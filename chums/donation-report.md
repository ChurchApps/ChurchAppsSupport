**How To Make A Donation Report**

<img width="1220" alt="Donation report 1" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/2b9a3b6a-2618-4ad5-b7b0-1866c3791f24">

In the Chums dashboard click on the Donation icon

<img width="1218" alt="Donation report 2" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/bc4d1315-6568-4796-b0fc-d51511f57b35">

Enter dates or leave the default dates. Click run report. This will produce a graph showing donations. You can print or download a csv with the data. This report will not show individual donors.

<img width="1217" alt="Donation report 3" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/7f166a00-41cb-472c-a35b-ee3c32070db1">

To view individual donations click on the batch you would like to see.

<img width="1214" alt="Donation report 4" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/4736ab61-3290-4f33-9d64-5c7c27e80a3d">

You can download a csv file showing the results. This will include the donor and the amount given.
