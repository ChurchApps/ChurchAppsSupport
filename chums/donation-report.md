**How To Make A Donation Report**

1. <img width="1220" alt="Donation report 1" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/2b9a3b6a-2618-4ad5-b7b0-1866c3791f24">
  In the Chums dashboard click on the Donation icon

2. <img width="1347" alt="Donation Reports 2" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/408c21eb-3fff-477c-ad25-23362795ba20">
  The default Donation Summary is for the past year. You can choose whatever dates you would like to show. Click "Run Report".

3. <img width="1345" alt="Donation Reports 3" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/6fcf5653-5164-4dad-b470-2de754d493f5">
  To download a CSV of the Donation Summary click the download icon. This will produce a dropdown menu. Choose "Fund Summary" and a CSV file will download to your Downloads File. This file does not include donors names.

4. <img width="1345" alt="Donation Reports 4" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/6f63574c-b8f3-4ade-ae77-54c122601093">
  Click on the "Detailed Summary" to produce a CSV file which includes each donors name, how much they have given and which fund they gave to during the "Filter Report" dates. 

5. https://www.dropbox.com/scl/fi/h3mdsaiqd1mzcensyyqwj/Pages-Donation-Report-Template.pages?rlkey=1crsmvkz14grrq0myzsvlw7hx&st=h63ut6pm&dl=0
   Download this template or create your own Pages document to mail merge with the CSV file (open in Numbers) for a donation report.

6. https://www.dropbox.com/scl/fi/j0oajnvqjsg9lsrpy1k5v/Word-Donation-Report-Template.docx?rlkey=zkmvjij7t732yd6p6xaxh8dm1&st=y6dqk6ig&dl=0
   Download this template or create your own Word document to mail merge with the CSV file (open in Excel) for a donation report.

7. <img width="1217" alt="Donation report 3" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/7f166a00-41cb-472c-a35b-ee3c32070db1">
  To view individual donations click on the batch you would like to see.

8. <img width="1214" alt="Donation report 4" src="https://github.com/ChurchApps/ChurchAppsSupport/assets/127863068/4736ab61-3290-4f33-9d64-5c7c27e80a3d">
  You can download a csv file showing the results. This will include the donor and the amount given in that batch.
