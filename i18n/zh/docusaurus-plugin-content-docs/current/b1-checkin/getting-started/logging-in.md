# 登录

<div class="article-intro">

在终端可用于签到之前，您需要使用具有教会访问权限的 ChurchApps 帐户登录。本页介绍首次登录过程、教会选择、自动登录行为以及如何登出。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在 Android 平板电脑上安装 B1 Church Checkin（请参阅[入门指南](./index.md)）
- 准备好您的 ChurchApps 帐户电子邮件和密码

</div>

## 首次登录

1. 打开 B1 Church Checkin 应用。启动画面之后，您将看到带有 B1 Church Checkin 标志的登录界面。
2. 在第一个字段中输入您的**电子邮件地址**。
3. 在第二个字段中输入您的**密码**。
4. 点击 **Login** 按钮。

应用验证您的凭据并登录。如果电子邮件或密码不正确，您将在屏幕底部看到错误消息。

:::info
这与您用于 B1 Admin 或任何其他 ChurchApps 产品的电子邮件和密码相同。您不需要为签到终端单独创建帐户。
:::

## 选择您的教会

成功登录后，应用会加载与您帐户关联的教会列表。每个教会显示为一张卡片，显示教会名称和标志（如果在 B1 Admin 中配置了标志）。

- 如果您的帐户只关联了**一个教会**，您仍然会看到选择界面。点击您的教会继续。
- 如果您的帐户关联了**多个教会**，请点击您希望此终端服务的教会。

选择教会后，应用会加载该教会的品牌和外观设置。您的教会标志将显示在所有后续界面的标题栏中。

## 再次访问时自动登录

首次成功登录后，应用会保存您的电子邮件、密码和所选教会。后续启动时：

- 应用会使用已保存的凭据自动登录。
- 如果您之前选择了教会，它会恢复该选择并直接带您进入**聚会界面**。
- 如果已保存的凭据不再有效，应用会返回登录界面，以便您重新登录。

:::tip
由于终端会自动登录，您只需在初始设置时输入一次凭据。之后，打开平板电脑即可使用。
:::

## 登出

登出功能故意隐藏，以防使用终端的教会成员意外退出。要登出：

1. 在两秒内快速点击标题栏中的**教会标志** **七次**。
2. 会出现一个确认对话框，询问您是否要登出。
3. 点击 **Logout** 确认，或点击 **Cancel** 保持登录状态。

这将清除已保存的凭据并返回登录界面。

:::warning
登出将清除所有已保存的凭据和教会选择。您需要重新输入电子邮件和密码才能使用终端。仅在需要切换帐户或重新配置终端时才登出。
:::

## 后续步骤

登录后，您可以[设置打印机](./printer-setup)或直接进入[签到流程](../check-in/)。
