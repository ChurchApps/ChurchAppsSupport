# 打印机设置

<div class="article-intro">

B1 Church Checkin 可以在网络标签打印机上打印姓名标签和家长接送凭条。打印机设置是可选的——您可以在没有打印机的情况下使用应用，仅用于考勤追踪。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [登录](./logging-in) B1 Church Checkin 应用并选择您的教会
- 将兼容的网络标签打印机连接到与平板电脑相同的 Wi-Fi 网络

</div>

## 访问打印机设置

您可以随时通过点击屏幕最顶部的**打印机状态栏**来打开打印机设置界面。这条细栏显示应用版本号和当前打印机连接状态。

## 扫描打印机

打开打印机设置界面时，应用会自动扫描您的本地网络以查找兼容的打印机。扫描过程中，您将看到带有"正在扫描打印机"消息的加载指示器。

扫描完成后，您将看到已发现的打印机列表。每张打印机卡片显示：

- **打印机型号**名称
- 打印机在网络上的 **IP 地址**

列表底部始终包含**无打印机**选项。

:::warning
平板电脑和打印机必须连接到同一本地网络，扫描才能发现打印机。如果您的打印机未出现，请验证两台设备是否在同一 Wi-Fi 网络上，以及打印机是否已开启。
:::

## 选择打印机

1. 点击您要使用的打印机。所选打印机卡片将以勾选标记高亮显示。
2. 要在没有物理打印机的情况下运行（用于测试或仅考勤模式），请选择**无打印机**。

## 测试打印

在保存选择之前，您可以验证连接是否正常：

1. 从列表中选择一台打印机。
2. 点击屏幕底部的**测试打印**按钮。
3. 应用会向打印机发送测试标签。如果打印机连接正确，将打印出测试页。

如果您在选择**无打印机**时点击**测试打印**，应用将显示提醒您未选择打印机的警告。

:::tip
选择打印机后，请务必进行测试打印以确认连接，然后再进行下次聚会。
:::

## 保存您的选择

当您对打印机选择满意时：

1. 点击屏幕底部的**保存并重启**按钮。
2. 应用保存您的打印机选择并重启以应用设置。

您的打印机选择会在应用重启之间保留。除非您想更换打印机，否则不需要再次设置。

:::info
如果您将终端移至不同位置或更改了网络设置，请返回打印机设置，重新扫描打印机并更新您的选择。
:::

## 后续步骤

打印机配置完成后，您就可以开始签到了。请参阅[签到流程](../check-in/)了解完整流程的工作方式，从[选择聚会](../check-in/selecting-service)开始。
