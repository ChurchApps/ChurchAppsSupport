# B1 Church Checkin 入门指南

<div class="article-intro">

设置 B1 Church Checkin 只需几分钟。本节将引导您在 Android 平板电脑上安装应用、使用 ChurchApps 帐户登录，以及可选地配置网络标签打印机。

</div>

## 您需要准备

- **一台 Android 平板电脑** -- 10 英寸平板电脑最适合终端体验。
- **一个 ChurchApps 帐户** -- 您用于 B1 Admin 或其他 ChurchApps 产品的电子邮件和密码。
- **一台网络标签打印机**（可选）-- 用于打印姓名标签和家长接送凭条。应用可以扫描您的本地网络以查找兼容的打印机。

## 安装

1. 在您的 Android 平板电脑上打开 **Google Play Store**。
2. 搜索 **B1 Church Checkin**。
3. 点击 **安装** 并等待下载完成。
4. 打开应用。初始化时您将看到启动画面。

## 首次设置

首次启动应用时，您将经历三个步骤：

1. **[登录](./logging-in)** -- 输入您的电子邮件和密码进行验证。
2. **[选择您的教会](./logging-in)** -- 如果您的帐户关联了多个教会，请选择此终端应服务的教会。
3. **[设置打印机](./printer-setup)** -- 配置网络打印机以打印标签，如果不需要标签，可以跳过此步骤。

首次登录后，应用会保存您的凭据和教会选择。后续启动时，它将自动直接进入聚会界面。

:::info
应用每次启动时都会检查空中更新。如果有可用更新，它会在您到达登录界面之前自动下载并应用。
:::

:::tip
请确保您的教会管理员在设置终端之前已在 B1 Admin 中配置了聚会、聚会时间和小组。详情请参阅[考勤设置指南](../../b1-admin/attendance/setup.md)。
:::
