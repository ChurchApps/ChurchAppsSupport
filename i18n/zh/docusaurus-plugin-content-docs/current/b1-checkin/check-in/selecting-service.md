# 选择聚会

<div class="article-intro">

每次签到的第一步是选择您要参加的聚会。聚会界面在应用加载完成后立即显示，它决定了签到流程中可用的聚会时间和小组。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [登录](../getting-started/logging-in) B1 Church Checkin 应用并选择您的教会
- 确保教会管理员已在 B1 Admin 中[配置了聚会和聚会时间](../../b1-admin/attendance/setup.md)

</div>

## 工作原理

1. 登录后（或再次访问时自动登录后），应用显示**聚会界面**。
2. 您将看到为您的教会配置的所有聚会列表。每个聚会显示为一张卡片，显示聚会名称（例如，"周日上午"或"周三晚间"）。
3. 点击您要参加的聚会。

应用加载与该聚会关联的聚会时间和小组，然后带您进入[成员查找界面](./looking-up-members)。

:::info
聚会由教会管理员在 B1 Admin 的考勤部分配置。如果您没有看到预期的聚会，请让管理员验证它是否已在[考勤设置](../../b1-admin/attendance/setup.md)中创建。
:::

## 幕后发生了什么

当您选择聚会时，应用从服务器获取三样东西：

- 该聚会的**聚会时间**（例如，一个聚会可能有 9:00 和 11:00 两个时间段）。
- 每个聚会时间可用的**小组**（例如，婴儿室、学前班、小学部）。
- **小组与聚会时间的关联**，决定了哪些小组在哪些时间可用。

这些数据在本地缓存，使签到流程的其余部分快速且响应灵敏。

:::tip
如果您的教会只配置了一个聚会，您仍然需要点击它才能继续。应用不会自动选择唯一的聚会。
:::

## 下一步

选择聚会后，您将通过电话号码或姓名[查找您的家庭](./looking-up-members)。
