# 审查您的家庭

<div class="article-intro">

选择一个人后，应用程序加载您的整个家庭并在家庭审查屏幕上显示它。这是签到流程的中心枢纽，您可以看到家庭中的每个人并在签到前管理其小组分配。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 从服务屏幕[选择服务](./selecting-service)
- [查找成员](./looking-up-members)并从搜索结果中选择他们

</div>

## 查看家庭成员

每个家庭成员显示为一个卡片，具有：

- 其**照片**（如果存档中没有照片则为占位符图标）
- 其**显示名称**
- **其当前小组分配的摘要**（卡片折叠时显示为名称下方的小芯片）
- 右侧的**展开/折叠箭头**

## 展开成员

点击成员的卡片以展开它。展开后，您将看到选定服务中每个**服务时间**的一行。每行显示：

- **服务时间名称**（例如，"9:00 AM" 或 "11:00 AM"）
- 一个显示当前分配的小组的**按钮**，或"选择小组"如果还没有分配小组

点击小组按钮进入[小组选择屏幕](./group-assignment)，您可以在其中为该人和服务时间选择或更改小组。

展开的卡片还显示**成员**、**访客**和**志愿者**芯片。点击一个以设置此人如何签到 -- 成员是默认的。类型对儿童房间很重要：志愿者计入房间的志愿者与儿童比例，访客计入您的教会设置的任何访客容量。

再次点击成员的卡片以折叠它。

:::tip
您不需要为每个家庭成员分配小组。参加主礼拜服务的成人可以不分配小组。仅为需要签到特定班级或计划的成员分配小组。
:::

## 从此屏幕导航

从家庭审查屏幕，您有三个选项：

- **展开成员**并点击服务时间按钮以[分配或更改小组](./group-assignment)。
- **点击成员列表下方的添加访客按钮**以[添加访客](./adding-guests)到您的家庭。
- **点击屏幕底部的签到按钮**以[完成签到](./completing-checkin)家庭中的每个人。

:::info
来自前一次签到到同一服务的任何小组分配都会自动预加载。如果您的孩子每周参加相同的小组，您可能不需要更改任何内容 -- 只需审查和签到。
:::

## 后续步骤

- [分配小组](./group-assignment)给需要签到特定班级或计划的家庭成员。
- [添加访客](./adding-guests)如果有人和您的家庭一起访问。
- [完成签到](./completing-checkin)当每个人的分配看起来正确时。
