# 完成签到

<div class="article-intro">

审查了您的家庭并进行了任何需要的小组分配后，您就可以完成签到了。这是自助亭工作流中的最后一步 -- 应用程序提交出勤情况、打印标签并为下一个家庭重置。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 在家庭审查屏幕上[审查您的家庭](./household-review)
- [分配小组](./group-assignment)给任何需要签到特定班级或计划的家庭成员
- 可选[添加任何访客](./adding-guests)访问您的家庭

</div>

## 如何签到

1. 从**家庭审查屏幕**，点击屏幕底部的**签到**按钮。
2. 应用程序向服务器提交出勤数据并显示带有绿色复选标记和欢迎消息的**成功屏幕**。

就这样。您家庭的出勤情况已被记录。

## 房间已满和志愿者比例

如果您的教会在其房间上配置了[安全限制](../../b1-admin/attendance/checkin-safety)，服务器在保存前检查它们：

- 如果选定的房间**已满或已关闭**，签到不会进行，应用程序会命名房间，以便您可以选择不同的房间。
- 如果儿童房间**缺少志愿者**来达到其比例，应用程序要么显示警告，工作人员可以确认继续进行，要么根据您的教会配置比例执行方式完全阻止签到。

## 标签打印

如果配置了网络打印机，应用程序将在签到后自动打印标签：

- **姓名标签**为分配给启用了**打印名牌**设置的小组的每个人打印。姓名标签包括该人的姓名、其小组分配以及存档中的任何过敏症/备注信息。
- **家长取件单**在任何签到的人在启用了**家长取件**设置的小组中时打印。取件单列出了儿童、其小组分配和唯一的**4 字符安全代码**。

:::info
同一安全代码出现在儿童的姓名标签和家长的取件单上。在取件时，志愿者匹配代码以验证正确的成人取走每个儿童。
:::

安全代码为每次签到生成，仅使用辅音和数字（排除元音以避免形成不当单词）。

:::warning
如果标签没有打印，通过点击**教会徽标**七次来打开管理员设置，然后点击**更改打印机**以验证打印机连接。有关故障排除步骤，请参见[打印机设置](../getting-started/printer-setup)。
:::

## 签到后发生的事情

- 如果配置了打印机，应用程序将打印所有标签，然后自动返回到**查找屏幕**，准备下一个家庭。
- 如果未配置打印机，成功屏幕将显示几秒钟，然后自动返回到**查找屏幕**。

您不需要点击任何内容来回到查找屏幕 -- 应用程序自动处理过渡。

:::tip
应用程序在每次签到后完全重置，因此没有一个家庭看到另一个家庭信息的风险。
:::

## 记录的内容

当您点击**签到**时，应用程序向服务器为有小组分配的每个家庭成员发送以下内容：

- **被签到的人**
- **他们参加的服务**
- **服务时间**和**他们分配的小组**

此数据出现在 B1 Admin 的出勤部分中，您的教会管理员可以在其中查看和管理出勤记录。有关详细信息，请参见[签到管理指南](../../b1-admin/attendance/check-in.md)。
