# 完成签到

<div class="article-intro">

查看完家庭并完成所需的小组分配后，您就可以完成签到了。这是终端流程的最后一步——应用提交考勤数据、打印标签，然后为下一个家庭重置。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在家庭查看界面上[查看您的家庭](./household-review)
- 为需要签到特定班级或项目的家庭成员[分配小组](./group-assignment)
- 可选地[添加访客](./adding-guests)（如果有人随您家庭来访）

</div>

## 如何签到

1. 在**家庭查看界面**上，点击屏幕底部的 **Check-in** 按钮。
2. 应用将考勤数据提交到服务器，并显示带有绿色勾选标记和欢迎消息的**成功界面**。

就这么简单。您家庭的考勤已经记录。

## 标签打印

如果配置了网络打印机，应用会在签到后自动打印标签：

- **姓名标签**为分配到启用了**打印姓名标签**设置的小组的每个人打印。姓名标签包括人员姓名、小组分配以及过敏/备注信息（如果档案中有记录）。
- **家长接送凭条**在任何签到人员所在小组启用了**家长接送**设置时打印。接送凭条列出儿童姓名、小组分配和唯一的 **4 字符安全码**。

:::info
相同的安全码同时出现在儿童的姓名标签和家长的接送凭条上。接送时，志愿者核对代码以验证正确的成人接走每个孩子。
:::

安全码在每次签到时重新生成，仅使用辅音字母和数字（排除元音以避免形成不当词语）。

:::warning
如果标签未打印，请检查屏幕顶部的打印机状态栏。您可以点击它访问打印机设置并验证连接。有关故障排除步骤，请参阅[打印机设置](../getting-started/printer-setup)。
:::

## 签到后会发生什么

- 如果配置了打印机，应用打印所有标签后自动返回**查找界面**，准备迎接下一个家庭。
- 如果未配置打印机，成功界面显示几秒钟后自动返回**查找界面**。

您不需要点击任何东西即可返回查找界面——应用会自动处理过渡。

:::tip
应用在每次签到后完全重置，因此不存在一个家庭看到另一个家庭信息的风险。
:::

## 记录的内容

当您点击 **Check-in** 时，应用会为每个有小组分配的家庭成员向服务器发送以下信息：

- 正在签到的**人员**
- 他们参加的**聚会**
- 他们被分配到的**聚会时间**和**小组**

这些数据出现在 B1 Admin 的考勤部分，教会管理员可以在那里查看和管理考勤记录。详情请参阅[签到管理指南](../../b1-admin/attendance/check-in.md)。
