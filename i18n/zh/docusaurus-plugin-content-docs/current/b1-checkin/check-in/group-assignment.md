# 分配小组

<div class="article-intro">

当您点击家庭审查屏幕上的服务时间按钮时，应用程序打开小组选择屏幕。在这里，您选择家庭成员应该为该特定服务时间参加哪个小组或班级。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 在家庭审查屏幕上[审查您的家庭](./household-review)
- 展开您想要分配的成员并点击服务时间按钮
- 确保您的教会管理员已在 B1 Admin 中[创建小组](../../b1-admin/groups/creating-groups.md)并将它们链接到服务时间

</div>

## 小组如何组织

小组按**类别**显示。每个类别显示为一个卡片，显示：

- **类别名称**（例如，"儿童"、"青年"、"成人"）
- **该类别中小组的数量**

点击类别卡片以展开它并查看其中的单个小组。再次点击它以折叠它。

## 选择小组

1. 点击**类别**以展开它。
2. 审查该类别内的小组列表。每个小组显示其名称及右侧的选择图标。
3. 点击您想要分配的**小组名称**。

应用程序保存您的选择并返回您到[家庭审查屏幕](./household-review)。该成员的服务时间按钮现在将显示您选择的小组的名称。

## 年龄和年级指导

如果您的教会在其房间上设置了年龄或年级范围，列表会指导您找到正确的房间：该人符合的房间被**突出显示**，该人年龄或年级之外的房间被**变淡**。变淡的房间仍然可以被选择，但工作人员必须确认该选择。年龄和年级是从您的教会的年级晋升日期计算的，因此房间在学年翻转期间保持准确。

## 选择"无"

如果您决定家庭成员不应该在此服务时间为任何小组分配，点击屏幕底部的**无**按钮。这会清除该服务时间的任何现有小组分配并返回您到家庭审查屏幕。

:::tip
对于参加主礼拜服务且不需要签到特定小组或班级的成人，使用**无**选项。
:::

## 更改小组分配

要更改已进行的小组分配：

1. 回到[家庭审查屏幕](./household-review)。
2. 展开您想要更改其小组的成员。
3. 点击服务时间按钮（现在显示当前分配的小组名称）。
4. 选择不同的小组或点击**无**。

:::info
小组和类别由您的教会管理员在 B1 Admin 中配置。如果小组缺失或在错误的类别中，要求您的管理员更新[小组设置](../../b1-admin/groups/creating-groups.md)。
:::

## 后续步骤

在分配小组后，返回到[家庭审查屏幕](./household-review)以：

- 为其他家庭成员分配小组
- [添加访客](./adding-guests)如果有人和您的家庭一起访问
- [完成签到](./completing-checkin)当所有分配准备好时
