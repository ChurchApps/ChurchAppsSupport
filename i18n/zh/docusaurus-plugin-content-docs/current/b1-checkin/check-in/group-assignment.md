# 分配小组

<div class="article-intro">

当您在家庭查看界面上点击聚会时间按钮时，应用会打开小组选择界面。在这里，您可以选择家庭成员在该特定聚会时间应参加的小组或班级。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在家庭查看界面上[查看您的家庭](./household-review)
- 展开要分配的成员并点击聚会时间按钮
- 确保教会管理员已在 B1 Admin 中[创建了小组](../../b1-admin/groups/creating-groups.md)并将其关联到聚会时间

</div>

## 小组的组织方式

小组按**类别**显示。每个类别显示为一张卡片，包含：

- **类别名称**（例如，"儿童"、"青少年"、"成人"）
- 该类别中的**小组数量**

点击类别卡片展开它，查看其中的各个小组。再次点击可折叠。

## 选择小组

1. 点击一个**类别**展开它。
2. 查看该类别中的小组列表。每个小组显示其名称，右侧有选择图标。
3. 点击您要分配的**小组名称**。

应用保存您的选择并返回[家庭查看界面](./household-review)。该成员的聚会时间按钮现在将显示您选择的小组名称。

## 选择"无"

如果您决定某个家庭成员在此聚会时间不应分配到任何小组，请点击屏幕底部的**无**按钮。这将清除该聚会时间的任何现有小组分配，并返回家庭查看界面。

:::tip
对于参加主要敬拜聚会且不需要签到特定小组或班级的成人，请使用**无**选项。
:::

## 更改小组分配

要更改已经完成的小组分配：

1. 返回[家庭查看界面](./household-review)。
2. 展开要更改小组的成员。
3. 点击聚会时间按钮（现在显示当前分配的小组名称）。
4. 选择不同的小组或点击**无**。

:::info
小组和类别由教会管理员在 B1 Admin 中配置。如果缺少小组或小组在错误的类别中，请让管理员更新[小组设置](../../b1-admin/groups/creating-groups.md)。
:::

## 后续步骤

分配完小组后，返回[家庭查看界面](./household-review)以：

- 为其他家庭成员分配小组
- 如果有人随您家庭来访，[添加访客](./adding-guests)
- 当所有分配完成后，[完成签到](./completing-checkin)
