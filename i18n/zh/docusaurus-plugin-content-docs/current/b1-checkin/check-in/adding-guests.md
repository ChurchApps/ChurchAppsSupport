# 添加访客

<div class="article-intro">

如果有人正在拜访您的家庭并需要与您的家庭一起签到，您可以直接从终端将他们添加为访客。这对于来访的儿童、亲戚或尚未在教会系统中的朋友非常有用。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [查找您的家庭](./looking-up-members)并从搜索结果中选择一位成员
- 在[家庭查看界面](./household-review)上，您可以看到您的家庭成员

</div>

## 如何添加访客

1. 在**家庭查看界面**上，点击位于家庭成员列表下方的**添加访客**按钮。
2. 添加访客表单出现，包含两个字段：
   - **名字** -- 输入访客的名字。
   - **姓氏** -- 输入访客的姓氏。
3. 点击**添加访客**按钮保存。

访客被添加到您的家庭中，您将返回家庭查看界面。新访客将出现在您的成员列表中，与其他家庭成员并列。

:::info
如果访客的姓名与系统中已有的非成员人员匹配，应用将关联到该现有记录而不是创建重复记录。这有助于保持教会数据库的整洁。
:::

## 添加访客后

访客出现在您的家庭列表中后，您可以：

- **展开他们的卡片**并[将他们分配到小组](./group-assignment)，就像其他家庭成员一样。
- **[签到](./completing-checkin)**整个家庭，包括访客。

## 取消

如果您改变主意，请点击添加访客表单底部的**取消**按钮，返回家庭查看界面而不添加任何人。

:::tip
在终端添加访客会在教会数据库中创建一条人员记录。教会管理员稍后可以在 B1 Admin 的[人员](../../b1-admin/people/adding-people.md)部分更新访客的完整详细信息。
:::

## 下一步

添加访客并根据需要[分配小组](./group-assignment)后，您就可以[完成签到](./completing-checkin)了。
