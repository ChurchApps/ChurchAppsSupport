# 签到流程

<div class="article-intro">

B1 Church Checkin 的签到过程遵循简单的分步流程。教会成员走到终端前，找到他们的家庭，确认小组分配，然后签到——全程不到一分钟。本节介绍终端签到体验的每一个步骤。

</div>

## 签到流程概览

1. **[选择聚会](./selecting-service)** -- 选择您要参加的聚会（例如，周日上午、周日下午、周三聚会）。
2. **[查找您的家庭](./looking-up-members)** -- 通过电话号码或姓氏在系统中查找自己。
3. **[查看您的家庭](./household-review)** -- 查看所有家庭成员及其当前的小组分配。
4. **[分配小组](./group-assignment)** -- 为每个聚会时间选择每个家庭成员应该参加的班级或小组。
5. **[添加访客](./adding-guests)**（可选）-- 如果有人随您一同来访，将访客添加到您的家庭。
6. **[完成签到](./completing-checkin)** -- 点击签到按钮提交考勤并打印标签。

签到完成后，应用显示成功界面并自动返回查找界面，准备迎接下一个家庭。

:::tip
整个流程设计为教会成员可以独立使用。志愿者不需要操作终端，但在附近安排人员帮助初次来访者总是个好主意。
:::

## 开始之前

在使用签到终端之前，请确保在 B1 Admin 中配置了以下内容：

- **聚会和聚会时间**已在考勤设置中设置。请参阅[考勤设置指南](../../b1-admin/attendance/setup.md)。
- **小组**已创建并关联到相应的聚会时间。请参阅[创建小组](../../b1-admin/groups/creating-groups.md)。
- **小组设置**已配置用于标签打印：为需要姓名标签的小组启用**打印姓名标签**，为需要接送凭条的儿童小组启用**家长接送**。
- **人员**已添加到系统中并关联到家庭。请参阅[添加人员](../../b1-admin/people/adding-people.md)。

:::info
如果您尚未设置终端本身，请参阅[入门指南](../getting-started/)了解安装、登录和打印机配置说明。
:::
