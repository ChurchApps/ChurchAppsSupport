# 签到工作流

<div class="article-intro">

B1 Church Checkin 中的签到流程遵循简单的分步流程。教会成员走到自助亭，找到他们的家庭，确认小组分配并签到 -- 所有这些都在不到一分钟内完成。此部分涵盖自助亭签到体验的每个步骤。

</div>

## 一目了然的签到流程

1. **[选择服务](./selecting-service)** -- 选择您参加的服务（例如，周日上午、周日下午、平日）。
2. **[查找您的家庭](./looking-up-members)** -- 按电话号码或姓氏搜索以在系统中找到自己。
3. **[审查您的家庭](./household-review)** -- 查看所有家庭成员和其当前小组分配。
4. **[分配小组](./group-assignment)** -- 为每个服务时间选择每个家庭成员应该参加的班级或小组。
5. **[添加访客](./adding-guests)**（可选）-- 如果访客和您一起加入，将其添加到您的家庭。
6. **[完成签到](./completing-checkin)** -- 点击签到按钮以提交出勤并打印标签。
7. **[签出](./checking-out)**（有人值守的台） -- 在取件时，验证安全代码和取件人，使用可信取件列表、寻呼家长和标签重印。

签到完成后，应用程序显示成功屏幕并自动返回到查找屏幕，为下一个家庭做好准备。

:::tip
整个流程专为教会成员独立使用而设计。志愿者不需要操作自助亭，尽管在附近帮助首次访问者始终是个好主意。
:::

## 开始前

在使用签到自助亭之前，确保在 B1 Admin 中配置了以下内容：

- **服务和服务时间**在出勤设置下设置。请参见[出勤设置指南](../../b1-admin/attendance/setup.md)。
- **小组**已创建并链接到相应的服务时间。请参见[创建小组](../../b1-admin/groups/creating-groups.md)。
- **小组设置**为标签打印配置：为需要姓名标签的小组启用**打印名牌**，为需要取件单的儿童小组启用**家长取件**。
- **人员**已添加到系统中，具有家庭关联。请参见[添加人员](../../b1-admin/people/adding-people.md)。

:::info
如果您还没有设置自助亭本身，请参见[入门](../getting-started/)以了解安装、登录和打印机配置说明。
:::
