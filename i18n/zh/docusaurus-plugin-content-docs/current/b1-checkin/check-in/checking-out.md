# 签出与儿童安全

<div class="article-intro">

签出结束了儿童签到过程：家长出示他们接送标签上的安全代码，信息亭验证谁要接孩子，然后儿童被签出。配备人员的工作站还可获得安全工具 — 信任接送人验证、给家长发页面短信、安全标签重新打印和紧急广播。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 签出可在信息亭管理员设置中设置为**配备人员**模式的工作站上使用
- 儿童必须已通过[完成签到](./completing-checkin)并打印了载有安全代码的接送标签
- 分页和紧急广播需要您的教会在B1 Admin中连接了短信提供商

</div>

## 开始签出

1. 在配备人员的工作站上，点击查找屏幕上的**签出**。
2. 从家族接送标签上输入4个字符的**安全代码**。您可以输入它、使用屏幕上的键盘或用USB或蓝牙扫描器扫描标签的条形码 — 输入所有4个字符后，代码会自动提交。
3. 信息亭显示在该代码下签入的儿童。

## 验证谁在接孩子

签出屏幕询问谁在接孩子：

- **信任接送人**对于该家庭显示为可点击的卡片，带有他们的照片和关系 — 点击站在你面前的人。
- **家庭成员**也显示在照片网格中。
- **其他**让您为列表上没有的人输入名字。

如果输入的名字与标记为**未授权**用于该家庭的人匹配，信息亭会显示警告并阻止签出。工作人员可以选择**覆盖**继续 — 覆盖会记录在出勤记录中，包括该人的名字。

一旦确认接送人，点击签出。接送人的名字会与出勤记录一起存储。

:::info
信任和未授权接送人由教会工作人员在B1 Admin中每个人的页面上管理 — 请参见[签到安全](../../b1-admin/attendance/checkin-safety#trusted-and-not-authorized-pickup-people)。
:::

## 给家长发页面

在服事期间需要一位家长 — 换尿布、哭泣的孩子？从配备人员的工作站的签出屏幕，工作人员可以发送**页面**：通过教会的短信提供商向孩子的父母或监护人发送短信。选择退出短信或没有手机号码的父母会被跳过，信息亭会显示发送了多少条消息。

## 重新打印标签

如果姓名标签或接送标签丢失或损坏，配备人员的工作站的工作人员可以在输入安全代码后从签出屏幕**重新打印**家族的标签。重新打印使用与原始签到相同的打印机和标签模板。

## 紧急广播

在紧急情况下，工作人员可以立即给当前服事的**每个签入的孩子**的监护人发短信：

1. 打开信息亭**管理员设置**（快速点击标题徽标7次，加上PIN，如果设置了的话）。
2. 点击**紧急广播**。
3. 输入消息，然后在确认字段中输入**紧急情况** — **发送广播**按钮会一直禁用，直到您这样做。
4. 信息亭报告有多少部电话收到了消息，以及有多少人被跳过（选择退出或没有手机号码）。

:::warning
广播会发送给所选服事的每个签入家庭。仅在真正的紧急情况下使用它 — 疏散、禁闭、严重天气。
:::
