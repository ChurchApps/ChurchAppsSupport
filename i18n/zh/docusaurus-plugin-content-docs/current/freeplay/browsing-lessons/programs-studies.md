# 课程系列与学习单元

<div class="article-intro">

课程浏览器采用逐步引导的方式，帮助您精确找到要播放的内容。每一步都显示一个包含图片和标题的三列卡片网格，方便您使用电视遥控器浏览完整的课程库。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 确保您的电视已连接互联网（浏览课程目录需要联网）
- 使用侧边栏导航到**浏览课程系列**屏幕

</div>

## 浏览课程系列

1. 打开**浏览课程系列**屏幕
2. 课程系列以图片卡片网格形式显示，每行三个
3. 使用遥控器上的方向键高亮选中一个课程系列
4. 按**确认**键打开它

每张课程系列卡片显示其封面图片。如果没有可用图片，将显示占位卡片。

## 选择学习单元

选择课程系列后，**选择学习单元**屏幕将加载该系列中的所有学习单元。

1. 学习单元卡片以三列网格形式显示，包含封面图片和名称
2. 导航到您想要的学习单元并按**确认**键
3. 按**返回**键回到课程系列列表

## 选择课程

选择学习单元后，**选择课程**屏幕将显示该单元中的所有课程。

1. 课程卡片显示封面图片和课程标题
2. 导航到您想要的课程并按**确认**键
3. 按**返回**键回到学习单元列表

## 选择场景

课程详情屏幕显示课程名称、标题和描述，叠加在课程封面图片上。描述下方会显示一个或多个场景按钮。

场景代表同一课程的不同版本。例如，一节课可能有分别针对"学龄前"、"小学"和"青少年"年龄组的场景。每个场景包含适合该教学环境的媒体文件。

1. 使用方向键高亮选中您想要的场景
2. 按**确认**键开始加载内容
3. FreePlay 获取该场景的播放列表并启动播放器

:::tip
如果您只看到一个场景按钮，说明该课程只有一个版本。选择它即可立即开始播放。
:::

## 返回导航

在浏览过程中的任何位置，按遥控器上的**返回**键可回到上一级：

- 从课程详情屏幕，**返回**回到课程列表
- 从课程列表，**返回**回到学习单元列表
- 从学习单元列表，**返回**回到课程系列网格
- 从课程系列网格，**返回**打开侧边栏
