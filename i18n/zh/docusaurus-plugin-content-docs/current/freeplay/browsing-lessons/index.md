# 浏览课程

<div class="article-intro">

FreePlay 内置了课程浏览器，让您可以浏览完整的课程、学习系列和单元库。当您想手动选择要播放的课程，而不依赖教室日程或计划时，这个功能非常实用。

</div>

## 浏览流程

课程按以下层级组织：

1. **课程系列** - 顶级课程分类（例如，儿童事工系列）
2. **学习单元** - 课程系列中的课程分组（例如，多周教学单元）
3. **课程** - 包含独立内容的单节课程
4. **场景** - 针对特定年龄组或教学环境定制的课程版本

您首先选择一个课程系列，然后进入学习单元，选择课程，最后选择场景开始播放。

## 访问课程浏览器

课程浏览器可通过**浏览课程系列**屏幕访问。使用侧边栏导航到该页面，或从导航中选择浏览选项。

:::info
浏览课程需要互联网连接以加载课程目录。选择场景并开始播放后，内容将直接流式传输，无需单独的下载步骤。
:::
