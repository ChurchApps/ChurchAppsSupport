# 教室模式

<div class="article-intro">

教室模式将您的电视连接到教会中的特定房间。配对后，FreePlay 会自动下载该房间排定的课程，这样您的志愿者只需在上课时按**开始课程**即可。

</div>

## 工作原理

1. **找到您的教会** - 按名称搜索您的教会
2. **选择教室** - 选择该电视所属的教室
3. **下载课程** - FreePlay 获取排定的课程并下载所有媒体文件
4. **播放** - 按**开始课程**按钮，课程自动播放

电视会记住配对的教会和教室，因此您只需进行一次设置。每周，FreePlay 会检查当天排定的课程并下载所有新内容。

:::tip
在配对电视之前，请确保您的教室已在 [freeplay.church](https://freeplay.church) 上配置完成。如果选择教会后没有显示教室，请访问网站进行设置。
:::
