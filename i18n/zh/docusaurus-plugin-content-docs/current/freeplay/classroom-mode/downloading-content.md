# 下载课程内容

<div class="article-intro">

选择教会和教室后，FreePlay 会自动获取当天排定的课程并开始下载所有媒体文件。此屏幕是您等待内容准备就绪后再开始课程的页面。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 选择您的教会和教室——参见[选择您的教会和教室](./selecting-room)
- 确保您的电视已连接互联网以完成初始下载

</div>

## 下载工作原理

1. FreePlay 检查您教室的日程并加载指定的课程
2. 课程名称、标题和描述显示在屏幕上，叠加在课程封面图片上
3. 进度指示器显示**正在下载第 X 项，共 Y 项**，每个文件被保存到设备
4. 当所有文件下载完成后，带有播放图标的**开始课程**按钮出现

下载期间您无需进行任何操作。FreePlay 会自动处理一切。

## 分辨率

FreePlay 默认以 720p 下载内容。分辨率设置（720p 或 1080p）决定了下载视频和图片文件的质量。更高的分辨率在大屏电视上看起来更清晰，但下载时间更长，占用更多存储空间。

:::info
分辨率默认设置为 720p。内容按照您设备配置的分辨率下载。
:::

## 开始课程

下载完成后，**开始课程**按钮会在屏幕底部淡入显示。使用电视遥控器高亮选中它并按**确认**键开始播放。

## 自动刷新

FreePlay 每小时检查一次内容更新。如果当天日程发生变化，应用会自动获取新课程。您也可以按遥控器上的**返回**键回到教室选择屏幕，重新进入下载屏幕以强制刷新。

:::tip
在周日上午之前，趁您的建筑有良好的互联网连接时下载课程。下载完成后，内容可以离线播放，无需任何网络依赖。
:::

## 如果日程为空

如果所选教室没有排定课程，您将看到消息"无法加载日程。请确保已为此班级排定课程。"请在 [freeplay.church](https://freeplay.church) 检查您的日程，确保正确的课程已分配到此教室。
