# 播放课程

<div class="article-intro">

FreePlay 媒体播放器是为电视遥控器设计的全屏体验。它可以播放视频和图片幻灯片，自动或手动推进课程内容，让教师完全掌控课程节奏。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 下载课程——参见[下载课程内容](./downloading-content)
- 按**开始课程**按钮进入播放器

</div>

## 播放器控制

使用电视遥控器控制播放：

| 遥控器按钮 | 操作 |
|---|---|
| **确认 / 播放-暂停** | 暂停或恢复当前视频 |
| **右方向键 / 快进** | 跳到下一张幻灯片或视频 |
| **左方向键 / 快退** | 回到上一张幻灯片或视频 |
| **上方向键** | 打开消息选择器，跳转到特定项目 |
| **下方向键 / 返回 / 菜单** | 退出播放器并返回上一屏幕 |

:::tip
您也可以在支持触摸的遥控器上使用滑动手势。向左滑动前进，向右滑动后退，向下滑动打开消息选择器，向上滑动退出播放器。
:::

## 视频播放

播放视频时，您将看到全屏视频内容。按**确认**键或**播放-暂停**键会暂停视频，并显示暂停图标覆盖层和屏幕底部的进度条。再次按下即可恢复播放。

视频播放完成后会自动推进到课程中的下一项。如果视频设置为循环播放，它将重复播放直到您手动推进。

## 图片幻灯片

图片幻灯片以全屏方式显示。与视频不同，图片幻灯片不会自动推进——它们会停留在屏幕上，直到您按右方向键前进。这让教师可以完全掌控教学节奏。

## 消息选择器

在播放过程中按**上**方向键可打开消息选择器。它会显示课程中所有项目的网格，包含缩略图和标题。导航到任意项目并按**确认**键即可直接跳转。这在需要跳过或返回课程特定部分时非常实用。

## 退出播放器

按遥控器上的**下方向键**、**返回**或**菜单**键退出播放器。您将返回到下载屏幕（教室模式）、计划下载屏幕（计划模式）或内容浏览器（播放提供商内容时）。

:::info
播放器激活期间屏幕保持常亮，因此您无需担心电视在课程进行中进入休眠状态。
:::
