# 选择您的教会和教室

<div class="article-intro">

在 FreePlay 可以为您的教室下载课程之前，您需要告诉它使用哪个教会和教室。这是一次性设置——电视会记住您的选择，供以后使用。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 安装并启动 FreePlay——参见[入门指南](../getting-started/)
- 在 [freeplay.church](https://freeplay.church) 上注册您的教会并配置教室
- 从[配对模式屏幕](../getting-started/pairing-modes.md)选择**配对到教室**

</div>

## 查找您的教会

1. 从**选择配对模式**屏幕，选择**配对到教室**
2. 您将看到**查找您的教会**屏幕，顶部有一个搜索栏
3. 使用电视遥控器选择搜索栏，输入教会名称的至少四个字母
4. 搜索栏下方会出现匹配的教会列表
5. 在列表中导航到您的教会，按遥控器上的**确认**键

:::warning
如果您的教会未出现在搜索结果中，可能尚未注册。请访问 [freeplay.church](https://freeplay.church) 注册您的教会并配置教室。
:::

## 选择教室

选择教会后，**选择教室**屏幕会显示所有已配置的教室列表。

1. 使用遥控器上的上下方向键浏览教室列表
2. 高亮选中该电视所分配的教室
3. 按遥控器上的**确认**键

FreePlay 保存您的选择，并立即跳转到下载屏幕以获取排定的课程。

:::tip
如果您看到"未找到教室"，说明您教会的教室尚未配置。管理员可以在 [freeplay.church](https://freeplay.church) 上进行设置。
:::

## 更改选择

如需稍后切换到其他教会或教室，请在下载屏幕上按电视遥控器的**返回**键。这将带您回到教室选择屏幕，您可以选择其他教室，或继续按**返回**键回到教会搜索。
