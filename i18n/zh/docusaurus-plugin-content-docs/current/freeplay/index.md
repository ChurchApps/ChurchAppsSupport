# FreePlay 概览

<div class="article-intro">

FreePlay 是一款为教会设计的电视应用，用于在教室电视上显示课程内容。它运行在 Android TV、Amazon Fire TV 和 Apple TV 上，设计为周日早上无需互联网连接即可使用。志愿者可以在一周内预先下载课程材料，然后在上课时只需按下播放即可。

</div>

## 主要功能

- **离线播放** - 提前下载视频和图片幻灯片，在礼拜期间不依赖 Wi-Fi
- **教室配对** - 将电视连接到特定房间，自动接收排定的课程内容
- **计划配对** - 将设备链接到教会礼拜计划，协调各房间的内容
- **内容提供商** - 从 [Lessons.church](/docs/lessons-church/) 和 Bible Project 等外部来源浏览和下载内容
- **遥控器友好** - 使用标准电视遥控器的方向键和确认键即可导航整个应用
- **可折叠侧边栏** - 纤细的侧边栏保持导航便捷，同时不干扰屏幕显示

## 工作原理

启动 FreePlay 后，它会连接到您的内容提供商并检查可用内容。然后您可以浏览文件夹、下载媒体和播放课程——所有操作均针对大屏电视体验进行了优化。
