# FreePlay 入门指南

<div class="article-intro">

FreePlay 可在 Android TV、Amazon Fire TV 和 Apple TV 上使用。本指南将引导您完成应用安装、首次启动，以及了解在应用中使用的基本电视遥控器操作。

</div>

## 安装 FreePlay

### Android TV / Amazon Fire TV

1. 打开电视上的应用商店（**Google Play 商店**或 **Amazon 应用商店**）
2. 搜索 **FreePlay**
3. 选择应用并点击**安装**

### Apple TV

1. 打开 Apple TV 上的 **App Store**
2. 搜索 **FreePlay**
3. 选择应用并点击**获取**

## 启动应用

首次打开 FreePlay 时，您将看到带有 FreePlay 标志的动画启动画面。应用会检查之前连接的内容提供商。如果您已经连接了提供商，它将直接带您进入该提供商的内容浏览器。否则，它将打开**内容提供商**屏幕，您可以在此连接第一个内容来源。

## 使用电视遥控器导航

FreePlay 专为电视遥控器设计。以下是您在应用中使用的控制操作：

- **方向键（上/下/左/右）** - 在屏幕上的项目之间移动
- **确认 / OK 按钮** - 选择高亮的项目
- **左方向键** - 从任何屏幕打开侧边栏
- **右方向键** - 关闭侧边栏并返回主内容区域
- **返回 / 菜单按钮** - 返回上一屏幕

:::tip
在浏览内容时，侧边栏会折叠为窄条图标。按遥控器上的左方向键可展开侧边栏并查看完整标签。
:::
