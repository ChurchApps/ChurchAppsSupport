# 选择配对模式

<div class="article-intro">

FreePlay 提供两种配对模式用于将电视连接到您的教会，同时还可以直接浏览内容提供商。**选择配对模式**屏幕以大卡片形式展示这些选项，您可以使用电视遥控器进行导航。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 安装并首次启动 FreePlay——参见[入门指南](./index.md)
- 准备好电视遥控器用于导航

</div>

## 配对到教室

当您希望特定电视始终显示为特定房间排定的课程时，选择**配对到教室**。这是儿童事工教室的最佳选择，课程按每周日程安排。

**工作原理：**您按名称搜索教会，选择教室，FreePlay 自动获取当天为该教室排定的课程。每个周日早上，您只需打开电视，正确的课程就已准备就绪。

**最适合：**应始终显示该房间排定内容的专用教室电视。

:::tip
在配对电视之前，请在 [freeplay.church](https://freeplay.church) 上设置您的教室。这样可以确保您搜索教会时教室能够正常显示。
:::

## 配对到计划

当您希望跟随教会礼拜计划时，选择**配对到计划**。此模式会生成一个配对码，您需要在 **B1 Admin** 中输入该代码以将电视链接到特定的计划类型。电视随后下载并播放分配给当前计划的内容。

**工作原理：**FreePlay 在屏幕上显示一个配对码。管理员在 **B1 Admin** 的**计划类型**设置中输入该代码。配对后，电视自动加载当前计划的内容。

**最适合：**显示与每周礼拜计划相关内容的电视，例如敬拜歌词或公告幻灯片。

## 内容提供商

如果您不需要配对到教室或计划，可以直接浏览内容提供商。[Lessons.church](../../lessons-church/) 和 Bible Project 等提供商提供可按需下载和播放的内容。

**最适合：**志愿者每周手动选择播放内容的电视，或当您想浏览可用内容时。

:::info
您可以随时从导航底部的侧边栏访问**提供商**屏幕。您无需使用配对模式即可浏览和播放内容。
:::
