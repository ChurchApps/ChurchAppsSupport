# 计划模式

<div class="article-intro">

计划模式将您的电视连接到教会礼拜计划。它不是根据教室日程获取内容，而是遵循教会领导在 B1 Admin 中配置的计划。当您希望电视精确显示当前周计划中分配的内容时，这是理想的选择。

</div>

## 与教室模式的区别

- **[教室模式](../classroom-mode/)** 根据特定房间在指定日期的日程获取课程
- **计划模式** 获取与计划类型关联的内容，可以包括课程、提供商演示文稿、附加组件和其他按自定义顺序排列的媒体

计划模式在您的礼拜计划包含来自不同来源的混合内容，并且这些内容被组织成单一序列时特别有用。

## 工作原理

1. FreePlay 在屏幕上生成配对码
2. 管理员在 **B1 Admin** 的计划类型设置中输入该代码
3. 配对后，FreePlay 加载当前计划并下载所有相关媒体
4. 按**开始计划**开始播放

电视保持与计划类型的配对。每周它会自动获取最新的计划及其内容。
