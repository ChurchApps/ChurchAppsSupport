# 配对您的设备

<div class="article-intro">

要使用计划模式，您需要在 B1 Admin 中将电视与计划类型配对。FreePlay 会生成一个唯一的配对码，将设备链接到您教会的计划，每周自动提供内容。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 安装并启动 FreePlay——参见[入门指南](../getting-started/)
- 从[配对模式屏幕](../getting-started/pairing-modes.md)选择**配对到计划**
- 在电脑或手机上准备好 **B1 Admin** 的访问权限以完成配对

</div>

## 生成配对码

1. 从**选择配对模式**屏幕，选择**配对到计划**
2. FreePlay 联系服务器并生成一个短配对码
3. 配对码以大字符显示在屏幕中央
4. 配对码下方，脉冲指示器显示**等待连接**

配对码以单独字符形式显示，方便从房间对面阅读。

## 在 B1 Admin 中输入代码

1. 在电脑或手机上，打开您教会的 **B1 Admin**
2. 导航到您要分配的计划的**计划类型**设置
3. 输入电视上显示的配对码
4. 保存设置

FreePlay 每隔几秒轮询服务器，检查配对是否完成。一旦 **B1 Admin** 确认连接，电视会自动跳转到计划下载屏幕。

## 下载计划内容

配对后，FreePlay 加载该计划类型的当前计划。它显示：

- 计划名称和礼拜日期
- 关联的课程名称（如果计划包含课程内容）
- 进度指示器显示**正在下载第 X 项，共 Y 项**

当所有媒体文件下载完成后，**开始计划**按钮出现。按遥控器上的**确认**键开始播放。

:::tip
计划每小时自动刷新一次。如果当天计划有更新，FreePlay 会自动获取更改，无需手动干预。
:::

## 替代方式：按教会名称搜索

如果您不想使用配对码流程，可以选择配对屏幕底部的**或按教会名称搜索**。这将带您进入教会搜索屏幕，您可以在此找到教会并连接到教室。

## 如果配对失败

如果无法生成配对码（例如网络问题），您将看到一条错误消息和**重试**按钮。确保电视已连接互联网，然后重试。

:::warning
配对码在一定时间后会过期。如果等待时间过长，请返回配对屏幕生成新的代码。
:::
