# 浏览和下载内容

<div class="article-intro">

连接到内容提供商后，您可以浏览其资源库、下载文件并在电视上播放。内容浏览器提供了一个可视化的文件夹和文件网格，您可以使用电视遥控器进行导航。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 连接至少一个内容提供商——参见[连接提供商](./connecting-providers)
- 准备好电视遥控器用于导航

</div>

## 内容浏览器

内容浏览器以三列卡片网格的形式显示项目。每张卡片显示缩略图和标题。有两种类型的项目：

- **文件夹** - 以缩略图或文件夹图标表示。选择文件夹可查看其内容。
- **文件** - 单独的媒体项目（视频或图片）。视频文件在缩略图上显示播放图标。

屏幕顶部的标题栏显示当前文件夹的名称，如果您在根目录则显示提供商名称。

## 浏览文件夹

1. 使用遥控器上的方向键高亮选中一个文件夹
2. 按**确认**键打开它
3. 浏览器加载文件夹内容，可能是更多文件夹或可播放的文件
4. 按遥控器上的**返回**键返回上一级

您可以深入到提供商内容结构允许的任何层级。浏览器会记录您的文件夹路径，以便您可以逐级返回。

## 下载和播放内容

当您选择一个包含可播放文件的文件夹（或选择单个文件）时，FreePlay 会跳转到下载屏幕：

1. 内容标题和所选文件夹的封面图片显示在屏幕上
2. 进度条显示**正在下载第 X 项，共 Y 项**，文件被保存到您的设备
3. 下载完成后，带有播放图标的**开始**按钮出现
4. 按**确认**键开始播放

:::tip
下载的内容会缓存在您的电视上。如果您稍后再次播放相同的内容，它会立即加载，无需重新下载。
:::

## 播放提供商内容

播放方式与课程播放相同。使用以下控制：

- **右方向键** - 下一项
- **左方向键** - 上一项
- **确认 / 播放-暂停** - 暂停或恢复视频
- **上方向键** - 打开消息选择器，跳转到特定项目
- **返回** - 退出到内容浏览器

退出播放器后，您将返回到当前提供商的内容浏览器根目录。

:::info
有关所有播放器控制的完整参考，请参见[播放课程](../classroom-mode/playing-lessons)。
:::

## 切换提供商

已连接的提供商显示在左侧侧边栏中。要切换到其他提供商：

1. 按遥控器上的**左**方向键展开侧边栏
2. 选择您想浏览的提供商
3. 内容浏览器刷新显示该提供商的资源库

:::warning
如果某个文件夹显示"无可用内容"，说明该提供商在该层级没有任何项目。请尝试浏览其他文件夹或稍后再查看。
:::
