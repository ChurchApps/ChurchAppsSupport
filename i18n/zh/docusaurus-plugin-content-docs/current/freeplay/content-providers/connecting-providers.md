# 连接提供商

<div class="article-intro">

在浏览提供商的内容之前，您需要先连接到它。某些提供商需要通过二维码或邮箱登录进行身份验证，而其他提供商只需点击一下即可连接。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 安装并启动 FreePlay——参见[入门指南](../getting-started/)
- 准备好电视遥控器用于导航
- 对于需要登录的提供商，请准备好您的账号凭据

</div>

## 浏览可用提供商

1. 从侧边栏打开**内容提供商**屏幕（选择底部的**提供商**图标）
2. 您将看到提供商卡片网格，每张卡片显示提供商的标志和名称
3. 已连接的提供商在名称下方显示绿色**已连接**标记
4. 尚未可用的提供商显示**即将推出**标签

## 无需身份验证即可连接

部分提供商不需要登录。选择这些提供商时，FreePlay 会立即连接并打开内容浏览器。无需任何凭据。

## 设备流程身份验证（二维码）

某些提供商使用设备流程，类似于在电视上登录流媒体应用的方式：

1. 在**内容提供商**屏幕上选择提供商卡片
2. FreePlay 显示一个二维码和验证网址
3. 用手机扫描二维码，或在任何设备上访问显示的网址
4. 输入电视屏幕上显示的用户代码
5. 在手机或电脑上完成登录流程
6. FreePlay 检测到登录成功并显示**已连接！**
7. 内容浏览器自动打开

:::info
脉冲显示的**等待授权**指示器表示 FreePlay 正在检查您的登录状态。代码会在几分钟后过期，请及时完成登录流程。
:::

## 表单登录

其他提供商使用传统的邮箱和密码登录：

1. 选择提供商卡片
2. 使用屏幕键盘输入您的**邮箱**和**密码**
3. 选择**登录**按钮
4. 如果凭据正确，FreePlay 显示**已连接！**并打开内容浏览器

:::tip
使用遥控器上的方向键在邮箱字段、密码字段和登录按钮之间移动。按**确认**键选中文本字段以打开屏幕键盘。
:::

## 断开提供商连接

要断开已连接的提供商：

1. 进入**内容提供商**屏幕
2. 选择显示**已连接**标记的提供商卡片
3. 确认提示询问您是否要断开连接
4. 选择**断开连接**以移除连接

断开连接后，该提供商的内容将不再显示在您的侧边栏中。

:::warning
断开连接会从您的设备上移除已保存的身份验证信息。如果您想重新连接，需要再次登录。
:::
