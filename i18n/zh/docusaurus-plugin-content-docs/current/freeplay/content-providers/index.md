# 内容提供商

<div class="article-intro">

内容提供商是提供课程材料、视频和图片以在电视上播放的外部服务。FreePlay 支持多个提供商，让您可以在一个应用中浏览和下载来自不同来源的内容。

</div>

## 可用提供商

FreePlay 目前包含以下提供商：

- **[Lessons.church](../../lessons-church/)** - 免费、开源的儿童事工课程
- **Bible Project** - 探索圣经的视频和资源
- **SignPresenter** - 教会演示和标牌内容
- **B1.church** - 来自您自己的 B1 教会设置的内容

:::info
部分提供商仍在添加中。如果提供商卡片显示"即将推出"，表示该提供商尚不可用于连接。
:::

## 提供商工作原理

每个提供商都有自己的内容库，按文件夹和文件组织。连接到提供商后，其内容会出现在文件夹浏览器中，您可以深入分类、选择项目并下载播放。

已连接的提供商显示在屏幕左侧的侧边栏中。您可以通过在侧边栏中选择来切换提供商，方便在设置过程中快速切换内容来源。
