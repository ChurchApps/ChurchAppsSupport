# 自定义

<div class="article-intro">

Lessons.church 提供了一个即用型课程内容库，但每个教会都不同。自定义功能让您可以调整课程内容以适应您特定的教室、教学风格和年龄组，让志愿者只看到与他们相关的内容。

</div>

## 为什么要自定义？

Lessons.church 上的课程内容设计为灵活可用。一节课程可能包含多个年龄组的章节、可选活动或不适用于每个教室的角色。与其要求志愿者在脑海中跳过不适用的部分，不如自定义课程，让他们只看到相关内容。

例如，一节课程可能包含一个对学龄前儿童很合适但不适合五年级教室的手工活动。通过自定义，您可以为较大年龄组隐藏该章节，同时为较小年龄组保留。

## 您可以自定义什么

- **移除章节** -- 隐藏不适用于特定教室或年龄组的完整课程章节。
- **隐藏角色或操作** -- 移除不需要的章节内角色或操作。
- **重新排列内容** -- 重新排列章节和操作以匹配您教室偏好的流程。

## 针对教室或全局

保存自定义时，您可以选择适用范围：

- **单个教室** -- 自定义仅影响当前教室。其他教室看到原始内容。
- **所有教室** -- 自定义适用于您教会的所有教室。

这种灵活性意味着您可以为不同年龄组设置同一课程的不同视图，或一次性进行适用于所有教室的更改。

:::tip
如果您是自定义新手，先从单个教室开始了解工作方式，然后再将更改应用到所有教室。
:::
