# 自定义课程

<div class="article-intro">

本指南将引导您为教室自定义课程。自定义功能让您可以移除、隐藏或重新排列内容，确保志愿者只看到他们需要的内容——不多不少。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在门户中[设置至少一个教室](../classrooms/setting-up-classrooms.md)
- 为要自定义的教室[安排课程](../classrooms/scheduling-lessons.md)
- 阅读[自定义概览](./index.md)了解可以自定义的内容

</div>

## 打开自定义视图

1. 登录并导航到您的**门户**。
2. 选择包含要自定义课程的**教室**。
3. 点击**已安排的课程**打开它。
4. 点击**自定义**进入自定义视图。

您将看到完整的课程内容，分为章节、角色和操作。

## 移除章节

如果某个章节不适用于您的教室，可以将其完全移除。当课程包含多个年龄组的内容而您只需要其中一个时，这很有用。例如，如果课程同时包含"学龄前活动"和"小学活动"章节，您可以移除不适合您教室的那个。

1. 找到要移除的章节。
2. 点击**移除**或**隐藏**该章节。
3. 该章节将不再对查看此课程的志愿者显示。

## 隐藏角色或操作

在章节中，您可以隐藏特定的角色或操作。例如，如果某个章节包含"敬拜带领"角色但您的教室没有此角色，您可以只隐藏该角色而不移除整个章节。

1. 展开章节查看其角色和操作。
2. 点击**隐藏**您不需要的角色或操作。

## 重新排列内容

您可以重新排列章节和操作以匹配您偏好的流程：

1. 拖动章节或操作上下移动以调整顺序。
2. 新的顺序就是志愿者查看课程时看到的顺序。

## 保存自定义

完成更改后，保存自定义。系统会要求您选择适用范围：

- **仅此教室** -- 更改仅适用于当前教室。其他教室不受影响。
- **所有教室** -- 更改适用于您教会的所有教室。
- **移除自定义** -- 如果您之前自定义了此课程并想恢复原始内容，选择此选项清除更改。

:::tip
先为一个教室自定义单节课程。熟悉流程后，您可以更广泛地将自定义应用到所有教室。
:::

## 何时自定义

自定义在以下情况最为有用：

- 课程包含多个年龄组的内容，您只想显示相关部分。
- 您的教室不使用某些角色（如专职敬拜带领或技术志愿者）。
- 您偏好与默认提供的不同的课程流程顺序。

:::info
您不需要自定义每节课程。如果内容对您的教室已经适用，可以完全跳过此步骤，直接进入[演示](../presenting/index.md)。
:::
