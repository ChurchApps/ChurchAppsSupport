# 志愿者指南

<div class="article-intro">

Lessons.church 上的每节课程都包含针对特定志愿者角色的指南。您不必把整个课程计划交给每位志愿者，而是可以让他们查看只显示与其角色相关指导的指南，保持简洁和专注。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 阅读[查看课程](../browsing/viewing-lessons.md)了解课程结构
- 导航到您要准备指南的课程

</div>

## 指南的组织方式

课程分为**章节**（例如，"欢迎"、"圣经故事"、"小组活动"）。在每个章节中，**角色**被分配特定的**操作**。有四种操作类型：

- **说** -- 志愿者需要朗读或复述的脚本文本。
- **做** -- 需要执行的实际活动或动作。
- **注意** -- 为志愿者准备的背景信息或提醒（不需口头表达）。
- **播放** -- 为班级播放的视频或音频片段。

当志愿者打开他们的指南时，他们按顺序看到每个章节，且只包含分配给其角色的操作。这保持了简洁和专注。

## 查看志愿者指南

1. 打开您要准备的课程。
2. 找到与课程场景关联的**角色**列表。
3. 点击角色名称打开该角色的指南。

指南以逐步引导的方式显示每个章节。志愿者可以在课堂上用手机、平板或电脑滚动查看。

## 打印指南

如果志愿者更喜欢纸质版本，您可以直接从浏览器打印任何指南。

1. 打开所需角色的指南。
2. 使用浏览器的**打印**功能（**Ctrl+P** 或 **Cmd+P**）。
3. 打印页面或另存为 PDF 以便稍后分发。

:::tip
提前与志愿者分享指南链接，让他们可以在课前查看自己的指导。一周内的快速阅读可以帮助每个人在周日早上更有准备。
:::

## 多个角色

一些课程有多个角色——例如，主讲教师、小组组长和技术志愿者。每个角色都有自己的指南，因此您可以拥有课程所需的任意数量的专门指南。

:::info
如果某个角色不适用于您的教室，您可以使用[自定义](../customization/customizing-lessons.md)功能将其隐藏，这样您的志愿者只会看到相关的角色。
:::
