# 演示

<div class="article-intro">

Lessons.church 提供工具帮助您的志愿者自信地教授每节课程。无论您是在屏幕上投影内容、分发打印指南，还是发放活动表，您需要的一切都内置在平台中。

</div>

## 演示模式

当需要教学时，您可以直接从任何课程启动**演示模式**。这为您提供了专为教室屏幕和投影仪设计的全屏投影友好视图。它还与 FreeShow 演示软件和 Venue TV Android 应用集成，用于自动化显示。

## 志愿者指南

每节课程都包含基于角色的指南，让志愿者确切知道要说什么和做什么。迎宾者看到他们的说明，小组组长看到他们的，依此类推。指南可以在课堂上用手机或平板查看，也可以提前打印。

## 可下载材料

课程附带可下载的资源，如活动表、涂色页、带领者指南和补充 PDF。这些材料在课程系列、学习单元和课程级别都可用，因此您可以一站式获取所需的一切。

## 整合使用

一个典型的周日早上工作流程可能如下：

1. 在一周早些时候下载并打印所有活动表或涂色页。
2. 与团队分享志愿者指南链接，让他们可以查看自己的角色。
3. 在教室电脑或电视上以**演示模式**打开课程。
4. 在志愿者跟随各自指南的同时，逐步引导课程章节。

:::tip
在课前几天准备好材料并分享指南链接，可以给志愿者时间查看，让他们在周日早上更有信心。
:::
