# 下载与材料

<div class="article-intro">

Lessons.church 上的课程附带可下载的资源，您可以打印或以数字方式分享。这些材料支持课程内容，为志愿者和学生在课堂期间和课后提供可使用的实物资料。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [浏览到课程](../browsing/viewing-lessons.md)或从您的[教室日程](../classrooms/scheduling-lessons.md)打开已安排的课程

</div>

## 材料类型

根据课程和提供商，您可能会找到以下资源：

- **活动表** -- 学生在课堂上完成的动手练习表。
- **涂色页** -- 适合较小年龄组的可打印页面。
- **带领者指南** -- 为教师准备的详细说明和背景信息。
- **补充 PDF** -- 额外的阅读材料、带回家的资料或家长讲义。

并非每节课程都包含所有这些。可用材料取决于内容提供商提供的内容。

## 在哪里找到下载

资源分布在三个级别：

- **课程系列级别** -- 适用于整个课程体系的通用资源。
- **学习单元级别** -- 与课程系列中特定学习单元相关的材料。
- **课程级别** -- 单节课程的资源。

下载材料：

1. 导航到课程、学习单元或课程系列页面。
2. 在页面上找到**下载**或**资源**部分。
3. 点击文件名将其下载到您的设备。

文件通常为 PDF 格式，可直接打印。

## 课前准备

一个好的习惯是在课前几天检查可下载材料：

1. 打开即将到来的课程并查看可用的下载。
2. 根据预期出席人数打印足够的活动表和涂色页。
3. 查看带领者指南，熟悉课程流程。
4. 准备好课后分发的带回材料。

:::info
一些提供商将所有材料放在学习单元级别，而不是附加到单个课程。如果您在特定课程页面上没有看到下载，请检查上级学习单元或课程系列页面。
:::

:::tip
为志愿者创建每周检查清单，包括下载和打印材料。这样可以确保在周日早上之前不会遗漏任何东西。
:::
