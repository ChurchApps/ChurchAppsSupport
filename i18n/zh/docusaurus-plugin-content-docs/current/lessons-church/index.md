# Lessons.church

<div class="article-intro">

Lessons.church 是一个完全免费的教会课程平台。没有订阅费、没有费用、没有使用限制。它提供来自 12 多个提供商的 189 多个学习单元中的 1,100 多节基于视频的课程，涵盖从学龄前到成人的各个年龄组。

</div>

## 您可以做什么

使用 Lessons.church，您的教会可以：

- **浏览内容** -- 探索来自多个提供商的不断增长的课程系列和学习单元库，按年龄组和主题分类。
- **设置教室** -- 创建与您事工结构匹配的教室（例如，学龄前、小学、青少年或成人）。
- **安排课程** -- 在需要的日期为教室分配特定课程。
- **根据需求自定义** -- 调整课程内容以适应您教会的教学风格和偏好。
- **向班级演示** -- 使用内置的演示模式在课堂上逐步引导课程。

## 内容组织方式

Lessons.church 按照清晰的层级组织课程：

1. **课程系列** -- 来自提供商的顶级课程体系（例如，儿童事工课程系列）。
2. **学习单元** -- 课程系列中的一组相关课程（例如，关于特定主题的四周系列）。
3. **课程** -- 学习单元中的单独教学会话，通常用于一次课堂教学。
4. **场景** -- 针对特定年龄组或教学环境定制的课程版本（例如，学龄前版本和小学版本）。

## 集成

Lessons.church 与其他 ChurchApps 工具协同工作，帮助您传递内容：

- **B1.church** -- 志愿者和家长可以通过 B1 应用访问即将到来和最近的课程。
- **FreeShow** -- 使用 FreeShow 演示软件在屏幕上显示课程内容。
- **Venue TV 应用** -- 使用 Android TV 应用直接在电视上显示课程。
