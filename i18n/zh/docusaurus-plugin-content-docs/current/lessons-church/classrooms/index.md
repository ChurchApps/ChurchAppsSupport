# 教室

<div class="article-intro">

教室是您在 Lessons.church 中组织教会教学的基础。教室代表一个教学场所——可以是实体或虚拟的——例如，"学龄前教室"、"小学 K-2"或"成人查经班"。本节介绍创建教室和为教室安排课程。

</div>

## 为什么教室很重要

当您设置教室时，就为您的儿童事工、青少年团契或任何教学环境创建了清晰的结构。每个教室都有自己的日程，这样您的志愿者始终知道要教什么、什么时候教。不必再传递纸质日程表或发送临时邮件，每个人都可以查看门户了解即将到来的安排。

## 您可以用教室做什么

- **创建无限教室** -- 根据教会需求设置任意数量的教室。没有限制，因此您可以为每个年龄组、校区或课程系列设置单独的教室。
- **安排课程** -- 在特定日期为每个教室分配特定课程。您的日程一目了然地显示过去和即将到来的课程。
- **自定义内容** -- 为每个教室定制课程内容，让志愿者只看到与其年龄组相关的内容。详见[自定义](../customization/index.md)。

## 教室如何协同工作

教室将 Lessons.church 上的课程内容连接到您教会的实际教学日程。以下是典型的工作流程：

1. 在 Lessons.church 上[浏览](../browsing/index.md)可用的课程系列和学习单元。
2. 设置教室以匹配您教会的房间和年龄组。
3. 为即将到来的日期为每个教室安排课程。
4. 志愿者打开他们的教室查看日程安排并做好准备。
