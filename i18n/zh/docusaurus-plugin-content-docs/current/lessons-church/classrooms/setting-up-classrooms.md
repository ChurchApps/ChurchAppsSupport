# 设置教室

<div class="article-intro">

在安排课程之前，您需要创建代表教学环境的教室。本指南将引导您在 Lessons.church 门户中创建、命名和管理教室。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [创建账号](../getting-started/creating-account.md)并连接到您的教会
- 熟悉[门户概览](../getting-started/portal-overview.md)

</div>

## 创建教室

1. 在 [Lessons.church](https://lessons.church) 登录您的账号。
2. 导航到您的**门户**。这是您教会管理教室和日程的中心。
3. 点击**添加教室**。
4. 输入教室名称。选择一个志愿者能够识别的描述性名称——例如，"学龄前"、"小学 K-2"或"初中"。
5. 保存新教室。它将立即出现在您的门户仪表板上。

为教会的每个房间或年龄组重复这些步骤。您可以创建任意数量的教室。

## 首次设置

如果您尚未设置任何教室，门户将显示一个设置视频来指导您完成此过程。该视频涵盖创建教室和安排第一节课程的基础知识。如果您是 Lessons.church 的新用户，这是一个很好的起点。

:::tip
按照志愿者对房间的常用叫法来命名教室。如果大家都叫它"学前班教室"，就命名为"学前班"而不是"幼儿早期教育教室 101"这样正式的名称。
:::

## 管理教室

您的教室显示在门户仪表板上，您可以随时管理它们。在这里您可以：

- **重命名教室** -- 如果房间或年龄组发生变化，更新名称。
- **删除教室** -- 删除不再需要的教室。
- **查看日程** -- 点击任何教室查看其课程日程。

## 下一步

教室设置完成后，您就可以开始[安排课程](./scheduling-lessons.md)，让志愿者每周知道要教什么。
