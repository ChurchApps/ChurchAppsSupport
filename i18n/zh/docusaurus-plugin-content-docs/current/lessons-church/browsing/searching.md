# 搜索

<div class="article-intro">

当您知道想教什么主题，但不确定哪个课程系列或学习单元包含它时，搜索功能可以帮助您快速找到合适的内容。搜索涵盖整个 Lessons.church 库，包括来自外部提供商的内容。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 阅读[浏览内容](./index.md)概览，了解所有可用的导航选项

</div>

## 如何搜索

1. 从 Lessons.church 网站顶部导航到**搜索页面**。
2. 在**搜索框**中输入关键词或短语（例如，"饶恕"、"大卫与歌利亚"或"祷告"）。
3. 按**回车键**或点击搜索按钮查看结果。

## 搜索原理

搜索基于对整个 Lessons.church 库的全文搜索。它会检索课程标题、描述和内容以匹配您的关键词。结果包括来自所有可用课程系列和提供商的课程，因此您可以全面了解某个主题的可用内容。

## 浏览结果

搜索结果显示匹配的课程以及每节课程所属的课程系列和学习单元信息。点击任何结果可直接进入该课程页面，在那里您可以查看完整的内容、章节、角色和可下载材料。

## 何时使用搜索

搜索在以下情况最为有用：

- 您有特定的圣经故事或主题，想查看所有涵盖它的可用课程。
- 您正在规划一个特别系列，需要找到关于特定主题的课程。
- 您还不熟悉可用的课程系列，想快速了解有什么内容。

:::tip
如果关键词返回的结果太多，请尝试更具体的短语。例如，搜索"好撒玛利亚人"而不是仅搜索"撒玛利亚人"来缩小范围。
:::

如果您已经知道想要哪个课程系列，按课程系列浏览通常更快。详见[课程系列与学习单元](./programs-and-studies.md)了解如何浏览内容层级。
