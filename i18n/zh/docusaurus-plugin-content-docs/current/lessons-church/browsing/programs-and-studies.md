# 课程系列与学习单元

<div class="article-intro">

Lessons.church 上的所有课程按照清晰的层级结构组织。了解这个结构将帮助您快速找到适合教会的内容，并在相关课程之间轻松导航。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 阅读[浏览内容](./index.md)概览，了解查找内容的不同方式

</div>

## 内容层级

内容从最广泛到最具体的层次组织：

- **提供商** -- 发布课程的组织（Lessons.church 或第三方提供商）。
- **课程系列** -- 一个课程体系，如 "Ark Kids" 或 "NextLevel"。每个课程系列是围绕一个主题或年龄组设计的相关学习单元集合。
- **学习单元** -- 课程系列中的一个单元，如"好撒玛利亚人"。每个学习单元包含一组旨在一起教授的课程，通常持续数周。
- **课程** -- 学习单元中的单独教学会话。每节课程包含章节、角色、操作和可下载材料。
- **场景** -- 课程的年龄组版本，如学龄前或小学。场景让您可以将同一课程针对不同受众进行调整。

## 浏览课程系列

要浏览课程系列：

1. 访问 Lessons.church 首页。
2. 滚动浏览可用课程系列列表，或查看顶部的精选课程系列。
3. 点击**课程系列名称**或其图片打开课程系列页面。

每个课程系列页面包含：

- **描述** -- 课程内容和目标受众的概述。
- **课程系列图片** -- 该系列的视觉标识。
- **视频预告** -- 一些课程系列包含简短的视频预览，让您在决定使用之前了解内容的风格和质量。
- **学习单元** -- 该课程系列中所有学习单元的列表，可能按分类组织。

## 浏览学习单元

从课程系列页面：

1. 浏览学习单元列表。如果课程系列将学习单元组织成分类，您会看到它们相应分组。
2. 点击**学习单元名称**打开学习单元页面。
3. 学习单元页面显示该单元中的所有课程，按教学顺序列出。

在这里，您可以点击任何课程查看其完整内容，包括章节、角色和可下载材料。详见[查看课程](./viewing-lessons.md)获取详细教程。

:::info
课程系列和学习单元由课程提供商管理。如果您是内容创作者，可以通过[管理后台](../admin/index.md)区域管理自己的课程系列和学习单元。
:::
