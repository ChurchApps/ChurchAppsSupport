# 查看课程

<div class="article-intro">

Lessons.church 上的每节课程都是一个完整的教学会话，包含结构化内容、可下载材料，以及对多个年龄组的支持。本指南介绍如何浏览和使用课程页面来准备课堂教学。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 阅读[课程系列与学习单元](./programs-and-studies.md)了解内容的组织方式
- 浏览到课程系列和学习单元，找到您想查看的课程

</div>

## 进入课程

要查看课程，请按照内容层级导航：

1. 在首页点击一个**课程系列**。
2. 从课程系列页面选择一个**学习单元**。
3. 从学习单元页面点击您想查看的**课程**。

## 您将看到的内容

课程页面分为几个区域：

### 视频

如果课程包含视频，它会显示在页面顶部。点击即可在浏览器中直接播放。视频通常用作主要教学环节，尤其在儿童事工中。

### 章节和角色

每节课程分为**章节**（例如，"开场"、"圣经故事"、"活动"、"结束"）。在每个章节中，内容按**角色**组织：

- **带领者** -- 主要教师或演讲者
- **协助者** -- 辅助志愿者
- 课程创建者定义的其他自定义角色

每个角色列出其**操作**，告诉志愿者具体要做什么：

- **说** -- 需要朗读的脚本或要点
- **做** -- 需要执行的实际指导或活动
- **注意** -- 为志愿者准备的背景信息或提示
- **播放** -- 播放视频或音频片段的提示

### 可下载材料

课程可能包含您可以下载和打印的文件，如：

- PDF 讲义
- 活动表
- 涂色页
- 手工制作说明
- 给家长的带回材料

在课程页面中寻找下载链接以获取这些资源。详见[下载与材料](../presenting/downloads.md)了解更多关于准备打印资源的信息。

## 切换场景

许多课程支持多个**场景**，代表不同的年龄组，如学龄前、小学或青少年。场景让同一课程主题可以针对每个群体的适当水平进行调整。

要切换场景：

1. 在课程页面上找到**场景选择器**。
2. 点击不同的场景名称（例如，从**小学**切换到**学龄前**）。
3. 页面更新显示为该年龄组设计的内容、脚本和材料。

:::tip
如果您教授多个年龄组，请在课前查看每个场景，以了解内容如何针对不同水平进行调整。
:::
