# 外部提供商

<div class="article-intro">

Lessons.church 不仅限于自己的课程。第三方课程提供商可以通过**开放课程格式**发布他们的内容，这些内容将与 Lessons.church 的内置课程系列一起显示，提供统一的浏览体验。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 阅读[课程系列与学习单元](./programs-and-studies.md)了解内容的组织方式
- 要添加提供商连接，您需要一个可以访问教会门户的 [Lessons.church 账号](../getting-started/creating-account.md)

</div>

## 什么是外部提供商？

外部提供商是使用开放课程格式发布教会课程的任何组织——这是一种允许课程内容在平台间共享的标准。当提供商以这种格式提供内容时，Lessons.church 可以将其拉入并像自己的课程系列一样显示。

这意味着您可以获得更丰富的课程选择，而无需访问多个网站。

## 浏览外部内容

外部提供商的内容与内置内容的使用方式完全相同：

1. 外部提供商的课程系列与其他课程系列一起显示在 Lessons.church 首页上。
2. 点击一个**课程系列**查看其学习单元。
3. 点击一个**学习单元**查看其课程。
4. 使用相同的布局查看课程——章节、角色、操作、视频和可下载材料。

外部内容没有特殊流程或单独区域。所有内容都集成在一个库中，方便浏览。

## 添加提供商连接

如果您的教会使用支持开放课程格式的课程提供商，并且您希望在 Lessons.church 门户中获取他们的内容：

1. 登录您的 Lessons.church 账号。
2. 在门户中导航到**外部提供商**页面。
3. 添加外部提供商的连接 URL。
4. 他们的课程系列和学习单元将可供您的教会浏览、安排和在教室中使用。

:::info
添加外部提供商连接是通过 Lessons.church 门户完成的，需要已登录的账号。浏览平台已添加的内容不需要账号。
:::

## 面向内容创作者

如果您是有兴趣在 Lessons.church 上提供内容的课程出版商，开放课程格式提供了一种标准化方式来共享您的课程系列、学习单元和课程。请联系 Lessons.church 团队或访问开发者文档了解集成详情。

:::tip
如果您是管理员，需要在平台级别管理第三方提供商源，请参见管理后台部分的[第三方提供商](../admin/third-party-providers.md)。
:::
