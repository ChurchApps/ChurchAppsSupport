# 浏览内容

<div class="article-intro">

Lessons.church 提供一个免费的教会课程库，任何人都可以浏览。您无需账号或登录即可浏览可用内容——只需访问网站即可开始查看。本节介绍查找和浏览课程的所有方式。

</div>

## 首页

当您首次访问 Lessons.church 时，首页为您提供所有可用内容的快速概览：

- **精选课程系列** -- 突出显示的课程系列出现在顶部，方便发现热门或最近更新的内容。
- **所有课程系列** -- 在精选部分下方，您会看到每个可用课程系列的完整列表，以及课程和学习单元数量等统计信息。
- **提供商内容** -- 来自 Lessons.church 和外部课程提供商的内容一起显示，为您提供一个探索的统一位置。

## 查找内容的方式

有多种方式可以浏览课程库：

1. **按课程系列浏览** -- 点击首页上的任何课程系列查看其学习单元和课程。当您想浏览特定课程体系时，这是最佳方式。详见[课程系列与学习单元](./programs-and-studies.md)。

2. **搜索** -- 使用搜索页面按关键词在整个库中查找课程。当您知道某个主题但不确定哪个课程系列涵盖它时，这很有帮助。详见[搜索](./searching.md)。

3. **浏览外部提供商** -- 一些内容来自以开放课程格式发布的第三方课程提供商。您可以像浏览内置内容一样浏览他们的课程系列和学习单元。详见[外部提供商](./external-providers.md)。

## 无需登录

浏览完全开放。您可以查看课程系列、学习单元、课程、可下载材料和视频，无需创建账号。如果您决定要为教室安排课程或自定义内容，可以在此时[创建免费账号](../getting-started/creating-account.md)。

:::tip
先在首页浏览几个课程系列，了解课程的组织方式，然后再深入研究特定的学习单元。
:::
