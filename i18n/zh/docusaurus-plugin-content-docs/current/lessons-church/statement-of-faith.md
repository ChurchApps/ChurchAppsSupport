# 信仰声明

<div class="article-intro">

虽然我们承认任何人为的声明都无法完全囊括圣经对信仰的全部理解，但我们提交此声明以阐明 ChurchApps 作为组织的核心信仰。这些信仰指导着 Lessons.church 上所有可用课程的内容策划和审查过程。

</div>

**神：**独一、圣洁的神永恒地存在于三个位格中——圣父、圣子和圣灵。神创造了一切可见和不可见的事物。神在智慧、能力和爱中是完美的，知晓过去、现在和未来的一切事物，他救赎的主权计划在创世之前就已确立。(创 1:1-2, 申 6:4, 来 11:3, 弗 1:9-10; 启 13:8)

**耶稣：**耶稣基督是神的独生子，由童女所生，完全神性和完全人性，是我们的救主和主。耶稣没有罪，他代替我们死，作为我们罪的替代性牺牲，承受了神的愤怒，使所有信靠他的人与神和好。耶稣在胜过罪和死亡后身体复活。他升天坐在圣父的右边，在那里以我们的君王、大祭司和中保身份掌权，直到他荣耀地再来。(约 3:16, 西 1:15; 2:9-15; 林前 15:3-8, 20-28; 林后 5:18-21; 来 4:14-15)

**圣灵：**圣灵是完全神性的，活跃在教会和世界中。圣灵通过光照福音和定罪来吸引所有人归向基督。圣灵住在信徒的生命中，使他们得改变、引导、确据和能力，过有果实的基督徒生活。(约 16:8-11; 徒 2:38; 林后 3:17-18; 加 3:2)

**圣经：**神在圣经中启示自己，圣经是独特的受默示的书面神的话语，在其所确认的一切事上都是无误的。圣经是信仰和实践一切事务的最终权威。(提后 3:16; 彼后 1:20-21)

**人类：**神按照自己的形象创造了所有人，男和女，因此所有人都有内在的价值和目的。由于第一个男人和女人（亚当和夏娃）的罪，死亡进入了世界。所有人都犯了罪，亏缺了神的荣耀，与神隔绝，除了靠耶稣基督的宝血别无盼望。(创 1:26-27; 创 3; 罗 3:23; 弗 2:1-3)

**救恩：**救恩唯独在基督里才能找到，通过恩典藉着信心赐给所有人。活的信心通过悔改、认罪、浸水洗礼和顺服的生活来证明。(罗 3:23; 5:12, 徒 2:38, 加 3:26-29; 弗 2:4-10)

**教会：**教会是基督在地上的身体，基督是元首。神的教会由信徒皆祭司组成，按照神赐给他们的恩赐作为福音的执事服事。教会共同受命使万民成为门徒，直到基督再来。(太 28:18-20; 弗 3:10; 4:11-13; 西 1:18; 彼前 2:9-10)

**基督再来：**基督将有形地再来，恢复创造并审判世界。信徒将身体复活获得与神在天堂的永生，不信者将身体复活面对地狱的永恒审判。在天堂，罪将不再存在，在基督里的人将永远与神相交。(徒 1:11; 帖后 1:5-12; 帖前 4:13-18; 启 20:11-15)

## 课程指导方针

Lessons.church 使用来自各种不同基督教来源的课程。我们提供的每节课程都经过我们内部审查，确保圣经的准确性和教义的健全性。我们理解教会内部存在各种不同的观点，我们选择在共同点上团结一致。关于每个课程的作者信息可以在各课程的"关于"部分找到。

我们鼓励用户就他们对特定课程或系列的疑问与我们联系。
