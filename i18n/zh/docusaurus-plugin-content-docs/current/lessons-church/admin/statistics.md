# 统计数据

<div class="article-intro">

统计页面为管理员提供内容使用情况的洞察。您可以查看哪些学习单元最受欢迎、哪些教会正在下载您的资源，以及这些教会的地理位置分布。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 拥有 Lessons.church 的管理员权限（参见[管理后台概览](./index.md)）
- 至少有一个[课程系列](./managing-programs.md)包含已发布的内容

</div>

## 访问统计数据

1. 导航到**管理后台**区域。
2. 选择**统计**或**统计数据**页面。
3. 选择您要查看统计数据的**课程系列**。

## 您可以看到什么

统计页面提供多种下载数据视图：

### 按学习单元的下载量

所选课程系列中每个学习单元的独立下载量明细。这显示哪些系列最受欢迎，帮助您决定在内容创建方面的投入重点。

### 教会列表

从您的课程系列中下载资源的教会列表。这让您了解您的受众以及内容的采用范围。

### 地理分布

一个地图视图，显示下载教会的地理位置。这帮助您了解课程的地理覆盖范围，并识别您的内容受欢迎的地区。

### 日期范围筛选

您可以按日期范围筛选所有统计数据，聚焦于特定时间段。这对于查看随时间变化的趋势或衡量新发布学习单元的影响非常有用。

1. 使用日期选择器设置**开始日期**和**结束日期**。
2. 统计数据将更新为仅反映该范围内的活动。

## 有效利用统计数据

以下是一些利用下载数据的方式：

- **识别热门内容** -- 专注于创建更多与获得最多下载的风格和主题相似的材料。
- **跟踪新发布** -- 发布新学习单元后，按日期筛选查看其被采用的速度。
- **了解您的受众** -- 使用教会列表和地图查看谁在使用您的内容，并考虑与这些社区建立联系。

:::info
统计数据基于独立下载量。如果同一教会多次下载某个资源，只计算一次。
:::
