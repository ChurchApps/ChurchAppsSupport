# 第三方提供商

<div class="article-intro">

Lessons.church 通过开放课程格式支持外部课程提供商。这意味着您可以从其他组织导入内容，并将其与内置库一起提供，让教会可以在一个地方访问多个来源的课程。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 拥有 Lessons.church 的管理员权限（参见[管理后台概览](./index.md)）
- 获取您要连接的外部提供商的源地址

</div>

## 添加提供商

1. 导航到**管理后台**区域。
2. 打开**第三方**页面。
3. 点击**添加提供商**（或相应的添加新连接按钮）。
4. 输入提供商的**源地址**。这是提供商以开放课程格式发布课程数据的 URL。
5. 保存连接。

保存后，Lessons.church 将导入提供商的内容。他们的课程系列、学习单元和课程将与您自己的内容一起出现在课程库中。

## 管理提供商

在**第三方**页面，您可以查看所有已连接的提供商。对于每个提供商，您可以：

- **查看**导入的内容以确认是否正确拉取。
- **更新**源地址（如果提供商更改了端点）。
- **移除**不再需要的提供商。

## 什么是开放课程格式？

开放课程格式是一种标准化架构，以结构化方式描述课程内容——课程系列、学习单元、课程、场景、章节和操作。任何以此格式发布内容的组织都可以作为提供商添加到 Lessons.church。

如果您是有兴趣通过 Lessons.church 提供课程的内容提供商，您可以在 [Open Lesson Schema](https://github.com/LiveChurchSolutions/LessonsScreen) 仓库中找到完整的架构文档。

## 为什么使用第三方提供商？

- **为教会提供更多选择** -- 教会可以浏览更多种类的课程，无需离开平台。
- **一站式解决** -- 志愿者和协调员无需访问多个网站来查找和安排课程。
- **设置简单** -- 添加提供商只需要一个源地址。无需手动数据输入或文件上传。

:::info
第三方内容由外部提供商管理。如果您注意到提供商的内容缺失或过时，可能需要在他们那边解决问题。
:::

:::tip
如需教会级别的外部提供商配置（非管理员级别），请参见浏览部分的[外部提供商](../browsing/external-providers.md)页面。
:::
