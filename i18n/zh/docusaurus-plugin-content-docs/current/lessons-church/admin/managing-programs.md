# 管理课程系列

<div class="article-intro">

课程系列是 Lessons.church 中课程的顶级容器。一个课程系列代表完整的课程体系——例如，"Ark Kids" 或 "Youth Foundations"。本指南介绍在管理后台中创建、编辑和组织课程系列。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 拥有 Lessons.church 的管理员权限（参见[管理后台概览](./index.md)）
- 准备好课程系列名称、描述和缩略图

</div>

## 创建课程系列

1. 导航到**管理后台**区域。
2. 点击**添加课程系列**。
3. 填写以下字段：
   - **名称** -- 课程系列的显示名称（例如，"Ark Kids"）。
   - **别名** -- URL 友好的标识符（例如，"ark-kids"）。这用于网址中。
   - **描述** -- 课程系列内容的简短摘要。
   - **图片** -- 上传代表该课程系列的缩略图。
4. 保存课程系列。

您的新课程系列将出现在管理课程系列列表中，一旦包含学习单元和课程，也将出现在面向公众的网站上。

## 编辑课程系列

1. 在管理列表中点击课程系列名称。
2. 更新任何字段（**名称**、**别名**、**描述**或**图片**）。
3. 保存更改。

## 在课程系列中组织学习单元

课程系列创建后，您可以向其添加学习单元。学习单元在管理视图中嵌套显示在课程系列下。点击课程系列旁的展开箭头可查看其学习单元，或从那里添加新的。详见[管理学习单元](./managing-studies.md)。

## 学习单元分类

如果您的课程系列包含许多学习单元，可以将它们组织成**分类**。分类按主题、年龄范围或您选择的任何其他标准对学习单元进行分组。这使用户更容易浏览大型课程系列。

## 上传课程系列级别资源

您可以将可下载文件附加到课程系列。这些资源适用于整个课程系列，而非特定的学习单元或课程——例如，范围与序列文档或志愿者手册。

1. 在管理视图中打开课程系列。
2. 导航到**资源**部分。
3. 上传您的文件。

## 清除缓存

对课程系列进行更改后，您可能需要清除缓存以使更新显示在公开网站上。使用管理后台中的**清除缓存**选项来刷新内容。

:::tip
选择一个描述性的课程系列名称并撰写清晰的描述。这是教会浏览课程库时首先看到的内容，好的摘要可以帮助他们判断该课程系列是否适合他们的需求。
:::
