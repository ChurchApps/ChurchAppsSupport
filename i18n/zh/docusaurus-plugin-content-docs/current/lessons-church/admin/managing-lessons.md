# 管理课程

<div class="article-intro">

课程是学习单元中的单独教学会话。每节课程代表一次课堂教学的内容。在这里您可以构建详细的内容——场景、章节、角色和操作——供志愿者教学时使用。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [创建课程系列](./managing-programs.md)并[添加学习单元](./managing-studies.md)
- 拥有 Lessons.church 的管理员权限（参见[管理后台概览](./index.md)）

</div>

## 创建课程

1. 导航到**管理后台**区域，展开课程系列和学习单元，找到要添加课程的学习单元。
2. 点击**添加课程**。
3. 输入课程的**名称**和其他详细信息。
4. 保存课程。

## 了解课程结构

每节课程分为几个层次：

- **场景** -- 针对不同年龄组或教学环境的课程版本。例如，同一课程可以有"学龄前"场景和"小学"场景。
- **章节** -- 场景中的分段，如"欢迎"、"视频"、"圣经故事"和"活动"。章节控制课堂时间的流程。
- **角色和操作** -- 在每个章节中，您可以分配角色（例如，"主讲教师"或"小组组长"）并为每个角色指定具体的操作。

## 操作类型

您可以在章节中为角色分配四种类型的操作：

- **说** -- 志愿者应向班级朗读或复述的文本。
- **做** -- 需要执行的实际活动或指导。
- **注意** -- 为志愿者准备的背景信息或提醒（不需朗读）。
- **播放** -- 在该章节期间播放的视频或音频片段。

## 添加场景

1. 在管理视图中打开课程。
2. 点击**添加场景**。
3. 输入场景名称（例如，"学龄前"或"小学"）。
4. 保存。现在您可以为此场景添加章节。

## 添加章节和操作

1. 展开场景查看其章节。
2. 点击**添加章节**并为其命名（例如，"欢迎"）。
3. 在章节中，点击**添加操作**。
4. 选择**角色**、**操作类型**（说、做、注意或播放），并输入内容。
5. 保存。

重复此过程以构建完整的课程流程。

## 上传课程级别资源

您可以直接将可下载材料附加到课程——例如，该周特定的活动表或涂色页。

1. 在管理视图中打开课程。
2. 导航到**资源**部分。
3. 上传您的文件。

:::tip
从志愿者的角度构建课程。按顺序浏览章节和操作，问自己一个首次参与的志愿者是否可以在没有其他准备的情况下跟上。
:::
