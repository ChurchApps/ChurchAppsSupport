# 管理学习单元

<div class="article-intro">

学习单元是课程系列中的一系列相关课程。例如，名为"好撒玛利亚人"的四周系列将是大型儿童事工课程系列中的一个学习单元。本指南介绍在管理后台中创建、编辑和组织学习单元。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在管理后台[创建至少一个课程系列](./managing-programs.md)
- 拥有 Lessons.church 的管理员权限（参见[管理后台概览](./index.md)）

</div>

## 创建学习单元

1. 导航到**管理后台**区域并展开要添加学习单元的课程系列。
2. 点击**添加学习单元**。
3. 填写以下字段：
   - **名称** -- 学习单元的显示名称（例如，"好撒玛利亚人"）。
   - **别名** -- URL 友好的标识符（例如，"the-good-samaritan"）。
   - **描述** -- 此学习单元涵盖的内容摘要以及包含的课程数量。
   - **图片** -- 上传学习单元的缩略图。
4. 保存学习单元。

## 编辑学习单元

1. 在管理列表中点击学习单元名称。
2. 根据需要更新任何字段。
3. 保存更改。

## 分配分类

如果父级课程系列使用学习单元分类，您可以将学习单元分配到其中：

1. 在管理视图中打开学习单元。
2. 从下拉菜单中选择一个**分类**。
3. 保存。

分类帮助用户浏览包含许多学习单元的课程系列。例如，您可能有"旧约"、"新约"和"专题"等分类。

## 管理学习单元中的课程

学习单元包含一节或多节课程。在管理视图中展开学习单元可查看其课程，或点击**添加课程**创建新课程。详见[管理课程](./managing-lessons.md)了解如何构建课程内容。

## 上传学习单元级别资源

您可以附加适用于整个学习单元的可下载文件——例如，系列概览、宣传图片或家长通讯模板。

1. 在管理视图中打开学习单元。
2. 导航到**资源**部分。
3. 上传您的文件。

:::tip
撰写学习单元描述时要考虑教会领导的视角。对主题、目标年龄组和课程数量的清晰摘要可以帮助教会决定是否使用这个系列。
:::
