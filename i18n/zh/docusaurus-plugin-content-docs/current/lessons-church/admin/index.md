# 管理后台

<div class="article-intro">

Lessons.church 的管理后台是内容创建者和管理员管理课程库的地方。在这里您可以构建课程体系结构、上传资源、查看使用统计数据，以及连接第三方内容提供商。

</div>

:::warning
管理后台访问需要相应的权限。如果您没有看到管理选项，请联系您的教会管理员申请访问权限。
:::

## 您可以做什么

管理后台让您可以完全控制您的内容：

- **创建和编辑课程系列** -- 设置顶级课程体系，包含名称、描述、图片和别名。
- **管理学习单元** -- 在每个课程系列中组织相关课程分组。
- **构建课程** -- 添加包含场景、章节、角色和操作的单独课程。
- **上传资源** -- 在课程系列、学习单元或课程级别附加可下载文件（PDF、活动表、图片）。
- **查看统计数据** -- 查看哪些内容被下载、由谁下载以及下载地点。
- **配置提供商** -- 连接使用开放课程格式的外部课程提供商。

## 内容层级

作为管理员，了解内容的组织结构会很有帮助：

1. **课程系列**包含一个或多个学习单元。
2. **学习单元**包含一个或多个课程。
3. **课程**包含一个或多个场景（年龄组变体）。
4. **场景**包含章节，章节包含基于角色的操作。

每个层级都可以附加自己的可下载资源。
