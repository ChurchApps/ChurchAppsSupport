# 门户概览

<div class="article-intro">

登录 Lessons.church 后，您将进入**门户**——您教会管理教室和课程日程的仪表板。这是您在组织教会课程时花费大部分时间的地方。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [创建账号](./creating-account.md)并连接到您的教会
- 登录 [Lessons.church](https://lessons.church)

</div>

## 您看到的内容

门户分为两个主要区域：

- **教室**（右侧）-- 您教会的教室列表。点击教室名称查看其已安排的课程。您也可以在此添加新教室或编辑现有教室。
- **已安排的课程**（左侧）-- 选择教室后，此区域显示已安排的课程。如果您尚未设置任何教室，将看到一个设置说明视频帮助您入门。

## 您可以在门户中做什么

从门户，您可以：

1. **管理教室** -- 添加、编辑或删除教室。每个教室可以关联志愿者团队（这样他们可以在 B1 应用中查看即将到来的课程）和家长/学生群组（这样他们可以访问最近的课程）。
2. **查看和管理已安排的课程** -- 查看每个教室即将到来的课程并根据需要进行更改。
3. **订阅教室源** -- 点击教室旁边的源图标获取播放列表源 URL，可与兼容的应用配合使用。
4. **访问外部提供商** -- 使用**外部提供商**页面配置支持开放课程格式的第三方内容来源。这让您可以引入 Lessons.church 内置库之外的其他提供商的课程。
5. **访问管理后台** -- 如果您有管理员权限，您将在导航中看到**管理后台**选项。管理后台让您可以在内容级别管理课程系列、学习单元和课程。

## 导航

门户导航包括：

- **首页** -- 返回 Lessons.church 公开首页浏览内容。
- **日程** -- 管理教室和课程日程的主门户视图。
- **外部提供商** -- 配置第三方课程提供商。
- **管理后台** -- 管理课程系列、学习单元和课程内容（仅在有管理员权限时显示）。

:::info
**管理后台**选项仅在您的账号被授予内容编辑权限时才会显示。大多数教会工作人员使用**日程**和**外部提供商**页面进行日常管理。
:::

## 入门

如果您是第一次访问门户且尚未创建任何教室，请先在教室面板中点击**添加第一个教室**。为教室命名（例如，"学龄前"或"3-5 年级"）并保存。然后您就可以开始[安排课程](../classrooms/scheduling-lessons.md)了。
