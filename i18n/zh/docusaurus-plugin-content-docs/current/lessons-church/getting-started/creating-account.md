# 创建账号

<div class="article-intro">

Lessons.church 使用与 B1.church 相同的 ChurchApps 账号系统。如果您已有 ChurchApps 账号，可以直接登录。否则，注册只需几分钟，即可获得完整课程库的访问权限。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 准备好一个有效的电子邮件地址用于账号验证
- 如果您的教会已在使用 Lessons.church，请在创建新教会配置文件之前咨询您的管理员

</div>

## 注册新账号

1. 在浏览器中访问 [Lessons.church](https://lessons.church)。
2. 点击顶部导航栏中的**登录**。
3. 在登录页面，点击**注册**创建新账号。
4. 输入您的**名字**、**姓氏**和**电子邮件地址**。
5. 为您的账号创建**密码**。
6. 提交注册表单。

注册后，系统会提示您连接到现有教会或创建新的教会配置文件。

## 连接您的教会

**创建新教会：**

1. 选择**创建新教会**选项。
2. 输入您的**教会名称**和基本信息。
3. 完成设置。您将成为教会在 Lessons.church 上的管理员。

**加入现有教会：**

1. 按名称搜索您的教会。
2. 从结果中选择您的教会。
3. 您教会的管理员可能需要授予您相应的权限。

:::tip
如果您教会中已有人设置了 Lessons.church，请让他们确保您拥有管理教室和日程的正确权限。
:::

## 使用现有账号登录

如果您已有来自 B1.church 或其他 ChurchApps 工具的 ChurchApps 账号：

1. 访问 [Lessons.church](https://lessons.church) 并点击**登录**。
2. 输入您现有的**电子邮件**和**密码**。
3. 您将被带到门户，在那里可以管理教室和日程。

:::info
您的 ChurchApps 账号适用于所有 ChurchApps 产品。一个账号即可访问 B1 Admin、B1.church、Lessons.church 等。
:::

## 下一步

登录并连接到教会后，请前往[门户概览](./portal-overview.md)了解如何浏览仪表板并开始设置教室。
