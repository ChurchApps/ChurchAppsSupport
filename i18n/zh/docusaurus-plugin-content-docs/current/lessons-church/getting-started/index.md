# 入门指南

<div class="article-intro">

Lessons.church 完全免费使用。没有订阅费、没有试用期、没有功能限制。本节将引导您完成账号创建、连接教会和浏览门户，以便您可以立即开始安排课程。

</div>

## 您需要什么

要开始使用 Lessons.church，您需要：

1. **创建账号** -- 在 [Lessons.church](https://lessons.church) 使用您的姓名、电子邮件和密码注册。这使用与 B1.church 相同的 ChurchApps 账号。
2. **连接您的教会** -- 注册后，连接到您现有的教会配置文件或创建新的。这将您的教室和日程与教会关联。
3. **设置教室** -- 创建与您事工相匹配的教室（例如，"学龄前"、"1-3 年级"或"青少年"）。教室是您安排和组织课程的地方。
4. **开始安排课程** -- 浏览内容库，找到适合您需求的学习单元，并为教室安排课程。

教室设置好并安排了课程后，您的志愿者可以通过门户访问内容并向班级演示课程。

:::tip
如果您已有来自 B1.church 的 ChurchApps 账号，可以使用相同的凭据登录 Lessons.church——无需重新注册。
:::
