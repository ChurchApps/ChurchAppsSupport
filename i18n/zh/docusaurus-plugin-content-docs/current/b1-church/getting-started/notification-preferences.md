# 通知偏好设置

<div class="article-intro">

通知偏好设置让你决定你从你的教堂接收哪些通知以及如何接收它们 — 推送通知、电子邮件或应用内铃声。你可以微调每种通知类型、设置静音时段或一次性静音所有内容，全部从一个屏幕上。

</div>

## 打开你的偏好设置

1. 在 B1 成员应用中打开**通知**区域（铃声图标）。
2. 点击**通知偏好设置**。

## 全局控制

在屏幕顶部你会找到适用于所有内容的设置：

- **静音所有** — 临时暂停所有非必要通知。
- **推送通知** — 为你的设备打开或关闭推送通知。
- **电子邮件频率** — 选择电子邮件通知如何到达：
  - **每个通知** — 在事情发生时发送电子邮件。
  - **每日摘要** — 将它们捆绑到一个每日电子邮件中。
  - **从不** — 不发送通知电子邮件。
- **静音时段** — 设置你宁愿不被打扰的开始和结束时间（以及你的时区）。非紧急通知等到静音时段结束。

## 按类型选择通知

全局控制下面是一个通知类型网格。对于每个，你可以打开或关闭你想要的频道：

- **推送** — 你设备上的通知。
- **电子邮件** — 你收件箱中的消息。
- **应用内** — 应用内的铃声/徽章。

常见的类型包括**事件提醒**、**服务与安排**、**直接消息**、**组群聊天**、**祈祷请求**和**教堂公告**。例如，你可能对直接消息保持推送通知打开，但对公告关闭它们。

:::info
一些通知类型**始终打开**且无法关闭 — **账户和安全**、**捐赠收据和报表**和**签到安全警报**。这些是出于你的安全和法律原因所需的，所以它们显示一个小锁定图标。其他一切都由你控制。
:::

一些类型**默认关闭**（例如宣传或筹款消息），只有在你选择选择加入时才能联系你。

## 保存你的更改

调整你的偏好后，点击**保存**。你的更改立即生效并适用于你使用 B1 的任何地方。

:::tip
时间敏感通知 — 如安全警报或儿童签到安全消息 — 即使在静音时段也会联系你。日常更新等到静音时段结束。
:::
