# 登录

<div class="article-intro">

登录 B1.church 后，您可以访问会员门户，在其中查看时间线、管理小组、在线奉献以及使用教会设置的所有工具。本文介绍如何登录、使用用户菜单以及退出登录。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个 ChurchApps 账户。请参阅[入门指南](../getting-started/)创建账户。
- 您需要教会的 B1.church 网址（例如 `https://yourchurchname.b1.church`）。

</div>

## 如何登录

1. 前往教会的 B1.church 网址（例如 `https://yourchurchname.b1.church`）。
2. 点击页头右上角的**登录**按钮。
3. 输入您的**电子邮件地址**和**密码**。
4. 点击**登录**。

登录后，您将返回到点击登录前所在的页面。如果您在未登录状态下直接导航到会员门户页面，登录后将自动返回该页面。

## 访问会员门户

登录后，您可以通过点击页面右上角的**姓名**，并从菜单中选择**会员门户**来访问会员门户。会员门户是您所有个人教会工具的所在之处，包括[时间线](../community/timeline.md)、[小组](../groups/)、[奉献](../giving/)等。

## 您的用户菜单

登录后，您的姓名会以标签形式显示在右上角。点击它可打开一个菜单，包含以下选项：

- **会员门户** -- 前往您的个人会员门户仪表板。
- **编辑资料** -- 更新您的账户设置或编辑教会个人资料。
- **退出登录** -- 退出您的账户。

## 管理员访问

如果您是拥有相应权限的教会管理员，您还将在用户菜单中看到**管理门户**选项。点击它将进入 B1 Admin，您可以在其中管理教会的设置、人员、小组和内容。

:::tip
如果您忘记了密码，请使用登录页面上的**忘记密码**链接进行重置。您将收到一封包含创建新密码说明的电子邮件。
:::

## 退出登录

要退出登录，请点击右上角的**姓名**并选择**退出登录**。您将退出登录并返回到公开网站。
