# 登录

<div class="article-intro">

登录 B1.church 为您提供对成员门户的访问权限，您可以在其中查看您的时间线、管理组、进行在线捐赠并使用您的教会设置的所有工具。本文介绍了如何登录、导航您的用户菜单和登出。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要一个 ChurchApps 账户。参阅[入门](../getting-started/)来创建一个。
- 您需要您教会的 B1.church URL（例如，`https://yourchurchname.b1.church`）。

</div>

## 如何登录

1. 转到您教会的 B1.church URL（例如，`https://yourchurchname.b1.church`）。
2. 点击标题右上角的 **Login** 按钮。
3. 输入您的**电子邮件地址**和**密码**。
4. 点击 **Sign In**。

登录后，您将被带回到您在点击登录前所在的页面。如果您在登出时直接导航到成员门户页面，登录后您将自动返回到该页面。

## 访问成员门户

登录后，您可以通过点击页面右上角的您的**名字**并从菜单中选择 **Member Portal** 来访问成员门户。成员门户是您所有个人教会工具的所在地，包括您的[时间线](../community/timeline.md)、[小组](../groups/)、[捐赠](../giving/)等。

## 您的用户菜单

登录后，您的名字显示为右上角的芯片。点击它来打开包含以下选项的菜单：

- **Member Portal**——转到您的个人成员门户仪表板。
- **Edit Profile**——更新您的账户设置或编辑您的教会资料。
- **Logout**——登出您的账户。

## 管理员访问

如果您是具有适当权限的教会管理员，您还会在用户菜单中看到 **Admin Portal** 选项。点击它会带您到 B1 Admin，您可以在那里管理您的教会设置、人员、小组和内容。

## 忘记密码

如果您记不住您的密码：

1. 点击登录页面上的 **Forgot Password**。
2. 输入与您的 ChurchApps 账户关联的电子邮件地址并提交。
3. 查看您的电子邮件中包含**6 位验证码**的邮件。该代码的有效期为 15 分钟。
4. 在验证屏幕上输入代码来登录。
5. 登录后，访问您的资料来选择新密码。

:::tip
如果代码没有在几分钟内到达，请检查您的垃圾邮件文件夹。五次错误尝试后代码被锁定——如果发生这种情况，请从"忘记密码"页面请求一个新代码。
:::

## 登出

要登出，点击右上角的您的**名字**并选择 **Logout**。您将被登出并返回到公共网站。
