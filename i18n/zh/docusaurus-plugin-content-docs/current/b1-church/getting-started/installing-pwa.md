# 安装为应用（PWA）

<div class="article-intro">

B1.church支持作为Progressive Web App（PWA）安装，这让您可以像原生应用一样将教会的成员门户添加到手机的主屏幕。这提供了完整屏幕、应用式体验，而无需通过App Store或Google Play。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要教会的B1.church URL：`https://yourchurchname.b1.church`
- 使用支持的浏览器：iOS上的**Safari**或Android上的**Chrome**

</div>

## 什么是PWA?

Progressive Web App（PWA）是一个可以安装在您设备上并表现得像原生应用的网站。当您将B1.church安装为PWA时，它：

- 全屏打开，没有浏览器导航栏
- 以您教会的图标和名称显示在主屏幕上
- 与教会的品牌和主题颜色顺利协作

:::tip
如果您不想从App Store或Google Play安装单独的应用，PWA是一个很好的选择。它提供与您在浏览器中获得的相同成员门户体验，以更方便的包形式。
:::

## 从何处安装

您可以从教会B1.church网站上的**任何页面**安装PWA——公开主页、讲道页面、登录屏幕或成员门户内的任何地方。没有单独的"安装URL"要记住；如果您可以在网络上访问教会，您可以从那里安装应用。

当您启动已安装的应用时，它会直接打开到成员仪表板，准备好让您登录。

## 在iPhone上安装（iOS——Safari）

1. 在iPhone上打开**Safari**，转到`https://yourchurchname.b1.church`（用教会的URL替换`yourchurchname`）。
2. 点击屏幕底部的**共享**按钮（带向上箭头的框）。
3. 在共享表中向下滚动，然后点击**添加到主屏幕**。
4. 名称将默认为教会的名称。点击右上角的**添加**。

应用图标将出现在主屏幕上。点击它以全屏模式打开成员门户。

:::info
iOS上的PWA安装需要Safari。如果您在iPhone上使用Chrome或其他浏览器，请先切换到Safari。
:::

## 在Android上安装（Chrome）

1. 在Android设备上打开**Chrome**，转到`https://yourchurchname.b1.church`。
2. Chrome将自动在屏幕底部显示**"添加到主屏幕"**横幅，或点击右上角的三点菜单（⋮）并选择**添加到主屏幕**。
3. 出现提示时点击**添加**。

应用图标将出现在主屏幕上。点击它以打开成员门户。

## 安装后

安装PWA后，它会直接在专用全屏窗口中打开到成员门户。您可以：

- 使用现有的ChurchApps账户登录
- 访问教会启用的所有成员功能——小组、捐献、签到、服务计划等
- 随时通过点击主屏幕上的图标返回

当您有未读消息时，应用图标会显示**通知徽章**（小数字徽章），所以您可以一目了然地看到是否有新的东西在等您。

:::info
PWA需要互联网连接。与某些原生应用不同，它不会缓存所有内容供离线使用。
:::

## 旧的B1 Mobile应用呢？

iOS和Android上的遗留原生**B1 Mobile**应用已被弃用，不再更新。PWA是今后在手机上访问B1.church成员门户的支持方式，它涵盖原生应用提供的所有成员面向功能。如果您之前安装了原生应用，请在方便时切换到PWA。

请参阅[B1 Mobile参考页](/docs/b1-mobile/)了解已弃用应用的历史文档。
