# 作为应用安装 (PWA)

<div class="article-intro">

B1.church 支持作为 Progressive Web App (PWA) 安装，这使您可以像原生应用一样将您的教会成员门户添加到您的手机主屏幕。这提供了完整屏幕的类应用体验，而无需通过 App Store 或 Google Play。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要您教会的 B1.church URL：`https://yourchurchname.b1.church`
- 使用支持的浏览器：iOS 上的 **Safari** 或 Android 上的 **Chrome**

</div>

## 什么是 PWA？

Progressive Web App (PWA) 是可以安装在您的设备上并表现得像原生应用的网站。当您将 B1.church 安装为 PWA 时，它：

- 以全屏方式打开，不显示浏览器导航栏
- 显示在您的主屏幕上，带有您教会的图标和名称
- 与您教会的品牌和主题色彩顺利协作

:::tip
如果您不想从 App Store 或 Google Play 安装单独的应用，PWA 是一个很好的选择。它提供了您在浏览器中获得的相同成员门户体验，但以更便捷的方式包装。
:::

## 从哪里安装

您可以从您教会 B1.church 网站上的**任何页面**安装 PWA——公共主页、讲道页面、登录屏幕或成员门户内的任何地方。没有需要记住的单独"安装 URL"；如果您可以在网络上访问您的教会，您就可以从那里安装应用。

当您启动已安装的应用时，它会直接打开到您的成员仪表板，准备好让您登录。

## 在 iPhone 上安装 (iOS — Safari)

1. 在您的 iPhone 上打开 **Safari** 并转到 `https://yourchurchname.b1.church`（用您的教会的 URL slug 替换 `yourchurchname`）。
2. 点击屏幕底部的 **Share** 按钮（带有向上箭头的框）。
3. 在分享表单中向下滚动并点击 **Add to Home Screen**。
4. 名称将默认为您的教会名称。点击右上角的 **Add**。

应用图标将出现在您的主屏幕上。点击它来以全屏模式打开成员门户。

:::info
iOS 上的 PWA 安装需要 Safari。如果您在 iPhone 上使用 Chrome 或其他浏览器，请先切换到 Safari。
:::

## 在 Android 上安装 (Chrome)

1. 在您的 Android 设备上打开 **Chrome** 并转到 `https://yourchurchname.b1.church`。
2. Chrome 会自动在屏幕底部显示 **"Add to Home screen"** 横幅，或点击右上角的三点菜单 (⋮) 并选择 **Add to Home screen**。
3. 收到提示时点击 **Add**。

应用图标将出现在您的主屏幕上。点击它来打开成员门户。

## 安装后

安装后，PWA 直接打开到成员门户中的专用全屏窗口。您可以：

- 使用您现有的 ChurchApps 账户登录
- 访问您的教会启用的所有成员功能——小组、捐赠、签到、服侍计划等
- 随时通过点击主屏幕上的图标返回到它

:::info
PWA 需要互联网连接。与某些原生应用不同，它不会缓存所有内容以供离线使用。
:::

## 旧版 B1 Mobile 应用怎样？

原生 **B1 Mobile** 应用的旧版本在 iOS 和 Android 上已被弃用，不再更新。PWA 是今后访问手机上 B1.church 成员门户的支持方式，它涵盖了原生应用提供的所有成员面向的功能。如果您之前安装了原生应用，请在方便时切换到 PWA。

有关已弃用应用的历史文档，请参阅 [B1 Mobile 参考页面](/docs/b1-mobile/)。
