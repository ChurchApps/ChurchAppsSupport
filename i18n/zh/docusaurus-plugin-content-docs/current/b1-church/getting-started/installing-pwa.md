# 安装为应用程序 (PWA)

<div class="article-intro">

B1.church 支持安装为渐进式网络应用程序 (PWA)，它让您可以将您的教会的成员门户添加到手机的主屏幕，就像原生应用一样。这为您提供了完整屏幕的、类似应用程序的体验，而无需通过应用商店或 Google Play。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要您的教会的 B1.church 移动网址：`https://yourchurchname.b1.church/mobile`
- 使用支持的浏览器：iOS 上的 **Safari** 或 Android 上的 **Chrome**

</div>

## 什么是 PWA？

渐进式网络应用程序 (PWA) 是一个可以安装在您的设备上并行为像原生应用的网站。当您将 B1.church 安装为 PWA 时，它会：

- 无浏览器导航栏全屏打开
- 在您的主屏幕上显示您的教会的图标和名称
- 与您的教会的品牌和主题颜色平稳配合

:::tip
如果您不想从应用商店或 Google Play 安装单独的应用程序，PWA 是一个很好的选择。它提供您在浏览器中获得的相同成员门户体验，采用更便利的方案。
:::

## 在 iPhone (iOS — Safari) 上安装

1. 在您的 iPhone 上打开 **Safari**，转到 `https://yourchurchname.b1.church/mobile`（将 `yourchurchname` 替换为您的教会的网址段）。
2. 点击屏幕底部的**共享**按钮（带向上箭头的方框）。
3. 在共享表中向下滚动，点击**添加到主屏幕**。
4. 名称将默认为您的教会的名称。点击右上角的**添加**。

应用图标将出现在您的主屏幕上。点击它以全屏模式打开成员门户。

:::info
iOS 上的 PWA 安装需要 Safari。如果您在 iPhone 上使用 Chrome 或其他浏览器，请先切换到 Safari。
:::

## 在 Android (Chrome) 上安装

1. 在您的 Android 设备上打开 **Chrome**，转到 `https://yourchurchname.b1.church/mobile`。
2. Chrome 将自动在屏幕底部显示**"添加到主屏幕"**横幅，或点击右上角的三点菜单 (⋮)，选择**添加到主屏幕**。
3. 在提示时点击**添加**。

应用图标将出现在您的主屏幕上。点击它以打开成员门户。

## 安装后

安装后，PWA 的工作方式与在浏览器中访问移动网站相同，但在专用的全屏窗口中。您可以：

- 使用现有的 ChurchApps 帐户登录
- 访问您的教会启用的所有成员功能 -- 小组、奉献、签到、服务计划等
- 通过点击主屏幕上的图标随时返回到它

:::info
PWA 需要互联网连接。与某些原生应用不同，它不会缓存所有内容以供离线使用。
:::

## PWA 与原生应用

两个选项都可让您访问 B1.church 成员门户。正确的选择取决于您的偏好：

| | **PWA** | **B1 Mobile 应用** |
|---|---|---|
| 安装 | 从浏览器添加到主屏幕 | 从应用商店 / Google Play 下载 |
| 更新 | 自动（无需应用商店更新） | 通过应用商店 / Google Play |
| 推送通知 | 不支持 | 支持 |
| 平台 | 任何具有浏览器的设备 | iOS 和 Android |

:::tip
如果您的教会使用群组对话，并且希望收到有关新消息和活动的推送通知，请改用 [B1 Mobile 应用](/docs/b1-mobile/)。PWA 目前不支持推送通知。
:::
