# B1.church 开始使用

<div class="article-intro">

B1.church 是您教会的在线主页。这是您保持与会众联系、查看更新、在线奉献、办理签到参加服务等的地方。本部分将引导您了解访问和使用成员门户的基础知识。

</div>

## 访问您的教会网站

您的教会有自己的 B1.church 网址，通常格式为 `https://yourchurchname.b1.church`。您可以在任何计算机、平板电脑或手机上的网络浏览器中访问此网址。您的教会领导或欢迎团队可以为您提供确切的地址。

首次访问该网站时，您将看到您的教会的公共网站，其中包含自定义品牌、导航菜单和内容页面。

## 创建帐户

要访问成员门户及其功能，您需要 ChurchApps 帐户。

1. 访问您的教会的 B1.church 网址。
2. 单击页面右上角的**登录**按钮。
3. 在登录屏幕上，选择**注册**新帐户的选项。
4. 输入您的**名字**、**姓氏**、**电子邮件地址**，然后选择**密码**。
5. 完成注册表单。

注册后，您可以登录并访问成员门户及其所有工具。

:::tip
如果您的教会已将您添加到他们的系统中，请使用他们文件中的相同电子邮件地址。这会将您的帐户链接到您现有的教会档案。
:::

## 您可以做什么

登录后，成员门户可让您访问：

- 您个人的**时间线**，包含来自您的小组和教会的更新
- 您所属的**小组**，包括对话和活动
- 用于捐赠的在线**奉献**
- 显示您的志愿者日程和服务详情的**计划**
- 服务和活动的**签到**
- 用于查找和连接其他成员的**社区**目录
- 用于圣经研究和课程的**课程**

## 在手机上访问

B1.church 适用于任何移动浏览器，但您也可以通过两种方式获得更像应用程序的体验：

- **安装为 PWA** -- 直接从浏览器将您的教会的成员门户添加到主屏幕。无需应用商店。有关说明，请参阅[安装为应用程序 (PWA)](./installing-pwa.md)。
- **下载 B1 Mobile** -- 从应用商店或 Google Play 安装原生应用程序以获得完整的移动体验，包括推送通知。请参阅 [B1 Mobile](/docs/b1-mobile/)。
