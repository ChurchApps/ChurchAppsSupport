# B1.church 入门指南

<div class="article-intro">

B1.church 是您教会的线上家园。在这里，您可以与会众保持联系、查看更新、在线奉献、礼拜签到等等。本节将引导您了解访问和使用会员门户的基础知识。

</div>

## 访问教会网站

您的教会拥有自己的 B1.church 网址，通常格式为 `https://yourchurchname.b1.church`。您可以在电脑、平板或手机的任何网络浏览器中访问此网址。教会领袖或欢迎团队可以为您提供确切的地址。

首次访问该网站时，您将看到教会的公开网站，包含其自定义品牌、导航菜单和内容页面。

## 创建账户

要访问会员门户及其功能，您需要一个 ChurchApps 账户。

1. 访问教会的 B1.church 网址。
2. 点击页面右上角的**登录**按钮。
3. 在登录界面，选择**注册**新账户的选项。
4. 输入您的**名字**、**姓氏**、**电子邮件地址**，并选择一个**密码**。
5. 完成注册表单。

注册后，您即可登录并使用会员门户中的所有工具。

:::tip
如果教会已将您添加到其系统中，请使用他们存档的同一电子邮件地址。这将把您的账户链接到您现有的教会个人资料。
:::

## 您可以做什么

登录后，会员门户让您可以访问：

- 您的个人**时间线**，包含来自小组和教会的更新
- 您所属的**小组**，包括对话和活动
- 在线**奉献**功能
- **计划**显示您的志愿者排班和礼拜详情
- 礼拜和活动的**签到**
- **社区**目录，查找和联系其他成员
- 用于查经和课程的**课程**
