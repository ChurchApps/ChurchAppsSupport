# 导航 B1App

<div class="article-intro">

B1.church 中的成员门户使用侧边栏导航布局，可以轻松在可用工具之间移动。此页面解释导航如何在桌面和移动设备上工作。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要[登录](./logging-in.md)才能访问成员门户。

</div>

## 侧边栏导航

当您进入成员门户时，您将在屏幕左侧看到一个垂直侧边栏。此侧边栏包含门户每个部分的选项卡：

- **时间线** -- 您的个性化更新和来自您的小组和教会的对话反馈。这是打开成员门户时加载的默认选项卡。
- **我** -- 您的个人仪表板，包括即将推出的服侍任务、注册、小组事件和最近通知。请参见[我的页面](./me-page)。
- **小组** -- 查看和交互您[所属的小组](../groups/)。
- **社区** -- 浏览教会[成员目录](../community/member-directory.md)。
- **计划** -- 查看即将推出的[服侍计划](../plans/)和您分配的志愿者职位。
- **签到** -- 为服务和活动[签到](../checkin/)自己和您的家庭。
- **课程** *(已弃用)* -- 门户中的课程选项卡已被[FreePlay](/docs/freeplay/)取代用于课堂播放。在 [Lessons.church](https://lessons.church) 直接浏览课程或请参见[课程参考页面](../content/lessons.md)了解详细信息。
- **捐赠** -- 进行在线[捐赠](../giving/)并查看您的捐赠历史。

点击任何选项卡以切换到该部分。活动选项卡会被突出显示，因此您始终知道您在哪里。

:::info
您看到的选项卡可能因您的教会启用的功能而异。教会管理员通过 B1 Admin 控制成员可见哪些部分。如果您看不到特定选项卡，您的教会可能尚未打开该功能。
:::

## 移动导航

在手机和平板电脑等较小屏幕上，侧边栏折叠以节省空间。相反，您将在成员门户顶部看到**菜单**按钮。点击它以打开所有可用选项卡的列表。选择一个选项卡以导航到该部分，菜单将自动关闭。

## 管理员选项卡

如果您在教会有管理员权限，您将在侧边栏底部看到额外的**管理员**选项卡。点击它将您带到 B1 Admin，您可以在其中管理您的教会的设置和数据。

## 顶部标题导航

在成员门户之外，主网站标题包含您的教会的自定义导航链接。这些由您的教会的管理员配置，可能包括链接到[讲道](../content/sermons.md)、[圣经](../content/bible.md)、[直播](../content/live-streaming.md)和其他内容的页面。在移动设备上，这些链接可通过标题右上角的汉堡菜单图标访问。
