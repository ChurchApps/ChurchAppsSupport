# 导航 B1App

<div class="article-intro">

B1.church 的会员门户采用侧边栏导航布局，方便您在不同的可用工具之间切换。本页说明导航在桌面和移动设备上的工作方式。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要[登录](./logging-in.md)才能访问会员门户。

</div>

## 侧边栏导航

进入会员门户后，您将在屏幕左侧看到一个垂直侧边栏。该侧边栏包含门户各个板块的选项卡：

- **时间线** -- 您的个性化动态，包含来自小组和教会的更新和对话。这是打开会员门户时默认加载的选项卡。
- **小组** -- 查看并与您所属的[小组](../groups/)互动。
- **社区** -- 浏览教会[会员目录](../community/member-directory.md)。
- **计划** -- 查看即将到来的[礼拜计划](../plans/)和您分配的志愿者岗位。
- **签到** -- 为自己和家庭成员进行礼拜和活动[签到](../checkin/)。
- **课程** -- 访问查经[课程](../content/lessons.md)和课程内容。
- **奉献** -- 进行在线[奉献](../giving/)并查看您的奉献历史。

点击任意选项卡即可切换到该板块。当前活跃的选项卡会高亮显示，让您始终知道自己在哪里。

:::info
您看到的选项卡可能因教会启用的功能而异。教会管理员通过 B1 Admin 控制哪些板块对会员可见。如果您没有看到某个选项卡，可能是教会尚未开启该功能。
:::

## 移动端导航

在手机和平板等较小屏幕上，侧边栏会折叠以节省空间。取而代之的是，您将在会员门户顶部看到一个**菜单**按钮。点击它可打开所有可用选项卡的列表。选择一个选项卡即可导航到该板块，菜单将自动关闭。

## 管理选项卡

如果您拥有教会的管理员权限，您将在侧边栏底部看到一个额外的**管理**选项卡。点击它将进入 B1 Admin，您可以在其中管理教会的设置和数据。

## 顶部页头导航

在会员门户之外，主网站页头包含教会自定义的导航链接。这些由教会管理员配置，可能包含指向[讲道](../content/sermons.md)、[圣经](../content/bible.md)、[直播](../content/live-streaming.md)和其他内容页面的链接。在移动端，这些链接可通过页头右上角的汉堡菜单图标访问。
