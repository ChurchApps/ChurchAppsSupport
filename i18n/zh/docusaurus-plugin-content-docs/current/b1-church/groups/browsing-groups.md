# 浏览小组

<div class="article-intro">

教会的公开小组页面让您可以轻松发现感兴趣的小组。小组按类别组织，您可以快速浏览不同类型的小组并找到合适的。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 浏览小组无需登录。但是，[登录](../getting-started/logging-in.md)后可以在每个小组的详情页面访问额外功能。

</div>

## 找到小组页面

在教会的 B1.church 网站上导航到**小组**页面。此页面是公开访问的，因此您无需登录即可浏览。

## 小组的组织方式

小组按**类别**（也称为标签）显示。类别由教会管理员设置，可能包括"小组团契"、"事工"、"青年"或"查经班"等标签。每个类别显示其下的所有小组。

## 阅读小组卡片

每个小组以卡片形式显示，包含以下信息：

- **小组照片** -- 代表小组的封面图片。如果未设置照片，则显示默认图片。
- **小组名称** -- 醒目地显示在照片上。
- **聚会时间** -- 小组通常的聚会时间（例如"每周三晚 7 点"）。如果未设置时间，则显示占位符。
- **聚会地点** -- 小组聚会的地点。如果未设置地点，则显示占位符。
- **描述** -- 关于小组内容的简短预览，限制在几行内。点击小组可阅读完整描述。

## 查看小组

点击任意小组卡片可打开完整的小组详情页面。在那里您可以找到完整描述、小组负责人信息、即将举行的日历活动以及联系方式。

:::tip
如果您正在寻找特定类型的小组，请先浏览类别标题。这有助于您直接跳到感兴趣的板块，而无需滚动浏览每个小组。
:::
