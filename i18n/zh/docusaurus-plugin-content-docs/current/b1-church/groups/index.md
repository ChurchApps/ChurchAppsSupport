# 小组

<div class="article-intro">

小组是与教会其他成员建立联系的好方式。无论您是在寻找小组团契、事工团队、查经班还是志愿服务机会，您都可以直接从教会的 B1.church 网站浏览和加入小组。

</div>

## 您可以做什么

以下是您在 B1App 中可以使用小组功能做的事情：

- **浏览公开小组** -- 查看按类别组织的所有可用小组。每个小组显示照片、名称、聚会时间、地点和简短描述。
- **查看小组详情** -- 点击任意小组可查看其完整描述、负责人、即将举行的日历活动以及联系负责人的表单。
- **加入小组** -- 与小组建立联系，成为成员并获得额外功能的访问权限。
- **参与对话** -- 成为小组成员后，您可以通过对话选项卡发布消息并与其他小组成员互动。
- **访问日历和资源** -- 在小组日历上查看即将举行的活动，从资源选项卡下载共享文件或链接。

## 公开与登录后的体验

任何人都可以访问教会的小组页面并浏览可用的小组。未登录的访客可以查看小组描述、了解负责人、查看即将举行的日历活动，以及使用联系表单向负责人发送消息。

[登录](../getting-started/logging-in.md)后，您将在小组详情页面解锁额外的选项卡，包括**小组详情**、**日历**、**对话**、**资源**、**成员**和**出勤**。小组负责人在每个选项卡上拥有额外的编辑功能。

:::tip
小组也会显示在您的[时间线](../community/timeline.md)上——来自小组的对话和更新将自动出现在您的活动动态中。
:::
