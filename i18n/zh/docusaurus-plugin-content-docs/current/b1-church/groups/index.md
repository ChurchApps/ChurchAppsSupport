# 小组

<div class="article-intro">

小组是与您的教会中其他人联系的好方法。无论您是在寻找小组、事工团队、圣经研究还是志愿机会，您都可以直接从您教会的 B1.church 网站浏览和加入小组。

</div>

## 您可以做什么

以下是您可以在 B1App 中对小组做的事情：

- **浏览公开小组** -- 查看按类别组织的所有可用小组。每个小组显示照片、名称、会议时间、位置和简短描述。
- **查看小组详细信息** -- 点击任何小组以查看其完整描述、领导者、即将推出的日历事件和联系表格以联系小组的领导者。
- **加入小组** -- 连接到小组以成为成员。开放小组允许您立即加入；需要批准的小组允许您提交加入请求。请参见[请求加入小组](./join-requests.md)。
- **参与对话** -- 成为小组成员后，您可以发布消息并通过对话选项卡与其他小组成员交互。
- **访问日历和资源** -- 查看小组日历上的即将推出的活动并从资源选项卡下载共享文件或链接。

## 公开与登录体验

任何人都可以访问您教会的小组页面并浏览可用小组。未登录的访问者可以查看小组描述、查看谁是领导者、检查即将推出的日历事件并使用联系表格向领导者发送消息。

当您[登录](../getting-started/logging-in.md)时，您在小组详细信息页面上解锁额外的选项卡，包括**小组详细信息**、**日历**、**对话**、**资源**、**成员**和**出勤**。小组领导者在每个选项卡上具有额外的编辑功能。

:::tip
小组也出现在您的[时间线](../community/timeline.md)上 -- 来自您的小组的对话和更新将自动显示在您的活动反馈中。
:::
