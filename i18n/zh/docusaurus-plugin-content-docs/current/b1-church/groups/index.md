# 小组

<div class="article-intro">

小组是与教会中其他人联系的好方法。无论您是在寻找小组、事工小组、圣经学习还是志愿机会，您都可以直接从教会的B1.church网站浏览并加入小组。

</div>

## 您可以做什么

以下是您在B1App中可以对小组进行的操作：

- **浏览公开小组** ——查看按类别组织的所有可用小组。每个小组显示照片、名称、会议时间、位置和简短描述。
- **查看小组详情** ——点击任何小组以查看其完整描述、领导者、即将到来的日历事件和与小组领导者联系的表。
- **加入小组** ——与小组联系以成为成员。开放小组让您立即加入；需要批准的小组让您提交加入请求。请参阅[请求加入小组](./join-requests.md)。
- **参与对话** ——成为小组成员后，您可以通过对话标签与其他小组成员发送消息和互动。
- **访问日历和资源** ——查看小组日历上的即将到来的事件并从资源标签下载共享文件或链接。

## 公开与已登录体验

任何人都可以访问教会的小组页面并浏览可用的小组。未登录的访客可以查看小组描述、看到谁是领导者、检查即将到来的日历事件，并使用联系表向领导者发送消息。

当您[登录](../getting-started/logging-in.md)时，您会解锁小组详情页面上的额外标签，包括**小组详情**、**日历**、**对话**、**资源**、**成员**和**出席**。小组领导者在每个标签上都有额外的编辑功能。

:::tip
小组也出现在您的[时间线](../community/timeline.md)上——来自小组的对话和更新会自动显示在您的活动源中。
:::
