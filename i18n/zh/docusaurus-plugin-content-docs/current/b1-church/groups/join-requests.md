# 请求加入小组

<div class="article-intro">

有些小组在您可以成为成员之前需要批准。当小组有此设置时，您提交加入请求——可选地向小组领导者包括消息——并等待领导者批准或拒绝。您可以查看待处理请求的状态并随时取消它们。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您必须[登录](../getting-started/logging-in.md)到您的B1.church账户才能请求加入小组。

</div>

## 加入与请求加入

小组可以有其管理员设置的不同加入政策：

- **开放** ——在小组的详情页面上点击**加入小组**，您会立即被添加为成员。
- **需要请求** ——点击**请求加入**提交小组领导者必须批准才能成为成员的请求。
- **关闭** ——没有加入按钮出现。成员必须由领导者手动添加。

## 如何提交加入请求

1. 浏览到您想要加入的小组。如果您需要帮助查找，请参阅[浏览小组](./browsing-groups.md)。
2. 打开小组的详情页面。
3. 点击**请求加入**。
4. 出现一个对话框，您可以选择向小组领导者写一条消息，解释您为什么想加入。
5. 点击**提交**发送请求。

小组的领导者会收到关于您请求的通知。一旦他们审核，您会收到一个带有结果的通知。

:::tip
包括一条简短的消息——比如您如何听说这个小组或您希望从中得到什么——可以帮助领导者更快地做出决定。
:::

## 检查请求的状态

提交请求后，您可以从**小组**页面查看其状态：

1. 导航到教会B1.church网站上的**小组**页面。
2. 向下滚动到**待处理请求**部分。
3. 每个待处理请求显示小组名称和您提交的日期。

如果请求已被批准或拒绝，您会收到通知——批准的请求会将您作为成员移入小组。

## 取消请求

如果您在领导者审核之前改变主意：

1. 转到**小组**页面。
2. 在**待处理请求**部分找到您的请求。
3. 点击请求旁的**取消**。

请求会立即被撤销，领导者将不再看到它。
