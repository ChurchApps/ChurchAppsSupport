# 小组详情

<div class="article-intro">

从小组页面点击某个小组后，您将进入小组详情页面。您看到的内容取决于您是否已登录，已登录的成员可以访问额外的选项卡和功能。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 通过[浏览小组](./browsing-groups.md)找到要查看的小组。
- 要获得所有选项卡的完整访问权限，请[登录](../getting-started/logging-in.md)您的账户。

</div>

## 访客视图（未登录）

如果您未登录，您将看到以下内容：

- **主图区** -- 显示小组照片、名称、聚会时间和聚会地点的大型横幅区域。
- **描述** -- 小组的完整描述，说明其目的和预期内容。
- **负责人** -- 小组负责人的头像和姓名。您可以点击负责人的姓名查看其个人资料。
- **日历活动** -- 最多显示三个即将举行的小组活动，每个显示日期和活动标题。如果小组没有即将举行的活动，此板块将隐藏。
- **联系表单** -- 允许您直接向小组负责人发送消息的表单。填写您的**名字**、**姓氏**、**电子邮件**、**电话号码**和**消息**，然后点击**提交**。如果有多位负责人，您可以从下拉菜单中选择要联系的人。

## 成员视图（已登录）

登录后，小组详情页面会显示主图区以及侧边栏选项卡。可用的选项卡包括：

- **小组详情** -- 小组的完整描述，以富文本格式呈现。
- **成员** -- 小组所有成员的列表，显示其照片和姓名。您可以点击成员姓名在[会员目录](../community/member-directory.md)中访问其个人资料。
- **日历** -- 小组日历，显示聚会和活动。您可以浏览即将到来和过去的活动。
- **对话** -- 小组成员发布消息和相互交流的空间。详见[小组对话](./conversations.md)。
- **资源** -- 为小组发布的共享文件和链接。您可以下载上传的文件或打开链接资源。
- **出勤** -- 仅对小组负责人可见。允许跟踪小组聚会的出勤情况。

## 负责人功能

如果您是小组的负责人，您拥有额外的编辑功能：

- **编辑小组详情** -- 直接从**详情**选项卡更新小组的名称、描述、聚会时间和其他设置。
- **管理成员** -- 从**成员**选项卡添加新成员或移除现有成员。
- **管理资源** -- 上传文件（最多 100 MB 存储空间）、添加链接或删除现有资源。
- **负责人资源** -- 访问单独的**资源（负责人）**选项卡，其中包含仅对小组负责人可见的文件和链接。
- **记录出勤** -- 从**出勤**选项卡记录小组会议的出勤情况。

:::info
如果您尚未成为小组成员，可以使用访客视图中的联系表单联系小组负责人，咨询如何加入。
:::
