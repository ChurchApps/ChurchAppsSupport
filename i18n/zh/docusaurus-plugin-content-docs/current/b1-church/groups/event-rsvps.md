# 事件RSVP

<div class="article-intro">

小组事件可以收集RSVP，以便领导者知道谁会来。在您小组的日历上，每个即将到来的事件都会问"您会参加吗？" — 点击回答，随时改变您的想法，一目了然地看到有多少其他人要去。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您必须[登录](../getting-started/logging-in.md)并且是小组的成员
- 事件必须允许RSVP（默认情况下打开；领导者可以按事件关闭）

</div>

## 回复事件

1. 在成员门户中打开您的小组并进入其**事件**/日历视图。
2. 每个即将到来的事件卡片都显示**您会参加吗？**带有三个按钮：**会去**、**也许**和**不能去**。
3. 点击您的答案。您的选择会突出显示，每个按钮显示有多少人给出了该回复的实时计数。
4. 再次点击您突出显示的答案以清除它，或点击不同的按钮以更改您的回复。

您的RSVP适用于该特定事件 — 对于每周一次的会议，您分别回答每周的事件。

## 对于小组领导者

- **查看回复** -- 领导者在每个事件卡片上看到一个**查看回复**按钮。它打开**回复**表，按回答列出每个人 — 会去、也许和不能去。如果还没有人回答，它显示"尚无回复"。
- **打开或关闭RSVP** -- 创建或编辑事件时，使用**允许RSVP**开关。关闭RSVP的事件不显示回复按钮。从B1 Admin批量创建事件的工作人员有相同的选项 — 请参见[小组日历](../../b1-admin/groups/group-calendar#event-rsvps)。

:::tip
RSVP是一个轻量级的人数统计。如果您的事件需要注册详情、容量管理或付款，请改为使用[事件注册](../events/registering)。
:::
