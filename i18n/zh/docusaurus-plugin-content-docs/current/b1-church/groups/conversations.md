# 小组对话

<div class="article-intro">

小组页面上的对话标签是小组成员可以相互发送消息和互动的空间。使用对话来讨论话题、共享更新，并在会议之间与小组保持联系。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您必须[登录](../getting-started/logging-in.md)到您的B1.church账户。
- 您必须是小组成员才能访问对话标签。有关如何加入小组，请参阅[小组详情](./group-details.md)。

</div>

## 访问对话

1. 通过从[小组页面](./browsing-groups.md)点击小组来导航到小组详情页面。
2. 确保您已登录。
3. 点击侧栏中的**对话**标签。

## 讨论和公告

对话标签可能显示两个子标签：

- **讨论** ——任何小组成员都可以发送消息并回复他人的空间。这是默认视图，始终可用。
- **公告** ——小组领导者发送的对所有成员都重要的消息。常规成员可以阅读公告，但只有领导者可以创建它们。

如果没有公告且您不是领导者，您将仅看到讨论视图，没有标签栏。

## 发送消息

1. 在小组页面上打开**对话**标签。
2. 确保您在**讨论**子标签上（如果标签栏可见）。
3. 在文本字段中输入您的消息。
4. 提交您的消息以与小组共享。

您的消息将对小组的所有成员可见。

## 编辑或删除您的消息

您可以编辑或删除您发布的任何消息：

1. 将鼠标悬停在您的消息上（或在移动设备上点击它）以显示**更多选项**按钮（⋮）。
2. 点击按钮打开消息菜单。
3. 选择**编辑**以修改消息文本，或**删除**以删除它。
   - 编辑时，消息文本加载回输入字段。进行更改并提交以保存。
   - 删除时，出现确认对话框。点击**确认**以永久删除消息。

:::info
您只能编辑或删除自己的消息。其他成员发送的消息不显示编辑/删除选项。
:::

:::tip
小组对话也出现在您的[时间线](../community/timeline.md)上，所以您可以跟上讨论，而不必单独访问每个小组。
:::

## 领导者功能

如果您是小组领导者，您在对话标签中有额外的功能：

- **发送公告** ——切换到**公告**子标签并发送所有小组成员都可以阅读的消息。
- **参与讨论** ——您也可以像其他任何成员一样参与常规讨论。

:::info
对话特定于每个小组。您在一个小组中发布的消息不会出现在另一个小组的对话中。
:::
