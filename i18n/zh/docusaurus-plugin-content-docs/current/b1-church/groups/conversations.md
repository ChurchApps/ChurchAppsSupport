# 小组对话

<div class="article-intro">

小组页面上的对话选项卡是小组成员可以发布消息并相互交流的空间。使用对话功能来讨论话题、分享更新，并在聚会间隙与小组保持联系。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您必须[登录](../getting-started/logging-in.md)您的 B1.church 账户。
- 您必须是该小组的成员才能访问对话选项卡。请参阅[小组详情](./group-details.md)了解如何加入小组。

</div>

## 访问对话

1. 从[小组页面](./browsing-groups.md)点击某个小组，导航到小组详情页面。
2. 确保您已登录。
3. 在侧边栏中点击**对话**选项卡。

## 讨论和公告

对话选项卡可能显示两个子选项卡：

- **讨论** -- 任何小组成员都可以发布消息和回复他人的空间。这是默认视图，始终可用。
- **公告** -- 由小组负责人发布的、所有成员都应关注的重要消息。普通成员可以阅读公告，但只有负责人可以创建公告。

如果没有公告且您不是负责人，您将只看到讨论视图而没有选项卡栏。

## 发布消息

1. 在小组页面上打开**对话**选项卡。
2. 确保您在**讨论**子选项卡上（如果选项卡栏可见）。
3. 在文本框中输入您的消息。
4. 提交消息以与小组分享。

您的消息将对小组所有成员可见。

:::tip
小组对话也会显示在您的[时间线](../community/timeline.md)上，因此您无需逐个访问每个小组即可跟进讨论。
:::

## 负责人功能

如果您是小组负责人，您在对话选项卡中拥有额外的功能：

- **发布公告** -- 切换到**公告**子选项卡并发布所有小组成员都能阅读的消息。
- **发布讨论** -- 您也可以像其他成员一样参与普通讨论。

:::info
对话是每个小组独立的。您在一个小组中发布的消息不会出现在另一个小组的对话中。
:::
