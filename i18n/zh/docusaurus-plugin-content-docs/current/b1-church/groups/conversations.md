# 小组对话

<div class="article-intro">

您小组页面上的对话选项卡是一个空间，小组成员可以发布消息并相互交互。使用对话来讨论主题、共享更新并与您的小组保持联系。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您必须[登录](../getting-started/logging-in.md)到您的 B1.church 账户。
- 您必须是小组的成员才能访问对话选项卡。请参见[小组详细信息](./group-details.md)了解如何加入小组。

</div>

## 访问对话

1. 通过从[小组页面](./browsing-groups.md)点击一个小组来导航到小组详细信息页面。
2. 确保您已登录。
3. 在侧边栏中点击**对话**选项卡。

## 讨论和公告

对话选项卡可能显示两个子选项卡：

- **讨论** -- 任何小组成员可以发布消息并回复其他成员的空间。这是默认视图，始终可用。
- **公告** -- 小组领导者发布的消息，对所有成员都很重要。普通成员可以阅读公告，但只有领导者可以创建它们。

如果没有公告，您不是领导者，您将只看到不带标签栏的讨论视图。

## 发布消息

1. 在您小组页面上打开**对话**选项卡。
2. 确保您在**讨论**子选项卡上（如果标签栏可见）。
3. 在文本字段中输入您的消息。
4. 提交您的消息以与小组共享。

您的消息将对小组的所有成员可见。

## 编辑或删除您的消息

您可以编辑或删除您发布的任何消息：

1. 将鼠标悬停在您的消息上（或在移动设备上点击它）以显示**更多选项**按钮（⋮）。
2. 点击按钮以打开消息菜单。
3. 选择**编辑**以修改消息文本，或**删除**以将其删除。
   - 编辑时，消息文本加载回输入字段。进行更改并提交以保存它们。
   - 删除时，确认对话出现。点击**确认**以永久删除消息。

:::info
您只能编辑或删除您自己的消息。其他成员发布的消息不显示编辑/删除选项。
:::

## 对消息做出反应

在小组聊天中，您可以用表情符号对任何消息做出反应：

1. 点击或将鼠标悬停在消息上并选择**添加反应**。
2. 从六个表情符号中选择一个：👍 ❤️ 😂 🎉 🙏 😮。

反应显示为消息下方的小芯片，用计数说明有多少人使用了每个。您自己的反应被突出显示 -- 点击芯片（或再次选择相同的表情符号）将其删除，或选择不同的表情符号来添加另一个。反应实时显示给聊天中的每个人。

:::tip
小组对话也出现在您的[时间线](../community/timeline.md)上，因此您可以跟上讨论而无需单独访问每个小组。
:::

## 领导者功能

如果您是小组领导者，您在对话选项卡中具有额外的功能：

- **发布公告** -- 切换到**公告**子选项卡并发布所有小组成员可以阅读的消息。
- **发布讨论** -- 您也可以像任何其他成员一样参与常规讨论。

:::info
对话特定于每个小组。您在一个小组中发布的消息不会出现在另一个小组的对话中。
:::
