# 捐赠历史

<div class="article-intro">

当您登录时，您可以查看您过去捐赠的完整记录并生成给税收目的的捐赠陈述。您的捐赠历史可从您 B1.church 账户的**捐赠**部分获得。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您必须[登录](../getting-started/logging-in.md)到您的 B1.church 账户。
- 您需要至少进行过一笔捐赠。请参见[进行捐赠](./making-donations.md)以开始。

</div>

## 查看您的历史

1. 登录到您的 B1.church 账户。
2. 导航到**捐赠**部分。
3. 点击左侧面板中的**历史**以查看您的过去捐赠。

您的捐赠历史显示为包含以下列的表：

- **日期** -- 捐赠何时进行。
- **方法** -- 您如何支付（例如，"卡 - Visa" 或 "银行 - ACH"）。
- **基金** -- 捐赠指向的基金（例如，普通基金、建筑基金）。
- **金额** -- 捐赠的美元金额。

:::info
仍在处理中的捐赠将以"待处理"标签显示在基金名称旁边，金额以不同颜色显示。
:::

## 导出和打印陈述

点击捐赠历史标题中的**下载**按钮以访问导出选项：

- **当前年份 (CSV)** -- 下载您当前年份所有捐赠的 CSV 文件。您可以在电子表格应用程序中打开它。
- **当前年份 (打印)** -- 打开当前年份的可打印捐赠陈述。
- **上一年 (CSV)** -- 下载您上一年所有捐赠的 CSV 文件。
- **上一年 (打印)** -- 打开上一年的可打印捐赠陈述。

:::tip
在新年早期打印您的上一年捐赠陈述，以便在税收季节准备好。可打印陈述包括您的记录需要的所有详细信息。
:::

## 定期捐赠

从**捐赠**部分，您也可以点击**定期捐赠**来管理您设置的任何预定给予。此部分允许您查看、更新或取消定期捐赠并管理您保存的支付方法。

如果定期捐赠通过 **Stripe** 收费，您也可以**暂停**它而不是取消 -- 点击捐赠旁边的暂停图标并确认。给予停止直到您点击相同图标**恢复**；暂停的捐赠在列表中标记为**已暂停**。通过其他支付提供商的定期捐赠仍然可以编辑或取消，但不能暂停。
