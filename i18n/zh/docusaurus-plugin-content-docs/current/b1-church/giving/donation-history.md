# 奉献历史

<div class="article-intro">

登录后，您可以查看过去奉献的完整记录，并生成用于税务目的的奉献报表。您的奉献历史可在 B1.church 账户的**奉献**板块中找到。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您必须[登录](../getting-started/logging-in.md)您的 B1.church 账户。
- 您需要至少有一笔奉献记录。请参阅[进行奉献](./making-donations.md)了解如何开始。

</div>

## 查看历史记录

1. 登录您的 B1.church 账户。
2. 导航到**奉献**板块。
3. 点击左侧面板中的**历史记录**查看您的过去奉献。

您的奉献历史以表格形式显示，包含以下列：

- **日期** -- 奉献的时间。
- **方式** -- 付款方式（例如"信用卡 - Visa"或"银行 - ACH"）。
- **基金** -- 奉献指定的基金（例如一般基金、建堂基金）。
- **金额** -- 奉献的美元金额。

:::info
仍在处理中的奉献将在基金名称旁显示"待处理"标签，金额以不同颜色显示。
:::

## 导出和打印报表

点击奉献历史标题中的**下载**按钮可访问导出选项：

- **本年度（CSV）** -- 下载本年度所有奉献的 CSV 文件。您可以在电子表格应用程序中打开此文件。
- **本年度（打印）** -- 打开本年度的可打印奉献报表。
- **上一年度（CSV）** -- 下载上一年度所有奉献的 CSV 文件。
- **上一年度（打印）** -- 打开上一年度的可打印奉献报表。

:::tip
在新年初期尽早打印上一年度的奉献报表，以便在报税季节备用。可打印报表包含您记录所需的所有详细信息。
:::

## 定期奉献

在**奉献**板块中，您还可以点击**定期奉献**来管理您设置的任何定期捐赠。此板块允许您查看、更新或取消定期奉献，以及管理已保存的付款方式。
