# 进行奉献

<div class="article-intro">

通过教会的 B1.church 网站有两种方式进行奉献：公开奉献页面（无需登录）和会员奉献页面（需要登录）。两者都使用由 Stripe 驱动的安全奉献表单。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 公开奉献无需账户——只需访问教会的**奉献**页面。
- 如需享受历史记录跟踪和已保存付款方式的会员奉献功能，请先[登录](../getting-started/logging-in.md)您的账户。

</div>

## 公开奉献页面（无需登录）

任何人无需创建账户即可进行奉献：

1. 导航到教会的**奉献**页面。
2. 奉献表单显示在页面左侧。选择您想要奉献的**基金**。
3. 输入您的付款信息和奉献金额。
4. 提交表单完成奉献。

在页面右侧，您将看到 **ACH 付款说明**的链接（如果您倾向于银行转账）。如果您想登录以访问奉献历史等额外功能，还有一个**登录**按钮。

:::tip
如果您在访问公开奉献页面时已经登录，将自动重定向到会员奉献页面，在那里您可以访问完整的奉献历史和已保存的付款方式。
:::

## 会员奉献页面（已登录）

登录后，奉献体验包含额外功能：

1. 点击**奉献**选项卡（或导航到**我的奉献**）。
2. 您将看到一个包含三个板块的面板：
   - **立即奉献** -- 进行一次性或定期奉献。
   - **历史记录** -- 查看过去的奉献并导出记录。
   - **定期奉献** -- 管理定期捐赠和付款方式。
3. 选择**立即奉献**打开奉献表单。
4. 选择您想要奉献的**基金**并输入奉献金额。
5. 选择已保存的付款方式或输入新的付款信息。
6. 提交表单完成奉献。

奉献处理完成后，将显示成功消息，您的奉献将出现在[奉献历史](./donation-history.md)中。

## 付款方式

奉献通过 **Stripe** 安全处理。您可以使用信用卡、借记卡或设置 ACH 银行转账。登录后，您的付款方式可以保存以备将来使用，让下次奉献更加快捷。

:::warning
如果您的奉献未能成功，请仔细检查付款信息并重试。如果问题持续存在，请联系教会办公室获取帮助。
:::
