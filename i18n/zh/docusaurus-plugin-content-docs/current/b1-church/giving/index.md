# 奉献

<div class="article-intro">

B1App 让您可以轻松地通过在线奉献支持教会。无论您想进行快速的一次性捐赠还是管理定期奉献，您所需的一切都可以在教会的 B1.church 网站上找到。

</div>

## 概述

B1App 支持两种奉献方式：

- **公开奉献** -- 任何人无需登录即可进行奉献。访问教会的奉献页面，选择基金，输入付款信息即可。不需要账户。
- **登录奉献** -- 当您在奉献前[登录](../getting-started/logging-in.md)时，您的捐赠将关联到您的账户。这使您可以访问完整的奉献历史、可打印的奉献报表、定期奉献管理和已保存的付款方式。

## 您可以做什么

以下是奉献板块中可用的功能：

- **指定基金奉献** -- 您的教会可能设置了多个基金，如一般基金、建堂基金或宣教基金。您可以选择将奉献指定到哪个基金。
- **使用信用卡或银行账户支付** -- 奉献通过 Stripe 安全处理。您可以使用信用卡或借记卡支付。也提供 ACH 银行转账说明。
- **设置定期奉献** -- 已登录的会员可以设置自动定期奉献，确保您不会错过任何一次捐赠。
- **查看奉献历史** -- 查看过去奉献的完整列表，包括日期、方式、基金和金额。
- **打印奉献报表** -- 生成当前年度或上一年度的可打印奉献报表，用于税务记录。
- **管理付款方式** -- 保存和管理您的付款方式，以便将来更快地奉献。

:::info
奉献通过 **Stripe** 安全处理。您的付款信息绝不会存储在 B1.church 服务器上。
:::
