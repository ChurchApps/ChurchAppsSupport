# 时间线

<div class="article-intro">

**时间线**是 B1.church 会员门户的默认首页。它提供活动动态，显示来自您的小组和教会的最新更新，将所有内容集中在一个便捷的位置。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您必须[登录](../getting-started/logging-in.md)您的 B1.church 账户。

</div>

## 时间线显示的内容

时间线显示帖子和更新的动态，可能包括：

- **小组对话** -- 来自您所属[小组](../groups/)的消息和讨论。
- **任务** -- 分配给您或您所在小组的任务。
- **活动** -- 与您和您的小组相关的即将举行的活动。
- **课程** -- 分享给您的小组的课程和[课程内容](../content/lessons.md)。
- **讲道** -- 教会发布的最新[讲道](../content/sermons.md)。

动态中的每篇帖子显示帖子标题、所属小组（如适用）以及发布时间。

## 查看帖子详情

时间线采用主从布局。您的动态列在页面左侧。点击任意帖子即可在右侧面板中打开其完整详情。在详情视图中，您可以阅读帖子的完整内容并与之互动。

点击详情面板顶部的**返回更新**即可回到动态列表。

## 保持联系

时间线将您所有小组的活动汇集到一个动态中，因此您无需逐一查看每个小组。养成定期查看时间线的习惯，有助于您了解教会的最新动态。

:::tip
如果您的时间线为空，可能是您尚未加入任何小组。访问[小组](../groups/browsing-groups.md)页面查找可加入的小组，或请您的教会管理员或小组负责人将您添加进去。
:::
