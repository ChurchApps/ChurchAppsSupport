# 社区

<div class="article-intro">

B1.church 中的**社区**板块帮助您与教会大家庭保持联系。它整合了会员目录，方便您查找和了解教会成员，同时还有时间线动态，让您随时了解小组和教会的最新动态。

</div>

## 您将看到的内容

- **[会员目录](./member-directory.md)** -- 浏览和搜索教会的会员目录。按姓名查找或按小组筛选来找到您要找的人。点击某位成员即可查看其个人资料详情。
- **[时间线](./timeline.md)** -- 您的活动动态，显示来自小组和教会的最新更新。时间线是您访问会员门户时的默认首页。

## 开始使用

您必须[登录](../getting-started/logging-in.md)才能访问社区功能。如果您尚未登录，当您导航到**社区**选项卡或**时间线**时，系统将提示您登录。

**时间线**选项卡始终显示在侧边栏中，作为会员门户的首页。**社区**选项卡提供会员目录的访问入口。

:::info
您在目录中可以看到的成员和信息取决于教会的隐私设置。您可能无法查看每位成员的完整个人资料。如有疑问，请联系您的教会管理员。
:::

:::tip
结合使用社区功能和[小组](../groups/)功能，在一周中保持与教会的紧密联系。
:::
