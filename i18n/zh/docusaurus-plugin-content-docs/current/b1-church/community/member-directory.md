# 会员目录

<div class="article-intro">

会员目录让您可以浏览和搜索教会中的人员。无论您需要查找某位小组成员还是认识新朋友，目录都能帮助您轻松地与教会大家庭建立联系。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您必须[登录](../getting-started/logging-in.md)您的 B1.church 账户。

</div>

## 访问目录

1. 在[侧边栏导航](../getting-started/navigating.md)中点击**社区**。
2. 目录会加载并在页面左侧显示所有可见的教会成员列表。

## 搜索成员

页面采用主从布局。在左侧，您有两种方式查找人员：

### 按姓名搜索

1. 确保搜索区域顶部选中了**人员**筛选器。
2. 在搜索框中输入姓名。
3. 点击**搜索**或按 **Enter** 键查看结果。

如果您将搜索框留空并点击**搜索**，将显示完整的目录列表。

### 按小组搜索

1. 点击搜索区域顶部的**小组**筛选器。
2. 从下拉列表中选择一个小组。
3. 点击**搜索**查看该小组的所有成员。

:::tip
按小组搜索是一种快速找到您在[小组聚会](../groups/browsing-groups.md)中遇到但记不太清姓名的人的好方法。
:::

## 浏览目录

成员按姓氏字母顺序排列，并按字母标题分组（A、B、C，依此类推）。每个条目显示成员的照片、显示名称和家庭名称，方便您快速浏览列表。

## 查看成员资料

点击列表中的任意成员，即可在右侧详情面板中打开其个人资料。资料显示的成员信息由教会配置，可能包括联系方式、家庭成员和小组成员身份。

您可以点击个人资料中的家庭成员，跳转到他们的资料页面。

:::info
每份资料中可见的信息取决于教会的隐私和目录设置。并非每位成员的所有字段都会显示。
:::
