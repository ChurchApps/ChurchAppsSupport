# B1.church

<div class="article-intro">

B1.church 是教会面向会员的门户。这是您的会众保持联系、访问教会工具和在一周中参与社区活动的地方。每个教会都有自己的网址 `https://yourchurchname.b1.church`。

</div>

当会员访问您的 B1.church 网站时，他们会看到教会的公开网站，包含品牌、导航链接和内容页面。登录后，他们可以访问**会员门户**，所有个人教会工具都集中在一个地方。

## 主要功能

- **时间线** -- 个性化动态，包含来自小组和教会的更新、对话和活动。
- **小组** -- 查看您所属的小组、查看小组成员、参与对话和跟踪出勤。
- **奉献** -- 在线向教会奉献并查看奉献历史。
- **计划** -- 查看即将到来的礼拜计划、您分配的岗位以及访问歌曲详情和礼拜流程。
- **签到** -- 为自己和家庭成员进行礼拜和活动签到。
- **社区目录** -- 浏览教会会员目录、查看个人资料和发送私信。
- **内容** -- 访问讲道、阅读圣经、学习课程和观看直播，全部来自教会网站。
