# 礼拜流程

<div class="article-intro">

打开计划后，**礼拜流程**板块显示该礼拜的项目顺序。这为您和团队提供了礼拜流程的共享视图，包括时间安排、歌曲详情和每个环节的描述。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您必须[登录](../getting-started/logging-in.md)您的 B1.church 账户。
- 从您的[计划列表](./viewing-plans.md)中打开一个计划以查看其礼拜流程。

</div>

## 礼拜流程显示的内容

礼拜流程以列表形式显示各项目，按礼拜中的发生顺序排列。每个项目包括：

- **累计时间** -- 从礼拜开始的运行时间，让您知道每个环节预计何时开始。
- **项目标签和描述** -- 环节的名称（如歌曲标题、祷告或公告）以及任何备注或描述。
- **持续时间** -- 该项目预计需要多长时间。

项目按**标题**分组，将相关环节归在一起（例如"敬拜组曲"或"信息"）。这使您可以轻松浏览礼拜的整体结构。

## 课程和教学内容

如果计划关联了来自 [Lessons.church](../content/lessons.md) 等提供商的课程或教学内容，您将在礼拜流程中看到该内容的预览。这可能包括课程章节、演示材料和团队在礼拜中需要的其他资源。

## 打印礼拜流程

您可以打印格式化的礼拜流程版本，在礼拜当天随身携带。

1. 从**计划**列表中打开计划。
2. 在**礼拜流程**板块中，点击右上角的**打印**图标。
3. 新窗口将打开一个可打印的布局，包括：
   - 顶部的**礼拜日期**。
   - 左侧按类别分组的**团队分配**，显示每个岗位及其负责人。
   - 右侧的**完整礼拜流程**，包含时间、项目详情和持续时间列。
4. 浏览器的打印对话框将自动打开，您可以打印或保存为 PDF。

:::tip
打印的礼拜流程是排练或礼拜当天随手参考的好资料。它在一页上同时包含团队名单和完整的礼拜流程。
:::
