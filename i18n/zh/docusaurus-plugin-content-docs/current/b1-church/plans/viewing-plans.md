# 查看计划

<div class="article-intro">

计划页面清楚地显示了您被分配的所有服务计划。它使用主-详细布局，以便您可以快速浏览计划并深入了解任何一个的详情。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您必须[登录](../getting-started/logging-in.md)到您的B1.church账户。
- 您必须被分配到至少一个服务计划。如果您没有看到任何计划，请与教会管理员或事工领导者联系。

</div>

## 访问您的计划

1. 在[侧边栏导航](../getting-started/navigating.md)中点击**计划**。
2. 您将在页面左侧看到您的计划列表。

## 计划列表

您的计划在左面板上的**我的计划**下列出。每个条目显示：

- **计划名称**（例如"主日敬拜"或"儿童事工"）
- **服务日期**
- 您的**职位**（您填充的角色）
- 您的**状态**——彩色编码，所以您可以一目了然地看到您是否已接受、拒绝或尚未响应

在列表顶部，您还会看到**停工日期**选项。点击它以管理您无法服务的日期。您可以添加、编辑和删除停工日期范围，以便您的团队领导者知道何时安排您。

## 查看计划详情

点击列表中的任何计划以在右面板中打开其详情。详情视图包括：

- **职位详情** ——您分配的职位和需要的时间以及您当前的状态。如果您还没有响应，您可以直接从此视图点击**接受**或**拒绝**。
- **注释** ——计划创建者为团队添加的任何注释。
- **服务顺序** ——完整的服务顺序，包括时间、项目和描述。有关更多详情，请参阅[服务顺序](./service-order.md)。
- **团队分配** ——所有团队成员及其职位的摘要，所以您可以看到与您一起服务的其他人。

:::tip
如果您在不同计划中有多个分配，列表按日期排序，所以您即将到来的承诺首先出现。定期检查以了解最新信息。
:::

## 课程教师视图

如果您的计划包括来自内容提供者（如Lessons.church）的相关课程，您将在**服务顺序**标签中看到一个切换按钮，让您在**摘要**和**教**视图之间切换。

- **摘要视图** ——显示服务顺序、时间和基本课程部分，非常适合快速查看将发生什么
- **教视图** ——打开为教师设计的沉浸式全屏课程演示，包括格式化的说明、嵌入式媒体、注释和逐步指导

教视图将课程内容组织成显示为顶部标签的可导航部分。您可以：

- 点击部分标签直接跳转到课程的该部分
- 滚动浏览课程内容，活动标签会自动更新以匹配您的位置
- 点击媒体项目（视频、图像、活动）以在模态对话框中查看它们
- 查看格式不同的操作类型：**说**出现在对话气泡中，**做**以粗体文本显示，**注释**以突出显示的标注显示

:::tip
教视图在领导教室或小组时特别有帮助，因为它以为移动和平板电脑使用优化的清洁、无干扰格式呈现所有课程内容。
:::
