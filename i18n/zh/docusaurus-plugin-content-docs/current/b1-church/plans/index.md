# 计划

<div class="article-intro">

B1.church 中的**计划**板块让您可以查看分配给您的礼拜计划。无论您是敬拜团队成员、儿童事工帮手还是其他服事岗位，这里都是您了解何时需要服事、您的岗位以及礼拜安排的地方。

</div>

## 您将看到的内容

计划可能包含几种类型的内容，具体取决于教会的设置：

- **您的分配** -- 查看您被分配的岗位及需要到场的时间。每个分配显示岗位名称、礼拜日期和您当前的状态（已接受、已拒绝或未确认）。
- **礼拜流程** -- 查看礼拜流程，包含时间安排、歌曲信息和项目描述，让您提前了解礼拜的流程。
- **课程内容** -- 某些计划包含来自 [Lessons.church](../content/lessons.md) 等提供商的相关课程内容。您可以直接在计划中预览这些内容。
- **屏蔽日期** -- 通过管理屏蔽日期让团队负责人知道您何时无法服事。

## 开始使用

要访问您的计划，请在会员门户的侧边栏导航中点击**计划**。您必须[登录](../getting-started/logging-in.md)才能查看计划——如果您尚未登录，系统将提示您先登录。

:::info
您只能看到分配给您的计划。如果您没有看到任何计划列表，说明您尚未被添加到服事团队。请与教会管理员或事工负责人联系以参与服事。
:::
