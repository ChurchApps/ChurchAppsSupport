# 志愿者报名

<div class="article-intro">

浏览可用的志愿者职位，直接从B1.church为即将进行的服侍报名。职位按类别组织，让您快速找到需要帮助的地方并加入一个团队。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- B1.church账户 -- 您必须登录才能报名
- 您的教会管理员必须至少在一个职位上启用自助报名

</div>

## 浏览可用职位

1. 导航到您教会的B1.church网站上的**志愿者**页面。
2. 您将看到有空缺志愿者职位的即将进行服侍计划的列表。
3. 每个计划显示**计划名称**、**服侍日期**和**服侍时间**。

职位按类别分组（例如，"乐队"、"待客"、"儿童事工"）。每个职位显示：

- **职位名称** -- 您将担任的角色（例如，"鼓手"、"招待员"、"咖啡服务员"）。
- **描述** -- 关于该角色的详情（由管理员提供）。
- **可用名额** -- 一个进度条，显示已填充的位置数量和剩余的位置数量。

## 报名

1. 找到您想志愿服侍的职位。
2. 点击该职位旁边的**报名**按钮。
3. 出现确认信息："您已报名！感谢您的志愿服侍。"
4. 该职位现在显示绿色指示符，确认您已报名。

:::info
您必须登录才能报名。如果您未登录，系统会提示您先登录。
:::

## 取消报名

如果您的计划有变，无法再服侍：

1. 找到您报名的职位 -- 它将显示绿色的"您已报名"指示符。
2. 点击**移除**按钮。
3. 确认您想从该职位中移除自己。
4. 一条信息确认："您已从此职位中被移除。"

## 截止日期

您的教会管理员可能会为每个计划设置报名截止日期。截止日期是服侍日期前的设定小时数。截止日期过后：

- 您无法为新职位报名。
- 您无法移除现有报名。
- 计划顶部会显示一条信息，表示报名截止日期已过。

## 职位容量

每个职位的名额数量有限。当所有名额都已填满时：

- 该职位显示为**已满**。
- 报名按钮被禁用。
- 如果有人取消报名，该名额会重新开放。

:::tip
定期检查志愿者页面，特别是在周初。热门职位很快就会填满，所以早早报名能给您最好的选择。
:::
