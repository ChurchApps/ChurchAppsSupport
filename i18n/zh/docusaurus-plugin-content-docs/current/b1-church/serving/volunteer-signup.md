# 志愿注册

<div class="article-intro">

直接从B1.church浏览可用的志愿职位并为即将到来的服务注册自己。职位按类别组织，所以您可以快速找到需要帮助的地方并加入团队。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 一个B1.church账户——您必须登录才能注册
- 教会管理员必须已在至少一个职位上启用自助注册

</div>

## 浏览可用职位

1. 在教会的B1.church网站上导航到**志愿者**页面。
2. 您将看到有开放志愿职位的即将到来的服务计划列表。
3. 每个计划显示**计划名称**、**服务日期**和**服务时间**。

职位按类别分组（例如"乐队"、"款待"、"儿童事工"）。每个职位显示：

- **职位名称** ——您将填充的角色（例如"鼓手"、"迎接者"、"咖啡服务员"）。
- **描述** ——关于角色的详情，如果由管理员提供。
- **可用位置** ——显示有多少位置已填充以及还剩多少位置的进度条。

## 注册

1. 找到您想要志愿的职位。
2. 点击职位旁的**注册**按钮。
3. 出现确认消息："您已注册！谢谢您的志愿服务。"
4. 该职位现在显示绿色指示符，确认您已注册。

:::info
您必须登录才能注册。如果您未登录，系统会提示您首先登录。
:::

## 删除注册

如果您的计划改变，您不能再服务：

1. 找到您注册的职位——它会显示绿色的"您已注册"指示符。
2. 点击**删除**按钮。
3. 确认您想从职位中删除自己。
4. 一条消息确认："您已从此职位被删除。"

## 截止日期

教会管理员可能为每个计划设置注册截止日期。截止日期是服务日期前的设定小时数。截止日期过后：

- 您不能注册新职位。
- 您不能删除现有注册。
- 计划顶部的消息指示注册截止日期已过。

## 职位容量

每个职位的位置有限。当所有位置都被填满时：

- 职位显示为**已满**。
- 注册按钮被禁用。
- 如果有人删除其注册，位置会再次打开。

:::tip
定期检查志愿者页面，特别是在周早期。受欢迎的职位填满很快，所以早期注册为您提供最佳选择。
:::
