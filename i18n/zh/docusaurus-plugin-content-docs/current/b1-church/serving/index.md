# 服务

<div class="article-intro">

B1.church上的服务部分让您浏览志愿机会并为教会中的职位注册。除了查看您分配的计划和服务顺序，您现在还可以自选需要填充的职位。

</div>

## 您可以做什么

- **浏览志愿机会** ——查看需要志愿者的即将到来的服务计划和开放职位。
- **注册职位** ——选择一个职位并一键为自己注册。
- **查看您的计划** ——查看您即将到来和过去的[服务分配](../plans/viewing-plans)。
- **删除注册** ——如果计划更改，取消自助注册（截止日期前）。

## 如何工作

您的教会管理员创建服务计划并标记某些职位可用于自助注册。这些职位出现在志愿注册页面上，按类别组织（如"乐队"、"款待"或"儿童事工"）。您可以看到每个职位还有多少位置空缺，并为您有时间的职位注册。

:::info
只有教会管理员为自助注册启用的职位才会出现在志愿者页面上。某些职位只能通过直接分配来填充。
:::
