# 服侍

<div class="article-intro">

B1.church上的服侍部分让您浏览志愿者机会并在您的教会报名职位。除了查看您分配的计划和服务顺序外，您现在还可以自选需要填补的职位。

</div>

## 您可以做什么

- **浏览志愿者机会** — 查看需要志愿者的即将推出的服务计划和空缺职位。
- **报名职位** — 选择职位并用一次点击注册自己。
- **查看您的计划** — 查看您即将推出和过去的[服侍分配](../plans/viewing-plans)。
- **删除报名** — 如果您的计划发生变化（在截止日期前），取消自助报名。

## 工作原理

您的教会管理员创建服务计划并标记某些职位可用于自助报名。这些职位出现在志愿者注册页面上，按类别组织（例如"乐队"、"招待"或"儿童事工"）。您可以看到每个职位有多少个空缺位置，并注册您可用的位置。

:::info
只有您的教会管理员为自助报名启用的职位才会出现在志愿者页面上。某些职位可能仅通过直接分配来填补。
:::
