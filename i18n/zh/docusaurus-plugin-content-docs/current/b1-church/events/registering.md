# 为事件注册

<div class="article-intro">

当教会为事件启用注册时，您可以通过一个简单的三步过程进行注册。您可以注册自己并添加家庭成员，无论您是否有账户或作为访客访问。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 教会必须为事件启用事件注册
- 注册窗口必须开放（在管理员设置的开放和关闭日期之间）

</div>

## 第1步：输入您的信息

1. 从教会日历中，打开您想要注册的事件，然后点击**为此事件注册**。
2. 注册页面加载，显示事件标题、描述和日期。
3. 如果您是**已登录**，表单会显示您的名称并确认您在账户下注册。
4. 如果您是**访客**（未登录），请填写：
   - **名字**（必填）
   - **姓氏**（必填）
   - **电子邮件**（必填）
   - **电话**（可选）

:::info
如果事件有容量限制，进度条显示还剩多少位置。如果事件已满，您将无法注册。
:::

## 第2步：添加家庭成员

1. 如果您想注册额外的家庭成员，请为每个人输入其**名字**和**姓氏**。
2. 点击**添加成员**添加更多人员（总共最多10人）。
3. 点击任何成员旁的删除按钮将其从注册中删除。
4. 如果您只注册自己，可以跳过此步骤。

## 第3步：确认

1. 查看事件详情和您的注册信息。
2. 确认您的注册。
3. 出现一条确认消息，显示事件详情和您注册的每个人的列表。

您还将在与账户关联的电子邮件地址（或作为访客提供的电子邮件）收到**确认电子邮件**。

## 取消注册

如果您需要取消注册，请与教会管理员联系。他们可以从B1 Admin注册页面取消您的注册。
