# 活动注册

<div class="article-intro">

当您的教会为活动启用注册时，您可以通过简单的三步过程进行注册。无论您是否有账户或作为客人访问，您都可以注册自己并添加家庭成员。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您的教会必须为该活动启用了活动注册
- 注册窗口必须是开放的（介于管理员设置的开放和关闭日期之间）

</div>

## 步骤1：输入您的信息

1. 从教会日历中，打开您想注册的活动并单击**注册此活动**。
2. 注册页面加载，显示活动标题、描述和日期。
3. 如果您**已登录**，表单显示您的名称并确认您在您的账户下注册。
4. 如果您是**客人**（未登录），请填写：
   - **名字**（必需）
   - **姓氏**（必需）
   - **电子邮件**（必需）
   - **电话**（可选）

:::info
如果活动有容量限制，进度条显示剩余多少个位置。如果活动已满，您将无法注册。
:::

## 步骤2：添加家庭成员

1. 如果您想注册额外的家庭成员，为每个人输入他们的**名字**和**姓氏**。
2. 单击**添加成员**来添加更多人（最多10人）。
3. 单击任何成员旁的移除按钮将他们从注册中移除。
4. 如果您只注册自己，可以跳过此步骤。

## 步骤3：确认

1. 审查活动详细信息和您的注册信息。
2. 确认您的注册。
3. 确认消息出现，显示活动详细信息和您注册的所有人的列表。

您还会收到**确认电子邮件**，邮件地址是与您的账户相关联的电子邮件地址（或您作为客人提供的电子邮件）。

## 取消注册

如果您需要取消注册，请联系您的教会管理员。他们可以从B1 Admin注册页面取消您的注册。
