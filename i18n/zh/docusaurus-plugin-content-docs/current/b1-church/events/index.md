# 事件

<div class="article-intro">

B1.church 上的事件部分允许您浏览即将举行的教会活动并直接从网站进行注册。当您的教会为事件启用注册时，您可以在几个简单的步骤中注册自己和您的家庭成员 -- 对于付费活动，选择参与者类型和选项、应用折扣代码并安全支付作为注册的一部分。

</div>

## 浏览事件

事件显示在教会日历上。当您打开启用注册的事件时，会显示**为此事件注册**按钮以及事件详细信息。

启用注册的事件显示：

- **事件标题和描述** -- 事件是关于什么的。
- **日期和时间** -- 事件何时发生。
- **注册按钮** -- 一个蓝色按钮，将您带到注册表。

## 注册状态

事件可能显示不同的状态，具体取决于时间和容量：

- **开放** -- 注册可用，您可以注册。
- **尚未开放** -- 注册窗口尚未开始。在开放日期后查看。
- **已关闭** -- 注册截止日期已过。
- **已满** -- 所有可用位置已被占用。

:::tip
如果事件已满，当您的教会启用了等候名单时，您可能仍然能够加入**等候名单** -- 如果有位置打开，您将自动收到电子邮件。否则，稍后查看；如果有人取消，可能会打开位置。
:::

## 您需要什么

- B1.church 账户（或者您可以作为访客注册）
- 访问您的教会的 B1.church 网站
