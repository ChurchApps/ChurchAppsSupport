# 活动

<div class="article-intro">

B1.church上的活动部分让您浏览即将举办的教会活动并直接从网站注册。当您的教会为活动启用注册时，您可以在短短几个步骤内注册自己和您的家庭成员。

</div>

## 浏览活动

活动出现在教会日历上。当您打开已启用注册的活动时，会显示**注册此活动**按钮以及活动详细信息。

启用注册的活动显示：

- **活动标题和描述** — 活动的相关内容。
- **日期和时间** — 活动发生的时间。
- **注册按钮** — 一个蓝色按钮，带您进入注册表单。

## 注册状态

根据时间和容量，活动可能显示不同的状态：

- **开放** — 注册可用，您可以注册。
- **尚未开放** — 注册窗口尚未开始。在开放日期后回来检查。
- **已关闭** — 注册截止日期已过。
- **已满** — 所有可用位置已满。

:::tip
如果活动已满，稍后回来检查 — 如果有人取消注册，可能会有位置空出来。
:::

## 您需要什么

- B1.church账户（或您可以作为客人注册）
- 您教会的B1.church网站的访问权限
