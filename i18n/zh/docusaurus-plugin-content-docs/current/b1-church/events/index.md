# 事件

<div class="article-intro">

B1.church上的事件部分让您浏览即将到来的教会事件并直接从网站为它们注册。当教会为事件启用注册时，您可以在几个简单的步骤中为自己和家庭成员注册。

</div>

## 浏览事件

事件出现在教会日历上。当您打开启用了注册的事件时，会在事件详情旁边显示**为此事件注册**按钮。

启用了注册的事件显示：

- **事件标题和描述** ——事件的内容。
- **日期和时间** ——事件何时进行。
- **注册按钮** ——一个蓝色按钮，带您到注册表。

## 注册状态

事件可能根据时间和容量显示不同的状态：

- **开放** ——注册可用，您可以注册。
- **未开放** ——注册窗口尚未开始。在开放日期后检查。
- **已关闭** ——注册截止日期已过。
- **已满** ——所有可用位置都已被占用。

:::tip
如果事件已满，稍后检查——如果有人取消其注册，位置可能会打开。
:::

## 需要什么

- 一个B1.church账户（或您可以作为访客注册）
- 访问教会的B1.church网站
