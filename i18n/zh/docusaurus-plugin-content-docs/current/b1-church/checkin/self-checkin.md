# 自助签到

<div class="article-intro">

自助签到流程让您只需几个步骤即可为自己和家庭成员记录出勤。这是使用教会实体签到终端的快速替代方式。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您必须[登录](../getting-started/logging-in.md)您的 B1.church 账户。
- 您的家庭成员必须关联到您的账户。如果缺少某位家庭成员，请联系您的教会管理员将其添加到您的家庭中。

</div>

## 签到流程

签到遵循简单的三步流程。在[侧边栏](../getting-started/navigating.md)中导航到**签到**选项卡即可开始。

### 第 1 步：选择礼拜

打开签到页面后，您将看到可用礼拜的列表（例如"周日上午"或"周三晚间"）。每张礼拜卡片显示礼拜名称，如果您的教会有多个地点，还会显示校区信息。

点击某场礼拜以选中它并进入下一步。

### 第 2 步：选择家庭成员和小组

选择礼拜后，您将看到家庭成员列表。对于每位出席的人：

1. 点击一位家庭成员以展开其条目。
2. 您将看到所选礼拜的可用**礼拜时段**。
3. 点击礼拜时段旁边的**选择小组**，选择该人应签到的小组（如"主堂敬拜"、"儿童教室 1"或"青年团契"）。
4. 对每位出席的家庭成员重复以上操作。

您不需要为每位家庭成员都签到——只需为实际在场的人展开并分配小组即可。

### 第 3 步：确认

为所有出席人员选择好小组后，点击页面底部的**完成签到**按钮。

您将看到确认屏幕，显示"签到完成！"的消息以及出勤已保存的提示。点击**返回我的页面**即可回到[时间线](../community/timeline.md)。

:::tip
如果您需要在完成签到前更改小组选择，请点击任意礼拜时段旁边的**更改**按钮以选择其他小组。
:::

## 注意事项

- 您只能为关联到您账户的家庭成员签到。如果缺少某位家庭成员，请联系您的教会管理员将其添加到您的家庭中。
- 如果签到页面上没有显示任何礼拜，可能是您的教会目前没有配置可签到的礼拜。

:::warning
签到功能仅在您的教会配置了活跃礼拜时可用。如果您没有看到任何礼拜列表，请与教会办公室确认自助签到是否已启用。
:::
