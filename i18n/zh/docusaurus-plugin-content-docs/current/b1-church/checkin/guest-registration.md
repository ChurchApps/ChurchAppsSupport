# 客人注册

<div class="article-intro">

如果您是第一次访问教会，您可以使用客人注册页面注册自己和您的家人作为客人。这会在教会系统中创建一条记录，以便志愿者可以在您到达时为您签到。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您的教会必须启用了QR客人注册
- 扫描在签到信息亭显示的二维码，或从教会志愿者那里收到链接

</div>

## 客人注册的工作原理

客人注册是为尚未有账户的访客设计的。教会的签到信息亭显示一个二维码，链接到注册页面。您也可以从志愿者或迎宾人员那里收到链接。

## 注册您的家人

1. **扫描签到信息亭的二维码**或在您的手机上打开客人注册链接。
2. 客人注册表单出现，其中包含**主要联系人**的字段：
   - **名字**（必需）
   - **姓氏**（必需）
   - **电子邮件**（可选）
   - **电话**（可选）
3. 要添加**家庭成员**，请按**添加家庭成员**并输入他们的名字和姓氏。您最多可以添加10个家庭成员。
4. 要移除家庭成员，请按他们名字旁的移除按钮。
5. 按**注册**来提交。

## 注册后

一旦您提交表单，确认屏幕出现，显示消息："注册完成 — 您都准备好了！工作人员将很快为您签到。"

您的注册在教会数据库中创建记录：

- 使用主要联系人的姓氏创建一个**家户**。
- 为每个家庭成员创建**人员记录**，具有"客人"成员资格状态。
- 教会志愿者或管理员随后可以使用[B1 Checkin](../../b1-checkin/check-in/looking-up-members)信息亭为您的家人签到。

:::info
如果您的电子邮件与教会数据库中的现有记录匹配，系统会链接到该记录而不是创建重复项。这保持教会的数据清洁。
:::

:::tip
提供您的电子邮件地址可以帮助教会在您访问后跟进您。这是可选的但推荐。
:::
