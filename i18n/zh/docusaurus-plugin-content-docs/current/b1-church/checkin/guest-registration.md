# 访客注册

<div class="article-intro">

如果您是首次访问教会，您可以使用访客注册页面注册自己和您的家庭成员为访客。这会在教会的系统中创建一条记录，以便志愿者可以在您到达时为您签到。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您的教会必须启用了QR访客注册
- 扫描签到亭上显示的二维码，或从教会志愿者处获取链接

</div>

## 访客注册如何工作

访客注册是为还没有账户的访客设计的。教会签到亭上显示一个二维码，链接到注册页面。您也可以从志愿者或迎接者那里获得链接。

## 注册您的家庭

1. **扫描** 签到亭上的二维码或在您的手机上打开访客注册链接。
2. 访客注册表单出现，带有**主要联系人**的字段：
   - **名字**（必填）
   - **姓氏**（必填）
   - **电子邮件**（可选）
   - **电话**（可选）
3. 要添加**家庭成员**，请点击**添加家庭成员**并输入他们的名字和姓氏。您最多可以添加10个家庭成员。
4. 要删除家庭成员，请点击他们名称旁的删除按钮。
5. 点击**注册**进行提交。

## 注册后

提交表单后，出现一个确认屏幕，显示消息："注册完成——您已准备好！工作人员将很快为您签到。"

您的注册在教会数据库中创建了记录：

- 使用主要联系人的姓氏创建**家庭**。
- 为每个家庭成员创建**人员记录**，"访客"成员资格状态。
- 教会志愿者或管理员随后可以使用[B1 Checkin](../../b1-checkin/check-in/looking-up-members)亭为您的家庭签到。

:::info
如果您的电子邮件与教会数据库中的现有记录匹配，系统会链接到该记录，而不是创建重复。这保持教会的数据整洁。
:::

:::tip
提供您的电子邮件地址可以帮助教会在您访问后跟进。这是可选的，但建议提供。
:::
