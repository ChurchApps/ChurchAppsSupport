# 签到

<div class="article-intro">

B1.church 中的**签到**功能允许您直接从会员门户为自己和家庭成员进行教会礼拜签到。这是一种无需前往实体签到终端即可记录出勤的便捷方式。

</div>

## 工作原理

使用自助签到时，您需要选择一场礼拜，选择哪些家庭成员将出席，并为每个人分配每个礼拜时段对应的小组。确认后，您的出勤记录将自动保存。

## 开始使用

要访问签到功能，请在会员门户的[侧边栏导航](../getting-started/navigating.md)中点击**签到**。您必须先[登录](../getting-started/logging-in.md)才能使用此功能——如果您尚未登录，系统将提示您先进行登录。

:::info
签到功能需要您的教会在管理设置中配置礼拜和出勤小组。如果您打开签到页面时没有可用的礼拜，可能是您的教会尚未配置此功能。请联系您的教会管理员获取帮助。
:::
