# 内容

<div class="article-intro">

B1.church 为您的会众提供了一个集中的平台来访问教会分享的内容。根据教会的配置，您可以访问讲道、直播、圣经阅读、每日经文和课程资料——所有这些都可以在会员门户中找到。

</div>

## 可用的内容功能

### 讲道

浏览和观看教会的录制讲道。教会的讲道库按播放列表和系列组织，方便您查找过去的信息。

### 直播

直接在 B1.church 中观看教会的直播。当礼拜正在直播时，您可以加入直播流并通过聊天和祷告功能与其他观众互动。

### 圣经

使用内置的圣经阅读器阅读经文。您可以在不离开网站的情况下浏览书卷、章节和经节。

### 每日经文

查看每天更换的精美圣经经文图片。图片会自动调整以适应您的屏幕。

### 课程

通过嵌入式 Lessons.church 平台访问教会的课程和学习资料。您的登录状态会自动继承，因此您可以直接进入内容。

## 查找内容

内容页面可通过教会的网站导航访问。教会管理员决定启用哪些内容功能以及它们在菜单中的显示方式。如果您没有看到上述某项内容功能，可能是您的教会尚未进行相应配置。

:::info
某些内容功能（如[课程](./lessons.md)）需要您[登录](../getting-started/logging-in.md)，而其他功能（如[讲道](./sermons.md)和[圣经](./bible.md)）则公开可用。
:::
