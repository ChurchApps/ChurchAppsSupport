# 课程教师视图

<div class="article-intro">
教师视图提供了一个扩展的全屏界面,用于查看具有结构化部分的课程内容,使教师和领导者在课堂时间更容易浏览课程材料。
</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您的教会必须在计划中安排了课程。有关详情,请参见 B1 Admin 指南中的[安排课程](../../b1-admin/serving/scheduling-lessons.md)。
- 您需要被分配到计划或有权在 B1.church 网站上查看它。
</div>

## 打开教师视图

要访问课程的扩展教师视图:

1. 导航到 B1.church 上的**计划**部分
2. 打开包含您要教授的课程的计划
3. 在计划项目中找到课程
4. 单击课程查看其详情
5. 单击**教师视图**或**扩展视图**按钮

课程将以针对教学优化的全屏界面打开。

:::tip
教师视图在将课程内容投影到屏幕或电视上供班级观看时,或从平板电脑教学时特别有用。
:::

## 教师视图功能

### 选项卡导航

教师视图使用顶部的选项卡将课程内容组织成部分:

- 课程的每个主要部分都显示为单独的选项卡
- 单击任何选项卡可直接跳转到该部分
- 活动部分在选项卡栏中突出显示
- 选项卡使跳转到课程的不同部分变得容易

### 可滚动内容

主内容区域显示完整的课程材料:

- 自然滚动课程
- 滚动时,活动选项卡自动更新以匹配您正在查看的部分
- 所有格式化文本、图像和说明都清晰显示

### 媒体播放

当课程包含视频、图像或其他媒体时:

- 单击任何媒体项目在弹出播放器中打开它
- 视频在带有标准控件的专用视频播放器中播放
- 图像在灯箱中打开以进行全屏查看
- 关闭媒体播放器返回课程内容

### 可下载资源

如果课程包含可下载文件(活动表、涂色页等):

- 每个资源都显示下载按钮
- 单击可将文件直接下载到您的设备
- 文件可以包括内容创建者提供的 PDF、图像或其他材料

## 在教室中使用教师视图

### 课前

1. 在课程开始前在教师视图中打开课程
2. 审查所有部分并熟悉内容
3. 下载您需要的任何可打印资源
4. 测试任何视频或媒体以确保它们正常播放

### 课堂中

1. 在您的设备上保持教师视图打开或将其投影到屏幕上
2. 使用选项卡在课程进展时在各部分之间跳转
3. 在适当的时候播放视频或显示图像
4. 根据需要滚动浏览说明而不会迷失位置

### 在投影屏幕上

在为班级投影时:

- 全屏布局消除了干扰
- 大而易读的文本使内容从房间对面也能看清
- 单击媒体项目向全班展示视频或图像
- 使用无线鼠标或触控板导航

## 关闭教师视图

要退出教师视图:

- 单击右上角的**关闭**按钮(X)
- 或按键盘上的 **Esc** 键
- 您将返回到计划详情视图

## 移动设备和平板电脑使用

教师视图适用于平板电脑和移动设备:

- **平板电脑** -- 非常适合想要便携式课程指南的教师
- **手机** -- 也支持,尽管较小的屏幕可能不太适合教学
- **横向模式** -- 在移动设备上提供最佳观看体验

## 与常规课程视图的区别

| 功能 | 常规视图 | 教师视图 |
|---------|--------------|--------------|
| **布局** | 紧凑,与计划内联 | 全屏,专用视图 |
| **导航** | 仅滚动 | 选项卡 + 滚动并自动高亮 |
| **媒体** | 嵌入页面 | 弹出播放器用于集中查看 |
| **屏幕空间** | 显示其他计划项目 | 最大化课程内容 |
| **最适合** | 浏览和计划 | 教学和展示 |

## 最佳实践

- **提前准备** -- 在课前在教师视图中审查课程以熟悉布局
- **标记部分** -- 注意哪些选项卡包含关键活动或讨论
- **测试媒体** -- 确保视频和图像在课前正确加载
- **提前下载资源** -- 不要等到课堂时间才下载可打印材料
- **使用更大的屏幕** -- 投影或使用平板电脑获得最佳教学体验

## 故障排除

### 课程无法在教师视图中打开

- 确保课程已在计划中正确安排
- 检查您是否有查看计划的权限
- 刷新页面并重试

### 视频无法播放

- 检查您的互联网连接
- 确保您的设备允许媒体播放
- 尝试在单独的选项卡中打开视频
- 某些内容提供商可能对视频播放有限制

### 移动设备上内容显示被截断

- 将设备旋转到横向模式
- 如需要可稍微缩小
- 考虑使用平板电脑或更大的设备以获得更好的可见性
