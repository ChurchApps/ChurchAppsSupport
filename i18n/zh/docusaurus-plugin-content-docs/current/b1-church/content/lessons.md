# 课程

<div class="article-intro">

**课程**选项卡让您可以访问教会的课程和学习资料。内容通过 Lessons.church 平台提供，并直接嵌入 B1.church 中，因此您无需导航到其他网站。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您必须[登录](../getting-started/logging-in.md)您的 B1.church 账户。课程选项卡位于会员区域，需要身份验证。
- 您的教会必须通过 Lessons.church 配置了课程内容。

</div>

## 访问课程

1. 登录您的 B1.church 会员门户。
2. 在[会员导航](../getting-started/navigating.md)中点击**课程**选项卡。
3. Lessons.church 的内容将直接在页面中加载。

当您打开课程选项卡时，您的登录状态会自动继承。您无需再次单独登录 Lessons.church。

## 您可以做什么

进入课程选项卡后，您可以浏览教会提供的课程内容。根据教会的设置，您可能会看到：

- **课程计划** -- 包含章节和活动的结构化大纲
- **演示和媒体** -- 配合课程的幻灯片、视频和图片
- **可下载资源** -- 用于深入学习的文件和附加材料

您可以点击课程中的各个章节，在对话框窗口中查看其内容。使用**返回**按钮回到章节列表，或点击**关闭**退出对话框。

:::tip
分享到您所在小组的课程也会显示在您的[时间线](../community/timeline.md)上，方便您查找分配的课程内容，而无需直接导航到课程选项卡。
:::

## 可用性

当您的教会通过 Lessons.church 配置了课程内容时，课程选项卡才会出现。如果您在会员导航中没有看到**课程**选项卡，可能是您的教会尚未设置课程内容。请联系教会管理员了解更多信息。
