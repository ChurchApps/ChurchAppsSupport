# 直播

<div class="article-intro">

直播页面让您可以直接在 B1.church 上实时观看教会的礼拜。当教会正在直播时，您可以收看并使用视频旁边的互动功能参与其中。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 观看直播无需登录。直播页面是公开访问的。
- 您的教会必须配置了直播时间表。请查看教会的礼拜时间以了解直播可用的时段。

</div>

## 观看直播

1. 从教会网站导航进入**直播**页面。
2. 如果当前有礼拜正在直播或即将开始，视频播放器将自动加载。
3. 直接在页面上观看直播。

当前没有直播时，您可能会看到离线内容或下一场礼拜时间的信息。

## 互动功能

直播页面可以在视频旁边包含互动选项卡，具体取决于教会的配置。这些可能包括：

- **聊天** -- 在直播期间发送消息并与其他观众互动。聊天通常在礼拜开始前几分钟启用，并在结束后保持一段时间可用。
- **祷告** -- 在礼拜期间提交祷告请求。教会直播团队的主持人可以私下回复。
- **其他选项卡** -- 您的教会可能会添加自定义选项卡，包含公告单、讲道笔记、[奉献页面](../giving/making-donations.md)或其他资源的链接。

点击选项卡图标或标签，在观看时切换不同的可用功能。

## 礼拜前后

教会会设置直播和互动功能可用的具体时间。您可能在礼拜正式开始前就能加入聊天，给您时间与他人打招呼。礼拜结束后，聊天可能会短暂保持开放以便继续交流。

:::info
如果直播页面显示离线内容或倒计时，说明下一场礼拜尚未开始。请在列出的礼拜时间再回来查看。
:::

:::tip
想找您错过的录制讲道？访问[讲道](./sermons.md)页面浏览教会过去的信息。
:::

## 可用性

当您的教会配置了直播时间表时，直播页面才可用。如果您在导航中没有看到**直播**链接，可能是您的教会尚未设置直播。请联系教会管理员了解详情。
