# 查看计划

<div class="article-intro">

B1 Mobile 的计划界面为您提供服事排班的完整视图。它分为两个标签，方便您快速找到下一个分配或回顾服事历史。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 由教会管理员将您分配到至少一个服事计划中

</div>

## 访问您的计划

1. 在底部导航中点击**服事**。
2. 计划界面打开，顶部有两个标签：**即将到来**和**过往**。

## 即将到来标签

**即将到来**标签是您查看下一步安排的首选视图。顶部会看到一张主卡片，突出显示您的下一个分配，包括计划名称、日期和您的岗位。这让您一眼就能知道何时需要您。

在主卡片下方，您会看到一组汇总统计，显示各状态的分配数量：

- **请求中** -- 您已被邀请服事但尚未回复。
- **已确认** -- 您已接受该分配。
- **待处理** -- 该分配正在等待进一步操作。

再往下，一个表格列出了所有即将到来的服事时间及其日期、岗位和状态。

## 过往标签

点击**过往**标签切换到服事历史。此视图按时间倒序显示您之前参与的所有计划。这是跟踪您一段时间内贡献的好方法。

## 查看计划详情

点击任何计划——无论是即将到来的还是过往的——即可打开其完整详情。详情视图包括：

- **礼拜流程** -- 完整的礼拜程序，包含时间和项目详情，让您了解礼拜的流程。
- **您的分配** -- 您的岗位、需要到场的时间和当前状态。
- **团队分配** -- 计划中所有服事人员的汇总，让您了解团队中还有谁。

:::tip
定期查看**即将到来**标签以掌握您的排班。主卡片始终显示您的下一个安排，因此打开应用程序即可立即知道下次何时服事。
:::

:::info
当您被分配到新计划或分配状态发生变化时，您会收到[推送通知](../community/notifications.md)，确保您不会错过任何更新。
:::
