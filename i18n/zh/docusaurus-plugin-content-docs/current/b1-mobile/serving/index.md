# 服事

<div class="article-intro">

B1 Mobile 的**服事**部分让您可以快捷方便地掌握您的服事安排。无论您在敬拜团队、接待组还是儿童事工帮忙，这里都可以查看您的排班时间和角色。

</div>

## 您将找到的内容

- **即将到来的分配** -- 一览所有未来的服事时间，让您始终知道下次何时需要您。
- **过往服事记录** -- 回顾您已完成的服事，跟踪您一段时间内的参与情况。
- **岗位分配** -- 每个分配显示您被安排的岗位及其当前状态——是请求中、已确认还是等待您的回复。

## 开始使用

要查看您的服事排班，请在应用程序底部导航中点击**服事**。您必须登录才能访问您的分配。如果尚未登录，系统会提示您先[登录](../getting-started/logging-in)。

:::info
您只能看到已被分配的计划。如果没有看到任何分配，说明您尚未被添加到服事团队中。请联系教会管理员或事工负责人参与服事。
:::
