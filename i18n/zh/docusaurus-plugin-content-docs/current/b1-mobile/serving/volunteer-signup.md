# 志愿服侍报名

<div class="article-intro">

浏览可用的志愿者职位并直接从 B1 Mobile 注册即将进行的服侍。查看需要填补哪些职位、有多少个开放位置，并仅需轻轻一点就可加入一个团队。

</div>

<div class="prereqs">
<h4>开始前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in) 到您的帐户
- 您的教会管理员必须在至少一个职位上启用自助报名

</div>

## 浏览志愿者机会

1. 从应用导航中打开**志愿者**部分。
2. 出现有开放职位的即将进行的服侍计划列表。
3. 每个计划卡显示：
   - **计划名称**和**服侍日期**
   - **总体进度条**显示填充的总位置
   - **状态** -- 还有多少个开放位置，或如果所有位置都被占据则显示"已满"
   - **职位芯片**显示每个角色及其开放位置数
4. 点击**查看并报名**以查看计划的完整职位列表。

## 报名一个职位

1. 点击一个计划以打开其详情。
2. 职位按类别分组（例如"乐队"、"待客"、"儿童事工"）。
3. 每个职位显示角色名称、描述和可用位置。
4. 点击您想要的职位旁边的**报名**。
5. 出现确认提示："您已报名！感谢您的志愿服侍。"
6. 该职位显示绿色指示符确认您的报名。

:::info
您必须登录才能报名职位。如果需要，应用会提示您登录。
:::

## 移除报名

1. 查找您报名的职位 -- 它显示绿色的"您已报名"指示符。
2. 点击**移除**。
3. 确认对话框询问您是否确定。
4. 确认从职位中移除自己。

## 截止日期和容量

- **截止日期** -- 您的教会可能在服侍前设置截止时间。截止日期后，您无法报名或移除报名。计划顶部的横幅表示截止日期已过。
- **容量** -- 每个职位都有有限数量的位置。当已满时，报名按钮被禁用。如果有人取消，位置会重新打开。

:::tip
启用[推送通知](../community/notifications.md)以了解新的志愿者机会和您分配的变化。
:::
