# 志愿者报名

<div class="article-intro">

直接从B1 Mobile浏览可用的志愿者职位并为即将推出的服务注册。查看需要填补的职位、有多少个位置空开，并通过几次点击加入一个团队。

</div>

<div class="prereqs">
<h4>开始前</h4>

- [安装B1 Mobile](../getting-started/installing)并[登录](../getting-started/logging-in)您的账户
- 您的教会管理员必须在至少一个职位上启用自助报名

</div>

## 浏览志愿者机会

1. 从应用导航中打开**志愿者**部分。
2. 出现带有开放职位的即将推出的服务计划列表。
3. 每个计划卡显示：
   - **计划名称**和**服务日期**
   - **整体进度条**显示已填补的总位置
   - **状态** — 仍然空开的位置数，或如果所有位置都已占用则显示"已满"
   - **职位芯片**显示每个角色及其空开的位置数
4. 点击**查看和报名**来查看计划的职位完整列表。

## 报名职位

1. 点击计划来打开其详细信息。
2. 职位按类别分组（例如，"乐队"、"招待"、"儿童事工"）。
3. 每个职位显示角色名称、描述和可用位置。
4. 点击您想要的职位旁的**报名**。
5. 确认警报出现："您已报名！感谢您的志愿服务。"
6. 职位显示绿色指示器确认您的报名。

:::info
您必须登录才能报名职位。如果需要，应用会提示您登录。
:::

## 移除报名

1. 找到您报名的职位 — 它显示绿色的"您已报名"指示器。
2. 点击**移除**。
3. 确认对话框询问您是否确定。
4. 确认从职位移除自己。

## 截止时间和容量

- **截止时间** — 您的教会可能在服务前设置截止时间。超过截止时间后，您无法报名或移除报名。计划顶部的横幅指示何时已超过截止时间。
- **容量** — 每个职位有有限的位置。满时，报名按钮被禁用。如果有人取消，位置会重新空出。

:::tip
启用[推送通知](../community/notifications.md)以保持了解新的志愿者机会和您分配的更改。
:::
