# B1 Mobile

<div class="article-intro">

B1 Mobile 是 B1.church 的配套应用程序，适用于 iOS 和 Android。它让教会成员可以随时随地访问教会提供的所有工具——小组、奉献、签到、服事计划、成员目录、私信、讲道、圣经、每日经文、课程等。

</div>

该应用程序使用您的 ChurchApps 账户，与您在 B1.church 上使用的账户相同。登录并选择您的教会后，应用程序会自动适配您教会的品牌和配置。如果您属于多个教会，可以随时在它们之间切换。

B1 Mobile 支持推送通知，确保您不会错过教会或小组的任何重要更新。首页仪表盘会根据您的使用习惯进行个性化定制——您最常访问的功能会被提升到顶部，方便快速访问。

## 主要功能

- **小组** -- 查看您的小组，参与对话，与小组成员保持联系。
- **奉献** -- 直接从手机向您的教会进行奉献。
- **签到** -- 为您自己和家庭成员在礼拜和活动中签到。
- **服事计划** -- 查看即将到来的服事计划和您被分配的志愿者岗位。
- **成员目录** -- 浏览教会成员目录并查看成员资料。
- **私信** -- 与其他教会成员收发私信。
- **讲道** -- 观看或收听教会的讲道。
- **圣经** -- 在应用程序内阅读圣经。
- **每日经文** -- 每天以一节经文开始新的一天。
- **课程** -- 访问并学习 Lessons.church 的课程内容。
