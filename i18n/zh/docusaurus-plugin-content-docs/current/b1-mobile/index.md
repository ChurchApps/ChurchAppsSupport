# B1 Mobile（已弃用）

:::danger 已弃用——改为使用 B1.church
iOS 和 Android 的原生 B1 Mobile 应用已被**弃用**且不再被开发。其功能现在通过位于 `https://yourchurchname.b1.church` 的 **B1.church Progressive Web App (PWA)** 提供，它可以添加到任何 iPhone 或 Android 主屏幕，并提供相同的成员门户体验，而无需 App Store 下载。

**如果您是教会成员：**参阅[作为应用安装 (PWA)](../b1-church/getting-started/installing-pwa.md)来在您的手机上设置替代品。

**如果您是教会管理员：**将您的成员指向您的教会网站（`https://yourchurchname.b1.church`），而不是 App Store / Google Play 列表——他们可以从任何页面安装 PWA。

下面的页面仅为仍在运行旧版原生应用的教会保留供参考。
:::

<div class="article-intro">

B1 Mobile *曾是* B1.church 在 iOS 和 Android 上的伴侣原生应用。它为教会成员提供了对小组、捐赠、签到、服侍计划、成员目录、私人消息、讲道、圣经、每日经文、课程等的随时访问。

</div>

该应用使用您的 ChurchApps 账户，与您在 B1.church 上使用的账户相同。登录并选择您的教会后，应用会适应您的教会品牌和配置。如果您属于多个教会，您可以随时在它们之间切换。

B1 Mobile 支持推送通知，因此您不会错过来自您的教会或小组的重要更新。主页仪表板根据您的使用情况个性化——您访问最频繁的功能会被提升到顶部以便快速访问。

## 主要功能

- **小组**——查看您的小组，参与对话，并与小组成员保持联系。
- **捐赠**——直接从您的手机进行教会捐赠。
- **事件**——浏览即将到来的事件并直接从您的手机注册。
- **签到**——为服务和事件签到自己和您的家庭成员。
- **服侍计划**——查看即将到来的服务计划、您的分配志愿职位并注册开放职位。
- **成员目录**——浏览您的教会目录并查看成员资料。
- **消息**——与其他教会成员发送和接收私人消息。
- **讲道**——在应用中观看或收听您的教会的讲道。
- **圣经**——在应用中阅读圣经。
- **每日经文**——以每日经文开始您的一天。
- **课程**（已弃用——被 [FreePlay](/docs/freeplay/) 取代）——用于在应用内嵌入来自 Lessons.church 的课程。
