# 社区

<div class="article-intro">

B1 Mobile 中的**社区**功能帮助您直接通过手机与教会大家庭保持联系。无论您需要查找某位成员、发送快速消息还是查看最新通知，一切只需轻点即可。

</div>

## 您将找到的内容

- **[成员目录](./member-directory)** -- 浏览和搜索教会的成员目录。按姓名查找成员，点击成员可查看其个人资料，包括联系信息和家庭关系。
- **[私信](./messaging)** -- 与其他教会成员收发私信。开始新对话或继续之前的对话。
- **[通知](./notifications)** -- 通过推送通知了解小组动态、消息、服事分配和教会的其他重要更新。

## 开始使用

您必须登录才能访问社区功能。如果尚未登录，当您尝试使用这些功能时，系统会提示您[登录](../getting-started/logging-in)。

成员目录、私信和通知均可通过应用程序的导航栏访问。它们共同为您提供了一种简便的方式来查找人员、私密沟通，以及了解教会的最新动态。

:::info
您在目录中可以看到的成员和信息取决于教会的隐私设置。您可以在[隐私设置](../profile/privacy-settings.md)界面控制自己的可见性。如有疑问，请联系您的教会管理员。
:::
