# 成员目录

<div class="article-intro">

B1 Mobile 中的成员目录让您可以浏览和搜索教会中的成员。无论您需要查找某位小组成员还是寻找周日认识的某个人，目录都能帮助您轻松建立联系。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户

</div>

## 访问目录

1. 在应用程序导航中点击**目录**选项。
2. 目录会加载教会成员的按字母排列的列表。

## 浏览目录

成员按姓氏字母顺序排列，并按字母标题（A、B、C 等）分组。您可以滚动浏览列表来查看教会大家庭。每个条目显示成员的姓名和照片，方便您找到想要查找的人。

## 搜索成员

如果您知道要查找的人，目录顶部的搜索栏是最快的方式。

1. 点击屏幕顶部的**搜索栏**。
2. 输入姓名。
3. 列表会实时筛选，显示匹配的结果。

:::tip
这在大型教会中特别有用，因为滚动完整列表可能需要较长时间。只需输入姓名的前几个字母即可快速缩小结果范围。
:::

## 查看成员资料

点击列表中的任何成员即可打开其个人资料。资料页面可能包含：

- **联系信息** -- 电话号码、电子邮件地址以及教会提供的其他详细信息。
- **家庭关系** -- 与此人资料关联的[家庭成员](../profile/household.md)。您可以点击某位家庭成员跳转到其个人资料。

:::info
每个资料中可见的信息取决于教会的隐私和目录设置。并非所有字段都会为每位成员显示。成员可以在[隐私设置](../profile/privacy-settings.md)中控制自己的可见性。如有疑问，请联系您的教会管理员。
:::
