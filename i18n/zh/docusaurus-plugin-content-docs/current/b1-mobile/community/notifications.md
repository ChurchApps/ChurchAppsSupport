# 通知

<div class="article-intro">

B1 Mobile 使用推送通知来让您了解您的教会正在发生的事情。您无需打开应用来检查更新，重要信息会自动发送给您。

</div>

<div class="prereqs">
<h4>开始前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 在您的设备设置中允许 B1 Mobile 的通知

</div>

## 您将收到什么通知

通知可以涵盖一系列活动，包括：

- **消息**——当某人给您发送[私人消息](./messaging)时。
- **小组活动**——来自您所属[小组](../groups/)的更新、帖子和对话。
- **服侍分配**——当您被分配到一个[计划](../serving/viewing-plans.md)或您的分配状态改变时。
- **其他更新**——与您相关的一般公告和活动。

## 查看您的通知

1. 点击应用右上角的**铃声图标**，或在导航中点击 **Notifications** 选项。
2. 通知屏幕在选项卡视图中显示您所有最近的通知，组织得让您可以快速找到您正在寻找的内容。

每条通知都包括更新的简短摘要。点击任何通知来直接转到相关内容——例如，点击消息通知打开对话，点击服侍通知带您到计划。

## 移除通知

您可以清除已阅读的通知，以使您的列表专注于新内容。

- **删除单个通知**——点击任何通知行右侧的垃圾箱图标来仅移除该项。
- **清除所有通知**——点击通知列表顶部的 **Clear All** 来一次性移除每条通知。您会被要求确认再清除所有内容。

移除通知仅影响您的视图；它不会更改基础活动（消息、计划或小组帖子仍然存在）。如果您在多个设备上启用了通知，在一个设备上清除将同步到其他设备。

## 推送通知

B1 Mobile 向您的设备发送推送通知，以便您即使在应用未打开时也能收到更新。要接收推送通知，请确保您在设备设置中允许了 B1 Mobile 的通知。

### 启用推送通知

- **iOS**——转到 **Settings** > **Notifications** > **B1 Mobile** 并打开 **Allow Notifications**。
- **Android**——转到 **Settings** > **Apps** > **B1 Mobile** > **Notifications** 并确保通知已启用。

:::warning
如果您没有收到通知，请检查您的设备设置以确保它们已启用。另外确认您已登录到应用，因为通知与您的账户相关联。
:::

:::tip
您可以在设备级别管理通知偏好。如果您发现通知过于频繁，请调整您的设备设置来控制警报的显示方式和时间。
:::
