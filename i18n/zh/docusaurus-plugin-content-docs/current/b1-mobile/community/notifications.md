# 通知

<div class="article-intro">

B1 Mobile 使用推送通知让您了解教会的最新动态。您无需打开应用程序检查更新，重要信息会自动推送给您。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 在设备设置中允许 B1 Mobile 发送通知

</div>

## 您将收到的通知

通知可以涵盖各种活动，包括：

- **消息** -- 当有人给您发送[私信](./messaging)时。
- **小组动态** -- 您所在[小组](../groups/)的更新、帖子和对话。
- **服事分配** -- 当您被分配到某个[服事计划](../serving/viewing-plans.md)或您的分配状态发生变化时。
- **其他更新** -- 与您相关的一般公告和活动。

## 查看您的通知

1. 点击应用程序右上角的**铃铛图标**，或在导航中点击**通知**选项。
2. 通知界面以标签视图显示您所有的最近通知，便于您快速找到所需内容。

每条通知包含更新的简要摘要。点击任何通知即可直接跳转到相关内容——例如，点击消息通知会打开对话，点击服事通知会带您进入服事计划。

## 推送通知

B1 Mobile 会向您的设备发送推送通知，即使应用程序未打开也能收到更新。要接收推送通知，请确保在设备设置中允许了 B1 Mobile 的通知。

### 启用推送通知

- **iOS** -- 前往**设置** > **通知** > **B1 Mobile**，开启**允许通知**。
- **Android** -- 前往**设置** > **应用** > **B1 Mobile** > **通知**，确保通知已启用。

:::warning
如果您没有收到通知，请检查设备设置确保通知已启用。同时确认您已登录应用程序，因为通知与您的账户绑定。
:::

:::tip
您可以在设备层面管理通知偏好设置。如果您觉得通知太频繁，可以调整设备设置来控制提醒的方式和时间。
:::
