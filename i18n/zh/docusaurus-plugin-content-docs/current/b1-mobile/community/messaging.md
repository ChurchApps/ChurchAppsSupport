# 消息

<div class="article-intro">

B1 Mobile 包括私人消息，因此您可以直接从应用与其他教会成员进行一对一的对话。这是一种简单的方式来保持联系，而无需共享个人电话号码或电子邮件地址。

</div>

<div class="prereqs">
<h4>开始前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户

</div>

## 访问您的消息

1. 在应用导航中点击 **Messaging**。
2. 您以前的对话会被列出以便快速访问。

## 开始新对话

1. 从消息屏幕中，点击选项来开始新对话。
2. 使用搜索字段来查找您想给其消息的成员。您也可以通过[成员目录](./member-directory)找到成员。
3. 从搜索结果中选择成员。
4. 输入您的消息并点击 **Send**。

您的新对话现在将出现在您的对话列表中，以便您稍后可以轻松地继续。

## 阅读和发送消息

当您打开对话时，您会看到完整的消息线程。每条消息都显示发送者的头像和名字，使得跟踪很容易。要回复，在屏幕底部的文本字段中输入您的消息并点击 **Send**。

## 编辑消息

如果您需要在发送消息后纠正什么，您可以编辑它。点击您想更改的消息，选择编辑选项，进行您的更正并保存。更新的消息将替换对话中的原始消息。

## 您的对话列表

您所有的活跃对话都显示在主消息屏幕上的列表中。列表显示另一个人的名字和头像，使其很容易找到您正在寻找的对话。点击任何对话来打开它并继续您离开的地方。

## 删除对话

如果您想从列表中移除对话，点击对话行右侧的垃圾箱图标并确认。对话从您的视图中被移除；另一个人仍然可以在他们这边看到他们的线程副本。要稍后再与同一个人交谈，只需开始新对话。

:::tip
使用消息来协调与志愿者同伴、跟进您在教会中认识的某人，或只是检查朋友。它将您的教会通信保存在一个便捷的地方。
:::

:::info
启用[推送通知](./notifications)来在某人给您发送新消息时收到警报，即使应用未打开。
:::
