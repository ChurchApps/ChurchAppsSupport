# 私信

<div class="article-intro">

B1 Mobile 包含私信功能，让您可以直接在应用程序中与其他教会成员进行一对一对话。这是一种保持联系的简便方式，无需分享个人电话号码或电子邮件地址。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户

</div>

## 访问您的消息

1. 在应用程序导航中点击**私信**。
2. 您之前的对话会列出以便快速访问。

## 开始新对话

1. 在私信界面，点击开始新对话的选项。
2. 使用搜索栏查找您想要发送消息的成员。您也可以通过[成员目录](./member-directory)查找成员。
3. 从搜索结果中选择该成员。
4. 输入您的消息并点击**发送**。

您的新对话将出现在对话列表中，方便您稍后继续。

## 阅读和发送消息

打开对话后，您将看到完整的消息记录。每条消息都显示发送者的头像和姓名，方便您跟踪对话。要回复，请在屏幕底部的文本框中输入消息并点击**发送**。

## 编辑消息

如果您需要在发送后修改某条消息，可以对其进行编辑。点击要修改的消息，选择编辑选项，进行更正并保存。更新后的消息将替换对话中的原始消息。

## 您的对话列表

所有活跃的对话都显示在私信主界面的列表中。列表显示对方的姓名和头像，方便您找到要查找的对话。点击任何对话即可打开它并从上次中断的地方继续。

:::tip
使用私信功能与志愿者同伴协调、跟进在教会认识的人，或者只是问候朋友。它将您的教会通讯集中在一个便利的地方。
:::

:::info
启用[推送通知](./notifications)即可在有人给您发送新消息时收到提醒，即使应用程序未打开也能收到。
:::
