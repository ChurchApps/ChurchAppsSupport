# 入门

:::danger B1 Mobile 已弃用
原生 B1 Mobile 应用已被 [B1.church Progressive Web App (PWA)](../../b1-church/getting-started/installing-pwa.md) 取代。新用户应该从 `https://yourchurchname.b1.church` 上的任何页面安装 PWA，而不是从 App Store 或 Google Play 下载 B1 Mobile。
:::

B1 Mobile 入门只需几分钟。下载应用，使用您的 ChurchApps 账户登录，选择您的教会，您就可以开始了。

</div>

## 快速开始

1. **下载**应用来自 [App Store](https://apps.apple.com)（iOS）或 [Google Play Store](https://play.google.com)（Android）。搜索"B1.church"。
2. **登录**使用您现有的 ChurchApps 账户——与您在 B1.church 上使用的相同电子邮件和密码。如果您还没有账户，点击 **Register** 来创建一个。
3. **选择您的教会**通过按名称搜索。输入至少 3 个字符来查看结果。
4. **探索仪表板**——您的个性化主屏幕显示您的教会提供的功能的快速链接。

就这样。选择您的教会后，应用会加载您的教会品牌、导航链接和内容。您可以开始使用小组、捐赠、签到、和您的教会启用的其他所有功能。
