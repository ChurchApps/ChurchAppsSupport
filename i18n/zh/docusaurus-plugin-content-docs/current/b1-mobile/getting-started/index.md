# 快速入门

<div class="article-intro">

开始使用 B1 Mobile 只需几分钟。下载应用程序，使用您的 ChurchApps 账户登录，选择您的教会，即可开始使用。

</div>

## 快速开始

1. 从 [App Store](https://apps.apple.com)（iOS）或 [Google Play Store](https://play.google.com)（Android）**下载**应用程序。搜索"B1.church"。
2. 使用您现有的 ChurchApps 账户**登录**——与您在 B1.church 上使用的电子邮件和密码相同。如果您还没有账户，请点击**注册**创建一个。
3. 通过搜索教会名称**选择您的教会**。输入至少3个字符即可查看结果。
4. **浏览仪表盘**——您的个性化主页显示了教会提供的各项功能的快捷链接。

就是这样。选择教会后，应用程序会加载您教会的品牌、导航链接和内容。您可以开始使用教会已启用的小组、奉献、签到等所有功能。
