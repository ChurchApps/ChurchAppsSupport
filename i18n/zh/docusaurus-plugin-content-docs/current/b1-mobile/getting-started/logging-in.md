# 登录

<div class="article-intro">

B1 Mobile 使用您在 B1.church 和其他 ChurchApps 产品上使用的相同 ChurchApps 账户。如果您已经拥有账户，您可以使用您现有的电子邮件和密码登录。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 在您的 iOS 或 Android 设备上[安装 B1 Mobile](./installing)

</div>

## 如何登录

1. 打开 B1 Mobile 应用。
2. 从抽屉菜单或登录提示中点击 **Login** 选项。
3. 输入您的**电子邮件地址**和**密码**。
4. 点击 **Sign In**。

登录后，您将被带到[仪表板](../dashboard/)。如果您还没有选择教会，您会被提示搜索并选择一个。

## 创建新账户

如果您还没有 ChurchApps 账户，您可以直接在应用中创建一个：

1. 在登录屏幕上，点击底部的 **Register** 链接。
2. 输入您的**名字**、**姓氏**和**电子邮件地址**。
3. 表单会根据您的信息自动预填充您的教会（如果找到匹配）。
4. 点击 **Register**。
5. 查看您的电子邮件中包含临时密码的邮件。
6. 返回应用，点击 **Login**，并使用您的电子邮件和电子邮件中的临时密码登录。

首次登录后，您可以从您的[资料设置](../profile/editing-profile.md)更改您的密码。

:::info
如果您尝试使用尚未注册的电子邮件登录，您将自动被定向到注册屏幕，以便您可以创建账户。
:::

:::warning
如果您在几分钟内没有收到注册电子邮件，请务必检查您的垃圾邮件或垃圾文件夹。
:::

## 忘记密码

如果您忘记了您的密码，在登录屏幕上点击 **Forgot Password?** 并输入您的电子邮件地址。您将收到一封包含**6 位验证码**的电子邮件，该代码的有效期为 15 分钟。在验证屏幕上输入代码来登录，然后从您的[资料设置](../profile/editing-profile.md)更新您的密码。

:::warning
五次错误的代码尝试后，代码被锁定。如果发生这种情况，返回"忘记密码"屏幕并请求一个新代码。
:::

## 保持登录

B1 Mobile 在会话之间保持您登录。您不需要每次打开应用时都登录。如果您因任何原因被登出，只需使用您的电子邮件和密码再次登录。

## 登出

要从应用中登出：

1. 通过点击左上角的**汉堡菜单**（三条水平线）来打开抽屉菜单。
2. 向下滚动抽屉菜单。
3. 点击 **Sign Out**。

您将被登出，可以随时使用您的凭证再次登录。
