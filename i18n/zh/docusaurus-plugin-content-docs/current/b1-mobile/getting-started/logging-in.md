# 登录

<div class="article-intro">

B1 Mobile 使用您在 B1.church 和其他 ChurchApps 产品上使用的同一个 ChurchApps 账户。如果您已有账户，可以使用现有的电子邮件和密码登录。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在您的 iOS 或 Android 设备上[安装 B1 Mobile](./installing)

</div>

## 如何登录

1. 打开 B1 Mobile 应用程序。
2. 从侧边栏菜单或登录提示中点击**登录**选项。
3. 输入您的**电子邮件地址**和**密码**。
4. 点击**登录**。

登录后，您将进入[仪表盘](../dashboard/)。如果您尚未选择教会，系统会提示您搜索并选择一个教会。

## 创建新账户

如果您还没有 ChurchApps 账户，可以直接在应用程序中创建一个：

1. 在登录界面，点击底部的**注册**链接。
2. 输入您的**名字**、**姓氏**和**电子邮件地址**。
3. 点击**注册**。
4. 查收电子邮件，其中包含您的临时密码。
5. 返回应用程序，点击**登录**，使用您的电子邮件和邮件中的临时密码登录。

首次登录后，您可以在[个人资料设置](../profile/editing-profile.md)中更改密码。

:::warning
如果几分钟内未收到注册邮件，请检查您的垃圾邮件或垃圾箱文件夹。
:::

## 忘记密码

如果您忘记了密码，请在登录界面点击**忘记密码？**。这将在浏览器中打开一个页面，您可以在其中输入电子邮件地址并请求密码重置链接。

## 保持登录状态

B1 Mobile 会在会话之间保持您的登录状态。您不需要每次打开应用程序时都登录。如果因任何原因被登出，只需使用您的电子邮件和密码重新登录即可。

## 退出登录

要退出应用程序：

1. 点击左上角的**菜单图标**（三条横线）打开侧边栏菜单。
2. 滚动到侧边栏菜单底部。
3. 点击**退出登录**。

您将被登出，随时可以使用您的凭据重新登录。
