# 切换教会

<div class="article-intro">

如果您是多个教会的成员，B1 Mobile 可以轻松地在它们之间切换。您的账户适用于所有使用 ChurchApps 的教会，因此您只需一个登录凭据。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](./installing) 并[登录](./logging-in)您的账户

</div>

## 如何切换教会

1. 点击左上角的**菜单图标**（三条横线）打开侧边栏菜单。
2. 在侧边栏顶部，点击显示当前教会名称的按钮（如果未选择教会则显示**查找教会**）。
3. 出现**教会搜索**界面。您最近访问的教会显示在顶部，方便快速访问。
4. 点击最近的教会进行切换，或在搜索栏中输入以查找其他教会。

切换教会后，应用程序会重新加载该教会的品牌、导航链接和内容。您的[仪表盘](../dashboard/)、[小组](../groups/)和其他数据将反映新选择的教会。

## 搜索教会

在教会搜索界面，在搜索栏中输入至少3个字符的教会名称。匹配的教会会出现在下方列表中。点击一个教会即可选择它。

:::info
如果没有出现结果，请尝试不同的拼写或教会名称的较短部分。搜索会匹配在 ChurchApps 中注册的教会名称。
:::

## 最近访问的教会

教会搜索界面会记住您之前访问过的教会。当您没有输入搜索查询时，最近访问的教会会自动显示。这使得切换回您常用的教会变得非常快捷。

:::tip
切换教会不需要退出登录再重新登录。您的单一 ChurchApps 账户适用于您的所有教会。只需从教会搜索界面选择不同的教会即可。
:::
