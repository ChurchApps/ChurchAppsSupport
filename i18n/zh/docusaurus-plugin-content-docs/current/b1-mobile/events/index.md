# 活动

<div class="article-intro">

B1 Mobile 中的活动功能让您可以浏览即将举行的教会活动，并直接从手机注册。当您的教会为某个活动启用注册功能时，您可以注册自己和家庭成员，只需轻轻点击几下。

</div>

## 查找有注册的活动

启用了注册的活动会出现在您的团体日历中。当您点击一个有可用注册的活动时，您会在活动详情中看到一个**注册**按钮。

活动可能显示不同的状态：

- **开放** -- 注册可用，您可以报名。
- **尚未开放** -- 注册窗口尚未开始。
- **已关闭** -- 注册截止日期已过。
- **已满** -- 所有可用位置已被填满。

## 您需要什么

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)
- 您的教会必须为该活动启用活动注册
