# 活动

<div class="article-intro">

B1 Mobile中的活动功能让您浏览即将举办的教会活动并直接从您的手机注册。当您的教会为活动启用注册时，您可以通过几次点击为自己和您的家庭成员注册。

</div>

## 查找启用了注册的活动

启用了注册的活动出现在您的小组日历中。当您点击有可用注册的活动时，您会在活动详细信息中看到**注册**按钮。

活动可能显示不同的状态：

- **开放** — 注册可用，您可以注册。
- **尚未开放** — 注册窗口尚未开始。
- **已关闭** — 注册截止日期已过。
- **已满** — 所有可用位置已满。

## 您需要什么

- [B1 Mobile已安装](../getting-started/installing)并[已登录](../getting-started/logging-in)
- 您的教会必须为该活动启用了活动注册
