# 活动注册

<div class="article-intro">

使用简单的三步过程直接从B1 Mobile注册教会活动。您可以注册自己并从您的手机添加家庭成员。

</div>

<div class="prereqs">
<h4>开始前</h4>

- [安装B1 Mobile](../getting-started/installing)并[登录](../getting-started/logging-in)您的账户
- 活动必须启用了注册，且注册窗口必须是开放的

</div>

## 步骤1：输入您的信息

1. 打开启用了注册的小组日历活动，然后点击**注册**。
2. 注册屏幕显示活动标题、描述和日期。
3. 由于您已登录，表单确认您在您的账户下注册。

:::info
如果活动有容量限制，进度条显示剩余多少个位置。如果活动已满，您将无法注册。
:::

## 步骤2：添加家庭成员

1. 要为额外的家庭成员注册，为每个人输入**名字**和**姓氏**。
2. 点击**添加成员**来添加更多人（最多10人）。
3. 点击任何成员旁的移除按钮将他们从注册中移除。
4. 如果您只注册自己，请跳过此步骤。

## 步骤3：确认

1. 审查活动详细信息和您的注册信息。
2. 确认您的注册。
3. 确认消息出现，显示活动详细信息和您注册的所有人。

您还会收到一封**确认电子邮件**，地址是与您的账户相关联的电子邮件。

## 取消注册

如果您需要取消，请联系您的教会管理员。他们可以从B1 Admin注册页面取消您的注册。
