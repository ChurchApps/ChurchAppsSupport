# 活动注册

<div class="article-intro">

直接从 B1 Mobile 通过简单的三步流程注册教会活动。您可以注册自己并从手机上添加家庭成员。

</div>

<div class="prereqs">
<h4>开始前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in) 到您的帐户
- 活动必须启用注册且注册窗口必须是开放的

</div>

## 第 1 步：输入您的信息

1. 打开已启用注册的团体日历活动，然后点击**注册**。
2. 注册屏幕显示活动标题、描述和日期。
3. 由于您已登录，表格确认您在您的帐户下注册。

:::info
如果活动有容量限制，进度条会显示还有多少个位置。如果活动已满，您将无法注册。
:::

## 第 2 步：添加家庭成员

1. 要注册其他家庭成员，请为每个人输入**名**和**姓**。
2. 点击**添加成员**以添加更多人（最多 10 人）。
3. 点击任何成员旁边的删除按钮将其从注册中删除。
4. 如果您只注册自己，请跳过此步骤。

## 第 3 步：确认

1. 查看活动详情和您的注册信息。
2. 确认您的注册。
3. 出现确认消息，其中包含活动详情和您注册的所有人。

您还将收到一封**确认电子邮件**至与您的帐户关联的电子邮件地址。

## 取消注册

如果您需要取消，请联系您的教会管理员。他们可以从 B1 Admin 注册页面取消您的注册。
