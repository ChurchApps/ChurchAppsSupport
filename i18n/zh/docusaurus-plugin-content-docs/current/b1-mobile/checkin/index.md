# 签到

<div class="article-intro">

B1 Mobile 让您可以直接通过手机为自己和家庭成员办理教会礼拜签到。无需在自助签到机前排队，在您走进教会大门之前就能完成出席记录。

</div>

## 工作原理

签到流程是一个快速的多步骤操作，引导您选择礼拜、选择出席人员，并为每个人分配小组。确认后，您的出席记录会自动生成。

## 开始使用

要使用签到功能，请在 B1 Mobile 中打开**签到**部分。您必须登录才能使用此功能。如果尚未登录，系统会提示您先[登录](../getting-started/logging-in)。

:::info
签到功能要求您的教会在管理设置中配置了礼拜和出席小组。如果打开签到界面时没有显示任何礼拜，可能是您的教会尚未配置此功能。请联系您的教会管理员寻求帮助。
:::

## 您可以做什么

- **为家庭成员签到** -- 选择哪些家庭成员出席，并一次性记录所有人的出席情况。参见[家庭成员](../profile/household.md)了解如何管理您的家庭成员。
- **选择礼拜和时间** -- 从教会配置的可用礼拜和时段中选择。
- **分配小组** -- 将每个人分配到礼拜中的相应小组，例如主堂敬拜、儿童教室或青年小组。
