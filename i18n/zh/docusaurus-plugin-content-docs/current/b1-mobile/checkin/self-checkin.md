# 自助签到

<div class="article-intro">

通过 B1 Mobile 签到是一个简单的多步骤流程。只需轻点几下，即可为您自己和家庭成员记录出席情况，无需在自助签到机前排队等候。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 确保您的[家庭成员](../profile/household.md)已在个人资料中设置

</div>

## 签到流程

### 第一步：选择礼拜和时间

打开签到界面后，您将看到可用的礼拜和时间列表。这些由您的教会配置，可能包括"周日上午 9:00"或"周三晚上 6:30"等选项。

1. 查看可用的礼拜选项。
2. 点击您要参加的礼拜和时间。

### 第二步：选择家庭成员

选择礼拜后，您将看到家庭成员列表。

1. 查看家庭成员列表。
2. 选择将要参加礼拜的每个人。

您不需要为所有人签到——只需选择实际出席的人。

### 第三步：分配小组

对于您选择的每个人，您需要为他们分配礼拜中的小组。小组可能包括主堂敬拜、儿童教室1、儿童教室2或青年小组等选项。

1. 为每位选定的家庭成员，点击选择相应的**小组**。
2. 确保每个人被分配到与其年龄或角色对应的正确小组。

### 第四步：确认

为所有出席人员分配好小组后，完成签到。

1. 检查您的选择，确保一切正确。
2. 确认签到。
3. 您将看到确认界面，通知您出席情况已被记录。

## 注意事项

- 您只能为关联到您账户的家庭成员签到。如果列表中缺少某人，请联系您的教会管理员将其添加到您的[家庭](../profile/household.md)中。
- 如果签到界面没有显示任何礼拜，说明您的教会目前可能未配置签到礼拜。
- 如果需要在确认前更改选择，您可以返回上一步调整。

:::warning
请在确认前确保选择了正确的礼拜时间和小组分配。出席记录一旦生成，可能需要由教会管理员进行更改。
:::

:::tip
在去教会的路上打开签到界面，这样到达后就能立即签到。整个过程只需几秒钟。
:::
