# 内容

<div class="article-intro">

B1 Mobile 为您提供了一个集中的地方，直接从手机访问教会分享的所有内容。根据教会的配置，您可以访问讲道、直播、圣经阅读、每日经文和课程材料——所有这些都无需离开应用程序。

</div>

## 可用的内容功能

### 讲道

浏览和观看教会录制的讲道。讲道按播放列表和系列组织，方便您查找过往信息或跟进当前系列。

### 直播

直接在应用程序中观看教会的直播。当礼拜正在直播时，您可以随时随地收看。

### 圣经

使用内置的圣经阅读器阅读经文。在书卷、章节和经文之间导航，无需切换应用程序。

### 每日经文

查看每天更新的精美圣经经文图片。图片会自动调整以适应您设备的屏幕方向。

### 课程

通过嵌入的 Lessons.church 平台访问教会的课程和学习材料。直接在 B1 Mobile 中浏览和查看已安排的课程。

## 查找内容

内容功能可通过应用程序的侧边栏菜单访问。您的教会管理员决定启用哪些内容功能，因此您可能看不到上面列出的所有选项。如果缺少某个功能，可能是教会尚未配置。

:::info
内容功能由您的教会管理员通过 [B1 Admin](../../b1-admin/) 配置。如果您没有看到预期的功能，请联系教会办公室。
:::
