# 直播

<div class="article-intro">

B1 Mobile 让您可以直接从手机观看教会的直播。当礼拜正在播出时，您可以随时随地实时收看。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 您的教会必须配置了直播计划

</div>

## 观看直播

1. 从侧边栏菜单打开**直播**界面，或当直播即将开始时点击[讲道](./sermons)界面上的倒计时。
2. 如果礼拜正在直播，视频播放器会自动加载并开始播放。
3. 直接在应用程序中观看直播。

当没有正在进行的直播时，界面可能会显示离线内容或下一场预定礼拜的信息。

## 进入直播

有两种方式可以访问直播：

- **侧边栏菜单** -- 打开侧边栏菜单，点击**直播**即可直接进入直播界面。
- **讲道界面** -- 当直播预定在未来24小时内开始时，[讲道](./sermons)界面上会出现倒计时。直播开始后，您可以从那里导航到直播。

## 礼拜前后

您的教会会设置直播可用的具体时间。在礼拜开始前，您可能会看到倒计时或离线消息。直播结束后，直播界面会恢复到离线状态。

:::info
如果直播界面显示倒计时或离线消息，说明下一场预定的礼拜尚未开始。请在列出的礼拜时间回来查看。
:::

## 可用性

当您的教会配置了直播计划时，直播界面即可使用。如果您在应用程序的侧边栏菜单中没有看到**直播**选项，可能是教会尚未设置直播。请联系教会管理员了解详情。

:::tip
查看[讲道](./sermons)界面上的倒计时计时器，了解下一场直播何时开始。
:::
