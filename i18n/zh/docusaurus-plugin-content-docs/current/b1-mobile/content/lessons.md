# 课程

<div class="article-intro">

课程界面让您可以访问教会的课程和学习材料。内容通过 Lessons.church 平台提供，并直接嵌入在 B1 Mobile 中，方便您无需切换到其他应用程序即可浏览和查看课程。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 您的教会必须通过 Lessons.church 配置了课程内容

</div>

## 访问课程

1. 从侧边栏菜单打开**课程**界面。
2. Lessons.church 的内容会直接在应用程序中加载。
3. 浏览教会提供的课程内容。

您的教会会配置此处显示哪些课程和项目。您将看到教会团队已安排或分配的内容。

## 您可以做什么

进入课程界面后，您可以浏览和查看教会设置的材料。根据教会的配置，您可能会找到：

- **课程计划** -- 包含各部分和活动的结构化大纲
- **演示文稿和媒体** -- 配合课程的幻灯片、视频和图片
- **可下载资源** -- 用于进一步学习的文件和材料

点击课程中的各个部分查看其内容。使用导航控件在各部分之间切换或返回课程列表。

:::info
Lessons.church 是一个独立的 ChurchApps 平台，与 B1 Mobile 集成。您的教会管理员通过 Lessons.church 门户管理课程内容。
:::

## 可用性

当您的教会通过 Lessons.church 配置了课程内容时，课程界面才会显示。如果您在侧边栏菜单中没有看到**课程**选项，可能是教会尚未设置课程内容。请联系教会管理员了解更多信息。
