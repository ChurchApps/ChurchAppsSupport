# 讲道

<div class="article-intro">

讲道界面让您可以浏览和观看教会录制的讲道。这是您回顾错过的信息或重温触动您的讲道的首选之处。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 您的教会必须在管理门户中配置了讲道内容

</div>

## 您将看到的内容

讲道界面以标签视图组织，包含两个主要部分：

- **播放列表标签** -- 显示教会的讲道播放列表（系列），按日期排序。每个播放列表将相关讲道分组在一起，方便您从头到尾跟进多周系列。
- **最近标签** -- 按时间倒序显示各个讲道列表，最新的信息始终在最上方。

在屏幕顶部，最新的讲道以特色卡片的形式显示，让您可以快速访问最新的信息。

## 直播倒计时

当[直播](./live-streaming)即将在未来24小时内开始时，讲道界面上会出现倒计时计时器。这让您知道下一场礼拜将于何时播出，以便您计划观看直播。

## 观看讲道

1. 从侧边栏菜单打开**讲道**界面。
2. 浏览**播放列表**或**最近**标签，找到您想要的讲道。
3. 点击讲道即可打开并开始观看。

您也可以点击某个播放列表查看该系列中的所有讲道，然后选择要观看的具体讲道。

讲道可能包含托管在 YouTube、Vimeo 或 Facebook 等平台上的视频或音频内容。点击讲道后，媒体会直接在应用程序内播放。

## 每个讲道显示的内容

每个讲道条目可以显示：

- **标题** -- 讲道的名称
- **讲员** -- 讲道的传讲者
- **日期** -- 讲道的录制或发布日期
- **缩略图** -- 讲道的预览图片

:::tip
如果您在讲道界面上看到倒计时，当直播开始时点击它即可直接跳转到[直播](./live-streaming)。
:::

## 可用性

当您的教会通过管理门户添加了讲道内容时，讲道界面即可使用。如果您在应用程序中没有看到**讲道**，可能是教会尚未配置此功能。
