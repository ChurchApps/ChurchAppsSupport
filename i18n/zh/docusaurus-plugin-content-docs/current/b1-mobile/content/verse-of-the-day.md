# 每日经文

<div class="article-intro">

每日经文界面以精美的图片格式显示每天的圣经经文。这是一种简单的方式，让您每天以经文开始新的一天，或在打开应用程序时获得一刻鼓励。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 您的教会必须启用每日经文功能

</div>

## 查看经文

1. 从侧边栏菜单打开**每日经文**界面。
2. 界面以精美图片的形式显示今天的经文。
3. 每天会自动更新一节新经文。

您不需要做任何操作——只需打开界面，当天的经文就会显示。

## 自适应显示

经文图片会自动调整以适应您设备的屏幕方向。根据您持握手机或平板电脑的方式，您将看到三种格式之一：

- **竖屏（9:16）** -- 针对竖直持握手机优化
- **横屏（16:9）** -- 适合平板电脑或横向持握手机
- **方形（1:1）** -- 当屏幕尺寸接近正方形时使用

应用程序会检测您的屏幕方向并选择最佳格式，因此无论您如何查看，经文都会呈现出最佳效果。如果您旋转设备，图片会更新以匹配新的方向。

:::tip
将每日经文作为您早晨例行的一部分。每天打开应用程序，在开始新的一天之前进行片刻的反思。
:::

## 可用性

每日经文是 B1 Mobile 的内置内容功能。如果您在侧边栏菜单中没有看到它，可能是教会管理员尚未将其添加到应用程序中。请联系教会管理员提出请求。
