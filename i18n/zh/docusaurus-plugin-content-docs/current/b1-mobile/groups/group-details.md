# 小组详情

<div class="article-intro">

当您从小组界面点击某个小组时，会进入该小组的详情页面。在这里，您可以查看小组的所有信息并与其内容互动，包括帖子、成员列表和小组信息。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 打开[小组](./index.md)界面

</div>

## 您将看到的内容

小组详情页面为您提供小组的完整资料：

- **小组信息** -- 小组的名称、描述、聚会详情以及小组带领者提供的其他信息。
- **帖子和时间线** -- 该小组特定的帖子和活动动态。您可以看到带领者和成员分享的内容，包括更新、公告和对话。
- **成员** -- 查看小组的成员。这帮助您了解与谁建立联系并认出熟悉的面孔。

## 与小组内容互动

从小组详情页面，您可以通过多种方式参与小组：

- **阅读帖子** -- 滚动浏览小组的时间线，了解最新活动。
- **查看成员资料** -- 点击某个成员在[成员目录](../community/member-directory.md)中查看其个人资料信息。
- **保持知情** -- 定期查看以了解小组带领者和其他成员发布的新帖子、公告和更新。

## 如何进入

1. 在 B1 Mobile 中打开**小组**界面。
2. 点击任何小组——您的主小组、精选卡片或列表中的任何小组。
3. 小组的详情页面会打开，显示所有内容和信息。

:::tip
您最常访问的小组会作为主卡片显示在小组界面的顶部，因此进入您最喜欢的小组始终只需一键。
:::
