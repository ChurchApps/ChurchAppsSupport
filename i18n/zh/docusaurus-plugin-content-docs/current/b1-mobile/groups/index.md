# 小组

<div class="article-intro">

B1 Mobile 中的小组功能帮助您与重要的人和事工保持联系。无论您属于小组团契、查经班、志愿者团队还是事工，小组界面都是您了解教会社区动态的中心。

</div>

## 您将看到的内容

打开**小组**界面后，您的小组按访问频率排列：

- **主小组** -- 您最常访问的小组以大型主卡片的形式醒目地显示在屏幕顶部。点击它即可直接进入该小组的详情页面。
- **精选小组** -- 您接下来最常访问的小组以精选卡片的形式显示在主卡片下方。这些卡片让您可以快速访问最常参与的小组。
- **所有小组** -- 在精选区域下方，您可以浏览教会的其他小组。

这种布局让您只需一键即可进入最关心的小组，同时也能访问其他所有小组。

## 您可以做什么

以下是您可以在小组界面执行的操作：

- **浏览小组** -- 滚动浏览教会的可用小组，发现新的小组加入。
- **查看小组详情** -- 点击任何小组查看其完整资料，包括帖子、成员和小组信息。
- **关注时间线** -- 统一的时间线动态在一个地方显示所有小组的帖子和活动，确保您不会错过任何更新。
- **保持联系** -- 查看小组成员分享的内容，关注对话，与小组内容互动。

:::tip
小组界面会像[仪表盘](../dashboard/)一样根据您的使用习惯进行调整。您最常访问的小组会被提升到顶部以便快速访问。
:::
