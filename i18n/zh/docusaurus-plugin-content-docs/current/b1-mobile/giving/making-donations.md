# 进行奉献

<div class="article-intro">

通过 B1 Mobile 奉献快捷而简单。奉献部分的**捐赠**标签只需几个步骤即可引导您完成整个过程，让您可以随时随地支持教会。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- 准备好信用卡或借记卡，或提前[保存支付方式](./managing-payments)

</div>

## 如何奉献

1. 在 B1 Mobile 中打开**奉献**部分。
2. 点击**捐赠**标签。
3. 选择您想要奉献的**基金**。您的教会可能有多个可用基金，例如一般基金、宣教基金或建堂基金。
4. 输入您想要奉献的**金额**。
5. 提供您的**支付信息**。如果您已保存支付方式，可以在此选择。否则，请输入您的卡片信息。
6. 提交您的奉献。

奉献处理完成后，您将看到确认信息，表明您的奉献已成功。该奉献也会出现在您的**历史**标签中。

## 安全支付

所有支付都通过 **Stripe** 处理，这是一个可信赖且广泛使用的支付平台。您的卡片信息由 Stripe 安全处理，绝不会存储在 ChurchApps 服务器上。您可以放心，您的财务信息受到保护。

:::warning
提交前请仔细检查奉献金额和基金。奉献会立即处理，无法在应用程序内撤回。
:::

## 快速重复奉献

如果您想奉献与上次相同的金额，无需经过完整的奉献流程。**概览**标签显示您最近的奉献以及一个**重复**按钮。点击它即可快速再次提交相同的奉献。

:::tip
如果您计划定期奉献，请在**管理**标签中保存您的支付方式，这样下次奉献会更快。详见[管理支付方式](./managing-payments)。
:::
