# 奉献历史

<div class="article-intro">

B1 Mobile 通过两个标签让您清楚了解自己的奉献情况：**概览**提供快速摘要，**历史**提供每笔奉献的详细记录。使用这些功能来跟踪您全年的慷慨奉献。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户
- [至少进行一次奉献](./making-donations)才能查看历史记录

</div>

## 概览标签

**概览**标签提供您奉献活动的概况。它显示：

- **年度累计总额** -- 您今年迄今为止的奉献总额。
- **奉献次数** -- 您今年进行的单独奉献次数。
- **上次奉献** -- 您最近一次奉献的详情，包括金额和日期。
- **重复按钮** -- 上次奉献旁边的快速**重复**按钮，让您只需一键即可再次奉献相同金额。如果您定期奉献相同金额，这非常方便。

概览标签是您打开奉献部分时的理想起点。它告诉您当前的奉献状况，并让您可以立即再次奉献。

## 历史标签

**历史**标签显示您所有过往奉献的完整详细列表。每个条目包含以下信息：

- **日期** -- 奉献的日期。
- **基金** -- 奉献指向的基金。
- **金额** -- 奉献的金额。

滚动列表即可查看完整的奉献记录。这对于跟踪您全年的奉献情况或查找特定的过往奉献非常有用。

:::tip
定期查看**概览**标签以跟踪您的奉献目标。年度累计总额让您一目了然地了解进度。
:::
