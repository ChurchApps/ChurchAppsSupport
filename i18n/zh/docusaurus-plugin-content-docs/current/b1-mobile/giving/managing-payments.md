# 管理支付方式

<div class="article-intro">

奉献部分的**管理**标签让您可以查看和管理已保存的支付方式。通过保存卡片，您可以更快地进行后续奉献，无需每次重新输入信息。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户

</div>

## 查看支付方式

打开**管理**标签后，您将看到所有已保存支付方式的列表。每个条目显示卡片类型和卡号的后四位，方便您轻松识别各张卡片。

## 添加新的支付方式

要添加新卡片：

1. 在 B1 Mobile 中打开**奉献**部分。
2. 点击**管理**标签。
3. 点击添加新支付方式的选项。
4. 输入卡片信息。信息会安全地发送到 **Stripe** 进行处理。
5. 保存后，新卡片将出现在您的支付方式列表中。

下次您从**捐赠**标签[进行奉献](./making-donations)时，已保存的卡片将作为支付选项可用。

## 删除支付方式

如果您不再需要某张保存的卡片，可以从**管理**标签中将其删除：

1. 在已保存的支付方式列表中找到要删除的卡片。
2. 点击卡片旁边的删除选项。
3. 确认删除。

该卡片将不再作为奉献时的支付选项显示。

:::info
删除支付方式不会影响任何过往的奉献。您的[奉献历史](./donation-history)保持不变。
:::

:::tip
至少保存一种支付方式以便快速奉献。这样，您可以使用**概览**标签上的**重复**按钮一键再次奉献。
:::
