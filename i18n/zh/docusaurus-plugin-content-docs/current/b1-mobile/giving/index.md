# 奉献

<div class="article-intro">

B1 Mobile 让您可以直接从手机轻松支持教会，提供完整的奉献体验。无论您想快速奉献、查看奉献历史还是管理支付方式，一切都集中在一个地方。

</div>

## 奉献标签

奉献部分分为四个标签：

- **概览** -- 您的奉献活动摘要。查看年度累计总额、奉献次数以及上次奉献的详情，包括快速**重复**按钮可再次奉献相同金额。
- **捐赠** -- 进行新的奉献。选择基金、输入金额并提供支付信息以完成奉献。
- **管理** -- 查看和管理您保存的支付方式。添加新卡片或删除不再使用的卡片。
- **历史** -- 所有过往奉献的详细列表，方便您查看完整的奉献记录。

## 支付方式

通过 B1 Mobile 进行的所有奉献都通过 **Stripe** 安全处理。您的支付信息由 Stripe 的安全基础设施处理，绝不会存储在 ChurchApps 服务器上。您可以放心奉献，因为您的财务信息受到保护。

:::info
您的教会管理员通过 [B1 Admin](../../b1-admin/) 设置可用的基金和支付处理。如果您在应用程序中没有看到奉献选项，可能是教会尚未配置此功能。
:::

## 您可以做什么

以下是您可以在奉献部分执行的操作：

- **向特定基金奉献** -- 选择将奉献定向到哪个基金，例如一般基金、宣教基金或建堂基金。
- **查看您的奉献** -- 检查年度累计摘要并浏览完整的奉献历史。
- **保存支付方式** -- 保存卡片信息以便将来更快地奉献。
- **重复奉献** -- 只需一键即可快速奉献与上次相同的金额。
