# 个人资料

<div class="article-intro">

B1 Mobile 中的个人资料是您管理个人信息、家庭成员和隐私设置的地方。您在此处更新的所有内容都会反映在所有 ChurchApps 服务中，因此您只需在一个地方进行更改。

</div>

## 资料标签

个人资料界面分为几个标签：

### 个人资料

**个人资料**标签让您查看和编辑个人信息，包括姓名、联系方式和头像照片。保持资料更新有助于教会与您保持联系。

### 家庭成员

**家庭成员**标签显示您的家庭成员及其在家庭中的角色。此信息用于[签到](../checkin/)等功能，您可以一次为全家签到。

### 可见性

**可见性**标签让您控制其他教会成员在[目录](../community/member-directory.md)中可以看到的关于您的信息。您决定哪些个人信息是公开的，哪些保持私密。

### 待审核

**待审核**标签显示您已提交但等待管理员审批的资料更改。

### 账户

**账户**标签提供对安全设置的访问。

## 进入个人资料

1. 打开侧边栏菜单。
2. 点击**个人资料**打开个人资料界面。
3. 使用顶部的标签在各部分之间切换。
