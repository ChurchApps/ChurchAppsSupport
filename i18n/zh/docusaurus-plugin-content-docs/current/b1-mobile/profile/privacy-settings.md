# 隐私设置

<div class="article-intro">

个人资料界面的**可见性**标签让您控制其他教会成员在目录中可以看到的关于您的信息。您决定哪些个人信息是公开的，哪些保持私密。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户

</div>

## 控制您的可见性

1. 打开侧边栏菜单，点击**个人资料**。
2. 选择**可见性**标签。
3. 查看每个字段并选择是否对其他成员可见。
4. 保存更改。

您可以控制以下信息的可见性：

- **电话号码** -- 其他成员是否可以看到您的电话号码
- **电子邮件地址** -- 您的电子邮件是否显示在[目录](../community/member-directory.md)中
- **地址** -- 您的家庭住址是否对其他人可见

默认情况下，教会可能已将某些字段设为可见或隐藏。您可以随时调整这些设置以匹配您的舒适度。

:::tip
如果您不确定其他人能看到什么，请访问**可见性**标签查看当前设置。您随时可以进行更改。
:::

## 待审核的更改

**待审核**标签显示您已提交但等待管理员审批的资料更新。某些教会要求管理员在更改生效前进行审核和批准。如果您所做的更改需要审批，它将在此处列出并显示当前状态。

:::info
如果您所做的更改尚未生效，请检查**待审核**标签。教会管理员可能需要先进行审批。
:::

## 账户设置

**账户**标签提供对安全设置的访问。使用此标签管理与账户访问和安全相关的设置。

## 个人资料标签汇总

| 标签 | 用途 |
|---|---|
| **个人资料** | [编辑个人信息](./editing-profile)和照片 |
| **家庭成员** | [查看家庭成员](./household) |
| **可见性** | 控制在目录中其他人可以看到的内容 |
| **待审核** | 查看等待审批的资料更改 |
| **账户** | 管理安全设置 |

这些标签共同为您提供对个人信息及其在教会社区中共享方式的完全控制。
