# 编辑个人资料

<div class="article-intro">

**个人资料**标签让您更新个人信息，确保教会始终拥有正确的详细信息。您在此处所做的更改会反映在所有 ChurchApps 服务中，包括 B1.church 和教会使用的任何其他应用程序。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户

</div>

## 您可以编辑的内容

您的个人资料包含以下个人信息：

- **名字和姓氏** -- 您在 ChurchApps 中的显示名称
- **电子邮件地址** -- 用于教会通讯和账户恢复
- **电话号码** -- 方便教会在需要时联系您
- **地址** -- 您的家庭住址，用于教会记录
- **头像照片** -- 在[目录](../community/member-directory.md)和整个应用程序中代表您的图片

## 更新信息

1. 打开侧边栏菜单，点击**个人资料**。
2. 如果尚未激活，请选择**个人资料**标签。
3. 点击要更新的字段并进行更改。
4. 完成后保存更改。

您更新的信息会自动同步。任何查看您资料的人（取决于您的[隐私设置](./privacy-settings)）将立即看到新的详细信息。

:::tip
保持电子邮件地址和电话号码为最新。教会使用这些信息向您发送重要更新和提醒。
:::

## 头像照片

您可以从**个人资料**标签添加或更改头像照片。您的照片会显示在[成员目录](../community/member-directory.md)、小组成员列表以及显示您资料的其他任何地方。

## 跨 ChurchApps 的更改

由于所有 ChurchApps 服务共享相同的账户系统，您在 B1 Mobile 中所做的更改会立即显示在 [B1.church](../../b1-church/)、教会的管理工具以及任何其他连接的服务中。您无需在每个地方分别更新信息。

:::info
某些教会要求管理员审批特定的资料更改。如果您的更新没有立即显示，请检查个人资料界面上的**待审核**标签查看其状态。
:::
