# 家庭成员

<div class="article-intro">

个人资料界面的**家庭成员**标签显示您的家庭成员。在这里您可以查看谁是您家庭的成员以及成员之间的关系，这对签到等功能至关重要。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [安装 B1 Mobile](../getting-started/installing) 并[登录](../getting-started/logging-in)您的账户

</div>

## 您将看到的内容

打开**家庭成员**标签后，您将看到家庭成员列表。每位成员的条目显示：

- **姓名** -- 该人的名字和姓氏
- **角色** -- 其在家庭中的角色（如户主或其他）
- **头像照片** -- 其照片（如已上传）

您的家庭是一个共享群组，将您的家庭成员在教会系统中关联在一起。

## 家庭成员的重要性

您的家庭信息被 B1 Mobile 和整个 ChurchApps 的多项功能使用：

- **签到** -- 当您为礼拜或活动[签到](../checkin/self-checkin.md)时，可以一次为全家签到，无需逐个签到。
- **教会记录** -- 教会使用家庭数据来组织家庭记录，并以家庭为单位与您的家人沟通。
- **目录** -- 其他成员可能会在[教会目录](../community/member-directory.md)中看到您的家庭信息，具体取决于您的[隐私设置](./privacy-settings)。

## 查看家庭成员

1. 打开侧边栏菜单，点击**个人资料**。
2. 选择**家庭成员**标签。
3. 浏览家庭成员列表。

:::info
家庭成员关系通常由教会管理员管理。如果您需要在家庭中添加或删除成员，请联系教会办公室寻求帮助。
:::

## 保持信息更新

如果家庭成员的信息需要更新，他们可以[编辑自己的资料](./editing-profile)，或者您可以联系教会管理员进行更改。保持家庭信息准确可以确保[签到](../checkin/)等功能为全家顺利运行。
