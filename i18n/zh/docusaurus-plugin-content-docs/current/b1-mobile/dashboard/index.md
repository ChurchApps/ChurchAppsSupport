# 仪表盘

<div class="article-intro">

仪表盘是您在 B1 Mobile 中的主页。它为您提供教会各项功能的快捷访问，并根据您的使用习惯进行组织。布局会随着时间推移自动调整，将您最常用的功能放在最显眼的位置。

</div>

## 欢迎区域

在仪表盘顶部，您会看到欢迎信息，以及教会的标志（如果未设置标志则显示教会名称）和"保持联系"的标语。此区域确认您当前正在查看的教会。

## 精选链接

在欢迎区域下方，您最常访问的功能以大型**主卡片**形式显示，带有背景图片和功能名称。这是您使用最频繁的功能，被提升到顶部以便一键访问。

## 精选卡片

在主卡片下方，您会看到最多两个并排显示的**精选卡片**。这些代表您第二和第三常用的功能。每张卡片显示背景图片和功能名称。点击任一卡片即可直接进入该功能。

## 快捷操作

其余功能以**快捷操作**网格的形式显示在精选卡片下方。每个项目显示图标和标签。这些是教会配置的所有其他导航链接，让您可以访问从[小组](../groups/)、[奉献](../giving/)到[签到](../checkin/)、[讲道](../content/sermons.md)等所有功能。

## 布局如何自适应

仪表盘布局根据您的使用模式进行个性化定制。每次您点击某个功能时，应用程序会记录您访问的频率。使用越频繁的功能会越靠近仪表盘顶部：

- 您**最常访问**的功能成为主卡片。
- 您**第二和第三常访问**的功能成为精选卡片。
- **其他所有功能**显示在快捷操作网格中。

如果您还没有使用过应用程序，功能会按照教会配置的默认顺序显示。随着时间推移，仪表盘会重新排列以匹配您的使用习惯。

:::info
仪表盘上可用的功能由教会管理员通过 [B1 Admin](../../b1-admin/) 配置。如果您没有看到预期的功能，请联系教会办公室。
:::

## 刷新仪表盘

在仪表盘上下拉即可刷新。这将重新加载教会的链接和内容，确保您看到的是最新信息。

## 导航

从仪表盘，您可以到达应用程序的任何部分：

- 点击左上角的**菜单图标**（三条横线）打开包含所有导航选项的侧边栏菜单。
- 点击右上角的**铃铛图标**查看您的[通知](../community/notifications.md)。
- 点击任何卡片或快捷操作项即可直接进入该功能。
