# 创建日历

<div class="article-intro">

在 B1 Admin 中创建日历可让您通过连接一个或多个小组来构建精选的活动视图。活动由小组负责人在其小组内管理,您的日历会在一个地方显示这些活动。即使是域管理员也不能直接在日历部分添加或编辑活动,除非他们是活动所属小组的负责人。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 设置要包含在日历中的[小组](../groups/creating-groups.md)及其活动
- 您需要对 B1 Admin 中的日历部分拥有管理访问权限

</div>

## 创建新日历

1. 在 B1 Admin 中,导航到 **Website**,然后转到 **Calendars** 部分。
2. 点击 **Add Calendar**。
3. 为您的日历输入**名称**(例如,"青年事工活动"或"主教会日历")。
4. 添加可选的**描述**以帮助您的团队了解此日历的用途。
5. 点击 **Create** 保存您的新日历。

## 日历详情页面

创建日历后,点击它以打开详情页面。此页面有两个主要区域:

- **左列** -- 显示从已连接小组中提取的活动的日历视图。
- **右列** -- 关联的小组列表。这是您管理要包含在此日历中的小组的地方。

## 连接小组

在日历中有活动的小组会自动出现在详情页面右侧的小组列表中。

1. 在小组部分点击 **Add** 将小组与您的日历关联。
2. 从下拉列表中选择小组。
3. 选择是包含该小组的**所有活动**还是仅包含**特定活动**。
4. 点击 **Save**。

:::tip
将小组连接到您的日历是自动汇总活动的强大方式。当小组负责人向其[小组](../groups/creating-groups.md)添加活动时,它可以流入您的全教会日历,而无需您进行任何额外工作。
:::

:::info
如果您想创建一个从教会中的许多小组提取活动的单一日历,请参阅[精选日历](curated-calendar)以获取简化的方法。
:::

## 启用活动注册

您可以为任何日历活动启用注册,以便成员可以通过 B1 网站或移动应用程序注册。

1. 点击现有活动或创建新活动。
2. 在活动编辑器中,切换 **Registration** 以启用它。
3. 配置注册设置:
   - **Capacity**(可选)-- 设置最大注册人数。留空表示无限制。
   - **Registration Opens** -- 注册开始的日期和时间。
   - **Registration Closes** -- 注册结束的日期和时间。
   - **Tags** -- 逗号分隔的标签(例如,"youth, retreat, vbs")以帮助分类可注册的活动。
4. 保存活动。

启用注册后,成员在 [B1 网站](../../b1-church/events/registering)或 [B1 Mobile 应用](../../b1-mobile/events/registering)上查看活动时将看到 **Register for this Event** 按钮。

### 管理注册

要查看和管理活动的注册:

1. 导航到 B1 Admin 中的 **Registrations** 页面。
2. 您将看到所有启用注册的活动表格,显示活动标题、日期、当前注册人数与容量以及标签。
3. 点击活动以查看完整的注册列表,包括姓名、成员数、状态和注册日期。
4. 从详情页面,您可以:
   - **取消**单个注册
   - **永久删除**注册
   - **导出**所有注册为 CSV

:::tip
使用容量进度条监控活动填满的速度。当活动达到或超过容量时,进度条会变成红色。
:::
