# 创建日历

<div class="article-intro">

在 B1 Admin 中创建日历为您提供了一个集中管理事工、部门或整个教会活动的场所。每个日历可以从一个或多个小组中提取活动，让一切井然有序。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 设置您想要在日历中包含其活动的[小组](../groups/creating-groups.md)
- 您需要拥有 B1 Admin 中日历部分的管理员权限

</div>

## 创建新日历

1. 在 B1 Admin 中导航到**日历**部分。
2. 点击**添加日历**。
3. 为您的日历输入一个**名称**（例如，"青年事工活动"或"教会主日历"）。
4. 添加一个可选的**描述**，帮助您的团队了解此日历的用途。
5. 点击**创建**以保存您的新日历。

## 日历详情页面

创建日历后，点击它以打开详情页面。此页面有两个主要区域：

- **左栏** -- 日历和活动编辑器。在这里您可以查看、添加和管理日历上的活动。
- **右栏** -- 关联小组列表。显示哪些小组的活动包含在此日历中。

## 添加活动

1. 在日历详情页面上，使用左侧的日历编辑器添加新活动。
2. 设置活动名称、日期、时间及其他详细信息。
3. 您添加的活动将显示在日历上，并可以发布到您的网站。

## 连接小组

在日历中有活动的小组会自动显示在详情页面右侧的小组列表中。

1. 在小组部分点击**添加**，将小组与您的日历关联。
2. 从下拉菜单中选择小组。
3. 选择是包含该小组的**所有活动**还是仅包含**特定活动**。
4. 点击**保存**。

:::tip
将小组连接到日历是自动汇总活动的强大方式。当小组负责人在其[小组](../groups/creating-groups.md)中添加活动时，该活动可以自动流入您的全教会日历，无需您进行任何额外操作。
:::

:::info
如果您想创建一个从教会多个小组中提取活动的统一日历，请参阅[精选日历](curated-calendar)了解简化的方法。
:::
