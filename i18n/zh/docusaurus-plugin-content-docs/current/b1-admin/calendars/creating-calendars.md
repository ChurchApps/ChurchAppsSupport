# 创建日历

<div class="article-intro">

在 B1 Admin 中创建日历可让您通过连接一个或多个小组来构建事件的精选视图。事件由小组领导者在其小组内管理，您的日历在一个地方显示这些事件。即使是域管理员，除非他们是事件所属小组的领导者，否则也无法在日历部分直接添加或编辑事件。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 设置您想在日历中包含的[小组](../groups/creating-groups.md)
- 您需要对 B1 Admin 中的日历部分有管理权限

</div>

## 创建新日历

1. 在 B1 Admin 中，导航到**网站**，然后进入**日历**部分。
2. 单击**添加日历**。
3. 为您的日历输入**名称**（例如"青年事工活动"或"主教堂日历"）。
4. 添加可选的**描述**以帮助您的团队理解这个日历的用途。
5. 单击**创建**保存您的新日历。

## 日历详情页面

创建日历后，单击它打开详情页面。此页面有两个主要区域：

- **左栏** -- 日历视图，显示从连接的小组中提取的事件。
- **右栏** -- 关联的小组列表。这是您管理这个日历中包含哪些小组的地方。

## 连接小组

具有日历中事件的小组会自动出现在详情页面右侧的小组列表中。

1. 在小组部分单击**添加**以将小组与您的日历关联。
2. 从下拉菜单中选择小组。
3. 选择是包括该小组的**所有事件**还是仅包括**特定事件**。
4. 单击**保存**。

:::tip
将小组连接到您的日历是自动聚合事件的强大方式。当小组领导者向他们的[小组](../groups/creating-groups.md)添加事件时，它可以流入您的教堂范围日历，而不需要您做任何额外的工作。
:::

:::info
如果您想创建一个从教堂内许多小组提取事件的单一日历，请参阅[精选日历](curated-calendar)了解更精简的方法。
:::

## 启用事件注册

您可以为任何日历事件启用注册，以便成员可以通过 B1 网站或移动应用注册。

1. 单击现有事件或创建新事件。
2. 在事件编辑器中，切换**注册**以启用它。
3. 配置注册设置：
   - **容量**（可选）-- 设置最大注册数。留空表示无限制。
   - **注册开始** -- 注册可用的日期和时间。
   - **注册关闭** -- 注册关闭的日期和时间。
   - **标签** -- 逗号分隔的标签（例如"青年、撤退、暑期圣经学校"），帮助分类可注册的事件。
   - **注册问题** -- 可选地附加一个[表单](../forms/creating-forms.md)，以便注册者在注册时回答额外问题（饮食限制、T 恤尺寸、紧急联系人等）。选择**无**跳过问题。
   - **启用候补列表** -- 当活动满满时，让额外的注册者加入候补列表，而不是被拒绝。请参阅[付费注册](paid-registrations#waitlist)。
4. 保存事件。

对于付费活动，同一设置页面允许您定义已定价的**与会者类型**、可选的**选项**（附加服务）和**折扣代码**，通过您教堂的捐赠提供商收集付款。有关完整演练，请参阅[付费注册](paid-registrations)。

启用注册后，成员在[B1 网站](../../b1-church/events/registering)或 [B1 移动应用](../../b1-mobile/events/registering)上查看事件时会看到**注册此事件**按钮。如果您附加了表单，注册者会在注册期间看到**问题**步骤，他们的答案会与他们的注册一起保存。

:::info
注册问题仅适用于**未标记为受限**的表单。受限表单在注册期间会被自动跳过而不是显示，因此在将问题附加到事件时使用不受限的表单。
:::

### 管理注册

要查看和管理事件的注册：

1. 在 B1 Admin 中导航到**注册**页面。
2. 您将看到所有启用了注册的事件的表格，显示事件标题、日期、当前注册数与容量以及标签。
3. 单击事件以查看完整的注册列表，包括名称、成员数、与会者类型、付款状态和注册日期。
4. 从详情页面，您可以：
   - **添加与会者** -- 手动为离线或电话注册的人员注册。
   - **取消**单个注册
   - **删除**永久注册
   - **升级**候补列表中的注册当有空位时
   - **导出 CSV** -- 下载所有注册，包括与会者类型、选项、付款金额和问题答案

如果事件附加了注册问题，详情页面还会显示**仅未回答的问题**筛选器，以快速查找尚未提交答案的注册者，以及每个已回答注册上的**查看答案**按钮以查看他们的回复。付费活动增加**类型**栏、**已付款/总计**栏、按类型计数和付款详情对话框 -- 请参阅[付费注册](paid-registrations#the-registration-roster)。

:::tip
使用容量进度条监控事件填满的速度。当事件达到或超过容量时，条形图会变为红色。
:::
