# 日历审批

<div class="article-intro">

"审批"页面是管理员审查和处理待决房间和资源预订请求以及需要审批后才能发布的日历事件的地方。

</div>

<div class="prereqs">
<h4>开始前的准备</h4>

- 在[房间和资源](rooms-resources)中使用**审批组**配置房间或资源
- 需要**日历管理员**权限或**content.edit**权限

</div>

## 打开审批

在 B1 Admin 中，转到**日历**并选择**审批**。待决预订请求和等待审查的事件列表在这里。

## 预订请求

当小组创建事件并请求房间或资源时，请求出现在**预订请求**面板中。每行显示：

- 请求的房间或资源
- 事件名称和日期/时间
- 请求的小组

### 冲突指示器

如果两个请求对同一房间或资源重叠，冲突警告图标会出现。在批准任何一个之前，请仔细审查冲突的请求。

### 批准或拒绝

点击任何预订请求上的**✓**（批准）或**✗**（拒绝）图标。请求小组会收到该决定的通知。批准的预订被锁定到该事件的房间或资源；拒绝的预订为其他人释放该时间段。

## 待决事件

如果你的日历工作流要求事件审批才能对公众可见，待决事件出现在**待决事件**面板中。批准事件以将其发布到日历，或拒绝它以通知提交者需要进行更改。

:::tip
在[房间和资源](rooms-resources)中的房间上设置审批组以要求对该房间进行审批。有权限的小组可以在创建事件时请求房间，这些请求会进入此页面。
:::
