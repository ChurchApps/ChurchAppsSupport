# 日历

<div class="article-intro">

B1 Admin 中的日历部分允许您为教会创建和管理活动日历。您可以为不同的事工设置日历，汇总多个小组的活动，并将日历发布到您的 B1.church 网站上供会众查看。

</div>

## 查看您的日历

1. 在 B1 Admin 中，导航到**日历**部分。
2. 日历页面列出了您所有日历的名称和状态。
3. 每个日历还会显示与其关联的小组。

## 您可以做什么

- **添加新日历** -- 点击添加按钮，为特定事工、活动类型或用途创建日历。请参阅[创建日历](creating-calendars)获取分步说明。
- **编辑日历** -- 点击任何日历以打开它，管理其活动和关联的小组。
- **发布日历** -- 日历可以显示在您的 B1.church 网站上，让访客查看教会即将举办的活动。
- **创建精选日历** -- 将多个小组的活动汇集到一个日历视图中，全面了解教会正在发生的一切。请参阅[精选日历](curated-calendar)了解详情。

:::info
日历与[小组](../groups/creating-groups.md)紧密配合。当小组安排了活动时，这些活动可以被纳入您的日历中。请参阅[创建日历](creating-calendars)页面，了解如何将小组连接到日历。
:::

- [精选日历](curated-calendar) -- 将多个小组的活动汇总到一个视图中
