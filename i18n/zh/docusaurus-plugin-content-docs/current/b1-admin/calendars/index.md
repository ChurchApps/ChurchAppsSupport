# 日历

<div class="article-intro">

B1 Admin 中的日历部分允许您通过汇集小组中的活动为您的教会创建精选日历。小组负责人管理他们自己小组的活动,作为管理员,您可以构建在您的 B1.church 网站上显示这些活动的日历,供您的会众查看。

</div>

## 查看您的日历

1. 在 B1 Admin 中,导航到 **Website**,然后转到 **Calendars** 部分。
2. 日历页面列出了所有日历及其名称和状态。
3. 每个日历还显示与之关联的小组。

## 您可以做什么

- **添加新日历** -- 点击添加按钮为特定事工、活动类型或目的创建日历。有关分步说明,请参阅[创建日历](creating-calendars)。
- **编辑日历** -- 点击任何日历以打开它并管理其中包含的小组。
- **发布日历** -- 日历可以显示在您的 B1.church 网站上,以便访客可以看到您教会即将举行的活动。
- **创建精选日历** -- 将多个小组的活动提取到单个日历视图中,以全面了解您教会正在进行的所有活动。有关详细信息,请参阅[精选日历](curated-calendar)。

:::info
日历与[小组](../groups/creating-groups.md)密切配合。当小组安排活动时,这些活动可以被提取到您的日历中。请参阅[创建日历](creating-calendars)页面以了解如何将小组连接到日历。
:::
