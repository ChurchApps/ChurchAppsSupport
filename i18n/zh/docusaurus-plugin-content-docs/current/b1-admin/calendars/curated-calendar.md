# 精选日历

<div class="article-intro">

精选日历将多个小组的活动汇总到一个统一的日历视图中。这非常适合创建一个全教会日历，在一个地方展示所有事工和小组中正在发生的一切。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 设置您想要包含其活动的[小组](../groups/creating-groups.md)
- 确保小组负责人已在各自的小组中添加了活动
- 熟悉[创建日历](creating-calendars)的基础知识

</div>

## 创建精选日历

1. 在 B1 Admin 中，导航到**网站**部分。
2. 点击顶部导航中的**日历**。
3. 点击**添加日历**以创建新的精选日历。
4. 为您的日历起一个**名称**（例如，"教会活动"或"本周教会动态"）。
5. 点击**创建**以保存。

## 向日历中添加小组

1. 点击您的新日历以打开它。
2. 在右侧的**日历中的小组**部分，点击**添加**。
3. 从下拉菜单中选择一个小组。
4. 选择是添加该小组的**所有活动**还是仅添加**特定活动**。
5. 点击**保存**。
6. 重复此过程以添加您想要包含的所有小组。

:::tip
添加您所有活跃的事工小组，创建一个全面的日历，让您的会众完整了解每周教会正在发生的一切。
:::

## 分享您的日历

精选日历设置完成后，您可以通过多种方式分享它：

- **嵌入您的网站** -- 日历可以直接显示在您的 B1.church 网站页面上，供访客浏览。
- **通过 ICS 订阅** -- 点击**订阅**以复制 ICS 链接。您的会众可以将此链接粘贴到 Google Calendar、Apple Calendar、Outlook 或任何其他日历应用程序中以保持同步。
- **下载 ICS 文件** -- 下载日历文件以将其导入日历应用程序。

:::info
当小组负责人在其[小组](../groups/creating-groups.md)中添加新活动时，这些活动会自动出现在您的精选日历中。每次添加新活动时，您无需手动更新精选日历。
:::
