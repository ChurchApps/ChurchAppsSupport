# 房间、资源和日程安排

<div class="article-intro">

房间和资源让你跟踪教会的可预订空间和设备。当小组或团队想为事件使用房间或借用资源时，他们可以通过 B1 请求——你可以批准或拒绝这些请求，以防止双重预订。

</div>

<div class="prereqs">
<h4>开始前的准备</h4>

- 需要对 B1 Admin 中的"日历"部分拥有管理权限
- 设置[小组](../groups/creating-groups)，其成员将提交预订请求

</div>

## 打开房间和资源

在 B1 Admin 中，转到**日历**并选择**房间和资源**。该页面有四个选项卡：**房间**、**资源**、**封锁**和**模板**。

## 房间

房间是可以随日历事件一起预订的物理空间——圣所、团契厅、教室等。

### 添加房间

1. 在**房间**选项卡上，点击**添加房间**。
2. 输入房间**名称**。
3. 可选地设置**容量**（最大人数）。
4. 设置**审批组** - 当小组请求此房间时，所选小组的领导者将被通知并必须批准预订。如果不需要审批，则留空。
5. 点击**保存**。

### 编辑或删除房间

点击任何房间旁的编辑图标，进行更改，并点击**保存**。使用**删除**来永久移除房间。

## 资源

资源是可以随事件一起预订的项目——AV 设备、投影仪、椅子、面包车等。

### 添加资源

1. 点击**资源**选项卡，然后点击**添加资源**。
2. 输入**名称**和可选**描述**。
3. 如果有人需要签署资源贷款，则设置**审批组**。
4. 点击**保存**。

## 封锁

封锁防止房间或资源在特定时间被预订——用于翻新、预留期或维护。

### 添加封锁

1. 点击**封锁**选项卡，然后点击**添加封锁**。
2. 选择要封锁**房间**、**资源**还是所有房间和资源。
3. 设置封锁的**开始**和**结束**日期/时间。
4. 可选地添加**原因**备注。
5. 点击**保存**。

## 事件模板

模板让你保存可重复使用的事件配置——标题、描述和关联的小组——以便可以一致地创建重复事件。

### 添加模板

1. 点击**模板**选项卡，然后点击**添加模板**。
2. 输入模板的**名称**。
3. 可选地链接一个**小组**，以便从模板创建事件时自动将事件连接到该小组。
4. 点击**保存**。

创建新事件时，员工可以选择现有模板来预填充事件详情。
