# 可用时间日历

<div class="article-intro">

可用时间日历为您提供了教堂所有房间和资源预订的整体视图。从这里，您可以看到已安排的内容，在冲突发生前发现它们，并直接为任何活动预订房间或资源。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 在房间和资源部分设置至少一个[房间或资源](rooms-resources)
- 您需要对 B1 Admin 中的日历部分有编辑权限

</div>

## 打开可用时间日历

在 B1 Admin 中，转到**日历**并从侧边栏选择**可用时间**。

## 阅读日历

日历默认显示当前月份。您可以使用顶部的箭头向前或向后导航，或在月、周和日视图之间切换。

每个活动都按预订状态进行颜色编码：

| 颜色 | 含义 |
|-------|---------|
| 绿色 | 已批准 |
| 橙色 | 待批准 |
| 灰色 | 已锁定（不可用） |

将鼠标悬停在活动上会显示活动标题和它所关联的房间或资源。

## 按房间或资源筛选

使用顶部左角的**筛选**下拉菜单将日历范围缩小到单个房间或资源。选择**所有房间和资源**可返回完整视图。

## 预订房间或资源

1. 单击页面右上角的**预订**按钮。
2. 在打开的对话框中，填写活动详情：
   - **标题** — 活动的名称
   - **开始**和**结束**日期/时间
   - **可见性** — 公开或私密
   - **房间** — 选择一个或多个要预留的房间
   - **资源** — 选择一个或多个要预留的资源
3. 可选地设置**设置**和**拆卸**时间（以分钟为单位）。这些在预订的两端填充，以便为设置和清理预留空间，尽管活动的开始/结束时间保持不变。
4. 要重复预订，请选中**重复**并配置重复周期：
   - **重复间隔** -- 设置间隔（例如，每 2 周）。
   - **频率** -- 每日、每周或每月。每周可让您选择特定日期；每月可让您选择一个固定的月日期或相对模式，如"第二个星期二"。
   - **结束** -- 永不结束、在特定日期或在设定数量的重复次数后。
5. 要指定自定义预订窗口（与活动的开始/结束不同），切换**自定义预订窗口**并输入窗口的开始和结束时间。当房间需要在活动的列出时间之外可访问时使用此功能。
6. 单击**保存**提交预订。

:::info
如果房间或资源已配置**批准小组**，预订将显示为**待批准**，直到该小组的领导者批准它。有关批准工作流，请参阅[日历批准](approvals)。
:::

:::tip
日历将在您保存之前突出显示任何冲突。如果您看到冲突警告，请调整您的时间或选择不同的房间。
:::
