# 打印目录

<div class="article-intro">

打印目录功能为您的教会生成一个格式化的、可打印的成员目录。它按家庭组织成员，包括联系信息和照片，并设计为在标准信纸上打印时看起来专业。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要一个活跃的 B1 Admin 账户，有权查看人员。
- 您目录中的人应该填有照片和联系信息以获得最佳结果。

</div>

## 打开打印目录

1. 在 B1 Admin 中导航至 **人员**。
2. 点击人员结果卡顶部工具栏中的 **打印图标**（打印机图标）。它出现在导出按钮旁边。
3. 打印目录在新浏览器标签中打开并开始加载您的目录数据。
4. 约 1.5 秒后，您的浏览器的打印对话框将自动出现。

:::tip
目录在单独的标签中打开，因此您可以打印它而不离开人员页面。打印对话框出现后，标签自动导航回。
:::

## 包含的内容

打印目录包含您教会的所有 **活跃成员**。具体来说：

- 具有 **非活跃** 或 **访客** 成员资格状态的人会自动排除。
- 已 **退出** 目录的人会被排除。
- 所有其他人按 **家庭** 分组。

## 目录布局

打印的目录有两个部分：

### 封面

第一页包括：
- 您的教会名称
- 标题"成员目录"
- 当前年份
- 您教会的位置
- 包含的总家庭数量

### 成员列表

剩余页面按姓氏按字母顺序列出家庭。每个家庭卡显示：

- **照片** -- 每个家庭成员的圆形缩略图，下方有他们的名字
- **家庭显示名称** -- 例如，"Smith 一家"、"John & Jane Doe" 或仅个人的名字
- **地址** -- 街道地址、城市、州和邮编
- **电话** -- 住宅或手机号码
- **电子邮件** -- 主要电子邮件地址
- **生日** -- 为每个家庭成员格式化为单个日期
- **纪念日** -- 如果已记录

家庭被组织成 **字母部分**（A、B、C、...），每个字母组前有装饰性部分标题。每行显示两个家庭卡。

## 打印设置

目录格式为 **信纸尺寸**（8.5" × 11"），边距为 0.5 英寸。当浏览器打印对话框出现时：

- 将纸张尺寸保持为 **信纸**
- 如果您想要部分标题用其颜色样式打印，请在您的浏览器打印设置中启用 **背景图形**（有时称为"打印背景"）
- 建议纵向方向

:::info
目录尊重分页符 -- 家庭卡和部分标题不会跨页面分割。
:::
