# 搜索人员

<div class="article-intro">

**人员**页面以可搜索、可排序的表格显示您的教会目录。您可以快速找到会众中的任何人、自定义显示的信息并导出结果。高效的搜索对于日常教会管理任务至关重要，如跟进访客、准备联系人列表和管理成员记录。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个具有查看人员权限的活跃 B1 Admin 账户。如果不确定您的访问级别，请参阅[角色与权限](roles-permissions.md)。
- 您的教会目录中应该有人员。如果尚未添加任何人，请参阅[添加人员](adding-people.md)或[导入数据](importing-data.md)。

</div>

## 快速搜索

人员页面顶部的搜索栏允许您实时查找成员：

1. 点击人员页面顶部的**搜索框**。
2. 开始输入姓名、电子邮件或其他关键词。
3. 结果将在您输入时自动过滤（有大约半秒的短暂延迟，以避免每次按键都触发搜索）。
4. 下方的表格将更新，仅显示匹配的结果。

:::tip
您不需要按 Enter。搜索会在您停止输入后自动运行。
:::

## 排序结果

您可以通过点击表格中的任何列标题来排序目录：

1. 点击**列标题**（例如**姓名**或**电子邮件**）按该列排序。
2. 再次点击同一标题以反转排序顺序。

这使得按字母顺序、年龄或任何其他可见列查找人员变得轻松。

## 自定义列

不需要同时显示所有信息。您可以选择表格中显示的列：

1. 在表格顶部附近找到**列选择器下拉菜单**。
2. 勾选或取消勾选列以显示或隐藏它们。可用的列包括：
   - **照片**
   - **姓名**
   - **电子邮件**
   - **电话**
   - **地址**
   - **出生日期**
   - **年龄**
   - **性别**
   - **会员身份状态**
3. 表格将立即更新以反映您的选择。

:::info
您的列选择会影响导出到 CSV 时包含的内容。在导出前自定义列，以获取您所需的精确数据。
:::

## 导出搜索结果

您可以随时将当前搜索结果下载为 CSV 文件：

1. 应用您想要的任何搜索或过滤器。
2. 自定义列以包含您需要的数据。
3. 点击 **Export** 按钮。
4. CSV 文件将下载到您的电脑，可在 Excel、Google Sheets 或任何电子表格应用程序中打开。

有关导出的更多详情，请参阅[导出数据](./exporting-data.md)。

:::tip
对于更高级的查询——如查找过去三个月未出席的所有人——请尝试 [AI 搜索](./ai-search.md)功能，它允许您使用自然语言问题进行搜索。
:::
