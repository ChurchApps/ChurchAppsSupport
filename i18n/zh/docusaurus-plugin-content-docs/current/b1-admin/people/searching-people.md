# 搜索人员

<div class="article-intro">

**人员** 页面以可搜索的、可排序的表格显示您的教会目录。您可以快速查找会众中的任何人、自定义显示的信息，并导出您的结果。高效搜索对于日常教会管理任务（如跟进访客、准备联系人列表和管理成员记录）至关重要。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要一个活跃的 B1 Admin 账户，有权查看人员。如果您不确定您的访问级别，请参阅 [角色与权限](roles-permissions.md)。
- 您的教会目录中应该有人。如果您还没有添加任何人，请参阅 [添加人员](adding-people.md) 或 [导入数据](importing-data.md)。

</div>

## 快速搜索

人员页面顶部的搜索栏可让您实时查找成员：

1. 点击人员页面顶部的 **搜索框**。
2. 开始输入名字、电子邮件或其他关键字。
3. 当您输入时，结果会自动过滤（大约半秒的延迟，以便搜索不会在每次按键时运行）。
4. 下面的表格更新为仅显示匹配的结果。

:::tip
您无需按 Enter 键。搜索在您停止输入后自动运行。
:::

## 排序结果

您可以通过点击表格中的任何列标题来排序目录：

1. 点击 **列标题**（例如，**名字** 或 **电子邮件**）按该列排序。
2. 再次点击相同的标题以反向排序顺序。

这使得按字母顺序、年龄或任何其他可见列查找人员变得容易。

## 自定义列

不是每一条信息都需要一次显示。您可以选择表格中显示哪些列：

1. 在表格顶部查找 **列选择器下拉菜单**。
2. 检查或取消检查列以显示或隐藏它们。可用的列包括：
   - **照片**
   - **名字**
   - **电子邮件**
   - **电话**
   - **地址**
   - **出生日期**
   - **年龄**
   - **性别**
   - **成员资格状态**
3. 表格立即更新以反映您的选择。

:::info
您的列选择会影响导出为 CSV 时包含的内容。在导出前自定义列以获取您需要的确切数据。
:::

## 分页

当您的目录有许多记录时，结果被分成多个页面。使用表格底部的 **分页控件** 在页面之间移动。当前页面和总记录数显示，以便您始终知道您在列表中的位置。

:::tip
如果您想一次看到更多结果，请优化您的搜索以缩小列表范围，而不是通过大目录进行分页。
:::

## 导出搜索结果

您可以随时将您当前的搜索结果下载为 CSV 文件：

1. 应用您想要的任何搜索或过滤。
2. 自定义您的列以包含您需要的数据。
3. 点击 **导出** 按钮。
4. CSV 文件将下载到您的计算机，准备在 Excel、Google Sheets 或任何电子表格应用中打开。

有关导出的更多详细信息，请参阅 [导出数据](./exporting-data.md)。

:::tip
对于更高级的查询 -- 如找到过去三个月未参加的每个人 -- 尝试 [AI 搜索](./ai-search.md) 功能，它让您使用纯语言问题进行搜索。
:::
