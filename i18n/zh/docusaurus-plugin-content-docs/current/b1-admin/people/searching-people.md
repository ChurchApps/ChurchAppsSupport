# 搜索成员

<div class="article-intro">

**成员**页面显示你的教会目录，以可搜索、可排序的表格形式。你可以快速找到会众中的任何人、自定义显示哪些信息，并导出你的结果。高效的搜索对于日常教会管理任务（如跟进访客、准备联系名单和管理成员记录）至关重要。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 你需要一个有权限查看成员的有效 B1 Admin 账户。如果你不确定自己的访问权限，详见[角色和权限](roles-permissions.md)。
- 你的教会目录中应该有成员。如果你还没有添加任何人，详见[添加成员](adding-people.md)或[导入数据](importing-data.md)。

</div>

## 快速搜索

成员页面顶部的搜索栏让你实时查找成员：

1. 单击**成员**页面顶部的**搜索框**。
2. 开始输入名字、电子邮件或其他关键词。
3. 当你输入时结果会自动过滤（有大约半秒的延迟，所以搜索不会对每次击键都触发）。
4. 下方的表格会更新以仅显示匹配的结果。

:::tip
你不需要按 Enter。搜索会在你停止输入后自动运行。
:::

## 排序结果

你可以通过单击表格中的任何列标题来排序目录：

1. 单击一个**列标题**（例如**名字**或**电子邮件**）按该列排序。
2. 再次单击同一标题可反转排序顺序。

这样可以轻松按字母顺序、年龄或任何其他可见列查找人员。

## 自定义列

不是所有信息都需要一次性可见。你可以选择在表格中显示哪些列：

1. 查找表格顶部附近的**列选择器下拉菜单**。
2. 勾选或取消勾选列以显示或隐藏它们。可用的列包括：
   - **照片**
   - **名字**
   - **电子邮件**
   - **电话**
   - **地址**
   - **出生日期**
   - **年龄**
   - **性别**
   - **成员身份**
   - **分校区**
3. 表格会立即更新以反映你的选择。

:::info
你的列选择会影响导出到 CSV 时包含的内容。在导出前自定义列以获得你需要的确切数据。
:::

## 分页

当你的目录有很多记录时，结果会跨多个页面显示。使用表格底部的**分页控件**在页面之间移动。当前页和总记录数会显示，所以你总是知道自己在列表中的位置。

:::tip
如果你想一次看到更多结果，请精化你的搜索以缩小列表，而不是通过大型目录分页。
:::

## 导出搜索结果

你可以随时将当前搜索结果下载为 CSV 文件：

1. 应用你想要的任何搜索或过滤。
2. 自定义你的列以包含你需要的数据。
3. 单击**导出**按钮。
4. CSV 文件将下载到你的计算机，准备好在 Excel、Google Sheets 或任何电子表格应用中打开。

有关导出的更多详情，详见[导出数据](./exporting-data.md)。

:::tip
对于更高级的查询 -- 如查找过去三个月内没有出勤的所有人 -- 尝试[AI 搜索](./ai-search.md)功能，它让你可以使用普通语言问题进行搜索。
:::

## 将搜索保存为列表

运行搜索后，**保存为列表**按钮（书签图标）会出现在成员页面标题中。单击它可以将你当前的查询存储在一个名字和可选类别下，这样你就可以在未来的会话中立即重新加载它。详见[保存的列表](./lists.md)获取完整详情。
