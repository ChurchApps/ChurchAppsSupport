# 搜索人员

<div class="article-intro">

**人员**页面在可搜索、可排序的表格中显示您的教堂目录。您可以快速找到您的会众中的任何人、自定义显示哪些信息以及导出您的结果。高效的搜索对于日常教堂管理任务（如跟进访客、准备联系列表和管理成员记录）至关重要。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要一个有权限查看人员的有效 B1 Admin 账户。如果您不确定您的访问权限，请参阅[角色和权限](roles-permissions.md)。
- 您的教堂目录中应该有人员。如果您还没有添加任何人，请参见[添加人员](adding-people.md)或[导入数据](importing-data.md)。

</div>

## 快速搜索

人员页面顶部的搜索栏允许您实时查找成员：

1. 单击人员页面顶部的**搜索框**。
2. 开始输入名称、电子邮件或其他关键字。
3. 当您输入时，结果会自动筛选（大约有半秒的延迟，所以搜索不会在每次按键时触发）。
4. 下面的表格更新为仅显示匹配的结果。

:::tip
您无需按 Enter。搜索会在您停止输入后自动运行。
:::

## 排序结果

您可以通过单击表格中的任何列标题来排序目录：

1. 单击**列标题**（例如**名称**或**电子邮件**）按该列排序。
2. 再次单击相同的标题以反转排序顺序。

这使得按字母顺序、按年龄或任何其他可见列查找人员变得容易。

## 自定义列

不是所有信息都需要一次显示。您可以选择在表格中显示哪些列：

1. 查找表格顶部附近的**列选择器下拉菜单**。
2. 选中或取消选中列以显示或隐藏它们。可用的列包括：
   - **照片**
   - **名称**
   - **电子邮件**
   - **电话**
   - **地址**
   - **出生日期**
   - **年龄**
   - **性别**
   - **成员身份状态**
   - **校区**
3. 表格立即更新以反映您的选择。

:::info
您的列选择会影响导出到 CSV 时包含的内容。在导出前自定义列以获得您需要的确切数据。
:::

## 分页

当您的目录有许多记录时，结果会分页。使用表格底部的**分页控件**在页面之间移动。当前页面和总记录数被显示，以便您始终知道您在列表中的位置。

:::tip
如果您想一次看到更多结果，请完善您的搜索以缩小列表范围，而不是通过大型目录分页。
:::

## 导出搜索结果

您可以随时将当前搜索结果下载为 CSV 文件：

1. 应用您想要的任何搜索或筛选器。
2. 自定义您的列以包含您需要的数据。
3. 单击**导出**按钮。
4. 一个 CSV 文件将下载到您的计算机，准备在 Excel、Google Sheets 或任何电子表格应用中打开。

有关导出的更多详细信息，请参阅[导出数据](./exporting-data.md)。

:::tip
对于更高级的查询 -- 如查找过去三个月未参加过的每个人 -- 尝试[AI 搜索](./ai-search.md)功能，它允许您使用纯语言问题进行搜索。
:::

## 高级搜索

高级搜索允许您通过组合条件构建精确筛选器。从人员页面打开它，然后展开一个类别并检查您想要筛选的字段，为每个字段选择一个运算符和值。类别包括**名称**、**人口统计**、**联系**、**成员身份**、**活动**（捐赠和考勤）和**自定义字段**。

**自定义字段**类别列出了您教堂的[自定义字段](../settings/custom-fields.md)-- 您在设置中定义的用于追踪您自己信息的字段（例如背景检查过期日期）。提供的运算符与每个字段的类型匹配：文本字段支持*包含/等于/开头为/结尾为*，数字字段支持比较运算符，日期字段支持*等于/之后/之前*，是/否和多项选择字段让您选择一个值。您可以在此筛选的任何字段都可以保存为实时[列表](./lists.md)。

## 将搜索保存为列表

运行搜索后，**保存为列表**按钮（书签图标）会出现在人员页面标题中。单击它以在名称和可选类别下存储您当前的查询，这样您可以在将来的会话中立即重新加载它。有关完整详细信息，请参阅[已保存的列表](./lists.md)。
