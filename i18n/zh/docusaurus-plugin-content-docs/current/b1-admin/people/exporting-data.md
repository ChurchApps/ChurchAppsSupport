# 导出数据

<div class="article-intro">

B1 Admin 允许您导出教会数据，以便在电子表格中使用、与团队共享或保留备份。无论您需要一份简单的姓名和电子邮件列表还是完整的数据库导出，都有适合您需求的选项。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个具有查看要导出数据权限的活跃 B1 Admin 账户。如果不确定您的访问级别，请参阅[角色与权限](roles-permissions.md)。
- 进行完整数据库导出需要访问 **Settings** 区域。

</div>

## 从人员页面导出

导出目录最快的方法是直接从 **People** 页面：

1. 导航到左侧边栏中的 **People**。
2. 使用搜索栏或过滤器缩小要导出的结果范围（或不设过滤器以导出所有人员）。过滤技巧请参阅[搜索人员](searching-people.md)。
3. 使用**列选择器**选择要包含在导出中的列（例如姓名、电子邮件、电话、地址）。
4. 点击 **Export** 按钮。
5. 包含表格中当前显示数据的 CSV 文件将下载到您的电脑。

:::tip
在导出前自定义列。CSV 文件将精确包含您可见的列，因此您可以根据需要定制导出内容，无需事后编辑文件。
:::

## 从设置进行完整数据导出

要完整导出所有 B1 数据（不仅仅是人员），请使用设置中的导出工具：

1. 点击左侧边栏中的 **Settings**。
2. 点击顶部导航中的 **Import/Export**。
3. 从 **Data Source** 下拉菜单中选择 **B1 Database**。
4. 查看数据预览并点击 **Continue to Destination**。
5. 选择 **B1 Export Zip** 作为导出目标。
6. 监控导出进度，直到所有项目显示绿色对勾。
7. 导出文件将自动下载。在您的下载文件夹中查找 `B1Export` 文件。
8. 解压文件以访问各个 CSV 文件（如 `people.csv`），可以在 Excel、Google Sheets 或 Numbers 中打开。

:::info
完整数据导出包括人员、小组、奉献、考勤等——您 B1 数据库中的所有内容。这也是定期备份教会记录的好方法。
:::

## 导出小组数据

您还可以导出各个小组的成员名单。从 **Groups** 页面打开一个小组，然后点击**下载图标**以导出该小组的成员名单。更多详情请参阅[小组成员](../groups/group-members.md)。

:::info
导出的 CSV 文件适用于所有主要电子表格应用程序，包括 Microsoft Excel、Google Sheets 和 Apple Numbers。
:::
