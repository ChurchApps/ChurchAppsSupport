# 添加人员

<div class="article-intro">

人员部分是 B1 Admin 的基础——它是您教会的成员数据库。所有其他功能（小组、考勤、奉献、表单）都与人员记录关联。本指南将引导您完成将某人添加到数据库、编辑其详细信息以及将家庭成员链接到家庭中。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个具有管理人员权限的活跃 B1 Admin 账户。如果不确定您的访问级别，请参阅[角色与权限](roles-permissions.md)。
- 如果您需要添加的人员超过几个，请考虑使用 [CSV 导入](importing-data.md)工具。

</div>

## 添加一个人

1. 导航到 B1.church Admin 仪表板。
2. 点击左侧边栏中的 **People**。
3. 点击右上角的 **Add Person** 按钮。
4. 填写姓名、姓氏和电子邮件地址，然后点击 **Add**。

该人员的个人资料页面将打开，准备添加更多详细信息。

:::tip
如果您正在从其他教会管理系统迁移，[导入数据](importing-data.md)功能允许您从 CSV 文件导入整个目录——比逐个添加人员要快得多。
:::

## 编辑详细信息

1. 在人员的个人资料页面上，点击姓名旁边的**编辑铅笔**图标。
2. 填写其他信息，如中间名、会员身份状态、日期、地址和电话号码。
3. 点击 **Save** 保存个人信息。

个人资料还包含几个用于相关信息的选项卡：

- **Notes** — 添加关于此人的备注（牧养关怀、跟进等）
- **Groups** — 查看和管理[小组成员身份](../groups/group-members.md)
- **Attendance** — 查看[考勤记录](../attendance/tracking-attendance.md)
- **Donations** — 查看[奉献记录](../donations/recording-donations.md)

## 使用表单

您可以直接从人员的个人资料填写自定义表单。这些是您可以按照[创建表单](../forms/creating-forms.md)指南构建的用户定义表单。

1. 在人员的个人资料中，点击 **Forms** 下拉菜单选择一个表单。
2. 点击 **Add Form** 打开它。
3. 填写表单详情并点击 **Save**。

:::info
链接到人员个人资料的表单使用 **People** 表单类型。如果您需要独立表单（如活动注册），请参阅表单指南中的 [Stand Alone 表单选项](../forms/creating-forms.md)。
:::

## 管理家庭

家庭功能允许您将家庭成员链接在一起。这对于[签到](../attendance/check-in.md)特别有用，家长可以一次性为所有孩子签到。

1. 在人员的个人资料中，点击家庭名称旁边的**编辑铅笔**图标。
2. 家庭编辑器将打开。为当前人员选择**家庭角色**（例如户主、配偶、子女）。
3. 点击 **Add** 添加另一个家庭成员。
4. 在搜索框中输入人员姓名并点击 **Search**。
5. 当该人员出现在搜索结果中时，点击 **Select**。
6. 选择其家庭角色并点击 **Save** 完成家庭设置。
