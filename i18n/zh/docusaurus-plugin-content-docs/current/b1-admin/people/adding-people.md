# 添加人员

<div class="article-intro">

人员部分是 B1 Admin 的基础 -- 它是您的教堂成员数据库。所有其他功能（小组、考勤、捐赠、表单）都与人员记录相关。本指南将指导您添加某人到您的数据库、编辑他们的详情以及将家庭成员链接到家庭。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要一个有权限管理人员的有效 B1 Admin 账户。如果您不确定您的访问权限，请参阅[角色和权限](roles-permissions.md)。
- 如果您添加超过几个人员，请考虑改用[CSV 导入](importing-data.md)工具。

</div>

## 添加人员

1. 导航到 B1.church Admin 仪表板。
2. 单击左侧边栏中的**人员**。
3. 单击右上角的**添加人员**按钮。
4. 填写个人的名字、姓氏和电子邮件地址，然后单击**添加**。

个人的个人资料页面将打开，准备供您添加更多详情。

:::tip
如果您正在从另一个教堂管理系统迁移，[导入数据](importing-data.md)功能可让您从 CSV 文件导入整个目录 -- 比一次添加一个人快得多。
:::

## 编辑详情

1. 在人员的个人资料页面上，单击他们名字旁边的**编辑铅笔**。
2. 填写其他信息，如中间名、成员身份、日期、地址和电话号码。
3. 单击**保存**存储个人信息。

个人资料还包括相关信息的几个选项卡：

- **备注** — 添加有关个人的备注（牧养关怀、跟进等）
- **小组** — 查看和管理[小组成员身份](../groups/group-members.md)
- **考勤** — 查看[考勤记录](../attendance/tracking-attendance.md)
- **捐赠** — 查看[捐赠历史](../donations/recording-donations.md)

## 使用表单

您可以直接从人员的个人资料填出自定义表单。这些是用户定义的表单，您可以按照[创建表单](../forms/creating-forms.md)指南构建。

1. 在人员的个人资料上，单击**表单**下拉菜单选择表单。
2. 单击**添加表单**打开它。
3. 填写表单详情并单击**保存**。

:::info
链接到人员个人资料的表单使用**人员**表单类型。如果您需要独立表单（如活动注册），请参阅表单指南中的[独立表单选项](../forms/creating-forms.md)。
:::

:::tip
如果您只需要追踪人员上的一两条额外信息 -- 日期、数字、是/否答案 -- 使用[自定义字段](../settings/custom-fields.md)而不是表单。它们填充速度更快，并且在高级搜索中可直接搜索。
:::

## 管理家庭

家庭让您将家庭成员链接在一起。这对于[签到](../attendance/check-in.md)特别有用，其中父母可以一次为所有子女签到。

1. 在人员的个人资料上，单击家庭名称旁边的**编辑铅笔**。
2. 家庭编辑器将打开。为当前人员选择**家庭角色**（例如家长、配偶、儿童）。
3. 单击**添加**添加另一个家庭成员。
4. 在搜索框中输入个人的名称并单击**搜索**。
5. 当个人出现在搜索结果中时，单击**选择**。
6. 选择他们的家庭角色并单击**保存**完成家庭设置。
