# AI 搜索

<div class="article-intro">

**AI 搜索**功能允许您使用简单的自然语言问题来搜索教会目录，而不是使用传统的搜索字段和过滤器。您无需使用特定的过滤条件构建查询，只需用日常语言描述您要查找的内容——即可快速回答关于您会众的复杂问题。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个具有查看人员权限的活跃 B1 Admin 账户。如果不确定您的访问级别，请参阅[角色与权限](roles-permissions.md)。
- 熟悉标准的[搜索人员](searching-people.md)工具会有帮助，但不是必需的。

</div>

## 工作原理

AI 搜索使用人工智能来解读您的问题，并从人员数据库中返回匹配的结果。您可以像询问同事一样提问。

1. 在 B1 Admin 中导航到 **People** 页面。
2. 在搜索控件下方找到 **AI Search** 选项。
3. 用自然语言输入您的问题。
4. 按 **Enter** 或点击 **Search** 执行查询。
5. 结果表格将更新，显示匹配的人员。

## 查询示例

以下是您可以提出的一些问题示例：

- "谁在过去 3 个月没有出席？"
- "显示所有 30 岁以下的成员"
- "列出有孩子在青年小组中的家庭"
- "谁是今年加入教会的？"
- "显示所有没有电子邮件地址的人"
- "哪些成员住在邮编 75001 区域？"

:::tip
在提问时尽可能具体。您提供的细节越多，结果就越准确。
:::

## 何时使用 AI 搜索

AI 搜索在以下情况下特别有用：

- 您需要组合多个条件，这些条件使用传统过滤器很难设置。
- 您不确定应该按哪些字段搜索。
- 您想快速回答关于会众的问题，而无需创建报告。

:::info
AI 搜索与标准的快速搜索和列过滤器并行工作。使用最适合您需求的方法。对于简单的姓名查找，[快速搜索](searching-people.md)是最快的选择。对于更复杂的问题，AI 搜索表现最佳。
:::

有关其他搜索选项，请参阅[搜索人员](./searching-people.md)。
