# 人口统计

<div class="article-intro">

**人口统计**仪表板给你一个关于你的会众的视觉快照 -- 他们是谁、有多少、以及你的成员身份如何分布在年龄组、性别、婚姻状态和分校区中。每个图表都是交互式的：单击任何段来直接跳转到过滤的成员搜索，显示确切的那些人。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 你的目录中必须存在人员。详见[添加成员](./adding-people.md)或[导入数据](./importing-data.md)来开始。
- 要看到分校区图表，你首先需要在[设置 → 分校区](../settings/campuses.md)中配置分校区。

</div>

## 打开人口统计

从**成员**页面，单击页面标题中的**人口统计**按钮（条形图图标）。页面在顶部显示你的总人数，后跟下面描述的图表。

## 图表

### 年龄分布

一个分组柱状图，显示多少人属于每个年龄段（例如 0-9、10-17、18-24 等，到 65+）。每个柱子分成三个系列 -- **女性**、**男性**和**未分配** -- 这样你可以一览无余地看到每个年龄组内的性别分布。

### 成员身份

一个甜甜圈图，按成员身份（例如成员、定期参加者、访客等）分解你的会众。

### 性别

一个甜甜圈图，显示你目录中女性、男性和未分配记录的比例。

### 婚姻状态

一个甜甜圈图，显示跨婚姻状态值的分布（例如已婚、单身、丧偶）。

### 分校区

一个甜甜圈图，显示人们如何分布在你教会的分校区中。这个图表仅在你配置了分校区且至少有一个人被分配到分校区时出现。没有被分配分校区的人显示为**未分配**。

## 深入数据

单击甜甜圈图的任何切片或年龄图上的任何柱子段来导航到**成员**页面，其中预先过滤以显示确切的那些人。使用这个来快速构建一个联系名单、运行批量操作或导出一个段 -- 例如，在成员身份中单击**访客**切片来查看所有访客，然后将他们发送到一个工作流。

:::tip
深入到过滤结果后，在成员搜索中使用**保存为列表**来将该过滤保存为将来使用。详见[保存的列表](./lists.md)。
:::
