# 成员

<div class="article-intro">

**成员**部分是你在 B1 Admin 中教会管理的核心。这是你维护教会目录的地方 -- 搜索成员、添加新人、查看和编辑档案、追踪家庭单位。无论你的会众规模大小，保持成员记录的组织有序是驱动 B1 中每个功能的基础。

</div>

## 你可以做什么

这是成员部分中可用的关键功能概览：

1. **搜索你的目录** -- 使用快速搜索栏按名字查找任何人，或利用高级过滤器和人工智能驱动的自然语言搜索来查找你需要的确切人员。详见[搜索成员](./searching-people.md)和[AI 搜索](./ai-search.md)。

2. **将搜索查询保存为列表** -- 将任何搜索或过滤结果保存为命名列表，这样你就可以在将来一键重新加载。详见[保存的列表](./lists.md)。

3. **查看人口统计** -- 按年龄、性别、婚姻状态、成员身份和分校区查看你的会众的视觉分解 -- 单击任何图表段可深入查看这些人。详见[人口统计](./demographics.md)。

4. **添加新人** -- 快速将个人添加到你的目录中，填写他们的档案详情、联系信息和家庭关系。详见[添加成员](./adding-people.md)以获取分步说明。

5. **批量编辑成员** -- 通过改变他们的成员身份、婚姻状态、性别、选择退出偏好或小组成员身份，一次更新多个人。这可以节省对许多个人进行相同更改的时间。详见[批量编辑](./bulk-editing.md)。

6. **打印你的目录** -- 生成按家庭单位分组的格式化、可打印的成员目录，包含照片和联系信息。详见[打印目录](./print-directory.md)。

7. **导入和导出数据** -- 从 CSV 文件导入成员数据或从另一个教会管理系统（如 Breeze）迁移。你也可以随时将你的目录导出为 CSV。详见[导入数据](./importing-data.md)和[导出数据](./exporting-data.md)。

8. **管理档案** -- 查看和编辑每个人的详细档案，包括联系信息、家庭成员、[小组成员身份](../groups/group-members.md)、[出勤历史](../attendance/tracking-attendance.md)、[捐赠](../donations/recording-donations.md)和[自定义表格](../forms/creating-forms.md)。

9. **分配角色和权限** -- 通过分配角色控制你团队中的谁可以访问 B1 Admin 的不同部分。详见[分配角色](./roles-permissions.md)。

## 开始使用

要访问成员部分，请单击 B1 Admin 仪表板左侧边栏中的**成员**。你将看到完整的教会目录，顶部有搜索栏，下方是成员列表。

:::tip
如果你是第一次设置 B1，请从[导入你现有的成员数据](./importing-data.md)开始。然后根据需要审查和清理档案。
:::
