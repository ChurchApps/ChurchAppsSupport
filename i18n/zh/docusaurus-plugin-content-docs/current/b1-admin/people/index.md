# 人员

<div class="article-intro">

**人员**部分是 B1 Admin 中教会管理的核心。这里是您维护教会目录的地方——搜索成员、添加新人员、查看和编辑档案以及跟踪家庭。无论您的会众规模大小,保持人员记录的有序性是 B1 中所有其他功能的基础。

</div>

## 您可以做什么

以下是人员部分中可用的主要功能概览:

1. **搜索您的目录** -- 使用快速搜索栏按姓名查找任何人,或利用高级过滤器和 AI 驱动的自然语言搜索来准确定位您需要的人。详见[搜索人员](./searching-people.md)和 [AI 搜索](./ai-search.md)。

2. **添加新人员** -- 快速将个人添加到您的目录中,填写他们的档案详情、联系信息和家庭关系。有关分步说明,请参见[添加人员](./adding-people.md)。

3. **批量编辑人员** -- 一次更新多个人员,更改他们的会员状态、婚姻状况、性别、退出偏好或小组成员资格。这在对许多人进行相同更改时可以节省时间。请参见[批量编辑](./bulk-editing.md)。

4. **导入和导出数据** -- 从 CSV 文件导入成员数据或从其他教会管理系统(如 Breeze)迁移。您也可以随时将目录导出为 CSV。请参见[导入数据](./importing-data.md)和[导出数据](./exporting-data.md)。

5. **管理档案** -- 查看和编辑每个人的详细档案,包括联系信息、家庭成员、[小组成员资格](../groups/group-members.md)、[出勤历史](../attendance/tracking-attendance.md)、[捐赠](../donations/recording-donations.md)和[自定义表单](../forms/creating-forms.md)。

6. **分配角色和权限** -- 通过分配角色来控制团队中谁可以访问 B1 Admin 的不同部分。请参见[分配角色](./roles-permissions.md)。

## 入门指南

要访问人员部分,请单击 B1 Admin 仪表板左侧边栏中的**人员**。您将看到完整的教会目录,顶部有一个搜索栏,下面是成员列表。

:::tip
如果您是首次设置 B1,请从[导入现有成员数据](./importing-data.md)开始。然后根据需要审查和清理档案。
:::
