# 人员

<div class="article-intro">

**人员** 部分是您在 B1 Admin 中的教会管理的核心。这是您维护教会目录的地方 -- 搜索成员、添加新人、查看和编辑个人资料以及跟踪家庭。无论您的会众规模如何，保持您的人员记录有组织是赋予 B1 中的每个其他功能权力的基础。

</div>

## 您可以做什么

以下是人员部分中可用的主要功能概述：

1. **搜索您的目录** -- 使用快速搜索栏按名称找到任何人，或利用高级过滤和人工智能驱动的自然语言搜索来找到您需要的正确人员。有关详细信息，请参阅 [搜索人员](./searching-people.md) 和 [AI 搜索](./ai-search.md)。

2. **添加新人** -- 快速将个人添加到您的目录并填写他们的个人资料详细信息、联系信息和家庭联系。有关分步说明，请参阅 [添加人员](./adding-people.md)。

3. **批量编辑人员** -- 通过改变他们的成员资格状态、婚姻状况、性别、退出偏好或小组成员资格一次更新多人。在对许多个人进行相同更改时节省时间。参阅 [批量编辑](./bulk-editing.md)。

4. **打印您的目录** -- 生成一个格式化的、可打印的成员目录，按家庭分组，包括照片和联系信息。参阅 [打印目录](./print-directory.md)。

5. **导入和导出数据** -- 从 CSV 文件中导入成员数据或从另一个教会管理系统（如 Breeze）迁移。您也可以随时将您的目录导出为 CSV。参阅 [导入数据](./importing-data.md) 和 [导出数据](./exporting-data.md)。

6. **管理个人资料** -- 查看和编辑每个人的详细个人资料，包括联系信息、家庭成员、[小组成员资格](../groups/group-members.md)、[出席历史](../attendance/tracking-attendance.md)、[捐献](../donations/recording-donations.md) 和 [自定义表单](../forms/creating-forms.md)。

7. **分配角色和权限** -- 通过分配角色控制您团队的哪些成员可以访问 B1 Admin 的不同部分。参阅 [分配角色](./roles-permissions.md)。

## 开始

要访问人员部分，请点击 B1 Admin 仪表板左边栏中的 **人员**。您将看到您的完整教会目录，顶部有搜索栏，下方有成员列表。

:::tip
如果您是第一次设置 B1，请从 [导入您现有的成员数据](./importing-data.md) 开始。然后根据需要审查和清理个人资料。
:::
