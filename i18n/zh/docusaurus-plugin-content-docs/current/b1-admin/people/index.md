# 人员

<div class="article-intro">

**人员**部分是 B1 Admin 中教会管理的核心。在这里您可以维护教会目录——搜索成员、添加新人员、查看和编辑个人资料以及跟踪家庭。无论您的会众规模大小，保持人员记录的有序组织是 B1 中所有其他功能的基础。

</div>

## 您可以执行的操作

以下是人员部分可用的关键功能概述：

1. **搜索目录** -- 使用快速搜索栏按姓名查找任何人，或利用高级过滤器和 AI 驱动的自然语言搜索精确定位您需要的人。详情请参阅[搜索人员](./searching-people.md)和 [AI 搜索](./ai-search.md)。

2. **添加新人员** -- 快速将个人添加到目录中，并填写其个人资料详情、联系信息和家庭关联。分步说明请参阅[添加人员](./adding-people.md)。

3. **导入和导出数据** -- 从 CSV 文件导入成员数据或从 Breeze 等其他教会管理系统迁移。您也可以随时将目录导出为 CSV。请参阅[导入数据](./importing-data.md)和[导出数据](./exporting-data.md)。

4. **管理个人资料** -- 查看和编辑每个人的详细资料，包括联系信息、家庭成员、[小组成员身份](../groups/group-members.md)、[出席记录](../attendance/tracking-attendance.md)、[奉献](../donations/recording-donations.md)和[自定义表单](../forms/creating-forms.md)。

5. **分配角色和权限** -- 通过分配角色来控制团队中的谁可以访问 B1 Admin 的不同部分。请参阅[分配角色](./roles-permissions.md)。

## 入门

要访问人员部分，请在 B1 Admin 仪表板的左侧边栏中点击 **People**。您将看到完整的教会目录，顶部有搜索栏，下方是成员列表。

:::tip
如果您是首次设置 B1，请先[导入现有的成员数据](./importing-data.md)。然后根据需要检查和清理个人资料。
:::
