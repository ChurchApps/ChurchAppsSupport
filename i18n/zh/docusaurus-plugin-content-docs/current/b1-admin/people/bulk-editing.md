# 批量编辑人员

<div class="article-intro">
批量编辑允许您一次更新多个人员,在对许多人进行相同更改时节省时间。您可以在单个操作中更新会员状态、婚姻状况、性别、退出偏好和小组成员资格。
</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要管理人员数据的权限。详见[角色与权限](./roles-permissions.md)。
- 您应该已经添加或导入了要编辑的人员。如需要,请参见[添加人员](./adding-people.md)。
</div>

## 选择要批量编辑的人员

1. 在 B1 Admin 中导航到**人员**
2. 使用搜索或过滤工具查找要更新的人员
3. 勾选每个人姓名旁边的复选框来选择他们
   - 您可以单独选择人员
   - 或使用标题复选框选择当前页面上所有可见的人员
4. 一旦您选择了至少一个人,**批量操作**按钮将出现

:::tip
如果您需要根据特定条件更新大量人员,请先使用 [AI 搜索](./ai-search.md)功能或过滤器缩小列表范围,然后选择所有匹配的人员。
:::

## 可用的批量操作

**批量操作**菜单提供了几个选项:

### 更新会员状态

为所有选定的人员更新会员状态:

1. 单击**批量操作** → **设置会员状态**
2. 选择新状态:
   - **访客** -- 首次或偶尔参加者
   - **常规参加者** -- 频繁参加但不是会员的人
   - **会员** -- 正式教会会员
   - **员工** -- 教会员工
   - **不活跃** -- 不再参加的人
3. 选择更新模式:
   - **覆盖全部** -- 替换所有选定人员的当前状态
   - **仅更新空值** -- 仅为没有状态的人设置状态
4. 单击**更新**

### 更新婚姻状况

批量更新婚姻状况:

1. 单击**批量操作** → **设置婚姻状况**
2. 选择新状态:
   - **未知**
   - **单身**
   - **已婚**
   - **离婚**
   - **丧偶**
3. 选择是覆盖现有值还是仅更新空字段
4. 单击**更新**

### 更新性别

为多个人员更新性别信息:

1. 单击**批量操作** → **设置性别**
2. 选择值:
   - **未指定**
   - **男性**
   - **女性**
3. 选择更新模式(覆盖全部或仅更新空值)
4. 单击**更新**

### 更新退出状态

控制人员是否选择退出通信:

1. 单击**批量操作** → **设置已退出**
2. 选择:
   - **否** -- 允许通信(移除退出)
   - **是** -- 阻止通信(设置退出)
3. 选择更新模式
4. 单击**更新**

:::warning
更改退出状态时要小心。明确选择退出的人员不应收到通信,除非他们给予了新的同意。
:::

### 添加到小组

将所有选定的人员添加到一个或多个小组:

1. 单击**批量操作** → **添加到小组**
2. 搜索并选择要将人员添加到的小组
3. 您可以选择多个小组将人员添加到所有这些小组
4. 单击**添加到小组**

每个人将作为选定小组的常规成员添加。如需要,您稍后可以从[小组成员](../groups/group-members.md)页面将个人提升为小组领袖。

### 从小组中移除

从一个或多个小组中移除所有选定的人员:

1. 单击**批量操作** → **从小组中移除**
2. 搜索并选择要从中移除人员的小组
3. 您可以选择多个小组
4. 单击**从小组中移除**

:::info
此操作仅从指定的小组中移除人员。它不会删除他们的人员记录。
:::

### 删除人员

从您的教会数据库中永久删除选定的人员:

1. 单击**批量操作** → **删除**
2. 审查将被删除的人员列表
3. 在确认字段中输入 **DELETE**
4. 单击**确认删除**

:::danger
删除人员是永久性的,无法撤销。这将删除他们的所有数据,包括:
- 个人信息
- 小组成员资格
- 出勤记录
- 捐赠历史
- 表单提交

仅在您绝对确定要从系统中移除这些人员时才使用此操作。
:::

## 批量编辑结果

完成批量操作后,您将看到显示以下内容的摘要:

- **总选定数** -- 操作中包含了多少人
- **成功更新数** -- 更改了多少条记录
- **失败数** -- 无法更新的记录(如适用)
- **未更改数** -- 不需要更改的记录(例如,使用"仅更新空值"模式时)

如果有任何更新失败,您将看到解释原因的错误详情。

## 最佳实践

- **从小处开始** -- 先在少量记录上测试批量操作,以确保您进行的是正确的更改
- **使用过滤器** -- 在选择人员之前使用过滤器或 AI 搜索缩小列表范围,以确保您仅更新正确的个人
- **仔细检查选择** -- 在应用批量更改之前审查选定的人员
- **使用"仅更新空值"模式** -- 当您想填写缺失的数据而不覆盖现有信息时
- **记录重大更改** -- 保留批量更新的记录,以便以后需要参考时使用
- **与团队协调** -- 在进行大规模批量更改时让其他管理员知晓

## 常见用例

### 更新新会员

在会员班之后,将所有参加者更新为会员状态:

1. 搜索参加班级的人员
2. 全选他们
3. 使用**批量操作** → **设置会员状态** → **会员**

### 组织小组

将多个人员添加到新的小组:

1. 搜索您想加入小组的人员
2. 选择他们
3. 使用**批量操作** → **添加到小组**并选择小组

### 清理数据

为已婚夫妇填写缺失的婚姻状况:

1. 过滤已婚的人员(使用家庭信息)
2. 选择婚姻状况为空的人员
3. 使用**批量操作** → **设置婚姻状况** → **已婚** → **仅更新空值**
