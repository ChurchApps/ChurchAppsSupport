# 记录捐赠

<div class="article-intro">

在 B1 Admin 中记录捐赠通过批次系统完成。您创建一个批次来代表一次收集（例如周日奉献），然后向该批次添加单个捐赠。这使您的奉献记录保持有序且易于核对。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 设置您的[基金](funds.md)，以便将捐赠分配到正确的类别
- 创建一个[批次](batches.md)来存放您即将输入的捐赠
- 确保捐赠者在您的[人员名录](../people/adding-people.md)中，以便在输入奉献时可以查找他们

</div>

## 创建批次并添加捐赠

1. 在 **B1 Admin** 中，点击侧边栏中的**捐赠**，然后点击**批次**。
2. 点击**添加批次**。
3. 输入批次名称（例如，"周日奉献 - 1月5日"）并选择日期。点击**保存**。
4. 您的新批次出现在列表中，显示零笔捐赠和 $0.00。
5. 点击**批次名称**打开它。

## 输入单个捐赠

1. 在批次详情页面中，在**搜索字段**中输入捐赠者姓名来查找他们。
2. 选择人员后，捐赠输入表单出现，包含**日期**、**支付方式**、**基金**、**金额**和**支票号码**字段。
3. 填写详情并点击**添加捐赠**。
4. 捐赠被添加到下方的表格中，表单重置以便您输入下一笔。

:::tip
您可以快速连续输入多笔捐赠，无需离开批次页面。表单在每次输入后重置，让您可以高效地处理一叠支票或信封。
:::

## 将捐赠拆分到多个基金

有时一个捐赠者在一笔交易中向多个基金奉献。处理方法：

1. 点击捐赠行上的**编辑**按钮。
2. 在编辑表单中，为不同基金添加金额。总金额将从各基金金额自动计算。
3. 点击**保存**更新捐赠。

:::info
将捐赠拆分到多个基金在捐赠者开出一张指定用于多个用途（如一般基金和宣教基金）的支票时很常见。
:::

## 编辑或删除捐赠

要编辑捐赠，点击批次中该行的**编辑**按钮。您可以更改日期、金额、基金、支付方式或任何其他详情。完成后点击**保存**。

:::tip
批次页面标题会在您添加或编辑条目时自动更新，显示捐赠总数和合计金额。使用此功能与存款单进行核对。
:::
