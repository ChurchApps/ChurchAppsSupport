# 管理资金

<div class="article-intro">

资金让您按用途对捐赠进行分类，以便您可以追踪您的教堂为每个事工领域收到多少捐赠。常见的资金包括一般基金、建筑基金、宣教和青年事工，但您可以根据需要创建尽可能多的资金。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要访问 B1 Admin 中的**捐赠**部分
- 决定您的教堂需要哪些捐赠类别，以及每个类别是否应该免税

</div>

## 资金页面

导航到**捐赠 > 资金**以查看所有资金的列表。每一行显示资金名称和其**可免税**状态，这决定了向该资金的捐赠是否显示在年终[捐赠凭证](giving-statements.md)上。

## 创建新资金

1. 单击页面顶部的**添加资金**。
2. 输入资金的名称（例如"建筑基金"）。
3. 如果向此资金的捐赠应包含在捐赠凭证中，请选中**可免税**复选框。
4. 保持**向捐赠者显示**被选中，以在公开和成员捐赠表单中显示资金，或取消选中以将资金从捐赠者隐藏，同时仍然将其用于手动记录的捐赠。
5. 单击**保存**。

新资金立即出现在您的列表中，并在输入捐赠时可用。

## 编辑和删除资金

- 要编辑资金，单击其旁边的**编辑**按钮。您可以更改名称、切换可免税设置或切换**向捐赠者显示**以在捐赠表单中显示或隐藏它。
- 要删除资金，打开编辑表单并单击**删除**。

:::warning
删除资金不会删除以前分配给它的捐赠。这些捐赠记录保留在系统中，但资金将不再可用于新捐赠。
:::

:::info
从捐赠者隐藏的资金在此列表中的名称旁边显示一个**隐藏**标记。隐藏的资金仍然适用于在 B1 Admin 中手动记录的捐赠 -- 它们只是不会显示为公开或成员捐赠表单上的选项。
:::

## 查看资金详情

单击**资金名称**打开其详情页面。此页面显示：

- **捐赠历史** -- 向此资金的所有捐赠列表
- **日期范围筛选器** -- 将结果缩小到特定时间段
- **摘要统计** -- 总捐赠、唯一捐赠者和总金额

当您需要报告特定的筹款活动或检查筹款目标的进展时，此视图很有帮助。

## 排序和导出

- 使用列标题按名称对资金进行排序。
- 单击**导出为 CSV**以将资金列表下载为电子表格。

:::tip
在开始[记录捐赠](recording-donations.md)之前设置您的资金。有了所有资金就位，确保捐赠从一开始就被正确分类。
:::

## 后续步骤

设置了资金后，您可以在[记录捐赠](recording-donations.md)时将它们分配给捐赠。确保在生成[捐赠凭证](giving-statements.md)之前将适当的资金标记为可免税。
