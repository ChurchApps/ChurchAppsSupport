# 奉献报表

<div class="article-intro">

每年年底，您的捐赠者需要一份可抵税奉献的摘要用于记录。B1 Admin 可以轻松地一次性为所有捐赠者生成这些报表，节省数小时的手动工作。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 验证您的[基金](funds.md)已正确标记为**可抵税** -- 只有对可抵税基金的捐赠才会出现在报表上
- 确保所有捐赠已[记录](recording-donations.md)，所有在线交易已[从 Stripe 导入](stripe-import.md)

</div>

## 访问奉献报表

1. 在 **B1 Admin** 中，点击侧边栏中的**捐赠**。
2. 点击**报表**。

## 生成报表

1. 从页面顶部的下拉菜单中选择**年份**。您可以选择当前年份或之前五年中的任何一年。
2. 页面显示该年度的摘要统计信息，包括：
   - **捐赠者总数** -- 进行奉献的人数
   - **捐赠总数** -- 单个捐赠记录的数量
   - **总金额** -- 所有奉献的合计金额

## 下载报表

您有两种方式将报表发送给捐赠者：

### 下载为 CSV 文件

点击**下载 ZIP** 下载一个 ZIP 文件，其中包含每个捐赠者的单独 CSV 文件。如果您想通过电子邮件单独发送报表或将其导入其他系统，这很有用。

### 打印所有报表

点击**打印全部**在浏览器中打开每个捐赠者报表的可打印视图。从那里，使用浏览器的打印功能将其发送到打印机。每份报表从新页开始，可以直接折叠和邮寄。

:::tip
在一月初趁记录还清晰时生成报表。在生成报表之前，请仔细检查基金是否已正确标记为可抵税——只有对可抵税基金的捐赠才会被包含。
:::

:::info
奉献报表仅包括分配给启用了**可抵税**设置的基金的捐赠。如果基金未标记为可抵税，其捐赠将不会出现在报表上。您可以在[基金](funds.md)页面管理此设置。
:::

## 后续步骤

如果您需要在生成报表之前查看捐赠详情，请访问[捐赠报告](donation-reports.md)页面或查看单个[批次](batches.md)。
