# 捐赠批次

<div class="article-intro">

批次将您的捐赠分组，以便于跟踪和核对。一个典型的批次代表一次收集，例如周日奉献或特别活动。使用批次可帮助您保持有序，并方便验证您的记录是否与实际存款一致。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 确保已[设置基金](funds.md)，以便在记录捐赠时可用
- 您需要有权访问 B1 Admin 中的**捐赠**部分

</div>

## 批次页面

导航到**捐赠 > 批次**时，您将看到所有批次的列表。每行显示：

- **名称** -- 您给批次的标签
- **日期** -- 收集日期
- **捐赠** -- 批次中的单个捐赠数量
- **合计** -- 总金额

顶部的标题显示摘要统计信息，包括批次总数、所有批次的捐赠总数和总金额。

## 创建新批次

1. 点击页面顶部的**添加批次**。
2. 输入描述性名称（例如，"周日奉献 - 2月9日"）。
3. 选择收集日期。
4. 点击**保存**。

您的新批次出现在列表中，可以开始添加捐赠。

## 使用批次

- **查看捐赠** -- 点击批次名称打开它，查看其中包含的所有单个捐赠。从那里您可以添加、编辑或删除捐赠。
- **编辑批次详情** -- 点击批次行上的**编辑**按钮更改其名称或日期。
- **排序** -- 使用列标题按名称或日期对批次排序。
- **导出** -- 点击**导出为 CSV** 将批次列表下载为电子表格。

:::tip
一致地命名批次，以便日后容易找到。包含日期和收集类型（例如，"周日上午 - 2025-02-09"）可以在列表增长时保持有序。
:::

## 后续步骤

有了批次后，请参阅[记录捐赠](recording-donations.md)了解如何向其中添加单个捐赠。您还可以[导入 Stripe 交易](stripe-import.md)从在线奉献自动创建批次。
