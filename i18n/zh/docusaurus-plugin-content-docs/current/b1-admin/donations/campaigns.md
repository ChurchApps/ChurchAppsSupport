# 活动和承诺

<div class="article-intro">

活动让你开展针对具体目标的筹款活动——建筑基金、宣教旅行、特殊项目——并跟踪成员的承诺和实际捐款，以便你可以看到实时进展。

</div>

<div class="prereqs">
<h4>开始前的准备</h4>

- 设置你的[捐款基金](funds) - 每个活动与一个基金相关联
- 需要访问 B1 Admin 的"捐款"部分

</div>

## 打开活动

在 B1 Admin 中，在左侧边栏中转到**捐款**并选择**活动**。你将看到所有活动的列表，包含其目标金额、总承诺金额和迄今为止的总捐款金额。

## 创建活动

1. 点击**添加活动**。
2. 填写活动详情：
   - **名称** *（必需）* - 此活动的显示名称（例如，"2026 建筑基金"）。
   - **基金** - 此活动关联的捐款基金。
   - **开始日期** / **结束日期** - 筹款窗口。
   - **目标** - 活动的目标美元金额。
3. 点击**保存**。

## 跟踪进展

每个活动卡显示：

- **目标** - 你的目标金额
- **承诺** - 成员承诺要捐赠的总金额
- **已捐** - 在活动窗口期间实际捐赠到此活动基金的总金额
- **进度条** - 显示你朝目标迈进的进度

点击任何活动打开详情视图，其中列出个别承诺及其履行状态。

## 添加承诺

承诺是成员承诺要为活动捐赠的金额。要记录承诺：

1. 打开一个活动。
2. 点击**添加承诺**。
3. 选择进行承诺的**人**。
4. 输入承诺**金额**。
5. 可选地为承诺承诺设置**日期**。
6. 点击**保存**。

承诺出现在活动详情中并为活动卡上的**承诺**总数做出贡献。

:::tip
承诺与实际捐款不同。承诺跟踪一个承诺；捐款记录实际礼物。两者都显示在活动上，以便你可以看到承诺的履行情况。
:::

## 编辑或删除活动

点击任何活动卡上的编辑图标来更新其名称、目标、日期或基金。点击**删除**来永久移除活动及其承诺。删除活动不会删除记录到其基金的任何捐款。
