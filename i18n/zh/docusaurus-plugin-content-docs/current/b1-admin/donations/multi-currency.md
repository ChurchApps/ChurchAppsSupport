# 多货币支持

<div class="article-intro">

B1 的多货币功能允许您的教会接受和跟踪不同货币的奉献。这对于拥有国际成员、宣教士或位于不同国家的多个校区的教会特别有用。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要管理奉献的权限。详情请参阅[角色与权限](../people/roles-permissions.md)。
- 使用 Stripe 设置您的[在线奉献](./online-giving-setup.md),Stripe 支持多货币交易。
- 了解您教会处理多种货币的会计需求。

</div>

## 启用多货币

多货币支持现在在 B1 中默认启用。启用后:

- 成员在在线奉献时可以使用本地货币奉献
- 您可以手动记录任何货币的奉献
- 奉献报告以原始货币显示金额
- Stripe 自动处理在线奉献的货币转换

## 支持的货币

系统支持所有主要世界货币,包括:

- **USD** -- 美元
- **EUR** -- 欧元
- **GBP** -- 英镑
- **CAD** -- 加元
- **AUD** -- 澳元
- **MXN** -- 墨西哥比索
- **BRL** -- 巴西雷亚尔
- **INR** -- 印度卢比
- **CNY** -- 人民币
- **JPY** -- 日元
- 以及更多...

在线奉献可用的货币取决于您的 Stripe 账户支持的货币。

## 记录不同货币的奉献

### 在线奉献

当成员通过 Stripe 在线奉献时:

1. 他们在结账时选择首选货币
2. Stripe 以该货币处理付款
3. 奉献以原始货币金额记录在 B1 中
4. Stripe 自动处理任何必要的货币转换到您账户的默认货币

### 手动输入

要记录不同货币的现金或支票奉献:

1. 在 B1 Admin 中导航到 **奉献**
2. 点击 **添加奉献**
3. 从货币下拉列表中选择货币
4. 以该货币输入金额
5. 完成奉献的其余详细信息
6. 点击 **保存**

## 查看多货币奉献

### 奉献报告

奉献报告以原始货币显示金额:

- 个别奉献记录显示货币代码(例如,"$100.00 USD")
- 总额按货币计算
- 您可以按特定货币筛选

### 奉献报表

生成奉献报表时:

- 每笔奉献都以其原始货币显示
- 总额按货币细分
- 成员可以准确看到他们以每种货币奉献的金额

## Stripe 集成

对于在线奉献,Stripe 处理多货币交易:

- **自动转换** -- Stripe 将货币转换为您账户的默认货币
- **汇率** -- Stripe 使用当前市场汇率
- **费用** -- 货币转换可能产生额外的 Stripe 费用
- **支付货币** -- 资金以您账户的默认货币存入

:::info
查看您的 Stripe 仪表板以了解当前汇率和与多货币交易相关的任何费用。
:::

## 会计注意事项

处理多种货币时:

- **记录保存** -- 跟踪原始奉献金额和货币以进行准确报告
- **汇率** -- 请注意,Stripe 的转换率可能与您银行的利率不同
- **税务收据** -- 咨询您的会计师,了解如何报告不同货币的奉献以用于税务目的
- **基金分配** -- 您可以将奉献分配到特定基金,无论货币如何

## 最佳实践

- **默认货币** -- 将您教会的主要货币设置为大多数交易的默认货币
- **清晰沟通** -- 在结账过程中告诉奉献者他们使用的货币
- **一致的报告** -- 决定是以原始货币报告还是转换为单一货币进行汇总
- **定期对账** -- 将 Stripe 支付与您的奉献记录对账,考虑货币转换

## 限制

- 货币转换仅由 Stripe 处理在线奉献
- 手动奉献按输入时记录,无自动转换
- 历史报告以原始货币显示奉献
- 总额计算按货币进行,不跨货币计算
