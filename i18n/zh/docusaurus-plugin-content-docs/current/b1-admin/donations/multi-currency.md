# 多币种支持

<div class="article-intro">

B1 的多币种功能允许您的教会接受并跟踪不同货币的捐献。这对于拥有国际成员、传教士或位于不同国家的多个校区的教会特别有用。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要管理捐献的权限。有关详细信息，请参阅 [角色与权限](../people/roles-permissions.md)。
- 使用支持多币种交易的 Stripe 设置您的 [在线捐献](./online-giving-setup.md)。
- 了解您的教会处理多种货币的会计需求。

</div>

## 启用多币种

多币种支持现已在 B1 中默认启用。启用后：

- 成员在在线捐献时可以用本地货币捐献
- 您可以手动记录任何货币的捐献
- 捐献报告显示原始货币的金额
- Stripe 自动处理在线捐献的货币转换

## 支持的货币

该系统支持所有主要世界货币，包括：

- **USD** -- 美元
- **EUR** -- 欧元
- **GBP** -- 英镑
- **CAD** -- 加元
- **AUD** -- 澳元
- **MXN** -- 墨西哥比索
- **BRL** -- 巴西里亚尔
- **INR** -- 印度卢比
- **CNY** -- 中国人民币
- **JPY** -- 日元
- 还有许多其他货币...

在线捐献的可用货币取决于您的 Stripe 账户支持的货币。

## 记录不同货币的捐献

### 在线捐献

当成员通过 Stripe 在线捐献时：

1. 他们在结账时选择首选货币
2. Stripe 以该货币处理付款
3. 捐献以原始货币金额记录在 B1 中
4. Stripe 自动处理任何必要的货币转换至您账户的默认货币

### 手动输入

要记录不同货币的现金或支票捐献：

1. 在 B1 Admin 中导航至 **捐献**
2. 点击 **添加捐献**
3. 从货币下拉菜单中选择货币
4. 以该货币输入金额
5. 完成捐献的其余详细信息
6. 点击 **保存**

## 查看多币种捐献

### 捐献报告

捐献报告以原始货币显示金额：

- 单个捐献记录显示货币代码（例如，".00 USD"）
- 总计按货币计算
- 您可以按特定货币过滤

### 捐献声明

生成捐献声明时：

- 每笔捐献以原始货币显示
- 总计按货币分类
- 成员看到他们用每种货币捐献的金额

## Stripe 集成

对于在线捐献，Stripe 处理多币种交易：

- **自动转换** -- Stripe 将货币转换为您账户的默认货币
- **汇率** -- Stripe 使用当前市场汇率
- **费用** -- 货币转换可能会产生额外的 Stripe 费用
- **付款货币** -- 资金以您账户的默认货币存入

:::info
检查您的 Stripe 仪表板以查看当前转换汇率和与多币种交易相关的任何费用。
:::

## 会计考虑因素

处理多种货币时：

- **记录保存** -- 保持原始捐献金额和货币的记录，以便准确报告
- **汇率** -- 请注意 Stripe 的转换汇率可能与您银行的汇率不同
- **税收收据** -- 请咨询您的会计师了解如何报告不同货币的捐献以用于税务目的
- **资金分配** -- 您可以将捐献分配给特定资金，无论货币如何

## 最佳实践

- **默认货币** -- 将您教会的主要货币设置为大多数交易的默认值
- **清晰沟通** -- 在结账过程中告诉捐献者他们用什么货币捐献
- **一致报告** -- 决定是否以原始货币报告或转换为单一货币汇总
- **定期对账** -- 定期对账 Stripe 付款与您的捐献记录，考虑货币转换

## 限制

- 货币转换仅由 Stripe 处理在线捐献
- 手动捐献按输入值记录，不进行自动转换
- 历史报告显示原始货币的捐献
- 总计计算按货币计算，不跨货币计算
