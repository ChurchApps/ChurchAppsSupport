# 捐赠

<div class="article-intro">

B1 Admin 中的捐赠部分提供了管理教会奉献所需的一切。从记录每周的奉献到生成年终税务报表，所有捐赠工具都组织在一个地方，让您轻松掌握教会的财务状况。

</div>

## 子部分

您将在捐赠侧边栏中找到以下页面：

### 摘要

**摘要**页面为您提供所有奉献活动的可视化概览。它显示图表和筛选报告，让您快速查看整个教会的捐赠趋势。有关使用摘要和报告工具的更多详情，请参阅[捐赠报告](donation-reports.md)。

### 批次

**[批次](batches.md)**让您将捐赠分组——例如，在一次周日奉献中收集的所有支票和现金。每个批次跟踪捐赠数量和总金额，便于核对。

### 基金

**[基金](funds.md)**按用途对您的捐赠进行分类。常见示例包括一般基金、建设基金和宣教基金。您可以将每个基金标记为可抵税或不可抵税。

### 报表

**[报表](giving-statements.md)**页面为您的捐赠者生成年终奉献报表。您可以下载单个 CSV 文件或一次性打印所有报表。

### Stripe 导入

如果您通过 Stripe 接受在线捐赠，**[Stripe 导入](stripe-import.md)**工具可让您将这些交易导入 B1 Admin，使所有数据集中在一个系统中。

## 在线奉献

B1 Admin 集成了 **Stripe** 和 **PayPal** 用于接受在线捐赠。配置完成后，会员可以通过您的 B1.church 网站奉献，这些捐赠会自动出现在您的记录中。请参阅[在线奉献设置](online-giving-setup.md)开始配置。

:::tip
如果您是 B1 Admin 捐赠功能的新用户，请先[设置基金](funds.md)，然后[创建第一个批次](batches.md)并向其中[记录捐赠](recording-donations.md)。
:::
