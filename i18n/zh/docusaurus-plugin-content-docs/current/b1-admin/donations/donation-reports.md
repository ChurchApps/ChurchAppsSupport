# 捐赠报告

<div class="article-intro">

B1 Admin 为您提供了多种查看和分析教会奉献数据的方式。捐赠摘要页面提供带有图表和筛选器的可视化概览，而报告部分提供更详细的捐赠摘要报告。使用这些工具来跟踪奉献趋势、为理事会会议做准备或核对您的记录。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 确保捐赠已[记录在批次中](recording-donations.md)或[从 Stripe 导入](stripe-import.md)
- 验证您的[基金](funds.md)设置正确，以便捐赠被正确分类

</div>

## 捐赠摘要页面

**摘要**页面是您点击侧边栏中**捐赠**时首先看到的内容。它提供奉献活动的高级概览。

1. 在侧边栏中导航到**捐赠**以打开摘要页面。
2. 使用**日期范围筛选器**选择要查看的时间段。将较早的日期设在上方，较近的日期设在下方。
3. 页面显示每周奉献图表，让您一目了然地查看趋势。
4. 点击**下载**导出包含捐赠总额、奉献周和基金的 CSV 文件。

:::info
摘要页面显示汇总的奉献数据，不包括单个捐赠者姓名。如需捐赠者级别的详细信息，请使用[批次](batches.md)页面。
:::

## 查看捐赠者级别的详细信息

要查看谁捐赠了、捐了多少以及捐给了哪个基金：

1. 导航到**捐赠 > 批次**。
2. 点击**批次名称**打开它。
3. 批次详情页面列出每笔捐赠，包括捐赠者姓名、金额、基金、日期和支付方式。
4. 点击**捐赠者姓名**查看他们捐赠次数和每次金额的明细。
5. 点击**捐赠 ID** 打开侧边面板，显示该笔捐赠的完整详情。
6. 点击**下载**导出包含该批次所有捐赠者和捐赠信息的 CSV。

## 捐赠摘要报告

B1 Admin 还在报告部分包含**捐赠摘要**报告：

1. 点击侧边栏中的**报告**。
2. 选择**捐赠摘要**报告。
3. 选择筛选条件（日期范围、基金、校区等）并运行报告。

## 导出数据

您可以从多个位置导出捐赠数据：

- **摘要页面** -- 下载按基金分类的每周奉献总额 CSV
- **批次详情页面** -- 下载包含捐赠者详情的单笔捐赠 CSV
- **基金详情页面** -- 下载特定基金的捐赠历史

:::tip
对于年终报告，将摘要页面导出与[奉献报表](giving-statements.md)工具结合使用，以获取汇总趋势和单个捐赠者报表。
:::
