# 小组

<div class="article-intro">

**小组**部分允许您将教会成员组织成团队、事工、小组、查经班、委员会等。小组是您教会社区生活的支柱，本部分为您提供在一处创建、管理和与每个小组沟通的工具。

</div>

## 概述

小组按**类别**组织。类别是顶级标签，例如"小组"、"事工"或"委员会"。每个类别可以包含多个小组，便于将相关小组归类在一起。

当您打开小组页面时，顶部会显示摘要信息：

- **总小组数** -- 所有类别中的小组数量
- **类别数** -- 您创建的小组类别数量
- **总成员数** -- 所有小组中成员的总计数量

摘要下方，您的小组按类别列出，每个小组显示成员数量。

## 您可以执行的操作

以下是您可以在小组部分完成的操作：

1. **创建和组织小组** -- 在类别中添加新小组以反映您教会的结构。请参阅[创建小组](./creating-groups.md)。

2. **管理小组成员** -- 添加和移除成员、指定组长并与您的小组沟通。请参阅[小组成员](./group-members.md)。

3. **安排活动** -- 每个小组可以拥有自己的日历，用于聚会、活动和定期安排。请参阅[小组日历](./group-calendar.md)。

4. **导出小组数据** -- 将任何小组的成员名单下载为 CSV 文件，用于电子表格或邮件。导出说明请参阅[小组成员](./group-members.md)。

## 入门

要访问小组，请在 B1 Admin 仪表板的左侧边栏中点击 **People**，然后选择 **Groups**。您将看到所有现有的小组按类别组织。

:::tip
如果您刚开始使用，请先创建几个与您教会结构匹配的类别（例如"事工"、"小组"、"团队"）。然后在每个类别中添加各个小组。分步指南请参阅[创建小组](./creating-groups.md)。
:::

:::info
小组成员必须先存在于您的[人员](../people/adding-people.md)目录中，才能被添加到小组。如果您是首次设置 B1，请在创建小组之前[导入您的成员数据](../people/importing-data.md)。
:::
