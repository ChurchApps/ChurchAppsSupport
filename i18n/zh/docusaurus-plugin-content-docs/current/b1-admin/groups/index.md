# 小组

<div class="article-intro">

**小组** 部分让您将教会成员组织为事工、小组、圣经学习、委员会等。小组是教会社区生活的支柱，本部分为您提供在一个地方创建、管理和与教会中的每个小组沟通的工具。请注意 **团队** 与小组分开 -- 团队用于服事和志愿者安排。

</div>

## 概述

小组按 **类别** 组织。类别是一个顶级标签，例如"小组"、"事工"或"委员会"。每个类别可以包含多个小组，使相关小组易于保持组织。

当您打开小组页面时，您将在顶部看到一个摘要，显示：

- **总小组数** -- 所有类别中的小组数
- **类别数** -- 您创建的小组类别数
- **总成员数** -- 所有小组中成员的合计数

在摘要下方，您的小组按类别列出，每个小组显示成员数。

## 您可以做什么

以下是您在小组部分可以完成的事项：

1. **创建和组织小组** -- 在类别中添加新小组，以反映您的教会的组织方式。参阅 [创建小组](./creating-groups.md)。

2. **管理小组成员** -- 添加和移除成员、指定领导者并与您的小组沟通。参阅 [小组成员](./group-members.md)。

3. **安排事件** -- 每个小组都可以有自己的日历用于集会、事件和定期活动。参阅 [小组日历](./group-calendar.md)。

4. **管理加入申请** -- 当小组使用 **申请** 加入政策时，审查和批准或拒绝想要加入的人的进来成员申请。参阅 [小组加入申请](./group-join-requests.md)。

5. **导出小组数据** -- 将任何小组的成员列表下载为 CSV 文件供电子表格或邮件列表使用。有关导出说明，请参阅 [小组成员](./group-members.md)。

## 开始

要访问小组，请点击 B1 Admin 仪表板左边栏中的 **人员**，然后选择 **小组**。您将看到所有按类别组织的现有小组。

:::tip
如果您刚开始，首先创建几个与您的教会结构相匹配的类别（例如，"事工"、"小组"、"委员会"）。然后在每个类别中添加单个小组。有关分步演练，请参阅 [创建小组](./creating-groups.md)。
:::

:::info
小组成员必须首先存在于您的 [人员](../people/adding-people.md) 目录中，才能被添加到小组。如果您是首次设置 B1，请在创建小组之前 [导入您的成员数据](../people/importing-data.md)。
:::
