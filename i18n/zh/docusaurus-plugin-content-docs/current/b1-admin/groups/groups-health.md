# 小组健康

<div class="article-intro">

小组健康仪表板为你提供所有小组状况的概览——成员趋势、平均出席人数和过去 90 天的增长或损耗——全部在单个可排序的表格中。

</div>

<div class="prereqs">
<h4>开始前的准备</h4>

- 需要至少几个有成员的小组来查看有意义的数据。请参阅[创建小组](creating-groups)。
- 出席数据从记录的会议中提取。请参阅[考勤](../attendance/index)部分。

</div>

## 打开小组健康

在 B1 Admin 中，在左侧边栏中转到**小组**并选择**健康**。仪表板加载一个表格，每个小组一行。

## 列

| 列 | 显示的内容 |
|--------|--------------|
| **名称** | 小组名称，链接到小组详情页面 |
| **类别** | 小组的类别 |
| **成员** | 当前活跃成员数 |
| **加入（90天）** | 过去 90 天内加入的成员 |
| **离开（90天）** | 过去 90 天内离开的成员 |
| **流失率（90天）** | 过去 90 天内的净流失率百分比 |
| **平均出席** | 每个出席会议的平均人数 |

点击任何列标题按该列对表格排序。再次点击以反向排序方向。

## 使用健康数据

- **高流失 + 低加入** - 一个正在缩小且没有替换失去成员的小组。值得与小组领导者谈话。
- **高加入 + 低出席** - 人们正在注册但没有出现。考虑进行参与跟进。
- **高平均出席** - 一个健康、活跃的小组。其他小组的潜在模型。

:::tip
点击小组名称会直接带你到小组的详情页面，你可以在那里查看个别成员、出席记录和日历事件。
:::
