# 组群健康

<div class="article-intro">

组群健康仪表盘让你一目了然地了解所有组群的情况 — 会员趋势、出勤平均值和过去 90 天的增长或流失 — 全部都在一个可排序的表格中。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 你至少需要一些有成员的组群才能看到有意义的数据。参见[创建组群](creating-groups)。
- 出勤数据从记录的课程中提取。参见[出勤](../attendance/)部分。

</div>

## 打开组群健康

在 B1 Admin 中，从左侧菜单栏进入**组群**，然后选择**健康**。仪表盘加载一个表格，每个组群占一行。

## 列

| 列 | 它显示 |
|--------|--------------|
| **名称** | 组群名称，链接到组群详情页面 |
| **分类** | 组群的分类 |
| **成员** | 当前活跃成员数 |
| **加入 (90天)** | 过去 90 天内加入的成员 |
| **离开 (90天)** | 过去 90 天内离开的成员 |
| **流失 (90天)** | 过去 90 天的净流失率，以百分比表示 |
| **平均出勤** | 每次出勤课程的平均人数 |

点击任何列标题来按该列排序表格。再次点击来反转排序方向。

## 使用健康数据

- **高流失 + 低加入** — 一个正在缩小且没有替换丧失的成员的组群。值得与组群负责人进行交谈。
- **高加入 + 低出勤** — 人们正在注册但没有出现。考虑进行参与度跟进。
- **高平均出勤** — 一个健康、活跃的组群。其他组群的潜在模型。

:::tip
点击组群名称直接带你到组群的详情页面，在那里你可以查看单个成员、出勤记录和日历事件。
:::
