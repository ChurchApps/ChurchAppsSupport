# 创建小组

<div class="article-intro">

在 B1 Admin 中创建小组非常简单。您将设置一个类别、为小组命名，然后配置其设置。小组帮助您将教会组织成有意义的单元，如小组团契、事工团队、委员会和课程。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个具有管理小组权限的活跃 B1 Admin 账户。如果不确定您的访问级别，请参阅[角色与权限](../people/roles-permissions.md)。
- 为您的小组确定一个类别结构（例如"小组"、"事工"、"委员会"）。类别有助于将相关小组组织在一起。

</div>

## 添加新小组

1. 导航到 **B1 Admin 仪表板**。
2. 点击 **Groups** 选项卡。
3. 点击 **Add Group** 并输入**类别名称**。类别帮助您将相关小组组织在一起（例如"小组"、"事工"或"委员会"）。如果类别已存在，您可以从列表中选择。
4. 输入**小组名称**。
5. 点击 **Add**。您的新小组将出现在所选类别下的列表中。

## 配置小组设置

创建小组后，您可以填写其他详细信息：

1. 点击列表中的**小组名称**以打开它。
2. 点击**铅笔图标**以编辑小组设置。
3. 配置以下选项：
   - **描述** -- 小组简介。此内容对成员可见。
   - **聚会时间** -- 小组通常何时聚会（例如"每周三晚上 7 点"）。
   - **考勤跟踪** -- 如果您想为该小组记录[考勤](../attendance/tracking-attendance.md)，请启用此选项。
   - **礼拜时间** -- 如适用，将小组与特定的教会礼拜时间关联。有关礼拜时间的详细信息，请参阅[考勤设置](../attendance/setup.md)。
4. 点击 **Save** 以应用更改。

:::tip
添加清晰的描述和聚会时间可以帮助成员了解加入小组后的期望。
:::

## 后续步骤

创建和配置小组后，您可以：

- **添加成员** -- 搜索人员并将其添加到小组。使用绿色钥匙图标指定小组组长。请参阅[小组成员](./group-members.md)。
- **设置日历** -- 为小组创建活动和定期聚会。请参阅[小组日历](./group-calendar.md)。
- **沟通** -- 直接从小组页面向所有小组成员发送消息。
- **导出数据** -- 点击下载图标以导出小组成员名单。

:::info
您教会的所有小组都按类别组织在小组主页上。随着教会的发展，您可以随时重新排列或重命名类别。
:::
