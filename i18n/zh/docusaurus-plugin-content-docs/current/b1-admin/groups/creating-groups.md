# 创建小组

<div class="article-intro">

在 B1 Admin 中创建小组很简单。您将设置一个类别、为您的小组命名，然后配置其设置。小组帮助您将教堂组织成有意义的单位，如小型小组、委员会和课程。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要一个有权限管理小组的有效 B1 Admin 账户。如果您不确定您的访问权限，请参阅[角色和权限](../people/roles-permissions.md)。
- 决定您的小组的类别结构（例如"小型小组"、"事工"、"委员会"）。类别有助于保持相关小组的组织。

</div>

## 添加新小组

1. 导航到 **B1 Admin 仪表板**。
2. 单击**小组**选项卡。
3. 单击**添加小组**并输入**类别名称**。类别可以帮助您将相关小组组织在一起（例如"小型小组"、"事工"或"委员会"）。如果类别已存在，您可以从列表中选择它。
4. 输入**小组名称**。
5. 单击**添加**。您的新小组将出现在所选类别下的列表中。

## 配置小组设置

一旦小组创建好后，您可以填写额外的详情：

1. 单击列表中的**小组名称**以打开它。
2. 单击**铅笔图标**以编辑小组设置。
3. 配置以下选项：
   - **描述** -- 小组的简要摘要。这对成员可见。
   - **会议时间** -- 小组通常在什么时候开会（例如"星期三晚上 7 点"）。
   - **加入政策** -- 选择谁可以加入此小组：
     - **开放** -- 任何人都可以立即加入而无需批准
     - **请求** -- 人员必须提交需要批准的加入请求（参见[小组加入请求](./group-join-requests.md)）
     - **封闭** -- 成员必须由领导者或管理员手动添加
   - **标签** -- 为小组分配一个或多个描述标签（例如"面对面"、"在线"、"欢迎新成员"）。标签是您定义的自由形式标签；检查所有适用的。标签可以用于筛选小组浏览器网站元素中的小组。
   - **机密小组** -- 从公开页面、小组查找器和非成员中隐藏此小组及其花名册。使用此功能处理敏感小组，如恢复或辅导事工；只有小组的成员和教堂工作人员可以看到它。
   - **考勤跟踪** -- 如果您想为此小组记录[考勤](../attendance/tracking-attendance.md)，请启用此功能。
   - **服务时间** -- 如果适用，将小组与特定的教堂服务时间关联。有关服务时间的详细信息，请参阅[考勤设置](../attendance/setup.md)。
4. 单击**保存**以应用您的更改。

:::tip
添加清晰的描述和会议时间可以帮助成员在加入小组时了解期望。
:::

## 复制小组

开始定期上课或事工的新一季？与其重新输入所有设置，不如复制现有小组：

1. 打开小组并单击小组横幅中的**复制**图标（在编辑旁边）。
2. 确认复制。

副本继承了原始的设置 -- 类别、描述、会议时间/地点、加入政策、标签和校区 -- 但**不包括**其成员。新小组以原始名称命名，附加" (副本)"；从小组设置重命名它。

## 归档小组

当小组不再活跃但您想保留其历史而不是删除它时：

1. 打开小组并单击**铅笔图标**以编辑其设置。
2. 单击**归档**并确认。

归档的小组从主小组列表中消失。要再次找到它，请打开小组页面顶部的**显示归档**切换，然后单击小组旁边的**还原**以将其带回。

:::info
归档小组不会删除其成员、考勤历史或日历事件 -- 它仅在您还原它之前隐藏主列表中的小组。
:::

## 后续步骤

创建并配置小组后，您已准备好：

- **添加成员** -- 搜索人员并将其添加到小组中。使用绿色钥匙图标指定小组领导者。参见[小组成员](./group-members.md)。
- **设置日历** -- 为小组创建事件和重复会议。参见[小组日历](./group-calendar.md)。
- **沟通** -- 直接从小组页面向所有小组成员发送消息。
- **导出数据** -- 单击下载图标以导出您小组的成员列表。

:::info
所有教堂小组都在主小组页面上按类别组织。随着教堂的增长，您始终可以重新排列或重命名类别。
:::
