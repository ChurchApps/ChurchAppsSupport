# 创建小组

<div class="article-intro">

在 B1 Admin 中创建小组非常简单。您将设置类别、命名小组,然后配置其设置。小组帮助您将教会组织成有意义的单位,如小组、委员会和课程。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个活跃的 B1 Admin 账户,并具有管理小组的权限。如果您不确定自己的访问级别,请参阅[角色与权限](../people/roles-permissions.md)。
- 决定小组的类别结构(例如,"小组"、"事工"、"委员会")。类别有助于保持相关小组的组织性。

</div>

## 添加新小组

1. 导航到 **B1 Admin 仪表板**。
2. 点击 **小组** 选项卡。
3. 点击 **添加小组**,输入 **类别名称**。类别帮助您将相关小组组织在一起(例如,"小组"、"事工"或"委员会")。如果类别已存在,您可以从列表中选择它。
4. 输入 **小组名称**。
5. 点击 **添加**。您的新小组将出现在所选类别下的列表中。

## 配置小组设置

创建小组后,您可以填写其他详细信息:

1. 点击列表中的 **小组名称** 打开它。
2. 点击 **铅笔图标** 编辑小组设置。
3. 配置以下选项:
   - **描述** -- 关于小组内容的简要摘要。这对成员可见。
   - **聚会时间** -- 小组通常何时聚会(例如,"周三晚上7点")。
   - **加入政策** -- 选择谁可以加入此小组:
     - **开放** -- 任何人都可以立即加入,无需批准
     - **请求** -- 人们必须提交需要批准的加入请求(参见[小组加入请求](./group-join-requests.md))
     - **封闭** -- 成员必须由领导者或管理员手动添加
   - **考勤跟踪** -- 如果您想记录此小组的[考勤](../attendance/tracking-attendance.md),请启用此功能。
   - **服务时间** -- 如果适用,将小组与特定的教会服务时间关联。有关服务时间的详细信息,请参阅[考勤设置](../attendance/setup.md)。
4. 点击 **保存** 应用您的更改。

:::tip
添加清晰的描述和聚会时间可以帮助成员了解加入小组后的期望。
:::

## 后续步骤

创建和配置小组后,您可以:

- **添加成员** -- 搜索人员并将他们添加到小组。使用绿色钥匙图标指定小组领导者。参见[小组成员](./group-members.md)。
- **设置日历** -- 为小组创建活动和定期聚会。参见[小组日历](./group-calendar.md)。
- **沟通** -- 直接从小组页面向所有小组成员发送消息。
- **导出数据** -- 点击下载图标导出小组成员列表。

:::info
您教会的所有小组都在主小组页面上按类别组织。随着教会的发展,您可以随时重新排列或重命名类别。
:::
