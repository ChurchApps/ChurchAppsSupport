# 创建小组

<div class="article-intro">

在 B1 Admin 中创建小组很简单。您将设置一个类别，为您的小组命名，然后配置其设置。小组帮助您将您的教会组织成有意义的单位，如小组、委员会和课程。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要一个具有小组管理权限的活跃 B1 Admin 账户。如果您不确定您的访问级别，请参阅 [角色与权限](../people/roles-permissions.md)。
- 为您的小组决定一个类别结构（例如，"小组"、"事工"、"委员会"）。类别帮助保持相关小组的组织。

</div>

## 添加新小组

1. 导航至 **B1 Admin 仪表板**。
2. 点击 **小组** 标签。
3. 点击 **添加小组** 并输入 **类别名称**。类别帮助您将相关小组组织在一起（例如，"小组"、"事工"或"委员会"）。如果类别已存在，您可以从列表中选择。
4. 输入 **小组名称**。
5. 点击 **添加**。您的新小组将在选定的类别下的列表中出现。

## 配置小组设置

创建小组后，您可以填写其他详细信息：

1. 点击列表中的 **小组名称** 打开它。
2. 点击 **铅笔图标** 编辑小组设置。
3. 配置以下选项：
   - **描述** -- 小组是什么的简要总结。成员可以看到这个。
   - **集会时间** -- 小组通常集会的时间（例如，"周三晚上 7 点"）。
   - **加入政策** -- 选择谁可以加入这个小组：
     - **开放** -- 任何人都可以立即加入，无需批准
     - **申请** -- 人们必须提交需要批准的加入申请（参阅 [小组加入申请](./group-join-requests.md)）
     - **关闭** -- 成员必须由领导者或管理员手动添加
   - **标签** -- 为小组分配一个或多个描述性标签（例如，"面对面"、"在线"、"欢迎新成员"）。标签是您定义的自由形式标签；检查所有适用的标签。标签可用于在小组浏览器网站元素中过滤小组。
   - **出席跟踪** -- 如果您想为该小组记录 [出席](../attendance/tracking-attendance.md)，请启用此选项。
   - **服务时间** -- 如果适用，将小组与特定的教会服务时间关联。有关服务时间的详细信息，请参阅 [出席设置](../attendance/setup.md)。
4. 点击 **保存** 应用您的更改。

:::tip
添加清晰的描述和集会时间有助于成员了解加入小组时的预期。
:::

## 后续步骤

创建并配置小组后，您已准备好：

- **添加成员** -- 搜索人员并将他们添加到小组。使用绿色钥匙图标指定小组领导者。参阅 [小组成员](./group-members.md)。
- **设置日历** -- 为小组创建事件和定期集会。参阅 [小组日历](./group-calendar.md)。
- **沟通** -- 直接从小组页面向所有小组成员发送消息。
- **导出数据** -- 点击下载图标导出您的小组成员列表。

:::info
所有教会小组都按类别在主要小组页面上组织。当您的教会增长时，您总是可以重新安排或重命名类别。
:::
