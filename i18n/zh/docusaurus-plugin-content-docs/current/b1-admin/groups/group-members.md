# 小组成员

<div class="article-intro">

创建小组后，下一步是添加成员。在小组详情页面中，您可以搜索人员、将其添加到小组、分配组长、发送消息和导出成员名单。管理小组成员对于协调事工、小组和团队至关重要。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要在 B1 Admin 中至少设置一个小组。如果尚未创建，请参阅[创建小组](creating-groups.md)。
- 您要添加的人员必须已存在于您的[人员](../people/adding-people.md)目录中。

</div>

## 向小组添加成员

1. 导航到 **Groups** 页面并点击您要管理的小组。
2. 点击 **Members** 选项卡。
3. 在搜索框中输入您要添加的人员姓名。
4. 在搜索结果中点击该人员姓名旁边的 **Add**。
5. 该人员现在出现在小组成员名单中。

:::tip
将搜索框留空并点击 **Search** 可以浏览整个目录。当您不确定某人姓名的确切拼写时，这很有用。
:::

## 指定小组组长

小组组长拥有特殊权限——他们可以编辑[小组日历](group-calendar.md)、管理活动并帮助协调小组。

1. 在小组成员名单中，找到您想指定为组长的人员。
2. 点击其姓名旁边的**绿色钥匙图标**。
3. 该人员现在被指定为小组组长。

要取消组长身份，请再次点击绿色钥匙图标。

:::info
任何小组成员都可以查看小组日历和活动，但只有组长可以添加或编辑日历活动。
:::

## 向小组成员发送消息

您可以直接从 B1 Admin 与小组的所有成员沟通：

1. 在小组详情页面，找到消息区域。
2. 在文本框中输入您的消息。
3. 点击 **Send**。

您的消息将发送给小组的所有成员。

## 导出小组数据

要将小组成员名单下载为文件：

1. 在小组详情页面，点击**下载图标**。
2. 包含小组成员信息的 CSV 文件将下载到您的电脑。

这对于创建打印名册、将数据导入其他工具或保留离线记录非常有用。有关更多导出选项，请参阅[导出数据](../people/exporting-data.md)。

## 移除成员

要从小组中移除某人，在成员名单中找到其姓名，然后点击其条目旁边的**移除**按钮。

:::info
从小组中移除某人不会将其从教会目录中删除。他们仍会出现在[人员](../people/adding-people.md)部分中，并可以随时重新添加到小组。
:::
