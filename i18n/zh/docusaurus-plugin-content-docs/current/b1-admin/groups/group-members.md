# 小组成员

<div class="article-intro">

创建小组后，下一步就是添加成员。在小组详情页面中，你可以搜索人员、将他们添加到小组、指定领导者、发送消息以及导出成员列表。管理小组成员对于协调小型团体、委员会和班级是必不可少的。

</div>

<div class="prereqs">
<h4>开始前的准备</h4>

- 需要在 B1 Admin 中至少设置一个小组。如果尚未创建，请参阅[创建小组](creating-groups.md)。
- 要添加的人员必须已存在于你的[人员](../people/adding-people.md)目录中。

</div>

## 向小组添加成员

1. 导航到**小组**页面，点击要管理的小组。
2. 点击**成员**选项卡。
3. 在搜索框中输入要添加的人员的姓名。
4. 点击搜索结果中该人员姓名旁的**添加**按钮。
5. 该人员现在将显示在小组成员列表中。

:::tip
留空搜索框并点击**搜索**以浏览整个目录。如果你不确定某人的确切拼写，这种方法会很有帮助。
:::

## 指定小组领导者

小组领导者拥有特殊权限——他们可以编辑[小组日历](group-calendar.md)、管理事件并帮助协调小组。

1. 在小组成员列表中，找到要设为领导者的人员。
2. 点击其姓名旁的**绿色钥匙图标**。
3. 该人员现已被指定为小组领导者。

要移除领导者身份，请再次点击绿色钥匙图标。

:::info
任何小组成员都可以查看小组日历和事件，但只有领导者才能添加或编辑日历事件。
:::

## 向小组成员发送消息

你可以直接从 B1 Admin 与小组的所有成员进行交流：

1. 从小组详情页面，找到消息传递区域。
2. 在文本框中输入你的消息。
3. 点击**发送**。

你的消息将被传递给小组的所有成员。

## 向小组成员发送电子邮件

你可以向小组的所有成员发送格式化的电子邮件：

1. 从小组详情页面，点击**电子邮件图标**。
2. "发送电子邮件"对话框打开，显示将收到电子邮件的成员数量和没有电子邮件地址的成员数量。
3. 可选地从下拉菜单中选择一个**电子邮件模板**，或从头开始撰写消息。点击**管理模板**来创建或编辑模板。
4. 输入**主题行**。你可以通过点击字段芯片来插入合并字段：`{{firstName}}`、`{{lastName}}`、`{{displayName}}`、`{{email}}`、`{{churchName}}`。
5. 使用 HTML 编辑器撰写**电子邮件正文**。相同的合并字段也可在此处使用。
6. 点击**发送**。
7. 摘要将显示成功发送的电子邮件数量和跳过的成员数量（没有电子邮件地址）。

:::tip
为周期性通信（如每周更新、事件公告或祷告请求）创建可重复使用的电子邮件模板。模板可以节省时间并确保消息传递的一致性。
:::

## 导出小组数据

要将小组成员列表下载为文件：

1. 从小组详情页面，点击**下载图标**。
2. 包含小组成员信息的 CSV 文件将下载到你的计算机。

这对于创建打印名单、将数据导入其他工具或保留离线记录非常有用。有关更多导出选项，请参阅[导出数据](../people/exporting-data.md)。

## 向小组成员发送推送通知

你可以直接向在其设备上安装了 B1.church 应用并启用了推送通知的所有小组成员发送推送通知。

1. 从小组详情页面，点击标题工具栏中的**铃声图标**（在电子邮件和短信图标旁）。
2. 对话框打开，显示小组中有多少成员启用了推送功能。
3. 填写通知详情：
   - **标题** *（必需）* - 简短摘要，最多 80 个字符。
   - **消息** *（必需）* - 通知正文，最多 240 个字符。
   - **打开链接或传单 URL** *（可选）* - 相对应用路径（例如 `/mobile/groups`）或完整 `https://` URL，点击通知时打开。
   - **图像 URL** *（可选）* - 在支持的设备上与通知一起显示的 `https://` URL 图像。
4. 实时预览显示通知在设备上的显示方式。
5. 点击**发送通知**。

:::info
推送通知仅传递给已安装 B1.church PWA 且未禁用推送通知的小组成员。没有注册推送设备或推送已关闭的成员将被计为跳过，发送摘要显示有多少成员被成功推送与跳过。
:::

:::tip
发送后，对话框显示成功排队的通知数量。如果大多数成员显示为跳过，请提醒他们访问其 B1.church 网站、将其安装为主屏幕应用并在提示时允许通知。
:::

## 移除成员

要从小组中移除某人，在成员列表中找到其姓名，并点击其条目旁的**移除**按钮。

:::info
从小组中移除人员不会从你的教会目录中删除他们。他们仍将显示在[人员](../people/adding-people.md)部分中，并可随时重新添加到小组。
:::
