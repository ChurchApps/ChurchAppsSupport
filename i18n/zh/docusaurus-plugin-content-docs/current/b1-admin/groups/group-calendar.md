# 小组日历

<div class="article-intro">

B1 中的每个小组都可以拥有自己的活动日历。小组组长使用它来安排聚会、活动和定期事件，所有小组成员都可以查看即将到来的活动并参与相关讨论。日历使您的小组保持有序，确保每个人都了解即将发生的事情。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要在 B1 Admin 中创建一个至少有一名成员的小组。请参阅[创建小组](creating-groups.md)和[小组成员](group-members.md)。
- 要添加或编辑日历活动，您必须被指定为小组组长。相关说明请参阅[指定小组组长](group-members.md)。

</div>

## 访问小组日历

小组日历可通过 B1.church 面向成员的应用程序访问：

1. 在 B1 Admin 中，点击屏幕右上角的**下拉菜单**。
2. 选择 **Switch App**。
3. 选择 **B1.church**。
4. 点击左侧边栏中的 **Groups**。
5. 从列表中选择您的小组。
6. 点击屏幕左侧的 **Calendar**。

:::info
小组日历通过 B1.church（成员应用程序）管理，而非 B1 Admin。小组组长通过从管理仪表板切换应用程序来访问它。
:::

## 添加活动

1. 在小组日历中，点击 **Add an Event**。
2. 输入活动的**标题**。
3. 写一个简短的**描述**说明活动内容。
4. 设置**开始时间**和**结束时间**。
5. 点击 **Save** 将活动添加到日历。

## 设置重复活动

如果您的小组按固定时间表聚会，您可以设置活动自动重复：

1. 创建或编辑活动时，勾选 **Recurring** 复选框。
2. 选择活动的重复方式：
   - **Interval** -- 设置两次之间的天数。例如，输入"7"表示每 7 天重复一次。
   - **Frequency** -- 选择按**天**、**周**或**月**重复。
3. 保存活动。它现在将在每个预定日期出现在日历上。

:::tip
对于每周六聚会的小组，将频率设置为"周"并选择周六。活动将自动填充到未来的日历中。
:::

## 查看日历

小组日历提供多种视图选项：

- **月** -- 一览整个月份。
- **周** -- 聚焦当前一周。
- **日** -- 详细查看单天。
- **议程** -- 按时间顺序查看即将到来的活动列表。

使用日历顶部的按钮在视图之间切换。

## 活动详情和讨论

点击任何活动以查看其详情。在活动详情视图中：

- **成员和组长**可以发起讨论并对活动发表评论。
- **组长**可以编辑活动详情、更改时间或删除活动。

:::warning
删除活动将永久移除它。如果您需要取消重复活动的单次发生，请考虑编辑它。
:::

## 订阅日历

您可以将小组日历与个人日历同步：

1. 在小组日历页面中，点击 **Subscribe**。
2. 按照提示将日历添加到 Google Calendar、Apple Calendar、Outlook 或任何其他支持订阅的日历应用程序。

:::tip
订阅可以使您的个人日程与小组活动保持同步，无需单独查看 B1。小组组长所做的任何更改都会自动出现在您同步的日历中。
:::
