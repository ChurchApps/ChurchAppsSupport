# 小组日历

<div class="article-intro">

B1 中的每个小组都可以有自己的事件日历。小组领导者使用它来安排会议、活动和重复事件，而所有小组成员都可以查看即将发生的事件并参与关于它们的对话。日历保持您的小组组织有序，确保每个人都了解即将发生的事情。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 您需要在 B1 Admin 中创建至少有一个成员的小组。参见[创建小组](creating-groups.md)和[小组成员](group-members.md)。
- 要添加或编辑日历事件，您必须被指定为小组领导者。有关说明，请参见[指定小组领导者](group-members.md)。

</div>

## 访问小组日历

小组日历可通过 B1.church 成员应用获得：

1. 从 B1 Admin，单击屏幕右上角的**下拉菜单**。
2. 选择**切换应用**。
3. 选择 **B1.church**。
4. 单击左侧边栏中的**小组**。
5. 从列表中选择您的小组。
6. 单击屏幕左侧的**日历**。

:::info
小组日历是通过 B1.church（成员应用）而不是 B1 Admin 管理的。小组领导者通过从管理员仪表板切换应用来访问它。
:::

## 添加事件

1. 从小组日历中，单击**添加事件**。
2. 为事件输入**标题**。
3. 写一个关于事件内容的简短**描述**。
4. 设置**开始时间**和**结束时间**。
5. 单击**保存**将事件添加到日历。

## 设置重复事件

如果您的小组定期集会，您可以将事件设置为自动重复：

1. 创建或编辑事件时，选中**重复**框。
2. 选择事件应如何重复：
   - **间隔** -- 设置两次发生之间的天数。例如，输入"7"每 7 天重复一次事件。
   - **频率** -- 选择按**日**、**周**或**月**重复。
3. 保存事件。它现在将在每个预定的发生时出现在日历上。

:::tip
对于每周六开会的小组，将频率设置为"周"并选择星期六。事件将自动在日历上向前展示。
:::

## 在 B1 Admin 中批量创建事件

B1 Admin 中的小组页面也有一个**日历**选项卡，员工可以一次安排整个赛季的会议：

1. 在 B1 Admin 中打开小组并选择**日历**选项卡。
2. 单击**添加事件**打开**添加重复事件**对话框。
3. 选择时间表（每周、每 2 周、每月或一次），**首次会议**和**最后会议**日期，以及可选的**跳过假日**。
4. 查看会议日期的预览并取消选中任何要跳过的内容。
5. 如果您希望成员能够响应这些事件，请保持**允许 RSVP**被选中（见下文），然后保存。

## 事件 RSVP

小组事件可以收集 RSVP -- 为事件的每次发生选择**是**、**否**或**也许**。

- **允许 RSVP** 对通过批量添加对话框创建的事件默认打开；取消选中它以关闭对这些事件的响应。关闭 RSVP 的事件在日历选项卡上显示为**禁用**。
- 在 B1 Admin 小组日历选项卡上，每个事件显示一个响应摘要（例如"5 是 · 2 也许 · 1 否"）。单击打开**RSVP 响应**花名册，列出谁为该发生选择了是、也许和否。

:::tip
RSVP 为领导者提供了人数而无需完整活动注册的正式性。对于付费注册或容量管理的活动，请改用[事件注册](../calendars/creating-calendars.md#enabling-event-registration)。
:::

## 查看日历

小组日历提供多个视图选项：

- **月** -- 一次查看整个月。
- **周** -- 关注当前周。
- **日** -- 详细查看单一日。
- **议程** -- 按时间顺序查看即将发生的事件列表。

使用日历顶部的按钮在视图之间切换。

## 事件详情和对话

单击任何事件以查看其详情。从事件详情视图：

- **成员和领导者**可以开始对话并留下关于事件的评论。
- **领导者**可以编辑事件详情、更改时间或删除事件。

:::warning
删除事件会永久删除它。如果您需要取消重复事件的单次发生，请考虑改为编辑它。
:::

## 订阅日历

您可以将小组日历与您的个人日历同步：

1. 从小组日历页面，单击**订阅**。
2. 按照提示将日历添加到 Google 日历、Apple 日历、Outlook 或任何其他支持订阅的日历应用。

:::tip
订阅使您的个人日程与您的小组事件保持同步，而无需单独检查 B1。小组领导者所做的任何更改都会自动显示在您的同步日历中。
:::
