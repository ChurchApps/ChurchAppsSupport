# 让您的教会开始使用 B1

<div class="article-intro">

为新教会提供的完整入门指南——从创建账户到导入会员通讯录，再到配置您的第一批功能。按顺序执行这些步骤，让您的教会在 B1 上运行起来。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 一个有效的电子邮件地址
- 您的教会标志和品牌素材（颜色、图片）——建议准备但不是开始的必要条件

</div>

## 步骤 1：创建您的账户

注册您的教会并成为第一位管理员。

按照[创建账户](../../getting-started/create-account.md)指南：

1. 前往 b1.church 并使用您的电子邮件注册
2. 验证您的电子邮件地址
3. 选择"创建新教会"并输入您的教会名称
4. 您将自动成为域管理员，拥有所有内容的完全访问权限

## 步骤 2：配置教会设置

输入您教会的基本信息和品牌标识。

按照[教会设置](../settings/church-settings.md)指南：

1. 输入您教会的名称、地址、电话号码和电子邮件
2. 设置您的子域名（这将变成 yourchurch.b1.church）
3. 上传您的教会标志和图片

:::tip
您的子域名会出现在您的网站 URL 和奉献页面 URL 中，所以请选择简短且易于识别的名称（例如，恩典社区教会可以用"grace"）。
:::

## 步骤 3：设置角色并邀请您的团队

控制谁可以访问什么，并让您的团队加入。

按照[角色与权限](../settings/roles-permissions.md)指南：

1. 为您的团队创建角色（例如"牧师"、"办公室管理员"、"财务团队"、"敬拜主领"）
2. 为每个角色分配特定权限，使人们只能看到他们需要的内容
3. 通过添加团队成员并分配角色来邀请他们

:::info
您不必自己做所有事情。尽早邀请您的行政人员、牧师和事工负责人，这样他们可以帮助设置各自的领域。
:::

## 步骤 4：导入您的成员

将现有的会员通讯录导入 B1。

按照[导入数据](../people/importing-data.md)指南。您有两个选项：

1. **CSV 上传**——从您当前的系统或电子表格中导出，格式化为 CSV，上传到 B1
2. **Breeze 迁移**——一键导入，可带来人员、照片、小组、捐款和出席记录

:::tip
从 Breeze 迁移？导入工具可以一步完成所有数据迁移——人员、照片、小组、捐款和出席历史。这是最快的入门方式。
:::

## 步骤 5：配置移动应用

为您的会众自定义 B1 Mobile 应用体验。

按照[移动应用设置](../settings/mobile-app.md)指南：

1. 自定义您的成员在 B1 Mobile 应用中看到哪些标签页
2. 设置标签页名称、图标和类型（圣经、直播、奉献、网站等）
3. 控制哪些功能对您的会众可见

## 步骤 6：开始设置功能

现在基础已经就绪，按照以下指南设置具体功能：

- [设置在线奉献](./online-giving.md)——通过您的网站和应用接受捐款
- [启动您的教会网站](./launch-website.md)——构建您的教会网站
- [运营小组](./small-groups.md)——组织小组、查经班和事工
- [设置儿童签到](./childrens-checkin.md)——儿童事工的自助签到
- [管理周日志愿者](./sunday-volunteers.md)——协调您的服事团队
- [建立您的讲道库](./sermon-library.md)——将讲道放到网上并设置直播
- [追踪出席情况](./track-attendance.md)——监控谁参加了礼拜和小组

## 完成！

您的教会已在 B1 上设置完毕。您的团队拥有了访问权限，您的成员已导入，您已准备好开始使用对您的事工最重要的功能。浏览上方的功能指南来设置各个领域。
