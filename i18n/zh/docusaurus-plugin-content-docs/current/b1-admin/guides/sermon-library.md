# 建立讲道库并设置直播

<div class="article-intro">

将您的讲道存档上线并设置直播——从导入您现有的 YouTube 或 Vimeo 讲道库，到安排定期直播并配置互动聊天功能。完成后，您的会众可以在网站和移动应用上观看往期讲道并加入直播礼拜。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 拥有内容管理权限的 B1 Admin 帐户
- 包含讲道视频的 YouTube 或 Vimeo 频道（用于导入您的存档）
- YouTube 频道 ID（用于直播——可在 YouTube Studio 设置中找到）

</div>

## 第 1 步：创建讲道播放列表

播放列表将您的讲道整理成系列和合集，方便浏览。

按照[播放列表](../sermons/playlists.md)指南：

1. 创建系列/合集，如"2024 主日信息"、"婚姻系列"、"圣诞礼拜"
2. 为每个播放列表添加名称、描述、发布日期和缩略图

:::tip
在添加讲道之前先创建播放列表。这样在添加讲道时可以更方便地将其归入正确的系列中。
:::

## 第 2 步：导入您现有的讲道

如果您已经在 YouTube 或 Vimeo 上有讲道视频，可以一次性全部导入。

按照[批量导入](../sermons/bulk-import.md)指南：

1. 输入您的 YouTube 或 Vimeo 频道链接
2. 预览导入内容，查看将导入哪些视频
3. 点击导入，一次性全部添加

:::tip
启用自动导入功能后，您上传到 YouTube/Vimeo 的新视频将自动出现在 B1 中——无需手动操作。
:::

## 第 3 步：添加单个讲道

对于单独添加或不属于批量导入的讲道，可以逐个添加。

按照[管理讲道](../sermons/managing-sermons.md)指南：

1. 粘贴视频链接（YouTube、Vimeo 或 Facebook）
2. 标题、描述、缩略图和时长将从视频元数据中自动填充
3. 将讲道分配到播放列表

## 第 4 步：设置直播

连接您的 YouTube 频道，使您的 B1 网站自动获取直播流。

按照[直播](../sermons/live-streaming.md)指南：

1. 在直播设置中输入您的 YouTube 频道 ID
2. 配置您的定期礼拜时间表（例如，每周日上午 10:00）
3. 自定义直播页面标签：聊天、祷告请求和自定义链接

:::info
当您在 YouTube 上开始直播时，B1 直播页面会自动获取直播流。周日早上无需额外操作。
:::

## 第 5 步：在您的网站上展示

让访客方便地找到您的讲道库和直播。

按照[页面管理](../website/managing-pages.md)指南：

1. 在您的 B1 网站上添加讲道版块，让访客可以浏览您的讲道库
2. 添加直播版块，提供直播观看体验

## 第 6 步：移动端访问

会众可以从任何设备观看讲道和参加直播礼拜。

1. 会众可以在 [B1 Mobile 应用](../../b1-mobile/content/sermons.md)和 [B1.church](../../b1-church/content/sermons.md) 上观看讲道
2. 直播聊天和祷告请求功能在两个平台上均可使用

:::info
直播聊天和祷告请求功能让您的线上会众能够实时参与，就像身临其境一样。
:::

## 完成！

您的讲道库已上线，直播也已配置完成。会众可以随时回顾往期讲道，也可以从任何地方加入直播礼拜。新上传的讲道可以自动同步。
