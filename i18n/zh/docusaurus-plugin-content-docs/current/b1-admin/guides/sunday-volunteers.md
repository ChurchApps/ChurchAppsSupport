# 管理主日志愿者

<div class="article-intro">

设置您的志愿者团队，创建每周礼拜计划，分配岗位，编排包含敬拜诗歌的礼拜流程，并自动化定期任务。完成后，您的志愿者将在 B1 网站和移动应用上看到他们的分配，并可以直接接受或拒绝。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 拥有管理员权限的 B1 Admin 帐户
- 志愿者已[添加到您的人员目录](../people/adding-people.md)

</div>

## 第 1 步：创建志愿者团队

将您的志愿者组织成团队，以便按事工领域管理分配。

按照[创建小组](../groups/creating-groups.md)和[小组成员](../groups/group-members.md)指南：

1. 创建一个分类（例如"主日早晨团队"）
2. 创建小组，如"敬拜团队"、"音响与媒体"、"迎宾组"、"儿童事工志愿者"
3. 通过搜索志愿者姓名并点击添加，将志愿者加入各个小组

:::tip
在成员标签页上使用绿色钥匙图标指定团队领导。领导可以管理自己的小组日历并协调团队。
:::

## 第 2 步：建立诗歌库

添加敬拜诗歌，以便每周将其纳入礼拜计划。

按照[诗歌](../serving/songs.md)指南：

1. 添加敬拜诗歌，包括标题、艺术家和时长
2. 为每首诗歌添加编排，包含歌词和调号
3. 添加外部链接，指向 YouTube 演唱视频、和弦谱或乐谱

:::info
在此添加的诗歌可以随后纳入任何礼拜计划。只需建立一次诗歌库，每周重复使用。
:::

## 第 3 步：创建礼拜计划

计划是告知志愿者何处需要他们的每周分配表。

按照[计划](../serving/plans.md)指南：

1. 导航到事奉，选择您的事工标签
2. 点击"添加计划"并选择礼拜日期
3. 在分配标签页上，展开每个团队并为志愿者分配岗位
4. 志愿者将看到他们的分配状态（已请求、已确认、待处理）

## 第 4 步：编排礼拜流程

从头到尾规划您的礼拜流程。

按照[礼拜流程](../serving/service-order.md)指南：

1. 在您的计划中切换到"礼拜流程"标签
2. 点击"添加项目"添加元素：敬拜诗歌（来自您的诗歌库）、祷告、经文朗读、公告、讲道、奉献、结束
3. 拖放项目以重新排列礼拜流程

:::info
礼拜流程是针对每个计划的。此处的更改不会影响其他计划或主诗歌库。
:::

## 第 5 步：设置定期任务

自动化每周职责，确保万无一失。

按照[任务](../serving/tasks.md)和[自动化](../serving/automations.md)指南：

1. 为每周定期职责创建自动化任务（例如"准备圣餐用品"、"打印教会公告"、"设置音响设备"）
2. 将每个自动化任务分配给负责人或小组
3. 任务会自动出现在他们的 B1 仪表板和移动应用上

:::tip
在假期或特殊时期将自动化设置为非活跃状态，准备好后再重新激活。无需删除和重新创建。
:::

## 第 6 步：通知志愿者

被分配的志愿者会自动看到他们的计划并可以回应。

1. 志愿者可以在 [B1.church](../../b1-church/plans/viewing-plans.md) 和 [B1 Mobile 应用](../../b1-mobile/serving/viewing-plans.md)上查看他们的计划
2. 他们可以直接从任一平台接受或拒绝分配
3. 他们可以设置不可用日期，标记无法参加的周次
4. 可以打印计划用于排练或张贴在后台

:::tip
移动应用的推送通知会在志愿者收到新分配或计划变更时提醒他们。
:::

## 完成！

您的志愿者管理系统已设置完成。每周创建新计划、分配岗位，您的团队就能在手机或电脑上看到所有内容。定期任务会自动处理日常准备工作。
