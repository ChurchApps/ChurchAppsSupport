# 设置活动报名

<div class="article-intro">

创建活动报名表单，收集参加者信息和可选的付款，将其嵌入您的教会网站，并管理收到的提交。完成后，您将拥有一个可分享的报名页面，适用于任何教会活动。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 具有管理员权限的 B1 Admin 账户
- 如需收取费用：需先[配置 Stripe](../donations/online-giving-setup.md)

</div>

## 步骤 1：创建独立表单

独立表单拥有自己的公共 URL，任何人都可以访问——非常适合活动报名。

按照[创建表单](../forms/creating-forms.md)指南：

1. 导航到表单，点击添加表单
2. 选择"Stand Alone"类型——这将为您的表单提供自己的公共 URL
3. 以活动名称命名（例如"男性退修会报名"、"VBS 报名"）

## 步骤 2：添加问题

构建您需要从报名者那里收集信息的字段。

按照[创建表单](../forms/creating-forms.md#adding-questions)指南添加问题：

1. 转到"问题"标签页，添加所需信息的字段：姓名、电子邮件、电话、饮食限制、T恤尺码、紧急联系人等
2. 对于餐饮偏好或课程选择等选项，使用多项选择

:::warning
付款字段类型需要配置 Stripe。如果您尚未设置在线奉献，请在添加付款字段之前参阅[在线奉献设置](../donations/online-giving-setup.md)。
:::

## 步骤 3：配置表单设置

控制您的报名表单何时以及如何可用。

1. 如果报名只应在有限时间内开放，请设置可用日期
2. 复制公共 URL——您可以直接分享
3. 添加具有管理员或仅查看角色的表单成员，以帮助管理提交

## 步骤 4：嵌入到您的网站

通过将报名表单添加到您的教会网站，使其易于找到。

按照[管理页面](../website/managing-pages.md)指南：

1. 在您的 B1 网站编辑器中，向页面添加新的部分并选择表单元素
2. 从列表中选择您的报名表单

:::tip
也可以通过电子邮件、社交媒体和教会公报分享独立 URL——它出现的地方越多，您获得的报名就越多。
:::

## 步骤 5：管理提交

随着报名的到来跟踪注册情况，并在需要时导出数据。

按照[管理提交](../forms/managing-submissions.md)指南：

1. 在提交标签页上查看收到的回复
2. 导出为 CSV，用于电子表格、人数统计或与活动协调员分享

## 完成！

您的活动报名已上线。分享链接，将其嵌入您的网站，并从 B1 Admin 跟踪报名。活动结束后，导出最终名单作为记录。
