# 跟踪教会出勤

<div class="article-intro">

为您的礼拜和小组设置出勤跟踪——从配置堂点和礼拜时间，到手动记录出勤或通过自助签到应用记录，再到在报告中查看趋势。完成后，您将全面了解谁在参加以及何时参加。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 拥有管理员权限的 B1 Admin 帐户
- [已将人员添加到您的通讯录](../people/adding-people.md)
- 已为您要跟踪的事工[创建小组](../groups/creating-groups.md)

</div>

## 第 1 步：配置堂点和礼拜

定义您的教会聚会地点和时间。

按照[出勤设置](../attendance/setup.md)指南：

1. 添加您的堂点（例如"主堂"）
2. 添加礼拜时间（例如"周日上午 9:00"、"周日上午 11:00"、"周三晚上 7:00"）
3. 为每个礼拜时间分配小组，以定义该礼拜期间有哪些班级/事工在聚会

## 第 2 步：启用小组出勤跟踪

为您要监控的小组开启出勤跟踪。

按照[创建小组](../groups/creating-groups.md)指南：

1. 编辑您要跟踪的每个小组，开启"跟踪出勤"
2. 将小组与礼拜时间关联，以便出勤记录对应到正确的礼拜

:::info
并非每个小组都需要出勤跟踪。只为您想监控参与情况的小组启用它——儿童班级、小组、主日学等。
:::

## 第 3 步：手动记录出勤

勾选每场礼拜和小组的出勤人员。

按照[记录出勤](../attendance/tracking-attendance.md)指南：

1. 导航到出勤，选择日期和礼拜
2. 勾选每个小组的出勤人员

:::tip
手动跟踪非常适合成人礼拜、小组以及不便使用签到终端的任何场景。
:::

## 第 4 步：设置自助签到（可选）

让会众使用平板终端自行签到。

按照[签到](../attendance/check-in.md)指南：

1. 在平板电脑上安装 B1 Checkin 应用，用于自助签到
2. 会众搜索自己的姓名，为自己和家人签到
3. 签到出勤记录与手动输入的记录一起显示——数据没有区别

:::info
如需完整的儿童事工签到设置（包括硬件和打印机配置），请参阅[儿童签到指南](./childrens-checkin.md)。
:::

## 第 5 步：查看出勤报告

全面了解您教会的参与情况。

按照[出勤报告](../reports/attendance-reports.md)指南。提供三种报告视图：

1. **出勤趋势**——查看一段时间内的出勤变化，发现增长或下降趋势
2. **小组出勤**——比较各小组的出勤总数
3. **每日小组出勤**——逐日明细，用于详细分析

:::info
报告同时包含手动记录和自助签到的出勤数据。数据显示方式没有区别——两种方法的数据汇入同一报告。
:::

## 完成！

您的出勤跟踪已设置完成。无论您是手动记录出勤、使用签到应用，还是两者兼用——所有数据都汇入相同的报告，让您清晰了解教会各项活动的参与情况。
