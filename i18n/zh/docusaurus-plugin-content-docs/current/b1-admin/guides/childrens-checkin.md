# 设置儿童事工签到

<div class="article-intro">

本指南将引导您完成在教会中运行儿童签到系统所需的一切——从将家庭信息输入数据库，到配置适龄分组，到在周日早上打印姓名标签。完成后，家长将能够在自助平板电脑上为孩子签到，并收到配对的安全标签。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在 [admin.b1.church](https://admin.b1.church) 创建您的教会账户
- 确保您拥有管理员权限——如需帮助，请参阅[角色与权限](../people/roles-permissions.md)
- 可选：如果您从其他系统迁移，请准备一个包含家庭信息的 CSV 文件

</div>

## 步骤 1：将家庭添加到数据库

在签到功能运行之前，系统需要了解您的家庭信息。每个孩子都需要通过家庭功能与家长关联。

按照[添加人员](../people/adding-people.md)指南添加至少一个家庭。请确保：

- 先添加父母
- 添加每个孩子
- 使用[家庭编辑器](../people/adding-people.md#managing-households)将他们关联到同一个家庭

:::tip
如果您需要添加的家庭不止几个，请使用 [CSV 导入](../people/importing-data.md)工具，而不是逐个添加。您可以在几分钟内导入整个通讯录。
:::

## 步骤 2：创建儿童分组

分组定义了孩子们签到的班级。通常您需要为每个年龄段创建一个分组。

按照[创建分组](../groups/creating-groups.md)指南创建如下分组：

- **婴儿班**（0-2 岁）
- **学前班**（3-5 岁）
- **小学班**（6-10 岁）

您可以根据您的事工结构调整名称和年龄范围。

## 步骤 3：配置校区和礼拜

签到与特定的礼拜时间绑定。您至少需要配置一个校区和一个礼拜。

按照[出席设置](../attendance/setup.md)指南：

1. 添加您的校区（例如"主校区"）
2. 添加一个礼拜（例如"周日早间礼拜"）
3. 设置礼拜时间（例如"上午 9:00"）
4. 将您的儿童分组分配到礼拜

## 步骤 4：设置签到应用

现在通过在平板电脑上安装签到应用来连接一切。

1. 安装 **B1 Checkin 应用**——请参阅[签到](../attendance/check-in.md)文章获取下载链接
2. 使用您的 B1 Admin 凭据登录
3. 选择您的校区和礼拜时间

请参阅完整的[签到](../attendance/check-in.md)文章了解详细的设置步骤。

## 步骤 5：准备硬件

您需要一台用于自助终端的平板电脑，以及可选的 Brother 标签打印机来打印姓名标签。

至少需要：
- **一台 Android 或 Amazon Fire 平板电脑**——请参阅[推荐平板电脑](../attendance/check-in.md#recommended-hardware)
- **一台 Brother 标签打印机**——推荐 QL-1110NWB，因其支持蓝牙和 WiFi
- **Brother DK-1201 标签**（1-1/7" x 3-1/2"）

:::warning
只有 Brother 标签打印机与 B1 Checkin 应用兼容。其他品牌的打印机将无法使用。
:::

## 步骤 6：运行测试签到

在周日早上之前，进行一次测试：

1. 在平板电脑上打开 B1 Checkin 应用
2. 选择您的校区和正确的礼拜时间
3. 搜索您添加的一个家庭
4. 为一个孩子签到并验证：
   - 出席记录显示在 B1 Admin 的**出席**部分
   - 如果使用打印机，姓名标签正确打印

:::tip
在正式启用之前，对您的迎宾团队志愿者进行签到流程培训。通常只需要 5 分钟的快速演示。
:::

## 完成！

您的儿童事工签到已就绪。家长可以搜索他们的家庭，选择孩子，并在自助终端签到。出席记录会自动记录在 B1 Admin 中。
