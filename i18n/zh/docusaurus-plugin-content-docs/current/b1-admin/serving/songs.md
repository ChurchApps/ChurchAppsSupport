# 诗歌

<div class="article-intro">

诗歌页面管理您的敬拜诗歌库。将所有诗歌详情、歌词和编排集中在一个地方，让您的敬拜团队能够轻松为即将到来的礼拜做准备并保持一致的曲目库。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在 B1 Admin 中导航到**事奉**部分以访问诗歌页面
- 可选地，设置好您的[礼拜计划](./plans.md)，以便可以将诗歌直接链接到[礼拜流程](./service-order.md)中

</div>

## 浏览您的诗歌库

打开**诗歌**页面时，您的诗歌以卡片形式显示。每张卡片显示诗歌的缩略图、标题、艺术家和时长。滚动浏览诗歌库或使用搜索功能查找所需内容。

## 搜索诗歌

1. 点击诗歌页面上的**搜索**按钮。
2. 输入诗歌**标题**或**艺术家**名称。
3. 结果会随您输入而筛选，显示诗歌库中匹配的诗歌。

## 添加诗歌

1. 在诗歌页面上点击**添加诗歌**。
2. 在外部诗歌数据库中按标题或艺术家搜索。
3. 选择您要添加的诗歌。
4. 该诗歌将被添加到您的诗歌库，基本详情已预先填写。

:::tip
添加诗歌后，点击它以查看和更新详细信息、添加编排，或附加外部链接如 YouTube 视频或和弦谱。
:::

## 查看和编辑诗歌

点击任意诗歌卡片以打开其详情页面。从这里您可以：

- **编辑诗歌详情**——更新标题、艺术家、时长和其他元数据。
- **管理编排**——每首诗歌可以有多个编排（例如不同的调号或版本）。根据需要添加、编辑或删除编排。
- **添加歌词**——输入或更新每个编排的歌词。
- **设置调号**——为每个编排指定音乐调号，以便团队知道如何准备。
- **添加外部链接**——附加链接到 YouTube 视频、和弦谱或其他平台上托管的乐谱等资源。

## 编排

一首诗歌可以有多个编排，以适应不同的礼拜或团队配置。每个编排包括：

- 用于显示或投影的**歌词**
- **调号**标注
- 指向额外资源的**外部链接**

:::info
诗歌库中的诗歌可以直接添加到计划中的[礼拜流程](./service-order.md)，将诗歌详情和编排链接到该特定礼拜。
:::
