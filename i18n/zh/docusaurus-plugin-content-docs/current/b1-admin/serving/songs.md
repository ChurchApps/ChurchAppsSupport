# 歌曲

<div class="article-intro">

歌曲页面管理您的敬拜歌曲库。将所有歌曲详情、歌词和编排保留在一个地方，以便您的敬拜团队可以轻松为即将进行的服务做准备并维持一致的曲目库。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 导航到 B1 Admin 中的**服务**部分以访问歌曲页面
- 可选地，设置您的[服务计划](./plans.md)，以便您可以直接将歌曲链接到[服务顺序](./service-order.md)中

</div>

## 浏览您的歌曲库

当您打开**歌曲**页面时，您的歌曲显示为卡片。每张卡片显示歌曲的缩略图、标题、艺术家和持续时间。滚动浏览库或使用搜索来找到您需要的内容。

## 搜索歌曲

1. 单击歌曲页面上的**搜索**按钮。
2. 输入歌曲**标题**或**艺术家**名称。
3. 当您输入时，结果将筛选，从您的库中显示匹配的歌曲。

## 添加歌曲

1. 在歌曲页面上单击**添加歌曲**。
2. 按标题或艺术家搜索外部歌曲数据库。
3. 选择您想要添加的歌曲。
4. 歌曲将添加到您的库中，其基本详情已预填充。

:::tip
添加歌曲后，单击它以查看和更新其详情、添加编排或附加外部链接，如 YouTube 视频或和弦图。
:::

## 查看和编辑歌曲

单击任何歌曲卡片以打开其详情页面。从这里您可以：

- **编辑歌曲详情** -- 更新标题、艺术家、持续时间和其他元数据。
- **管理编排** -- 每首歌曲可以有多个编排（例如，不同的调或版本）。根据需要添加、编辑或删除编排。
- **添加歌词** -- 为每个编排输入或更新歌词。
- **设置调性** -- 使用歌词旁的调性选择器为每个编排指定音乐调性。更改调性会立即转调显示的和弦，包括在编排页面上以及从服务计划打开歌曲时，因此您的团队始终看到他们实际演奏的调性中的和弦。
- **添加外部链接** -- 附加链接到资源，如 YouTube 视频、和弦图或其他地方托管的乐谱。

## 编排

编排代表歌曲的不同版本 -- 例如，较快的现代版本与较慢的原声版本，或不同调性中供不同团队演奏的相同歌曲。同一歌曲的每个编排共享歌曲标题和艺术家，但有自己的独立设置：

- **调性** -- 编排的音乐调性
- **BPM** -- 每分钟拍数的节奏
- **长度** -- 持续时间（秒）
- **节拍** -- 拍号（例如 4/4、3/4、6/8）
- **序列** -- 和弦/部分结构（例如引奏 → 诗节 → 副歌 → 桥接）
- **歌词** -- 用于显示或投影
- **外部链接** -- 和弦图、YouTube 录音、乐谱或其他资源

当您将歌曲添加到[服务顺序](./service-order.md)时，您可以选择为该特定服务使用哪个编排，因此您的团队始终排练和演奏正确的版本。

:::info
来自您库的歌曲可以直接添加到计划内的[服务顺序](./service-order.md)中，将歌曲详情和选定的编排链接到该特定服务。
:::

## 后续步骤

- 使用[任务](./tasks.md)与您的敬拜团队协调以进行歌曲准备和练习
