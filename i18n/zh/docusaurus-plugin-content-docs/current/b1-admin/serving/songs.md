# 歌曲

<div class="article-intro">

歌曲页面管理您的敬拜歌曲库。将所有歌曲详情、歌词和编排保存在一个地方，以便您的敬拜团队可以轻松为即将进行的服事做准备并维持一致的歌曲库。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 导航到 B1 Admin 中的 **Serving**（服事）部分以访问歌曲页面
- 可选地，设置您的[服事计划](./plans.md)，以便您可以直接在[服事顺序](./service-order.md)中链接歌曲

</div>

## 浏览您的歌曲库

当您打开 **Songs**（歌曲）页面时，您的歌曲以卡片的形式显示。每张卡片都显示歌曲的缩略图、标题、艺术家和时长。滚动浏览库或使用搜索找到您需要的内容。

## 搜索歌曲

1. 在歌曲页面上点击 **Search**（搜索）按钮。
2. 输入歌曲 **title**（标题）或 **artist**（艺术家）名称。
3. 结果将在您输入时过滤，显示您库中匹配的歌曲。

## 添加歌曲

1. 在歌曲页面上点击 **Add Song**（添加歌曲）。
2. 按标题或艺术家搜索外部歌曲数据库。
3. 选择您想添加的歌曲。
4. 歌曲将被添加到您的库中，其基本详情预先填充。

:::tip
添加歌曲后，点击它以查看和更新其详情、添加编排或附加外部链接，如 YouTube 视频或和弦图。
:::

## 查看和编辑歌曲

点击任何歌曲卡片以打开其详细页面。从这里您可以：

- **编辑歌曲详情** — 更新标题、艺术家、时长和其他元数据。
- **管理编排** — 每首歌曲可以有多个编排（例如，不同的音调或版本）。根据需要添加、编辑或移除编排。
- **添加歌词** — 为每个编排输入或更新歌词。
- **设置音调** — 为每个编排指定音乐音调，以便您的团队知道如何准备。
- **添加外部链接** — 附加资源链接，如 YouTube 视频、托管在其他地方的和弦图或乐谱。

## 编排

编排代表歌曲的不同版本 — 例如，更快的现代版本与更慢的原声版本，或同一首歌曲的不同音调版本。同一首歌曲的每个编排共享歌曲标题和艺术家，但有自己的独立设置：

- **Key**（音调）— 编排演奏的音乐音调
- **BPM** — 每分钟节拍数的速度
- **Length**（长度）— 时长，以秒为单位
- **Meter**（时拍）— 拍号（例如 4/4、3/4、6/8）
- **Sequence**（序列）— 和弦/部分结构（例如前奏 → 诗节 → 副歌 → 桥段）
- **Lyrics**（歌词）— 用于显示或投影
- **External links**（外部链接）— 和弦图、YouTube 录音、乐谱或其他资源

当您向[服事顺序](./service-order.md)添加歌曲时，您可以为该特定服事选择使用哪个编排，这样您的团队总是排练和演奏正确的版本。

:::info
您库中的歌曲可以直接添加到计划内的[服事顺序](./service-order.md)中，将歌曲详情和所选编排链接到该特定服事。
:::

## 后续步骤
