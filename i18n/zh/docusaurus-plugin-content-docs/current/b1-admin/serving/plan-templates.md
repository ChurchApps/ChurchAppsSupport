# 计划模板

<div class="article-intro">

计划模板让您保存计划的服事顺序或职位的可重用快照，并一键将其应用于未来的计划。与其每周重新构建相同的结构，您可以保存一次，然后将其邮票应用到所需的多个即将进行的计划。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 创建至少一个[服事计划](plans.md)，其中定义了服事顺序或职位
- 您需要访问 B1 Admin 中的服事区域

</div>

## 将计划保存为模板

1. 从 **Serving**（服事）区域打开现有计划。
2. 点击 **Templates**（模板）按钮（计划页面的顶部）。
3. 选择 **Save as Template**（保存为模板）。
4. 给模板起名（例如"Standard Sunday"或"Worship + Sermon"）。
5. 保存。模板会在该时刻捕获计划的当前服事顺序和职位。

## 管理模板

从任何计划类型页面上的 **Templates**（模板）按钮，您可以：

- **Rename**（重命名）— 更改模板的显示名称
- **Update from plan**（从计划更新）— 用计划的当前状态替换模板的内容（在您细化标准结构后很有用）
- **Apply**（应用）— 将模板邮票应用到一个或多个即将进行的计划
- **Delete**（删除）— 移除您不再需要的模板

## 将模板应用于计划

1. 点击计划类型页面上的 **Templates**（模板）按钮。
2. 点击您想使用的模板旁边的 **Apply**（应用）图标（复制图标）。
3. 通过检查他们的方框选择要应用模板的即将进行的计划。
4. 选择要复制的内容：
   - **Order of Service**（服事顺序）— 用模板的项目替换所选计划的服事顺序项目
   - **Positions**（职位）— 用模板的职位替换所选计划的职位（及其分派）
5. 点击 **Apply**（应用）。

:::warning
应用职位将替换所选计划上的所有现有职位和分派。在应用前，确保您还没有为这些职位安排志愿者。
:::

:::tip
当您想要一致的歌曲/讲道结构但志愿者每周分派不同时，应用不带职位的服事顺序。
:::

## 保持模板最新

如果您的标准服事结构发生变化，打开模板管理器并使用 **Update from plan**（从计划更新）从您的最新计划同步模板，而不是从头开始重建。
