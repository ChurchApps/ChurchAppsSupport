# 任务

<div class="article-intro">

任务让您可以将待办事项分配给教会中的人员或小组。无论是一次性待办事项还是定期职责，任务帮助您跟踪需要完成的工作以及由谁负责，确保不遗漏任何事项。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 确保您要分配任务的人员或[小组](../groups/creating-groups.md)已存在于 B1 Admin 中
- 导航到**事奉**部分以访问任务页面

</div>

## 查看任务

导航到**事奉**并打开**任务**页面。您将看到所有未完成任务的列表。每个任务显示标题、分配对象和状态。

:::tip
默认情况下，只显示未完成的任务。切换**显示已关闭**以同时查看已完成的任务。
:::

## 创建任务

1. 在任务页面上，点击**添加任务**。
2. 输入任务的**标题**。
3. 点击**分配给**框并输入人员姓名，或切换到**小组**并输入小组名称。
4. 点击**搜索**并从结果中选择人员或小组。
5. 添加**备注**，写下给分配对象的详细信息或说明。
6. 点击**保存**。

任务将出现在任务列表中，并在分配对象的仪表板和移动应用上可见。

## 管理任务

点击任意任务以打开其详情页面。从这里您可以：

- **更改状态**——使用状态下拉菜单在**未完成**和**已关闭**之间切换。
- **重新分配任务**——将分配对象更新为其他人员或小组。
- **添加备注**——为分配对象包含额外信息或更新。
- **编辑标题**——根据需要更新任务标题。

## 任务生命周期

1. 新任务以**未完成**状态开始。
2. 分配对象收到通知，可以从仪表板、任务页面或移动应用查看任务。
3. 工作完成后，分配对象（或管理员）将状态更改为**已关闭**。
4. 已关闭的任务将从默认视图中移除，但仍可通过启用**显示已关闭**来访问。

:::info
您可以自动化定期任务，使其按计划自动创建。有关设置详情，请参阅[自动化](./automations.md)。
:::
