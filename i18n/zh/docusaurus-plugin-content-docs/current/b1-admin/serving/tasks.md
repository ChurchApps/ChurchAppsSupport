# 任务

<div class="article-intro">

任务让你将行动项分配给你教会中的人员或小组。无论是一次性待办事项还是重复的责任，任务帮助你追踪需要完成什么以及谁负责，这样就不会有任何事情被忽视。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 确保你想要分配任务的人员或[小组](../groups/creating-groups.md)在 B1 Admin 中存在
- 导航到**服侍**部分来访问任务页面

</div>

## 查看任务

导航到**服侍**并打开**任务**页面。你会看到你的开放任务列表。每个任务显示其标题、被分配者和状态。

:::tip
默认情况下，仅显示开放任务。切换**显示已关闭**来查看已完成的任务。
:::

## 创建任务

1. 在任务页面上，单击**添加任务**。
2. 输入任务的**标题**。
3. 单击**分配给**框并输入一个人的名字，或切换到**小组**并输入小组名字。
4. 单击**搜索**并从结果中选择该人或小组。
5. 添加一个**备注**，其中包含被分配者的任何详情或说明。
6. 单击**保存**。

该任务现在将出现在任务列表中，并在被分配者的仪表板和移动应用上可见。

## 管理任务

单击任何任务来打开其详情页面。从这里你可以：

- **更改状态** -- 使用状态下拉菜单在**开放**和**已关闭**之间切换。
- **重新分配任务** -- 将被分配者更新为不同的人或小组。
- **添加备注** -- 为被分配者包含其他信息或更新。
- **编辑标题** -- 根据需要更新任务标题。

## 任务生命周期

1. 一个新任务开始时的状态是**开放**。
2. 被分配者会收到通知，并可以从他们的仪表板、任务页面或移动应用中查看任务。
3. 一旦工作完成，被分配者（或管理员）将状态更改为**已关闭**。
4. 已关闭的任务会从默认视图中移除，但仍可以通过启用**显示已关闭**来访问。

:::info
你可以自动化重复任务，使其按计划创建。详见[自动化](./automations.md)了解如何设置此功能。
:::

:::tip
当你需要将许多人移动通过相同的一组步骤时 -- 如新访客跟进或成员过程 -- 使用[工作流](./workflows.md)。工作流将任务转变为拖放板上的卡片，具有步骤、到期日期和路由。
:::
