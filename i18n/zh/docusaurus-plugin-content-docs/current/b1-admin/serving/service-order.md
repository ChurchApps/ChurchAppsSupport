# 礼拜流程

<div class="article-intro">

礼拜流程定义了敬拜礼拜中各项内容的顺序。使用它来安排从开场诗歌到结束祷告的所有环节，让您的整个团队提前了解礼拜的流程。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 为您要编排礼拜流程的日期[创建一个礼拜计划](./plans.md)
- 可选地，将[诗歌](./songs.md)添加到您的诗歌库中，以便可以直接链接到礼拜流程中

</div>

## 访问礼拜流程

1. 导航到**事奉**并选择您的事工。
2. 点击一个**计划类型**，然后打开一个具体的**计划**。
3. 在计划详情页面上点击**礼拜流程**标签。

## 编排您的礼拜流程

礼拜流程是按照礼拜期间发生顺序排列的项目列表。常见元素包括：

- 敬拜诗歌
- 祷告
- 经文朗读
- 公告
- 讲道或信息
- 奉献
- 结束

## 将项目添加到礼拜流程

1. 在**礼拜流程**标签上，点击**添加项目**。
2. 选择您要添加的元素类型（例如，来自诗歌库的诗歌、自定义项目或标题）。
3. 输入项目的详细信息，如标题和备注。
4. 点击**保存**。

:::tip
您可以直接从[诗歌](./songs.md)库中添加诗歌。这会将诗歌详情（包括歌词和编排）链接到您的礼拜计划中，让您的敬拜团队拥有所需的一切。
:::

## 重新排序项目

拖放项目以重新排列顺序。屏幕上显示的顺序就是您的团队在礼拜期间将遵循的顺序。

## 编辑和删除项目

- 点击礼拜流程中的任意项目以**编辑**其详细信息。
- 使用**删除**选项从流程中移除项目。

:::info
礼拜流程是针对每个计划的。您在此处所做的更改仅影响所选计划，不会改变其他计划或您的诗歌库。
:::

## 分享礼拜流程

完成礼拜流程编排后，您可以从计划详情页面打印完整计划（包括礼拜流程）。这为您的团队提供了完整的礼拜安排。

## 下一步
