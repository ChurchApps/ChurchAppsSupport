# 将教室电视连接到 B1 Admin 课程

<div class="article-intro">

您可以使用 **FreePlay**（ChurchApps 为教会提供的免费电视应用）将教室电视连接到 B1 Admin 课程。连接后，电视将显示您在 B1 Admin 中安排的课程内容，让教师可以直接在教室屏幕上使用。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在 B1 Admin 中至少安排一节课程（请参阅[安排课程](./scheduling-lessons.md)）
- 在您要使用的电视或 Fire Stick 上安装 **FreePlay**

</div>

## 第 1 步——在您的电视上安装 FreePlay

在您的教室显示设备上安装 **FreePlay** 应用：

- **Amazon Fire Stick / Fire TV**——在 Amazon Appstore 中搜索 **FreePlay**
- **Android TV / Google TV**——在 Google Play Store 中搜索 **FreePlay**
- **Apple TV**——在 App Store 中搜索 **FreePlay**

有关完整的设置说明，请参阅 [FreePlay 入门指南](/docs/freeplay/getting-started/)。

## 第 2 步——将 B1.church 添加为内容提供商

1. 在电视上打开 **FreePlay** 应用
2. 导航到**内容提供商**屏幕（在侧边栏中选择**提供商**图标）
3. 从可用提供商列表中找到并选择 **B1.church**
4. 按照屏幕提示使用您的 B1 教会凭据登录
5. FreePlay 显示**已连接！**表示连接成功

连接后，B1.church 将出现在您的 FreePlay 侧边栏中。FreePlay 将获取您在 B1 Admin 中安排的课程内容，以便教师可以直接在教室电视上播放。

:::tip
连接后，教师可以使用电视遥控器按照课程进度自行控制视频和幻灯片的切换。无需单独设置或使用 USB 驱动器。
:::

:::info
有关使用内容提供商进行身份验证的详细步骤（二维码、表单登录等），请参阅[连接提供商](/docs/freeplay/content-providers/connecting-providers)。
:::
