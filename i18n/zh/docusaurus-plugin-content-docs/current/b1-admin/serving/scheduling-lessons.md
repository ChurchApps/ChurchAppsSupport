# 从 Lessons.church 安排课程

<div class="article-intro">

B1 Admin 与 [Lessons.church](https://lessons.church) 直接集成，让您可以在服务计划中直接为教室安排课程。这将一切 -- 志愿者、分配和课程内容 -- 保存在一个地方。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在事工服务区域设置您的事工
- 拥有一个活跃的 [Lessons.church](https://lessons.church) 账户 -- 如果您的教会还没有，请先在那里注册

</div>

## 步骤 1 -- 链接您的 Lessons.church 账户

这是每个事工的一次性设置。您需要先连接 Lessons.church 账户，然后才能浏览和安排内容。

1. 登录 [B1 Admin](https://admin.b1.church/) 并转到**事工服务**
2. 打开您想要连接的事工（例如，儿童事工）
3. 向下滚动到**内容提供商账户**部分
4. 点击**链接新提供商**
5. 从列表中选择 **Lessons.church**
6. 将出现一个带有代码的设备授权屏幕
7. 前往 [lessons.church](https://lessons.church)，登录并输入代码以授权连接
8. 批准后，您将在 B1 Admin 中 Lessons.church 下看到**"账户已链接"**

:::info
内容提供商账户部分按事工划分。如果您管理多个事工（例如，儿童和青年），则需要为每个事工分别链接 Lessons.church。
:::

## 步骤 2 -- 安排课程

链接后，您可以直接从计划中安排课程。

1. 在 B1 Admin 中，转到**事工服务 → 计划**
2. 选择您的事工标签页并点击**添加计划** -- **安排课程**表单将打开
3. 选择课程的**日期**（默认为即将到来的周日）
4. 点击**选择课程** -- 将打开内容浏览对话框
5. 在对话框顶部，确认 **Lessons.church** 已被选为提供商
6. 浏览内容：
   - 选择一个**项目**（例如，"儿童圣经故事"）
   - 在该项目中选择一个**系列**（例如，"创世与早期故事"）
   - 选择具体的**课程**
   - 选择**场所** -- 这是课程的年龄组版本（例如，"学前班"、"小学"）
7. 点击**关联课程**以确认
8. 选择您的志愿者**复制选项**：
   - **不复制** -- 全新计划，不转移志愿者
   - **仅复制职位** -- 从上一个计划复制志愿者角色，但不复制分配的人员
   - **职位和分配** -- 同时复制角色和分配的志愿者 *（最常见）*
9. 点击**保存**

计划会自动创建并命名（例如，"2月23日 - 小学"）。志愿者可以打开计划查看他们的分配并在周日前预习课程内容。

:::warning
确保为您教室的年龄组选择正确的**场所**。选择错误的场所（例如，为小学班级选择学前班内容）意味着您的志愿者将看到为不同年龄段设计的内容。
:::

:::tip
**提前规划** -- 您可以一次安排多周的课程，以便您的团队提前准备。使用计划视图中的过往课程列表，避免意外重复内容。
:::

## 自定义课程内容

课程安排好后，您可以针对特定教室进行调整 -- 删除不适用的部分，隐藏您教室不使用的角色，或重新排列内容以匹配您偏好的流程。自定义可以仅保存给一间教室，也可以应用于教会的所有教室。

请参阅[自定义课程](/docs/lessons-church/customization/customizing-lessons)指南获取逐步说明。

## 使用 FreePlay 在教室电视上播放课程

在 B1 Admin 中安排课程与 **[FreePlay](/docs/freeplay/)** -- ChurchApps 为教室电视和 Fire Stick 提供的免费媒体播放器 -- 完美搭配。当您的计划设置好后，FreePlay 可以直接从 Lessons.church 拉取课程内容并在教室全屏播放。您的老师使用电视遥控器控制进度，随着课程进行在视频和幻灯片之间切换。

这意味着您的志愿者在手机上查看计划，而内容在教室电视上播放 -- 无需单独设置、无需U盘、无需手忙脚乱。

[了解如何将 FreePlay 连接到内容提供商 →](/docs/freeplay/content-providers/connecting-providers)

## 找不到您的课程提供商？

可用提供商列表正在不断增长。如果您的教会使用的课程提供商尚未在 B1 Admin 中显示，请联系我们，我们将努力将其添加进来。

您可以将以下消息直接复制发送给您的课程提供商 -- 一旦他们与我们联系，我们就会设置集成：

---

> **主题：ChurchApps 集成请求**
>
> 您好 [课程提供商团队]，
>
> 我们非常喜欢您的课程，每周都与我们的孩子们一起使用。我们还使用 ChurchApps 来管理志愿者和服务计划，并使用 FreePlay (freeplay.church) 直接在教室电视上播放课程内容。这对我们的老师来说是一个巨大的改变。
>
> 目前我们必须单独管理您的课程，但如果您与 ChurchApps 集成，我们可以直接在服务计划中安排您的课程，并通过 FreePlay 在教室屏幕上播放 -- 所有这些都无需离开我们已经在使用的工具。
>
> ChurchApps 已经与多家课程提供商合作，他们的团队也准备好与您合作。您能否联系他们 **support@churchapps.org** 开始对话？我们非常期待看到这一切实现！
>
> 谢谢！
