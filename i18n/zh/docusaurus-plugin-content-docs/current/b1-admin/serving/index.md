# 事工服务

<div class="article-intro">

事工服务版块是您协调志愿者和规划教会礼拜的中心。无论您是在组织主日崇拜、周中聚会还是特别活动，这里是您将一切整合在一起的地方，让您的团队确切知道谁在何时服务。

</div>

## 事工服务页面如何运作

当您打开**事工服务**页面时，您会看到您的事工以标签页的形式显示在顶部。每个事工都有自己的计划类型和团队，因此您可以独立管理多个事工。例如，您可能有敬拜、儿童事工和接待事工的独立标签页。

:::tip
如果您是管理员，请切换**显示全部**以同时查看所有事工。当您需要检查多个团队的分配情况时，这非常有用。
:::

## 子版块

事工服务版块包含三个关键区域：

- **[计划](./plans.md)** -- 为特定日期创建服务计划并将志愿者分配到团队。计划帮助您组织谁在何时服务，让每个人提前知道自己的角色。
- **[诗歌](./songs.md)** -- 管理您的敬拜诗歌库，包括歌词、编曲和外部链接。您的敬拜团队可以用它来准备即将到来的礼拜。
- **[任务](./tasks.md)** -- 为团队成员分配和跟踪任务。设置[自动化](./automations.md)来自动创建重复性任务，确保万无一失。

## 开始使用

1. 从 B1 Admin 的主菜单导航到**事工服务**。
2. 选择一个**事工标签页**以查看其计划类型和团队。
3. 选择一个计划类型开始[创建服务计划](./plans.md)，或探索[诗歌](./songs.md)和[任务](./tasks.md)子版块。

:::info
在创建计划之前，您需要在设置区域中设置您的事工和团队。有关配置服务结构的详细信息，请参阅设置文档。
:::
