# 计划概览

<div class="article-intro">

计划概览为您提供所有志愿者分派跨多个服事日期的鸟瞰图。您可以在一个网格中看到谁在即将进行的周内的每个职位上服事，而不是逐个打开每个计划 — 快速发现需要填补的任何空缺。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 在服事区域创建至少一个事工和计划类型
- 创建[服事计划](./plans.md)，包括日期和志愿者分派
- 确保您的志愿者已添加到您的[人员名录](../people/adding-people.md)

</div>

## 访问概览

1. 从 B1 Admin 的主菜单导航到 **Serving**（服事）。
2. 在页面顶部选择一个 **ministry tab**（事工标签页）。
3. 点击一个 **plan type**（计划类型）打开其计划列表。
4. 点击页面顶部附近的 **Overview**（概览）按钮。

## 阅读概览网格

概览显示一个网格，其中：

- **行**代表每个职位（例如"音乐：吉他"、"技术：投影"）按类别分组
- **列**代表即将进行的服事日期（例如"4 月 14 日"、"4 月 21 日"）
- **单元格**显示分派到该日期该职位的志愿者的名字

以 **红色** 突出显示的单元格是未分派的 — 尚未分派任何志愿者。这使您可以一眼看出人员配置空缺，而无需打开每个计划。

:::tip
志愿者名字以缩写格式显示（名字和姓氏首字母，例如"John D."），以便在您有许多职位时保持网格紧凑。
:::

## 直接从概览分派志愿者

您不需要打开单个计划来填充空槽。点击网格中的任何单元格以打开该职位和日期的分派面板。从那里您可以：

- 从您的团队中选择一个人分派到该职位
- 替换或移除现有分派
- 保存更改而无需离开概览

这使您可以在一次中完成整个日程表的人员配置 — 跨周和职位工作，无需进出单个计划。

## 过滤概览

您可以使用顶部的过滤器控件调整概览显示的内容：

- **Start Date / End Date**（开始日期/结束日期）— 默认情况下，概览显示未来 12 周。输入自定义日期以扩展或缩小范围。
- **Ministry**（事工）— 在不离开概览的情况下切换到不同的事工。
- **Plan Type**（计划类型）— 过滤到所选事工内的特定计划类型。
- **Gaps Only**（仅显示空缺）— 切换此选项以隐藏每个日期都已填充的行，这样您可以专注于仍需志愿者的职位。

进行更改后点击 **Filter** 以更新网格。

## 导出到 CSV

点击 **Export CSV** 以将当前网格下载为电子表格。导出包括所有职位和志愿者分派，适用于筛选的日期范围，使其易于与事工领导者分享或打印以供规划会议使用。

:::info
CSV 导出反映当前应用的任何过滤器 — 只有网格中显示的日期和计划类型会包括在下载中。
:::
