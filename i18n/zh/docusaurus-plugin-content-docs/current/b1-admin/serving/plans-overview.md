# 计划概览

<div class="article-intro">

计划概览为您提供了一个鸟瞰视图，可以同时查看多个服务日期的所有志愿者分配。与其逐个打开每个计划，您可以在单个网格中看到谁在每个职位上服侍，以及快速发现任何仍需要填补的空缺。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 在服侍区域创建至少一个事工和计划类型
- 创建带有日期和志愿者分配的[服务计划](./plans.md)
- 确保您的志愿者已添加到[人员目录](../people/adding-people.md)

</div>

## 访问概览

1. 从B1 Admin的主菜单导航到**服侍**。
2. 在页面顶部选择一个**事工选项卡**。
3. 单击**计划类型**以打开其计划列表。
4. 单击页面顶部附近的**概览**按钮。

## 阅读概览网格

概览显示一个网格，其中：

- **行**代表每个职位（例如，"音乐：吉他"、"技术：投影"），按类别分组
- **列**代表即将到来的服务日期（例如，"4月14日"、"4月21日"）
- **单元格**显示分配到该日期该职位的志愿者的名称

用**红色**突出显示的职位未填补 — 尚未分配志愿者。这使得您可以一目了然地看到员工配置空缺，而无需逐个打开每个计划。

:::tip
志愿者名称以缩写格式显示（名字和姓氏首字母，例如，"John D."），以在您有许多职位时保持网格的紧凑性。
:::

## 筛选概览

您可以使用顶部的筛选器控件来调整概览显示的内容：

- **开始日期/结束日期** — 默认情况下，概览显示未来12周。输入自定义日期以扩展或缩小范围。
- **事工** — 切换到不同的事工而不离开概览。
- **计划类型** — 筛选到所选事工内的特定计划类型。

进行更改后单击**筛选**以更新网格。

## 导出到CSV

单击**导出CSV**以将当前网格下载为电子表格。导出包括筛选日期范围的所有职位和志愿者分配，便于与事工领导者分享或打印用于规划会议。

:::info
CSV导出反映当前应用的任何筛选器 — 网格中显示的日期和计划类型才会包含在下载中。
:::
