# 计划概览

<div class="article-intro">

计划概览为您提供了在一次查看多个服务日期的所有志愿者分配的鸟瞰视图。与其打开每个计划，您可以在一个网格中看到在每个即将到来的周内谁在服务每个职位 -- 并快速发现仍需要填补的任何空位。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 在服务区创建至少一个事工和计划类型
- 创建[服务计划](./plans.md)，包括日期和志愿者分配
- 确保您的志愿者已添加到您的[人员目录](../people/adding-people.md)

</div>

## 访问概览

1. 从 B1 Admin 的主菜单导航到**服务**。
2. 选择页面顶部的**事工选项卡**。
3. 单击一个**计划类型**打开其计划列表。
4. 单击页面顶部附近的**概览**按钮。

## 阅读概览网格

概览显示一个网格，其中：

- **行**代表每个职位（例如"音乐：吉他"、"技术：投影"），按类别分组
- **列**代表即将到来的服务日期（例如"4 月 14 日"、"4 月 21 日"）
- **单元格**显示在该日期分配给该职位的志愿者的名称

以**红色**突出显示的单元格未填补 -- 尚未分配志愿者。这样可以轻松一目了然地看到人员配置缺口，而无需打开每个计划。

:::tip
志愿者名称以缩短格式显示（名字和姓氏首字母，例如"John D."），以在您有许多职位时保持网格紧凑。
:::

## 直接从概览分配志愿者

您无需打开单个计划来填补空位。单击网格中的任何单元格以打开该职位和日期的分配面板。从那里您可以：

- 从您的团队中选择一个人分配给该职位
- 单击已分配的任何人旁边的**删除**以将其从该位置删除
- 保存更改而无需离开概览

这使得在一次经过中为整个日程安排人员成为可能 -- 跨周和职位工作，而无需在单个计划中导航。

## 从概览自动排期

单击**自动排期**让 B1 一次填补当前网格中的每个打开的未填补的位置。对于视图中的每个计划，它从链接到每个职位的小组中提取候选人并自动填补空位，报告能够填补多少个可见计划。计划一次填补一个，因此同一志愿者在同一次经过中不会在两个日期上重复预订。

:::info
自动排期只填补已经为空的位置 -- 它从不替换现有分配。
:::

## 向所有排期的人发送电子邮件

单击**电子邮件志愿者**以一次操作向在当前筛选的日期范围和事工中分配的每个志愿者发送通知，而不是逐计划发送电子邮件。B1 报告发送了多少封电子邮件以及多少个失败。

## 突出显示志愿者的日程

使用**突出显示**下拉菜单从您的团队中选择一个人 -- 他们被分配的每个单元格都在网格中突出显示，因此您可以在将他们添加到另一个位置之前看到他们已经在服务的地方。选择**所有人**关闭突出显示。

## 筛选概览

您可以使用顶部的筛选控件调整概览显示的内容：

- **开始日期/结束日期** -- 默认情况下，概览显示未来 12 周。输入自定义日期以扩展或缩小范围。
- **事工** -- 切换到不同的事工而无需离开概览。
- **计划类型** -- 筛选到所选事工中的特定计划类型。
- **仅未填补** -- 打开此选项以隐藏每个日期都已填补的行，以便您只关注仍需要志愿者的职位。

进行更改后单击**筛选**以更新网格。

## 导出到 CSV

单击**导出 CSV**以将当前网格下载为电子表格。导出包括筛选的日期范围的所有职位和志愿者分配，使其易于与事工领导分享或为计划会议打印。

:::info
CSV 导出反映当前应用的任何筛选器 -- 网格中显示的只有日期和计划类型包含在下载中。
:::
