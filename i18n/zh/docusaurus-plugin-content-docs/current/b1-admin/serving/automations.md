# 自动化

<div class="article-intro">

自动化功能按照重复的时间表自动创建任务。无需每周或每月手动创建相同的任务，您只需设置一次自动化，让系统为您处理，确保万无一失。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 熟悉 B1 Admin 中[任务](./tasks.md)的工作方式
- 确定您希望自动化哪些重复性职责

</div>

## 查看自动化

导航到**事工服务**并打开**自动化**页面。您将看到所有自动化的列表，每个自动化都显示其标题、活动状态和重复模式。

## 创建自动化

1. 在自动化页面上，点击**添加自动化**。
2. 为自动化输入一个**标题**。它也将用作其创建的任务的标题。
3. 设置**重复模式** -- 选择任务应多久创建一次（例如，每周、每月或在特定日期）。
4. 将任务分配给每次运行时负责的**个人或群组**。
5. 将自动化设置为**活动**状态，使其开始按计划创建任务。
6. 点击**保存**。

## 编辑自动化

1. 点击列表中现有的自动化以打开它。
2. 更新标题、重复模式、分配或任何其他设置。
3. 点击**保存**以应用您的更改。

## 激活和停用

您可以控制自动化是否运行而无需删除它：

- **活动** -- 自动化将按其计划创建任务。
- **非活动** -- 自动化已暂停，在您重新激活之前不会创建任何新任务。

:::tip
在假期休息或不需要重复任务的特殊时段，将自动化设置为**非活动**状态。您可以随时重新激活它，而不会丢失配置。
:::

## 工作原理

当自动化运行时，它会使用配置的标题和分配创建一个新的[任务](./tasks.md)。该任务的行为与其他任何任务一样 -- 被分配人会收到通知，并可以从他们的仪表板或任务页面管理该任务。

:::info
自动化只会向前创建新任务。它们不会为过去的日期追溯创建任务。
:::
