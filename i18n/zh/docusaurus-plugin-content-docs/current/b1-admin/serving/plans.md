# 服务计划

<div class="article-intro">

服务计划组织谁在什么时候提供服务。每个计划与特定的日期和事工相关联，使您能够轻松地逐周协调志愿者团队，并确保每场服务都配备充足的人员。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 在事工区域设置您的事工和团队
- 确保志愿者已被添加到您的[人员目录](../people/adding-people.md)并分配到团队

</div>

## 访问计划

1. 从主菜单导航到**事工**。
2. 在页面顶部选择一个**事工选项卡**。
3. 单击**计划类型**以查看该类型的计划列表。
4. 单击特定计划以打开它。

:::info
查看计划不需要完整的管理员访问权限。作为事工成员的事工成员可以导航到事工并查看其事工的计划，无需管理员权限。
:::

## 创建计划

1. 从计划类型视图中，单击**新建计划**。
2. 为计划命名或使用日期作为名称。选择服务的**日期**。
3. 如果您想从之前的计划复制，请选择仅位置或位置和分配。如果不想复制，只需选择任何内容即可。您也可以从我的上一个计划复制服务顺序。
4. 保存计划。现在您可以开始分配团队成员并构建[服务顺序](./service-order.md)。

## 计划详情页面

当您打开计划时，您将看到两个选项卡：

- **分配** -- 管理分配给此计划的团队成员。您可以从现有团队中添加人员，并查看谁已确认或仍在待定中。
- **[服务顺序](./service-order.md)** -- 使用敬拜歌曲、祈祷、通知和讲道等元素构建服务顺序。

## 分配团队成员

1. 打开计划并转到**分配**选项卡。
2. 单击**添加位置**以展开它。填写"添加位置"表单中的信息。对于类别名称，添加您喜欢的任何类别。
3. 单击**所需人员**并选择志愿者来填充该位置。
4. 通过单击**添加**从您的团队花名册中添加成员。
5. 分配的成员将与其团队一起出现，并显示其分配状态。
6. 单击通知志愿者以在B1应用程序中或通过电子邮件通知他们。

:::tip
在创建计划之前在事工设置中设置您的团队。这样，您将拥有一个可随时分配的志愿者库。
:::

## 打印计划

您可以打印计划以分发给您的团队。打开计划，打开服务顺序选项卡，使用**打印**选项生成可打印版本，其中包括分配和服务顺序。这对于在排练中分发或在公共区域张贴很有用。

:::info
计划由事工组织。创建或查看计划之前，请确保您在正确的事工选项卡上。
:::

## 后续步骤

- 使用[计划概览](./plans-overview.md)在一个网格中查看多周的所有即将进行的分配，并发现未填充的位置
- 使用歌曲、阅读和其他元素构建您的[服务顺序](./service-order.md)
- 直接将库中的[歌曲](./songs.md)添加到服务顺序中
- 使用[任务](./tasks.md)向团队成员分配后续行动项
