# 服务计划

<div class="article-intro">

服务计划组织谁在何时服务。每个计划都与特定的日期和事工相关联，便于您逐周协调志愿者团队，确保每次礼拜都有充足的人员配置。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 在设置区域中设置您的事工和团队
- 确保志愿者已添加到您的[人员目录](../people/adding-people.md)并已分配到团队

</div>

## 访问计划

1. 从主菜单导航到**事工服务**。
2. 在页面顶部选择一个**事工标签页**。
3. 点击一个**计划类型**以查看该类型的计划列表。
4. 点击特定计划以打开它。

## 创建计划

1. 在计划类型视图中，点击**添加计划**。
2. 选择礼拜的**日期**。
3. 保存计划。现在您可以开始分配团队成员并构建[礼拜流程](./service-order.md)。

## 计划详情页面

当您打开一个计划时，您会看到两个标签页：

- **分配** -- 管理哪些团队成员被分配到此计划。您可以从现有团队中添加人员，并查看谁已确认或仍在等待中。
- **[礼拜流程](./service-order.md)** -- 构建礼拜流程，包括敬拜诗歌、祷告、公告和讲道等环节。

## 分配团队成员

1. 打开一个计划并转到**分配**标签页。
2. 点击一个**团队**以展开它。
3. 通过点击**添加**从您的团队名单中添加成员。
4. 已分配的成员将显示在其团队下方，并附有分配状态。

:::tip
在创建计划之前，先在事工设置中设置您的团队。这样，您将拥有一个可供分配的志愿者储备。
:::

## 打印计划

您可以打印计划以分发给您的团队。打开计划并使用**打印**选项生成包含分配和礼拜流程的可打印版本。这对于在排练时分发或张贴在公共区域非常有用。

:::info
计划按事工组织。在创建或查看计划之前，请确保您在正确的事工标签页上。
:::
