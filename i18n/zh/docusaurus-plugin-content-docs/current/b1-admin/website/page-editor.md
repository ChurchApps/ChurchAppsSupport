# 使用页面编辑器

<div class="article-intro">

B1 页面编辑器是一个可视化拖放构建器,可让您无需编写任何代码即可设计教会网站页面。您可以添加部分和内容块、自定义样式、预览您的工作并撤消更改 -- 全部在浏览器中完成。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 完成[初始设置](initial-setup)以配置您的网站
- 在[管理页面](managing-pages)中至少创建一个页面
- 您需要 **content.edit** 权限才能访问编辑器

</div>

## 打开编辑器

1. 在 B1 Admin 中,点击左侧菜单中的 **Website**。
2. 在页面表格中找到要编辑的页面并点击 **Edit**。

编辑器以全屏模式打开。左侧面板显示您的页面结构和可用的内容元素;中心区域显示页面的实时预览。

:::info
编辑器始终以浅色模式显示,无论您的 B1 Admin 主题设置如何。这可确保预览准确匹配页面对网站访客的外观。
:::

## 页面结构:部分和元素

每个页面都由两个级别构建:

- **Sections** -- 将页面划分为水平带的顶级容器(例如,英雄部分、内容块或页脚条)。在可以添加内容之前,每个页面必须至少有一个部分。
- **Elements** -- 放置在部分内的各个内容片段,例如文本、图像、按钮、卡片、表单和日历。

### 添加部分

1. 点击 **Add Section**(或左侧面板顶部的 **+** 按钮)。
2. 为您的部分选择布局 -- 选项包括单列、两列、三列等。
3. 新部分出现在预览中。点击它以选择它并配置其背景颜色、内边距和其他样式选项。

### 向部分添加元素

1. 在预览中点击部分内部以选择它。
2. 点击 **Add Content** 并从列表中选择元素类型:
   - **Text** -- 标题、段落和富文本
   - **Image** -- 上传或链接到照片
   - **Button** -- 可点击的号召性用语链接
   - **Card** -- 带有标题和描述的图像
   - **Form** -- 将[表单](../forms/creating-forms)直接嵌入页面
   - **Calendar** -- 显示活动日历
   - **FAQ** -- 手风琴式问答块
   - **Video** -- 通过 URL 嵌入视频
3. 使用出现的设置面板配置元素。

### 重新排序内容

使用每个项目左侧的手柄图标(六个点)拖动部分或元素以重新排序它们。您可以在部分内拖动元素或在部分之间移动它们。

## 设置页面样式

### 部分样式

点击任何部分以打开其样式面板。您可以设置:

- **Background** -- 纯色、渐变或图像
- **Padding** -- 部分内的上下间距
- **Width** -- 全宽或居中/包含

### 元素样式

点击任何元素以打开其样式面板。常见选项包括字体大小、颜色、对齐方式、边距和内边距。对于图像,您可以设置替代文本和链接目标。

### 自定义 CSS

对于高级样式,每个部分和元素都有一个 **Custom CSS** 字段,您可以在其中编写自己的 CSS 规则。这些规则的作用域限于该元素,因此它们不会无意中影响页面的其余部分。

:::tip
如果您需要在整个网站上应用样式 -- 例如自定义字体或全局颜色 -- 请使用[外观](appearance)设置,而不是在单个页面上使用自定义 CSS。
:::

## 预览您的页面

使用工具栏中的预览控件检查您的页面在不同屏幕尺寸下的外观:

- **Desktop** -- 全宽浏览器视图
- **Mobile** -- 窄手机尺寸视图

点击 **Preview** 在新浏览器选项卡中打开页面的实时版本,与访客看到的完全一样。

## 撤消更改

编辑器会自动跟踪您的编辑历史记录。使用工具栏按钮或键盘快捷键导航:

- **Undo**(Ctrl+Z / Cmd+Z)-- 撤消您的上一个操作
- **Redo**(Ctrl+Y / Cmd+Y)-- 重新应用撤消的操作

您还可以将页面恢复到较早的快照。点击工具栏中的 **History** 以查看带有描述的已保存快照列表,然后点击任何条目以恢复到该时间点。

:::warning
恢复快照会用快照版本替换您当前的页面内容。这无法使用标准撤消按钮撤消。如果要保留返回选项,请在恢复旧快照之前保存当前状态的快照。
:::

## 保存您的工作

在您工作时会自动保存更改。工具栏中的状态指示器显示您的更改是否已保存。您还可以随时点击 **Save** 以强制保存。
