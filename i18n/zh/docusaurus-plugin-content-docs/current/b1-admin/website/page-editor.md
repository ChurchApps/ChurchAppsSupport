# 使用页面编辑器

<div class="article-intro">

B1 页面编辑器是一个可视化拖放构建器，让你无需编写任何代码即可设计教会网站页面。你可以添加部分和内容块、自定义样式、预览你的工作和撤消更改——所有这些都在浏览器中进行。

</div>

<div class="prereqs">
<h4>开始前的准备</h4>

- 完成[初始设置](initial-setup)以配置你的网站
- 在[管理页面](managing-pages)中至少创建一个页面
- 你需要**content.edit**权限才能访问编辑器

</div>

## 打开编辑器

1. 在 B1 Admin 中，点击左侧菜单中的**网站**。
2. 在"页面"表中找到要编辑的页面，点击**编辑**。

编辑器以全屏模式打开。左侧面板显示你的页面结构和可用内容元素；中央区域显示你的页面的实时预览。

:::info
编辑器始终以浅色模式显示，无论你的 B1 Admin 主题设置如何。这确保预览准确匹配你的页面对网站访问者的外观。
:::

## 页面结构：部分和元素

每个页面由两个级别构建：

- **部分** - 将页面分为水平带的顶级容器（例如，英雄部分、内容块或页脚带）。每个页面必须至少有一个部分，然后才能添加内容。
- **元素** - 放在部分内的单个内容，如文本、图像、按钮、卡片、表格和日历。

### 添加部分

1. 点击**添加部分**（或左侧面板顶部的**+**按钮）。
2. 选择如何开始：
   - **从模板** - 浏览按类别（英雄、关于、服务、奉献等）组织的部分模板库，点击一个将其作为完全样式化、预填充的部分插入。添加后你可以自定义所有内容。
   - **空白部分** - 选择列布局（单列、两列、三列等）并从头开始构建。
3. 新部分出现在预览中。点击它来选择它并配置其背景颜色、填充和其他样式选项。

### 向部分添加元素

1. 在预览中点击部分内以选择它。
2. 点击**添加内容**并从列表中选择元素类型：
   - **文本** - 标题、段落和富文本
   - **图像** - 上传或链接到照片
   - **按钮** - 可点击的号召性行动链接
   - **卡片** - 带有标题和描述的图像
   - **表格** - 直接在页面上嵌入[表格](../forms/creating-forms)
   - **日历** - 显示事件日历
   - **常见问题** - 手风琴式问题和答案块
   - **视频** - 通过 URL 嵌入视频
   - **小组浏览器** - 所有教会小组的可过滤目录，支持可选的搜索、类别过滤和标签过滤
3. 使用出现的设置面板配置元素。

### 重新排序内容

使用每项左侧的句柄图标（六个点）拖动部分或元素以重新排序。你可以在部分内拖动元素或在部分之间移动它们。

## 样式化你的页面

### 部分样式

点击任何部分打开其样式面板。你可以设置：

- **背景** - 纯色、渐变或图像
- **填充** - 部分内的上下间距
- **宽度** - 全宽或居中/容器化

### 元素样式

点击任何元素打开其样式面板。常见选项包括字体大小、颜色、对齐、外边距和填充。对于图像，你可以设置替代文本和链接目标。

### 自定义 CSS

对于高级样式，每个部分和元素都有一个**自定义 CSS**字段，你可以在其中编写自己的 CSS 规则。这些规则限定到该元素，因此不会意外影响页面的其余部分。

:::tip
如果你需要在整个网站上应用样式——例如自定义字体或全局颜色——请使用[外观](appearance)设置而不是单个页面上的自定义 CSS。
:::

## 预览你的页面

使用工具栏中的预览控件来检查你的页面在不同屏幕尺寸下的外观：

- **桌面** - 全宽浏览器视图
- **移动** - 窄手机尺寸视图

点击**预览**在新浏览器标签中打开页面的实时版本，就像访问者将看到的那样。

## 撤消更改

编辑器自动跟踪你的编辑历史记录。使用工具栏按钮或键盘快捷键导航：

- **撤消**（Ctrl+Z / Cmd+Z） - 恢复你的最后一个操作
- **重做**（Ctrl+Y / Cmd+Y） - 重新应用已撤消的操作

你也可以将页面恢复到早期的快照。点击工具栏中的**历史记录**以查看带有描述的保存快照列表，并点击任何条目以恢复到该点。

:::warning
恢复快照会用快照版本替换你的当前页面内容。这不能用标准撤消按钮撤消。在恢复旧快照之前保存当前状态的快照，如果你想保留返回的选项。
:::

## 保存和发布

当你工作时，更改会自动保存。工具栏中的状态指示器显示你的更改是否已保存。

### 草稿和发布状态

页面可以有一个**发布**状态，控制访问者何时看到你的更改。工具栏显示一个状态芯片，显示当前状态：

- **保存时上线** - 页面不使用发布工作流。每个保存的更改都会立即上线。这是新页面的默认值。
- **未发布的更改** - 页面之前已发布，但你自上次发布以来进行了更改。访问者仍然看到以前发布的版本。
- **已发布** - 页面已上线，你的保存内容与访问者看到的内容相匹配。

要发布你的更改，点击工具栏中的**发布**按钮。页面立即上线。

要恢复到最后发布的版本而不影响访问者看到的内容，打开溢出菜单（⋮）并点击**放弃更改**。

要完全将页面下线，打开溢出菜单并点击**取消发布**。访问者将不再看到该页面，直到你重新发布它。

:::tip
当你想准备一个页面——例如，为即将发生的事件——并仅在正确的时刻上线时，使用草稿/发布工作流。构建和预览页面，然后在准备好时点击"发布"。
:::
