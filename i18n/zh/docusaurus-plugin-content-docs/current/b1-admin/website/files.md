# 文件

<div class="article-intro">

文件页面是教会网站上所有媒体和文档的集中存储库。从这里您可以上传、整理、浏览和管理网站所需的每个图片、文档和媒体文件。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 完成网站的[初始设置](initial-setup)
- 在上传之前优化图片以加快页面加载速度
- 以常见的网页格式（PNG、JPG、PDF 等）准备您的文件

</div>

## 上传文件

1. 在 B1 Admin 中导航到**网站**部分，点击**文件**标签。
2. 将文件直接拖放到上传区域，或点击上传按钮从您的电脑中浏览选择。
3. 您的文件将立即上传并出现在文件库中。

:::tip
为获得最佳效果，请在上传前优化图片。较小的文件大小有助于网站更快地为访客加载。TinyPNG 或 Squoosh 等工具可以在不明显损失质量的情况下压缩图片。
:::

## 使用文件夹整理

1. 创建文件夹将相关文件分组——例如"讲道图片"、"活动横幅"或"同工照片"。
2. 在文件夹之间移动文件，随着文件库增长保持整洁。
3. 使用清晰、描述性的文件夹名称，以便团队能够快速找到文件。

## 浏览和搜索

- 滚动文件库以查看所有上传文件的缩略图。
- 当文件库变大时，使用**搜索**功能按名称查找文件。
- 点击任意文件以查看其详细信息，包括文件名、大小和链接。

## 在网站上使用文件

1. 点击文件以查看其详细信息。
2. **复制文件链接**以在网站页面中使用。您可以将此链接粘贴到图片元素、下载链接或任何接受文件链接的地方。
3. 您在此上传的文件可在所有网站页面和内容块中使用。

:::info
在[页面管理](managing-pages)中建设页面时，您可以通过链接引用此处上传的任何文件。这样所有媒体都集中整理在一个地方。
:::

## 删除文件

1. 选择您要删除的文件。
2. 点击**删除**按钮将其从文件库中永久删除。
3. 在删除之前，请确保该文件当前未在任何页面上使用，因为这会导致引用它的链接或图片失效。

:::warning
已删除的文件无法恢复。在删除之前，请仔细确认该文件不再在任何页面上使用。
:::
