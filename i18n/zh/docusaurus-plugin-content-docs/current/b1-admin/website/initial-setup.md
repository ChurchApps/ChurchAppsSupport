# 初始设置

<div class="article-intro">

每个 B1 帐户都附带一个即用型网站。本指南将引导您完成教会域名设置、网站外观配置、创建第一个页面以及组织导航。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个拥有管理员权限的 B1.church 帐户
- 如果使用自定义域名，请准备好您的 DNS 提供商登录凭据（例如 GoDaddy、Cloudflare 或 AWS）
- 准备好 PNG 格式的教会标志，带透明背景以获得最佳效果

</div>

## 设置您的域名

您的教会将自动获得一个 B1.church 上的子域名（例如 `yourchurch.b1.church`）。您也可以将自己的自定义域名指向您的 B1 网站。

1. 访问 admin.b1.church 或点击个人头像下拉菜单选择**切换应用**，前往 **B1.church Admin**。
2. 点击左侧边栏的**仪表板**，然后从下拉菜单中选择**设置**。
3. 点击**管理**查看您的子域名。将其设置为简短易记且不含空格的名称。
4. 要使用自定义域名，请登录您的 DNS 提供商（如 GoDaddy、Cloudflare 或 AWS）并添加两条记录：
   - 一条根域名的 **A 记录**，指向 `3.23.251.61`
   - 一条 `www` 的 **CNAME 记录**，指向 `proxy.b1.church`
5. 返回 B1.church Admin，将您的自定义域名添加到列表中，点击**添加**然后**保存**。几分钟内您的网站就可以通过自定义域名访问了。

:::tip
如果您没有看到设置选项，请让设置教会帐户的人授予您"编辑教会设置"权限。详情请参阅[角色与权限](../settings/roles-permissions.md)。
:::

## 创建您的第一个页面

1. 在 B1 Admin 中，点击左侧菜单的**网站**以打开网站页面视图。
2. 点击右上角的**添加页面**。
3. 选择**空白**作为页面类型并将其命名为"首页"。
4. 点击**页面设置**，将 URL 路径设置为 `/`（仅一个斜杠，不含文字）作为首页。其他页面使用 `/page-name`。
5. 点击**编辑内容**开始建设。每个页面必须以一个**版块**开始——这是所有其他元素的容器。
6. 添加版块后，再次点击**添加内容**，通过拖放将文字、图片、视频、卡片、表单等插入您的版块中。

:::info
有关页面、导航和页面类型的详细说明，请参阅[页面管理](managing-pages)。
:::

## 配置网站外观

1. 在网站页面视图中，点击顶部的**外观**标签。
2. 使用**调色板**设置您的品牌颜色，包括主色、次色和强调色调。
3. 在**字体排版设置**中，从字体浏览器中选择标题和正文字体。
4. 在样式设置中的**标志**下上传您的教会标志。提供浅色背景和深色背景两个版本。
5. 配置**网站页脚**，填入教会的联系信息和链接。

:::info
您在外观中所做的更改将应用到整个网站。有关每项设置的详细说明，请参阅[外观](appearance)页面。
:::

## 设置导航

您的导航链接显示在网站页面视图的左侧边栏中。要组织它们：

1. 点击**添加**创建新的导航链接，将其指向您的某个页面。
2. 拖放链接以重新排序，或将其嵌套在父级项目下以创建下拉菜单。
3. 预览您的网站以确认导航显示正确。
