# 网站

<div class="article-intro">

B1 Admin 中的网站版块为您的教会提供了一个功能齐全的网站建设工具。您可以创建页面、自定义网站的外观和风格、管理媒体文件以及发布活动日历——全部无需编写任何代码。

</div>

## 您可以做什么

网站建设工具分为五个主要区域：

### 页面

创建和管理教会网站上的所有页面。**页面**表格显示每个页面及其状态——**自动生成**（由系统自动创建）或**自定义**（由您创建）。您可以添加新的自定义页面、编辑现有页面，或将自动生成的页面转换为自定义页面以完全控制其内容。

左侧边栏以树状结构显示您的导航链接。您可以使用拖放来重新排列链接，设置您希望访客看到的确切层次和顺序。

### 区块

建立可重复使用的内容组件，可以放置在多个页面上。当您有在多个地方出现的内容（如行动号召、公告横幅或联系版块）时，区块可以节省您的时间。更新一次区块，更改就会应用到所有使用它的地方。

### 外观

自定义网站的主题、颜色、字体、标志和间距。您可以应用预设样式主题快速开始，或使用调色板编辑器、字体排版设置和自定义 CSS 来微调每个细节。详情请参阅[外观](appearance)指南。

### 文件

在一个集中位置上传和整理所有媒体资源——图片、文档和其他文件。使用文件夹保持整洁，并复制文件链接在网站任何位置使用。详情请参阅[文件](files)。

### 日历

在网站上发布活动日历，让访客了解您教会正在举办的活动。创建精选日历，将多个小组的活动汇集到一个视图中。

:::tip
如果您是第一次配置网站，请从[初始设置](initial-setup)开始。它将引导您完成域名设置、创建第一个页面以及配置网站外观。
:::
