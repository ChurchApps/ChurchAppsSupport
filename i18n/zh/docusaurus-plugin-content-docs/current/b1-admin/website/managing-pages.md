# 页面管理

<div class="article-intro">

网站页面视图是您创建、编辑和组织教会网站所有页面的中心枢纽。您可以在这个页面上同时管理页面内容和网站导航。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 完成[初始设置](initial-setup)以配置您的域名和基本网站设置
- 准备好您的内容和图片。请先使用[文件](files)管理器上传媒体资源。

</div>

## 了解页面类型

**页面**表格列出您网站上的每个页面及其状态：

- **自动生成**——由系统根据教会数据自动创建的页面（例如小组页面或讲道页面）。这些页面会随着数据变化自动更新。
- **自定义**——您使用自己的内容和布局创建的页面。

如果您想完全控制某个自动生成页面的内容和设计，可以将其转换为自定义页面。

## 添加和编辑页面

1. 点击页面表格右上角的**添加页面**按钮。
2. 选择页面类型（空白或模板）并为其命名。
3. 点击任意页面旁边的**编辑**以打开页面建设器，在其中添加版块、文字、图片和其他元素。
4. 点击**页面设置**以更新页面标题、URL 路径和其他元数据。
5. 使用**预览**按钮在新窗口中打开页面，查看访客将看到的确切效果。

:::tip
对于首页，将 URL 路径设置为 `/` 即可。对于所有其他页面，使用描述性路径如 `/about` 或 `/contact`。
:::

## 管理导航

网站页面视图的左侧边栏显示您的导航链接。这些链接控制访客在您网站上看到的菜单。

1. 点击**添加**创建新的导航链接。您可以将其指向网站上的任何页面或外部链接。
2. 要重新排序链接，拖放它们到您想要的顺序。您也可以将链接嵌套在父级项目下以创建下拉菜单。
3. 点击任意链接旁边的**编辑**图标以更改其标签、链接或位置。
4. 要从导航中移除链接，点击**删除**图标。

:::info
移除导航链接不会删除页面本身。页面仍然存在并可以通过其链接直接访问——只是不会出现在菜单中。
:::

## 网站组织技巧

- 将顶级导航保持在五到六个项目以内，以便访客能够快速找到内容。
- 为相关子页面使用嵌套链接（例如"关于"下拉菜单包含"我们的团队"、"信仰声明"和"历史"）。
- 通过点击**移动端预览**在移动设备上检查导航，确保在较小屏幕上效果良好。
- 为页面使用清晰、描述性的名称，帮助访客了解他们将找到什么内容。

:::tip
您可以在页面中添加[表单](../forms/creating-forms.md)，以收集访客的报名注册、祷告请求或其他信息。
:::
