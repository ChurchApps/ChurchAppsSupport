# 管理页面

<div class="article-intro">

网站页面视图是创建、编辑和组织教会网站上所有页面的中央枢纽。你可以从单一界面管理页面内容和网站导航。

</div>

<div class="prereqs">
<h4>开始前的准备</h4>

- 完成[初始设置](initial-setup)以配置你的域名和基本网站设置
- 准备好内容和图像。首先使用[文件](files)管理器上传媒体资源。

</div>

## 理解页面类型

**页面**表列出网站上的每个页面及其状态：

- **生成的页面** - 系统基于你的教会数据自动创建的页面（例如小组页面或讲道页面）。这些页面会随着你的数据变化而自动更新。
- **自定义页面** - 你用自己的内容和布局创建的页面。

你可以将任何自动生成的页面转换为自定义页面，以获得对其内容和设计的完全控制。

## 添加和编辑页面

1. 点击"页面"表右上角的**添加页面**按钮。
2. 选择页面类型（空白或模板）并给它起一个名称。
3. 点击任何页面旁的**编辑**以打开[页面编辑器](page-editor)，在那里你可以添加部分、文本、图像和其他元素。
4. 点击**页面设置**来更新页面标题、URL 路径和其他元数据。
5. 使用**预览**按钮在新窗口中打开你的页面，准确查看访问者将如何看到它。

:::tip
对于你的主页，将 URL 路径设置为 `/`。对于所有其他页面，使用描述性路径，如 `/about` 或 `/contact`。
:::

## 管理导航

网站页面视图的左侧栏显示你的导航链接。这些链接控制访问者在你的网站上看到的菜单。

1. 点击**添加**来创建新的导航链接。你可以将其指向网站上的任何页面或外部 URL。
2. 要重新排序链接，将它们拖放到你想要的顺序。你也可以将链接嵌套在父项目下来创建下拉菜单。
3. 点击任何链接旁的**编辑**图标来更改其标签、URL 或位置。
4. 要从导航中移除链接，点击**删除**图标。

:::info
移除导航链接不会删除页面本身。页面仍然存在，可以通过其 URL 直接访问——它只是不会显示在菜单中。
:::

## 组织网站的提示

- 将顶级导航保持在五到六项左右，以便访问者快速找到内容。
- 对相关的子页面使用嵌套链接（例如，带有"我们的团队"、"信念"和"历史"的"关于"下拉菜单）。
- 通过点击**移动预览**在移动设备上查看导航，确保在较小屏幕上正常工作。
- 给页面清晰、描述性的名称，帮助访问者理解他们将找到什么。

:::tip
你可以在页面上添加[表格](../forms/creating-forms.md)来收集访问者的注册、祷告请求或其他信息。
:::

## 从网站模板开始

如果你从头开始构建网站，你可以使用**网站模板**而不是逐个创建页面来快速启动。网站模板创建一组预建页面——主页、关于、连接、奉献等——已经准备好内容和导航链接。

1. 在"页面"屏幕上，点击**网站模板**按钮（在**添加页面**按钮旁）。
2. 浏览可用模板并点击一个来预览其页面结构。
3. 当你找到喜欢的模板时，点击**应用模板**。
4. 不存在的页面会被创建并添加到你的导航。现有页面保持原样。

应用模板后，在[页面编辑器](page-editor)中打开每个页面，用你的教会的真实内容替换占位符文本和图像。

:::info
网站模板创建页面结构和导航。它们不会覆盖你网站的配色方案或字体——这些由[外观](appearance)控制。
:::

## 图像灯箱

当访问者点击你网站上的图像时，它会在全屏灯箱叠加层中打开。这让人们可以在不离开页面的情况下以更大的尺寸查看照片。无需配置——灯箱会自动为页面内容中的图像启用。
