# 创建表单

<div class="article-intro">

构建自定义表单以从您的会众收集信息。您可以创建用于活动注册、调查、访客卡、会员申请等的表单。表单可以链接到数据库中的人员记录，也可以作为具有独立公开 URL 的独立页面使用。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 对于**人员**表单（链接到人员记录），您需要先在[数据库中添加人员](../people/adding-people.md)。
- 对于收集**付款**的表单，您必须[配置 Stripe 以进行在线奉献](../donations/online-giving-setup.md)。

</div>

## 创建新表单

1. 从主菜单导航到 **Forms**。
2. 点击 **Add Form**。
3. 输入表单的**名称**。
4. 从下拉菜单中选择表单类型：
   - **People** — 将提交与数据库中的[人员记录](../people/adding-people.md)关联。
   - **Stand Alone** — 创建具有独立公开 URL 的独立表单，适合外部注册。
5. 点击 **Save** 创建表单。

您的新表单将出现在列表中。点击它开始添加问题。

## 添加问题

1. 打开您的表单并转到 **Questions** 选项卡。
2. 点击 **Add Question**。
3. 从 Provider 下拉菜单中选择**字段类型**。可用类型包括：
   - **Textbox** — 用于简短文本回答
   - **Date** — 用于选择日期
   - **Email** — 用于电子邮件地址
   - **Phone Number** — 用于电话号码输入
   - **Multiple Choice** — 用于从预定义选项中选择
   - **Payment** — 用于收取付款
4. 输入问题的**标题**和可选的**描述**。
5. 如果该字段为必填项，请勾选 **Require an answer**。
6. 点击 **Save**。
7. 重复以上步骤添加更多问题。

:::warning
**Payment** 字段类型需要配置 Stripe。如果您尚未设置在线奉献，请在添加付款字段之前参阅[在线奉献设置](../donations/online-giving-setup.md)。
:::

## 管理表单成员

1. 打开您的表单并转到 **Members** 选项卡。
2. 搜索人员并为其分配角色：
   - **Admin** — 可以编辑表单并查看所有提交。
   - **View Only** — 可以查看提交但不能编辑表单。

## 配置表单属性

您可以随时更新表单的名称和设置。对于 Stand Alone 表单，您还会看到一个独特的**公开 URL**，可以与任何人分享。

:::tip
Stand Alone 表单非常适合活动注册。通过电子邮件、社交媒体分享公开 URL，或将表单直接嵌入您的教会网站。
:::

:::info
要在您的 B1 网站上嵌入表单，请转到网站编辑器，添加新部分，然后选择 **Form** 元素。然后选择您想要显示的表单。有关编辑网站的详细信息，请参阅[管理页面](../website/managing-pages.md)。
:::
