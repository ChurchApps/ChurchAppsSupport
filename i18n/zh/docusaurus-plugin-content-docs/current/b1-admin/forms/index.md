# 表单

<div class="article-intro">

表单部分允许您创建自定义表单，用于注册、调查、数据收集等。无论您需要访客卡、活动报名还是代祷请求表单，都可以在这里创建，并通过公开链接或您的 B1.church 网站与您的会众分享。

</div>

## 表单页面

当您打开**表单**时，您将看到两个选项卡：

- **Forms** -- 显示所有活动表单。
- **Archived** -- 显示已归档的表单。您可以随时恢复已归档的表单。

列表中的每个表单都会显示其名称，如果已配置，还会显示一个可以对外分享的公开 URL。

## 您可以执行的操作

在表单页面，您可以：

- **添加表单** -- 点击 **Add Form** 创建新的自定义表单。
- **编辑表单** -- 点击表单名称以打开并修改其问题、成员或设置。
- **归档表单** -- 将不再需要的表单移至**归档**选项卡，以保持列表整洁。
- **恢复表单** -- 切换到**归档**选项卡，恢复您想重新激活的任何表单。

## 使用表单

点击任意表单名称以打开它。表单编辑器有三个选项卡：

- **Questions** -- 通过添加不同类型的字段来构建表单结构。详情请参阅[创建表单](./creating-forms.md)。
- **Members** -- 控制谁可以访问和管理表单。
- **Submissions** -- 查看已提交的回复。详情请参阅[管理提交](./managing-submissions.md)。

:::tip
具有公开 URL 的表单可以通过链接分享、嵌入您的网站或包含在电子邮件中。这使得从可能没有教会账户的人那里收集信息变得轻松。
:::

:::info
您可以使用 **Members** 选项卡来控制谁有权访问每个表单。分配**管理员**或**仅查看**角色，为您的团队提供适当的访问级别。
:::
