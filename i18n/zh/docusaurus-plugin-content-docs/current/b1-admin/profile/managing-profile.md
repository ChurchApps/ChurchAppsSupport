# 管理您的个人资料

<div class="article-intro">

个人资料页面允许您编辑个人账户设置。您的个人资料在所有 ChurchApps 产品之间共享，因此您在此处所做的任何更改都将反映在您使用 ChurchApps 的所有地方。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您必须登录到您的 ChurchApps 账户
- 如果您计划更新电子邮件地址或密码，请提前准备好

</div>

## 个人资料信息

个人资料页面的顶部包含您的基本账户详情：

1. **电子邮件** -- 用于登录和接收通知的电子邮件地址。
2. **名字**和**姓氏** -- 您在 ChurchApps 中的显示名称。
3. **密码** -- 更新您的账户密码。

要更新您的信息，请编辑字段并点击 **Save Changes** 按钮。

## 更改密码

1. 滚动到个人资料页面上的密码字段。
2. 在 **Password** 字段中输入您的新密码。
3. 在 **Confirm Password** 字段中重新输入。
4. 点击 **Save Changes**。

:::info
您的密码必须至少为 8 个字符。确保密码和确认字段完全匹配。
:::

## 关联账户

**关联账户**部分显示与您的 ChurchApps 账户连接的任何外部服务。您可以查看哪些账户已关联，并从此部分管理这些连接。

## 主题偏好

ChurchApps 支持浅色和深色两种显示模式：

1. 在您的个人资料页面上找到 **Theme** 切换开关。
2. 在 **Light** 和 **Dark** 模式之间切换。
3. 界面将立即更新以反映您的选择。

您的主题偏好将保存到您的账户中，并将应用于您的所有会话。

## 账户删除

在个人资料页面底部，您会找到永久删除账户的选项。此操作将删除您的账户及相关数据，且无法撤销。

:::warning
账户删除是永久性的，无法恢复。在继续之前，请确保您确实想要删除账户。
:::

:::info
如果您是 **Domain Admin**，请确保在删除账户之前将管理职责转移给其他用户。有关管理管理员角色的详细信息，请参阅[角色和权限](../settings/roles-permissions.md)。
:::

:::info
在演示模式下，个人资料编辑功能已禁用。如果您使用的是演示账户，将无法更改个人资料设置。
:::
