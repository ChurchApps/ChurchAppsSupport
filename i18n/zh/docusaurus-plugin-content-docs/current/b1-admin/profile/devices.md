# 管理设备

<div class="article-intro">

设备页面允许您管理连接到您账户的 ChurchAppsPlayer 设备。这些设备用于流媒体传输内容和在您所在位置的屏幕上显示教会媒体。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 确保 ChurchAppsPlayer 设备已开启并连接到互联网
- 记下设备屏幕上显示的配对代码
- 如果您尚未查看，请先查看您的[个人资料设置](./managing-profile.md)

</div>

## 查看您的设备

打开设备页面时，您将看到一个表格，列出所有已注册的设备及以下信息：

- **标签** -- 您为设备分配的名称。
- **注册日期** -- 设备首次与您的账户配对的时间。
- **最后活跃日期** -- 设备最近一次连接到您账户的时间。

## 添加新设备

1. 点击设备页面顶部的 **Add Device** 按钮。
2. 系统将提示您输入**配对代码**。
3. 找到您要连接的 ChurchAppsPlayer 设备上显示的配对代码。
4. 输入代码并确认。
5. 成功配对后，设备将出现在您的设备表格中。

:::tip
在尝试配对之前，请确保 ChurchAppsPlayer 设备已开启并连接到互联网。配对代码显示在设备屏幕上。
:::

## 编辑设备标签

1. 在设备表格中点击**设备名称**。
2. 将标签更新为描述性名称（例如，"主堂主屏幕"或"大厅显示屏"）。
3. 保存您的更改。

:::tip
使用清晰、描述性的标签可以轻松识别每个设备，尤其是当您的教会设置了多个屏幕时。
:::

## 管理设备访问权限

使用设备页面跟踪哪些设备有权访问您教会的流媒体和内容。如果某个设备不再使用，您可以将其从账户中移除，以保持设备列表的整洁和安全。

:::warning
定期检查您的已连接设备，确保只有活跃的、经授权的设备链接到您的账户。移除不再使用的设备以维护安全性。
:::
