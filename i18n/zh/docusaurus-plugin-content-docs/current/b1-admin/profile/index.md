# 个人资料

<div class="article-intro">

个人资料部分允许您管理 ChurchApps 中的个人账户设置。您的个人资料在所有 ChurchApps 工具之间共享，因此您在此处所做的更改将应用于所有地方——从 B1 Admin 到 B1 移动应用。

</div>

## 您可以做什么

在个人资料部分，您可以：

- **更新您的姓名和电子邮件** -- 保持您的联系信息为最新状态。
- **更改密码** -- 为您的账户设置新密码。
- **切换主题** -- 根据您的偏好在浅色和深色模式之间切换。
- **管理已连接的设备** -- 查看和控制哪些 ChurchAppsPlayer 设备链接到您的账户。
- **查看关联账户** -- 查看和管理与您的 ChurchApps 账户关联的任何外部服务。

## 个人资料子部分

个人资料区域包含以下页面：

1. **[管理您的个人资料](./managing-profile.md)** -- 编辑您的个人信息、更改密码、管理关联账户、调整主题偏好以及处理账户删除。
2. **[管理设备](./devices.md)** -- 查看、添加和管理与您账户配对的 ChurchAppsPlayer 设备。

:::tip
请保持您的电子邮件地址为最新状态。它用于账户恢复和来自您教会的重要通知。
:::
