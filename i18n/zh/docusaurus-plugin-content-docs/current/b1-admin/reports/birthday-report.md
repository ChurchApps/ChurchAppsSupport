# 生日报告

<div class="article-intro">

生日报告帮助您通过显示即将过生日的会众来保持与他们的联系。使用它来策划庆祝活动、寄送贺卡或在礼拜中进行公告。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 确保每个人的个人档案中已输入出生日期。详情请参阅[添加人员](../people/adding-people.md)。
- 您需要相应的[权限](../settings/roles-permissions.md)才能访问报告

</div>

## 运行报告

1. 从主菜单导航到**报告**。
2. 点击**生日报告**卡片。
3. 使用日期筛选器设置**日期范围**，以确定您要查看哪些生日。
4. 报告将显示该范围内有生日的会众列表。

## 报告显示内容

生日报告显示：

- **会员姓名**——来自您教会数据库中的人员姓名。
- **出生日期**——他们的生日日期。

结果按您选择的日期范围列出，方便查看谁即将过生日。

## 实际用途

- **每周公告**——运行下一周的报告，将生日纳入教会公告或通知中。
- **生日贺卡**——生成当月列表，让团队寄送个人生日贺卡。
- **小组组长**——小组组长可以运行报告来记住其小组成员的生日。
- **里程碑庆祝**——按特定日期范围筛选，策划里程碑生日庆祝活动。

:::tip
每月初运行此报告以提前规划。您可以将日期范围设置为覆盖整个月份，以便有时间准备贺卡或公告。
:::

:::info
生日数据来自每个人个人档案中的出生日期字段。确保您的[会员记录](../people/adding-people.md)保持最新，以获得准确结果。
:::
