# 考勤报告

<div class="article-intro">

B1 Admin 提供三种考勤报告，帮助您了解人们参与礼拜和小组活动的情况。每种报告从不同角度呈现您的考勤数据，从高层次趋势到每日明细。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 确保您的礼拜和小组[持续进行考勤跟踪](../attendance/tracking-attendance.md)
- 确保您的[小组](../groups/creating-groups.md)和礼拜已在 B1 Admin 中配置
- 您需要相应的[权限](../settings/roles-permissions.md)才能访问报告

</div>

## 考勤趋势

考勤趋势报告显示您的礼拜出席人数随时间的变化情况。

1. 导航到 **Reports** 并点击 **Attendance Trend** 卡片。
2. 设置**日期范围**以定义时间段。
3. 选择一个**礼拜**或**小组**来筛选结果。
4. 报告将显示一条趋势线，展示所选时间段内的出席人数。

此报告有助于发现模式，如季节性下降、增长趋势或特殊活动的影响。

## 小组考勤

小组考勤报告显示所选日期范围内各小组的出席总数。

1. 导航到 **Reports** 并点击 **Group Attendance** 卡片。
2. 设置报告的**日期范围**。
3. 选择您要查看的**小组**。
4. 报告将显示每个小组的总出席数据。

使用此报告比较各小组的出席情况，确定哪些小组在增长或需要关注。

## 每日小组考勤

每日小组考勤报告提供您小组考勤数据的逐日明细。

1. 导航到 **Reports** 并点击 **Daily Group Attendance** 卡片。
2. 设置报告的**日期范围**。
3. 选择您要查看的**小组**。
4. 报告显示所选范围内每个单独日期的出席数据。

此报告提供细粒度的详细信息，有助于了解每周的变化或识别出席异常高或低的特定日期。

:::tip
使用考勤趋势报告进行高层次概览，使用每日小组考勤报告深入研究特定日期。
:::

## 实际用途

- **规划** -- 使用考勤趋势为即将到来的礼拜规划座位、人员和资源。
- **外展** -- 尽早发现出席下降的模式，以便及时跟进成员。
- **领导报告** -- 在定期领导报告中包含考勤数据，以展示事工健康状况。
- **活动评估** -- 比较特殊活动前后的出席情况，以衡量其影响。

:::warning
考勤数据通过您的小组和礼拜签到流程记录。如果考勤没有被持续跟踪，您的报告将无法准确反映实际参与情况。设置说明请参阅[考勤跟踪](../attendance/tracking-attendance.md)。
:::
