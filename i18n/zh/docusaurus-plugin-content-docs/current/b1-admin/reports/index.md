# 报告

<div class="article-intro">

报告页面为您提供快速访问预构建报告的功能，帮助您了解教会数据。使用这些报告来跟踪出席趋势、监控捐赠活动，并通过生日庆祝与您的会众保持联系。

</div>

## 可用报告

报告页面显示五个报告卡片。点击任意卡片以打开带有筛选器的报告。

- **[生日报告](./birthday-report)** -- 查看您会众成员即将到来的生日。
- **[出席趋势](./attendance-reports)** -- 查看您各项礼拜的出席模式随时间如何变化。
- **[小组出席](./attendance-reports)** -- 查看所选日期范围内按小组划分的出席总数。
- **[每日小组出席](./attendance-reports)** -- 获取小组出席数据的逐日分析。
- **[捐赠摘要](./donation-summary)** -- 查看捐赠总额和趋势，用于财务规划。

## 使用报告

1. 从主菜单导航到 **Reports**。
2. 点击一个**报告卡片**以打开它。
3. 使用每个报告顶部的**筛选器**按日期范围、小组、礼拜或其他条件缩小数据范围。
4. 查看报告中显示的结果。

:::tip
定期运行报告，以掌握出席趋势、即将到来的生日和捐赠模式。这些数据帮助您更有效地规划事工和外展活动。
:::

:::info
报告数据取决于一致的数据录入。确保您的团队正在[跟踪出席](../attendance/tracking-attendance.md)、[记录捐赠](../donations/recording-donations.md)，并保持[人员记录](../people/adding-people.md)为最新状态，以获得最准确的结果。
:::
