# 审计日志

<div class="article-intro">

审计日志跟踪您教会管理系统中的所有重要操作和更改。使用它来查看登录活动、跟踪谁对人员记录进行了更改、监控权限更新并在整个团队中保持问责制。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 具有服务器管理员访问权限的 B1 Admin 帐户
- 导航到 **Settings** 以查找审计日志

</div>

## 查看审计日志

1. 在 B1 Admin 中转到 **Settings**。
2. 选择 **Audit Log**。
3. 日志在表格中显示最近的条目,包含以下列:
   - **Date** -- 操作发生的时间。
   - **Category** -- 操作类型(颜色编码以便快速扫描)。
   - **Action** -- 执行了什么操作(例如,create、update、delete、login_success)。
   - **Entity** -- 受影响记录的类型和 ID。
   - **IP Address** -- 执行操作的用户的 IP 地址。
   - **Details** -- 所做具体更改的摘要。

## 过滤日志

使用页面顶部的过滤器缩小结果范围:

- **Category** -- 按操作类型过滤:
  - **All Categories** -- 显示所有内容。
  - **Login** -- 登录成功和失败。
  - **People** -- 创建、更新或删除人员记录。
  - **Permissions** -- 权限授予和撤销。
  - **Donations** -- 捐赠记录更改。
  - **Groups** -- 小组管理操作。
  - **Forms** -- 表单提交活动。
  - **Settings** -- 配置更改。
- **Start Date** -- 从此日期开始显示条目。
- **End Date** -- 显示截至此日期的条目。

设置过滤器后点击 **Search** 更新结果。

## 了解类别

每个类别都用颜色编码以便快速识别:

- **Login** -- 蓝色标签。跟踪成功和失败的登录尝试。
- **People** -- 紫色标签。跟踪人员记录的创建、更新和删除。
- **Permissions** -- 红色标签。跟踪访问权限的授予或撤销。
- **Donations** -- 绿色标签。跟踪捐赠记录更改。
- **Groups** -- 灰色标签。跟踪小组管理操作。
- **Forms** -- 橙色标签。跟踪表单提交活动。
- **Settings** -- 黄色标签。跟踪配置更改。

## 导出日志

显示日志条目时,会出现 **CSV download** 按钮。点击它可将当前过滤的结果导出到电子表格以进行离线审查或记录保存。

## 分页

使用表格底部的分页控件浏览结果。您可以每页显示 25、50 或 100 个条目。

:::info
审计日志条目自动保留一年。超过 365 天的条目将被删除以保持系统性能。
:::

:::tip
定期查看审计日志,尤其是在加入新团队成员或进行重大配置更改后。它有助于及早发现意外活动。
:::
