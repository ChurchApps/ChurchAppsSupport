# 捐赠汇总

<div class="article-intro">

捐赠汇总报告让您清晰了解教会的奉献活动。使用它来跟踪捐赠总额、监控奉献趋势，并为领导团队准备财务报告。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 确保捐赠在系统中[持续记录](../donations/recording-donations.md)
- 您需要相应的[权限](../settings/roles-permissions.md)才能访问财务报告

</div>

## 运行报告

1. 从主菜单导航到**报告**。
2. 点击**捐赠汇总**卡片。
3. 设置**日期范围**以定义报告期间。
4. 应用其他筛选条件，如**基金**或其他条件以缩小结果范围。
5. 报告将显示所选期间的奉献汇总。

## 报告显示内容

捐赠汇总报告包括：

- **捐赠总额**——所选期间的总奉献金额。
- **捐赠者数量**——做出贡献的不重复捐赠者人数。
- **各基金金额**——各基金的奉献明细（如适用）。

## 实际用途

- **财务规划**——审查奉献趋势以预测未来收入并制定预算。
- **董事会报告**——为定期的董事会或领导会议生成具有明确奉献数据的汇总。
- **年终审查**——运行全年报告以准备年度财务汇总。
- **专项活动跟踪**——按基金筛选以跟踪特别奉献活动或建堂项目的进度。
- **季节性分析**——比较不同时期的奉献，了解季节性规律。

:::tip
每月运行此报告以了解教会的财务状况。逐月比较数据有助于及早发现趋势。
:::

:::warning
捐赠数据来自系统中记录的贡献。确保捐赠记录一致——无论是通过在线奉献、支票处理还是手动输入——以获得最准确的报告。详情请参阅[记录捐赠](../donations/recording-donations.md)。
:::
