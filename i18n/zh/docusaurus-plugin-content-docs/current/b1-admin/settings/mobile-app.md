# 移动应用设置

<div class="article-intro">

移动应用设置页面允许您配置教会成员在 B1 Mobile 应用中看到的导航标签。您可以控制哪些标签可见、它们链接到哪里以及如何显示。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要"Edit Church Settings"权限。如果您没有访问权限，请参阅[角色与权限](./roles-permissions.md)。
- 首先配置您的[教会设置](./church-settings.md)，包括教会名称和品牌形象

</div>

## 访问移动应用设置

1. 导航到左侧边栏中的**设置**页面。
2. 点击页头中的**移动应用**按钮。
3. 移动应用设置页面显示您当前的应用标签。

## 添加新标签

1. 点击页面顶部的**添加标签**按钮。
2. 填写标签详情：
   - **名称** -- 标签上显示的标签名（例如，"讲道"或"奉献"）。
   - **图标** -- 点击图标选择器为您的标签选择一个图标。您也可以上传自定义图片。
   - **标签类型** -- 从 Bible、Live Stream、Donation、Website 等选项中选择。
   - **URL** -- 输入标签应链接到的网址。
   - **可见性** -- 控制谁可以看到此标签（所有人、仅成员等）。
3. 点击**保存标签**将其添加到您的应用中。

## 编辑现有标签

1. 点击**应用标签**列表中的任何现有标签。
2. 更新标签的名称、图标、URL、类型或可见性设置。
3. 点击**保存标签**以应用您的更改。

## 重新排列标签

您可以更改标签在移动应用中的显示顺序。拖放列表中的标签来重新排列它们。此页面上显示的顺序与您的成员在应用中看到的顺序一致。

:::info
某些标签可能会在满足特定条件时自动出现——例如，当直播活跃时，Live Stream 标签可能会显示。手动添加的标签让您可以完全控制成员在任何时候看到的内容。
:::

:::tip
保持标签数量适度。三到五个标签对大多数教会来说效果很好。标签太多会让成员的导航变得混乱。
:::

## 这些标签显示在哪里

您在此处配置的标签显示在**B1 Mobile 应用**（B1.church 应用）中，您的教会成员可以从 App Store 或 Google Play 下载。您在此页面上所做的更改将在成员下次打开应用时反映出来。
