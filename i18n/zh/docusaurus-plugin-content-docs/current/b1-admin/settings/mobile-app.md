# 移动应用设置

<div class="article-intro">

移动应用设置页面允许您配置在**B1.church 移动体验 (PWA)** 中为您的教会成员显示的导航选项卡。您可以控制哪些选项卡可见、它们链接到什么，以及如何显示它们。

</div>

:::info 原生 B1 Mobile 应用已弃用
这里配置的选项卡通过 [B1.church Progressive Web App (PWA)](/docs/b1-church/getting-started/installing-pwa) 提供，该应用已取代原生 B1 Mobile 应用。成员可以从 `https://yourchurchname.b1.church` 上的任何页面在其主屏幕上安装 PWA，而不是从 App Store 或 Google Play 下载旧应用。
:::

<div class="prereqs">
<h4>开始前</h4>

- 您需要"Edit Church Settings"权限。如果您没有访问权限，请参阅[角色和权限](./roles-permissions.md)
- 首先配置您的[教会设置](./church-settings.md)，包括您的教会名称和品牌

</div>

## 访问移动应用设置

1. 在左侧栏中导航到 **Settings** 页面。
2. 点击标题中的 **Mobile Apps** 按钮。
3. "移动应用设置"页面显示您当前的应用选项卡。

## 添加新选项卡

1. 点击页面顶部的 **Add Tab** 按钮。
2. 填写选项卡详细信息：
   - **Name**——出现在选项卡上的标签（例如，"Sermons"或"Give"）。
   - **Icon**——点击图标选择器来选择您的选项卡的图标。您也可以上传自定义图像。
   - **Tab Type**——从圣经、直播流、捐赠、网站等选项中选择。
   - **URL**——输入选项卡应链接到的网址。
   - **Visibility**——控制谁可以看到此选项卡（所有人、仅成员等）。
3. 点击 **Save Tab** 来将其添加到您的应用。

## 编辑现有选项卡

1. 在 **App Tabs** 列表中点击任何现有选项卡。
2. 更新选项卡的名称、图标、URL、类型或可见性设置。
3. 点击 **Save Tab** 来应用您的更改。

## 重新排列选项卡

您可以更改选项卡在移动应用中出现的顺序。拖放列表中的选项卡来重新排列它们。此页面上显示的顺序与您的成员在应用中看到的顺序相匹配。

:::info
当满足某些条件时，某些选项卡可能会自动出现——例如，当流处于活动状态时，直播流选项卡可能会显示。手动添加的选项卡使您可以完全控制您的成员始终看到的内容。
:::

:::tip
保持选项卡数量可控。三到五个选项卡对大多数教会效果很好。选项卡过多会使您的成员导航混乱。
:::

## 这些选项卡出现在哪里

您在此处配置的选项卡显示在**B1.church PWA** 中，您的成员从 `https://yourchurchname.b1.church` 上的任何页面安装。您在此页面上进行的更改会在成员下次打开应用时反映出来。（这些选项卡也由旧版 [B1 Mobile 原生应用](/docs/b1-mobile/)呈现给仍在运行该应用的任何成员，但该应用已被弃用且不再更新。）
