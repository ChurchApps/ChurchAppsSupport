# 设置

<div class="article-intro">

设置部分是您在 B1 Admin 中配置教会账户、品牌形象和管理选项的地方。在这里，您可以管理从教会名称和子域名到用户权限和移动应用配置的一切内容。

</div>

## 您将在这里找到什么

设置区域分为四个子部分，可从侧边栏访问：

1. **设置** -- 配置您教会的基本信息、品牌形象和子域名。详情请参阅[教会设置](./church-settings.md)。
2. **移动应用** -- 设置和自定义 B1 Mobile 应用中为您的成员显示的导航标签。详情请参阅[移动应用设置](./mobile-app.md)。
3. **服务器管理** -- 访问高级管理工具来管理您教会的服务器级别设置。
4. **表单** -- 创建和管理自定义表单，用于从您的会众收集信息。

## 主设置页面

当您首次打开设置页面时，您会看到顶部显示的**教会名称**和**子域名**。页头提供了常见任务的快捷按钮：

- **编辑设置** -- 更新您的教会名称、地址、联系信息和品牌形象。
- **移动应用** -- 直接跳转到移动应用配置。
- **角色** -- 管理您团队的用户角色和权限。
- **导入/导出** -- 使用导入/导出工具在系统之间传输数据。

:::tip
首先在**编辑设置**下配置您的教会名称和品牌形象，然后设置[角色](./roles-permissions.md)以邀请您的团队成员并赋予适当的访问级别。
:::

:::info
有关您的数据如何受到保护的信息，请参阅[数据安全](./data-security.md)。
:::
