# 设置

<div class="article-intro">

设置部分是你在 B1 Admin 中配置教会账户、品牌和管理选项的地方。从这里你可以管理从教会名字和子域到用户权限和移动应用配置的一切。

</div>

## 你会在这里找到什么

设置区域组织为从侧边栏可访问的以下子部分：

1. **设置** -- 配置你教会的基本信息、品牌和子域。详见[教会设置](./church-settings.md)了解详情。
2. **分校区** -- 为你教会的多个物理位置添加和管理分校区。一旦创建，分校区会出现在人员档案上、在出勤设置中以及在人口统计仪表板中。详见[分校区](./campuses.md)了解详情。
3. **移动应用** -- 设置和自定义出现在 [B1.church PWA](/docs/b1-church/getting-started/installing-pwa) 中的导航选项卡（相同的选项卡也由已弃用的 B1 Mobile 本地应用呈现）。详见[移动应用设置](./mobile-app.md)了解详情。
4. **服务器管理员** -- 访问高级管理工具以管理你教会的服务器级设置。
5. **表格** -- 创建和管理自定义表格以从你的会众收集信息。

## 主设置页面

当你首次打开设置页面时，你会看到你的**教会名字**和**子域**显示在顶部。标题提供快速访问按钮用于常见任务：

- **编辑设置** -- 更新你教会的名字、地址、联系信息和品牌。
- **移动应用** -- 直接跳转到移动应用配置。
- **角色** -- 为你的团队管理用户角色和权限。
- **导入/导出** -- 使用导入/导出工具在系统之间传输数据。

:::tip
首先在**编辑设置**下配置你的教会名字和品牌，然后设置[角色](./roles-permissions.md)以使用正确的访问级别邀请你的团队成员。
:::

:::info
有关你的数据是如何保护的信息，详见[数据安全](./data-security.md)。
:::
