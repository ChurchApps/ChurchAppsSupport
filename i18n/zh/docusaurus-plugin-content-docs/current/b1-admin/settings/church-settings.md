# 教会设置

<div class="article-intro">

教会设置页面是您配置教会基本信息、联系方式和品牌形象的地方。这些信息将在所有 ChurchApps 工具中使用，包括您的 B1.church 网站和 B1 Mobile 应用。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要"Edit Church Settings"权限。如果您没有访问权限，请参阅[角色与权限](./roles-permissions.md)。
- 准备好您教会的地址、联系信息和标志

</div>

## 编辑您的教会信息

1. 导航到左侧边栏中的**设置**页面。
2. 点击页头中的**编辑设置**按钮。
3. 更新以下任何字段：
   - **教会名称** -- 在所有 ChurchApps 产品中显示的名称。
   - **地址** -- 您教会的实际地址。
   - **联系信息** -- 电话号码、电子邮件和其他联系方式。
4. 点击**保存**以应用您的更改。

## 设置您的子域名

您的教会将获得一个免费的 **yourchurch.b1.church** 子域名。这是成员和访客可以访问您教会在线形象的网址。

1. 在设置页面上，找到**子域名**字段。
2. 输入您首选的子域名（例如，gracechurch.b1.church 对应 "gracechurch"）。
3. 保存您的更改。

:::info
您的子域名在所有 ChurchApps 教会中必须是唯一的。如果您首选的名称已被使用，请尝试添加您的城市或州名（例如，"gracechurch-dallas"）。
:::

## 配置品牌形象

自定义您的教会在所有 ChurchApps 工具中的外观：

1. 点击标志区域并选择图片文件来上传您的**教会标志**。
2. 添加在您的网站和[移动应用](./mobile-app.md)上使用的其他**教会图片**。

:::tip
为获得最佳效果，请使用 PNG 格式的透明背景标志。这可确保它在浅色和深色背景上都看起来很好。
:::

## 导入和导出

设置页头中的**导入/导出**按钮会在新浏览器窗口中打开一个专用工具。使用它来：

- 从其他教会管理系统导入成员数据。
- 导出您的 ChurchApps 数据以进行备份或迁移。

当您首次设置教会并需要将现有记录转移到 ChurchApps 时，这尤其有用。

:::warning
导入数据时，请始终先备份现有记录。导入操作会向您的系统添加数据，如果多次运行可能会创建重复条目。
:::
