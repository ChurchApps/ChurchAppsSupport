# 分校区

<div class="article-intro">

如果你的教会在多个地点聚会，**分校区**让你追踪每个人和小组属于哪个站点。配置后，分校区会在人员档案上显示、在出勤设置中以及在人口统计仪表板中。多站点教会可以在整个 B1 Admin 中按分校区过滤、搜索和报告。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 你需要**编辑教会设置**权限来管理分校区。详见[角色和权限](./roles-permissions.md)。

</div>

## 打开分校区设置

在 B1 Admin 中，转到左侧边栏中的**设置**并从设置导航中选择**分校区**。你会看到所有配置的分校区的列表，包括它们的名字、位置和时区。

## 添加分校区

1. 单击**添加分校区**（或**+**按钮，如果还没有分校区）。
2. 填写分校区详情：
   - **名字** *(必需)* -- 在整个 B1 Admin 中显示的显示名字（例如"主校区"或"北校区"）。
   - **地址** -- 分校区街道地址（用于信息显示；与教会设置中的主教会地址不同）。
   - **城市 / 州 / 邮编** -- 分校区位置。
   - **时区** -- 此分校区的 IANA 时区（例如*America/Chicago*）。在分校区位于不同时区时有用。
   - **网站** -- 此分校区自己的网络存在的可选 URL。
3. 单击**保存**。

## 编辑分校区

在列表中单击任何分校区行来在右侧的面板中打开其编辑器。更新字段并单击**保存**。

## 删除分校区

打开分校区来编辑并单击**删除**。你会被要求确认。删除分校区不会移除分配给它的人 -- 他们的分校区字段只是变成空白。

## 将人员分配给分校区

创建分校区后，员工可以从他们的档案将一个人分配给分校区：

1. 在**成员**中打开一个人的记录。
2. 单击**编辑**。
3. 从**分校区**下拉菜单中选择分校区。
4. 单击**保存**。

你也可以从成员页面批量更新分校区。选择多个人，使用**批量编辑**，并一次为所有人设置分校区字段。

## 按分校区过滤

一旦分校区被设置，你可以在整个 B1 Admin 中按分校区过滤：

- **成员搜索** -- 在高级搜索中添加分校区条件，或加载一个限定到分校区的[保存的列表](../people/lists.md)。
- **人口统计** -- [人口统计仪表板](../people/demographics.md)在至少有一个人分配到分校区时显示分校区甜甜圈图。
- **出勤设置** -- 出勤中的每个服务时间都可以绑定到一个分校区。

:::tip
单位置教会不需要配置分校区。所有分校区功能都是可选的 -- 如果没有分校区存在，分校区字段和图表只是不出现。
:::
