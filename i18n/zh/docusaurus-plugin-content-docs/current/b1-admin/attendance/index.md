# 出席

<div class="article-intro">

B1 Admin 的出席版块为您提供了教会出席情况的完整视图，帮助您了解谁在参加礼拜以及各小组的增长趋势。从配置校区和礼拜时间到查看趋势和设置自助签到，所有出席跟踪和报告功能都在此页面集中管理。

</div>

## 页面概览

打开出席页面后，您会看到顶部显示教会出席设置的关键统计信息：

- **校区** -- 您已配置的实体地点数量
- **礼拜时间** -- 所有校区的礼拜时间总数
- **已排班小组** -- 分配到特定礼拜时间的小组
- **未排班小组** -- 独立跟踪出席的小组，不绑定特定礼拜时间

在标题下方，三个标签页组织了您的出席工具。

## 标签页

### 设置

**设置**标签页用于配置出席跟踪的基础架构。您可以在此定义校区、为每个校区添加礼拜时间，并将[小组](../groups/creating-groups.md)分配到这些礼拜时间。此操作只需进行一次，但您可以在日程变更时随时更新。详见[出席设置](setup.md)的完整操作指南。

### 出席

**出席**标签页显示一段时间内的出席趋势。使用筛选器按日期范围、校区或礼拜时间缩小结果，快速发现数据中的规律。详见[出席跟踪](tracking-attendance.md)了解如何使用报表和筛选器。

### 小组

**小组**标签页按各个小组细分出席数据。当您想了解特定小组、班级或事工团队的情况，而非查看全教会整体数据时，此功能非常有用。
