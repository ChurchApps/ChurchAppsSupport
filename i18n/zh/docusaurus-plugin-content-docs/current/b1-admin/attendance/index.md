# 出席记录

<div class="article-intro">

B1 Admin 中的出席记录部分为您提供了教会出席情况的完整视图，包括谁在参加以及您的小组如何随时间增长。从配置校区和礼拜时间到查看趋势和设置自助签到，跟踪和了解出席情况所需的一切都可以从此页面进行管理。

</div>

## 页面概览

当您打开出席记录页面时，您将看到一个包含教会出席设置关键统计数据的标题栏：

- **校区** -- 您已配置的物理地点数量
- **礼拜时间** -- 所有校区的礼拜总数
- **已排期小组** -- 分配到特定礼拜时间的小组
- **未排期小组** -- 独立于礼拜时间跟踪出席情况的小组

标题栏下方有三个选项卡，用于组织您的出席工具。

## 选项卡

### 设置

**设置**选项卡是您配置出席跟踪基本要素的地方。在这里，您可以定义校区、为每个校区添加礼拜时间，并将[小组](../groups/creating-groups.md)分配到这些礼拜时间。您只需执行一次此操作，但可以在日程变更时随时更新。请参阅[出席设置](setup.md)获取完整指南。

### 出席记录

**出席记录**选项卡显示一段时间内的出席趋势。使用筛选器按日期范围、校区或礼拜时间缩小结果，快速发现数据中的规律。请参阅[跟踪出席情况](tracking-attendance.md)了解使用报告和筛选器的详细信息。

### 小组

**小组**选项卡按各个小组细分出席情况。当您想查看特定小组、班级或事工团队的表现，而不是查看全教会数据时，此功能非常有用。

- [签到](check-in.md) -- 为礼拜设置自助签到
