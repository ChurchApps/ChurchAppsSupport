# 考勤

<div class="article-intro">

B1 Admin 中的考勤部分为您提供了完整的教堂出席情况视图，以及您的小组如何随时间增长的情况。从配置校区和服务时间到审查趋势和设置自动签到，管理出席所需的一切都在该页面上进行。

</div>

## 页面概览

当您打开考勤页面时，您将看到一个标题，其中包含有关您教堂考勤设置的关键统计数据：

- **校区** -- 您已配置的物理位置数量
- **服务时间** -- 所有校区中的服务总数
- **已排期的小组** -- 分配给特定服务时间的小组
- **未排期的小组** -- 独立于服务时间跟踪出席情况的小组

在标题下方，三个选项卡可以组织您的考勤工具。

## 选项卡

### 设置

**设置**选项卡是您配置考勤跟踪基础的地方。在这里，您定义校区，为每个校区添加服务时间，并将[小组](../groups/creating-groups.md)分配给这些服务时间。您只需要做一次，尽管您可以在日程更改时更新它。有关完整演练，请参阅[考勤设置](setup.md)。

### 考勤

**考勤**选项卡显示随时间推移的出席趋势。使用筛选器按日期范围、校区或服务时间缩小结果范围，并快速发现数据中的模式。有关使用报告和筛选器的详细信息，请参阅[跟踪考勤](tracking-attendance.md)。

### 小组

**小组**选项卡将考勤分解为各个小组。当您想查看特定小组、班级或事工团队的表现如何，而不是查看教堂范围内的数字时，这很有帮助。
