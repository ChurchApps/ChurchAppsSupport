# 签到标签设计器

<div class="article-intro">

标签设计器让你创建和自定义当家庭为其孩子签到时打印的名牌和领取单模板。你可以精确控制每个标签上显示的信息、其位置和外观。

</div>

<div class="prereqs">
<h4>开始前的准备</h4>

- 设置[考勤](setup)并配置至少一个启用签到的服务时间
- 设置[签到](check-in)以便标签可以打印
- 需要对"考勤"部分拥有管理权限

</div>

## 打开标签设计器

在 B1 Admin 中，转到左侧边栏中的**考勤**并选择**标签**。你将看到已保存标签模板的列表，按类型分开：**名牌**和**领取单**。

## 标签类型

- **名牌** - 打印并粘贴到孩子身上。通常包括孩子的姓名、他们的教室/会议和安全代码。
- **领取单** - 给父母或监护人。通常包括安全代码和他们签到的孩子列表。

B1 提供默认名牌和默认领取单模板，适用于标准 3.5 × 1.1 英寸热敏标签。

## 创建标签模板

1. 点击**添加名牌**或**添加领取单**（或使用下拉菜单选择）。
2. 新模板在标签编辑器中打开。

### 标签编辑器

编辑器显示配置尺寸的标签的缩放预览。沿着左侧面板，你可以配置：

- **名称** - 模板名称（仅供你参考）
- **标签类型** - 名牌或领取单
- **宽度 / 高度** - 标签大小（英寸）

### 添加块

标签由块组成——放在标签画布上的单个内容片段。点击**添加块**以插入新块并选择其类型：

- **字段** - 在打印时拉取数据值：
  - `person.displayName` - 人员的全名
  - `sessions` - 他们签到的服务/教室
  - `securityCode` - 随机生成的领取安全代码
  - `children` - 孩子列表（用于领取单）
  - `person.nametagNotes` - 人员记录中的任何特殊注释
  - `campus` - 校园名称
- **文本** - 你输入的静态文本（用于标题、标签或说明）
- **条形码** - 编码安全代码的条形码

### 定位块

每个块有**X**、**Y**、**宽度**和**高度**字段，表示为标签画布的百分比（0-100）。调整这些以精确定位内容。你也可以设置：

- **字体大小** - 文本大小（点数）
- **加粗** - 切换加粗文本
- **对齐** - 左、中或右文本对齐
- **条件** - 可选地在字段为空时隐藏块（例如，仅当其有值时才显示 nametagNotes）

### 保存

点击**保存**以保存模板。更新的模板将在 B1 Checkin 中下次打印标签时使用。

## 重新排序模板

如果你有多个名牌或领取单模板，B1 Checkin 将默认使用列表中的第一个模板。拖动模板以重新排序。

## 删除模板

点击任何模板行上的删除图标并确认。删除某类型的最后一个模板会恢复内置默认模板。

:::tip
编辑模板后进行测试打印，以在下次服务前确认布局看起来正确。
:::
