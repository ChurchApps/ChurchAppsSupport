# 签到标签设计工具

<div class="article-intro">

标签设计工具让你创建和自定义打印的名字牌和接送单据模板，当家庭为他们的孩子签到时会打印出来。你可以精确控制每个标签上显示的信息、位置和外观。

</div>

<div class="prereqs">
<h4>开始前</h4>

- 设置[签到](setup)并至少配置一个启用了签到功能的服务时间
- 设置[签到](check-in)以便标签打印
- 你需要签到部分的管理员访问权限

</div>

## 打开标签设计工具

在 B1 Admin 中，从左侧菜单栏进入**签到**，然后选择**标签**。你将看到已保存的标签模板列表，按类型分离：**名字牌**和**接送单据**。

## 标签类型

- **名字牌** — 打印并贴在孩子身上。通常包括孩子的名字、他们所在的教室/课程和安全代码。
- **接送单据** — 给父母或监护人。通常包括安全代码和他们签到的孩子的列表。

B1 为你提供了一个默认的名字牌和一个默认的接送单据模板，尺寸为标准的 3.5 × 1.1 英寸热敏标签。

## 创建标签模板

1. 点击**添加名字牌**或**添加接送单据**（或使用下拉菜单选择）。
2. 一个新模板会在标签编辑器中打开。

### 标签编辑器

编辑器显示一个按配置大小缩放的标签预览。在左侧面板，你可以配置：

- **名称** — 模板名称（仅供你参考）
- **标签类型** — 名字牌或接送单据
- **宽度 / 高度** — 标签尺寸（英寸）

### 添加块

标签由块组成 — 个别内容块位置在标签画布上。点击**添加块**插入新块并选择其类型：

- **字段** — 在打印时提取数据值：
  - `person.displayName` — 人员全名
  - `sessions` — 签到所在的服务/教室
  - `securityCode` — 随机生成的接送安全代码
  - `children` — 孩子列表（接送单据）
  - `person.nametagNotes` — 人员记录上的任何特殊说明
  - `campus` — 校园名称
- **文本** — 你输入的静态文本（用于标题、标签或说明）
- **条形码** — 编码安全代码的条形码

### 定位块

每个块都有 **X**、**Y**、**宽度**和**高度**字段，以标签画布的百分比表示（0–100）。调整这些来精确定位内容。你也可以设置：

- **字体大小** — 文本大小（磅）
- **粗体** — 切换粗体文本
- **对齐** — 左、中或右文本对齐
- **条件** — 可选地隐藏块，如果字段为空（例如，只有在 nametagNotes 有值时才显示）

### 保存

点击**保存**保存模板。下次在 B1 Checkin 中打印标签时，将使用更新的模板。

## 重新排序模板

如果你有多个名字牌或接送单据模板，B1 Checkin 默认会使用列表中的第一个模板。拖动模板来重新排序它们。

## 删除模板

点击任何模板行上的删除图标并确认。删除类型的最后一个模板会恢复默认的内置模板。

:::tip
在编辑模板后进行测试打印，以确认下一次服务之前布局看起来正确。
:::
