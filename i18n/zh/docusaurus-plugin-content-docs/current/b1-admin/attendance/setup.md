# 出席设置

<div class="article-intro">

在跟踪出席情况之前，您需要告诉 B1 Admin 您教会的物理地点、礼拜时间以及每场礼拜中有哪些小组聚会。这项一次性设置创建了驱动教会所有出席跟踪和报告的结构。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个具有出席管理权限的有效 B1 Admin 账户。如果您不确定自己的访问级别，请参阅[角色与权限](../people/roles-permissions.md)。
- 如果您计划将小组分配到礼拜时间，请确保您的[小组已创建](../groups/creating-groups.md)。

</div>

## 关键概念

- **校区** -- 您教会聚会的物理地点（例如，"主校区"、"北校区"）。
- **礼拜时间** -- 校区中的定期聚会（例如，"周日上午 9:00"、"周三晚上 7:00"）。
- **已排期小组** -- 分配到特定礼拜时间的小组。出席记录在该礼拜的上下文中进行跟踪。
- **未排期小组** -- 自行跟踪出席情况的小组，不与任何礼拜时间关联。

## 设置您的出席结构

1. 打开 **B1 Admin** 并在侧边栏中点击 **Attendance**。
2. 选择 **Setup** 选项卡。
3. 点击 **Add Campus** 并输入您的地点名称。点击 **Save**。
4. 选中您的校区后，点击 **Add Service Time**。输入名称，例如 "Sunday 9:00 AM"，然后点击 **Save**。
5. 对该校区的每个礼拜时间重复上述操作。
6. 要将小组分配到礼拜时间，请选择礼拜时间并点击 **Add Group**。从列表中选择小组，然后点击 **Save**。

:::tip
如果某个小组在常规礼拜之外聚会——例如一个自行跟踪出席情况的周中小组——您可以将其保留为未排期小组。它仍然会显示在小组选项卡中用于出席报告。
:::

## 编辑您的设置

您可以随时更新设置。选择一个校区、礼拜时间或小组，点击 **Edit** 更改其详细信息，或点击 **Delete** 删除它。

:::info
删除礼拜时间不会删除过去的出席记录。即使您更改了日程，您的历史数据也会被保留。
:::

## 下一步

一旦您的校区、礼拜时间和小组就位，您就可以开始[跟踪出席情况](tracking-attendance.md)或为礼拜设置[自助签到](check-in.md)了。
