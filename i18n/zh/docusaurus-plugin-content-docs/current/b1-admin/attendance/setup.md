# 出席设置

<div class="article-intro">

在开始跟踪出席之前，您需要告诉 B1 Admin 您教会的实体位置、礼拜时间以及每个礼拜有哪些小组参加。这项一次性设置构建了支撑整个教会出席跟踪和报告功能的基础架构。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要一个拥有出席管理权限的 B1 Admin 账户。如果不确定您的访问级别，请参阅[角色与权限](../people/roles-permissions.md)。
- 如果计划将小组分配到礼拜时间，请确保先[创建小组](../groups/creating-groups.md)。

</div>

## 关键概念

- **校区** -- 您教会聚会的实体地点（例如"主校区"、"北校区"）。
- **礼拜时间** -- 校区中的定期聚会（例如"周日上午 9:00"、"周三晚上 7:00"）。
- **已排班小组** -- 分配到特定礼拜时间的小组。出席记录在该礼拜的上下文中进行跟踪。
- **未排班小组** -- 独立跟踪出席的小组，不绑定礼拜时间。

## 设置出席结构

1. 打开 **B1 Admin**，点击侧边栏中的 **出席**。
2. 选择**设置**标签页。
3. 点击**添加校区**并输入您的地点名称。点击**保存**。
4. 选中您的校区后，点击**添加礼拜时间**。输入名称（如"周日上午 9:00"）并点击**保存**。
5. 对该校区的每个礼拜时间重复此操作。
6. 要将小组分配到礼拜时间，选择该礼拜时间并点击**添加小组**。从列表中选择小组并点击**保存**。

### 为小组启用出席跟踪

在记录小组出席之前，必须先为该小组开启"跟踪出席"功能。

1. 点击侧边栏中的**小组**并选择该小组。
2. 点击**编辑**铅笔图标。
3. 将**跟踪出席**设置为**是**。
4. 点击**保存**。

:::tip
如果您在上一步中已将小组分配到礼拜时间，也可以在小组编辑页面中使用**添加礼拜时间**选项将其关联到正确的礼拜。这确保会议与正确的校区和时间关联。
:::

:::tip
如果某个小组在常规礼拜之外聚会——例如周中小组自行跟踪出席——可以将其保留为未排班小组。它仍会出现在"小组"标签页的出席报告中。
:::

## 编辑设置

您可以随时更新设置。选择一个校区、礼拜时间或小组，点击**编辑**更改详情，或点击**删除**将其移除。

:::info
删除礼拜时间不会删除过去的出席记录。即使您更改了日程，历史数据仍会保留。
:::

## 下一步

校区、礼拜时间和小组配置完成后，您就可以开始[手动记录出席](recording-attendance.md)或为礼拜设置[自助签到](check-in.md)了。
