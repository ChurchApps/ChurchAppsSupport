# 记录出席

<div class="article-intro">

校区、礼拜时间和小组设置完成后，您可以在每次聚会结束后手动记录出席。B1 Admin 围绕**会议**组织出席记录——每个小组每次聚会日期对应一个会议。您创建会议、标记出席人员，数据会直接汇入出席报表。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您的校区、礼拜时间和小组必须已配置完成。如果尚未完成，请参阅[出席设置](setup.md)。
- 您要跟踪的小组必须启用了**跟踪出席**功能。详见[出席设置](setup.md)。

</div>

## 创建会议

一个会议代表小组的一次聚会——例如，某个周日的 K-3 年级班级。

1. 打开 **B1 Admin**，点击侧边栏中的 **小组**。
2. 选择您要记录出席的小组。
3. 点击**会议**标签页。
4. 点击**新建**创建一个新会议。
5. 如果该小组已分配到礼拜时间，选择**礼拜时间**。如果是未排班小组，此字段不会出现。
6. 选择**会议日期**——可以是今天、过去的日期或未来的日期。
7. 点击**保存**。

:::tip
您可以为过去的日期创建会议以补录未记录的出席，也可以提前创建会议以便小组聚会时直接使用。
:::

## 标记出席

保存会议后，小组成员会显示在页面右侧。

1. 勾选每位出席人员旁边的复选框。
2. 更改会自动保存——无需额外点击保存按钮。

:::info
会议列表中仅显示当前小组成员。如果某人出席了但尚未加入小组，请先[将其添加到小组](../groups/group-members.md)，然后再记录其出席。
:::

## 导出出席记录到电子表格

您可以将会议记录下载为 CSV 文件，在 Excel、Numbers 或 Google Sheets 中使用。

1. 打开您要导出的会议。
2. 点击**下载 CSV** 按钮。
3. 在电子表格应用中打开下载的文件。

## 查看已记录的出席

记录会议后，数据会出现在出席报表中。

- **出席标签页** -- 显示全教会的出席趋势。参见[出席跟踪](tracking-attendance.md)。
- **小组标签页** -- 按各个小组显示出席细分。

:::tip
如果刚创建的会议没有立即出现在报表中，请确保会议日期在报表筛选器中选定的日期范围内。
:::
