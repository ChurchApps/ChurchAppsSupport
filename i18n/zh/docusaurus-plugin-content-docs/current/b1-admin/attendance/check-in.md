# 签到

<div class="article-intro">

B1 Admin 通过配套的 **B1 Checkin** 应用支持礼拜时的自助签到。成员到达时可以在自助终端或专用设备上为自己和家人签到，使流程更加快捷并减轻志愿者的工作量。每次签到都会自动记录为出席。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您的校区、礼拜时间和小组必须在[出席设置](setup.md)中配置完成。
- 您的数据库中需要有[人员](../people/adding-people.md)，并设置好[家庭](../people/adding-people.md#managing-households)，以便家庭成员可以一起签到。
- 您需要一台平板电脑，以及可选的 Brother 标签打印机（请参阅下方的[推荐硬件](#recommended-hardware)）。

</div>

## 工作原理

B1 Checkin 应用连接到您的 B1 Admin 出席设置。当成员签到时，其出席记录会自动关联到正确的校区、礼拜时间和小组。使用签到系统的人员无需手动输入出席记录。

## 设置签到

1. **首先配置您的出席结构。** 在 B1 Admin 中，转到 **Attendance > Setup**，确保您的校区、礼拜时间和小组已就位。签到应用依赖于此配置。详情请参阅[出席设置](setup.md)。
2. **在您计划使用的设备上安装 B1 Checkin 应用。** 该应用可在以下平台获取：
   - **Android/Samsung 平板电脑：** [Google Play Store](https://play.google.com/store/apps/details?id=church.b1.checkin)
   - **Amazon Fire 平板电脑：** [Amazon App Store](https://www.amazon.com/Live-Church-Solutions-B1-Check-In/dp/B0FW5HKRB5/)
3. **使用您教会的账户凭据登录 B1 Checkin 应用。**
4. **为当前聚会选择校区和礼拜时间。**
5. 成员现在可以在设备上搜索自己的姓名并签到。

:::tip
将签到设备放置在醒目且易于到达的位置，如大厅入口或迎宾台。在礼拜期间做简短通告有助于让成员知道此选项可用。
:::

:::tip
如果您的教会有多个校区，您需要在[出席设置](setup.md)中为每个校区重复设置。每个签到设备可以配置为不同的校区。
:::

## 推荐硬件

**平板电脑** — 以下任何一款都能与应用良好配合：

- **紧凑型：** Samsung Galaxy Tab A7 Lite 8.7"
- **大屏幕：** Samsung Galaxy Tab A8 10.5"
- **经济型：** Amazon Fire HD 10

**打印机** — 签到可配合 Brother 标签打印机打印姓名标签：

- **最佳：** Brother QL-1110NWB（通过蓝牙和 WiFi 支持多台平板电脑）
- **良好：** Brother QL-810W（通过 WiFi 支持多台平板电脑）
- **经济型：** Brother QL-1100（仅 WiFi）

**标签：** Brother DK-1201 (1-1/7" x 3-1/2")

:::warning
仅 Brother 标签打印机与 B1 Checkin 应用兼容。其他品牌的打印机无法用于打印姓名标签。
:::

:::info
请按照打印机的设置说明将其连接到与平板电脑相同的 WiFi 网络。您可以在 [Brother 支持网站](https://support.brother.com)上找到 Brother 打印机驱动程序和设置指南。
:::

## 记录内容

每次签到都会在 B1 Admin 中创建一条出席记录。您可以在[出席记录](tracking-attendance.md)和[小组](../groups/group-members.md)选项卡中查看这些记录，就像手动输入的出席记录一样。数据的显示方式没有区别——两种方法都汇入相同的报告。
