# 签到

<div class="article-intro">

B1 Admin 通过配套的 **B1 Checkin** 应用支持礼拜自助签到功能。会众到达时可以在自助终端或专用设备上为自己和家人签到，使整个过程快速便捷，同时减轻志愿者的工作负担。每次签到都会自动记录为出席记录。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您的校区、礼拜时间和小组必须已在[出席设置](setup.md)中配置完成。
- 您需要在[数据库中添加人员](../people/adding-people.md)，并设置好[家庭](../people/adding-people.md#managing-households)，以便家庭成员可以一起签到。
- 您需要一台平板电脑，以及可选的 Brother 标签打印机（参见下方的[推荐硬件](#recommended-hardware)）。

</div>

## 工作原理

B1 Checkin 应用连接到您的 B1 Admin 出席设置。当会众签到时，系统会自动将其出席记录关联到正确的校区、礼拜时间和小组。使用签到系统的人员无需手动输入出席记录。

## 设置签到

1. **首先配置您的出席结构。** 在 B1 Admin 中，前往 **出席 > 设置**，确保您的校区、礼拜时间和小组已就位。签到应用依赖此配置。详见[出席设置](setup.md)。
2. **在您计划使用的设备上安装 B1 Checkin 应用。** 该应用可在以下平台获取：
   - **Android/Samsung 平板：** [Google Play 商店](https://play.google.com/store/apps/details?id=church.b1.checkin)
   - **Amazon Fire 平板：** [Amazon 应用商店](https://www.amazon.com/Live-Church-Solutions-B1-Check-In/dp/B0FW5HKRB5/)
3. **使用您教会的账户凭据登录 B1 Checkin 应用。**
4. **选择当前聚会的校区和礼拜时间。**
5. 会众现在可以在设备上搜索自己的姓名并完成签到。

:::tip
将签到设备放置在大厅入口或欢迎台等显眼、便于触及的位置。在礼拜期间做简短说明，帮助会众了解此功能。
:::

:::tip
如果您的教会有多个校区，您需要在[出席设置](setup.md)中为每个校区重复设置步骤。每台签到设备可以配置为不同的校区。
:::

## 推荐硬件

**平板电脑** — 以下设备均可与应用良好配合：

- **紧凑型：** Samsung Galaxy Tab A7 Lite 8.7"
- **大屏幕：** Samsung Galaxy Tab A8 10.5"
- **经济型：** Amazon Fire HD 10

**打印机** — 签到功能支持 Brother 标签打印机来打印姓名标签：

- **最佳：** Brother QL-1110NWB（通过蓝牙和 WiFi 支持多台平板）
- **良好：** Brother QL-810W（通过 WiFi 支持多台平板）
- **经济型：** Brother QL-1100（仅支持 WiFi）

**标签：** Brother DK-1201 (1-1/7" x 3-1/2")

:::warning
仅 Brother 标签打印机与 B1 Checkin 应用兼容。其他品牌的打印机无法用于打印姓名标签。
:::

:::info
按照打印机的设置说明将其连接到与平板电脑相同的 WiFi 网络。您可以在 [Brother 支持网站](https://support.brother.com)上找到 Brother 打印机驱动程序和设置指南。
:::

## 自定义终端外观

您可以自定义 B1 Checkin 应用的外观以匹配您教会的品牌形象。在 B1 Admin 中，前往 **出席 > 终端主题** 进行配置：

### 颜色

自定义八种颜色设置以匹配您教会的品牌：

- **主色调**和**主色调对比色** -- 主要品牌颜色及其文字颜色。
- **辅助色**和**辅助色对比色** -- 强调颜色及其文字颜色。
- **页眉背景**和**副标题背景** -- 终端页眉区域的颜色。
- **按钮背景**和**按钮文字** -- 交互按钮的颜色。

### 背景图片

为终端欢迎页面和查找页面上传可选的背景图片。建议尺寸为 1920x1080 像素。

### 空闲屏幕 / 屏幕保护

配置一段时间无操作后激活的屏幕保护程序：

1. 开启或关闭空闲屏幕。
2. 设置**超时时间**（无操作多少秒后启动屏幕保护，最少 10 秒）。
3. 添加一个或多个**幻灯片** -- 每张幻灯片包含一张图片和显示时长（最少 3 秒）。

:::tip
利用空闲屏幕在终端未被使用时展示公告、即将举行的活动或欢迎信息。
:::

## 通过二维码进行访客登记

签到终端可以显示一个二维码，访客扫描后可以在自己的手机上为自己和家人进行登记。这加快了首次来访客人的签到流程。

当访客扫描二维码后，他们会被引导到一个[访客登记页面](../../b1-church/checkin/guest-registration)，在那里输入姓名、电子邮件和家庭成员信息。然后志愿者可以在终端上查找他们并完成签到。

### 启用二维码访客登记

要开启二维码显示：

1. 在 B1 Admin 中，点击左侧边栏中的 **移动端**（手机图标）。
2. 选择 **签到** 标签页。
3. 开启 **二维码访客登记**。

:::note
此设置位于**移动端**下，而非"出席 > 终端主题"中。
:::

## 记录内容

每次签到都会在 B1 Admin 中创建一条出席记录。您可以在[出席](tracking-attendance.md)和[小组](../groups/group-members.md)标签页中查看这些记录，与手动输入的出席记录完全相同。数据显示方式没有区别 -- 两种方式的数据都汇入相同的报表。
