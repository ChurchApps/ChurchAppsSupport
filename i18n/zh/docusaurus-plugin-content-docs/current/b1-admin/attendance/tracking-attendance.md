# 跟踪出席情况

<div class="article-intro">

一旦您的校区、礼拜时间和小组配置完成，B1 Admin 使查看出席数据和发现趋势变得轻而易举。出席页面提供两种报告视图——**出席记录**选项卡用于查看全教会趋势，**小组**选项卡用于查看小组级别的详细信息。使用这些工具来了解增长规律、识别参与度下降，并为您的教会做出数据驱动的决策。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您的出席结构必须至少设置一个校区和礼拜时间。如果您尚未完成此操作，请参阅[出席设置](setup.md)。
- 在报告显示结果之前，需要先录入出席数据。数据可以来自手动输入或[自助签到](check-in.md)。

</div>

## 查看出席趋势

1. 打开 **B1 Admin** 并在侧边栏中点击 **Attendance**。
2. 选择 **Attendance** 选项卡。
3. 打开选项卡时报告会自动运行，显示默认日期范围内的出席情况。

## 筛选您的数据

使用页面顶部的筛选器来缩小结果范围：

- **日期范围** -- 选择开始和结束日期以关注特定时段。
- **校区** -- 选择一个校区以仅查看该地点的出席情况。
- **礼拜时间** -- 选择一个礼拜时间以深入了解特定聚会。

更改筛选器后，图表和数据会立即更新，因此您可以快速比较不同的时间段或地点。

:::info
每次打开出席记录选项卡时，报告会自动运行，因此您无需点击刷新按钮即可看到最新数据。
:::

## 小组出席情况

**小组**选项卡按各个小组显示出席情况。当您想要监控特定班级、事工团队或小组，而不是查看整体礼拜数据时，此功能非常有用。

1. 选择**小组**选项卡。
2. 从列表中选择一个小组以查看其出席历史。
3. 使用日期范围筛选器调整报告窗口。

:::tip
小组出席情况对于[小组](../groups/creating-groups.md)负责人特别有价值，他们希望跟踪小组中成员随时间变化的参与度。
:::

## 使用出席数据的建议

- 每月查看趋势，及早发现季节性规律。
- 比较校区级别的数据，了解哪些地点在增长。
- 使用小组级别的报告跟进出席率下降的[小组](../groups/group-members.md)。
- 将出席洞察与 [AI 搜索](../people/ai-search.md)工具结合使用，找出最近未出席的人员。
