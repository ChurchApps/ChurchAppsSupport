# 出席跟踪

<div class="article-intro">

校区、礼拜时间和小组配置完成后，B1 Admin 让您能够轻松查看出席数据并发现趋势。出席页面提供两种报告视图——**出席**标签页用于查看全教会趋势，**小组**标签页用于查看小组级别的详细数据。使用这些工具了解增长规律、识别参与度下降情况，并为教会做出数据驱动的决策。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您的出席结构必须至少设置了一个校区和礼拜时间。如果尚未完成，请参阅[出席设置](setup.md)。
- 需要先记录出席数据，报表才会显示结果。数据可来自[手动录入](recording-attendance.md)或[自助签到](check-in.md)。

</div>

## 查看出席趋势

1. 打开 **B1 Admin**，点击侧边栏中的 **出席**。
2. 选择**出席**标签页。
3. 报表在标签页打开时自动运行，显示默认日期范围内的出席情况。

## 筛选数据

使用页面顶部的筛选器缩小结果范围：

- **日期范围** -- 选择开始和结束日期，聚焦于特定时期。
- **校区** -- 选择一个校区，仅查看该地点的出席情况。
- **礼拜时间** -- 选择一个礼拜时间，深入了解某场特定聚会。

更改筛选器后，图表和数据会立即更新，方便您快速比较不同时期或地点。

:::info
每次打开出席标签页时报表都会自动运行，因此您无需点击刷新按钮即可看到最新数据。
:::

## 小组出席

**小组**标签页按各个小组显示出席细分。当您想监控特定班级、事工团队或小组的情况，而非查看整体礼拜数据时，此功能非常有用。

1. 选择**小组**标签页。
2. 从列表中选择一个小组查看其出席历史。
3. 使用日期范围筛选器调整报告时间窗口。

:::tip
小组出席数据对于想要跟踪小组参与度变化的[小组](../groups/creating-groups.md)负责人尤为重要。
:::

## 使用出席数据的建议

- 每月查看趋势，及早发现季节性规律。
- 比较校区级别的数据，了解哪些地点在增长。
- 使用小组级别的报表跟进出席率下降的[小组](../groups/group-members.md)。
- 将出席洞察与 [AI 搜索](../people/ai-search.md)工具结合使用，查找近期未出席的人员。
