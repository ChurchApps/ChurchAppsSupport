# B1 Admin 概述

<div class="article-intro">

B1 Admin 是您的教会管理仪表板。它提供工具来管理教会运营的各个方面——从成员目录和奉献到网站建设和直播——所有功能都集中在一个地方。

</div>

## 入门

1. 打开浏览器并访问 [admin.b1.church](https://admin.b1.church)
2. 使用您的用户名和密码登录
3. 您将进入 **仪表板**,该页面提供教会数据概览以及 **快速操作**,用于执行常见任务,如添加人员、创建小组、记录奉献等

:::tip
刚接触 B1 Admin?从[简介](./introduction)开始观看视频演示,然后访问[设置](./settings/)配置您的教会信息并邀请您的团队。
:::

## 主要功能

- **[人员](./people/)** - 管理您的教会目录、添加成员、跟踪家庭
- **[小组](./groups/)** - 创建和组织教会小组,处理加入请求和成员管理
- **[考勤](./attendance/)** - 设置校区、服务时间并跟踪考勤
- **[奉献](./donations/)** - 记录多种货币的奉献、管理基金、生成报表
- **[服侍](./serving/)** - 协调志愿者、创建服务计划、管理任务
- **[表单](./forms/)** - 为注册和数据收集构建自定义表单
- **[报告](./reports/)** - 查看生日、考勤和奉献报告
- **[网站](./website/)** - 使用自定义导航样式构建和管理您的教会网站
- **[讲道](./sermons/)** - 管理您的讲道库和直播
- **[日历](./calendars/)** - 通过连接小组活动创建精选日历
- **[设置](./settings/)** - 配置教会信息、角色和权限
