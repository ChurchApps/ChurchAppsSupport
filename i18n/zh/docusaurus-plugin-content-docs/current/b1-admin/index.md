# B1 Admin 概览

<div class="article-intro">

B1 Admin 是您的教堂管理仪表板。它提供了管理教堂运营各个方面的工具 -- 从成员目录和捐赠到网站建设和直播 -- 所有一切都在一个地方。

</div>

## 开始入门

1. 打开您的浏览器并转到 [admin.b1.church](https://admin.b1.church)
2. 用您的用户名和密码登录
3. 您将进入**仪表板**，该仪表板提供了教堂数据的概览以及用于常见任务（如添加人员、创建小组、记录捐赠等）的**快速操作**

:::tip
初次接触 B1 Admin？从[介绍](./introduction)开始，观看视频演练，然后访问[设置](./settings/)来配置您的教堂信息并邀请您的团队。
:::

## 主要功能

- **[人员](./people/)** - 管理您的教堂目录、添加成员、批量编辑数据、追踪家庭
- **[小组](./groups/)** - 创建和组织教堂小组，提供加入请求和成员管理
- **[考勤](./attendance/)** - 设置校区、服务时间和追踪考勤
- **[捐赠](./donations/)** - 记录多种货币的捐赠、管理资金、生成凭证
- **[服务](./serving/)** - 协调志愿者、创建服务计划、管理任务和通过分步工作流移动人员
- **[表单](./forms/)** - 为注册和数据收集构建自定义表单
- **[报告](./reports/)** - 查看生日、考勤和捐赠报告
- **[网站](./website/)** - 使用自定义导航样式构建和管理您的教堂网站
- **[讲道](./sermons/)** - 管理您的讲道库和直播
- **[日历](./calendars/)** - 创建精选日历、管理房间/资源预订和查看可用性
- **[设置](./settings/)** - 配置教堂信息、角色、权限和自定义人员字段
