# B1 Admin 概览

<div class="article-intro">

B1 Admin 是您的教会管理控制面板。它提供管理教会运营各个方面的工具——从会员目录和捐赠到网站建设和直播——全部集中在一个地方。

</div>

## 快速入门

1. 打开浏览器，前往 [admin.b1.church](https://admin.b1.church)
2. 使用您的用户名和密码登录
3. 您将进入**仪表板**，可以快速访问所有主要功能

:::tip
初次使用 B1 Admin？请先查看[简介](./introduction)获取视频教程，然后访问[设置](./settings/)配置您的教会信息并邀请团队成员。
:::

## 主要功能

- **[人员](./people/)** - 管理您的教会通讯录、添加会员、跟踪家庭信息
- **[小组](./groups/)** - 创建和组织教会小组、团队和事工
- **[出勤](./attendance/)** - 设置堂点、礼拜时间并跟踪出勤
- **[捐赠](./donations/)** - 记录奉献、管理基金、生成报告单
- **[事奉](./serving/)** - 协调志愿者、创建礼拜计划、管理任务
- **[表单](./forms/)** - 创建自定义表单用于报名注册和数据收集
- **[报告](./reports/)** - 查看生日、出勤和捐赠报告
- **[网站](./website/)** - 建设和管理您的教会网站
- **[讲道](./sermons/)** - 管理您的讲道库和直播
- **[日历](./calendars/)** - 创建和发布活动日历
- **[设置](./settings/)** - 配置教会信息、角色和权限
