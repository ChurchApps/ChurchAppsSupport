# B1 Admin 概览

<div class="article-intro">

B1 Admin 是你的教会管理仪表板。它提供工具来管理教会运营的各个方面 -- 从成员目录和捐赠到网站建设和直播 -- 一切都可以在一个地方完成。

</div>

## 开始使用

1. 打开浏览器并访问 [admin.b1.church](https://admin.b1.church)
2. 用你的用户名和密码登录
3. 你会进入**仪表板**，它提供教会数据的概览，以及**快速操作**按钮用于常见任务，如添加成员、创建小组、记录捐赠等

:::tip
第一次使用 B1 Admin？从[简介](./introduction)开始观看视频演练，然后访问[设置](./settings/)来配置教会信息并邀请你的团队。
:::

## 主要功能

- **[成员](./people/)** - 管理教会目录、添加成员、批量编辑数据、追踪家庭单位
- **[小组](./groups/)** - 创建和组织教会小组，支持加入请求和成员管理
- **[出勤](./attendance/)** - 设置分校区、服务时间并追踪出勤情况
- **[捐赠](./donations/)** - 记录多种货币的捐赠、管理基金、生成报表
- **[服侍](./serving/)** - 协调志愿者、创建服侍计划、管理任务并通过分步骤工作流移动成员
- **[表格](./forms/)** - 为注册和数据收集构建自定义表格
- **[报告](./reports/)** - 查看生日、出勤和捐赠报告
- **[网站](./website/)** - 使用自定义导航样式构建和管理教会网站
- **[讲道](./sermons/)** - 管理讲道库和直播
- **[日历](./calendars/)** - 通过连接小组事件创建策划的日历
- **[设置](./settings/)** - 配置教会信息、角色和权限
