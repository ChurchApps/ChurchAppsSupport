# B1 Admin 概述

<div class="article-intro">

B1 Admin 是您的教会管理仪表板。它提供了管理您教会运营方方面面的工具 -- 从成员目录和捐献到网站建设和直播 -- 所有这些都在一个地方。

</div>

## 开始

1. 打开您的浏览器并访问 [admin.b1.church](https://admin.b1.church)
2. 使用您的用户名和密码登录
3. 您将进入 **仪表板**，它提供您的教会数据的概述以及 **快速操作** 用于常见任务，例如添加人员、创建小组、记录捐献等

:::tip
B1 Admin 初学者？从 [简介](./introduction) 开始观看视频演练，然后访问 [设置](./settings/) 来配置您的教会信息并邀请您的团队。
:::

## 主要功能

- **[人员](./people/)** - 管理您的教会目录、添加成员、批量编辑数据、跟踪家庭
- **[小组](./groups/)** - 创建和组织教会小组，包括加入申请和成员管理
- **[出席](./attendance/)** - 设置校区、服务时间和跟踪出席
- **[捐献](./donations/)** - 以多种货币记录捐献、管理资金、生成声明
- **[服事](./serving/)** - 协调志愿者、创建服事计划、管理任务
- **[表单](./forms/)** - 为注册和数据收集构建自定义表单
- **[报告](./reports/)** - 查看生日、出席和捐献报告
- **[网站](./website/)** - 使用自定义导航样式构建和管理您的教会网站
- **[讲道](./sermons/)** - 管理您的讲道库和直播
- **[日历](./calendars/)** - 通过连接小组事件创建策划日历
- **[设置](./settings/)** - 配置教会信息、角色和权限
