# B1 Admin 概览

<div class="article-intro">

B1 Admin 是您的教会管理仪表板。它提供了管理教会运营各个方面的工具——从成员目录和捐赠到网站建设和直播——一切都在一个地方完成。

</div>

## 入门指南

1. 打开浏览器并访问 [admin.b1.church](https://admin.b1.church)
2. 使用您的用户名和密码登录
3. 您将进入**仪表板**,它提供了教会数据的概览以及常见任务的**快速操作**,如添加人员、创建小组、记录捐赠等

:::tip
首次使用 B1 Admin?从[介绍](./introduction)开始观看视频演示,然后访问[设置](./settings/)配置您的教会信息并邀请您的团队。
:::

## 主要功能

- **[人员](./people/)** - 管理您的教会目录,添加成员,批量编辑数据,跟踪家庭
- **[小组](./groups/)** - 创建和组织教会小组,处理加入请求和成员管理
- **[出勤](./attendance/)** - 设置校区、聚会时间并跟踪出勤情况
- **[捐赠](./donations/)** - 记录多币种奉献,管理基金,生成报表
- **[服事](./serving/)** - 协调志愿者,创建服事计划,管理任务
- **[表单](./forms/)** - 为登记和数据收集创建自定义表单
- **[报告](./reports/)** - 查看生日、出勤和捐赠报告
- **[网站](./website/)** - 使用自定义导航样式构建和管理您的教会网站
- **[讲道](./sermons/)** - 管理您的讲道库和直播
- **[日历](./calendars/)** - 通过连接小组活动创建精选日历
- **[设置](./settings/)** - 配置教会信息、角色和权限
