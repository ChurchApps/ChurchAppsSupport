## 操作指引

1. 使用 Google Chrome 浏览器，前往 [admin.b1.church](https://admin.b1.church) 并使用您的用户名和密码登录。
2. 这将带您进入 **B1.church Admin 仪表板**。在这里，您可以管理教会的数据库、网站、移动应用、课程等。
3. 点击**设置**来配置您的教会信息、管理角色和其他详细信息。
4. 点击**编辑铅笔**图标来设置您的教会信息。
5. 回到仪表板，点击右上角的**问号图标**获取帮助和教程。
6. 点击**查看文档**以查看帮助您入门 B1.church 的教程列表。
7. 返回仪表板，点击**个人头像旁边的箭头**并选择**切换应用**。
8. 选择 **Lessons.church** 来管理您教会的在线课程和课程计划。
9. 点击个人头像旁边的箭头并再次选择**切换应用**，即可切换回 B1.church。
10. 您现在可以通过**会员门户**或**管理员门户**进入 B1.church。会员门户让您可以查看 B1.church 对会众的显示效果、查看您的小组以及管理您的个人捐赠。

:::tip
从[设置](./settings/)页面开始，配置您的教会名称、品牌标识和团队权限。这将确保您后续设置的所有内容都能正确显示。
:::
