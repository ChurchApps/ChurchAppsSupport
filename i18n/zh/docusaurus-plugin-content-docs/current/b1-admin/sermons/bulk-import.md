# 批量导入

<div class="article-intro">

批量导入页面让您可以通过从 YouTube 或 Vimeo 导入视频来快速填充讲道库。这是导入现有讲道内容的最快方式，无需逐个添加每个视频。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要 **contentApi.streamingServices.edit** 权限。如果您没有访问权限，请参阅[角色与权限](../settings/roles-permissions.md)。
- 至少创建一个[播放列表](playlists)以将讲道导入其中
- 准备好您的 YouTube 频道 ID 或 Vimeo 帐户信息

</div>

## 选择来源

1. 在 B1 Admin 中，点击左侧边栏的**讲道**，然后点击**批量导入**标签。
2. 您会看到两个可点击的卡片：
   - **YouTube**（红色）——从 YouTube 频道导入视频
   - **Vimeo**（蓝色）——从 Vimeo 帐户导入视频
3. 点击您讲道所在平台对应的卡片。

:::tip
如果您在两个平台上都有视频，可以随时返回并切换来源。
:::

## 导入前准备

在导入视频之前，您至少需要一个播放列表来整理它们。如果您还没有创建播放列表：

1. 点击**播放列表**标签。
2. 点击**创建第一个播放列表**，填写名称、描述、发布日期和缩略图。
3. 点击**保存**，然后返回**批量导入**标签。

有关创建和管理播放列表的详细说明，请参阅[播放列表](playlists)。

## 导入视频

1. 选择 YouTube 或 Vimeo 后，在提供的字段中输入您的 **YouTube 频道 ID** 或 **Vimeo 帐户信息**。
2. 点击**获取**按钮以从您的频道或帐户检索所有可用视频。
3. 获取完成后，您将看到包含复选框的所有视频列表。
4. 勾选您要导入的视频旁边的复选框。您可以全选或选择特定视频。
5. 可选择启用**自动导入新视频**，以自动将未来上传的视频添加到您的讲道库。
6. 点击**导入到播放列表**下拉菜单，选择这些视频应添加到哪个播放列表。
7. 点击**导入**按钮完成批量导入。

您的视频将连同所有详细信息一起导入，包括标题、描述、日期和缩略图。

:::info
批量导入非常适合从其他平台迁移时入门使用。对于后续添加单个讲道，请改用[管理讲道](managing-sermons)页面。
:::
