# 讲道

<div class="article-intro">

B1 Admin 中的讲道版块让您管理教会的讲道库和直播。您可以添加单个讲道、将其组织成播放列表、配置直播时间表，以及从 YouTube 或 Vimeo 批量导入现有内容。

</div>

:::info
要访问讲道版块，您需要 **contentApi.streamingServices.edit** 权限。如果您在菜单中看不到讲道选项，请让教会管理员授予您相应的角色。有关管理权限的详细信息，请参阅[角色与权限](../settings/roles-permissions.md)。
:::

## 您可以做什么

讲道版块分为四个主要区域：

### 讲道

在您的讲道库中添加和管理单个讲道条目。每个讲道可以包含标题、讲员、日期、描述、缩略图，以及视频或音频内容的链接。您可以将讲道链接到 YouTube、Vimeo、Facebook 或自定义视频链接。详情请参阅[管理讲道](managing-sermons)。

### 播放列表

将您的讲道组织成系列或合集。播放列表让您的会众可以方便地按顺序浏览讲道系列。每个播放列表有自己的名称、描述和缩略图。详情请参阅[播放列表](playlists)。

### 直播时间

配置教会的直播时间表。设置每周定期礼拜或一次性活动，管理聊天和视频设置，并控制直播开始时间。访客将在您的直播页面上看到下一次预定的礼拜时间。设置说明请参阅[直播](live-streaming)。

### 批量导入

通过从 YouTube 或 Vimeo 批量导入视频来快速填充您的讲道库。这是将现有讲道内容迁移到 B1 的最快方式。请参阅[批量导入](bulk-import)开始使用。

:::tip
如果您是第一次设置讲道，请先创建一个[播放列表](playlists)，然后[单独添加讲道](managing-sermons)或从 YouTube 或 Vimeo [批量导入](bulk-import)。
:::
