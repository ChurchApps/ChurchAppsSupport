# 直播

<div class="article-intro">

直播时间页面让您配置教会的直播时间表、管理礼拜时间并自定义观众体验。设置每周定期礼拜或一次性活动，配置聊天和视频设置，并控制直播开始时间。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要 **contentApi.streamingServices.edit** 权限。如果您没有访问权限，请参阅[角色与权限](../settings/roles-permissions.md)。
- 如果您计划使用自动直播功能，请准备好您的 YouTube 频道 ID
- 至少添加一个[讲道](managing-sermons)或永久直播链接作为直播源

</div>

该页面有两个主要标签：**礼拜**用于管理直播时间表，**设置**用于配置直播页面。

## 管理礼拜

### 添加礼拜

1. 在 B1 Admin 中，点击左侧边栏的**讲道**，然后点击**直播时间**标签。
2. 点击**添加礼拜**按钮创建新的预定礼拜。
3. 输入**礼拜名称**（例如"主日早晨"）。
4. 设置**礼拜时间**——选择礼拜开始的日期和时间。
5. 将**每周重复**设置为**是**（定期每周礼拜）或**否**（一次性活动）。

### 配置聊天和视频设置

6. 在**聊天设置**中，设置礼拜前后多少分钟启用聊天。这让访客可以在礼拜开始前就开始聊天，并在礼拜后继续交流。
7. 在**视频设置**中，设置视频流提前多久开始播放倒计时或礼拜前内容。
8. 从下拉菜单中选择要播放的讲道：
   - **最新讲道**——自动播放您最近添加的视频。
   - **当前直播礼拜**——使用您的频道 ID 播放当前的 YouTube 直播。
   - 您也可以选择任何已保存的特定讲道。
9. 点击**保存**以安排您的礼拜。

:::info
如果设置为定期重复，您的礼拜将每周自动更新。您可以添加任意数量的礼拜。访客访问您的直播页面时将看到下一次预定的礼拜时间。
:::

## 直播页面设置

点击**设置**标签来自定义直播旁边显示的标签和链接。

### 添加标签

1. 点击**添加**按钮在直播页面上添加新标签。
2. 从预设标签（**聊天**或**祷告**）中选择，或添加带有外部链接的自定义标签。
3. 对于预设标签，只需在**标签文本**框中输入名称即可完成设置。
4. 对于链接标签，输入标签名称，点击图标按钮选择图标，然后输入链接地址。
5. 您配置的标签将出现在直播页面上，供观众访问额外的资源和互动功能。

### 预览您的直播

点击**查看您的直播**按钮，查看直播页面对访客的确切显示效果，包括您的标志、礼拜时间和已配置的标签。

## 设置 YouTube 直播

要连接您的 YouTube 频道以实现自动直播：

1. 前往**讲道**，点击**添加讲道**，然后选择**添加永久直播链接**。
2. 视频提供商默认为**当前 YouTube 直播**。输入您的 **YouTube 频道 ID**。
3. 添加标题和描述，然后点击**保存**。
4. 在**直播时间**中，创建一个礼拜并从讲道下拉菜单中选择您的永久直播链接。

:::tip
要找到您的 YouTube 频道 ID，请前往 YouTube 频道的高级设置并复制频道 ID 值。
:::

## 自定义颜色和标志

您的直播页面使用网站的[外观](../website/appearance)设置：

- **浅色强调色**配深色文字用于页头。
- **深色强调色**配浅色文字用于侧边栏。
- 您的**浅色背景标志**显示在直播页面上。使用具有透明背景和 4:1 宽高比的图片。

要更改这些设置，请前往**网站**然后**外观**，更新您的[调色板](../website/appearance#color-palette)和[标志](../website/appearance#logo-and-branding)设置。

## 添加直播主持人

要赋予团队成员主持人功能（聊天管理、祷告请求回应）：

1. 前往左侧边栏的**设置**，点击**角色**。
2. 点击加号按钮，选择**添加自定义角色**。
3. 将角色命名为"直播主持人"，然后点击**保存**。
4. 点击新角色，然后在成员部分点击**添加**来添加人员。
5. 向下滚动到**编辑权限**，展开**内容**部分，勾选**主持聊天**。

当主持人登录直播页面时，他们将拥有特殊功能，包括聊天管理和祷告请求管理。

:::info
有关创建角色和管理权限的更多详情，请参阅[角色与权限](../settings/roles-permissions.md)。
:::

## 故障排除

如果您使用"当前 YouTube 直播"选项和频道 ID 时，自动 YouTube 直播未正确显示，请尝试以下方法：

**症状：**
- 直播嵌入显示"视频不可用"
- 页面加载但没有视频出现
- 直接 YouTube 嵌入可用，但自动频道直播不行

**解决方案：**
检查您的 YouTube 频道是否有旧的或即将到来的已安排直播并删除它们：

1. 前往您的 YouTube Studio。
2. 导航到**内容**然后**直播**。
3. 查找任何旧的已安排直播或即将到来的已安排直播。
4. 删除这些旧的或已安排的直播条目。
5. 再次测试您的直播页面。

:::warning
当您的频道中有多个已安排或过去的直播条目时，YouTube 的自动频道直播嵌入可能会被阻止。删除这些条目后，YouTube 可以正确识别并提供您当前的直播。
:::

**其他要求：**
- 您的直播必须设置为**公开**（不是未列出或私密）。
- 必须在 YouTube 直播设置中允许嵌入。
- 确保您使用的是**当前 YouTube 直播**提供商（使用频道 ID），而不是 **YouTube** 提供商（使用视频 ID）。
