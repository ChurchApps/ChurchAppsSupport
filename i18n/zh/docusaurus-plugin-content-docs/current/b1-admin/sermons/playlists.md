# 播放列表

<div class="article-intro">

播放列表让您将讲道组织成系列或合集。当访客在您的 B1.church 网站上浏览讲道库时，播放列表帮助他们按正确的顺序查找和跟进完整的讲道系列。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要 **contentApi.streamingServices.edit** 权限。如果您没有访问权限，请参阅[角色与权限](../settings/roles-permissions.md)。
- 准备好播放列表的名称、描述和缩略图图片

</div>

## 查看播放列表

1. 在 B1 Admin 中，点击左侧边栏的**讲道**。
2. 点击页面顶部的**播放列表**标签。
3. 您将看到所有现有播放列表及其名称和描述的列表。

## 创建播放列表

1. 点击**创建第一个播放列表**按钮（如果您还没有播放列表）或点击**添加播放列表**。
2. 输入播放列表的**名称**——例如，讲道系列标题如"信仰根基"或"诗篇之夏"。
3. 添加**描述**，为访客提供系列的简要概述。
4. 设置**发布日期**以控制播放列表何时可见。
5. 上传**缩略图图片**以直观地代表该系列。
6. 点击**保存**创建播放列表。

:::tip
在添加讲道之前先创建播放列表。这样您可以在添加讲道时将每个讲道分配到正确的播放列表中，而不是之后再重新整理。
:::

## 将讲道添加到播放列表

当您[添加或编辑讲道](managing-sermons)时，使用**播放列表**下拉菜单将其分配到播放列表。分配到同一播放列表的所有讲道将在您的网站上按顺序分组显示。

:::info
您也可以在[批量导入](bulk-import)期间将讲道分配到播放列表。从 YouTube 或 Vimeo 导入时，在点击**导入**之前从**导入到播放列表**下拉菜单中选择播放列表。
:::

## 播放列表在网站上的显示方式

播放列表在您的 B1.church 网站上以可浏览的合集形式展示。访客可以：

- 一目了然地查看所有可用的讲道系列
- 点击进入播放列表以查看该系列中的所有讲道
- 按顺序观看或收听讲道
