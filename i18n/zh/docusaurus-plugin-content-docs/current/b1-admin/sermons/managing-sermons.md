# 管理讲道

<div class="article-intro">

讲道页面显示您的整个讲道库。从这里您可以添加新讲道、编辑现有条目，并按播放列表组织内容。每个讲道可以链接到 YouTube、Vimeo、Facebook 或自定义链接上托管的视频或音频。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 您需要 **contentApi.streamingServices.edit** 权限。如果您没有访问权限，请参阅[角色与权限](../settings/roles-permissions.md)。
- 至少创建一个[播放列表](playlists)以将讲道组织其中
- 准备好来自 YouTube、Vimeo 或 Facebook 的视频 ID 或链接

</div>

## 查看您的讲道库

1. 在 B1 Admin 中，点击左侧边栏的**讲道**。
2. 讲道页面显示所有讲道条目，按播放列表组织。每个讲道显示缩略图、标题和日期。
3. 点击任意讲道以查看或编辑其详细信息。

## 添加讲道

1. 点击右上角的**添加讲道**按钮，从下拉菜单中选择**添加讲道**。
2. 选择一个**播放列表**以将讲道分配到其中。
3. 选择您的**视频提供商**——YouTube、Vimeo、Facebook 或自定义链接。我们推荐使用 YouTube，因为它与 B1 系统配合效果最佳。
4. 输入视频 ID 或链接，然后点击**获取**。对于 YouTube，视频 ID 是 YouTube 链接中 `v=` 后面的字符串。
5. 点击**获取**后，讲道详细信息将自动导入，包括发布日期、时长、标题、描述和缩略图。
6. 进行所需的更改并点击**保存**。

:::tip
您也可以通过在**添加讲道**下拉菜单中选择**添加永久直播链接**来添加永久直播流链接。这将使用您的频道 ID 创建与 YouTube 频道直播的持久连接。详情请参阅[直播](live-streaming)。
:::

## 编辑讲道

1. 点击讲道库中的任意讲道以打开其详细信息。
2. 根据需要更新标题、讲员、日期、描述、缩略图或媒体链接。
3. 点击**保存**应用更改。

## 讲道详情

每个讲道条目可以包含：

- **标题**——显示给访客的讲道名称
- **讲员**——讲道的传讲者
- **日期**——发布或传讲日期
- **描述**——讲道内容摘要
- **缩略图**——显示在讲道库中的预览图片
- **视频/音频链接**——指向 YouTube、Vimeo、Facebook 或自定义托管平台上讲道媒体的链接

## 安排讲道用于直播

添加讲道后，您可以将其安排在直播页面上播放：

1. 前往**直播时间**标签。
2. 编辑一个礼拜，在**视频设置**下从下拉菜单中选择您的讲道。
3. 该讲道将在预定的礼拜时间播放。

:::info
如需一次性导入多个讲道而非逐个添加，请使用[批量导入](bulk-import)工具直接从您的 YouTube 或 Vimeo 帐户导入视频。
:::
