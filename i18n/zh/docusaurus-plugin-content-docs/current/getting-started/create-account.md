# 创建您的账户

<div class="article-intro">

使用 ChurchApps 的第一步是创建您的账户。整个过程只需几分钟，即可让您访问所有 ChurchApps 工具，包括 B1 Admin、B1.church 和 Lessons.church。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- 准备一个有效的电子邮件地址用于账户验证
- 查看[入门指南概述](./index.md)以了解 ChurchApps 提供的功能

</div>

## 分步注册

1. 在您的网络浏览器中访问 [B1.church](https://b1.church)。
2. 点击 **注册** 按钮。
3. 输入您的 **名字**、**姓氏** 和 **电子邮件地址**。
4. 为您的账户创建密码。
5. 提交注册表单。

## 确认您的电子邮件

注册后，您将在提供的地址收到一封确认电子邮件。

1. 检查您的收件箱中是否有来自 ChurchApps 的确认邮件。
2. 点击邮件中的确认链接。
3. 您的账户现已验证并可以使用。

:::tip
如果您没有看到确认邮件，请检查您的垃圾邮件文件夹。邮件来自 ChurchApps，可能需要一两分钟才能送达。
:::

## 设置您的教会

确认电子邮件并登录后，您可以选择创建新教会或加入现有教会。

**创建新教会：**

1. 选择 **创建新教会** 选项。
2. 输入您的 **教会名称** 和基本信息。
3. 完成设置流程。
4. 您将被带到 **B1 Admin 仪表板**，在那里您可以开始配置您的教会。

**加入现有教会：**

1. 选择 **加入现有教会** 选项。
2. 按名称搜索您的教会。
3. 请求加入。您教会的管理员需要批准您的请求。

:::info
创建新教会的人会自动被分配 **Domain Admin** 角色，该角色拥有对所有设置和功能的完全访问权限。您可以稍后在 **角色** 页面添加其他管理员。
:::

## 接下来做什么

教会设置完成后，请前往[初始设置](./initial-setup.md)指南，配置教会的设置、品牌形象和团队访问权限。
