# 入门指南

<div class="article-intro">

欢迎使用 ChurchApps -- 专为帮助您的教会蓬勃发展而设计的免费开源工具。无论您是小型会众还是大型多校区组织，ChurchApps 都能为您提供管理和发展教会社区所需的一切。

</div>

## ChurchApps 产品

ChurchApps 包含多个协同工作的产品：

- **B1 Admin** -- 您的教会管理仪表板。在一个集中位置管理人员、小组、考勤、奉献、网站内容等。
- **B1.church** -- 您的教会公共网站和会员门户。会员可以观看讲道、访问通讯录、在线奉献并保持联系。
- **Lessons.church** -- 免费的教会课程，适用于主日学、小组和圣经学习。免费访问不断增长的课程库。
- **FreeShow** -- 免费的敬拜服务演示软件。在聚会期间展示歌词、幻灯片和媒体内容。

:::info
所有 ChurchApps 工具均 **100% 免费**，没有限制、没有高级层级、没有隐藏费用。ChurchApps 是一个由捐赠和志愿者支持的非营利组织。
:::

## 本文档涵盖的内容

本文档将引导您完成使用 ChurchApps 所需的一切准备工作。您将学习如何：

1. 创建您的账户并注册您的教会。
2. 配置教会的基本设置和品牌形象。
3. 为您的团队设置正确的角色和权限。
4. 开始使用工具来管理您的会众。
