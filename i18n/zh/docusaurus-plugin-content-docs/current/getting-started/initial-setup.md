# 初始设置

<div class="article-intro">

创建账户和教会后，在邀请团队之前需要完成一些重要的配置步骤。本指南将引导您按照推荐的设置顺序操作，确保您的教会从第一天起就准备就绪。

</div>

<div class="prereqs">
<h4>开始之前</h4>

- [创建您的账户](./create-account.md)并注册您的教会
- 准备好教会的徽标和品牌素材（可选但推荐）

</div>

## 步骤 1：配置教会信息

1. 在左侧边栏中导航到 **设置**。
2. 点击 **编辑设置**。
3. 输入教会的 **名称**、**地址** 和 **联系方式**。
4. 点击 **保存** 以应用更改。

## 步骤 2：设置品牌形象

1. 在 **设置** 页面上传您的 **教会徽标**。
2. 配置其他品牌选项，如颜色和图片。
3. 您的品牌形象将显示在 B1.church 网站和 B1 移动应用中。

:::tip
在邀请团队成员之前准备好徽标和品牌形象，可以在他们登录时留下良好的第一印象。
:::

## 步骤 3：配置角色并邀请团队

1. 在 **设置** 页面点击 **角色**。
2. 为您的团队创建角色（例如"牧师"、"秘书"、"财务"）。
3. 为每个角色分配适当的权限。
4. 通过搜索团队成员的账户并将其分配到角色来添加成员。

## 步骤 4：设置关键区域

团队就位后，开始构建 B1 Admin 的核心区域：

1. **人员** -- 将成员添加到教会通讯录中。这是其他一切的基础，所以从这里开始。您可以手动添加人员，也可以使用 **设置** 中的 **导入** 工具。
2. **小组** -- 为小组、事工团队、课程和其他聚会创建小组。
3. **考勤** -- 配置校区、礼拜和礼拜时间以跟踪出勤情况。
4. **奉献** -- 设置奉献基金并连接您的支付网关（Stripe）以接受在线奉献。
5. **网站** -- 使用 B1.church 网站构建器创建教会的公共网站。

:::info
您不需要一次设置所有区域。许多教会从 **人员** 和 **小组** 开始，然后在熟悉平台后添加 **考勤** 和 **奉献**。
:::

## 推荐顺序

为获得最顺畅的设置体验，我们建议按以下顺序操作：

1. **人员** -- 首先添加您的成员和固定参加者。
2. **小组** -- 将人员组织到有意义的小组中。
3. **考勤** -- 开始跟踪每周的出勤情况。
4. **奉献** -- 在人员录入系统后设置奉献。
5. **网站** -- 在数据就绪后最后构建公共网站。

完成这些步骤后，您的教会就准备就绪了。浏览 [B1 Admin](/docs/b1-admin/) 文档获取每个区域的详细指南。
