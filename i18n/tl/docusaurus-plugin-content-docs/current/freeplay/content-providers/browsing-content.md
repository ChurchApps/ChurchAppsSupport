# Pagba-browse at Pag-download ng Nilalaman

<div class="article-intro">

Kapag nakakonekta ka na sa isang content provider, maaari mong i-browse ang library nito, mag-download ng mga file, at i-play ang mga ito sa iyong TV. Ang content browser ay nagbibigay ng visual na grid ng mga folder at file na maaari mong i-navigate gamit ang iyong TV remote.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kumonekta sa hindi bababa sa isang content provider -- tingnan ang [Pagkonekta sa mga Provider](./connecting-providers)
- Ihanda ang iyong TV remote para sa navigation

</div>

## Ang Content Browser

Ang content browser ay nagpapakita ng mga item sa isang 3-column na grid ng mga card. Ang bawat card ay nagpapakita ng thumbnail image at pamagat. May dalawang uri ng mga item:

- **Mga Folder** - Kinakatawan ng isang thumbnail o folder icon. Pumili ng folder upang makita ang nilalaman nito.
- **Mga File** - Mga indibidwal na media item (mga video o larawan). Ang mga video file ay nagpapakita ng play icon overlay sa kanilang thumbnail.

Ang header sa itaas ng screen ay nagpapakita ng pangalan ng kasalukuyang folder, o ang pangalan ng provider kung ikaw ay nasa root level.

## Pag-navigate ng mga Folder

1. Gamitin ang directional pad sa iyong remote upang i-highlight ang isang folder
2. Pindutin ang **Select** upang buksan ito
3. Ilo-load ng browser ang nilalaman ng folder, na maaaring mas maraming folder o mga playable na file
4. Pindutin ang **Back** sa iyong remote upang umakyat ng isang antas

Maaari kang mag-drill hanggang sa kung gaano kalalim ang pinapayagan ng istruktura ng nilalaman ng provider. Tina-track ng browser ang iyong folder path upang maka-navigate ka pabalik sa bawat antas.

## Pag-download at Pag-play ng Nilalaman

Kapag pumili ka ng folder na naglalaman ng mga playable na file (o pumili ng isang indibidwal na file), lilipat ang FreePlay sa isang download screen:

1. Ang pamagat ng nilalaman ay lalabas sa screen na may cover image mula sa napiling folder
2. Isang progress bar ang nagpapakita ng **Downloading item X of Y** habang ang mga file ay sine-save sa iyong device
3. Kapag kumpleto na ang pag-download, lalabas ang button na **Start** na may play icon
4. Pindutin ang **Select** upang simulan ang playback

:::tip
Ang na-download na nilalaman ay naka-cache sa iyong TV. Kung i-play mo muli ang parehong nilalaman mamaya, agad itong maglo-load nang hindi na kailangang mag-download muli.
:::

## Pag-play ng Nilalaman mula sa Provider

Ang playback ay gumagana sa parehong paraan tulad ng pag-playback ng aralin. Gamitin ang mga kontrol na ito:

- **Kanang arrow** - Susunod na item
- **Kaliwang arrow** - Nakaraang item
- **Select / Play-Pause** - I-pause o ipagpatuloy ang video
- **Pataas na arrow** - Buksan ang message selector upang tumalon sa isang partikular na item
- **Back** - Lumabas sa content browser

Kapag lumabas ka sa player, babalik ka sa content browser sa root level ng kasalukuyang provider.

:::info
Para sa kumpletong sanggunian ng lahat ng kontrol ng player, tingnan ang [Pag-play ng mga Aralin](../classroom-mode/playing-lessons).
:::

## Paglipat sa Pagitan ng mga Provider

Ang mga nakakonektang provider ay lalabas bilang mga item sa sidebar sa kaliwa. Upang lumipat sa ibang provider:

1. Pindutin ang **Kaliwang** arrow sa iyong remote upang palawakin ang sidebar
2. Piliin ang provider na gusto mong i-browse
3. Magre-refresh ang content browser na may library ng provider na iyon

:::warning
Kung ang isang folder ay nagpapakita ng "No content available," ang provider ay walang anumang item sa antas na iyon. Subukang mag-navigate sa ibang folder o suriin muli mamaya.
:::
