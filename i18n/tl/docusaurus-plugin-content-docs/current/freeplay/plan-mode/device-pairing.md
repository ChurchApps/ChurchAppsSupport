# Pag-pair ng Iyong Device

<div class="article-intro">

Upang magamit ang Plan Mode, kailangan mong i-pair ang iyong TV sa isang uri ng plano sa B1 Admin. Ang FreePlay ay bumubuo ng natatanging pairing code na nagkokonekta sa device sa plano ng iyong simbahan, na nagbibigay-daan sa awtomatikong paghahatid ng nilalaman bawat linggo.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- I-install at ilunsad ang FreePlay -- tingnan ang [Pagsisimula](../getting-started/)
- Piliin ang **Pair to Plan** mula sa [screen ng pairing mode](../getting-started/pairing-modes.md)
- Magkaroon ng access sa **B1 Admin** sa isang computer o telepono upang kumpletuhin ang pairing

</div>

## Pagbuo ng Pairing Code

1. Mula sa screen ng **Select Pairing Mode**, piliin ang **Pair to Plan**
2. Nakikipag-ugnayan ang FreePlay sa server at bumubuo ng maikling pairing code
3. Ang code ay lalabas sa malalaking karakter sa gitna ng screen
4. Sa ibaba ng code, isang pulsing na indicator ang nagpapakita ng **Waiting for connection**

Ang code ay ipinapakita bilang mga indibidwal na karakter para sa madaling pagbasa mula sa kabila ng silid.

## Paglagay ng Code sa B1 Admin

1. Sa isang computer o telepono, buksan ang **B1 Admin** para sa iyong simbahan
2. Mag-navigate sa mga setting ng **Plan Type** para sa plano na gusto mong italaga
3. Ilagay ang pairing code na ipinapakita sa TV
4. I-save ang mga setting

Nagpo-poll ang FreePlay sa server tuwing ilang segundo, sinusuri kung kumpleto na ang pairing. Kapag nakumpirma ng **B1 Admin** ang koneksyon, awtomatikong lilipat ang TV sa plan download screen.

## Pag-download ng Nilalaman ng Plano

Pagkatapos ng pairing, nilo-load ng FreePlay ang kasalukuyang plano para sa uri ng plano na iyon. Ipinapakita nito ang:

- Ang pangalan ng plano at petsa ng serbisyo
- Ang kaugnay na pangalan ng aralin (kung ang plano ay may kasamang nilalaman ng aralin)
- Isang progress indicator na nagpapakita ng **Downloading item X of Y**

Kapag na-download na ang lahat ng media file, lalabas ang button na **Start Plan**. Pindutin ang **Select** sa iyong remote upang simulan ang playback.

:::tip
Awtomatikong nagre-refresh ang plano tuwing isang oras. Kung na-update ang plano sa loob ng araw, kukunin ng FreePlay ang mga pagbabago nang walang manu-manong interbensyon.
:::

## Alternatibo: Maghanap Ayon sa Pangalan ng Simbahan

Kung mas gusto mong hindi gamitin ang daloy ng pairing code, maaari mong piliin ang **or search by church name** sa ibaba ng pairing screen. Dadalhin ka nito sa screen ng paghahanap ng simbahan kung saan maaari mong hanapin ang iyong simbahan at kumonekta sa isang silid-aralan sa halip.

## Kung Nabigo ang Pairing

Kung hindi mabuo ang pairing code (halimbawa, dahil sa problema sa network), makikita mo ang isang error message na may button na **Try Again**. Siguraduhing nakakonekta ang iyong TV sa internet at subukan muli.

:::warning
Ang mga pairing code ay nag-e-expire pagkatapos ng itinakdang panahon. Kung masyadong matagal kang naghintay, bumuo ng bagong code sa pamamagitan ng pagbabalik sa pairing screen.
:::
