# Classroom Mode

<div class="article-intro">

Ang Classroom Mode ay nagkokonekta ng iyong TV sa isang partikular na silid sa iyong simbahan. Kapag naka-pair na, awtomatikong dina-download ng FreePlay ang aralin na naka-iskedyul para sa silid na iyon, kaya ang iyong mga boluntaryo ay basta na lang pipindutin ang **Start Lesson** kapag nagsimula na ang klase.

</div>

## Paano Ito Gumagana

1. **Hanapin ang iyong simbahan** - Maghanap ng iyong simbahan ayon sa pangalan
2. **Pumili ng silid** - Piliin ang silid-aralan kung saan nakatalaga ang TV na ito
3. **I-download ang aralin** - Kinukuha ng FreePlay ang naka-iskedyul na aralin at dina-download ang lahat ng mga media file
4. **I-play** - Pindutin ang button na **Start Lesson** at awtomatikong magpe-play ang aralin

Naaalala ng TV kung aling simbahan at silid ang naka-pair dito, kaya isang beses mo lang kailangang dumaan sa setup. Bawat linggo, sinusuri ng FreePlay ang naka-iskedyul na aralin para sa kasalukuyang araw at dina-download ang anumang bagong nilalaman.

:::tip
Siguraduhing naka-configure ang iyong mga silid-aralan sa [freeplay.church](https://freeplay.church) bago i-pair ang iyong TV. Kung walang lumabas na mga silid-aralan kapag pinili mo ang iyong simbahan, bisitahin ang website upang i-set up ang mga ito.
:::
