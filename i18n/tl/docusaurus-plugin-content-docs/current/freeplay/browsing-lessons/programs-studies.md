# Mga Programa at Pag-aaral

<div class="article-intro">

Ang browser ng aralin ay gumagamit ng hakbang-hakbang na daloy upang paliitin ang eksaktong nilalaman na gusto mong i-play. Ang bawat hakbang ay nagpapakita ng 3-column na grid ng mga card na may mga larawan at pamagat, na nagpapadali sa pag-navigate sa buong library ng aralin gamit ang iyong TV remote.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Siguraduhing may koneksyon sa internet ang iyong TV (kailangan para sa pagba-browse ng catalog ng programa)
- Mag-navigate sa screen ng **Browse Programs** gamit ang sidebar

</div>

## Pagba-browse ng mga Programa

1. Buksan ang screen ng **Browse Programs**
2. Ang mga programa ay lalabas bilang isang grid ng mga image card, tatlo sa bawat hanay
3. Gamitin ang directional pad sa iyong remote upang i-highlight ang isang programa
4. Pindutin ang **Select** upang buksan ito

Ang bawat card ng programa ay nagpapakita ng cover image ng programa. Kung walang available na larawan, isang placeholder card ang ipapakita.

## Pagpili ng Pag-aaral

Pagkatapos pumili ng programa, ang screen ng **Select a Study** ay maglo-load kasama ang lahat ng pag-aaral sa programang iyon.

1. Ang mga card ng pag-aaral ay lalabas sa isang 3-column na grid na may mga cover image at pangalan ng pag-aaral
2. Mag-navigate sa pag-aaral na gusto mo at pindutin ang **Select**
3. Pindutin ang **Back** upang bumalik sa listahan ng mga programa

## Pagpili ng Aralin

Pagkatapos pumili ng pag-aaral, ang screen ng **Select a Lesson** ay magpapakita ng lahat ng aralin sa pag-aaral na iyon.

1. Ang mga card ng aralin ay nagpapakita ng mga cover image at pamagat ng aralin
2. Mag-navigate sa aralin na gusto mo at pindutin ang **Select**
3. Pindutin ang **Back** upang bumalik sa listahan ng mga pag-aaral

## Pagpili ng Venue

Ang screen ng detalye ng aralin ay nagpapakita ng pangalan ng aralin, pamagat, at paglalarawan na nakapatong sa cover image ng aralin. Sa ibaba ng paglalarawan, makakakita ka ng isa o higit pang mga button ng venue.

Ang mga venue ay kumakatawan sa iba't ibang bersyon ng parehong aralin. Halimbawa, ang isang aralin ay maaaring may hiwalay na mga venue para sa "Preschool," "Elementary," at "Pre-Teen" na mga grupo ng edad. Ang bawat venue ay naglalaman ng mga media file na angkop para sa kapaligiran na iyon.

1. Gamitin ang directional pad upang i-highlight ang venue na gusto mo
2. Pindutin ang **Select** upang simulan ang pag-load ng nilalaman
3. Kukunin ng FreePlay ang playlist ng venue at sisimulan ang player

:::tip
Kung isang venue button lang ang nakikita mo, isang bersyon lang ng aralin ang available. Piliin ito upang magsimulang mag-play kaagad.
:::

## Pag-navigate Pabalik

Sa anumang punto ng daloy ng pagba-browse, pindutin ang **Back** sa iyong remote upang bumalik sa nakaraang antas:

- Mula sa screen ng detalye ng aralin, ang **Back** ay babalik sa listahan ng mga aralin
- Mula sa listahan ng mga aralin, ang **Back** ay babalik sa listahan ng mga pag-aaral
- Mula sa listahan ng mga pag-aaral, ang **Back** ay babalik sa grid ng mga programa
- Mula sa grid ng mga programa, ang **Back** ay magbubukas ng sidebar
