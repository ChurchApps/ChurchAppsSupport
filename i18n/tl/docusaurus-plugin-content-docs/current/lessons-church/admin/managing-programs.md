# Pamamahala ng mga Programa

<div class="article-intro">

Ang mga programa ay ang top-level na mga lalagyan para sa kurikulum sa Lessons.church. Ang isang programa ay kumakatawan sa isang buong serye ng kurikulum -- halimbawa, "Ark Kids" o "Youth Foundations." Sinasaklaw ng gabay na ito ang paglikha, pag-edit, at pag-aayos ng mga programa sa admin area.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Magkaroon ng mga pahintulot ng admin para sa Lessons.church (tingnan ang [Pangkalahatang-tanaw ng Administrasyon](./index.md))
- Maghanda ng pangalan ng programa, paglalarawan, at thumbnail image

</div>

## Paglikha ng Programa

1. Mag-navigate sa **Admin** area.
2. I-click ang **Add Program**.
3. Punan ang mga sumusunod na field:
   - **Name** -- Ang display name para sa programa (hal., "Ark Kids").
   - **Slug** -- Isang URL-friendly na identifier (hal., "ark-kids"). Ginagamit ito sa mga web address.
   - **Description** -- Isang maikling buod ng kung ano ang sinasaklaw ng programa.
   - **Image** -- Mag-upload ng thumbnail image na kumakatawan sa programa.
4. I-save ang programa.

Ang iyong bagong programa ay lalabas na ngayon sa admin program list at, kapag mayroon na itong mga pag-aaral at aralin, sa public-facing na site.

## Pag-edit ng Programa

1. I-click ang pangalan ng programa sa admin list.
2. I-update ang alinman sa mga field (**Name**, **Slug**, **Description**, o **Image**).
3. I-save ang iyong mga pagbabago.

## Pag-aayos ng mga Pag-aaral sa Loob ng Programa

Kapag mayroon nang programa, maaari kang magdagdag ng mga pag-aaral dito. Ang mga pag-aaral ay lalabas na naka-nest sa ilalim ng programa sa admin view. I-click ang expand arrow sa tabi ng isang programa upang makita ang mga pag-aaral nito, o magdagdag ng bago mula doon. Tingnan ang [Pamamahala ng mga Pag-aaral](./managing-studies.md) para sa mga detalye.

## Mga Kategorya ng Pag-aaral

Kung ang iyong programa ay may maraming pag-aaral, maaari mo itong ayusin sa mga **kategorya**. Ang mga kategorya ay naggu-grupo ng mga pag-aaral ayon sa paksa, saklaw ng edad, o anumang ibang pamantayan na iyong piliin. Nagpapadali ito para sa mga gumagamit na mag-browse ng isang malaking programa.

## Pag-upload ng mga Mapagkukunan sa Antas ng Programa

Maaari kang mag-attach ng mga nada-download na file sa isang programa. Ito ay mga mapagkukunan na nauugnay sa kabuuang programa sa halip na isang partikular na pag-aaral o aralin -- halimbawa, isang scope-and-sequence na dokumento o handbook ng boluntaryo.

1. Buksan ang programa sa admin view.
2. Mag-navigate sa seksyon ng **Resources**.
3. I-upload ang iyong mga file.

## Pag-clear ng Cache

Pagkatapos gumawa ng mga pagbabago sa isang programa, maaaring kailangan mong i-clear ang cache upang lumabas ang mga update sa public site. Gamitin ang opsyon na **Clear Cache** sa admin area upang i-refresh ang nilalaman.

:::tip
Pumili ng isang deskriptibong pangalan ng programa at sumulat ng malinaw na paglalarawan. Ito ang unang nakikita ng mga simbahan kapag nagba-browse sa library ng kurikulum, kaya ang isang magandang buod ay nakakatulong sa kanila na magpasya kung angkop ang programa para sa kanilang mga pangangailangan.
:::
