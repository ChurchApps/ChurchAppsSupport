# Pamamahala ng mga Aralin

<div class="article-intro">

Ang mga aralin ay ang mga indibidwal na sesyon ng pagtuturo sa loob ng isang pag-aaral. Ang bawat aralin ay kumakatawan sa kung ano ang nangyayari sa isang pagpupulong ng klase. Dito mo bubuo ang detalyadong nilalaman -- mga venue, seksyon, tungkulin, at aksyon -- na gagamitin ng mga boluntaryo kapag nagtuturo sila.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [Gumawa ng programa](./managing-programs.md) at [magdagdag ng pag-aaral](./managing-studies.md) dito
- Magkaroon ng mga pahintulot ng admin para sa Lessons.church (tingnan ang [Pangkalahatang-tanaw ng Administrasyon](./index.md))

</div>

## Paglikha ng Aralin

1. Mag-navigate sa **Admin** area, palawakin ang programa at pag-aaral, at hanapin ang pag-aaral kung saan mo gustong magdagdag ng aralin.
2. I-click ang **Add Lesson**.
3. Maglagay ng **Name** at iba pang detalye para sa aralin.
4. I-save ang aralin.

## Pag-unawa sa Istruktura ng Aralin

Ang bawat aralin ay nakaayos sa ilang mga layer:

- **Mga Venue** -- Iba't ibang bersyon ng aralin para sa iba't ibang grupo ng edad o kapaligiran. Halimbawa, maaari kang magkaroon ng "Preschool" venue at "Elementary" venue para sa parehong aralin.
- **Mga Seksyon** -- Ang mga segment sa loob ng isang venue, tulad ng "Welcome", "Video", "Bible Story", at "Activity". Kinokontrol ng mga seksyon ang daloy ng oras ng klase.
- **Mga Tungkulin at Aksyon** -- Sa loob ng bawat seksyon, nagtatakda ka ng mga tungkulin (halimbawa, "Lead Teacher" o "Small Group Leader") at binibigyan ang bawat tungkulin ng mga partikular na aksyon.

## Mga Uri ng Aksyon

May apat na uri ng aksyon na maaari mong italaga sa isang tungkulin sa loob ng isang seksyon:

- **Say** -- Teksto na dapat sabihin o i-paraphrase ng boluntaryo sa klase.
- **Do** -- Isang pisikal na aktibidad o instruksiyon na gagawin.
- **Note** -- Impormasyon sa background o paalala para sa boluntaryo (hindi binabasa nang malakas).
- **Play** -- Isang video o audio clip na ipe-play habang nasa seksyon.

## Pagdaragdag ng Venue

1. Buksan ang aralin sa admin view.
2. I-click ang **Add Venue**.
3. Maglagay ng pangalan para sa venue (hal., "Preschool" o "Elementary").
4. I-save. Maaari ka nang magdagdag ng mga seksyon sa venue na ito.

## Pagdaragdag ng mga Seksyon at Aksyon

1. Palawakin ang isang venue upang makita ang mga seksyon nito.
2. I-click ang **Add Section** at bigyan ito ng pangalan (hal., "Welcome").
3. Sa loob ng seksyon, i-click ang **Add Action**.
4. Piliin ang **tungkulin**, **uri ng aksyon** (Say, Do, Note, o Play), at ilagay ang nilalaman.
5. I-save.

Ulitin ang prosesong ito upang buuin ang buong daloy ng aralin.

## Pag-upload ng mga Mapagkukunan sa Antas ng Aralin

Maaari kang mag-attach ng mga nada-download na materyal nang direkta sa isang aralin -- halimbawa, isang activity sheet o coloring page na partikular para sa linggong iyon.

1. Buksan ang aralin sa admin view.
2. Mag-navigate sa seksyon ng **Resources**.
3. I-upload ang iyong mga file.

:::tip
Buuin ang iyong aralin mula sa pananaw ng boluntaryo. Dumaan sa mga seksyon at aksyon nang sunud-sunod at tanungin ang iyong sarili kung maaaring sumunod ang isang first-time na boluntaryo nang walang ibang paghahanda.
:::
