# Pamamahala ng mga Pag-aaral

<div class="article-intro">

Ang pag-aaral ay isang serye ng magkakaugnay na aralin sa loob ng isang programa. Halimbawa, ang isang apat na linggong serye na tinatawag na "The Good Samaritan" ay isang pag-aaral sa loob ng isang mas malaking programa ng ministeryo ng mga bata. Sinasaklaw ng gabay na ito ang paglikha, pag-edit, at pag-aayos ng mga pag-aaral sa admin area.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [Gumawa ng hindi bababa sa isang programa](./managing-programs.md) sa admin area
- Magkaroon ng mga pahintulot ng admin para sa Lessons.church (tingnan ang [Pangkalahatang-tanaw ng Administrasyon](./index.md))

</div>

## Paglikha ng Pag-aaral

1. Mag-navigate sa **Admin** area at palawakin ang programa kung saan mo gustong magdagdag ng pag-aaral.
2. I-click ang **Add Study**.
3. Punan ang mga sumusunod na field:
   - **Name** -- Ang display name para sa pag-aaral (hal., "The Good Samaritan").
   - **Slug** -- Isang URL-friendly na identifier (hal., "the-good-samaritan").
   - **Description** -- Isang buod ng kung ano ang sinasaklaw ng pag-aaral na ito at kung ilang aralin ang kasama.
   - **Image** -- Mag-upload ng thumbnail image para sa pag-aaral.
4. I-save ang pag-aaral.

## Pag-edit ng Pag-aaral

1. I-click ang pangalan ng pag-aaral sa admin list.
2. I-update ang alinman sa mga field kung kinakailangan.
3. I-save ang iyong mga pagbabago.

## Pagtatalaga ng Kategorya

Kung ang parent program ay gumagamit ng mga kategorya ng pag-aaral, maaari mong italaga ang iyong pag-aaral sa isa:

1. Buksan ang pag-aaral sa admin view.
2. Pumili ng **Category** mula sa dropdown.
3. I-save.

Nakakatulong ang mga kategorya sa mga gumagamit na mag-browse ng mga programa na may maraming pag-aaral. Halimbawa, maaaring mayroon kang mga kategorya tulad ng "Old Testament", "New Testament", at "Topical".

## Pamamahala ng mga Aralin sa Loob ng Pag-aaral

Ang mga pag-aaral ay naglalaman ng isa o higit pang aralin. Palawakin ang isang pag-aaral sa admin view upang makita ang mga aralin nito, o i-click ang **Add Lesson** upang gumawa ng bago. Tingnan ang [Pamamahala ng mga Aralin](./managing-lessons.md) para sa mga detalye sa pagbuo ng nilalaman ng aralin.

## Pag-upload ng mga Mapagkukunan sa Antas ng Pag-aaral

Maaari kang mag-attach ng mga nada-download na file na nauugnay sa buong pag-aaral -- halimbawa, isang pangkalahatang-tanaw ng serye, mga promotional na graphics, o template ng newsletter para sa mga magulang.

1. Buksan ang pag-aaral sa admin view.
2. Mag-navigate sa seksyon ng **Resources**.
3. I-upload ang iyong mga file.

:::tip
Isulat ang paglalarawan ng iyong pag-aaral na may pag-iisip sa mga lider ng simbahan. Ang malinaw na buod ng tema, target na grupo ng edad, at bilang ng mga aralin ay nakakatulong sa mga simbahan na magpasya kung gagamitin ang serye.
:::
