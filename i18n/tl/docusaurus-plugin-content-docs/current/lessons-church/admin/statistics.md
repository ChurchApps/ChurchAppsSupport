# Mga Istatistika

<div class="article-intro">

Ang pahina ng mga istatistika ay nagbibigay sa mga admin ng kaalaman kung paano ginagamit ang kanilang nilalaman. Makikita mo kung aling mga pag-aaral ang pinakasikat, kung aling mga simbahan ang nagda-download ng iyong mga mapagkukunan, at kung saan matatagpuan ang mga simbahang iyon sa heograpiya.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Magkaroon ng mga pahintulot ng admin para sa Lessons.church (tingnan ang [Pangkalahatang-tanaw ng Administrasyon](./index.md))
- Magkaroon ng hindi bababa sa isang [programa](./managing-programs.md) na may nai-publish na nilalaman

</div>

## Pag-access ng mga Istatistika

1. Mag-navigate sa **Admin** area.
2. Piliin ang pahina ng **Stats** o **Statistics**.
3. Piliin ang **programa** na gusto mong tingnan ang mga istatistika.

## Ano ang Maaari Mong Makita

Ang pahina ng mga istatistika ay nagbibigay ng ilang mga view ng iyong data sa pag-download:

### Mga Download Ayon sa Pag-aaral

Isang breakdown ng mga natatanging download para sa bawat pag-aaral sa loob ng napiling programa. Ipinapakita nito sa iyo kung aling mga serye ang nakakakuha ng pinakamaraming atensyon at nakakatulong sa iyo na magpasya kung saan mag-invest ng iyong mga pagsisikap sa paglikha ng nilalaman.

### Listahan ng mga Simbahan

Isang listahan ng mga simbahan na nag-download ng mga mapagkukunan mula sa iyong programa. Nagbibigay ito sa iyo ng ideya tungkol sa iyong audience at kung gaano kalawak na inaangkop ang iyong nilalaman.

### Distribusyon ayon sa Heograpiya

Isang map view na nagpapakita kung saan matatagpuan ang iyong mga nagda-download na simbahan. Nakakatulong ito sa iyo na maunawaan ang heograpikal na abot ng iyong kurikulum at matukoy ang mga rehiyon kung saan sikat ang iyong nilalaman.

### Pag-filter ayon sa Saklaw ng Petsa

Maaari mong i-filter ang lahat ng istatistika ayon sa saklaw ng petsa upang mag-focus sa isang partikular na panahon. Ito ay kapaki-pakinabang para sa pagtingin ng mga trend sa paglipas ng panahon o pagsukat ng epekto ng isang bagong inilabas na pag-aaral.

1. Itakda ang **petsa ng simula** at **petsa ng pagtatapos** gamit ang mga date picker.
2. Mag-a-update ang mga istatistika upang ipakita lamang ang aktibidad sa loob ng saklaw na iyon.

## Epektibong Paggamit ng mga Istatistika

Narito ang ilang paraan upang magamit ang iyong data sa pag-download:

- **Tukuyin ang sikat na nilalaman** -- Ituon ang iyong pagsisikap sa paglikha ng mas maraming materyal sa estilo at mga paksa na nakakakuha ng pinakamaraming download.
- **Subaybayan ang mga bagong release** -- Pagkatapos mag-publish ng bagong pag-aaral, mag-filter ayon sa petsa upang makita kung gaano kabilis itong pinapansin.
- **Unawain ang iyong audience** -- Gamitin ang listahan ng mga simbahan at mapa upang makita kung sino ang gumagamit ng iyong nilalaman at isaalang-alang ang pakikipag-ugnayan sa mga komunidad na iyon.

:::info
Ang mga istatistika ay batay sa mga natatanging download. Kung ang parehong simbahan ay nag-download ng mapagkukunan nang maraming beses, ito ay binibilang nang isang beses.
:::
