# Administrasyon

<div class="article-intro">

Ang admin area sa Lessons.church ay kung saan pinamamahalaan ng mga tagalikha ng nilalaman at mga administrator ang library ng aralin. Mula rito maaari mong buuin ang istruktura ng iyong kurikulum, mag-upload ng mga mapagkukunan, suriin ang mga istatistika ng paggamit, at ikonekta ang mga third-party na content provider.

</div>

:::warning
Ang admin access ay nangangailangan ng naaangkop na mga pahintulot. Kung hindi mo nakikita ang mga opsyon ng admin, makipag-ugnayan sa administrator ng iyong simbahan upang humiling ng access.
:::

## Ano ang Maaari Mong Gawin

Ang admin area ay nagbibigay sa iyo ng ganap na kontrol sa iyong nilalaman:

- **Gumawa at mag-edit ng mga programa** -- Mag-set up ng mga top-level na serye ng kurikulum na may mga pangalan, paglalarawan, larawan, at slug.
- **Pamahalaan ang mga pag-aaral** -- Ayusin ang mga grupo ng magkakaugnay na aralin sa loob ng bawat programa.
- **Gumawa ng mga aralin** -- Magdagdag ng mga indibidwal na aralin na may mga venue, seksyon, tungkulin, at aksyon.
- **Mag-upload ng mga mapagkukunan** -- Mag-attach ng mga nada-download na file (PDF, activity sheet, larawan) sa antas ng programa, pag-aaral, o aralin.
- **Tingnan ang mga istatistika** -- Alamin kung aling nilalaman ang dina-download, kung sino ang nagda-download, at kung saan.
- **I-configure ang mga provider** -- Ikonekta ang mga external na provider ng kurikulum na gumagamit ng Open Lesson Format.

## Hierarchy ng Nilalaman

Bilang isang admin, nakakatulong na maunawaan kung paano nakaayos ang nilalaman:

1. Ang **Mga Programa** ay naglalaman ng isa o higit pang pag-aaral.
2. Ang **Mga Pag-aaral** ay naglalaman ng isa o higit pang aralin.
3. Ang **Mga Aralin** ay naglalaman ng isa o higit pang venue (mga pagkakaiba-iba ayon sa grupo ng edad).
4. Ang **Mga Venue** ay naglalaman ng mga seksyon, at ang mga seksyon ay naglalaman ng mga aksyon na batay sa tungkulin.

Ang bawat antas ay maaaring magkaroon ng sariling nada-download na mga mapagkukunan na naka-attach.
