# Mga Download at Materyal

<div class="article-intro">

Ang mga aralin sa Lessons.church ay may kasamang mga nada-download na mapagkukunan na maaari mong i-print o ibahagi nang digital. Sinusuportahan ng mga materyal na ito ang nilalaman ng aralin at nagbibigay sa iyong mga boluntaryo at estudyante ng isang bagay na kongkreto na magagamit habang at pagkatapos ng klase.

</div>

<h4>Bago Ka Magsimula</h4>

- [Mag-browse sa isang aralin](../browsing/viewing-lessons.md) o buksan ang isang naka-iskedyul na aralin mula sa iyong [iskedyul ng silid-aralan](../classrooms/scheduling-lessons.md)

</div>

## Mga Uri ng Materyal

Depende sa aralin at provider, maaari kang makahanap ng mga mapagkukunan tulad ng:

- **Mga activity sheet** -- Mga hands-on na worksheet para kumpletuhin ng mga estudyante habang nasa klase.
- **Mga coloring page** -- Mga napi-print na pahina para sa mas batang grupo ng edad.
- **Mga gabay para sa lider** -- Detalyadong mga instruksiyon at background na impormasyon para sa mga guro.
- **Mga supplementary PDF** -- Karagdagang pagbasa, take-home sheet, o mga handout para sa mga magulang.

Hindi lahat ng aralin ay may kasamang lahat ng ito. Ang mga available na materyal ay depende sa kung ano ang ibinigay ng provider ng nilalaman.

## Saan Makikita ang mga Download

Ang mga mapagkukunan ay available sa tatlong antas:

- **Antas ng programa** -- Mga pangkalahatang mapagkukunan na naaangkop sa buong serye ng kurikulum.
- **Antas ng pag-aaral** -- Mga materyal na nakatali sa isang partikular na pag-aaral sa loob ng isang programa.
- **Antas ng aralin** -- Mga mapagkukunan para sa isang indibidwal na aralin.

Upang mag-download ng mga materyal:

1. Mag-navigate sa pahina ng aralin, pag-aaral, o programa.
2. Hanapin ang seksyon ng **Downloads** o **Resources** sa pahina.
3. Mag-click sa pangalan ng file upang i-download ito sa iyong device.

Ang mga file ay karaniwang nasa PDF format at handa nang i-print.

## Paghahanda para sa Klase

Ang magandang gawain ay suriin ang mga nada-download na materyal ilang araw bago ang klase:

1. Buksan ang paparating na aralin at suriin ang mga available na download.
2. Mag-print ng sapat na kopya ng mga activity sheet at coloring page para sa iyong inaasahang bilang ng dumadalo.
3. Suriin ang gabay para sa lider upang pamilyar ka sa daloy ng aralin.
4. Maglaan ng anumang take-home na materyal na ipamimigay sa dulo ng klase.

:::info
Ang ilang provider ay nagsasama ng lahat ng kanilang mga materyal sa antas ng pag-aaral sa halip na i-attach ang mga ito sa mga indibidwal na aralin. Kung hindi ka nakakakita ng mga download sa isang partikular na pahina ng aralin, suriin ang parent na pag-aaral o pahina ng programa.
:::

:::tip
Gumawa ng lingguhang checklist para sa iyong mga boluntaryo na kasama ang pag-download at pag-print ng mga materyal. Tinitiyak nito na walang makakalimutan bago ang umaga ng Linggo.
:::
