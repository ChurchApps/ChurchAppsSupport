# Mga Silid-aralan

<div class="article-intro">

Ang mga silid-aralan ay ang pundasyon kung paano mo inaayos ang pagtuturo sa iyong simbahan sa Lessons.church. Ang isang silid-aralan ay kumakatawan sa isang pisikal o virtual na silid kung saan itinuturo ang mga aralin -- halimbawa, "Preschool Room," "Elementary K-2," o "Adult Bible Study." Sinasaklaw ng seksyong ito ang paglikha ng mga silid-aralan at pag-iskedyul ng mga aralin sa kanila.

</div>

## Bakit Mahalaga ang mga Silid-aralan

Kapag nag-set up ka ng mga silid-aralan, gumagawa ka ng malinaw na istruktura para sa iyong ministeryo ng mga bata, mga grupo ng kabataan, o anumang kapaligiran ng pagtuturo. Ang bawat silid-aralan ay may sariling iskedyul, kaya palaging alam ng iyong mga boluntaryo kung ano ang ituturo at kailan. Sa halip na magpasa ng mga papel na iskedyul o magpadala ng mga huling minutong email, maaaring suriin ng lahat ang portal upang makita kung ano ang paparating.

## Ano ang Maaari Mong Gawin sa mga Silid-aralan

- **Gumawa ng walang limitasyong mga silid-aralan** -- Mag-set up ng kasing dami ng silid-aralan na kailangan ng iyong simbahan. Walang limitasyon, kaya maaari kang magkaroon ng hiwalay na silid-aralan para sa bawat grupo ng edad, campus, o programa.
- **Mag-iskedyul ng mga aralin** -- Italaga ang mga partikular na aralin sa bawat silid-aralan sa mga partikular na petsa. Ang iyong iskedyul ay nagpapakita ng parehong mga nakaraang at paparating na aralin sa isang sulyap.
- **I-customize ang nilalaman** -- I-angkop ang nilalaman ng aralin para sa bawat silid-aralan upang ang mga boluntaryo ay nakikita lang ang kung ano ang nauugnay sa kanilang grupo ng edad. Tingnan ang [Pag-customize](../customization/index.md) para sa mga detalye.

## Paano Magkakaugnay ang mga Silid-aralan

Ikinokonekta ng mga silid-aralan ang nilalaman ng aralin sa Lessons.church sa tunay na iskedyul ng pagtuturo ng iyong simbahan. Narito ang karaniwang daloy ng trabaho:

1. [Mag-browse](../browsing/index.md) ng mga available na programa at pag-aaral sa Lessons.church.
2. I-set up ang iyong mga silid-aralan upang tumugma sa mga silid at grupo ng edad ng iyong simbahan.
3. Mag-iskedyul ng mga aralin sa bawat silid-aralan para sa mga paparating na petsa.
4. Binubuksan ng mga boluntaryo ang kanilang silid-aralan upang makita kung ano ang naka-iskedyul at maghanda.
