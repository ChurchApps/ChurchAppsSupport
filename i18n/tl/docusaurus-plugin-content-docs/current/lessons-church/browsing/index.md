# Pagba-browse ng Nilalaman

<div class="article-intro">

Ang Lessons.church ay nag-aalok ng libreng library ng kurikulum ng simbahan na maaaring galugarin ng kahit sino. Hindi mo kailangan ng account o login upang mag-browse ng available na nilalaman -- bisitahin lang ang site at magsimulang tumingin. Sinasaklaw ng seksyong ito ang lahat ng paraan upang maghanap at mag-explore ng kurikulum.

</div>

## Ang Home Page

Kapag unang dumating ka sa Lessons.church, ang home page ay nagbibigay sa iyo ng mabilisang pangkalahatang-tanaw ng lahat ng available:

- **Mga featured na programa** -- Ang mga naka-highlight na serye ng kurikulum ay lalabas sa itaas, na nagpapadali sa pagdiskubre ng sikat o kamakailang na-update na nilalaman.
- **Lahat ng programa** -- Sa ibaba ng featured na seksyon, makikita mo ang kumpletong listahan ng bawat available na programa ng kurikulum kasama ang mga istatistika tulad ng bilang ng mga aralin at pag-aaral sa bawat isa.
- **Nilalaman mula sa provider** -- Ang nilalaman mula sa Lessons.church at mga external na provider ng kurikulum ay magkasamang ipinapakita, na nagbibigay sa iyo ng iisang lugar upang mag-explore.

## Mga Paraan upang Maghanap ng Nilalaman

May ilang paraan upang mag-navigate sa library ng kurikulum:

1. **Mag-browse ayon sa programa** -- Mag-click sa anumang programa sa home page upang makita ang mga pag-aaral at aralin nito. Ito ang pinakamahusay na paraan kapag gusto mong galugarin ang isang partikular na serye ng kurikulum. Tingnan ang [Mga Programa at Pag-aaral](./programs-and-studies.md) para sa mga detalye.

2. **Maghanap** -- Gamitin ang pahina ng paghahanap upang maghanap ng mga aralin ayon sa keyword sa buong library. Ito ay kapaki-pakinabang kapag alam mo ang paksa ngunit hindi ka sigurado kung aling programa ang sumasaklaw dito. Tingnan ang [Paghahanap](./searching.md) para sa mga detalye.

3. **Galugarin ang mga external na provider** -- Ang ilang nilalaman ay nagmumula sa mga third-party na provider ng kurikulum na nag-publish sa Open Lesson Format. Maaari mong i-browse ang kanilang mga programa at pag-aaral tulad ng built-in na nilalaman. Tingnan ang [Mga External na Provider](./external-providers.md) para sa mga detalye.

## Hindi Kailangan ng Login

Ang pagba-browse ay ganap na bukas. Maaari mong tingnan ang mga programa, pag-aaral, aralin, mga nada-download na materyal, at video nang hindi gumagawa ng account. Kung magpasya kang gusto mong mag-iskedyul ng mga aralin para sa iyong mga silid-aralan o mag-customize ng nilalaman, maaari kang [gumawa ng libreng account](../getting-started/creating-account.md) sa puntong iyon.

:::tip
Magsimula sa pamamagitan ng pag-explore ng ilang programa sa home page upang makuha ang pakiramdam kung paano nakaayos ang kurikulum bago sumisid sa isang partikular na pag-aaral.
:::
