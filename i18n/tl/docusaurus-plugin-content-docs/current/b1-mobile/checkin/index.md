# Check-In

<div class="article-intro">

Ang B1 Mobile ay nagbibigay-daan sa iyong i-check in ang iyong sarili at ang mga miyembro ng iyong sambahayan para sa mga serbisyo ng simbahan nang direkta mula sa iyong telepono. Laktawan ang pila sa kiosk at i-record ang iyong attendance bago ka pa man pumasok sa pintuan.

</div>

## Paano Ito Gumagana

Ang proseso ng check-in ay isang mabilis na multi-step na daloy na gagabay sa iyo sa pagpili ng serbisyo, pagpili kung sino ang dadalo, at pag-assign ng bawat tao sa isang grupo. Kapag na-confirm mo na, awtomatikong nire-record ang iyong attendance.

## Pagsisimula

Para ma-access ang check-in, buksan ang seksyong **Check-In** sa B1 Mobile. Kailangan mong naka-log in para magamit ang feature na ito. Kung hindi ka naka-sign in, hihilingin sa iyong [mag-log in](../getting-started/logging-in) muna.

:::info
Ang feature ng check-in ay nangangailangan na ang iyong simbahan ay nag-set up ng mga serbisyo at attendance group sa kanilang mga setting ng administrasyon. Kung walang lumabas na mga serbisyo kapag binuksan mo ang check-in screen, maaaring hindi pa na-configure ng iyong simbahan ang feature na ito. Makipag-ugnayan sa iyong church administrator para sa tulong.
:::

## Ano ang Maaari Mong Gawin

- **I-check in ang iyong sambahayan** -- Piliin kung aling mga miyembro ng pamilya ang dadalo at i-record ang attendance ng lahat nang sabay-sabay. Tingnan ang [Sambahayan](../profile/household.md) para malaman ang tungkol sa pamamahala ng iyong mga miyembro ng pamilya.
- **Pumili ng mga serbisyo at oras** -- Pumili mula sa mga available na serbisyo at time slot na na-configure ng iyong simbahan.
- **Mag-assign ng mga grupo** -- Ilagay ang bawat tao sa naaangkop na grupo para sa serbisyo, tulad ng Main Worship, Kids Room, o Youth Group.
