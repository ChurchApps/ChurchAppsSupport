# Dashboard

<div class="article-intro">

Ang dashboard ang iyong home screen sa B1 Mobile. Nagbibigay ito sa iyo ng mabilis na access sa mga feature na inaalok ng iyong simbahan, na nakaayos batay sa kung paano mo ginagamit ang app. Ang layout ay umaangkop sa paglipas ng panahon para ilagay ang iyong pinakaginagamit na mga feature sa harapan at gitna.

</div>

## Seksyon ng Pagbati

Sa itaas ng dashboard, makikita mo ang isang welcome message kasama ang logo ng iyong simbahan (o pangalan ng simbahan kung walang logo na nakatakda) at isang "stay connected" na tagline. Kinukumpirma ng seksyong ito kung aling simbahan ang kasalukuyan mong tinitingnan.

## Mga Featured Link

Sa ibaba ng seksyon ng pagbati, ang iyong pinakabinibisitang feature ay lumalabas bilang isang malaking **hero card** na may background image at pangalan ng feature. Ito ang feature na pinakamadalas mong gamitin, na inilalabas sa itaas para sa isang-tap na access.

## Mga Featured Card

Sa ilalim ng hero card, makikita mo ang hanggang dalawang **featured card** na magkatabi. Kinakatawan ng mga ito ang iyong pangalawa at pangatlong pinakabinibisitang feature. Ang bawat card ay nagpapakita ng background image at pangalan ng feature. I-tap ang alinmang card para pumunta nang direkta sa feature na iyon.

## Mga Mabilisang Aksyon

Ang natitirang mga feature ay lumalabas sa isang **Quick Actions** grid sa ibaba ng mga featured card. Ang bawat item ay nagpapakita ng icon at label. Ang mga ito ay lahat ng iba pang navigation link na na-configure ng iyong simbahan, na nagbibigay sa iyo ng access sa lahat mula sa [mga grupo](../groups/) at [pagbibigay](../giving/) hanggang sa [check-in](../checkin/), [mga sermon](../content/sermons.md), at marami pa.

## Paano Umaangkop ang Layout

Ang layout ng dashboard ay personalisado batay sa iyong mga pattern ng paggamit. Tuwing mag-ta-tap ka ng feature, tina-track ng app kung gaano mo kadalas bisitahin ito. Ang mga feature na mas madalas mong gamitin ay lumilipat pataas ng dashboard:

- Ang iyong **pinakabinibisitang** feature ang nagiging hero card.
- Ang iyong **pangalawa at pangatlong pinakabinibisitang** feature ang nagiging mga featured card.
- **Lahat ng iba pa** ay lumalabas sa Quick Actions grid.

Kung hindi mo pa nagagamit ang app, ang mga feature ay lumalabas sa default na pagkakasunud-sunod na na-configure ng iyong simbahan. Sa paglipas ng panahon, nagre-reshuffle ang dashboard para tumugma sa iyong mga gawi.

:::info
Ang mga feature na available sa iyong dashboard ay na-configure ng iyong church administrator sa pamamagitan ng [B1 Admin](../../b1-admin/). Kung hindi mo makita ang isang feature na inaasahan mo, makipag-ugnayan sa opisina ng iyong simbahan.
:::

## Pagre-refresh ng Dashboard

Hilahin pababa ang dashboard para i-refresh ito. Nire-reload nito ang mga link at nilalaman ng iyong simbahan para matiyak na nakikita mo ang pinakabagong impormasyon.

## Navigation

Mula sa dashboard, maaari mong maabot ang anumang bahagi ng app:

- I-tap ang **hamburger icon** (tatlong pahalang na linya) sa kaliwang itaas na sulok para buksan ang drawer menu na may lahat ng mga opsyon sa navigation.
- I-tap ang **bell icon** sa kanang itaas na sulok para tingnan ang iyong [mga notification](../community/notifications.md).
- I-tap ang anumang card o quick action item para pumunta nang direkta sa feature na iyon.
