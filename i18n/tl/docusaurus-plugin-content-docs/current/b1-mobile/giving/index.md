# Pagbibigay

<div class="article-intro">

Pinapadali ng B1 Mobile ang pagsuporta sa iyong simbahan na may kumpletong karanasan sa pagbibigay nang direkta mula sa iyong telepono. Gusto mo mang magbigay ng mabilisang regalo, suriin ang iyong kasaysayan ng pagbibigay, o pamahalaan ang iyong mga paraan ng pagbabayad, ang lahat ay available sa isang lugar.

</div>

## Mga Tab ng Pagbibigay

Ang seksyon ng pagbibigay ay nakaayos sa apat na tab:

- **Overview** -- Isang buod ng iyong aktibidad sa pagbibigay. Tingnan ang iyong year-to-date na kabuuan, ang bilang ng mga regalo na ibinigay mo, at mga detalye tungkol sa iyong huling donasyon, kasama ang mabilisang **Repeat** button para magbigay ulit ng parehong halaga.
- **Donate** -- Magbigay ng bagong donasyon. Pumili ng fund, maglagay ng halaga, at ibigay ang iyong mga detalye ng pagbabayad para makumpleto ang regalo.
- **Manage** -- Tingnan at pamahalaan ang iyong mga naka-save na paraan ng pagbabayad. Magdagdag ng mga bagong card o mag-alis ng mga hindi mo na ginagamit.
- **History** -- Isang detalyadong listahan ng lahat ng iyong nakaraang donasyon, para masuri mo ang iyong kumpletong rekord ng pagbibigay.

## Paano Gumagana ang mga Pagbabayad

Lahat ng donasyong ginawa sa pamamagitan ng B1 Mobile ay secure na pinoproseso sa pamamagitan ng **Stripe**. Ang iyong impormasyon sa pagbabayad ay hinahawakan ng secure na imprastraktura ng Stripe at hindi kailanman iniimbak sa mga server ng ChurchApps. Maaari kang magbigay nang may kumpiyansa na alam mong protektado ang iyong mga detalyeng pinansyal.

:::info
Ang iyong church administrator ang nagse-set up ng mga available na fund at payment processing sa pamamagitan ng [B1 Admin](../../b1-admin/). Kung hindi mo makita ang opsyon ng pagbibigay sa app, maaaring hindi pa na-configure ng iyong simbahan ang feature na ito.
:::

## Ano ang Maaari Mong Gawin

Narito ang maaari mong gawin mula sa seksyon ng pagbibigay:

- **Magbigay sa mga partikular na fund** -- Pumili kung saang fund mo idi-direct ang iyong donasyon, tulad ng General Fund, Missions Fund, o Building Fund.
- **Suriin ang iyong pagbibigay** -- Tingnan ang iyong year-to-date na buod at i-browse ang iyong buong kasaysayan ng donasyon.
- **Mag-save ng mga paraan ng pagbabayad** -- Itago ang iyong mga detalye ng card para sa mas mabilis na pagbibigay sa hinaharap.
- **Ulitin ang isang regalo** -- Mabilis na magbigay ng parehong halaga gaya ng iyong huling donasyon sa isang tap lang.
