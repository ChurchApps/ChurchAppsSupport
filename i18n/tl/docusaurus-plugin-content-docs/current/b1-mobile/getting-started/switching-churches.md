# Paglipat ng Simbahan

<div class="article-intro">

Kung kasapi ka sa higit sa isang simbahan, pinapadali ng B1 Mobile ang paglipat sa pagitan nila. Gumagana ang iyong account sa lahat ng simbahang gumagamit ng ChurchApps, kaya isang login lang ang kailangan mo.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [I-install ang B1 Mobile](./installing) at [mag-log in](./logging-in) sa iyong account

</div>

## Paano Lumipat ng Simbahan

1. Buksan ang drawer menu sa pamamagitan ng pag-tap sa **hamburger icon** (tatlong pahalang na linya) sa kaliwang itaas na sulok.
2. Sa drawer header, i-tap ang button na nagpapakita ng pangalan ng iyong kasalukuyang simbahan (o **Find a Church** kung walang napiling simbahan).
3. Lalabas ang screen na **Church Search**. Ang iyong kamakailang binisitang mga simbahan ay nasa itaas para sa mabilis na access.
4. I-tap ang isang kamakailang simbahan para lumipat dito, o mag-type sa search field para maghanap ng ibang simbahan.

Kapag lumipat ka ng simbahan, magre-reload ang app na may branding, navigation link, at nilalaman ng simbahang iyon. Ang iyong [dashboard](../dashboard/), [mga grupo](../groups/), at iba pang data ay magpapakita ng bagong napiling simbahan.

## Paghahanap ng Simbahan

Sa Church Search screen, mag-type ng hindi bababa sa 3 character ng pangalan ng simbahan sa search field. Ang mga tumutugmang simbahan ay lalabas sa isang listahan sa ibaba. I-tap ang isang simbahan para piliin ito.

:::info
Kung walang lumabas na resulta, subukan ang ibang spelling o mas maikling bahagi ng pangalan ng simbahan. Ang paghahanap ay tumutugma sa mga pangalan ng simbahang nakarehistro sa ChurchApps.
:::

## Mga Kamakailang Simbahan

Naaalala ng Church Search screen ang mga simbahang nabisita mo na dati. Kapag wala kang ni-type na search query, awtomatikong ipinapakita ang iyong mga kamakailang simbahan. Pinapabilis nito ang paglipat pabalik sa isang simbahang regular mong ginagamit.

:::tip
Hindi mo kailangang mag-log out at mag-log back in para lumipat ng simbahan. Gumagana ang iyong iisang ChurchApps account sa lahat ng iyong mga simbahan. Pumili lang ng ibang simbahan mula sa Church Search screen at handa ka na.
:::
