# Pagsisimula

:::danger Ang B1 Mobile ay deprecated
Ang native B1 Mobile app ay pinalitan ng [B1.church Progressive Web App (PWA)](../../b1-church/getting-started/installing-pwa.md). Ang mga bagong user ay dapat mag-install ng PWA mula sa anumang pahina sa `https://yourchurchname.b1.church` sa halip na mag-download ng B1 Mobile mula sa App Store o Google Play.
:::

<div class="article-intro">

Ang pagsisimula sa B1 Mobile ay tumatagal lamang ng ilang minuto. I-download ang app, mag-sign in gamit ang iyong ChurchApps account, pumili ng iyong simbahan, at handa na ikaw.

</div>

## Quick Start

1. **I-download** ang app mula sa [App Store](https://apps.apple.com) (iOS) o [Google Play Store](https://play.google.com) (Android). Maghanap ng "B1.church".
2. **Mag-log in** gamit ang iyong umiiral na ChurchApps account -- ang parehong email at password na ginagamit mo sa B1.church. Kung wala ka pang account, i-tap ang **Register** upang lumikha ng isa.
3. **Pumili ng iyong simbahan** sa pamamagitan ng paghahanap para sa ito ayon sa pangalan. I-type ang hindi bababa sa 3 characters upang makita ang mga resulta.
4. **Tuklasin ang dashboard** -- ang iyong personalized home screen ay nagpapakita ng quick links sa mga features na inaalok ng iyong simbahan.

Iyon na. Kapag pumili ka na ng iyong simbahan, ang app ay nag-load ng branding, navigation links, at nilalaman ng iyong simbahan. Maaari kang magsimulang gumamit ng groups, giving, check-in, at lahat ng iba pang ini-enable ng iyong simbahan.
