# Pagsisimula

<div class="article-intro">

Ang pagsisimula sa B1 Mobile ay ilang minuto lamang. I-download ang app, mag-sign in gamit ang iyong ChurchApps account, piliin ang iyong simbahan, at handa ka na.

</div>

## Mabilis na Pagsisimula

1. **I-download** ang app mula sa [App Store](https://apps.apple.com) (iOS) o [Google Play Store](https://play.google.com) (Android). Hanapin ang "B1.church".
2. **Mag-log in** gamit ang iyong kasalukuyang ChurchApps account -- ang parehong email at password na ginagamit mo sa B1.church. Kung wala ka pang account, i-tap ang **Register** para gumawa ng isa.
3. **Piliin ang iyong simbahan** sa pamamagitan ng paghahanap gamit ang pangalan nito. Mag-type ng hindi bababa sa 3 character para makakita ng mga resulta.
4. **I-explore ang dashboard** -- ang iyong personalisadong home screen ay nagpapakita ng mga mabilisang link sa mga feature na inaalok ng iyong simbahan.

Iyon lang. Kapag napili mo na ang iyong simbahan, nilo-load ng app ang branding, navigation link, at nilalaman ng iyong simbahan. Maaari mo nang gamitin ang mga grupo, pagbibigay, check-in, at lahat ng iba pang na-enable ng iyong simbahan.

## Sa Seksyong Ito

- [Pag-install ng App](./installing) -- Kung saan ida-download ang B1 Mobile at mga kinakailangan ng device.

:::
