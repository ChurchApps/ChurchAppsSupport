# Pag-install ng App

:::danger Ang B1 Mobile ay deprecated — mag-install ng PWA sa halip
Ang native B1 Mobile app ay hindi na ina-update. Inirekomenda namin ang pag-install ng **B1.church Progressive Web App (PWA)** mula sa site ng iyong simbahan (`https://yourchurchname.b1.church`) sa halip — ito ay maaaring i-install mula sa anumang pahina. Ang PWA ay maaaring idagdag sa iyong home screen sa iOS o Android sa loob ng ilang segundo at pinagsasama ang bawat feature ng native app.

Tingnan ang [Pag-install bilang App (PWA)](../../b1-church/getting-started/installing-pwa.md) para sa step-by-step na mga tagubilin.
:::

<div class="article-intro">

Ang B1 Mobile ay available para sa iOS at Android devices. Ang app ay libre na i-download at tumatagal lamang ng sandali upang mag-install upang maaari kang magsimulang kumonekta sa iyong simbahan.

</div>

## I-download

- **iOS** -- Buksan ang **App Store** sa iyong iPhone o iPad at maghanap ng **B1.church**. I-tap ang **Get** upang mag-download at mag-install.
- **Android** -- Buksan ang **Google Play Store** sa iyong Android device at maghanap ng **B1.church**. I-tap ang **Install** upang mag-download at mag-install.

Maaari mo ring maghanap ng "B1 Mobile" kung ang unang paghahanap ay hindi nagbabalik ng mga resulta.

## Pagkatapos Mag-install

Kapag naka-install na ang app, buksan ito mula sa iyong home screen. Makikita mo ang church search screen kung saan maaari kang mahanap at pumili ng iyong simbahan. Kung mayroon ka na ng ChurchApps account, maaari kang [mag-log in](./logging-in) kaagad. Kung hindi, maaari kang lumikha ng account mula sa login screen.

Ang app ay magsasangot ng pahintulot upang magpadala ng mga push notifications. Inirekomenda namin ang pagpayag ng mga notipikasyon upang makatanggap ka ng mga update mula sa iyong simbahan at mga grupo.

:::tip
Payagan ang mga push notifications kapag hiniling sa panahon ng setup. Tinitiyak nito na makakatanggap ka ng mapapanahong mga update tungkol sa [group activity](../groups/group-timeline.md), [messages](../community/messaging.md), at [serving assignments](../serving/viewing-plans.md).
:::

## Pag-update ng App

Ang B1 Mobile ay nakakatanggap ng regular na mga update na may mga bagong features at pagpapabuti. Ang iyong device ay dapat awtomatikong i-update ang app, ngunit maaari mo ring manual na suriin ang mga update:

1. Buksan ang **App Store** (iOS) o **Google Play Store** (Android).
2. Maghanap ng **B1.church**.
3. Kung available ang isang update, i-tap ang **Update**.

:::info
Ang B1 Mobile ay nangangailangan ng iOS 14 o mas bago para sa mga device ng Apple at Android 6.0 o mas bago para sa mga device ng Android.
:::

## Gamitin ang PWA Sa halip

Ang native B1 Mobile app ay na-deprecate na kapalit ng **B1.church Progressive Web App (PWA)**. Ang PWA ay nag-install mula sa Safari o Chrome — walang pangangailangan ng App Store download — at ang inirekomendadong opsyon para sa bawat bagong pag-install. Tingnan ang [Pag-install bilang App (PWA)](/docs/b1-church/getting-started/installing-pwa) para sa mga tagubilin.
