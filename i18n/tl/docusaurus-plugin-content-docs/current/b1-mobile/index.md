# B1 Mobile

<div class="article-intro">

Ang B1 Mobile ay ang kasamang app para sa B1.church, na makukuha sa iOS at Android. Nagbibigay ito sa mga miyembro ng simbahan ng access kahit saan sa lahat ng mga tool na ibinibigay ng iyong simbahan -- mga grupo, pagbibigay, check-in, mga plano sa paglilingkod, direktoryo ng mga miyembro, pribadong mensahe, mga sermon, Bibliya, bersikulo ng araw, mga aralin, at marami pa.

</div>

Ginagamit ng app ang iyong ChurchApps account, ang parehong ginagamit mo sa B1.church. Pagkatapos mag-log in at piliin ang iyong simbahan, ina-adjust ng app ang branding at configuration ng iyong simbahan. Kung kasapi ka sa maraming simbahan, maaari kang lumipat sa pagitan nila anumang oras.

Sinusuportahan ng B1 Mobile ang mga push notification para hindi mo mapalampas ang mahalagang update mula sa iyong simbahan o mga grupo. Ang home dashboard ay nag-pe-personalize batay sa iyong paggamit -- ang mga feature na madalas mong bisitahin ay inilalabas sa itaas para sa mabilis na access.

## Mga Pangunahing Feature

- **Mga Grupo** -- Tingnan ang iyong mga grupo, sumali sa mga pag-uusap, at manatiling konektado sa mga miyembro ng grupo.
- **Pagbibigay** -- Magbigay ng donasyon sa iyong simbahan nang direkta mula sa iyong telepono.
- **Check-In** -- I-check in ang iyong sarili at ang mga miyembro ng iyong sambahayan para sa mga serbisyo at kaganapan.
- **Mga Plano sa Paglilingkod** -- Tingnan ang mga paparating na plano ng serbisyo at ang iyong mga nakatakdang posisyon bilang boluntaryo.
- **Direktoryo ng mga Miyembro** -- I-browse ang direktoryo ng iyong simbahan at tingnan ang mga profile ng miyembro.
- **Pagmemensahe** -- Magpadala at tumanggap ng mga pribadong mensahe sa ibang miyembro ng simbahan.
- **Mga Sermon** -- Manood o makinig ng mga sermon mula sa iyong simbahan.
- **Bibliya** -- Basahin ang Bibliya sa loob ng app.
- **Bersikulo ng Araw** -- Simulan ang iyong araw sa isang pang-araw-araw na bersikulo.
- **Mga Aralin** -- I-access at pagdaanan ang mga aralin mula sa Lessons.church.
