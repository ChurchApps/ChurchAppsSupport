# Mga Grupo

<div class="article-intro">

Ang mga Grupo sa B1 Mobile ay tumutulong sa iyong manatiling konektado sa mga tao at ministeryo na mahalaga sa iyo. Kung kasapi ka sa isang small group, Bible study, volunteer team, o ministeryo, ang Groups screen ang iyong sentro para sa lahat ng nangyayari sa iyong komunidad sa simbahan.

</div>

## Ang Makikita Mo

Kapag binuksan mo ang screen ng **Groups**, ang iyong mga grupo ay nakaayos batay sa kung gaano mo kadalas bisitahin ang mga ito:

- **Hero group** -- Ang iyong pinakabinibisitang grupo ay kitang-kitang ipinapakita sa itaas ng screen bilang isang malaking hero card. I-tap ito para pumunta nang direkta sa detail page ng grupong iyon.
- **Mga featured group** -- Ang iyong mga susunod na pinakabinibisitang grupo ay lumalabas bilang mga featured card sa ibaba ng hero. Nagbibigay ang mga ito ng mabilis na access sa mga grupong pinaka-ine-engage mo.
- **Lahat ng mga grupo** -- Sa ibaba ng featured section, maaari mong i-browse ang iba pa sa mga grupo ng iyong simbahan.

Ang layout na ito ay nagpapadali sa pagpunta sa mga grupo na pinakamahalaga sa iyo sa isang tap lang, habang nagbibigay pa rin ng access sa lahat ng iba pa.

## Ano ang Maaari Mong Gawin

Narito ang maaari mong gawin mula sa Groups screen:

- **Mag-browse ng mga grupo** -- Mag-scroll sa mga available na grupo ng iyong simbahan at tuklasin ang mga bago para salihan.
- **Tingnan ang mga detalye ng grupo** -- I-tap ang anumang grupo para makita ang buong profile nito, kasama ang mga post, miyembro, at impormasyon ng grupo.
- **Sundan ang timeline** -- Isang pinag-isang timeline feed ang nagpapakita ng mga post at aktibidad mula sa lahat ng iyong mga grupo sa isang lugar, para hindi mo mapalampas ang isang update.
- **Manatiling konektado** -- Tingnan kung ano ang ibinabahagi ng iyong mga miyembro ng grupo, sumabay sa mga pag-uusap, at makisali sa nilalaman ng grupo.

:::tip
Ang Groups screen ay umaangkop sa iyong paggamit, tulad ng [dashboard](../dashboard/). Ang mga grupo na pinakamadalas mong bisitahin ay inilalabas sa itaas para sa mabilis na access.
:::
