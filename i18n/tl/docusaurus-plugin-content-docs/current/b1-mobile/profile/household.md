# Sambahayan

<div class="article-intro">

Ang tab na **Household** sa iyong Profile screen ay nagpapakita ng iyong mga miyembro ng pamilya. Dito mo makikita kung sino ang bahagi ng iyong sambahayan at kung paano konektado ang mga miyembro, na mahalaga para sa mga feature tulad ng check-in.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [I-install ang B1 Mobile](../getting-started/installing) at [mag-log in](../getting-started/logging-in) sa iyong account

</div>

## Ang Makikita Mo

Kapag binuksan mo ang tab na **Household**, makikita mo ang listahan ng mga tao sa iyong sambahayan. Ang bawat entry ng miyembro ay nagpapakita ng:

- **Pangalan** -- Ang pangalan at apelyido ng tao
- **Papel** -- Ang kanilang papel sa loob ng sambahayan (tulad ng Head o Other)
- **Profile photo** -- Ang kanilang larawan, kung may na-upload

Ang iyong sambahayan ay isang shared group na nag-uugnay sa iyong mga miyembro ng pamilya sa loob ng sistema ng simbahan.

## Bakit Mahalaga ang Sambahayan

Ang impormasyon ng iyong sambahayan ay ginagamit ng ilang feature sa B1 Mobile at sa buong ChurchApps:

- **Check-in** -- Kapag [nag-check in](../checkin/self-checkin.md) ka para sa isang serbisyo o kaganapan, maaari mong i-check in ang buong sambahayan mo nang sabay-sabay sa halip na i-check in ang bawat tao nang paisa-isa.
- **Mga rekord ng simbahan** -- Ginagamit ng iyong simbahan ang data ng sambahayan para panatilihing maayos ang mga rekord ng pamilya at para makipag-usap sa iyong pamilya bilang isang yunit.
- **Direktoryo** -- Maaaring makita ng ibang miyembro ang iyong impormasyon ng sambahayan sa [direktoryo ng simbahan](../community/member-directory.md), depende sa iyong [mga setting ng privacy](./privacy-settings).

## Pagtingin sa Iyong Sambahayan

1. Buksan ang drawer menu at i-tap ang **Profile**.
2. Piliin ang tab na **Household**.
3. I-browse ang listahan ng mga miyembro ng pamilya sa iyong sambahayan.

:::info
Ang pagiging miyembro ng sambahayan ay karaniwang pinapamahalaan ng iyong church administrator. Kung kailangan mong magdagdag o mag-alis ng tao mula sa iyong sambahayan, makipag-ugnayan sa opisina ng iyong simbahan para sa tulong.
:::

## Pananatiling Updated ng Impormasyon

Kung kailangang i-update ang mga detalye ng isang miyembro ng pamilya, maaari nilang [i-edit ang kanilang sariling profile](./editing-profile), o maaari kang makipag-ugnayan sa iyong church administrator para gumawa ng mga pagbabago. Ang pagpapanatiling tama ng impormasyon ng sambahayan ay tinitiyak na maayos na gumagana ang mga feature tulad ng [check-in](../checkin/) para sa buong pamilya mo.
