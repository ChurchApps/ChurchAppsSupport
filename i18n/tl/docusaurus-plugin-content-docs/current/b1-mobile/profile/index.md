# Profile

<div class="article-intro">

Ang iyong profile sa B1 Mobile ay kung saan mo pinapamahalaan ang iyong personal na impormasyon, ang iyong sambahayan, at ang iyong mga setting ng privacy. Lahat ng ina-update mo dito ay nare-reflect sa lahat ng serbisyo ng ChurchApps, kaya isang lugar lang ang kailangan mong baguhin.

</div>

## Mga Tab ng Profile

Ang Profile screen ay nakaayos sa ilang tab:

### Profile

Ang tab na **Profile** ay nagbibigay-daan sa iyong tingnan at i-edit ang iyong personal na impormasyon, kasama ang iyong pangalan, mga detalye ng pakikipag-ugnayan, at profile photo. Ang pagpapanatiling updated ng iyong profile ay tumutulong sa iyong simbahan na manatiling konektado sa iyo.

### Sambahayan

Ang tab na **Household** ay nagpapakita ng iyong mga miyembro ng pamilya at kanilang mga papel sa loob ng iyong sambahayan. Ang impormasyong ito ay ginagamit para sa mga feature tulad ng [check-in](../checkin/), kung saan maaari mong i-check in ang buong pamilya mo nang sabay-sabay.

### Visibility

Ang tab na **Visibility** ay nagbibigay-daan sa iyong kontrolin kung ano ang makikita ng ibang miyembro ng simbahan tungkol sa iyo sa [direktoryo](../community/member-directory.md). Ikaw ang magpapasya kung aling mga personal na detalye ang ibabahagi at alin ang mananatiling pribado.

### Pending

Ang tab na **Pending** ay nagpapakita ng anumang mga pagbabago sa profile na isinumite mo na naghihintay ng pag-apruba ng admin.

### Account

Ang tab na **Account** ay nagbibigay ng access sa iyong mga setting ng seguridad.

## Pagpunta sa Iyong Profile

1. Buksan ang drawer menu.
2. I-tap ang **Profile** para buksan ang iyong profile screen.
3. Gamitin ang mga tab sa itaas para mag-navigate sa pagitan ng mga seksyon.
