# Pag-edit ng Iyong Profile

<div class="article-intro">

Ang tab na **Profile** ay nagbibigay-daan sa iyong i-update ang iyong personal na impormasyon para palaging tama ang mga detalyeng naka-file sa iyong simbahan. Ang mga pagbabagong ginawa mo dito ay nare-reflect sa lahat ng serbisyo ng ChurchApps, kasama ang B1.church at anumang iba pang app na ginagamit ng iyong simbahan.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [I-install ang B1 Mobile](../getting-started/installing) at [mag-log in](../getting-started/logging-in) sa iyong account

</div>

## Ano ang Maaari Mong I-edit

Kasama sa iyong profile ang mga sumusunod na personal na detalye:

- **Pangalan at apelyido** -- Ang iyong display name sa lahat ng ChurchApps
- **Email address** -- Ginagamit para sa mga komunikasyon ng simbahan at pagbawi ng account
- **Numero ng telepono** -- Para maabot ka ng iyong simbahan kapag kinakailangan
- **Address** -- Ang iyong address sa bahay para sa mga rekord ng simbahan
- **Profile photo** -- Ang larawang kumakatawan sa iyo sa [direktoryo](../community/member-directory.md) at sa buong app

## Pag-update ng Iyong Impormasyon

1. Buksan ang drawer menu at i-tap ang **Profile**.
2. Piliin ang tab na **Profile** kung hindi pa ito aktibo.
3. I-tap ang field na gusto mong i-update at gawin ang iyong mga pagbabago.
4. I-save ang iyong mga pagbabago kapag tapos ka na.

Awtomatikong na-sync ang iyong na-update na impormasyon. Sinumang tumingin sa iyong profile (batay sa iyong [mga setting ng privacy](./privacy-settings)) ay makikita agad ang mga bagong detalye.

:::tip
Panatilihing updated ang iyong email address at numero ng telepono. Ginagamit ng iyong simbahan ang impormasyong ito para magpadala sa iyo ng mahahalagang update at paalala.
:::

## Profile Photo

Maaari kang magdagdag o magpalit ng iyong profile photo mula sa tab na **Profile**. Ang iyong larawan ay lumalabas sa [direktoryo ng mga miyembro](../community/member-directory.md), mga listahan ng miyembro ng grupo, at kahit saan pa na ipinapakita ang iyong profile.

## Mga Pagbabago sa Lahat ng ChurchApps

Dahil lahat ng serbisyo ng ChurchApps ay gumagamit ng parehong sistema ng account, ang pagbabagong ginawa mo sa B1 Mobile ay agad na nakikita sa [B1.church](../../b1-church/), sa mga admin tool ng iyong simbahan, at sa anumang iba pang konektadong serbisyo. Hindi mo na kailangang i-update ang iyong impormasyon nang hiwalay sa bawat lugar.

:::info
Ang ilang simbahan ay nangangailangan ng pag-apruba ng administrator para sa ilang pagbabago sa profile. Kung hindi agad lumabas ang iyong update, tingnan ang tab na **Pending** sa iyong profile screen para sa status nito.
:::
