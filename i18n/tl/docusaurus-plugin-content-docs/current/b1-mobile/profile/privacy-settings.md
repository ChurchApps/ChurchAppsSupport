# Mga Setting ng Privacy

<div class="article-intro">

Ang tab na **Visibility** sa iyong Profile screen ay nagbibigay-daan sa iyong kontrolin kung ano ang makikita ng ibang miyembro ng simbahan tungkol sa iyo sa direktoryo. Ikaw ang magpapasya kung aling mga personal na detalye ang ibabahagi nang publiko at alin ang mananatiling pribado.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [I-install ang B1 Mobile](../getting-started/installing) at [mag-log in](../getting-started/logging-in) sa iyong account

</div>

## Pagkontrol ng Iyong Visibility

1. Buksan ang drawer menu at i-tap ang **Profile**.
2. Piliin ang tab na **Visibility**.
3. Suriin ang bawat field at piliin kung ito ay nakikita ng ibang miyembro.
4. I-save ang iyong mga pagbabago.

Maaari mong kontrolin ang visibility para sa mga detalye tulad ng:

- **Numero ng telepono** -- Kung makikita ng ibang miyembro ang iyong numero ng telepono
- **Email address** -- Kung ipinapakita ang iyong email sa [direktoryo](../community/member-directory.md)
- **Address** -- Kung nakikita ng iba ang iyong address sa bahay

Bilang default, maaaring may ilang field na visible o nakatago ang iyong simbahan. Maaari mong i-adjust ang mga setting na ito anumang oras para tumugma sa iyong kaginhawaan.

:::tip
Kung hindi ka sigurado kung ano ang nakikita ng iba, bisitahin ang tab na **Visibility** at suriin ang iyong kasalukuyang mga setting. Maaari kang gumawa ng mga pagbabago anumang oras.
:::

## Mga Pending na Pagbabago

Ang tab na **Pending** ay nagpapakita ng anumang mga update sa profile na isinumite mo na naghihintay ng pag-apruba ng admin. Ang ilang simbahan ay nangangailangan ng administrator na suriin at aprubahan ang ilang pagbabago bago magkabisa ang mga ito. Kung gumawa ka ng pagbabagong nangangailangan ng pag-apruba, makikita mo ito dito kasama ang kasalukuyang status nito.

:::info
Kung hindi pa nagkakabisa ang pagbabagong ginawa mo, tingnan ang tab na **Pending**. Maaaring kailanganin muna ng iyong church administrator na aprubahan ito.
:::

## Mga Setting ng Account

Ang tab na **Account** ay nagbibigay ng access sa iyong mga setting ng seguridad. Gamitin ang tab na ito para pamahalaan ang mga setting na nauugnay sa iyong access at seguridad ng account.

## Buod ng mga Tab ng Profile

| Tab | Layunin |
|---|---|
| **Profile** | [I-edit ang iyong personal na impormasyon](./editing-profile) at larawan |
| **Household** | [Tingnan ang iyong mga miyembro ng pamilya](./household) |
| **Visibility** | Kontrolin kung ano ang nakikita ng iba sa direktoryo |
| **Pending** | Suriin ang mga pagbabago sa profile na naghihintay ng pag-apruba |
| **Account** | Pamahalaan ang mga setting ng seguridad |

Ang mga tab na ito ay nagtutulungan para bigyan ka ng buong kontrol sa iyong personal na impormasyon at kung paano ito ibinabahagi sa loob ng iyong komunidad sa simbahan.
