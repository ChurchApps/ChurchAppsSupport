# Direktoryo ng mga Miyembro

<div class="article-intro">

Ang direktoryo ng mga miyembro sa B1 Mobile ay nagbibigay-daan sa iyong mag-browse at maghanap ng mga tao sa iyong simbahan. Kung kailangan mong hanapin ang isang kapwa miyembro ng grupo o mahanap ang isang taong nakilala mo noong Linggo, pinapadali ng direktoryo ang pagkonekta.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [I-install ang B1 Mobile](../getting-started/installing) at [mag-log in](../getting-started/logging-in) sa iyong account

</div>

## Pag-access sa Direktoryo

1. I-tap ang opsyong **Directory** sa navigation ng app.
2. Nilo-load ng direktoryo ang mga miyembro ng iyong simbahan na nakaayos ayon sa alpabeto.

## Pagba-browse sa Direktoryo

Ang mga miyembro ay naka-index ayon sa alpabeto ng apelyido at nakaayos sa ilalim ng mga heading na letra (A, B, C, at iba pa). Maaari kang mag-scroll sa listahan para i-browse ang iyong pamilya sa simbahan. Ang bawat entry ay nagpapakita ng pangalan at larawan ng miyembro, na nagpapadali sa paghahanap ng hinahanap mo.

## Paghahanap ng mga Miyembro

Kung alam mo kung sino ang hinahanap mo, ang search bar sa itaas ng direktoryo ang pinakamabilis na paraan para mahanap sila.

1. I-tap ang **search bar** sa itaas ng screen.
2. Mag-type ng pangalan.
3. Nagfi-filter ang listahan sa real time para ipakita ang mga tumutugmang resulta.

:::tip
Ito ay lalo nang kapaki-pakinabang sa mga mas malalaking simbahan kung saan ang pag-scroll sa buong listahan ay magtatagal. Mag-type lang ng unang ilang letra ng pangalan para mabilis na mapaliit ang mga resulta.
:::

## Pagtingin sa Profile ng Miyembro

I-tap ang sinumang miyembro sa listahan para buksan ang kanilang profile. Maaaring kasama sa profile page ang:

- **Impormasyon sa pakikipag-ugnayan** -- Numero ng telepono, email address, at iba pang mga detalyeng ginawang available ng iyong simbahan.
- **Mga koneksyon sa pamilya** -- [Mga miyembro ng sambahayan](../profile/household.md) na naka-link sa profile ng taong ito. Maaari mong i-tap ang isang miyembro ng pamilya para ma-navigate sa kanilang profile din.

:::info
Ang impormasyong nakikita sa bawat profile ay depende sa mga privacy at directory setting ng iyong simbahan. Hindi lahat ng field ay maaaring ipakita para sa bawat miyembro. Maaaring kontrolin ng mga miyembro ang kanilang sariling visibility mula sa [Mga Setting ng Privacy](../profile/privacy-settings.md). Kung may mga tanong ka tungkol sa kung ano ang ibinabahagi, makipag-ugnayan sa iyong church administrator.
:::
