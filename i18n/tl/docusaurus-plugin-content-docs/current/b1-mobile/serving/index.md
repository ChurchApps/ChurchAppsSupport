# Paglilingkod

<div class="article-intro">

Ang seksyon ng **Serving** sa B1 Mobile ay nagbibigay sa iyo ng mabilis at madaling paraan para manatiling updated sa iyong mga tungkulin sa paglilingkod. Kung ikaw man ay nasa worship team, bumabati sa pintuan, o tumutulong sa kids ministry, dito mo makikita kung kailan ka naka-schedule at kung ano ang iyong papel.

</div>

## Ang Makikita Mo

- **Mga paparating na takdang-aralin** -- Tingnan ang lahat ng iyong hinaharap na oras ng paglilingkod sa isang tingin para palagi mong alam kung kailan ka susunod na kakailanganin.
- **Nakaraang kasaysayan ng serbisyo** -- Suriin ang mga serbisyong napaglingkuran mo na para subaybayan ang iyong paglahok sa paglipas ng panahon.
- **Mga takdang-aralin sa posisyon** -- Ang bawat takdang-aralin ay nagpapakita ng posisyong ibinigay sa iyo kasama ang kasalukuyang status nito -- kung ito ay hinihiling, nakumpirma, o naghihintay ng iyong tugon.

## Pagsisimula

Para tingnan ang iyong schedule ng paglilingkod, i-tap ang **Serving** sa bottom navigation ng app. Kailangan mong naka-log in para ma-access ang iyong mga takdang-aralin. Kung hindi ka naka-sign in, hihilingin sa iyong [mag-log in](../getting-started/logging-in) muna.

:::info
Makikita mo lamang ang mga plano kung saan ka na-assign. Kung hindi ka nakakakita ng mga takdang-aralin, ibig sabihin ay hindi ka pa naidadagdag sa isang serving team. Makipag-ugnayan sa iyong church administrator o ministry leader para makilahok.
:::
