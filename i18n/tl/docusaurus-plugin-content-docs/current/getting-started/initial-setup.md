# Paunang Setup

<div class="article-intro">

Pagkatapos gumawa ng inyong account at simbahan, may ilang mahahalagang hakbang na kailangang i-configure bago mag-imbita ng inyong team. Gagabayan kayo ng gabay na ito sa inirerekomendang pagkakasunod-sunod ng setup upang handa na ang inyong simbahan mula sa unang araw.

</div>

<div class="prereqs">
<h4>Bago Kayo Magsimula</h4>

- [Gumawa ng inyong account](./create-account.md) at irehistro ang inyong simbahan
- Ihanda ang logo at mga branding asset ng inyong simbahan (opsyonal ngunit inirerekomenda)

</div>

## Hakbang 1: I-configure ang Impormasyon ng Inyong Simbahan

1. Pumunta sa **Settings** sa kaliwang sidebar.
2. I-click ang **Edit Settings**.
3. Ilagay ang **pangalan**, **address**, at **contact details** ng inyong simbahan.
4. I-click ang **Save** upang i-apply ang inyong mga pagbabago.

## Hakbang 2: I-set Up ang Inyong Branding

1. Mula sa pahina ng **Settings**, i-upload ang **logo ng inyong simbahan**.
2. I-configure ang mga karagdagang opsyon sa branding tulad ng mga kulay at larawan.
3. Ang inyong branding ay lalabas sa inyong B1.church website at sa B1 Mobile app.

:::tip
Ang pagkakaroon ng inyong logo at branding na handa bago mag-imbita ng mga miyembro ng team ay nagbibigay ng magandang unang impresyon kapag nag-log in sila.
:::

## Hakbang 3: I-configure ang mga Tungkulin at Imbitahan ang Inyong Team

1. I-click ang **Mga Tungkulin** mula sa pahina ng **Settings**.
2. Gumawa ng mga tungkulin para sa inyong team (halimbawa, "Pastor", "Kalihim", "Ingat-yaman").
3. Italaga ang naaangkop na mga pahintulot sa bawat tungkulin.
4. Magdagdag ng mga miyembro ng team sa pamamagitan ng paghahanap ng kanilang mga account at pag-assign sa kanila ng mga tungkulin.

## Hakbang 4: I-set Up ang Inyong Mga Pangunahing Lugar

Kapag handa na ang inyong team, simulan ang pagbuo ng mga pangunahing lugar ng B1 Admin:

1. **Mga Tao** -- Magdagdag ng mga miyembro sa direktoryo ng inyong simbahan. Ito ang pundasyon ng lahat ng iba pa, kaya magsimula dito. Maaari kayong magdagdag ng mga tao nang mano-mano o gamitin ang kasangkapang **Import** mula sa **Settings**.
2. **Mga Grupo** -- Gumawa ng mga grupo para sa maliliit na grupo, ministry team, klase, at iba pang pagtitipon.
3. **Attendance** -- I-configure ang inyong mga kampus, serbisyo, at oras ng serbisyo para sa pagsubaybay ng attendance.
4. **Mga Donasyon** -- Mag-set up ng mga donation fund at ikonekta ang inyong payment gateway (Stripe) upang tumanggap ng online giving.
5. **Website** -- Buuin ang pampublikong website ng inyong simbahan gamit ang B1.church website builder.

:::info
Hindi ninyo kailangang i-set up ang bawat lugar nang sabay-sabay. Maraming simbahan ang nagsisimula sa **Mga Tao** at **Mga Grupo**, pagkatapos ay dinadagdagan ng **Attendance** at **Mga Donasyon** habang nasasanay sa platform.
:::

## Inirerekomendang Pagkakasunod-sunod

Para sa pinakamaayos na karanasan sa setup, inirerekomenda namin ang pagkakasunod-sunod na ito:

1. **Mga Tao** -- Idagdag muna ang inyong mga miyembro at regular na dumadalo.
2. **Mga Grupo** -- Ayusin ang mga tao sa makabuluhang mga grupo.
3. **Attendance** -- Simulan ang pagsubaybay kung sino ang dumadalo bawat linggo.
4. **Mga Donasyon** -- I-set up ang giving kapag nasa sistema na ang inyong mga tao.
5. **Website** -- Buuin ang inyong pampublikong site sa huli, pagkatapos na maayos ang inyong data.

Kapag nakumpleto na ninyo ang mga hakbang na ito, handa na ang inyong simbahan. Tuklasin ang dokumentasyon ng [B1 Admin](/docs/b1-admin/) para sa mga detalyadong gabay sa bawat lugar.
