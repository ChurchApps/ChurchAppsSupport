# Pagsisimula

<div class="article-intro">

Maligayang pagdating sa ChurchApps -- mga libreng, open-source na kasangkapan na dinisenyo upang tulungan ang inyong simbahan na umunlad. Maliit man o malaking organisasyon na may maraming kampus ang inyong kongregasyon, nagbibigay ang ChurchApps ng lahat ng kailangan ninyo upang pamahalaan at palaguin ang inyong komunidad ng simbahan.

</div>

## Mga Produkto ng ChurchApps

Kasama sa ChurchApps ang ilang mga produkto na magkakasamang gumagana:

- **B1 Admin** -- Ang dashboard ng pamamahala ng inyong simbahan. Pamahalaan ang mga tao, grupo, attendance, donasyon, nilalaman ng website, at marami pa mula sa isang sentral na lokasyon.
- **B1.church** -- Ang pampublikong website ng inyong simbahan at portal ng mga miyembro. Maaaring manood ng mga sermon, ma-access ang mga direktoryo, mag-donate online, at manatiling konektado ang mga miyembro.
- **Lessons.church** -- Libreng kurikulum ng simbahan para sa Sunday school, maliliit na grupo, at pag-aaral ng Bibliya. Mag-access ng lumalagong library ng mga aralin nang walang bayad.
- **FreeShow** -- Libreng presentation software para sa inyong mga worship service. Magpakita ng mga liriko, slide, at media sa inyong mga pagtitipon.

:::info
Lahat ng kasangkapan ng ChurchApps ay **100% libre** na walang limitasyon, walang premium tier, at walang nakatagong bayarin. Ang ChurchApps ay isang nonprofit na organisasyon na sinusuportahan ng mga donasyon at mga boluntaryo.
:::

## Ano ang Sakop ng Dokumentasyong Ito

Gagabayan kayo ng dokumentasyong ito sa lahat ng kailangan ninyo upang magsimula sa ChurchApps. Matututo kayo kung paano:

1. Gumawa ng inyong account at irehistro ang inyong simbahan.
2. I-configure ang mga pangunahing setting at branding ng inyong simbahan.
3. I-set up ang inyong team na may tamang mga tungkulin at pahintulot.
4. Simulan ang paggamit ng mga kasangkapan upang pamahalaan ang inyong kongregasyon.
