# Direktoryo ng Miyembro

<div class="article-intro">

Ang direktoryo ng miyembro ay nagbibigay-daan sa iyo na mag-browse at maghanap ng mga tao sa iyong simbahan. Kailangan mo mang hanapin ang isang kapwa miyembro ng grupo o makahanap ng bagong kakilala, pinapadali ng direktoryo ang pagkonekta sa pamilya ng iyong simbahan.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Dapat kang naka-[log in](../getting-started/logging-in.md) sa iyong B1.church account.

</div>

## Pag-access sa Direktoryo

1. I-click ang **Komunidad** sa [sidebar navigation](../getting-started/navigating.md).
2. Ang direktoryo ay naglo-load na may listahan ng lahat ng nakikitang mga miyembro ng simbahan sa kaliwang bahagi ng pahina.

## Paghahanap ng mga Miyembro

Ang pahina ay gumagamit ng master-detail na layout. Sa kaliwang bahagi, mayroon kang dalawang paraan para makahanap ng mga tao:

### Maghanap ayon sa Pangalan

1. Siguraduhing napili ang **People** filter sa itaas ng search area.
2. Mag-type ng pangalan sa search box.
3. I-click ang **Search** o pindutin ang **Enter** para makita ang mga resulta.

Kung iiwan mong blangko ang search box at i-click ang **Search**, ipapakita ang buong listahan ng direktoryo.

### Maghanap ayon sa Grupo

1. I-click ang **Groups** filter sa itaas ng search area.
2. Pumili ng grupo mula sa dropdown list.
3. I-click ang **Search** para makita ang lahat ng miyembro ng grupong iyon.

:::tip
Ang paghahanap ayon sa grupo ay isang mabilis na paraan para mahanap ang isang taong nakilala mo sa isang [group meeting](../groups/browsing-groups.md) ngunit hindi mo masyadong natatandaan ang pangalan.
:::

## Pag-browse ng Direktoryo

Ang mga miyembro ay nakalista nang alphabetical ayon sa apelyido at nakagrupo sa ilalim ng mga letter heading (A, B, C, at iba pa). Ang bawat entry ay nagpapakita ng larawan ng miyembro, display name, at pangalan ng sambahayan, na nagpapadali sa pagbrowse ng listahan.

## Pagtingin ng Profile ng Miyembro

I-click ang sinumang miyembro sa listahan para buksan ang kanilang profile sa detail panel sa kanan. Ang profile ay nagpapakita ng impormasyon ng miyembro ayon sa na-configure ng iyong simbahan, na maaaring kasama ang mga contact detail, mga miyembro ng sambahayan, at mga pagiging miyembro sa grupo.

Maaari mong i-click ang mga miyembro ng sambahayan sa loob ng isang profile para mag-navigate sa kanilang mga profile.

:::info
Ang impormasyong nakikita sa bawat profile ay depende sa mga privacy at directory setting ng iyong simbahan. Hindi lahat ng field ay maaaring ipakita para sa bawat miyembro.
:::
