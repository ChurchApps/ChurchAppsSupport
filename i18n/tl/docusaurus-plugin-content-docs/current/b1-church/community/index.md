# Komunidad

<div class="article-intro">

Ang seksyon ng **Komunidad** sa B1.church ay tumutulong sa iyo na manatiling konektado sa pamilya ng iyong simbahan. Pinagsasama nito ang isang direktoryo ng miyembro para mahanap at makilala ang ibang mga miyembro ng simbahan, at isang timeline feed na nagpapanatiling updated ka sa mga nangyayari sa iyong mga grupo at simbahan.

</div>

## Ano ang Makikita Mo

- **[Direktoryo ng Miyembro](./member-directory.md)** -- Mag-browse at maghanap sa direktoryo ng mga miyembro ng iyong simbahan. Hanapin ang mga tao ayon sa pangalan o i-filter ayon sa grupo para mahanap ang hinahanap mo. I-click ang isang miyembro para tingnan ang mga detalye ng profile.
- **[Timeline](./timeline.md)** -- Ang iyong activity feed na nagpapakita ng mga pinakabagong update mula sa iyong mga grupo at simbahan. Ang timeline ang default na landing page kapag binibisita mo ang portal ng miyembro.

## Pagsisimula

Dapat kang naka-[log in](../getting-started/logging-in.md) para ma-access ang mga feature ng komunidad. Kung hindi ka naka-sign in, hihilingin sa iyong mag-log in kapag nag-navigate ka sa **Komunidad** tab o sa **Timeline**.

Ang **Timeline** tab ay palaging available sa sidebar at nagsisilbing home page ng iyong portal ng miyembro. Ang **Komunidad** tab ay nagbibigay ng access sa direktoryo ng miyembro.

:::info
Ang mga miyembro at impormasyon na makikita mo sa direktoryo ay depende sa mga privacy setting ng iyong simbahan. Maaaring hindi mo makita ang buong profile ng bawat miyembro. Kung may mga tanong ka tungkol sa kung ano ang ibinabahagi, makipag-ugnayan sa administrator ng iyong simbahan.
:::

:::tip
Gamitin ang mga feature ng komunidad kasabay ng [Mga Grupo](../groups/) para manatiling engaged sa iyong simbahan sa buong linggo.
:::
