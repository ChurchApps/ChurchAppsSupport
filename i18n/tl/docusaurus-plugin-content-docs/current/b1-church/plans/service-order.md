# Order ng Serbisyo

<div class="article-intro">

Kapag binuksan mo ang isang plano, ang seksyon ng **Order ng Serbisyo** ay nagpapakita ng pagkakasunud-sunod ng mga item para sa serbisyong iyon. Nagbibigay ito sa iyo at sa iyong team ng shared na view ng takbo ng serbisyo, kasama ang timing, mga detalye ng kanta, at paglalarawan para sa bawat elemento.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Dapat kang naka-[log in](../getting-started/logging-in.md) sa iyong B1.church account.
- Buksan ang isang plano mula sa iyong [listahan ng Mga Plano](./viewing-plans.md) para makita ang order ng serbisyo nito.

</div>

## Ano ang Ipinapakita ng Order ng Serbisyo

Ang order ng serbisyo ay ipinapakita bilang isang listahan ng mga item sa pagkakasunud-sunod kung kailan mangyayari sa serbisyo. Ang bawat item ay may kasamang:

- **Cumulative time** -- Ang running time mula sa simula ng serbisyo, para malaman mo kung kailan inaasahang magsisimula ang bawat elemento.
- **Label at paglalarawan ng item** -- Ang pangalan ng elemento (tulad ng titulo ng kanta, panalangin, o anunsyo) kasama ang anumang mga nota o paglalarawan.
- **Tagal** -- Gaano katagal inaasahang aabutin ng item.

Ang mga item ay naka-organisa sa ilalim ng mga **header** na naggugrupo ng mga kaugnay na elemento (halimbawa, "Worship Set" o "Message"). Pinapadali nito ang pag-scan ng kabuuang istraktura ng serbisyo.

## Curriculum at Nilalaman ng Leksyon

Kung ang plano ay may kasamang curriculum o lesson content mula sa isang provider tulad ng [Lessons.church](../content/lessons.md), makikita mo ang preview ng nilalamang iyon sa loob ng order ng serbisyo. Maaaring kasama dito ang mga seksyon ng leksyon, presentation material, at iba pang resources na kailangan ng iyong team para sa serbisyo.

## Pag-print ng Order ng Serbisyo

Maaari mong i-print ang isang formatted na bersyon ng order ng serbisyo para dalhin mo sa araw ng serbisyo.

1. Buksan ang plano mula sa iyong **Mga Plano** na listahan.
2. Sa **Order ng Serbisyo** na seksyon, i-click ang **print** icon sa kanang itaas.
3. Magbubukas ang isang bagong window na may printable layout na may kasamang:
   - Ang **petsa ng serbisyo** sa itaas.
   - **Mga team assignment** na nakagrupo ayon sa kategorya sa kaliwa, na nagpapakita ng bawat posisyon at kung sino ang nagpupuno nito.
   - Ang **buong order ng serbisyo** sa kanan na may mga column para sa oras, detalye ng item, at tagal.
4. Awtomatikong magbubukas ang print dialog ng iyong browser para maka-print ka o mag-save bilang PDF.

:::tip
Ang naka-print na order ng serbisyo ay isang magandang reference na dadalhin habang nagre-rehearse o sa mismong serbisyo. Kasama dito ang parehong team roster at ang buong order ng serbisyo sa isang pahina.
:::
