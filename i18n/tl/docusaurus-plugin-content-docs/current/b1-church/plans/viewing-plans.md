# Pagtingin ng mga Plano

<div class="article-intro">

Ang Plans page ay nagbibigay sa iyo ng malinaw na view ng lahat ng plano ng serbisyo na itinalaga sa iyo. Gumagamit ito ng master-detail layout para mabilis mong ma-browse ang iyong mga plano at ma-dive sa mga detalye ng alinman.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Dapat kang naka-[log in](../getting-started/logging-in.md) sa iyong B1.church account.
- Dapat kang naka-assign sa kahit isang plano ng serbisyo. Kung walang planong makikita, makipag-ugnayan sa administrator ng iyong simbahan o ministry leader.

</div>

## Pag-access sa Iyong mga Plano

1. I-click ang **Mga Plano** sa [sidebar navigation](../getting-started/navigating.md).
2. Makikita mo ang listahan ng iyong mga plano sa kaliwang bahagi ng pahina.

## Ang Listahan ng Plano

Ang iyong mga plano ay nakalista sa kaliwang panel sa ilalim ng **My Plans**. Ang bawat entry ay nagpapakita ng:

- Ang **pangalan ng plano** (halimbawa, "Sunday Worship" o "Kids Ministry")
- Ang **petsa ng serbisyo**
- Ang iyong **posisyon** (ang role na pinupunan mo)
- Ang iyong **status** -- color-coded para makita mo agad kung nag-accept, nag-decline, o hindi pa sumasagot

Sa itaas ng listahan, makikita mo rin ang opsyon na **Blockout Dates**. I-click ito para pamahalaan ang mga petsa kung kailan ka hindi available para magsilbi. Maaari kang magdagdag, mag-edit, at mag-alis ng mga blockout date range para malaman ng iyong mga team leader kung kailan ka hindi available.

## Pagtingin ng mga Detalye ng Plano

I-click ang anumang plano sa listahan para buksan ang mga detalye nito sa kanang panel. Ang detail view ay may kasamang:

- **Mga detalye ng posisyon** -- Ang itinalaga sa iyong posisyon kasama ang mga kinakailangang oras at ang iyong kasalukuyang status. Kung hindi ka pa sumasagot, maaari mong i-click ang **Accept** o **Decline** nang direkta mula sa view na ito.
- **Mga nota** -- Anumang mga nota na idinagdag ng plan creator para sa team.
- **Order ng Serbisyo** -- Ang buong order ng serbisyo na may timing, mga item, at paglalarawan. Tingnan ang [Order ng Serbisyo](./service-order.md) para sa higit pang mga detalye.
- **Mga team assignment** -- Isang buod ng lahat ng mga miyembro ng team at kanilang mga posisyon, para makita mo kung sino pa ang nagsisilbi kasama mo.

:::tip
Kung mayroon kang maraming assignment sa iba't ibang plano, ang listahan ay naka-sort ayon sa petsa para lumabas muna ang iyong mga paparating na commitment. Regular na suriin para manatiling updated sa iyong schedule.
:::
