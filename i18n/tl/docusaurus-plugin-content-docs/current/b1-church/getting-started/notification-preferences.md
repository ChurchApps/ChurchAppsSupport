# Mga Kagustuhan sa Notification

<div class="article-intro">

Ang mga kagustuhan sa notification ay nagpapahintulot sa iyo na magpasya kung aling mga notification ang matatanggap mo mula sa iyong simbahan at kung paano mo sila matatanggap — push notifications, email, o ang in-app bell. Maaari mong tumpukin ang bawat uri ng notification, magtakda ng quiet hours, o i-mute ang lahat nang sabay-sabay, lahat mula sa isang screen.

</div>

## Pagbubukas ng Iyong Mga Kagustuhan

1. Buksan ang **Notifications** area (ang bell icon) sa B1 member app.
2. I-tap ang **Notification Preferences**.

## Mga Global Control

Sa tuktok ng screen ay makikita mo ang mga setting na nag-apply sa lahat:

- **Mute all** — Pansamantalang itinigil ang lahat ng non-essential notifications.
- **Push notifications** — Nag-turn on o off ng push notifications para sa iyong device.
- **Email frequency** — Pumili kung paano darating ang mga email notification:
  - **Per notification** — Magpadala ng email habang nangyayari ang mga bagay.
  - **Daily digest** — I-bundle ang mga ito sa isang pang-araw-araw na email.
  - **Never** — Huwag magpadala ng mga notification email.
- **Quiet hours** — Magtakda ng oras ng pagsisimula at pagtatapos (at ang iyong time zone) kapag hindi ka nais na mawalan ng tulog. Ang mga non-urgent notification ay naghihintay hanggang sa matapos ang quiet hours.

## Pagpili ng mga Notification ayon sa Uri

Sa ibaba ng mga pandaigdigang kontrol ay isang grid ng mga uri ng notification. Para sa bawat isa maaari mong i-turn on o off ang mga channel na gusto mo:

- **Push** — Isang notification sa iyong device.
- **Email** — Isang mensahe sa iyong inbox.
- **In-App** — Ang bell/badge sa loob ng app.

Ang mga karaniwang uri ay kinabibilangan ng **Event Reminders**, **Serving & Schedule**, **Direct Messages**, **Group Chat**, **Prayer Requests**, at **Church Announcements**. Halimbawa, maaari mong panatilihing naka-on ang push notification para sa Direct Messages ngunit i-turn off ang mga ito para sa Announcements.

:::info
Ang ilang uri ng notification ay **laging naka-on** at hindi maaaring i-turn off — **Account & Security**, **Giving Receipts & Statements**, at **Check-In Safety Alerts**. Ang mga ito ay kinakailangan para sa iyong seguridad at para sa mga legal na dahilan, kaya ipinakita nila ang isang maliit na lock icon. Lahat ay iyong kontrolin.
:::

Ang ilang mga uri ay **off by default** (tulad ng promotional o fundraising messages) at umabot lamang sa iyo kung pipiliin mong mag-opt in.

## Pag-save ng Iyong mga Pagbabago

Pagkatapos ayusin ang iyong mga kagustuhan, i-tap ang **Save**. Ang iyong mga pagbabago ay nakakaranas ng epekto kaagad at nag-apply saanman ka gumagamit ng B1.

:::tip
Ang mga time-sensitive notifications — tulad ng isang security alert o isang mensahe sa kaligtasan ng check-in ng bata — naaabot ka pa rin kahit sa quiet hours. Ang pang-araw-araw na update ay naghihintay hanggang matapos ang quiet hours.
:::
