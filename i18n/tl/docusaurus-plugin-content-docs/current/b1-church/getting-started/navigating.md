# Pag-navigate sa B1App

<div class="article-intro">

Ang portal ng miyembro sa B1.church ay gumagamit ng sidebar navigation layout na nagpapadali ng paglipat sa iba't ibang mga tool na available sa iyo. Ipinapaliwanag ng pahinang ito kung paano gumagana ang navigasyon sa parehong desktop at mobile device.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kailangan mong maka-[log in](./logging-in.md) para ma-access ang portal ng miyembro.

</div>

## Sidebar Navigation

Kapag pumasok ka sa portal ng miyembro, makikita mo ang isang vertical na sidebar sa kaliwang bahagi ng screen. Ang sidebar na ito ay naglalaman ng mga tab para sa bawat seksyon ng portal:

- **Timeline** -- Ang iyong personalized na feed ng mga update at pag-uusap mula sa iyong mga grupo at simbahan. Ito ang default na tab na naglo-load kapag binuksan mo ang portal ng miyembro.
- **Mga Grupo** -- Tingnan at makipag-ugnayan sa mga [grupo](../groups/) na kinabibilangan mo.
- **Komunidad** -- I-browse ang [direktoryo ng mga miyembro](../community/member-directory.md) ng simbahan.
- **Mga Plano** -- Tingnan ang mga paparating na [plano ng serbisyo](../plans/) at ang iyong mga nakatalagang posisyon bilang boluntaryo.
- **Check-in** -- [I-check in](../checkin/) ang iyong sarili at ang iyong sambahayan para sa mga serbisyo at kaganapan.
- **Mga Leksyon** -- I-access ang mga [leksyon](../content/lessons.md) ng Bible study at curriculum.
- **Mga Donasyon** -- Gumawa ng online na [donasyon](../giving/) at tingnan ang iyong kasaysayan ng pagbibigay.

I-click ang anumang tab para lumipat sa seksyong iyon. Ang aktibong tab ay naka-highlight para lagi mong malaman kung nasaan ka.

:::info
Ang mga tab na makikita mo ay maaaring mag-iba depende sa kung ano ang na-enable ng iyong simbahan. Kinokontrol ng mga church administrator kung aling mga seksyon ang makikita ng mga miyembro sa pamamagitan ng B1 Admin. Kung hindi mo makita ang isang partikular na tab, maaaring hindi pa na-activate ng iyong simbahan ang feature na iyon.
:::

## Mobile Navigation

Sa mas maliliit na screen tulad ng mga telepono at tablet, nako-collapse ang sidebar para makatipid ng espasyo. Sa halip, makikita mo ang isang **Menu** na button sa itaas ng portal ng miyembro. I-tap ito para buksan ang isang listahan ng lahat ng available na mga tab. Pumili ng tab para mag-navigate sa seksyong iyon, at awtomatikong magsasara ang menu.

## Tab ng Administrator

Kung mayroon kang mga pahintulot bilang administrator sa iyong simbahan, makikita mo ang karagdagang **Admin** na tab sa ibaba ng sidebar. Ang pag-click dito ay magdadala sa iyo sa B1 Admin kung saan maaari mong pamahalaan ang mga setting at data ng iyong simbahan.

## Navigasyon ng Upper Header

Sa labas ng portal ng miyembro, ang pangunahing header ng site ay naglalaman ng mga custom na navigation link ng iyong simbahan. Ang mga ito ay naka-configure ng mga administrator ng iyong simbahan at maaaring magsama ng mga link sa mga pahina tulad ng [mga sermon](../content/sermons.md), ang [Bibliya](../content/bible.md), [live streaming](../content/live-streaming.md), at iba pang nilalaman. Sa mobile, ang mga link na ito ay accessible sa pamamagitan ng hamburger menu icon sa kanang itaas ng header.
