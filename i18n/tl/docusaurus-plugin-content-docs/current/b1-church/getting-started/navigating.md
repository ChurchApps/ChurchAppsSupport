# Pag-navigate ng B1App

<div class="article-intro">

Ang member portal sa B1.church ay gumagamit ng sidebar navigation layout na nagsisiguro na madali ang paglipat sa pagitan ng iba't ibang tool na available sa iyo. Ang pahinang ito ay nagpapaliwanag kung paano gumagana ang navigation sa parehong desktop at mobile device.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kailangan mong maging [logged in](./logging-in.md) upang makuha ang access sa member portal.

</div>

## Sidebar Navigation

Kapag pumasok ka sa member portal, makikita mo ang isang vertical sidebar sa kaliwang gilid ng screen. Ang sidebar na ito ay naglalaman ng mga tab para sa bawat seksyon ng portal:

- **Timeline** -- Ang iyong personalized feed ng mga update at conversation mula sa iyong grupo at simbahan. Ito ang default tab na nag-load kapag bukas mo ang member portal.
- **Me** -- Ang iyong personal dashboard ng mga darating na serving assignment, registration, group event, at mga bagong notification. Tingnan ang [The Me Page](./me-page).
- **Groups** -- Tingnan at makipag-interact sa [mga grupo](../groups/) na kabilang mo.
- **Community** -- Tingnan ang church [member directory](../community/member-directory.md).
- **Plans** -- Tingnan ang mga darating na [service plan](../plans/) at ang iyong naitalang volunteer position.
- **Check-in** -- [Mag-check in](../checkin/) ng iyong sarili at ng iyong household para sa mga serbisyo at kaganapan.
- **Lessons** *(deprecated)* -- Ang in-portal Lessons tab ay pinalitan ng [FreePlay](/docs/freeplay/) para sa classroom playback. Tingnan ang curriculum direkta sa [Lessons.church](https://lessons.church) o tingnan ang [Lessons reference page](../content/lessons.md) para sa detalye.
- **Donations** -- Gumawa ng online [donation](../giving/) at tingnan ang iyong giving history.

I-click ang anumang tab upang magpalipat sa seksyong iyon. Ang aktibong tab ay naka-highlight upang palagi mong alam kung nasaan ka.

:::info
Ang mga tab na makikita mo ay maaaring mag-iba depende sa kung ano ang nag-enable ang iyong simbahan. Ang mga church administrator ay kumokontrol kung aling mga seksyon ay nakikita ng mga miyembro sa pamamagitan ng B1 Admin. Kung hindi mo nakikita ang partikular na tab, ang iyong simbahan ay maaaring hindi nag-turn on ng feature na iyon.
:::

## Mobile Navigation

Sa mas maliit na screen tulad ng telepono at tablet, ang sidebar ay bumubukod upang makatipid ng espasyo. Sa halip, makikita mo ang isang **Menu** button sa tuktok ng member portal. I-tap ito upang buksan ang listahan ng lahat ng available na tab. Pumili ng tab upang mag-navigate sa seksyong iyon, at ang menu ay awtomatikong magsasara.

## Admin Tab

Kung mayroon kang administrator permission sa iyong simbahan, makikita mo ang isang karagdagang **Admin** tab sa ibaba ng sidebar. Ang pag-click nito ay dadalhin ka sa B1 Admin kung saan maaari mong pamahalaan ang setting at data ng iyong simbahan.

## Top Header Navigation

Sa labas ng member portal, ang main site header ay naglalaman ng custom navigation link ng iyong simbahan. Ang mga ito ay kinokonfigura ng mga administrator ng iyong simbahan at maaaring magsama ng mga link sa mga pahina tulad ng [sermon](../content/sermons.md), ang [Bible](../content/bible.md), [live streaming](../content/live-streaming.md), at iba pang content. Sa mobile, ang mga link na ito ay accessible sa pamamagitan ng hamburger menu icon sa top-right corner ng header.
