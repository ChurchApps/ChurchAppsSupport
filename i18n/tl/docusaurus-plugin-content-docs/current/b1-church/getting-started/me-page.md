# Ang Me Page

<div class="article-intro">

Ang **Me** page ay ang iyong personal dashboard sa member portal — isang chronological view ng lahat ng darating para sa iyo: serving assignments, event registrations, at mga darating na events ng iyong mga grupo, kasama ang iyong mga kamakailang notification.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Kailangan mong maging [naka-log in](./logging-in.md) — ang Me page ay lumalitaw lamang para sa mga naka-sign in na miyembro

</div>

## Pagbubukas ng Me Page

Buksan ang portal menu at i-tap ang **Me**. Ang page ay nag-load ng iyong personal na impormasyon na inorganisa sa mga seksiyon.

## Darating

Ang **Upcoming** section ay naglalista ng lahat ng darating para sa iyo sa order ng petsa, bawat item ay may tag ng uri:

- **Serving** -- Ang iyong mga volunteerong posisyon sa mga darating na service plans. I-tap ito upang buksan ang plan.
- **Registration** -- Ang mga events na iyong [narehistro na](../events/registering). I-tap upang pumunta sa iyong [registrations](../events/my-registrations).
- **Event** -- Mga darating na events mula sa mga grupo na kabilang ka. I-tap ito upang buksan ang mga events ng grupo.

Kung walang naka-schedule, makikita mo ang "Walang darating" — ang iyong serving times, events, at registrations ay lalitaw dito habang sila ay naka-schedule.

## Mga Kamakailang Notification

Sa ibaba ng Upcoming, ang **Recent Notifications** section ay nagpapakita ng iyong mga kamakailang notification upang makasabay ka nang hindi bumubukas ng bawat area ng portal.

## Mga Shortcut

Kung ang iyong simbahan ay may mga bookable rooms o resources, ang Me page ay nagpapakita rin ng **Shortcuts** section:

- **Request an Event** -- Humiling na magsagawa ng isang event kasama ang mga kuwarto at kagamitan na kailangan mo. Tingnan ang [Requesting Events & Rooms](../events/requesting-events).
- **My Requests** -- Subaybayan ang status ng iyong mga hiling at kanselahin ang mga naghihintay.

Kung ang iyong simbahan ay wala pang nag-set up ng mga kuwarto o resources, ang mga shortcut na ito ay nakatago.
