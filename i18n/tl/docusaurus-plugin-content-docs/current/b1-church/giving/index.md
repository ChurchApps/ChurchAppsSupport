# Pagbibigay

<div class="article-intro">

Pinapadali ng B1App ang pagsuporta sa iyong simbahan sa pamamagitan ng online na donasyon. Gusto man gumawa ng mabilis na one-time na regalo o pamahalaan ang mga regular na pagbibigay, lahat ng kailangan mo ay available mula sa B1.church site ng iyong simbahan.

</div>

## Pangkalahatang-ideya

Sinusuportahan ng B1App ang dalawang paraan ng pagbibigay:

- **Mga pampublikong donasyon** -- Sinuman ay maaaring mag-donate nang hindi nag-lo-log in. Bisitahin ang donate page ng iyong simbahan, pumili ng fund, at ilagay ang iyong mga detalye ng pagbabayad. Hindi kinakailangan ang account.
- **Naka-authenticate na pagbibigay** -- Kapag [nag-log in](../getting-started/logging-in.md) ka bago mag-donate, ang iyong mga regalo ay nakakonekta sa iyong account. Nagbibigay ito sa iyo ng access sa buong kasaysayan ng donasyon, mga napi-print na giving statement, pamamahala ng regular na donasyon, at mga naka-save na paraan ng pagbabayad.

## Ano ang Magagawa Mo

Narito ang available sa seksyon ng pagbibigay:

- **Mag-donate sa mga tiyak na fund** -- Maaaring may maraming fund na naka-set up ang iyong simbahan, tulad ng General Fund, Building Fund, o Missions Fund. Maaari mong piliin kung saang fund ididirekta ang iyong donasyon.
- **Magbayad gamit ang card o bank account** -- Ang mga donasyon ay ligtas na pinoproseso sa pamamagitan ng Stripe. Maaari kang magbayad gamit ang credit o debit card. Available din ang mga tagubilin para sa ACH bank payment.
- **Mag-set up ng mga regular na donasyon** -- Ang mga naka-log in na miyembro ay maaaring mag-schedule ng mga awtomatikong regular na donasyon para hindi mo makalimutan.
- **Tingnan ang kasaysayan ng donasyon** -- Tingnan ang kumpletong listahan ng iyong mga nakaraang donasyon, kasama ang petsa, paraan, fund, at halaga.
- **Mag-print ng mga giving statement** -- Mag-generate ng mga napi-print na giving statement para sa kasalukuyan o nakaraang taon para sa iyong mga tax record.
- **Pamahalaan ang mga paraan ng pagbabayad** -- I-save at pamahalaan ang iyong mga paraan ng pagbabayad para sa mas mabilis na pagbibigay sa hinaharap.

:::info
Ang mga donasyon ay ligtas na pinoproseso sa pamamagitan ng **Stripe**. Ang iyong impormasyon sa pagbabayad ay hindi kailanman naka-store sa mga server ng B1.church.
:::
