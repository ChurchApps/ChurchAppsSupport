# Talata ng Araw

<div class="article-intro">

Ang Talata ng Araw na pahina ay nagpapakita ng pang-araw-araw na talata sa Bibliya bilang isang magandang naka-format na imahe. Ito ay isang simpleng paraan para simulan ang iyong araw sa Kasulatan o makahanap ng pampalakas ng loob sa buong linggo.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Hindi kailangan mag-log in para tumingin ng Talata ng Araw. Ang pahina ay pampublikong accessible.
- Dapat ay nagdagdag ang iyong simbahan ng Talata ng Araw na pahina sa kanilang site navigation.

</div>

## Pagtingin sa Talata

1. Mag-navigate sa **Talata ng Araw** na pahina mula sa site navigation ng iyong simbahan.
2. Ipinapakita ng pahina ang talata ngayon bilang isang imahe.
3. Awtomatikong lumalabas ang bagong talata araw-araw.

Wala kang kailangang gawin -- bisitahin lang ang pahina at ipapakita ang talata ng kasalukuyang araw.

## Responsive na Pagpapakita

Awtomatikong nagsasaayos ang imahe ng talata para magkasya sa iyong screen. Depende sa iyong device at laki ng window, makikita mo ang isa sa tatlong format:

- **Widescreen (16:9)** -- Pinakamainam para sa mga desktop monitor at laptop
- **Parisukat (1:1)** -- Angkop para sa mga screen na katulad ng laki ng tablet
- **Portrait (9:16)** -- Na-optimize para sa mga telepono na hawak nang patayo

Nakaka-detect ang pahina ng mga dimensyon ng iyong screen at pinipili ang pinakamainam na format, kaya palaging maganda ang hitsura ng talata anuman ang iyong paraan ng pagtingin. Kung ire-resize mo ang window ng iyong browser, mag-a-update ang imahe para tumugma sa mga bagong dimensyon.

:::tip
Gawin mong pang-araw-araw na gawi ang pagsuri ng Talata ng Araw, pagkatapos ay tuklasin pa ang talata sa [Bible reader](./bible.md).
:::

## Availability

Ang Talata ng Araw ay isang built-in na feature ng nilalaman ng B1.church. Kung hindi mo ito makita sa navigation ng iyong simbahan, maaaring hindi ito idinagdag ng administrator ng iyong simbahan sa site menu. Makipag-ugnayan sa administrator ng iyong simbahan para hilingin ito.
