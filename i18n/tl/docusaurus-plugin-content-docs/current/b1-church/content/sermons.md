# Mga Sermon

<div class="article-intro">

Ang Sermons page ay nagbibigay-daan sa iyo na mag-browse at manood ng mga naka-record na sermon mula sa iyong simbahan. Ang sermon library ng iyong simbahan ay ipinapakita gamit ang hitsura at layout na na-configure ng iyong simbahan, kaya ang dating ay katugma ng branding ng iyong simbahan.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Hindi kailangan mag-log in para tumingin ng mga sermon. Ang Sermons page ay pampublikong accessible.
- Dapat ay nagdagdag ang iyong simbahan ng sermon content sa pamamagitan ng kanilang admin portal.

</div>

## Pag-browse ng mga Sermon

1. Mag-navigate sa **Mga Sermon** page mula sa site navigation ng iyong simbahan.
2. Makikita mo ang listahan ng mga available na sermon, karaniwang naka-organisa ayon sa playlist o serye.
3. I-click ang anumang sermon para manood o makinig.

Ang mga sermon ay maaaring may kasamang video o audio content na naka-host sa mga platform tulad ng YouTube, Vimeo, o Facebook. Kapag nag-click ka sa isang sermon, ang media ay nagpe-play nang direkta sa pahina.

## Ano ang Makikita Mo

Ang bawat sermon entry ay maaaring magpakita ng:

- **Titulo** -- Ang pangalan ng sermon
- **Tagapagsalita** -- Kung sino ang nagbigay ng mensahe
- **Petsa** -- Kailan naka-record o na-publish ang sermon
- **Paglalarawan** -- Isang buod ng nilalaman ng sermon
- **Thumbnail** -- Isang preview image para sa sermon

Inoorganisa ng iyong simbahan ang mga sermon sa mga playlist o serye. Pinapadali nito ang pagsunod sa isang multi-week na serye o paghahanap ng mga sermon sa isang partikular na paksa.

:::tip
Kung naghahanap ka ng live broadcast sa halip na isang naka-record na sermon, tingnan ang [Live Streaming](./live-streaming.md) page.
:::

## Availability

Ang Sermons page ay available kapag ang iyong simbahan ay nagdagdag ng sermon content sa pamamagitan ng kanilang admin portal. Kung hindi mo makita ang **Mga Sermon** link sa navigation ng iyong simbahan, maaaring hindi pa na-configure ng iyong simbahan ang feature na ito. Makipag-ugnayan sa administrator ng iyong simbahan para sa karagdagang impormasyon.
