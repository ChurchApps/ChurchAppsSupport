# Teacher View para sa mga Aralin

<div class="article-intro">
Ang Teacher View ay nagbibigay ng expanded, full-screen na interface para sa pagtingin ng nilalaman ng aralin na may nakabalangkas na mga seksyon, na ginagawang mas madali para sa mga guro at lider na mag-navigate sa mga materyales ng aralin sa oras ng klase.
</div>

<div class="prereqs">
<h4>Bago Kayo Magsimula</h4>

- Ang inyong simbahan ay dapat mayroong mga naka-iskedyul na aralin sa isang plano. Tingnan ang gabay ng B1 Admin sa [Scheduling Lessons](../../b1-admin/serving/scheduling-lessons.md) para sa mga detalye.
- Kailangan ninyong maging nakatakda sa plano o magkaroon ng access upang tingnan ito sa inyong B1.church site.
</div>

## Pagbubukas ng Teacher View

Upang ma-access ang expanded na teacher view para sa isang aralin:

1. Mag-navigate sa seksyon ng **Plans** sa B1.church
2. Buksan ang plano na naglalaman ng aralin na gusto ninyong ituro
3. Hanapin ang aralin sa mga item ng plano
4. I-click ang aralin upang tingnan ang mga detalye nito
5. I-click ang **Teacher View** o **Expanded View** button

Ang aralin ay bubuksan sa isang full-screen na interface na na-optimize para sa pagtuturo.

:::tip
Ang Teacher View ay lalo na kapaki-pakinabang kapag nagpro-project ng nilalaman ng aralin sa screen o TV para makita ng klase, o kapag nagtuturo mula sa tablet.
:::

## Mga Feature ng Teacher View

### Tabbed Navigation

Ang teacher view ay nag-oorganisa ng nilalaman ng aralin sa mga seksyon gamit ang mga tab sa itaas:

- Ang bawat pangunahing seksyon ng aralin ay lumalabas bilang hiwalay na tab
- I-click ang anumang tab upang tumalon nang direkta sa seksyon na iyon
- Ang aktibong seksyon ay naka-highlight sa tab bar
- Ang mga tab ay ginagawang madaling lumipat-lipat sa iba't ibang bahagi ng aralin

### Scrollable na Nilalaman

Ang pangunahing content area ay nagpapakita ng buong mga materyales ng aralin:

- Mag-scroll sa aralin nang natural
- Habang nag-scroll kayo, ang aktibong tab ay awtomatikong nag-aupdate upang tumugma sa seksyon na inyong tinitingnan
- Ang lahat ng naka-format na teksto, larawan, at mga tagubilin ay malinaw na ipinapakita

### Pagpapatugtog ng Media

Kapag ang mga aralin ay may kasamang mga video, larawan, o ibang media:

- I-click ang anumang item ng media upang buksan ito sa popup player
- Ang mga video ay nagpe-play sa isang dedicated na video player na may mga karaniwang kontrol
- Ang mga larawan ay bubuksan sa lightbox para sa full-screen na pagtingin
- Isara ang media player upang bumalik sa nilalaman ng aralin

### Mga Mada-download na Resource

Kung ang aralin ay may kasamang mga mada-download na file (mga activity sheet, coloring page, atbp.):

- Lumalabas ang mga download button para sa bawat resource
- I-click upang direktang i-download ang mga file sa inyong device
- Ang mga file ay maaaring maglaman ng mga PDF, larawan, o ibang mga materyales na ibinigay ng content creator

## Paggamit ng Teacher View sa Silid-aralan

### Bago ang Klase

1. Buksan ang aralin sa Teacher View bago magsimula ang klase
2. Suriin ang lahat ng seksyon at pamilyarin ang inyong sarili sa nilalaman
3. I-download ang anumang mga printable na resource na kailangan ninyo
4. Subukan ang anumang mga video o media upang matiyak na gumagana silang maayos

### Sa Panahon ng Klase

1. Panatilihing bukas ang Teacher View sa inyong device o i-project ito sa screen
2. Gamitin ang mga tab upang tumalon sa pagitan ng mga seksyon habang umuusad kayo sa aralin
3. Mag-play ng mga video o magpakita ng mga larawan sa tamang oras
4. Mag-scroll sa mga tagubilin kung kinakailangan nang hindi nawawalan ng inyong lugar

### Sa Naka-project na Screen

Kapag nagpro-project para sa klase:

- Ang full-screen na layout ay nag-aalis ng mga distraksyon
- Ang malalaking, nababasang teksto ay ginagawang nakikita ang nilalaman mula sa kabila ng silid
- I-click ang mga item ng media upang magpakita ng mga video o larawan sa buong klase
- Mag-navigate gamit ang wireless mouse o trackpad

## Pagsasara ng Teacher View

Upang lumabas sa teacher view:

- I-click ang **Close** button (X) sa itaas na sulok
- O pindutin ang **Esc** key sa inyong keyboard
- Babalik kayo sa plan details view

## Gamit sa Mobile at Tablet

Gumagana ang Teacher View sa mga tablet at mobile device:

- **Mga Tablet** -- Perpekto para sa mga guro na gumagamit ng portable na gabay sa aralin
- **Mga Telepono** -- Sinusuportahan din, bagaman ang mas maliit na screen ay maaaring hindi gaanong perpekto para sa pagtuturo
- **Landscape mode** -- Nagbibigay ng pinakamahusay na karanasan sa pagtingin sa mga mobile device

## Mga Pagkakaiba sa Regular na Lesson View

| Feature | Regular View | Teacher View |
|---------|--------------|--------------|
| **Layout** | Compact, inline sa plano | Full-screen, dedicated na view |
| **Navigation** | Scroll lamang | Mga tab + scroll na may auto-highlighting |
| **Media** | Naka-embed sa pahina | Popup player para sa nakatuong pagtingin |
| **Espasyo sa screen** | Nagpapakita ng ibang mga item ng plano | Pinalaki para sa nilalaman ng aralin |
| **Pinakamahusay para sa** | Pag-browse at pagpaplano | Pagtuturo at pagtatanghal |

## Mga Best Practice

- **Maghanda nang maaga** -- Suriin ang mga aralin sa Teacher View bago ang klase upang pamilyarin ang inyong sarili sa layout
- **Mag-bookmark ng mga seksyon** -- Tandaan kung aling mga tab ang naglalaman ng mga pangunahing aktibidad o mga talakayan
- **Subukan ang media** -- Tiyakin na maayos na naglo-load ang mga video at larawan bago ang oras ng klase
- **Mag-download ng mga resource nang maaga** -- Huwag maghintay hanggang oras ng klase upang mag-download ng mga printable
- **Gumamit ng mas malaking screen** -- Mag-project o gumamit ng tablet para sa pinakamahusay na karanasan sa pagtuturo

## Troubleshooting

### Hindi Bumubukas ang Aralin sa Teacher View

- Tiyakin na ang aralin ay maayos na naka-iskedyul sa plano
- Suriin na mayroon kayong pahintulot upang tingnan ang plano
- I-refresh ang pahina at subukan muli

### Hindi Nagpe-play ang mga Video

- Suriin ang inyong internet connection
- Tiyakin na pinapayagan ng inyong device ang pagpapatugtog ng media
- Subukang buksan ang video sa hiwalay na tab
- Ang ilang content provider ay maaaring may mga paghihigpit sa pagpapatugtog ng video

### Ang Nilalaman ay Mukhang Naputol sa Mobile

- I-rotate ang inyong device sa landscape mode
- Mag-zoom out nang kaunti kung kinakailangan
- Isaalang-alang ang paggamit ng tablet o mas malaking device para sa mas mahusay na visibility
