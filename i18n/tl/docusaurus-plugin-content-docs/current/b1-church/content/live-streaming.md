# Live Streaming

<div class="article-intro">

Ang Live Stream page ay nagbibigay-daan sa iyo na manood ng mga serbisyo ng iyong simbahan habang nangyayari, nang direkta mula sa B1.church. Kapag nagbo-broadcast ang iyong simbahan, maaari kang sumali at makibahagi gamit ang mga interactive na feature sa tabi ng video.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Hindi kailangan mag-log in para manood ng live stream. Ang Stream page ay pampublikong accessible.
- Dapat ay may naka-configure na streaming schedule ang iyong simbahan. Suriin ang mga oras ng serbisyo ng iyong simbahan para malaman kung kailan available ang mga live broadcast.

</div>

## Panonood ng Live Stream

1. Mag-navigate sa **Stream** page mula sa site navigation ng iyong simbahan.
2. Kung kasalukuyang live o malapit nang magsimula ang isang serbisyo, awtomatikong maglo-load ang video player.
3. Manood ng broadcast nang direkta sa pahina.

Kapag walang aktibong live stream, maaari kang makakita ng offline na nilalaman o impormasyon tungkol sa susunod na naka-schedule na oras ng serbisyo.

## Mga Interactive na Feature

Ang live stream page ay maaaring may kasamang mga interactive na tab sa tabi ng video, depende sa kung ano ang na-configure ng iyong simbahan. Maaaring kasama ang:

- **Chat** -- Magpadala ng mga mensahe at makipag-ugnayan sa ibang manonood habang nagbo-broadcast. Ang chat ay karaniwang na-enable ilang minuto bago magsimula ang serbisyo at nananatiling available sa loob ng isang panahon pagkatapos.
- **Panalangin** -- Magpadala ng prayer request habang may serbisyo. Ang isang streaming host sa team ng iyong simbahan ay maaaring sumagot nang pribado.
- **Karagdagang mga tab** -- Ang iyong simbahan ay maaaring magdagdag ng custom na mga tab na may mga link sa mga bulletin, sermon notes, [giving pages](../giving/making-donations.md), o iba pang resources.

I-click ang mga tab icon o label para lumipat sa pagitan ng mga available na feature habang nanonood.

## Bago at Pagkatapos ng Serbisyo

Nagtatakda ang iyong simbahan ng mga tiyak na oras para sa kung kailan magiging available ang stream at mga interactive na feature. Maaari kang sumali sa chat bago opisyal na magsimula ang serbisyo, na nagbibigay sa iyo ng oras para bumati sa iba. Pagkatapos matapos ang serbisyo, maaaring manatiling bukas ang chat nang sandali para sa tuloy-tuloy na pag-uusap.

:::info
Kung ang stream page ay nagpapakita ng offline na nilalaman o countdown, ang susunod na naka-schedule na serbisyo ay hindi pa nagsisimula. Bumalik sa nakalistang oras ng serbisyo.
:::

:::tip
Naghahanap ng naka-record na sermon na napalampas mo? Bisitahin ang [Mga Sermon](./sermons.md) page para mag-browse ng mga nakaraang mensahe mula sa iyong simbahan.
:::

## Availability

Ang Live Stream page ay available kapag ang iyong simbahan ay may naka-configure na streaming schedule. Kung hindi mo makita ang **Stream** link sa iyong navigation, maaaring walang naka-set up na live streaming ang iyong simbahan. Makipag-ugnayan sa administrator ng iyong simbahan para sa mga detalye.
