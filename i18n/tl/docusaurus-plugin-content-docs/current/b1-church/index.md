# B1.church

<div class="article-intro">

Ang B1.church ay ang portal para sa mga miyembro ng iyong simbahan. Ito ang lugar kung saan ang iyong kongregasyon ay nakakakonekta, naa-access ang mga tool ng simbahan, at nakikipag-ugnayan sa komunidad sa buong linggo. Ang bawat simbahan ay may sariling URL sa `https://pangalanngsimbahan.b1.church`.

</div>

Kapag binisita ng mga miyembro ang iyong B1.church site, makikita nila ang pampublikong website ng iyong simbahan kasama ang iyong branding, mga link ng navigasyon, at mga pahina ng nilalaman. Pagkatapos mag-log in, maa-access na nila ang **Portal ng Miyembro** kung saan nakalagay ang lahat ng kanilang personal na mga tool ng simbahan.

## Mga Pangunahing Feature

- **Timeline** -- Isang personalized na feed ng mga update, pag-uusap, at aktibidad mula sa iyong mga grupo at simbahan.
- **Mga Grupo** -- Tingnan ang mga grupo na kinabibilangan mo, makita ang mga miyembro ng grupo, sumali sa mga pag-uusap, at subaybayan ang attendance.
- **Pagbibigay** -- Mag-donate sa iyong simbahan online at tingnan ang iyong kasaysayan ng pagbibigay.
- **Mga Plano** -- Tingnan ang mga paparating na plano ng serbisyo, ang iyong mga nakatalagang posisyon, at i-access ang mga detalye ng kanta at order ng serbisyo.
- **Check-In** -- I-check in ang iyong sarili at ang mga miyembro ng iyong sambahayan para sa mga serbisyo at kaganapan.
- **Direktoryo ng Komunidad** -- I-browse ang direktoryo ng mga miyembro ng iyong simbahan, tingnan ang mga profile, at magpadala ng direktang mensahe.
- **Nilalaman** -- I-access ang mga sermon, basahin ang Bibliya, mag-aral ng mga leksyon, at manood ng live stream, lahat mula sa site ng iyong simbahan.
