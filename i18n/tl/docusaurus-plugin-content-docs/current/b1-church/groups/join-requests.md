# Naghihiling ng Pagsali sa isang Grupo

<div class="article-intro">

Ang ilang mga grupo ay nangangailangan ng pagsang-ayon bago ka magiging miyembro. Kapag mayroon ang isang grupo ng itaas na setting, nagsasabing sumali ng join request — opsyonal na kasama ang isang mensahe sa pinuno ng grupo — at naghihintay para sa pinuno na aprubahan o tanggihan. Maaari mong tingnan ang status ng iyong mga naghihintay na request at kanselahin ang mga ito sa anumang oras.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Dapat kang maging [naka-log in](../getting-started/logging-in.md) sa iyong B1.church account upang humiling ng pagsali sa isang grupo.

</div>

## Pagsali kumpara sa Naghihiling ng Pagsali

Ang mga grupo ay maaaring magkaroon ng iba't ibang mga patakaran sa pagsali na itinakda ng kanilang mga administrator:

- **Bukas** -- Mag-click sa **Join Group** sa detalye ng grupo at ikaw ay agad na idadagdag bilang miyembro.
- **Kinakailangan ang request** -- Mag-click sa **Request to Join** upang sumali ng isang request na kailangang aprubahan ng isang pinuno ng grupo bago ka maging miyembro.
- **Sarado** -- Walang pindot na pagsali na lumilitaw. Ang mga miyembro ay dapat idagdag nang manu-mano ng mga pinuno.

## Paano Magsumite ng Join Request

1. Tuklasin ang grupo na gusto mong sumali. Tingnan ang [Browsing Groups](./browsing-groups.md) kung kailangan mo ng tulong sa paghahanap dito.
2. Buksan ang detalye ng grupo.
3. Mag-click sa **Request to Join**.
4. Lumilitaw ang isang dialog kung saan maaari kang optional na magsulat ng isang mensahe sa pinuno ng grupo na nagpapaliwanag kung bakit gusto mong sumali.
5. Mag-click sa **Submit** upang ipadala ang iyong request.

Ang mga pinuno ng grupo ay makakatanggap ng isang notification tungkol sa iyong request. Kapag sinuri nila ito, makakatanggap ka ng notification sa resulta.

:::tip
Ang kasama ng isang maikling mensahe — tulad ng kung paano mo narinig tungkol sa grupo o kung ano ang iyong inaasahan na makuha dito — ay maaaring tumulong sa pinuno na gumawa ng mas mabilis na desisyon.
:::

## Pagsusuri ng Status ng Iyong mga Request

Pagkatapos magsumite ng request, maaari mong tingnan ang status nito mula sa **Groups** page:

1. Mag-navigate sa **Groups** page sa website ng iyong simbahan B1.church.
2. I-scroll down sa **Pending Requests** section.
3. Bawat naghihintay na request ay nagpapakita ng pangalan ng grupo at ang petsa na isinumite mo ito.

Kung ang isang request ay aprubado o tinanggihan, makakatanggap ka ng notification — ang mga aprubadong request ay maglalagay sa iyo sa grupo bilang miyembro.

## Kinokansela ang Request

Kung nagbabago ang iyong isip bago tinukoy ng isang pinuno ang iyong request:

1. Pumunta sa **Groups** page.
2. Hanapin ang iyong request sa **Pending Requests** section.
3. Mag-click sa **Cancel** sa tabi ng request.

Ang request ay agad na nababawi at ang pinuno ay hindi na makikita ito.
