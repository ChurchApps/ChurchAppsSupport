# Mga Detalye ng Grupo

<div class="article-intro">

Kapag nag-click ka sa isang grupo mula sa groups page, dadalhin ka sa group detail page. Ang makikita mo ay depende sa kung naka-log in ka o hindi, kung saan ang mga naka-log in na miyembro ay may access sa karagdagang mga tab at feature.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Humanap ng grupo na titingnan sa pamamagitan ng [pag-browse ng mga grupo](./browsing-groups.md).
- Para sa buong access sa lahat ng tab, [mag-log in](../getting-started/logging-in.md) sa iyong account.

</div>

## View ng Bisita (Hindi Naka-log In)

Kung hindi ka naka-log in, makikita mo ang sumusunod:

- **Hero section** -- Isang malaking banner area na nagpapakita ng larawan, pangalan, oras ng pagtitipon, at lokasyon ng pagtitipon ng grupo.
- **Paglalarawan** -- Ang buong paglalarawan ng grupo na nagpapaliwanag ng layunin nito at kung ano ang maaasahan.
- **Mga Lider** -- Mga avatar at pangalan ng mga lider ng grupo. Maaari mong i-click ang pangalan ng isang lider para tingnan ang kanilang profile.
- **Mga kaganapan sa kalendaryo** -- Hanggang tatlong paparating na kaganapan para sa grupo ang ipinapakita, bawat isa ay nagpapakita ng petsa at titulo ng kaganapan. Kung walang paparating na kaganapan ang grupo, nakatago ang seksyong ito.
- **Contact form** -- Isang form na nagbibigay-daan sa iyo na magpadala ng mensahe nang direkta sa mga lider ng grupo. Punan ang iyong **pangalan**, **apelyido**, **email**, **numero ng telepono**, at **mensahe**, pagkatapos ay i-click ang **Submit**. Kung maraming lider, maaari mong piliin kung sino ang makikipag-ugnayan mula sa isang dropdown.

## View ng Miyembro (Naka-log In)

Kapag naka-log in ka, ang group detail page ay nagpapakita ng hero section kasama ang isang sidebar ng mga tab. Ang mga available na tab ay:

- **Mga Detalye ng Grupo** -- Ang buong paglalarawan ng grupo, naka-render na may rich text formatting.
- **Mga Miyembro** -- Isang listahan ng lahat ng miyembro sa grupo, nagpapakita ng kanilang larawan at pangalan. Maaari mong i-click ang pangalan ng isang miyembro para bisitahin ang kanilang profile sa [direktoryo ng miyembro](../community/member-directory.md).
- **Kalendaryo** -- Ang kalendaryo ng grupo na nagpapakita ng mga pagtitipon at kaganapan. Maaari mong i-browse ang mga paparating at nakaraang kaganapan.
- **Mga Pag-uusap** -- Isang espasyo para sa mga miyembro ng grupo na mag-post ng mga mensahe at makipag-ugnayan sa isa't isa. Tingnan ang [Mga Pag-uusap sa Grupo](./conversations.md) para sa higit pang mga detalye.
- **Mga Resources** -- Mga shared file at link na nai-post para sa grupo. Maaari mong i-download ang mga na-upload na file o buksan ang mga naka-link na resources.
- **Attendance** -- Nakikita lamang ng mga lider ng grupo. Nagbibigay-daan sa pagsubaybay kung sino ang dumalo sa mga pagtitipon ng grupo.

## Mga Kakayahan ng Lider

Kung ikaw ay isang lider ng grupo, mayroon kang karagdagang kakayahan sa pag-edit:

- **I-edit ang mga detalye ng grupo** -- I-update ang pangalan, paglalarawan, oras ng pagtitipon, at iba pang mga setting ng grupo nang direkta mula sa **Details** tab.
- **Pamahalaan ang mga miyembro** -- Magdagdag ng bagong mga miyembro sa grupo o mag-alis ng mga kasalukuyan mula sa **Members** tab.
- **Pamahalaan ang mga resources** -- Mag-upload ng mga file (hanggang 100 MB ng storage), magdagdag ng mga link, o mag-delete ng mga kasalukuyang resources.
- **Mga resources para sa lider** -- I-access ang isang hiwalay na **Resources (Leaders)** tab na may mga file at link na nakikita lamang ng mga lider ng grupo.
- **Subaybayan ang attendance** -- I-record ang attendance para sa mga session ng grupo mula sa **Attendance** tab.

:::info
Kung hindi ka pa miyembro ng isang grupo, maaari mong gamitin ang contact form sa visitor view para makipag-ugnayan sa mga lider ng grupo at magtanong tungkol sa pagsali.
:::
