# Mga Pag-uusap sa Grupo

<div class="article-intro">

Ang Conversations tab sa iyong group page ay isang espasyo kung saan ang mga miyembro ng grupo ay maaaring mag-post ng mga mensahe at makipag-ugnayan sa isa't isa. Gamitin ang mga pag-uusap para talakayin ang mga paksa, ibahagi ang mga update, at manatiling konektado sa iyong grupo sa pagitan ng mga pagtitipon.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Dapat kang naka-[log in](../getting-started/logging-in.md) sa iyong B1.church account.
- Dapat kang miyembro ng grupo para ma-access ang Conversations tab. Tingnan ang [Mga Detalye ng Grupo](./group-details.md) para malaman kung paano sumali sa isang grupo.

</div>

## Pag-access sa mga Pag-uusap

1. Mag-navigate sa group detail page sa pamamagitan ng pag-click sa isang grupo mula sa [groups page](./browsing-groups.md).
2. Siguraduhing naka-log in ka.
3. I-click ang **Conversations** tab sa sidebar.

## Mga Diskusyon at Anunsyo

Ang Conversations tab ay maaaring magpakita ng dalawang sub-tab:

- **Mga Diskusyon** -- Isang espasyo kung saan ang sinumang miyembro ng grupo ay maaaring mag-post ng mga mensahe at sumagot sa iba. Ito ang default na view at laging available.
- **Mga Anunsyo** -- Mga mensaheng nai-post ng mga lider ng grupo na mahalaga para sa lahat ng miyembro. Ang mga regular na miyembro ay maaaring magbasa ng mga anunsyo ngunit tanging mga lider lang ang maaaring gumawa ng mga ito.

Kung walang mga anunsyo at hindi ka lider, makikita mo lang ang Discussions view nang walang tab bar.

## Pag-post ng Mensahe

1. Buksan ang **Conversations** tab sa iyong group page.
2. Siguraduhing nasa **Mga Diskusyon** sub-tab ka (kung nakikita ang tab bar).
3. I-type ang iyong mensahe sa text field.
4. I-submit ang iyong mensahe para ibahagi ito sa grupo.

Ang iyong mensahe ay makikita ng lahat ng miyembro ng grupo.

:::tip
Ang mga pag-uusap sa grupo ay lumalabas din sa iyong [Timeline](../community/timeline.md), para makasubaybay ka sa mga diskusyon nang hindi binibisita ang bawat grupo nang isa-isa.
:::

## Mga Feature para sa Lider

Kung ikaw ay isang lider ng grupo, mayroon kang karagdagang kakayahan sa Conversations tab:

- **Mag-post ng mga anunsyo** -- Lumipat sa **Mga Anunsyo** sub-tab at mag-post ng mga mensaheng mababasa ng lahat ng miyembro ng grupo.
- **Mag-post ng mga diskusyon** -- Maaari ka ring makibahagi sa mga regular na diskusyon tulad ng sinumang ibang miyembro.

:::info
Ang mga pag-uusap ay tiyak sa bawat grupo. Ang mga mensaheng nai-post mo sa isang grupo ay hindi lalabas sa mga pag-uusap ng ibang grupo.
:::
