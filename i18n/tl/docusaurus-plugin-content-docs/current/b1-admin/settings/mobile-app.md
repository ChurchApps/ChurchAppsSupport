# Mga Setting ng Mobile App

<div class="article-intro">

Ang pahina ng Mga Setting ng Mobile App ay nagbibigay-daan sa iyong i-configure ang mga navigation tab na lumalabas sa B1 Mobile app para sa mga miyembro ng iyong simbahan. Ikaw ang kumokontrol kung aling mga tab ang nakikita, kung saan sila nagli-link, at kung paano sila ipinapakita.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kailangan mo ang pahintulot na "Edit Church Settings". Tingnan ang [Mga Tungkulin at Pahintulot](./roles-permissions.md) kung wala kang access.
- I-configure muna ang iyong [Mga Setting ng Simbahan](./church-settings.md), kabilang ang pangalan ng iyong simbahan at branding

</div>

## Pag-access sa Mga Setting ng Mobile App

1. Mag-navigate sa pahina ng **Mga Setting** sa kaliwang sidebar.
2. I-click ang button na **Mga Mobile App** sa header.
3. Ipinapakita ng pahina ng Mga Setting ng Mobile App ang iyong kasalukuyang mga app tab.

## Pagdagdag ng Bagong Tab

1. I-click ang button na **Magdagdag ng Tab** sa itaas ng pahina.
2. Punan ang mga detalye ng tab:
   - **Pangalan** -- Ang label na lumalabas sa tab (halimbawa, "Mga Sermon" o "Mag-donate").
   - **Icon** -- I-click ang icon selector upang pumili ng icon para sa iyong tab. Maaari ka ring mag-upload ng custom na imahe.
   - **Uri ng Tab** -- Pumili mula sa mga opsyon tulad ng Bible, Live Stream, Donation, Website, at iba pa.
   - **URL** -- Ilagay ang web address na dapat i-link ng tab.
   - **Visibility** -- Kontrolin kung sino ang makakakita sa tab na ito (lahat, mga miyembro lamang, atbp.).
3. I-click ang **I-save ang Tab** upang idagdag ito sa iyong app.

## Pag-edit ng Umiiral na Tab

1. I-click ang anumang umiiral na tab sa listahan ng **Mga App Tab**.
2. I-update ang pangalan, icon, URL, uri, o mga setting ng visibility ng tab.
3. I-click ang **I-save ang Tab** upang i-apply ang iyong mga pagbabago.

## Muling Pag-aayos ng mga Tab

Maaari mong baguhin ang pagkakasunud-sunod kung paano lumalabas ang mga tab sa mobile app. I-drag at i-drop ang mga tab sa listahan upang ayusin muli ang mga ito. Ang pagkakasunud-sunod na ipinapakita sa pahinang ito ay tumutugma sa pagkakasunud-sunod na makikita ng iyong mga miyembro sa app.

:::info
Ang ilang mga tab ay maaaring awtomatikong lumabas kapag natugunan ang ilang mga kundisyon -- halimbawa, ang isang Live Stream tab ay maaaring lumabas kapag aktibo ang isang stream. Ang mga manu-manong idinagdag na tab ay nagbibigay sa iyo ng buong kontrol sa kung ano ang nakikita ng iyong mga miyembro sa lahat ng oras.
:::

:::tip
Panatilihing mapapamahalaan ang bilang ng iyong mga tab. Tatlo hanggang limang tab ang gumagana nang maayos para sa karamihan ng mga simbahan. Masyadong maraming tab ay maaaring gawing nakakalito ang navigation para sa iyong mga miyembro.
:::

## Saan Lumalabas ang mga Tab na Ito

Ang mga tab na iko-configure mo rito ay ipinapakita sa **B1 Mobile app** (B1.church app) na dina-download ng mga miyembro ng iyong simbahan mula sa App Store o Google Play. Ang mga pagbabagong ginawa mo sa pahinang ito ay makikita sa app sa susunod na buksan ito ng isang miyembro.
