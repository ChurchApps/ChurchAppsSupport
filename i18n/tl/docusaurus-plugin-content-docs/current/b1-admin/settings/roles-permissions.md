# Mga Tungkulin at Pahintulot

<div class="article-intro">

Ang mga tungkulin ay nagbibigay-daan sa iyong kontrolin kung ano ang maa-access ng iba't ibang user sa loob ng iyong ChurchApps account. Maaari kang lumikha ng mga custom na tungkulin para sa staff, mga volunteer, at iba pang miyembro ng team, bawat isa ay may kani-kanilang antas ng access upang panatilihing ligtas ang iyong data.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kailangan mo ang Domain Admin access o isang tungkulin na may pahintulot na mamahala ng mga tungkulin
- Maghanda ng listahan ng mga miyembro ng team at ang mga lugar na kailangan nilang ma-access
- Suriin ang mga available na kategorya ng pahintulot sa ibaba upang planuhin ang iyong mga tungkulin

</div>

## Pag-access sa Mga Tungkulin

1. Mag-navigate sa pahina ng **Mga Setting** sa kaliwang sidebar.
2. I-click ang button na **Mga Tungkulin** sa header.
3. Ipinapakita ng pahina ng Mga Tungkulin ang lahat ng kasalukuyang tinukoy na tungkulin para sa iyong simbahan.

## Pag-unawa sa Pahina ng Mga Tungkulin

Ang pahina ng Mga Tungkulin ay nahahati sa dalawang panel:

- **Kaliwang bahagi** -- Ipinapakita ang listahan ng mga miyembrong nakatalaga sa napiling tungkulin.
- **Kanang bahagi** -- Ipinapakita ang mga setting ng pahintulot na maaari mong i-configure para sa tungkuling iyon.

I-click ang anumang pangalan ng tungkulin upang tingnan at pamahalaan ang mga miyembro at pahintulot nito.

## Pagdagdag ng mga User sa isang Tungkulin

1. Piliin ang tungkulin na gusto mong dagdagan ng mga miyembro.
2. Gamitin ang **search field** sa kaliwang bahagi upang hanapin ang taong gusto mong idagdag.
3. Piliin ang tao mula sa mga resulta ng paghahanap.
4. Sila ay agad na idadagdag sa tungkulin.

## Pag-alis ng mga User mula sa isang Tungkulin

1. Piliin ang tungkulin na naglalaman ng user na gusto mong alisin.
2. Hanapin ang tao sa listahan ng miyembro sa kaliwang bahagi.
3. I-click ang **button na alisin** sa tabi ng kanilang pangalan.

## Pag-configure ng mga Pahintulot

Ang bawat tungkulin ay maaaring bigyan ng access sa mga partikular na lugar ng B1 Admin. Ang mga pahintulot ay nakaayos ayon sa seksyon:

- **Mga Tao** -- Access sa direktoryo ng miyembro at mga record ng tao.
- **Mga Donasyon** -- Access sa mga record ng donasyon at pamamahala ng pondo.
- **Attendance** -- Access sa pagsubaybay ng attendance at mga ulat.
- **Nilalaman** -- Access sa website at pamamahala ng nilalaman.
- At mga karagdagang lugar habang available ang mga ito.

Gamitin ang mga checkbox sa kanang bahagi ng pahina ng Mga Tungkulin upang i-enable o i-disable ang access para sa bawat lugar.

:::warning
Ang mga **Domain Admin** ay may buong access sa lahat ng lugar ng iyong ChurchApps account. Ang kanilang mga pahintulot ay hindi maaaring baguhin o limitahan. Gamitin ang tungkuling ito para lamang sa iyong mga pinaka-pinagkakatiwalaang administrator.
:::

:::tip
Gumawa ng mga partikular na tungkulin tulad ng "Ingat-yaman" na may **Mga Donasyon** access lamang, o "Check-In Volunteer" na may **Attendance** access lamang. Sinusunod nito ang prinsipyo ng pinakamababang pribilehiyo at pinapanatiling ligtas ang iyong data. Tingnan ang [Seguridad ng Data](./data-security.md) para sa higit pa tungkol sa kung paano pinoprotektahan ng ChurchApps ang iyong impormasyon.
:::
