# Mga Setting ng Simbahan

<div class="article-intro">

Ang pahina ng Mga Setting ng Simbahan ay kung saan mo iko-configure ang pangunahing impormasyon ng iyong simbahan, mga detalye ng pakikipag-ugnayan, at branding. Ang mga detalyeng ito ay ginagamit sa lahat ng ChurchApps na tool, kabilang ang iyong B1.church na website at ang B1 Mobile app.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kailangan mo ang pahintulot na "Edit Church Settings". Tingnan ang [Mga Tungkulin at Pahintulot](./roles-permissions.md) kung wala kang access.
- Ihanda ang address ng iyong simbahan, impormasyon sa pakikipag-ugnayan, at logo

</div>

## Pag-edit ng Impormasyon ng Iyong Simbahan

1. Mag-navigate sa pahina ng **Mga Setting** sa kaliwang sidebar.
2. I-click ang button na **I-edit ang Mga Setting** sa header.
3. I-update ang alinman sa mga sumusunod na field:
   - **Pangalan ng Simbahan** -- Ang pangalang ipinapakita sa lahat ng produkto ng ChurchApps.
   - **Address** -- Ang pisikal na address ng iyong simbahan.
   - **Impormasyon sa Pakikipag-ugnayan** -- Numero ng telepono, email, at iba pang mga detalye ng pakikipag-ugnayan.
4. I-click ang **Save** para i-apply ang iyong mga pagbabago.

## Pag-set Up ng Iyong Subdomain

Ang iyong simbahan ay nakakakuha ng libreng subdomain sa **yourchurch.b1.church**. Ito ang web address kung saan maa-access ng mga miyembro at bisita ang online na presensya ng iyong simbahan.

1. Sa pahina ng Mga Setting, hanapin ang field na **Subdomain**.
2. Ilagay ang iyong gustong subdomain (halimbawa, "gracechurch" para sa gracechurch.b1.church).
3. I-save ang iyong mga pagbabago.

:::info
Ang iyong subdomain ay dapat natatangi sa lahat ng ChurchApps na simbahan. Kung ang gusto mong pangalan ay nakuha na, subukang idagdag ang iyong lungsod o estado (halimbawa, "gracechurch-dallas").
:::

## Pag-configure ng Branding

I-customize kung paano lumalabas ang iyong simbahan sa lahat ng ChurchApps na tool:

1. I-upload ang iyong **logo ng simbahan** sa pamamagitan ng pag-click sa lugar ng logo at pagpili ng image file.
2. Magdagdag ng anumang karagdagang **mga larawan ng simbahan** na ginagamit sa iyong website at [mobile app](./mobile-app.md).

:::tip
Para sa pinakamainam na resulta, gumamit ng logo na may transparent na background sa PNG format. Tinitiyak nito na maganda itong tingnan sa parehong maliwanag at madilim na mga background.
:::

## Import at Export

Ang button na **Import/Export** sa header ng Mga Setting ay nagbubukas ng isang dedikadong tool sa bagong browser window. Gamitin ito para:

- Mag-import ng data ng miyembro mula sa ibang sistema ng pamamahala ng simbahan.
- Mag-export ng iyong ChurchApps na data para sa backup o migration.

Ito ay partikular na kapaki-pakinabang kapag una mong sine-set up ang iyong simbahan at kailangan mong ilipat ang mga umiiral na record sa ChurchApps.

:::warning
Kapag nag-i-import ng data, palaging i-back up muna ang iyong mga umiiral na record. Ang mga operasyon ng import ay nagdadagdag ng data sa iyong sistema at maaaring lumikha ng mga duplicate na entry kung tatakbo nang maraming beses.
:::
