# Pag-record ng mga Donasyon

<div class="article-intro">

Ang pag-record ng mga donasyon sa B1 Admin ay ginagawa sa pamamagitan ng Batches system. Gumagawa ka ng batch para kumatawan sa isang koleksyon (tulad ng Sunday offering), pagkatapos ay magdagdag ng mga indibidwal na donasyon sa batch na iyon. Pinapanatili nitong organisado at madaling i-reconcile ang iyong mga giving record.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- I-setup ang iyong mga [fund](funds.md) para ma-assign mo ang mga donasyon sa tamang mga kategorya
- Gumawa ng [batch](batches.md) para paglagyan ng mga donasyon na ila-lagay mo
- Siguraduhing ang mga donor ay nasa iyong [people directory](../people/adding-people.md) para mahanap mo sila kapag naglalagay ng mga alay

</div>

## Paggawa ng Batch at Pagdaragdag ng mga Donasyon

1. Sa **B1 Admin**, i-click ang **Donations** sa sidebar, pagkatapos i-click ang **Batches**.
2. I-click ang **Add Batch**.
3. Maglagay ng pangalan para sa batch (hal., "Sunday Offering - Jan 5") at piliin ang petsa. I-click ang **Save**.
4. Lalabas ang iyong bagong batch sa listahan na nagpapakita ng zero donation at $0.00.
5. I-click ang **pangalan ng batch** para buksan ito.

## Paglalagay ng mga Indibidwal na Donasyon

1. Sa batch detail page, i-type ang pangalan ng donor sa **search field** para hanapin sila.
2. Pagkatapos pumili ng tao, lalabas ang donation entry form na may mga field para sa **Date**, **Payment Method**, **Fund**, **Amount**, at **Check Number**.
3. Punan ang mga detalye at i-click ang **Add Donation**.
4. Ang donasyon ay idadagdag sa table sa ibaba, at mare-reset ang form para mailagay mo ang susunod.

:::tip
Maaari kang mabilis na maglagay ng maraming donasyon nang sunod-sunod nang hindi umaalis sa batch page. Nagre-reset ang form pagkatapos ng bawat entry para makapag-move ka sa isang stack ng mga tseke o sobre nang mahusay.
:::

## Pag-split ng Donasyon sa Maraming Fund

Minsan ang isang donor ay nagbibigay sa higit sa isang fund sa isang transaksyon. Para i-handle ito:

1. I-click ang **Edit** button sa donation row.
2. Sa edit form, magdagdag ng mga halaga sa iba't ibang fund. Ang kabuuan ay awtomatikong makakalkula mula sa mga indibidwal na halaga ng fund.
3. I-click ang **Save** para i-update ang donasyon.

:::info
Ang pag-split ng mga donasyon sa mga fund ay karaniwan kapag ang isang donor ay nagsulat ng isang tseke na itinalaga para sa maraming layunin, tulad ng General Fund at Missions.
:::

## Pag-edit o Pag-alis ng mga Donasyon

Para mag-edit ng donasyon, i-click ang **Edit** button sa row nito sa batch. Maaari mong baguhin ang petsa, halaga, fund, paraan ng pagbabayad, o anumang iba pang detalye. I-click ang **Save** kapag tapos ka na.

:::tip
Awtomatikong nag-a-update ang batch page header para ipakita ang kabuuang bilang ng mga donasyon at pinagsamang halaga sa dolyar habang nagdadagdag o nag-e-edit ka ng mga entry. Gamitin ito para i-reconcile laban sa iyong deposit slip.
:::

## Mga Susunod na Hakbang

- Suriin ang iyong mga entry gamit ang [Mga Report ng Donasyon](donation-reports.md) para i-verify ang katumpakan
- Sa year-end, bumuo ng [Mga Giving Statement](giving-statements.md) para sa iyong mga donor
