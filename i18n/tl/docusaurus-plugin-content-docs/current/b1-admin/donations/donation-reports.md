# Mga Report ng Donasyon

<div class="article-intro">

Ang B1 Admin ay nagbibigay sa iyo ng ilang paraan para tingnan at suriin ang giving data ng iyong simbahan. Ang Donations Summary page ay nagbibigay ng visual na pangkalahatang-tanaw na may mga chart at filter, habang ang Reports section ay nag-aalok ng mas detalyadong Donation Summary report. Gamitin ang mga tool na ito para subaybayan ang mga trend ng pag-aalay, maghanda para sa mga board meeting, o i-reconcile ang iyong mga record.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Siguraduhing ang mga donasyon ay [nai-record na sa mga batch](recording-donations.md) o [na-import mula sa Stripe](stripe-import.md)
- I-verify na ang iyong mga [fund](funds.md) ay tama ang setup para maayos na makategorya ang mga donasyon

</div>

## Donations Summary Page

Ang **Summary** page ang unang makikita mo kapag nag-click ka ng **Donations** sa sidebar. Nagbibigay ito ng mataas na antas na tanaw ng iyong giving activity.

1. Mag-navigate sa **Donations** sa sidebar para buksan ang Summary page.
2. Gamitin ang **date range filter** para piliin ang panahon na gusto mong suriin. Ilagay ang mas naunang petsa sa itaas at ang mas bagong petsa sa ibaba.
3. Ang page ay nagpapakita ng weekly giving chart para makita mo ang mga trend sa isang tingin.
4. I-click ang **Download** para mag-export ng CSV file na may kabuuang halaga ng donasyon, ang linggo ng pagbibigay, at ang fund kung saan ito ibinigay.

:::info
Ang Summary page ay nagpapakita ng pinagsama-samang giving data. Hindi ito nagsasama ng mga indibidwal na pangalan ng donor. Para sa mga detalye sa antas ng donor, gamitin ang [Batches](batches.md) page.
:::

## Pagtingin ng mga Detalye sa Antas ng Donor

Para sa breakdown kung sino ang nagbigay, magkano, at sa aling fund:

1. Mag-navigate sa **Donations > Batches**.
2. I-click ang **pangalan ng batch** para buksan ito.
3. Ang batch detail page ay naglilista ng bawat donasyon kasama ang pangalan ng donor, halaga, fund, petsa, at paraan ng pagbabayad.
4. I-click ang **pangalan ng donor** para makita ang breakdown kung ilang beses silang nagbigay at magkano sa bawat pagkakataon.
5. I-click ang **donation ID** para buksan ang side panel na may buong detalye para sa indibidwal na donasyong iyon.
6. I-click ang **Download** para mag-export ng CSV na may lahat ng impormasyon ng donor at donasyon para sa batch na iyon.

## Donation Summary Report

Kasama rin sa B1 Admin ang isang **Donation Summary** report sa Reports section:

1. I-click ang **Reports** sa sidebar.
2. Piliin ang **Donation Summary** report.
3. Piliin ang iyong mga filter (date range, fund, campus, atbp.) at i-run ang report.

## Pag-export ng Data

Maaari kang mag-export ng donation data mula sa maraming lugar:

- **Summary page** -- mag-download ng CSV ng weekly giving total ayon sa fund
- **Batch detail page** -- mag-download ng CSV ng mga indibidwal na donasyon kasama ang mga detalye ng donor
- **Fund detail page** -- mag-download ng donation history para sa isang partikular na fund

:::tip
Para sa year-end reporting, pagsamahin ang Summary page export sa [Giving Statements](giving-statements.md) tool para makuha ang parehong aggregate trend at indibidwal na donor statement.
:::

## Mga Susunod na Hakbang

- Bumuo ng [Mga Giving Statement](giving-statements.md) para sa iyong mga donor sa year-end
- Suriin ang mga indibidwal na [batch](batches.md) para i-verify ang mga detalye ng donasyon
- Tingnan ang mga [fund](funds.md) detail page para sa mga giving breakdown ayon sa kategorya
