# Mga Batch ng Donasyon

<div class="article-intro">

Ang mga batch ay nagpapangkat ng iyong mga donasyon para sa mas madaling pag-track at pag-reconcile. Ang isang karaniwang batch ay kumakatawan sa isang koleksyon, tulad ng Sunday offering o special event. Ang paggamit ng mga batch ay tumutulong sa iyo na manatiling organisado at pinapadali ang pag-verify na tumutugma ang iyong mga record sa aktwal na mga deposito.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Siguraduhing [na-setup mo na ang iyong mga fund](funds.md) para available ang mga ito kapag nagre-record ng mga donasyon
- Kakailanganin mo ang access sa **Donations** section sa B1 Admin

</div>

## Ang Batches Page

Kapag nag-navigate ka sa **Donations > Batches**, makikita mo ang listahan ng lahat ng iyong mga batch. Ang bawat row ay nagpapakita ng:

- **Name** -- ang label na ibinigay mo sa batch
- **Date** -- ang petsa ng koleksyon
- **Donations** -- ang bilang ng mga indibidwal na donasyon sa batch
- **Total** -- ang kabuuang halaga sa dolyar

Ang header sa itaas ay nagpapakita ng summary statistics kabilang ang kabuuang bilang ng mga batch, kabuuang bilang ng mga donasyon sa lahat ng batch, at kabuuang halaga sa dolyar.

## Paggawa ng Bagong Batch

1. I-click ang **Add Batch** sa itaas ng page.
2. Maglagay ng deskriptibong pangalan (hal., "Sunday Offering - Feb 9").
3. Piliin ang petsa ng koleksyon.
4. I-click ang **Save**.

Ang iyong bagong batch ay lalabas sa listahan, handa na para magdagdag ng mga donasyon.

## Pagtatrabaho sa mga Batch

- **Tingnan ang mga donasyon** -- i-click ang pangalan ng batch para buksan ito at makita ang lahat ng indibidwal na donasyon na nilalaman nito. Mula doon maaari kang magdagdag, mag-edit, o mag-alis ng mga donasyon.
- **I-edit ang mga detalye ng batch** -- i-click ang **Edit** button sa batch row para baguhin ang pangalan o petsa nito.
- **I-sort** -- gamitin ang mga column header para i-sort ang mga batch ayon sa pangalan o petsa.
- **I-export** -- i-click ang **Export to CSV** para i-download ang iyong batch list bilang spreadsheet.

:::tip
Pangalanan ang iyong mga batch nang consistent para madaling mahanap sa ibang pagkakataon. Ang pagsasama ng petsa at uri ng koleksyon (hal., "Sunday AM - 2025-02-09") ay nagpapanatiling organisado ng iyong listahan habang lumalaki ito.
:::

## Mga Susunod na Hakbang

Kapag mayroon ka nang batch, tingnan ang [Pag-record ng mga Donasyon](recording-donations.md) para malaman kung paano magdagdag ng mga indibidwal na donasyon dito. Maaari mo ring [mag-import ng mga Stripe transaction](stripe-import.md) para awtomatikong gumawa ng mga batch mula sa online giving.
