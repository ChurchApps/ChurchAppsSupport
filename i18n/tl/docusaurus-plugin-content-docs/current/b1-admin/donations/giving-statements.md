# Mga Giving Statement

<div class="article-intro">

Sa pagtatapos ng bawat taon, ang iyong mga donor ay nangangailangan ng buod ng kanilang tax-deductible na pag-aalay para sa kanilang mga record. Pinapadali ng B1 Admin ang pagbuo ng mga statement na ito para sa lahat ng donor nang sabay-sabay, na nakatitipid ng mga oras ng manu-manong trabaho.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- I-verify na ang iyong mga [fund](funds.md) ay tama ang marka bilang **Tax Deductible** -- ang mga donasyon lamang sa tax-deductible na fund ang lalabas sa mga statement
- Siguraduhing ang lahat ng donasyon ay [nai-record na](recording-donations.md) at ang anumang online transaction ay [na-import na mula sa Stripe](stripe-import.md)

</div>

## Pag-access ng mga Giving Statement

1. Sa **B1 Admin**, i-click ang **Donations** sa sidebar.
2. I-click ang **Statements**.

## Pagbuo ng mga Statement

1. Piliin ang **taon** mula sa dropdown sa itaas ng page. Maaari mong piliin ang kasalukuyang taon o alinman sa limang nakaraang taon.
2. Ang page ay nagpapakita ng summary statistics para sa taong iyon, kabilang ang:
   - **Kabuuang mga donor** -- ang bilang ng mga taong nagbigay
   - **Kabuuang mga donasyon** -- ang bilang ng mga indibidwal na donation record
   - **Kabuuang halaga** -- ang pinagsamang halaga sa dolyar ng lahat ng pag-aalay

## Pag-download ng mga Statement

Mayroon kang dalawang opsyon para maipaabot ang mga statement sa iyong mga donor:

### I-download bilang mga CSV File

I-click ang **Download ZIP** para mag-download ng ZIP file na naglalaman ng indibidwal na CSV file para sa bawat donor. Kapaki-pakinabang ito kung gusto mong i-email ang mga statement nang isa-isa o i-import ang mga ito sa ibang sistema.

### I-print ang Lahat ng Statement

I-click ang **Print All** para buksan ang printable view ng statement ng bawat donor sa iyong browser. Mula doon, gamitin ang print function ng iyong browser para ipadala ang mga ito sa printer. Ang bawat statement ay nagsisimula sa bagong page para handa nang itiklop at ipadala sa koreo.

:::tip
I-run ang iyong mga statement nang maaga sa Enero habang sariwa pa ang iyong mga record. I-double check na tama ang marka ng iyong mga fund bilang tax-deductible bago bumuo ng mga statement -- ang mga donasyon lamang sa tax-deductible na fund ang kasama.
:::

:::info
Ang mga giving statement ay nagsasama lamang ng mga donasyon na naka-assign sa mga fund na may naka-enable na **Tax Deductible** setting. Kung ang isang fund ay hindi namarkahan bilang tax-deductible, ang mga donasyon nito ay hindi lalabas sa statement. Maaari mong pamahalaan ang setting na ito sa [Funds](funds.md) page.
:::

## Mga Susunod na Hakbang

Kung kailangan mong suriin ang mga detalye ng donasyon bago bumuo ng mga statement, bisitahin ang [Mga Report ng Donasyon](donation-reports.md) page o suriin ang mga indibidwal na [batch](batches.md).
