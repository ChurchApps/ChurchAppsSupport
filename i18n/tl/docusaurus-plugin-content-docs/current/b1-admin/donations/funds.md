# Pamamahala ng mga Fund

<div class="article-intro">

Ang mga fund ay nagpapahintulot sa iyo na i-kategorya ang mga donasyon ayon sa layunin para masubaybayan mo kung magkano ang natatanggap ng iyong simbahan para sa bawat lugar ng ministeryo. Ang mga karaniwang fund ay General Fund, Building Fund, Missions, at Youth Ministry, pero maaari kang gumawa ng ilan man ang kailangan mo.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kakailanganin mo ang access sa **Donations** section sa B1 Admin
- Magpasya kung aling mga kategorya ng pag-aalay ang kailangan ng iyong simbahan at kung alin ang dapat na tax-deductible

</div>

## Ang Funds Page

Mag-navigate sa **Donations > Funds** para makita ang listahan ng lahat ng iyong mga fund. Ang bawat row ay nagpapakita ng pangalan ng fund at ang **tax-deductible** status nito, na tumutukoy kung ang mga donasyon sa fund na iyon ay lalabas sa year-end [giving statement](giving-statements.md).

## Paggawa ng Bagong Fund

1. I-click ang **Add Fund** sa itaas ng page.
2. Maglagay ng pangalan para sa fund (hal., "Building Fund").
3. Lagyan ng check ang **Tax Deductible** box kung ang mga donasyon sa fund na ito ay dapat isama sa mga giving statement.
4. I-click ang **Save**.

Ang bagong fund ay agad na lalabas sa iyong listahan at available na kapag naglalagay ng mga donasyon.

## Pag-edit at Pag-delete ng mga Fund

- Para mag-edit ng fund, i-click ang **Edit** button sa tabi nito. Maaari mong baguhin ang pangalan o i-toggle ang tax-deductible setting.
- Para mag-delete ng fund, buksan ang edit form at i-click ang **Delete**.

:::warning
Ang pag-delete ng fund ay hindi nag-aalis ng mga donasyon na dati nang naka-assign dito. Ang mga donation record na iyon ay nananatili sa sistema, pero ang fund ay hindi na magiging available para sa mga bagong donasyon.
:::

## Pagtingin ng mga Detalye ng Fund

I-click ang **pangalan ng fund** para buksan ang detail page nito. Ang page na ito ay nagpapakita ng:

- **Donation history** -- listahan ng lahat ng donasyon sa fund na ito
- **Date range filter** -- i-narrow ang mga resulta sa isang partikular na panahon
- **Summary statistics** -- kabuuang donasyon, natatanging mga donor, at kabuuang halaga sa dolyar

Ang view na ito ay kapaki-pakinabang kapag kailangan mong mag-report sa isang partikular na giving initiative o suriin ang progress tungo sa isang fundraising goal.

## Pag-sort at Pag-export

- Gamitin ang mga column header para i-sort ang mga fund ayon sa pangalan.
- I-click ang **Export to CSV** para i-download ang iyong fund list bilang spreadsheet.

:::tip
I-setup ang iyong mga fund bago ka magsimulang [mag-record ng mga donasyon](recording-donations.md). Ang pagkakaroon ng lahat ng fund na handa na ay nagsisiguro na ang mga donasyon ay nakakategorya nang tama mula sa simula.
:::

## Mga Susunod na Hakbang

Kapag naka-setup na ang iyong mga fund, maaari mo nang i-assign ang mga ito sa mga donasyon kapag [nagre-record ng mga donasyon](recording-donations.md) sa isang batch. Siguraduhing namarkahan ang mga naaangkop na fund bilang tax-deductible bago bumuo ng [mga giving statement](giving-statements.md).
