# Pagtatala ng Attendance

<div class="article-intro">

Kapag naka-set up na ang inyong mga campus, oras ng serbisyo, at mga grupo, maaari nang manu-manong magtala ng attendance pagkatapos ng bawat pagtitipon. Inoorganisa ng B1 Admin ang attendance sa mga **sesyon** -- isang sesyon bawat grupo bawat petsa ng pagpupulong. Ginagawa ninyo ang sesyon, minamarkahan kung sino ang dumalo, at direktang pumupunta ang datos sa inyong mga ulat ng attendance.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Kailangang na-configure na ang inyong mga campus, oras ng serbisyo, at mga grupo. Tingnan ang [Pag-set Up ng Attendance](setup.md) kung hindi ninyo pa ito nagagawa.
- Ang mga grupong gusto ninyong subaybayan ay kailangang may naka-enable na **Track Attendance**. Tingnan ang [Pag-set Up ng Attendance](setup.md) para sa mga detalye.

</div>

## Paglikha ng Sesyon

Ang sesyon ay kumakatawan sa isang pagkakataon ng pagpupulong ng grupo -- halimbawa, ang inyong klase ng K--3rd grade sa isang partikular na Linggo.

1. Buksan ang **B1 Admin** at i-click ang **Groups** sa sidebar.
2. Piliin ang grupong gusto ninyong italang ang attendance.
3. I-click ang **Sessions** tab.
4. I-click ang **New** para gumawa ng bagong sesyon.
5. Kung ang grupo ay naka-assign sa isang oras ng serbisyo, pumili ng **Service Time**. Kung ito ay hindi naka-schedule na grupo, hindi lalabas ang field na ito.
6. Piliin ang **Session Date** -- maaari itong ngayon, nakaraang petsa, o hinaharap na petsa.
7. I-click ang **Save**.

:::tip
Maaari kayong gumawa ng mga sesyon para sa mga nakaraang petsa para makaabot sa attendance na hindi pa ninyo naitala, o gawin ang mga ito nang maaga para maging handa kapag nagpulong ang inyong grupo.
:::

## Pag-mark ng Attendance

Pagkatapos i-save ang sesyon, lalabas ang mga miyembro ng grupo sa kanang bahagi ng pahina.

1. I-check ang kahon sa tabi ng bawat taong dumalo.
2. Awtomatikong naise-save ang mga pagbabago -- walang karagdagang Save button para sa mga marka ng attendance.

:::info
Tanging mga kasalukuyang miyembro ng grupo lang ang lalabas sa listahan ng sesyon. Kung may dumalo na hindi pa miyembro ng grupo, [idagdag muna sila sa grupo](../groups/group-members.md), pagkatapos ay itala ang kanilang attendance.
:::

## Pag-export ng Attendance sa Spreadsheet

Maaari ninyong i-download ang isang tala ng sesyon bilang CSV file para gamitin sa Excel, Numbers, o Google Sheets.

1. Buksan ang sesyon na gusto ninyong i-export.
2. I-click ang **Download CSV** button.
3. Buksan ang na-download na file sa inyong spreadsheet application.

## Pagtingin sa Naitalang Attendance

Pagkatapos magtala ng mga sesyon, lalabas ang datos sa inyong mga ulat ng attendance.

- **Attendance tab** -- nagpapakita ng mga trend sa buong simbahan sa paglipas ng panahon. Tingnan ang [Pagsubaybay ng Attendance](tracking-attendance.md).
- **Groups tab** -- nagpapakita ng attendance na hinati ayon sa indibidwal na grupo.

:::tip
Kung ang sesyon na kaka-create lang ninyo ay hindi agad lumabas sa mga ulat, siguraduhing ang petsa ng sesyon ay nasa loob ng saklaw ng petsa na napili sa mga filter ng ulat.
:::
