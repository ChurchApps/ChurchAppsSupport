# Pagsubaybay ng Attendance

<div class="article-intro">

Kapag na-configure na ang iyong mga campus, oras ng serbisyo, at mga grupo, pinapadali ng B1 Admin ang pagsusuri ng datos ng attendance at pagtukoy ng mga trend. Ang pahina ng Attendance ay nagbibigay ng dalawang view ng ulat -- ang **Attendance** tab para sa mga trend ng buong simbahan at ang **Mga Grupo** tab para sa detalye sa antas ng grupo. Gamitin ang mga tool na ito para maunawaan ang mga pattern ng paglago, matukoy ang bumababang pakikilahok, at gumawa ng mga desisyon na batay sa datos para sa iyong simbahan.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Ang iyong attendance structure ay dapat na-set up na may kahit isang campus at oras ng serbisyo. Tingnan ang [Attendance Setup](setup.md) kung hindi mo pa ito nagawa.
- Kailangang may naitatalang datos ng attendance bago magpakita ng mga resulta ang mga ulat. Ang datos ay maaaring manggaling sa manu-manong pag-input o [self check-in](check-in.md).

</div>

## Pagtingin ng Mga Trend ng Attendance

1. Buksan ang **B1 Admin** at i-click ang **Attendance** sa sidebar.
2. Piliin ang **Attendance** tab.
3. Awtomatikong tumatakbo ang ulat kapag binuksan ang tab, na nagpapakita ng attendance sa isang default na saklaw ng petsa.

## Pag-filter ng Iyong Datos

Gamitin ang mga filter sa itaas ng pahina para paliitin ang mga resulta:

- **Saklaw ng Petsa** -- pumili ng simula at katapusang petsa para tumutok sa isang partikular na panahon.
- **Campus** -- pumili ng campus para makita ang attendance para sa lokasyong iyon lamang.
- **Oras ng Serbisyo** -- pumili ng oras ng serbisyo para suriin nang malalim ang isang partikular na pagtitipon.

Ang tsart at datos ay nag-a-update agad kapag nagpalit ka ng filter, kaya mabilis mong maikukumpara ang iba't ibang panahon o lokasyon.

:::info
Awtomatikong tumatakbo ang mga ulat sa tuwing bubuksan mo ang Attendance tab, kaya palaging makikita mo ang pinakabagong mga numero nang hindi kailangang mag-click ng refresh button.
:::

## Attendance ng Grupo

Ang **Mga Grupo** tab ay nagpapakita ng attendance na hinati ayon sa indibidwal na grupo. Ito ay kapaki-pakinabang kapag gusto mong subaybayan ang isang partikular na klase, ministry team, o maliit na grupo sa halip na tingnan ang pangkalahatang mga numero ng serbisyo.

1. Piliin ang **Mga Grupo** tab.
2. Pumili ng grupo mula sa listahan para makita ang kasaysayan ng attendance nito.
3. Gamitin ang filter ng saklaw ng petsa para i-adjust ang window ng pag-uulat.

:::tip
Ang attendance ng grupo ay partikular na mahalaga para sa mga lider ng [mga maliit na grupo](../groups/creating-groups.md) na gustong subaybayan ang pakikilahok sa kanilang grupo sa paglipas ng panahon.
:::

## Mga Tip para sa Paggamit ng Datos ng Attendance

- Suriin ang mga trend buwan-buwan para maaga matukoy ang mga seasonal na pattern.
- Ikumpara ang datos sa antas ng campus para maunawaan kung aling mga lokasyon ang lumalaki.
- Gamitin ang mga ulat sa antas ng grupo para mag-follow up sa mga [grupo](../groups/group-members.md) na nagpapakita ng bumababang attendance.
- Pagsamahin ang mga insight sa attendance sa [AI Search](../people/ai-search.md) tool para mahanap ang mga taong hindi na nakapunta kamakailan.
