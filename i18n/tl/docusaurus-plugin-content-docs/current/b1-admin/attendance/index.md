# Attendance

<div class="article-intro">

Ang seksyon ng Attendance sa B1 Admin ay nagbibigay sa iyo ng kumpletong pagtingin sa kung sino ang pumupunta sa iyong simbahan at kung paano lumalaki ang iyong mga grupo sa paglipas ng panahon. Mula sa pag-configure ng mga campus at oras ng serbisyo hanggang sa pagsusuri ng mga trend at pag-set up ng self check-in, lahat ng kailangan mo para subaybayan at maunawaan ang attendance ay pinamamahalaan mula sa pahinang ito.

</div>

## Pangkalahatang-tanaw ng Pahina

Kapag binuksan mo ang pahina ng Attendance, makikita mo ang isang header na may mga pangunahing istatistika tungkol sa setup ng attendance ng iyong simbahan:

- **Mga Campus** -- ang bilang ng mga pisikal na lokasyon na na-configure mo
- **Mga Oras ng Serbisyo** -- ang kabuuang bilang ng mga serbisyo sa lahat ng campus
- **Mga Naka-iskedyul na Grupo** -- mga grupo na nakatalaga sa isang partikular na oras ng serbisyo
- **Mga Hindi Naka-iskedyul na Grupo** -- mga grupo na sumusubaybay ng attendance nang independyente sa oras ng serbisyo

Sa ibaba ng header, tatlong tab ang nag-oorganisa ng iyong mga tool sa attendance.

## Mga Tab

### Setup

Ang **Setup** tab ay kung saan mo kino-configure ang mga pangunahing bahagi ng pagsubaybay ng attendance. Dito mo itinatakda ang iyong mga campus, nagdaragdag ng mga oras ng serbisyo sa bawat campus, at nagtatalaga ng mga [grupo](../groups/creating-groups.md) sa mga oras ng serbisyo na iyon. Kailangan mo lang gawin ito nang isang beses, bagama't maaari mo itong i-update kapag nagbago ang iyong iskedyul. Tingnan ang [Attendance Setup](setup.md) para sa kumpletong gabay.

### Attendance

Ang **Attendance** tab ay nagpapakita ng mga trend ng attendance sa paglipas ng panahon. Gamitin ang mga filter para paliitin ang mga resulta ayon sa saklaw ng petsa, campus, o oras ng serbisyo at mabilis na matukoy ang mga pattern sa iyong data. Tingnan ang [Pagsubaybay ng Attendance](tracking-attendance.md) para sa mga detalye sa paggamit ng mga ulat at filter.

### Mga Grupo

Ang **Mga Grupo** tab ay hinahati ang attendance ayon sa indibidwal na grupo. Ito ay kapaki-pakinabang kapag gusto mong makita kung kamusta ang isang partikular na maliit na grupo, klase, o ministry team sa halip na tingnan ang mga numero ng buong simbahan.

## Mga Susunod na Hakbang

- [Attendance Setup](setup.md) -- i-configure ang mga campus, oras ng serbisyo, at mga grupo
- [Pagsubaybay ng Attendance](tracking-attendance.md) -- tingnan ang mga trend at i-filter ang mga ulat
- [Check-In](check-in.md) -- i-set up ang self check-in para sa mga serbisyo

:::tip
Kung sine-set up mo ang attendance sa unang pagkakataon, magsimula sa gabay na [Attendance Setup](setup.md) para itakda ang iyong mga campus at oras ng serbisyo. Kapag nailagay na ito, maaari ka nang magsimulang sumubaybay at suriin ang datos ng attendance kaagad.
:::
