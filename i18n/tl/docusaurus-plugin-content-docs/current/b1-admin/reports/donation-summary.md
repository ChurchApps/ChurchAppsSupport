# Buod ng Donasyon

<div class="article-intro">

Ang ulat ng Buod ng Donasyon ay nagbibigay sa inyo ng malinaw na larawan ng aktibidad ng pagbibigay ng inyong simbahan. Gamitin ito para subaybayan ang mga kabuuan ng donasyon, bantayan ang mga trend ng pagbibigay, at maghanda ng mga ulat sa pananalapi para sa inyong koponan ng pamunuan.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Siguraduhing ang mga donasyon ay patuloy na [naitatala](../donations/recording-donations.md) sa inyong sistema
- Kailangan mo ng naaangkop na [mga pahintulot](../settings/roles-permissions.md) para ma-access ang mga ulat sa pananalapi

</div>

## Pagpapatakbo ng Ulat

1. Pumunta sa **Mga Ulat** mula sa pangunahing menu.
2. I-click ang **Buod ng Donasyon** card.
3. Itakda ang **saklaw ng petsa** para tukuyin ang panahon ng pag-uulat.
4. Mag-apply ng mga karagdagang filter tulad ng **fund** o iba pang pamantayan para paliitin ang mga resulta.
5. Ipapakita ng ulat ang buod ng pagbibigay para sa napiling panahon.

## Ano ang Ipinapakita ng Ulat

Kasama sa ulat ng Buod ng Donasyon ang:

- **Kabuuang donasyon** -- Ang kabuuang halaga ng pagbibigay para sa napiling panahon.
- **Bilang ng donor** -- Ang bilang ng mga natatanging donor na nag-ambag.
- **Mga halaga ayon sa fund** -- Isang breakdown ng pagbibigay sa iba't ibang fund, kung naaangkop.

## Mga Praktikal na Gamit

- **Pagpaplano sa pananalapi** -- Suriin ang mga trend ng pagbibigay para hulaan ang hinaharap na kita at planuhin ang inyong badyet.
- **Mga ulat sa board** -- Gumawa ng mga buod para sa mga regular na pulong ng board o pamunuan na may malinaw na data ng pagbibigay.
- **Mga pagsusuri sa katapusan ng taon** -- Patakbuhin ang ulat para sa buong taon para maghanda ng mga taunang buod sa pananalapi.
- **Pagsubaybay ng kampanya** -- Mag-filter ayon sa fund para subaybayan ang pag-unlad ng mga espesyal na kampanya ng pagbibigay o mga proyektong pagtatayo.
- **Seasonal na pagsusuri** -- Ihambing ang pagbibigay sa iba't ibang panahon para maunawaan ang mga seasonal na pattern.

:::tip
Patakbuhin ang ulat na ito buwanan para manatiling may kaalaman tungkol sa kalusugan sa pananalapi ng inyong simbahan. Ang paghahambing ng data buwan-buwan ay tumutulong sa inyo na matukoy ang mga trend nang maaga.
:::

:::warning
Ang data ng donasyon ay nagmumula sa mga nairekord na kontribusyon sa inyong sistema. Siguraduhing patuloy na nailalagay ang mga donasyon -- maging sa pamamagitan ng online giving, pagproseso ng tseke, o mano-manong paglagay -- para sa pinakatamang mga ulat. Tingnan ang [Pagtatala ng mga Donasyon](../donations/recording-donations.md) para sa mga detalye.
:::
