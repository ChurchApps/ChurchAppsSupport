# Mga Ulat ng Attendance

<div class="article-intro">

Ang B1 Admin ay nagbibigay ng tatlong ulat ng attendance upang tulungan kang maunawaan kung paano nakikibahagi ang mga tao sa iyong mga serbisyo at grupo. Ang bawat ulat ay nag-aalok ng iba't ibang pananaw sa iyong datos ng attendance, mula sa mataas na antas na mga trend hanggang sa pang-araw-araw na breakdown.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Tiyaking ang attendance ay [sinusubaybayan nang tuluy-tuloy](../attendance/tracking-attendance.md) para sa iyong mga serbisyo at grupo
- Tiyaking ang iyong mga [grupo](../groups/creating-groups.md) at mga serbisyo ay na-configure sa B1 Admin
- Kailangan mo ng angkop na [mga pahintulot](../settings/roles-permissions.md) upang ma-access ang mga ulat

</div>

## Trend ng Attendance

Ang ulat na Attendance Trend ay nagpapakita kung paano nagbabago ang attendance sa paglipas ng panahon para sa iyong mga serbisyo.

1. Mag-navigate sa **Reports** at i-click ang card na **Attendance Trend**.
2. I-set ang **date range** upang tukuyin ang panahon.
3. Pumili ng **serbisyo** o **grupo** upang i-filter ang mga resulta.
4. Ang ulat ay magpapakita ng trend line na nagpapakita ng bilang ng attendance sa napiling panahon.

Ang ulat na ito ay kapaki-pakinabang para sa paghanap ng mga pattern tulad ng seasonal na pagbaba, mga trend ng paglago, o epekto ng mga espesyal na event.

## Attendance ng Grupo

Ang ulat na Group Attendance ay nagpapakita ng kabuuang attendance ayon sa grupo para sa napiling date range.

1. Mag-navigate sa **Reports** at i-click ang card na **Group Attendance**.
2. I-set ang **date range** para sa ulat.
3. Piliin ang **(mga) grupo** na gusto mong suriin.
4. Ang ulat ay magpapakita ng kabuuang bilang ng attendance para sa bawat grupo.

Gamitin ang ulat na ito upang ikumpara ang attendance sa iba't ibang grupo at tukuyin kung alin ang lumalaki o nangangailangan ng atensyon.

## Pang-araw-araw na Attendance ng Grupo

Ang ulat na Daily Group Attendance ay nagbibigay ng araw-araw na breakdown ng datos ng attendance para sa iyong mga grupo.

1. Mag-navigate sa **Reports** at i-click ang card na **Daily Group Attendance**.
2. I-set ang **date range** para sa ulat.
3. Piliin ang **(mga) grupo** na gusto mong suriin.
4. Ang ulat ay nagpapakita ng bilang ng attendance para sa bawat indibidwal na araw sa loob ng range.

Ang ulat na ito ay nagbibigay ng granular na detalye, na kapaki-pakinabang para sa pag-unawa ng week-to-week na pagkakaiba-iba o pagtukoy ng mga partikular na araw na may hindi karaniwang mataas o mababang attendance.

:::tip
Gamitin ang ulat na Attendance Trend para sa mataas na antas na pangkalahatang-tanaw at ang ulat na Daily Group Attendance kapag kailangan mong suriin ang mga partikular na petsa.
:::

## Mga Praktikal na Gamit

- **Pagpaplano** -- Gamitin ang mga trend ng attendance upang magplano ng upuan, staffing, at mga resource para sa mga paparating na serbisyo.
- **Outreach** -- Agad na tukuyin ang mga bumababang pattern ng attendance upang makapag-follow up sa mga miyembro.
- **Mga ulat para sa board** -- Isama ang datos ng attendance sa iyong mga regular na ulat sa leadership upang ipakita ang kalusugan ng ministeryo.
- **Pagsusuri ng event** -- Ikumpara ang attendance bago at pagkatapos ng mga espesyal na event upang sukatin ang kanilang epekto.

:::warning
Ang datos ng attendance ay nire-record sa pamamagitan ng iyong mga proseso ng check-in sa grupo at serbisyo. Kung ang attendance ay hindi sinusubaybayan nang tuluy-tuloy, ang iyong mga ulat ay hindi tumpak na magpapakita ng aktwal na pakikilahok. Tingnan ang [Pagsubaybay ng Attendance](../attendance/tracking-attendance.md) para sa mga tagubilin sa setup.
:::
