# Mga Ulat

<div class="article-intro">

Ang pahina ng mga Ulat ay nagbibigay sa iyo ng mabilis na access sa mga pre-built na ulat na tumutulong sa iyo na maunawaan ang data ng iyong simbahan. Gamitin ang mga ulat na ito upang subaybayan ang mga trend ng pagdalo, bantayan ang aktibidad ng donasyon, at manatiling konektado sa iyong kongregasyon sa pamamagitan ng mga pagdiriwang ng kaarawan.

</div>

## Mga Available na Ulat

Ang pahina ng mga Ulat ay nagpapakita ng limang report card. I-click ang anumang card upang buksan ang ulat kasama ang mga filter nito.

- **[Ulat ng Kaarawan](./birthday-report)** -- Tingnan ang mga paparating na kaarawan ng mga miyembro ng iyong kongregasyon.
- **[Trend ng Pagdalo](./attendance-reports)** -- Tingnan kung paano nagbabago ang mga pattern ng pagdalo sa paglipas ng panahon sa iyong mga serbisyo.
- **[Pagdalo sa Grupo](./attendance-reports)** -- Tingnan ang mga kabuuang pagdalo ayon sa grupo para sa isang napiling saklaw ng petsa.
- **[Araw-araw na Pagdalo sa Grupo](./attendance-reports)** -- Kunin ang araw-araw na breakdown ng data ng pagdalo sa grupo.
- **[Buod ng Donasyon](./donation-summary)** -- Suriin ang mga kabuuang donasyon at mga trend para sa pagpaplano ng pananalapi.

## Paggamit ng mga Ulat

1. Mag-navigate sa **Reports** mula sa pangunahing menu.
2. I-click ang isang **report card** upang buksan ito.
3. Gamitin ang mga **filter** sa itaas ng bawat ulat upang paliitin ang data ayon sa saklaw ng petsa, grupo, serbisyo, o iba pang pamantayan.
4. Suriin ang mga resulta na ipinapakita sa ulat.

:::tip
Regular na magpatakbo ng mga ulat upang manatiling updated sa mga trend ng pagdalo, mga paparating na kaarawan, at mga pattern ng donasyon. Ang data na ito ay tumutulong sa iyo na magplano ng mga ministeryo at outreach nang mas epektibo.
:::

:::info
Ang data ng ulat ay nakadepende sa pare-parehong pag-input ng data. Tiyaking ang iyong team ay [nagta-track ng pagdalo](../attendance/tracking-attendance.md), [nagre-record ng mga donasyon](../donations/recording-donations.md), at pinapanatiling updated ang [mga rekord ng tao](../people/adding-people.md) para sa pinakatumpak na resulta.
:::
