# Mga File

<div class="article-intro">

Ang pahina ng Mga File ay ang sentral na repositoryo para sa lahat ng media at dokumento na ginagamit sa website ng inyong simbahan. Mula rito maaari kayong mag-upload, mag-ayos, mag-browse, at mamahala ng bawat larawan, dokumento, at media file na kailangan ng inyong site.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Kumpletuhin ang [Paunang Setup](initial-setup) para sa inyong website
- I-optimize ang mga larawan bago mag-upload para sa mas mabilis na oras ng pag-load ng pahina
- Ihanda ang inyong mga file sa mga karaniwang web format (PNG, JPG, PDF, atbp.)

</div>

## Pag-upload ng mga File

1. Pumunta sa seksyon ng **Website** sa B1 Admin at i-click ang **Files** tab.
2. I-drag and drop ang mga file nang direkta sa upload area, o i-click ang upload button para mag-browse sa inyong computer.
3. Ang inyong mga file ay mag-a-upload at agad na lalabas sa file library.

:::tip
Para sa pinakamahusay na resulta, i-optimize ang mga larawan bago mag-upload. Ang mas maliliit na laki ng file ay tumutulong na mas mabilis mag-load ang inyong website para sa mga bisita. Ang mga tool tulad ng TinyPNG o Squoosh ay maaaring mag-compress ng mga larawan nang walang kapansin-pansing pagkawala ng kalidad.
:::

## Pag-aayos gamit ang mga Folder

1. Gumawa ng mga folder para pagsamahin ang mga nauugnay na file -- halimbawa, "Sermon Graphics," "Event Banners," o "Staff Photos."
2. Ilipat ang mga file sa pagitan ng mga folder para mapanatiling maayos ang inyong library habang lumalaki ito.
3. Gumamit ng malinaw at mapaglarawang mga pangalan ng folder para mabilis na mahanap ng inyong koponan ang mga file.

## Pagba-browse at Paghahanap

- Mag-scroll sa file library para makita ang mga thumbnail ng lahat ng inyong nai-upload na file.
- Gamitin ang **search** function para maghanap ng mga file ayon sa pangalan kapag lumalaki na ang inyong library.
- I-click ang anumang file para tingnan ang mga detalye nito, kabilang ang pangalan ng file, laki, at URL.

## Paggamit ng mga File sa Inyong Website

1. I-click ang isang file para tingnan ang mga detalye nito.
2. **Kopyahin ang file URL** para gamitin ito sa mga pahina ng inyong website. Maaari ninyong i-paste ang URL na ito sa mga image element, download link, o kahit saan na tumatanggap ng file URL.
3. Ang mga file na ina-upload ninyo rito ay available para gamitin sa lahat ng pahina at content block ng inyong website.

:::info
Kapag nagbubuo ng mga pahina sa [Pamamahala ng mga Pahina](managing-pages), maaari ninyong i-reference ang anumang file na na-upload dito sa pamamagitan ng URL nito. Pinapanatili nitong nakaayos ang lahat ng inyong media sa isang sentral na lokasyon.
:::

## Pagbura ng mga File

1. Piliin ang file na gusto ninyong alisin.
2. I-click ang **Delete** button para permanenteng alisin ito mula sa inyong library.
3. Siguraduhing hindi kasalukuyang ginagamit ang file sa anumang pahina bago ito burahin, dahil masisira nito ang anumang mga link o larawan na nagrereference dito.

:::warning
Ang mga nabura na file ay hindi na maaaring ma-recover. I-double-check na hindi na ginagamit ang isang file sa anumang pahina bago ito alisin.
:::
