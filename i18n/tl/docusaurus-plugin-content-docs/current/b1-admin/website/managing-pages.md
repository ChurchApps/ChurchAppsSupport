# Pamamahala ng mga Pahina

<div class="article-intro">

Ang Website Pages view ang inyong sentral na hub para sa paggawa, pag-edit, at pag-aayos ng lahat ng pahina sa website ng inyong simbahan. Maaari ninyong pamahalaan ang parehong content ng inyong pahina at navigation ng inyong site mula sa iisang screen na ito.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Kumpletuhin ang [Paunang Setup](initial-setup) para i-configure ang inyong domain at mga pangunahing setting ng site
- Ihanda ang inyong content at mga larawan. Gamitin ang [Mga File](files) manager para mag-upload ng mga media asset muna.

</div>

## Pag-unawa sa mga Uri ng Pahina

Inilalista ng **Pages** table ang bawat pahina sa inyong site kasama ang status nito:

- **Generated** -- Mga pahinang awtomatikong ginawa ng sistema batay sa data ng inyong simbahan (halimbawa, isang pahina ng Mga Grupo o pahina ng Mga Sermon). Awtomatikong nag-a-update ang mga pahinang ito habang nagbabago ang inyong data.
- **Custom** -- Mga pahinang ginawa ninyo na may sarili ninyong content at layout.

Maaari ninyong gawing custom na pahina ang anumang auto-generated na pahina kung gusto ninyo ng buong kontrol sa content at disenyo nito.

## Pagdagdag at Pag-edit ng mga Pahina

1. I-click ang **Add Page** button sa kanang sulok sa itaas ng Pages table.
2. Pumili ng uri ng pahina (blank o template) at bigyan ito ng pangalan.
3. I-click ang **Edit** sa tabi ng anumang pahina para buksan ang page builder, kung saan maaari kayong magdagdag ng mga section, teksto, larawan, at iba pang elemento.
4. I-click ang **Page Settings** para i-update ang pamagat ng pahina, URL path, at iba pang metadata.
5. Gamitin ang **Preview** button para buksan ang inyong pahina sa bagong window at makita kung paano eksaktong ang hitsura nito sa mga bisita.

:::tip
Para sa inyong home page, itakda ang URL path sa `/` lamang. Para sa lahat ng iba pang pahina, gumamit ng mapaglarawang path tulad ng `/about` o `/contact`.
:::

## Pamamahala ng Navigation

Ipinapakita ng kaliwang sidebar ng Website Pages view ang inyong mga navigation link. Kinokontrol ng mga link na ito ang menu na nakikita ng mga bisita sa inyong website.

1. I-click ang **Add** para gumawa ng bagong navigation link. Maaari ninyo itong ituro sa anumang pahina sa inyong site o sa isang external na URL.
2. Para muling ayusin ang mga link, i-drag and drop ang mga ito sa pagkakasunod-sunod na gusto ninyo. Maaari rin ninyong i-nest ang mga link sa ilalim ng parent item para gumawa ng mga dropdown menu.
3. I-click ang **Edit** icon sa tabi ng anumang link para baguhin ang label, URL, o posisyon nito.
4. Para alisin ang isang link mula sa navigation, i-click ang **Delete** icon.

:::info
Ang pag-alis ng navigation link ay hindi nagbubura ng pahina mismo. Ang pahina ay umiiral pa rin at maaaring ma-access nang direkta sa pamamagitan ng URL nito -- hindi lang ito lalabas sa menu.
:::

## Mga Tip para sa Pag-aayos ng Inyong Site

- Panatilihin ang inyong top-level navigation sa lima o anim na item para mabilis na mahanap ng mga bisita ang mga bagay-bagay.
- Gumamit ng mga nested link para sa mga kaugnay na sub-page (halimbawa, isang "About" dropdown na may "Our Team," "Beliefs," at "History").
- Suriin ang inyong navigation sa mobile sa pamamagitan ng pag-click ng **Mobile Preview** para siguraduhing gumagana itong mabuti sa mas maliliit na screen.
- Bigyan ang mga pahina ng malinaw at mapaglarawang mga pangalan na tumutulong sa mga bisita na maunawaan kung ano ang matatagpuan nila.

:::tip
Maaari kayong magdagdag ng [mga form](../forms/creating-forms.md) sa inyong mga pahina para mangolekta ng mga pagpaparehistro, kahilingan sa panalangin, o iba pang impormasyon mula sa mga bisita.
:::
