# Pangkalahatang-tanaw ng B1 Admin

<div class="article-intro">

Ang B1 Admin ang inyong dashboard sa pamamahala ng simbahan. Nagbibigay ito ng mga tool para pamahalaan ang bawat aspeto ng operasyon ng inyong simbahan -- mula sa mga directory ng miyembro at donasyon hanggang sa pagbuo ng website at live streaming -- lahat mula sa isang lugar.

</div>

## Pagsisimula

1. Buksan ang inyong browser at pumunta sa [admin.b1.church](https://admin.b1.church)
2. Mag-sign in gamit ang inyong username at password
3. Dadalhin kayo sa **Dashboard**, na nagbibigay ng mabilis na access sa lahat ng pangunahing feature

:::tip
Bago pa lang sa B1 Admin? Magsimula sa [Panimula](./introduction) para sa isang video walkthrough, pagkatapos bisitahin ang [Mga Setting](./settings/) para i-configure ang impormasyon ng inyong simbahan at mag-imbita ng inyong koponan.
:::

## Mga Pangunahing Feature

- **[Mga Tao](./people/)** - Pamahalaan ang directory ng inyong simbahan, magdagdag ng mga miyembro, subaybayan ang mga sambahayan
- **[Mga Grupo](./groups/)** - Gumawa at ayusin ang mga grupo, koponan, at ministeryo ng simbahan
- **[Attendance](./attendance/)** - Mag-set up ng mga campus, oras ng serbisyo, at subaybayan ang attendance
- **[Mga Donasyon](./donations/)** - Itala ang pagbibigay, pamahalaan ang mga fund, gumawa ng mga statement
- **[Paglilingkod](./serving/)** - I-coordinate ang mga boluntaryo, gumawa ng mga service plan, pamahalaan ang mga gawain
- **[Mga Form](./forms/)** - Bumuo ng mga custom na form para sa mga pagpaparehistro at pagkolekta ng data
- **[Mga Ulat](./reports/)** - Tingnan ang mga ulat ng kaarawan, attendance, at donasyon
- **[Website](./website/)** - Buuin at pamahalaan ang website ng inyong simbahan
- **[Mga Sermon](./sermons/)** - Pamahalaan ang inyong sermon library at live streaming
- **[Mga Kalendaryo](./calendars/)** - Gumawa at mag-publish ng mga kalendaryo ng kaganapan
- **[Mga Setting](./settings/)** - I-configure ang impormasyon ng simbahan, mga tungkulin, at mga pahintulot

:::info
I-click ang icon ng tandang pananong sa kanang sulok sa itaas ng anumang pahina para sa mabilis na access sa tulong at dokumentasyon.
:::

Para sa suporta, mag-email sa [support@churchapps.org](mailto:support@churchapps.org).
