# B1 Admin Overview

<div class="article-intro">

Ang B1 Admin ay ang dashboard ng pamamahala ng inyong parokya. Ito ay nagbibigay ng mga kagamitan upang pamahalaan ang bawat aspeto ng operasyon ng inyong parokya -- mula sa mga direktoryo ng miyembro at mga donasyon hanggang sa pagbuo ng website at live streaming -- lahat mula sa isang lugar.

</div>

## Magsimula

1. Buksan ang inyong browser at magpunta sa [admin.b1.church](https://admin.b1.church)
2. Mag-sign in gamit ang inyong username at password
3. Kayo ay makakarating sa **Dashboard**, na nagbibigay ng overview ng inyong data ng parokya kasama ang **Quick Actions** para sa mga karaniwang gawain tulad ng pagdagdag ng mga tao, paglikha ng mga grupo, pag-record ng mga donasyon, at marami pang iba

:::tip
Bago pa lang sa B1 Admin? Magsimula sa [Introduction](./introduction) para sa isang video walkthrough, pagkatapos ay bisitahin ang [Settings](./settings/) upang makakonfigure ng inyong impormasyon ng parokya at mag-imbitahan ng inyong koponan.
:::

## Mga Pangunahing Feature

- **[People](./people/)** - Pamahalaan ang inyong direktoryo ng parokya, magdagdag ng mga miyembro, bulk edit data, subaybayan ang mga tahanan
- **[Groups](./groups/)** - Lumikha at i-organize ang mga grupo ng parokya na may join requests at member management
- **[Attendance](./attendance/)** - I-setup ang mga campus, service times, at subaybayan ang attendance
- **[Donations](./donations/)** - Mag-record ng mga pagbibigay sa maraming pera, pamahalaan ang mga pondo, lumikha ng mga statement
- **[Serving](./serving/)** - I-coordinate ang mga volunteer, lumikha ng mga service plans, pamahalaan ang mga gawain
- **[Forms](./forms/)** - Bumuo ng mga customized na form para sa mga pagpaparehistro at pagkolekta ng data
- **[Reports](./reports/)** - Tingnan ang birthday, attendance, at donation reports
- **[Website](./website/)** - Bumuo at pamahalaan ang inyong website ng parokya na may customized na navigation styles
- **[Sermons](./sermons/)** - Pamahalaan ang inyong sermon library at live streaming
- **[Calendars](./calendars/)** - Lumikha ng mga curated na kalendaryo sa pamamagit ng pag-uugnay ng mga kaganapan ng grupo
- **[Settings](./settings/)** - Makakonfigure ng impormasyon ng parokya, mga papel, at mga pahintulot

:::info
I-click ang question mark icon sa tuktok na sulok ng anumang pahina para sa mabilis na access sa tulong at dokumentasyon.
:::
