## Walkthrough

1. Gamit ang Google Chrome, pumunta sa [admin.b1.church](https://admin.b1.church) at mag-sign in gamit ang inyong username at password.
2. Dadalhin kayo nito sa **B1.church Admin dashboard**. Mula rito, maaari ninyong pamahalaan ang database, website, mobile app, mga aralin, at iba pa ng inyong simbahan.
3. I-click ang **Settings** para i-set up ang impormasyon ng inyong simbahan, mga administrative role, at iba pang mga detalye.
4. I-click ang **edit pencil** para i-set up ang impormasyon ng inyong simbahan.
5. Bumalik sa dashboard, i-click ang **icon ng tandang pananong** sa kanang sulok sa itaas para sa tulong at mga tutorial.
6. I-click ang **View Documentation** para makita ang listahan ng mga tutorial na makakatulong sa inyo na magsimula sa B1.church.
7. Bumalik sa dashboard at i-click ang **arrow sa tabi ng inyong profile picture** at piliin ang **Switch App**.
8. Piliin ang **Lessons.church** para pamahalaan ang mga online na aralin at kurikulum ng inyong simbahan.
9. Bumalik sa B1.church sa pamamagitan ng pag-click sa arrow sa tabi ng inyong profile picture at pagpili ng **Switch App** muli.
10. Maaari na ninyong makapasok sa B1.church sa pamamagitan ng **member portal** o **admin portal**. Ang member portal ay nagbibigay-daan sa inyo na makita kung paano ang hitsura ng B1.church sa inyong mga miyembro, tingnan ang inyong mga grupo, at pamahalaan ang inyong mga personal na donasyon.

:::tip
Magsimula sa pahinang [Mga Setting](./settings/) para i-configure ang pangalan ng inyong simbahan, branding, at mga pahintulot ng koponan. Tinitiyak nito na tama ang lalagyan ng lahat ng iba pang i-set up ninyo.
:::

## Mga Susunod na Hakbang

Pagkatapos panoorin ang panimulang ito, tuklasin ang mga pangunahing feature na ito:

- [Pagdagdag ng mga Tao](./people/adding-people) -- Matuto kung paano magdagdag ng mga miyembro sa inyong database
- [Mga Grupo](./groups/) -- Mag-set up at pamahalaan ang mga grupo ng simbahan
- [Mga Plano sa Paglilingkod](./serving/plans) -- Ayusin ang inyong serving ministry
- [Attendance](./attendance/) -- Subaybayan ang attendance at check-in
- [Website](./website/) -- Buuin at i-customize ang website ng inyong simbahan
- [Mga Setting](./settings/) -- I-configure ang impormasyon at mga pahintulot ng inyong simbahan
