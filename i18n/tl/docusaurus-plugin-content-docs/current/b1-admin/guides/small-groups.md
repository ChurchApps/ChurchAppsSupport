# Pamahalaan ang mga Small Group

<div class="article-intro">

Mag-set up ng mga small group, Bible study, o ministry team na may pamamahala ng miyembro, mga kalendaryo ng grupo, at mga pag-uusap. Mag-publish ng church-wide na kalendaryo ng kaganapan sa inyong website para makita ng mga bisita kung ano ang nangyayari at manatiling konektado ang mga miyembro sa pamamagitan ng web portal at mobile app.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- B1 Admin account na may admin access
- [Mga taong naidagdag sa inyong directory](../people/adding-people.md) (kailangan munang umiiral ang mga miyembro ng grupo)

</div>

## Hakbang 1: Gumawa ng mga Kategorya ng Grupo at mga Grupo

Ayusin ang inyong mga grupo sa mga kategorya para madaling mahanap ng mga miyembro ang kanilang hinahanap.

Sundin ang gabay sa [Paggawa ng mga Grupo](../groups/creating-groups.md) para:

1. Gumawa ng mga kategorya para ayusin ang inyong mga grupo (hal., "Small Groups", "Bible Studies", "Ministry Teams", "Committees")
2. Gumawa ng mga grupo sa loob ng bawat kategorya (hal., "Tuesday Night Men's Group", "Women's Bible Study", "Youth Ministry")
3. Magdagdag ng paglalarawan at oras ng pulong para sa bawat grupo

## Hakbang 2: Magdagdag ng mga Miyembro at mga Lider

Punuin ang inyong mga grupo ng mga miyembro at italaga ang mga lider na maaaring mamahala ng kanilang sariling mga koponan.

Sundin ang gabay sa [Mga Miyembro ng Grupo](../groups/group-members.md) para:

1. Maghanap ng mga tao at idagdag sila sa naaangkop na mga grupo
2. Italaga ang mga lider ng grupo gamit ang berdeng icon ng susi — nakakakuha ang mga lider ng karagdagang mga pahintulot

:::tip
Maaaring pamahalaan ng mga lider ng grupo ang kanilang sariling kalendaryo ng grupo, mag-post ng mga pag-uusap, at mag-coordinate ng mga kaganapan nang hindi kailangan ng buong admin access.
:::

## Hakbang 3: I-set Up ang mga Kalendaryo ng Grupo

Bigyan ang bawat grupo ng sariling kalendaryo para sa mga pulong at kaganapan.

Sundin ang gabay sa [Kalendaryo ng Grupo](../groups/group-calendar.md) para:

1. Magdagdag ng mga paulit-ulit na pulong (hal., "Tuwing Martes ng 7 PM") at mga isahang kaganapan (hal., "Group Cookout - Marso 15")
2. Maaaring magdagdag at mamahala ng mga kaganapan ang mga lider ng grupo para sa kanilang sariling grupo

## Hakbang 4: Gumawa ng Church-Wide na Kalendaryo

I-aggregate ang mga kaganapan mula sa lahat ng inyong mga grupo sa isang pinag-isang kalendaryo para sa buong simbahan.

Sundin ang mga gabay sa [Paggawa ng mga Kalendaryo](../calendars/creating-calendars.md) at [Curated na Kalendaryo](../calendars/curated-calendar.md) para:

1. Gumawa ng isang curated na kalendaryo na nagku-pull ng mga kaganapan mula sa maraming grupo sa isang pinag-isang view
2. Ibahagi ang ICS subscription link para maidagdag ng mga miyembro ang mga kaganapan sa kanilang personal na calendar app (Google Calendar, Apple Calendar, Outlook)

:::info
Awtomatikong nag-a-aggregate ng mga kaganapan ang isang curated na kalendaryo mula sa mga grupong ikinonekta mo. Kapag nagdagdag ng kaganapan ang isang lider ng grupo, lalabas din ito sa kalendaryo ng simbahan.
:::

## Hakbang 5: I-publish sa Inyong Website

Gawing nakikita ang inyong mga grupo at kalendaryo sa mga bisita at miyembro sa inyong B1 website.

Sundin ang gabay sa [Pamamahala ng mga Pahina](../website/managing-pages.md) para:

1. Magdagdag ng seksyon ng Kalendaryo sa inyong B1 website para ipakita ang curated na kalendaryo
2. Magdagdag ng seksyon ng mga Grupo para ma-browse ng mga bisita ang mga available na grupo at malaman kung paano sumali

## Hakbang 6: Karanasan ng Miyembro

Maaaring tumuklas, sumali, at lumahok ang mga miyembro sa mga grupo mula sa kanilang telepono o computer.

1. Nagba-browse at sumasali ang mga miyembro sa mga grupo sa [B1.church](../../b1-church/groups/browsing-groups.md)
2. Maaaring tingnan ng mga miyembro ng grupo ang mga detalye, kalendaryo, at mag-post sa [mga pag-uusap](../../b1-church/groups/conversations.md)
3. Lahat ng feature ng grupo ay available din sa [B1 Mobile app](../../b1-mobile/groups/)

:::info
Nakikita ng mga miyembro ang kanilang mga grupo, paparating na mga kaganapan, at mga pag-uusap ng grupo sa parehong web at mobile — lahat ay naka-sync.
:::

## Tapos Ka Na!

Na-set up na at nakakonekta na ang inyong mga small group. Maaaring tumuklas ng mga grupo ang mga miyembro, tingnan ang mga kaganapan, at manatiling konektado sa pamamagitan ng mga pag-uusap — lahat sa inyong website at mobile app. Maaaring pamahalaan ng mga lider ng grupo ang kanilang sariling mga koponan nang hiwalay.
