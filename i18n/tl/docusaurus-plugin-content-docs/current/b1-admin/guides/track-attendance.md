# Subaybayan ang Attendance ng Simbahan

<div class="article-intro">

I-set up ang pagsubaybay ng attendance para sa inyong mga serbisyo at grupo — mula sa pag-configure ng mga campus at oras ng serbisyo, hanggang sa pagtatala ng attendance nang mano-mano o sa pamamagitan ng self-service check-in app, hanggang sa pagtingin ng mga trend sa mga ulat. Sa huli, magkakaroon kayo ng kumpletong larawan kung sino ang dumadalo at kailan.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- B1 Admin account na may admin access
- [Mga taong naidagdag sa inyong directory](../people/adding-people.md)
- [Mga grupong nagawa na](../groups/creating-groups.md) para sa mga ministeryong gusto ninyong subaybayan

</div>

## Hakbang 1: I-configure ang mga Campus at Serbisyo

Tukuyin kung saan at kailan nagtitipon ang inyong simbahan.

Sundin ang gabay sa [Pag-setup ng Attendance](../attendance/setup.md) para:

1. Idagdag ang inyong campus (hal., "Main Campus")
2. Idagdag ang mga oras ng serbisyo (hal., "Linggo 9:00 AM", "Linggo 11:00 AM", "Miyerkules 7:00 PM")
3. I-assign ang mga grupo sa bawat oras ng serbisyo para tukuyin kung anong mga klase/ministeryo ang nagtitipon sa serbisyong iyon

## Hakbang 2: I-enable ang Pagsubaybay ng Attendance ng Grupo

I-on ang pagsubaybay ng attendance para sa mga grupong gusto ninyong bantayan.

Sundin ang gabay sa [Paggawa ng mga Grupo](../groups/creating-groups.md) para:

1. I-edit ang bawat grupong gusto ninyong subaybayan at i-on ang "Track Attendance"
2. Iugnay ang mga grupo sa mga oras ng serbisyo para maitala ang attendance laban sa tamang serbisyo

:::info
Hindi lahat ng grupo ay nangangailangan ng pagsubaybay ng attendance. I-enable ito para sa mga grupong gusto ninyong bantayan ang pakikilahok — mga klase ng mga bata, mga small group, Sunday school, atbp.
:::

## Hakbang 3: Mano-manong Itala ang Attendance

I-check off kung sino ang dumalo sa bawat serbisyo at grupo.

Sundin ang gabay sa [Pagsubaybay ng Attendance](../attendance/tracking-attendance.md) para:

1. Pumunta sa Attendance, piliin ang petsa at serbisyo
2. I-check off kung sino ang dumalo sa bawat grupo

:::tip
Mahusay na gumagana ang mano-manong pagsubaybay para sa mga serbisyo ng mga matatanda, mga small group, at anumang sitwasyon kung saan hindi praktikal ang check-in kiosk.
:::

## Hakbang 4: I-set Up ang Self Check-In (Opsyonal)

Hayaan ang mga miyembro na mag-check in ng kanilang sarili gamit ang isang tablet kiosk.

Sundin ang gabay sa [Check-In](../attendance/check-in.md) para:

1. I-install ang B1 Checkin app sa isang tablet para sa self-service check-in
2. Hinahanap ng mga miyembro ang kanilang pangalan at nag-che-check in ng kanilang sarili at kanilang pamilya
3. Ang mga talaan ng check-in attendance ay lumalabas kasama ng mano-manong nailagay na mga talaan — walang pagkakaiba sa data

:::info
Para sa kumpletong setup ng children's ministry kasama ang hardware at printer configuration, tingnan ang [Gabay sa Children's Check-In](./childrens-checkin.md).
:::

## Hakbang 5: Tingnan ang mga Ulat ng Attendance

Tingnan ang malaking larawan ng pakikilahok sa buong simbahan.

Sundin ang gabay sa [Mga Ulat ng Attendance](../reports/attendance-reports.md). May tatlong view ng ulat na available:

1. **Trend ng Attendance** — tingnan ang attendance sa paglipas ng panahon para matukoy ang paglago o pagbaba
2. **Attendance ng Grupo** — ihambing ang mga kabuuan sa mga grupo
3. **Pang-araw-araw na Attendance ng Grupo** — araw-araw na breakdown para sa detalyadong pagsusuri

:::info
Pinagsasama ng mga ulat ang parehong mano-manong naitala at self-check-in na attendance. Walang pagkakaiba sa kung paano lumilitaw ang data — ang parehong pamamaraan ay pumapasok sa parehong mga ulat.
:::

## Tapos Ka Na!

Na-set up na ang inyong pagsubaybay ng attendance. Mano-mano man ninyong itinatala ang attendance, ginagamit ang check-in app, o pareho — lahat ng data ay dumadaloy sa parehong mga ulat kaya mayroon kayong malinaw na larawan ng pakikilahok sa buong simbahan.
