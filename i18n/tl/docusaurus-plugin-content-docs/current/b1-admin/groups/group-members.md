# Mga Miyembro ng Grupo

<div class="article-intro">

Kapag nakalikha ka na ng grupo, ang susunod na hakbang ay ang pagdaragdag ng mga miyembro. Mula sa pahina ng detalye ng grupo, maaari kang maghanap ng mga tao, idagdag sila sa grupo, mag-assign ng mga lider, magpadala ng mga mensahe, at i-export ang listahan ng miyembro. Ang pamamahala ng membership ng grupo ay mahalaga para sa pag-coordinate ng mga ministeryo, maliliit na grupo, at mga team.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kailangan mo ng kahit isang grupo na na-set up sa B1 Admin. Tingnan ang [Paglikha ng Mga Grupo](creating-groups.md) kung hindi ka pa nakakalikha.
- Ang mga tao na gusto mong idagdag ay kailangang umiiral na sa iyong direktoryo ng [People](../people/adding-people.md).

</div>

## Pagdaragdag ng mga Miyembro sa Grupo

1. Mag-navigate sa pahina ng **Groups** at i-click ang grupo na gusto mong pamahalaan.
2. I-click ang tab na **Members**.
3. Sa search box, i-type ang pangalan ng tao na gusto mong idagdag.
4. I-click ang **Add** sa tabi ng pangalan ng tao sa mga resulta ng paghahanap.
5. Ang tao ay lalabas na sa listahan ng miyembro ng grupo.

:::tip
Iwanang blangko ang search box at i-click ang **Search** upang mag-browse sa buong direktoryo. Ito ay kapaki-pakinabang kung hindi ka sigurado sa eksaktong baybay ng pangalan ng isang tao.
:::

## Pag-designate ng mga Lider ng Grupo

Ang mga lider ng grupo ay may mga espesyal na pribilehiyo -- maaari silang mag-edit ng [kalendaryo ng grupo](group-calendar.md), pamahalaan ang mga event, at tumulong sa pag-coordinate ng grupo.

1. Sa listahan ng miyembro ng grupo, hanapin ang tao na gusto mong gawing lider.
2. I-click ang **green key icon** sa tabi ng kanyang pangalan.
3. Ang tao ay itinalaga na bilang lider ng grupo.

Upang alisin ang status ng lider, i-click muli ang green key icon.

:::info
Anumang miyembro ng grupo ay maaaring tingnan ang kalendaryo ng grupo at mga event, ngunit ang mga lider lamang ang maaaring magdagdag o mag-edit ng mga event sa kalendaryo.
:::

## Pagpapadala ng mga Mensahe sa mga Miyembro ng Grupo

Maaari kang makipag-ugnayan sa lahat ng miyembro ng grupo nang direkta mula sa B1 Admin:

1. Mula sa pahina ng detalye ng grupo, hanapin ang lugar para sa mensahe.
2. I-type ang iyong mensahe sa text box.
3. I-click ang **Send**.

Ang iyong mensahe ay ipapadala sa lahat ng miyembro ng grupo.

## Pag-export ng Datos ng Grupo

Upang i-download ang listahan ng miyembro ng grupo bilang file:

1. Mula sa pahina ng detalye ng grupo, i-click ang **download icon**.
2. Isang CSV file na naglalaman ng impormasyon ng miyembro ng grupo ang mada-download sa iyong computer.

Ito ay kapaki-pakinabang para sa paggawa ng mga naka-print na roster, pag-import ng datos sa ibang mga tool, o pagpapanatili ng mga offline record. Para sa higit pang mga opsyon sa pag-export, tingnan ang [Pag-export ng Datos](../people/exporting-data.md).

## Pag-alis ng mga Miyembro

Upang alisin ang isang tao mula sa grupo, hanapin ang kanyang pangalan sa listahan ng miyembro at i-click ang button na **remove** sa tabi ng kanyang entry.

:::info
Ang pag-alis ng tao mula sa grupo ay hindi nagde-delete sa kanya mula sa direktoryo ng iyong simbahan. Siya ay patuloy na lalabas sa seksyon ng [People](../people/adding-people.md) at maaaring muling idagdag sa grupo anumang oras.
:::
