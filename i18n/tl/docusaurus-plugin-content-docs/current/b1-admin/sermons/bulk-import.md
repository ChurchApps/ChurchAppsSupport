# Bulk Import

<div class="article-intro">

Ang pahina ng Bulk Import ay nagbibigay-daan sa inyo na mabilis na mapunan ang inyong sermon library sa pamamagitan ng pag-import ng mga video mula sa YouTube o Vimeo. Ito ang pinakamabilis na paraan para dalhin ang inyong kasalukuyang sermon content sa halip na idagdag ang bawat video nang isa-isa.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Kailangan mo ng **contentApi.streamingServices.edit** na pahintulot. Tingnan ang [Mga Tungkulin at Pahintulot](../settings/roles-permissions.md) kung wala kang access.
- Gumawa ng kahit isang [playlist](playlists) para ma-import ang inyong mga sermon doon
- Ihanda ang inyong YouTube Channel ID o impormasyon ng Vimeo account

</div>

## Pagpili ng Inyong Pinagmulan

1. Sa B1 Admin, i-click ang **Sermons** sa kaliwang sidebar, pagkatapos i-click ang **Bulk Import** tab.
2. Makakakita kayo ng dalawang naki-click na card:
   - **YouTube** (pula) -- Mag-import ng mga video mula sa isang YouTube channel
   - **Vimeo** (asul) -- Mag-import ng mga video mula sa isang Vimeo account
3. I-click ang card para sa platform kung saan naka-host ang inyong mga sermon.

:::tip
Maaari kayong bumalik at magpalit ng pinagmulan anumang oras kung mayroon kayong mga video sa parehong platform.
:::

## Bago Kayo Mag-import

Bago mag-import ng mga video, kailangan ninyo ng kahit isang playlist para ayusin ang mga ito. Kung hindi pa kayo nakagawa ng anumang playlist:

1. I-click ang **Playlists** tab.
2. I-click ang **Create First Playlist** at punan ang pangalan, paglalarawan, petsa ng paglalathala, at thumbnail.
3. I-click ang **Save**, pagkatapos bumalik sa **Bulk Import** tab.

Tingnan ang [Mga Playlist](playlists) para sa mga detalyadong tagubilin sa paggawa at pamamahala ng mga playlist.

## Pag-import ng mga Video

1. Pagkatapos pumili ng YouTube o Vimeo, ilagay ang inyong **YouTube Channel ID** o **impormasyon ng Vimeo account** sa ibinigay na field.
2. I-click ang **Fetch** button para kunin ang lahat ng available na video mula sa inyong channel o account.
3. Pagkatapos mag-fetch, makikita ninyo ang listahan ng lahat ng inyong mga video na may mga checkbox.
4. I-check ang mga kahon sa tabi ng mga video na gusto ninyong i-import. Maaari ninyong piliin lahat o pumili ng mga tiyak na video.
5. Opsyonal, i-enable ang **Auto Import New Videos** para awtomatikong maidagdag ang mga susunod na upload sa inyong library.
6. I-click ang **Import Into Playlist** dropdown para piliin kung aling playlist ang pagdadagdagan ng mga video na ito.
7. I-click ang **Import** button para makumpleto ang bulk import.

Ang inyong mga video ay mai-import kasama ang lahat ng kanilang mga detalye, kabilang ang mga pamagat, paglalarawan, petsa, at thumbnail.

:::info
Ang bulk import ay perpekto para sa pagsisimula kapag lumilipat kayo mula sa ibang platform. Para sa pagdaragdag ng mga indibidwal na sermon sa hinaharap, gamitin sa halip ang pahina ng [Pamamahala ng mga Sermon](managing-sermons).
:::
