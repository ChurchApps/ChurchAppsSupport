# Mga Sermon

<div class="article-intro">

Ang seksyon ng Mga Sermon sa B1 Admin ay nagbibigay-daan sa inyo na pamahalaan ang sermon library at live streaming ng inyong simbahan. Maaari kayong magdagdag ng mga indibidwal na sermon, ayusin ang mga ito sa mga playlist, i-configure ang mga iskedyul ng live stream, at mag-import ng kasalukuyang content nang maramihan mula sa YouTube o Vimeo.

</div>

:::info
Para ma-access ang seksyon ng Mga Sermon, kailangan ninyo ang **contentApi.streamingServices.edit** na pahintulot. Kung hindi ninyo nakikita ang Sermons sa inyong menu, hilingin sa administrator ng inyong simbahan na bigyan kayo ng naaangkop na tungkulin. Tingnan ang [Mga Tungkulin at Pahintulot](../settings/roles-permissions.md) para sa mga detalye sa pamamahala ng mga pahintulot.
:::

## Ano ang Maaari Ninyong Gawin

Ang seksyon ng Mga Sermon ay nakaayos sa apat na pangunahing bahagi:

### Mga Sermon

Magdagdag at pamahalaan ng mga indibidwal na sermon entry sa inyong library. Bawat sermon ay maaaring magsama ng pamagat, tagapagsalita, petsa, paglalarawan, thumbnail, at mga link sa video o audio content. Maaari ninyong i-link ang mga sermon sa YouTube, Vimeo, Facebook, o custom na video URL. Tingnan ang [Pamamahala ng mga Sermon](managing-sermons) para sa buong detalye.

### Mga Playlist

Ayusin ang inyong mga sermon sa mga serye o koleksyon. Pinapadali ng mga playlist para sa inyong kongregasyon na mag-browse sa isang serye ng sermon nang maayos. Ang bawat playlist ay may sariling pangalan, paglalarawan, at thumbnail. Tingnan ang [Mga Playlist](playlists) para sa karagdagang impormasyon.

### Mga Oras ng Live Stream

I-configure ang iskedyul ng live streaming ng inyong simbahan. Mag-set up ng mga paulit-ulit na lingguhang serbisyo o isahang kaganapan, pamahalaan ang mga setting ng chat at video, at kontrolin kung kailan mag-live ang inyong stream. Makikita ng mga bisita ang susunod na naka-iskedyul na oras ng serbisyo sa inyong streaming page. Tingnan ang [Live Streaming](live-streaming) para sa mga tagubilin sa pag-setup.

### Bulk Import

Mabilis na mapunan ang inyong sermon library sa pamamagitan ng pag-import ng mga video nang maramihan mula sa YouTube o Vimeo. Ito ang pinakamabilis na paraan para i-migrate ang inyong kasalukuyang sermon content sa B1. Tingnan ang [Bulk Import](bulk-import) para magsimula.

:::tip
Kung unang beses ninyong nagse-set up ng mga sermon, magsimula sa paggawa ng [playlist](playlists), pagkatapos ay [magdagdag ng mga sermon nang isa-isa](managing-sermons) o [i-import ang mga ito nang maramihan](bulk-import) mula sa YouTube o Vimeo.
:::
