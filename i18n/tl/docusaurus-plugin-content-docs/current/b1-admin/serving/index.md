# Paglilingkod

<div class="article-intro">

Ang seksyon ng Paglilingkod ang iyong sentro para sa pag-coordinate ng mga boluntaryo at pagpaplano ng mga serbisyo ng simbahan. Maging ikaw ay nag-oorganisa ng Sunday worship, midweek gatherings, o mga espesyal na kaganapan, dito mo pinagsasama-sama ang lahat para malaman ng iyong mga team kung sino ang naglilingkod at kailan.

</div>

## Paano gumagana ang pahina ng Paglilingkod

Kapag binuksan mo ang pahina ng **Paglilingkod**, makikita mo ang iyong mga ministeryo na ipinapakita bilang mga tab sa itaas. Ang bawat ministeryo ay may sariling mga uri ng plano at mga team, kaya maaari mong pamahalaan ang maraming ministeryo nang hiwalay. Halimbawa, maaari kang magkaroon ng magkakahiwalay na tab para sa Pagsamba, Ministeryong Pambata, at Hospitality.

:::tip
Kung ikaw ay isang admin, i-toggle ang **Ipakita Lahat** para makita ang lahat ng ministeryo nang sabay-sabay. Ito ay nakakatulong kapag kailangan mong suriin ang mga assignment sa maraming team.
:::

## Mga Sub-Seksyon

Kasama sa seksyon ng Paglilingkod ang tatlong pangunahing lugar:

- **[Mga Plano](./plans.md)** -- Lumikha ng mga plano ng serbisyo para sa mga tiyak na petsa at magtalaga ng mga boluntaryo sa mga team. Tumutulong ang mga plano na ayusin kung sino ang naglilingkod at kailan, para malaman ng lahat ang kanilang papel nang maaga.
- **[Mga Awit](./songs.md)** -- Pamahalaan ang iyong library ng mga awit ng pagsamba na may mga liriko, arrangement, at panlabas na link. Maaari itong gamitin ng iyong worship team para maghanda para sa mga paparating na serbisyo.
- **[Mga Gawain](./tasks.md)** -- Magtalaga at subaybayan ang mga gawain para sa mga miyembro ng iyong team. Mag-set up ng [Mga Automation](./automations.md) para awtomatikong lumikha ng mga paulit-ulit na gawain, para walang malagpasan.

## Pagsisimula

1. Mag-navigate sa **Paglilingkod** mula sa pangunahing menu sa B1 Admin.
2. Pumili ng **tab ng ministeryo** para makita ang mga uri ng plano at mga team nito.
3. Pumili ng uri ng plano para magsimulang [lumikha ng mga plano ng serbisyo](./plans.md), o i-explore ang mga sub-seksyon na [Mga Awit](./songs.md) at [Mga Gawain](./tasks.md).

:::info
Bago ka makagawa ng mga plano, kakailanganin mong i-set up ang iyong mga ministeryo at team sa lugar ng Settings. Tingnan ang dokumentasyon ng Settings para sa mga detalye sa pag-configure ng iyong istraktura ng paglilingkod.
:::
