# Mga Kanta

<div class="article-intro">

Ang pahina ng Mga Kanta ay namamahala sa inyong worship song library. Panatilihin ang lahat ng detalye ng kanta, lyrics, at arrangement sa isang lugar para madaling makapaghanda ang inyong worship team para sa mga paparating na serbisyo at mapanatili ang isang consistent na repertoire.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Pumunta sa seksyon ng **Serving** sa B1 Admin para ma-access ang pahina ng Mga Kanta
- Opsyonal, ihanda ang inyong [mga service plan](./plans.md) para direkta ninyong mai-link ang mga kanta sa [service order](./service-order.md)

</div>

## Pagba-browse ng Inyong Song Library

Kapag binuksan ninyo ang pahina ng **Mga Kanta**, ipinapakita ang inyong mga kanta bilang mga card. Ipinapakita ng bawat card ang thumbnail na larawan, pamagat, artist, at tagal ng kanta. Mag-scroll sa library o gumamit ng search para mahanap ang kailangan ninyo.

## Paghahanap ng mga Kanta

1. I-click ang **Search** button sa pahina ng Mga Kanta.
2. Mag-type ng **pamagat** o pangalan ng **artist** ng kanta.
3. Magfi-filter ang mga resulta habang nagta-type kayo, na nagpapakita ng mga tumutugmang kanta mula sa inyong library.

## Pagdagdag ng Kanta

1. I-click ang **Add Song** sa pahina ng Mga Kanta.
2. Maghanap sa external na song database ayon sa pamagat o artist.
3. Piliin ang kantang gusto ninyong idagdag.
4. Ang kanta ay idadagdag sa inyong library na may mga pangunahing detalyeng pre-filled na.

:::tip
Pagkatapos magdagdag ng kanta, i-click ito para suriin at i-update ang mga detalye nito, magdagdag ng mga arrangement, o mag-attach ng mga external link tulad ng mga YouTube video o chord chart.
:::

## Pagtingin at Pag-edit ng Kanta

I-click ang anumang song card para buksan ang pahina ng detalye nito. Mula rito maaari ninyong:

- **I-edit ang mga detalye ng kanta** -- I-update ang pamagat, artist, tagal, at iba pang metadata.
- **Pamahalaan ang mga arrangement** -- Bawat kanta ay maaaring magkaroon ng maraming arrangement (halimbawa, iba't ibang key o bersyon). Magdagdag, mag-edit, o mag-alis ng mga arrangement kung kinakailangan.
- **Magdagdag ng lyrics** -- Ilagay o i-update ang lyrics para sa bawat arrangement.
- **Itakda ang key** -- Tukuyin ang musikal na key para sa bawat arrangement para malaman ng inyong koponan kung paano maghahanda.
- **Magdagdag ng mga external link** -- Mag-attach ng mga link sa mga resource tulad ng mga YouTube video, chord chart, o sheet music na naka-host sa ibang lugar.

## Mga Arrangement

Ang isang kanta ay maaaring magkaroon ng maraming arrangement para ma-accommodate ang iba't ibang serbisyo o configuration ng koponan. Bawat arrangement ay kinabibilangan ng:

- **Lyrics** para sa display o projection
- **Key** designation
- **Mga external link** sa mga karagdagang resource

:::info
Ang mga kanta mula sa inyong library ay maaaring idagdag nang direkta sa isang [Service Order](./service-order.md) sa loob ng isang plano, na nili-link ang mga detalye at arrangement ng kanta sa partikular na serbisyong iyon.
:::

## Mga Susunod na Hakbang

- Idagdag ang mga kanta sa inyong [Service Order](./service-order.md) kapag binubuo ang isang [service plan](./plans.md)
- Mag-coordinate sa inyong worship team gamit ang [Mga Gawain](./tasks.md) para sa paghahanda at pagsasanay ng kanta
