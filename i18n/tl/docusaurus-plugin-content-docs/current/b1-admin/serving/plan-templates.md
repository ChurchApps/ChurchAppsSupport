# Mga Template ng Plano

<div class="article-intro">

Ang mga plan templates ay nagbibigay-daan sa iyo na magsave ng isang reusable snapshot ng order of service o posisyon ng isang plan at i-apply ito sa mga darating na plano sa isang click. Sa halip na muling bumuo ng parehong istraktura bawat linggo, i-save mo ito nang minsan at i-stamp ito sa maraming darating na mga plano kung kinakailangan mo.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Lumikha ng hindi bababa sa isang [service plan](plans.md) na may isang service order o mga posisyon na natukoy
- Kailangan mo ng access sa Serving area sa B1 Admin

</div>

## Pagsisave ng isang Plan bilang Template

1. Buksan ang isang umiiral na plan mula sa **Serving** area.
2. I-click ang **Templates** button (tuktok ng pahina ng plan).
3. Pumili ng **Save as Template**.
4. Bigyan ng pangalan ang template (halimbawa, "Standard Sunday" o "Worship + Sermon").
5. I-save. Ang template ay kumukuha ng kasalukuyang order of service at posisyon ng plan sa sandaling iyon.

## Pagpapamahalaan ng Mga Template

Mula sa **Templates** button sa anumang pahina ng plan type maaari mong:

- **Magpangalan ulit** — baguhin ang display name ng template
- **I-update mula sa plan** — palitan ang nilalaman ng template gamit ang kasalukuyang estado ng isang plan (kapaki-pakinabang pagkatapos mong pinino ang iyong karaniwang istraktura)
- **I-apply** — i-stamp ang template sa isa o maraming mga darating na plano
- **Tanggalin** — alisin ang isang template na hindi mo na kailangan

## Pag-apply ng isang Template sa Mga Plano

1. I-click ang **Templates** button sa pahina ng plan type.
2. I-click ang **Apply** icon (copy icon) sa tabi ng template na gusto mong gamitin.
3. Pumili kung aling mga darating na plano ang i-apply ito sa pamamagitan ng pagChecking ng kanilang mga kahon.
4. Pumili kung ano ang kopyahin:
   - **Order of Service** — palitan ang selected plans' service order items gamit ang items ng template
   - **Posisyon** — palitan ang selected plans' posisyon (at ang kanilang mga assignment) gamit ang posisyon ng template
5. I-click ang **Apply**.

:::warning
Ang pag-apply ng mga posisyon ay nagpapalitan sa lahat ng umiiral na mga posisyon at mga assignment sa mga napiling plans. Siguraduhin na hindi mo pa naka-schedule ng mga boluntaryo para sa mga posisyong iyon bago mag-apply.
:::

:::tip
I-apply ang Order of Service nang wala ang Posisyon kung gusto mo ng isang pare-parehong kanta/sermon na istraktura ngunit ang mga boluntaryo ay naiiba ang assignment bawat linggo.
:::

## Pagpapanatili ng Mga Template na Nakatuon

Kung ang iyong karaniwang istraktura ng serbisyo ay nagbago, buksan ang template manager at gamitin ang **Update from plan** upang i-sync ang template mula sa iyong pinakabagong plan sa halip na muling bumuo ito mula sa simula.
