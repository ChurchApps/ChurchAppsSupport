# Mga Plano ng Serbisyo

<div class="article-intro">

Ang mga plano ng serbisyo ay nag-oorganisa kung sino ang naglilingkod at kailan. Ang bawat plano ay nakatali sa isang tiyak na petsa at ministeryo, na ginagawang madali ang pag-coordinate ng iyong mga volunteer team linggo-linggo at pagtiyak na bawat serbisyo ay may kumpletong tauhan.

</div>

<div class="prereqs">
<h4>Bago ka magsimula</h4>

- I-set up ang iyong mga ministeryo at team sa lugar ng Settings
- Tiyaking naidagdag na ang mga boluntaryo sa iyong [direktoryo ng mga tao](../people/adding-people.md) at naitala sa mga team

</div>

## Pag-access sa mga plano

1. Mag-navigate sa **Paglilingkod** mula sa pangunahing menu.
2. Pumili ng **tab ng ministeryo** sa itaas ng pahina.
3. I-click ang isang **uri ng plano** para makita ang listahan ng mga plano para sa uri na iyon.
4. I-click ang isang partikular na plano para buksan ito.

## Paggawa ng isang plano

1. Mula sa view ng uri ng plano, i-click ang **Magdagdag ng Plano**.
2. Piliin ang **petsa** para sa serbisyo.
3. I-save ang plano. Maaari ka nang magsimulang mag-assign ng mga miyembro ng team at bumuo ng [order ng serbisyo](./service-order.md).

## Ang pahina ng detalye ng plano

Kapag binuksan mo ang isang plano, makikita mo ang dalawang tab:

- **Mga Assignment** -- Pamahalaan kung aling mga miyembro ng team ang naka-assign sa planong ito. Maaari kang magdagdag ng mga tao mula sa iyong mga kasalukuyang team at makita kung sino ang nakumpirma o nasa pending pa.
- **[Order ng Serbisyo](./service-order.md)** -- Buuin ang order ng serbisyo na may mga elemento tulad ng mga awit ng pagsamba, panalangin, anunsyo, at sermon.

## Pag-assign ng mga miyembro ng team

1. Buksan ang isang plano at pumunta sa tab na **Mga Assignment**.
2. I-click ang isang **team** para i-expand ito.
3. Magdagdag ng mga miyembro mula sa iyong team roster sa pamamagitan ng pag-click ng **Idagdag**.
4. Ang mga na-assign na miyembro ay lalabas sa ilalim ng kanilang team kasama ang kanilang assignment status.

:::tip
I-set up ang iyong mga team sa ministry settings bago gumawa ng mga plano. Sa ganitong paraan, magkakaroon ka ng handang pool ng mga boluntaryo para i-assign.
:::

## Pag-print ng mga plano

Maaari kang mag-print ng isang plano para ipamahagi sa iyong team. Buksan ang plano at gamitin ang opsyon na **I-print** para mag-generate ng printable na bersyon na kasama ang mga assignment at ang order ng serbisyo. Ito ay kapaki-pakinabang para ipamahagi sa mga rehearsal o i-post sa isang common area.

:::info
Ang mga plano ay naka-organisa ayon sa ministeryo. Tiyaking nasa tamang tab ng ministeryo ka bago gumawa o tumingin ng mga plano.
:::

## Mga susunod na hakbang

- Buuin ang iyong [Order ng Serbisyo](./service-order.md) na may mga awit, pagbasa, at iba pang mga elemento
- Magdagdag ng [mga awit](./songs.md) mula sa iyong library nang direkta sa order ng serbisyo
- Gamitin ang [Mga Gawain](./tasks.md) para mag-assign ng follow-up action items sa mga miyembro ng team
