# Service Order

<div class="article-intro">

Ang service order ay tumutukoy sa pagkakasunod-sunod ng mga elemento sa inyong worship service. Gamitin ito para ilatag ang lahat mula sa pambungad na kanta hanggang sa pangwakas na panalangin, para malaman ng buong koponan ninyo ang daloy ng serbisyo nang maaga.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- [Gumawa ng service plan](./plans.md) para sa petsang gusto ninyong buuin ang service order
- Opsyonal, magdagdag ng [mga kanta](./songs.md) sa inyong library para direkta ninyong mai-link ang mga ito sa service order

</div>

## Pag-access sa Service Order

1. Pumunta sa **Serving** at piliin ang inyong ministeryo.
2. I-click ang isang **plan type**, pagkatapos buksan ang isang tiyak na **plano**.
3. I-click ang **Service Order** tab sa pahina ng detalye ng plano.

## Pagbuo ng Inyong Service Order

Ang service order ay isang listahan ng mga item na nakaayos sa pagkakasunod-sunod na mangyayari sa panahon ng serbisyo. Ang mga karaniwang elemento ay kinabibilangan ng:

- Mga worship song
- Mga panalangin
- Mga pagbasa ng banal na kasulatan
- Mga anunsyo
- Sermon o mensahe
- Handog
- Pagsasara

## Pagdagdag ng mga Item sa Service Order

1. Sa **Service Order** tab, i-click ang **Add Item**.
2. Piliin ang uri ng elementong gusto ninyong idagdag (halimbawa, isang kanta mula sa inyong library, isang custom na item, o isang heading).
3. Ilagay ang mga detalye para sa item, tulad ng pamagat at anumang mga tala.
4. I-click ang **Save**.

:::tip
Maaari kayong magdagdag ng mga kanta nang direkta mula sa inyong [Mga Kanta](./songs.md) library. Nili-link nito ang mga detalye ng kanta, kabilang ang mga lyrics at arrangement, sa inyong service plan para magkaroon ang inyong worship team ng lahat ng kailangan nila.
:::

## Muling Pag-aayos ng mga Item

I-drag and drop ang mga item para muling ayusin ang pagkakasunod-sunod. Ang pagkakasunod-sunod na ipinapakita sa screen ang siyang susundin ng inyong koponan sa panahon ng serbisyo.

## Pag-edit at Pag-alis ng mga Item

- I-click ang anumang item sa service order para **i-edit** ang mga detalye nito.
- Gamitin ang opsyong **delete** para alisin ang isang item mula sa pagkakasunod-sunod.

:::info
Ang service order ay tiyak sa bawat plano. Ang mga pagbabagong ginagawa ninyo rito ay nakakaapekto lamang sa napiling plano at hindi babaguhin ang ibang mga plano o ang inyong song library.
:::

## Pagbabahagi ng Service Order

Kapag kumpleto na ang inyong service order, maaari ninyong i-print ang buong plano (kasama ang service order) mula sa pahina ng detalye ng plano. Nagbibigay ito sa inyong koponan ng kumpletong rundown ng serbisyo.

## Mga Susunod na Hakbang

- Pamahalaan ang inyong worship repertoire sa pahina ng [Mga Kanta](./songs.md)
- Mag-assign ng mga boluntaryo sa serbisyo sa tab na Assignments ng [Mga Plano](./plans.md)
- Gumawa ng [Mga Gawain](./tasks.md) para sa anumang mga follow-up item na nauugnay sa serbisyo
