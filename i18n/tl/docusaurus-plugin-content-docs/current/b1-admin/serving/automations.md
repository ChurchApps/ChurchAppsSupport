# Mga Automation

<div class="article-intro">

Ang mga automation ay awtomatikong lumilikha ng mga gawain sa isang paulit-ulit na iskedyul. Sa halip na manu-manong gawin ang parehong mga gawain bawat linggo o buwan, maaari mong i-set up ang isang automation nang isang beses at hayaan ang sistema na asikasuhin ito para sa iyo, na tinitiyak na walang makakalagpas.

</div>

<div class="prereqs">
<h4>Bago ka magsimula</h4>

- Pamilyarisahin ang iyong sarili kung paano gumagana ang [Mga Gawain](./tasks.md) sa B1 Admin
- Tukuyin kung aling mga paulit-ulit na responsibilidad ang gusto mong i-automate

</div>

## Pagtingin sa mga automation

Mag-navigate sa **Paglilingkod** at buksan ang pahina ng **Mga Automation**. Makikita mo ang isang listahan ng lahat ng iyong mga automation, na nagpapakita ng kanilang pamagat, aktibong katayuan, at pattern ng pag-uulit.

## Paggawa ng isang automation

1. Sa pahina ng Mga Automation, i-click ang **Magdagdag ng Automation**.
2. Maglagay ng **pamagat** para sa automation. Ito rin ang gagamitin bilang pamagat ng mga gawain na nililikha nito.
3. Itakda ang **pattern ng pag-uulit** -- piliin kung gaano kadalas dapat gawin ang gawain (halimbawa, lingguhan, buwanan, o sa mga tiyak na araw).
4. Italaga ang gawain sa isang **tao o grupo** na magiging responsable sa bawat pagpapatakbo nito.
5. Itakda ang automation sa **Aktibo** para magsimula itong lumikha ng mga gawain ayon sa iskedyul.
6. I-click ang **I-save**.

## Pag-edit ng isang automation

1. I-click ang isang umiiral na automation sa listahan para buksan ito.
2. I-update ang pamagat, pattern ng pag-uulit, assignment, o anumang iba pang setting.
3. I-click ang **I-save** para ilapat ang iyong mga pagbabago.

## Pag-activate at pag-deactivate

Maaari mong kontrolin kung tumatakbo ang isang automation nang hindi ito dini-delete:

- **Aktibo** -- Ang automation ay lilikha ng mga gawain ayon sa iskedyul nito.
- **Hindi aktibo** -- Ang automation ay naka-pause at hindi lilikha ng anumang bagong gawain hangga't hindi mo ito muling ina-activate.

:::tip
Itakda ang isang automation sa **Hindi aktibo** sa panahon ng holiday breaks o mga espesyal na panahon kung kailan hindi kailangan ang paulit-ulit na gawain. Maaari mo itong muling i-activate anumang oras nang hindi nawawala ang iyong configuration.
:::

## Paano ito gumagana

Kapag tumakbo ang isang automation, gumagawa ito ng bagong [gawain](./tasks.md) na may naka-configure na pamagat at assignment. Ang gawain ay kumikilos tulad ng anumang iba pang gawain -- ang mga itinalaga ay tumatanggap ng mga notification at maaaring pamahalaan ang gawain mula sa kanilang dashboard o sa pahina ng Mga Gawain.

:::info
Ang mga automation ay lumilikha lamang ng mga bagong gawain pasulong. Hindi sila retroactively na lumilikha ng mga gawain para sa mga nakaraang petsa.
:::

## Mga susunod na hakbang

- Matuto pa tungkol sa pamamahala ng mga indibidwal na [Gawain](./tasks.md) na ginagawa ng mga automation
- Gamitin ang mga automation kasabay ng [Mga Plano ng Serbisyo](./plans.md) para mapanatili ang iyong lingguhang paghahanda ng serbisyo sa tamang landas
