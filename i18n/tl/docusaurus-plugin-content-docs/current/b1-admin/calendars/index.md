# Mga Kalendaryo

<div class="article-intro">

Ang seksyon ng Mga Kalendaryo sa B1 Admin ay nagbibigay-daan sa iyo na lumikha at mamahala ng mga kalendaryo ng kaganapan para sa iyong simbahan. Maaari kang mag-set up ng mga kalendaryo para sa iba't ibang ministeryo, mag-aggregate ng mga kaganapan mula sa maraming grupo, at mag-publish ng mga kalendaryo sa iyong B1.church na website para makita ng iyong kongregasyon.

</div>

## Pagtingin sa Iyong Mga Kalendaryo

1. Sa B1 Admin, pumunta sa seksyon ng **Mga Kalendaryo**.
2. Ang pahina ng Mga Kalendaryo ay naglilista ng lahat ng iyong mga kalendaryo kasama ang kanilang pangalan at katayuan.
3. Ipinapakita rin ng bawat kalendaryo ang mga grupong nauugnay dito.

## Ano ang Maaari Mong Gawin

- **Magdagdag ng bagong kalendaryo** -- I-click ang button na magdagdag upang lumikha ng kalendaryo para sa isang partikular na ministeryo, uri ng kaganapan, o layunin. Tingnan ang [Paglikha ng Mga Kalendaryo](creating-calendars) para sa mga hakbang-hakbang na tagubilin.
- **Mag-edit ng kalendaryo** -- I-click ang anumang kalendaryo upang buksan ito at pamahalaan ang mga kaganapan at kaugnay na mga grupo nito.
- **Mag-publish ng mga kalendaryo** -- Ang mga kalendaryo ay maaaring ipakita sa iyong B1.church na website upang makita ng mga bisita ang mga paparating na kaganapan sa iyong simbahan.
- **Lumikha ng mga curated na kalendaryo** -- Pagsamahin ang mga kaganapan mula sa maraming grupo sa isang kalendaryo view para sa isang komprehensibong pagtingin sa lahat ng nangyayari sa iyong simbahan. Tingnan ang [Curated na Kalendaryo](curated-calendar) para sa mga detalye.

:::info
Ang mga kalendaryo ay malapit na gumagana kasama ang [Mga Grupo](../groups/creating-groups.md). Kapag ang mga grupo ay may naka-iskedyul na mga kaganapan, ang mga kaganapang iyon ay maaaring idagdag sa iyong mga kalendaryo. Tingnan ang pahina ng [Paglikha ng Mga Kalendaryo](creating-calendars) upang malaman kung paano ikonekta ang mga grupo sa mga kalendaryo.
:::
