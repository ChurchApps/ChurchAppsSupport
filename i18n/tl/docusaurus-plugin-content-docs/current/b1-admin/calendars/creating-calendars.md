# Paglikha ng Mga Kalendaryo

<div class="article-intro">

Ang paglikha ng kalendaryo sa B1 Admin ay nagbibigay sa iyo ng isang sentral na lugar upang pamahalaan ang mga kaganapan para sa isang ministeryo, departamento, o buong simbahan. Ang bawat kalendaryo ay maaaring mag-pull ng mga kaganapan mula sa isa o higit pang mga grupo, na nagpapadali sa pag-organisa ng lahat.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- I-set up ang mga [grupo](../groups/creating-groups.md) na ang mga kaganapan ay gusto mong isama sa iyong kalendaryo
- Kailangan mo ng administrative na access sa seksyon ng Mga Kalendaryo sa B1 Admin

</div>

## Paglikha ng Bagong Kalendaryo

1. Pumunta sa seksyon ng **Mga Kalendaryo** sa B1 Admin.
2. I-click ang **Magdagdag ng Kalendaryo**.
3. Maglagay ng **pangalan** para sa iyong kalendaryo (halimbawa, "Mga Kaganapan ng Kabataang Ministeryo" o "Pangunahing Kalendaryo ng Simbahan").
4. Magdagdag ng opsyonal na **paglalarawan** upang matulungan ang iyong team na maunawaan kung para saan ang kalendaryong ito.
5. I-click ang **Lumikha** upang i-save ang iyong bagong kalendaryo.

## Ang Pahina ng Detalye ng Kalendaryo

Pagkatapos gumawa ng kalendaryo, i-click ito upang buksan ang pahina ng detalye. Ang pahinang ito ay may dalawang pangunahing bahagi:

- **Kaliwang kolum** -- Ang kalendaryo at editor ng mga kaganapan. Dito maaari mong tingnan, idagdag, at pamahalaan ang mga kaganapan sa iyong kalendaryo.
- **Kanang kolum** -- Ang listahan ng mga kaugnay na grupo. Ipinapakita nito kung aling mga grupo ang may mga kaganapang kasama sa kalendaryong ito.

## Pagdaragdag ng mga Kaganapan

1. Sa pahina ng detalye ng kalendaryo, gamitin ang editor ng kalendaryo sa kaliwa upang magdagdag ng mga bagong kaganapan.
2. Itakda ang pangalan ng kaganapan, petsa, oras, at iba pang mga detalye.
3. Ang mga kaganapang idadagdag mo ay lalabas sa kalendaryo at maaaring i-publish sa iyong website.

## Pagkonekta ng mga Grupo

Ang mga grupong may mga kaganapan sa kalendaryo ay awtomatikong lalabas sa listahan ng mga grupo sa kanang bahagi ng pahina ng detalye.

1. I-click ang **Magdagdag** sa seksyon ng mga grupo upang i-associate ang isang grupo sa iyong kalendaryo.
2. Piliin ang grupo mula sa dropdown.
3. Piliin kung isasama ang **lahat ng kaganapan** mula sa grupong iyon o **mga partikular na kaganapan** lamang.
4. I-click ang **I-save**.

:::tip
Ang pagkonekta ng mga grupo sa iyong kalendaryo ay isang malakas na paraan upang awtomatikong mag-aggregate ng mga kaganapan. Kapag ang isang lider ng grupo ay nagdagdag ng kaganapan sa kanilang [grupo](../groups/creating-groups.md), maaari itong dumaloy sa iyong pangkalahatang kalendaryo ng simbahan nang walang dagdag na trabaho mula sa iyo.
:::

:::info
Kung gusto mong lumikha ng isang kalendaryo na kumukuha ng mga kaganapan mula sa maraming grupo sa iyong simbahan, tingnan ang [Curated na Kalendaryo](curated-calendar) para sa isang pinasimpleng paraan.
:::
