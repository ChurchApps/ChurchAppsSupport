# Mga Kwarto, Mapagkukunan at Pag-schedule

<div class="article-intro">

Ang Mga Kwarto at Mapagkukunan ay nagbibigay-daan sa iyo na subaybayan ang mga bookable na espasyo at kagamitan ng iyong simbahan. Kapag nais ng isang grupo o koponan na gumamit ng isang kwarto o humiram ng isang mapagkukunan para sa isang kaganapan, maaari nilang ihingin ito sa pamamagitan ng B1 -- at maaari mong aprubahan o tanggihan ang mga hiling na iyon upang maiwasan ang double-booking.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Kailangan mo ng administrative access sa Calendars na seksyon sa B1 Admin
- I-set up ang [mga grupo](../groups/creating-groups) na ang mga miyembro ay magsusumite ng mga booking request

</div>

## Pagbubukas ng Mga Kwarto at Mapagkukunan

Sa B1 Admin, pumunta sa **Calendars** at piliin ang **Rooms & Resources**. Ang pahina ay may apat na tab: **Rooms**, **Resources**, **Blockouts**, at **Templates**.

## Mga Kwarto

Ang mga kwarto ay mga pisikal na espasyo na maaaring maireserba sa tabi ng mga kaganapan sa kalendaryo -- isang sanctuary, isang fellowship hall, isang silid-aralan, atbp.

### Pagdagdag ng isang Kwarto

1. Sa **Rooms** tab, mag-click **Add Room**.
2. Magpasok ng **pangalan** para sa kwarto.
3. Opsyonal na itakda ang **kapasidad** (maximum na bilang ng mga tao).
4. Itakda ang **Approval Group** -- kapag humihiling ang isang grupo ng kwartong ito, ang mga lider ng piniling grupo ay sinasabihan at dapat aprubahan ang booking. Iwanan ang blangko kung walang pag-aprubahan na kinakailangan.
5. Mag-click **Save**.

### Pag-edit o Pag-delete ng isang Kwarto

Mag-click sa edit icon sa tabi ng anumang kwarto, gumawa ng iyong mga pagbabago, at mag-click **Save**. Gamitin ang **Delete** upang permanenteng alisin ang isang kwarto.

## Mga Mapagkukunan

Ang mga mapagkukunan ay mga bagay na maaaring i-book sa tabi ng isang kaganapan -- AV equipment, mga projector, mga upuan, isang van, atbp.

### Pagdagdag ng isang Mapagkukunan

1. Mag-click sa **Resources** tab, pagkatapos **Add Resource**.
2. Magpasok ng **pangalan** at opsyonal na isang **paglalarawan**.
3. Itakda ang **Approval Group** kung ang sinuman ay dapat mag-sign off sa mga loan ng mapagkukunan.
4. Mag-click **Save**.

## Mga Blockout

Ang mga blockout ay nakakaiwang maprotektahan ang mga kwarto o mapagkukunan mula sa pagiging i-book sa ilang oras -- para sa renovasyon, nakalaan na mga panahon, o pagpapanatili.

### Pagdagdag ng isang Blockout

1. Mag-click sa **Blockouts** tab, pagkatapos **Add Blockout**.
2. Pumili kung blokahin ang **Room**, **Resource**, o lahat ng mga kwarto at mapagkukunan.
3. Itakda ang **simula** at **pagtatapos** ng petsa/oras para sa blockout.
4. Opsyonal na magdagdag ng **dahilan** na tala.
5. Mag-click **Save**.

## Mga Template ng Kaganapan

Ang mga template ay nagbibigay-daan sa iyo na mag-save ng isang reusable event configuration -- pamagat, paglalarawan, at mga kaugnay na grupo -- upang ang mga paulit-ulit na kaganapan ay maaaring likhain nang pare-pareho.

### Pagdagdag ng isang Template

1. Mag-click sa **Templates** tab, pagkatapos **Add Template**.
2. Magpasok ng **pangalan** para sa template.
3. Opsyonal na mag-link ng **grupo** upang ang kaganapan ay awtomatikong konektado sa grupo na iyon kapag ginawa mula sa template.
4. Mag-click **Save**.

Kapag lumilikha ng isang bagong kaganapan, ang staff ay maaaring pumili ng isang umiiral na template upang pre-fill ang mga detalye ng kaganapan.
