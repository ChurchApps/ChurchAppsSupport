# Paghahanap ng mga Tao

<div class="article-intro">

Ang pahina ng **People** ay nagpapakita ng direktoryo ng iyong simbahan sa isang nahahanap at naso-sort na talahanayan. Mabilis mong mahahanap ang sinuman sa iyong kongregasyon, mako-customize kung aling impormasyon ang ipinapakita, at mae-export ang iyong mga resulta. Ang mahusay na paghahanap ay mahalaga para sa pang-araw-araw na mga gawain sa church administration tulad ng pag-follow up sa mga bisita, paghahanda ng mga contact list, at pamamahala ng mga record ng miyembro.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kailangan mo ng aktibong B1 Admin account na may pahintulot na tingnan ang mga tao. Tingnan ang [Mga Role at Pahintulot](roles-permissions.md) kung hindi ka sigurado sa iyong antas ng access.
- Ang direktoryo ng iyong simbahan ay dapat may mga tao na. Kung hindi ka pa nagdagdag ng sinuman, tingnan ang [Pagdaragdag ng mga Tao](adding-people.md) o [Pag-import ng Datos](importing-data.md).

</div>

## Mabilis na Paghahanap

Ang search bar sa itaas ng pahina ng People ay nagbibigay-daan sa iyo na mahanap ang mga miyembro sa real time:

1. I-click ang **search box** sa itaas ng pahina ng People.
2. Magsimulang mag-type ng pangalan, email, o ibang keyword.
3. Ang mga resulta ay awtomatikong mag-fi-filter habang nagta-type ka (may maikling delay na mga kalahating segundo upang hindi mag-fire ang paghahanap sa bawat keystroke).
4. Ang talahanayan sa ibaba ay mag-a-update upang ipakita lamang ang mga tumutugmang resulta.

:::tip
Hindi mo kailangang pindutin ang Enter. Ang paghahanap ay awtomatikong tumatakbo pagkatapos mong huminto sa pag-type.
:::

## Pag-sort ng mga Resulta

Maaari mong i-sort ang direktoryo sa pamamagitan ng pag-click sa anumang column header sa talahanayan:

1. I-click ang **column header** (halimbawa, **Name** o **Email**) upang mag-sort ayon sa column na iyon.
2. I-click muli ang parehong header upang baligtarin ang pagkasunud-sunod ng sort.

Pinapadali nito ang paghahanap ng mga tao ayon sa alpabeto, edad, o anumang iba pang nakikitang column.

## Pag-customize ng mga Column

Hindi lahat ng impormasyon ay kailangang nakikita nang sabay-sabay. Maaari mong piliin kung aling mga column ang lalabas sa talahanayan:

1. Hanapin ang **column selector dropdown** malapit sa itaas ng talahanayan.
2. I-check o i-uncheck ang mga column upang ipakita o itago ang mga ito. Ang mga available na column ay kinabibilangan ng:
   - **Photo**
   - **Name**
   - **Email**
   - **Phone**
   - **Address**
   - **Birth Date**
   - **Age**
   - **Gender**
   - **Membership Status**
3. Agad na mag-a-update ang talahanayan upang ipakita ang iyong mga napili.

:::info
Ang iyong mga napiling column ay nakakaapekto sa kung ano ang kasama kapag nag-export ka sa CSV. I-customize ang mga column bago mag-export upang makuha ang eksaktong datos na kailangan mo.
:::

## Pag-export ng mga Resulta ng Paghahanap

Maaari mong i-download ang iyong kasalukuyang mga resulta ng paghahanap bilang CSV file anumang oras:

1. I-apply ang anumang paghahanap o mga filter na gusto mo.
2. I-customize ang iyong mga column upang isama ang datos na kailangan mo.
3. I-click ang button na **Export**.
4. Isang CSV file ang mada-download sa iyong computer, handa nang buksan sa Excel, Google Sheets, o anumang spreadsheet application.

Para sa higit pang mga detalye tungkol sa pag-export, tingnan ang [Pag-export ng Datos](./exporting-data.md).

:::tip
Para sa mas advanced na mga query -- tulad ng paghahanap ng lahat ng hindi nag-attend sa nakalipas na tatlong buwan -- subukan ang feature na [AI Search](./ai-search.md), na nagbibigay-daan sa iyo na maghanap gamit ang mga tanong sa natural na wika.
:::
