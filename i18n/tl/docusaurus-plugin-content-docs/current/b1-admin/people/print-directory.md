# I-print ang Direktoryo

<div class="article-intro">

Ang Print Directory feature ay lumilikha ng isang formatted, printable member directory para sa inyong parokya. Ito ay inorganisa ang mga miyembro ayon sa tahanan, nagsasama ng contact information at mga larawan, at idinisenyo upang magmukhang propesyonal kapag nai-print sa standard letter-size paper.

</div>

<div class="prereqs">
<h4>Bago Magsimula</h4>

- Kailangan ninyo ng aktibong B1 Admin account na may pahintulot upang tingnan ang mga tao.
- Ang mga tao sa inyong direktoryo ay dapat may mga larawan at contact information na napuno para sa pinakamahusay na mga resulta.

</div>

## Pagbubukas ng Print Directory

1. Mag-navigate sa **People** sa B1 Admin.
2. I-click ang **print icon** (printer icon) sa toolbar sa itaas ng Mga Resulta ng Tao card. Ito ay lilitaw sa tabi ng Export button.
3. Ang Print Directory ay bumubukas sa isang bagong browser tab at nagsisimulang mag-load ng inyong data ng direktoryo.
4. Pagkatapos ng tungkol sa 1.5 segundo, ang inyong browser's print dialog ay awtomatikong lalitaw.

:::tip
Ang direktoryo ay bumubukas sa isang hiwalay na tab upang maaari ninyong i-print nang hindi umaalis sa Mga Tao page. Pagkatapos ang print dialog ay lilitaw, ang tab ay automatic na bumabalik sa Mga Tao page.
:::

## Ano Ang Kasama

Ang print directory ay nagsasama ng lahat ng **active members** ng inyong parokya. Partikular:

- Ang mga taong may **Inactive** o **Visitor** membership status ay awtomatikong inalis.
- Ang mga taong **opted out** ng direktoryo ay inalis.
- Ang lahat ng iba ay nakagrupo ayon sa **tahanan**.

## Directory Layout

Ang printed directory ay may dalawang bahagi:

### Cover Page

Ang unang pahina ay nagsasama ng:
- Ang pangalan ng inyong parokya
- Ang pamagat na "Member Directory"
- Ang kasalukuyang taon
- Ang lokasyon ng inyong parokya
- Ang bilang ng kabuuang mga taong kasama

### Member Listings

Ang mga natitirang pahina ay naglalista ng mga tahanan sa alphabetical ayon sa pangalawang pangalan. Bawat card ng tahanan ay nagpapakita ng:

- **Photos** -- Circular thumbnails para sa bawat miyembro ng tahanan, na may kanilang pangalan sa ilalim
- **Household display name** -- Halimbawa, "The Smith Family", "John & Jane Doe", o lamang ang pangalan ng tao para sa mga indibidwal
- **Address** -- Street address, siyudad, estado, at zip
- **Phone** -- Home o mobile number
- **Email** -- Pangunahing email address
- **Birthdays** -- Formatted bilang mga indibidwal na petsa para sa bawat miyembro ng tahanan
- **Anniversary** -- Kung nai-record

Ang mga tahanan ay inorganisado sa **alphabetical sections** (A, B, C, ...) na may isang decorative section header bago ang bawat letra na grupo. Dalawang household cards ay lilitaw nang magkakasama sa bawat hilera.

## Print Settings

Ang direktoryo ay naka-format para sa **letter-size paper** (8.5" × 11") na may 0.5-inch margins. Kapag ang browser print dialog ay lumilitaw:

- Iwanang ang paper size na itakda sa **Letter**
- I-enable ang **Background graphics** (minsan ay tinatawag na "Print backgrounds") sa inyong browser's print settings kung gusto ninyong ang section headers na mag-print na may kanilang color styling
- Ang Portrait orientation ay inirekomenda

:::info
Ang direktoryo ay gumagalang sa mga page breaks — ang mga household cards at section headers ay hindi sisikat sa mga pahina.
:::
