# Pag-export ng Datos

<div class="article-intro">

Ang B1 Admin ay nagbibigay-daan sa iyo na i-export ang datos ng iyong simbahan upang magamit ito sa mga spreadsheet, ibahagi sa iyong team, o mag-imbak ng backup. Kung kailangan mo man ng mabilis na listahan ng mga pangalan at email o kumpletong database export, may mga opsyon na akma sa iyong mga pangangailangan.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- Kailangan mo ng aktibong B1 Admin account na may pahintulot na tingnan ang datos na gusto mong i-export. Tingnan ang [Mga Role at Pahintulot](roles-permissions.md) kung hindi ka sigurado sa iyong antas ng access.
- Para sa buong database export, kailangan mo ng access sa lugar ng **Settings**.

</div>

## Pag-export mula sa Pahina ng People

Ang pinakamabilis na paraan upang i-export ang iyong direktoryo ay direkta mula sa pahina ng **People**:

1. Mag-navigate sa **People** sa kaliwang sidebar.
2. Gamitin ang search bar o mga filter upang paliitin ang mga resulta na gusto mong i-export (o iwanang walang filter upang i-export ang lahat). Tingnan ang [Paghahanap ng mga Tao](searching-people.md) para sa mga tip sa pag-filter.
3. Gamitin ang **column selector** upang piliin kung aling mga column ang gusto mong isama sa export (halimbawa, Name, Email, Phone, Address).
4. I-click ang button na **Export**.
5. Isang CSV file ang mada-download sa iyong computer na may datos na kasalukuyang ipinapakita sa talahanayan.

:::tip
I-customize ang iyong mga column bago mag-export. Ang CSV file ay maglalaman ng eksaktong mga column na nakikita mo, kaya maaari mong i-tailor ang export sa iyong mga pangangailangan nang hindi ine-edit ang file pagkatapos.
:::

## Buong Data Export mula sa Settings

Para sa kumpletong export ng lahat ng iyong B1 data (hindi lang mga tao), gamitin ang export tool sa Settings:

1. I-click ang **Settings** sa kaliwang sidebar.
2. I-click ang **Import/Export** sa itaas na navigation.
3. Piliin ang **B1 Database** mula sa dropdown na **Data Source**.
4. Suriin ang data preview at i-click ang **Continue to Destination**.
5. Piliin ang **B1 Export Zip** bilang export destination.
6. Subaybayan ang progreso ng export hanggang sa magpakita ng berdeng checkmark ang lahat ng item.
7. Awtomatikong mada-download ang export file. Hanapin ang `B1Export` file sa iyong downloads folder.
8. I-unzip ang file upang ma-access ang mga indibidwal na CSV file (tulad ng `people.csv`) na maaari mong buksan sa Excel, Google Sheets, o Numbers.

:::info
Ang buong data export ay kinabibilangan ng mga tao, grupo, donasyon, attendance, at marami pa -- lahat ng nasa iyong B1 database. Ito rin ay isang magandang paraan upang lumikha ng pana-panahong backup ng mga record ng iyong simbahan.
:::

## Pag-export ng Datos ng Grupo

Maaari mo ring i-export ang mga listahan ng miyembro para sa mga indibidwal na grupo. Mula sa pahina ng **Groups**, buksan ang isang grupo at i-click ang **download icon** upang i-export ang listahan ng miyembro ng grupong iyon. Tingnan ang [Mga Miyembro ng Grupo](../groups/group-members.md) para sa higit pang mga detalye.

:::info
Ang mga na-export na CSV file ay gumagana sa lahat ng pangunahing spreadsheet application kabilang ang Microsoft Excel, Google Sheets, at Apple Numbers.
:::
