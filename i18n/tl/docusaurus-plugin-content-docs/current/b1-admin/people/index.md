# Mga Tao

<div class="article-intro">

Ang seksyon ng **Mga Tao** ang puso ng pamamahala ng iyong simbahan sa B1 Admin. Dito mo pinapanatili ang direktoryo ng iyong simbahan -- naghahanap ng mga miyembro, nagdaragdag ng mga bagong tao, tinitingnan at ine-edit ang mga profile, at sinusubaybayan ang mga sambahayan. Maliit man o malaki ang iyong kongregasyon, ang pagpapanatiling organisado ng iyong mga record ng tao ang pundasyon na nagpapatakbo sa lahat ng iba pang feature sa B1.

</div>

## Ano ang Maaari Mong Gawin

Narito ang pangkalahatang-tanaw ng mga pangunahing feature na available sa seksyon ng Mga Tao:

1. **Maghanap sa iyong direktoryo** -- Gamitin ang quick search bar upang mahanap ang sinuman ayon sa pangalan, o samantalahin ang advanced na mga filter at AI-powered na natural language search upang mahanap nang eksakto ang kailangan mo. Tingnan ang [Paghahanap ng mga Tao](./searching-people.md) at [AI Search](./ai-search.md) para sa mga detalye.

2. **Magdagdag ng mga bagong tao** -- Mabilis na magdagdag ng mga indibidwal sa iyong direktoryo at punan ang kanilang mga detalye ng profile, contact information, at koneksyon sa sambahayan. Tingnan ang [Pagdaragdag ng mga Tao](./adding-people.md) para sa sunud-sunod na mga tagubilin.

3. **Mag-import at mag-export ng datos** -- Magdala ng datos ng miyembro mula sa CSV file o mag-migrate mula sa ibang church management system tulad ng Breeze. Maaari mo ring i-export ang iyong direktoryo sa CSV anumang oras. Tingnan ang [Pag-import ng Datos](./importing-data.md) at [Pag-export ng Datos](./exporting-data.md).

4. **Pamahalaan ang mga profile** -- Tingnan at i-edit ang mga detalyadong profile para sa bawat tao, kabilang ang contact info, mga miyembro ng sambahayan, [membership sa grupo](../groups/group-members.md), [kasaysayan ng attendance](../attendance/tracking-attendance.md), [mga donasyon](../donations/recording-donations.md), at [mga custom form](../forms/creating-forms.md).

5. **Mag-assign ng mga role at pahintulot** -- Kontrolin kung sino sa iyong team ang maaaring mag-access sa iba't ibang bahagi ng B1 Admin sa pamamagitan ng pag-assign ng mga role. Tingnan ang [Pag-assign ng mga Role](./roles-permissions.md).

## Pagsisimula

Upang ma-access ang seksyon ng Mga Tao, i-click ang **People** sa kaliwang sidebar ng iyong B1 Admin dashboard. Makikita mo ang iyong buong direktoryo ng simbahan na may search bar sa itaas at listahan ng mga miyembro sa ibaba.

:::tip
Kung nagse-set up ka ng B1 sa unang pagkakataon, magsimula sa [pag-import ng iyong mga kasalukuyang datos ng miyembro](./importing-data.md). Pagkatapos ay suriin at linisin ang mga profile kung kinakailangan.
:::
