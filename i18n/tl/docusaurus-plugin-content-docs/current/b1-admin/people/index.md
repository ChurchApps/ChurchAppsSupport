# Mga Tao

<div class="article-intro">

Ang seksyon ng **People** ay ang puso ng pamamahala ng inyong simbahan sa B1 Admin. Dito ninyo pinapanatili ang direktoryo ng inyong simbahan -- paghahanap ng mga miyembro, pagdagdag ng mga bagong tao, pagtingin at pag-edit ng mga profile, at pagsubaybay sa mga sambahayan. Maging maliit o malaking kongregasyon, ang pagpapanatili ng organisado ng mga rekord ng inyong mga tao ay ang pundasyon na nagpapagana ng bawat iba pang feature sa B1.

</div>

## Ano ang Maaari Ninyong Gawin

Narito ang pangkalahatang tanawin ng mga pangunahing feature na available sa seksyon ng People:

1. **Maghanap sa inyong direktoryo** -- Gamitin ang quick search bar upang makahanap ng kahit sino sa pamamagitan ng pangalan, o samantalahin ang mga advanced na filter at AI-powered na natural language search upang mahanap nang eksakto kung sino ang kailangan ninyo. Tingnan ang [Searching People](./searching-people.md) at [AI Search](./ai-search.md) para sa mga detalye.

2. **Magdagdag ng mga bagong tao** -- Mabilis na magdagdag ng mga indibidwal sa inyong direktoryo at punan ang kanilang mga detalye ng profile, impormasyon sa pakikipag-ugnayan, at mga koneksyon sa sambahayan. Tingnan ang [Adding People](./adding-people.md) para sa step-by-step na mga tagubilin.

3. **Mag-bulk edit ng mga tao** -- Mag-update ng maraming tao nang sabay-sabay sa pamamagitan ng pagbabago ng kanilang membership status, marital status, kasarian, mga kagustuhan sa opt-out, o mga pagiging miyembro ng grupo. Nakakatipid ito ng oras kapag gumagawa ng parehong pagbabago sa maraming indibidwal. Tingnan ang [Bulk Editing](./bulk-editing.md).

4. **Mag-import at mag-export ng datos** -- Magdala ng datos ng miyembro mula sa CSV file o maglipat mula sa ibang church management system tulad ng Breeze. Maaari rin ninyong i-export ang inyong direktoryo sa CSV anumang oras. Tingnan ang [Importing Data](./importing-data.md) at [Exporting Data](./exporting-data.md).

5. **Pamahalaan ang mga profile** -- Tingnan at i-edit ang mga detalyadong profile para sa bawat tao, kabilang ang impormasyon sa pakikipag-ugnayan, mga miyembro ng sambahayan, [mga pagiging miyembro ng grupo](../groups/group-members.md), [kasaysayan ng pagdalo](../attendance/tracking-attendance.md), [mga donasyon](../donations/recording-donations.md), at [mga custom na form](../forms/creating-forms.md).

6. **Magtakda ng mga tungkulin at pahintulot** -- Kontrolin kung sino sa inyong koponan ang makaka-access ng iba't ibang bahagi ng B1 Admin sa pamamagitan ng pagtatalaga ng mga tungkulin. Tingnan ang [Assigning Roles](./roles-permissions.md).

## Pagsisimula

Upang ma-access ang seksyon ng People, i-click ang **People** sa kaliwang sidebar ng inyong B1 Admin dashboard. Makikita ninyo ang inyong buong direktoryo ng simbahan na may search bar sa itaas at listahan ng mga miyembro sa ibaba.

:::tip
Kung nag-set up kayo ng B1 sa unang pagkakataon, magsimula sa pamamagitan ng [pag-import ng inyong umiiral na datos ng miyembro](./importing-data.md). Pagkatapos ay suriin at linisin ang mga profile kung kinakailangan.
:::
