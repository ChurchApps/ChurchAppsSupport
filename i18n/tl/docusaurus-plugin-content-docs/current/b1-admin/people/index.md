# Mga Tao

<div class="article-intro">

Ang **People** section ay ang puso ng inyong pamamahala ng parokya sa B1 Admin. Dito ay pinananatili ninyo ang inyong direktoryo ng parokya -- paghahanap ng mga miyembro, pagdagdag ng mga bagong tao, pagtingin at pag-edit ng mga profile, at pagsubaybay sa mga tahanan. Kung mayroon kayong maliit na congregation o malaki, ang panatili ng inyong mga record ng tao na organisado ay ang pundasyon na nagpapalakas sa bawat iba pang feature sa B1.

</div>

## Ano Ang Maaari Ninyong Gawin

Dito ay isang overview ng mga pangunahing feature na available sa People section:

1. **Maghanap sa inyong direktoryo** -- Gamitin ang quick search bar upang mahanap ang sinuman ayon sa pangalan, o makinabang sa advanced filters at AI-powered natural language search upang mahanap kung sino ang kinakailangan ninyo. Tingnan ang [Searching People](./searching-people.md) at [AI Search](./ai-search.md) para sa mga detalye.

2. **Magdagdag ng mga bagong tao** -- Mabilis na magdagdag ng mga indibidwal sa inyong direktoryo at punan ang kanilang profile details, contact information, at household connections. Tingnan ang [Adding People](./adding-people.md) para sa step-by-step instructions.

3. **Bulk edit people** -- I-update ang maraming tao nang sabay-sabay sa pamamagitan ng pagbabago ng kanilang membership status, marital status, gender, opt-out preferences, o group memberships. Ito ay nakakatipid ng oras kapag gumagawa ng parehong pagbabago sa maraming indibidwal. Tingnan ang [Bulk Editing](./bulk-editing.md).

4. **I-print ang inyong direktoryo** -- Lumikha ng isang formatted, printable member directory na naka-group ayon sa tahanan, kumpleto ng mga larawan at contact information. Tingnan ang [Print Directory](./print-directory.md).

5. **I-import at i-export ang data** -- Magdala ng miyembro ng data mula sa isang CSV file o mag-migrate mula sa ibang church management system tulad ng Breeze. Maaari din ninyong i-export ang inyong direktoryo sa CSV anumang oras. Tingnan ang [Importing Data](./importing-data.md) at [Exporting Data](./exporting-data.md).

6. **Pamahalaan ang mga profile** -- Tingnan at i-edit ang mga detalyadong profile para sa bawat tao, kasama ang contact info, mga miyembro ng tahanan, [group memberships](../groups/group-members.md), [attendance history](../attendance/tracking-attendance.md), [donations](../donations/recording-donations.md), at [custom forms](../forms/creating-forms.md).

7. **Magtalok ng mga papel at pahintulot** -- Kontrolin kung sino sa inyong koponan ang maaaring mag-access sa iba't ibang bahagi ng B1 Admin sa pamamagitan ng pagtatalok ng mga papel. Tingnan ang [Assigning Roles](./roles-permissions.md).

## Magsimula

Upang ma-access ang People section, i-click ang **People** sa kaliwang sidebar ng inyong B1 Admin dashboard. Makikita ninyo ang inyong buong direktoryo ng parokya na may search bar sa itaas at isang listahan ng mga miyembro sa ibaba.

:::tip
Kung nagseset-up kayo ng B1 sa unang pagkakataon, magsimula sa pamamagitan ng [pag-import ng inyong kasalukuyang miyembro ng data](./importing-data.md). Pagkatapos ay suriin at linisin ang mga profile ayon sa pangangailangan.
:::
