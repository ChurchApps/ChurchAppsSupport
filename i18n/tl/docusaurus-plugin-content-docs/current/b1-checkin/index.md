# Pangkalahatang-tanaw ng B1 Church Checkin

<div class="article-intro">

Ang B1 Church Checkin ay isang self-service kiosk app para sa mga Android tablet. Pinapayagan nito ang mga miyembro ng simbahan na hanapin ang kanilang pamilya, italaga ang mga bata sa mga grupo o klase, at mag-print ng mga name label at parent pickup slip -- lahat nang walang kailangang volunteer sa desk.

</div>

## Paano Ito Gumagana

Maglagay ng Android tablet sa iyong lobby o welcome area. Ang mga miyembro ay lalapit, maghahanap ng kanilang household sa pamamagitan ng numero ng telepono o apelyido, susuriin ang kanilang pamilya, pipili kung aling mga grupo ang dapat daluhan ng bawat tao, at ita-tap ang **Check-in**. Awtomatikong magpi-print ang app ng mga name label at pickup slip, pagkatapos ay mare-reset para sa susunod na pamilya.

## Mga Pangunahing Tampok

- **Self-service na paghahanap ng miyembro** -- Maghanap sa pamamagitan ng numero ng telepono o apelyido para mabilis na mahanap ang iyong household.
- **Pagsusuri ng household** -- Tingnan ang lahat ng miyembro ng pamilya na may mga larawan at kasalukuyang mga group assignment sa isang tingin.
- **Pagtatalaga ng grupo** -- Italaga ang bawat miyembro ng pamilya sa tamang klase o grupo para sa napiling oras ng serbisyo.
- **Check-in ng bisita** -- Magdagdag ng bisita sa iyong household kaagad gamit ang pangalan at apelyido.
- **Pag-print ng label** -- Mag-print ng mga name label para sa mga bata at parent pickup slip na may natatanging security code.
- **Suporta sa network printer** -- Tumuklas at mag-configure ng mga network printer nang direkta mula sa app.
- **Auto-login** -- Pagkatapos ng unang login, tinatandaan ng app ang iyong mga credential at dumiretso sa services screen sa mga susunod na pagbukas.
- **Suporta sa maraming simbahan** -- Kung ang iyong account ay naka-link sa higit sa isang simbahan, maaari kang pumili kung saan mag-check in.
- **Branding ng simbahan** -- Ipinapakita ng app ang logo ng iyong simbahan sa header kapag napili na ang isang simbahan.
