# Pagsusuri ng Iyong Household

<div class="article-intro">

Pagkatapos mong pumili ng tao mula sa search results, nilo-load ng app ang iyong buong household at ipinapakita ito sa household review screen. Ito ang sentro ng proseso ng check-in kung saan maaari mong makita ang lahat sa iyong pamilya at pamahalaan ang kanilang mga group assignment bago mag-check in.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [Pumili ng serbisyo](./selecting-service) mula sa services screen
- [Maghanap ng miyembro](./looking-up-members) at piliin sila mula sa search results

</div>

## Pagtingin sa mga Miyembro ng Household

Ang bawat miyembro ng household ay lumalabas bilang isang card na may:

- Kanilang **larawan** (o isang placeholder icon kung walang larawan sa file)
- Kanilang **display name**
- Isang **buod ng kanilang kasalukuyang mga group assignment** (ipinapakita bilang maliliit na chip sa ibaba ng pangalan kapag naka-collapse ang card)
- Isang **expand/collapse arrow** sa kanang bahagi

## Pag-expand ng Miyembro

I-tap ang card ng isang miyembro para i-expand ito. Kapag na-expand, makikita mo ang isang row para sa bawat **oras ng serbisyo** sa loob ng napiling serbisyo. Ang bawat row ay nagpapakita ng:

- Ang **pangalan ng oras ng serbisyo** (halimbawa, "9:00 AM" o "11:00 AM")
- Isang **button** na nagpapakita ng kasalukuyang naka-assign na grupo, o "Select Group" kung wala pang naka-assign na grupo

I-tap ang group button para pumunta sa [group selection screen](./group-assignment) kung saan maaari kang pumili o baguhin ang grupo para sa taong iyon at oras ng serbisyo.

I-tap ulit ang card ng miyembro para i-collapse ito.

:::tip
Hindi mo kailangang i-assign ang bawat miyembro ng pamilya sa isang grupo. Ang mga adult na dumadalo sa main worship service ay maaaring iwan nang walang group assignment. Mag-assign lamang ng mga grupo para sa mga miyembro na kailangang mag-check in sa isang partikular na klase o programa.
:::

## Pag-navigate Mula sa Screen na Ito

Mula sa household review screen, mayroon kang tatlong opsyon:

- **I-expand ang isang miyembro** at i-tap ang service time button para [mag-assign o magpalit ng grupo](./group-assignment).
- **I-tap ang Add Guest button** sa ibaba ng member list para [magdagdag ng bisita](./adding-guests) sa iyong household.
- **I-tap ang Check-in button** sa ibaba ng screen para [kumpletuhin ang check-in](./completing-checkin) para sa lahat ng nasa household.

:::info
Ang anumang group assignment mula sa nakaraang check-in sa parehong serbisyo ay awtomatikong nilo-load. Kung ang iyong mga anak ay dumadalo sa parehong mga grupo bawat linggo, maaaring hindi mo na kailangang baguhin ang anuman -- suriin lang at mag-check in.
:::

## Mga Susunod na Hakbang

- [Mag-assign ng mga grupo](./group-assignment) sa mga miyembro ng pamilya na kailangang mag-check in sa isang partikular na klase o programa.
- [Magdagdag ng bisita](./adding-guests) kung may bumibisita sa iyong pamilya.
- [Kumpletuhin ang check-in](./completing-checkin) kapag tama na ang lahat ng assignment.
