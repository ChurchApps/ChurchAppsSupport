# Paghahanap ng mga Miyembro

<div class="article-intro">

Pagkatapos pumili ng serbisyo, darating ka sa member lookup screen. Dito hinahanap mo ang iyong pamilya para makapag-check in. Sinusuportahan ng app ang paghahanap sa pamamagitan ng numero ng telepono o pangalan.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [Pumili ng serbisyo](./selecting-service) mula sa services screen
- Siguraduhing ang iyong household ay [naidagdag na sa sistema](../../b1-admin/people/adding-people.md) ng isang administrator ng simbahan

</div>

## Mga Mode ng Paghahanap

Ang lookup screen ay nag-aalok ng dalawang search mode. Maaari kang magpalipat-lipat sa kanila sa pamamagitan ng pag-tap sa toggle sa itaas ng screen:

- **Phone** -- Maghanap sa pamamagitan ng numero ng telepono. Ito ang default na mode.
- **Name** -- Maghanap sa pamamagitan ng pangalan.

## Paghahanap sa Pamamagitan ng Numero ng Telepono

1. Siguraduhing **Phone** ang aktibong search mode (naka-highlight ito sa itaas).
2. I-tap ang search field at maglagay ng hindi bababa sa **4 na digit** ng iyong numero ng telepono gamit ang on-screen keypad.
3. I-tap ang **Search** button o pindutin ang search key sa keyboard.

Hinahanap ng app ang mga miyembro na may numero ng telepono na tumutugma sa mga digit na iyong nilagay. Kung naglagay ka ng higit sa 4 na digit, ginagamit nito ang buong numero. Kung eksaktong 4 ang nilagay mo, tinutugma nito ang huling 4 na digit ng mga numero ng telepono sa file.

## Paghahanap sa Pamamagitan ng Pangalan

1. I-tap ang **Name** toggle sa itaas ng screen para lumipat sa name search mode.
2. Maglagay ng hindi bababa sa **2 character** ng pangalan ng tao sa search field.
3. I-tap ang **Search** button.

Hinahanap ng app ang mga miyembro na may pangalang tumutugma sa iyong ni-type.

## Pagpili ng Tao

Ang search results ay lumalabas bilang mga card sa ibaba ng search field. Ang bawat card ay nagpapakita ng **pangalan** at **larawan** ng tao (kung mayroon sa file).

I-tap ang card ng isang tao para piliin sila. Nilo-load ng app ang lahat ng miyembro ng household ng taong iyon at dinadala ka sa [household review screen](./household-review).

:::tip
Ang paghahanap sa pamamagitan ng huling 4 na digit ng numero ng telepono ang pinakamabilis na paraan para hanapin ang iyong pamilya. Hikayatin ang mga miyembro na gamitin ang paraang ito para sa mas mabilis na check-in experience.
:::

:::warning
Kung walang nahanap na resulta, nagpapakita ang app ng mensahe na nagpapaalam sa iyo. I-double check ang mga digit o pangalan na iyong nilagay, o subukan ang kabilang search mode. Kung ikaw ay first-time visitor, humingi ng tulong sa isang volunteer para [maidagdag sa sistema](../../b1-admin/people/adding-people.md).
:::

## Susunod na Hakbang

Pagkatapos pumili ng tao, [susuriin mo ang iyong household](./household-review) at pamamahalaan ang mga group assignment.
