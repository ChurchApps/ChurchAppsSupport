# Pagpili ng Serbisyo

<div class="article-intro">

Ang unang hakbang ng bawat check-in ay ang pagpili kung aling serbisyo ang iyong dadaluhan. Ang services screen ay lumalabas pagkatapos mag-load ng app at tinutukoy nito kung aling mga oras ng serbisyo at grupo ang available para sa natitirang bahagi ng check-in flow.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [Mag-log in](../getting-started/logging-in) sa B1 Church Checkin app at piliin ang iyong simbahan
- Siguraduhing [naka-configure na ng administrator ng iyong simbahan ang mga serbisyo at oras ng serbisyo](../../b1-admin/attendance/setup.md) sa B1 Admin

</div>

## Paano Ito Gumagana

1. Pagkatapos mag-log in (o pagkatapos ng auto-login sa mga return visit), ipinapakita ng app ang **services screen**.
2. Makikita mo ang listahan ng lahat ng serbisyong naka-configure para sa iyong simbahan. Ang bawat serbisyo ay lumalabas bilang isang card na nagpapakita ng pangalan ng serbisyo (halimbawa, "Sunday Morning" o "Wednesday Evening").
3. I-tap ang serbisyong iyong dadaluhan.

Nilo-load ng app ang mga oras ng serbisyo at grupo na nauugnay sa serbisyong iyon, pagkatapos ay dinadala ka sa [member lookup screen](./looking-up-members).

:::info
Ang mga serbisyo ay naka-configure ng administrator ng iyong simbahan sa B1 Admin sa ilalim ng Attendance section. Kung hindi mo makita ang serbisyong inaasahan mo, hilingin sa iyong admin na i-verify na nagawa na ito sa [attendance setup](../../b1-admin/attendance/setup.md).
:::

## Ano ang Nangyayari sa Likod

Kapag pumili ka ng serbisyo, kumukuha ang app ng tatlong bagay mula sa server:

- Ang **mga oras ng serbisyo** para sa serbisyong iyon (halimbawa, ang isang serbisyo ay maaaring may 9:00 AM at 11:00 AM na time slot).
- Ang **mga grupo** na available sa bawat oras ng serbisyo (halimbawa, Nursery, Preschool, Elementary).
- Ang **mga group-to-service-time link** na tumutukoy kung aling mga grupo ang available sa aling oras.

Ang data na ito ay naka-cache nang lokal para maging mabilis at responsive ang natitirang proseso ng check-in.

:::tip
Kung isang serbisyo lang ang naka-configure sa iyong simbahan, kailangan mo pa ring i-tap ito para magpatuloy. Hindi awtomatikong pinipili ng app ang isang serbisyo.
:::

## Susunod na Hakbang

Pagkatapos pumili ng serbisyo, [hahanapin mo ang iyong pamilya](./looking-up-members) sa pamamagitan ng numero ng telepono o pangalan.
