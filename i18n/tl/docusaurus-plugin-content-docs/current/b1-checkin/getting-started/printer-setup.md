# Pag-setup ng Printer

<div class="article-intro">

Ang B1 Church Checkin ay maaaring mag-print ng mga name label at parent pickup slip sa isang network label printer. Ang pag-setup ng printer ay opsyonal -- maaari mong gamitin ang app nang walang printer para sa attendance tracking lang.

</div>

<div class="prereqs">
<h4>Bago Ka Magsimula</h4>

- [Mag-log in](./logging-in) sa B1 Church Checkin app at piliin ang iyong simbahan
- Ikonekta ang isang compatible na network label printer sa parehong Wi-Fi network ng tablet

</div>

## Pag-access sa Printer Settings

Maaari mong buksan ang printer settings screen anumang oras sa pamamagitan ng pag-tap sa **printer status bar** sa pinakataas ng screen. Ang manipis na bar na ito ay nagpapakita ng version number ng app at kasalukuyang printer connection status.

## Pag-scan ng mga Printer

Kapag bumukas ang printer settings screen, awtomatikong i-scan ng app ang iyong local network para sa mga compatible na printer. Makikita mo ang loading indicator na may mensaheng "Scanning for printers" habang nagsa-scan.

Kapag natapos ang scan, makikita mo ang listahan ng mga natuklasang printer. Ang bawat printer card ay nagpapakita ng:

- Ang **modelo ng printer**
- Ang **IP address** ng printer sa iyong network

Ang opsyong **No Printer** ay laging kasama sa ibaba ng listahan.

:::warning
Ang tablet at printer ay dapat na nakakonekta sa parehong local network para matukoy ng scan ang printer. Kung hindi lumalabas ang iyong printer, i-verify na pareho ang Wi-Fi network ng dalawang device at naka-on ang printer.
:::

## Pagpili ng Printer

1. I-tap ang printer na gusto mong gamitin. Ang napiling printer card ay made-highlight na may checkmark.
2. Para mag-print nang walang pisikal na printer (para sa testing o attendance-only mode), piliin ang **No Printer**.

## Test Printing

Bago i-save ang iyong pagpili, maaari mong i-verify na gumagana ang koneksyon:

1. Pumili ng printer mula sa listahan.
2. I-tap ang **Test Print** button sa ibaba ng screen.
3. Nagpapadala ang app ng test label sa printer. Kung maayos ang koneksyon ng printer, magpi-print ng test page.

Kung ita-tap mo ang **Test Print** habang **No Printer** ang napili, magpapakita ang app ng alert na nagpapaalala na walang napiling printer.

:::tip
Laging mag-run ng test print pagkatapos pumili ng printer para kumpirmahin ang koneksyon bago ang susunod na serbisyo.
:::

## Pag-save ng Iyong Pagpili

Kapag satisfied ka na sa iyong pagpili ng printer:

1. I-tap ang **Save & Restart** button sa ibaba ng screen.
2. Sine-save ng app ang iyong pagpili ng printer at nagre-restart para i-apply ang mga setting.

Ang iyong pagpili ng printer ay naaalala sa pagitan ng mga app restart. Hindi mo na kailangang i-setup ito ulit maliban kung gusto mong magpalit ng printer.

:::info
Kung inilipat mo ang kiosk sa ibang lokasyon o binago ang iyong network setup, bumalik sa printer settings para mag-scan ulit ng mga printer at i-update ang iyong pagpili.
:::

## Mga Susunod na Hakbang

Kapag naka-configure na ang iyong printer, handa ka nang magsimulang mag-check in. Tingnan ang [workflow ng check-in](../check-in/) para malaman kung paano gumagana ang buong proseso, simula sa [pagpili ng serbisyo](../check-in/selecting-service).
