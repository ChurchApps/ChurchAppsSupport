# Pagsisimula sa B1 Church Checkin

<div class="article-intro">

Ang pag-setup ng B1 Church Checkin ay ilang minuto lang. Inaakay ka ng seksyong ito sa pag-install ng app sa isang Android tablet, pag-log in gamit ang iyong ChurchApps account, at opsyonal na pag-configure ng network label printer.

</div>

## Ano ang Kailangan Mo

- **Isang Android tablet** -- Pinakamahusay ang 10-inch na tablet para sa kiosk experience.
- **Isang ChurchApps account** -- Ang email at password na ginagamit mo para sa B1 Admin o iba pang mga produkto ng ChurchApps.
- **Isang network label printer** (opsyonal) -- Para sa pag-print ng mga name label at parent pickup slip. Maaaring mag-scan ang app ng iyong local network para mahanap ang mga compatible na printer.

## Pag-install

1. Buksan ang **Google Play Store** sa iyong Android tablet.
2. Hanapin ang **B1 Church Checkin**.
3. I-tap ang **Install** at hintayin na matapos ang download.
4. Buksan ang app. Makikita mo ang splash screen habang nag-i-initialize ito.

## Unang Beses na Pag-setup

Kapag binuksan mo ang app sa unang pagkakataon, dadaan ka sa tatlong hakbang:

1. **[Mag-log in](./logging-in)** -- Ilagay ang iyong email at password para mag-authenticate.
2. **[Pumili ng simbahan](./logging-in)** -- Kung ang iyong account ay naka-link sa maraming simbahan, piliin kung alin ang dapat pagsilbihan ng kiosk na ito.
3. **[Mag-setup ng printer](./printer-setup)** -- Mag-configure ng network printer para sa label printing, o laktawan ang hakbang na ito kung hindi mo kailangan ng mga label.

Pagkatapos ng unang login, sine-save ng app ang iyong mga credential at pagpili ng simbahan. Sa mga susunod na pagbukas, dumiretso na ito sa services screen nang awtomatiko.

:::info
Sinusuri ng app ang mga over-the-air update sa bawat pagsisimula. Kung may available na update, awtomatiko itong magda-download at mag-a-apply bago ka makarating sa login screen.
:::

:::tip
Siguraduhing na-configure na ng administrator ng iyong simbahan ang mga serbisyo, oras ng serbisyo, at mga grupo sa B1 Admin bago i-setup ang kiosk. Tingnan ang [gabay sa pag-setup ng attendance](../../b1-admin/attendance/setup.md) para sa mga detalye.
:::
