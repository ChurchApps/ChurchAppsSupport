# Plan Mode

<div class="article-intro">

Plan Mode verbindt uw TV met een kerkserviceplan. In plaats van inhoud op basis van een klaslokaalschema op te halen, volgt het het plan dat is geconfigureerd door uw kerkleiding in B1 Admin. Dit is ideaal wanneer u wilt dat een TV precies weergeeft wat in het huidige weekplan is toegewezen.

</div>

## Hoe Het Verschilt Van Classroom Mode

- **[Classroom Mode](../classroom-mode/)** haalt de les op die is gepland voor een specifieke kamer op een bepaalde datum
- **Plan Mode** haalt de inhoud op die aan een plantype is gekoppeld, wat lessen, providerpresentaties, add-ons en ander media in aangepaste volgorde kan bevatten

Plan Mode is vooral nuttig wanneer uw serviceplan een mix van inhoud uit verschillende bronnen bevat, allemaal in één volgorde georganiseerd.

## Hoe Het Werkt

1. FreePlay genereert een koppelingscode op het scherm
2. Een beheerder voert die code in **B1 Admin** onder de instellingen voor het plantype in
3. Na koppeling laadt FreePlay het huidige plan en downloadt alle gerelateerde media
4. Druk op **Start Plan** om afspelen te starten

De TV blijft aan het plantype gekoppeld. Elke week pikt het automatisch het nieuwste plan en de inhoud ervan op.
