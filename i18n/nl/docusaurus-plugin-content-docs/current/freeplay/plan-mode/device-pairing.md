# Pairing Your Device

<div class="article-intro">

Als u Plan Mode wilt gebruiken, moet u uw TV met een plantype in B1 Admin koppelen. FreePlay genereert een unieke koppelingscode die het apparaat aan het plan van uw kerk koppelt, waardoor automatische inhoudslevering elke week mogelijk is.

</div>

<div class="prereqs">
<h4>Voordat u begint</h4>

- Installeer en start FreePlay -- zie [Getting Started](../getting-started/)
- Kies **Pair to Plan** uit het [pairing mode screen](../getting-started/pairing-modes.md)
- Hebt u toegang tot **B1 Admin** op een computer of telefoon om de koppeling in te stellen

</div>

## De Koppelingscode Genereren

1. Kies **Pair to Plan** in het scherm **Select Pairing Mode**
2. FreePlay neemt contact op met de server en genereert een korte koppelingscode
3. De code verschijnt in grote letters in het midden van het scherm
4. Onder de code toont een pulserende indicator **Waiting for connection**

De code wordt weergegeven als afzonderlijke tekens voor gemakkelijk lezen van over de kamer.

## De Code In B1 Admin Invoeren

1. Open op een computer of telefoon **B1 Admin** voor uw kerk
2. Ga naar de instellingen **Plan Type** voor het plan dat u wilt toewijzen
3. Voer de koppelingscode in die op de TV wordt weergegeven
4. Sla de instellingen op

FreePlay peilt de server elke paar seconden om te controleren of de koppeling is voltooid. Zodra **B1 Admin** de verbinding bevestigt, gaat de TV automatisch naar het plandownloadscherm.

## Plan-inhoud Downloaden

Na koppeling laadt FreePlay het huidige plan voor dat plantype. Het geeft weer:

- De plannaam en servicedatum
- De naam van de gerelateerde les (als het plan lesseninhoud bevat)
- Een voortgangsindicator met **Downloading item X of Y**

Wanneer alle mediabestanden zijn gedownload, verschijnt de **Start Plan**-knop. Druk op **Select** op uw afstandsbediening om afspelen te starten.

:::tip
Het plan wordt elk uur automatisch vernieuwd. Als het plan gedurende de dag wordt bijgewerkt, pikt FreePlay de wijzigingen zonder handmatige bemoeienis op.
:::

## Alternatief: Zoeken Op Kerknaam

Als u het codeflow-codeproces liever niet gebruikt, kunt u onderaan het koppeelscherm **or search by church name** selecteren. Dit brengt u naar het kerkenzoekscherm waar u uw kerk kunt vinden en in plaats daarvan verbinding kunt maken met een klaslokaal.

## Als De Koppeling Mislukt

Als de koppelingscode niet kan worden gegenereerd (bijvoorbeeld vanwege een netwerkprobleem), ziet u een foutbericht met een **Try Again**-knop. Zorg ervoor dat uw TV is verbonden met internet en probeer het opnieuw.

:::warning
Koppelingscodes verlopen na een bepaalde periode. Als u te lang wacht, genereert u een nieuwe code door naar het koppeelscherm terug te keren.
:::
