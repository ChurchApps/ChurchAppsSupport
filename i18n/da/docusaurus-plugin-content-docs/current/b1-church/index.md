# B1.church

<div class="article-intro">

B1.church er medlemmernes portal for din kirke. Det er det sted, hvor din menighed går for at holde sig forbundet, få adgang til kirkeværktøjer og engagere sig i dit fællesskab hele ugen. Hver kirke får sin egen URL på `https://yourchurchname.b1.church`.

</div>

Når medlemmer besøger din B1.church-side, ser de din kirkes offentlige website med dit branding, navigationlinks og indholdssider. Efter login kan de få adgang til **medlemsportalen**, hvor alle deres personlige kirkeværktøjer lever på ét sted.

## Vigtigste funktioner

- **Tidslinje** -- Et personaliseret feed med opdateringer, samtaler og aktivitet fra dine grupper og kirke.
- **Grupper** -- Se grupperne, du tilhører, se gruppemedlemmer, deltag i samtaler og spor tilstedeværelse.
- **Giver** -- Donér til din kirke online og se din givelhistorik.
- **Arrangementer** -- Gennemse kommende arrangementer og registrer dig selv og din familie.
- **Tjening** -- Se serviceplaner, dine frivilligpositioner og tilmeld dig åbne frivillige muligheder.
- **Planer** -- Se kommende serviceplaner, se dine tildelte positioner og få adgang til sanginformation og serviceordrer.
- **Tjek ind** -- Tjek ind dig selv og dine husholdsmedlemmer til gudstjenester og arrangementer. Gæster kan registrere sig via QR-kode.
- **Fællesskabsmappe** -- Gennemse din kirkes medlemsmappe, se profiler og send direkte meddelelser.
- **Indhold** -- Få adgang til prædikener, læs Bibelen, arbejd gennem lektioner og se live streams, alt fra din kirkes side.
