# Tjening

<div class="article-intro">

Afsnittet Tjening på B1.church lader dig gennemse frivillige muligheder og tilmeld dig positioner på din kirke. Ud over at se dine tildelte planer og serviceordrer, kan du nu selvvælge positioner, der skal besættes.

</div>

## Hvad du kan gøre

- **Gennemse frivillige muligheder** -- Se kommende serviceplaner med åbne positioner, der har brug for frivillige.
- **Tilmeld dig positioner** -- Vælg en position og tilmeld dig selv med et enkelt klik.
- **Se dine planer** -- Se dine kommende og tidligere [tjeneste tildelte opgaver](../plans/viewing-plans).
- **Fjern tilmeldinger** -- Aflys en selvtilmelding, hvis dine planer ændres (før fristen).

## Hvordan det fungerer

Din kirkes administrator opretter serviceplaner og markerer visse positioner som tilgængelige for selvtilmelding. Disse positioner vises på frivilligtilmeldingssiden, organiseret efter kategori (såsom "Band", "Gæstevenlighed" eller "Børneministerium"). Du kan se, hvor mange pladser der er åbne for hver position og tilmelde dig de, som du er tilgængelig til.

:::info
Kun positioner, som din kirkes administrator har aktiveret til selvtilmelding, vises på frivilligsiden. Nogle positioner kan kun blive besæt gennem direkte tildeling.
:::
