# B1 Mobile

<div class="article-intro">

B1 Mobile er companion-appen til B1.church, tilgængelig på iOS og Android. Det giver kirkebedlemmer på farten adgang til alle værktøjer din kirke leverer -- grupper, giver, tjek ind, tjeneste planer, medlemsmappen, private meddelelser, prædikener, Bibelen, vers af dagen, lektioner og mere.

</div>

Appen bruger din ChurchApps-konto, den samme som du bruger på B1.church. Efter at have logget ind og valgt din kirke, tilpasser appen sig til din kirkes branding og konfiguration. Hvis du tilhører flere kirker, kan du skifte mellem dem når som helst.

B1 Mobile understøtter push-meddelelser, så du aldrig misser en vigtig opdatering fra din kirke eller grupper. Hjem instrumentbrættet personaliserer sig selv baseret på din brug -- de funktioner du besøger oftest, bliver promoveret til toppen for hurtig adgang.

## Vigtigste funktioner

- **Grupper** -- Se dine grupper, deltag i samtaler og holdt forbundet med gruppemedlemmer.
- **Giver** -- Donér til din kirke direkte fra din telefon.
- **Arrangementer** -- Gennemse kommende arrangementer og registrér direkte fra din telefon.
- **Tjek ind** -- Tjek ind dig selv og dine husholdsmedlemmer til gudstjenester og arrangementer.
- **Tjeneste planer** -- Se kommende serviceplaner, dine tildelte frivillige positioner og tilmeld dig åbne positioner.
- **Medlemsmappe** -- Gennemse din kirkes mappe og se medlemsprofiler.
- **Messaging** -- Send og modtag private meddelelser med andre kirkebedlemmer.
- **Prædikener** -- Se eller lyt til prædikener fra din kirke.
- **Bibelen** -- Læs Bibelen i appen.
- **Vers af dagen** -- Start din dag med et dagligt vers.
- **Lektioner** -- Få adgang til og arbejd gennem lektioner fra Lessons.church.
