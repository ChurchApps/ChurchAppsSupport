# Profil

<div class="article-intro">

Din profil i B1 Mobile er hvor du administrer dine personlige oplysninger, din husstand og dine privatlivsindstillinger. Alt, hvad du opdaterer her, afspejles på tværs af alle ChurchApps-tjenester, så du kun behøver foretage ændringer ét sted.

</div>

## Profil-faner

Profilskærmen er organiseret i flere faner:

### Profil

Fanen **Profil** lader dig se og redigere dine personlige oplysninger, herunder dit navn, kontaktdetaljer og profilbillede. At holde din profil opdateret hjælper din kirke med at holde forbindelse til dig.

### Husstand

Fanen **Husstand** viser dine familiemedlemmer og deres roller inden for din husstand. Disse oplysninger bruges til funktioner som [tjek ind](../checkin/), hvor du kan tjekke hele din familie ind på samme tid.

### Synlighed

Fanen **Synlighed** lader dig kontrollere, hvad andre kirkebedlemmer kan se om dig i [mappen](../community/member-directory.md). Du bestemmer, hvilke personlige detaljer der deles, og hvilke der forbliver private.

### Afventer

Fanen **Afventer** viser eventuelle profiländringer, du har indsendt, der afventer administratorgodkendelse.

### Konto

Fanen **Konto** giver adgang til dine sikkerhedsindstillinger.

## Komme til din profil

1. Åbn skuffelementet.
2. Tryk på **Profil** for at åbne din profilskærm.
3. Brug fanerne øverst for at navigere mellem afsnit.
