# B1 Admin Oversigt

<div class="article-intro">

B1 Admin er dit kirkestyrings dashboard. Det giver værktøjer til at administrere ethvert aspekt af din kirkes operationer -- fra medlemmer kataloger og donationer til websted opbygning og live streaming -- alt fra ét sted.

</div>

## Kom i gang

1. Åbn din browser og gå til [admin.b1.church](https://admin.b1.church)
2. Log ind med dit brugernavn og adgangskode
3. Du lander på **Dashboard**, som giver en oversigt over dine kirkedata sammen med **Hurtige handlinger** til almindelige opgaver som tilføj folk, opret grupper, registrer donationer og mere

:::tip
Ny til B1 Admin? Start med [Introduktion](./introduction) til en video gennemgang, derefter besøg [Indstillinger](./settings/) for at konfigurere dine kirkeoplysninger og inviter dit hold.
:::

## Vigtigste funktioner

- **[Folk](./people/)** - Administrer dit kirkekatolog, tilføj medlemmer, spor husstande
- **[Grupper](./groups/)** - Opret og organiser kirkegrupper, hold og ministerier
- **[Tilstedeværelse](./attendance/)** - Opsætning af campusser, tjeneste gange, og spor tilstedeværelse
- **[Donationer](./donations/)** - Registrer givning, administrer fonde, generer sætninger
- **[Tjeneste](./serving/)** - Koordiner frivillige, opret tjenesteplaner, administrer opgaver
- **[Formularer](./forms/)** - Byg tilpassede formularer til registreringer og dataindsamling
- **[Rapporter](./reports/)** - Se fødselsdags-, tilstedeværelse- og donationsrapporter
- **[Websted](./website/)** - Byg og administrer dit kirkewebsted
- **[Prædikener](./sermons/)** - Administrer dit prædikenbibliotek og live streaming
- **[Kalendere](./calendars/)** - Opret og udgiv event kalendere
- **[Indstillinger](./settings/)** - Konfigurer kirke info, roller og tilladelser

:::info
Klik spørgsmålstegnet ikonet i øverste højre hjørne af enhver side for hurtig adgang til hjælp og dokumentation.
:::
