# kirkens indstillinger

<div class="article-intro">

kirkens indstillinger siden er hvor du konfigurér dit kirkes basis information, kontakt detaljer og branding. Disse detaljer bliver brugt på tværs af alle ChurchApps værktøjer, inkluderet dit B1.church websted og B1 Mobile app.

</div>

<div class="prereqs">
<h4>Før du begynder</h4>

- Du har brug for "rediger kirkens indstillinger" tilladelse. Se [roller og tilladelser](./roles-permissions.md) hvis du ikke har adgang.
- Har dine kirkes adresse, kontakt information og logo klar

</div>

## rediger dine kirkens information

1. Gå til **indstillinger** siden i venstre sidebar.
2. Klik **rediger indstillinger** knap i header.
3. opdater nogen af følgende felter:
   - **kirkenavn** -- navn vist på tværs af alle ChurchApps produkter.
   - **adresse** -- dit kirkes fysisk adresse.
   - **kontakt information** -- telefon nummer, e-mail og anden kontakt detaljer.
4. Klik **gem** for at anvende dine ændringer.

## opsætning af din underdomæne

Dit kirke får en gratis underdomæne på **dinkirke.b1.church**. Dette er web adressen hvor medlemmer og besøgende kan få adgang til dit kirkes online tilstedeværelse.

1. På indstillinger siden, lokalisér **underdomæne** felt.
2. Anføre dit ønskede underdomæne (for eksempel, "gracechurch" til gracechurch.b1.church).
3. gem dine ændringer.

:::info
dit underdomæne skal være unik på tværs af alle ChurchApps kirker. Hvis dit ønskede navn er taget, prøv at tilføje din by eller stat (for eksempel, "gracechurch-dallas").
:::

## konfigurering af branding

tilpas hvordan din kirke vises på tværs af alle ChurchApps værktøjer:

1. upload dit **kirkelogo** ved at klikke logo område og vælge en billedfil.
2. Tilføj ethvert yderligere **kirke billeder** brugt på dit websted og [mobilapp](./mobile-app.md).

:::tip
til de bedste resultater, brug et logo med transparent baggrund i PNG format. Dette sikrer det ser godt ud på både lyst og mørk baggrunde.
:::

## import og eksport

**import/eksport** knap i indstillinger header åbner et dedikeret værktøj i et nyt browser vindue. Brug det til:

- import medlemsdata fra en anden kirke administrationssystem.
- eksport dit ChurchApps data til backup eller migration formål.

Dette er særligt nyttigt når du først opsætning din kirke og skal overføre eksisterende registreringer ind i ChurchApps.

:::warning
når import af data, altid backup dine eksisterende registreringer først. import operationer tilføj data til dit system og kan opbygge duplikat poster hvis kørsel flere gange.
:::
