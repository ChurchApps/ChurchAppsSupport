# Donationer

<div class="article-intro">

Donationssektionen i B1 Admin giver alt hvad du har brug for at administrere din kirkes giving. Fra registrering af ugentlig offring til generering af år-slut momsopgørelser, alle dine donationsværktøjer er organiseret på ét sted, så du kan holde øje med din kirkes økonomi.

</div>

## Undersektioner

Du vil finde følgende sider i sidebjælken Donationer:

### Opsummering

Siden **Opsummering** giver dig et visuelt overblik over all giveraktivitet. Det viser diagrammer og filtrerede rapporter, så du hurtigt kan se donationstrends på tværs af din kirke. Se [Donationsrapporter](donation-reports.md) for mere detaljer om brug af opsummeringes- og rapportværktøjer.

### Batcher

**[Batcher](batches.md)** lader dig gruppere donationer sammen -- for eksempel alle checks og kontanter indsamlet under en søndag offring. Hver batch sporer sit donationsantal og samlet beløb for let afstemning.

### Fonde

**[Fonde](funds.md)** kategoriserer dine donationer efter formål. Almindelige eksempler inkluderer Generel Fond, Byggefondsgivende og Missions. Du kan markere hver fond som skattefradragsberettiget eller ej.

### Erklæringer

Siden **[Erklæringer](giving-statements.md)** genererer år-slut giverbekendtelser for dine givere. Du kan downloade individuelle CSV-filer eller udskrive alle erklæringer på engang.

### Stripe import

Hvis du accepterer onlinedonationer gennem Stripe, lader værktøjet **[Stripe Import](stripe-import.md)** dig trække disse transaktioner ind i B1 Admin, så alt er i ét system.

## Online Giving

B1 Admin integreres med **Stripe** og **PayPal** for at acceptere onlinedonationer. Når det er konfigureret, kan medlemmer give gennem dit B1.church-sted, og disse donationer vises automatisk i dine records. Se [Online Giving Setup](online-giving-setup.md) for at komme i gang.

:::tip
Hvis du er ny på donationer i B1 Admin, skal du starte med [opsætning af dine fonde](funds.md), derefter [oprette din første batch](batches.md) og [registrere donationer](recording-donations.md) ind i det.
:::
