# Administrer din profil

<div class="article-intro">

Profil siden lader dig redigere dine personlige kontindelstillinger. Din profil deles på tværs af alle ChurchApps produkter, så enhver ændringer du foretager her vil blive afspejlet overalt hvor du bruger ChurchApps.

</div>

<div class="prereqs">
<h4>Før du begynder</h4>

- Du skal være logget ind på din ChurchApps-konto
- Har din nye e-mail-adresse eller adgangskode klar hvis du planlægger at opdatere dem

</div>

## Profil oplysninger

Top sektionen på profil siden indeholder dine basis kontodetaljer:

1. **e-mail** -- Din e-mail-adresse bruges til at logge ind og modtage underretninger.
2. **fornavn** og **efternavn** -- Dit displaynavn på tværs af ChurchApps.
3. **adgangskode** -- Opdater din kontoadgangskode.

For at opdatere dine oplysninger, rediger felterne og klik **gem ændringer** knap.

## Ændrer din adgangskode

1. Rul ned til adgangskode feltene på profil siden.
2. Anføre din nye adgangskode i **adgangskode** felt.
3. Anføre den igen i **bekræft adgangskode** felt.
4. Klik **gem ændringer**.

:::info
Din adgangskode skal være mindst 8 tegn lang. Sørg for at både adgangskode og bekræftelse felter matcher præcis.
:::

## Forbundne konti

**Forbundne konti** sektion viser enhver eksterne tjenester forbundne til din ChurchApps-konto. Du kan se hvilke konti som er forbundne og administrere disse forbindelser fra denne sektion.

## tema præferencer

ChurchApps understøtter både lys og mørk display tilstande:

1. Find **tema** skifte på din profil siden.
2. Skift mellem **lys** og **mørk** tilstand.
3. Interfacet opdaterer sig øjeblikkelig for at afspejle dit valg.

Dit tema præference lagres til din konto og vil gælde på tværs af alle dine sessioner.

## kontosletning

I bunden af profil siden, vil du finde mulighed for permanent at slette din konto. Denne handling fjerner din konto og tilhørende data og kan ikke fortrylles.

:::warning
Konto sletning er permanent og kan ikke omvendt. Sørg for at du virkelig ønsker at fjerne din konto før du fortsætter.
:::

:::info
Hvis du er en **domæne admin**, sørg for at overføre administrative ansvarsområder til en anden bruger før du sletter din konto. Se [roller og tilladelser](../settings/roles-permissions.md) for detaljer på administrering af admin roller.
:::

:::info
Profil redigering er deaktiveret i demo tilstand. Hvis du bruger en demo konto, vil du ikke kunne foretage ændringer til dine profil indstillinger.
:::
