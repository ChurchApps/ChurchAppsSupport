# Profil

<div class="article-intro">

Profil sektionen lader dig administrere dine personlige kontindelstillinger i ChurchApps. Din profil er delt på tværs af alle ChurchApps værktøjer, så ændringer du gør her gælder overalt -- fra B1 Admin til B1 mobilappen.

</div>

## Hvad du kan gøre

Fra profilsektionen kan du:

- **opdater dit navn og e-mail** -- Hold dine kontaktoplysninger aktuelle.
- **Ændre din adgangskode** -- Sæt en ny adgangskode til din konto.
- **Skift temaer** -- skifte mellem lys og mørk tilstand for at matche dine præferencer.
- **Administrer forbundne enheder** -- Se og kontroller hvilke ChurchAppsPlayer enheder som er knyttet til din konto.
- **Se forbundne konti** -- Se og administrer enhver eksterne tjenester forbundne til din ChurchApps konto.

## profil undersektioner

Profil området har følgende sider:

1. **[administrer din profil](./managing-profile.md)** -- rediger dine personlige oplysninger, ændre din adgangskode, administrer forbundne konti, tilpas tema præferencer og håndter konto sletning.
2. **[administrer enheder](./devices.md)** -- Se, tilføj og administrer ChurchAppsPlayer enheder parret med din konto.

:::tip
Hold din e-mail-adresse opdateret. Den bruges til kontoindlevering og vigtige meddelelser fra din kirke.
:::
