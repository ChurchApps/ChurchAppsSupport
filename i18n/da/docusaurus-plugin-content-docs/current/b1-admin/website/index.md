# websted

<div class="article-intro">

websted sektionen i B1 Admin giver dig en fuld-udstyr websted bygger til dit kirke. Du kan opbygge sider, tilpas dine websted udsende og fornemmelse, administrer medie filer og udgiv event kalendere -- alt uden at skrive nogen kode.

</div>

## hvad du kan gøre

websted bygger er organiseret ind i fem hovedområder:

### sider

opbygge og administrer alle sider på dit kirkewebsted. **sider** tabel viser hver siden sammen med dets status -- enten **genererede** (automatisk opbygget af systemet) eller **brugerdefineret** (opbygget af dig). Du kan tilføje nye brugerdefinerede sider, rediger eksisterende eller konvertere auto-genererede sider til brugerdefinerede sider til fuld styre over deres indhold.

Venstre sidebar viser dine navigations links i træ struktur. Du kan omorden links ved hjælp af træg-og-slip for at indstille præcis hierarki og orden du ønsker besøgende at se.

### blokke

byg genbrug af indhold komponenter som du kan placere på flere sider. blokke sparer dig tid når du har indhold som opfordring-til-handling, meddelelse banner eller kontakt sektion som vises i mere end ét sted. opdater en blok engang og ændringen gælder overalt den bliver brugt.

### udseende

tilpas dit websteds tema, farver, skrifttyper, logo og mellemrum. Du kan anvende forudbyggede stil temaer til hurtig start eller fine-tune ethvert detalje med farve palet redaktør, typografi indstillinger og brugerdefineret CSS. Se [udseende](appearance) vejledning til fulde detaljer.

### filer

upload og organiser alle dine medie aktiver -- billeder, dokumenter og anden filer -- i ét centralt sted. Brug mapper for at holde ting ryddeligt og kopier fil URL'er til bruger overalt på dit websted. Se [filer](files) til mere information.

### kalendere

udgiv event kalendere på dit websted så besøgende kan se hvad som sker på dit kirke. opbygge kurateret kalendere som trække arrangementer fra flere grupper ind i enkelt visning.

:::tip
start med [oprindelig opsætning](initial-setup) hvis du er konfigurering dit websted til første gang. Det går dig gennem domæne opsætning, opbygning af dit første side og konfigurering af dit websteds udseende.
:::
