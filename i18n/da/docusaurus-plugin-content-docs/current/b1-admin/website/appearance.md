# udseende

<div class="article-intro">

udseende siden lader dig tilpas det samlede udsende og fornemmelse af dit kirkewebsted. fra farver og skrifttyper til mellemrum og brugerdefineret CSS, kan du styre ethvert visuelt aspekt af dit websted fra ét sted.

</div>

<div class="prereqs">
<h4>Før du begynder</h4>

- fuldført [oprindelig opsætning](initial-setup) til dit websted
- Har dit kirkelogo klar i PNG format med transparent baggrund og en 4:1 aspect ratio
- ved dit kirkes mærkningsfarver (hex værdier) hvis du har en eksisterende stilguide

</div>

## adgang til udseende indstillinger

1. I B1 Admin, klik **websted** i venstre menu.
2. Klik **udseende** fane øverst på websted sider visning.
3. websted stilene siden indlæser med live forhåndsvisning af dit websted på venstre og **stil indstillinger** muligheder på højre.

## farve palet

1. Klik **farve palet** i stil indstillinger panel.
2. Du vil se **basis farver** (lyst, accent og mørk skygger) og **semantisk farver** (primær, sekundær, succes, advarsel og fejl).
3. Klik ethvert farve prøvefelt for at åbne farve vælger. træg vælgeren eller anføre hex værdien for at vælge dit farve.
4. **farve kombinationer forhåndsvisning** viser hvordan dit valgte farver arbejde sammen.
5. Brug **foreslåede paletter** for at hurtigt anvende en foruddesignet farve skema.
6. Klik **gem** når du er tilfreds.

## typografi

1. Klik **typografi indstillinger** i stil indstillinger panel.
2. Klik **vælg font** for at åbne font browser. Du kan søgning efter navn eller gennemse kategorier som serif, sans serif, display, handwriting og monospace.
3. indstil skrifttyper til både overskrifter og brødtekst.
4. Klik **typografi skala** for at justere størrelse hierarki til overskrift 1 gennem overskrift 4. Brug skala multiplikator og basis størrelse felter til at fine-tune.
5. Klik **gem** for at anvende dine skrift valg.

## mellemrum

1. Klik **mellemrum skala** i stil indstillinger panel.
2. justér mellemrum værdier til ekstra lille gennem ekstra stor. praktisk eksempler vise hvordan hver værdier påvirker layout.
3. Klik **gem mellemrum** for at anvende værdierne på tværs af dit hele websted.

## logo og branding

1. Klik **logo** i stil indstillinger panel.
2. upload dit **lys baggrund logo** og **mørk baggrund logo**. Brug billeder med transparent baggrund og en 4:1 aspect ratio til de bedste resultater.
3. upload et **social medie billede** til link forhåndsvisninger og en **favicon** til browser tab ikon.

:::tip
til de bedste resultater, brug logo med transparent baggrund i PNG format. Dette sikrer det ser godt ud på både lyst og mørk baggrunde på tværs af dit websted og [mobilapp](../settings/mobile-app.md).
:::

## brugerdefineret CSS og javascript

1. Klik **CSS og javascript** i stil indstillinger panel.
2. Tilføj **brugerdefineret CSS** for at overskygge standard stilene til avanceret tilpasning.
3. Tilføj **brugerdefineret HTML** til sporing koder eller andet scripts.
4. Brug **almindelige javascript eksempler** sektion til kodebidder som Google analytik integration.

:::warning
brugerdefineret CSS er kraftigt men kan bryde dit websteds layout hvis brugt forkert. de fleste kirker kan opnå udseendet de ønsker ved hjælp af indbygget farve, skrift og mellemrum kontroller. brug kun brugerdefineret CSS hvis du er komfortabel med web udvikling.
:::

## stil temaer

Hvis du ønsker hurtig startpunkt, **foreslåede paletter** i farve palet sektion tilbyder forudbyggede temaer som sæt koordineret farver i ét klik. Du kan altid fine-tune individuelle indstillinger efter anvending af tema.
