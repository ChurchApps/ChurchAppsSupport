# Gestion des programmes

<div class="article-intro">

Les programmes sont les conteneurs de niveau supérieur pour les programmes d'enseignement dans Lessons.church. Un programme représente une série complète de programmes d'enseignement -- par exemple, « Ark Kids » ou « Youth Foundations ». Ce guide couvre la création, la modification et l'organisation des programmes dans l'espace d'administration.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Disposer des autorisations d'administrateur pour Lessons.church (voir la [vue d'ensemble de l'administration](./index.md))
- Préparer un nom de programme, une description et une image miniature

</div>

## Créer un programme

1. Naviguez vers l'espace **Admin**.
2. Cliquez sur **Add Program**.
3. Remplissez les champs suivants :
   - **Name** -- Le nom d'affichage du programme (par ex., « Ark Kids »).
   - **Slug** -- Un identifiant adapté aux URL (par ex., « ark-kids »). Celui-ci est utilisé dans les adresses web.
   - **Description** -- Un bref résumé du contenu du programme.
   - **Image** -- Téléversez une image miniature représentant le programme.
4. Enregistrez le programme.

Votre nouveau programme apparaîtra désormais dans la liste des programmes de l'administration et, une fois qu'il contiendra des études et des leçons, sur le site public.

## Modifier un programme

1. Cliquez sur le nom du programme dans la liste d'administration.
2. Mettez à jour l'un des champs (**Name**, **Slug**, **Description** ou **Image**).
3. Enregistrez vos modifications.

## Organiser les études au sein d'un programme

Une fois qu'un programme existe, vous pouvez y ajouter des études. Les études apparaissent imbriquées sous le programme dans la vue d'administration. Cliquez sur la flèche de développement à côté d'un programme pour voir ses études, ou ajoutez-en une nouvelle à partir de là. Voir [Gestion des études](./managing-studies.md) pour plus de détails.

## Catégories d'études

Si votre programme comporte de nombreuses études, vous pouvez les organiser en **catégories**. Les catégories regroupent les études par thème, tranche d'âge ou tout autre critère de votre choix. Cela facilite la navigation pour les utilisateurs au sein d'un programme volumineux.

## Téléverser des ressources au niveau du programme

Vous pouvez joindre des fichiers téléchargeables à un programme. Ce sont des ressources qui s'appliquent au programme dans son ensemble plutôt qu'à une étude ou une leçon spécifique -- par exemple, un document de planification annuelle ou un guide du bénévole.

1. Ouvrez le programme dans la vue d'administration.
2. Naviguez vers la section **Resources**.
3. Téléversez vos fichiers.

## Vider le cache

Après avoir apporté des modifications à un programme, vous devrez peut-être vider le cache pour que les mises à jour apparaissent sur le site public. Utilisez l'option **Clear Cache** dans l'espace d'administration pour actualiser le contenu.

:::tip
Choisissez un nom de programme descriptif et rédigez une description claire. C'est la première chose que les églises voient lorsqu'elles parcourent la bibliothèque de programmes d'enseignement, donc un bon résumé les aide à décider si le programme convient à leurs besoins.
:::
