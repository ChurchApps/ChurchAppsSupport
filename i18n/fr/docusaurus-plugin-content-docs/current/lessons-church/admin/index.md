# Administration de Lessons.church

<div class="article-intro">

Administrez votre instance de Lessons.church pour gérer les salles de classe, les enseignants et les leçons.

</div>

## Sections d'administration

- **Salles de classe** — Créez et gérez les salles de classe
- **Enseignants** — Ajoutez et gérez les comptes des enseignants
- **Fournisseurs de contenu** — Connectez les sources de leçons
- **Rapports** — Consultez l'activité et le suivi des étudiants

## Gestion des salles de classe

1. Allez à **Salles de classe**
2. Cliquez sur **Ajouter une classe**
3. Entrez les détails de la classe :
   - Nom de la classe
   - Niveau d'âge
   - Enseignant affecté
4. Cliquez sur **Enregistrer**
