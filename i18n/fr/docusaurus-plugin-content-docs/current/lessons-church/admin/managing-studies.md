# Gestion des études

<div class="article-intro">

Une étude est une série de leçons connexes au sein d'un programme. Par exemple, une série de quatre semaines intitulée « Le Bon Samaritain » serait une étude au sein d'un programme plus vaste de ministère pour enfants. Ce guide couvre la création, la modification et l'organisation des études dans l'espace d'administration.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Créer au moins un programme](./managing-programs.md) dans l'espace d'administration
- Disposer des autorisations d'administrateur pour Lessons.church (voir la [vue d'ensemble de l'administration](./index.md))

</div>

## Créer une étude

1. Naviguez vers l'espace **Admin** et développez le programme auquel vous souhaitez ajouter une étude.
2. Cliquez sur **Add Study**.
3. Remplissez les champs suivants :
   - **Name** -- Le nom d'affichage de l'étude (par ex., « Le Bon Samaritain »).
   - **Slug** -- Un identifiant adapté aux URL (par ex., « the-good-samaritan »).
   - **Description** -- Un résumé du contenu de cette étude et du nombre de leçons qu'elle comprend.
   - **Image** -- Téléversez une image miniature pour l'étude.
4. Enregistrez l'étude.

## Modifier une étude

1. Cliquez sur le nom de l'étude dans la liste d'administration.
2. Mettez à jour l'un des champs selon vos besoins.
3. Enregistrez vos modifications.

## Attribuer une catégorie

Si le programme parent utilise des catégories d'études, vous pouvez attribuer votre étude à l'une d'entre elles :

1. Ouvrez l'étude dans la vue d'administration.
2. Sélectionnez une **Category** dans le menu déroulant.
3. Enregistrez.

Les catégories aident les utilisateurs à parcourir les programmes qui comportent de nombreuses études. Par exemple, vous pourriez avoir des catégories comme « Ancien Testament », « Nouveau Testament » et « Thématique ».

## Gérer les leçons au sein d'une étude

Les études contiennent une ou plusieurs leçons. Développez une étude dans la vue d'administration pour voir ses leçons, ou cliquez sur **Add Lesson** pour en créer une nouvelle. Voir [Gestion des leçons](./managing-lessons.md) pour plus de détails sur la construction du contenu des leçons.

## Téléverser des ressources au niveau de l'étude

Vous pouvez joindre des fichiers téléchargeables qui s'appliquent à l'ensemble de l'étude -- par exemple, un aperçu de la série, des visuels promotionnels ou un modèle de bulletin pour les parents.

1. Ouvrez l'étude dans la vue d'administration.
2. Naviguez vers la section **Resources**.
3. Téléversez vos fichiers.

:::tip
Rédigez la description de votre étude en pensant aux responsables d'église. Un résumé clair du thème, de la tranche d'âge ciblée et du nombre de leçons aide les églises à décider si elles souhaitent utiliser cette série.
:::
