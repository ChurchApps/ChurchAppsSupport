# Configurer les salles de classe

<div class="article-intro">

Avant de pouvoir planifier des leçons, vous devez créer des salles de classe qui représentent vos environnements d'enseignement. Ce guide vous accompagne dans la création, la dénomination et la gestion des salles de classe dans le portail Lessons.church.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Créez un compte](../getting-started/creating-account.md) et connectez-vous à votre église
- Familiarisez-vous avec l'[Aperçu du portail](../getting-started/portal-overview.md)

</div>

## Créer une salle de classe

1. Connectez-vous à votre compte sur [Lessons.church](https://lessons.church).
2. Accédez à votre **Portail**. C'est le centre de gestion de votre église pour administrer les salles de classe et les calendriers.
3. Cliquez sur **Ajouter une salle de classe**.
4. Entrez un nom pour votre salle de classe. Choisissez quelque chose de descriptif que vos bénévoles reconnaîtront -- par exemple, « Préscolaire », « Élémentaire CP-CE1 » ou « Collège ».
5. Enregistrez votre nouvelle salle de classe. Elle apparaîtra immédiatement sur le tableau de bord de votre portail.

Répétez ces étapes pour chaque salle ou groupe d'âge de votre église. Vous pouvez créer autant de salles de classe que nécessaire.

## Configuration initiale

Si vous n'avez pas encore configuré de salle de classe, votre portail affichera une vidéo de configuration pour vous guider dans le processus. Cette vidéo couvre les bases de la création de salles de classe et de la planification de vos premières leçons. C'est un excellent point de départ si vous êtes nouveau sur Lessons.church.

:::tip
Nommez vos salles de classe en fonction de la façon dont vos bénévoles pensent déjà à vos salles. Si tout le monde l'appelle « la salle des petits », nommez-la « Salle des petits » plutôt que quelque chose de formel comme « Salle d'éducation de la petite enfance 101 ».
:::

## Gérer les salles de classe

Vos salles de classe apparaissent sur le tableau de bord du portail où vous pouvez les gérer à tout moment. De là, vous pouvez :

- **Renommer une salle de classe** -- Mettre à jour le nom si vos salles ou groupes d'âge changent.
- **Supprimer une salle de classe** -- Supprimer les salles de classe dont vous n'avez plus besoin.
- **Consulter le calendrier** -- Cliquer sur n'importe quelle salle de classe pour voir son calendrier de leçons.

## Et ensuite

Une fois vos salles de classe configurées, vous êtes prêt(e) à commencer à [planifier des leçons](./scheduling-lessons.md) afin que vos bénévoles sachent quoi enseigner chaque semaine.
