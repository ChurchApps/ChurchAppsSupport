# Aperçu du portail

<div class="article-intro">

Après vous être connecté(e) à Lessons.church, vous accédez au **Portail** -- le tableau de bord de votre église pour gérer les salles de classe et les calendriers de leçons. C'est ici que vous passerez la plupart de votre temps lorsque vous organiserez le programme d'enseignement de votre église.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Créez un compte](./creating-account.md) et connectez-vous à votre église
- Connectez-vous à [Lessons.church](https://lessons.church)

</div>

## Ce que vous voyez

Le portail est organisé en deux zones principales :

- **Salles de classe** (côté droit) -- Une liste des salles de classe de votre église. Cliquez sur le nom d'une salle de classe pour consulter ses leçons planifiées. Vous pouvez également ajouter de nouvelles salles de classe ou modifier celles existantes depuis cet endroit.
- **Leçons planifiées** (côté gauche) -- Une fois que vous sélectionnez une salle de classe, cette zone affiche les leçons qui ont été planifiées pour celle-ci. Si vous n'avez pas encore configuré de salle de classe, vous verrez une vidéo d'instructions de configuration pour vous aider à démarrer.

## Ce que vous pouvez faire depuis le portail

Depuis le portail, vous pouvez :

1. **Gérer les salles de classe** -- Ajouter, modifier ou supprimer des salles de classe. Chaque salle de classe peut être liée à une équipe de bénévoles (afin qu'ils voient les leçons à venir dans l'application B1) et à un groupe parents/élèves (afin qu'ils puissent accéder aux leçons récentes).
2. **Consulter et gérer les leçons planifiées** -- Voir quelles leçons sont à venir pour chaque salle de classe et effectuer des modifications si nécessaire.
3. **S'abonner aux flux des salles de classe** -- Cliquez sur l'icône de flux à côté d'une salle de classe pour obtenir une URL de flux de playlist que vous pouvez utiliser avec des applications compatibles.
4. **Accéder aux fournisseurs externes** -- Utilisez la page **Fournisseurs externes** pour configurer des sources de contenu tierces qui prennent en charge l'Open Lesson Format. Cela vous permet d'intégrer des leçons d'autres fournisseurs au-delà de la bibliothèque intégrée de Lessons.church.
5. **Accéder à la zone d'administration** -- Si vous disposez des autorisations d'administration, vous verrez une option **Admin** dans la navigation. La zone d'administration vous permet de gérer les programmes, études et leçons au niveau du contenu.

## Navigation

La navigation du portail comprend :

- **Accueil** -- Retourner à la page d'accueil publique de Lessons.church pour parcourir le contenu.
- **Calendriers** -- La vue principale du portail pour gérer les salles de classe et les calendriers de leçons.
- **Fournisseurs externes** -- Configurer les fournisseurs de leçons tiers.
- **Admin** -- Gérer les programmes, études et contenu des leçons (visible uniquement si vous avez les autorisations d'administration).

:::info
L'option **Admin** n'apparaît que si votre compte a reçu les autorisations d'édition de contenu. La plupart du personnel de l'église utilisera les pages **Calendriers** et **Fournisseurs externes** pour la gestion quotidienne.
:::

## Pour commencer

Si vous visitez le portail pour la première fois et n'avez pas encore créé de salle de classe, commencez par cliquer sur **Ajouter une première salle de classe** dans le panneau des salles de classe. Donnez un nom à votre salle de classe (par exemple, « Préscolaire » ou « CE2-CM2 ») et enregistrez-la. À partir de là, vous pouvez commencer à [planifier des leçons](../classrooms/scheduling-lessons.md).
