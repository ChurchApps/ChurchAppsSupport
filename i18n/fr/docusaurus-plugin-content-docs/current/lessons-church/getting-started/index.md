# Premiers pas

<div class="article-intro">

Lessons.church est entièrement gratuit. Il n'y a pas d'abonnement, pas de période d'essai et pas de limitation de fonctionnalités. Cette section vous guide dans la création de votre compte, la connexion de votre église et la navigation dans le portail afin que vous puissiez commencer à planifier des leçons immédiatement.

</div>

## Ce dont vous avez besoin

Pour commencer avec Lessons.church, vous allez :

1. **Créer un compte** -- Inscrivez-vous sur [Lessons.church](https://lessons.church) avec votre nom, votre adresse courriel et un mot de passe. Cela utilise le même compte ChurchApps que celui que vous possédez peut-être déjà via B1.church.
2. **Connecter votre église** -- Après votre inscription, connectez-vous à votre profil d'église existant ou créez-en un nouveau. Cela lie vos salles de classe et vos plannings à votre église.
3. **Configurer les salles de classe** -- Créez des salles de classe qui correspondent à votre ministère (par exemple, « Préscolaire », « CP-CE2 » ou « Jeunesse »). Les salles de classe sont l'endroit où vous planifiez et organisez les leçons.
4. **Commencer à planifier des leçons** -- Parcourez la bibliothèque de contenu, trouvez une étude qui correspond à vos besoins et planifiez des leçons pour vos salles de classe.

Une fois que vos salles de classe sont configurées et que les leçons sont planifiées, vos bénévoles peuvent accéder au contenu via le portail et présenter les leçons à leurs classes.

:::tip
Si vous avez déjà un compte ChurchApps via B1.church, vous pouvez vous connecter à Lessons.church avec les mêmes identifiants -- pas besoin de vous inscrire à nouveau.
:::
