# Programmes et études

<div class="article-intro">

Tous les programmes d'enseignement sur Lessons.church sont organisés selon une hiérarchie claire. Comprendre cette structure vous aidera à trouver rapidement le bon contenu pour votre église et à naviguer facilement entre les leçons connexes.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Familiarisez-vous avec la vue d'ensemble [Parcourir le contenu](./index.md) pour comprendre les différentes façons de trouver du contenu

</div>

## Hiérarchie du contenu

Le contenu est organisé du plus général au plus spécifique :

- **Fournisseur** -- L'organisation qui publie le programme d'enseignement (Lessons.church ou un fournisseur tiers).
- **Programme** -- Une série de programmes d'enseignement, telle que « Ark Kids » ou « NextLevel ». Chaque programme est un ensemble d'études connexes conçues autour d'un thème ou d'une tranche d'âge.
- **Étude** -- Une unité au sein d'un programme, telle que « Le Bon Samaritain ». Chaque étude contient un ensemble de leçons destinées à être enseignées ensemble, souvent sur plusieurs semaines.
- **Leçon** -- Une session d'enseignement individuelle au sein d'une étude. Chaque leçon comprend des sections, des rôles, des actions et du matériel téléchargeable.
- **Lieu** -- Une version d'une leçon adaptée à une tranche d'âge, telle que Maternelle ou Primaire. Les lieux vous permettent d'adapter la même leçon à différents publics.

## Parcourir les programmes

Pour explorer les programmes :

1. Visitez la page d'accueil de Lessons.church.
2. Faites défiler la liste des programmes disponibles ou regardez les programmes en vedette en haut de la page.
3. Cliquez sur le **nom d'un programme** ou sur son image pour ouvrir la page du programme.

Chaque page de programme comprend :

- **Description** -- Un aperçu du contenu du programme d'enseignement et du public auquel il est destiné.
- **Image du programme** -- Une identité visuelle pour la série.
- **Bande-annonce vidéo** -- Certains programmes incluent une courte prévisualisation vidéo pour que vous puissiez voir le style et la qualité du contenu avant de vous engager.
- **Études** -- Une liste de toutes les études du programme, qui peuvent être organisées en catégories.

## Parcourir les études

Depuis une page de programme :

1. Parcourez la liste des études. Si le programme organise les études en catégories, vous les verrez regroupées en conséquence.
2. Cliquez sur le **nom d'une étude** pour ouvrir la page de l'étude.
3. La page de l'étude affiche toutes les leçons incluses dans cette étude, listées dans l'ordre d'enseignement.

De là, vous pouvez cliquer sur n'importe quelle leçon pour voir son contenu complet, y compris les sections, les rôles et le matériel téléchargeable. Voir [Consulter les leçons](./viewing-lessons.md) pour une présentation détaillée.

:::info
Les programmes et les études sont gérés par les fournisseurs de programmes d'enseignement. Si vous êtes un créateur de contenu, vous pouvez gérer vos propres programmes et études via l'espace [Administration](../admin/index.md).
:::
