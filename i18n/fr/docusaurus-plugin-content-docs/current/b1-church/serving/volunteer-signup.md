# Inscription des bénévoles

<div class="article-intro">

L'inscription des bénévoles vous permet de parcourir les positions ouvertes à votre église et de vous inscrire pour servir. C'est un moyen facile de contribuer sans avoir besoin d'attendre une affectation directe.

</div>

## Parcourir les positions

1. Sur B1.church, accédez à la section **Service**
2. Vous verrez les positions ouvertes organisées par catégorie
3. Chaque position affiche :
   - Le **nom de la position**
   - Le **nombre de places ouvertes**
   - La **date de service**
   - La **description de la position** (si fournie)

## S'inscrire à une position

1. Cliquez sur la position pour laquelle vous souhaitez vous inscrire
2. Consultez les détails et la description
3. Cliquez sur **Inscrivez-vous**
4. Confirmez votre inscription

Vous verrez une confirmation indiquant que vous êtes inscrit.

## Gérer vos inscriptions

1. Allez à la section **Service**
2. Votre liste d'inscriptions affiche toutes les positions pour lesquelles vous vous êtes auto-inscrit
3. Pour annuler une inscription, cliquez sur la position et sélectionnez **Annuler l'inscription**

:::warning
Vous ne pouvez annuler une inscription que avant la date limite définie par votre administrateur d'église.
:::
