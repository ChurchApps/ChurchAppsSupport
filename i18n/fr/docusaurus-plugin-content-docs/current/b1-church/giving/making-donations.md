# Faire un don

<div class="article-intro">

Il existe deux façons de faire un don via le site B1.church de votre église : la page de don publique (aucune connexion requise) et la page de don membre (connexion requise). Les deux utilisent un formulaire de don sécurisé propulsé par Stripe.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Pour les dons publics, aucun compte n'est nécessaire -- il suffit de visiter la page **Donner** de votre église.
- Pour les dons membre avec suivi de l'historique et moyens de paiement enregistrés, [connectez-vous](../getting-started/logging-in.md) à votre compte d'abord.

</div>

## Page de don publique (aucune connexion requise)

N'importe qui peut faire un don sans créer de compte :

1. Accédez à la page **Donner** de votre église.
2. Le formulaire de don s'affiche sur le côté gauche de la page. Sélectionnez le **fonds** auquel vous souhaitez donner.
3. Entrez vos informations de paiement et le montant que vous souhaitez donner.
4. Soumettez le formulaire pour compléter votre don.

Sur le côté droit de la page, vous verrez un lien vers les **Instructions pour le paiement ACH** si vous préférez donner par virement bancaire. Il y a également un bouton **Connexion** si vous souhaitez vous connecter pour accéder à des fonctionnalités supplémentaires comme l'historique des dons.

:::tip
Si vous êtes déjà connecté lorsque vous visitez la page de don publique, vous serez automatiquement redirigé vers la page de don membre où vous avez accès à votre historique complet de dons et à vos moyens de paiement enregistrés.
:::

## Page de don membre (connecté)

Lorsque vous êtes connecté, l'expérience de don inclut des fonctionnalités supplémentaires :

1. Cliquez sur l'onglet **Donner** (ou naviguez vers **Mes dons**).
2. Vous verrez un panneau avec trois sections :
   - **Donner maintenant** -- Faites un don ponctuel ou récurrent.
   - **Historique** -- Consultez vos dons passés et exportez les relevés.
   - **Dons récurrents** -- Gérez les dons programmés et les moyens de paiement.
3. Sélectionnez **Donner maintenant** pour ouvrir le formulaire de don.
4. Choisissez le **fonds** auquel vous souhaitez donner et entrez le montant de votre don.
5. Sélectionnez un moyen de paiement enregistré existant ou entrez de nouvelles informations de paiement.
6. Soumettez le formulaire pour compléter votre don.

Après le traitement de votre don, un message de succès apparaîtra et votre don s'affichera dans votre [historique des dons](./donation-history.md).

## Moyens de paiement

Les dons sont traités de manière sécurisée via **Stripe**. Vous pouvez payer par carte de crédit, carte de débit ou configurer un paiement bancaire ACH. Lorsque vous êtes connecté, vos moyens de paiement peuvent être enregistrés pour une utilisation future, rendant vos prochains dons plus rapides.

:::warning
Si votre don ne passe pas, vérifiez vos informations de paiement et réessayez. Si le problème persiste, contactez le secrétariat de votre église pour obtenir de l'aide.
:::
