# Communauté

<div class="article-intro">

La section **Communauté** dans B1.church vous aide à rester en contact avec votre famille d'église. Elle rassemble un annuaire des membres pour trouver et en apprendre davantage sur les autres membres de l'église, ainsi qu'un fil d'actualité qui vous tient informé de ce qui se passe dans vos groupes et votre église.

</div>

## Ce que vous y trouverez

- **[Annuaire des membres](./member-directory.md)** -- Parcourez et recherchez dans l'annuaire des membres de votre église. Cherchez des personnes par nom ou filtrez par groupe pour trouver qui vous cherchez. Cliquez sur un membre pour voir les détails de son profil.
- **[Fil d'actualité](./timeline.md)** -- Votre flux d'activité affichant les dernières mises à jour de vos groupes et de votre église. Le fil d'actualité est la page d'accueil par défaut lorsque vous visitez le portail membre.

## Pour commencer

Vous devez être [connecté](../getting-started/logging-in.md) pour accéder aux fonctionnalités communautaires. Si vous n'êtes pas connecté, vous serez invité à vous connecter lorsque vous naviguerez vers l'onglet **Communauté** ou le **Fil d'actualité**.

L'onglet **Fil d'actualité** est toujours disponible dans la barre latérale et sert de page d'accueil de votre portail membre. L'onglet **Communauté** donne accès à l'annuaire des membres.

:::info
Les membres et les informations visibles pour vous dans l'annuaire dépendent des paramètres de confidentialité de votre église. Vous ne pourrez peut-être pas voir le profil complet de chaque membre. Si vous avez des questions sur ce qui est partagé, contactez l'administrateur de votre église.
:::

:::tip
Utilisez les fonctionnalités communautaires en complément des [Groupes](../groups/) pour rester engagé avec votre église tout au long de la semaine.
:::
