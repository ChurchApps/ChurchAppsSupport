# Événements

<div class="article-intro">

La section Événements sur B1.church vous permet de parcourir les événements d'église à venir et de vous inscrire directement à partir du site Web. Lorsque votre église active l'enregistrement pour un événement, vous pouvez vous inscrire vous et les membres de votre famille en quelques étapes.

</div>

## Navigation dans les événements

Les événements apparaissent sur le calendrier de l'église. Quand vous ouvrez un événement pour lequel l'enregistrement est activé, un bouton **Inscrivez-vous à cet événement** s'affiche aux côtés des détails de l'événement.

Les événements pour lesquels l'enregistrement est activé affichent :

- **Titre et description de l'événement** — De quoi parle l'événement.
- **Date et heure** — Quand l'événement a lieu.
- **Bouton d'enregistrement** — Un bouton bleu qui vous mène au formulaire d'enregistrement.

## Statut de l'enregistrement

Les événements peuvent afficher des statuts différents selon le timing et la capacité :

- **Ouvert** — L'enregistrement est disponible et vous pouvez vous inscrire.
- **Pas encore ouvert** — La fenêtre d'enregistrement n'a pas commencé. Revenez après la date d'ouverture.
- **Fermé** — La date limite d'enregistrement a passé.
- **Complet** — Tous les spots disponibles ont été pris.

:::tip
Si un événement est complet, revenez plus tard -- des places peuvent s'ouvrir si quelqu'un annule son enregistrement.
:::

## Ce dont vous avez besoin

- Un compte B1.church (ou vous pouvez vous enregistrer en tant qu'invité)
- Accès au site Web B1.church de votre église
