# S'inscrire pour les événements

<div class="article-intro">

Lorsque votre église active l'enregistrement pour un événement, vous pouvez vous inscrire via un simple processus étape par étape. Vous pouvez vous inscrire et ajouter des membres de la famille, que vous ayez un compte ou que vous visitiez en tant qu'invité. Pour les événements payants, vous choisissez un type d'assistant, ajoutez des suppléments optionnels, appliquez un code de réduction et payez de manière sécurisée à la fin.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Votre église doit avoir l'enregistrement des événements activé pour l'événement
- La fenêtre d'enregistrement doit être ouverte (entre les dates d'ouverture et de fermeture définies par l'administrateur)

</div>

## Étape 1 : Entrez vos informations

1. À partir du calendrier de l'église, ouvrez l'événement auquel vous souhaitez vous inscrire et cliquez sur **Register for this Event**.
2. La page d'enregistrement se charge en affichant le titre, la description et la date de l'événement.
3. Si vous êtes **connecté**, le formulaire affiche votre nom et confirme que vous vous inscrivez sous votre compte.
4. Si vous êtes un **invité** (non connecté), remplissez :
   - **Prénom** (requis)
   - **Nom de famille** (requis)
   - **E-mail** (requis)
   - **Téléphone** (optionnel)
5. Si l'événement a des types d'assistants (par exemple, Adulte et Enfant à des prix différents), choisissez **votre type d'enregistrement**. Chaque type affiche son prix, et les types épuisés affichent **Sold out**.

:::info
Si l'événement a une limite de capacité, une barre de progression indique combien de places restent. Si l'événement est complet et votre église a activé une liste d'attente, vous pouvez toujours terminer l'inscription pour rejoindre la liste d'attente, consultez [Rejoindre une liste d'attente](#rejoindre-une-liste-dattente) ci-dessous.
:::

## Étape 2 : Ajouter des membres de la famille

1. Si vous souhaitez inscrire des membres supplémentaires de la famille, entrez leur **Prénom** et leur **Nom de famille** pour chaque personne.
2. Si l'événement a des types d'assistants, choisissez un type pour chaque personne que vous ajoutez.
3. Cliquez sur **Add Member** pour ajouter plus de personnes (jusqu'à 10 au total).
4. Cliquez sur le bouton supprimer à côté de tout membre pour le retirer de l'enregistrement.
5. Si vous vous inscrivez uniquement vous-même, vous pouvez ignorer cette étape.

## Étape 3 : Choisir les options et les sélections (si proposées)

Certains événements offrent des compléments optionnels payants : T-shirts, repas, améliorations d'activité.

1. Si l'événement a des sélections, une étape **Options & Selections** apparaît.
2. Utilisez les graduateurs de quantité pour choisir le nombre de chaque élément que vous souhaitez. Les options gratuites affichent **Free** et les options limitées indiquent le nombre restant.
3. Cliquez sur **Continue**.

## Étape 4 : Répondre aux questions (si demandé)

Certains événements demandent aux inscrits de répondre à quelques questions supplémentaires : restrictions alimentaires, taille de T-shirt, contact d'urgence et détails similaires.

1. Si l'événement a des questions attachées, une étape **Questions** apparaît ensuite.
2. Remplissez vos réponses et continuez.

Si l'événement n'a pas de questions attachées, cette étape est automatiquement ignorée.

## Étape 5 : Payer (Événements payants)

Si votre enregistrement a un coût, l'étape **Payment** affiche un **Résumé de la commande** listant chaque assistant, chaque sélection, toute réduction et le **Total**.

- **Code de réduction** -- Si votre église vous a donné un code, entrez-le et cliquez sur **Apply**. La réduction est reflétée immédiatement, et vous verrez un message clair si le code a expiré, n'a pas encore commencé ou nécessite plus d'assistants.
- Les **membres connectés** peuvent payer avec une carte enregistrée ou choisir **Use a new card**.
- Les **invités** entrent directement les détails de leur carte.

Cliquez sur **Pay & Register** pour compléter le paiement et l'enregistrement en une seule étape. Les détails de votre carte vont directement au fournisseur de paiement de l'église et ne sont jamais stockés par B1.

Les enregistrements gratuits ignorent cette étape et vont directement à la confirmation.

## Étape 6 : Confirmer

Un message de confirmation apparaît avec les détails de l'événement et une liste de tous les utilisateurs auxquels vous vous êtes inscrit. Vous recevrez également un **e-mail de confirmation** à l'adresse e-mail associée à votre compte (ou l'e-mail que vous avez fourni en tant qu'invité).

## Rejoindre une liste d'attente

Si l'événement est complet et votre église a activé la liste d'attente, la première étape affiche : *« Cet événement est complet. Vous pouvez rejoindre la liste d'attente et nous vous notifierons si une place se libère. »* Complétez l'inscription comme d'habitude, le paiement est ignoré tant que vous êtes sur la liste d'attente. Quand une place se libère, vous recevez un e-mail ; si un paiement est dû, il comprend un lien pour le compléter. Vous pouvez également vérifier votre statut et payer à partir de [Mes inscriptions](./my-registrations).

## Modification ou annulation d'une enregistrement

Si vous êtes connecté, vous pouvez modifier ou annuler votre enregistrement vous-même à partir de la page **Registrations** du portail des membres : consultez [Mes inscriptions](./my-registrations). Les invités doivent contacter le bureau de l'église, et le personnel peut apporter la modification à partir de la page d'enregistrements de B1 Admin.
