# Leçons

<div class="article-intro">

L'onglet **Leçons** vous donne accès aux programmes et supports d'apprentissage de votre église. Le contenu est fourni via la plateforme Lessons.church et est intégré directement dans B1.church, vous n'avez donc pas besoin de naviguer vers un site séparé.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous devez être [connecté](../getting-started/logging-in.md) à votre compte B1.church. L'onglet Leçons se trouve dans l'espace membre et nécessite une authentification.
- Votre église doit avoir configuré un programme via Lessons.church.

</div>

## Accéder aux leçons

1. Connectez-vous à votre portail membre B1.church.
2. Cliquez sur l'onglet **Leçons** dans la [navigation membre](../getting-started/navigating.md).
3. Le contenu Lessons.church se charge directement dans la page.

Votre connexion est transférée automatiquement lorsque vous ouvrez l'onglet Leçons. Vous n'avez pas besoin de vous connecter séparément à Lessons.church.

## Ce que vous pouvez faire

Une fois dans l'onglet Leçons, vous pouvez parcourir le programme que votre église a mis à disposition. Selon ce que votre église a configuré, vous pouvez trouver :

- **Plans de leçons** -- Plans structurés avec des sections et des activités
- **Présentations et médias** -- Diapositives, vidéos et images accompagnant les leçons
- **Ressources téléchargeables** -- Fichiers et compléments pour approfondir l'étude

Vous pouvez cliquer sur les sections individuelles d'une leçon pour voir leur contenu dans une fenêtre de dialogue. Utilisez le bouton **Retour** pour revenir à la liste des sections, ou **Fermer** pour quitter la fenêtre de dialogue.

:::tip
Les leçons partagées avec vos groupes apparaîtront également dans votre [Fil d'actualité](../community/timeline.md), ce qui facilite la recherche du programme assigné sans naviguer directement vers l'onglet Leçons.
:::

## Disponibilité

L'onglet Leçons apparaît lorsque votre église a configuré un programme via Lessons.church. Si vous ne voyez pas d'onglet **Leçons** dans votre navigation membre, votre église n'a peut-être pas de contenu de leçons configuré. Contactez l'administrateur de votre église pour plus d'informations.
