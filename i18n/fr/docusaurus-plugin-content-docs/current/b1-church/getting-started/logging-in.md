# Se connecter

<div class="article-intro">

Se connecter à B1.church vous donne accès au portail membre où vous pouvez consulter votre fil d'actualité, gérer vos groupes, donner en ligne et utiliser tous les outils que votre église a mis en place. Cet article explique comment se connecter, naviguer dans votre menu utilisateur et se déconnecter.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin d'un compte ChurchApps. Consultez [Pour commencer](../getting-started/) pour en créer un.
- Vous avez besoin de l'URL B1.church de votre église (par exemple, `https://nomdevotreeglise.b1.church`).

</div>

## Comment se connecter

1. Rendez-vous sur l'URL B1.church de votre église (par exemple, `https://nomdevotreeglise.b1.church`).
2. Cliquez sur le bouton **Connexion** dans le coin supérieur droit de l'en-tête.
3. Entrez votre **adresse e-mail** et votre **mot de passe**.
4. Cliquez sur **Se connecter**.

Après la connexion, vous serez redirigé vers la page sur laquelle vous étiez avant de cliquer sur connexion. Si vous aviez navigué directement vers une page du portail membre alors que vous étiez déconnecté, vous serez automatiquement redirigé vers cette page après la connexion.

## Accéder au portail membre

Une fois connecté, vous pouvez accéder au portail membre en cliquant sur votre **nom** dans le coin supérieur droit de la page et en sélectionnant **Portail membre** dans le menu. Le portail membre est l'endroit où se trouvent tous vos outils d'église personnels, y compris votre [fil d'actualité](../community/timeline.md), vos [groupes](../groups/), vos [dons](../giving/), et plus encore.

## Votre menu utilisateur

Après la connexion, votre nom apparaît sous forme de badge dans le coin supérieur droit. Cliquez dessus pour ouvrir un menu avec les options suivantes :

- **Portail membre** -- Accédez au tableau de bord de votre portail membre personnel.
- **Modifier le profil** -- Mettez à jour les paramètres de votre compte ou modifiez votre profil d'église.
- **Déconnexion** -- Déconnectez-vous de votre compte.

## Accès administrateur

Si vous êtes administrateur d'église avec les permissions appropriées, vous verrez également une option **Portail administrateur** dans votre menu utilisateur. Cliquer dessus vous amène à B1 Admin où vous pouvez gérer les paramètres, les personnes, les groupes et le contenu de votre église.

:::tip
Si vous avez oublié votre mot de passe, utilisez le lien **Mot de passe oublié** sur la page de connexion pour le réinitialiser. Vous recevrez un e-mail avec les instructions pour créer un nouveau mot de passe.
:::

## Se déconnecter

Pour vous déconnecter, cliquez sur votre **nom** dans le coin supérieur droit et sélectionnez **Déconnexion**. Vous serez déconnecté et redirigé vers le site public.
