# Connexion

<div class="article-intro">

La connexion à B1.church vous donne accès au portail des membres où vous pouvez consulter votre chronologie, gérer les groupes, faire des dons en ligne et utiliser tous les outils que votre église a mis en place. Cet article explique comment vous connecter, naviguer dans votre menu utilisateur et vous déconnecter.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin d'un compte ChurchApps. Consultez [Getting Started](../getting-started/) pour en créer un.
- Vous avez besoin de l'URL B1.church de votre église (par exemple, `https://yourchurchname.b1.church`).

</div>

## Comment se connecter

1. Allez à l'URL B1.church de votre église (par exemple, `https://yourchurchname.b1.church`).
2. Cliquez sur le bouton **Login** dans le coin supérieur droit de l'en-tête.
3. Entrez votre **adresse e-mail** et **mot de passe**.
4. Cliquez sur **Sign In**.

Après la connexion, vous serez ramené à la page sur laquelle vous étiez avant de cliquer sur la connexion. Si vous avez accédé directement à une page du portail des membres alors que vous étiez déconnecté, vous serez automatiquement renvoyé à cette page après la connexion.

## Accès au portail des membres

Une fois connecté, vous pouvez accéder au portail des membres en cliquant sur votre **nom** dans le coin supérieur droit de la page et en sélectionnant **Member Portal** dans le menu. Le portail des membres est l'endroit où tous vos outils personnels de l'église vivent, y compris votre [chronologie](../community/timeline.md), [groupes](../groups/), [dons](../giving/), et plus.

## Votre menu utilisateur

Après la connexion, votre nom apparaît comme une puce dans le coin supérieur droit. Cliquez dessus pour ouvrir un menu avec les options suivantes :

- **Member Portal** -- Allez à votre tableau de bord du portail des membres personnels.
- **Edit Profile** -- Mettez à jour vos paramètres de compte ou modifiez votre profil d'église.
- **Logout** -- Déconnectez-vous de votre compte.

## Accès administrateur

Si vous êtes un administrateur d'église avec les autorisations appropriées, vous verrez également une option **Admin Portal** dans votre menu utilisateur. En cliquant dessus, vous accédez à B1 Admin où vous pouvez gérer les paramètres, les personnes, les groupes et le contenu de votre église.

## Mot de passe oublié

Si vous ne vous souvenez pas de votre mot de passe :

1. Cliquez sur **Forgot Password** sur la page de connexion.
2. Entrez l'adresse e-mail liée à votre compte ChurchApps et soumettez.
3. Vérifiez votre e-mail pour un message contenant un **code de vérification à 6 chiffres**. Le code est valide pendant 15 minutes.
4. Entrez le code sur l'écran de vérification pour vous connecter.
5. Une fois connecté, visitez votre profil pour choisir un nouveau mot de passe.

:::tip
Si le code n'arrive pas dans quelques minutes, vérifiez votre dossier de spam. Après cinq tentatives incorrectes, le code est verrouillé — demandez une nouvelle tentative à partir de la page Mot de passe oublié si cela se produit.
:::

## Déconnexion

Pour vous déconnecter, cliquez sur votre **nom** dans le coin supérieur droit et sélectionnez **Logout**. Vous serez déconnecté et renvoyé au site public.
