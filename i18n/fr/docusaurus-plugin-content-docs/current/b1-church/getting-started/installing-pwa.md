# Installation comme une app (PWA)

<div class="article-intro">

B1.church supporte l'installation comme une Progressive Web App (PWA), ce qui vous permet d'ajouter le portail des membres de votre église à l'écran d'accueil de votre téléphone comme une app native. Cela vous donne une expérience en plein écran, ressemblant à une app, sans passer par l'App Store ou Google Play.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin de l'URL mobile B1.church de votre église : `https://yourchurchname.b1.church/mobile`
- Utilisez un navigateur supporté : **Safari** sur iOS ou **Chrome** sur Android

</div>

## Qu'est-ce qu'une PWA ?

Une Progressive Web App (PWA) est un site web qui peut être installé sur votre appareil et se comporte comme une app native. Lorsque vous installez B1.church comme une PWA, elle :

- S'ouvre en plein écran sans les barres de navigation du navigateur
- Apparaît sur votre écran d'accueil avec l'icône et le nom de votre église
- Fonctionne bien avec la marque de votre église et les couleurs du thème

:::tip
La PWA est une excellente option si vous préférez ne pas installer une app séparée depuis l'App Store ou Google Play. Elle fournit la même expérience du portail des membres que vous obtenez dans le navigateur, dans un paquet plus pratique.
:::

## Installation sur iPhone (iOS — Safari)

1. Ouvrez **Safari** sur votre iPhone et allez à `https://yourchurchname.b1.church/mobile` (remplacez `yourchurchname` par le slug URL de votre église).
2. Appuyez sur le bouton **Partager** (la boîte avec une flèche pointant vers le haut) en bas de l'écran.
3. Faites défiler vers le bas dans la feuille de partage et appuyez sur **Ajouter à l'écran d'accueil**.
4. Le nom sera par défaut celui de votre église. Appuyez sur **Ajouter** en haut à droite.

L'icône de l'app apparaîtra sur votre écran d'accueil. Appuyez dessus pour ouvrir le portail des membres en mode plein écran.

:::info
L'installation de PWA sur iOS nécessite Safari. Si vous utilisez Chrome ou un autre navigateur sur iPhone, basculez d'abord vers Safari.
:::

## Installation sur Android (Chrome)

1. Ouvrez **Chrome** sur votre appareil Android et allez à `https://yourchurchname.b1.church/mobile`.
2. Chrome affichera automatiquement une bannière **"Ajouter à l'écran d'accueil"** en bas de l'écran, ou appuyez sur le menu à trois points (⋮) en haut à droite et sélectionnez **Ajouter à l'écran d'accueil**.
3. Appuyez sur **Ajouter** lorsque vous y êtes invité.

L'icône de l'app apparaîtra sur votre écran d'accueil. Appuyez dessus pour ouvrir le portail des membres.

## Après l'installation

Une fois installée, la PWA fonctionne de la même manière que de visiter le site mobile dans votre navigateur, mais dans une fenêtre plein écran dédiée. Vous pouvez :

- Vous connecter avec votre compte ChurchApps existant
- Accéder à toutes les fonctionnalités des membres que votre église a activées — groupes, dons, enregistrement, plans de service et plus
- Y revenir à tout moment en appuyant sur l'icône sur votre écran d'accueil

:::info
La PWA nécessite une connexion Internet. Contrairement à certaines apps natives, elle ne met pas en cache tout le contenu pour une utilisation hors ligne.
:::

## PWA vs. App native

Les deux options vous donnent accès au portail des membres B1.church. Le bon choix dépend de votre préférence :

| | **PWA** | **App B1 Mobile** |
|---|---|---|
| Installation | Ajouter à l'écran d'accueil à partir du navigateur | Télécharger depuis l'App Store / Google Play |
| Mises à jour | Automatiques (aucune mise à jour d'app store nécessaire) | Via l'App Store / Google Play |
| Notifications push | Non supportées | Supportées |
| Plateforme | N'importe quel appareil avec un navigateur | iOS et Android |

:::tip
Si votre église utilise les conversations de groupe et que vous voulez les notifications push pour les nouveaux messages et l'activité, utilisez l'[app B1 Mobile](/docs/b1-mobile/) à la place. Les notifications push ne sont actuellement pas supportées dans la PWA.
:::
