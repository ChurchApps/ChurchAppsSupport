# Installation en tant qu'application (PWA)

<div class="article-intro">

B1.church supporte l'installation en tant qu'application Web progressive (PWA), qui vous permet d'ajouter le portail de votre église à l'écran d'accueil de votre téléphone comme une application native. Cela vous donne une expérience en plein écran, similaire à une application, sans passer par l'App Store ou Google Play.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin de l'URL B1.church de votre église : `https://yourchurchname.b1.church`
- Utilisez un navigateur supporté : **Safari** sur iOS ou **Chrome** sur Android

</div>

## Qu'est-ce qu'une PWA ?

Une application Web progressive (PWA) est un site Web qui peut être installé sur votre appareil et se comporte comme une application native. Lorsque vous installez B1.church en tant que PWA, il :

- S'ouvre en plein écran sans barres de navigation du navigateur
- Apparaît sur votre écran d'accueil avec l'icône et le nom de votre église
- Fonctionne parfaitement avec la marque et les couleurs de thème de votre église

:::tip
La PWA est une excellente option si vous préférez ne pas installer une application séparée depuis l'App Store ou Google Play. Elle offre la même expérience du portail de membres que vous obtenez dans le navigateur, dans un paquet plus pratique.
:::

## Où installer à partir

Vous pouvez installer la PWA de **n'importe quelle page** sur le site B1.church de votre église — la page d'accueil publique, une page de sermon, l'écran de connexion ou n'importe où dans le portail des membres. Il n'y a pas d'URL "d'installation" séparée à retenir ; si vous pouvez visiter votre église sur le Web, vous pouvez installer l'application à partir de là.

Lorsque vous lancez l'application installée, elle s'ouvre directement sur votre tableau de bord des membres, prêt pour vous de vous connecter.

## Installation sur iPhone (iOS — Safari)

1. Ouvrez **Safari** sur votre iPhone et allez à `https://yourchurchname.b1.church` (remplacez `yourchurchname` par le slug d'URL de votre église).
2. Appuyez sur le bouton **Share** (la boîte avec une flèche pointant vers le haut) en bas de l'écran.
3. Faites défiler vers le bas dans la feuille de partage et appuyez sur **Add to Home Screen**.
4. Le nom prendra par défaut le nom de votre église. Appuyez sur **Add** dans le coin supérieur droit.

L'icône de l'application apparaîtra sur votre écran d'accueil. Appuyez dessus pour ouvrir le portail des membres en mode plein écran.

:::info
L'installation de PWA sur iOS nécessite Safari. Si vous utilisez Chrome ou un autre navigateur sur iPhone, passez d'abord à Safari.
:::

## Installation sur Android (Chrome)

1. Ouvrez **Chrome** sur votre appareil Android et allez à `https://yourchurchname.b1.church`.
2. Chrome affichera automatiquement une bannière **"Add to Home screen"** en bas de l'écran, ou appuyez sur le menu à trois points (⋮) dans le coin supérieur droit et sélectionnez **Add to Home screen**.
3. Appuyez sur **Add** lorsqu'on vous y invite.

L'icône de l'application apparaîtra sur votre écran d'accueil. Appuyez dessus pour ouvrir le portail des membres.

## Après l'installation

Une fois installée, la PWA s'ouvre directement dans le portail des membres dans une fenêtre dédiée en plein écran. Vous pouvez :

- Vous connecter avec votre compte ChurchApps existant
- Accéder à toutes les fonctionnalités de membres que votre église a activées — groupes, dons, enregistrement, plans de service et plus
- Y revenir à tout moment en appuyant sur l'icône sur votre écran d'accueil

:::info
La PWA nécessite une connexion Internet. Contrairement à certaines applications natives, elle n'enregistre pas tout le contenu pour une utilisation hors ligne.
:::

## Qu'en est-il de l'ancienne application B1 Mobile ?

L'application native B1 Mobile héritée sur iOS et Android a été obsolète et n'est plus mise à jour. La PWA est le moyen supporté d'accéder au portail de membres B1.church sur un téléphone à l'avenir, et elle couvre chaque fonction de membre que l'application native offrait. Si vous avez précédemment installé l'application native, passez à la PWA à votre convenance.

Consultez la page de référence [B1 Mobile](/docs/b1-mobile/) pour la documentation historique sur l'application obsolète.
