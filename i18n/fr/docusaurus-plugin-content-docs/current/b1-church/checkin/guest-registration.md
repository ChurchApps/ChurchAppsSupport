# Enregistrement des invités

<div class="article-intro">

Si vous visitez une église pour la première fois, vous pouvez vous enregistrer vous et votre famille en tant qu'invités en utilisant la page d'enregistrement des invités. Cela crée un dossier dans le système de l'église afin que les bénévoles puissent vous enregistrer à votre arrivée.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Votre église doit avoir l'enregistrement des invités par code QR activé
- Scannez le code QR affiché au kiosque d'accueil ou recevez un lien d'un bénévole de l'église

</div>

## Comment fonctionne l'enregistrement des invités

L'enregistrement des invités est conçu pour les visiteurs qui n'ont pas encore de compte. Un code QR s'affiche au kiosque d'accueil de l'église qui mène à la page d'enregistrement. Vous pouvez également recevoir le lien d'un bénévole ou d'un accueillant.

## Enregistrement de votre famille

1. **Scannez le code QR** au kiosque d'accueil ou ouvrez le lien d'enregistrement des invités sur votre téléphone.
2. Le formulaire d'enregistrement des invités apparaît avec les champs du **contact principal** :
   - **Prénom** (requis)
   - **Nom** (requis)
   - **E-mail** (facultatif)
   - **Téléphone** (facultatif)
3. Pour ajouter des **membres de la famille**, appuyez sur **Ajouter un membre de la famille** et entrez son prénom et son nom. Vous pouvez ajouter jusqu'à 10 membres de la famille.
4. Pour supprimer un membre de la famille, appuyez sur le bouton de suppression à côté de son nom.
5. Appuyez sur **Enregistrer** pour soumettre.

## Après l'enregistrement

Une fois que vous avez soumis le formulaire, un écran de confirmation apparaît avec le message : « Enregistrement terminé — Vous êtes prêt ! Un membre du personnel vous enregistrera sous peu. »

Votre enregistrement crée des dossiers dans la base de données de l'église :

- Un **ménage** est créé en utilisant le nom de famille du contact principal.
- Des **dossiers de personne** sont créés pour chaque membre de la famille avec un statut d'adhésion « Invité ».
- Un bénévole ou administrateur de l'église peut ensuite enregistrer votre famille en utilisant le kiosque [B1 Checkin](../../b1-checkin/check-in/looking-up-members).

:::info
Si votre e-mail correspond à un dossier existant dans la base de données de l'église, le système établit un lien avec ce dossier au lieu de créer un doublon. Cela garde les données de l'église propres.
:::

:::tip
Fournir votre adresse e-mail aide l'église à vous suivre après votre visite. C'est facultatif mais recommandé.
:::
