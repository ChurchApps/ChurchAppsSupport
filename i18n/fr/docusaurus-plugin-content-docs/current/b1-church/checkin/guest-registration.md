# Enregistrement des invités

<div class="article-intro">

Si vous visitez une église pour la première fois, vous pouvez vous enregistrer vous-même et votre famille en tant qu'invités en utilisant la page d'enregistrement des invités. Cela crée un enregistrement dans le système de l'église afin que les bénévoles puissent vous enregistrer à votre arrivée.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Votre église doit avoir l'enregistrement des invités par code QR activé
- Scannez le code QR affiché au kiosque d'enregistrement, ou recevez un lien d'un bénévole de l'église

</div>

## Comment fonctionne l'enregistrement des invités

L'enregistrement des invités est conçu pour les visiteurs qui n'ont pas encore de compte. Un code QR est affiché au kiosque d'enregistrement de l'église qui renvoie à la page d'enregistrement. Vous pouvez également recevoir le lien d'un bénévole ou d'un hôte.

## Enregistrement de votre famille

1. **Scannez le code QR** au kiosque d'enregistrement ou ouvrez le lien d'enregistrement des invités sur votre téléphone.
2. Le formulaire d'enregistrement des invités s'affiche avec les champs du **contact principal** :
   - **Prénom** (obligatoire)
   - **Nom** (obligatoire)
   - **E-mail** (optionnel)
   - **Téléphone** (optionnel)
3. Pour ajouter des **membres de la famille**, appuyez sur **Ajouter un membre de la famille** et entrez leur prénom et nom. Vous pouvez ajouter jusqu'à 10 membres de la famille.
4. Pour supprimer un membre de la famille, appuyez sur le bouton de suppression à côté de son nom.
5. Appuyez sur **Enregistrer** pour soumettre.

## Après l'enregistrement

Une fois le formulaire soumis, un écran de confirmation s'affiche avec le message : « Enregistrement terminé — vous êtes prêt ! Un membre du personnel vous enregistrera sous peu. »

Votre enregistrement crée des enregistrements dans la base de données de l'église :

- Un **foyer** est créé en utilisant le nom de famille du contact principal.
- Les **enregistrements de personne** sont créés pour chaque membre de la famille avec un statut d'adhésion « Invité ».
- Un bénévole ou un administrateur de l'église peut ensuite enregistrer votre famille en utilisant le [kiosque B1 Checkin](../../b1-checkin/check-in/looking-up-members).

:::info
Si votre e-mail correspond à un enregistrement existant dans la base de données de l'église, le système se lie à cet enregistrement au lieu de créer un doublon. Cela garde les données de l'église propres.
:::

:::tip
Fournir votre adresse e-mail aide l'église à vous suivre après votre visite. C'est optionnel mais recommandé.
:::
