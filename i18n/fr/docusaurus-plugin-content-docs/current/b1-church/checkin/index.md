# Accueil

<div class="article-intro">

La fonctionnalité **Accueil** dans B1.church vous permet de vous enregistrer vous et les membres de votre ménage pour les services d'église directement à partir du portail des membres. C'est un moyen pratique d'enregistrer votre présence sans avoir besoin de visiter un kiosque d'accueil physique.

</div>

## Comment ça marche

Lorsque vous utilisez l'auto-accueil, vous sélectionnez un service, choisissez les membres du ménage qui assistent et assignez chaque personne au groupe approprié pour chaque heure de service. Une fois que vous confirmez, votre présence est enregistrée automatiquement.

## Démarrage

Pour accéder à l'accueil, cliquez sur **Accueil** dans la [navigation de la barre latérale](../getting-started/navigating.md) de votre portail des membres. Vous devez être [connecté](../getting-started/logging-in.md) pour utiliser cette fonction -- si vous n'êtes pas connecté, vous serez invité à vous connecter d'abord.

:::info
La fonction d'accueil exige que votre église ait configuré des services et des groupes de présence dans ses paramètres d'administration. S'il n'y a pas de services disponibles lorsque vous ouvrez la page d'accueil, votre église n'a peut-être pas encore configuré cette fonction. Contactez l'administrateur de votre église pour obtenir de l'aide.
:::
