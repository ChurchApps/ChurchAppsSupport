# Enregistrement

<div class="article-intro">

La fonctionnalité **Enregistrement** dans B1.church vous permet de vous enregistrer, vous et les membres de votre foyer, pour les cultes directement depuis le portail membre. C'est un moyen pratique d'enregistrer votre présence sans avoir besoin de vous rendre à une borne d'enregistrement physique.

</div>

## Comment ça fonctionne

Lorsque vous utilisez l'auto-enregistrement, vous sélectionnez un culte, choisissez quels membres du foyer sont présents, et assignez chaque personne au groupe approprié pour chaque horaire de culte. Une fois que vous confirmez, votre présence est enregistrée automatiquement.

## Pour commencer

Pour accéder à l'enregistrement, cliquez sur **Enregistrement** dans la [barre de navigation latérale](../getting-started/navigating.md) de votre portail membre. Vous devez être [connecté](../getting-started/logging-in.md) pour utiliser cette fonctionnalité -- si vous n'êtes pas connecté, vous serez invité à vous connecter d'abord.

:::info
La fonctionnalité d'enregistrement nécessite que votre église ait configuré des cultes et des groupes de présence dans ses paramètres d'administration. Si aucun culte n'est disponible lorsque vous ouvrez la page d'enregistrement, votre église n'a peut-être pas encore configuré cette fonctionnalité. Contactez l'administrateur de votre église pour obtenir de l'aide.
:::
