# Plans

<div class="article-intro">

La section **Plans** dans B1.church vous permet de voir les plans de culte auxquels vous avez été assigné. Que vous fassiez partie de l'équipe de louange, que vous aidiez au ministère des enfants ou que vous serviez dans un autre domaine, c'est ici que vous pouvez savoir quand vous êtes programmé, quel poste vous occupez et à quoi ressemble le culte.

</div>

## Ce que vous y trouverez

Les plans peuvent inclure plusieurs types de contenu selon la façon dont votre église les configure :

- **Vos assignations** -- Voyez à quels postes vous avez été assigné et quand vous êtes attendu. Chaque assignation affiche le nom du poste, la date du culte et votre statut actuel (Accepté, Refusé ou Non confirmé).
- **Déroulement du culte** -- Consultez le déroulement du culte avec les détails de timing, les informations sur les chants et les descriptions des éléments pour connaître le déroulement du culte à l'avance.
- **Contenu de programme** -- Certains plans incluent du contenu de leçons ou de programme associé provenant de fournisseurs comme [Lessons.church](../content/lessons.md). Vous pouvez prévisualiser ce contenu directement dans le plan.
- **Dates d'indisponibilité** -- Informez les responsables de votre équipe de vos indisponibilités en gérant vos dates d'indisponibilité.

## Pour commencer

Pour accéder à vos plans, cliquez sur **Plans** dans la barre de navigation latérale de votre portail membre. Vous devez être [connecté](../getting-started/logging-in.md) pour voir vos plans -- si vous n'êtes pas connecté, vous serez invité à vous connecter d'abord.

:::info
Vous ne verrez que les plans auxquels vous avez été assigné. Si vous ne voyez aucun plan listé, cela signifie que vous n'avez pas encore été ajouté à une équipe de service. Parlez à l'administrateur de votre église ou au responsable de votre ministère pour vous impliquer.
:::
