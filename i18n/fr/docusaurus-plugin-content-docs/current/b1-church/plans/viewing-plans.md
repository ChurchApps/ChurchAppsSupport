# Consulter les plans

<div class="article-intro">

La page Plans vous offre une vue claire de tous les plans de culte auxquels vous êtes assigné. Elle utilise une mise en page maître-détail pour que vous puissiez rapidement parcourir vos plans et consulter les détails de chacun d'entre eux.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous devez être [connecté](../getting-started/logging-in.md) à votre compte B1.church.
- Vous devez être assigné à au moins un plan de culte. Si vous ne voyez aucun plan, contactez l'administrateur de votre église ou le responsable de votre ministère.

</div>

## Accéder à vos plans

1. Cliquez sur **Plans** dans la [barre de navigation latérale](../getting-started/navigating.md).
2. Vous verrez une liste de vos plans sur le côté gauche de la page.

## La liste des plans

Vos plans sont listés dans le panneau de gauche sous **Mes plans**. Chaque entrée affiche :

- Le **nom du plan** (par exemple, « Culte du dimanche » ou « Ministère des enfants »)
- La **date du culte**
- Votre **poste** (le rôle que vous occupez)
- Votre **statut** -- codé par couleur pour que vous puissiez voir en un coup d'œil si vous avez accepté, refusé ou pas encore répondu

En haut de la liste, vous verrez également une option **Dates d'indisponibilité**. Cliquez dessus pour gérer les dates où vous n'êtes pas disponible pour servir. Vous pouvez ajouter, modifier et supprimer des plages de dates d'indisponibilité afin que les responsables de votre équipe sachent quand vous planifier en fonction de vos disponibilités.

## Voir les détails d'un plan

Cliquez sur n'importe quel plan dans la liste pour ouvrir ses détails dans le panneau de droite. La vue détaillée comprend :

- **Détails du poste** -- Votre poste assigné avec les horaires nécessaires et votre statut actuel. Si vous n'avez pas encore répondu, vous pouvez cliquer sur **Accepter** ou **Refuser** directement depuis cette vue.
- **Notes** -- Les notes ajoutées par le créateur du plan pour l'équipe.
- **Déroulement du culte** -- Le déroulement complet du culte avec le timing, les éléments et les descriptions. Consultez [Déroulement du culte](./service-order.md) pour plus de détails.
- **Assignations de l'équipe** -- Un résumé de tous les membres de l'équipe et de leurs postes, pour que vous puissiez voir qui d'autre sert à vos côtés.

:::tip
Si vous avez plusieurs assignations dans différents plans, la liste est triée par date afin que vos prochains engagements apparaissent en premier. Consultez régulièrement pour rester à jour avec votre planning.
:::
