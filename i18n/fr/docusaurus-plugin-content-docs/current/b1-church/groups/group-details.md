# Détails du groupe

<div class="article-intro">

Quand vous cliquez sur un groupe dans la page des groupes, vous êtes amené à la page des détails du groupe. Ce que vous voyez dépend de si vous êtes connecté ou non, les membres connectés ayant accès à des onglets et des fonctionnalités supplémentaires.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Trouvez un groupe à afficher en [parcourant les groupes](./browsing-groups.md).
- Pour un accès complet à tous les onglets, [connectez-vous](../getting-started/logging-in.md) à votre compte.

</div>

## Vue des visiteurs (Non connecté)

Si vous n'êtes pas connecté, vous verrez les éléments suivants :

- **Section héros** — Une grande zone de bannière montrant la photo, le nom, l'heure de réunion et le lieu de réunion du groupe.
- **Description** — La description complète du groupe expliquant son objectif et à quoi s'attendre.
- **Leaders** — Avatars et noms des leaders du groupe. Vous pouvez cliquer sur le nom d'un leader pour voir son profil.
- **Événements du calendrier** — Jusqu'à trois événements à venir pour le groupe sont affichés, chacun montrant la date et le titre de l'événement. Si le groupe n'a pas d'événements à venir, cette section est masquée.
- **Formulaire de contact** — Un formulaire qui vous permet d'envoyer un message directement aux leaders du groupe. Remplissez votre **prénom**, **nom**, **e-mail**, **numéro de téléphone** et **message**, puis cliquez sur **Soumettre**. S'il y a plusieurs leaders, vous pouvez choisir lequel contacter dans une liste déroulante.

## Vue des membres (Connecté)

Quand vous êtes connecté, la page des détails du groupe affiche la section héros avec une barre latérale d'onglets. Les onglets disponibles sont :

- **Détails du groupe** — La description complète du groupe, rendue avec un formatage de texte enrichi.
- **Membres** — Une liste de tous les membres du groupe, montrant leur photo et leur nom. Vous pouvez cliquer sur le nom d'un membre pour visiter son profil dans l'[annuaire des membres](../community/member-directory.md).
- **Calendrier** — Le calendrier du groupe montrant les réunions et les événements. Vous pouvez parcourir les événements à venir et passés.
- **Conversations** — Un espace pour que les membres du groupe publient des messages et interagissent les uns avec les autres. Consultez [Conversations de groupe](./conversations.md) pour plus de détails.
- **Ressources** — Fichiers et liens partagés qui ont été publiés pour le groupe. Vous pouvez télécharger les fichiers téléchargés ou ouvrir les ressources liées.
- **Présence** — Visible uniquement par les leaders du groupe. Permet le suivi de qui a assisté aux réunions du groupe.

## Capacités du leader

Si vous êtes un leader du groupe, vous avez des capacités de modification supplémentaires :

- **Modifier les détails du groupe** — Mettez à jour le nom, la description, l'heure de réunion et d'autres paramètres du groupe directement à partir de l'onglet **Détails**.
- **Gérer les membres** — Ajoutez de nouveaux membres au groupe ou supprimez les existants à partir de l'onglet **Membres**.
- **Gérer les ressources** — Téléchargez des fichiers (jusqu'à 100 Mo de stockage), ajoutez des liens ou supprimez les ressources existantes.
- **Ressources de leader** — Accédez à un onglet **Ressources (Leaders)** séparé avec les fichiers et les liens visibles uniquement par les leaders du groupe.
- **Suivre la présence** — Enregistrez la présence pour les sessions du groupe à partir de l'onglet **Présence**.

## Rejoindre un groupe

Quand vous êtes connecté et que vous n'êtes pas encore membre, un bouton de connexion apparaît sur la page des détails du groupe. L'étiquette du bouton dépend de la politique d'adhésion du groupe :

- **Rejoindre le groupe** — Le groupe est ouvert. Cliquer dessus vous ajoute immédiatement en tant que membre.
- **Demander à rejoindre** — Le groupe nécessite une approbation. Cliquer dessus ouvre une boîte de dialogue où vous pouvez inclure un message optionnel au leader du groupe, puis soumettre votre demande. Consultez [Demander à rejoindre un groupe](./join-requests.md) pour plus de détails sur la gestion de vos demandes.

Si aucun bouton n'apparaît, le groupe est fermé et les membres doivent être ajoutés manuellement par un leader ou un administrateur.

:::info
Si un groupe est fermé et que vous voulez vous joindre, utilisez le formulaire de contact sur la vue du visiteur pour communiquer avec les leaders du groupe.
:::
