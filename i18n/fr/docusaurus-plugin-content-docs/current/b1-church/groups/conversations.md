# Conversations de groupe

<div class="article-intro">

L'onglet Conversations sur votre page de groupe est un espace où les membres du groupe peuvent publier des messages et interagir les uns avec les autres. Utilisez les conversations pour discuter de sujets, partager des mises à jour et rester connecté avec votre groupe entre les réunions.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous devez être [connecté](../getting-started/logging-in.md) à votre compte B1.church.
- Vous devez être un membre du groupe pour accéder à l'onglet Conversations. Consultez les [Détails du groupe](./group-details.md) pour savoir comment rejoindre un groupe.

</div>

## Accès aux conversations

1. Accédez à la page de détails du groupe en cliquant sur un groupe à partir de la [page groupes](./browsing-groups.md).
2. Assurez-vous que vous êtes connecté.
3. Cliquez sur l'onglet **Conversations** dans la barre latérale.

## Discussions et annonces

L'onglet Conversations peut afficher deux sous-onglets :

- **Discussions** -- Un espace où n'importe quel membre du groupe peut publier des messages et répondre aux autres. C'est la vue par défaut et elle est toujours disponible.
- **Announcements** -- Messages publiés par les responsables de groupe qui sont importants pour que tous les membres les voient. Les membres ordinaires peuvent lire les annonces, mais seuls les responsables peuvent les créer.

S'il n'y a pas d'annonces et que vous n'êtes pas un responsable, vous verrez uniquement la vue Discussions sans la barre d'onglets.

## Publier un message

1. Ouvrez l'onglet **Conversations** sur votre page de groupe.
2. Assurez-vous que vous êtes sur le sous-onglet **Discussions** (si la barre d'onglets est visible).
3. Tapez votre message dans le champ de texte.
4. Soumettez votre message pour le partager avec le groupe.

Votre message sera visible à tous les membres du groupe.

## Modification ou suppression de vos messages

Vous pouvez modifier ou supprimer n'importe quel message que vous avez publié :

1. Survolez votre message (ou appuyez dessus sur mobile) pour révéler le bouton **more options** (⋮).
2. Cliquez sur le bouton pour ouvrir le menu des messages.
3. Choisissez **Edit** pour modifier le texte du message ou **Delete** pour le supprimer.
   - Lors de l'édition, le texte du message se recharge dans le champ de saisie. Faites vos modifications et soumettez-les pour les enregistrer.
   - Lors de la suppression, une boîte de dialogue de confirmation apparaît. Cliquez sur **Confirm** pour supprimer définitivement le message.

:::info
Vous pouvez uniquement modifier ou supprimer vos propres messages. Les messages publiés par d'autres membres ne montrent pas les options d'édition/suppression.
:::

## Réaction aux messages

Dans le chat de groupe, vous pouvez réagir à n'importe quel message avec un emoji :

1. Appuyez ou survolez un message et choisissez **Add reaction**.
2. Choisissez l'un des six emoji : 👍 ❤️ 😂 🎉 🙏 😮.

Les réactions apparaissent sous forme de petites puces sous le message avec un décompte du nombre de personnes qui ont utilisé chacune. Votre propre réaction est surlignée : appuyez sur la puce (ou choisissez le même emoji à nouveau) pour la supprimer, ou choisissez un emoji différent pour en ajouter un autre. Les réactions s'affichent en direct pour tous les utilisateurs du chat.

:::tip
Les conversations de groupe apparaissent également sur votre [Timeline](../community/timeline.md), vous pouvez donc suivre les discussions sans visiter chaque groupe individuellement.
:::

## Fonctionnalités de responsable

Si vous êtes un responsable de groupe, vous avez des capacités supplémentaires dans l'onglet Conversations :

- **Publier des annonces** -- Basculez vers le sous-onglet **Announcements** et publiez des messages que tous les membres du groupe peuvent lire.
- **Publier des discussions** -- Vous pouvez également participer aux discussions régulières comme n'importe quel autre membre.

:::info
Les conversations sont spécifiques à chaque groupe. Les messages que vous publiez dans un groupe n'apparaîtront pas dans les conversations d'un autre groupe.
:::
