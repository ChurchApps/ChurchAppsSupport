# Groupes

<div class="article-intro">

Les groupes sont un excellent moyen de vous connecter avec d'autres personnes à votre église. Que vous recherchiez un petit groupe, une équipe de ministère, une étude biblique ou une opportunité de bénévolat, vous pouvez parcourir et rejoindre des groupes directement à partir du site B1.church de votre église.

</div>

## Ce que vous pouvez faire

Voici ce que vous pouvez faire avec les groupes dans B1App :

- **Parcourir les groupes publics** — Affichlez tous les groupes disponibles organisés par catégorie. Chaque groupe montre une photo, un nom, une heure de réunion, un lieu et une brève description.
- **Consulter les détails du groupe** — Cliquez sur un groupe pour voir sa description complète, ses leaders, les événements du calendrier à venir et un formulaire de contact pour contacter les leaders du groupe.
- **Rejoindre les groupes** — Connectez-vous à un groupe pour devenir membre. Les groupes ouverts vous permettent de rejoindre immédiatement ; les groupes nécessitant une approbation vous permettent de soumettre une demande d'adhésion. Consultez [Demander à rejoindre un groupe](./join-requests.md).
- **Participer aux conversations** — Une fois que vous êtes membre d'un groupe, vous pouvez publier des messages et interagir avec d'autres membres du groupe via l'onglet Conversations.
- **Accès aux calendriers et ressources** — Affichlez les événements à venir sur le calendrier du groupe et téléchargez les fichiers partagés ou les liens à partir de l'onglet Ressources.

## Expérience publique versus connectée

Quiconque peut visiter la page des groupes de votre église et parcourir les groupes disponibles. Les visiteurs qui ne sont pas connectés peuvent consulter les descriptions des groupes, voir qui sont les leaders, vérifier les événements du calendrier à venir et envoyer un message à un leader en utilisant le formulaire de contact.

Quand vous [vous connectez](../getting-started/logging-in.md), vous déverrouillez des onglets supplémentaires sur la page des détails du groupe, y compris **Détails du groupe**, **Calendrier**, **Conversations**, **Ressources**, **Membres** et **Présence**. Les leaders du groupe ont des capacités de modification supplémentaires sur chacun de ces onglets.

:::tip
Les groupes apparaissent également sur votre [Timeline](../community/timeline.md) — les conversations et les mises à jour de vos groupes apparaîtront dans votre flux d'activité automatiquement.
:::
