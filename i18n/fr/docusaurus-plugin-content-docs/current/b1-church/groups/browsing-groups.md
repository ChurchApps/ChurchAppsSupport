# Parcourir les groupes

<div class="article-intro">

La page publique des groupes de votre église facilite la découverte des groupes qui vous intéressent. Les groupes sont organisés par catégorie, vous pouvez donc parcourir rapidement les différents types de groupes et trouver celui qui vous convient.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Aucune connexion n'est requise pour parcourir les groupes. Cependant, [se connecter](../getting-started/logging-in.md) vous donne accès à des fonctionnalités supplémentaires sur la page de détails de chaque groupe.

</div>

## Trouver la page des groupes

Accédez à la page **Groupes** sur le site B1.church de votre église. Cette page est accessible publiquement, vous n'avez donc pas besoin d'être connecté pour parcourir les groupes.

## Comment les groupes sont organisés

Les groupes sont affichés par **catégorie** (également appelée libellé). Les catégories sont configurées par les administrateurs de votre église et peuvent inclure des libellés tels que « Petits groupes », « Ministères », « Jeunesse » ou « Études bibliques ». Chaque catégorie affiche tous les groupes qui lui appartiennent.

## Lire les fiches de groupe

Chaque groupe est affiché sous forme de fiche avec les informations suivantes :

- **Photo du groupe** -- Une image de couverture représentant le groupe. Si aucune photo n'a été définie, une image par défaut est affichée.
- **Nom du groupe** -- Affiché en évidence sur la photo.
- **Horaire de réunion** -- Quand le groupe se réunit habituellement (par exemple, « Mercredis à 19h »). Si aucun horaire n'est défini, un texte de remplacement est affiché.
- **Lieu de réunion** -- Où le groupe se réunit. Si aucun lieu n'est défini, un texte de remplacement est affiché.
- **Description** -- Un bref aperçu du sujet du groupe, limité à quelques lignes. Cliquez sur le groupe pour lire la description complète.

## Voir un groupe

Cliquez sur n'importe quelle fiche de groupe pour ouvrir la page de détails complète du groupe. Vous y trouverez la description complète, des informations sur les responsables du groupe, les événements à venir au calendrier et un moyen de prendre contact.

:::tip
Si vous cherchez un type de groupe spécifique, parcourez d'abord les en-têtes de catégories. Cela vous permet d'accéder directement à la section qui vous intéresse plutôt que de faire défiler tous les groupes.
:::
