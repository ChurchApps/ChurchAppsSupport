# Applications mobiles

<div class="article-intro">

Les applications mobiles ChurchApps sont construites avec **React Native** et **Expo**, fournissant des expériences natives sur Android, iOS et Android TV. Toutes les applications mobiles partagent la même chaîne d'outils basée sur Expo, donc si vous pouvez en construire une, vous pouvez toutes les construire.

</div>

## Applications en un coup d'œil

| Application | Objectif | Plateforme |
|-----|---------|----------|
| [B1 Mobile](./b1-mobile) | Application principale orientée vers les membres pour les membres de l'église | Android, iOS |
| B1 Checkin | Application de kiosque pour les stations de check-in | Android, iOS |
| LessonsScreen | Application Android TV pour afficher les leçons dans les salles de classe | Android TV |

### B1 Mobile

L'application principale pour les membres de l'église. Les membres peuvent voir les répertoires, accéder aux donations, vérifier la présence, recevoir des notifications et interagir avec leur communauté religieuse. Voir le guide de configuration [B1 Mobile](./b1-mobile) pour les détails.

### B1 Checkin

Une application de kiosque spécialement conçue pour les stations de check-in à l'église. Elle fournit une interface simple et tactile pour que les membres enregistrent les enfants et notent la présence aux services et aux événements.

### LessonsScreen

Une application Android TV qui affiche le contenu des leçons dans les salles de classe. Les enseignants peuvent envoyer les diapositives des leçons, les vidéos et les activités sur un écran TV, ce qui facilite la présentation du matériel du curriculum lors des classes.

:::info
Toutes les applications mobiles partagent la même chaîne d'outils basée sur Expo et suivent des modèles de configuration similaires. Le guide [B1 Mobile](./b1-mobile) sert de référence pour le processus de configuration générale sur tous les projets mobiles.
:::
