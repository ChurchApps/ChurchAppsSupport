# Pour commencer

<div class="article-intro">

Démarrer avec B1 Mobile ne prend que quelques minutes. Téléchargez l'application, connectez-vous avec votre compte ChurchApps, sélectionnez votre église, et vous êtes prêt.

</div>

## Démarrage rapide

1. **Téléchargez** l'application depuis l'[App Store](https://apps.apple.com) (iOS) ou le [Google Play Store](https://play.google.com) (Android). Recherchez « B1.church ».
2. **Connectez-vous** avec votre compte ChurchApps existant -- le même e-mail et mot de passe que vous utilisez sur B1.church. Si vous n'avez pas encore de compte, appuyez sur **S'inscrire** pour en créer un.
3. **Sélectionnez votre église** en la recherchant par nom. Tapez au moins 3 caractères pour voir les résultats.
4. **Explorez le tableau de bord** -- votre écran d'accueil personnalisé affiche des raccourcis vers les fonctionnalités proposées par votre église.

C'est tout. Une fois votre église sélectionnée, l'application charge l'identité visuelle, les liens de navigation et le contenu de votre église. Vous pouvez commencer à utiliser les groupes, les dons, l'enregistrement et tout ce que votre église a activé.
