# Premiers pas

:::danger B1 Mobile est obsolète
L'application native B1 Mobile a été remplacée par l'[application Web progressive B1.church (PWA)](../../b1-church/getting-started/installing-pwa.md). Les nouveaux utilisateurs doivent installer la PWA à partir de n'importe quelle page sur `https://yourchurchname.b1.church` au lieu de télécharger B1 Mobile depuis l'App Store ou Google Play.
:::

<div class="article-intro">

Se familiariser avec B1 Mobile ne prend que quelques minutes. Téléchargez l'application, connectez-vous avec votre compte ChurchApps, sélectionnez votre église, et vous êtes prêt à commencer.

</div>

## Démarrage rapide

1. **Téléchargez** l'application depuis l'[App Store](https://apps.apple.com) (iOS) ou [Google Play Store](https://play.google.com) (Android). Recherchez "B1.church".
2. **Connectez-vous** avec votre compte ChurchApps existant -- le même e-mail et mot de passe que vous utilisez sur B1.church. Si vous n'avez pas encore de compte, appuyez sur **Register** pour en créer un.
3. **Sélectionnez votre église** en la recherchant par nom. Tapez au moins 3 caractères pour voir les résultats.
4. **Explorez le tableau de bord** -- votre écran d'accueil personnalisé affiche des liens rapides vers les fonctionnalités que votre église offre.

C'est tout. Une fois que vous avez sélectionné votre église, l'application charge la marque, les liens de navigation et le contenu de votre église. Vous pouvez commencer à utiliser les groupes, les dons, l'enregistrement et tout ce que votre église a activé.
