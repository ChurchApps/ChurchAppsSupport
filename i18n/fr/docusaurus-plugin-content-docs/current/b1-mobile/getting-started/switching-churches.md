# Changer d'église

<div class="article-intro">

Si vous êtes membre de plusieurs églises, B1 Mobile facilite le passage de l'une à l'autre. Votre compte fonctionne avec toutes les églises qui utilisent ChurchApps, vous n'avez donc besoin que d'un seul identifiant.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](./installing) et [connectez-vous](./logging-in) à votre compte

</div>

## Comment changer d'église

1. Ouvrez le menu latéral en appuyant sur l'**icône hamburger** (trois lignes horizontales) dans le coin supérieur gauche.
2. Dans l'en-tête du menu, appuyez sur le bouton affichant le nom de votre église actuelle (ou **Trouver une église** si aucune église n'est sélectionnée).
3. L'écran de **Recherche d'église** apparaît. Vos églises récemment visitées sont affichées en haut pour un accès rapide.
4. Appuyez sur une église récente pour y basculer, ou tapez dans le champ de recherche pour trouver une autre église.

Lorsque vous changez d'église, l'application se recharge avec l'identité visuelle, les liens de navigation et le contenu de cette église. Votre [tableau de bord](../dashboard/), vos [groupes](../groups/) et autres données refléteront l'église nouvellement sélectionnée.

## Rechercher une église

Sur l'écran de Recherche d'église, tapez au moins 3 caractères du nom de l'église dans le champ de recherche. Les églises correspondantes apparaissent dans une liste en dessous. Appuyez sur une église pour la sélectionner.

:::info
Si aucun résultat n'apparaît, essayez une orthographe différente ou une partie plus courte du nom de l'église. La recherche compare les noms des églises enregistrées auprès de ChurchApps.
:::

## Églises récentes

L'écran de Recherche d'église mémorise les églises que vous avez visitées précédemment. Lorsque vous n'avez pas saisi de requête de recherche, vos églises récentes sont affichées automatiquement. Cela facilite le retour rapide vers une église que vous utilisez régulièrement.

:::tip
Vous n'avez pas besoin de vous déconnecter et de vous reconnecter pour changer d'église. Votre compte ChurchApps unique fonctionne avec toutes vos églises. Il suffit de sélectionner une autre église depuis l'écran de Recherche d'église et vous êtes prêt.
:::
