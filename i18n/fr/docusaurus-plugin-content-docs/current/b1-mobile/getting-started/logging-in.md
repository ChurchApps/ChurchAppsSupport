# Se connecter

<div class="article-intro">

B1 Mobile utilise le même compte ChurchApps que celui que vous utilisez sur B1.church et les autres produits ChurchApps. Si vous avez déjà un compte, vous pouvez vous connecter avec votre e-mail et votre mot de passe existants.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](./installing) sur votre appareil iOS ou Android

</div>

## Comment se connecter

1. Ouvrez l'application B1 Mobile.
2. Appuyez sur l'option **Connexion** depuis le menu latéral ou l'invite de connexion.
3. Saisissez votre **adresse e-mail** et votre **mot de passe**.
4. Appuyez sur **Se connecter**.

Après la connexion, vous serez dirigé vers le [tableau de bord](../dashboard/). Si vous n'avez pas encore sélectionné d'église, vous serez invité à en rechercher et en sélectionner une.

## Créer un nouveau compte

Si vous n'avez pas encore de compte ChurchApps, vous pouvez en créer un directement dans l'application :

1. Sur l'écran de connexion, appuyez sur le lien **S'inscrire** en bas.
2. Saisissez votre **prénom**, votre **nom** et votre **adresse e-mail**.
3. Appuyez sur **S'inscrire**.
4. Vérifiez votre e-mail pour un message contenant votre mot de passe temporaire.
5. Retournez dans l'application, appuyez sur **Connexion** et connectez-vous avec votre e-mail et le mot de passe temporaire reçu par e-mail.

Après votre première connexion, vous pouvez modifier votre mot de passe depuis les [paramètres de votre profil](../profile/editing-profile.md).

:::warning
Pensez à vérifier votre dossier spam ou courrier indésirable si vous ne recevez pas l'e-mail d'inscription dans les quelques minutes.
:::

## Mot de passe oublié

Si vous avez oublié votre mot de passe, appuyez sur **Mot de passe oublié ?** sur l'écran de connexion. Cela ouvre une page dans votre navigateur où vous pouvez saisir votre adresse e-mail et demander un lien de réinitialisation de mot de passe.

## Rester connecté

B1 Mobile vous maintient connecté entre les sessions. Vous n'avez pas besoin de vous connecter à chaque fois que vous ouvrez l'application. Si vous êtes déconnecté pour une raison quelconque, reconnectez-vous simplement avec votre e-mail et votre mot de passe.

## Se déconnecter

Pour vous déconnecter de l'application :

1. Ouvrez le menu latéral en appuyant sur l'**icône hamburger** (trois lignes horizontales) dans le coin supérieur gauche.
2. Faites défiler jusqu'en bas du menu latéral.
3. Appuyez sur **Se déconnecter**.

Vous serez déconnecté et pourrez vous reconnecter à tout moment avec vos identifiants.
