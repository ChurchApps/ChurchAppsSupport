# Enregistrement

<div class="article-intro">

B1 Mobile vous permet de vous enregistrer, ainsi que les membres de votre foyer, pour les cultes de votre église directement depuis votre téléphone. Évitez la file d'attente à la borne et enregistrez votre présence avant même de franchir les portes.

</div>

## Comment ça fonctionne

Le processus d'enregistrement est un flux rapide en plusieurs étapes qui vous guide pour sélectionner un culte, choisir les personnes présentes et assigner chaque personne à un groupe. Une fois la confirmation effectuée, votre présence est enregistrée automatiquement.

## Pour commencer

Pour accéder à l'enregistrement, ouvrez la section **Enregistrement** dans B1 Mobile. Vous devez être connecté pour utiliser cette fonctionnalité. Si vous n'êtes pas connecté, vous serez invité à vous [connecter](../getting-started/logging-in) au préalable.

:::info
La fonctionnalité d'enregistrement nécessite que votre église ait configuré des cultes et des groupes de présence dans ses paramètres d'administration. Si aucun culte n'apparaît lorsque vous ouvrez l'écran d'enregistrement, il se peut que votre église n'ait pas encore configuré cette fonctionnalité. Contactez l'administrateur de votre église pour obtenir de l'aide.
:::

## Ce que vous pouvez faire

- **Enregistrer votre foyer** -- Sélectionnez les membres de votre famille qui sont présents et enregistrez la présence de tous en une seule fois. Consultez [Foyer](../profile/household.md) pour apprendre à gérer les membres de votre famille.
- **Choisir les cultes et les horaires** -- Sélectionnez parmi les cultes et créneaux horaires disponibles que votre église a configurés.
- **Assigner des groupes** -- Placez chaque personne dans le groupe approprié pour le culte, comme le Culte principal, la Salle des enfants ou le Groupe de jeunes.
