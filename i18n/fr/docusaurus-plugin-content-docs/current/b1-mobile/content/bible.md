# Bible

<div class="article-intro">

B1 Mobile inclut un lecteur biblique intégré pour que vous puissiez lire les Écritures directement depuis l'application. Le lecteur est intégré directement dans l'écran Bible, vous offrant un accès rapide à n'importe quel passage sans changer d'application.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](../getting-started/installing) et [connectez-vous](../getting-started/logging-in) à votre compte
- Votre église doit avoir activé la fonctionnalité Bible

</div>

## Lire la Bible

1. Ouvrez l'écran **Bible** depuis le menu latéral.
2. Le lecteur se charge avec un passage par défaut.
3. Utilisez les contrôles de navigation dans le lecteur pour vous déplacer vers un autre livre, chapitre ou verset.

Le lecteur biblique est alimenté par YouVersion ou un autre fournisseur configuré par votre église. L'expérience de lecture est la même quel que soit le fournisseur utilisé.

## Naviguer entre les passages

Le lecteur intégré fournit des contrôles pour parcourir les Écritures. Vous pouvez :

- Accéder à un livre spécifique de la Bible
- Aller à un chapitre particulier
- Faire défiler les versets au sein d'un chapitre

Le lecteur occupe tout l'écran, vous offrant suffisamment d'espace pour lire confortablement sur votre appareil.

:::tip
Intégrez l'écran Bible à votre routine quotidienne -- c'est un moyen rapide de lire les Écritures où que vous soyez. Associez-le au [Verset du jour](./verse-of-the-day) pour une inspiration quotidienne.
:::

## Disponibilité

L'écran Bible est disponible lorsque votre église a activé cette fonctionnalité. Si vous ne voyez pas d'option **Bible** dans le menu latéral, il se peut que l'administrateur de votre église ne l'ait pas ajouté à l'application. Contactez l'administrateur de votre église pour en faire la demande.
