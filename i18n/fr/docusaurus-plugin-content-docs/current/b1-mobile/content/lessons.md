# Cours

<div class="article-intro">

L'écran Cours vous donne accès au programme et aux supports d'apprentissage de votre église. Le contenu est fourni via la plateforme Lessons.church et est intégré directement dans B1 Mobile, de sorte que vous pouvez parcourir et consulter les cours sans changer d'application.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](../getting-started/installing) et [connectez-vous](../getting-started/logging-in) à votre compte
- Votre église doit avoir configuré un programme via Lessons.church

</div>

## Accéder aux cours

1. Ouvrez l'écran **Cours** depuis le menu latéral.
2. Le contenu de Lessons.church se charge directement dans l'application.
3. Parcourez le programme que votre église a mis à disposition.

Votre église configure les cours et programmes qui apparaissent ici. Vous verrez le contenu qui a été programmé ou assigné par l'équipe de votre église.

## Ce que vous pouvez faire

Une fois sur l'écran Cours, vous pouvez parcourir et consulter les supports que votre église a mis en place. Selon la configuration de votre église, vous pouvez trouver :

- **Plans de cours** -- Des plans structurés avec des sections et des activités
- **Présentations et médias** -- Des diaporamas, vidéos et images pour accompagner les cours
- **Ressources téléchargeables** -- Des fichiers et supports pour approfondir l'étude

Appuyez sur les sections individuelles d'un cours pour en afficher le contenu. Utilisez les contrôles de navigation pour vous déplacer entre les sections ou revenir à la liste des cours.

:::info
Lessons.church est une plateforme ChurchApps distincte qui s'intègre à B1 Mobile. L'administrateur de votre église gère le contenu des cours via le portail Lessons.church.
:::

## Disponibilité

L'écran Cours apparaît lorsque votre église a configuré un programme via Lessons.church. Si vous ne voyez pas d'option **Cours** dans le menu latéral, il se peut que votre église n'ait pas mis en place de contenu de cours. Contactez l'administrateur de votre église pour plus d'informations.
