# Groupes sur B1 Mobile

<div class="article-intro">

Restez connecté avec les groupes de votre église sur B1 Mobile. Consultez les mises à jour, les conversations et les événements de groupe.

</div>

## Accès à vos groupes

1. Ouvrez B1 Mobile
2. Appuyez sur l'icône **Groupes** dans le menu inférieur
3. Vous verrez une liste de tous les groupes dont vous faites partie

## Affichage de la chronologie du groupe

1. Appuyez sur un groupe pour ouvrir les détails
2. Appuyez sur **Chronologie**
3. Vous verrez tous les messages, annonces et mises à jour du groupe
4. Appuyez sur un message pour voir les réponses et les réactions

## Participation aux groupes

1. Appuyez sur un groupe
2. Appuyez sur **Membres**
3. Vous verrez une liste de tous les membres du groupe
4. Appuyez sur un membre pour voir son profil

## Paramètres du groupe

1. Appuyez sur un groupe
2. Appuyez sur **Paramètres** ou l'icône d'engrenage
3. Vous pouvez :
   - Quitter le groupe
   - Gérer les notifications
   - Afficher les informations du groupe
