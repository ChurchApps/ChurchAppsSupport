# Chronologie du groupe

<div class="article-intro">

La chronologie du groupe vous permet de rester connecté aux groupes dont vous faites partie. Consultez les messages, les mises à jour et les conversations en temps réel.

</div>

## Accès à votre chronologie du groupe

1. Ouvrez B1 Mobile
2. Appuyez sur l'icône **Groupes**
3. Sélectionnez un groupe
4. Vous verrez la **Chronologie** du groupe avec tous les messages et mises à jour

## Types de messages

- **Annonces** — Messages importants des chefs de groupe
- **Conversations** — Discussions générales du groupe
- **Mises à jour** — Événements de groupe et changements
- **Réactions** — Réactions des membres aux messages

## Participation aux conversations

1. Appuyez sur un message pour répondre
2. Tapez votre réaction ou réponse
3. Appuyez sur **Envoyer**

Votre message sera affiché dans la conversation pour que les autres puissent voir.

## Notifications

Les notifications push vous alertent des nouveaux messages du groupe. Vous pouvez gérer vos paramètres de notification dans les **Paramètres** de l'application.
