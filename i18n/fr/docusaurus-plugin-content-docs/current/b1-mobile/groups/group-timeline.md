# Fil d'actualité du groupe

<div class="article-intro">

L'écran Groupes dans B1 Mobile comprend un fil d'actualité qui rassemble l'activité de tous vos groupes en un flux unifié. Au lieu de vérifier chaque groupe individuellement, vous pouvez faire défiler un seul fil pour voir tout ce qui se passe.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](../getting-started/installing) et [connectez-vous](../getting-started/logging-in) à votre compte
- Être membre d'au moins un groupe dans votre église

</div>

## Comment ça fonctionne

Le fil d'actualité du groupe fusionne les publications, conversations et mises à jour de tous les groupes auxquels vous appartenez. Lorsque les responsables et les membres du groupe partagent du contenu, celui-ci apparaît dans le fil par ordre chronologique. Cela signifie que vous obtenez une image complète de votre vie de groupe sans avoir à basculer entre différents groupes.

## Ce qui apparaît dans le fil d'actualité

Le fil d'actualité comprend une variété de contenus provenant de tous vos groupes :

- **Publications** -- Messages et mises à jour partagés par les responsables et les membres du groupe.
- **Conversations** -- Discussions en cours au sein de vos groupes.
- **Mises à jour** -- Annonces, rappels d'événements et autres activités de vos groupes.

Chaque élément du fil d'actualité indique de quel groupe il provient, pour que vous connaissiez toujours le contexte de ce que vous lisez.

## Rester à jour

Le fil d'actualité est un excellent moyen de rester connecté avec votre communauté d'église en un coup d'oeil. Voici quelques conseils pour en tirer le meilleur parti :

- **Consultez-le régulièrement** -- Ouvrez l'écran Groupes et faites défiler le fil d'actualité pour rattraper l'activité récente.
- **Appuyez pour en savoir plus** -- Si une publication attire votre attention, appuyez dessus ou sur le nom du groupe pour accéder à la page complète des [détails du groupe](./group-details) pour plus de contexte.
- **Tout au même endroit** -- Vous n'avez pas besoin de visiter chaque groupe séparément. Le fil d'actualité rassemble toute l'activité pour vous.

:::info
Le fil d'actualité n'affiche que l'activité des groupes associés à votre église. Si vous ne voyez aucune publication, cela peut signifier que vos groupes n'ont pas eu d'activité récente, ou que vous n'êtes pas encore membre d'un groupe.
:::

:::tip
Activez les [notifications push](../community/notifications.md) pour être alerté lorsque de nouvelles publications apparaissent dans vos groupes, même lorsque vous n'êtes pas dans l'application.
:::
