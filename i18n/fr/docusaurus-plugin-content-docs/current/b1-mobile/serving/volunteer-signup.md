# Inscription des bénévoles

<div class="article-intro">

Inscrivez-vous pour les positions ouvertes directement depuis B1 Mobile. Parcourez les opportunités de bénévolat disponibles et confirmez votre participation en quelques appels.

</div>

## Parcourir les positions ouvertes

1. Ouvrez B1 Mobile
2. Appuyez sur l'icône **Service** 
3. Appuyez sur **Positions ouvertes** ou **Inscription**
4. Vous verrez une liste des positions disponibles organisées par catégorie

## S'inscrire

1. Appuyez sur une position qui vous intéresse
2. Consultez les détails :
   - **Description de la position**
   - **Date et heure du service**
   - **Nombre de places disponibles**
3. Appuyez sur **S'inscrire**
4. Confirmez votre inscription

Un message de confirmation affichera que vous êtes inscrit pour cette position.

## Gérer vos inscriptions

1. Appuyez sur **Service** dans le menu
2. Appuyez sur **Mes inscriptions**
3. Vous verrez toutes les positions pour lesquelles vous vous êtes auto-inscrit
4. Pour annuler, appuyez sur la position et sélectionnez **Annuler l'inscription**

:::tip
Vous pouvez aussi voir vos plans assignés et vos inscriptions d'auto-signup au même endroit.
:::
