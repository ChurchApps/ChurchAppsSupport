# Modifier votre profil

<div class="article-intro">

L'onglet **Profil** vous permet de mettre à jour vos informations personnelles afin que votre église dispose toujours des bonnes informations. Les modifications que vous effectuez ici sont reflétées dans tous les services ChurchApps, y compris B1.church et toutes les autres applications utilisées par votre église.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](../getting-started/installing) et [connectez-vous](../getting-started/logging-in) à votre compte

</div>

## Ce que vous pouvez modifier

Votre profil comprend les informations personnelles suivantes :

- **Prénom et nom** -- Votre nom d'affichage dans ChurchApps
- **Adresse e-mail** -- Utilisée pour les communications de l'église et la récupération de compte
- **Numéro de téléphone** -- Pour que votre église puisse vous joindre en cas de besoin
- **Adresse** -- Votre adresse postale pour les dossiers de l'église
- **Photo de profil** -- L'image qui vous représente dans l'[annuaire](../community/member-directory.md) et dans toute l'application

## Mettre à jour vos informations

1. Ouvrez le menu latéral et appuyez sur **Profil**.
2. Sélectionnez l'onglet **Profil** s'il n'est pas déjà actif.
3. Appuyez sur le champ que vous souhaitez modifier et effectuez vos changements.
4. Enregistrez vos modifications lorsque vous avez terminé.

Vos informations mises à jour sont synchronisées automatiquement. Toute personne qui consulte votre profil (sous réserve de vos [paramètres de confidentialité](./privacy-settings)) verra les nouvelles informations immédiatement.

:::tip
Gardez votre adresse e-mail et votre numéro de téléphone à jour. Votre église utilise ces informations pour vous envoyer des mises à jour et des rappels importants.
:::

## Photo de profil

Vous pouvez ajouter ou modifier votre photo de profil depuis l'onglet **Profil**. Votre photo apparaît dans l'[annuaire des membres](../community/member-directory.md), les listes de membres des groupes et partout où votre profil est affiché.

## Modifications dans ChurchApps

Comme tous les services ChurchApps partagent le même système de compte, une modification effectuée dans B1 Mobile est immédiatement visible dans [B1.church](../../b1-church/), les outils d'administration de votre église et tous les autres services connectés. Vous n'avez pas besoin de mettre à jour vos informations séparément dans chaque endroit.

:::info
Certaines églises exigent l'approbation d'un administrateur pour certaines modifications de profil. Si votre mise à jour n'apparaît pas immédiatement, vérifiez l'onglet **En attente** sur votre écran de profil pour connaître son statut.
:::
