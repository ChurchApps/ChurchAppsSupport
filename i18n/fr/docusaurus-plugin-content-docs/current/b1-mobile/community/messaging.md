# Messagerie

<div class="article-intro">

B1 Mobile inclut la messagerie privée pour que vous puissiez avoir des conversations individuelles avec d'autres membres de l'église directement à partir de l'application. C'est un moyen simple de rester en contact sans avoir besoin de partager les numéros de téléphone personnels ou les adresses e-mail.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](../getting-started/installing) et [connectez-vous](../getting-started/logging-in) à votre compte

</div>

## Accès à vos messages

1. Appuyez sur **Messaging** dans la navigation de l'application.
2. Vos conversations précédentes sont listées pour un accès rapide.

## Commencer une nouvelle conversation

1. À partir de l'écran de messagerie, appuyez sur l'option pour commencer une nouvelle conversation.
2. Utilisez le champ de recherche pour trouver le membre que vous souhaitez contacter. Vous pouvez également trouver des membres via l'[Annuaire des membres](./member-directory).
3. Sélectionnez le membre dans les résultats de la recherche.
4. Tapez votre message et appuyez sur **Send**.

Votre nouvelle conversation apparaîtra maintenant dans votre liste de conversations pour que vous puissiez facilement la continuer plus tard.

## Lecture et envoi de messages

Lorsque vous ouvrez une conversation, vous verrez le fil de message complet. Chaque message affiche l'avatar et le nom de l'expéditeur, ce qui facilite le suivi. Pour répondre, tapez votre message dans le champ de texte en bas de l'écran et appuyez sur **Send**.

## Édition de messages

Si vous avez besoin de corriger quelque chose après l'envoi d'un message, vous pouvez l'éditer. Appuyez sur le message que vous souhaitez modifier, sélectionnez l'option d'édition, apportez votre correction et enregistrez. Le message mis à jour remplacera l'original dans la conversation.

## Votre liste de conversations

Toutes vos conversations actives apparaissent dans une liste sur l'écran de messagerie principal. La liste affiche le nom et l'avatar de l'autre personne, ce qui facilite la recherche de la conversation que vous cherchez. Appuyez sur n'importe quelle conversation pour l'ouvrir et continuer là où vous l'aviez laissée.

## Suppression d'une conversation

Si vous souhaitez supprimer une conversation de votre liste, appuyez sur l'icône de la corbeille sur le côté droit de la ligne de conversation et confirmez. La conversation est supprimée de votre vue ; l'autre personne peut toujours voir sa copie du fil de son côté. Pour continuer à parler avec la même personne plus tard, commencez simplement une nouvelle conversation.

:::tip
Utilisez la messagerie pour coordonner avec des bénévoles, faire un suivi auprès de quelqu'un que vous avez rencontré à l'église, ou simplement prendre des nouvelles d'un ami. Cela garde vos communications d'église dans un endroit pratique.
:::

:::info
Activez les [notifications push](./notifications) pour être alerté lorsque quelqu'un vous envoie un nouveau message, même lorsque l'application n'est pas ouverte.
:::
