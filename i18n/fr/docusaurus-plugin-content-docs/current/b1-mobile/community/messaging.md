# Messagerie

<div class="article-intro">

B1 Mobile inclut une messagerie privée pour que vous puissiez avoir des conversations individuelles avec d'autres membres de l'église directement depuis l'application. C'est un moyen simple de rester en contact sans avoir besoin de partager vos numéros de téléphone personnels ou vos adresses e-mail.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](../getting-started/installing) et [connectez-vous](../getting-started/logging-in) à votre compte

</div>

## Accéder à vos messages

1. Appuyez sur **Messagerie** dans la navigation de l'application.
2. Vos conversations précédentes sont listées pour un accès rapide.

## Démarrer une nouvelle conversation

1. Depuis l'écran de messagerie, appuyez sur l'option pour démarrer une nouvelle conversation.
2. Utilisez le champ de recherche pour trouver le membre que vous souhaitez contacter. Vous pouvez également trouver des membres via l'[Annuaire des membres](./member-directory).
3. Sélectionnez le membre dans les résultats de recherche.
4. Tapez votre message et appuyez sur **Envoyer**.

Votre nouvelle conversation apparaîtra désormais dans votre liste de conversations pour que vous puissiez la reprendre facilement plus tard.

## Lire et envoyer des messages

Lorsque vous ouvrez une conversation, vous verrez l'intégralité du fil de messages. Chaque message affiche l'avatar et le nom de l'expéditeur, ce qui facilite le suivi. Pour répondre, tapez votre message dans le champ de texte en bas de l'écran et appuyez sur **Envoyer**.

## Modifier des messages

Si vous devez corriger quelque chose après avoir envoyé un message, vous pouvez le modifier. Appuyez sur le message que vous souhaitez changer, sélectionnez l'option de modification, effectuez votre correction et enregistrez. Le message mis à jour remplacera l'original dans la conversation.

## Votre liste de conversations

Toutes vos conversations actives apparaissent dans une liste sur l'écran principal de messagerie. La liste affiche le nom et l'avatar de l'autre personne, ce qui facilite la recherche de la conversation souhaitée. Appuyez sur une conversation pour l'ouvrir et reprendre là où vous en étiez.

:::tip
Utilisez la messagerie pour coordonner avec d'autres bénévoles, faire un suivi avec quelqu'un que vous avez rencontré à l'église, ou simplement prendre des nouvelles d'un ami. Cela regroupe vos communications d'église en un seul endroit pratique.
:::

:::info
Activez les [notifications push](./notifications) pour être alerté lorsque quelqu'un vous envoie un nouveau message, même lorsque l'application n'est pas ouverte.
:::
