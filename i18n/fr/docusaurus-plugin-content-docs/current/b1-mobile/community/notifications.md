# Notifications

<div class="article-intro">

B1 Mobile utilise les notifications push pour vous tenir informé de ce qui se passe dans votre église. Au lieu de devoir ouvrir l'application pour vérifier les mises à jour, les informations importantes vous parviennent automatiquement.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](../getting-started/installing) et [connectez-vous](../getting-started/logging-in) à votre compte
- Autorisez les notifications pour B1 Mobile dans les paramètres de votre appareil

</div>

## Ce qui vous sera notifié

Les notifications peuvent couvrir un éventail d'activités, notamment :

- **Messages** -- Lorsque quelqu'un vous envoie un [message privé](./messaging).
- **Activité des groupes** -- Mises à jour, publications et conversations des [groupes](../groups/) auxquels vous appartenez.
- **Affectations de service** -- Lorsque vous êtes assigné à un [plan](../serving/viewing-plans.md) ou lorsque le statut de votre affectation change.
- **Autres mises à jour** -- Annonces générales et activités vous concernant.

## Consulter vos notifications

1. Appuyez sur l'**icône de cloche** dans le coin supérieur droit de l'application, ou appuyez sur l'option **Notifications** dans la navigation.
2. L'écran des notifications affiche toutes vos notifications récentes dans une vue à onglets, organisée pour que vous puissiez rapidement trouver ce que vous recherchez.

Chaque notification inclut un bref résumé de la mise à jour. Appuyez sur n'importe quelle notification pour accéder directement au contenu concerné -- par exemple, appuyer sur une notification de message ouvre la conversation, et appuyer sur une notification de service vous amène au plan.

## Notifications push

B1 Mobile envoie des notifications push à votre appareil pour que vous receviez les mises à jour même lorsque l'application n'est pas ouverte. Pour recevoir les notifications push, assurez-vous d'avoir autorisé les notifications pour B1 Mobile dans les paramètres de votre appareil.

### Activer les notifications push

- **iOS** -- Allez dans **Réglages** > **Notifications** > **B1 Mobile** et activez **Autoriser les notifications**.
- **Android** -- Allez dans **Paramètres** > **Applications** > **B1 Mobile** > **Notifications** et assurez-vous que les notifications sont activées.

:::warning
Si vous ne recevez pas de notifications, vérifiez les paramètres de votre appareil pour vous assurer qu'elles sont activées. Confirmez également que vous êtes connecté à l'application, car les notifications sont liées à votre compte.
:::

:::tip
Vous pouvez gérer les préférences de notification au niveau de l'appareil. Si vous trouvez les notifications trop fréquentes, ajustez les paramètres de votre appareil pour contrôler comment et quand les alertes apparaissent.
:::
