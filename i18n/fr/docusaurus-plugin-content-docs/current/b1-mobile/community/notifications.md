# Notifications

<div class="article-intro">

B1 Mobile utilise les notifications push pour vous tenir informé de ce qui se passe à votre église. Au lieu de devoir ouvrir l'application pour vérifier les mises à jour, les informations importantes vous arrivent automatiquement.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Installez B1 Mobile](../getting-started/installing) et [connectez-vous](../getting-started/logging-in) à votre compte
- Autorisez les notifications pour B1 Mobile dans les paramètres de votre appareil

</div>

## À propos de quoi vous serez notifié

Les notifications peuvent couvrir une gamme d'activités, notamment :

- **Messages** -- Quand quelqu'un vous envoie un [message privé](./messaging).
- **Activité de groupe** -- Mises à jour, messages et conversations des [groupes](../groups/) auxquels vous appartenez.
- **Assignations de service** -- Quand vous êtes assigné à un [plan](../serving/viewing-plans.md) ou quand l'état de votre assignation change.
- **Autres mises à jour** -- Annonces générales et activités pertinentes pour vous.

## Affichage de vos notifications

1. Appuyez sur l'**icône de cloche** dans le coin supérieur droit de l'application, ou appuyez sur l'option **Notifications** dans la navigation.
2. L'écran des notifications affiche toutes vos notifications récentes dans une vue à onglets, organisées pour que vous puissiez rapidement trouver ce que vous cherchez.

Chaque notification inclut un bref résumé de la mise à jour. Appuyez sur n'importe quelle notification pour aller directement au contenu pertinent -- par exemple, appuyer sur une notification de message ouvre la conversation, et appuyer sur une notification de service vous amène au plan.

## Suppression de notifications

Vous pouvez effacer les notifications une fois que vous les avez lues pour que votre liste reste concentrée sur ce qui est nouveau.

- **Supprimer une seule notification** -- Appuyez sur l'icône de la corbeille sur le côté droit de n'importe quelle ligne de notification pour supprimer juste cet élément.
- **Effacer toutes les notifications** -- Appuyez sur **Clear All** en haut de la liste des notifications pour supprimer chaque notification à la fois. Vous serez invité à confirmer avant que tout soit effacé.

La suppression d'une notification n'affecte que votre vue ; cela ne change pas l'activité sous-jacente (le message, le plan ou le message du groupe est toujours là). Si vous avez les notifications activées sur plusieurs appareils, l'effacement sur un appareil se synchronisera avec les autres.

## Notifications push

B1 Mobile envoie les notifications push à votre appareil pour que vous receviez les mises à jour même lorsque l'application n'est pas ouverte. Pour recevoir les notifications push, assurez-vous d'avoir autorisé les notifications pour B1 Mobile dans les paramètres de votre appareil.

### Activation des notifications push

- **iOS** -- Allez à **Settings** > **Notifications** > **B1 Mobile** et activez **Allow Notifications**.
- **Android** -- Allez à **Settings** > **Apps** > **B1 Mobile** > **Notifications** et assurez-vous que les notifications sont activées.

:::warning
Si vous ne recevez pas les notifications, vérifiez les paramètres de votre appareil pour vous assurer qu'elles sont activées. Confirmez également que vous êtes connecté à l'application, car les notifications sont liées à votre compte.
:::

:::tip
Vous pouvez gérer les préférences de notification au niveau de l'appareil. Si vous trouvez les notifications trop fréquentes, ajustez les paramètres de votre appareil pour contrôler comment et quand les alertes apparaissent.
:::
