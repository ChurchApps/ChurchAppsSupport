# Examen de votre ménage

<div class="article-intro">

Après avoir sélectionné une personne dans les résultats de recherche, l'application charge votre ménage entier et l'affiche sur l'écran d'examen du ménage. C'est le hub central du processus d'enregistrement où vous pouvez voir toutes les personnes de votre famille et gérer leurs attributions de groupe avant de vous enregistrer.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- [Sélectionnez un service](./selecting-service) à partir de l'écran des services
- [Recherchez un membre](./looking-up-members) et sélectionnez-le dans les résultats de recherche

</div>

## Affichage des membres du ménage

Chaque membre du ménage apparaît sous la forme d'une carte avec :

- Leur **photo** (ou une icône de placeholder si aucune photo n'est sur fichier)
- Leur **nom d'affichage**
- Un **résumé de leurs attributions de groupe actuelles** (affiché sous forme de petites puces sous le nom quand la carte est réduite)
- Une **flèche d'expansion/réduction** sur le côté droit

## Expansion d'un membre

Appuyez sur la carte d'un membre pour l'étendre. Lorsqu'elle est étendue, vous verrez une rangée pour chaque **heure de service** au sein du service sélectionné. Chaque rangée affiche :

- Le **nom de l'heure de service** (par exemple, « 9h00 » ou « 11h00 »)
- Un **bouton** affichant le groupe actuellement assigné, ou « Select Group » si aucun groupe n'a encore été assigné

Appuyez sur le bouton du groupe pour aller à l'[écran de sélection du groupe](./group-assignment) où vous pouvez choisir ou modifier le groupe pour cette personne et cette heure de service.

La carte développée affiche également les puces **Member**, **Guest** et **Volunteer**. Appuyez sur une pour définir comment cette personne s'enregistre : Member est la valeur par défaut. Le type a de l'importance pour les salles pour enfants : les bénévoles comptent pour le ratio bénévole-enfant de la salle, et les invités comptent par rapport à toute capacité d'invité définie par votre église.

Appuyez à nouveau sur la carte du membre pour la réduire.

:::tip
Vous n'avez pas besoin d'attribuer chaque membre de la famille à un groupe. Les adultes qui assistent au service de culte principal peuvent rester sans attribution de groupe. Attribuez uniquement des groupes aux membres qui doivent être enregistrés dans une classe ou un programme spécifique.
:::

## Navigation à partir de cet écran

À partir de l'écran d'examen du ménage, vous avez trois options :

- **Développez un membre** et appuyez sur un bouton d'heure de service pour [assigner ou modifier un groupe](./group-assignment).
- **Appuyez sur le bouton Add Guest** sous la liste des membres pour [ajouter un invité](./adding-guests) à votre ménage.
- **Appuyez sur le bouton Check-in** au bas de l'écran pour [finaliser l'enregistrement](./completing-checkin) pour toutes les personnes du ménage.

:::info
Toutes les attributions de groupe d'un enregistrement précédent au même service sont préchargées automatiquement. Si vos enfants assistent aux mêmes groupes chaque semaine, vous n'aurez peut-être rien à modifier : consultez simplement et enregistrez-vous.
:::
