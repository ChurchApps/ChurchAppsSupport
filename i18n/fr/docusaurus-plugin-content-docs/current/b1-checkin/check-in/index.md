# Processus d'enregistrement

<div class="article-intro">

Le processus d'enregistrement dans B1 Church Checkin suit un flux simple, étape par étape. Un membre de l'église s'approche du kiosque, trouve sa famille, confirme les affectations de groupe et s'enregistre -- le tout en moins d'une minute. Cette section couvre chaque étape de l'expérience d'enregistrement au kiosque.

</div>

## Le processus d'enregistrement en un coup d'œil

1. **[Sélectionner un culte](./selecting-service)** -- Choisissez le culte auquel vous assistez (par exemple, Dimanche matin, Dimanche soir, Mi-semaine).
2. **[Rechercher votre famille](./looking-up-members)** -- Recherchez par numéro de téléphone ou nom de famille pour vous trouver dans le système.
3. **[Vérifier votre foyer](./household-review)** -- Consultez tous les membres de la famille et leurs affectations de groupe actuelles.
4. **[Assigner des groupes](./group-assignment)** -- Choisissez la classe ou le groupe auquel chaque membre de la famille doit participer pour chaque créneau horaire de culte.
5. **[Ajouter des visiteurs](./adding-guests)** (optionnel) -- Ajoutez un visiteur à votre foyer s'il vous accompagne.
6. **[Finaliser l'enregistrement](./completing-checkin)** -- Appuyez sur le bouton d'enregistrement pour soumettre la présence et imprimer les étiquettes.

Une fois l'enregistrement terminé, l'application affiche un écran de confirmation et revient automatiquement à l'écran de recherche, prête pour la famille suivante.

:::tip
L'ensemble du processus est conçu pour que les membres de l'église puissent l'utiliser de manière autonome. Les bénévoles n'ont pas besoin d'opérer le kiosque, bien qu'il soit toujours judicieux d'avoir quelqu'un à proximité pour aider les visiteurs qui viennent pour la première fois.
:::

## Avant de commencer

Assurez-vous que les éléments suivants sont configurés dans B1 Admin avant d'utiliser le kiosque d'enregistrement :

- **Les cultes et les horaires de culte** sont configurés dans les paramètres de présence. Consultez le [guide de configuration de la présence](../../b1-admin/attendance/setup.md).
- **Les groupes** sont créés et associés aux horaires de culte appropriés. Consultez [création de groupes](../../b1-admin/groups/creating-groups.md).
- **Les paramètres de groupe** sont configurés pour l'impression d'étiquettes : activez **Print Nametag** pour les groupes nécessitant des étiquettes nominatives, et **Parent Pickup** pour les groupes d'enfants nécessitant des reçus de récupération.
- **Les personnes** sont ajoutées au système avec des associations de foyer. Consultez [ajout de personnes](../../b1-admin/people/adding-people.md).

:::info
Si vous n'avez pas encore configuré le kiosque lui-même, consultez [Premiers pas](../getting-started/) pour les instructions d'installation, de connexion et de configuration de l'imprimante.
:::
