# Flux de travail d'enregistrement

<div class="article-intro">

Le processus d'enregistrement dans B1 Church Checkin suit un flux simple étape par étape. Un membre de l'église se présente au kiosque, trouve sa famille, confirme les attributions de groupe et s'enregistre, le tout en moins d'une minute. Cette section couvre chaque étape de l'expérience d'enregistrement du kiosque.

</div>

## Le flux d'enregistrement en un coup d'œil

1. **[Sélectionnez un service](./selecting-service)** -- Choisissez le service auquel vous assistez (par exemple, dimanche AM, dimanche PM, entre-semaine).
2. **[Recherchez votre famille](./looking-up-members)** -- Recherchez par numéro de téléphone ou nom de famille pour vous trouver dans le système.
3. **[Examinez votre ménage](./household-review)** -- Voyez tous les membres de la famille et leurs attributions de groupe actuelles.
4. **[Attribuez des groupes](./group-assignment)** -- Choisissez la classe ou le groupe auquel chaque membre de la famille doit assister pour chaque heure de service.
5. **[Ajouter des invités](./adding-guests)** (optionnel) -- Ajoutez un visiteur à votre ménage s'il vous rejoint.
6. **[Finaliser l'enregistrement](./completing-checkin)** -- Appuyez sur le bouton d'enregistrement pour soumettre la présence et imprimer les étiquettes.
7. **[Retrait](./checking-out)** (postes gérés) -- Au moment de la récupération, vérifiez le code de sécurité et la personne de récupération, avec les listes de récupération de confiance, la notification de parent et les réimpressions d'étiquettes.

Après la fin de l'enregistrement, l'application affiche un écran de succès et revient automatiquement à l'écran de recherche, prêt pour la famille suivante.

:::tip
L'ensemble du flux est conçu pour que les membres de l'église l'utilisent indépendamment. Les bénévoles n'ont pas besoin d'exploiter le kiosque, bien qu'avoir quelqu'un à proximité pour aider les visiteurs pour la première fois soit toujours une bonne idée.
:::

## Avant de commencer

Assurez-vous que les éléments suivants sont configurés dans B1 Admin avant d'utiliser le kiosque d'enregistrement :

- Les **services et les heures de service** sont configurés dans les paramètres de présence. Consultez le [guide de configuration de la présence](../../b1-admin/attendance/setup.md).
- Les **groupes** sont créés et liés aux heures de service appropriées. Consultez [créer des groupes](../../b1-admin/groups/creating-groups.md).
- Les **paramètres de groupe** sont configurés pour l'impression d'étiquettes : activez **Print Nametag** pour les groupes qui ont besoin d'étiquettes nominatives, et **Parent Pickup** pour les groupes d'enfants qui ont besoin de bordereaux de récupération.
- Les **personnes** sont ajoutées au système avec les associations de ménage. Consultez [ajouter des personnes](../../b1-admin/people/adding-people.md).

:::info
Si vous n'avez pas encore configuré le kiosque lui-même, consultez [Démarrage](../getting-started/) pour les instructions d'installation, de connexion et de configuration de l'imprimante.
:::
