# Connexion

<div class="article-intro">

Avant que le kiosque puisse être utilisé pour l'enregistrement, vous devez vous connecter avec un compte ChurchApps ayant accès à votre église. Cette page couvre le processus de première connexion, la sélection de l'église, le comportement de connexion automatique et la procédure de déconnexion.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Installez B1 Church Checkin sur votre tablette Android (voir [Premiers pas](./index.md))
- Ayez votre adresse e-mail et votre mot de passe de compte ChurchApps à portée de main

</div>

## Première connexion

1. Ouvrez l'application B1 Church Checkin. Après l'écran de démarrage, vous verrez l'écran de connexion avec le logo B1 Church Checkin.
2. Saisissez votre **adresse e-mail** dans le premier champ.
3. Saisissez votre **mot de passe** dans le second champ.
4. Appuyez sur le bouton **Login**.

L'application valide vos identifiants et vous connecte. Si l'adresse e-mail ou le mot de passe est incorrect, un message d'erreur s'affichera en bas de l'écran.

:::info
Il s'agit des mêmes adresse e-mail et mot de passe que vous utilisez pour B1 Admin ou tout autre produit ChurchApps. Vous n'avez pas besoin d'un compte séparé pour le kiosque d'enregistrement.
:::

## Sélection de votre église

Après une connexion réussie, l'application charge la liste des églises liées à votre compte. Chaque église est affichée sous forme de carte montrant le nom de l'église et son logo (si celui-ci a été configuré dans B1 Admin).

- Si votre compte est lié à **une seule église**, vous verrez tout de même l'écran de sélection. Appuyez sur votre église pour continuer.
- Si votre compte est lié à **plusieurs églises**, appuyez sur l'église que vous souhaitez que ce kiosque desserve.

Une fois que vous avez sélectionné une église, l'application charge l'image de marque et les paramètres d'apparence de cette église. Le logo de votre église apparaîtra dans l'en-tête sur tous les écrans suivants.

## Connexion automatique lors des visites suivantes

L'application enregistre votre adresse e-mail, votre mot de passe et l'église sélectionnée après la première connexion réussie. Lors des prochains lancements :

- L'application se connecte automatiquement avec vos identifiants enregistrés.
- Si vous aviez précédemment sélectionné une église, elle restaure cette sélection et vous amène directement à l'**écran des cultes**.
- Si les identifiants enregistrés ne sont plus valides, l'application revient à l'écran de connexion pour que vous puissiez vous reconnecter.

:::tip
Comme le kiosque se connecte automatiquement, vous n'avez besoin de saisir vos identifiants qu'une seule fois lors de la configuration initiale. Après cela, il suffit d'allumer la tablette et elle sera prête à fonctionner.
:::

## Déconnexion

La déconnexion est intentionnellement masquée afin que les membres de l'église utilisant le kiosque ne puissent pas se déconnecter accidentellement. Pour vous déconnecter :

1. Appuyez sur le **logo de l'église** dans l'en-tête **sept fois** rapidement (en moins de deux secondes).
2. Une boîte de dialogue de confirmation apparaîtra vous demandant si vous souhaitez vous déconnecter.
3. Appuyez sur **Logout** pour confirmer, ou sur **Cancel** pour rester connecté.

Cela efface vos identifiants enregistrés et vous ramène à l'écran de connexion.

:::warning
La déconnexion efface tous les identifiants enregistrés et la sélection d'église. Vous devrez saisir à nouveau votre adresse e-mail et votre mot de passe pour utiliser le kiosque. Ne vous déconnectez que lorsque vous devez changer de compte ou reconfigurer le kiosque.
:::
