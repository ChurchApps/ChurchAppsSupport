# Connexion

<div class="article-intro">

Avant que le kiosque puisse être utilisé pour l'accueil, vous devez vous connecter avec un compte ChurchApps qui a accès à votre église. Cette page couvre le processus de première connexion, la sélection de l'église, le comportement de la connexion automatique et comment se déconnecter.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Installez B1 Church Checkin sur votre tablette Android (voir [Démarrage](./index.md))
- Ayez votre e-mail et votre mot de passe du compte ChurchApps prêts

</div>

## Première connexion

1. Ouvrez l'application B1 Church Checkin. Après l'écran de démarrage, vous verrez l'écran de connexion avec le logo B1 Church Checkin.
2. Entrez votre **adresse e-mail** dans le premier champ.
3. Entrez votre **mot de passe** dans le deuxième champ.
4. Appuyez sur le bouton **Connexion**.

L'application valide vos identifiants et vous connecte. Si l'e-mail ou le mot de passe est incorrect, vous verrez un message d'erreur au bas de l'écran.

:::info
C'est le même e-mail et mot de passe que vous utilisez pour B1 Admin ou tout autre produit ChurchApps. Vous n'avez pas besoin d'un compte séparé pour le kiosque d'accueil.
:::

## Sélection de votre église

Après une connexion réussie, l'application charge une liste des églises liées à votre compte. Chaque église s'affiche sous forme de carte montrant le nom et le logo de l'église (s'il a été configuré dans B1 Admin).

- Si votre compte est lié à **une seule église**, vous verrez toujours l'écran de sélection. Appuyez sur votre église pour continuer.
- Si votre compte est lié à **plusieurs églises**, appuyez sur l'église que ce kiosque doit servir.

Une fois que vous sélectionnez une église, l'application charge les paramètres de marque et d'apparence de cette église. Votre logo d'église apparaîtra dans l'en-tête sur tous les écrans suivants.

## Connexion automatique lors des visites de retour

L'application enregistre votre e-mail, votre mot de passe et l'église sélectionnée après la première connexion réussie. Lors de lancements ultérieurs :

- L'application se connecte automatiquement avec vos identifiants enregistrés.
- Si vous avez précédemment sélectionné une église, elle restaure cette sélection et vous mène directement à l'**écran des services**.
- Si les identifiants enregistrés ne sont plus valides, l'application revient à l'écran de connexion pour que vous puissiez vous connecter à nouveau.

:::tip
Parce que le kiosque se connecte automatiquement, vous n'avez besoin d'entrer les identifiants qu'une seule fois lors de la configuration initiale. Après cela, la tablette peut être allumée et elle sera prête à fonctionner.
:::

## Paramètres d'administration

Les paramètres d'administration sont intentionnellement cachés pour que les membres de l'église utilisant le kiosque ne puissent pas modifier accidentellement la configuration. Pour y accéder :

1. Appuyez sur le **logo de l'église** dans l'en-tête **sept fois** en rapide succession (en deux secondes).
2. Si un code PIN de kiosque est défini, entrez le code PIN quand vous y êtes invité.
3. L'écran **Paramètres d'administration** s'ouvrira avec les options suivantes :
   - **Changer le service** — Basculez l'heure de service sélectionnée
   - **Changer l'imprimante** — Ouvrir [paramètres d'imprimante](./printer-setup) pour scanner et sélectionner une imprimante
   - **Définir/Changer le code PIN** — Définir ou mettre à jour le code PIN du kiosque qui protège l'accès administrateur
   - **Se déconnecter** — Se déconnecter et revenir à l'écran de connexion

:::warning
La déconnexion efface tous les identifiants et la sélection d'église enregistrés. Vous devrez entrer à nouveau votre e-mail et votre mot de passe pour utiliser le kiosque. Déconnectez-vous uniquement lorsque vous avez besoin de changer de comptes ou de reconfigurer le kiosque.
:::
