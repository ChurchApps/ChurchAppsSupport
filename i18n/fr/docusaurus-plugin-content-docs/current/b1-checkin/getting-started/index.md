# Premiers pas avec B1 Church Checkin

<div class="article-intro">

La mise en place de B1 Church Checkin ne prend que quelques minutes. Cette section vous guide à travers l'installation de l'application sur une tablette Android, la connexion avec votre compte ChurchApps, et la configuration optionnelle d'une imprimante d'étiquettes réseau.

</div>

## Ce dont vous avez besoin

- **Une tablette Android** -- Une tablette de 10 pouces est idéale pour l'expérience kiosque.
- **Un compte ChurchApps** -- L'adresse e-mail et le mot de passe que vous utilisez pour B1 Admin ou d'autres produits ChurchApps.
- **Une imprimante d'étiquettes réseau** (optionnel) -- Pour imprimer les étiquettes nominatives et les reçus de récupération parentale. L'application peut scanner votre réseau local pour trouver les imprimantes compatibles.

## Installation

1. Ouvrez le **Google Play Store** sur votre tablette Android.
2. Recherchez **B1 Church Checkin**.
3. Appuyez sur **Installer** et attendez la fin du téléchargement.
4. Ouvrez l'application. Vous verrez un écran de démarrage pendant l'initialisation.

## Configuration initiale

Lorsque vous lancez l'application pour la première fois, vous passerez par trois étapes :

1. **[Connexion](./logging-in)** -- Saisissez votre adresse e-mail et votre mot de passe pour vous authentifier.
2. **[Sélection de votre église](./logging-in)** -- Si votre compte est lié à plusieurs églises, choisissez celle que ce kiosque doit desservir.
3. **[Configuration d'une imprimante](./printer-setup)** -- Configurez une imprimante réseau pour l'impression d'étiquettes, ou ignorez cette étape si vous n'avez pas besoin d'étiquettes.

Après la première connexion, l'application enregistre vos identifiants et votre sélection d'église. Lors des prochains lancements, elle accédera automatiquement à l'écran des cultes.

:::info
L'application vérifie les mises à jour automatiques à chaque démarrage. Si une mise à jour est disponible, elle est téléchargée et appliquée automatiquement avant que vous n'atteigniez l'écran de connexion.
:::

:::tip
Assurez-vous que l'administrateur de votre église a configuré les cultes, les horaires de culte et les groupes dans B1 Admin avant de mettre en place le kiosque. Consultez le [guide de configuration de la présence](../../b1-admin/attendance/setup.md) pour plus de détails.
:::
