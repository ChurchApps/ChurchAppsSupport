# Fournisseurs de contenu

<div class="article-intro">

Les fournisseurs de contenu sont des services externes qui fournissent des supports de cours, des vidéos et des images pour la lecture sur votre téléviseur. FreePlay prend en charge plusieurs fournisseurs, vous permettant de parcourir et de télécharger du contenu provenant de différentes sources, le tout dans une seule application.

</div>

## Fournisseurs disponibles

FreePlay inclut actuellement les fournisseurs suivants :

- **[Lessons.church](../../lessons-church/)** - Programme gratuit et open source pour le ministère des enfants
- **Bible Project** - Vidéos et ressources explorant la Bible
- **SignPresenter** - Contenu de présentation et d'affichage pour les églises
- **B1.church** - Contenu de votre propre configuration d'église B1

:::info
Certains fournisseurs sont encore en cours d'ajout. Si une carte de fournisseur affiche « Bientôt disponible », il n'est pas encore possible de s'y connecter.
:::

## Comment fonctionnent les fournisseurs

Chaque fournisseur possède sa propre bibliothèque de contenu organisée en dossiers et fichiers. Une fois que vous vous connectez à un fournisseur, son contenu apparaît dans un navigateur de dossiers où vous pouvez explorer les catégories, sélectionner des éléments et les télécharger pour la lecture.

Les fournisseurs connectés apparaissent dans la barre latérale sur le côté gauche de l'écran. Vous pouvez basculer entre les fournisseurs en les sélectionnant dans la barre latérale, ce qui facilite le passage d'une source de contenu à une autre pendant la configuration.
