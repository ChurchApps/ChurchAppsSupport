# Parcourir et télécharger du contenu

<div class="article-intro">

Une fois que vous vous êtes connecté à un fournisseur de contenu, vous pouvez parcourir sa bibliothèque, télécharger des fichiers et les lire sur votre téléviseur. Le navigateur de contenu fournit une grille visuelle de dossiers et de fichiers dans laquelle vous naviguez avec votre télécommande TV.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Connectez-vous à au moins un fournisseur de contenu -- voir [Se connecter aux fournisseurs](./connecting-providers)
- Ayez votre télécommande TV à portée de main pour la navigation

</div>

## Le navigateur de contenu

Le navigateur de contenu affiche les éléments dans une grille de cartes à 3 colonnes. Chaque carte affiche une image miniature et un titre. Il existe deux types d'éléments :

- **Dossiers** - Représentés par une miniature ou une icône de dossier. Sélectionnez un dossier pour voir son contenu.
- **Fichiers** - Éléments multimédias individuels (vidéos ou images). Les fichiers vidéo affichent une icône de lecture superposée sur leur miniature.

L'en-tête en haut de l'écran affiche le nom du dossier actuel, ou le nom du fournisseur si vous êtes au niveau racine.

## Naviguer dans les dossiers

1. Utilisez le pavé directionnel de votre télécommande pour mettre un dossier en surbrillance
2. Appuyez sur **Sélectionner** pour l'ouvrir
3. Le navigateur charge le contenu du dossier, qui peut contenir d'autres dossiers ou des fichiers lisibles
4. Appuyez sur **Retour** sur votre télécommande pour remonter d'un niveau

Vous pouvez naviguer aussi profondément que la structure de contenu du fournisseur le permet. Le navigateur garde une trace du chemin de vos dossiers afin que vous puissiez revenir en arrière à travers chaque niveau.

## Télécharger et lire du contenu

Lorsque vous sélectionnez un dossier contenant des fichiers lisibles (ou sélectionnez un fichier individuel), FreePlay passe à un écran de téléchargement :

1. Le titre du contenu apparaît à l'écran avec une image de couverture du dossier sélectionné
2. Une barre de progression affiche **Téléchargement de l'élément X sur Y** au fur et à mesure que les fichiers sont enregistrés sur votre appareil
3. Une fois le téléchargement terminé, le bouton **Démarrer** apparaît avec une icône de lecture
4. Appuyez sur **Sélectionner** pour commencer la lecture

:::tip
Le contenu téléchargé est mis en cache sur votre téléviseur. Si vous lisez le même contenu ultérieurement, il se charge instantanément sans re-téléchargement.
:::

## Lire le contenu d'un fournisseur

La lecture fonctionne de la même manière que la lecture des leçons. Utilisez ces commandes :

- **Flèche droite** - Élément suivant
- **Flèche gauche** - Élément précédent
- **Sélectionner / Lecture-Pause** - Mettre en pause ou reprendre la vidéo
- **Flèche haut** - Ouvrir le sélecteur de messages pour accéder à un élément spécifique
- **Retour** - Quitter vers le navigateur de contenu

Lorsque vous quittez le lecteur, vous revenez au navigateur de contenu au niveau racine du fournisseur actuel.

:::info
Pour une référence complète de toutes les commandes du lecteur, voir [Lire les leçons](../classroom-mode/playing-lessons).
:::

## Basculer entre les fournisseurs

Les fournisseurs connectés apparaissent comme des éléments dans la barre latérale à gauche. Pour basculer vers un autre fournisseur :

1. Appuyez sur la flèche **Gauche** de votre télécommande pour développer la barre latérale
2. Sélectionnez le fournisseur que vous souhaitez parcourir
3. Le navigateur de contenu s'actualise avec la bibliothèque de ce fournisseur

:::warning
Si un dossier affiche « Aucun contenu disponible », le fournisseur ne possède aucun élément à ce niveau. Essayez de naviguer vers un autre dossier ou vérifiez ultérieurement.
:::
