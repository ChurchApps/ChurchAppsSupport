# Connexion aux fournisseurs de contenu

<div class="article-intro">

Connectez FreePlay à vos fournisseurs de contenu préférés pour accéder à des leçons, des vidéos et d'autres contenus pédagogiques.

</div>

## Connexion d'un fournisseur

1. Ouvrez FreePlay
2. Allez à **Paramètres** > **Fournisseurs de contenu**
3. Appuyez sur **Ajouter un fournisseur**
4. Sélectionnez un fournisseur dans la liste (B1.church, Lessons.church, etc.)
5. Vous verrez une option d'authentification :
   - **Code QR** — Scannez le code QR
   - **Connexion au formulaire** — Entrez e-mail et mot de passe
   - **Authentification par appareil** — Suivez les instructions à l'écran

## Authentification commune

### B1.church
- Connectez-vous avec votre compte ChurchApps
- Autorisez la connexion FreePlay dans B1 Admin

### Lessons.church
- Connectez-vous avec votre compte Lessons.church
- Suivez les instructions de code de dispositif

### Autres fournisseurs
- Chaque fournisseur peut avoir un processus d'authentification différent
- Suivez les invites spécifiques du fournisseur

## Gestion des fournisseurs

1. Allez à **Paramètres** > **Fournisseurs de contenu**
2. Vous verrez tous les fournisseurs connectés
3. Appuyez sur un fournisseur pour voir les options :
   - **Reconnecter** — Si votre jeton a expiré
   - **Déconnecter** — Supprimer l'accès
