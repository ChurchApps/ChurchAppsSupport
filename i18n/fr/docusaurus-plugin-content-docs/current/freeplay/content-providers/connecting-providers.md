# Se connecter aux fournisseurs

<div class="article-intro">

Avant de pouvoir parcourir le contenu d'un fournisseur, vous devez vous y connecter. Certains fournisseurs nécessitent une authentification via un code QR ou une connexion par e-mail, tandis que d'autres peuvent être connectés en un seul clic.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Installez et lancez FreePlay -- voir [Premiers pas](../getting-started/)
- Ayez votre télécommande TV à portée de main pour la navigation
- Pour les fournisseurs nécessitant une connexion, ayez vos identifiants de compte disponibles

</div>

## Parcourir les fournisseurs disponibles

1. Ouvrez l'écran **Fournisseurs de contenu** depuis la barre latérale (sélectionnez l'icône **Fournisseurs** en bas)
2. Vous verrez une grille de cartes de fournisseurs, chacune affichant le logo et le nom du fournisseur
3. Les fournisseurs connectés affichent un badge vert **Connecté** sous leur nom
4. Les fournisseurs qui ne sont pas encore disponibles affichent un libellé **Bientôt disponible**

## Connexion sans authentification

Certains fournisseurs ne nécessitent pas de connexion. Lorsque vous sélectionnez l'un de ces fournisseurs, FreePlay se connecte immédiatement et ouvre le navigateur de contenu. Aucun identifiant n'est nécessaire.

## Authentification par flux d'appareil (code QR)

Certains fournisseurs utilisent un flux d'appareil, similaire à la façon dont vous vous connectez aux applications de streaming sur un téléviseur :

1. Sélectionnez la carte du fournisseur sur l'écran **Fournisseurs de contenu**
2. FreePlay affiche un code QR et une URL de vérification
3. Scannez le code QR avec votre téléphone, ou visitez l'URL affichée sur n'importe quel appareil
4. Entrez le code utilisateur affiché sur l'écran du téléviseur
5. Complétez le processus de connexion sur votre téléphone ou ordinateur
6. FreePlay détecte la connexion réussie et affiche **Connecté !**
7. Le navigateur de contenu s'ouvre automatiquement

:::info
Un indicateur pulsant **En attente d'autorisation** indique que FreePlay vérifie votre connexion. Le code expire après plusieurs minutes, complétez donc le processus rapidement.
:::

## Connexion par formulaire

D'autres fournisseurs utilisent une connexion traditionnelle par e-mail et mot de passe :

1. Sélectionnez la carte du fournisseur
2. Entrez votre **E-mail** et votre **Mot de passe** à l'aide du clavier à l'écran
3. Sélectionnez le bouton **Se connecter**
4. Si vos identifiants sont corrects, FreePlay affiche **Connecté !** et ouvre le navigateur de contenu

:::tip
Utilisez le pavé directionnel de votre télécommande pour vous déplacer entre le champ e-mail, le champ mot de passe et le bouton de connexion. Appuyez sur **Sélectionner** sur un champ de texte pour ouvrir le clavier à l'écran.
:::

## Déconnecter un fournisseur

Pour vous déconnecter d'un fournisseur auquel vous êtes déjà connecté :

1. Accédez à l'écran **Fournisseurs de contenu**
2. Sélectionnez la carte du fournisseur qui affiche le badge **Connecté**
3. Une demande de confirmation vous demande si vous souhaitez vous déconnecter
4. Choisissez **Déconnecter** pour supprimer la connexion

Après la déconnexion, le contenu du fournisseur n'apparaîtra plus dans votre barre latérale.

:::warning
La déconnexion supprime l'authentification enregistrée de votre appareil. Vous devrez vous reconnecter si vous souhaitez rétablir la connexion ultérieurement.
:::
