# Choisir un mode d'appairage

<div class="article-intro">

FreePlay propose deux modes d'appairage pour connecter votre téléviseur à votre église, ainsi que la possibilité de parcourir directement les fournisseurs de contenu. L'écran **Sélectionner le mode d'appairage** présente ces options sous forme de grandes cartes que vous pouvez naviguer avec votre télécommande TV.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Installez FreePlay et lancez-le pour la première fois -- voir [Premiers pas](./index.md)
- Ayez votre télécommande TV à portée de main pour la navigation

</div>

## Appairer à une salle de classe

Choisissez **Appairer à une salle de classe** lorsque vous souhaitez qu'un téléviseur spécifique affiche toujours la leçon programmée pour une salle particulière. C'est la meilleure option pour les salles de classe du ministère des enfants où les leçons sont assignées selon un programme hebdomadaire.

**Comment ça fonctionne :** Vous recherchez votre église par nom, choisissez une salle de classe, et FreePlay récupère automatiquement la leçon programmée pour cette salle aujourd'hui. Chaque dimanche matin, vous allumez simplement le téléviseur et la bonne leçon est prête.

**Idéal pour :** Les téléviseurs de salle de classe dédiés qui doivent toujours afficher le contenu programmé pour cette salle.

:::tip
Configurez vos salles de classe sur [freeplay.church](https://freeplay.church) avant d'appairer votre téléviseur. Cela garantit que vos salles apparaissent lorsque vous recherchez votre église.
:::

## Appairer à un plan

Choisissez **Appairer à un plan** lorsque vous souhaitez suivre un plan de service d'église. Ce mode génère un code d'appairage que vous entrez dans **B1 Admin** pour associer le téléviseur à un type de plan spécifique. Le téléviseur télécharge et lit ensuite le contenu assigné au plan en cours.

**Comment ça fonctionne :** FreePlay affiche un code d'appairage à l'écran. Un administrateur entre ce code dans les paramètres du **Type de plan** dans **B1 Admin**. Une fois appairé, le téléviseur charge automatiquement le contenu du plan en cours.

**Idéal pour :** Les téléviseurs qui affichent du contenu lié à votre plan de service hebdomadaire, comme les paroles de louange ou les diapositives d'annonces.

## Fournisseurs de contenu

Si vous n'avez pas besoin d'appairer à une salle de classe ou à un plan, vous pouvez parcourir directement les fournisseurs de contenu. Des fournisseurs comme [Lessons.church](../../lessons-church/) et Bible Project proposent du contenu que vous pouvez télécharger et lire à la demande.

**Idéal pour :** Les téléviseurs où les bénévoles choisissent manuellement quoi lire chaque semaine, ou lorsque vous souhaitez explorer le contenu disponible.

:::info
Vous pouvez accéder à l'écran **Fournisseurs** à tout moment depuis la barre latérale en bas de la navigation. Vous n'avez pas besoin d'utiliser un mode d'appairage pour parcourir et lire du contenu.
:::
