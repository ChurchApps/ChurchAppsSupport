# Démarrage avec FreePlay

<div class="article-intro">

Installez et configurez FreePlay sur votre téléviseur de salle de classe en quelques minutes.

</div>

## Installation

1. **Trouvez FreePlay** dans l'app store de votre appareil :
   - Amazon Appstore (Fire Stick/Fire TV)
   - Google Play Store (Android TV, Google TV)
   - App Store (Apple TV)
2. Tapez **FreePlay** dans la recherche
3. Appuyez sur **Installer**

## Configuration initiale

1. Ouvrez FreePlay après installation
2. Acceptez les conditions d'utilisation
3. Sélectionnez votre langue
4. Vous arriverez à l'écran **Paramètres** ou **Contenu**

## Connexion d'un fournisseur de contenu

1. Allez à **Paramètres** > **Fournisseurs de contenu**
2. Appuyez sur **Ajouter un fournisseur**
3. Sélectionnez un fournisseur (B1.church, Lessons.church, etc.)
4. Suivez les instructions d'authentification
