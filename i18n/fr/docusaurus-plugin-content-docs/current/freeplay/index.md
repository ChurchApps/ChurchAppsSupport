# Aperçu de FreePlay

<div class="article-intro">

FreePlay est le lecteur multimédia gratuit de ChurchApps pour les écrans de salle de classe. Affichage de contenu vidéo en plein écran, calendriers de leçons et support interactif pour les enseignants.

</div>

## Caractéristiques principales

- **Lecture vidéo plein écran** — Lecteur vidéo dédié sans distraction
- **Contenu de leçon intégré** — Lisez les leçons planifiées depuis B1 Admin ou Lessons.church
- **Support des contrôles de télévision** — Naviguez avec une télécommande standard
- **Calendrier des leçons** — Affichez les leçons planifiées pour les semaines à venir
- **Compatible avec plusieurs appareils** — Fonctionne sur Amazon Fire Stick, Android TV, Google TV et Apple TV
