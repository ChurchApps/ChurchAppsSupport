# Présentation de FreePlay

<div class="article-intro">

FreePlay est une application TV pour les églises qui affiche le contenu des leçons sur les téléviseurs des salles de classe. Elle fonctionne sur Android TV, Amazon Fire TV et Apple TV, et est conçue pour fonctionner sans connexion internet le dimanche matin. Les bénévoles peuvent pré-télécharger les supports de cours pendant la semaine, puis simplement appuyer sur lecture lorsque le cours commence.

</div>

## Fonctionnalités principales

- **Lecture hors ligne** - Téléchargez les vidéos et les diapositives d'images à l'avance pour ne jamais dépendre du Wi-Fi pendant les services
- **Appairage à une salle de classe** - Connectez votre téléviseur à une salle spécifique et recevez automatiquement le contenu de la leçon programmée
- **Appairage à un plan** - Associez votre appareil à un plan de service d'église pour coordonner le contenu entre les salles
- **Fournisseurs de contenu** - Parcourez et téléchargez du contenu provenant de sources externes comme [Lessons.church](/docs/lessons-church/) et Bible Project
- **Compatible avec la télécommande TV** - Naviguez dans toute l'application avec une télécommande TV standard en utilisant les boutons directionnels et le bouton de sélection
- **Barre latérale rétractable** - Une barre latérale fine garde la navigation accessible sans encombrer l'écran

## Comment ça fonctionne

Lorsque vous lancez FreePlay, l'application se connecte à vos fournisseurs de contenu et vérifie le contenu disponible. De là, vous pouvez parcourir les dossiers, télécharger les médias et lire les leçons -- le tout optimisé pour une expérience sur grand écran de télévision.
