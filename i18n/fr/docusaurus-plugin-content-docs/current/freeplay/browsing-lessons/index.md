# Parcourir les leçons

<div class="article-intro">

FreePlay inclut un navigateur de leçons intégré qui vous permet d'explorer la bibliothèque complète des programmes, études et leçons disponibles. Cela est utile lorsque vous souhaitez choisir manuellement une leçon à lire plutôt que de vous fier à un programme de salle de classe ou à un plan.

</div>

## Le flux de navigation

Les leçons sont organisées de manière hiérarchique :

1. **Programmes** - Catégories de cursus de premier niveau (par exemple, une série pour le ministère des enfants)
2. **Études** - Groupes de leçons au sein d'un programme (par exemple, une unité sur plusieurs semaines)
3. **Leçons** - Sessions de cours individuelles avec leur propre contenu
4. **Lieux** - Différentes versions d'une leçon adaptées à des groupes d'âge ou des contextes spécifiques

Vous commencez par sélectionner un programme, puis vous accédez à une étude, choisissez une leçon et enfin sélectionnez un lieu pour commencer la lecture.

## Accéder au navigateur de leçons

Le navigateur de leçons est accessible via l'écran **Parcourir les programmes**. Accédez-y en utilisant la barre latérale ou en sélectionnant l'option de navigation correspondante.

:::info
Parcourir les leçons nécessite une connexion internet pour charger le catalogue de programmes. Une fois que vous avez sélectionné un lieu et commencé la lecture, le contenu est diffusé directement sans étape de téléchargement séparée.
:::
