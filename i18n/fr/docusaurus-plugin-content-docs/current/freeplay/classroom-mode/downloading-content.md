# Téléchargement de contenu

<div class="article-intro">

Téléchargez le contenu de la leçon pour une lecture hors ligne sur FreePlay, utile pour les salles sans connexion Internet fiable.

</div>

## Téléchargement des leçons

1. Ouvrez FreePlay
2. Allez à **Contenu**
3. Sélectionnez le fournisseur de contenu (Lessons.church, B1.church, etc.)
4. Trouvez la leçon que vous souhaitez télécharger
5. Appuyez sur le bouton **Télécharger**

Le contenu sera téléchargé sur votre appareil pour une lecture hors ligne.

## Gestion des fichiers téléchargés

1. Allez à **Paramètres** > **Contenu téléchargé**
2. Vous verrez une liste de tout le contenu téléchargé
3. Appuyez sur **Supprimer** pour libérer de l'espace

## Lecture du contenu téléchargé

1. Allez à **Contenu hors ligne**
2. Sélectionnez une leçon téléchargée
3. Appuyez sur **Lire**

Le contenu se lit normalement sans avoir besoin d'une connexion Internet.
