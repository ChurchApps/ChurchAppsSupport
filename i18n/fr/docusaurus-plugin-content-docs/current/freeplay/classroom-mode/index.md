# Mode Salle de classe

<div class="article-intro">

Le Mode Salle de classe connecte votre téléviseur à une salle spécifique de votre église. Une fois appairé, FreePlay télécharge automatiquement la leçon programmée pour cette salle, afin que vos bénévoles puissent simplement appuyer sur **Démarrer la leçon** lorsque le cours commence.

</div>

## Comment ça fonctionne

1. **Trouvez votre église** - Recherchez votre église par nom
2. **Sélectionnez une salle** - Choisissez la salle de classe à laquelle ce téléviseur appartient
3. **Téléchargez la leçon** - FreePlay récupère la leçon programmée et télécharge tous les fichiers multimédias
4. **Lecture** - Appuyez sur le bouton **Démarrer la leçon** et la leçon se lance automatiquement

Le téléviseur mémorise l'église et la salle auxquelles il est appairé, vous n'avez donc besoin de faire la configuration qu'une seule fois. Chaque semaine, FreePlay vérifie la leçon programmée pour le jour en cours et télécharge tout nouveau contenu.

:::tip
Assurez-vous que vos salles de classe sont configurées sur [freeplay.church](https://freeplay.church) avant d'appairer votre téléviseur. Si aucune salle de classe n'apparaît lorsque vous sélectionnez votre église, rendez-vous sur le site web pour les configurer.
:::
