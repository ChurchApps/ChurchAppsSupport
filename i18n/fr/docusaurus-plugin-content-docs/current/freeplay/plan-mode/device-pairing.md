# Appairer votre appareil

<div class="article-intro">

Pour utiliser le Mode Plan, vous devez appairer votre téléviseur avec un type de plan dans B1 Admin. FreePlay génère un code d'appairage unique qui associe l'appareil au plan de votre église, permettant la livraison automatique de contenu chaque semaine.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Installez et lancez FreePlay -- voir [Premiers pas](../getting-started/)
- Choisissez **Appairer à un plan** depuis l'[écran de mode d'appairage](../getting-started/pairing-modes.md)
- Ayez accès à **B1 Admin** sur un ordinateur ou un téléphone pour compléter l'appairage

</div>

## Générer le code d'appairage

1. Depuis l'écran **Sélectionner le mode d'appairage**, choisissez **Appairer à un plan**
2. FreePlay contacte le serveur et génère un court code d'appairage
3. Le code apparaît en gros caractères au centre de l'écran
4. Sous le code, un indicateur pulsant affiche **En attente de connexion**

Le code est affiché sous forme de caractères individuels pour une lecture facile à travers la salle.

## Entrer le code dans B1 Admin

1. Sur un ordinateur ou un téléphone, ouvrez **B1 Admin** pour votre église
2. Accédez aux paramètres du **Type de plan** pour le plan que vous souhaitez assigner
3. Entrez le code d'appairage affiché sur le téléviseur
4. Enregistrez les paramètres

FreePlay interroge le serveur toutes les quelques secondes pour vérifier si l'appairage est terminé. Une fois que **B1 Admin** confirme la connexion, le téléviseur passe automatiquement à l'écran de téléchargement du plan.

## Télécharger le contenu du plan

Après l'appairage, FreePlay charge le plan en cours pour ce type de plan. Il affiche :

- Le nom du plan et la date du service
- Le nom de la leçon associée (si le plan inclut du contenu de leçon)
- Un indicateur de progression affichant **Téléchargement de l'élément X sur Y**

Lorsque tous les fichiers multimédias sont téléchargés, le bouton **Démarrer le plan** apparaît. Appuyez sur **Sélectionner** sur votre télécommande pour commencer la lecture.

:::tip
Le plan s'actualise automatiquement toutes les heures. Si le plan est mis à jour pendant la journée, FreePlay récupère les modifications sans aucune intervention manuelle.
:::

## Alternative : rechercher par nom d'église

Si vous préférez ne pas utiliser le flux de code d'appairage, vous pouvez sélectionner **ou rechercher par nom d'église** en bas de l'écran d'appairage. Cela vous amène à l'écran de recherche d'église où vous pouvez trouver votre église et vous connecter à une salle de classe à la place.

## Si l'appairage échoue

Si le code d'appairage ne peut pas être généré (par exemple, en raison d'un problème réseau), vous verrez un message d'erreur avec un bouton **Réessayer**. Assurez-vous que votre téléviseur est connecté à internet et réessayez.

:::warning
Les codes d'appairage expirent après un certain temps. Si vous attendez trop longtemps, générez un nouveau code en revenant à l'écran d'appairage.
:::
