# Créer un compte

<div class="article-intro">

Créez votre compte ChurchApps pour accéder à tous les services de votre église incluant B1 Admin, B1 Mobile, B1.church et d'autres outils.

</div>

## Étapes de création de compte

1. Visitez [churchapps.org](https://churchapps.org) ou accédez directement à votre URL B1.church de l'église
2. Cliquez sur **Créer un compte**
3. Entrez votre **adresse e-mail** et créez un **mot de passe**
4. Remplissez votre **prénom**, **nom de famille** et acceptez les conditions d'utilisation
5. Cliquez sur **Créer un compte**
6. Consultez l'e-mail de confirmation et cliquez sur le lien pour vérifier votre adresse e-mail
7. Vous pouvez maintenant vous connecter et accéder à votre église

:::tip
Utilisez l'adresse e-mail et le mot de passe du même compte pour tous les produits ChurchApps - B1 Admin, B1 Mobile, B1.church, etc.
:::
