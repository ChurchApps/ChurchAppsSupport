# Premiers pas

<div class="article-intro">

Bienvenue sur ChurchApps -- des outils gratuits et open source conçus pour aider votre église à prospérer. Que vous soyez une petite congrégation ou une grande organisation multi-sites, ChurchApps fournit tout ce dont vous avez besoin pour gérer et développer votre communauté ecclésiale.

</div>

## Produits ChurchApps

ChurchApps comprend plusieurs produits qui fonctionnent ensemble :

- **B1 Admin** -- Votre tableau de bord de gestion d'église. Gérez les personnes, les groupes, la fréquentation, les dons, le contenu du site web et bien plus encore depuis un emplacement centralisé.
- **B1.church** -- Votre site web public d'église et portail des membres. Les membres peuvent visionner les sermons, accéder aux répertoires, donner en ligne et rester connectés.
- **Lessons.church** -- Programme d'études gratuit pour l'école du dimanche, les petits groupes et les études bibliques. Accédez à une bibliothèque de leçons en constante croissance, sans aucun coût.
- **FreeShow** -- Logiciel de présentation gratuit pour vos cultes. Affichez les paroles, les diapositives et les médias pendant vos rassemblements.

:::info
Tous les outils ChurchApps sont **100 % gratuits** sans limites, sans niveaux premium et sans frais cachés. ChurchApps est une organisation à but non lucratif soutenue par des dons et des bénévoles.
:::

## Ce que couvre cette documentation

Cette documentation vous guidera à travers tout ce dont vous avez besoin pour être opérationnel avec ChurchApps. Vous apprendrez à :

1. Créer votre compte et enregistrer votre église.
2. Configurer les paramètres de base et l'image de marque de votre église.
3. Mettre en place votre équipe avec les rôles et permissions appropriés.
4. Commencer à utiliser les outils pour gérer votre congrégation.
