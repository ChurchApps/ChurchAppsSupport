# Configuration initiale

<div class="article-intro">

Après création de votre compte ChurchApps, effectuez la configuration initiale pour configurer votre profil, vos paramètres de notification et vos préférences.

</div>

## Configuration de votre profil

1. Connectez-vous à votre compte
2. Accédez à **Paramètres** ou **Mon profil**
3. Remplissez vos informations personnelles :
   - Prénom et nom
   - Numéro de téléphone (facultatif)
   - Photo de profil (facultatif)
4. Sélectionnez votre église (si vous appartenez à plusieurs églises)
5. Enregistrez vos modifications

## Paramètres de notification

1. Dans **Paramètres**, allez à **Notifications**
2. Choisissez comment recevoir les notifications :
   - **Notifications push** — Mises à jour en temps réel sur votre téléphone
   - **Notifications e-mail** — Mises à jour envoyées à votre e-mail
   - **Pas de notifications** — Désactiver les notifications
3. Configurez vos préférences pour :
   - Affectations de service
   - Mises à jour de groupe
   - Annonces d'événements
   - Messages directs
4. Enregistrez vos modifications

## Exploration de votre église

1. Accédez à **Groupes** pour voir les groupes disponibles
2. Consultez les **Événements** à venir
3. Affichez les **Plans de service** pour voir vos affectations
4. Consultez le **Répertoire** pour voir les autres membres
