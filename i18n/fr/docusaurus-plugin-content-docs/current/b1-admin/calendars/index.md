# Calendriers

<div class="article-intro">

La section Calendriers dans B1 Admin vous permet de créer des calendriers organisés pour votre église en regroupant les événements de vos groupes. Les responsables de groupe gèrent les événements de leur propre groupe, et en tant qu'administrateur, vous pouvez créer des calendriers qui affichent ces événements sur votre site web B1.church pour que votre congrégation les voie.

</div>

## Consultation de vos calendriers

1. Dans B1 Admin, naviguez vers **Site Web**, puis vers la section **Calendriers**.
2. La page Calendriers liste tous vos calendriers avec leur nom et leur statut.
3. Chaque calendrier affiche également les groupes qui lui sont associés.

## Ce que vous pouvez faire

- **Ajouter un nouveau calendrier** -- Cliquez sur le bouton d'ajout pour créer un calendrier pour un ministère, un type d'événement ou un objectif spécifique. Consultez [Création de calendriers](creating-calendars) pour des instructions étape par étape.
- **Modifier un calendrier** -- Cliquez sur n'importe quel calendrier pour l'ouvrir et gérer les groupes qui y sont inclus.
- **Publier des calendriers** -- Les calendriers peuvent être affichés sur votre site web B1.church afin que les visiteurs puissent voir les événements à venir de votre église.
- **Créer des calendriers organisés** -- Extrayez des événements de plusieurs groupes dans une vue calendrier unique pour une vue d'ensemble complète de tout ce qui se passe dans votre église. Consultez [Calendrier organisé](curated-calendar) pour plus de détails.

:::info
Les calendriers fonctionnent étroitement avec les [Groupes](../groups/creating-groups.md). Lorsque les groupes ont des événements planifiés, ces événements peuvent être intégrés dans vos calendriers. Consultez la page [Création de calendriers](creating-calendars) pour apprendre à connecter des groupes aux calendriers.
:::
