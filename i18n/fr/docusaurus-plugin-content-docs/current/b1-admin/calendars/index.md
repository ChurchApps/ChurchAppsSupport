# Calendriers

<div class="article-intro">

La section Calendriers dans B1 Admin vous permet de créer et de gérer des calendriers d'événements pour votre église. Vous pouvez configurer des calendriers pour différents ministères, regrouper des événements de plusieurs groupes et publier des calendriers sur votre site B1.church pour que votre assemblée puisse les consulter.

</div>

## Consulter vos calendriers

1. Dans B1 Admin, accédez à la section **Calendriers**.
2. La page Calendriers liste tous vos calendriers avec leur nom et leur statut.
3. Chaque calendrier affiche également les groupes qui lui sont associés.

## Ce que vous pouvez faire

- **Ajouter un nouveau calendrier** -- Cliquez sur le bouton d'ajout pour créer un calendrier pour un ministère, un type d'événement ou un objectif spécifique. Consultez [Créer des calendriers](creating-calendars) pour des instructions étape par étape.
- **Modifier un calendrier** -- Cliquez sur n'importe quel calendrier pour l'ouvrir et gérer ses événements et ses groupes associés.
- **Publier des calendriers** -- Les calendriers peuvent être affichés sur votre site B1.church afin que les visiteurs puissent voir les événements à venir de votre église.
- **Créer des calendriers organisés** -- Rassemblez les événements de plusieurs groupes dans une vue calendrier unique pour une vue d'ensemble complète de tout ce qui se passe dans votre église. Consultez [Calendrier organisé](curated-calendar) pour plus de détails.

:::info
Les calendriers fonctionnent en étroite collaboration avec les [Groupes](../groups/creating-groups.md). Lorsque des groupes ont des événements planifiés, ces événements peuvent être intégrés à vos calendriers. Consultez la page [Créer des calendriers](creating-calendars) pour apprendre comment connecter des groupes aux calendriers.
:::
