# Salles, ressources et planification

<div class="article-intro">

Les salles et ressources vous permettent de suivre les espaces et équipements réservables de votre église. Quand un groupe ou une équipe veut utiliser une salle ou emprunter une ressource pour un événement, ils peuvent la demander via B1 -- et vous pouvez approuver ou refuser ces demandes pour éviter les réservations doubles.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous devez avoir un accès administratif à la section Calendriers dans B1 Admin
- Configurez les [groupes](../groups/creating-groups) dont les membres soumettront des demandes de réservation

</div>

## Ouverture de Salles et ressources

Dans B1 Admin, allez à **Calendriers** et sélectionnez **Salles et ressources**. La page a quatre onglets : **Salles**, **Ressources**, **Blocages** et **Modèles**.

## Salles

Les salles sont des espaces physiques qui peuvent être réservés aux côtés des événements du calendrier -- un sanctuaire, une salle de communion, une classe, etc.

### Ajouter une salle

1. Sur l'onglet **Salles**, cliquez sur **Ajouter une salle**.
2. Entrez un **nom** pour la salle.
3. Définissez éventuellement une **capacité** (nombre maximal de personnes).
4. Définissez un **Groupe d'approbation** -- quand un groupe demande cette salle, les responsables du groupe choisi sont notifiés et doivent approuver la réservation. Laissez vide si aucune approbation n'est nécessaire.
5. Cliquez sur **Enregistrer**.

### Modifier ou supprimer une salle

Cliquez sur l'icône de modification à côté de n'importe quelle salle, effectuez vos modifications et cliquez sur **Enregistrer**. Utilisez **Supprimer** pour supprimer définitivement une salle.

## Ressources

Les ressources sont des éléments qui peuvent être réservés aux côtés d'un événement -- équipement AV, projecteurs, chaises, un van, etc.

### Ajouter une ressource

1. Cliquez sur l'onglet **Ressources**, puis sur **Ajouter une ressource**.
2. Entrez un **nom** et éventuellement une **description**.
3. Définissez un **Groupe d'approbation** si quelqu'un doit approuver les prêts de ressources.
4. Cliquez sur **Enregistrer**.

## Blocages

Les blocages empêchent les salles ou les ressources d'être réservées pendant certains moments -- pour des rénovations, des périodes réservées ou de la maintenance.

### Ajouter un blocage

1. Cliquez sur l'onglet **Blocages**, puis sur **Ajouter un blocage**.
2. Choisissez de bloquer une **Salle**, une **Ressource** ou toutes les salles et ressources.
3. Définissez la date/heure de **début** et de **fin** du blocage.
4. Ajoutez éventuellement une note de **raison**.
5. Cliquez sur **Enregistrer**.

## Modèles d'événements

Les modèles vous permettent de sauvegarder une configuration d'événement réutilisable -- titre, description et groupes associés -- pour que les événements récurrents puissent être créés de manière cohérente.

### Ajouter un modèle

1. Cliquez sur l'onglet **Modèles**, puis sur **Ajouter un modèle**.
2. Entrez un **nom** pour le modèle.
3. Liez éventuellement un **groupe** pour que l'événement soit automatiquement connecté à ce groupe lors de sa création à partir du modèle.
4. Cliquez sur **Enregistrer**.

Lors de la création d'un nouvel événement, le personnel peut choisir un modèle existant pour pré-remplir les détails de l'événement.
