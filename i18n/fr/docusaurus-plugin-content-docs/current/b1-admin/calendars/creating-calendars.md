# Création de calendriers

<div class="article-intro">

La création d'un calendrier dans B1 Admin vous donne un lieu central pour gérer les événements d'un ministère, d'un département ou de toute votre église. Chaque calendrier peut extraire les événements d'un ou plusieurs groupes, ce qui facilite la tenue de tout en ordre.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Configurez les [groupes](../groups/creating-groups.md) dont vous voulez inclure les événements dans votre calendrier
- Vous avez besoin d'un accès administratif à la section Calendriers dans B1 Admin

</div>

## Création d'un nouveau calendrier

1. Allez à la section **Calendriers** dans B1 Admin.
2. Cliquez sur **Ajouter un calendrier**.
3. Entrez un **nom** pour votre calendrier (par exemple, "Événements du ministère jeunesse" ou "Calendrier de l'église").
4. Ajoutez une **description** optionnelle pour aider votre équipe à comprendre l'objectif de ce calendrier.
5. Cliquez sur **Créer** pour enregistrer votre nouveau calendrier.

## La page de détail du calendrier

Après la création d'un calendrier, cliquez dessus pour ouvrir la page de détail. Cette page a deux zones principales :

- **Colonne de gauche** -- L'éditeur de calendrier et d'événements. C'est ici que vous pouvez afficher, ajouter et gérer les événements sur votre calendrier.
- **Colonne de droite** -- La liste des groupes associés. Ceci montre quels groupes ont des événements inclus dans ce calendrier.

## Ajout d'événements

1. Sur la page de détail du calendrier, utilisez l'éditeur de calendrier à gauche pour ajouter de nouveaux événements.
2. Définissez le nom de l'événement, la date, l'heure et tous les autres détails.
3. Les événements que vous ajoutez apparaîtront sur le calendrier et pourront être publiés sur votre site.

## Connexion des groupes

Les groupes qui ont des événements dans le calendrier apparaissent automatiquement dans la liste des groupes sur le côté droit de la page de détail.

1. Cliquez sur **Ajouter** dans la section des groupes pour associer un groupe à votre calendrier.
2. Sélectionnez le groupe dans la liste déroulante.
3. Choisissez si vous voulez inclure **tous les événements** de ce groupe ou seulement des **événements spécifiques**.
4. Cliquez sur **Enregistrer**.

:::tip
La connexion de groupes à votre calendrier est un moyen puissant d'agréger automatiquement les événements. Lorsqu'un chef de groupe ajoute un événement à son [groupe](../groups/creating-groups.md), il peut s'intégrer à votre calendrier de l'église sans travail supplémentaire de votre part.
:::

:::info
Si vous voulez créer un seul calendrier qui extrait les événements de nombreux groupes dans toute votre église, voir [Calendrier organisé](curated-calendar) pour une approche rationalisée.
:::

## Activation de l'enregistrement aux événements

Vous pouvez activer l'enregistrement pour n'importe quel événement du calendrier afin que les membres puissent s'inscrire par le site B1 ou l'application mobile.

1. Cliquez sur un événement existant ou créez un nouveau.
2. Dans l'éditeur d'événements, basculez **Enregistrement** pour l'activer.
3. Configurez les paramètres d'enregistrement :
   - **Capacité** (optionnel) -- Définissez un nombre maximum d'enregistrements. Laissez vide pour illimité.
   - **L'enregistrement s'ouvre** -- La date et l'heure lorsque l'enregistrement devient disponible.
   - **L'enregistrement se ferme** -- La date et l'heure lorsque l'enregistrement se ferme.
   - **Balises** -- Étiquettes séparées par des virgules (par exemple, "jeunesse, retraite, école du dimanche") pour aider à catégoriser les événements enregistrables.
4. Enregistrez l'événement.

Une fois que l'enregistrement est activé, les membres verront un bouton **S'inscrire à cet événement** lorsqu'ils verront l'événement sur le [site web B1](../../b1-church/events/registering) ou l'[application mobile B1](../../b1-mobile/events/registering).

### Gestion des enregistrements

Pour afficher et gérer les enregistrements pour vos événements :

1. Allez à la page **Enregistrements** dans B1 Admin.
2. Vous verrez un tableau de tous les événements avec l'enregistrement activé, affichant le titre de l'événement, la date, le nombre d'enregistrements actuel par rapport à la capacité et les balises.
3. Cliquez sur un événement pour voir la liste complète des enregistrements, incluant les noms, le nombre de membres, le statut et la date d'enregistrement.
4. Depuis la page de détail, vous pouvez :
   - **Annuler** les enregistrements individuels
   - **Supprimer** les enregistrements définitivement
   - **Exporter** tous les enregistrements en CSV

:::tip
Utilisez la barre de progression de la capacité pour surveiller la rapidité avec laquelle les événements se remplissent. La barre devient rouge lorsqu'un événement est à ou dépasse la capacité.
:::
