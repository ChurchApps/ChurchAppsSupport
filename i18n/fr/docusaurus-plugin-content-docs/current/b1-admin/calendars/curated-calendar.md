# Calendrier organisé

<div class="article-intro">

Un calendrier organisé regroupe les événements de plusieurs groupes dans une vue calendrier unifiée. C'est idéal pour créer un calendrier d'église global qui affiche tout ce qui se passe dans l'ensemble de vos ministères et groupes en un seul endroit.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Configurez les [groupes](../groups/creating-groups.md) dont vous souhaitez inclure les événements
- Assurez-vous que les responsables de groupe ont ajouté des événements à leurs groupes respectifs
- Familiarisez-vous avec les bases de la [Création de calendriers](creating-calendars)

</div>

## Créer un calendrier organisé

1. Depuis B1 Admin, accédez à la section **Site web**.
2. Cliquez sur **Calendriers** dans la navigation supérieure.
3. Cliquez sur **Ajouter un calendrier** pour créer un nouveau calendrier organisé.
4. Donnez un **nom** à votre calendrier (par exemple, « Événements de l'église » ou « Cette semaine dans notre église »).
5. Cliquez sur **Créer** pour l'enregistrer.

## Ajouter des groupes à votre calendrier

1. Cliquez sur votre nouveau calendrier pour l'ouvrir.
2. Dans la section **Groupes du calendrier** sur le côté droit, cliquez sur **Ajouter**.
3. Sélectionnez un groupe dans le menu déroulant.
4. Choisissez d'ajouter **tous les événements** de ce groupe ou uniquement des **événements spécifiques**.
5. Cliquez sur **Enregistrer**.
6. Répétez ce processus pour ajouter autant de groupes que vous le souhaitez.

:::tip
Ajoutez tous vos groupes de ministère actifs pour créer un calendrier complet qui offre à votre assemblée une vue d'ensemble de tout ce qui se passe dans votre église chaque semaine.
:::

## Partager votre calendrier

Une fois votre calendrier organisé configuré, vous pouvez le partager de plusieurs façons :

- **Intégrer sur votre site web** -- Le calendrier peut être affiché directement sur les pages de votre site B1.church pour que les visiteurs puissent le consulter.
- **S'abonner via ICS** -- Cliquez sur **S'abonner** pour copier le lien ICS. Votre assemblée peut coller ce lien dans Google Calendar, Apple Calendar, Outlook ou toute autre application de calendrier pour rester synchronisée.
- **Télécharger le fichier ICS** -- Téléchargez le fichier calendrier pour l'importer dans une application de calendrier.

:::info
Lorsque les responsables de groupe ajoutent de nouveaux événements à leurs [groupes](../groups/creating-groups.md), ces événements apparaissent automatiquement dans votre calendrier organisé. Vous n'avez pas besoin de mettre à jour manuellement le calendrier organisé chaque fois qu'un nouvel événement est ajouté.
:::
