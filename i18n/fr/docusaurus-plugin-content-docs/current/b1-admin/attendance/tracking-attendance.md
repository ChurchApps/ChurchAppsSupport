# Suivi de la présence

<div class="article-intro">

Une fois vos campus, horaires de culte et groupes configurés, B1 Admin facilite la consultation des données de présence et l'identification des tendances. La page Présence propose deux vues de rapport -- l'onglet **Présence** pour les tendances à l'échelle de l'église et l'onglet **Groupes** pour les détails par groupe. Utilisez ces outils pour comprendre les schémas de croissance, identifier les baisses d'engagement et prendre des décisions basées sur les données pour votre église.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Votre structure de présence doit être configurée avec au moins un campus et un horaire de culte. Voir [Configuration de la présence](setup.md) si vous ne l'avez pas encore fait.
- Des données de présence doivent être enregistrées avant que les rapports n'affichent des résultats. Les données peuvent provenir de la [saisie manuelle](recording-attendance.md) ou de [l'enregistrement en libre-service](check-in.md).

</div>

## Consulter les tendances de présence

1. Ouvrez **B1 Admin** et cliquez sur **Présence** dans la barre latérale.
2. Sélectionnez l'onglet **Présence**.
3. Le rapport s'exécute automatiquement à l'ouverture de l'onglet, affichant la présence sur une plage de dates par défaut.

## Filtrer vos données

Utilisez les filtres en haut de la page pour affiner les résultats :

- **Plage de dates** -- choisissez une date de début et de fin pour vous concentrer sur une période spécifique.
- **Campus** -- sélectionnez un campus pour voir la présence uniquement pour cet emplacement.
- **Horaire de culte** -- choisissez un horaire de culte pour examiner un rassemblement particulier.

Le graphique et les données se mettent à jour dès que vous modifiez un filtre, ce qui vous permet de comparer rapidement différentes périodes ou emplacements.

:::info
Les rapports s'exécutent automatiquement chaque fois que vous ouvrez l'onglet Présence, vous verrez donc toujours des chiffres à jour sans avoir besoin de cliquer sur un bouton d'actualisation.
:::

## Présence par groupe

L'onglet **Groupes** affiche la présence détaillée par groupe individuel. C'est utile lorsque vous souhaitez suivre une classe, une équipe ministérielle ou un petit groupe spécifique plutôt que de regarder les chiffres globaux des cultes.

1. Sélectionnez l'onglet **Groupes**.
2. Choisissez un groupe dans la liste pour voir son historique de présence.
3. Utilisez le filtre de plage de dates pour ajuster la fenêtre de rapport.

:::tip
La présence par groupe est particulièrement précieuse pour les responsables de [petits groupes](../groups/creating-groups.md) qui souhaitent suivre l'engagement au sein de leur groupe au fil du temps.
:::

## Conseils pour utiliser les données de présence

- Examinez les tendances mensuellement pour détecter les variations saisonnières tôt.
- Comparez les données par campus pour comprendre quels emplacements sont en croissance.
- Utilisez les rapports par groupe pour suivre les [groupes](../groups/group-members.md) dont la présence est en baisse.
- Combinez les analyses de présence avec l'outil [Recherche IA](../people/ai-search.md) pour trouver les personnes qui n'ont pas assisté récemment.
