# Suivi des présences

<div class="article-intro">

Une fois vos lieux de culte, horaires de culte et groupes configurés, B1 Admin facilite la consultation des données de présences et l'identification des tendances. La page Présences propose deux vues de rapports -- l'onglet **Présences** pour les tendances à l'échelle de l'église et l'onglet **Groupes** pour les détails au niveau des groupes. Utilisez ces outils pour comprendre les schémas de croissance, identifier les baisses d'engagement et prendre des décisions basées sur les données pour votre église.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Votre structure de présences doit être configurée avec au moins un lieu de culte et un horaire de culte. Consultez [Configuration des présences](setup.md) si vous ne l'avez pas encore fait.
- Des données de présences doivent être enregistrées avant que les rapports n'affichent des résultats. Les données peuvent provenir de la saisie manuelle ou de l'[enregistrement en libre-service](check-in.md).

</div>

## Consulter les tendances de présences

1. Ouvrez **B1 Admin** et cliquez sur **Présences** dans la barre latérale.
2. Sélectionnez l'onglet **Présences**.
3. Le rapport s'exécute automatiquement à l'ouverture de l'onglet, affichant les présences sur une plage de dates par défaut.

## Filtrer vos données

Utilisez les filtres en haut de la page pour affiner les résultats :

- **Plage de dates** -- choisissez une date de début et de fin pour vous concentrer sur une période spécifique.
- **Lieu de culte** -- sélectionnez un lieu de culte pour voir les présences uniquement pour cet emplacement.
- **Horaire de culte** -- choisissez un horaire de culte pour examiner un rassemblement en particulier.

Le graphique et les données se mettent à jour dès que vous modifiez un filtre, ce qui vous permet de comparer rapidement différentes périodes ou emplacements.

:::info
Les rapports s'exécutent automatiquement chaque fois que vous ouvrez l'onglet Présences, de sorte que vous verrez toujours des chiffres à jour sans avoir besoin de cliquer sur un bouton d'actualisation.
:::

## Présences par groupe

L'onglet **Groupes** affiche les présences ventilées par groupe individuel. Cela est utile lorsque vous souhaitez suivre une classe spécifique, une équipe de ministère ou un petit groupe plutôt que de regarder les chiffres globaux des cultes.

1. Sélectionnez l'onglet **Groupes**.
2. Choisissez un groupe dans la liste pour voir son historique de présences.
3. Utilisez le filtre de plage de dates pour ajuster la fenêtre de rapport.

:::tip
Les présences par groupe sont particulièrement utiles pour les responsables de [petits groupes](../groups/creating-groups.md) qui souhaitent suivre l'engagement au sein de leur groupe au fil du temps.
:::

## Conseils pour utiliser les données de présences

- Examinez les tendances mensuellement pour détecter les schémas saisonniers en amont.
- Comparez les données par lieu de culte pour comprendre quels emplacements sont en croissance.
- Utilisez les rapports au niveau des groupes pour assurer le suivi des [groupes](../groups/group-members.md) dont les présences diminuent.
- Combinez les informations sur les présences avec l'outil de [Recherche IA](../people/ai-search.md) pour trouver les personnes qui n'ont pas assisté récemment.
