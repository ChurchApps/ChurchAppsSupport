# Enregistrement de la présence

<div class="article-intro">

Une fois que vos campus, heures de service et groupes sont configurés, vous pouvez enregistrer manuellement la présence après chaque réunion. B1 Admin organise la présence autour de **sessions** -- une session par groupe par date de réunion. Vous créez la session, marquez qui a assisté, et les données alimentent directement vos rapports de présence.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vos campus, heures de service et groupes doivent être configurés. Voir [Configuration de la présence](setup.md) si vous n'avez pas encore fait cela.
- Les groupes que vous voulez suivre doivent avoir **Suivre la présence** activé. Voir [Configuration de la présence](setup.md) pour les détails.

</div>

## Création d'une session

Une session représente une occurrence d'une réunion de groupe -- par exemple, votre classe de K-3ème année un dimanche spécifique.

1. Ouvrez **B1 Admin** et cliquez sur **Groupes** dans la barre latérale.
2. Sélectionnez le groupe pour lequel vous voulez enregistrer la présence.
3. Cliquez sur l'onglet **Sessions**.
4. Cliquez sur **Nouveau** pour créer une nouvelle session.
5. Si le groupe est assigné à une heure de service, choisissez l'**Heure de service**. Si c'est un groupe non programmé, ce champ n'apparaîtra pas.
6. Sélectionnez la **Date de session** -- cela peut être aujourd'hui, une date passée ou une date future.
7. Cliquez sur **Enregistrer**.

:::tip
Vous pouvez créer des sessions pour les dates passées pour récupérer la présence que vous n'avez pas encore enregistrée, ou les créer à l'avance pour qu'elles soient prêtes lorsque votre groupe se réunit.
:::

## Marquage de la présence

Après avoir sauvegardé la session, les membres du groupe apparaissent sur le côté droit de la page.

1. Cochez la case à côté de chaque personne qui a assisté.
2. Les modifications sont sauvegardées automatiquement -- il n'y a pas de bouton Enregistrer supplémentaire pour les marques de présence.

:::info
Seuls les membres actuels du groupe apparaissent dans la liste des sessions. Si quelqu'un a assisté mais n'est pas encore dans le groupe, [ajoutez-le au groupe](../groups/group-members.md) en premier, puis enregistrez sa présence.
:::

## Export de la présence vers une feuille de calcul

Vous pouvez télécharger un enregistrement de la session en tant que fichier CSV à utiliser dans Excel, Numbers ou Google Sheets.

1. Ouvrez la session que vous voulez exporter.
2. Cliquez sur le bouton **Télécharger CSV**.
3. Ouvrez le fichier téléchargé dans votre application de feuille de calcul.

## Affichage de la présence enregistrée

Après l'enregistrement des sessions, les données apparaissent dans vos rapports de présence.

- **Onglet Présence** -- affiche les tendances à l'échelle de l'église au fil du temps. Voir [Suivi de la présence](tracking-attendance.md).
- **Onglet Groupes** -- affiche la présence décomposée par groupe individuel.

:::tip
Si une session que vous venez de créer n'apparaît pas immédiatement dans les rapports, assurez-vous que la date de la session se situe dans la plage de dates sélectionnée dans les filtres des rapports.
:::
