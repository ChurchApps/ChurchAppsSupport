# Enregistrer la présence

<div class="article-intro">

Une fois vos campus, horaires de culte et groupes configurés, vous pouvez enregistrer manuellement la présence après chaque rassemblement. B1 Admin organise la présence autour de **sessions** -- une session par groupe et par date de réunion. Vous créez la session, indiquez qui était présent, et les données alimentent directement vos rapports de présence.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vos campus, horaires de culte et groupes doivent être configurés. Voir [Configuration de la présence](setup.md) si vous ne l'avez pas encore fait.
- Les groupes que vous souhaitez suivre doivent avoir le **Suivi de présence** activé. Voir [Configuration de la présence](setup.md) pour plus de détails.

</div>

## Créer une session

Une session représente une occurrence d'une réunion de groupe -- par exemple, votre classe maternelle-CE2 un dimanche spécifique.

1. Ouvrez **B1 Admin** et cliquez sur **Groupes** dans la barre latérale.
2. Sélectionnez le groupe pour lequel vous souhaitez enregistrer la présence.
3. Cliquez sur l'onglet **Sessions**.
4. Cliquez sur **Nouveau** pour créer une nouvelle session.
5. Si le groupe est assigné à un horaire de culte, choisissez l'**Horaire de culte**. S'il s'agit d'un groupe non planifié, ce champ n'apparaîtra pas.
6. Sélectionnez la **Date de session** -- cela peut être aujourd'hui, une date passée ou une date future.
7. Cliquez sur **Enregistrer**.

:::tip
Vous pouvez créer des sessions pour des dates passées pour rattraper la présence que vous n'avez pas encore enregistrée, ou les créer à l'avance pour qu'elles soient prêtes lorsque votre groupe se réunit.
:::

## Marquer la présence

Après avoir enregistré la session, les membres du groupe apparaissent sur le côté droit de la page.

1. Cochez la case à côté de chaque personne qui était présente.
2. Les modifications sont enregistrées automatiquement -- il n'y a pas de bouton Enregistrer supplémentaire pour les marquages de présence.

:::info
Seuls les membres actuels du groupe apparaissent dans la liste de session. Si quelqu'un a assisté mais n'est pas encore dans le groupe, [ajoutez-le au groupe](../groups/group-members.md) d'abord, puis enregistrez sa présence.
:::

## Exporter la présence vers un tableur

Vous pouvez télécharger un enregistrement de la session sous forme de fichier CSV à utiliser dans Excel, Numbers ou Google Sheets.

1. Ouvrez la session que vous souhaitez exporter.
2. Cliquez sur le bouton **Télécharger CSV**.
3. Ouvrez le fichier téléchargé dans votre application de tableur.

## Consulter la présence enregistrée

Après avoir enregistré des sessions, les données apparaissent dans vos rapports de présence.

- **Onglet Présence** -- affiche les tendances à l'échelle de l'église au fil du temps. Voir [Suivi de la présence](tracking-attendance.md).
- **Onglet Groupes** -- affiche la présence détaillée par groupe individuel.

:::tip
Si une session que vous venez de créer n'apparaît pas immédiatement dans les rapports, assurez-vous que la date de session se situe dans la plage de dates sélectionnée dans les filtres du rapport.
:::
