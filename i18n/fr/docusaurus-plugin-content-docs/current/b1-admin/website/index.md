# Site web

<div class="article-intro">

La section Site web dans B1 Admin vous offre un constructeur de site web complet pour votre église. Vous pouvez créer des pages, personnaliser l'apparence de votre site, gérer les fichiers média et publier des calendriers d'événements -- le tout sans écrire de code.

</div>

## Ce que vous pouvez faire

Le constructeur de site web est organisé en cinq zones principales :

### Pages

Créez et gérez toutes les pages de votre site web d'église. Le tableau **Pages** affiche chaque page avec son statut -- soit **Généré** (créé automatiquement par le système) soit **Personnalisé** (créé par vous). Vous pouvez ajouter de nouvelles pages personnalisées, modifier les existantes ou convertir des pages générées automatiquement en pages personnalisées pour un contrôle total sur leur contenu.

La barre latérale gauche affiche vos liens de navigation sous forme d'arborescence. Vous pouvez réorganiser les liens par glisser-déposer pour définir exactement la hiérarchie et l'ordre que vous souhaitez que les visiteurs voient.

### Blocs

Construisez des composants de contenu réutilisables que vous pouvez placer sur plusieurs pages. Les blocs vous font gagner du temps lorsque vous avez du contenu comme un appel à l'action, une bannière d'annonce ou une section de contact qui apparaît à plusieurs endroits. Mettez à jour un bloc une seule fois et la modification s'applique partout où il est utilisé.

### Apparence

Personnalisez le thème, les couleurs, les polices, le logo et l'espacement de votre site web. Vous pouvez appliquer des thèmes de style prédéfinis pour un démarrage rapide ou affiner chaque détail avec l'éditeur de palette de couleurs, les paramètres de typographie et le CSS personnalisé. Consultez le guide [Apparence](appearance) pour tous les détails.

### Fichiers

Téléversez et organisez toutes vos ressources média -- images, documents et autres fichiers -- dans un emplacement central. Utilisez des dossiers pour garder les choses ordonnées et copiez les URL des fichiers pour les utiliser n'importe où sur votre site. Consultez [Fichiers](files) pour plus d'informations.

### Calendriers

Publiez des calendriers d'événements sur votre site web pour que les visiteurs puissent voir ce qui se passe dans votre église. Créez des calendriers organisés qui rassemblent les événements de plusieurs groupes dans une seule vue.

:::tip
Commencez par la [Configuration initiale](initial-setup) si vous configurez votre site web pour la première fois. Elle vous guide à travers la configuration du domaine, la création de votre première page et la configuration de l'apparence de votre site.
:::
