# Fichiers

<div class="article-intro">

La page Fichiers est le dépôt central de tous les médias et documents utilisés sur votre site web d'église. Depuis cette page, vous pouvez téléverser, organiser, parcourir et gérer chaque image, document et fichier média dont votre site a besoin.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Complétez la [Configuration initiale](initial-setup) de votre site web
- Optimisez les images avant le téléversement pour des temps de chargement plus rapides
- Préparez vos fichiers dans des formats web courants (PNG, JPG, PDF, etc.)

</div>

## Téléverser des fichiers

1. Accédez à la section **Site web** dans B1 Admin et cliquez sur l'onglet **Fichiers**.
2. Glissez-déposez les fichiers directement sur la zone de téléversement, ou cliquez sur le bouton de téléversement pour parcourir votre ordinateur.
3. Vos fichiers seront téléversés et apparaîtront immédiatement dans la bibliothèque de fichiers.

:::tip
Pour de meilleurs résultats, optimisez les images avant de les téléverser. Des tailles de fichiers plus petites aident votre site web à se charger plus rapidement pour les visiteurs. Des outils comme TinyPNG ou Squoosh peuvent compresser les images sans perte de qualité perceptible.
:::

## Organiser avec des dossiers

1. Créez des dossiers pour regrouper les fichiers liés -- par exemple, « Visuels de prédications », « Bannières d'événements » ou « Photos du personnel ».
2. Déplacez les fichiers entre les dossiers pour garder votre bibliothèque ordonnée au fur et à mesure qu'elle grandit.
3. Utilisez des noms de dossiers clairs et descriptifs pour que votre équipe puisse trouver les fichiers rapidement.

## Parcourir et rechercher

- Faites défiler la bibliothèque de fichiers pour voir les miniatures de tous vos fichiers téléversés.
- Utilisez la fonction de **recherche** pour trouver des fichiers par nom lorsque votre bibliothèque devient volumineuse.
- Cliquez sur n'importe quel fichier pour voir ses détails, y compris le nom du fichier, la taille et l'URL.

## Utiliser les fichiers sur votre site web

1. Cliquez sur un fichier pour voir ses détails.
2. **Copiez l'URL du fichier** pour l'utiliser dans les pages de votre site web. Vous pouvez coller cette URL dans des éléments d'image, des liens de téléchargement ou partout où une URL de fichier est acceptée.
3. Les fichiers que vous téléversez ici sont disponibles pour utilisation sur toutes les pages et blocs de contenu de votre site web.

:::info
Lors de la construction de pages dans [Gestion des pages](managing-pages), vous pouvez référencer n'importe quel fichier téléversé ici par son URL. Cela permet de garder tous vos médias organisés dans un emplacement central.
:::

## Supprimer des fichiers

1. Sélectionnez le fichier que vous souhaitez supprimer.
2. Cliquez sur le bouton **Supprimer** pour le retirer définitivement de votre bibliothèque.
3. Assurez-vous que le fichier n'est actuellement utilisé sur aucune page avant de le supprimer, car cela cassera tous les liens ou images qui le référencent.

:::warning
Les fichiers supprimés ne peuvent pas être récupérés. Vérifiez bien qu'un fichier n'est plus utilisé sur aucune page avant de le supprimer.
:::
