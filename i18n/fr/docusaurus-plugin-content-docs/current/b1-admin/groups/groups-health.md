# Santé des groupes

<div class="article-intro">

Le tableau de bord Santé des groupes vous donne une vue d'ensemble sur le fonctionnement de tous vos groupes — tendances d'adhésion, moyennes de participation et croissance ou attrition au cours des 90 derniers jours — le tout dans une seule table triable.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin d'au moins quelques groupes avec des membres pour voir des données significatives. Voir [Création de groupes](creating-groups).
- Les données de présence sont tirées des sessions enregistrées. Voir la section [Présence](../attendance/).

</div>

## Ouverture de la santé des groupes

Dans B1 Admin, allez à **Groupes** dans la barre latérale gauche et sélectionnez **Santé**. Le tableau de bord charge un tableau avec une ligne par groupe.

## Colonnes

| Colonne | Ce qu'elle affiche |
|--------|--------------|
| **Nom** | Le nom du groupe, lié à la page de détails du groupe |
| **Catégorie** | La catégorie du groupe |
| **Membres** | Nombre actuel de membres actifs |
| **Adhérés (90j)** | Membres qui ont adhéré au cours des 90 derniers jours |
| **Partis (90j)** | Membres qui ont quitté au cours des 90 derniers jours |
| **Attrition (90j)** | Taux d'attrition net en pourcentage sur 90 jours |
| **Présence moyenne** | Nombre moyen de participants par session de présence |

Cliquez sur n'importe quel en-tête de colonne pour trier le tableau par cette colonne. Cliquez à nouveau pour inverser l'ordre de tri.

## Utilisation des données de santé

- **Attrition élevée + adhésions faibles** — un groupe qui rétrécit et ne remplace pas les membres perdus. Cela vaut la peine d'en parler avec le leader du groupe.
- **Adhésions élevées + présence faible** — les gens s'inscrivent mais ne se présentent pas. Envisagez un suivi d'engagement.
- **Présence moyenne élevée** — un groupe sain et actif. Modèle potentiel pour d'autres groupes.

:::tip
Cliquer sur le nom d'un groupe vous amène directement à la page de détails du groupe où vous pouvez examiner les membres individuels, les dossiers de présence et les événements du calendrier.
:::
