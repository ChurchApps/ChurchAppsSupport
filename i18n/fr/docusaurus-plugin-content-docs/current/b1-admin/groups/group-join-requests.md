# Demandes d'adhésion aux groupes

<div class="article-intro">

Lorsqu'un groupe est configuré avec une politique d'adhésion basée sur l'approbation, les personnes peuvent soumettre des demandes pour rejoindre. Les responsables de groupe et les administrateurs examinent ces demandes et les approuvent ou les refusent. Cela donne à votre église un contrôle sur l'appartenance aux groupes tout en facilitant l'expression de l'intérêt des personnes à rejoindre.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin de la permission de gérer les groupes, ou vous devez être responsable du groupe spécifique. Consultez [Rôles et permissions](../people/roles-permissions.md) pour plus de détails.
- Le groupe doit avoir sa politique d'adhésion définie sur **Demande** (approbation requise). Consultez [Créer des groupes](./creating-groups.md) pour savoir comment configurer les politiques d'adhésion.

</div>

## Comprendre les politiques d'adhésion

Les groupes peuvent avoir trois politiques d'adhésion différentes :

- **Ouvert** -- Tout le monde peut rejoindre immédiatement sans approbation
- **Demande** -- Les personnes soumettent une demande d'adhésion qui nécessite une approbation
- **Fermé** -- Personne ne peut demander à rejoindre (les membres doivent être ajoutés manuellement)

Lorsqu'un groupe utilise la politique **Demande**, toutes les tentatives d'adhésion passent par le flux de travail d'approbation décrit sur cette page.

## Consulter les demandes en attente

### Pour les responsables de groupe

1. Accédez à **Groupes** dans B1 Admin
2. Cliquez sur le nom du groupe
3. Cliquez sur l'onglet **Demandes d'adhésion**
4. Vous verrez toutes les demandes en attente pour ce groupe

### Pour les administrateurs

Les administrateurs avec des permissions de gestion de groupe peuvent consulter les demandes en attente pour tous les groupes :

1. Accédez à **Groupes** dans B1 Admin
2. Recherchez un badge de notification ou un indicateur de demandes en attente
3. Cliquez pour consulter toutes les demandes en attente de l'église

## Examiner une demande d'adhésion

Chaque demande d'adhésion affiche :

- **Nom et photo de la personne** -- La personne demandant à rejoindre
- **Message optionnel** -- Un message personnel expliquant pourquoi elle veut rejoindre (si fourni)
- **Date de la demande** -- Quand la demande a été soumise

Pour examiner une demande :

1. Lisez le message de la personne si elle en a fourni un
2. Cliquez sur le nom de la personne pour consulter son profil si nécessaire
3. Décidez d'approuver ou de refuser

## Approuver une demande

1. Cliquez sur **Approuver** sur la demande d'adhésion
2. La personne est immédiatement ajoutée au groupe en tant que membre
3. Le demandeur reçoit une notification indiquant que sa demande a été approuvée
4. La demande est marquée comme approuvée dans le système

:::tip
Lorsque vous approuvez une demande, la personne devient un membre régulier du groupe. Vous pouvez ultérieurement la promouvoir responsable de groupe si nécessaire depuis la page [Membres du groupe](./group-members.md).
:::

## Refuser une demande

1. Cliquez sur **Refuser** sur la demande d'adhésion
2. Fournissez éventuellement une raison du refus (jusqu'à 500 caractères)
3. Cliquez sur **Confirmer**
4. Le demandeur reçoit une notification avec votre raison de refus (si fournie)
5. La demande est marquée comme refusée

:::info
Fournir une raison de refus aide la personne à comprendre pourquoi sa demande n'a pas été approuvée et peut l'encourager à réessayer plus tard ou à explorer d'autres groupes.
:::

## Notifications

Le système de demandes d'adhésion envoie automatiquement des notifications :

- **Lorsqu'une demande est soumise** -- Tous les responsables de groupe reçoivent une notification
- **Lorsqu'une demande est approuvée** -- Le demandeur reçoit une confirmation
- **Lorsqu'une demande est refusée** -- Le demandeur reçoit une notification avec toute raison de refus

Les notifications apparaissent dans le centre de notifications de l'utilisateur sur B1.church et dans l'application mobile.

## Gérer les demandes du côté des membres

Les personnes peuvent gérer leurs propres demandes d'adhésion depuis B1.church :

- Consulter le statut de leurs demandes en attente sur la page de détails du groupe
- Annuler une demande en attente si elles changent d'avis
- Voir si leur demande a été approuvée ou refusée

## Bonnes pratiques

- **Répondez rapidement** -- Essayez d'examiner les demandes dans les 24 à 48 heures pour que les gens ne restent pas en attente
- **Soyez clair dans les raisons de refus** -- Aidez les gens à comprendre les prochaines étapes ou les options alternatives
- **Vérifiez les profils** -- Examinez le profil de la personne pour voir si elle convient bien au groupe
- **Communiquez les attentes** -- Assurez-vous que votre description de groupe indique clairement à qui le groupe s'adresse
