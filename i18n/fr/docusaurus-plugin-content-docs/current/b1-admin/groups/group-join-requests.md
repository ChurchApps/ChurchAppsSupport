# Demandes d'Adhésion aux Groupes

<div class="article-intro">

Lorsqu'un groupe est configuré avec une politique d'adhésion basée sur l'approbation, les gens peuvent soumettre des demandes d'adhésion. Les dirigeants et administrateurs de groupe examinent ces demandes et les approuvent ou les rejettent. Cela donne à votre église le contrôle sur l'adhésion au groupe tout en permettant aux gens d'exprimer facilement leur intérêt à rejoindre.

</div>

<div class="prereqs">
<h4>Avant de Commencer</h4>

- Vous avez besoin d'une permission pour gérer les groupes, ou vous devez être un dirigeant du groupe spécifique. Voir [Rôles et Permissions](../people/roles-permissions.md) pour plus de détails.
- Le groupe doit avoir sa politique d'adhésion définie sur **Demande** (approbation requise). Voir [Créer des Groupes](./creating-groups.md) pour savoir comment configurer les politiques d'adhésion.

</div>

## Comprendre les Politiques d'Adhésion

Les groupes peuvent avoir trois politiques d'adhésion différentes :

- **Ouvert** -- N'importe qui peut rejoindre immédiatement sans approbation
- **Demande** -- Les gens soumettent une demande d'adhésion qui nécessite une approbation
- **Fermé** -- Personne ne peut demander à rejoindre (les membres doivent être ajoutés manuellement)

Lorsqu'un groupe utilise la politique **Demande**, toutes les tentatives d'adhésion passent par le flux d'approbation décrit sur cette page.

## Afficher les Demandes Mises en Attente

### Pour les Dirigeants du Groupe

1. Allez à **Groupes** dans B1 Admin
2. Cliquez sur le nom du groupe
3. Cliquez sur l'onglet **Demandes d'Adhésion**
4. Vous verrez toutes les demandes mises en attente pour ce groupe

### Pour les Administrateurs

Les administrateurs ayant la permission de gérer les groupes peuvent afficher les demandes mises en attente dans tous les groupes :

1. Allez à **Groupes** dans B1 Admin
2. Recherchez un badge de notification ou un indicateur de demandes mises en attente
3. Cliquez pour afficher toutes les demandes mises en attente au niveau de l'église

## Examiner une Demande d'Adhésion

Chaque demande d'adhésion affiche :

- **Nom et photo du demandeur** -- La personne demandant à rejoindre
- **Message optionnel** -- Un message personnel expliquant pourquoi il veut rejoindre (le cas échéant)
- **Date de la demande** -- Quand la demande a été soumise

Pour examiner une demande :

1. Lisez le message de la personne s'il a fourni un
2. Cliquez sur le nom de la personne pour afficher son profil si nécessaire
3. Décidez d'approuver ou de rejeter

## Approuver une Demande

1. Cliquez sur **Approuver** dans la demande d'adhésion
2. La personne est immédiatement ajoutée au groupe comme membre
3. Le demandeur reçoit une notification que sa demande a été approuvée
4. La demande est marquée comme approuvée dans le système

:::tip
Lorsque vous approuvez une demande, la personne devient un membre régulier du groupe. Vous pouvez ensuite la promouvoir au rôle de dirigeant du groupe si nécessaire à partir de la page [Membres du Groupe](./group-members.md).
:::

## Rejeter une Demande

1. Cliquez sur **Rejeter** dans la demande d'adhésion
2. Fournissez éventuellement une raison du rejet (jusqu'à 500 caractères)
3. Cliquez sur **Confirmer**
4. Le demandeur reçoit une notification avec votre raison de rejet (le cas échéant)
5. La demande est marquée comme rejetée

:::info
Fournir une raison de rejet aide la personne à comprendre pourquoi sa demande n'a pas été approuvée et peut l'encourager à réessayer plus tard ou à explorer d'autres groupes.
:::

## Notifications

Le système de demande d'adhésion envoie automatiquement des notifications :

- **Quand une demande est soumise** -- Tous les dirigeants du groupe reçoivent une notification
- **Quand une demande est approuvée** -- Le demandeur reçoit une confirmation
- **Quand une demande est rejetée** -- Le demandeur reçoit une notification avec votre raison de rejet

Les notifications apparaissent dans le centre de notifications de l'utilisateur sur B1.church et dans l'application mobile.

## Gérer les Demandes du Côté du Membre

Les gens peuvent gérer leurs propres demandes d'adhésion depuis B1.church :

- Afficher le statut de leurs demandes mises en attente sur la page de détails du groupe
- Annuler une demande mise en attente s'ils changent d'avis
- Voir si leur demande a été approuvée ou rejetée

## Bonnes Pratiques

- **Répondre rapidement** -- Essayez d'examiner les demandes dans les 24-48 heures pour que les gens ne restent pas dans l'attente
- **Être clair dans les raisons de rejet** -- Aidez les gens à comprendre les prochaines étapes ou les options alternatives
- **Vérifier les profils** -- Examinez le profil de la personne pour voir si elle convient bien au groupe
- **Communiquer les attentes** -- Assurez-vous que la description de votre groupe indique clairement à qui le groupe s'adresse
