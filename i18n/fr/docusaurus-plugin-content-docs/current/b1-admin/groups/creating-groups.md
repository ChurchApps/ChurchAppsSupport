# Création de groupes

<div class="article-intro">

La création d'un groupe dans B1 Admin est simple. Vous allez configurer une catégorie, nommer votre groupe, puis configurer ses paramètres. Les groupes vous aident à organiser votre église en unités significatives comme les petits groupes, les comités et les classes.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin d'un compte B1 Admin actif avec l'autorisation de gérer les groupes. Consultez [Rôles et autorisations](../people/roles-permissions.md) si vous n'êtes pas sûr de votre niveau d'accès.
- Décidez d'une structure de catégories pour vos groupes (par exemple, « Petits groupes », « Ministères », « Comités »). Les catégories aident à organiser les groupes connexes.

</div>

## Ajout d'un nouveau groupe

1. Allez au **tableau de bord B1 Admin**.
2. Cliquez sur l'onglet **Groupes**.
3. Cliquez sur **Ajouter un groupe** et entrez un **Nom de catégorie**. Les catégories vous aident à organiser les groupes connexes ensemble (par exemple, « Petits groupes », « Ministères » ou « Comités »). Si une catégorie existe déjà, vous pouvez la sélectionner dans la liste.
4. Entrez le **Nom du groupe**.
5. Cliquez sur **Ajouter**. Votre nouveau groupe apparaîtra dans la liste sous la catégorie choisie.

## Configuration des paramètres du groupe

Une fois votre groupe créé, vous pouvez remplir des détails supplémentaires :

1. Cliquez sur le **nom du groupe** dans la liste pour l'ouvrir.
2. Cliquez sur l'**icône crayon** pour modifier les paramètres du groupe.
3. Configurez les options suivantes :
   - **Description** -- Un bref résumé de ce que le groupe fait. Ceci est visible aux membres.
   - **Heures de réunion** -- Quand le groupe se réunit habituellement (par exemple, « Mercredi à 19h »).
   - **Politique de participation** -- Choisissez qui peut rejoindre ce groupe :
     - **Ouvert** -- N'importe qui peut rejoindre immédiatement sans approbation
     - **Demande** -- Les gens doivent soumettre une demande d'adhésion qui nécessite une approbation (consultez [Demandes d'adhésion au groupe](./group-join-requests.md))
     - **Fermé** -- Les membres doivent être ajoutés manuellement par les leaders ou administrateurs
   - **Étiquettes** -- Affectez une ou plusieurs étiquettes descriptives au groupe (par exemple, « En personne », « En ligne », « Les nouveaux membres bienvenus »). Les étiquettes sont des balises libres que vous définissez ; cochez tous les éléments qui s'appliquent. Les étiquettes peuvent être utilisées pour filtrer les groupes dans l'élément du site Groups Browser.
   - **Groupe confidentiel** -- Masquez ce groupe et sa liste de membres des pages publiques, du chercheur de groupe et des non-membres. Utilisez ceci pour les groupes sensibles comme les ministères de rétablissement ou de conseil ; seuls les membres du groupe et le personnel de l'église peuvent le voir.
   - **Suivi de présence** -- Activez ceci si vous souhaitez enregistrer la [présence](../attendance/tracking-attendance.md) de ce groupe.
   - **Heures de service** -- Associez le groupe avec des heures de service d'église spécifiques le cas échéant. Consultez [Configuration de la présence](../attendance/setup.md) pour plus de détails sur les heures de service.
4. Cliquez sur **Enregistrer** pour appliquer vos modifications.

:::tip
Ajouter une description claire et une heure de réunion aide les membres à savoir à quoi s'attendre quand ils rejoignent un groupe.
:::

## Duplication d'un groupe

Commencez une nouvelle session d'un cours ou ministère récurrent ? Au lieu de re-saisir tous les paramètres, dupliquez un groupe existant :

1. Ouvrez le groupe et cliquez sur l'**icône dupliquer** dans la bannière du groupe (à côté de Modifier).
2. Confirmez la duplication.

La copie reprend les paramètres de l'original -- catégorie, description, heure/lieu de réunion, politique de participation, étiquettes et site -- mais **pas** ses membres. Le nouveau groupe est nommé d'après l'original avec « (Copie) » ajouté ; renommez-le à partir des paramètres du groupe.

## Archivage d'un groupe

Lorsqu'un groupe n'est plus actif mais que vous souhaitez conserver son historique au lieu de le supprimer :

1. Ouvrez le groupe et cliquez sur l'**icône crayon** pour modifier ses paramètres.
2. Cliquez sur **Archiver** et confirmez.

Les groupes archivés disparaissent de la liste principale des groupes. Pour en trouver un à nouveau, activez la bascule **Afficher les archivés** en haut de la page Groupes, puis cliquez sur **Restaurer** à côté du groupe pour le ramener.

:::info
L'archivage d'un groupe ne supprime pas ses membres, son historique de présence ou ses événements de calendrier -- il masque simplement le groupe de la liste par défaut jusqu'à ce que vous le restauriez.
:::

## Prochaines étapes

Après la création et la configuration de votre groupe, vous êtes prêt à :

- **Ajouter des membres** -- Recherchez des personnes et ajoutez-les au groupe. Utilisez l'icône de clé verte pour désigner les leaders du groupe. Consultez [Membres du groupe](./group-members.md).
- **Configurer un calendrier** -- Créez des événements et des réunions récurrentes pour le groupe. Consultez [Calendrier du groupe](./group-calendar.md).
- **Communiquer** -- Envoyez des messages à tous les membres du groupe directement depuis la page du groupe.
- **Exporter les données** -- Cliquez sur l'icône de téléchargement pour exporter la liste des membres de votre groupe.

:::info
Tous les groupes de votre église sont organisés par catégories sur la page principale des groupes. Vous pouvez toujours réorganiser ou renommer les catégories à mesure que votre église se développe.
:::
