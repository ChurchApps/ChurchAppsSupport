# Calendrier du groupe

<div class="article-intro">

Chaque groupe dans B1 peut avoir son propre calendrier d'événements. Les responsables de groupe l'utilisent pour planifier des réunions, des activités et des événements récurrents, tandis que tous les membres du groupe peuvent consulter les événements à venir et participer aux conversations à leur sujet. Le calendrier garde votre groupe organisé et s'assure que tout le monde reste informé de ce qui se prépare.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous devez avoir un groupe créé dans B1 Admin avec au moins un membre. Consultez [Créer des groupes](creating-groups.md) et [Membres du groupe](group-members.md).
- Pour ajouter ou modifier des événements du calendrier, vous devez être désigné comme responsable de groupe. Consultez [Désigner des responsables de groupe](group-members.md) pour les instructions.

</div>

## Accéder au calendrier du groupe

Le calendrier du groupe est disponible via l'application B1.church destinée aux membres :

1. Depuis B1 Admin, cliquez sur le **menu déroulant** dans le coin supérieur droit de votre écran.
2. Sélectionnez **Changer d'application**.
3. Choisissez **B1.church**.
4. Cliquez sur **Groupes** dans la barre latérale gauche.
5. Sélectionnez votre groupe dans la liste.
6. Cliquez sur **Calendrier** sur le côté gauche de l'écran.

:::info
Le calendrier du groupe est géré via B1.church (l'application membre) plutôt que via B1 Admin. Les responsables de groupe y accèdent en changeant d'application depuis le tableau de bord administrateur.
:::

## Ajouter un événement

1. Depuis le calendrier du groupe, cliquez sur **Ajouter un événement**.
2. Entrez un **titre** pour l'événement.
3. Rédigez une courte **description** de l'événement.
4. Définissez l'**heure de début** et l'**heure de fin**.
5. Cliquez sur **Enregistrer** pour ajouter l'événement au calendrier.

## Configurer des événements récurrents

Si votre groupe se réunit selon un programme régulier, vous pouvez configurer des événements pour qu'ils se répètent automatiquement :

1. Lors de la création ou de la modification d'un événement, cochez la case **Récurrent**.
2. Choisissez comment l'événement doit se répéter :
   - **Intervalle** -- Définissez un nombre de jours entre les occurrences. Par exemple, entrer « 7 » répète l'événement tous les 7 jours.
   - **Fréquence** -- Choisissez de répéter par **jour**, **semaine** ou **mois**.
3. Enregistrez l'événement. Il apparaîtra maintenant dans le calendrier à chaque occurrence prévue.

:::tip
Pour un groupe qui se réunit tous les samedis, réglez la fréquence sur « semaine » et choisissez samedi. L'événement se remplira automatiquement dans le calendrier à l'avenir.
:::

## Consulter le calendrier

Le calendrier du groupe offre plusieurs options d'affichage :

- **Mois** -- Voir le mois complet en un coup d'œil.
- **Semaine** -- Se concentrer sur la semaine en cours.
- **Jour** -- Voir une seule journée en détail.
- **Agenda** -- Voir une liste des événements à venir dans l'ordre chronologique.

Basculez entre les vues à l'aide des boutons en haut du calendrier.

## Détails de l'événement et conversations

Cliquez sur n'importe quel événement pour voir ses détails. Depuis la vue de détail de l'événement :

- **Les membres et les responsables** peuvent démarrer des conversations et laisser des commentaires sur l'événement.
- **Les responsables** peuvent modifier les détails de l'événement, changer les horaires ou supprimer l'événement.

:::warning
La suppression d'un événement le retire définitivement. Si vous devez annuler une seule occurrence d'un événement récurrent, envisagez plutôt de le modifier.
:::

## S'abonner au calendrier

Vous pouvez synchroniser le calendrier du groupe avec votre calendrier personnel :

1. Depuis la page du calendrier du groupe, cliquez sur **S'abonner**.
2. Suivez les instructions pour ajouter le calendrier à Google Calendar, Apple Calendar, Outlook ou toute autre application de calendrier prenant en charge les abonnements.

:::tip
S'abonner permet de garder votre agenda personnel synchronisé avec les événements de votre groupe sans avoir à consulter B1 séparément. Toute modification apportée par les responsables de groupe apparaîtra automatiquement dans votre calendrier synchronisé.
:::
