# Groupes

<div class="article-intro">

La section **Groupes** vous permet d'organiser les membres de votre église en équipes, ministères, petits groupes, études bibliques, comités et bien plus encore. Les groupes sont l'épine dorsale de la vie communautaire de votre église, et cette section vous donne les outils pour créer, gérer et communiquer avec chaque groupe en un seul endroit.

</div>

## Aperçu

Les groupes sont organisés par **catégorie**. Une catégorie est une étiquette de niveau supérieur comme « Petits groupes », « Ministères » ou « Comités ». Chaque catégorie peut contenir plusieurs groupes, ce qui facilite le regroupement des groupes apparentés.

Lorsque vous ouvrez la page Groupes, vous verrez un résumé en haut affichant :

- **Total des groupes** -- le nombre de groupes dans toutes les catégories
- **Catégories** -- le nombre de catégories de groupes que vous avez créées
- **Total des membres** -- le nombre combiné de membres dans tous les groupes

Sous le résumé, vos groupes sont listés par catégorie avec le nombre de membres affiché pour chaque groupe.

## Ce que vous pouvez faire

Voici ce que vous pouvez accomplir dans la section Groupes :

1. **Créer et organiser des groupes** -- Ajoutez de nouveaux groupes au sein de catégories pour refléter la structure de votre église. Consultez [Créer des groupes](./creating-groups.md).

2. **Gérer les membres du groupe** -- Ajoutez et supprimez des membres, désignez des responsables et communiquez avec vos groupes. Consultez [Membres du groupe](./group-members.md).

3. **Planifier des événements** -- Chaque groupe peut avoir son propre calendrier pour les réunions, événements et activités récurrentes. Consultez [Calendrier du groupe](./group-calendar.md).

4. **Exporter les données du groupe** -- Téléchargez les listes de membres pour n'importe quel groupe sous forme de fichier CSV pour une utilisation dans des tableurs ou des envois postaux. Consultez [Membres du groupe](./group-members.md) pour les instructions d'exportation.

## Pour commencer

Pour accéder aux Groupes, cliquez sur **Personnes** dans la barre latérale gauche de votre tableau de bord B1 Admin, puis sélectionnez **Groupes**. Vous verrez tous vos groupes existants organisés par catégorie.

:::tip
Si vous débutez, commencez par créer quelques catégories qui correspondent à la structure de votre église (par exemple, « Ministères », « Petits groupes », « Équipes »). Ensuite, ajoutez des groupes individuels dans chaque catégorie. Consultez [Créer des groupes](./creating-groups.md) pour un guide étape par étape.
:::

:::info
Les membres du groupe doivent d'abord exister dans votre annuaire [Personnes](../people/adding-people.md) avant de pouvoir être ajoutés à un groupe. Si vous configurez B1 pour la première fois, [importez vos données de membres](../people/importing-data.md) avant de créer des groupes.
:::
