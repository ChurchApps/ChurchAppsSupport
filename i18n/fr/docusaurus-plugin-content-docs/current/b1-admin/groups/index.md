# Groupes

<div class="article-intro">

La section **Groupes** vous permet d'organiser les membres de votre église en ministères, petits groupes, études bibliques, comités et bien d'autres. Les groupes sont l'épine dorsale de la vie communautaire de votre église, et cette section vous donne les outils pour créer, gérer et communiquer avec chaque groupe en un seul endroit. Notez que les **Équipes** sont séparées des Groupes — les Équipes sont utilisées pour le service et la planification des bénévoles.

</div>

## Vue d'ensemble

Les groupes sont organisés par **catégorie**. Une catégorie est une étiquette de haut niveau comme "Petits groupes", "Ministères" ou "Comités". Chaque catégorie peut contenir plusieurs groupes, ce qui facilite le regroupement des groupes connexes.

Lorsque vous ouvrez la page Groupes, vous verrez un résumé en haut affichant :

- **Total des groupes** -- le nombre de groupes dans toutes les catégories
- **Catégories** -- le nombre de catégories de groupes que vous avez créées
- **Total des membres** -- le compte combiné des membres dans tous les groupes

Sous le résumé, vos groupes sont listés par catégorie avec le nombre de membres affichés pour chaque groupe.

## Ce que vous pouvez faire

Voici ce que vous pouvez accomplir dans la section Groupes :

1. **Créer et organiser des groupes** -- Ajoutez de nouveaux groupes dans les catégories pour refléter comment votre église est structurée. Voir [Création de groupes](./creating-groups.md).

2. **Gérer les membres du groupe** -- Ajoutez et supprimez des membres, désignez des responsables et communiquez avec vos groupes. Voir [Membres du groupe](./group-members.md).

3. **Planifier des événements** -- Chaque groupe peut avoir son propre calendrier pour les réunions, les événements et les activités récurrentes. Voir [Calendrier du groupe](./group-calendar.md).

4. **Exporter les données du groupe** -- Téléchargez les listes des membres de tout groupe en tant que fichier CSV à utiliser dans les feuilles de calcul ou les envois. Voir [Membres du groupe](./group-members.md) pour les instructions d'export.

## Démarrage

Pour accéder à Groupes, cliquez sur **Personnes** dans la barre latérale gauche de votre tableau de bord B1 Admin, puis sélectionnez **Groupes**. Vous verrez tous vos groupes existants organisés par catégorie.

:::tip
Si vous commencez tout juste, commencez par créer quelques catégories qui correspondent à votre structure d'église (par exemple, "Ministères", "Petits groupes", "Comités"). Puis ajoutez des groupes individuels au sein de chaque catégorie. Voir [Création de groupes](./creating-groups.md) pour une présentation étape par étape.
:::

:::info
Les membres du groupe doivent d'abord exister dans votre répertoire [Personnes](../people/adding-people.md) avant de pouvoir être ajoutés à un groupe. Si vous configurez B1 pour la première fois, [importez vos données de membres](../people/importing-data.md) avant de créer des groupes.
:::
