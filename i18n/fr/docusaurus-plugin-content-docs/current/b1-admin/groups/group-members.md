# Membres du groupe

<div class="article-intro">

Une fois que vous avez créé un groupe, l'étape suivante est l'ajout de membres. À partir de la page de détail d'un groupe, vous pouvez chercher des personnes, les ajouter au groupe, assigner des responsables, envoyer des messages et exporter la liste des membres. La gestion de l'adhésion aux groupes est essentielle pour coordonner les petits groupes, les comités et les classes.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin d'au moins un groupe configuré dans B1 Admin. Voir [Création de groupes](creating-groups.md) si vous n'en avez pas encore créé.
- Les personnes que vous voulez ajouter doivent déjà exister dans votre répertoire [Personnes](../people/adding-people.md).

</div>

## Ajout de membres à un groupe

1. Allez à la page **Groupes** et cliquez sur le groupe que vous voulez gérer.
2. Cliquez sur l'onglet **Membres**.
3. Dans la boîte de recherche, tapez le nom de la personne que vous voulez ajouter.
4. Cliquez sur **Ajouter** à côté du nom de la personne dans les résultats de recherche.
5. La personne apparaît maintenant dans la liste des membres du groupe.

:::tip
Laissez la boîte de recherche vide et cliquez sur **Rechercher** pour parcourir tout votre répertoire. Ceci est utile si vous n'êtes pas sûr de l'orthographe exacte du nom de quelqu'un.
:::

## Désignation des chefs de groupe

Les chefs de groupe ont des privilèges spéciaux -- ils peuvent éditer le [calendrier du groupe](group-calendar.md), gérer les événements et aider à coordonner le groupe.

1. Dans la liste des membres du groupe, trouvez la personne que vous voulez faire chef.
2. Cliquez sur l'icône de **clé verte** à côté de son nom.
3. La personne est maintenant désignée comme chef de groupe.

Pour supprimer le statut de chef, cliquez à nouveau sur l'icône de clé verte.

:::info
Tout membre du groupe peut voir le calendrier et les événements du groupe, mais seuls les chefs peuvent ajouter ou modifier les événements du calendrier.
:::

## Envoi de messages aux membres du groupe

Vous pouvez communiquer avec tous les membres d'un groupe directement depuis B1 Admin :

1. Depuis la page de détail du groupe, recherchez la zone de messagerie.
2. Tapez votre message dans la boîte de texte.
3. Cliquez sur **Envoyer**.

Votre message sera livré à tous les membres du groupe.

## Envoi d'e-mails aux membres du groupe

Vous pouvez envoyer des e-mails formatés à tous les membres d'un groupe :

1. Depuis la page de détail du groupe, cliquez sur l'icône **e-mail**.
2. La boîte de dialogue Envoyer un e-mail s'ouvre, affichant le nombre de membres qui recevront l'e-mail et le nombre n'ayant pas d'adresse e-mail en dossier.
3. Optionnellement sélectionnez un **modèle d'e-mail** dans la liste déroulante, ou composez un message de zéro. Cliquez sur **Gérer les modèles** pour créer ou éditer des modèles.
4. Entrez une **ligne d'objet**. Vous pouvez insérer des champs de fusion en cliquant sur les puces de champ : `{{firstName}}`, `{{lastName}}`, `{{displayName}}`, `{{email}}`, `{{churchName}}`.
5. Composez le **corps de l'e-mail** en utilisant l'éditeur HTML. Les mêmes champs de fusion sont disponibles ici.
6. Cliquez sur **Envoyer**.
7. Un résumé affiche le nombre d'e-mails envoyés avec succès et le nombre de membres ignorés (pas d'e-mail en dossier).

:::tip
Créez des modèles d'e-mail réutilisables pour les communications récurrentes comme les mises à jour hebdomadaires, les annonces d'événements ou les demandes de prière. Les modèles économisent du temps et assurent une messagerie cohérente.
:::

## Export de données du groupe

Pour télécharger la liste des membres du groupe en tant que fichier :

1. Depuis la page de détail du groupe, cliquez sur l'icône de **téléchargement**.
2. Un fichier CSV contenant les informations des membres du groupe téléchargera sur votre ordinateur.

Ceci est utile pour créer des listes imprimées, importer des données dans d'autres outils ou garder des enregistrements hors ligne. Pour plus d'options d'export, voir [Export de données](../people/exporting-data.md).

## Suppression de membres

Pour supprimer quelqu'un d'un groupe, localisez son nom dans la liste des membres et cliquez sur le bouton **supprimer** à côté de son entrée.

:::info
Supprimer une personne d'un groupe ne la supprime pas de votre répertoire d'église. Elle apparaîtra toujours dans la section [Personnes](../people/adding-people.md) et peut être réajoutée au groupe à tout moment.
:::
