# Gérer les petits groupes

<div class="article-intro">

Mettez en place des petits groupes, des études bibliques ou des équipes de ministère avec la gestion des membres, les calendriers de groupe et les conversations. Publiez un calendrier d'événements à l'échelle de l'église sur votre site web afin que les visiteurs puissent voir ce qui se passe et que les membres restent connectés via le portail web et l'application mobile.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Un compte B1 Admin avec un accès administrateur
- [Des personnes ajoutées à votre répertoire](../people/adding-people.md) (les membres des groupes doivent exister au préalable)

</div>

## Étape 1 : Créer les catégories et les groupes

Organisez vos groupes en catégories afin que les membres puissent facilement trouver ce qu'ils cherchent.

Suivez le guide [Créer des groupes](../groups/creating-groups.md) pour :

1. Créer des catégories pour organiser vos groupes (par ex. « Petits groupes », « Études bibliques », « Équipes de ministère », « Comités »)
2. Créer des groupes au sein de chaque catégorie (par ex. « Groupe hommes du mardi soir », « Étude biblique femmes », « Ministère jeunesse »)
3. Ajouter une description et un horaire de réunion pour chaque groupe

## Étape 2 : Ajouter des membres et des responsables

Remplissez vos groupes avec des membres et désignez des responsables qui pourront gérer leurs propres équipes.

Suivez le guide [Membres du groupe](../groups/group-members.md) pour :

1. Rechercher des personnes et les ajouter aux groupes appropriés
2. Désigner les responsables de groupe à l'aide de l'icône de clé verte -- les responsables obtiennent des permissions supplémentaires

:::tip
Les responsables de groupe peuvent gérer le calendrier de leur propre groupe, publier des conversations et coordonner des événements sans avoir besoin d'un accès administrateur complet.
:::

## Étape 3 : Configurer les calendriers de groupe

Donnez à chaque groupe son propre calendrier pour les réunions et les événements.

Suivez le guide [Calendrier du groupe](../groups/group-calendar.md) pour :

1. Ajouter des réunions récurrentes (par ex. « Chaque mardi à 19h ») et des événements ponctuels (par ex. « Barbecue du groupe - 15 mars »)
2. Les responsables de groupe peuvent ajouter et gérer les événements de leur propre groupe

## Étape 4 : Créer un calendrier à l'échelle de l'église

Regroupez les événements de tous vos groupes dans un calendrier unifié pour toute l'église.

Suivez les guides [Créer des calendriers](../calendars/creating-calendars.md) et [Calendrier organisé](../calendars/curated-calendar.md) pour :

1. Créer un calendrier organisé qui rassemble les événements de plusieurs groupes dans une vue unifiée
2. Partager le lien d'abonnement ICS pour que les membres puissent ajouter les événements à leurs applications de calendrier personnel (Google Calendar, Apple Calendar, Outlook)

:::info
Un calendrier organisé regroupe automatiquement les événements des groupes que vous connectez. Lorsqu'un responsable de groupe ajoute un événement, il apparaît également sur le calendrier de l'église.
:::

## Étape 5 : Publier sur votre site web

Rendez vos groupes et votre calendrier visibles aux visiteurs et aux membres sur votre site B1.

Suivez le guide [Gestion des pages](../website/managing-pages.md) pour :

1. Ajouter une section Calendrier à votre site B1 pour afficher le calendrier organisé
2. Ajouter une section Groupes pour que les visiteurs puissent parcourir les groupes disponibles et apprendre comment les rejoindre

## Étape 6 : Expérience des membres

Les membres peuvent découvrir, rejoindre et participer aux groupes depuis leur téléphone ou leur ordinateur.

1. Les membres parcourent et rejoignent les groupes sur [B1.church](../../b1-church/groups/browsing-groups.md)
2. Les membres du groupe peuvent consulter les détails, les calendriers et publier dans les [conversations](../../b1-church/groups/conversations.md)
3. Toutes les fonctionnalités de groupe sont également disponibles sur l'[application B1 Mobile](../../b1-mobile/groups/)

:::info
Les membres voient leurs groupes, les événements à venir et les conversations de groupe à la fois sur le web et sur mobile -- tout reste synchronisé.
:::

## C'est terminé !

Vos petits groupes sont configurés et connectés. Les membres peuvent découvrir des groupes, consulter les événements et rester connectés grâce aux conversations -- le tout sur votre site web et votre application mobile. Les responsables de groupe peuvent gérer leurs propres équipes de manière autonome.
