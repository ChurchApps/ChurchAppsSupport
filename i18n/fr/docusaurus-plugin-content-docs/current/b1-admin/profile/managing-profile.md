# Gérer votre profil

<div class="article-intro">

La page Profil vous permet de modifier les paramètres de votre compte personnel. Votre profil est partagé entre tous les produits ChurchApps, de sorte que toute modification effectuée ici sera reflétée partout où vous utilisez ChurchApps.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous devez être connecté à votre compte ChurchApps
- Préparez votre nouvelle adresse e-mail ou votre nouveau mot de passe si vous prévoyez de les mettre à jour

</div>

## Informations du profil

La section supérieure de la page Profil contient les détails de base de votre compte :

1. **E-mail** -- Votre adresse e-mail utilisée pour la connexion et la réception des notifications.
2. **Prénom** et **Nom** -- Votre nom d'affichage dans ChurchApps.
3. **Mot de passe** -- Mettez à jour le mot de passe de votre compte.

Pour mettre à jour vos informations, modifiez les champs et cliquez sur le bouton **Enregistrer les modifications**.

## Changer votre mot de passe

1. Faites défiler jusqu'aux champs de mot de passe sur la page Profil.
2. Saisissez votre nouveau mot de passe dans le champ **Mot de passe**.
3. Saisissez-le à nouveau dans le champ **Confirmer le mot de passe**.
4. Cliquez sur **Enregistrer les modifications**.

:::info
Votre mot de passe doit comporter au moins 8 caractères. Assurez-vous que les champs mot de passe et confirmation correspondent exactement.
:::

## Comptes associés

La section **Comptes associés** affiche les services externes connectés à votre compte ChurchApps. Vous pouvez voir quels comptes sont liés et gérer ces connexions depuis cette section.

## Préférences de thème

ChurchApps prend en charge les modes d'affichage clair et sombre :

1. Trouvez le bouton bascule **Thème** sur votre page Profil.
2. Basculez entre le mode **Clair** et le mode **Sombre**.
3. L'interface se mettra à jour immédiatement pour refléter votre choix.

Votre préférence de thème est enregistrée dans votre compte et s'appliquera à toutes vos sessions.

## Suppression du compte

En bas de la page Profil, vous trouverez l'option de supprimer définitivement votre compte. Cette action supprime votre compte et les données associées et ne peut pas être annulée.

:::warning
La suppression du compte est définitive et irréversible. Assurez-vous de vraiment vouloir supprimer votre compte avant de procéder.
:::

:::info
Si vous êtes un **Administrateur de domaine**, assurez-vous de transférer les responsabilités administratives à un autre utilisateur avant de supprimer votre compte. Consultez [Rôles et autorisations](../settings/roles-permissions.md) pour plus de détails sur la gestion des rôles d'administration.
:::

:::info
La modification du profil est désactivée en mode démonstration. Si vous utilisez un compte de démonstration, vous ne pourrez pas apporter de modifications à vos paramètres de profil.
:::
