# Profil

<div class="article-intro">

La section Profil vous permet de gérer les paramètres de votre compte personnel dans ChurchApps. Votre profil est partagé entre tous les outils ChurchApps, de sorte que les modifications que vous apportez ici s'appliquent partout -- de B1 Admin à l'application B1 Mobile.

</div>

## Ce que vous pouvez faire

Depuis la section Profil, vous pouvez :

- **Mettre à jour votre nom et votre adresse e-mail** -- Gardez vos coordonnées à jour.
- **Changer votre mot de passe** -- Définissez un nouveau mot de passe pour votre compte.
- **Changer de thème** -- Basculez entre le mode clair et le mode sombre selon votre préférence.
- **Gérer les appareils connectés** -- Consultez et contrôlez les appareils ChurchAppsPlayer liés à votre compte.
- **Consulter les comptes associés** -- Visualisez et gérez les services externes connectés à votre compte ChurchApps.

## Sous-sections du profil

La zone Profil contient les pages suivantes :

1. **[Gérer votre profil](./managing-profile.md)** -- Modifiez vos informations personnelles, changez votre mot de passe, gérez les comptes associés, ajustez les préférences de thème et procédez à la suppression de compte.
2. **[Gérer les appareils](./devices.md)** -- Consultez, ajoutez et gérez les appareils ChurchAppsPlayer associés à votre compte.

:::tip
Gardez votre adresse e-mail à jour. Elle est utilisée pour la récupération de compte et les notifications importantes de votre église.
:::
