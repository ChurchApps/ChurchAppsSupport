# Paramètres de l'église

<div class="article-intro">

La page Paramètres de l'église est l'endroit où vous configurez les informations de base de votre église, les coordonnées et l'image de marque. Ces informations sont utilisées dans tous les outils ChurchApps, y compris votre site web B1.church et l'application B1 Mobile.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous devez disposer de l'autorisation « Modifier les paramètres de l'église ». Consultez [Rôles et autorisations](./roles-permissions.md) si vous n'avez pas accès.
- Préparez l'adresse, les coordonnées et le logo de votre église

</div>

## Modifier les informations de votre église

1. Accédez à la page **Paramètres** dans la barre latérale gauche.
2. Cliquez sur le bouton **Modifier les paramètres** dans l'en-tête.
3. Mettez à jour l'un des champs suivants :
   - **Nom de l'église** -- Le nom affiché dans tous les produits ChurchApps.
   - **Adresse** -- L'adresse physique de votre église.
   - **Coordonnées** -- Numéro de téléphone, adresse e-mail et autres informations de contact.
4. Cliquez sur **Enregistrer** pour appliquer vos modifications.

## Configuration de votre sous-domaine

Votre église bénéficie d'un sous-domaine gratuit à l'adresse **votreeglise.b1.church**. Il s'agit de l'adresse web où les membres et les visiteurs peuvent accéder à la présence en ligne de votre église.

1. Sur la page Paramètres, repérez le champ **Sous-domaine**.
2. Saisissez le sous-domaine de votre choix (par exemple, « eglisegrace » pour eglisegrace.b1.church).
3. Enregistrez vos modifications.

:::info
Votre sous-domaine doit être unique parmi toutes les églises ChurchApps. Si le nom que vous souhaitez est déjà pris, essayez d'ajouter votre ville ou votre région (par exemple, « eglisegrace-paris »).
:::

## Configuration de l'image de marque

Personnalisez l'apparence de votre église dans tous les outils ChurchApps :

1. Téléchargez le **logo de votre église** en cliquant sur la zone du logo et en sélectionnant un fichier image.
2. Ajoutez des **images supplémentaires de l'église** utilisées sur votre site web et dans l'[application mobile](./mobile-app.md).

:::tip
Pour de meilleurs résultats, utilisez un logo avec un fond transparent au format PNG. Cela garantit un rendu optimal sur les fonds clairs comme sur les fonds sombres.
:::

## Import et export

Le bouton **Import/Export** dans l'en-tête des Paramètres ouvre un outil dédié dans une nouvelle fenêtre de navigateur. Utilisez-le pour :

- Importer les données des membres depuis un autre système de gestion d'église.
- Exporter vos données ChurchApps à des fins de sauvegarde ou de migration.

Cela est particulièrement utile lorsque vous configurez votre église pour la première fois et que vous devez transférer des dossiers existants dans ChurchApps.

:::warning
Lors de l'importation de données, sauvegardez toujours vos dossiers existants au préalable. Les opérations d'importation ajoutent des données à votre système et peuvent créer des doublons si elles sont exécutées plusieurs fois.
:::
