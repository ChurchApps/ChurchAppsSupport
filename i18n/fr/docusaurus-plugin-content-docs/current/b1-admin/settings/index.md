# Paramètres

<div class="article-intro">

La section Paramètres est l'endroit où vous configurez le compte de votre église, la marque et les options administratives dans B1 Admin. À partir d'ici, vous pouvez gérer tout, du nom et du sous-domaine de votre église aux permissions des utilisateurs et à la configuration de l'application mobile.

</div>

## Ce que vous trouverez ici

La zone Paramètres est organisée dans les sous-sections suivantes accessibles depuis la barre latérale :

1. **Paramètres** -- Configurez les informations de base, la marque et le sous-domaine de votre église. Voir [Paramètres de l'église](./church-settings.md) pour les détails.
2. **Campus** -- Ajoutez et gérez plusieurs emplacements physiques pour votre église. Une fois créés, les campus apparaissent sur les profils des personnes, dans la configuration de la présence et dans le tableau de bord des données démographiques. Voir [Campus](./campuses.md) pour les détails.
3. **Applications mobiles** -- Configurez et personnalisez les onglets de navigation qui apparaissent dans le [B1.church PWA](/docs/b1-church/getting-started/installing-pwa) pour vos membres (les mêmes onglets sont également rendus par l'application native B1 Mobile dépréciée). Voir [Paramètres de l'application mobile](./mobile-app.md) pour les détails.
4. **Administrateur du serveur** -- Accédez aux outils d'administration avancés pour gérer les paramètres au niveau du serveur de votre église.
5. **Formulaires** -- Créez et gérez des formulaires personnalisés pour collecter des informations de votre congrégation.

## La page Paramètres principale

Lorsque vous ouvrez d'abord la page Paramètres, vous verrez le **nom de votre église** et le **sous-domaine** affichés en haut. L'en-tête fournit des boutons d'accès rapide pour les tâches courantes :

- **Modifier les paramètres** -- Mettez à jour le nom, l'adresse, les coordonnées et la marque de votre église.
- **Applications mobiles** -- Accédez directement à la configuration de l'application mobile.
- **Rôles** -- Gérez les rôles et les permissions des utilisateurs pour votre équipe.
- **Importer/Exporter** -- Transférez les données entre les systèmes en utilisant l'outil d'importation/exportation.

:::tip
Commencez par configurer le nom et la marque de votre église sous **Modifier les paramètres**, puis configurez les [Rôles](./roles-permissions.md) pour inviter vos membres d'équipe avec le bon niveau d'accès.
:::

:::info
Pour plus d'informations sur la façon dont vos données sont protégées, voir [Sécurité des données](./data-security.md).
:::
