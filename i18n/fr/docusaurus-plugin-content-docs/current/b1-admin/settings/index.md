# Paramètres

<div class="article-intro">

La section Paramètres est l'endroit où vous configurez le compte, l'image de marque et les options administratives de votre église dans B1 Admin. De là, vous pouvez gérer tout, du nom de votre église et de votre sous-domaine aux autorisations d'utilisateur et à la configuration de l'application mobile.

</div>

## Ce que vous trouverez ici

La zone des paramètres est organisée en sous-sections suivantes accessibles à partir de la barre latérale :

1. **Paramètres** -- Configurez les informations de base, l'image de marque et le sous-domaine de votre église. Consultez [Paramètres de l'église](./church-settings.md) pour plus de détails.
2. **Sites** -- Ajoutez et gérez plusieurs emplacements physiques pour votre église. Une fois créés, les sites apparaissent sur les profils de personnes, dans la configuration de la présence et dans le tableau de bord de démographie. Consultez [Sites](./campuses.md) pour plus de détails.
3. **Champs personnalisés** -- Définissez vos propres champs à suivre sur les personnes (une date, un nombre, une réponse oui/non ou une liste de sélection), puis remplissez-les sur les profils et recherchez-les. Consultez [Champs personnalisés](./custom-fields.md) pour plus de détails.
3. **Applications mobiles** -- Configurez et personnalisez les onglets de navigation qui apparaissent dans la [PWA B1.church](/docs/b1-church/getting-started/installing-pwa) pour vos membres (les mêmes onglets sont également rendus par l'application native B1 Mobile dépréciée). Consultez [Paramètres de l'application mobile](./mobile-app.md) pour plus de détails.
4. **Administrateur serveur** -- Accédez aux outils d'administration avancés pour gérer les paramètres au niveau du serveur de votre église.
5. **Formulaires** -- Créer et gérer des formulaires personnalisés pour collecter des informations de votre congrégation.

## La page des paramètres principaux

Quand vous ouvrez d'abord la page Paramètres, vous verrez le **nom de votre église** et votre **sous-domaine** affichés en haut. L'en-tête fournit des boutons d'accès rapide pour les tâches courantes :

- **Modifier les paramètres** -- Mettez à jour le nom, l'adresse, les coordonnées et l'image de marque de votre église.
- **Applications mobiles** -- Allez directement à la configuration de l'application mobile.
- **Rôles** -- Gérer les rôles des utilisateurs et les autorisations de votre équipe.
- **Importer/Exporter** -- Transférez les données entre les systèmes à l'aide de l'outil d'importation/exportation.

:::tip
Commencez par configurer le nom de votre église et l'image de marque sous **Modifier les paramètres**, puis configurez les [Rôles](./roles-permissions.md) pour inviter les membres de votre équipe avec le bon niveau d'accès.
:::

:::info
Pour plus d'informations sur la façon dont vos données sont protégées, consultez [Sécurité des données](./data-security.md).
:::
