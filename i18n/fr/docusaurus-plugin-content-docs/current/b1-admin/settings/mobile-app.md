# Paramètres de l'application mobile

<div class="article-intro">

La page Paramètres de l'application mobile vous permet de configurer les onglets de navigation qui apparaissent dans l'application B1 Mobile pour les membres de votre église. Vous contrôlez quels onglets sont visibles, vers quoi ils renvoient et comment ils sont affichés.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous devez disposer de l'autorisation « Modifier les paramètres de l'église ». Consultez [Rôles et autorisations](./roles-permissions.md) si vous n'avez pas accès.
- Configurez d'abord vos [Paramètres de l'église](./church-settings.md), y compris le nom et l'image de marque de votre église

</div>

## Accéder aux paramètres de l'application mobile

1. Accédez à la page **Paramètres** dans la barre latérale gauche.
2. Cliquez sur le bouton **Applications mobiles** dans l'en-tête.
3. La page Paramètres de l'application mobile affiche vos onglets actuels.

## Ajouter un nouvel onglet

1. Cliquez sur le bouton **Ajouter un onglet** en haut de la page.
2. Remplissez les détails de l'onglet :
   - **Nom** -- Le libellé qui apparaît sur l'onglet (par exemple, « Prédications » ou « Donner »).
   - **Icône** -- Cliquez sur le sélecteur d'icônes pour choisir une icône pour votre onglet. Vous pouvez également télécharger une image personnalisée.
   - **Type d'onglet** -- Sélectionnez parmi les options telles que Bible, Diffusion en direct, Don, Site web, et plus encore.
   - **URL** -- Saisissez l'adresse web vers laquelle l'onglet doit renvoyer.
   - **Visibilité** -- Contrôlez qui peut voir cet onglet (tout le monde, membres uniquement, etc.).
3. Cliquez sur **Enregistrer l'onglet** pour l'ajouter à votre application.

## Modifier un onglet existant

1. Cliquez sur un onglet existant dans la liste des **Onglets de l'application**.
2. Mettez à jour le nom, l'icône, l'URL, le type ou les paramètres de visibilité de l'onglet.
3. Cliquez sur **Enregistrer l'onglet** pour appliquer vos modifications.

## Réorganiser les onglets

Vous pouvez modifier l'ordre dans lequel les onglets apparaissent dans l'application mobile. Faites glisser et déposez les onglets dans la liste pour les réorganiser. L'ordre affiché sur cette page correspond à l'ordre que vos membres verront dans l'application.

:::info
Certains onglets peuvent apparaître automatiquement lorsque certaines conditions sont remplies -- par exemple, un onglet Diffusion en direct peut s'afficher lorsqu'une diffusion est active. Les onglets ajoutés manuellement vous donnent un contrôle total sur ce que vos membres voient en permanence.
:::

:::tip
Limitez le nombre d'onglets à un niveau raisonnable. Trois à cinq onglets conviennent à la plupart des églises. Trop d'onglets peuvent rendre la navigation confuse pour vos membres.
:::

## Où ces onglets apparaissent

Les onglets que vous configurez ici sont affichés dans l'**application B1 Mobile** (application B1.church) que les membres de votre église téléchargent depuis l'App Store ou Google Play. Les modifications que vous apportez sur cette page sont reflétées dans l'application la prochaine fois qu'un membre l'ouvre.
