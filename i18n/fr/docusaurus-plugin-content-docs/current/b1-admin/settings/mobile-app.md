# Paramètres de l'application mobile

<div class="article-intro">

La page Paramètres de l'application mobile vous permet de configurer les onglets de navigation qui apparaissent dans l'**expérience mobile B1.church (PWA)** pour les membres de votre église. Vous contrôlez les onglets visibles, ce à quoi ils se connectent et comment ils sont affichés.

</div>

:::info L'application native B1 Mobile est obsolète
Les onglets configurés ici sont fournis via l'[application Web progressive B1.church (PWA)](/docs/b1-church/getting-started/installing-pwa), qui a remplacé l'application native B1 Mobile. Les membres peuvent installer la PWA sur leur écran d'accueil de n'importe quelle page sur `https://yourchurchname.b1.church` au lieu de télécharger l'application héritée depuis l'App Store ou Google Play.
:::

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin de la permission "Edit Church Settings". Consultez [Roles & Permissions](./roles-permissions.md) si vous n'avez pas accès.
- Configurez d'abord vos [Church Settings](./church-settings.md), y compris le nom de votre église et votre marque

</div>

## Accès aux paramètres de l'application mobile

1. Accédez à la page **Settings** dans la barre latérale de gauche.
2. Cliquez sur le bouton **Mobile Apps** dans l'en-tête.
3. La page Paramètres de l'application mobile affiche vos onglets actuels.

## Ajout d'un nouvel onglet

1. Cliquez sur le bouton **Add Tab** en haut de la page.
2. Remplissez les détails de l'onglet :
   - **Name** -- L'étiquette qui apparaît sur l'onglet (par exemple, "Sermons" ou "Give").
   - **Icon** -- Cliquez sur le sélecteur d'icône pour choisir une icône pour votre onglet. Vous pouvez également télécharger une image personnalisée.
   - **Tab Type** -- Sélectionnez parmi les options telles que Bible, Live Stream, Donation, Website, et plus.
   - **URL** -- Entrez l'adresse Web à laquelle l'onglet doit être lié.
   - **Visibility** -- Contrôlez qui peut voir cet onglet (tout le monde, membres uniquement, etc.).
3. Cliquez sur **Save Tab** pour l'ajouter à votre application.

## Édition d'un onglet existant

1. Cliquez sur n'importe quel onglet existant dans la liste **App Tabs**.
2. Mettez à jour le nom, l'icône, l'URL, le type ou les paramètres de visibilité de l'onglet.
3. Cliquez sur **Save Tab** pour appliquer vos modifications.

## Réorganisation des onglets

Vous pouvez modifier l'ordre dans lequel les onglets apparaissent dans l'application mobile. Faites glisser et déposez les onglets dans la liste pour les réorganiser. L'ordre affiché sur cette page correspond à l'ordre que vos membres verront dans l'application.

:::info
Certains onglets peuvent apparaître automatiquement lorsque certaines conditions sont remplies -- par exemple, un onglet Live Stream peut apparaître quand un flux est actif. Les onglets ajoutés manuellement vous donnent un contrôle total sur ce que vos membres voient à tout moment.
:::

:::tip
Gardez le nombre d'onglets gérable. Trois à cinq onglets fonctionnent bien pour la plupart des églises. Trop d'onglets peuvent rendre la navigation confuse pour vos membres.
:::

## Où ces onglets apparaissent

Les onglets que vous configurez ici sont affichés dans la **PWA B1.church** que vos membres installent de n'importe quelle page sur `https://yourchurchname.b1.church`. Les modifications que vous apportez sur cette page sont reflétées la prochaine fois qu'un membre ouvre l'application. (Les onglets sont également rendus par l'[application native B1 Mobile](/docs/b1-mobile/) héritée pour tout membre qui l'exécute encore, mais cette application est obsolète et n'est plus mise à jour.)
