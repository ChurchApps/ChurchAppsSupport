# Connexion d'un téléviseur de salle de classe à vos leçons B1 Admin

<div class="article-intro">

Vous pouvez connecter un téléviseur de salle de classe à vos leçons B1 Admin en utilisant **FreePlay** — l'application télévisée gratuite de ChurchApps pour les églises. Une fois connectée, la télévision affiche le contenu de la leçon que vous avez planifiée dans B1 Admin, le mettant à disposition de vos enseignants directement sur l'écran de la salle de classe.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Planifiez au moins une leçon dans B1 Admin (voir [Planification des leçons](./scheduling-lessons.md))
- Installez **FreePlay** sur le téléviseur ou Fire Stick que vous souhaitez utiliser

</div>

## Étape 1 — Installation de FreePlay sur votre téléviseur

Installez l'application **FreePlay** sur votre affichage de salle de classe :

- **Amazon Fire Stick / Fire TV** — Recherchez **FreePlay** dans l'Amazon Appstore
- **Android TV / Google TV** — Recherchez **FreePlay** dans le Google Play Store
- **Apple TV** — Recherchez **FreePlay** dans l'App Store

Pour les instructions de configuration complètes, consultez le [guide de démarrage FreePlay](/docs/freeplay/getting-started/).

## Étape 2 — Connectez B1.church en tant que fournisseur de contenu

1. Ouvrez l'application **FreePlay** sur la télévision
2. Accédez à l'écran **Fournisseurs de contenu** (sélectionnez l'icône **Fournisseurs** dans la barre latérale)
3. Trouvez et sélectionnez **B1.church** dans la liste des fournisseurs disponibles
4. Suivez les invites à l'écran pour vous connecter avec vos identifiants d'église B1
5. FreePlay affiche **Connecté !** une fois la connexion établie avec succès

Une fois connecté, B1.church apparaît dans votre barre latérale FreePlay. FreePlay extraira le contenu de la leçon que vous avez planifiée dans B1 Admin afin que vos enseignants puissent le lire directement sur le téléviseur de la salle de classe.

:::tip
Après la connexion, vos enseignants peuvent utiliser la télécommande du téléviseur pour avancer dans les vidéos et les diapositives à leur rythme à mesure que la leçon se déroule. Aucune configuration séparée ou clé USB nécessaire.
:::

:::info
Pour connaître les étapes détaillées d'authentification auprès d'un fournisseur de contenu (code QR, connexion par formulaire, etc.), consultez [Connexion aux fournisseurs](/docs/freeplay/content-providers/connecting-providers).
:::
