# Rechercher des Gens

<div class="article-intro">

La page **Gens** affiche votre répertoire d'église dans un tableau consultable et triable. Vous pouvez trouver rapidement n'importe qui dans votre congrégation, personnaliser les informations affichées et exporter vos résultats. Une recherche efficace est essentielle pour les tâches quotidiennes de gestion d'église comme le suivi des visiteurs, la préparation des listes de contacts et la gestion des dossiers des membres.

</div>

<div class="prereqs">
<h4>Avant de Commencer</h4>

- Vous avez besoin d'un compte B1 Admin actif avec la permission d'afficher les gens. Voir [Rôles et Permissions](roles-permissions.md) si vous ne savez pas quel est votre niveau d'accès.
- Votre répertoire d'église devrait avoir des gens dedans. Si vous n'avez pas encore ajouté quelqu'un, voir [Ajouter des Gens](adding-people.md) ou [Importer les Données](importing-data.md).

</div>

## Recherche Rapide

La barre de recherche en haut de la page Gens vous permet de trouver les membres en temps réel :

1. Cliquez sur la **boîte de recherche** en haut de la page Gens.
2. Commencez à taper un nom, un email ou autre mot-clé.
3. Les résultats se filtreront automatiquement au fur et à mesure que vous tapez (il y a un délai d'environ une demi-seconde pour que la recherche ne s'exécute pas à chaque caractère).
4. Le tableau ci-dessous se met à jour pour afficher uniquement les résultats correspondants.

:::tip
Vous n'avez pas besoin d'appuyer sur Entrée. La recherche s'exécute automatiquement après que vous arrêtiez de taper.
:::

## Tri des Résultats

Vous pouvez trier le répertoire en cliquant sur n'importe quel en-tête de colonne du tableau :

1. Cliquez sur un **en-tête de colonne** (par exemple, **Nom** ou **Email**) pour trier par cette colonne.
2. Cliquez sur le même en-tête à nouveau pour inverser l'ordre de tri.

Cela facilite la recherche de personnes par ordre alphabétique, par âge ou par n'importe quelle autre colonne visible.

## Personnaliser les Colonnes

Pas chaque information n'a besoin d'être visible à la fois. Vous pouvez choisir quelles colonnes apparaissent dans le tableau :

1. Recherchez le **menu déroulant du sélecteur de colonne** près du haut du tableau.
2. Cochez ou décochez les colonnes pour les afficher ou les masquer. Les colonnes disponibles incluent :
   - **Photo**
   - **Nom**
   - **Email**
   - **Téléphone**
   - **Adresse**
   - **Date de Naissance**
   - **Âge**
   - **Sexe**
   - **Statut d'Adhésion**
3. Le tableau se met à jour immédiatement pour refléter vos sélections.

:::info
Vos choix de colonnes affectent ce qui est inclus lorsque vous exportez vers CSV. Personnalisez les colonnes avant d'exporter pour obtenir exactement les données dont vous avez besoin.
:::

## Pagination

Lorsque votre répertoire a beaucoup d'enregistrements, les résultats sont répartis sur plusieurs pages. Utilisez les **contrôles de pagination** en bas du tableau pour passer d'une page à l'autre. La page actuelle et le nombre total d'enregistrements sont affichés afin que vous sachiez toujours où vous êtes dans la liste.

:::tip
Si vous voulez voir plus de résultats à la fois, affinez votre recherche pour réduire la liste plutôt que de passer d'une page à l'autre dans un grand répertoire.
:::

## Exporter les Résultats de Recherche

Vous pouvez télécharger vos résultats de recherche actuels en fichier CSV à tout moment :

1. Appliquez la recherche ou les filtres que vous souhaitez.
2. Personnalisez vos colonnes pour inclure les données dont vous avez besoin.
3. Cliquez sur le bouton **Exporter**.
4. Un fichier CSV téléchargera sur votre ordinateur, prêt à être ouvert dans Excel, Google Sheets ou toute application de feuille de calcul.

Pour plus de détails sur l'exportation, voir [Exporter les Données](./exporting-data.md).

:::tip
Pour les requêtes plus avancées -- comme trouver tous ceux qui n'ont pas assisté au cours des trois derniers mois -- essayez la fonctionnalité [Recherche IA](./ai-search.md), qui vous permet de rechercher en utilisant des questions en langage simple.
:::
