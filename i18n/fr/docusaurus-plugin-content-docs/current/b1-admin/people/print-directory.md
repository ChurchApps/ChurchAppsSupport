# Imprimer le Répertoire

<div class="article-intro">

La fonctionnalité Imprimer le Répertoire génère un répertoire de membres formaté et imprimable pour votre église. Il organise les membres par ménage, inclut les informations de contact et les photos, et est conçu pour avoir l'air professionnel lorsqu'il est imprimé sur du papier lettre standard.

</div>

<div class="prereqs">
<h4>Avant de Commencer</h4>

- Vous avez besoin d'un compte B1 Admin actif avec la permission d'afficher les gens.
- Les gens dans votre répertoire devraient avoir des photos et des informations de contact remplies pour les meilleurs résultats.

</div>

## Ouvrir le Répertoire d'Impression

1. Allez à **Gens** dans B1 Admin.
2. Cliquez sur l'**icône d'impression** (icône d'imprimante) dans la barre d'outils en haut de la carte des résultats Gens. Il apparaît à côté du bouton Exporter.
3. Le Répertoire d'Impression s'ouvre dans un nouvel onglet du navigateur et commence à charger les données de votre répertoire.
4. Après environ 1,5 secondes, la boîte de dialogue d'impression de votre navigateur apparaîtra automatiquement.

:::tip
Le répertoire s'ouvre dans un onglet séparé afin que vous puissiez l'imprimer sans quitter la page Gens. Après l'apparition de la boîte de dialogue d'impression, l'onglet revient automatiquement en arrière.
:::

## Ce Qui Est Inclus

Le répertoire d'impression inclut tous les **membres actifs** de votre église. Spécifiquement :

- Les personnes ayant un statut d'adhésion **Inactif** ou **Visiteur** sont automatiquement exclues.
- Les gens qui ont **opté hors du** répertoire sont exclus.
- Tous les autres sont groupés par **ménage**.

## Disposition du Répertoire

Le répertoire imprimé a deux parties :

### Page de Couverture

La première page inclut :
- Votre nom d'église
- Le titre « Répertoire des Membres »
- L'année en cours
- L'emplacement de votre église
- Un décompte du nombre total de ménages inclus

### Listes des Membres

Les pages restantes énumèrent les ménages par ordre alphabétique par nom de famille. Chaque carte de ménage affiche :

- **Photos** -- Vignettes circulaires pour chaque membre du ménage, avec son nom ci-dessous
- **Nom d'affichage du ménage** -- Par exemple, « La Famille Smith », « John et Jane Doe », ou juste le nom de la personne pour les individus
- **Adresse** -- Rue, ville, état et code postal
- **Téléphone** -- Numéro de maison ou mobile
- **Email** -- Adresse email principale
- **Anniversaires** -- Formatés comme des dates individuelles pour chaque membre du ménage
- **Anniversaire** -- S'il est enregistré

Les ménages sont organisés en **sections alphabétiques** (A, B, C, ...) avec un en-tête de section décoratif avant chaque groupe de lettres. Deux cartes de ménage apparaissent côte à côte sur chaque ligne.

## Paramètres d'Impression

Le répertoire est formaté pour **papier lettre** (8,5 « × 11 ») avec des marges de 0,5 pouce. Lorsque la boîte de dialogue d'impression du navigateur apparaît :

- Laissez le format du papier défini sur **Lettre**
- Activez **Graphiques en arrière-plan** (parfois appelé « Imprimer les arrière-plans ») dans les paramètres d'impression de votre navigateur si vous souhaitez que les en-têtes de section s'impriment avec leur style de couleur
- L'orientation portrait est recommandée

:::info
Le répertoire respecte les sauts de page -- les cartes de ménage et les en-têtes de section ne seront pas divisés entre les pages.
:::
