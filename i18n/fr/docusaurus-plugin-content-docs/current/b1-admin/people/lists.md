# Listes enregistrées

<div class="article-intro">

Les **listes enregistrées** vous permettent de stocker une requête de recherche sous un nom et de la réexécuter en un clic à tout moment. Construisez une liste une fois -- « Membres actifs au campus Nord », « Visiteurs au cours des 90 derniers jours » ou tout filtre avancé -- et elle reste dans votre barre latérale afin que vous n'ayez jamais à reconstruire la requête.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Exécutez au moins une recherche dans la section Personnes afin d'avoir des résultats à enregistrer. Consultez [Recherche de personnes](./searching-people.md) ou [Recherche AI](./ai-search.md).

</div>

## Enregistrement d'une recherche en tant que liste

1. Exécutez une recherche sur la page **Personnes** -- une recherche rapide, une recherche AI ou une recherche de filtre avancé.
2. Quand les résultats apparaissent, un bouton **Enregistrer comme liste** (icône signet) s'affiche dans l'en-tête de la page Personnes.
3. Cliquez sur **Enregistrer comme liste**.
4. Entrez un **Nom** pour la liste (obligatoire).
5. Éventuellement, entrez une **Catégorie** pour regrouper les listes connexes dans la barre latérale (par exemple, « Sensibilisation » ou « Adhésion »).
6. Cliquez sur **Enregistrer**.

La liste est maintenant enregistrée dans le compte de votre église et apparaîtra dans le panneau **Listes enregistrées** sur le côté gauche de la page Personnes.

:::info
Les listes sont partagées dans votre église -- toute personne ayant accès aux personnes peut les voir et les charger. Seul le personnel ayant la permission appropriée peut renommer ou supprimer les listes.
:::

## Chargement d'une liste enregistrée

Dans le panneau **Listes enregistrées** sur le côté gauche de la page Personnes, cliquez sur n'importe quel nom de liste. La recherche se réexécute instantanément avec la requête stockée, mettant à jour la table des résultats.

Les listes sont groupées par catégorie (si vous en avez attribué une), les listes non catégorisées apparaissant en dernier.

:::tip
Les listes sont des **requêtes actives**, pas des instantanés. Chaque fois que vous chargez une liste, la recherche s'exécute de manière nouvelle contre vos données actuelles, de sorte que les nouvelles personnes qui correspondent maintenant aux critères apparaîtront.
:::

## Renommage et suppression de listes

Le personnel ayant la permission de gestion peut renommer ou supprimer n'importe quelle liste directement à partir du panneau Listes enregistrées.

- Cliquez sur l'**icône crayon** à côté d'un nom de liste pour le renommer.
- Cliquez sur l'**icône poubelle** pour le supprimer (vous serez invité à confirmer).

## Cas d'utilisation

| Scénario | Comment construire la liste |
|---|---|
| Tous les visiteurs | Recherche avancée : Statut d'adhésion = Visiteur |
| Membres à un campus spécifique | Recherche avancée : Campus = [votre campus] |
| Personnes sans adresse e-mail | Recherche avancée : L'e-mail est vide |
| Bénévoles avec une vérification des antécédents expirée | Recherche avancée : [Champ personnalisé](../settings/custom-fields.md) « La vérification des antécédents expire » est avant aujourd'hui |
| Résultats d'une question AI | Posez une question dans la recherche AI, puis enregistrez |
