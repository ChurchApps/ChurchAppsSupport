# Playlists

<div class="article-intro">

Les playlists vous permettent d'organiser vos prédications en séries ou collections. Lorsque les visiteurs parcourent votre bibliothèque de prédications sur votre site B1.church, les playlists les aident à trouver et à suivre une série complète de prédications dans le bon ordre.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin de la permission **contentApi.streamingServices.edit**. Consultez [Rôles et permissions](../settings/roles-permissions.md) si vous n'avez pas accès.
- Ayez un nom, une description et une image miniature prêts pour votre playlist

</div>

## Consulter les playlists

1. Dans B1 Admin, cliquez sur **Prédications** dans la barre latérale gauche.
2. Cliquez sur l'onglet **Playlists** en haut de la page.
3. Vous verrez une liste de toutes vos playlists existantes avec leurs noms et descriptions.

## Créer une playlist

1. Cliquez sur le bouton **Créer la première playlist** (si vous n'avez pas encore de playlists) ou cliquez sur **Ajouter une playlist**.
2. Saisissez un **nom** pour la playlist -- par exemple, le titre de la série de prédications comme « Fondements de la foi » ou « Un été dans les Psaumes ».
3. Ajoutez une **description** pour donner aux visiteurs un bref aperçu de la série.
4. Définissez une **date de publication** pour contrôler quand la playlist devient visible.
5. Téléversez une **image miniature** pour représenter visuellement la série.
6. Cliquez sur **Enregistrer** pour créer votre playlist.

:::tip
Créez vos playlists avant d'ajouter des prédications. De cette façon, vous pouvez affecter chaque prédication à la bonne playlist au fur et à mesure, plutôt que de réorganiser plus tard.
:::

## Ajouter des prédications à une playlist

Lorsque vous [ajoutez ou modifiez une prédication](managing-sermons), vous l'affectez à une playlist à l'aide du menu déroulant **Playlist**. Toutes les prédications affectées à la même playlist sont regroupées et affichées dans l'ordre sur votre site web.

:::info
Vous pouvez également affecter des prédications à des playlists lors de l'[Import en masse](bulk-import). Lors de l'importation depuis YouTube ou Vimeo, sélectionnez une playlist dans le menu déroulant **Importer dans la playlist** avant de cliquer sur **Importer**.
:::

## Comment les playlists apparaissent sur votre site web

Les playlists sont affichées sur votre site B1.church sous forme de collections navigables. Les visiteurs peuvent :

- Voir toutes les séries de prédications disponibles en un coup d'œil
- Cliquer sur une playlist pour voir toutes les prédications de cette série
- Regarder ou écouter les prédications dans l'ordre
