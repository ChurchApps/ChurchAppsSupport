# Import en masse

<div class="article-intro">

La page Import en masse vous permet de remplir rapidement votre bibliothèque de prédications en important des vidéos depuis YouTube ou Vimeo. C'est le moyen le plus rapide d'intégrer votre contenu de prédications existant plutôt que d'ajouter chaque vidéo une par une.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous avez besoin de la permission **contentApi.streamingServices.edit**. Consultez [Rôles et permissions](../settings/roles-permissions.md) si vous n'avez pas accès.
- Créez au moins une [playlist](playlists) dans laquelle importer vos prédications
- Ayez votre identifiant de chaîne YouTube ou vos informations de compte Vimeo à portée de main

</div>

## Choisir votre source

1. Dans B1 Admin, cliquez sur **Prédications** dans la barre latérale gauche, puis cliquez sur l'onglet **Import en masse**.
2. Vous verrez deux cartes cliquables :
   - **YouTube** (rouge) -- Importer des vidéos depuis une chaîne YouTube
   - **Vimeo** (bleu) -- Importer des vidéos depuis un compte Vimeo
3. Cliquez sur la carte correspondant à la plateforme où vos prédications sont hébergées.

:::tip
Vous pouvez revenir en arrière et changer de source à tout moment si vous avez des vidéos sur les deux plateformes.
:::

## Avant d'importer

Avant d'importer des vidéos, vous avez besoin d'au moins une playlist pour les organiser. Si vous n'avez pas encore créé de playlists :

1. Cliquez sur l'onglet **Playlists**.
2. Cliquez sur **Créer la première playlist** et remplissez le nom, la description, la date de publication et la miniature.
3. Cliquez sur **Enregistrer**, puis retournez à l'onglet **Import en masse**.

Consultez [Playlists](playlists) pour des instructions détaillées sur la création et la gestion des playlists.

## Importer des vidéos

1. Après avoir sélectionné YouTube ou Vimeo, saisissez votre **identifiant de chaîne YouTube** ou vos **informations de compte Vimeo** dans le champ prévu.
2. Cliquez sur le bouton **Récupérer** pour obtenir toutes les vidéos disponibles de votre chaîne ou compte.
3. Après la récupération, vous verrez une liste de toutes vos vidéos avec des cases à cocher.
4. Cochez les cases à côté des vidéos que vous souhaitez importer. Vous pouvez tout sélectionner ou en choisir des spécifiques.
5. Facultativement, activez **Importer automatiquement les nouvelles vidéos** pour ajouter automatiquement les futurs téléversements à votre bibliothèque.
6. Cliquez sur le menu déroulant **Importer dans la playlist** pour choisir dans quelle playlist ces vidéos doivent être ajoutées.
7. Cliquez sur le bouton **Importer** pour finaliser l'import en masse.

Vos vidéos seront importées avec tous leurs détails, y compris les titres, descriptions, dates et miniatures.

:::info
L'import en masse est idéal pour démarrer lorsque vous migrez depuis une autre plateforme. Pour ajouter des prédications individuelles par la suite, utilisez plutôt la page [Gérer les prédications](managing-sermons).
:::
