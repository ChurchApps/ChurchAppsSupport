# Formulaires

<div class="article-intro">

La section Formulaires vous permet de créer des formulaires personnalisés pour les inscriptions, les sondages, la collecte de données et bien plus encore. Que vous ayez besoin d'une fiche visiteur, d'une inscription à un événement ou d'un formulaire de demande de prière, vous pouvez le créer ici et le partager avec votre communauté via un lien public ou votre site web B1.church.

</div>

## La page Formulaires

Lorsque vous ouvrez **Formulaires**, vous verrez deux onglets :

- **Formulaires** -- Affiche tous vos formulaires actifs.
- **Archivés** -- Affiche les formulaires qui ont été archivés. Vous pouvez restaurer les formulaires archivés à tout moment.

Chaque formulaire de la liste affiche son nom et, s'il est configuré, une URL publique que vous pouvez partager à l'extérieur.

## Ce que vous pouvez faire

Depuis la page Formulaires, vous pouvez :

- **Ajouter un formulaire** -- Cliquez sur **Ajouter un formulaire** pour créer un nouveau formulaire personnalisé.
- **Modifier un formulaire** -- Cliquez sur le nom d'un formulaire pour l'ouvrir et modifier ses questions, ses membres ou ses paramètres.
- **Archiver un formulaire** -- Déplacez les formulaires dont vous n'avez plus besoin vers l'onglet **Archivés** pour garder votre liste propre.
- **Restaurer un formulaire** -- Passez à l'onglet **Archivés** et restaurez tout formulaire que vous souhaitez rendre à nouveau actif.

## Travailler avec un formulaire

Cliquez sur le nom d'un formulaire pour l'ouvrir. L'éditeur de formulaire comporte trois onglets :

- **Questions** -- Construisez la structure du formulaire en ajoutant des champs de différents types. Consultez [Créer des formulaires](./creating-forms.md) pour plus de détails.
- **Membres** -- Contrôlez qui peut accéder au formulaire et le gérer.
- **Soumissions** -- Consultez les réponses qui ont été soumises. Consultez [Gérer les soumissions](./managing-submissions.md) pour plus de détails.

:::tip
Les formulaires avec une URL publique peuvent être partagés par lien, intégrés à votre site web ou inclus dans des e-mails. Cela facilite la collecte d'informations auprès de personnes qui n'ont pas forcément de compte dans votre église.
:::

:::info
Vous pouvez contrôler qui a accès à chaque formulaire en utilisant l'onglet **Membres**. Attribuez les rôles **Administrateur** ou **Lecture seule** pour donner à votre équipe le bon niveau d'accès.
:::
