# Dons

<div class="article-intro">

La section Dons dans B1 Admin fournit tout ce dont vous avez besoin pour gérer les offrandes de votre église. De l'enregistrement des offrandes hebdomadaires à la génération des reçus fiscaux de fin d'année, tous vos outils de gestion des dons sont organisés en un seul endroit afin que vous puissiez garder le contrôle des finances de votre église.

</div>

## Sous-sections

Vous trouverez les pages suivantes dans la barre latérale Dons :

### Résumé

La page **Résumé** vous offre un aperçu visuel de l'ensemble de l'activité de dons. Elle affiche des graphiques et des rapports filtrés pour que vous puissiez rapidement voir les tendances des dons dans votre église. Consultez [Rapports de dons](donation-reports.md) pour plus de détails sur l'utilisation des outils de résumé et de rapports.

### Lots

Les **[Lots](batches.md)** vous permettent de regrouper les dons -- par exemple, tous les chèques et espèces collectés lors d'une offrande du dimanche. Chaque lot suit le nombre de dons et le montant total pour un rapprochement facile.

### Fonds

Les **[Fonds](funds.md)** catégorisent vos dons par objectif. Les exemples courants incluent le Fonds général, le Fonds de construction et les Missions. Vous pouvez marquer chaque fonds comme déductible d'impôts ou non.

### Relevés

La page **[Relevés](giving-statements.md)** génère les reçus fiscaux de fin d'année pour vos donateurs. Vous pouvez télécharger des fichiers CSV individuels ou imprimer tous les relevés en une seule fois.

### Import Stripe

Si vous acceptez les dons en ligne via Stripe, l'outil **[Import Stripe](stripe-import.md)** vous permet d'importer ces transactions dans B1 Admin afin que tout soit dans un seul système.

## Dons en ligne

B1 Admin s'intègre avec **Stripe** et **PayPal** pour accepter les dons en ligne. Une fois configuré, les membres peuvent donner via votre site B1.church et ces dons apparaissent automatiquement dans vos registres. Consultez [Configuration des dons en ligne](online-giving-setup.md) pour commencer.

:::tip
Si vous débutez avec les dons dans B1 Admin, commencez par [configurer vos fonds](funds.md), puis [créez votre premier lot](batches.md) et [enregistrez des dons](recording-donations.md) dedans.
:::
