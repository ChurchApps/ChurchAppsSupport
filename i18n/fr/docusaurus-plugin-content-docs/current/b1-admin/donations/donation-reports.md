# Rapports de dons

<div class="article-intro">

B1 Admin vous offre plusieurs façons de consulter et d'analyser les données de dons de votre église. La page Résumé des dons fournit un aperçu visuel avec des graphiques et des filtres, tandis que la section Rapports propose un rapport de synthèse des dons plus détaillé. Utilisez ces outils pour suivre les tendances de dons, préparer les réunions du conseil ou rapprocher vos registres.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Assurez-vous que les dons ont été [enregistrés dans des lots](recording-donations.md) ou [importés depuis Stripe](stripe-import.md)
- Vérifiez que vos [fonds](funds.md) sont correctement configurés afin que les dons soient correctement catégorisés

</div>

## Page de résumé des dons

La page **Résumé** est la première chose que vous voyez lorsque vous cliquez sur **Dons** dans la barre latérale. Elle fournit une vue d'ensemble de votre activité de dons.

1. Accédez à **Dons** dans la barre latérale pour ouvrir la page Résumé.
2. Utilisez le **filtre de plage de dates** pour sélectionner la période que vous souhaitez examiner. Placez la date la plus ancienne en haut et la date la plus récente en bas.
3. La page affiche un graphique des dons hebdomadaires pour que vous puissiez voir les tendances en un coup d'œil.
4. Cliquez sur **Télécharger** pour exporter un fichier CSV avec le montant total des dons, la semaine du don et le fonds concerné.

:::info
La page Résumé affiche les données de dons agrégées. Elle n'inclut pas les noms des donateurs individuels. Pour les détails au niveau des donateurs, utilisez la page [Lots](batches.md).
:::

## Consulter les détails par donateur

Pour obtenir le détail de qui a donné, combien et à quel fonds :

1. Accédez à **Dons > Lots**.
2. Cliquez sur un **nom de lot** pour l'ouvrir.
3. La page de détails du lot liste chaque don avec le nom du donateur, le montant, le fonds, la date et le mode de paiement.
4. Cliquez sur le **nom d'un donateur** pour voir le détail du nombre de dons effectués et du montant de chacun.
5. Cliquez sur un **identifiant de don** pour ouvrir un panneau latéral avec tous les détails de ce don individuel.
6. Cliquez sur **Télécharger** pour exporter un CSV avec toutes les informations sur les donateurs et les dons pour ce lot.

## Rapport de synthèse des dons

B1 Admin inclut également un rapport **Synthèse des dons** dans la section Rapports :

1. Cliquez sur **Rapports** dans la barre latérale.
2. Sélectionnez le rapport **Synthèse des dons**.
3. Choisissez vos filtres (plage de dates, fonds, campus, etc.) et lancez le rapport.

## Exporter les données

Vous pouvez exporter les données de dons depuis plusieurs endroits :

- **Page Résumé** -- télécharger un CSV des totaux hebdomadaires des dons par fonds
- **Page de détails du lot** -- télécharger un CSV des dons individuels avec les détails des donateurs
- **Page de détails du fonds** -- télécharger l'historique des dons pour un fonds spécifique

:::tip
Pour les rapports de fin d'année, combinez l'export de la page Résumé avec l'outil [Reçus fiscaux](giving-statements.md) pour obtenir à la fois les tendances agrégées et les relevés individuels des donateurs.
:::
