# Gestion des fonds

<div class="article-intro">

Les fonds vous permettent de catégoriser les dons par objet afin que vous puissiez suivre le montant que votre église reçoit pour chaque domaine du ministère. Les fonds courants incluent Fonds général, Fonds de bâtiment, Missions et Ministère jeunesse, mais vous pouvez en créer autant que vous le souhaitez.

</div>

<div class="prereqs">
<h4>Avant de commencer</h4>

- Vous aurez besoin d'un accès à la section **Dons** dans B1 Admin
- Décidez des catégories de dons dont votre église a besoin et si chacun doit être déductible des impôts

</div>

## La page des fonds

Allez à **Dons > Fonds** pour voir une liste de tous vos fonds. Chaque ligne affiche le nom du fonds et son statut de **déductibilité fiscale**, qui détermine si les dons à ce fonds apparaissent sur les [relevés de dons](giving-statements.md) de fin d'année.

## Création d'un nouveau fonds

1. Cliquez sur **Ajouter un fonds** en haut de la page.
2. Entrez un nom pour le fonds (par exemple, « Fonds de bâtiment »).
3. Cochez la case **Déductible des impôts** si les dons à ce fonds doivent être inclus dans les relevés de dons.
4. Laissez **Visible pour les donateurs** coché pour afficher le fonds sur les formulaires de don publics et membres, ou décochez-le pour masquer le fonds aux donateurs tout en l'utilisant pour les dons enregistrés manuellement.
5. Cliquez sur **Enregistrer**.

Le nouveau fonds apparaît immédiatement dans votre liste et est disponible lors de la saisie des dons.

## Modification et suppression de fonds

- Pour modifier un fonds, cliquez sur le bouton **Modifier** à côté. Vous pouvez modifier le nom, basculer le paramètre de déductibilité fiscale ou basculer **Visible pour les donateurs** pour l'afficher ou le masquer sur les formulaires de don.
- Pour supprimer un fonds, ouvrez le formulaire de modification et cliquez sur **Supprimer**.

:::warning
La suppression d'un fonds ne supprime pas les dons qui lui ont été précédemment attribués. Ces enregistrements de don restent dans le système, mais le fonds ne sera plus disponible pour les nouveaux dons.
:::

:::info
Les fonds masqués aux donateurs affichent une étiquette **Masqué** à côté de leur nom dans cette liste. Les fonds masqués fonctionnent toujours pour les dons enregistrés manuellement dans B1 Admin -- ils n'apparaissent simplement pas comme une option sur les formulaires de don publics ou membres.
:::

## Affichage des détails des fonds

Cliquez sur un **nom de fonds** pour ouvrir sa page de détail. Cette page affiche :

- **Historique des dons** -- une liste de tous les dons à ce fonds
- **Filtre de plage de dates** -- affiner les résultats à une période spécifique
- **Statistiques récapitulatives** -- total des dons, donateurs uniques et montant en dollars total

Cette vue est utile lorsque vous devez faire un rapport sur une initiative de dons spécifique ou vérifier les progrès vers un objectif de collecte de fonds.

## Tri et exportation

- Utilisez les en-têtes de colonne pour trier les fonds par nom.
- Cliquez sur **Exporter en CSV** pour télécharger votre liste de fonds sous forme de feuille de calcul.

:::tip
Configurez vos fonds avant de commencer à [enregistrer les dons](recording-donations.md). Avoir tous les fonds en place assure que les dons sont catégorisés correctement dès le départ.
:::

## Prochaines étapes

Une fois vos fonds configurés, vous pouvez les attribuer aux dons lors de l'[enregistrement des dons](recording-donations.md) en lot. Assurez-vous de marquer les fonds appropriés comme déductibles des impôts avant de générer les [relevés de dons](giving-statements.md).
