# कक्षा मोड

<div class="article-intro">

कक्षा मोड आपके टीवी को आपके चर्च के एक विशिष्ट कमरे से कनेक्ट करता है। एक बार पेयर होने के बाद, FreePlay स्वचालित रूप से उस कमरे के लिए निर्धारित पाठ डाउनलोड करता है, ताकि आपके स्वयंसेवक कक्षा शुरू होने पर बस **पाठ शुरू करें** दबा सकें।

</div>

## यह कैसे काम करता है

1. **अपना चर्च खोजें** - नाम से अपना चर्च खोजें
2. **कमरा चुनें** - वह कक्षा चुनें जिससे यह टीवी संबंधित है
3. **पाठ डाउनलोड करें** - FreePlay निर्धारित पाठ प्राप्त करता है और सभी मीडिया फ़ाइलें डाउनलोड करता है
4. **चलाएं** - **पाठ शुरू करें** बटन दबाएं और पाठ स्वचालित रूप से चलता है

टीवी याद रखता है कि वह किस चर्च और कमरे से पेयर है, इसलिए आपको सेटअप केवल एक बार करना होगा। प्रत्येक सप्ताह, FreePlay वर्तमान दिन के निर्धारित पाठ की जांच करता है और कोई भी नई सामग्री डाउनलोड करता है।

:::tip
अपने टीवी को पेयर करने से पहले सुनिश्चित करें कि आपकी कक्षाएं [freeplay.church](https://freeplay.church) पर कॉन्फ़िगर हैं। यदि आपका चर्च चुनने पर कोई कक्षा दिखाई नहीं देती, तो उन्हें सेट करने के लिए वेबसाइट पर जाएं।
:::
