# Plans Overview

<div class="article-intro">

Plans Overview आपको एक ही बार में multiple service dates में सभी volunteer assignments का bird's-eye view देता है। प्रत्येक plan को individually खोलने के बजाय, आप एक single grid में upcoming weeks में हर position में कौन serving कर रहा है यह देख सकते हैं - और जल्दी से gaps को spot कर सकते हैं जिन्हें भरने की अभी भी आवश्यकता है।

</div>

<div class="prereqs">
<h4>शुरुआत से पहले</h4>

- Serving area में कम से कम एक Ministry और Plan Type create करें
- [service plans](./plans.md) को dates और volunteer assignments के साथ create करें
- सुनिश्चित करें कि आपके volunteers को आपकी [people directory](../people/adding-people.md) में add किया गया है

</div>

## Overview को Access करना

1. B1 Admin में main menu से **Serving** पर navigate करें।
2. पृष्ठ के शीर्ष पर एक **ministry tab** को select करें।
3. इसकी plan list को खोलने के लिए एक **plan type** पर click करें।
4. पृष्ठ के near top में **Overview** button पर click करें।

## Overview Grid को पढ़ना

Overview एक grid display करता है जहां:

- **Rows** प्रत्येक position को represent करती हैं (उदा। "Music: Guitar", "Tech: Projection") category द्वारा grouped
- **Columns** upcoming service dates को represent करती हैं (उदा। "Apr 14", "Apr 21")
- **Cells** उस position में assigned volunteer का नाम दिखाती हैं उस date पर

**Red** में highlighted cells unfilled हैं - कोई volunteer अभी तक assign नहीं हुआ है। यह आपको प्रत्येक plan को individually खोले बिना staffing gaps को एक glance में देखना आसान बनाता है।

:::tip
Volunteer names एक shortened format में दिखाई देते हैं (first name और last initial, उदा। "John D.") जब आपके पास बहुत सारी positions हों तो grid को compact रखने के लिए।
:::

## Overview से सीधे Volunteers को Assign करना

Empty slots को भरने के लिए आपको individual plans को खोलने की आवश्यकता नहीं है। Grid में किसी भी cell पर click करें उस position और date के लिए एक assignment panel खोलने के लिए। वहां से आप:

- अपनी team से एक person को select करके position में assign करें
- Existing assignment को replace या remove करें
- Overview को छोड़े बिना change को save करें

यह एक ही pass में एक entire schedule को staff करना संभव बनाता है - weeks और positions में काम करें individual plans में navigate किए बिना।

## Overview को Filter करना

आप filter controls का use करके adjust कर सकते हैं कि overview क्या दिखाता है:

- **Start Date / End Date** — Default में overview 12 weeks को future में दिखाता है। Custom date range को expand या narrow करने के लिए enter करें।
- **Ministry** — Overview को छोड़े बिना एक अलग ministry पर switch करें।
- **Plan Type** — Selected ministry के भीतर एक specific plan type पर filter करें।
- **Gaps Only** — इसे on करें rows को hide करने के लिए जहां हर date already filled है, ताकि आप केवल positions पर focus कर सकें जिन्हें अभी भी volunteer की आवश्यकता है।

Grid को update करने के लिए changes करने के बाद **Filter** पर click करें।

## CSV में Export करना

Current grid को एक spreadsheet के रूप में download करने के लिए **Export CSV** पर click करें। Export में filtered date range के लिए सभी positions और volunteer assignments शामिल हैं, जिससे ministry leaders के साथ share करना या planning meetings के लिए print करना आसान है।

:::info
CSV export जो भी filters currently लागू हैं उन्हें reflect करता है - केवल dates और plan type जो grid में दिखाई देते हैं वे download में शामिल होते हैं।
:::
