# समूह स्वास्थ्य

<div class="article-intro">

समूह स्वास्थ्य डैशबोर्ड आपको अपने सभी समूहों कैसे कर रहे हैं का एक पक्षी की दृष्टि देता है -- सदस्यता रुझान, उपस्थिति औसत और विगत 90 दिनों में वृद्धि या नुकसान -- सभी एक एकल क्रमबद्ध तालिका में।

</div>

<div class="prereqs">

- सार्थक डेटा देखने के लिए आपके पास कम से कम कुछ सदस्य वाले समूह होने चाहिए। [समूह बनाना](creating-groups) देखें।
- उपस्थिति डेटा दर्ज किए गए सत्र से खींचा जाता है। [उपस्थिति](../attendance/index) अनुभाग देखें।

</div>

## समूह स्वास्थ्य खोलना

B1 Admin में, बाईं ओर साइडबार में **समूह** पर जाएँ और **स्वास्थ्य** चुनें। डैशबोर्ड एक तालिका एक पंक्ति प्रति समूह के साथ लोड करता है।

## स्तंभ

| स्तंभ | यह क्या दिखाता है |
|--------|--------------|
| **नाम** | समूह नाम, समूह विवरण पृष्ठ के लिए लिंक किया गया |
| **श्रेणी** | समूह की श्रेणी |
| **सदस्य** | वर्तमान सक्रिय सदस्य गणना |
| **शामिल (90d)** | सदस्य जो विगत 90 दिनों में शामिल हुए |
| **छोड़ा गया (90d)** | सदस्य जो विगत 90 दिनों में चले गए |
| **नुकसान (90d)** | 90 दिनों में एक प्रतिशत के रूप में शुद्ध नुकसान दर |
| **औसत उपस्थिति** | प्रति उपस्थिति सत्र औसत हेडकाउंट |

तालिका को उस स्तंभ द्वारा क्रमबद्ध करने के लिए किसी भी स्तंभ हेडर पर क्लिक करें। पुनः क्रम दिशा को उलट करने के लिए फिर से क्लिक करें।

## स्वास्थ्य डेटा का उपयोग

- **उच्च नुकसान + कम शामिल** -- एक समूह जो सिकुड़ रहा है और खोए हुए सदस्यों को प्रतिस्थापित नहीं कर रहा है। समूह नेता के साथ एक बातचीत के लायक।
- **उच्च शामिल + कम उपस्थिति** -- लोग साइन अप कर रहे हैं लेकिन दिखाई नहीं दे रहे हैं। जुड़ाव अनुवर्ती पर विचार करें।
- **उच्च औसत उपस्थिति** -- एक स्वस्थ, सक्रिय समूह। अन्य समूहों के लिए संभावित मॉडल।

:::tip
किसी समूह के नाम पर क्लिक करने से आप सीधे समूह के विवरण पृष्ठ पर जाते हैं जहाँ आप व्यक्तिगत सदस्यों, उपस्थिति रिकॉर्ड और कैलेंडर ईवेंट की समीक्षा कर सकते हैं।
:::
