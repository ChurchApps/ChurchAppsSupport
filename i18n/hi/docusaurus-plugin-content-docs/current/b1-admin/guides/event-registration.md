# इवेंट पंजीकरण सेट करें

<div class="article-intro">

एक इवेंट पंजीकरण फॉर्म बनाएं, प्रतिभागी जानकारी और वैकल्पिक भुगतान एकत्र करें, इसे अपनी चर्च वेबसाइट पर एम्बेड करें, और आने वाले सबमिशन को प्रबंधित करें। अंत में, आपके पास किसी भी चर्च इवेंट के लिए एक साझा करने योग्य पंजीकरण पृष्ठ होगा।

</div>

<div class="prereqs">
<h4>शुरू करने से पहले</h4>

- व्यवस्थापक (admin) पहुंच वाला B1 Admin खाता
- भुगतान एकत्र करने के लिए: पहले [Stripe कॉन्फ़िगर होना चाहिए](../donations/online-giving-setup.md)

</div>

## चरण 1: एक स्टैंड अलोन फॉर्म बनाएं

स्टैंड अलोन फॉर्म का अपना सार्वजनिक URL होता है जिसे कोई भी एक्सेस कर सकता है — इवेंट पंजीकरण के लिए बिल्कुल सही।

[फॉर्म बनाना](../forms/creating-forms.md) गाइड का पालन करें:

1. फॉर्म्स पर जाएं और फॉर्म जोड़ें पर क्लिक करें
2. "Stand Alone" प्रकार चुनें — इससे आपके फॉर्म को अपना सार्वजनिक URL मिलता है
3. इसे इवेंट के नाम पर रखें (जैसे, "पुरुषों का रिट्रीट पंजीकरण", "VBS साइन-अप")

## चरण 2: प्रश्न जोड़ें

पंजीकरणकर्ताओं से एकत्र करने के लिए आवश्यक फ़ील्ड बनाएं।

अपने प्रश्न जोड़ने के लिए [फॉर्म बनाना](../forms/creating-forms.md#adding-questions) गाइड का पालन करें:

1. प्रश्न टैब पर जाएं और आवश्यक जानकारी के लिए फ़ील्ड जोड़ें: नाम, ईमेल, फोन, आहार प्रतिबंध, टी-शर्ट साइज, आपातकालीन संपर्क, आदि।
2. भोजन प्राथमिकताओं या सत्र चयन जैसे विकल्पों के लिए बहुविकल्पी का उपयोग करें

:::warning
भुगतान फ़ील्ड प्रकार के लिए Stripe कॉन्फ़िगर होना आवश्यक है। यदि आपने अभी तक ऑनलाइन दान सेट नहीं किया है, तो भुगतान फ़ील्ड जोड़ने से पहले [ऑनलाइन दान सेटअप](../donations/online-giving-setup.md) देखें।
:::

## चरण 3: फॉर्म सेटिंग्स कॉन्फ़िगर करें

नियंत्रित करें कि आपका पंजीकरण फॉर्म कब और कैसे उपलब्ध है।

1. यदि पंजीकरण केवल सीमित समय के लिए खुला होना चाहिए तो उपलब्धता तिथियां सेट करें
2. सार्वजनिक URL कॉपी करें — आप इसे सीधे साझा कर सकते हैं
3. सबमिशन प्रबंधित करने में मदद के लिए Admin या View Only भूमिकाओं के साथ फॉर्म सदस्य जोड़ें

## चरण 4: अपनी वेबसाइट पर एम्बेड करें

पंजीकरण फॉर्म को अपनी चर्च वेबसाइट पर जोड़कर आसानी से खोजने योग्य बनाएं।

[पेज प्रबंधन](../website/managing-pages.md) गाइड का पालन करें:

1. अपने B1 वेबसाइट एडिटर में, एक पेज पर एक नया सेक्शन जोड़ें और फॉर्म एलीमेंट चुनें
2. सूची से अपना पंजीकरण फॉर्म चुनें

:::tip
ईमेल, सोशल मीडिया और चर्च बुलेटिन के माध्यम से भी स्टैंडअलोन URL साझा करें — जितनी अधिक जगहों पर यह दिखाई देगा, उतने अधिक साइनअप आपको मिलेंगे।
:::

## चरण 5: सबमिशन प्रबंधित करें

आने वाले पंजीकरणों को ट्रैक करें और जब जरूरत हो तब डेटा निर्यात करें।

[सबमिशन प्रबंधन](../forms/managing-submissions.md) गाइड का पालन करें:

1. सबमिशन टैब पर आने वाली प्रतिक्रियाओं की समीक्षा करें
2. स्प्रेडशीट, हेडकाउंट ट्रैकिंग, या इवेंट समन्वयकों के साथ साझा करने के लिए CSV में निर्यात करें

## आप तैयार हैं!

आपका इवेंट पंजीकरण लाइव है। लिंक साझा करें, इसे अपनी वेबसाइट पर एम्बेड करें, और B1 Admin से साइनअप ट्रैक करें। जब इवेंट समाप्त हो जाए, तो अपने रिकॉर्ड के लिए अंतिम सूची निर्यात करें।
