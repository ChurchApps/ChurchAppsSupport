# B1 Admin सारांश

<div class="article-intro">

B1 Admin आपका चर्च प्रबंधन डैशबोर्ड है। यह आपके चर्च के संचालन के हर पहलू को प्रबंधित करने के लिए उपकरण प्रदान करता है -- सदस्य निर्देशिका और दान से लेकर वेबसाइट निर्माण और लाइव स्ट्रीमिंग तक -- सब कुछ एक ही स्थान से।

</div>

## शुरुआत करना

1. अपना ब्राउज़र खोलें और [admin.b1.church](https://admin.b1.church) पर जाएं
2. अपने उपयोगकर्ता नाम और पासवर्ड के साथ साइन इन करें
3. आप **डैशबोर्ड** पर पहुंचेंगे, जो आपके चर्च डेटा का अवलोकन प्रदान करता है और लोगों को जोड़ने, समूह बनाने, दान रिकॉर्ड करने और अधिक जैसे सामान्य कार्यों के लिए **त्वरित कार्य** प्रदान करता है

:::tip
B1 Admin में नए हैं? वीडियो वॉकथ्रू के लिए [परिचय](./introduction) से शुरू करें, फिर अपनी चर्च की जानकारी कॉन्फ़िगर करने और अपनी टीम को आमंत्रित करने के लिए [सेटिंग्स](./settings/) पर जाएं।
:::

## मुख्य विशेषताएं

- **[लोग](./people/)** - अपनी चर्च निर्देशिका प्रबंधित करें, सदस्य जोड़ें, डेटा बल्क संपादित करें, परिवारों को ट्रैक करें
- **[समूह](./groups/)** - शामिल होने के अनुरोधों और सदस्य प्रबंधन के साथ चर्च समूह बनाएं और व्यवस्थित करें
- **[उपस्थिति](./attendance/)** - परिसर, सेवा समय सेट करें और उपस्थिति को ट्रैक करें
- **[दान](./donations/)** - कई मुद्राओं में दान रिकॉर्ड करें, फंड प्रबंधित करें, विवरण तैयार करें
- **[सेवा](./serving/)** - स्वयंसेवकों का समन्वय करें, सेवा योजनाएं बनाएं, कार्य प्रबंधित करें
- **[फॉर्म](./forms/)** - पंजीकरण और डेटा संग्रह के लिए कस्टम फॉर्म बनाएं
- **[रिपोर्ट](./reports/)** - जन्मदिन, उपस्थिति और दान रिपोर्ट देखें
- **[वेबसाइट](./website/)** - कस्टम नेविगेशन शैलियों के साथ अपनी चर्च वेबसाइट बनाएं और प्रबंधित करें
- **[उपदेश](./sermons/)** - अपनी उपदेश लाइब्रेरी और लाइव स्ट्रीमिंग प्रबंधित करें
- **[कैलेंडर](./calendars/)** - समूह इवेंट को जोड़कर क्यूरेटेड कैलेंडर बनाएं
- **[सेटिंग्स](./settings/)** - चर्च जानकारी, भूमिकाएं और अनुमतियां कॉन्फ़िगर करें

:::info

:::
