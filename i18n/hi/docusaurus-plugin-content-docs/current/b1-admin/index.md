# B1 Admin अवलोकन

<div class="article-intro">

B1 Admin आपकी चर्च प्रबंधन डैशबोर्ड है। यह सदस्य निर्देशिकाओं और दान से लेकर वेबसाइट निर्माण और लाइव स्ट्रीमिंग तक -- आपके चर्च के संचालन के हर पहलू को एक ही स्थान से प्रबंधित करने के उपकरण प्रदान करता है।

</div>

## शुरू करना

1. अपना ब्राउज़र खोलें और [admin.b1.church](https://admin.b1.church) पर जाएं
2. अपने उपयोगकर्ता नाम और पासवर्ड से साइन इन करें
3. आप **डैशबोर्ड** पर पहुंचेंगे, जो आपको सभी प्रमुख सुविधाओं तक त्वरित पहुंच प्रदान करता है

:::tip
B1 Admin में नए हैं? वीडियो वॉकथ्रू के लिए [परिचय](./introduction) से शुरू करें, फिर अपनी चर्च जानकारी कॉन्फ़िगर करने और अपनी टीम को आमंत्रित करने के लिए [सेटिंग्स](./settings/) पर जाएं।
:::

## प्रमुख सुविधाएं

- **[लोग](./people/)** - अपनी चर्च निर्देशिका प्रबंधित करें, सदस्य जोड़ें, परिवारों को ट्रैक करें
- **[समूह](./groups/)** - चर्च समूह, टीमें और सेवकाइयां बनाएं और व्यवस्थित करें
- **[उपस्थिति](./attendance/)** - कैंपस, सेवा समय सेट करें और उपस्थिति ट्रैक करें
- **[दान](./donations/)** - दान रिकॉर्ड करें, फंड प्रबंधित करें, विवरण तैयार करें
- **[सेवा](./serving/)** - स्वयंसेवकों का समन्वय करें, सेवा योजनाएं बनाएं, कार्य प्रबंधित करें
- **[फॉर्म](./forms/)** - पंजीकरण और डेटा संग्रह के लिए कस्टम फॉर्म बनाएं
- **[रिपोर्ट](./reports/)** - जन्मदिन, उपस्थिति और दान रिपोर्ट देखें
- **[वेबसाइट](./website/)** - अपनी चर्च वेबसाइट बनाएं और प्रबंधित करें
- **[प्रवचन](./sermons/)** - अपनी प्रवचन लाइब्रेरी और लाइव स्ट्रीमिंग प्रबंधित करें
- **[कैलेंडर](./calendars/)** - इवेंट कैलेंडर बनाएं और प्रकाशित करें
- **[सेटिंग्स](./settings/)** - चर्च जानकारी, भूमिकाएं और अनुमतियां कॉन्फ़िगर करें

:::info
सहायता और दस्तावेज़ीकरण तक त्वरित पहुंच के लिए किसी भी पृष्ठ के ऊपरी-दाएं कोने में प्रश्न चिह्न आइकन पर क्लिक करें।
:::

सहायता के लिए, [support@churchapps.org](mailto:support@churchapps.org) पर ईमेल करें।
