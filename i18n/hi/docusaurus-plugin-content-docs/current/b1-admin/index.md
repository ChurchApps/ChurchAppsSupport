# B1 Admin अवलोकन

<div class="article-intro">

B1 Admin आपका चर्च प्रबंधन डैशबोर्ड है। यह आपके चर्च के संचालन के हर पहलू को प्रबंधित करने के लिए उपकरण प्रदान करता है -- सदस्य निर्देशिकाओं और दान से वेबसाइट निर्माण और लाइव स्ट्रीमिंग तक -- सभी एक जगह से।

</div>

## शुरुआत करना

1. अपने ब्राउज़र को खोलें और [admin.b1.church](https://admin.b1.church) पर जाएँ
2. अपने उपयोगकर्ता नाम और पासवर्ड के साथ साइन इन करें
3. आप **डैशबोर्ड** पर पहुँचेंगे, जो आपके चर्च डेटा का अवलोकन प्रदान करता है साथ ही लोगों को जोड़ने, समूह बनाने, दान रिकॉर्ड करने और अन्य जैसे सामान्य कार्यों के लिए **त्वरित कार्य**

:::tip
B1 Admin में नए हैं? [परिचय](./introduction) से शुरू करें एक वीडियो पूर्वाभास के लिए, फिर अपनी चर्च की जानकारी कॉन्फ़िगर करने और अपनी टीम को आमंत्रित करने के लिए [सेटिंग्स](./settings/) पर जाएँ।
:::

## मुख्य विशेषताएँ

- **[लोग](./people/)** - अपनी चर्च निर्देशिका प्रबंधित करें, सदस्यों को जोड़ें, डेटा को बल्क संपादित करें, घरों को ट्रैक करें
- **[समूह](./groups/)** - शामिल होने के अनुरोधों और सदस्य प्रबंधन के साथ चर्च समूह बनाएँ और व्यवस्थित करें
- **[उपस्थिति](./attendance/)** - परिसरों, सेवा समय और ट्रैक उपस्थिति सेटअप करें
- **[दान](./donations/)** - कई मुद्राओं में दान रिकॉर्ड करें, निधि प्रबंधित करें, विवरण बनाएँ
- **[सेवा](./serving/)** - स्वयंसेवकों का समन्वय करें, सेवा योजनाएँ बनाएँ, कार्य प्रबंधित करें
- **[फॉर्म](./forms/)** - पंजीकरण और डेटा संग्रह के लिए कस्टम फॉर्म बनाएँ
- **[रिपोर्ट](./reports/)** - जन्मदिन, उपस्थिति और दान रिपोर्ट देखें
- **[वेबसाइट](./website/)** - कस्टम नेविगेशन शैलियों के साथ अपनी चर्च वेबसाइट बनाएँ और प्रबंधित करें
- **[सर्मन](./sermons/)** - अपनी सर्मन लाइब्रेरी प्रबंधित करें और लाइव स्ट्रीमिंग करें
- **[कैलेंडर](./calendars/)** - समूह घटनाओं को कनेक्ट करके क्यूरेटेड कैलेंडर बनाएँ
- **[सेटिंग्स](./settings/)** - चर्च की जानकारी, भूमिकाएँ और अनुमतियों को कॉन्फ़िगर करें
