# B1 Admin अवलोकन

<div class="article-intro">

B1 Admin आपके चर्च का प्रबंधन डैशबोर्ड है। यह आपके चर्च के संचालन के हर पहलू को प्रबंधित करने के लिए उपकरण प्रदान करता है -- सदस्य निर्देशिकाओं और दान से लेकर वेबसाइट निर्माण और लाइव स्ट्रीमिंग तक -- सब कुछ एक जगह से।

</div>

## शुरुआत कर रहे हैं

1. अपने ब्राउज़र को खोलें और [admin.b1.church](https://admin.b1.church) पर जाएं
2. अपने उपयोगकर्ता नाम और पासवर्ड से साइन इन करें
3. आप **डैशबोर्ड** पर पहुंचेंगे, जो आपके चर्च डेटा का अवलोकन प्रदान करता है साथ ही लोगों को जोड़ने, समूह बनाने, दान रिकॉर्ड करने आदि जैसे सामान्य कार्यों के लिए **त्वरित कार्य**

:::tip
B1 Admin में नए हैं? [परिचय](./introduction) के साथ एक वीडियो वॉकथ्रू से शुरुआत करें, फिर [सेटिंग्स](./settings/) पर जाएं अपनी चर्च जानकारी को कॉन्फ़िगर करने और अपनी टीम को आमंत्रित करने के लिए।
:::

## मुख्य सुविधाएं

- **[लोग](./people/)** - अपनी चर्च निर्देशिका प्रबंधित करें, सदस्यों को जोड़ें, डेटा को बल्क संपादित करें, घरों को ट्रैक करें
- **[समूह](./groups/)** - चर्च समूह बनाएं और व्यवस्थित करें जॉइन अनुरोधों और सदस्य प्रबंधन के साथ
- **[उपस्थिति](./attendance/)** - कैम्पस सेटअप करें, सेवा समय और उपस्थिति को ट्रैक करें
- **[दान](./donations/)** - कई मुद्राओं में दान रिकॉर्ड करें, निधियों को प्रबंधित करें, विवरण जनरेट करें
- **[सेवा](./serving/)** - स्वयंसेवकों को समन्वित करें, सेवा योजनाएं बनाएं, कार्यों को प्रबंधित करें, और लोगों को चरण-दर-चरण वर्कफ़्लोज़ के माध्यम से आगे बढ़ाएं
- **[फॉर्म](./forms/)** - पंजीकरण और डेटा संग्रह के लिए कस्टम फॉर्म बनाएं
- **[रिपोर्ट](./reports/)** - जन्मदिन, उपस्थिति और दान रिपोर्ट देखें
- **[वेबसाइट](./website/)** - कस्टम नेविगेशन शैलियों के साथ अपनी चर्च वेबसाइट बनाएं और प्रबंधित करें
- **[उपदेश](./sermons/)** - अपनी उपदेश लाइब्रेरी और लाइव स्ट्रीमिंग प्रबंधित करें
- **[कैलेंडर](./calendars/)** - समूह इवेंट को जोड़कर क्यूरेटेड कैलेंडर बनाएं
- **[सेटिंग्स](./settings/)** - चर्च जानकारी, भूमिकाएं और अनुमतियां कॉन्फ़िगर करें
