# ऑनलाइन दान सेटअप

<div class="article-intro">

B1 Admin **Stripe** और **PayPal** के साथ एकीकृत होता है ताकि आपके सदस्य आपकी B1.church साइट के माध्यम से ऑनलाइन दान कर सकें। एक बार कॉन्फ़िगर करने के बाद, ऑनलाइन दान स्वचालित रूप से मैन्युअल रूप से दर्ज किए गए उपहारों के साथ आपके दान रिकॉर्ड में दिखाई देते हैं, जिससे सब कुछ एक ही सिस्टम में रहता है।

</div>

<div class="prereqs">
<h4>शुरू करने से पहले</h4>

- अपने [दान फंड](funds.md) सेट अप करें ताकि दानदाता अपने उपहार निर्दिष्ट कर सकें
- [stripe.com](https://stripe.com) पर एक Stripe खाता बनाएं और उसे सक्रिय करें (टेस्ट मोड से बाहर निकालें)
- अपने B1 Admin लॉगिन क्रेडेंशियल तैयार रखें

</div>

## Stripe सेटअप करना

1. यदि आपके पास पहले से नहीं है तो [stripe.com](https://stripe.com) पर एक खाता बनाएं। **अपना खाता सक्रिय** करना और टेस्ट मोड से बाहर निकालना सुनिश्चित करें।
2. Stripe में, **Developers > API Keys** पर जाएं।
3. अपनी **Publishable Key** कॉपी करें।
4. [B1 Admin](https://admin.b1.church/) में लॉग इन करें।
5. शीर्ष नेविगेशन में **Church** पर क्लिक करें, फिर **Edit Church Settings** पर क्लिक करें।
6. **Church Settings** के बगल में संपादन आइकन पर क्लिक करें।
7. **Giving** अनुभाग तक स्क्रॉल करें।
8. **Provider** को **Stripe** पर सेट करें।
9. अपनी Publishable Key को **Public Key** फ़ील्ड में पेस्ट करें।
10. Stripe पर वापस जाएं और अपनी **Secret Key** प्रकट करें (आप इसे केवल एक बार देख सकते हैं, इसलिए बैकअप सहेजें)।
11. Secret Key को **Secret Key** फ़ील्ड में पेस्ट करें और **Save** पर क्लिक करें।

:::warning
आपकी Stripe Secret Key केवल एक बार दिखाई जाती है। Stripe डैशबोर्ड से दूर जाने से पहले इसे एक सुरक्षित स्थान पर कॉपी करें। यदि आप इसे खो देते हैं, तो आपको एक नई कुंजी जनरेट करनी होगी।
:::

## अपनी B1.church साइट पर दान पृष्ठ जोड़ना

1. [b1.church](https://b1.church/) पर जाएं और लॉग इन करें।
2. **Settings** आइकन पर क्लिक करें।
3. **Add Tab** पर क्लिक करें।
4. प्रकार के रूप में **Donation** चुनें।
5. टैब के लिए एक नाम दर्ज करें (उदा., "दान दें") और **Save** पर क्लिक करें।
6. वैकल्पिक रूप से, टैब आइकन बदलें -- दान संबंधित आइकन के लिए आइकन खोज में "Giv" टाइप करें।

आपका दान पृष्ठ अब लाइव है। सदस्य इसे `yoursubdomain.b1.church/donate` पर देख सकते हैं।

## अपना दान लिंक साझा करना

अपना दान URL खोजने के लिए, **B1 Admin** पर जाएं और अपना सबडोमेन देखने के लिए **Settings** आइकन पर क्लिक करें। आपका दान लिंक इस प्रारूप का पालन करता है:

`https://yoursubdomain.b1.church/donate`

इस लिंक को अपनी वेबसाइट पर, ईमेल में, या अपने बुलेटिन में साझा करें ताकि सदस्यों को पता चले कि ऑनलाइन कहां दान करना है।

## दान सूचनाएं

Stripe हर बार दान प्राप्त होने पर एक ईमेल सूचना भेजता है। सूचना ईमेल पता बदलने के लिए, Stripe डैशबोर्ड पर जाएं, ऊपर दाईं ओर अपने प्रोफ़ाइल पर क्लिक करें, **Profile** चुनें, और अपना ईमेल पता अपडेट करें।

## प्रोसेसिंग शुल्क विकल्प

आप अपने दान पृष्ठ को कॉन्फ़िगर कर सकते हैं ताकि दानदाता वैकल्पिक रूप से प्रोसेसिंग शुल्क वहन कर सकें ताकि आपके चर्च को पूरी दान राशि प्राप्त हो। यह सेटिंग B1 Admin के भीतर आपकी चर्च सेटिंग्स में प्रबंधित की जाती है।

:::tip
सेटअप के बाद, अपनी मंडली को ऑनलाइन दान की घोषणा करने से पहले सब कुछ काम कर रहा है यह पुष्टि करने के लिए एक छोटा परीक्षण दान करें।
:::

## अगले कदम

- यदि ऑनलाइन लेनदेन स्वचालित रूप से सिंक नहीं हो रहे हैं तो उन्हें B1 Admin में लाने के लिए [Stripe आयात](stripe-import.md) का उपयोग करें
- ऑनलाइन दान सही ढंग से दिखाई दे रहे हैं यह सत्यापित करने के लिए अपनी [दान रिपोर्ट](donation-reports.md) जांचें
- ऑनलाइन और ऑफलाइन दोनों दान शामिल करने वाले [दान विवरण](giving-statements.md) तैयार करें
