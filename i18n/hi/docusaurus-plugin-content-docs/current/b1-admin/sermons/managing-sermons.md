# प्रवचन प्रबंधन

<div class="article-intro">

प्रवचन पृष्ठ आपकी संपूर्ण प्रवचन लाइब्रेरी प्रदर्शित करता है। यहां से आप नए प्रवचन जोड़ सकते हैं, मौजूदा प्रविष्टियां संपादित कर सकते हैं, और प्लेलिस्ट के अनुसार अपनी सामग्री व्यवस्थित कर सकते हैं। प्रत्येक प्रवचन YouTube, Vimeo, Facebook, या कस्टम URL पर होस्ट किए गए वीडियो या ऑडियो से लिंक हो सकता है।

</div>

<div class="prereqs">
<h4>शुरू करने से पहले</h4>

- आपको **contentApi.streamingServices.edit** अनुमति की आवश्यकता है। यदि आपके पास एक्सेस नहीं है तो [भूमिकाएं और अनुमतियां](../settings/roles-permissions.md) देखें।
- अपने प्रवचनों को व्यवस्थित करने के लिए कम से कम एक [प्लेलिस्ट](playlists) बनाएं
- YouTube, Vimeo, या Facebook से अपने वीडियो ID या URL तैयार रखें

</div>

## अपनी प्रवचन लाइब्रेरी देखना

1. B1 Admin में, बाएं साइडबार में **Sermons** पर क्लिक करें।
2. प्रवचन पृष्ठ आपकी सभी प्रवचन प्रविष्टियां प्लेलिस्ट के अनुसार व्यवस्थित दिखाता है। प्रत्येक प्रवचन अपना थंबनेल, शीर्षक और तिथि प्रदर्शित करता है।
3. इसका विवरण देखने या संपादित करने के लिए किसी भी प्रवचन पर क्लिक करें।

## प्रवचन जोड़ना

1. ऊपरी दाएं कोने में **Add Sermon** बटन पर क्लिक करें और ड्रॉपडाउन से **Add Sermon** चुनें।
2. प्रवचन को असाइन करने के लिए एक **Playlist** चुनें।
3. अपना **Video Provider** चुनें -- YouTube, Vimeo, Facebook, या Custom URL। हम YouTube की अनुशंसा करते हैं क्योंकि यह B1 सिस्टम के साथ सबसे अच्छा काम करता है।
4. वीडियो ID या URL दर्ज करें और **Fetch** पर क्लिक करें। YouTube के लिए, वीडियो ID YouTube URL में `v=` के बाद वर्णों की स्ट्रिंग है।
5. जब आप **Fetch** पर क्लिक करते हैं, तो प्रकाशन तिथि, अवधि, शीर्षक, विवरण और थंबनेल सहित प्रवचन विवरण स्वचालित रूप से आयात हो जाते हैं।
6. कोई भी बदलाव करें और **Save** पर क्लिक करें।

:::tip
आप **Add Sermon** ड्रॉपडाउन से **Add Permanent Live URL** चुनकर एक स्थायी लाइव स्ट्रीम URL भी जोड़ सकते हैं। यह आपके YouTube चैनल की लाइव स्ट्रीम से आपके Channel ID का उपयोग करके एक स्थायी कनेक्शन बनाता है। अधिक विवरण के लिए [लाइव स्ट्रीमिंग](live-streaming) देखें।
:::

## प्रवचन संपादित करना

1. विवरण खोलने के लिए अपनी लाइब्रेरी में किसी भी प्रवचन पर क्लिक करें।
2. आवश्यकतानुसार शीर्षक, वक्ता, तिथि, विवरण, थंबनेल, या मीडिया लिंक अपडेट करें।
3. अपने परिवर्तन लागू करने के लिए **Save** पर क्लिक करें।

## प्रवचन विवरण

प्रत्येक प्रवचन प्रविष्टि में शामिल हो सकते हैं:

- **शीर्षक** -- आगंतुकों को प्रदर्शित प्रवचन का नाम
- **वक्ता** -- प्रवचन किसने दिया
- **तिथि** -- प्रकाशन या प्रसारण तिथि
- **विवरण** -- प्रवचन सामग्री का सारांश
- **थंबनेल** -- आपकी प्रवचन लाइब्रेरी में दिखाई गई पूर्वावलोकन छवि
- **वीडियो/ऑडियो लिंक** -- YouTube, Vimeo, Facebook, या कस्टम होस्ट पर प्रवचन मीडिया के URL

## लाइव स्ट्रीम के लिए प्रवचन शेड्यूल करना

प्रवचन जोड़ने के बाद, आप इसे अपने लाइव स्ट्रीम पृष्ठ पर प्रसारण के लिए शेड्यूल कर सकते हैं:

1. **Live Stream Times** टैब पर जाएं।
2. एक सेवा संपादित करें और **Video Settings** के तहत, ड्रॉपडाउन से अपना प्रवचन चुनें।
3. प्रवचन शेड्यूल सेवा समय पर चलेगा।

:::info
एक-एक करके जोड़ने के बजाय एक साथ कई प्रवचन आयात करने के लिए, अपने YouTube या Vimeo खाते से सीधे वीडियो खींचने के लिए [बल्क आयात](bulk-import) टूल का उपयोग करें।
:::
