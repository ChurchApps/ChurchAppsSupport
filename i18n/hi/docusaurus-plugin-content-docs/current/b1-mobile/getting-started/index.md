# शुरू करना

<div class="article-intro">

B1 Mobile के साथ शुरू करने में बस कुछ ही मिनट लगते हैं। ऐप डाउनलोड करें, अपने ChurchApps खाते से साइन इन करें, अपना चर्च चुनें, और आप तैयार हैं।

</div>

## त्वरित शुरुआत

1. **डाउनलोड करें** ऐप को [App Store](https://apps.apple.com) (iOS) या [Google Play Store](https://play.google.com) (Android) से। "B1.church" खोजें।
2. **लॉग इन करें** अपने मौजूदा ChurchApps खाते से -- वही ईमेल और पासवर्ड जो आप B1.church पर उपयोग करते हैं। यदि आपके पास अभी तक खाता नहीं है, तो एक बनाने के लिए **Register** पर टैप करें।
3. **अपना चर्च चुनें** नाम से खोजकर। परिणाम देखने के लिए कम से कम 3 अक्षर टाइप करें।
4. **डैशबोर्ड एक्सप्लोर करें** -- आपकी वैयक्तिकृत होम स्क्रीन आपके चर्च द्वारा प्रदान की जाने वाली सुविधाओं के त्वरित लिंक दिखाती है।

बस इतना ही। एक बार जब आपने अपना चर्च चुन लिया, तो ऐप आपके चर्च की ब्रांडिंग, नेविगेशन लिंक और सामग्री लोड कर लेता है। आप समूह, दान, चेक-इन, और बाकी सब कुछ जो आपके चर्च ने सक्षम किया है, उपयोग करना शुरू कर सकते हैं।

## इस अनुभाग में

- [ऐप इंस्टॉल करना](./installing) -- B1 Mobile कहाँ से डाउनलोड करें और डिवाइस आवश्यकताएँ।
- [लॉग इन करना](./logging-in) -- साइन इन कैसे करें या नया खाता कैसे बनाएँ।
- [चर्च स्विच करना](./switching-churches) -- यदि आप एक से अधिक चर्चों से जुड़े हैं तो उनके बीच कैसे स्विच करें।

:::tip
B1 Mobile सभी अन्य ChurchApps उत्पादों के समान ChurchApps खाते का उपयोग करता है। यदि आपके पास पहले से [B1.church](../../b1-church/), B1 Admin, या Lessons.church से कोई खाता है, तो आप लॉग इन करने के लिए उन्हीं क्रेडेंशियल्स का उपयोग कर सकते हैं।
:::
