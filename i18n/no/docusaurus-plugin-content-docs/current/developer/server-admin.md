# Serveradministrasjon

:::info Teknisk dokumentasjon
Denne siden inneholder teknisk administrasjonsdokumentasjon.
For fullstendig engelsk dokumentasjon, se [Server Administration](/docs/developer/server-admin) (engelsk versjon).
:::

<div class="article-intro">

Serveradministrasjonsfunksjoner i ChurchApps er kun tilgjengelige for brukere med **Server.Admin**-tillatelsen. Disse verktøyene brukes for plattformoperasjoner, støtte og feilsøking på tvers av alle kirker i systemet.

</div>

:::warning Tilgang begrenset
Funksjonene beskrevet på denne siden krever **Server.Admin**-tillatelse og er ikke tilgjengelige for vanlige kirkeadministratorer. De er ment kun for plattformoperatører og støttepersonell.
:::

## Tilgang til Server Admin

Brukere med Server.Admin-tillatelse kan få tilgang til serveradmin-panelet fra B1 Admin:

1. Logg inn på [admin.b1.church](https://admin.b1.church)
2. Klikk **Admin**-fanen i hovednavigatoren
3. Serveradmin-panelet inkluderer faner for å administrere kirker, brukere og systemoperasjoner

For fullstendig dokumentasjon på engelsk, se [Server Administration](/docs/developer/server-admin).
