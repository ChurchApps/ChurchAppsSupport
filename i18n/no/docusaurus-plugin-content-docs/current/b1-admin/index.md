# B1 Admin-oversikt

<div class="article-intro">

B1 Admin er kirkens styringsinstrumenter. Det gir verktøy for å administrere alle aspekter av kirkens driftingning -- fra medlemsmapper og donasjoner til nettstedbygging og direktekringkasting -- alt fra ett sted.

</div>

## Kom i gang

1. Åpne nettleseren og gå til [admin.b1.church](https://admin.b1.church)
2. Logg på med brukernavn og passord
3. Du vil lande på **Instrumentbordet**, som gir en oversikt over kirkens data sammen med **Hurtighandlinger** for vanlige oppgaver som å legge til personer, opprette grupper, registrere donasjoner og mer

:::tip
Ny på B1 Admin? Start med [Introduksjonen](./introduction) for en videogjenomgang, og besøk deretter [Innstillinger](./settings/) for å konfigurere kirkeinformasjonen og invitere teamet.
:::

## Nøkkelfunksjoner

- **[Personer](./people/)** - Administrer kirkemappen, legg til medlemmer, spor hushold
- **[Grupper](./groups/)** - Opprett og organiser kirkgrupper, team og ministerier
- **[Oppmøte](./attendance/)** - Sett opp campuser, servicetider og spor oppmøte
- **[Donasjoner](./donations/)** - Registrer giving, administrer fond, generer erklæringer
- **[Tjeneste](./serving/)** - Koordiner frivillige, opprett serviceplan, administrer oppgaver
- **[Skjemaer](./forms/)** - Bygg egendefinerte skjemaer for registreringer og datainnsamling
- **[Rapporter](./reports/)** - Se fødselsdag-, oppmøte- og donasjonsrapporter
- **[Nettsted](./website/)** - Bygg og administrer kirkens nettsted
- **[Prekener](./sermons/)** - Administrer prekebiliteket og direktekringkasting
- **[Kalendere](./calendars/)** - Opprett og publiser hendelseskalendere
- **[Innstillinger](./settings/)** - Konfigurer kirkeinformasjon, roller og tillatelser

:::info
Klikk spørsmålstegnet i det øvre høyre hjørnet på enhver side for rask tilgang til hjelp og dokumentasjon.
:::
