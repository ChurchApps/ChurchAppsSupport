# B1 Admin-oversikt

<div class="article-intro">

B1 Admin er ditt administrasjonsdashboard for menigheten. Det gir verktøy for å administrere alle aspekter av menighetens drift -- fra medlemskatalogen og donasjoner til nettstedbygging og direktestrømming -- alt fra ett sted.

</div>

## Komme i gang

1. Åpne nettleseren din og gå til [admin.b1.church](https://admin.b1.church)
2. Logg inn med brukernavn og passord
3. Du kommer til **Dashbordet**, som gir en oversikt over menighetsdataene dine sammen med **Hurtighandlinger** for vanlige oppgaver som å legge til personer, opprette grupper, registrere donasjoner og mer

:::tip
Ny i B1 Admin? Start med [Introduksjonen](./introduction) for en videogennemgang, og besøk deretter [Innstillinger](./settings/) for å konfigurere menighetsinformasjonen din og invitere teamet ditt.
:::

## Hovedfunksjoner

- **[Personer](./people/)** - Administrer menighetskatalogen din, legg til medlemmer, spor husholdninger
- **[Grupper](./groups/)** - Opprett og organiser menighetsgrupper med tilmeldingsforespørsler og medlemsadministrasjon
- **[Oppmøte](./attendance/)** - Sett opp lokasjoner, gudstjenestetider og spor oppmøte
- **[Donasjoner](./donations/)** - Registrer gaver i flere valutaer, administrer fond, generer kvitteringer
- **[Tjeneste](./serving/)** - Koordiner frivillige, opprett tjenesteplaner, administrer oppgaver
- **[Skjemaer](./forms/)** - Bygg tilpassede skjemaer for påmelding og datainnsamling
- **[Rapporter](./reports/)** - Se bursdags-, oppmøte- og donasjonsrapporter
- **[Nettsted](./website/)** - Bygg og administrer menighetsnettstedet ditt med tilpassede navigasjonsstiler
- **[Prekener](./sermons/)** - Administrer prekenbiblioteket ditt og direktestrømming
- **[Kalendere](./calendars/)** - Opprett kuraterte kalendere ved å koble gruppearrangementer
- **[Innstillinger](./settings/)** - Konfigurer menighetsinformasjon, roller og tillatelser

:::info
Klikk på spørsmålstegn-ikonet øverst til høyre på en hvilken som helst side for rask tilgang til hjelp og dokumentasjon.
:::
