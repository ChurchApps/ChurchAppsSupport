# B1 Admin-oversikt

<div class="article-intro">

B1 Admin er din kirkeadministrasjonsdashboard. Det gir verktøy for å administrere alle aspekter av kirkens drift -- fra medlemskatalogen og donasjoner til nettstedbygging og direktestrømming -- alt fra ett sted.

</div>

## Komme i gang

1. Åpne nettleseren din og gå til [admin.b1.church](https://admin.b1.church)
2. Logg inn med brukernavn og passord
3. Du kommer til **Dashbordet**, som gir en oversikt over kirkedataene dine sammen med **Hurtighandlinger** for vanlige oppgaver som å legge til personer, opprette grupper, registrere donasjoner og mer

:::tip
Ny i B1 Admin? Start med [Introduksjonen](./introduction) for en videogennemgang, besøk deretter [Innstillinger](./settings/) for å konfigurere kirkeinformasjonen din og invitere teamet ditt.
:::

## Hovedfunksjoner

- **[Personer](./people/)** - Administrer kirkekatalogen din, legg til medlemmer, spor husholdninger
- **[Grupper](./groups/)** - Opprett og organiser kirkegrupper, team og tjenestesteder
- **[Fremmøte](./attendance/)** - Sett opp campus, gudstjenestetider og spor fremmøte
- **[Donasjoner](./donations/)** - Registrer gaver, administrer fond, generer kvitteringer
- **[Tjeneste](./serving/)** - Koordiner frivillige, opprett tjenesteplaner, administrer oppgaver
- **[Skjemaer](./forms/)** - Bygg tilpassede skjemaer for påmelding og datainnsamling
- **[Rapporter](./reports/)** - Se bursdags-, fremmøte- og donasjonsrapporter
- **[Nettsted](./website/)** - Bygg og administrer kirkenettstedet ditt
- **[Prekener](./sermons/)** - Administrer prekenbiblioteket ditt og direktestrømming
- **[Kalendere](./calendars/)** - Opprett kurerte kalendere ved å koble gruppehendelser
- **[Innstillinger](./settings/)** - Konfigurer kirkeinformasjon, roller og tillatelser

:::info
Klikk spørsmålstegnikonet øverst til høyre på enhver side for rask tilgang til hjelp og dokumentasjon.
:::
