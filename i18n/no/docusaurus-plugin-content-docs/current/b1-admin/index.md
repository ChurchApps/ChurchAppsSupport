# B1 Admin oversikt

<div class="article-intro">

B1 Admin er ditt dashbord for menighetsadministrasjon. Det gir verktoy for a administrere alle aspekter av menighetens drift -- fra medlemskataloger og donasjoner til nettstedsbygging og direktesending -- alt fra ett sted.

</div>

## Kom i gang

1. Apne nettleseren din og ga til [admin.b1.church](https://admin.b1.church)
2. Logg inn med brukernavn og passord
3. Du kommer til **Dashbordet**, som gir deg rask tilgang til alle hovedfunksjoner

:::tip
Ny pa B1 Admin? Start med [Introduksjonen](./introduction) for en videogjennomgang, og besok deretter [Innstillinger](./settings/) for a konfigurere menighetsinfo og invitere teamet ditt.
:::

## Hovedfunksjoner

- **[Mennesker](./people/)** - Administrer menighetsregisteret, legg til medlemmer, spor husholdninger
- **[Grupper](./groups/)** - Opprett og organiser menighetsgrupper, team og tjenester
- **[Oppmote](./attendance/)** - Sett opp campus, gudstjenestetider og spor oppmote
- **[Donasjoner](./donations/)** - Registrer gaver, administrer fond, generer oppgaver
- **[Tjeneste](./serving/)** - Koordiner frivillige, lag tjenesteplaner, administrer oppgaver
- **[Skjemaer](./forms/)** - Bygg tilpassede skjemaer for registrering og datainnsamling
- **[Rapporter](./reports/)** - Se bursdag-, oppmote- og donasjonsrapporter
- **[Nettsted](./website/)** - Bygg og administrer menighetens nettsted
- **[Prekener](./sermons/)** - Administrer prekenbiblioteket og direktesending
- **[Kalendere](./calendars/)** - Opprett og publiser arrangementskalendere
- **[Innstillinger](./settings/)** - Konfigurer menighetsinfo, roller og tillatelser

:::info
Klikk pa sporsmalsmerke-ikonet i ovre hoyre hjorne av en hvilken som helst side for rask tilgang til hjelp og dokumentasjon.
:::

For brukerstotte, send e-post til [support@churchapps.org](mailto:support@churchapps.org).
