# Donasjoner

<div class="article-intro">

Donasjonsseksjonen i B1 Admin gir deg alt du trenger for å administrere menighetens givertjeneste. Fra registrering av ukentlige ofringer til generering av skatteoppgaver ved årsslutt, alle donasjonsverktøyene dine er organisert på ett sted slik at du kan holde oversikt over menighetens økonomi.

</div>

## Underseksjoner

Du finner følgende sider i sidepanelet for Donasjoner:

### Sammendrag

**Sammendrag**-siden gir deg en visuell oversikt over all giveraktivitet. Den viser diagrammer og filtrerte rapporter slik at du raskt kan se donasjonstrender i menigheten din. Se [Donasjonsrapporter](donation-reports.md) for mer informasjon om bruk av sammendrags- og rapportverktøyene.

### Grupper

**[Grupper](batches.md)** lar deg samle donasjoner sammen -- for eksempel alle sjekker og kontanter som ble samlet inn under en søndagsofring. Hver gruppe sporer antall donasjoner og totalbeløp for enkel avstemming.

### Fond

**[Fond](funds.md)** kategoriserer donasjonene dine etter formål. Vanlige eksempler inkluderer Generelt fond, Bygningsfond og Misjoner. Du kan merke hvert fond som skattefradragsberettiget eller ikke.

### Giveroppgaver

**[Giveroppgaver](giving-statements.md)**-siden genererer årlige giveroppgaver for giverne dine. Du kan laste ned individuelle CSV-filer eller skrive ut alle oppgaver samtidig.

### Stripe-import

Hvis du aksepterer donasjoner på nett gjennom Stripe, lar **[Stripe-import](stripe-import.md)**-verktøyet deg hente disse transaksjonene inn i B1 Admin slik at alt er i ett system.

## Nettbasert givertjeneste

B1 Admin integreres med **Stripe** og **PayPal** for å akseptere donasjoner på nett. Når det er konfigurert, kan medlemmer gi gjennom din B1.church-side, og donasjonene vises automatisk i dine registre. Se [Oppsett av nettbasert givertjeneste](online-giving-setup.md) for å komme i gang.

:::tip
Hvis du er ny med donasjoner i B1 Admin, start med å [sette opp fondene dine](funds.md), deretter [opprett din første gruppe](batches.md) og [registrer donasjoner](recording-donations.md) i den.
:::
