# Innstillinger

<div class="article-intro">

Innstillinger-seksjonen er hvor du konfigurerer kirkekontoen, merkevarebygging og administrative alternativer i B1 Admin. Herfra kan du administrere alt fra kirkenavn og subdomene til brukertillatelser og mobilappkonfigurasjon.

</div>

## Hva du finner her

Innstillinger-området er organisert i fire underseksjoner tilgjengelig fra sidelinjen:

1. **Innstillinger** -- Konfigurer kirkens grunnleggende informasjon, merkevarebygging og subdomene. Se [Kirkeinnstillinger](./church-settings.md) for detaljer.
2. **Mobilapper** -- Sett opp og tilpass navigasjonslanene som vises i [B1.church PWA](/docs/b1-church/getting-started/installing-pwa) for medlemmene dine (de samme fanene gjengis også av den utdaterte B1 Mobile native appen). Se [Mobilappinnstillinger](./mobile-app.md) for detaljer.
3. **Serveradministrator** -- Tilgang til avanserte administrasjonsverktøy for å administrere kirkens servernivåinnstillinger.
4. **Skjemaer** -- Opprett og administrer tilpassede skjemaer for å samle informasjon fra menigheten din.

## Hovedinnstillingssiden

Når du først åpner Innstillinger-siden, vil du se **kirkenavnet** og **subdomene** vist øverst. Overskriften gir hurtigtilgangsknapper for vanlige oppgaver:

- **Rediger innstillinger** -- Oppdater kirkens navn, adresse, kontaktinformasjon og merkevarebygging.
- **Mobilapper** -- Hopp direkte til mobilappkonfigurasjon.
- **Roller** -- Administrer brukerroller og tillatelser for teamet ditt.
- **Import/Eksport** -- Overfør data mellom systemer ved hjelp av import/eksport-verktøyet.

:::tip
Start med å konfigurere kirkenavnet og merkevarebyggingen under **Rediger innstillinger**, sett deretter opp [Roller](./roles-permissions.md) for å invitere teammedlemmene dine med riktig tilgangsnivå.
:::

:::info
For informasjon om hvordan dataene dine beskyttes, se [Datasikkerhet](./data-security.md).
:::
