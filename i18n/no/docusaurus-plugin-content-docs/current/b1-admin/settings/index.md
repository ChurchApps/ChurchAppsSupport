# Innstillinger

<div class="article-intro">

Innstillinger-seksjonen er der du konfigurerer kirkens konto, merkevarebygging og administrative alternativer i B1 Admin. Herfra kan du administrere alt fra kirkenavnet og underdomenet ditt til brukertillatelser og mobilappkonfigurasjon.

</div>

## Hva du finner her

Innstillinger-området er organisert i fire underseksjoner som er tilgjengelige fra sidefeltet:

1. **Innstillinger** -- Konfigurer kirkens grunnleggende informasjon, merkevarebygging og underdomene. Se [Kirkeinnstillinger](./church-settings.md) for detaljer.
2. **Mobilapper** -- Sett opp og tilpass navigasjonsfanene som vises i B1 Mobile-appen for medlemmene dine. Se [Mobilappinnstillinger](./mobile-app.md) for detaljer.
3. **Serveradministrasjon** -- Få tilgang til avanserte administrasjonsverktøy for å administrere kirkens innstillinger på servernivå.
4. **Skjemaer** -- Opprett og administrer egendefinerte skjemaer for å samle inn informasjon fra menigheten din.

## Hovedsiden for innstillinger

Når du først åpner Innstillinger-siden, vil du se **kirkenavnet** og **underdomenet** ditt vist øverst. Overskriften gir hurtigtilgangsknapper for vanlige oppgaver:

- **Rediger innstillinger** -- Oppdater kirkens navn, adresse, kontaktinformasjon og merkevarebygging.
- **Mobilapper** -- Gå direkte til mobilappkonfigurasjon.
- **Roller** -- Administrer brukerroller og tillatelser for teamet ditt.
- **Import/Eksport** -- Overfør data mellom systemer ved hjelp av import/eksport-verktøyet.

:::tip
Start med å konfigurere kirkenavnet og merkevarebyggingen under **Rediger innstillinger**, og sett deretter opp [Roller](./roles-permissions.md) for å invitere teammedlemmene dine med riktig tilgangsnivå.
:::

:::info
For informasjon om hvordan dataene dine er beskyttet, se [Datasikkerhet](./data-security.md).
:::
