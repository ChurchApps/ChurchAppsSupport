# Kalendere

<div class="article-intro">

Kalender-seksjonen i B1 Admin lar deg opprette kurerte kalendere for kirken din ved å samle hendelser fra dine grupper. Gruppeledere administrerer sine egne gruppers hendelser, og som administrator kan du bygge kalendere som viser disse hendelsene på B1.church-nettstedet ditt for menigheten å se.

</div>

## Se kalenderne dine

1. I B1 Admin, naviger til **Nettsted**, deretter til **Kalendere**-seksjonen.
2. Kalender-siden viser alle kalenderne dine med navn og status.
3. Hver kalender viser også gruppene som er knyttet til den.

## Hva du kan gjøre

- **Legg til en ny kalender** -- Klikk legg til-knappen for å opprette en kalender for et spesifikt tjenestested, hendelsestype eller formål. Se [Opprette kalendere](creating-calendars) for trinn-for-trinn-instruksjoner.
- **Rediger en kalender** -- Klikk på en hvilken som helst kalender for å åpne den og administrere hvilke grupper som er inkludert i den.
- **Publiser kalendere** -- Kalendere kan vises på B1.church-nettstedet ditt slik at besøkende kan se kommende hendelser i kirken din.
- **Opprett kurerte kalendere** -- Hent hendelser fra flere grupper inn i én kalendervisning for en omfattende oversikt over alt som skjer i kirken din. Se [Kurert kalender](curated-calendar) for detaljer.

:::info
Kalendere fungerer tett sammen med [Grupper](../groups/creating-groups.md). Når grupper har planlagte hendelser, kan disse hendelsene hentes inn i kalenderne dine. Se [Opprette kalendere](creating-calendars)-siden for å lære hvordan du kobler grupper til kalendere.
:::
