# Kalendere

<div class="article-intro">

Kalender-seksjonen i B1 Admin lar deg opprette og administrere arrangementskalendere for kirken din. Du kan sette opp kalendere for forskjellige tjenester, samle arrangementer fra flere grupper, og publisere kalendere på B1.church-nettsiden din slik at menigheten kan se dem.

</div>

## Vise kalenderne dine

1. I B1 Admin, naviger til **Kalendere**-seksjonen.
2. Kalender-siden viser alle kalenderne dine med navn og status.
3. Hver kalender viser også gruppene som er tilknyttet den.

## Hva du kan gjøre

- **Legge til en ny kalender** -- Klikk på legg til-knappen for å opprette en kalender for en bestemt tjeneste, arrangementstype eller formål. Se [Opprette kalendere](creating-calendars) for trinnvise instruksjoner.
- **Redigere en kalender** -- Klikk på en kalender for å åpne den og administrere arrangementene og tilknyttede grupper.
- **Publisere kalendere** -- Kalendere kan vises på B1.church-nettsiden din slik at besøkende kan se kommende arrangementer i kirken.
- **Opprette kuraterte kalendere** -- Hent arrangementer fra flere grupper inn i én enkelt kalendervisning for en omfattende oversikt over alt som skjer i kirken din. Se [Kuratert kalender](curated-calendar) for detaljer.

:::info
Kalendere fungerer tett sammen med [Grupper](../groups/creating-groups.md). Når grupper har planlagte arrangementer, kan disse hentes inn i kalenderne dine. Se siden [Opprette kalendere](creating-calendars) for å lære hvordan du kobler grupper til kalendere.
:::
