# B1.church

<div class="article-intro">

B1.church er medlemsportalen for din kirke. Det er stedet hvor menigheten din kan holde seg tilkoblet, få tilgang til kirkens verktøy og engasjere seg med fellesskapet gjennom hele uken. Hver kirke får sin egen URL på `https://dinkirke.b1.church`.

</div>

Når medlemmer besøker B1.church-nettstedet ditt, ser de kirkens offentlige nettside med deres merkevare, navigasjonslenker og innholdssider. Etter innlogging kan de få tilgang til **Medlemsportalen** der alle de personlige kirkeverktøyene finnes på ett sted.

## Hovedfunksjoner

- **Tidslinje** -- En personlig feed med oppdateringer, samtaler og aktivitet fra gruppene dine og kirken.
- **Grupper** -- Se gruppene du tilhører, se gruppemedlemmer, delta i samtaler og følg oppmøte.
- **Givertjeneste** -- Gi gaver til kirken din på nett og se giverhistorikken din.
- **Planer** -- Se kommende tjenesteplaner, vis dine tildelte posisjoner og få tilgang til sangdetaljer og tjenesterekkefølge.
- **Innsjekking** -- Sjekk inn deg selv og husstandsmedlemmene dine for gudstjenester og arrangementer.
- **Fellesskapskatalog** -- Bla gjennom kirkens medlemskatalog, vis profiler og send direktemeldinger.
- **Innhold** -- Få tilgang til prekener, les Bibelen, arbeid gjennom leksjoner og se direktestrømming, alt fra kirkens nettsted.
