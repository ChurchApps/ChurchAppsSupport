# B1 Kirke

<div class="article-intro">

B1 Kirke er medlemsportalen for kirken din. Medlemmer kan se hendelser, melde seg på aktiviteter, se preken, få informasjon om frivilligheten, og sjekke inn for tjenester -- alt fra en plass.

</div>

## Nøkkelfunksjoner

- Se kommende hendelser og registrer deg
- Se prekenehistorikk
- Delta i direktekringkasting av tjenester
- Meld deg frivillig for servicer
- Sjekk inn for tjenester
- Administrer gruppemeldskap

## Dokumentasjonseksjoner

- **[Innsjekking](./checkin/)** -- Sjekk inn for tjenester
- **[Hendelser](./events/)** -- Se og registrer for hendelser
- **[Tjeneste](./serving/)** -- Meld deg frivillig
