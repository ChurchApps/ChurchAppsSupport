# Kom i gang

<div class="article-intro">

Velkommen til ChurchApps -- gratis, open source-verktoy designet for a hjelpe kirken din til a trives. Enten du er en liten menighet eller en stor organisasjon med flere lokasjoner, gir ChurchApps deg alt du trenger for a administrere og utvikle kirkefellesskapet ditt.

</div>

## ChurchApps-produkter

ChurchApps inkluderer flere produkter som fungerer sammen:

- **B1 Admin** -- Administrasjonspanelet for kirken din. Administrer personer, grupper, oppmote, donasjoner, nettstedinnhold og mer fra ett sentralt sted.
- **B1.church** -- Det offentlige nettstedet for kirken din og medlemsportalen. Medlemmer kan se prekener, fa tilgang til kataloger, gi gaver pa nett og holde kontakten.
- **Lessons.church** -- Gratis kirkelig undervisningsmateriell for sondagsskole, smagrupper og bibelstudier. Fa tilgang til et voksende bibliotek med leksjoner uten kostnad.
- **FreeShow** -- Gratis presentasjonsprogramvare for gudstjenestene dine. Vis sangtekster, lysbilder og media under samlingene dine.

:::info
Alle ChurchApps-verktoy er **100 % gratis** uten begrensninger, premiumnivaer eller skjulte avgifter. ChurchApps er en ideell organisasjon som stettes av donasjoner og frivillige.
:::

## Hva denne dokumentasjonen dekker

Denne dokumentasjonen vil veilede deg gjennom alt du trenger for a komme i gang med ChurchApps. Du vil laere hvordan du:

1. Oppretter kontoen din og registrerer kirken din.
2. Konfigurerer kirkens grunnleggende innstillinger og merkevarebygging.
3. Setter opp teamet ditt med riktige roller og tillatelser.
4. Begynner a bruke verktoyene for a administrere menigheten din.
