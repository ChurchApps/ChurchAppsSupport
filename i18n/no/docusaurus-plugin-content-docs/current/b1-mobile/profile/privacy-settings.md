# Personverninnstillinger

<div class="article-intro">

**Synlighet**-fanen pa Profilskjermen lar deg kontrollere hva andre menighetsmedlemmer kan se om deg i katalogen. Du bestemmer hvilke personlige detaljer som deles offentlig og hvilke som forblir private.

</div>

<div class="prereqs">
<h4>For Du Begynner</h4>

- [Installer B1 Mobile](../getting-started/installing) og [logg inn](../getting-started/logging-in) pa kontoen din

</div>

## Kontrollere Synligheten

1. Apne sidemenyen og trykk pa **Profil**.
2. Velg **Synlighet**-fanen.
3. Gjennomga hvert felt og velg om det er synlig for andre medlemmer.
4. Lagre endringene.

Du kan kontrollere synligheten for detaljer som:

- **Telefonnummer** -- Om andre medlemmer kan se telefonnummeret ditt
- **E-postadresse** -- Om e-posten din vises i [katalogen](../community/member-directory.md)
- **Adresse** -- Om hjemmeadressen din er synlig for andre

Som standard kan kirken ha visse felt synlige eller skjulte. Du kan justere disse innstillingene nar som helst for a matche komfortnivaaet ditt.

:::tip
Hvis du er usikker pa hva andre kan se, besok **Synlighet**-fanen og gjennomga de navaerende innstillingene. Du kan alltid gjore endringer senere.
:::

## Ventende Endringer

**Ventende**-fanen viser profilendringer du har sendt inn som venter pa administratorgodkjenning. Noen kirker krever at en administrator gjennomgar og godkjenner visse endringer for de trer i kraft. Hvis du har gjort en endring som krever godkjenning, ser du den oppfort her med dens navaerende status.

:::info
Hvis en endring du har gjort ikke har tradd i kraft enna, sjekk **Ventende**-fanen. Kirkens administrator ma kanskje godkjenne den forst.
:::

## Kontoinnstillinger

**Konto**-fanen gir tilgang til sikkerhetsinnstillingene dine. Bruk denne fanen til a administrere innstillinger relatert til kontotilgang og sikkerhet.

## Sammendrag av Profilfaner

| Fane | Formal |
|---|---|
| **Profil** | [Rediger personlig informasjon](./editing-profile) og bilde |
| **Familie** | [Se familiemedlemmene dine](./household) |
| **Synlighet** | Kontroller hva andre ser i katalogen |
| **Ventende** | Gjennomga profilendringer som venter pa godkjenning |
| **Konto** | Administrer sikkerhetsinnstillinger |

Disse fanene fungerer sammen for a gi deg full kontroll over personlig informasjon og hvordan den deles i kirkefellesskapet ditt.
