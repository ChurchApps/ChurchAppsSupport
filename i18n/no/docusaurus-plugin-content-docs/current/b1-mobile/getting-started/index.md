# Kom i Gang

<div class="article-intro">

A komme i gang med B1 Mobile tar bare noen minutter. Last ned appen, logg inn med ChurchApps-kontoen din, velg kirken din, og du er klar.

</div>

## Hurtigstart

1. **Last ned** appen fra [App Store](https://apps.apple.com) (iOS) eller [Google Play Store](https://play.google.com) (Android). Sok etter "B1.church".
2. **Logg inn** med din eksisterende ChurchApps-konto -- samme e-post og passord som du bruker pa B1.church. Hvis du ikke har en konto enna, trykk pa **Registrer** for a opprette en.
3. **Velg kirken din** ved a soke etter den med navn. Skriv minst 3 tegn for a se resultater.
4. **Utforsk hjemmesiden** -- din personlige startside viser hurtiglenker til funksjonene kirken din tilbyr.

Det er alt. Nar du har valgt kirken din, laster appen inn kirkens merkevare, navigasjonslenker og innhold. Du kan begynne a bruke grupper, givertjeneste, innsjekking og alt annet kirken din har aktivert.
