# B1 Mobile

<div class="article-intro">

B1 Mobile er den tilhorende appen for B1.church, tilgjengelig pa iOS og Android. Den gir menighetsmedlemmer tilgang til alle verktoyene kirken din tilbyr pa farten -- grupper, givertjeneste, innsjekking, tjenesteplaner, medlemskatalog, privat meldingstjeneste, prekener, Bibelen, dagens bibelvers, leksjoner og mer.

</div>

Appen bruker ChurchApps-kontoen din, den samme som du bruker pa B1.church. Etter at du har logget inn og valgt kirken din, tilpasser appen seg til kirkens merkevare og konfigurasjon. Hvis du tilhorer flere kirker, kan du bytte mellom dem nar som helst.

B1 Mobile stotter push-varsler slik at du aldri gar glipp av en viktig oppdatering fra kirken eller gruppene dine. Hjemmesiden tilpasser seg basert pa bruken din -- funksjonene du besokrer oftest blir loftet til toppen for rask tilgang.

## Hovedfunksjoner

- **Grupper** -- Se gruppene dine, delta i samtaler og hold kontakten med gruppemedlemmene.
- **Givertjeneste** -- Gi gaver til kirken din direkte fra telefonen.
- **Innsjekking** -- Sjekk inn deg selv og familiemedlemmene dine for gudstjenester og arrangementer.
- **Tjenesteplaner** -- Se kommende tjenesteplaner og dine tildelte frivilligposisjoner.
- **Medlemskatalog** -- Bla gjennom kirkens katalog og se medlemsprofiler.
- **Meldingstjeneste** -- Send og motta private meldinger med andre menighetsmedlemmer.
- **Prekener** -- Se eller lytt til prekener fra kirken din.
- **Bibelen** -- Les Bibelen i appen.
- **Dagens Bibelvers** -- Start dagen med et daglig bibelvers.
- **Leksjoner** -- Fa tilgang til og arbeid gjennom leksjoner fra Lessons.church.
