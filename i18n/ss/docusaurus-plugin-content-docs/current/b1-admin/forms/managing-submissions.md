# Kuphatsa Lokutfunyelweko

<div class="article-intro">

Nase ifomu yakho isisebentako, lokutfunyelwe kutawucala kufika. Ithebhu Lokutfunyelweko likuvumela kutsi uhlolisise timphendvulo tonkhe, ulandzelele kutsi ngubani lotfumeleko, futsi usebentise loludzaba kute ulandzelele ebandleni lakho ngesikhatsi lesifanele.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- [Dala ifomu](./creating-forms.md) lenemibuto lengasiyo ngephansi kwalinye
- Yabelana ngefomu nge-URL yayo yemphakatsi nobe uyifake ku[sayithi lakho](../website/managing-pages.md)

</div>

## Kubuka Lokutfunyelweko

1. Yiya ku-**Emafomu** kusuka kumenyu lesikhulu.
2. Chofota ifomu lofuna kuyihlolisisa.
3. Yiya kuthebhu le-**Lokutfunyelweko**.

Utawubona luhlu lwato tonkhe timphendvulo letemukelwe kuleyo fomu. Lokutfunyelwe ngakunye kukhombisa imininingwane yalowo lophendvulile kanye nelusuku lekutfunyelwa kwakho.

## Kuhlolisisa Lokutfunyelweko Nga Munye

Chofota nome nguluphi lokutfunyelwe kute ubone imininingwane yonkhe. Utawubona tonkhe timphendvulo lomuntfu layiniketa embutweni ngamunye wefomu.

## Kutsi Lokutfunyelweko Kuqoqwa Njani

Lokutfunyelwe kungangena ngetindlela letinyenti:

- **I-URL Yemphakatsi** -- Uma ifomu yakho ine-URL yemphakatsi (emafomu Lasimeleko), noma ngubani lonelinki angatfumela imphendvulo. Yabelana nge-URL nge-imeyili, kunkhulumo-mgca yemaphephandzaba, nobe ngemeseji.
- **Kufakwa esayithini lakho** -- Emafomu lafakwe kusayithi lakho leB1 aqoqa lokutfunyelwe ngekutitomatika nabavakashi bagcwalisa wona.
- **Kusetjentiswa ngekhatsi** -- Sicuku netihlengi nabo bangatfumela emafomu esikhundleni sebanye ngekhatsi ku-B1 Admin.

:::tip
Hlola ithebhu Lokutfunyelweko njalonjalo kumafomu lafana nemakhadi ebahambeli nobe tibuto temthandazo, kuze ukwati kulandzela ngekushesha.
:::

## Kusebentisa Idatha Yalokutfunyelweko

Idatha leqoqwe ngemafomu ingakusita:

- Ulandzelele kubhaliswa kwemcimbi bese uhlela ngendlela lefanele
- Ulandzele bahambeli nobe emalunga lamasha
- Uqoqe imiphendvulo yetibuto tekuhlela nekulungiselela inkonzo
- Ucubungule tinkhokhelo temicimbi nobe tinhlelo

:::info
Emalunga efomu lanetikhundla te-**Admin** nobe **Kubuka Kuphela** angafinyelela kuthebhu Lokutfunyelweko. Cinisekisa kutsi bantfu labafanele kusicuku sakho banefinyelelo emafomini labawadzingako ngekuphatsa tikhundla kuthebhu **Emalunga**.
:::

## Tinyatselo Letilandzelako
