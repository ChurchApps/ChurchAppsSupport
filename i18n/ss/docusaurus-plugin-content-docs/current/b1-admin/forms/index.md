# Emafomu

<div class="article-intro">

Sicele Semafomu sikuvumela kudala emafomu lakhetsekile ekubhalisa, tibuto tekuhlolisisa, kucoca lwati, nakunye. Nobe udzinga likhadi lesihambeli, kubhalisa emcimbini, nobe ifomu yesicelo semthandazo, ungayakha lapha bese uyabelana nayo nelibandla lakho ngelinki lemphakatsi nobe ngesayithi lakho leB1.church.

</div>

## Likhasi Lemafomu

Nawuvula **Emafomu**, utawubona tithebhu letimbili:

- **Emafomu** -- Ikhombisa onkhe emafomu akho lasebentako.
- **Lagcinwe** -- Ikhombisa emafomu lasegciniwe. Ungabuyisela emafomu lagciniwe nome nini.

Ifomu ngayinye eluhlwini ikhombisa ligama layo bese, uma ihleliwe, i-URL yemphakatsi longayabelana ngayo ngaphandle.

## Loko Longakwenta

Kusuka ekhasini Lemafomu, ungakwati:

- **Kwengeta ifomu** -- Chofota **Engeta Fomu** kute udale ifomu lensha lekhetsekile.
- **Kuhlela ifomu** -- Chofota ligama lefomu kute uyivule bese ushintje imibuto yayo, emalunga, nobe tilungiselelo tayo.
- **Kugcina ifomu** -- Hambisa emafomu longasadzingi kutsi asebente kuya kuthebhu **Lagcinwe** kuze uhlale nelusha lolucocekile.
- **Kubuyisela ifomu** -- Shintjela kuthebhu **Lagcinwe** bese ubuyisela nome nguliphi ifomu lofuna kutsi lisebente futsi.

## Kusebenta Nefomu

Chofota nome nguliphi ligama lefomu kute uyivule. Sihleli sefomu sinetithebhu letintsatfu:

- **Imibuto** -- Yakha sakhiwo sefomu ngekwengeta tinsimu ngetinhlobo letahlukene. Buka [Kudala Emafomu](./creating-forms.md) kute utfole imininingwane.
- **Emalunga** -- Lawula kutsi ngubani longafinyelela nekuphatsa ifomu.
- **Lokutfunyelweko** -- Buka timphendvulo letitfunyelwe. Buka [Kuphatsa Lokutfunyelweko](./managing-submissions.md) kute utfole imininingwane.

:::tip
Emafomu lanele-URL yemphakatsi angabelanwa ngelinki, afakwe kusayithi lakho, nobe afakwe kwe-imeyili. Loku kwenta kube lula kucoca lwati kubantfu labangenayo i-akhawunti yelibandla mhlawumbe.
:::

:::info
Ungakwati kulawula kutsi ngubani lonefinyelelo kufomu ngalinye usebentisa lithebhu le-**Emalunga**. Nika tikhundla te-**Admin** nobe **Kubuka Kuphela** kute unike sicuku sakho lizinga lelifanele lefinyelelo.
:::
