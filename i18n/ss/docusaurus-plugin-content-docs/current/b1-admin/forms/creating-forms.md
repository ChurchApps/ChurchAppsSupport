# Kudala Emafomu

<div class="article-intro">

Yakha emafomu lakhetsekile kute uqoqe lwati kulibandla lakho. Ungadala emafomu ekubhaliswa kwemicimbi, tibuto tekuhlolisisa, emakhadi ebahambeli, ticelo tebulunga, nakunye. Emafomu angahlotjaniswa nebantfu ku-dathabhesi yakho nobe asetjentiswe njengemakhasi lasimeleko lanembala webawo i-URL yemphakatsi.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Kumafomu e-**Bantfu** (lahlotjaniswe nemarekhodi ebantfu), udzinga [bantfu ku-dathabhesi yakho](../people/adding-people.md) kucala.
- Kumafomu lacoca **tinkhokhelo**, kufanele ube [neStripe lehleliwe kunikela kwenkhulumo-mgca](../donations/online-giving-setup.md).

</div>

## Kudala Ifomu Lensha

1. Yiya ku-**Emafomu** kusuka kumenyu lesikhulu.
2. Chofota **Engeta Fomu**.
3. Faka **ligama** lefomu yakho.
4. Khetsa luhlobo lwefomu kuluhlu lolwehlako:
   - **Bantfu** — Ihlotjanisa lokutfunyelwe [nemarekhodi ebantfu](../people/adding-people.md) ku-dathabhesi yakho.
   - **Lasimeleko** — Idala ifomu lelisimeleko lenayo i-URL yayo yemphakatsi, lelifanele kubhaliswa kwangaphandle.
5. Chofota **Gcina** kute udale ifomu.

Ifomu lakho lelisha litawuvela eluhlwini. Chofota kuyo kute ucale kwengeta imibuto.

## Kwengeta Imibuto

1. Vula ifomu yakho bese uya kuthebhu le-**Imibuto**.
2. Chofota **Engeta Umbuto**.
3. Khetsa **luhlobo lwensimu** kuluhlu lolwehlako lweProvider. Tinhlobo letikhona tifaka:
   - **Ibhokisi Lombhalo** — Yemiphendvulo lemifisha yembhalo
   - **Lusuku** — Yekukhetsa tinsuku
   - **I-imeyili** — Yemakheli e-imeyili
   - **Inombolo Yelucingo** — Yekufaka inombolo yelucingo
   - **Kukhetsa Lokunyenti** — Yekukhetsa kusuka etinketfweni letihleliwe ngaphambilini
   - **Inkhokhelo** — Yekuqoqa tinkhokhelo
4. Faka **Sihloko** kanye ne-**Sichasiso** lesingakafanele kombuto.
5. Hlolisisa **Dzinga Imphendvulo** uma insimu iyimfuneko.
6. Chofota **Gcina**.
7. Phindza kute wengete lokunyenti kwemibuto.

:::warning
Luhlobo lwensimu ye-**Inkhokhelo** ludzinga kutsi iStripe ihleliwe. Uma ungakayihleli kunikela kwenkhulumo-mgca, buka [Kuhlela Kunikela Kwenkhulumo-mgca](../donations/online-giving-setup.md) ngaphambi kwekwengeta tinsimu tetinkhokhelo.
:::

## Kuphatsa Emalunga Efomu

1. Vula ifomu yakho bese uya kuthebhu le-**Emalunga**.
2. Sesha umuntfu bese umengeta ngesikhundla:
   - **Admin** — Angakwati kuhlela ifomu nekubona lokutfunyelwe konkhe.
   - **Kubuka Kuphela** — Angakwati kubona lokutfunyelwe kodvwa angeke ahlele ifomu.

## Kuphindzaphindza Ifomu

Kute usebentise kabusha ifomu njengendzawo yekucala yelensha, chofota luphawu lwe-**Kuphindzaphindza** (luphawu lwekukopisha) lolusedvutane nefomu eluhlwini Lemafomu. IB1 idala kopi lecondzile yefomu — kufaka yonkhe imibuto — loyingahlela bese uyishintjisa ligama ngekutimela.

:::tip
Kuphindzaphindza kuyasita emicimbini lephindzaphindzako lapho imibuto yekubhalisa isala ifana umnyaka nemnyaka. Phindzaphindza ifomu yanyalonyalo, buyeketa ligama netinsuku, bese ususukuma.
:::

## Kuhlela Tici Tefomu

Ungabuyeketa ligama lefomu yakho netilungiselelo nome nini. Kumafomu Lasimeleko, utawubuye ubone i-**URL yemphakatsi** lekhetsekile longayabelana nome ngubani.

:::tip
Emafomu Lasimeleko akahle kakhulu ekubhaliseni kwemicimbi. Yabelana nge-URL yemphakatsi nge-imeyili, kunkhulumo-mgca yemaphephandzaba, nobe ufake ifomu ngalokucondzile kusayithi lelibandla lakho.
:::

:::info
Kute ufake ifomu kusayithi lakho leB1, yiya kusihleli sesayithi lakho, wengete sigaba lesisha, bese ukhetsa sitselo se-**Fomu**. Bese ukhetsa ifomu lofuna kuyikhombisa. Buka [Kuphatsa Emakhasi](../website/managing-pages.md) kute utfole imininingwane yekuhlela sayithi lakho.
:::
