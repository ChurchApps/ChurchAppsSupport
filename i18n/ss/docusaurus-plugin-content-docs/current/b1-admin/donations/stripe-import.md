# Kungenisa kweStripe

<div class="article-intro">

Uma wemukela iminikelo yenkhulumo-mgca nge-Stripe, lithuluti Lekungenisa kweStripe likuvumela kutsi udvonse leto ticoceleco tingene ku-B1 Admin kuze yonkhe imininingwane yakho yeminikelo ihlale endzaweni yinye. Loku kusita kakhulu nawubamba noma nguticoceleco tini letingakahlanganiswanga ngekutitomatika.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Cedza [Kuhlela Kunikela Kwenkhulumo-mgca](online-giving-setup.md) kute uhlanganise i-akhawunti yakho yeStripe ku-B1 Admin
- Cinisekisa kutsi unayo iminikelo ku-idashbhodi yakho yeStripe yesikhatsi lofuna kusingenisa

</div>

## Kusebenta Kwaso

Inchubo yekungenisa isebentisa umsebenti lonetinyatselo letimbili: kucala ubuka kucala kutsi yini letawungeniswa, bese ungcina kungenisa. Le ndlela yekuhlola kucala ikuvumela kutsi uhlolisise konkhe ngaphambi kwekutsi kudalwe nome ngukuphi kwedatha.

## Kungenisa Ticoceleco

1. Ku-**B1 Admin**, yiya ku-**Iminikelo > Emaqoqo**.
2. Chofota sixibonono se-**Kungenisa kweStripe** lesisenzansi kwelikhasi Lemaqoqo.
3. Khetsa **sikhatsi** seticoceleco lofuna kutingenisa.
4. Chofota **Buka Ngaphambili** kute usebentise kuhlolwa kwekucala.

## Kuhlolisisa Umbono Wekucala

Umbono wekucala ukhombisa umcoceleko ngamunye lovela kuStripe kanye nesikhombisi sesimo:

- **Losha** -- lomcoceleko awukakangeniswa futsi utawufakwa uma uchubeka.
- **Sekungenisiwe** -- lomcoceleko sewukhona ku-B1 Admin futsi utawuyekwa.
- **Kwentiwaphi lokunye** -- lomcoceleko usariwe ngenca yelinye lizatfu (sib.e., kubuyiselwa kwemali nobe kukhokha lokungaphumelelanga).

Indzawo yesifinyeto lesetulu ikhombisa inani lelisonkhe leticoceleco esigabeni ngasinye kanye nemanani emali lahilelekile.

:::info
Sinyatselo sekubuka umbono wekucala asidali nome ngumaphi emarekhodi. Kuhlolwa kwekufundza kuphela kute ukwati kufakazela kutsi kutakwenteka njani ngaphambi kwekunikela sivumelwano.
:::

## Kucedza Kungenisa

1. Hlolisisa imiphumela yombono wekucala kanye nemanani esifinyeto.
2. Chofota **Ngenisa Lokusaphutfumako** kute ungenise tonkhe ticoceleco letiphawulwe njenge-**Zisha**.
3. Ngemuva kwekutsi kungenisa kucedvwe, tichipi tesimo lesiseceleni komcoceleko ngamunye tibuyeketa kute tikhombise umphumela.

## Emacebo Ekusebentisa Kungenisa kweStripe

- Sebentisa kungenisa njalonjalo (ngesonto nobe ngenyanga) kute ugcine emarekhodi akho asezingeni.
- Uma umcoceleko ukhombisa njenge-**Sekungenisiwe**, kusho kutsi B1 Admin sewunayo irekhodi lelifananako -- akukho sento lesidzingekako.
- Sebentisa sihlungi sesikhatsi kute ugcile esikhatsini lesitsite uma ufuna ticoceleco letitsite.

:::tip
Ngemuva kwekungenisa, vakashela likhasi le-[Emaqoqo](batches.md) kute uhlolisise kutsi iminikelo lengenisiwe ivela kahle nekutsi emanani ahambisana naloko lokubona ku-idashbhodi yakho yeStripe.
:::

## Tinyatselo Letilandzelako

- Cinisekisa kutsi iminikelo lengenisiwe yabelwe [etikhwameni](funds.md) letifanele kute [tincwadzi temnikelo](giving-statements.md) tibe ncinile
