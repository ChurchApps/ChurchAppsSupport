# Iminikelo

<div class="article-intro">

Sicele Seminikelo ku-B1 Admin sinika konkhe lokudzingekile kuphatsa kunikela kwelibandla lakho. Kusuka ekubhaleni iminikelo yangesonto kuya ekukhiqizeni tincwadzi tentela tekuphela komnyaka, tonkhe tithuluti takho teminikelo tihleliwe endzaweni yinye kuze ukwati kulawula timali telibandla lakho.

</div>

## Titselo Letincane

Utawutfola emakhasi lalandzelako kuliphaneli lelisecaleni Leminikelo:

### Sifinyeto

Likhasi le-**Sifinyeto** likunika umbono lobonakalako wawo onkhe umsebenti weminikelo. Likhombisa emadulo nemibiko lehlungiwe kuze ubone ngekushesha ematrendi eminikelo kulo lonkhe libandla lakho. Buka [Imibiko Yeminikelo](donation-reports.md) kute utfole imininingwane leyengeziwe ngekusebentisa tithuluti tesifinyeto nemibiko.

### Emaqoqo

**[Emaqoqo](batches.md)** akuvumela kuhlanganisa iminikelo ndzawonye -- sibonelo, wonkhe emasheke nemali lecashiwe leqoqiwe ngesikhatsi semnikelo weSonto. Liqoqo ngalinye lilandzelela inani laso leminikelo nelinani lelisonkhe kute kube lula kuhlolisisa.

### Tikhwama

**[Tikhwama](funds.md)** tihlukanisa iminikelo yakho ngenhloso yayo. Tibonelo letijwayelekile tifaka Sikhwama Lesijwayelekile, Sikhwama Sekwakha, kanye Nemishini. Ungaphawula sikhwama ngasinye njengalesehliswa entelweni nobe cha.

### Tincwadzi

Likhasi le-**[Tincwadzi](giving-statements.md)** likhiqiza tincwadzi teminikelo tekuphela komnyaka tebanikeli bakho. Ungadawuniloda emafayela e-CSV nga munye nobe uphrinte tonkhe tincwadzi kanyekanye.

### Kungenisa kweStripe

Uma wemukela iminikelo yenkhulumo-mgca nge-Stripe, lithuluti le-**[Kungenisa kweStripe](stripe-import.md)** likuvumela kutsi udvonse leto ticoceleco tingene ku-B1 Admin kuze konkhe kube kusistimu yinye.

## Kunikela Kwenkhulumo-mgca

B1 Admin ihlanganisa ne-**Stripe** ne-**PayPal** kute yemukele iminikelo yenkhulumo-mgca. Nase kuhleliwe, emalunga angakwati kunikela ngesayithi lakho leB1.church, futsi leyo minikelo ivele ngekutitomatika emarekhodini akho. Buka [Kuhlela Kunikela Kwenkhulumo-mgca](online-giving-setup.md) kute ucale.

:::tip
Uma usemusha ekunikeleni ku-B1 Admin, cala nge[kuhlela tikhwama takho](funds.md), bese [udala liqoqo lakho lekucala](batches.md) bese [ubhala iminikelo](recording-donations.md) kulo.
:::
