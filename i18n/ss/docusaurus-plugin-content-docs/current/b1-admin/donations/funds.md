# Kuphatsa Tikhwama

<div class="article-intro">

Tikhwama tikuvumela kuhlukanisa iminikelo ngenhloso yayo kuze ukwati kulandzelela linani lelamukelwa libandla lakho ngesigaba ngasinye senkonzo. Tikhwama letijwayelekile tifaka Sikhwama Lesijwayelekile, Sikhwama Sekwakha, Umishini, kanye neNkonzo Yebasha, kodvwa ungadala lokunyenti njengoba udzinga.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Utawudzinga kufinyelela esiceleni se-**Iminikelo** ku-B1 Admin
- Ncuma kutsi ngutiphi titselo tekunikela letidzingekako libandla lakho kanye nekutsi ngasinye kufanele sibe lesehliswa entelweni yini

</div>

## Likhasi Letikhwama

Yiya ku-**Iminikelo > Tikhwama** kute ubone luhlu lwato tonkhe tikhwama takho. Umugca ngamunye ukhombisa ligama lesikhwama kanye nesimo saso se-**lokwehliswa entelweni**, lokucondzisako kutsi iminikelo yesikhwama leso ivela yini ku[tincwadzi temnikelo](giving-statements.md) tekuphela komnyaka.

## Kudala Sikhwama Lesisha

1. Chofota **Engeta Sikhwama** esicongweni selikhasi.
2. Faka ligama lesikhwama (sib.e., "Sikhwama Sekwakha").
3. Hlolisisa libhokisi le-**Lokwehliswa Entelweni** uma iminikelo yalesikhwama kufanele ifakwe etincwadzini temnikelo.
4. Shiya **Kubonakala Kubanikeli** lihlolisisiwe kute ukhombise sikhwama emafomini emphakatsi nemalunga ekunikela, nobe uyisuse kute ufihle sikhwama kubanikeli ngesikhatsi usasebentisa kuminikelo lebhalwe ngesandla.
5. Chofota **Gcina**.

Sikhwama lesisha sivela ngekushesha eluhlwini lwakho futsi siyatfolakala nawubhala iminikelo.

## Kuhlela Nekususa Tikhwama

- Kute uhlele sikhwama, chofota inkinobho ye-**Hlela** lesedvutane naso. Ungashintja ligama, ushintje silungiselelo selokwehliswa entelweni, nobe ushintje **Kubonakala Kubanikeli** kute ukhombise nobe ufihle emafomini ekunikela.
- Kute ususe sikhwama, vula ifomu yekuhlela bese uchofota **Susa**.

:::warning
Kususa sikhwama akususi iminikelo leyayikadze yabelwe kuso. Loo marekhodi eminikelo asala kusistimu, kodvwa sikhwama singeke sisaba khona kuminikelo lemisha.
:::

:::info
Tikhwama letifihliwe kubanikeli tikhombisa sichipi se-**Kufihliwe** lesisedvutane negama lato kuloluhlu. Tikhwama letifihliwe tisasebenta kuminikelo lebhalwe ngesandla ku-B1 Admin -- tingaveli nje njengalithuba emafomini emphakatsi nobe emalunga ekunikela.
:::

## Kubuka Imininingwane Yesikhwama

Chofota **ligama lesikhwama** kute uvule likhasi lemininingwane yaso. Lelikhasi likhombisa:

- **Umlandvo weminikelo** -- luhlu lwayo yonkhe iminikelo yalesikhwama
- **Sihlungi sesikhatsi** -- yancisa imiphumela kusikhatsi lesitsite
- **Titatimende letihlanganisiwe** -- iminikelo yonkhe, banikeli labehlukene, kanye nelinani lelisonkhe

Lombono uyasita nawudzinga kubika ngemsebenti lotsite wekucoca imali nobe uhlole inchubekelo yesihloko sekucoca imali.

## Kuhlela Ngendlela Nekukhipha

- Sebentisa tihloko telikholomu kute uhlele tikhwama ngeligama.
- Chofota **Khipha ku-CSV** kute wehlise luhlu lwesikhwama sakho njengesipredishithi.

:::tip
Hlela tikhwama takho ngaphambi kwekutsi ucale [kubhala iminikelo phansi](recording-donations.md). Kutsi tonkhe tikhwama tibe khona kuvikela kutsi iminikelo ihlukaniswe kahle kusukela ekucaleni.
:::

## Tinyatselo Letilandzelako

Nase tikhwama takho tihleliwe, ungatikhomba nawu[bhala iminikelo](recording-donations.md) eliqoqweni. Cinisekisa kutsi uphawula tikhwama letifanele njengaletehliswa entelweni ngaphambi kwekukhiqiza [tincwadzi temnikelo](giving-statements.md).
