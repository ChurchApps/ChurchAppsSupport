# Imibiko Yeminikelo

<div class="article-intro">

B1 Admin ikunika tindlela letinyenti tekubuka nekuhlaziya imininingwane yeminikelo yelibandla lakho. Likhasi le-Sifinyeto Seminikelo linika umbono lobonakalako ngemadulo netihlungi, kuyilapho siceleni Semibiko sinikeza umbiko Wesifinyeto Seminikelo lonemininingwane lebanti kakhulu. Sebentisa letithuluti kute ulandzelele ematrendi eminikelo, ulungiselele imihlangano yebomeli, nobe uhlolisise tincwadzi takho.

## Idashbhodi Yeminikelo

**Idashbhodi Yeminikelo** yintfo yekucala loyibonako nawuchofota **Iminikelo** kuliphaneli lelisecaleni. Inika umbono losetulu wemsebenti wakho weminikelo kanye netincomo tekusebenta letibalulekile.

1. Yiya ku-**Iminikelo** kuliphaneli lelisecaleni kute uvule idashbhodi.
2. Etulu, emakhadi lamane e-**KPI** akhombisa titsi tekukala kweminikelo yakho ngekushesha:
   - **Sonkhe Kunikela** -- Linani lelisonkhe lelinikelwe esikhatsini lesikhetsiwe.
   - **Sipho Lesijwayelekile** -- Linani lelijwayelekile lemnikelo.
   - **Banikeli Labehlukene** -- Inani labantfu labehlukene labanikelile.
   - **Sonkhe Iminikelo** -- Inani lelisonkhe leminikelo nga munye.
3. Sebentisa **sishintjanisi sesikhatsi** kute ushintje emkhatsini kwe-**Ngesonto**, **Ngenyanga**, nobe **Ngekota**.
4. Ngephansi kwe-KPI, ledulo likhombisa ematrendi eminikelo esikhatsi lesikhetsiwe.
5. Chofota **Dawuniloda** kute ukhiphe ifayela le-CSV lenetinani letiphelele teminikelo.

## Likhasi Lesifinyeto Seminikelo

Likhasi le-**Sifinyeto** linika imininingwane leyengeziwe yeminikelo lehlangene.

1. Yiya ku-**Iminikelo** kuliphaneli lelisecaleni kute uvule likhasi Lesifinyeto.
2. Sebentisa **sihlungi sesikhatsi** kute ukhetse sikhatsi lofuna kusibuka. Beka lusuku loluphambili etulu nelusuku lolusandza kwenteka ngephansi.
3. Likhasi likhombisa idulo leminikelo yangesonto kuze ubone ematrendi ngekushesha.
4. Chofota **Dawuniloda** kute ukhiphe ifayela le-CSV lelinelinani lelisonkhe lelinikelwe, liviki lelinikelwe kulo, kanye nesikhwama lesinikelwe kuso.

:::info
Likhasi Lesifinyeto likhombisa imininingwane yeminikelo lehlangene. Alifaki emagama ebanikeli nga munye. Uma udzinga imininingwane yebanikeli nga munye, sebentisa likhasi le-[Emaqoqo](batches.md).
:::

## Kubuka Imininingwane Yebanikeli Nga Munye

Kute ubone kutsi ngubani lonikela, malini, naku sikhwama sini:

1. Yiya ku-**Iminikelo > Emaqoqo**.
2. Chofota **ligama leliqoqo** kute ulivule.
3. Likhasi lemininingwane yeliqoqo lidvwebe umnikelo ngamunye ngeligama lomniki, linani, sikhwama, lusuku, kanye nendlela yekukhokha.
4. Chofota **ligama lomniki** kute ubone imininingwane yekutsi banikele kangaki nekutsi ngiyoni koma nga koma.
5. Chofota ku-**ID yomnikelo** kute uvule iphaneli yaseceleni lenemininingwane yonkhe yalowo mnikelo nga munye.
6. Chofota **Dawuniloda** kute ukhiphe i-CSV lenelwati lonkhe lwebanikeli neminikelo yaleliqoqo.

## Umbiko Wesifinyeto Seminikelo

B1 Admin ibuye ifake nembiko we-**Sifinyeto Seminikelo** esiceleni Semibiko:

1. Chofota **Imibiko** kuliphaneli lelisecaleni.
2. Khetsa umbiko we-**Sifinyeto Seminikelo**.
3. Khetsa tihlungi takho (sikhatsi, sikhwama, likhempasi, njll.) bese usebentisa umbiko.

## Kukhipha Imininingwane

Ungakhipha imininingwane yeminikelo etindzaweni letinyenti:

- **Likhasi Lesifinyeto** -- dawuniloda i-CSV yemanani eminikelo yangesonto ngesikhwama
- **Likhasi lemininingwane yeliqoqo** -- dawuniloda i-CSV yeminikelo nga munye lenemininingwane yebanikeli
- **Likhasi lemininingwane yesikhwama** -- dawuniloda umlandvo weminikelo wesikhwama lesitsite

:::tip
Kutebiko yekuphela komnyaka, hlanganisa kukhishwa kwelikhasi Lesifinyeto nelithuluti le-[Tincwadzi Temnikelo](giving-statements.md) kute utfole kokubili ematrendi lahlangene kanye netincwadzi tebanikeli nga munye.
:::

## Tinyatselo Letilandzelako

- Khiqiza [Tincwadzi Temnikelo](giving-statements.md) tebanikeli bakho ekupheleni komnyaka
- Hlolisisa [emaqoqo](batches.md) nga munye kute ufakazele imininingwane yeminikelo
- Hlola emakhasi emininingwane ye[sikhwama](funds.md) kute ubone kwehlukaniswa kweminikelo ngeliphothiliyo
