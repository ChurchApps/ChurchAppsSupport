# Imikhankaso Netithembiso

<div class="article-intro">

Imikhankaso ikuvumela kutsi ucalise umkhankaso wekucoca imali ngenhloso lecondzile — sikhwama sekwakha, luhambo lwetithunywa, umsebenti lokhetsekile — futsi ulandzelele tithembiso letentiwa emalunga kanye neminikelo lengempela kute ubone inchubekelo yakho ngesikhatsi lesiphetsiwe.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Hlela [tikhwama tekunikela](funds) takho — umkhankaso ngamunye uhlotjaniswa nesikhwama
- Udzinga kufinyelela esiceleni Seminikelo se-B1 Admin

</div>

## Kuvula Imikhankaso

Ku-B1 Admin, yiya ku-**Iminikelo** kuliphaneli lelisecaleni bese ukhetsa **Imikhankaso**. Utawubona luhlu lwayo yonkhe imikhankaso kanye nelinani layo lelihlosiwe, sonkhe lesitsembisiwe, kanye nesonkhe lesenele kunikelwa.

## Kudala Umkhankaso

1. Chofota **Engeta Umkhankaso**.
2. Gcwalisa imininingwane yemkhankaso:
   - **Ligama** *(kuyadzingeka)* — ligama leliyakuboniswa lomkhankaso (sibonelo, "Sikhwama Sekwakha 2026").
   - **Sikhwama** — sikhwama sekunikela lesihlotjaniswe nalomkhankaso.
   - **Lusuku Lekucala** / **Lusuku Lekugcina** — sikhatsi sekucoca imali.
   - **Sihloko** — linani lemali lelihlosiwe lomkhankaso.
3. Chofota **Gcina**.

## Kulandzelela Inchubekelo

Ikhadi lomkhankaso ngamunye likhombisa:

- **Sihloko** — linani lakho lelihlosiwe
- **Kutsembisiwe** — linani lelisonkhe lelo emalunga lawutsembisile kunikela
- **Kunikelwe** — linani lelisonkhe lelinikelwe ngempela kusikhwama saloumkhankaso ngesikhatsi seliwindi lomkhankaso
- **Bha yenchubekelo** ekhombisa kutsi usufike kanjani eduze nesihloko sakho

Chofota nome nguwuphi umkhankaso kute uvule umbono wemininingwane, lodwebha tithembiso nga munye kanye nesimo saso sekugcwaliseka.

## Kwengeta Tithembiso

Tithembiso timibhaliso yemalunga lekunikela kumkhankaso. Kute ubhale sithembiso:

1. Vula umkhankaso.
2. Chofota **Engeta Sithembiso**.
3. Khetsa **umuntfu** lenta sithembiso.
4. Faka **linani** lesithembiso.
5. Ngesihlengeto, beka **lusuku** lwesibopho sesithembiso.
6. Chofota **Gcina**.

Tithembiso tivela kumininingwane yomkhankaso futsi tingeta ku-**Kutsembisiwe** kwesonkhe kwelikhadi lomkhankaso.

:::tip
Tithembiso tehlukile kuletinikelo letingempela. Sithembiso silandzelela sibopho; umnikelo ubhala sipho lesingempela. Kokubili kuboniswa kumkhankaso kuze ubone kutsi tithembiso tigcwaliseka kahle kangakanani.
:::

## Kuhlela nobe Kususa Umkhankaso

Chofota luphawu lwekuhlela kunome yikuphi ikhadi lomkhankaso kute ubuyekete ligama laso, sihloko, tinsuku, nobe sikhwama saso. Chofota **Susa** kute ususe umkhankaso netithembiso taso kuphela. Kususa umkhankaso akususi nome yimuphi umnikelo lowabhalwa esikhwameni saso.
