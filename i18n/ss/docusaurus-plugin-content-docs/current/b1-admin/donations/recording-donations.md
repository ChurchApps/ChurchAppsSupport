# Kubhala Iminikelo Phansi

<div class="article-intro">

Kubhala iminikelo phansi ku-B1 Admin kwentiwa nge-sistimu Yemaqoqo. Udala liqoqo kute limele kucoceleka (njengemnikelo weSonto), bese ufaka iminikelo nga munye kuleliyo liqoqo. Loku kugcina tincwadzi takho teminikelo tihlelekile futsi kulula kutihlolisisa.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Hlela [tikhwama](funds.md) takho kuze ukwati kwabela iminikelo etitsselweni letifanele
- Dala [liqoqo](batches.md) lelitawubamba iminikelo losacela kuyifaka
- Cinisekisa kutsi banikeli bakhona ku[luhlu lwebantfu](../people/adding-people.md) lwakho kuze ukwati kubatfola nawubhala iminikelo

</div>

## Kudala Liqoqo Nekwengeta Iminikelo

1. Ku-**B1 Admin**, chofota **Iminikelo** kuliphaneli lelisecaleni, bese uchofota **Emaqoqo**.
2. Chofota **Engeta Liqoqo**.
3. Faka ligama leliqoqo (sib.e., "Umnikelo weSonto - Jan 5") bese ukhetsa lusuku. Chofota **Gcina**.
4. Liqoqo lakho lelisha livela eluhlwini likhombisa iminikelo lelicace kutsi ayikho kanye ne-$0.00.
5. Chofota **ligama leliqoqo** kute ulivule.

## Kufaka Iminikelo Nga Munye

1. Ekhasini lemininingwane yeliqoqo, bhala ligama lomniki ku-**nsimu yekusesha** kute umtfole.
2. Ngemuva kwekukhetsa umuntfu, ifomu yekufaka umnikelo iyavela inetinsimu te-**Lusuku**, **Indlela Yekukhokha**, **Sikhwama**, **Linani**, kanye **Nenombolo Yesheke**.
3. Gcwalisa imininingwane bese uchofota **Engeta Umnikelo**.
4. Umnikelo wengetwa etafuleni lelingephansi, futsi ifomu iyabuyela endzaweni yayo yekucala kuze ukwati kufaka lolandzelako.

:::tip
Ungakwati kufaka iminikelo lemikhulu nga munye ngekushesha ngaphandle kwekushiya likhasi leliqoqo. Ifomu iyabuyela endzaweni yayo yekucala ngemuva kwekufaka nga munye kuze ukwati kuhamba etulu kwenqwaba yemasheke nobe temawrafu ngendlela lesheshako.
:::

## Kwehlukanisa Umnikelo Emkhatsini Wetikhwama Letinyenti

Ngaletinye tikhatsi umnikeli munye anikela kuletsikhwama tetinyenti kunkhokhelo yinye. Kute uphatse loku:

1. Chofota inkinobho ye-**Hlela** emugceni wemnikelo.
2. Efomini yekuhlela, ngeta emanani etikhwameni letahlukene. Linani lelisonkhe litawubalwa ngekutitomatika kusuka kumanani esikhwama nga sinye.
3. Chofota **Gcina** kute ubuyekete umnikelo.

:::info
Kwehlukanisa iminikelo emkhatsini wetikhwama kujwayelekile nawumniki abhala sheke sinye lesihloselwe tinhloso letinyenti, njengeSikhwama Lesijwayelekile kanye Nemishini.
:::

## Kuhlela nobe Kususa Iminikelo

Kute uhlele umnikelo, chofota inkinobho ye-**Hlela** emugceni waso eliqoqweni. Ungashintja lusuku, linani, sikhwama, indlela yekukhokha, nobe nome yiphi imininingwane lenye. Chofota **Gcina** nawucedza.

:::tip
Sihloko selikhasi leliqoqo sibuyeketa ngekutitomatika kute sikhombise inani lelisonkhe leminikelo kanye nelinani lemali lelihlangene njengoba wengeta nobe uhlela emarekhodi. Sebentisa loku kute uhlolisise ngekumelene neliphepha lakho lekudiposita.
:::
