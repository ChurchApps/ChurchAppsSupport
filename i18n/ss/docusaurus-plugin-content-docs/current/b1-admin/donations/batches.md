# Emaqoqo Emnikelo

<div class="article-intro">

Emaqoqo ahlanganisa iminikelo yakho ndzawonye kuze kube lula kuyilandzelela nekuyihlolisisa. Liqoqo lelijwayelekile limele kucoceleka lokukodvwa, njengemnikelo weSonto nobe umcimbi lokhetsekile. Kusebentisa emaqoqo kukusita kuhlela kahle futsi kwenta kube lula kufakazela kutsi tincwadzi takho tiyahambisana nemadipositi lawenteka ngempela.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Cinisekisa kutsi sewentile [kuhlela tikhwama takho](funds.md) kuze titfole nawubhala iminikelo phansi
- Utawudzinga kufinyelela esiceleni se-**Iminikelo** ku-B1 Admin

</div>

## Likhasi Lemaqoqo

Nawuya ku-**Iminikelo > Emaqoqo**, utawubona luhlu lwawo onkhe emaqoqo akho. Umugca ngamunye ukhombisa:

- **Ligama** -- lilebula lowaliniketa liqoqo
- **Lusuku** -- lusuku lwekucoceleka
- **Iminikelo** -- inani leminikelo lengiyo ekuqoqweni
- **Sonkhe** -- linani lemali lelihlangene

Sihloko lesisetulu sikhombisa titatimende letihlanganisiwe letifaka phakathi inani lelisonkhe lemaqoqo, inani lelisonkhe leminikelo kuwo onkhe emaqoqo, kanye nelinani lemali lelisonkhe.

## Kudala Liqoqo Lelisha

1. Chofota **Engeta Liqoqo** esicongweni selikhasi.
2. Faka ligama leliyichaza (sib.e., "Umnikelo weSonto - Feb 9").
3. Khetsa lusuku lwekucoceleka.
4. Chofota **Gcina**.

Liqoqo lakho lelisha liyavela eluhlwini, seliyakulungela kutsi ufake iminikelo kulo.

## Kusebenta Nemaqoqo

- **Buka iminikelo** -- chofota ligama leliqoqo kuze ulivule futsi ubone yonkhe iminikelo lengiyo. Kusuka lapho ungangeta, uhlele, nobe ususe iminikelo.
- **Hlela imininingwane yeliqoqo** -- chofota inkinobho ye-**Hlela** emugceni weliqoqo kuze ushintje ligama nobe lusuku lwalo.
- **Hlela ngendlela** -- sebentisa tihloko telikholomu kute uhlele emaqoqo ngeligama nobe ngelusuku.
- **Khipha** -- chofota **Khipha ku-CSV** kute wehlise luhlu lwemaqoqo akho njengesipredishithi.

:::tip
Niketa emaqoqo akho emagama langashintjishintji kuze kube lula kuwatfola nasamuva. Kufaka lusuku netinhlobo takho tekucoceleka (sib.e., "Sonto ekuseni - 2025-02-09") kugcina luhlu lwakho luhleliwe njengoba lukhula.
:::

## Tinyatselo Letilandzelako

Nase unaliqoqo, buka [Kubhala Iminikelo Phansi](recording-donations.md) kute ufundze kutsi ungenta njani kufaka iminikelo nga munye kulo. Ungabuye [ungenise ticoceleco teStripe](stripe-import.md) kute wente emaqoqo ngekutitomatika kusuka ekunikeleni kwenkhulumo-mgca.
