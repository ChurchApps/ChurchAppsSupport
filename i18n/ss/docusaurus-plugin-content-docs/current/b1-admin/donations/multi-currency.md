# Kwesekelwa Kwetinhlobo Letinyenti Temali

<div class="article-intro">

Sici seB1 setinhlobo letinyenti temali sivumela libandla lakho kwemukela nekulandzelela iminikelo ngetinhlobo letahlukene temali. Loku kusita kakhulu emabandleni lanemalunga emave lahlukene, tithunywa, nobe amakhempasi lamanyenti emaveni lahlukene.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga imvumo yekuphatsa iminikelo. Buka [Tikhundla Netimvumo](../people/roles-permissions.md) kute utfole imininingwane.
- Hlela [kunikela kwakho kwenkhulumo-mgca](./online-giving-setup.md) nge-Stripe, lesekela ticoceleco tetinhlobo letinyenti temali.
- Cala imidzingo yelibandla lakho yekugcinwa kwetincwadzi mayelana nekuphatsa tinhlobo letinyenti temali.

</div>

## Kuvula Tinhlobo Letinyenti Temali

Kwesekelwa kwetinhlobo letinyenti temali sekuvuliwe njengekumiswa lokujwayelekile ku-B1. Nasekuvuliwe:

- Emalunga angakwati kunikela ngemali yesive sabo nabanikela kwenkhulumo-mgca
- Ungabhala iminikelo ngesandla nganoma yiluphi luhlobo lwemali
- Imibiko yeminikelo ikhombisa emanani ngeluhlobo lwayo lwekucala
- IStripe iphatsa kushintjwa kwemali ngekutitomatika kunikela kwenkhulumo-mgca

## Tinhlobo Temali Letisekelwako

Sistimu isekela tonkhe tinhlobo temali temhlaba letinkhulu, letifaka:

- **USD** -- Dolela weMelika
- **EUR** -- Euro
- **GBP** -- Phawundi lesiNgisi
- **CAD** -- Dolela weCanada
- **AUD** -- Dolela we-Australia
- **MXN** -- Peso yeMexico
- **BRL** -- Real yeBrazil
- **INR** -- Rupee yeNdiya
- **CNY** -- Yuan yeShayina
- **JPY** -- Yen yeJapan
- Naletinyenti letinye...

Tinhlobo temali letikhona kunikela kwenkhulumo-mgca titsembela etinhlotini letisekelwa yi-akhawunti yakho yeStripe.

## Kubhala Iminikelo Ngetinhlobo Letahlukene Temali

### Iminikelo Yenkhulumo-mgca

Nalilunga linikela ngenkhulumo-mgca nge-Stripe:

1. Bakhetsa luhlobo lwemali labalitsandzako ekukhokheni
2. IStripe icubungula inkhokhelo ngaloluya hlobo lwemali
3. Umnikelo ubhalwa ku-B1 ngelinani loluhlobo lwemali lwekucala
4. IStripe iphatsa ngekutitomatika nome yikuphi kushintjwa lokudzingekile lwemali kuya kuluhlobo lwemali lolujwayelekile lwe-akhawunti yakho

### Kufaka Ngesandla

Kute ubhale umnikelo wemali lecashiwe nobe wesheke ngaluhlobo lwemali loluhlukile:

1. Yiya ku-**Iminikelo** ku-B1 Admin
2. Chofota **Engeta Umnikelo**
3. Khetsa luhlobo lwemali kuluhlu lolwehlako lwetinhlobo temali
4. Faka linani ngaloluya hlobo lwemali
5. Cedza lokunye kwemininingwane yemnikelo
6. Chofota **Gcina**

## Kubuka Iminikelo Yetinhlobo Letinyenti Temali

### Imibiko Yeminikelo

Imibiko yeminikelo ikhombisa emanani ngetinhlobo tayo temali tekucala:

- Emarekhodi eminikelo nga munye akhombisa likhodi lwemali (sib.e., "$100.00 USD")
- Emanani lelisonkhe abaliwa ngaluhlobo ngalunye lwemali
- Ungahlungela ngetinhlobo letitsite temali

### Tincwadzi Temnikelo

Nakukhiqizwa tincwadzi temnikelo:

- Umnikelo ngamunye uvela ngeluhlobo lwawo lwemali lwekucala
- Emanani lelisonkhe ahlukaniswa ngaluhlobo lwemali
- Emalunga abona ncamashi kutsi banikeleni kuluhlobo ngalunye lwemali

## Kuhlanganiswa NeStripe

Kunikela kwenkhulumo-mgca, iStripe iphatsa ticoceleco tetinhlobo letinyenti temali:

- **Kushintjwa ngekutitomatika** -- iStripe ishintja tinhlobo temali kuya kuloluya lolujwayelekile lwe-akhawunti yakho
- **Emarengo ekushintjaniswa** -- iStripe isebentisa emarengo ekushintjaniswa lakhona njengamanje emakethe
- **Tindleko** -- kushintjwa kwemali kungangeta tindleko letengetiwe teStripe
- **Luhlobo lwemali lwekukhokhwa** -- Timali tifakwa ku-akhawunti yakho ngaluhlobo lolujwayelekile lwemali

:::info
Hlola idashbhodi yakho yeStripe kute ubone emarengo ekushintjaniswa lakhona manje kanye nome yiziphi tindleko letihambisana neticoceleco tetinhlobo letinyenti temali.
:::

## Tintfo Letifanele Kucatjangelwa Ngekugcina Tincwadzi

Nawusebenta ngetinhlobo letinyenti temali:

- **Kugcina emarekhodi** -- Landzelela emanani nemihlobo yemali yeminikelo yekucala kute utfole imibiko lecinile
- **Emarengo ekushintjaniswa** -- Caphela kutsi emarengo ekushintjaniswa eStripe angahlukaniswa naloawenkhulu wakho
- **Emarisidi entela** -- Buta umbali wakho mayelana nekutsi ungayibika njani iminikelo yetinhlobo letahlukene temali ngenhloso yentela
- **Kwabelwa kwesikhwama** -- Ungakwati kwabela iminikelo kutikhwama letitsite kungakhatsateki luhlobo lwemali

## Tindlela Letincono

- **Luhlobo lwemali lolujwayelekile** -- Beka luhlobo lwemali lwelibandla lakho lolusetulu njengalolujwayelekile ekuningini kweticoceleco
- **Kukhulumisana lokucacile** -- Tjela banikeli ngaluphi luhlobo lwemali labanikela ngalo esikhatsini sekukhokha
- **Kubika lokuhambisana** -- Ncuma nawukubona kutsi ubike ngetinhlobo tekucala noma ushintje kuluhlobo lunye lwemali tetifinyeto
- **Kuhlolisisa njalonjalo** -- Hlolisisa tinkhokhelo teStripe netincwadzi tebumnikelo bakho, ubhale kwehlukaniswa kwemali

## Imikhawulo

- Kushintjwa kwemali kuphatswa yiStripe kunikela kwenkhulumo-mgca kuphela
- Iminikelo lebhalwe ngesandla ibhalwa njengoba ifakiwe kungeko kushintjwa ngekutitomatika
- Imibiko yesikhatsi lesendlulile ikhombisa iminikelo ngetinhlobo tayo tekucala temali
- Kubalwa kwemanani lelisonkhe kwentiwa ngaluhlobo ngalunye lwemali, hhayi emkhatsini wato tonkhe
