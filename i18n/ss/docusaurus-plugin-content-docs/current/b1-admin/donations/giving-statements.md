# Tincwadzi Temnikelo

<div class="article-intro">

Ekupheleni komnyaka ngamunye, banikeli bakho badzinga sifinyeto seminikelo yabo lengumtselo lokwehliswa entelweni kute basigcine emarekhodini abo. B1 Admin yenta kube lula kukhiqiza letincwadzi tebanikeli bonkhe kanyekanye, kukugcinela emahora lamanyenti emsebenti lowentiwa ngesandla.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Cinisekisa kutsi [tikhwama](funds.md) takho tiphawulwe kahle njenge-**Lokwehliswa Entelweni** -- kuphela iminikelo yetikhwama letehliswa entelweni ivela etincwadzini
- Cinisekisa kutsi yonkhe iminikelo [ibhaliwe](recording-donations.md) nekutsi noma ngutiphi ticoceleco tenkhulumo-mgca [tingenisiwe kusuka kuStripe](stripe-import.md)

</div>

## Kufinyelela Tincwadzi Temnikelo

1. Ku-**B1 Admin**, chofota **Iminikelo** kuliphaneli lelisecaleni.
2. Chofota **Tincwadzi**.

## Kukhiqiza Tincwadzi

1. Khetsa **umnyaka** kuluhlu lolwehlako esicongweni selikhasi. Ungakhetsa umnyaka wanyalo nobe nome yimuphi wemnyaka lesihlanu leyendlulile.
2. Likhasi likhombisa titatimende letihlanganisiwe taloyo mnyaka, letifaka:
   - **Banikeli labasonkhe** -- inani labantfu labanikelile
   - **Iminikelo lesonkhe** -- inani lemarekhodi eminikelo nga munye
   - **Linani lelisonkhe** -- linani lemali lelihlangene lakho konkhe kunikela

## Kudawuniloda Tincwadzi

Unetindlela letimbili tekutfumela tincwadzi kubanikeli bakho:

### Dawuniloda Njengemafayela e-CSV

Chofota **Dawuniloda i-ZIP** kute udawunilode ifayela le-ZIP leliphetse ifayela le-CSV nga munye lomniki nga munye. Loku kuyasita uma ufuna kutfumela tincwadzi nga munye nge-imeyili nobe kutingenisa kusinye sistimu.

### Phrinta Tonkhe Tincwadzi

Chofota **Phrinta Tonkhe** kute uvule umbono lophrintekako wencwadzi yemniki nga munye ku-brawuza yakho. Kusuka lapho, sebentisa umsebenti wekuphrinta webrawuza yakho kute utitfumele ku-phrinta. Incwadzi nga munye icala ngelikhasi lelisha kuze titolungiselelwa kusongwa nekutfunyelwa.

:::tip
Sebentisa tincwadzi takho ekucaleni kwaJanuary ngesikhatsi tincwadzi tisenhle. Hlola kabili kutsi tikhwama takho tiphawulwe kahle njengaletehliswa entelweni ngaphambi kwekukhiqiza tincwadzi -- kuphela iminikelo yetikhwama letehliswa entelweni ifakwa.
:::

:::info
Tincwadzi Temnikelo tifaka kuphela iminikelo lenikelwe kutikhwama letinesilungiselelo se-**Lokwehliswa Entelweni** lesivuliwe. Uma sikhwama singaphawulwanga njengalesehliswa entelweni, iminikelo yaso ngeke ivele encwadzini. Ungaphatsa lesilungiselelo ekhasini le-[Tikhwama](funds.md).
:::

## Tinyatselo Letilandzelako

Uma udzinga kuhlolisisa imininingwane yeminikelo ngaphambi kwekukhiqiza tincwadzi, vakashela likhasi le-[Imibiko Yeminikelo](donation-reports.md) nobe uhlole [emaqoqo](batches.md) nga munye.
