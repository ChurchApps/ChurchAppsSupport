# Tilungiselelo teLibandla

<div class="article-intro">

Likhasi le-Church Settings yindzawo lapho ulungisa khona imininingwane lesisekelo yelibandla lakho, imininingwane yekutsintsana, kanye nekuvezwa kwesihloko sakho (branding). Le mininingwane isetjentiswa kuto tonkhe tinsita te-ChurchApps, kuhlanganise neliwebhusayithi lakho le-B1.church kanye ne-app ye-B1 Mobile.

</div>

<div class="prereqs">
<h4>Before You Begin</h4>

- Udzinga imvumo lelitsi "Edit Church Settings". Buka [Roles & Permissions](./roles-permissions.md) uma ungakabinawo lelifikelelo.
- Yenta kutsi likheli lelibandla lakho, imininingwane yekutsintsana, kanye nelogo kube kulungile

</div>

## Kuhlela Imininingwane Yelibandla Lakho

1. Yendlulela ekhasini le- **Settings** kulisayidbha lengesencele.
2. Chofota inkinobho le- **Edit Settings** encenyeni lengetulu.
3. Buyeketa noma nguyiphi yalamasikhala lalandzelako:
   - **Church Name** -- Ligama leliboniswa kuto tonkhe timikhicito te-ChurchApps.
   - **Address** -- Likheli lelibandla lakho langempela.
   - **Contact Information** -- Inombolo yelucingo, imeyili, kanye nemininingwane lenye yekutsintsana.
4. Chofota **Save** kute usebentise tinshintjo takho.

## Kulungisa i-Subdomain Yakho

Libandla lakho litfola i-subdomain yamahhala ku- **yourchurch.b1.church**. Leli likheli lewebhu lapho emalunga netivakashi tingafinyelela khona kubukhona belibandla lakho ku-inthanethi.

1. Ekhasini le-Settings, tfola sikhala se- **Subdomain**.
2. Faka i-subdomain loyitsandzako (sibonelo, "gracechurch" ye-gracechurch.b1.church).
3. Gcina tinshintjo takho.

:::info
I-subdomain yakho kufanele ibe ngeyakhona wedvwa kuwo onkhe emabandla e-ChurchApps. Uma ligama lelitsandzako selisatfwe, zama kungeta lidolobha lakho nome sifundza (sibonelo, "gracechurch-dallas").
:::

## Kulungisa i-Branding

Lungisa kutsi libandla lakho livela njani kuto tonkhe tinsita te-ChurchApps:

1. Layisha i- **church logo** yakho ngekuchofota indzawo yelogo bese ukhetsa ifayela lesitfombe.
2. Ngeta noma yimuphi **church images** letengeziwe letisetjentiswa kuliwebhusayithi lakho kanye [nase-app yeselula](./mobile-app.md).

:::tip
Kute utfole imiphumela lemihle kakhulu, sebentisa ilogo lenesizinda lesicondzekako (transparent) kusifomeni se-PNG. Loku kucinisekisa kutsi ibukeka kahle kuletinsika letikhanyako naletimnyama.
:::

## Kungenisa Nekukhipha (Import and Export)

Inkinobho le- **Import/Export** encenyeni lengetulu yeSettings ivula insita lekhetsekile kuwindi lensha ye-browser. Sebentisa loku kute:

- Ungenise imininingwane yemalunga lisuka kuhlelo lolunye lekuphatsa emabandla.
- Ukhiphe imininingwane yakho ye-ChurchApps ngenhloso yekugcina sipele nome kutfutfusa.

Loku kusita kakhulu ngesikhatsi ucala kulungisa libandla lakho futsi udzinga kutfutfusela emarekhodi lasekhona ku-ChurchApps.

:::warning
Nawungenisa imininingwane, njalo gcina sipele semarekhodi akho lasekhona kucala. Kungenisa imininingwane kungeta imininingwane kuhlelweni lakho futsi kungadala emarekhodi lafanako uma kwentiwa kaningi.
:::
