# Tikhundla & Timvumo

<div class="article-intro">

Tikhundla (Roles) tikuvumela kutsi ulawule kutsi bantfu labehlukene bangafinyelela kuphi ku-akhawunti yakho ye-ChurchApps. Ungakhona kudala tikhundla takho letikhetsekile tetisebenti, tivolontile, kanye nalamanye emalunga ecembu, ngayinye inelizinga layo lelifikelelo lekugcina imininingwane yakho iphephile.

</div>

<div class="prereqs">
<h4>Before You Begin</h4>

- Udzinga lifikelelo le-Domain Admin nome sikhundla lesinemvumo yekuphatsa tikhundla
- Yenta luhlu lwemalunga ecembu kanye netindzawo ladzinga kutifinyelela
- Buka tigaba temvumo letikhona ngentasi kute ulungisele tikhundla takho

</div>

## Kufinyelela Kutikhundla

1. Yendlulela ekhasini le- **Settings** kulisayidbha lengesencele.
2. Chofota inkinobho le- **Roles** encenyeni lengetulu.
3. Likhasi le-Roles liboniswa tonkhe tikhundla letisatfwe manje telibandla lakho.

## Kucondza Likhasi Le-Roles

Likhasi le-Roles lehlukaniswe kutincenye letimbili:

- **Ngesencele** -- Kukhomba luhlu lwemalunga labekelwe sikhundla lesikhetsiwe.
- **Ngesekudla** -- Kukhomba tilungiselelo temvumo longatilungisa kuleso sikhundla.

Chofota liligama lanoma ngusiphi sikhundla kute ubone futsi uphatse emalunga awo netimvumo tawo.

## Kungeta Basebentisi Kusikhundla

1. Khetsa sikhundla lofuna kungeta emalunga kuso.
2. Sebentisa **sikhala sekusesha** ngesencele kute utfole umuntfu lofuna kumungeta.
3. Khetsa lomuntfu kuya emiphumeleni yekusesha.
4. Batawungetwa kulesikhundla ngalesikhatsi lesincane.

## Kususa Basebentisi Kusikhundla

1. Khetsa sikhundla lesinalowo muntfu lofuna kumususa.
2. Tfola lomuntfu kuluhlu lwemalunga ngesencele.
3. Chofota inkinobho le- **remove** eceleni kweligama lakhe.

## Kulungisa Timvumo

Ngayinye sikhundla singaphiwa lifikelelo kutindzawo letitsite te-B1 Admin. Timvumo tihlelwe ngetigaba:

- **People** -- Lifikelelo kudirekthri yemalunga naku-emarekhodi emuntfu ngamunye.
- **Donations** -- Lifikelelo kumarekhodi etimnikelo naku-kuphatfwa kwetimali.
- **Attendance** -- Lifikelelo kukulandzelela kuya kanye nemibiko.
- **Content** -- Lifikelelo kuliwebhusayithi naku-kuphatfwa kwekucuketfwa.
- Naletinye tindzawo njengobe tikhona.

Sebentisa emabhokisi ekukhetsa ngesekudla kwelikhasi le-Roles kute uvumele nome ucishe lifikelelo kuyo yonkhe indzawo.

:::warning
**Domain Admins** banelifikelelo lelugcwele kuto tonkhe tindzawo te-akhawunti yakho ye-ChurchApps. Timvumo tabo atikwati kushintjwa nome kunciphiswa. Sebentisa lesikhundla kubaphatsi bakho labatsembekile kakhulu kuphela.
:::

:::tip
Dala tikhundla letikhetsekile njenge "Treasurer" lokunelifikelelo le- **Donations** kuphela, nome "Check-In Volunteer" lokunelifikelelo le- **Attendance** kuphela. Loku kulandzela umtsetfo wekuniketwa lifikelelo lelincane kakhulu (least privilege) futsi kugcina imininingwane yakho iphephile. Buka [Data Security](./data-security.md) kute ufundze kabanti ngendlela i-ChurchApps evikela ngayo imininingwane yakho.
:::
