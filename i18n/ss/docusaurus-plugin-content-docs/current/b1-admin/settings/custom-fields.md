# Emasikhala Lakhetsile

<div class="article-intro">

**Emasikhala Lakhetsile (Custom Fields)** akuvumela kutsi ulandzelele imininingwane yakho ngamunye kuwo onkhe umrekhodi wemuntfu — tintfo B1 lengakabinaso sikhala lesakhelwe sona, njengelilanga lekuphela kwesikhatsi sekuhlolwa kwemlandvo, sayizi yebhatje, nome simo se-baptism class. Uchaza sikhala kanye ku-Settings, bese ugcwalisa linani kuphrofayili yemuntfu ngamunye bese usesha nome udala luhlu ngaso. Loku kutsatsela indzawo yendlela yakadzeni yekudala ifomu ye-People kuphela kute kugcinwe incenye yinye yemininingwane lekhetsekile.

</div>

<div class="prereqs">
<h4>Before You Begin</h4>

- Udzinga imvumo ye- **People** edit kute uchaze emasikhala futsi ugcwalise emanani, kanye nelifikelelo endzaweni ye-Settings. Nome ngubani lonemvumo ye-People view angabona emanani. Buka [Roles & Permissions](./roles-permissions.md).
- Nquma kutsi yini lofuna kuyilandzelela kanye nekutsi luphi luhlobo lolufanele kahle (umbhalo, linani, lilanga, imphendvulo lengu-ye/cha, nome luhlu lwekukhetsa) ngaphambi kwekutsi ucale.

</div>

## Kuvula Emasikhala Lakhetsile

Ku-B1 Admin, yendlulela ku- **Settings** kulisayidbha lengesencele bese ukhetsa likhadi le- **Custom Fields**. Ungakwati futsi kuya khona ngco ku- **/settings/custom-fields**. Utawubona luhlu lwato tonkhe emasikhala lowachazile, kukhombisa **Name** ne **Field Type** yawo. Uma ungakadali nakunye, iphaneli ifundzeka itsi *"No custom fields have been added yet."*

## Kungeta Sikhala

1. Chofota **Add Field**.
2. Kusihleli lesivulekako ngesekudla, faka **Name** — leli libhalo lelitawubonwa titsebenti emaphrofayilini ebantfu nasekuseshweni (sibonelo, *Background check expires*).
3. Khetsa **Field Type**:
   - **Textbox** — umbhalo lomfisha longahleliwe.
   - **Whole Number** — emanani langenato ema-decimal (sibonelo, sibalo).
   - **Decimal** — emanani langaba ne-decimal.
   - **Date** — lilanga lekhalendasi.
   - **Yes/No** — imphendvulo lelula yengu-ye/cha.
   - **Multiple Choice** — luhlu lwekukhetsa. Nawukhetsa lolu hlobo, kuvela **sihleli setinketfo** kute ungete inketfo ngayinye lebantfu bangayikhetsa.
4. Chofota **Save**.

Sikhala manje sikhona kuphrofayili yemuntfu ngamunye.

:::info
Tinhlobo tesikhala tifana naletisetjentiswa [emibutweni yemafomu](../forms/creating-forms.md), ngako emanani asebenta ngendlela lefanako kuyo yonkhe i-B1.
:::

## Kuhlela Sikhala

Chofota noma ngumuphi umugca wesikhala kuluhlu kute uvule kabusha kusihleli. Shintja ligama, luhlobo, nome tinketfo bese uchofota **Save**.

:::warning
Kushintja **Field Type** yesikhala lesesinemanani (sibonelo, kusuka ku-Textbox kuya ku-Date) kungasala emanani langenele langakasafanani nesimo lesisha. Shintja tinhlobo ngenakekelo nase titsebenti sekucele tagcwalisa sikhala.
:::

## Kususa Sikhala

Vula sikhala kute usihlele bese uchofota **Delete**. Utawubutwa kucinisekisa: *"Are you sure you wish to delete this custom field? Its stored values will also be removed."* Kususa sikhala kukukhipha ngalokuphelele **kanye nalinye linani lelingigciniwe ngaso** kubo bonkhe bantfu — loku akukwati kubuyiselwa emuva.

## Kugcwalisa Emanani Kumuntfu

Nakukhona kungenani sikhala sinye lesikhona, emanani aso ahlala eceleni kwemininingwane leyakhelwe endzaweni yayo kumrekhodi wemuntfu ngamunye — uyawabona ku- **Personal Details** futsi uwahlela kufomu lelifana nalelisetjentiswa yeminye imininingwane yemuntfu. Akukho lokungetiwe uze uchaze sikhala sakho sekucala.

1. Vula umrekhodi wemuntfu ku- **People**.
2. Esigabeni se- **Personal Details**, chofota inkinobho ye- **Edit** (ipensela).
3. Yehlela phansi uye endzaweni ye- **Custom Fields** ekugcineni kwefomu lekuhlela bese ugcwalisa linani sikhala ngasinye. Sikhala ngasinye sikhombisa lokufakwako lokuhambelana nesimo saso — sikhetsi selilanga kumasikhala e-Date, i-dropdown yengu-ye/cha kumasikhala e-Yes/No, luhlu lwekukhetsa ku-Multiple Choice, njll.
4. Chofota **Save**. Emanani akho emasikhala lakhetsile agcinwa kanyekanye nalokunye kokungenani lokusemuntfu.

Nasekubuyela ephrofayilini, sikhala ngasinye lesinelinani manje sibonakala esigabeni se- **Personal Details** (imiphendvulo yengu-ye/cha ifundzeka njenge *Yebo* nome *Cha*, futsi i-Multiple Choice ikhombisa libhalo lenketfo). Emasikhala lasele engenalutfo ayafihlwa nje. Kute ususe linani, hlela umuntfu, sula sikhala, bese ugcina — linani lelingenalutfo liyakhitjwa kumrekhodi esikhundleni sekugcinwa lingenalutfo.

:::tip
Sibonelo lesijwayelekile ngukuphepha kwetivolontile: dala sikhala se- **Date** lesitsi *Background check expires*, bhala lilanga lelivolontile ngalinye, bese udala [Saved List](../people/lists.md) leyekhomba nome ngubani lelilanga lakhe seliphelile.
:::

## Kusesha Nekwakha Luhlu Ngemasikhala Lakhetsile

Emasikhala lakhetsile ayasesheka ngalokuphelele:

1. Ekhasini le- **People**, vula i- [Advanced Search](../people/searching-people.md).
2. Vula sigaba se- **Custom Fields**.
3. Hlola sikhala lofuna kuhlunga ngaso, khetsa umsebenti wekuhlunga, bese ufaka linani. Imisebenti yekuhlunga leniketwako ihambisana neluhlobo lwesikhala:
   - **Textbox** — kucuketse, kulingana, kucala nge, kuphetsa nge.
   - **Whole Number / Decimal** — kulingana, kukhulu kunaleli, kukhulu kunaleli nome kulingana naleli, kuncane kunaleli, kuncane kunaleli nome kulingana naleli.
   - **Date** — kulingana, ngemuva kwaleli (kukhulu kunaleli), ngaphambi kwaleli (kuncane kunaleli).
   - **Yes/No** — kulingana neYebo nome Cha.
   - **Multiple Choice** — kulingana nome kucuketse yinye yetinketfo.

Gcina noma ngusiphi sesho sesikhala lesikhetsile njenge [List](../people/lists.md). Luhlu ngumbuto lophilako, ngako luhlu loluhlelwe nge *Background check expires is before today* luyaphindze luhlole umuntfu ngamunye ngaso sonkhe sikhatsi luvulwa — akukho kunakekela ngesandla.

## Kwenteka Nakuhlanganiswa (Merge)

Nawe [uhlanganisa emarekhodi lamabili emuntfu](../people/adding-people.md), emanani emasikhala lakhetsile ayatfutfuseleka ngokutentekela. Umuntfu lomgcinako uhlala nalamanani akhe; kumasikhala lapho kwakunemininingwane kumuntfu losusiwe kuphela, lelo linani lidluliselwa kuze kungabikho lokulahleka.
