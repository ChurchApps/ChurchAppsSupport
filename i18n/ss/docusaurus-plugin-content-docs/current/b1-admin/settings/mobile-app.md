# Tilungiselelo Te-App Yeselula

<div class="article-intro">

Likhasi le-Mobile App Settings likuvumela kutsi ulungise ema-tab wekuzulazula (navigation tabs) laboniswa ku- **B1.church mobile experience (PWA)** yemalunga elibandla lakho. Ulawula kutsi ngutiphi tab letibonakalako, kutsi tikhomba kuphi, kanye nekutsi tiboniswa njani.

</div>

:::info I-app ye-B1 Mobile langempela isayilelwe ekupheleni
Ema-tab lalungiswa lapha aletfwa nge- [B1.church Progressive Web App (PWA)](/docs/b1-church/getting-started/installing-pwa), leseyisikhundleni se-app ye-B1 Mobile langempela. Emalunga angafaka i-PWA kusikrini lekhaya (home screen) yawo kusuka kunoma ngukuphi likhasi ku-`https://yourchurchname.b1.church`, esikhundleni sekulanda i-app yakadzeni ku-App Store nome ku-Google Play.
:::

<div class="prereqs">
<h4>Before You Begin</h4>

- Udzinga imvumo lelitsi "Edit Church Settings". Buka [Roles & Permissions](./roles-permissions.md) uma ungakabinawo lelifikelelo.
- Lungisa kucala [Church Settings](./church-settings.md) yakho, kufaka ekhatsi ligama lelibandla lakho kanye ne-branding

</div>

## Kufinyelela Kutilungiselelo Te-App Yeselula

1. Yendlulela ekhasini le- **Settings** kulisayidbha lengesencele.
2. Chofota inkinobho le- **Mobile Apps** encenyeni lengetulu.
3. Likhasi le-Mobile App Settings liboniswa ema-tab akho lasekhona.

## Kungeta i-Tab Lensha

1. Chofota inkinobho le- **Add Tab** engetulu lekhasi.
2. Gcwalisa imininingwane yetab:
   - **Name** -- Libhalo leliboniswa etabini (sibonelo, "Sermons" nome "Give").
   - **Icon** -- Chofota sikhetsi selithemba (icon) kute ukhetse i-icon yetab yakho. Ungakwati kulayisha nesitfombe sakho.
   - **Tab Type** -- Khetsa kuletinketfo letinjenge Bible, Live Stream, Donation, Website, nakunye lokunyenti.
   - **URL** -- Faka likheli lewebhu lapho itab lifanele likhombe khona.
   - **Visibility** -- Lawula kutsi ngubani longabona letab (bonkhe bantfu, emalunga kuphela, njll.).
3. Chofota **Save Tab** kute uyingete ku-app yakho.

## Kuhlela i-Tab Lesikhona

1. Chofota noma yiyiphi tab lesikhona kuluhlu lwe- **App Tabs**.
2. Buyeketa ligama letab, i-icon, i-URL, luhlobo, nome tilungiselelo te-visibility.
3. Chofota **Save Tab** kute usebentise tinshintjo takho.

## Kuguculisa Indzawo Yetema-Tab

Ungashintja indlela letab liboniswa ngayo ku-app yeselula. Dvonsa bese ulahla ema-tab kuluhlu kute utihlele kabusha. Indzawo leboniswe kuleli khasi iyafana neyekutsi emalunga akho atawubona kutiphi ku-app.

:::info
Letinye tema-tab tingavela ngekutentekela nakugcwaliseka tibonelo letitsite -- sibonelo, i-tab ye-Live Stream ingavela nakuhamba i-stream lesebentako. Ema-tab langetwe ngesandla akunika lawulo lolugcwele kutsi emalunga akho abonani ngato tonkhe tikhatsi.
:::

:::tip
Gcina inani letema-tab liphatseka. Emathathu kuya kuma-tab lasihlanu asebenta kahle emabandleni lamanyenti. Emangi kakhulu ema-tab angenta kuzulazula (navigation) kudzida emalungeni akho.
:::

## Lapho Lama-Tab Abonakala Khona

Ema-tab lawulungisa lapha aboniswa ku- **B1.church PWA** leyifakwa emalunga akho kusuka kunoma ngukuphi likhasi ku-`https://yourchurchname.b1.church`. Tinshintjo lotenta kuleli khasi tiyabonakala ngesikhatsi silunga selunga sivula i-app kabusha. (Ema-tab futsi aboniswa yi- [app yakadzeni ye-B1 Mobile](/docs/b1-mobile/) kunoma ngubani lomalunga losasebentisa yona, kodvwa loyo-app seyisayilelwe ekupheleni futsi ayisabuyekeziwa.)
