# Emakhampasi

<div class="article-intro">

Uma libandla lakho lihlangana etindzaweni letengeti nakunye, **Emakhampasi (Campuses)** akuvumela kutsi ulandzelele kutsi umuntfu ngamunye nelicembu ngalinye ligcina liphi. Nase kulungisiwe, emakhampasi avela njengenketfo emaphrofayilini ebantfu, ekulungiseleleni kweAttendance, kanye ku-Demographics dashboard. Emabandla lanetindzawo letinyenti angahlunga, asesha, futsi enta imibiko ngekhampasi kuyo yonkhe i-B1 Admin.

</div>

<div class="prereqs">
<h4>Before You Begin</h4>

- Udzinga imvumo le- **Edit Church Settings** kute uphatse emakhampasi. Buka [Roles & Permissions](./roles-permissions.md).

</div>

## Kuvula Tilungiselelo Tekhampasi

Ku-B1 Admin, yendlulela ku- **Settings** kulisayidbha lengesencele bese ukhetsa **Campuses** kuluhlu lweSettings. Utawubona luhlu lwato tonkhe emakhampasi lasalungiswa, kanye neligama lawo, indzawo, ne-timezone.

## Kungeta Ikhampasi

1. Chofota **Add Campus** (nome inkinobho le- **+** uma kungakabikho emakhampasi lasekhona).
2. Gcwalisa imininingwane yekhampasi:
   - **Name** *(kuyadzingeka)* — ligama leliboniswa kuyo yonkhe i-B1 Admin (sibonelo, "Main Campus" nome "North Campus").
   - **Address** — likheli lelisitaladi lekhampasi (lisetjentiswa ekukhombiseni umniningwane; akulingani nelikheli lelisisekelo lelibandla lakho ku-Church Settings).
   - **City / State / Zip** — indzawo yekhampasi.
   - **Timezone** — i-IANA timezone yalekhampasi (sibonelo, *America/Chicago*). Kuyasita nakuba emakhampasi asetindzaweni tesikhatsi letehlukene.
   - **Website** — i-URL lekhetsekile yebukhona bekhampasi bewebhu.
3. Chofota **Save**.

## Kuhlela Ikhampasi

Chofota noma ngumuphi umugca wekhampasi kuluhlu kute uvule sihleli sayo esengeni lesekudla. Buyeketa emasikhala bese uchofota **Save**.

## Kususa Ikhampasi

Vula ikhampasi kute uyihlele bese uchofota **Delete**. Utawuceliwa kucinisekisa. Kususa ikhampasi akususi bantfu labekelwe kuyo — sikhala sabo sekhampasi sisala singenalutfo.

## Kubekela Bantfu Kukhampasi

Ngemuva kwekudala emakhampasi, tisebenti tingabekela umuntfu kukhampasi kusuka kuphrofayili yakhe:

1. Vula umrekhodi wemuntfu ku- **People**.
2. Chofota **Edit**.
3. Khetsa ikhampasi kuluhlu lwe- **Campus** dropdown.
4. Chofota **Save**.

Ungakwati futsi kubuyeketa ikhampasi ngebunyenti kusuka ekhasini le-People. Khetsa bantfu labanyenti, sebentisa **Bulk Edit**, bese usimisa sikhala se-Campus sabo bonkhe kanyekanye.

## Kuhlunga Ngekhampasi

Nasekulungisiwe emakhampasi, ungahlunga kuyo yonkhe i-B1 Admin ngekhampasi:

- **Kusesha kwe-People** — ngeta simo se-Campus kukusesha lokungenele, nome layisha [Saved List](../people/lists.md) lebekelwe khampasi lelitsite.
- **Demographics** — i- [Demographics dashboard](../people/demographics.md) ikhombisa i-donut chart yeKhampasi nakukhona umuntfu lomunye lonekhampasi lebekelwe.
- **Attendance Setup** — sikhatsi ngasinye senkonzo ku-Attendance singabotjwa kukhampasi.

:::tip
Emabandla lanendzawo yinye akudzingi kulungisa emakhampasi. Tonkhe tici temakhampasi tikhetsekile — uma kungekho emakhampasi lakhona, tikhala tekhampasi kanye netichathi atibonakali nje.
:::
