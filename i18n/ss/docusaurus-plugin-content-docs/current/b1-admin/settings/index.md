# Tilungiselelo

<div class="article-intro">

Sigaba se-Settings yindzawo lapho ulungisa khona i-akhawunti yelibandla lakho, i-branding, kanye netinketfo tekuphatsa ku-B1 Admin. Kusuka lapha ungaphatsa konkhe kusukela eligameni lelibandla lakho ne-subdomain kuya kutimvumo tebasebentisi kanye nekulungiswa kwe-app yeselula.

</div>

## Yini Loyitfolako Lapha

Indzawo yeSettings ihlelwe yaba tincenye letilandzelako letifinyelelwa kusuka kulisayidbha:

1. **Settings** -- Lungisa imininingwane lesisekelo yelibandla lakho, i-branding, kanye ne-subdomain. Buka [Church Settings](./church-settings.md) kutfola imininingwane.
2. **Campuses** -- Ngeta futsi uphatse tindzawo letinyenti tebumzimba telibandla lakho. Nase kudaliwe, emakhampasi avela emaphrofayilini ebantfu, ekulungiseleleni kweAttendance, kanye ku-Demographics dashboard. Buka [Campuses](./campuses.md) kutfola imininingwane.
3. **Custom Fields** -- Chaza emasikhala akho lawakhetsile lekulandzelela kubantfu (lilanga, linani, imphendvulo lengu-ye/cha, nome luhlu lwekukhetsa), bese uwagcwalisa emaphrofayilini futsi usesha ngawo. Buka [Custom Fields](./custom-fields.md) kutfola imininingwane.
4. **Mobile Apps** -- Lungisa futsi uhlele ema-tab ekuzulazula laboniswa ku- [B1.church PWA](/docs/b1-church/getting-started/installing-pwa) yemalunga akho (lawo mama-tab afanako futsi aboniswa yi-app yakadzeni ye-B1 Mobile leseyisayilelwe ekupheleni). Buka [Mobile App Settings](./mobile-app.md) kutfola imininingwane.
5. **Server Admin** -- Finyelela kutinsita letitfutfukile tekuphatsa tilungiselelo telizinga leseva lelibandla lakho.
6. **Forms** -- Dala futsi uphatse emafomu akho lakhetsile ekucosha imininingwane kubantfu belibandla lakho.

## Likhasi Lelisisekelo Le-Settings

Ngesikhatsi uvula kucala likhasi le-Settings, utawubona **ligama lelibandla lakho** ne **subdomain** yakho kuboniswa engetulu. Sihloko sinika emankinobho ekufinyelela ngekushesha kumisebenti lejwayelekile:

- **Edit Settings** -- Buyeketa ligama lelibandla lakho, likheli, imininingwane yekutsintsana, kanye ne-branding.
- **Mobile Apps** -- Yeqa uye ngco ekulungiseni i-app yeselula.
- **Roles** -- Phatsa tikhundla nemvumo tebasebentisi becembu lakho.
- **Import/Export** -- Tfutfusa imininingwane emkhatsini wetinhlelo ngekusebentisa insita ye-import/export.

:::tip
Cala ngekulungisa ligama lelibandla lakho ne-branding ngaphansi kwe- **Edit Settings**, bese ulungisa [Roles](./roles-permissions.md) kute umeme emalunga ecembu lakho ngelizinga lelifanele lelifikelelo.
:::

:::info
Kutfola imininingwane mayelana nendlela imininingwane yakho leyivikelwa ngayo, buka [Data Security](./data-security.md).
:::
