# Wemukelekile ku-B1.church Admin!

<div class="article-intro">

Siyabonga ngekukhetsa B1.church! Lesingeniso sikuhambisa etinhlokweni tekucala kusebentisa i-B1.church Admin platform, kusukela ekungeneni nasekuhambeni ku-dashboard, kuya ekulungiseni imniningwane yelibandla lakho nasekuhloleni emathulusi latfolakalako.

</div>

## Tinyatselo Tekucala

1. Usebentisa i-Google Chrome, hamba ku-[admin.b1.church](https://admin.b1.church) bese ungena ngelibito lakho lekungena nelighama lakho lemfihlo.
2. Loku kutakuyisa ku-**dashboard ye-B1.church Admin**. Kusukela lapha, ungaphatsa i-database yelibandla lakho, liwebusayithi, i-app yeselula, tifundvo, nalokunye lokunyenti.
3. Chofota ku-**Tilungiselelo** kute ulungise imniningwane yelibandla lakho, tikhundla tebuphatsi, kanye nemininye imniningwane.
4. Chofota **lipentshisi lekuhlela** kute ulungise imniningwane yelibandla lakho.
5. Ubuyele ku-dashboard, uchofote **liayikhoni lelifonti lemibuto** lelisekhoneni lesekudla langetulu kute utfole lusito netifundvo.
6. Chofota **Buka Emadokhumenti** kute ubone luhlu lwetifundvo letakusita ucale kusebentisa i-B1.church.
7. Buyela ku-dashboard bese uchofota **umcibisholo losedvutane nesitfombe sakho sephrofayela** bese ukhetsa **Shintja i-App**.
8. Khetsa **Lessons.church** kute uphatse tifundvo teliBandla lakho letikhona ku-inthanethi kanye nekharikhulamu.
9. Buyela ku-B1.church ngekuchofota umcibisholo losedvutane nesitfombe sakho sephrofayela bese ukhetsa **Shintja i-App** futsi.
10. Manje ungangena ku-B1.church nge-**member portal** noma i-**admin portal**. I-member portal ikuvumela kubona kutsi i-B1.church ibukeka njani emalungwini akho, ubuke emacembu akho, futsi uphatse iminikelo yakho yebuwena.

:::tip
Cala ngelikhasi le-[Tilungiselelo](./settings/) kute ulungise ligama lelibandla lakho, kuvela kwemake, netimvumo tebasebenti bakho. Loku kucinisekisa kutsi konkhe lokunye lokutakulungisa kutakuvela kahle.
:::
