# Emakamelo, Tinsita Nekuhlelwa Kwesikhatsi

<div class="article-intro">

Emakamelo Netinsita akuvumela kutsi ulandzelele tindzawo tebandla lakho letingabhukhwa kanye netimphahla. Nangabe licembu nome licembu lebasebenti lifuna kusebentisa likamelo nome likweleta insita yemcimbi, bangayicela ngekusebentisa B1 — futsi ungakhona kuvumela nome kwenqaba letetisandzo kuze kungabi khona kubhukha kabili.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- Udzinga kufinyelela kwebuphatsi ku-Calendars ku-B1 Admin
- Hlela [emacembu](../groups/creating-groups) lalunga lawo latawufaka tetisandzo tekubhukha

</div>

## Kuvula Emakamelo Netinsita

Ku-B1 Admin, hamba uye ku-**Calendars** bese ukhetsa **Rooms & Resources**. Likhasi linetabhu letine: **Rooms**, **Resources**, **Blockouts**, ne-**Templates**.

## Emakamelo

Emakamelo tindzawo letibonakalako letingabekelwa kucaleka nemicimbi yekhalenda — indlu yekukhonza, holo yebudlelwane, ikilasi, njalonjalo.

### Kwengeta Likamelo

1. Kutabhu ye-**Rooms**, chofota **Add Room**.
2. Faka **ligama** lelikamelo.
3. Ungakwati kuhlela **linani lelivumelekile** (linani lelikhulu labantfu).
4. Hlela **Approval Group** — nangabe licembu licela lelikamelo, baholi belicembu lelikhetsiwe bayaziswa futsi kufanele bavumele kubhukha. Yekela ingene ize nangabe kute imvumo ledzingekile.
5. Chofota **Save**.

### Kuhlela nome Kususa Likamelo

Chofota sibonakaliso sekuhlela ngasekhatsi kwanoma yikuphi likamelo, wente lushintjo lwakho, bese uchofota **Save**. Sebentisa **Delete** kususa likamelo unomphelo.

## Tinsita

Tinsita timphahla letingabhukhwa kucaleka nemcimbi — timphahla te-AV, emaprojector, tihlalo, iveni, njalonjalo.

### Kwengeta Insita

1. Chofota tabhu ye-**Resources**, bese **Add Resource**.
2. Faka **ligama** futsi ungakwati kufaka **sichachiso**.
3. Hlela **Approval Group** nangabe kunemuntfu lodzinga kuvumela kukweletiswa kwensita.
4. Chofota **Save**.

## Kuvinjelwa

Kuvinjelwa kuvimbela emakamelo nome tinsita kutsi tibhukhwe ngesikhatsi lesitsite — ngenca yekulungiswa, tikhatsi letibekelwe, nome kulondvolozwa.

### Kwengeta Kuvinjelwa

1. Chofota tabhu ye-**Blockouts**, bese **Add Blockout**.
2. Khetsa kutsi ufuna kuvimbela **Likamelo**, **Insita**, nome onkhe emakamelo netinsita.
3. Hlela lusuku/sikhatsi se-**start** ne-**end** sekuvinjelwa.
4. Ungakwati kwengeta incwadzi ye-**reason**.
5. Chofota **Save**.

## Tifanekiso Temicimbi

Tifanekiso tikuvumela kugcina inhlelo yomcimbi lengasetjentiswa kabili — sihloko, sichachiso, kanye nemacembu lahlobene — kuze imicimbi lephindzaphindzako idalwe ngendlela lefanako njalo.

### Kwengeta Sifanekiso

1. Chofota tabhu ye-**Templates**, bese **Add Template**.
2. Faka **ligama** lesifanekiso.
3. Ungakwati kuhlanganisa **licembu** kuze umcimbi uhlangane ngalokutentekelako nalelocembu nawudalwa kusuka kusifanekiso.
4. Chofota **Save**.

Nakudalwa umcimbi lomusha, basebenti bangakhetsa sifanekiso lesikhona kuze bagcwalise imininingwane yomcimbi kucala.
