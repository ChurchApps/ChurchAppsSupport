# Kubhalisa Lokukhokhelwako

<div class="article-intro">

Kubhalisela umcimbi kungadlula kubala tinhloko lokulula. Ungachaza tinhlobo talabatakubakhona letinemtsengo (njenge Bantfu Labadzala ne Bantfwana), unikete tintfo letengetiwe letingakhetsa ngemtsengo netibalo tato, udale emakhodi esaphulelo, futsi ubutse imali yekukhokha ngesikhatsi sekubhalisa ngekusebentisa umniki-msebenti wemnikelo losewukhona webandla lakho. Nangabe umcimbi ugcwala, luhlu lolungakhetseka lwekulindza lugcina emalunga latitsandzelako esitini futsi liwenyusa ngalokutentekelako njengobe sikhala sivuleka.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- Vula kubhalisa kumcimbi kucala — buka [Dala Emakhalenda](creating-calendars#enabling-event-registration)
- Kubutsa imali yekukhokha, libandla lakho lidzinga [imnikelo we-online lohleliwe](../donations/online-giving-setup.md) (Stripe, PayPal, nome Kingdom Funding). Imicimbi lemahhala ayidzingi kuhlelwa kwemnikelo.

</div>

## Kuvula Tinhlelo Tekubhalisa

1. Ku-B1 Admin, hamba uye ekhasini le-**Registrations** bese uvula umcimbi wakho (nome uvule umcimbi kusuka ekhalendeni lawo).
2. Ikhadi le-**Registration Settings** liveta tisekelo — **Enable Registration**, **Capacity**, **Registration Opens/Closes**, **Tags**, ne-**Registration Questions**.
3. Ngentasi kwetisekelo kunema-accordion lamatsatfu: **Attendee Types**, **Selections**, ne-**Discount Codes**.

## Tinhlobo Talabatakubakhona

Tinhlobo talabatakubakhona tikuvumela kukhokhisa imitsengo lehlukene ngetinhlobo letehlukene talabatakubakhona — futsi ubekele linani lelivunyelwe ngalinye ngekwehlukanisa.

1. Vula i-accordion ye-**Attendee Types** bese uchofota **Add Type**.
2. Faka **Ligama** (sib. "Bantfu Labadzala", "Bantfwana", "Sitfundzeni").
3. Hlela **Mtsengo**. Sebentisa 0 kuhlobo lelimahhala.
4. Ungakwati kuhlela **Linani Lelivumelekile** kulelohlobo kuphela (sib. tikhala letingu-20 tebantfwana kuphela). Yekela ingene ize nangabe kute umkhawulo ngeluhlobo ngalunye.
5. Chofota **Save**.

Ngesikhatsi sekubhalisa, labatakubakhona ngamunye ukhetsa luhlobo; tinhlobo letigcwele tiveta njenge-**Sold out** futsi atikhetseki. Luhlu luveta luhlobo lwalabatakubakhona ngamunye kanye netibalo letiseluhambeni ngeluhlobo ngalunye.

## Tikhetso

Tikhetso tintfo letengetiwe letikhetsekako letinemtsengo — emahembe langu-T-shirt, tinhlelo tekudla, kunyuswa kwemisebenti.

1. Vula i-accordion ye-**Selections** bese uchofota **Add Selection**.
2. Faka **Ligama**, **Sichachiso** lesikhetsekako, ne-**Mtsengo** (0 iveta "Mahhala").
3. Ungakwati kuhlela **Linani Lelivumelekile** (linani lelipheleleko lelikhona kuko konkhe kubhalisa) ne-**Linani Lelikhulu** (linani lelikhulu lelinye kubhalisa lelingaodela).
4. Chofota **Save**.

Labhalisile bakhetsa tibalo ngesikhatsi sekubhalisa, futsi tinani letiphelele tibalwa kulinani lelivumelekile kuze ungakaze udayise ngetulu.

## Emakhodi Esaphulelo

1. Vula i-accordion ye-**Discount Codes** bese uchofota **Add Discount Code**.
2. Faka i-**Khodi** labhalisile labatakuyibhala.
3. Khetsa **Luhlobo** — **Liphesenti** nome **Linani** — kanye **Nelinani** lalo.
4. Ungakwati kukhawulisa likhodi nge-**Start Date** / **End Date**, i-**Min Members** (linani lelincane lalabatakubakhona kubhalisa), ne-**Max Uses**.
5. Chofota **Save**.

Likhodi ngalinye liveta linani le-**Uses** kuze ubone kutsi lisetjentiswe kangaki. Labhalisile batfola imphendvulo ngalokushesha nabasebentisa likhodi — kuhlanganise imilayeto lecacile nangabe likhodi seliphelelwe sikhatsi, lingakacali, nome lidzinga labatakubakhona labanyenti.

## Luhlu Lwekulindza

Vula **Enable Waitlist** ekhadini le-Registration Settings. Nangabe umcimbi ufinyelela linani lelivumelekile:

- Labhalisile labasha baniketwa sikhala eluhlwini lwekulindza esikhundleni sekutalwa. Bagcwalisa kubhalisa lokufanako (kukhokha kuyaeqiwa nasebasesikhaleni sekulindza).
- Nangabe umuntfu ekhansele, kubhalisa lokudzala kakhulu eluhlwini lwekulindza kuyanyuswa **ngalokutentekelako** futsi kutfola imeyili yekutsi sikhala sivuliwe. Nangabe basakweleta imali, imeyili iyabahambisela kutsi bagcwalise kukhokha.
- Ungakwati kunyusa umuntfu ngesandla nanini nawufuna nge-**Promote** action emgceni wekulindza — loku kusiza ngemva kwekwenyusa linani lelivumelekile lomcimbi.

:::info
Kubhalisa lokunyusiwe kuhlala *kusalindzwe* kuze noma yimuphi imali lesele ikhokhwe; kukhokha (nome kute lekukhokhwa) kuyakucinisekisa.
:::

## Luhlu Lwekubhalisa

Vula umcimbi kusuka ekhasini le-Registrations kubona konkhe kubhalisa. Ithebula liveta **Name**, **Members**, **Type** (luhlobo lwalabatakubakhona ngamunye), **Paid / Total** (nesecwayiso semali lesekweletwako nangabe isekhona), **Status**, ne-**Date**, kanye netibalo temaluhlobo ngetulu kwethebula.

- Chofota sibonakaliso semininingwane semgca kuvula i-dialog ye-**Registration Details** — emalunga, tikhetso, kukhokhiwe/kwelesele, kanye nethebula le-**Payments** lelibala tonkhe tikhokhelo (imali, indlela, lusuku).
- **Export CSV** ilayisha luhlu lolugcwele lolunetinsika temalunga, tinhlobo talabatakubakhona, tikhetso, kukhokhiwe/kuphelele/kwelesele, simo, kanye nensika yinye ngembuzo ngamunye wekubhalisa.
- **Add Attendee** isakuvumela kubhala kubhalisa lokungakenteki nge-online ngesandla.

:::info
Imali lebuyiselwako ayicutjwa ngekhatsi kwe-B1. Nangabe udzinga kubuyisela imali yekubhalisa lokukhokhelwe lokukhanseliwe, khokha imali ebuyiselwako kusuka ku-dashboard yemniki-msebenti wemnikelo wakho (sib. Stripe).
:::

## Kusebenta Kwekukhokha

Kukhokha kucutjwa ngesango lemnikelo lelifanako lelisebentiswa nguleli bandla lakho seliminikelo — imininingwane yekhadi ihamba ngalokucondzile emniki-msebentini futsi ayisoze icabane netiseva teB1. Emitsengo ihlala ibalwa kuseva kusuka etinhlotini takho letihleliwe, tikhetso, kanye nemakhodi esaphulelo, kuze umbhalisi angeke akwati kuguca linani leliphelele. Emalunga langenile angakhokha ngekhadi lelesekelwe; titamu tifaka ikhadi ekukhokheni.
