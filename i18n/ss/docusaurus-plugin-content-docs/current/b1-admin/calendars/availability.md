# Ikhalenda Lekutfolakala

<div class="article-intro">

Ikhalenda Lekutfolakala likuniketa kubuka lokusetulu kwako konkhe kubhukha kwemakamelo netinsita kulo lonkhe libandla lakho. Kusuka lapha ungabona loko lokuhleliwe, ubone kungahambisani ngembikwe kwenteka, futsi ubhukhe likamelo nome insita yanoma ngumuphi umcimbi ngalokucondzile.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- Hlela [likamelo nome insita](rooms-resources) linye noma ngesikhatsi ku-Rooms & Resources
- Udzinga kufinyelela kwekuhlela ku-Calendars ku-B1 Admin

</div>

## Kuvula Ikhalenda Lekutfolakala

Ku-B1 Admin, hamba uye ku-**Calendars** bese ukhetsa **Availability** ku-sidebar.

## Kufundza Ikhalenda

Ikhalenda liveta inyanga yamanje ngalokujwayelekile. Ungahamba embili nasemuva ngemicibi lesetulu, nome ushintje emkhatsini wekubuka kwenyanga, kweviki, nekwelanga.

Umcimbi ngamunye unemibala lehlukanisiwe ngesimo sekubhukha:

| Umbala | Lokusho |
|-------|---------|
| Luhlata | Kuvunyiwe |
| Orenji | Kusalindzwe imvumo |
| Mpunga | Kuvinjiwe (akutfolakali) |

Kwenta i-cursor ihlale etulu kwemcimbi kuveta sihloko somcimbi kanye nelikamelo nome insita lehlobene nawo.

## Kucabangisela Ngelikamelo nome Insita

Sebentisa i-**Filter** dropdown ngesencele lengetulu kuze unciphise ikhalenda liye elikamelweni linye nome insita yinye. Khetsa **All Rooms & Resources** kubuyela ekubukeni lokugcwele.

## Kubhukha Likamelo nome Insita

1. Chofota inkinobho ye-**Book** ekhoneni langesekudla lengetulu lelikhasi.
2. Ku-dialog leyivulekako, gcwalisa imininingwane yomcimbi:
   - **Title** — ligama lomcimbi
   - Lusuku/sikhatsi lwe-**Start** ne-**End**
   - **Visibility** — Public nome Private
   - **Rooms** — khetsa likamelo lelinye nome lamanyenti kutsi ubekele
   - **Resources** — khetsa insita yinye nome letinyenti kutsi ubekele
3. Ungakwati kuhlela emasikhatsi e-**Setup** ne-**Teardown** (ngemaminithi). Loku kwengeta sikhatsi kuwo womabili emakhona ekubhukha kuze indzawo ibekelwe kulungiswa nekuhlanjululwa, ngisho nome emasikhatsi ekucala/kuphela komcimbi ahlala afanele.
4. Kuphindza kubhukha, hlola **Repeats** bese uhlela kuphindzaphindza:
   - **Repeat every** -- hlela sikhatsi lesiphakathi (sibonelo, njalo ngemaviki lamabili).
   - **Frequency** -- Nsuku Nonkhe, Ngeviki, nome Ngenyanga. Ngeviki kukuvumela kukhetsa emalanga latsite eviki; Ngenyanga kukuvumela kukhetsa lilanga lelicondzile lenyanga nome iphetheni lelihlobene njenge "lilanga lesibili laLesibili."
   - **Ends** -- Akusoze, ngelusuku loluthize, nome ngemva kwenani lelibekiwe letento.
5. Kukhomba wingdo lekubhukha lecondzile (leyehlukile kucala/kuphela komcimbi), vula **Custom Booking Window** bese ufaka emasikhatsi ekucala nekuphela alelowingdo. Sebentisa loku nangabe likamelo lidzinga kufinyeleleka ngaphandle kwemahora labekwe emcimbini.
6. Chofota **Save** kutfumela kubhukha.

:::info
Nangabe likamelo nome insita line-**Approval Group** lehleliwe, kubhukha kutakuvela njenge-**Pending** kuze kufike umholi walelocembu ekuyivumeleni. Buka [Kuvunywa Kwemakhalenda](approvals) kutfola inchubo yekuvunywa.
:::

:::tip
Ikhalenda litawukhombisa nome ngukuphi kungahambisani ngembikwe usindza. Nangabe ubona secwayiso sekungahambisani, lungisa emasikhatsi akho nome ukhetse likamelo lelehlukile.
:::
