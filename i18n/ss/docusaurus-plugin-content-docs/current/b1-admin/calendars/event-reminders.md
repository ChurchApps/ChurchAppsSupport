# Tikhumbuto Temicimbi

<div class="article-intro">

Tikhumbuto temicimbi tiyaziswa ngalokutentekelako bantfu labafanele ngembikwe umcimbi wente -- sibonelo, "Ungakulahleki! I-workshop yemphilo iyacala kusasa nga-9:00 ekuseni." Uhlela sikhumbuto kanye kumcimbi, futsi B1 iyasithumela ngesikhatsi lesihleliwe ngekusebentisa tatiso te-push kanye ne-imeyili. Emalunga angalawula kutsi ngutuphi tikhumbuto labatitfola kusuka ku-[Tincamelo Tetatiso](../../b1-church/getting-started/notification-preferences) tabo.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- Dala umcimbi lofuna kukhumbuta bantfu ngawo (buka [Dala Emakhalenda](creating-calendars))
- Kufinyelela kubantfu labhalisile, [vula kubhalisa](creating-calendars) kumcimbi
- Kufinyelela licembu lonkhe, cinisekisa kutsi umcimbi ungewe [licembu](../groups/creating-groups) lelinemalunga

</div>

## Kuhlela Sikhumbuto

Uhlela tikhumbuto endzaweni ye-**Reminders** yemcimbi.

- Nawe**dala umcimbi lomusha**, vula sigaba se-**Reminders** ku-event editor ngembikwe usindza.
- Ku**mcimbi lokhona nakadze**, vula likhasi lemcimbi le-**Registration Details** (kusuka esigabeni se-**Registrations**) kwengeta nome kushintja sikhumbuto salo.

1. Vula **Enable reminders**.
2. Khetsa **When** yekutfumela. Khetsa kufika kumatsatfu emasikhatsi: **emalanga lasikhombisa ngembili**, **emalanga lamatsatfu ngembili**, **lilanga linye ngembili**, ne-**Lilanga Lomcimbi**.
3. Hlela **Sikhatsi Selanga** lokufanele sikhumbuto sitfunyelwe ngaso (kokujwayelekile ku-**9:00 ekuseni**, ngesikhatsi sendzawo yebandla lakho).
4. Khetsa **Ngubani** lokufanele akhunjulwe (buka [Ngubani Lokhunjulwako](#who-gets-reminded) ngentasi).
5. Ungakwati kwengeta **Umlayeto**. Yekela ingene ize kutsi usebentise emagama ajwayelekile, nome ubhale awakho -- ungafaka `{{eventTitle}}` futsi lokhu kutawushintjwa ngeligama lomcimbi.
6. Khetsa **Tindlela**: sitatiso se-**Push**, i-**Email**, nome kokubili.
7. Sindza umcimbi.

Njengobe wenta tishintjo, **sibonakaliso lesiphilako** siveta ngalokusondzelene kutsi bangaki bantfu labatawukhunjulwa, bangaki labazakubhalisile labangeke bafinyelelwe, kanye netikhatsi letilandzelako tekutfumela lesihleliwe -- kuze ukwati kucinisekisa kutsi sikhumbuto sibukeka kahle ngembikwe usindza.

## Ngubani Lokhunjulwako

Sihlelo se-**Who** silawula kutsi sikhumbuto siya kubani:

- **Labhalisile kuphela** -- Wonkhe umuntfu lobhalisele umcimbi lohlangene nerekhodi lemuntfu. Loku kokujwayelekile nangabe umcimbi unekubhalisa lokuvuliwe, kuze sikhumbuto semcimbi lomncane lobhalisiwe singaze siye elicenjini lonkhe ngengoti.
- **Tinhloko / labhalisile kuphela** -- Sikhumbuto sinye kubhalisela ngalinye (umuntfu lobhalisile), kunekutsi kube ngemalunga onkhe emndeni lasekubhalisweni.
- **Emalunga elicembu** -- Wonkhe umuntfu losecenjini lomcimbi. Loku kokujwayelekile nangabe umcimbi awukusebentisi kubhalisa.
- **Ngekutitjintja (Auto)** -- Isebentisa labhalisile nangabe kubhalisa kuvuliwe, kanye kanjalo isebentise licembu.

:::info
Titamu letingenwe ngeligama kuphela (ngaphandle kwerekhodi lemuntfu lelihlobene) atikwati kutfola sikhumbuto, ngobe kute akhawundi, sisetjentiswa, nome imeyili yekutfumela kuyo. Sibonakaliso sikutjela kutsi bangaki labazakubhalisile labangena kuleli chembu kuze kungabikho kumangala. Emalunga lakhetse kutsi angatfoli tatiso nawo ayeqiwa.
:::

## Nini Lapho Tikhumbuto Titfunyelwa Khona

- Tikhumbuto titfunywa nga-**sikhatsi selanga lokhetsile**, ngesikhatsi sendzawo yebandla lakho, kuko onkhe emasikhatsi lawakhetsile.
- Nawe**shintja lusuku nome sikhatsi somcimbi**, tikhumbuto letisalindzile tiyahlelwa kabusha ngalokutentekelako -- awudzingi kuhlela sikhumbuto.
- Nawe**susa umcimbi** (nome ukhansele sento sinye somcimbi lophindzaphindzako), tikhumbuto talo letisalindzile tiyakhanselwa ngalokutentekelako.
- Imicimbi lephindzaphindzako iphatfwa ngalokutentekelako: sento ngasinye lesetako sitfola sikhumbuto saso.

:::tip
Tikhumbuto titfunywa **nge-push kucala, ne-imeyili njengendlela yekugcina**. Nangabe lilunga line-push notification lisebenta, litawutfola i-push; nangabe kute, litawutfola imeyili endzaweni yaleyo. Emalunga akhetsa kutsi ngutuphi tindlela lafuna kutona kuluhlobo ngalunye lwesatiso ku-[Tincamelo Tetatiso](../../b1-church/getting-started/notification-preferences) tabo.
:::

## Loko Emalunga Langakulawula

Tikhumbuto tihlala tihlonipha [Tincamelo Tetatiso](../../b1-church/getting-started/notification-preferences) talilunga ngalinye. Lilunga lingakwati:

- Kuvala **Tikhumbuto Temicimbi** ku-push nome ku-imeyili nasagcina letinye tatiso tivuliwe.
- Kuhlela **emahora ekuthula** kuze tatiso letingasiphutfumele tilinde kufika sikhatsi lesifanele.

Awukwati kudlula sincumo selunga sekungafuni kutfola tikhumbuto temicimbi -- loku kugcina B1 ihambisana nemitsetfo lelwisana ne-spam futsi kugcina emalunga alawula i-inbox yawo.

## Tikhumbuto Tekukhonta

Tivolontiya letihleliwe ku-plan titfola **sikhumbuto sekukhonta** lesihlukene lesinemininingwane ye-plan futsi, nangabe bangakaphenduli, tinkinobho te-**Accept / Decline** ngalokucondzile ku-imeyili. Letokhumbuto tihlelwa ku-plan type kunekutsi tibe kumcimbi wekhalenda -- buka [Tivolontiya TeSonto](../guides/sunday-volunteers) kutsi kusebenta njani kuhlelwa kwetivolontiya nemtikhumbuto.
