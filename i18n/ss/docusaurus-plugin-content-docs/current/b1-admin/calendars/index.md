# Emakhalenda

<div class="article-intro">

Sigaba se-Calendars ku-B1 Admin sikuvumela kudala emakhalenda lahlelwe ebandla lakho ngekubutsa imicimbi lesuka emacenjini akho. Baholi bemacembu baphatsa imicimbi yemacembu abo, futsi njengemphatsi ungakhona kwakha emakhalenda laveta lemicimbi kuwebusayithi yakho ye-B1.church kutsi libandla lakho libone.

</div>

## Kubuka Emakhalenda Akho

1. Ku-B1 Admin, hamba uye ku-**Calendars** kumenu lomkhulu.
2. Likhasi le-Calendars libala onkhe emakhalenda akho kanye nemagama kanye nesimo sawo.
3. Ikhalenda ngayinye ibuye iveta emacembu lahlobene nayo.

## Loko Longakwenta

- **Kwengeta ikhalenda lelisha** -- Chofota inkinobho yekwengeta kudala ikhalenda lemkhonto lotsite, luhlobo lomcimbi, nome injongo. Buka [Dala Emakhalenda](creating-calendars) kutfola imiyalo ngayinye ngayinye.
- **Kuhlela ikhalenda** -- Chofota kunoma yikuphi ikhalenda kuze liyivule futsi uphatse kutsi ngutuphi emacembu lafakiwe kuyo.
- **Kushicilela emakhalenda** -- Emakhalenda angavezwa kuwebusayithi yakho ye-B1.church kuze labo labetako babone imicimbi lesetayo ebandleni lakini.
- **Kudala emakhalenda lahlelwe** -- Butsa imicimbi lesuka emacenjini lamanyenti liyifake kukubuka kwekhalenda linye kutfola sitfombe lesigcwele saloko lokwentekako ebandleni lakini. Buka [Ikhalenda Lelihlelwe](curated-calendar) kutfola imininingwane.
- **Phatsa emakamelo netinsita** -- Landzelela tindzawo letingabhukhwa kanye netimphahla futsi uphatse tetisandzo tekuvunywa. Buka [Emakamelo Netinsita](rooms-resources) kutfola imininingwane.
- **Buka kutfolakala** -- Bona konkhe kubhukha kwemakamelo netinsita ekhalendeni linye futsi ubhukhe tindzawo temicimbi lesetayo. Buka [Ikhalenda Lekutfolakala](availability) kutfola imininingwane.

:::info
Emakhalenda asebenta kalufatane [nemacembu](../groups/creating-groups.md). Nangabe emacembu anemicimbi lehlelwe, lemicimbi ingabuthwa iyiswe emakhalendeni akho. Buka likhasi le-[Dala Emakhalenda](creating-calendars) kufundza kutsi uhlanganisa njani emacembu nemakhalenda.
:::
