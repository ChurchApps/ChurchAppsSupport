# Ikhalenda Lelihlelwe

<div class="article-intro">

Ikhalenda lelihlelwe libutsa imicimbi lesuka emacenjini lamanyenti liyifake kukubuka kwekhalenda linye lelihlangene. Loku kufaneleka kakhulu ekwakheni ikhalenda lelimele libandla lonkhe leliveta konkhe lokwentekako kuwo onkhe emakhonza netinhlangano takho endzaweni yinye.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- Hlela [emacembu](../groups/creating-groups.md) lofuna imicimbi yawo ifakwe
- Cinisekisa kutsi baholi bemacembu bafake imicimbi emacenjini abo
- Tijwayeze netisekelo teku[Dala Emakhalenda](creating-calendars)

</div>

## Kudala Ikhalenda Lelihlelwe

1. Kusuka ku-B1 Admin, hamba uye ku-**Website**.
2. Chofota **Calendars** kunavigeshini lengetulu.
3. Chofota **Add Calendar** kudala ikhalenda lelisha lelihlelwe.
4. Nika ikhalenda lakho **ligama** (sibonelo, "Imicimbi Yebandla" nome "Liviki Leli Ebandleni Lakitsi").
5. Chofota **Create** kulisindza.

## Kwengeta Emacembu Kikhalenda Lakho

1. Chofota ekhalendeni lakho lelisha kuze liyivule.
2. Endzaweni ye-**Groups in Calendar** ngesekudla, chofota **Add**.
3. Khetsa licembu ku-menu leyehlako.
4. Khetsa kutsi ufuna kwengeta **yonkhe imicimbi** yalelocembu nome **imicimbi lecondzile kuphela**.
5. Chofota **Save**.
6. Phindza lenchubo kangetulu kwengeta emacembu lamanyenti njengobe ufisa.

:::tip
Engeta wonkhe emacembu akho asebentako emkhontweni kudala ikhalenda leliphelele leliniketa libandla lakho sitfombe lesigcwele saloko lokwentekako ebandleni lakini liviki nelilinye.
:::

## Kwabelana Ngekhalenda Lakho

Nasesihlelwe kahle ikhalenda lakho lelihlelwe, ungakwati kulyabelana ngato ngetindlela letinyenti:

- **Faka kuwebusayithi yakho** -- Ikhalenda lingavezwa ngalokucondzile emakhasini ewebusayithi yakho ye-B1.church kutsi labo labetako bakubuke.
- **Bhalisela nge-ICS** -- Chofota **Subscribe** kucosha i-link ye-ICS. Libandla lakho lingayifaka lelilinki ku-Google Calendar, Apple Calendar, Outlook, nome nome ngiyiphi lenye application yekhalenda kuze bahlale bahambisana.
- **Layisha lifayela le-ICS** -- Layisha lifayela lekhalenda kuze ulifake kuapplication yekhalenda.

:::info
Nabaholi bemacembu bengeta imicimbi lemisha [emacembu](../groups/creating-groups.md) abo, lemicimbi ngalokutentekelako iyavela ekhalendeni lakho lelihlelwe. Awudzingi kubuyeketa ngesandla ikhalenda lelihlelwe njalo nakwengetwa umcimbi lomusha.
:::
