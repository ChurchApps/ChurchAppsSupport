# Kuvunywa Kwemakhalenda

<div class="article-intro">

Likhasi Lekuvunywa (Approvals) yindzawo lapho baphatsi babuyeketa khona futsi benta sincumo ngetetisandzo letisalindzile tekubhukha emakamelo netinsita, kanye nemicimbi yekhalenda ledzinga imvumo ngembikwe kushicilelwa.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- Hlela emakamelo nome tinsita nge-**Approval Group** ku-[Emakamelo Netinsita](rooms-resources)
- Udzinga imvumo ye-**Calendars Admin** nome imvumo ye-**content.edit**

</div>

## Kuvula Likhasi Lekuvunywa

Ku-B1 Admin, hamba uye ku-**Calendars** bese ukhetsa **Approvals**. Tetisandzo letisalindzile tekubhukha kanye nemicimbi lokusalindzwe kubuyeketwa kwato kubalulwe lapha.

## Tetisandzo Tekubhukha

Nangabe licembu lidala umcimbi futsi licela likamelo nome insita, lesitisandzo sivela ku-**Booking Requests** phaneli. Umugca ngamunye uveta:

- Likamelo nome insita lecelwako
- Ligama lomcimbi kanye nelusuku/sikhatsi
- Licembu lelicelako

### Tibonakaliso Tekungahambisani

Uma tetisandzo letimbili tigcwelana ngelikamelo nome insita lefanako, kuvela sibonakaliso sesecwayiso sekungahambisani. Buyeketa ngekucophelela letisandzo letingahambisani ngembikwe uvumela lolunye kuto.

### Kuvumela nome Kwenqaba

Chofota sibonakaliso se-**✓** (vumela) nome se-**✗** (nqaba) kunoma yisiphi sitisandzo sekubhukha. Licembu lelicelako liyaziswa ngesincumo. Kubhukha lokuvunyiwe kukhiywa kulelokamelo nome insita mayelana nemcimbi; kubhukha lokwenqatjiwe kukhulula sikhala kwabanye.

## Imicimbi Lesalindzile

Nangabe inchubo yakho yekhalenda idzinga imvumo yemicimbi ngembikwe iyaboniswa esiveni, imicimbi lesalindzile ivela ku-**Pending Events** phaneli. Vumela umcimbi kuze ushicilelwe ekhalendeni, nome uwunqaba kuze wati lomuntfu lowufakile kutsi kudzingeka kwentiwe lushintjo.

:::tip
Hlela Licembu Lekuvumela likamelo ku-[Emakamelo Netinsita](rooms-resources) kudzingekisela imvumo yalelokamelo. Emacembu lanelilungelo angase acele likamelo nawadala imicimbi, futsi letetisandzo tigelete kulelikhasi.
:::
