# Imisebenti

<div class="article-intro">

Imisebenti ikuvumela kwabela tintfo letidzingeka kwentiwa kubantfu nobe emacembu elibandla lakho. Kungakhathaliseki kutsi kusebenti sekwentiwa kanye nobe sibopho lesiphindzaphindzako, imisebenti iyakusita kulandzelela kutsi yini lekufanele yentiwe futsi ngubani lonesibopho kuze kute lokushiywako.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- Cinisekisa kutsi bantfu nobe [emacembu](../groups/creating-groups.md) lofuna kubabela imisebenti bakhona ku-B1 Admin
- Hamba uye esigabeni seKukhonta kute ufinyelele likhasi leMisebenti

</div>

## Kubuka Imisebenti

Hamba uye ku-**Kukhonta** bese uvula likhasi le-**Misebenti**. Utawubona luhlu lwemisebenti yakho lesavuliwe. Yonkhe umsebenti ukhombisa sihloko saso, lowabelwe wona, kanye nesimo saso.

:::tip
Ngekwehluleka, kukhonjiswa kuphela imisebenti lesavuliwe. Phendukisa **Show Closed** kute ubone nemisebenti lesecediwe.
:::

## Kwakha Umsebenti

1. Kulikhasi leMisebenti, cindzetela **Add Task**.
2. Faka **sihloko** somsebenti.
3. Cindzetela libhokisi le-**Assign To** bese ubhala ligama lemuntfu, nobe ushintjele ku-**Group** bese ubhala ligama lelicembu.
4. Cindzetela **Search** bese ukhetsa umuntfu nobe licembu kuletiphumo.
5. Engeta **inothi** ngayo yonkhe imininingwane nobe imiyalo yalowo lowabelwe umsebenti.
6. Cindzetela **Save**.

Umsebenti utawuvela manje eluhlwini lwemisebenti futsi utabonakala kulowo lowabelwe ku-dashboard yakhe nase-app yefowni.

## Kuphatsa Umsebenti

Cindzetela nayiphi umsebenti kute uvule likhasi laso lelinemininingwane. Kusuka lapha ungakhona:

- **Kushintja simo** -- Shintjela emkhatsini we-**Open** kanye ne-**Closed** usebentisa i-dropdown yesimo.
- **Kuphindze kwabela** -- Shintja lowabelwe kube ngumuntfu nobe licembu lelihlukene.
- **Kwengeta emanothi** -- Faka imininingwane leyengetiwe nobe kuvuselelwa kulowo lowabelwe.
- **Kuhlela sihloko** -- Shintja sihloko somsebenti njengobe kudzingekile.

## Indlela Lokuhamba Ngayo Umsebenti

1. Umsebenti lomusha ucala ngesimo se-**Open**.
2. Lowabelwe utfola satiso futsi angabona umsebenti kusuka ku-dashboard yakhe, likhasi leMisebenti, nobe i-app yefowni.
3. Nasekucedvwe umsebenti, lowabelwe (nobe umuntfu we-admin) ushintja simo kuya ku-**Closed**.
4. Imisebenti lesivaliwe iyasuswa kumbukiso wekwehluleka kodvwa ingakafinyelelwa ngekuvula **Show Closed**.

:::info
Ungasetha kutitenzela kwemisebenti lephindzaphindzako kute yakhiwe ngekulandzelana kwesikhatsi. Bona [Kutitenzela](./automations.md) kute utfole imininingwane yekusetha loku.
:::

:::tip
Nawudzinga kuhambisa bantfu labaningi ngendlela lefanako yetigaba -- njengekulandzelela tivakashi letisha nobe inchubo yebulunga -- sebentisa [Tinchubo Temisebenti](./workflows.md). Tinchubo Temisebenti tiphendula imisebenti ibe ngemakhadi ku-bhodi lekudonswa nekubekwa, kanye netinyatselo, emalanga ekugcina, kanye nekuhambisa.
:::
