# Tingoma

<div class="article-intro">

Likhasi leTingoma liphatsa umtapo wakho wetingoma tekukhulekela. Gcina yonkhe imininingwane yetingoma, emavi, kanye nekuhlelwa endzaweni yinye kute licembu lakho lekukhulekela likhone kutilungiselela kalula tinkonzo letitako futsi ligcine luhlu lwetingoma loluhambelanako.

</div>

- Ngekukhetsa, yiba ne-[tinhlelo tenkonzo](./plans.md) letisetiwe kute ukhone kuhlanganisa tingoma ngco ku[lulandzelo lwenkonzo](./service-order.md)

## Kubukela Umtapo Wakho Wetingoma

Nawuvula likhasi le-**Tingoma**, tingoma takho tikhonjiswa njengemakhadi. Likhadi ngalinye likhombisa sitfombe lesincane sengoma, sihloko, umculi, kanye nesikhatsi lesihamba sona. Gcuma emtapweni nobe usebentise kusesha kute utfole lokudzingako.

## Kusesha Tingoma

1. Cindzetela inkhinobho ye-**Search** ekhasini leTingoma.
2. Bhala **sihloko** sengoma nobe ligama **lomculi**.
3. Timphumela titohlungwa njengoba ubhala, tikhombisa tingoma letihambelanako kumtapo wakho.

## Kwengeta Ingoma

1. Cindzetela **Add Song** ekhasini leTingoma.
2. Sesha ku-database yengoma yangaphandle ngesihloko nobe ngemculi.
3. Khetsa ingoma lofuna kuyengeta.
4. Ingoma itawungetwa emtapweni wakho ngemininingwane yayo lesisekelo isesigcwalisiwe.

:::tip
Ngemuva kwekwengeta ingoma, cindzetela kuyo kute ubuyekete futsi ubuyekete imininingwane yayo, wengete kuhlelwa, nobe uhlanganise tixhumanisi tangaphandle njengetividiyo te-YouTube nobe emashadi ephimbo.
:::

## Kubuka Nekuhlela Ingoma

Cindzetela nayiphi likhadi lengoma kute uvule likhasi layo lelinemininingwane. Kusuka lapha ungakhona:

- **Hlela imininingwane yengoma** -- Buyekete sihloko, umculi, sikhatsi lesihamba sona, kanye nalenye imininingwane.
- **Phatsa kuhlelwa** -- Ingoma nganye ingaba nekuhlelwa lokuningi (sibonelo, tinophu letehlukene nobe tinhlobo tehlukene). Engeta, hlela, nobe susa kuhlelwa njengobe kudzingekile.
- **Engeta emavi** -- Faka nobe ubuyekete emavi kuhlelwa ngakunye.
- **Setha inophu** -- Chaza inophu yemculo kuhlelwa ngakunye usebentisa sikhetsi senophu eceleni kwemavi. Kushintja inophu kushintja masinyane emakhodi lakhonjiswako, kokubili ekhasini lekuhlelwa nanawuvula ingoma kusuka eluhlelweni lwenkonzo, kute licembu lakho libone njalo emakhodi enophu leyisebentako ngempela.
- **Engeta tixhumanisi tangaphandle** -- Hlanganisa tixhumanisi kutfolakalo letifana netividiyo te-YouTube, emashadi ephimbo, nobe imibhalo yemculo legcinwe kwenye indzawo.

## Kuhlelwa

Kuhlelwa kumele luhlobo loluhlukile lwengoma — sibonelo, luhlobo lolusheshako lwanamuhla ngokumelene neluhlobo lolucotfo, nobe leyo ngoma efanako kunye inophu lehlukene yelicembu lehlukene. Kuhlelwa ngakunye kwengoma lefanako kwabelana ngesihloko sengoma nemculi kodvwa kunemiseto yako lengatimeleko:

- **Inophu** — inophu yemculo lokuhlelwa okusetjentiswa kuyo
- **BPM** — lijubane ngeminithi
- **Bude** — sikhatsi ngemasekondi
- **Sigijimi** — luphawu lwesikhatsi (sib. 4/4, 3/4, 6/8)
- **Kulandzelana** — sakhiwo semakhodi/tigaba (sib. Sicalo → Ivesi → Ikhorasi → Ibhulitji)
- **Emavi** — kukhonjiswa nobe kudlaliswa
- **Tixhumanisi tangaphandle** — emashadi ephimbo, kurekhodwa kwe-YouTube, imibhalo yemculo, nobe letinye tintfo

Nawengeta ingoma ku-[Kulandzelana Kwenkonzo](./service-order.md), ungakhetsa kutsi ngukuphi kuhlelwa lokufuna kukusebentisa kuleyo nkonzo lekhetsekile, kute licembu lakho lihlale lifundza futsi lidlala luhlobo lolulungile.

:::info
Tingoma temtapo wakho tingengetwa ngco ku-[Kulandzelana Kwenkonzo](./service-order.md) ngekhatsi kweluhlelo, kuhlanganisa imininingwane yengoma kanye nekuhlelwa lokukhetsiwe kuleyo nkonzo lekhetsekile.
:::
