# Tifanekiso Tetinhlelo

<div class="article-intro">

Tifanekiso tetinhlelo tikuvumela kugcina sithombe lesingasetjentiswa kabusha selulandzelo lwenkonzo yeluhlelo nobe tikhundla bese ulisebentisa etinhlelweni letitako ngekucindzetela kanye. Esikhundleni sekwakha kabusha lokwakhiwo lokufanako yonkhe liviki, ulisindza kanye bese ulifaka etinhlelweni letiningi letitako ngendlela lodzingako.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- Yakha lesincane [luhlelo lwenkonzo](plans.md) olulandzelo lwenkonzo nobe tikhundla tichaziwe
- Udzinga kufinyelela endzaweni yeKukhonta ku-B1 Admin

</div>

## Kusindza Luhlelo Njengesifanekiso

1. Vula luhlelo lolusele lukhona kusuka endzaweni ye-**Kukhonta**.
2. Cindzetela inkhinobho ye-**Templates** (esicongweni sekhasi leluhlelo).
3. Khetsa **Save as Template**.
4. Nika sifanekiso ligama (sibonelo, "Standard Sunday" nobe "Worship + Sermon").
5. Sindza. Sifanekiso sitsatsa simo saluhlelo salulandzelo lwenkonzo kanye netikhundla ngaleso sikhatsi.

## Kuphatsa Tifanekiso

Kusuka enkhinobhweni ye-**Templates** kunayo yonkhe likhasi lohlobo lweluhlelo ungakhona:

- **Kuphindze ubite** — shintja ligama leliboniswako lesifanekiso
- **Buyeketa kusuka eluhlelweni** — shintjanisela kucuketfwe kwesifanekiso ngesimo saluhlelo sanyalo (kuyasita ngemuva kwekutsi ulungisile lokwakhiwo lwakho lolujwayelekile)
- **Sebentisa** — faka sifanekiso etinhlelweni letinye nobe letinyenti letitako
- **Susa** — susa sifanekiso longasadzingi

## Kufaka Sifanekiso Etinhlelweni

1. Cindzetela inkhinobho ye-**Templates** ekhasini lohlobo lweluhlelo.
2. Cindzetela lithicho le-**Apply** (lithicho lekukopisha) eceleni kwesifanekiso lofuna kusisebentisa.
3. Khetsa kutsi ngutiphi tinhlelo letitako lofuna kusisebentisa kuto ngekumaka emabhokisi wato.
4. Khetsa lokufuna kukopisha:
   - **Lulandzelo Lwenkonzo** — kushintjanisela tintfo telulandzelo lwenkonzo letinhlelo lekhetsiwe ngetinye tesifanekiso
   - **Tikhundla** — kushintjanisela tikhundla (kanye nesabelo sato) tetinhlelo letikhetsiwe ngeteso sifanekiso
5. Cindzetela **Apply**.

:::warning
Kusebentisa tikhundla kushintjanisela tonkhe tikhundla nesabelo lesele sikhona etinhlelweni letikhetsiwe. Cinisekisa kutsi awukakahleli bavolontiya kuletikhundla ngaphambi kwekusebentisa lesifanekiso.
:::

:::tip
Sebentisa Lulandzelo Lwenkonzo ngaphandle Kwetikhundla nawufuna sakhiwo lesifanako lwengoma/lentshumayelo kodvwa bavolontiya sebasabelwe ngendlela lehlukene liviki ngaliviki.
:::

## Kugcina Tifanekiso Tibuyekekile

Nangabe sakhiwo sakho lesijwayelekile senkonzo sishintja, vula umphatsi wesifanekiso bese usebentisa **Update from plan** kute uhambise sifanekiso kusuka eluhlelweni lwakho lwakamuva esikhundleni sekusakha kabusha kusukela ekucaleni.
