# Kuhlanganisa i-TV Yekilasi Netifundvo Takho Te-B1 Admin

<div class="article-intro">

Ungahlanganisa i-TV yekilasi netifundvo takho te-B1 Admin usebentisa **i-FreePlay** — i-app yasimahla yeChurchApps yema-TV emabandla. Nasekuhlanganisiwe, i-TV ikhombisa kucuketfwe kwesifundvo losukuhlelile ku-B1 Admin, kukwenta kutfolakale kubatitjhere bakho ngco esikrinini sekilasi.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- Hlela sifundvo lesisincane ku-B1 Admin (bona [Kuhlela Tifundvo](./scheduling-lessons.md))
- Faka **i-FreePlay** kuTV nobe Fire Stick lofuna kuyisebentisa

</div>

## Sinyatselo 1 — Faka i-FreePlay kuTV Yakho

Faka i-app ye-**FreePlay** kusikrini sekilasi lakho:

- **Amazon Fire Stick / Fire TV** — Sesha **FreePlay** ku-Amazon Appstore
- **Android TV / Google TV** — Sesha **FreePlay** ku-Google Play Store
- **Apple TV** — Sesha **FreePlay** ku-App Store

Kutfola imiyalo lengcile yekusetha bona luchazo lwe-[FreePlay Getting Started](/docs/freeplay/getting-started/).

## Sinyatselo 2 — Hlanganisa i-B1.church Njengemhlinzeki Wekucuketfwe

1. Vula i-app ye-**FreePlay** kuTV
2. Hamba uye kusikrini se-**Content Providers** (khetsa lithicho le-**Providers** ku-sidebar)
3. Tfola bese ukhetsa i-**B1.church** eluhlwini lwebahlinzeki labatfolakalako
4. Landzela imiyalo lesesikrinini kute ungene usebentisa imininingwane yakho yelibandla ye-B1
5. I-FreePlay ikhombisa **Connected!** nasekuphumelele luxhumano

Nasekuhlanganisiwe, i-B1.church ivela ku-sidebar yakho ye-FreePlay. I-FreePlay itokhipha kucuketfwe kwesifundvo losukuhlelile ku-B1 Admin kute batitjhere bakho bakhone kukudlala ngco kuTV yekilasi.

:::tip
Ngemuva kwekuhlanganisa, batitjhere bakho bangasebentisa i-remote yeTV kute bachubeke etividiyweni nasesilayidini ngelijubane labo njengoba sifundvo sihamba. Akudzingeki kusetjenta ngekwehlukene nobe ema-USB.
:::

:::info
Kutfola imiyalo lengcile yekugunyatwa nemhlinzeki wekucuketfwe (i-QR code, kungena nge-form, njll.), bona [Kuhlanganisa Nebahlinzeki](/docs/freeplay/content-providers/connecting-providers).
:::
