# Tinhlelo Tenkonzo

<div class="article-intro">

Tinhlelo tenkonzo tihlela kutsi ngubani lokhontako futsi ngunini. Luhlelo ngalunye luhlotjaniswa nelilanga lekhetsekile kanye nemnyango, kukwenta kube lula kuhlanganisa emacembu akho ebavolontiya liviki ngeliviki futsi ucinisekise kutsi inkonzo ngayinye igcwele ngebantfu.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- Setha imnyango kanye nemacembu akho endzaweni yeKukhonta
- Cinisekisa kutsi bavolontiya sebengetiwe ku[luhlu lwebantfu](../people/adding-people.md) futsi babelwe emacenjini

</div>

## Kufinyelela Etinhlelweni

1. Hamba uye ku-**Kukhonta** kusuka kumenyu lesikhulu.
2. Khetsa **lithebhu lemnyango** esicongweni sekhasi.
3. Cindzetela **luhlobo lweluhlelo** kute ubone luhlu lwetinhlelo taloluhlobo.
4. Cindzetela luhlelo lolukhetsekile kute ulivule.

:::info
Kufinyelela lokugcwele kwe-admin akudzingeki kute uphatse tinhlelo. Nobani lolilunga lemnyango angahamba aye ku-Kukhonta bese wakha, ahlele, futsi ahlele tinhlelo temnyango wakhe ngaphandle kwekudzinga lilungelo le-Plans Edit. Bahleli labanelilungelo le-Plans Edit bangaphatsa tinhlelo kuyo yonkhe imnyango.
:::

## Kwakha Luhlelo

1. Kusuka embukisweni wohlobo lweluhlelo, cindzetela **New Plan**.
2. Nika luhlelo ligama nobe usebentise lilanga njengeligama. Khetsa **lilanga** lenkonzo.
3. Nangabe ufuna kukopisha kusuka eluhlelweni lolwendlulile, khetsa tikhundla kuphela nobe tikhundla nesabelo. Nangabe ungafuni kukopisha, khetsa nje kute lutfo. Ungaphindze ukopishe lulandzelo lwenkonzo kusuka eluhlelweni lwami lolwendlulile.
4. Sindza luhlelo. Manje ungacala kwabela emalunga elicembu futsi wakhe [lulandzelo lwenkonzo](./service-order.md).

## Likhasi Lemininingwane Yeluhlelo

Nawuvula luhlelo, utawubona emathebhu lamabili:

- **Sabelo** -- Phatsa kutsi ngimalunga aphi elicembu lasabelwe kuloluhlelo. Ungakhona kwengeta bantfu kusuka emacenjini akho lasele akhona futsi ubone kutsi ngubani lolisicinisekisile nobe losalindzile.
- **[Kulandzelana Kwenkonzo](./service-order.md)** -- Yakha lulandzelo lwenkonzo ngetintfo letifana netingoma tekukhulekela, imithandazo, tatiso, kanye nentshumayelo.

## Kwabela Emalunga Elicembu

1. Vula luhlelo bese uya ku-thebhu ye-**Sabelo**.
2. Cindzetela **add Position** kute uyandise. Gcwalisa imininingwane ku-form yekwengeta sikhundla. Kuligama lesigaba, engeta noma yini loyifunako.
3. Cindzetela **People Needed** bese ukhetsa bavolontiya kute bagcwalise sikhundla.
4. Engeta emalunga kusuka eluhlwini lwelicembu lakho ngekucindzetela **Add**.
5. Emalunga lasabelwe atokwenta atovela ngaphansi kwelicembu lawo nesimo sabo sesabelo.
6. Cindzetela kwatisa bavolontiya kute ubatise ngekhatsi kwe-app ye-B1 nobe nge-imeyili.

:::tip
Setha emacembu akho ku-timiseto temnyango ngaphambi kwekwakha tinhlelo. Ngendlela lenjalo, utoba nesiciko lesilungele sebavolontiya lokungakhetswa kuso.
:::

## Tikhumbuto Tebavolontiya

B1 ingakhumbuta bavolontiya ngekutigcina ngaphambi kwetinkonzo labasabelwe kuto, kute ungabi nekulandzelela licembu lakho liviki ngeliviki. Tikhumbuto tiya **kuwo wonkhe losabelwe** — kokubili labo labacinisekisile kanye nalabo labangakaphenduli — nge-imeyili kanye njengesatiso lesingekhatsi-kwe-app/lesitfunyelwa efownini. Sikhumbuto ngasinye sifaka tikhundla temvolontiya, lilanga lenkonzo, emanothi eluhlelo, kanye nemlayeto wakho lokhetsekile.

Sikhatsi kanye nekucuketfwe kwesikhumbuto kusethwa nga **luhlobo lweluhlelo**, ngako luhlobo ngalunye lwenkonzo lungagcina luhlelo lwaso lolukhetsekile.

1. Kusuka endzaweni ye-**Kukhonta**, khetsa umnyango lonaloluhlobo lweluhlelo.
2. Cindzetela lithicho le-**hlela (pensela)** eceleni kwaloluhlobo lweluhlelo.
3. Kusigabeni se-**Reminders**, setha:
   - **Emalanga ekukhumbuta ngaphambi kwenkonzo** — luhlu loluhlukaniswe ngekhefu lwekutsi kufanele kutfunyelwe emalanga lamangaki ngaphambili, sibonelo `7,1,0`. Sebentisa `0` kute utfumele sikhumbuto ngelilanga lenkonzo. Shiya lensimu ingenalutfo kute ucise tikhumbuto kulolu hlobo lweluhlelo.
   - **Umlayeto wesikhumbuto lokhetsekile** *(ngekukhetsa)* — umutsi lowengetiwe esikhumbuteni, njengekutsi "Fika emaminithi langu-30 ngaphambili kute utilungiselele."
4. Sindza luhlobo lweluhlelo.

Tinhlobo tetinhlelo letitisha tikhumbuta bavolontiya **emalanga lamabili ngaphambili** kwenkonzo ngayinye ngekwehluleka kuze ulishintje loku.

:::tip
Bavolontiya labangakacinisekisi batfola tinkhinobho te-**Accept** kanye ne-**Decline** ngco ngekhatsi kwe-imeyili yesikhumbuto, kute bangaphendvula ngaphandle kwekungena.
:::

:::info
Sikhumbuto ngasinye sitfunyelwa kanye kuphela. Tinhlelo letisesetimeni lekulotjwa (lokungakatfunyelwa elicemjini) atisusi tikhumbuto.
:::

## Kuhlotjaniswa Kwemacembu Neluhlobo Lweluhlelo

Ngaphansi kweluhlu lwetinhlelo ekhasini lohlobo lweluhlelo, sigaba se-**Groups** sikuvumela kukhetsa kutsi ngimaphi emacembu langabona tinhlelo taloluhlobo lweluhlelo kusuka ku-portal yawo yemalunga. Loku yindlela lesheshako yekukhombisa tinkonzo letitako kumacembu lafanele ngaphandle kwekuwanika kufinyelela kwe-admin.

1. Ekhasini lohlobo lweluhlelo, yehlela phansi uye ku-sigaba se-**Groups**.
2. Cindzetela **Add Group** bese ukhetsa licembu ku-dropdown.
3. Ku-kholomu ye-**Shows**, khetsa kutsi emalunga aleliso licembu kufanele abone yini **Past**, **Future**, nobe **Both** tinhlelo taloluhlobo lweluhlelo.
4. Phindza kute uhlotjanise lamanye emacembu, nobe ucindzetele lithicho lekulahla kute ususe licembu.

:::info
Kuphela emacembu lamakiwe njenge-**Standard** avela ekukhetseni. Emalunga elicembu lelihlotjanisiwe atobona ngekutigcina tinhlelo talolu hlobo lweluhlelo kuthebhu ye-[Plans](/docs/b1-church/plans/) ku-portal yemalunga ye-B1 — kuvinjelwe kufingqi lolwendlulile/lolutako/kokubili lolukhetsile.
:::

## Kuphrinta Tinhlelo

Ungaphrinta luhlelo kute usabalalise elicemjini lakho. Vula luhlelo, vula thebhu yelulandzelo lwenkonzo bese usebentisa incenye ye-**Print** kute wakhe simo lesiphrintekako lesifaka sabelo kanye nelulandzelo lwenkonzo. Loku kuyasita nawukhipha etifundvweni nobe nawubeka endzaweni levamile.

:::info
Tinhlelo tihlelwe ngemnyango. Cinisekisa kutsi ukulithebhu lelifanele lemnyango ngaphambi kwekwakha nobe kubuka tinhlelo.
:::

## Tinyatselo Letilandzelako

- Sebentisa i-[Kubuka Konkhe Kwetinhlelo](./plans-overview.md) kute ubone yonkhe imisebenti letatawenteka emavikini lamaningi kuthebula linye futsi utfole tikhundla letingakagcwaliswa — futsi wabele bavolontiya ngco kusuka kuthebula
- Sindza sakhiwo saluhlelo njenge-[Sifanekiso Seluhlelo](./plan-templates.md) kute ulisebentise etinhlelweni letitako ngekucindzetela kanye
- Yakha [Kulandzelana Kwenkonzo](./service-order.md) yakho ngetingoma, kufundza, kanye netinye tintfo
- Engeta [tingoma](./songs.md) kusuka emtapweni wakho ngco kululandzelo lwenkonzo
- Sebentisa [Imisebenti](./tasks.md) kute wabele tintfo tekulandzela emalungeni elicembu
