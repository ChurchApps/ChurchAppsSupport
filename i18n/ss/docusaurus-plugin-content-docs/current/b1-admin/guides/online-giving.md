# Setinga Imnikelo Ye-Inthanethi

<div class="article-intro">

Hamba kuko konkhe lokudzingekako kute wemukele imnikelo ye-inthanethi ebandleni lakho — kusukela ekwakheni tikhwama temnikelo, kuya ekuxhumaneni ne-Stripe kwekucubungula tinkhokhelo, kuya ekwabelaneni ngekhasi lemnikelo nelicembu lakho. Nasekugcineni, emalunga atawukwati kunikela nge-inthanethi kusebentisa iwebhusayithi ne-app yeselula yakho.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- I-akhawunti ye-B1 Admin lenekufinyelela kwemphatsi — buka [Tindzima & Imvumo](../people/roles-permissions.md)
- I-akhawunti ye-Stripe (yakha lenye mahhala ku-[stripe.com](https://stripe.com) nangabe kudzingeka)

</div>

## Sinyatselo 1: Yakha Tikhwama Temnikelo

Tikhwama tinhlobo tebasunikeli labangaphela kuto. Kudzingeka ube nesikhwama linye kucala ngembi kwekutsi wemukele imnikelo.

Landzela umhlahlandlela we-[Tikhwama](../donations/funds.md) kute uhlele tigaba takho temnikelo:

1. Yakha tikhwama takho letijwayelekile kakhulu (sib., "Sikhwama Lesijwayelekile", "Sikhwama Sekwakha", "Emisebenti Yekushumayela")
2. Maka tikhwama letikhululekile entelweni ngendlela lefanele — loku kutfinta tincwadzi temnikelo takwelicala lomnyaka

:::tip
Ungangeta tikhwama letinyenti nome nini. Cala ngetigaba takho temnikelo letijwayelekile kakhulu.
:::

## Sinyatselo 2: Xhumana Ne-Stripe

I-Stripe icubungula tonkhe tinkhokhelo. Utawuxhumanisa i-akhawunti yakho ye-Stripe ne-B1 Admin kute imnikelo igcwale ku-akhawunti yakho yebhange.

Landzela umhlahlandlela we-[Kuhlela Imnikelo Ye-Inthanethi](../donations/online-giving-setup.md) kute uxhumane ne-Stripe:

1. Ngena ku-dashbhodi yakho ye-Stripe bese uthola Sikhiya Sakho Sekubonakala kanye Nesikhiya Sakho Semfihlo
2. Ku-B1 Admin, yiya kuTilungiselelo bese ufaka tikhiya tobabili

:::warning
I-Stripe iveta Sikhiya Sakho Semfihlo kanye kuphela. Kopisha uyigcine ngembi kwekushiya i-dashbhodi ye-Stripe. Nangabe ulahlekelwa yiso, kudzingeka wente lesinye lesisha.
:::

## Sinyatselo 3: Faka Likhasi Lemnikelo Kuwebhusayithi Yakho

Yenta imnikelo itfolakale kalula ngekufaka likhasi lemnikelo kuwebhusayithi yakho ye-B1.

Landzela imihlahlandlela ye-[Kuhlela Imnikelo Ye-Inthanethi](../donations/online-giving-setup.md) ne-[Kuphatsa Emakhasi](../website/managing-pages.md) kute:

1. Faka ithebhu "Nikela" kusayithi yakho ye-B1.church
2. I-URL yakho yemnikelo itoba: `https://yoursubdomain.b1.church/donate`
3. Emalunga angakwati kunikela ngaphandle kwekungena (likhasi lemphakatsi) nobe angene kute atsatse tindlela tekukhokha letigciniwe kanye nemlandvo wemnikelo

## Sinyatselo 4: Yenta Umnikelo Wekuhlola

Ngembi kwekumemetela kulicembu lakho, cinisekisa kutsi konkhe kusebenta.

1. Yenta umnikelo lomncane wekuhlola kute ucinisekise kutsi luhambo lonkhe lusebenta kusukela ekucaleni kuya ekugcineni
2. Hlola kutsi umnikelo uyavela ku-B1 Admin ngaphansi kweMnikelo

:::tip
Sebentisa simo sekuhlola se-Stripe kucala nangabe ufuna kucinisekisa ngaphandle kwetinkhokhelo takahle, bese ushintjela kusimo lesisebentako ngembi kwekumemetela kulicembu lakho.
:::

## Sinyatselo 5: Memetela Kulicembu Lakho

Sakata indzaba kute emalunga atati kutsi angakwati kunikela nge-inthanethi.

1. Yabelana nge-URL yemnikelo nge-website, ema-imeyili, ticwatibho, kanye ne-social media
2. Emalunga angakwati futsi kunikela nge-[app ye-B1 Mobile](../../b1-mobile/giving/) — sici semnikelo sakhiwe ngekhatsi

:::info
Emalunga langena angagcina tindlela tekukhokha, ahlele imnikelo lephindaphindakako, futsi abuke umlandvo wawo wemnikelo. Kunikela ngaphandle kwelibito nako kuyasebenta — akudzingeki kungena.
:::

## Sinyatselo 6: Kuphatsa Lokuchubekako

Gcina emarekhodi akho emnikelo alungile futsi wente imibiko unyaka wonkhe.

1. [Ngenisa tinkhokhelo te-Stripe](../donations/stripe-import.md) njalo (ngesonto nobe ngenyanga) kute emarekhodi akho asale alungile
2. [Buka imibiko yemnikelo](../donations/donation-reports.md) kute ulandzelele tinjani temnikelo kanye netamba ngesikhwama
3. [Yenta tincwadzi temnikelo telicala lomnyaka](../donations/giving-statements.md) kwemarekhodi entela yebanikeli bakho

:::tip
Yenta kungenisa kwe-Stripe kungenani ngenyanga kute emarekhodi akho asale asha. Buka [Umhlahlandlela Wemibiko Yelicala Lomnyaka](./year-end-reports.md) kutfola inchubo yonkhe yelicala lomnyaka.
:::

## Usucedzile!

Libandla lakho selivemukela manje imnikelo ye-inthanethi. Emalunga angakwati kunikela nge-website yakho, i-app ye-B1 Mobile, nobe nome ngukuphi luchungechunge lolunebhrawuza. Yonkhe imnikelo ilandzelelwa ngekutinelela ku-B1 Admin.
