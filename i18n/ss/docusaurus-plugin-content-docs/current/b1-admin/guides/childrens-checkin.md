# Setinga Kungena Kwebantfwana Enkonzweni

<div class="article-intro">

Lomhlahlandlela ukuhola kuko konkhe lokudzingekako kute kutfole luhlelo lwekungena kwebantfwana lolusebentako ebandleni lakho — kusukela ekufakeni imindeni kudatabhesi, kuya ekuhleleni emacembu lahambisana neminyaka, kuya ekushicileleni emathegi elibito ngeliSontfo ekuseni. Nasekugcineni, batali batawukhona kungenisa bantfwababo kwi-kiosk yeliteblethi futsi batfole ithegi yekuphepha lehambisana nayo.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- Yakha i-akhawunti yelibandla lakho ku-[admin.b1.church](https://admin.b1.church)
- Cinisekisa kutsi unako kufinyelela kwemphatsi — buka [Tindzima & Imvumo](../people/roles-permissions.md) nangabe kudzingeka
- Kungakhetseka: Lungisa lifayela le-CSV lemindeni nangabe udzimula kusuka kolunye luhlelo

</div>

## Sinyatselo 1: Faka Imindeni Kudatabhesi Yakho

Ngembi kwekutsi kungena kusebente, luhlelo kudzingeka kwati ngemindeni yakho. Ngamunye umntfwana kudzingeka ahlotjaniswe nemtali kusebentisa insita yemuti.

Landzela umhlahlandlela we-[Kufaka Bantfu](../people/adding-people.md) kute ufake umndeni longeti munye. Cinisekisa kutsi:

- Faka batali kucala
- Faka wonkhe umntfwana
- Bahlanganise emutini munye kusebentisa [sihleli semuti](../people/adding-people.md#managing-households)

:::tip
Nangabe unemindeni leminyenti lokufanele uyifake, sebentisa lithulusi le-[CSV Import](../people/importing-data.md) esikhundleni sekufaka munye ngamunye. Ungangenisa lonkhe luhlu lwakho lwebantfu ngemaminithi lambalwa.
:::

## Sinyatselo 2: Yakha Emacembu Ebantfwana

Emacembu achaza emakilasi bantfwana labangena kuwo. Ngalokujwayelekile utawufuna licembu linye ngasibanga ngasinye seminyaka.

Landzela umhlahlandlela we-[Kudala Emacembu](../groups/creating-groups.md) kute udale emacembu lafana na:

- **Umnyango Wetinsana** (iminyaka 0–2)
- **Ngembi Kwesikolwa** (iminyaka 3–5)
- **Ibanga Laphansi** (iminyaka 6–10)

Ungashintja emabito nemabanga eminyaka kute kuhambisane naluhlelo lwenkonzo yakho.

## Sinyatselo 3: Hlela Emakhempasi Netinkonzo

Kungena kuhlotjaniswe netikhatsi letitsite tenkonzo. Kudzingeka ube nelikhempasi linye nenkonzo linye lehleliwe.

Landzela umhlahlandlela we-[Kuhlela Kubakhona](../attendance/setup.md) kute:

1. Faka likhempasi lakho (sib., "Likhempasi Lelikhulu")
2. Faka inkonzo (sib., "LiSontfo Ekuseni")
3. Setinga sikhatsi senkonzo (sib., "9:00 Ekuseni")
4. Nika emacembu ebantfwana bakho kuloyo nkonzo

## Sinyatselo 4: Setinga Uhlelo Lwekungena

Manje hlanganisa konkhe ngekufaka uhlelo lwekungena kwi-tablet.

1. Faka **i-app ye-B1 Checkin** — buka sihloko se-[Kungena](../attendance/check-in.md) kutfola emalinki ekulanda
2. Ngena ngemvumo yakho ye-B1 Admin
3. Khetsa likhempasi lakho nesikhatsi senkonzo

Buka sihloko lesigcwele se-[Kungena](../attendance/check-in.md) kutfola tinyatselo letigcwele tekuhlela.

## Sinyatselo 5: Tfola Emathulusi Akho

Udzinga i-tablet ye-kiosk kanye, ngekukhetsa, i-printha yemalebula ye-Brother yemathegi elibito.

Okuncane lokudzingekako:
- **I-tablet ye-Android nobe ye-Amazon Fire** — buka [emathebulethi lancoketwe](../attendance/check-in.md#recommended-hardware)
- **I-printha yemalebula ye-Brother** — i-QL-1110NWB iyancoketwa ngekusekela kwayo i-Bluetooth ne-WiFi
- **Emalebula e-Brother DK-1201** (1-1/7" x 3-1/2")

:::warning
Ngutimphatha temalebula te-Brother kuphela letihambisana ne-app ye-B1 Checkin. Letinye timphatha temaphrintha atisebenti.
:::

## Sinyatselo 6: Yenta Kuhlola Kwekungena

Ngembi kweliSontfo ekuseni, yenta kuhlola:

1. Vula i-app ye-B1 Checkin ku-tablet yakho
2. Khetsa likhempasi lakho nesikhatsi lesifanele senkonzo
3. Sesha lomunye wemindeni lowake wayifaka
4. Ngenisa umntfwana bese ucinisekisa kutsi:
   - Kubakhona kuvela ku-B1 Admin ngaphansi kwe-**Kubakhona**
   - Nangabe usebentisa iphrintha, ithegi yelibito ishicilela kahle

:::tip
Cecesha emavolontiya elicembu lekwemukela ngendlela yekungena ngembi kwekucalisa. Kuhamba kancane kwemaminithi lasihlanu ngalokujwayelekile kuyoke.
:::

## Usucedzile!

Kungena kwenkonzo yebantfwana bakho sekulungile. Batali bangasesha umndeni wabo, bakhetse bantfwababo, bangene kwi-kiosk. Kubakhona kubhalwe ngekutinelela ku-B1 Admin.
