# Casisa Iwebhusayithi Yelibandla Lakho

<div class="article-intro">

B1.church ifaka umakhi longalo webhusayithi lophelele ngaphandle kwelidzinga elingetiwe. Lomhlahlandlela ukuhola ekwakheni iwebhusayithi yelibandla lakho kusukela ekucaleni — kuhlela likhasi lakho lekucala, kuhlela sitfo sakho, kufaka emakhasi labalulekile, kanye ngekukhetsa kuhlanganisa imnikelo ye-inthanethi kanye nemafomu ekubhalisa emicimbi.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- Yiba nelogo yelibandla lakho lilungile (i-PNG lenesikhundla lesibonakalako ivele ingcono)
- Khetsa imibala lemi-2–3 yebrendi yesayithi yakho
- Nangabe usebentisa idomeini yakho leyakho (sib., yourchurch.com), yiba nekufinyelela ku-DNS provider yakho (GoDaddy, Cloudflare, njalonjalo)
- Nangabe ufuna imnikelo ye-inthanethi kusayithi yakho, cedza [Kuhlela Imnikelo Ye-Inthanethi](../donations/online-giving-setup.md) (Stripe) kucala

</div>

## Sinyatselo 1: Kuhlela Kwekucala Kwewebhusayithi

Cala ngekwakha likhasi lakho lekucala kanye nesakhiwo sesisekelo sesayithi.

Landzela umhlahlandlela we-[Kuhlela Kwekucala Kwewebhusayithi](../website/initial-setup.md) kute:

1. Yiya ku-**Website** ku-B1 Admin
2. Yakha likhasi lakho lekucala nge incenye lenkhulu, umlayeto wekwemukela, kanye nemininingwane lebalulekile
3. Faka libito lelibandla lakho kanye nemagama lamafisha

## Sinyatselo 2: Hlela Sitfo

Setinga sibonelo lesibonakalako sesayithi yakho — imibala, tinhlobo temagama, ilogo, ne-footer.

Landzela umhlahlandlela we-[Sitfo](../website/appearance.md) kute:

1. Layisha ilogo yelibandla lakho
2. Setinga umbala wakho wekucala kanye nemibala lehambisanako
3. Hlela ibha yekuzulazula ne-footer
4. Buka lokushintjile ngembi kwekushicilela

:::tip
Gcina imibala yakho ilula — umbala munye wekucala kanye nombala munye lohambisanako uvamise kwenele. Umakhi wewebhusayithi utawenta lokunye konkhe.
:::

## Sinyatselo 3: Faka Emakhasi Elwati

Yakha emakhasi labavakashi bakho labawadzingako kakhulu.

Landzela umhlahlandlela we-[Kuphatsa Emakhasi](../website/managing-pages.md) kute wakhe emakhasi lafana na:

- **Ngatsi** — indzaba yelibandla lakho, tikholelwo, nebaholi
- **Tintshumayelo** — linki eku-[luhla lwetintshumayelo](../sermons/managing-sermons.md)
- **Emicimbi** — emicimbi letako kanye nekubhalisa
- **Nikela** — likhasi lemnikelo ye-inthanethi (kudzinga [kuhlela kwe-Stripe](../donations/online-giving-setup.md))
- **Sixhumana** — indzawo, tikhatsi tenkonzo, nemininingwane yekuxhumana

## Sinyatselo 4: Xhumana Nedomeini Yakho

Nangabe ufuna kusebentisa libito lakho ledomeini (njenge yourchurch.com) esikhundleni se-URL yekucala ye-B1:

1. Yiya ku-**Website > Settings** ku-B1 Admin
2. Faka idomeini yakho leyakho
3. Buyeketa emarekhodi akho e-DNS ku-domain provider yakho kute akhombe ku-B1

:::info
Kushintja kwe-DNS kungatsatsa emahora langaba ngu-48 kute kusakateke. Isayithi yakho ingase ingatfolakali kusuka kudomeini yakho leyakho ngekushesha. I-URL yekucala ye-B1 itawuchubeka isebenta ngesikhatsi leso.
:::

## Sinyatselo 5: Faka Imnikelo Nemafomu

Chumisa isayithi yakho ngetintfo letiphindaphindako:

- **Imnikelo Ye-Inthanethi** — Faka incenye yemnikelo kute emalunga akwati kunikela ngco kusuka kuwebhusayithi yakho. Buka [Kuhlela Imnikelo Ye-Inthanethi](../donations/online-giving-setup.md) kute uhlele Stripe kucala.
- **Emafomu Ekubhalisa** — Faka [emafomu lalizimele](../forms/creating-forms.md) kwekubhalisa emicimbi, emakhadi evakashi, nobe kufaka sicelo semavolontiya. Buka [Kuphatsa Emakhasi](../website/managing-pages.md) kutfola kutsi ungayifaka njani i-elementi yefomu kunome ngukuphi likhasi.

## Usucedzile!

Iwebhusayithi yelibandla lakho isisebenta. Yabelana nge-URL nelicembu lakho kanye ne-social media. Ungashintja lokucushiwe, ufake emakhasi lamasha, futsi ulungise sitfo nanini kusuka ku-dashbhodi ye-B1 Admin.
