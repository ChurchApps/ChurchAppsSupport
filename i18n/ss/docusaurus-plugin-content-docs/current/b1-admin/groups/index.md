# Emaqembu

<div class="article-intro">

Sigaba se-**Emaqembu** sikuvumela kutsi uhlele emalunga ebandla lakho abe ngemasonto lasematiko, emaqembu lamancane, tifundvo teliBhayibheli, emakomiti, nakunye lokunyenti. Emaqembu sisekelo semphilo yebumbano ebandleni lakho, futsi lesigaba sikunika emathulusi ekudala, kuphatsa, nekukhulumisana nawo onkhe emaqembu endzaweni yinye. Caphela kutsi **Tinhlangano** tehlukile Kumaqembu — Tinhlangano tisetjentiselwa kukhonta netekuhlela ekuzitandeleni.

</div>

## Kubukwa Konkhe

Emaqembu ahlelwe nge-**sigaba**. Sigaba luphawu lolungetulu lofana ne-"Emaqembu Lamancane," "Emasonto," nobe "Emakomiti." Ngasinye sigaba singaba nemaqembu lamanyenti, lokwenta kube lula kugcina emaqembu lahambelanako endzaweni yinye.

Nawuvula likhasi Lemaqembu, utawubona sifingqo etulu lesikhombisa:

- **Emaqembu Onkhe** -- linani lemaqembu kuwo onkhe tigaba
- **Tigaba** -- linani letigaba temaqembu losulidalile
- **Emalunga Onkhe** -- linani lelihlangene lemalunga kuwo onkhe emaqembu

Ngaphansi kwesifingqo, emaqembu akho ahlelwe ngesigaba nelinani lemalunga likhonjiswa kulo lonkhe liqembu.

## Loko Longakwenta

Naku loko longakufeza kusigaba se-Emaqembu:

1. **Dala futsi uhlele emaqembu** -- Engeta emaqembu lamasha ngasitigaba kute akhombise indlela libandla lakho lihlelekile ngayo. Bona [Kudala Emaqembu](./creating-groups.md).

2. **Phatsa emalunga eliqembu** -- Engeta futsi ususe emalunga, ukhombise baholi, futsi ukhulume nemaqembu akho. Bona [Emalunga Eliqembu](./group-members.md).

3. **Hlela imicimbi** -- Ngasinye liqembu lingaba nelikhalendari lalo lemihlangano, imicimbi, netentfo letiphindzaphindzekako. Bona [Likhalendari Leliqembu](./group-calendar.md).

4. **Phatsa tesicelo tekungenela** -- Nangabe liqembu lisebentisa umtsetfo wekungenela we-**Sicelo**, buka bese uyavumela nobe uyenqaba tesicelo letingenako tekungena letivela kubantfu labafuna kungenela. Bona [Tesicelo Tekungenela Eliqenjini](./group-join-requests.md).

5. **Khiphela imininingwane yeliqembu** -- Layisha luhlu lwemalunga ngalo lonkhe liqembu njengelifayela le-CSV lekusetjentiswa emaphepheni ekubalwa nobe kutfunyelwa. Bona [Emalunga Eliqembu](./group-members.md) kutfola tinyatselo tekukhipha.

## Kucala

Kufinyelela Emaqenjini, chofota **Bantfu** kulohlu lolusesandleni sancele senkomba yakho ye-B1 Admin, bese ukhetsa **Emaqembu**. Utawubona onkhe emaqembu akho lakhona lahlelwe ngesigaba.

:::tip
Nangabe usacala, cala ngekudala tigaba letimbalwa letihambelana nendlela libandla lakho lakheke ngayo (sibonelo, "Emasonto," "Emaqembu Lamancane," "Emakomiti"). Bese ungeta emaqembu lodvwa ngasitigaba. Bona [Kudala Emaqembu](./creating-groups.md) kutfola indlela lelandzelwako ngayinye.
:::

:::info
Emalunga eliqembu kufanele abe khona kucalanga kulushicilelo lwakho lwe-[Bantfu](../people/adding-people.md) ngaphambi kwekutsi angetwe eliqenjini. Nangabe usungisa i-B1 kwekucala, [ngenisa imininingwane yemalunga akho](../people/importing-data.md) ngaphambi kwekudala emaqembu.
:::
