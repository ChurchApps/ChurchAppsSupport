# Emalunga Eliqembu

<div class="article-intro">

Nawukadala liqembu, sinyatselo lesilandzelako kungeta emalunga. Kusuka ekhasini lemininingwane yeliqembu ungafuna bantfu, ubengete eliqenjini, ukhombise baholi, utfumele imilayeto, futsi ukhiphe luhlu lwemalunga. Kuphatsa bulunga beliqembu kubalulekile ekuhleleni emaqembu lamancane, emakomiti, netikilasi.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga sikhatsi lesilodwa selicembu lelisungiwe ku-B1 Admin. Bona [Kudala Emaqembu](creating-groups.md) nangabe akukho lolidalile.
- Bantfu lofuna kubengeta kufanele babe khona kucala kulushicilelo lwakho lwe-[Bantfu](../people/adding-people.md).

</div>

## Kungeta Emalunga Eliqenjini

1. Yiya kulikhasi le-**Emaqembu** bese uchofota liqembu lofuna kulisebentisa.
2. Chofota ithebu **Emalunga**.
3. Ebhokisini lekufuna, bhala ligama lemuntfu lofuna kumungeta.
4. Chofota **Ngeta** eceleni kwaligama lalomuntfu emiphumeleni yekufuna.
5. Lomuntfu manje uvela kuluhlu lwemalunga eliqembu.

:::tip
Shiya libhokisi lekufuna lingenalutfo bese uchofota **Funa** kute ubuke lonkhe lushicilelo lwakho. Loku kuyasita nangabe awucacisisi indlela lelibhalwa ngayo ligama lemuntfu.
:::

## Kukhombisa Baholi Beliqembu

Baholi beliqembu banemalungelo lakhetsekile -- bangahlela [likhalendari leliqembu](group-calendar.md), baphatse imicimbi, futsi basite kuhlela liqembu.

1. Kuluhlu lwemalunga eliqembu, funa umuntfu lofuna kummenta abe ngumholi.
2. Chofota **luphawu lwelikhono leliluhlata** eceleni kwaligama lakhe.
3. Lomuntfu manje ukhonjiswe njengumholi weliqembu.

Kususa simo semholi, chofota luphawu lwelikhono leliluhlata futsi.

:::info
Nome ngubani lilunga leliqembu angabuka likhalendari lemicimbi yeliqembu, kepha ngubaholi kuphela labangengeta nobe bahlele imicimbi yelikhalendari.
:::

## Kutfumela Imilayeto Kumalunga Eliqembu

Ungakhulumisana nawo onkhe emalunga eliqembu ngalokucondzile kusuka ku-B1 Admin:

1. Kusuka ekhasini lemininingwane yeliqembu, funa indzawo yemlayeto.
2. Bhala umlayeto wakho ebhokisini lembhalo.
3. Chofota **Tfumela**.

Umlayeto wakho utawufinyelela kuwo onkhe emalunga eliqembu.

## Kutfumela Ema-imeyili Kumalunga Eliqembu

Ungatfumela ema-imeyili lahlelekile kuwo onkhe emalunga eliqembu:

1. Kusuka ekhasini lemininingwane yeliqembu, chofota **luphawu lwe-imeyili**.
2. Ibhokisi lekutfumela i-imeyili liyavuleka, likhombisa kutsi malunga lamangaki latawutfola i-imeyili nekutsi lamangaki angena i-imeyili lebhalisiwe.
3. Ungakhetsa i-**thempuleti ye-imeyili** kuluhlu, nobe wakhe umlayeto kusukela ekucaleni. Chofota **Phatsa Emathempuleti** kutsi udale nobe uhlele emathempuleti.
4. Bhala **umutsi wemlayeto**. Ungafaka emabhokisi ekuhlanganisa ngekuchofota emalungu lachaza inkhomba: `{{firstName}}`, `{{lastName}}`, `{{displayName}}`, `{{email}}`, `{{churchName}}`.
5. Bhala **umutimba we-imeyili** usebentisa i-HTML editor. Lamalungu lafanako lekuhlanganisa ayatfolakala nalapha.
6. Chofota **Tfumela**.
7. Sifingqo sikhombisa kutsi ema-imeyili lamangaki latfunyelwe ngalempumelelo nekutsi malunga lamangaki asalele (angenayo i-imeyili lebhalisiwe).

:::tip
Dala emathempuleti e-imeyili langasetjentiswa kaningi ekukhulumisaneni lokuphindzaphindzekako njengetibiko taviki, tati so temicimbi, nobe tesicelo temithandazo. Emathempuleti agcina sikhatsi futsi acinisekisa umlayeto lofananako.
:::

## Kukhipha Imininingwane Yeliqembu

Kukhiphela luhlu lwemalunga eliqembu njengelifayela:

1. Kusuka ekhasini lemininingwane yeliqembu, chofota **luphawu lwekukhipha**.
2. Lifayela le-CSV leliphetse imininingwane yemalunga eliqembu lelikhishwa likhishwe kukhompyutha wakho.

Loku kuyasita ekudaleni luhlu loluphrintiwe, ekungeniseni imininingwane kwamanye emathulusi, nobe kugcina imiyalo lengelona umuncu we-inthanethi. Kutfola letinye tinketfo tekukhipha, bona [Kukhipha Imininingwane](../people/exporting-data.md).

## Kutfumela Tati so Letishukumako Kumalunga Eliqembu

Ungatfumela satiso lesishukumako ngalokucondzile kuwo onkhe emalunga eliqembu lange-B1.church app lefakwe kudivayisi lwabo nesati so leshukumako livuliwe.

1. Kusuka ekhasini lemininingwane yeliqembu, chofota **luphawu lwensimbi** kubar yemathulusi ekhasini (eceleni kwaluphawu lwe-imeyili neye-SMS).
2. Ibhokisi liyavuleka likhombisa kutsi malunga lamangaki eliqembu lakho anesati so leshukumako livuliwe.
3. Gcwalisa imininingwane yesati so:
   - **Sihloko** *(kudzingekile)* -- Sifingqo lesifisha, kuya kuma-80 wetimpawu.
   - **Umlayeto** *(kudzingekile)* -- Umtimba wesati so, kuya kuma-240 wetimpawu.
   - **Vula i-link nobe i-URL ye-flyer** *(kwekukhetsa)* -- Indlela le-app lehambisana nayo (sibonelo, `/mobile/groups`) nobe i-URL legcwele ye-`https://` lekuvulwa yisati so nasichojwa.
   - **I-URL yesitfombe** *(kwekukhetsa)* -- I-URL ye-`https://` yesitfombe lesivela eceleni kwesati so kumadivayisi lasekelekile.
4. Kubuka lokuphilako kukhombisa indlela sati so lesitawukhonjiswa ngayo kudivayisi.
5. Chofota **Tfumela Satiso**.

:::info
Tati so letishukumako titfunyelwa kuphela kumalunga eliqembu lanefakelwe i-B1.church PWA futsi labangazivimbanga tati so letishukumako. Emalunga langenayo idivayisi lebhalisiwe yesati so nobe lanesati so lesivaliwe abalwa njengalasalele, futsi sifingqo sekutfumela sikhombisa kutsi bangaki labafinyelelekile bahlangana nalabasalele.
:::

:::tip
Ngemuva kwekutfumela, ibhokisi likhombisa kutsi satiso lesingaki lesibekwe kuhla ngalempumelelo. Nangabe emalunga lamanyenti akhonjiswa asalele, bakhumbute kutsi bavakashele indzawo yabo ye-B1.church, bayifakele njengekhasi lasekhaya kudivayisi, futsi bavumele tati so nabacelwa.
:::

## Kususa Emalunga

Kususa umuntfu eliqenjini, funa ligama lakhe kuluhlu lwemalunga bese uchofota inkhinombho ye-**kususa** eceleni kwemibhalo yakhe.

:::info
Kususa umuntfu eliqenjini akumsuli kulushicilelo lwebandla lakho. Utawuchubeka avele ku-[Bantfu](../people/adding-people.md) futsi angengetwa futsi eliqenjini nome ngasiphi sikhatsi.
:::
