# Simo Semaqembu

<div class="article-intro">

Inkomba ye-Simo Semaqembu ikunika umbono lophakeme wendlela onkhe emaqembu akho atiphatsa ngayo — inkambiso yebulunga, umyinge wekubakhona, nekukhula nobe kwehla eminyakeni lengu-90 leyendlulile — konkhe kuthebula linye lelingahlelwa.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga kungenani emaqembu lambalwa lanemalunga kute ubone imininingwane lenengcondzo. Bona [Kudala Emaqembu](creating-groups).
- Imininingwane yekubakhona itsatfwa kutimhlangano letibhalisiwe. Bona sigaba se-[Kubakhona](../attendance/).

</div>

## Kuvula Simo Semaqembu

Ku-B1 Admin, yiya ku-**Emaqembu** kulohlu lolusesandleni sancele bese ukhetsa **Simo**. Inkomba iyalayisha ithebula lelinemugca munye ngaliqembu.

## Emakholomu

| Ikholomu | Loko lekukhombisako |
|--------|--------------|
| **Ligama** | Ligama leliqembu, lelihlanganiswe nekhasi lemininingwane yeliqembu |
| **Sigaba** | Sigaba seliqembu |
| **Emalunga** | Linani lemalunga lasebentako langekwamanje |
| **Angene (emalanga langu-90)** | Emalunga langene eminyakeni lengu-90 leyendlulile |
| **Ashiyile (emalanga langu-90)** | Emalunga lashiyile eminyakeni lengu-90 leyendlulile |
| **Kwehla (emalanga langu-90)** | Linani lekwehla kwemalunga njengephesenti eminyakeni lengu-90 |
| **Umyinge Wekubakhona** | Umyinge welinani lebantfu ngamunye kumhlangano wekubakhona |

Chofota nome ngusiphi sihloko sekholomu kute uhlele ithebula ngaleyokholomu. Chofota futsi kute uphendvule luhlelo.

## Kusebentisa Imininingwane Yesimo

- **Kwehla lokukhulu + kungena lokuncane** — liqembu lelincipha nelingatfoli emalunga lamasha ekushintsa lashiyile. Kufanele kubonisane nemholi weliqembu.
- **Kungena lokukhulu + kubakhona lokuncane** — bantfu bayabhalisa kepha abeti. Cabanga ngekulandzelela kubandzakanyeka.
- **Umyinge wekubakhona lophakeme** — liqembu leliphilile, lelisebentako. Lingaba sibonelo kumanye emaqembu.

:::tip
Kuchofota ligama leliqembu kukuyisa ngalokucondzile ekhasini lemininingwane yeliqembu lapho ungaphenya emalunga lodvwa, imibhalo yekubakhona, netetamo tekhalendari.
:::
