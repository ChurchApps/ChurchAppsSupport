# Likhalendari Leliqembu

<div class="article-intro">

Nganye liqembu ku-B1 lingaba nelikhalendari lalo lemicimbi. Baholi beliqembu bayalisebentisa kuhlela imihlangano, imisebenti, netentfo letiphindzaphindzekako, ngesikhatsi onkhe emalunga eliqembu angabuka imicimbi letako futsi ajoyine tinkhulumo mayelana nayo. Likhalendari ligcina liqembu lakho lihleliwe futsi licinisekisa kutsi wonkhe umuntfu uhlala atiwa ngaloko lokutako.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga liqembu lelidaliwe ku-B1 Admin lelinelilunga lilinye kancane. Bona [Kudala Emaqembu](creating-groups.md) ne-[Emalunga Eliqembu](group-members.md).
- Kungeta nobe kuhlela imicimbi yelikhalendari, kufanele ukhonjiswe njengumholi weliqembu. Bona [Kukhombisa Baholi Beliqembu](group-members.md) kutfola imiyalo.

</div>

## Kufinyelela Likhalendari Leliqembu

Likhalendari leliqembu litfolakala ngale-B1.church app lekhombene nemsebentisi:

1. Kusuka ku-B1 Admin, chofota **imenyu lengumbukiso** ekhoneni langesekudla etulu esikhrini sakho.
2. Khetsa **Shintjsa i-App**.
3. Khetsa **B1.church**.
4. Chofota **Emaqembu** kulohlu lolusesandleni sancele.
5. Khetsa liqembu lakho kuluhlu.
6. Chofota **Likhalendari** ngasesandleni sancele sesikhrini.

:::info
Likhalendari leliqembu liphatswa nge-B1.church (i-app yemalunga) kunale-B1 Admin. Baholi beliqembu bafinyelela kilo ngekushintjsa ema-app kusuka enkombeni yekuphatsa.
:::

## Kungeta Umcimbi

1. Kusuka elikhalendarini leliqembu, chofota **Ngeta Umcimbi**.
2. Faka **sihloko** semcimbi.
3. Bhala **incazelo** lemfishane yekutsi umcimbi umayelana naloku.
4. Beka **sikhatsi sekucala** nesikhatsi sekuphela.
5. Chofota **Gcina** kute wengete umcimbi elikhalendarini.

## Kuhlela Tetamo Letiphindzaphindzekako

Nangabe liqembu lakho lihlangana ngesikhatsi lesihlelekile, ungabeka tetamo kutsi tiphindze ngekutitjhentisa:

1. Nawudala nobe uhlela umcimbi, hlolisa libhokisi le-**Kuphindzaphindzeka**.
2. Khetsa indlela umcimbi ofanele uphindze ngayo:
   - **Sikhala** -- Beka linani lemalanga emkhatsini wekwenteka. Sibonelo, kufaka "7" kuphindza umcimbi njalo ngemalanga langu-7.
   - **Kwenteka** -- Khetsa kuphindza nge-**lilanga**, **liviki**, nobe **inyanga**.
3. Gcina umcimbi. Manje utawuvela elikhalendarini kuko konkhe kwenteka lokuhleliwe.

:::tip
Kuliqembu lelihlangana njalo ngeMgcibelo, beka kwenteka kube "liviki" bese ukhetsa liMgcibelo. Umcimbi utawutigcwalisa ngekutigcwalisa elikhalendarini kusukela nyalo kuya embili.
:::

## Kudala Tetamo Ngebunyenti ku-B1 Admin

Likhasi leliqembu ku-B1 Admin liphindze libe nethebu le-**Likhalendari** lapho bantfu labasebentako bangahlela lonkhe sizini semihlangano ngasikhatsi sinye:

1. Vula liqembu ku-B1 Admin bese ukhetsa ithebu le-**Likhalendari**.
2. Chofota **Ngeta Tetamo** kute uvule ibhokisi le-**Ngeta Tetamo Letiphindzaphindzekako**.
3. Khetsa sikhatsi (njalo ngeviki, njalo ngemaviki lamabili, njalo ngenyanga, nobe sikhatsi sinye), emalanga e-**Umhlangano Wekucala** ne-**Umhlangano Wekugcina**, kwekukhetsa **Yeka emaholideni**.
4. Buka kubuka kwemalanga emihlangano bese ususa luphawu kuloko lofuna kukuyeka.
5. Shiya **Vumela ema-RSVP** luphawu nangabe ufuna kutsi emalunga akhone kuphendvula letemcimbi (bona ngentasi), bese ugcina.

## Ema-RSVP Emicimbi

Imicimbi yeliqembu ingacoca ema-RSVP — **Yebo**, **Cha**, nobe **Mhlawumbe** kuko kwenteka ngakunye kwemcimbi.

- **Vumela ema-RSVP** livuliwe ngekwehluleka kuletemcimbi letidalwe ngebhokisi lekungeta ngebunyenti; susa luphawu kute uvale kuphendvula kuletemcimbi. Temcimbi letinema-RSVP avaliwe atikhonjiswa njenge-**Kuvaliwe** ethebu lelikhalendari.
- Ethebu lelikhalendari leliqembu ku-B1 Admin, ngamunye umcimbi ukhombisa sifingqo semphendvulo (sibonelo, "5 yebo · 2 mhlawumbe · 1 cha"). Chofota kuvula luhlu lwe-**Miphendvulo Ye-RSVP**, loluluhlu lolubhala kutsi ngubani lophendvule yebo, mhlawumbe, nobe cha kuloko kwenteka.

:::tip
Ema-RSVP anika baholi linani lebantfu ngaphandle kwendlela legcwele yekubhalisa umcimbi. Kumabhalisela lakhokhelwako nobe imicimbi lelawulwa ngesikhala, sebentisa [kubhalisa umcimbi](../calendars/creating-calendars.md#enabling-event-registration) esikhundleni saloko.
:::

## Kubuka Likhalendari

Likhalendari leliqembu linika tinketfo letinyenti tekubuka:

- **Inyanga** -- Bona inyanga yonkhe ngalikhatsana lelilinye.
- **Liviki** -- Gcizelela kuliviki lanyalo.
- **Lilanga** -- Buka lilanga linye ngalokugcwele.
- **Luhlu Lwemicimbi** -- Buka luhlu lwemicimbi letako ngekulandzelana kwesikhatsi.

Shintjsa emkhatsini wekubuka usebentisa emabhatani laphezulu elikhalendarini.

## Imininingwane Yemicimbi Netinkhulumo

Chofota nome nguwuphi umcimbi kute ubuke imininingwane yawo. Kusuka ekubukweni kwemininingwane yemcimbi:

- **Emalunga nebaholi** bangacala tinkhulumo futsi bashiye imiyalo mayelana nemcimbi.
- **Baholi** bangahlela imininingwane yemcimbi, bashintjse tikhatsi, nobe basule umcimbi.

:::warning
Kususa umcimbi kuwususa ngalokuphelele. Nangabe udzinga kukhansela kwenteka kunye kwemcimbi lophindzaphindzekako, cabanga ngekuwuhlela esikhundleni saloko.
:::

## Kubhalisela Likhalendari

Ungahlanganisa likhalendari leliqembu nelikhalendari lakho lemuntfu siqu:

1. Kusuka ekhasini lelikhalendari leliqembu, chofota **Bhalisela**.
2. Landzela imiyalo kute wengete likhalendari ku-Google Calendar, Apple Calendar, Outlook, nobe nome ngusiphi lolunye luhlelo lwelikhalendari lolusekela kubhalisela.

:::tip
Kubhalisela kugcina uhlelo lwakho lomuntfu siqu luhambelana neimicimbi yeliqembu lakho ngaphandle kwekudzinga kuhlola i-B1 ngasosonkhe lolo. Nome ngusiphi shintjso lesentiwe baholi beliqembu sitawuvela ngekutigcwalisa elikhalendarini lakho lelihlanganisiwe.
:::
