# Kudala Emaqembu

<div class="article-intro">

Kudala liqembu ku-B1 Admin kulula. Utawusungula sigaba, ubike ligama leliqembu lakho, bese uhlela emasethingi alo. Emaqembu akusita kuhlela libandla lakho abe tincenye letinengcondzo njengemaqembu lamancane, emakomiti, netikilasi.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga i-akhawunti ye-B1 Admin lesebentako lenemvumo yekuphatsa emaqembu. Bona [Tindzima Netimvumo](../people/roles-permissions.md) nangabe awucacisisi lizinga lakho lekufinyelela.
- Nquma sakhiwo setigaba temaqembu akho (sibonelo, "Emaqembu Lamancane," "Emasonto," "Emakomiti"). Tigaba tisita kugcina emaqembu lahambelanako ahleliwe.

</div>

## Kungeta Liqembu Lelisha

1. Yiya ku-**inkomba ye-B1 Admin**.
2. Chofota ithebu le-**Emaqembu**.
3. Chofota **Ngeta Liqembu** bese ubhala **Ligama Lesigaba**. Tigaba tikusita kuhlela emaqembu lahambelanako endzaweni yinye (sibonelo, "Emaqembu Lamancane," "Emasonto," nobe "Emakomiti"). Nangabe sigaba sesikhona, ungasikhetsa kuluhlu.
4. Bhala **Ligama Leliqembu**.
5. Chofota **Ngeta**. Liqembu lakho lelisha litawuvela kuluhlu ngaphansi kwesigaba losikhetsile.

## Kuhlela Emasethingi Eliqembu

Nawukadala liqembu lakho, ungagcwalisa imininingwane lengetiwe:

1. Chofota **ligama leliqembu** kuluhlu kute ulivule.
2. Chofota **luphawu lwepensele** kute uhlele emasethingi eliqembu.
3. Hlela letinketfo letilandzelako:
   - **Incazelo** -- Sifingqo lesimfishane sekutsi liqembu limayelana naloku. Loku kuyabonakala emalungeni.
   - **Tikhatsi Temihlangano** -- Nakuvamile liqembu lihlangana (sibonelo, "NgeLesitsatfu nga-7 kusihlwa").
   - **Umtsetfo Wekungenela** -- Khetsa kutsi ngubani longangenela leliqembu:
     - **Kuvulekile** -- Nome ngubani angangenela ngalesikhatsi ngaphandle kwemvumo
     - **Sicelo** -- Bantfu kufanele bafake sicelo sekungenela lesidzinga imvumo (bona [Tesicelo Tekungenela Eliqenjini](./group-join-requests.md))
     - **Kuvaliwe** -- Emalunga kufanele engetwe ngesandla ngebaholi nobe tiphatsimandla
   - **Luphawu** -- Nika liqembu luphawu lolulodwa nobe lolunyenti lolutsi kutsi limayelana naloku (sibonelo, "Endzaweni," "Kuline," "Emalunga Lamasha Ayemukelekile"). Luphawu tinkomba letivulekile lotichazako; hlolisa tonkhe letifanelana. Luphawu lungasetjentiswa kuhlunga emaqembu ku-Groups Browser website element.
   - **Liqembu lelifihlekile** -- Fihla leliqembu neluhlu lwalo emakhasini emphakatsi, ku-group finder, nakulabangesibo emalunga. Sebentisa loku kumaqembu latfobekile njengeteluleko lokuqala kabusha nobe teluleko. Emalunga eliqembu kuphela nebasebenti belibandla labangalibona.
   - **Kulandzelela Kubakhona** -- Vula loku nangabe ufuna kubhalisa [kubakhona](../attendance/tracking-attendance.md) yaleliqembu.
   - **Tikhatsi Tenkhonzo** -- Hlanganisa liqembu netikhatsi letitsite tenkhonzo yebandla nangabe kufanele. Bona [Kuhlelwa Kwekubakhona](../attendance/setup.md) kutfola imininingwane ngetikhatsi tenkhonzo.
4. Chofota **Gcina** kute usebentise tishintjso takho.

:::tip
Kungeta incazelo lecacile nesikhatsi semhlangano kusita emalunga atiwe kutsi kufanele balindzele njani nabangena eliqenjini.
:::

## Kuphindza Liqembu

Ucala sizini lesisha selikilasi nobe lenkhonzo leliphindzaphindzekako? Esikhundleni sekufaka onkhe emasethingi kabusha, phindza liqembu lelikhona:

1. Vula liqembu bese uchofota luphawu lwe-**kuphindza** ku-banner yeliqembu (eceleni kwe-Edit).
2. Cinisekisa kuphindza.

Likhophi liyatsatsa emasethingi weliqembu lekucala -- sigaba, incazelo, sikhatsi/indzawo yemhlangano, umtsetfo wekungenela, luphawu, nekhampasi -- kepha **hhayi** emalunga alo. Liqembu lelisha libitwa ngeligama lelikhona nge- " (Kuphindza)" lokungeteliwe; libite kabusha kusuka emasethingini eliqembu.

## Kufaka Liqembu Emafayilini

Nangabe liqembu alisasebenti kepha ufuna kugcina umlandvo walo esikhundleni sekulisusa:

1. Vula liqembu bese uchofota **luphawu lwepensele** kute uhlele emasethingi alo.
2. Chofota **Faka Emafayilini** bese ucinisekisa.

Emaqembu lafakwe emafayilini ayawa kuluhlu Lwemaqembu lolusemcoka. Kutfola linye futsi, vula i-**Khombisa lafakwe emafayilini** ephezulu kwelikhasi lemaqembu, bese uchofota **Buyisela** eceleni kweliqembu kute ulibuyisele.

:::info
Kufaka liqembu emafayilini akususi emalunga alo, umlandvo wekubakhona, nobe tetamo telikhalendari -- kufihla kuphela liqembu kuluhlu lolujwayelekile kuze ulibuyisele.
:::

## Tinyatselo Letilandzelako

Nawukadala futsi wahlela liqembu lakho, usukulungele:

- **Kungeta emalunga** -- Funa bantfu bese ubengeta eliqenjini. Sebentisa luphawu lwelikhono leliluhlata kute ukhombise baholi beliqembu. Bona [Emalunga Eliqembu](./group-members.md).
- **Kusungula likhalendari** -- Dala tetamo netemihlangano leliphindzaphindzekako yeliqembu. Bona [Likhalendari Leliqembu](./group-calendar.md).
- **Kukhulumisana** -- Tfumela imilayeto kuwo onkhe emalunga eliqembu ngalokucondzile kusuka ekhasini leliqembu.
- **Kukhipha imininingwane** -- Chofota luphawu lwekukhipha kute ukhiphe luhlu lwemalunga eliqembu lakho.

:::info
Onkhe emaqembu ebandla lakho ahlelwe ngetigaba elikhasini lelisemcoka Lemaqembu. Ungahlala uhlela kabusha nobe ubita kabusha tigaba njengoba libandla lakho likhula.
:::
