# Tesicelo Tekungenela Eliqenjini

<div class="article-intro">

Nangabe liqembu lihlelwe ngemtsetfo wekungenela lodzinga imvumo, bantfu bangafaka tesicelo tekungenela. Baholi beliqembu netiphatsimandla babuka letesicelo bese bayativumela nobe bayatenqaba. Loku kunika libandla lakho emandla ekulawula kungenela kwelilunga ngesikhatsi kwenta kube lula kubantfu kukhombisa intfsandvo yekufuna kungenela.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga imvumo yekuphatsa emaqembu, nobe udzinga kuba ngumholi weliqembu lelitsite. Bona [Tindzima Netimvumo](../people/roles-permissions.md) kute utfole imininingwane.
- Liqembu kufanele libe nemtsetfo wekungenela lohlelwe ku-**Sicelo** (imvumo iyadzingeka). Bona [Kudala Emaqembu](./creating-groups.md) kute utfole indlela yekuhlela imitsetfo yekungenela.

</div>

## Kucondzisisa Imitsetfo Yekungenela

Emaqembu angaba nemitsetfo lemitsatfu lehlukene yekungenela:

- **Kuvulekile** -- Nome ngubani angangenela ngalesikhatsi ngaphandle kwemvumo
- **Sicelo** -- Bantfu bafaka sicelo sekungenela lesidzinga imvumo
- **Kuvaliwe** -- Kute longangenela (emalunga kufanele engetwe ngesandla)

Nangabe liqembu lisebentisa umtsetfo we**Sicelo**, tonkhe tetamo tekungenela tihamba ngendlela yemvumo lechachiswe kulelikhasi.

## Kubuka Tesicelo Letisalindzile

### Kubaholi Beliqembu

1. Yiya ku-**Emaqembu** ku-B1 Admin
2. Chofota ligama leliqembu
3. Chofota ithebu **Tesicelo Tekungenela**
4. Utawubona tonkhe tesicelo letisalindzile taleliqembu

### Kutiphatsimandla

Tiphatsimandla letinemvumo yekuphatsa emaqembu tingabuka tesicelo letisalindzile kuwo onkhe emaqembu:

1. Yiya ku-**Emaqembu** ku-B1 Admin
2. Funa luphawu lwesatiso nobe lukhombisa tesicelo letisalindzile
3. Chofota kute ubone tonkhe tesicelo letisalindzile kulibandla lonkhe

## Kubuka Sicelo Sekungenela

Ngasinye sesicelo sekungenela sikhombisa:

- **Ligama nesitfombe salomuntfu** -- Umuntfu lofuna kungenela
- **Umlayeto lokhetsekile** -- Umlayeto lochaza kutsi kungani afuna kungenela (nangabe unikwe)
- **Lusuku lwesicelo** -- Ngesikhatsi sicelo sifakwa

Kubuka sicelo:

1. Fundza umlayeto walomuntfu nangabe akhona
2. Chofota ligama lalomuntfu kute ubuke iphrofayela yakhe nangabe kudzingeka
3. Nquma kutsi utawuvumela nobe wenqaba

## Kuvumela Sicelo

1. Chofota **Vumela** kulesicelo sekungenela
2. Umuntfu ufakwa ngekushesha eliqenjini njengelilunga
3. Lofake sicelo utfola satiso setsi sicelo sakhe sivunyelwe
4. Sicelo sibekwa njengesivunyelwe kuhlelo

:::tip
Nangabe uvumela sicelo, umuntfu uba lilunga lelijwayelekile leliqembu. Ungamkhweletsa kamuva abe ngumholi weliqembu nangabe kudzingeka usuka ekhasini le-[Emalunga Eliqembu](./group-members.md).
:::

## Kwenqaba Sicelo

1. Chofota **Yenqaba** kulesicelo sekungenela
2. Ungeta sizatfu sekwenqaba nangabe ufuna (kuya kuma-500 wetimpawu)
3. Chofota **Chaza**
4. Lofake sicelo utfola satiso lesinesizatfu sakho sekwenqaba (nangabe sinikiwe)
5. Sicelo sibekwa njengesenqatjwe

:::info
Kunika sizatfu sekwenqaba kusita umuntfu kucondzisisa kutsi kungani sicelo sakhe singavunyelwanga futsi kungamkhutsata kutsi azame futsi kamuva nobe afune emanye emaqembu.
:::

## Tati so

Luhlelo lwesicelo sekungenela lutfumela satiso ngekutigcinela:

- **Nasifakwa sicelo** -- Bonkhe baholi beliqembu batfola satiso
- **Nasivunyelwa sicelo** -- Lofake sicelo utfola siciniseko
- **Nasenqatjwa sicelo** -- Lofake sicelo utfola satiso lesinaso sonkhe sizatfu sekwenqaba

Tati so tivela endzaweni yesati so yemsebentisi ku-B1.church nasekusebentiseni lidivayisi lelihambako.

## Kuphatsa Tesicelo Kusuka Kulunga

Bantfu bangaphatsa tesicelo tabo tekungenela kusuka ku-B1.church:

- Buka simo sesicelo sabo lesisalindzile ekhasini leliqembu
- Khansela sicelo lesisalindzile nangabe bashintsha umcabango wabo
- Babone kutsi sicelo sabo sivunyelwe nobe senqatjwe

## Timitsi Letinhle

- **Phendvula ngekushesha** -- Zama kubuka tesicelo emahoreni la-24-48 kuze bantfu bangalindzi sikhatsi lesidze
- **Caphelisisa etitsatfweni tekwenqaba** -- Sita bantfu bacondzisise tinyatselo letilandzelako nobe letinye tinketfo
- **Hlola tiphrofayela** -- Buka iphrofayela yalomuntfu kubona nangabe uyafanela liqembu
- **Cocisana ngalokulindzelwe** -- Cinisekisa kutsi inchazelo yeliqembu lakho ichaza kahle kutsi liqembu limele bani
