# Tinhlu

<div class="article-intro">

Tinhlu tikuvumela kuhlela tintshumayelo takho tibe tichungechunge nome imitsamo. Nangabe bavakashi baphequla umtsamo wakho wetintshumayelo eliwebusayithini lakho le-B1.church, tinhlu tibasita kutfola futsi balandzele chungechunge lolucedzelekile lwetintshumayelo ngendlela lelungile.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- Udzinga imvumo **contentApi.streamingServices.edit**. Buka [Tikhundla Netimvumo](../settings/roles-permissions.md) uma ungenawo umfanelo.
- Yiba nelibito, inchazelo, kanye nesitfombe lesincane lokulungele luhlu lwakho

</div>

## Kubuka Tinhlu

1. Ku-B1 Admin, chofota **Tintshumayelo** kulacala lelisesandleni sancele.
2. Chofota ithebhu ye-**Tinhlu** etulu kwelikhasi.
3. Utawubona luhlu lwato tonkhe tinhlu takho leseletikhona nemabito nenchazelo yato.

## Kwakha Luhlu

1. Chofota inkhinobho **Yakha Luhlu Lwekucala** (uma ungakabi nato tinhlu) nome uchofote **Engeta Luhlu**.
2. Faka **ligama** loluhlu -- sibonelo, sihloko sechungechunge lwetintshumayelo lokufana ne "Sisekelo Selukholo" nome "Lihlobo eTihlabelelweni."
3. Engeta **inchazelo** kute unike bavakashi umbono lomfishane wechungechunge.
4. Lungisa **lusuku lwekushicelela** kute ulawule kutsi luhlu lucala nini kuvela.
5. Layisha **sitfombe lesincane** kute simele chungechunge ngendlela lebonakalako.
6. Chofota **Gcina** kute wakhe luhlu lwakho.

:::tip
Yakha tinhlu takho kucala ngembi kwekwengeta tintshumayelo. Ngaleyo ndlela ungakwati kunikeza intshumayelo nga yinye luhlu lolufanele njengoba uhamba, esikhundleni sekuphindza uhlele emuva.
:::

## Kwengeta Tintshumayelo Kuluhlu

Nawu-[engeta nome uhlela intshumayelo](managing-sermons), uyinikeza luhlu ngekusebentisa iluhlu lolwehlako le-**Luhlu**. Tonkhe tintshumayelo letinikezwe luhlu lolufananako tihlanganiswa ndzawonye futsi tikhonjiswa ngendlela lelungile eliwebusayithini lakho.

:::info
Ungakwati futsi kunikeza tintshumayelo tinhlu ngesikhatsi se-[Kungeniswa Ngebuningi](bulk-import). Nawungenisa kusuka ku-YouTube nome Vimeo, khetsa luhlu eluhlwini lolwehlako lwe-**Ngenisa Kuluhlu** ngembi kwekutsi uchofote **Ngenisa**.
:::

## Indlela Tinhlu Letivela Ngayo Eliwebusayithini Lakho

Tinhlu tikhonjiswa eliwebusayithini lakho le-B1.church njengemitsamo lengaphequlwa. Bavakashi bangakwati:

- Kubona tonkhe tichungechunge letitfolakalako tetintshumayelo ngekushesha
- Kuchofota kungena kuluhlu kute babone tonkhe tintshumayelo kulelo chungechunge
- Kubuka nome kulalela tintshumayelo ngendlela lelungile

## Tinyatselo Letilandzelako

- [Kuphatsa Tintshumayelo](managing-sermons) -- Engeta tintshumayelo nga ngayinye etinhlwini takho
- [Kungeniswa Ngebuningi](bulk-import) -- Ngenisa tintshumayelo letinyenti bese uzinikeza tinhlu
