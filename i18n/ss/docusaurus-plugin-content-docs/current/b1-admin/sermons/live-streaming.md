# Kusakaza Bukhoma

<div class="article-intro">

Likhasi le-Tikhatsi Tekusakaza Bukhoma likuvumela kulungisa shedyuli yekusakaza yelibandla lakho, uphatse tikhatsi temakhonzo, futsi ulungise indlela lababukeli labatawuyibona ngayo. Hlela emakhonzo lenyakelako lasonto naso wonkhe, nome imicimbi yekwenteka kanye kanye, lungisa tilungiselelo tekucoca netevidiyo, futsi ulawule ngesikhatsi lapho umsakazo wakho utawucala khona.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- Udzinga imvumo **contentApi.streamingServices.edit**. Buka [Tikhundla Netimvumo](../settings/roles-permissions.md) uma ungenawo umfanelo.
- Yiba ne-ID Yeshanela yakho YeYouTube lelungele, uma ucata kusebentisa kusakaza bukhoma lokuyingeniso yesekwe.
- Engeta [intshumayelo](managing-sermons) lulodvwa okungenani nome i-URL yekusakaza bukhoma lehlala njalo, kute usebentise njengemtfombo wemsakazo wakho

</div>

Likhasi linetheb hu letimbili letiyinhloko: **Emakhonzo** yekuphatsa shedyuli yakho yekusakaza bukhoma kanye ne-**Tilungiselelo** yekulungisa likhasi lakho lekusakaza.

## Kuphatsa Emakhonzo

### Kwengeta Inkhonzo

1. Ku-B1 Admin, chofota **Tintshumayelo** kulacala lelisesandleni sancele, bese uchofota ithebhu ye-**Tikhatsi Tekusakaza Bukhoma**.
2. Chofota inkhinobho **Engeta Inkhonzo** kute wakhe inkhonzo lensha lelungisiwe.
3. Faka **Ligama Lenkhonzo** (sibonelo, "Ekuseni Ngelisonto").
4. Lungisa **Sikhatsi Senkhonzo** -- khetsa lilanga kanye nesikhatsi lapho inkhonzo yakho icala khona.
5. Lungisa **Iphindzaphindza Ngesonto** kutsi ibe **Yebo** temakhonzo lenyakelako lasonto naso wonkhe, nome **Cha** temcimbi wekwenteka kanye kanye.

### Kulungisa Tilungiselelo Tekucoca Netevidiyo

6. Ngaphansi kwa-**Tilungiselelo Tekucoca**, lungisa emaminithi lamangaki ngembi nangemuva kwenkhonzo lapho kucoca kufanele kuvulwe khona. Loku kuvumela bavakashi kucala kucoca ngembi kwekutsi inkhonzo icale futsi baciphelele ngemuva kwaloko.
7. Ngaphansi kwa-**Tilungiselelo Tevidiyo**, lungisa kutsi umsakazo wevidiyo kufanele ucale ngesikhatsi lesingakanani ngembi, kute kube khona kucubunga noma kucuketfwe lokungembi kwenkhonzo.
8. Khetsa kutsi ngukuphi intshumayelo lekufanele kudlalwe kusuka kuluhlu:
   - **Intshumayelo Yamuva** -- Idlala ngekutentekwako ividiyo yakho lengeniswe muva nje.
   - **Inkhonzo Yamanje Lesakazwa Bukhoma** -- Idlala umsakazo wakho wamanje losakazwa bukhoma kusuka ku-YouTube kusebentisa i-ID Yeshanela yakho.
   - Ungakhetsa nome ngabe yiyiphi intshumayelo lotsite loseyigcinile.
9. Chofota **Gcina** kute uhlele inkhonzo yakho.

:::info
Inkhonzo yakho itakutivuselela ngekutentekwako liviki nga lelinye uma ilungisiwe kutsi iphindzaphindze. Ungengeta emakhonzo lamanyenti njengoba udzinga. Bavakashi batakubona sikhatsi lesilandzelako lesilungisiwe senkhonzo lapho bavakashela khasi lakho lekusakaza.
:::

## Tilungiselelo Telikhasi Lekusakaza

Chofota ithebhu ye-**Tilungiselelo** kute ulungise tithebhu netixhumanisi letivela eceleni komsakazo wakho lobukhoma.

### Kwengeta Tithebhu

1. Chofota inkhinobho **Engeta** kute wengete ithebhu lensha ekhasini lakho lekusakaza bukhoma.
2. Khetsa ithebhu **Kucoca** lelisekwe kadze, nome wengete ithebhu lelitsite line-URL yangaphandle.
3. Ethebhini yeKucoca, nika lona ligama kuphela ebhokisini le-**Umbhalo Wethebhu**, bese kulungiselelo kuphelile.
4. Ethebhini lelixhunyeniswe, faka ligama lethebhu, khetsa liayikhoni ngekuchofota inkhinobho leliayikhoni, bese ufaka i-URL.
5. Tithebhu takho letilungisiwe titakuvela ekhasini lekusakaza bukhoma kute bababukeli babe nemtfombo lengetiwe kanye netici letisebentako.

### Kubuka Umsakazo Wakho Ngembi Kwesikhatsi

Chofota inkhinobho **Buka Umsakazo Wakho** kute ubone kucacile kutsi likhasi lakho lekusakaza bukhoma litawubukeka njani kubavakashi, kufaka ekhatsi ilogo yakho, tikhatsi temakhonzo, kanye netithebhu letilungisiwe.

## Kulungisa Umsakazo Wakho Wabukhoma we-YouTube

Kute uxhumanise ishanela yakho ye-YouTube kusakaza bukhoma ngekutentekwako:

1. Hamba ku-**Tintshumayelo** bese uchofota **Engeta Intshumayelo**, bese ukhetsa **Engeta i-URL Yekusakaza Bukhoma Lohlala Njalo**.
2. Umniki wevidiyo uba **Umsakazo Wamanje Wa-YouTube Lobukhoma** ngekwendlela lesetjentiswako. Faka **I-ID Yeshanela Yakho YeYouTube**.
3. Engeta sihloko nenchazelo, bese uchofota **Gcina**.
4. Ku-**Tikhatsi Tekusakaza Bukhoma**, yakha inkhonzo bese ukhetsa i-URL yakho lehlala njalo eluhlwini lwetintshumayelo.

:::tip
Kute utfole i-ID Yeshanela yakho YeYouTube, hamba etilungiselelweni letiphakeme teshanela yakho ye-YouTube bese ukhawopisha inani le-ID Yeshanela.
:::

## Kulungisa Imibala Kanye Nelogo

Likhasi lakho lekusakaza bukhoma lisebentisa tilungiselelo te-[Kubukeka](../website/appearance) teliwebusayithi lakho:

- **Umbala wekucacisa lokhanyako** unembhalo lomnyama usetjentiswa etulu.
- **Umbala wekucacisa lomnyama** unembhalo lokhanyako usetjentiswa lucaleni.
- **Ilogo Yakho Yangemuva Lekhanyako** ivela ekhasini lekusakaza. Sebentisa sitfombe lesingenambala wangemuva (transparent) kanye nesilinganiso se-4:1.

Kute ushintje loku, hamba ku-**Liwebusayithi** bese ku-**Kubukeka**, bese ubuyekete **Luhlu Lwemibala** kanye netilungiselelo te-**Logo**.

## Kwengeta Baphatsi Bekusakaza

Kute unike emalunga eticukutfwana imvumo yekungena ku-inkulumo yebaphatsi bodvwa eceleni kwekucoca lokuvulekile:

1. Hamba ku-**Tilungiselelo** kulacala lelisesandleni sancele bese uchofota **Tikhundla**.
2. Chofota inkhinobho lene-plus bese ukhetsa **Engeta Sikhundla Lesitsite**.
3. Nika sikhundla ligama letsi "Umphatsi Wekusakaza" bese uchofota **Gcina**.
4. Chofota sikhundla lesisha, bese uchofota **Engeta** esigabeni Semalunga kute wengete bantfu.
5. Skrolela phansi kuya ku-**Hlela Timvumo**, uvule sigaba se-**Kucuketfwe**, bese uhluza **Inkulumo Yebaphatsi**.

Nangabe baphatsi bangena ekhasini lekusakaza bukhoma, ithebhu lelifihlekile letsi **Inkulumo Yebaphatsi** livela eceleni kwenkulumo lekuvulekile, kute ticukutfwana ticoce todvwa ngesikhatsi semsakazo.

:::info
Kute utfole imininingwane lengephi mayelana nekwakha tikhundla nekuphatsa timvumo, buka [Tikhundla Netimvumo](../settings/roles-permissions.md).
:::

## Kulungisa Tinkinga

Uma umsakazo wakho wangekutentekwako wa-YouTube ungakhomboliseki kahle nawusebentisa lokutfunywa "Umsakazo Wamanje Wa-YouTube Lobukhoma" ne-ID Yeshanela yakho, sebentisa loku lokulandzelako:

**Tiphawu:**
- Umfaki wemsakazo bukhoma ukhombisa "Ividiyo Ayitfolakali"
- Likhasi liyalayisha kodvwa ayikho ividiyo levelako
- Emafaki wangaphandle wa-YouTube ayasebenta, kodvwa umsakazo wamanje wetentekwako weshanela awusebenti

**Sisombululo:**
Hlola eshaneleni yakho ye-YouTube kutsi kukhona yini imisakazo lemidzala nome lelungisiwe yekwenteka kanye, bese uyicoshe:

1. Hamba ku-YouTube Studio yakho.
2. Hamba ku-**Kucuketfwe** bese uya ku-**Bukhoma**.
3. Funa nome ngukuphi umsakazo lomdzala loshedulisiwe nome lolungiselelwe kwenteka.
4. Coshe letinsimo temsakazo lomdzala nome loshedulisiwe.
5. Hlola likhasi lakho lekusakaza futsi.

:::warning
Umfaki wemsakazo wangekutentekwako weshanela ye-YouTube ungavinjelwa uma kunetinsimo letinyenti letishedulisiwe nome letendlulile eshaneleni yakho. Kucosha lena kuvumela YouTube kutsi ikhombise kahle futsi inikeze umsakazo wakho wamanje.
:::

**Tidzingo letengetiwe:**
- Umsakazo wakho wobukhoma kufanele ulungiswe kutsi ube **Wemphakatsi** (hhayi Lofihlekile nome Lomuye).
- Kufakwa (embedding) kufanele kuvunyelwe etilungiselelweni takho temsakazo we-YouTube.
- Cinisekisa kutsi usebentisa umniki **Umsakazo Wamanje Wa-YouTube Lobukhoma** (one-ID Yeshanela), hhayi umniki **YouTube** (one-ID Yevidiyo).
