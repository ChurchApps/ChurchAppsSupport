# Kungeniswa Ngebuningi

<div class="article-intro">

Likhasi le-Kungeniswa Ngebuningi likuvumela kugcwalisa ngekushesha umtsamo wakho wetintshumayelo ngekungenisa emavidiyo asuka ku-YouTube nome i-Vimeo. Le yindlela lesheshako kakhulu yekungenisa kucuketfwe kwakho kwetintshumayelo lokuselokhona, kunekwengeta ividiyo ngayinye ngesikhatsi.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- Udzinga imvumo **contentApi.streamingServices.edit**. Buka [Tikhundla Netimvumo](../settings/roles-permissions.md) uma ungenawo umfanelo.
- Yakha ku-[luhlu](playlists) lulodvwa okungenani kute ungenise tintshumayelo takho kulo
- Yiba nemniningwane we-ID Yeshanela yakho YeYouTube nome imniningwane ye-akhawundi ye-Vimeo lelungele

</div>

## Kukhetsa Umtfombo Wakho

1. Ku-B1 Admin, chofota **Tintshumayelo** kulacala lelisesandleni sancele, bese uchofota ithebhu ye-**Kungeniswa Ngebuningi**.
2. Utawubona emakhadi lamabili langachofotwa:
   - **YouTube** (libomvu) -- Ngenisa emavidiyo asuka kushanela ye-YouTube
   - **Vimeo** (luhlata) -- Ngenisa emavidiyo asuka ku-akhawundi ye-Vimeo
3. Chofota likhadi lelisicalo lalapho kugcinwe khona tintshumayelo takho.

:::tip
Ungabuyela emumva futsi ushintje imtfombo nomangabe unaso sikhatsi, uma unemavidiyo kuletinsimu tomabili.
:::

## Ngembi Kwekungenisa

Ngembi kwekungenisa emavidiyo, udzinga luhlu lulodvwa okungenani kute uwahlele kulo. Uma ungakabi nato tinhlu:

1. Chofota ithebhu ye-**Tinhlu**.
2. Chofota **Yakha Luhlu Lwekucala** bese ugcwalisa ligama, inchazelo, lusuku lwekushicelela, kanye nesitfombe lesincane.
3. Chofota **Gcina**, bese ubuyela ethebhini ye-**Kungeniswa Ngebuningi**.

Buka [Tinhlu](playlists) kute utfole imiyalo lecacile mayelana nekwakha nekuphatsa tinhlu.

## Kungenisa Emavidiyo

1. Ngemuva kwekukhetsa YouTube nome Vimeo, faka i-**ID Yeshanela Yakho YeYouTube** nome **imniningwane ye-akhawundi ye-Vimeo** endzaweni lenikeziwe.
2. Chofota inkhinobho letsi **Tfola** kute utfole onkhe emavidiyo latfolakalako kushanela nome ku-akhawundi yakho.
3. Ngemuva kwekutfola, utawubona luhlu lwawo onkhe emavidiyo akho anemabhokisi ekuhluza.
4. Hluza emabhokisi lasedvutane nemavidiyo lofisa kuwangenisa. Ungakhetsa onkhe nome ukhetse latsite ngekutsite.
5. Ngekukhetsa kwakho, vula **Ngenisa Emavidiyo Lamasha Ngokutentekwako** kute wengete ngekutentekwako emavidiyo latakulayishwa esikhatsini lesitako.
6. Chofota imenyu leyehlako letsi **Ngenisa Kuluhlu** kute ukhetse kutsi ngukuphi luhlu lokufanele emavidiyo lawa engetwe kulo.
7. Chofota inkhinobho **Ngenisa** kute ucedzele kungeniswa ngebuningi.

Emavidiyo akho atakungeniswa nayo yonkhe imniningwane yawo, lokufaka ekhatsi tihloko, tinchazelo, tilusuku, kanye netitfombe letincane.

:::info
Kungeniswa ngebuningi kuncono kakhulu nawucala, uma ususuka kwenye i-platform. Nawengeta tintshumayelo nga ngayinye kusukela manje, sebentisa likhasi le-[Kuphatsa Tintshumayelo](managing-sermons) esikhundleni saloko.
:::
