# Tinhlobo Te-Navigation

<div class="article-intro">

Lungisa imibala ye-navigation bar yeliwebhusayithi lelibandla lakho kute ihambelane ne-branding yakho. Ungalungisa imibala yato tobabili tinsika letinesizinda kanye ne-overlay lecondzekako, lokukunika lawulo lolugcwele lwendlela i-navigation yakho lebukeka ngayo emakhasini lahlukene.

</div>

<div class="prereqs">
<h4>Before You Begin</h4>

- Udzinga imvumo yekuphatsa liwebhusayithi lelibandla lakho. Buka [Roles & Permissions](../people/roles-permissions.md) kutfola imininingwane.
- Yenta kutsi imibala yakho yebrendi ilungile, kufaka ekhatsi emakhodi embala e-hex (sib. #03A9F4).
- Condza umehluko emkhatsini wetinhlobo te-navigation lecinile ne-lecondzekako kulisayithi lakho.

</div>

## Kucondza Tinhlobo Te-Navigation

I-navigation yeliwebhusayithi lakho ingavela ngetinhlobo letimbili letehlukene kuya ngekhasi:

- **Solid navigation** -- I-navigation bar lenesizinda sembala, ivamise kusetjentiswa emakhasini ekucuketfwa
- **Transparent navigation** -- I-navigation legcwala phezu kwekucuketfwa kwelikhasi, ivamise kusetjentiswa emakhasini lanetitfombe letinkhulu (hero images) nome tizinda letigcwele sikrini

Ungalungisa imibala yalezi tinhlobo tobabili ngekwehlukana.

## Kufinyelela Kutinhlobo Te-Navigation

1. Yendlulela ku- **Website** ku-B1 Admin
2. Chofota **Appearance** kulisayidbha
3. Yehlela uye esigabeni se- **Navigation Styles**
4. Chofota **Edit Navigation Styles**

## Kulungisa I-Solid Navigation

I-Solid navigation ivela nesizinda sembala ngemuva kwe-navigation bar. Ungalungisa:

### I-Background Color

1. Vula i-switch ye- **Override** ye- **Background Color**
2. Chofota sikhetsi sembala
3. Khetsa umbala lofunako wesizinda
4. Umbala lojwayelekile ngumhlophe (#FFFFFF)

### I-Link Color

1. Vula i-switch ye- **Override** ye- **Link Color**
2. Khetsa umbala wembhalo wetilinki te-navigation
3. Loku kutsintsa tilinki esimeni sato sejwayelekile
4. Umbala lojwayelekile ngumphunga lomnyama (#555555)

### I-Link Hover Color

1. Vula i-switch ye- **Override** ye- **Link Hover Color**
2. Khetsa umbala lotilinki titshintjela kuwo nasekutsi umsebentisi wafaka i-cursor phezu kwato
3. Loku kunika imphendvulo lebonakalako yetilinki letingachofotwa
4. Umbala lojwayelekile ngumbala lophuzi lokhanyako (#03A9F4)

### I-Active Color

1. Vula i-switch ye- **Override** ye- **Active Color**
2. Khetsa umbala welinki lelikhasi lelisebentako njengamanje
3. Loku kusita basebentisi kutsi bati kutsi bakuliphi likhasi
4. Umbala lojwayelekile ngumbala lophuzi lokhanyako (#03A9F4)

## Kulungisa I-Transparent Navigation

I-Transparent navigation igcwala phezu kwekucuketfwa kwelikhasi lakho ngaphandle kwesizinda. Ungalungisa:

### I-Link Color

1. Vula i-switch ye- **Override** ye- **Link Color**
2. Khetsa umbala loyphambana kahle nesizinda sekhasi lakho
3. Kanyenti umhlophe nome imibala lekhanyako isebenta kahle etizindeni letimnyama
4. Umbala lojwayelekile ngumphunga lomnyama (#555555)

### I-Link Hover Color

1. Vula i-switch ye- **Override** ye- **Link Hover Color**
2. Khetsa umbala wesimo se-hover
3. Cinisekisa kutsi uyabonakala kumelene nesizinda sekhasi lakho
4. Umbala lojwayelekile ngumbala lophuzi lokhanyako (#03A9F4)

### I-Active Color

1. Vula i-switch ye- **Override** ye- **Active Color**
2. Khetsa umbala wekukhomba likhasi lelisebentako
3. Kufanele uvele kahle kepha uhambisane nesakhiwo sakho
4. Umbala lojwayelekile ngumbala lophuzi lokhanyako (#03A9F4)

:::info
I-Transparent navigation ayinawo lungiselelo lwe-background color njengobe igcwala phezu kwekucuketfwa kwelikhasi ngco.
:::

## Kugcina Tinshintjo Takho

1. Ngemuva kwekulungisa imibala yakho, chofota **Save Navigation Styles**
2. Tinshintjo takho tisebenta ngalesikhatsi lesincane kuliwebhusayithi lakho leliphilako
3. Vakashela liwebhusayithi lakho kute ubone i-navigation kuto tobabili tinhlobo

## Kubuyisela Kumanani Ejwayelekile

Uma ufuna kubuyela kumibala yejwayelekile:

1. Vala ema-switch e- **Override** ya noma yimuphi imibala lekhetsile
2. Chofota **Save Navigation Styles**
3. I-navigation ibuyela kuluhlu lwemibala lolujwayelekile

Nome uchofote **Cancel** kute ulahle tonkhe tinshintjo ngaphandle kwekutigcina.

## Tindlela Letinhle Kakhulu

### Kuphambana Kwemibala (Color Contrast)

- **Kufundzeka** -- Cinisekisa kutsi imibala yetilinki inekuphambana lokwenele nesizinda
- **Kuhambisana ne-WCAG** -- Zama kufinyelela lizinga lekuphambana lelingashoni ngephansi kwa-4.5:1 kute kube nekufinyeleleka (accessibility)
- **Hlola tinhlobo tobabili** -- Buka lisayithi lakho ngato tobabili i-solid ne-transparent navigation

### Kuhambisana Nebrendi

- **Sebentisa imibala yebrendi wakho** -- Yihambisane nelogo yakho kanye nesihloko selisayithi
- **Nciphisa luhlu lwakho lwemibala** -- Bambelela kumibala lemibili kuya kwemitsatfu kutfola kubukeka lokuhambisanako
- **Cabanga ngetitfombe takho** -- Uma usebentisa i-transparent navigation, yihlole kumelene netizinda letijwayelekile temakhasi

### Simo Se-Hover Nese-Active

- **Imphendvulo lecacile** -- Yenta simo se-hover sibonakale sehlukile kutilinki tejwayelekile
- **Cacisa emakhasi lasebentako** -- Sebentisa umbala lohlukile kute basebentisi bati kutsi bakuphi
- **Kutjintjana lokusheshako** -- Luhlelo luyahlanganisa ngekutentekela kutjintja kwemibala

## Kucatulula Tinkinga

### Imibala Ayibukeki Kahle

- **Sula i-cache yakho** -- I-cache ye-browser ingakhombisa imibala lendzala
- **Hlola emakhodi e-hex** -- Cinisekisa kutsi ufake emakhodi embala e-hex lasemtsetfweni
- **Hlola etizindeni letehlukene** -- Imibala ingabukeka ihlukile kuya ngekhasi

### I-Navigation Ayiboniki

- **Simo se-Transparent** -- Uma usebentisa transparent navigation phezu kwetitfombe letikhanyako, mbhalo lomnyama ungaba nzima kubonwa
- **Sisombululo** -- Buyeketa imibala yetilinki takho nome usebentise tizinda temakhasi letimnyama
- **Enye indlela** -- Ngeta litfunti lelincane nome sizinda se-overlay endzaweni ye-navigation

## Imininingwane Yebuchwepheshe

Tinhlobo te-navigation tigcinwe njenge-JSON futsi tisetjentiswa ngekusebentisa ema-CSS variable:

- Tinshintjo tisebenta ngalesikhatsi lesincane ngaphandle kwekwakha kabusha lisayithi
- Imibala iyandlulela kuto tonkhe ticenye te-navigation
- Ema-override akhetsekile; imibala lengakabekwa isebentisa yalokujwayelekile yesihloko
