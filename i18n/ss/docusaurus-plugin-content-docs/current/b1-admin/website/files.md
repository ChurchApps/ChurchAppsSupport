# Emafayela

<div class="article-intro">

Likhasi le-Files sisikhungo sesitolo salo lonkhe umdidiya nemibhalo lesetjentiswa kuliwebhusayithi lelibandla lakho. Kusuka lapha ungalayisha, uhlele, ubuke, futsi uphatse sonkhe sitfombe, umbhalo, nefayela lemidiya lelidzingwa lisayithi lakho.

</div>

<div class="prereqs">
<h4>Before You Begin</h4>

- Cedza [Initial Setup](initial-setup) yeliwebhusayithi lakho
- Nciphisa titfombe ngaphambi kwekutsi ulayishe kute emakhasi alayishe ngekushesha
- Lungisa emafayela akho ngetifometi letijwayelekile tewebhu (PNG, JPG, PDF, njll.)

</div>

## Kulayisha Emafayela

1. Yendlulela esigabeni se- **Website** ku-B1 Admin bese uchofota i-tab le- **Files**.
2. Dvonsa bese ulahla emafayela ngco endzaweni yekulayisha, nome uchofote inkinobho yekulayisha kute uhambe ku-khompyutha yakho.
3. Emafayela akho atawulayisha futsi avele emtapweni wemafayela ngalesikhatsi lesincane.

:::tip
Kute utfole imiphumela lemihle kakhulu, nciphisa titfombe ngaphambi kwekulayisha. Emafayela lamancane kusayizi asita liwebhusayithi lakho kutsi lilayishe ngekushesha kubavakashi. Tinsita njenge TinyPNG nome Squoosh tingankihlisa titfombe ngaphandle kwekulahlekelwa lokubonakalako kwekubukeka lokuhle.
:::

## Kuhlela Ngemafolda

1. Dala emafolda kute uhlanganise emafayela lahlobene ndzawonye -- sibonelo, "Sermon Graphics," "Event Banners," nome "Staff Photos."
2. Hambisa emafayela emkhatsini wemafolda kute ugcine umtapo wakho uhlelekile njengobe ukhula.
3. Sebentisa emagama emafolda lacacile futsi lachazako kute licembu lakho litfole emafayela ngekushesha.

## Kubuka Nekusesha

- Yehlela phansi kumtapo wemafayela kute ubone titfombe letincane (thumbnails) tawo onkhe emafayela lowalayishile.
- Sebentisa umsebenti we- **search** kute utfole emafayela ngeligama nase umtapo wakho sewukhulile.
- Chofota noma ngeliphi ifayela kute ubuke imininingwane yalo, kufaka ekhatsi ligama lefayela, sayizi, kanye ne-URL.

## Kusebentisa Emafayela Kuliwebhusayithi Lakho

1. Chofota ifayela kute ubuke imininingwane yalo.
2. **Kophela i-URL yefayela** kute uyisebentise emakhasini eliwebhusayithi lakho. Ungakwati kunamatsela le-URL kuticenye tesitfombe, kutilinki tekulanda, nome nome ngukuphi lapho kwemukelwa khona i-URL yefayela.
3. Emafayela lowalayisha lapha akhona kute asetjentiswe kuwo onkhe emakhasi eliwebhusayithi lakho nemibhalo yekucuketfwa.

:::info
Nawakha emakhasi ku- [Managing Pages](managing-pages), ungakhomba noma ngeliphi ifayela lelilayishwe lapha nge-URL yalo. Loku kugcina umdidiya wakho wonkhe uhlelekile endzaweni yinye lesisekelo.
:::

## Kususa Emafayela

1. Khetsa ifayela lofuna kususa.
2. Chofota inkinobho le- **Delete** kute uyisuse ngalokuphelele emtapweni wakho.
3. Cinisekisa kutsi ifayela alisasetjentiswa nakuliphi likhasi ngaphambi kwekulisusa, njengobe loku kutawuphula tilinki nome titfombe letikhomba kulo.

:::warning
Emafayela lasusiwe akakwati kubuyiselwa. Cinisekisa kabili kutsi ifayela alisasetjentiswa nakuliphi likhasi ngaphambi kwekulisusa.
:::
