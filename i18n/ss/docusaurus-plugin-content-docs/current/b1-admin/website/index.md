# Liwebhusayithi

<div class="article-intro">

Sigaba se-Website ku-B1 Admin sikunika sakhi seliwebhusayithi lesigcwele setintfo tonkhe selibandla lakho. Ungadala emakhasi, uhlele kubukeka nesimo selisayithi lakho, uphatse emafayela emidiya, futsi ushicilele emakhalendha etehlakalo -- konkhe ngaphandle kwekubhala noma nguyiphi ikhodi.

</div>

## Loko Longakwenta

Sakhi se-Website sihlelwe tindzawo letinhlanu letikhulu:

### Emakhasi (Pages)

Dala futsi uphatse wonkhe emakhasi eliwebhusayithi lelibandla lakho. Litafula le- **Pages** likhombisa likhasi ngalinye kanye nesimo salo -- nome ku- **Generated** (lidalwe ngekutentekela luhlelo) nome **Custom** (lidalwe nguwe). Ungangeta emakhasi lamasha lakhetsile, uhlele lasekhona, nome uguculele emakhasi ladalwe ngekutentekela abe ngemakhasi lakhetsile kute ube nelawulo lolugcwele lwekucuketfwa kwawo.

Isayidbha yangesencele ikhombisa tilinki takho te-navigation ngendlela ye-tree structure. Ungahlela kabusha tilinki ngekudvonsa nekulahla kute usimise indlela nekulandzelana loku funako kutsi tivakashi titibone ngayo.

### Tincenye (Blocks)

Yakha ticenye tekucuketfwa letingasetjentiswa kaningi lotingabeka emakhasini lamanyenti. Tincenye tikonga sikhatsi sakho nawuneNkucetfwa njenge call-to-action, ibhena yesimemetelo, nome sigaba sekutsintsana lesivela endzaweni lengaphezu kwayinye. Buyeketa incenye kanye bese lutjintjo lusebenta yonkhe indzawo lapho isetjentiswe khona.

### Kubukeka (Appearance)

Lungisa sihloko, imibala, imibhalo, ilogo, kanye nekukhweleta kwendzawo yeliwebhusayithi lakho. Ungasebentisa tihloko letakhiwe ngaphambilini kute ucale ngekushesha nome uhlele imininingwane yonkhe nge-color palette editor, tilungiselelo temibhalo, kanye ne-CSS lekhetsile. Buka umculu we- [Appearance](appearance) kutfola imininingwane leyphelele.

### Emafayela (Files)

Layisha futsi uhlele tonkhe tintfo temidiya yakho -- titfombe, imibhalo, naletinye tifayela -- endzaweni yinye lesisekelo. Sebentisa emafolda kute ugcine tintfo tihlelekile bese ukhopha tilinki temafayela kute utisebentise nakuphi kulisayithi lakho. Buka [Files](files) kutfola imininingwane lengetiwe.

### Emakhalendha (Calendars)

Shicilela emakhalendha etehlakalo kuliwebhusayithi lakho kute tivakashi tibone loko lokwenteka kulibandla lakho. Dala emakhalendha lahlelwe lakhipha tehlakalo etigcele letinyenti kubukeko banye.

:::tip
Cala nge- [Initial Setup](initial-setup) uma ulungisela liwebhusayithi lakho kwekucala. Ikukhokhela ekulungiseni domain, kudala likhasi lakho lekucala, kanye nekulungisa kubukeka kwelisayithi lakho.
:::
