# Kuphatsa Emakhasi

<div class="article-intro">

Embukisi we-Website Pages sisikhungo sakho lesisekelo sekudala, kuhlela, nekuhlela wonkhe emakhasi eliwebhusayithi lelibandla lakho. Ungaphatsa tobabili kucuketfwa kwekhasi lakho kanye ne-navigation yelisayithi lakho kusuka kulesikrini sinye.

</div>

<div class="prereqs">
<h4>Before You Begin</h4>

- Cedza [Initial Setup](initial-setup) kute ulungise i-domain yakho kanye netilungiselelo letisisekelo telisayithi
- Yenta kutsi kucuketfwa kwakho netitfombe kulungile. Sebentisa umphatsi we- [Files](files) kucala kute ulayishe tintfo temidiya.

</div>

## Kucondza Tinhlobo Temakhasi

Litafula le- **Pages** likhomba likhasi ngalinye kulisayithi lakho kanye nesimo salo:

- **Generated** -- Emakhasi ladalwe ngekutentekela luhlelo kususela kumininingwane yelibandla lakho (sibonelo, likhasi le-Groups nome likhasi le-Sermons). Lamakhasi atibuyeketa nakushintja imininingwane yakho.
- **Custom** -- Emakhasi lowatakhile ngco ngekucuketfwa kwakho kanye nesakhiwo sakho.

Ungaguqula noma ngeliphi likhasi leladalwe ngekutentekela libe likhasi lelikhetsile uma ufuna lawulo lolugcwele lwekucuketfwa nesakhiwo salo.

## Kungeta Nekuhlela Emakhasi

1. Chofota inkinobho le- **Add Page** engoshweni lesekudla lengetulu kwelitafula le-Pages.
2. Khetsa luhlobo lwekhasi (lelingenalutfo nome i-template) bese ulinika ligama.
3. Chofota **Edit** eceleni kwanoma ngeliphi likhasi kute uvule [umhleli wekhasi](page-editor), lapho ungangeta khona tigaba, mbhalo, titfombe, naletinye ticenye.
4. Chofota **Page Settings** kute ubuyeketa sihloko sekhasi, indlela ye-URL, kanye naletinye tintfo temininingwane.
5. Sebentisa inkinobho ye- **Preview** kute uvule likhasi lakho kuwindi lensha bese ubona kabanti indlela leliyakubukeka ngayo kutivakashi.

:::tip
Kulikhasi lakho lasekhaya, simisa indlela ye-URL kuba `/` nje. Kuwo onkhe lamanye emakhasi, sebentisa indlela lechazako njenge `/about` nome `/contact`.
:::

## Kuphatsa I-Navigation

Isayidbha yangesencele yembukisi we-Website Pages ikhombisa tilinki takho te-navigation. Letilinki tilawula imenyu lebonwa tivakashi kuliwebhusayithi lakho.

1. Chofota **Add** kute udale linki lelisha le-navigation. Ungalikhomba kuliphi likhasi kulisayithi lakho nome ku-URL yangephandle.
2. Kute uhlele kabusha tilinki, tidvonse futsi utilahle endleleni loyifunako. Ungakwati futsi kufaka tilinki ngaphansi kwentfo yenhloko kute udale ema-menyu ekwehla (dropdown menus).
3. Chofota i-icon ye- **Edit** eceleni kwanoma ngeliphi linki kute ushintje libhalo lalo, i-URL, nome indzawo yalo.
4. Kute ususe linki ku-navigation, chofota i-icon ye- **Delete**.

:::info
Kususa linki le-navigation akulisusi likhasi ngco. Likhasi liyachubeka libakhona futsi lingafinyeleleka ngco nge-URL yalo -- alisoze lavela ku-menyu nje.
:::

## Emacebo Ekuhlela Lisayithi Lakho

- Gcina i-navigation yakho lengetulu kutindzawo letisihlanu nome letisitfupha kute tivakashi titfole tintfo ngekushesha.
- Sebentisa tilinki letifakwe ngaphansi kwentfo yenhloko yemakhasi lahlobene (sibonelo, i-dropdown ye-"About" leneti"Our Team," "Beliefs," ne-"History").
- Buka i-navigation yakho ku-mobile ngekuchofota **Mobile Preview** kute ucinisekise kutsi isebenta kahle ku-sikrini lesincane.
- Nika emakhasi emagama lacacile futsi lachazako lasita tivakashi kutsi baticondzele kutsi batawutfola ini.

:::tip
Ungangeta [emafomu](../forms/creating-forms.md) emakhasini akho kute ucoshe kubhalisa, ticelo temkhuleko, nome lenye imininingwane kutivakashi.
:::

## Kucala Ngesibonelo Sesayithi (Site Template)

Uma wakha lisayithi lakho kusuka ekucaleni, ungasicala ngekusebentisa **Site Template** esikhundleni sekudala emakhasi lilinye ngalinye. I-site template idala luhlu lwemakhasi lasengakhiwe ngaphambilini — likhaya, mayelana natsi, kutsintsana, kunikela, nakunye — kunekucuketfwa lokusayindlala kanye netilinki te-navigation setilungile kucala.

1. Kusikrini se-Pages, chofota inkinobho le- **Site Templates** (eceleni kwenkinobho le- **Add Page**).
2. Hamba etemplate letikhona bese uchofota lenye kute ubuke sakhiwo sayo semakhasi.
3. Nawutfola leyoyitsandzako, chofota **Apply Template**.
4. Emakhasi langakabikho kudalwa futsi angetwe ku-navigation yakho. Emakhasi lasekhona asalwa njengobe enjalo.

Ngemuva kwekusebentisa template, vula likhasi ngalinye ku- [umhleli wekhasi](page-editor) kute ushintje umbhalo netitfombe letisayindlala nge kucuketfwa langempela kwelibandla lakho.

:::info
Ema-Site templates adala sakhiwo semakhasi kanye ne-navigation. Awabuyeketi luhlu lwemibala lwelisayithi lakho nome imibhalo -- loko kulawulwa yi- [Appearance](appearance).
:::

## I-Image Lightbox

Nakuchofota tivakashi sitfombe kuliwebhusayithi lakho, sivuleka ku-lightbox overlay legcwala sikrini sonkhe. Loku kuvumela bantfu kubuka titfombe ngesayizi lenkhulu ngaphandle kwekushiya likhasi. Akukho lungiselelo ledzingekako -- i-lightbox ivunyelwa ngekutentekela yetitfombe kucuketfweni kwekhasi lakho.
