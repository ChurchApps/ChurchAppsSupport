# Kubukeka

<div class="article-intro">

Likhasi le-Appearance likuvumela kutsi ulungise sonkhe simo nekubukeka kweliwebhusayithi lelibandla lakho. Kusuka kumibala neminhlobo yemibhalo (fonts) kuya ekukhweleni kwendzawo (spacing) ne-CSS lekhetsile, ungalawula sonkhe simo lesibonakalako selisayithi lakho endzaweni yinye.

</div>

<div class="prereqs">
<h4>Before You Begin</h4>

- Cedza [Initial Setup](initial-setup) yeliwebhusayithi lakho
- Yenta kutsi ilogo yelibandla lakho ilungile ngesifomeni se-PNG lenesizinda lesicondzekako kanye ne-aspect ratio ye-4:1
- Yati imibala yebrendi welibandla lakho (emanani e-hex) uma unayo i-style guide lesele ikhona

</div>

## Kufinyelela Kutilungiselelo Te-Appearance

1. Ku-B1 Admin, chofota **Website** kumenyu wangesencele.
2. Chofota i-tab le- **Appearance** engetulu kwembukisi we-Website Pages.
3. Likhasi le-Site Styles liyalayisha nekubukisa lokuphilako kweliwebhusayithi lakho ngesencele kanye netinketfo te- **Style Settings** ngesekudla.

## I-Color Palette

1. Chofota **Color Palette** epanelini ye-Style Settings.
2. Utawubona **Base Colors** (indzawo lekhanyako, le-accent, kanye netinsika letimnyama) kanye ne **Semantic Colors** (Primary, Secondary, Success, Warning, ne-Error).
3. Chofota noma ngusiphi sitfombe sembala kute uvule sikhetsi sembala. Dvonsa sikhetsi nome ufake linani le-hex kute ukhetse umbala wakho.
4. **Color Combinations Preview** ikhombisa kutsi imibala loyikhetsile isebentana njani ndzawonye.
5. Sebentisa **Suggested Palettes** kute usebentise ngekushesha luhlu lwemibala lolusalungiswe.
6. Chofota **Save** nasewenelisekile.

## Umbhalo (Typography)

1. Chofota **Typography Settings** epanelini ye-Style Settings.
2. Chofota **Select a Font** kute uvule sikhangisi semibhalo. Ungasesha ngeligama nome uhambe etigabeni njenge-Serif, Sans Serif, Display, Handwriting, ne-Monospace.
3. Simisa imibhalo yato tobabili tihloko kanye nemibhalo yemtimba.
4. Chofota **Typography Scale** kute ubuyeketa lizinga lesayizi kusukela ku-Heading 1 kuya ku-Heading 4. Sebentisa emasikhala e-scale multiplier ne-base size kute uhlele kabanti.
5. Chofota **Save** kute usebentise tinketfo takho temibhalo.

## Kukhweleta Indzawo (Spacing)

1. Chofota **Spacing Scale** epanelini ye-Style Settings.
2. Buyeketa emanani ekukhweleta indzawo kusukela ku-Extra Small kuya ku-Extra Large. Tibonelo letisebentako tikhombisa kutsi linani ngalinye litsintsa njani indlela lekuhlelwe ngayo.
3. Chofota **Save Spacing** kute usebentise emanani kulo lonkhe lisayithi lakho.

## Ilogo ne-Branding

1. Chofota **Logo** epanelini ye-Style Settings.
2. Layisha i- **Light Background Logo** ne- **Dark Background Logo** yakho. Sebentisa titfombe letinesizinda lesicondzekako kanye ne-aspect ratio ye-4:1 kutfola imiphumela lemihle kakhulu.
3. Layisha i- **Social Media Image** yekubukiswa kwetilinki kanye ne- **Favicon** ye-icon yethebhu ye-browser.

:::tip
Kute utfole imiphumela lemihle kakhulu, sebentisa ilogo lenesizinda lesicondzekako kusifomeni se-PNG. Loku kucinisekisa kutsi ibukeka kahle kuletinsika letikhanyako naletimnyama kuliwebhusayithi lakho kanye [nase-app yeselula](../settings/mobile-app.md).
:::

## Tinhlobo Te-Navigation

Lungisa imibala ye-navigation bar yeliwebhusayithi lakho yato tobabili tinhlobo, lecinile ne-lecondzekako:

1. Yehlela uye esigabeni se- **Navigation Styles**
2. Chofota **Edit Navigation Styles**
3. Lungisa imibala ye-navigation lecinile (lenesizinda) kanye ne-navigation lecondzekako (i-overlay mode)
4. Chofota **Save** kute usebentise imibala yakho ye-navigation

Kutfola imiyalo lecacile, buka [Navigation Styles](./navigation-styles.md).

## I-CSS ne-JavaScript Lekhetsile

1. Chofota **CSS and Javascript** epanelini ye-Style Settings.
2. Ngeta **Custom CSS** kute ubuyeketa tinhlobo letijwayelekile ekulungiseni lokutfutfukile.
3. Ngeta **Custom HTML** yemakhodi ekulandzelela nome letinye tinhlelo.
4. Sebentisa sigaba se- **Common Javascript Examples** ngetibonelo njenge kuhlanganiswa kwe-Google Analytics.

:::warning
I-Custom CSS inemandla kodvwa ingephula simo selisayithi lakho uma isetjentiswa ngendlela lengalungi. Emabandla lamanyenti angatfola kubukeka lakufunako ngekusebentisa kulawula kwemibala, kwemibhalo, nekwendzawo lokwakhelwe ngekhatsi. Sebentisa i-custom CSS kuphela uma utikhulule ngekutfutfukiswa kwewebhu.
:::

## Tihloko Te-Style (Style Themes)

Uma ufuna indzawo lekhawuleta yekucala, i- **Suggested Palettes** esigabeni se-Color Palette inika tihloko letakhiwe ngaphambilini letimisa imibala lehambelanako ngekuchofota kanye. Ungahlala uhlela tilungiselelo ngatinye ngemuva kwekusebentisa sihloko.
