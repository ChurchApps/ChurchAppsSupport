# Kubakhona

<div class="article-intro">

Sigaba se-Kubakhona ku-B1 Admin sikunika umbono lophelele wekutsi ngubani lofikako ebandleni lakho nekutsi emaqembu akho akhula njani ngesikhatsi. Kusuka ekuhleleni emakhampasi netikhatsi tenkhonzo kuya ekubukeni inkambiso nekuhlela kungena ngekutingenisa, konkhe lokudzingekile kulandzelela nekucondzisisa kubakhona kuphatswa kulelikhasi.

</div>

## Kubukwa Kwelikhasi

Nawuvula likhasi Lekubakhona, utawubona hedi lenetibalo letisemcoka mayelana nekuhlelwa kwekubakhona kwebandla lakho:

- **Emakhampasi** -- linani lendzawo temzimba lohlelile
- **Tikhatsi Tenkhonzo** -- linani lelipheleleyo lwetinkhonzo kuwo onkhe emakhampasi
- **Emaqembu Lahleliwe** -- emaqembu lahlanganiswe nesikhatsi lesitsite senkhonzo
- **Emaqembu Langahlelwanga** -- emaqembu lalandzelela kubakhona ngekwawo kungakhatsateki sikhatsi senkhonzo

Ngaphansi kwehedi, ema-thebu lamatsatfu ahlela emathulusi akho ekubakhona.

## Ema-Thebu

### Kuhlelwa

Ithebu ye-**Kuhlelwa** yindzawo lapho uhlela khona tisekelo tekulandzelela kubakhona. Lapha uchaza emakhampasi akho, wengete tikhatsi tenkhonzo ekhampasini ngalinye, futsi uhlanganise [emaqembu](../groups/creating-groups.md) naletikhatsi tenkhonzo. Udzinga kwenta loku kanye kuphela, nome ungakwenta kabusha nome nini nakushintjsa uhlelo lwakho. Bona [Kuhlelwa Kwekubakhona](setup.md) kutfola luhambo lolugcwele.

### Kubakhona

Ithebu ye-**Kubakhona** ikhombisa inkambiso yekubakhona ngesikhatsi. Sebentisa tihlungi kute uncipise imiphumela ngesikhala semalanga, likhampasi, nobe sikhatsi senkhonzo bese ubona ngekushesha emaphetheni kumniningwane wakho. Bona [Kulandzelela Kubakhona](tracking-attendance.md) kutfola imininingwane yekusebentisa timbiko netihlungi.

### Emaqembu

Ithebu ye-**Emaqembu** yehlukanisa kubakhona ngeliqembu ngalinye. Loku kuyasita nawufuna kubona kutsi liqembu lelincane, ikilasi, nobe licembu lekukhonta liphila njani esikhundleni sekubuka linani lebandla lonkhe.
