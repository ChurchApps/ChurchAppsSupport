# Kungena

<div class="article-intro">

I-B1 Admin isekela kutingenisa emalunga etinkhonzweni ngale-app lehambisana nayo yebeka i-**B1 Checkin**. Emalunga angatingenisa wona nemindeni yawo kuma-kiosk nobe emadivayisi lakhetsekile nangabe afika, lokwenta inchubo ibe sheshako futsi kunciphise umsebenti kubasebentindlela bakho. Konkhe kungena kubhaliswa ngekutigcinela njengekubakhona.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Emakhampasi akho, tikhatsi tenkhonzo, netiqembu takho kufanele tihlelwe ku-[Kuhlelwa Kwekubakhona](setup.md).
- Udzinga [bantfu kulushicilelo lwakho](../people/adding-people.md) labanemindeni [lehleliwe](../people/adding-people.md#managing-households) kuze imindeni ikhone kungena ndzawonye.
- Utawudzinga i-tablet nangabe ufuna, i-Brother label printer (bona [tincomo tema-hardware](#recommended-hardware) ngentasi).

</div>

## Indlela Lekusebenta Ngayo

I-B1 Checkin app ihlanganiswa ne-B1 Admin setup yekubakhona. Nalilunga lingena, kubakhona kwalo kubhaliswa ngekutigcinela ekhampasini lelifanele, sikhatsi senkhonzo, neliqembu. Awudzingi kubhala kubakhona ngesandla kunobe ngubani losebentisa luhlelo lwekungena.

## Kuhlela Kungena

1. **Cala ngekuhlela sakhiwo sakho sekubakhona.** Ku-B1 Admin, yiya ku-**Kubakhona > Kuhlelwa** bese ucinisekisa kutsi emakhampasi akho, tikhatsi tenkhonzo, netiqembu takho tikhona. I-app yekungena isekelwe kuloku kuhlelwa. Bona [Kuhlelwa Kwekubakhona](setup.md) kutfola imininingwane.
2. **Faka i-B1 Checkin app** kumadivayisi lohlela kuwasebentisa. I-app itfolakala kuletinkundla letilandzelako:
   - **Ema-Tablet e-Android/Samsung:** [Google Play Store](https://play.google.com/store/apps/details?id=church.b1.checkin)
   - **Ema-Amazon Fire Tablet:** [Amazon App Store](https://www.amazon.com/Live-Church-Solutions-B1-Check-In/dp/B0FW5HKRB5/)
3. **Ngena ku-B1 Checkin app** usebentisa imininingwane ye-akhawunti yebandla lakho.
4. **Khetsa likhampasi nesikhatsi senkhonzo** yemhlangano wanyalo.
5. Emalunga manje angafuna ligama lawo kudivayisi bese angena.

:::tip
Beka emadivayisi ekungena endzaweni letibonakalako, letifinyeleleka lula njengasemnyango wekungena nobe etafuleni lekwemukela. Sitiso lesimfishane ngesikhatsi setinkhonzo sisita emalunga atiwe kutsi lenketfo iyatfolakala.
:::

:::tip
Nangabe libandla lakho linemakhampasi lamanyenti, kufanele uphindze kuhlelwa ngalinye likhampasi ku-[Kuhlelwa Kwekubakhona](setup.md). Idivayisi ngayinye yekungena ingahlelwa yelinye likhampasi lelihlukile.
:::

## Ema-Hardware Lancomekako

**Ema-Tablet** — nome ngusiphi salawa asebenta kahle ne-app:

- **Lincane:** Samsung Galaxy Tab A7 Lite 8.7"
- **Sikrini Lesikhulu:** Samsung Galaxy Tab A8 10.5"
- **Lekusindza:** Amazon Fire HD 10

**Emaphrinta** — kungena kusebenta nema-Brother label printer ekuphrinta emalebula emagama:

- **Lelihle Kunako:** Brother QL-1110NWB (isekela ema-tablet lamanyenti nge-Bluetooth ne-WiFi)
- **Lelihle:** Brother QL-810W (isekela ema-tablet lamanyenti nge-WiFi)
- **Lekusindza:** Brother QL-1100 (WiFi kuphela)

**Emalebula:** Brother DK-1201 (1-1/7" x 3-1/2")

:::warning
Ema-Brother label printer kuphela lahambelana ne-B1 Checkin app. Lezinye tinhlobo temaphrinta atiwotisebenti ekuphrinteni emalebula emagama.
:::

:::info
Landzela imiyalo yekuhlela lweliphrinta lakho kute ulihlanganise nemtsandzo lofanako we-WiFi wale-tablet yakho. Ungatfola tidraivi te-Brother printer netikhombiso tekuhlela ku-[iwebusayithi lesekelo le-Brother](https://support.brother.com).
:::

## Kuhlela Kubukeka Kwe-Kiosk

Ungahlela kubukeka kwe-B1 Checkin app kutsi kuhambisane netimake yebandla lakho. Ku-B1 Admin, yiya ku-**Kubakhona > Kiosk Theme** kute uhlele:

### Imibala

Hlela imibala lesiphohlongo kute ihambisane netimake yebandla lakho:

- **Lokusemcoka** ne-**Lokuphambana Nekusemcoka** -- Umbala lomkhulu wemake netembala walo wembhalo.
- **Lokwesibili** ne-**Lokuphambana Nokwesibili** -- Umbala wekugcizelela netembala walo wembhalo.
- **Lisiko Lehedi** ne-**Lisiko Lesabhu-hedi** -- Imibala yendzawo yehedi ye-kiosk.
- **Lisiko Lebhatani** ne-**Umbhalo Webhatani** -- Imibala yemabhatani lasebentako.

### Sitfombe Selisiko

Layisha sitfombe selisiko lesingakhetseka kumakhasi ekwemukela nekufuna e-kiosk. Lusayizi lolucondziwe ngu-1920x1080 emaphikseli.

### Sikrini Sekungasebenti / Screensaver

Hlela i-screensaver leyicalako ngemuva kwesikhatsi sekungasebenti:

1. Vula nobe uvale sikrini sekungasebenti.
2. Beka **sikhatsi**  (ngemasekhondi lamangaki ekungasebenti ngaphambi kwekutsi i-screensaver icale, kancane kunaka-10 emasekhondi).
3. Ngeta lisilayidi lelilodwa nobe lamanyenti -- ngasinye sisilayidi sinesitfombe nesikhatsi sekukhonjiswa (kancane kunemasekhondi lamatsatfu).

:::tip
Sebentisa sikrini sekungasebenti kute ukhombise tati so, imicimbi letako, nobe imilayeto yekwemukela nasi-kiosk singasebenti.
:::

## Kubhaliswa Kwetivakashi Nge-QR Code

Ikiosk yekungena ingakhombisa i-QR code leti vakashi tingayikalisela kuzibhalisa wona nemindeni yato kwafoni lato. Loku kusheshisa inchubo yekungena kutivakashi tekucala.

Nasikalisela QR code, tivakashi tiyisiwa [ekhasini lekubhaliswa kwesivakashi](../../b1-church/checkin/guest-registration) lapho tifaka ligama lato, i-imeyili, nemalunga emndeni. Umsebenti lofakako angabe abafuna kuma-kiosk bese abangenisa.

### Kuvula Kubhaliswa Kwesivakashi nge-QR

Kuvula kukhonjiswa kwe-QR code:

1. Ku-B1 Admin, yiya ku-**Mobile** kulohlu lolusesandleni sancele (luphawu lwefoni).
2. Khetsa ithebu ye-**Kungena**.
3. Vula **QR Guest Registration**.

:::note
Lesethingi likhona ngaphansi kwe-**Mobile**, hhayi ngaphansi kwe-Attendance > Kiosk Theme.
:::

## Loko Lokubhaliswako

Konkhe kungena kudala umbhalo wekubakhona ku-B1 Admin. Ungabuka lemibhalo kuma-thebu e-[Kubakhona](tracking-attendance.md) ne-[Emaqembu](../groups/group-members.md) njengoba unjalo nge-kubakhona lokubhalwe ngesandla. Akukho mehluko ekutseni imininingwane ivela njani — tonkhe letindlela tembili tiyangena kutimbiko letifanako.
