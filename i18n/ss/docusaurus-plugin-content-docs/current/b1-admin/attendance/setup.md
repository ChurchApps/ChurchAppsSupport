# Kuhlelwa Kwekubakhona

<div class="article-intro">

Ngaphambi kwekutsi ulandzelele kubakhona, udzinga kutjela i-B1 Admin ngendzawo yebandla lakho, ngesikhatsi tinkhonzo tenteka, netiqembu letihlangana kunkhonzo ngayinye. Loku kuhlelwa lokwentiwa kanye kudala sakhiwo lesipha emandla konkhe kulandzelela nekubika kwekubakhona kubandla lakho.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga i-akhawunti ye-B1 Admin lesebentako lenemvumo yekuphatsa kubakhona. Bona [Tindzima Netimvumo](../people/roles-permissions.md) nangabe awucacisisi lizinga lakho lekufinyelela.
- Nangabe uhlose kuhlanganisa emaqembu netikhatsi tenkhonzo, cinisekisa kutsi [emaqembu akho asetenti](../groups/creating-groups.md) kucala.

</div>

## Imibono Lesemcoka

- **Likhampasi** -- indzawo lekhona lapho libandla lakho lihlangana khona (sib., "Likhampasi Lelisemcoka," "Likhampasi Lasenyakatfo").
- **Sikhatsi Senkhonzo** -- umhlangano lophindzaphindzekako ekhampasini (sib., "Sonto nga-9:00 ekuseni," "Lesitsatfu nga-7:00 kusihlwa").
- **Liqembu Lelihleliwe** -- liqembu lelihlanganiswe nesikhatsi lesitsite senkhonzo. Kubakhona kulandzelelwa ngekwalesosikhatsi senkhonzo.
- **Liqembu Lelingahleliwe** -- liqembu leliyalandzelela kubakhona kwalo ngekwalo, ngaphandle kwekuhlanganiswa nesikhatsi senkhonzo.

## Kuhlela Sakhiwo Sakho Sekubakhona

1. Vula **B1 Admin** bese uchofota **Kubakhona** kulohlu lolusesandleni sancele.
2. Khetsa ithebu **Kuhlelwa**.
3. Chofota **Ngeta Likhampasi** bese ubhala ligama lendzawo yakho. Chofota **Gcina**.
4. Nalikhampasi lakho likhetsiwe, chofota **Ngeta Sikhatsi Senkhonzo**. Bhala ligama lifana ne-"Sonto nga-9:00 ekuseni" bese uchofota **Gcina**.
5. Phindza loku ngasosonkhe sikhatsi senkhonzo kulelolikhampasi.
6. Kuhlanganisa liqembu nesikhatsi senkhonzo, khetsa sikhatsi senkhonzo bese uchofota **Ngeta Liqembu**. Khetsa liqembu kuluhlu bese uchofota **Gcina**.

### Kuvula Kulandzelela Kubakhona Kuliqembu

Ngaphambi kwekutsi liqembu libhaliswe kubakhona, Kulandzelela Kubakhona kufanele kuvulwe kulelo liqembu.

1. Chofota **Emaqembu** kulohlu lolusesandleni sancele bese ukhetsa liqembu.
2. Chofota luphawu lwepensele le-**Hlela**.
3. Beka **Kulandzelela Kubakhona** ku-**Yebo**.
4. Chofota **Gcina**.

:::tip
Nangabe uhlanganise liqembu nesikhatsi senkhonzo esinyatselweni lesendlulile, sebentisa futsi inketfo ye-**Ngeta Sikhatsi Senkhonzo** kusikrini sekuhlela seliqembu kute ulihlanganise nesikhatsi lesifanele. Loku kucinisekisa kutsi ema-session ahlanganiswe nelikhampasi nesikhatsi lelifanele.
:::

:::tip
Nangabe liqembu lihlangana ngaphandle kwenkhonzo lejwayelekile -- njengeliqembu lelincane lelihlangana emkhatsini weviki lelilandzelela kubakhona kwalo -- ungalishiya njengeliqembu lelingahleliwe. Lisatawuvela ethebu Lemaqembu ekubikeni kwekubakhona.
:::

## Kuhlela Kuhlelwa Kwakho

Ungashintjsa kuhlelwa kwakho nome ngasiphi sikhatsi. Khetsa likhampasi, sikhatsi senkhonzo, nobe liqembu bese uchofota **Hlela** kute ushintjse imininingwane yalo, nobe **Susa** kute ulisuse.

:::info
Kususa sikhatsi senkhonzo akususi imibhalo yekubakhona leyendlulile. Imininingwane yakho yemlandvo igcinwa nome ushintjsa uhlelo lwakho.
:::

## Loko Lokulandzelako

Nawukacala emakhampasi akho, tikhatsi tenkhonzo, netiqembu takho tikhona, usukulungele kucala [kubhala kubakhona](recording-attendance.md) ngesandla nobe uhlele [kungena ngekutingenisa](check-in.md) yetinkhonzo takho.
