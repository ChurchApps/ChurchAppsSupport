# Kulandzelela Kubakhona

<div class="article-intro">

Nawukahlelwa emakhampasi akho, tikhatsi tenkhonzo, netiqembu, i-B1 Admin yenta kube lula kubuka imininingwane yekubakhona nekubona inkambiso. Likhasi Lekubakhona linika tindlela letimbili tekubika -- ithebu ye-**Kubakhona** yenkambiso yebandla lonkhe nethebu ye-**Emaqembu** yemininingwane yeliqembu ngalinye. Sebentisa lamathulusi kucondzisisa emaphetheni ekukhula, kukhomba kwehla kwekubandzakanyeka, futsi wente tincumo letisekelwe kwimininingwane yebandla lakho.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Sakhiwo sakho sekubakhona kufanele sihlelwe nekungenani likhampasi lilinye nesikhatsi senkhonzo. Bona [Kuhlelwa Kwekubakhona](setup.md) nangabe awukakwenti loku.
- Imininingwane yekubakhona idzinga kubhalwa ngaphambi kwekutsi timbiko tikhombise imiphumela. Imininingwane ingavela [kubhalweni ngesandla](recording-attendance.md) nobe [kungena ngekutingenisa](check-in.md).

</div>

## Kubuka Inkambiso Yekubakhona

1. Vula **B1 Admin** bese uchofota **Kubakhona** kulohlu lolusesandleni sancele.
2. Khetsa ithebu **Kubakhona**.
3. Umbiko usebenta ngekutigcinela nayivulwa ithebu, ukhombise kubakhona ngesikhala sesikhatsi lesejwayelekile.

## Kuhlunga Imininingwane Yakho

Sebentisa tihlungi letisephasini lelikhasi kute uncipise imiphumela:

- **Sikhala Semalanga** -- khetsa lusuku lwekucala nolwekuphela kute ugcizelele esikhatsini lesitsite.
- **Likhampasi** -- khetsa likhampasi kute ubone kubakhona kwaleyondzawo kuphela.
- **Sikhatsi Senkhonzo** -- khetsa sikhatsi senkhonzo kute ugcizelele emhlanganweni lotsite.

Igrafu nemininingwane kuyashintjsa ngekushesha nawushintjsa sihlungi, kuze ukhone kucatsanisa ngekushesha tikhatsi netindzawo letihlukene.

:::info
Timbiko tisebenta ngekutigcinela njalo nawuvula ithebu Yekubakhona, ngako uhlala ubona linani lakamuva ngaphandle kwekudzinga kuchofota inkhinombho yekuvuselela.
:::

## Kubakhona Kweliqembu

Ithebu ye-**Emaqembu** ikhombisa kubakhona lokwehlukaniswe ngeliqembu ngalinye. Loku kuyasita nawufuna kulandzelela ikilasi lelitsite, licembu lekukhonta, nobe liqembu lelincane esikhundleni sekubuka linani lelinkhonzo lonkhe.

1. Khetsa ithebu **Emaqembu**.
2. Khetsa liqembu kuluhlu kute ubone umlandvo walo wekubakhona.
3. Sebentisa sihlungi sesikhala semalanga kute uhlele sikhala sekubika.

:::tip
Kubakhona kweliqembu kubalulekile kakhulu kubaholi be-[emaqembu lamancane](../groups/creating-groups.md) labafuna kulandzelela kubandzakanyeka ngesikhatsi eliqenjini labo.
:::

## Timitsi Tekusebentisa Imininingwane Yekubakhona

- Buka inkambiso ngenyanga kute ubambe emaphetheni etinkhatsi tomnyaka kusengaphambili.
- Catsanisa imininingwane yemakhampasi kute ucondzisise kutsi ngutiphi tindzawo letikhulako.
- Sebentisa timbiko temaqembu kute ulandzelele [emaqembu](../groups/group-members.md) lakhombisa kwehla kwekubakhona.
- Hlanganisa imibono yekubakhona nelithulusi le-[AI Search](../people/ai-search.md) kute utfole bantfu labangakabikho kutsi baye masinyane.
