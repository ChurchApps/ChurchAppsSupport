# Kubhala Kubakhona

<div class="article-intro">

Nawukacala emakhampasi akho, tikhatsi tenkhonzo, netiqembu, ungabhala kubakhona ngesandla ngemuva kwemhlangano ngamunye. I-B1 Admin ihlela kubakhona ngema-**session** -- session yinye ngeliqembu ngelilanga lemhlangano. Udala session, ubike kutsi ngubani lofikile, futsi imininingwane ingena ngalokucondzile kutimbiko takho tekubakhona.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Emakhampasi akho, tikhatsi tenkhonzo, netiqembu kufanele tihlelwe. Bona [Kuhlelwa Kwekubakhona](setup.md) nangabe awukakwenti loku.
- Emaqembu lofuna kuwalandzelela kufanele abe ne-**Kulandzelela Kubakhona** kuvuliwe. Bona [Kuhlelwa Kwekubakhona](setup.md) kutfola imininingwane.

</div>

## Kudala Session

Session imele kwenteka kunye kwemhlangano weliqembu -- sibonelo, ikilasi yakho ye-K--3rd grade ngeSonto lelitsite.

1. Vula **B1 Admin** bese uchofota **Emaqembu** kulohlu lolusesandleni sancele.
2. Khetsa liqembu lofuna kulibhalela kubakhona.
3. Chofota ithebu **Ema-Session**.
4. Chofota **Lesisha** kute udale i-session lensha.
5. Nangabe liqembu lihlanganiswe nesikhatsi senkhonzo, khetsa **Sikhatsi Senkhonzo**. Nangabe kuliqembu lelingahlelwanga, lensimu ayivele.
6. Khetsa **Lusuku Lwe-Session** -- loku kungaba lamuhla, lusuku loluendlulile, nobe lusuku loluta.
7. Chofota **Gcina**.

:::tip
Ungadala ema-session emalangeni lendlulile kute ubambisane nekubakhona longakakubhali, nobe uwadale kusengaphambili kuze abe kulungele nasihlangana liqembu lakho.
:::

## Kubhala Kubakhona

Nawukagcina i-session, emalunga eliqembu avela ngasesandla sekudla kwelikhasi.

1. Hlolisa ibhokisi eceleni kwemuntfu ngamunye lofikile.
2. Tishintjso tigcinwa ngekutigcinela -- kute inkinombho lengetiwe ye-Gcina kuletimake tekubakhona.

:::info
Kuphela emalunga eliqembu langekwamanje avela kuluhlu lwe-session. Nangabe kukhona lofikile kepha longakakabi khona eliqenjini, [mengete eliqenjini](../groups/group-members.md) kucala, bese ubhala kubakhona kwakhe.
:::

## Kukhipha Kubakhona Ku-Spreadsheet

Ungakhipha umbhalo we-session njengelifayela le-CSV lekusetjentiswa ku-Excel, Numbers, nobe Google Sheets.

1. Vula i-session lofuna kuyikhipha.
2. Chofota inkhinombho **Khipha CSV**.
3. Vula lifayela lelikhishiwe ku-app yakho yespreadsheet.

## Kubuka Kubakhona Lokubhaliwe

Ngemuva kwekubhala ema-session, imininingwane ivela kutimbiko takho tekubakhona.

- **Ithebu Yekubakhona** -- ikhombisa inkambiso yebandla lonkhe ngesikhatsi. Bona [Kulandzelela Kubakhona](tracking-attendance.md).
- **Ithebu Yemaqembu** -- ikhombisa kubakhona lokwehlukaniswe ngeliqembu ngalinye.

:::tip
Nangabe i-session losandza kuyidala ingaveli ngekushesha kutimbiko, cinisekisa kutsi lusuku lwe-session lungaphakathi kwesikhala sesikhatsi lesikhetsiwe kuluhlungo lwetimbiko.
:::
