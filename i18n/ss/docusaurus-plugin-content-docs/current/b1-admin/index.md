# Sibuko Se-B1 Admin

<div class="article-intro">

B1 Admin sibhodi sakho sekuphatsa libandla. Siniketa emathulusi ekuphatsa yonkhe indzawo yemsebenti welibandla lakho -- kusukela kuluhlu lwemalunga neminikelo kuya ekwakhiweni kwewebhusayithi nekusakata bukhoma -- konkhe endzaweni yinye.

</div>

## Kucala

1. Vula i-browser yakho bese uya ku [admin.b1.church](https://admin.b1.church)
2. Ngena ngeligama lakho lekusebentisa nelifasikhodi
3. Utawufika ku **Dashboard**, leniketa simo lesibonisa live lelibandla lakho kanye **netento letisheshako** temisebenti lejwayelekile njengekwengeta bantfu, kudala emacembu, kubhala phasi iminikelo, netinye

:::tip
Musha ku-B1 Admin? Cala nge-[Introduction](./introduction) yekubona ividiyo lechaza konkhe, bese uvakashela ku-[Settings](./settings/) kucondzisa imininingwane yelibandla lakho futsi umeme ticu takho.
:::

## Tintfo Letibalulekile

- **[Bantfu](./people/)** - Phatsa luhlu lwemalunga elibandla lakho, wengete emalunga, uhlele imininingwane ngebunyenti, ulandzelele imindeni
- **[Emacembu](./groups/)** - Dala uhlele emacembu elibandla ngesicelo sekungena nekuphatsa emalunga
- **[Kubakhona](./attendance/)** - Setapisha likhampasi, tikhatsi tenkhonzo, futsi ulandzelele kubakhona
- **[Iminikelo](./donations/)** - Bhala phasi imininikelo ngetimali letahlukene, uphatse tikhwama, ukhicite tincwadzi temininikelo
- **[Kukhonta](./serving/)** - Hlela bavolontiya, udale tinhlelo tenkhonzo, uphatse imisebenti, futsi ushukumise bantfu ngetinyatselo tenchubo yemsebenti
- **[Emafomu](./forms/)** - Yakha emafomu lakhetsiwe ekubhalisa nekucoca imininingwane
- **[Emabhikho](./reports/)** - Buka emabhikho ekutalwa, ekubakhona, neminikelo
- **[Iwebhusayithi](./website/)** - Yakha uphatse iwebhusayithi yelibandla lakho ngetitayela letikhetsiwe tekuhamba
- **[Tintshumayelo](./sermons/)** - Phatsa layibrari yakho yetintshumayelo nekusakata bukhoma
- **[Emakhalendari](./calendars/)** - Dala emakhalendari lakhetsiwe, uphatse kubhukwa kwendlu/situlo, futsi ubuke sikhatsi lesikhona
- **[Settings](./settings/)** - Condzisa imininingwane yelibandla, tikhundla, imvumo, nemasimu lakhetsiwe emuntfu

:::info
Chofoza sithombe semfundo (?) lesisesitulwini sesekudla sangenhla saso sonkhe likhasi kutfola lusito nemibhalo ngekushesha.
:::

Kulusito, thumela imeyili ku [support@churchapps.org](mailto:support@churchapps.org).
