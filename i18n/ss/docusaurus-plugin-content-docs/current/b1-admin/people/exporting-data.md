# Kukhipha Imininingwane

<div class="article-intro">

I-B1 Admin ikuvumela kukhipha imininingwane yebandla lakho kute ungayisebentisa ku-spreadsheet, uyabelane nebutfu bakini, nobe ugcine sipele. Nome udzinga luhlu lolusheshako lwemabito ne-imeyili nobe kukhipha lokuphelele kwesichibi, kukhona tinketho letifanele tidzingo takho.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga i-akhawundi ye-B1 Admin lesebentako lenemvumo yekubona imininingwane lofuna kuyikhipha. Bona [Tikhundla & Timvumo](roles-permissions.md) uma ungacinisekile ngelizinga lakho lekufinyelela.
- Kwekukhipha lokuphelele kwesichibi, udzinga kufinyelela endzaweni ye-**Settings**.

</div>

## Kukhipha Kusuka Ekhasini Lebantfu

Indlela lesheshako yekukhipha sichibi sakho ngucondzana ekhasini le-**Bantfu**:

1. Yendlela ku-**Bantfu** kubar lesesandleni sancele.
2. Sebentisa insimu yekusesha nobe imikhawulo kute ucwaninge imiphumela lofuna kuyikhipha (nobe ushiye kungakhawulwa kute ukhiphe wonkhe muntfu). Bona [Kusesha Bantfu](searching-people.md) tetiluleko tekukhawula.
3. Sebentisa **sikhethi semakholomu** kute ukhetse kutsi ngumaphi emakholomu lofuna afakwe ekukhishweni (sibonelo, Libito, I-imeyili, Ucingo, Likheli).
4. Chofoza inkinobho ye-**Export**.
5. Ifayela le-CSV litakwehla kukhompyutha wakho lelinemininingwane lebonakala nyalo ethebulini.

:::tip
Lungisa emakholomu akho ngaphambi kwekukhipha. Ifayela le-CSV litafaka kanye emakholomu lowabonakalisile, ngako ungakwati kulungisa kukhipha kudzinga kwakho ngaphandle kwekuhlela ifayela ngemuva kwaloko.
:::

## Kukhipha Lokuphelele Kwemininingwane Kusuka ku-Settings

Kwekukhipha lokuphelele kwayo yonkhe imininingwane yakho ye-B1 (hhayi bantfu kuphela), sebentisa lithulusi lekukhipha ku-Settings:

1. Chofoza **Settings** kubar lesesandleni sancele.
2. Chofoza **Import/Export** kunavigeyishini lengetulu.
3. Khetsa **B1 Database** kudropdown ye-**Data Source**.
4. Buyeketa sibonelo semininingwane bese uchofoza **Continue to Destination**.
5. Khetsa **B1 Export Zip** njengendzawo lekuyiwa kuyo yekukhipha.
6. Bukisisa inchubo yekukhipha kuze konkhe kukhomba luphawu loluluhlata.
7. Ifayela lekukhipha litakwehla ngekutentela. Funa ifayela le-`B1Export` kufolda yakho yemakwehla.
8. Vula i-zip kute utfole emafayela e-CSV lahlukene (njenge-`people.csv`) longawavula ku-Excel, Google Sheets, nobe Numbers.

:::info
Kukhipha lokuphelele kwemininingwane kufaka bantfu, emacembu, tipho, kufika, kanye nakunye -- konkhe lokusesichibini sakho se-B1. Loku futsi kuyindlela lenhle yekwenta sipele lesihlala sikhona semarekhodi ebandla lakho.
:::

## Kukhipha Imininingwane Yelicembu

Ungakhona futsi kukhipha luhlu lwemalunga lemacembu latsite. Kusuka ekhasini le-**Groups**, vula licembu bese uchofoza **iconi yekwehlisela** kute ukhiphe luhlu lwemalunga alelocembu. Bona [Emalunga Elicembu](../groups/group-members.md) kwemininingwane leyengeti.

:::info
Emafayela e-CSV lakhishiwe asebenta netonkhe ticelo letikhulu tespreadsheet lekufaka Microsoft Excel, Google Sheets, ne-Apple Numbers.
:::
