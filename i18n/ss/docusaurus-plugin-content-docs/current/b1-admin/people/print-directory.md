# Phrinta Luhlu Lwemalunga

<div class="article-intro">

Insita yekuPhrinta Luhlu Lwemalunga yenta luhlu lwemalunga lwelibandla lakho oluhlelekile nolungaphrintwa. Ihlela emalunga ngekwemakhaya, ifaka imininingwane yekuxhumana kanye netitfombe, futsi yakhelwe kubukeka ngendlela yebuchwepheshe nayiphrintwa emaphepheni lasayindza sagcina (letter-size).

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga i-akhawundi ye-B1 Admin lesebentako lenemvume yekubona bantfu.
- Bantfu labakuluhlu lwakho kufanele babe netitfombe kanye nemininingwane yekuxhumana legcwelisiwe kuze utfole imiphumela lemihle kakhulu.

</div>

## Kuvula Luhlu Lolutakuphrintwa

1. Yendlulela ku-**People** ku-B1 Admin.
2. Chofota i-**print icon** (luphawu lwephrinta) ku-toolbar lesetulu kwelikhadi lemiphumela ye-People. Ivela eceleni kwenkinobho ye-Export.
3. I-Print Directory ivula kuthebhu lensha ye-browser bese icala kulayisha imininingwane yeluhlu lwakho.
4. Ngemuva kwemazuzwana lacishe abe ngu-1.5, ibhokisi lekuphrinta le-browser yakho litovela ngekutitjenga.

:::tip
Luhlu luvula kuthebhu lehlukene kuze ukwati kuluphrinta ngaphandle kwekushiya likhasi le-People. Ngemuva kwekuvela kwebhokisi lekuphrinta, ithebhu ibuyela emuva ngekutitjenga.
:::

## Yini Lefakwako

Luhlu lolutakuphrintwa lufaka onkhe emalunga lasebentako elibandla lakho. Ngalokucacile:

- Bantfu labanesimo selilunga lesitsi Inactive nobe Visitor bayakhishwa ngekutitjenga.
- Bantfu labakhetse kungabi khona kuluhlu bayakhishwa.
- Bonkhe labanye bahlelwa ngekwemakhaya.

## Simo Seluhlu

Luhlu loluphrintiwe lunetincenye letimbili:

### Likhasi Lelicembe

Likhasi lekucala lifaka:
- Ligama lelibandla lakho
- Sihloko lesitsi "Member Directory"
- Umnyaka wanyalo
- Indzawo yelibandla lakho
- Inani lawonkhe emakhaya lafakiwe

### Luhlu Lwemalunga

Emakhasi lasele abala emakhaya ngekulandzelana kwesinyatsi ngekwesibongo. Likhadi lelikhaya ngalinye likhombisa:

- **Titfombe** -- Titfombe letiyindingilizi ngelilunga ngalinye lelikhaya, negama lalo ngaphansi
- **Ligama leliboniswako lelikhaya** -- Sibonelo, "The Smith Family", "John & Jane Doe", nobe nje ligama lemuntfu kubantfu labasodvwa
- **Likheli** -- Likheli lemgwaco, idolobha, sifundza, kanye ne-zip
- **Lucingo** -- Inombolo yasekhaya nobe yeselula
- **Imeyili** -- Likheli leliyinhloko le-imeyili
- **Emalanga ekutalwa** -- Ahlelwe njengetinsuku letihlukene ngelilunga ngalinye lelikhaya
- **Anniversary** -- Nangabe ibhaliwe

Emakhaya ahlelwe ngetigaba tesinyatsi (A, B, C, ...) kunesihloko lesihlobisiwe embili kwelicembu ngalinye lelitinyatsi. Emakhadi lamabili emakhaya avela eceleni nakelinye kumugca ngamunye.

## Emasethingi Ekuphrinta

Luhlu luhlelelwe liphepha lelisayindza sagcina (letter-size) (8.5" × 11") ngemamajini langu-0.5-inch. Nayivela ibhokisi lekuphrinta ye-browser:

- Yekela lisayindza leliphepha lisetwe ku-Letter
- Vula i-Background graphics (kwesinye sikhatsi kubitwa "Print backgrounds") kumasethingi ekuphrinta e-browser yakho nangabe ufuna tihloko tetigaba tiphrinte nesitayela sato sembala
- Kuncoyiwe kusetjentiswa i-portrait orientation

:::info
Luhlu luyakuhlonipha kwehlukaniswa kwemakhasi — emakhadi emakhaya netihloko tetigaba angeke tihlukaniswe emakhasini lamaningi.
:::
