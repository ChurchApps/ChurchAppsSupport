# Kwengeta Bantu

<div class="article-intro">

Incenye ye-People iyisisekelo se-B1 Admin — yidatabase yemalunga ebandla lakho. Onkhe emanye emakhono (emacembu, kuba khona, iminikelo, emafomu) ahlanganiswa nemarekhodi emuntfu ngamunye. Lolucwaningo lukuhambisa ekufakeni umuntfu kudatabase yakho, ekuhleleni imininingwane yakhe, nasekuhlanganiseni emalunga emndeni abe yindlu yinye.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga i-akhawunti ye-B1 Admin lesebentako lenemvumo yekuphatsa bantu. Buka [Roles & Permissions](roles-permissions.md) nangabe awucacelwa ngemvumo yakho.
- Nangabe ufaka bantu labaningi kunalikhulu, cabangela kusebentisa lithulusi le-[CSV Import](importing-data.md) esikhundleni.

</div>

## Kwengeta Umuntfu

1. Iya ku-B1.church Admin dashboard.
2. Chofota **People** kuluhlu lolusesandleni sancele.
3. Chofota inkinobho **Add Person** lesesikhulwini sasekudla.
4. Faka ligama lekucala lemuntfu, sibongo, kanye ne-imeyili, bese uchofota **Add**.

Likhasi lephrofayela yemuntfu litawuvuleka, lilungele kutsi wengete imininingwane leyengetiwe.

:::tip
Nangabe usuka kuhlelo lolunye lekuphatsa bandla, sici se-[Import Data](importing-data.md) sikuvumela kutsi ulethe yonkhe idatha yakho yebantfu kusuka kufayela le-CSV — lokushesha kakhulu kunekufaka bantfu munye ngamunye.
:::

## Kuhlela Imininingwane

1. Ekhasini lephrofayela yemuntfu, chofota **iphenisela lekuhlela** eceleni kwelibito lakhe.
2. Faka imininingwane lengetiwe njengelibito lemphakatsi, simo sebulunga, tinsuku, likheli, kanye netinombolo tebucingo.
3. Chofota **Save** kute ugcine imininingwane yemuntfu.

Iphrofayela nayo inetimenyu letinyenti tetintfo letihambelana:

- **Notes** — Engeta emanotsi ngemuntfu (kunakekelwa kwemuntfu, kulandzelela, njll.)
- **Groups** — Buka futsi uphatse [bulunga belicembu](../groups/group-members.md)
- **Attendance** — Buka [emarekhodi ekuya](../attendance/tracking-attendance.md)
- **Donations** — Buka [umlandvo weminikelo](../donations/recording-donations.md)

## Kusebenta Netifomu

Ungagcwalisa emafomu langakhelwe wena macal ekhasini lephrofayela yemuntfu. Lawa ngemafomu lacalwe ngumsebentisi longawakha ngekulandzela lucwaningo lwe- [Creating Forms](../forms/creating-forms.md).

1. Ekhasini lephrofayela yemuntfu, chofota luhlu lolwehlako **Forms** kute ukhetse ifomu.
2. Chofota **Add Form** kute uyivule.
3. Gcwalisa imininingwane yefomu bese uchofota **Save**.

:::info
Emafomu lahlotjaniswe nephrofayela yemuntfu asebentisa luhlobo lwefomu **People**. Nangabe udzinga ifomu lengaziphileko (njengekubhalisela umcimbi), buka [sici se-Stand Alone form](../forms/creating-forms.md) elucwaningweni lwemafomu.
:::

:::tip
Nangabe udzinga kulandzelela kancane kwemininingwane lengetiwe kubantu — lusuku, linombolo, imphendvulo ye-yebo/cha — sebentisa [Custom Fields](../settings/custom-fields.md) esikhundleni sefomu. Ziphangisa kugcwaliswa futsi ziyatfolakala nakusetjentiswa i-Advanced Search.
:::

## Kuphatsa Imindeni

Imindeni ikuvumela kutsi uhlanganise emalunga emndeni ndzawonye. Loku kusita kakhulu ku-[kubhalisa nasufika](../attendance/check-in.md), lapho umtali angabhalisela bonkhe bantfwana bakhe kanyekanye.

1. Ekhasini lemuntfu, chofota **iphenisela lekuhlela** eceleni kwelibito lemndeni.
2. Sisebentiso sekuhlela umndeni sitawuvuleka. Khetsa **indzima yemndeni** yemuntfu wamanje (sib. Inhloko, Umyeni/Umfati, Umntfwana).
3. Chofota **Add** kute wengete lilunga lelinye lemndeni.
4. Bhala libito lemuntfu ebhokisini lekusesha bese uchofota **Search**.
5. Nasekavela umuntfu emiphumeleni yekusesha, chofota **Select**.
6. Khetsa indzima yakhe yemndeni bese uchofota **Save** kute ucedze kumisa umndeni.
