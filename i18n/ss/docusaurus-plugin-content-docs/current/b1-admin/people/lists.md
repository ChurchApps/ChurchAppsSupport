# Emalulu Lagciniwe

<div class="article-intro">

**Saved Lists** ikuvumela kugcina sicelo sekusesha ngaphansi kweligama futsi usiphinde ngekuchofota kanye nome ngunini. Yakha luhlu kanyekanye — "Active Members in Campus North," "Visitors in the last 90 days," nobe nome ngukuphi kuhlunga lokuphakeme — bese luhlala ku-sidebar yakho kuze ungaphindzi kwakha kabusha sicelo futsi.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Chubeka kusesha okuncane kunye esigabeni se-People kuze ube nemiphumela yekugcina. Bona [Searching People](./searching-people.md) nobe [AI Search](./ai-search.md).

</div>

## Kugcina Kusesha Njengeluhlu

1. Chubeka kusesha kulikhasi le-People — kusesha ngekushesha, kusesha nge-AI, nobe kusesha ngekuhlunga lokuphakeme.
2. Nayivela imiphumela, inkinobho ye-**Save as List** (luphawu lwe-bookmark) iyabonakala ku-header yelikhasi le-People.
3. Chofota **Save as List**.
4. Faka **Name** yeluhlu (kuyadzingeka).
5. Ngekukhetsa, faka **Category** kuze uhlanganise emalulu lahlobene ku-sidebar (sibonelo, "Outreach" nobe "Membership").
6. Chofota **Save**.

Luhlu selugciniwe ku-akhawundi yelibandla lakho futsi lutawuvela ku-**Saved Lists panel** lengakwesancele lelikhasi le-People.

:::info
Emalulu abelwana kulo lonkhe libandla lakho — nome ngubani lonekufinyelela ku-People angawabona futsi awalayishe. Ngusebenti lokuphela lonemvume lefanele longaphendula ligama nobe asuse emalulu.
:::

## Kulayisha Luhlu Lolugciniwe

Ku-**Saved Lists panel** lengakwesancele lelikhasi le-People, chofota nome ngeliphi ligama leluhlu. Kusesha kuyaphindza msinya ngesicelo lesigciniwe, kufakwe lokusha ithebula lemiphumela.

Emalulu ahlanganiswa ngesigaba (nangabe wanika lesinye), kuthi emalulu langakabelwa sigaba avele ekugcineni.

:::tip
Emalulu ticelo letiphilako, hhayi tinshothi (snapshots). Sikhatsi ngasinye nawulayisha luhlu, kusesha kuchubeka busha kumelene nemininingwane yakho yamanje, kuze bantfu labasha labamanje labafanako nesipho babonakale.
:::

## Kuphendula Ligama Nekususa Emalulu

Sebenti lonemvume yekuphatsa angaphendula ligama nobe asuse nome nguluphi luhlu ngco ku-**Saved Lists panel**.

- Chofota **luphawu lwepensela** eceleni kweligama leluhlu kuze uliphendule.
- Chofota **luphawu lwendzaweni yekulahla** (trash icon) kuze ulisuse (utawubutwa kucinisekisa).

## Tibonelo Tekusebentisa

| Simo | Kwakha luhlu njani |
|---|---|
| Tonkhe tivakashi | Advanced search: Membership Status = Visitor |
| Emalunga esikhungo lesitsite | Advanced search: Campus = [sikhungo sakho] |
| Bantfu labangenayo imeyili | Advanced search: Email is empty |
| Bavolontiya labanembhalo we-background check lophelelwe sikhatsi | Advanced search: [Custom Field](../settings/custom-fields.md) "Background check expires" is before today |
| Imiphumela lesuka embutweni we-AI | Buta umbuto ku-AI Search, bese ugcina |
