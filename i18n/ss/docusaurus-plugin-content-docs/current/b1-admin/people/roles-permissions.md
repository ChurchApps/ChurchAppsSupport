# Kunikeza Tikhundla

<div class="article-intro">

I-B1 Admin isebentisa luhlelo lwemvumo lolusekelwe etikhundleni kute kulawulwe kutsi ngumaphi lomsebenti ngamunye webutfu bakini langakubona nekukwenta. Ngekunikeza tikhundla, ungakhona kunika bomsebenti kanye netisebenti tekuvolontiya kufinyelela endzaweni lebadzinga kona -- kanye kuphela. Kuphatsa kahle tikhundla kugcina imininingwane yebandla lakho iphephile ngesikhatsi kunika butfu bakini emandla kusebenta ngendlela lesheshako.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga kufinyelela kwe-**Domain Admin** nobe sikhundla lesinemvumo yekuphatsa i-**Settings** ku-B1 Admin.
- Bantfu lofuna kubanika tikhundla kufanele sebebakhona echibini lakho. Bona [Kwengeta Bantfu](adding-people.md) uma udzinga kubengeta kucala.

</div>

## Kucondzisisa Tikhundla

Sikhundla luhlu lwemvumo losinikeza umsebenti munye nobe labanyenti. Sibonelo, ungakhona kudala sikhundla se-"Butfu Betimali" lesinika kufinyelela [emarekhodini etipho](../donations/recording-donations.md), nobe sikhundla se-"Sivolontiya Sekungena" lesivumela kufinyelela [tibonelo tekufika](../attendance/check-in.md) kuphela.

Sikhundla ngasinye silawula kufinyelela etindzaweni letitsite te-B1 Admin, lekufaka:

- **Bantfu** -- kubona nekuhlela tiphrofayili temalunga
- **Tipho** -- kuphatsa umnikelo kanye netincwadzi tetimali
- **Kufika** -- kubhalisa nekubona imininingwane yekufika
- **Emafomu** -- kudala nekuphatsa [emafomu lakhethekile](../forms/creating-forms.md)
- **Emacembu** -- kuphatsa [bulunga bemacembu](../groups/group-members.md) kanye netikhalendari
- **Settings** -- kulungisa tilungiselelo tebandla lonkhe

:::warning
**Bo-Domain Admin** banekufinyelela lokuphelele kuyo yonkhe indzawo ye-B1 Admin. Timvumo tabo atikwati kuhlelwa nobe kukhawulwa. Sebentisa lesikhundla kubaphatsi bakho labakhulu kuphela.
:::

## Kubuka Nekuphatsa Tikhundla

1. Chofoza **Settings** kubar lesesandleni sancele.
2. Chofoza **Roles** kunavigeyishini lengetulu.
3. Utawubona luhlu lwato tonkhe tikhundla letilungiselelwe libandla lakini.
4. Chofoza nasiphi sikhundla kute ubone emalunga aso kanye netimvumo taso.

## Kwengeta Bomsebenti Esikhundleni

1. Yendlela ku-**Settings** bese ku-**Roles**.
2. Chofoza sikhundla lofuna kwengeta umsebenti kuso.
3. Esigabeni se-**Members**, sesha umuntfu ngelibito.
4. Chofoza **Add** kute umnikeze sikhundla.

Umsebenti utawuba nato tonkhe timvumo letihambisana nesikhundla lesito sikhatsi lesitako angena.

## Kuhlela Timvumo Tesikhundla

1. Yendlela ku-**Settings** bese ku-**Roles**.
2. Chofoza sikhundla lofuna kusishintja.
3. Esigabeni se-**Permissions**, faka nobe ususe luphawu etindzaweni lofuna sikhundla sikwati kutifinyelela.
4. Chofoza **Save** kute usebentise tishintjo takho.

:::tip
Landzela umgomo wekuniketa livi lelincane -- nika sikhundla ngasinye timvumo lesitidzingako ngempela kuphela. Loku kugcina imininingwane yakho iphephile futsi kwehlisa lithuba lekushintja lokungakahleliwe.
:::

## Tibonelo Letivamile Tetikhundla

- **Bomsebenti Behhovisi** -- kufinyelela ku-Bantfu, Tipho, Kufika, kanye Nemafomu
- **Baholi Bemacembu** -- kufinyelela ku-[Emacembu](../groups/creating-groups.md) kuphela
- **Bavolontiya Bekungena** -- kufinyelela ku-[Kufika](../attendance/check-in.md) kuphela
- **Butfu Betimali** -- kufinyelela ku-[Tipho](../donations/recording-donations.md) kanye nekubika
