# Kusesha Bantfu

<div class="article-intro">

Likhasi le-**People** likhombisa luhlu lwakho lwelibandla kuthebula lelingaseshwa nelingahlelwa. Ungatfola ngekushesha nome ngubani elibandleni lakho, uhlele imininingwane lekhonjiswako, futsi ukhiphe imiphumela yakho. Kusesha ngendlela lesebentako kubalulekile emisebentini yemihla ngemihla yekuphatsa libandla njengekulandzelela tivakashi, kulungisa luhlu lwekuxhumana, kanye nekuphatsa emarekhodi emalunga.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga i-akhawundi ye-B1 Admin lesebentako lenemvume yekubona bantfu. Bona [Roles & Permissions](roles-permissions.md) nangabe awucacelwa kutsi kufinyelela kwakho kufike kuphi.
- Luhlu lwelibandla lakho kufanele lube nebantfu kulo. Nangabe ungakabi wengeta muntfu, bona [Adding People](adding-people.md) nobe [Importing Data](importing-data.md).

</div>

## Kusesha Ngekushesha

Ibha yekusesha lesetulu kwelikhasi le-**People** ikuvumela kutfola emalunga ngesikhatsi lesiphilako:

1. Chofota **libhokisi lekusesha** letulu kwelikhasi le-People.
2. Cala kubhala ligama, imeyili, nobe lelinye ligama lelidzingekako.
3. Imiphumela itohlungwa ngekutitjenga njengobe ubhala (kunekulibala lokufishane lokungatsi ngukhona lengalinye lesekhondi kuze kusesha kungacali kunkinobho ngayinye leshayiwe).
4. Ithebula lelingentasi liyafakwa lokusha kuze likhombise kuphela imiphumela lefanako.

:::tip
Awudzingi kucindzetela i-Enter. Kusesha kuchubeka ngekutitjenga ngemuva kwekutsi uyekele kubhala.
:::

## Kuhlela Imiphumela

Ungahlela luhlu ngekuchofota nayiphi inhloko yekholomu kuthebula:

1. Chofota **inhloko yekholomu** (sibonelo, **Name** nobe **Email**) kuze uhlele ngaleyokholomu.
2. Chofota lenhloko lefanako futsi kuze uphendule luhlelo lekuhlela.

Loku kwenta kubelula kutfola bantfu ngekulandzelana kwesinyatsi, ngeminyaka, nobe ngayo yonkhe lenye ikholomu lebonakalako.

## Kulungisa Ngekwakho Emakholomu

Akusiko konkhe kwemininingwane lokudzingeka kubonakale ngesikhatsi sinye. Ungakhetsa kutsi ngutiphi tikholomu letivelako kuthebula:

1. Funa i-**column selector dropdown** ecishe etulu kwethebula.
2. Hlola nobe ususe luphawu kumakholomu kuze ubonise nobe ufihle. Emakholomu lakhona afaka:
   - **Photo**
   - **Name**
   - **Email**
   - **Phone**
   - **Address**
   - **Birth Date**
   - **Age**
   - **Gender**
   - **Membership Status**
   - **Campus**
3. Ithebula lifakwa lokusha ngekushesha kuze likhombise lokukhetsile.

:::info
Lokukhetsa kwakho emakholomu kutsintsa loko lokufakwako nawukhipha kuya ku-CSV. Lungisa emakholomu ngaphambi kwekukhipha kuze utfole imininingwane loyidzingako ngco.
:::

## Emakhasi (Pagination)

Nangabe luhlu lwakho lunemarekhodi lamaningi, imiphumela ihlukaniswa emakhasini. Sebentisa **emalawulo ekuphenda emakhasi** ngephasi kwethebula kuze unyakate emkhasini nemkhasi. Likhasi lamanje kanye nenani lelilonkhe lemarekhodi kuyakhonjiswa kuze wati sikhatsi sonkhe kutsi ukuphi kuluhlu.

:::tip
Nangabe ufuna kubona imiphumela leminingi ngesikhatsi sinye, lungisa kusesha kwakho kuze unciphise luhlu kunekuphenda emakhasini eluhlwini lolukhulu.
:::

## Kukhipha Imiphumela Yekusesha

Ungakhipha imiphumela yakho yamanje yekusesha njengefayela le-CSV nome ngunini:

1. Sebentisa nome ngukuphi kusesha nobe emafilta lowufuna.
2. Lungisa emakholomu akho kuze afake imininingwane loyidzingako.
3. Chofota inkinobho ye-**Export**.
4. Ifayela le-CSV litawukhishwa kuya kukhompyutha yakho, lilungele kuvulwa ku-Excel, Google Sheets, nobe nome nguluphi luhlelo lwe-spreadsheet.

Kuze utfole imininingwane leyengetiwe ngekukhipha, bona [Exporting Data](./exporting-data.md).

:::tip
Kuletisipho letiphakeme kakhulu -- njengekutfola wonkhe umuntfu longakayi emihlanganweni kuleinyanga letintsatfu tekugcina -- zama insita ye-[AI Search](./ai-search.md), leyikuvumela kusesha usebentisa imibuto ngelulwimi lolulula.
:::

## Kusesha Lokuphakeme (Advanced Search)

I-Advanced Search ikuvumela kwakha emafilta lacacile ngekuhlanganisa timo. Yivule kusuka likhasini le-People, bese wandzisa sigaba bese uhlola emasimu lofuna kuwahlunga, ukhetsa i-operator kanye nenani ngasinye. Tigaba tifaka **Names**, **Demographics**, **Contact**, **Membership**, **Activity** (iminikelo nekuya emihlanganweni), kanye ne-**Custom Fields**.

Sigaba se-**Custom Fields** sibala [Custom Fields](../settings/custom-fields.md) telibandla lakho — emasimu lowawachaza ku-Settings kuze ulandzelele imininingwane yakho lekhetsekile (njengelilanga lekuphela kwe-background check). Ema-operator lanikeziwe ahambisana nemhlobo wesimu ngasinye: emasimu embhalo asekela *contains / equals / starts with / ends with*, emasimu enombolo asekela ema-operator ekucatsanisa, emasimu elilanga asekela *equals / after / before*, kanye nemasimu e-Yes/No ne-Multiple Choice akuvumela kukhetsa inani. Nome nguliphi lisimu longalihlunga lapha lingagcinwa njenge-[List](./lists.md) leliphilako.

## Kugcina Kusesha Njengema-List

Ngemuva kwekuchubela kusesha, inkinobho ye-**Save as List** (luphawu lwe-bookmark) ivela ku-header yelikhasi le-People. Yichofote kuze ugcine sicelo sakho samanje ngaphansi kweligama kanye nesigaba lesingakhetsakala, kuze ukwati kululayisha kabusha msinya kutinkhathi letitako. Bona [Saved Lists](./lists.md) kuze utfole imininingwane lephelele.
