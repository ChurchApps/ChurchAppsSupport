# Kuhlela Bantfu Ngebuningi

<div class="article-intro">
Kuhlela ngebuningi kukuvumela kubuyeketa bantfu labanyenti ngesikhatsi sinye, kunonga sikhatsi nawenta lushintjo lolufanako kubantfu labanyenti. Ungabuyeketa simo selilunga, simo semshado, bulili, tinketho tekungangeni, kanye nebulunga bemacembu ngekusebenta kunye.
</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga imvumo yekuphatsa imininingwane yebantfu. Bona [Tikhundla & Timvumo](./roles-permissions.md) kwemininingwane.
- Kufanele ube sewabafaka noma wangenisa bantfu lofuna kubahlela. Bona [Kwengeta Bantfu](./adding-people.md) uma kudzingeka.
</div>

## Kukhetsa Bantfu Kwekuhlela Ngebuningi

1. Yendlela ku-**Bantfu** ku-B1 Admin
2. Sebentisa emathulusi ekusesha nobe kukhawula kute utfole bantfu lofuna kubabuyeketa
3. Faka luphawu emabhokisini langakwelibito lemuntfu ngamunye kute umkhethe
   - Ungakhetsa bantfu ngamunye ngamunye
   - Nobe usebentise ibhokisi lesihloko kute ukhetse bonkhe bantfu lababonakalako ekhasini lanyalo
4. Nawusuke wakhetsa umuntfu lomunye noma ngetulu, inkinobho ye-**Tento Tebuningi** itakuvela

:::tip
Nangabe udzinga kubuyeketa licembu lelikhulu lebantfu ngekusekelwe timo letitsite, sebentisa insita ye-[Kusesha nge-AI](./ai-search.md) nobe imikhawulo kute ucwaninge luhlu lwakho kucala, bese ukhetsa bonkhe bantfu labafanako.
:::

## Tento Tebuningi Letikhona

Imenyu ye-**Tento Tebuningi** iniketa tinketho letinyenti:

### Buyeketa Simo Selilunga

Buyeketa simo selilunga sebantfu bonkhe labakhetsiwe:

1. Chofoza **Tento Tebuningi** → **Beka Simo Selilunga**
2. Khetsa simo lesisha:
   - **Sivakashi** -- Labatfola bafike kucala nobe ngaletinye tikhatsi
   - **Sivakashi Sesikhatsi** -- Labafika njalo kepha bangesiwo emalunga
   - **Lilunga** -- Emalunga alibandla lasemtsetfweni
   - **Sisebenti** -- Emalunga esisebenti selibandla
   - **Akasebenti** -- Bantfu labangasafiki
3. Khetsa indlela yekubuyeketa:
   - **Bhala phetulu konkhe** -- Shintja simo sanyalo sebonkhe bantfu labakhetsiwe
   - **Buyeketa lokungenalutfo kuphela** -- Beka simo kubantfu labangenaso kuphela
4. Chofoza **Buyeketa**

### Buyeketa Simo Semshado

Buyeketa simo semshado ngebuningi:

1. Chofoza **Tento Tebuningi** → **Beka Simo Semshado**
2. Khetsa simo lesisha:
   - **Akwatiwa**
   - **Ngashadi**
   - **Ngishadile**
   - **Ngehlukanisile**
   - **Ngumfelokati/Ngumfelwa**
3. Khetsa kutsi ubhale phetulu tinani letikhona nobe ubuyeketa lokungenalutfo kuphela
4. Chofoza **Buyeketa**

### Buyeketa Bulili

Buyeketa imininingwane yebulili yebantfu labanyenti:

1. Chofoza **Tento Tebuningi** → **Beka Bulili**
2. Khetsa linani:
   - **Akuchazwanga**
   - **Lomdvuna**
   - **Lomsikati**
3. Khetsa indlela yekubuyeketa (bhala phetulu konkhe nobe lokungenalutfo kuphela)
4. Chofoza **Buyeketa**

### Buyeketa Simo Sekungangeni

Lawula kutsi bantfu bakhiphile yini kunobuchwepheshe:

1. Chofoza **Tento Tebuningi** → **Beka Kukhipha**
2. Khetsa:
   - **Cha** -- Vumela nobuchwepheshe (susa kukhishwa)
   - **Yebo** -- Vala nobuchwepheshe (beka kukhishwa)
3. Khetsa indlela yekubuyeketa
4. Chofoza **Buyeketa**

:::warning
Yicophelela nawushintja simo sekukhipha. Bantfu labebakhiphe ngalokucacile akufanele bemukele nobuchwepheshe ngaphandle kwekuvuma kabusha.
:::

### Ngeta Ecenjini

Ngeta bonkhe bantfu labakhetsiwe kulicembu linye nobe kumacembu lamanyenti:

1. Chofoza **Tento Tebuningi** → **Ngeta Ecenjini**
2. Sesha bese ukhetsa licembu (nemacembu) lofuna kungeta bantfu kulo
3. Ungakhetsa emacembu lamanyenti kute ungete bantfu kuwo onkhe
4. Chofoza **Ngeta Emacenjini**

Umuntfu ngamunye utawungetwa njengelilunga lesejwayelekile lelicembu (nemacembu) lakhetsiwe. Ungakhona kuphakamisa umuntfu abe ngumholi welicembu ngesikhatsi lesitako kusuka ekhasini le-[Emalunga Elicembu](../groups/group-members.md).

### Susa Ecenjini

Susa bonkhe bantfu labakhetsiwe kulicembu linye nobe kumacembu lamanyenti:

1. Chofoza **Tento Tebuningi** → **Susa Ecenjini**
2. Sesha bese ukhetsa licembu (nemacembu) lofuna kususa bantfu kulo
3. Ungakhetsa emacembu lamanyenti
4. Chofoza **Susa Emacenjini**

:::info
Lesento sisusa kuphela bantfu emacenjini lakhonjisiwe. Asisisuli imininingwane yebo njengebantfu.
:::

### Susa Bantfu

Susa ngalokuphelele bantfu labakhetsiwe echibini lakho lebandla:

1. Chofoza **Tento Tebuningi** → **Susa**
2. Buyeketa luhlu lwebantfu labatakususwa
3. Bhala **DELETE** ensimini yekucinisekisa
4. Chofoza **Cinisekisa Kususa**

:::danger
Kususa bantfu kuphelele futsi akukwati kuhlehliswa. Loku kutakususa yonkhe imininingwane yabo, lekufaka:
- Imininingwane yemuntfu
- Bulunga bemacembu
- Emarekhodi ekufika
- Umlandvo wetipho
- Kutfunyelwa kwemafomu

Sebentisa lesento kuphela nangabe ucinisekile ngalokuphelele kutsi ufuna kususa lababantfu ohlelweni lwakho.
:::

## Imiphumela Yekuhlela Ngebuningi

Ngemuva kwekucedza sento sebuningi, utawubona simiso lesikhombisa:

- **Sonkhe lokukhetsiwe** -- Bangaki bantfu lebebafakwe kulesento
- **Lokubuyeketwe ngempumelelo** -- Emarekhodi langaki lashintjiwe
- **Kwehlulekile** -- Nome ngu emarekhodi langakwazanga kubuyeketwa (uma kukhona)
- **Akushintjwanga** -- Emarekhodi langakadzingi kushintjwa (sibonelo, nawusebentisa simo se-"buyeketa lokungenalutfo kuphela")

Nangabe kukhona lokubuyeketwa lokwehlulekile, utawubona imininingwane yeliphutsa lechaza kutsi kungani.

## Tento Letinhle Kakhulu

- **Cala ncane** -- Hlola tento tebuningi kumarekhodi lambalwa kucala kute ucinisekise kutsi wenta lushintjo lolufanele
- **Sebentisa imikhawulo** -- Cwaninga luhlu lwakho ngemikhawulo nobe kusesha nge-AI ngaphambi kwekukhetsa bantfu kute ucinisekise kutsi ubuyeketa bantfu lokufanele kuphela
- **Hlola kabili tikhetho** -- Buyeketa bantfu labakhetsiwe ngaphambi kwekusebentisa tishintjo tebuningi
- **Sebentisa simo se-"buyeketa lokungenalutfo kuphela"** -- Nawufuna kugcwalisa imininingwane leyintekelako ngaphandle kwekubhala phetulu imininingwane lekhona
- **Bhala tinshwana letikhulu** -- Gcina emanothi ngetishintjo tebuningi uma kudzingeka kutibhekisa kamuva
- **Hlanganyela nebutfu bakini** -- Tjela labanye labaphetse kutsi bawati nawenta tishintjo tebuningi letikhulu

## Tindlela Letivamile Tekusebentisa

### Kubuyeketa Emalunga Lamasha

Ngemuva kwesifundvo semalunga, buyeketa bonkhe labebekhona base baba simo se-Lilunga:

1. Sesha bantfu labebekhona kulesifundvo
2. Bakhetse bonkhe
3. Sebentisa **Tento Tebuningi** → **Beka Simo Selilunga** → **Lilunga**

### Kuhlela Emacembu Lamancane

Ngeta bantfu labanyenti ecenjini lelisha lelincane:

1. Sesha bantfu lofuna kubafaka ecenjini
2. Bakhetse
3. Sebentisa **Tento Tebuningi** → **Ngeta Ecenjini** bese ukhetsa licembu lelincane

### Kuhlanta Imininingwane

Gcwalisa simo semshado lesitekelako semacembu lashadile:

1. Khawula bantfu labashadile (usebentisa imininingwane yemndeni)
2. Khetsa labo labangenaso simo semshado
3. Sebentisa **Tento Tebuningi** → **Beka Simo Semshado** → **Ngishadile** → **Buyeketa lokungenalutfo kuphela**
