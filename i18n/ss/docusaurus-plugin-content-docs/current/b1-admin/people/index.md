# Bantfu

<div class="article-intro">

Sigaba se-**People** siyinhlitiyo yekuphatsa libandla lakho ku-B1 Admin. Lapha kuwukugcina luhlu lwakho lwelibandla -- kusesha emalunga, kwengeta bantfu labasha, kubuka nekuhlela emaphrofayela, kanye nekulandzela emakhaya. Nobe unelibandla lelincane nobe lelikhulu, kugcina emarekhodi ebantfu bakho ahlelekile kuyisisekelo lesisebentisa yonkhe insita lenye ku-B1.

</div>

## Yini Longayenta

Naku kubukwa lokujwayelekile kwetintfo letibalulekile letikhona esigabeni se-**People**:

1. **Sesha luhlu lwakho** -- Sebentisa i-quick search bar kuze utfole nome ngubani ngeligama, nobe usebentise emafilta laphakeme kanye nekusesha ngelulwimi lwemvelo lesetjentiswa yi-AI kuze utfole ngco lomudzingako. Bona [Kusesha Bantfu](./searching-people.md) kanye ne-[AI Search](./ai-search.md) kuze utfole imininingwane.

2. **Gcina kusesha njengemaLists** -- Gcina nome ngukuphi kusesha nobe kuhlunga njengeluhlu lolunelugama kuze ukwati kululayisha kabusha ngekuchofota kanye. Bona [Saved Lists](./lists.md).

3. **Buka i-Demographics** -- Bona kuhlukaniswa lokubonakalako kwelibandla lakho ngeminyaka, bulili, simo semshado, simo selilunga, kanye nesikhungo — bese uchofota nayiphi indzawo yeshathi kuze ugubhe ngekhatsi kwalabo bantfu. Bona [Demographics](./demographics.md).

4. **Engeta bantfu labasha** -- Engeta ngekushesha bantfu labasodvwa kuluhlu lwakho bese ugcwalisa imininingwane yemaphrofayela abo, imininingwane yekuxhumana, kanye nekuxhumana kwemakhaya. Bona [Kwengeta Bantfu](./adding-people.md) kuze utfole imiyalo lelandzelanako.

5. **Hlela bantfu labaningi kanyekanye (bulk edit)** -- Faka lokusha kubantfu labaningi ngesikhatsi sinye ngekushintja simo sabo selilunga, simo semshado, bulili, tinhlobo tekungabikho, nobe bulunga betincumo. Loku kongelela sikhatsi nangabe wenta lushintjo lolufanako kubantfu labaningi. Bona [Bulk Editing](./bulk-editing.md).

6. **Phrinta luhlu lwakho** -- Yenta luhlu lwemalunga lolungaphrintwa, lolihlelekile futsi lolihlanganiswe ngekwemakhaya, lolinetitfombe kanye nemininingwane yekuxhumana. Bona [Print Directory](./print-directory.md).

7. **Ngenisa nekukhipha imininingwane** -- Ngenisa imininingwane yemalunga kusuka kufayela le-CSV nobe fudukisa kusuka kwenye insita yekuphatsa libandla njenge-Breeze. Ungakhipha nobe ngunini luhlu lwakho lube yi-CSV. Bona [Importing Data](./importing-data.md) kanye ne-[Exporting Data](./exporting-data.md).

8. **Phatsa emaphrofayela** -- Buka futsi uhlele emaphrofayela lananiswe lemuntfu ngamunye, lokufaka imininingwane yekuxhumana, emalunga emakhaya, [bulunga betincumo](../groups/group-members.md), [umlandvo wekuya emihlanganweni](../attendance/tracking-attendance.md), [iminikelo](../donations/recording-donations.md), kanye [nemafomu lakhetsekile](../forms/creating-forms.md).

9. **Nika tikhundla netimvume** -- Lawula kutsi ngubani ethimbeni lakho longafinyelela tincenye letehlukene te-B1 Admin ngekunika tikhundla. Bona [Assigning Roles](./roles-permissions.md).

## Kucala

Kuze ufinyelele esigabeni se-**People**, chofota **People** ku-sidebar lengakwesancele ye-dashboard yakho ye-B1 Admin. Utawubona luhlu lwakho lolugcwele lwelibandla ngebha yekusesha etulu naluhlu lwemalunga ngentasi.

:::tip
Nangabe usukela i-B1 kwekucala, cala ngekungenisa imininingwane yemalunga akho lesele ikhona. Bese ubuyeketa futsi uhlanze emaphrofayela njengobe kudzingeka.
:::
