# Kuphatsa Emadivayisi

<div class="article-intro">

Likhasi le-Devices likuvumela kutsi uphatse emadivayisi e-ChurchAppsPlayer lahlanganiswe ne-akhawunti yakho. Lamadivayisi asetjentiswa ekusakazeni sicuku netekubonisa umbhalo webandla etikrinini endzaweni yakho.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Yiba nedivayisi le-ChurchAppsPlayer lesivuliwe futsi lihlanganiswe ne-inthanethi
- Bhala phasi likhodi lekuhlanganisa leliboniswa esikrinini sedivayisi
- Buka [tilungiselelo tephrofayela yakho](./managing-profile.md) nangabe ungakakenti

</div>

## Kubuka Emadivayisi Akho

Nawuvula likhasi le-Devices, utawubona thebula lelibalula onkhe emadivayisi labhalisiwe kanye nemininingwane lelandzelako:

- **Label** -- Libito lolinike idivayisi.
- **Registration Date** -- Ngesikhatsi idivayisi yahlanganiswa kucala ne-akhawunti yakho.
- **Last Active Date** -- Sikhatsi sagcina lapho idivayisi ihlangana khona ne-akhawunti yakho.

## Kwengeta Idivayisi Lensha

1. Chofota inkinobho **Add Device** phezulu ekhasini le-Devices.
2. Utawuceliwa kufaka **likhodi lekuhlanganisa**.
3. Tfola likhodi lekuhlanganisa leliboniswa kudivayisi ye-ChurchAppsPlayer lofuna kuyihlanganisa.
4. Faka likhodi bese uyachaza.
5. Idivayisi itawuvela ethebulini yemadivayisi akho nasingahlanganiswa ngephumelelo.

:::tip
Chaka kutsi idivayisi ye-ChurchAppsPlayer ivuliwe futsi ihlanganiswe ne-inthanethi ngaphambi kwekutama kuyihlanganisa. Likhodi lekuhlanganisa liboniswa esikrinini sedivayisi.
:::

## Kuhlela Libito Ledivayisi

1. Chofota **libito ledivayisi** ethebulini yemadivayisi.
2. Guculisa libito kube ngulelicacisako (sib. "Sanctuary Main Screen" nome "Lobby Display").
3. Gcina lutjintjo lwakho.

:::tip
Kusebentisa emabito lacacile futsi lachazako kwenta kube lula kwati kutsi ngiyiphi idivayisi, ikakhulu nawunemakhono lamaningi etikrinini lahlanganisiwe ebandleni lakho.
:::

## Kuphatsa Kutfola Kwemadivayisi

Sebentisa likhasi le-Devices kute ugcine umlandvo wekutsi ngutiphi tidivayisi letinemvumo yekufinyelela sicuku sekusakaza nekwakha kwebandla lakho. Nangabe idivayisi ingasasetjentiswa, ungayisusa ku-akhawunti yakho kute ugcine luhlu lwakho lwemadivayisi luhlanjululiwe futsi luphephile.

:::warning
Buyeketa njalo emadivayisi ahlanganisiwe kute uchaze kutsi ngulawaphela lasebentako futsi lanemvumo lahlanganiswe ne-akhawunti yakho. Susa nome ngutiphi tidivayisi letingasasetjentiswa kute ugcine kuphepha.
:::
