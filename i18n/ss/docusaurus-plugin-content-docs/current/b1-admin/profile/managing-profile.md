# Kuphatsa Iphrofayela Yakho

<div class="article-intro">

Likhasi le-Profile likuvumela kutsi uhlele tilungiselelo tekhawunti yakho lomuntfu. Iphrofayela yakho yabelwana kuto tonkhe emakhiciso e-ChurchApps, ngako nome ngutiphi tutjintjo lotenta lapha titawubonakala yonkhe indzawo losebentisa khona i-ChurchApps.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Kufanele ube ungenile ku-akhawunti yakho ye-ChurchApps
- Yiba nemeyili yakho lensha nome iphasiwedi lilungile nangabe uhlela kutitjintja

</div>

## Imininingwane Yephrofayela

Incenye lesetulu ekhasini le-Profile icuketse imininingwane yakho lesisekelo ye-akhawunti:

1. **Email** -- I-imeyili yakho lesetjentiswa ekungeneni nasekutfoleni tatiso.
2. **First Name** ne**Last Name** -- Libito lakho leliboniswako kuyo yonkhe i-ChurchApps.
3. **Password** -- Buyeketa iphasiwedi ye-akhawunti yakho.

Kute ubuyekete imininingwane yakho, hlela emabhokisi bese uchofota inkinobho **Save Changes**.

## Kutjintja Iphasiwedi Yakho

1. Yehla uye emabhokisini ephasiwedi ekhasini le-Profile.
2. Faka iphasiwedi yakho lensha ebhokisini **Password**.
3. Phindze uyifake ebhokisini **Confirm Password**.
4. Chofota **Save Changes**.

:::info
Iphasiwedi yakho kufanele ibengenani tinhlamvu letisi-8 kubudze. Chaka kutsi iphasiwedi kanye nesichaziso silingana ngalokugcwele.
:::

## Ema-akhawunti Lahlanganisiwe

Incenye ye-**Linked Accounts** ikhombisa noma ngutiphi tinsita tangaphandle letihlanganiswe ne-akhawunti yakho ye-ChurchApps. Ungabona kutsi ngutiphi ema-akhawunti lahlanganisiwe futsi uphatse loko kuhlanganiswa kusuka kule ncenye.

## Tintsandvo Tethimu

I-ChurchApps isekela tindlela letimbili tekubonisa: le-light ne-dark:

1. Tfola sisketshi se-**Theme** ekhasini lakho le-Profile.
2. Shintjanisa emkhatsini we-**Light** ne-**Dark** mode.
3. Sici sitawubuyeketa ngco kute sibonakalise lokukhetsile kwakho.

Intsandvo yakho yethimu igcinwa ku-akhawunti yakho futsi itawusebenta kuto tonkhe tikhatsi tawe.

## Kususwa Kwe-akhawunti

Ngentasi ekhasini le-Profile, utawutfola sikhetho sekususa i-akhawunti yakho ngalokuphelele. Lesento sisusa i-akhawunti yakho kanye nedatha lehlobene nayo futsi asikwati kuguculwa.

:::warning
Kususwa kwe-akhawunti kuphelele futsi akukwati kuguculwa. Chaka kutsi ufuna ngalokweqiniso kususa i-akhawunti yakho ngaphambi kwekuchubeka.
:::

:::info
Nangabe ungu-**Domain Admin**, chaka kutsi udlulisela imisebenti yekuphatsa kumuntfu lomunye ngaphambi kwekususa i-akhawunti yakho. Buka [Roles & Permissions](../settings/roles-permissions.md) kute utfole imininingwane ngekuphatsa tindzima tekuphatsa.
:::

:::info
Kuhlela iphrofayela kuvaliwe ku-demo mode. Nangabe usebentisa i-akhawunti ye-demo, ngeke ukwati kwenta tutjintjo etilungiselelweni takho tephrofayela.
:::
