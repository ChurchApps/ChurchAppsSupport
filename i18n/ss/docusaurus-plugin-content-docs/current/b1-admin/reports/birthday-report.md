# Umbiko Wemalanga Ekutalwa

<div class="article-intro">

Umbiko Wemalanga Ekutalwa ukusita kuhlala uhlangene nelibandla lakho ngekuveta emalunga lanemalanga ekutalwa lasetako. Wusebentise kuhlela imigubho, kutfumela emakhadi, nome kwenta tatiso ngesikhatsi setinsita.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- Cinisekisa kutsi emalanga ekutalwa afakiwe kuphrofayela yemuntfu ngamunye. Buka [Kwengeta Bantfu](../people/adding-people.md) kutfola imininingwane.
- Udzinga [timvumo](../settings/roles-permissions.md) letifanele kufinyelela imibiko

</div>

## Kucalisa Umbiko

1. Hamba uye ku-**Reports** kusuka kumenu lomkhulu.
2. Chofota ikhadi le-**Birthday Report**.
3. Hlela **luhla lwelusuku** ngekusebentisa tihlungo telusuku kuchaza kutsi ngutuphi emalanga ekutalwa lofuna kuwabona.
4. Umbiko utawuveta luhlu lwemalunga lanemalanga ekutalwa kuloluhla.

## Loko Umbiko Lokuvetako

Umbiko Wemalanga Ekutalwa uveta:

- **Ligama lelilunga** -- Ligama lemuntfu kusuka ku-database yebandla lakho.
- **Lilanga lekutalwa** -- Lusuku lwelilanga lakhe lekutalwa.

Imiphumela ibalwa ngekhatsi kwaluhla lwelusuku lokhetsile, kwenta kube lula kubona kutsi ngubani lonelilanga lekutalwa lelisetako.

## Kusetjentiswa Lokusemsebentini

- **Tatiso teviki** -- Calisa umbiko yeviki lelisetako futsi ufake emalanga ekutalwa kuphephetheni lakho nome tatiso.
- **Emakhadi elilanga lekutalwa** -- Yakha luhlu lwenyanga futsi lichembu lakho litfumele emakhadi elilanga lekutalwa lakhetsekile.
- **Baholi bemacembu lamancane** -- Baholi bemacembu bangacalisa umbiko kukhumbula emalanga ekutalwa emalunga elicembu labo.
- **Imigubho yetigaba letibalulekile** -- Cwayisa tikhatsi letitsite kuhlela imigubho yemalanga ekutalwa lamakhulu.

:::tip
Calisa lombiko ekucaleni kwenyanga ngayinye kuhlela kusengaphambili. Ungahlela luhla lwelusuku kuze lugcwale yonkhe inyanga kuze ube nesikhatsi sekulungisa emakhadi nome tatiso.
:::

:::info
Mininingwane yemalanga ekutalwa ivela endzaweni yelilanga lekutalwa kuphrofayela yemuntfu ngamunye. Cinisekisa kutsi [emarekhodi emalunga](../people/adding-people.md) alandzelele sikhatsi kuze utfole imiphumela lecondzile.
:::
