# Sifinyeto Semnikelo

<div class="article-intro">

Umbiko we-Sifinyeto Semnikelo ukuniketa sitfombe lesicacile semsebenti webandla lakho weminikelo. Wusebentise kulandzelela tinani letiphelele teminikelo, kucwaya emaphetheni eminikelo, kanye nekulungiselela imibiko yetimali yechembu lakho lebuholi.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- Cinisekisa kutsi iminikelo [iyabhalwa ngalokuhambisanako](../donations/recording-donations.md) enhlelweni yakho
- Udzinga [timvumo](../settings/roles-permissions.md) letifanele kufinyelela imibiko yetimali

</div>

## Kucalisa Umbiko

1. Hamba uye ku-**Reports** kusuka kumenu lomkhulu.
2. Chofota ikhadi le-**Donation Summary**.
3. Hlela **luhla lwelusuku** kuchaza sikhatsi sembiko.
4. Faka tihlungo letengetiwe njenge **fund** nome letinye tinsika kunciphisa imiphumela.
5. Umbiko utawuveta sifinyeto seminikelo yesikhatsi lesikhetsiwe.

## Loko Umbiko Lokuvetako

Umbiko we-Donation Summary uhlanganisa:

- **Tinani letiphelele teminikelo** -- Linani leliphelele leminikelo yesikhatsi lesikhetsiwe.
- **Linani labaniki beminikelo** -- Linani labaniki beminikelo labehlukene labenta umnikelo.
- **Tinani ngekhaya lemali (fund)** -- Kuhlukaniswa kweminikelo etikhaya letehlukene temali, nangabe kusebenta.

## Kusetjentiswa Lokusemsebentini

- **Kuhlela tetimali** -- Buyeketa emaphetheni eminikelo kubikezela imali letako futsi uhlele ibhajethi yakho.
- **Imibiko yebhodi** -- Yakha tifinyeto temihlangano yebhodi nome yebuholi lejwayelekile nemininingwane leminikelo lecacile.
- **Kubuyeketwa kwekuphela komnyaka** -- Calisa umbiko wemnyaka lophelele kulungiselela tifinyeto temimali temnyaka.
- **Kulandzelela imikhankaso** -- Cwayisa nge-fund kulandzelela intfutfuko yemikhankaso yeminikelo lekhetsekile nome emaprojekthi ekwakha.
- **Kuhlaziywa kwesizini** -- Catsanisa iminikelo kuletikhatsi letehlukene kucondza emaphetheni esizini.

:::tip
Calisa lombiko inyanga ngayinye kuhlala watiswa ngemphilo yemali yebandla lakho. Kucatsanisa mininingwane yenyanga ngenyanga kukusita kubona emaphetheni kucala.
:::

:::warning
Mininingwane yeminikelo ivela emnikelweni lobhaliwe enhlelweni yakho. Cinisekisa kutsi iminikelo ifakwa ngalokuhambisanako -- kungaba ngekusebentisa umnikelo we-online, inchubo yesheke, nome kufaka ngesandla -- kutfola imibiko lecondzile kakhulu. Buka [Kubhala Iminikelo](../donations/recording-donations.md) kutfola imininingwane.
:::
