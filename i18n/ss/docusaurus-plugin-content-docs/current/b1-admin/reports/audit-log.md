# Irekhodi Lekuhlolwa

<div class="article-intro">

Irekhodi lekuhlolwa lilandzelela tonkhe tento letibalulekile kanye netishintjo endleleni yakho yekuphatsa libandla. Lisebentise kubuyeketa umsebenti wekungena, kulandzelela kutsi ngubani lowente tishintjo emarekhodini ebantfu, kucwaya tibuyeketo temvumo, kanye nekugcina umtfwalo etinhlolweni kuwo wonkhe emcembu wakho.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- I-akhawundi ye-B1 Admin lenekufinyelela kwe-server admin
- Hamba uye ku-**Settings** kutfola i-Audit Log

</div>

## Kubuka Irekhodi Lekuhlolwa

1. Hamba uye ku-**Settings** ku-B1 Admin.
2. Khetsa **Audit Log**.
3. Irekhodi liveta tingeniso letisandza kwenteka kuthebula lelinetinsika letilandzelako:
   - **Date** -- Ngubemi lapho sento sentekile khona.
   - **Category** -- Luhlobo lwesento (lunemibala lehlukanisiwe kufundza ngekushesha).
   - **Action** -- Loko lokwentiwe (sib. dala, buyeketa, susa, login_success).
   - **Entity** -- Luhlobo kanye ne-ID yerekhodi lelitsintfekile.
   - **IP Address** -- I-IP Address yemsebentisi lowente sento.
   - **Details** -- Sifinyeto setishintjo letitsite letentiwe.

## Kucwayisa Irekhodi

Sebentisa tihlungo letisetulu kwelikhasi kuncipisa imiphumela:

- **Category** -- Cwayisa ngeluhlobo lwesento:
  - **All Categories** -- Veta konkhe.
  - **Login** -- Kungena lokuphumelele nalokungaphumelelanga.
  - **People** -- Kudala, kubuyeketa, nome kususa marekhodi ebantfu.
  - **Permissions** -- Kunikwa nekususwa kwemvumo.
  - **Donations** -- Tishintjo temarekhodi eminikelo.
  - **Groups** -- Tento tekuphatsa emacembu.
  - **Forms** -- Umsebenti wekufakwa kwemafomu.
  - **Settings** -- Tishintjo tekuhlelwa.
- **Start Date** -- Veta tingeniso kusuka kuleli langa kuya embili.
- **End Date** -- Veta tingeniso kuze kufike kuleli langa.

Chofota **Search** ngemva kwekuhlela tihlungo takho kubuyeketa imiphumela.

## Kucondza Emakategori

Ikategori ngayinye inemibala lehlukanisiwe kutsi ikhonjiswe ngekushesha:

- **Login** -- Chipu lelimbala luhlata luphuzi. Lilandzelela kungena lokuphumelele nalokungaphumelelanga.
- **People** -- Chipu lelimbala luphephulini. Lilandzelela kudalwa, kubuyeketwa, nekususwa kwemarekhodi ebantfu.
- **Permissions** -- Chipu lelimbala bomvu. Lilandzelela nakunikwa nome kususwa emalungelo ekufinyelela.
- **Donations** -- Chipu lelimbala luhlata. Lilandzelela tishintjo temarekhodi eminikelo.
- **Groups** -- Chipu lelimbala mpunga. Lilandzelela imisebenti yekuphatsa emacembu.
- **Forms** -- Chipu lelimbala orenji. Lilandzelela umsebenti wekufakwa kwemafomu.
- **Settings** -- Chipu lelimbala luphuzi. Lilandzelela tishintjo tekuhlelwa.

## Kulayisha Irekhodi Ngephandle

Nangabe tingeniso teIrekhodi tiveziwe, inkinobho ye-**CSV download** iyavela. Yichofote kulayisha imiphumela lehlungiwe yamanje ihambe iye ku-spreadsheet kubuyeketwa ngephandle nome kugcinwa kwemarekhodi.

## Kwehlukanisa Emakhasi

Sebentisa tilawuli tekwehlukanisa emakhasi ngentasi kwethebula kuhamba emkhatsini wemiphumela. Ungakwati kuveta tingeniso letingu-25, 50, nome 100 ngekhasi ngalinye.

:::info
Tingeniso teIrekhodi Lekuhlolwa tigcinwa ngalokutentekelako umnyaka munye. Tingeniso letidzala kunemalanga langu-365 tiyasuswa kugcina inhlelo isebenta kahle.
:::

:::tip
Buyeketa irekhodi lekuhlolwa njalonjalo, ikakhulu ngemva kwekwemukela emalunga lamasha emcembu nome kwenta tishintjo letibalulekile tekuhlelwa. Kusita kukhomba umsebenti longakalindzelekile kucala.
:::
