# Imibiko Yekubakhona

<div class="article-intro">

B1 Admin iniketa imibiko lemitsatfu yekubakhona kukusita kucondza kutsi bantfu batibandzakanya njani netinsita takho kanye nemacembu. Umbiko ngamunye uniketa umbono lohlukile ngemininingwane yakho yekubakhona, kusuka kumaphetheni lasetulu kuye ekuhlukanisweni kwelilanga nelilanga.

</div>

<div class="prereqs">
<h4>Ngembikwe Kucala</h4>

- Cinisekisa kutsi kubakhona [kuyalandzelelwa ngalokuhambisanako](../attendance/tracking-attendance.md) tinsita takho kanye nemacembu
- Cinisekisa kutsi [emacembu](../groups/creating-groups.md) netinsita takho kuhlelwe ku-B1 Admin
- Udzinga [timvumo](../settings/roles-permissions.md) letifanele kufinyelela imibiko

</div>

## Iphetheni Yekubakhona

Umbiko we-Attendance Trend uveta kutsi kubakhona kushintja njani ngesikhatsi tinsita takho.

1. Hamba uye ku-**Reports** bese uchofota ikhadi le-**Attendance Trend**.
2. Hlela **luhla lwelusuku** kuchaza sikhatsi.
3. Khetsa **insita** nome **licembu** kuhlukanisa imiphumela.
4. Umbiko uveta umgca weliphetheni loveta tibalo tekubakhona ngesikhatsi lesikhetsiwe.

Lombiko usita ekuboneni emaphetheni njengekuwa kwesizini, emaphetheni ekukhula, nome umtselela wemicimbi lekhetsekile.

## Kubakhona Kwelicembu

Umbiko we-Group Attendance uveta tibalo letiphelele tekubakhona ngelicembu kuluhla lwelusuku lolukhetsiwe.

1. Hamba uye ku-**Reports** bese uchofota ikhadi le-**Group Attendance**.
2. Hlela **luhla lwelusuku** lwembiko.
3. Khetsa **emacembu** lofuna kuwabuyeketa.
4. Umbiko uveta tibalo letiphelele tekubakhona kulicembu ngalinye.

Sebentisa lombiko kucatsanisa kubakhona emkhatsini wemacembu futsi ukhombe kutsi ngutuphi emacembu lakhulako nome ladzinga kunakwa.

## Kubakhona Kwelicembu Lilanga Ngelilanga

Umbiko we-Daily Group Attendance uniketa kuhlukaniswa kwelilanga nelilanga kwemininingwane yekubakhona yemacembu akho.

1. Hamba uye ku-**Reports** bese uchofota ikhadi le-**Daily Group Attendance**.
2. Hlela **luhla lwelusuku** lwembiko.
3. Khetsa **emacembu** lofuna kuwabuyeketa.
4. Umbiko uveta tibalo tekubakhona lilanga ngalinye ngekhatsi kwaloluhla.

Lombiko ukuniketa imininingwane lecolisisiwe, lokusita kucondza kwehlukana kweviki ngeviki nome kukhomba emalanga latsite lanekubakhona lokungajwayelekile kuphakama nome kuphansi.

:::tip
Sebentisa umbiko we-Attendance Trend kubuka lokusetulu kanye nombiko we-Daily Group Attendance nawudzinga kungena etikwemalanga latsite.
:::

## Kusetjentiswa Lokusemsebentini

- **Kuhlela** -- Sebentisa emaphetheni ekubakhona kuhlela tihlalo, basebenti, netinsita tetinsita letisetako.
- **Kufinyelela Bantfu** -- Khomba emaphetheni ekwehla kwekubakhona kucala kuze ukwati kulandzelela emalunga.
- **Imibiko yebhodi** -- Faka mininingwane yekubakhona kuyo yonkhe imibiko yakho yebuholi kukhombisa imphilo yemkhonto.
- **Kuhlolwa kwemicimbi** -- Catsanisa kubakhona ngembikwe nangemva kwemicimbi lekhetsekile kulinganisa umtselela wayo.

:::warning
Mininingwane yekubakhona ibhalwa ngekusebentisa inchubo yakho yekungena kwelicembu netinsita. Nangabe kubakhona akulandzelelwa ngalokuhambisanako, imibiko yakho ngeke ikhombise ngalokucondzile kubandzakanyeka lokwenteka. Buka [Kulandzelela Kubakhona](../attendance/tracking-attendance.md) kutfola imiyalo yekuhlela.
:::
