# Imibiko

<div class="article-intro">

Likhasi le-Reports likuniketa kufinyelela ngekushesha kumibiko lesheyakhelwe lekusita kucondza mininingwane yebandla lakho. Sebentisa lemibiko kulandzelela emaphetheni ekubakhona, kucwaya umsebenti weminikelo, kanye nekuhlala uhlangene nelibandla lakho ngekugujwa kwemalanga ekutalwa.

</div>

## Imibiko Lekhona

Likhasi le-Reports liveta emakhadi embiko wembiko ngawunye lokhona. Chofota nome ngiliphi ikhadi kuvula umbiko kanye netihlungo taso.

- **[Umbiko Wemalanga Ekutalwa](./birthday-report)** -- Buka emalanga ekutalwa lasetako emalunga elibandla lakho.
- **[Iphetheni Yekubakhona](./attendance-reports)** -- Bona kutsi emaphetheni ekubakhona ashintja njani ngesikhatsi kuwo onkhe tinsita takho.
- **[Kubakhona Kwelicembu](./attendance-reports)** -- Buka tinani letiphelele tekubakhona ngelicembu kuluhla lwelusuku lolukhetsiwe.
- **[Kubakhona Kwelicembu Lilanga Ngelilanga](./attendance-reports)** -- Tfola kuhlukaniswa kwelilanga nelilanga kwemininingwane yekubakhona kwelicembu.
- **[Sifinyeto Semnikelo](./donation-summary)** -- Buyeketa tinani letiphelele teminikelo kanye nemaphetheni ekuhlela timali.
- **[Irekhodi Lekuhlolwa](./audit-log)** -- Landzelela tonkhe tishintjo netento kuyo yonkhe inhlelo yakho yekuphatsa libandla (litfolakala ngentasi kwe-Settings > Audit Log).

## Kusebentisa Imibiko

1. Hamba uye ku-**Reports** kusuka kumenu lomkhulu.
2. Chofota **ikhadi lembiko** kuze liyivule.
3. Sebentisa **tihlungo** letisetulu kombiko ngamunye kunciphisa mininingwane ngeluhla lwelusuku, licembu, insita, nome tinye tinsika.
4. Buyeketa imiphumela leyetiwe embikweni.

:::tip
Calisa imibiko njalonjalo kuhlala uphambili emaphetheni ekubakhona, emalanga ekutalwa lasetako, kanye nemaphetheni eminikelo. Lemininingwane ikusita kuhlela imikhonto nekufinyelela bantfu ngalokusebentako kakhulu.
:::

:::info
Mininingwane yembiko incike ekufakweni kwemininingwane ngalokuhambisanako. Cinisekisa kutsi lichembu lakho [liyalandzelela kubakhona](../attendance/tracking-attendance.md), [libhale iminikelo](../donations/recording-donations.md), futsi [ligcine emarekhodi ebantfu](../people/adding-people.md) alandzelele sikhatsi kutfola imiphumela lecondzile kakhulu.
:::
