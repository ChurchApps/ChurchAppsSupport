# Simo Lesijikelele Se-B1 Church Checkin

<div class="article-intro">

I-B1 Church Checkin yi-app ye-kiosk letisebentela yona ku-tablet ye-Android. Ivumela emalunga ebandla kutsi afune umndeni wawo, abele tingane emacenjini nobe emakilasini, futsi aphrinte emalebula emagama netincwadzi tekulanda kubatali -- konkhe ngaphandle kwekudzinga ivolontiya lesedeskini.

</div>

## Kusebenta Njani

Beka ithebulethi ye-Android ngaphandle noma endzaweni yekwemukela. Emalunga ayasondzela, afune umndeni wawo ngenombolo yelucingo nobe ngesibongo, abuyekete umndeni wawo, akhetse kutsi ngutiphi emacembu lomuntfu ngamunye kufanele aye kuwo, bese acindzetela **Check-in**. I-app iyaphrinta emalebula emagama netincwadzi tekulanda ngekutentela, bese isetulwa kabusha kutekulindzela umndeni lolandzelako.

## Timphawu Letibalulekile

- **Kufuna lilunga ngekutentelwa** -- Funa ngenombolo yelucingo nobe ngesibongo kute utfole umndeni wakho ngekushesha.
- **Kubuyeketa umndeni** -- Bona onkhe emalunga emndeni anetitfombe kanye nekwabelwa kwemacembu kwamanje ngekulunguza kanye.
- **Kwabela emacembu** -- Yabela lilunga lemndeni ngalinye kulikilasi nobe licembu lelifanele sikhatsi senkhonzo lesikhetsiwe.
- **Kubhalisa tivakashi** -- Engeta sivakashi emndenini wakho ngco lapho ngeligama lekucala kanye nesibongo.
- **Kubhalisa tivakashi nge-QR** -- Khombisa ikhodi ye-QR letivakashi tingayikalisa kute [titibhalise](../b1-church/checkin/guest-registration) ngemafoni ato ngembi kwekubhalisa.
- **Kuphrinta emalebula** -- Phrinta emalebula emagama etinganeni netincwadzi tekulanda kubatali letinemakhodi lekuphepha lahlukene.
- **Kwesekwa kweprinta lehlanganiswe nenethiwekhi** -- Tfola futsi uhlele emaprinta lahlanganiswe nenethiwekhi ngco kusuka ku-app.
- **Kungena ngekutentela** -- Ngemuva kwekungena kwekucala, i-app iyakhumbula imininingwane yakho bese iya ngco eskrinini setinkhonzo ekuvuleni lokutako.
- **Kwesekwa kwemabandla lamanyenti** -- Nangabe i-akhawundi yakho ihlanganiswe nemabandla langetulu kwelinye, ungakhetsa kutsi ngutsini lofuna kubhalisela kulo.
- **Ibulogo yebandla** -- I-app ikhombisa ilogo yebandla lakho esihlokweni nayisiye ikhetsiwe.
