# Kuhlela Iprinta

<div class="article-intro">

I-B1 Church Checkin iyakhona kuphrinta emalebula emagama netincwadzi tekulanda kubatali kuya eprinteni yemalebula lehlanganiswe nenethiwekhi. Kuhlela iprinta kungalokutsandvekako -- ungasebentisa i-app ngaphandle kweprinta kuphela kute ugcine umlandvo wekubakhona.

</div>

## Kufinyelela Kuhlelweni Kweprinta

Ungavula skrini sekuhlela iprinta nobe ngasiphi sikhatsi kusebentisa imenyu ye-**Kuhlela Kwemlawuli**:

1. Cindzetela **ilogo yebandla** esihlokweni **kasikhombisa** ngekushesha (ngekhatsi kwemasekhondi lamabili).
2. Nangabe kukhona i-PIN ye-kiosk lehleliwe, faka i-PIN nawucelwa.
3. Kuskrini se-Kuhlela Kwemlawuli, cindzetela **Shintja Iprinta**.

Loku kuvula skrini sekuhlela iprinta lapho ungahlola khona emaprinta, ukhetse linye, bese wenta luphrinto lwekuhlola.

## Kuhlola Emaprinta

Nawuvula skrini sekuhlela iprinta, i-app iyahlola ngekutentela inethiwekhi yakho yalapha kute itfole emaprinta lahambisanako. Utawubona luphawu lwekulinda nemlayeto lotsi "Scanning for printers" njengoba kuhlolwa kuchubeka.

Nangabe kuhlola sekucedzile, utawubona luhlu lwemaprinta latfoliwe. Likhadi leprinta ngalinye likhombisa:

- Ligama **lemodeli yeprinta**
- **Ikheli le-IP** leprinta kunethiwekhi yakho

Kukhetso lekutsi **Kute Iprinta** kuhlala kukhona ngephansi kweluhlu.

:::warning
Ithebulethi neprinta kufanele tihlanganiswe kunethiwekhi yalapha lefanako kute kuhlola kutfole iprinta. Nangabe iprinta yakho ingaveli, cinisekisa kutsi tonkhe timbili tidivayisi tikunethiwekhi yeWi-Fi lefanako nekutsi iprinta ivulekile.
:::

## Kukhetsa Iprinta

1. Cindzetela iprinta lofuna kuyisebentisa. Likhadi leprinta lelikhetsiwe liyacaciswa ngephawu lekuhlola.
2. Kute uphrinte ngaphandle kweprinta yangempela (kutekuhlola nobe kwendlela yekubakhona kuphela), khetsa **Kute Iprinta**.

## Kuprinta Kwekuhlola

Ngembi kwekugcina kukhetsa kwakho, ungahlola kutsi luchumano lusebenta:

1. Khetsa iprinta eluhlwini.
2. Cindzetela inkinobho letsi **Test Print** ngaphansi kweskrini.
3. I-app itfumela lilebula lekuhlola eprinteni. Nangabe iprinta ihlanganisiwe kahle, likhasi lekuhlola litawuphrinteka.

Nawucindzetela **Test Print** kuphetse **Kute Iprinta** kukhetsiwe, i-app itawukhombisa sikhumbuto sekutsi akukho iprinta lekhetsiwe.

:::tip
Hlala wenta luphrinto lwekuhlola ngemuva kwekukhetsa iprinta kute ucinisekise luchumano ngembi kwenkhonzo lelandzelako.
:::

## Kugcina Kukhetsa Kwakho

Nawusanelisekile ngekukhetsa kwakho kweprinta:

1. Cindzetela inkinobho letsi **Save & Restart** ngaphansi kweskrini.
2. I-app iyagcina kukhetsa kwakho kweprinta bese iphindza icale kabusha kute isebentise kuhlelwa.

Kukhetsa kwakho kweprinta kuyakhunjulwa nangabe i-app icala kabusha. Awudzingi kukuhlela futsi ngaphandle kwekutsi ufuna kushintja emaprinta.

:::info
Nangabe uhambisa i-kiosk kulenye indzawo nobe ushintja kuhlelwa kwakho kwenethiwekhi, buyela ekuhleleni kweprinta kute uhlole emaprinta futsi bese wentula kukhetsa kwakho.
:::
