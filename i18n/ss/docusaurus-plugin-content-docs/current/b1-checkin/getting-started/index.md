# Kucala Nge-B1 Church Checkin

<div class="article-intro">

Kuhlela B1 Church Checkin kutsatsa imizuzu lembalwa nje. Lesigaba sikuhambisa esinyatselweni sekufaka i-app ku-tablet ye-Android, kungena nge-akhawundi yakho ye-ChurchApps, nekuhlela ngalokutsandvekako iprinta yemalebula lehlanganiswe nenethiwekhi.

</div>

## Loko Lokudzingekako

- **Ithebulethi ye-Android** -- Ithebulethi lene-10-inch isebenta kahle kakhulu ekusebentiseni kwe-kiosk.
- **Akhawundi ye-ChurchApps** -- Imeyili nelipasiwedi lokusebentisako ku-B1 Admin nobe kwenye imikhicito ye-ChurchApps.
- **Iprinta yemalebula lehlanganiswe nenethiwekhi** (ngalokutsandvekako) -- Yekuphrinta emalebula emagama netincwadzi tekulanda kubatali. I-app iyakhona kuhlola inethiwekhi yakho kute itfole emaprinta lahambisana nayo.

## Kufaka I-App

1. Vula i-**Google Play Store** ku-tablet yakho ye-Android.
2. Funa i-**B1 Church Checkin**.
3. Cindzetela **Install** bese ulinda kutsi kudawundwelwa kuchede.
4. Vula i-app. Utawubona skrini se-splash njengoba iyacala.

## Kuhlela Kwekucala

Nawuvula i-app kwekucala, utawuchubeka etinyatselweni letintsatfu:

1. **[Ngena](./logging-in)** -- Faka imeyili nelipasiwedi yakho kute ucinisekiswe.
2. **[Khetsa libandla lakho](./logging-in)** -- Nangabe i-akhawundi yakho ihlanganiswe nemabandla lamanyenti, khetsa kutsi ngutsini lelizakusetjentiswa yile kiosk.
3. **[Hlela iprinta](./printer-setup)** -- Hlela iprinta lehlanganiswe nenethiwekhi yekuphrinta emalebula, nobe weqe lesinyatselo nangabe ungawadzingi emalebula.

Ngemuva kwekungena kwekucala, i-app iyagcina imininingwane yakho yekungena kanye nekukhetsa kwakho kwebandla. Ekuvuleni lokutako, itawweqa iye ngco eskrinini setinkhonzo ngekutentela.

:::info
I-app iyahlola tibuyekezo letisebenta nge-inthanethi njalo nayicala. Nangabe kukhona sibuyekezo lesitfolakalako, siyadawundwelwa futsi sisebentiswe ngekutentela ngembi kwekutsi ufike eskrinini sekungena.
:::

:::tip
Cinisekisa kutsi umlawuli webandla lakho usewahlele tinkhonzo, tikhatsi tenkhonzo, nemacembu ku-B1 Admin ngembi kwekuhlela i-kiosk. Bona [umhlahlandlela wekuhlela kubakhona](../../b1-admin/attendance/setup.md) kute utfole imininingwane.
:::
