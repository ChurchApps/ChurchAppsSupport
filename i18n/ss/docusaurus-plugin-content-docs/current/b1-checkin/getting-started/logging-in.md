# Kungena

<div class="article-intro">

Ngembi kutsi i-kiosk isetjentiswe kubhalisa, kudzingeka ungene nge-akhawundi ye-ChurchApps lenemvumo yekufinyelela ebandleni lakho. Leli khasi likhava luchungechunge lwekungena kwekucala, kukhetsa libandla, kutiphatsa kwekungena ngekutentela, kanye nendlela yekuphuma.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- Faka i-B1 Church Checkin ku-tablet yakho ye-Android (bona [Kucala](./index.md))
- Yiba nemeyili nelipasiwedi ye-akhawundi yakho ye-ChurchApps kulungele

</div>

## Kungena Kwekucala

1. Vula i-app ye-B1 Church Checkin. Ngemuva kweskrini se-splash, utawubona skrini sekungena lesinelogo ye-B1 Church Checkin.
2. Faka **likheli lakho lemeyili** ensimini yekucala.
3. Faka **lipasiwedi** lakho ensimini yesibili.
4. Cindzetela inkinobho letsi **Login**.

I-app iyahlola imininingwane yakho bese ikungenisa. Nangabe imeyili nobe lipasiwedi lingakalungi, utawubona umlayeto weliphutsa ngaphansi kweskrini.

:::info
Lena yimeyili nelipasiwedi lofanako lokusebentisako ku-B1 Admin nobe nakuwuphi umkhicito lomunye we-ChurchApps. Awudzingi i-akhawundi lehlukile ye-kiosk yekubhalisa.
:::

## Kukhetsa Libandla Lakho

Ngemuva kwekungena ngempumelelo, i-app iyalayisha luhlu lwemabandla lahlanganiswe ne-akhawundi yakho. Libandla ngalinye likhonjiswa njengekhadi lelikhombisa ligama nelogo yebandla (nangabe seyihleliwe ku-B1 Admin).

- Nangabe i-akhawundi yakho ihlanganiswe **nebandla linye kuphela**, utawubuye ubone skrini sekukhetsa. Cindzetela libandla lakho kute uchubeke.
- Nangabe i-akhawundi yakho ihlanganiswe **nemabandla lamanyenti**, cindzetela libandla lofuna kutsi lesi si-kiosk sisisebentele sona.

Nawukhetsa libandla, i-app iyalayisha tinhlelo tebulogo netekubukeka taloo bandla. Ilogo yebandla lakho itawuvela esihlokweni kuto tonkhe tiskrini letilandzelako.

## Kungena Ngekutentela Nawuphindze Uvakashe

I-app iyagcina imeyili yakho, lipasiwedi, kanye nebandla lokhetsile ngemuva kwekungena kwekucala ngempumelelo. Ekuvuleni lokutako:

- I-app iyangena ngekutentela isebentisa imininingwane yakho legciniwe.
- Nangabe wakhetsa libandla nangaphambili, iyabuyisa loko kukhetsa bese ikuyisa ngco **eskrinini setinkhonzo**.
- Nangabe imininingwane legciniwe ingasakuhambisani, i-app iyabuyela eskrinini sekungena kute ukhone kungena kabusha.

:::tip
Njengobe i-kiosk ingena ngekutentela, udzinga kufaka imininingwane kanye kuphela ngesikhatsi sekuhlela kwekucala. Ngemuva kwaloko, ithebulethi ingavulwa nje itfole ilungele kusebenta.
:::

## Kuhlela Kwemlawuli

Kuhlela kwemlawuli kufihlwe ngephutsumo kute emalunga ebandla lasebentisa i-kiosk angeke ashintje kuhlelwa ngephutsa. Kute ufinyelele kuko:

1. Cindzetela **ilogo yebandla** esihlokweni **kasikhombisa** ngekushesha (ngekhatsi kwemasekhondi lamabili).
2. Nangabe kukhona i-PIN ye-kiosk lehleliwe, faka i-PIN nawucelwa.
3. Skrini se-**Kuhlela Kwemlawuli** sitawuvula sinaleto tikhetso:
   - **Shintja Inkhonzo** — shintja sikhatsi senkhonzo lesikhetsiwe
   - **Shintja Iprinta** — vula [kuhlelwa kweprinta](./printer-setup) kute uhlole bese ukhetsa iprinta
   - **Hlela/Shintja i-PIN** — hlela nobe wentule i-PIN ye-kiosk levikela kufinyelela kwemlawuli
   - **Phuma** — phuma bese ubuyela eskrinini sekungena

:::warning
Kuphuma kususa yonkhe imininingwane legciniwe kanye nekukhetsa kwebandla. Utawudzinga kubuye ufake imeyili nelipasiwedi yakho kute usebentise i-kiosk futsi. Phuma kuphela nawudzinga kushintja emakhawundi nobe kuhlela kabusha i-kiosk.
:::
