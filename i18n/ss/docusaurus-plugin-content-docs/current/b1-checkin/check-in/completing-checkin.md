# Kucedza Kubhalisa

<div class="article-intro">

Nase ubuyeketile umndeni wakho futsi wenta konkhe kwabelwa kwemacembu lokudzingekile, usulungele kuchubeka kubhalisa. Lesi ngunyatselo wekugcina eluchungechungeni lwe-kiosk -- i-app iyatfumela umlandvo wekubakhona, iphrinte emalebula, bese isetulwa kabusha kutekulindzela umndeni lolandzelako.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- [Buyeketa umndeni wakho](./household-review) eskrinini sekubuyeketa umndeni
- [Yabela emacembu](./group-assignment) kunobe ngumuphi umuntfu wemndeni ledzinga kubhaliswa kulikilasi nobe kuluhlelo lolutsite
- Ngalokutsandvekako [engeta tivakashi](./adding-guests) letivakashile nemndeni wakho

</div>

## Kubhalisa Njani

1. Kusukela kuskrini sekubuyeketa umndeni, cindzetela inkinobho letsi **Check-in** ngaphansi kweskrini.
2. I-app iyatfumela imininingwane yekubakhona kusevha bese ikhombisa **skrini wempumelelo** lonelphawu leliluhlata kanye nemlayeto wekwemukela.

Yikho konkhe lokudzingekako. Kubakhona kwemndeni wakho sekubhaliwe.

## Emakamelo Lagcwele Netilinganiso Tebavolontiya

Nangabe libandla lakho lisewahlele [emandla ekuphepha](../../b1-admin/attendance/checkin-safety) ekamelweni lawo, isevha iyawahlola ngembi kwekugcina:

- Nangabe likamelo lelikhetsiwe **ligcwele nobe livaliwe**, kubhalisa akwenteki futsi i-app iyalibita likamelo kute ukhone kukhetsa lelinye.
- Nangabe likamelo letingane **lisilela ngebavolontiya** kutilinganiso talo, i-app ingakhombisa sikhumbuto lesingacinisekiswa ngumsebenti webandla kute kuchubeke, nobe ivimbe kubhalisa ngalokuphelele -- konkhe kuya ngendlela libandla lakho lelihlele ngayo kuphocelelwa kwetilinganiso.

## Kuphrinta Emalebula

Nangabe kunayo iprinta lehlanganiswe nenethiwekhi, i-app iyaphrinta emalebula ngekutentela ngemuva kwekubhalisa:

- **Emalebula emagama** aphrintwa umuntfu ngamunye lowabelwe licembu lelinesethingi letsi **Print Nametag** levuliwe. Emalebula emagama afaka ligama lemuntfu, kwabelwa kwakhe kwelicembu, kanye nemininingwane yekutingiwa nobe yemibiko nangabe ikhona.
- **Tincwadzi tekulanda kubatali** taphrintwa nangabe nabani lobhalisiwe ukulicembu lelinesethingi letsi **Parent Pickup** levuliwe. Incwadzi yekulanda ibala tingane, kwabelwa kwato kwemacembu, kanye **nekhodi yekuphepha yetinhlamvu letine** lengakafani nalenye.

:::info
Ikhodi lefanako yekuphepha ivela kokubili elebulini legama lengane naencwadzini yekulanda kubatali. Ngesikhatsi sekulanda, bavolontiya bayahlanganisa emakhodi kute bacinisekise kutsi ngumuntfu lofanele lolanda ingane ngayinye.
:::

Ikhodi yekuphepha yakhiwa isha kubhalisa ngakunye futsi isebentisa tinhlamvu letimayelana nemidvumo kuphela (tinhlamvu tesikwehla tiyekwe kute kungabi khona emagama langakafanele akhiwe).

:::warning
Nangabe emalebula angaphrinti, vula Kuhlelwa Kwemlawuli ngekucindzetela **ilogo yebandla** kasikhombisa, bese ucindzetele **Shintja Iprinta** kute uhlole luchumano lweprinta. Bona [Kuhlela Iprinta](../getting-started/printer-setup) kutetinyatselo tekulungisa.
:::

## Loko Lokwentekako Ngemuva Kwekubhalisa

- Nangabe kunayo iprinta lehleliwe, i-app iyaphrinta onkhe emalebula bese ibuyela ngekutentela **eskrinini sekufuna**, lulungele umndeni lolandzelako.
- Nangabe kute iprinta lehleliwe, skrini sempumelelo sikhonjiswa emasekhondi lambalwa bese sibuyela ngekutentela eskrinini sekufuna.

Awudzingi kucindzetela lutfo kute ubuyele eskrinini sekufuna -- i-app iyalulawula loku ngekutentela.

:::tip
I-app iyasetulwa kabusha ngalokuphelele ngemuva kwekubhalisa ngakunye, ngako akukho ngoti yekutsi umndeni munye ubone imininingwane yalomunye umndeni.
:::

## Loko Lokubhalwako

Nawucindzetela **Check-in**, i-app itfumela loku kulandzelako kusevha lilunga ngalinye lemndeni lelinekwabelwa kwelicembu:

- **Umuntfu** lobhaliswako
- **Inkhonzo** layiya kuyo
- **Sikhatsi senkhonzo** kanye **nelicembu** layabelwe kulo

Lolwati lubonakala ku-B1 Admin ngaphansi kwesigaba sekubakhona, lapho balawuli bebandla lakho bangabuka futsi balawule khona imilandvo yekubakhona. Bona [umhlahlandlela wekulawula kubhalisa](../../b1-admin/attendance/check-in.md) kutetinsayo.
