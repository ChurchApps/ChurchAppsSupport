# Kuphuma Kanye Nekuphepha Kwetingane

<div class="article-intro">

Kuphuma kuvala luchungechunge lwekubhalisa kwetingane: umtali ukhombisa ikhodi yekuphepha lesuka elebulini lakhe lekulanda, i-kiosk iyahlola kutsi ngubani lolandzako, bese tingane tikhishwa. Tikhungo letinabantfu naletisebenta tona nato tinetintfo tekuphepha -- kuhlolwa kwebantfu labatsenjiwe bekulanda, imilayeto yekubita umtali, kuphrintwa kabusha kwemalebula ekuphepha, kanye nembiko wesimo lesiphuthumako.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- Kuphuma kutfolakala etikhungweni letihleliwe esimeni se **sebantfu** ekuhleleni kwemlawuli we-kiosk
- Tingane kufanele [tibe setibhalisiwe](./completing-checkin) tinelilebula leliphrintiwe lelekulanda lelinemakhodi ekuphepha
- Kubita nembiko wesimo lesiphuthumako kudzinga kutsi libandla lakho lihlanganiswe nemniniti we-SMS ku-B1 Admin

</div>

## Kucala Kuphuma

1. Esikhungweni lesinabantfu, cindzetela **Check Out** eskrinini sekufuna.
2. Faka **ikhodi yekuphepha** yetinhlamvu letine lesuka elebulini lekulanda lemndeni. Ungayibhala, usebentise likhibhodi lelisekrinini, nobe uhlole ibhakhodi yelilebula ngesihloli se-USB nobe se-Bluetooth -- ikhodi iyatfunyelwa ngekutentela nangabe tonkhe tinhlamvu letine setifakiwe.
3. I-kiosk ikhombisa tingane letibhalisiwe ngaphansi kwaleyo khodi.

## Kuhlola Kutsi Ngubani Lolandzako

Skrini sekuphuma sibuta kutsi ngubani lotalanda tingane:

- **Bantfu labatsenjiwe bekulanda** kulomndeni bavela njengemakhadi langacindzetelwa anesitfombe sabo kanye nebuhlobo babo -- cindzetela umuntfu lome embikweni wakho.
- **Bantfu labadzala bemndeni** nabo bavela ku-grid yetitfombe.
- **Lokunye** kukuvumela kutsi ubhale ligama lemuntfu longekho eluhlwini.

Nangabe ligama lelibhaliwe lifanana nalomuntfu lophawulwe kutsi **Akavunyelwe** kulomndeni, i-kiosk iyavimba kuphuma ngesikhumbuto. Umsebenti webandla angakhetsa **Yeqa** kute kuchubeke nakunjalo -- lokweqiwe kuyabhalwa embikweni wekubakhona kanye neligama lemuntfu.

Nase kucinisekisiwe kutsi ngubani lolandzako, cindzetela check out. Ligama lemuntfu lolandzako liyagcinwa embikweni wekubakhona.

:::info
Bantfu labatsenjiwe kanye nalabangavunyelwe kulanda balawulwa ngumsebenti webandla ekhasini lemuntfu ngamunye ku-B1 Admin -- bona [Kuphepha Kwekubhalisa](../../b1-admin/attendance/checkin-safety#trusted-and-not-authorized-pickup-people).
:::

## Kubita Umtali

Udzinga umtali ngesikhatsi senkhonzo -- kushintjwa kwenabukhini, ingane lekhalako? Kusukela eskrinini sekuphuma esikhungweni lesinabantfu, umsebenti angatfumela **umlayeto**: umbiko wemshini kubatali nobe kubagcini betingane ngeliniti le-SMS lebandla. Batali labangavumanga imibiko nobe labangenayo inombolo yelucingo yeselula bayeqwa, futsi i-kiosk ikhombisa kutsi yimingaki imibiko letfunyelwe.

## Kuphrinta Kabusha Emalebula

Nangabe lilebula legama nobe lelekulanda lilahlekile nobe lonakele, umsebenti esikhungweni lesinabantfu angaphrinta kabusha emalebula alomndeni kusukela eskrinini sekuphuma ngemuva kwekufaka ikhodi yekuphepha. Kuphrinta kabusha kusebentisa iprinta lefanako kanye nemifanekiso yemalebula lofanako naleyo yekubhalisa kwekucala.

## Umbiko Wesimo Lesiphuthumako

Esimeni lesiphuthumako, umsebenti angatfumela umlayeto kubagcini **beti-ngane tonkhe letibhalisiwe** kuloko lokukhona senkhonzo ngasikhatsi sinye:

1. Vula **kuhlelwa kwemlawuli** ye-kiosk (kucindzetela ilogo esihlokweni kasikhombisa ngekushesha, kwengeta ne-PIN nangabe ikhona).
2. Cindzetela **Emergency broadcast**.
3. Faka umlayeto, bese ubhala **EMERGENCY** ensimini yekucinisekisa -- inkinobho letsi **Send broadcast** ihlala ivaliwe kuze kwenteke loko.
4. I-kiosk ibika kutsi mangaki emafoni lawutfole umlayeto kanye nekutsi bangaki bantfu labeqiwe (labangavumanga nobe labangenayo inombolo yeselula).

:::warning
Umbiko uya kuyo yonkhe imindeni lebhalisiwe yenkhonzo lekhetsiwe. Yisebentise etimweni tesibili tesimo lesiphuthumako -- kukhishwa kwebantfu, kuvalwa kwendzawo, simo lesibi selitulu.
:::
