# Kufuna Emalunga

<div class="article-intro">

Ngemuva kwekukhetsa inkhonzo, ufika kuskrini sekufuna lilunga. Lapha ufuna khona umndeni wakho kute ukhone kubhalisa. I-app iyakhona kufuna ngenombolo yelucingo nobe ngeligama.

</div>

<div class="prereqs">
<h4>Ngembi Kwekucala</h4>

- [Khetsa inkhonzo](./selecting-service) kuskrini setinkhonzo
- Cinisekisa kutsi umndeni wakho [usewengetiwe kusistimu](../../b1-admin/people/adding-people.md) ngumlawuli webandla

</div>

## Tindlela Tekufuna

Skrini sekufuna siniketa tindlela letimbili tekufuna. Ungashintjashintja emkhatsini wato ngekucindzetela inkinobho lesetulu eskrinini:

- **Lucingo** -- Funa ngenombolo yelucingo. Lena yindlela lesetjentiswa ngekwehluleka.
- **Ligama** -- Funa ngeligama.

## Kufuna Ngenombolo Yelucingo

1. Cinisekisa kutsi **Lucingo** yindlela lesebentako yekufuna (icaciswa ngembala lophezulu eskrinini).
2. Cindzetela ensimini yekufuna bese ufaka **tinombolo letine tebuncane** tenombolo yakho yelucingo usebentisa likhibhodi lelisekrinini.
3. Cindzetela inkinobho letsi **Funa** noma ucindzetele inkinobho yekufuna kulikhibhodi.

I-app iyafuna emalunga latinombolo talo telucingo tifanana naletinombolo lotifakile. Nangabe ufaka tinombolo letingetulu kwalatine, isebentisa inombolo lephelele. Nangabe ufaka letine kanye, iyahlanganisa netinombolo letine tekugcina tetinombolo telucingo letikuluhlu.

## Kufuna Ngeligama

1. Cindzetela inkinobho letsi **Ligama** lesetulu eskrinini kute ushintjele endleleni yekufuna ngeligama.
2. Faka **tinhlamvu letimbili tebuncane** teligama lemuntfu ensimini yekufuna.
3. Cindzetela inkinobho letsi **Funa**.

I-app iyafuna emalunga lamagama awo afanana naloko lokubhalile.

## Kukhetsa Umuntfu

Timphumela tekufuna tivela njengemakhadi ngephansi kwensimu yekufuna. Likhadi ngalinye likhombisa **ligama** kanye **nesitfombe** semuntfu (nangabe sikhona).

Cindzetela likhadi lemuntfu kute umkhetse. I-app iyalayisha onkhe emalunga emndeni waloo muntfu bese ikuyisa [kuskrini sekubuyeketa umndeni](./household-review).

:::tip
Kufuna ngetinombolo letine tekugcina tenombolo yelucingo yindlela lesheshako kakhulu yekufuna umndeni wakho. Khutsata emalunga kutsi asebentise lendlela kute abhalise ngekushesha.
:::

:::warning
Nangabe kute timphumela letitfolakele, i-app iyakhombisa umlayeto lokwatisako. Phindze uhlole tinombolo nobe ligama lolibhalile, nobe uzame lenye indlela yekufuna. Nangabe ungumvakashi wekucala, cela lusito kumvolontiya kute [wengetwe kusistimu](../../b1-admin/people/adding-people.md).
:::

## Sinyatselo Lesilandzelako

Ngemuva kwekukhetsa umuntfu, [utawubuyeketa umndeni wakho](./household-review) bese ulawula kwabelwa kwemacembu.
