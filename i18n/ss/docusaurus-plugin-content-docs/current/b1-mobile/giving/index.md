# Kunikela

<div class="article-intro">

I-B1 Mobile yenta kube lula kusekela libandla lakho ngemsebenti lopheleleko wekunikela ngco usukela efonini lakho. Nome ufuna kwenta umnikelo losheshako, ubuyekete umlandvo wakho wekunikela, nome uphatse tindlela takho tekukhokha, konkhe kutfolakala endzaweni yinye.

</div>

## Timenyu Tekunikela

Ingcenye yekunikela ihlelwe kutimenyu letine:

- **Simo Lesijikelele** -- Sifingqo semsebenti wakho wekunikela. Bona inani lakho lelipheleleko lomnyaka lonkhe, inani leminikelo loyentile, kanye nemininingwane yemnikelo wakho wekugcina, kufaka nenkinombho lesheshako ye-**Repeat** yekuphindza unikele ngalelo nani.
- **Nikela** -- Wenta umnikelo lomusha. Khetsa sikhwama, faka inani, bese unika imininingwane yakho yekukhokha kute ucedzele umnikelo.
- **Phatsa** -- Buka futsi uphatse tindlela takho tekukhokha letigciniwe. Engeta emakhadi lamasha nome ususe latungasakhi lawasebentisi.
- **Umlandvo** -- Lucwecwe locacile leminikelo yakho yonkhe yangaphambilini, kute ukhone kubuyeketa umlandvo wakho wekunikela lopheleleko.

## Indlela Tinkhokhelo Letisebenta Ngayo

Yonkhe iminikelo lentiwa nge-B1 Mobile icutjwa ngekuphepha nge-**Stripe**. Imininingwane yakho yekukhokha iphatswa luhlelo lwaphephile lwe-Stripe futsi ayikaze igcinwe kumaseva e-ChurchApps. Ungakhona kunikela ungakhatsateki, wati kutsi imininingwane yakho yetimali ivikelekile.

:::info
Umphatsi welibandla lakho usungula tikhwama letitfolakalako kanye nekucutjwa kwetinkhokhelo nge-[B1 Admin](../../b1-admin/). Nangabe ungasiboni sinketho sekunikela ku-app, kungenteka libandla lakho lingakalihleli lesi sici.
:::

## Loko Longakwenta

Nakuloko longakwenta kusuka engcenyeni yekunikela:

- **Nikela etikhwameni letitsite** -- Khetsa kutsi umnikelo wakho utfunyelwe kuluphi sikhwama, njengesikhwama sesijikelele, sikhwama semisebenti yeliVangeli, nome sikhwama sekwakha.
- **Buyeketa kunikela kwakho** -- Hlola sifingqo sakho semnyaka lonkhe bese uzulazula emlandvweni wakho wekunikela lopheleleko.
- **Gcina tindlela tekukhokha** -- Gcina imininingwane yekhadi lakho kute unikele ngekushesha ngesikhatsi lesitako.
- **Phindza unikele** -- Nikela ngekushesha ngalelo nani lomnikelo wakho wekugcina ngekucindzetela kanye.
