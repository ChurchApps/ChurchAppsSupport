# Kuphatsa Tindlela Tekukhokha

<div class="article-intro">

Ithebhu ye-**Phatsa** esigabeni sekunikela ikuvumela kubuka nekuphatsa tindlela takho tekukhokha letigciniwe. Ngekugcina likhadi, ungakwati kwenta iminikelo yakusasa ngekushesha ngaphandle kwekuphindze ufake imininingwane yakho onkhe malanga.

## Kubuka Tindlela Takho Tekukhokha

Nawuvula ithebhu ye-**Phatsa**, utawubona luhlu lwato tonkhe tindlela takho tekukhokha letigciniwe. Yinye ngayinye ikhombisa luhlobo lwelikhadi kanye nemanambala lamane wekugcina welikhadi, kute ukwati kuhlukanisa likhadi ngalinye ngekulula.

## Kwengeta Indlela Lensha Yekukhokha

Kute wengete likhadi lelisha:

1. Vula sigaba se-**Kunikela** ku-B1 Mobile.
2. Cindzetela ithebhu ye-**Phatsa**.
3. Cindzetela inketfo yekwengeta indlela lensha yekukhokha.
4. Faka imininingwane yelikhadi lakho. Imininingwane itfunyelwa ngendlela lephephile ku-**Stripe** kute icutjwe.
5. Nasekugciniwe, likhadi lelisha lilotawuvela eluhlwini lwakho lwetindlela tekukhokha.

Likhadi lakho lelisagciniwe litawube seliyatfola njengendlela yekukhokha ngesikhatsi lesilandzelako [wenta umnikelo](./making-donations) kusuka ethebhini ye-**Nikela**.

## Kususa Indlela Yekukhokha

Nangabe ungasafuni likhadi legciniwe, ungalisusa ethebhini ye-**Phatsa**:

1. Tfola likhadi lofuna kulisusa eluhlwini lwakho lwetindlela tekukhokha letigciniwe.
2. Cindzetela inketfo yekususa noma yekucisha lecantsi kwelikhadi.
3. Cinisekisa kususa.

Likhadi angeke lisavela njengendlela yekukhokha nawenta umnikelo.

:::info
Kususa indlela yekukhokha akutfintani neminikelo lengasadzale. [Umlando wemnikelo](./donation-history) wakho uhlala ungashintjwa.
:::

:::tip
Gcina tenkhe indlela yinye yekukhokha legciniwe kute unikele ngekushesha. Ngendlela leyi, ungasebentisa inkinobho ye-**Phindza** ethebhini ye-**Kuhlolwa Konkhe** kute unikele futsi ngekucindzetela kanye kuphela.
:::
