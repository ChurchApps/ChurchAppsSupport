# Kwenta Umnikelo

<div class="article-intro">

Kunikela nge-B1 Mobile kusheshako futsi kulula. Ithebhu ye-**Nikela** engcenyeni yekunikela ikuhola enchubweni ngetigaba letimbalwa, kute ukhone kusekela libandla lakho kuloko lapho ukhona.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- [Faka B1 Mobile](../getting-started/installing) futsi [ungene](../getting-started/logging-in) ku-akhawundi yakho
- Yiba nelikhadi lesikweleti nome le-debit lelilungele, nome [ugcine indlela yekukhokha](./managing-payments) kusengaphambili

</div>

## Indlela Yekunikela

1. Vula ingcenye ye-**Kunikela** ku-B1 Mobile.
2. Cindzetela ithebhu ye-**Nikela**.
3. Khetsa **sikhwama** lofuna kunikela kuso. Libandla lakho lingaba netikhwama letinyenti letitfolakalako, njengesikhwama sesijikelele, sikhwama semisebenti yeliVangeli, nome sikhwama sekwakha.
4. Faka **inani** lofuna kulinikela.
5. Nika **imininingwane yakho yekukhokha**. Nangabe unendlela yekukhokha legciniwe, ungayikhetsa lapha. Nangabe kutjho, faka imininingwane yekhadi lakho.
6. Tfumela umnikelo wakho.

Ngemuva kwekutsi umnikelo wakho wentiwe, utawubona kucinisekisa kutsi umnikelo wakho uphumelele. Umnikelo utaphindze uvele kuthebhu yakho ye-**Umlandvo**.

## Tinkhokhelo Letiphephile

Tonkhe tinkhokhelo tentiwa nge-**Stripe**, luhlelo lwetinkhokhelo loluthenjwako futsi lolusetjentiswa kabanti. Imininingwane yekhadi lakho iphatswa ngekuphepha nge-Stripe futsi ayikaze igcinwe kumaseva e-ChurchApps. Ungakhona kunikela ungakhatsateki, wati kutsi imininingwane yakho yetimali ivikelekile.

:::warning
Hlola kabili inani lemnikelo kanye nesikhwama ngaphambi kwekutfumela. Iminikelo icutjwa ngco futsi ayikhoni kubuyiswa ngekhatsi kwe-app.
:::

## Kuphindzaphindza Kunikela Ngekushesha

Nangabe ufuna kunikela ngalelo nani lomnikelo wakho wekugcina, awudzingi kuhamba yonkhe inchubo yekunikela. Ithebhu ye-**Simo Lesijikelele** ikhombisa umnikelo wakho wekamuva kanye nenkinombho ye-**Repeat**. Yicindzetele kute utfumele lomunye umnikelo lofananako ngekushesha.

:::tip
Nangabe uhlela kunikela njalonjalo, gcina indlela yakho yekukhokha ethebhini ye-**Phatsa** kute ukhone kunikela ngekushesha nakuvela sikhatsi lesitako. Bona [Kuphatsa Tindlela Tekukhokha](./managing-payments) kutfola imininingwane.
:::
