# Umlandvo Wemnikelo

<div class="article-intro">

I-B1 Mobile ikunika sitfombe lesicacile sekunikela kwakho ngeethebhu letimbili: **Simo Lesijikelele** kutfola sifingqo lesisheshako kanye **Ne-Umlandvo** kutfola umbhalo locacile weminikelo yonkhe. Sebentisa loku kulandzelela sipho sakho emnyakeni wonkhe.

## Ithebhu Yesimo Lesijikelele

Ithebhu ye-**Simo Lesijikelele** inika sifingqo semsebenti wakho wekunikela ngekubuka kanyekanye. Ikhombisa:

- **Inani lelipheleleko lomnyaka lonkhe** -- Inani lelipheleleko losewulinikele kulomnyaka.
- **Inani leminikelo** -- Ngumangaki eminikelo lohlukene loyentile kulomnyaka.
- **Umnikelo wekugcina** -- Imininingwane yemnikelo wakho wekamuva, kufaka inani nelusuku.
- **Inkinombho ye-Repeat** -- Inkinombho lesheshako ye-**Repeat** eceleni komnikelo wakho wekugcina yenta kutsi unikele ngalelo nani futsi ngekucindzetela kanye. Loku kuluncedvo nangabe unikela ngalelo nani njalonjalo.

Ithebhu ye-Simo Lesijikelele inguyona ndzawo lenhle yekucala nawuvula ingcenye yekunikela. Ikubonisa kutsi umile ngaphi futsi yenta kube lula kuphindze unikele.

## Ithebhu Yomlandvo

Ithebhu ye-**Umlandvo** ikhombisa lucwecwe lolupheleleko, locacile, leminikelo yakho yonkhe yangaphambilini. Sinhlanyelo ngasinye sifaka imininingwane lenjenge:

- **Lusuku** -- Ngenkhatsi umnikelo wentiwa ngayo.
- **Sikhwama** -- Kutsi umnikelo watfunyelwa kuluphi sikhwama.
- **Inani** -- Inani lemali lemnikelo.

Skrola ulucwecwe kute ubuyekete umlandvo wakho wonkhe wekunikela. Loku kuluncedvo ekugcineni sipho sakho emnyakeni wonkhe nome ekufunani umnikelo lotsite wangaphambilini.

:::tip
Hlola ithebhu ye-**Simo Lesijikelele** njalonjalo kute uhlale usemgomeni wetinjongo takho tekunikela. Inani lelipheleleko lomnyaka lonkhe lenta kube lula kubona intfutfuko yakho ngekubuka kanyekanye.
:::
