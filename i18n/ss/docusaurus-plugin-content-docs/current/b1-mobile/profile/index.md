# Iphrofayela

<div class="article-intro">

Iphrofayela yakho ku-B1 Mobile yindzawo lapho uphatsa khona imininingwane yakho yebumunye, umndeni wakho, netilungiselelo takho tekubukwa. Konkhe lokubuyeketako lapha kutfomela kuto tonkhe tinsita te-ChurchApps, ngako awudzingi kwenta tinguquko ngaphandle kwendzawo yinye.

</div>

## Emathebhu Eliphrofayela

Sikrini seliphrofayela sihlelwe kumathebhu lamanyenti:

### Iphrofayela

Ithebhu ye-**Iphrofayela** ikuvumela kubuka nekuhlela imininingwane yakho yebumunye, kufaka ekhatsi ligama lakho, imininingwane yekutfintana, nesitfombe seliphrofayela. Kugcina iphrofayela yakho ilungile kusita libandla lakho lihlale litfintana nawe.

### Umndeni

Ithebhu ye-**Umndeni** ikhombisa emalunga emndeni wakho netikhundla tabo ngekhatsi kwemndeni wakho. Lemininingwane isetjentiswa ticini letifana ne[kungena](../checkin/), lapho ungangenisa umndeni wakho wonkhe kanyekanye.

### Kubukwa

Ithebhu ye-**Kubukwa** ikuvumela kulawula loko lamanye emalunga elibandla labangakubona ngawe [eluhlwini lwemalunga](../community/member-directory.md). Wena ukhetsa kutsi ngiyiphi imininingwane yebumunye leyabelwanako neyihlala iyimfihlo.

### Kulindzelwe

Ithebhu ye-**Kulindzelwe** ikhombisa nome ngutiphi tinguquko teliphrofayela lotifakile letilindzela kuvunywa ngumphatsi.

### I-Akhawunti

Ithebhu ye-**I-Akhawunti** iniketa kufinyelela etilungiselelweni takho tekuphepha.

## Kufinyelela Eliphrofayeleni Yakho

1. Vula imenyu leyivulekako.
2. Cindzetela **Iphrofayela** kute uvule sikrini sakho seliphrofayela.
3. Sebentisa emathebhu latulu kute uzulazule emkatini wetigaba.
