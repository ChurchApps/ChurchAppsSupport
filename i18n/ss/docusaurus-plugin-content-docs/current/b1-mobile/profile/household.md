# Umndeni

<div class="article-intro">

Ithebhu ye-**Umndeni** esikrinini sakho seliphrofayela ikhombisa emalunga emndeni wakho. Lapha-ke lapho ungabona khona kutsi ngubani longulilunga lemndeni wakho nekutsi emalunga ahlangana njani, loko lokubalulekile ticini letifana nekungena.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- [Faka B1 Mobile](../getting-started/installing) futsi [ungene](../getting-started/logging-in) ku-akhawunti yakho

</div>

## Loko Lotawukubona

Nawuvula ithebhu ye-**Umndeni**, utawubona luhlu lwebantfu labasemndenini wakho. Kufakwa kwelilunga ngalinye kukhombisa:

- **Ligama** -- Ligama nesibongo semuntfu
- **Sikhundla** -- Indzima yakhe ngekhatsi kwemndeni (njengeNhloko nome Lolunye)
- **Sitfombe seliphrofayela** -- Sitfombe sakhe, nangabe silandiwe

Umndeni wakho licembu lelabelwanako lelihlanganisa emalunga emndeni wakho ngekhatsi kwesimiso selibandla.

## Kutsi Kubaluleke Ngani Umndeni

Imininingwane yemndeni wakho isetjentiswa ticini letinyenti ku-B1 Mobile naku-ChurchApps yonkhe:

- **Kungena** -- Nawu[ngena](../checkin/self-checkin.md) enkonzweni nome emcimbini, ungangenisa umndeni wakho wonkhe kanyekanye esikhundleni sekungenisa muntfu ngamunye ngekwehlukana.
- **Emarekhodi elibandla** -- Libandla lakho lisebentisa imininingwane yemndeni kute ligcine emarekhodi emndeni ahlelekile futsi litfumelelane nemndeni wakho njengelicembu.
- **Luhlu lwemalunga** -- Lamanye emalunga angase abone imininingwane yemndeni wakho [eluhlwini lwemalunga elibandla](../community/member-directory.md), kuya ngetilungiselelo takho [tekubukwa](./privacy-settings).

## Kubuka Umndeni Wakho

1. Vula imenyu leyivulekako futsi ucindzetele **Iphrofayela**.
2. Khetsa ithebhu ye-**Umndeni**.
3. Hlola luhlu lwemalunga emndeni langekhatsi kwemndeni wakho.

:::info
Bulunga bemndeni buvamise kuphatswa ngumphatsi welibandla lakho. Nangabe udzinga kwengeta nome kususa umuntfu emndenini wakho, tsindzana nelihhovisi lelibandla lakho kute utfole lusito.
:::

## Kugcina Imininingwane Ihlala Ilungile

Nangabe imininingwane yelilunga lemndeni idzinga kubuyeketwa, bangahlela [iphrofayela yabo](./editing-profile), nome ungatsindzana nemphatsi welibandla lakho kute wente tinguquko. Kugcina imininingwane yemndeni ilungile kucinisekisa kutsi ticini letifana ne[kungena](../checkin/) tisebenta kahle kumndeni wakho wonkhe.
