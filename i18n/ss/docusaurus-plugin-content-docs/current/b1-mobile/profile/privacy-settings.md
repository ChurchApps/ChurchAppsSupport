# Tilungiselelo Tekubukwa

<div class="article-intro">

Ithebhu ye-**Kubukwa** esikrinini sakho seliphrofayela ikuvumela kulawula loko lamanye emalunga elibandla labangakubona ngawe eluhlwini lwemalunga. Wena ukhetsa kutsi ngiyiphi imininingwane yebumunye leyabelwana obala neyihlala iyimfihlo.

</div>

<h4>Ngaphambi Kwekucala</h4>

</div>

## Kulawula Kubukwa Kwakho

1. Vula imenyu leyivulekako futsi ucindzetele **Iphrofayela**.
2. Khetsa ithebhu ye-**Kubukwa**.
3. Buyekeza ensimi ngayinye futsi ukhetse kutsi ingabonakala yini kulamanye emalunga.
4. Gcina tinguquko takho.

Ungalawula kubukwa kwemininingwane lefana ne:

- **Inambala yelucingo** -- Ingabe lamanye emalunga angayibona inambala yakho yelucingo
- **Likheli le-imeyili** -- Ingabe i-imeyili yakho ikhonjiswa [eluhlwini lwemalunga](../community/member-directory.md)
- **Likheli** -- Ingabe likheli lakho lasekhaya libonakala kulabanye

Ngesimo lesijwayelekile, libandla lakho lingase libe netinsimi letitsite letibonakalako nome letifihliwe. Ungalungisa letilungiselelo nomakunini kute tihambisane nendlela lophatseka kahle ngayo.

:::tip
Nangabe ungakacinisekisi kutsi labanye bangabonani, vakashela ithebhu ye-**Kubukwa** futsi ubuyekete tilungiselelo takho takwamanje. Ungahlala wenta tinguquko emuva.
:::

## Tinguquko Letilindzelwe

Ithebhu ye-**Kulindzelwe** ikhombisa nome ngutiphi tibuyekezo teliphrofayela lotifakile letilindzela kuvunywa ngumphatsi. Emanye emabandla adzinga umphatsi kutsi abukeze futsi avume tinguquko letitsite ngaphambi kwekutsi tisebente. Nangabe wentile inguquko ledzinga kuvunywa, utawuyibona ibhaliwe lapha kanye nesimo sayo samanje.

:::info
Nangabe inguquko loyentile ingakaqali kusebenta, hlola ithebhu ye-**Kulindzelwe**. Kungenteka mphatsi welibandla lakho adzinga kuyivuma kucala.
:::

## Tilungiselelo Te-Akhawunti

Ithebhu ye-**I-Akhawunti** iniketa kufinyelela etilungiselelweni takho tekuphepha. Sebentisa lelithebhu kuphatsa tilungiselelo letihambelana nekufinyelela nekuphepha kwe-akhawunti yakho.

## Sifingqo Semathebhu Eliphrofayela

| Ithebhu | Injongo |
|---|---|
| **Iphrofayela** | [Hlela imininingwane yakho yebumunye](./editing-profile) nesitfombe |
| **Umndeni** | [Buka emalunga emndeni wakho](./household) |
| **Kubukwa** | Lawula loko labanye labakubona eluhlwini lwemalunga |
| **Kulindzelwe** | Buyekeza tinguquko teliphrofayela letilindzele kuvunywa |
| **I-Akhawunti** | Phatsa tilungiselelo tekuphepha |

Lamathebhu asebenta ndzawonye kute akunike kulawula lokuphelele kwemininingwane yakho yebumunye nendlela leyabelwana ngayo ngekhatsi kwemphakatsi welibandla lakho.
