# Kuhlela Iphrofayela Yakho

<div class="article-intro">

Ithebhu ye-**Iphrofayela** ikuvumela kubuyeketa imininingwane yakho yebumunye kute libandla lakho lihlale linemininingwane lelungile lekugcinwe. Tinguquko lotenta lapha titfomela kuto tonkhe tinsita te-ChurchApps, kufaka ekhatsi B1.church nanome nguluphi lolunye luhlelo lolusetjentiswa libandla lakho.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- [Faka B1 Mobile](../getting-started/installing) futsi [ungene](../getting-started/logging-in) ku-akhawunti yakho

</div>

## Loko Longakuhlela

Iphrofayela yakho ifaka lemininingwane yebumunye lelandzelako:

- **Ligama nesibongo** -- Ligama lakho lelibonakalako kuto tonkhe te-ChurchApps
- **Likheli le-imeyili** -- Lisetjentiselwa kutfumelelana kwelibandla nekubuyisela i-akhawunti
- **Inambala yelucingo** -- Kute libandla lakho likwati kukufinyelela nawudzingeka
- **Likheli** -- Likheli lakho lasekhaya lemarekhodi elibandla
- **Sitfombe seliphrofayela** -- Sitfombe lesikumelelako [eluhlwini lwemalunga](../community/member-directory.md) naendzaweni yonkhe ku-app

## Kubuyeketa Imininingwane Yakho

1. Vula imenyu leyivulekako futsi ucindzetele **Iphrofayela**.
2. Khetsa ithebhu ye-**Iphrofayela** nangabe ingakasebenti.
3. Cindzetela ensimini lofuna kuyibuyeketa futsi wente tinguquko takho.
4. Gcina tinguquko takho nawucedzile.

Imininingwane yakho lebuyeketiwe ivumelanisa ngekutenteka. Nome ngubani lobuka iphrofayela yakho (kuya ngetilungiselelo takho [tekubukwa](./privacy-settings)) utawubona imininingwane lensha ngekushesha.

:::tip
Gcina likheli lakho le-imeyili nenambala yakho yelucingo tihlale tilungile. Libandla lakho lisebentisa lemininingwane kukutfumelela tatiso netikhumbuto letibalulekile.
:::

## Sitfombe Seliphrofayela

Ungakwati kwengeta nome kutjintja sitfombe sakho seliphrofayela kusuka ethebhini ye-**Iphrofayela**. Sitfombe sakho sivela [eluhlwini lwemalunga](../community/member-directory.md), eluhlwini lwemalunga elicembu, nanome ngukuphi lapho iphrofayela yakho ikhonjiswa khona.

## Tinguquko Kuto Tonkhe Te-ChurchApps

Ngobe tonkhe tinsita te-ChurchApps tabelana ngasinye sekusebentisa i-akhawunti, inguquko loyentako ku-B1 Mobile ibonakala ngekushesha ku-[B1.church](../../b1-church/), tinsimba tekuphatsa telibandla lakho, nanome nguluphi lolunye luhlelo loluhlanganisiwe. Awudzingi kubuyeketa imininingwane yakho ngekwehlukana endzaweni ngayinye.

:::info
Emanye emabandla adzinga kuvunywa ngumphatsi kutinguquko letitsite teliphrofayela. Nangabe inguquko yakho ingaboniseki ngekushesha, hlola ithebhu ye-**Kulindzelwe** esikrinini sakho seliphrofayela kute ubone simo saso.
:::
