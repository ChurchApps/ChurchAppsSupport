# Kutibhalisa

<div class="article-intro">

Kutibhalisa nge-B1 Mobile yinchubo lelula, lenetitebe letinyenti. Ngekucindzetela lokumbalwa nje, ungabhalisa kuba khona kwakho nekwemalunga emndeni wakho ngaphandle kwekulinda elayinini le-kiosk.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- [Faka i-B1 Mobile](../getting-started/installing) bese [ungena](../getting-started/logging-in) ku-akhawundi yakho
- Cinisekisa kutsi [emalunga emndeni wakho](../profile/household.md) asetwe kuphrofayela yakho

</div>

## Inchubo Yekubhalisa

### Nyatselo 1: Khetsa Inkhonzo Nesikhatsi

Nawuvula sikrini sekubhalisa, utawubona luhlu lwetinkhonzo netikhatsi letikhona. Leti tilungiselelwe libandla lakho futsi tingafaka kucedza njengekutsi "Sonto Ekuseni nga-9:00" nobe "Lesihlanu Kusihlwa nga-6:30."

1. Buka tinkhonzo letikhona.
2. Cindzetela inkhonzo nesikhatsi lotawuba kikho.

### Nyatselo 2: Khetsa Emalunga Emndeni

Ngemuva kwekukhetsa inkhonzo, utawubona luhlu lwemalunga emndeni wakho.

1. Buka luhlu lwemalunga emndeni.
2. Khetsa umuntfu ngamunye lotawuba khona enkhonzweni.

Awudzingi kubhalisa bonkhe -- khetsa kuphela labo labakhona ngiko.

### Nyatselo 3: Yabela Emacembu

Kumuntfu ngamunye lomkhetsile, utawumyabela licembu lenkhonzo. Emacembu angafaka kucedza njengekutsi Kukhonta Lokukhulu, Ligumbi Lebantfwana 1, Ligumbi Lebantfwana 2, nobe Licembu Lebasha.

1. Kulunga ngamunye lomndeni lomkhetsile, cindzetela kute ukhetse **licembu** lelifanele.
2. Cinisekisa kutsi umuntfu ngamunye wabelwe licembu lelifanele ngeminyaka yakhe nobe indzima yakhe.

### Nyatselo 4: Kucinisekisa

Nasewubabele emacembu bonkhe labakhona, cedzelisa kubhalisa.

1. Buka tikhetso takho kutsi konkhe kulungile.
2. Cinisekisa kubhalisa.
3. Utawubona sikrini sekucinisekisa lesikwatisa kutsi kuba khona kwakho sekubhaliswe.

## Tintfo Lokufanele Utati

- Ungabhalisa kuphela emalunga emndeni lahlotjaniswe ne-akhawundi yakho. Uma umuntfu angeko eluhlwini, cela mphatsi welibandla lakho kutsi amfake [emndenini](../profile/household.md) wakho.
- Uma kute tinkhonzo letivela esikrinini sekubhalisa, kungenteka libandla lakho alikabi natinkhonzo letilungiselelwe kubhalisa ngalesikhatsi.
- Uma udzinga kushintja lokukhetsile ngaphambi kwekucinisekisa, ungabuyela emuva bese ulungisa tikhetso takho.

:::warning
Cinisekisa kutsi ukhetse sikhatsi senkhonzo lesifanele kanye nemabelo emacembu lafanele ngaphambi kwekucinisekisa. Nasekubalwe kuba khona, kungadzingeka kutsi shintjo tentiwe ngumphatsi welibandla lakho.
:::

:::tip
Vula sikrini sekubhalisa nawusesendleleni uya ebandleni kuze utolungele kubhalisa nawufika. Kutsatsa emasekondi lambalwa kuphela kucedzelisa inchubo.
:::
