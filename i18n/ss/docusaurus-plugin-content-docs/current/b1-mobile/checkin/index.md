# Kubhalisa Kuya Enkhonzweni

<div class="article-intro">

B1 Mobile iyakuvumela kutsi utibhalise wena kanye nemalunga emndeni wakho ekuya etinkhonzweni telibandla, ngco ngelucingo lwakho. Egcina sikhatsi sekulindza elayinini le-kiosk, bese ubhalisa kuba khona kwakho ngaphambi kwekutsi ungene ngisho emnyango.

</div>

## Kusebenta Njani

Inchubo yekubhalisa yindlela lesheshako, lenetitebe letinyenti, lekuhola ekukhetseni inkhonzo, ekukhetseni labo labakhona, nasekwabeleni umuntfu ngamunye elicenjini. Nawesucinisekisa, kuba khona kwakho kuyabhaliswa ngekutentekela.

## Kucala

Kute ufinyelele ekubhaliseni, vula liphesheya lelitsi **Kubhalisa Kuya Enkhonzweni** ku-B1 Mobile. Kufanele ube ungenile ku-akhawundi yakho kutsi usebentise lesisici. Uma ungakangeni, utawucelwa kutsi [ungene](../getting-started/logging-in) kucala.

:::info
Lesisici sekubhalisa sidzinga kutsi libandla lakho selisungule tinkhonzo nemacembu ekuba khona etilungiselelweni tekuphatsa. Uma kute tinkhonzo letivela nawuvula sikrini sekubhalisa, kungenteka libandla lakho lisengakalulungiseleli lesisici. Tsindzana nemphatsi welibandla lakho kutfola lusito.
:::

## Loko Longakwenta

- **Bhalisa umndeni wakho** -- Khetsa kutsi ngimalunga muphi emndeni lakhona, bese ubhalisa kuba khona kwabo bonkhe ngasikhatsi sinye. Bona [Umndeni](../profile/household.md) kutfola kwati kabanti ngekuphatsa emalunga emndeni wakho.
- **Khetsa tinkhonzo netikhatsi** -- Khetsa kuletindzawo tetinkhonzo netikhatsi letikhona letilungiselelwe libandla lakho.
- **Yabela emacembu** -- Beka umuntfu ngamunye elicenjini lelifanele lenkhonzo, njengekutsi Kukhonta Lokukhulu, Ligumbi Lebantfwana, nobe Licembu Lebasha.
