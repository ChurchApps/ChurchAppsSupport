# Tifundvo (Kwesula)

:::danger Sikrini se-Tifundvo sesitsatselwe indzawo yiFreePlay
Sikrini se-**Tifundvo** lesasingekhatsi kwe-app sesiyekile kusetjentiswa. Kucuketfwe kwetifundvo lokuvela ku-Lessons.church manje kulethwa nge-**[FreePlay](/docs/freeplay/)**, i-app yeligagasi yemahhala yeChurchApps yemabhokisini ekilasini, iFire TV, i-Apple TV, ne-Android TV. IFreePlay inikeza kudlala ngaphandle kweluchungechunge, kuhlanganiswa kwekilasi, kanye nendlela lesebentisana kahle ne-remote yeligagasi lesikrini se-Tifundvo lesekhatsi kwe-app besingakwati kuyinika.

Kutisebenti-ngemahhala nebafundzisi labadzinga kucwaninga tifundvo ngelucingo nobe kwitablet, ngenani ku-[Lessons.church](https://lessons.church) ngco kubhrawuza yakho.
:::

<div class="article-intro">

Sikrini se-Tifundvo besikunika kufinyelela etifundvweni netintfweni tekufundza telibandla lakho. Kucuketfwe bekulethwa nge-Lessons.church futsi kufakwe ngco ngekhatsi kwe-B1 Mobile, kute ukwati kucwaninga nekubuka tifundvo ngaphandle kwekushintja kuya kulenye i-app.

## Kufinyelela Tifundvo

1. Vula sikrini se-**Tifundvo** endzaweni yekuhamba-hamba.
2. Kucuketfwe kwe-Lessons.church kuyalayisha ngco ngekhatsi kwe-app.
3. Cwaninga tifundvo libandla lakho lelitenta tatfolakala.

Libandla lakho lilungiselela kutsi tifundvo naluphi luhlelo lolubonakala lapha. Utawubona kucuketfwe lokuhlelwe nobe kwabelwe cembu lelibandla lakho.

## Loko Longakwenta

Nasewungekhatsi kwesikrini se-Tifundvo, ungacwaninga futsi ubuke tintfo libandla lakho lelitisetsile. Kuya ngaloko libandla lakho lelilungiselele, ungatfola:

- **Tinhlelo tetifundvo** -- Tincwadzi letakhelwe letinetigaba nemisebenti
- **Tetulo nemidiya** -- Tigcawu, emavidiyo, netitfombe letihambisana netifundvo
- **Tinsita letilayishekako** -- Emafayela nentfo tekufundza kabanti

Cindzetela tigaba letitsite ngekhatsi kwesifundvo kute ubone kucuketfwe kwato. Sebentisa tilawuli tekuhamba-hamba kute ushintje emkhatsini wetigaba nobe ubuyele eluhlwini lwetifundvo.

:::info
Lessons.church yipulatifomu lelehlukile yeChurchApps lehlanganiswa ne-B1 Mobile. Mphatsi welibandla lakho uphatsa kucuketfwe kwetifundvo nge-portal ye-Lessons.church.
:::

## Kutfolakala

Sikrini se-Tifundvo sivela nakuba libandla lakho lilungiselele tifundvo nge-Lessons.church. Uma ungasiboni sikhetso se-**Tifundvo** endzaweni yekuhamba-hamba, kungenteka libandla lakho alikakalungiseleli kucuketfwe kwetifundvo. Tsindzana nemphatsi welibandla lakho kutfola imininingwane lengetiwe.
