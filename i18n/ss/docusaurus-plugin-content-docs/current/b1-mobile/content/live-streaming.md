# Kusakata Bukhoma

<div class="article-intro">

B1 Mobile ikuvumela kutsi ubukele kusakateka bukhoma kwelibandla lakho ngco ngelucingo lwakho. Nakusakateka inkhonzo, ungangenela nomakuphi lapho ukhona bese ubukela ngesikhatsi lesifanako.

## Kubukela Kusakata Bukhoma

1. Vula sikrini se-**Stream** endzaweni yekuhamba-hamba, nobe cindzetela sibalo sekubala lesibonakala esikrinini se[Tintshumayelo](./sermons) nakusesitawucala kusakata bukhoma.
2. Uma kunenkhonzo lesakateka njengamanje, sisebenti sevidiyo siyalayisha ngekutentekela bese kusakata kucala kudlala.
3. Bukela kusakata ngco ku-app.

Nakute lokusakata bukhoma khona njengamanje, sikrini singakhombisa kucuketfwe lokungekho bukhoma nobe lwati mayelana nenkhonzo lelandzelako lelihleliwe.

## Kufika Kusakata

Kunetindlela letimbili tekufinyelela kusakata bukhoma:

- **Indzawo yekuhamba-hamba** -- Vula indzawo yekuhamba-hamba bese ucindzetela **Stream** kute uye ngco esikrinini sekusakata bukhoma.
- **Sikrini setintshumayelo** -- Nakuhleliwe kusakata bukhoma ngekhatsi kwemahora langu-24 lalandzelako, sibalo sekubala sibonakala esikrinini se[Tintshumayelo](./sermons). Nakucala kusakata, ungahamba uye ekusakateni kusuka lapho.

## Ngaphambi Nangemuva Kwenkhonzo

Libandla lakho libeka tikhatsi letitsite tekutsi kusakata kutfolakala nini. Ungabona sibalo sekubala nobe umbiko wekutsi akukho lokusakateka ngaphambi kwekucala kwenkhonzo. Ngemuva kwekutsi kusakata kuphele, sikrini sekusakata sibuyela esimeni saso sekungabi bukhoma.

:::info
Uma sikrini sekusakata sikhombisa sibalo sekubala nobe umbiko wekutsi akukho lokusakateka, inkhonzo lelandzelako lelihleliwe ayikakacali. Buyela ubuke ngesikhatsi lesibaliwe senkhonzo.
:::

## Kutfolakala

Sikrini sekusakata bukhoma sikhona nakuba libandla lakho lilungiselele luhlelo lwekusakata. Uma ungasiboni sikhetso se-**Stream** endzaweni yekuhamba-hamba ye-app yakho, kungenteka libandla lakho alikasilungiseleli kusakata bukhoma. Tsindzana nemphatsi welibandla lakho kutfola imininingwane.

:::tip
Hlola sikrini se-[Tintshumayelo](./sermons) kute ubone sibalo sekubala lesikhombisa kutsi kusakata lokulandzelako kutawucala nini.
:::
