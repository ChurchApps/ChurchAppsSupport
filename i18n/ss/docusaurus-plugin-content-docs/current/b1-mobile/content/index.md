# Kucuketfwe

<div class="article-intro">

B1 Mobile ikunika indzawo lenye lapho ungafinyelela khona konkhe kucuketfwe libandla lakho lokwabelanako, ngco ngelucingo lwakho. Kuya ngaloko libandla lakho lelikulungiselele, ungaba nekufinyelela etintshumayelweni, ekusakateni bukhoma, ekufundzeni liBhayibheli, evesini lemihla ngemihla, netintfweni tetifundvo -- konkhe ngaphandle kwekushiya i-app.

</div>

## Ticini Tekucuketfwe Letikhona

### Tintshumayelo

Buka futsi ubukele tintshumayelo letibhaliwe tivela elibandleni lakho. Tintshumayelo tihlelwe eluhlwini netinchubo kute ukwati kufumana kalula imilayeto yakadzeni nobe ulandzele inchubo lekhona.

### Kusakata Bukhoma

Bukela kusakateka bukhoma kwelibandla lakho ngco ku-app. Nakusakateka inkhonzo, ungangenela bese ubukela nomakuphi lapho ukhona.

### LiBhayibheli

Fundza imiBhalo usebentisa umfundzi weliBhayibheli lowakhelwe ngekhatsi. Hamba-hamba emincwadzini, tehlukwaneni, nasemavesini ngaphandle kwekushintja emaphini.

### Vesi Lwelilanga

Buka sitfombe seliBhayibheli lesihlelwe kahle lesishintja mihla yonkhe. Sitfombe siyatilungisa ngekutentekela kute sifanele indlela lidivayisi lakho lelibhekise ngayo.

### Tifundvo (Kwesula)

Sikrini se-Tifundvo lesasingekhatsi kwe-app sesitsatselwe indzawo ngu-**[FreePlay](/docs/freeplay/)** kwekudlala ekilasini, kanye nekuvakashela [Lessons.church](https://lessons.church) ngco kwekucwaninga. Bona [likhasi lelihlobene neTifundvo](./lessons) kute utfole imininingwane.

## Kufumana Kucuketfwe

Ticini tekucuketfwe titfolakala ngendzawo yekuhamba-hamba ye-app. Mphatsi welibandla lakho unquma kutsi tici tiphi tekucuketfwe letivuliwe, ngako-ke kungenteka ungatiboni tonkhe tikhetso letibalwe ngenhla. Uma sisici lesitsite singekho, kungenteka libandla lakho lisengakasilungiseleli.

:::info
Ticini tekucuketfwe tilungiselelwa ngumphatsi welibandla lakho nge-[B1 Admin](../../b1-admin/). Uma ungasiboni sisici losilindzele, tsindzana nelihhovisi lelibandla lakho.
:::
