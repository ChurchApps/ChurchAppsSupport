# Tintshumayelo

<div class="article-intro">

Sikrini se-Tintshumayelo sikuvumela kutsi ubuke futsi ubukele tintshumayelo letibhaliwe tivela elibandleni lakho. Yindzawo yakho lengcono yekulandzela imilayeto lowayiphuthelwa nobe kuphindze ubuke intshumayelo leyakukhutsata.

</div>

## Loko Lotawukubona

Sikrini se-Tintshumayelo sihlelwe ngekhatsi kwemibono lehlukene lenetigaba letimbili letiyinhloko:

- **Ithebhu ye-Playlists** -- Ikhombisa tinhlelo tetintshumayelo telibandla lakho (tichungechunge), letihlelwe ngelilanga. Luhlelo ngalunye luhlanganisa tintshumayelo letihlobene, kwenta kube lula kulandzela luchungechunge lwemaviki lamanyenti kusukela ekucaleni kuya ekugcineni.
- **Ithebhu ye-Recent** -- Ikhombisa luhlu lwetintshumayelo ngayinye ngendlela leyembuyela emuva ngesikhatsi, kute imilayeto lemisha ihlale isetulu.

Esiketfweni sesikrini, intshumayelo yekugcina yakamuva ikhonjiswa njengekhadi lelibalulekile, likunika kufinyelela ngekushesha emlayetweni wakamuva.

## Sibalo Sekubala Kusakata Bukhoma

Nakuletwa [kusakata bukhoma](./live-streaming) ngekhatsi kwemahora langu-24 lalandzelako, sibalo sekubala sibonakala esikrinini se-Tintshumayelo. Loku kukwatisa kutsi inkhonzo lelandzelako itawusakata nini kute uhlele kubukela ubukhoma.

## Kubukela Intshumayelo

1. Vula sikrini se-**Tintshumayelo** endzaweni yekuhamba-hamba.
2. Cwaninga ithebhu ye-**Playlists** nobe ye-**Recent** kute utfole intshumayelo loyifunako.
3. Cindzetela intshumayelo kute uyivule bese ucala kubukela.

Ungaphindze ucindzetele luhlelo kute ubone tonkhe tintshumayelo taloluchungechunge, bese ukhetsa intshumayelo lotsite loyifunako kubukela.

Tintshumayelo tingafaka kucuketfwe kwevidiyo nobe kwelizwi lokuhlelwe kuma-platform lanjengaYouTube, iVimeo, nobe iFacebook. Nawucindzetela intshumayelo, umdiya udlala ngco ngekhatsi kwe-app.

## Loko Intshumayelo Ngayinye Lekukhombisako

Ngulinye lentshumayelo lingakhombisa:

- **Sihloko** -- Libito lentshumayelo
- **Umshumayeli** -- Ngubani lowashumayela umlayeto
- **Lilanga** -- Nakuphi noma nakushicilelwe intshumayelo
- **Sitfombe** -- Sitfombe sekubuka intshumayelo ngaphambili

:::tip
Uma ubona sibalo sekubala esikrinini se-Tintshumayelo, sicindzetele kute uye ngco e[kusakateni bukhoma](./live-streaming) nakucala.
:::

## Kutfolakala

Sikrini se-Tintshumayelo sikhona nakuba libandla lakho lifake kucuketfwe kwetintshumayelo nge-portal yalo yekuphatsa. Uma ungasiboni **Tintshumayelo** ku-app yakho, kungenteka libandla lakho lisengakasilungiseleli lesisici.
