# Kubhalisela Imicimbi

<div class="article-intro">

Bhalisela imicimbi yelibandla ngco usukela ku-B1 Mobile ngenchubo lelula yetigaba letintsatfu. Ungabhalisa wena kanye nekwengeta emalunga emndeni ngco efonini lakho.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- [Faka B1 Mobile](../getting-started/installing) futsi [ungene](../getting-started/logging-in) ku-akhawundi yakho
- Umcimbi kufanele ube nekubhalisa lokuvuliwe futsi sikhatsi sekubhalisa kufanele sibe sivulekile

</div>

## Sigaba 1: Faka Imininingwane Yakho

1. Vula umcimbi wekhalenda lelicembu lonekubhalisa lokuvuliwe bese ucindzetela **Bhalisa**.
2. Sikrini sekubhalisa sikhombisa sihloko semcimbi, incazelo, kanye nelusuku.
3. Njengobe ungenile, lifomu liyacinisekisa kutsi ubhalisa ngaphansi kwe-akhawundi yakho.

:::info
Nangabe umcimbi unemkhawulo wetindzawo, ibha yenchubo ikhombisa kutsi kusele tindzawo letingaki. Nangabe umcimbi ugcwele, ngeke ukhone kubhalisa.
:::

## Sigaba 2: Engeta Emalunga Emndeni

1. Kute ubhalisele emanye emalunga emndeni, faka **Libito Lekucala** kanye **Nesibongo** salinye nalinye lilunga.
2. Cindzetela **Engeta Lilunga** kute wengete labanye bantfu (kufikela ku-10 sebentonkhe).
3. Cindzetela inkinombho yekususa eceleni kwanoma nguliphi lilunga kute ulikhiphe ekubhalisweni.
4. Nangabe ubhalisa wena wedvwa, yeqa lesigaba.

## Sigaba 3: Cinisekisa

1. Buyeketa imininingwane yemcimbi kanye nemininingwane yakho yekubhalisa.
2. Cinisekisa kubhalisa kwakho.
3. Umlayeto wekucinisekisa uyavela unemininingwane yemcimbi kanye nabo bonkhe lobabhalise.

Utawuphindze utfole **imeyili yekucinisekisa** ku-adresi ye-imeyili lehambisana ne-akhawundi yakho.

## Kukhansela Kubhalisa

Nangabe udzinga kukhansela, tsintsana nemphatsi welibandla lakho. Bangakhansela kubhalisa kwakho kusuka ekhasini lekubhalisa le-B1 Admin.
