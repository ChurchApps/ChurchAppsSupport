# Imicimbi

<div class="article-intro">

Sici seMicimbi ku-B1 Mobile sikuvumela kutsi uhlole imicimbi yelibandla lelitako bese uyabhalisa ngco usukela efonini lakho. Nangabe libandla lakho linika kubhalisa emcimbini, ungabhalisa wena kanye nemalunga emndeni wakho ngekucindzetela kanyekanye.

</div>

## Kutfola Imicimbi Lenekubhalisa

Imicimbi lenekubhalisa lokuvuliwe ivela kumakhalenda emacembu akho. Nawucindzetela emcimbini lonekubhalisa lokutfolakalako, utawubona inkinombho letsi **Bhalisa** kuminingwane yemcimbi.

Imicimbi ingakhombisa timo letahlukene:

- **Kuvulekile** -- Kubhalisa kuyatfolakala futsi ungabhalisa.
- **Akukakavulwa** -- Sikhatsi sekubhalisa asikakacali.
- **Kuvaliwe** -- Sikhatsi sekugcina sekubhalisa sesiphelile.
- **Kugcwele** -- Tindzawo tonkhe letitfolakalako sesitsatsiwe.
