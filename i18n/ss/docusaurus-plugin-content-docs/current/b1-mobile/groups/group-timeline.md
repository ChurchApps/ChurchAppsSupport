# Sikhatsi Selicembu

<div class="article-intro">

Sikrini se-Emacembu ku-B1 Mobile sifaka luhlu lwesikhatsi loluhlanganisa tento kuvela kuwo onkhe emacembu akho lube luhlu lolulodwa loluhlangene. Esikhundleni sekuhlola licembu ngalinye ngekwehlukana, ungaskorola kulandzela luhlu lolulodwa kute ubone konkhe lokwentekako.

</div>

:::info
Sici sesikhatsi selicembu asikho njengamanje ngesikhatsi kwentiwa tintfutfuko. Ngalesikhatsi, usengakwati kuhlola emacembu akho ngekwehlukana kusuka esikrinini se-Emacembu.
:::

</div>

## Kutsi Kusebenta Njani

Sikhatsi selicembu sihlanganisa tibhalo, ingcoco, netento letivela emacenjini onkhe lawubalunga kuwo. Njengoba baholi nemalunga elicembu babelana ngekucuketfwe, kuvela esikhatsini ngendlela lelandzelana ngesikhatsi. Loku kusho kutsi utfola simo lesiphelele sekuphila kwelicembu lakho ngaphandle kwekutjintjatjintja emkatini wemacembu lahlukene.

## Loko Lokuvela Esikhatsini

Sikhatsi sifaka intfo letinyenti letivela emacenjini akho:

- **Tibhalo** -- Imilayeto netibuyekezo letabelwane baholi nemalunga elicembu.
- **Ingcoco** -- Tingcoco letisachubeka letentekako ngekhatsi emacenjini akho.
- **Tatiso** -- Tatiso, tikhumbuto temicimbi, naletinye tento letisuka emacenjini akho.

Intfo yinye ngayinye esikhatsini ikhombisa kutsi ivela kuliphi licembu, kute uhlale wati sizinda saloko lokufundzako.

## Kuhlala Uphucukile

Sikhatsi yindlela lenhle yekuhlala uhlangene nemphakatsi welibandla lakho ngekubuka kanye. Naku kunye kwesikhombiso sekutfola lokunyenti kuso:

- **Buyela njalo** -- Vula sikrini se-Emacembu futsi uskorole ulandzele sikhatsi kute ukhawulele nento lensha.
- **Cindzetela kute utfole lokunyenti** -- Nangabe sibhalo sikubamba emehlo, sicindzetele nome ucindzetele ligama lelicembu kuze uye ekhasini leliphelele le-[imininingwane yelicembu](./group-details) kute utfole lokunyenti.
- **Konkhe endzaweni yinye** -- Awudzingi kuvakashela licembu ngalinye ngekwehlukana. Sikhatsi sikuletsela tento tonkhe.

:::info
Sikhatsi sikhombisa kuphela tento letivela emacenjini lahambisana nelibandla lakho. Nangabe ungaboni tibhalo, kusho kutsi emacembu akho abenganawo tento letisha, nome kungenteka ungakabi lilunga la-nome ngiliphi licembu.
:::

:::tip
Vumela [titfunyiselwa letihlwatjelwako](../community/notifications.md) kute wentelwe naku vela tibhalo letinsha emacenjini akho, ngisho nanawungeko ku-app.
:::
