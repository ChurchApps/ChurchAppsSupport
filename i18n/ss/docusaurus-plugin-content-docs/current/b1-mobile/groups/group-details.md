# Imininingwane Yelicembu

<div class="article-intro">

Nawucindzetela licembu kusuka esikrinini se-Emacembu, uyisiwa ekhasini lemininingwane yalelocembu. Lapha-ke lapho ungabona khona konkhe ngelicembu futsi uhlanganyele nekucuketfwe kwalo, kufaka ekhatsi tibhalo, luhlu lwemalunga, nemininingwane yelicembu.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- [Faka B1 Mobile](../getting-started/installing) futsi [ungene](../getting-started/logging-in) ku-akhawunti yakho
- Vula sikrini se-[Emacembu](./index.md)

</div>

## Loko Lotawukubona

Ikhasi lemininingwane yelicembu likunika sifingqo lesiphelele selicembu:

- **Imininingwane yelicembu** -- Ligama lelicembu, sichachiso, imininingwane yekuhlangana, nayo yonkhe lenye imininingwane leniketwe baholi belicembu.
- **Tibhalo netikhatsi** -- Luhlu lwetibhalo netento letihambelana nalelicembu. Ungabona loko lokwabelwane baholi nemalunga, kufaka ekhatsi tibuyekezo, tatiso, kanye nengcoco.
- **Emalunga** -- Buka labangemalunga elicembu. Loku kukusita ubone kutsi uhlangana nabani futsi ukhumbule bubuso lobati.

## Kuhlanganyela Nekucuketfwe Kwelicembu

Kusuka ekhasini lemininingwane yelicembu, ungahlanganyela nelicembu ngetindlela letinyenti:

- **Fundza tibhalo** -- Skorola kulandzela sikhatsi selicembu kute ukhawulele nento lensha.
- **Buka tiphrofayela temalunga** -- Cindzetela lilunga kute ubone imininingwane yeliphrofayela lakhe ku-[luhlu lwemalunga](../community/member-directory.md).
- **Hlala watiseka** -- Buyela njalo kubona tibhalo letinsha, tatiso, netento letisuka kubaholi belicembu nakumalunga akho.

## Kufinyelela Lapho

1. Vula sikrini se-**Emacembu** ku-B1 Mobile.
2. Cindzetela lelinye licembu -- licembu lakho lelidzumile, likhadi leliqoshiwe, nome nome ngucuphi ecwecweni.
3. Ikhasi lemininingwane yelicembu liyavuleka nako konkhe kucuketfwe nemininingwane yalo.

:::tip
Licembu lakho lelivakashelwa kakhulu livela njengelikhadi leliqoshiwe etulu kwesikrini se-Emacembu, kute kufinyelela elicenjini lakho lelitsandzekako kuhlale kucindzetela kanye kuphela.
:::
