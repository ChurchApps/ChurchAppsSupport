# Emacembu

<div class="article-intro">

Emacembu ku-B1 Mobile akusita uhlale uhlangene nebantfu nemisebenti lekubalulekile kuwe. Nome ubulilunga lelicembu lelincane, sifundvo seliBhayibheli, licembu lebavolontiya, nome umsebenti wenkonzo, sikrini se-Emacembu sikuyindzawo yakho lapho konkhe lokwentekako emphakatsini welibandla lakho kutfolakala khona.

</div>

## Loko Lotawukubona

Nawuvula sikrini se-**Emacembu**, emacembu akho ahlelwe ngekwekutsi uwavakashela kangakhi:

- **Licembu leliphambili** -- Licembu lakho lelivakashelwa kakhulu likhonjiswa ngendlela lecacile etulu kwesikrini njengelikhadi lelikhulu leliphambili. Licindzetele kute ungene ngco ekhasini lemininingwane yalelicembu.
- **Emacembu lakhetsiwe** -- Emacembu akho lalandzelako lavakashelwa kakhulu avela njengemakhadi lakhetsiwe ngentasi kwelelikhulu. Loku kukunika kufinyelela ngekushesha emacenjini lohlanganyela nawo kakhulu.
- **Wonkhe emacembu** -- Ngentasi kwesigaba salakhetsiwe, ungahlola lonkhe lamanye emacembu elibandla lakho.

Sakhiwo lesi senta kube lula kufinyelela emacenjini lawakhatsalelako kakhulu ngekucindzetela kanye kuphela, ngesikhatsi kusengasenta kufinyelela kuko konkhe lokunye.

## Loko Longakwenta

Nakuku loko longakwenta usuka esikrinini se-Emacembu:

- **Hlola emacembu** -- Skorola ulandzele emacembu latfolakalako elibandleni lakho futsi utfole lamasha wangenele.
- **Buka imininingwane yelicembu** -- Cindzetela nome ngiliphi licembu kubona sifingqo salo lesiphelele, kufaka ekhatsi tibhalo, emalunga, nemininingwane yelicembu.
- **Landzela sikhatsi** -- Luhlu lwesikhatsi loluhlangene lukhombisa tibhalo netento letivela kuwo onkhe emacembu akho endzaweni yinye, kute ungasoze wehluleke kuntfo.
- **Hlala uhlangene** -- Bona loko lokwabelwane bemalunga elicembu lakho, ulandzele ingcoco, futsi uhlanganyele nekucuketfwe kwelicembu.

Sikrini se-Emacembu sitivumelanisa nekusetjentiswa kwakho, njengalokufananako [i-dashboard](../dashboard/). Emacembu lowavakashela kakhulu akhushulelwa etulu kute ufinyelele kalula.
:::
