# Tati

<div class="article-intro">

B1 Mobile isebentisa tati letitfunyelwako kute ihlale ikwatisa ngaloko lokwentekako elibandleni lakho. Esikhundleni sekudzinga kuvula i-app kute uhlole tinguquko, lwati lolubalulekile luta kuwe ngekutentekela.

</div>

## Loko Lotawutiswa Ngako

Tati tingafaka umsebenti lohlukahlukene, lokufaka:

- **Imilayeto** -- Nakukhona umuntfu lokutfumela [umlayeto wangasese](./messaging).
- **Umsebenti wemacembu** -- Tinguquko, timiso, netingcoco tavela [emacembwini](../groups/) longu lilunga lawo.
- **Kwabelwa umsebenti wekukhonta** -- Nawabelwa [luhlelo](../serving/viewing-plans.md) nobe nakushintja simo sekwabelwa kwakho.
- **Letinye tinguquko** -- Tetfulo letivamile netintfo letikuphatsako.

## Kubuka Tati Takho

1. Cindzetela **luphawu lwensimbi** ekhoneni lelisetulu langakwesekudla kwe-app, nobe cindzetela sikhetso se-**Tati** endzaweni yekuhamba-hamba.
2. Sikrini setati sikhombisa tonkhe tati takho takamuva ngendlela lehlelwe ngemathebhu, lehlelwe kute ukwati kufumana ngekushesha loko lokufunako.

Sati ngasinye sifaka sifingqo lesifisha senguquko. Cindzetela sati sinye kute uye ngco ekucuketfweni lokuhlobene -- sibonelo, kucindzetela sati semlayeto kuvula ingcoco, bese kucindzetela sati sekukhonta kukusa eluhlelweni.

## Kususa Tati

Ungasula tati nasewutifundzile kute luhlu lwakho lulokhu lugxile kuloko lokusha.

- **Susa satsi sinye** -- Cindzetela luphawu lwendlwana yekulahla emalungwini lasekudleni kwemugca wesati kute ususe intfo leyodwa nje.
- **Susa tonkhe tati** -- Cindzetela **Susa Konkhe** esiketfweni lesisetulu kweluhlu lwetati kute ususe onkhe emati kanyekanye. Utawucelwa kucinisekisa ngaphambi kwekutsi konkhe kususwe.

Kususa sati kuphatsa kuphela kubuko bakho; akushintji umsebenti longaphansi (umlayeto, luhlelo, nobe sitfulo semacembu sisekhona). Uma unetati letivuliwe emadivayisini lamanyenti, kususa kulodwa kutawuhambisana nakuwo lamanye.

## Tati Letitfunyelwako

B1 Mobile itfumela tati letitfunyelwako edivayisini lakho kute wemukele tinguquko ngisho nome i-app ingavulekile. Kute wemukele tati letitfunyelwako, cinisekisa kutsi uvumele tati te-B1 Mobile etilungiselelweni telidivayisi lakho.

### Kuvumela Tati Letitfunyelwako

- **iOS** -- Hamba ku-**Settings** > **Notifications** > **B1 Mobile** bese uvula **Allow Notifications**.
- **Android** -- Hamba ku-**Settings** > **Apps** > **B1 Mobile** > **Notifications** bese ucinisekisa kutsi tati tivuliwe.

:::warning
Uma ungawemukeli tati, hlola tilungiselelo telidivayisi lakho kute ucinisekise kutsi tivuliwe. Futsi cinisekisa kutsi ungenile ku-app, njengobe tati tihlotjaniswe ne-akhawundi yakho.
:::

:::tip
Ungalawula tikhetfo tetati eveni lelidivayisi. Uma utfola tati tivama kakhulu, lungisa tilungiselelo telidivayisi lakho kute ulawule kutsi tiveta njani nangakusiphi sikhatsi.
:::
