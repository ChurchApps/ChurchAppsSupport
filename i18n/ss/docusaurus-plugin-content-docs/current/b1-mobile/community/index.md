# Umphakatsi

<div class="article-intro">

Tici te-**Umphakatsi** ku-B1 Mobile tikusita kuhlala uxhumene nemndeni wakho welibandla ngco ngelucingo lwakho. Nome udzinga kufuna lilunga lebandla, kutfumela umlayeto lomfisha, nobe kuhlola tati takho tekugcina, konkhe kusesandleni sakho.

</div>

## Loko Lotawukutfola

- **[Luhlu Lwemalunga](./member-directory)** -- Cwaninga futsi usesha eluhlwini lwemalunga elibandla lakho. Funa bantfu ngemabito abo bese ucindzetela lilunga kute ubone iphrofayela yalo lenelwati lekutsindzana kanye nebuhlobo bemndeni.
- **[Kutfumelana Imilayeto](./messaging)** -- Tfumela futsi wemukele imilayeto yangasese nalamanye emalunga elibandla. Cala ingcoco lensha nobe uchubeke lapho ushiye khona nengcoco lesikhona.
- **[Tati](./notifications)** -- Hlala watiswa ngetati letitfunyelwako mayelana nemsebenti wemacembu, imilayeto, kwabelwa umsebenti wekukhonta, naletinye tintfo letibalulekile letivela elibandleni lakho.

## Kucala

Kufanele ube ungenile kutsi ufinyelele ticini temphakatsi. Uma ungakangeni, utawucelwa kutsi [ungene](../getting-started/logging-in) nawulinga kusebentisa nawuphi walamacenjana.

Luhlu lwemalunga, kutfumelana imilayeto, netati kungafinyeleleka ngakunye endzaweni yekuhamba-hamba ye-app. Ndzawonye, likunika indlela lelula yekufuna bantfu, kutsindzana ngasese, nekuhlala watiswa ngaloko lokwentekako elibandleni lakho.

:::info
Emalunga nelwati lokubonakala kuwe eluhlwini kuya ngetilungiselelo teimfihlo telibandla lakho. Ungalawula kubonakala kwakho ngekwakho esikrinini se[Tilungiselelo Teimfihlo](../profile/privacy-settings.md). Uma unemibuto ngaloko lokwabelwanako, tsindzana nemphatsi welibandla lakho.
:::
