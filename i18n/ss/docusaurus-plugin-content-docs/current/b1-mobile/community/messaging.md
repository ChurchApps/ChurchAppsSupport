# Kutfumelana Imilayeto

<div class="article-intro">

B1 Mobile ifaka kutfumelana imilayeto yangasese kute ube nengcoco yemuntfu munye nemunye nalamanye emalunga elibandla ngco kusuka ku-app. Yindlela lelula yekuhlala uxhumene ngaphandle kwekudzinga kwabelana ngetinombolo telucingo tebumdvutane nobe emakheli e-imeyili.

</div>

## Kufinyelela Imilayeto Yakho

1. Cindzetela **Kutfumelana Imilayeto** endzaweni yekuhamba-hamba ye-app.
2. Tingcoco takho tangaphambilini tilotshiwe kute ufinyelele kalula.

## Kucala Ingcoco Lensha

1. Kusuka esikrinini sekutfumelana imilayeto, cindzetela sikhetso sekucala ingcoco lensha.
2. Sebentisa lensimu yekusesha kute utfole lilunga lofuna kulitfumela umlayeto. Ungatewati kanjalo kufumana emalunga nge[Luhlu Lwemalunga](./member-directory).
3. Khetsa lilunga emiphumeleni yekusesha.
4. Bhala umlayeto wakho bese ucindzetela **Tfumela**.

Ingcoco yakho lensha itawubonakala luhlwini lwakho lwetingcoco kute ukwati kuyichubeka kamuva ngekulula.

## Kufundza Nekutfumela Imilayeto

Nawuvula ingcoco, utawubona luhlu lolugcwele lwemilayeto. Umlayeto ngamunye ukhombisa sitfombe kanye nelibito lalotfumele, kwenta kube lula kulandzela. Kute uphendvule, bhala umlayeto wakho ensimini yembhalo lesetasi kwesikrini bese ucindzetela **Tfumela**.

## Kuhlela Imilayeto

Uma udzinga kulungisa intfo tsite ngemuva kwekutfumela umlayeto, ungawuhlela. Cindzetela umlayeto lofuna kuwushintja, khetsa sikhetso sekuhlela, wente sichibiyelo sakho, bese ugcina. Umlayeto lolungisiwe utawutsatsela indzawo yalo lowekucala engcocweni.

## Luhlu Lwakho Lwetingcoco

Tonkhe tingcoco takho letisebentako tibonakala eluhlwini esikrinini sesiseko sekutfumelana imilayeto. Luhlu lukhombisa libito nesitfombe somunye umuntfu, kwenta kube lula kufumana ingcoco lodzinga kuyichubeka. Cindzetela nome ngukuphi ingcoco kute uyivule futsi uchubeke lapho ushiye khona.

## Kususa Ingcoco

Uma ufuna kususa ingcoco eluhlwini lwakho, cindzetela luphawu lwendlwana yekulahla emalungwini lasekudleni kwemugca wengcoco bese ucinisekisa. Ingcoco iyasuswa kubuko bakho; lomunye umuntfu usengakhona akubone likhophi lakhe lengcoco ehlangotsini lakhe. Kute uphindze ukhulume nalomuntfu munye kamuva, cala nje ingcoco lensha.

:::tip
Sebentisa kutfumelana imilayeto kuhlanganisa netisebenti-ngemahhala bakho, kulandzelela umuntfu lowahlangana naye ebandleni, nobe kuvele kuhlola umngani. Kugcina kutsindzana kwebandla lakho endzaweni yinye lelula.
:::

:::info
Vula [tati](./notifications) kute utiswe nakukhona umuntfu lokutfumela umlayeto lomusha, ngisho nanome i-app ingavulekile.
:::
