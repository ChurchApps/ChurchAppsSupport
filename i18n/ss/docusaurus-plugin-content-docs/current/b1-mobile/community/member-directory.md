# Luhlu Lwemalunga

<div class="article-intro">

Luhlu lwemalunga ku-B1 Mobile lukuvumela kutsi ucwaninge futsi usesha bantfu elibandleni lakho. Nome udzinga kufuna lilunga lelicembu lakho, nobe kufuna umuntfu lowahlangana naye ngeliSonto, luhlu lulwenta kube lula kutsindzana.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- [Faka i-B1 Mobile](../getting-started/installing) bese [ungena](../getting-started/logging-in) ku-akhawundi yakho

</div>

## Kufinyelela Kuluhlu

1. Cindzetela sikhetso se-**Luhlu** endzaweni yekuhamba-hamba ye-app.
2. Luhlu luyalayisha nemalunga elibandla lakho ahlelwe nge-alfabhethi.

## Kucwaninga Luhlu

Emalunga ahlelwe nge-alfabhethi ngesibongo bese ahlelwa ngaphansi kwetihloko taletinhlamvu (A, B, C, njalo njalo). Ungasikrola luhlu kute ucwaninge umndeni wakho welibandla. Lilunga ngalinye likhombisa libito nesitfombe salo, kwenta kube lula kufumana loko lokufunako.

## Kusesha Emalunga

Uma uwati loko lokufunako, ibha yekusesha lesetulu kwe-luhlu yindlela lesheshako yekumfumana.

1. Cindzetela **ibha yekusesha** lesetulu kwesikrini.
2. Bhala libito.
3. Luhlu luyahlungwa ngasikhatsi lesifanako kute lukhombise imiphumela lefanako.

:::tip
Loku kusita kakhulu emabandleni lamakhulu lapho kukrola khona luhlu loluphelele kungatsatsa sikhatsi. Bhala nje tinhlamvu tekucala tebito kute unciphise imiphumela ngekushesha.
:::

## Kubuka Iphrofayela Yelilunga

Cindzetela lilunga lilonkhe eluhlwini kute uvule iphrofayela yalo. Likhasi lephrofayela lingafaka:

- **Lwati lekutsindzana** -- Inombolo yelucingo, likheli le-imeyili, nalokunye lwati libandla lakho lelenta kutfolakala.
- **Buhlobo bemndeni** -- [Emalunga emndeni](../profile/household.md) lahlotjaniswe nephrofayela yalomuntfu. Ungacindzetela lilunga lemndeni kute uye ephrofayeleni yalo naye.

:::info
Lwati lolubonakala ephrofayeleni ngayinye luya ngetilungiselelo teimfihlo naluhlu telibandla lakho. Akusiwo onkhe emavi latawukhonjiswa kulo lonkhe lilunga. Emalunga angalawula kubonakala kwawo ngekwawo ku[Tilungiselelo Teimfihlo](../profile/privacy-settings.md). Uma unemibuto ngaloko lokwabelwanako, tsindzana nemphatsi welibandla lakho.
:::
