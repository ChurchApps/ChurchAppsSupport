# B1 Mobile (Kuyekelwe Kusetjentiswa)

:::danger Kuyekelwe Kusetjentiswa — sebentisa B1.church esikhundleni
Luhlelo lwemvelo lwe-B1 Mobile lwe-iOS ne-Android **selulikhelwe** futsi alusakhatjatjiswa. Tici tawo manje titfunyelwa nge-**B1.church Progressive Web App (PWA)** e `https://yourchurchname.b1.church`, lengafakwa esikrinini sasekhaya sanoma nguphi umakhalekhukhwini we-iPhone nome i-Android futsi iniketa lokufananako kwekusebenta kwe-portal yelilunga ngaphandle kwekulanda ku-App Store.

**Nangabe ulilunga lelibandla:** buka [Ukufaka Njenge-App (PWA)](../b1-church/getting-started/installing-pwa.md) kute usetiwe lokufakiwe lokusesikhundleni kumakhalekhukhwini wakho.

**Nangabe ungumphatsi welibandla:** khombisa emalunga akho eyusayithi yelibandla lenu (`https://yourchurchname.b1.church`) esikhundleni sekukhomba ku-App Store / Google Play — bangafaka i-PWA kusuka kunoma ngukuphi likhasi.

Emakhasi langentasi agcinelwe kubhekwa mabhandla lasesakusebentisa luhlelo lwemvelo lwakadzeni.
:::

<div class="article-intro">

B1 Mobile *bekuyi*-app yemvelo lehambisana ne-B1.church ku-iOS ne-Android. Beyiniketa emalunga elibandla lokufinyelela emacembu, kunikela, kungena, tinhlelo tekukhonta, luhlu lwemalunga, imilayeto yangasese, timshumayelo, liBhayibheli, ivesi lelilanga, tifundvo, nakunye lokunyenti, ngisho nasendleleni.
</div>

I-app beyisebentisa i-akhawunti yakho ye-ChurchApps, leyona lefanako loyisebentisa ku-B1.church. Ngemuva kwekungena futsi ukhetsa libandla lakho, i-app beyitivumelanisa nemfanekiso welibandla lakho netilungiselelo tato. Nangabe bewulilunga lemabandla lamanyenti, bewungatjintjana emkatini wawo nomakunini.

B1 Mobile beyisekela titfunyiselwa letihlwatjelwako kute ungasoze wehluleke kuluhlelo lolubalulekile lolusuka elibandleni lakho nasemacembini akho. I-dashboard yasekhaya beyititfomela ngekusekelwa kusetjentiswa kwakho -- tici lobewutivakashela kakhulu betikhushulelwa etulu kute ufinyelele kalula.

## Tici Letibalulekile

- **Emacembu** -- Buka emacembu akho, ubambe incenye enkulumeni, futsi uhlale uhlangene nemalunga emacembu.
- **Kunikela** -- Wenta iminikelo elibandleni lakho ngqo kusuka kumakhalekhukhwini wakho.
- **Imicimbi** -- Cwaninga imicimbi letako futsi ubhalise ngqo kusuka kumakhalekhukhwini wakho.
- **Kungena** -- Ngena wena kanye nemalunga emndeni wakho ekhaya emitandzweni nasemicimbini.
- **Tinhlelo Tekukhonta** -- Buka tinhlelo takusasa tekukhonta, tikhundla takho letinikelwe wena tekukhonta, futsi ubhalise etikhundleni letivulekile.
- **Luhlu Lwemalunga** -- Cwaninga luhlu lwemalunga elibandla lakho futsi ubuke tiphrofayela temalunga.
- **Kutfumela Imilayeto** -- Tfumela futsi wemukele imilayeto yangasese kanye nalamanye emalunga elibandla.
- **Timshumayelo** -- Buka nome ulalele timshumayelo tebandla lakho.
- **LiBhayibheli** -- Fundza liBhayibheli ngekhatsi ku-app.
- **Ivesi Lelilanga** -- Chala lilanga lakho ngevesi lemihla ngemihla.
- **Tifundvo** *(kuyekelwe kusetjentiswa — kutsatselwe indzawo ngu-[FreePlay](/docs/freeplay/))* -- Beyisetjentiswa kufaka tifundvo letisuka ku-Lessons.church ngekhatsi ku-app.
