# Kubuka Tinhlelo

<div class="article-intro">

Sikrini setinhlelo ku-B1 Mobile sikunika simo lesiphelele seluhlelo lwakho lwekukhonta. Sihlelwe emathebhini lamabili kute ukwati kutfola ngekushesha sabelo sakho lesilandzelako nome ubuke emuva emlandweni wakho wekukhonta.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- [Faka B1 Mobile](../getting-started/installing) futsi [ungene](../getting-started/logging-in) ku-akhawunti yakho
- Ube unikwe okungenani luhlelo lolulodwa lwekukhonta ngumphatsi welibandla lakho

</div>

## Kufinyelela Etinhlelweni Takho

1. Cindzetela **Kukhonta** kukudukisa lokusetasi.
2. Sikrini setinhlelo siyavuleka nemathebhu lamabili etulu: **Letitako** ne-**Letendlulile**.

## Ithebhu Yeletitako

Ithebhu ye-**Letitako** yindzawo yakho yekucala kubona loko lokutako. Etulu utawubona likhadi leliphambili leligcizelela sabelo sakho lesilandzelako, kufaka ekhatsi ligama loluhlelo, lusuku, nesikhundla sakho. Loku kwenta kube lula kwati ngekubuka kanye nawudzingekile.

Ngentasi kwelikhadi leliphambili, utawutfola luhlu lwetibalo tesifingqo lolukhombisa kutsi mangaki emasabelo akho kusimo ngasinye:

- **Kuceliwe** -- Ucelwe kukhonta kodvwa ungakaphenduli.
- **Kuvunyiwe** -- Wemukele lelisabelo.
- **Kulindzelwe** -- Lelisabelo lilindzele sento lesinye.

Kuye phansi esikrinini, ithebula likhombisa tonkhe tikhatsi takho tekukhonta letitako, ndzawonye nemalanga awo, tikhundla, netimo tato.

## Ithebhu Yeletendlulile

Cindzetela ithebhu ye-**Letendlulile** kute utjintjele emlandweni wakho wekukhonta. Loku kukhombisa tonkhe tinhlelo lowake wakhonta kuto ngaphambilini, kubhaliswe ngesikhatsi lesibuyela emuva. Yindlela lenhle yekugcina kutfola bulunga bakho ngesikhatsi.

## Kubuka Imininingwane Yeluhlelo

Cindzetela nome ngeluphi luhlelo -- nome ngabe lutako nome ludzendlulile -- kute uvule imininingwane yalo leyiphelele. Kubuka kwemininingwane kufaka:

- **Luhlu lwenkonzo** -- Luhlu loluphelele lwenkonzo kanye nesikhatsi nemininingwane yetintfo kute ubone lulandzelane lwenkonzo.
- **Sabelo sakho** -- Sikhundla sakho, sikhatsi lodzingekile ngaso, nesimo sakho samanje.
- **Tabelo Teliciembu** -- Sifingqo salabo bonkhe labakhonta eluhlelweni, kute ubone kutsi ngobani labanye labaseciembini.

## Kucinda Tinhlelo

Ungacinda imininingwane yeluhlelo kute uzisebentisele ngasese ngaphandle kwe-inthanethi. Kusuka ekubukeni kwemininingwane yeluhlelo, sebentisa inketfo ye-**cinda** kute wente inguqulo leyacindzwako yeluhlu lwenkonzo nemasabelo. Loku kusita ekulingiselelweni, kubukwa ngemuva kwesitezi, nome kwabelana nemalunga eciembu labatsandza ikhophi yephepha.

:::tip
Hlola ithebhu ye-**Letitako** njalo kute uhlale welele eluhlelweni lwakho. Likhadi leliphambili lihlala likhombisa sabelo sakho lesilandzelako, ngako ungavula i-app futsi wati ngekushesha nawudzingekile.
:::

Utawutfola [titfunyiselwa letihlwatjelwako](../community/notifications.md) nawunikwa luhlelo lolusha nome nasimo sakho sesabelo sitjintja, kute ungasoze wehluleke kuluhlelo.
:::
