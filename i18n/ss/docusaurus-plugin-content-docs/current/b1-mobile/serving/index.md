# Kukhonta

<div class="article-intro">

Sigaba se-**Kukhonta** ku-B1 Mobile sikunika indlela lelula naleshesha yekuhlala welele etinjongweni takho tekukhonta. Nome ubulicembu lekuhlabelela, ubamukela emnyango, nome usita eNkonzweni yebantfwana, lapha-ke lapho ungabona khona nawuhlelwe kanye nekutsi indzima yakho yini.

</div>

## Loko Lotawukutfola

- **Tabelo Letitako** -- Bona tonkhe tikhatsi takho tekukhonta letitako ngekubuka kanye kute uhlale wati nawudzingekile.
- **Umlando Wekukhonta Lokwendlulile** -- Buyekeza tinkonzo lowake wakhonta kuto kute ugcine kutfola bulunga bakho ngesikhatsi.
- **Tabelo Tetikhundla** -- Sabelo ngasinye sikhombisa sikhundla loniketwe sona kanye nesimo saso samanje -- ingabe siceliwe, sivunyiwe, nome silindzele imphendvulo yakho.
- **Tfuba Tekutivolontiya** -- Hlola tikhundla letivulekile futsi [utibhalise](./volunteer-signup) tinkonzweni letitako.

## Kucala

Kute ubuke luhlelo lwakho lwekukhonta, cindzetela **Kukhonta** kukudukisa lokusetasi kwe-app. Kufanele ube ungenile kute ufinyelele tabelweni takho. Nangabe ungakangeni, utawubotjelwa [kungena](../getting-started/logging-in) kucala.

:::info
Utawubona kuphela tinhlelo lotinikwe wena. Nangabe ungaboni tabelo, kusho kutsi ungakafakwa ecenjini lekukhonta. Tsindzana nemphatsi welibandla lakho nome umholi wenkonzo kute utfole indlela yekungenela.
:::
