# Kubhalisela Kutivolontiya

<div class="article-intro">

Hlola tikhundla letivulekile tekutivolontiya futsi ubhalisele tinkonzo letitako ngco kusuka ku-B1 Mobile. Bona kutsi ngutiphi tikhundla letidzinga kugcwaliswa, mangaki emalungu latfolakalako, futsi ungenele ecienjini ngekucindzetela kambalwa nje.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- [Faka B1 Mobile](../getting-started/installing) futsi [ungene](../getting-started/logging-in) ku-akhawunti yakho
- Mphatsi welibandla lakho kufanele abe avumele kutibhalisela ngekwakhe okungenani sikhundla lesinye

</div>

## Kuhlola Tfuba Tekutivolontiya

1. Vula sigaba se-**Kutivolontiya** kusuka ekudukisweni kwe-app.
2. Kuvela luhlu lwetinhlelo tenkonzo letitako letinetikhundla letivulekile.
3. Likhadi loluhlelo ngalunye likhombisa:
   - **Ligama loluhlelo** ne-**lusuku lwenkonzo**
   - **Bha yenchubo yonkhe** lekhombisa emalungu lagcwaliswe wonkhe
   - **Simo** -- inani lemalungu lasengavulekile, nome "Kugcwele" nangabe onkhe emalungu asetsatiwe
   - **Emachips esikhundla** lakhombisa indzima ngayinye kanye nenani layo lemalungu lasengavulekile
4. Cindzetela **Buka Ubhalise** kute ubone luhlu loluphelele lwetikhundla teluhlelo.

## Kubhalisela Sikhundla

1. Cindzetela luhlelo kute uvule imininingwane yalo.
2. Tikhundla tihlelwe ngekwesigaba (sibonelo, "Licembu Lekuhlabelela", "Kwemukela Titfwele", "Nkonzo Yebantfwana").
3. Sikhundla ngasinye sikhombisa ligama lendzima, sichachiso, nemalungu latfolakalako.
4. Cindzetela **Bhalisa** cantsi kwesikhundla losifunako.
5. Kuvela sitsembiso: "Usubhalisiwe! Siyabonga ngekutivolontiya."
6. Sikhundla sikhombisa luphawu loluluhlata luciniseka ngekubhalisa kwakho.

:::info
Kufanele ube ungenile kute ubhalisele tikhundla. I-app itakubotjela kungena nangabe kudzingeka.
:::

## Kususa Kubhalisa

1. Tfola sikhundla losibhalisele -- sikhombisa luphawu loluluhlata "Usubhalisiwe".
2. Cindzetela **Susa**.
3. Kuvela lidayalogi lekucinisekisa lelibuta nangabe ucinisekile.
4. Cinisekisa kute utikhiphe kulesikhundla.

## Sikhatsi Sekuvala Netfuba

- **Tikhatsi Tekuvala** -- Libandla lakho lingase libeke sikhatsi sekugcina ngaphambi kwenkonzo. Ngemuva kwesikhatsi lesivaliwe, awukwati kubhalisa nome kususa kubhalisa. Libhena etulu kweluhlelo likhombisa nawusuvele sedlule sikhatsi sekuvala.
- **Emalungu** -- Sikhundla ngasinye sinenani lelikhawulekile lemalungu. Nasekugcwele, inkinobho yekubhalisa iyavaleka. Emalungu ayavuleka kabusha nangabe umuntfu akhansela.

:::tip
Vumela [titfunyiselwa letihlwatjelwako](../community/notifications.md) kute uhlale wati ngetfuba letinsha tekutivolontiya nangetinguquko etabelweni takho.
:::
