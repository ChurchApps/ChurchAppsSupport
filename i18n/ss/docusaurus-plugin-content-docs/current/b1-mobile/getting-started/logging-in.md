# Kungena

<div class="article-intro">

I-B1 Mobile isebentisa i-akhawundi lefananako ye-ChurchApps loyisebentisako ku-B1.church nakuletinye timphahla te-ChurchApps. Nangabe sewunayo i-akhawundi, ungangena nge-imeyili neliphasiwedi lakho lelisele likhona.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- [Faka B1 Mobile](./installing) emshinini wakho we-iOS nome Android

</div>

## Indlela Yekungena

1. Vula i-app ye-B1 Mobile.
2. Cindzetela inketho **Login** kusuka kumenyu ye-drawer nome kumyalo wekungena.
3. Faka **liadresi lakho le-imeyili** kanye **neliphasiwedi**.
4. Cindzetela **Sign In**.

Ngemuva kwekungena, utakuyiswa [ekhasini lasekhaya](../dashboard/). Nangabe ungakakhetsi libandla, utabutwa kutsi usesha bese ukhetsa lelinye.

## Kwenta I-akhawundi Lensha

Nangabe ungakabi nayo i-akhawundi ye-ChurchApps, ungayenta ngco ku-app:

1. Esikrinini sekungena, cindzetela sixhumanisi **Register** ngasentansi.
2. Faka **libito lakho lekucala**, **sibongo**, kanye **neliadresi le-imeyili**.
3. Lifomu litakutigcwalisela libandla lakho ngekutitentela nangabe kutfolwe lokufananako kuya ngemininingwane yakho.
4. Cindzetela **Register**.
5. Hlola i-imeyili yakho kutfola umlayeto lonelifasiwedi lelisikhashana.
6. Buyela ku-app, cindzetela **Login**, bese ungena nge-imeyili yakho kanye neliphasiwedi lelisikhashana lelivela ku-imeyili.

Ngemuva kwekungena kwakho kwekucala, ungashintja liphasiwedi lakho kusuka ku-[tilungiselelo temininingwane yakho](../profile/editing-profile.md).

:::info
Nangabe uzama kungena nge-imeyili lengakabhaliswa, utawuyiswa ngekutitentela esikrinini sekubhalisa kute wente i-akhawundi.
:::

:::warning
Cinisekisa kutsi uhlola i-spam nome i-junk folder yakho nangabe ungayitfoli i-imeyili yekubhalisa ngekhatsi kwemizuzu lomcane.
:::

## Nangabe Ukhohliwe Liphasiwedi Lakho

Nangabe ukhohliwe liphasiwedi lakho, cindzetela **Forgot Password?** esikrinini sekungena bese ufaka liadresi lakho le-imeyili. Utawutfola i-imeyili lenelikhodi lekucinisekisa lelinetinhlamvu letisi-6 lelisebenta imizuzu lelishumi nesihlanu. Faka likhodi esikrinini sekucinisekisa kute ungene, bese uvuselela liphasiwedi lakho kusuka ku-[tilungiselelo temininingwane yakho](../profile/editing-profile.md).

:::warning
Ngemuva kwemizamo lemihlanu lengakalungi yelikhodi, likhodi liyavalwa. Nangabe loku kwenteka, buyela esikrinini se-Forgot Password bese ucela likhodi lelisha.
:::

## Kuhlala Ungenile

I-B1 Mobile iyakugcina ungenile emkhatsini wetikhatsi. Awudzingi kungena onkhe malanga uvula i-app. Nangabe uphunyiwe ngenye indlela, ngena kabusha nge-imeyili neliphasiwedi lakho.

## Kuphuma

Kute uphume ku-app:

1. Vula imenyu ye-drawer ngekucindzetela **luphawu lwe-hamburger** (imicondvo lemitsatfu lehambisana) esikhungweni sangesekudla lengetulu.
2. Skrola uye ngasentansi kwemenyu ye-drawer.
3. Cindzetela **Sign Out**.

Utawuphuma futsi ungaphindze ungene nomanini nge-imininingwane yakho.
