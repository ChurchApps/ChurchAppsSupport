# Kushintja Emabandla

<div class="article-intro">

Nangabe uyilunga lelibandla lelingaphezu kwakunye, i-B1 Mobile yenta kube lula kushintja emkhatsini wawo. I-akhawundi yakho isebenta kuwo onkhe emabandla lasebentisa i-ChurchApps, ngako awudzingi ngaphandle kwekungena kunye.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- [Faka B1 Mobile](./installing) futsi [ungene](./logging-in) ku-akhawundi yakho

</div>

## Indlela Yekushintja Emabandla

1. Vula imenyu ye-drawer ngekucindzetela **luphawu lwe-hamburger** (imicondvo lemitsatfu lehambisana) esikhungweni sangesekudla lengetulu.
2. Esihlokweni semenyu ye-drawer, cindzetela inkinombho lekhombisa libito lelibandla lakho lelisele likhona (nome **Find a Church** nangabe kute libandla lelikhetsiwe).
3. Sikrini se-**Church Search** siyavela. Emabandla lowakavakashela kamuva akhonjiswa engetulu kute ukhone kufinyelela ngekushesha.
4. Cindzetela libandla lakamuva kute ushintjele kulo, nome bhala esikhaleni sekusesha kute utfole lelinye libandla.

Nawushintja emabandla, i-app iyalayisha kabusha ngeluphawu lwalelo bandla, tixhumanisi tekuhamba, kanye nekucuketfwe. [Likhasi lakho lasekhaya](../dashboard/), [emacembu](../groups/), kanye naletinye imininingwane utakukhombisa libandla lelisha lelikhetsiwe.

## Kusesha Libandla

Esikrinini se-Church Search, bhala tinhlamvu letingatiphindze tibe ntsatfu telibito lelibandla esikhaleni sekusesha. Emabandla lafanako avela ebhalweni ngaphansi. Cindzetela libandla kute ulikhetse.

:::info
Nangabe kute imiphumela levelako, zama kubhala ngenye indlela nome incenye lemfushane yelibito lelibandla. Kusesha kuhambisana nemabito emabandla lasabhalisiwe ku-ChurchApps.
:::

## Emabandla Akamuva

Sikrini se-Church Search siyakhumbula emabandla lowake wawavakashela. Nangabe ungakabhali nasiphi sicelo sekusesha, emabandla akho akamuva akhonjiswa ngekutitentela. Loku kwenta kube lula kushintjela emuva kulibandla loli sebentisa njalo.

:::tip
Awudzingi kuphuma bese uphindze ungene kute ushintje emabandla. I-akhawundi yakho yinye ye-ChurchApps isebenta kuwo onkhe emabandla akho. Vele ukhetse lelinye libandla kusuka esikrinini se-Church Search bese usukhona kucala.
:::
