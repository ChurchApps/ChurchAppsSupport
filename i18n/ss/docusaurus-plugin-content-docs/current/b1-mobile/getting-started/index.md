# Kucala

:::danger B1 Mobile seyiyimidzala
Luhlelo lwakhona lwe-B1 Mobile lususwe lwafakwa yi-[B1.church Progressive Web App (PWA)](../../b1-church/getting-started/installing-pwa.md). Bantfu labasha kufanele bafake i-PWA kusuka kunoma ngukuphi likhasi ku `https://yourchurchname.b1.church` esikhundleni sekulanda i-B1 Mobile ku-App Store nome Google Play.
:::

<div class="article-intro">

Kucala ku-B1 Mobile kutsatsa nje imizuzu lomcane. Landza i-app, ungene nge-akhawundi yakho ye-ChurchApps, ukhetse libandla lakho, bese usukhona kucala.

</div>

## Kucala Ngekushesha

1. **Landza** i-app kusuka ku-[App Store](https://apps.apple.com) (iOS) nome [Google Play Store](https://play.google.com) (Android). Sesha "B1.church".
2. **Ngena** nge-akhawundi yakho ye-ChurchApps lesele ikhona -- lelo imeyili neliphasiwedi lolisebentisako ku-B1.church. Nangabe ungakabi nayo i-akhawundi, cindzetela **Register** kute wente lensha.
3. **Khetsa libandla lakho** ngekulisesha ngelibito. Bhala tinhlamvu letingatiphindze tibe ntsatfu kute ubone imiphumela.
4. **Hlola likhasi lasekhaya** -- sikrini sakho lesenteliwe sikhombisa tixhumanisi letisheshako etici letinikelwa libandla lakho.

Nako-ke lokho. Ngemuva kwekutsi ukhetse libandla lakho, i-app iyalayisha luphawu lwelibandla lakho, tixhumanisi tekuhamba, kanye nekucuketfwe. Ungacala kusebentisa emacembu, kunikela, kungena, kanye nakho konkhe lokunye lokuvunyelwe libandla lakho.

## Kulesi Sigaba

- [Kufaka i-App](./installing) -- Kutsi ulande kuphi i-B1 Mobile kanye netidzingo temshini.
- [Kungena](./logging-in) -- Indlela yekungena nome yekwenta i-akhawundi lensha.
- [Kushintja Emabandla](./switching-churches) -- Indlela yekushintja emkhatsini wemabandla nangabe ungewelibandla lelingaphezu kwakunye.

:::tip
I-B1 Mobile isebentisa i-akhawundi lefananako ye-ChurchApps njengato tonkhe letinye timphahla te-ChurchApps. Nangabe sewunayo i-akhawundi kusuka ku-[B1.church](../../b1-church/), B1 Admin, nome Lessons.church, ungasebentisa lawomanye emafuphi ekungena kute ungene.
:::
