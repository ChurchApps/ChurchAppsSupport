# Kufaka i-App

:::danger B1 Mobile seyiyimidzala — faka i-PWA esikhundleni sayo
Luhlelo lwakhona lwe-B1 Mobile alusavuselelwa. Sincoma kufaka **i-B1.church Progressive Web App (PWA)** kusuka esiteni lelibandla lakho (`https://yourchurchname.b1.church`) esikhundleni sayo -- ingafakwa kusuka kunoma ngukuphi likhasi. I-PWA ingengetwa esikrinini sasekhaya ku-iOS nome Android ngemizuzwana bese ibamele tonkhe tici te-app yakhona.

Bona [Kufaka Njenge-App (PWA)](../../b1-church/getting-started/installing-pwa.md) kutfola imiyalo lehambisana ngesigaba nesigaba.
:::

<div class="article-intro">

I-B1 Mobile itfolakala kuwo womabili emishini we-iOS nawe-Android. I-app mahhala kulandwa futsi itsatsa nje umzuzwana kuyifaka kute ucale kuxhumana nelibandla lakho.

</div>

## Kulanda

- **iOS** -- Vula i-**App Store** ku-iPhone yakho nome i-iPad, sesha **B1.church**. Cindzetela **Get** kute ulande futsi ufake.
- **Android** -- Vula i-**Google Play Store** emshinini wakho we-Android, sesha **B1.church**. Cindzetela **Install** kute ulande futsi ufake.

Ungaphindze usesha "B1 Mobile" nangabe kusesha kwekucala akutfoli imiphumela.

## Ngemuva Kwekufaka

Uma i-app seyifakiwe, uyivule esikrinini sakho sasekhaya. Utawubona sikrini sekusesha emabandla lapho ungakhona kutfola futsi ukhetse libandla lakho. Nangabe sewunayo i-akhawundi ye-ChurchApps, unga[ngena](./logging-in) khona manje. Nangabe ungakabi nayo, ungenta i-akhawundi kusuka esikrinini sekungena.

I-app itakubuta imvumo yekutfumela tatiso te-push. Sincoma kuvumela tatiso kute ukhone kutfola imibiko yesikhatsi lesifanele kusuka elibandleni lakho nasemacembeni.

:::tip
Vumela tatiso te-push nasekubutwa ngesikhatsi sekufaka. Loku kucinisekisa kutsi utfola imibiko yesikhatsi lesifanele mayelana [nemsebenti wecembu](../groups/group-timeline.md), [imilayeto](../community/messaging.md), kanye [nekwabelwa umsebenti wekukhonta](../serving/viewing-plans.md).
:::

## Kuvuselela i-App

I-B1 Mobile itfola kuvuselelwa njalo ngetici letinsha kanye nekutfutfukiswa. Umshini wakho kufanele uvuselele i-app ngekutitentela, kodvwa ungaphindze uhlole kuvuselelwa ngesandla:

1. Vula i-**App Store** (iOS) nome i-**Google Play Store** (Android).
2. Sesha i-**B1.church**.
3. Nangabe kunekuvuselelwa lokutfolakalako, cindzetela **Update**.

:::info
I-B1 Mobile idzinga i-iOS 14 nome lengetulu emishinini ye-Apple kanye ne-Android 6.0 nome lengetulu emishinini ye-Android.
:::

## Sebentisa i-PWA Esikhundleni

Luhlelo lwakhona lwe-B1 Mobile selusatjelwa yi-**B1.church Progressive Web App (PWA)**. I-PWA ifakwa kusuka ku-Safari nome ku-Chrome — akudzingeki kulanda ku-App Store — futsi ingulokhu lokuncomekako kuko konkhe kufakwa lokusha. Bona [Kufaka Njenge-App (PWA)](/docs/b1-church/getting-started/installing-pwa) kutfola imiyalo.
