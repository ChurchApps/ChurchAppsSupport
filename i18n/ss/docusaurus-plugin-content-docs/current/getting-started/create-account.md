# Kudala I-Akhawunti Yakho

<div class="article-intro">

Kucala kusebentisa ChurchApps kucala ngekudala i-akhawunti yakho. Inchubo ithatsa emaminithi lambalwa kuphela futsi ikunika kufinyelela kuto tonkhe tinsimbi ta-ChurchApps, kufaka ekhatsi i-B1 Admin, i-B1.church, ne-Lessons.church.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Yiba nelikheli le-imeyili lelisebentako lelikulungele kucinisekisa i-akhawunti
- Buyeketa [sifingqo sekucala](./index.md) kuze uve kutsi yini ChurchApps leyihlinzekako

</div>

## Kubhalisa Kuya Ngezinyathelo

1. Iya ku-[B1.church](https://b1.church) kusiphequluli sakho seweb.
2. Chofota inkinobho ye-**Bhalisa**.
3. Faka **libito lakho lekucala**, **sibongo**, nelikheli **le-imeyili**.
4. Dala iphasiwedi ye-akhawunti yakho.
5. Fakela liphepha lekubhalisa.

## Kucinisekisa I-Imeyili Yakho

Ngemuva kwekubhalisa, utawutfola i-imeyili yekucinisekisa kulelikheli lolinikele.

1. Hlola ibhokisi lakho lemilayeto (inbox) kutfola i-imeyili yekucinisekisa levela ku-ChurchApps.
2. Chofota lisixhumanisi lekucinisekisa lelikuyo i-imeyili.
3. I-akhawunti yakho seyicinisekisiwe futsi seyikulungele kusetjentiswa.

:::tip
Uma ungayiboni i-imeyili yekucinisekisa, hlola ifolda yakho ye-spam nome yemfucumfucu. I-imeyili ivela ku-ChurchApps futsi ingase itsatse umzuzu nome imizuzu lemibili kutsi ifike.
:::

## Kucala Ibandla Lakho

Ngemuva kwekucinisekisa i-imeyili yakho nekungena, utawuba nelikhetfo lekudala ibandla lelisha nome kujoyina lelikhona kadze.

**Kudala ibandla lelisha:**

1. Khetsa likhetfo lekutsi **udale ibandla lelisha**.
2. Faka **libito lebandla lakho** nelwati loluyisisekelo. Liphepha lekubhalisa litawutigcwalisa ngekutiphindzaphindza imininingwane yebandla lakho uma kutfolakala lokufananako.
3. Cedza inchubo yekucala.
4. Utawuyiswa **kubhodi lekulawula le-B1 Admin (dashboard)** lapho ungacala khona kulungisa ibandla lakho.

**Kujoyina ibandla lelikhona kadze:**

1. Khetsa likhetfo lekutsi **ujoyine ibandla lelikhona kadze**.
2. Sesha ibandla lakho ngelibito.
3. Cela kujoyina. Umlawuli webandla lakho utawudzinga kuvuma sicelo sakho.

:::info
Uma uzama kungena nge-imeyili lengakabhaliswa, utawuholelwa ngekutentekela endleleni yekubhalisa kuze ukhone kudala i-akhawunti.
:::

:::info
Umuntfu lodala ibandla lelisha unikwa indzima ye-**Domain Admin** ngekutentekela, lenika kufinyelela lokugcwele kuto tonkhe tilungiselelo netici. Ungangeta balawuli labanyenti kamuva kusuka ekhasini le-**Roles**.
:::

## Loku Kulandzelako

Uma ibandla lakho selicaliwe, iya kuchibhukhu lwe-[Kulungiswa Kwasekucaleni](./initial-setup.md) kuze ulungise tilungiselelo tebandla lakho, luphawu, nekufinyelela kwelicembu.
