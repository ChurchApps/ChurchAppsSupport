# Kucala

<div class="article-intro">

Wemukelekile ku-ChurchApps -- emathulusi lamahhala, la-open-source lakhelwe kusita ibandla lakho kutsi lichume. Nome ungumhlangano lomncane nome inhlangano lenkhulu lene-campus letinyenti, ChurchApps ihlinzeka ngako konkhe lokudzingekako kuphatsa nekukhulisa umphakatsi webandla lakho.

</div>

## Imikhicito Ye-ChurchApps

ChurchApps ihlanganisa imikhicito lemhlanganweni lesebenta ndzawonye:

- **B1 Admin** -- Ibhodi lakho lekuphatsa ibandla (dashboard). Phatsa bantfu, emacembu, kuba khona, imnikelo, sicuku sewebusayithi, nalokunye lokunyenti kusuka endzaweni yinye lesemkhatsini.
- **B1.church** -- Iwebusayithi yakho yebandla lomphakatsi kanye nemnyango wemalunga (portal). Emalunga angabuka tishumayelo, afinyelele emalungelo, aphe imnikelo nge-inthanethi, futsi ahlale axhumene.
- **Lessons.church** -- Luhlelo lwekufundzisa lwebandla lolumahhala lwesikolwa seSonto, emacembu lamancane, nekufundza liBhayibheli. Finyelela ku-layibrari lekhulako yetifundvo mahhala.
- **FreeShow** -- Isofthiwe yekwetulwa lemahhala yeminkhonzo yakho yekukhonta. Bonisa emavi engoma, tibonakaliso, nemidiya ngesikhatsi semihlangano yakho.

:::info
Onkhe emathulusi e-ChurchApps **ngemahhala nge-100%** ngaphandle kwemikhawulo, emazinga langu-premium, kanye netindleko letifihliwe. ChurchApps yinhlangano lengakhi inzuzo lesekelwa ngeminikelo netivolontiya.
:::

## Loku Lokucuketfwe Kulencwadzi

Lencwadzi yekuchaza itakuhambisa kuto konkhe lokudzingekako kuze ucale futsi usebentise i-ChurchApps. Utawufundza kutsi:

1. Dala i-akhawunti yakho futsi ubhalise ibandla lakho.
2. Lungisa tilungiselelo letiyisisekelo tebandla lakho kanye neluphawu.
3. Setha licembu lakho ngetindzima netivumelwano letifanele.
4. Cala kusebentisa emathulusi kuphatsa umphakatsi wakho.
