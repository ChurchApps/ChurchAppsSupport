# Kungena

<div class="article-intro">

Sici se-**Kungena** ku-B1.church sikuvumela kutsi utingenise wena kanye nemalunga elikhaya lakho emitimbeni yelibandla ngco kusuka ku-member portal. Lena yindlela lelula yekubhalisa kubakhona kwakho ngaphandle kwekudzinga kuya e-kiosk yekungena lebonakalako.

</div>

## Kusebenta Njani

Nawusebentisa kutingenisa wena matfupha, ukhetsa umsebenti, ukhetse kutsi ngumaphi emalunga elikhaya latakuba khona, bese wabela wonkhe muntfu kucembu lelifanele ngesikhatsi ngasinye semsebenti. Nawucinisekisa, kubakhona kwakho kubhalwa ngco.

## Kucala

Kuze ufinyelele ekungeneni, cindzetela **Kungena** ku-[sidebar navigation](../getting-started/navigating.md) ye-member portal yakho. Kufanele ube [ungenile](../getting-started/logging-in.md) kuze usebentise lesi sici -- nangabe ungakangeni, utawucelwa kutsi ungene kucala.

:::info
Sici sekungena sidzinga kutsi libandla lakho selifake imisebenti nemacembu ekubakhona etilungiselelweni tekuphatsa. Nangabe kute imisebenti lekhona nawuvula likhasi lekungena, kungenteka libandla lakho lingakayihleli lesi sici. Tsindzetela umphatsi welibandla lakho kusita.
:::
