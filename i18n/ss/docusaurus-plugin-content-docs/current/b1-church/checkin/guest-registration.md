# Kubhalisa Tivakashi

<div class="article-intro">

Nangabe uvakashela libandla kwekucala, ungatibhalisa wena kanye nemndeni wakho njengetivakashi ngekusebentisa likhasi lekubhalisa tivakashi. Loku kudala umbhalo emshinini welibandla kuze bavolontiya bakwati kukungenisa (check in) nawufika.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Libandla lakho kufanele libe nekubhalisa tivakashi nge-QR kuvuliwe
- Skena i-QR code lebonakala e-kiosk yekungena, kumbe ufumane linki kuvela kuvolontiya welibandla

</div>

## Kubhalisa Tivakashi Kusebenta Kanjani

Kubhalisa tivakashi kwakhelwe tivakashi letingakabi nayo i-akhawunti. I-QR code ibonakala e-kiosk yekungena yelibandla lekuxhumana nelikhasi lekubhalisa. Ungatfola nalo linki kuvela kuvolontiya kumbe kumamukeli.

## Kubhalisa Umndeni Wakho

1. **Skena i-QR code** e-kiosk yekungena kumbe uvule linki lekubhalisa tivakashi kuselula sakho.
2. Ifomu lekubhalisa tivakashi liyavela linetinkhala **temuntfu lotsite naye kucala**:
   - **Ligama Lekucala** (kuyadzingeka)
   - **Sibongo** (kuyadzingeka)
   - **I-imeyili** (akusiko lokudzingekako)
   - **Lucingo** (akusiko lokudzingekako)
3. Kwengeta **emalunga emndeni**, cindzetela **Engeta Lilunga Lemndeni** bese ufaka ligama nesibongo sabo. Ungengeta emalunga emndeni langeta 10.
4. Kususa lilunga lemndeni, cindzetela inkhinobho yekususa eceleni kwelibito labo.
5. Cindzetela **Bhalisa** kutfumela.

## Ngemuva Kwekubhalisa

Nawusuwutfumele ifomu, kuvela sikrini sekucinisekisa lesinemlayeto lotsi: "Kubhalisa Kuphelele -- Usukulungele! Sisebenti selibandla sitakungenisa (check in) masinyane."

Kubhalisa kwakho kudala imibhalo emshinini welikhaya welibandla:

- **Likhaya** (household) lakhiwa kusetjentiswa sibongo salomuntfu lotsite kucala.
- **Imibhalo yemuntfu** yakhiwa yalelilunga ngalinye lemndeni ngesimo se-"Sivakashi" kubulunga.
- Volontiya kumbe umphatsi welibandla angabe angenisa (check in) umndeni wakho asebentisa i-kiosk ye-[B1 Checkin](../../b1-checkin/check-in/looking-up-members).

:::info
Nangabe i-imeyili yakho ifana neminye lesivele ikhona emshinini welibandla, luhlelo luyaxhumana naleyo mibhalo esikhundleni sekwakha lokuphindziwe. Loku kugcina imininingwane yelibandla ihlanzekile.
:::

:::tip
Kunikeza i-imeyili yakho kusita libandla kutsi likulandzele ngemuva kwekuvakashela kwakho. Akusiko lokudzingekako kodvwa kuyanconywa.
:::
