# Kukhonta

<div class="article-intro">

Incenye yeKukhonta ku-B1.church ikuvumela kuphequlula emathuba ekutinikela kanye nekubhalisa tikhundla ebandleni lakini. Ngaphandle kwekubuka tinhlelo takho lowabelwe kuto kanye netinhlu tenkhonzo, manje ungatikhetsela wena tikhundla letidzinga kugcwaliswa.

</div>

## Loko Longakwenta

- **Buka emathuba ekutinikela** -- Buka tinhlelo tenkhonzo letako letinetikhundla letivulekile letidzinga labatinikelako.
- **Bhalisela tikhundla** -- Khetsa sikhundla bese utibhalisela ngekuchofota kanye.
- **Buka tinhlelo takho** -- Buka [kwabelwa kwekukhonta](../plans/viewing-plans) kwakho lokutako naloko lokudlulile.
- **Susa kubhalisa** -- Khansela kutibhalisela nawe nangabe tinhlelo takho tishintja (ngembi kwesikhatsi lesigcinako).

## Indlela Lokusebenta Ngayo

Umphatsi welibandla lakini wakha tinhlelo tenkhonzo bese aphawula letinye tikhundla kutsi tiyatfolakala kutibhalisa wena matfupha. Letikhundla tibonakala ekhasini lekubhalisela kutinikela, tihlelwe ngetigaba (njenge-"Bhendi", "Kwemukela Tihambeli", nobe "Bufundisi Bebantfwana"). Ungabona kutsi mangaki emasikhundla lasekhona kusikhundla ngasinye bese ubhalisela lawo lokukhona nawo.

:::info
Kuphela tikhundla lomphatsi welibandla lakini akavule kutibhalisela ngekwakho letivela ekhasini lekutinikela. Letinye tikhundla tingase tigcwaliswe kuphela ngekwabelwa ngalokucondzile.
:::
