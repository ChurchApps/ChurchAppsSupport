# Kubhalisa Kuzinikela

<div class="article-intro">

Phequlula tikhundla tekutinikela letitfolakalako bese utibhalisela tinkhonzo letitako ngalokucondzile kusuka ku-B1.church. Tikhundla tihlelwe ngetigaba kuze ukhone kutfola ngekushesha lapho kudzingeka lusito khona bese ungenela tfimba.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Akhawundi ye-B1.church -- kufanele ube ungenile kuze utibhalisele
- Umphatsi welibandla lakini kufanele avule kutibhalisela ngekwakho kulokungenani sikhundla sinye

</div>

## Kuphequlula Tikhundla Letitfolakalako

1. Yendlulela ekhasini le-**Kutinikela** ku-website yelandla lakini ye-B1.church.
2. Utawubona luhlu lwetinhlelo tenkhonzo letako letinetikhundla tekutinikela letivulekile.
3. Uhlelo ngalunye lukhombisa **ligama loluhlelo**, **lusuku lwenkhonzo**, kanye ne**tikhatsi tenkhonzo**.

Tikhundla tihlelwe ngetigaba (sibonelo, "Bhendi", "Kwemukela Tihambeli", "Bufundisi Bebantfwana"). Sikhundla ngasinye sikhombisa:

- **Ligama lesikhundla** -- Indzima longayigcwalisa (sib., "Umshayi Wesigubhu", "Umemukeli", "Umphakeli Wekhofi").
- **Inchazelo** -- Imininingwane mayelana nendzima, nangabe ifakwe ngumphatsi.
- **Tikhala letitfolakalako** -- Bha lekhombisa kutsi mangaki emasikhala lagcwele kanye nalasele.

## Kutibhalisa

1. Tfola sikhundla lofuna kutinikela kuso.
2. Chofota inkhinobho ye-**Bhalisa** eceleni kwesikhundla.
3. Kuvela umlayeto wekucinisekisa: "Sewubhalisile! Siyabonga ngekutinikela kwakho."
4. Sikhundla manje sikhombisa luphawu loluluhlata lolucinisekisa kutsi ubhalisile.

:::info
Kufanele ube ungenile kuze utibhalisele. Nangabe awukangeni, utawucelwa kutsi ungene kucala.
:::

## Kususa Kubhalisa

Nangabe tinhlelo takho tishintja bese ungasakhoni kukhonta:

1. Tfola sikhundla losubhalisele kuso -- sitawukhombisa luphawu loluluhlata lolutsi "Sewubhalisile".
2. Chofota inkhinobho ye-**Susa**.
3. Cinisekisa kutsi ufuna kutikhipha kulesikhundla.
4. Umlayeto uyacinisekisa: "Sewukhishiwe kulesikhundla."

## Emalanga Lekugcina

Umphatsi welibandla lakini angabeka lilanga lekugcina lekubhalisa kuhlelo ngalinye. Lilanga lekugcina linani lemahora labekiwe ngembi kwelusuku lwenkhonzo. Nase seliphelile lelilanga:

- Awusakhoni kubhalisela tikhundla letitasha.
- Awusakhoni kususa kubhalisa lokukhona.
- Umlayeto etulu kohlelo ukhombisa kutsi lilanga lekugcina lekubhalisa selidlulile.

## Emasikhala Etikhundla

Sikhundla ngasinye sinenani lelilinganiselwe lemasikhala. Nawo onkhe emasikhala sewagcwele:

- Sikhundla sikhombisa njenge **Sigcwele**.
- Inkhinobho yekubhalisa iyavaliwe.
- Nangabe umuntfu asusa kubhalisa kwakhe, sikhala siyavuleka futsi.

:::tip
Hlola likhasi lekutinikela njalo, ikakhulu ekucaleni kweviki. Tikhundla letitsandvwako tigcwala ngekushesha, ngako kubhalisa kusenesikhatsi kukupha likhetfo lelincono.
:::
