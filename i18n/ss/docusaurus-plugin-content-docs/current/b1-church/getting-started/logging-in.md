# Kungena

<div class="article-intro">

Kungena ku-B1.church kukunika kufinyelela ephothalini yemalungu lapho ungabona khona i-Timeline yakho, uphatse emacembu, unikele nge-inthanethi, futsi usebentise tonkhe tinsiza lelibandla lakini selitihlele. Lomlobo ubeka kutsi ungangena njani, uhambe njani ku-imenyu yakho yemsebentisi, kanye nekuphuma.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Udzinga i-akhawunti ye-ChurchApps. Bona [Kucala](../getting-started/) kutsi udale leyakho.
- Udzinga likheli le-B1.church lelibandla lakini (sibonelo, `https://yourchurchname.b1.church`).

</div>

## Kungena Njani

1. Vakashela likheli le-B1.church lelibandla lakini (sibonelo, `https://yourchurchname.b1.church`).
2. Chofoza inkhinobho **Ngena** lesekoneni lesekudla etulu.
3. Faka **likheli lakho le-imeyili** kanye **neliphasiwedi** lakho.
4. Chofoza **Ngena**.

Ngemuva kwekungena, utawubuyiselwa ekhasini lobukulo ngaphambi kwekutsi uchofoze kungena. Nangabe uye ngco ekhasini leliphothali yemalungu ungakangeni, utawubuyiselwa kulelo khasi ngco ngemuva kwekungena.

## Kufinyelela Iphothali Yemalungu

Nawusengenile, ungafinyelela iphothali yemalungu ngekuchofoza **libito** lakho lesekoneni lesekudla etulu ekhasini bese ukhetsa **Iphothali Yemalungu** ku-imenyu. Iphothali yemalungu yindzawo lapho tonkhe tinsiza takho tebandla tekwakho tikhona, kufaka [i-timeline](../community/timeline.md), [emacembu](../groups/), [kunikela](../giving/), kanye nalokunye.

## Imenyu Yakho Yemsebentisi

Ngemuva kwekungena, libito lakho livela njengechip esekoneni lesekudla etulu. Yichofote kuze uvule imenyu lenalokukhetsiwe lokulandzelako:

- **Iphothali Yemalungu** -- Iya ku-dashbhodi yakho yemuntfu ngamunye leseiphothalini yemalungu.
- **Hlela Iprofayela** -- Ntjintja tilungiselelo te-akhawunti yakho nome uhlele iprofayela yakho yebandla.
- **Phuma** -- Phuma ku-akhawunti yakho.

## Kufinyelela Kwebalawuli

Nangabe ungumphatsi welibandla lonemvumo lefanele, utawubona nalokukhetsiwe **Iphothali Yebalawuli** ku-imenyu yakho yemsebentisi. Kuyichofota kukuyisa ku-B1 Admin lapho ungaphatsa khona tilungiselelo telibandla lakini, bantfu, emacembu, nenshila.

## Kukhohlwa Liphasiwedi Lakho

Nangabe ungasakhumbuli liphasiwedi lakho:

1. Chofoza **Ngikhohliwe Liphasiwedi** ekhasini lekungena.
2. Faka likheli le-imeyili lelihlotjaniswe ne-akhawunti yakho ye-ChurchApps bese uyathumela.
3. Hlola i-imeyili yakho utfole umlayeto lonekhodi **yekucinisekisa lenetinombolo letisitfupha**. Ikhodi isebenta emiminithi lelishumi nesihlanu.
4. Faka ikhodi ekhasini lekucinisekisa kuze ungene.
5. Nawusesungenile, vakashela iprofayela yakho kuze ukhetse liphasiwedi lelisha.

:::tip
Nangabe ikhodi ingafiki ngemphelamehlo lombalwa, hlola ifolda yakho ye-spam. Ngemuva kwemizamo lemihlanu lengakalungi, ikhodi iyakhiyeka — cela lenye ekhasini lekukhohlwa Liphasiwedi nangabe kwenteka loko.
:::

## Kuphuma

Kuze uphume, chofoza **libito** lakho lesekoneni lesekudla etulu bese ukhetsa **Phuma**. Utawukhishwa ku-akhawunti yakho futsi ubuyiselwe kusayithi lelivulekele wonkhe umuntfu.
