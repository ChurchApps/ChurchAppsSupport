# Tinketho Tetatiso

<div class="article-intro">

Tinketho tetatiso tikuvumela kutsi ukhetse kutsi ngutiphi tatiso lotitfolako kusuka elibandleni lakini futsi utitfola njani -- tatiso letisunyelwako, i-imeyili, nome i-bell lengekhatsi kwe-app. Ungalungisa lonkhe luhlobo lwesatiso ngalunye, ubeke emahora ekuthula, nome ucedze konkhe kanyekanye, konkhe kusuka esikhrinini sinye.

</div>

## Kuvula Tinketho Takho

1. Vula ingcenye ye-**Tatiso** (i-icon ye-bell) ku-B1 member app.
2. Chofoza **Tinketho Tetatiso**.

## Kulawula Konkhe

Etulu esikhrinini utawutfola tilungiselelo letisebenta kukho konkhe:

- **Cedza konkhe** -- Yima kanyenkani tonkhe tatiso letingesibalulekile.
- **Tatiso letisunyelwako** -- Vula nome ucedze tatiso letisunyelwako edivayisini yakho.
- **Kuvama kwe-imeyili** -- Khetsa kutsi ti-imeyili tetatiso tifika njani:
  - **Ngasinye sesatiso** -- Tfumela i-imeyili njengoba kwenteka tintfo.
  - **Sifingqo sansuku sinye** -- Hlanganisa tonkhe ku-imeyili yinye yalelilanga.
  - **Neze** -- Ungatfumeli ti-imeyili tetatiso.
- **Emahora ekuthula** -- Beka sikhatsi sekucala nesekugcina (kanye netaimzoni yakho) lapho ungatsandzi kuphazanyiswa khona. Tatiso letingesiphutfumako tilinda kuze emahora ekuthula acedze.

## Kukhetsa Tatiso Ngenhlobo

Ngaphansi kwekulawula konkhe kunelibhokisi lalezinhlobo tetatiso. Kunhlobo ngayinye ungavula nome ucedze tindlela lotitsandzako:

- **Kusunyelwa** -- Satiso edivayisini yakho.
- **I-imeyili** -- Umlayeto ebhokisini lakho le-imeyili.
- **Ngekhatsi kwe-App** -- I-bell/badge ngekhatsi kwe-app.

Tinhlobo letivamile tifaka **Tikhumbuto Temicimbi**, **Kukhonta Neluhlelo**, **Imilayeto Lecondzile**, **Inkulumo Yelicembu**, **Titfombiso Temithandazo**, kanye **netimemetelo Telibandla**. Sibonelo, ungangayivula i-tatiso lesunyelwako yeMilayeto Lecondzile kepha uyicedze yeTimemetelo.

:::info
Letinye tinhlobo tetatiso tihlala **tivuliwe** futsi angeke tacedzeka -- **I-Akhawunti Nekuphepha**, **Tincwadzi Nemibiko Yemnikelo**, kanye **Netexwayiso Tekuphepha Kwekungena**. Loku kudzingekile kutsi uphephe futsi ngenca yemitsetfo, ngako-ke tikhombisa luphawu loluncane lweloko. Konkhe lokunye kwakho kutsi ukulawule.
:::

Letinye tinhlobo **tivalekile** ngekuzenzekela (njengetatiso letimayelana nekutsengiswa nome imali) futsi angeke tikufinyelele ngaphandle kwekutsi ukhetse kuvumelana nato.

## Kugcina Lutjintjo Lwakho

Ngemuva kwekulungisa tinketho takho, chofoza **Gcina**. Lutjintjo lwakho lusebenta ngco futsi lusebenta yonkhe indzawo losebentisa khona i-B1.

:::tip
Tatiso letiphutfumako -- njengesexwayiso sekuphepha nome umlayeto wekuphepha kwekungena kwemntfwana -- ziyakufinyelela ngisho nase emahoreni ekuthula. Tibiko tansukuzonkhe tilinda kuze emahora ekuthula acedze.
:::
