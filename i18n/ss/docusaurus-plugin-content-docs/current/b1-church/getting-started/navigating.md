# Kuhamba nge-B1App

<div class="article-intro">

Iphothali yemalungu ku-B1.church isebentisa luhlaka lwekuhamba lwebha lelisemaceleni lokwenta kube lula kunyakata emkhatsini wetinsiza letahlukene lotifinyelelako. Lelikhasi lichaza kutsi kuhamba kusebenta njani kuwo womabili emakhompyutha nemadivayisi lasetjenziswako lucingo.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Kudzingeka kutsi [ube ungenile](./logging-in.md) kuze ufinyelele iphothali yemalungu.

</div>

## Kuhamba Nge-Bha Lesemaceleni

Nawungena eliphothalini yemalungu, utawubona bha lelimile lelisemaceleni lesenshoni yesikhrini. Lebha ine-tabs tengcenye ngayinye yeliphothali:

- **I-Timeline** -- I-feed yakho lekhethekile yetibiko netinkulumo letisuka emacenjini akho nasebandleni. Lena yitabs levele ilayishwa nawuvula iphothali yemalungu.
- **Me** -- I-dashbhodi yakho yemuntfu ngamunye yetabelo tekukhonta letitako, kubhaliswa, imicimbi yemacembu, kanye netatiso takho letisha kakhulu. Bona [Likhasi Lami](./me-page).
- **Emacembu** -- Bona futsi uhlangane [nemacembu](../groups/) lawulunga kuwo.
- **Umphakatsi** -- Cwaninga [luhlu lwemalungu](../community/member-directory.md) lelibandla.
- **Tinhlelo** -- Bona [tinhlelo temsebenti](../plans/) letitako kanye netikhundla takho tebuvolontiya lotinikiwe.
- **Kubhalisa Ekufikeni** -- [Bhalisa Ekufikeni](../checkin/) wena nemndeni wakho emitini netimicimbini.
- **Tifundvo** *(seyiyekiwe)* -- I-tab yeTifundvo ngekhatsi kweliphothali seyifakwe endzaweni [ye-FreePlay](/docs/freeplay/) yekudlala kwendzawo yekufundzisa. Cwaninga inhlelo yekufundza ngco ku-[Lessons.church](https://lessons.church) nome bona [likhasi lekubhekiswa kwetifundvo](../content/lessons.md) kutfola imininingwane.
- **Kunikela** -- Yenta [imnikelo](../giving/) nge-inthanethi futsi ubone umlandvo wakho wekunikela.

Chofoza noma ngutiphi i-tab kuze ushintjele kuloyo ngcenye. I-tab lesebentako iyakhonjiswa ngekukhanya kuze uhlale wati kutsi ukuphi.

:::info
Ti-tabs lotibonako tingahluka kuye ngaloko lelibandla lakini selikuvumele. Balawuli belibandla balawula kutsi tingcenye tini letibonakala emalungwini kusebentisa B1 Admin. Nangabe ungayiboni i-tab lotsite, kungenteka lelibandla lakini alikakayivuli leyo mphawu.
:::

## Kuhamba Kwelucingo

Etikhrinini letincane njengematiluphone nemathebulethi, ibha lelisemaceleni liyavaleka kuze kongwe indzawo. Esikhundleni saloko, utawubona inkhinobho ye-**Menu** etulu eliphothalini yemalungu. Yichofote kuze uvule luhlu lwato tonkhe ti-tabs letikhona. Khetsa i-tab kuze uhambe uye kuleyo ngcenye, bese imenyu ivaleka ngekutentela.

## I-Tab Yebalawuli

Nangabe unemvumo yebulawuli elibandleni lakini, utawubona nge-**Admin** tab lengetiwe ephansi kwebha lelisemaceleni. Kuyichofota kukuyisa ku-B1 Admin lapho ungaphatsa khona tilungiselelo nenshila yelibandla lakini.

## Kuhamba Kwehedani Lesetulu

Ngaphandle kweliphothali yemalungu, ihedani lesayithi lelikhulu inetixhumanisi tekuhamba letikhethekile telibandla lakini. Leti tilungiselelwa babalawuli belibandla lakini futsi tingafaka tixhumanisi temakhasi njenge[tintshumayelo](../content/sermons.md), [liBhayibheli](../content/bible.md), [kusakaza ngco](../content/live-streaming.md), nenye inshila. Kulucingo, letixhumanisi tifinyeleleka nge-icon yemenyu ye-hamburger esekoneni lesekudla etulu ehedani.
