# Emacembu

<div class="article-intro">

Emacembu yindlela lenhle yekuxhumana nalabanye ebandleni lakini. Nangabe ufuna licembu lelincane, ithimba yebufundisi, sifundvo seliBhayibheli, nobe litfuba lekutinikela, ungaphequlula futsi ungenele emacembu ngalokucondzile kusuka kusayithi ye-B1.church yelandla lakini.

</div>

## Loko Longakwenta

Naku loko longakwenta ngemacembu ku-B1App:

- **Buka emacembu lavulekile** -- Buka wonkhe emacembu latfolakalako ahleliwe ngetigaba. Licembu ngalinye likhombisa sitfombe, ligama, sikhatsi sekuhlangana, indzawo, kanye nenchazelo lemfishane.
- **Buka imininingwane yelicembu** -- Chofota nawuphi licembu kuze ubone inchazelo yalo lephelele, baholi, imicimbi yekhalendari letako, kanye nefomu yekutsindzabana nebaholi belicembu.
- **Ngenela emacembu** -- Xhumana nelicembu kuze ube lilunga. Emacembu lavulekile akuvumela kungenela ngalokushesha; emacembu ladzinga imvumo akuvumela kwenta sicelo sekungenela. Buka [Kucela Kungenela Licembu](./join-requests.md).
- **Bamba iincenye etingcotweni** -- Nase sewulilunga lelicembu, ungatfumela imilayeto futsi usebentisane nalamanye emalunga elicembu kusetjentiswa ithebhu yeTingcoco.
- **Finyelela emakhalendari netinsita** -- Buka imicimbi letako ekhalendeni yelicembu bese ukhwishisa emafayela lasabelwe nobe emalinki kusuka kuthebhu yeTinsita.

## Kubuka Kwelibanana Neko Longenile

Nome ngubani angavakashela likhasi lemacembu elandla lakini bese aphequlula emacembu latfolakalako. Tihambeli letingakangeni tingabuka tinchazelo temacembu, tibone baholi bakhona, tihlole imicimbi yekhalendari letako, futsi titfumele umlayeto kumholi kusetjentiswa ifomu yekutsindzabana.

Nawu [ngena](../getting-started/logging-in.md), uyavula emathebhu lengetiwe ekhasini lelinemininingwane yelicembu, lokuhlanganisa **Imininingwane Yelicembu**, **Khalendari**, **Tingcoco**, **Tinsita**, **Emalunga**, kanye ne**Kuya Ekhatsi**. Baholi belicembu banemakhono lengetiwe ekuhleleni kuwo onkhe lamathebhu.

:::tip
Emacembu futsi abonakala ku-[Tayimilayini](../community/timeline.md) yakho -- tingcoco netimininingwane letimisha kusuka kumacembu akho titabonakala ku-feed yakho yemsebenti ngalokutentakalelayo.
:::
