# Kuvuma Kuya Emicimbini

<div class="article-intro">

Imicimbi yemacembu ingacoceka imiphendvulo yekutsi ngubani lotako, kuze baholi bati ngubani lotako. Ekhalendeni yelicembu lakho, umcimbi ngamunye lotako uyabuta "Utawuya yini?" — phendvula ngekuchofota, shintsha umcabango wakho nanini, futsi ubone ngekushesha kutsi bangaki labanye labatako.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Kufanele [ube ungenile](../getting-started/logging-in.md) futsi ube lilunga lelicembu
- Umcimbi kufanele uvumele kuphendvula (kuvuliwe ngekwehluleka; baholi bangakuvala umcimbi ngamunye)

</div>

## Kuphendvula Umcimbi

1. Vula licembu lakho kuphothi yemalunga bese uya kuma-Events / kuma-khalendari yalo.
2. Ikhadi lomcimbi ngamunye lotako likhombisa **Utawuya yini?** kanye nenkhinobho letintsatfu: **Ngiyeta**, **Mhlawumbe**, kanye ne**Angeke**.
3. Chofota impendvulo yakho. Lokukhetsile kwakho kuyakhanya, futsi inkhinobho ngayinye ikhombisa inani lelibalwa ngesikhatsi lelikhombisa bangaki labaphendvule ngaleyo ndlela.
4. Chofota lokukhetsile kwakho lokukhanyako futsi kuze ukususe, nobe uchofote lenye inkhinobho kuze ushintshe impendvulo yakho.

Impendvulo yakho isebenta kulomcimbi lotsite kuphela — kumhlangano wangeviki, uphendvula umcimbi weviki ngayinye ngekwehlukana.

## Kubaholi Belicembu

- **Kubuka Timphendvulo** -- Baholi babona inkhinobho yeka **Buka timphendvulo** ekhadini lomcimbi ngamunye. Iyavula liphepha lelithi **Timphendvulo** lelibala bonkhe ngeimpendvulo — Ngiyeta, Mhlawumbe, kanye ne-Angeke. Nangabe kute lophendvule, kubonakala "Akukabi khona timphendvulo."
- **Kuvula nobe Kuvala Kuphendvula** -- Nawakha nobe uhlela umcimbi, sebentisa insimbi yeka **Vumela Kuphendvula**. Imicimbi lengavumeli kuphendvula ayikhombisi tinkhinobho tekuphendvula. Sitafu lesakha imicimbi ngebunyenti kusuka ku-B1 Admin sinalo lelikhetfo lelifananako — buka [Khalendari Yelicembu](../../b1-admin/groups/group-calendar#event-rsvps).

:::tip
Kuphendvula kuyindlela lelula yekubala bantfu. Nangabe umcimbi wakho udzinga imininingwane yekubhalisa, kuphatsa buningi, nobe kukhokha, sebentisa [kubhalisela imicimbi](../events/registering) kunaloko.
:::
