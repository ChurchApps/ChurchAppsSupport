# Kubukela Emacembu

<div class="article-intro">

Likhasi lelivulekile lemacembu elandla lakini lenta kube lula kutfola emacembu latakuhehako. Emacembu ahleliwe ngetigaba, kuze ukhone kubuka ngekushesha tinhlobo letahlukene temacembu bese utfola lelikulungele.

</div>

</div>

## Kutfola Likhasi Lemacembu

Yendlulela ekhasini le-**Emacembu** kusayithi ye-B1.church yelandla lakini. Leli khasi liyafinyeleleka ngalokuphelele, ngako awudzingi kungena kuze ubukele.

## Indlela Emacembu Lahleleke Ngayo

Emacembu aboniswa nge-**sigaba** (lokuphindze kubitwe nge-label). Tigaba tihlelwa bantfu labaphetse libandla lakini futsi kungahlanganisa emagama lafana ne-"Emacembu Lamancane," "Imisebenti Yebufundisi," "Bantfu Labasha," nobe "Kufundza liBhayibheli." Sigaba ngasinye siveta wonkhe emacembu lawakuso.

## Kufundza Emakhadi Emacembu

Licembu ngalinye liboniswa njengekhadi lelinelulwati lolulandzelako:

- **Sitfombe salicembu** -- Sitfombe lesimela licembu. Nangabe kute sitfombe lesibekiwe, kuvela sitfombe lesijwayelekile.
- **Ligama lelicembu** -- Liboniswa ngalokucacile etikwesitfombe.
- **Sikhatsi sekuhlangana** -- Ngabe licembu lihlangana nini ngalokujwayelekile (sibonelo, "NgemaLavia ngu-7 kusihlwa"). Nangabe kute sikhatsi lesibekiwe, kuvela indzawo lengakagcwaliswa.
- **Indzawo yekuhlangana** -- Kuhlangana kuphi. Nangabe kute indzawo lebekiwe, kuvela indzawo lengakagcwaliswa.
- **Inchazelo** -- Incenye lencane lekhombisa kutsi licembu limayelana naliphi, ligcinwe emalayini lambalwa kuphela. Chofota licembu kuze ufundze inchazelo lephelele.

## Kubuka Licembu

Chofota nayiphi ikhadi lelicembu kuze uvule likhasi lelinemininingwane lephelele yalicembu. Lapho utawutfola inchazelo lephelele, imininingwane mayelana nebaholi belicembu, imicimbi letako ekhalendeni, kanye nendlela yekutsindzabana nabo.

:::tip
Nangabe udzinga luhlobo lolutsite lwelicembu, cala ngekubuka emagama etigaba. Loku kukusita kuze uye ngalokushesha encenyeni lekutsandzako kunekutsi uphequlule onkhe emacembu.
:::
