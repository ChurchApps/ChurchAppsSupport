# Kucela Kungenela Licembu

<div class="article-intro">

Lamanye emacembu adzinga imvumo ngembi kwekutsi ube lilunga. Nangabe licembu linalesi setinga, wenta sicelo sekungenela — mhlawumbe uphindze ufake umlayeto kumholi welicembu — bese ulindza kutsi umholi avume nobe angavumi. Ungabuka simo saticelo takho letisalindzile futsi utikhansele nangaso sonkhe sikhatsi.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Kufanele [ube ungenile](../getting-started/logging-in.md) ku-akhawundi yakho ye-B1.church kuze wente sicelo sekungenela licembu.

</div>

## Kungenela Naku Kucela Kungenela

Emacembu angaba nemitsetfo lehlukene yekungena lebekwe bantfu labawaphetse:

- **Livulekile** -- Chofota **Ngenela Licembu** ekhasini lelinemininingwane yelicembu bese wenta ube lilunga ngalokushesha.
- **Kudzingeka sicelo** -- Chofota **Cela Kungenela** kuze wente sicelo lokufanele sivunywe ngumholi welicembu ngembi kwekutsi ube lilunga.
- **Livalekile** -- Kute inkhinobho yekungenela lebonakalako. Emalunga kufanele afakwe ngesandla ngubaholi.

## Indlela Yekwenta Sicelo Sekungenela

1. Phequlula uye elicenjini lofuna kulingenela. Buka [Kubukela Emacembu](./browsing-groups.md) nangabe udzinga lusito ekulitfoleni.
2. Vula likhasi lelinemininingwane yelicembu.
3. Chofota **Cela Kungenela**.
4. Kuvela ibhokisi lapho ungabhala khona umlayeto kumholi welicembu, uchaza kutsi kungani ufuna kungenela.
5. Chofota **Tfumela** kuze utfumele sicelo sakho.

Baholi belicembu batawutfola sexwayiso mayelana nesicelo sakho. Nase basicubungula, utawutfola sexwayiso lesikhombisa umphumela.

:::tip
Kufaka umlayeto lomfishane — njengekutsi wativa njani ngelicembu nobe yini lofuna kuyitfola kilo — kungasita umholi kutsi atsatse sincumo ngekushesha.
:::

## Kubuka Simo Saticelo Takho

Nase sewentile sicelo, ungabuka simo saso ekhasini le-**Emacembu**:

1. Yendlulela ekhasini le-**Emacembu** kusayithi ye-B1.church yelandla lakini.
2. Yehlela phansi uye encenyeni yeka **Ticelo Letisalindzile**.
3. Sicelo ngasinye lesisalindzile sikhombisa ligama lelicembu kanye nelusuku lowasenta ngalo.

Nangabe sicelo sesivunyiwe nobe sesalwe, utawutfola sexwayiso — ticelo letivunyiwe tikungenisa elicenjini njengelilunga.

## Kukhansela Sicelo

Nawushintsha umcabango wakho ngembi kwekutsi umholi acubungule sicelo sakho:

1. Yendlulela ekhasini le-**Emacembu**.
2. Tfola sicelo sakho encenyeni ye-**Ticelo Letisalindzile**.
3. Chofota **Khansela** eceleni kwesicelo.

Sicelo sikhishwa ngalokushesha, futsi umholi angeke asabe asibona.
