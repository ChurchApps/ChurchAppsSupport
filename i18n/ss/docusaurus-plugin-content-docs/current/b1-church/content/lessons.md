# Tifundvo (Sekuyekelwe)

:::danger Lithebhu leTifundvo selentjentiswa yi-FreePlay
Lithebhu le-**Tifundvo** lelifakwe ngekhatsi ku-B1.church selentjentiswa. Kudlalwa kwelucwaningo lolusuka ku-Lessons.church manje kulethwa nge-**[FreePlay](/docs/freeplay/)** — luhlelo lwe-TV lwemahhala lwe-ChurchApps lolwenteliwe emaskrini ekilasi, Fire TV, Apple TV, ne-Android TV — lolunikeza kudlalwa ngaphandle kwe-inthanethi, kuphatselana kwelikilasi, kanye nelivo le-TV-remote lelithebhu lelifakiwe belingeke likwati kulinikeza.

Kute uzulazule, uhlele, kumbe ugucule lucwaningo ngalokucondzile, ngena ku-[Lessons.church](https://lessons.church) ku-browser wakho. Kute udlale tifundvo ku-TV yasekilasini, faka i-[FreePlay](/docs/freeplay/getting-started/) kudivayisi lwakho lwe-TV.
:::

<div class="article-intro">

Lithebhu le-**Tifundvo** belikunika kufinyelela kulucwaningo nakuletinsita tekufundza tebandla lakho. Sicuketfwe lesi besinikelwa ngendlela ye-Lessons.church futsi sifakwe ngokucondzile ngekhatsi ku-B1.church, ngako bewungadzingi kuya kulelinye likhasi.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Kufanele [ube ungenile](../getting-started/logging-in.md) ku-akhawunti yakho ye-B1.church. Lithebhu leTifundvo litfolakala endzaweni yemalunga futsi lidzinga kucinisekiswa.
- Ibandla lakho kufanele libe selihlele lucwaningo nge-Lessons.church.

</div>

## Kufinyelela Etifundvweni

1. Ngena ku-portal yemalunga ye-B1.church.
2. Chofoza lithebhu le-**Tifundvo** [ku-luhlu lwekuzulazula lwemalunga](../getting-started/navigating.md).
3. Sicuketfwe se-Lessons.church sitfwela ngokucondzile ngekhatsi kulelikhasi.

Kungena kwakho kuyachubeka ngekutinhlalisela nawuvula lithebhu leTifundvo. Awudzingi kungena futsi ngekwehlukanako ku-Lessons.church.

## Loko Longakwenta

Nawusekhatsi kulithebhu leTifundvo, ungazulazula lucwaningo ibandla lakho lelenta lube khona. Kuya ngaloko ibandla lakho likuhlelile, ungatfola:

- **Tinhlelo tetifundvo** -- Tilulwane letihleliwe letinetincenye netintfo tekwenta
- **Tethulo nemidiya** -- Ema-slide, emavidiyo, netitfombe letihambisana netifundvo
- **Tinsita letingakhishelwa** -- Emafayela nalokunye lokungengetwa kufundza kabanti

Ungachofoza etincenyeni letitsite ngekhatsi kwesifundvo kute ubone sicuketfwe saso kuliwindi. Sebentisa inkinobho ye-**Emuva** kute ubuyele eluhlwini lwetincenye, kumbe **Vala** kute uphume kuliwindi.

:::tip
Tifundvo letabelwene nemacembu akho futsi titawuvela [ku-Umugca Wesikhatsi](../community/timeline.md) wakho, lokwenta kube lula kutfola lucwaningo lolunikeliwe ngaphandle kwekuya ngqo ethebhini leTifundvo.
:::

## Kutfolakala

Lithebhu leTifundvo livela nangabe ibandla lakho selihlele lucwaningo nge-Lessons.church. Nangabe ungalitfoli lithebhu le-**Tifundvo** eluhlwini lwakho lwekuzulazula lwemalunga, kungenteka ibandla lakho alikafaki sicuketfwe setifundvo. Tsindzana nemphatsi webandla lakho kutfola imininingwane leyengetiwe.
