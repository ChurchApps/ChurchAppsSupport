# B1.church

<div class="article-intro">

I-B1.church yi-portal lebhekene nemalunga ebandla lakho. Yindzawo lapho umphakatsi wakho uya khona kuze uhlale uxhunyiwe, ufinyelele ematulo ebandla, futsi uhlanganyele nemphakatsi wakho ngesikhatsi seviki. Libandla ngalinye litfola i-URL yalo lecacile ku-`https://yourchurchname.b1.church`.

</div>

Nangabe emalunga avakashela isayithi yakho ye-B1.church, abona sayithi yebandla lakho lesobala nge-branding yakho, emalinki e-navigation, nemakhasi ekucuketfwe. Ngemuva kwekungena (logging in), bangakhona kufinyelela ku-**Member Portal** lapho onkhe ematulo abo ebandla ngamunye atfolakala endzaweni yinye.

## Tici Letibalulekile

- **Timeline** -- Umsakazo lohlelwe ngekwemuntu wetibuyeketo, tinkulumo, netento kusuka emacembeni akho nasebandleni.
- **Emacembe (Groups)** -- Buka emacembe lawungewo kuwo, ubone emalunga ecembe, ujoyine tinkulumo, futsi ulandzelele kubakhona.
- **Kunikela (Giving)** -- Wenta iminikelo ebandleni lakho nge-inthanethi futsi ubone umlandvo wemuko wakho.
- **Imicimbi (Events)** -- Bhrawuza imicimbi letako futsi ubhalise wena nemndeni wakho.
- **Kukhonta (Serving)** -- Buka tinhlelo tenkhonzo, tikhundla takho tebuvolontiya, futsi ubhalise emathubeni lavulekile ebuvolontiya.
- **Tinhlelo (Plans)** -- Bona tinhlelo tenkhonzo letitako, ubone tikhundla lokwabelwe wena, futsi ufinyelele imininingwane yetingoma netinhlelo tenkhonzo.
- **Kungena (Check-In)** -- Bhalisa wena nemalunga emndeni wakho ngesikhatsi setinkhonzo netemicimbi. Tivakashi tingabhalisa nge-QR code.
- **Luhlu Lwemphakatsi (Community Directory)** -- Bhrawuza luhlu lwemalunga ebandla lakho, ubone tiphrofayela, futsi utfumele imilayezo lecondzile.
- **Kucuketfwe (Content)** -- Finyelela tishumayelo, fundza liBhayibheli, bukela kusakaza lokuphilako, futsi ubone ivesi lemihla ngemihla, konkhe kusuka esayitini yebandla lakho. (Kudlala kwekharikhulamu manje kutfolakala ku-[FreePlay](/docs/freeplay/) — i-tab ye-Lessons lengekhatsi kwe-portal seyintfo lengasasetjentiswa.)
