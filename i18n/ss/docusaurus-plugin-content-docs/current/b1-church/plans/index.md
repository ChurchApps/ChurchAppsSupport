# Tinhlelo

<div class="article-intro">

Incenye ye-**Tinhlelo** ku-B1.church ikuvumela kubona tinhlelo tenkhonzo lohleliwe kuto. Nome ungathimba lekuculweni, usita ebufundisini bebantfwana, nobe ukhonta kwenye indzawo, lapha ngukuphi lapho ungatfola khona kutsi ubhukiwe nini, ngusikhundla sini losigcwalisako, kanye naloko lekuhlangana kutobonakala njengako.

</div>

## Loko Lotakutfola

Tinhlelo tingahlanganisa tinhlobo letinyenti tekucuketfwe kusuka kutsi libandla lakini litihlela njani:

- **Kwabelwa kwakho** -- Bona kutsi ngutiphi tikhundla lokwabelwe kuto kanye nanini lodzingeka. Kwabelwa ngakunye kukhombisa ligama lesikhundla sakho, lusuku lwenkhonzo, kanye nesimo sakho samanje (Kuvumile, Alile, nobe Akakaphendvuli).
- **Luhlu lwenkhonzo** -- Buka luhlu lwenkhonzo kanye nemininingwane yesikhatsi, imininingwane yengoma, kanye netichazo tetintfo, kuze wati kucondza kwenkhonzo ngembi kwesikhatsi.
- **Kucuketfwe kwesifundvo** -- Letinye tinhlelo tihlanganisa kucuketfwe kwesifundvo nobe kwekufundzisa lokusuka kubaniki-nsita njenge-[Lessons.church](../content/lessons.md). Ungabuka loku kucuketfwe ngalokucondzile ngekhatsi kohlelo.
- **Emalanga Lengekho** -- Yatisa baholi bakho betfimba ngesikhatsi longeko yekusebenta ngekuphatsa emalanga akho lengekho.

## Kucala

Kuze ufinyelele tinhlelo takho, chofota **Tinhlelo** kubha yekuhamba yeceleni yephothi yakho yemalunga. Kufanele [ube ungenile](../getting-started/logging-in.md) kuze ubone tinhlelo takho -- nangabe awukangeni, utawucelwa kutsi ungene kucala.

:::info
Utawubona kuphela tinhlelo lowabelwe kuto. Nangabe awuboni tinhlelo letibalisiwe, kusho kutsi awukafakwa ethimbeni lekukhonta. Khuluma nemphatsi welibandla lakini nobe umholi webufundisi kuze utfole indlela yekubamba incenye.
:::
