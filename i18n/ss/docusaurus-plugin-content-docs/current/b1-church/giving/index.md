# Kunikela

<div class="article-intro">

I-B1App yenta kube lula kusekela libandla lakini ngemnikelo nge-inthanethi. Nome ufuna kwenta sipho lesisheshako sinye nome uphatse kunikela lokuphindzaphindzako, konkhe lokudzingako kutfolakala kusuka esayithini lelibandla lakini le-B1.church.

</div>

## Simo Sonkhe

I-B1App iseketela tindlela letimbili tekunikela:

- **Umnikelo lovulekele wonkhe umuntfu** -- Nome ngubani angenta umnikelo ngaphandle kwekungena. Vakashela likhasi lelibandla lakini lekunikela, khetsa sikhwama, bese ufaka imininingwane yakho yenkhokhelo. Akudzingeki i-akhawunti.
- **Kunikela lokucinisekisiwe** -- Nawu[ngena](../getting-started/logging-in.md) ngaphambi kwekunikela, timipho takho tihlotjaniswa ne-akhawunti yakho. Loku kukunika kufinyelela emlandvweni wakho lophelele wemnikelo, tincwadzi letinganyatselwa tekunikela, kuphatsa kunikela lokuphindzaphindzako, kanye netindlela tekukhokha letigciniwe.

## Loko Longakwenta

Nasi loko lokutfolakalako engcenyeni yekunikela:

- **Nikela ku tikhwama letikhetsekile** -- Lelibandla lakini kungenteka linetikhwama letinyenti letihlelwe, njengeSikhwama Lesijwayelekile, iSikhwama Sekwakha, nome iSikhwama Semisebenti Yebuvangeli. Ungakhetsa kutsi umnikelo wakho ucondziswa kuyiphi sikhwama.
- **Khokha nge likhadi nome i-akhawunti yebhange** -- Imnikelo icutjwa ngekuphepha kusebentisa Stripe. Ungakhokha ngelikhadi lekukweleta nome lekukhokha ngco. Imiyalo yenkhokhelo ye-ACH nayo iyatfolakala.
- **Hlela kunikela lokuphindzaphindzako** -- Emalungu lasengenile angahlela kunikela lokuphindzaphindzako ngekutentela kuze ungaze washiywa ngumnikelo.
- **Bona umlandvo wemnikelo** -- Bona luhlu loluphelele lwemnikelo wakho wangaphambili, kufaka lilanga, indlela, sikhwama, kanye nenani.
- **Nyatselisa tincwadzi tekunikela** -- Khokhisa tincwadzi letinganyatselwa tekunikela tanyaka wamanje nome wangaphambili kwemininingwane yakho yentela.
- **Phatsa tindlela tekukhokha** -- Gcina futsi uphatse tindlela takho tekukhokha kuze kunikela kube ngcono ngesikhatsi lesitako.

:::info
Imnikelo icutjwa ngekuphepha kusebentisa **Stripe**. Imininingwane yakho yenkhokhelo ayikagcinwa kumaseva e-B1.church.
:::
