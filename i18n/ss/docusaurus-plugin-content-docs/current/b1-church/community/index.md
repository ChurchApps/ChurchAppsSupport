# Umphakatsi

<div class="article-intro">

Sigaba se **Umphakatsi** ku-B1.church sikusita kutsi uhlale uxhumene nemndeni wakho webandla. Silethela ndzawonye luhlu lwemalungu lolusita kutfola nekwati ngemalungu lakini ebandleni, kanye ne-Layini Yesikhatsi lekugcina wati ngaloko lokwentekako emacembini akho nasebandleni.

</div>

## Loko Lotawukutfola

- **[Luhlu Lwemalungu](./member-directory.md)** -- Cwaninga futsi ufune eluhlwini lwemalungu ebandla lakho. Funa bantfu ngeligama nome uhlunge ngelicembu kute utfole loyo lomfunako. Chofota kulilunga kute ubone imininingwane yephrofayela yalo.
- **[Layini Yesikhatsi](./timeline.md)** -- I-feed yakho yemsebenti lekhombisa buchwepheshe bekugcina emacembini nasebandleni lakho. Layini Yesikhatsi likhasi lekucala nawuvakashela iphothali yemalungu.

## Kucala

Kufanele ube [ungenile](../getting-started/logging-in.md) kute ukhone kusebentisa tinsita te-Umphakatsi. Nangabe awukangeni, utawucelwa kutsi ungene nawuya ethebhini ye-**Umphakatsi** nome ye-**Layini Yesikhatsi**.

Thebhu ye-**Layini Yesikhatsi** ihlala ikhona ku-sidebar futsi isebenta njengekhasi lekucala lephothali yakho yemalungu. Thebhu ye-**Umphakatsi** inika kufinyelela eluhlwini lwemalungu.

:::info
Emalungu nemininingwane lokubonakala kuwe eluhlwini kuya ngetilungiselelo temfihlo tebandla lakho. Kungenteka ungeke ukhone kubona iphrofayela lephelele yalelo nalelo lilunga. Nangabe unemibuto ngaloko lokwabelwanako, tsintsana nemphatsi webandla lakho.
:::

:::tip
Sebentisa tinsita te-Umphakatsi kanye ne[Emacembu](../groups/) kute uhlale ubambe iciniso ebandleni lakho ngesonto lonkhe.
:::
