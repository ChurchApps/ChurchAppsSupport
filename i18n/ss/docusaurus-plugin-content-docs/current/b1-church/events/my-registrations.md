# Kubhaliswa Kwami

<div class="article-intro">

Likhasi le-**Registrations** eliphothalini yemalungu yindzawo yakho lekhaya kwawo wonkhe umcimbi lobhalisele wona. Kusuka lapha ungahlola simo sakho, ukhokhe imali losele uyonelwe, uhlele kutsi ngubani lotako neloko lokuwodile, nome ukhansele kubhaliswa ngalokuphelele.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Kudzingeka kutsi [ube ungenile](../getting-started/logging-in.md) — kubhaliswa kwetivakashi akuveli lapha
- Kudzingeka kutsi ube nekubhaliswa lokungenani okukodwa. Bona [Kubhalisela Imicimbi](./registering).

</div>

## Kubona Kubhaliswa Kwakho

Vula **Registrations** eliphothalini yemalungu. Kubhaliswa ngakunye kuvela njenge-kadi lekhombisa:

- **Sihloko nelilanga** lemcimbi
- Ipili ye-**status** — **confirmed**, **pending**, **waitlisted**, nome **cancelled**
- **Emalunga** lowabhalisele
- Umucu wenkhokhelo — nome ngukuphi inani lelikhokhiwe, nome **Balance due** nangabe kusesele imali lengakakhokhwa

Nangabe ungakabhaliseli lutfo, utawubona "No registrations yet."

## Kucedzela Inkhokhelo

Inkhinobho ye-**Complete payment** ivela kunome yikuphi kubhaliswa lokusesele nemali lengakakhokhwa — ngalokuvamile ngemuva kwekutsi wenyuselwe kusuka eluhlwini lwekulindza, nome nasebasebenti bahlele intfo letjintje inani lakho lelilonkhe.

1. Chofoza **Complete payment**.
2. Hlola ifingqo yeodolo, khetsa likhadi lelisele ligciniwe nome ufake lelisha, bese ukhokha.
3. Nasekukhokhiwe imali losele uyonelwe, kubhaliswa kuba **confirmed**.

:::tip
Nakuvula lisikhala kuluhlu lwekulindza lokwenzelwa wena, i-imeyili lelibandla lakini lelitfumelako ifaka sixhumanisi lesihamba ngco kulesi sinyatselo senkhokhelo.
:::

## Kuhlela Kubhaliswa

Emahlelo ayashintja. Chofoza **Edit** kubhaliswa kuze uvule ibhokisi le-**Edit Registration**, lapho ungakwati khona:

- Ntjintja **imininingwane yakho yekutsintsana**
- Shintja **libito** kanye **noluhlobo lwesamukeli** lwalilunga ngalinye
- Lungisa **linani lalokukhethiwe** (tintfo letengetiwe njengemahembe nome kudla)

Umkhawulo webungako usasebenta nawuhlela — nangabe luhlobo lwesamukeli nome sikhetho sesiseleteme kusukela ngaleso sikhatsi, utawucelwa kukhetsa lokunye. Kugcina kubala kabusha inani lakho lelilonkhe; nangabe lutjintjo lushiya kusaseleko imali lengakakhokhwa, inkhinobho ye-**Complete payment** iyavela ekhadini.

:::info
Kuhlela akukaze kukhokhise likhadi lakho ngekutentela. Nangabe inani lakho lelisha likhulu kakhulu, khokha umehluko nge-**Complete payment**; nangabe liphansi, tsintsana nelibandla lakini ngembuyiselo wemali — imbuyiselo iphatswa bebasebenti belibandla kusetjentiswa umcutji wabo wenkhokhelo.
:::

## Kukhansela Kubhaliswa

1. Chofoza **Cancel Registration** ekhadini.
2. Cinisekisa ebhokisini — kukhanjelwa akukaze kwatsi kubuyiswe.

Sikhala sakho siyakhululwa ngco. Nangabe umcimbi unelinye luhlu lwekulindza, umuntfu lolandzelako eluhlwini uyanyuselwa ngekutentela. Nome yini lenkhokhelo loyentile ayibuyiselwa ngekutentela — tsintsana nehhovisi lelibandla lakini ngembuyiselo.
