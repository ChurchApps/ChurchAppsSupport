# Kucela Imicimbi Nemakamelo

<div class="article-intro">

Nangabe lelibandla lakini lilandzelela emakamelo netinsiza letingabhukwa, ungacela umcimbi — kanye nemakamelo netintfo letiwudzingako — ngco kusuka eliphothalini yemalungu. Kungqubuzana nemabhukingi lasekhona kuyakhonjiswa njengoba ukhetsa, sicelo sakho siya elibandleni kuze sivunywe, futsi ungasilandzela nome usikhansele kusuka ku-My Requests.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Kudzingeka kutsi [ube ungenile](../getting-started/logging-in.md) eliphothalini yemalungu
- Lelibandla lakini kudzingeka kutsi libe nemakamelo nome tinsiza tihleliwe ku-B1 Admin — tindlela letisheshako tesicelo tivela kuphela nakukhona loku

</div>

## Kutfumela Sicelo

1. Vula likhasi le-**Me** eliphothalini yemalungu bese uchofoza **Request an Event** ngaphansi kwe-Shortcuts.
2. Gcwalisa tisekelo temcimbi: **Sihloko semcimbi**, **Sichasiso** lesingakhetseki, sikhatsi se-**Start** kanye ne-**End** (nome **All day**), kanye nekutsi uyaku**phindza** yini.
3. Khetsa **Emakamelo** ne**Tinsiza** lotidzingako. Kwetinsiza ungabeka **Qty**.
4. Cophela tixwayiso tekungqubuzana — nangabe sikhatsi sakho singenelela kubhukingi lelisele likhona lekamelo nome insiza, utawubona *"This time conflicts with an existing booking."* Ungalungisa sikhatsi sakho nome ukhetse lokunye.
5. Chofoza **Submit Request**.

Ngemuva kwekutfumela, ifingqo ye-**Request submitted** ikhombisa simo sekamelo nome insiza ngayinye loyicelile:

- **Approved** -- Kubhukiwe ngco (likamelo nome insiza akunalo licembu lekuvuma).
- **Pending approval** -- Kudzingeka kutsi umuntfu welibandla avume kucala.

## Kulandzela Titfiso Takho

Vula **My Requests** kusuka ekhasini le-Me. Sicelo ngasinye sikhombisa umcimbi, nini lapho ubucele khona, kanye nepili ye-status:

- **Pending** -- Kulindzelwe sincumo
- **Approved** -- Ibhukingi lakho selicinisekisiwe
- **Rejected** -- Sicelo senqatjiwe

Ngesikhatsi sicelo sisalindzele, ungakwati kusihlehlisa nge-**Cancel request** (utawucelwa kucinisekisa — kukhanjelwa akukaze kwatsi kubuyiswe).

:::info
Ticelo tibuywa bebasebenti belibandla ekhasini le-B1 Admin le-[Approvals](../../b1-admin/calendars/approvals). Kutsi ngutiphi emakamelo netinsiza letidzinga imvumo kuya ngekutsi lelibandla lakini lifuna njani — bona [Emakamelo Netinsiza](../../b1-admin/calendars/rooms-resources).
:::
