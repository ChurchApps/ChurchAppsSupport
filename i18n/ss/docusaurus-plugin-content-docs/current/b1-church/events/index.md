# Imicimbi

<div class="article-intro">

Ingcenye ye-Events ku-B1.church ikuvumela kutsi ucwaninge imicimbi yelibandla letako futsi ubhalisele yona ngco kusuka esayithini. Nawulibandla lakini livumela kubhaliswa kwemcimbi, ungatibhalisela wena futsi wembandzakanye emalunga emndeni ngetinyatselo letimbalwa — futsi kwemicimbi lekhokhelwako, khetsa tinhlobo tesamukeli kanye netikhetho, sebentisa emakhodi ekusaphulelwa, futsi ukhokhe ngekuphepha njengengcenye yekubhalisela.

</div>

## Kucwaninga Imicimbi

Imicimbi ivela ekhalendeni lelibandla. Nawuvula umcimbi lonekubhaliswa lokuvunyiwe, inkhinobho ye-**Register for this Event** ikhonjiswa eceleni kwemininingwane yemcimbi.

Imicimbi lenekubhaliswa lokuvunyiwe ikhombisa:

- **Sihloko nesichasiso semcimbi** -- Kutsi umcimbi umayelana nani.
- **Lilanga nesikhatsi** -- Nini lapho umcimbi wenteka khona.
- **Inkhinobho yekubhalisa** -- Inkhinobho lensundu lekuyisa eliformini lekubhalisa.

## Simo Sekubhaliswa

Imicimbi ingakhombisa simo lesehlukene kuye ngesikhatsi kanye nebungako:

- **Open** -- Kubhaliswa kuyatfolakala futsi ungakwati kubhalisela.
- **Not Yet Open** -- Sikhatsi sekubhaliswa asikakacali. Buya emuva ngemuva kwelilanga lekuvula.
- **Closed** -- Sikhatsi sekugcina sekubhaliswa sesidlulile.
- **Full** -- Tonkhe tikhala letikhona setitsatfiwe.

:::tip
Nangabe umcimbi sewugcwele, kungenteka usakwati kungena eluhlwini **lwekulindza** nangabe lelibandla lakini seluluvumele -- utawutfunyelwa i-imeyili ngekutentela nangabe kuvuleka sikhala. Nangabe akunjalo, buya emuva ngemuva kwesikhashana; tikhala tingavuleka nangabe umuntfu akhansela.
:::

## Loko Lokudzingekako

- I-akhawunti ye-B1.church (nome ungabhalisela njengesivakashi)
- Kufinyelela esayithini le-B1.church lelibandla lakini
