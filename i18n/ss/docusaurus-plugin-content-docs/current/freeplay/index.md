# Sifinyeto se-FreePlay

<div class="article-intro">

FreePlay yi-app ye-TV yemabandla lebonisa kucuketfwe kwetifundzo kuma-TV asemakilasini. Isebenta ku-Android TV, ku-Amazon Fire TV, naku-Apple TV, futsi yakhelwe kusebenta ngisho nangaphandle kwelusix humano lwe-internete ngemakusasa eSonto. Bantfu labativele bangalayisha imibhalo yetifundzo ngesikhatsi seliviki, base bacindzetela inkhinobho yekudlala nakucala ikilasi.

</div>

:::tip Umusha ku-FreePlay? Cala lapha
Buka **<a href="/guides/freeplay-quickstart" target="_blank">vidiyo yetfu yekucala ngekushesha</a>** — faka i-FreePlay bese ucala kudlala kucuketfwe emaminithini lambalwa nje. Akudzingeki bugcisa betekhnoloji.
:::

:::tip Usulungele kuya phambili?
**<a href="/guides/freeplay-b1admin" target="_blank">Umhlahlandlela wetfu lophelele wekusetjenta</a>** ukuhambisa ngekuxhuma i-B1 Admin, kuze sifundzo lesifanele silayishwe ngekutitjintela ku-TV yakho yekilasi njalo ngeliviki. Kufakwe emavidiyo netinyatselo letibhaliwe.
:::

## Timphawu Letibalulekile

- **Kudlala ngaphandle kwe-internete** - Layisha emavidiyo netithombe kusengakadze, kuze ungakaze utsembele ku-Wi-Fi ngesikhatsi setinkonzo
- **Kuphatfwa kwekulayishwa** - Buka futsi uphatse konkhe kucuketfwe lokulayishiwe kusuka kuthebhu yeDownloads leyakhelwe loko
- **Kuhlanganiswa kwelikilasi** - Xhuma i-TV yakho negumbi lelitsite futsi utfole ngekutitjintela kucuketfwe kwesifundzo lesihlelekile
- **Kuhlanganiswa kwelicebo** - Xhuma lidivayisi lakho nelicebo lenkonzo yebandla kuze kube nekucuketfwe lokuhambisana emagunjini onkhe
- **Baniketi bekucuketfwe** - Hlola futsi ulayishe kucuketfwe kusuka etindzaweni tangephandle njenge-[Lessons.church](/docs/lessons-church/) ne-Bible Project
- **Ihambisana ne-remote ye-TV** - Hamba kuyo yonkhe i-app nge-remote yeTV lejwayelekile usebentisa tinkhinobho tenkambo nekukhetsa
- **Incenye yaseceleni lengagoqeka** - Incenye yaseceleni lencane igcina kuhambeka kufinyeleleka ngaphandle kwekugcwalisa sikrini

## Kusebenta Njani

Nawucalisa i-FreePlay, ixhumana nebaniketi bakho bekucuketfwe futsi ihlole kucuketfwe lokukhona. Kusukela lapho ungahlola emafolda, ulayishe umidiya, futsi udlale tifundzo -- konkhe kulungiselelwe kubukela ngesikrini lesikhulu se-TV.
