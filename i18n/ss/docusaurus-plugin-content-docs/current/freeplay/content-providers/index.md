# Baniketi Bekucuketfwe

<div class="article-intro">

Baniketi bekucuketfwe timisela tangephandle letiniketa imibhalo yetifundzo, emavidiyo, netitfombe tekudlala ku-TV yakho. FreePlay isekela baniketi labanyenti, kukuvumela kuhlola futsi ulayishe kucuketfwe kusuka etindzaweni letehlukene, konkhe ku-app yinye.

</div>

## Baniketi Labatfolakalako

FreePlay njengamanje ifaka labafolokati labalandzelako:

- **[Lessons.church](../../lessons-church/)** - Luhlelo lwekufundzisa bantfwana lolumahhala, loluvuliwe (open-source)
- **Bible Project** - Emavidiyo netinsita letihlola liBhayibheli
- **SignPresenter** - Kucuketfwe kwekwetfulwa nekutfumela imiyalo emabandleni
- **B1.church** - Kucuketfwe lokusuka elungiselelweni lwakho lwe-B1 church

:::info
Labanye baniketi basakhelwa. Uma likhadi lemniketi libonisa "Coming Soon," akukafikwa esikhatsini sekuxhumana naye.
:::

## Indlela Baniketi Labasebenta Ngayo

Umniketi ngamunye unenchubomlomo yakhe yekucuketfwe lehlelwe ngemafolda nemafayela. Nawusuxhumene nemniketi, kucuketfwe kwakhe kubonakala esisebentini sekuhlola emafolda lapho ungangena khona emicokwini, ukhetse tintfo, futsi ulayishe kute udlale.

Baniketi labaxhunyiwe babonakala encenyeni yaseceleni ngakwesencele sesikrini. Ungakwati kushintjela emkatsini webaniketi ngekubakhetsa encenyeni yaseceleni, lokwenta kube lula kuyeca emkatsini wetindzawo tekucuketfwe ngesikhatsi selungiselelo.
