# Kuxhumana Nebaniketi

<div class="article-intro">

Ngaphambi kwekutsi uhlole kucuketfwe kusuka kumniketi, udzinga kuxhumana naye. Labanye baniketi badzinga sicinisekiso ngelikhodi le-QR nome kungena nge-imeyili, kutsi labanye bangaxhunywa ngekucindzetela kanye.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Faka bese ucalisa i-FreePlay -- buka [Kucala](../getting-started/)
- Yiba ne-remote yakho ye-TV lulungele kuhamba
- Kubaniketi labadzinga kungena, yiba nemininingwane yakho yeakhawundi itfolakala

</div>

**<a href="/guides/freeplay-b1admin" target="_blank">Umhlahlandlela wetfu wenyatselo ngenyatselo</a>** uhamba ekuxhumeni i-B1 Admin, ekuhleleni sifundzo, nasekuxhumaneni ne-FreePlay — konkhe endzaweni yinye. Vula endzaweni lensha ye-tab kuze ulandzele.
:::

## Kuhlola Baniketi Labatfolakalako

1. Vula sikrini se-Content Providers encenyeni yaseceleni (khetsa siphawu se-Providers ngaphansi)
2. Utawubona luhlu lwemakhadi ebaniketi, ngalinye libonisa lugo neligama lemniketi
3. Baniketi labaxhunyiwe babonisa luphawu lolutima lolutsi **Connected** ngaphansi kweligama labo
4. Baniketi labangakatfolakali babonisa libheji letsi **Coming Soon**

## Kuxhumana Ngaphandle Kwesicinisekiso

Labanye baniketi abadzingi kungena. Nawukhetsa munye walabo baniketi, FreePlay iyaxhuma ngalokushesha futsi ivule sisebenti sekuhlola kucuketfwe. Akukho mininingwane yeakhawundi ledzingekako.

## Sicinisekiso Ngendlela Yelidivayisi (Likhodi le-QR)

Labanye baniketi basebentisa indlela yelidivayisi, lefanana nendlela yekungena kuma-app ekudlala ku-TV:

1. Khetsa likhadi lemniketi esikrinini se-Content Providers
2. FreePlay ibonisa likhodi le-QR ne-URL yesicinisekiso
3. Skena likhodi le-QR ngelusiba lwakho, nome uvakashele i-URL leboniswako kunome ngukuphi lidivayisi
4. Faka likhodi lemsebentisi leliboniswe esikrinini se-TV
5. Cedza inchubo yekungena ngelusiba lwakho nome ngekhompyuta
6. FreePlay ibona kungena lokuphumelele futsi ibonise **Connected!**
7. Sisebenti sekuhlola kucuketfwe siyavuleka ngekutitjintela

:::info
Luphawu lolugidzimako lolutsi **Waiting for authorization** lubonisa kutsi FreePlay ihlola kungena kwakho. Likhodi liphelelwa emuva kwemaminithi lambalwa, ngako-ke cedza inchubo ngalokushesha.
:::

## Kungena Ngefomu

Labanye baniketi basebentisa indlela lejwayelekile ye-imeyili nelipasiwedi:

1. Khetsa likhadi lemniketi
2. Faka i-**Email** ne-**Password** yakho usebentisa i-khibhodi lesesikrinini
3. Khetsa inkhinobho ye-**Sign In**
4. Uma imininingwane yakho ilungile, FreePlay ibonisa **Connected!** futsi ivule sisebenti sekuhlola kucuketfwe

:::tip
Sebentisa i-pad yenkambo ku-remote yakho kuze uhambe emkatsini wensimu ye-imeyili, insimu yelipasiwedi, nenkhinobho yekungena. Cindzetela **Select** ensimini yembhalo kuze uvule i-khibhodi lesesikrinini.
:::

## Kuxhumana Ngaphandle Nemniketi

Kuxhumana ngaphandle nemniketi losewuxhunywe naye:

1. Hamba esikrinini se-Content Providers
2. Khetsa likhadi lemniketi leliboniswa luphawu lolutsi **Connected**
3. Sicelo sekucinisekisa siyabuta kutsi ufuna yini kuxhumana ngaphandle
4. Khetsa **Disconnect** kuze ususe luxhumano

Ngemuva kwekuxhumana ngaphandle, kucuketfwe kwemniketi angeke kusaba kubonakala encenyeni yaseceleni.

:::warning
Kuxhumana ngaphandle kususa sicinisekiso lesigciniwe kulidivayisi lakho. Utawudzinga kungena kabusha uma ufisa kuxhumana kabusha ngesikhatsi lesitako.
:::
