# Simo Selikilasi

<div class="article-intro">

Simo Selikilasi sixhuma TV yakho negumbi lelitsite ebandleni lakho. Ngemuva kwekuhlanganiswa, FreePlay iyalayisha ngekutitjintela sifundzo lesihlelekile kulelogumbi, kuze bantfu labativele bakwati nje kucindzetela **Start Lesson** nakucala ikilasi.

</div>

## Kusebenta Njani

1. **Tfola libandla lakho** - Sesha libandla lakho ngeligama
2. **Khetsa ligumbi** - Khetsa likilasi le-TV le ingelilo
3. **Layisha sifundzo** - FreePlay ilanda sifundzo lesihlelekile futsi ilayishe onkhe emafayela emidiya
4. **Dlala** - Cindzetela inkhinobho ye-Start Lesson bese sifundzo sidlala ngekutitjintela

I-TV iyakhumbula kutsi lihlanganiswe naliphi libandla neligumbi, ngako-ke udzinga kuchubeka nekulungiselela kanye kuphela. Liviki ngalinye, FreePlay ihlola sifundzo lesihlelekile selilanga lanamuhla bese ilayisha nome ngukuphi kucuketfwe lokusha.

:::tip
Cinisekisa kutsi emakilasi akho alungiselelwe ku-[freeplay.church](https://freeplay.church) ngaphambi kwekuhlanganisa i-TV yakho. Uma kute emakilasi labonakalako nawukhetsa libandla lakho, vakashela iwebhusayithi kuze uwalungiselele.
:::
