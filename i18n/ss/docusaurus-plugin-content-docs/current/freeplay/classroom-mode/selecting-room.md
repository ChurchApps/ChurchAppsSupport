# Kukhetsa Libandla Neligumbi Lakho

<div class="article-intro">

Ngaphambi kwekutsi FreePlay ikwati kulayisha tifundzo telikilasi lakho, udzinga kuyitjela kutsi ngutiphi libandla neligumbi lekusetjentiswa. Loku kulungiselelo lolwentiwa kanye kuphela -- i-TV iyakhumbula lokukhetsile kwakho kuletinye tikhatsi letitako.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Faka bese ucalisa i-FreePlay -- buka [Kucala](../getting-started/)
- Bhalisa libandla lakho futsi ulungiselele emakilasi ku-[freeplay.church](https://freeplay.church)
- Khetsa **Pair to Classroom** kusuka [esikrinini sesimo sekuhlanganiswa](../getting-started/pairing-modes.md)

</div>

## Kutfola Libandla Lakho

1. Kusuka esikrinini se-Select Pairing Mode, khetsa **Pair to Classroom**
2. Utawubona sikrini se-Find Your Church, sinesikhala sekusesha etulu
3. Usebentisa i-remote yakho ye-TV, khetsa sikhala sekusesha bese ubhala tinhlamvu letine tebuncane teligama lebandla lakho
4. Luhlu lwemabandla lafananako luyabonakala ngaphansi kwesikhala sekusesha
5. Hamba uye ebandleni lakho eluhlwini bese ucindzetela **Select** ku-remote yakho

:::warning
Uma libandla lakho lingabonakali emiphumeleni yekusesha, kungenteka kutsi likakabhaliswa. Vakashela [freeplay.church](https://freeplay.church) kuze ubhalise libandla lakho futsi ulungiselele emakilasi akho.
:::

## Kukhetsa Likilasi

Ngemuva kwekukhetsa libandla lakho, sikrini se-Select a Classroom sibonakala sinaluhlu lwawo onkhe emagumbi lalungiselelwe.

1. Sebentisa titfwebula tenkambo letiya etulu naphansi ku-remote yakho kuze uhlole luhlu lwemakilasi
2. Khombisa ligumbi lelinikwe le-TV
3. Cindzetela **Select** ku-remote yakho

FreePlay igcina lokukhetsile kwakho futsi ihambe ngalokushesha esikrinini sekulayisha kuze ilande sifundzo lesihlelekile.

:::tip
Uma ubona "No classrooms found," emakilasi elibandla lakho akakalungiselelwa. Umphatsi angawalungiselela ku-[freeplay.church](https://freeplay.church).
:::

## Kushintja Lokukhetsile Kwakho

Kuze ushintjele kuleliny libandla nome ligumbi ngaleso sikhatsi, cindzetela inkhinobho **Back** ku-remote yakho ye-TV kusuka esikrinini sekulayisha. Loku kukubuyisela esikrinini sekukhetsa ligumbi lapho ungakhetsa khona lelinye likilasi, kumbe uchubeke ucindzetela **Back** kuze ubuyele ekuseshweni kwelibandla.
