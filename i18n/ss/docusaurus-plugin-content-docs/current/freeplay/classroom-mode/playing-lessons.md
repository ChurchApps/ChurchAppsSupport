# Kudlalwa Kwetifundzo

<div class="article-intro">

Umdlali wemidiya we-FreePlay kungumhlangano logcwele sikrini lowakhelwe ema-remote e-TV. Udlala emavidiyo netitfombe letibonisako, uhamba ekucuketfweni kwesifundzo ngekutitjintela nome ngesandla, kukhipha bafundzisi emandla lagcwele ekulawuleni lijubane lesifundzo sabo.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Layisha sifundzo -- buka [Kulayishwa Kwekucuketfwe Kwesifundzo](./downloading-content)
- Cindzetela inkhinobho ye-**Start Lesson** kuze ungene emdlalini

</div>

## Kulawulwa Kwemdlali

Sebentisa i-remote yakho ye-TV kuze ulawule kudlala:

| Inkhinobho ye-Remote | Sento |
|---|---|
| **Select / Play-Pause** | Misa nome cala kabusha ividiyo yamanje |
| **Umtimba longakwesekudla / Fast Forward** | Yeca uye kusitfombe nome ividiyo lelandzelako |
| **Umtimba longakwesencele / Rewind** | Buyela kusitfombe nome ividiyo lengaphambili |
| **Umtimba longetulu** | Vula sikhetsi semilayeto kuze uyeke enteweni letsite |
| **Ngaphansi / Back / Menu** | Phuma emdlalini bese ubuyela esikrinini lesandzulele |

:::tip
Ungakwati futsi kususha ema-remote lasebenta ngekutsintsa. Susha ngakwesencele kuze uye phambili, susha ngakwesekudla kuze ubuyele emuva, susha phansi kuze uvule sikhetsi semilayeto, susha etulu kuze uphume emdlalini.
:::

## Kudlala Kwevidiyo

Nawudlala ividiyo, utawubona konkhe kucuketfwe kwevidiyo kugcwele sikrini. Kucindzetela **Select** nome **Play-Pause** kumisa ividiyo futsi kubonise siphawu sekumisa nemudla wenchubo ngaphansi kwesikrini. Cindzetela futsi kuze uchubeke.

Emavidiyo ahamba ngekutitjintela aye enteweni lelandzelako esifundzweni nawaphela. Uma ividiyo ilungiselelwe kuphindzaphindza, iyachubeka kuze uyihambise ngesandla.

## Titfombe Letibonisako

Titfombe letibonisako tibonakala tigcwele sikrini. Ngeklulu kwemavidiyo, titfombe letibonisako atihambi ngekutitjintela -- tihlala esikrinini kuze ucindzetele umtimba longakwesekudla kuze uhambe phambili. Loku kunika bafundzisi emandla lagcwele ekulawuleni lijubane.

## Sikhetsi Semilayeto

Cindzetela umtimba **longetulu** ngesikhatsi sekudlala kuze uvule sikhetsi semilayeto. Loku kubonisa luhlu lwato tonkhe tintfo tesifundzo tinetitfombe letincane nemagama. Hamba kunome ngukuphi intfo bese ucindzetela **Select** kuze uye kuyo ngco.  Loku kuyasita ekuyecweni phambili nome kubuyeleni encenyeni letsite yesifundzo.

## Kuphuma Emdlalini

Cindzetela **Ngaphansi**, **Back**, nome **Menu** ku-remote yakho kuze uphume emdlalini. Utawubuyela esikrinini sekulayisha (kuSimo Selikilasi), esikrinini sekulayisha kwelicebo (kuSimo Selicebo), nome kusisebenti sekuhlola kucuketfwe (nawudlala kucuketfwe kwemniketi).

:::info
Sikrini sihlala sivukile ngesikhatsi umdlali usebentako, ngako-ke akudzingi kukhatsateka kutsi TV iyolala ngesikhatsi sesifundzo.
:::
