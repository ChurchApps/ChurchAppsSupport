# Kulayishwa Kwekucuketfwe Kwesifundzo

<div class="article-intro">

Nawusukhetsile libandla nelikilasi, FreePlay iyalanda ngekutitjintela sifundzo lesihlelekile lamuhla futsi icale kulayisha onkhe emafayela awo emidiya. Lesi sikrini kulapho ulinda khona kucuketfwe kulungele ngaphambi kwekucala sifundzo.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Khetsa libandla neligumbi lakho -- buka [Kukhetsa Libandla Neligumbi Lakho](./selecting-room)
- Cinisekisa kutsi TV yakho ixhunywe ku-internete kulokulayishwa kwekucala

</div>

## Indlela Lesebenta Ngayo Kulayisha

1. FreePlay ihlola hlelo lwesikhatsi selikilasi lakho futsi ilande sifundzo lesinikiwe
2. Ligama lesifundzo, sihloko, nencazelo kubonakala esikrinini, kwembelwe etulu kwesitfombe sesembozo sesifundzo
3. Luphawu lwenchubo lubonisa **Downloading item X of Y** njengoba lifayela ngalinye ligcinwa kulidivayisi
4. Nawo onkhe emafayela sewalayishiwe, inkhinobho ye-**Start Lesson** iyabonakala nesiphawu sekudlala

Akudzingeki wente lutfo ngesikhatsi sekulayishwa. FreePlay iyakuphatsa konkhe ngekutitjintela.

## Kucaca (Resolution)

FreePlay ilayisha kucuketfwe ku-720p ngekwemvamisa. Kulungiselelwa kwekucaca (720p nome 1080p) kunquma buhle bemafayela emavidiyo netithombe lalayishiwe. Kucaca lokuphakeme kubukeka kuhle kakhulu kuma-TV lamakhulu kodvwa kudzinga sikhatsi lesengeti kuze kulayishwe futsi kusebentisa indzawo lengeti yekulondvolota.

:::info
Kucaca kulungiselelwe ku-720p ngekwemvamisa. Kucuketfwe kulayishwa ngekucaca lokulungiselelwe kulidivayisi lakho.
:::

## Kucala Sifundzo

Nawuseligcine kulayishwa, inkhinobho ye-**Start Lesson** ivela ngesikhashana ngaphansi kwesikrini. Sebentisa i-remote yakho ye-TV kuze uyikhombise bese ucindzetela **Select** kuze kucale kudlalwa.

## Kuvuselelwa Ngekutitjintela

FreePlay ihlola kucuketfwe lokubuyekeziwe njalo ngelihora. Uma hlelo lwesikhatsi lishintja ngesikhatsi selilanga, i-app itfola sifundzo lesisha ngekutitjintela. Ungakwati futsi kucindzetela **Back** ku-remote yakho kuze ubuyele esikrinini sekukhetsa ligumbi bese ungena futsi esikrinini sekulayisha kuze ubike kutsi kubuyekeziwe.

:::tip
Layisha tifundzo ngaphambi kwekusa kweliSonto, ngesikhatsi lesakho sinelusix humano lwe-internete loluchubekile. Nasekulayishiwe, kucuketfwe kudlala ngaphandle kwe-internete ngaphandle kwekutsembela nome ngukuphi enethiwekini.
:::

## Thebhu ye-Downloads

FreePlay ifaka thebhu ye-**Downloads** lapho ungabona futsi uphatse khona konkhe kucuketfwe lokulayishiwe kulidivayisi. Loku kukupha kubona kutsi yini legciniwe endzaweni futsi ilungele kudlala ngaphandle kwe-internete. Sebentisa thebhu ye-Downloads kuze uchekhe kutsi tifundzo tilayishwe ngalokuphelele ngaphambi kweliSonto ekuseni, nome kuze ukhulule indzawo yekulondvolota ngekususa kucuketfwe lokudzala.

## Uma Hlelo Lwesikhatsi Lungenalutfo

Uma kute sifundzo lesihlelekile likilasi lelikhetsiwe, utawubona umlayeto lotsi "The schedule could not be loaded. Make sure a lesson is scheduled for this class." Chekha hlelo lwakho lwesikhatsi ku-[freeplay.church](https://freeplay.church) futsi ucinisekise kutsi sifundzo lesifanele sinikwe lelogumbi.
