# Kukhetsa Simo Sekuhlanganiswa

<div class="article-intro">

FreePlay iniketa timo letimbili tekuhlanganisa kuze uxhume TV yakho nebandla lakho, kanye nemandla ekuhlola baniketi bekucuketfwe ngco. Sikrini se-Select Pairing Mode sethula letinketfo njengemakhadi lamakhulu longahamba kuwo nge-remote yakho ye-TV.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Faka i-FreePlay futsi uyicalise kwekucala -- buka [Kucala](./index.md)
- Yiba ne-remote yakho ye-TV lulungele kuhamba

</div>

## Kuhlanganiswa Nelikilasi

Khetsa **Pair to Classroom** nawufuna kutsi TV letsite ihlale ibonisa sifundzo lesihlelekile ligumbi lelitsite. Loku kuyintfo lengcono kakhulu emakilasini enkonzo yebantfwana lapho tifundzo tinikwa khona ngehlelo lweliviki.

**Indlela lesebenta ngayo:** Usesha libandla lakho ngeligama, ukhetse likilasi, futsi FreePlay ilande ngekutitjintela nome ngutsiphi sifundzo lesihlelekile lelogumbi lamuhla. Ngelisonto ngalinye, uvula nje TV bese sifundzo lesifanele silungile.

**Ikahle kakhulu ye:** Ema-TV emakilasi lahlala enikwe kutsi ahlale abonisa kucuketfwe lokuhlelekile kwaloo ligumbi.

:::tip
Lungiselela emakilasi akho ku-[freeplay.church](https://freeplay.church) ngaphambi kwekuhlanganisa TV yakho. Loku kucinisekisa kutsi emagumbi akho ayabonakala nawusesha libandla lakho.
:::

## Kuhlanganiswa Nelicebo

Khetsa **Pair to Plan** nawufuna kulandzela icebo lenkonzo yebandla. Lesi simo sikhipha likhodi lekuhlanganisa lofaka ku-**B1 Admin** kuze uxhume TV nenhlobo letsite yelicebo. TV iyabese ilayisha futsi idlale kucuketfwe lokuniketwe kulelo cebo lamanje.

**Indlela lesebenta ngayo:** FreePlay ibonisa likhodi lekuhlanganisa esikrinini. Umphatsi ufaka lelokhodi etilungiselelweni te-**Plan Type** nge-**B1 Admin**. Nasekuhlanganisiwe, TV iyalayisha ngekutitjintela kucuketfwe kwelicebo lamanje.

**Ikahle kakhulu ye:** Ema-TV labonisa kucuketfwe lokuhlotjaniswa nelicebo lakho lenkonzo lweliviki, njenga emagama etinkondlo kumbe tibonelo temitsakato.

## Baniketi Bekucuketfwe

Uma ungadzingi kuhlanganiswa nelikilasi nome nelicebo, ungakwati kuhlola baniketi bekucuketfwe ngco. Baniketi labafana ne-[Lessons.church](../../lessons-church/) ne-Bible Project baniketa kucuketfwe longakulayisha bese uwudlala nome nini.

**Ikahle kakhulu ye:** Ema-TV lapho bantfu labativele bakhetsa ngesandla yini letawudlalwa liviki ngalinye, nome nawufuna kuhlola kucuketfwe lokutfolakalako.

:::info
Ungafinyelela esikrinini se-**Providers** nome ngasiphi sikhatsi kusuka encenyeni yaseceleni ngaphansi kwekuhamba. Awudzingi kusebentisa simo sekuhlanganiswa kuze uhlole futsi udlale kucuketfwe.
:::
