# Kucala Ngekusebentisa i-FreePlay

<div class="article-intro">

FreePlay iyatfolakala ku-Android TV, Amazon Fire TV, nase-Apple TV. Lomhlahlandlela ukuhambisa ekufakeni i-app, ekuyicaliseni ngekucala, nasekufundzeni kulawulwa lokusisiseko kwe-remote ye-TV lotawukusebentisa kuyo yonkhe i-app.

</div>

:::

## Kufaka i-FreePlay

### Android TV / Amazon Fire TV

1. Vula sitolo se-app ku-TV yakho (**Google Play Store** nome **Amazon Appstore**)
2. Sesha **FreePlay**
3. Khetsa i-app bese ukhetsa **Install**

### Apple TV

1. Vula i-**App Store** ku-Apple TV yakho
2. Sesha **FreePlay**
3. Khetsa i-app bese ukhetsa **Get**

## Kucalisa I-App

Nawuvula i-FreePlay kwekucala, utawubona sikrini lesihamba sinemswenya we-FreePlay. I-app ihlola nome ngukuphi baniketi bekucuketfwe labaxhunywe ngaphambilini. Uma sewuke waxhumana nemniketi, itawukuyisa ngco esisebentini sekuhlola kucuketfwe salowo mniketi. Ngaphandle kwaloko, itawuvula sikrini se-Content Providers lapho ungaxhumana khona nesisu sakho sekucala sekucuketfwe.

## Kuhamba Nge-remote Yakho Ye-TV

FreePlay yakhelwe ema-remote e-TV. Nali lokulawulwa lotawukusebentisa kuyo yonkhe i-app:

- **I-pad yenkambo (etulu/phansi/kwesencele/kwesekudla)** - Hamba emkatsini wetintfo esikrinini
- **Inkhinobho ye-Select / OK** - Khetsa intfo lekhonjisiwe
- **Umtimba longakwesencele** - Vula incenye yaseceleni kusuka kunome ngukuphi sikrini
- **Umtimba longakwesekudla** - Vala incenye yaseceleni bese ubuyela endzaweni lenkhulu yekucuketfwe
- **Inkhinobho ye-Back / Menu** - Buyela esikrinini lesandzulele

:::tip
Incenye yaseceleni iyagoqeka ibe siphawu lesincane nawuhlola kucuketfwe. Cindzetela umtimba longakwesencele ku-remote yakho kuze uyandzise futsi ubone onkhe emagama lagcwele.
:::
