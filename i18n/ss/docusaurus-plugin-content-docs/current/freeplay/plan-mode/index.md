# Simo Selicebo

<div class="article-intro">

Simo Selicebo sixhuma TV yakho nelicebo lenkonzo yebandla. Esikhundleni sekulanda kucuketfwe ngekusekelwa ehlelweni lwesikhatsi selikilasi, silandzela icebo lelilungiselelwe bucwephesha bebandla lakho ku-B1 Admin. Loku kuhle nawufuna kutsi TV ibonise ngco loko lokunikwe elicebweni leliviki lamanje.

</div>

## Indlela Lehlukene Ngayo Ku-Simo Selikilasi

- **[Simo Selikilasi](../classroom-mode/)** ilanda sifundzo lesihlelekile ligumbi lelitsite ngelilanga lelinikiwe
- **Simo Selicebo** silanda kucuketfwe lokuhlotjaniswa nenhlobo yelicebo, lokungafaka tifundzo, kwetfulwa kwebaniketi, tintfo letengetiwe, naleminye imidiya lehleliwe ngekulandzelana lokwakhelwe

Simo Selicebo sisita kakhulu nawe icebo lakho lenkonzo lifaka kuhlangana kwekucuketfwe lokusuka etindzaweni letehlukene, konkhe kuhlelwe kube kulandzelana kunye.

## Kusebenta Njani

1. FreePlay ikhipha likhodi lekuhlanganisa esikrinini
2. Umphatsi ufaka lelokhodi ku-**B1 Admin** ngaphansi kwetilungiselelo tenhlobo yelicebo
3. Nasekuhlanganisiwe, FreePlay ilayisha icebo lamanje futsi ilayishe wonkhe umidiya lohlotjaniswa nalo
4. Cindzetela **Start Plan** kuze kucale kudlalwa

TV ihlala ihlanganisiwe nenhlobo yelicebo. Liviki ngalinye, ilanda ngekutitjintela icebo lelisha nekucuketfwe kwalo.
