# Kuhlanganiswa Kwelidivayisi Lakho

<div class="article-intro">

Kuze usebentise Simo Selicebo, udzinga kuhlanganisa TV yakho nenhlobo yelicebo ku-B1 Admin. FreePlay ikhipha likhodi lekuhlanganisa lelikhetsekile lelixhuma lidivayisi nelicebo lebandla lakho, kuvumele kuniketwa kwekucuketfwe ngekutitjintela liviki ngalinye.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Faka bese ucalisa i-FreePlay -- buka [Kucala](../getting-started/)
- Khetsa **Pair to Plan** kusuka [esikrinini sesimo sekuhlanganiswa](../getting-started/pairing-modes.md)
- Yiba nemfinyelelo ku-**B1 Admin** ngekhompyuta nome ngelusiba kuze ucedze kuhlanganiswa

</div>

## Kukhipha Likhodi Lekuhlanganisa

1. Kusuka esikrinini se-Select Pairing Mode, khetsa **Pair to Plan**
2. FreePlay ixhumana neseva futsi ikhipha likhodi lekuhlanganisa lelifisha
3. Likhodi libonakala ngetinhlamvu letinkhulu emkhatsini wesikrini
4. Ngaphansi kwelikhodi, luphawu lolugidzimako lubonisa **Waiting for connection**

Likhodi liboniswa njengetinhlamvu letodvwa kuze kubaluleke kufundza kusuka kudze endlini.

## Kufaka Likhodi ku-B1 Admin

1. Ngekhompyuta nome ngelusiba, vula **B1 Admin** yebandla lakho
2. Hamba uye etilungiselelweni te-**Plan Type** telicebo lofuna kulinika
3. Faka likhodi lekuhlanganisa leliboniswa ku-TV
4. Gcina tilungiselelo

FreePlay ibuta seva njalo emaminithini lambalwa, ihlola kutsi kuhlanganiswa sekuphelele yini. Nase-**B1 Admin** icinisekisa luxhumano, TV iyahamba ngekutitjintela iye esikrinini sekulayisha kwelicebo.

## Kulayishwa Kwekucuketfwe Kwelicebo

Ngemuva kwekuhlanganiswa, FreePlay ilayisha icebo lamanje lalelo nhlobo yelicebo. Ibonisa:

- Ligama lelicebo nelilanga lenkonzo
- Ligama lesifundzo lelihambisanako (uma icebo lifaka kucuketfwe kwesifundzo)
- Luphawu lwenchubo lolubonisa **Downloading item X of Y**

Nawo onkhe emafayela emidiya sewalayishiwe, inkhinobho ye-**Start Plan** iyabonakala. Cindzetela **Select** ku-remote yakho kuze kucale kudlalwa.

:::tip
Icebo liyavuseleleka ngekutitjintela njalo ngelihora. Uma icebo libuyekeziwe ngesikhatsi selilanga, FreePlay ilanda tishintjo ngaphandle kwekungenelela ngesandla.
:::

## Lenye Indlela: Kusesha Ngeligama Lebandla

Uma ungatsandzi kusebentisa inchubo yelikhodi lekuhlanganisa, ungakhetsa **or search by church name** ngaphansi kwesikrini sekuhlanganisa. Loku kukuyisa esikrinini sekusesha kwebandla lapho ungatfola khona libandla lakho bese uxhumana nelikilasi esikhundleni saloko.

## Uma Kuhlanganiswa Kungaphumeleli

Uma likhodi lekuhlanganisa lingakwati kukhishwa (sibonelo, ngenca yenkinga yenethiwekhi), utawubona umlayeto wesiphosiso nenkhinobho ye-**Try Again**. Cinisekisa kutsi TV yakho ixhunywe ku-internete bese uzama futsi.

:::warning
Emakhodi ekuhlanganisa aphelelwa ngemuva kwesikhatsi lesibekiwe. Uma ulinde sikhatsi lesidze kakhulu, khipha likhodi lelisha ngekubuyela esikrinini sekuhlanganisa.
:::
