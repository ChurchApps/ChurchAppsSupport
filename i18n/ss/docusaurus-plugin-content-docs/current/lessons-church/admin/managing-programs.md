# Kuphatsa Tinhlelo

<div class="article-intro">

Tinhlelo tiwutakhiwo letiphakeme kakhulu tekufundzisa ku-Lessons.church. Luhlelo lumele sicu sonkhe sekufundzisa -- sibonelo, "Ark Kids" nome "Youth Foundations." Lomculo ubeka kwakha, kuhlela, nekuhlela tinhlelo endzaweni yekuphatsa.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- Iba nemvumo yekuphatsa ye-Lessons.church (bona [Sifingqo Sekuphatsa](./index.md))
- Lungisela ligama lelinhlelo, incazelo, nesitfombe lesincane

</div>

## Kwakha Luhlelo

1. Yaphuma uye endzaweni ye-**Admin**.
2. Chofota **Add Program**.
3. Gcwalisa emabala lalandzelako:
   - **Name** -- Ligama lelibonakalako lelinhlelo (sib., "Ark Kids").
   - **Slug** -- Sikhombisi lesihambisana ne-URL (sib., "ark-kids"). Lokhu kusetjentiswa emakhelweni ewebhu.
   - **Description** -- Sifingqo lesifushane sekutsi luhlelo lumbonisa ini.
   - **Image** -- Layisha sitfombe lesincane lesimela luhlelo.
4. Gcina luhlelo.

Luhlelo lwakho lolusha manje seluzawuvela ebuluhlwini bemahlelo endzaweni yekuphatsa, futsi, nasekunetifundvo netifundzo, esiteni lesibukwa ngumphakatsi.

## Kuhlela Luhlelo

1. Chofota ligama lelinhlelo ebuluhlwini bekuphatsa.
2. Vuselela nawuphi emabala (**Name**, **Slug**, **Description**, nome **Image**).
3. Gcina lokuguculiwe kwakho.

## Kuhlela Tifundvo Ngekhatsi Kwelinhlelo

Nasekunelinhlelo, ungengeta tifundvo kulo. Tifundvo tivela tibonakala ngephasi kwelinhlelo endzaweni yekuphatsa. Chofota lisikhombisi lekuvula eceleni kwelinhlelo kuze ubone tifundvo talo, nome wengete lesisha lapho. Bona [Kuphatsa Tifundvo](./managing-studies.md) kutfola imininingwane.

## Tinhlobo Tetifundvo

Nangabe luhlelo lwakho lunetifundvo letinyenti, ungatihlela tibe **tinhlobo** (categories). Tinhlobo tihlanganisa tifundvo ngesihloko, ngeliciniso lemadvuna, nome nome ngukuphi lokunye lokuzikhetsela. Loku kwenta kubalula kubantfu kuphequlula luhlelo loluludzala.

## Kulayisha Tinsizakalo Temtindze Welinhlelo

Ungakwati kunamatsela emafayela langalandwa elinhlelweni. Leti tinsizakalo letisebenta luhlelo lonkhe kunekutsi tisebente sifundvo nome sifundzo lesitsite -- sibonelo, umbhalo wekuhlelwa nekulandzelana, nome incwadzi yemsiti.

1. Vula luhlelo endzaweni yekuphatsa.
2. Yaphuma uye kulingeniso le-**Resources**.
3. Layisha emafayela akho.

## Kususa i-Cache

Ngemuva kwekwenta lokuguculiwe elinhlelweni, kungadzingeka kususe i-cache kuze lokuguculiwe kubonakale esiteni lesibukwa ngumphakatsi. Sebentisa lokukhetsa **Clear Cache** endzaweni yekuphatsa kuze uvuselele kontenti.

:::tip
Khetsa ligama lelinhlelo leliluchaza kahle bese ubhala incazelo lecacile. Loku kuyintfo yekucala emabandla layibonako nawaphequlula umtapo wekufundzisa, ngako-ke sifingqo lesihle sitasita bacoshise kutsi luhlelo lufanele tidzingo tabo yini.
:::
