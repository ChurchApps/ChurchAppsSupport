# Kuphatsa Tifundvo

<div class="article-intro">

Sifundvo sitichungechunge tetifundzo letihlobene ngekhatsi kwelinhlelo. Sibonelo, tichungechunge temaviki lamane letibitwa "The Good Samaritan" bekungaba sifundvo ngekhatsi kwelinhlelo lelikhulu lenkonzo yebantfwana. Lomculo ubeka kwakha, kuhlela, nekuhlela tifundvo endzaweni yekuphatsa.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- [Yakha luhlelo lulodvwa kungenani](./managing-programs.md) endzaweni yekuphatsa
- Iba nemvumo yekuphatsa ye-Lessons.church (bona [Sifingqo Sekuphatsa](./index.md))

</div>

## Kwakha Sifundvo

1. Yaphuma uye endzaweni ye-**Admin** bese wandisa luhlelo lofuna kwengeta sifundvo kulo.
2. Chofota **Add Study**.
3. Gcwalisa emabala lalandzelako:
   - **Name** -- Ligama lelibonakalako lesifundvo (sib., "The Good Samaritan").
   - **Slug** -- Sikhombisi lesihambisana ne-URL (sib., "the-good-samaritan").
   - **Description** -- Sifingqo sekutsi sifundvo lesi simbonisa ini nekutsi sinetifundzo letingaki.
   - **Image** -- Layisha sitfombe lesincane sesifundvo.
4. Gcina sifundvo.

## Kuhlela Sifundvo

1. Chofota ligama lesifundvo ebuluhlwini bekuphatsa.
2. Vuselela nawuphi emabala njengobe kudzingeka.
3. Gcina lokuguculiwe kwakho.

## Kwabela Luhlobo

Nangabe luhlelo loluphetsele lusebentisa tinhlobo tetifundvo, ungakwati kwabela sifundvo sakho kulunye:

1. Vula sifundvo endzaweni yekuphatsa.
2. Khetsa **Category** kudropdown.
3. Gcina.

Tinhlobo tisita bantfu baphequlule tinhlelo letinetifundvo letinyenti. Sibonelo, ungaba netinhlobo letinjenge "Testamente Lelidzala", "Testamente Lelisha", ne"Ngesihloko".

## Kuphatsa Tifundzo Ngekhatsi Kwesifundvo

Tifundvo tiba nesifundzo lesisodvwa nome ngetulu. Yandisa sifundvo endzaweni yekuphatsa kuze ubone tifundzo taso, nome chofota **Add Lesson** kwakha lesisha. Bona [Kuphatsa Tifundzo](./managing-lessons.md) kutfola imininingwane yekwakha kontenti yesifundzo.

## Kulayisha Tinsizakalo Temtindze Wesifundvo

Ungakwati kunamatsela emafayela langalandwa lasebenta sifundvo sonkhe -- sibonelo, sifingqo setichungechunge, titfombe tekumemetela, nome sicu semculo wakubatali.

1. Vula sifundvo endzaweni yekuphatsa.
2. Yaphuma uye kulingeniso le-**Resources**.
3. Layisha emafayela akho.

:::tip
Bhala incazelo yesifundvo sakho ucabanga ngebaholi belibandla. Sifingqo lesicacile sesihloko, licembu leminyaka lelicoshiwe, nenani letifundzo, kusita emabandla kucoshisa nangabe angasebentisa tichungechunge.
:::
