# Kuphatsa Tifundzo

<div class="article-intro">

Tifundzo tiwutinhlangano temfundvo letitsite ngekhatsi kwesifundvo. Sifundzo ngasinye simele lokwentekako ngesikhatsi selingeniso lelilinye lekilasi. Lapha yindzawo lapho wakha khona kontenti lenemininingwane lecatsulwe -- ema-venue, tigaba, tindzima, netento -- lokutakusetjentiswa ngulabasita labazawufundzisa.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekucala</h4>

- [Yakha luhlelo](./managing-programs.md) bese [wengeta sifundvo](./managing-studies.md) kulo
- Iba nemvumo yekuphatsa ye-Lessons.church (bona [Sifingqo Sekuphatsa](./index.md))

</div>

## Kwakha Sifundzo

1. Yaphuma uye endzaweni ye-**Admin**, wandise luhlelo nesifundvo, bese utfola sifundvo lofuna kwengeta kulo sifundzo.
2. Chofota **Add Lesson**.
3. Faka **Name** neminye imininingwane yesifundzo.
4. Gcina sifundzo.

## Kucondzisisa Sakhiwo Sesifundzo

Sifundzo ngasinye sihlelwe ngemitindze lemininigi:

- **Ema-Venue** -- Tinhlobo letahlukene tesifundzo temacembu ahlukene eminyaka nome tetimo. Sibonelo, ungaba ne-venue "yaseNazaleti" (Preschool) ne-venue "yaBancane" (Elementary) yesifundzo lesifanako.
- **Tigaba** -- Tincenye ngekhatsi kwe-venue, njengekutsi "Kwemukela", "Vidiyo", "Indzaba yeliBhayibheli", ne"Umsebenti". Tigaba tilawula kuhamba kwesikhatsi sekilasi.
- **Tindzima Netento** -- Ngekhatsi kwesigaba ngasinye, wabela tindzima (sibonelo, "Umfundzisi Lomkhulu" nome "Umholi Wecembu Lelincane") bese unika indzima ngayinye tento letitsite.

## Tinhlobo Tetento

Kunetinhlobo letine tetento longatiwabela indzima ngekhatsi kwesigaba:

- **Say** (Sho) -- Umbhalo lokufanele umsiti awukhulume nome awuchaze kubantfwana bekilasi.
- **Do** (Yenta) -- Umsebenti wemtimba nome umyalo lokufanele wenteke.
- **Note** (Phawula) -- Imininingwane yangemuva nome sikhumbuto semsiti (akufundvwa ngalizwi lelikhulu).
- **Play** (Dlala) -- Vidiyo nome umsindvo lokufanele udlalwe ngesikhatsi sesigaba.

## Kwengeta i-Venue

1. Vula sifundzo endzaweni yekuphatsa.
2. Chofota **Add Venue**.
3. Faka ligama le-venue (sib., "Preschool" nome "Elementary").
4. Gcina. Manje sewungengeta tigaba kule venue.

## Kwengeta Tigaba Netento

1. Yandisa i-venue kuze ubone tigaba tayo.
2. Chofota **Add Section** bese uyinika ligama (sib., "Kwemukela").
3. Ngekhatsi kwesigaba, chofota **Add Action**.
4. Khetsa **indzima**, **luhlobo lwesento** (Say, Do, Note, nome Play), bese ufaka kontenti.
5. Gcina.

Phindzaphinda lenyatselo kuze wakhe kuhamba kwesifundzo lokugcwele.

## Kulayisha Tinsizakalo Temtindze Wesifundzo

Ungakwati kunamatsela imibhalo langalandwa mgceka kusifundzo -- sibonelo, likhasi lemsebenti nome likhasi lekupendza lelibhekene naleliviki lelitsite.

1. Vula sifundzo endzaweni yekuphatsa.
2. Yaphuma uye kulingeniso le-**Resources**.
3. Layisha emafayela akho.

:::tip
Yakha sifundzo sakho kusukela embonweni wemsiti. Hamba etigabeni netentweni ngendzelo bese utibuta kutsi ngabe umsiti locala ngesikhatsi angakwati yini kulandzela ngaphandle kwekulungiselela lokunye.
:::
