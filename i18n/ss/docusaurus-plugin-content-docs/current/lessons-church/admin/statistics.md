# Tibalo

<div class="article-intro">

Likhasi letibalo linika baphatsi kucondzisisa ngendlela kontenti yabo lesetjentiswa ngayo. Ungabona kutsi ngutiphi tifundvo letitfolwa kakhulu, ngutiphi emabandla lalanda tinsizakalo takho, nekutsi lawomabandla akuphi ngeliciniso lendzawo.

</div>

## Kufinyelela Etibalweni

1. Yaphuma uye endzaweni ye-**Admin**.
2. Khetsa likhasi le-**Stats** nome **Statistics**.
3. Khetsa **luhlelo** lofuna kubona tibalo talo.

## Yini Longayibona

Likhasi letibalo linika imibono lemininigi yemniningwane yakho yekulandwa:

### Kulandwa Ngesifundvo

Kwehlukanisa kwekulandwa lokuhlukile ngesifundvo ngasinye ngekhatsi kwelinhlelo lelikhetsiwe. Loku kukukhombisa kutsi ngutiphi tichungechunge letitfola kakhulu, futsi kukusita ukhetse kutsi ufaka kuphi umtamo wakho wekwakha kontenti.

### Luhlu Lwemabandla

Luhlu lwemabandla lalande tinsizakalo elinhlelweni lakho. Loku kukunika umuzwa welilaleli lakho nekutsi kontenti yakho yemukelwa kangakanani.

### Kusabalala Ngeliciniso Lendzawo

Umbono wemephu lokhombisa kutsi emabandla lakho lalandzako akuphi. Loku kukusita ucondzisise kufinyelela ngeliciniso lendzawo kwekufundzisa kwakho futsi ukhombe tindzawo lapho kontenti yakho ithandvwa khona.

### Kuhlungwa Ngesikhatsi

Ungahlunga tibalo tonkhe ngesigaba sesikhatsi kuze ugcile esikhatsini lesitsite. Loku kuwusito ekuboneni tinguquko esikhatsini nome ekulinganiseni umtselela wesifundvo lesisha lesishicilelwe.

1. Setja **lusuku lwekucala** ne**lusuku lwekugcina** usebentisa tikhetsi telusuku.
2. Tibalo tizawuvuseleleka kutsi tikhombise kuphela kusebenta ngekhatsi kwaleso sigaba.

## Kusebentisa Tibalo Ngendlela Lenemsebenti

Nato tindlela letimbalwa tekusebentisa imininingwane yakho yekulandwa:

- **Khomba kontenti letfolwako** -- Gcila umtamo wakho ekwakheni tintfo letinyenti ngesitayela nangetihloko letitfola kulandwa lokunyenti.
- **Landzelela kushicilelwa lokusha** -- Ngemuva kwekushicilela sifundvo lesisha, hlunga ngelusuku kuze ubone kutsi sitfolwa ngekushesha njani.
- **Condzisisa lilaleli lakho** -- Sebentisa luhlu lwemabandla nemephu kubona kutsi ngubani losebentisa kontenti yakho bese ucabangela kuxhumana nalawomiphakatsi.

:::info
Tibalo tisekelwe ekulandweni lokuhlukile. Nangabe libandla lelifanako lilanda insita imikhawulo leminyenti, kubalwa kanye kuphela.
:::
