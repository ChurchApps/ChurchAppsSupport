# Kusungula I-akhawunti

<div class="article-intro">

Lessons.church isebentisa lona luhlelo lwe-akhawunti lwe-ChurchApps lolufananako ne-B1.church. Nangabe usunayo i-akhawunti ye-ChurchApps, ungangena ngco. Nangabe kungenjalo, kubhalisa kutsatsa emaminithi lambalwa kuphela futsi kukunike kufinyelela emtapeni wonkhe wetifundzo.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- Yiba nendlela yemeyili lesebentako lokulungele kucinisekiswa kwe-akhawunti
- Nangabe libandla lakho selisebentisa Lessons.church, buta umphatsi wakho ngemininingwane ngaphambi kwekusungula umfanekiso lomusha welibandla

</div>

## Kubhalisa I-akhawunti Lensha

1. Yiya ku- [Lessons.church](https://lessons.church) ku-brawuza yakho ye-inthanethi.
2. Chofota **Ngena** ku-bar yekuzulazula lengetulu.
3. Ekhasini lekungena, chofota **Bhalisa** kute usungule i-akhawunti lensha.
4. Faka **libito lakho lekucala**, **sibongo sakho**, nendlela **yemeyili** yakho.
5. Sungula **liphasiwedi** ye-akhawunti yakho.
6. Tfumela lifomu lekubhalisa.

Ngemuva kwekubhalisa, utawucelwa kutsi utihlanganise nelibandla lelikhona nobe usungule umfanekiso lomusha welibandla.

## Kutihlanganisa Nelibandla Lakho

**Kusungula libandla lelisha:**

1. Khetsa lesikhetso sekusungula **libandla lelisha**.
2. Faka **libito lelibandla** lakho nemininingwane leyisisekelo.
3. Cedzela kusungula. Utawuba ngumphatsi welibandla lakho ku-Lessons.church.

**Kuhlanganiswa nelibandla lelikhona:**

1. Sesha libandla lakho ngelibito.
2. Khetsa libandla lakho kuletiphumo.
3. Umphatsi welibandla lakho angase adzinge kukunika emalungelo lafanele.

:::tip
Nangabe umuntfu ebandleni lakho sewasungula Lessons.church, mcele kutsi acinisekise kutsi unemalungelo lafanele ekuphatseni emakilasi netinhlelo.
:::

## Kungena Nge-akhawunti Lekhona

Nangabe sewuneyakho i-akhawunti ye-ChurchApps kusukela ku-B1.church nobe kwenye insita ye-ChurchApps:

1. Yiya ku- [Lessons.church](https://lessons.church) futsi uchofote **Ngena**.
2. Faka **imeyili** nel **liphasiwedi** lakho lelikhona.
3. Utawuyiswa eliphothalini lapho ungaphatsa khona emakilasi netinhlelo takho.

:::info
I-akhawunti yakho ye-ChurchApps isebenta kuwo onkhe emikhicito ye-ChurchApps. I-akhawunti yinye ikunika kufinyelela ku-B1 Admin, B1.church, Lessons.church, nakunye lokunyenti.
:::

## Loko Lokulandzelako

Nase usungenile futsi utihlanganise nelibandla lakho, yiya ku- [Kubuka Konkhe Liphothali](./portal-overview.md) kute ufundze kutsi ungazulazula njani ku-dashboard futsi ucale kusungula emakilasi.
