# Kucala

<div class="article-intro">

Lessons.church ayikhokhelwa ngalokuphelele kusetjentiswa. Akukho kubhalisa ngenyanga, akukho sikhatsi sekulinga, futsi akukho emkhawulo etifeni. Lesigaba sikuhambisa ekusunguleni i-akhawunti yakho, kutihlanganisa nelibandla lakho, nasekuzulazuleni eliphothalini kuze ukwati kucala kuhlela tifundzo ngco.

</div>

## Loko Lokudzingekako

Kute ucale nge-Lessons.church, utawudzinga:

1. **Kusungula i-akhawunti** -- Bhalisa ku- [Lessons.church](https://lessons.church) ngeligama lakho, imeyili, neliphasiwedi. Loku kusebentisa i-akhawunti lefananako ye-ChurchApps longase usenayo kusukela ku-B1.church.
2. **Kutihlanganisa nelibandla lakho** -- Ngemuva kwekubhalisa, tihlanganise nemfanekiso welibandla lakho lelikhona nobe usungule lomusha. Loku kuhlanganisa emakilasi netinhlelo takho nelibandla lakho.
3. **Kusungula emakilasi** -- Sungula emakilasi lahambisana nenkonzo yakho (sibonelo, "Sikolo Lesincane", "Libanga Lekucala kuya Kwesitsatfu", nobe "Bantfu Labasha"). Emakilasi yindzawo lapho uhlela khona futsi wahlele tifundzo.
4. **Kucala kuhlela tifundzo** -- Hlolisisa umtapo welokucuketfwe, utfole sifundvo lesihambisana netidzingo takho, futsi uhlele tifundzo emakilasini akho.

Nase emakilasi akho asengiwe futsi tifundzo tihlelwe, emavolontiya akho angafinyelela lokucuketfwe ngeliphothali futsi akhiphe tifundzo emakilasini abo.

:::tip
Nangabe usunayo i-akhawunti ye-ChurchApps kusukela ku-B1.church, ungangena ku-Lessons.church ngemininingwane lefananako -- akudzingeki kubhalisa kabusha.
:::
