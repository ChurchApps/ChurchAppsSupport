# Kuphequlula Kontenti

<div class="article-intro">

I-Lessons.church inikeza umtapo wamahhala wekufundzisa kwelibandla lonkhe umuntfu langakwati kuwuhlola. Awudzingi i-akhawunti nome kungena kuze uphequlule kontenti letfolakalako -- vakashela nje sayithi ucale ukhangela. Lesigaba sicutsa tonkhe tindlela tekutfola nekuhlola kufundzisa.

</div>

## Likhasi Lasekhaya

Nawufika kucala ku-Lessons.church, likhasi lasekhaya likunika umbono lonesivinini waloko konkhe lokutfolakalako:

- **Tinhlelo letikhangelwe** -- Tichungechunge tekufundzisa letikhonjisiwe tivela etulu, tenta kubalula kutfola kontenti letfolwako kakhulu nome levuselelwe kabusha.
- **Tonkhe tinhlelo** -- Ngephasi kwesigaba lesikhangelwe, utawutfola luhlu lolugcwele lwato tonkhe tinhlelo tekufundzisa letitfolakalako kanye netibalo letinjenge inani letifundzo netifundvo kilinye.
- **Kontenti yebanikeli** -- Kontenti letivela ku-Lessons.church kanye nakubanikeli bekufundzisa bangaphandle ibonakaliswa ndzawonye, ikunika indzawo yinye yekuhlola.

## Tindlela Tekutfola Kontenti

Kunetindlela letimbalwa tekuhamba emtapeni wekufundzisa:

1. **Phequlula ngeluhlelo** -- Chofota nome ngutuphi luhlelo ekhasini lasekhaya kubona tifundvo netifundzo talo. Le yindlela lengcono nawufuna kuhlola sicu lesitsite sekufundzisa. Bona [Tinhlelo Netifundvo](./programs-and-studies.md) kutfola imininingwane.

2. **Sesha** -- Sebentisa likhasi lekusesha kutfola tifundzo ngeligama lekusesha kuwo wonkhe umtapo. Loku kuyasita nawati sihloko kodvwa ungaciniseki kutsi ngeluphi luhlelo lolucutsa sona. Bona [Kusesha](./searching.md) kutfola imininingwane.

3. **Hlola banikeli bangaphandle** -- Lenye kontenti ivela kubanikeli bekufundzisa bangaphandle labashicilela nge-Open Lesson Format. Ungaphequlula tinhlelo netifundvo tabo njengekontenti yemgceka. Bona [Banikeli Bangaphandle](./external-providers.md) kutfola imininingwane.

## Akudzingeki Kungena

Kuphequlula kuvulekile ngalokuphelele. Ungakwati kubuka tinhlelo, tifundvo, tifundzo, imibhalo langalandwa, nemavidiyo ngaphandle kwekwakha i-akhawunti. Nangabe ukhetsa kutsi ufuna kuhlela tifundzo temakilasi akho nome kulungisa kontenti, ungakwati [kwakha i-akhawunti yamahhala](../getting-started/creating-account.md) ngaleso sikhatsi.

:::tip
Cala ngekuhlola tinhlelo letimbalwa ekhasini lasekhaya kuze utfole umuzwa wendlela kufundzisa lokuhlelwe ngayo ngaphambi kwekungena esifundvweni lesitsite.
:::
