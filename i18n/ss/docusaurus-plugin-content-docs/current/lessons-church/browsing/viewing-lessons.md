# Kubuka Tifundzo

<div class="article-intro">

Sifundzo ngasinye ku-Lessons.church silingeniso lekufundzisa lelipheleleko lelinekontenti lehleliwe, imibhalo langalandwa, nekusekelwa kwemacembu lamanyenti eminyaka. Lomculo uchaza indlela yekuhamba nekusebentisa likhasi lesifundzo kuze ulungiselele kilasi lakho.

</div>

## Kufinyelela Kusifundzo

Kubuka sifundzo, hamba ngesakhiwo sekontenti:

1. Chofota **luhlelo** ekhasini lasekhaya.
2. Khetsa **sifundvo** ekhasini lelinhlelo.
3. Chofota **sifundzo** lofuna kusibuka ekhasini lesifundvo.

## Yini Longayibona

Likhasi lesifundzo lihleliwe libe netindzawo letimbalwa:

### Vidiyo

Nangabe sifundzo sihlanganisa vidiyo, ivela etulu kwelikhasi. Chofota kuyidlala ngco kubhrawuza yakho. Emavidiyo avamise kusetjentiswa njengengeniso yekufundzisa lesemgceka, ikakhulukati kunkonzo yebantfwana.

### Tigaba Netindzima

Sifundzo ngasinye sihlukaniswa tibe **tigaba** (sibonelo, "Kwemukela," "Indzaba yeliBhayibheli," "Umsebenti," "Kuvala"). Ngekhatsi kwesigaba ngasinye, kontenti ihleliwe nge**ndzima**:

- **Leader** (Umholi) -- Umfundzisi lomkhulu nome umethuli
- **Helper** (Umsiti) -- Basiti labasekelako
- Letinye tindzima letitsite lekhetsiwe ngumchami wekufundzisa

Indzima ngayinye ihlanganisa **tento** tayo, letitjela umsiti kutsi kufanele enteni ngco:

- **Say** (Sho) -- Umbhalo nome emaphuzu ekukhuluma lokufundzwa ngalizwi lelikhulu
- **Do** (Yenta) -- Imiyalo yemtimba nome imisebenti lekufanele yentiwe
- **Note** (Phawula) -- Imininingwane yangemuva nome emacebo emsiti
- **Play** (Dlala) -- Iphuzu lekudlala vidiyo nome umsindvo

### Imibhalo Langalandwa

Tifundzo tingahlanganisa emafayela longawalanda futsi uwaphrinte, njengekutsi:

- Emakhasi e-PDF
- Emakhasi emisebenti
- Emakhasi ekupendza
- Imiyalo yemsebenti wetandla
- Emakhasi ekuya ekhaya wabatali

Khangela emalinki ekulanda ngekhatsi kwelikhasi lesifundzo kutfola letinsizakalo. Bona [Kulanda Netintfo Tekusebenta](../presenting/downloads.md) kutfola imininingwane leyengetiwe yekulungisela tinsizakalo letiphrintiwe.

## Kushintja Emkhatsini We-Venue

Tifundzo letinyenti tesekela ema-venue lamanyenti, lamela emacembu eminyaka lahlukene njengekutsi Preschool, Elementary, nome Pre-Teen. Ema-venue avumela sihloko sesifundzo lesifanako kutsi silungiselelwe linani lelifanele licembu ngalinye.

Kushintja emkhatsini wema-venue:

1. Khangela **sikhetsi se-venue** ekhasini lesifundzo.
2. Chofota ligama le-venue lelihlukile (sibonelo, shintja kusukela ku-**Elementary** uye ku-**Preschool**).
3. Likhasi liyavuseleleka kukhombisa kontenti, imibhalo, nemibhalo lelungiselelwe lelocembu leminyaka.

:::tip
Nangabe ufundzisa emacembu lamanyenti eminyaka, buyeketa i-venue ngayinye ngaphambi kwelingeniso lakho kuze ubone indlela kontenti ilungiselelwe ngayo emazingeni lahlukene.
:::
