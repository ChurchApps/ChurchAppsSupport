# Kuhlela Tifundzo

<div class="article-intro">

Nase emakilasi akho asengiwe, sinyatselo lesilandzelako kuhlela tifundzo kuze emavolontiya akho atati kahle kutsi afanele afundzise ini futsi nini. Kuhlela kuhlanganisa lokucuketfwe kwesifundzo lesivela ku-Lessons.church nekhalenda yelikilasi lakho.

</div>

## Kubuka Luhlelo Lwelikilasi

1. Ngena futsi uye **Eliphothalini** lakho.
2. Chofota **likilasi** lofuna kulihlela.
3. Kubuka kweluhlelo kukhombisa **tifundzo letine letendlulile** kanye **netifundzo letine letitako**. Loku kukupha kubuka lokushesha kwaloko lokwakufundziswe kungakadlula nekwaloko lokutako.

## Kungeta Sifundzo Eluhlelweni

1. Kusukela ekubukeni kweluhlelo lwelikilasi, chofota kute ungete sifundzo lesisha.
2. Khetsa **lilanga** lesifundzo.
3. Khetsa **luhlelo** lofuna kufundzisa kulo (sibonelo, "Tindzaba TeliBhayibheli Tebantfwana").
4. Khetsa **sifundvo** lesingekhatsi kwaloluhlelo (sibonelo, "Kudalwa Netindzaba Takucala").
5. Khetsa **sifundzo** lesitsite lofuna kusihlela.
6. Khetsa **indzawo** lefaneleko yalelikilasi. Tindzawo timela tinguqulo letahlukene teliqembu leminyaka telokucuketfwe kwesifundzo (sibonelo, "Sikolo Lesincane" nobe "Sikolo Lesiphansi"). Kukhetsa indzawo lefanele kucinisekisa kutsi emavolontiya akho abona lokucuketfwe lokwakhelwe liqembu leminyaka lelisekilasini labo.
7. Gcina sifundzo lesihleliwe.

Sifundzo manje sitawuvela eluhlelweni lwelikilasi ngalelolilanga.

:::info
Tinhlelo, tifundvo, netifundzo kuluhlu lwekulandzelana lwekucuketfwe ku-Lessons.church. Luhlelo lunetifundvo letinyenti, futsi sifundvo ngasinye sinetifundzo letahlukene ngekhatsi kwaso. Tindzawo tinikeza tinguqulo temacembu eminyaka lahlukene tesifundzo lesifananako. Bona [Tinhlelo Netifundvo](../browsing/programs-and-studies.md) kute utfole incazelo legcwele.
:::

## Kutsi Emavolontiya Asebentisa Luhlelo Njani

Nase uhlelile tifundzo, emavolontiya akho angavula liphothali, akhetse likilasi labo, futsi ababone ngco loko lokutako. Bangahlolisisa lokucuketfwe kwesifundzo kusengaphambili, badawunilodze naluphi lolucuketfwe lolwakhona, futsi bafike balungele. Akusekho kucabanga lokungacinisekile ngaloko lokufanele kufundziswe kulandzelako.

## Emacebo Ekuhlela

- **Hlela kusengaphambili** -- Hlela emaviki lamanyenti etifundzo ngesikhatsi sinye kuze licembu lakho lilungiselele kusengaphambili.
- **Hlala uhlala ufana** -- Kulandzela luhlelo ngekulandzelana kusita bantfwana kwakha phetulu kwaloko labakufundze ngeliviki leledlule.
- **Hlola umlandvo** -- Sebentisa luhlu lwetifundzo letendlulile kucinisekisa kutsi awuphindzi lokucuketfwe ngephutsa.

:::warning
Cinisekisa kutsi ukhetsa indzawo lefaneleko yeliqembu leminyaka lelisekilasini lakho. Kukhetsa indzawo lengafanele kusho kutsi emavolontiya akho atawubona lokucuketfwe lokwakhelwe lizinga leminyaka lelahlukile.
:::

## Loko Lokulandzelako

Ngemuva kwekuhlela tifundzo, ungase ufune [kulungisela lokucuketfwe kwesifundzo](../customization/customizing-lessons.md) emakilasini latsite kuze kuhambisane nesitayela sakho sekufundzisa netidzingo teliqembu leminyaka. Nakusesikhatsi sekufundzisa, fundza nge- [Kwetfula](../presenting/index.md) kute ukhiphe tifundzo ngendlela lephumelelako.
