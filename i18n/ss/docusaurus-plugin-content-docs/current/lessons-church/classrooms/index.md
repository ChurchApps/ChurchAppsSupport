# Emakilasi

<div class="article-intro">

Emakilasi siyisisekelo saloko kuhleleka kwekufundzisa ebandleni lakho ku-Lessons.church. Likilasi limele ligumbi lelibonakalako nobe le-inthanethi lapho tifundzo tifundziswa khona -- sibonelo, "Ligumbi Lebantfwana," "Sikolo Lesiphansi K-2," nobe "Sifundzo SeliBhayibheli Sebantfu Labadzala." Lesigaba sicoca ngekusungula emakilasi nangekuhlela tifundzo kuwo.

</div>

## Kutsi Kungani Emakilasi Abaluleke

Nawusungula emakilasi, wakha sakhiwo lesicacile senkonzo yakho yebantfwana, emacembu ebantfu labasha, nobe nasiphi sizinda sekufundzisa. Likilasi ngalinye lithola luhlelo lwaso, ngako emavolontiya akho ahlala akwati kahle kutsi kufanele afundzise ini futsi nini. Kunekutfumela emaphepha eluhlelo endzaweni yendzawo nobe kutfumela imeyili yemuva kwesikhatsi, wonkhe umuntfu angahlolisisa liphothali abone loko lokutako.

## Loko Longakwenta Ngemakilasi

- **Sungula emakilasi langenamkhawulo** -- Sungula emakilasi lamanyenti njengobe libandla lakho lidzinga. Akukho mkhawulo, ngako ungaba nemakilasi lahlukene ngaliqembu leminyaka ngalinye, ikhempasi, nobe luhlelo.
- **Hlela tifundzo** -- Nikeza tifundzo letitsite kilasi ngalinye ngemalanga latsite. Luhlelo lwakho lukhombisa tifundzo letendlulile naletitako ngekubuka kanye.
- **Lungisela lokucuketfwe** -- Lungisela lokucuketfwe kwesifundzo kilasi ngalinye kuze emavolontiya abone kuphela loko lokuhambisana neliqembu labo leminyaka. Bona [Kulungiselwa](../customization/index.md) kute utfole imininingwane.

## Kutsi Emakilasi Ahlangana Njani

Emakilasi ahlanganisa lokucuketfwe kwesifundzo ku-Lessons.church nemhlelo wekufundzisa wangempela welibandla lakho. Nayi inchubo lejwayelekile:

1. [Hlolisisa](../browsing/index.md) tinhlelo netifundvo letitfolakalako ku-Lessons.church.
2. Sungula emakilasi akho kuze ahambisane nemagumbi elibandla lakho nemacembu eminyaka.
3. Hlela tifundzo kilasi ngalinye ngemalanga latako.
4. Emavolontiya avula likilasi labo kute babone loko lokuhleliwe futsi balungiselele.
