# Kulungiselwa

<div class="article-intro">

Lessons.church inikeza umtapo welokucuketfwe kwesifundzo lokusetayo kusetjentiswa, kodvwa libandla ngalinye lehlukile. Tici tekulungiselwa tikuvumela kulungisela lokucuketfwe kwesifundzo kuze kuhambisane nemakilasi akho latsite, sitayela sekufundzisa, kanye nemacembu eminyaka kuze emavolontiya akho abone kuphela loko lokuhambelanako kubo.

</div>

## Kungani Kulungisela?

Lokucuketfwe kwesifundzo ku-Lessons.church kwakhelwe kutsi kube nekuguculeka. Sifundzo sinye singahlanganisa tincenye temacembu eminyaka lamanyenti, imisebenti lengakafuneki, nobe tindzima letingasebenti kuwo onkhe emakilasi. Kunekucela emavolontiya akho kutsi bacabange bacedzele tincenye letingasebenti, ungalungisela sifundzo kuze babone kuphela loko lokuhambelanako kubo.

Sibonelo, sifundzo singahlanganisa umsebenti wekuchubeka losebenta kahle kubantfwana besikolo lesincane kodvwa ungasilungeli kilasi lakho lelibanga lesihlanu. Ngekulungisela, ungafihla lencenye kuliqembu lalabadzala kusenjalo uyigcine kulelincane.

## Loko Longakulungisela

- **Kususa tincenye** -- Fihla tincenye tonkhe tesifundzo letingasebenti kulelikilasi lelitsite nobe liqembu leminyaka.
- **Kufihla tindzima nobe tento** -- Susa tindzima nobe tento letitsite ngekhatsi kwencenye nangabe atidzingeki.
- **Kuhlela kabusha lokucuketfwe** -- Hlela kabusha tincenye netento kuze kuhambisane nemgudvu lowutsandzako welikilasi lakho.

## Kwelikilasi Kuphela Nobe Kwabo Bonkhe

Nawugcina kulungiselwa, ukhetsa kutsi kusebenta ngalobubanti bani:

- **Likilasi lodvwa** -- Kulungiselwa kutsintsa kuphela lelikilasi lelikhona. Lamanye emakilasi abona lokucuketfwe kwangempela.
- **Wonkhe emakilasi** -- Kulungiselwa kusebenta kuwo onkhe emakilasi ebandleni lakho.

Loku kuguculeka kusho kutsi ungasungula tinguqulo letahlukene tesifundzo lesifananako emacembini eminyaka lahlukene, nobe wente lushintjo lolusebenta yonkhe indzawo ngasikhatsi sinye.

:::tip
Nangabe umutjha ekulungiseleni, cala ngelikilasi linye kute ubone kutsi kusebenta njani ngaphambi kwekusebentisa tinguculo kuwo onkhe emakilasi.
:::
