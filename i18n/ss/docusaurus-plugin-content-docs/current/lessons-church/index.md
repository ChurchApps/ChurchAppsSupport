# Lessons.church

<div class="article-intro">

Lessons.church liyipulatifomu yetifundvo yemabandla lengakhokhelwa nakancane. Akukho kubhalisa ngenyanga, akukho tindleko, futsi akukho emkhawulo ekusetjentisweni. Iniketa tifundzo letingetulu kwe-1,100 letisekelwe kumavidiyo kuto tonkhe tifundvo letingetulu kwe-189 letivela kubahlinzeki labangetulu kwe-12, letimbandzakanya emacembu eminyaka kusukela kubantfwana labasesikoleni lesincane kuya kubantfu labadzala.

</div>

## Loko Longakwenta

Ngekusebentisa Lessons.church, libandla lakho lingakwati:

- **Kuhlolisisa lokucuketfwe** -- Hlolisisa umtapo lokhulako wetinhlelo netifundvo letivela kubahlinzeki labaningi, lahlelwe ngeliqembu leminyaka nangesihloko.
- **Kusungula emakilasi** -- Sungula emakilasi lahambisana nesakhiwo senkonzo yakho (sibonelo, sikolo lesincane, sikolo lesiphansi, bantfu labasha, nobe bantfu labadzala).
- **Kuhlela tifundzo** -- Nikeza emakilasi akho tifundzo letitsite ngemalanga lawadzingako.
- **Kulungisela tidzingo takho** -- Guqula lokucuketfwe kwesifundzo kuze kuhambisane nesitayela nekutsandza kwelibandla lakho ekufundziseni.
- **Kwetfula emakilasini** -- Sebentisa simo se-presenter lesakhelwe ngekhatsi kuhamba tifundzo kancane kancane ngesikhatsi sekilasi.

## Kuhleleka Kwalokucuketfwe

Lessons.church ihlela tifundzo ngekulandzelana lokucacile:

1. **Tinhlelo** -- Luhlu lwetifundzo loluphakeme oluvela kumhlinzeki (sibonelo, luhlelo lwenkonzo yebantfwana).
2. **Tifundvo** -- Licembu letifundzo letihlobene ngekhatsi kweluhlelo (sibonelo, luhlu lwemaviki lamane mayelana nesihloko lesitsite).
3. **Tifundzo** -- Sikhatsi sekufundzisa lesisodvwa ngekhatsi kwesifundvo, lesivamise kusetjentiswa emhlanganweni wekilasi munye.
4. **Tindzawo** -- Tinguqulo letahlukene tesifundzo letakhelwe emacembu eminyaka lahlukene nobe tindzawo letitsite (sibonelo, inguqulo yesikolo lesincane nenguqulo yesikolo lesiphansi).

## Kuhlanganiswa

Lessons.church isebenta kanye netinye tinsita ta-ChurchApps kute ikusite ukhipha lokucuketfwe:

- **B1.church** -- Emavolontiya nebatali bangafinyelela tifundzo letitako naletisandza kubakhona ngekusebentisa i-app ye-B1.
- **FreeShow** -- Sebentisa insakatiselo yekwetfula ye-FreeShow kuveta lokucuketfwe kwesifundzo esikrinini.
- **Venue TV App** -- Veta tifundzo ngco kuthelevishini ngekusebentisa insakatiselo ye-Android TV.
