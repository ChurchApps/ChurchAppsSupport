# Simo Sekwetfula

<div class="article-intro">

Simo Sekwetfula siguqula sifundzo sibe sikrini lesigcwele, lokwakhelwe kuveta. Sakhiwe kute sikhombise lokucuketfwe kwesifundzo emakrinini elikilasi, emaphrojekthi, nobe kumathelevishini kuze wonkhe umuntfu ekamelweni akwati kulandzela ngesikhatsi seklasi.

</div>

<div class="prereqs">
<h4>Ngaphambi Kwekutsi Ucale</h4>

- [Hlela sifundzo](../classrooms/scheduling-lessons.md) elikilasini, nobe uye kusifundzo nge [umtapo welokucuketfwe](../browsing/index.md)
- Cinisekisa kutsi sikrini sakho (iphrojekthi, thelevishini, nobe monitha) sihlanganisiwe futsi silungisiwe

</div>

## Kungena Ku-Simo Sekwetfula

1. Vula sifundzo lofuna kusetfula.
2. Chofota inkhinobho **Etfula** ekhasini lesifundzo.
3. Sifundzo sitawuguqukela ekubukeni kwesikrini lesigcwele lesinemuva lemnyama nembhalo lomkhulu, lolula kufundza.

Nawuse ku-Simo Sekwetfula, ungahamba etincenyeni tesifundzo incenye nge-incenye usebentisa tilawuli tekuzulazula letisesikrinini.

## Kutsi Kubukeka Njani

Simo Sekwetfula sisebentisa umnyama lohlobile lohlanti kute lokucuketfwe kuvele ngco esikrinini lesivetiwe. Incenye ngayinye yesifundzo ikhonjiswa ngasikhatsi sinye, kugcina kucondzana kusekwenteni lesikhona. Umbhalo, titfombe, nemavidiyo tonkhe tihleliwe kute kufundzeke kusukela kudze ekamelweni.

## Kusebentisa Ne-FreeShow

Lessons.church ihlanganiswe ne- [FreeShow](https://freeshow.app), insakatiselo yekwetfula lengakhokhelwa. FreeShow ingadonsa lokucuketfwe kwesifundzo ngco kusukela ku-Lessons.church, ikuvumela kufaka emaslayidi esifundzo emgudvini wakho lowakhona wekwetfula. Loku kusita kakhulu nangabe usewusebentisa i-FreeShow yemavi ekukhonta nobe tatiso.

## Kusebentisa Ne-Venue TV App

Insakatiselo ye- **Venue TV** ye-Android ikuvumela kukhombisa tifundzo kuthelevishini ngaphandle kwekhompyutha lehlukile. Faka insakatiselo kudivayisi ye-Android TV, uyihlanganise nelikilasi lakho le-Lessons.church, sifundzo lesihleliwe sitawuvela ngokutentekela. Loku kuyintfo lenhle yekungasebentisi tandla kumakilasi lanesikrini lesikhethiwe.

:::tip
Nangabe usebentisa iphrojekthi, cinisekisa kutsi sikrini sakho sisetiwe kutsi sikhombise nobe sengete kuphrojekthi ngaphambi kwekungena Ku-Simo Sekwetfula. Loku kucinisekisa kutsi kubuka kwesikrini lesigcwele kuvela esikrinini lesifanele.
:::

## Kuphuma Ku-Simo Sekwetfula

Kute uphume Ku-Simo Sekwetfula, cindzetela inkhinobho **Escape** nobe uchofote inkhinobho yekuphuma ekhoneni lesikrini. Utawubuyela ekubukeni lokujwayelekile kwesifundzo.
