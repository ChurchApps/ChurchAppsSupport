# Kudawunilodwa Nemitfombo

<div class="article-intro">

Tifundzo ku-Lessons.church tita netinsita letingadawunilodwa longatiphrinta nobe utabelane ngato nge-inthanethi. Letinsita tisekela lokucuketfwe kwesifundzo futsi tinike emavolontiya nebafundzi bakho intfo lengabanjwa yekusebenta ngayo ngesikhatsi semklaso nasemuva kwaso.

</div>

</div>

## Tinhlobo Temitfombo

Kuya ngesifundzo nemhlinzeki, ungatfola tinsita letinjenge:

- **Emaphepha emisebenti** -- Emaphepha emisebenti yesandla lekufanele afundzi bawacedzele ngesikhatsi semklaso.
- **Emakhasi ekupendza** -- Emakhasi langaphrintwa emacembini eminyaka lamancane.
- **Imihlahlandlela yebaholi** -- Imiyalo lenemininingwane nemininingwane yesisekelo yetitjeli.
- **Emaphepha e-PDF esengeta** -- Kufundza lokwengetiwe, emaphepha ekuya ekhaya, nobe emaphepha ebatali.

Akusiwo onkhe sifundzo lesinato tonkhe leti. Imitfombo letfolakalako incike kuloko umhlinzeki welokucuketfwe akuhlinzekile.

## Lapho Kutfolwa Khona Kudawunilodwa

Imitfombo itfolakala kumazinga lamatsatfu:

- **Lizinga lwaluhlelo** -- Imitfombo lejwayelekile lesebenta kulo lonkhe luchungechunge lwetifundzo.
- **Lizinga lwesifundvo** -- Imitfombo lehlangene nesifundvo lesitsite ngekhatsi kwaluhlelo.
- **Lizinga lwesifundzo** -- Imitfombo yesifundzo lesisodvwa.

Kudawunilodwa imitfombo:

1. Yiya ekhasini lesifundzo, sifundvo, nobe luhlelo.
2. Fune incenye ye- **Kudawunilodwa** nobe **Imitfombo** ekhasini.
3. Chofota libito lelifayela kute ulidawunilode kudivayisi yakho.

Emafayela avamise kuba ngesimo se-PDF futsi alungele kuphrintwa.

## Kulungiselwa Iklasi

Umgudvu lomuhle kuhlola imitfombo legadawunilodwa emalanga lambalwa ngaphambi kweklasi:

1. Vula sifundzo lesitako futsi uhlole kudawunilodwa lokutfolakalako.
2. Phrinta emakhophi lanele emaphepha emisebenti nemakhasi ekupendza kubantfu lababhekekile.
3. Hlola umhlahlandlela webaholi kute utfole kwaziwa umgudvu wesifundzo.
4. Beka bucala nayiphi imitfombo yekuya ekhaya kute uyiphe ekugcineni kweklasi.

:::info
Labanye bahlinzeki bafaka yonkhe imitfombo yabo elizingeni lesifundvo kunekuyihlanganisa etifundzweni letitsite. Nangabe ungaboni kudawunilodwa ekhasini lesifundzo lesitsite, hlola likhasi lesifundvo nobe luhlelo.
:::

:::tip
Sungula luhlu lwesikhecelo lweliviki nalilinye lwemavolontiya akho lolubandzakanya kudawunilodwa nekuphrinta imitfombo. Loku kucinisekisa kutsi akukho lokusalibalako ngaphambi kwekusa ngeSonto.
:::
