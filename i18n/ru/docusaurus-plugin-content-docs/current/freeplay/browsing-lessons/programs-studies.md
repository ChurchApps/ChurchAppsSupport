# Программы и курсы

<div class="article-intro">

Обозреватель уроков использует пошаговый процесс для выбора нужного контента. На каждом шаге отображается сетка карточек из 3 столбцов с изображениями и заголовками, что позволяет легко перемещаться по всей библиотеке уроков с помощью пульта ТВ.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Убедитесь, что телевизор подключён к интернету (требуется для просмотра каталога программ)
- Перейдите на экран **Обзор программ** через боковую панель

</div>

## Просмотр программ

1. Откройте экран **Обзор программ**
2. Программы отображаются в виде сетки карточек с изображениями, по три в ряд
3. Используйте кнопки направления на пульте для выделения программы
4. Нажмите **Выбрать**, чтобы открыть её

Каждая карточка программы показывает обложку программы. Если изображение недоступно, отображается карточка-заполнитель.

## Выбор курса

После выбора программы загружается экран **Выберите курс** со всеми курсами в этой программе.

1. Карточки курсов отображаются в сетке из 3 столбцов с обложками и названиями курсов
2. Перейдите к нужному курсу и нажмите **Выбрать**
3. Нажмите **Назад**, чтобы вернуться к списку программ

## Выбор урока

После выбора курса на экране **Выберите урок** отображаются все уроки этого курса.

1. Карточки уроков показывают обложки и названия уроков
2. Перейдите к нужному уроку и нажмите **Выбрать**
3. Нажмите **Назад**, чтобы вернуться к списку курсов

## Выбор площадки

На экране деталей урока отображается название урока, заголовок и описание, наложенные на обложку урока. Под описанием вы увидите одну или несколько кнопок площадок.

Площадки представляют собой различные версии одного и того же урока. Например, у урока могут быть отдельные площадки для возрастных групп «Дошкольники», «Начальная школа» и «Подростки». Каждая площадка содержит медиафайлы, соответствующие данной возрастной группе.

1. Используйте кнопки направления для выделения нужной площадки
2. Нажмите **Выбрать**, чтобы начать загрузку контента
3. FreePlay загружает плейлист площадки и запускает проигрыватель

:::tip
Если вы видите только одну кнопку площадки, значит, доступна только одна версия урока. Выберите её, чтобы сразу начать воспроизведение.
:::

## Навигация назад

В любой момент при просмотре нажмите **Назад** на пульте, чтобы вернуться к предыдущему уровню:

- С экрана деталей урока кнопка **Назад** возвращает к списку уроков
- Из списка уроков **Назад** возвращает к списку курсов
- Из списка курсов **Назад** возвращает к сетке программ
- Из сетки программ **Назад** открывает боковую панель
