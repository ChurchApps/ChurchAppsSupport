# Просмотр уроков

<div class="article-intro">

FreePlay включает встроенный обозреватель уроков, который позволяет изучать полную библиотеку доступных программ, курсов и уроков. Это полезно, когда вы хотите вручную выбрать урок для воспроизведения, а не полагаться на расписание класса или план.

</div>

## Порядок просмотра

Уроки организованы в иерархию:

1. **Программы** — категории учебных планов верхнего уровня (например, серия для детского служения)
2. **Курсы** — группы уроков в рамках программы (например, многонедельный блок)
3. **Уроки** — отдельные учебные занятия со своим контентом
4. **Площадки** — различные версии урока, адаптированные для определённых возрастных групп или условий

Вы начинаете с выбора программы, затем переходите к курсу, выбираете урок и, наконец, выбираете площадку для начала воспроизведения.

## Доступ к обозревателю уроков

Обозреватель уроков доступен на экране **Обзор программ**. Перейдите к нему через боковую панель или выбрав опцию просмотра в навигации.

:::info
Для просмотра уроков требуется интернет-соединение для загрузки каталога программ. После выбора площадки и начала воспроизведения контент транслируется напрямую без отдельного этапа загрузки.
:::
