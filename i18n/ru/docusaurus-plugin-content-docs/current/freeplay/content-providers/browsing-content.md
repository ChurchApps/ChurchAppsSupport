# Просмотр и загрузка контента

<div class="article-intro">

После подключения к поставщику контента вы можете просматривать его библиотеку, загружать файлы и воспроизводить их на телевизоре. Обозреватель контента представляет собой визуальную сетку папок и файлов, по которой вы перемещаетесь с помощью пульта ТВ.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Подключитесь хотя бы к одному поставщику контента — см. [Подключение к поставщикам](./connecting-providers)
- Подготовьте пульт ТВ для навигации

</div>

## Обозреватель контента

Обозреватель контента отображает элементы в сетке карточек из 3 столбцов. Каждая карточка показывает миниатюру и заголовок. Есть два типа элементов:

- **Папки** — обозначаются миниатюрой или значком папки. Выберите папку, чтобы увидеть её содержимое.
- **Файлы** — отдельные медиаэлементы (видео или изображения). На миниатюрах видеофайлов отображается значок воспроизведения.

В заголовке вверху экрана отображается название текущей папки или название поставщика, если вы находитесь на корневом уровне.

## Навигация по папкам

1. Используйте кнопки направления на пульте для выделения папки
2. Нажмите **Выбрать**, чтобы открыть её
3. Обозреватель загружает содержимое папки, которое может содержать другие папки или воспроизводимые файлы
4. Нажмите **Назад** на пульте, чтобы перейти на один уровень вверх

Вы можете углубляться настолько, насколько позволяет структура контента поставщика. Обозреватель отслеживает ваш путь по папкам, чтобы вы могли возвращаться через каждый уровень.

## Загрузка и воспроизведение контента

Когда вы выбираете папку с воспроизводимыми файлами (или отдельный файл), FreePlay переходит на экран загрузки:

1. На экране отображается заголовок контента с обложкой из выбранной папки
2. Индикатор прогресса показывает **Загрузка элемента X из Y** по мере сохранения файлов на устройство
3. После завершения загрузки появляется кнопка **Начать** со значком воспроизведения
4. Нажмите **Выбрать** для начала воспроизведения

:::tip
Загруженный контент кэшируется на телевизоре. Если вы воспроизведёте тот же контент позже, он загрузится мгновенно без повторного скачивания.
:::

## Воспроизведение контента от поставщиков

Воспроизведение работает так же, как воспроизведение уроков. Используйте следующие элементы управления:

- **Стрелка вправо** — следующий элемент
- **Стрелка влево** — предыдущий элемент
- **Выбрать / Пауза-воспроизведение** — поставить на паузу или возобновить видео
- **Стрелка вверх** — открыть выбор сообщений для перехода к определённому элементу
- **Назад** — выйти в обозреватель контента

При выходе из проигрывателя вы возвращаетесь в обозреватель контента на корневом уровне текущего поставщика.

:::info
Полный справочник по всем элементам управления проигрывателем см. в разделе [Воспроизведение уроков](../classroom-mode/playing-lessons).
:::

## Переключение между поставщиками

Подключённые поставщики отображаются как элементы боковой панели слева. Чтобы переключиться на другого поставщика:

1. Нажмите **стрелку влево** на пульте, чтобы развернуть боковую панель
2. Выберите поставщика, которого хотите просмотреть
3. Обозреватель контента обновится и покажет библиотеку этого поставщика

:::warning
Если в папке отображается сообщение «Контент недоступен», у поставщика нет элементов на этом уровне. Попробуйте перейти в другую папку или проверьте позже.
:::
