# Поставщики контента

<div class="article-intro">

Поставщики контента — это внешние сервисы, предоставляющие учебные материалы, видео и изображения для воспроизведения на вашем телевизоре. FreePlay поддерживает нескольких поставщиков, позволяя вам просматривать и загружать контент из различных источников в одном приложении.

</div>

## Доступные поставщики

В настоящее время FreePlay включает следующих поставщиков:

- **[Lessons.church](../../lessons-church/)** — бесплатная программа для детского служения с открытым исходным кодом
- **Bible Project** — видео и ресурсы для изучения Библии
- **SignPresenter** — контент для презентаций и информационных табло в церквях
- **B1.church** — контент из вашей собственной настройки B1

:::info
Некоторые поставщики ещё добавляются. Если на карточке поставщика написано «Скоро», он пока недоступен для подключения.
:::

## Как работают поставщики

У каждого поставщика есть своя библиотека контента, организованная в папки и файлы. После подключения к поставщику его контент отображается в обозревателе папок, где вы можете переходить по категориям, выбирать элементы и загружать их для воспроизведения.

Подключённые поставщики отображаются в боковой панели в левой части экрана. Вы можете переключаться между поставщиками, выбирая их в боковой панели, что позволяет легко перемещаться между источниками контента во время настройки.
