# Подключение к поставщикам

<div class="article-intro">

Прежде чем вы сможете просматривать контент от поставщика, вам нужно подключиться к нему. Некоторые поставщики требуют аутентификацию через QR-код или вход по электронной почте, в то время как другие могут быть подключены одним щелчком.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Установите и запустите FreePlay — смотрите [Начало работы](../getting-started/)
- Держите пульт телевизора готовым для навигации
- Для поставщиков, требующих входа, имейте готовыми учетные данные вашей учетной записи

</div>

Наш **<a href="/guides/freeplay-b1admin" target="_blank">пошаговый руководство</a>** проходит через связь B1 Admin, планирование урока и подключение FreePlay — все в одном месте. Откройте в новой вкладке для следования.
:::

## Просмотр доступных поставщиков

1. Откройте экран **Поставщики контента** из боковой панели (выберите значок **Поставщики** внизу)
2. Вы увидите сетку карточек поставщиков, каждая показывает логотип и имя поставщика
3. Подключенные поставщики показывают зеленый значок **Подключено** ниже их имени
4. Поставщики, которые еще недоступны, показывают метку **Скоро**

## Подключение без аутентификации

Некоторые поставщики не требуют входа. Когда вы выбираете одного из этих поставщиков, FreePlay подключается немедленно и открывает браузер контента. Учетные данные не требуются.

## Аутентификация потока устройства (QR-код)

Определенные поставщики используют поток устройства, похожий на то, как вы входите в потоковые приложения на телевизоре:

1. Выберите карточку поставщика на экране **Поставщики контента**
2. FreePlay отображает QR-код и URL проверки
3. Отсканируйте QR-код с помощью телефона или посетите отображаемый URL на любом устройстве
4. Введите код пользователя, показанный на экране телевизора
5. Завершите процесс входа на телефоне или компьютере
6. FreePlay обнаруживает успешный вход и показывает **Подключено!**
7. Браузер контента открывается автоматически

:::info
Пульсирующий индикатор **Ожидание авторизации** показывает, что FreePlay проверяет ваш вход. Код истекает через несколько минут, поэтому завершите процесс оперативно.
:::

## Вход в форму

Другие поставщики используют традиционный вход по электронной почте и пароль:

1. Выберите карточку поставщика
2. Введите свой **Email** и **Пароль**, используя встроенную клавиатуру
3. Выберите кнопку **Войти**
4. Если ваши учетные данные верны, FreePlay показывает **Подключено!** и открывает браузер контента

:::tip
Используйте клавиши со стрелками на пульте для перемещения между полем электронной почты, полем пароля и кнопкой входа. Нажмите **Выбрать** на текстовом поле, чтобы открыть встроенную клавиатуру.
:::

## Отключение поставщика

Чтобы отключиться от поставщика, к которому вы уже подключались:

1. Перейдите на экран **Поставщики контента**
2. Выберите карточку поставщика, которая показывает значок **Подключено**
3. Приглашение подтверждения спрашивает, хотите ли вы отключиться
4. Выберите **Отключить**, чтобы удалить соединение

После отключения контент поставщика больше не будет отображаться в вашей боковой панели.

:::warning
Отключение удаляет сохраненную аутентификацию с вашего устройства. Вам нужно будет войти снова, если вы захотите переподключиться позже.
:::
