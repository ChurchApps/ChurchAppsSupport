# Подключение к поставщикам

<div class="article-intro">

Прежде чем вы сможете просматривать контент от поставщика, необходимо подключиться к нему. Некоторые поставщики требуют аутентификацию через QR-код или вход по электронной почте, а к другим можно подключиться одним нажатием.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Установите и запустите FreePlay — см. [Начало работы](../getting-started/)
- Подготовьте пульт ТВ для навигации
- Для поставщиков, требующих входа, подготовьте данные учётной записи

</div>

## Просмотр доступных поставщиков

1. Откройте экран **Поставщики контента** из боковой панели (выберите значок **Поставщики** внизу)
2. Вы увидите сетку карточек поставщиков, каждая из которых показывает логотип и название поставщика
3. Подключённые поставщики отображают зелёный значок **Подключено** под названием
4. Поставщики, которые ещё не доступны, показывают пометку **Скоро**

## Подключение без аутентификации

Некоторые поставщики не требуют входа. Когда вы выбираете одного из таких поставщиков, FreePlay подключается мгновенно и открывает обозреватель контента. Учётные данные не требуются.

## Аутентификация через устройство (QR-код)

Некоторые поставщики используют аутентификацию через устройство, аналогично входу в потоковые приложения на телевизоре:

1. Выберите карточку поставщика на экране **Поставщики контента**
2. FreePlay отображает QR-код и URL для подтверждения
3. Отсканируйте QR-код телефоном или откройте отображаемый URL на любом устройстве
4. Введите код пользователя, показанный на экране телевизора
5. Завершите процесс входа на телефоне или компьютере
6. FreePlay обнаруживает успешный вход и отображает **Подключено!**
7. Обозреватель контента открывается автоматически

:::info
Пульсирующий индикатор **Ожидание авторизации** показывает, что FreePlay проверяет ваш вход. Код истекает через несколько минут, поэтому завершите процесс своевременно.
:::

## Вход через форму

Другие поставщики используют традиционный вход по электронной почте и паролю:

1. Выберите карточку поставщика
2. Введите **электронную почту** и **пароль**, используя экранную клавиатуру
3. Нажмите кнопку **Войти**
4. Если учётные данные верны, FreePlay отображает **Подключено!** и открывает обозреватель контента

:::tip
Используйте кнопки направления на пульте для перемещения между полем электронной почты, полем пароля и кнопкой входа. Нажмите **Выбрать** на текстовом поле, чтобы открыть экранную клавиатуру.
:::

## Отключение поставщика

Чтобы отключиться от поставщика, к которому вы уже подключены:

1. Перейдите на экран **Поставщики контента**
2. Выберите карточку поставщика со значком **Подключено**
3. Появится запрос подтверждения на отключение
4. Выберите **Отключить**, чтобы удалить подключение

После отключения контент этого поставщика больше не будет отображаться в боковой панели.

:::warning
Отключение удаляет сохранённую аутентификацию с вашего устройства. Для повторного подключения потребуется снова выполнить вход.
:::
