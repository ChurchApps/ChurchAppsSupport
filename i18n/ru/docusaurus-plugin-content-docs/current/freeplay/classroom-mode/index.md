# Режим класса

<div class="article-intro">

Режим класса подключает ваш телевизор к определённой комнате в вашей церкви. После привязки FreePlay автоматически загружает урок, запланированный для этой комнаты, и вашим волонтёрам остаётся просто нажать **Начать урок**, когда начинается занятие.

</div>

## Как это работает

1. **Найдите свою церковь** — выполните поиск церкви по названию
2. **Выберите комнату** — выберите класс, к которому относится этот телевизор
3. **Загрузите урок** — FreePlay получает запланированный урок и загружает все медиафайлы
4. **Воспроизведение** — нажмите кнопку **Начать урок**, и урок начнёт воспроизводиться автоматически

Телевизор запоминает, к какой церкви и комнате он привязан, поэтому настройку нужно выполнить только один раз. Каждую неделю FreePlay проверяет расписание на текущий день и загружает новый контент.

:::tip
Убедитесь, что ваши классы настроены на [freeplay.church](https://freeplay.church), прежде чем привязывать телевизор. Если при выборе церкви классы не отображаются, посетите сайт для их настройки.
:::
