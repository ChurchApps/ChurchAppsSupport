# Привязка устройства

<div class="article-intro">

Для использования режима плана необходимо привязать телевизор к типу плана в B1 Admin. FreePlay генерирует уникальный код привязки, который связывает устройство с планом вашей церкви, обеспечивая автоматическую доставку контента каждую неделю.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Установите и запустите FreePlay — см. [Начало работы](../getting-started/)
- Выберите **Привязка к плану** на [экране выбора режима привязки](../getting-started/pairing-modes.md)
- Обеспечьте доступ к **B1 Admin** на компьютере или телефоне для завершения привязки

</div>

## Генерация кода привязки

1. На экране **Выбор режима привязки** выберите **Привязка к плану**
2. FreePlay связывается с сервером и генерирует короткий код привязки
3. Код отображается крупными символами в центре экрана
4. Под кодом пульсирующий индикатор показывает **Ожидание подключения**

Код отображается в виде отдельных символов для удобного считывания на расстоянии.

## Ввод кода в B1 Admin

1. На компьютере или телефоне откройте **B1 Admin** для вашей церкви
2. Перейдите к настройкам **типа плана** для плана, который вы хотите назначить
3. Введите код привязки, показанный на телевизоре
4. Сохраните настройки

FreePlay опрашивает сервер каждые несколько секунд, проверяя завершение привязки. Как только **B1 Admin** подтверждает подключение, телевизор автоматически переходит на экран загрузки плана.

## Загрузка контента плана

После привязки FreePlay загружает текущий план для данного типа плана. Отображается:

- Название плана и дата служения
- Название связанного урока (если план включает учебный контент)
- Индикатор прогресса, показывающий **Загрузка элемента X из Y**

Когда все медиафайлы загружены, появляется кнопка **Начать план**. Нажмите **Выбрать** на пульте для начала воспроизведения.

:::tip
План обновляется автоматически каждый час. Если план обновляется в течение дня, FreePlay подхватывает изменения без какого-либо ручного вмешательства.
:::

## Альтернатива: поиск по названию церкви

Если вы предпочитаете не использовать процесс с кодом привязки, можно выбрать **или выполнить поиск по названию церкви** в нижней части экрана привязки. Это перенаправит вас на экран поиска церкви, где можно найти свою церковь и подключиться к классу.

## Если привязка не удалась

Если код привязки не может быть сгенерирован (например, из-за проблем с сетью), вы увидите сообщение об ошибке с кнопкой **Попробовать снова**. Убедитесь, что телевизор подключён к интернету, и повторите попытку.

:::warning
Коды привязки истекают через определённое время. Если вы ждёте слишком долго, сгенерируйте новый код, вернувшись на экран привязки.
:::
