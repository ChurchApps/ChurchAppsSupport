# Режим плана

<div class="article-intro">

Режим плана подключает ваш телевизор к плану церковного служения. Вместо получения контента на основе расписания класса, он следует плану, настроенному руководством вашей церкви в B1 Admin. Это идеальный вариант, когда вы хотите, чтобы телевизор отображал именно тот контент, который назначен в плане текущей недели.

</div>

## Отличия от режима класса

- **[Режим класса](../classroom-mode/)** загружает урок, запланированный для конкретной комнаты на определённую дату
- **Режим плана** загружает контент, привязанный к типу плана, который может включать уроки, презентации поставщиков, дополнения и другие медиа, организованные в определённом порядке

Режим плана особенно полезен, когда ваш план служения включает контент из различных источников, объединённый в единую последовательность.

## Как это работает

1. FreePlay генерирует код привязки на экране
2. Администратор вводит этот код в **B1 Admin** в настройках типа плана
3. После привязки FreePlay загружает текущий план и скачивает все связанные медиафайлы
4. Нажмите **Начать план** для начала воспроизведения

Телевизор остаётся привязанным к типу плана. Каждую неделю он автоматически подхватывает последний план и его контент.
