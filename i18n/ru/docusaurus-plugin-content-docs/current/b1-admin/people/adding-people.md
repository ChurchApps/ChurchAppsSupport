# Добавление людей

<div class="article-intro">

Раздел "Люди" — это основа B1 Admin, ваша база данных членов церкви. Все остальные функции (группы, посещаемость, пожертвования, формы) связаны с записями о людях. Это руководство покажет вам, как добавить человека в базу данных, редактировать его данные и связывать членов семьи в домохозяйства.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Вам нужна активная учётная запись B1 Admin с разрешением на управление людьми. См. [Роли и разрешения](roles-permissions.md), если вы не уверены в своём уровне доступа.
- Если вы добавляете более нескольких человек, рассмотрите использование инструмента [импорта из CSV](importing-data.md).

</div>

## Добавление человека

1. Перейдите на панель управления B1.church Admin.
2. Нажмите **People** на левой панели навигации.
3. Нажмите кнопку **Add Person** в правом верхнем углу.
4. Заполните имя, фамилию и адрес электронной почты, затем нажмите **Add**.

Откроется страница профиля человека, готовая для добавления дополнительных данных.

:::tip
Если вы переходите с другой системы управления церковью, функция [Импорт данных](importing-data.md) позволяет загрузить весь справочник из CSV-файла — это гораздо быстрее, чем добавлять людей по одному.
:::

## Редактирование данных

1. На странице профиля человека нажмите на **значок карандаша** рядом с именем.
2. Заполните дополнительную информацию: отчество, статус членства, даты, адрес и номера телефонов.
3. Нажмите **Save**, чтобы сохранить личную информацию.

Профиль также включает несколько вкладок для связанной информации:

- **Заметки** — Добавляйте заметки о человеке (пастырская забота, необходимость связаться и т. д.)
- **Группы** — Просматривайте и управляйте [членством в группах](../groups/group-members.md)
- **Посещаемость** — Просматривайте [записи о посещаемости](../attendance/tracking-attendance.md)
- **Пожертвования** — Просматривайте [историю пожертвований](../donations/recording-donations.md)

## Работа с формами

Вы можете заполнять пользовательские формы непосредственно из профиля человека. Это пользовательские формы, которые вы можете создать, следуя руководству [Создание форм](../forms/creating-forms.md).

1. В профиле человека нажмите на выпадающий список **Формы**, чтобы выбрать форму.
2. Нажмите **Add Form**, чтобы открыть её.
3. Заполните данные формы и нажмите **Save**.

:::info
Формы, привязанные к профилю человека, используют тип формы **People**. Если вам нужна отдельная форма (например, для регистрации на мероприятие), см. [вариант Stand Alone](../forms/creating-forms.md) в руководстве по формам.
:::

## Управление домохозяйствами

Домохозяйства позволяют связывать членов семьи вместе. Это особенно полезно для [регистрации](../attendance/check-in.md), когда родитель может отметить всех своих детей одновременно.

1. В профиле человека нажмите на **значок карандаша** рядом с названием домохозяйства.
2. Откроется редактор домохозяйства. Выберите **роль в домохозяйстве** для текущего человека (например, Глава, Супруг(а), Ребёнок).
3. Нажмите **Add**, чтобы добавить ещё одного члена домохозяйства.
4. Введите имя человека в поле поиска и нажмите **Search**.
5. Когда человек появится в результатах поиска, нажмите **Select**.
6. Выберите его роль в домохозяйстве и нажмите **Save**, чтобы завершить настройку домохозяйства.
