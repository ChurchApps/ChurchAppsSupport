# Печать справочника

<div class="article-intro">

Функция печати справочника создает отформатированный справочник членов вашей церкви для печати. Он организует членов по семьям, включает контактную информацию и фотографии и предназначен для красивого вида при печати на стандартной бумаге формата письма.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Вам нужна активная учетная запись B1 Admin с разрешением на просмотр людей.
- Люди в вашем справочнике должны иметь заполненные фотографии и контактную информацию для получения наилучших результатов.

</div>

## Открытие справочника для печати

1. Перейдите к **Люди** в B1 Admin.
2. Нажмите на **значок печати** (значок принтера) на панели инструментов в верхней части карточки результатов "Люди". Он появляется рядом с кнопкой Экспорт.
3. Справочник для печати откроется в новой вкладке браузера и начнет загружаться ваши данные справочника.
4. Примерно через 1,5 секунды будет автоматически открыт диалог печати браузера.

:::tip
Справочник открывается в отдельной вкладке, поэтому вы можете печатать его, не покидая страницу "Люди". После появления диалога печати вкладка автоматически вернется назад.
:::

## Что включается

Печатный справочник включает всех **активных членов** вашей церкви. В частности:

- Люди со статусом членства **Неактивный** или **Посетитель** автоматически исключаются.
- Люди, которые **отказались** от справочника, исключаются.
- Все остальные сгруппированы по **семье**.

## Макет справочника

Печатный справочник состоит из двух частей:

### Обложка

На первой странице указано:
- Название вашей церкви
- Название "Справочник членов"
- Текущий год
- Местоположение вашей церкви
- Количество включенных семей

### Списки членов

На остальных страницах перечислены семьи в алфавитном порядке по фамилии. Каждая карточка семьи показывает:

- **Фотографии** — Круглые миниатюры для каждого члена семьи с их именем ниже
- **Имя дисплея семьи** — Например, "Семья Смит", "Джон и Джейн Доу" или просто имя человека для отдельных лиц
- **Адрес** — Почтовый адрес, город, штат и индекс
- **Телефон** — Домашний или мобильный номер
- **Электронная почта** — Основной адрес электронной почты
- **Дни рождения** — Отформатированы как отдельные даты для каждого члена семьи
- **Годовщина** — Если записано

Семьи организованы в **алфавитные секции** (А, Б, В, ...) с декоративным заголовком секции перед каждой группой букв. На каждой строке рядом друг с другом располагаются две карточки семьи.

## Параметры печати

Справочник отформатирован для **бумаги формата письма** (8,5" × 11") с полями 0,5 дюйма. Когда появляется диалог печати браузера:

- Оставьте размер бумаги установленным на **Письмо**
- Включите **Графика фона** (иногда называется "Печать фона") в параметрах печати вашего браузера, если вы хотите, чтобы заголовки секций печатались с цветовым оформлением
- Рекомендуется портретная ориентация

:::info
Справочник соблюдает разрывы страниц — карточки семей и заголовки секций не будут разбиты на разные страницы.
:::
