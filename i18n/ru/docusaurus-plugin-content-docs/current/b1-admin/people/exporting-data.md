# Экспорт данных

<div class="article-intro">

B1 Admin позволяет экспортировать данные вашей церкви для использования в электронных таблицах, передачи команде или создания резервной копии. Независимо от того, нужен ли вам быстрый список имён и email-адресов или полный экспорт базы данных, есть варианты, подходящие для ваших нужд.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Вам нужна активная учётная запись B1 Admin с разрешением на просмотр данных, которые вы хотите экспортировать. См. [Роли и разрешения](roles-permissions.md), если вы не уверены в своём уровне доступа.
- Для полного экспорта базы данных необходим доступ к разделу **Settings**.

</div>

## Экспорт со страницы "Люди"

Самый быстрый способ экспортировать справочник — непосредственно со страницы **People**:

1. Перейдите в раздел **People** на левой панели навигации.
2. Используйте панель поиска или фильтры, чтобы сузить результаты для экспорта (или оставьте без фильтрации, чтобы экспортировать всех). Советы по фильтрации см. в [Поиск людей](searching-people.md).
3. Используйте **выбор столбцов**, чтобы указать, какие столбцы включить в экспорт (например, Имя, Email, Телефон, Адрес).
4. Нажмите кнопку **Export**.
5. CSV-файл с данными, отображаемыми в таблице, будет загружен на ваш компьютер.

:::tip
Настройте столбцы перед экспортом. CSV-файл будет содержать именно те столбцы, которые у вас видимы, поэтому вы можете подготовить экспорт под свои нужды без редактирования файла после загрузки.
:::

## Полный экспорт данных из настроек

Для полного экспорта всех данных B1 (не только людей) используйте инструмент экспорта в настройках:

1. Нажмите **Settings** на левой панели навигации.
2. Нажмите **Import/Export** в верхней навигации.
3. Выберите **B1 Database** из выпадающего списка **Data Source**.
4. Просмотрите предварительный просмотр данных и нажмите **Continue to Destination**.
5. Выберите **B1 Export Zip** как назначение экспорта.
6. Отслеживайте прогресс экспорта, пока все элементы не покажут зелёные галочки.
7. Файл экспорта загрузится автоматически. Найдите файл `B1Export` в папке загрузок.
8. Распакуйте файл, чтобы получить доступ к отдельным CSV-файлам (таким как `people.csv`), которые можно открыть в Excel, Google Sheets или Numbers.

:::info
Полный экспорт данных включает людей, группы, пожертвования, посещаемость и многое другое — всё из вашей базы данных B1. Это также отличный способ создавать периодические резервные копии церковных записей.
:::

## Экспорт данных групп

Вы также можете экспортировать списки участников отдельных групп. На странице **Groups** откройте группу и нажмите **значок загрузки**, чтобы экспортировать список участников этой группы. Подробнее см. [Участники группы](../groups/group-members.md).

:::info
Экспортированные CSV-файлы работают со всеми основными приложениями для электронных таблиц, включая Microsoft Excel, Google Sheets и Apple Numbers.
:::
