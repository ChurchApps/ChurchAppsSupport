# Сохранённые списки

<div class="article-intro">

**Сохранённые списки** позволяют вам сохранить поисковый запрос под названием и переиграть его одним щелчком в любое время. Создайте список один раз — «Активные члены в кампусе Север», «Посетители за последние 90 дней» или любой продвинутый фильтр — и он останется в вашей боковой панели, чтобы вам никогда не нужно было пересоздавать запрос заново.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Выполните по крайней мере один поиск в разделе Люди, чтобы у вас были результаты для сохранения. Подробнее см. [Поиск людей](./searching-people.md) или [Поиск ИИ](./ai-search.md).

</div>

## Сохранение поиска как списка

1. Выполните поиск на странице **Люди** — быстрый поиск, поиск ИИ или поиск с продвинутым фильтром.
2. Когда результаты появляются, в заголовке страницы Люди появляется кнопка **Сохранить как список** (значок закладки).
3. Щелкните **Сохранить как список**.
4. Введите **Название** для списка (обязательно).
5. Дополнительно введите **Категорию** для группировки связанных списков на боковой панели (например, «Распространение» или «Членство»).
6. Нажмите **Сохранить**.

Список теперь сохранён в аккаунт вашей церкви и будет отображаться на панели **Сохранённые списки** слева от страницы Люди.

:::info
Списки общие по всей церкви — любой человек с доступом к Люди может видеть и загружать их. Только сотрудники с надлежащим разрешением могут переименовывать или удалять списки.
:::

## Загрузка сохранённого списка

На панели **Сохранённые списки** слева от страницы Люди щелкните любое название списка. Поиск переиграется мгновенно с сохранённым запросом, обновляя таблицу результатов.

Списки сгруппированы по категории (если вы назначили одну), при этом некатегоризированные списки появляются в конце.

:::tip
Списки — это **активные запросы**, а не снимки. Каждый раз, когда вы загружаете список, поиск выполняется заново по вашим текущим данным, чтобы новые люди, которые теперь соответствуют критериям, появились.
:::

## Переименование и удаление списков

Сотрудники с разрешением на управление могут переименовывать или удалять любой список непосредственно с панели Сохранённые списки.

- Щелкните **значок карандаша** рядом с названием списка, чтобы переименовать его.
- Щелкните **значок мусоропровода**, чтобы удалить его (вам будет предложено подтверждение).

## Варианты использования

| Сценарий | Как построить список |
|---|---|
| Все посетители | Продвинутый поиск: Статус членства = Посетитель |
| Члены на конкретном кампусе | Продвинутый поиск: Кампус = [ваш кампус] |
| Люди без адреса электронной почты | Продвинутый поиск: Электронная почта пуста |
| Результаты из вопроса ИИ | Задайте вопрос в Поиск ИИ, затем сохраните |
