# Сохраненные списки

<div class="article-intro">

**Сохраненные списки** позволяют вам сохранить поисковой запрос под именем и повторно запустить его одним щелчком в любой момент. Создайте список один раз -- "Активные члены в кампусе Север", "Посетители за последние 90 дней" или любой расширенный фильтр -- и он останется в вашей боковой панели, так что вам не придется пересчитывать запрос снова.

</div>

<div class="prereqs">
<h4>Перед тем как начать</h4>

- Выполните по крайней мере один поиск в разделе "Люди", чтобы у вас были результаты для сохранения. См. [Поиск людей](./searching-people.md) или [AI Search](./ai-search.md).

</div>

## Сохранение поиска как списка

1. Выполните поиск на странице **Люди** -- быстрый поиск, поиск AI или расширенный поиск с фильтром.
2. Когда появляются результаты, в заголовке страницы "Люди" появляется кнопка **Сохранить как список** (значок закладки).
3. Нажмите **Сохранить как список**.
4. Введите **имя** для списка (обязательно).
5. Дополнительно введите **категорию**, чтобы группировать связанные списки в боковой панели (например, "Охват" или "Членство").
6. Нажмите **Сохранить**.

Список теперь сохранен на аккаунте вашей церкви и появится в панели **Сохраненные списки** с левой стороны страницы "Люди".

:::info
Списки совместно используются по всей вашей церкви -- любой с доступом к "Люди" может видеть и загружать их. Только персонал с соответствующим разрешением может переименовывать или удалять списки.
:::

## Загрузка сохраненного списка

В панели **Сохраненные списки** с левой стороны страницы "Люди" нажмите на любое имя списка. Поиск мгновенно повторяется с сохраненным запросом, обновляя таблицу результатов.

Списки группируются по категориям (если вы назначили одну), с некатегоризованными списками, появляющимися в последний раз.

:::tip
Списки - это **live queries**, а не снимки. Каждый раз, когда вы загружаете список, поиск выполняется заново по вашим текущим данным, поэтому новые люди, которые теперь соответствуют критериям, будут отображаться.
:::

## Переименование и удаление списков

Персонал с разрешением управления может переименовать или удалить любой список непосредственно с панели сохраненных списков.

- Нажмите на **значок карандаша** рядом с названием списка, чтобы переименовать его.
- Нажмите на **значок мусоропровода**, чтобы удалить его (вас попросят подтвердить).

## Сценарии использования

| Сценарий | Как построить список |
|---|---|
| Все посетители | Расширенный поиск: Membership Status = Visitor |
| Члены на определенном кампусе | Расширенный поиск: Campus = [ваш кампус] |
| Люди без адреса электронной почты | Расширенный поиск: Email is empty |
| Волонтеры с истекшей проверкой фона | Расширенный поиск: [Custom Field](../settings/custom-fields.md) "Background check expires" is before today |
| Результаты из вопроса AI | Задайте вопрос в AI Search, затем сохраните |
