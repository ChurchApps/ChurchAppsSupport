# Массовое редактирование людей

<div class="article-intro">
Массовое редактирование позволяет вам обновлять несколько человек одновременно, экономя время при внесении одинаковых изменений для многих людей. Вы можете обновлять статус членства, семейное положение, пол, настройки отказа и членство в группах в одной операции.
</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Вам необходимо разрешение на управление данными о людях. См. [Роли и разрешения](./roles-permissions.md) для получения подробностей.
- Вы должны уже добавили или импортировали людей, которых хотите редактировать. См. [Добавление людей](./adding-people.md) при необходимости.
</div>

## Выбор людей для массового редактирования

1. Перейдите в раздел **Люди** в B1 Admin
2. Используйте инструменты поиска или фильтрации, чтобы найти людей, которых вы хотите обновить
3. Отметьте флажки рядом с именем каждого человека, чтобы выбрать их
   - Вы можете выбирать людей индивидуально
   - Или используйте флажок в заголовке, чтобы выбрать всех видимых людей на текущей странице
4. После того как вы выбрали хотя бы одного человека, появится кнопка **Массовые действия**

:::tip
Если вам нужно обновить большую группу людей на основе определенных критериев, используйте функцию [Поиск с помощью ИИ](./ai-search.md) или фильтры, чтобы сначала сузить ваш список, затем выберите всех соответствующих людей.
:::

## Доступные массовые действия

Меню **Массовые действия** предоставляет несколько опций:

### Обновление статуса членства

Обновите статус членства для всех выбранных людей:

1. Нажмите **Массовые действия** → **Установить статус членства**
2. Выберите новый статус:
   - **Посетитель** - Посетители впервые или периодические
   - **Регулярный посетитель** - Частые посетители, которые не являются членами
   - **Член** - Официальные члены церкви
   - **Персонал** - Сотрудники церкви
   - **Неактивный** - Люди, которые больше не посещают
3. Выберите режим обновления:
   - **Перезаписать все** - Заменить текущий статус для всех выбранных людей
   - **Обновить только пустые** - Установить статус только для людей, у которых его нет
4. Нажмите **Обновить**

### Обновление семейного положения

Обновите семейное положение массово:

1. Нажмите **Массовые действия** → **Установить семейное положение**
2. Выберите новый статус:
   - **Неизвестно**
   - **Одинокий**
   - **Женат/Замужем**
   - **Разведен/Разведена**
   - **Вдовец/Вдова**
3. Выберите, перезаписывать ли существующие значения или обновлять только пустые поля
4. Нажмите **Обновить**

### Обновление пола

Обновите информацию о поле для нескольких человек:

1. Нажмите **Массовые действия** → **Установить пол**
2. Выберите значение:
   - **Не указан**
   - **Мужской**
   - **Женский**
3. Выберите режим обновления (перезаписать все или только пустые)
4. Нажмите **Обновить**

### Обновление статуса отказа

Контролируйте, отказались ли люди от общения:

1. Нажмите **Массовые действия** → **Установить отказ**
2. Выберите:
   - **Нет** - Разрешить общение (снять отказ)
   - **Да** - Заблокировать общение (установить отказ)
3. Выберите режим обновления
4. Нажмите **Обновить**

:::warning
Будьте осторожны при изменении статуса отказа. Люди, которые явно отказались, не должны получать сообщения, если только они не дали новое согласие.
:::

### Добавление в группу

Добавьте всех выбранных людей в одну или несколько групп:

1. Нажмите **Массовые действия** → **Добавить в группу**
2. Найдите и выберите группу(ы), в которую нужно добавить людей
3. Вы можете выбрать несколько групп, чтобы добавить людей во все из них
4. Нажмите **Добавить в группы**

Каждый человек будет добавлен как обычный член выбранной группы(групп). Вы можете позже повысить отдельных людей до лидеров группы при необходимости со страницы [Члены группы](../groups/group-members.md).

### Удаление из группы

Удалите всех выбранных людей из одной или нескольких групп:

1. Нажмите **Массовые действия** → **Удалить из группы**
2. Найдите и выберите группу(ы), из которой нужно удалить людей
3. Вы можете выбрать несколько групп
4. Нажмите **Удалить из групп**

:::info
Это действие только удаляет людей из указанных групп. Оно не удаляет их записи о людях.
:::

### Удаление людей

Окончательно удалите выбранных людей из базы данных вашей церкви:

1. Нажмите **Массовые действия** → **Удалить**
2. Просмотрите список людей, которые будут удалены
3. Введите **DELETE** в поле подтверждения
4. Нажмите **Подтвердить удаление**

:::danger
Удаление людей является окончательным и не может быть отменено. Это удалит все их данные, включая:
- Личную информацию
- Членство в группах
- Записи о посещаемости
- Историю пожертвований
- Отправленные формы

Используйте это действие только в том случае, если вы абсолютно уверены, что хотите удалить этих людей из вашей системы.
:::

## Результаты массового редактирования

После завершения массового действия вы увидите сводку, показывающую:

- **Всего выбрано** - Сколько людей было включено в операцию
- **Успешно обновлено** - Сколько записей было изменено
- **Не удалось** - Любые записи, которые не удалось обновить (если применимо)
- **Без изменений** - Записи, которые не нуждались в изменениях (например, при использовании режима "обновить только пустые")

Если какие-либо обновления не удались, вы увидите детали ошибок, объясняющие причину.

## Лучшие практики

- **Начинайте с малого** - Тестируйте массовые операции на нескольких записях сначала, чтобы убедиться, что вы вносите правильные изменения
- **Используйте фильтры** - Сузьте свой список с помощью фильтров или поиска ИИ перед выбором людей, чтобы убедиться, что вы обновляете только нужных людей
- **Дважды проверяйте выбор** - Просмотрите выбранных людей перед применением массовых изменений
- **Используйте режим "обновить только пустые"** - Когда вы хотите заполнить отсутствующие данные без перезаписи существующей информации
- **Документируйте крупные изменения** - Сохраняйте заметки о массовых обновлениях на случай, если вам нужно будет сослаться на них позже
- **Координируйтесь с вашей командой** - Сообщайте другим администраторам, когда вносите крупные массовые изменения

## Распространенные случаи использования

### Обновление новых членов

После класса членства обновите всех посетителей до статуса Член:

1. Найдите людей, которые посетили класс
2. Выберите их всех
3. Используйте **Массовые действия** → **Установить статус членства** → **Член**

### Организация малых групп

Добавьте несколько человек в новую малую группу:

1. Найдите людей, которых вы хотите в группе
2. Выберите их
3. Используйте **Массовые действия** → **Добавить в группу** и выберите малую группу

### Очистка данных

Заполните отсутствующее семейное положение для супружеских пар:

1. Отфильтруйте людей, которые женаты (используя информацию о домохозяйстве)
2. Выберите тех, у кого пустое семейное положение
3. Используйте **Массовые действия** → **Установить семейное положение** → **Женат/Замужем** → **Обновить только пустые**
