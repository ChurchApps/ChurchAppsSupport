# Обзор B1 Admin

<div class="article-intro">

B1 Admin - это ваша панель управления церковью. Она предоставляет инструменты для управления всеми аспектами деятельности вашей церкви -- от справочников членов и пожертвований до создания веб-сайтов и прямых трансляций -- все в одном месте.

</div>

## Начало работы

1. Откройте браузер и перейдите на [admin.b1.church](https://admin.b1.church)
2. Войдите, используя ваше имя пользователя и пароль
3. Вы попадете на **Панель управления**, которая предоставляет обзор данных вашей церкви вместе с **Быстрыми действиями** для распространенных задач, таких как добавление людей, создание групп, запись пожертвований и многое другое

:::tip
Новичок в B1 Admin? Начните с [Введения](./introduction) для видеообзора, затем посетите [Настройки](./settings/), чтобы настроить информацию о вашей церкви и пригласить свою команду.
:::

## Ключевые функции

- **[Люди](./people/)** - Управляйте справочником вашей церкви, добавляйте членов, отслеживайте домохозяйства
- **[Группы](./groups/)** - Создавайте и организуйте церковные группы с запросами на присоединение и управлением участниками
- **[Посещаемость](./attendance/)** - Настраивайте кампусы, время служений и отслеживайте посещаемость
- **[Пожертвования](./donations/)** - Записывайте пожертвования в нескольких валютах, управляйте фондами, создавайте отчеты
- **[Служение](./serving/)** - Координируйте волонтеров, создавайте планы служения, управляйте задачами
- **[Формы](./forms/)** - Создавайте пользовательские формы для регистрации и сбора данных
- **[Отчеты](./reports/)** - Просматривайте отчеты о днях рождения, посещаемости и пожертвованиях
- **[Веб-сайт](./website/)** - Создавайте и управляйте веб-сайтом вашей церкви с пользовательскими стилями навигации
- **[Проповеди](./sermons/)** - Управляйте библиотекой проповедей и прямыми трансляциями
- **[Календари](./calendars/)** - Создавайте курируемые календари, связывая события групп
- **[Настройки](./settings/)** - Настройте информацию о церкви, роли и права доступа

:::info

:::
