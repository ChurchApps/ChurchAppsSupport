# Обзор B1 Admin

<div class="article-intro">

B1 Admin - это панель управления вашей церковью. Он предоставляет инструменты для управления каждым аспектом деятельности вашей церкви -- от справочников членов и пожертвований до создания веб-сайтов и прямых трансляций -- все в одном месте.

</div>

## Начало работы

1. Откройте браузер и перейдите на [admin.b1.church](https://admin.b1.church)
2. Войдите со своим именем пользователя и паролем
3. Вы попадете на **Панель управления**, которая предоставляет обзор данных вашей церкви вместе с **Быстрыми действиями** для общих задач, таких как добавление людей, создание групп, запись пожертвований и многое другое

:::tip
Вы новичок в B1 Admin? Начните с [Введения](./introduction) для видео-обзора, затем посетите [Параметры](./settings/) для конфигурации информации о вашей церкви и приглашения вашей команды.
:::

## Ключевые возможности

- **[Люди](./people/)** - управление справочником вашей церкви, добавление участников, массовое редактирование данных, отслеживание домохозяйств
- **[Группы](./groups/)** - создание и организация групп церкви с запросами присоединения и управлением участниками
- **[Посещаемость](./attendance/)** - установка кампусов, времени служений и отслеживание посещаемости
- **[Пожертвования](./donations/)** - запись благодарностей в нескольких валютах, управление фондами, создание отчетов
- **[Служение](./serving/)** - координация волонтеров, создание планов служений, управление задачами и перемещение людей через пошаговые процессы
- **[Формы](./forms/)** - создание пользовательских форм для регистрации и сбора данных
- **[Отчеты](./reports/)** - просмотр отчетов о днях рождения, посещаемости и пожертвованиях
- **[Веб-сайт](./website/)** - создание и управление веб-сайтом вашей церкви с пользовательскими стилями навигации
- **[Проповеди](./sermons/)** - управление библиотекой проповедей и прямыми трансляциями
- **[Календари](./calendars/)** - создание организованных календарей, управление бронированием комнат/ресурсов и просмотр доступности
- **[Параметры](./settings/)** - конфигурация информации о церкви, ролей, разрешений и пользовательских полей для люде
