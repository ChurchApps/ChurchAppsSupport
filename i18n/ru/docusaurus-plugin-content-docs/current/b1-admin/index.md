# Обзор B1 Admin

<div class="article-intro">

B1 Admin - это панель управления вашей церковью. Она предоставляет инструменты для управления всеми аспектами работы вашей церкви - от справочников членов и пожертвований до создания веб-сайтов и прямых трансляций - все в одном месте.

</div>

## Начало работы

1. Откройте браузер и перейдите на [admin.b1.church](https://admin.b1.church)
2. Войдите, используя ваше имя пользователя и пароль
3. Вы попадете на **Панель управления**, которая предоставляет обзор данных вашей церкви вместе с **Быстрыми действиями** для распространенных задач, таких как добавление людей, создание групп, запись пожертвований и многое другое

:::tip
Впервые в B1 Admin? Начните с [Введения](./introduction) для видео-обзора, затем посетите [Настройки](./settings/), чтобы настроить информацию о вашей церкви и пригласить вашу команду.
:::

## Основные возможности

- **[Люди](./people/)** - Управляйте справочником вашей церкви, добавляйте членов, массово редактируйте данные, отслеживайте домохозяйства
- **[Группы](./groups/)** - Создавайте и организуйте церковные группы с запросами на присоединение и управлением членами
- **[Посещаемость](./attendance/)** - Настраивайте кампусы, время служб и отслеживайте посещаемость
- **[Пожертвования](./donations/)** - Записывайте пожертвования в нескольких валютах, управляйте фондами, создавайте отчеты
- **[Служение](./serving/)** - Координируйте волонтеров, создавайте планы служения, управляйте задачами
- **[Формы](./forms/)** - Создавайте настраиваемые формы для регистрации и сбора данных
- **[Отчеты](./reports/)** - Просматривайте отчеты о днях рождения, посещаемости и пожертвованиях
- **[Веб-сайт](./website/)** - Создавайте и управляйте веб-сайтом вашей церкви с настраиваемыми стилями навигации
- **[Проповеди](./sermons/)** - Управляйте библиотекой проповедей и прямыми трансляциями
- **[Календари](./calendars/)** - Создавайте кураторские календари, связывая групповые события
- **[Настройки](./settings/)** - Настраивайте информацию о церкви, роли и разрешения

:::
