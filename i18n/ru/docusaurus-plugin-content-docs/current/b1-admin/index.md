# Обзор B1 Admin

<div class="article-intro">

B1 Admin - это приборная панель управления вашей церкви. Она предоставляет инструменты для управления всеми аспектами деятельности вашей церкви — от справочников членов и пожертвований до создания сайта и прямой трансляции — все в одном месте.

</div>

## Начало работы

1. Откройте браузер и перейдите на [admin.b1.church](https://admin.b1.church)
2. Войдите с вашим именем пользователя и паролем
3. Вы попадете на **Приборную панель**, которая предоставляет обзор данных вашей церкви вместе с **Быстрыми действиями**

:::tip
Новичок в B1 Admin? Начните с [Введения](./introduction) для видео обзора, затем посетите [Параметры](./settings/) для настройки информации вашей церкви.
:::

## Ключевые функции

- **[Люди](./people/)** - Управляйте справочником, добавляйте членов
- **[Группы](./groups/)** - Создавайте и организуйте группы
- **[Посещаемость](./attendance/)** - Настройте кампусы и отслеживайте посещаемость
- **[Пожертвования](./donations/)** - Записывайте пожертвования и управляйте фондами
- **[Служение](./serving/)** - Координируйте добровольцев и планы служения
- **[Формы](./forms/)** - Создавайте пользовательские формы
- **[Отчеты](./reports/)** - Просматривайте отчеты о днях рождений и посещаемости
- **[Сайт](./website/)** - Создавайте и управляйте веб-сайтом церкви
- **[Проповеди](./sermons/)** - Управляйте библиотекой проповедей
- **[Календари](./calendars/)** - Создавайте и публикуйте календари событий
- **[Параметры](./settings/)** - Настройте информацию церкви и разрешения

:::info
Нажмите значок вопроса в верхнем правом углу для быстрого доступа к справке.
:::

Для поддержки напишите на [support@churchapps.org](mailto:support@churchapps.org).
