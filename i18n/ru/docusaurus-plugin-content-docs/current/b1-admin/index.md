# Обзор B1 Admin

<div class="article-intro">

B1 Admin — это панель управления вашей церковью. Она предоставляет инструменты для управления всеми аспектами деятельности вашей церкви — от справочников прихожан и пожертвований до создания сайта и прямых трансляций — всё в одном месте.

</div>

## Начало работы

1. Откройте браузер и перейдите на [admin.b1.church](https://admin.b1.church)
2. Войдите с вашим именем пользователя и паролем
3. Вы попадёте на **Панель управления**, которая обеспечивает быстрый доступ ко всем основным функциям

:::tip
Впервые в B1 Admin? Начните с [Введения](./introduction) для видео-обзора, затем перейдите в [Настройки](./settings/), чтобы настроить информацию о вашей церкви и пригласить команду.
:::

## Основные функции

- **[Люди](./people/)** — Управление справочником церкви, добавление прихожан, отслеживание семей
- **[Группы](./groups/)** — Создание и организация церковных групп, команд и служений
- **[Посещаемость](./attendance/)** — Настройка кампусов, времени богослужений и учёт посещаемости
- **[Пожертвования](./donations/)** — Запись пожертвований, управление фондами, формирование справок
- **[Служение](./serving/)** — Координация волонтёров, создание планов богослужений, управление задачами
- **[Формы](./forms/)** — Создание пользовательских форм для регистрации и сбора данных
- **[Отчёты](./reports/)** — Просмотр отчётов о днях рождения, посещаемости и пожертвованиях
- **[Сайт](./website/)** — Создание и управление сайтом церкви
- **[Проповеди](./sermons/)** — Управление библиотекой проповедей и прямыми трансляциями
- **[Календари](./calendars/)** — Создание и публикация календарей событий
- **[Настройки](./settings/)** — Настройка информации о церкви, ролей и разрешений
