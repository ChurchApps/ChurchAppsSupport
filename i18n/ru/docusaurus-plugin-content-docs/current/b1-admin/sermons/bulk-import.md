# Массовый импорт

<div class="article-intro">

Страница «Массовый импорт» позволяет быстро наполнить библиотеку проповедей, импортируя видео с YouTube или Vimeo. Это самый быстрый способ перенести существующий контент с проповедями вместо добавления каждого видео по одному.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Вам необходимо разрешение **contentApi.streamingServices.edit**. См. [Роли и разрешения](../settings/roles-permissions.md), если у вас нет доступа.
- Создайте хотя бы один [плейлист](playlists), в который будут импортированы проповеди
- Подготовьте идентификатор вашего канала YouTube или данные учётной записи Vimeo

</div>

## Выбор источника

1. В B1 Admin нажмите **Проповеди** в левой боковой панели, затем нажмите вкладку **Массовый импорт**.
2. Вы увидите две карточки:
   - **YouTube** (красная) — импорт видео с канала YouTube
   - **Vimeo** (синяя) — импорт видео с учётной записи Vimeo
3. Нажмите на карточку платформы, на которой размещены ваши проповеди.

:::tip
Вы можете в любой момент вернуться и переключить источник, если у вас есть видео на обеих платформах.
:::

## Перед импортом

Перед импортом видео вам нужен хотя бы один плейлист для их организации. Если вы ещё не создали плейлисты:

1. Нажмите вкладку **Плейлисты**.
2. Нажмите **Создать первый плейлист** и заполните название, описание, дату публикации и миниатюру.
3. Нажмите **Сохранить**, затем вернитесь на вкладку **Массовый импорт**.

Подробные инструкции по созданию и управлению плейлистами см. в разделе [Плейлисты](playlists).

## Импорт видео

1. После выбора YouTube или Vimeo введите **идентификатор канала YouTube** или **данные учётной записи Vimeo** в указанное поле.
2. Нажмите кнопку **Загрузить**, чтобы получить все доступные видео с вашего канала или учётной записи.
3. После загрузки вы увидите список всех ваших видео с чекбоксами.
4. Отметьте видео, которые хотите импортировать. Вы можете выбрать все или отдельные.
5. При необходимости включите **Автоматический импорт новых видео**, чтобы будущие загрузки автоматически добавлялись в библиотеку.
6. Нажмите выпадающий список **Импортировать в плейлист**, чтобы выбрать, в какой плейлист будут добавлены эти видео.
7. Нажмите кнопку **Импорт** для завершения массового импорта.

Ваши видео будут импортированы со всеми деталями, включая названия, описания, даты и миниатюры.

:::info
Массовый импорт идеально подходит для начала работы при переходе с другой платформы. Для добавления отдельных проповедей в дальнейшем используйте страницу [Управление проповедями](managing-sermons).
:::
