# Управление проповедями

<div class="article-intro">

На странице «Проповеди» отображается вся ваша библиотека проповедей. Здесь вы можете добавлять новые проповеди, редактировать существующие записи и организовывать контент по плейлистам. Каждая проповедь может содержать ссылку на видео или аудио, размещённое на YouTube, Vimeo, Facebook или по пользовательскому URL.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Вам необходимо разрешение **contentApi.streamingServices.edit**. См. [Роли и разрешения](../settings/roles-permissions.md), если у вас нет доступа.
- Создайте хотя бы один [плейлист](playlists) для организации ваших проповедей
- Подготовьте идентификаторы видео или URL-адреса с YouTube, Vimeo или Facebook

</div>

## Просмотр библиотеки проповедей

1. В B1 Admin нажмите **Проповеди** в левой боковой панели.
2. На странице «Проповеди» отображаются все ваши записи проповедей, организованные по плейлистам. Каждая проповедь показывает миниатюру, название и дату.
3. Нажмите на любую проповедь, чтобы просмотреть или отредактировать её детали.

## Добавление проповеди

1. Нажмите кнопку **Добавить проповедь** в правом верхнем углу и выберите **Добавить проповедь** из выпадающего списка.
2. Выберите **Плейлист**, в который будет добавлена проповедь.
3. Выберите **Провайдер видео** — YouTube, Vimeo, Facebook или пользовательский URL. Мы рекомендуем YouTube, так как он лучше всего работает с системой B1.
4. Введите идентификатор видео или URL и нажмите **Загрузить**. Для YouTube идентификатор видео — это строка символов после `v=` в URL YouTube.
5. При нажатии **Загрузить** детали проповеди импортируются автоматически, включая дату публикации, продолжительность, название, описание и миниатюру.
6. Внесите необходимые изменения и нажмите **Сохранить**.

:::tip
Вы также можете добавить постоянный URL прямой трансляции, выбрав **Добавить постоянный URL трансляции** из выпадающего списка **Добавить проповедь**. Это создаёт постоянное подключение к прямой трансляции вашего канала YouTube через идентификатор канала. Подробнее см. [Прямые трансляции](live-streaming).
:::

## Редактирование проповеди

1. Нажмите на любую проповедь в библиотеке, чтобы открыть её детали.
2. Обновите название, проповедника, дату, описание, миниатюру или ссылки на медиа по необходимости.
3. Нажмите **Сохранить**, чтобы применить изменения.

## Детали проповеди

Каждая запись проповеди может включать:

- **Название** — название проповеди, отображаемое для посетителей
- **Проповедник** — кто произнёс проповедь
- **Дата** — дата публикации или произнесения
- **Описание** — краткое содержание проповеди
- **Миниатюра** — превью-изображение, отображаемое в библиотеке проповедей
- **Ссылки на видео/аудио** — URL-адреса медиа проповеди на YouTube, Vimeo, Facebook или пользовательском хостинге

## Планирование проповеди для прямой трансляции

После добавления проповеди вы можете запланировать её для трансляции на странице прямой трансляции:

1. Перейдите на вкладку **Время прямых трансляций**.
2. Отредактируйте богослужение и в разделе **Настройки видео** выберите вашу проповедь из выпадающего списка.
3. Проповедь будет воспроизведена в запланированное время богослужения.

:::info
Для импорта нескольких проповедей сразу вместо добавления по одной используйте инструмент [Массовый импорт](bulk-import), чтобы загрузить видео напрямую с вашей учётной записи YouTube или Vimeo.
:::
