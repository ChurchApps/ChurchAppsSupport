# Профиль

<div class="article-intro">

Раздел «Профиль» позволяет управлять настройками вашей личной учётной записи в ChurchApps. Ваш профиль является общим для всех инструментов ChurchApps, поэтому изменения, внесённые здесь, применяются повсюду -- от B1 Admin до мобильного приложения B1.

</div>

## Что вы можете сделать

В разделе «Профиль» вы можете:

- **Обновить имя и электронную почту** -- Поддерживайте актуальность своей контактной информации.
- **Изменить пароль** -- Установите новый пароль для своей учётной записи.
- **Сменить тему** -- Переключайтесь между светлым и тёмным режимом по своему предпочтению.
- **Управлять подключёнными устройствами** -- Просматривайте и контролируйте, какие устройства ChurchAppsPlayer привязаны к вашей учётной записи.
- **Просматривать связанные аккаунты** -- Просматривайте и управляйте внешними сервисами, подключёнными к вашей учётной записи ChurchApps.

## Подразделы профиля

Область «Профиль» содержит следующие страницы:

1. **[Управление профилем](./managing-profile.md)** -- Редактируйте личную информацию, меняйте пароль, управляйте связанными аккаунтами, настраивайте предпочтения темы и удаляйте учётную запись.
2. **[Управление устройствами](./devices.md)** -- Просматривайте, добавляйте и управляйте устройствами ChurchAppsPlayer, связанными с вашей учётной записью.

:::tip
Поддерживайте актуальность вашего адреса электронной почты. Он используется для восстановления учётной записи и получения важных уведомлений от вашей церкви.
:::
