# Создание форм

<div class="article-intro">

Создавайте пользовательские формы для сбора информации от вашей общины. Вы можете создавать формы для регистрации на мероприятия, опросов, карточек посетителей, заявок на членство и многого другого. Формы могут быть связаны с людьми в вашей базе данных или использоваться как отдельные страницы со своим публичным URL.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Для форм типа **Люди** (привязанных к записям о людях) сначала необходимо [добавить людей в базу данных](../people/adding-people.md).
- Для форм, собирающих **платежи**, необходимо [настроить Stripe для онлайн-пожертвований](../donations/online-giving-setup.md).

</div>

## Создание новой формы

1. Перейдите в раздел **Формы** из главного меню.
2. Нажмите **Add Form**.
3. Введите **название** вашей формы.
4. Выберите тип формы из выпадающего списка:
   - **People** — Связывает ответы с [записями о людях](../people/adding-people.md) в вашей базе данных.
   - **Stand Alone** — Создаёт независимую форму со своим публичным URL, идеально подходящую для внешней регистрации.
5. Нажмите **Save**, чтобы создать форму.

Ваша новая форма появится в списке. Нажмите на неё, чтобы начать добавлять вопросы.

## Добавление вопросов

1. Откройте форму и перейдите на вкладку **Вопросы**.
2. Нажмите **Add Question**.
3. Выберите **тип поля** из выпадающего списка Provider. Доступные типы включают:
   - **Textbox** — Для коротких текстовых ответов
   - **Date** — Для выбора даты
   - **Email** — Для адресов электронной почты
   - **Phone Number** — Для ввода телефонного номера
   - **Multiple Choice** — Для выбора из предложенных вариантов
   - **Payment** — Для сбора платежей
4. Введите **Название** и необязательное **Описание** для вопроса.
5. Отметьте **Require an answer**, если поле является обязательным.
6. Нажмите **Save**.
7. Повторите для добавления дополнительных вопросов.

:::warning
Тип поля **Payment** требует настройки Stripe. Если вы ещё не настроили онлайн-пожертвования, см. [Настройка онлайн-пожертвований](../donations/online-giving-setup.md) перед добавлением полей оплаты.
:::

## Управление участниками формы

1. Откройте форму и перейдите на вкладку **Участники**.
2. Найдите человека и добавьте его с ролью:
   - **Admin** — Может редактировать форму и просматривать все ответы.
   - **View Only** — Может просматривать ответы, но не может редактировать форму.

## Настройка свойств формы

Вы можете обновить название и настройки формы в любое время. Для форм типа Stand Alone вы также увидите уникальный **публичный URL**, которым можно поделиться с кем угодно.

:::tip
Формы типа Stand Alone отлично подходят для регистрации на мероприятия. Делитесь публичным URL по электронной почте, в социальных сетях или встраивайте форму прямо на сайт вашей церкви.
:::

:::info
Чтобы встроить форму на ваш сайт B1, перейдите в редактор сайта, добавьте новый раздел и выберите элемент **Form**. Затем выберите форму, которую хотите отобразить. Подробнее см. [Управление страницами](../website/managing-pages.md).
:::
