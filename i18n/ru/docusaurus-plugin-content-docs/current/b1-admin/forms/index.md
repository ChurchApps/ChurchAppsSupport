# Формы

<div class="article-intro">

Раздел "Формы" позволяет создавать пользовательские формы для регистрации, опросов, сбора данных и многого другого. Независимо от того, нужна ли вам карточка посетителя, запись на мероприятие или форма молитвенной просьбы, вы можете создать её здесь и поделиться с вашей общиной через публичную ссылку или ваш сайт B1.church.

</div>

## Страница форм

Когда вы открываете раздел **Формы**, вы увидите две вкладки:

- **Формы** -- Показывает все ваши активные формы.
- **Архив** -- Показывает формы, которые были архивированы. Вы можете восстановить архивированные формы в любое время.

Каждая форма в списке отображает своё название и, если настроено, публичный URL, который можно использовать для внешнего распространения.

## Что вы можете сделать

На странице форм вы можете:

- **Добавить форму** -- Нажмите **Add Form**, чтобы создать новую пользовательскую форму.
- **Редактировать форму** -- Нажмите на название формы, чтобы открыть её и изменить вопросы, участников или настройки.
- **Архивировать форму** -- Перенесите формы, которые вам больше не нужны, на вкладку **Архив**, чтобы поддерживать порядок в списке.
- **Восстановить форму** -- Перейдите на вкладку **Архив** и восстановите любую форму, которую хотите сделать активной снова.

## Работа с формой

Нажмите на название любой формы, чтобы открыть её. Редактор формы содержит три вкладки:

- **Вопросы** -- Создайте структуру формы, добавляя поля различных типов. Подробнее см. [Создание форм](./creating-forms.md).
- **Участники** -- Управляйте доступом к форме и её администрированием.
- **Ответы** -- Просматривайте полученные ответы. Подробнее см. [Управление ответами](./managing-submissions.md).

:::tip
Формы с публичным URL можно распространять по ссылке, встраивать на ваш сайт или включать в электронные письма. Это упрощает сбор информации от людей, у которых может не быть церковной учётной записи.
:::

:::info
Вы можете контролировать доступ к каждой форме с помощью вкладки **Участники**. Назначайте роли **Администратор** или **Только просмотр**, чтобы предоставить вашей команде соответствующий уровень доступа.
:::
