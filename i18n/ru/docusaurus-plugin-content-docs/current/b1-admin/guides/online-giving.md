# Настройка онлайн-пожертвований

<div class="article-intro">

Пошаговая инструкция по всему необходимому для приёма онлайн-пожертвований в вашей церкви — от создания фондов пожертвований до подключения Stripe для обработки платежей и публикации страницы для пожертвований. По завершении прихожане смогут жертвовать онлайн через ваш сайт и мобильное приложение.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Учётная запись B1 Admin с правами администратора — см. [Роли и разрешения](../people/roles-permissions.md)
- Учётная запись Stripe (создайте бесплатно на [stripe.com](https://stripe.com), если нужно)

</div>

## Шаг 1: Создайте фонды пожертвований

Фонды — это категории, в которые жертвователи могут направлять свои пожертвования. Вам нужен хотя бы один фонд, прежде чем вы сможете принимать пожертвования.

Следуйте руководству [Фонды](../donations/funds.md), чтобы настроить категории пожертвований:

1. Создайте наиболее распространённые фонды (например, «Общий фонд», «Строительный фонд», «Миссии»)
2. Отметьте фонды, подлежащие налоговому вычету — это влияет на годовые отчёты о пожертвованиях

:::tip
Вы можете добавить больше фондов в любое время. Начните с наиболее распространённых категорий пожертвований.
:::

## Шаг 2: Подключите Stripe

Stripe обрабатывает все платежи. Вы подключите свою учётную запись Stripe к B1 Admin, чтобы пожертвования поступали на ваш банковский счёт.

Следуйте руководству [Настройка онлайн-пожертвований](../donations/online-giving-setup.md), чтобы подключить Stripe:

1. Войдите в панель управления Stripe и получите Публичный ключ (Publishable Key) и Секретный ключ (Secret Key)
2. В B1 Admin перейдите в Настройки и введите оба ключа

:::warning
Stripe показывает ваш Секретный ключ только один раз. Скопируйте и сохраните его перед тем, как покинуть панель управления Stripe. Если вы его потеряете, вам придётся сгенерировать новый.
:::

## Шаг 3: Добавьте страницу пожертвований на ваш сайт

Сделайте пожертвования доступными, добавив страницу пожертвований на ваш сайт B1.

Следуйте руководствам [Настройка онлайн-пожертвований](../donations/online-giving-setup.md) и [Управление страницами](../website/managing-pages.md), чтобы:

1. Добавить вкладку «Пожертвовать» на ваш сайт B1.church
2. Ваш URL для пожертвований будет: `https://yoursubdomain.b1.church/donate`
3. Прихожане могут жертвовать без входа в систему (публичная страница) или войти для сохранённых методов оплаты и истории пожертвований

## Шаг 4: Сделайте тестовое пожертвование

Прежде чем объявить прихожанам, убедитесь, что всё работает.

1. Сделайте небольшое тестовое пожертвование, чтобы проверить работу процесса от начала до конца
2. Проверьте, что пожертвование отображается в B1 Admin в разделе «Пожертвования»

:::tip
Сначала используйте тестовый режим Stripe, если хотите проверить без реальных списаний, затем переключитесь в рабочий режим перед объявлением прихожанам.
:::

## Шаг 5: Сообщите прихожанам

Распространите информацию, чтобы прихожане знали о возможности жертвовать онлайн.

1. Поделитесь URL для пожертвований через ваш сайт, рассылки по электронной почте, бюллетени и социальные сети
2. Прихожане также могут жертвовать через [приложение B1 Mobile](../../b1-mobile/giving/) — функция пожертвований встроена

:::info
Прихожане, которые войдут в систему, могут сохранять методы оплаты, настраивать регулярные пожертвования и просматривать историю пожертвований. Анонимные пожертвования тоже возможны — вход не требуется.
:::

## Шаг 6: Текущее управление

Поддерживайте актуальность записей о пожертвованиях и формируйте отчёты в течение года.

1. [Импортируйте транзакции Stripe](../donations/stripe-import.md) регулярно (еженедельно или ежемесячно), чтобы ваши записи были актуальными
2. [Просматривайте отчёты о пожертвованиях](../donations/donation-reports.md), чтобы отслеживать тенденции и итоги по фондам
3. [Формируйте годовые отчёты о пожертвованиях](../donations/giving-statements.md) для налоговой отчётности жертвователей

:::tip
Выполняйте импорт из Stripe не реже одного раза в месяц, чтобы ваши записи оставались актуальными. См. [Руководство по годовым отчётам](./year-end-reports.md) для полного описания процесса подготовки к концу года.
:::

## Готово!

Ваша церковь теперь принимает онлайн-пожертвования. Прихожане могут жертвовать через ваш сайт, приложение B1 Mobile или любое устройство с веб-браузером. Все пожертвования автоматически отслеживаются в B1 Admin.
