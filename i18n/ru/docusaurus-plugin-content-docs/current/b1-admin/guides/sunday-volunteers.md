# Управление добровольцами в воскресенье

Настройте команды добровольцев, создавайте еженедельные планы служения, назначайте позиции и автоматизируйте повторяющиеся задачи.

## Шаг 1: Создание команд добровольцев

Организуйте добровольцев в команды по направлениям служения.

Следуйте руководствам [Создание групп](../groups/creating-groups.md) и [Члены группы](../groups/group-members.md):

1. Создайте категорию (например, "Команды воскресенского утра")
2. Создайте группы (например, "Команда поклонения", "Звук и медиа")
3. Добавьте добровольцев в каждую группу

:::tip
Назначьте лидеров команд используя зеленый ключ.
:::

## Шаг 2: Создание библиотеки песен

Добавьте песни поклонения.

Следуйте руководству [Песни](../serving/songs.md).

## Шаг 3: Создание плана служения

Планы - это еженедельные задания.

Следуйте руководству [Планы](../serving/plans.md).

## Шаг 4: Создание порядка служения

Разложите поток вашей службы.

Следуйте руководству [Порядок служения](../serving/service-order.md).

## Шаг 5: Настройка повторяющихся задач

Автоматизируйте еженедельные обязанности.

Следуйте руководствам [Задачи](../serving/tasks.md) и [Автоматизация](../serving/automations.md).

## Шаг 6: Включение самостоятельной регистрации (необязательно)

Позвольте добровольцам регистрироваться на позиции сами.

1. При редактировании плана установите **Сроком регистрации** (часы перед служением)
2. Опционально переключите **Показывать имена добровольцев**
3. Для каждой позиции установите флажок **Разрешить самостоятельную регистрацию**
4. Добавьте **описание** для каждой позиции

## Шаг 7: Уведомление добровольцев

Назначенные добровольцы автоматически видят свои планы.

## Готово!

Ваша система управления добровольцами установлена.
