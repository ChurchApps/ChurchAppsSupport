# Учёт посещаемости церкви

<div class="article-intro">

Настройте учёт посещаемости для ваших богослужений и групп — от настройки кампусов и времени богослужений до записи посещаемости вручную или через приложение самостоятельной регистрации, до просмотра тенденций в отчётах. По завершении у вас будет полная картина того, кто посещает и когда.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Учётная запись B1 Admin с правами администратора
- [Люди добавлены в ваш справочник](../people/adding-people.md)
- [Группы созданы](../groups/creating-groups.md) для служений, которые вы хотите отслеживать

</div>

## Шаг 1: Настройте кампусы и богослужения

Определите, где и когда проходят встречи вашей церкви.

Следуйте руководству [Настройка посещаемости](../attendance/setup.md), чтобы:

1. Добавить ваш кампус (например, «Основной кампус»)
2. Добавить время богослужений (например, «Воскресенье 9:00», «Воскресенье 11:00», «Среда 19:00»)
3. Привязать группы к каждому времени богослужения, чтобы определить, какие занятия/служения проходят в это время

## Шаг 2: Включите учёт посещаемости для групп

Включите учёт посещаемости для групп, которые вы хотите отслеживать.

Следуйте руководству [Создание групп](../groups/creating-groups.md), чтобы:

1. Отредактировать каждую группу, которую хотите отслеживать, и включить «Учёт посещаемости»
2. Связать группы с временем богослужений, чтобы посещаемость записывалась для правильного богослужения

:::info
Не каждой группе нужен учёт посещаемости. Включайте его для групп, где вы хотите отслеживать участие — детские занятия, малые группы, воскресная школа и т.д.
:::

## Шаг 3: Записывайте посещаемость вручную

Отмечайте, кто посещал каждое богослужение и группу.

Следуйте руководству [Запись посещаемости](../attendance/tracking-attendance.md), чтобы:

1. Перейти в раздел «Посещаемость», выбрать дату и богослужение
2. Отметить, кто посещал каждую группу

:::tip
Ручная запись отлично подходит для взрослых богослужений, малых групп и любых ситуаций, где терминал регистрации непрактичен.
:::

## Шаг 4: Настройте самостоятельную регистрацию (необязательно)

Позвольте прихожанам регистрироваться самостоятельно через терминал на планшете.

Следуйте руководству [Регистрация](../attendance/check-in.md), чтобы:

1. Установить приложение B1 Checkin на планшет для самостоятельной регистрации
2. Прихожане ищут своё имя и регистрируют себя и свою семью
3. Записи регистрации появляются вместе с введёнными вручную записями — никакой разницы в данных

:::info
Для полной настройки детского служения, включая оборудование и конфигурацию принтера, см. [Руководство по регистрации детей](./childrens-checkin.md).
:::

## Шаг 5: Просмотр отчётов о посещаемости

Получите общую картину участия в вашей церкви.

Следуйте руководству [Отчёты о посещаемости](../reports/attendance-reports.md). Доступны три вида отчётов:

1. **Тенденции посещаемости** — отслеживайте посещаемость во времени для выявления роста или спада
2. **Посещаемость по группам** — сравнивайте итоги по группам
3. **Ежедневная посещаемость по группам** — подневная разбивка для детального анализа

:::info
Отчёты объединяют как вручную записанную, так и данные самостоятельной регистрации. Нет разницы в отображении данных — оба метода попадают в одни и те же отчёты.
:::

## Готово!

Учёт посещаемости настроен. Записываете ли вы посещаемость вручную, используете приложение регистрации или оба варианта — все данные попадают в одни и те же отчёты, и вы получаете чёткую картину участия в вашей церкви.
