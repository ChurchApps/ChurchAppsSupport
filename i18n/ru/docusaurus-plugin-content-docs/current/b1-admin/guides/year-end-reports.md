# Формирование годовых отчётов о пожертвованиях

<div class="article-intro">

Пошаговое руководство по процессу завершения года: финализация записей о пожертвованиях, проверка настроек фондов и формирование справок о налоговых вычетах для каждого жертвователя. Обычно это делается в начале января за предыдущий календарный год.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Учётная запись B1 Admin с доступом к финансам
- Пожертвования записывались в течение года (онлайн через Stripe и/или вручную)
- Доступ к вашей учётной записи Stripe, если вы принимаете онлайн-пожертвования

</div>

## Шаг 1: Импортируйте последние транзакции Stripe

Убедитесь, что все онлайн-пожертвования за конец года есть в вашей системе.

Следуйте руководству [Импорт из Stripe](../donations/stripe-import.md), чтобы:

1. Перейти в Пожертвования > Пакеты > Импорт из Stripe
2. Выбрать диапазон дат, охватывающий конец года (например, 1 декабря — 31 декабря)
3. Сначала нажать «Предпросмотр» для проверки, затем «Импортировать отсутствующие» для завершения

:::warning
Выполните этот импорт перед формированием справок. Любые неимпортированные транзакции не появятся в справках жертвователей.
:::

## Шаг 2: Проверьте отчёты о пожертвованиях

Убедитесь в точности ваших записей перед формированием справок.

Следуйте руководству [Отчёты о пожертвованиях](../donations/donation-reports.md), чтобы:

1. Проверить страницу сводки пожертвований за весь год
2. Сравнить итоги по фондам с банковскими выписками для выявления расхождений
3. При необходимости просмотреть отдельные пакеты для проверки данных на уровне жертвователей

## Шаг 3: Проверьте налоговый статус фондов

Убедитесь, что настройка налогового вычета для каждого фонда корректна, чтобы справки были точными.

Следуйте руководству [Фонды](../donations/funds.md), чтобы:

1. Открыть каждый фонд и подтвердить правильность настройки налогового вычета

:::info
Только пожертвования в фонды, отмеченные как подлежащие налоговому вычету, появятся в справках. Если фонд должен быть налоговым вычетом, но не отмечен соответствующим образом, обновите его перед формированием справок.
:::

## Шаг 4: Сформируйте справки о пожертвованиях

Создайте официальные справки о пожертвованиях для ваших жертвователей.

Следуйте руководству [Справки о пожертвованиях](../donations/giving-statements.md), чтобы:

1. Перейти в Пожертвования > Справки
2. Выбрать год из выпадающего списка и просмотреть сводную статистику
3. Выбрать способ загрузки:
   - **Скачать ZIP** — отдельные CSV-файлы, по одному на жертвователя
   - **Печать всех** — версия для печати, каждая справка на новой странице

:::tip
Формируйте справки в начале января, пока записи свежи. Это даст вам время обнаружить возможные ошибки перед отправкой.
:::

## Шаг 5: Распределите жертвователям

Доставьте справки жертвователям.

1. Распечатайте и отправьте справки по почте или разошлите отдельные CSV-файлы по электронной почте
2. Прихожане также могут просмотреть свою историю пожертвований и распечатать справки на [B1.church](../../b1-church/giving/donation-history.md) и в [приложении B1 Mobile](../../b1-mobile/giving/donation-history.md)

## Готово!

Ваши годовые отчёты о пожертвованиях готовы. Жертвователи получили справки о налоговых вычетах, а финансовые записи за год завершены.
