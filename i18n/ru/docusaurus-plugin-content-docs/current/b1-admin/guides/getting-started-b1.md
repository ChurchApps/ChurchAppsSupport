# Начните работу с B1 для вашей церкви

<div class="article-intro">

Полное руководство по началу работы для новых церквей — от создания учётной записи до импорта справочника членов и настройки первых функций. Следуйте этим шагам по порядку, чтобы запустить вашу церковь на B1.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Действующий адрес электронной почты
- Логотип и материалы фирменного стиля вашей церкви (цвета, изображения) — рекомендуется, но не обязательно для начала

</div>

## Шаг 1: Создайте учётную запись

Зарегистрируйте вашу церковь и станьте первым администратором.

Следуйте руководству [Создание учётной записи](../../getting-started/create-account.md):

1. Перейдите на b1.church и зарегистрируйтесь, используя вашу электронную почту
2. Подтвердите ваш адрес электронной почты
3. Выберите «Создать новую церковь» и введите название вашей церкви
4. Вы автоматически становитесь администратором домена с полным доступом ко всему

## Шаг 2: Настройте параметры церкви

Введите основную информацию и фирменный стиль вашей церкви.

Следуйте руководству [Настройки церкви](../settings/church-settings.md):

1. Введите название церкви, адрес, номер телефона и электронную почту
2. Установите ваш поддомен (он станет вашацерковь.b1.church)
3. Загрузите логотип и изображения вашей церкви

:::tip
Ваш поддомен отображается в URL сайта и страницы пожертвований, поэтому выберите что-то короткое и узнаваемое (например, «blagodat» для Церкви Благодати).
:::

## Шаг 3: Настройте роли и пригласите свою команду

Управляйте тем, у кого есть доступ к чему, и подключите свою команду.

Следуйте руководству [Роли и разрешения](../settings/roles-permissions.md):

1. Создайте роли для вашей команды (например, «Пастор», «Офис-администратор», «Финансовая группа», «Руководитель поклонения»)
2. Назначьте конкретные разрешения каждой роли, чтобы люди видели только то, что им нужно
3. Пригласите членов команды, добавив их и назначив роли

:::info
Вам не нужно делать всё самостоятельно. Пригласите офисный персонал, пасторов и руководителей служений заранее, чтобы они могли помочь настроить свои направления.
:::

## Шаг 4: Импортируйте ваших людей

Перенесите существующий справочник членов в B1.

Следуйте руководству [Импорт данных](../people/importing-data.md). У вас есть два варианта:

1. **Загрузка CSV** — экспортируйте из текущей системы или электронной таблицы, отформатируйте как CSV, загрузите в B1
2. **Миграция из Breeze** — импорт одним кликом, который переносит людей, фотографии, группы, пожертвования и посещаемость

:::tip
Переходите с Breeze? Инструмент импорта переносит всё за один шаг — людей, фотографии, группы, пожертвования и историю посещаемости. Это самый быстрый способ начать.
:::

## Шаг 5: Настройте мобильное приложение

Настройте приложение B1 Mobile для вашей общины.

Следуйте руководству [Настройки мобильного приложения](../settings/mobile-app.md):

1. Настройте, какие вкладки отображаются в приложении B1 Mobile для ваших членов
2. Установите названия вкладок, значки и типы (Библия, Прямой эфир, Пожертвование, Веб-сайт и т.д.)
3. Управляйте тем, какие функции видны вашей общине

## Шаг 6: Начните настройку функций

Теперь, когда основа готова, следуйте этим руководствам для настройки конкретных функций:

- [Настройте онлайн-пожертвования](./online-giving.md) — принимайте пожертвования через сайт и приложение
- [Запустите сайт вашей церкви](./launch-website.md) — создайте сайт вашей церкви
- [Организуйте малые группы](./small-groups.md) — организуйте группы, изучение Библии и служения
- [Настройте регистрацию детей](./childrens-checkin.md) — регистрация через киоск для детского служения
- [Управляйте воскресными волонтёрами](./sunday-volunteers.md) — координируйте команды служения
- [Создайте библиотеку проповедей](./sermon-library.md) — выложите проповеди онлайн и настройте прямые трансляции
- [Отслеживайте посещаемость](./track-attendance.md) — следите за тем, кто посещает богослужения и группы

## Готово!

Ваша церковь настроена на B1. У вашей команды есть доступ, ваши люди импортированы, и вы готовы начать использовать функции, наиболее важные для вашего служения. Изучите руководства по функциям выше, чтобы настроить каждое направление.
