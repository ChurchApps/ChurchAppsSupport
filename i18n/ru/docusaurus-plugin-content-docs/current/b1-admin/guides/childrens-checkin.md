# Настройка регистрации для детского служения

<div class="article-intro">

Это руководство проведёт вас через всё необходимое для запуска системы регистрации детей в вашей церкви — от внесения семей в базу данных, до настройки возрастных групп, до печати именных бейджей в воскресное утро. В итоге родители смогут регистрировать своих детей на планшете-киоске и получать соответствующий бейдж безопасности.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Создайте учётную запись вашей церкви на [admin.b1.church](https://admin.b1.church)
- Убедитесь, что у вас есть доступ администратора — при необходимости см. [Роли и разрешения](../people/roles-permissions.md)
- По желанию: Подготовьте CSV-файл с данными семей, если вы переходите с другой системы

</div>

## Шаг 1: Добавьте семьи в вашу базу данных

Прежде чем регистрация начнёт работать, система должна знать о ваших семьях. Каждый ребёнок должен быть связан с родителем через функцию домохозяйства.

Следуйте руководству [Добавление людей](../people/adding-people.md), чтобы добавить хотя бы одну семью. Обязательно:

- Сначала добавьте родителя(ей)
- Добавьте каждого ребёнка
- Свяжите их в одном домохозяйстве с помощью [редактора домохозяйств](../people/adding-people.md#managing-households)

:::tip
Если вам нужно добавить больше нескольких семей, используйте инструмент [импорта CSV](../people/importing-data.md) вместо добавления по одной. Вы можете импортировать весь справочник за несколько минут.
:::

## Шаг 2: Создайте детские группы

Группы определяют классы, в которые дети регистрируются. Обычно нужна одна группа на каждый возрастной диапазон.

Следуйте руководству [Создание групп](../groups/creating-groups.md), чтобы создать группы, например:

- **Ясли** (возраст 0–2)
- **Дошкольная группа** (возраст 3–5)
- **Начальная школа** (возраст 6–10)

Вы можете настроить названия и возрастные диапазоны в соответствии со структурой вашего служения.

## Шаг 3: Настройте кампусы и богослужения

Регистрация привязана к определённому времени богослужений. Вам нужен хотя бы один кампус и одно богослужение.

Следуйте руководству [Настройка посещаемости](../attendance/setup.md):

1. Добавьте свой кампус (например, «Основной кампус»)
2. Добавьте богослужение (например, «Воскресное утреннее богослужение»)
3. Установите время богослужения (например, «9:00»)
4. Назначьте детские группы на богослужение

## Шаг 4: Настройте приложение для регистрации

Теперь свяжите всё, установив приложение для регистрации на планшет.

1. Установите приложение **B1 Checkin** — ссылки на скачивание см. в статье [Регистрация](../attendance/check-in.md)
2. Войдите с учётными данными B1 Admin
3. Выберите свой кампус и время богослужения

См. полную статью [Регистрация](../attendance/check-in.md) для подробных шагов настройки.

## Шаг 5: Подготовьте оборудование

Вам понадобится планшет для киоска и, при желании, принтер этикеток Brother для именных бейджей.

Минимально:
- **Один планшет Android или Amazon Fire** — см. [рекомендуемые планшеты](../attendance/check-in.md#recommended-hardware)
- **Один принтер этикеток Brother** — рекомендуется QL-1110NWB за поддержку Bluetooth и WiFi
- **Этикетки Brother DK-1201** (1-1/7" x 3-1/2")

:::warning
С приложением B1 Checkin совместимы только принтеры этикеток Brother. Принтеры других марок работать не будут.
:::

## Шаг 6: Проведите тестовую регистрацию

Перед воскресным утром проведите пробный запуск:

1. Откройте приложение B1 Checkin на планшете
2. Выберите свой кампус и правильное время богослужения
3. Найдите одну из добавленных вами семей
4. Зарегистрируйте ребёнка и проверьте:
   - Посещаемость отображается в B1 Admin в разделе **Посещаемость**
   - Если используется принтер, именной бейдж печатается правильно

:::tip
Обучите волонтёров команды приветствия процессу регистрации перед запуском. Обычно достаточно быстрого 5-минутного инструктажа.
:::

## Готово!

Регистрация для детского служения готова. Родители могут найти свою семью, выбрать детей и зарегистрировать их на киоске. Посещаемость автоматически записывается в B1 Admin.
