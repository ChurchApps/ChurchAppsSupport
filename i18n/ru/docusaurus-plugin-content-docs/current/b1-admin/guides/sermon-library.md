# Создание библиотеки проповедей и прямых трансляций

<div class="article-intro">

Разместите архив проповедей онлайн и настройте прямые трансляции — от импорта существующей библиотеки YouTube или Vimeo до планирования регулярных богослужений с интерактивным чатом. По завершении ваши прихожане смогут смотреть прошлые проповеди и присоединяться к прямым трансляциям на вашем сайте и в мобильном приложении.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Учётная запись B1 Admin с правами управления контентом
- Канал YouTube или Vimeo с видеозаписями проповедей (для импорта архива)
- Идентификатор канала YouTube (для прямых трансляций — найдите его в настройках YouTube Studio)

</div>

## Шаг 1: Создайте плейлисты проповедей

Плейлисты организуют ваши проповеди в серии и коллекции для удобного просмотра.

Следуйте руководству [Плейлисты](../sermons/playlists.md), чтобы:

1. Создать серии/коллекции, например «Воскресные послания 2024», «Серия о браке», «Рождественские богослужения»
2. Добавить название, описание, дату публикации и миниатюру для каждого плейлиста

:::tip
Создайте плейлисты перед добавлением проповедей. Так будет проще организовать проповеди в нужные серии по мере их добавления.
:::

## Шаг 2: Импортируйте существующие проповеди

Если у вас уже есть видеозаписи проповедей на YouTube или Vimeo, вы можете импортировать их все сразу.

Следуйте руководству [Массовый импорт](../sermons/bulk-import.md), чтобы:

1. Ввести URL вашего канала YouTube или Vimeo
2. Просмотреть предварительный результат импорта, чтобы увидеть, какие видео будут загружены
3. Нажать «Импорт», чтобы добавить их все сразу

:::tip
Включите автоимпорт, и новые видео, загруженные на YouTube/Vimeo, будут автоматически появляться в B1 — без ручной работы.
:::

## Шаг 3: Добавьте отдельные проповеди

Для единичных добавлений или проповедей, не входящих в массовый импорт, добавляйте их по одной.

Следуйте руководству [Управление проповедями](../sermons/managing-sermons.md), чтобы:

1. Вставить URL видео (YouTube, Vimeo или Facebook)
2. Название, описание, миниатюра и продолжительность заполнятся автоматически из метаданных видео
3. Назначить проповедь в плейлист

## Шаг 4: Настройте прямую трансляцию

Подключите ваш канал YouTube, чтобы ваш сайт B1 автоматически подхватывал прямую трансляцию.

Следуйте руководству [Прямые трансляции](../sermons/live-streaming.md), чтобы:

1. Ввести идентификатор канала YouTube в настройках трансляции
2. Настроить расписание регулярных богослужений (например, по воскресеньям в 10:00)
3. Настроить вкладки страницы трансляции: Чат, Молитвенные просьбы и пользовательские ссылки

:::info
Когда вы начинаете трансляцию на YouTube, страница трансляции B1 автоматически подхватывает поток. Никаких дополнительных действий в воскресное утро не требуется.
:::

## Шаг 5: Отображение на вашем сайте

Сделайте библиотеку проповедей и прямую трансляцию легкодоступными для посетителей.

Следуйте руководству [Управление страницами](../website/managing-pages.md), чтобы:

1. Добавить раздел «Проповеди» на ваш сайт B1, чтобы посетители могли просматривать библиотеку
2. Добавить раздел «Прямая трансляция» для просмотра в реальном времени

## Шаг 6: Мобильный доступ

Прихожане могут получить доступ к проповедям и прямым трансляциям с любого устройства.

1. Прихожане могут смотреть проповеди в [приложении B1 Mobile](../../b1-mobile/content/sermons.md) и на [B1.church](../../b1-church/content/sermons.md)
2. Прямая трансляция с чатом и молитвенными просьбами работает на обеих платформах

:::info
Функции чата и молитвенных просьб в прямой трансляции позволяют вашим онлайн-прихожанам участвовать в реальном времени — как если бы они были в зале.
:::

## Готово!

Ваша библиотека проповедей размещена онлайн, а прямая трансляция настроена. Прихожане могут просматривать прошлые проповеди в любое время и присоединяться к прямым трансляциям откуда угодно. Новые загруженные проповеди могут синхронизироваться автоматически.
