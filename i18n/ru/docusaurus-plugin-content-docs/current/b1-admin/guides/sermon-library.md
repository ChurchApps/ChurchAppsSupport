# Создание библиотеки проповедей и трансляции

Получите свой архив проповедей в Интернет и настройте прямую трансляцию с интерактивным чатом.

## Шаг 1: Создание плейлистов проповедей

Плейлисты организуют проповеди в серии для легкого просмотра.

Следуйте руководству [Плейлисты](../sermons/playlists.md):

1. Создайте серии/коллекции
2. Добавьте название, описание, дату и миниатюру

## Шаг 2: Импорт существующих проповедей

Если у вас уже есть проповеди на YouTube или Vimeo, вы можете импортировать их все сразу.

Следуйте руководству [Массовый импорт](../sermons/bulk-import.md).

## Шаг 3: Добавление отдельных проповедей

Для добавления отдельных проповедей.

Следуйте руководству [Управление проповедями](../sermons/managing-sermons.md).

## Шаг 4: Настройка прямой трансляции

Подключите ваш канал YouTube.

Следуйте руководству [Прямая трансляция](../sermons/live-streaming.md).

## Шаг 5: Отображение на вашем сайте

Сделайте библиотеку легко доступной.

Следуйте руководству [Управление страницами](../website/managing-pages.md).

## Шаг 6: Доступ с мобильного устройства

Члены могут получать доступ с любого устройства через приложение B1 Mobile или сайт.

## Готово!

Ваша библиотека проповедей в Интернете и прямая трансляция настроены.
