# Запустите сайт вашей церкви

<div class="article-intro">

B1.church включает полноценный конструктор сайтов без дополнительной платы. Это руководство проведёт вас через создание сайта вашей церкви с нуля — настройку главной страницы, оформление внешнего вида, добавление ключевых страниц и, при желании, подключение онлайн-пожертвований и форм регистрации на мероприятия.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Подготовьте логотип вашей церкви (лучше всего подойдёт PNG с прозрачным фоном)
- Выберите 2–3 фирменных цвета для вашего сайта
- Если используете собственный домен (например, вашацерковь.com), обеспечьте доступ к вашему DNS-провайдеру (GoDaddy, Cloudflare и т.д.)
- Если вы хотите онлайн-пожертвования на сайте, сначала завершите [Настройку онлайн-пожертвований](../donations/online-giving-setup.md) (Stripe)

</div>

## Шаг 1: Начальная настройка сайта

Начните с создания главной страницы и базовой структуры сайта.

Следуйте руководству [Начальная настройка сайта](../website/initial-setup.md):

1. Перейдите в раздел **Сайт** в B1 Admin
2. Создайте главную страницу с героическим блоком, приветственным сообщением и ключевой информацией
3. Добавьте название и слоган вашей церкви

## Шаг 2: Настройте внешний вид

Задайте визуальную идентичность вашего сайта — цвета, шрифты, логотип и подвал.

Следуйте руководству [Внешний вид](../website/appearance.md):

1. Загрузите логотип вашей церкви
2. Установите основной и акцентный цвета
3. Настройте панель навигации и подвал
4. Просмотрите предварительный вид изменений

:::tip
Держите цветовую палитру простой — основного цвета плюс одного акцентного обычно достаточно. Конструктор сайтов позаботится об остальном.
:::

## Шаг 3: Добавьте страницы с контентом

Создайте страницы, которые больше всего нужны вашим посетителям.

Следуйте руководству [Управление страницами](../website/managing-pages.md), чтобы создать страницы, такие как:

- **О нас** — История вашей церкви, вероучение и руководство
- **Проповеди** — Ссылка на вашу [библиотеку проповедей](../sermons/managing-sermons.md)
- **Мероприятия** — Предстоящие мероприятия и регистрация
- **Пожертвовать** — Страница онлайн-пожертвований (требуется [настройка Stripe](../donations/online-giving-setup.md))
- **Контакты** — Расположение, время богослужений и контактная информация

## Шаг 4: Подключите ваш домен

Если вы хотите использовать собственное доменное имя (например, вашацерковь.com) вместо стандартного URL B1:

1. Перейдите в **Сайт > Настройки** в B1 Admin
2. Введите ваш пользовательский домен
3. Обновите DNS-записи у вашего регистратора домена, чтобы они указывали на B1

:::info
Распространение изменений DNS может занять до 48 часов. Ваш сайт может быть недоступен по пользовательскому домену сразу. Стандартный URL B1 будет продолжать работать в это время.
:::

## Шаг 5: Добавьте пожертвования и формы

Дополните ваш сайт интерактивными элементами:

- **Онлайн-пожертвования** — Добавьте раздел пожертвований, чтобы члены могли жертвовать прямо с вашего сайта. См. [Настройка онлайн-пожертвований](../donations/online-giving-setup.md) для предварительной настройки Stripe.
- **Формы регистрации** — Встройте [автономные формы](../forms/creating-forms.md) для записи на мероприятия, карточек посетителей или заявок волонтёров. См. [Управление страницами](../website/managing-pages.md) о том, как добавить элемент формы на любую страницу.

## Готово!

Сайт вашей церкви запущен. Поделитесь URL с вашей общиной и в социальных сетях. Вы можете обновлять контент, добавлять новые страницы и настраивать внешний вид в любое время из панели управления B1 Admin.
