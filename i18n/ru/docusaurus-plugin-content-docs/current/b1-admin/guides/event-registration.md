# Настройка регистрации на мероприятия

<div class="article-intro">

Создайте форму регистрации на мероприятие, собирайте информацию об участниках и необязательные платежи, встройте форму на сайт вашей церкви и управляйте заявками по мере их поступления. В итоге у вас будет общедоступная страница регистрации для любого церковного мероприятия.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Учётная запись B1 Admin с правами администратора
- Для сбора платежей: сначала [необходимо настроить Stripe](../donations/online-giving-setup.md)

</div>

## Шаг 1: Создайте автономную форму

Автономные формы имеют собственный публичный URL, доступный каждому — идеально для регистрации на мероприятия.

Следуйте руководству [Создание форм](../forms/creating-forms.md):

1. Перейдите в раздел «Формы» и нажмите «Добавить форму»
2. Выберите тип «Stand Alone» — это даёт вашей форме собственный публичный URL
3. Назовите её по имени мероприятия (например, «Регистрация на мужской ретрит», «Запись на VBS»)

## Шаг 2: Добавьте вопросы

Создайте поля для сбора информации от участников.

Следуйте руководству [Создание форм](../forms/creating-forms.md#adding-questions), чтобы добавить вопросы:

1. Перейдите на вкладку «Вопросы» и добавьте поля для необходимой информации: имя, электронная почта, телефон, диетические ограничения, размер футболки, контакт для экстренной связи и т.д.
2. Используйте «Множественный выбор» для вариантов, таких как предпочтения в питании или выбор сессий

:::warning
Тип поля «Оплата» требует настройки Stripe. Если вы ещё не настроили онлайн-пожертвования, см. [Настройка онлайн-пожертвований](../donations/online-giving-setup.md) перед добавлением полей оплаты.
:::

## Шаг 3: Настройте параметры формы

Управляйте тем, когда и как ваша форма регистрации доступна.

1. Установите даты доступности, если регистрация должна быть открыта только ограниченное время
2. Скопируйте публичный URL — вы можете поделиться им напрямую
3. Добавьте участников формы с ролями «Администратор» или «Только просмотр» для помощи в управлении заявками

## Шаг 4: Встройте на свой сайт

Сделайте форму регистрации легко доступной, добавив её на сайт вашей церкви.

Следуйте руководству [Управление страницами](../website/managing-pages.md):

1. В редакторе сайта B1 добавьте новый раздел на страницу и выберите элемент «Форма»
2. Выберите вашу форму регистрации из списка

:::tip
Делитесь автономным URL также через электронную почту, социальные сети и церковные бюллетени — чем больше мест, где он виден, тем больше регистраций вы получите.
:::

## Шаг 5: Управляйте заявками

Отслеживайте регистрации по мере их поступления и экспортируйте данные, когда это необходимо.

Следуйте руководству [Управление заявками](../forms/managing-submissions.md):

1. Просматривайте ответы по мере их поступления на вкладке «Заявки»
2. Экспортируйте в CSV для таблиц, учёта участников или передачи координаторам мероприятия

## Готово!

Регистрация на ваше мероприятие активна. Поделитесь ссылкой, встройте её на свой сайт и отслеживайте регистрации через B1 Admin. Когда мероприятие закончится, экспортируйте окончательный список для ваших записей.
