# Настройка регистрации на события

Создайте форму регистрации на событие, собирайте информацию участников и управляйте отправками.

:::info
**Два способа обработки регистрации на события:** это руководство охватывает **регистрацию на основе форм**, которая дает вам полный контроль над полями и сбором платежей. Для более простых событий см. **встроенную регистрацию на события** в календаре — см. [Создание календарей](../calendars/creating-calendars.md#enabling-event-registration).
:::

## Шаг 1: Создание отдельной формы

Отдельные формы имеют свой собственный общедоступный URL.

Следуйте руководству [Создание форм](../forms/creating-forms.md):

1. Откройте Формы и нажмите "Добавить форму"
2. Выберите тип "Отдельная"
3. Назовите по названию события

## Шаг 2: Добавление вопросов

Построите поля для сбора информации от регистрантов.

1. Перейдите на вкладку "Вопросы" и добавьте поля
2. Используйте "Множественный выбор" для вариантов

## Шаг 3: Настройка параметров формы

Контролируйте, когда и как ваша форма доступна.

1. Установите даты доступности
2. Скопируйте общедоступный URL
3. Добавьте членов формы

## Шаг 4: Встроение на ваш сайт

Сделайте форму легко найденной.

Следуйте руководству [Управление страницами](../website/managing-pages.md).

## Шаг 5: Управление отправками

Отслеживайте регистрации и экспортируйте данные.

Следуйте руководству [Управление отправками](../forms/managing-submissions.md).

## Готово!

Ваша регистрация на событие активна.
