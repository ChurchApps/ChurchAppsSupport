# Члены группы

После создания группы вы можете добавлять членов. Со страницы деталей группы вы можете искать людей, добавлять их, назначать лидеров, отправлять сообщения и экспортировать список членов.

## Добавление членов в группу

1. Откройте **Группы** и нажмите на группу для управления
2. Нажмите вкладку **Члены**
3. В поле поиска введите имя человека
4. Нажмите **Добавить** рядом с именем в результатах поиска
5. Человек появляется в списке членов группы

:::tip
Оставьте поле поиска пустым и нажмите **Поиск** для просмотра всех людей в справочнике.
:::

## Назначение лидеров группы

Лидеры группы имеют специальные привилегии:

1. В списке членов найдите человека, которого хотите сделать лидером
2. Нажмите **зеленый ключ** рядом с его именем
3. Человек теперь назначен лидером группы

Для удаления статуса лидера нажмите зеленый ключ снова.

:::info
Любой член группы может просматривать календарь и события, но только лидеры могут добавлять или редактировать события.
:::

## Отправка сообщений членам группы

1. Со страницы деталей группы найдите область сообщений
2. Введите ваше сообщение в текстовое поле
3. Нажмите **Отправить**

Сообщение будет доставлено всем членам группы.

## Отправка электронной почты членам группы

1. Со страницы деталей группы нажмите **иконку электронной почты**
2. Откроется диалог отправки с показом количества членов
3. Выберите **шаблон электронной почты** или напишите с нуля
4. Нажмите **Управление шаблонами** для создания или редактирования
5. Введите **строку темы** с полями слияния: `{{firstName}}`, `{{lastName}}`, `{{displayName}}`, `{{email}}`, `{{churchName}}`
6. Напишите **тело письма** используя HTML редактор
7. Нажмите **Отправить**
8. Итоговая информация показывает успешно отправленные письма

:::tip
Создавайте многоразовые шаблоны электронной почты для регулярных коммуникаций.
:::

## Экспорт данных группы

1. Со страницы деталей группы нажмите **иконку загрузки**
2. CSV файл со списком членов загружается

Это полезно для создания печатных списков или импорта в другие инструменты.

## Удаление членов

Найдите человека в списке членов и нажмите **кнопку удаления**.

:::info
Удаление человека из группы не удаляет его из справочника вашей церкви.
:::
