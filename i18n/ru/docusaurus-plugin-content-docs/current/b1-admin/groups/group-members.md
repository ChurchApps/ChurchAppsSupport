# Участники группы

<div class="article-intro">

После создания группы следующий шаг — добавление участников. На странице деталей группы вы можете искать людей, добавлять их в группу, назначать лидеров, отправлять сообщения и экспортировать список участников. Управление членством в группе необходимо для координации служений, малых групп и команд.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- В B1 Admin должна быть создана хотя бы одна группа. Если вы ещё не создали группу, см. [Создание групп](creating-groups.md).
- Люди, которых вы хотите добавить, должны уже существовать в вашем справочнике [Люди](../people/adding-people.md).

</div>

## Добавление участников в группу

1. Перейдите на страницу **Groups** и нажмите на группу, которой хотите управлять.
2. Нажмите на вкладку **Members**.
3. В поле поиска введите имя человека, которого хотите добавить.
4. Нажмите **Add** рядом с именем человека в результатах поиска.
5. Человек теперь отображается в списке участников группы.

:::tip
Оставьте поле поиска пустым и нажмите **Search**, чтобы просмотреть весь справочник. Это полезно, если вы не уверены в точном написании чьего-то имени.
:::

## Назначение лидеров группы

Лидеры группы имеют особые привилегии — они могут редактировать [календарь группы](group-calendar.md), управлять мероприятиями и помогать координировать группу.

1. В списке участников группы найдите человека, которого хотите сделать лидером.
2. Нажмите на **значок зелёного ключа** рядом с его именем.
3. Человек теперь назначен лидером группы.

Чтобы снять статус лидера, нажмите на значок зелёного ключа снова.

:::info
Любой участник группы может просматривать календарь группы и мероприятия, но только лидеры могут добавлять или редактировать события календаря.
:::

## Отправка сообщений участникам группы

Вы можете общаться со всеми участниками группы прямо из B1 Admin:

1. На странице деталей группы найдите область для сообщений.
2. Введите ваше сообщение в текстовое поле.
3. Нажмите **Send**.

Ваше сообщение будет доставлено всем участникам группы.

## Экспорт данных группы

Чтобы скачать список участников группы в виде файла:

1. На странице деталей группы нажмите **значок загрузки**.
2. CSV-файл с информацией об участниках группы будет загружен на ваш компьютер.

Это полезно для создания печатных списков, импорта данных в другие инструменты или ведения оффлайн-записей. Дополнительные варианты экспорта см. в [Экспорт данных](../people/exporting-data.md).

## Удаление участников

Чтобы удалить кого-то из группы, найдите его имя в списке участников и нажмите кнопку **удаления** рядом с его записью.

:::info
Удаление человека из группы не удаляет его из справочника вашей церкви. Он по-прежнему будет отображаться в разделе [Люди](../people/adding-people.md) и может быть повторно добавлен в группу в любое время.
:::
