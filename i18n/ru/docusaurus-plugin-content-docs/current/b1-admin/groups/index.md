# Группы

<div class="article-intro">

Раздел **Группы** позволяет организовать членов вашей церкви в команды, служения, малые группы, библейские занятия, комитеты и многое другое. Группы являются основой общинной жизни вашей церкви, и этот раздел предоставляет инструменты для создания, управления и общения с каждой группой в одном месте.

</div>

## Обзор

Группы организованы по **категориям**. Категория — это метка верхнего уровня, такая как "Малые группы", "Служения" или "Комитеты". Каждая категория может содержать несколько групп, что упрощает объединение связанных групп.

Когда вы открываете страницу "Группы", вверху отображается сводка:

- **Всего групп** -- количество групп во всех категориях
- **Категории** -- количество созданных вами категорий групп
- **Всего участников** -- общее количество участников во всех группах

Под сводкой ваши группы перечислены по категориям с указанием количества участников для каждой группы.

## Что вы можете сделать

Вот что вы можете сделать в разделе "Группы":

1. **Создавать и организовывать группы** -- Добавляйте новые группы в категории, отражающие структуру вашей церкви. См. [Создание групп](./creating-groups.md).

2. **Управлять участниками групп** -- Добавляйте и удаляйте участников, назначайте лидеров и общайтесь с вашими группами. См. [Участники группы](./group-members.md).

3. **Планировать мероприятия** -- У каждой группы может быть свой календарь для встреч, мероприятий и регулярных активностей. См. [Календарь группы](./group-calendar.md).

4. **Экспортировать данные группы** -- Скачивайте списки участников любой группы в формате CSV для использования в таблицах или рассылках. Инструкции по экспорту см. в [Участники группы](./group-members.md).

## Начало работы

Чтобы получить доступ к группам, нажмите **People** на левой панели навигации B1 Admin, затем выберите **Groups**. Вы увидите все существующие группы, организованные по категориям.

:::tip
Если вы только начинаете, создайте несколько категорий, соответствующих структуре вашей церкви (например, "Служения", "Малые группы", "Команды"). Затем добавьте отдельные группы в каждую категорию. Пошаговое руководство см. в [Создание групп](./creating-groups.md).
:::

:::info
Участники группы должны сначала существовать в вашем справочнике [Люди](../people/adding-people.md), прежде чем их можно будет добавить в группу. Если вы настраиваете B1 впервые, [импортируйте данные о членах](../people/importing-data.md) перед созданием групп.
:::
