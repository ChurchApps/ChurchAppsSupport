# Группы

<div class="article-intro">

Раздел **Группы** позволяет организовать членов церкви в служения, небольшие группы, библейские исследования, комитеты и прочее. Группы - основа жизни сообщества в вашей церкви, и этот раздел дает вам инструменты для создания, управления и коммуникации. Отметим, что **Команды** отделены от групп — команды используются для служения и планирования волонтеров.

</div>

## Обзор

Группы организованы по **категориям**. Категория - это основная метка, такая как "Небольшие группы", "Служения" или "Комитеты". Каждая категория может содержать несколько групп.

Когда вы откроете страницу групп, вы увидите сводку вверху:

- **Всего групп** — количество групп по всем категориям
- **Категории** — количество созданных категорий
- **Всего членов** — объединенное количество членов по всем группам

Ниже группы перечислены по категориям с показом количества членов в каждой.

## Что вы можете делать

Вот что вы можете сделать в разделе "Группы":

1. **Создавать и организовывать группы** — добавляйте новые группы в категории. См. [Создание групп](./creating-groups.md).

2. **Управлять членами группы** — добавляйте и удаляйте членов, назначайте лидеров и общайтесь. См. [Члены группы](./group-members.md).

3. **Планировать события** — каждая группа может иметь свой календарь для встреч и событий. См. [Календарь группы](./group-calendar.md).

4. **Экспортировать данные группы** — загружайте списки членов как CSV файлы. См. [Члены группы](./group-members.md).

## Начало работы

Для доступа к группам нажмите **Люди** в левой боковой панели, затем выберите **Группы**. Вы увидите все существующие группы, организованные по категориям.

:::tip
Если вы только начинаете, создайте несколько категорий, которые соответствуют вашей структуре церкви (например, "Служения", "Небольшие группы", "Комитеты"). Затем добавляйте отдельные группы в каждую категорию. См. [Создание групп](./creating-groups.md).
:::

:::info
Члены группы должны сначала существовать в [Люди](../people/adding-people.md) перед добавлением в группу. Если вы настраиваете B1 впервые, [импортируйте данные членов](../people/importing-data.md) перед созданием групп.
:::
