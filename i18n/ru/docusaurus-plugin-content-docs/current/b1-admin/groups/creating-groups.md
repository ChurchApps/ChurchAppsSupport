# Создание групп

<div class="article-intro">

Создание группы в B1 Admin - это просто. Вы настроите категорию, назовете свою группу, а затем настроите её параметры. Группы помогают организовать вашу церковь в значимые единицы, такие как малые группы, комитеты и классы.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Вам нужна активная учетная запись B1 Admin с правами на управление группами. См. [Роли и права доступа](../people/roles-permissions.md), если вы не уверены в своем уровне доступа.
- Определитесь со структурой категорий для ваших групп (например, "Малые группы", "Служения", "Комитеты"). Категории помогают поддерживать порядок среди связанных групп.

</div>

## Добавление новой группы

1. Перейдите в **панель управления B1 Admin**.
2. Нажмите на вкладку **Группы**.
3. Нажмите **Добавить группу** и введите **Название категории**. Категории помогают организовать связанные группы вместе (например, "Малые группы", "Служения" или "Комитеты"). Если категория уже существует, вы можете выбрать её из списка.
4. Введите **Название группы**.
5. Нажмите **Добавить**. Ваша новая группа появится в списке под выбранной категорией.

## Настройка параметров группы

После создания группы вы можете заполнить дополнительные сведения:

1. Нажмите на **название группы** в списке, чтобы открыть её.
2. Нажмите **значок карандаша**, чтобы изменить настройки группы.
3. Настройте следующие параметры:
   - **Описание** -- Краткое резюме о том, что представляет собой группа. Это видно участникам.
   - **Время встреч** -- Когда группа обычно собирается (например, "Среды в 19:00").
   - **Политика присоединения** -- Выберите, кто может присоединиться к этой группе:
     - **Открытая** -- Любой может присоединиться немедленно без одобрения
     - **По запросу** -- Люди должны подать запрос на присоединение, требующий одобрения (см. [Запросы на присоединение к группе](./group-join-requests.md))
     - **Закрытая** -- Участники должны быть добавлены вручную лидерами или администраторами
   - **Отслеживание посещаемости** -- Включите это, если хотите записывать [посещаемость](../attendance/tracking-attendance.md) для этой группы.
   - **Время служений** -- Свяжите группу с конкретным временем церковных служб, если применимо. См. [Настройка посещаемости](../attendance/setup.md) для получения подробной информации о времени служений.
4. Нажмите **Сохранить**, чтобы применить изменения.

:::tip
Добавление четкого описания и времени встреч помогает участникам знать, чего ожидать, когда они присоединяются к группе.
:::

## Следующие шаги

После создания и настройки группы вы готовы:

- **Добавить участников** -- Найдите людей и добавьте их в группу. Используйте зеленый значок ключа, чтобы назначить лидеров группы. См. [Участники группы](./group-members.md).
- **Настроить календарь** -- Создавайте события и регулярные встречи для группы. См. [Календарь группы](./group-calendar.md).
- **Общаться** -- Отправляйте сообщения всем участникам группы прямо со страницы группы.
- **Экспортировать данные** -- Нажмите значок загрузки, чтобы экспортировать список участников вашей группы.

:::info
Все группы вашей церкви организованы по категориям на главной странице Группы. Вы всегда можете переупорядочивать или переименовывать категории по мере роста вашей церкви.
:::
