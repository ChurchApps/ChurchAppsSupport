# Создание групп

<div class="article-intro">

Создание группы в B1 Admin — это просто. Вы настроите категорию, назовёте свою группу и затем сконфигурируете её параметры. Группы помогают организовать вашу церковь в значимые единицы, такие как малые группы, команды служения, комитеты и классы.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Вам нужна активная учётная запись B1 Admin с разрешением на управление группами. См. [Роли и разрешения](../people/roles-permissions.md), если вы не уверены в своём уровне доступа.
- Определите структуру категорий для ваших групп (например, "Малые группы", "Служения", "Комитеты"). Категории помогают организовать связанные группы.

</div>

## Добавление новой группы

1. Перейдите на **панель управления B1 Admin**.
2. Нажмите на вкладку **Groups**.
3. Нажмите **Add Group** и введите **название категории**. Категории помогают организовать связанные группы (например, "Малые группы", "Служения" или "Комитеты"). Если категория уже существует, вы можете выбрать её из списка.
4. Введите **название группы**.
5. Нажмите **Add**. Ваша новая группа появится в списке под выбранной категорией.

## Настройка параметров группы

После создания группы вы можете заполнить дополнительные данные:

1. Нажмите на **название группы** в списке, чтобы открыть её.
2. Нажмите на **значок карандаша**, чтобы редактировать параметры группы.
3. Настройте следующие параметры:
   - **Описание** -- Краткое описание группы. Оно видно участникам.
   - **Время встреч** -- Когда группа обычно встречается (например, "Среды в 19:00").
   - **Учёт посещаемости** -- Включите, если хотите вести [учёт посещаемости](../attendance/tracking-attendance.md) для этой группы.
   - **Время служб** -- Привяжите группу к определённым временам церковных служб, если применимо. Подробнее о временах служб см. [Настройка посещаемости](../attendance/setup.md).
4. Нажмите **Save**, чтобы применить изменения.

:::tip
Добавление понятного описания и времени встреч помогает участникам знать, чего ожидать при вступлении в группу.
:::

## Следующие шаги

После создания и настройки группы вы готовы:

- **Добавить участников** -- Найдите людей и добавьте их в группу. Используйте значок зелёного ключа для назначения лидеров группы. См. [Участники группы](./group-members.md).
- **Настроить календарь** -- Создайте мероприятия и регулярные встречи для группы. См. [Календарь группы](./group-calendar.md).
- **Общаться** -- Отправляйте сообщения всем участникам группы прямо со страницы группы.
- **Экспортировать данные** -- Нажмите значок загрузки, чтобы экспортировать список участников группы.

:::info
Все группы вашей церкви организованы по категориям на главной странице "Группы". Вы всегда можете переименовать или реорганизовать категории по мере роста вашей церкви.
:::
