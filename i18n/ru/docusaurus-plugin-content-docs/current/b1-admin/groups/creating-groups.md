# Создание групп

Создание группы в B1 Admin просто. Вы установите категорию, назовете группу и настроите параметры. Группы помогают организовать членов церкви на значимые единицы.

## Добавление новой группы

1. Откройте **B1 Admin**
2. Нажмите на вкладку **Группы**
3. Нажмите **Добавить группу** и введите **название категории**
4. Категории помогают организовать связанные группы (например, "Небольшие группы", "Служения", "Комитеты")
5. Введите **название группы**
6. Нажмите **Добавить**

## Настройка параметров группы

1. Нажмите на **название группы** в списке
2. Нажмите **иконку карандаша** для редактирования
3. Настройте:
   - **Описание** — краткое резюме о группе
   - **Время встреч** — когда группа встречается (например, "Среда в 19:00")
   - **Отслеживание посещаемости** — включить, если хотите отслеживать посещаемость
   - **Время служений** — связите с определенным временем служения
4. Нажмите **Сохранить**

:::tip
Добавьте четкое описание и время встреч, чтобы члены знали, чего ожидать.
:::

## Следующие шаги

После создания и настройки группы вы можете:

- **Добавить членов** — см. [Члены группы](./group-members.md)
- **Установить календарь** — создайте события и встречи
- **Общаться** — отправляйте сообщения членам группы
- **Экспортировать данные** — загрузите список членов

:::info
Все группы организованы по категориям на главной странице групп.
:::
