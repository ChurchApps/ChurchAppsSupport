# Здоровье групп

<div class="article-intro">

Приборная панель "Здоровье групп" дает вам обзор того, как работают все ваши группы — тенденции членства, средняя посещаемость и рост или отток за последние 90 дней — все в одной таблице для сортировки.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Вам нужны хотя бы несколько групп с членами, чтобы увидеть значимые данные. См. раздел [Создание групп](creating-groups).
- Данные посещаемости извлекаются из записанных сеансов. See the [Регистрация](../attendance/index) раздел.

</div>

## Открытие здоровья групп

В B1 Admin перейдите на **Группы** на левой боковой панели и выберите **Здоровье**. Приборная панель загружает таблицу с одной строкой на группу.

## Столбцы

| Столбец | Что он показывает |
|--------|--------------|
| **Имя** | Имя группы, связанное со страницей деталей группы |
| **Категория** | Категория группы |
| **Члены** | Текущее количество активных членов |
| **Присоединено (90 дней)** | Члены, присоединившиеся за последние 90 дней |
| **Ушли (90 дней)** | Члены, ушедшие за последние 90 дней |
| **Отток (90 дней)** | Чистый процент отттока за 90 дней |
| **Средняя посещаемость** | Среднее количество людей на сеанс посещаемости |

Нажмите на любой заголовок столбца, чтобы отсортировать таблицу по этому столбцу. Нажмите снова, чтобы развернуть направление сортировки.

## Использование данных здоровья

- **Высокий отток + низкие присоединения** — группа, которая сокращается и не заменяет потерянные члены. Стоит разговор с лидером группы.
- **Высокие присоединения + низкая посещаемость** — люди регистрируются, но не показываются. Рассмотрите дальнейшее привлечение.
- **Высокая средняя посещаемость** — здоровая, активная группа. Потенциальная модель для других групп.

:::tip
Нажатие на имя группы берет вас прямо на страницу деталей группы, где вы можете просмотреть отдельных членов, записи посещаемости и события календаря.
:::
