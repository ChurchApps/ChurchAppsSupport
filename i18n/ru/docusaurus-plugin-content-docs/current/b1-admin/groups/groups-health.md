# Здоровье групп

<div class="article-intro">

Панель управления Health Groups дает вам полный обзор того, как работают все ваши группы — тенденции членства, средняя посещаемость и рост или сокращение за последние 90 дней — все в одной сортируемой таблице.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Вам нужна по крайней мере несколько групп с членами, чтобы увидеть значимые данные. См. [Создание групп](creating-groups).
- Данные о посещаемости берутся из записанных сеансов. См. раздел [Посещаемость](../attendance/).

</div>

## Открытие Groups Health

В B1 Admin перейдите к **Группы** на левой боковой панели и выберите **Здоровье**. Панель управления загружает таблицу с одной строкой на группу.

## Столбцы

| Столбец | Что он показывает |
|--------|--------------|
| **Имя** | Имя группы, связанное со страницей деталей группы |
| **Категория** | Категория группы |
| **Члены** | Текущее количество активных членов |
| **Присоединились (90д)** | Члены, присоединившиеся в последние 90 дней |
| **Ушли (90д)** | Члены, ушедшие в последние 90 дней |
| **Отток (90д)** | Чистый уровень оттока в процентах за 90 дней |
| **Средняя посещаемость** | Среднее количество людей на сеанс посещаемости |

Нажмите на любой заголовок столбца, чтобы отсортировать таблицу по этому столбцу. Нажмите еще раз, чтобы изменить направление сортировки.

## Использование данных о здоровье

- **Высокий отток + низкое присоединение** — группа, которая сокращается и не пополняется ушедших членов. Стоит поговорить с лидером группы.
- **Высокое присоединение + низкая посещаемость** — люди регистрируются, но не появляются. Рассмотрите дальнейшее привлечение.
- **Высокая средняя посещаемость** — здоровая, активная группа. Потенциальная модель для других групп.

:::tip
Нажав на имя группы, вы перейдете непосредственно на страницу деталей группы, где вы можете просмотреть отдельных членов, записи о посещаемости и события календаря.
:::
