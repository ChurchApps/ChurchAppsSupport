# Календарь группы

<div class="article-intro">

Каждая группа в B1 может иметь свой собственный календарь событий. Лидеры групп используют его для планирования встреч, мероприятий и повторяющихся событий, в то время как все участники группы могут просматривать предстоящие события и участвовать в беседах о них. Календарь держит вашу группу организованной и гарантирует, что все в курсе того, что предстоит.

</div>

<div class="prereqs">
<h4>Перед тем как начать</h4>

- Вам нужна группа, созданная в B1 Admin, с по крайней мере одним участником. См. [Создание групп](creating-groups.md) и [Участники группы](group-members.md).
- Чтобы добавлять или редактировать события календаря, вы должны быть назначены лидером группы. См. [Назначение лидеров групп](group-members.md) для получения инструкций.

</div>

## Доступ к календарю группы

Календарь группы доступен через приложение B1.church для участников:

1. Из B1 Admin нажмите на **раскрывающееся меню** в верхнем правом углу экрана.
2. Выберите **Переключить приложение**.
3. Выберите **B1.church**.
4. Нажмите **Группы** в левой боковой панели.
5. Выберите вашу группу из списка.
6. Нажмите **Календарь** с левой стороны экрана.

:::info
Календарь группы управляется через B1.church (приложение для участников), а не через B1 Admin. Лидеры групп получают к нему доступ, переключаясь между приложениями с панели управления администратора.
:::

## Добавление события

1. Из календаря группы нажмите **Добавить событие**.
2. Введите **название** для события.
3. Напишите краткое **описание** того, о чем идет речь в событии.
4. Установите **время начала** и **время окончания**.
5. Нажмите **Сохранить**, чтобы добавить событие в календарь.

## Установка повторяющихся событий

Если ваша группа встречается по регулярному расписанию, вы можете установить автоматическое повторение событий:

1. При создании или редактировании события отметьте флажок **Повторяющееся**.
2. Выберите, как должно повторяться событие:
   - **Интервал** -- установите количество дней между вхождениями. Например, введение "7" повторяет событие каждые 7 дней.
   - **Частота** -- выберите повтор по **дню**, **неделе** или **месяцу**.
3. Сохраните событие. Теперь оно будет отображаться на календаре при каждом запланированном вхождении.

:::tip
Для группы, которая встречается каждую субботу, установите частоту на "неделю" и выберите субботу. Событие будет автоматически заполняться на календаре в дальнейшем.
:::

## Групповое создание событий в B1 Admin

На странице группы в B1 Admin также есть вкладка **Календарь**, где персонал может запланировать целый сезон встреч одновременно:

1. Откройте группу в B1 Admin и выберите вкладку **Календарь**.
2. Нажмите **Добавить события**, чтобы открыть диалоговое окно **Добавить повторяющиеся события**.
3. Выберите расписание (еженедельное, каждые 2 недели, ежемесячное или один раз), даты **Первой встречи** и **Последней встречи** и опционально **Пропустить праздники**.
4. Просмотрите предпросмотр дат встреч и снимите флажки, чтобы пропустить.
5. Оставьте **Разрешить ответы** отмеченным, если вы хотите, чтобы участники могли отвечать на эти события (см. ниже), затем сохраните.

## Ответы на события

События группы могут собирать ответы — "Да", "Нет" или "Может быть" для каждого вхождения события.

- **Разрешить ответы** включено по умолчанию для событий, созданных через диалоговое окно группового добавления; снимите флажок, чтобы отключить ответы для этих событий. События с отключенными ответами показывают как **Отключено** на вкладке календаря.
- На вкладке календаря группы в B1 Admin каждое событие показывает резюме ответов (например, "5 да · 2 может быть · 1 нет"). Нажмите, чтобы открыть **Ответы на вопросы**, который показывает список того, кто ответил да, может быть, и нет для этого вхождения.

:::tip
Ответы дают лидерам подсчет без формальности полной регистрации на событие. Для платных регистраций или событий, управляемых по вместимости, используйте вместо этого [регистрацию на событие](../calendars/creating-calendars.md#enabling-event-registration).
:::

## Просмотр календаря

Календарь группы предлагает несколько вариантов просмотра:

- **Месяц** -- посмотрите весь месяц с одного взгляда.
- **Неделя** -- сосредоточьтесь на текущей неделе.
- **День** -- посмотрите один день подробно.
- **Список** -- посмотрите список предстоящих событий в хронологическом порядке.

Переключайтесь между представлениями, используя кнопки в верхней части календаря.

## Детали события и беседы

Нажмите на любое событие, чтобы просмотреть его детали. Из представления деталей события:

- **Участники и лидеры** могут начинать беседы и оставлять комментарии о событии.
- **Лидеры** могут редактировать детали события, изменять время или удалять событие.

:::warning
Удаление события удаляет его окончательно. Если вам нужно отменить одно вхождение повторяющегося события, рассмотрите возможность его редактирования вместо этого.
:::

## Подписка на календарь

Вы можете синхронизировать календарь группы с вашим личным календарем:

1. На странице календаря группы нажмите **Подписаться**.
2. Следуйте подсказкам, чтобы добавить календарь в Google Calendar, Apple Calendar, Outlook или любое другое приложение календаря, поддерживающее подписку.

:::tip
Подписка держит ваше личное расписание в синхронизации с событиями вашей группы без необходимости проверять B1 отдельно. Любые изменения, внесенные лидерами групп, автоматически появятся в вашем синхронизированном календаре.
:::
