# Запись посещаемости

<div class="article-intro">

После настройки кампусов, времени служений и групп вы можете вручную записывать посещаемость после каждого собрания. B1 Admin организует посещаемость вокруг **сессий** -- одна сессия на группу на дату встречи. Вы создаёте сессию, отмечаете, кто присутствовал, и данные поступают непосредственно в ваши отчёты о посещаемости.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Ваши кампусы, время служений и группы должны быть настроены. Если вы ещё не сделали этого, см. [Настройка посещаемости](setup.md).
- Для групп, которые вы хотите отслеживать, должно быть включено **Отслеживание посещаемости**. Подробнее см. [Настройка посещаемости](setup.md).

</div>

## Создание сессии

Сессия представляет одно собрание группы -- например, ваш класс для детей от 5 до 9 лет в определённое воскресенье.

1. Откройте **B1 Admin** и нажмите **Группы** в боковой панели.
2. Выберите группу, для которой хотите записать посещаемость.
3. Нажмите на вкладку **Сессии**.
4. Нажмите **Создать**, чтобы создать новую сессию.
5. Если группа привязана к времени служения, выберите **Время служения**. Если это незапланированная группа, это поле не появится.
6. Выберите **Дату сессии** -- это может быть сегодняшний день, прошедшая или будущая дата.
7. Нажмите **Сохранить**.

:::tip
Вы можете создавать сессии для прошлых дат, чтобы внести посещаемость, которую ещё не записали, или создавать их заранее, чтобы они были готовы к моменту встречи группы.
:::

## Отметка посещаемости

После сохранения сессии в правой части страницы появляются члены группы.

1. Поставьте галочку напротив каждого человека, который присутствовал.
2. Изменения сохраняются автоматически -- дополнительной кнопки сохранения для отметок посещаемости нет.

:::info
В списке сессии отображаются только текущие члены группы. Если кто-то присутствовал, но ещё не состоит в группе, сначала [добавьте его в группу](../groups/group-members.md), а затем отметьте посещаемость.
:::

## Экспорт посещаемости в таблицу

Вы можете скачать запись сессии в виде CSV-файла для использования в Excel, Numbers или Google Sheets.

1. Откройте сессию, которую хотите экспортировать.
2. Нажмите кнопку **Скачать CSV**.
3. Откройте скачанный файл в вашем табличном приложении.

## Просмотр записанной посещаемости

После записи сессий данные появляются в ваших отчётах о посещаемости.

- **Вкладка «Посещаемость»** -- показывает общецерковные тенденции во времени. См. [Отслеживание посещаемости](tracking-attendance.md).
- **Вкладка «Группы»** -- показывает посещаемость в разбивке по отдельным группам.

:::tip
Если только что созданная сессия не отображается в отчётах, убедитесь, что дата сессии попадает в диапазон дат, выбранный в фильтрах отчёта.
:::
