# Запись посещаемости

<div class="article-intro">

После настройки вы можете вручную записывать посещаемость. B1 Admin организует это вокруг **сеансов**.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Структура должна быть установлена
- **Отслеживание посещаемости** должно быть включено

</div>

## Создание сеанса

1. Откройте **B1 Admin** и нажмите **Группы**
2. Выберите группу
3. Нажмите вкладку **Сеансы**
4. Нажмите **Новый**
5. Выберите **Дату сеанса**
6. Нажмите **Сохранить**

## Отметка посещаемости

1. Установите флажок рядом с каждым присутствующим
2. Изменения сохраняются автоматически

## Экспорт

1. Откройте сеанс
2. Нажмите **Загрузить CSV**

## Просмотр

Данные отображаются в отчетах.

:::tip
Убедитесь, что дата попадает в диапазон фильтров.
:::
