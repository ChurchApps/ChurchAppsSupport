# Конструктор этикеток для регистрации

<div class="article-intro">

Конструктор этикеток позволяет создавать и настраивать шаблоны бейджиков и квитанций для выдачи, которые печатаются при регистрации детей в семьях. Вы можете точно контролировать, какая информация появляется на каждой этикетке, где она расположена и как она выглядит.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Настройте [Посещаемость](setup) и настройте по крайней мере одно время службы с включенной регистрацией
- Настройте [Регистрацию](check-in) чтобы этикетки печатались
- Вам нужен административный доступ к разделу Посещаемость

</div>

## Открытие конструктора этикеток

В B1 Admin перейдите к **Посещаемость** на левой боковой панели и выберите **Этикетки**. Вы увидите список сохраненных шаблонов этикеток, отделенные по типам: **Бейджик** и **Квитанция для выдачи**.

## Типы этикеток

- **Бейджик** — печатается и прикрепляется к ребенку. Обычно включает имя ребенка, его класс/сеанс и код безопасности.
- **Квитанция для выдачи** — отдается родителю или опекуну. Обычно включает код безопасности и список детей, которых они зарегистрировали.

B1 предоставляет вам шаблон бейджика по умолчанию и шаблон квитанции по умолчанию, рассчитанные на стандартные термальные этикетки размером 3,5 × 1,1 дюйма.

## Создание шаблона этикетки

1. Нажмите **Добавить бейджик** или **Добавить квитанцию для выдачи** (или используйте раскрывающееся меню для выбора).
2. Новый шаблон откроется в редакторе этикеток.

### Редактор этикеток

Редактор показывает масштабный предпросмотр этикетки в настроенном размере. На левой панели вы можете настроить:

- **Имя** — имя шаблона (только для вашей справки)
- **Тип этикетки** — бейджик или квитанция для выдачи
- **Ширина / Высота** — размер этикетки в дюймах

### Добавление блоков

Этикетка состоит из блоков — отдельных частей содержимого, расположенных на холсте этикетки. Нажмите **Добавить блок** для вставления нового блока и выберите его тип:

- **Поле** — вытягивает значение данных во время печати:
  - `person.displayName` — полное имя человека
  - `sessions` — служба/класс, на которые они зарегистрировались
  - `securityCode` — случайно сгенерированный код безопасности для выдачи
  - `children` — список детей (для квитанций для выдачи)
  - `person.nametagNotes` — все специальные примечания в записи человека
  - `campus` — имя кампуса
- **Текст** — статический текст, который вы вводите (для заголовков, меток или инструкций)
- **Штрихкод** — штрихкод, кодирующий код безопасности

### Позиционирование блоков

Каждый блок имеет поля **X**, **Y**, **Ширина** и **Высота**, выраженные в процентах от холста этикетки (0–100). Отрегулируйте их, чтобы точно позиционировать содержимое. Вы также можете установить:

- **Размер шрифта** — размер текста в пунктах
- **Жирный** — переключатель для жирного текста
- **Выравнивание** — выравнивание текста влево, по центру или вправо
- **Условие** — дополнительно скрыть блок, если поле пусто (например, показать nametagNotes только если оно имеет значение)

### Сохранение

Нажмите **Сохранить** для сохранения шаблона. Обновленный шаблон будет использован в следующий раз при печати этикеток в B1 Checkin.

## Переупорядочение шаблонов

Если у вас есть несколько шаблонов бейджиков или квитанций для выдачи, B1 Checkin будет использовать первый шаблон в списке по умолчанию. Перетащите шаблоны для переупорядочения.

## Удаление шаблона

Нажмите значок удаления на любой строке шаблона и подтвердите. Удаление последнего шаблона типа восстанавливает встроенный шаблон по умолчанию.

:::tip
Сделайте пробный отпечаток после редактирования шаблона, чтобы подтвердить, что макет выглядит правильно перед следующей службой.
:::
