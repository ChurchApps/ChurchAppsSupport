# Отслеживание посещаемости

<div class="article-intro">

Когда ваши кампусы, расписание служб и группы настроены, B1 Admin позволяет легко просматривать данные о посещаемости и выявлять тенденции. Страница посещаемости предоставляет два представления отчётов — вкладку **Посещаемость** для общецерковных тенденций и вкладку **Группы** для детализации по группам. Используйте эти инструменты для понимания закономерностей роста, выявления снижения вовлечённости и принятия решений на основе данных для вашей церкви.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Структура посещаемости должна быть настроена с хотя бы одним кампусом и временем служения. Если вы ещё не сделали этого, см. раздел [Настройка посещаемости](setup.md).
- Данные о посещаемости должны быть записаны до того, как отчёты покажут результаты. Данные могут поступать из ручного ввода или [самостоятельной регистрации](check-in.md).

</div>

## Просмотр тенденций посещаемости

1. Откройте **B1 Admin** и нажмите **Attendance** на боковой панели.
2. Выберите вкладку **Attendance**.
3. Отчёт запускается автоматически при открытии вкладки, показывая посещаемость за период по умолчанию.

## Фильтрация данных

Используйте фильтры в верхней части страницы для сужения результатов:

- **Диапазон дат** — выберите начальную и конечную даты для фокусировки на определённом периоде.
- **Кампус** — выберите кампус, чтобы видеть посещаемость только этой локации.
- **Время служения** — выберите время служения для детального анализа конкретного собрания.

Диаграмма и данные обновляются сразу при изменении фильтра, поэтому вы можете быстро сравнивать различные периоды времени или локации.

:::info
Отчёты запускаются автоматически каждый раз при открытии вкладки «Посещаемость», поэтому вы всегда будете видеть актуальные данные без необходимости нажимать кнопку обновления.
:::

## Посещаемость по группам

Вкладка **Группы** показывает посещаемость в разрезе отдельных групп. Это полезно, когда вы хотите отслеживать конкретный класс, команду служения или малую группу, а не общие показатели служений.

1. Выберите вкладку **Группы**.
2. Выберите группу из списка, чтобы увидеть историю её посещаемости.
3. Используйте фильтр диапазона дат для настройки периода отчётности.

:::tip
Посещаемость по группам особенно ценна для лидеров [малых групп](../groups/creating-groups.md), которые хотят отслеживать вовлечённость в своей группе с течением времени.
:::

## Советы по использованию данных о посещаемости

- Ежемесячно просматривайте тенденции, чтобы заблаговременно выявлять сезонные закономерности.
- Сравнивайте данные на уровне кампусов, чтобы понять, какие локации растут.
- Используйте отчёты по группам для работы с [группами](../groups/group-members.md), показывающими снижение посещаемости.
- Сочетайте данные о посещаемости с инструментом [Поиск с ИИ](../people/ai-search.md), чтобы найти людей, которые давно не посещали церковь.
