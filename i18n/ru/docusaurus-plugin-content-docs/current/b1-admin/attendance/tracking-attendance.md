# Отслеживание посещаемости

<div class="article-intro">

После настройки кампусов и групп B1 Admin облегчает анализ данных и выявление тенденций.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Структура посещаемости должна быть установлена
- Данные о посещаемости должны быть записаны

</div>

## Просмотр тенденций

1. Откройте **B1 Admin** и нажмите **Посещаемость**
2. Выберите вкладку **Посещаемость**

## Фильтрация данных

- **Диапазон дат**
- **Кампус**
- **Время служения**

## Посещаемость группы

Вкладка **Группы** показывает данные по группам.

## Советы

- Анализируйте ежемесячно
- Сравнивайте кампусы
- Используйте AI Search
