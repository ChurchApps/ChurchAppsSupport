# Посещаемость

<div class="article-intro">

Раздел "Посещаемость" в B1 Admin предоставляет полный обзор того, кто посещает вашу церковь и как ваши группы растут с течением времени. От конфигурации кампусов и времени служений до анализа тенденций и настройки самостоятельной регистрации - все необходимое для отслеживания и понимания посещаемости управляется с этой страницы.

</div>

## Обзор страницы

Когда вы откроете страницу Посещаемость, вы увидите заголовок с ключевой статистикой о настройке посещаемости вашей церкви:

- **Кампусы** -- количество физических мест, которые вы настроили
- **Время служений** -- общее количество служений во всех кампусах
- **Запланированные группы** -- группы, назначенные на определенное время служения
- **Незапланированные группы** -- группы, которые отслеживают посещаемость независимо от времени служения

Ниже заголовка находятся три вкладки для организации ваших инструментов отслеживания посещаемости.

## Вкладки

### Настройка

Вкладка **Настройка** предназначена для конфигурации основных компонентов отслеживания посещаемости. Здесь вы определяете кампусы, добавляете время служения для каждого кампуса и назначаете [группы](../groups/creating-groups.md) к этим временам служения. Вам нужно сделать это только один раз, хотя вы можете обновить это в любое время, когда изменится ваше расписание. Полное руководство см. в разделе [Настройка посещаемости](setup.md).

### Посещаемость

Вкладка **Посещаемость** отображает тенденции посещаемости с течением времени. Используйте фильтры, чтобы сузить результаты по диапазону дат, кампусу или времени служения и быстро определить закономерности в ваших данных. Подробную информацию об использовании отчетов и фильтров см. в разделе [Отслеживание посещаемости](tracking-attendance.md).

### Группы

Вкладка **Группы** разбивает посещаемость по отдельным группам. Это полезно, когда вы хотите увидеть, как работает конкретная малая группа, класс или команда служения, а не смотреть на статистику по всей церкви.
