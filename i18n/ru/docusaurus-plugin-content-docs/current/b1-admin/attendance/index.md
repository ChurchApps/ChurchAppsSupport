# Посещаемость

<div class="article-intro">

Раздел "Посещаемость" в B1 Admin дает вам полное представление о том, кто приходит в вашу церковь и как растут ваши группы. От настройки кампусов и времени служб до анализа тенденций — все инструменты здесь.

</div>

## Обзор страницы

- **Кампусы** — количество физических мест
- **Время служб** — количество служб
- **Запланированные группы** — группы, назначенные службам
- **Незапланированные группы** — независимые группы

## Вкладки

### Настройка
Настройка кампусов, времени служб и групп. См. [Настройка посещаемости](setup.md).

### Посещаемость
Отображение тенденций с течением времени. См. [Отслеживание посещаемости](tracking-attendance.md).

### Группы
Посещаемость по отдельным группам.
