# Посещаемость

<div class="article-intro">

Раздел «Посещаемость» в B1 Admin предоставляет полную картину того, кто приходит в вашу церковь и как ваши группы растут со временем. От настройки кампусов и расписания служб до анализа тенденций и настройки самостоятельной регистрации — всё, что вам нужно для отслеживания и понимания посещаемости, управляется с этой страницы.

</div>

## Обзор страницы

Когда вы открываете страницу посещаемости, вы увидите заголовок с ключевой статистикой настройки посещаемости вашей церкви:

- **Кампусы** — количество настроенных физических локаций
- **Расписание служб** — общее количество служб по всем кампусам
- **Запланированные группы** — группы, привязанные к определённому времени служения
- **Незапланированные группы** — группы, которые отслеживают посещаемость независимо от расписания служб

Под заголовком три вкладки упорядочивают ваши инструменты посещаемости.

## Вкладки

### Настройка

Вкладка **Настройка** — это место, где вы конфигурируете основные элементы отслеживания посещаемости. Здесь вы определяете свои кампусы, добавляете расписание служб для каждого кампуса и назначаете [группы](../groups/creating-groups.md) этим служениям. Вам нужно сделать это только один раз, хотя вы можете обновлять настройки при изменении расписания. Подробное руководство см. в разделе [Настройка посещаемости](setup.md).

### Посещаемость

Вкладка **Посещаемость** отображает тенденции посещаемости с течением времени. Используйте фильтры для сужения результатов по диапазону дат, кампусу или времени служения и быстрого выявления закономерностей в данных. Подробнее об использовании отчётов и фильтров см. в разделе [Отслеживание посещаемости](tracking-attendance.md).

### Группы

Вкладка **Группы** разбивает посещаемость по отдельным группам. Это полезно, когда вы хотите увидеть, как обстоят дела у конкретной малой группы, класса или команды служения, а не рассматривать общие показатели церкви.

:::
