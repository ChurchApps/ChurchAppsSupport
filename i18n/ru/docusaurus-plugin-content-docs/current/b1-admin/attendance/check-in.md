# Регистрация

<div class="article-intro">

B1 Admin поддерживает самостоятельную регистрацию на служениях через сопутствующее приложение **B1 Checkin**. Члены церкви могут зарегистрировать себя и свои семьи на киосках или специальных устройствах по прибытии, что ускоряет процесс и снижает нагрузку на ваших волонтёров. Каждая регистрация автоматически записывается как посещение.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Ваши кампусы, расписание служб и группы должны быть настроены в разделе [Настройка посещаемости](setup.md).
- Вам нужны [люди в вашей базе данных](../people/adding-people.md) с настроенными [домохозяйствами](../people/adding-people.md#managing-households), чтобы семьи могли регистрироваться вместе.
- Вам понадобится планшет и, по желанию, принтер этикеток Brother (см. [рекомендуемое оборудование](#recommended-hardware) ниже).

</div>

## Как это работает

Приложение B1 Checkin подключается к настройке посещаемости в B1 Admin. Когда член церкви регистрируется, его посещение автоматически записывается для правильного кампуса, времени служения и группы. Вам не нужно вручную вводить посещаемость для тех, кто использует систему регистрации.

## Настройка регистрации

1. **Сначала настройте структуру посещаемости.** В B1 Admin перейдите в **Attendance > Setup** и убедитесь, что ваши кампусы, расписание служб и группы настроены. Приложение регистрации зависит от этой конфигурации. Подробнее см. в разделе [Настройка посещаемости](setup.md).
2. **Установите приложение B1 Checkin** на устройства, которые вы планируете использовать. Приложение доступно на следующих платформах:
   - **Планшеты Android/Samsung:** [Google Play Store](https://play.google.com/store/apps/details?id=church.b1.checkin)
   - **Планшеты Amazon Fire:** [Amazon App Store](https://www.amazon.com/Live-Church-Solutions-B1-Check-In/dp/B0FW5HKRB5/)
3. **Войдите в приложение B1 Checkin**, используя учётные данные аккаунта вашей церкви.
4. **Выберите кампус и время служения** для текущего собрания.
5. Теперь члены церкви могут найти своё имя на устройстве и зарегистрироваться.

:::tip
Размещайте устройства для регистрации в видных, легкодоступных местах, таких как входы в фойе или стойки приветствия. Краткое объявление во время служений поможет членам церкви узнать о доступности этой возможности.
:::

:::tip
Если ваша церковь имеет несколько кампусов, вам нужно будет повторить настройку для каждого кампуса в разделе [Настройка посещаемости](setup.md). Каждое устройство для регистрации может быть настроено для другого кампуса.
:::

## Рекомендуемое оборудование

**Планшеты** — любой из них хорошо работает с приложением:

- **Компактный:** Samsung Galaxy Tab A7 Lite 8.7"
- **Большой экран:** Samsung Galaxy Tab A8 10.5"
- **Бюджетный:** Amazon Fire HD 10

**Принтеры** — регистрация работает с принтерами этикеток Brother для печати именных бейджей:

- **Лучший:** Brother QL-1110NWB (поддерживает несколько планшетов через Bluetooth и WiFi)
- **Хороший:** Brother QL-810W (поддерживает несколько планшетов через WiFi)
- **Бюджетный:** Brother QL-1100 (только WiFi)

**Этикетки:** Brother DK-1201 (1-1/7" x 3-1/2")

:::warning
Только принтеры этикеток Brother совместимы с приложением B1 Checkin. Принтеры других марок не будут работать для печати именных бейджей.
:::

:::info
Следуйте инструкциям по настройке вашего принтера, чтобы подключить его к той же сети WiFi, что и ваш планшет. Драйверы и руководства по настройке принтеров Brother можно найти на [сайте поддержки Brother](https://support.brother.com).
:::

## Что записывается

Каждая регистрация создаёт запись о посещении в B1 Admin. Вы можете просматривать эти записи на вкладках [Посещаемость](tracking-attendance.md) и [Группы](../groups/group-members.md) так же, как и данные о посещаемости, введённые вручную. Нет никакой разницы в отображении данных — оба метода попадают в одни и те же отчёты.
