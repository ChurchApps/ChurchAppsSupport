# Настройка посещаемости

<div class="article-intro">

Перед отслеживанием посещаемости вам нужно рассказать B1 Admin о физических местоположениях, времени служб и группах. Эта одноразовая настройка создает структуру для всего отслеживания.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Вам нужен активный аккаунт B1 Admin
- Убедитесь, что ваши [группы созданы](../groups/creating-groups.md)

</div>

## Ключевые концепции

- **Кампус** — физическое место
- **Время служения** — повторяющееся собрание
- **Запланированная группа** — группа, назначенная службе
- **Незапланированная группа** — независимая группа

## Настройка структуры

1. Откройте **B1 Admin** и нажмите **Посещаемость**
2. Выберите вкладку **Настройка**
3. Нажмите **Добавить кампус**
4. Нажмите **Добавить время служения**
5. Назначьте группы временам служения

### Включение отслеживания посещаемости

1. Нажмите **Группы** и выберите группу
2. Нажмите **Редактировать**
3. Установите **Отслеживание посещаемости** на **Да**
4. Нажмите **Сохранить**

## Редактирование

Вы можете обновить настройку в любое время.

:::info
Удаление времени служения не удаляет предыдущие записи.
:::
