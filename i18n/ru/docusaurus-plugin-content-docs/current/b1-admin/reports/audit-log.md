# Журнал аудита

<div class="article-intro">

Журнал аудита отслеживает все значительные действия и изменения в вашей системе управления церкви. Используйте его для проверки деятельности входа и отслеживания ответственности.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Аккаунт B1 Admin с доступом администратора сервера
- Перейдите в **Параметры** для поиска журнала аудита

</div>

## Просмотр журнала аудита

1. Перейдите в **Параметры** в B1 Admin
2. Выберите **Журнал аудита**
3. Журнал отображает недавние записи в таблице со следующими колонками:
   - **Дата** — когда произошло действие
   - **Категория** — тип действия (цветной код)
   - **Действие** — что было сделано
   - **Сущность** — тип и ID затронутой записи
   - **IP адрес** — адрес пользователя
   - **Подробности** — сводка изменений

## Фильтрация журнала

Используйте фильтры вверху:

- **Категория** — тип действия (Логин, Люди, Разрешения, Пожертвования, Группы, Формы, Параметры)
- **Начальная дата** — начало периода
- **Конечная дата** — конец периода

## Понимание категорий

Каждая категория имеет цветовое кодирование:

- **Логин** — голубая
- **Люди** — фиолетовая
- **Разрешения** — красная
- **Пожертвования** — зеленая
- **Группы** — серая
- **Формы** — оранжевая
- **Параметры** — желтая

## Экспорт журнала

Нажмите **Загрузить CSV** для экспорта результатов в электронную таблицу.

## Нумерация страниц

Используйте элементы управления внизу таблицы для навигации.

:::info
Записи журнала аудита автоматически сохраняются в течение одного года.
:::
