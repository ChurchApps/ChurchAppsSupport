# Отчёты

<div class="article-intro">

Страница «Отчёты» предоставляет быстрый доступ к готовым отчётам, которые помогают понять данные вашей церкви. Используйте эти отчёты для отслеживания тенденций посещаемости, мониторинга активности пожертвований и поддержания связи с вашей общиной через празднование дней рождения.

</div>

## Доступные отчёты

Страница «Отчёты» отображает пять карточек отчётов. Нажмите на любую карточку, чтобы открыть отчёт с его фильтрами.

- **[Отчёт о днях рождения](./birthday-report)** -- Просмотр предстоящих дней рождения членов вашей общины.
- **[Тенденции посещаемости](./attendance-reports)** -- Узнайте, как меняются модели посещаемости ваших богослужений с течением времени.
- **[Посещаемость по группам](./attendance-reports)** -- Просмотр итогов посещаемости по группам за выбранный период.
- **[Ежедневная посещаемость по группам](./attendance-reports)** -- Получите ежедневную разбивку данных о посещаемости групп.
- **[Сводка пожертвований](./donation-summary)** -- Обзор итогов и тенденций пожертвований для финансового планирования.

## Использование отчётов

1. Перейдите в **Reports** из главного меню.
2. Нажмите на **карточку отчёта**, чтобы открыть его.
3. Используйте **фильтры** в верхней части каждого отчёта для сужения данных по диапазону дат, группе, богослужению или другим критериям.
4. Просмотрите результаты, отображённые в отчёте.

:::tip
Регулярно запускайте отчёты, чтобы быть в курсе тенденций посещаемости, предстоящих дней рождения и моделей пожертвований. Эти данные помогают более эффективно планировать служения и миссионерскую деятельность.
:::

:::info
Данные отчётов зависят от последовательного ввода данных. Убедитесь, что ваша команда [ведёт учёт посещаемости](../attendance/tracking-attendance.md), [записывает пожертвования](../donations/recording-donations.md) и поддерживает [записи о людях](../people/adding-people.md) в актуальном состоянии для получения наиболее точных результатов.
:::
