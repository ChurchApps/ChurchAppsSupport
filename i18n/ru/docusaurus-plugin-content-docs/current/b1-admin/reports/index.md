# Отчеты

<div class="article-intro">

Страница отчетов дает вам быстрый доступ к предварительно созданным отчетам, которые помогают вам понять данные вашей церкви. Используйте эти отчеты для отслеживания тенденций посещаемости и мониторинга деятельности пожертвований.

</div>

## Доступные отчеты

Страница отчетов отображает карточки отчетов. Нажмите на любую карточку для открытия отчета.

- **[Отчет о днях рождений](./birthday-report)** - Предстоящие дни рождений членов
- **[Тренд посещаемости](./attendance-reports)** - Изменение посещаемости во времени
- **[Посещаемость группы](./attendance-reports)** - Посещаемость по группам
- **[Ежедневная посещаемость группы](./attendance-reports)** - Ежедневные данные
- **[Сводка пожертвований](./donation-summary)** - Тенденции пожертвований
- **[Журнал аудита](./audit-log)** - Отслеживание всех изменений

## Использование отчетов

1. Откройте **Отчеты** из главного меню
2. Нажмите на **карточку отчета** для открытия
3. Используйте **фильтры** в верхней части отчета
4. Просмотрите результаты

:::tip
Запускайте отчеты регулярно, чтобы быть в курсе тенденций.
:::

:::info
Данные отчетов зависят от последовательного ввода данных. Убедитесь, что ваша команда отслеживает посещаемость, записывает пожертвования и держит записи людей в актуальном состоянии.
:::
