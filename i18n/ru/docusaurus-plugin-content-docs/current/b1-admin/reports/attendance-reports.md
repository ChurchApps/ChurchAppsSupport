# Отчёты о посещаемости

<div class="article-intro">

B1 Admin предоставляет три отчёта о посещаемости, которые помогут вам понять, как люди участвуют в ваших служениях и группах. Каждый отчёт даёт свой взгляд на данные о посещаемости — от общих тенденций до ежедневной статистики.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Убедитесь, что посещаемость [отслеживается регулярно](../attendance/tracking-attendance.md) для ваших служб и групп
- Убедитесь, что ваши [группы](../groups/creating-groups.md) и службы настроены в B1 Admin
- Вам нужны соответствующие [разрешения](../settings/roles-permissions.md) для доступа к отчётам

</div>

## Тенденции посещаемости

Отчёт "Тенденции посещаемости" показывает, как посещаемость меняется с течением времени для ваших служб.

1. Перейдите в **Reports** и нажмите на карточку **Attendance Trend**.
2. Установите **диапазон дат** для определения периода.
3. Выберите **службу** или **группу** для фильтрации результатов.
4. Отчёт отобразит линию тренда, показывающую количество посещений за выбранный период.

Этот отчёт полезен для выявления закономерностей: сезонных спадов, тенденций роста или влияния специальных мероприятий.

## Посещаемость групп

Отчёт "Посещаемость групп" показывает общие показатели посещаемости по группам за выбранный период.

1. Перейдите в **Reports** и нажмите на карточку **Group Attendance**.
2. Установите **диапазон дат** для отчёта.
3. Выберите **группу(ы)**, которые хотите просмотреть.
4. Отчёт отобразит общие показатели посещаемости для каждой группы.

Используйте этот отчёт для сравнения посещаемости групп и определения тех, которые растут или нуждаются во внимании.

## Ежедневная посещаемость групп

Отчёт "Ежедневная посещаемость групп" предоставляет поденную разбивку данных о посещаемости для ваших групп.

1. Перейдите в **Reports** и нажмите на карточку **Daily Group Attendance**.
2. Установите **диапазон дат** для отчёта.
3. Выберите **группу(ы)**, которые хотите просмотреть.
4. Отчёт покажет показатели посещаемости за каждый отдельный день в выбранном диапазоне.

Этот отчёт даёт детальную информацию, полезную для понимания изменений от недели к неделе или выявления конкретных дней с необычно высокой или низкой посещаемостью.

:::tip
Используйте отчёт "Тенденции посещаемости" для общего обзора и отчёт "Ежедневная посещаемость групп", когда нужно детально изучить конкретные даты.
:::

## Практическое применение

- **Планирование** -- Используйте тенденции посещаемости для планирования рассадки, персонала и ресурсов для предстоящих служб.
- **Работа с прихожанами** -- Выявляйте снижение посещаемости на ранней стадии, чтобы вовремя связаться с членами.
- **Отчёты для руководства** -- Включайте данные о посещаемости в регулярные отчёты для руководства, чтобы показать состояние служений.
- **Оценка мероприятий** -- Сравнивайте посещаемость до и после специальных мероприятий для оценки их эффективности.

:::warning
Данные о посещаемости записываются через процессы регистрации на ваших группах и службах. Если посещаемость не отслеживается регулярно, ваши отчёты не будут точно отражать реальное участие. Инструкции по настройке см. в [Отслеживание посещаемости](../attendance/tracking-attendance.md).
:::
