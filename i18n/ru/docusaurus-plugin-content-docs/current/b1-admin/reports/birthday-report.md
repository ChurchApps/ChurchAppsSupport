# Отчёт о днях рождения

<div class="article-intro">

Отчёт о днях рождения помогает поддерживать связь с вашими прихожанами, показывая тех, у кого скоро день рождения. Используйте его для планирования празднований, отправки открыток или объявлений во время богослужений.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Убедитесь, что даты рождения введены в профиле каждого человека. Подробнее см. [Добавление людей](../people/adding-people.md).
- Вам необходимы соответствующие [разрешения](../settings/roles-permissions.md) для доступа к отчётам

</div>

## Формирование отчёта

1. Перейдите в раздел **Отчёты** из главного меню.
2. Нажмите на карточку **Отчёт о днях рождения**.
3. Установите **диапазон дат** с помощью фильтров, чтобы определить, какие дни рождения вы хотите увидеть.
4. Отчёт отобразит список прихожан с днями рождения в указанном диапазоне.

## Что показывает отчёт

Отчёт о днях рождения отображает:

- **Имя прихожанина** — имя человека из базы данных вашей церкви.
- **Дата рождения** — дата его рождения.

Результаты отображаются в выбранном вами диапазоне дат, что позволяет легко увидеть, у кого скоро день рождения.

## Практическое применение

- **Еженедельные объявления** — сформируйте отчёт на предстоящую неделю и включите дни рождения в бюллетень или объявления.
- **Поздравительные открытки** — сформируйте список на месяц, и ваша команда сможет отправить личные поздравительные открытки.
- **Лидеры малых групп** — лидеры групп могут формировать отчёт, чтобы помнить о днях рождения участников своей группы.
- **Юбилейные празднования** — фильтруйте по определённым диапазонам дат для планирования юбилейных торжеств.

:::tip
Формируйте этот отчёт в начале каждого месяца для планирования. Вы можете установить диапазон дат на весь месяц, чтобы иметь время для подготовки открыток или объявлений.
:::

:::info
Данные о днях рождения берутся из поля даты рождения в профиле каждого человека. Убедитесь, что ваши [записи о прихожанах](../people/adding-people.md) актуальны для получения точных результатов.
:::
