# Сводка по пожертвованиям

<div class="article-intro">

Отчёт «Сводка по пожертвованиям» даёт чёткую картину активности пожертвований вашей церкви. Используйте его для отслеживания итогов, мониторинга тенденций пожертвований и подготовки финансовых отчётов для вашего руководства.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Убедитесь, что пожертвования [записываются регулярно](../donations/recording-donations.md) в вашей системе
- Вам необходимы соответствующие [разрешения](../settings/roles-permissions.md) для доступа к финансовым отчётам

</div>

## Формирование отчёта

1. Перейдите в раздел **Отчёты** из главного меню.
2. Нажмите на карточку **Сводка по пожертвованиям**.
3. Установите **диапазон дат** для определения отчётного периода.
4. Примените дополнительные фильтры, такие как **фонд** или другие критерии, для сужения результатов.
5. Отчёт отобразит сводку пожертвований за выбранный период.

## Что показывает отчёт

Отчёт «Сводка по пожертвованиям» включает:

- **Общая сумма пожертвований** — общий объём пожертвований за выбранный период.
- **Количество жертвователей** — число уникальных жертвователей, сделавших вклад.
- **Суммы по фондам** — разбивка пожертвований по различным фондам, если применимо.

## Практическое применение

- **Финансовое планирование** — анализируйте тенденции пожертвований для прогнозирования будущих доходов и планирования бюджета.
- **Отчёты для правления** — формируйте сводки для регулярных заседаний правления или руководства с чёткими данными о пожертвованиях.
- **Годовые обзоры** — формируйте отчёт за весь год для подготовки годовых финансовых сводок.
- **Отслеживание кампаний** — фильтруйте по фонду для отслеживания хода специальных кампаний по сбору средств или строительных проектов.
- **Сезонный анализ** — сравнивайте пожертвования за разные периоды для понимания сезонных закономерностей.

:::tip
Формируйте этот отчёт ежемесячно, чтобы быть в курсе финансового состояния вашей церкви. Сравнение данных от месяца к месяцу помогает заблаговременно выявлять тенденции.
:::

:::warning
Данные о пожертвованиях берутся из записанных вкладов в вашей системе. Убедитесь, что пожертвования вводятся регулярно — будь то через онлайн-пожертвования, обработку чеков или ручной ввод — для получения наиболее точных отчётов. Подробнее см. [Запись пожертвований](../donations/recording-donations.md).
:::
