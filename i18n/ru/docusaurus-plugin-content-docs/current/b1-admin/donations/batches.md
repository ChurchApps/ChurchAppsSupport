# Пакеты пожертвований

<div class="article-intro">

Пакеты группируют ваши пожертвования для удобного отслеживания и сверки. Типичный пакет представляет один сбор, например воскресное пожертвование или специальное мероприятие. Использование пакетов помогает поддерживать порядок и упрощает проверку соответствия записей фактическим депозитам.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Убедитесь, что вы [настроили фонды](funds.md), чтобы они были доступны при записи пожертвований
- Вам понадобится доступ к разделу **Пожертвования** в B1 Admin

</div>

## Страница пакетов

При переходе в **Пожертвования > Пакеты** вы увидите список всех ваших пакетов. Каждая строка отображает:

- **Название** — метка, которую вы присвоили пакету
- **Дата** — дата сбора
- **Пожертвования** — количество отдельных пожертвований в пакете
- **Итого** — общая сумма в долларах

Заголовок вверху показывает сводную статистику, включая общее количество пакетов, общее количество пожертвований во всех пакетах и итоговую сумму в долларах.

## Создание нового пакета

1. Нажмите **Добавить пакет** вверху страницы.
2. Введите описательное название (например, «Воскресное пожертвование — 9 фев»).
3. Выберите дату сбора.
4. Нажмите **Сохранить**.

Ваш новый пакет появится в списке, готовый для добавления пожертвований.

## Работа с пакетами

- **Просмотр пожертвований** — нажмите на название пакета, чтобы открыть его и увидеть все отдельные пожертвования. Оттуда вы можете добавлять, редактировать или удалять пожертвования.
- **Редактирование пакета** — нажмите кнопку **Редактировать** в строке пакета, чтобы изменить его название или дату.
- **Сортировка** — используйте заголовки столбцов для сортировки пакетов по названию или дате.
- **Экспорт** — нажмите **Экспорт в CSV**, чтобы скачать список пакетов в виде таблицы.

:::tip
Называйте пакеты последовательно, чтобы их было легко найти позже. Включение даты и типа сбора (например, «Воскресное утро — 2025-02-09») поддерживает порядок в списке по мере его роста.
:::
