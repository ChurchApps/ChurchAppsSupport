# Импорт из Stripe

<div class="article-intro">

Если вы принимаете онлайн-пожертвования через Stripe, инструмент «Импорт из Stripe» позволяет перенести эти транзакции в B1 Admin, чтобы все данные о пожертвованиях были в одном месте. Это особенно полезно для обнаружения транзакций, которые не были автоматически синхронизированы.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Завершите [Настройку онлайн-пожертвований](online-giving-setup.md) для подключения учётной записи Stripe к B1 Admin
- Убедитесь, что в вашей панели управления Stripe есть пожертвования за нужный диапазон дат

</div>

## Как это работает

Процесс импорта использует двухэтапный подход: сначала вы просматриваете, что будет импортировано, затем подтверждаете импорт. Этот подход с предварительным просмотром позволяет проверить всё до создания каких-либо данных.

## Импорт транзакций

1. В **B1 Admin** перейдите в **Пожертвования > Пакеты**.
2. Нажмите ссылку **Импорт из Stripe** внизу страницы «Пакеты».
3. Выберите **диапазон дат** для транзакций, которые хотите импортировать.
4. Нажмите **Предпросмотр** для выполнения пробной проверки.

## Просмотр предпросмотра

Предпросмотр отображает каждую транзакцию из Stripe с индикатором статуса:

- **Новая** — эта транзакция ещё не была импортирована и будет включена, если вы продолжите.
- **Уже импортирована** — эта транзакция уже существует в B1 Admin и будет пропущена.
- **Пропущена** — эта транзакция исключена по другой причине (например, возврат или неудачное списание).

Раздел сводки вверху показывает общее количество транзакций в каждой категории и суммы в долларах.

:::info
Этап предпросмотра не создаёт никаких записей. Это проверка только для чтения, чтобы вы могли убедиться, что произойдёт, прежде чем подтверждать.
:::

## Завершение импорта

1. Просмотрите результаты предпросмотра и итоговые суммы.
2. Нажмите **Импортировать отсутствующие**, чтобы импортировать все транзакции, отмеченные как **Новая**.
3. После завершения импорта индикаторы статуса рядом с каждой транзакцией обновляются, показывая результат.

## Советы по использованию импорта из Stripe

- Запускайте импорт регулярно (еженедельно или ежемесячно), чтобы поддерживать записи в актуальном состоянии.
- Если транзакция показана как **Уже импортирована**, это означает, что в B1 Admin уже есть соответствующая запись — никаких действий не требуется.
- Используйте фильтр диапазона дат, чтобы сосредоточиться на определённом периоде при поиске конкретных транзакций.

:::tip
После импорта посетите страницу [Пакеты](batches.md), чтобы убедиться, что импортированные пожертвования отображаются корректно и суммы совпадают с тем, что вы видите в панели управления Stripe.
:::

## Следующие шаги

- Убедитесь, что импортированные пожертвования назначены правильным [фондам](funds.md) для точных [справок о пожертвованиях](giving-statements.md)
