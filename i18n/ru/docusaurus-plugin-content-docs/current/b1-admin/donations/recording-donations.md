# Запись пожертвований

<div class="article-intro">

Запись пожертвований в B1 Admin выполняется через систему пакетов. Вы создаёте пакет для представления сбора (например, воскресного пожертвования), затем добавляете в него отдельные пожертвования. Это поддерживает порядок в записях о пожертвованиях и упрощает сверку.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Настройте [фонды](funds.md), чтобы можно было назначать пожертвования правильным категориям
- Создайте [пакет](batches.md) для хранения пожертвований, которые вы собираетесь внести
- Убедитесь, что жертвователи есть в вашем [справочнике людей](../people/adding-people.md), чтобы их можно было найти при внесении даров

</div>

## Создание пакета и добавление пожертвований

1. В **B1 Admin** нажмите **Пожертвования** в боковой панели, затем нажмите **Пакеты**.
2. Нажмите **Добавить пакет**.
3. Введите название пакета (например, «Воскресное пожертвование — 5 янв») и выберите дату. Нажмите **Сохранить**.
4. Ваш новый пакет появляется в списке с нулём пожертвований и $0,00.
5. Нажмите на **название пакета**, чтобы открыть его.

## Внесение отдельных пожертвований

1. На странице деталей пакета введите имя жертвователя в **поле поиска**, чтобы найти его.
2. После выбора человека появляется форма ввода пожертвования с полями **Дата**, **Способ оплаты**, **Фонд**, **Сумма** и **Номер чека**.
3. Заполните данные и нажмите **Добавить пожертвование**.
4. Пожертвование добавляется в таблицу ниже, и форма сбрасывается для ввода следующего.

:::tip
Вы можете быстро вносить несколько пожертвований подряд, не покидая страницу пакета. Форма сбрасывается после каждой записи, чтобы вы могли эффективно обработать стопку чеков или конвертов.
:::

## Разделение пожертвования между несколькими фондами

Иногда один жертвователь жертвует в несколько фондов одной транзакцией. Для этого:

1. Нажмите кнопку **Редактировать** в строке пожертвования.
2. В форме редактирования добавьте суммы для разных фондов. Общая сумма автоматически рассчитывается из сумм отдельных фондов.
3. Нажмите **Сохранить** для обновления пожертвования.

:::info
Разделение пожертвований между фондами часто встречается, когда жертвователь выписывает один чек, предназначенный для нескольких целей, например Общий фонд и Миссии.
:::

## Редактирование или удаление пожертвований

Чтобы отредактировать пожертвование, нажмите кнопку **Редактировать** в его строке в пакете. Вы можете изменить дату, сумму, фонд, способ оплаты или любой другой параметр. Нажмите **Сохранить** по завершении.

:::tip
Заголовок страницы пакета автоматически обновляется, показывая общее количество пожертвований и совокупную сумму по мере добавления или редактирования записей. Используйте это для сверки с квитанцией депозита.
:::

## Следующие шаги

- Проверьте записи с помощью [Отчётов о пожертвованиях](donation-reports.md) для проверки точности
- В конце года сгенерируйте [Справки о пожертвованиях](giving-statements.md) для ваших жертвователей
