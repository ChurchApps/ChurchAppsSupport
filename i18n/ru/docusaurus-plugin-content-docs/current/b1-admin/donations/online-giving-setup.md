# Настройка онлайн-сбора пожертвований

<div class="article-intro">

B1 Admin интегрируется с **Stripe** и **PayPal**, поэтому ваши прихожане могут дарить пожертвования онлайн через сайт B1.church. После настройки онлайн-пожертвования автоматически появляются в записях о пожертвованиях вместе с вручную введенными взносами, сохраняя всё в одной системе.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Настройте свои [фонды пожертвований](funds.md), чтобы жертвователи могли определить назначение своих взносов
- Создайте учетную запись Stripe на [stripe.com](https://stripe.com) и активируйте её (выведите из тестового режима)
- Держите под рукой учетные данные для входа в B1 Admin

</div>

## Настройка Stripe

1. Создайте учетную запись на [stripe.com](https://stripe.com), если её ещё нет. Убедитесь, что вы **активировали учетную запись** и вывели её из тестового режима.
2. В Stripe перейдите к **Developers > API Keys**.
3. Скопируйте ваш **Publishable Key** (публичный ключ).
4. Войдите в [B1 Admin](https://admin.b1.church/).
5. Нажмите **Church** (Церковь) в верхней навигации, затем нажмите **Edit Church Settings** (Редактировать параметры церкви).
6. Нажмите значок редактирования рядом с **Church Settings** (Параметры церкви).
7. Прокрутите вниз до раздела **Giving** (Пожертвования).
8. Установите **Provider** (Провайдер) на **Stripe**.
9. Вставьте ваш Publishable Key в поле **Public Key** (Публичный ключ).
10. Вернитесь в Stripe и откройте ваш **Secret Key** (секретный ключ) (его можно просмотреть только один раз, поэтому сохраните копию).
11. Вставьте Secret Key в поле **Secret Key** (Секретный ключ) и нажмите **Save** (Сохранить).

:::warning
Ваш Stripe Secret Key показывается только один раз. Скопируйте его в безопасное место перед тем, как покинуть панель управления Stripe. Если вы его потеряете, вам потребуется создать новый ключ.
:::

## Выбор валюты

После выбора Stripe в качестве провайдера рядом с вашими ключами API появляется раскрывающееся меню **Currency** (Валюта). Выберите валюту, соответствующую валюте расчёта вашего счета Stripe, чтобы пожертвования были начислены правильно.

Поддерживаемые валюты включают USD, EUR, GBP, CAD, AUD, INR, JPY, SGD, HKD, SEK, NOK, DKK, CHF, MXN и BRL. Вы можете подтвердить или изменить валюту по умолчанию вашего счета на [панели управления Stripe](https://dashboard.stripe.com/settings/currencies).

:::info
Выбранная здесь валюта используется для разовых пожертвований, повторяющихся подписок, расчётов комиссий и отчётов о пожертвованиях. Если вы позже измените валюту, только новые пожертвования и подписки будут использовать новую валюту — существующие повторяющиеся взносы остаются в той валюте, в которой они были созданы.
:::

:::warning
Убедитесь, что ваша учетная запись Stripe настроена на принятие выбранной вами валюты. Если ваша учетная запись Stripe не поддерживает выбранную валюту, пожертвования не будут выполнены при оформлении заказа.
:::

## Добавление страницы пожертвований на сайт B1.church

1. Перейдите на [b1.church](https://b1.church/) и войдите.
2. Нажмите значок **Settings** (Параметры).
3. Нажмите **Add Tab** (Добавить вкладку).
4. Выберите **Donation** (Пожертвование) в качестве типа.
5. Введите название вкладки (например, "Give" (Дарить)) и нажмите **Save** (Сохранить).
6. При необходимости измените значок вкладки — введите "Giv" в поле поиска значков, чтобы найти значок, связанный с пожертвованиями.

Ваша страница пожертвований теперь в сети. Прихожане могут посетить её по адресу yoursubdomain.b1.church/donate.

## Обмен ссылкой на пожертвования

Чтобы найти URL пожертвований, перейдите в **B1 Admin** и нажмите значок **Settings** (Параметры), чтобы увидеть ваш поддомен. Ваша ссылка на пожертвования следует формату:

\https://yoursubdomain.b1.church/donate\

Поделитесь этой ссылкой на своем веб-сайте, в электронных письмах или в информационном бюллетене, чтобы прихожане знали, где можно дарить пожертвования онлайн.

## Уведомления о пожертвованиях

Stripe отправляет уведомление электронной почты каждый раз при получении пожертвования. Чтобы изменить адрес электронной почты для уведомлений, перейдите на панель управления Stripe, нажмите на свой профиль в верхнем правом углу, выберите **Profile** (Профиль) и обновите свой адрес электронной почты.

## Варианты комиссий за обработку

Вы можете настроить страницу пожертвований так, чтобы жертвователи могли дополнительно покрыть комиссии за обработку, чтобы ваша церковь получила полную сумму пожертвования. Этот параметр управляется в параметрах вашей церкви в B1 Admin.

:::tip
После настройки сделайте небольшое тестовое пожертвование, чтобы убедиться, что всё работает правильно, прежде чем объявлять онлайн-пожертвования своей общине.
:::
