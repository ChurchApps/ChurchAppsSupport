# Настройка онлайн-пожертвований

<div class="article-intro">

B1 Admin интегрируется с **Stripe**, **PayPal** и **Kingdom Funding**, позволяя вашим членам делать пожертвования онлайн через сайт B1.church. После настройки онлайн-пожертвования автоматически появляются в ваших записях пожертвований наряду с вручную введённые дарами, сохраняя всё в одной системе.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Установите [фонды пожертвований](funds.md), чтобы жертвователи могли обозначить назначение своих подарков
- Создайте аккаунт Stripe на [stripe.com](https://stripe.com) и активируйте его (выведите из тестового режима)
- Подготовьте учётные данные для входа в B1 Admin

</div>

## Настройка Stripe

1. Создайте аккаунт на [stripe.com](https://stripe.com), если у вас его ещё нет. Убедитесь, что **активировали аккаунт** и вывели его из тестового режима.
2. В Stripe перейдите на **Developers > API Keys** (Разработчики > Ключи API).
3. Скопируйте ваш **Publishable Key** (Публичный ключ).
4. Войдите в [B1 Admin](https://admin.b1.church/).
5. Нажмите **Church** (Церковь) в верхней навигации, затем нажмите **Edit Church Settings** (Редактировать параметры церкви).
6. Нажмите значок редактирования рядом с **Church Settings** (Параметры церкви).
7. Прокрутите вниз до раздела **Giving** (Пожертвования).
8. Установите **Provider** (Поставщик) на **Stripe**.
9. Вставьте ваш Publishable Key в поле **Public Key** (Публичный ключ).
10. Вернитесь на Stripe и откройте ваш **Secret Key** (Секретный ключ) (вы сможете посмотреть его только один раз, поэтому сохраните копию).
11. Вставьте Secret Key в поле **Secret Key** (Секретный ключ) и нажмите **Save** (Сохранить).

:::warning
Ваш Secret Key в Stripe показывается только один раз. Скопируйте его в безопасное место перед тем, как уходить с панели Stripe. Если вы его потеряете, вам нужно будет создать новый ключ.
:::

## Выбор валюты

После выбора Stripe в качестве поставщика рядом с ключами API появляется раскрывающийся список **Currency** (Валюта). Выберите валюту, соответствующую валюте расчётов вашего аккаунта Stripe, чтобы пожертвования правильно обрабатывались.

Поддерживаемые валюты включают USD, EUR, GBP, CAD, AUD, INR, JPY, SGD, HKD, SEK, NOK, DKK, CHF, MXN и BRL. Вы можете подтвердить или изменить стандартную валюту вашего аккаунта на [Панели Stripe](https://dashboard.stripe.com/settings/currencies).

:::info
Выбранная здесь валюта используется для одноразовых пожертвований, повторяющихся подписок, расчёта комиссий и отчётов о пожертвованиях. Если вы измените валюту позже, новые пожертвования и подписки будут использовать новую валюту, а существующие повторяющиеся дары будут продолжать использовать валюту, в которой они были созданы.
:::

:::warning
Убедитесь, что ваш аккаунт Stripe настроен на приёмку выбранной вами валюты. Если ваш аккаунт Stripe не поддерживает выбранную валюту, пожертвования будут отклонены при оформлении.
:::

## Добавление страницы пожертвований на ваш сайт B1.church

1. Перейдите на [b1.church](https://b1.church/) и войдите.
2. Нажмите значок **Settings** (Параметры).
3. Нажмите **Add Tab** (Добавить вкладку).
4. Выберите **Donation** (Пожертвование) в качестве типа.
5. Введите название вкладки (например, "Give" (Пожертвовать)) и нажмите **Save** (Сохранить).
6. Если необходимо, измените значок вкладки — введите "Giv" в поиск значков, чтобы найти значок, связанный с пожертвованиями.

Ваша страница пожертвований теперь доступна. Члены могут посетить её по адресу `yoursubdomain.b1.church/donate`.

## Совместное использование ссылки для пожертвований

Чтобы найти URL-адрес пожертвований, перейдите в **B1 Admin** и нажмите значок **Settings** (Параметры), чтобы увидеть ваш поддомен. Ссылка на пожертвования следует формату:

`https://yoursubdomain.b1.church/donate`

Поделитесь этой ссылкой на вашем сайте, в письмах или в пресс-релизе, чтобы члены знали, где могут делать пожертвования онлайн.

## Уведомления о пожертвованиях

Stripe отправляет уведомление по электронной почте каждый раз, когда получено пожертвование. Чтобы изменить адрес электронной почты для уведомлений, перейдите на панель Stripe, нажмите на ваш профиль в правом верхнем углу, выберите **Profile** (Профиль) и обновите адрес электронной почты.

## Варианты платёжной комиссии

Вы можете настроить страницу пожертвований так, чтобы жертвователи могли опционально покрыть платёжные комиссии, чтобы ваша церковь получила полную сумму пожертвования. Этот параметр управляется в параметрах церкви в B1 Admin.

:::tip
После настройки сделайте небольшое тестовое пожертвование, чтобы убедиться, что всё работает, прежде чем объявлять онлайн-пожертвования вашей общине.
:::

## Настройка Kingdom Funding

Kingdom Funding — это христианский платёжный процессор, поддерживающий кредитные/дебетовые карты и банковские переводы ACH. Если ваша церковь зарегистрирована в Kingdom Funding, вы можете подключить его в качестве вашего шлюза пожертвований.

:::info
Интеграция Kingdom Funding в настоящее время находится на стадии бета-тестирования. Свяжитесь с вашим представителем B1, чтобы включить её для вашей церкви.
:::

1. Зарегистрируйтесь или войдите на [kingdomfunding.org](https://kingdomfunding.org).
2. Получите ваш **Security Key** (Ключ безопасности) (публичный) и **Private Key** (Приватный ключ) из портала продавца Kingdom Funding.
3. В B1 Admin перейдите на **Settings** (Параметры) и откройте **Church Settings** (Параметры церкви).
4. В разделе **Giving** (Пожертвования) установите **Provider** (Поставщик) на **Kingdom Funding**.
5. Вставьте ваш Security Key в поле **Security Key** (Ключ безопасности) и ваш Private Key в поле **Private Key** (Приватный ключ).
6. Установите **Webhook Key** (Ключ вебхука), полученный от Kingdom Funding, и скопируйте отображённый URL вебхука в параметры продавца Kingdom Funding, чтобы Kingdom Funding мог уведомлять B1 о завершённых транзакциях.
7. Сохраните.

После подключения члены будут видеть переключатель карта/банк на странице пожертвований и смогут делать пожертвования кредитной картой или банковским переводом.
