# Настройка онлайн-пожертвований

<div class="article-intro">

B1 Admin интегрируется со **Stripe** и **PayPal**, чтобы ваши члены могли жертвовать онлайн через ваш сайт B1.church. После настройки онлайн-пожертвования автоматически появляются в ваших записях наряду с пожертвованиями, внесёнными вручную, сохраняя всё в одной системе.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Настройте [фонды пожертвований](funds.md), чтобы жертвователи могли указать назначение своих даров
- Создайте учётную запись Stripe на [stripe.com](https://stripe.com) и активируйте её (выведите из тестового режима)
- Подготовьте учётные данные для входа в B1 Admin

</div>

## Настройка Stripe

1. Создайте учётную запись на [stripe.com](https://stripe.com), если у вас её ещё нет. Обязательно **активируйте учётную запись** и выведите её из тестового режима.
2. В Stripe перейдите в **Developers > API Keys**.
3. Скопируйте ваш **Publishable Key** (публичный ключ).
4. Войдите в [B1 Admin](https://admin.b1.church/).
5. Нажмите **Church** в верхней навигации, затем нажмите **Edit Church Settings**.
6. Нажмите значок редактирования рядом с **Church Settings**.
7. Прокрутите вниз до раздела **Giving**.
8. Установите **Provider** на **Stripe**.
9. Вставьте Publishable Key в поле **Public Key**.
10. Вернитесь в Stripe и откройте ваш **Secret Key** (секретный ключ) (вы можете просмотреть его только один раз, поэтому сохраните копию).
11. Вставьте Secret Key в поле **Secret Key** и нажмите **Save**.

:::warning
Ваш Secret Key от Stripe показывается только один раз. Скопируйте его в безопасное место, прежде чем покинуть панель управления Stripe. Если вы потеряете его, вам потребуется сгенерировать новый ключ.
:::

## Добавление страницы пожертвований на ваш сайт B1.church

1. Перейдите на [b1.church](https://b1.church/) и войдите в систему.
2. Нажмите значок **Настройки**.
3. Нажмите **Add Tab**.
4. Выберите **Donation** в качестве типа.
5. Введите название вкладки (например, «Пожертвовать») и нажмите **Save**.
6. При желании измените значок вкладки — введите «Giv» в поиске значков для значка, связанного с пожертвованиями.

Ваша страница пожертвований теперь активна. Члены могут посетить её по адресу `вашподдомен.b1.church/donate`.

## Распространение ссылки для пожертвований

Чтобы найти URL для пожертвований, перейдите в **B1 Admin** и нажмите значок **Настройки**, чтобы увидеть ваш поддомен. Ссылка для пожертвований имеет формат:

`https://вашподдомен.b1.church/donate`

Поделитесь этой ссылкой на вашем сайте, в электронных письмах или в объявлениях, чтобы члены знали, где можно пожертвовать онлайн.

## Уведомления о пожертвованиях

Stripe отправляет уведомление по электронной почте каждый раз при получении пожертвования. Чтобы изменить адрес электронной почты для уведомлений, перейдите в панель управления Stripe, нажмите на ваш профиль в правом верхнем углу, выберите **Profile** и обновите адрес электронной почты.

## Варианты комиссии за обработку

Вы можете настроить страницу пожертвований так, чтобы жертвователи могли по желанию покрыть комиссию за обработку, и ваша церковь получала полную сумму пожертвования. Эта настройка управляется в настройках церкви в B1 Admin.

:::tip
После настройки сделайте небольшое тестовое пожертвование, чтобы убедиться, что всё работает, прежде чем объявлять об онлайн-пожертвованиях вашей общине.
:::

## Следующие шаги

- Используйте [Импорт из Stripe](stripe-import.md) для переноса онлайн-транзакций в B1 Admin, если они не синхронизируются автоматически
- Проверьте [Отчёты о пожертвованиях](donation-reports.md), чтобы убедиться, что онлайн-пожертвования отображаются корректно
- Сгенерируйте [Справки о пожертвованиях](giving-statements.md), включающие и онлайн, и офлайн-пожертвования
