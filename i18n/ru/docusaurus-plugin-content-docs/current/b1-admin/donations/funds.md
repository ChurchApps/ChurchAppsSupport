# Управление фондами

<div class="article-intro">

Фонды позволяют классифицировать пожертвования по назначению, чтобы вы могли отслеживать, сколько ваша церковь получает для каждого направления служения. Типичные фонды включают Общий фонд, Строительный фонд, Миссии и Молодёжное служение, но вы можете создать столько, сколько необходимо.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Вам понадобится доступ к разделу **Пожертвования** в B1 Admin
- Определите, какие категории пожертвований нужны вашей церкви и должна ли каждая из них быть налоговым вычетом

</div>

## Страница фондов

Перейдите в **Пожертвования > Фонды**, чтобы увидеть список всех ваших фондов. Каждая строка показывает название фонда и его статус **налогового вычета**, который определяет, будут ли пожертвования в этот фонд включены в годовые [справки о пожертвованиях](giving-statements.md).

## Создание нового фонда

1. Нажмите **Добавить фонд** вверху страницы.
2. Введите название фонда (например, «Строительный фонд»).
3. Установите флажок **Налоговый вычет**, если пожертвования в этот фонд должны включаться в справки о пожертвованиях.
4. Нажмите **Сохранить**.

Новый фонд сразу появляется в вашем списке и доступен при внесении пожертвований.

## Редактирование и удаление фондов

- Чтобы отредактировать фонд, нажмите кнопку **Редактировать** рядом с ним. Вы можете изменить название или переключить настройку налогового вычета.
- Чтобы удалить фонд, откройте форму редактирования и нажмите **Удалить**.

:::warning
Удаление фонда не удаляет пожертвования, ранее назначенные ему. Эти записи о пожертвованиях остаются в системе, но фонд больше не будет доступен для новых пожертвований.
:::

## Просмотр деталей фонда

Нажмите на **название фонда**, чтобы открыть страницу его деталей. Эта страница показывает:

- **Историю пожертвований** — список всех пожертвований в этот фонд
- **Фильтр диапазона дат** — ограничение результатов определённым периодом
- **Сводную статистику** — общее количество пожертвований, уникальные жертвователи и общая сумма в долларах

Этот вид полезен, когда нужно отчитаться по конкретной инициативе пожертвований или проверить прогресс в достижении цели сбора средств.

## Сортировка и экспорт

- Используйте заголовки столбцов для сортировки фондов по названию.
- Нажмите **Экспорт в CSV**, чтобы скачать список фондов в виде таблицы.

:::tip
Настройте фонды до того, как начнёте [записывать пожертвования](recording-donations.md). Наличие всех фондов обеспечивает правильную категоризацию пожертвований с самого начала.
:::

## Следующие шаги

После настройки фондов вы можете назначать их пожертвованиям при [записи пожертвований](recording-donations.md) в пакет. Убедитесь, что соответствующие фонды отмечены как налоговый вычет перед генерацией [справок о пожертвованиях](giving-statements.md).
