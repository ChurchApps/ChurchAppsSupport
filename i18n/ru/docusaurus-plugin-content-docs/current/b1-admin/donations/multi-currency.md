# Поддержка нескольких валют

<div class="article-intro">

Функция нескольких валют B1 позволяет вашей церкви принимать и отслеживать пожертвования в различных валютах. Это особенно полезно для церквей с международными членами, миссионерами или несколькими кампусами в разных странах.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Вам нужно разрешение на управление пожертвованиями. См. [Роли и права доступа](../people/roles-permissions.md) для получения подробной информации.
- Настройте [онлайн-пожертвования](./online-giving-setup.md) с помощью Stripe, который поддерживает транзакции в нескольких валютах.
- Понимайте потребности вашей церкви в бухгалтерском учете для работы с несколькими валютами.

</div>

## Включение нескольких валют

Поддержка нескольких валют теперь включена по умолчанию в B1. После включения:

- Члены могут жертвовать в своей местной валюте при онлайн-пожертвованиях
- Вы можете вручную записывать пожертвования в любой валюте
- Отчеты о пожертвованиях показывают суммы в их исходной валюте
- Stripe автоматически обрабатывает конвертацию валют для онлайн-пожертвований

## Поддерживаемые валюты

Система поддерживает все основные мировые валюты, включая:

- **USD** -- Доллар США
- **EUR** -- Евро
- **GBP** -- Британский фунт
- **CAD** -- Канадский доллар
- **AUD** -- Австралийский доллар
- **MXN** -- Мексиканское песо
- **BRL** -- Бразильский реал
- **INR** -- Индийская рупия
- **CNY** -- Китайский юань
- **JPY** -- Японская иена
- И многие другие...

Доступные валюты для онлайн-пожертвований зависят от поддерживаемых валют вашей учетной записи Stripe.

## Запись пожертвований в различных валютах

### Онлайн-пожертвования

Когда член жертвует онлайн через Stripe:

1. Они выбирают предпочтительную валюту при оформлении заказа
2. Stripe обрабатывает платеж в этой валюте
3. Пожертвование записывается в B1 с исходной суммой в валюте
4. Stripe автоматически обрабатывает любую необходимую конвертацию валюты в валюту вашей учетной записи по умолчанию

### Ручной ввод

Чтобы записать пожертвование наличными или чеком в другой валюте:

1. Перейдите в раздел **Пожертвования** в B1 Admin
2. Нажмите **Добавить пожертвование**
3. Выберите валюту из выпадающего списка валют
4. Введите сумму в этой валюте
5. Заполните остальные детали пожертвования
6. Нажмите **Сохранить**

## Просмотр пожертвований в нескольких валютах

### Отчеты о пожертвованиях

Отчеты о пожертвованиях отображают суммы в их исходной валюте:

- Отдельные записи о пожертвованиях показывают код валюты (например, "$100.00 USD")
- Итоги рассчитываются по каждой валюте
- Вы можете фильтровать по конкретным валютам

### Отчеты о пожертвованиях

При создании отчетов о пожертвованиях:

- Каждое пожертвование отображается с исходной валютой
- Итоги разбиты по валютам
- Члены видят точно то, что они пожертвовали в каждой валюте

## Интеграция с Stripe

Для онлайн-пожертвований Stripe обрабатывает транзакции в нескольких валютах:

- **Автоматическая конвертация** -- Stripe конвертирует валюты в валюту вашей учетной записи по умолчанию
- **Обменные курсы** -- Stripe использует текущие рыночные обменные курсы
- **Комиссии** -- Конвертация валюты может повлечь дополнительные комиссии Stripe
- **Валюта выплаты** -- Средства депонируются в валюте вашей учетной записи по умолчанию

:::info
Проверьте панель управления Stripe, чтобы увидеть текущие курсы конвертации и любые комиссии, связанные с транзакциями в нескольких валютах.
:::

## Соображения по бухгалтерскому учету

При работе с несколькими валютами:

- **Ведение записей** -- Отслеживайте исходные суммы пожертвований и валюты для точной отчетности
- **Обменные курсы** -- Обратите внимание, что курсы конвертации Stripe могут отличаться от курсов вашего банка
- **Налоговые квитанции** -- Проконсультируйтесь со своим бухгалтером о том, как сообщать о пожертвованиях в различных валютах для налоговых целей
- **Распределение по фондам** -- Вы можете распределять пожертвования по конкретным фондам независимо от валюты

## Лучшие практики

- **Валюта по умолчанию** -- Установите основную валюту вашей церкви как валюту по умолчанию для большинства транзакций
- **Четкая коммуникация** -- Сообщайте донорам, в какой валюте они жертвуют, во время процесса оформления заказа
- **Последовательная отчетность** -- Решите, следует ли отчитываться в исходных валютах или конвертировать в единую валюту для сводок
- **Регулярная сверка** -- Сверяйте выплаты Stripe с записями о пожертвованиях, учитывая конвертацию валют

## Ограничения

- Конвертация валюты обрабатывается Stripe только для онлайн-пожертвований
- Ручные пожертвования записываются как введенные без автоматической конвертации
- Исторические отчеты показывают пожертвования в их исходных валютах
- Общие расчеты выполняются по каждой валюте, а не между валютами
