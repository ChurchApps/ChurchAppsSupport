# Справки о пожертвованиях

<div class="article-intro">

В конце каждого года ваши жертвователи нуждаются в справке о своих пожертвованиях, подлежащих налоговому вычету, для своих записей. B1 Admin упрощает генерацию этих справок для всех жертвователей одновременно, экономя часы ручной работы.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Убедитесь, что ваши [фонды](funds.md) правильно отмечены как **Налоговый вычет** — только пожертвования в фонды с налоговым вычетом включаются в справки
- Убедитесь, что все пожертвования [записаны](recording-donations.md) и все онлайн-транзакции [импортированы из Stripe](stripe-import.md)

</div>

## Доступ к справкам о пожертвованиях

1. В **B1 Admin** нажмите **Пожертвования** в боковой панели.
2. Нажмите **Справки**.

## Генерация справок

1. Выберите **год** из выпадающего списка вверху страницы. Вы можете выбрать текущий год или любой из пяти предыдущих лет.
2. Страница отображает сводную статистику за этот год, включая:
   - **Всего жертвователей** — количество людей, которые пожертвовали
   - **Всего пожертвований** — количество отдельных записей о пожертвованиях
   - **Общая сумма** — совокупная сумма в долларах всех пожертвований

## Скачивание справок

У вас есть два варианта передачи справок жертвователям:

### Скачать как CSV-файлы

Нажмите **Скачать ZIP**, чтобы скачать ZIP-файл с отдельным CSV-файлом для каждого жертвователя. Это удобно, если вы хотите отправить справки по электронной почте индивидуально или импортировать их в другую систему.

### Распечатать все справки

Нажмите **Печать всех**, чтобы открыть печатный вид справки каждого жертвователя в браузере. Оттуда используйте функцию печати браузера для отправки на принтер. Каждая справка начинается с новой страницы, готовая к складыванию и отправке по почте.

:::tip
Сгенерируйте справки в начале января, пока записи свежие. Перепроверьте, что фонды правильно отмечены как налоговый вычет перед генерацией справок — включаются только пожертвования в фонды с налоговым вычетом.
:::

:::info
Справки о пожертвованиях включают только пожертвования, назначенные фондам с включённой настройкой **Налоговый вычет**. Если фонд не отмечен как налоговый вычет, его пожертвования не появятся в справке. Вы можете управлять этой настройкой на странице [Фонды](funds.md).
:::
