# Отчеты о пожертвованиях

B1 Admin дает несколько способов просмотра и анализа данных о пожертвованиях. Используйте эти инструменты для отслеживания тенденций пожертвований и подготовки отчетов.

## Приборная панель пожертвований

1. Откройте **Пожертвования** в боковой панели
2. Вы увидите четыре KPI карточки с ключевыми метриками:
   - **Всего пожертвовано**
   - **Среднее пожертвование**
   - **Уникальные донаторы**
   - **Всего пожертвований**
3. Используйте переключатель периода для недельного, месячного и квартального представления
4. Диаграмма показывает тенденции пожертвований
5. Нажмите **Загрузить** для экспорта CSV

## Страница сводки пожертвований

1. Откройте **Пожертвования** в боковой панели
2. Используйте **фильтр диапазона дат**
3. Просмотрите диаграмму еженедельных пожертвований
4. Нажмите **Загрузить** для экспорта CSV

:::info
Страница сводки показывает агрегированные данные без имен отдельных доноров.
:::

## Просмотр деталей донора

1. Откройте **Пожертвования > Партии**
2. Нажмите на **название партии**
3. Вы увидите каждое пожертвование с деталями донора
4. Нажмите на имя донора для подробного просмотра
5. Нажмите на ID пожертвования для полных деталей
6. Нажмите **Загрузить** для экспорта CSV

## Отчет сводки пожертвований

В разделе **Отчеты**:

1. Нажмите **Отчеты**
2. Выберите **Сводка пожертвований**
3. Установите фильтры и запустите отчет

## Экспорт данных

Вы можете экспортировать данные из разных мест:

- **Страница сводки** — еженедельные суммы по фондам
- **Страница деталей партии** — отдельные пожертвования с деталями
- **Страница деталей фонда** — история пожертвований для фонда
