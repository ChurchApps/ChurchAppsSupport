# Отчёты о пожертвованиях

<div class="article-intro">

B1 Admin предоставляет несколько способов просмотра и анализа данных о пожертвованиях вашей церкви. Страница «Сводка пожертвований» даёт визуальный обзор с графиками и фильтрами, а раздел «Отчёты» содержит более детальный отчёт «Сводка пожертвований». Используйте эти инструменты для отслеживания тенденций, подготовки к совещаниям или сверки записей.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Убедитесь, что пожертвования были [записаны в пакеты](recording-donations.md) или [импортированы из Stripe](stripe-import.md)
- Проверьте, что ваши [фонды](funds.md) настроены правильно для корректной категоризации пожертвований

</div>

## Страница сводки пожертвований

Страница **Сводка** — первое, что вы видите при нажатии **Пожертвования** в боковой панели. Она даёт обзор активности пожертвований верхнего уровня.

1. Перейдите в **Пожертвования** в боковой панели, чтобы открыть страницу «Сводка».
2. Используйте **фильтр диапазона дат**, чтобы выбрать период для просмотра. Укажите более раннюю дату вверху и более позднюю внизу.
3. Страница отображает график еженедельных пожертвований для быстрого обзора тенденций.
4. Нажмите **Скачать**, чтобы экспортировать CSV-файл с общей суммой, неделей пожертвования и фондом назначения.

:::info
Страница «Сводка» показывает агрегированные данные о пожертвованиях. Она не включает имена отдельных жертвователей. Для детальной информации по жертвователям используйте страницу [Пакеты](batches.md).
:::

## Детальная информация по жертвователям

Для разбивки по тому, кто пожертвовал, сколько и в какой фонд:

1. Перейдите в **Пожертвования > Пакеты**.
2. Нажмите на **название пакета**, чтобы открыть его.
3. Страница деталей пакета показывает каждое пожертвование с именем жертвователя, суммой, фондом, датой и способом оплаты.
4. Нажмите на **имя жертвователя**, чтобы увидеть, сколько раз он жертвовал и какие суммы.
5. Нажмите на **ID пожертвования**, чтобы открыть боковую панель с полными деталями этого пожертвования.
6. Нажмите **Скачать**, чтобы экспортировать CSV со всей информацией о жертвователях и пожертвованиях этого пакета.

## Отчёт «Сводка пожертвований»

B1 Admin также включает отчёт **Сводка пожертвований** в разделе «Отчёты»:

1. Нажмите **Отчёты** в боковой панели.
2. Выберите отчёт **Сводка пожертвований**.
3. Выберите фильтры (диапазон дат, фонд, кампус и т.д.) и запустите отчёт.

## Экспорт данных

Вы можете экспортировать данные о пожертвованиях из нескольких мест:

- **Страница «Сводка»** — скачать CSV еженедельных итогов по фондам
- **Страница деталей пакета** — скачать CSV отдельных пожертвований с данными жертвователей
- **Страница деталей фонда** — скачать историю пожертвований для конкретного фонда

:::tip
Для годовой отчётности совместите экспорт со страницы «Сводка» с инструментом [Справки о пожертвованиях](giving-statements.md), чтобы получить и агрегированные тенденции, и индивидуальные справки жертвователей.
:::
