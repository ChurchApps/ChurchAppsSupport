# Управление страницами

<div class="article-intro">

Представление «Страницы сайта» — это ваш центр для создания, редактирования и организации всех страниц вашего церковного сайта. Вы можете управлять как контентом страниц, так и навигацией сайта с одного экрана.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Выполните [Начальную настройку](initial-setup) для настройки домена и базовых параметров сайта
- Подготовьте контент и изображения. Используйте менеджер [Файлов](files) для предварительной загрузки медиаресурсов.

</div>

## Типы страниц

Таблица **Страницы** перечисляет каждую страницу вашего сайта вместе с её статусом:

- **Сгенерированная** — страницы, автоматически созданные системой на основе данных вашей церкви (например, страница «Группы» или страница «Проповеди»). Эти страницы обновляются автоматически при изменении данных.
- **Пользовательская** — страницы, которые вы создали самостоятельно с собственным контентом и макетом.

Вы можете преобразовать любую автоматически сгенерированную страницу в пользовательскую, если хотите получить полный контроль над её контентом и дизайном.

## Добавление и редактирование страниц

1. Нажмите кнопку **Добавить страницу** в правом верхнем углу таблицы «Страницы».
2. Выберите тип страницы (пустая или шаблон) и дайте ей название.
3. Нажмите **Редактировать** рядом с любой страницей, чтобы открыть конструктор страниц, где можно добавлять секции, текст, изображения и другие элементы.
4. Нажмите **Настройки страницы**, чтобы обновить название страницы, URL-путь и другие метаданные.
5. Используйте кнопку **Предпросмотр**, чтобы открыть страницу в новом окне и увидеть, как она будет выглядеть для посетителей.

:::tip
Для главной страницы установите URL-путь просто `/`. Для всех остальных страниц используйте описательный путь, например `/about` или `/contact`.
:::

## Управление навигацией

Боковая панель слева в представлении «Страницы сайта» отображает ваши навигационные ссылки. Эти ссылки управляют меню, которое видят посетители на вашем сайте.

1. Нажмите **Добавить**, чтобы создать новую навигационную ссылку. Вы можете направить её на любую страницу вашего сайта или на внешний URL.
2. Для изменения порядка ссылок перетаскивайте их. Вы также можете вложить ссылки в родительский элемент для создания выпадающих меню.
3. Нажмите значок **Редактировать** рядом с любой ссылкой, чтобы изменить её название, URL или позицию.
4. Чтобы удалить ссылку из навигации, нажмите значок **Удалить**.

:::info
Удаление навигационной ссылки не удаляет саму страницу. Страница по-прежнему существует и доступна напрямую по её URL — она просто не будет отображаться в меню.
:::

## Советы по организации сайта

- Ограничьте навигацию верхнего уровня пятью-шестью элементами, чтобы посетители могли быстро находить нужное.
- Используйте вложенные ссылки для связанных подстраниц (например, выпадающий список «О нас» с пунктами «Наша команда», «Вероучение» и «История»).
- Проверьте навигацию на мобильных устройствах, нажав **Мобильный предпросмотр**, чтобы убедиться, что она хорошо работает на маленьких экранах.
- Давайте страницам понятные, описательные названия, которые помогут посетителям понять, что они найдут.

:::tip
Вы можете добавлять [формы](../forms/creating-forms.md) на ваши страницы для сбора регистраций, молитвенных просьб или другой информации от посетителей.
:::
