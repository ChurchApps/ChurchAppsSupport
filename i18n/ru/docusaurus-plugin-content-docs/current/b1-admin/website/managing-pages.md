# Управление страницами

<div class="article-intro">

Вид страниц веб-сайта — это ваш центр для создания, редактирования и организации всех страниц вашего веб-сайта церкви. Вы можете управлять содержимым страниц и навигацией сайта с этого единственного экрана.

</div>

- Подготовьте ваше содержимое и изображения

</div>

## Понимание типов страниц

Таблица **Страницы** содержит все страницы на вашем сайте:

- **Сгенерированные** — автоматически созданные системой
- **Пользовательские** — созданные вами

## Добавление и редактирование страниц

1. Нажмите кнопку **Добавить страницу**
2. Выберите тип страницы (пусто или шаблон)
3. Нажмите **Редактировать** для открытия редактора страниц
4. Нажмите **Параметры страницы** для обновления заголовка и пути URL
5. Используйте кнопку **Предпросмотр** для просмотра страницы

## Управление навигацией

1. Нажмите **Добавить** для создания новой навигационной ссылки
2. Перетащите ссылки для переупорядочивания
3. Нажмите **Редактировать** для изменения ссылки
4. Нажмите **Удалить** для удаления ссылки

## Советы по организации вашего сайта

- Сохраняйте навигацию на 5-6 элементов верхнего уровня
- Используйте вложенные ссылки для подстраниц
- Проверьте навигацию на мобильных устройствах
