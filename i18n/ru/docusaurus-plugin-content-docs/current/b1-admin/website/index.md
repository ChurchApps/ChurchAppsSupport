# Сайт

<div class="article-intro">

Раздел «Сайт» в B1 Admin предоставляет полнофункциональный конструктор сайтов для вашей церкви. Вы можете создавать страницы, настраивать внешний вид сайта, управлять медиафайлами и публиковать календари событий — всё это без написания кода.

</div>

## Что вы можете сделать

Конструктор сайтов организован в пять основных областей:

### Страницы

Создавайте и управляйте всеми страницами вашего церковного сайта. Таблица **Страницы** показывает каждую страницу и её статус — либо **Сгенерированная** (автоматически созданная системой), либо **Пользовательская** (созданная вами). Вы можете добавлять новые пользовательские страницы, редактировать существующие или преобразовывать автоматически сгенерированные страницы в пользовательские для полного контроля над их контентом.

Боковая панель слева отображает навигационные ссылки в виде древовидной структуры. Вы можете переупорядочивать ссылки перетаскиванием, чтобы установить точную иерархию и порядок, который увидят посетители.

### Блоки

Создавайте переиспользуемые компоненты контента, которые можно размещать на нескольких страницах. Блоки экономят время, когда у вас есть контент — например, призыв к действию, баннер с объявлением или раздел контактов — который появляется более чем в одном месте. Обновите блок один раз, и изменение применится везде, где он используется.

### Внешний вид

Настройте тему вашего сайта, цвета, шрифты, логотип и отступы. Вы можете применить готовые темы стилей для быстрого старта или точно настроить каждую деталь с помощью редактора цветовой палитры, настроек типографики и пользовательского CSS. Подробнее см. руководство [Внешний вид](appearance).

### Файлы

Загружайте и организуйте все медиаресурсы — изображения, документы и другие файлы — в одном центральном месте. Используйте папки для поддержания порядка и копируйте URL файлов для использования в любом месте вашего сайта. Подробнее см. [Файлы](files).

### Календари

Публикуйте календари событий на вашем сайте, чтобы посетители видели, что происходит в вашей церкви. Создавайте сводные календари, которые собирают события из нескольких групп в единое представление.

:::tip
Начните с [Начальной настройки](initial-setup), если вы настраиваете сайт впервые. Она проведёт вас через настройку домена, создание первой страницы и настройку внешнего вида сайта.
:::
