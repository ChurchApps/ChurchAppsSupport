# Файлы

<div class="article-intro">

Страница «Файлы» — это центральное хранилище всех медиафайлов и документов, используемых на вашем церковном сайте. Здесь вы можете загружать, организовывать, просматривать и управлять каждым изображением, документом и медиафайлом, необходимым вашему сайту.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Выполните [Начальную настройку](initial-setup) вашего сайта
- Оптимизируйте изображения перед загрузкой для более быстрой загрузки страниц
- Подготовьте файлы в распространённых веб-форматах (PNG, JPG, PDF и т.д.)

</div>

## Загрузка файлов

1. Перейдите в раздел **Сайт** в B1 Admin и нажмите вкладку **Файлы**.
2. Перетащите файлы непосредственно в область загрузки или нажмите кнопку загрузки, чтобы выбрать файлы на компьютере.
3. Ваши файлы будут загружены и сразу появятся в библиотеке файлов.

:::tip
Для лучших результатов оптимизируйте изображения перед загрузкой. Меньший размер файлов помогает вашему сайту загружаться быстрее для посетителей. Такие инструменты, как TinyPNG или Squoosh, могут сжать изображения без заметной потери качества.
:::

## Организация с помощью папок

1. Создавайте папки для группировки связанных файлов — например, «Графика проповедей», «Баннеры событий» или «Фото сотрудников».
2. Перемещайте файлы между папками, чтобы поддерживать порядок в библиотеке по мере её роста.
3. Используйте понятные, описательные названия папок, чтобы ваша команда могла быстро находить файлы.

## Просмотр и поиск

- Прокручивайте библиотеку файлов, чтобы видеть миниатюры всех загруженных файлов.
- Используйте функцию **поиска**, чтобы найти файлы по имени, когда библиотека станет большой.
- Нажмите на любой файл, чтобы просмотреть его детали, включая имя файла, размер и URL.

## Использование файлов на вашем сайте

1. Нажмите на файл, чтобы просмотреть его детали.
2. **Скопируйте URL файла** для использования на страницах вашего сайта. Вы можете вставить этот URL в элементы изображений, ссылки для скачивания или в любое другое место, принимающее URL файла.
3. Загруженные здесь файлы доступны для использования на всех страницах вашего сайта и в блоках контента.

:::info
При создании страниц в [Управлении страницами](managing-pages) вы можете ссылаться на любой загруженный здесь файл по его URL. Это позволяет хранить все медиафайлы в одном центральном месте.
:::

## Удаление файлов

1. Выберите файл, который хотите удалить.
2. Нажмите кнопку **Удалить**, чтобы навсегда удалить его из библиотеки.
3. Убедитесь, что файл не используется ни на одной странице перед удалением, так как это нарушит все ссылки или изображения, которые на него ссылаются.

:::warning
Удалённые файлы невозможно восстановить. Перед удалением дважды проверьте, что файл больше не используется ни на одной странице.
:::
