# Начальная настройка

<div class="article-intro">

Каждая учётная запись B1 поставляется с готовым сайтом. Это руководство проведёт вас через настройку домена вашей церкви, конфигурацию внешнего вида сайта, создание первых страниц и организацию навигации.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Вам нужна учётная запись B1.church с административным доступом
- Если вы используете собственный домен, подготовьте учётные данные вашего DNS-провайдера (например, GoDaddy, Cloudflare или AWS)
- Подготовьте логотип вашей церкви в формате PNG с прозрачным фоном для лучших результатов

</div>

## Настройка домена

Ваша церковь автоматически получает поддомен на B1.church (например, `yourchurch.b1.church`). Вы также можете направить собственный домен на ваш сайт B1.

1. Перейдите в **B1.church Admin**, посетив admin.b1.church или нажав на выпадающее меню профиля и выбрав **Переключить приложение**.
2. Нажмите **Панель управления** в левой боковой панели, затем выберите **Настройки** из выпадающего меню.
3. Нажмите **Управление**, чтобы просмотреть ваш поддомен. Установите его коротким и узнаваемым, без пробелов.
4. Чтобы использовать собственный домен, войдите в панель вашего DNS-провайдера (например, GoDaddy, Cloudflare или AWS) и добавьте две записи:
   - **A-запись** для корневого домена, указывающую на `3.23.251.61`
   - **CNAME-запись** для `www`, указывающую на `proxy.b1.church`
5. Вернитесь в B1.church Admin, добавьте ваш собственный домен в список и нажмите **Добавить**, затем **Сохранить**. Ваш сайт будет доступен по собственному домену в течение нескольких минут.

:::tip
Если вы не видите опцию «Настройки», попросите человека, создавшего учётную запись вашей церкви, предоставить вам разрешение «Редактирование настроек церкви». Подробнее см. [Роли и разрешения](../settings/roles-permissions.md).
:::

## Создание первой страницы

1. В B1 Admin нажмите **Сайт** в левом меню, чтобы открыть представление «Страницы сайта».
2. Нажмите **Добавить страницу** в правом верхнем углу.
3. Выберите **Пустая** в качестве типа страницы и назовите её «Главная».
4. Нажмите **Настройки страницы** и установите URL-путь `/` (только косая черта без текста) для главной страницы. Другие страницы используют `/page-name`.
5. Нажмите **Редактировать контент**, чтобы начать создание. Каждая страница должна начинаться с **Секции** — это контейнер для всех остальных элементов.
6. После добавления секции нажмите **Добавить контент** снова, чтобы вставить текст, изображения, видео, карточки, формы и многое другое, перетаскивая их в вашу секцию.

:::info
Подробные инструкции по работе со страницами, навигацией и типами страниц см. в [Управлении страницами](managing-pages).
:::

## Настройка внешнего вида сайта

1. В представлении «Страницы сайта» нажмите вкладку **Внешний вид** вверху.
2. Используйте **Цветовую палитру** для установки фирменных цветов — основного, вторичного и акцентного.
3. В разделе **Настройки типографики** выберите шрифты для заголовков и основного текста из обозревателя шрифтов.
4. Загрузите логотип вашей церкви в разделе **Логотип** в «Настройках стилей». Предоставьте версии для светлого и тёмного фона.
5. Настройте **Подвал сайта** с контактной информацией и ссылками вашей церкви.

:::info
Изменения, которые вы вносите во «Внешнем виде», применяются ко всему вашему сайту. Подробные инструкции по каждой настройке см. на странице [Внешний вид](appearance).
:::

## Настройка навигации

Навигационные ссылки отображаются в левой боковой панели представления «Страницы сайта». Чтобы организовать их:

1. Нажмите **Добавить**, чтобы создать новую навигационную ссылку и направить её на одну из ваших страниц.
2. Перетаскивайте ссылки для изменения порядка или вложения их в родительские элементы.
3. Просмотрите свой сайт, чтобы убедиться, что навигация выглядит правильно.
