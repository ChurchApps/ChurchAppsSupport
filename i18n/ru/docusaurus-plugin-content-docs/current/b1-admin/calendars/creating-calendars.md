# Создание календарей

Создание календаря в B1 Admin дает центральное место для управления событиями. Каждый календарь может включать события из одной или нескольких групп.

## Создание нового календаря

1. Откройте **Календари** в B1 Admin
2. Нажмите **Добавить календарь**
3. Введите **название** (например, "События молодежного служения")
4. Добавьте опциональное **описание**
5. Нажмите **Создать**

## Страница деталей календаря

После создания нажмите на календарь:

- **Левая колонка** — редактор календаря и событий
- **Правая колонка** — список связанных групп

## Добавление событий

1. Используйте редактор календаря слева
2. Установите название, дату, время события
3. События появятся на календаре

## Подключение групп

1. Нажмите **Добавить** в разделе групп
2. Выберите группу из меню
3. Выберите все события или конкретные
4. Нажмите **Сохранить**

## Включение регистрации на события

1. Откройте или создайте событие
2. Переключите **Регистрация**
3. Настройте параметры:
   - **Вместимость**
   - **Регистрация открывается**
   - **Регистрация закрывается**
   - **Теги**
4. Сохраните

Члены увидят кнопку **Зарегистрироваться на это событие**.

## Управление регистрациями

1. Откройте **Регистрации** в B1 Admin
2. Вы увидите все события с регистрацией
3. Нажмите для просмотра списка регистраций
4. Отменяйте, удаляйте или экспортируйте
