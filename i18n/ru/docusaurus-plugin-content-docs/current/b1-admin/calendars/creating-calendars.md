# Создание календарей

<div class="article-intro">

Создание календаря в B1 Admin предоставляет вам централизованное место для управления мероприятиями служения, отдела или всей церкви. Каждый календарь может включать события из одной или нескольких групп, что упрощает организацию всего.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Настройте [группы](../groups/creating-groups.md), события которых вы хотите включить в свой календарь
- Вам необходим административный доступ к разделу «Календари» в B1 Admin

</div>

## Создание нового календаря

1. Перейдите в раздел **Календари** в B1 Admin.
2. Нажмите **Добавить календарь**.
3. Введите **название** для вашего календаря (например, «Мероприятия молодёжного служения» или «Основной календарь церкви»).
4. Добавьте необязательное **описание**, чтобы ваша команда понимала, для чего предназначен этот календарь.
5. Нажмите **Создать**, чтобы сохранить новый календарь.

## Страница сведений о календаре

После создания календаря нажмите на него, чтобы открыть страницу сведений. Эта страница имеет две основные области:

- **Левая колонка** -- Редактор календаря и событий. Здесь вы можете просматривать, добавлять и управлять событиями вашего календаря.
- **Правая колонка** -- Список связанных групп. Здесь показано, какие группы имеют события, включённые в этот календарь.

## Добавление событий

1. На странице сведений о календаре используйте редактор календаря слева для добавления новых событий.
2. Укажите название мероприятия, дату, время и другие детали.
3. Добавленные события появятся в календаре и могут быть опубликованы на вашем сайте.

## Подключение групп

Группы, у которых есть события в календаре, автоматически отображаются в списке групп в правой части страницы сведений.

1. Нажмите **Добавить** в разделе групп, чтобы связать группу с вашим календарём.
2. Выберите группу из выпадающего списка.
3. Выберите, включить ли **все события** из этой группы или только **определённые события**.
4. Нажмите **Сохранить**.

:::tip
Подключение групп к вашему календарю — это мощный способ автоматического объединения событий. Когда лидер группы добавляет мероприятие в свою [группу](../groups/creating-groups.md), оно может автоматически появиться в общецерковном календаре без каких-либо дополнительных действий с вашей стороны.
:::

:::info
Если вы хотите создать единый календарь, который собирает события из множества групп вашей церкви, см. [Курированный календарь](curated-calendar) для упрощённого подхода.
:::
