# Календари

<div class="article-intro">

Раздел «Календари» в B1 Admin позволяет создавать и управлять календарями событий для вашей церкви. Вы можете настраивать календари для различных служений, объединять события из нескольких групп и публиковать календари на вашем сайте B1.church, чтобы ваша община могла их видеть.

</div>

## Просмотр ваших календарей

1. В B1 Admin перейдите в раздел **Календари**.
2. На странице «Календари» отображается список всех ваших календарей с их названием и статусом.
3. Каждый календарь также показывает связанные с ним группы.

## Что вы можете делать

- **Добавить новый календарь** -- Нажмите кнопку добавления, чтобы создать календарь для конкретного служения, типа мероприятия или цели. Пошаговые инструкции см. в разделе [Создание календарей](creating-calendars).
- **Редактировать календарь** -- Нажмите на любой календарь, чтобы открыть его и управлять событиями и связанными группами.
- **Публиковать календари** -- Календари могут отображаться на вашем сайте B1.church, чтобы посетители могли видеть предстоящие мероприятия вашей церкви.
- **Создавать курированные календари** -- Собирайте события из нескольких групп в единое представление календаря для полного обзора всего, что происходит в вашей церкви. Подробности см. в разделе [Курированный календарь](curated-calendar).

:::info
Календари тесно связаны с [Группами](../groups/creating-groups.md). Когда в группах запланированы мероприятия, эти события могут быть добавлены в ваши календари. Чтобы узнать, как связать группы с календарями, см. страницу [Создание календарей](creating-calendars).
:::
