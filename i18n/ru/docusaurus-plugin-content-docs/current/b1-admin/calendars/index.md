# Календари

<div class="article-intro">

Раздел Календари в B1 Admin позволяет создавать кураторские календари для вашей церкви, объединяя события из ваших групп. Руководители групп управляют событиями своих групп, а как администратор вы можете создавать календари, которые отображают эти события на вашем веб-сайте B1.church для вашей общины.

</div>

## Просмотр ваших календарей

1. В B1 Admin перейдите в раздел **Веб-сайт**, затем в раздел **Календари**.
2. На странице Календари перечислены все ваши календари с их названием и статусом.
3. Каждый календарь также показывает связанные с ним группы.

## Что вы можете делать

- **Добавить новый календарь** -- Нажмите кнопку добавления, чтобы создать календарь для конкретного служения, типа события или цели. См. [Создание календарей](creating-calendars) для пошаговых инструкций.
- **Редактировать календарь** -- Нажмите на любой календарь, чтобы открыть его и управлять тем, какие группы в него включены.
- **Публиковать календари** -- Календари могут отображаться на вашем веб-сайте B1.church, чтобы посетители могли видеть предстоящие события в вашей церкви.
- **Создавать кураторские календари** -- Объединяйте события из нескольких групп в один календарный вид для полного обзора всего, что происходит в вашей церкви. См. [Кураторский календарь](curated-calendar) для подробностей.

:::info
Календари тесно работают с [Группами](../groups/creating-groups.md). Когда в группах запланированы события, эти события можно включить в ваши календари. См. страницу [Создание календарей](creating-calendars), чтобы узнать, как подключать группы к календарям.
:::
