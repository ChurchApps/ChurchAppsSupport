# Курированный календарь

<div class="article-intro">

Курированный календарь объединяет события из нескольких групп в единое представление календаря. Это идеальное решение для создания общецерковного календаря, который показывает всё, что происходит во всех ваших служениях и группах, в одном месте.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Настройте [группы](../groups/creating-groups.md), события которых вы хотите включить
- Убедитесь, что лидеры групп добавили мероприятия в свои группы
- Ознакомьтесь с основами [Создания календарей](creating-calendars)

</div>

## Создание курированного календаря

1. В B1 Admin перейдите в раздел **Сайт**.
2. Нажмите **Календари** в верхней навигации.
3. Нажмите **Добавить календарь**, чтобы создать новый курированный календарь.
4. Дайте вашему календарю **название** (например, «Мероприятия церкви» или «На этой неделе в нашей церкви»).
5. Нажмите **Создать**, чтобы сохранить его.

## Добавление групп в ваш календарь

1. Нажмите на ваш новый календарь, чтобы открыть его.
2. В разделе **Группы в календаре** справа нажмите **Добавить**.
3. Выберите группу из выпадающего меню.
4. Выберите, добавить ли **все события** из этой группы или только **определённые события**.
5. Нажмите **Сохранить**.
6. Повторите этот процесс, чтобы добавить столько групп, сколько хотите включить.

:::tip
Добавьте все ваши активные группы служений, чтобы создать полный календарь, который даёт вашей общине полную картину того, что происходит в вашей церкви каждую неделю.
:::

## Публикация вашего календаря

После настройки курированного календаря вы можете поделиться им несколькими способами:

- **Встроить на ваш сайт** -- Календарь может отображаться непосредственно на страницах вашего сайта B1.church, чтобы посетители могли его просматривать.
- **Подписаться через ICS** -- Нажмите **Подписаться**, чтобы скопировать ссылку ICS. Ваша община может вставить эту ссылку в Google Calendar, Apple Calendar, Outlook или любое другое приложение-календарь для автоматической синхронизации.
- **Скачать файл ICS** -- Скачайте файл календаря, чтобы импортировать его в приложение-календарь.

:::info
Когда лидеры групп добавляют новые мероприятия в свои [группы](../groups/creating-groups.md), эти мероприятия автоматически появляются в вашем курированном календаре. Вам не нужно вручную обновлять курированный календарь каждый раз, когда добавляется новое событие.
:::
