# Настройки

<div class="article-intro">

Раздел «Настройки» — это место, где вы настраиваете учётную запись вашей церкви, брендинг и административные параметры в B1 Admin. Здесь вы можете управлять всем: от названия церкви и поддомена до разрешений пользователей и конфигурации мобильного приложения.

</div>

## Что здесь можно найти

Область настроек организована в четыре подраздела, доступных из боковой панели:

1. **Настройки** -- Настройте основную информацию о вашей церкви, брендинг и поддомен. Подробнее см. [Настройки церкви](./church-settings.md).
2. **Мобильные приложения** -- Настройте и персонализируйте вкладки навигации, которые отображаются в приложении B1 Mobile для ваших прихожан. Подробнее см. [Настройки мобильного приложения](./mobile-app.md).
3. **Администрирование сервера** -- Доступ к расширенным инструментам администрирования для управления настройками на уровне сервера.
4. **Формы** -- Создавайте и управляйте пользовательскими формами для сбора информации от вашей общины.

## Главная страница настроек

Когда вы впервые откроете страницу настроек, вы увидите **название церкви** и **поддомен**, отображаемые вверху. Заголовок содержит кнопки быстрого доступа для распространённых задач:

- **Редактировать настройки** -- Обновите название церкви, адрес, контактную информацию и брендинг.
- **Мобильные приложения** -- Перейдите непосредственно к настройке мобильного приложения.
- **Роли** -- Управляйте ролями пользователей и разрешениями для вашей команды.
- **Импорт/Экспорт** -- Переносите данные между системами с помощью инструмента импорта/экспорта.

:::tip
Начните с настройки названия церкви и брендинга в разделе **Редактировать настройки**, затем настройте [Роли](./roles-permissions.md), чтобы пригласить членов вашей команды с нужным уровнем доступа.
:::

:::info
Информацию о том, как защищены ваши данные, см. в разделе [Безопасность данных](./data-security.md).
:::
