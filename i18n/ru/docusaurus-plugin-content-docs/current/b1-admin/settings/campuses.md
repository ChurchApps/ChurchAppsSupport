# Кампусы

<div class="article-intro">

Если ваша церковь собирается в более чем одном месте, **Кампусы** позволяют вам отслеживать, какой сайт принадлежит каждому человеку и группе. После настройки кампусы появляются как опция в профилях людей, в настройке посещения и на панели управления Демография. Многокампусные церкви могут фильтровать, искать и создавать отчёты по кампусам во всём B1 Admin.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Вам нужно разрешение **Редактировать параметры церкви** для управления кампусами. Подробнее см. [Роли и разрешения](./roles-permissions.md).

</div>

## Открытие параметров кампусов

В B1 Admin перейдите в **Параметры** на левой боковой панели и выберите **Кампусы** из навигации Параметры. Вы увидите список всех настроенных кампусов с их названием, местоположением и часовым поясом.

## Добавление кампуса

1. Нажмите **Добавить кампус** (или кнопку **+**, если кампусов ещё нет).
2. Заполните детали кампуса:
   - **Название** *(обязательно)* — отображаемое название, показываемое во всём B1 Admin (например, «Главный кампус» или «Северный кампус»).
   - **Адрес** — уличный адрес кампуса (используется для информационного отображения; не совпадает с адресом вашей главной церкви в параметрах Параметры церкви).
   - **Город / Штат / Почтовый индекс** — местоположение кампуса.
   - **Часовой пояс** — часовой пояс IANA для этого кампуса (например, *America/Chicago*). Полезен, когда кампусы находятся в разных часовых поясах.
   - **Веб-сайт** — дополнительный URL для наличия собственного веб-присутствия этого кампуса.
3. Нажмите **Сохранить**.

## Редактирование кампуса

Щелкните по любой строке кампуса в списке, чтобы открыть его редактор на панели справа. Обновите поля и нажмите **Сохранить**.

## Удаление кампуса

Откройте кампус для редактирования и нажмите **Удалить**. Вам будет предложено подтверждение. Удаление кампуса не удаляет назначенных ему людей — их поле кампуса просто становится пустым.

## Назначение людей кампусу

После создания кампусов сотрудники могут назначить человека кампусу из его профиля:

1. Откройте запись человека в разделе **Люди**.
2. Нажмите **Редактировать**.
3. Выберите кампус из раскрывающегося списка **Кампус**.
4. Нажмите **Сохранить**.

Вы также можете обновить кампус в массовом порядке со страницы Люди. Выберите нескольких людей, используйте **Массовое редактирование**, и установите поле Кампус для всех сразу.

## Фильтрация по кампусу

После настройки кампусов вы можете фильтровать во всём B1 Admin по кампусам:

- **Поиск Люди** — добавьте условие Кампус в продвинутый поиск или загрузите [Сохранённый список](../people/lists.md), охватывающий кампус.
- **Демография** — панель управления [Демография](../people/demographics.md) показывает круговую диаграмму Кампус, когда по крайней мере один человек имеет назначенный кампус.
- **Настройка посещения** — каждое время служения в разделе Посещение может быть привязано к кампусу.

:::tip
Односайтовые церкви не должны настраивать кампусы. Все функции кампусов опциональны — если кампусов не существует, поля кампусов и диаграммы просто не появляются.
:::
