# Оценка риска передачи

<div class="article-intro">

Этот документ содержит оценку ChurchApps рисков, связанных с международной передачей личных данных из Великобритании/ЕЭЗ в Соединенные Штаты.

</div>

**Последний пересмотр:** апрель 2026

## 1. Детали передачи

| Элемент | Подробность |
|---|---|
| **Экспортер данных** | Церкви, использующие ChurchApps |
| **Импортер данных** | ChurchApps, работающий в США |
| **Страны назначения** | Соединенные Штаты |
| **Механизм передачи** | EU Standard Contractual Clauses (SCCs) |

## 2. Под-процессоры

| Под-процессор | Роль | Местоположение |
|---|---|---|
| **Amazon Web Services (AWS)** | Хостинг инфраструктуры | США |
| **Stripe** | Обработка платежей | США |

Данные кредитной карты передаются непосредственно в Stripe, а не через серверы ChurchApps.

## 3. Оценка риска

Риск оценивается как **низкий**.
