# Безопасность данных

<div class="article-intro">

Хотя идеально безопасной системы не существует, ChurchApps серьезно относится к безопасности данных. На этой странице объясняются меры защиты всех данных, введенных в B1 Admin и другие продукты ChurchApps.

</div>

## Ограничение сохранения чувствительных данных

ChurchApps никогда не хранит данные кредитной карты или банковского счета. Данные карты передаются непосредственно в Stripe, а не через наши серверы. Пароли никогда не хранятся — они обрабатываются через одностороннее хеширование.

Единственные чувствительные данные, которые остаются, — это список имен и контактной информации.

## Использование стандартных лучших практик

Все данные в пути и на наших серверах зашифрованы с использованием HTTPS. Серверы размещены в защищенном центре обработки данных Amazon Web Services.

## Разделение данных

Данные разделены на различные базы данных по области. Каждый из наших API (Членство, Пожертвования, Посещаемость, Обмен сообщениями, Служение и Уроки) является независимой системой с собственной базой данных.

## Ограниченный доступ

Доступ к производственным серверам строго ограничен двумя лицами, которые также являются членами правления.

## GDPR соответствие

ChurchApps поддерживает соответствие GDPR для церквей с членами в Великобритании или ЕС.
