# Роли и разрешения

<div class="article-intro">

Роли позволяют вам контролировать, к чему могут получить доступ различные пользователи в вашей учётной записи ChurchApps. Вы можете создавать пользовательские роли для сотрудников, волонтёров и других членов команды, каждая со своим уровнем доступа для обеспечения безопасности ваших данных.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Вам необходим доступ Domain Admin или роль с разрешением на управление ролями
- Подготовьте список членов команды и областей, к которым им нужен доступ
- Ознакомьтесь с доступными категориями разрешений ниже, чтобы спланировать ваши роли

</div>

## Доступ к ролям

1. Перейдите на страницу **Настройки** в левой боковой панели.
2. Нажмите кнопку **Роли** в заголовке.
3. На странице ролей отображаются все текущие роли, определённые для вашей церкви.

## Понимание страницы ролей

Страница ролей разделена на две панели:

- **Левая сторона** -- Показывает список членов, назначенных выбранной роли.
- **Правая сторона** -- Отображает настройки разрешений, которые вы можете настроить для этой роли.

Нажмите на название любой роли, чтобы просмотреть и управлять её членами и разрешениями.

## Добавление пользователей к роли

1. Выберите роль, в которую хотите добавить членов.
2. Используйте **поле поиска** на левой стороне, чтобы найти человека, которого хотите добавить.
3. Выберите человека из результатов поиска.
4. Он будет добавлен к роли немедленно.

## Удаление пользователей из роли

1. Выберите роль, содержащую пользователя, которого хотите удалить.
2. Найдите человека в списке членов на левой стороне.
3. Нажмите **кнопку удаления** рядом с его именем.

## Настройка разрешений

Каждой роли может быть предоставлен доступ к определённым областям B1 Admin. Разрешения организованы по разделам:

- **Люди** -- Доступ к справочнику прихожан и записям о людях.
- **Пожертвования** -- Доступ к записям о пожертвованиях и управлению фондами.
- **Посещаемость** -- Доступ к отслеживанию посещаемости и отчётам.
- **Контент** -- Доступ к сайту и управлению контентом.
- И дополнительные области по мере их появления.

Используйте флажки на правой стороне страницы ролей, чтобы включить или отключить доступ для каждой области.

:::warning
**Domain Admin** имеют полный доступ ко всем областям вашей учётной записи ChurchApps. Их разрешения не могут быть изменены или ограничены. Используйте эту роль только для самых доверенных администраторов.
:::

:::tip
Создавайте конкретные роли, такие как «Казначей» с доступом только к **Пожертвованиям**, или «Волонтёр на регистрации» с доступом только к **Посещаемости**. Это следует принципу наименьших привилегий и обеспечивает безопасность ваших данных. Подробнее о том, как ChurchApps защищает вашу информацию, см. в разделе [Безопасность данных](./data-security.md).
:::
