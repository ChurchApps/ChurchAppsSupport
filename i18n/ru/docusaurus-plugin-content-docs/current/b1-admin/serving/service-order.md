# Порядок богослужения

<div class="article-intro">

Порядок богослужения определяет последовательность элементов вашего богослужения. Используйте его для планирования всего — от открывающей песни до заключительной молитвы, чтобы вся команда заранее знала ход богослужения.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- [Создайте план богослужения](./plans.md) для даты, для которой хотите составить порядок
- По желанию добавьте [песни](./songs.md) в вашу библиотеку, чтобы можно было связать их напрямую с порядком богослужения

</div>

## Доступ к порядку богослужения

1. Перейдите в раздел **Служение** и выберите ваше направление служения.
2. Нажмите на **тип плана**, затем откройте конкретный **план**.
3. Нажмите вкладку **Порядок богослужения** на странице деталей плана.

## Составление порядка богослужения

Порядок богослужения — это список элементов, расположенных в последовательности их выполнения во время богослужения. Типичные элементы включают:

- Песни прославления
- Молитвы
- Чтение Писания
- Объявления
- Проповедь или послание
- Сбор пожертвований
- Заключение

## Добавление элементов в порядок богослужения

1. На вкладке **Порядок богослужения** нажмите **Добавить элемент**.
2. Выберите тип элемента, который хотите добавить (например, песню из вашей библиотеки, пользовательский элемент или заголовок).
3. Введите детали элемента, такие как название и примечания.
4. Нажмите **Сохранить**.

:::tip
Вы можете добавлять песни напрямую из вашей библиотеки [Песен](./songs.md). Это связывает детали песни, включая тексты и аранжировки, с вашим планом богослужения, чтобы у группы прославления было всё необходимое.
:::

## Изменение порядка элементов

Перетаскивайте элементы для изменения порядка. Последовательность на экране — это порядок, которому ваша команда будет следовать во время богослужения.

## Редактирование и удаление элементов

- Нажмите на любой элемент в порядке богослужения, чтобы **отредактировать** его детали.
- Используйте опцию **удалить**, чтобы убрать элемент из порядка.

:::info
Порядок богослужения привязан к конкретному плану. Изменения, которые вы вносите здесь, влияют только на выбранный план и не затрагивают другие планы или вашу библиотеку песен.
:::

## Распространение порядка богослужения

Когда порядок богослужения готов, вы можете распечатать полный план (включая порядок богослужения) со страницы деталей плана. Это даёт вашей команде полное описание хода богослужения.

## Следующие шаги

- Управляйте репертуаром прославления на странице [Песни](./songs.md)
- Назначайте волонтёров на богослужение на вкладке «Назначения» в [Планах](./plans.md)
- Создавайте [Задачи](./tasks.md) для любых последующих действий, связанных с богослужением
