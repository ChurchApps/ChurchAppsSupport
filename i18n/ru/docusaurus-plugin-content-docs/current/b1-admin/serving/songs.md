# Песни

<div class="article-intro">

Страница «Песни» управляет библиотекой песен прославления. Храните все детали песен, тексты и аранжировки в одном месте, чтобы ваша группа прославления могла легко подготовиться к предстоящим богослужениям и поддерживать постоянный репертуар.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Перейдите в раздел **Служение** в B1 Admin, чтобы получить доступ к странице «Песни»
- По желанию настройте [планы богослужений](./plans.md), чтобы можно было связывать песни напрямую с [порядком богослужения](./service-order.md)

</div>

## Просмотр библиотеки песен

Когда вы открываете страницу **Песни**, песни отображаются в виде карточек. Каждая карточка показывает миниатюру, название, исполнителя и продолжительность. Прокручивайте библиотеку или используйте поиск, чтобы найти нужное.

## Поиск песен

1. Нажмите кнопку **Поиск** на странице «Песни».
2. Введите **название** песни или имя **исполнителя**.
3. Результаты будут фильтроваться по мере ввода, показывая подходящие песни из вашей библиотеки.

## Добавление песни

1. Нажмите **Добавить песню** на странице «Песни».
2. Поищите во внешней базе данных песен по названию или исполнителю.
3. Выберите песню, которую хотите добавить.
4. Песня будет добавлена в вашу библиотеку с предварительно заполненными основными данными.

:::tip
После добавления песни нажмите на неё, чтобы просмотреть и обновить детали, добавить аранжировки или прикрепить внешние ссылки, такие как видео на YouTube или таблицы аккордов.
:::

## Просмотр и редактирование песни

Нажмите на любую карточку песни, чтобы открыть страницу деталей. Здесь вы можете:

- **Редактировать детали песни** — обновить название, исполнителя, продолжительность и другие метаданные.
- **Управлять аранжировками** — каждая песня может иметь несколько аранжировок (например, в разных тональностях или версиях). Добавляйте, редактируйте или удаляйте аранжировки по необходимости.
- **Добавить тексты** — введите или обновите тексты для каждой аранжировки.
- **Установить тональность** — укажите музыкальную тональность для каждой аранжировки, чтобы команда знала, как подготовиться.
- **Добавить внешние ссылки** — прикрепите ссылки на ресурсы, такие как видео YouTube, таблицы аккордов или ноты, размещённые в другом месте.

## Аранжировки

Одна песня может иметь несколько аранжировок для различных богослужений или составов команды. Каждая аранжировка включает:

- **Тексты** для отображения или проекции
- Обозначение **тональности**
- **Внешние ссылки** на дополнительные ресурсы

:::info
Песни из вашей библиотеки можно добавлять напрямую в [Порядок богослужения](./service-order.md) в рамках плана, связывая детали песни и аранжировки с конкретным богослужением.
:::

## Следующие шаги

- Добавляйте песни в [Порядок богослужения](./service-order.md) при составлении [плана богослужения](./plans.md)
- Координируйте работу с группой прославления с помощью [Задач](./tasks.md) для подготовки и репетиций песен
