# Автоматизации

<div class="article-intro">

Автоматизации создают задачи автоматически по повторяющемуся расписанию. Вместо того чтобы вручную создавать одни и те же задачи каждую неделю или месяц, вы можете настроить автоматизацию один раз и позволить системе делать это за вас, гарантируя, что ничего не будет упущено.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Ознакомьтесь с тем, как работают [Задачи](./tasks.md) в B1 Admin
- Определите, какие повторяющиеся обязанности вы хотите автоматизировать

</div>

## Просмотр автоматизаций

Перейдите в раздел **Служение** и откройте страницу **Автоматизации**. Вы увидите список всех ваших автоматизаций, каждая из которых показывает название, статус активности и шаблон повторения.

## Создание автоматизации

1. На странице «Автоматизации» нажмите **Добавить автоматизацию**.
2. Введите **название** для автоматизации. Оно также будет использоваться в качестве названия создаваемых задач.
3. Установите **шаблон повторения** -- выберите, как часто должна создаваться задача (например, еженедельно, ежемесячно или в определённые дни).
4. Назначьте задачу **человеку или группе**, которые будут ответственными при каждом запуске.
5. Установите автоматизацию в состояние **Активна**, чтобы она начала создавать задачи по расписанию.
6. Нажмите **Сохранить**.

## Редактирование автоматизации

1. Нажмите на существующую автоматизацию в списке, чтобы открыть её.
2. Обновите название, шаблон повторения, назначение или любые другие настройки.
3. Нажмите **Сохранить**, чтобы применить изменения.

## Активация и деактивация

Вы можете контролировать, работает ли автоматизация, не удаляя её:

- **Активна** -- Автоматизация будет создавать задачи в соответствии со своим расписанием.
- **Неактивна** -- Автоматизация приостановлена и не будет создавать новые задачи, пока вы не активируете её снова.

:::tip
Установите автоматизацию в состояние **Неактивна** во время праздничных перерывов или особых сезонов, когда повторяющаяся задача не нужна. Вы можете повторно активировать её в любое время, не теряя конфигурацию.
:::

## Как это работает

Когда автоматизация запускается, она создаёт новую [задачу](./tasks.md) с настроенным названием и назначением. Задача ведёт себя так же, как и любая другая задача -- назначенные лица получают уведомления и могут управлять задачей из своей панели управления или со страницы «Задачи».

:::info
Автоматизации создают только новые задачи на будущее. Они не создают задачи задним числом для прошлых дат.
:::
