# Планирование уроков из Lessons.church

<div class="article-intro">

B1 Admin интегрируется напрямую с [Lessons.church](https://lessons.church) так, чтобы вы могли планировать учебный план для ваших классов прямо в планах служения. Это держит все в одном месте.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Установите ваши служения в разделе служения
- Иметь активный аккаунт [Lessons.church](https://lessons.church)

</div>

## Шаг 1 — Связите ваш аккаунт Lessons.church

1. Войдите в [B1 Admin](https://admin.b1.church/) и перейдите в **Служение**
2. Откройте служение, которое вы хотите подключить
3. Прокрутите вниз к разделу **Учетные записи поставщика содержимого**
4. Нажмите **Связать новый поставщик**
5. Выберите **Lessons.church** из списка
6. Появится экран авторизации устройства с кодом
7. Перейдите на [lessons.church](https://lessons.church), войдите и введите код
8. После утверждения вы увидите **"Учетная запись связана"**

## Шаг 2 — Планирование урока

1. В B1 Admin перейдите в **Служение → Планы**
2. Выберите вкладку служения и нажмите **Добавить тип плана**
3. Нажмите на созданный тип плана и нажмите **Планировать урок**
4. Выберите **дату** для урока
5. Нажмите **Выберите урок** — откроется диалог браузера содержимого
6. Просмотрите содержимое:
   - Выберите **Программу**
   - Выберите **Исследование**
   - Выберите конкретный **Урок**
   - Выберите **Место** (версия по возрастной группе)
7. Нажмите **Связать урок**
8. Выберите опцию копирования для добровольцев
