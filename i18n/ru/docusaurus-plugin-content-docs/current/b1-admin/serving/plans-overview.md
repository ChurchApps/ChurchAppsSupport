# Обзор планов

<div class="article-intro">

Обзор планов дает вам полный обзор всех назначений добровольцев на несколько дат служения одновременно. Вместо открытия каждого плана отдельно, вы можете видеть, кто служит в каждой позиции во всех предстоящих неделях в одной сетке.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Создайте по крайней мере одно служение и тип плана
- Создайте [планы служения](./plans.md) с датами и назначениями
- Убедитесь, что добровольцы добавлены в справочник людей

</div>

## Доступ к обзору

1. Откройте **Служение** из главного меню
2. Выберите **вкладку служения** вверху
3. Нажмите на **тип плана** для открытия списка планов
4. Нажмите кнопку **Обзор** вверху

## Чтение сетки обзора

Обзор отображает сетку где:

- **Строки** — каждая позиция (например, "Музыка: Гитара")
- **Столбцы** — предстоящие даты служения
- **Ячейки** — имя назначенного добровольца

Позиции, выделенные **красным**, — незаполненные. Это облегчает выявление пробелов в расписании.

## Фильтрация обзора

Используйте элементы управления фильтром вверху:

- **Начальная/конечная дата** — по умолчанию показывает 12 недель
- **Служение** — переключитесь на другое служение
- **Тип плана** — отфильтруйте конкретный тип плана

## Экспорт в CSV

Нажмите **Экспорт CSV** для загрузки сетки в виде электронной таблицы.
