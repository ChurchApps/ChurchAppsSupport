# Обзор планов

<div class="article-intro">

Обзор планов дает вам полное представление о всех назначениях волонтёров на несколько дат служб сразу. Вместо открытия каждого плана отдельно, вы можете увидеть, кто служит в каждой позиции на предстоящих неделях в одной таблице — и быстро найти пробелы, которые ещё нужно заполнить.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Создайте по крайней мере одно служение и тип плана в области служения
- Создайте [планы служб](./plans.md) с датами и назначениями волонтёров
- Убедитесь, что ваши волонтёры добавлены в ваш [справочник контактов](../people/adding-people.md)

</div>

## Доступ к обзору

1. Перейдите на **Serving** (Служение) из главного меню в B1 Admin.
2. Выберите **ministry tab** (вкладку служения) в верхней части страницы.
3. Нажмите на **plan type** (тип плана) для открытия его списка планов.
4. Нажмите кнопку **Overview** (Обзор) рядом с верхней частью страницы.

## Чтение таблицы обзора

Обзор отображает таблицу, в которой:

- **Rows** (Строки) представляют каждую позицию (например, «Music: Guitar» (Музыка: гитара), «Tech: Projection» (Техника: проектор)), сгруппированные по категориям
- **Columns** (Столбцы) представляют предстоящие даты служб (например, «Apr 14» (14 апреля), «Apr 21» (21 апреля))
- **Cells** (Ячейки) показывают имя волонтёра, назначенного на эту позицию на эту дату

Ячейки, выделенные **красным**, не заполнены — волонтёр ещё не назначен. Это облегчает видение пробелов в штате с первого взгляда без открытия каждого плана отдельно.

:::tip
Имена волонтёров показаны в сокращённом формате (имя и инициал фамилии, например «John D.» (Джон Д.)), чтобы таблица оставалась компактной, когда у вас много позиций.
:::

## Назначение волонтёров прямо из обзора

Вам не нужно открывать отдельные планы для заполнения пустых мест. Нажмите на любую ячейку в таблице, чтобы открыть панель назначений для этой позиции и даты. Оттуда вы можете:

- Выбрать человека из вашей команды для назначения на позицию
- Заменить или удалить существующее назначение
- Сохранить изменение, не покидая обзор

Это позволяет укомплектовать полное расписание за один проход — работайте на неделях и позициях, не переходя из одного плана в другой.

## Фильтрация обзора

Вы можете отрегулировать то, что показывает обзор, используя элементы управления фильтром в верхней части:

- **Start Date / End Date** (Дата начала / Дата окончания) — по умолчанию обзор показывает 12 недель в будущее. Введите пользовательские даты, чтобы расширить или сузить диапазон.
- **Ministry** (Служение) — переключитесь на другое служение, не покидая обзор.
- **Plan Type** (Тип плана) — отфильтруйте по конкретному типу плана в выбранном служении.
- **Gaps Only** (Только пробелы) — включите это, чтобы скрыть строки, где каждая дата уже заполнена, чтобы вы могли сосредоточиться только на позициях, которым ещё нужен волонтёр.

Нажмите **Filter** (Фильтр) после внесения изменений, чтобы обновить таблицу.

## Экспорт в CSV

Нажмите **Export CSV** (Экспортировать CSV), чтобы загрузить текущую таблицу в виде электронной таблицы. Экспорт включает все позиции и назначения волонтёров для отфильтрованного диапазона дат, что облегчает совместное использование с лидерами служений или печать для планирования встреч.

:::info
Экспорт CSV отражает текущие применённые фильтры — только даты и тип плана, показанные в таблице, включены в загрузку.
:::
