# Обзор планов

<div class="article-intro">

Обзор планов дает вам полный обзор всех назначений волонтеров в несколько дат служения одновременно. Вместо открытия каждого плана отдельно, вы можете видеть, кто служит в каждой позиции во всех предстоящих неделях в одной сетке -- и быстро заметить любые пробелы, которые все еще нужно заполнить.

</div>

<div class="prereqs">
<h4>Перед тем как начать</h4>

- Создайте по крайней мере одно служение и тип плана в области служения
- Создайте [планы служений](./plans.md) с датами и назначениями волонтеров
- Убедитесь, что ваши волонтеры добавлены в ваш [справочник людей](../people/adding-people.md)

</div>

## Доступ к обзору

1. Перейдите на **Служение** из главного меню в B1 Admin.
2. Выберите **вкладку служения** в верхней части страницы.
3. Нажмите на **тип плана**, чтобы открыть список его планов.
4. Нажмите на кнопку **Обзор** рядом с верхней частью страницы.

## Чтение сетки обзора

Обзор отображает сетку, где:

- **Строки** представляют каждую позицию (например, "Музыка: гитара", "Техника: проекция"), сгруппированные по категориям
- **Столбцы** представляют предстоящие даты служений (например, "14 апреля", "21 апреля")
- **Ячейки** показывают имя волонтера, назначенного на эту позицию в эту дату

Ячейки, выделенные **красным**, не заполнены -- волонтер еще не назначен. Это облегчает видение пробелов в укомплектовании с первого взгляда без открытия каждого плана отдельно.

:::tip
Имена волонтеров отображаются в сокращенном формате (имя и инициал фамилии, например, "Иван К."), чтобы сохранить сетку компактной, когда у вас много позиций.
:::

## Назначение волонтеров непосредственно из обзора

Вам не нужно открывать отдельные планы, чтобы заполнить пустые слоты. Нажмите на любую ячейку в сетке, чтобы открыть панель назначения для этой позиции и даты. Оттуда вы можете:

- Выбрать человека из вашей команды для назначения на позицию
- Нажмите **Удалить** рядом с уже назначенным, чтобы удалить их из этого слота
- Сохраните изменение, не покидая обзор

Это дает возможность укомплектовать весь график за один проход -- работайте по неделям и позициям, не переходя в отдельные планы.

## Автоматическое планирование из обзора

Нажмите **Автоплан**, чтобы B1 заполнил каждый открытый, незаполненный слот в текущей сетке одновременно. Для каждого плана в представлении он вытягивает кандидатов из группы, привязанной к каждой позиции, и автоматически заполняет пустые слоты, сообщая, сколько видимых планов он смог заполнить. Планы заполняются один за другим, поэтому один и тот же волонтер не дважды бронируется на двух датах в одном проходе.

:::info
Автоплан заполняет только уже пустые слоты -- никогда не заменяет существующее назначение.
:::

## Отправка писем всем запланированным

Нажмите **Электронная почта волонтеров**, чтобы отправить уведомление каждому волонтеру, назначенному где-нибудь в отфильтрованном диапазоне дат и служении, в одном действии, вместо отправки письма план за планом. B1 сообщает, сколько писем было отправлено и сколько не удалось.

## Выделение расписания волонтера

Используйте раскрывающийся список **Выделить**, чтобы выбрать человека из вашей команды -- каждая ячейка, где они назначены, выделяется в сетке, поэтому вы можете видеть, где они уже служат перед добавлением их в другой слот. Выберите **Все**, чтобы отключить выделение.

## Фильтрация обзора

Вы можете отрегулировать то, что обзор показывает, используя элементы управления фильтрацией в верхней части:

- **Дата начала / Дата окончания** -- по умолчанию обзор показывает 12 недель в будущее. Введите пользовательские даты, чтобы расширить или сузить диапазон.
- **Служение** -- переключитесь на другое служение, не покидая обзор.
- **Тип плана** -- фильтр к определенному типу плана в выбранном служении.
- **Только незаполненные** -- переключите это на, чтобы скрыть строки, где каждая дата уже заполнена, чтобы вы могли сосредоточиться только на позициях, которые все еще нуждаются в волонтере.

Нажмите **Фильтр** после внесения изменений, чтобы обновить сетку.

## Экспорт в CSV

Нажмите **Экспорт CSV**, чтобы загрузить текущую сетку в виде электронной таблицы. Экспорт включает все позиции и назначения волонтеров для отфильтрованного диапазона дат, облегчая обмен с лидерами служения или печать для встреч планирования.

:::info
Экспорт CSV отражает все применяемые в данный момент фильтры -- только даты и тип плана, показанные в сетке, включены в загрузку.
:::
