# Планы служения

<div class="article-intro">

Планы служения организуют то, кто служит и когда. Каждый план привязан к определенной дате и служению, что облегчает координацию добровольцев неделя за неделей.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Установите ваши служения и команды в разделе служения
- Убедитесь, что добровольцы добавлены в справочник людей и назначены командам

</div>

## Доступ к планам

1. Откройте **Служение** из главного меню
2. Выберите **вкладку служения** вверху
3. Нажмите на **тип плана** для просмотра списка планов
4. Нажмите на конкретный план для его открытия

## Создание плана

1. Из вида типа плана нажмите **Новый план**
2. Дайте плану название или используйте дату
3. Выберите **дату** для служения
4. Если хотите скопировать из предыдущего плана, выберите опцию копирования
5. Сохраните план

## Страница деталей плана

Когда вы откроете план, вы увидите две вкладки:

- **Назначения** — управляйте назначениями добровольцев
- **[Порядок служения](./service-order.md)** — создавайте порядок служения

## Назначение членов команды

1. Откройте план и перейдите на вкладку **Назначения**
2. Нажмите **добавить позицию**
3. Нажмите **Нужны люди** и выберите добровольцев
4. Нажмите **Добавить** для добавления членов команды
5. Назначенные члены появятся с их статусом
6. Нажмите "уведомить добровольцев" для уведомления

## Печать планов

1. Откройте план
2. Откройте вкладку порядка служения
3. Используйте **Печать** для создания печатной версии
