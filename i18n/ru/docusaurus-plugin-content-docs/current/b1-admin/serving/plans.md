# Планы служения

<div class="article-intro">

Планы служения организуют, кто и когда служит. Каждый план связан с конкретной датой и служением, что упрощает координацию ваших команд волонтёров неделю за неделей и гарантирует, что каждое богослужение полностью укомплектовано.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Установите свои служения и команды в разделе Serving (Служения)
- Убедитесь, что волонтёры добавлены в [справочник людей](../people/adding-people.md) и назначены в команды

</div>

## Доступ к планам

1. Перейдите на **Serving** (Служения) из главного меню.
2. Выберите вкладку **ministry** (служения) в верхней части страницы.
3. Нажмите на **plan type** (тип плана), чтобы увидеть список планов этого типа.
4. Нажмите на конкретный план, чтобы открыть его.

:::info
Полный доступ администратора не требуется для управления планами. Любой, кто является членом служения, может перейти в Serving и создавать, редактировать и планировать планы для своего служения без необходимости иметь разрешение Plans Edit. Редакторы с ролью Plans Edit могут управлять планами для каждого служения.
:::

## Создание плана

1. На странице типа плана нажмите **New Plan** (Новый план).
2. Дайте плану название или используйте дату в качестве названия. Выберите **date** (дату) служения.
3. Если вы хотите скопировать из предыдущего плана, выберите только должности или должности и назначения. Если вы не хотите копировать, просто ничего не выбирайте. Вы также можете скопировать порядок служения из предыдущего плана.
4. Сохраните план. Теперь вы можете начать назначать членов команды и строить [порядок служения](./service-order.md).

## Страница с деталями плана

Когда вы откроете план, вы увидите две вкладки:

- **Assignments** (Назначения) — управляйте тем, какие члены команды назначены этому плану. Вы можете добавить людей из существующих команд и увидеть, кто подтвердил или ещё ждёт.
- **[Service Order](./service-order.md)** (Порядок служения) — постройте порядок служения с элементами, такими как хваления, молитвы, объявления и проповедь.

## Назначение членов команды

1. Откройте план и перейдите на вкладку **Assignments** (Назначения).
2. Нажмите на **add Position** (добавить должность), чтобы развернуть его. Заполните информацию в форме добавления должности. Для названия категории добавьте любую категорию, которую вам нравится.
3. Нажмите на **People Needed** (Требуемые люди) и выберите волонтёров для заполнения этой должности.
4. Добавьте членов из списка вашей команды, нажав **Add** (Добавить).
5. Назначенные члены появятся под своей командой вместе со статусом назначения.
6. Нажмите уведомить волонтёров, чтобы уведомить их в приложении B1 или по электронной почте.

:::tip
Установите свои команды в параметрах служения перед созданием планов. Таким образом, у вас будет готовый пул волонтёров для назначения.
:::

## Связывание групп с типом плана

Ниже списка планов на странице типа плана раздел **Groups** (Группы) позволяет вам решить, какие группы смогут видеть планы для этого типа плана из своего портала для членов. Это быстрый способ отобразить предстоящие служения нужным командам без предоставления им доступа администратора.

1. На странице типа плана прокрутите вниз до раздела **Groups** (Группы).
2. Нажмите **Add Group** (Добавить группу) и выберите группу из раскрывающегося списка.
3. В столбце **Shows** (Показывает) выберите, должны ли члены этой группы видеть **Past** (Прошлые), **Future** (Будущие) или **Both** (Оба) планы для этого типа плана.
4. Повторите это, чтобы связать дополнительные группы, или нажмите значок корзины, чтобы удалить группу.

:::info
В средство выбора появляются только группы с меткой **Standard** (Стандартные). Члены связанной группы автоматически видят планы этого типа плана на вкладке [Plans](/docs/b1-church/plans/) в портале для членов B1 — ограничено окном прошлого/будущего/обоих, который вы выбрали.
:::

## Печать планов

Вы можете распечатать план для распределения между своей командой. Откройте план, откройте вкладку порядка служения и используйте опцию **Print** (Печать), чтобы создать версию для печати, которая включает назначения и порядок служения. Это полезно для раздачи на репетициях или размещения в общей зоне.

:::info
Планы организованы по служениям. Убедитесь, что вы находитесь на правильной вкладке служения перед созданием или просмотром планов.
:::
