# Подключение классной доски телевизора к вашим урокам B1 Admin

<div class="article-intro">

Вы можете подключить телевизор в классе к вашим урокам B1 Admin, используя **FreePlay** — бесплатное приложение для телевизора ChurchApps. После подключения телевизор отображает содержимое урока, которое вы запланировали в B1 Admin.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Запланируйте по крайней мере один урок в B1 Admin (см. [Планирование уроков](./scheduling-lessons.md))
- Установите **FreePlay** на телевизор или Fire Stick

</div>

## Шаг 1 — Установка FreePlay на телевизор

1. Откройте app store на вашем телевизоре
2. Найдите **FreePlay**
3. Установите приложение

Поддерживаемые платформы:
- Amazon Fire Stick / Fire TV
- Android TV / Google TV
- Apple TV

## Шаг 2 — Подключение B1.church как поставщика содержимого

1. Откройте приложение **FreePlay** на телевизоре
2. Откройте экран **Поставщики содержимого**
3. Найдите и выберите **B1.church** из списка
4. Следуйте подсказкам на экране для входа со своими учетными данными B1
5. FreePlay отображает **Подключено!**

После подключения B1.church появляется в боковой панели FreePlay.
