# Подключение телевизора в классе к урокам B1 Admin

<div class="article-intro">

Вы можете подключить телевизор в классе к вашим урокам B1 Admin с помощью **FreePlay** — бесплатного телевизионного приложения ChurchApps для церквей. После подключения телевизор отображает контент уроков, запланированных в B1 Admin, делая его доступным для учителей прямо на экране в классе.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Запланируйте хотя бы один урок в B1 Admin (см. [Планирование уроков](./scheduling-lessons.md))
- Установите **FreePlay** на телевизор или Fire Stick, который хотите использовать

</div>

## Шаг 1 — Установите FreePlay на ваш телевизор

Установите приложение **FreePlay** на дисплей в вашем классе:

- **Amazon Fire Stick / Fire TV** — найдите **FreePlay** в Amazon Appstore
- **Android TV / Google TV** — найдите **FreePlay** в Google Play Store
- **Apple TV** — найдите **FreePlay** в App Store

Подробные инструкции по установке см. в [руководстве по началу работы с FreePlay](/docs/freeplay/getting-started/).

## Шаг 2 — Подключите B1.church как поставщика контента

1. Откройте приложение **FreePlay** на телевизоре
2. Перейдите на экран **Поставщики контента** (выберите значок **Поставщики** на боковой панели)
3. Найдите и выберите **B1.church** из списка доступных поставщиков
4. Следуйте инструкциям на экране для входа с учётными данными вашей церкви B1
5. FreePlay отобразит **Подключено!** после успешного подключения

После подключения B1.church появится в боковой панели FreePlay. FreePlay загрузит контент уроков, запланированных в B1 Admin, чтобы учителя могли воспроизводить его прямо на телевизоре в классе.

:::tip
После подключения учителя могут использовать пульт телевизора для переключения между видео и слайдами в своём темпе по мере продвижения урока. Не требуется отдельная настройка или USB-накопители.
:::

:::info
Подробные инструкции по аутентификации с поставщиком контента (QR-код, форма входа и т.д.) см. в разделе [Подключение к поставщикам](/docs/freeplay/content-providers/connecting-providers).
:::
