# Вход в систему

<div class="article-intro">

Прежде чем киоск можно будет использовать для регистрации входа, вам нужно войти с учетной записью ChurchApps, которая имеет доступ к вашей церкви.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Установите B1 Church Checkin на ваш Android планшет
- Имейте готовыми адрес электронной почты и пароль вашей учетной записи ChurchApps

</div>

## Первоначальный вход в систему

1. Откройте приложение B1 Church Checkin
2. Введите **адрес электронной почты** в первое поле
3. Введите **пароль** во второе поле
4. Нажмите кнопку **Вход в систему**

:::info
Это тот же адрес электронной почты и пароль, который вы используете для B1 Admin.
:::

## Выбор вашей церкви

После успешного входа в систему приложение загружает список церквей, связанных с вашей учетной записью.

- Если ваша учетная запись связана с **одной церковью**, вы все еще увидите экран выбора
- Если ваша учетная запись связана с **несколькими церквями**, выберите церковь, которую этот киоск должен обслуживать

## Автоматический вход при возврате

Приложение сохраняет ваш адрес электронной почты, пароль и выбранную церковь после первого успешного входа.

:::tip
Так как киоск автоматически входит в систему, вам нужно вводить учетные данные только один раз при начальной настройке.
:::

## Параметры администратора

Параметры администратора специально скрыты, чтобы члены церкви не могли случайно изменить конфигурацию. Для доступа:

1. Нажмите **логотип церкви** в заголовке **семь раз** в быстрой последовательности (в течение двух секунд)
2. Введите PIN киоска, если он установлен
3. На экране параметров администратора вы сможете:
   - Изменить службу
   - Изменить принтер
   - Установить/изменить PIN
   - Выйти из системы
