# Вход в систему

<div class="article-intro">

Прежде чем киоск можно будет использовать для регистрации, необходимо войти в систему с учётной записью ChurchApps, имеющей доступ к вашей церкви. На этой странице описан процесс первого входа, выбор церкви, поведение автоматического входа и выход из системы.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Установите B1 Church Checkin на планшет Android (см. [Начало работы](./index.md))
- Подготовьте адрес электронной почты и пароль вашей учётной записи ChurchApps

</div>

## Первый вход

1. Откройте приложение B1 Church Checkin. После заставки вы увидите экран входа с логотипом B1 Church Checkin.
2. Введите ваш **адрес электронной почты** в первое поле.
3. Введите ваш **пароль** во второе поле.
4. Нажмите кнопку **Login**.

Приложение проверяет ваши учётные данные и выполняет вход. Если адрес электронной почты или пароль неверны, в нижней части экрана появится сообщение об ошибке.

:::info
Это тот же адрес электронной почты и пароль, которые вы используете для B1 Admin или любого другого продукта ChurchApps. Вам не нужна отдельная учётная запись для киоска регистрации.
:::

## Выбор церкви

После успешного входа приложение загружает список церквей, привязанных к вашей учётной записи. Каждая церковь отображается в виде карточки с названием церкви и логотипом (если он настроен в B1 Admin).

- Если ваша учётная запись привязана к **одной церкви**, вы всё равно увидите экран выбора. Нажмите на вашу церковь, чтобы продолжить.
- Если ваша учётная запись привязана к **нескольким церквям**, нажмите на ту церковь, которую должен обслуживать этот киоск.

После выбора церкви приложение загружает фирменный стиль и настройки внешнего вида этой церкви. Логотип вашей церкви будет отображаться в заголовке на всех последующих экранах.

## Автоматический вход при повторных посещениях

Приложение сохраняет ваш адрес электронной почты, пароль и выбранную церковь после первого успешного входа. При последующих запусках:

- Приложение автоматически входит с сохранёнными учётными данными.
- Если вы ранее выбрали церковь, этот выбор восстанавливается, и вы попадаете прямо на **экран служений**.
- Если сохранённые учётные данные больше не действительны, приложение возвращается на экран входа, чтобы вы могли войти снова.

:::tip
Поскольку киоск входит в систему автоматически, вам нужно ввести учётные данные только один раз при первоначальной настройке. После этого планшет можно включить, и он будет готов к работе.
:::

## Выход из системы

Выход из системы намеренно скрыт, чтобы члены церкви, использующие киоск, не могли случайно выйти. Для выхода:

1. Нажмите на **логотип церкви** в заголовке **семь раз** подряд (в течение двух секунд).
2. Появится диалоговое окно с запросом подтверждения выхода.
3. Нажмите **Logout** для подтверждения или **Cancel**, чтобы остаться в системе.

Это очищает сохранённые учётные данные и возвращает на экран входа.

:::warning
Выход из системы очищает все сохранённые учётные данные и выбор церкви. Вам потребуется повторно ввести адрес электронной почты и пароль для использования киоска. Выходите только тогда, когда нужно сменить учётную запись или перенастроить киоск.
:::
