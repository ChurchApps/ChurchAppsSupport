# Начало работы с B1 Church Checkin

<div class="article-intro">

Настройка B1 Church Checkin занимает всего несколько минут. Этот раздел проведёт вас через установку приложения на планшет Android, вход в систему с учётной записью ChurchApps и, при необходимости, настройку сетевого принтера для бирок.

</div>

## Что вам понадобится

- **Планшет Android** — 10-дюймовый планшет лучше всего подходит для работы в режиме киоска.
- **Учётная запись ChurchApps** — Электронная почта и пароль, которые вы используете для B1 Admin или других продуктов ChurchApps.
- **Сетевой принтер для бирок** (необязательно) — Для печати именных бирок и квитанций для родителей. Приложение может сканировать вашу локальную сеть для поиска совместимых принтеров.

## Установка

1. Откройте **Google Play Store** на вашем планшете Android.
2. Найдите **B1 Church Checkin**.
3. Нажмите **Установить** и дождитесь завершения загрузки.
4. Откройте приложение. Вы увидите заставку во время инициализации.

## Первоначальная настройка

При первом запуске приложения вам нужно пройти три шага:

1. **[Войти в систему](./logging-in)** — Введите адрес электронной почты и пароль для аутентификации.
2. **[Выбрать церковь](./logging-in)** — Если ваша учётная запись связана с несколькими церквями, выберите, какую из них обслуживает этот киоск.
3. **[Настроить принтер](./printer-setup)** — Настройте сетевой принтер для печати бирок или пропустите этот шаг, если бирки вам не нужны.

После первого входа приложение сохраняет ваши учётные данные и выбор церкви. При следующих запусках оно автоматически переходит на экран служений.

:::info
Приложение проверяет наличие обновлений при каждом запуске. Если обновление доступно, оно загружается и применяется автоматически до появления экрана входа.
:::

:::tip
Убедитесь, что администратор вашей церкви настроил служения, время служений и группы в B1 Admin до настройки киоска. Подробности см. в [руководстве по настройке посещаемости](../../b1-admin/attendance/setup.md).
:::
