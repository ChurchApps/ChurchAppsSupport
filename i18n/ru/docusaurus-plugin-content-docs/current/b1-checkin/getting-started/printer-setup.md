# Настройка принтера

<div class="article-intro">

B1 Church Checkin может печатать бирки с именами и квитанции для родителей на принтере сетевых этикеток. Настройка принтера необязательна — вы можете использовать приложение без принтера только для отслеживания посещаемости.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- [Войдите](./logging-in) в приложение B1 Church Checkin и выберите вашу церковь
- Подключите совместимый принтер сетевых этикеток к той же сети Wi-Fi, что и планшет

</div>

## Доступ к параметрам принтера

Вы можете открыть экран параметров принтера в любое время через меню **Параметры администратора**:

1. Нажмите **логотип церкви** в заголовке **семь раз** в быстрой последовательности
2. Если установлен PIN киоска, введите PIN
3. На экране параметров администратора нажмите **Изменить принтер**

## Сканирование принтеров

Когда откроется экран параметров принтера, приложение автоматически сканирует вашу локальную сеть для совместимых принтеров.

## Выбор принтера

1. Нажмите принтер, который хотите использовать
2. Выбранная карточка принтера будет выделена с галочкой
3. Для печати без физического принтера выберите **Нет принтера**

## Пробная печать

Перед сохранением выбора вы можете проверить соединение:

1. Выберите принтер из списка
2. Нажмите кнопку **Пробная печать** внизу экрана
3. Приложение отправит пробную этикетку на принтер

## Сохранение выбора

Когда вы удовлетворены выбором принтера:

1. Нажмите кнопку **Сохранить и перезагрузить** внизу экрана
2. Приложение сохранит выбор принтера и перезагрузится
