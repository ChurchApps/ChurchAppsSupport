# Обзор B1 Church Checkin

<div class="article-intro">

B1 Church Checkin — это приложение-киоск самообслуживания для планшетов Android. Оно позволяет членам церкви найти свою семью, назначить детей в группы или классы, а также распечатать именные бирки и квитанции для получения детей родителями — всё без помощи волонтёра на стойке.

</div>

## Как это работает

Разместите планшет Android в вестибюле или зоне приёма вашей церкви. Члены церкви подходят, ищут своё домохозяйство по номеру телефона или фамилии, просматривают свою семью, выбирают, в какие группы должен пойти каждый, и нажимают **Check-in**. Приложение автоматически печатает именные бирки и квитанции для родителей, а затем сбрасывается для следующей семьи.

## Основные возможности

- **Самостоятельный поиск членов** — Поиск по номеру телефона или фамилии для быстрого нахождения вашего домохозяйства.
- **Обзор домохозяйства** — Просмотр всех членов семьи с фотографиями и текущими назначениями групп одним взглядом.
- **Назначение групп** — Назначение каждого члена семьи в правильный класс или группу для выбранного времени служения.
- **Регистрация гостей** — Добавление гостя в ваше домохозяйство на месте, указав имя и фамилию.
- **Печать бирок** — Печать именных бирок для детей и квитанций для родителей с уникальными кодами безопасности.
- **Поддержка сетевых принтеров** — Обнаружение и настройка сетевых принтеров прямо из приложения.
- **Автоматический вход** — После первого входа приложение запоминает ваши учётные данные и переходит сразу на экран служений при следующих запусках.
- **Поддержка нескольких церквей** — Если ваша учётная запись связана с несколькими церквями, вы можете выбрать, в какую из них регистрироваться.
- **Фирменный стиль церкви** — Приложение отображает логотип вашей церкви в заголовке после выбора церкви.
