# Завершение регистрации

<div class="article-intro">

После того как вы проверили свою семью и выполнили необходимые назначения групп, вы готовы завершить регистрацию. Это последний шаг в рабочем процессе киоска — приложение отправляет записи о посещении, печатает ярлыки и переходит в режим ожидания для следующей семьи.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- [Просмотрите свою семью](./household-review) на экране просмотра семьи
- [Назначьте группы](./group-assignment) любым членам семьи, которые должны зарегистрироваться в определённом классе или программе
- При необходимости [добавьте гостей](./adding-guests), которые посещают вместе с вашей семьёй

</div>

## Как зарегистрироваться

1. На **экране просмотра семьи** нажмите кнопку **Регистрация** внизу экрана.
2. Приложение отправляет данные о посещении на сервер и показывает **экран успеха** с зелёной галочкой и приветственным сообщением.

Это всё, что нужно. Посещение вашей семьи записано.

## Переполненные помещения и соотношение волонтёров

Если ваша церковь настроила [ограничения безопасности](../../b1-admin/attendance/checkin-safety) для своих помещений, сервер проверяет их перед сохранением:

- Если выбранное помещение **переполнено или закрыто**, регистрация не произойдёт и приложение назовёт помещение, чтобы вы могли выбрать другое.
- Если в детском помещении **недостаточно волонтёров** для его соотношения, приложение либо показывает предупреждение, которое сотрудник может подтвердить для продолжения, либо полностью блокирует регистрацию — в зависимости от того, как ваша церковь настроила применение соотношений.

## Печать ярлыков

Если настроен сетевой принтер, приложение автоматически печатает ярлыки после регистрации:

- **Ярлыки с именами** печатаются для каждого человека, назначенного в группу, для которой включена параметр **Печать бирки с именем**. Ярлыки с именами включают имя человека, назначение его группы и информацию об аллергии/примечаниях, если они имеются.
- **Квитанции для родителей** печатаются, когда любой зарегистрировавшийся человек находится в группе, для которой включена параметр **Получение родителями**. Квитанция содержит список детей, их назначения групп и уникальный **четырёхзначный код безопасности**.

:::info
Одинаковый код безопасности появляется на бирке с именем ребёнка и на квитанции для родителей. При получении волонтёры сопоставляют коды, чтобы убедиться, что нужный взрослый забирает каждого ребёнка.
:::

Код безопасности генерируется заново при каждой регистрации и содержит только согласные буквы и цифры (гласные исключены, чтобы избежать образования неприличных слов).

:::warning
Если ярлыки не печатаются, откройте параметры администратора, нажав на **логотип церкви** семь раз, затем нажмите **Изменить принтер** для проверки подключения принтера. Инструкции по устранению неполадок см. в разделе [Настройка принтера](../getting-started/printer-setup).
:::

## Что происходит после регистрации

- Если принтер настроен, приложение печатает все ярлыки и затем автоматически возвращается на **экран поиска**, готовясь к регистрации следующей семьи.
- Если принтер не настроен, экран успеха отображается несколько секунд, а затем автоматически возвращается на **экран поиска**.

Вам не нужно ничего нажимать для возврата на экран поиска — приложение автоматически выполняет переход.

:::tip
Приложение полностью сбрасывается после каждой регистрации, поэтому нет риска, что одна семья увидит информацию другой семьи.
:::

## Что записывается

Когда вы нажимаете **Регистрация**, приложение отправляет на сервер следующие данные для каждого члена семьи, которому назначена группа:

- **Человека**, который регистрируется
- **Служение**, которое он посещает
- **Время служения** и **группу**, которые ему назначены

Эти данные появляются в B1 Admin в разделе посещаемости, где администраторы вашей церкви могут просматривать и управлять записями о посещении. Подробности см. в [руководстве администратора регистрации](../../b1-admin/attendance/check-in.md).
