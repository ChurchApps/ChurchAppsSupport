# Завершение регистрации входа

<div class="article-intro">

После проверки домохозяйства и назначения групп вы готовы завершить регистрацию входа. Это последний шаг в рабочем процессе киоска.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Проверьте ваше домохозяйство на экране проверки домохозяйства
- Назначьте группы членам семьи
- Добавьте гостей, если необходимо

</div>

## Как зарегистрироваться

1. На **экране проверки домохозяйства** нажмите кнопку **Регистрация входа** внизу экрана
2. Приложение отправляет данные о посещении на сервер и показывает **экран успеха** с зеленой галочкой

## Печать этикеток

Если настроен принтер сети, приложение автоматически печатает этикетки:

- **Бирки с именами** печатаются для каждого человека
- **Квитанции для родителей** печатаются с детьми и уникальным 4-символьным кодом безопасности

## После регистрации входа

Если принтер настроен, приложение печатает все этикетки и возвращается к экрану поиска. Если принтер не настроен, экран успеха отображается несколько секунд, а затем возвращается к экрану поиска.

## Что записывается

Когда вы нажимаете **Регистрация входа**, приложение отправляет следующее на сервер для каждого члена домохозяйства с назначенной группой:

- Человека, который регистрируется
- Сервис, который они посещают
- Время служения и группу
