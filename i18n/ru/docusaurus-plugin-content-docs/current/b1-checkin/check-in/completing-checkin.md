# Завершение регистрации

<div class="article-intro">

После того как вы просмотрели своё домохозяйство и сделали необходимые назначения групп, вы готовы завершить регистрацию. Это последний шаг в процессе работы киоска — приложение отправляет данные о посещаемости, печатает бирки и сбрасывается для следующей семьи.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- [Просмотрите домохозяйство](./household-review) на экране обзора домохозяйства
- [Назначьте группы](./group-assignment) для членов семьи, которым нужно зарегистрироваться в определённом классе или программе
- При необходимости [добавьте гостей](./adding-guests), которые посещают вашу семью

</div>

## Как зарегистрироваться

1. На **экране обзора домохозяйства** нажмите кнопку **Check-in** в нижней части экрана.
2. Приложение отправляет данные о посещаемости на сервер и показывает **экран успеха** с зелёной галочкой и приветственным сообщением.

Вот и всё. Посещаемость вашей семьи зарегистрирована.

## Печать бирок

Если настроен сетевой принтер, приложение автоматически печатает бирки после регистрации:

- **Именные бирки** печатаются для каждого человека, назначенного в группу с включённой настройкой **Печать бирки**. Именные бирки содержат имя человека, назначение группы и информацию об аллергиях/заметках, если они есть в профиле.
- **Квитанции для родителей** печатаются, когда зарегистрированный человек находится в группе с включённой настройкой **Получение родителями**. Квитанция содержит имена детей, их назначения групп и уникальный **4-символьный код безопасности**.

:::info
Один и тот же код безопасности указывается как на именной бирке ребёнка, так и на квитанции для родителей. При получении ребёнка волонтёры сверяют коды, чтобы убедиться, что правильный взрослый забирает каждого ребёнка.
:::

Код безопасности генерируется заново при каждой регистрации и использует только согласные и цифры (гласные исключены, чтобы избежать формирования нежелательных слов).

:::warning
Если бирки не печатаются, проверьте строку состояния принтера в верхней части экрана. Нажмите на неё, чтобы открыть настройки принтера и проверить подключение. Подробнее см. [Настройка принтера](../getting-started/printer-setup).
:::

## Что происходит после регистрации

- Если принтер настроен, приложение печатает все бирки, а затем автоматически возвращается на **экран поиска**, готовое для следующей семьи.
- Если принтер не настроен, экран успеха отображается несколько секунд, а затем автоматически возвращается на **экран поиска**.

Вам не нужно ничего нажимать, чтобы вернуться на экран поиска — приложение выполняет переход автоматически.

:::tip
Приложение полностью сбрасывается после каждой регистрации, поэтому нет риска, что одна семья увидит информацию другой семьи.
:::

## Что фиксируется

Когда вы нажимаете **Check-in**, приложение отправляет на сервер следующие данные для каждого члена домохозяйства с назначением группы:

- **Человек**, который регистрируется
- **Служение**, которое он посещает
- **Время служения** и **группа**, в которую он назначен

Эти данные отображаются в B1 Admin в разделе «Посещаемость», где администраторы вашей церкви могут просматривать и управлять записями посещаемости. Подробности см. в [руководстве по администрированию регистрации](../../b1-admin/attendance/check-in.md).
