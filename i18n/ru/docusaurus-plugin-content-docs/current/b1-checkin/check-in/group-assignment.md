# Назначение групп

<div class="article-intro">

Когда вы нажимаете кнопку времени служения на экране обзора домохозяйства, приложение открывает экран выбора группы. Здесь вы выбираете, в какую группу или класс должен пойти член семьи на конкретное время служения.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- [Просмотрите домохозяйство](./household-review) на экране обзора домохозяйства
- Раскройте карточку члена, которого хотите назначить, и нажмите кнопку времени служения
- Убедитесь, что администратор вашей церкви [создал группы](../../b1-admin/groups/creating-groups.md) и связал их с временами служений в B1 Admin

</div>

## Как организованы группы

Группы отображаются по **категориям**. Каждая категория представлена карточкой, показывающей:

- **Название категории** (например, «Дети», «Молодёжь», «Взрослые»)
- **Количество групп** в этой категории

Нажмите на карточку категории, чтобы раскрыть её и увидеть отдельные группы внутри. Нажмите снова, чтобы свернуть.

## Выбор группы

1. Нажмите на **категорию**, чтобы раскрыть её.
2. Просмотрите список групп в этой категории. Каждая группа показывает своё название с значком выбора справа.
3. Нажмите на **название группы**, которую хотите назначить.

Приложение сохраняет ваш выбор и возвращает на [экран обзора домохозяйства](./household-review). Кнопка времени служения для этого члена теперь показывает название выбранной группы.

## Выбор «Нет»

Если вы решите, что члену семьи не нужно назначение в группу на это время служения, нажмите кнопку **Нет** в нижней части экрана. Это очищает любое существующее назначение группы для этого времени служения и возвращает на экран обзора домохозяйства.

:::tip
Используйте вариант **Нет** для взрослых, которые посещают основное служение и которым не нужно регистрироваться в определённой группе или классе.
:::

## Изменение назначения группы

Чтобы изменить уже сделанное назначение группы:

1. Вернитесь на [экран обзора домохозяйства](./household-review).
2. Раскройте карточку члена, чью группу хотите изменить.
3. Нажмите кнопку времени служения (которая теперь показывает название текущей назначенной группы).
4. Выберите другую группу или нажмите **Нет**.

:::info
Группы и категории настраиваются администратором вашей церкви в B1 Admin. Если группа отсутствует или находится в неправильной категории, попросите администратора обновить [настройки группы](../../b1-admin/groups/creating-groups.md).
:::
