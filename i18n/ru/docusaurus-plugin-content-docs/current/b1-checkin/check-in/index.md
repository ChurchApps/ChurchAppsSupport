# Процесс регистрации

<div class="article-intro">

Процесс регистрации в B1 Church Checkin следует простому пошаговому алгоритму. Член церкви подходит к киоску, находит свою семью, подтверждает назначения групп и регистрируется — всё менее чем за минуту. Этот раздел описывает каждый шаг процесса регистрации на киоске.

</div>

## Обзор процесса регистрации

1. **[Выбрать служение](./selecting-service)** — Выберите, на какое служение вы идёте (например, Воскресное утро, Воскресный вечер, Среда).
2. **[Найти свою семью](./looking-up-members)** — Поиск по номеру телефона или фамилии, чтобы найти себя в системе.
3. **[Просмотреть домохозяйство](./household-review)** — Просмотр всех членов семьи и их текущих назначений групп.
4. **[Назначить группы](./group-assignment)** — Выбор класса или группы для каждого члена семьи на каждое время служения.
5. **[Добавить гостей](./adding-guests)** (необязательно) — Добавление посетителя в ваше домохозяйство, если он пришёл с вами.
6. **[Завершить регистрацию](./completing-checkin)** — Нажмите кнопку регистрации, чтобы отправить данные о посещаемости и распечатать бирки.

После завершения регистрации приложение показывает экран успеха и автоматически возвращается на экран поиска, готовое к следующей семье.

:::tip
Весь процесс разработан так, чтобы члены церкви могли использовать его самостоятельно. Волонтёрам не нужно управлять киоском, хотя наличие кого-то поблизости для помощи новым посетителям — всегда хорошая идея.
:::

## Перед началом работы

Убедитесь, что в B1 Admin настроено следующее перед использованием киоска регистрации:

- **Служения и время служений** настроены в разделе «Посещаемость». См. [руководство по настройке посещаемости](../../b1-admin/attendance/setup.md).
- **Группы** созданы и привязаны к соответствующим временам служений. См. [создание групп](../../b1-admin/groups/creating-groups.md).
- **Настройки групп** настроены для печати бирок: включите **Печать бирки** для групп, которым нужны именные бирки, и **Получение родителями** для детских групп, которым нужны квитанции для родителей.
- **Люди** добавлены в систему с привязкой к домохозяйствам. См. [добавление людей](../../b1-admin/people/adding-people.md).

:::info
Если вы ещё не настроили сам киоск, см. [Начало работы](../getting-started/) для инструкций по установке, входу и настройке принтера.
:::
