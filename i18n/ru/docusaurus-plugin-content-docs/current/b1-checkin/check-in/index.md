# Рабочий процесс регистрации

<div class="article-intro">

Процесс регистрации в B1 Church Checkin следует простому пошаговому потоку. Член церкви подходит к киоску, находит свою семью, подтверждает назначения групп и регистрируется — всё это занимает менее минуты. Этот раздел охватывает каждый шаг опыта регистрации на киоске.

</div>

## Поток регистрации на первый взгляд

1. **[Выберите служение](./selecting-service)** — выберите, какое служение вы посещаете (например, воскресенье утром, воскресенье вечером, в середине недели).
2. **[Найдите свою семью](./looking-up-members)** — найдите себя в системе по номеру телефона или фамилии.
3. **[Просмотрите свою семью](./household-review)** — посмотрите всех членов семьи и их текущие назначения групп.
4. **[Назначьте группы](./group-assignment)** — выберите, в какой класс или группу должен посещать каждый член семьи в течение каждого времени служения.
5. **[Добавьте гостей](./adding-guests)** (опционально) — добавьте посетителя в вашу семью, если он присоединяется к вам.
6. **[Завершите регистрацию](./completing-checkin)** — нажмите кнопку регистрации, чтобы отправить данные о посещении и напечатать ярлыки.
7. **[Оформить выезд](./checking-out)** (контролируемые станции) — в момент получения убедитесь в коде безопасности и человеке, берущем ребёнка, с использованием списков доверенных получателей, страницы вызова родителей и повторной печати ярлыков.

После завершения регистрации приложение отображает экран успеха и автоматически возвращается на экран поиска, готовясь к регистрации следующей семьи.

:::tip
Весь поток разработан для самостоятельного использования членами церкви. Волонтёрам не нужно управлять киоском, хотя иметь рядом человека, который поможет первому посетителю, всегда хорошая идея.
:::

## Перед началом

Убедитесь, что в B1 Admin настроено следующее перед использованием киоска регистрации:

- **Служения и время служений** установлены в параметрах посещаемости. См. [руководство по настройке посещаемости](../../b1-admin/attendance/setup.md).
- **Группы** созданы и привязаны к соответствующему времени служений. См. [создание групп](../../b1-admin/groups/creating-groups.md).
- **Параметры группы** настроены для печати ярлыков: включите **Печать бирки с именем** для групп, которые требуют ярлыки с именами, и **Получение родителями** для детских групп, которые требуют квитанции для получения.
- **Люди** добавлены в систему с ассоциациями семей. См. [добавление людей](../../b1-admin/people/adding-people.md).

:::info
Если вы ещё не настроили сам киоск, см. [Начало работы](../getting-started/) для инструкций по установке, входу и настройке принтера.
:::
