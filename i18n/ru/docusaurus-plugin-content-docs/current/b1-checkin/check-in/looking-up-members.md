# Поиск членов

<div class="article-intro">

После выбора служения вы попадаете на экран поиска членов. Здесь вы ищете свою семью для регистрации. Приложение поддерживает поиск по номеру телефона или по имени.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- [Выберите служение](./selecting-service) на экране служений
- Убедитесь, что ваше домохозяйство было [добавлено в систему](../../b1-admin/people/adding-people.md) администратором церкви

</div>

## Режимы поиска

Экран поиска предлагает два режима. Вы можете переключаться между ними, нажимая на переключатель в верхней части экрана:

- **Телефон** — Поиск по номеру телефона. Это режим по умолчанию.
- **Имя** — Поиск по имени.

## Поиск по номеру телефона

1. Убедитесь, что **Телефон** — активный режим поиска (он выделен вверху).
2. Нажмите на поле поиска и введите не менее **4 цифр** вашего номера телефона с помощью экранной клавиатуры.
3. Нажмите кнопку **Поиск** или клавишу поиска на клавиатуре.

Приложение ищет членов, чей номер телефона совпадает с введёнными цифрами. Если вы вводите более 4 цифр, используется полный номер. Если вводите ровно 4, поиск выполняется по последним 4 цифрам номеров телефонов в базе данных.

## Поиск по имени

1. Нажмите переключатель **Имя** в верхней части экрана, чтобы переключиться в режим поиска по имени.
2. Введите не менее **2 символов** имени человека в поле поиска.
3. Нажмите кнопку **Поиск**.

Приложение ищет членов, чьё имя совпадает с тем, что вы ввели.

## Выбор человека

Результаты поиска отображаются в виде карточек под полем поиска. Каждая карточка показывает **имя** и **фотографию** человека (если она есть в профиле).

Нажмите на карточку человека, чтобы выбрать его. Приложение загружает всех членов домохозяйства этого человека и переводит вас на [экран обзора домохозяйства](./household-review).

:::tip
Поиск по последним 4 цифрам номера телефона — самый быстрый способ найти свою семью. Поощряйте членов использовать этот метод для более быстрой регистрации.
:::

:::warning
Если результаты не найдены, приложение отображает соответствующее сообщение. Перепроверьте введённые цифры или имя, или попробуйте другой режим поиска. Если вы пришли впервые, обратитесь к волонтёру за помощью в [добавлении в систему](../../b1-admin/people/adding-people.md).
:::

## Следующий шаг

После выбора человека вы перейдёте к [обзору домохозяйства](./household-review) и управлению назначениями групп.
