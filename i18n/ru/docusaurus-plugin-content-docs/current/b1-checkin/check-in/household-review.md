# Обзор домохозяйства

<div class="article-intro">

После выбора человека из результатов поиска приложение загружает всё ваше домохозяйство и отображает его на экране обзора домохозяйства. Это центральный экран процесса регистрации, где вы можете видеть всех в вашей семье и управлять назначениями групп перед регистрацией.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- [Выберите служение](./selecting-service) на экране служений
- [Найдите члена](./looking-up-members) и выберите его из результатов поиска

</div>

## Просмотр членов домохозяйства

Каждый член домохозяйства отображается в виде карточки с:

- **Фотографией** (или значком-заполнителем, если фотографии нет в профиле)
- **Отображаемым именем**
- **Обзором текущих назначений групп** (показаны маленькими метками под именем, когда карточка свёрнута)
- **Стрелкой раскрытия/свёртывания** с правой стороны

## Раскрытие карточки члена

Нажмите на карточку члена, чтобы раскрыть её. В раскрытом виде вы увидите строку для каждого **времени служения** в рамках выбранного служения. Каждая строка показывает:

- **Название времени служения** (например, «9:00» или «11:00»)
- **Кнопку** с названием текущей назначенной группы или «Выбрать группу», если группа ещё не назначена

Нажмите кнопку группы, чтобы перейти на [экран выбора группы](./group-assignment), где можно выбрать или изменить группу для этого человека и времени служения.

Нажмите на карточку члена снова, чтобы свернуть её.

:::tip
Вам не нужно назначать всех членов семьи в группы. Взрослые, посещающие основное служение, могут оставаться без назначения группы. Назначайте группы только для членов, которым нужно зарегистрироваться в определённом классе или программе.
:::

## Навигация с этого экрана

С экрана обзора домохозяйства у вас три варианта:

- **Раскрыть карточку члена** и нажать кнопку времени служения, чтобы [назначить или изменить группу](./group-assignment).
- **Нажать кнопку «Добавить гостя»** под списком членов, чтобы [добавить гостя](./adding-guests) в ваше домохозяйство.
- **Нажать кнопку «Check-in»** в нижней части экрана, чтобы [завершить регистрацию](./completing-checkin) для всех в домохозяйстве.

:::info
Любые назначения групп из предыдущей регистрации на то же служение загружаются автоматически. Если ваши дети посещают одни и те же группы каждую неделю, возможно, ничего менять не потребуется — просто проверьте и зарегистрируйтесь.
:::
