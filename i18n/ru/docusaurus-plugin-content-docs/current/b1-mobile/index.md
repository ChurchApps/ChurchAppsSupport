# B1 Mobile (Устарело)

:::danger Устарело — используйте B1.church вместо этого
Встроенное приложение B1 Mobile для iOS и Android **устарело** и больше не разрабатывается. Его функции теперь доставляются через **B1.church Progressive Web App (PWA)** по адресу `https://yourchurchname.b1.church`, которое можно добавить на главный экран любого iPhone или Android и обеспечивает тот же опыт портала членов без загрузки из App Store.

**Если вы член церкви:** см. [Installing as an App (PWA)](../b1-church/getting-started/installing-pwa.md), чтобы установить замену на вашем телефоне.

**Если вы администратор церкви:** указывайте своих членов на сайт вашей церкви (`https://yourchurchname.b1.church`) вместо App Store / Google Play — они могут установить PWA с любой страницы.

Страницы ниже сохраняются для справки для церквей, которые всё ещё запускают старое встроенное приложение.
:::

<div class="article-intro">

B1 Mobile *было* сопутствующее встроенное приложение для B1.church на iOS и Android. Оно дало членам церкви мобильный доступ к группам, пожертвованиям, регистрации, планам служения, справочнику членов, частным сообщениям, проповедям, Библии, стиху дня, урокам и многому другому.

</div>

Приложение использовало вашу учётную запись ChurchApps, ту же, которую вы используете на B1.church. После входа и выбора вашей церкви приложение адаптировалось к брендингу и конфигурации вашей церкви. Если вы принадлежали нескольким церквям, вы могли переключаться между ними в любой момент.

B1 Mobile поддерживал push-уведомления, поэтому вы не пропустили важное обновление из вашей церкви или групп. Главная панель управления персонализировалась на основе вашего использования — функции, которые вы посещали чаще всего, поднимались в верхнюю часть для быстрого доступа.

## Ключевые функции

- **Groups** (Группы) — просмотрите свои группы, участвуйте в разговорах и оставайтесь в контакте с членами группы.
- **Giving** (Пожертвования) — давайте пожертвования своей церкви прямо с телефона.
- **Events** (События) — просматривайте предстоящие события и зарегистрируйтесь прямо с телефона.
- **Check-In** (Регистрация) — зарегистрируйтесь и члены вашего домохозяйства для служений и событий.
- **Serving Plans** (Планы служения) — просмотрите предстоящие планы служения, свои назначенные должности волонтёров и зарегистрируйтесь для открытых должностей.
- **Member Directory** (Справочник членов) — просмотрите справочник вашей церкви и профили членов.
- **Messaging** (Обмен сообщениями) — отправляйте и получайте частные сообщения с другими членами церкви.
- **Sermons** (Проповеди) — смотрите или слушайте проповеди из вашей церкви.
- **Bible** (Библия) — читайте Библию внутри приложения.
- **Verse of the Day** (Стих дня) — начните свой день с ежедневного стиха.
- **Lessons** *(устарело — заменено на [FreePlay](/docs/freeplay/))* — использовалось для встраивания уроков из Lessons.church внутри приложения.
