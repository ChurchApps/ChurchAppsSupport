# Начало работы

:::danger B1 Mobile устарело
Встроенное приложение B1 Mobile заменено на [B1.church Progressive Web App (PWA)](../../b1-church/getting-started/installing-pwa.md). Новые пользователи должны установить PWA со страницы на `https://yourchurchname.b1.church` вместо загрузки B1 Mobile из App Store или Google Play.
:::

<div class="article-intro">

Начало работы с B1 Mobile займёт всего несколько минут. Загрузите приложение, войдите с помощью вашей учётной записи ChurchApps, выберите вашу церковь, и вы готовы начинать.

</div>

## Быстрый старт

1. **Download** (Загрузите) приложение из [App Store](https://apps.apple.com) (iOS) или [Google Play Store](https://play.google.com) (Android). Поищите "B1.church".
2. **Log in** (Войдите) со своей существующей учётной записью ChurchApps — тем же адресом электронной почты и паролем, который вы используете на B1.church. Если у вас ещё нет учётной записи, нажмите **Register** (Регистр), чтобы создать учётную запись.
3. **Select your church** (Выберите вашу церковь), ищя её по названию. Введите как минимум 3 символа, чтобы увидеть результаты.
4. **Explore the dashboard** (Изучите панель управления) — ваш персонализированный главный экран показывает быстрые ссылки на функции, которые ваша церковь предлагает.

Это всё. После того, как вы выберете вашу церковь, приложение загружает брендинг вашей церкви, ссылки навигации и содержимое. Вы можете начать использовать группы, пожертвования, регистрацию и всё остальное, что включила ваша церковь.

:::
