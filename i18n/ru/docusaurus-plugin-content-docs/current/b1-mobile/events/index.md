# События

<div class="article-intro">

Функция события в B1 Mobile позволяет вам просмотреть предстоящие события церкви и зарегистрироваться прямо со своего телефона. Когда ваша церковь включает регистрацию для события, вы можете зарегистрировать себя и членов семьи всего за несколько касаний.

</div>

## Поиск событий с регистрацией

События с включенной регистрацией появляются в календарях вашей группы. Когда вы нажимаете на событие, доступное для регистрации, вы видите кнопку **Зарегистрировать** в деталях события.

События могут показывать разные статусы:

- **Открыто** — Регистрация доступна и вы можете зарегистрироваться.
- **Еще не открыто** — Окно регистрации еще не началось.
- **Закрыто** — Крайний срок регистрации прошел.
- **Полно** — Все доступные места заняты.
