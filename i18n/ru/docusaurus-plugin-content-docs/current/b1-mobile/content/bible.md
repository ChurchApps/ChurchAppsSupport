# Библия

<div class="article-intro">

B1 Mobile включает встроенный библейский ридер для чтения Писания прямо из приложения. Ридер встроен непосредственно в экран Библии, обеспечивая быстрый доступ к любому отрывку без переключения на другое приложение.

</div>

<div class="prereqs">
<h4>Прежде Чем Начать</h4>

- [Установите B1 Mobile](../getting-started/installing) и [войдите](../getting-started/logging-in) в свою учетную запись
- В вашей церкви должна быть включена функция Библии

</div>

## Чтение Библии

1. Откройте экран **Библия** из бокового меню.
2. Ридер загружается с отрывком по умолчанию.
3. Используйте навигационные элементы управления внутри ридера для перехода к другой книге, главе или стиху.

Библейский ридер работает на основе YouVersion или альтернативного провайдера, настроенного вашей церковью. Опыт чтения одинаков независимо от используемого провайдера.

## Навигация по Отрывкам

Встроенный ридер предоставляет элементы управления для перемещения по Писанию. Вы можете:

- Перейти к определенной книге Библии
- Перейти к определенной главе
- Прокручивать стихи внутри главы

Ридер занимает весь экран, предоставляя достаточно пространства для комфортного чтения на вашем устройстве.

:::tip
Сделайте экран Библии частью вашей ежедневной рутины -- это быстрый способ читать Писание, где бы вы ни были. Сочетайте его со [Стихом Дня](./verse-of-the-day) для ежедневного вдохновения.
:::

## Доступность

Экран Библии доступен, когда ваша церковь включила эту функцию. Если вы не видите опцию **Библия** в боковом меню, администратор вашей церкви может не добавить ее в приложение. Обратитесь к администратору вашей церкви, чтобы запросить ее.
