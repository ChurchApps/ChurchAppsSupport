# Управление Способами Оплаты

<div class="article-intro">

Вкладка **Управление** в разделе пожертвований позволяет просматривать и управлять сохраненными способами оплаты. Сохранив карту, вы можете делать пожертвования быстрее без повторного ввода данных каждый раз.

</div>

<div class="prereqs">
<h4>Прежде Чем Начать</h4>

- [Установите B1 Mobile](../getting-started/installing) и [войдите](../getting-started/logging-in) в свою учетную запись

</div>

## Просмотр Способов Оплаты

На вкладке **Управление** вы увидите список всех сохраненных способов оплаты. Каждая запись показывает тип карты и последние четыре цифры номера, чтобы легко определить нужную карту.

## Добавление Нового Способа Оплаты

Чтобы добавить новую карту:

1. Откройте раздел **Пожертвования** в B1 Mobile.
2. Нажмите на вкладку **Управление**.
3. Нажмите на опцию добавления нового способа оплаты.
4. Введите данные карты. Информация безопасно отправляется в **Stripe** для обработки.
5. После сохранения новая карта появится в списке способов оплаты.

Сохраненная карта будет доступна как вариант оплаты при следующем [пожертвовании](./making-donations) на вкладке **Пожертвовать**.

## Удаление Способа Оплаты

Если вы больше не хотите иметь карту в файле, удалите ее на вкладке **Управление**:

1. Найдите карту в списке сохраненных способов оплаты.
2. Нажмите на опцию удаления рядом с картой.
3. Подтвердите удаление.

Карта больше не будет отображаться как вариант оплаты при пожертвовании.

:::info
Удаление способа оплаты не влияет на прошлые пожертвования. Ваша [история пожертвований](./donation-history) остается без изменений.
:::

:::tip
Сохраняйте хотя бы один способ оплаты для быстрых пожертвований. Так вы сможете использовать кнопку **Повторить** на вкладке **Обзор**, чтобы пожертвовать одним нажатием.
:::
