# Регистрация волонтеров

<div class="article-intro">

Просматривайте доступные волонтерские позиции и регистрируйтесь на предстоящие услуги прямо из B1 Mobile. Узнайте, какие позиции нуждаются в заполнении, сколько открыто мест и присоединитесь к команде всего за несколько касаний.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- [Установите B1 Mobile](../getting-started/installing) и [войдите](../getting-started/logging-in) в свой аккаунт
- Администратор вашей церкви должен включить самостоятельную регистрацию по крайней мере на одной позиции

</div>

## Просмотр волонтерских возможностей

1. Откройте раздел **Волонтеры** из навигации приложения.
2. Появляется список предстоящих планов услуг с открытыми позициями.
3. Каждая карточка плана показывает:
   - **Название плана** и **дата услуги**
   - **Общую прогресс-полосу**, показывающую общее количество заполненных мест
   - **Статус** — количество оставшихся мест или "Полно", если все места заняты
   - **Фишки позиций**, показывающие каждую роль и количество открытых мест
4. Нажмите **Просмотреть и зарегистрироваться**, чтобы увидеть полный список позиций для плана.

## Регистрация на позицию

1. Нажмите на план, чтобы открыть его детали.
2. Позиции сгруппированы по категориям (например, "Группа", "Гостеприимство", "Детское служение").
3. Каждая позиция показывает название роли, описание и доступные места.
4. Нажмите **Зарегистрироваться** рядом с позицией, на которую вы хотите зарегистрироваться.
5. Появляется диалог подтверждения: "Вы зарегистрировались! Спасибо за волонтерство."
6. Позиция показывает зеленый индикатор, подтверждающий вашу регистрацию.

:::info
Вы должны войти, чтобы зарегистрироваться на позицию. Приложение попросит вас войти, если необходимо.
:::

## Удаление регистрации

1. Найдите позицию, на которую вы зарегистрировались — она показывает зеленый индикатор "Вы зарегистрировались".
2. Нажмите **Удалить**.
3. Диалог подтверждения спросит, уверены ли вы.
4. Подтвердите, чтобы удалить себя из позиции.

## Сроки и вместимость

- **Сроки** — Ваша церковь может установить крайний срок до услуги. После крайнего срока вы не можете зарегистрироваться или удалить регистрации. Баннер в верхней части плана указывает, когда крайний срок прошел.
- **Вместимость** — Каждая позиция имеет ограниченное количество мест. Когда заполнено, кнопка регистрации отключена. Места открываются, если кто-то отменяет регистрацию.

:::tip
Включите [push-уведомления](../community/notifications.md), чтобы оставаться в курсе новых волонтерских возможностей и изменений в ваших назначениях.
:::
