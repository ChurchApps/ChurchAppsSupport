# Служение

<div class="article-intro">

Раздел "Служение" на B1.church позволяет вам просмотреть добровольческие возможности и зарегистрироваться на позиции в вашей церкви. Кроме просмотра назначенных вам планов и порядков служения, вы теперь можете самостоятельно выбирать позиции, которые нужно заполнить.

</div>

## Что вы можете делать

- **Просмотрите возможности для волонтёров** — посмотрите предстоящие планы служения с открытыми позициями, которые требуют волонтёров.
- **Зарегистрируйтесь на позиции** — выберите позицию и зарегистрируйте себя одним нажатием.
- **Просмотрите ваши планы** — посмотрите ваши предстоящие и прошлые [назначения служения](../plans/viewing-plans).
- **Удалите регистрации** — отмените самостоятельную регистрацию, если ваши планы изменятся (до крайнего срока).

## Как это работает

Администратор вашей церкви создаёт планы служения и отмечает определённые позиции как доступные для самостоятельной регистрации. Эти позиции появляются на странице добровольческой регистрации, организованные по категориям (например, "Группа", "Гостеприимство" или "Служение детям"). Вы можете видеть, сколько мест открыто для каждой позиции, и зарегистрироваться на те, для которых вы доступны.

:::info
На странице волонтёра отображаются только позиции, которые администратор вашей церкви включил для самостоятельной регистрации. Некоторые позиции могут заполняться только через прямое назначение.
:::
