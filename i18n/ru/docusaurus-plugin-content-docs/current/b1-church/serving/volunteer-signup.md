# Волонтерская регистрация

<div class="article-intro">

Просмотрите доступные добровольческие должности и зарегистрируйтесь на предстоящие служения прямо из B1.church. Должности организованы по категориям, чтобы вы быстро найти, где нужна помощь и присоединиться к команде.

</div>

<div class="prereqs">
<h4>Перед началом работы</h4>

- Учетная запись B1.church — вы должны быть в системе для регистрации
- Администратор вашей церкви должен включить самостоятельную регистрацию хотя бы на одну должность

</div>

## Просмотр доступных должностей

1. Перейдите на страницу **Волонтер** на веб-сайте B1.church вашей церкви.
2. Вы увидите список предстоящих планов служений с открытыми добровольческими должностями.
3. Каждый план показывает **название плана**, **дату служения** и **время служения**.

Должности сгруппированы по категориям (например, «Группа», «Гостеприимство», «Служение детям»). Каждая должность показывает:

- **Название должности** — Роль, которую вы заполните (напр., "Барабанщик", "Встречающий", "Сервер кофе").
- **Описание** — Подробности о роли, если предоставлены администратором.
- **Доступные места** — Полоса прогресса, показывающая, сколько слотов заполнено и сколько осталось.

## Регистрация

1. Найдите должность, в которой хотите помочь.
2. Нажмите кнопку **Зарегистрировать** рядом с должностью.
3. Появляется сообщение подтверждения: "Вы зарегистрированы! Спасибо за помощь."
4. Должность теперь показывает зеленый индикатор, подтверждающий вашу регистрацию.

:::info
Вы должны быть в системе для регистрации. Если вы не в системе, вам будет предложено сначала войти.
:::

## Удаление регистрации

Если ваши планы изменились и вы больше не можете служить:

1. Найдите должность, на которую вы зарегистрировались — она будет показывать зеленый индикатор "Вы зарегистрированы".
2. Нажмите кнопку **Удалить**.
3. Подтвердите, что вы хотите удалить себя с этой должности.
4. Сообщение подтверждает: "Вы удалены с этой должности."

## Сроки

Администратор вашей церкви может установить срок регистрации для каждого плана. Срок — это установленное количество часов до даты служения. После истечения срока:

- Вы не можете зарегистрироваться на новые должности.
- Вы не можете удалить существующие регистрации.
- Сообщение в верхней части плана указывает, что срок регистрации прошел.

## Вместимость должности

Каждая должность имеет ограниченное количество мест. Когда все места заполнены:

- Должность показывается как **Полно**.
- Кнопка регистрации отключена.
- Если кто-то удалит свою регистрацию, место откроется снова.

:::tip
Регулярно проверяйте страницу волонтера, особенно в начале недели. Популярные должности заполняются быстро, поэтому ранняя регистрация дает вам лучший выбор.
:::
