# Регистрация

<div class="article-intro">

Функция **Регистрация** в B1.church позволяет зарегистрировать себя и членов вашей семьи на богослужения церкви прямо из портала участника. Это удобный способ отметить посещение без необходимости использовать физический киоск регистрации.

</div>

## Как это работает

При использовании саморегистрации вы выбираете богослужение, указываете, какие члены семьи присутствуют, и назначаете каждого в соответствующую группу для каждого времени служения. После подтверждения ваше посещение автоматически записывается.

## Начало работы

Для доступа к регистрации нажмите **Регистрация** в [боковой панели навигации](../getting-started/navigating.md) вашего портала участника. Вам необходимо [войти в систему](../getting-started/logging-in.md), чтобы использовать эту функцию — если вы не авторизованы, вам будет предложено войти.

:::info
Функция регистрации требует, чтобы ваша церковь настроила богослужения и группы посещаемости в настройках администрирования. Если при открытии страницы регистрации нет доступных богослужений, ваша церковь может ещё не настроить эту функцию. Обратитесь к администратору вашей церкви за помощью.
:::
