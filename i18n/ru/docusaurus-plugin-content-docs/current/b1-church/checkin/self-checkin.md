# Саморегистрация

<div class="article-intro">

Процесс саморегистрации позволяет отметить посещение для себя и вашей семьи за несколько простых шагов. Это быстрая альтернатива использованию физического киоска в церкви.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Вам необходимо [войти в систему](../getting-started/logging-in.md) вашей учётной записи B1.church.
- Члены вашей семьи должны быть привязаны к вашей учётной записи. Если кто-то из семьи отсутствует, попросите администратора церкви добавить их в вашу семью.

</div>

## Процесс регистрации

Регистрация проходит в три простых шага. Перейдите на вкладку **Регистрация** в [боковой панели](../getting-started/navigating.md), чтобы начать.

### Шаг 1: Выберите богослужение

При открытии страницы регистрации вы увидите список доступных богослужений (например, «Воскресное утро» или «Среда вечер»). Каждая карточка показывает название богослужения и кампус, если у вашей церкви несколько мест.

Нажмите на богослужение, чтобы выбрать его и перейти к следующему шагу.

### Шаг 2: Выберите членов семьи и группы

После выбора богослужения вы увидите список членов вашей семьи. Для каждого присутствующего:

1. Нажмите на члена семьи, чтобы развернуть его запись.
2. Вы увидите доступные **времена богослужений**.
3. Нажмите **Выбрать группу** рядом с временем богослужения, чтобы выбрать группу для регистрации (например, «Основное богослужение», «Детская комната 1» или «Молодёжная группа»).
4. Повторите для каждого присутствующего члена семьи.

Не обязательно регистрировать всех — разверните и назначьте группы только для тех, кто действительно присутствует.

### Шаг 3: Подтверждение

Выбрав группы для всех присутствующих, нажмите кнопку **Завершить регистрацию** внизу страницы.

Вы увидите экран подтверждения с сообщением «Регистрация завершена!» и уведомлением о том, что посещение сохранено. Нажмите **Назад на мою страницу**, чтобы вернуться к [ленте](../community/timeline.md).

:::tip
Если вам нужно изменить выбор группы перед завершением регистрации, нажмите кнопку **Изменить** рядом с любым временем богослужения, чтобы выбрать другую группу.
:::

## Важно знать

- Вы можете регистрировать только членов семьи, привязанных к вашей учётной записи. Если кто-то отсутствует, попросите администратора церкви добавить их.
- Если на странице регистрации нет доступных богослужений, ваша церковь может не иметь настроенных богослужений на данный момент.

:::warning
Регистрация доступна только при наличии активных богослужений, настроенных вашей церковью. Если вы не видите ни одного богослужения, обратитесь в офис церкви для подтверждения, включена ли саморегистрация.
:::
