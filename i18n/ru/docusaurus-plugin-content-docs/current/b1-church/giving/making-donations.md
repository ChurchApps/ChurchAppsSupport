# Сделать пожертвование

<div class="article-intro">

Есть два способа сделать пожертвование через сайт B1.church вашей церкви: публичная страница пожертвований (вход не требуется) и страница пожертвований для участников (вход требуется). Обе используют безопасную форму пожертвования на базе Stripe.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Для публичных пожертвований учётная запись не нужна — просто посетите страницу **Пожертвования** вашей церкви.
- Для пожертвований участника с отслеживанием истории и сохранёнными способами оплаты, сначала [войдите в систему](../getting-started/logging-in.md).

</div>

## Публичная страница пожертвований (без входа)

Любой может сделать пожертвование без создания учётной записи:

1. Перейдите на страницу **Пожертвования** вашей церкви.
2. Форма пожертвования отображается в левой части страницы. Выберите **фонд**, в который хотите пожертвовать.
3. Введите платёжные данные и сумму пожертвования.
4. Отправьте форму, чтобы завершить пожертвование.

В правой части страницы вы найдёте ссылку на **Инструкции для ACH-платежа**, если предпочитаете банковский перевод. Также есть кнопка **Войти**, если вы хотите авторизоваться для доступа к дополнительным функциям, таким как история пожертвований.

:::tip
Если вы уже авторизованы при посещении публичной страницы пожертвований, вы будете автоматически перенаправлены на страницу пожертвований для участников, где доступны полная история и сохранённые способы оплаты.
:::

## Страница пожертвований для участников (с авторизацией)

Когда вы авторизованы, опыт пожертвования включает дополнительные функции:

1. Нажмите на вкладку **Пожертвования** (или перейдите к **Мои пожертвования**).
2. Вы увидите панель с тремя разделами:
   - **Пожертвовать сейчас** — Сделать разовое или регулярное пожертвование.
   - **История** — Просмотреть прошлые пожертвования и экспортировать записи.
   - **Регулярные пожертвования** — Управлять запланированными пожертвованиями и способами оплаты.
3. Выберите **Пожертвовать сейчас**, чтобы открыть форму.
4. Выберите **фонд** и введите сумму пожертвования.
5. Выберите существующий сохранённый способ оплаты или введите новые платёжные данные.
6. Отправьте форму, чтобы завершить пожертвование.

После обработки пожертвования появится сообщение об успехе, и ваше пожертвование отобразится в [истории пожертвований](./donation-history.md).

## Способы оплаты

Пожертвования обрабатываются безопасно через **Stripe**. Вы можете оплатить кредитной картой, дебетовой картой или настроить ACH-платёж. При авторизации ваши способы оплаты могут быть сохранены для будущего использования, что ускорит пожертвование в следующий раз.

:::warning
Если ваше пожертвование не прошло, перепроверьте платёжные данные и попробуйте снова. Если проблема сохраняется, обратитесь в офис вашей церкви за помощью.
:::
