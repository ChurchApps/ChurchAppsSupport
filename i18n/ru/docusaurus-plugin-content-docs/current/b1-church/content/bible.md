# Библия

<div class="article-intro">

B1.church включает встроенный читатель Библии, чтобы вы могли читать Священное Писание, не покидая сайт. Читатель встроен непосредственно на страницу, давая вам быстрый доступ к любому отрывку рядом с другими инструментами вашей церкви.

</div>

<div class="prereqs">
<h4>Перед началом</h4>

- Вход в систему не требуется для использования читателя Библии. Страница Библии общедоступна.
- Ваша церковь должна добавить страницу Библии в навигацию сайта.

</div>

## Чтение Библии

1. Перейдите на страницу **Библия** из навигации сайта вашей церкви.
2. Читатель открывается по умолчанию в Бытие 1:1.
3. Используйте элементы управления навигацией в читателе для перемещения в другую книгу, главу или стих.

Читатель Библии работает на основе **YouVersion** и даёт вам доступ к широкому выбору переводов Библии на многих языках.

## Навигация по отрывкам

Встроенный читатель предоставляет элементы управления для перемещения по Священному Писанию. Вы можете:

- Перейти в определённую книгу Библии
- Перепрыгнуть в конкретную главу
- Прокручивать стихи в главе
- Переключиться на другой перевод Библии, используя селектор версии

Читатель заполняет содержимое страницы, давая вам достаточно места для комфортного чтения.

:::tip
Выбранная вами книга, глава и перевод запоминаются, поэтому читатель открывается там, где вы остановились.
:::

:::tip
Вы можете добавить страницу Библии в закладки браузера, чтобы быстро вернуться к ней для ежедневного чтения. Комбинируйте с [Стихом дня](./verse-of-the-day.md) для ежедневного вдохновения Священным Писанием.
:::

## Доступность

Страница Библии — это встроенная функция содержимого B1.church. Если вы не видите ссылку **Библия** в навигации вашей церкви, администратор вашей церкви, возможно, не добавил её в меню сайта. Обратитесь к администратору вашей церкви с просьбой добавить её.
