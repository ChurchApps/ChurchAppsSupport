# Библия

<div class="article-intro">

B1.church включает встроенную программу для чтения Библии, чтобы вы могли читать Священное Писание, не покидая сайт. Программа встроена непосредственно в страницу, обеспечивая быстрый доступ к любому отрывку наряду с другими церковными инструментами.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Для использования программы чтения Библии вход в систему не требуется. Страница Библии доступна публично.
- Ваша церковь должна добавить страницу Библии в навигацию сайта.

</div>

## Чтение Библии

1. Перейдите на страницу **Библия** через навигацию сайта вашей церкви.
2. Программа открывается на Бытие 1:1 по умолчанию.
3. Используйте элементы навигации для перехода к другой книге, главе или стиху.

Программа чтения Библии использует по умолчанию перевод NIRV (New International Reader's Version).

## Навигация по отрывкам

Встроенная программа предоставляет элементы управления для перемещения по Писанию. Вы можете:

- Перейти к конкретной книге Библии
- Перейти к определённой главе
- Прокручивать стихи внутри главы

Программа заполняет область содержимого страницы, предоставляя достаточно места для комфортного чтения.

:::tip
Вы можете добавить страницу Библии в закладки браузера для быстрого возвращения к ежедневному чтению. Сочетайте с [Стихом дня](./verse-of-the-day.md) для ежедневного вдохновения Писанием.
:::

## Доступность

Страница Библии — это встроенная функция контента B1.church. Если вы не видите ссылку **Библия** в навигации вашей церкви, администратор может не добавить её в меню сайта. Обратитесь к администратору вашей церкви для запроса.
