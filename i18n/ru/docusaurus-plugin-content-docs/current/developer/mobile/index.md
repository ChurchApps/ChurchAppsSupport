# Мобильные приложения

<div class="article-intro">

Мобильные приложения ChurchApps построены на **React Native** и **Expo**, обеспечивая нативный опыт на Android, iOS и Android TV. Все мобильные приложения используют один и тот же набор инструментов на основе Expo, поэтому если вы можете собрать одно, вы можете собрать их все.

</div>

## Приложения

| Приложение | Назначение | Платформа |
|-----|---------|----------|
| [B1 Mobile](./b1-mobile) | Основное приложение для членов церкви | Android, iOS |
| B1 Checkin | Приложение для киосков регистрации | Android, iOS |
| LessonsScreen | Приложение Android TV для отображения уроков в аудиториях | Android TV |

### B1 Mobile

Основное приложение для членов церкви. Члены могут просматривать справочники, получать доступ к пожертвованиям, проверять посещаемость, получать уведомления и взаимодействовать с церковным сообществом. Подробности см. в [руководстве по настройке B1 Mobile](./b1-mobile).

### B1 Checkin

Специализированное приложение для киосков, предназначенное для станций регистрации в церквях. Оно предоставляет простой, удобный для сенсорного управления интерфейс для регистрации детей и записи посещаемости на служениях и мероприятиях.

### LessonsScreen

Приложение Android TV, отображающее учебный контент в аудиториях. Преподаватели могут отправлять слайды уроков, видео и задания на ТВ-экран, что упрощает демонстрацию учебных материалов во время занятий.

:::info
Все мобильные приложения используют один и тот же набор инструментов на основе Expo и следуют схожим паттернам настройки. Руководство по [B1 Mobile](./b1-mobile) служит справочником для общего процесса настройки всех мобильных проектов.
:::
