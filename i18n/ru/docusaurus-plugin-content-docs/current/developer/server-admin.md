# Администрирование сервера

<div class="article-intro">

Функции администрирования сервера в ChurchApps доступны только пользователям с разрешением **Server.Admin**. Эти инструменты используются для операций платформы, поддержки и устранения неполадок на всех церквях в системе.

</div>

:::warning Доступ ограничен
Функции, описанные на этой странице, требуют разрешение **Server.Admin** и недоступны обычным администраторам церквей. Они предназначены только для операторов платформы и персонала поддержки.
:::

## Доступ к администрированию сервера

Пользователи с разрешением Server.Admin могут получить доступ к панели администратора сервера из B1 Admin:

1. Войдите в [admin.b1.church](https://admin.b1.church)
2. Нажмите вкладку **Admin** в основной навигации
3. Панель включает вкладки для управления церквями, пользователями и операциями системы

## Олицетворение пользователя

Функция олицетворения позволяет администраторам сервера войти как другой пользователь в целях поддержки и устранения неполадок.

### Как олицетворить пользователя

1. Перейдите на вкладку **Олицетворить** в панели администратора
2. Введите имя или адрес электронной почты пользователя
3. Нажмите **Поиск** или нажмите Enter
4. Выберите пользователя из результатов
5. Подтвердите олицетворение
6. Вы будете вошли как этот пользователь

### Важные примечания

- Олицетворение создает новую сессию с разрешениями целевого пользователя
- Ваша сессия администратора заканчивается при олицетворении
- Все действия регистрируются в журнале аудита
- Выйдите и повторно войдите, чтобы вернуться к своему аккаунту
- Используйте олицетворение только для целей поддержки

### Соображения безопасности

- Олицетворение требует разрешение Server.Admin
- Все события записываются с ID администратора и целевого пользователя
- Церкви не уведомляются об олицетворении
- Документируйте события олицетворения в системе поддержки
