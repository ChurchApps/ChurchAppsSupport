# Эндпоинты контента

<div class="article-intro">

Модуль контента управляет страницами веб-сайта, проповедями, плейлистами, потоковыми сервисами, событиями, файлами, галереями, переводами Библии и песнями. Это самый большой модуль в API.

</div>

**Базовый путь:** `/content`

## Страницы

Базовый путь: `/content/pages`

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/tree` | Загрузить дерево страниц |
| GET | `/:id` | Получить страницу |
| POST | `/` | Создать страницу |
| DELETE | `/:id` | Удалить страницу |

## Проповеди
