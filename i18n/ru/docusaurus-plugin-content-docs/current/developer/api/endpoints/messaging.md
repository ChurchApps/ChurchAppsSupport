# Эндпоинты сообщений

<div class="article-intro">

Модуль сообщений управляет разговорами, чатом, push-уведомлениями, SMS/email доставкой и текстовыми сообщениями.

</div>

**Базовый путь:** `/messaging`

## Разговоры

Базовый путь: `/messaging/conversations`

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/:id` | Получить разговор |
| POST | `/` | Создать разговор |
| DELETE | `/:id` | Удалить разговор |

## Сообщения

Базовый путь: `/messaging/messages`

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/:id` | Получить сообщение |
| POST | `/` | Отправить сообщение |
| DELETE | `/:id` | Удалить сообщение |

## Приватные сообщения

Базовый путь: `/messaging/privatemessages`

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/` | Список сообщений |
| POST | `/` | Отправить сообщение |

## Уведомления

Базовый путь: `/messaging/notifications`

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/my` | Мои уведомления |
| POST | `/` | Создать уведомление |

## SMS

Базовый путь: `/messaging/texting`

| Метод | Путь | Описание |
|-------|------|---------|
| POST | `/send` | Отправить SMS |

## Email

Базовый путь: `/messaging/emailTemplates`

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/` | Список шаблонов |
| POST | `/send` | Отправить email |
