# Эндпоинты посещаемости

<div class="article-intro">

Модуль посещаемости управляет кампусами, служениями и записями о посещаемости.

</div>

**Базовый путь:** `/attendance`

## Кампусы

Базовый путь: `/attendance/campuses`

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/` | Список всех кампусов |
| GET | `/:id` | Получить кампус |
| POST | `/` | Создать кампус |
| DELETE | `/:id` | Удалить кампус |

## Служения

Базовый путь: `/attendance/services`

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/` | Список служений |
| GET | `/:id` | Получить служение |
| POST | `/` | Создать служение |
| DELETE | `/:id` | Удалить служение |

## Посещаемость

Базовый путь: `/attendance/attendancerecords`

| Метод | Путь | Описание |
|-------|------|---------|
| GET | `/` | Загрузить записи |
| GET | `/tree` | Полное дерево |
| GET | `/trend` | Тренд посещаемости |
