# Начало работы

<div class="article-intro">

Добро пожаловать в ChurchApps -- бесплатные инструменты с открытым исходным кодом, созданные для того, чтобы помочь вашей церкви процветать. Независимо от того, являетесь ли вы небольшой общиной или крупной многокампусной организацией, ChurchApps предоставляет все необходимое для управления и развития вашего церковного сообщества.

</div>

## Продукты ChurchApps

ChurchApps включает несколько продуктов, которые работают вместе:

- **B1 Admin** -- Панель управления вашей церковью. Управляйте людьми, группами, посещаемостью, пожертвованиями, содержимым сайта и многим другим из одного центрального места.
- **B1.church** -- Публичный сайт вашей церкви и портал для членов. Члены могут просматривать проповеди, получать доступ к справочникам, делать пожертвования онлайн и оставаться на связи.
- **Lessons.church** -- Бесплатные учебные материалы для воскресной школы, малых групп и изучения Библии. Получите доступ к растущей библиотеке уроков бесплатно.
- **FreeShow** -- Бесплатное программное обеспечение для презентаций на ваших богослужениях. Отображайте тексты песен, слайды и медиа во время ваших собраний.

:::info
Все инструменты ChurchApps **100% бесплатны** без ограничений, премиум-уровней и скрытых платежей. ChurchApps -- это некоммерческая организация, поддерживаемая пожертвованиями и волонтерами.
:::

## Что охватывает эта документация

Эта документация проведет вас через все, что нужно для начала работы с ChurchApps. Вы узнаете, как:

1. Создать учетную запись и зарегистрировать свою церковь.
2. Настроить основные параметры и фирменный стиль вашей церкви.
3. Настроить команду с правильными ролями и разрешениями.
4. Начать использовать инструменты для управления вашей общиной.
