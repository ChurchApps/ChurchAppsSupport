# Lessons.church

<div class="article-intro">

Lessons.church — это полностью бесплатная платформа учебных программ для церквей. Никаких подписок, никаких сборов и никаких ограничений на использование. Она предоставляет более 1100 видеоуроков в более чем 189 курсах от более чем 12 поставщиков, охватывая возрастные группы от дошкольников до взрослых.

</div>

## Что вы можете делать

С помощью Lessons.church ваша церковь может:

- **Просматривать контент** — изучать растущую библиотеку программ и курсов от различных поставщиков, организованную по возрастным группам и темам.
- **Настраивать классы** — создавать классы, соответствующие структуре вашего служения (например, дошкольники, начальная школа, молодёжь или взрослые).
- **Планировать уроки** — назначать конкретные уроки для ваших классов на нужные даты.
- **Настраивать под свои нужды** — корректировать содержание уроков в соответствии со стилем преподавания и предпочтениями вашей церкви.
- **Проводить занятия** — использовать встроенный режим презентации для пошагового проведения уроков во время занятий.

## Организация контента

Lessons.church организует учебные программы в чёткую иерархию:

1. **Программы** — серия учебных материалов верхнего уровня от поставщика (например, программа для детского служения).
2. **Курсы** — группа связанных уроков в рамках программы (например, четырёхнедельная серия по определённой теме).
3. **Уроки** — отдельное учебное занятие в рамках курса, обычно используемое для одного занятия.
4. **Площадки** — различные версии урока, адаптированные для конкретных возрастных групп или условий (например, версия для дошкольников и версия для начальной школы).

## Интеграции

Lessons.church работает с другими инструментами ChurchApps для доставки контента:

- **B1.church** — волонтёры и родители могут получить доступ к предстоящим и недавним урокам через приложение B1.
- **FreeShow** — используйте программу для презентаций FreeShow для отображения содержимого уроков на экране.
- **Venue TV App** — отображайте уроки непосредственно на телевизоре с помощью приложения для Android TV.
