# Руководства для волонтёров

<div class="article-intro">

Каждый урок в Lessons.church включает руководства, адаптированные для конкретных ролей волонтёров. Вместо того чтобы давать каждому волонтёру весь план урока, вы можете направить их к руководству, которое показывает только инструкции, относящиеся к их роли, — просто и по делу.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Ознакомьтесь со структурой урока, прочитав [Просмотр уроков](../browsing/viewing-lessons.md)
- Перейдите к уроку, для которого хотите подготовить руководства

</div>

## Организация руководств

Уроки разделены на **разделы** (например, «Приветствие», «Библейская история», «Активность в малых группах»). Внутри каждого раздела **ролям** назначены конкретные **действия**. Существует четыре типа действий:

- **Сказать** — сценарий текста, который волонтёр должен прочитать или пересказать.
- **Сделать** — физическое действие или активность для выполнения.
- **Заметка** — справочная информация или напоминания для волонтёра (не произносится вслух).
- **Воспроизвести** — видео или аудиоклип для воспроизведения классу.

Когда волонтёр открывает своё руководство, он видит каждый раздел по порядку, но только с действиями, назначенными его роли. Это делает руководство простым и сфокусированным.

## Просмотр руководства для волонтёра

1. Откройте урок, к которому хотите подготовиться.
2. Найдите список **ролей**, связанных с площадкой урока.
3. Нажмите на название роли, чтобы открыть руководство для этой роли.

Руководство отображает каждый раздел в виде пошаговой инструкции. Волонтёры могут просматривать его на телефоне, планшете или компьютере во время занятия.

## Печать руководств

Если волонтёры предпочитают бумажную копию, вы можете распечатать любое руководство прямо из браузера.

1. Откройте руководство для нужной роли.
2. Используйте функцию **Печать** браузера (**Ctrl+P** или **Cmd+P**).
3. Распечатайте страницу или сохраните как PDF для последующего распространения.

:::tip
Делитесь ссылками на руководства с волонтёрами заранее, чтобы они могли ознакомиться с инструкциями до занятия. Быстрый просмотр в течение недели помогает всем чувствовать себя увереннее в воскресное утро.
:::

## Несколько ролей

В некоторых уроках предусмотрено несколько ролей — например, ведущий преподаватель, руководитель малой группы и технический волонтёр. У каждой роли есть собственное руководство, поэтому у вас может быть столько специализированных руководств, сколько предусмотрено в уроке.

:::info
Если какая-либо роль не применима к вашему классу, вы можете скрыть её с помощью функций [настройки](../customization/customizing-lessons.md), чтобы волонтёры видели только актуальные роли.
:::
