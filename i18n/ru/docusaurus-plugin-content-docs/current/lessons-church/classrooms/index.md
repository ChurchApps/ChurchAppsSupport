# Классы

<div class="article-intro">

Классы являются основой организации преподавания в вашей церкви в Lessons.church. Класс представляет собой физическую или виртуальную комнату, где проводятся уроки — например, «Комната для дошкольников», «Начальная школа 1-2 класс» или «Взрослое изучение Библии». В этом разделе описано создание классов и планирование уроков для них.

</div>

## Почему классы важны

Когда вы настраиваете классы, вы создаёте чёткую структуру для детского служения, молодёжных групп или любой учебной среды. Каждый класс получает собственное расписание, поэтому ваши волонтёры всегда точно знают, что преподавать и когда. Вместо того чтобы раздавать бумажные расписания или отправлять письма в последний момент, каждый может проверить портал, чтобы увидеть, что запланировано.

## Что вы можете делать с классами

- **Создавать неограниченное количество классов** — настраивайте столько классов, сколько нужно вашей церкви. Ограничений нет, поэтому вы можете создать отдельные классы для каждой возрастной группы, кампуса или программы.
- **Планировать уроки** — назначайте конкретные уроки для каждого класса на определённые даты. Ваше расписание показывает как прошедшие, так и предстоящие уроки.
- **Настраивать контент** — адаптируйте содержание уроков для каждого класса, чтобы волонтёры видели только то, что соответствует их возрастной группе. Подробности см. в разделе [Настройка](../customization/index.md).

## Как классы связаны между собой

Классы связывают учебный контент на Lessons.church с реальным расписанием преподавания вашей церкви. Вот типичный рабочий процесс:

1. [Просмотрите](../browsing/index.md) доступные программы и курсы на Lessons.church.
2. Настройте классы в соответствии с комнатами и возрастными группами вашей церкви.
3. Запланируйте уроки для каждого класса на предстоящие даты.
4. Волонтёры открывают свой класс, чтобы увидеть, что запланировано, и подготовиться.
