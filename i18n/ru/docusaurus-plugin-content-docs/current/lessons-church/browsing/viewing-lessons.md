# Просмотр уроков

<div class="article-intro">

Каждый урок на Lessons.church — это полноценное учебное занятие со структурированным контентом, загружаемыми материалами и поддержкой нескольких возрастных групп. Это руководство объясняет, как ориентироваться на странице урока и использовать её для подготовки к занятию.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Ознакомьтесь с организацией контента, прочитав [Программы и курсы](./programs-and-studies.md)
- Перейдите к программе и курсу, чтобы найти урок, который хотите просмотреть

</div>

## Переход к уроку

Чтобы просмотреть урок, пройдите по иерархии контента:

1. Нажмите на **программу** на главной странице.
2. Выберите **курс** на странице программы.
3. Нажмите на **урок**, который хотите просмотреть, на странице курса.

## Что вы увидите

Страница урока организована в несколько областей:

### Видео

Если урок включает видео, оно отображается в верхней части страницы. Нажмите для воспроизведения прямо в браузере. Видео часто используются в качестве основного учебного сегмента, особенно в детском служении.

### Разделы и роли

Каждый урок разделён на **разделы** (например, «Вступление», «Библейская история», «Активность», «Заключение»). В каждом разделе контент организован по **ролям**:

- **Ведущий** — основной преподаватель или ведущий
- **Помощник** — волонтёры поддержки
- Другие пользовательские роли, определённые создателем учебной программы

Для каждой роли указаны **действия**, которые говорят волонтёру, что именно делать:

- **Сказать** — сценарий или тезисы для озвучивания
- **Сделать** — физические инструкции или активности для выполнения
- **Заметка** — справочная информация или советы для волонтёра
- **Воспроизвести** — указания на воспроизведение видео или аудиоклипа

### Загружаемые материалы

Уроки могут включать файлы, которые можно загрузить и распечатать, например:

- PDF-раздаточные материалы
- Рабочие листы
- Раскраски
- Инструкции для поделок
- Листы для родителей

Ищите ссылки для загрузки на странице урока. Подробности о подготовке печатных ресурсов см. в разделе [Загрузки и материалы](../presenting/downloads.md).

## Переключение между площадками

Многие уроки поддерживают несколько **площадок**, которые представляют разные возрастные группы, такие как «Дошкольники», «Начальная школа» или «Подростки». Площадки позволяют адаптировать тему одного урока к соответствующему уровню для каждой группы.

Для переключения между площадками:

1. Найдите **переключатель площадок** на странице урока.
2. Нажмите на другое название площадки (например, переключитесь с **Начальная школа** на **Дошкольники**).
3. Страница обновится и покажет контент, сценарии и материалы, разработанные для этой возрастной группы.

:::tip
Если вы преподаёте для нескольких возрастных групп, просмотрите каждую площадку перед занятием, чтобы увидеть, как контент адаптирован для разных уровней.
:::
