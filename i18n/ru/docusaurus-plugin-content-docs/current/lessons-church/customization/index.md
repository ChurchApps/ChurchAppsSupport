# Настройка

<div class="article-intro">

Lessons.church предоставляет библиотеку готовых учебных материалов, но каждая церковь уникальна. Функции настройки позволяют адаптировать содержание уроков к вашим конкретным классам, стилю преподавания и возрастным группам, чтобы ваши волонтёры видели только то, что для них актуально.

</div>

## Зачем настраивать?

Учебный контент на Lessons.church разработан с учётом гибкости. Один урок может включать разделы для нескольких возрастных групп, дополнительные активности или роли, которые применимы не в каждом классе. Вместо того чтобы просить волонтёров мысленно пропускать неактуальные части, вы можете настроить урок так, чтобы они видели только то, что для них важно.

Например, урок может включать активность с поделкой, отлично подходящую для дошкольников, но не подходящую для класса пятиклассников. С помощью настройки вы можете скрыть этот раздел для старшей группы, сохранив его для младшей.

## Что можно настраивать

- **Удаление разделов** — скрытие целых разделов урока, которые не подходят для конкретного класса или возрастной группы.
- **Скрытие ролей или действий** — удаление отдельных ролей или действий внутри раздела, если они не нужны.
- **Изменение порядка контента** — перестановка разделов и действий в соответствии с предпочитаемым ходом занятия.

## Для класса или для всех

При сохранении настройки вы выбираете, насколько широко она применяется:

- **Один класс** — настройка затрагивает только текущий класс. Другие классы видят оригинальный контент.
- **Все классы** — настройка применяется ко всем классам вашей церкви.

Эта гибкость означает, что вы можете настроить разные представления одного и того же урока для разных возрастных групп или сделать изменение, которое применяется везде одновременно.

:::tip
Если вы новичок в настройке, начните с одного класса, чтобы понять, как это работает, прежде чем применять изменения ко всем классам.
:::
