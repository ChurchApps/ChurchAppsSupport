# Статистика

<div class="article-intro">

Страница статистики предоставляет администраторам информацию об использовании их контента. Вы можете увидеть, какие курсы наиболее популярны, какие церкви загружают ваши ресурсы и где эти церкви расположены географически.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- Имейте права администратора для Lessons.church (см. [Обзор администрирования](./index.md))
- Имейте хотя бы одну [программу](./managing-programs.md) с опубликованным контентом

</div>

## Доступ к статистике

1. Перейдите в раздел **Администрирование**.
2. Выберите страницу **Статистика**.
3. Выберите **программу**, для которой хотите просмотреть статистику.

## Что вы можете увидеть

Страница статистики предоставляет несколько представлений данных о загрузках:

### Загрузки по курсам

Разбивка уникальных загрузок для каждого курса в выбранной программе. Это показывает, какие серии получают наибольший отклик, и помогает решить, куда направить усилия по созданию контента.

### Список церквей

Список церквей, которые загрузили ресурсы из вашей программы. Это даёт представление о вашей аудитории и о том, насколько широко ваш контент используется.

### Географическое распределение

Карта, показывающая расположение церквей, загрузивших ваш контент. Это помогает понять географический охват вашей учебной программы и определить регионы, где ваш контент популярен.

### Фильтрация по диапазону дат

Всю статистику можно фильтровать по диапазону дат для фокусировки на определённом периоде. Это полезно для отслеживания трендов во времени или оценки влияния недавно выпущенного курса.

1. Установите **начальную дату** и **конечную дату** с помощью элементов выбора даты.
2. Статистика обновится, отражая только активность в указанном диапазоне.

## Эффективное использование статистики

Вот несколько способов использования данных о загрузках:

- **Определение популярного контента** — сосредоточьте усилия на создании материалов в стиле и по темам, которые получают наибольшее количество загрузок.
- **Отслеживание новых выпусков** — после публикации нового курса отфильтруйте по дате, чтобы увидеть, как быстро он подхватывается.
- **Понимание вашей аудитории** — используйте список церквей и карту, чтобы узнать, кто использует ваш контент, и рассмотрите возможность связаться с этими сообществами.

:::info
Статистика основана на уникальных загрузках. Если одна и та же церковь загружает ресурс несколько раз, это считается как одна загрузка.
:::
