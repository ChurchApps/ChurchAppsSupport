# Управление курсами

<div class="article-intro">

Курс — это серия связанных уроков в рамках программы. Например, четырёхнедельная серия под названием «Добрый Самаритянин» будет курсом внутри более крупной программы детского служения. Это руководство описывает создание, редактирование и организацию курсов в разделе администрирования.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- [Создайте хотя бы одну программу](./managing-programs.md) в разделе администрирования
- Имейте права администратора для Lessons.church (см. [Обзор администрирования](./index.md))

</div>

## Создание курса

1. Перейдите в раздел **Администрирование** и разверните программу, в которую хотите добавить курс.
2. Нажмите **Добавить курс**.
3. Заполните следующие поля:
   - **Название** — отображаемое название курса (например, «Добрый Самаритянин»).
   - **ЧПУ-ссылка** — URL-дружественный идентификатор (например, «the-good-samaritan»).
   - **Описание** — краткое описание содержания курса и количества уроков.
   - **Изображение** — загрузите миниатюру для курса.
4. Сохраните курс.

## Редактирование курса

1. Нажмите на название курса в списке администрирования.
2. Обновите любое из полей по необходимости.
3. Сохраните изменения.

## Назначение категории

Если родительская программа использует категории курсов, вы можете назначить курсу категорию:

1. Откройте курс в режиме администрирования.
2. Выберите **Категорию** из выпадающего списка.
3. Сохраните.

Категории помогают пользователям просматривать программы, содержащие много курсов. Например, вы можете иметь категории «Ветхий Завет», «Новый Завет» и «Тематические».

## Управление уроками в курсе

Курсы содержат один или несколько уроков. Разверните курс в режиме администрирования, чтобы увидеть его уроки, или нажмите **Добавить урок** для создания нового. Подробности о создании содержимого уроков см. в разделе [Управление уроками](./managing-lessons.md).

## Загрузка ресурсов на уровне курса

Вы можете прикрепить загружаемые файлы, относящиеся ко всему курсу — например, обзор серии, промоматериалы или шаблон информационного письма для родителей.

1. Откройте курс в режиме администрирования.
2. Перейдите в раздел **Ресурсы**.
3. Загрузите ваши файлы.

:::tip
Составляйте описание курса с учётом церковных руководителей. Чёткое описание темы, целевой возрастной группы и количества уроков помогает церквям решить, подходит ли данная серия для них.
:::
