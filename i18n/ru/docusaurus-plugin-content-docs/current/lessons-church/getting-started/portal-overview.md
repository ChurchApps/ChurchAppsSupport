# Обзор портала

<div class="article-intro">

После входа в Lessons.church вы попадаете в **Портал** — панель управления вашей церкви для управления классами и расписаниями уроков. Именно здесь вы будете проводить большую часть времени при организации учебных программ для вашей церкви.

</div>

<div class="prereqs">
<h4>Прежде чем начать</h4>

- [Создайте учётную запись](./creating-account.md) и подключитесь к церкви
- Войдите на [Lessons.church](https://lessons.church)

</div>

## Что вы видите

Портал организован в две основные области:

- **Классы** (справа) — список классов вашей церкви. Нажмите на название класса, чтобы просмотреть его запланированные уроки. Здесь же можно добавлять новые классы или редактировать существующие.
- **Запланированные уроки** (слева) — после выбора класса в этой области отображаются запланированные для него уроки. Если вы ещё не настроили классы, вы увидите видеоинструкцию для начала работы.

## Что можно делать из портала

Из портала вы можете:

1. **Управлять классами** — добавлять, редактировать или удалять классы. Каждый класс можно связать с командой волонтёров (чтобы они видели предстоящие уроки в приложении B1) и группой родителей/учеников (чтобы они имели доступ к недавним урокам).
2. **Просматривать и управлять запланированными уроками** — видеть, какие уроки предстоят для каждого класса, и вносить изменения при необходимости.
3. **Подписываться на каналы классов** — нажмите значок канала рядом с классом, чтобы получить URL плейлиста, который можно использовать с совместимыми приложениями.
4. **Доступ к внешним поставщикам** — используйте страницу **Внешние поставщики** для настройки сторонних источников контента, поддерживающих Open Lesson Format. Это позволяет подключать уроки от других поставщиков помимо встроенной библиотеки Lessons.church.
5. **Доступ к разделу администрирования** — если у вас есть права администратора, в навигации появится пункт **Администрирование**. Раздел администрирования позволяет управлять программами, курсами и уроками на уровне контента.

## Навигация

Навигация портала включает:

- **Главная** — возврат на публичную главную страницу Lessons.church для просмотра контента.
- **Расписания** — основной вид портала для управления классами и расписаниями уроков.
- **Внешние поставщики** — настройка сторонних поставщиков уроков.
- **Администрирование** — управление программами, курсами и содержанием уроков (видно только при наличии прав администратора).

:::info
Пункт **Администрирование** отображается только при наличии прав на редактирование контента. Большинство сотрудников церкви будут использовать страницы **Расписания** и **Внешние поставщики** для повседневного управления.
:::

## Начало работы

Если вы впервые посещаете портал и ещё не создали классы, начните с нажатия **Добавить первый класс** в панели классов. Дайте классу название (например, «Дошкольники» или «3-5 класс») и сохраните. После этого вы можете начать [планирование уроков](../classrooms/scheduling-lessons.md).
