# Navegando por Grupos

<div class="article-intro">

A página pública de grupos da sua igreja facilita a descoberta de grupos que te interessam. Grupos são organizados por categoria, para que você possa rapidamente verificar diferentes tipos de grupos e encontrar o ideal.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Não é necessário login para navegar por grupos. No entanto, [fazer login](../getting-started/logging-in.md) dá acesso a funcionalidades adicionais na página de detalhes de cada grupo.

</div>

## Encontrando a Página de Grupos

Navegue até a página **Grupos** no site B1.church da sua igreja. Esta página é publicamente acessível, então você não precisa estar logado para navegar.

## Como os Grupos São Organizados

Grupos são exibidos por **categoria** (também chamada de rótulo). As categorias são configuradas pelos administradores da sua igreja e podem incluir rótulos como "Pequenos Grupos", "Ministérios", "Jovens" ou "Estudos Bíblicos". Cada categoria mostra todos os grupos que pertencem a ela.

## Lendo os Cartões dos Grupos

Cada grupo é exibido como um cartão com as seguintes informações:

- **Foto do grupo** -- Uma imagem de capa representando o grupo. Se nenhuma foto foi definida, uma imagem padrão é mostrada.
- **Nome do grupo** -- Exibido de forma destacada sobre a foto.
- **Horário de encontro** -- Quando o grupo normalmente se encontra (por exemplo, "Quartas às 19h"). Se nenhum horário está definido, um espaço reservado é mostrado.
- **Local de encontro** -- Onde o grupo se encontra. Se nenhum local está definido, um espaço reservado é mostrado.
- **Descrição** -- Uma prévia breve do que o grupo é sobre, limitada a poucas linhas. Clique no grupo para ler a descrição completa.

## Visualizando um Grupo

Clique em qualquer cartão de grupo para abrir a página completa de detalhes do grupo. Lá você encontrará a descrição completa, informações sobre os líderes do grupo, eventos próximos no calendário e uma forma de entrar em contato.

:::tip
Se você está procurando um tipo específico de grupo, verifique primeiro os títulos das categorias. Isso ajuda você a ir direto à seção que te interessa em vez de percorrer todos os grupos.
:::
