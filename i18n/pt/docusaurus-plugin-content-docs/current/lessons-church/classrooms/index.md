# Salas de Aula

<div class="article-intro">

As salas de aula são a base de como você organiza o ensino na sua igreja no Lessons.church. Uma sala de aula representa um espaço físico ou virtual onde as lições são ensinadas -- por exemplo, "Sala Pré-escolar", "Fundamental 1-2" ou "Estudo Bíblico para Adultos". Esta seção abrange a criação de salas de aula e o agendamento de lições para elas.

</div>

## Por Que as Salas de Aula São Importantes

Quando você configura salas de aula, cria uma estrutura clara para o ministério infantil, grupos de jovens ou qualquer ambiente de ensino da sua igreja. Cada sala de aula tem seu próprio cronograma, então seus voluntários sempre sabem exatamente o que ensinar e quando. Em vez de distribuir cronogramas em papel ou enviar e-mails de última hora, todos podem verificar o portal para ver o que está por vir.

## O Que Você Pode Fazer com Salas de Aula

- **Criar salas de aula ilimitadas** -- Configure quantas salas de aula sua igreja precisar. Não há limite, então você pode ter salas separadas para cada faixa etária, campus ou programa.
- **Agendar lições** -- Atribua lições específicas a cada sala de aula em datas específicas. Seu cronograma mostra tanto lições passadas quanto próximas de forma rápida.
- **Personalizar conteúdo** -- Adapte o conteúdo das lições para cada sala de aula para que os voluntários vejam apenas o que é relevante para sua faixa etária. Veja [Personalização](../customization/index.md) para detalhes.

## Como as Salas de Aula se Conectam

As salas de aula conectam o conteúdo das lições no Lessons.church ao cronograma de ensino real da sua igreja. Aqui está o fluxo de trabalho típico:

1. [Navegue](../browsing/index.md) pelos programas e estudos disponíveis no Lessons.church.
2. Configure suas salas de aula para corresponder às salas e faixas etárias da sua igreja.
3. Agende lições para cada sala de aula nas datas futuras.
4. Os voluntários abrem sua sala de aula para ver o que está agendado e se preparar.
