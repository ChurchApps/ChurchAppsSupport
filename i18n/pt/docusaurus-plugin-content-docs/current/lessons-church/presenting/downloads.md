# Downloads e Materiais

<div class="article-intro">

As lições no Lessons.church vêm com recursos para download que você pode imprimir ou compartilhar digitalmente. Esses materiais apoiam o conteúdo da lição e dão aos seus voluntários e alunos algo tangível para trabalhar durante e após a aula.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- [Navegue até uma lição](../browsing/viewing-lessons.md) ou abra uma lição agendada do [cronograma da sala de aula](../classrooms/scheduling-lessons.md)

</div>

## Tipos de Materiais

Dependendo da lição e do provedor, você pode encontrar recursos como:

- **Folhas de atividades** -- Exercícios práticos para os alunos completarem durante a aula.
- **Páginas para colorir** -- Páginas para impressão para faixas etárias mais jovens.
- **Guias do líder** -- Instruções detalhadas e informações de contexto para os professores.
- **PDFs complementares** -- Leituras adicionais, folhetos para levar para casa ou informativos para os pais.

Nem toda lição inclui todos esses materiais. Os materiais disponíveis dependem do que o provedor de conteúdo forneceu.

## Onde Encontrar Downloads

Os recursos estão disponíveis em três níveis:

- **Nível do programa** -- Recursos gerais que se aplicam à série de currículo inteira.
- **Nível do estudo** -- Materiais vinculados a um estudo específico dentro de um programa.
- **Nível da lição** -- Recursos para uma lição individual.

Para baixar materiais:

1. Navegue até a página da lição, estudo ou programa.
2. Procure a seção de **Downloads** ou **Recursos** na página.
3. Clique no nome do arquivo para baixá-lo no seu dispositivo.

Os arquivos são geralmente em formato PDF e prontos para impressão.

## Preparando-se para a Aula

Uma boa rotina é verificar os materiais para download alguns dias antes da aula:

1. Abra a próxima lição e revise os downloads disponíveis.
2. Imprima cópias suficientes de folhas de atividades e páginas para colorir para a frequência esperada.
3. Revise o guia do líder para que você esteja familiarizado com o fluxo da lição.
4. Separe os materiais para levar para casa para distribuir no final da aula.

:::info
Alguns provedores incluem todos os seus materiais no nível do estudo em vez de anexá-los a lições individuais. Se você não encontrar downloads em uma página de lição específica, verifique a página do estudo ou programa pai.
:::

:::tip
Crie uma lista de verificação semanal para seus voluntários que inclua baixar e imprimir materiais. Isso garante que nada seja esquecido antes do domingo de manhã.
:::
