# Gerenciando Programas

<div class="article-intro">

Programas são os contêineres de nível superior para currículo no Lessons.church. Um programa representa uma série completa de currículo -- por exemplo, "Ark Kids" ou "Youth Foundations". Este guia abrange a criação, edição e organização de programas na área de administração.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Tenha permissões de administrador no Lessons.church (veja a [Visão geral da Administração](./index.md))
- Prepare o nome do programa, descrição e imagem em miniatura

</div>

## Criando um Programa

1. Navegue até a área de **Administração**.
2. Clique em **Adicionar Programa**.
3. Preencha os seguintes campos:
   - **Nome** -- O nome de exibição do programa (por exemplo, "Ark Kids").
   - **Slug** -- Um identificador amigável para URL (por exemplo, "ark-kids"). Usado em endereços web.
   - **Descrição** -- Um breve resumo do que o programa abrange.
   - **Imagem** -- Envie uma imagem em miniatura que represente o programa.
4. Salve o programa.

Seu novo programa aparecerá na lista de programas da administração e, uma vez que tenha estudos e lições, no site público.

## Editando um Programa

1. Clique no nome do programa na lista de administração.
2. Atualize qualquer um dos campos (**Nome**, **Slug**, **Descrição** ou **Imagem**).
3. Salve suas alterações.

## Organizando Estudos Dentro de um Programa

Uma vez que um programa existe, você pode adicionar estudos a ele. Os estudos aparecem aninhados sob o programa na visualização de administração. Clique na seta de expansão ao lado de um programa para ver seus estudos, ou adicione um novo a partir daí. Veja [Gerenciando Estudos](./managing-studies.md) para detalhes.

## Categorias de Estudo

Se seu programa tem muitos estudos, você pode organizá-los em **categorias**. Categorias agrupam estudos por tema, faixa etária ou qualquer outro critério que você escolher. Isso facilita a navegação dos usuários em um programa grande.

## Enviando Recursos no Nível do Programa

Você pode anexar arquivos para download a um programa. Estes são recursos que se aplicam ao programa como um todo, em vez de um estudo ou lição específica -- por exemplo, um documento de escopo e sequência ou um manual do voluntário.

1. Abra o programa na visualização de administração.
2. Navegue até a seção de **Recursos**.
3. Envie seus arquivos.

## Limpando o Cache

Após fazer alterações em um programa, pode ser necessário limpar o cache para que as atualizações apareçam no site público. Use a opção **Limpar Cache** na área de administração para atualizar o conteúdo.

:::tip
Escolha um nome descritivo para o programa e escreva uma descrição clara. Esta é a primeira coisa que as igrejas veem ao navegar pela biblioteca de currículo, então um bom resumo ajuda a decidir se o programa é adequado para suas necessidades.
:::
