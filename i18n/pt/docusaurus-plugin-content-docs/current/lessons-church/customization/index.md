# Personalização

<div class="article-intro">

O Lessons.church fornece uma biblioteca de conteúdo de lições pronto para usar, mas cada igreja é diferente. Os recursos de personalização permitem que você adapte o conteúdo das lições para se adequar às suas salas de aula específicas, estilo de ensino e faixas etárias, para que seus voluntários vejam apenas o que é relevante para eles.

</div>

## Por Que Personalizar?

O conteúdo das lições no Lessons.church é projetado para ser flexível. Uma única lição pode incluir seções para múltiplas faixas etárias, atividades opcionais ou funções que não se aplicam a todas as salas de aula. Em vez de pedir aos seus voluntários que pulem mentalmente partes que não se aplicam, você pode personalizar a lição para que eles vejam apenas o que é relevante.

Por exemplo, uma lição pode incluir uma atividade de artesanato que funciona muito bem para pré-escolares, mas não é adequada para sua turma de quinto ano. Com a personalização, você pode ocultar essa seção para o grupo mais velho enquanto a mantém para o mais novo.

## O Que Você Pode Personalizar

- **Remover seções** -- Ocultar seções inteiras de uma lição que não se aplicam a uma sala de aula ou faixa etária específica.
- **Ocultar funções ou ações** -- Remover funções ou ações individuais dentro de uma seção se não forem necessárias.
- **Reordenar conteúdo** -- Reorganizar seções e ações para corresponder ao fluxo que você preferir para sua sala de aula.

## Específico por Sala ou Global

Quando você salva uma personalização, escolhe quão amplamente ela se aplica:

- **Sala de aula única** -- A personalização afeta apenas a sala de aula atual. Outras salas de aula veem o conteúdo original.
- **Todas as salas de aula** -- A personalização se aplica a todas as salas de aula da sua igreja.

Essa flexibilidade significa que você pode configurar diferentes visualizações da mesma lição para diferentes faixas etárias, ou fazer uma alteração que se aplica a todos de uma vez.

:::tip
Se você é novo na personalização, comece com uma única sala de aula para ver como funciona antes de aplicar alterações em todas as salas de aula.
:::
