# Navegando pelo Conteúdo

<div class="article-intro">

O Lessons.church oferece uma biblioteca gratuita de currículo para igrejas que qualquer pessoa pode explorar. Você não precisa de uma conta ou login para navegar pelo conteúdo disponível -- basta visitar o site e começar a explorar. Esta seção abrange todas as formas de encontrar e explorar currículo.

</div>

## A Página Inicial

Quando você chega ao Lessons.church pela primeira vez, a página inicial oferece uma visão geral rápida de tudo que está disponível:

- **Programas em destaque** -- Séries de currículo destacadas aparecem no topo, facilitando a descoberta de conteúdo popular ou recentemente atualizado.
- **Todos os programas** -- Abaixo da seção de destaques, você encontrará uma listagem completa de todos os programas de currículo disponíveis junto com estatísticas como o número de lições e estudos em cada um.
- **Conteúdo de provedores** -- Conteúdo tanto do Lessons.church quanto de provedores de currículo externos é exibido junto, oferecendo um único lugar para explorar.

## Formas de Encontrar Conteúdo

Existem várias maneiras de navegar pela biblioteca de currículo:

1. **Navegar por programa** -- Clique em qualquer programa na página inicial para ver seus estudos e lições. Esta é a melhor abordagem quando você quer explorar uma série de currículo específica. Veja [Programas e Estudos](./programs-and-studies.md) para detalhes.

2. **Pesquisar** -- Use a página de pesquisa para encontrar lições por palavra-chave em toda a biblioteca. Isso é útil quando você sabe um tema, mas não tem certeza qual programa o aborda. Veja [Pesquisando](./searching.md) para detalhes.

3. **Explorar provedores externos** -- Algum conteúdo vem de provedores de currículo de terceiros que publicam no Open Lesson Format. Você pode navegar pelos programas e estudos deles da mesma forma que o conteúdo integrado. Veja [Provedores Externos](./external-providers.md) para detalhes.

## Login Não Necessário

A navegação é completamente aberta. Você pode visualizar programas, estudos, lições, materiais para download e vídeos sem criar uma conta. Se decidir que quer agendar lições para suas salas de aula ou personalizar conteúdo, você pode [criar uma conta gratuita](../getting-started/creating-account.md) nesse momento.

:::tip
Comece explorando alguns programas na página inicial para ter uma noção de como o currículo é organizado antes de mergulhar em um estudo específico.
:::
