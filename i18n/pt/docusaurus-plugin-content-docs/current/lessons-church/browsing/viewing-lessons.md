# Visualizando Lições

<div class="article-intro">

Cada lição no Lessons.church é uma sessão de ensino completa com conteúdo estruturado, materiais para download e suporte para múltiplas faixas etárias. Este guia explica como navegar e usar uma página de lição para se preparar para sua aula.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Entenda como o conteúdo é organizado lendo [Programas e Estudos](./programs-and-studies.md)
- Navegue até um programa e estudo para encontrar a lição que deseja visualizar

</div>

## Chegando a uma Lição

Para visualizar uma lição, navegue pela hierarquia de conteúdo:

1. Clique em um **programa** na página inicial.
2. Selecione um **estudo** na página do programa.
3. Clique na **lição** que deseja visualizar na página do estudo.

## O Que Você Verá

A página da lição é organizada em várias áreas:

### Vídeo

Se a lição inclui um vídeo, ele aparece no topo da página. Clique para reproduzi-lo diretamente no seu navegador. Os vídeos são frequentemente usados como segmento principal de ensino, especialmente para ministério infantil.

### Seções e Funções

Cada lição é dividida em **seções** (por exemplo, "Abertura", "História Bíblica", "Atividade", "Encerramento"). Dentro de cada seção, o conteúdo é organizado por **função**:

- **Líder** -- O professor ou apresentador principal
- **Auxiliar** -- Voluntários de apoio
- Outras funções personalizadas definidas pelo criador do currículo

Cada função lista suas **ações**, que dizem ao voluntário exatamente o que fazer:

- **Falar** -- Roteiro ou pontos para falar em voz alta
- **Fazer** -- Instruções físicas ou atividades a realizar
- **Nota** -- Informação de contexto ou dicas para o voluntário
- **Reproduzir** -- Indicações para reproduzir um clipe de vídeo ou áudio

### Materiais para Download

As lições podem incluir arquivos que você pode baixar e imprimir, como:

- Folhetos em PDF
- Folhas de atividades
- Páginas para colorir
- Instruções de artesanato
- Folhetos para levar para casa para os pais

Procure os links de download na página da lição para acessar esses recursos. Veja [Downloads e Materiais](../presenting/downloads.md) para mais detalhes sobre a preparação de recursos impressos.

## Alternando Entre Ambientes

Muitas lições suportam múltiplos **ambientes**, que representam diferentes faixas etárias como Pré-escolar, Fundamental ou Pré-adolescente. Os ambientes permitem que o mesmo tema de lição seja adaptado ao nível apropriado para cada grupo.

Para alternar entre ambientes:

1. Procure o **seletor de ambiente** na página da lição.
2. Clique em um nome de ambiente diferente (por exemplo, mude de **Fundamental** para **Pré-escolar**).
3. A página é atualizada para mostrar o conteúdo, roteiros e materiais projetados para aquela faixa etária.

:::tip
Se você ensina para múltiplas faixas etárias, revise cada ambiente antes da sua sessão para ver como o conteúdo é adaptado para diferentes níveis.
:::
