# Programas e Estudos

<div class="article-intro">

Todo currículo no Lessons.church é organizado em uma hierarquia clara. Entender essa estrutura ajudará você a encontrar o conteúdo certo para sua igreja rapidamente e navegar entre lições relacionadas com facilidade.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Familiarize-se com a visão geral de [Navegando pelo Conteúdo](./index.md) para entender as diferentes formas de encontrar conteúdo

</div>

## Hierarquia de Conteúdo

O conteúdo é organizado do mais amplo ao mais específico:

- **Provedor** -- A organização que publica o currículo (Lessons.church ou um provedor de terceiros).
- **Programa** -- Uma série de currículo, como "Ark Kids" ou "NextLevel". Cada programa é uma coleção de estudos relacionados projetados em torno de um tema ou faixa etária.
- **Estudo** -- Uma unidade dentro de um programa, como "O Bom Samaritano". Cada estudo contém um conjunto de lições que devem ser ensinadas juntas, geralmente ao longo de várias semanas.
- **Lição** -- Uma sessão de ensino individual dentro de um estudo. Cada lição inclui seções, funções, ações e materiais para download.
- **Ambiente** -- Uma versão da lição para uma faixa etária específica, como Pré-escolar ou Fundamental. Os ambientes permitem adaptar a mesma lição para diferentes públicos.

## Navegando pelos Programas

Para explorar os programas:

1. Visite a página inicial do Lessons.church.
2. Percorra a lista de programas disponíveis ou veja os programas em destaque no topo.
3. Clique no **nome de um programa** ou na sua imagem para abrir a página do programa.

Cada página de programa inclui:

- **Descrição** -- Uma visão geral do que o currículo abrange e para quem é projetado.
- **Imagem do programa** -- Uma identidade visual para a série.
- **Trailer em vídeo** -- Alguns programas incluem uma prévia em vídeo curta para que você possa ver o estilo e qualidade do conteúdo antes de se comprometer.
- **Estudos** -- Uma listagem de todos os estudos no programa, que podem estar organizados em categorias.

## Navegando pelos Estudos

A partir de uma página de programa:

1. Percorra a lista de estudos. Se o programa organiza estudos em categorias, você os verá agrupados de acordo.
2. Clique no **nome de um estudo** para abrir a página do estudo.
3. A página do estudo exibe todas as lições incluídas naquele estudo, listadas na ordem de ensino.

A partir daqui, você pode clicar em qualquer lição para ver seu conteúdo completo, incluindo seções, funções e materiais para download. Veja [Visualizando Lições](./viewing-lessons.md) para um passo a passo detalhado.

:::info
Programas e estudos são gerenciados por provedores de currículo. Se você é um criador de conteúdo, pode gerenciar seus próprios programas e estudos através da área de [Administração](../admin/index.md).
:::
