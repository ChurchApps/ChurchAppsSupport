# Relatórios de Doações

<div class="article-intro">

O B1 Admin oferece várias maneiras de visualizar e analisar os dados de contribuições da sua igreja. A página de Resumo de Doações fornece uma visão geral visual com gráficos e filtros, enquanto a seção de Relatórios oferece um relatório de Resumo de Doações mais detalhado. Use essas ferramentas para acompanhar tendências de contribuições, preparar-se para reuniões de diretoria ou conciliar seus registros.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Certifique-se de que as doações foram [registradas em lotes](recording-donations.md) ou [importadas do Stripe](stripe-import.md)
- Verifique se seus [fundos](funds.md) estão configurados corretamente para que as doações sejam categorizadas adequadamente

</div>

## Página de Resumo de Doações

A página de **Resumo** é a primeira coisa que você vê ao clicar em **Doações** na barra lateral. Ela fornece uma visão geral de alto nível da sua atividade de contribuições.

1. Navegue até **Doações** na barra lateral para abrir a página de Resumo.
2. Use o **filtro de intervalo de datas** para selecionar o período que deseja revisar. Defina a data mais antiga no topo e a data mais recente na parte inferior.
3. A página exibe um gráfico de contribuições semanais para que você possa ver tendências rapidamente.
4. Clique em **Download** para exportar um arquivo CSV com o valor total doado, a semana em que foi doado e o fundo para o qual foi destinado.

:::info
A página de Resumo mostra dados agregados de contribuições. Ela não inclui nomes individuais de doadores. Para detalhes por doador, use a página de [Lotes](batches.md).
:::

## Visualizando Detalhes por Doador

Para uma análise de quem doou, quanto e para qual fundo:

1. Navegue até **Doações > Lotes**.
2. Clique no **nome de um lote** para abri-lo.
3. A página de detalhes do lote lista cada doação com o nome do doador, valor, fundo, data e método de pagamento.
4. Clique no **nome de um doador** para ver uma análise de quantas vezes ele doou e quanto a cada vez.
5. Clique em um **ID de doação** para abrir um painel lateral com os detalhes completos daquela doação individual.
6. Clique em **Download** para exportar um CSV com todas as informações de doadores e doações daquele lote.

## Relatório de Resumo de Doações

O B1 Admin também inclui um relatório de **Resumo de Doações** na seção de Relatórios:

1. Clique em **Relatórios** na barra lateral.
2. Selecione o relatório de **Resumo de Doações**.
3. Escolha seus filtros (intervalo de datas, fundo, campus, etc.) e execute o relatório.

## Exportando Dados

Você pode exportar dados de doações de vários lugares:

- **Página de Resumo** -- baixe um CSV dos totais semanais de contribuições por fundo
- **Página de detalhes do lote** -- baixe um CSV das doações individuais com detalhes do doador
- **Página de detalhes do fundo** -- baixe o histórico de doações para um fundo específico

:::tip
Para relatórios de fim de ano, combine a exportação da página de Resumo com a ferramenta de [Declarações de Contribuições](giving-statements.md) para obter tanto as tendências agregadas quanto as declarações individuais dos doadores.
:::
