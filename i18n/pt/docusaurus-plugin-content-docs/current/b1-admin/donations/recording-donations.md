# Registrando Doações

<div class="article-intro">

O registro de doações no B1 Admin é feito pelo sistema de Lotes. Você cria um lote para representar uma coleta (como uma oferta de domingo), e então adiciona doações individuais a esse lote. Isso mantém seus registros de contribuição organizados e fáceis de conciliar.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Configure seus [fundos](funds.md) para poder atribuir doações às categorias corretas
- Crie um [lote](batches.md) para conter as doações que você está prestes a inserir
- Certifique-se de que os doadores estão no seu [diretório de pessoas](../people/adding-people.md) para que você possa buscá-los ao inserir contribuições

</div>

## Criando um Lote e Adicionando Doações

1. No **B1 Admin**, clique em **Doações** na barra lateral, depois clique em **Lotes**.
2. Clique em **Adicionar Lote**.
3. Insira um nome para o lote (por exemplo, "Oferta de Domingo - 5 Jan") e selecione a data. Clique em **Salvar**.
4. Seu novo lote aparece na lista mostrando zero doações e $0,00.
5. Clique no **nome do lote** para abri-lo.

## Inserindo Doações Individuais

1. Na página de detalhes do lote, digite o nome do doador no **campo de pesquisa** para encontrá-lo.
2. Após selecionar uma pessoa, o formulário de entrada de doação aparece com campos para **Data**, **Método de Pagamento**, **Fundo**, **Valor** e **Número do Cheque**.
3. Preencha os detalhes e clique em **Adicionar Doação**.
4. A doação é adicionada à tabela abaixo, e o formulário é reiniciado para que você possa inserir a próxima.

:::tip
Você pode inserir rapidamente várias doações seguidas sem sair da página do lote. O formulário é reiniciado após cada entrada para que você possa processar uma pilha de cheques ou envelopes de forma eficiente.
:::

## Dividindo uma Doação Entre Múltiplos Fundos

Às vezes, um único doador contribui para mais de um fundo em uma transação. Para lidar com isso:

1. Clique no botão **Editar** na linha da doação.
2. No formulário de edição, adicione valores para diferentes fundos. O total será calculado automaticamente a partir dos valores individuais de cada fundo.
3. Clique em **Salvar** para atualizar a doação.

:::info
Dividir doações entre fundos é comum quando um doador emite um único cheque destinado a múltiplos propósitos, como Fundo Geral e Missões.
:::

## Editando ou Removendo Doações

Para editar uma doação, clique no botão **Editar** na sua linha no lote. Você pode alterar a data, valor, fundo, método de pagamento ou qualquer outro detalhe. Clique em **Salvar** quando terminar.

:::tip
O cabeçalho da página do lote é atualizado automaticamente para mostrar o número total de doações e o valor combinado em dólares à medida que você adiciona ou edita entradas. Use isso para conciliar com seu comprovante de depósito.
:::
