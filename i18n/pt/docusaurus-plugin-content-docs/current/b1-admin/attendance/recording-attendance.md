# Registrar Presença

<div class="article-intro">

Uma vez que seus campi, horários de culto e grupos estejam configurados, você pode registrar manualmente a presença após cada reunião. O B1 Admin organiza a presença em torno de **sessões** -- uma sessão por grupo por data de reunião. Você cria a sessão, marca quem compareceu, e os dados alimentam diretamente seus relatórios de presença.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Seus campi, horários de culto e grupos devem estar configurados. Consulte [Configuração de Presença](setup.md) se ainda não fez isso.
- Os grupos que deseja acompanhar devem ter o **Acompanhamento de Presença** ativado. Consulte [Configuração de Presença](setup.md) para detalhes.

</div>

## Criando uma Sessão

Uma sessão representa uma ocorrência de uma reunião de grupo -- por exemplo, sua classe de crianças de 1 a 3 anos em um domingo específico.

1. Abra o **B1 Admin** e clique em **Grupos** na barra lateral.
2. Selecione o grupo para o qual deseja registrar a presença.
3. Clique na aba **Sessões**.
4. Clique em **Nova** para criar uma nova sessão.
5. Se o grupo estiver atribuído a um horário de culto, escolha o **Horário de Culto**. Se for um grupo não programado, este campo não aparecerá.
6. Selecione a **Data da Sessão** -- pode ser hoje, uma data passada ou uma data futura.
7. Clique em **Salvar**.

:::tip
Você pode criar sessões para datas passadas para recuperar presenças que ainda não registrou, ou criá-las antecipadamente para que estejam prontas quando seu grupo se reunir.
:::

## Marcando Presença

Após salvar a sessão, os membros do grupo aparecem no lado direito da página.

1. Marque a caixa ao lado de cada pessoa que compareceu.
2. As alterações são salvas automaticamente -- não há um botão adicional de Salvar para as marcações de presença.

:::info
Apenas os membros atuais do grupo aparecem na lista da sessão. Se alguém compareceu mas ainda não está no grupo, [adicione-o ao grupo](../groups/group-members.md) primeiro e depois registre sua presença.
:::

## Exportando Presença para uma Planilha

Você pode baixar um registro da sessão como arquivo CSV para usar no Excel, Numbers ou Google Sheets.

1. Abra a sessão que deseja exportar.
2. Clique no botão **Baixar CSV**.
3. Abra o arquivo baixado no seu aplicativo de planilhas.

## Visualizando Presença Registrada

Após registrar sessões, os dados aparecem nos seus relatórios de presença.

- **Aba Presença** -- mostra tendências gerais da igreja ao longo do tempo. Consulte [Acompanhamento de Presença](tracking-attendance.md).
- **Aba Grupos** -- mostra a presença detalhada por grupo individual.

:::tip
Se uma sessão que você acabou de criar não aparecer nos relatórios imediatamente, certifique-se de que a data da sessão esteja dentro do período selecionado nos filtros do relatório.
:::
