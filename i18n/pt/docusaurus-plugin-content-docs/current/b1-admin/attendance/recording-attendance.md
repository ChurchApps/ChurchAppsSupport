# Registrando Presença

<div class="article-intro">

Uma vez que seus campi, horários de serviço e grupos estão configurados, você pode registrar manualmente a presença após cada reunião. O B1 Admin organiza presença em torno de **sessions** -- uma sessão por grupo por data de reunião. Você cria a sessão, marca quem compareceu, e os dados alimentam diretamente seus relatórios de presença.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Seus campi, horários de serviço e grupos devem estar configurados. Veja [Configuração de Presença](setup.md) se você ainda não tiver feito isso.
- Os grupos que você quer rastrear devem ter **Track Attendance** ativado. Veja [Configuração de Presença](setup.md) para detalhes.

</div>

## Criando uma Sessão

Uma sessão representa uma ocorrência de uma reunião de grupo -- por exemplo, sua classe de K-3ª série em um domingo específico.

1. Abra **B1 Admin** e clique em **Groups** na barra lateral.
2. Selecione o grupo para o qual você quer registrar presença.
3. Clique na aba **Sessions**.
4. Clique em **New** para criar uma nova sessão.
5. Se o grupo é atribuído a um horário de serviço, escolha o **Service Time**. Se for um grupo não programado, este campo não aparecerá.
6. Selecione a **Session Date** -- isso pode ser hoje, uma data anterior ou uma data futura.
7. Clique em **Save**.

:::tip
Você pode criar sessões para datas anteriores para recuperar a presença que você ainda não registrou, ou criá-las com antecedência para que estejam prontas quando seu grupo se reunir.
:::

## Marcando Presença

Após salvar a sessão, os membros do grupo aparecem no lado direito da página.

1. Marque a caixa ao lado de cada pessoa que compareceu.
2. As alterações são salvas automaticamente -- não há botão Save adicional para marcas de presença.

:::info
Apenas membros do grupo atuais aparecem na lista de sessões. Se alguém compareceu mas ainda não está no grupo, [adicione-o ao grupo](../groups/group-members.md) primeiro, então registre sua presença.
:::

## Exportando Presença para uma Planilha

Você pode baixar um registro da sessão como um arquivo CSV para usar em Excel, Numbers ou Google Sheets.

1. Abra a sessão que você quer exportar.
2. Clique no botão **Download CSV**.
3. Abra o arquivo baixado em seu aplicativo de planilha.

## Visualizando Presença Registrada

Após registrar sessões, os dados aparecem em seus relatórios de presença.

- **Aba Attendance** -- mostra tendências em toda a igreja ao longo do tempo. Veja [Rastreando Presença](tracking-attendance.md).
- **Aba Groups** -- mostra presença dividida por grupo individual.

:::tip
Se uma sessão que você criou não aparecer nos relatórios imediatamente, certifique-se de que a data da sessão está dentro do intervalo de datas selecionado nos filtros do relatório.
:::
