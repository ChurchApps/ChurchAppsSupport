# Listas Salvas

<div class="article-intro">

**Listas Salvas** permitem que você armazene uma consulta de pesquisa sob um nome e a execute novamente com um clique a qualquer momento. Crie uma lista uma vez -- "Membros Ativos no Campus Norte", "Visitantes nos últimos 90 dias" ou qualquer filtro avançado -- e ela permanece em sua barra lateral para que você nunca tenha que reconstruir a consulta novamente.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Execute pelo menos uma busca na seção Pessoas para que você tenha resultados para salvar. Veja [Procurando Pessoas](./searching-people.md) ou [Pesquisa com IA](./ai-search.md).

</div>

## Salvando uma Pesquisa como uma Lista

1. Execute uma busca na página **Pessoas** -- uma busca rápida, uma busca com IA ou uma busca de filtro avançado.
2. Quando os resultados aparecerem, um botão **Salvar como Lista** (ícone de marcador) aparece no cabeçalho da página Pessoas.
3. Clique em **Salvar como Lista**.
4. Digite um **Nome** para a lista (obrigatório).
5. Opcionalmente, digite uma **Categoria** para agrupar listas relacionadas na barra lateral (por exemplo, "Alcance" ou "Membro").
6. Clique em **Salvar**.

A lista agora é salva na conta da sua igreja e aparecerá no painel **Listas Salvas** no lado esquerdo da página Pessoas.

:::info
As listas são compartilhadas em sua igreja -- qualquer pessoa com acesso a Pessoas pode vê-las e carregá-las. Apenas a equipe com a permissão apropriada pode renomear ou deletar listas.
:::

## Carregando uma Lista Salva

No painel **Listas Salvas** no lado esquerdo da página Pessoas, clique em qualquer nome de lista. A busca é executada instantaneamente com a consulta armazenada, atualizando a tabela de resultados.

As listas são agrupadas por categoria (se você atribuiu uma), com listas sem categoria aparecendo por último.

:::tip
As listas são **consultas ao vivo**, não snapshots. Toda vez que você carrega uma lista, a busca é executada novamente com seus dados atuais, para que novas pessoas que agora correspondem aos critérios apareçam.
:::

## Renomeando e Deletando Listas

Equipe com permissão de gerenciar pode renomear ou deletar qualquer lista diretamente do painel Listas Salvas.

- Clique no **ícone de lápis** próximo a um nome de lista para renomeá-la.
- Clique no **ícone de lixo** para deletá-la (você será solicitado a confirmar).

## Casos de Uso

| Cenário | Como construir a lista |
|---|---|
| Todos os visitantes | Busca avançada: Status de Membro = Visitante |
| Membros em um campus específico | Busca avançada: Campus = [seu campus] |
| Pessoas sem endereço de e-mail | Busca avançada: E-mail está vazio |
| Resultados de uma pergunta com IA | Faça uma pergunta em Pesquisa com IA, depois salve |
