# Relatório de Aniversários

<div class="article-intro">

O Relatório de Aniversários ajuda você a manter contato com sua congregação mostrando os membros com aniversários próximos. Use-o para planejar celebrações, enviar cartões ou fazer anúncios durante os cultos.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Certifique-se de que as datas de nascimento estejam inseridas no perfil de cada pessoa. Veja [Adicionar Pessoas](../people/adding-people.md) para detalhes.
- Você precisa das [permissões](../settings/roles-permissions.md) apropriadas para acessar relatórios

</div>

## Executando o Relatório

1. Navegue até **Relatórios** no menu principal.
2. Clique no cartão **Relatório de Aniversários**.
3. Defina o **intervalo de datas** usando os filtros de data para definir quais aniversários você deseja ver.
4. O relatório exibirá uma lista de membros com aniversários naquele intervalo.

## O Que o Relatório Mostra

O Relatório de Aniversários exibe:

- **Nome do membro** -- O nome da pessoa no banco de dados da sua igreja.
- **Data de nascimento** -- A data do aniversário.

Os resultados são listados dentro do intervalo de datas que você selecionou, facilitando ver quem tem aniversário chegando.

## Usos Práticos

- **Anúncios semanais** -- Execute o relatório para a próxima semana e inclua os aniversários no seu boletim ou avisos.
- **Cartões de aniversário** -- Gere uma lista para o mês e peça à sua equipe que envie cartões de aniversário pessoais.
- **Líderes de pequenos grupos** -- Líderes de grupo podem executar o relatório para lembrar dos aniversários dos membros do seu grupo.
- **Celebrações de marcos** -- Filtre por intervalos de datas específicos para planejar celebrações de aniversários marcantes.

:::tip
Execute este relatório no início de cada mês para se planejar com antecedência. Você pode definir o intervalo de datas para cobrir o mês inteiro para ter tempo de preparar cartões ou anúncios.
:::

:::info
Os dados de aniversário vêm do campo de data de nascimento no perfil de cada pessoa. Certifique-se de que seus [registros de membros](../people/adding-people.md) estejam atualizados para resultados precisos.
:::
