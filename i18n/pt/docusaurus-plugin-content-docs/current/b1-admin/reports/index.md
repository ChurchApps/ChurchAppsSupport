# Relatórios

<div class="article-intro">

A página Relatórios oferece acesso rápido a relatórios pré-construídos que ajudam você a entender os dados da sua igreja. Use esses relatórios para acompanhar tendências de presença, monitorar atividades de doação e manter-se conectado com sua congregação através de celebrações de aniversário.

</div>

## Relatórios disponíveis

A página Relatórios exibe cinco cartões de relatório. Clique em qualquer cartão para abrir o relatório com seus filtros.

- **[Relatório de Aniversários](./birthday-report)** -- Veja os próximos aniversários dos membros da sua congregação.
- **[Tendência de Presença](./attendance-reports)** -- Veja como os padrões de presença mudam ao longo do tempo nos seus cultos.
- **[Presença por Grupo](./attendance-reports)** -- Veja os totais de presença por grupo para um intervalo de datas selecionado.
- **[Presença Diária por Grupo](./attendance-reports)** -- Obtenha uma análise dia a dia dos dados de presença dos grupos.
- **[Resumo de Doações](./donation-summary)** -- Revise os totais de doações e tendências para planejamento financeiro.

## Usando os Relatórios

1. Navegue até **Reports** no menu principal.
2. Clique em um **cartão de relatório** para abri-lo.
3. Use os **filtros** no topo de cada relatório para restringir os dados por intervalo de datas, grupo, culto ou outros critérios.
4. Revise os resultados exibidos no relatório.

:::tip
Execute relatórios regularmente para ficar por dentro das tendências de presença, próximos aniversários e padrões de doação. Esses dados ajudam você a planejar ministérios e ações de alcance de forma mais eficaz.
:::

:::info
Os dados dos relatórios dependem de uma entrada de dados consistente. Certifique-se de que sua equipe está [registrando presença](../attendance/tracking-attendance.md), [registrando doações](../donations/recording-donations.md) e mantendo os [registros de pessoas](../people/adding-people.md) atualizados para obter os resultados mais precisos.
:::
