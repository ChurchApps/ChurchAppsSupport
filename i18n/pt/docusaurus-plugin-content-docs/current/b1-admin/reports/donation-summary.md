# Resumo de Doações

<div class="article-intro">

O relatório de Resumo de Doações oferece uma visão clara da atividade de doações da sua igreja. Use-o para acompanhar totais de doações, monitorar tendências de contribuições e preparar relatórios financeiros para sua equipe de liderança.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Certifique-se de que as doações estejam sendo [registradas consistentemente](../donations/recording-donations.md) no seu sistema
- Você precisa das [permissões](../settings/roles-permissions.md) apropriadas para acessar relatórios financeiros

</div>

## Executando o Relatório

1. Navegue até **Relatórios** no menu principal.
2. Clique no cartão **Resumo de Doações**.
3. Defina o **intervalo de datas** para definir o período do relatório.
4. Aplique filtros adicionais como **fundo** ou outros critérios para refinar os resultados.
5. O relatório exibirá o resumo de doações para o período selecionado.

## O Que o Relatório Mostra

O relatório de Resumo de Doações inclui:

- **Total de doações** -- O valor total de contribuições para o período selecionado.
- **Contagem de doadores** -- O número de doadores únicos que contribuíram.
- **Valores por fundo** -- Um detalhamento das doações por diferentes fundos, se aplicável.

## Usos Práticos

- **Planejamento financeiro** -- Revise tendências de doações para prever receitas futuras e planejar seu orçamento.
- **Relatórios para a diretoria** -- Gere resumos para reuniões regulares da diretoria ou liderança com dados claros de doações.
- **Revisões de final de ano** -- Execute o relatório para o ano inteiro para preparar resumos financeiros anuais.
- **Acompanhamento de campanhas** -- Filtre por fundo para acompanhar o progresso de campanhas especiais de doação ou projetos de construção.
- **Análise sazonal** -- Compare doações em diferentes períodos para entender padrões sazonais.

:::tip
Execute este relatório mensalmente para se manter informado sobre a saúde financeira da sua igreja. Comparar dados mês a mês ajuda a identificar tendências com antecedência.
:::

:::warning
Os dados de doações vêm das contribuições registradas no seu sistema. Certifique-se de que as doações sejam inseridas consistentemente -- seja por doações online, processamento de cheques ou entrada manual -- para os relatórios mais precisos. Veja [Registrar Doações](../donations/recording-donations.md) para detalhes.
:::
