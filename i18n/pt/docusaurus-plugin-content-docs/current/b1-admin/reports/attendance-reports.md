# Relatórios de Frequência

<div class="article-intro">

O B1 Admin oferece três relatórios de frequência para ajudá-lo a entender como as pessoas estão participando dos seus cultos e grupos. Cada relatório oferece uma perspectiva diferente dos seus dados de frequência, desde tendências gerais até detalhamentos diários.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Certifique-se de que a frequência está sendo [registrada consistentemente](../attendance/tracking-attendance.md) para seus cultos e grupos
- Certifique-se de que seus [grupos](../groups/creating-groups.md) e cultos estejam configurados no B1 Admin
- Você precisa das [permissões](../settings/roles-permissions.md) apropriadas para acessar relatórios

</div>

## Tendência de Frequência

O relatório de Tendência de Frequência mostra como a frequência muda ao longo do tempo para seus cultos.

1. Navegue até **Relatórios** e clique no cartão **Tendência de Frequência**.
2. Defina o **intervalo de datas** para definir o período.
3. Selecione um **culto** ou **grupo** para filtrar os resultados.
4. O relatório exibe uma linha de tendência mostrando as contagens de frequência ao longo do período selecionado.

Este relatório é útil para identificar padrões como quedas sazonais, tendências de crescimento ou o impacto de eventos especiais.

## Frequência por Grupo

O relatório de Frequência por Grupo mostra os totais de frequência por grupo para um intervalo de datas selecionado.

1. Navegue até **Relatórios** e clique no cartão **Frequência por Grupo**.
2. Defina o **intervalo de datas** para o relatório.
3. Selecione o(s) **grupo(s)** que deseja revisar.
4. O relatório exibe os totais de frequência para cada grupo.

Use este relatório para comparar a frequência entre grupos e identificar quais grupos estão crescendo ou precisam de atenção.

## Frequência Diária por Grupo

O relatório de Frequência Diária por Grupo fornece um detalhamento dia a dia dos dados de frequência dos seus grupos.

1. Navegue até **Relatórios** e clique no cartão **Frequência Diária por Grupo**.
2. Defina o **intervalo de datas** para o relatório.
3. Selecione o(s) **grupo(s)** que deseja revisar.
4. O relatório mostra os números de frequência para cada dia individual dentro do intervalo.

Este relatório oferece detalhamento granular, útil para entender variações semana a semana ou identificar dias específicos com frequência excepcionalmente alta ou baixa.

:::tip
Use o relatório de Tendência de Frequência para uma visão geral e o relatório de Frequência Diária por Grupo quando precisar analisar datas específicas.
:::

## Usos Práticos

- **Planejamento** -- Use tendências de frequência para planejar assentos, equipe e recursos para próximos cultos.
- **Alcance** -- Identifique padrões de declínio de frequência cedo para que você possa acompanhar os membros.
- **Relatórios para a diretoria** -- Inclua dados de frequência nos seus relatórios regulares de liderança para mostrar a saúde do ministério.
- **Avaliação de eventos** -- Compare a frequência antes e depois de eventos especiais para medir seu impacto.

:::warning
Os dados de frequência são registrados através dos seus processos de check-in de grupos e cultos. Se a frequência não estiver sendo registrada consistentemente, seus relatórios não refletirão com precisão a participação real. Veja [Registro de Frequência](../attendance/tracking-attendance.md) para instruções de configuração.
:::
