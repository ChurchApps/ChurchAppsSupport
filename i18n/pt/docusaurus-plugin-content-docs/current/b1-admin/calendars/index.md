# Calendários

<div class="article-intro">

A seção Calendários no B1 Admin permite criar calendários curados para sua igreja, reunindo eventos de seus grupos. Os líderes de grupo gerenciam os eventos de seu próprio grupo, e como administrador você pode criar calendários que exibem esses eventos no site B1.church para sua congregação ver.

</div>

## Visualizando Seus Calendários

1. No B1 Admin, navegue até **Website** e depois até a seção **Calendars**.
2. A página Calendários lista todos os seus calendários com seu nome e status.
3. Cada calendário também mostra os grupos associados a ele.

## O Que Você Pode Fazer

- **Adicionar um novo calendário** -- Clique no botão adicionar para criar um calendário para um ministério específico, tipo de evento ou propósito. Consulte [Creating Calendars](creating-calendars) para instruções passo a passo.
- **Editar um calendário** -- Clique em qualquer calendário para abri-lo e gerenciar quais grupos estão incluídos nele.
- **Publicar calendários** -- Os calendários podem ser exibidos no site B1.church para que os visitantes possam ver os próximos eventos em sua igreja.
- **Criar calendários curados** -- Extraia eventos de vários grupos em uma única visualização de calendário para uma visão abrangente de tudo o que está acontecendo em sua igreja. Consulte [Curated Calendar](curated-calendar) para detalhes.

:::info
Os calendários funcionam em estreita colaboração com [Groups](../groups/creating-groups.md). Quando os grupos têm eventos agendados, esses eventos podem ser incluídos em seus calendários. Consulte a página [Creating Calendars](creating-calendars) para aprender como conectar grupos aos calendários.
:::
