# Calendários

<div class="article-intro">

A seção Calendários no B1 Admin permite que você crie e gerencie calendários de eventos para sua igreja. Você pode configurar calendários para diferentes ministérios, agregar eventos de vários grupos e publicar calendários no seu site B1.church para que sua congregação possa visualizá-los.

</div>

## Visualizando seus calendários

1. No B1 Admin, navegue até a seção **Calendários**.
2. A página de Calendários lista todos os seus calendários com nome e status.
3. Cada calendário também mostra os grupos associados a ele.

## O que você pode fazer

- **Adicionar um novo calendário** -- Clique no botão adicionar para criar um calendário para um ministério, tipo de evento ou finalidade específica. Consulte [Criando calendários](creating-calendars) para instruções passo a passo.
- **Editar um calendário** -- Clique em qualquer calendário para abri-lo e gerenciar seus eventos e grupos associados.
- **Publicar calendários** -- Os calendários podem ser exibidos no seu site B1.church para que os visitantes possam ver os próximos eventos da sua igreja.
- **Criar calendários curados** -- Reúna eventos de vários grupos em uma única visualização de calendário para ter uma visão abrangente de tudo o que está acontecendo na sua igreja. Consulte [Calendário curado](curated-calendar) para mais detalhes.

:::info
Os calendários trabalham em conjunto com os [Grupos](../groups/creating-groups.md). Quando os grupos têm eventos agendados, esses eventos podem ser incluídos nos seus calendários. Consulte a página [Criando calendários](creating-calendars) para saber como conectar grupos aos calendários.
:::
