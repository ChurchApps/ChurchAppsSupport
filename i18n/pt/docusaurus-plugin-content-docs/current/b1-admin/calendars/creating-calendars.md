# Criando Calendários

<div class="article-intro">

Criar um calendário no B1 Admin oferece um local central para gerenciar eventos para um ministério, departamento ou sua igreja inteira. Cada calendário pode puxar eventos de um ou mais grupos, tornando fácil manter tudo organizado.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Configure os [grupos](../groups/creating-groups.md) cujos eventos você quer incluir em seu calendário
- Você precisa de acesso administrativo à seção Calendários no B1 Admin

</div>

## Criando um Novo Calendário

1. Navegue até a seção **Calendars** no B1 Admin.
2. Clique em **Add Calendar**.
3. Insira um **name** para seu calendário (por exemplo, "Eventos do Ministério de Jovens" ou "Calendário da Igreja Principal").
4. Adicione uma **description** opcional para ajudar sua equipe a entender para que este calendário serve.
5. Clique em **Create** para salvar seu novo calendário.

## A Página de Detalhes do Calendário

Depois de criar um calendário, clique nele para abrir a página de detalhes. Esta página tem duas áreas principais:

- **Coluna esquerda** -- O editor de calendário e eventos. Aqui você pode visualizar, adicionar e gerenciar eventos em seu calendário.
- **Coluna direita** -- A lista de grupos associados. Isso mostra quais grupos têm eventos incluídos neste calendário.

## Adicionando Eventos

1. Na página de detalhes do calendário, use o editor de calendário à esquerda para adicionar novos eventos.
2. Defina o nome do evento, data, hora e quaisquer outros detalhes.
3. Os eventos que você adiciona aparecerão no calendário e podem ser publicados em seu site.

## Conectando Grupos

Grupos que têm eventos no calendário aparecem automaticamente na lista de grupos no lado direito da página de detalhes.

1. Clique em **Add** na seção de grupos para associar um grupo ao seu calendário.
2. Selecione o grupo no dropdown.
3. Escolha se quer incluir **all events** daquele grupo ou apenas **specific events**.
4. Clique em **Save**.

:::tip
Conectar grupos ao seu calendário é uma forma poderosa de agregar automaticamente eventos. Quando um líder de grupo adiciona um evento ao seu [grupo](../groups/creating-groups.md), ele pode fluir para seu calendário em toda a igreja sem nenhum trabalho extra de sua parte.
:::

:::info
Se você quer criar um calendário único que puxe eventos de muitos grupos em sua igreja, veja [Curated Calendar](curated-calendar) para uma abordagem simplificada.
:::

## Ativando Registro de Evento

Você pode ativar o registro para qualquer evento de calendário para que os membros possam se inscrever através do site B1 ou aplicativo móvel.

1. Clique em um evento existente ou crie um novo.
2. No editor de eventos, ative **Registration** para ativá-lo.
3. Configure as configurações de registro:
   - **Capacity** (opcional) -- Defina um número máximo de inscrições. Deixe em branco para ilimitado.
   - **Registration Opens** -- A data e hora em que o registro fica disponível.
   - **Registration Closes** -- A data e hora em que o registro fecha.
   - **Tags** -- Rótulos separados por vírgula (por exemplo, "youth, retreat, vbs") para ajudar a categorizar eventos registráveis.
4. Salve o evento.

Depois que o registro estiver ativado, os membros verão um botão **Register for this Event** quando visualizarem o evento no [site B1](../../b1-church/events/registering) ou [aplicativo B1 Mobile](../../b1-mobile/events/registering).

### Gerenciando Registros

Para visualizar e gerenciar registros para seus eventos:

1. Navegue até a página **Registrations** no B1 Admin.
2. Você verá uma tabela de todos os eventos com registro ativado, mostrando o título do evento, data, contagem de registro atual versus capacidade e tags.
3. Clique em um evento para ver a lista completa de registros, incluindo nomes, contagem de membros, status e data de registro.
4. A partir da página de detalhes, você pode:
   - **Cancel** registros individuais
   - **Delete** registros permanentemente
   - **Export** todos os registros para CSV

:::tip
Use a barra de progresso de capacidade para monitorar como os eventos estão se preenchendo rapidamente. A barra fica vermelha quando um evento está em ou acima da capacidade.
:::
