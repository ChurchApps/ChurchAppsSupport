# Formularios

<div class="article-intro">

A secao Formularios permite que voce crie formularios personalizados para inscricoes, pesquisas, coleta de dados e muito mais. Seja para um cartao de visitante, uma inscricao em evento ou um formulario de pedido de oracao, voce pode cria-lo aqui e compartilha-lo com sua congregacao atraves de um link publico ou do seu site B1.church.

</div>

## A pagina de Formularios

Quando voce abre **Formularios**, vera duas abas:

- **Formularios** -- Mostra todos os seus formularios ativos.
- **Arquivados** -- Mostra formularios que foram arquivados. Voce pode restaurar formularios arquivados a qualquer momento.

Cada formulario na lista exibe seu nome e, se configurado, uma URL publica que voce pode compartilhar externamente.

## O que voce pode fazer

Na pagina de Formularios, voce pode:

- **Adicionar um formulario** -- Clique em **Add Form** para criar um novo formulario personalizado.
- **Editar um formulario** -- Clique no nome de um formulario para abri-lo e modificar suas perguntas, membros ou configuracoes.
- **Arquivar um formulario** -- Mova formularios que voce nao precisa mais para a aba **Arquivados** para manter sua lista organizada.
- **Restaurar um formulario** -- Mude para a aba **Arquivados** e restaure qualquer formulario que deseja tornar ativo novamente.

## Trabalhando com um formulario

Clique em qualquer nome de formulario para abri-lo. O editor de formularios tem tres abas:

- **Perguntas** -- Construa a estrutura do formulario adicionando campos com diferentes tipos. Veja [Criando formularios](./creating-forms.md) para detalhes.
- **Membros** -- Controle quem pode acessar e gerenciar o formulario.
- **Envios** -- Visualize as respostas que foram enviadas. Veja [Gerenciando envios](./managing-submissions.md) para detalhes.

:::tip
Formularios com uma URL publica podem ser compartilhados via link, incorporados no seu site ou incluidos em e-mails. Isso facilita a coleta de informacoes de pessoas que podem nao ter uma conta na igreja.
:::

:::info
Voce pode controlar quem tem acesso a cada formulario usando a aba **Membros**. Atribua funcoes **Admin** ou **View Only** para dar a sua equipe o nivel certo de acesso.
:::
