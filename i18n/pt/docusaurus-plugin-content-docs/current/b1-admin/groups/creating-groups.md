# Criando Grupos

<div class="article-intro">

Criar um grupo no B1 Admin é simples. Você irá configurar uma categoria, nomear seu grupo e então configurar suas definições. Grupos ajudam você a organizar sua igreja em unidades significativas como pequenos grupos, comitês e classes.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Você precisa de uma conta B1 Admin ativa com permissão para gerenciar grupos. Veja [Funções e Permissões](../people/roles-permissions.md) se não tiver certeza sobre seu nível de acesso.
- Decida sobre uma estrutura de categoria para seus grupos (por exemplo, "Pequenos Grupos", "Ministérios", "Comitês"). Categorias ajudam a manter grupos relacionados organizados.

</div>

## Adicionando um Novo Grupo

1. Navegue até o **painel B1 Admin**.
2. Clique na aba **Groups**.
3. Clique em **Add Group** e insira um **Category Name**. Categorias ajudam você a organizar grupos relacionados juntos (por exemplo, "Pequenos Grupos", "Ministérios" ou "Comitês"). Se uma categoria já existe, você pode selecioná-la da lista.
4. Insira o **Group Name**.
5. Clique em **Add**. Seu novo grupo aparecerá na lista sob a categoria escolhida.

## Configurando Definições de Grupo

Depois que seu grupo é criado, você pode preencher detalhes adicionais:

1. Clique no **group name** na lista para abri-lo.
2. Clique no **ícone de lápis** para editar as definições do grupo.
3. Configure as seguintes opções:
   - **Description** -- Um breve resumo do que o grupo é sobre. Isso é visível para membros.
   - **Meeting Times** -- Quando o grupo normalmente se reúne (por exemplo, "Quartas-feiras às 7 PM").
   - **Attendance Tracking** -- Ative isso se você quer registrar [presença](../attendance/tracking-attendance.md) para este grupo.
   - **Service Times** -- Associe o grupo com horários de serviço específicos da igreja se aplicável. Veja [Configuração de Presença](../attendance/setup.md) para detalhes sobre horários de serviço.
4. Clique em **Save** para aplicar suas mudanças.

:::tip
Adicionar uma descrição clara e horário de reunião ajuda membros a saber o que esperar quando se juntam a um grupo.
:::

## Próximas Etapas

Depois de criar e configurar seu grupo, você está pronto para:

- **Adicionar membros** -- Procure por pessoas e adicione-as ao grupo. Use o ícone de chave verde para designar líderes de grupo. Veja [Group Members](./group-members.md).
- **Configurar um calendário** -- Crie eventos e reuniões recorrentes para o grupo. Veja [Group Calendar](./group-calendar.md).
- **Comunicar** -- Envie mensagens para todos os membros do grupo diretamente da página do grupo.
- **Exportar dados** -- Clique no ícone de download para exportar a lista de membros do seu grupo.

:::info
Todos os seus grupos de igreja são organizados por categorias na página principal de Grupos. Você pode sempre reorganizar ou renomear categorias conforme sua igreja cresce.
:::
