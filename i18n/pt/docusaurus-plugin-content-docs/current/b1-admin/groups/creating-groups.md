# Criando Grupos

<div class="article-intro">

Criar um grupo no B1 Admin é simples. Você configurará uma categoria, nomeará seu grupo e depois configurará suas definições. Os grupos ajudam você a organizar sua igreja em unidades significativas como pequenos grupos, comitês e classes.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Você precisa de uma conta ativa no B1 Admin com permissão para gerenciar grupos. Consulte [Funções e Permissões](../people/roles-permissions.md) se não tiver certeza sobre seu nível de acesso.
- Decida sobre uma estrutura de categorias para seus grupos (por exemplo, "Pequenos Grupos", "Ministérios", "Comitês"). As categorias ajudam a manter os grupos relacionados organizados.

</div>

## Adicionando um Novo Grupo

1. Navegue até o **painel do B1 Admin**.
2. Clique na aba **Grupos**.
3. Clique em **Adicionar Grupo** e insira um **Nome da Categoria**. As categorias ajudam você a organizar grupos relacionados juntos (por exemplo, "Pequenos Grupos", "Ministérios" ou "Comitês"). Se uma categoria já existir, você pode selecioná-la da lista.
4. Insira o **Nome do Grupo**.
5. Clique em **Adicionar**. Seu novo grupo aparecerá na lista sob a categoria escolhida.

## Configurando as Definições do Grupo

Depois que seu grupo for criado, você pode preencher detalhes adicionais:

1. Clique no **nome do grupo** na lista para abri-lo.
2. Clique no **ícone de lápis** para editar as definições do grupo.
3. Configure as seguintes opções:
   - **Descrição** -- Um breve resumo sobre o que é o grupo. Isso é visível para os membros.
   - **Horários de Reunião** -- Quando o grupo normalmente se reúne (por exemplo, "Quartas-feiras às 19h").
   - **Política de Ingresso** -- Escolha quem pode ingressar neste grupo:
     - **Aberto** -- Qualquer pessoa pode ingressar imediatamente sem aprovação
     - **Solicitação** -- As pessoas devem enviar uma solicitação de ingresso que requer aprovação (consulte [Solicitações de Ingresso em Grupos](./group-join-requests.md))
     - **Fechado** -- Os membros devem ser adicionados manualmente por líderes ou administradores
   - **Rastreamento de Presença** -- Ative isso se desejar registrar a [presença](../attendance/tracking-attendance.md) para este grupo.
   - **Horários de Culto** -- Associe o grupo a horários específicos de culto da igreja, se aplicável. Consulte [Configuração de Presença](../attendance/setup.md) para detalhes sobre horários de culto.
4. Clique em **Salvar** para aplicar suas alterações.

:::tip
Adicionar uma descrição clara e horário de reunião ajuda os membros a saberem o que esperar quando ingressarem em um grupo.
:::

## Próximos Passos

Depois de criar e configurar seu grupo, você está pronto para:

- **Adicionar membros** -- Procure pessoas e adicione-as ao grupo. Use o ícone de chave verde para designar líderes de grupo. Consulte [Membros do Grupo](./group-members.md).
- **Configurar um calendário** -- Crie eventos e reuniões recorrentes para o grupo. Consulte [Calendário do Grupo](./group-calendar.md).
- **Comunicar** -- Envie mensagens para todos os membros do grupo diretamente da página do grupo.
- **Exportar dados** -- Clique no ícone de download para exportar a lista de membros do seu grupo.

:::info
Todos os grupos da sua igreja são organizados por categorias na página principal de Grupos. Você sempre pode reorganizar ou renomear categorias à medida que sua igreja cresce.
:::
