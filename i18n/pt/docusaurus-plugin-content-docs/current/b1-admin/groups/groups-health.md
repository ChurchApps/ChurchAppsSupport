# Saúde dos Grupos

<div class="article-intro">

O dashboard de Saúde dos Grupos oferece uma visão geral de como todos os seus grupos estão indo — tendências de associação, médias de frequência e crescimento ou atrito nos últimos 90 dias — tudo em uma única tabela classificável.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Você precisa ter pelo menos alguns grupos com membros para ver dados significativos. Veja [Criando Grupos](creating-groups).
- Os dados de frequência são extraídos de sessões registradas. Veja a seção [Attendance](../attendance/index).

</div>

## Abrindo Saúde dos Grupos

Em B1 Admin, acesse **Grupos** na barra lateral esquerda e selecione **Saúde**. O dashboard carrega uma tabela com uma linha por grupo.

## Colunas

| Coluna | O que mostra |
|--------|--------------|
| **Nome** | O nome do grupo, vinculado à página de detalhes do grupo |
| **Categoria** | A categoria do grupo |
| **Membros** | Contagem atual de membros ativos |
| **Ingressou (90d)** | Membros que ingressaram nos últimos 90 dias |
| **Saiu (90d)** | Membros que saíram nos últimos 90 dias |
| **Churn (90d)** | Taxa de churn líquida como porcentagem em 90 dias |
| **Frequência Média** | Contagem média de presentes por sessão de frequência |

Clique em qualquer cabeçalho de coluna para classificar a tabela por essa coluna. Clique novamente para reverter a direção de classificação.

## Usando Dados de Saúde

- **Alto churn + poucos ingressos** — um grupo que está encolhendo e não substituindo membros perdidos. Vale a pena conversar com o líder do grupo.
- **Muitos ingressos + baixa frequência** — as pessoas estão se inscrevendo mas não aparecem. Considere o acompanhamento de engajamento.
- **Alta frequência média** — um grupo saudável e ativo. Modelo potencial para outros grupos.

:::tip
Clicar em um nome de grupo leva você diretamente à página de detalhes do grupo onde você pode revisar membros individuais, registros de frequência e eventos de calendário.
:::
