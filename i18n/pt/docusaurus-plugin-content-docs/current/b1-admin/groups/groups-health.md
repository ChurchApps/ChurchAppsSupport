# Saúde dos Grupos

<div class="article-intro">

O painel de Saúde dos Grupos oferece uma visão geral de como todos os seus grupos estão indo — tendências de associação, médias de presença e crescimento ou atrito nos últimos 90 dias — tudo em uma única tabela classificável.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Você precisa ter pelo menos alguns grupos com membros para ver dados significativos. Veja [Creating Groups](creating-groups).
- Os dados de presença são obtidos de sessões registradas. Veja a seção [Attendance](../attendance/).

</div>

## Abrindo a Saúde dos Grupos

Em B1 Admin, acesse **Groups** na barra lateral esquerda e selecione **Health**. O painel carrega uma tabela com uma linha por grupo.

## Colunas

| Coluna | O que mostra |
|--------|--------------|
| **Name** | O nome do grupo, vinculado à página de detalhes do grupo |
| **Category** | A categoria do grupo |
| **Members** | Contagem atual de membros ativos |
| **Joined (90d)** | Membros que se juntaram nos últimos 90 dias |
| **Left (90d)** | Membros que saíram nos últimos 90 dias |
| **Churn (90d)** | Taxa de churn líquida como porcentagem ao longo de 90 dias |
| **Avg Attendance** | Contagem média de cabeças por sessão de presença |

Clique em qualquer cabeçalho de coluna para classificar a tabela por essa coluna. Clique novamente para inverter a direção da classificação.

## Usando Dados de Saúde

- **Churn alto + adesões baixas** — um grupo que está encolhendo e não substituindo membros perdidos. Vale a pena uma conversa com o líder do grupo.
- **Adesões altas + presença baixa** — as pessoas estão se inscrevendo, mas não aparecem. Considere um acompanhamento de engajamento.
- **Presença média alta** — um grupo saudável e ativo. Potencial modelo para outros grupos.

:::tip
Clicar em um nome de grupo o leva diretamente à página de detalhes do grupo, onde você pode revisar membros individuais, registros de presença e eventos do calendário.
:::
