# Pareando Seu Dispositivo

<div class="article-intro">

Para usar o Modo Plano, você precisa parear sua TV com um tipo de plano no B1 Admin. O FreePlay gera um código de pareamento único que vincula o dispositivo ao plano da sua igreja, permitindo a entrega automática de conteúdo a cada semana.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Instale e inicie o FreePlay -- veja [Primeiros Passos](../getting-started/)
- Escolha **Parear com Plano** na [tela de modo de pareamento](../getting-started/pairing-modes.md)
- Tenha acesso ao **B1 Admin** em um computador ou celular para completar o pareamento

</div>

## Gerando o Código de Pareamento

1. Na tela **Selecionar Modo de Pareamento**, escolha **Parear com Plano**
2. O FreePlay contacta o servidor e gera um código curto de pareamento
3. O código aparece em caracteres grandes no centro da tela
4. Abaixo do código, um indicador pulsante mostra **Aguardando conexão**

O código é exibido como caracteres individuais para facilitar a leitura do outro lado da sala.

## Inserindo o Código no B1 Admin

1. Em um computador ou celular, abra o **B1 Admin** da sua igreja
2. Navegue até as configurações de **Tipo de Plano** para o plano que deseja atribuir
3. Insira o código de pareamento mostrado na TV
4. Salve as configurações

O FreePlay consulta o servidor a cada poucos segundos, verificando se o pareamento foi concluído. Assim que o **B1 Admin** confirma a conexão, a TV passa automaticamente para a tela de download do plano.

## Baixando o Conteúdo do Plano

Após o pareamento, o FreePlay carrega o plano atual para aquele tipo de plano. Ele exibe:

- O nome do plano e a data do culto
- O nome da lição associada (se o plano incluir conteúdo de lição)
- Um indicador de progresso mostrando **Baixando item X de Y**

Quando todos os arquivos de mídia são baixados, o botão **Iniciar Plano** aparece. Pressione **Selecionar** no controle remoto para iniciar a reprodução.

:::tip
O plano é atualizado automaticamente a cada hora. Se o plano for alterado durante o dia, o FreePlay busca as mudanças sem qualquer intervenção manual.
:::

## Alternativa: Pesquisar por Nome da Igreja

Se preferir não usar o fluxo de código de pareamento, você pode selecionar **ou pesquisar por nome da igreja** na parte inferior da tela de pareamento. Isso o leva à tela de pesquisa de igrejas, onde você pode encontrar sua igreja e se conectar a uma sala de aula.

## Se o Pareamento Falhar

Se o código de pareamento não puder ser gerado (por exemplo, devido a um problema de rede), você verá uma mensagem de erro com um botão **Tentar Novamente**. Certifique-se de que sua TV está conectada à internet e tente novamente.

:::warning
Os códigos de pareamento expiram após um período determinado. Se você demorar muito, gere um novo código retornando à tela de pareamento.
:::
