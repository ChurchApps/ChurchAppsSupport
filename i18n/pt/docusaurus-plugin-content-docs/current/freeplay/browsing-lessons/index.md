# Navegando pelas Lições

<div class="article-intro">

FreePlay inclui um navegador de lições integrado que permite explorar a biblioteca completa de programas, estudos e lições disponíveis. Isso é útil quando você deseja escolher manualmente uma lição para reproduzir em vez de depender de um cronograma de sala de aula ou plano.

</div>

## O Fluxo de Navegação

As lições são organizadas em uma hierarquia:

1. **Programas** - Categorias de currículo de nível superior (por exemplo, uma série de ministério infantil)
2. **Estudos** - Grupos de lições dentro de um programa (por exemplo, uma unidade de várias semanas)
3. **Lições** - Sessões de lição individuais com seu próprio conteúdo
4. **Ambientes** - Diferentes versões de uma lição adaptadas para faixas etárias ou contextos específicos

Você começa selecionando um programa, depois navega até um estudo, escolhe uma lição e, por fim, seleciona um ambiente para iniciar a reprodução.

## Acessando o Navegador de Lições

O navegador de lições está disponível na tela **Explorar Programas**. Navegue até lá usando a barra lateral ou selecionando a opção de explorar na navegação.

:::info
Navegar pelas lições requer uma conexão com a internet para carregar o catálogo de programas. Depois de selecionar um ambiente e iniciar a reprodução, o conteúdo é transmitido diretamente sem uma etapa de download separada.
:::
