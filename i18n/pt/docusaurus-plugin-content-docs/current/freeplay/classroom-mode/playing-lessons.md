# Reproduzindo Lições

<div class="article-intro">

O reprodutor de mídia do FreePlay é uma experiência em tela cheia projetada para controles remotos de TV. Ele reproduz tanto vídeos quanto slides de imagem, avançando pelo conteúdo da lição automaticamente ou manualmente, dando aos professores controle total sobre o ritmo da aula.

</div>

<div class="prereqs">
<h4>Antes de Começar</h4>

- Baixe uma lição -- veja [Baixando Conteúdo da Lição](./downloading-content)
- Pressione o botão **Iniciar Lição** para entrar no reprodutor

</div>

## Controles do Reprodutor

Use o controle remoto da TV para controlar a reprodução:

| Botão do Controle | Ação |
|---|---|
| **Selecionar / Play-Pause** | Pausar ou retomar o vídeo atual |
| **Seta direita / Avançar** | Pular para o próximo slide ou vídeo |
| **Seta esquerda / Retroceder** | Voltar para o slide ou vídeo anterior |
| **Seta para cima** | Abrir o seletor de mensagens para ir a um item específico |
| **Seta para baixo / Voltar / Menu** | Sair do reprodutor e retornar à tela anterior |

:::tip
Você também pode deslizar em controles remotos com toque. Deslize para a esquerda para avançar, deslize para a direita para voltar, deslize para baixo para abrir o seletor de mensagens e deslize para cima para sair do reprodutor.
:::

## Reprodução de Vídeo

Quando um vídeo está sendo reproduzido, você verá o conteúdo do vídeo em tela cheia. Pressionar **Selecionar** ou **Play-Pause** pausa o vídeo e exibe um ícone de pausa sobreposto com uma barra de progresso na parte inferior da tela. Pressione novamente para retomar.

Os vídeos avançam automaticamente para o próximo item da lição quando terminam. Se um vídeo estiver configurado para repetir, ele será reproduzido novamente até que você avance manualmente.

## Slides de Imagem

Os slides de imagem são exibidos em tela cheia. Diferente dos vídeos, os slides de imagem não avançam automaticamente -- eles permanecem na tela até que você pressione a seta direita para avançar. Isso dá ao professor controle total sobre o ritmo.

## Seletor de Mensagens

Pressione a seta **Para cima** durante a reprodução para abrir o seletor de mensagens. Isso mostra uma grade de todos os itens da lição com miniaturas e títulos. Navegue até qualquer item e pressione **Selecionar** para ir diretamente a ele. Isso é útil para pular adiante ou retornar a uma parte específica da lição.

## Saindo do Reprodutor

Pressione **Para baixo**, **Voltar** ou **Menu** no controle remoto para sair do reprodutor. Você retornará à tela de download (no Modo Sala de Aula), à tela de download do plano (no Modo Plano) ou ao navegador de conteúdo (ao reproduzir conteúdo de provedores).

:::info
A tela permanece ligada enquanto o reprodutor está ativo, então você não precisa se preocupar com a TV entrando em modo de espera durante uma lição.
:::
