# Modo Sala de Aula

<div class="article-intro">

O Modo Sala de Aula conecta sua TV a uma sala específica na sua igreja. Uma vez pareado, o FreePlay baixa automaticamente a lição agendada para aquela sala, para que seus voluntários possam simplesmente pressionar **Iniciar Lição** quando a aula começar.

</div>

## Como Funciona

1. **Encontre sua igreja** - Pesquise sua igreja pelo nome
2. **Selecione uma sala** - Escolha a sala de aula à qual esta TV pertence
3. **Baixe a lição** - O FreePlay busca a lição agendada e baixa todos os arquivos de mídia
4. **Reproduza** - Pressione o botão **Iniciar Lição** e a lição é reproduzida automaticamente

A TV lembra qual igreja e sala estão pareadas, então você só precisa fazer a configuração uma vez. A cada semana, o FreePlay verifica a lição agendada para o dia atual e baixa qualquer conteúdo novo.

:::tip
Certifique-se de que suas salas de aula estão configuradas em [freeplay.church](https://freeplay.church) antes de parear sua TV. Se nenhuma sala de aula aparecer quando você selecionar sua igreja, visite o site para configurá-las.
:::
