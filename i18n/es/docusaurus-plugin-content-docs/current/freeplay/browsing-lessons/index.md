# Explorar lecciones

<div class="article-intro">

FreePlay incluye un explorador de lecciones integrado que le permite explorar la biblioteca completa de programas, estudios y lecciones disponibles. Esto es util cuando desea elegir manualmente una leccion para reproducir en lugar de depender de un calendario de aula o un plan.

</div>

## El flujo de exploracion

Las lecciones estan organizadas en una jerarquia:

1. **Programas** - Categorias de curriculo de nivel superior (por ejemplo, una serie del ministerio infantil)
2. **Estudios** - Grupos de lecciones dentro de un programa (por ejemplo, una unidad de varias semanas)
3. **Lecciones** - Sesiones de lecciones individuales con su propio contenido
4. **Sedes** - Diferentes versiones de una leccion adaptadas para grupos de edad o entornos especificos

Comienza seleccionando un programa, luego profundiza en un estudio, elige una leccion y finalmente selecciona una sede para comenzar la reproduccion.

## Acceder al explorador de lecciones

El explorador de lecciones esta disponible a traves de la pantalla **Explorar programas**. Navegue alli usando la barra lateral o seleccionando la opcion de explorar desde la navegacion.

:::info
Explorar lecciones requiere una conexion a internet para cargar el catalogo de programas. Una vez que selecciona una sede y comienza la reproduccion, el contenido se transmite directamente sin un paso de descarga separado.
:::
