# Vincular su dispositivo

<div class="article-intro">

Para usar el Modo Plan, necesita vincular su TV con un tipo de plan en B1 Admin. FreePlay genera un codigo de vinculacion unico que enlaza el dispositivo con el plan de su iglesia, permitiendo la entrega automatica de contenido cada semana.

</div>

<div class="prereqs">
<h4>Antes de comenzar</h4>

- Instale e inicie FreePlay -- consulte [Primeros pasos](../getting-started/)
- Elija **Vincular a un plan** desde la [pantalla de modo de vinculacion](../getting-started/pairing-modes.md)
- Tenga acceso a **B1 Admin** en una computadora o telefono para completar la vinculacion

</div>

## Generar el codigo de vinculacion

1. Desde la pantalla **Seleccionar modo de vinculacion**, elija **Vincular a un plan**
2. FreePlay contacta al servidor y genera un codigo de vinculacion corto
3. El codigo aparece en caracteres grandes en el centro de la pantalla
4. Debajo del codigo, un indicador pulsante muestra **Esperando conexion**

El codigo se muestra como caracteres individuales para facilitar su lectura desde el otro lado de la sala.

## Ingresar el codigo en B1 Admin

1. En una computadora o telefono, abra **B1 Admin** para su iglesia
2. Navegue a la configuracion de **Tipo de plan** para el plan que desea asignar
3. Ingrese el codigo de vinculacion que se muestra en el TV
4. Guarde la configuracion

FreePlay consulta el servidor cada pocos segundos para verificar si la vinculacion se ha completado. Una vez que **B1 Admin** confirma la conexion, el TV se mueve automaticamente a la pantalla de descarga del plan.

## Descargar contenido del plan

Despues de la vinculacion, FreePlay carga el plan actual para ese tipo de plan. Muestra:

- El nombre del plan y la fecha del servicio
- El nombre de la leccion asociada (si el plan incluye contenido de lecciones)
- Un indicador de progreso que muestra **Descargando elemento X de Y**

Cuando todos los archivos multimedia estan descargados, aparece el boton **Iniciar plan**. Presione **Seleccionar** en su control remoto para comenzar la reproduccion.

:::tip
El plan se actualiza automaticamente cada hora. Si el plan se modifica durante el dia, FreePlay detecta los cambios sin ninguna intervencion manual.
:::

## Alternativa: Buscar por nombre de iglesia

Si prefiere no usar el flujo de codigo de vinculacion, puede seleccionar **o buscar por nombre de iglesia** en la parte inferior de la pantalla de vinculacion. Esto le lleva a la pantalla de busqueda de iglesia donde puede encontrar su iglesia y conectarse a un aula en su lugar.

## Si la vinculacion falla

Si no se puede generar el codigo de vinculacion (por ejemplo, debido a un problema de red), vera un mensaje de error con un boton **Intentar de nuevo**. Asegurese de que su TV este conectado a internet e intentelo nuevamente.

:::warning
Los codigos de vinculacion expiran despues de un periodo determinado. Si espera demasiado tiempo, genere un nuevo codigo regresando a la pantalla de vinculacion.
:::
