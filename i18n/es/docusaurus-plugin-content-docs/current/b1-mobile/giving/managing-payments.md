# Administrar Métodos de Pago

<div class="article-intro">

La pestaña **Administrar** en la sección de ofrendas te permite ver y administrar tus métodos de pago guardados. Al guardar una tarjeta, puedes hacer donaciones futuras más rápido sin tener que volver a ingresar tus datos cada vez.

</div>

<div class="prereqs">
<h4>Antes de Comenzar</h4>

- [Instala B1 Mobile](../getting-started/installing) e [inicia sesión](../getting-started/logging-in) en tu cuenta

</div>

## Ver tus Métodos de Pago

Cuando abras la pestaña **Administrar**, verás una lista de todos tus métodos de pago guardados. Cada entrada muestra el tipo de tarjeta y los últimos cuatro dígitos del número de tarjeta, para que puedas identificar fácilmente cuál es cuál.

## Agregar un Nuevo Método de Pago

Para agregar una nueva tarjeta:

1. Abre la sección de **Ofrendas** en B1 Mobile.
2. Toca la pestaña **Administrar**.
3. Toca la opción para agregar un nuevo método de pago.
4. Ingresa los datos de tu tarjeta. La información se envía de forma segura a **Stripe** para su procesamiento.
5. Una vez guardada, la nueva tarjeta aparecerá en tu lista de métodos de pago.

Tu tarjeta guardada estará disponible como opción de pago la próxima vez que [realices una donación](./making-donations) desde la pestaña **Donar**.

## Eliminar un Método de Pago

Si ya no deseas tener una tarjeta registrada, puedes eliminarla desde la pestaña **Administrar**:

1. Encuentra la tarjeta que deseas eliminar en tu lista de métodos de pago guardados.
2. Toca la opción de eliminar junto a la tarjeta.
3. Confirma la eliminación.

La tarjeta ya no aparecerá como opción de pago cuando realices una donación.

:::info
Eliminar un método de pago no afecta las donaciones anteriores. Tu [historial de donaciones](./donation-history) permanece sin cambios.
:::

:::tip
Mantén al menos un método de pago guardado para dar rápidamente. De esta manera, puedes usar el botón **Repetir** en la pestaña **Resumen** para dar nuevamente con un solo toque.
:::
