# Primeros Pasos

<div class="article-intro">

Comenzar con B1 Mobile toma solo unos minutos. Descarga la aplicación, inicia sesión con tu cuenta de ChurchApps, selecciona tu iglesia y estarás listo para empezar.

</div>

## Inicio Rápido

1. **Descarga** la aplicación desde la [App Store](https://apps.apple.com) (iOS) o [Google Play Store](https://play.google.com) (Android). Busca "B1.church".
2. **Inicia sesión** con tu cuenta existente de ChurchApps, el mismo correo electrónico y contraseña que usas en B1.church. Si aún no tienes una cuenta, toca **Registrarse** para crear una.
3. **Selecciona tu iglesia** buscándola por nombre. Escribe al menos 3 caracteres para ver resultados.
4. **Explora el panel de inicio** -- tu pantalla de inicio personalizada muestra accesos rápidos a las funciones que tu iglesia ofrece.

Eso es todo. Una vez que hayas seleccionado tu iglesia, la aplicación carga la imagen, los enlaces de navegación y el contenido de tu iglesia. Puedes comenzar a usar grupos, ofrendas, registro de asistencia y todo lo demás que tu iglesia haya habilitado.
