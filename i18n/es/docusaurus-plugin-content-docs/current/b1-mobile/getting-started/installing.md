# Instalando la Aplicación

<div class="article-intro">

B1 Mobile está disponible para dispositivos iOS y Android. La aplicación es gratuita para descargar y solo toma un momento instalarla para que puedas comenzar a conectarte con tu iglesia.

</div>

## Descarga

- **iOS** -- Abre la **App Store** en tu iPhone o iPad y busca **B1.church**. Toca **Obtener** para descargar e instalar.
- **Android** -- Abre la **Google Play Store** en tu dispositivo Android y busca **B1.church**. Toca **Instalar** para descargar e instalar.

También puedes buscar "B1 Mobile" si la primera búsqueda no devuelve resultados.

## Después de Instalar

Una vez que la aplicación está instalada, ábrela desde tu pantalla de inicio. Verás la pantalla de búsqueda de iglesia donde puedes encontrar y seleccionar tu iglesia. Si ya tienes una cuenta de ChurchApps, puedes [iniciar sesión](./logging-in) inmediatamente. Si no, puedes crear una cuenta desde la pantalla de inicio de sesión.

La aplicación te pedirá permiso para enviar notificaciones push. Recomendamos permitir notificaciones para que recibas actualizaciones de tu iglesia y grupos.

:::tip
Permite notificaciones push cuando se te pida durante la configuración. Esto asegura que recibas actualizaciones oportunas sobre [actividad de grupo](../groups/group-timeline.md), [mensajes](../community/messaging.md) y [asignaciones de servicio](../serving/viewing-plans.md).
:::

## Actualizando la Aplicación

B1 Mobile recibe actualizaciones regulares con nuevas características y mejoras. Tu dispositivo debe actualizar la aplicación automáticamente, pero también puedes buscar actualizaciones manualmente:

1. Abre la **App Store** (iOS) o **Google Play Store** (Android).
2. Busca **B1.church**.
3. Si una actualización está disponible, toca **Actualizar**.

:::info
B1 Mobile requiere iOS 14 o posterior para dispositivos Apple y Android 6.0 o posterior para dispositivos Android.
:::

## ¿Prefieres No Instalar una Aplicación Nativa?

Si prefieres no instalar una aplicación desde la App Store o Google Play, puedes acceder al portal de miembros de B1.church directamente desde el navegador de tu teléfono. B1.church admite instalación como **Aplicación Web Progresiva (PWA)**, que añade el portal de miembros a la pantalla de inicio desde Safari o Chrome -- sin necesidad de descargar. Ver [Instalación como Aplicación (PWA)](/docs/b1-church/getting-started/installing-pwa) para instrucciones.

La aplicación nativa de B1 Mobile y la PWA ofrecen características de portal de miembros similares, pero las notificaciones push solo están disponibles en la aplicación nativa.
