# Instalar la Aplicación

<div class="article-intro">

B1 Mobile está disponible para dispositivos iOS y Android. La aplicación es gratuita para descargar y solo toma un momento instalarla para que puedas comenzar a conectarte con tu iglesia.

</div>

## Descarga

- **iOS** -- Abre la **App Store** en tu iPhone o iPad y busca **B1.church**. Toca **Obtener** para descargar e instalar.
- **Android** -- Abre la **Google Play Store** en tu dispositivo Android y busca **B1.church**. Toca **Instalar** para descargar e instalar.

También puedes buscar "B1 Mobile" si la primera búsqueda no arroja resultados.

## Después de Instalar

Una vez instalada la aplicación, ábrela desde tu pantalla de inicio. Verás la pantalla de búsqueda de iglesias donde puedes encontrar y seleccionar tu iglesia. Si ya tienes una cuenta de ChurchApps, puedes [iniciar sesión](./logging-in) de inmediato. Si no, puedes crear una cuenta desde la pantalla de inicio de sesión.

La aplicación te pedirá permiso para enviar notificaciones push. Recomendamos permitir las notificaciones para que puedas recibir actualizaciones de tu iglesia y grupos.

:::tip
Permite las notificaciones push cuando se te solicite durante la configuración. Esto asegura que recibas actualizaciones oportunas sobre [actividad de grupos](../groups/group-timeline.md), [mensajes](../community/messaging.md) y [asignaciones de servicio](../serving/viewing-plans.md).
:::

## Actualizar la Aplicación

B1 Mobile recibe actualizaciones periódicas con nuevas funciones y mejoras. Tu dispositivo debería actualizar la aplicación automáticamente, pero también puedes buscar actualizaciones manualmente:

1. Abre la **App Store** (iOS) o **Google Play Store** (Android).
2. Busca **B1.church**.
3. Si hay una actualización disponible, toca **Actualizar**.

:::info
B1 Mobile requiere iOS 14 o posterior para dispositivos Apple y Android 6.0 o posterior para dispositivos Android.
:::
