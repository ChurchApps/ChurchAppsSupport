# Instalando la Aplicación

:::danger B1 Mobile está obsoleto — instala la PWA en su lugar
La aplicación nativa B1 Mobile ya no se está actualizando. Recomendamos instalar la **Aplicación Web Progresiva (PWA) de B1.church** desde tu sitio de iglesia (`https://yourchurchname.b1.church`) en su lugar; se puede instalar desde cualquier página. La PWA se puede agregar a tu pantalla de inicio en iOS o Android en segundos y reemplaza todas las características de la aplicación nativa.

Consulta [Installing as an App (PWA)](../../b1-church/getting-started/installing-pwa.md) para obtener instrucciones paso a paso.
:::

<div class="article-intro">

B1 Mobile está disponible para dispositivos iOS y Android. La aplicación es gratuita para descargar y solo toma un momento instalar para que puedas comenzar a conectarte con tu iglesia.

</div>

## Descargar

- **iOS** -- Abre la **App Store** en tu iPhone o iPad y busca **B1.church**. Toca **Get** para descargar e instalar.
- **Android** -- Abre la **Google Play Store** en tu dispositivo Android y busca **B1.church**. Toca **Install** para descargar e instalar.

También puedes buscar "B1 Mobile" si la primera búsqueda no devuelve resultados.

## Después de Instalar

Una vez que la aplicación esté instalada, ábrela desde tu pantalla de inicio. Verás la pantalla de búsqueda de iglesia donde puedes encontrar y seleccionar tu iglesia. Si ya tienes una cuenta de ChurchApps, puedes [iniciar sesión](./logging-in) de inmediato. Si no, puedes crear una cuenta desde la pantalla de inicio de sesión.

La aplicación te pedirá permiso para enviar notificaciones push. Recomendamos permitir notificaciones para que recibas actualizaciones de tu iglesia y grupos.

:::tip
Permite notificaciones push cuando se solicite durante la configuración. Esto asegura que recibas actualizaciones oportunas sobre la [actividad del grupo](../groups/group-timeline.md), [mensajes](../community/messaging.md) y [asignaciones de servicio](../serving/viewing-plans.md).
:::

## Actualizando la Aplicación

B1 Mobile recibe actualizaciones regulares con nuevas características y mejoras. Tu dispositivo debe actualizar la aplicación automáticamente, pero también puedes buscar actualizaciones manualmente:

1. Abre la **App Store** (iOS) o **Google Play Store** (Android).
2. Busca **B1.church**.
3. Si hay una actualización disponible, toca **Update**.

:::info
B1 Mobile requiere iOS 14 o posterior para dispositivos Apple y Android 6.0 o posterior para dispositivos Android.
:::

## Usa la PWA en su Lugar

La aplicación nativa B1 Mobile ha sido obsoleta a favor de la **Aplicación Web Progresiva (PWA) de B1.church**. La PWA se instala desde Safari o Chrome; no se requiere descarga de App Store; y es la opción recomendada para cada nueva instalación. Consulta [Installing as an App (PWA)](/docs/b1-church/getting-started/installing-pwa) para obtener instrucciones.
