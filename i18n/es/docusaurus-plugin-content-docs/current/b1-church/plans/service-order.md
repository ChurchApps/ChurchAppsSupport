# Orden del Servicio

<div class="article-intro">

Cuando abres un plan, la seccion de **Orden del Servicio** muestra la secuencia de elementos para ese servicio. Esto te da a ti y a tu equipo una vista compartida del flujo del servicio, incluyendo tiempos, detalles de canciones y descripciones de cada elemento.

</div>

<div class="prereqs">
<h4>Antes de Comenzar</h4>

- Debes haber [iniciado sesion](../getting-started/logging-in.md) en tu cuenta de B1.church.
- Abre un plan desde tu [lista de Planes](./viewing-plans.md) para ver su orden del servicio.

</div>

## Lo que Muestra el Orden del Servicio

El orden del servicio se muestra como una lista de elementos en el orden en que ocurriran durante el servicio. Cada elemento incluye:

- **Tiempo acumulado** -- El tiempo transcurrido desde el inicio del servicio, para que sepas cuando se espera que comience cada elemento.
- **Etiqueta y descripcion del elemento** -- El nombre del elemento (como el titulo de una cancion, oracion o anuncio) junto con cualquier nota o descripcion.
- **Duracion** -- Cuanto tiempo se espera que dure el elemento.

Los elementos estan organizados bajo **encabezados** que agrupan elementos relacionados (por ejemplo, "Bloque de Alabanza" o "Mensaje"). Esto facilita revisar la estructura general del servicio.

## Contenido de Curriculo y Lecciones

Si el plan tiene curriculo o contenido de leccion asociado de un proveedor como [Lessons.church](../content/lessons.md), veras una vista previa de ese contenido dentro del orden del servicio. Esto puede incluir secciones de leccion, materiales de presentacion y otros recursos que tu equipo necesita para el servicio.

## Imprimir el Orden del Servicio

Puedes imprimir una version formateada del orden del servicio para llevar contigo el dia del servicio.

1. Abre el plan desde tu lista de **Planes**.
2. En la seccion de **Orden del Servicio**, haz clic en el icono de **imprimir** en la esquina superior derecha.
3. Se abre una nueva ventana con un diseno imprimible que incluye:
   - La **fecha del servicio** en la parte superior.
   - Las **asignaciones del equipo** agrupadas por categoria a la izquierda, mostrando cada posicion y quien la esta cubriendo.
   - El **orden completo del servicio** a la derecha con columnas de tiempo, detalles de elementos y duracion.
4. El dialogo de impresion de tu navegador se abrira automaticamente para que puedas imprimir o guardar como PDF.

:::tip
El orden del servicio impreso es una excelente referencia para tener a mano durante los ensayos o el servicio en si. Incluye tanto la lista del equipo como el orden completo del servicio en una sola pagina.
:::
