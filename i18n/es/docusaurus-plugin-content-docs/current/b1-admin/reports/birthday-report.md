# Reporte de Cumpleaños

<div class="article-intro">

El Reporte de Cumpleaños le ayuda a mantenerse conectado con su congregación mostrando a los miembros con próximos cumpleaños. Úselo para planificar celebraciones, enviar tarjetas o hacer anuncios durante los servicios.

</div>

<div class="prereqs">
<h4>Antes de Comenzar</h4>

- Asegúrese de que las fechas de nacimiento estén ingresadas en el perfil de cada persona. Consulte [Agregar Personas](../people/adding-people.md) para más detalles.
- Necesita los [permisos](../settings/roles-permissions.md) apropiados para acceder a los reportes

</div>

## Ejecutar el Reporte

1. Navegue a **Reportes** desde el menú principal.
2. Haga clic en la tarjeta **Reporte de Cumpleaños**.
3. Establezca el **rango de fechas** usando los filtros de fecha para definir qué cumpleaños desea ver.
4. El reporte mostrará una lista de miembros con cumpleaños en ese rango.

## Qué Muestra el Reporte

El Reporte de Cumpleaños muestra:

- **Nombre del miembro** -- El nombre de la persona en la base de datos de su iglesia.
- **Fecha de nacimiento** -- La fecha de su cumpleaños.

Los resultados se listan dentro del rango de fechas que seleccionó, lo que facilita ver quién tiene un cumpleaños próximo.

## Usos Prácticos

- **Anuncios semanales** -- Ejecute el reporte para la próxima semana e incluya los cumpleaños en su boletín o anuncios.
- **Tarjetas de cumpleaños** -- Genere una lista para el mes y haga que su equipo envíe tarjetas de cumpleaños personales.
- **Líderes de grupos pequeños** -- Los líderes de grupo pueden ejecutar el reporte para recordar los cumpleaños de los miembros en su grupo.
- **Celebraciones especiales** -- Filtre por rangos de fechas específicos para planificar celebraciones de cumpleaños importantes.

:::tip
Ejecute este reporte al comienzo de cada mes para planificar con anticipación. Puede establecer el rango de fechas para cubrir todo el mes y así tener tiempo de preparar tarjetas o anuncios.
:::

:::info
Los datos de cumpleaños provienen del campo de fecha de nacimiento en el perfil de cada persona. Asegúrese de que sus [registros de membresía](../people/adding-people.md) estén actualizados para obtener resultados precisos.
:::
