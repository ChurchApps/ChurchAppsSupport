# Reportes de Asistencia

<div class="article-intro">

B1 Admin proporciona tres reportes de asistencia para ayudarle a comprender cómo las personas están participando en sus servicios y grupos. Cada reporte ofrece una perspectiva diferente de sus datos de asistencia, desde tendencias generales hasta desgloses diarios.

</div>

<div class="prereqs">
<h4>Antes de Comenzar</h4>

- Asegúrese de que la asistencia se esté [registrando de manera consistente](../attendance/tracking-attendance.md) para sus servicios y grupos
- Asegúrese de que sus [grupos](../groups/creating-groups.md) y servicios estén configurados en B1 Admin
- Necesita los [permisos](../settings/roles-permissions.md) apropiados para acceder a los reportes

</div>

## Tendencia de Asistencia

El reporte de Tendencia de Asistencia muestra cómo cambia la asistencia a lo largo del tiempo para sus servicios.

1. Navegue a **Reportes** y haga clic en la tarjeta **Tendencia de Asistencia**.
2. Establezca el **rango de fechas** para definir el período de tiempo.
3. Seleccione un **servicio** o **grupo** para filtrar los resultados.
4. El reporte muestra una línea de tendencia con los conteos de asistencia durante el período seleccionado.

Este reporte es útil para detectar patrones como caídas estacionales, tendencias de crecimiento o el impacto de eventos especiales.

## Asistencia por Grupo

El reporte de Asistencia por Grupo muestra los totales de asistencia por grupo para un rango de fechas seleccionado.

1. Navegue a **Reportes** y haga clic en la tarjeta **Asistencia por Grupo**.
2. Establezca el **rango de fechas** para el reporte.
3. Seleccione el o los **grupos** que desea revisar.
4. El reporte muestra las cifras totales de asistencia para cada grupo.

Use este reporte para comparar la asistencia entre grupos e identificar cuáles están creciendo o necesitan atención.

## Asistencia Diaria por Grupo

El reporte de Asistencia Diaria por Grupo proporciona un desglose día a día de los datos de asistencia de sus grupos.

1. Navegue a **Reportes** y haga clic en la tarjeta **Asistencia Diaria por Grupo**.
2. Establezca el **rango de fechas** para el reporte.
3. Seleccione el o los **grupos** que desea revisar.
4. El reporte muestra los números de asistencia para cada día individual dentro del rango.

Este reporte le brinda un detalle granular, lo cual es útil para comprender la variación semana a semana o identificar días específicos con asistencia inusualmente alta o baja.

:::tip
Use el reporte de Tendencia de Asistencia para una visión general de alto nivel y el reporte de Asistencia Diaria por Grupo cuando necesite profundizar en fechas específicas.
:::

## Usos Prácticos

- **Planificación** -- Use las tendencias de asistencia para planificar asientos, personal y recursos para los próximos servicios.
- **Alcance** -- Identifique patrones de asistencia en declive de manera temprana para poder dar seguimiento a los miembros.
- **Informes para la junta** -- Incluya datos de asistencia en sus informes regulares de liderazgo para mostrar la salud del ministerio.
- **Evaluación de eventos** -- Compare la asistencia antes y después de eventos especiales para medir su impacto.

:::warning
Los datos de asistencia se registran a través de sus procesos de registro de grupos y servicios. Si la asistencia no se registra de manera consistente, sus reportes no reflejarán con precisión la participación real. Consulte [Registro de Asistencia](../attendance/tracking-attendance.md) para instrucciones de configuración.
:::
