# Formularios

<div class="article-intro">

La sección de Formularios le permite crear formularios personalizados para inscripciones, encuestas, recopilación de datos y más. Ya sea que necesite una tarjeta de visitante, un registro para un evento o un formulario de solicitud de oración, puede crearlo aquí y compartirlo con su congregación a través de un enlace público o su sitio web de B1.church.

</div>

## La Página de Formularios

Cuando abra **Formularios**, verá dos pestañas:

- **Formularios** -- Muestra todos sus formularios activos.
- **Archivados** -- Muestra los formularios que han sido archivados. Puede restaurar formularios archivados en cualquier momento.

Cada formulario en la lista muestra su nombre y, si está configurado, una URL pública que puede compartir externamente.

## Lo que Puede Hacer

Desde la página de Formularios, puede:

- **Agregar un formulario** -- Haga clic en **Agregar Formulario** para crear un nuevo formulario personalizado.
- **Editar un formulario** -- Haga clic en el nombre de un formulario para abrirlo y modificar sus preguntas, miembros o configuraciones.
- **Archivar un formulario** -- Mueva los formularios que ya no necesita a la pestaña de **Archivados** para mantener su lista organizada.
- **Restaurar un formulario** -- Vaya a la pestaña de **Archivados** y restaure cualquier formulario que desee activar nuevamente.

## Trabajar con un Formulario

Haga clic en cualquier nombre de formulario para abrirlo. El editor de formularios tiene tres pestañas:

- **Preguntas** -- Construya la estructura del formulario agregando campos con diferentes tipos. Consulte [Crear Formularios](./creating-forms.md) para más detalles.
- **Miembros** -- Controle quién puede acceder y gestionar el formulario.
- **Respuestas** -- Vea las respuestas que se han enviado. Consulte [Gestión de Respuestas](./managing-submissions.md) para más detalles.

:::tip
Los formularios con una URL pública se pueden compartir mediante enlace, incrustar en su sitio web o incluir en correos electrónicos. Esto facilita la recopilación de información de personas que pueden no tener una cuenta en la iglesia.
:::

:::info
Puede controlar quién tiene acceso a cada formulario usando la pestaña de **Miembros**. Asigne roles de **Administrador** o **Solo Lectura** para dar a su equipo el nivel de acceso adecuado.
:::
