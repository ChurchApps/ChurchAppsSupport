# Calendarios

<div class="article-intro">

La sección de Calendarios en B1 Admin te permite crear y gestionar calendarios de eventos para tu iglesia. Puedes configurar calendarios para diferentes ministerios, agregar eventos de múltiples grupos y publicar calendarios en tu sitio web B1.church para que tu congregación pueda verlos.

</div>

## Ver tus calendarios

1. En B1 Admin, navega a la sección **Calendarios**.
2. La página de Calendarios muestra todos tus calendarios con su nombre y estado.
3. Cada calendario también muestra los grupos asociados a él.

## Lo que puedes hacer

- **Agregar un nuevo calendario** -- Haz clic en el botón de agregar para crear un calendario para un ministerio, tipo de evento o propósito específico. Consulta [Crear calendarios](creating-calendars) para instrucciones paso a paso.
- **Editar un calendario** -- Haz clic en cualquier calendario para abrirlo y gestionar sus eventos y grupos asociados.
- **Publicar calendarios** -- Los calendarios pueden mostrarse en tu sitio web B1.church para que los visitantes puedan ver los próximos eventos de tu iglesia.
- **Crear calendarios curados** -- Reúne eventos de múltiples grupos en una sola vista de calendario para una visión completa de todo lo que sucede en tu iglesia. Consulta [Calendario curado](curated-calendar) para más detalles.

:::info
Los calendarios trabajan en estrecha relación con los [Grupos](../groups/creating-groups.md). Cuando los grupos tienen eventos programados, esos eventos pueden incorporarse a tus calendarios. Consulta la página [Crear calendarios](creating-calendars) para aprender cómo conectar grupos a calendarios.
:::
