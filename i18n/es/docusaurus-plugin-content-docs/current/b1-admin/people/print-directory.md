# Imprimir directorio

<div class="article-intro">

La función de Imprimir Directorio genera un directorio de miembros formateado e imprimible para tu iglesia. Organiza los miembros por hogar, incluye información de contacto y fotos, y está diseñado para verse profesional cuando se imprime en papel de tamaño carta estándar.

</div>

<div class="prereqs">
<h4>Antes de comenzar</h4>

- Necesitas una cuenta activa de B1 Admin con permiso para ver personas.
- Las personas en tu directorio deben tener fotos e información de contacto completadas para los mejores resultados.

</div>

## Abriendo el directorio de impresión

1. Navega a **Personas** en B1 Admin.
2. Haz clic en el **icono de impresión** (icono de impresora) en la barra de herramientas en la parte superior de la tarjeta de resultados de Personas. Aparece junto al botón Exportar.
3. El Directorio de Impresión se abre en una nueva pestaña del navegador y comienza a cargar los datos de tu directorio.
4. Después de aproximadamente 1.5 segundos, el diálogo de impresión de tu navegador aparecerá automáticamente.

:::tip
El directorio se abre en una pestaña separada para que puedas imprimirlo sin dejar la página de Personas. Después de que aparezca el diálogo de impresión, la pestaña se navega automáticamente de vuelta.
:::

## Lo que se incluye

El directorio de impresión incluye todos los **miembros activos** de tu iglesia. Específicamente:

- Las personas con un estado de membresía **Inactivo** o **Visitante** se excluyen automáticamente.
- Las personas que han **optado por no participar** del directorio se excluyen.
- Todos los demás se agrupan por **hogar**.

## Disposición del directorio

El directorio impreso tiene dos partes:

### Página de portada

La primera página incluye:
- El nombre de tu iglesia
- El título "Directorio de miembros"
- El año actual
- La ubicación de tu iglesia
- Un recuento de los hogares totales incluidos

### Listados de miembros

Las páginas restantes listan hogares alfabéticamente por apellido. Cada tarjeta de hogar muestra:

- **Fotos** -- Miniaturas circulares para cada miembro del hogar, con su nombre debajo
- **Nombre de visualización del hogar** -- Por ejemplo, "La familia Smith", "John & Jane Doe", o solo el nombre de la persona para individuos
- **Dirección** -- Dirección de calle, ciudad, estado y código postal
- **Teléfono** -- Número de hogar o móvil
- **Correo electrónico** -- Dirección de correo electrónico principal
- **Cumpleaños** -- Formateado como fechas individuales para cada miembro del hogar
- **Aniversario** -- Si se registró

Los hogares se organizan en **secciones alfabéticas** (A, B, C, ...) con un encabezado de sección decorativo antes de cada grupo de letras. Dos tarjetas de hogar aparecen lado a lado en cada fila.

## Configuración de impresión

El directorio está formateado para **papel de tamaño carta** (8.5" × 11") con márgenes de 0.5 pulgadas. Cuando aparezca el diálogo de impresión del navegador:

- Deja el tamaño de papel configurado en **Carta**
- Habilita **gráficos de fondo** (a veces llamado "Imprimir fondos") en la configuración de impresión de tu navegador si quieres que los encabezados de sección se impriman con su estilo de color
- Se recomienda la orientación vertical

:::info
El directorio respeta los saltos de página -- las tarjetas de hogar y los encabezados de sección no se dividirán entre páginas.
:::
