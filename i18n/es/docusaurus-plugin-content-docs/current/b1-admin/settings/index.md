# Configuración

<div class="article-intro">

La sección de Configuración es donde configuras la cuenta, marca, y opciones administrativas de tu iglesia en B1 Admin. Desde aquí puedes gestionar todo, desde el nombre de tu iglesia y subdominio hasta permisos de usuario y configuración de aplicación móvil.

</div>

## Qué Encontrarás Aquí

El área de Configuración se organiza en las siguientes subsecciones accesibles desde la barra lateral:

1. **Configuración** -- Configura la información básica, marca, y subdominio de tu iglesia. Consulta [Configuración de la Iglesia](./church-settings.md) para más detalles.
2. **Campuses** -- Agrega y gestiona múltiples ubicaciones físicas de tu iglesia. Una vez creados, los campuses aparecen en perfiles de personas, en la configuración de asistencia, y en el panel de Demografía. Consulta [Campuses](./campuses.md) para más detalles.
3. **Campos Personalizados** -- Define tus propios campos para rastrear en personas (una fecha, número, respuesta sí/no, o lista de selección), luego complétalos en perfiles y busca en ellos. Consulta [Campos Personalizados](./custom-fields.md) para más detalles.
3. **Aplicaciones Móviles** -- Configura y personaliza las pestañas de navegación que aparecen en la [PWA de B1 Church](/docs/b1-church/getting-started/installing-pwa) para tus miembros (las mismas pestañas también se renderizan por la aplicación nativa B1 Mobile obsoleta). Consulta [Configuración de Aplicación Móvil](./mobile-app.md) para más detalles.
4. **Administrador del Servidor** -- Accede a herramientas avanzadas de administración para gestionar la configuración a nivel de servidor de tu iglesia.
5. **Formularios** -- Crea y gestiona formularios personalizados para recopilar información de tu congregación.

## La Página Principal de Configuración

Cuando abres por primera vez la página de Configuración, verás el **nombre de tu iglesia** y **subdominio** mostrados en la parte superior. El encabezado proporciona botones de acceso rápido para tareas comunes:

- **Editar Configuración** -- Actualiza el nombre, dirección, información de contacto, y marca de tu iglesia.
- **Aplicaciones Móviles** -- Ve directamente a la configuración de aplicación móvil.
- **Roles** -- Gestiona roles de usuario y permisos para tu equipo.
- **Importar/Exportar** -- Transfiere datos entre sistemas usando la herramienta de importación/exportación.

:::tip
Comienza configurando el nombre de tu iglesia y marca bajo **Editar Configuración**, luego configura [Roles](./roles-permissions.md) para invitar a los miembros de tu equipo con el nivel de acceso correcto.
:::

:::info
Para obtener información sobre cómo se protegen tus datos, consulta [Seguridad de Datos](./data-security.md).
:::
