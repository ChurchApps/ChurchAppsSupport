# Configuración

<div class="article-intro">

La sección de Configuración es donde configuras la cuenta, marca e opciones administrativas de tu iglesia en B1 Admin. Desde aquí puedes gestionar todo, desde el nombre de tu iglesia y subdominio hasta permisos de usuario y configuración de aplicaciones móviles.

</div>

## Qué encontrarás aquí

El área de Configuración se organiza en las siguientes subsecciones accesibles desde la barra lateral:

1. **Configuración** -- Configura la información básica, marca y subdominio de tu iglesia. Ve [Configuración de la iglesia](./church-settings.md) para más detalles.
2. **Sedes** -- Agrega y gestiona múltiples ubicaciones físicas de tu iglesia. Una vez creadas, las sedes aparecen en perfiles de personas, en la configuración de asistencia y en el panel de Demografía. Ve [Sedes](./campuses.md) para más detalles.
3. **Campos personalizados** -- Define tus propios campos para rastrear en personas (una fecha, número, respuesta sí/no o lista de selección), luego complétales en perfiles y busca en ellos. Ve [Campos personalizados](./custom-fields.md) para más detalles.
3. **Aplicaciones móviles** -- Configura y personaliza las fichas de navegación que aparecen en [B1.church PWA](/docs/b1-church/getting-started/installing-pwa) para tus miembros (las mismas fichas también se renderizan por la aplicación nativa B1 Mobile deprecada). Ve [Configuración de aplicación móvil](./mobile-app.md) para más detalles.
4. **Administrador del servidor** -- Accede a herramientas avanzadas de administración para gestionar la configuración del servidor de tu iglesia.
5. **Formularios** -- Crea y gestiona formularios personalizados para recopilar información de tu congregación.

## La página principal de Configuración

Cuando abres por primera vez la página de Configuración, verás el **nombre de tu iglesia** y **subdominio** mostrados en la parte superior. El encabezado proporciona botones de acceso rápido para tareas comunes:

- **Editar configuración** -- Actualiza el nombre, dirección, información de contacto y marca de tu iglesia.
- **Aplicaciones móviles** -- Ir directamente a la configuración de aplicaciones móviles.
- **Roles** -- Gestiona roles de usuario y permisos para tu equipo.
- **Importar/Exportar** -- Transfiere datos entre sistemas usando la herramienta de importación/exportación.

:::tip
Comienza configurando el nombre de tu iglesia y marca bajo **Editar configuración**, luego configura [Roles](./roles-permissions.md) para invitar a miembros de tu equipo con el nivel de acceso correcto.
:::

:::info
Para obtener información sobre cómo se protegen tus datos, ve [Seguridad de datos](./data-security.md).
:::
