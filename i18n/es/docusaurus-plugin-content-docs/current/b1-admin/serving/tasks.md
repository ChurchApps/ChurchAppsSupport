# Tareas

<div class="article-intro">

Las tareas le permiten asignar elementos de acción a personas o grupos en su iglesia. Ya sea una actividad puntual o una responsabilidad recurrente, las tareas le ayudan a hacer seguimiento de lo que necesita hacerse y quién es el responsable, para que nada se pase por alto.

</div>

<div class="prereqs">
<h4>Antes de Comenzar</h4>

- Asegúrese de que las personas o [grupos](../groups/creating-groups.md) a los que desea asignar tareas existan en B1 Admin
- Navegue a la sección **Servicio Voluntario** para acceder a la página de Tareas

</div>

## Ver Tareas

Navegue a **Servicio Voluntario** y abra la página de **Tareas**. Verá una lista de sus tareas abiertas. Cada tarea muestra su título, persona asignada y estado.

:::tip
Por defecto, solo se muestran las tareas abiertas. Active **Mostrar Cerradas** para ver también las tareas completadas.
:::

## Crear una Tarea

1. En la página de Tareas, haga clic en **Agregar Tarea**.
2. Ingrese un **título** para la tarea.
3. Haga clic en el cuadro **Asignar A** y escriba el nombre de una persona, o cambie a **Grupo** y escriba el nombre del grupo.
4. Haga clic en **Buscar** y seleccione la persona o grupo de los resultados.
5. Agregue una **nota** con cualquier detalle o instrucción para la persona asignada.
6. Haga clic en **Guardar**.

La tarea aparecerá ahora en la lista de tareas y será visible para la persona asignada en su panel de control y aplicación móvil.

## Gestionar una Tarea

Haga clic en cualquier tarea para abrir su página de detalle. Desde aquí puede:

- **Cambiar el estado** -- Alterne entre **Abierta** y **Cerrada** usando el menú desplegable de estado.
- **Reasignar la tarea** -- Actualice la persona asignada a una persona o grupo diferente.
- **Agregar notas** -- Incluya información adicional o actualizaciones para la persona asignada.
- **Editar el título** -- Actualice el título de la tarea según sea necesario.

## Ciclo de Vida de la Tarea

1. Una nueva tarea comienza con el estado **Abierta**.
2. La persona asignada recibe una notificación y puede ver la tarea desde su panel de control, la página de Tareas o la aplicación móvil.
3. Una vez completado el trabajo, la persona asignada (o un administrador) cambia el estado a **Cerrada**.
4. Las tareas cerradas se eliminan de la vista predeterminada, pero aún se puede acceder a ellas activando **Mostrar Cerradas**.

:::info
Puede automatizar tareas recurrentes para que se creen según un cronograma. Consulte [Automatizaciones](./automations.md) para obtener detalles sobre cómo configurarlo.
:::
