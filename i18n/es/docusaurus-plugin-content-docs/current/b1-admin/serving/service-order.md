# Orden del Servicio

<div class="article-intro">

El orden del servicio define la secuencia de elementos en tu servicio de adoración. Úsalo para diseñar todo, desde la canción de apertura hasta la oración final, para que todo tu equipo conozca el flujo del servicio con anticipación.

</div>

<div class="prereqs">
<h4>Antes de Empezar</h4>

- [Crea un plan de servicio](./plans.md) para la fecha para la que deseas construir un orden del servicio
- Opcionalmente, agrega [canciones](./songs.md) a tu biblioteca para poder vincularlas directamente en el orden del servicio

</div>

## Accediendo al Orden del Servicio

1. Navega a **Serving** y selecciona tu ministerio.
2. Haz clic en un **tipo de plan**, luego abre un **plan** específico.
3. Haz clic en la pestaña **Service Order** en la página de detalles del plan.

## Construyendo Tu Orden del Servicio

El orden del servicio es una lista de elementos organizados en la secuencia en la que ocurrirán durante el servicio. Los elementos comunes incluyen:

- Canciones de adoración
- Oraciones
- Lecturas de las Escrituras
- Anuncios
- Sermón o mensaje
- Ofrenda
- Cierre

## Agregar Elementos al Orden del Servicio

1. En la pestaña **Service Order**, haz clic en **Add Item**.
2. Elige el tipo de elemento que deseas agregar (por ejemplo, una canción de tu biblioteca, un elemento personalizado o un encabezado).
3. Ingresa los detalles del elemento, como el título y cualquier nota.
4. Haz clic en **Save**.

:::tip
Puedes agregar canciones directamente desde tu biblioteca de [Songs](./songs.md). Esto vincula los detalles de la canción, incluida la letra y arreglos, a tu plan de servicio para que tu equipo de adoración tenga todo lo que necesita.
:::

## Reordenar Elementos

Arrastra y suelta elementos para reorganizar el orden. La secuencia que se muestra en pantalla es el orden que tu equipo seguirá durante el servicio.

## Editar y Eliminar Elementos

- Haz clic en cualquier elemento en el orden del servicio para **editarlo**.
- Usa la opción **delete** para eliminar un elemento del orden.

:::info
El orden del servicio es específico para cada plan. Los cambios que realices aquí solo afectan al plan seleccionado y no alterarán otros planes ni tu biblioteca de canciones.
:::

## Incluir Elementos en Tiempos de Servicio Específicos

Si tu plan tiene más de un tiempo de servicio (por ejemplo, un servicio a las 8 a.m. y uno a las 10 a.m.), puedes elegir en qué servicios debe aparecer cada elemento. Esto es útil cuando un anuncio es relevante solo para un servicio o cuando una canción se canta en un servicio pero no en otro.

1. Abre un plan que tenga dos o más tiempos de servicio definidos en la pestaña **Times**.
2. En la pestaña **Service Order**, haz clic en un elemento para editarlo.
3. Bajo **Include in Services**, verás una casilla de verificación para cada tiempo de servicio, etiquetada con la hora.
4. Desactiva cualquier servicio donde el elemento debe saltarse.
5. Haz clic en **Save**.

Por defecto, cada elemento se incluye en cada servicio. Los tiempos de servicio excluidos se ocultan cuando imprimes el plan filtrado a ese servicio, para que cada servicio reciba una hoja de ejecución limpia con solo los elementos relevantes.

:::tip
La sección **Include in Services** solo aparece cuando el plan tiene más de un tiempo de servicio. Si solo ves un servicio, edita la pestaña **Times** del plan para agregar servicios adicionales primero.
:::

## Compartir el Orden del Servicio

Una vez que tu orden del servicio esté completo, puedes imprimir el plan completo (incluyendo el orden del servicio) desde la página de detalles del plan. Esto le da a tu equipo un resumen completo del servicio.
