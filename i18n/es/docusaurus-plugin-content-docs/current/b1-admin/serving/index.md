# Serving

<div class="article-intro">

La sección Serving es su centro de operaciones para coordinar voluntarios y planificar servicios de iglesia. Ya sea que esté organizando adoración dominical, reuniones entre semana o eventos especiales, aquí es donde junta todo para que sus equipos sepan exactamente quién está sirviendo y cuándo.

</div>

## Cómo Funciona la Página de Serving

Cuando abre la página **Serving**, verá sus ministerios mostrados como pestañas en la parte superior. Cada ministerio tiene sus propios tipos de plan y equipos, para que pueda gestionar múltiples ministerios de forma independiente. Por ejemplo, podría tener pestañas separadas para Adoración, Ministerio Infantil y Hospitalidad.

El personal no administrativo que es miembro de un ministerio también puede acceder a la página de Serving. Solo verán los ministerios a los que pertenecen y pueden ver sus planes y asignaciones sin necesidad de permisos administrativos completos.

:::tip
Si es un administrador, alterne **Mostrar Todo** para ver todos los ministerios a la vez. Esto es útil cuando necesita verificar asignaciones en múltiples equipos.
:::

## Sub-Secciones

La sección Serving incluye tres áreas clave:

- **[Planes](./plans.md)** -- Cree planes de servicio para fechas específicas y asigne voluntarios a equipos. Los planes lo ayudan a organizar quién está sirviendo cuándo, para que todos sepan su rol con anticipación. Use la **[Descripción General de Planes](./plans-overview.md)** para ver todas las asignaciones próximas en múltiples semanas en una cuadrícula.
- **[Canciones](./songs.md)** -- Gestione su biblioteca de canciones de adoración con letra, arreglos y enlaces externos. Su equipo de adoración puede usar esto para prepararse para servicios próximos.
- **[Tareas](./tasks.md)** -- Asigne y rastreee tareas para los miembros de su equipo. Configure [Automatizaciones](./automations.md) para crear tareas recurrentes automáticamente, para que nada se pierda.

## Primeros Pasos

1. Navegue a **Serving** desde el menú principal en B1 Admin.
2. Haga clic en el botón agregar ministerio para crear un nuevo ministerio o alterne "mostrar todo" para ver pestañas de ministerio existentes. Seleccione una **pestaña de ministerio** para ver sus tipos de plan y equipos.
3. Elija "crear un tipo de plan" u elija un tipo de plan existente para comenzar [creando planes de servicio](./plans.md), o explore las sub-secciones [Canciones](./songs.md) y [Tareas](./tasks.md).

:::info
Antes de poder crear planes, necesitará crear sus Ministerios en el área de Serving y Teams dentro de cada Ministerio.
:::
