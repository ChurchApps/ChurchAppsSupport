# Servicio

<div class="article-intro">

La sección Servicio es tu hub para coordinar voluntarios y planificar servicios de iglesia. Ya sea que estés organizando el culto dominical, reuniones entre semana o eventos especiales, aquí es donde reúnes todo para que tus equipos sepan exactamente quién está sirviendo y cuándo.

</div>

## Cómo Funciona la Página de Servicio

Cuando abres la página **Servicio**, verás tus ministerios mostrados como pestañas en la parte superior. Cada ministerio tiene sus propios tipos de planes y equipos, para que puedas administrar múltiples ministerios de forma independiente. Por ejemplo, podrías tener pestañas separadas para Adoración, Ministerio Infantil y Hospitalidad.

El personal no administrador que es miembro de un ministerio también puede acceder a la página Servicio. Solo verán los ministerios a los que pertenecen y pueden crear y editar planes, asignaciones, tipos de planes y el orden de servicio para su ministerio sin necesidad de permisos de administrador completo.

:::tip
Si eres administrador, activa **Mostrar Todo** para ver todos los ministerios a la vez. Esto es útil cuando necesitas verificar asignaciones en múltiples equipos.
:::

## Subsecciones

La sección Servicio incluye estas áreas clave:

- **[Planes](./plans.md)** -- Crea planes de servicio para fechas específicas y asigna voluntarios a equipos. Los planes te ayudan a organizar quién está sirviendo cuándo, para que todos sepan su rol con anticipación. Usa la **[Descripción General de Planes](./plans-overview.md)** para ver todas las asignaciones próximas en múltiples semanas en una cuadrícula.
- **[Canciones](./songs.md)** -- Administra tu biblioteca de canciones de adoración con letras, arreglos y enlaces externos. Tu equipo de adoración puede usarlo para prepararse para los próximos servicios.
- **[Tareas](./tasks.md)** -- Asigna y registra tareas para tus miembros del equipo. Configura [Automatizaciones](./automations.md) para crear tareas recurrentes automáticamente, para que nada se quede sin hacer.
- **[Flujos de Trabajo](./workflows.md)** -- Mueve personas a través de una serie de pasos en un tablero Kanban visual, como seguimiento de nuevos visitantes o un proceso de membresía. Los pasos pueden ejecutar acciones automatizadas (enviar un correo, esperar unos días, agregar a un grupo), y las tarjetas avanzan por botones de resultado, enrutamiento condicional o disparadores de eventos y horarios -- con reportes incorporados.

## Comenzando

1. Navega a **Servicio** desde el menú principal en B1 Admin.
2. Haz clic en el botón agregar ministerio para crear un nuevo ministerio o activa "mostrar todo" para ver las pestañas de ministerio existentes. Selecciona una **pestaña de ministerio** para ver sus tipos de planes y equipos.
3. Elige "crear un tipo de plan" o elige un tipo de plan existente para empezar a [crear planes de servicio](./plans.md), o explora las subsecciones [Canciones](./songs.md) y [Tareas](./tasks.md).

:::info
Antes de que puedas crear planes, necesitarás crear tus Ministerios en el área de Servicio y Equipos dentro de cada Ministerio.
:::
