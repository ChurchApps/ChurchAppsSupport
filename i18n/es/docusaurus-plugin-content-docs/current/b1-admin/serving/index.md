# Servicio Voluntario

<div class="article-intro">

La sección de Servicio Voluntario es su centro de coordinación de voluntarios y planificación de servicios de la iglesia. Ya sea que esté organizando el culto dominical, reuniones entre semana o eventos especiales, aquí es donde reúne todo para que sus equipos sepan exactamente quién está sirviendo y cuándo.

</div>

## Cómo Funciona la Página de Servicio Voluntario

Cuando abre la página de **Servicio Voluntario**, verá sus ministerios mostrados como pestañas en la parte superior. Cada ministerio tiene sus propios tipos de planes y equipos, lo que le permite gestionar múltiples ministerios de forma independiente. Por ejemplo, podría tener pestañas separadas para Adoración, Ministerio Infantil y Hospitalidad.

:::tip
Si es administrador, active **Mostrar Todos** para ver todos los ministerios a la vez. Esto es útil cuando necesita verificar las asignaciones en varios equipos.
:::

## Subsecciones

La sección de Servicio Voluntario incluye tres áreas clave:

- **[Planes](./plans.md)** -- Cree planes de servicio para fechas específicas y asigne voluntarios a los equipos. Los planes le ayudan a organizar quién está sirviendo y cuándo, para que todos conozcan su rol con anticipación.
- **[Canciones](./songs.md)** -- Gestione su biblioteca de canciones de adoración con letras, arreglos y enlaces externos. Su equipo de adoración puede usar esto para prepararse para los próximos servicios.
- **[Tareas](./tasks.md)** -- Asigne y haga seguimiento de tareas para los miembros de su equipo. Configure [Automatizaciones](./automations.md) para crear tareas recurrentes automáticamente, para que nada se pase por alto.

## Primeros Pasos

1. Navegue a **Servicio Voluntario** desde el menú principal en B1 Admin.
2. Seleccione una **pestaña de ministerio** para ver sus tipos de planes y equipos.
3. Elija un tipo de plan para comenzar a [crear planes de servicio](./plans.md), o explore las subsecciones de [Canciones](./songs.md) y [Tareas](./tasks.md).

:::info
Antes de poder crear planes, necesitará configurar sus ministerios y equipos en el área de Configuración. Consulte la documentación de Configuración para obtener detalles sobre cómo estructurar su servicio voluntario.
:::
