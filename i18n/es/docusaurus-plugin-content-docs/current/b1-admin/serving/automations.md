# Automatizaciones

<div class="article-intro">

Las automatizaciones crean tareas automáticamente según un cronograma recurrente. En lugar de crear manualmente las mismas tareas cada semana o mes, puede configurar una automatización una vez y dejar que el sistema se encargue, asegurándose de que nada se pase por alto.

</div>

<div class="prereqs">
<h4>Antes de Comenzar</h4>

- Familiarícese con el funcionamiento de las [Tareas](./tasks.md) en B1 Admin
- Identifique qué responsabilidades recurrentes desea automatizar

</div>

## Ver Automatizaciones

Navegue a **Servicio Voluntario** y abra la página de **Automatizaciones**. Verá una lista de todas sus automatizaciones, cada una mostrando su título, estado activo y patrón de recurrencia.

## Crear una Automatización

1. En la página de Automatizaciones, haga clic en **Agregar Automatización**.
2. Ingrese un **título** para la automatización. Este también se usará como título de las tareas que cree.
3. Configure el **patrón de recurrencia** -- elija con qué frecuencia debe crearse la tarea (por ejemplo, semanal, mensual o en días específicos).
4. Asigne la tarea a una **persona o grupo** que será responsable cada vez que se ejecute.
5. Configure la automatización como **Activa** para que comience a crear tareas según el cronograma.
6. Haga clic en **Guardar**.

## Editar una Automatización

1. Haga clic en una automatización existente en la lista para abrirla.
2. Actualice el título, patrón de recurrencia, asignación u otras configuraciones.
3. Haga clic en **Guardar** para aplicar sus cambios.

## Activar y Desactivar

Puede controlar si una automatización está en ejecución sin eliminarla:

- **Activa** -- La automatización creará tareas según su cronograma.
- **Inactiva** -- La automatización está pausada y no creará nuevas tareas hasta que la reactive.

:::tip
Configure una automatización como **Inactiva** durante períodos de vacaciones o temporadas especiales cuando la tarea recurrente no sea necesaria. Puede reactivarla en cualquier momento sin perder su configuración.
:::

## Cómo Funciona

Cuando se ejecuta una automatización, crea una nueva [tarea](./tasks.md) con el título y la asignación configurados. La tarea se comporta como cualquier otra tarea -- las personas asignadas reciben notificaciones y pueden gestionar la tarea desde su panel de control o la página de Tareas.

:::info
Las automatizaciones solo crean nuevas tareas hacia adelante. No crean tareas retroactivamente para fechas pasadas.
:::
