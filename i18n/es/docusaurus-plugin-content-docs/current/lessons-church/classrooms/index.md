# Aulas

<div class="article-intro">

Las aulas son la base de cómo organiza la enseñanza en su iglesia en Lessons.church. Un aula representa un espacio físico o virtual donde se imparten las lecciones, por ejemplo, "Sala de Preescolar", "Primaria K-2" o "Estudio Bíblico de Adultos". Esta sección cubre la creación de aulas y la programación de lecciones en ellas.

</div>

## Por qué son importantes las aulas

Cuando configura aulas, crea una estructura clara para su ministerio infantil, grupos juveniles o cualquier entorno de enseñanza. Cada aula tiene su propio horario, para que sus voluntarios siempre sepan exactamente qué enseñar y cuándo. En lugar de pasar horarios en papel o enviar correos electrónicos de último momento, todos pueden consultar el portal para ver lo que viene.

## Lo que puede hacer con las aulas

- **Crear aulas ilimitadas** -- Configure tantas aulas como su iglesia necesite. No hay límite, así que puede tener aulas separadas para cada grupo de edad, sede o programa.
- **Programar lecciones** -- Asigne lecciones específicas a cada aula en fechas específicas. Su horario muestra tanto las lecciones pasadas como las próximas de un vistazo.
- **Personalizar contenido** -- Adapte el contenido de las lecciones para cada aula para que los voluntarios solo vean lo que es relevante para su grupo de edad. Consulte [Personalización](../customization/index.md) para más detalles.

## Cómo se conectan las aulas

Las aulas conectan el contenido de lecciones en Lessons.church con el horario de enseñanza real de su iglesia. Este es el flujo de trabajo típico:

1. [Explore](../browsing/index.md) los programas y estudios disponibles en Lessons.church.
2. Configure sus aulas para que coincidan con los salones y grupos de edad de su iglesia.
3. Programe lecciones en cada aula para las próximas fechas.
4. Los voluntarios abren su aula para ver lo que está programado y prepararse.
