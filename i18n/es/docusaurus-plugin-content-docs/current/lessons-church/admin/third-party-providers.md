# Proveedores de terceros

<div class="article-intro">

Lessons.church admite proveedores de currículo externos a través del Formato Abierto de Lecciones. Esto significa que puede importar contenido de otras organizaciones y hacerlo disponible junto con la biblioteca integrada, brindando a las iglesias acceso a currículo de múltiples fuentes en un solo lugar.

</div>

<div class="prereqs">
<h4>Antes de comenzar</h4>

- Tenga permisos de administrador para Lessons.church (consulte la [descripción general de Administración](./index.md))
- Obtenga la URL del feed del proveedor externo que desea conectar

</div>

## Agregar un proveedor

1. Navegue al área de **Admin**.
2. Abra la página de **Third-Party**.
3. Haga clic en **Agregar proveedor** (o el botón equivalente para agregar una nueva conexión).
4. Ingrese la **URL del feed** del proveedor. Esta es la URL donde el proveedor publica sus datos de currículo en el Formato Abierto de Lecciones.
5. Guarde la conexión.

Una vez guardado, Lessons.church importará el contenido del proveedor. Sus programas, estudios y lecciones aparecerán en la biblioteca de currículo junto con su propio contenido.

## Gestionar proveedores

Desde la página de **Third-Party**, puede ver todos los proveedores conectados. Para cada proveedor, puede:

- **Ver** el contenido importado para confirmar que se importó correctamente.
- **Actualizar** la URL del feed si el proveedor cambia su punto de acceso.
- **Eliminar** un proveedor si ya no desea incluir su contenido.

## ¿Qué es el Formato Abierto de Lecciones?

El Formato Abierto de Lecciones es un esquema estandarizado que describe el contenido del currículo -- programas, estudios, lecciones, ambientes, secciones y acciones -- de manera estructurada. Cualquier organización que publique su contenido usando este formato puede ser agregada como proveedor en Lessons.church.

Si usted es un proveedor de contenido interesado en hacer que su currículo esté disponible a través de Lessons.church, puede encontrar la documentación completa del esquema en el repositorio de [Open Lesson Schema](https://github.com/LiveChurchSolutions/LessonsScreen).

## ¿Por qué usar proveedores de terceros?

- **Más opciones para las iglesias** -- Las iglesias pueden explorar una mayor variedad de currículo sin salir de la plataforma.
- **Todo en un solo lugar** -- Los voluntarios y coordinadores no necesitan visitar múltiples sitios web para encontrar y programar lecciones.
- **Configuración fácil** -- Agregar un proveedor solo requiere una URL de feed. No hay entrada manual de datos ni carga de archivos involucrada.

:::info
El contenido de terceros es gestionado por el proveedor externo. Si nota contenido faltante o desactualizado de un proveedor, es posible que el problema deba resolverse de su parte.
:::

:::tip
Para la configuración de proveedores externos a nivel de iglesia (no a nivel de administrador), consulte la página de [Proveedores externos](../browsing/external-providers.md) en la sección de Exploración.
:::
