# Proveedores de Terceros

<div class="article-intro">

Lessons.church admite proveedores de currículum externos a través del Formato Abierto de Lecciones. Esto significa que puede importar contenido de otras organizaciones y ponerlo a disposición junto con la biblioteca integrada, dando a las iglesias acceso a currículum de múltiples fuentes en un solo lugar.

</div>

<div class="prereqs">
<h4>Antes de Comenzar</h4>

- Tener permisos de administrador para Lessons.church (consulte la [descripción general de Administración](./index.md))
- Obtener la URL de feed del proveedor externo que desea conectar

</div>

## Agregar un Proveedor

1. Navegue al área **Admin**.
2. Abra la página **Third-Party** (Terceros).
3. Haga clic en **Add Provider** (Agregar Proveedor) (o el botón equivalente para agregar una nueva conexión).
4. Ingrese el **feed URL** del proveedor. Esta es la URL donde el proveedor publica sus datos de currículum en el Formato Abierto de Lecciones.
5. Guarde la conexión.

Una vez guardado, Lessons.church importará el contenido del proveedor. Sus programas, estudios y lecciones aparecerán en la biblioteca de currículum junto con su propio contenido.

## Administración de Proveedores

Desde la página **Third-Party** (Terceros), puede ver todos los proveedores conectados. Para cada proveedor, puede:

- **Ver** el contenido importado para confirmar que se extrajo correctamente.
- **Actualizar** la URL de feed si el proveedor cambia su punto final.
- **Eliminar** un proveedor si ya no desea incluir su contenido.

## ¿Qué es el Formato Abierto de Lecciones?

El Formato Abierto de Lecciones es un esquema estandarizado que describe contenido de currículum -- programas, estudios, lecciones, lugares, secciones y acciones -- de manera estructurada. Cualquier organización que publique su contenido usando este formato puede agregarse como proveedor en Lessons.church.

Si es un proveedor de contenido interesado en poner su currículum disponible a través de Lessons.church, puede encontrar la documentación completa del esquema en la guía [Formato Abierto de Lecciones](./open-lesson-format.md).

## ¿Por Qué Usar Proveedores de Terceros?

- **Más opciones para iglesias** -- Las iglesias pueden explorar una variedad más amplia de currículum sin salir de la plataforma.
- **Un lugar para todo** -- Los voluntarios y coordinadores no necesitan visitar múltiples sitios web para encontrar y programar lecciones.
- **Configuración fácil** -- Agregar un proveedor requiere solo una URL de feed. No hay entrada de datos manual ni carga de archivos.

:::info
El contenido de terceros es administrado por el proveedor externo. Si nota contenido faltante o desactualizado de un proveedor, el problema puede necesitar resolverse de su parte.
:::

:::tip
Para la configuración de proveedor externo a nivel de iglesia (no a nivel de administrador), consulte la página [Proveedores Externos](../browsing/external-providers.md) en la sección Exploración.
:::
