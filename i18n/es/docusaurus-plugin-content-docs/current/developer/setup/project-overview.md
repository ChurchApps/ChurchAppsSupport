# Descripción del proyecto

<div class="article-intro">

ChurchApps consta de aproximadamente 20 repositorios independientes, cada uno publicado bajo la Organización ChurchApps de GitHub.

</div>

## APIs de backend

Todas las APIs se construyen con Node.js, Express y TypeScript.

| Proyecto | Propósito |
|---------|----------|
| **Api** | Monolito modular principal |
| **LessonsApi** | Backend de Lessons.church |

## Aplicaciones web

| Proyecto | Propósito |
|---------|----------|
| **B1Admin** | Panel de administración de la iglesia |
| **B1App** | Aplicación de miembro |

## Librerías compartidas

| Paquete | Propósito |
|---------|---------|
| `@churchapps/helpers` | Utilidades base |
| `@churchapps/apihelper` | Utilidades del servidor Express |
| `@churchapps/apphelper` | Componentes React |
