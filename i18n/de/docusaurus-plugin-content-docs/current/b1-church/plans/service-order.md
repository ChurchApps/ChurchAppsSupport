# Gottesdienstablauf

<div class="article-intro">

Wenn Sie einen Plan öffnen, zeigt der Abschnitt **Gottesdienstablauf** die Reihenfolge der Elemente für diesen Gottesdienst. Dies gibt Ihnen und Ihrem Team eine gemeinsame Übersicht über den Ablauf des Gottesdienstes, einschließlich Zeitplanung, Lieddetails und Beschreibungen für jedes Element.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie müssen in Ihrem B1.church-Konto [angemeldet](../getting-started/logging-in.md) sein.
- Öffnen Sie einen Plan aus Ihrer [Planliste](./viewing-plans.md), um dessen Gottesdienstablauf einzusehen.

</div>

## Was der Gottesdienstablauf zeigt

Der Gottesdienstablauf wird als Liste von Elementen in der Reihenfolge angezeigt, in der sie während des Gottesdienstes stattfinden. Jedes Element enthält:

- **Kumulative Zeit** -- Die laufende Zeit ab Beginn des Gottesdienstes, sodass Sie wissen, wann jedes Element voraussichtlich beginnt.
- **Elementbezeichnung und Beschreibung** -- Der Name des Elements (z. B. ein Liedtitel, Gebet oder Ankündigung) zusammen mit Notizen oder Beschreibungen.
- **Dauer** -- Wie lange das Element voraussichtlich dauert.

Elemente sind unter **Überschriften** organisiert, die zusammengehörige Elemente gruppieren (zum Beispiel „Lobpreis-Set" oder „Predigt"). So können Sie die Gesamtstruktur des Gottesdienstes leicht überblicken.

## Lehrplan- und Lektionsinhalte

Wenn der Plan zugehörige Lehrplan- oder Lektionsinhalte von einem Anbieter wie [Lessons.church](../content/lessons.md) enthält, sehen Sie eine Vorschau dieser Inhalte innerhalb des Gottesdienstablaufs. Dies kann Lektionsabschnitte, Präsentationsmaterialien und andere Ressourcen umfassen, die Ihr Team für den Gottesdienst benötigt.

## Den Gottesdienstablauf drucken

Sie können eine formatierte Version des Gottesdienstablaufs drucken, um sie am Tag des Gottesdienstes mitzubringen.

1. Öffnen Sie den Plan aus Ihrer **Pläne**-Liste.
2. Klicken Sie im Abschnitt **Gottesdienstablauf** auf das **Druck**-Symbol in der oberen rechten Ecke.
3. Ein neues Fenster öffnet sich mit einem druckbaren Layout, das Folgendes enthält:
   - Das **Gottesdienstdatum** oben.
   - **Teamzuweisungen** links nach Kategorie gruppiert, mit jeder Position und der Person, die sie ausfüllt.
   - Den **vollständigen Gottesdienstablauf** rechts mit Spalten für Zeit, Elementdetails und Dauer.
4. Der Druckdialog Ihres Browsers öffnet sich automatisch, sodass Sie drucken oder als PDF speichern können.

:::tip
Der gedruckte Gottesdienstablauf ist eine großartige Referenz für Proben oder den Gottesdienst selbst. Er enthält sowohl die Teamübersicht als auch den vollständigen Gottesdienstablauf auf einer einzigen Seite.
:::
