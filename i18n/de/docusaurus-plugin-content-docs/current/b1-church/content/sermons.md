# Predigten

<div class="article-intro">

Auf der Predigten-Seite können Sie aufgezeichnete Predigten Ihrer Gemeinde durchstöbern und anschauen. Die Predigtbibliothek Ihrer Gemeinde wird im Erscheinungsbild und Layout angezeigt, das Ihre Gemeinde konfiguriert hat, sodass das Aussehen zum Branding Ihrer Gemeinde passt.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Zum Anzeigen von Predigten ist keine Anmeldung erforderlich. Die Predigten-Seite ist öffentlich zugänglich.
- Ihre Gemeinde muss Predigtinhalte über ihr Admin-Portal hinzugefügt haben.

</div>

## Predigten durchstöbern

1. Navigieren Sie über die Seitennavigation Ihrer Gemeinde zur **Predigten**-Seite.
2. Sie sehen eine Auflistung der verfügbaren Predigten, in der Regel nach Playlist oder Serie geordnet.
3. Klicken Sie auf eine Predigt, um sie anzuschauen oder anzuhören.

Predigten können Video- oder Audioinhalte enthalten, die auf Plattformen wie YouTube, Vimeo oder Facebook gehostet werden. Wenn Sie auf eine Predigt klicken, wird das Medium direkt auf der Seite abgespielt.

## Was Sie sehen werden

Jeder Predigteintrag kann Folgendes anzeigen:

- **Titel** -- Der Name der Predigt
- **Redner** -- Wer die Botschaft gehalten hat
- **Datum** -- Wann die Predigt aufgenommen oder veröffentlicht wurde
- **Beschreibung** -- Eine Zusammenfassung des Predigtinhalts
- **Vorschaubild** -- Ein Vorschaubild der Predigt

Ihre Gemeinde organisiert Predigten in Playlists oder Serien. So können Sie einer mehrteiligen Serie leicht folgen oder Predigten zu einem bestimmten Thema finden.

:::tip
Wenn Sie nach einer Live-Übertragung statt einer aufgezeichneten Predigt suchen, schauen Sie stattdessen auf die [Livestreaming](./live-streaming.md)-Seite.
:::

## Verfügbarkeit

Die Predigten-Seite ist verfügbar, wenn Ihre Gemeinde Predigtinhalte über ihr Admin-Portal hinzugefügt hat. Wenn Sie keinen **Predigten**-Link in der Navigation Ihrer Gemeinde sehen, hat Ihre Gemeinde diese Funktion möglicherweise noch nicht konfiguriert. Wenden Sie sich für weitere Informationen an Ihren Gemeindeadministrator.
