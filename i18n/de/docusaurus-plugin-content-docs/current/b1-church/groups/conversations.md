# Gruppengespräche

<div class="article-intro">

Die Registerkarte „Gespräche" auf Ihrer Gruppenseite ist ein Ort, an dem Gruppenmitglieder Nachrichten posten und miteinander interagieren können. Nutzen Sie Gespräche, um Themen zu diskutieren, Neuigkeiten zu teilen und mit Ihrer Gruppe zwischen den Treffen in Verbindung zu bleiben.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie müssen bei Ihrem B1.church-Konto angemeldet sein.
- Sie müssen Mitglied der Gruppe sein, um auf die Registerkarte „Gespräche" zuzugreifen. Siehe „Gruppendetails" für Informationen zum Beitritt einer Gruppe.

</div>

## Auf Gespräche zugreifen

1. Navigieren Sie zur Gruppendetailseite, indem Sie auf eine Gruppe von der Gruppenseite aus klicken.
2. Stellen Sie sicher, dass Sie angemeldet sind.
3. Klicken Sie in der Seitenleiste auf die Registerkarte „Gespräche".

## Diskussionen und Ankündigungen

Die Registerkarte „Gespräche" kann zwei Unterregisterkarten anzeigen:

- Diskussionen -- Ein Bereich, in dem jedes Gruppenmitglied Nachrichten posten und auf andere antworten kann. Dies ist die Standardansicht und immer verfügbar.
- Ankündigungen -- Nachrichten von Gruppenleitern, die für alle Mitglieder wichtig sind. Reguläre Mitglieder können Ankündigungen lesen, aber nur Leiter können sie erstellen.

Wenn es keine Ankündigungen gibt und Sie kein Leiter sind, sehen Sie nur die Diskussionsansicht ohne die Registerkartenleiste.

## Eine Nachricht posten

1. Öffnen Sie die Registerkarte „Gespräche" auf Ihrer Gruppenseite.
2. Stellen Sie sicher, dass Sie sich auf der Unterregisterkarte „Diskussionen" befinden (falls die Registerkartenleiste sichtbar ist).
3. Geben Sie Ihre Nachricht in das Textfeld ein.
4. Senden Sie Ihre Nachricht, um sie mit der Gruppe zu teilen.

Ihre Nachricht wird für alle Mitglieder der Gruppe sichtbar sein.

## Ihre Nachrichten bearbeiten oder löschen

Sie können jede von Ihnen geposteter Nachricht bearbeiten oder löschen:

1. Fahren Sie mit der Maus über Ihre Nachricht (oder tippen Sie darauf auf Mobilgeräten), um die Schaltfläche „Weitere Optionen" (⋮) anzuzeigen.
2. Klicken Sie auf die Schaltfläche, um das Nachrichtenmenü zu öffnen.
3. Wählen Sie „Bearbeiten", um den Nachrichtentext zu ändern, oder „Löschen", um sie zu entfernen.
   - Beim Bearbeiten wird der Nachrichtentext wieder in das Eingabefeld geladen. Nehmen Sie Ihre Änderungen vor und senden Sie sie ab, um sie zu speichern.
   - Beim Löschen erscheint ein Bestätigungsdialog. Klicken Sie auf „Bestätigen", um sie dauerhaft zu entfernen.

:::info
Sie können nur Ihre eigenen Nachrichten bearbeiten oder löschen. Von anderen Mitgliedern geposteter Nachrichten zeigen die Bearbeiten-/Löschen-Optionen nicht an.
:::

## Auf Nachrichten reagieren

Im Gruppenchat können Sie auf jede Nachricht mit einem Emoji reagieren:

1. Tippen Sie auf eine Nachricht oder fahren Sie mit der Maus darüber und wählen Sie „Reaktion hinzufügen".
2. Wählen Sie eines der sechs Emojis: 👍 ❤️ 😂 🎉 🙏 😮.

Reaktionen erscheinen als kleine Chips unter der Nachricht mit einer Zählung, wie viele Personen jedes verwendet haben. Ihre eigene Reaktion wird hervorgehoben -- tippen Sie auf den Chip (oder wählen Sie dasselbe Emoji erneut), um sie zu entfernen, oder wählen Sie ein anderes Emoji, um eine weitere hinzuzufügen. Reaktionen erscheinen live für alle im Chat.

:::tip
Gruppengespräche erscheinen auch auf Ihrer Timeline, sodass Sie Diskussionen verfolgen können, ohne jede Gruppe einzeln zu besuchen.
:::

## Leiter-Funktionen

Wenn Sie ein Gruppenleiter sind, haben Sie zusätzliche Möglichkeiten auf der Registerkarte „Gespräche":

- Ankündigungen posten -- Wechseln Sie zur Unterregisterkarte „Ankündigungen" und posten Sie Nachrichten, die alle Gruppenmitglieder lesen können.
- Diskussionen posten -- Sie können sich auch wie jedes andere Mitglied an regulären Diskussionen beteiligen.

:::info
Gespräche sind spezifisch für jede Gruppe. Von Ihnen in einer Gruppe geposteter Nachrichten erscheinen nicht in den Gesprächen einer anderen Gruppe.
:::
