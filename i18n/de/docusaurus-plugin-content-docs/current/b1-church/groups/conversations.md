# Gruppenunterhaltungen

<div class="article-intro">

Der Unterhaltungen-Tab auf Ihrer Gruppenseite ist ein Bereich, in dem Gruppenmitglieder Nachrichten posten und miteinander interagieren können. Nutzen Sie die Unterhaltungen, um Themen zu diskutieren, Neuigkeiten zu teilen und zwischen den Treffen mit Ihrer Gruppe in Verbindung zu bleiben.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie müssen in Ihrem B1.church-Konto [angemeldet](../getting-started/logging-in.md) sein.
- Sie müssen Mitglied der Gruppe sein, um auf den Unterhaltungen-Tab zugreifen zu können. Siehe [Gruppendetails](./group-details.md) für Informationen zum Beitritt zu einer Gruppe.

</div>

## Auf Unterhaltungen zugreifen

1. Navigieren Sie zur Gruppendetailseite, indem Sie auf eine Gruppe von der [Gruppenseite](./browsing-groups.md) aus klicken.
2. Stellen Sie sicher, dass Sie angemeldet sind.
3. Klicken Sie auf den **Unterhaltungen**-Tab in der Seitenleiste.

## Diskussionen und Ankündigungen

Der Unterhaltungen-Tab kann zwei Unter-Tabs anzeigen:

- **Diskussionen** -- Ein Bereich, in dem jedes Gruppenmitglied Nachrichten posten und auf andere antworten kann. Dies ist die Standardansicht und immer verfügbar.
- **Ankündigungen** -- Nachrichten, die von Gruppenleitern gepostet werden und für alle Mitglieder wichtig sind. Reguläre Mitglieder können Ankündigungen lesen, aber nur Leiter können sie erstellen.

Wenn es keine Ankündigungen gibt und Sie kein Leiter sind, sehen Sie nur die Diskussionsansicht ohne die Tab-Leiste.

## Eine Nachricht posten

1. Öffnen Sie den **Unterhaltungen**-Tab auf Ihrer Gruppenseite.
2. Stellen Sie sicher, dass Sie sich auf dem Unter-Tab **Diskussionen** befinden (falls die Tab-Leiste sichtbar ist).
3. Geben Sie Ihre Nachricht in das Textfeld ein.
4. Senden Sie Ihre Nachricht, um sie mit der Gruppe zu teilen.

Ihre Nachricht wird für alle Mitglieder der Gruppe sichtbar sein.

:::tip
Gruppenunterhaltungen erscheinen auch auf Ihrer [Chronik](../community/timeline.md), sodass Sie Diskussionen verfolgen können, ohne jede Gruppe einzeln besuchen zu müssen.
:::

## Leiterfunktionen

Wenn Sie ein Gruppenleiter sind, haben Sie im Unterhaltungen-Tab zusätzliche Möglichkeiten:

- **Ankündigungen posten** -- Wechseln Sie zum Unter-Tab **Ankündigungen** und posten Sie Nachrichten, die alle Gruppenmitglieder lesen können.
- **An Diskussionen teilnehmen** -- Sie können auch wie jedes andere Mitglied an regulären Diskussionen teilnehmen.

:::info
Unterhaltungen sind gruppenspezifisch. Nachrichten, die Sie in einer Gruppe posten, erscheinen nicht in den Unterhaltungen einer anderen Gruppe.
:::
