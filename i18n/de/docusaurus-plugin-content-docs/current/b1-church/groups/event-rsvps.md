# Event RSVPs

<div class="article-intro">

Gruppen-Events können RSVPs sammeln, damit Leiter wissen, wer kommt. Im Kalender deiner Gruppe fragt jedes bevorstehende Event "Wirst du teilnehmen?" – antworte mit einem Tippen, ändere deine Meinung jederzeit und sehe auf einen Blick, wie viele andere gehen.

</div>

<div class="prereqs">
<h4>Bevor du beginnst</h4>

- Du musst [angemeldet](../getting-started/logging-in.md) sein und ein Mitglied der Gruppe
- Das Event muss RSVPs zulassen (sie sind standardmäßig an; Leiter können sie pro Event ausschalten)

</div>

## Reagiere auf einen Event

1. Öffne deine Gruppe im Mitglieder-Portal und gehe zu seiner **Events**-/Kalender-Ansicht.
2. Jede bevorstehende Event-Karte zeigt **Wirst du teilnehmen?** mit drei Schaltflächen: **Ja**, **Vielleicht** und **Nein**.
3. Tippe deine Antwort. Deine Auswahl wird hervorgehoben, und jede Schaltfläche zeigt eine Live-Zählung, wie viele Personen diese Antwort gegeben haben.
4. Tippe deine hervorgehobene Antwort erneut an, um sie zu löschen, oder tippe eine andere Schaltfläche, um deine Antwort zu ändern.

Dein RSVP gilt für dieses spezifische Vorkommen – für ein wöchentliches Treffen antwortest du auf die Events jeder Woche separat.

## Für Gruppenleiter

- **Anzeigen von Antworten** – Leiter sehen eine Schaltfläche **Antworten anzeigen** auf jeder Event-Karte. Sie öffnet das Blatt **Antworten**, das jeden nach Antwort auflistet – Ja, Vielleicht und Nein. Wenn niemand noch geantwortet hat, zeigt es "Keine Antworten noch".
- **RSVPs ein- oder ausschalten** – Beim Erstellen oder Bearbeiten eines Events verwende den Umschalter **RSVPs zulassen**. Events mit RSVPs ausgeschaltet zeigen keine Antwort-Schaltflächen. Personal, das Events in B1 Admin massenweise erstellt, hat die gleiche Option – siehe [Gruppenkalender](../../b1-admin/groups/group-calendar#event-rsvps).

:::tip
RSVPs sind eine leichte Kopfzahl. Wenn dein Event Anmeldungsdetails, Kapazitätsverwaltung oder Zahlung benötigt, verwende stattdessen [Event-Registrierung](../events/registering).
:::
