# Anfrage zum Beitreten einer Gruppe

<div class="article-intro">

Einige Gruppen erfordern Genehmigung, bevor Sie Mitglied werden können. Wenn eine Gruppe diese Einstellung hat, senden Sie eine Beitrittanfrage ein – optional mit einer Nachricht an den Gruppenleiter – und warten auf die Genehmigung oder Ablehnung durch den Leiter. Sie können den Status Ihrer ausstehenden Anfragen anschauen und diese jederzeit stornieren.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie müssen in Ihrem B1.church-Konto [angemeldet](../getting-started/logging-in.md) sein, um einer Gruppe beizutreten.

</div>

## Beitritt vs. Anfrage zum Beitreten

Gruppen können unterschiedliche Beitrittrichtlinien haben, die von ihren Administratoren festgelegt werden:

- **Open** – Klicken Sie auf **Join Group** auf der Seite mit Gruppendetails und Sie werden sofort als Mitglied hinzugefügt.
- **Request required** – Klicken Sie auf **Request to Join**, um eine Anfrage einzureichen, die ein Gruppenleiter genehmigen muss, bevor Sie Mitglied werden.
- **Closed** – Es wird keine Join-Schaltfläche angezeigt. Mitglieder müssen manuell von Leitern hinzugefügt werden.

## Wie Sie eine Beitrittanfrage einreichen

1. Navigieren Sie zur Gruppe, der Sie beitreten möchten. Weitere Informationen zum Finden finden Sie in [Browsing Groups](./browsing-groups.md).
2. Öffnen Sie die Seite mit Gruppendetails.
3. Klicken Sie auf **Request to Join**.
4. Ein Dialog wird angezeigt, in dem Sie optional eine Nachricht an den Gruppenleiter schreiben können, um zu erklären, warum Sie beitreten möchten.
5. Klicken Sie auf **Submit**, um Ihre Anfrage zu senden.

Die Gruppenleiter erhalten eine Benachrichtigung über Ihre Anfrage. Sobald sie diese überprüft haben, erhalten Sie eine Benachrichtigung mit dem Ergebnis.

:::tip
Das Schreiben einer kurzen Nachricht – z. B. wie Sie von der Gruppe erfahren haben oder was Sie daraus erreichen möchten – kann dem Leiter helfen, schneller eine Entscheidung zu treffen.
:::

## Überprüfung des Status Ihrer Anfragen

Nachdem Sie eine Anfrage eingereicht haben, können Sie ihren Status von der Seite **Groups** anschauen:

1. Navigieren Sie zur Seite **Groups** auf der B1.church-Website Ihrer Kirche.
2. Scrollen Sie nach unten zum Abschnitt **Pending Requests**.
3. Jede ausstehende Anfrage zeigt den Gruppennamen und das Datum, an dem Sie sie eingereicht haben.

Wenn eine Anfrage genehmigt oder abgelehnt wurde, erhalten Sie eine Benachrichtigung – genehmiigte Anfragen machen Sie zum Mitglied der Gruppe.

## Stornieren einer Anfrage

Wenn Sie Ihre Meinung ändern, bevor ein Leiter Ihre Anfrage überprüft:

1. Gehen Sie zur Seite **Groups**.
2. Finden Sie Ihre Anfrage im Abschnitt **Pending Requests**.
3. Klicken Sie neben der Anfrage auf **Cancel**.

Die Anfrage wird sofort zurückgezogen und der Leiter sieht sie nicht mehr.
