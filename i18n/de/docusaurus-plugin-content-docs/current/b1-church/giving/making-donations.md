# Eine Spende tätigen

<div class="article-intro">

Es gibt zwei Möglichkeiten, über die B1.church-Seite Ihrer Gemeinde zu spenden: die öffentliche Spendenseite (keine Anmeldung erforderlich) und die Mitglieder-Spendenseite (Anmeldung erforderlich). Beide verwenden ein sicheres Spendenformular, das von Stripe betrieben wird.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Für öffentliche Spenden wird kein Konto benötigt -- besuchen Sie einfach die **Spenden**-Seite Ihrer Gemeinde.
- Für Mitgliederspenden mit Historienverfolgung und gespeicherten Zahlungsmethoden [melden Sie sich](../getting-started/logging-in.md) zuerst bei Ihrem Konto an.

</div>

## Öffentliche Spendenseite (keine Anmeldung erforderlich)

Jeder kann ohne Erstellung eines Kontos spenden:

1. Navigieren Sie zur **Spenden**-Seite Ihrer Gemeinde.
2. Das Spendenformular wird auf der linken Seite angezeigt. Wählen Sie den **Fonds** aus, an den Sie spenden möchten.
3. Geben Sie Ihre Zahlungsdaten und den gewünschten Betrag ein.
4. Senden Sie das Formular ab, um Ihre Spende abzuschließen.

Auf der rechten Seite sehen Sie einen Link zu **Anweisungen für ACH-Zahlungen**, falls Sie per Banküberweisung spenden möchten. Es gibt auch eine **Login**-Schaltfläche, falls Sie sich anmelden möchten, um auf zusätzliche Funktionen wie die Spendenhistorie zuzugreifen.

:::tip
Wenn Sie bereits angemeldet sind, wenn Sie die öffentliche Spendenseite besuchen, werden Sie automatisch zur Mitglieder-Spendenseite weitergeleitet, wo Sie Zugriff auf Ihre vollständige Spendenhistorie und gespeicherte Zahlungsmethoden haben.
:::

## Mitglieder-Spendenseite (angemeldet)

Wenn Sie angemeldet sind, bietet die Spendenerfahrung zusätzliche Funktionen:

1. Klicken Sie auf den **Spenden**-Tab (oder navigieren Sie zu **Meine Spenden**).
2. Sie sehen einen Bereich mit drei Abschnitten:
   - **Jetzt spenden** -- Tätigen Sie eine einmalige oder wiederkehrende Spende.
   - **Historie** -- Sehen Sie Ihre vergangenen Spenden ein und exportieren Sie Aufzeichnungen.
   - **Wiederkehrende Spenden** -- Verwalten Sie geplante Gaben und Zahlungsmethoden.
3. Wählen Sie **Jetzt spenden**, um das Spendenformular zu öffnen.
4. Wählen Sie den **Fonds**, an den Sie spenden möchten, und geben Sie Ihren Spendenbetrag ein.
5. Wählen Sie eine bestehende gespeicherte Zahlungsmethode oder geben Sie neue Zahlungsdaten ein.
6. Senden Sie das Formular ab, um Ihre Spende abzuschließen.

Nach der Verarbeitung Ihrer Spende erscheint eine Erfolgsmeldung und Ihre Spende wird in Ihrer [Spendenhistorie](./donation-history.md) angezeigt.

## Zahlungsmethoden

Spenden werden sicher über **Stripe** verarbeitet. Sie können mit Kreditkarte, Debitkarte zahlen oder eine ACH-Bankzahlung einrichten. Wenn Sie angemeldet sind, können Ihre Zahlungsmethoden für zukünftige Nutzung gespeichert werden, sodass das Spenden beim nächsten Mal schneller geht.

:::warning
Wenn Ihre Spende nicht durchgeht, überprüfen Sie Ihre Zahlungsdaten und versuchen Sie es erneut. Wenn das Problem weiterhin besteht, wenden Sie sich an Ihr Gemeindebüro.
:::
