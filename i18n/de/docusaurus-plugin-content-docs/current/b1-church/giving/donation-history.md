# Spendenhistorie

<div class="article-intro">

Wenn Sie angemeldet sind, können Sie eine vollständige Aufzeichnung Ihrer vergangenen Spenden einsehen und Spendenbescheinigungen für steuerliche Zwecke erstellen. Ihre Spendenhistorie ist im Bereich **Spenden** Ihres B1.church-Kontos verfügbar.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie müssen in Ihrem B1.church-Konto [angemeldet](../getting-started/logging-in.md) sein.
- Sie müssen mindestens eine Spende getätigt haben. Siehe [Eine Spende tätigen](./making-donations.md), um zu beginnen.

</div>

## Ihre Historie einsehen

1. Melden Sie sich bei Ihrem B1.church-Konto an.
2. Navigieren Sie zum Bereich **Spenden**.
3. Klicken Sie im linken Bereich auf **Historie**, um Ihre vergangenen Spenden anzuzeigen.

Ihre Spendenhistorie wird als Tabelle mit folgenden Spalten angezeigt:

- **Datum** -- Wann die Spende getätigt wurde.
- **Methode** -- Wie Sie bezahlt haben (zum Beispiel „Karte - Visa" oder „Bank - ACH").
- **Fonds** -- An welchen Fonds die Spende gerichtet war (zum Beispiel Allgemeiner Fonds, Baufonds).
- **Betrag** -- Der Betrag der Spende.

:::info
Spenden, die noch verarbeitet werden, erscheinen mit dem Label „Ausstehend" neben dem Fondsnamen und der Betrag wird in einer anderen Farbe angezeigt.
:::

## Exportieren und Bescheinigungen drucken

Klicken Sie auf die **Download**-Schaltfläche in der Kopfzeile der Spendenhistorie, um auf Exportoptionen zuzugreifen:

- **Aktuelles Jahr (CSV)** -- Laden Sie eine CSV-Datei aller Spenden des aktuellen Jahres herunter. Sie können diese in einer Tabellenkalkulationsanwendung öffnen.
- **Aktuelles Jahr (Drucken)** -- Öffnen Sie eine druckbare Spendenbescheinigung für das aktuelle Jahr.
- **Vorjahr (CSV)** -- Laden Sie eine CSV-Datei aller Spenden des Vorjahres herunter.
- **Vorjahr (Drucken)** -- Öffnen Sie eine druckbare Spendenbescheinigung für das Vorjahr.

:::tip
Drucken Sie Ihre Spendenbescheinigung des Vorjahres frühzeitig im neuen Jahr aus, damit Sie sie für die Steuererklärung bereit haben. Die druckbare Bescheinigung enthält alle Details, die Sie für Ihre Unterlagen benötigen.
:::

## Wiederkehrende Spenden

Im Bereich **Spenden** können Sie auch auf **Wiederkehrende Spenden** klicken, um geplante Gaben zu verwalten, die Sie eingerichtet haben. In diesem Abschnitt können Sie wiederkehrende Spenden einsehen, aktualisieren oder stornieren und Ihre gespeicherten Zahlungsmethoden verwalten.
