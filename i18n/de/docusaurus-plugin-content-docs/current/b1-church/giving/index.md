# Spenden

<div class="article-intro">

B1App macht es einfach, Ihre Gemeinde durch Online-Spenden zu unterstützen. Egal ob Sie eine schnelle Einzelspende tätigen oder wiederkehrende Spenden verwalten möchten -- alles, was Sie brauchen, ist auf der B1.church-Seite Ihrer Gemeinde verfügbar.

</div>

## Überblick

B1App unterstützt zwei Möglichkeiten zu spenden:

- **Öffentliche Spenden** -- Jeder kann eine Spende tätigen, ohne sich anzumelden. Besuchen Sie die Spendenseite Ihrer Gemeinde, wählen Sie einen Fonds aus und geben Sie Ihre Zahlungsdaten ein. Es ist kein Konto erforderlich.
- **Authentifiziertes Spenden** -- Wenn Sie sich vor dem Spenden [anmelden](../getting-started/logging-in.md), werden Ihre Gaben mit Ihrem Konto verknüpft. Dadurch erhalten Sie Zugriff auf Ihre vollständige Spendenhistorie, druckbare Spendenbescheinigungen, Verwaltung wiederkehrender Spenden und gespeicherte Zahlungsmethoden.

## Was Sie tun können

Folgendes ist im Spendenbereich verfügbar:

- **An bestimmte Fonds spenden** -- Ihre Gemeinde hat möglicherweise mehrere Fonds eingerichtet, wie zum Beispiel einen Allgemeinen Fonds, Baufonds oder Missionsfonds. Sie können wählen, an welchen Fonds Ihre Spende gehen soll.
- **Mit Karte oder Bankkonto zahlen** -- Spenden werden sicher über Stripe verarbeitet. Sie können mit Kredit- oder Debitkarte zahlen. Anweisungen für ACH-Bankzahlungen sind ebenfalls verfügbar.
- **Wiederkehrende Spenden einrichten** -- Angemeldete Mitglieder können automatische wiederkehrende Spenden planen, damit Sie nie eine Gabe verpassen.
- **Spendenhistorie einsehen** -- Sehen Sie eine vollständige Liste Ihrer vergangenen Spenden ein, einschließlich Datum, Methode, Fonds und Betrag.
- **Spendenbescheinigungen drucken** -- Erstellen Sie druckbare Spendenbescheinigungen für das aktuelle oder vorherige Jahr für Ihre Steuerunterlagen.
- **Zahlungsmethoden verwalten** -- Speichern und verwalten Sie Ihre Zahlungsmethoden für schnelleres Spenden in der Zukunft.

:::info
Spenden werden sicher über **Stripe** verarbeitet. Ihre Zahlungsinformationen werden niemals auf B1.church-Servern gespeichert.
:::
