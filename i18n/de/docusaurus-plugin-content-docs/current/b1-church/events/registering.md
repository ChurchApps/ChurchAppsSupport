# Für Veranstaltungen anmelden

<div class="article-intro">

Wenn Ihre Kirche die Anmeldung für eine Veranstaltung aktiviert hat, können Sie sich über einen einfachen Schritt-für-Schritt-Prozess anmelden. Sie können sich selbst anmelden und Familienmitglieder hinzufügen, unabhängig davon, ob Sie ein Konto haben oder als Gast besuchen. Bei bezahlten Veranstaltungen wählen Sie einen Teilnehmertyp, fügen optionale Extras hinzu, wenden einen Rabattcode an und zahlen am Ende sicher.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Ihre Kirche muss die Veranstaltungsanmeldung für die Veranstaltung aktiviert haben
- Das Anmeldezeitfenster muss offen sein (zwischen den vom Administrator festgelegten Öffnungs- und Schließungsdaten)

</div>

## Schritt 1: Geben Sie Ihre Informationen ein

1. Öffnen Sie im Kirchenkalender die Veranstaltung, für die Sie sich anmelden möchten, und klicken Sie auf **Für diese Veranstaltung anmelden**.
2. Die Anmeldeseite wird angezeigt und zeigt den Veranstaltungstitel, die Beschreibung und das Datum.
3. Wenn Sie **angemeldet** sind, wird das Formular mit Ihrem Namen ausgefüllt und bestätigt, dass Sie sich unter Ihrem Konto anmelden.
4. Wenn Sie **Gast** sind (nicht angemeldet), füllen Sie aus:
   - **Vorname** (erforderlich)
   - **Nachname** (erforderlich)
   - **E-Mail** (erforderlich)
   - **Telefon** (optional)
5. Wenn die Veranstaltung Teilnehmertypen hat (zum Beispiel Erwachsene und Kinder zu unterschiedlichen Preisen), wählen Sie **Ihren Anmeldetyp**. Jeder Typ zeigt seinen Preis an, und Typen, die ausverkauft sind, werden als **Ausverkauft** angezeigt.

:::info
Wenn die Veranstaltung ein Kapazitätslimit hat, zeigt eine Fortschrittsleiste, wie viele Plätze noch verfügbar sind. Wenn die Veranstaltung voll ist und Ihre Kirche eine Warteliste aktiviert hat, können Sie die Anmeldung trotzdem abschließen, um der Warteliste beizutreten — siehe [Warteliste beitreten](#warteliste-beitreten) unten.
:::

## Schritt 2: Familienmitglieder hinzufügen

1. Wenn Sie zusätzliche Familienmitglieder anmelden möchten, geben Sie für jede Person **Vorname** und **Nachname** ein.
2. Wenn die Veranstaltung Teilnehmertypen hat, wählen Sie für jede hinzugefügte Person einen Typ.
3. Klicken Sie auf **Mitglied hinzufügen**, um weitere Personen hinzuzufügen (bis zu 10 insgesamt).
4. Klicken Sie auf die Schaltfläche zum Entfernen neben einem Mitglied, um es von der Anmeldung zu entfernen.
5. Wenn Sie sich nur selbst anmelden, können Sie diesen Schritt überspringen.

## Schritt 3: Wählen Sie Optionen & Auswahlmöglichkeiten (falls angeboten)

Einige Veranstaltungen bieten optionale bezahlte Extras an — T-Shirts, Mahlzeiten, Aktivitätsupgrades.

1. Wenn die Veranstaltung Auswahlmöglichkeiten hat, wird ein Schritt **Optionen & Auswahlmöglichkeiten** angezeigt.
2. Verwenden Sie die Menge-Stepper, um auszuwählen, wie viele von jedem Sie möchten. Kostenlose Optionen werden als **Kostenlos** angezeigt, und begrenzte Optionen zeigen, wie viele noch verfügbar sind.
3. Klicken Sie auf **Weiter**.

## Schritt 4: Beantworten Sie Fragen (falls gefragt)

Einige Veranstaltungen bitten Teilnehmer, ein paar zusätzliche Fragen zu beantworten — Diätwünsche, T-Shirt-Größe, Notfallkontakt und ähnliche Details.

1. Wenn die Veranstaltung Fragen enthält, wird ein Schritt **Fragen** angezeigt.
2. Füllen Sie Ihre Antworten aus und fahren Sie fort.

Wenn die Veranstaltung keine Fragen enthält, wird dieser Schritt automatisch übersprungen.

## Schritt 5: Zahlen (bezahlte Veranstaltungen)

Wenn Ihre Anmeldung eine Gebühr hat, zeigt der Schritt **Zahlung** eine **Bestellübersicht**, die jeden Teilnehmer, jede Auswahlmöglichkeit, einen Rabatt und die **Summe** auflistet.

- **Rabattcode** — Wenn Ihre Kirche Ihnen einen Code gegeben hat, geben Sie ihn ein und klicken Sie auf **Anwenden**. Der Rabatt wird sofort widergespiegelt, und Sie sehen eine klare Meldung, wenn der Code abgelaufen ist, noch nicht gestartet wurde oder mehr Teilnehmer erfordert.
- **Angemeldete Mitglieder** können mit einer gespeicherten Karte zahlen oder **Neue Karte verwenden** wählen.
- **Gäste** geben ihre Kartendaten direkt ein.

Klicken Sie auf **Zahlen & Anmelden**, um Zahlung und Anmeldung in einem Schritt abzuschließen. Ihre Kartendaten gehen direkt an den Zahlungsanbieter der Kirche und werden nie von B1 gespeichert.

Kostenlose Anmeldungen überspringen diesen Schritt und gehen direkt zur Bestätigung.

## Schritt 6: Bestätigen

Eine Bestätigungsmeldung wird mit den Veranstaltungsdetails und einer Liste aller Personen angezeigt, die Sie angemeldet haben. Sie erhalten auch eine **Bestätigungsemail** unter der E-Mail-Adresse, die Ihrem Konto zugeordnet ist (oder der E-Mail, die Sie als Gast angegeben haben).

## Warteliste beitreten

Wenn die Veranstaltung voll ist und Ihre Kirche die Warteliste aktiviert hat, zeigt der erste Schritt: *„Diese Veranstaltung ist voll. Sie können der Warteliste beitreten und wir benachrichtigen Sie, wenn ein Platz frei wird."* Schließen Sie die Anmeldung wie gewohnt ab — die Zahlung wird übersprungen, während Sie auf der Warteliste sind. Wenn ein Platz frei wird, erhalten Sie eine E-Mail; wenn eine Zahlung erforderlich ist, enthält sie einen Link zum Abschließen. Sie können Ihren Status auch überprüfen und bezahlen von [Meine Anmeldungen](./my-registrations).

## Anmeldung ändern oder stornieren

Wenn Sie angemeldet sind, können Sie Ihre Anmeldung selbst von der Seite **Anmeldungen** im Mitglieder-Portal bearbeiten oder stornieren — siehe [Meine Anmeldungen](./my-registrations). Gäste sollten das Kirchenbüro kontaktieren, und der Mitarbeiter kann die Änderung von der B1 Admin-Seite mit Anmeldungen vornehmen.
