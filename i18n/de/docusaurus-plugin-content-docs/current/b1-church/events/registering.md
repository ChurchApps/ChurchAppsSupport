# Registrierung für Veranstaltungen

<div class="article-intro">

Wenn Ihre Kirche die Registrierung für eine Veranstaltung aktiviert, können Sie sich über einen einfachen dreistufigen Prozess anmelden. Sie können sich selbst anmelden und Familienmitglieder hinzufügen, egal ob Sie ein Konto haben oder als Gast besuchen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Ihre Kirche muss die Veranstaltungsregistrierung für die Veranstaltung aktiviert haben
- Das Registrierungsfenster muss offen sein (zwischen den Öffnungs- und Schließungsdaten, die vom Administrator festgelegt wurden)

</div>

## Schritt 1: Geben Sie Ihre Informationen ein

1. Öffnen Sie im Kirchenkalender die Veranstaltung, für die Sie sich registrieren möchten, und klicken Sie auf **Register for this Event**.
2. Die Registrierungsseite wird geladen und zeigt den Ereignistitel, die Beschreibung und das Datum an.
3. Wenn Sie **angemeldet** sind, zeigt das Formular Ihren Namen an und bestätigt, dass Sie sich unter Ihrem Konto registrieren.
4. Wenn Sie ein **Gast** sind (nicht angemeldet), füllen Sie Folgendes aus:
   - **Vorname** (erforderlich)
   - **Nachname** (erforderlich)
   - **E-Mail** (erforderlich)
   - **Telefon** (optional)

:::info
Wenn das Ereignis ein Kapazitätslimit hat, zeigt eine Fortschrittsleiste, wie viele Plätze verbleibend sind. Wenn die Veranstaltung voll ist, können Sie sich nicht registrieren.
:::

## Schritt 2: Fügen Sie Familienmitglieder hinzu

1. Wenn Sie zusätzliche Familienmitglieder registrieren möchten, geben Sie ihren **Vornamen** und **Nachnamen** für jede Person ein.
2. Klicken Sie auf **Add Member**, um weitere Personen hinzuzufügen (bis zu 10 insgesamt).
3. Klicken Sie auf die Schaltfläche „Entfernen" neben einem Mitglied, um ihn aus der Registrierung zu entfernen.
4. Wenn Sie sich nur selbst registrieren, können Sie diesen Schritt überspringen.

## Schritt 3: Bestätigen

1. Überprüfen Sie die Veranstaltungsdetails und Ihre Registrierungsinformationen.
2. Bestätigen Sie Ihre Registrierung.
3. Eine Bestätigungsmitteilung wird mit den Veranstaltungsdetails und einer Liste aller Personen angezeigt, die Sie registriert haben.

Sie erhalten auch eine **Bestätigungs-E-Mail** unter der E-Mail-Adresse, die mit Ihrem Konto verknüpft ist (oder die E-Mail-Adresse, die Sie als Gast angegeben haben).

## Stornieren einer Registrierung

Wenn Sie Ihre Registrierung stornieren müssen, wenden Sie sich an Ihren Kirchenadministrator. Sie können Ihre Registrierung auf der Registrierungsseite von B1 Admin stornieren.
