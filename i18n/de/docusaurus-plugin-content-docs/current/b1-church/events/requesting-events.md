# Ereignisse & Räume anfordern

<div class="article-intro">

Wenn Ihre Gemeinde buchbare Räume und Ressourcen verfolgt, können Sie ein Ereignis — mit den Räumen und Ausrüstung, die es benötigt — direkt aus dem Mitglieder-Portal anfordern. Konflikte mit bestehenden Buchungen werden flaggiert, während Sie auswählen, Ihre Anfrage geht an die Gemeinde zur Genehmigung, und Sie können sie von Meine Anfragen verfolgen oder stornieren.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie müssen [eingeloggt](../getting-started/logging-in.md) sein zum Mitglieder-Portal
- Ihre Gemeinde muss Räume oder Ressourcen in B1 Admin eingerichtet haben — die Anfrage-Verknüpfungen erscheinen nur dann

</div>

## Anfrage einreichen

1. Öffnen Sie die **Me**-Seite im Mitglieder-Portal und tippen Sie auf **Ereignis anfordern** unter Verknüpfungen.
2. Füllen Sie die Ereignis-Grundlagen aus: **Ereignis-Titel**, optionale **Beschreibung**, die **Start**- und **End**-Zeiten (oder **Ganztägig**) und ob es **Wiederholt**.
3. Wählen Sie die **Räume** und **Ressourcen**, die Sie benötigen. Für Ressourcen können Sie eine **Menge** setzen.
4. Achten Sie auf Konflikt-Warnungen — wenn Ihr Zeitslot eine bestehende Buchung für einen Raum oder eine Ressource überlappt, sehen Sie *„Dieser Zeitpunkt konfligiert mit einer bestehenden Buchung."* Sie können Ihren Zeitslot anpassen oder etwas anderes auswählen.
5. Tippen Sie auf **Anfrage senden**.

Nach dem Senden zeigt eine **Anfrage gesendet**-Zusammenfassung den Status jedes Raums und jeder Ressource, die Sie angefordert haben:

- **Genehmigt** -- Sofort gebucht (der Raum oder die Ressource hat keine Genehmigungsgruppe).
- **Genehmigung ausstehend** -- Jemand aus der Gemeinde muss zuerst zustimmen.

## Anfragen verfolgen

Öffnen Sie **Meine Anfragen** von der Me-Seite. Jede Anfrage zeigt das Ereignis, wann Sie es angefordert haben, und eine Status-Pille:

- **Ausstehend** -- Wartung auf eine Entscheidung
- **Genehmigt** -- Ihre Buchung ist bestätigt
- **Abgelehnt** -- Die Anfrage wurde abgelehnt

Während eine Anfrage noch ausstehend ist, können Sie sie mit **Anfrage stornieren** zurückziehen (Sie werden aufgefordert zu bestätigen — die Stornierung kann nicht rückgängig gemacht werden).

:::info
Anfragen werden von Kirchenmitarbeitern auf der B1 Admin [Genehmigungsseite](../../b1-admin/calendars/approvals) überprüft. Welche Räume und Ressourcen Genehmigung benötigen, ist bis zu Ihrer Gemeinde — siehe [Räume & Ressourcen](../../b1-admin/calendars/rooms-resources).
:::
