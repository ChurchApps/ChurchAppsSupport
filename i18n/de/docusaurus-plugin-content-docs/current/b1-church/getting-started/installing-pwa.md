# Als App Installieren (PWA)

<div class="article-intro">

B1.church unterstützt die Installation als Progressive Web App (PWA), mit der Sie das Gemeinde-Mitglieder-Portal Ihrer Kirche wie eine native App auf dem Home-Bildschirm Ihres Telefons hinzufügen können. Dies bietet Ihnen eine Vollbild-Erfahrung wie eine App, ohne den App Store oder Google Play durchgehen zu müssen.

</div>

<div class="prereqs">
<h4>Bevor Sie Beginnen</h4>

- Sie benötigen Ihre B1.church-URL der Kirche: `https://yourchurchname.b1.church`
- Verwenden Sie einen unterstützten Browser: **Safari** unter iOS oder **Chrome** unter Android

</div>

## Was ist eine PWA?

Eine Progressive Web App (PWA) ist eine Website, die auf Ihrem Gerät installiert werden kann und sich wie eine native App verhält. Wenn Sie B1.church als PWA installieren:

- Wird es im Vollbildmodus ohne Browser-Navigationsleisten geöffnet
- Wird auf Ihrem Home-Bildschirm mit dem Symbol und Namen Ihrer Kirche angezeigt
- Funktioniert reibungslos mit dem Branding und den Designfarben Ihrer Kirche

:::tip
Die PWA ist eine großartige Option, wenn Sie es bevorzugen, keine separate App aus dem App Store oder Google Play zu installieren. Sie bietet die gleiche Mitglieder-Portal-Erfahrung, die Sie im Browser erhalten, in einem bequemeren Paket.
:::

## Wo Sie Installieren Können

Sie können die PWA von **jeder Seite** auf der B1.church-Website Ihrer Kirche installieren – auf der öffentlichen Startseite, einer Predigtseite, dem Anmelde-Bildschirm oder überall im Mitglieder-Portal. Es gibt keine separate „Installations-URL" zu merken; wenn Sie Ihre Kirche im Web besuchen können, können Sie die App von dort aus installieren.

Wenn Sie die installierte App starten, öffnet sie sich direkt auf Ihrem Mitglieder-Dashboard und Sie sind bereit, sich anzumelden.

## Installieren auf dem iPhone (iOS – Safari)

1. Öffnen Sie **Safari** auf Ihrem iPhone und gehen Sie zu `https://yourchurchname.b1.church` (ersetzen Sie `yourchurchname` durch den URL-Slug Ihrer Kirche).
2. Tippen Sie auf die Schaltfläche **Share** (das Feld mit einem Pfeil nach oben) am unteren Rand des Bildschirms.
3. Scrollen Sie in der Freigabe-Tabelle nach unten und tippen Sie auf **Add to Home Screen**.
4. Der Name wird standardmäßig auf den Namen Ihrer Kirche gesetzt. Tippen Sie auf **Add** in der oberen rechten Ecke.

Das App-Symbol wird auf Ihrem Home-Bildschirm angezeigt. Tippen Sie darauf, um das Mitglieder-Portal im Vollbildmodus zu öffnen.

:::info
Die PWA-Installation unter iOS erfordert Safari. Wenn Sie Chrome oder einen anderen Browser auf dem iPhone verwenden, wechseln Sie zuerst zu Safari.
:::

## Installieren auf Android (Chrome)

1. Öffnen Sie **Chrome** auf Ihrem Android-Gerät und gehen Sie zu `https://yourchurchname.b1.church`.
2. Chrome zeigt automatisch ein Banner **„Add to Home screen"** am unteren Rand des Bildschirms an, oder tippen Sie auf das Menü mit drei Punkten (⋮) in der oberen rechten Ecke und wählen Sie **Add to Home screen**.
3. Tippen Sie auf **Add**, wenn Sie dazu aufgefordert werden.

Das App-Symbol wird auf Ihrem Home-Bildschirm angezeigt. Tippen Sie darauf, um das Mitglieder-Portal zu öffnen.

## Nach Der Installation

Nach der Installation öffnet sich die PWA direkt im Mitglieder-Portal in einem dedizierten Vollbild-Fenster. Sie können:

- Sich mit Ihrem vorhandenen ChurchApps-Konto anmelden
- Auf alle Mitglieder-Funktionen zugreifen, die Ihre Kirche aktiviert hat – Gruppen, Spenden, Check-in, Dienst-Pläne und mehr
- Sie jederzeit wiederfinden, indem Sie auf das Symbol auf Ihrem Home-Bildschirm tippen

:::info
Die PWA erfordert eine Internetverbindung. Im Gegensatz zu einigen nativen Apps werden nicht alle Inhalte für die Offline-Nutzung zwischengespeichert.
:::

## Was ist mit der alten B1 Mobile App?

Die veraltete native **B1 Mobile**-App für iOS und Android wurde als veraltet markiert und wird nicht mehr aktualisiert. Die PWA ist die unterstützte Möglichkeit, um auf das B1.church-Mitglieder-Portal auf einem Telefon zuzugreifen, und sie deckt jedes Mitglieder-Feature ab, das die native App angeboten hat. Wenn Sie zuvor die native App installiert haben, wechseln Sie nach Belieben zur PWA.

Siehe die Seite [B1 Mobile-Referenz](/docs/b1-mobile/) für historische Dokumentation zur veralteten App.
