# Als App installieren (PWA)

<div class="article-intro">

B1.church unterstützt die Installation als Progressive Web App (PWA), mit der Sie das Mitgliederportal Ihrer Gemeinde genauso wie eine native App auf dem Startbildschirm Ihres Telefons hinzufügen können. Dies bietet Ihnen ein Full-Screen-, app-ähnliches Erlebnis, ohne dass Sie durch den App Store oder Google Play gehen müssen.

</div>

<div class="prereqs">
<h4>Vorbereitung</h4>

- Sie benötigen die Mobile-URL des B1.church Ihrer Gemeinde: `https://yourchurchname.b1.church/mobile`
- Verwenden Sie einen unterstützten Browser: **Safari** auf iOS oder **Chrome** auf Android

</div>

## Was ist eine PWA?

Eine Progressive Web App (PWA) ist eine Website, die auf Ihrem Gerät installiert werden kann und sich wie eine native App verhält. Wenn Sie B1.church als PWA installieren, können Sie:

- Vollbildmodus ohne Browser-Navigationaleisten öffnen
- Auf Ihrem Startbildschirm mit dem Symbol und Namen Ihrer Gemeinde erscheinen
- Reibungslos mit dem Branding und den Designfarben Ihrer Gemeinde funktionieren

:::tip
Die PWA ist eine großartige Option, wenn Sie lieber keine separate App aus dem App Store oder Google Play installieren möchten. Sie bietet das gleiche Mitgliederportal-Erlebnis, das Sie im Browser erhalten, in einem bequemeren Paket.
:::

## Installation auf iPhone (iOS – Safari)

1. Öffnen Sie **Safari** auf Ihrem iPhone und gehen Sie zu `https://yourchurchname.b1.church/mobile` (ersetzen Sie `yourchurchname` durch Ihren Gemeinde-URL-Slug).
2. Tippen Sie auf die Schaltfläche **Freigeben** (das Feld mit einem nach oben zeigenden Pfeil) am unteren Bildschirmrand.
3. Scrollen Sie im Freigabeblatt nach unten und tippen Sie auf **Zum Home-Bildschirm hinzufügen**.
4. Der Name wird standardmäßig auf den Namen Ihrer Gemeinde eingestellt. Tippen Sie auf **Hinzufügen** in der oberen rechten Ecke.

Das App-Symbol wird auf Ihrem Startbildschirm angezeigt. Tippen Sie darauf, um das Mitgliederportal im Vollbildmodus zu öffnen.

:::info
Die PWA-Installation auf iOS erfordert Safari. Wenn Sie Chrome oder einen anderen Browser auf dem iPhone verwenden, wechseln Sie zunächst zu Safari.
:::

## Installation auf Android (Chrome)

1. Öffnen Sie **Chrome** auf Ihrem Android-Gerät und gehen Sie zu `https://yourchurchname.b1.church/mobile`.
2. Chrome zeigt automatisch ein Banner **"Zum Startbildschirm hinzufügen"** am unteren Bildschirmrand an, oder tippen Sie auf das Drei-Punkte-Menü (⋮) in der oberen rechten Ecke und wählen Sie **Zum Startbildschirm hinzufügen**.
3. Tippen Sie auf **Hinzufügen**, wenn Sie dazu aufgefordert werden.

Das App-Symbol wird auf Ihrem Startbildschirm angezeigt. Tippen Sie darauf, um das Mitgliederportal zu öffnen.

## Nach der Installation

Nach der Installation funktioniert die PWA genauso wie der Besuch der Mobile-Website in Ihrem Browser, aber in einem dedizierten Vollbildfenster. Sie können:

- Sich mit Ihrem vorhandenen ChurchApps-Konto anmelden
- Auf alle Mitglieder-Features zugreifen, die Ihre Gemeinde aktiviert hat – Gruppen, Spenden, Check-In, Servicepläne und mehr
- Jederzeit durch Tippen auf das Symbol auf Ihrem Startbildschirm darauf zurückgreifen

:::info
Die PWA benötigt eine Internetverbindung. Im Gegensatz zu einigen nativen Apps speichert sie nicht alle Inhalte für die Offline-Nutzung.
:::

## PWA vs. Native App

Beide Optionen geben Ihnen Zugriff auf das B1.church-Mitgliederportal. Die richtige Wahl hängt von Ihren Vorlieben ab:

| | **PWA** | **B1 Mobile App** |
|---|---|---|
| Installation | Zum Startbildschirm von Browser hinzufügen | Aus App Store / Google Play herunterladen |
| Updates | Automatisch (kein App Store Update erforderlich) | Über App Store / Google Play |
| Push-Benachrichtigungen | Nicht unterstützt | Unterstützt |
| Plattform | Alle Geräte mit Browser | iOS und Android |

:::tip
Wenn Ihre Gemeinde Gruppen-Gespräche nutzt und Sie Push-Benachrichtigungen für neue Nachrichten und Aktivitäten möchten, verwenden Sie stattdessen die [B1 Mobile App](/docs/b1-mobile/). Push-Benachrichtigungen werden derzeit nicht in der PWA unterstützt.
:::
