# Erste Schritte mit B1 Church Checkin

<div class="article-intro">

Die Einrichtung von B1 Church Checkin dauert nur wenige Minuten. Dieser Abschnitt fuehrt Sie durch die Installation der App auf einem Android-Tablet, die Anmeldung mit Ihrem ChurchApps-Konto und die optionale Konfiguration eines Netzwerk-Etikettendruckers.

</div>

## Was Sie benoetigen

- **Ein Android-Tablet** -- Ein 10-Zoll-Tablet eignet sich am besten fuer das Kiosk-Erlebnis.
- **Ein ChurchApps-Konto** -- Die E-Mail-Adresse und das Passwort, die Sie fuer B1 Admin oder andere ChurchApps-Produkte verwenden.
- **Ein Netzwerk-Etikettendrucker** (optional) -- Zum Drucken von Namensetiketten und Abholscheinen fuer Eltern. Die App kann Ihr lokales Netzwerk nach kompatiblen Druckern durchsuchen.

## Installation

1. Oeffnen Sie den **Google Play Store** auf Ihrem Android-Tablet.
2. Suchen Sie nach **B1 Church Checkin**.
3. Tippen Sie auf **Installieren** und warten Sie, bis der Download abgeschlossen ist.
4. Oeffnen Sie die App. Sie sehen einen Startbildschirm waehrend der Initialisierung.

## Ersteinrichtung

Wenn Sie die App zum ersten Mal starten, durchlaufen Sie drei Schritte:

1. **[Anmelden](./logging-in)** -- Geben Sie Ihre E-Mail-Adresse und Ihr Passwort zur Authentifizierung ein.
2. **[Gemeinde auswaehlen](./logging-in)** -- Wenn Ihr Konto mit mehreren Gemeinden verknuepft ist, waehlen Sie aus, welche dieser Kiosk bedienen soll.
3. **[Drucker einrichten](./printer-setup)** -- Konfigurieren Sie einen Netzwerkdrucker fuer den Etikettendruck, oder ueberspringen Sie diesen Schritt, wenn Sie keine Etiketten benoetigen.

Nach der ersten Anmeldung speichert die App Ihre Zugangsdaten und Gemeindeauswahl. Bei zukuenftigen Starts wechselt sie automatisch direkt zum Gottesdienst-Bildschirm.

:::info
Die App prueft bei jedem Start auf Over-the-Air-Updates. Wenn ein Update verfuegbar ist, wird es automatisch heruntergeladen und angewendet, bevor Sie den Anmeldebildschirm erreichen.
:::

:::tip
Stellen Sie sicher, dass Ihr Gemeinde-Administrator Gottesdienste, Gottesdienstzeiten und Gruppen in B1 Admin konfiguriert hat, bevor Sie den Kiosk einrichten. Weitere Informationen finden Sie in der [Anleitung zur Anwesenheitseinrichtung](../../b1-admin/attendance/setup.md).
:::
