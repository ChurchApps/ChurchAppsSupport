# Gruppen zuweisen

<div class="article-intro">

Wenn Sie auf dem Haushaltsuebersichts-Bildschirm auf eine Gottesdienstzeit-Schaltflaeche tippen, oeffnet die App den Gruppenauswahl-Bildschirm. Hier waehlen Sie aus, welche Gruppe oder Klasse ein Familienmitglied fuer diese bestimmte Gottesdienstzeit besuchen soll.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Ueberpruefen Sie Ihren Haushalt](./household-review) auf dem Haushaltsuebersichts-Bildschirm
- Klappen Sie das Mitglied auf, das Sie zuweisen moechten, und tippen Sie auf eine Gottesdienstzeit-Schaltflaeche
- Stellen Sie sicher, dass Ihr Gemeinde-Administrator [Gruppen erstellt](../../b1-admin/groups/creating-groups.md) und diese mit Gottesdienstzeiten in B1 Admin verknuepft hat

</div>

## Wie Gruppen organisiert sind

Gruppen werden in **Kategorien** angezeigt. Jede Kategorie erscheint als Karte mit:

- Dem **Kategorienamen** (zum Beispiel "Kinder", "Jugend", "Erwachsene")
- Der **Anzahl der Gruppen** in dieser Kategorie

Tippen Sie auf eine Kategoriekarte, um sie aufzuklappen und die einzelnen Gruppen darin zu sehen. Tippen Sie erneut, um sie zuzuklappen.

## Eine Gruppe auswaehlen

1. Tippen Sie auf eine **Kategorie**, um sie aufzuklappen.
2. Sehen Sie sich die Liste der Gruppen innerhalb dieser Kategorie an. Jede Gruppe zeigt ihren Namen mit einem Auswahlsymbol auf der rechten Seite.
3. Tippen Sie auf den **Gruppennamen**, den Sie zuweisen moechten.

Die App speichert Ihre Auswahl und kehrt zum [Haushaltsuebersichts-Bildschirm](./household-review) zurueck. Die Gottesdienstzeit-Schaltflaeche fuer dieses Mitglied zeigt nun den Namen der ausgewaehlten Gruppe an.

## "Keine" auswaehlen

Wenn Sie entscheiden, dass ein Familienmitglied fuer diese Gottesdienstzeit keiner Gruppe zugewiesen werden soll, tippen Sie auf die Schaltflaeche **None** am unteren Bildschirmrand. Dies loescht jede bestehende Gruppenzuweisung fuer diese Gottesdienstzeit und kehrt zum Haushaltsuebersichts-Bildschirm zurueck.

:::tip
Verwenden Sie die Option **None** fuer Erwachsene, die am regulaeren Gottesdienst teilnehmen und nicht in eine bestimmte Gruppe oder Klasse eingecheckt werden muessen.
:::

## Eine Gruppenzuweisung aendern

Um eine bereits vorgenommene Gruppenzuweisung zu aendern:

1. Kehren Sie zum [Haushaltsuebersichts-Bildschirm](./household-review) zurueck.
2. Klappen Sie das Mitglied auf, dessen Gruppe Sie aendern moechten.
3. Tippen Sie auf die Gottesdienstzeit-Schaltflaeche (die nun den aktuell zugewiesenen Gruppennamen anzeigt).
4. Waehlen Sie eine andere Gruppe oder tippen Sie auf **None**.

:::info
Gruppen und Kategorien werden von Ihrem Gemeinde-Administrator in B1 Admin konfiguriert. Wenn eine Gruppe fehlt oder in der falschen Kategorie ist, bitten Sie Ihren Administrator, die [Gruppeneinstellungen](../../b1-admin/groups/creating-groups.md) zu aktualisieren.
:::

## Naechste Schritte

Nachdem Sie Gruppen zugewiesen haben, kehren Sie zum [Haushaltsuebersichts-Bildschirm](./household-review) zurueck, um:

- Weiteren Familienmitgliedern Gruppen zuzuweisen
- [Einen Gast hinzuzufuegen](./adding-guests), wenn jemand mit Ihrer Familie zu Besuch ist
- [Den Check-in abzuschliessen](./completing-checkin), wenn alle Zuweisungen erledigt sind
