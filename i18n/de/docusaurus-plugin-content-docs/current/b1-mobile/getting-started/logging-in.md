# Anmeldung

<div class="article-intro">

B1 Mobile verwendet das gleiche ChurchApps-Konto, das Sie auf B1.church und anderen ChurchApps-Produkten verwenden. Wenn Sie bereits ein Konto haben, können Sie sich mit Ihrer bestehenden E-Mail und Ihrem Passwort anmelden.

</div>

<div class="prereqs">
<h4>Bevor Sie Beginnen</h4>

- [Installieren Sie B1 Mobile](./installing) auf Ihrem iOS- oder Android-Gerät

</div>

## Wie Sie Sich Anmelden

1. Öffnen Sie die B1 Mobile App.
2. Tippen Sie auf die Option **Login** aus dem Drawer-Menü oder der Anmelde-Eingabeaufforderung.
3. Geben Sie Ihre **E-Mail-Adresse** und Ihr **Passwort** ein.
4. Tippen Sie auf **Sign In**.

Nach der Anmeldung werden Sie zum [Dashboard](../dashboard/) gebracht. Wenn Sie noch keine Kirche ausgewählt haben, werden Sie aufgefordert, eine zu suchen und auszuwählen.

## Erstellen eines Neuen Kontos

Wenn Sie noch kein ChurchApps-Konto haben, können Sie eines direkt in der App erstellen:

1. Tippen Sie auf dem Anmelde-Bildschirm auf den Link **Register** am unteren Rand.
2. Geben Sie Ihren **Vornamen**, **Nachnamen** und Ihre **E-Mail-Adresse** ein.
3. Das Formular wird Ihre Kirche automatisch ausfüllen, falls eine Übereinstimmung basierend auf Ihren Informationen gefunden wird.
4. Tippen Sie auf **Register**.
5. Überprüfen Sie Ihre E-Mail auf eine Nachricht mit Ihrem vorübergehenden Passwort.
6. Kehren Sie zur App zurück, tippen Sie auf **Login** und melden Sie sich mit Ihrer E-Mail und dem vorübergehenden Passwort aus der E-Mail an.

Nach Ihrer ersten Anmeldung können Sie Ihr Passwort aus Ihren [Profileinstellungen](../profile/editing-profile.md) ändern.

:::info
Wenn Sie versuchen, sich mit einer E-Mail anzumelden, die nicht registriert ist, werden Sie automatisch zum Registrierungs-Bildschirm weitergeleitet, damit Sie ein Konto erstellen können.
:::

:::warning
Überprüfen Sie Ihren Spam- oder Junk-Ordner, falls Sie die Registrierungs-E-Mail nicht innerhalb weniger Minuten erhalten.
:::

## Vergessenes Passwort

Wenn Sie Ihr Passwort vergessen haben, tippen Sie auf dem Anmelde-Bildschirm auf **Forgot Password?** und geben Sie Ihre E-Mail-Adresse ein. Sie erhalten eine E-Mail mit einem **6-stelligen Verifikationscode**, der 15 Minuten lang gültig ist. Geben Sie den Code auf dem Verifikations-Bildschirm ein, um sich anzumelden, und aktualisieren Sie dann Ihr Passwort aus Ihren [Profileinstellungen](../profile/editing-profile.md).

:::warning
Nach fünf fehlgeschlagenen Code-Versuchen wird der Code gesperrt. Wenn dies geschieht, kehren Sie zum Bildschirm „Vergessenes Passwort" zurück und fordern Sie einen neuen Code an.
:::

## Angemeldet Bleiben

B1 Mobile hält Sie zwischen den Sitzungen angemeldet. Sie müssen sich nicht jedes Mal anmelden, wenn Sie die App öffnen. Wenn Sie aus irgendeinem Grund abgemeldet werden, melden Sie sich einfach wieder mit Ihrer E-Mail und Ihrem Passwort an.

## Abmeldung

Um sich aus der App abzumelden:

1. Öffnen Sie das Drawer-Menü, indem Sie auf das **Hamburger-Symbol** (drei horizontale Linien) in der oberen linken Ecke tippen.
2. Scrollen Sie zum unteren Ende des Drawer-Menüs.
3. Tippen Sie auf **Sign Out**.

Sie werden abgemeldet und können sich jederzeit wieder mit Ihren Anmeldedaten anmelden.
