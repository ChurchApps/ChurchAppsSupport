# Anmeldung

<div class="article-intro">

B1 Mobile verwendet das gleiche ChurchApps-Konto, das Sie auf B1.church und anderen ChurchApps-Produkten verwenden. Wenn Sie bereits ein Konto haben, können Sie sich mit Ihrer bestehenden E-Mail und Ihrem Passwort anmelden.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Installieren Sie B1 Mobile](./installing) auf Ihrem iOS- oder Android-Gerät

</div>

## Wie man sich anmeldet

1. Öffnen Sie die B1 Mobile-App.
2. Tippen Sie auf die Option **Anmeldung** aus dem Drawer-Menü oder dem Anmeldeeingabe-Prompt.
3. Geben Sie Ihre **E-Mail-Adresse** und **Passwort** ein.
4. Tippen Sie auf **Anmelden**.

Nach der Anmeldung werden Sie zum [Dashboard](../dashboard/) geleitet. Wenn Sie noch keine Kirche ausgewählt haben, werden Sie aufgefordert, eine zu suchen und auszuwählen.

## Ein neues Konto erstellen

Wenn Sie noch kein ChurchApps-Konto haben, können Sie eines direkt in der App erstellen:

1. Tippen Sie auf dem Anmeldebildschirm auf den Link **Registrierung** am unteren Ende.
2. Geben Sie Ihren **Vorname**, **Nachname** und **E-Mail-Adresse** ein.
3. Das Formular füllt automatisch Ihre Kirche auf, wenn basierend auf Ihren Informationen eine Übereinstimmung gefunden wird.
4. Tippen Sie auf **Registrierung**.
5. Überprüfen Sie Ihre E-Mail auf eine Nachricht mit Ihrem temporären Passwort.
6. Kehren Sie zur App zurück, tippen Sie auf **Anmeldung** und melden Sie sich mit Ihrer E-Mail und dem temporären Passwort aus der E-Mail an.

Nach Ihrer ersten Anmeldung können Sie Ihr Passwort über Ihre [Profileinstellungen](../profile/editing-profile.md) ändern.

:::info
Wenn Sie sich mit einer E-Mail anmelden, die noch nicht registriert ist, werden Sie automatisch zum Registrierungsbildschirm geleitet, damit Sie ein Konto erstellen können.
:::

:::warning
Überprüfen Sie unbedingt Ihren Spam- oder Junk-Ordner, wenn Sie die Registrierungs-E-Mail nicht innerhalb weniger Minuten erhalten.
:::

## Passwort vergessen

Wenn Sie Ihr Passwort vergessen haben, tippen Sie auf dem Anmeldebildschirm auf **Passwort vergessen?**. Dies öffnet eine Seite in Ihrem Browser, auf der Sie Ihre E-Mail-Adresse eingeben und einen Passwort-Zurücksetzen-Link anfordern können.

## Angemeldet bleiben

B1 Mobile hält Sie zwischen den Sitzungen angemeldet. Sie müssen sich nicht jedes Mal anmelden, wenn Sie die App öffnen. Wenn Sie aus irgendeinem Grund abgemeldet werden, melden Sie sich einfach erneut mit Ihrer E-Mail und Ihrem Passwort an.

## Abmelden

So melden Sie sich von der App ab:

1. Öffnen Sie das Drawer-Menü, indem Sie auf das **Hamburger-Symbol** (drei horizontale Linien) in der oberen linken Ecke tippen.
2. Scrollen Sie zum unteren Ende des Drawer-Menüs.
3. Tippen Sie auf **Abmelden**.

Sie werden abgemeldet und können sich jederzeit erneut mit Ihren Anmeldedaten anmelden.
