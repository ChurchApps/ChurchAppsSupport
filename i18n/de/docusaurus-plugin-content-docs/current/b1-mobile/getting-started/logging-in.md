# Anmelden

<div class="article-intro">

B1 Mobile verwendet dasselbe ChurchApps-Konto, das Sie auf B1.church und anderen ChurchApps-Produkten nutzen. Wenn Sie bereits ein Konto haben, koennen Sie sich mit Ihrer bestehenden E-Mail-Adresse und Ihrem Passwort anmelden.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Installieren Sie B1 Mobile](./installing) auf Ihrem iOS- oder Android-Geraet

</div>

## So melden Sie sich an

1. Oeffnen Sie die B1 Mobile App.
2. Tippen Sie auf die Option **Anmelden** im Seitenmenue oder in der Anmeldeaufforderung.
3. Geben Sie Ihre **E-Mail-Adresse** und Ihr **Passwort** ein.
4. Tippen Sie auf **Anmelden**.

Nach der Anmeldung werden Sie zum [Dashboard](../dashboard/) weitergeleitet. Wenn Sie noch keine Gemeinde ausgewaehlt haben, werden Sie aufgefordert, eine zu suchen und auszuwaehlen.

## Ein neues Konto erstellen

Wenn Sie noch kein ChurchApps-Konto haben, koennen Sie eines direkt in der App erstellen:

1. Tippen Sie auf dem Anmeldebildschirm auf den Link **Registrieren** am unteren Rand.
2. Geben Sie Ihren **Vornamen**, **Nachnamen** und Ihre **E-Mail-Adresse** ein.
3. Tippen Sie auf **Registrieren**.
4. Pruefen Sie Ihre E-Mails auf eine Nachricht mit Ihrem temporaeren Passwort.
5. Kehren Sie zur App zurueck, tippen Sie auf **Anmelden** und melden Sie sich mit Ihrer E-Mail-Adresse und dem temporaeren Passwort aus der E-Mail an.

Nach Ihrer ersten Anmeldung koennen Sie Ihr Passwort in Ihren [Profileinstellungen](../profile/editing-profile.md) aendern.

:::warning
Pruefen Sie unbedingt Ihren Spam- oder Junk-Ordner, falls Sie die Registrierungs-E-Mail nicht innerhalb weniger Minuten erhalten.
:::

## Passwort vergessen

Wenn Sie Ihr Passwort vergessen haben, tippen Sie auf dem Anmeldebildschirm auf **Passwort vergessen?**. Daraufhin oeffnet sich eine Seite in Ihrem Browser, auf der Sie Ihre E-Mail-Adresse eingeben und einen Link zum Zuruecksetzen des Passworts anfordern koennen.

## Angemeldet bleiben

B1 Mobile haelt Sie zwischen den Sitzungen angemeldet. Sie muessen sich nicht jedes Mal anmelden, wenn Sie die App oeffnen. Sollten Sie aus irgendeinem Grund abgemeldet werden, melden Sie sich einfach erneut mit Ihrer E-Mail-Adresse und Ihrem Passwort an.

## Abmelden

Um sich aus der App abzumelden:

1. Oeffnen Sie das Seitenmenue, indem Sie auf das **Hamburger-Symbol** (drei horizontale Linien) in der oberen linken Ecke tippen.
2. Scrollen Sie zum Ende des Seitenmenues.
3. Tippen Sie auf **Abmelden**.

Sie werden abgemeldet und koennen sich jederzeit mit Ihren Zugangsdaten wieder anmelden.
