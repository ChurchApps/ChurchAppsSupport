# Erste Schritte

:::danger B1 Mobile ist veraltet
Die native B1 Mobile App wurde durch die [B1.church Progressive Web App (PWA)](../../b1-church/getting-started/installing-pwa.md) ersetzt. Neue Benutzer sollten die PWA von jeder Seite auf `https://yourchurchname.b1.church` installieren, anstatt B1 Mobile aus dem App Store oder Google Play Store zu laden.
:::

<div class="article-intro">

Die ersten Schritte mit B1 Mobile dauern nur wenige Minuten. Laden Sie die App herunter, melden Sie sich mit Ihrem ChurchApps-Konto an, wählen Sie Ihre Kirche aus, und Sie können loslegen.

</div>

## Schnellstart

1. **Laden Sie die App** aus dem [App Store](https://apps.apple.com) (iOS) oder [Google Play Store](https://play.google.com) (Android) herunter. Suchen Sie nach „B1.church".
2. **Melden Sie sich an** mit Ihrem bestehenden ChurchApps-Konto – die gleiche E-Mail und das gleiche Passwort, die Sie auf B1.church verwenden. Wenn Sie noch kein Konto haben, tippen Sie auf **Register**, um eines zu erstellen.
3. **Wählen Sie Ihre Kirche** aus, indem Sie sie nach Name durchsuchen. Geben Sie mindestens 3 Zeichen ein, um Ergebnisse zu sehen.
4. **Erkunden Sie das Dashboard** – Ihr personalisierter Startbildschirm zeigt Links zu den Funktionen, die Ihre Kirche angeboten hat.

Das ist alles. Nachdem Sie Ihre Kirche ausgewählt haben, lädt die App das Branding, die Navigationslinks und den Inhalt Ihrer Kirche. Sie können mit Gruppen, Spenden, Check-in und allem anderen beginnen, was Ihre Kirche aktiviert hat.
