# Registrierung für Veranstaltungen

<div class="article-intro">

Melden Sie sich direkt von B1 Mobile aus für Kirchenveranstaltungen an – mit einem einfachen dreistufigen Verfahren. Sie können sich selbst anmelden und Familienmitglieder direkt vom Telefon hinzufügen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Installieren Sie B1 Mobile](../getting-started/installing) und [melden Sie sich](../getting-started/logging-in) in Ihrem Konto an
- Die Veranstaltung muss die Registrierung aktiviert haben und das Registrierungsfenster muss offen sein

</div>

## Schritt 1: Geben Sie Ihre Informationen ein

1. Öffnen Sie eine Gruppenkalender-Veranstaltung mit aktivierter Registrierung und tippen Sie auf **Registrieren**.
2. Der Registrierungsbildschirm zeigt den Veranstaltungstitel, die Beschreibung und das Datum.
3. Da Sie angemeldet sind, bestätigt das Formular, dass Sie sich unter Ihrem Konto registrieren.

:::info
Wenn die Veranstaltung ein Kapazitätslimit hat, zeigt eine Fortschrittsleiste, wie viele Plätze noch verfügbar sind. Wenn die Veranstaltung voll ist, können Sie sich nicht registrieren.
:::

## Schritt 2: Fügen Sie Familienmitglieder hinzu

1. Um zusätzliche Familienmitglieder zu registrieren, geben Sie einen **Vornamen** und **Nachnamen** für jede Person ein.
2. Tippen Sie auf **Mitglied hinzufügen**, um weitere Personen hinzuzufügen (bis zu 10 insgesamt).
3. Tippen Sie auf die Entfernen-Schaltfläche neben einem Mitglied, um es von der Registrierung zu entfernen.
4. Wenn Sie sich nur selbst registrieren, überspringen Sie diesen Schritt.

## Schritt 3: Bestätigen

1. Überprüfen Sie die Veranstaltungsdetails und Ihre Registrierungsinformationen.
2. Bestätigen Sie Ihre Registrierung.
3. Eine Bestätigungsmeldung mit den Veranstaltungsdetails und allen registrierten Personen wird angezeigt.

Sie erhalten auch eine **Bestätigungs-E-Mail** unter der E-Mail-Adresse, die mit Ihrem Konto verknüpft ist.

## Registrierung stornieren

Wenden Sie sich an Ihren Kirchenadministrator, wenn Sie stornieren müssen. Sie können Ihre Registrierung auf der Registrierungsseite von B1 Admin stornieren.
