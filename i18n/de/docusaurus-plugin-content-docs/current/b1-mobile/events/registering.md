# Registrierung für Ereignisse

<div class="article-intro">

Registrieren Sie sich für Kirchenereignisse direkt von B1 Mobile mit einem einfachen dreistufigen Prozess. Sie können sich selbst anmelden und Familienmitglieder direkt von Ihrem Telefon aus hinzufügen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Installieren Sie B1 Mobile](../getting-started/installing) und [melden Sie sich an](../getting-started/logging-in) bei Ihrem Konto
- Das Ereignis muss Registrierung aktiviert haben und das Registrierungsfenster muss offen sein

</div>

## Schritt 1: Geben Sie Ihre Informationen ein

1. Öffnen Sie ein Gruppen-Kalender-Ereignis, das Registrierung aktiviert hat, und tippen Sie auf **Registrieren**.
2. Der Registrierungs-Bildschirm zeigt den Ereignistitel, die Beschreibung und das Datum.
3. Da Sie angemeldet sind, bestätigt das Formular, dass Sie sich unter Ihrem Konto registrieren.

:::info
Wenn das Ereignis ein Kapazitätslimit hat, zeigt eine Fortschrittsleiste, wie viele Plätze verbleiben. Wenn das Ereignis voll ist, können Sie sich nicht registrieren.
:::

## Schritt 2: Fügen Sie Familienmitglieder hinzu

1. Um zusätzliche Familienmitglieder zu registrieren, geben Sie einen **Vorname** und **Nachname** für jede Person ein.
2. Tippen Sie auf **Mitglied hinzufügen**, um mehr Personen hinzuzufügen (bis zu 10 insgesamt).
3. Tippen Sie auf die Entfernen-Schaltfläche neben einem Mitglied, um sie von der Registrierung zu entfernen.
4. Wenn Sie nur sich selbst registrieren, überspringen Sie diesen Schritt.

## Schritt 3: Bestätigen

1. Überprüfen Sie die Ereignis-Detailseite und Ihre Registrierungs-Informationen.
2. Bestätigen Sie Ihre Registrierung.
3. Eine Bestätigungsnachricht erscheint mit den Ereignis-Detailseite und Personen, die Sie registriert haben.

Sie erhalten auch eine **Bestätigungs-E-Mail** unter der E-Mail-Adresse, die mit Ihrem Konto verknüpft ist.

## Registrierung absagen

Wenn Sie absagen müssen, kontaktieren Sie Ihren Kirchen-Administrator. Sie können Ihre Registrierung von der B1 Admin-Registrierungs-Seite absagen.
