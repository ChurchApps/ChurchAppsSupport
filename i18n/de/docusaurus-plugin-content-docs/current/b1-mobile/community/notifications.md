# Benachrichtigungen

<div class="article-intro">

B1 Mobile verwendet Push-Benachrichtigungen, um Sie ueber das Geschehen in Ihrer Gemeinde auf dem Laufenden zu halten. Anstatt die App oeffnen zu muessen, um nach Neuigkeiten zu schauen, kommen wichtige Informationen automatisch zu Ihnen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Installieren Sie B1 Mobile](../getting-started/installing) und [melden Sie sich an](../getting-started/logging-in) bei Ihrem Konto
- Erlauben Sie Benachrichtigungen fuer B1 Mobile in Ihren Geraeteeinstellungen

</div>

## Worueber Sie benachrichtigt werden

Benachrichtigungen koennen verschiedene Aktivitaeten abdecken, darunter:

- **Nachrichten** -- Wenn Ihnen jemand eine [private Nachricht](./messaging) sendet.
- **Gruppenaktivitaeten** -- Neuigkeiten, Beitraege und Unterhaltungen aus [Gruppen](../groups/), denen Sie angehoeren.
- **Dienstzuweisungen** -- Wenn Sie einem [Plan](../serving/viewing-plans.md) zugewiesen werden oder sich der Status Ihrer Zuweisung aendert.
- **Weitere Neuigkeiten** -- Allgemeine Ankuendigungen und fuer Sie relevante Aktivitaeten.

## Ihre Benachrichtigungen ansehen

1. Tippen Sie auf das **Glockensymbol** in der oberen rechten Ecke der App oder auf die Option **Benachrichtigungen** in der Navigation.
2. Der Benachrichtigungsbildschirm zeigt alle Ihre aktuellen Benachrichtigungen in einer Tab-Ansicht, uebersichtlich organisiert, damit Sie schnell finden, wonach Sie suchen.

Jede Benachrichtigung enthaelt eine kurze Zusammenfassung des Updates. Tippen Sie auf eine beliebige Benachrichtigung, um direkt zum entsprechenden Inhalt zu gelangen -- zum Beispiel oeffnet das Tippen auf eine Nachrichtenbenachrichtigung die Unterhaltung, und das Tippen auf eine Dienstbenachrichtigung fuehrt Sie zum Plan.

## Push-Benachrichtigungen

B1 Mobile sendet Push-Benachrichtigungen an Ihr Geraet, damit Sie Neuigkeiten erhalten, auch wenn die App nicht geoeffnet ist. Um Push-Benachrichtigungen zu erhalten, stellen Sie sicher, dass Sie Benachrichtigungen fuer B1 Mobile in Ihren Geraeteeinstellungen erlaubt haben.

### Push-Benachrichtigungen aktivieren

- **iOS** -- Gehen Sie zu **Einstellungen** > **Mitteilungen** > **B1 Mobile** und aktivieren Sie **Mitteilungen erlauben**.
- **Android** -- Gehen Sie zu **Einstellungen** > **Apps** > **B1 Mobile** > **Benachrichtigungen** und stellen Sie sicher, dass Benachrichtigungen aktiviert sind.

:::warning
Wenn Sie keine Benachrichtigungen erhalten, ueberpruefen Sie Ihre Geraeteeinstellungen, um sicherzustellen, dass sie aktiviert sind. Bestaetigen Sie ausserdem, dass Sie in der App angemeldet sind, da Benachrichtigungen an Ihr Konto gebunden sind.
:::

:::tip
Sie koennen die Benachrichtigungseinstellungen auf Geraeteebene verwalten. Wenn Ihnen die Benachrichtigungen zu haeufig sind, passen Sie Ihre Geraeteeinstellungen an, um zu steuern, wie und wann Benachrichtigungen angezeigt werden.
:::
