# Benachrichtigungen

<div class="article-intro">

B1 Mobile verwendet Push-Benachrichtigungen, um Sie über das, was in Ihrer Kirche vor sich geht, auf dem Laufenden zu halten. Anstatt die App zu öffnen, um Updates zu überprüfen, kommen wichtige Informationen automatisch zu Ihnen.

</div>

<div class="prereqs">
<h4>Bevor Sie Beginnen</h4>

- [Installieren Sie B1 Mobile](../getting-started/installing) und [melden Sie sich](../getting-started/logging-in) bei Ihrem Konto an
- Erlauben Sie Benachrichtigungen für B1 Mobile in Ihren Geräteeinstellungen

</div>

## Worüber Sie Benachrichtigt Werden

Benachrichtigungen können eine Reihe von Aktivitäten abdecken, einschließlich:

- **Nachrichten** – Wenn jemand Ihnen eine [private Nachricht](./messaging) sendet.
- **Gruppenaktivität** – Updates, Posts und Gespräche aus [Gruppen](../groups/), denen Sie angehören.
- **Dienst-Zuweisungen** – Wenn Sie einem [Plan](../serving/viewing-plans.md) zugewiesen werden oder sich Ihr Zuweisungsstatus ändert.
- **Andere Updates** – Allgemeine Ankündigungen und Aktivitäten, die für Sie relevant sind.

## Anzeigen Ihrer Benachrichtigungen

1. Tippen Sie auf das **Glockensymbol** in der oberen rechten Ecke der App, oder tippen Sie in der Navigation auf die Option **Notifications**.
2. Der Benachrichtigungs-Bildschirm zeigt alle Ihre kürzlichen Benachrichtigungen in einer Registerkartenansicht an, die organisiert ist, damit Sie schnell das finden können, das Sie suchen.

Jede Benachrichtigung enthält eine kurze Zusammenfassung des Updates. Tippen Sie auf eine Benachrichtigung, um direkt zum relevanten Inhalt zu gehen – beispielsweise öffnet das Tippen auf eine Nachrichtenbenachrichtigung das Gespräch, und das Tippen auf eine Dienst-Benachrichtigung bringt Sie zum Plan.

## Benachrichtigungen Entfernen

Sie können gelesene Benachrichtigungen löschen, damit Ihre Liste auf das Neue konzentriert bleibt.

- **Eine einzelne Benachrichtigung löschen** – Tippen Sie auf das Papierkorb-Symbol auf der rechten Seite einer beliebigen Benachrichtigungszeile, um nur dieses Element zu entfernen.
- **Alle Benachrichtigungen löschen** – Tippen Sie auf **Clear All** oben in der Benachrichtigungsliste, um jede Benachrichtigung auf einmal zu entfernen. Sie werden aufgefordert, diese zu bestätigen, bevor alles gelöscht wird.

Das Entfernen einer Benachrichtigung betrifft nur Ihre Ansicht; es ändert nicht die zugrunde liegende Aktivität (die Nachricht, der Plan oder der Gruppenbeitrag ist immer noch vorhanden). Wenn Sie Benachrichtigungen auf mehreren Geräten aktiviert haben, wird die Löschung auf einem Gerät auf die anderen synchronisiert.

## Push-Benachrichtigungen

B1 Mobile sendet Push-Benachrichtigungen an Ihr Gerät, damit Sie Updates erhalten, auch wenn die App nicht geöffnet ist. Um Push-Benachrichtigungen zu erhalten, stellen Sie sicher, dass Sie Benachrichtigungen für B1 Mobile in Ihren Geräteeinstellungen erlaubt haben.

### Aktivieren von Push-Benachrichtigungen

- **iOS** – Gehen Sie zu **Settings** > **Notifications** > **B1 Mobile** und schalten Sie **Allow Notifications** ein.
- **Android** – Gehen Sie zu **Settings** > **Apps** > **B1 Mobile** > **Notifications** und stellen Sie sicher, dass Benachrichtigungen aktiviert sind.

:::warning
Wenn Sie Benachrichtigungen nicht erhalten, überprüfen Sie Ihre Geräteeinstellungen, um sicherzustellen, dass sie aktiviert sind. Bestätigen Sie auch, dass Sie sich bei der App angemeldet haben, da Benachrichtigungen an Ihr Konto gebunden sind.
:::

:::tip
Sie können Benachrichtigungspräferenzen auf der Geräteebene verwalten. Wenn Sie Benachrichtigungen zu häufig finden, passen Sie Ihre Geräteeinstellungen an, um zu kontrollieren, wie und wann Warnungen angezeigt werden.
:::
