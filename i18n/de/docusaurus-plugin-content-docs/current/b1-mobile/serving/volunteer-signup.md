# Freiwilligen-Anmeldung

<div class="article-intro">

Durchsuchen Sie verfügbare Freiwilligenpositionen und melden Sie sich direkt von B1 Mobile aus für anstehende Gottesdienste an. Sehen Sie, welche Positionen besetzt werden müssen, wie viele Plätze frei sind, und treten Sie mit nur wenigen Tippen einem Team bei.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Installieren Sie B1 Mobile](../getting-started/installing) und [melden Sie sich](../getting-started/logging-in) in Ihrem Konto an
- Ihr Kirchenadministrator muss die Selbstanmeldung für mindestens eine Position aktiviert haben

</div>

## Durchsuchen Sie Freiwilligenmöglichkeiten

1. Öffnen Sie den Abschnitt **Freiwilliger** über die App-Navigation.
2. Es wird eine Liste der anstehenden Servicepläne mit offenen Positionen angezeigt.
3. Jede Plankarte zeigt:
   - **Planname** und **Servicedatum**
   - **Gesamte Fortschrittsleiste** mit besetzten Plätzen
   - **Status** -- Anzahl der noch verfügbaren Plätze oder "Voll", wenn alle Plätze belegt sind
   - **Positions-Chips**, die jede Rolle und ihre offenen Platzanzahl anzeigen
4. Tippen Sie auf **Anzeigen und Anmelden**, um die vollständige Liste der Positionen für einen Plan zu sehen.

## Melden Sie sich für eine Position an

1. Tippen Sie auf einen Plan, um seine Details zu öffnen.
2. Positionen sind nach Kategorie gruppiert (z. B. "Band", "Gastfreundschaft", "Kinderstunde").
3. Jede Position zeigt den Rollennamen, die Beschreibung und die verfügbaren Plätze.
4. Tippen Sie neben der Position, für die Sie sich anmelden möchten, auf **Anmelden**.
5. Ein Bestätigungsdialog wird angezeigt: "Sie haben sich angemeldet! Vielen Dank für Ihre Freiwilligenarbeit."
6. Die Position zeigt einen grünen Indikator, der Ihre Anmeldung bestätigt.

:::info
Sie müssen angemeldet sein, um sich für Positionen anzumelden. Die App fordert Sie auf, sich anzumelden, falls erforderlich.
:::

## Eine Anmeldung entfernen

1. Suchen Sie die Position, für die Sie sich angemeldet haben -- sie zeigt einen grünen Indikator "Sie haben sich angemeldet".
2. Tippen Sie auf **Entfernen**.
3. Ein Bestätigungsdialog fragt, ob Sie sicher sind.
4. Bestätigen Sie, um sich von der Position zu entfernen.

## Fristen und Kapazität

- **Fristen** -- Ihre Kirche kann eine Sperrfrist vor dem Gottesdienst festlegen. Nach der Frist können Sie sich nicht anmelden oder Anmeldungen entfernen. Ein Banner oben im Plan zeigt an, wann die Frist abgelaufen ist.
- **Kapazität** -- Jede Position hat eine begrenzte Anzahl von Plätzen. Wenn sie voll ist, ist die Anmeldungsschaltfläche deaktiviert. Plätze werden wieder verfügbar, wenn sich jemand abmeldet.

:::tip
Aktivieren Sie [Push-Benachrichtigungen](../community/notifications.md), um über neue Freiwilligenmöglichkeiten und Änderungen Ihrer Aufgaben informiert zu bleiben.
:::
