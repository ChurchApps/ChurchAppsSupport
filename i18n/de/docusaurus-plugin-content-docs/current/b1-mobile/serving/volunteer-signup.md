# Freiwilligs-Anmeldung

<div class="article-intro">

Durchsuchen Sie verfügbare Freiwilligs-Positionen und melden Sie sich für kommende Gottesdienste direkt von B1 Mobile an. Sehen Sie, welche Positionen Freiwillige benötigen, wie viele Plätze offen sind, und treten Sie einem Team mit nur ein paar Taps bei.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Installieren Sie B1 Mobile](../getting-started/installing) und [melden Sie sich an](../getting-started/logging-in) bei Ihrem Konto
- Ihr Kirchen-Administrator muss Selbstanmeldung für mindestens eine Position aktiviert haben

</div>

## Durchsuchen Sie Freiwilligs-Möglichkeiten

1. Öffnen Sie den Bereich **Freiwilligs** aus der App-Navigation.
2. Eine Liste kommender Dienst-Pläne mit offenen Positionen erscheint.
3. Jede Plan-Karte zeigt:
   - **Plan-Name** und **Dienst-Datum**
   - **Gesamtfortschrittsleiste** zeigt insgesamt Plätze gefüllt
   - **Status** — Anzahl verbleibender Plätze oder "Voll", wenn alle Plätze besetzt sind
   - **Positions-Chips** zeigen jede Rolle und ihre Anzahl offener Plätze
4. Tippen Sie auf **Ansicht und Anmeldung**, um die vollständige Liste der Positionen für einen Plan zu sehen.

## Melden Sie sich für eine Position an

1. Tippen Sie auf einen Plan, um seine Details zu öffnen.
2. Positionen sind nach Kategorie gruppiert (zum Beispiel "Band", "Gastlichkeit", "Kinderministerin").
3. Jede Position zeigt den Rollen-Namen, Beschreibung und verfügbare Plätze.
4. Tippen Sie auf **Anmelden** neben der Position, für die Sie wollen.
5. Eine Bestätigungs-Benachrichtigung erscheint: "Sie sind angemeldet! Danke für Ihren Freiwilligendienst."
6. Die Position zeigt einen grünen Indikator, der bestätigt, dass Sie angemeldet sind.

:::info
Sie müssen angemeldet sein, um sich für Positionen anzumelden. Die App fordert Sie auf, sich anzumelden, wenn erforderlich.
:::

## Entfernen Sie Ihre Anmeldung

1. Finden Sie die Position, für die Sie sich angemeldet haben — sie zeigt einen grünen "Sie sind angemeldet"-Indikator.
2. Tippen Sie auf **Entfernen**.
3. Ein Bestätigungs-Dialog fragt, ob Sie sicher sind.
4. Bestätigen Sie, um sich von der Position zu entfernen.

## Fristen und Kapazität

- **Fristen** — Ihre Kirche kann einen Cutoff-Zeit vor dem Dienst setzen. Nach der Frist können Sie sich nicht anmelden oder Anmeldungen entfernen. Ein Banner oben auf dem Plan zeigt an, wenn die Frist verstrichen ist.
- **Kapazität** — Jede Position hat eine begrenzte Anzahl von Plätzen. Wenn voll, ist die Anmeldungs-Schaltfläche deaktiviert. Plätze öffnen sich wieder, wenn sich jemand abmeldet.

:::tip
Aktivieren Sie [Push-Benachrichtigungen](../community/notifications.md), um über neue Freiwilligs-Möglichkeiten und Änderungen an Ihren Aufträgen informiert zu werden.
:::
