# Spenden

<div class="article-intro">

B1 Mobile macht es einfach, Ihre Gemeinde mit einem vollstaendigen Spendenerlebnis direkt von Ihrem Telefon aus zu unterstuetzen. Ob Sie eine schnelle Spende taetigen, Ihren Spendenverlauf einsehen oder Ihre Zahlungsmethoden verwalten moechten -- alles ist an einem Ort verfuegbar.

</div>

## Die Spenden-Tabs

Der Spendenbereich ist in vier Tabs organisiert:

- **Uebersicht** -- Eine Zusammenfassung Ihrer Spendenaktivitaet. Sehen Sie Ihre Jahressumme, die Anzahl Ihrer Spenden und Details zu Ihrer letzten Spende, einschliesslich einer schnellen **Wiederholen**-Schaltflaeche, um den gleichen Betrag erneut zu spenden.
- **Spenden** -- Taetigen Sie eine neue Spende. Waehlen Sie einen Fonds, geben Sie einen Betrag ein und geben Sie Ihre Zahlungsdaten an, um eine Spende abzuschliessen.
- **Verwalten** -- Sehen und verwalten Sie Ihre gespeicherten Zahlungsmethoden. Fuegen Sie neue Karten hinzu oder entfernen Sie solche, die Sie nicht mehr benoetigen.
- **Verlauf** -- Eine detaillierte Liste aller Ihrer vergangenen Spenden, damit Sie Ihren vollstaendigen Spendenverlauf einsehen koennen.

## So funktionieren Zahlungen

Alle ueber B1 Mobile getaetigten Spenden werden sicher ueber **Stripe** abgewickelt. Ihre Zahlungsinformationen werden von Stripes sicherer Infrastruktur verarbeitet und niemals auf ChurchApps-Servern gespeichert. Sie koennen mit der Gewissheit spenden, dass Ihre Finanzdaten geschuetzt sind.

:::info
Ihr Gemeindeadministrator richtet die verfuegbaren Fonds und die Zahlungsabwicklung ueber [B1 Admin](../../b1-admin/) ein. Wenn Sie keine Spendenoption in der App sehen, hat Ihre Gemeinde diese Funktion moeglicherweise noch nicht konfiguriert.
:::

## Was Sie tun koennen

Folgendes koennen Sie im Spendenbereich tun:

- **An bestimmte Fonds spenden** -- Waehlen Sie, an welchen Fonds Ihre Spende gerichtet werden soll, z. B. Allgemeiner Fonds, Missionsfonds oder Baufonds.
- **Ihre Spenden einsehen** -- Pruefen Sie Ihre Jahreszusammenfassung und durchsuchen Sie Ihren vollstaendigen Spendenverlauf.
- **Zahlungsmethoden speichern** -- Speichern Sie Ihre Kartendaten fuer schnelleres Spenden in der Zukunft.
- **Eine Spende wiederholen** -- Spenden Sie schnell den gleichen Betrag wie Ihre letzte Spende mit einem einzigen Fingertipp.
