# Profil verwalten

<div class="article-intro">

Die Profilseite ermöglicht es Ihnen, Ihre persönlichen Kontoeinstellungen zu bearbeiten. Ihr Profil wird in allen ChurchApps-Produkten geteilt, sodass alle Änderungen, die Sie hier vornehmen, überall dort widergespiegelt werden, wo Sie ChurchApps nutzen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie müssen in Ihrem ChurchApps-Konto angemeldet sein
- Halten Sie Ihre neue E-Mail-Adresse oder Ihr neues Passwort bereit, falls Sie diese aktualisieren möchten

</div>

## Profilinformationen

Der obere Bereich der Profilseite enthält Ihre grundlegenden Kontodaten:

1. **E-Mail** -- Ihre E-Mail-Adresse, die zur Anmeldung und zum Empfang von Benachrichtigungen verwendet wird.
2. **Vorname** und **Nachname** -- Ihr Anzeigename in ChurchApps.
3. **Passwort** -- Aktualisieren Sie Ihr Kontopasswort.

Um Ihre Informationen zu aktualisieren, bearbeiten Sie die Felder und klicken Sie auf **Änderungen speichern**.

## Passwort ändern

1. Scrollen Sie auf der Profilseite zu den Passwortfeldern.
2. Geben Sie Ihr neues Passwort im Feld **Passwort** ein.
3. Geben Sie es erneut im Feld **Passwort bestätigen** ein.
4. Klicken Sie auf **Änderungen speichern**.

:::info
Ihr Passwort muss mindestens 8 Zeichen lang sein. Stellen Sie sicher, dass das Passwort und die Bestätigung genau übereinstimmen.
:::

## Verknüpfte Konten

Der Bereich **Verknüpfte Konten** zeigt alle externen Dienste, die mit Ihrem ChurchApps-Konto verbunden sind. Sie können einsehen, welche Konten verknüpft sind, und diese Verbindungen von diesem Bereich aus verwalten.

## Designeinstellungen

ChurchApps unterstützt sowohl helles als auch dunkles Design:

1. Suchen Sie den **Design**-Schalter auf Ihrer Profilseite.
2. Wechseln Sie zwischen **Hell** und **Dunkel**.
3. Die Oberfläche wird sofort aktualisiert, um Ihre Wahl widerzuspiegeln.

Ihre Designeinstellung wird in Ihrem Konto gespeichert und gilt für alle Ihre Sitzungen.

## Kontolöschung

Am Ende der Profilseite finden Sie die Option, Ihr Konto dauerhaft zu löschen. Diese Aktion entfernt Ihr Konto und die zugehörigen Daten und kann nicht rückgängig gemacht werden.

:::warning
Die Kontolöschung ist dauerhaft und kann nicht rückgängig gemacht werden. Stellen Sie sicher, dass Sie Ihr Konto wirklich entfernen möchten, bevor Sie fortfahren.
:::

:::info
Wenn Sie ein **Domain Admin** sind, stellen Sie sicher, dass Sie administrative Verantwortlichkeiten an einen anderen Benutzer übertragen, bevor Sie Ihr Konto löschen. Siehe [Rollen & Berechtigungen](../settings/roles-permissions.md) für Details zur Verwaltung von Admin-Rollen.
:::

:::info
Die Profilbearbeitung ist im Demo-Modus deaktiviert. Wenn Sie ein Demo-Konto verwenden, können Sie keine Änderungen an Ihren Profileinstellungen vornehmen.
:::
