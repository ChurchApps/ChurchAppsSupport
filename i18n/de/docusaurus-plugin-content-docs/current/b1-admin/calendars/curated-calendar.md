# Kuratierter Kalender

<div class="article-intro">

Ein kuratierter Kalender fasst Veranstaltungen aus mehreren Gruppen in einer einheitlichen Kalenderansicht zusammen. Dies ist ideal, um einen gemeindeweiten Kalender zu erstellen, der alles zeigt, was in all Ihren Diensten und Gruppen an einem einzigen Ort passiert.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Richten Sie die [Gruppen](../groups/creating-groups.md) ein, deren Veranstaltungen Sie einbeziehen möchten
- Stellen Sie sicher, dass Gruppenleiter Veranstaltungen zu ihren jeweiligen Gruppen hinzugefügt haben
- Machen Sie sich mit den Grundlagen von [Kalender erstellen](creating-calendars) vertraut

</div>

## Einen kuratierten Kalender erstellen

1. Navigieren Sie in B1 Admin zum Bereich **Website**.
2. Klicken Sie in der oberen Navigation auf **Kalender**.
3. Klicken Sie auf **Kalender hinzufügen**, um einen neuen kuratierten Kalender zu erstellen.
4. Geben Sie Ihrem Kalender einen **Namen** (zum Beispiel „Gemeindeveranstaltungen" oder „Diese Woche in unserer Gemeinde").
5. Klicken Sie auf **Erstellen**, um ihn zu speichern.

## Gruppen zu Ihrem Kalender hinzufügen

1. Klicken Sie auf Ihren neuen Kalender, um ihn zu öffnen.
2. Klicken Sie im Abschnitt **Gruppen im Kalender** auf der rechten Seite auf **Hinzufügen**.
3. Wählen Sie eine Gruppe aus dem Dropdown-Menü.
4. Wählen Sie, ob Sie **alle Veranstaltungen** dieser Gruppe oder nur **bestimmte Veranstaltungen** hinzufügen möchten.
5. Klicken Sie auf **Speichern**.
6. Wiederholen Sie diesen Vorgang, um so viele Gruppen hinzuzufügen, wie Sie einbeziehen möchten.

:::tip
Fügen Sie alle Ihre aktiven Dienstgruppen hinzu, um einen umfassenden Kalender zu erstellen, der Ihrer Gemeinde ein vollständiges Bild davon gibt, was jede Woche in Ihrer Gemeinde passiert.
:::

## Ihren Kalender teilen

Sobald Ihr kuratierter Kalender eingerichtet ist, können Sie ihn auf verschiedene Weise teilen:

- **Auf Ihrer Website einbetten** -- Der Kalender kann direkt auf Ihren B1.church-Website-Seiten angezeigt werden, damit Besucher darin stöbern können.
- **Per ICS abonnieren** -- Klicken Sie auf **Abonnieren**, um den ICS-Link zu kopieren. Ihre Gemeindemitglieder können diesen Link in Google Calendar, Apple Kalender, Outlook oder jede andere Kalender-App einfügen, um synchron zu bleiben.
- **ICS-Datei herunterladen** -- Laden Sie die Kalenderdatei herunter, um sie in eine Kalenderanwendung zu importieren.

:::info
Wenn Gruppenleiter neue Veranstaltungen zu ihren [Gruppen](../groups/creating-groups.md) hinzufügen, erscheinen diese automatisch in Ihrem kuratierten Kalender. Sie müssen den kuratierten Kalender nicht jedes Mal manuell aktualisieren, wenn eine neue Veranstaltung hinzugefügt wird.
:::
