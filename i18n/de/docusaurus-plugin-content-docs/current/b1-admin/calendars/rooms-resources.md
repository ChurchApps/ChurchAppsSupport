# Räume, Ressourcen & Planung

<div class="article-intro">

Räume und Ressourcen ermöglichen es Ihnen, die buchbaren Räume und Ausrüstung Ihrer Kirche zu verfolgen. Wenn eine Gruppe oder ein Team einen Raum nutzen oder eine Ressource für ein Ereignis ausleihen möchte, kann sie B1 anfordern – und Sie können diese Anfragen genehmigen oder ablehnen, um Doppelbuchungen zu verhindern.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen administrative Berechtigung für den Abschnitt „Kalender" in B1 Admin
- Richten Sie [Gruppen](../groups/creating-groups) ein, deren Mitglieder Buchungsanfragen einreichen werden

</div>

## Räume & Ressourcen öffnen

Gehen Sie in B1 Admin zu **Kalender** und wählen Sie **Räume & Ressourcen**. Die Seite hat vier Registerkarten: **Räume**, **Ressourcen**, **Blockierungen** und **Vorlagen**.

## Räume

Räume sind physische Räume, die zusammen mit Kalender-Ereignissen reserviert werden können – ein Heiligtum, eine Gemeinschaftshalle, ein Klassenzimmer usw.

### Raum hinzufügen

1. Klicken Sie auf der Registerkarte **Räume** auf **Raum hinzufügen**.
2. Geben Sie einen **Namen** für den Raum ein.
3. Legen Sie optional eine **Kapazität** fest (maximale Personenzahl).
4. Legen Sie eine **Genehmigungsgruppe** fest – wenn eine Gruppe diesen Raum anfordert, werden die Leiter der gewählten Gruppe benachrichtigt und müssen die Buchung genehmigen. Lassen Sie das Feld leer, wenn keine Genehmigung erforderlich ist.
5. Klicken Sie auf **Speichern**.

### Raum bearbeiten oder löschen

Klicken Sie auf das Bearbeiten-Symbol neben einem beliebigen Raum, nehmen Sie Ihre Änderungen vor und klicken Sie auf **Speichern**. Verwenden Sie **Löschen**, um einen Raum dauerhaft zu entfernen.

## Ressourcen

Ressourcen sind Gegenstände, die zusammen mit einem Ereignis gebucht werden können – AV-Ausrüstung, Projektoren, Stühle, einen Van usw.

### Ressource hinzufügen

1. Klicken Sie auf die Registerkarte **Ressourcen** und dann auf **Ressource hinzufügen**.
2. Geben Sie einen **Namen** und optional eine **Beschreibung** ein.
3. Legen Sie eine **Genehmigungsgruppe** fest, wenn jemand die Ressourcenverleih genehmigen muss.
4. Klicken Sie auf **Speichern**.

## Blockierungen

Blockierungen verhindern, dass Räume oder Ressourcen während bestimmter Zeiten gebucht werden – für Renovierungen, reservierte Zeiträume oder Wartung.

### Blockierung hinzufügen

1. Klicken Sie auf die Registerkarte **Blockierungen** und dann auf **Blockierung hinzufügen**.
2. Wählen Sie, ob Sie einen **Raum**, eine **Ressource** oder alle Räume und Ressourcen blockieren möchten.
3. Legen Sie das **Start**- und **End**-Datum/die Uhrzeit für die Blockierung fest.
4. Fügen Sie optional eine **Grund**-Notiz hinzu.
5. Klicken Sie auf **Speichern**.

## Ereignisvorlagen

Vorlagen ermöglichen es Ihnen, eine wiederverwendbare Ereigniskonfiguration – Titel, Beschreibung und zugehörige Gruppen – zu speichern, damit wiederkehrende Ereignisse konsistent erstellt werden können.

### Vorlage hinzufügen

1. Klicken Sie auf die Registerkarte **Vorlagen** und dann auf **Vorlage hinzufügen**.
2. Geben Sie einen **Namen** für die Vorlage ein.
3. Verlinken Sie optional eine **Gruppe**, damit das Ereignis automatisch mit dieser Gruppe verbunden wird, wenn es aus der Vorlage erstellt wird.
4. Klicken Sie auf **Speichern**.

Wenn Sie ein neues Ereignis erstellen, können Mitarbeiter eine vorhandene Vorlage wählen, um die Ereignisdetails voreinzufüllen.
