# Personen suchen

<div class="article-intro">

Die Seite **Personen** zeigt Ihr Gemeindeverzeichnis in einer durchsuchbaren, sortierbaren Tabelle an. Sie können schnell jede Person in Ihrer Gemeinde finden, anpassen, welche Informationen angezeigt werden, und Ihre Ergebnisse exportieren. Effizientes Suchen ist essenziell für alltägliche Verwaltungsaufgaben wie die Nachverfolgung von Besuchern, das Erstellen von Kontaktlisten und die Verwaltung von Mitgliedereinträgen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen ein aktives B1 Admin-Konto mit Berechtigung zur Ansicht von Personen. Siehe [Rollen & Berechtigungen](roles-permissions.md), wenn Sie sich über Ihre Zugangsstufe unsicher sind.
- Ihr Gemeindeverzeichnis sollte Personen enthalten. Falls Sie noch niemanden hinzugefügt haben, siehe [Personen hinzufügen](adding-people.md) oder [Daten importieren](importing-data.md).

</div>

## Schnellsuche

Die Suchleiste oben auf der Personenseite ermöglicht es Ihnen, Mitglieder in Echtzeit zu finden:

1. Klicken Sie auf das **Suchfeld** oben auf der Personenseite.
2. Beginnen Sie, einen Namen, eine E-Mail oder ein anderes Stichwort einzugeben.
3. Die Ergebnisse werden automatisch gefiltert, während Sie tippen (es gibt eine kurze Verzögerung von etwa einer halben Sekunde, damit die Suche nicht bei jedem Tastendruck ausgelöst wird).
4. Die Tabelle unten aktualisiert sich und zeigt nur die passenden Ergebnisse.

:::tip
Sie müssen nicht Enter drücken. Die Suche wird automatisch ausgeführt, nachdem Sie mit dem Tippen aufgehört haben.
:::

## Ergebnisse sortieren

Sie können das Verzeichnis sortieren, indem Sie auf eine beliebige Spaltenüberschrift in der Tabelle klicken:

1. Klicken Sie auf eine **Spaltenüberschrift** (zum Beispiel **Name** oder **E-Mail**), um nach dieser Spalte zu sortieren.
2. Klicken Sie erneut auf dieselbe Überschrift, um die Sortierreihenfolge umzukehren.

So können Sie Personen leicht alphabetisch, nach Alter oder nach jeder anderen sichtbaren Spalte finden.

## Spalten anpassen

Nicht jede Information muss gleichzeitig sichtbar sein. Sie können wählen, welche Spalten in der Tabelle erscheinen:

1. Suchen Sie das **Spaltenauswahl-Dropdown** oben in der Tabelle.
2. Aktivieren oder deaktivieren Sie Spalten zum Anzeigen oder Ausblenden. Verfügbare Spalten sind:
   - **Foto**
   - **Name**
   - **E-Mail**
   - **Telefon**
   - **Adresse**
   - **Geburtsdatum**
   - **Alter**
   - **Geschlecht**
   - **Mitgliedschaftsstatus**
3. Die Tabelle aktualisiert sich sofort, um Ihre Auswahl widerzuspiegeln.

:::info
Ihre Spaltenauswahl beeinflusst, was beim CSV-Export enthalten ist. Passen Sie die Spalten vor dem Export an, um genau die benötigten Daten zu erhalten.
:::

## Suchergebnisse exportieren

Sie können Ihre aktuellen Suchergebnisse jederzeit als CSV-Datei herunterladen:

1. Wenden Sie die gewünschten Such- oder Filterkriterien an.
2. Passen Sie Ihre Spalten an, um die benötigten Daten einzubeziehen.
3. Klicken Sie auf die Schaltfläche **Exportieren**.
4. Eine CSV-Datei wird auf Ihren Computer heruntergeladen, bereit zum Öffnen in Excel, Google Sheets oder jeder Tabellenkalkulationsanwendung.

Weitere Details zum Export finden Sie unter [Daten exportieren](./exporting-data.md).

:::tip
Für fortgeschrittenere Abfragen -- wie das Finden aller Personen, die in den letzten drei Monaten nicht teilgenommen haben -- probieren Sie die [KI-Suche](./ai-search.md), die es Ihnen ermöglicht, in natürlicher Sprache zu suchen.
:::
