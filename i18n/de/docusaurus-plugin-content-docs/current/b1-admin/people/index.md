# Personen

<div class="article-intro">

Der **Personen**-Bereich ist das Herzstück Ihrer Kirchenverwaltung in B1 Admin. Hier pflegen Sie Ihr Kirchenverzeichnis -- suchen nach Mitgliedern, fügen neue Personen hinzu, sehen und bearbeiten Profile und verfolgen Haushalte. Ob Sie eine kleine Gemeinde oder eine große haben, die Organisation Ihrer Personendaten ist das Fundament, das jede andere Funktion in B1 antreibt.

</div>

## Was Sie tun können

Hier ist ein Überblick über die Hauptfunktionen im Personen-Bereich:

1. **Durchsuchen Sie Ihr Verzeichnis** -- Verwenden Sie die Schnellsuchleiste, um jede Person nach Namen zu finden, oder nutzen Sie erweiterte Filter und KI-gestützte natürlichsprachliche Suche, um genau die richtige Person zu finden. Siehe [Personen suchen](./searching-people.md) und [KI-Suche](./ai-search.md) für Details.

2. **Neue Personen hinzufügen** -- Fügen Sie schnell Einzelpersonen zu Ihrem Verzeichnis hinzu und füllen Sie deren Profildetails, Kontaktinformationen und Haushaltsverbindungen aus. Siehe [Personen hinzufügen](./adding-people.md) für schrittweise Anleitungen.

3. **Personen in großen Mengen bearbeiten** -- Aktualisieren Sie mehrere Personen gleichzeitig, indem Sie deren Mitgliedschaftsstatus, Familienstand, Geschlecht, Abmeldepräferenzen oder Gruppenmitgliedschaften ändern. Dies spart Zeit, wenn Sie dieselbe Änderung bei vielen Personen vornehmen. Siehe [Massenbearbeitung](./bulk-editing.md).

4. **Daten importieren und exportieren** -- Importieren Sie Mitgliederdaten aus einer CSV-Datei oder migrieren Sie von einem anderen Kirchenverwaltungssystem wie Breeze. Sie können Ihr Verzeichnis jederzeit auch als CSV exportieren. Siehe [Daten importieren](./importing-data.md) und [Daten exportieren](./exporting-data.md).

5. **Profile verwalten** -- Zeigen Sie detaillierte Profile für jede Person an und bearbeiten Sie diese, einschließlich Kontaktinformationen, Haushaltsmitglieder, [Gruppenmitgliedschaften](../groups/group-members.md), [Anwesenheitshistorie](../attendance/tracking-attendance.md), [Spenden](../donations/recording-donations.md) und [benutzerdefinierte Formulare](../forms/creating-forms.md).

6. **Rollen und Berechtigungen zuweisen** -- Kontrollieren Sie, wer in Ihrem Team auf verschiedene Teile von B1 Admin zugreifen kann, indem Sie Rollen zuweisen. Siehe [Rollen zuweisen](./roles-permissions.md).

## Erste Schritte

Um auf den Personen-Bereich zuzugreifen, klicken Sie auf **Personen** in der linken Seitenleiste Ihres B1 Admin Dashboards. Sie sehen Ihr vollständiges Kirchenverzeichnis mit einer Suchleiste oben und einer Liste der Mitglieder darunter.

:::tip
Wenn Sie B1 zum ersten Mal einrichten, beginnen Sie mit dem [Import Ihrer vorhandenen Mitgliederdaten](./importing-data.md). Überprüfen und bereinigen Sie dann die Profile nach Bedarf.
:::
