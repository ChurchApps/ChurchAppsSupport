# Personen

<div class="article-intro">

Der Bereich **Personen** ist das Herzstück Ihrer Kirchenverwaltung in B1 Admin. Hier verwalten Sie Ihr Kirchenverzeichnis – suchen Sie nach Mitgliedern, fügen Sie neue Personen hinzu, zeigen Sie Profile an und bearbeiten Sie sie, und verfolgen Sie Haushalte. Ob Sie eine kleine Gemeinde oder eine große haben, das Organisieren Ihrer Personendaten ist die Grundlage, die jede andere Funktion in B1 unterstützt.

</div>

## Was Sie tun können

Hier ist ein Überblick über die wichtigsten Funktionen, die im Bereich Personen verfügbar sind:

1. **Durchsuchen Sie Ihr Verzeichnis** – Verwenden Sie die Quick-Search-Leiste, um jemanden schnell nach Name zu finden, oder nutzen Sie erweiterte Filter und KI-gesteuerte Suche in natürlicher Sprache, um genau die Person zu finden, die Sie brauchen. Weitere Informationen finden Sie unter [Personen durchsuchen](./searching-people.md) und [KI-Suche](./ai-search.md).

2. **Speichern Sie Suchanfragen als Listen** – Speichern Sie jede Suche oder jeden Filter unter einem Namen als benannte Liste, damit Sie sie später mit einem Klick erneut laden können. Weitere Informationen finden Sie unter [Gespeicherte Listen](./lists.md).

3. **Sehen Sie sich die Demografie an** – Sehen Sie eine visuelle Aufschlüsselung Ihrer Gemeinde nach Alter, Geschlecht, Familienstand, Mitgliedschaftsstatus und Standort – und klicken Sie auf ein beliebiges Diagrammsegment, um sich diese Personen näher anzusehen. Weitere Informationen finden Sie unter [Demografie](./demographics.md).

4. **Fügen Sie neue Personen hinzu** – Fügen Sie schnell Einzelpersonen zu Ihrem Verzeichnis hinzu und füllen Sie ihre Profilinformationen, Kontaktinformationen und Haushaltszusammenhänge aus. Weitere Informationen finden Sie unter [Personen hinzufügen](./adding-people.md) für schrittweise Anweisungen.

5. **Bearbeiten Sie Personen in Massen** – Aktualisieren Sie mehrere Personen gleichzeitig, indem Sie deren Mitgliedschaftsstatus, Familienstand, Geschlecht, Abmeldungspräferenzen oder Gruppenmitgliedschaften ändern. Dies spart Zeit, wenn Sie bei vielen Personen die gleiche Änderung vornehmen müssen. Weitere Informationen finden Sie unter [Massen-Bearbeitung](./bulk-editing.md).

6. **Drucken Sie Ihr Verzeichnis** – Generieren Sie ein formatiertes, druckbares Mitgliedverzeichnis, das nach Haushalten gruppiert ist, komplett mit Fotos und Kontaktinformationen. Weitere Informationen finden Sie unter [Verzeichnis drucken](./print-directory.md).

7. **Importieren und exportieren Sie Daten** – Importieren Sie Mitgliedsdaten aus einer CSV-Datei oder migrieren Sie von einem anderen Kirchenverwaltungssystem wie Breeze. Sie können Ihr Verzeichnis auch jederzeit als CSV exportieren. Weitere Informationen finden Sie unter [Daten importieren](./importing-data.md) und [Daten exportieren](./exporting-data.md).

8. **Verwalten Sie Profile** – Zeigen Sie detaillierte Profile für jede Person an und bearbeiten Sie sie, einschließlich Kontaktinformationen, Haushaltsmitglieder, [Gruppenmitgliedschaften](../groups/group-members.md), [Anwesenheitsverlauf](../attendance/tracking-attendance.md), [Spenden](../donations/recording-donations.md) und [benutzerdefinierte Formulare](../forms/creating-forms.md).

9. **Weisen Sie Rollen und Berechtigungen zu** – Kontrollieren Sie, wer in Ihrem Team Zugriff auf verschiedene Teile von B1 Admin hat, indem Sie Rollen zuweisen. Weitere Informationen finden Sie unter [Rollen zuweisen](./roles-permissions.md).

## Erste Schritte

Um auf den Bereich Personen zuzugreifen, klicken Sie auf **Personen** in der linken Seitenleiste Ihres B1-Admin-Dashboards. Sie sehen Ihr komplettes Kirchenverzeichnis mit einer Suchleiste oben und einer Liste von Mitgliedern darunter.

:::tip
Wenn Sie B1 zum ersten Mal einrichten, beginnen Sie mit dem [Importieren Ihrer bestehenden Mitgliedsdaten](./importing-data.md). Überprüfen und bereinigen Sie dann die Profile nach Bedarf.
:::
