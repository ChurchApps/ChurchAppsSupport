# Gespeicherte Listen

<div class="article-intro">

**Gespeicherte Listen** ermöglichen es Ihnen, eine Suchabfrage unter einem Namen zu speichern und jederzeit mit einem Klick erneut auszuführen. Erstellen Sie eine Liste einmal -- "Aktive Mitglieder auf Campus Nord", "Besucher in den letzten 90 Tagen" oder einen beliebigen erweiterten Filter -- und sie bleibt in Ihrer Seitenleiste, damit Sie die Abfrage nie wieder neu aufbauen müssen.

</div>

<div class="prereqs">
<h4>Bevor Sie anfangen</h4>

- Führen Sie mindestens eine Suche im Bereich "Personen" durch, damit Sie die Ergebnisse speichern können. Siehe [Personen suchen](./searching-people.md) oder [KI-Suche](./ai-search.md).

</div>

## Speichern einer Suche als Liste

1. Führen Sie eine Suche auf der Seite **Personen** durch -- eine Schnellsuche, eine KI-Suche oder eine erweiterte Filter-Suche.
2. Wenn die Ergebnisse angezeigt werden, wird eine Schaltfläche **Als Liste speichern** (Lesezeichensymbol) in der Header der Personenseite angezeigt.
3. Klicken Sie auf **Als Liste speichern**.
4. Geben Sie einen **Namen** für die Liste ein (erforderlich).
5. Geben Sie optional eine **Kategorie** ein, um verwandte Listen in der Seitenleiste zu gruppieren (z.B. "Outreach" oder "Mitgliedschaft").
6. Klicken Sie auf **Speichern**.

Die Liste wird jetzt auf dem Konto Ihrer Kirche gespeichert und wird im Panel **Gespeicherte Listen** auf der linken Seite der Personenseite angezeigt.

:::info
Listen werden in Ihrer Kirche geteilt -- jeder mit Zugriff auf Personen kann sie sehen und laden. Nur Mitarbeiter mit der entsprechenden Berechtigung können Listen umbenennen oder löschen.
:::

## Laden einer gespeicherten Liste

Klicken Sie im Panel **Gespeicherte Listen** auf der linken Seite der Personenseite auf einen beliebigen Listennamen. Die Suche wird sofort mit der gespeicherten Abfrage erneut ausgeführt und die Ergebnistabelle aktualisiert.

Listen werden nach Kategorie gruppiert (falls Sie eine zugewiesen haben), mit nicht kategorisierten Listen, die zuletzt angezeigt werden.

:::tip
Listen sind **Live-Abfragen**, keine Snapshots. Jedes Mal, wenn Sie eine Liste laden, wird die Suche frisch gegen Ihre aktuellen Daten ausgeführt, sodass neue Personen, die die Kriterien erfüllen, angezeigt werden.
:::

## Umbenennen und Löschen von Listen

Mitarbeiter mit Verwaltungsberechtigung können Listen direkt aus dem Panel "Gespeicherte Listen" umbenennen oder löschen.

- Klicken Sie auf das **Stiftsymbol** neben einem Listennamen, um ihn umzubenennen.
- Klicken Sie auf das **Papierkorb-Symbol**, um ihn zu löschen (Sie werden aufgefordert, zu bestätigen).

## Anwendungsfälle

| Szenario | So erstellen Sie die Liste |
|---|---|
| Alle Besucher | Erweiterte Suche: Mitgliedschaftsstatus = Besucher |
| Mitglieder auf einem bestimmten Campus | Erweiterte Suche: Campus = [Ihr Campus] |
| Personen ohne E-Mail-Adresse | Erweiterte Suche: E-Mail ist leer |
| Freiwillige mit einer abgelaufenen Hintergrundüberprüfung | Erweiterte Suche: [Benutzerdefiniertes Feld](../settings/custom-fields.md) "Hintergrund-Check läuft ab" ist vor heute |
| Ergebnisse einer KI-Frage | Stellen Sie eine Frage in der KI-Suche, dann speichern Sie sie |
