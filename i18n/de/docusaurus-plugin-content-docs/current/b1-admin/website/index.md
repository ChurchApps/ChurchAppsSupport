# Website

<div class="article-intro">

Der Bereich Website in B1 Admin bietet Ihnen einen vollwertigen Website-Baukasten für Ihre Gemeinde. Sie können Seiten erstellen, das Erscheinungsbild Ihrer Website anpassen, Mediendateien verwalten und Veranstaltungskalender veröffentlichen -- alles ohne eine einzige Zeile Code zu schreiben.

</div>

## Was Sie tun können

Der Website-Baukasten ist in fünf Hauptbereiche unterteilt:

### Seiten

Erstellen und verwalten Sie alle Seiten Ihrer Gemeinde-Website. Die **Seiten**-Tabelle zeigt jede Seite zusammen mit ihrem Status -- entweder **Generiert** (automatisch vom System erstellt) oder **Benutzerdefiniert** (von Ihnen erstellt). Sie können neue benutzerdefinierte Seiten hinzufügen, bestehende bearbeiten oder automatisch generierte Seiten in benutzerdefinierte Seiten umwandeln, um die volle Kontrolle über deren Inhalt zu erhalten.

Die linke Seitenleiste zeigt Ihre Navigationslinks in einer Baumstruktur an. Sie können Links per Drag & Drop neu anordnen, um die gewünschte Hierarchie und Reihenfolge für Besucher festzulegen.

### Blöcke

Erstellen Sie wiederverwendbare Inhaltskomponenten, die Sie auf mehreren Seiten platzieren können. Blöcke sparen Ihnen Zeit, wenn Sie Inhalte wie eine Handlungsaufforderung, ein Ankündigungsbanner oder einen Kontaktbereich haben, die an mehreren Stellen erscheinen. Aktualisieren Sie einen Block einmal und die Änderung wird überall übernommen, wo er verwendet wird.

### Erscheinungsbild

Passen Sie das Theme, die Farben, Schriftarten, das Logo und die Abstände Ihrer Website an. Sie können vorgefertigte Stil-Themes für einen schnellen Start anwenden oder jedes Detail mit dem Farbpaletten-Editor, den Typografie-Einstellungen und benutzerdefiniertem CSS feinabstimmen. Siehe die Anleitung [Erscheinungsbild](appearance) für alle Details.

### Dateien

Laden Sie alle Ihre Medien-Assets hoch und organisieren Sie sie -- Bilder, Dokumente und andere Dateien -- an einem zentralen Ort. Verwenden Sie Ordner, um alles übersichtlich zu halten, und kopieren Sie Datei-URLs zur Verwendung an beliebiger Stelle auf Ihrer Website. Siehe [Dateien](files) für weitere Informationen.

### Kalender

Veröffentlichen Sie Veranstaltungskalender auf Ihrer Website, damit Besucher sehen können, was in Ihrer Gemeinde los ist. Erstellen Sie kuratierte Kalender, die Veranstaltungen aus mehreren Gruppen in einer einzigen Ansicht zusammenführen.

:::tip
Beginnen Sie mit der [Ersteinrichtung](initial-setup), wenn Sie Ihre Website zum ersten Mal konfigurieren. Sie führt Sie durch die Domain-Einrichtung, die Erstellung Ihrer ersten Seite und die Konfiguration des Erscheinungsbilds Ihrer Website.
:::
