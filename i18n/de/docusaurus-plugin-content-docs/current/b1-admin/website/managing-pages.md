# Seiten verwalten

<div class="article-intro">

Die Website-Seitenansicht ist Ihre zentrale Anlaufstelle zum Erstellen, Bearbeiten und Organisieren aller Seiten Ihrer Gemeinde-Website. Sie können sowohl Ihren Seiteninhalt als auch die Navigation Ihrer Website von diesem einen Bildschirm aus verwalten.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Schließen Sie die [Ersteinrichtung](initial-setup) ab, um Ihre Domain und die grundlegenden Website-Einstellungen zu konfigurieren
- Halten Sie Ihre Inhalte und Bilder bereit. Verwenden Sie den [Dateien](files)-Manager, um Medien-Assets vorab hochzuladen.

</div>

## Seitentypen verstehen

Die **Seiten**-Tabelle listet jede Seite Ihrer Website zusammen mit ihrem Status auf:

- **Generiert** -- Seiten, die automatisch vom System basierend auf den Daten Ihrer Gemeinde erstellt wurden (zum Beispiel eine Gruppen-Seite oder Predigten-Seite). Diese Seiten aktualisieren sich selbst, wenn sich Ihre Daten ändern.
- **Benutzerdefiniert** -- Seiten, die Sie selbst mit eigenem Inhalt und Layout erstellt haben.

Sie können jede automatisch generierte Seite in eine benutzerdefinierte Seite umwandeln, wenn Sie die volle Kontrolle über Inhalt und Design haben möchten.

## Seiten hinzufügen und bearbeiten

1. Klicken Sie oben rechts in der Seiten-Tabelle auf die Schaltfläche **Seite hinzufügen**.
2. Wählen Sie einen Seitentyp (leer oder eine Vorlage) und geben Sie einen Namen ein.
3. Klicken Sie neben einer beliebigen Seite auf **Bearbeiten**, um den Seitenbaukasten zu öffnen, in dem Sie Abschnitte, Text, Bilder und andere Elemente hinzufügen können.
4. Klicken Sie auf **Seiteneinstellungen**, um den Seitentitel, den URL-Pfad und andere Metadaten zu aktualisieren.
5. Verwenden Sie die Schaltfläche **Vorschau**, um Ihre Seite in einem neuen Fenster zu öffnen und genau zu sehen, wie sie für Besucher aussehen wird.

:::tip
Setzen Sie für Ihre Startseite den URL-Pfad auf einfach `/`. Für alle anderen Seiten verwenden Sie einen beschreibenden Pfad wie `/ueber-uns` oder `/kontakt`.
:::

## Navigation verwalten

Die linke Seitenleiste der Website-Seitenansicht zeigt Ihre Navigationslinks an. Diese Links steuern das Menü, das Besucher auf Ihrer Website sehen.

1. Klicken Sie auf **Hinzufügen**, um einen neuen Navigationslink zu erstellen. Sie können ihn auf eine beliebige Seite Ihrer Website oder auf eine externe URL verweisen lassen.
2. Um Links neu zu ordnen, ziehen Sie sie per Drag & Drop in die gewünschte Reihenfolge. Sie können Links auch unter einem übergeordneten Eintrag verschachteln, um Dropdown-Menüs zu erstellen.
3. Klicken Sie auf das **Bearbeiten**-Symbol neben einem beliebigen Link, um dessen Beschriftung, URL oder Position zu ändern.
4. Um einen Link aus der Navigation zu entfernen, klicken Sie auf das **Löschen**-Symbol.

:::info
Das Entfernen eines Navigationslinks löscht nicht die Seite selbst. Die Seite existiert weiterhin und kann direkt über ihre URL aufgerufen werden -- sie erscheint lediglich nicht mehr im Menü.
:::

## Tipps zur Organisation Ihrer Website

- Beschränken Sie Ihre Hauptnavigation auf fünf oder sechs Einträge, damit Besucher Inhalte schnell finden können.
- Verwenden Sie verschachtelte Links für zusammengehörige Unterseiten (zum Beispiel ein "Über uns"-Dropdown mit "Unser Team", "Glaubensbekenntnis" und "Geschichte").
- Überprüfen Sie Ihre Navigation auf Mobilgeräten, indem Sie auf **Mobile Vorschau** klicken, um sicherzustellen, dass sie auf kleineren Bildschirmen gut funktioniert.
- Geben Sie Seiten klare, beschreibende Namen, die Besuchern helfen zu verstehen, was sie dort finden werden.

:::tip
Sie können [Formulare](../forms/creating-forms.md) zu Ihren Seiten hinzufügen, um Anmeldungen, Gebetsanliegen oder andere Informationen von Besuchern zu erfassen.
:::
