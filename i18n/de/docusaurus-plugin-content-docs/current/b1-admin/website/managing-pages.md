# Seiten verwalten

<div class="article-intro">

Die Website-Seiten-Ansicht ist Ihr zentraler Hub zum Erstellen, Bearbeiten und Organisieren aller Seiten auf Ihrer Kirchenwebsite. Sie können beide Ihre Seiten-Inhalte und die Navigation Ihrer Website von diesem einzigen Bildschirm aus verwalten.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Schließen Sie die [Anfängliche Konfiguration](initial-setup) ab, um Ihre Domäne und grundlegende Website-Einstellungen zu konfigurieren
- Haben Sie Ihre Inhalte und Bilder bereit. Verwenden Sie zuerst den [Dateien](files)-Manager, um Medien-Assets hochzuladen.

</div>

## Seiten-Typen verstehen

Die Tabelle **Seiten** listet jede Seite auf Ihrer Website zusammen mit ihrem Status auf:

- **Erstellt** -- Seiten, die automatisch vom System basierend auf den Daten Ihrer Kirche erstellt wurden (z.B. eine Gruppen-Seite oder Predigten-Seite). Diese Seiten aktualisieren sich selbst, wenn sich Ihre Daten ändern.
- **Benutzerdefiniert** -- Seiten, die Sie selbst mit Ihrem eigenen Inhalt und Layout erstellt haben.

Sie können jede Auto-generierte Seite in eine benutzerdefinierte Seite konvertieren, wenn Sie volle Kontrolle über ihren Inhalt und Design möchten.

## Seiten hinzufügen und bearbeiten

1. Klicken Sie auf die Schaltfläche **Seite hinzufügen** in der oberen rechten Ecke der Seiten-Tabelle.
2. Wählen Sie einen Seiten-Typ (leer oder eine Vorlage) und geben Sie ihm einen Namen.
3. Klicken Sie auf **Bearbeiten** neben jeder Seite, um den Seiten-Builder zu öffnen, wo Sie Abschnitte, Text, Bilder und andere Elemente hinzufügen können.
4. Klicken Sie auf **Seiten-Einstellungen**, um den Seiten-Titel, den URL-Pfad und andere Metadaten zu aktualisieren.
5. Verwenden Sie die Schaltfläche **Vorschau**, um Ihre Seite in einem neuen Fenster zu öffnen und zu sehen, wie sie für Besucher aussieht.

:::tip
Für Ihre Homepage setzen Sie den URL-Pfad auf einfach `/`. Für alle anderen Seiten verwenden Sie einen beschreibenden Pfad wie `/about` oder `/contact`.
:::

## Navigation verwalten

Die linke Seitenleiste der Website-Seiten-Ansicht zeigt Ihre Navigations-Links. Diese Links steuern das Menü, das Besucher auf Ihrer Website sehen.

1. Klicken Sie auf **Hinzufügen**, um einen neuen Navigations-Link zu erstellen. Sie können ihn auf jede Seite auf Ihrer Website oder auf eine externe URL verweisen.
2. Um Links neu zu ordnen, ziehen Sie sie per Drag-and-Drop in die gewünschte Reihenfolge. Sie können auch Links unter einem übergeordneten Element verschachteln, um Dropdown-Menüs zu erstellen.
3. Klicken Sie auf das **Bearbeiten**-Symbol neben einem Link, um sein Label, URL oder Position zu ändern.
4. Um einen Link aus der Navigation zu entfernen, klicken Sie auf das **Löschen**-Symbol.

:::info
Das Entfernen eines Navigations-Links löscht die Seite selbst nicht. Die Seite existiert immer noch und kann über ihre URL direkt aufgerufen werden — sie wird einfach nicht im Menü angezeigt.
:::

## Tipps zum Organisieren Ihrer Website

- Halten Sie Ihre Top-Level-Navigation auf fünf oder sechs Elemente, damit Besucher Dinge schnell finden können.
- Verwenden Sie verschachtelte Links für verwandte Unter-Seiten (z.B. ein "Über"-Dropdown mit "Unser Team", "Überzeugungen" und "Geschichte").
- Überprüfen Sie Ihre Navigation auf Mobil, indem Sie auf **Mobile-Vorschau** klicken, um sicherzustellen, dass sie auf kleineren Bildschirmen gut funktioniert.
- Geben Sie Seiten klare, beschreibende Namen, die Besuchern helfen zu verstehen, was sie finden werden.

:::tip
Sie können [Formulare](../forms/creating-forms.md) zu Ihren Seiten hinzufügen, um Registrierungen, Gebetsanfragen oder andere Informationen von Besuchern zu sammeln.
:::

## Bild-Lightbox

Wenn Besucher auf ein Bild auf Ihrer Website klicken, öffnet es sich in einer Vollbild-Lightbox-Überlagerung. Dies ermöglicht es Menschen, Fotos in einer größeren Größe anzuzeigen, ohne die Seite zu verlassen. Keine Konfiguration ist erforderlich — die Lightbox ist automatisch für Bilder in Ihrem Seiten-Inhalt aktiviert.
