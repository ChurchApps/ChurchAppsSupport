# Seiten verwalten

<div class="article-intro">

Die Website-Seitenansicht ist Ihr zentraler Ort zum Erstellen, Bearbeiten und Organisieren aller Seiten auf Ihrer Kirchenwebsite. Sie können sowohl Ihren Seiteninhalt als auch die Navigation Ihrer Website über einen einzigen Bildschirm verwalten.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Führen Sie die [Anfangseinrichtung](initial-setup) durch, um Ihre Domain und grundlegende Website-Einstellungen zu konfigurieren
- Haben Sie Ihren Inhalt und Ihre Bilder bereit. Verwenden Sie zunächst den [Dateien](files)-Manager, um Mediendateien hochzuladen.

</div>

## Seitentypen verstehen

Die Tabelle **Seiten** listet jede Seite auf Ihrer Website zusammen mit ihrem Status auf:

- **Generiert** – Seiten, die automatisch vom System erstellt wurden, basierend auf den Daten Ihrer Kirche (zum Beispiel eine Gruppenseite oder Predigenseite). Diese Seiten aktualisieren sich selbst, wenn Ihre Daten sich ändern.
- **Benutzerdefiniert** – Seiten, die Sie selbst mit Ihrem eigenen Inhalt und Layout erstellt haben.

Sie können jede automatisch generierte Seite in eine benutzerdefinierte Seite umwandeln, wenn Sie vollständige Kontrolle über ihren Inhalt und Design möchten.

## Seiten hinzufügen und bearbeiten

1. Klicken Sie auf die Schaltfläche **Seite hinzufügen** in der oberen rechten Ecke der Tabelle „Seiten".
2. Wählen Sie einen Seitentyp (leer oder eine Vorlage) und geben Sie ihm einen Namen.
3. Klicken Sie auf **Bearbeiten** neben einer beliebigen Seite, um den [Seiten-Editor](page-editor) zu öffnen, wo Sie Abschnitte, Text, Bilder und andere Elemente hinzufügen können.
4. Klicken Sie auf **Seiteneinstellungen**, um den Seitentitel, den URL-Pfad und andere Metadaten zu aktualisieren.
5. Verwenden Sie die Schaltfläche **Vorschau**, um Ihre Seite in einem neuen Fenster zu öffnen und zu sehen, wie sie für Besucher aussieht.

:::tip
Für Ihre Startseite setzen Sie den URL-Pfad einfach auf `/`. Verwenden Sie für alle anderen Seiten einen beschreibenden Pfad wie `/about` oder `/contact`.
:::

## Navigation verwalten

Die linke Seitenleiste der Website-Seitenansicht zeigt Ihre Navigationslinks. Diese Links steuern das Menü, das Besucher auf Ihrer Website sehen.

1. Klicken Sie auf **Hinzufügen**, um einen neuen Navigationslink zu erstellen. Sie können ihn auf eine beliebige Seite Ihrer Website oder auf eine externe URL verweisen.
2. Um Links neu anzuordnen, ziehen Sie sie per Drag & Drop in die gewünschte Reihenfolge. Sie können Links auch unter einem übergeordneten Element verschachteln, um Dropdown-Menüs zu erstellen.
3. Klicken Sie auf das **Bearbeiten**-Symbol neben einem Link, um sein Etikett, seine URL oder Position zu ändern.
4. Um einen Link aus der Navigation zu entfernen, klicken Sie auf das **Löschen**-Symbol.

:::info
Das Entfernen eines Navigationslinks löscht die Seite selbst nicht. Die Seite existiert immer noch und kann direkt über ihre URL aufgerufen werden – sie wird einfach nicht im Menü angezeigt.
:::

## Tipps zum Organisieren Ihrer Website

- Halten Sie Ihre Navigation auf der obersten Ebene auf etwa fünf oder sechs Elemente, damit Besucher Dinge schnell finden können.
- Verwenden Sie verschachtelte Links für verwandte Unterseiten (zum Beispiel ein „Info"-Dropdown mit „Unser Team", „Unsere Überzeugungen" und „Geschichte").
- Überprüfen Sie Ihre Navigation auf Mobilgeräten, indem Sie auf **Handy-Vorschau** klicken, um sicherzustellen, dass sie auf kleineren Bildschirmen gut funktioniert.
- Geben Sie Seiten klare, beschreibende Namen, die Besuchern helfen zu verstehen, was sie finden werden.

:::tip
Sie können [Formulare](../forms/creating-forms.md) zu Ihren Seiten hinzufügen, um Registrierungen, Gebetsanfragen oder andere Informationen von Besuchern zu sammeln.
:::

## Von einer Website-Vorlage starten

Wenn Sie Ihre Website von Grund auf aufbauen, können Sie sie mit einer **Website-Vorlage** starten, anstatt Seiten einzeln zu erstellen. Eine Website-Vorlage erstellt eine Reihe von vordefinierten Seiten – Startseite, Info, Verbinden, Spenden und andere – mit Platzhalterinhalten und bereits verdrahteten Navigationslinks.

1. Klicken Sie auf dem Bildschirm „Seiten" auf die Schaltfläche **Website-Vorlagen** (neben der Schaltfläche **Seite hinzufügen**).
2. Durchsuchen Sie die verfügbaren Vorlagen und klicken Sie auf eine, um ihre Seitenstruktur in der Vorschau zu sehen.
3. Wenn Sie eine gefunden haben, die Ihnen gefällt, klicken Sie auf **Vorlage anwenden**.
4. Seiten, die noch nicht vorhanden sind, werden erstellt und zu Ihrer Navigation hinzugefügt. Vorhandene Seiten bleiben unverändert.

Öffnen Sie nach dem Anwenden einer Vorlage jede Seite im [Seiten-Editor](page-editor), um den Platzhaltertext und Bilder durch den echten Inhalt Ihrer Kirche zu ersetzen.

:::info
Website-Vorlagen erstellen die Seitenstruktur und Navigation. Sie überschreiben nicht das Farbschema oder die Schriftarten Ihrer Website – diese werden durch [Erscheinungsbild](appearance) gesteuert.
:::

## Bild-Lightbox

Wenn Besucher auf ein Bild auf Ihrer Website klicken, wird es in einem Lightbox-Overlay im Vollbildmodus geöffnet. Dies ermöglicht es Personen, Fotos in einer größeren Größe anzuzeigen, ohne die Seite zu verlassen. Es ist keine Konfiguration erforderlich – die Lightbox wird automatisch für Bilder in Ihrem Seiteninhalt aktiviert.
