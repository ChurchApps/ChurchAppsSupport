# Dateien

<div class="article-intro">

Die Seite Dateien ist das zentrale Verzeichnis für alle Medien und Dokumente, die auf Ihrer Gemeinde-Website verwendet werden. Von hier aus können Sie Bilder, Dokumente und Mediendateien hochladen, organisieren, durchsuchen und verwalten.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Schließen Sie die [Ersteinrichtung](initial-setup) für Ihre Website ab
- Optimieren Sie Bilder vor dem Hochladen für schnellere Seitenladezeiten
- Bereiten Sie Ihre Dateien in gängigen Webformaten vor (PNG, JPG, PDF usw.)

</div>

## Dateien hochladen

1. Navigieren Sie zum Bereich **Website** in B1 Admin und klicken Sie auf den Tab **Dateien**.
2. Ziehen Sie Dateien per Drag & Drop direkt auf den Upload-Bereich oder klicken Sie auf die Upload-Schaltfläche, um Ihren Computer zu durchsuchen.
3. Ihre Dateien werden hochgeladen und erscheinen sofort in der Dateibibliothek.

:::tip
Für beste Ergebnisse optimieren Sie Bilder vor dem Hochladen. Kleinere Dateigrößen helfen Ihrer Website, schneller für Besucher zu laden. Tools wie TinyPNG oder Squoosh können Bilder ohne merklichen Qualitätsverlust komprimieren.
:::

## Mit Ordnern organisieren

1. Erstellen Sie Ordner, um zusammengehörige Dateien zu gruppieren -- zum Beispiel "Predigtgrafiken", "Veranstaltungsbanner" oder "Mitarbeiterfotos".
2. Verschieben Sie Dateien zwischen Ordnern, um Ihre Bibliothek beim Wachsen übersichtlich zu halten.
3. Verwenden Sie klare, beschreibende Ordnernamen, damit Ihr Team Dateien schnell finden kann.

## Durchsuchen und Suchen

- Scrollen Sie durch die Dateibibliothek, um Miniaturansichten aller hochgeladenen Dateien zu sehen.
- Verwenden Sie die **Suchfunktion**, um Dateien nach Namen zu finden, wenn Ihre Bibliothek groß wird.
- Klicken Sie auf eine beliebige Datei, um ihre Details anzuzeigen, einschließlich Dateiname, Größe und URL.

## Dateien auf Ihrer Website verwenden

1. Klicken Sie auf eine Datei, um ihre Details anzuzeigen.
2. **Kopieren Sie die Datei-URL**, um sie auf Ihren Website-Seiten zu verwenden. Sie können diese URL in Bildelemente, Download-Links oder überall dort einfügen, wo eine Datei-URL akzeptiert wird.
3. Hier hochgeladene Dateien stehen für die Verwendung auf allen Ihren Website-Seiten und Inhaltsblöcken zur Verfügung.

:::info
Beim Erstellen von Seiten unter [Seiten verwalten](managing-pages) können Sie jede hier hochgeladene Datei über ihre URL referenzieren. So bleiben alle Ihre Medien an einem zentralen Ort organisiert.
:::

## Dateien löschen

1. Wählen Sie die Datei aus, die Sie entfernen möchten.
2. Klicken Sie auf die Schaltfläche **Löschen**, um sie dauerhaft aus Ihrer Bibliothek zu entfernen.
3. Stellen Sie sicher, dass die Datei derzeit auf keiner Seite verwendet wird, bevor Sie sie löschen, da dies alle Links oder Bilder, die sie referenzieren, beschädigt.

:::warning
Gelöschte Dateien können nicht wiederhergestellt werden. Überprüfen Sie sorgfältig, ob eine Datei auf keiner Seite mehr verwendet wird, bevor Sie sie entfernen.
:::
