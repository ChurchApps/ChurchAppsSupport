# Gruppenmitglieder

<div class="article-intro">

Sobald Sie eine Gruppe erstellt haben, ist der nächste Schritt das Hinzufügen von Mitgliedern. Von der Detailseite einer Gruppe aus können Sie nach Personen suchen, sie zur Gruppe hinzufügen, Leiter bestimmen, Nachrichten senden und die Mitgliederliste exportieren. Die Verwaltung der Gruppenmitgliedschaft ist unerlässlich für die Koordination von Kleingruppen, Ausschüssen und Klassen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen mindestens eine in B1 Admin eingerichtete Gruppe. Siehe [Gruppen erstellen](creating-groups.md), falls Sie noch keine erstellt haben.
- Die Personen, die Sie hinzufügen möchten, müssen bereits in Ihrem [Personen](../people/adding-people.md)-Verzeichnis vorhanden sein.

</div>

## Mitglieder zu einer Gruppe hinzufügen

1. Navigieren Sie zur Seite **Gruppen** und klicken Sie auf die Gruppe, die Sie verwalten möchten.
2. Klicken Sie auf den Tab **Mitglieder**.
3. Geben Sie im Suchfeld den Namen der Person ein, die Sie hinzufügen möchten.
4. Klicken Sie auf **Hinzufügen** neben dem Namen der Person in den Suchergebnissen.
5. Die Person erscheint nun in der Gruppenmitgliederliste.

:::tip
Lassen Sie das Suchfeld leer und klicken Sie auf **Suchen**, um Ihr gesamtes Verzeichnis zu durchblättern. Dies ist hilfreich, wenn Sie sich nicht sicher über die genaue Schreibweise eines Namens sind.
:::

## Gruppenleiter bestimmen

Gruppenleiter haben besondere Berechtigungen -- sie können den [Gruppenkalender](group-calendar.md) bearbeiten, Veranstaltungen verwalten und die Gruppe koordinieren.

1. Finden Sie in der Gruppenmitgliederliste die Person, die Sie zum Leiter machen möchten.
2. Klicken Sie auf das **grüne Schlüsselsymbol** neben deren Namen.
3. Die Person ist nun als Gruppenleiter bestimmt.

Um den Leiterstatus zu entfernen, klicken Sie erneut auf das grüne Schlüsselsymbol.

:::info
Jedes Gruppenmitglied kann den Gruppenkalender und die Veranstaltungen einsehen, aber nur Leiter können Kalenderveranstaltungen hinzufügen oder bearbeiten.
:::

## Nachrichten an Gruppenmitglieder senden

Sie können mit allen Mitgliedern einer Gruppe direkt aus B1 Admin kommunizieren:

1. Suchen Sie auf der Gruppen-Detailseite den Nachrichtenbereich.
2. Geben Sie Ihre Nachricht in das Textfeld ein.
3. Klicken Sie auf **Senden**.

Ihre Nachricht wird an alle Mitglieder der Gruppe zugestellt.

## E-Mails an Gruppenmitglieder senden

Sie können formatierte E-Mails an alle Mitglieder einer Gruppe senden:

1. Klicken Sie auf der Gruppen-Detailseite auf das **E-Mail-Symbol**.
2. Der Dialog zum Senden einer E-Mail öffnet sich und zeigt an, wie viele Mitglieder die E-Mail erhalten werden und wie viele keine E-Mail-Adresse hinterlegt haben.
3. Wählen Sie optional eine **E-Mail-Vorlage** aus dem Dropdown-Menü oder verfassen Sie eine Nachricht von Grund auf. Klicken Sie auf **Vorlagen verwalten**, um Vorlagen zu erstellen oder zu bearbeiten.
4. Geben Sie eine **Betreffzeile** ein. Sie können Seriendruckfelder einfügen, indem Sie auf die Feldchips klicken: `{{firstName}}`, `{{lastName}}`, `{{displayName}}`, `{{email}}`, `{{churchName}}`.
5. Verfassen Sie den **E-Mail-Text** mit dem HTML-Editor. Dieselben Seriendruckfelder stehen hier zur Verfügung.
6. Klicken Sie auf **Senden**.
7. Eine Zusammenfassung zeigt, wie viele E-Mails erfolgreich gesendet wurden und wie viele Mitglieder übersprungen wurden (keine E-Mail hinterlegt).

:::tip
Erstellen Sie wiederverwendbare E-Mail-Vorlagen für wiederkehrende Kommunikation wie wöchentliche Updates, Veranstaltungsankündigungen oder Gebetsanliegen. Vorlagen sparen Zeit und sorgen für einheitliche Kommunikation.
:::

## Gruppendaten exportieren

So laden Sie die Gruppenmitgliederliste als Datei herunter:

1. Klicken Sie auf der Gruppen-Detailseite auf das **Download-Symbol**.
2. Eine CSV-Datei mit den Mitgliederinformationen der Gruppe wird auf Ihren Computer heruntergeladen.

Dies ist nützlich für das Erstellen gedruckter Listen, den Import von Daten in andere Tools oder die Aufbewahrung von Offline-Aufzeichnungen. Für weitere Exportoptionen siehe [Daten exportieren](../people/exporting-data.md).

## Mitglieder entfernen

Um jemanden aus einer Gruppe zu entfernen, suchen Sie seinen Namen in der Mitgliederliste und klicken Sie auf die Schaltfläche **Entfernen** neben dem Eintrag.

:::info
Das Entfernen einer Person aus einer Gruppe löscht sie nicht aus Ihrem Gemeindeverzeichnis. Sie erscheint weiterhin im Bereich [Personen](../people/adding-people.md) und kann jederzeit wieder zur Gruppe hinzugefügt werden.
:::
