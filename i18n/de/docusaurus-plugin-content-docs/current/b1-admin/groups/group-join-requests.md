# Gruppenbeitrittsanfragen

<div class="article-intro">

Wenn eine Gruppe mit einer genehmigungsbasierten Beitrittsrichtlinie konfiguriert ist, können Personen Anfragen zum Beitritt einreichen. Gruppenleiter und Administratoren prüfen diese Anfragen und genehmigen oder lehnen sie ab. Dies gibt Ihrer Gemeinde Kontrolle über die Gruppenmitgliedschaft, während es für Personen einfach ist, ihr Interesse am Beitritt auszudrücken.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen die Berechtigung zur Verwaltung von Gruppen, oder Sie müssen Leiter der spezifischen Gruppe sein. Siehe [Rollen & Berechtigungen](../people/roles-permissions.md) für Details.
- Die Gruppe muss ihre Beitrittsrichtlinie auf **Anfrage** (Genehmigung erforderlich) gesetzt haben. Siehe [Gruppen erstellen](./creating-groups.md) für die Konfiguration von Beitrittsrichtlinien.

</div>

## Beitrittsrichtlinien verstehen

Gruppen können drei verschiedene Beitrittsrichtlinien haben:

- **Offen** -- Jeder kann sofort ohne Genehmigung beitreten
- **Anfrage** -- Personen reichen eine Beitrittsanfrage ein, die eine Genehmigung erfordert
- **Geschlossen** -- Niemand kann einen Beitritt beantragen (Mitglieder müssen manuell hinzugefügt werden)

Wenn eine Gruppe die **Anfrage**-Richtlinie verwendet, durchlaufen alle Beitrittsversuche den auf dieser Seite beschriebenen Genehmigungsworkflow.

## Ausstehende Anfragen anzeigen

### Für Gruppenleiter

1. Navigieren Sie zu **Gruppen** in B1 Admin
2. Klicken Sie auf den Gruppennamen
3. Klicken Sie auf die Registerkarte **Beitrittsanfragen**
4. Sie sehen alle ausstehenden Anfragen für diese Gruppe

### Für Administratoren

Administratoren mit Gruppenverwaltungsberechtigungen können ausstehende Anfragen über alle Gruppen hinweg anzeigen:

1. Navigieren Sie zu **Gruppen** in B1 Admin
2. Suchen Sie nach einem Benachrichtigungs-Badge oder einer Anzeige für ausstehende Anfragen
3. Klicken Sie, um alle ausstehenden Anfragen gemeindeweit anzuzeigen

## Eine Beitrittsanfrage prüfen

Jede Beitrittsanfrage zeigt:

- **Name und Foto der Person** -- Die Person, die den Beitritt beantragt
- **Optionale Nachricht** -- Eine persönliche Nachricht, die erklärt, warum sie beitreten möchten (falls angegeben)
- **Anfragedatum** -- Wann die Anfrage eingereicht wurde

Um eine Anfrage zu prüfen:

1. Lesen Sie die Nachricht der Person, falls sie eine angegeben hat
2. Klicken Sie auf den Namen der Person, um ihr Profil anzuzeigen, falls erforderlich
3. Entscheiden Sie, ob Sie genehmigen oder ablehnen möchten

## Eine Anfrage genehmigen

1. Klicken Sie bei der Beitrittsanfrage auf **Genehmigen**
2. Die Person wird sofort als Mitglied zur Gruppe hinzugefügt
3. Der Antragsteller erhält eine Benachrichtigung, dass seine Anfrage genehmigt wurde
4. Die Anfrage wird im System als genehmigt markiert

:::tip
Wenn Sie eine Anfrage genehmigen, wird die Person ein reguläres Gruppenmitglied. Sie können sie später bei Bedarf von der Seite [Gruppenmitglieder](./group-members.md) zum Gruppenleiter befördern.
:::

## Eine Anfrage ablehnen

1. Klicken Sie bei der Beitrittsanfrage auf **Ablehnen**
2. Geben Sie optional einen Grund für die Ablehnung an (bis zu 500 Zeichen)
3. Klicken Sie auf **Bestätigen**
4. Der Antragsteller erhält eine Benachrichtigung mit Ihrem Ablehnungsgrund (falls angegeben)
5. Die Anfrage wird als abgelehnt markiert

:::info
Die Angabe eines Ablehnungsgrundes hilft der Person zu verstehen, warum ihre Anfrage nicht genehmigt wurde, und kann sie ermutigen, es später erneut zu versuchen oder andere Gruppen zu erkunden.
:::

## Benachrichtigungen

Das Beitrittsanfragen-System sendet automatisch Benachrichtigungen:

- **Wenn eine Anfrage eingereicht wird** -- Alle Gruppenleiter erhalten eine Benachrichtigung
- **Wenn eine Anfrage genehmigt wird** -- Der Antragsteller erhält eine Bestätigung
- **Wenn eine Anfrage abgelehnt wird** -- Der Antragsteller erhält eine Benachrichtigung mit einem eventuellen Ablehnungsgrund

Benachrichtigungen erscheinen im Benachrichtigungscenter des Benutzers auf B1.church und in der mobilen App.

## Anfragen von der Mitgliederseite verwalten

Personen können ihre eigenen Beitrittsanfragen von B1.church aus verwalten:

- Den Status ihrer ausstehenden Anfragen auf der Gruppendetailseite anzeigen
- Eine ausstehende Anfrage abbrechen, wenn sie ihre Meinung ändern
- Sehen, ob ihre Anfrage genehmigt oder abgelehnt wurde

## Best Practices

- **Schnell antworten** -- Versuchen Sie, Anfragen innerhalb von 24-48 Stunden zu prüfen, damit Personen nicht warten müssen
- **Seien Sie klar bei Ablehnungsgründen** -- Helfen Sie Personen, die nächsten Schritte oder alternative Optionen zu verstehen
- **Profile überprüfen** -- Überprüfen Sie das Profil der Person, um zu sehen, ob sie gut zur Gruppe passt
- **Erwartungen kommunizieren** -- Stellen Sie sicher, dass Ihre Gruppenbeschreibung klar angibt, für wen die Gruppe ist
