# Gruppen-Beitrittanfragen

<div class="article-intro">

Wenn eine Gruppe mit einer genehmigungsgestützten Beitrittslrichtlinie konfiguriert ist, können Personen Anfragen zum Beitreten einreichen. Gruppenleitern und Administratoren überprüfen diese Anfragen und genehmigen oder lehnen sie ab. Dies gibt Ihrer Kirchengemeinde Kontrolle über die Gruppenmitgliedschaft, macht es aber einfach, dass Menschen ihr Interesse am Beitreten ausdrücken können.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen die Berechtigung zur Verwaltung von Gruppen oder müssen ein Leiter der spezifischen Gruppe sein. Siehe [Rollen & Berechtigungen](../people/roles-permissions.md) für Details.
- Die Gruppe muss ihre Beitrittslrichtlinie auf **Anfrage** (Genehmigung erforderlich) eingestellt haben. Siehe [Gruppen erstellen](./creating-groups.md) zum Konfigurieren von Beitrittslrichtlinien.

</div>

## Beitrittslrichtlinien verstehen

Gruppen können drei verschiedene Beitrittslrichtlinien haben:

- **Offen** -- Jeder kann sofort ohne Genehmigung beitreten
- **Anfrage** -- Personen reichen eine Beitrittsanfrage ein, die Genehmigung benötigt
- **Geschlossen** -- Niemand kann beantragen, beizutreten (Mitglieder müssen manuell hinzugefügt werden)

Wenn eine Gruppe die Richtlinie **Anfrage** verwendet, erfolgen alle Beitrittversuche über den auf dieser Seite beschriebenen Genehmigungsworkflow.

## Ausstehende Anfragen anzeigen

### Für Gruppenleitern

1. Navigieren Sie zu **Gruppen** in B1 Admin
2. Klicken Sie auf den Gruppennamen
3. Klicken Sie auf die Registerkarte **Beitrittanfragen**
4. Sie sehen alle ausstehenden Anfragen für diese Gruppe

### Für Administratoren

Administratoren mit Gruppenverwaltungsberechtigungen können ausstehende Anfragen über alle Gruppen hinweg anzeigen:

1. Navigieren Sie zu **Gruppen** in B1 Admin
2. Suchen Sie nach einem Benachrichtigungsabzeichen oder einem Indikator für ausstehende Anfragen
3. Klicken Sie, um alle ausstehenden Anfragen in der Kirchengemeinde anzuzeigen

## Eine Beitrittsanfrage überprüfen

Jede Beitrittsanfrage zeigt:

- **Name und Foto der Person** -- Die Person, die beitreten möchte
- **Optionale Nachricht** -- Eine persönliche Nachricht, die erklärt, warum sie beitreten möchte (falls vorhanden)
- **Anfragedatum** -- Wann die Anfrage eingereicht wurde

Um eine Anfrage zu überprüfen:

1. Lesen Sie die Nachricht der Person, falls vorhanden
2. Klicken Sie auf den Namen der Person, um ihr Profil anzusehen, falls nötig
3. Entscheiden Sie, ob Sie genehmigen oder ablehnen

## Eine Anfrage genehmigen

1. Klicken Sie auf die Bittrittsanfrage **Genehmigen**
2. Die Person wird sofort als Mitglied der Gruppe hinzugefügt
3. Die Person, die die Anfrage gestellt hat, erhält eine Benachrichtigung, dass ihre Anfrage genehmigt wurde
4. Die Anfrage wird im System als genehmigt gekennzeichnet

:::tip
Wenn Sie eine Anfrage genehmigen, wird die Person ein normales Gruppenmitglied. Sie können sie später zum Gruppenleiter befördern, falls nötig, von der [Gruppenmitglieder](./group-members.md)-Seite aus.
:::

## Eine Anfrage ablehnen

1. Klicken Sie auf die Bittrittsanfrage **Ablehnen**
2. Geben Sie optional einen Grund für die Ablehnung ein (bis zu 500 Zeichen)
3. Klicken Sie auf **Bestätigen**
4. Die Person, die die Anfrage gestellt hat, erhält eine Benachrichtigung mit Ihrem Ablehnungsgrund (falls vorhanden)
5. Die Anfrage wird im System als abgelehnt gekennzeichnet

:::info
Ein Ablehnungsgrund hilft der Person zu verstehen, warum ihre Anfrage nicht genehmigt wurde, und kann sie ermutigen, es später erneut zu versuchen oder andere Gruppen zu erkunden.
:::

## Benachrichtigungen

Das System für Beitrittanfragen sendet automatisch Benachrichtigungen:

- **Wenn eine Anfrage eingereicht wird** -- Alle Gruppenleitern erhalten eine Benachrichtigung
- **Wenn eine Anfrage genehmigt wird** -- Die Person, die die Anfrage gestellt hat, erhält eine Bestätigung
- **Wenn eine Anfrage abgelehnt wird** -- Die Person, die die Anfrage gestellt hat, erhält eine Benachrichtigung mit Ihrem Ablehnungsgrund

Benachrichtigungen werden im Benachrichtigungszentrum des Benutzers auf B1.church und in der mobilen App angezeigt.

## Anfragen von der Seite des Mitglieds verwalten

Personen können ihre eigenen Beitrittanfragen von B1.church aus verwalten:

- Sehen Sie den Status ihrer ausstehenden Anfragen auf der Gruppenseite
- Stornieren Sie eine ausstehende Anfrage, wenn Sie sich anders überlegen
- Sehen Sie, ob ihre Anfrage genehmigt oder abgelehnt wurde

## Best Practices

- **Schnell reagieren** -- Versuchen Sie, Anfragen innerhalb von 24-48 Stunden zu überprüfen, damit Personen nicht warten müssen
- **Klare Ablehnungsgründe** -- Helfen Sie den Menschen, nächste Schritte oder alternative Optionen zu verstehen
- **Profile überprüfen** -- Überprüfen Sie das Profil der Person, um zu sehen, ob sie gut zur Gruppe passt
- **Erwartungen kommunizieren** -- Stellen Sie sicher, dass Ihre Gruppenbeschreibung klar angibt, für wen die Gruppe bestimmt ist
