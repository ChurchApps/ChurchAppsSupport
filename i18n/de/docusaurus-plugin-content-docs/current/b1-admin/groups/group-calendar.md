# Gruppenkalender

<div class="article-intro">

Jede Gruppe in B1 kann einen eigenen Ereigniskalender haben. Gruppenleiter nutzen ihn, um Treffen, Aktivitäten und wiederkehrende Ereignisse zu planen, während alle Gruppenmitglieder bevorstehende Ereignisse einsehen und sich an Gesprächen darüber beteiligen können. Der Kalender hält Ihre Gruppe organisiert und stellt sicher, dass alle über anstehende Termine informiert bleiben.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen eine in B1 Admin erstellte Gruppe mit mindestens einem Mitglied. Siehe [Gruppen erstellen](creating-groups.md) und [Gruppenmitglieder](group-members.md).
- Um Kalendertermine hinzuzufügen oder zu bearbeiten, müssen Sie als Gruppenleiter bestimmt sein. Anweisungen finden Sie unter [Gruppenleiter bestimmen](group-members.md).

</div>

## Auf den Gruppenkalender zugreifen

Der Gruppenkalender ist über die mitgliederorientierte App B1.church verfügbar:

1. Klicken Sie in B1 Admin oben rechts auf dem Bildschirm auf das **Dropdown-Menü**.
2. Wählen Sie **App wechseln**.
3. Wählen Sie **B1.church**.
4. Klicken Sie in der linken Seitenleiste auf **Gruppen**.
5. Wählen Sie Ihre Gruppe aus der Liste.
6. Klicken Sie links auf dem Bildschirm auf **Kalender**.

:::info
Der Gruppenkalender wird über B1.church (die Mitglieder-App) verwaltet, nicht über B1 Admin. Gruppenleiter greifen darauf zu, indem sie vom Admin-Dashboard aus die App wechseln.
:::

## Ein Ereignis hinzufügen

1. Klicken Sie im Gruppenkalender auf **Ereignis hinzufügen**.
2. Geben Sie einen **Titel** für das Ereignis ein.
3. Schreiben Sie eine kurze **Beschreibung**, worum es bei dem Ereignis geht.
4. Legen Sie die **Startzeit** und **Endzeit** fest.
5. Klicken Sie auf **Speichern**, um das Ereignis zum Kalender hinzuzufügen.

## Wiederkehrende Ereignisse einrichten

Wenn sich Ihre Gruppe regelmäßig trifft, können Sie Ereignisse so einstellen, dass sie sich automatisch wiederholen:

1. Aktivieren Sie beim Erstellen oder Bearbeiten eines Ereignisses das Kontrollkästchen **Wiederkehrend**.
2. Wählen Sie, wie sich das Ereignis wiederholen soll:
   - **Intervall** -- Legen Sie eine Anzahl von Tagen zwischen den Vorkommen fest. Die Eingabe von „7" wiederholt das Ereignis beispielsweise alle 7 Tage.
   - **Häufigkeit** -- Wählen Sie eine Wiederholung nach **Tag**, **Woche** oder **Monat**.
3. Speichern Sie das Ereignis. Es erscheint nun bei jedem geplanten Vorkommen im Kalender.

:::tip
Für eine Gruppe, die sich jeden Samstag trifft, stellen Sie die Häufigkeit auf „Woche" und wählen Sie Samstag. Das Ereignis wird künftig automatisch im Kalender angezeigt.
:::

## Massenhafte Erstellung von Ereignissen in B1 Admin

Die Gruppenseite in B1 Admin hat außerdem eine Registerkarte **Kalender**, auf der Mitarbeiter eine ganze Saison von Treffen auf einmal planen können:

1. Öffnen Sie die Gruppe in B1 Admin und wählen Sie die Registerkarte **Kalender**.
2. Klicken Sie auf **Ereignisse hinzufügen**, um den Dialog **Wiederkehrende Ereignisse hinzufügen** zu öffnen.
3. Wählen Sie den Zeitplan (wöchentlich, alle 2 Wochen, monatlich oder einmalig), die Daten für **Erstes Treffen** und **Letztes Treffen** und optional **Feiertage überspringen**.
4. Überprüfen Sie die Vorschau der Treffdaten und deaktivieren Sie alle, die übersprungen werden sollen.
5. Lassen Sie **RSVPs zulassen** aktiviert, wenn Mitglieder auf diese Ereignisse antworten können sollen (siehe unten), und speichern Sie dann.

## Ereignis-RSVPs

Gruppenereignisse können RSVPs sammeln -- ein **Ja**, **Nein** oder **Vielleicht** für jedes Vorkommen des Ereignisses.

- **RSVPs zulassen** ist bei über den Massenhinzufügen-Dialog erstellten Ereignissen standardmäßig aktiviert; deaktivieren Sie es, um Antworten für diese Ereignisse abzuschalten. Ereignisse mit deaktivierten RSVPs werden auf der Kalender-Registerkarte als **Deaktiviert** angezeigt.
- Auf der Gruppenkalender-Registerkarte von B1 Admin zeigt jedes Ereignis eine Antwortzusammenfassung (zum Beispiel „5 Ja · 2 Vielleicht · 1 Nein"). Klicken Sie darauf, um das Verzeichnis **RSVP-Antworten** zu öffnen, das auflistet, wer für dieses Vorkommen mit Ja, Vielleicht und Nein geantwortet hat.

:::tip
RSVPs geben Leitern eine Kopfzahl ohne die Formalität einer vollständigen Ereignisregistrierung. Für kostenpflichtige Anmeldungen oder kapazitätsgesteuerte Ereignisse verwenden Sie stattdessen die [Ereignisregistrierung](../calendars/creating-calendars.md#enabling-event-registration).
:::

## Den Kalender anzeigen

Der Gruppenkalender bietet mehrere Anzeigeoptionen:

- **Monat** -- Den gesamten Monat auf einen Blick sehen.
- **Woche** -- Sich auf die aktuelle Woche konzentrieren.
- **Tag** -- Einen einzelnen Tag im Detail anzeigen.
- **Agenda** -- Eine Liste bevorstehender Ereignisse in chronologischer Reihenfolge anzeigen.

Wechseln Sie mit den Schaltflächen oben im Kalender zwischen den Ansichten.

## Ereignisdetails und Gespräche

Klicken Sie auf ein Ereignis, um dessen Details anzuzeigen. In der Ereignisdetailansicht können:

- **Mitglieder und Leiter** Gespräche beginnen und Kommentare zum Ereignis hinterlassen.
- **Leiter** Ereignisdetails bearbeiten, Zeiten ändern oder das Ereignis löschen.

:::warning
Das Löschen eines Ereignisses entfernt es dauerhaft. Wenn Sie nur ein einzelnes Vorkommen eines wiederkehrenden Ereignisses absagen möchten, sollten Sie es stattdessen bearbeiten.
:::

## Den Kalender abonnieren

Sie können den Gruppenkalender mit Ihrem persönlichen Kalender synchronisieren:

1. Klicken Sie auf der Gruppenkalenderseite auf **Abonnieren**.
2. Folgen Sie den Anweisungen, um den Kalender zu Google Kalender, Apple Kalender, Outlook oder einer anderen Kalenderanwendung hinzuzufügen, die Abonnements unterstützt.

:::tip
Durch das Abonnieren bleibt Ihr persönlicher Zeitplan mit den Ereignissen Ihrer Gruppe synchron, ohne dass Sie B1 separat prüfen müssen. Alle Änderungen von Gruppenleitern erscheinen automatisch in Ihrem synchronisierten Kalender.
:::
