# Gruppenkalender

<div class="article-intro">

Jede Gruppe in B1 kann ihren eigenen Veranstaltungskalender haben. Gruppenleiter verwenden ihn, um Treffen, Aktivitäten und wiederkehrende Events zu planen, während alle Gruppenmitglieder kommende Events anzeigen und Unterhaltungen darüber führen können. Der Kalender hält deine Gruppe organisiert und stellt sicher, dass jeder über kommende Ereignisse informiert bleibt.

</div>

<div class="prereqs">
<h4>Bevor du beginnst</h4>

- Du benötigst eine in B1 Admin erstellte Gruppe mit mindestens einem Mitglied. Siehe [Gruppen erstellen](creating-groups.md) und [Gruppenmitglieder](group-members.md).
- Um Kalender-Events hinzuzufügen oder zu bearbeiten, musst du als Gruppenleiter bestimmt werden. Siehe [Designation von Gruppenleitern](group-members.md) für Anweisungen.

</div>

## Zugriff auf den Gruppenkalender

Der Gruppenkalender ist über die B1.church-App für Mitglieder verfügbar:

1. Klicke im B1 Admin oben rechts auf der Seite auf das **Dropdown-Menü**.
2. Wähle **App wechseln**.
3. Wähle **B1.church**.
4. Klicke auf **Gruppen** in der linken Seitenleiste.
5. Wähle deine Gruppe aus der Liste.
6. Klicke auf der linken Seite des Bildschirms auf **Kalender**.

:::info
Der Gruppenkalender wird über B1.church (die Mitglieder-App) statt B1 Admin verwaltet. Gruppenleiter greifen darauf zu, indem sie Apps vom Admin-Dashboard wechseln.
:::

## Einen Event hinzufügen

1. Klicke im Gruppenkalender auf **Event hinzufügen**.
2. Gib einen **Titel** für das Event ein.
3. Schreibe eine kurze **Beschreibung**, worum es bei dem Event geht.
4. Stelle die **Startzeit** und **Endzeit** ein.
5. Klicke auf **Speichern**, um das Event zum Kalender hinzuzufügen.

## Einrichtung von wiederkehrenden Events

Wenn deine Gruppe nach einem regelmäßigen Plan trifft, kannst du Events automatisch wiederholen lassen:

1. Aktiviere beim Erstellen oder Bearbeiten eines Events das Kontrollkästchen **Wiederkehrend**.
2. Wähle, wie das Event wiederholt werden soll:
   - **Intervall** – Stelle eine Anzahl von Tagen zwischen den Vorkommen ein. Wenn du beispielsweise "7" eingibst, wiederholt sich das Event alle 7 Tage.
   - **Häufigkeit** – Wähle eine Wiederholung nach **Tag**, **Woche** oder **Monat**.
3. Speichere das Event. Es wird nun im Kalender bei jedem geplanten Vorkommen angezeigt.

:::tip
Für eine Gruppe, die jeden Samstag zusammenkommt, stelle die Häufigkeit auf "Woche" ein und wähle Samstag. Das Event wird automatisch im Kalender vorwärts ausfüllen.
:::

## Massenerstellen von Events in B1 Admin

Die Seite der Gruppe in B1 Admin hat auch einen Reiter **Kalender**, in dem das Personal eine ganze Saison von Treffen auf einmal planen kann:

1. Öffne die Gruppe in B1 Admin und wähle den Reiter **Kalender**.
2. Klicke auf **Events hinzufügen**, um das Dialogfeld **Wiederkehrende Events hinzufügen** zu öffnen.
3. Wähle den Zeitplan (wöchentlich, alle 2 Wochen, monatlich oder einmalig), die Termine **Erstes Treffen** und **Letztes Treffen** und optional **Feiertage überspringen**.
4. Überprüfe die Vorschau der Treffendaten und deaktiviere alle, die du überspringen möchtest.
5. Lasse **RSVPs zulassen** aktiviert, wenn du möchtest, dass Mitglieder auf diese Events antworten können (siehe unten), speichere dann.

## Event RSVPs

Gruppenereignisse können RSVPs sammeln – ein **Ja**, **Nein** oder **Vielleicht** für jedes Vorkommen des Events.

- **RSVPs zulassen** ist standardmäßig aktiviert für Events, die über den Massenhinzufügen-Dialog erstellt werden; deaktiviere es, um die Antworten für diese Events auszuschalten. Events mit RSVPs deaktiviert zeigen als **Deaktiviert** auf dem Kalender-Reiter.
- Im Kalender-Reiter der B1 Admin-Gruppe zeigt jedes Event eine Antwort-Zusammenfassung (z. B. "5 ja · 2 vielleicht · 1 nein"). Klicke durch, um das Roster **RSVP-Antworten** zu öffnen, das auflistet, wer auf dieses Vorkommen ja, vielleicht und nein geantwortet hat.

:::tip
RSVPs geben Leitern eine Kopfzahl ohne die Formalität der vollständigen Event-Registrierung. Für bezahlte Anmeldungen oder Kapazitätsmanagement-Events, siehe [Event-Registrierung](../calendars/creating-calendars.md#enabling-event-registration) stattdessen.
:::

## Viewing the Calendar

Der Gruppenkalender bietet mehrere Anzeigeoptionen:

- **Monat** – Sehe den ganzen Monat auf einen Blick.
- **Woche** – Konzentriere dich auf die aktuelle Woche.
- **Tag** – Zeige einen einzelnen Tag im Detail an.
- **Agenda** – Sehe eine Liste kommender Events in chronologischer Reihenfolge.

Wechsle zwischen Ansichten mit den Buttons oben im Kalender.

## Event-Details und Unterhaltungen

Klicke auf ein beliebiges Event, um seine Details anzuzeigen. Aus der Event-Detailansicht:

- **Mitglieder und Leiter** können Unterhaltungen beginnen und Kommentare zum Event hinterlassen.
- **Leiter** können Event-Details bearbeiten, Zeiten ändern oder das Event löschen.

:::warning
Das Löschen eines Events entfernt es dauerhaft. Wenn du ein einzelnes Vorkommen eines wiederkehrenden Events absagen möchtest, erwäge stattdessen, es zu bearbeiten.
:::

## Abonnement des Kalenders

Du kannst den Gruppenkalender mit deinem persönlichen Kalender synchronisieren:

1. Klicke auf der Gruppenseite des Kalenders auf **Abonnieren**.
2. Folge den Aufforderungen, um den Kalender zu Google Kalender, Apple Kalender, Outlook oder einer anderen Kalenderanwendung hinzuzufügen, die Abonnements unterstützt.

:::tip
Das Abonnement hält deinen persönlichen Zeitplan mit den Events deiner Gruppe synchronisiert, ohne dass du B1 separat überprüfen musst. Alle Änderungen, die von Gruppenleitern vorgenommen werden, werden automatisch in deinem synchronisierten Kalender angezeigt.
:::
