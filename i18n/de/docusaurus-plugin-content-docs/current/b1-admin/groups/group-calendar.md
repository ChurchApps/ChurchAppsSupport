# Gruppenkalender

<div class="article-intro">

Jede Gruppe in B1 kann ihren eigenen Veranstaltungskalender haben. Gruppenleiter nutzen ihn, um Treffen, Aktivitäten und wiederkehrende Veranstaltungen zu planen, während alle Gruppenmitglieder bevorstehende Veranstaltungen einsehen und sich an Gesprächen darüber beteiligen können. Der Kalender hält Ihre Gruppe organisiert und stellt sicher, dass alle über anstehende Termine informiert sind.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen eine in B1 Admin erstellte Gruppe mit mindestens einem Mitglied. Siehe [Gruppen erstellen](creating-groups.md) und [Gruppenmitglieder](group-members.md).
- Um Kalenderveranstaltungen hinzuzufügen oder zu bearbeiten, müssen Sie als Gruppenleiter bestimmt sein. Siehe [Gruppenleiter bestimmen](group-members.md) für Anweisungen.

</div>

## Auf den Gruppenkalender zugreifen

Der Gruppenkalender ist über die B1.church-Mitglieder-App verfügbar:

1. Klicken Sie in B1 Admin auf das **Dropdown-Menü** in der oberen rechten Ecke Ihres Bildschirms.
2. Wählen Sie **App wechseln**.
3. Wählen Sie **B1.church**.
4. Klicken Sie in der linken Seitenleiste auf **Gruppen**.
5. Wählen Sie Ihre Gruppe aus der Liste.
6. Klicken Sie auf der linken Seite auf **Kalender**.

:::info
Der Gruppenkalender wird über B1.church (die Mitglieder-App) und nicht über B1 Admin verwaltet. Gruppenleiter greifen darauf zu, indem sie vom Admin-Dashboard aus die App wechseln.
:::

## Eine Veranstaltung hinzufügen

1. Klicken Sie im Gruppenkalender auf **Veranstaltung hinzufügen**.
2. Geben Sie einen **Titel** für die Veranstaltung ein.
3. Schreiben Sie eine kurze **Beschreibung**, worum es bei der Veranstaltung geht.
4. Legen Sie die **Startzeit** und **Endzeit** fest.
5. Klicken Sie auf **Speichern**, um die Veranstaltung zum Kalender hinzuzufügen.

## Wiederkehrende Veranstaltungen einrichten

Wenn sich Ihre Gruppe nach einem regelmäßigen Zeitplan trifft, können Sie Veranstaltungen automatisch wiederholen lassen:

1. Beim Erstellen oder Bearbeiten einer Veranstaltung aktivieren Sie das Kontrollkästchen **Wiederkehrend**.
2. Wählen Sie, wie die Veranstaltung wiederholt werden soll:
   - **Intervall** -- Legen Sie eine Anzahl von Tagen zwischen den Wiederholungen fest. Zum Beispiel wiederholt die Eingabe von „7" die Veranstaltung alle 7 Tage.
   - **Häufigkeit** -- Wählen Sie die Wiederholung nach **Tag**, **Woche** oder **Monat**.
3. Speichern Sie die Veranstaltung. Sie erscheint nun im Kalender zu jedem geplanten Termin.

:::tip
Für eine Gruppe, die sich jeden Samstag trifft, setzen Sie die Häufigkeit auf „Woche" und wählen Sie Samstag. Die Veranstaltung wird automatisch für die Zukunft im Kalender eingetragen.
:::

## Den Kalender anzeigen

Der Gruppenkalender bietet mehrere Ansichtsoptionen:

- **Monat** -- Den gesamten Monat auf einen Blick sehen.
- **Woche** -- Auf die aktuelle Woche fokussieren.
- **Tag** -- Einen einzelnen Tag im Detail anzeigen.
- **Agenda** -- Eine Liste bevorstehender Veranstaltungen in chronologischer Reihenfolge sehen.

Wechseln Sie zwischen den Ansichten über die Schaltflächen oben im Kalender.

## Veranstaltungsdetails und Gespräche

Klicken Sie auf eine beliebige Veranstaltung, um ihre Details anzuzeigen. In der Veranstaltungsdetailansicht:

- **Mitglieder und Leiter** können Gespräche starten und Kommentare zur Veranstaltung hinterlassen.
- **Leiter** können Veranstaltungsdetails bearbeiten, Zeiten ändern oder die Veranstaltung löschen.

:::warning
Das Löschen einer Veranstaltung entfernt sie dauerhaft. Wenn Sie einen einzelnen Termin einer wiederkehrenden Veranstaltung absagen müssen, erwägen Sie stattdessen, ihn zu bearbeiten.
:::

## Den Kalender abonnieren

Sie können den Gruppenkalender mit Ihrem persönlichen Kalender synchronisieren:

1. Klicken Sie auf der Gruppenkalenderseite auf **Abonnieren**.
2. Folgen Sie den Anweisungen, um den Kalender zu Google Calendar, Apple Kalender, Outlook oder jeder anderen Kalenderanwendung hinzuzufügen, die Abonnements unterstützt.

:::tip
Das Abonnieren hält Ihren persönlichen Terminplan mit den Veranstaltungen Ihrer Gruppe synchron, ohne B1 separat überprüfen zu müssen. Alle Änderungen durch Gruppenleiter erscheinen automatisch in Ihrem synchronisierten Kalender.
:::
