# Gruppenkalender

<div class="article-intro">

Jede Gruppe in B1 kann ihren eigenen Ereigniskalender haben. Gruppenleitern verwenden ihn, um Treffen, Aktivitäten und wiederkehrende Ereignisse zu planen, während alle Gruppenmitglieder bevorstehende Ereignisse sehen und an Gesprächen darüber teilnehmen können. Der Kalender hält Ihre Gruppe organisiert und stellt sicher, dass jeder über die anstehenden Ereignisse informiert bleibt.

</div>

<div class="prereqs">
<h4>Bevor Sie anfangen</h4>

- Sie benötigen eine in B1 Admin erstellte Gruppe mit mindestens einem Mitglied. Siehe [Erstellen von Gruppen](creating-groups.md) und [Gruppenmitglieder](group-members.md).
- Um Kalenderereignisse hinzuzufügen oder zu bearbeiten, müssen Sie als Gruppenleiter bestimmt werden. Siehe [Bestimmen von Gruppenleitern](group-members.md) für Anweisungen.

</div>

## Zugriff auf den Gruppenkalender

Der Gruppenkalender ist über die B1.church-App für Mitglieder verfügbar:

1. Klicken Sie in B1 Admin auf das **Dropdown-Menü** in der oberen rechten Ecke Ihres Bildschirms.
2. Wählen Sie **App wechseln**.
3. Wählen Sie **B1.church**.
4. Klicken Sie in der linken Seitenleiste auf **Gruppen**.
5. Wählen Sie Ihre Gruppe aus der Liste.
6. Klicken Sie auf der linken Seite des Bildschirms auf **Kalender**.

:::info
Der Gruppenkalender wird über B1.church (die Mitglieder-App) verwaltet, nicht über B1 Admin. Gruppenleitern greifen darauf zu, indem sie von dem Admin-Dashboard aus die Apps wechseln.
:::

## Ereignis hinzufügen

1. Klicken Sie im Gruppenkalender auf **Ereignis hinzufügen**.
2. Geben Sie einen **Titel** für das Ereignis ein.
3. Schreiben Sie eine kurze **Beschreibung** über das Ereignis.
4. Legen Sie die **Startzeit** und **Endzeit** fest.
5. Klicken Sie auf **Speichern**, um das Ereignis zum Kalender hinzuzufügen.

## Einrichten wiederkehrender Ereignisse

Wenn Ihre Gruppe nach einem regulären Zeitplan trifft, können Sie Ereignisse so einstellen, dass sie automatisch wiederholt werden:

1. Wenn Sie ein Ereignis erstellen oder bearbeiten, markieren Sie das Kontrollkästchen **Wiederkehrend**.
2. Wählen Sie, wie das Ereignis wiederholt werden soll:
   - **Interval** -- Legen Sie eine Anzahl von Tagen zwischen Vorkommen fest. Zum Beispiel, "7" eingeben, wiederholt das Ereignis alle 7 Tage.
   - **Häufigkeit** -- Wählen Sie, um nach **Tag**, **Woche** oder **Monat** zu wiederholen.
3. Speichern Sie das Ereignis. Es wird nun im Kalender bei jedem geplanten Vorkommen angezeigt.

:::tip
Für eine Gruppe, die sich jeden Samstag trifft, stellen Sie die Häufigkeit auf "Woche" ein und wählen Sie Samstag. Das Ereignis wird automatisch im Kalender in der Zukunft angezeigt.
:::

## Massenerstellung von Ereignissen in B1 Admin

Die Gruppenseite in B1 Admin hat auch eine Registerkarte **Kalender**, auf der Mitarbeiter eine ganze Saison von Treffen auf einmal planen können:

1. Öffnen Sie die Gruppe in B1 Admin und wählen Sie die Registerkarte **Kalender**.
2. Klicken Sie auf **Ereignisse hinzufügen**, um das Dialogfeld **Wiederkehrende Ereignisse hinzufügen** zu öffnen.
3. Wählen Sie den Zeitplan (wöchentlich, alle 2 Wochen, monatlich oder einmalig), das Datum des **ersten Treffens** und des **letzten Treffens** und optionally **Feiertage überspringen**.
4. Überprüfen Sie die Vorschau der Treffendaten und deaktivieren Sie alle, die übersprungen werden sollen.
5. Lassen Sie **RSVPs zulassen** aktiviert, wenn Mitglieder auf diese Ereignisse antworten können sollen (siehe unten), und speichern Sie dann.

## Ereignis-RSVPs

Gruppenereignisse können RSVPs erfassen -- ein **Ja**, **Nein** oder **Vielleicht** für jedes Vorkommen des Ereignisses.

- **RSVPs zulassen** ist standardmäßig für Ereignisse aktiviert, die über das Massenhinzufügen-Dialogfeld erstellt wurden; deaktivieren Sie es, um Antworten für diese Ereignisse auszuschalten. Ereignisse mit ausgeschalteten RSVPs werden auf der Registerkarte "Kalender" als **Deaktiviert** angezeigt.
- Auf der Registerkarte des Gruppendarkalenders in B1 Admin zeigt jedes Ereignis eine Antwortenzusammenfassung (z.B. "5 ja · 2 vielleicht · 1 nein"). Klicken Sie durch, um das Verzeichnis **RSVP-Antworten** zu öffnen, das auflistet, wer ja, vielleicht und nein für dieses Vorkommen geantwortet hat.

:::tip
RSVPs geben Führungskräften eine Kopfzahl ohne die Formalität der vollständigen Ereignisregistrierung. Für bezahlte Anmeldungen oder kapazitätsverwaltete Ereignisse verwenden Sie stattdessen [Ereignisregistrierung](../calendars/creating-calendars.md#ereignisregistrierung-aktivieren).
:::

## Anzeigen des Kalenders

Der Gruppenkalender bietet mehrere Anzeigeoptionen:

- **Monat** -- Siehe den ganzen Monat auf einen Blick.
- **Woche** -- Konzentrieren Sie sich auf die aktuelle Woche.
- **Tag** -- Sehen Sie einen einzelnen Tag im Detail.
- **Agenda** -- Siehe eine Liste von bevorstehenden Ereignissen in chronologischer Reihenfolge.

Wechseln Sie zwischen den Ansichten mit den Schaltflächen oben im Kalender.

## Ereignisdetails und Gespräche

Klicken Sie auf ein beliebiges Ereignis, um seine Details anzuzeigen. Aus der Ereignisdetailansicht:

- **Mitglieder und Leiter** können Gespräche starten und Kommentare zum Ereignis hinterlassen.
- **Leiter** können Ereignisdetails bearbeiten, Zeiten ändern oder das Ereignis löschen.

:::warning
Das Löschen eines Ereignisses entfernt es dauerhaft. Wenn Sie nur ein Vorkommen eines wiederkehrenden Ereignisses stornieren möchten, sollten Sie es stattdessen bearbeiten.
:::

## Abonnieren des Kalenders

Sie können den Gruppenkalender mit Ihrem persönlichen Kalender synchronisieren:

1. Klicken Sie auf der Gruppenkendenseite auf **Abonnieren**.
2. Folgen Sie den Anweisungen, um den Kalender zu Google Calendar, Apple Calendar, Outlook oder einer anderen Kalenderanwendung hinzuzufügen, die Abos unterstützt.

:::tip
Das Abonnieren hält Ihren persönlichen Zeitplan mit den Ereignissen Ihrer Gruppe synchronisiert, ohne dass Sie B1 separat überprüfen müssen. Alle Änderungen, die von Gruppenleitern vorgenommen werden, werden automatisch in Ihrem synchronisierten Kalender angezeigt.
:::
