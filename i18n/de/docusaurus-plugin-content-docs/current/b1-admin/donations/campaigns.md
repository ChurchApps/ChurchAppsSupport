# Kampagnen & Zusagen

<div class="article-intro">

Kampagnen ermöglichen es Ihnen, eine Fundraising-Aktion mit einem bestimmten Ziel durchzuführen – ein Gebäudefonds, eine Missionsreise, ein spezielles Projekt – und Zusagen von Mitgliedern zusammen mit tatsächlichen Spenden zu verfolgen, damit Sie Ihren Echtzeitfortschritt sehen können.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Richten Sie Ihre [Spendendonds](funds) ein – jede Kampagne ist an einen Fonds gebunden
- Sie benötigen Zugriff auf den Abschnitt „Spenden" von B1 Admin

</div>

## Kampagnen öffnen

Gehen Sie in B1 Admin zu **Spenden** in der linken Seitenleiste und wählen Sie **Kampagnen**. Sie sehen eine Liste aller Kampagnen mit ihrem Zielwert, Gesamtzusagen und bisher insgesamt gegebener Summe.

## Kampagne erstellen

1. Klicken Sie auf **Kampagne hinzufügen**.
2. Füllen Sie die Kampagnendetails aus:
   - **Name** *(erforderlich)* – der Anzeigename für diese Kampagne (zum Beispiel „Gebäudefonds 2026").
   - **Fonds** – der Spendendonds, mit dem diese Kampagne verbunden ist.
   - **Startdatum** / **Enddatum** – das Fundraising-Fenster.
   - **Ziel** – der Zieldollarbetrag für die Kampagne.
3. Klicken Sie auf **Speichern**.

## Fortschritt verfolgen

Jede Kampagnenkarte zeigt:

- **Ziel** – Ihr Zielwert
- **Zusagen** – der Gesamtbetrag, den Mitglieder zusagen zu geben
- **Gegeben** – der tatsächliche Spendenbetrag für diesen Kampagnenfonds während des Kampagnenzeitraums
- Eine **Fortschrittsleiste**, die zeigt, wie weit Sie Ihrem Ziel gekommen sind

Klicken Sie auf eine beliebige Kampagne, um die Detailansicht zu öffnen, die einzelne Zusagen und ihren Erfüllungsstatus auflistet.

## Zusagen hinzufügen

Zusagen sind Verpflichtungen von Mitgliedern, eine Kampagne zu unterstützen. Um eine Zusage aufzuzeichnen:

1. Öffnen Sie eine Kampagne.
2. Klicken Sie auf **Zusage hinzufügen**.
3. Wählen Sie die **Person**, die die Zusage gibt.
4. Geben Sie den Zusagebetrag ein.
5. Legen Sie optional ein **Datum** für die Zusagezusage fest.
6. Klicken Sie auf **Speichern**.

Zusagen erscheinen in der Kampagnendetail und tragen zum **Zusagen**-Gesamtwert auf der Kampagnenkarte bei.

:::tip
Zusagen unterscheiden sich von tatsächlichen Spenden. Eine Zusage verfolgt eine Verpflichtung; eine Spende erfasst das tatsächliche Geschenk. Beide werden in der Kampagne angezeigt, damit Sie sehen können, wie gut Zusagen erfüllt werden.
:::

## Kampagne bearbeiten oder löschen

Klicken Sie auf das Bearbeiten-Symbol auf einer beliebigen Kampagnenkarte, um ihren Namen, Ziel, Termine oder Fonds zu aktualisieren. Klicken Sie auf **Löschen**, um die Kampagne und ihre Zusagen dauerhaft zu entfernen. Das Löschen einer Kampagne löscht keine Spenden, die für ihren Fonds aufgezeichnet wurden.
