# Fonds verwalten

<div class="article-intro">

Fonds ermöglichen es Ihnen, Spenden nach Zweck zu kategorisieren, damit Sie verfolgen können, wie viel Ihre Gemeinde für jeden Bereich des Dienstes erhält. Gängige Fonds sind Allgemeiner Fonds, Baufonds, Missionen und Jugendarbeit, aber Sie können so viele erstellen, wie Sie benötigen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen Zugang zum Bereich **Spenden** in B1 Admin
- Entscheiden Sie, welche Spendenkategorien Ihre Gemeinde benötigt und ob jede steuerlich absetzbar sein soll

</div>

## Die Fondsseite

Navigieren Sie zu **Spenden > Fonds**, um eine Liste aller Ihrer Fonds zu sehen. Jede Zeile zeigt den Fondsnamen und seinen Status als **steuerlich absetzbar**, der bestimmt, ob Spenden an diesen Fonds auf den Jahresend-[Spendenbescheinigungen](giving-statements.md) erscheinen.

## Einen neuen Fonds erstellen

1. Klicken Sie oben auf der Seite auf **Fonds hinzufügen**.
2. Geben Sie einen Namen für den Fonds ein (z. B. „Baufonds").
3. Aktivieren Sie das Kontrollkästchen **Steuerlich absetzbar**, wenn Spenden an diesen Fonds auf Spendenbescheinigungen erscheinen sollen.
4. Klicken Sie auf **Speichern**.

Der neue Fonds erscheint sofort in Ihrer Liste und ist bei der Spendenerfassung verfügbar.

## Fonds bearbeiten und löschen

- Um einen Fonds zu bearbeiten, klicken Sie auf die Schaltfläche **Bearbeiten** daneben. Sie können den Namen ändern oder die Einstellung für steuerliche Absetzbarkeit umschalten.
- Um einen Fonds zu löschen, öffnen Sie das Bearbeitungsformular und klicken Sie auf **Löschen**.

:::warning
Das Löschen eines Fonds entfernt keine Spenden, die zuvor diesem Fonds zugewiesen waren. Diese Spendeneinträge bleiben im System, aber der Fonds ist für neue Spenden nicht mehr verfügbar.
:::

## Fondsdetails anzeigen

Klicken Sie auf einen **Fondsnamen**, um seine Detailseite zu öffnen. Diese Seite zeigt:

- **Spendenhistorie** -- eine Liste aller Spenden an diesen Fonds
- **Zeitraumfilter** -- Ergebnisse auf einen bestimmten Zeitraum eingrenzen
- **Zusammenfassungsstatistiken** -- Gesamtspenden, einzigartige Spender und Gesamtbetrag

Diese Ansicht ist hilfreich, wenn Sie über eine bestimmte Spendeninitiative berichten oder den Fortschritt eines Spendenziels überprüfen müssen.

## Sortieren und Exportieren

- Verwenden Sie die Spaltenüberschriften, um Fonds nach Name zu sortieren.
- Klicken Sie auf **Als CSV exportieren**, um Ihre Fondsliste als Tabelle herunterzuladen.

:::tip
Richten Sie Ihre Fonds ein, bevor Sie mit der [Spendenerfassung](recording-donations.md) beginnen. Wenn alle Fonds vorhanden sind, wird sichergestellt, dass Spenden von Anfang an richtig kategorisiert werden.
:::

## Nächste Schritte

Sobald Ihre Fonds eingerichtet sind, können Sie sie bei der [Spendenerfassung](recording-donations.md) in einem Stapel zuweisen. Stellen Sie sicher, dass die entsprechenden Fonds als steuerlich absetzbar markiert sind, bevor Sie [Spendenbescheinigungen](giving-statements.md) erstellen.
