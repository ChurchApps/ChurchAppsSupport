# Fonds verwalten

<div class="article-intro">

Fonds ermöglichen es Ihnen, Spenden nach Zweck zu kategorisieren, sodass Sie verfolgen können, wie viel Ihre Kirche für jeden Dienstbereich erhält. Übliche Fonds sind Allgemeiner Fonds, Baufonds, Mission und Jugendarbeit, aber Sie können so viele erstellen, wie Sie benötigen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen Zugriff auf den Bereich **Spenden** in B1 Admin
- Entscheiden Sie, welche Spendenkategorien Ihre Kirche benötigt und ob jede davon steuerlich absetzbar sein soll

</div>

## Die Fondsseite

Navigieren Sie zu **Spenden > Fonds**, um eine Liste aller Ihrer Fonds anzuzeigen. Jede Zeile zeigt den Fondsnamen und seinen Status **steuerlich absetzbar**, der bestimmt, ob Spenden an diesen Fonds auf den [Spendenbescheinigungen](giving-statements.md) am Jahresende erscheinen.

## Einen neuen Fonds erstellen

1. Klicken Sie oben auf der Seite auf **Fonds hinzufügen**.
2. Geben Sie einen Namen für den Fonds ein (z. B. „Baufonds").
3. Aktivieren Sie das Kontrollkästchen **Steuerlich absetzbar**, wenn Spenden an diesen Fonds auf Spendenbescheinigungen aufgeführt werden sollen.
4. Lassen Sie **Für Spender sichtbar** aktiviert, um den Fonds auf öffentlichen und Mitglieder-Spendenformularen anzuzeigen, oder deaktivieren Sie es, um den Fonds vor Spendern zu verbergen, während Sie ihn weiterhin für manuell erfasste Spenden verwenden.
5. Klicken Sie auf **Speichern**.

Der neue Fonds erscheint sofort in Ihrer Liste und steht bei der Eingabe von Spenden zur Verfügung.

## Fonds bearbeiten und löschen

- Um einen Fonds zu bearbeiten, klicken Sie auf die Schaltfläche **Bearbeiten** daneben. Sie können den Namen ändern, die Einstellung „steuerlich absetzbar" umschalten oder **Für Spender sichtbar** umschalten, um ihn auf Spendenformularen anzuzeigen oder auszublenden.
- Um einen Fonds zu löschen, öffnen Sie das Bearbeitungsformular und klicken Sie auf **Löschen**.

:::warning
Das Löschen eines Fonds entfernt keine Spenden, die ihm zuvor zugewiesen wurden. Diese Spendendatensätze bleiben im System erhalten, aber der Fonds steht für neue Spenden nicht mehr zur Verfügung.
:::

:::info
Fonds, die vor Spendern verborgen sind, zeigen in dieser Liste neben ihrem Namen ein **Verborgen**-Etikett. Verborgene Fonds funktionieren weiterhin für manuell in B1 Admin erfasste Spenden -- sie erscheinen nur nicht als Option auf den öffentlichen oder Mitglieder-Spendenformularen.
:::

## Fondsdetails anzeigen

Klicken Sie auf einen **Fondsnamen**, um dessen Detailseite zu öffnen. Diese Seite zeigt:

- **Spendenverlauf** -- eine Liste aller Spenden an diesen Fonds
- **Datumsbereichsfilter** -- die Ergebnisse auf einen bestimmten Zeitraum eingrenzen
- **Zusammenfassende Statistiken** -- Gesamtspenden, eindeutige Spender und Gesamtbetrag

Diese Ansicht ist hilfreich, wenn Sie über eine bestimmte Spendeninitiative berichten oder den Fortschritt in Richtung eines Spendenziels überprüfen müssen.

## Sortieren und Exportieren

- Verwenden Sie die Spaltenüberschriften, um Fonds nach Namen zu sortieren.
- Klicken Sie auf **Als CSV exportieren**, um Ihre Fondsliste als Tabellenkalkulation herunterzuladen.

:::tip
Richten Sie Ihre Fonds ein, bevor Sie mit dem [Erfassen von Spenden](recording-donations.md) beginnen. Wenn alle Fonds bereits eingerichtet sind, werden Spenden von Anfang an korrekt kategorisiert.
:::

## Nächste Schritte

Sobald Ihre Fonds eingerichtet sind, können Sie sie beim [Erfassen von Spenden](recording-donations.md) in einem Stapel zuweisen. Stellen Sie sicher, dass die entsprechenden Fonds als steuerlich absetzbar markiert sind, bevor Sie [Spendenbescheinigungen](giving-statements.md) erstellen.
