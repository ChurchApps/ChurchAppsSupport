# Spenden

<div class="article-intro">

Der Bereich Spenden in B1 Admin bietet alles, was Sie für die Verwaltung der Spenden Ihrer Gemeinde benötigen. Von der Erfassung wöchentlicher Kollekten bis zur Erstellung von Jahresend-Steuerbescheinigungen sind alle Spendenwerkzeuge an einem Ort organisiert, damit Sie die Finanzen Ihrer Gemeinde im Blick behalten.

</div>

## Unterbereiche

Sie finden folgende Seiten in der Spenden-Seitenleiste:

### Zusammenfassung

Die Seite **Zusammenfassung** gibt Ihnen einen visuellen Überblick über alle Spendenaktivitäten. Sie zeigt Diagramme und gefilterte Berichte, damit Sie Spendentrends Ihrer Gemeinde schnell erkennen können. Siehe [Spendenberichte](donation-reports.md) für weitere Details zur Nutzung der Zusammenfassungs- und Berichtswerkzeuge.

### Stapel

**[Stapel](batches.md)** ermöglichen es Ihnen, Spenden zu gruppieren -- zum Beispiel alle Schecks und Bargeld, die während einer Sonntagskollekte gesammelt wurden. Jeder Stapel verfolgt die Anzahl der Spenden und den Gesamtbetrag für eine einfache Abstimmung.

### Fonds

**[Fonds](funds.md)** kategorisieren Ihre Spenden nach Zweck. Gängige Beispiele sind Allgemeiner Fonds, Baufonds und Missionen. Sie können jeden Fonds als steuerlich absetzbar kennzeichnen oder nicht.

### Bescheinigungen

Die Seite **[Bescheinigungen](giving-statements.md)** erstellt Jahresend-Spendenbescheinigungen für Ihre Spender. Sie können einzelne CSV-Dateien herunterladen oder alle Bescheinigungen auf einmal drucken.

### Stripe-Import

Wenn Sie Online-Spenden über Stripe akzeptieren, können Sie mit dem **[Stripe-Import](stripe-import.md)**-Werkzeug diese Transaktionen in B1 Admin übernehmen, damit alles in einem System ist.

## Online-Spenden

B1 Admin integriert sich mit **Stripe** und **PayPal** für die Annahme von Online-Spenden. Nach der Konfiguration können Mitglieder über Ihre B1.church-Website spenden, und diese Spenden erscheinen automatisch in Ihren Aufzeichnungen. Siehe [Online-Spenden einrichten](online-giving-setup.md) für den Einstieg.

:::tip
Wenn Sie neu im Spendenbereich von B1 Admin sind, beginnen Sie mit dem [Einrichten Ihrer Fonds](funds.md), dann [erstellen Sie Ihren ersten Stapel](batches.md) und [erfassen Sie Spenden](recording-donations.md).
:::
