# Massenimport

<div class="article-intro">

Die Massenimport-Seite ermöglicht es Ihnen, Ihre Predigtbibliothek schnell zu füllen, indem Sie Videos von YouTube oder Vimeo importieren. Dies ist der schnellste Weg, Ihre bestehenden Predigtinhalte einzubringen, anstatt jedes Video einzeln hinzuzufügen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Sie benötigen die Berechtigung **contentApi.streamingServices.edit**. Siehe [Rollen & Berechtigungen](../settings/roles-permissions.md), falls Sie keinen Zugang haben.
- Erstellen Sie mindestens eine [Playlist](playlists), in die Sie Ihre Predigten importieren
- Halten Sie Ihre YouTube-Kanal-ID oder Vimeo-Kontoinformationen bereit

</div>

## Ihre Quelle wählen

1. Klicken Sie in B1 Admin in der linken Seitenleiste auf **Predigten** und dann auf die Registerkarte **Massenimport**.
2. Sie sehen zwei anklickbare Karten:
   - **YouTube** (rot) -- Videos von einem YouTube-Kanal importieren
   - **Vimeo** (blau) -- Videos von einem Vimeo-Konto importieren
3. Klicken Sie auf die Karte der Plattform, auf der Ihre Predigten gehostet sind.

:::tip
Sie können jederzeit zurückgehen und die Quelle wechseln, wenn Sie Videos auf beiden Plattformen haben.
:::

## Vor dem Import

Bevor Sie Videos importieren, benötigen Sie mindestens eine Playlist, in die Sie sie einordnen können. Falls Sie noch keine Playlists erstellt haben:

1. Klicken Sie auf die Registerkarte **Playlists**.
2. Klicken Sie auf **Erste Playlist erstellen** und füllen Sie Name, Beschreibung, Veröffentlichungsdatum und Vorschaubild aus.
3. Klicken Sie auf **Speichern** und kehren Sie dann zur Registerkarte **Massenimport** zurück.

Siehe [Playlists](playlists) für detaillierte Anweisungen zum Erstellen und Verwalten von Playlists.

## Videos importieren

1. Geben Sie nach der Auswahl von YouTube oder Vimeo Ihre **YouTube-Kanal-ID** oder **Vimeo-Kontoinformationen** in das bereitgestellte Feld ein.
2. Klicken Sie auf **Abrufen**, um alle verfügbaren Videos von Ihrem Kanal oder Konto abzurufen.
3. Nach dem Abrufen sehen Sie eine Liste aller Ihrer Videos mit Kontrollkästchen.
4. Aktivieren Sie die Kontrollkästchen neben den Videos, die Sie importieren möchten. Sie können alle oder bestimmte auswählen.
5. Aktivieren Sie optional **Neue Videos automatisch importieren**, um zukünftige Uploads automatisch zu Ihrer Bibliothek hinzuzufügen.
6. Klicken Sie auf das Dropdown-Menü **In Playlist importieren**, um zu wählen, welcher Playlist diese Videos hinzugefügt werden sollen.
7. Klicken Sie auf **Importieren**, um den Massenimport abzuschließen.

Ihre Videos werden mit allen Details importiert, einschließlich Titel, Beschreibungen, Daten und Vorschaubilder.

:::info
Der Massenimport ist ideal für den Einstieg, wenn Sie von einer anderen Plattform migrieren. Für das Hinzufügen einzelner Predigten in Zukunft verwenden Sie stattdessen die Seite [Predigten verwalten](managing-sermons).
:::
