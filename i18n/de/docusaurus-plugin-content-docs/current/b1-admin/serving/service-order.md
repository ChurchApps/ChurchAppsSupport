# Serviceablauf

<div class="article-intro">

Der Serviceablauf definiert die Reihenfolge der Elemente in Ihrem Gottesdienst. Nutzen Sie ihn, um alles vom Eröffnungslied bis zum Schlussgebet festzulegen, damit Ihr gesamtes Team den Ablauf des Gottesdienstes im Voraus kennt.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Erstellen Sie einen Serviceplan](./plans.md) für das Datum, für das Sie einen Serviceablauf erstellen möchten
- Fügen Sie optional [Lieder](./songs.md) zu Ihrer Bibliothek hinzu, damit Sie sie direkt in den Serviceablauf verlinken können

</div>

## Auf den Serviceablauf zugreifen

1. Navigieren Sie zu **Dienst** und wählen Sie Ihren Dienstbereich aus.
2. Klicken Sie auf einen **Plantyp** und öffnen Sie dann einen bestimmten **Plan**.
3. Klicken Sie auf der Plandetailseite auf die Registerkarte **Serviceablauf**.

## Ihren Serviceablauf erstellen

Der Serviceablauf ist eine Liste von Elementen, die in der Reihenfolge angeordnet sind, in der sie während des Gottesdienstes stattfinden. Übliche Elemente sind:

- Lobpreislieder
- Gebete
- Schriftlesungen
- Ankündigungen
- Predigt oder Botschaft
- Kollekte
- Schluss

## Elemente zum Serviceablauf hinzufügen

1. Klicken Sie auf der Registerkarte **Serviceablauf** auf **Element hinzufügen**.
2. Wählen Sie die Art des hinzuzufügenden Elements (zum Beispiel ein Lied aus Ihrer Bibliothek, ein benutzerdefiniertes Element oder eine Überschrift).
3. Geben Sie die Details für das Element ein, wie Titel und Notizen.
4. Klicken Sie auf **Speichern**.

:::tip
Sie können Lieder direkt aus Ihrer [Lieder](./songs.md)-Bibliothek hinzufügen. Dies verknüpft die Liededetails, einschließlich Liedtexte und Arrangements, mit Ihrem Serviceplan, damit Ihr Lobpreisteam alles hat, was es braucht.
:::

## Elemente neu anordnen

Ziehen Sie Elemente am Griff links von jeder Zeile, um die Reihenfolge zu ändern -- lassen Sie sie über der blauen Linie los, an der das Element landen soll, oder über einer Abschnittsüberschrift, um es in diesen Abschnitt zu verschieben. Die auf dem Bildschirm angezeigte Reihenfolge ist die Reihenfolge, der Ihr Team während des Gottesdienstes folgt.

## Timing und Medien

Jedes Element und jeder Abschnitt zeigt, wie lange es dauert und wann es beginnt:

- Wenn der Plan eine einzelne ausgewählte Gottesdienstzeit hat, zeigt die linke Leiste die tatsächliche **Uhrzeit** an, zu der jedes Element beginnt (zum Beispiel 9:14 Uhr). Andernfalls wird die verstrichene Zeit seit Beginn des Gottesdienstes angezeigt.
- Abschnittsüberschriften summieren die Laufzeit von allem darin.
- Lieder und benutzerdefinierte Elemente zählen erst dann zu diesem Zeitplan, wenn Sie **Minuten**/**Sekunden** auf dem Bearbeitungsbildschirm des Elements eingeben.
- Elemente, die Sie aus Ihrer Medienbibliothek hinzufügen (Bilder oder Videoclips), zeigen ein **Miniaturbild** und werden automatisch getaktet -- die Dauer eines Videos wird aus der Datei selbst gemessen, während ein Bild eine Planungsschätzung von etwa 5:00 (kursiv angezeigt) beiträgt, da es auf dem Bildschirm bleibt, bis ein Operator weiterschaltet. Sie können beides trotzdem auf eine bestimmte Dauer bearbeiten.

## Elemente bearbeiten und entfernen

- Klicken Sie auf ein Element im Serviceablauf, um dessen Details zu **bearbeiten**.
- Verwenden Sie die Option **Löschen**, um ein Element aus dem Ablauf zu entfernen.

:::info
Der Serviceablauf ist spezifisch für jeden Plan. Änderungen, die Sie hier vornehmen, betreffen nur den ausgewählten Plan und verändern weder andere Pläne noch Ihre Liederbibliothek.
:::

## Elemente in bestimmten Gottesdienstzeiten einschließen

Wenn Ihr Plan mehr als eine Gottesdienstzeit hat (zum Beispiel einen 8-Uhr- und einen 10-Uhr-Gottesdienst), können Sie auswählen, in welchen Gottesdiensten jedes Element erscheinen soll. Dies ist nützlich, wenn eine Ankündigung nur für einen Gottesdienst relevant ist oder wenn ein Lied in einem Gottesdienst gesungen wird, in einem anderen jedoch nicht.

1. Öffnen Sie einen Plan mit zwei oder mehr Gottesdienstzeiten, die auf der Registerkarte **Zeiten** definiert sind.
2. Klicken Sie auf der Registerkarte **Serviceablauf** auf ein Element, um es zu bearbeiten.
3. Unter **In Gottesdiensten einschließen** sehen Sie ein Kontrollkästchen für jede Gottesdienstzeit, beschriftet mit der Uhrzeit.
4. Deaktivieren Sie jeden Gottesdienst, in dem das Element übersprungen werden soll.
5. Klicken Sie auf **Speichern**.

Standardmäßig ist jedes Element in jedem Gottesdienst enthalten. Ausgeschlossene Gottesdienstzeiten werden beim Drucken des auf diesen Gottesdienst gefilterten Plans ausgeblendet, sodass jeder Gottesdienst einen sauberen Ablaufplan mit nur den relevanten Elementen erhält.

:::tip
Der Abschnitt **In Gottesdiensten einschließen** erscheint nur, wenn der Plan mehr als eine Gottesdienstzeit hat. Wenn Sie nur einen Gottesdienst sehen, bearbeiten Sie zuerst die Registerkarte **Zeiten** des Plans, um weitere Gottesdienste hinzuzufügen.
:::

## Den Serviceablauf teilen

Sobald Ihr Serviceablauf vollständig ist, können Sie den vollständigen Plan (einschließlich des Serviceablaufs) von der Plandetailseite aus drucken. Dies gibt Ihrem Team einen vollständigen Überblick über den Gottesdienst.

## Nächste Schritte

- Verwalten Sie Ihr Lobpreisrepertoire auf der Seite [Lieder](./songs.md)
- Weisen Sie Freiwillige auf der Registerkarte Zuweisungen der [Servicepläne](./plans.md) dem Gottesdienst zu
- Erstellen Sie [Aufgaben](./tasks.md) für alle Nachverfolgungspunkte im Zusammenhang mit dem Gottesdienst
