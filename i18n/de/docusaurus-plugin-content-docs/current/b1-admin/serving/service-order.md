# Dienstreihenfolge

<div class="article-intro">

Die Dienstreihenfolge definiert die Abfolge der Elemente Ihres Gottesdiensts. Verwenden Sie es, um alles vom Eröffnungslied bis zum Abschlussgebet auszulegen, damit Ihr gesamtes Team den Ablauf des Gottesdiensts im Voraus kennt.

</div>

<div class="prereqs">
<h4>Bevor Sie Beginnen</h4>

- [Erstellen Sie einen Dienst-Plan](./plans.md) für das Datum, für das Sie eine Dienstreihenfolge erstellen möchten
- Fügen Sie optional [Lieder](./songs.md) zu Ihrer Bibliothek hinzu, damit Sie sie direkt in die Dienstreihenfolge verlinken können

</div>

## Zugriff auf die Dienstreihenfolge

1. Navigieren Sie zu **Serving** und wählen Sie Ihren Dienst.
2. Klicken Sie auf einen **Plantyp** und öffnen Sie dann einen bestimmten **Plan**.
3. Klicken Sie auf die Registerkarte **Service Order** auf der Plan-Detailseite.

## Erstellen Sie Ihre Dienstreihenfolge

Die Dienstreihenfolge ist eine Liste von Elementen, die in der Reihenfolge angeordnet sind, in der sie während des Gottesdiensts auftreten. Häufige Elemente sind:

- Anbetungslieder
- Gebete
- Schriftlesungen
- Ankündigungen
- Predigt oder Botschaft
- Opfer
- Abschluss

## Hinzufügen von Elementen zur Dienstreihenfolge

1. Klicken Sie auf der Registerkarte **Service Order** auf **Add Item**.
2. Wählen Sie den Typ des Elements aus, das Sie hinzufügen möchten (z. B. ein Lied aus Ihrer Bibliothek, ein benutzerdefiniertes Element oder eine Überschrift).
3. Geben Sie die Details für das Element ein, z. B. den Titel und alle Notizen.
4. Klicken Sie auf **Save**.

:::tip
Sie können Lieder direkt aus Ihrer [Songs](./songs.md)-Bibliothek hinzufügen. Dies verlinkt die Songdetails, einschließlich Texten und Arrangements, mit Ihrem Dienst-Plan, damit Ihr Anbetungsteam alles hat, was es braucht.
:::

## Neuanordnen von Elementen

Ziehen Sie Elemente per Drag-and-Drop, um die Reihenfolge zu ändern. Die auf dem Bildschirm angezeigte Abfolge ist die Reihenfolge, die Ihr Team während des Gottesdiensts befolgt.

## Bearbeiten und Entfernen von Elementen

- Klicken Sie auf ein Element in der Dienstreihenfolge, um seine Details zu **bearbeiten**.
- Verwenden Sie die Option **delete**, um ein Element aus der Reihenfolge zu entfernen.

:::info
Die Dienstreihenfolge ist spezifisch für jeden Plan. Änderungen, die Sie hier vornehmen, beeinflussen nur den ausgewählten Plan und ändern keine anderen Pläne oder Ihre Lied-Bibliothek.
:::

## Einbeziehen von Elementen in Spezifische Dienstzeiten

Wenn Ihr Plan mehr als eine Dienstzeit hat (z. B. einen 8-Uhr- und 10-Uhr-Gottesdienst), können Sie auswählen, welche Dienste jedes Element enthalten soll. Dies ist nützlich, wenn eine Ankündigung nur für einen Gottesdienst relevant ist oder wenn ein Lied in einem Gottesdienst gesungen wird, aber nicht in einem anderen.

1. Öffnen Sie einen Plan, auf dem auf der Registerkarte **Times** zwei oder mehr Dienstzeiten definiert sind.
2. Klicken Sie auf der Registerkarte **Service Order** auf ein Element, um es zu bearbeiten.
3. Unter **Include in Services** sehen Sie ein Kontrollkästchen für jede Dienstzeit mit der Zeitangabe.
4. Deaktivieren Sie alle Dienste, in denen das Element übersprungen werden soll.
5. Klicken Sie auf **Save**.

Standardmäßig sind alle Elemente in jedem Dienst enthalten. Ausgeschlossene Dienstzeiten werden ausgeblendet, wenn Sie den Plan gefiltert nach diesem Dienst drucken, damit jeder Dienst ein sauberes Ablauf-Blatt nur mit den relevanten Elementen erhält.

:::tip
Der Abschnitt **Include in Services** wird nur angezeigt, wenn der Plan mehr als eine Dienstzeit hat. Wenn Sie nur einen Dienst sehen, bearbeiten Sie zuerst die Registerkarte **Times** des Plans, um zusätzliche Dienste hinzuzufügen.
:::

## Freigeben der Dienstreihenfolge

Sobald Ihre Dienstreihenfolge vollständig ist, können Sie den vollständigen Plan (einschließlich der Dienstreihenfolge) von der Plan-Detailseite ausdrucken. Dies gibt Ihrem Team einen vollständigen Überblick über den Gottesdienst.

## Nächste Schritte

- Verwalten Sie Ihr Anbetungs-Repertoire auf der Seite [Songs](./songs.md)
- Weisen Sie Freiwillige dem Gottesdienst auf der Registerkarte [Plans](./plans.md) Assignments zu
- Erstellen Sie [Aufgaben](./tasks.md) für alle Folgemaßnahmen im Zusammenhang mit dem Gottesdienst
