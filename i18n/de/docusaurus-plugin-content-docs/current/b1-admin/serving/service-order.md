# Servicefolge

<div class="article-intro">

Die Servicefolge definiert die Abfolge von Elementen in Ihrem Gottesdienst. Verwenden Sie sie, um alles von dem öffnenden Lied bis zu dem schließenden Gebet zu legen, damit Ihr ganzes Team den Ablauf des Gottesdienstes im Voraus kennt.

</div>

<div class="prereqs">
<h4>Bevor Sie anfangen</h4>

- [Erstellen Sie einen Serviceplan](./plans.md) für das Datum, für das Sie eine Servicefolge bauen möchten
- Optional, fügen Sie [Lieder](./songs.md) zu Ihrer Bibliothek hinzu, damit Sie sie direkt in die Servicefolge verknüpfen können

</div>

## Zugriff auf die Servicefolge

1. Navigieren Sie zu **Dienst** und wählen Sie Ihr Ministerium.
2. Klicken Sie auf einen **Plantyp**, dann öffnen Sie einen bestimmten **Plan**.
3. Klicken Sie auf die Registerkarte **Servicefolge** auf der Plandetailseite.

## Bauen Sie Ihre Servicefolge auf

Die Servicefolge ist eine Liste von Elementen, die in der Reihenfolge angeordnet sind, in der sie während des Gottesdienstes stattfinden. Häufige Elemente enthalten:

- Anbetungslieder
- Gebete
- Schriftenlesungen
- Ankündigungen
- Predigt oder Nachricht
- Opfer
- Schließung

## Hinzufügen von Elementen zur Servicefolge

1. Klicken Sie auf der Registerkarte **Servicefolge** auf **Element hinzufügen**.
2. Wählen Sie den Typ des Elements aus, das Sie hinzufügen möchten (z.B. ein Lied aus Ihrer Bibliothek, ein benutzerdefiniertes Element oder eine Überschrift).
3. Geben Sie die Details für das Element ein, wie den Titel und alle Notizen.
4. Klicken Sie auf **Speichern**.

:::tip
Sie können Lieder direkt aus Ihrer [Lieder](./songs.md)-Bibliothek hinzufügen. Dies verknüpft die Lied-Details, einschließlich Texte und Arrangements, mit Ihrem Serviceplan, sodass Ihr Anbetungs-Team alles hat, was sie benötigen.
:::

## Neuordnung von Elementen

Ziehen Sie Elemente am Handle auf der linken Seite jeder Zeile, um die Reihenfolge zu ändern -- drop auf die blaue Linie, wo Sie das Element landen möchten, oder auf eine Abschnitts-Überschrift, um es in diesen Abschnitt zu verschieben. Die auf dem Bildschirm angezeigte Abfolge ist die Reihenfolge, die Ihr Team während des Gottesdienstes befolgen wird.

## Timing und Medien

Jedes Element und jeder Abschnitt zeigt, wie lange er dauert und wann er beginnt:

- Wenn der Plan eine einzelne Servicezeit ausgewählt hat, zeigt die linke Schiene die tatsächliche **Uhrzeit**, bei der jedes Element beginnt (z.B. 9:14 Uhr). Andernfalls zeigt es verstrichene Zeit vom Anfang des Gottesdienstes.
- Abschnitts-Überschriften addieren sich der Laufzeit von allem darin.
- Lieder und benutzerdefinierte Elemente zählen zu diesem Zeitplan, sobald Sie **Minuten**/**Sekunden** auf dem Bearbeitungsbildschirm des Elements eingeben.
- Elemente, die Sie aus Ihrer Medienbibliothek hinzufügen (Bilder oder Videoclips), zeigen eine **Miniaturansicht** und sind automatisch zeitgesteuert -- die Dauer eines Videos wird von der Datei selbst gemessen, während ein Bild einen ~5:00-Planungsschätzung beiträgt (in Kursivschrift angezeigt), da er auf dem Bildschirm bleibt, bis ein Betreiber ihn vorschiebt. Sie können immer noch beide auf eine bestimmte Dauer bearbeiten.

## Bearbeiten und Entfernen von Elementen

- Klicken Sie auf ein Element in der Servicefolge, um seine Details zu **bearbeiten**.
- Verwenden Sie die Option **Löschen**, um ein Element aus der Reihenfolge zu entfernen.

:::info
Die Servicefolge ist spezifisch für jeden Plan. Änderungen, die Sie hier vornehmen, beeinflussen nur den gewählten Plan und verändern keine anderen Pläne oder Ihre Lied-Bibliothek.
:::

## Eintrag von Elementen in bestimmten Servicezeiten

Wenn Ihr Plan mehr als eine Servicezeit hat (z.B. einen 8:00 Uhr und 10:00 Uhr Gottesdienst), können Sie wählen, welche Services jedes Element in sich haben sollte. Dies ist hilfreich, wenn eine Ankündigung nur für einen Gottesdienst relevant ist oder wenn ein Lied in einem Gottesdienst gesungen wird, nicht in einem anderen.

1. Öffnen Sie einen Plan mit zwei oder mehr Servicezeiten, die auf der Registerkarte **Zeiten** definiert sind.
2. Klicken Sie auf der Registerkarte **Servicefolge** auf ein Element, um es zu bearbeiten.
3. Unter **In Services enthalten**, sehen Sie ein Kontrollkästchen für jede Servicezeit, gekennzeichnet mit der Zeit.
4. Deaktivieren Sie einen Gottesdienst, in dem das Element übersprungen werden soll.
5. Klicken Sie auf **Speichern**.

Standardmäßig ist jedes Element in jedem Gottesdienst enthalten. Ausgeschlossene Servicezeiten sind ausgeblendet, wenn Sie den Plan gefiltert auf diesen Gottesdienst ausdrucken, sodass jeder Gottesdienst ein sauberes Datenblatt mit nur den relevanten Elementen erhält.

:::tip
Der Bereich **In Services enthalten** wird nur angezeigt, wenn der Plan mehr als eine Servicezeit hat. Wenn Sie nur einen Gottesdienst sehen, bearbeiten Sie zuerst die Registerkarte **Zeiten** des Plans, um zusätzliche Services hinzuzufügen.
:::

## Freigeben der Servicefolge

Nachdem Ihre Servicefolge abgeschlossen ist, können Sie den vollständigen Plan (einschließlich der Servicefolge) von der Plandetailseite aus ausdrucken. Dies gibt Ihrem Team einen vollständigen Überblick über den Gottesdienst.
