# Gottesdienstablauf

<div class="article-intro">

Der Gottesdienstablauf definiert die Abfolge der Elemente in Ihrem Gottesdienst. Nutzen Sie ihn, um alles vom Eröffnungslied bis zum Schlussgebet festzulegen, damit Ihr gesamtes Team den Ablauf des Gottesdienstes im Voraus kennt.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Erstellen Sie einen Dienstplan](./plans.md) für das Datum, für das Sie einen Gottesdienstablauf aufbauen möchten
- Fügen Sie optional [Lieder](./songs.md) zu Ihrer Bibliothek hinzu, damit Sie sie direkt in den Gottesdienstablauf verlinken können

</div>

## Auf den Gottesdienstablauf zugreifen

1. Navigieren Sie zu **Dienst** und wählen Sie Ihren Dienstbereich.
2. Klicken Sie auf einen **Plantyp** und öffnen Sie dann einen bestimmten **Plan**.
3. Klicken Sie auf die Registerkarte **Gottesdienstablauf** auf der Plan-Detailseite.

## Ihren Gottesdienstablauf aufbauen

Der Gottesdienstablauf ist eine Liste von Elementen, die in der Reihenfolge angeordnet sind, in der sie während des Gottesdienstes auftreten. Typische Elemente sind:

- Lobpreislieder
- Gebete
- Schriftlesungen
- Ankündigungen
- Predigt oder Botschaft
- Kollekte
- Abschluss

## Elemente zum Gottesdienstablauf hinzufügen

1. Klicken Sie auf der Registerkarte **Gottesdienstablauf** auf **Element hinzufügen**.
2. Wählen Sie den Typ des Elements, das Sie hinzufügen möchten (zum Beispiel ein Lied aus Ihrer Bibliothek, ein benutzerdefiniertes Element oder eine Überschrift).
3. Geben Sie die Details für das Element ein, wie Titel und Notizen.
4. Klicken Sie auf **Speichern**.

:::tip
Sie können Lieder direkt aus Ihrer [Lieder](./songs.md)-Bibliothek hinzufügen. Dadurch werden die Lieddetails einschließlich Texte und Arrangements mit Ihrem Dienstplan verknüpft, sodass Ihr Lobpreisteam alles hat, was es braucht.
:::

## Elemente neu ordnen

Ziehen Sie Elemente per Drag & Drop, um die Reihenfolge zu ändern. Die auf dem Bildschirm angezeigte Abfolge ist die Reihenfolge, der Ihr Team während des Gottesdienstes folgt.

## Elemente bearbeiten und entfernen

- Klicken Sie auf ein beliebiges Element im Gottesdienstablauf, um seine Details zu **bearbeiten**.
- Verwenden Sie die Option **Löschen**, um ein Element aus dem Ablauf zu entfernen.

:::info
Der Gottesdienstablauf ist spezifisch für jeden Plan. Änderungen, die Sie hier vornehmen, betreffen nur den ausgewählten Plan und ändern keine anderen Pläne oder Ihre Liederbibliothek.
:::

## Den Gottesdienstablauf teilen

Sobald Ihr Gottesdienstablauf fertig ist, können Sie den vollständigen Plan (einschließlich des Gottesdienstablaufs) von der Plan-Detailseite aus drucken. So erhält Ihr Team eine vollständige Übersicht des Gottesdienstes.

## Nächste Schritte

- Verwalten Sie Ihr Lobpreisrepertoire auf der Seite [Lieder](./songs.md)
- Weisen Sie Ehrenamtliche dem Gottesdienst auf der Registerkarte [Pläne](./plans.md) Einteilungen zu
- Erstellen Sie [Aufgaben](./tasks.md) für Nachfolge-Aktionspunkte im Zusammenhang mit dem Gottesdienst
