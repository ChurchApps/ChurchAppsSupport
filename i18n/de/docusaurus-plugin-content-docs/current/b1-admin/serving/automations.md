# Automatisierungen

<div class="article-intro">

Automatisierungen erstellen Aufgaben automatisch nach einem wiederkehrenden Zeitplan. Anstatt jede Woche oder jeden Monat dieselben Aufgaben manuell zu erstellen, können Sie eine Automatisierung einmal einrichten und das System die Arbeit erledigen lassen, damit nichts vergessen wird.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Machen Sie sich vertraut, wie [Aufgaben](./tasks.md) in B1 Admin funktionieren
- Identifizieren Sie, welche wiederkehrenden Verantwortlichkeiten Sie automatisieren möchten

</div>

## Automatisierungen anzeigen

Navigieren Sie zu **Dienst** und öffnen Sie die Seite **Automatisierungen**. Sie sehen eine Liste aller Ihrer Automatisierungen mit Titel, Aktivstatus und Wiederholungsmuster.

## Eine Automatisierung erstellen

1. Klicken Sie auf der Automatisierungsseite auf **Automatisierung hinzufügen**.
2. Geben Sie einen **Titel** für die Automatisierung ein. Dieser wird auch als Titel der erstellten Aufgaben verwendet.
3. Legen Sie das **Wiederholungsmuster** fest -- wählen Sie, wie oft die Aufgabe erstellt werden soll (zum Beispiel wöchentlich, monatlich oder an bestimmten Tagen).
4. Weisen Sie die Aufgabe einer **Person oder Gruppe** zu, die jedes Mal verantwortlich sein wird.
5. Setzen Sie die Automatisierung auf **Aktiv**, damit sie planmäßig mit der Erstellung von Aufgaben beginnt.
6. Klicken Sie auf **Speichern**.

## Eine Automatisierung bearbeiten

1. Klicken Sie auf eine bestehende Automatisierung in der Liste, um sie zu öffnen.
2. Aktualisieren Sie Titel, Wiederholungsmuster, Zuweisung oder andere Einstellungen.
3. Klicken Sie auf **Speichern**, um Ihre Änderungen zu übernehmen.

## Aktivieren und Deaktivieren

Sie können steuern, ob eine Automatisierung läuft, ohne sie zu löschen:

- **Aktiv** -- Die Automatisierung erstellt Aufgaben gemäß ihrem Zeitplan.
- **Inaktiv** -- Die Automatisierung ist pausiert und erstellt keine neuen Aufgaben, bis Sie sie reaktivieren.

:::tip
Setzen Sie eine Automatisierung während Feiertagen oder besonderen Zeiten auf **Inaktiv**, wenn die wiederkehrende Aufgabe nicht benötigt wird. Sie können sie jederzeit reaktivieren, ohne Ihre Konfiguration zu verlieren.
:::

## So funktioniert es

Wenn eine Automatisierung ausgeführt wird, erstellt sie eine neue [Aufgabe](./tasks.md) mit dem konfigurierten Titel und der Zuweisung. Die Aufgabe verhält sich wie jede andere Aufgabe -- Zugewiesene erhalten Benachrichtigungen und können die Aufgabe von ihrem Dashboard oder der Aufgabenseite aus verwalten.

:::info
Automatisierungen erstellen nur neue Aufgaben in die Zukunft. Sie erstellen keine Aufgaben rückwirkend für vergangene Daten.
:::
