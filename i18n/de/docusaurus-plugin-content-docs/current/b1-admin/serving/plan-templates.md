# Planvorlagen

<div class="article-intro">

Mit Planvorlagen können Sie eine wiederverwendbare Momentaufnahme der Dienstordnung oder Positionen eines Plans speichern und diese mit einem Klick auf zukünftige Pläne anwenden. Anstatt jede Woche dieselbe Struktur neu zu erstellen, speichern Sie sie einmal und stempeln sie auf beliebig viele bevorstehende Pläne auf.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Erstellen Sie mindestens einen [Dienstplan](plans.md) mit einer definierten Dienstordnung oder Positionen
- Sie benötigen Zugriff auf den Bereich „Dienste" in B1 Admin

</div>

## Speichern Sie einen Plan als Vorlage

1. Öffnen Sie einen bestehenden Plan aus dem Bereich **Dienste**.
2. Klicken Sie auf die Schaltfläche **Templates** (oben auf der Plan-Seite).
3. Wählen Sie **Save as Template**.
4. Geben Sie der Vorlage einen Namen (z. B. „Standard Sunday" oder „Worship + Sermon").
5. Speichern Sie. Die Vorlage erfasst die aktuelle Dienstordnung und Positionen des Plans zu diesem Zeitpunkt.

## Verwalten Sie Vorlagen

Von der Schaltfläche **Templates** auf einer beliebigen Plantyp-Seite können Sie:

- **Rename** – ändern Sie den Anzeigenamen der Vorlage
- **Update from plan** – ersetzen Sie den Vorlagenininhalt mit dem aktuellen Status eines Plans (nützlich, nachdem Sie Ihre Standardstruktur verfeinert haben)
- **Apply** – stempeln Sie die Vorlage auf einen oder mehrere bevorstehende Pläne auf
- **Delete** – entfernen Sie eine Vorlage, die Sie nicht mehr benötigen

## Wenden Sie eine Vorlage auf Pläne an

1. Klicken Sie auf der Plantyp-Seite auf die Schaltfläche **Templates**.
2. Klicken Sie auf das Symbol **Apply** (Kopiersymbol) neben der Vorlage, die Sie verwenden möchten.
3. Wählen Sie, welche bevorstehenden Pläne Sie anwenden möchten, indem Sie ihre Kontrollkästchen aktivieren.
4. Wählen Sie, was zu kopieren ist:
   - **Order of Service** – ersetzt die Dienstordnungselemente der ausgewählten Pläne durch die Elemente der Vorlage
   - **Positions** – ersetzt die Positionen (und ihre Aufträge) der ausgewählten Pläne durch die Positionen der Vorlage
5. Klicken Sie auf **Apply**.

:::warning
Das Anwenden von Positionen ersetzt alle bestehenden Positionen und Aufträge auf den ausgewählten Plänen. Stellen Sie sicher, dass Sie nicht bereits Freiwillige für diese Positionen eingeplant haben, bevor Sie anwenden.
:::

:::tip
Wenden Sie die Dienstordnung ohne Positionen an, wenn Sie eine konsistente Lied-/Predit-Struktur möchten, aber Freiwillige jede Woche unterschiedlich zugewiesen werden.
:::

## Halten Sie Vorlagen auf dem neuesten Stand

Wenn sich Ihre Standard-Dienststruktur ändert, öffnen Sie den Vorlagen-Manager und verwenden Sie **Update from plan**, um die Vorlage aus Ihrem neuesten Plan zu synchronisieren, anstatt sie von Grund auf neu zu erstellen.
