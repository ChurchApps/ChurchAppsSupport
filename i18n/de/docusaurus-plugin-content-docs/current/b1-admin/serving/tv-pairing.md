# Verbinden Sie einen Klassen-Fernseher mit Ihren B1 Admin-Lektionen

<div class="article-intro">

Sie können einen Klassen-Fernseher mit Ihren B1 Admin-Lektionen mit **FreePlay** verbinden — ChurchApps' kostenlose Fernseh-App für Kirchen. Sobald verbunden, zeigt der Fernseher den Lektion-Inhalt an, den Sie in B1 Admin geplant haben, was ihn für Ihre Lehrer direkt auf dem Klassen-Bildschirm verfügbar macht.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Planen Sie mindestens eine Lektion in B1 Admin (siehe [Lektionen planen](./scheduling-lessons.md))
- Installieren Sie **FreePlay** auf dem Fernseher oder Fire Stick, den Sie verwenden möchten

</div>

## Schritt 1 — Installieren Sie FreePlay auf Ihrem Fernseher

Installieren Sie die **FreePlay**-App auf Ihrem Klassen-Display:

- **Amazon Fire Stick / Fire TV** — Suchen Sie nach **FreePlay** im Amazon Appstore
- **Android TV / Google TV** — Suchen Sie nach **FreePlay** im Google Play Store
- **Apple TV** — Suchen Sie nach **FreePlay** im App Store

Eine vollständige Setup-Anleitung finden Sie im [FreePlay Getting Started-Anleitung](/docs/freeplay/getting-started/).

## Schritt 2 — Verbinden Sie B1.church als Inhalts-Anbieter

1. Öffnen Sie die **FreePlay**-App auf dem Fernseher
2. Navigieren Sie zum Bildschirm **Inhalts-Anbieter** (wählen Sie das **Anbieter**-Symbol in der Seitenleiste)
3. Finden und wählen Sie **B1.church** aus der Liste verfügbarer Anbieter
4. Folgen Sie den On-Screen-Aufforderungen, um sich mit Ihren B1-Kirchen-Anmeldedaten anzumelden
5. FreePlay zeigt **Verbunden!** an, sobald die Verbindung erfolgreich ist

Sobald verbunden, erscheint B1.church in Ihrer FreePlay-Seitenleiste. FreePlay wird den Lektion-Inhalt abrufen, den Sie in B1 Admin geplant haben, damit Ihre Lehrer ihn direkt auf dem Klassen-Fernseher spielen können.

:::tip
Nach dem Verbinden können Ihre Lehrer die Fernbedienung des Fernsehers verwenden, um Videos und Folien in ihrem eigenen Tempo vorzuführen, während die Lektion fließt. Kein separates Setup oder USB-Laufwerke erforderlich.
:::

:::info
Für ausführliche Schritte zur Authentifizierung bei einem Inhalts-Anbieter (QR-Code, Form-Anmeldung, etc.), siehe [Mit Anbietern verbinden](/docs/freeplay/content-providers/connecting-providers).
:::
