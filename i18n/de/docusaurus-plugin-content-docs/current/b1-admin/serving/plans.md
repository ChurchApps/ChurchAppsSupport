# Dienst-Pläne

<div class="article-intro">

Dienst-Pläne organisieren, wer dient und wann. Jeder Plan ist an ein bestimmtes Datum und einen bestimmten Dienst gebunden und macht es einfach, Ihre Freiwilligen-Teams Woche für Woche zu koordinieren und sicherzustellen, dass jeder Gottesdienst vollständig besetzt ist.

</div>

<div class="prereqs">
<h4>Bevor Sie Beginnen</h4>

- Richten Sie Ihre Dienste und Teams im Bereich Serving ein
- Stellen Sie sicher, dass Freiwillige Ihrem [Verzeichnis](../people/adding-people.md) hinzugefügt wurden und Teams zugewiesen wurden

</div>

## Zugriff auf Pläne

1. Navigieren Sie zu **Serving** aus dem Hauptmenü.
2. Wählen Sie einen **Dienst-Tab** oben auf der Seite.
3. Klicken Sie auf einen **Plantyp**, um die Liste der Pläne für diesen Typ anzuzeigen.
4. Klicken Sie auf einen bestimmten Plan, um ihn zu öffnen.

:::info
Für die Verwaltung von Plänen ist kein vollständiger Admin-Zugriff erforderlich. Jeder, der Mitglied eines Dienstes ist, kann zum Bereich Serving navigieren und Pläne für seinen eigenen Dienst erstellen, bearbeiten und planen, ohne die Berechtigung „Plans Edit" zu benötigen. Bearbeiter mit der Rolle „Plans Edit" können Pläne in allen Diensten verwalten.
:::

## Einen Plan Erstellen

1. Klicken Sie in der Plantyp-Ansicht auf **New Plan**.
2. Geben Sie dem Plan einen Namen oder verwenden Sie das Datum als Namen. Wählen Sie das **Datum** für den Gottesdienst.
3. Wenn Sie von einem vorherigen Plan kopieren möchten, wählen Sie nur Positionen oder Positionen und Zuweisungen. Wenn Sie nicht kopieren möchten, wählen Sie nichts. Sie können auch die Reihenfolge des Gottesdiensts aus meinem vorherigen Plan kopieren.
4. Speichern Sie den Plan. Sie können jetzt mit der Zuweisung von Teammitgliedern und dem Aufbau der [Dienstreihenfolge](./service-order.md) beginnen.

## Die Plan-Detailseite

Wenn Sie einen Plan öffnen, sehen Sie zwei Tabs:

- **Assignments** – Verwalten Sie, welche Teammitglieder diesem Plan zugewiesen sind. Sie können Personen aus Ihren vorhandenen Teams hinzufügen und sehen, wer bestätigt hat oder noch ausstehend ist.
- **[Service Order](./service-order.md)** – Erstellen Sie die Reihenfolge des Gottesdiensts mit Elementen wie Anbetungsliedern, Gebeten, Ankündigungen und der Predigt.

## Zuweisen von Teammitgliedern

1. Öffnen Sie einen Plan und gehen Sie zum Tab **Assignments**.
2. Klicken Sie auf **add Position**, um ihn zu erweitern. Füllen Sie die Informationen im Formular zum Hinzufügen einer Position aus. Für den Kategorienamen fügen Sie jede beliebige Kategorie hinzu.
3. Klicken Sie auf **People Needed** und wählen Sie Freiwillige für diese Position aus.
4. Fügen Sie Mitglieder aus Ihrem Team-Verzeichnis hinzu, indem Sie auf **Add** klicken.
5. Zugewiesene Mitglieder werden unter ihrem Team mit ihrem Zuweisungsstatus angezeigt.
6. Klicken Sie auf „Freiwillige benachrichtigen", um sie in der B1-App oder per E-Mail zu benachrichtigen.

:::tip
Richten Sie Ihre Teams in den Dienst-Einstellungen ein, bevor Sie Pläne erstellen. Auf diese Weise haben Sie einen einsatzbereiten Pool von Freiwilligen zum Zuweisen.
:::

## Verbinden von Gruppen mit einem Plantyp

Unterhalb der Planliste auf der Plantyp-Seite können Sie mit dem Abschnitt **Groups** entscheiden, welche Gruppen die Pläne für diesen Plantyp aus ihrem Mitglieder-Portal sehen können. Dies ist eine schnelle Möglichkeit, bevorstehende Gottesdienste für die richtigen Teams anzuzeigen, ohne ihnen Admin-Zugriff zu geben.

1. Scrollen Sie auf der Plantyp-Seite nach unten zum Abschnitt **Groups**.
2. Klicken Sie auf **Add Group** und wählen Sie eine Gruppe aus dem Dropdown aus.
3. Wählen Sie in der Spalte **Shows**, ob Mitglieder dieser Gruppe **Past**, **Future** oder **Both** Pläne für diesen Plantyp sehen sollten.
4. Wiederholen Sie dies, um zusätzliche Gruppen zu verknüpfen, oder klicken Sie auf das Trash-Symbol, um eine Gruppe zu entfernen.

:::info
Nur Gruppen, die als **Standard** gekennzeichnet sind, werden in der Auswahl angezeigt. Mitglieder einer verknüpften Gruppe sehen automatisch die Pläne dieses Plantyps auf der Registerkarte [Plans](/docs/b1-church/plans/) im B1-Mitglieder-Portal – begrenzt auf das Fenster für Vergangenheit/Zukunft/beide, das Sie ausgewählt haben.
:::

## Drucken von Plänen

Sie können einen Plan für die Verteilung an Ihr Team drucken. Öffnen Sie den Plan, öffnen Sie den Dienstreihenfolge-Tab und verwenden Sie die Option **Print**, um eine druckbare Version zu generieren, die Zuweisungen und die Dienstreihenfolge enthält. Dies ist nützlich, um bei Proben auszugeben oder an einem gemeinsamen Ort zu veröffentlichen.

:::info
Pläne werden nach Dienst organisiert. Stellen Sie sicher, dass Sie auf der richtigen Dienst-Registerkarte sind, bevor Sie Pläne erstellen oder anzeigen.
:::

## Nächste Schritte

- Verwenden Sie die [Pläne-Übersicht](./plans-overview.md), um alle bevorstehenden Zuweisungen in mehreren Wochen in einem Raster anzuzeigen und unbesetzte Positionen zu finden
- Erstellen Sie Ihre [Dienstreihenfolge](./service-order.md) mit Liedern, Lesungen und anderen Elementen
- Fügen Sie [Lieder](./songs.md) aus Ihrer Bibliothek direkt in die Dienstreihenfolge ein
- Verwenden Sie [Aufgaben](./tasks.md), um Folgemaßnahmen an Teammitglieder zu vergeben
