# Plan-Validierung & Freiwilligs-Benachrichtigungen

<div class="article-intro">

B1 Admin überprüft automatisch Ihre Pläne auf Probleme vor dem Sonntag — ungefüllte Positionen, Zeitplanungs-Konflikte und Freiwillige, die das Datum blockiert haben. Wenn alles gut aussieht, können Sie Ihr gesamtes Team mit einem einzigen Klick benachrichtigen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Erstellen Sie einen [Dienst-Plan](./plans.md) und weisen Sie Freiwillige Positionen zu
- Fügen Sie [Gottesdienst-Zeiten](./plans.md) zum Plan hinzu, damit die Konflikt-Erkennung Überlappungen überprüfen kann
- Stellen Sie sicher, dass Freiwillige die B1 Mobile-App installiert haben, um Push-Benachrichtigungen zu erhalten

</div>

## Das Validierungs-Panel

Jeder Plan hat ein **Validierungs**-Panel, das automatisch läuft, während Sie es aufbauen. Es überprüft drei Dinge:

### Ungefüllte Positionen
Wenn eine Position mehr Personen erfordert, als derzeit zugewiesen sind, listet das Validierungs-Panel genau auf, was noch benötigt wird — zum Beispiel, *"Sound Tech: 1 weitere Person benötigt."* Sie können auf einen Blick sehen, ob Ihr Plan vor der Woche vollständig besetzt ist.

### Zeitplanungs-Konflikte
Wenn ein Freiwilliger zwei Positionen zugewiesen wird, die zeitlich überlappen, kennzeichnet das Validierungs-Panel den Konflikt — zum Beispiel, *"Jane Smith: Zeit-Konflikt zwischen Lob-Leiter und Kindercheck-In während Sonntagsgottesdienst."* Dies erfasst Doppelreservierungen, bevor sie zu einem Sonntagsmorgen-Problem werden.

### Blockierungs-Daten
Freiwillige können Daten setzen, an denen sie in B1 Mobile nicht verfügbar sind. Wenn jemand einem Plan zugewiesen wird, der innerhalb einer ihrer Blockierungs-Daten fällt, zeigt das Validierungs-Panel den Konflikt automatisch auf, damit Sie einen Ersatz finden können.

### Übergreifende Plan-Konflikte
Die Validierung überprüft auch alle Ihre Pläne auf einmal. Wenn derselbe Freiwilliger in zwei verschiedenen Plänen zugewiesen ist, die zeitlich überlappen — zum Beispiel ein 9 Uhr Gottesdienst und ein 10 Uhr Gottesdienst, die beide bis 10:30 Uhr laufen — kennzeichnet B1 Admin diese Person als doppelt gebucht über Pläne hinweg.

:::tip
Sie müssen nichts tun, um Validierung auszuführen — es aktualisiert sich automatisch jedes Mal, wenn Sie einen Auftrag hinzufügen oder ändern. Halten Sie einfach ein Auge auf das Panel, während Sie den Plan aufbauen.
:::

## Benachrichtigen Sie Freiwillige

Sobald Ihr Plan gesetzt ist, können Sie alle zugewiesenen Freiwilligen auf einmal direkt vom Validierungs-Panel aus benachrichtigen.

1. Öffnen Sie den Plan und scrollen Sie zum **Validierungs**-Panel
2. Wenn es nicht benachrichtigte Freiwillige gibt, sehen Sie einen Link mit der Anzahl, die benachrichtigt werden muss (z.B., *"8 Freiwillige benachrichtigen"*)
3. Klicken Sie auf den Link, um Push-Benachrichtigungen an alle zu senden, die nicht benachrichtigt wurden
4. Freiwillige erhalten eine Benachrichtigung auf ihrem Telefon, dass sie eingeplant wurden und aufgefordert, ihren Auftrag zu bestätigen

:::info
Nur Freiwillige, die nicht benachrichtigt wurden, werden einbezogen. Wenn Sie später jemanden zum Plan hinzufügen, erscheint der Link erneut, damit Sie die neue Ergänzung benachrichtigen können, ohne den Rest des Teams erneut zu benachrichtigen.
:::

:::warning
Freiwillige müssen die B1 Mobile-App installiert haben und Benachrichtigungen aktiviert haben, um die Push-Benachrichtigung zu erhalten. Siehe das [Benachrichtigungs-Anleitung](/docs/b1-mobile/community/notifications), um wie Freiwillige dies auf ihrem Gerät aktivieren können.
:::
