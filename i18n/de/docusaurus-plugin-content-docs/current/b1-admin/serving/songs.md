# Lieder

<div class="article-intro">

Die Liederseite verwaltet Ihre Lobpreislieder-Bibliothek. Behalten Sie alle Lieddetails, Texte und Arrangements an einem Ort, damit Ihr Lobpreisteam sich einfach auf kommende Gottesdienste vorbereiten und ein einheitliches Repertoire pflegen kann.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Navigieren Sie zum Bereich **Dienst** in B1 Admin, um auf die Liederseite zuzugreifen
- Richten Sie optional Ihre [Dienstpläne](./plans.md) ein, damit Sie Lieder direkt in den [Gottesdienstablauf](./service-order.md) verlinken können

</div>

## Ihre Liederbibliothek durchstöbern

Wenn Sie die Seite **Lieder** öffnen, werden Ihre Lieder als Karten angezeigt. Jede Karte zeigt das Vorschaubild, den Titel, den Interpreten und die Dauer des Liedes. Scrollen Sie durch die Bibliothek oder nutzen Sie die Suche, um das Gewünschte zu finden.

## Nach Liedern suchen

1. Klicken Sie auf der Liederseite auf **Suchen**.
2. Geben Sie einen Lied-**Titel** oder **Interpreten**-Namen ein.
3. Die Ergebnisse werden während des Tippens gefiltert und zeigen passende Lieder aus Ihrer Bibliothek.

## Ein Lied hinzufügen

1. Klicken Sie auf der Liederseite auf **Lied hinzufügen**.
2. Suchen Sie in der externen Lied-Datenbank nach Titel oder Interpret.
3. Wählen Sie das Lied aus, das Sie hinzufügen möchten.
4. Das Lied wird mit vorausgefüllten Grunddaten zu Ihrer Bibliothek hinzugefügt.

:::tip
Klicken Sie nach dem Hinzufügen eines Liedes darauf, um seine Details zu überprüfen und zu aktualisieren, Arrangements hinzuzufügen oder externe Links wie YouTube-Videos oder Akkordblätter anzuhängen.
:::

## Ein Lied anzeigen und bearbeiten

Klicken Sie auf eine beliebige Liedkarte, um die Detailseite zu öffnen. Von hier aus können Sie:

- **Lieddetails bearbeiten** -- Titel, Interpret, Dauer und andere Metadaten aktualisieren.
- **Arrangements verwalten** -- Jedes Lied kann mehrere Arrangements haben (zum Beispiel verschiedene Tonarten oder Versionen). Fügen Sie Arrangements nach Bedarf hinzu, bearbeiten oder entfernen Sie sie.
- **Texte hinzufügen** -- Geben Sie Texte für jedes Arrangement ein oder aktualisieren Sie sie.
- **Tonart festlegen** -- Geben Sie die musikalische Tonart für jedes Arrangement an, damit Ihr Team weiß, wie es sich vorbereiten muss.
- **Externe Links hinzufügen** -- Hängen Sie Links zu Ressourcen wie YouTube-Videos, Akkordblättern oder Noten an, die anderswo gehostet werden.

## Arrangements

Ein einzelnes Lied kann mehrere Arrangements haben, um verschiedene Gottesdienste oder Teamkonfigurationen abzudecken. Jedes Arrangement enthält:

- **Texte** zur Anzeige oder Projektion
- **Tonart**-Bezeichnung
- **Externe Links** zu zusätzlichen Ressourcen

:::info
Lieder aus Ihrer Bibliothek können direkt zu einem [Gottesdienstablauf](./service-order.md) innerhalb eines Plans hinzugefügt werden, wodurch die Lieddetails und Arrangements mit diesem bestimmten Gottesdienst verknüpft werden.
:::

## Nächste Schritte

- Fügen Sie Lieder zu Ihrem [Gottesdienstablauf](./service-order.md) hinzu, wenn Sie einen [Dienstplan](./plans.md) aufbauen
- Koordinieren Sie sich mit Ihrem Lobpreisteam über [Aufgaben](./tasks.md) für die Liedvorbereitung und Proben
