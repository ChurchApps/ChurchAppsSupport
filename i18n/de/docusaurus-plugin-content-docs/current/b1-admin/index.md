# B1 Admin Überblick

<div class="article-intro">

B1 Admin ist Ihr Kirchenverwaltungs-Dashboard. Es bietet Tools zur Verwaltung jeden Aspekt des Betriebs Ihrer Kirche -- von Mitgliedverzeichnissen und Spenden bis hin zum Website-Bau und Live-Streaming -- alles an einem Ort.

</div>

## Erste Schritte

1. Öffnen Sie Ihren Browser und gehen Sie zu [admin.b1.church](https://admin.b1.church)
2. Melden Sie sich mit Ihrem Benutzernamen und Passwort an
3. Sie landen auf dem **Dashboard**, das einen Überblick über Ihre Kirchendaten zusammen mit **Schnellaktionen** für häufige Aufgaben wie das Hinzufügen von Personen, das Erstellen von Gruppen, das Aufzeichnen von Spenden und mehr bietet

:::tip
Neu bei B1 Admin? Beginnen Sie mit der [Einführung](./introduction) für eine Video-Anleitung, und besuchen Sie dann [Einstellungen](./settings/) um Ihre Kircheninformationen zu konfigurieren und Ihr Team einzuladen.
:::

## Hauptmerkmale

- **[Personen](./people/)** - Verwalten Sie Ihr Kirchenverzeichnis, fügen Sie Mitglieder hinzu, bearbeiten Sie Daten in Massen, verfolgen Sie Haushalte
- **[Gruppen](./groups/)** - Erstellen und organisieren Sie Kirchengruppen mit Beirittsanfragen und Mitgliederverwaltung
- **[Anwesenheit](./attendance/)** - Richten Sie Campi, Gottesdienstzeiten ein und verfolgen Sie die Anwesenheit
- **[Spenden](./donations/)** - Spenden in mehreren Währungen aufzeichnen, Fonds verwalten, Erklärungen generieren
- **[Dienste](./serving/)** - Koordinieren Sie Freiwillige, erstellen Sie Servicepläne, verwalten Sie Aufgaben und bewegen Sie Personen durch Schritt-für-Schritt-Arbeitsabläufe
- **[Formulare](./forms/)** - Erstellen Sie benutzerdefinierte Formulare für Registrierungen und Datenerfassung
- **[Berichte](./reports/)** - Anzeige von Geburtstags-, Anwesenheits- und Spendenbergichten
- **[Website](./website/)** - Erstellen und verwalten Sie Ihre Kirchenwebsite mit benutzerdefinierten Navigationsstilen
- **[Predigten](./sermons/)** - Verwalten Sie Ihre Predigtbibliothek und Live-Streaming
- **[Kalender](./calendars/)** - Erstellen Sie kuratierte Kalender, verwalten Sie Raum-/Ressourcenbuchungen und sehen Sie Verfügbarkeit
- **[Einstellungen](./settings/)** - Konfigurieren Sie Kircheninformationen, Rollen, Berechtigungen und benutzerdefinierte Personenfelder
