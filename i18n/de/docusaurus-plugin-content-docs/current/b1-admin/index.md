# B1 Admin Übersicht

<div class="article-intro">

B1 Admin ist Ihr Kirchenverwaltungs-Dashboard. Es bietet Tools zur Verwaltung aller Aspekte der Abläufe Ihrer Kirche -- von Mitgliederverzeichnissen und Spenden bis hin zum Website-Aufbau und Live-Streaming -- alles von einem Ort aus.

</div>

## Erste Schritte

1. Öffnen Sie Ihren Browser und gehen Sie zu [admin.b1.church](https://admin.b1.church)
2. Melden Sie sich mit Ihrem Benutzernamen und Passwort an
3. Sie landen auf dem **Dashboard**, das einen Überblick über Ihre Kirchendaten sowie **Quick Actions** für häufige Aufgaben wie das Hinzufügen von Personen, Erstellen von Gruppen, Erfassen von Spenden und mehr bietet

:::tip
Neu in B1 Admin? Beginnen Sie mit der [Introduction](./introduction) für eine Video-Einführung und besuchen Sie dann [Settings](./settings/), um Ihre Kircheninformationen zu konfigurieren und Ihr Team einzuladen.
:::

## Hauptfunktionen

- **[People](./people/)** - Verwalten Sie Ihr Kirchenverzeichnis, fügen Sie Mitglieder hinzu, verfolgen Sie Haushalte
- **[Groups](./groups/)** - Erstellen und organisieren Sie Kirchengruppen, Teams und Dienste
- **[Attendance](./attendance/)** - Richten Sie Gemeinden, Gottesdienstzeiten ein und verfolgen Sie die Teilnahme
- **[Donations](./donations/)** - Erfassen Sie Spenden, verwalten Sie Fonds, erstellen Sie Spendenquittungen
- **[Serving](./serving/)** - Koordinieren Sie Freiwillige, erstellen Sie Dienstpläne, verwalten Sie Aufgaben
- **[Forms](./forms/)** - Erstellen Sie benutzerdefinierte Formulare für Registrierungen und Datenerfassung
- **[Reports](./reports/)** - Zeigen Sie Geburtstags-, Teilnahme- und Spendenberichte an
- **[Website](./website/)** - Erstellen und verwalten Sie Ihre Kirchenwebsite
- **[Sermons](./sermons/)** - Verwalten Sie Ihre Predigtbibliothek und Live-Streaming
- **[Calendars](./calendars/)** - Erstellen Sie kuratierte Kalender durch Verbindung von Gruppenveranstaltungen
- **[Settings](./settings/)** - Konfigurieren Sie Kircheninformationen, Rollen und Berechtigungen
