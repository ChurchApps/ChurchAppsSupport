# Erstellen Sie Ihr Konto

<div class="article-intro">

Der Einstieg in ChurchApps beginnt mit der Erstellung Ihres Kontos. Der Prozess dauert nur wenige Minuten und gibt Ihnen Zugriff auf alle ChurchApps-Tools, einschließlich B1 Admin, B1.church und Lessons.church.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Haben Sie eine gültige E-Mail-Adresse für die Kontoüberprüfung bereit
- Überprüfen Sie die [Erste Schritte-Übersicht](./index.md), um zu verstehen, was ChurchApps bietet

</div>

## Schritt-für-Schritt-Registrierung

1. Gehen Sie zu [B1.church](https://b1.church) in Ihrem Webbrowser.
2. Klicken Sie auf die Schaltfläche **Registrierung**.
3. Geben Sie Ihren **Vorname**, **Nachname** und **E-Mail-Adresse** ein.
4. Erstellen Sie ein Passwort für Ihr Konto.
5. Senden Sie das Registrierungsformular ab.

## Bestätigen Sie Ihre E-Mail

Nach der Registrierung erhalten Sie eine Bestätigunge-Mail unter der Adresse, die Sie angegeben haben.

1. Überprüfen Sie Ihren Posteingang auf die Bestätigunge-Mail von ChurchApps.
2. Klicken Sie auf den Bestätigungslink in der E-Mail.
3. Ihr Konto ist nun überprüft und einsatzbereit.

:::tip
Wenn Sie die Bestätigunge-Mail nicht sehen, überprüfen Sie Ihren Spam- oder Junk-Ordner. Die E-Mail kommt von ChurchApps und kann ein oder zwei Minuten dauern.
:::

## Richten Sie Ihre Kirche ein

Nach dem Bestätigen Ihrer E-Mail und der Anmeldung haben Sie die Möglichkeit, eine neue Kirche zu erstellen oder einer bestehenden beizutreten.

**Um eine neue Kirche zu erstellen:**

1. Wählen Sie die Option **eine neue Kirche erstellen**.
2. Geben Sie Ihren **Kirchennamen** und grundlegende Informationen ein. Das Registrierungsformular füllt automatisch Ihre Kirchendaten aus, wenn eine Übereinstimmung gefunden wird.
3. Schließen Sie den Setup-Prozess ab.
4. Sie werden zum **B1 Admin-Dashboard** geleitet, wo Sie mit der Konfiguration Ihrer Kirche beginnen können.

**Um einer bestehenden Kirche beizutreten:**

1. Wählen Sie die Option **einer bestehenden Kirche beitreten**.
2. Suchen Sie Ihre Kirche nach Name.
3. Beantragen Sie Beitritt. Ein Administrator Ihrer Kirche muss Ihren Antrag genehmigen.

:::info
Wenn Sie sich mit einer E-Mail anmelden, die noch nicht registriert ist, werden Sie automatisch zum Registrierungsfluss geleitet, damit Sie ein Konto erstellen können.
:::

:::info
Die Person, die eine neue Kirche erstellt, erhält automatisch die Rolle **Domain Admin**, die vollständigen Zugriff auf alle Einstellungen und Funktionen gewährt. Sie können später weitere Administratoren von der Seite **Rollen** hinzufügen.
:::

## Was kommt danach

Nachdem Ihre Kirche eingerichtet ist, gehen Sie zur [Initial Setup](./initial-setup.md)-Anleitung, um die Einstellungen, das Branding und den Teamzugriff Ihrer Kirche zu konfigurieren.
