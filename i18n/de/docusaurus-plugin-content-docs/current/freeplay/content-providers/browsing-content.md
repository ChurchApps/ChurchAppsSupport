# Inhalte durchsuchen und herunterladen

<div class="article-intro">

Sobald Sie sich mit einem Inhaltsanbieter verbunden haben, koennen Sie dessen Bibliothek durchsuchen, Dateien herunterladen und auf Ihrem Fernseher abspielen. Der Inhaltsbrowser bietet ein visuelles Raster aus Ordnern und Dateien, das Sie mit Ihrer TV-Fernbedienung navigieren.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Verbinden Sie sich mit mindestens einem Inhaltsanbieter -- siehe [Mit Anbietern verbinden](./connecting-providers)
- Halten Sie Ihre TV-Fernbedienung zur Navigation bereit

</div>

## Der Inhaltsbrowser

Der Inhaltsbrowser zeigt Elemente in einem 3-spaltigen Kartenraster an. Jede Karte zeigt ein Vorschaubild und einen Titel. Es gibt zwei Arten von Elementen:

- **Ordner** - Dargestellt durch ein Vorschaubild oder Ordnersymbol. Waehlen Sie einen Ordner aus, um seinen Inhalt zu sehen.
- **Dateien** - Einzelne Medienelemente (Videos oder Bilder). Videodateien zeigen ein Play-Symbol auf ihrem Vorschaubild.

Die Kopfzeile oben am Bildschirm zeigt den Namen des aktuellen Ordners oder den Anbieternamen, wenn Sie sich auf der obersten Ebene befinden.

## Ordner navigieren

1. Verwenden Sie das Steuerkreuz auf Ihrer Fernbedienung, um einen Ordner hervorzuheben
2. Druecken Sie **Select**, um ihn zu oeffnen
3. Der Browser laedt den Inhalt des Ordners, der weitere Ordner oder abspielbare Dateien enthalten kann
4. Druecken Sie **Back** auf Ihrer Fernbedienung, um eine Ebene nach oben zu gehen

Sie koennen so tief navigieren, wie die Inhaltsstruktur des Anbieters es erlaubt. Der Browser merkt sich Ihren Ordnerpfad, sodass Sie durch jede Ebene zurueck navigieren koennen.

## Inhalte herunterladen und abspielen

Wenn Sie einen Ordner mit abspielbaren Dateien auswaehlen (oder eine einzelne Datei auswaehlen), wechselt FreePlay zu einem Download-Bildschirm:

1. Der Inhaltstitel erscheint auf dem Bildschirm mit einem Titelbild aus dem ausgewaehlten Ordner
2. Ein Fortschrittsbalken zeigt **Downloading item X of Y**, waehrend Dateien auf Ihrem Geraet gespeichert werden
3. Sobald der Download abgeschlossen ist, erscheint die Schaltflaeche **Start** mit einem Play-Symbol
4. Druecken Sie **Select**, um die Wiedergabe zu starten

:::tip
Heruntergeladene Inhalte werden auf Ihrem Fernseher zwischengespeichert. Wenn Sie denselben Inhalt spaeter erneut abspielen, wird er sofort geladen, ohne erneut heruntergeladen zu werden.
:::

## Anbieterinhalte abspielen

Die Wiedergabe funktioniert genauso wie die Lektionswiedergabe. Verwenden Sie diese Steuerungen:

- **Rechter Pfeil** - Naechstes Element
- **Linker Pfeil** - Vorheriges Element
- **Select / Play-Pause** - Video pausieren oder fortsetzen
- **Pfeil nach oben** - Nachrichtenauswahl oeffnen, um zu einem bestimmten Element zu springen
- **Back** - Zum Inhaltsbrowser zurueckkehren

Wenn Sie den Player verlassen, kehren Sie zum Inhaltsbrowser auf der obersten Ebene des aktuellen Anbieters zurueck.

:::info
Eine vollstaendige Uebersicht aller Player-Steuerungen finden Sie unter [Lektionen abspielen](../classroom-mode/playing-lessons).
:::

## Zwischen Anbietern wechseln

Verbundene Anbieter erscheinen als Eintraege in der Seitenleiste auf der linken Seite. Um zu einem anderen Anbieter zu wechseln:

1. Druecken Sie den **linken Pfeil** auf Ihrer Fernbedienung, um die Seitenleiste auszuklappen
2. Waehlen Sie den Anbieter, den Sie durchsuchen moechten
3. Der Inhaltsbrowser aktualisiert sich mit der Bibliothek dieses Anbieters

:::warning
Wenn ein Ordner "No content available" anzeigt, hat der Anbieter auf dieser Ebene keine Elemente. Versuchen Sie, zu einem anderen Ordner zu navigieren, oder schauen Sie spaeter noch einmal vorbei.
:::
