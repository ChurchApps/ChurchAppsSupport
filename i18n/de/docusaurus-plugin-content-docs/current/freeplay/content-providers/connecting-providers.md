# Mit Anbietern verbinden

<div class="article-intro">

Bevor Sie Inhalte eines Anbieters durchsuchen koennen, muessen Sie sich mit ihm verbinden. Einige Anbieter erfordern eine Authentifizierung ueber einen QR-Code oder eine E-Mail-Anmeldung, waehrend andere mit einem einzigen Klick verbunden werden koennen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Installieren und starten Sie FreePlay -- siehe [Erste Schritte](../getting-started/)
- Halten Sie Ihre TV-Fernbedienung zur Navigation bereit
- Halten Sie fuer Anbieter, die eine Anmeldung erfordern, Ihre Zugangsdaten bereit

</div>

## Verfuegbare Anbieter durchsuchen

1. Oeffnen Sie den Bildschirm **Content Providers** ueber die Seitenleiste (waehlen Sie das **Providers**-Symbol unten aus)
2. Sie sehen ein Raster aus Anbieterkarten, die jeweils das Logo und den Namen des Anbieters zeigen
3. Verbundene Anbieter zeigen ein gruenes **Connected**-Abzeichen unter ihrem Namen
4. Anbieter, die noch nicht verfuegbar sind, zeigen ein **Coming Soon**-Label

## Verbindung ohne Authentifizierung

Einige Anbieter erfordern keine Anmeldung. Wenn Sie einen dieser Anbieter auswaehlen, verbindet sich FreePlay sofort und oeffnet den Inhaltsbrowser. Es sind keine Zugangsdaten erforderlich.

## Geraetefluss-Authentifizierung (QR-Code)

Bestimmte Anbieter verwenden einen Geraetefluss, aehnlich wie die Anmeldung bei Streaming-Apps auf einem Fernseher:

1. Waehlen Sie die Anbieterkarte auf dem Bildschirm **Content Providers**
2. FreePlay zeigt einen QR-Code und eine Verifizierungs-URL an
3. Scannen Sie den QR-Code mit Ihrem Telefon oder besuchen Sie die angezeigte URL auf einem beliebigen Geraet
4. Geben Sie den auf dem Fernseher angezeigten Benutzercode ein
5. Schliessen Sie den Anmeldeprozess auf Ihrem Telefon oder Computer ab
6. FreePlay erkennt die erfolgreiche Anmeldung und zeigt **Connected!** an
7. Der Inhaltsbrowser oeffnet sich automatisch

:::info
Eine pulsierende Anzeige **Waiting for authorization** zeigt an, dass FreePlay auf Ihre Anmeldung wartet. Der Code laeuft nach einigen Minuten ab, fuehren Sie den Vorgang daher zuegig durch.
:::

## Formular-Anmeldung

Andere Anbieter verwenden eine traditionelle E-Mail- und Passwort-Anmeldung:

1. Waehlen Sie die Anbieterkarte aus
2. Geben Sie Ihre **E-Mail-Adresse** und Ihr **Passwort** ueber die Bildschirmtastatur ein
3. Waehlen Sie die Schaltflaeche **Sign In**
4. Wenn Ihre Zugangsdaten korrekt sind, zeigt FreePlay **Connected!** an und oeffnet den Inhaltsbrowser

:::tip
Verwenden Sie das Steuerkreuz auf Ihrer Fernbedienung, um zwischen dem E-Mail-Feld, dem Passwort-Feld und der Anmeldeschaltflaeche zu wechseln. Druecken Sie **Select** auf einem Textfeld, um die Bildschirmtastatur zu oeffnen.
:::

## Einen Anbieter trennen

Um einen bereits verbundenen Anbieter zu trennen:

1. Gehen Sie zum Bildschirm **Content Providers**
2. Waehlen Sie die Anbieterkarte mit dem **Connected**-Abzeichen
3. Eine Bestaetigung fragt, ob Sie die Verbindung trennen moechten
4. Waehlen Sie **Disconnect**, um die Verbindung zu entfernen

Nach dem Trennen werden die Inhalte des Anbieters nicht mehr in Ihrer Seitenleiste angezeigt.

:::warning
Das Trennen entfernt die gespeicherte Authentifizierung von Ihrem Geraet. Sie muessen sich erneut anmelden, wenn Sie sich spaeter wieder verbinden moechten.
:::
