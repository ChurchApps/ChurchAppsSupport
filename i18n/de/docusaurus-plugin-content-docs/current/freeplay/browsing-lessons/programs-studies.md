# Programme und Studienreihen

<div class="article-intro">

Der Lektionsbrowser verwendet einen schrittweisen Ablauf, um genau den Inhalt einzugrenzen, den Sie abspielen moechten. Jeder Schritt zeigt ein 3-spaltiges Raster aus Karten mit Bildern und Titeln, sodass Sie die gesamte Lektionsbibliothek einfach mit Ihrer TV-Fernbedienung navigieren koennen.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Stellen Sie sicher, dass Ihr Fernseher eine Internetverbindung hat (erforderlich zum Durchsuchen des Programmkatalogs)
- Navigieren Sie ueber die Seitenleiste zum Bildschirm **Browse Programs**

</div>

## Programme durchsuchen

1. Oeffnen Sie den Bildschirm **Browse Programs**
2. Programme erscheinen als Raster aus Bildkarten, drei nebeneinander
3. Verwenden Sie das Steuerkreuz auf Ihrer Fernbedienung, um ein Programm hervorzuheben
4. Druecken Sie **Select**, um es zu oeffnen

Jede Programmkarte zeigt das Titelbild des Programms. Wenn kein Bild verfuegbar ist, wird eine Platzhalterkarte angezeigt.

## Eine Studienreihe auswaehlen

Nach der Auswahl eines Programms wird der Bildschirm **Select a Study** mit allen Studienreihen in diesem Programm geladen.

1. Studienreihen-Karten erscheinen in einem 3-spaltigen Raster mit Titelbildern und Studienreihen-Namen
2. Navigieren Sie zu der gewuenschten Studienreihe und druecken Sie **Select**
3. Druecken Sie **Back**, um zur Programmliste zurueckzukehren

## Eine Lektion auswaehlen

Nach der Auswahl einer Studienreihe zeigt der Bildschirm **Select a Lesson** alle Lektionen in dieser Studienreihe an.

1. Lektionskarten zeigen Titelbilder und Lektionsnamen
2. Navigieren Sie zur gewuenschten Lektion und druecken Sie **Select**
3. Druecken Sie **Back**, um zur Studienreihenliste zurueckzukehren

## Einen Veranstaltungsort auswaehlen

Der Lektionsdetail-Bildschirm zeigt den Lektionsnamen, den Titel und die Beschreibung auf dem Titelbild der Lektion. Unterhalb der Beschreibung sehen Sie eine oder mehrere Veranstaltungsort-Schaltflaechen.

Veranstaltungsorte stellen verschiedene Versionen derselben Lektion dar. Zum Beispiel koennte eine Lektion separate Veranstaltungsorte fuer die Altersgruppen "Vorschule", "Grundschule" und "Teenager" haben. Jeder Veranstaltungsort enthaelt die fuer diese Umgebung passenden Mediendateien.

1. Verwenden Sie das Steuerkreuz, um den gewuenschten Veranstaltungsort hervorzuheben
2. Druecken Sie **Select**, um den Inhalt zu laden
3. FreePlay ruft die Wiedergabeliste des Veranstaltungsortes ab und startet den Player

:::tip
Wenn Sie nur eine Veranstaltungsort-Schaltflaeche sehen, gibt es nur eine Version der Lektion. Waehlen Sie sie aus, um sofort mit der Wiedergabe zu beginnen.
:::

## Zurueck navigieren

An jedem Punkt im Durchsuchungsablauf koennen Sie **Back** auf Ihrer Fernbedienung druecken, um zur vorherigen Ebene zurueckzukehren:

- Vom Lektionsdetail-Bildschirm fuehrt **Back** zurueck zur Lektionsliste
- Von der Lektionsliste fuehrt **Back** zurueck zur Studienreihenliste
- Von der Studienreihenliste fuehrt **Back** zurueck zum Programmraster
- Vom Programmraster oeffnet **Back** die Seitenleiste
