# Lektionen durchsuchen

<div class="article-intro">

FreePlay enthaelt einen integrierten Lektionsbrowser, mit dem Sie die gesamte Bibliothek verfuegbarer Programme, Studienreihen und Lektionen erkunden koennen. Dies ist nuetzlich, wenn Sie manuell eine Lektion zur Wiedergabe auswaehlen moechten, anstatt sich auf einen Klassenraumplan oder einen Plan zu verlassen.

</div>

## Der Durchsuchungsablauf

Lektionen sind in einer Hierarchie organisiert:

1. **Programme** - Uebergeordnete Lehrplankategorien (z. B. eine Kinderdienst-Reihe)
2. **Studienreihen** - Gruppen von Lektionen innerhalb eines Programms (z. B. eine mehrwoechige Einheit)
3. **Lektionen** - Einzelne Lektionssitzungen mit eigenen Inhalten
4. **Veranstaltungsorte** - Verschiedene Versionen einer Lektion, die fuer bestimmte Altersgruppen oder Umgebungen zugeschnitten sind

Sie beginnen mit der Auswahl eines Programms, gehen dann in eine Studienreihe, waehlen eine Lektion und waehlen schliesslich einen Veranstaltungsort, um die Wiedergabe zu starten.

## Zugriff auf den Lektionsbrowser

Der Lektionsbrowser ist ueber den Bildschirm **Browse Programs** zugaenglich. Navigieren Sie dorthin ueber die Seitenleiste oder indem Sie die Durchsuchen-Option in der Navigation auswaehlen.

:::info
Zum Durchsuchen von Lektionen ist eine Internetverbindung erforderlich, um den Programmkatalog zu laden. Sobald Sie einen Veranstaltungsort auswaehlen und die Wiedergabe starten, werden die Inhalte direkt gestreamt, ohne einen separaten Download-Schritt.
:::
