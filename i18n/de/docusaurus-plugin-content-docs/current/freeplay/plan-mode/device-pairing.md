# Ihr Gerät koppeln

<div class="article-intro">

Um Plan Mode zu verwenden, müssen Sie Ihren Fernseher mit einem Plan-Typ in B1 Admin koppeln. FreePlay erzeugt einen eindeutigen Pairing-Code, der das Gerät mit dem Gottesdienst-Plan Ihrer Kirche verknüpft und automatische Inhalts-Zustellung jede Woche ermöglicht.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Installieren und starten Sie FreePlay — siehe [Getting Started](../getting-started/)
- Wählen Sie **Pair to Plan** aus dem [Pairing-Modus-Bildschirm](../getting-started/pairing-modes.md)
- Haben Sie Zugang zu **B1 Admin** auf einem Computer oder Telefon, um die Kopplung zu vervollständigen

</div>

## Pairing-Code generieren

1. Wählen Sie **Pair to Plan** von dem **Select Pairing Mode**-Bildschirm
2. FreePlay kontaktiert den Server und erzeugt einen kurzen Pairing-Code
3. Der Code erscheint in großen Zeichen in der Mitte des Bildschirms
4. Unter dem Code zeigt ein pulsierender Indikator **Auf Verbindung warten**

Der Code wird als einzelne Zeichen für einfaches Lesen von überall im Raum angezeigt.

## Code in B1 Admin eingeben

1. Gehen Sie auf einem Computer oder Telefon zu der unten im Code angezeigten Adresse (oder scannen Sie den QR-Code) — dies öffnet die **Gerät autorisieren**-Seite in B1 Admin
2. Geben Sie den Pairing-Code ein, wenn er nicht bereits vom QR-Code ausgefüllt wurde
3. Unter **Pläne anzeigen für** wählen Sie den Plan-Typ, dem dieser Bildschirm folgen soll (z.B. "Sonntagsgottesdienst")
4. Genehmigen Sie das Gerät

FreePlay fragt den Server alle paar Sekunden ab und prüft, ob die Kopplung abgeschlossen ist. Sobald **B1 Admin** die Verbindung bestätigt, bewegt sich der Fernseher automatisch zum Plan-Download-Bildschirm.

:::tip
Geräte können auch von **Profile → Devices → Add Device** in B1 Admin mit dem gleichen Code gekoppelt werden.
:::

## Plan-Inhalte herunterladen

Nach der Kopplung lädt FreePlay den aktuellen Plan für diesen Plan-Typ. Es zeigt an:

- Der Plan-Name und das Dienstdatum
- Der zugehörige Unterricht-Name (wenn der Plan Unterricht-Inhalte enthält)
- Ein Fortschrittsindikator, der **Wird Element X von Y heruntergeladen** anzeigt

Wenn alle Mediendateien heruntergeladen sind, erscheint die **Start Plan**-Schaltfläche. Drücken Sie **Select** auf Ihrer Fernbedienung, um die Wiedergabe zu starten.

:::tip
Der Plan wird automatisch stündlich aufgefrischt. Wenn der Plan während des Tages aktualisiert wird, nimmt FreePlay die Änderungen ohne manuelle Intervention auf.
:::

## Wenn die Kopplung fehlschlägt

Wenn der Pairing-Code nicht generiert werden kann (z.B. aufgrund eines Netzwerkproblems), wird eine Fehlermeldung mit einer **Try Again**-Schaltfläche angezeigt. Stellen Sie sicher, dass Ihr Fernseher mit dem Internet verbunden ist und versuchen Sie erneut.

:::warning
Pairing-Codes verfallen nach einer bestimmten Zeit. Wenn Sie zu lange warten, erzeugen Sie einen neuen Code, indem Sie zum Pairing-Bildschirm zurückkehren.
:::
