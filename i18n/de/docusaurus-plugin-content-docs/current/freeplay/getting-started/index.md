# Erste Schritte mit FreePlay

<div class="article-intro">

FreePlay ist verfuegbar fuer Android TV, Amazon Fire TV und Apple TV. Diese Anleitung fuehrt Sie durch die Installation der App, den ersten Start und die grundlegenden TV-Fernbedienungssteuerungen, die Sie in der gesamten App verwenden werden.

</div>

## FreePlay installieren

### Android TV / Amazon Fire TV

1. Oeffnen Sie den App Store auf Ihrem Fernseher (**Google Play Store** oder **Amazon Appstore**)
2. Suchen Sie nach **FreePlay**
3. Waehlen Sie die App aus und waehlen Sie **Install**

### Apple TV

1. Oeffnen Sie den **App Store** auf Ihrem Apple TV
2. Suchen Sie nach **FreePlay**
3. Waehlen Sie die App aus und waehlen Sie **Get**

## Die App starten

Wenn Sie FreePlay zum ersten Mal oeffnen, sehen Sie einen animierten Startbildschirm mit dem FreePlay-Logo. Die App prueft auf zuvor verbundene Inhaltsanbieter. Wenn Sie bereits einen Anbieter verbunden haben, fuehrt sie Sie direkt zum Inhaltsbrowser dieses Anbieters. Andernfalls oeffnet sie den Bildschirm **Content Providers**, auf dem Sie sich mit Ihrer ersten Inhaltsquelle verbinden koennen.

## Navigation mit Ihrer TV-Fernbedienung

FreePlay ist fuer TV-Fernbedienungen entwickelt. Hier sind die Steuerungen, die Sie in der gesamten App verwenden werden:

- **Steuerkreuz (oben/unten/links/rechts)** - Zwischen Elementen auf dem Bildschirm wechseln
- **Select / OK-Taste** - Das hervorgehobene Element auswaehlen
- **Linker Pfeil** - Seitenleiste von jedem Bildschirm aus oeffnen
- **Rechter Pfeil** - Seitenleiste schliessen und zum Hauptinhaltsbereich zurueckkehren
- **Back / Menu-Taste** - Zum vorherigen Bildschirm zurueckkehren

:::tip
Die Seitenleiste klappt beim Durchsuchen von Inhalten zu einem schmalen Symbolstreifen zusammen. Druecken Sie den linken Pfeil auf Ihrer Fernbedienung, um sie auszuklappen und die vollstaendigen Beschriftungen zu sehen.
:::
