## Installieren Sie FreePlay

### Android TV / Amazon Fire TV

1. Öffnen Sie den App-Store auf Ihrem Fernseher (**Google Play Store** oder **Amazon Appstore**)
2. Suchen Sie nach **FreePlay**
3. Wählen Sie die App und wählen Sie **Installieren**

### Apple TV

1. Öffnen Sie den **App Store** auf Ihrem Apple TV
2. Suchen Sie nach **FreePlay**
3. Wählen Sie die App und wählen Sie **Installieren**

## Starten Sie die App

Wenn Sie FreePlay zum ersten Mal öffnen, sehen Sie einen animierten Splash-Bildschirm mit dem FreePlay-Logo. Die App überprüft auf alle zuvor verbundenen Inhalts-Anbieter. Wenn Sie bereits einen Anbieter verbunden haben, nimmt Sie die App direkt zum Inhalts-Browser dieses Anbieters. Andernfalls öffnet sich der Bildschirm **Inhalts-Anbieter**, wo Sie sich mit Ihrer ersten Inhaltsquelle verbinden können.

## Mit Ihrer Fernbedienung navigieren

FreePlay ist für Fernbedienungen gebaut. Hier sind die Steuerungen, die Sie in der gesamten App verwenden:

- **Richtungspad (oben/unten/links/rechts)** - Navigieren Sie zwischen Elementen auf dem Bildschirm
- **Wählen / OK-Schaltfläche** - Wählen Sie das hervorgehobene Element
- **Linker Pfeil** - Öffnen Sie die Seitenleiste von einem beliebigen Bildschirm
- **Rechter Pfeil** - Schließen Sie die Seitenleiste und kehren Sie zum Haupt-Inhaltsbereich zurück
- **Zurück / Menü-Schaltfläche** - Gehen Sie zum vorherigen Bildschirm zurück

:::tip
Die Seitenleiste wird zu einem schmalen Symbol-Streifen zusammengeklappt, wenn Sie Inhalte durchsuchen. Drücken Sie den linken Pfeil auf Ihrer Fernbedienung, um sie zu erweitern und die vollständigen Labels zu sehen.
:::
