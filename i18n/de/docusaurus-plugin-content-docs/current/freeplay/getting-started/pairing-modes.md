# Wählen Sie einen Pairing-Modus

<div class="article-intro">

FreePlay bietet zwei Pairing-Modi zum Verbinden Ihres Fernsehers mit Ihrer Kirche, plus die Möglichkeit, Content-Provider direkt zu durchsuchen. Der **Pairing-Modus-Auswahl**-Bildschirm präsentiert diese Optionen als große Karten, die Sie mit Ihrer Fernbedienung navigieren können.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Installieren und starten Sie FreePlay — siehe [Getting Started](./index.md)
- Haben Sie Ihre TV-Fernbedienung bereit zur Navigation

</div>

## Zu Klassenzimmer koppeln

Wählen Sie **Zu Klassenzimmer koppeln**, wenn ein bestimmtes Fernsehen immer die Unterricht anzeigen soll, die für einen bestimmten Raum eingeplant ist. Dies ist die beste Option für Klassenzimmer der Kinderbetreuung, wo Unterricht auf einem wöchentlichen Zeitplan zugewiesen ist.

**Wie es funktioniert:** Sie suchen Ihre Kirche nach Name, wählen ein Klassenzimmer und FreePlay ruft automatisch ab, welche Lektion für diesen Raum heute eingeplant ist. Jeden Sonntagmorgen schalten Sie einfach den Fernseher ein und die richtige Lektion ist bereit.

**Am besten für:** Dedizierte Klassenzimmer-Fernseher, die immer den eingeplanten Inhalte für diesen Raum anzeigen sollen.

:::tip
Richten Sie Ihre Klassenzimmer unter [freeplay.church](https://freeplay.church) ein, bevor Sie Ihren Fernseher koppeln.
:::

## Zu Plan koppeln

Wählen Sie **Zu Plan koppeln**, wenn Sie einem Kirchengottesdienst-Plan folgen möchten. Dieser Modus erzeugt einen Pairing-Code, den Sie in **B1 Admin** eingeben, um den Fernseher mit einem bestimmten Plan-Typ zu verknüpfen. Der Fernseher lädt dann automatisch den dem aktuellen Plan zugeordneten Inhalte herunter und spielt ihn ab.

**Wie es funktioniert:** FreePlay zeigt einen Pairing-Code und einen QR-Code auf dem Bildschirm an. Ein Administrator öffnet die verknüpfte Seite in **B1 Admin**, gibt den Code ein und wählt den Plan-Typ aus, dem der Bildschirm folgen soll. Sobald genehmigt, lädt der Fernseher automatisch den aktuellen Plan-Inhalte.

**Am besten für:** Fernseher, die Inhalte anzeigen, die an Ihren wöchentlichen Gottesdienst-Plan gebunden sind, wie Worship-Texte oder Ankündigungsfolien.

## Content-Provider

Wenn Sie nicht an ein Klassenzimmer oder Plan koppeln müssen, können Sie Content-Provider direkt durchsuchen. Provider wie [Lessons.church](../../lessons-church/) und Bible Project bieten Inhalte, die Sie auf Abruf herunterladen und abspielen können.

**Am besten für:** Fernseher, auf denen Freiwillige manuell wählen, was jede Woche zu spielen ist.

:::info
Sie können den **Providers**-Bildschirm jederzeit in der Seitenleiste aufrufen. Sie müssen keinen Pairing-Modus verwenden.
:::
