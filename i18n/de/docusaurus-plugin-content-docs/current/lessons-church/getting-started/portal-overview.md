# Portalübersicht

<div class="article-intro">

Nach der Anmeldung bei Lessons.church gelangen Sie zum **Portal** -- dem Dashboard Ihrer Gemeinde zur Verwaltung von Klassenzimmern und Lektionsplänen. Hier werden Sie die meiste Zeit verbringen, wenn Sie den Lehrplan für Ihre Gemeinde organisieren.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Erstellen Sie ein Konto](./creating-account.md) und verbinden Sie sich mit Ihrer Gemeinde
- Melden Sie sich bei [Lessons.church](https://lessons.church) an

</div>

## Was Sie sehen

Das Portal ist in zwei Hauptbereiche gegliedert:

- **Klassenzimmer** (rechte Seite) -- Eine Liste der Klassenzimmer Ihrer Gemeinde. Klicken Sie auf einen Klassenzimmernamen, um die geplanten Lektionen anzuzeigen. Sie können von hier aus auch neue Klassenzimmer hinzufügen oder bestehende bearbeiten.
- **Geplante Lektionen** (linke Seite) -- Sobald Sie ein Klassenzimmer ausgewählt haben, zeigt dieser Bereich die dafür geplanten Lektionen an. Wenn Sie noch keine Klassenzimmer eingerichtet haben, sehen Sie ein Einrichtungsvideo, das Ihnen den Einstieg erleichtert.

## Was Sie vom Portal aus tun können

Vom Portal aus können Sie:

1. **Klassenzimmer verwalten** -- Klassenzimmer hinzufügen, bearbeiten oder entfernen. Jedes Klassenzimmer kann mit einem Freiwilligen-Team verknüpft werden (damit sie bevorstehende Lektionen in der B1-App sehen) und mit einer Eltern-/Schülergruppe (damit diese auf kürzliche Lektionen zugreifen können).
2. **Geplante Lektionen ansehen und verwalten** -- Sehen Sie, welche Lektionen für jedes Klassenzimmer anstehen, und nehmen Sie bei Bedarf Änderungen vor.
3. **Klassenzimmer-Feeds abonnieren** -- Klicken Sie auf das Feed-Symbol neben einem Klassenzimmer, um eine Playlist-Feed-URL zu erhalten, die Sie mit kompatiblen Apps verwenden können.
4. **Auf externe Anbieter zugreifen** -- Nutzen Sie die Seite **External Providers**, um Drittanbieter-Inhaltsquellen zu konfigurieren, die das Open Lesson Format unterstützen. So können Sie Lektionen von anderen Anbietern über die integrierte Lessons.church-Bibliothek hinaus einbinden.
5. **Auf den Admin-Bereich zugreifen** -- Wenn Sie Administratorberechtigungen haben, sehen Sie eine **Admin**-Option in der Navigation. Der Admin-Bereich ermöglicht die Verwaltung von Programmen, Studienreihen und Lektionen auf Inhaltsebene.

## Navigation

Die Portal-Navigation umfasst:

- **Home** -- Zurück zur öffentlichen Lessons.church-Startseite, um Inhalte zu durchsuchen.
- **Schedules** -- Die Hauptportalansicht zur Verwaltung von Klassenzimmern und Lektionsplänen.
- **External Providers** -- Konfiguration von Drittanbieter-Lektionsanbietern.
- **Admin** -- Verwaltung von Programmen, Studienreihen und Lektionsinhalten (nur sichtbar, wenn Sie Administratorberechtigungen haben).

:::info
Die Option **Admin** erscheint nur, wenn Ihrem Konto Berechtigungen zur Inhaltsbearbeitung erteilt wurden. Die meisten Gemeindemitarbeiter nutzen die Seiten **Schedules** und **External Providers** für die tägliche Verwaltung.
:::

## Erste Schritte

Wenn Sie das Portal zum ersten Mal besuchen und noch keine Klassenzimmer erstellt haben, beginnen Sie mit einem Klick auf **Add First Classroom** im Klassenzimmer-Panel. Geben Sie Ihrem Klassenzimmer einen Namen (zum Beispiel „Vorschule" oder „3.-5. Klasse") und speichern Sie es. Von dort aus können Sie mit der [Planung von Lektionen](../classrooms/scheduling-lessons.md) beginnen.
