# Lektionen verwalten

<div class="article-intro">

Lektionen sind die einzelnen Unterrichtseinheiten innerhalb einer Studienreihe. Jede Lektion stellt dar, was während eines einzelnen Unterrichtstreffens passiert. Hier bauen Sie die detaillierten Inhalte auf -- Veranstaltungsorte, Abschnitte, Rollen und Aktionen -- die Freiwillige beim Unterrichten verwenden.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Erstellen Sie ein Programm](./managing-programs.md) und [fügen Sie eine Studienreihe hinzu](./managing-studies.md)
- Verfügen Sie über Administratorberechtigungen für Lessons.church (siehe [Administrationsübersicht](./index.md))

</div>

## Eine Lektion erstellen

1. Navigieren Sie zum **Admin**-Bereich, erweitern Sie das Programm und die Studienreihe, und finden Sie die Studienreihe, der Sie eine Lektion hinzufügen möchten.
2. Klicken Sie auf **Add Lesson**.
3. Geben Sie einen **Namen** und weitere Details für die Lektion ein.
4. Speichern Sie die Lektion.

## Die Lektionsstruktur verstehen

Jede Lektion ist in mehrere Ebenen gegliedert:

- **Veranstaltungsorte** -- Verschiedene Versionen der Lektion für unterschiedliche Altersgruppen oder Umgebungen. Zum Beispiel könnten Sie einen Veranstaltungsort „Vorschule" und einen „Grundschule" für dieselbe Lektion haben.
- **Abschnitte** -- Die Segmente innerhalb eines Veranstaltungsortes, wie „Begrüßung", „Video", „Bibelgeschichte" und „Aktivität". Abschnitte steuern den Ablauf der Unterrichtszeit.
- **Rollen und Aktionen** -- Innerhalb jedes Abschnitts weisen Sie Rollen zu (zum Beispiel „Hauptlehrer" oder „Kleingruppenleiter") und geben jeder Rolle bestimmte Aktionen.

## Aktionstypen

Es gibt vier Arten von Aktionen, die Sie einer Rolle innerhalb eines Abschnitts zuweisen können:

- **Say** -- Text, den der Freiwillige der Klasse vorlesen oder in eigenen Worten wiedergeben soll.
- **Do** -- Eine praktische Aktivität oder Anweisung, die durchgeführt werden soll.
- **Note** -- Hintergrundinformationen oder eine Erinnerung für den Freiwilligen (wird nicht laut vorgelesen).
- **Play** -- Ein Video- oder Audioclip, der während des Abschnitts abgespielt werden soll.

## Einen Veranstaltungsort hinzufügen

1. Öffnen Sie die Lektion in der Admin-Ansicht.
2. Klicken Sie auf **Add Venue**.
3. Geben Sie einen Namen für den Veranstaltungsort ein (z.B. „Vorschule" oder „Grundschule").
4. Speichern Sie. Sie können nun Abschnitte zu diesem Veranstaltungsort hinzufügen.

## Abschnitte und Aktionen hinzufügen

1. Erweitern Sie einen Veranstaltungsort, um seine Abschnitte zu sehen.
2. Klicken Sie auf **Add Section** und geben Sie ihm einen Namen (z.B. „Begrüßung").
3. Klicken Sie innerhalb des Abschnitts auf **Add Action**.
4. Wählen Sie die **Rolle**, den **Aktionstyp** (Say, Do, Note oder Play) und geben Sie den Inhalt ein.
5. Speichern Sie.

Wiederholen Sie diesen Vorgang, um den gesamten Lektionsablauf aufzubauen.

## Ressourcen auf Lektionsebene hochladen

Sie können herunterladbare Materialien direkt an eine Lektion anhängen -- zum Beispiel ein Arbeitsblatt oder eine Ausmalseite speziell für diese Woche.

1. Öffnen Sie die Lektion in der Admin-Ansicht.
2. Navigieren Sie zum Bereich **Resources**.
3. Laden Sie Ihre Dateien hoch.

:::tip
Erstellen Sie Ihre Lektion aus der Perspektive des Freiwilligen. Gehen Sie die Abschnitte und Aktionen der Reihe nach durch und fragen Sie sich, ob ein Freiwilliger, der zum ersten Mal dabei ist, ohne weitere Vorbereitung folgen könnte.
:::
