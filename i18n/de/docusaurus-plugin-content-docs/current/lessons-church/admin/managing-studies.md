# Studienreihen verwalten

<div class="article-intro">

Eine Studienreihe ist eine Reihe zusammengehöriger Lektionen innerhalb eines Programms. Zum Beispiel wäre eine vierwöchige Reihe namens „Der barmherzige Samariter" eine Studienreihe innerhalb eines größeren Kinderdienst-Programms. Diese Anleitung behandelt das Erstellen, Bearbeiten und Organisieren von Studienreihen im Administrationsbereich.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Erstellen Sie mindestens ein Programm](./managing-programs.md) im Admin-Bereich
- Verfügen Sie über Administratorberechtigungen für Lessons.church (siehe [Administrationsübersicht](./index.md))

</div>

## Eine Studienreihe erstellen

1. Navigieren Sie zum **Admin**-Bereich und erweitern Sie das Programm, dem Sie eine Studienreihe hinzufügen möchten.
2. Klicken Sie auf **Add Study**.
3. Füllen Sie die folgenden Felder aus:
   - **Name** -- Der Anzeigename der Studienreihe (z.B. „Der barmherzige Samariter").
   - **Slug** -- Eine URL-freundliche Kennung (z.B. „the-good-samaritan").
   - **Description** -- Eine Zusammenfassung dessen, was diese Studienreihe abdeckt und wie viele Lektionen sie enthält.
   - **Image** -- Laden Sie ein Vorschaubild für die Studienreihe hoch.
4. Speichern Sie die Studienreihe.

## Eine Studienreihe bearbeiten

1. Klicken Sie auf den Namen der Studienreihe in der Admin-Liste.
2. Aktualisieren Sie beliebige Felder nach Bedarf.
3. Speichern Sie Ihre Änderungen.

## Eine Kategorie zuweisen

Wenn das übergeordnete Programm Studienkategorien verwendet, können Sie Ihre Studienreihe einer zuweisen:

1. Öffnen Sie die Studienreihe in der Admin-Ansicht.
2. Wählen Sie eine **Kategorie** aus dem Dropdown-Menü.
3. Speichern Sie.

Kategorien helfen Nutzern beim Durchsuchen von Programmen mit vielen Studienreihen. Zum Beispiel könnten Sie Kategorien wie „Altes Testament", „Neues Testament" und „Thematisch" haben.

## Lektionen innerhalb einer Studienreihe verwalten

Studienreihen enthalten eine oder mehrere Lektionen. Erweitern Sie eine Studienreihe in der Admin-Ansicht, um ihre Lektionen zu sehen, oder klicken Sie auf **Add Lesson**, um eine neue zu erstellen. Siehe [Lektionen verwalten](./managing-lessons.md) für Details zum Aufbau von Lektionsinhalten.

## Ressourcen auf Studienreihenebene hochladen

Sie können herunterladbare Dateien anhängen, die für die gesamte Studienreihe gelten -- zum Beispiel eine Reihenübersicht, Werbegrafiken oder eine Vorlage für einen Elternbrief.

1. Öffnen Sie die Studienreihe in der Admin-Ansicht.
2. Navigieren Sie zum Bereich **Resources**.
3. Laden Sie Ihre Dateien hoch.

:::tip
Schreiben Sie Ihre Studienreihenbeschreibung mit Gemeindeleitern im Blick. Eine klare Zusammenfassung des Themas, der Zielgruppe und der Anzahl der Lektionen hilft Gemeinden bei der Entscheidung, ob sie die Reihe verwenden möchten.
:::
