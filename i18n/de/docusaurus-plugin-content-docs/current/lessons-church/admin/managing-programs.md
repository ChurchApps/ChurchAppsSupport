# Programme verwalten

<div class="article-intro">

Programme sind die übergeordneten Container für Lehrpläne in Lessons.church. Ein Programm repräsentiert eine gesamte Lehrplanreihe -- zum Beispiel „Ark Kids" oder „Youth Foundations". Diese Anleitung behandelt das Erstellen, Bearbeiten und Organisieren von Programmen im Administrationsbereich.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Verfügen Sie über Administratorberechtigungen für Lessons.church (siehe [Administrationsübersicht](./index.md))
- Bereiten Sie einen Programmnamen, eine Beschreibung und ein Vorschaubild vor

</div>

## Ein Programm erstellen

1. Navigieren Sie zum **Admin**-Bereich.
2. Klicken Sie auf **Add Program**.
3. Füllen Sie die folgenden Felder aus:
   - **Name** -- Der Anzeigename des Programms (z.B. „Ark Kids").
   - **Slug** -- Eine URL-freundliche Kennung (z.B. „ark-kids"). Diese wird in Webadressen verwendet.
   - **Description** -- Eine kurze Zusammenfassung dessen, was das Programm abdeckt.
   - **Image** -- Laden Sie ein Vorschaubild hoch, das das Programm repräsentiert.
4. Speichern Sie das Programm.

Ihr neues Programm erscheint nun in der Admin-Programmliste und, sobald es Studienreihen und Lektionen enthält, auf der öffentlichen Webseite.

## Ein Programm bearbeiten

1. Klicken Sie auf den Programmnamen in der Admin-Liste.
2. Aktualisieren Sie beliebige Felder (**Name**, **Slug**, **Description** oder **Image**).
3. Speichern Sie Ihre Änderungen.

## Studienreihen innerhalb eines Programms organisieren

Sobald ein Programm existiert, können Sie Studienreihen hinzufügen. Studienreihen erscheinen verschachtelt unter dem Programm in der Admin-Ansicht. Klicken Sie auf den Erweiterungspfeil neben einem Programm, um seine Studienreihen zu sehen, oder fügen Sie von dort eine neue hinzu. Siehe [Studienreihen verwalten](./managing-studies.md) für Details.

## Studienkategorien

Wenn Ihr Programm viele Studienreihen hat, können Sie diese in **Kategorien** organisieren. Kategorien gruppieren Studienreihen nach Thema, Altersbereich oder anderen Kriterien Ihrer Wahl. Dies erleichtert es den Nutzern, ein umfangreiches Programm zu durchsuchen.

## Ressourcen auf Programmebene hochladen

Sie können herunterladbare Dateien an ein Programm anhängen. Dies sind Ressourcen, die für das gesamte Programm gelten und nicht für eine bestimmte Studienreihe oder Lektion -- zum Beispiel ein Umfangs- und Ablaufdokument oder ein Handbuch für Freiwillige.

1. Öffnen Sie das Programm in der Admin-Ansicht.
2. Navigieren Sie zum Bereich **Resources**.
3. Laden Sie Ihre Dateien hoch.

## Cache leeren

Nach Änderungen an einem Programm müssen Sie möglicherweise den Cache leeren, damit die Aktualisierungen auf der öffentlichen Seite erscheinen. Verwenden Sie die Option **Clear Cache** im Admin-Bereich, um die Inhalte zu aktualisieren.

:::tip
Wählen Sie einen aussagekräftigen Programmnamen und schreiben Sie eine klare Beschreibung. Dies ist das Erste, was Gemeinden beim Durchsuchen der Lehrplanbibliothek sehen, daher hilft eine gute Zusammenfassung bei der Entscheidung, ob das Programm für ihre Bedürfnisse geeignet ist.
:::
