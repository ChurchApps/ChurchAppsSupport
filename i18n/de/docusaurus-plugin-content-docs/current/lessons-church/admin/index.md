# Administration

<div class="article-intro">

Der Administrationsbereich in Lessons.church ist der Ort, an dem Inhaltsersteller und Administratoren die Lektionsbibliothek verwalten. Von hier aus können Sie Ihre Lehrplanstruktur aufbauen, Ressourcen hochladen, Nutzungsstatistiken einsehen und externe Inhaltsanbieter anbinden.

</div>

:::warning
Für den Administratorzugang sind entsprechende Berechtigungen erforderlich. Wenn Sie die Admin-Optionen nicht sehen, wenden Sie sich an Ihren Gemeindeadministrator, um Zugang zu beantragen.
:::

## Was Sie tun können

Der Administrationsbereich gibt Ihnen die volle Kontrolle über Ihre Inhalte:

- **Programme erstellen und bearbeiten** -- Richten Sie übergeordnete Lehrplanreihen mit Namen, Beschreibungen, Bildern und Slugs ein.
- **Studienreihen verwalten** -- Organisieren Sie Gruppen zusammengehöriger Lektionen innerhalb jedes Programms.
- **Lektionen aufbauen** -- Fügen Sie einzelne Lektionen mit Veranstaltungsorten, Abschnitten, Rollen und Aktionen hinzu.
- **Ressourcen hochladen** -- Hängen Sie herunterladbare Dateien (PDFs, Arbeitsblätter, Bilder) auf Programm-, Studienreihen- oder Lektionsebene an.
- **Statistiken einsehen** -- Sehen Sie, welche Inhalte heruntergeladen werden, von wem und wo.
- **Anbieter konfigurieren** -- Binden Sie externe Lehrplananbieter an, die das Open Lesson Format verwenden.

## Inhaltshierarchie

Als Administrator ist es hilfreich, die Struktur der Inhalte zu verstehen:

1. **Programme** enthalten eine oder mehrere Studienreihen.
2. **Studienreihen** enthalten eine oder mehrere Lektionen.
3. **Lektionen** enthalten einen oder mehrere Veranstaltungsorte (Varianten nach Altersgruppe).
4. **Veranstaltungsorte** enthalten Abschnitte, und Abschnitte enthalten rollenbasierte Aktionen.

Jede Ebene kann eigene herunterladbare Ressourcen enthalten.
