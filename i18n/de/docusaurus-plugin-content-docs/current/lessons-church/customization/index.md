# Anpassung

<div class="article-intro">

Lessons.church bietet eine Bibliothek gebrauchsfertiger Lektionsinhalte, aber jede Gemeinde ist anders. Die Anpassungsfunktionen ermöglichen es Ihnen, Lektionsinhalte an Ihre spezifischen Klassenzimmer, Ihren Lehrstil und Ihre Altersgruppen anzupassen, damit Ihre Freiwilligen nur das sehen, was für sie relevant ist.

</div>

## Warum anpassen?

Lektionsinhalte auf Lessons.church sind so gestaltet, dass sie flexibel einsetzbar sind. Eine einzelne Lektion kann Abschnitte für mehrere Altersgruppen, optionale Aktivitäten oder Rollen enthalten, die nicht für jedes Klassenzimmer gelten. Anstatt Ihre Freiwilligen zu bitten, Teile gedanklich zu überspringen, die nicht zutreffen, können Sie die Lektion so anpassen, dass sie nur das sehen, was für sie relevant ist.

Zum Beispiel könnte eine Lektion eine Bastelaktivität enthalten, die für Vorschulkinder hervorragend geeignet ist, aber nicht gut zu Ihrem Klassenzimmer der fünften Klasse passt. Mit der Anpassung können Sie diesen Abschnitt für die ältere Gruppe ausblenden und gleichzeitig für die jüngere beibehalten.

## Was Sie anpassen können

- **Abschnitte entfernen** -- Blenden Sie ganze Abschnitte einer Lektion aus, die für ein bestimmtes Klassenzimmer oder eine Altersgruppe nicht zutreffen.
- **Rollen oder Aktionen ausblenden** -- Entfernen Sie einzelne Rollen oder Aktionen innerhalb eines Abschnitts, wenn sie nicht benötigt werden.
- **Inhalte neu ordnen** -- Ordnen Sie Abschnitte und Aktionen um, damit sie dem von Ihnen bevorzugten Ablauf für Ihr Klassenzimmer entsprechen.

## Klassenzimmerspezifisch oder global

Wenn Sie eine Anpassung speichern, wählen Sie, wie weit sie gelten soll:

- **Einzelnes Klassenzimmer** -- Die Anpassung betrifft nur das aktuelle Klassenzimmer. Andere Klassenzimmer sehen den Originalinhalt.
- **Alle Klassenzimmer** -- Die Anpassung gilt für alle Klassenzimmer in Ihrer Gemeinde.

Diese Flexibilität ermöglicht es Ihnen, verschiedene Ansichten derselben Lektion für verschiedene Altersgruppen einzurichten oder eine Änderung vorzunehmen, die überall gleichzeitig gilt.

:::tip
Wenn Sie neu bei der Anpassung sind, beginnen Sie mit einem einzelnen Klassenzimmer, um zu sehen, wie es funktioniert, bevor Sie Änderungen auf alle Klassenzimmer anwenden.
:::
