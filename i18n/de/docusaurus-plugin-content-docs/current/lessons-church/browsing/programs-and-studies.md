# Programme und Studienreihen

<div class="article-intro">

Alle Lehrpläne auf Lessons.church sind in einer klaren Hierarchie organisiert. Das Verständnis dieser Struktur hilft Ihnen, die richtigen Inhalte für Ihre Gemeinde schnell zu finden und mühelos zwischen verwandten Lektionen zu navigieren.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Machen Sie sich mit der Übersicht [Inhalte durchsuchen](./index.md) vertraut, um die verschiedenen Möglichkeiten der Inhaltsfindung zu verstehen

</div>

## Inhaltshierarchie

Inhalte sind vom Allgemeinsten zum Spezifischsten organisiert:

- **Anbieter** -- Die Organisation, die den Lehrplan veröffentlicht (Lessons.church oder ein Drittanbieter).
- **Programm** -- Eine Lehrplanreihe, wie „Ark Kids" oder „NextLevel". Jedes Programm ist eine Sammlung verwandter Studienreihen, die um ein Thema oder eine Altersgruppe herum gestaltet sind.
- **Studienreihe** -- Eine Einheit innerhalb eines Programms, wie „Der barmherzige Samariter". Jede Studienreihe enthält eine Reihe von Lektionen, die zusammen gelehrt werden sollen, oft über mehrere Wochen.
- **Lektion** -- Eine einzelne Unterrichtseinheit innerhalb einer Studienreihe. Jede Lektion umfasst Abschnitte, Rollen, Aktionen und herunterladbare Materialien.
- **Veranstaltungsort** -- Eine altersgruppenspezifische Version einer Lektion, wie Vorschule oder Grundschule. Veranstaltungsorte ermöglichen es, dieselbe Lektion an verschiedene Zielgruppen anzupassen.

## Programme durchsuchen

So erkunden Sie Programme:

1. Besuchen Sie die Lessons.church-Startseite.
2. Scrollen Sie durch die Liste der verfügbaren Programme oder schauen Sie sich die empfohlenen Programme oben an.
3. Klicken Sie auf einen **Programmnamen** oder sein Bild, um die Programmseite zu öffnen.

Jede Programmseite enthält:

- **Beschreibung** -- Ein Überblick darüber, was der Lehrplan abdeckt und für wen er gedacht ist.
- **Programmbild** -- Eine visuelle Identität für die Reihe.
- **Videotrailer** -- Einige Programme enthalten eine kurze Videovorschau, damit Sie Stil und Qualität der Inhalte sehen können, bevor Sie sich festlegen.
- **Studienreihen** -- Eine Auflistung aller Studienreihen im Programm, die möglicherweise in Kategorien geordnet sind.

## Studienreihen durchsuchen

Von einer Programmseite aus:

1. Durchsuchen Sie die Liste der Studienreihen. Wenn das Programm Studienreihen in Kategorien organisiert, sehen Sie sie entsprechend gruppiert.
2. Klicken Sie auf einen **Studienreihennamen**, um die Studienreihenseite zu öffnen.
3. Die Studienreihenseite zeigt alle enthaltenen Lektionen in Unterrichtsreihenfolge an.

Von hier aus können Sie in jede Lektion klicken, um deren vollständigen Inhalt zu sehen, einschließlich Abschnitten, Rollen und herunterladbaren Materialien. Siehe [Lektionen ansehen](./viewing-lessons.md) für eine detaillierte Anleitung.

:::info
Programme und Studienreihen werden von Lehrplananbietern verwaltet. Wenn Sie ein Inhaltsersteller sind, können Sie Ihre eigenen Programme und Studienreihen über den Bereich [Administration](../admin/index.md) verwalten.
:::
