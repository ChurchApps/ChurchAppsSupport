# Lessons.church

<div class="article-intro">

Lessons.church ist eine vollkommen kostenlose Lehrplan-Plattform für Gemeinden. Es gibt keine Abonnements, keine Gebühren und keine Nutzungsbeschränkungen. Die Plattform bietet über 1.100 videobasierte Lektionen in mehr als 189 Studienreihen von über 12 Anbietern, die Altersgruppen von der Vorschule bis zum Erwachsenenalter abdecken.

</div>

## Was Sie tun können

Mit Lessons.church kann Ihre Gemeinde:

- **Inhalte durchsuchen** -- Entdecken Sie eine wachsende Bibliothek von Programmen und Studienreihen verschiedener Anbieter, geordnet nach Altersgruppe und Thema.
- **Klassenzimmer einrichten** -- Erstellen Sie Klassenzimmer, die zu Ihrer Dienststruktur passen (zum Beispiel Vorschule, Grundschule, Jugend oder Erwachsene).
- **Lektionen planen** -- Weisen Sie Ihren Klassenzimmern bestimmte Lektionen an den gewünschten Terminen zu.
- **An Ihre Bedürfnisse anpassen** -- Passen Sie Lektionsinhalte an den Lehrstil und die Vorlieben Ihrer Gemeinde an.
- **Im Unterricht präsentieren** -- Nutzen Sie den integrierten Präsentationsmodus, um Lektionen Schritt für Schritt während der Unterrichtszeit durchzugehen.

## Wie Inhalte organisiert sind

Lessons.church organisiert den Lehrplan in einer klaren Hierarchie:

1. **Programme** -- Eine übergeordnete Lehrplanreihe eines Anbieters (zum Beispiel ein Programm für den Kinderdienst).
2. **Studienreihen** -- Eine Gruppe zusammengehöriger Lektionen innerhalb eines Programms (zum Beispiel eine vierwöchige Reihe zu einem bestimmten Thema).
3. **Lektionen** -- Eine einzelne Unterrichtseinheit innerhalb einer Studienreihe, die typischerweise für ein einzelnes Unterrichtstreffen verwendet wird.
4. **Veranstaltungsorte** -- Verschiedene Versionen einer Lektion, die auf bestimmte Altersgruppen oder Umgebungen zugeschnitten sind (zum Beispiel eine Version für die Vorschule und eine für die Grundschule).

## Integrationen

Lessons.church arbeitet mit anderen ChurchApps-Werkzeugen zusammen, um Ihnen bei der Bereitstellung von Inhalten zu helfen:

- **B1.church** -- Freiwillige und Eltern können über die B1-App auf bevorstehende und kürzlich durchgeführte Lektionen zugreifen.
- **FreeShow** -- Verwenden Sie die Präsentationssoftware FreeShow, um Lektionsinhalte auf einem Bildschirm anzuzeigen.
- **Venue TV App** -- Zeigen Sie Lektionen direkt auf einem Fernseher mit der Android TV App an.
