# Klassenzimmer einrichten

<div class="article-intro">

Bevor Sie Lektionen planen können, müssen Sie Klassenzimmer erstellen, die Ihre Unterrichtsumgebungen repräsentieren. Diese Anleitung führt Sie durch das Erstellen, Benennen und Verwalten von Klassenzimmern im Lessons.church-Portal.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Erstellen Sie ein Konto](../getting-started/creating-account.md) und verbinden Sie sich mit Ihrer Gemeinde
- Machen Sie sich mit der [Portalübersicht](../getting-started/portal-overview.md) vertraut

</div>

## Ein Klassenzimmer erstellen

1. Melden Sie sich bei Ihrem Konto auf [Lessons.church](https://lessons.church) an.
2. Navigieren Sie zu Ihrem **Portal**. Dies ist die zentrale Anlaufstelle Ihrer Gemeinde für die Verwaltung von Klassenzimmern und Zeitplänen.
3. Klicken Sie auf **Add Classroom**.
4. Geben Sie einen Namen für Ihr Klassenzimmer ein. Wählen Sie etwas Aussagekräftiges, das Ihre Freiwilligen wiedererkennen -- zum Beispiel „Vorschule", „Grundschule Klasse 1-2" oder „Mittelstufe".
5. Speichern Sie Ihr neues Klassenzimmer. Es erscheint sofort auf Ihrem Portal-Dashboard.

Wiederholen Sie diese Schritte für jeden Raum oder jede Altersgruppe in Ihrer Gemeinde. Sie können so viele Klassenzimmer erstellen, wie Sie benötigen.

## Ersteinrichtung

Wenn Sie noch keine Klassenzimmer eingerichtet haben, zeigt Ihr Portal ein Einrichtungsvideo an, das Sie durch den Prozess führt. Dieses Video behandelt die Grundlagen des Erstellens von Klassenzimmern und des Planens Ihrer ersten Lektionen. Es ist ein großartiger Ausgangspunkt, wenn Sie neu bei Lessons.church sind.

:::tip
Benennen Sie Ihre Klassenzimmer so, wie Ihre Freiwilligen bereits über Ihre Räume denken. Wenn alle ihn „den Vorschulraum" nennen, benennen Sie ihn „Vorschule" statt etwas Formelles wie „Frühkindliche Bildung Raum 101".
:::

## Klassenzimmer verwalten

Ihre Klassenzimmer erscheinen auf dem Portal-Dashboard, wo Sie sie jederzeit verwalten können. Von hier aus können Sie:

- **Ein Klassenzimmer umbenennen** -- Aktualisieren Sie den Namen, wenn sich Ihre Räume oder Altersgruppen ändern.
- **Ein Klassenzimmer entfernen** -- Löschen Sie Klassenzimmer, die Sie nicht mehr benötigen.
- **Den Zeitplan ansehen** -- Klicken Sie auf ein beliebiges Klassenzimmer, um seinen Lektionsplan zu sehen.

## Was kommt als Nächstes

Sobald Ihre Klassenzimmer eingerichtet sind, können Sie damit beginnen, [Lektionen zu planen](./scheduling-lessons.md), damit Ihre Freiwilligen jede Woche wissen, was sie unterrichten sollen.
