# Leitfäden für Freiwillige

<div class="article-intro">

Jede Lektion in Lessons.church enthält Leitfäden, die auf bestimmte Rollen von Freiwilligen zugeschnitten sind. Anstatt jedem Freiwilligen den gesamten Unterrichtsplan zu geben, können Sie sie auf einen Leitfaden verweisen, der nur die für ihre Rolle relevanten Anweisungen zeigt -- einfach und fokussiert.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- Verstehen Sie die Lektionsstruktur, indem Sie [Lektionen ansehen](../browsing/viewing-lessons.md) lesen
- Navigieren Sie zu der Lektion, für die Sie Leitfäden vorbereiten möchten

</div>

## Wie Leitfäden aufgebaut sind

Lektionen sind in **Abschnitte** unterteilt (zum Beispiel „Begrüßung", „Bibelgeschichte", „Kleingruppenaktivität"). Innerhalb jedes Abschnitts werden **Rollen** bestimmte **Aktionen** zugewiesen. Es gibt vier Aktionstypen:

- **Say** -- Vorformulierter Text, den der Freiwillige vorlesen oder in eigenen Worten wiedergeben soll.
- **Do** -- Eine praktische Handlung oder Aktivität, die durchgeführt werden soll.
- **Note** -- Hintergrundinformationen oder Erinnerungen für den Freiwilligen (wird nicht laut gesprochen).
- **Play** -- Ein Video- oder Audioclip, der für die Klasse abgespielt werden soll.

Wenn ein Freiwilliger seinen Leitfaden öffnet, sieht er jeden Abschnitt der Reihe nach mit nur den Aktionen, die seiner Rolle zugewiesen sind. Das hält die Sache einfach und fokussiert.

## Einen Freiwilligen-Leitfaden ansehen

1. Öffnen Sie die Lektion, die Sie vorbereiten möchten.
2. Suchen Sie die Liste der **Rollen**, die dem Veranstaltungsort der Lektion zugeordnet sind.
3. Klicken Sie auf einen Rollennamen, um den Leitfaden für diese Rolle zu öffnen.

Der Leitfaden zeigt jeden Abschnitt als schrittweise Anleitung an. Freiwillige können während des Unterrichts auf einem Smartphone, Tablet oder Computer durchscrollen.

## Leitfäden drucken

Wenn Ihre Freiwilligen eine gedruckte Version bevorzugen, können Sie jeden Leitfaden direkt aus dem Browser drucken.

1. Öffnen Sie den Leitfaden für die gewünschte Rolle.
2. Nutzen Sie die **Drucken**-Funktion Ihres Browsers (**Strg+P** oder **Cmd+P**).
3. Drucken Sie die Seite oder speichern Sie sie als PDF, um sie später zu verteilen.

:::tip
Teilen Sie Leitfaden-Links im Voraus mit Ihren Freiwilligen, damit sie ihre Anweisungen vor dem Unterricht durchsehen können. Ein kurzes Durchlesen unter der Woche hilft allen, am Sonntagmorgen besser vorbereitet zu sein.
:::

## Mehrere Rollen

Einige Lektionen haben mehrere Rollen -- zum Beispiel einen Hauptlehrer, einen Kleingruppenleiter und einen Technik-Freiwilligen. Jede Rolle hat ihren eigenen Leitfaden, sodass Sie so viele spezialisierte Leitfäden haben können, wie die Lektion vorsieht.

:::info
Wenn eine Rolle für Ihr Klassenzimmer nicht zutrifft, können Sie sie mit den [Anpassungsfunktionen](../customization/customizing-lessons.md) ausblenden, damit Ihre Freiwilligen nur die Rollen sehen, die relevant sind.
:::
