# Präsentationsmodus

<div class="article-intro">

Der Präsentationsmodus verwandelt eine Lektion in eine projektionsfreundliche Vollbildanzeige. Er ist dafür konzipiert, Lektionsinhalte auf Klassenzimmerbildschirmen, Projektoren oder Fernsehern anzuzeigen, damit alle im Raum während der Unterrichtszeit folgen können.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Planen Sie eine Lektion](../classrooms/scheduling-lessons.md) für ein Klassenzimmer, oder navigieren Sie über die [Inhaltsbibliothek](../browsing/index.md) zu einer Lektion
- Stellen Sie sicher, dass Ihr Anzeigegerät (Projektor, Fernseher oder Monitor) angeschlossen und konfiguriert ist

</div>

## Den Präsentationsmodus starten

1. Öffnen Sie die Lektion, die Sie präsentieren möchten.
2. Klicken Sie auf die Schaltfläche **Present** auf der Lektionsseite.
3. Die Lektion wechselt in eine Vollbildansicht mit schwarzem Hintergrund und großem, gut lesbarem Text.

Sobald Sie sich im Präsentationsmodus befinden, können Sie die Lektion Abschnitt für Abschnitt mit den Navigationsschaltflächen auf dem Bildschirm durchgehen.

## Wie es aussieht

Der Präsentationsmodus verwendet einen sauberen, dunklen Hintergrund, damit die Inhalte auf einem projizierten Bildschirm hervorstechen. Jeder Abschnitt der Lektion wird einzeln angezeigt, sodass der Fokus auf der aktuellen Aktivität bleibt. Text, Bilder und eingebettete Videos sind so formatiert, dass sie quer durch den Raum lesbar sind.

## Verwendung mit FreeShow

Lessons.church lässt sich mit [FreeShow](https://freeshow.app) integrieren, einer kostenlosen Präsentationsanwendung. FreeShow kann Lektionsinhalte direkt von Lessons.church abrufen, sodass Sie Lektionsfolien in Ihren bestehenden Präsentationsablauf einbinden können. Dies ist besonders nützlich, wenn Sie FreeShow bereits für Gottesdienstliedtexte oder Ankündigungen verwenden.

## Verwendung mit der Venue TV App

Die **Venue TV** Android-App ermöglicht es Ihnen, Lektionen auf einem Fernseher ohne separaten Computer anzuzeigen. Installieren Sie die App auf einem Android TV-Gerät, verbinden Sie sie mit Ihrem Lessons.church-Klassenzimmer, und die geplante Lektion erscheint automatisch. Dies ist eine großartige freihändige Option für Klassenzimmer mit einem eigenen Bildschirm.

:::tip
Wenn Sie einen Projektor verwenden, stellen Sie sicher, dass Ihr Display auf Spiegeln oder Erweitern auf den Projektor eingestellt ist, bevor Sie den Präsentationsmodus starten. So wird sichergestellt, dass die Vollbildansicht auf dem richtigen Bildschirm erscheint.
:::

## Den Präsentationsmodus verlassen

Um den Präsentationsmodus zu verlassen, drücken Sie die **Escape**-Taste oder klicken Sie auf die Schaltfläche zum Beenden in der Ecke des Bildschirms. Sie kehren zur normalen Lektionsansicht zurück.
