# Downloads und Materialien

<div class="article-intro">

Lektionen auf Lessons.church enthalten herunterladbare Ressourcen, die Sie ausdrucken oder digital teilen können. Diese Materialien unterstützen den Lektionsinhalt und geben Ihren Freiwilligen und Schülern etwas Greifbares, mit dem sie während und nach dem Unterricht arbeiten können.

</div>

<div class="prereqs">
<h4>Bevor Sie beginnen</h4>

- [Navigieren Sie zu einer Lektion](../browsing/viewing-lessons.md) oder öffnen Sie eine geplante Lektion aus Ihrem [Klassenzimmer-Zeitplan](../classrooms/scheduling-lessons.md)

</div>

## Arten von Materialien

Je nach Lektion und Anbieter finden Sie möglicherweise Ressourcen wie:

- **Arbeitsblätter** -- Praktische Übungsblätter, die Schüler während des Unterrichts bearbeiten können.
- **Ausmalseiten** -- Druckbare Seiten für jüngere Altersgruppen.
- **Leitfäden für Leiter** -- Detaillierte Anweisungen und Hintergrundinformationen für Lehrer.
- **Ergänzende PDFs** -- Zusätzliche Lektüre, Informationsblätter zum Mitnehmen oder Elternhandouts.

Nicht jede Lektion enthält all diese Materialien. Die verfügbaren Materialien hängen davon ab, was der Inhaltsanbieter bereitgestellt hat.

## Wo Sie Downloads finden

Ressourcen sind auf drei Ebenen verfügbar:

- **Programmebene** -- Allgemeine Ressourcen, die für die gesamte Lehrplanreihe gelten.
- **Studienreihenebene** -- Materialien, die an eine bestimmte Studienreihe innerhalb eines Programms gebunden sind.
- **Lektionsebene** -- Ressourcen für eine einzelne Lektion.

So laden Sie Materialien herunter:

1. Navigieren Sie zur Lektions-, Studienreihen- oder Programmseite.
2. Suchen Sie den Bereich **Downloads** oder **Resources** auf der Seite.
3. Klicken Sie auf einen Dateinamen, um ihn auf Ihr Gerät herunterzuladen.

Dateien liegen typischerweise im PDF-Format vor und sind druckfertig.

## Vorbereitung auf den Unterricht

Eine gute Routine ist es, einige Tage vor dem Unterricht nach herunterladbaren Materialien zu schauen:

1. Öffnen Sie die bevorstehende Lektion und prüfen Sie die verfügbaren Downloads.
2. Drucken Sie genügend Exemplare der Arbeitsblätter und Ausmalseiten für die erwartete Teilnehmerzahl.
3. Sehen Sie den Leitfaden für Leiter durch, damit Sie mit dem Lektionsablauf vertraut sind.
4. Legen Sie Materialien zum Mitnehmen bereit, die am Ende des Unterrichts verteilt werden sollen.

:::info
Einige Anbieter stellen alle ihre Materialien auf Studienreihenebene zur Verfügung, anstatt sie an einzelne Lektionen anzuhängen. Wenn Sie auf einer bestimmten Lektionsseite keine Downloads sehen, prüfen Sie die übergeordnete Studienreihen- oder Programmseite.
:::

:::tip
Erstellen Sie eine wöchentliche Checkliste für Ihre Freiwilligen, die das Herunterladen und Ausdrucken von Materialien umfasst. So wird sichergestellt, dass vor dem Sonntagmorgen nichts vergessen wird.
:::
