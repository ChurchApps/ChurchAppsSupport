# FreePlay 개요

<div class="article-intro">

FreePlay는 교실 TV에 레슨 콘텐츠를 표시하는 교회용 TV 앱입니다. Android TV, Amazon Fire TV, Apple TV에서 실행되며, 주일 오전에 인터넷 연결 없이 작동하도록 설계되었습니다. 봉사자들은 주중에 레슨 자료를 미리 다운로드한 후, 수업이 시작될 때 재생 버튼만 누르면 됩니다.

</div>

## 주요 기능

- **오프라인 재생** - 서비스 중 Wi-Fi에 의존하지 않도록 동영상과 이미지 슬라이드를 미리 다운로드합니다
- **교실 페어링** - TV를 특정 방에 연결하면 예정된 레슨 콘텐츠를 자동으로 받습니다
- **플랜 페어링** - 기기를 교회 서비스 플랜에 연결하여 방 간 콘텐츠를 조율합니다
- **콘텐츠 제공자** - [Lessons.church](/docs/lessons-church/) 및 Bible Project와 같은 외부 소스에서 콘텐츠를 탐색하고 다운로드합니다
- **TV 리모컨 친화적** - 방향 버튼과 선택 버튼을 사용하는 표준 TV 리모컨으로 앱 전체를 탐색합니다
- **접이식 사이드바** - 슬림한 사이드바가 화면을 어지럽히지 않으면서 탐색을 편리하게 유지합니다

## 작동 방식

FreePlay를 실행하면 콘텐츠 제공자에 연결하고 사용 가능한 콘텐츠를 확인합니다. 이후 폴더를 탐색하고, 미디어를 다운로드하며, 레슨을 재생할 수 있습니다. 모든 기능이 대형 화면 TV 경험에 최적화되어 있습니다.
