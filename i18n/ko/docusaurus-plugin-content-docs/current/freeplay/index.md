# FreePlay 개요

<div class="article-intro">

FreePlay는 교실 TV에 레슨 콘텐츠를 표시하는 교회용 TV 앱입니다. Android TV, Amazon Fire TV, Apple TV에서 실행되며 일요일 아침에 인터넷 연결 없이 작동하도록 설계되었습니다. 자원봉사자는 주 동안 레슨 자료를 미리 다운로드한 다음 수업이 시작될 때 재생을 누르면 됩니다.

</div>

:::tip FreePlay를 처음 사용하나요? 여기서 시작하세요
우리의 **<a href="/guides/freeplay-quickstart" target="_blank">빠른 시작 비디오</a>**를 봅니다 — 몇 분 만에 FreePlay를 설치하고 콘텐츠 재생을 시작합니다. 기술 기술이 필요하지 않습니다.
:::

:::tip 더 나아가고 싶으신가요?
우리의 **<a href="/guides/freeplay-b1admin" target="_blank">전체 설정 가이드</a>**는 B1 Admin을 연결하여 올바른 레슨이 매주 자동으로 교실 TV에 다운로드되도록 안내합니다. 비디오 및 작성된 단계가 포함되어 있습니다.
:::

## 주요 기능

- **오프라인 재생** - 예배 중에 Wi-Fi에 의존하지 않도록 미리 비디오 및 이미지 슬라이드 다운로드
- **다운로드 관리** - 전용 다운로드 탭에서 모든 다운로드된 콘텐츠 보기 및 관리
- **교실 페어링** - TV를 특정 방으로 연결하고 예정된 레슨 콘텐츠를 자동으로 받기
- **계획 페어링** - 기기를 교회 예배 계획에 연결하여 방 전체에서 조정된 콘텐츠
- **콘텐츠 제공자** - [Lessons.church](/docs/lessons-church/) 및 Bible Project와 같은 외부 소스에서 콘텐츠 찾아보기 및 다운로드
- **TV 리모컨 친화적** - 방향 버튼과 선택을 사용하는 표준 TV 리모컨으로 전체 앱을 탐색
- **축소 가능한 사이드바** - 슬림 사이드바는 네비게이션을 액세스 가능하게 유지하면서 화면을 지저분하게 하지 않습니다

## 작동 원리

FreePlay를 시작할 때 콘텐츠 제공자에 연결하고 사용 가능한 콘텐츠를 확인합니다. 여기에서 폴더를 찾아보고, 미디어를 다운로드하고, 레슨을 재생할 수 있습니다 -- 모두 대형 화면 TV 환경에 최적화되어 있습니다.
