# 플랜 모드

<div class="article-intro">

플랜 모드는 TV를 교회 서비스 플랜에 연결합니다. 교실 일정을 기반으로 콘텐츠를 가져오는 대신, B1 Admin에서 교회 리더십이 구성한 플랜을 따릅니다. 현재 주의 플랜에 배정된 콘텐츠를 정확히 TV에 표시하고 싶을 때 이상적입니다.

</div>

## 교실 모드와의 차이점

- **[교실 모드](../classroom-mode/)** 는 특정 날짜에 특정 방에 예정된 레슨을 가져옵니다
- **플랜 모드** 는 플랜 유형에 연결된 콘텐츠를 가져오며, 이는 레슨, 제공자 프레젠테이션, 추가 기능, 그 밖의 미디어가 사용자 지정 순서로 포함될 수 있습니다

플랜 모드는 서비스 플랜이 다양한 소스의 콘텐츠 혼합을 포함하고 모두 단일 순서로 구성된 경우에 특히 유용합니다.

## 작동 방식

1. FreePlay가 화면에 페어링 코드를 생성합니다
2. 관리자가 플랜 유형 설정에서 **B1 Admin**에 해당 코드를 입력합니다
3. 페어링이 완료되면 FreePlay가 현재 플랜을 로드하고 관련 미디어를 모두 다운로드합니다
4. **플랜 시작**을 눌러 재생을 시작합니다

TV는 플랜 유형에 페어링된 상태로 유지됩니다. 매주 자동으로 최신 플랜과 콘텐츠를 가져옵니다.
