# 기기 페어링하기

<div class="article-intro">

플랜 모드를 사용하려면 B1 Admin의 플랜 유형과 TV를 페어링해야 합니다. FreePlay가 기기를 교회의 플랜에 연결하는 고유한 페어링 코드를 생성하여 매주 자동으로 콘텐츠를 전달할 수 있도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- FreePlay를 설치하고 실행하십시오 -- [시작하기](../getting-started/)를 참조하십시오
- [페어링 모드 화면](../getting-started/pairing-modes.md)에서 **플랜에 페어링**을 선택하십시오
- 페어링을 완료하려면 컴퓨터나 휴대폰에서 **B1 Admin**에 접근할 수 있어야 합니다

</div>

## 페어링 코드 생성하기

1. **페어링 모드 선택** 화면에서 **플랜에 페어링**을 선택하십시오
2. FreePlay가 서버에 연결하여 짧은 페어링 코드를 생성합니다
3. 코드가 화면 중앙에 큰 글자로 표시됩니다
4. 코드 아래에 깜빡이는 표시기가 **연결 대기 중**을 표시합니다

코드는 방 건너편에서도 쉽게 읽을 수 있도록 개별 문자로 표시됩니다.

## B1 Admin에서 코드 입력하기

1. 컴퓨터 또는 휴대폰에서 교회의 **B1 Admin**을 여십시오
2. 배정하려는 플랜의 **플랜 유형** 설정으로 이동하십시오
3. TV 화면에 표시된 페어링 코드를 입력하십시오
4. 설정을 저장하십시오

FreePlay는 몇 초마다 서버에 페어링이 완료되었는지 확인합니다. **B1 Admin**이 연결을 확인하면 TV가 자동으로 플랜 다운로드 화면으로 이동합니다.

## 플랜 콘텐츠 다운로드하기

페어링 후 FreePlay가 해당 플랜 유형의 현재 플랜을 로드합니다. 다음이 표시됩니다:

- 플랜 이름과 서비스 날짜
- 연관된 레슨 이름 (플랜에 레슨 콘텐츠가 포함된 경우)
- **항목 X/Y 다운로드 중**을 표시하는 진행률 표시기

모든 미디어 파일이 다운로드되면 **플랜 시작** 버튼이 나타납니다. 리모컨의 **선택**을 눌러 재생을 시작하십시오.

:::tip
플랜은 매 시간마다 자동으로 새로고침됩니다. 낮 동안 플랜이 업데이트되면 FreePlay가 수동 개입 없이 변경 사항을 반영합니다.
:::

## 대안: 교회 이름으로 검색

페어링 코드 흐름을 사용하지 않으려면 페어링 화면 하단의 **또는 교회 이름으로 검색**을 선택하십시오. 이를 통해 교회 검색 화면으로 이동하여 교회를 찾고 교실에 연결할 수 있습니다.

## 페어링이 실패하는 경우

페어링 코드를 생성할 수 없는 경우 (예: 네트워크 문제), **다시 시도** 버튼이 있는 오류 메시지가 표시됩니다. TV가 인터넷에 연결되어 있는지 확인하고 다시 시도하십시오.

:::warning
페어링 코드는 일정 시간이 지나면 만료됩니다. 너무 오래 기다린 경우 페어링 화면으로 돌아가 새 코드를 생성하십시오.
:::
