# FreePlay 시작하기

<div class="article-intro">

FreePlay는 Android TV, Amazon Fire TV, Apple TV에서 사용할 수 있습니다. 이 가이드는 앱 설치, 처음 실행, 앱 전반에 걸쳐 사용할 기본 TV 리모컨 컨트롤 학습 과정을 안내합니다.

</div>

## FreePlay 설치하기

### Android TV / Amazon Fire TV

1. TV에서 앱 스토어(**Google Play Store** 또는 **Amazon Appstore**)를 여십시오
2. **FreePlay**를 검색하십시오
3. 앱을 선택하고 **설치**를 선택하십시오

### Apple TV

1. Apple TV에서 **App Store**를 여십시오
2. **FreePlay**를 검색하십시오
3. 앱을 선택하고 **받기**를 선택하십시오

## 앱 실행하기

FreePlay를 처음 열면 FreePlay 로고가 있는 애니메이션 스플래시 화면이 표시됩니다. 앱이 이전에 연결된 콘텐츠 제공자를 확인합니다. 이미 제공자를 연결한 경우 해당 제공자의 콘텐츠 브라우저로 바로 이동합니다. 그렇지 않으면 첫 번째 콘텐츠 소스를 연결할 수 있는 **콘텐츠 제공자** 화면이 열립니다.

## TV 리모컨으로 탐색하기

FreePlay는 TV 리모컨을 위해 설계되었습니다. 앱 전반에 걸쳐 사용할 컨트롤은 다음과 같습니다:

- **방향 패드 (위/아래/왼쪽/오른쪽)** - 화면의 항목 사이를 이동합니다
- **선택 / OK 버튼** - 강조 표시된 항목을 선택합니다
- **왼쪽 화살표** - 모든 화면에서 사이드바를 엽니다
- **오른쪽 화살표** - 사이드바를 닫고 메인 콘텐츠 영역으로 돌아갑니다
- **뒤로 / 메뉴 버튼** - 이전 화면으로 돌아갑니다

:::tip
콘텐츠를 탐색할 때 사이드바가 좁은 아이콘 띠로 축소됩니다. 리모컨의 왼쪽 화살표를 눌러 사이드바를 펼치면 전체 레이블을 볼 수 있습니다.
:::
