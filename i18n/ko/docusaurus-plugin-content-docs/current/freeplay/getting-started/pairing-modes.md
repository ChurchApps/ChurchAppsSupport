# 페어링 모드 선택하기

<div class="article-intro">

FreePlay는 TV를 교회에 연결하는 두 가지 페어링 모드와 콘텐츠 제공자를 직접 탐색하는 기능을 제공합니다. **페어링 모드 선택** 화면에서 이러한 옵션이 TV 리모컨으로 탐색할 수 있는 대형 카드로 표시됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- FreePlay를 설치하고 처음 실행하십시오 -- [시작하기](./index.md)를 참조하십시오
- 탐색을 위해 TV 리모컨을 준비하십시오

</div>

## 교실에 페어링

특정 방에 예정된 레슨을 특정 TV가 항상 표시하도록 하려면 **교실에 페어링**을 선택하십시오. 이 옵션은 주간 일정에 따라 레슨이 배정되는 어린이 사역 교실에 가장 적합합니다.

**작동 방식:** 교회를 이름으로 검색하고, 교실을 선택하면 FreePlay가 오늘 해당 방에 예정된 레슨을 자동으로 가져옵니다. 매주 일요일 아침에 TV를 켜면 올바른 레슨이 준비되어 있습니다.

**최적 용도:** 항상 해당 방에 예정된 콘텐츠를 표시해야 하는 전용 교실 TV.

:::tip
TV를 페어링하기 전에 [freeplay.church](https://freeplay.church)에서 교실을 설정하십시오. 이렇게 하면 교회를 검색할 때 방이 나타납니다.
:::

## 플랜에 페어링

교회 서비스 플랜을 따르려면 **플랜에 페어링**을 선택하십시오. 이 모드는 **B1 Admin**에서 입력하는 페어링 코드를 생성하여 TV를 특정 플랜 유형에 연결합니다. 그러면 TV가 현재 플랜에 배정된 콘텐츠를 다운로드하고 재생합니다.

**작동 방식:** FreePlay가 화면에 페어링 코드를 표시합니다. 관리자가 **B1 Admin**의 **플랜 유형** 설정에 해당 코드를 입력합니다. 페어링이 완료되면 TV가 자동으로 현재 플랜의 콘텐츠를 로드합니다.

**최적 용도:** 예배 가사나 공지 슬라이드와 같이 주간 서비스 플랜에 연결된 콘텐츠를 표시하는 TV.

## 콘텐츠 제공자

교실이나 플랜에 페어링할 필요가 없는 경우 콘텐츠 제공자를 직접 탐색할 수 있습니다. [Lessons.church](../../lessons-church/) 및 Bible Project와 같은 제공자는 주문형으로 다운로드하고 재생할 수 있는 콘텐츠를 제공합니다.

**최적 용도:** 봉사자가 매주 재생할 콘텐츠를 직접 선택하는 TV, 또는 사용 가능한 콘텐츠를 탐색하려는 경우.

:::info
**제공자** 화면은 탐색 하단의 사이드바에서 언제든지 접근할 수 있습니다. 콘텐츠를 탐색하고 재생하기 위해 페어링 모드를 사용할 필요는 없습니다.
:::
