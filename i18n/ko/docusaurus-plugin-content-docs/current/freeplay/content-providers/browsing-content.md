# 콘텐츠 탐색 및 다운로드

<div class="article-intro">

콘텐츠 제공자에 연결하면 라이브러리를 탐색하고, 파일을 다운로드하며, TV에서 재생할 수 있습니다. 콘텐츠 브라우저는 TV 리모컨으로 탐색하는 폴더와 파일의 시각적 그리드를 제공합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 최소 하나의 콘텐츠 제공자에 연결하십시오 -- [제공자에 연결하기](./connecting-providers)를 참조하십시오
- 탐색을 위해 TV 리모컨을 준비하십시오

</div>

## 콘텐츠 브라우저

콘텐츠 브라우저는 카드의 3열 그리드로 항목을 표시합니다. 각 카드에는 썸네일 이미지와 제목이 표시됩니다. 항목에는 두 가지 유형이 있습니다:

- **폴더** - 썸네일 또는 폴더 아이콘으로 표시됩니다. 폴더를 선택하면 내용을 볼 수 있습니다.
- **파일** - 개별 미디어 항목 (동영상 또는 이미지). 동영상 파일은 썸네일에 재생 아이콘 오버레이가 표시됩니다.

화면 상단의 헤더에는 현재 폴더의 이름이 표시되며, 루트 레벨에 있는 경우 제공자 이름이 표시됩니다.

## 폴더 탐색하기

1. 리모컨의 방향 패드를 사용하여 폴더를 강조 표시하십시오
2. **선택**을 눌러 여십시오
3. 브라우저가 폴더의 내용을 로드합니다. 내용은 더 많은 폴더이거나 재생 가능한 파일일 수 있습니다
4. 리모컨의 **뒤로**를 눌러 한 단계 위로 이동하십시오

제공자의 콘텐츠 구조가 허용하는 만큼 깊이 탐색할 수 있습니다. 브라우저가 폴더 경로를 추적하므로 각 레벨을 통해 되돌아갈 수 있습니다.

## 콘텐츠 다운로드 및 재생

재생 가능한 파일이 포함된 폴더를 선택하거나 개별 파일을 선택하면 FreePlay가 다운로드 화면으로 이동합니다:

1. 선택한 폴더의 커버 이미지와 함께 콘텐츠 제목이 화면에 표시됩니다
2. 진행률 표시줄에 파일이 기기에 저장될 때 **항목 X/Y 다운로드 중**이 표시됩니다
3. 다운로드가 완료되면 재생 아이콘과 함께 **시작** 버튼이 나타납니다
4. **선택**을 눌러 재생을 시작하십시오

:::tip
다운로드된 콘텐츠는 TV에 캐시됩니다. 나중에 동일한 콘텐츠를 다시 재생하면 재다운로드 없이 즉시 로드됩니다.
:::

## 제공자 콘텐츠 재생하기

재생은 레슨 재생과 동일하게 작동합니다. 다음 컨트롤을 사용하십시오:

- **오른쪽 화살표** - 다음 항목
- **왼쪽 화살표** - 이전 항목
- **선택 / 재생-일시정지** - 동영상 일시정지 또는 재개
- **위쪽 화살표** - 특정 항목으로 이동하는 메시지 선택기 열기
- **뒤로** - 콘텐츠 브라우저로 종료

플레이어를 종료하면 현재 제공자의 루트 레벨에 있는 콘텐츠 브라우저로 돌아갑니다.

:::info
모든 플레이어 컨트롤의 전체 참조는 [레슨 재생](../classroom-mode/playing-lessons)을 참조하십시오.
:::

## 제공자 간 전환하기

연결된 제공자는 왼쪽 사이드바에 항목으로 표시됩니다. 다른 제공자로 전환하려면:

1. 리모컨의 **왼쪽** 화살표를 눌러 사이드바를 펼치십시오
2. 탐색하려는 제공자를 선택하십시오
3. 콘텐츠 브라우저가 해당 제공자의 라이브러리로 새로고침됩니다

:::warning
폴더에 "사용 가능한 콘텐츠 없음"이 표시되면 해당 레벨에 제공자의 항목이 없는 것입니다. 다른 폴더로 이동하거나 나중에 다시 확인하십시오.
:::
