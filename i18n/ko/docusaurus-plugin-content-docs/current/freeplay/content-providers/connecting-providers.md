# 제공자에 연결하기

<div class="article-intro">

제공자의 콘텐츠를 탐색하려면 먼저 연결해야 합니다. 일부 제공자는 QR 코드나 이메일 로그인을 통한 인증이 필요하고, 다른 제공자는 한 번의 클릭으로 연결할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- FreePlay를 설치하고 실행하십시오 -- [시작하기](../getting-started/)를 참조하십시오
- 탐색을 위해 TV 리모컨을 준비하십시오
- 로그인이 필요한 제공자의 경우 계정 자격 증명을 준비하십시오

</div>

## 사용 가능한 제공자 탐색하기

1. 사이드바에서 **콘텐츠 제공자** 화면을 여십시오 (하단의 **제공자** 아이콘 선택)
2. 각 제공자의 로고와 이름이 표시된 제공자 카드 그리드가 표시됩니다
3. 연결된 제공자는 이름 아래에 녹색 **연결됨** 배지가 표시됩니다
4. 아직 사용할 수 없는 제공자는 **출시 예정** 라벨이 표시됩니다

## 인증 없이 연결하기

일부 제공자는 로그인이 필요하지 않습니다. 이러한 제공자 중 하나를 선택하면 FreePlay가 즉시 연결되고 콘텐츠 브라우저가 열립니다. 자격 증명이 필요하지 않습니다.

## 기기 흐름 인증 (QR 코드)

일부 제공자는 TV의 스트리밍 앱에 로그인하는 방식과 유사한 기기 흐름을 사용합니다:

1. **콘텐츠 제공자** 화면에서 제공자 카드를 선택하십시오
2. FreePlay가 QR 코드와 인증 URL을 표시합니다
3. 스마트폰으로 QR 코드를 스캔하거나, 다른 기기에서 표시된 URL을 방문하십시오
4. TV 화면에 표시된 사용자 코드를 입력하십시오
5. 스마트폰이나 컴퓨터에서 로그인 프로세스를 완료하십시오
6. FreePlay가 로그인 성공을 감지하고 **연결됨!**을 표시합니다
7. 콘텐츠 브라우저가 자동으로 열립니다

:::info
깜빡이는 **인증 대기 중** 표시기는 FreePlay가 로그인을 확인 중임을 나타냅니다. 코드는 몇 분 후에 만료되므로 빠르게 프로세스를 완료하십시오.
:::

## 폼 로그인

다른 제공자는 기존의 이메일과 비밀번호 로그인을 사용합니다:

1. 제공자 카드를 선택하십시오
2. 화면 키보드를 사용하여 **이메일**과 **비밀번호**를 입력하십시오
3. **로그인** 버튼을 선택하십시오
4. 자격 증명이 올바르면 FreePlay가 **연결됨!**을 표시하고 콘텐츠 브라우저를 엽니다

:::tip
리모컨의 방향 패드를 사용하여 이메일 필드, 비밀번호 필드, 로그인 버튼 사이를 이동하십시오. 텍스트 필드에서 **선택**을 눌러 화면 키보드를 여십시오.
:::

## 제공자 연결 해제하기

이미 연결한 제공자의 연결을 해제하려면:

1. **콘텐츠 제공자** 화면으로 이동하십시오
2. **연결됨** 배지가 표시된 제공자 카드를 선택하십시오
3. 연결 해제 여부를 묻는 확인 메시지가 나타납니다
4. **연결 해제**를 선택하여 연결을 제거하십시오

연결을 해제하면 해당 제공자의 콘텐츠가 더 이상 사이드바에 표시되지 않습니다.

:::warning
연결 해제 시 기기에서 저장된 인증 정보가 제거됩니다. 나중에 다시 연결하려면 다시 로그인해야 합니다.
:::
