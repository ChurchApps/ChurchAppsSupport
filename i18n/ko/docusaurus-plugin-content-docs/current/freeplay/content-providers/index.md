# 콘텐츠 제공자

<div class="article-intro">

콘텐츠 제공자는 TV에서 재생할 레슨 자료, 동영상, 이미지를 공급하는 외부 서비스입니다. FreePlay는 여러 제공자를 지원하므로 하나의 앱에서 다양한 소스의 콘텐츠를 탐색하고 다운로드할 수 있습니다.

</div>

## 사용 가능한 제공자

FreePlay는 현재 다음 제공자를 포함합니다:

- **[Lessons.church](../../lessons-church/)** - 무료 오픈소스 어린이 사역 커리큘럼
- **Bible Project** - 성경을 탐구하는 동영상 및 리소스
- **SignPresenter** - 교회를 위한 프레젠테이션 및 사이니지 콘텐츠
- **B1.church** - 자체 B1 교회 설정의 콘텐츠

:::info
일부 제공자는 아직 추가 중입니다. 제공자 카드에 "출시 예정"이 표시되면 아직 연결할 수 없는 상태입니다.
:::

## 제공자 작동 방식

각 제공자는 폴더와 파일로 구성된 자체 콘텐츠 라이브러리를 가지고 있습니다. 제공자에 연결하면 해당 콘텐츠가 폴더 브라우저에 표시되어 카테고리를 탐색하고, 항목을 선택하며, 재생을 위해 다운로드할 수 있습니다.

연결된 제공자는 화면 왼쪽의 사이드바에 표시됩니다. 사이드바에서 제공자를 선택하여 전환할 수 있으므로 설정 중에 콘텐츠 소스 간에 쉽게 이동할 수 있습니다.
