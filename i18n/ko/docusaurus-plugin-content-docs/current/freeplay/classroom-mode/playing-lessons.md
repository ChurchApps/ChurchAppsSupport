# 레슨 재생

<div class="article-intro">

FreePlay 미디어 플레이어는 TV 리모컨을 위해 설계된 전체 화면 경험을 제공합니다. 동영상과 이미지 슬라이드를 모두 재생하며, 레슨 콘텐츠를 자동 또는 수동으로 진행하여 교사가 레슨 속도를 완전히 제어할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 레슨을 다운로드하십시오 -- [레슨 콘텐츠 다운로드](./downloading-content)를 참조하십시오
- **레슨 시작** 버튼을 눌러 플레이어에 진입하십시오

</div>

## 플레이어 컨트롤

TV 리모컨으로 재생을 제어하십시오:

| 리모컨 버튼 | 동작 |
|---|---|
| **선택 / 재생-일시정지** | 현재 동영상을 일시정지하거나 재개합니다 |
| **오른쪽 화살표 / 빨리감기** | 다음 슬라이드 또는 동영상으로 건너뜁니다 |
| **왼쪽 화살표 / 되감기** | 이전 슬라이드 또는 동영상으로 돌아갑니다 |
| **위쪽 화살표** | 특정 항목으로 바로 이동하는 메시지 선택기를 엽니다 |
| **아래쪽 / 뒤로 / 메뉴** | 플레이어를 종료하고 이전 화면으로 돌아갑니다 |

:::tip
터치 지원 리모컨에서는 스와이프도 사용할 수 있습니다. 왼쪽으로 스와이프하면 앞으로 이동하고, 오른쪽으로 스와이프하면 뒤로 이동하며, 아래로 스와이프하면 메시지 선택기가 열리고, 위로 스와이프하면 플레이어가 종료됩니다.
:::

## 동영상 재생

동영상이 재생 중일 때 전체 화면 동영상 콘텐츠가 표시됩니다. **선택** 또는 **재생-일시정지**를 누르면 동영상이 일시정지되고 화면 하단에 진행률 표시줄과 함께 일시정지 아이콘 오버레이가 표시됩니다. 다시 누르면 재개됩니다.

동영상이 끝나면 레슨의 다음 항목으로 자동 진행됩니다. 동영상이 반복 재생으로 설정된 경우 수동으로 진행할 때까지 반복됩니다.

## 이미지 슬라이드

이미지 슬라이드는 전체 화면으로 표시됩니다. 동영상과 달리 이미지 슬라이드는 자동으로 넘어가지 않으며, 오른쪽 화살표를 눌러 앞으로 이동할 때까지 화면에 유지됩니다. 이를 통해 교사가 수업 속도를 완전히 제어할 수 있습니다.

## 메시지 선택기

재생 중 **위쪽** 화살표를 눌러 메시지 선택기를 여십시오. 레슨의 모든 항목이 썸네일과 제목이 있는 그리드로 표시됩니다. 원하는 항목으로 이동하여 **선택**을 누르면 해당 항목으로 바로 이동합니다. 이 기능은 레슨의 특정 부분으로 건너뛰거나 돌아갈 때 유용합니다.

## 플레이어 종료하기

리모컨의 **아래쪽**, **뒤로**, 또는 **메뉴** 버튼을 눌러 플레이어를 종료하십시오. 교실 모드에서는 다운로드 화면으로, 플랜 모드에서는 플랜 다운로드 화면으로, 제공자 콘텐츠 재생 시에는 콘텐츠 브라우저로 돌아갑니다.

:::info
플레이어가 활성화된 동안 화면이 켜진 상태로 유지되므로 레슨 중 TV가 절전 모드로 전환될 걱정을 하지 않아도 됩니다.
:::
