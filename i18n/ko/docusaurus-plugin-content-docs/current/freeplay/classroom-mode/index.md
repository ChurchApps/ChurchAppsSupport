# 교실 모드

<div class="article-intro">

교실 모드는 TV를 교회의 특정 방에 연결합니다. 페어링이 완료되면 FreePlay가 해당 방에 예정된 레슨을 자동으로 다운로드하므로, 봉사자들은 수업이 시작될 때 **레슨 시작** 버튼만 누르면 됩니다.

</div>

## 작동 방식

1. **교회 찾기** - 이름으로 교회를 검색합니다
2. **방 선택** - 이 TV가 속한 교실을 선택합니다
3. **레슨 다운로드** - FreePlay가 예정된 레슨을 가져와 모든 미디어 파일을 다운로드합니다
4. **재생** - **레슨 시작** 버튼을 누르면 레슨이 자동으로 재생됩니다

TV는 어떤 교회와 방에 페어링되어 있는지 기억하므로 설정은 한 번만 하면 됩니다. 매주 FreePlay는 당일 예정된 레슨을 확인하고 새 콘텐츠를 다운로드합니다.

:::tip
TV를 페어링하기 전에 [freeplay.church](https://freeplay.church)에서 교실이 구성되어 있는지 확인하십시오. 교회를 선택했을 때 교실이 나타나지 않으면 웹사이트를 방문하여 설정하십시오.
:::
