# 레슨 콘텐츠 다운로드

<div class="article-intro">

교회와 교실을 선택하면 FreePlay가 오늘 예정된 레슨을 자동으로 가져와 모든 미디어 파일 다운로드를 시작합니다. 이 화면은 레슨을 시작하기 전에 콘텐츠가 준비될 때까지 기다리는 곳입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교회와 교실을 선택하십시오 -- [교회 및 방 선택하기](./selecting-room)를 참조하십시오
- 초기 다운로드를 위해 TV가 인터넷에 연결되어 있는지 확인하십시오

</div>

## 다운로드 작동 방식

1. FreePlay가 교실 일정을 확인하고 배정된 레슨을 로드합니다
2. 레슨의 커버 이미지 위에 레슨 이름, 제목, 설명이 화면에 표시됩니다
3. 각 파일이 기기에 저장될 때 진행률 표시기에 **항목 X/Y 다운로드 중**이 표시됩니다
4. 모든 파일이 다운로드되면 재생 아이콘과 함께 **레슨 시작** 버튼이 나타납니다

다운로드 중에는 아무것도 하지 않아도 됩니다. FreePlay가 모든 것을 자동으로 처리합니다.

## 해상도

FreePlay는 기본적으로 720p로 콘텐츠를 다운로드합니다. 해상도 설정(720p 또는 1080p)에 따라 다운로드되는 동영상 및 이미지 파일의 품질이 결정됩니다. 해상도가 높을수록 대형 TV에서 더 선명하게 보이지만 다운로드 시간이 더 걸리고 저장 공간을 더 많이 사용합니다.

:::info
해상도는 기본적으로 720p로 설정됩니다. 콘텐츠는 기기에 구성된 해상도로 다운로드됩니다.
:::

## 레슨 시작하기

다운로드가 완료되면 화면 하단에 **레슨 시작** 버튼이 나타납니다. TV 리모컨을 사용하여 버튼을 강조 표시하고 **선택**을 눌러 재생을 시작하십시오.

## 자동 새로고침

FreePlay는 매 시간마다 업데이트된 콘텐츠를 확인합니다. 낮 동안 일정이 변경되면 앱이 자동으로 새 레슨을 가져옵니다. 리모컨의 **뒤로** 버튼을 눌러 방 선택 화면으로 돌아간 후 다운로드 화면에 다시 진입하여 강제로 새로고침할 수도 있습니다.

:::tip
건물에 강한 인터넷 연결이 있는 주중에 레슨을 미리 다운로드하십시오. 다운로드가 완료되면 네트워크 의존 없이 오프라인으로 콘텐츠를 재생할 수 있습니다.
:::

## 일정이 비어 있는 경우

선택한 교실에 레슨이 예정되어 있지 않으면 "일정을 로드할 수 없습니다. 이 학급에 레슨이 예정되어 있는지 확인하십시오."라는 메시지가 표시됩니다. [freeplay.church](https://freeplay.church)에서 일정을 확인하고 올바른 레슨이 이 방에 배정되어 있는지 확인하십시오.
