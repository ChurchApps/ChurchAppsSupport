# 교회 및 방 선택하기

<div class="article-intro">

FreePlay가 교실의 레슨을 다운로드하려면 사용할 교회와 방을 지정해야 합니다. 이것은 일회성 설정으로, TV가 이후 세션을 위해 선택 사항을 기억합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- FreePlay를 설치하고 실행하십시오 -- [시작하기](../getting-started/)를 참조하십시오
- [freeplay.church](https://freeplay.church)에서 교회를 등록하고 교실을 구성하십시오
- [페어링 모드 화면](../getting-started/pairing-modes.md)에서 **교실에 페어링**을 선택하십시오

</div>

## 교회 찾기

1. **페어링 모드 선택** 화면에서 **교실에 페어링**을 선택하십시오
2. 상단에 검색 필드가 있는 **교회 찾기** 화면이 표시됩니다
3. TV 리모컨을 사용하여 검색 필드를 선택하고 교회 이름을 최소 네 글자 이상 입력하십시오
4. 검색 필드 아래에 일치하는 교회 목록이 나타납니다
5. 목록에서 교회로 이동하여 리모컨의 **선택** 버튼을 누르십시오

:::warning
검색 결과에 교회가 나타나지 않으면 아직 등록되지 않았을 수 있습니다. [freeplay.church](https://freeplay.church)를 방문하여 교회를 등록하고 교실을 구성하십시오.
:::

## 교실 선택하기

교회를 선택하면 구성된 모든 방 목록이 있는 **교실 선택** 화면이 나타납니다.

1. 리모컨의 위아래 화살표를 사용하여 교실 목록을 탐색하십시오
2. 이 TV가 배정된 방을 강조 표시하십시오
3. 리모컨의 **선택** 버튼을 누르십시오

FreePlay가 선택 사항을 저장하고 즉시 다운로드 화면으로 이동하여 예정된 레슨을 가져옵니다.

:::tip
"교실을 찾을 수 없습니다"라고 표시되면 교회의 교실이 아직 구성되지 않은 것입니다. 관리자가 [freeplay.church](https://freeplay.church)에서 설정할 수 있습니다.
:::

## 선택 변경하기

나중에 다른 교회나 방으로 변경하려면 다운로드 화면에서 TV 리모컨의 **뒤로** 버튼을 누르십시오. 그러면 방 선택 화면으로 돌아가 다른 교실을 선택하거나, **뒤로**를 계속 눌러 교회 검색으로 돌아갈 수 있습니다.
