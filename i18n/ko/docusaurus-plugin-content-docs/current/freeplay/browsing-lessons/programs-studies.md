# 프로그램 및 스터디

<div class="article-intro">

레슨 브라우저는 단계별 흐름을 사용하여 원하는 콘텐츠를 정확히 찾아냅니다. 각 단계에서는 이미지와 제목이 있는 카드의 3열 그리드가 표시되어, TV 리모컨으로 전체 레슨 라이브러리를 쉽게 탐색할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- TV에 인터넷 연결이 되어 있는지 확인하십시오 (프로그램 카탈로그 탐색에 필요)
- 사이드바를 사용하여 **프로그램 탐색** 화면으로 이동하십시오

</div>

## 프로그램 탐색하기

1. **프로그램 탐색** 화면을 여십시오
2. 프로그램이 이미지 카드의 그리드로 세 열로 표시됩니다
3. 리모컨의 방향 패드를 사용하여 프로그램을 강조 표시하십시오
4. **선택** 버튼을 눌러 열어보십시오

각 프로그램 카드에는 프로그램의 커버 이미지가 표시됩니다. 이미지가 없는 경우 플레이스홀더 카드가 표시됩니다.

## 스터디 선택하기

프로그램을 선택하면 해당 프로그램의 모든 스터디가 포함된 **스터디 선택** 화면이 로드됩니다.

1. 스터디 카드가 커버 이미지와 스터디 이름이 있는 3열 그리드로 표시됩니다
2. 원하는 스터디로 이동하여 **선택**을 누르십시오
3. **뒤로** 버튼을 눌러 프로그램 목록으로 돌아갑니다

## 레슨 선택하기

스터디를 선택하면 해당 스터디의 모든 레슨이 포함된 **레슨 선택** 화면이 표시됩니다.

1. 레슨 카드에 커버 이미지와 레슨 제목이 표시됩니다
2. 원하는 레슨으로 이동하여 **선택**을 누르십시오
3. **뒤로** 버튼을 눌러 스터디 목록으로 돌아갑니다

## 베뉴 선택하기

레슨 상세 화면에는 레슨의 커버 이미지 위에 레슨 이름, 제목, 설명이 겹쳐서 표시됩니다. 설명 아래에 하나 이상의 베뉴 버튼이 표시됩니다.

베뉴는 동일한 레슨의 다양한 버전을 나타냅니다. 예를 들어, 레슨에는 "유아부", "초등부", "중고등부" 연령 그룹별로 별도의 베뉴가 있을 수 있습니다. 각 베뉴에는 해당 환경에 적합한 미디어 파일이 포함되어 있습니다.

1. 방향 패드를 사용하여 원하는 베뉴를 강조 표시하십시오
2. **선택**을 눌러 콘텐츠 로드를 시작하십시오
3. FreePlay가 베뉴의 재생 목록을 가져와 플레이어를 시작합니다

:::tip
베뉴 버튼이 하나만 표시되면 해당 레슨의 버전이 하나뿐입니다. 선택하면 바로 재생이 시작됩니다.
:::

## 뒤로 탐색하기

탐색 흐름의 어느 단계에서든 리모컨의 **뒤로** 버튼을 눌러 이전 단계로 돌아갈 수 있습니다:

- 레슨 상세 화면에서 **뒤로**를 누르면 레슨 목록으로 돌아갑니다
- 레슨 목록에서 **뒤로**를 누르면 스터디 목록으로 돌아갑니다
- 스터디 목록에서 **뒤로**를 누르면 프로그램 그리드로 돌아갑니다
- 프로그램 그리드에서 **뒤로**를 누르면 사이드바가 열립니다
