# 레슨 탐색

<div class="article-intro">

FreePlay에는 사용 가능한 프로그램, 스터디, 레슨의 전체 라이브러리를 탐색할 수 있는 내장 레슨 브라우저가 포함되어 있습니다. 교실 일정이나 플랜에 의존하지 않고 직접 재생할 레슨을 선택하고 싶을 때 유용합니다.

</div>

## 탐색 흐름

레슨은 계층 구조로 구성되어 있습니다:

1. **프로그램** - 최상위 커리큘럼 카테고리 (예: 어린이 사역 시리즈)
2. **스터디** - 프로그램 내 레슨 그룹 (예: 다주 단위)
3. **레슨** - 자체 콘텐츠를 가진 개별 레슨 세션
4. **베뉴** - 특정 연령대나 환경에 맞게 조정된 레슨의 다양한 버전

프로그램을 선택한 후 스터디로 들어가고, 레슨을 선택한 뒤, 마지막으로 재생을 시작할 베뉴를 선택합니다.

## 레슨 브라우저 접근하기

레슨 브라우저는 **프로그램 탐색** 화면을 통해 이용할 수 있습니다. 사이드바를 사용하거나 탐색 메뉴에서 탐색 옵션을 선택하여 해당 화면으로 이동하십시오.

:::info
레슨 탐색에는 프로그램 카탈로그를 로드하기 위한 인터넷 연결이 필요합니다. 베뉴를 선택하고 재생을 시작하면 별도의 다운로드 없이 콘텐츠가 직접 스트리밍됩니다.
:::
