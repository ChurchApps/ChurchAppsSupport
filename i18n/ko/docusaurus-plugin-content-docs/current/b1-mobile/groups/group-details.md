# 그룹 상세

<div class="article-intro">

그룹 화면에서 그룹을 탭하면 해당 그룹의 상세 페이지로 이동합니다. 여기에서 그룹에 대한 모든 것을 확인하고 게시물, 구성원 목록, 그룹 정보를 포함한 콘텐츠와 상호작용할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 모바일 설치](../getting-started/installing) 및 계정으로 [로그인](../getting-started/logging-in)
- [그룹](./index.md) 화면 열기

</div>

## 화면 구성

그룹 상세 페이지에서 그룹의 전체 프로필을 볼 수 있습니다:

- **그룹 정보** -- 그룹 이름, 설명, 모임 세부 정보, 그룹 리더가 제공한 기타 정보
- **게시물 및 타임라인** -- 이 그룹에 특정한 게시물과 활동의 피드. 리더와 구성원이 공유한 업데이트, 공지사항, 대화를 볼 수 있습니다.
- **구성원** -- 그룹에 속한 사람을 확인합니다. 누구와 연결되어 있는지 확인하고 익숙한 얼굴을 알아볼 수 있습니다.

## 그룹 콘텐츠와 상호작용

그룹 상세 페이지에서 여러 가지 방법으로 그룹에 참여할 수 있습니다:

- **게시물 읽기** -- 그룹의 타임라인을 스크롤하여 최근 활동을 확인합니다.
- **구성원 프로필 보기** -- 구성원을 탭하면 [교인 명부](../community/member-directory.md)에서 프로필 정보를 볼 수 있습니다.
- **최신 정보 유지** -- 정기적으로 확인하여 그룹 리더와 동료 구성원의 새로운 게시물, 공지사항, 업데이트를 확인합니다.

## 접근 방법

1. B1 모바일에서 **그룹** 화면을 엽니다.
2. 히어로 그룹, 추천 카드 또는 목록의 아무 그룹이나 탭합니다.
3. 그룹의 상세 페이지가 모든 콘텐츠와 정보와 함께 열립니다.

:::tip
가장 많이 방문하는 그룹이 그룹 화면 상단에 히어로 카드로 나타나므로, 즐겨찾는 그룹에 항상 한 번의 탭으로 접근할 수 있습니다.
:::
