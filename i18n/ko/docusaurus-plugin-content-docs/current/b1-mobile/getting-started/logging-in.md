# 로그인하기

<div class="article-intro">

B1 모바일은 B1.church와 다른 ChurchApps 제품에서 사용하는 것과 동일한 ChurchApps 계정을 사용합니다. 이미 계정이 있으면 기존 이메일과 비밀번호로 로그인할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- iOS 또는 Android 기기에 [B1 모바일 설치](./installing)

</div>

## 로그인 방법

1. B1 모바일 앱을 엽니다.
2. 서랍 메뉴 또는 로그인 안내에서 **로그인** 옵션을 탭합니다.
3. **이메일 주소**와 **비밀번호**를 입력합니다.
4. **로그인**을 탭합니다.

로그인 후 [대시보드](../dashboard/)로 이동합니다. 아직 교회를 선택하지 않은 경우, 교회를 검색하고 선택하라는 안내가 표시됩니다.

## 새 계정 만들기

아직 ChurchApps 계정이 없으면 앱에서 직접 만들 수 있습니다:

1. 로그인 화면 하단에서 **회원가입** 링크를 탭합니다.
2. **이름**, **성**, **이메일 주소**를 입력합니다.
3. **회원가입**을 탭합니다.
4. 임시 비밀번호가 포함된 이메일을 확인합니다.
5. 앱으로 돌아가서 **로그인**을 탭하고, 이메일과 이메일에서 받은 임시 비밀번호로 로그인합니다.

처음 로그인한 후 [프로필 설정](../profile/editing-profile.md)에서 비밀번호를 변경할 수 있습니다.

:::warning
몇 분 이내에 가입 이메일을 받지 못한 경우, 스팸 또는 정크 폴더를 확인하십시오.
:::

## 비밀번호를 잊은 경우

비밀번호를 잊은 경우, 로그인 화면에서 **비밀번호 찾기**를 탭하십시오. 브라우저에서 이메일 주소를 입력하고 비밀번호 재설정 링크를 요청할 수 있는 페이지가 열립니다.

## 로그인 상태 유지

B1 모바일은 세션 간에 로그인 상태를 유지합니다. 앱을 열 때마다 로그인할 필요가 없습니다. 어떤 이유로 로그아웃된 경우, 이메일과 비밀번호로 다시 로그인하면 됩니다.

## 로그아웃

앱에서 로그아웃하려면:

1. 왼쪽 상단의 **햄버거 아이콘**(가로 세 줄)을 탭하여 서랍 메뉴를 엽니다.
2. 서랍 메뉴 하단으로 스크롤합니다.
3. **로그아웃**을 탭합니다.

로그아웃되며, 언제든지 자격 증명으로 다시 로그인할 수 있습니다.
