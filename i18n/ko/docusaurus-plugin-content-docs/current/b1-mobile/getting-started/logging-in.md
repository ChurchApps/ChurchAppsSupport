# 로그인

<div class="article-intro">

B1 Mobile은 B1.church 및 기타 ChurchApps 제품에서 사용하는 것과 동일한 ChurchApps 계정을 사용합니다. 이미 계정이 있으면 기존 이메일과 암호로 로그인할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- iOS 또는 Android 기기에 [B1 Mobile 설치](./installing)

</div>

## 로그인 방법

1. B1 Mobile 앱을 엽니다.
2. 드로어 메뉴 또는 로그인 프롬프트에서 **로그인** 옵션을 탭합니다.
3. **이메일 주소**와 **암호**를 입력합니다.
4. **로그인**을 탭합니다.

로그인한 후 [대시보드](../dashboard/)로 이동합니다. 아직 교회를 선택하지 않았으면 검색하고 선택하라는 메시지가 표시됩니다.

## 새 계정 만들기

아직 ChurchApps 계정이 없으면 앱에서 직접 만들 수 있습니다:

1. 로그인 화면에서 하단의 **등록** 링크를 탭합니다.
2. **이름**, **성**, **이메일 주소**를 입력합니다.
3. 정보에 따라 일치 항목이 발견되면 양식이 교회를 자동으로 채웁니다.
4. **등록**을 탭합니다.
5. 임시 암호가 있는 메시지에 대해 이메일을 확인합니다.
6. 앱으로 돌아가서 **로그인**을 탭하고 이메일과 이메일의 임시 암호로 로그인합니다.

첫 로그인 후 [프로필 설정](../profile/editing-profile.md)에서 암호를 변경할 수 있습니다.

:::info
아직 등록되지 않은 이메일로 로그인을 시도하면 계정을 만들 수 있도록 등록 화면으로 자동 이동됩니다.
:::

:::warning
몇 분 이내에 등록 이메일을 받지 못하면 스팸 또는 정크 폴더를 확인하세요.
:::

## 암호 잊음

암호를 잊어버린 경우 로그인 화면에서 **암호를 잊으셨나요?**를 탭합니다. 이렇게 하면 이메일 주소를 입력하고 암호 재설정 링크를 요청할 수 있는 페이지가 브라우저에서 열립니다.

## 로그인 유지

B1 Mobile은 세션 사이에 로그인 상태를 유지합니다. 앱을 열 때마다 로그인할 필요가 없습니다. 어떤 이유로든 로그아웃되면 이메일과 암호로 다시 로그인하면 됩니다.

## 로그아웃

앱을 로그아웃하려면:

1. 왼쪽 상단 모서리의 **햄버거 아이콘**(3개의 가로줄)을 탭하여 드로어 메뉴를 엽니다.
2. 드로어 메뉴의 하단으로 스크롤합니다.
3. **로그아웃**을 탭합니다.

로그아웃되며 언제든지 자격증명으로 다시 로그인할 수 있습니다.
