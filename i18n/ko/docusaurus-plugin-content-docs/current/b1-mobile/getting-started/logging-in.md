# 로그인

<div class="article-intro">

B1 Mobile은 B1.church 및 기타 ChurchApps 제품에서 사용하는 동일한 ChurchApps 계정을 사용합니다. 이미 계정이 있으면 기존 이메일 및 비밀번호로 로그인할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- iOS 또는 Android 기기에 [B1 Mobile을 설치하세요](./installing)

</div>

## 로그인 방법

1. B1 Mobile 앱을 열으세요.
2. 드로어 메뉴 또는 로그인 프롬프트에서 **Login** 옵션을 탭하세요.
3. **이메일 주소**와 **비밀번호**를 입력하세요.
4. **Sign In**을 탭하세요.

로그인하면 [대시보드](../dashboard/)로 이동합니다. 아직 교회를 선택하지 않았으면 교회를 검색하고 선택하도록 요청받을 것입니다.

## 새 계정 만들기

아직 ChurchApps 계정이 없으면 앱에서 직접 계정을 만들 수 있습니다:

1. 로그인 화면에서 하단의 **Register** 링크를 탭하세요.
2. **first name**, **last name** 및 **email address**를 입력하세요.
3. 양식은 정보를 기반으로 찾은 경우 교회를 자동으로 채울 것입니다.
4. **Register**를 탭하세요.
5. 임시 비밀번호가 포함된 메시지를 이메일로 확인하세요.
6. 앱으로 돌아가 **Login**을 탭하고 이메일과 이메일에서 받은 임시 비밀번호로 로그인하세요.

첫 로그인 후 [profile settings](../profile/editing-profile.md)에서 비밀번호를 변경할 수 있습니다.

:::info
아직 등록되지 않은 이메일로 로그인하려고 하면 자동으로 등록 화면으로 이동하여 계정을 만들 수 있습니다.
:::

:::warning
몇 분 내에 등록 이메일을 받지 못하면 스팸이나 정크 폴더를 확인하세요.
:::

## 비밀번호를 잊으셨나요?

비밀번호를 잊었으면 로그인 화면에서 **Forgot Password?**를 탭하고 이메일 주소를 입력하세요. **6자리 확인 코드**가 포함된 이메일을 받을 것입니다. 이 코드는 15분 동안 유효합니다. 확인 화면에서 코드를 입력하여 로그인한 후 [profile settings](../profile/editing-profile.md)에서 비밀번호를 업데이트하세요.

:::warning
5번 틀린 코드 시도 후 코드가 잠깁니다. 이 경우 비밀번호 찾기 화면으로 돌아가 새 코드를 요청하세요.
:::

## 로그인 상태 유지

B1 Mobile은 세션 간 로그인 상태를 유지합니다. 앱을 열 때마다 로그인할 필요가 없습니다. 어떤 이유로 로그아웃된 경우 이메일 및 비밀번호로 다시 로그인하면 됩니다.

## 로그아웃

앱에서 로그아웃하려면:

1. 좌측 상단의 **hamburger icon**(세 개의 수평선)을 탭하여 드로어 메뉴를 열으세요.
2. 드로어 메뉴의 하단까지 스크롤하세요.
3. **Sign Out**을 탭하세요.

로그아웃되고 언제든지 자격증명으로 다시 로그인할 수 있습니다.
