# 교회 전환하기

<div class="article-intro">

둘 이상의 교회에 소속되어 있는 경우, B1 모바일에서 쉽게 교회를 전환할 수 있습니다. 계정은 ChurchApps를 사용하는 모든 교회에서 작동하므로 하나의 로그인만 있으면 됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 모바일 설치](./installing) 및 계정으로 [로그인](./logging-in)

</div>

## 교회 전환 방법

1. 왼쪽 상단의 **햄버거 아이콘**(가로 세 줄)을 탭하여 서랍 메뉴를 엽니다.
2. 서랍 헤더에서 현재 교회 이름이 표시된 버튼(교회가 선택되지 않은 경우 **교회 찾기**)을 탭합니다.
3. **교회 검색** 화면이 나타납니다. 최근 방문한 교회가 빠른 접근을 위해 상단에 표시됩니다.
4. 최근 교회를 탭하여 전환하거나, 검색 필드에 입력하여 다른 교회를 찾습니다.

교회를 전환하면 앱이 해당 교회의 브랜딩, 내비게이션 링크 및 콘텐츠로 다시 로드됩니다. [대시보드](../dashboard/), [그룹](../groups/) 및 기타 데이터가 새로 선택한 교회를 반영합니다.

## 교회 검색

교회 검색 화면에서 검색 필드에 교회 이름의 최소 3자 이상을 입력합니다. 일치하는 교회가 아래 목록에 나타납니다. 교회를 탭하여 선택합니다.

:::info
결과가 나타나지 않으면 다른 철자나 교회 이름의 더 짧은 부분을 시도해 보십시오. 검색은 ChurchApps에 등록된 교회 이름과 대조하여 일치시킵니다.
:::

## 최근 교회

교회 검색 화면은 이전에 방문한 교회를 기억합니다. 검색어를 입력하지 않으면 최근 교회가 자동으로 표시됩니다. 이렇게 하면 정기적으로 사용하는 교회로 빠르게 전환할 수 있습니다.

:::tip
교회를 전환하기 위해 로그아웃했다가 다시 로그인할 필요가 없습니다. 하나의 ChurchApps 계정으로 모든 교회에서 사용할 수 있습니다. 교회 검색 화면에서 다른 교회를 선택하기만 하면 준비 완료입니다.
:::
