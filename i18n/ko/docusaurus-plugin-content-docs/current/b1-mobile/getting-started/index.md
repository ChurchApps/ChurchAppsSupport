# 시작하기

:::danger B1 Mobile은 단종되었습니다
기본 B1 Mobile 앱은 [B1.church Progressive Web App (PWA)](../../b1-church/getting-started/installing-pwa.md)로 대체되었습니다. 새로운 사용자는 App Store 또는 Google Play에서 B1 Mobile을 다운로드하는 대신 https://yourchurchname.b1.church의 모든 페이지에서 PWA를 설치해야 합니다.
:::

<div class="article-intro">

B1 Mobile을 시작하는 것은 몇 분이면 됩니다. 앱을 다운로드하고, ChurchApps 계정으로 로그인하고, 교회를 선택하면 준비가 됩니다.

</div>

## 빠른 시작

1. **다운로드** [App Store](https://apps.apple.com)(iOS) 또는 [Google Play Store](https://play.google.com)(Android)에서 앱을 다운로드하세요. "B1.church"를 검색하세요.
2. **로그인** 기존 ChurchApps 계정으로 로그인하세요 -- B1.church에서 사용하는 동일한 이메일과 비밀번호입니다. 아직 계정이 없으면 **Register**를 탭하여 계정을 만드세요.
3. **교회 선택** 이름으로 검색하여 교회를 선택하세요. 결과를 보려면 최소 3자를 입력하세요.
4. **대시보드 탐색** -- 개인화된 홈 화면에 교회가 제공하는 기능으로 빠른 링크가 표시됩니다.

이제 끝입니다. 교회를 선택한 후 앱이 교회의 브랜딩, 네비게이션 링크 및 콘텐츠를 로드합니다. 교회가 활성화한 그룹, 헌금, 체크인 및 기타 모든 것을 사용하기 시작할 수 있습니다.
