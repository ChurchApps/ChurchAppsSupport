# 시작하기

<div class="article-intro">

B1 모바일을 시작하는 데 몇 분밖에 걸리지 않습니다. 앱을 다운로드하고, ChurchApps 계정으로 로그인하고, 교회를 선택하면 준비 완료입니다.

</div>

## 빠른 시작

1. **다운로드** -- [App Store](https://apps.apple.com)(iOS) 또는 [Google Play Store](https://play.google.com)(Android)에서 앱을 다운로드합니다. "B1.church"를 검색하십시오.
2. **로그인** -- 기존 ChurchApps 계정으로 로그인합니다 -- B1.church에서 사용하는 것과 동일한 이메일과 비밀번호입니다. 아직 계정이 없으면 **회원가입**을 탭하여 계정을 만드십시오.
3. **교회 선택** -- 이름으로 교회를 검색합니다. 결과를 보려면 최소 3자 이상 입력하십시오.
4. **대시보드 탐색** -- 개인화된 홈 화면에 교회에서 제공하는 기능의 바로가기가 표시됩니다.

이것이 전부입니다. 교회를 선택하면 앱이 교회의 브랜딩, 내비게이션 링크 및 콘텐츠를 로드합니다. 그룹, 헌금, 체크인 등 교회에서 활성화한 모든 기능을 바로 사용할 수 있습니다.
