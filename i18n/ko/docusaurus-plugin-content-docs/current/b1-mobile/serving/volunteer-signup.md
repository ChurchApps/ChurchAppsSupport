# 자원봉사 신청

<div class="article-intro">

사용 가능한 자원봉사 위치를 둘러보고 B1 Mobile에서 직접 예정된 예배 봉사에 신청하세요. 채워져야 할 위치, 개방된 자리 수를 확인하고 몇 번의 탭으로 팀에 참여하세요.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 Mobile 설치](../getting-started/installing) 및 계정에 [로그인](../getting-started/logging-in)
- 교회 관리자가 최소한 하나의 위치에서 자동 신청을 활성화해야 합니다

</div>

## 자원봉사 기회 찾아보기

1. 앱 네비게이션에서 **자원봉사** 섹션을 엽니다.
2. 개방된 위치가 있는 예정된 서비스 계획 목록이 나타납니다.
3. 각 계획 카드는 다음을 표시합니다:
   - **계획 이름**과 **서비스 날짜**
   - **전체 진행률 표시줄** 차있는 자리 표시
   - **상태** -- 아직 개방된 자리 수, 또는 모든 자리가 찬 경우 "꽉 찼음"
   - **위치 칩** 각 역할과 개방된 자리 수 표시
4. **보기 및 신청**을 탭하여 계획에 대한 전체 위치 목록을 봅니다.

## 위치에 신청하기

1. 계획을 탭하여 세부정보를 엽니다.
2. 위치는 카테고리로 그룹화됩니다(예: "밴드", "접대", "어린이 사역").
3. 각 위치는 역할 이름, 설명, 사용 가능한 자리를 표시합니다.
4. 원하는 위치 옆에 있는 **신청**을 탭합니다.
5. "신청되었습니다! 자원봉사해주셔서 감사합니다."라는 확인 알림이 나타납니다.
6. 위치는 신청 확인하는 초록색 표시를 표시합니다.

:::info
신청하려면 로그인해야 합니다. 필요하면 앱이 로그인하라는 메시지를 표시합니다.
:::

## 신청 제거하기

1. 신청한 위치를 찾습니다 -- 초록색 "신청됨" 표시가 표시됩니다.
2. **제거**를 탭합니다.
3. 확인 대화 상자에 확실한지 묻습니다.
4. 위치에서 자신을 제거하도록 확인합니다.

## 마감일 및 용량

- **마감일** -- 교회가 예배 전에 마감 시간을 설정할 수 있습니다. 마감일 이후에는 신청하거나 신청을 제거할 수 없습니다. 계획 상단의 배너는 마감일이 지났을 때를 표시합니다.
- **용량** -- 각 위치의 자리 수는 제한되어 있습니다. 가득 차면 신청 버튼이 비활성화됩니다. 누군가 취소하면 자리가 다시 열립니다.

:::tip
[푸시 알림](../community/notifications.md)을 활성화하여 새로운 자원봉사 기회 및 할당 변경 사항에 대해 알아두세요.
:::
