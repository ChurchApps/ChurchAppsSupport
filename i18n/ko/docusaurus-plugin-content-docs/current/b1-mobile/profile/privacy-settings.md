# 개인정보 설정

<div class="article-intro">

프로필 화면의 **공개 설정** 탭에서 다른 교인이 명부에서 볼 수 있는 정보를 제어할 수 있습니다. 어떤 개인 정보를 공개하고 어떤 정보를 비공개로 유지할지 결정합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 모바일 설치](../getting-started/installing) 및 계정으로 [로그인](../getting-started/logging-in)

</div>

## 공개 설정 제어

1. 서랍 메뉴를 열고 **프로필**을 탭합니다.
2. **공개 설정** 탭을 선택합니다.
3. 각 필드를 확인하고 다른 교인에게 표시할지 여부를 선택합니다.
4. 변경사항을 저장합니다.

다음 항목의 공개 여부를 제어할 수 있습니다:

- **전화번호** -- 다른 교인이 전화번호를 볼 수 있는지 여부
- **이메일 주소** -- [명부](../community/member-directory.md)에 이메일이 표시되는지 여부
- **주소** -- 자택 주소가 다른 사람에게 보이는지 여부

기본적으로 교회에서 특정 필드를 공개 또는 비공개로 설정할 수 있습니다. 편한 수준에 맞게 언제든지 이 설정을 조정할 수 있습니다.

:::tip
다른 사람이 무엇을 볼 수 있는지 잘 모르겠으면, **공개 설정** 탭을 방문하여 현재 설정을 확인하십시오. 나중에 언제든지 변경할 수 있습니다.
:::

## 보류 중인 변경사항

**보류 중** 탭에서 관리자 승인을 기다리는 프로필 업데이트를 확인할 수 있습니다. 일부 교회에서는 특정 변경사항이 적용되기 전에 관리자의 검토와 승인이 필요합니다. 승인이 필요한 변경을 한 경우, 현재 상태와 함께 여기에 표시됩니다.

:::info
변경사항이 아직 적용되지 않았다면 **보류 중** 탭을 확인하십시오. 교회 관리자가 먼저 승인해야 할 수 있습니다.
:::

## 계정 설정

**계정** 탭에서 보안 설정에 접근할 수 있습니다. 이 탭을 사용하여 계정 접근 및 보안과 관련된 설정을 관리합니다.

## 프로필 탭 요약

| 탭 | 용도 |
|---|---|
| **프로필** | [개인 정보](./editing-profile)와 사진 편집 |
| **가족** | [가족 구성원](./household) 조회 |
| **공개 설정** | 명부에서 다른 사람이 볼 수 있는 정보 제어 |
| **보류 중** | 승인 대기 중인 프로필 변경사항 확인 |
| **계정** | 보안 설정 관리 |

이 탭들이 함께 작동하여 개인 정보와 교회 공동체 내에서의 정보 공유 방식을 완전히 제어할 수 있습니다.
