# 프로필

<div class="article-intro">

B1 모바일의 프로필은 개인 정보, 가족 및 개인정보 설정을 관리하는 곳입니다. 여기서 업데이트한 모든 내용은 모든 ChurchApps 서비스에 반영되므로 한 곳에서만 변경하면 됩니다.

</div>

## 프로필 탭

프로필 화면은 여러 탭으로 구성되어 있습니다:

### 프로필

**프로필** 탭에서 이름, 연락처 정보, 프로필 사진 등 개인 정보를 조회하고 편집할 수 있습니다. 프로필을 최신 상태로 유지하면 교회에서 연락하는 데 도움이 됩니다.

### 가족

**가족** 탭에서 가족 구성원과 가족 내 역할을 확인합니다. 이 정보는 한 번에 가족 전체를 체크인할 수 있는 [체크인](../checkin/) 같은 기능에 사용됩니다.

### 공개 설정

**공개 설정** 탭에서 다른 교인이 [명부](../community/member-directory.md)에서 볼 수 있는 정보를 제어합니다. 어떤 개인 정보를 공유하고 어떤 정보를 비공개로 유지할지 결정합니다.

### 보류 중

**보류 중** 탭에서 관리자 승인을 기다리는 프로필 변경 사항을 확인합니다.

### 계정

**계정** 탭에서 보안 설정에 접근합니다.

## 프로필 접근 방법

1. 서랍 메뉴를 엽니다.
2. **프로필**을 탭하여 프로필 화면을 엽니다.
3. 상단의 탭을 사용하여 섹션 간 이동합니다.
