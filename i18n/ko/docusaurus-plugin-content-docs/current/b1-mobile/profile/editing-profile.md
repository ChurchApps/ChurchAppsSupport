# 프로필 편집

<div class="article-intro">

**프로필** 탭에서 개인 정보를 업데이트하여 교회에서 항상 올바른 정보를 보유할 수 있습니다. 여기서 변경한 내용은 B1.church 및 교회에서 사용하는 다른 앱을 포함한 모든 ChurchApps 서비스에 반영됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 모바일 설치](../getting-started/installing) 및 계정으로 [로그인](../getting-started/logging-in)

</div>

## 편집 가능한 항목

프로필에는 다음 개인 정보가 포함됩니다:

- **이름과 성** -- ChurchApps 전체에서 사용되는 표시 이름
- **이메일 주소** -- 교회 소통 및 계정 복구에 사용
- **전화번호** -- 필요할 때 교회에서 연락할 수 있도록
- **주소** -- 교회 기록을 위한 자택 주소
- **프로필 사진** -- [명부](../community/member-directory.md)와 앱 전체에서 사용되는 이미지

## 정보 업데이트

1. 서랍 메뉴를 열고 **프로필**을 탭합니다.
2. **프로필** 탭이 활성화되어 있지 않으면 선택합니다.
3. 업데이트하려는 필드를 탭하고 변경합니다.
4. 변경이 완료되면 저장합니다.

업데이트된 정보가 자동으로 동기화됩니다. 프로필을 보는 사람([개인정보 설정](./privacy-settings)에 따라)은 즉시 새로운 정보를 볼 수 있습니다.

:::tip
이메일 주소와 전화번호를 최신 상태로 유지하십시오. 교회에서 이 정보를 사용하여 중요한 업데이트와 알림을 보냅니다.
:::

## 프로필 사진

**프로필** 탭에서 프로필 사진을 추가하거나 변경할 수 있습니다. 사진은 [교인 명부](../community/member-directory.md), 그룹 구성원 목록 및 프로필이 표시되는 모든 곳에 나타납니다.

## ChurchApps 전체에서의 변경사항

모든 ChurchApps 서비스가 동일한 계정 시스템을 공유하기 때문에, B1 모바일에서 변경한 내용은 [B1.church](../../b1-church/), 교회의 관리 도구 및 기타 연결된 서비스에서 즉시 확인할 수 있습니다. 각 장소에서 별도로 정보를 업데이트할 필요가 없습니다.

:::info
일부 교회에서는 특정 프로필 변경에 관리자 승인이 필요합니다. 업데이트가 즉시 반영되지 않으면 프로필 화면의 **보류 중** 탭에서 상태를 확인하십시오.
:::
