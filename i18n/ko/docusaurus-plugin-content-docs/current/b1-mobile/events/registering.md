# 행사 등록하기

<div class="article-intro">

B1 Mobile에서 간단한 3단계 프로세스를 사용하여 교회 행사에 직접 등록하세요. 휴대폰에서 자신과 가족 구성원을 등록할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 Mobile 설치](../getting-started/installing) 및 계정에 [로그인](../getting-started/logging-in)
- 행사에 등록이 활성화되어 있고 등록 기간이 열려있어야 합니다

</div>

## 1단계: 정보 입력

1. 등록이 활성화된 그룹 달력 행사를 열고 **등록**을 탭합니다.
2. 등록 화면에는 행사 제목, 설명, 날짜가 표시됩니다.
3. 로그인했으므로 양식에서 계정으로 등록 중임을 확인합니다.

:::info
행사에 수용 제한이 있는 경우, 진행률 표시줄에 남은 자리 수가 표시됩니다. 행사가 가득 차면 등록할 수 없습니다.
:::

## 2단계: 가족 구성원 추가

1. 추가 가족 구성원을 등록하려면 각 사람의 **이름**과 **성**을 입력합니다.
2. **구성원 추가**를 탭하여 더 많은 사람을 추가합니다(최대 10명).
3. 구성원 옆의 제거 버튼을 탭하여 등록에서 제외합니다.
4. 자신만 등록하는 경우 이 단계를 건너뜁니다.

## 3단계: 확인

1. 행사 세부정보와 등록 정보를 검토합니다.
2. 등록을 확인합니다.
3. 행사 세부정보와 등록한 모든 사람이 포함된 확인 메시지가 나타납니다.

계정과 연결된 이메일 주소로 **확인 이메일**도 받게 됩니다.

## 등록 취소

취소해야 하는 경우 교회 관리자에게 문의하세요. 관리자가 B1 Admin 등록 페이지에서 등록을 취소할 수 있습니다.
