# 행사

<div class="article-intro">

B1 Mobile의 행사 기능을 사용하면 예정된 교회 행사를 둘러보고 휴대폰에서 직접 등록할 수 있습니다. 교회에서 행사에 대한 등록을 활성화하면 몇 번의 탭으로 자신과 가족 구성원을 등록할 수 있습니다.

</div>

## 등록이 있는 행사 찾기

등록이 활성화된 행사는 그룹 달력에 표시됩니다. 등록이 가능한 행사를 탭하면 행사 세부정보에서 **등록** 버튼을 볼 수 있습니다.

행사는 다양한 상태를 표시할 수 있습니다:

- **진행 중** -- 등록이 가능하며 등록할 수 있습니다.
- **아직 시작 안함** -- 등록 기간이 아직 시작되지 않았습니다.
- **종료됨** -- 등록 마감일이 지났습니다.
- **꽉 참** -- 사용 가능한 모든 자리가 차 있습니다.

## 필요한 것

- [B1 Mobile 설치](../getting-started/installing) 및 [로그인](../getting-started/logging-in)
- 교회에서 해당 행사에 대해 행사 등록을 활성화해야 합니다
