# 대시보드

<div class="article-intro">

대시보드는 B1 모바일의 홈 화면입니다. 교회에서 제공하는 기능에 빠르게 접근할 수 있으며, 앱 사용 방식에 따라 정리됩니다. 레이아웃은 시간이 지남에 따라 가장 많이 사용하는 기능을 전면에 배치하도록 조정됩니다.

</div>

## 환영 섹션

대시보드 상단에는 환영 메시지와 함께 교회 로고(로고가 설정되지 않은 경우 교회 이름)와 "소통하기" 태그라인이 표시됩니다. 이 섹션은 현재 어떤 교회를 보고 있는지 확인해 줍니다.

## 추천 링크

환영 섹션 아래에는 가장 많이 방문한 기능이 배경 이미지와 기능 이름이 있는 큰 **히어로 카드**로 표시됩니다. 이것은 가장 자주 사용하는 기능으로, 한 번의 탭으로 접근할 수 있도록 상단에 배치됩니다.

## 추천 카드

히어로 카드 아래에는 나란히 표시되는 최대 두 개의 **추천 카드**가 있습니다. 이것은 두 번째와 세 번째로 많이 방문한 기능을 나타냅니다. 각 카드에는 배경 이미지와 기능 이름이 표시됩니다. 카드를 탭하면 해당 기능으로 바로 이동합니다.

## 빠른 실행

나머지 기능은 추천 카드 아래의 **빠른 실행** 그리드에 표시됩니다. 각 항목에는 아이콘과 라벨이 표시됩니다. 이것은 교회에서 설정한 다른 모든 내비게이션 링크로, [그룹](../groups/)과 [헌금](../giving/)부터 [체크인](../checkin/), [설교](../content/sermons.md) 등에 이르기까지 모든 것에 접근할 수 있습니다.

## 레이아웃 적응 방식

대시보드 레이아웃은 사용 패턴에 따라 개인화됩니다. 기능을 탭할 때마다 앱이 방문 빈도를 추적합니다. 더 자주 사용하는 기능이 대시보드 상단으로 이동합니다:

- **가장 많이 방문한** 기능이 히어로 카드가 됩니다.
- **두 번째와 세 번째로 많이 방문한** 기능이 추천 카드가 됩니다.
- **나머지 모든 것**은 빠른 실행 그리드에 표시됩니다.

앱을 아직 사용하지 않은 경우, 기능은 교회에서 설정한 기본 순서로 표시됩니다. 시간이 지나면 대시보드가 사용 습관에 맞게 재배치됩니다.

:::info
대시보드에서 사용할 수 있는 기능은 교회 관리자가 [B1 관리자](../../b1-admin/)를 통해 설정합니다. 예상하는 기능이 보이지 않으면 교회 사무실에 문의하십시오.
:::

## 대시보드 새로고침

대시보드를 아래로 당기면 새로고침됩니다. 이렇게 하면 교회의 링크와 콘텐츠가 다시 로드되어 최신 정보를 볼 수 있습니다.

## 내비게이션

대시보드에서 앱의 모든 부분에 접근할 수 있습니다:

- 왼쪽 상단의 **햄버거 아이콘**(가로 세 줄)을 탭하면 모든 내비게이션 옵션이 있는 서랍 메뉴가 열립니다.
- 오른쪽 상단의 **종 아이콘**을 탭하면 [알림](../community/notifications.md)을 볼 수 있습니다.
- 카드나 빠른 실행 항목을 탭하면 해당 기능으로 바로 이동합니다.
