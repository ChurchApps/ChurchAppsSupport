# 결제 수단 관리

<div class="article-intro">

헌금 섹션의 **관리** 탭에서 저장된 결제 수단을 조회하고 관리할 수 있습니다. 카드를 저장하면 매번 정보를 다시 입력하지 않아도 되므로 향후 헌금을 더 빠르게 할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 모바일 설치](../getting-started/installing) 및 계정으로 [로그인](../getting-started/logging-in)

</div>

## 결제 수단 조회

**관리** 탭을 열면 저장된 모든 결제 수단 목록이 표시됩니다. 각 항목에는 카드 유형과 카드 번호의 마지막 네 자리가 표시되어 어떤 카드인지 쉽게 구별할 수 있습니다.

## 새 결제 수단 추가

새 카드를 추가하려면:

1. B1 모바일에서 **헌금** 섹션을 엽니다.
2. **관리** 탭을 탭합니다.
3. 새 결제 수단 추가 옵션을 탭합니다.
4. 카드 정보를 입력합니다. 정보는 **Stripe**에 안전하게 전송되어 처리됩니다.
5. 저장되면 새 카드가 결제 수단 목록에 나타납니다.

저장된 카드는 다음에 **헌금하기** 탭에서 [헌금](./making-donations)할 때 결제 옵션으로 사용할 수 있습니다.

## 결제 수단 삭제

더 이상 카드를 보관하지 않으려면 **관리** 탭에서 삭제할 수 있습니다:

1. 저장된 결제 수단 목록에서 삭제할 카드를 찾습니다.
2. 카드 옆의 삭제 옵션을 탭합니다.
3. 삭제를 확인합니다.

해당 카드는 헌금 시 결제 옵션으로 더 이상 나타나지 않습니다.

:::info
결제 수단을 삭제해도 과거 헌금에는 영향을 미치지 않습니다. [헌금 내역](./donation-history)은 변경되지 않습니다.
:::

:::tip
빠른 헌금을 위해 최소 하나의 결제 수단을 저장해 두십시오. 이렇게 하면 **개요** 탭의 **반복** 버튼을 사용하여 한 번의 탭으로 다시 헌금할 수 있습니다.
:::
