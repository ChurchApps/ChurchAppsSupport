# B1 Mobile

<div class="article-intro">

B1 Mobile은 iOS 및 Android에서 사용할 수 있는 B1.church의 동반 앱입니다. 교회 회원들이 교회에서 제공하는 모든 도구에 언제 어디서나 액세스할 수 있습니다 -- 그룹, 기부금, 체크인, 예배 계획, 회원 디렉터리, 개인 메시징, 설교, 성경, 오늘의 말씀, 레슨 등.

</div>

앱은 B1.church에서 사용하는 것과 동일한 ChurchApps 계정을 사용합니다. 로그인하고 교회를 선택한 후 앱이 교회의 브랜딩 및 구성에 적응합니다. 여러 교회에 속한 경우 언제든지 그들 사이를 전환할 수 있습니다.

B1 Mobile은 푸시 알림을 지원하므로 교회나 그룹의 중요한 업데이트를 놓치지 않습니다. 홈 대시보드는 사용에 따라 개인화됩니다 -- 가장 자주 방문하는 기능이 빠른 액세스를 위해 맨 위로 이동합니다.

## 주요 기능

- **그룹** -- 그룹을 보고, 대화에 참여하고, 그룹 회원들과 연결 유지합니다.
- **기부금** -- 휴대폰에서 직접 교회에 기부금을 합니다.
- **이벤트** -- 예정된 이벤트를 찾아보고 휴대폰에서 직접 등록합니다.
- **체크인** -- 예배 및 이벤트에 대해 자신과 가구 구성원을 체크인합니다.
- **예배 계획** -- 예정된 예배 계획, 할당된 자원봉사자 위치, 열린 위치에 가입을 봅니다.
- **회원 디렉터리** -- 교회 디렉터리를 찾아보고 회원 프로필을 봅니다.
- **메시징** -- 다른 교회 회원과 개인 메시지를 보내고 받습니다.
- **설교** -- 교회의 설교를 보거나 듣습니다.
- **성경** -- 앱 내에서 성경을 읽습니다.
- **오늘의 말씀** -- 매일 말씀으로 하루를 시작합니다.
- **레슨** -- Lessons.church의 레슨에 액세스하고 작업합니다.
