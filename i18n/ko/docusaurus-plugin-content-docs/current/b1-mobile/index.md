# B1 모바일

<div class="article-intro">

B1 모바일은 iOS와 Android에서 사용할 수 있는 B1.church의 동반 앱입니다. 교회 구성원들이 이동 중에도 교회에서 제공하는 모든 도구에 접근할 수 있습니다 -- 그룹, 헌금, 체크인, 봉사 계획, 교인 명부, 개인 메시지, 설교, 성경, 오늘의 말씀, 공과, 그 외 다양한 기능을 포함합니다.

</div>

이 앱은 B1.church에서 사용하는 것과 동일한 ChurchApps 계정을 사용합니다. 로그인하고 교회를 선택하면, 앱이 교회의 브랜딩과 설정에 맞게 조정됩니다. 여러 교회에 소속되어 있는 경우, 언제든지 교회를 전환할 수 있습니다.

B1 모바일은 푸시 알림을 지원하므로 교회나 그룹의 중요한 업데이트를 놓치지 않습니다. 홈 대시보드는 사용 패턴에 따라 자동으로 개인화됩니다 -- 가장 자주 방문하는 기능이 빠른 접근을 위해 상단에 표시됩니다.

## 주요 기능

- **그룹** -- 그룹을 조회하고, 대화에 참여하며, 그룹 구성원들과 연결을 유지합니다.
- **헌금** -- 휴대폰에서 직접 교회에 헌금합니다.
- **체크인** -- 예배와 행사에 본인과 가족 구성원을 체크인합니다.
- **봉사 계획** -- 예정된 예배 계획과 배정된 봉사 위치를 확인합니다.
- **교인 명부** -- 교회 명부를 탐색하고 교인 프로필을 확인합니다.
- **메시지** -- 다른 교인들과 개인 메시지를 주고받습니다.
- **설교** -- 교회의 설교를 시청하거나 청취합니다.
- **성경** -- 앱 내에서 성경을 읽습니다.
- **오늘의 말씀** -- 매일 말씀으로 하루를 시작합니다.
- **공과** -- Lessons.church의 공과에 접근하고 학습합니다.
