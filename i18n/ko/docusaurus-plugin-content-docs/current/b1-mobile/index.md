# B1 Mobile(단종됨)

:::danger 단종됨 -- 대신 B1.church를 사용하세요
iOS 및 Android의 기본 B1 Mobile 앱은 **단종되었으며** 더 이상 개발 중이 아닙니다. 이제 기능은 https://yourchurchname.b1.church의 **B1.church Progressive Web App (PWA)**을 통해 제공되며, 모든 페이지에서 추가할 수 있으며 App Store 다운로드 없이 동일한 회원 포털 경험을 제공합니다.

**교회 회원인 경우:** [앱으로 설치하기(PWA)](../b1-church/getting-started/installing-pwa.md)에서 휴대폰에 대체 앱을 설정하는 방법을 참조하세요.

**교회 관리자인 경우:** App Store / Google Play 목록 대신 교회 사이트(https://yourchurchname.b1.church)를 회원에게 안내하세요 -- 모든 페이지에서 PWA를 설치할 수 있습니다.

아래 페이지는 기본 앱을 계속 실행 중인 교회를 위한 참조용으로 보관됩니다.
:::

<div class="article-intro">

B1 Mobile은 iOS 및 Android의 B1.church 동반 기본 앱이었습니다. 그룹, 헌금, 체크인, 봉사 계획, 회원 디렉토리, 개인 메시징, 설교, 성경, 오늘의 말씀, 레슨 등에 모바일에서 접근할 수 있도록 해주었습니다.

</div>

ChurchApps 계정을 사용했으며, B1.church에서 사용하는 계정과 동일합니다. 로그인하고 교회를 선택한 후 앱이 교회의 브랜딩 및 구성에 맞게 적응했습니다. 여러 교회에 속해 있으면 언제든지 교회 간에 전환할 수 있었습니다.

B1 Mobile은 푸시 알림을 지원했으므로 교회 또는 그룹의 중요한 업데이트를 놓치지 않을 것입니다. 홈 대시보드는 사용량을 기반으로 개인화되어 가장 자주 방문한 기능이 빠른 접근을 위해 상단에 표시됩니다.

## 주요 기능

- **Groups** -- 그룹을 보고, 대화에 참여하고, 그룹 구성원과 연결을 유지하세요.
- **Giving** -- 휴대폰에서 직접 교회에 헌금하세요.
- **Events** -- 예정된 이벤트를 보고 휴대폰에서 직접 등록하세요.
- **Check-In** -- 예배 및 이벤트를 위해 자신과 가족 구성원을 체크인하세요.
- **Serving Plans** -- 예정된 봉사 계획, 배치된 자원봉사자 위치를 보고 열린 위치에 등록하세요.
- **Member Directory** -- 교회의 디렉토리를 보고 회원 프로필을 보세요.
- **Messaging** -- 다른 교회 회원과 개인 메시지를 보내고 받으세요.
- **Sermons** -- 교회의 설교를 시청하거나 들으세요.
- **Bible** -- 앱 내에서 성경을 읽으세요.
- **Verse of the Day** -- 매일 말씀으로 하루를 시작하세요.
- **Lessons** *(단종됨 -- [FreePlay](/docs/freeplay/)로 대체됨)* -- Lessons.church의 레슨을 앱 내에 포함했습니다.
