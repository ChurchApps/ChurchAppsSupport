# 설교

<div class="article-intro">

설교 화면에서 교회의 녹화된 설교를 탐색하고 시청할 수 있습니다. 놓친 메시지를 따라잡거나 감동받은 설교를 다시 보기에 적합한 공간입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 모바일 설치](../getting-started/installing) 및 계정으로 [로그인](../getting-started/logging-in)
- 교회의 관리 포털에 설교 콘텐츠가 설정되어 있어야 합니다

</div>

## 화면 구성

설교 화면은 두 개의 주요 섹션이 있는 탭 보기로 구성되어 있습니다:

- **재생목록 탭** -- 교회의 설교 재생목록(시리즈)을 날짜순으로 표시합니다. 각 재생목록은 관련 설교를 그룹으로 묶어 여러 주에 걸친 시리즈를 처음부터 끝까지 쉽게 따라갈 수 있습니다.
- **최근 탭** -- 개별 설교를 역순(최신순)으로 나열하여 최신 메시지가 항상 상단에 표시됩니다.

화면 상단에는 가장 최근 설교가 추천 카드로 표시되어 최신 메시지에 빠르게 접근할 수 있습니다.

## 실시간 스트리밍 카운트다운

[실시간 스트리밍](./live-streaming)이 24시간 이내에 예정되어 있으면 설교 화면에 카운트다운 타이머가 나타납니다. 이를 통해 다음 예배가 방송되는 시간을 알 수 있어 실시간 시청을 계획할 수 있습니다.

## 설교 시청

1. 서랍 메뉴에서 **설교** 화면을 엽니다.
2. **재생목록** 또는 **최근** 탭에서 원하는 설교를 찾습니다.
3. 설교를 탭하여 열고 시청을 시작합니다.

재생목록을 탭하면 해당 시리즈의 모든 설교를 볼 수 있으며, 시청하고 싶은 특정 설교를 선택할 수 있습니다.

설교에는 YouTube, Vimeo 또는 Facebook과 같은 플랫폼에서 호스팅되는 비디오 또는 오디오 콘텐츠가 포함될 수 있습니다. 설교를 탭하면 앱 내에서 직접 미디어가 재생됩니다.

## 각 설교에 표시되는 정보

각 설교 항목에는 다음이 표시될 수 있습니다:

- **제목** -- 설교 제목
- **설교자** -- 메시지를 전한 사람
- **날짜** -- 설교가 녹화 또는 게시된 날짜
- **미리보기 이미지** -- 설교의 미리보기 이미지

:::tip
설교 화면에 카운트다운이 표시되면, 탭하여 [실시간 스트리밍](./live-streaming)이 시작될 때 바로 이동하십시오.
:::

## 이용 가능 여부

설교 화면은 교회에서 관리 포털을 통해 설교 콘텐츠를 추가한 경우 사용할 수 있습니다. 앱에 **설교**가 보이지 않으면, 교회에서 아직 이 기능을 설정하지 않았을 수 있습니다.
