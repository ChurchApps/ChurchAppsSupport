# 실시간 스트리밍

<div class="article-intro">

B1 모바일에서 휴대폰으로 직접 교회의 실시간 스트리밍을 시청할 수 있습니다. 예배가 방송 중일 때, 어디에서든 실시간으로 시청할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 모바일 설치](../getting-started/installing) 및 계정으로 [로그인](../getting-started/logging-in)
- 교회에서 스트리밍 일정이 설정되어 있어야 합니다

</div>

## 실시간 스트리밍 시청

1. 서랍 메뉴에서 **스트리밍** 화면을 열거나, 실시간 스트리밍이 곧 시작될 때 [설교](./sermons) 화면의 카운트다운을 탭합니다.
2. 현재 예배가 라이브 중이면 비디오 플레이어가 자동으로 로드되고 방송이 재생됩니다.
3. 앱에서 직접 스트리밍을 시청합니다.

현재 실시간 스트리밍이 활성화되어 있지 않으면, 화면에 오프라인 콘텐츠나 다음 예정된 예배에 대한 정보가 표시될 수 있습니다.

## 스트리밍 접근 방법

실시간 스트리밍에 접근하는 두 가지 방법이 있습니다:

- **서랍 메뉴** -- 서랍 메뉴를 열고 **스트리밍**을 탭하면 실시간 스트리밍 화면으로 바로 이동합니다.
- **설교 화면** -- 실시간 스트리밍이 24시간 이내에 예정되어 있으면 [설교](./sermons) 화면에 카운트다운이 나타납니다. 방송이 시작되면 거기에서 스트리밍으로 이동할 수 있습니다.

## 예배 전후

교회에서 스트리밍이 가능한 특정 시간을 설정합니다. 예배 시작 전에 카운트다운이나 오프라인 메시지가 표시될 수 있습니다. 방송이 끝나면 스트리밍 화면이 오프라인 상태로 돌아갑니다.

:::info
스트리밍 화면에 카운트다운이나 오프라인 메시지가 표시되면, 다음 예정된 예배가 아직 시작되지 않은 것입니다. 표시된 예배 시간에 다시 확인하십시오.
:::

## 이용 가능 여부

실시간 스트리밍 화면은 교회에서 스트리밍 일정을 설정한 경우 사용할 수 있습니다. 앱의 서랍 메뉴에 **스트리밍** 옵션이 보이지 않으면, 교회에서 실시간 스트리밍을 설정하지 않았을 수 있습니다. 자세한 내용은 교회 관리자에게 문의하십시오.

:::tip
[설교](./sermons) 화면에서 다음 실시간 방송이 시작되는 시간을 보여주는 카운트다운 타이머를 확인하십시오.
:::
