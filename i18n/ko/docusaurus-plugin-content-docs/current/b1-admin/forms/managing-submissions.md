# 제출 관리

<div class="article-intro">

양식이 활성화되면 제출이 들어오기 시작합니다. 제출 탭에서 모든 응답을 검토하고, 누가 제출했는지 추적하며, 데이터를 활용하여 교인들에게 적시에 후속 조치를 취할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 최소 하나의 질문이 포함된 [양식을 만드십시오](./creating-forms.md)
- 공개 URL을 통해 양식을 공유하거나 [웹사이트](../website/managing-pages.md)에 삽입하십시오

</div>

## 제출 조회

1. 메인 메뉴에서 **양식**으로 이동합니다.
2. 검토하려는 양식을 클릭합니다.
3. **제출** 탭으로 이동합니다.

해당 양식에 대해 수신된 모든 응답 목록이 표시됩니다. 각 제출에는 응답자 정보와 제출 날짜가 표시됩니다.

## 개별 제출 검토

제출을 클릭하여 전체 세부 정보를 확인합니다. 양식의 각 질문에 대해 해당 사람이 제공한 모든 답변을 볼 수 있습니다.

## 제출 수집 경로

제출은 여러 경로를 통해 들어올 수 있습니다:

- **공개 URL** -- 양식에 공개 URL이 있는 경우(독립형 양식), 링크를 가진 누구나 응답을 제출할 수 있습니다. 이메일, 소셜 미디어 또는 문자 메시지를 통해 URL을 공유하십시오.
- **웹사이트에 삽입** -- B1 웹사이트에 삽입된 양식은 방문자가 작성하면 자동으로 제출을 수집합니다.
- **내부 사용** -- 교직원과 자원봉사자가 B1 Admin 내에서 다른 사람을 대신하여 양식을 제출할 수도 있습니다.

:::tip
방문자 카드나 기도 요청 같은 양식은 **제출** 탭을 정기적으로 확인하여 신속하게 후속 조치를 취하십시오.
:::

## 제출 데이터 활용

양식을 통해 수집된 데이터는 다음과 같이 활용할 수 있습니다:

- 행사 등록을 추적하고 그에 맞게 계획합니다
- 방문자나 새 교인에게 후속 조치를 취합니다
- 사역 기획을 위한 설문 피드백을 수집합니다
- 행사나 프로그램에 대한 결제를 처리합니다

:::info
**관리자** 또는 **조회만** 역할을 가진 양식 멤버가 제출 탭에 접근할 수 있습니다. **멤버** 탭에서 역할을 관리하여 팀의 적절한 사람이 필요한 양식에 접근할 수 있도록 하십시오.
:::
