# 양식 만들기

<div class="article-intro">

교인으로부터 정보를 수집하기 위한 맞춤 양식을 작성합니다. 행사 등록, 설문조사, 방문자 카드, 교인 가입 신청 등을 위한 양식을 만들 수 있습니다. 양식은 데이터베이스의 인원 정보와 연결하거나, 고유한 공개 URL을 가진 독립형 페이지로 사용할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- **인원** 양식(인원 기록과 연결)의 경우, 먼저 [데이터베이스에 인원](../people/adding-people.md)이 있어야 합니다.
- **결제**를 수집하는 양식의 경우, [온라인 헌금을 위한 Stripe 설정](../donations/online-giving-setup.md)이 완료되어야 합니다.

</div>

## 새 양식 만들기

1. 메인 메뉴에서 **양식**으로 이동합니다.
2. **양식 추가**를 클릭합니다.
3. 양식의 **이름**을 입력합니다.
4. 드롭다운에서 양식 유형을 선택합니다:
   - **인원** — 제출을 데이터베이스의 [인원 기록](../people/adding-people.md)과 연결합니다.
   - **독립형** — 고유한 공개 URL을 가진 독립적인 양식을 만들며, 외부 등록에 적합합니다.
5. **저장**을 클릭하여 양식을 생성합니다.

새 양식이 목록에 나타납니다. 클릭하여 질문 추가를 시작하십시오.

## 질문 추가

1. 양식을 열고 **질문** 탭으로 이동합니다.
2. **질문 추가**를 클릭합니다.
3. 제공자 드롭다운에서 **필드 유형**을 선택합니다. 사용 가능한 유형은 다음과 같습니다:
   - **텍스트 상자** — 짧은 텍스트 답변용
   - **날짜** — 날짜 선택용
   - **이메일** — 이메일 주소용
   - **전화번호** — 전화번호 입력용
   - **객관식** — 사전 정의된 옵션에서 선택용
   - **결제** — 결제 수집용
4. 질문의 **제목**과 선택적으로 **설명**을 입력합니다.
5. 필수 필드인 경우 **응답 필수**를 체크합니다.
6. **저장**을 클릭합니다.
7. 반복하여 더 많은 질문을 추가합니다.

:::warning
**결제** 필드 유형을 사용하려면 Stripe가 설정되어 있어야 합니다. 아직 온라인 헌금을 설정하지 않은 경우, 결제 필드를 추가하기 전에 [온라인 헌금 설정](../donations/online-giving-setup.md)을 참조하십시오.
:::

## 양식 멤버 관리

1. 양식을 열고 **멤버** 탭으로 이동합니다.
2. 사람을 검색하고 역할과 함께 추가합니다:
   - **관리자** — 양식을 편집하고 모든 제출을 조회할 수 있습니다.
   - **조회만** — 제출을 조회할 수 있지만 양식을 편집할 수 없습니다.

## 양식 속성 설정

양식의 이름과 설정은 언제든지 업데이트할 수 있습니다. 독립형 양식의 경우, 누구에게나 공유할 수 있는 고유한 **공개 URL**도 표시됩니다.

:::tip
독립형 양식은 행사 등록에 적합합니다. 이메일, 소셜 미디어를 통해 공개 URL을 공유하거나, 교회 웹사이트에 양식을 직접 삽입하십시오.
:::

:::info
B1 웹사이트에 양식을 삽입하려면, 웹사이트 편집기에서 새 섹션을 추가하고 **양식** 요소를 선택한 다음, 표시할 양식을 선택하십시오. 웹사이트 편집에 대한 자세한 내용은 [페이지 관리](../website/managing-pages.md)를 참조하십시오.
:::
