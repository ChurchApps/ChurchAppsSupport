# 프로필 관리

<div class="article-intro">

프로필 페이지에서는 개인 계정 설정을 편집할 수 있습니다. 프로필은 모든 ChurchApps 제품에서 공유되므로, 여기에서 변경한 내용은 ChurchApps를 사용하는 모든 곳에 반영됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- ChurchApps 계정에 로그인되어 있어야 합니다
- 이메일 주소 또는 비밀번호를 업데이트하려면 새 이메일 주소 또는 비밀번호를 준비하십시오

</div>

## 프로필 정보

프로필 페이지 상단 섹션에는 기본 계정 세부 정보가 포함되어 있습니다:

1. **이메일** -- 로그인 및 알림 수신에 사용되는 이메일 주소입니다.
2. **이름** 및 **성** -- ChurchApps 전체에서 표시되는 이름입니다.
3. **비밀번호** -- 계정 비밀번호를 업데이트합니다.

정보를 업데이트하려면 필드를 편집하고 **변경 사항 저장** 버튼을 클릭합니다.

## 비밀번호 변경

1. 프로필 페이지에서 비밀번호 필드로 스크롤합니다.
2. **비밀번호** 필드에 새 비밀번호를 입력합니다.
3. **비밀번호 확인** 필드에 다시 입력합니다.
4. **변경 사항 저장**을 클릭합니다.

:::info
비밀번호는 최소 8자 이상이어야 합니다. 비밀번호와 확인 필드가 정확히 일치하는지 확인하십시오.
:::

## 연결된 계정

**연결된 계정** 섹션에는 ChurchApps 계정에 연결된 외부 서비스가 표시됩니다. 이 섹션에서 어떤 계정이 연결되어 있는지 확인하고 해당 연결을 관리할 수 있습니다.

## 테마 환경 설정

ChurchApps는 밝은 모드와 어두운 모드를 모두 지원합니다:

1. 프로필 페이지에서 **테마** 토글을 찾습니다.
2. **밝은** 모드와 **어두운** 모드 간에 전환합니다.
3. 인터페이스가 선택한 내용을 즉시 반영하여 업데이트됩니다.

테마 환경 설정은 계정에 저장되며 모든 세션에 적용됩니다.

## 계정 삭제

프로필 페이지 하단에 계정을 영구적으로 삭제하는 옵션이 있습니다. 이 작업은 계정과 관련 데이터를 제거하며 취소할 수 없습니다.

:::warning
계정 삭제는 영구적이며 되돌릴 수 없습니다. 진행하기 전에 정말로 계정을 제거하려는 것인지 확인하십시오.
:::

:::info
**도메인 관리자**인 경우, 계정을 삭제하기 전에 관리 책임을 다른 사용자에게 이전하십시오. 관리자 역할 관리에 대한 자세한 내용은 [역할 및 권한](../settings/roles-permissions.md)을 참조하십시오.
:::

:::info
데모 모드에서는 프로필 편집이 비활성화됩니다. 데모 계정을 사용하는 경우 프로필 설정을 변경할 수 없습니다.
:::
