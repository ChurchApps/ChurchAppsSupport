# 프로필

<div class="article-intro">

프로필 섹션에서는 ChurchApps의 개인 계정 설정을 관리할 수 있습니다. 프로필은 모든 ChurchApps 도구에서 공유되므로, 여기에서 변경한 내용은 B1 Admin부터 B1 모바일 앱까지 모든 곳에 적용됩니다.

</div>

## 할 수 있는 작업

프로필 섹션에서 다음을 수행할 수 있습니다:

- **이름 및 이메일 업데이트** -- 연락처 정보를 최신 상태로 유지합니다.
- **비밀번호 변경** -- 계정의 새 비밀번호를 설정합니다.
- **테마 전환** -- 밝은 모드와 어두운 모드 간에 전환하여 선호에 맞게 조정합니다.
- **연결된 기기 관리** -- 계정에 연결된 ChurchAppsPlayer 기기를 보고 제어합니다.
- **연결된 계정 보기** -- ChurchApps 계정에 연결된 외부 서비스를 보고 관리합니다.

## 프로필 하위 섹션

프로필 영역에는 다음 페이지가 있습니다:

1. **[프로필 관리](./managing-profile.md)** -- 개인 정보 편집, 비밀번호 변경, 연결된 계정 관리, 테마 환경 설정 조정 및 계정 삭제를 처리합니다.
2. **[기기 관리](./devices.md)** -- 계정에 연결된 ChurchAppsPlayer 기기를 보고, 추가하고, 관리합니다.

:::tip
이메일 주소를 최신 상태로 유지하십시오. 계정 복구 및 교회의 중요한 알림에 사용됩니다.
:::
