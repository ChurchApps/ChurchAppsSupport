# 기기 관리

<div class="article-intro">

기기 페이지에서는 계정에 연결된 ChurchAppsPlayer 기기를 관리할 수 있습니다. 이 기기들은 콘텐츠 스트리밍과 교회 현장의 화면에 미디어를 표시하는 데 사용됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- ChurchAppsPlayer 기기의 전원이 켜져 있고 인터넷에 연결되어 있어야 합니다
- 기기 화면에 표시된 페어링 코드를 확인하십시오
- 아직 확인하지 않았다면 [프로필 설정](./managing-profile.md)을 검토하십시오

</div>

## 기기 보기

기기 페이지를 열면 다음 정보가 포함된 등록된 모든 기기의 표가 표시됩니다:

- **레이블** -- 기기에 지정한 이름입니다.
- **등록 날짜** -- 기기가 계정에 처음 페어링된 날짜입니다.
- **마지막 활성 날짜** -- 기기가 계정에 가장 최근에 연결된 시간입니다.

## 새 기기 추가

1. 기기 페이지 상단의 **기기 추가** 버튼을 클릭합니다.
2. **페어링 코드**를 입력하라는 메시지가 표시됩니다.
3. 연결하려는 ChurchAppsPlayer 기기에 표시된 페어링 코드를 확인합니다.
4. 코드를 입력하고 확인합니다.
5. 페어링이 성공적으로 완료되면 기기가 기기 표에 나타납니다.

:::tip
페어링을 시도하기 전에 ChurchAppsPlayer 기기의 전원이 켜져 있고 인터넷에 연결되어 있는지 확인하십시오. 페어링 코드는 기기 화면에 표시됩니다.
:::

## 기기 레이블 편집

1. 기기 표에서 **기기 이름**을 클릭합니다.
2. 설명적인 이름으로 레이블을 업데이트합니다 (예: "본당 메인 스크린" 또는 "로비 디스플레이").
3. 변경 사항을 저장합니다.

:::tip
명확하고 설명적인 레이블을 사용하면 기기를 쉽게 식별할 수 있으며, 특히 교회에 여러 화면이 설치되어 있을 때 유용합니다.
:::

## 기기 접근 관리

기기 페이지를 사용하여 교회의 스트리밍 및 콘텐츠에 접근하는 기기를 추적합니다. 더 이상 사용하지 않는 기기가 있으면 계정에서 제거하여 기기 목록을 깔끔하고 안전하게 유지할 수 있습니다.

:::warning
연결된 기기를 정기적으로 검토하여 활성화되고 승인된 기기만 계정에 연결되어 있는지 확인하십시오. 더 이상 사용하지 않는 기기는 보안 유지를 위해 제거하십시오.
:::
