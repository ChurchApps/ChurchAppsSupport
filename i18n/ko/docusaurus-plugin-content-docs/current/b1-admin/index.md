# B1 Admin 개요

<div class="article-intro">

B1 Admin은 교회 관리 대시보드입니다. 교인 명부 관리, 기부금 관리, 웹사이트 구축, 실시간 스트리밍까지 교회 운영의 모든 측면을 한 곳에서 관리할 수 있는 도구를 제공합니다.

</div>

## 시작하기

1. 브라우저를 열고 [admin.b1.church](https://admin.b1.church)에 접속합니다.
2. 사용자명과 비밀번호로 로그인합니다.
3. **대시보드**에 도착하면 교회 데이터 개요와 교인 추가, 그룹 생성, 기부금 기록 등의 일반적인 작업을 위한 **빠른 작업**을 확인할 수 있습니다.

:::tip
B1 Admin이 처음이신가요? 먼저 [소개](./introduction)에서 영상 안내를 확인한 후, [설정](./settings/)에서 교회 정보를 구성하고 팀원을 초대하세요.
:::

## 주요 기능

- **[교인](./people/)** - 교인 명부 관리, 교인 추가, 데이터 대량 편집, 가구 추적
- **[그룹](./groups/)** - 교회 그룹 생성 및 구성, 가입 요청 및 교인 관리
- **[참석](./attendance/)** - 캠퍼스 설정, 예배 시간 및 참석 추적
- **[기부금](./donations/)** - 여러 통화로 기부금 기록, 기금 관리, 명세서 생성
- **[봉사](./serving/)** - 자원봉사자 조율, 예배 계획 수립, 작업 관리, 단계적 워크플로우를 통한 교인 이동
- **[양식](./forms/)** - 등록 및 데이터 수집을 위한 사용자 정의 양식 작성
- **[보고서](./reports/)** - 생일, 참석, 기부금 보고서 조회
- **[웹사이트](./website/)** - 사용자 정의 탐색 스타일로 교회 웹사이트 구축 및 관리
- **[설교](./sermons/)** - 설교 라이브러리 및 실시간 스트리밍 관리
- **[달력](./calendars/)** - 그룹 행사를 연결하여 큐레이션된 달력 생성
- **[설정](./settings/)** - 교회 정보, 역할 및 권한 구성

:::info

:::
