# B1 Admin 개요

<div class="article-intro">

B1 Admin은 교회 관리 대시보드입니다. 교인 명부와 헌금부터 웹사이트 구축 및 라이브 스트리밍까지, 교회 운영의 모든 측면을 한 곳에서 관리할 수 있는 도구를 제공합니다.

</div>

## 시작하기

1. 브라우저를 열고 [admin.b1.church](https://admin.b1.church)로 이동합니다
2. 사용자 이름과 비밀번호로 로그인합니다
3. **대시보드**로 이동되며, 여기에서 모든 주요 기능에 빠르게 접근할 수 있습니다

:::tip
B1 Admin이 처음이신가요? [소개](./introduction)에서 동영상 안내를 먼저 확인한 후, [설정](./settings/)에서 교회 정보를 구성하고 팀원을 초대하세요.
:::

## 주요 기능

- **[교인 관리](./people/)** - 교회 명부 관리, 교인 추가, 가구 추적
- **[그룹](./groups/)** - 교회 그룹, 팀, 사역부서 생성 및 조직
- **[출석](./attendance/)** - 캠퍼스, 예배 시간 설정 및 출석 추적
- **[헌금](./donations/)** - 헌금 기록, 기금 관리, 명세서 생성
- **[봉사](./serving/)** - 자원봉사자 조정, 봉사 계획 수립, 업무 관리
- **[양식](./forms/)** - 등록 및 데이터 수집을 위한 맞춤 양식 작성
- **[보고서](./reports/)** - 생일, 출석, 헌금 보고서 조회
- **[웹사이트](./website/)** - 교회 웹사이트 구축 및 관리
- **[설교](./sermons/)** - 설교 자료 관리 및 라이브 스트리밍
- **[일정](./calendars/)** - 행사 일정 생성 및 게시
- **[설정](./settings/)** - 교회 정보, 역할 및 권한 구성

:::info
도움말과 문서에 빠르게 접근하려면 페이지 오른쪽 상단의 물음표 아이콘을 클릭하세요.
:::

지원이 필요하시면 [support@churchapps.org](mailto:support@churchapps.org)으로 이메일을 보내주세요.
