# 설교 라이브러리 구축 및 실시간 스트리밍

<div class="article-intro">

설교 아카이브를 온라인에 게시하고 실시간 스트리밍을 설정합니다 -- 기존 YouTube 또는 Vimeo 라이브러리 가져오기부터 인터랙티브 채팅이 포함된 정기 실시간 예배 예약까지. 이 가이드를 완료하면 교인들이 과거 설교를 시청하고 웹사이트와 모바일 앱에서 실시간 예배에 참여할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 콘텐츠 관리 권한이 있는 B1 Admin 계정
- 설교 동영상이 있는 YouTube 또는 Vimeo 채널 (아카이브 가져오기용)
- YouTube 채널 ID (실시간 스트리밍용 -- YouTube Studio 설정에서 확인 가능)

</div>

## 1단계: 설교 재생목록 만들기

재생목록은 설교를 시리즈와 컬렉션으로 정리하여 쉽게 탐색할 수 있게 합니다.

[재생목록](../sermons/playlists.md) 가이드를 따라 다음을 수행하십시오:

1. "2024 주일 메시지", "결혼 시리즈", "크리스마스 예배"와 같은 시리즈/컬렉션을 만듭니다
2. 각 재생목록에 이름, 설명, 게시 날짜 및 썸네일을 추가합니다

:::tip
설교를 추가하기 전에 재생목록을 먼저 만드십시오. 이렇게 하면 설교를 추가할 때 올바른 시리즈에 쉽게 정리할 수 있습니다.
:::

## 2단계: 기존 설교 가져오기

YouTube 또는 Vimeo에 설교 동영상이 이미 있는 경우, 한 번에 모두 가져올 수 있습니다.

[일괄 가져오기](../sermons/bulk-import.md) 가이드를 따라 다음을 수행하십시오:

1. YouTube 또는 Vimeo 채널 URL을 입력합니다
2. 가져오기 미리보기에서 어떤 동영상이 가져와질지 확인합니다
3. 가져오기를 클릭하여 한 번에 모두 추가합니다

:::tip
자동 가져오기를 활성화하면 YouTube/Vimeo에 업로드하는 새 동영상이 자동으로 B1에 나타납니다 -- 수동 작업이 필요 없습니다.
:::

## 3단계: 개별 설교 추가

일회성 추가나 일괄 가져오기에 포함되지 않은 설교의 경우, 개별적으로 추가합니다.

[설교 관리](../sermons/managing-sermons.md) 가이드를 따라 다음을 수행하십시오:

1. 동영상 URL (YouTube, Vimeo 또는 Facebook)을 붙여넣습니다
2. 제목, 설명, 썸네일 및 길이가 동영상 메타데이터에서 자동으로 채워집니다
3. 설교를 재생목록에 배정합니다

## 4단계: 실시간 스트리밍 설정

YouTube 채널을 연결하여 B1 사이트가 자동으로 실시간 스트림을 인식하도록 합니다.

[실시간 스트리밍](../sermons/live-streaming.md) 가이드를 따라 다음을 수행하십시오:

1. 스트리밍 설정에 YouTube 채널 ID를 입력합니다
2. 정기 예배 일정을 구성합니다 (예: 매주 일요일 오전 10:00)
3. 스트리밍 페이지 탭을 사용자 정의합니다: 채팅, 기도 요청 및 사용자 정의 링크

:::info
YouTube에서 라이브를 시작하면 B1 스트리밍 페이지가 자동으로 스트림을 인식합니다. 주일 아침에 추가 단계가 필요하지 않습니다.
:::

## 5단계: 웹사이트에 표시

설교 라이브러리와 실시간 스트림을 방문자들이 쉽게 접근할 수 있도록 합니다.

[페이지 관리](../website/managing-pages.md) 가이드를 따라 다음을 수행하십시오:

1. B1 웹사이트에 설교 섹션을 추가하여 방문자들이 라이브러리를 탐색할 수 있도록 합니다
2. 실시간 시청 경험을 위한 실시간 스트리밍 섹션을 추가합니다

## 6단계: 모바일 접근

교인들은 모든 기기에서 설교와 실시간 예배에 접근할 수 있습니다.

1. 교인들은 [B1 모바일 앱](../../b1-mobile/content/sermons.md) 및 [B1.church](../../b1-church/content/sermons.md)에서 설교를 시청할 수 있습니다
2. 채팅과 기도 요청이 포함된 실시간 스트리밍은 두 플랫폼 모두에서 작동합니다

:::info
실시간 채팅과 기도 요청 기능은 온라인 교인들이 마치 현장에 있는 것처럼 실시간으로 참여할 수 있게 해줍니다.
:::

## 완료!

설교 라이브러리가 온라인에 게시되었고 실시간 스트림이 구성되었습니다. 교인들은 언제든지 과거 설교를 시청하고 어디서든 실시간 예배에 참여할 수 있습니다. 새로운 설교 업로드도 자동으로 동기화할 수 있습니다.
