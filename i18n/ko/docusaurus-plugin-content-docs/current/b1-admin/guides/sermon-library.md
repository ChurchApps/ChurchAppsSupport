# 설교 라이브러리 및 라이브 스트림 구축

<div class="article-intro">

기존 YouTube 또는 Vimeo 라이브러리 가져오기부터 대화형 채팅으로 정기적인 라이브 예배 예약까지 설교 보관소를 온라인으로 옮기고 라이브 스트리밍을 설정합니다. 마지막에 회중은 웹사이트와 모바일 앱에서 과거 설교를 보고 라이브 예배에 참여할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 콘텐츠 관리 권한이 있는 B1 Admin 계정
- 설교 비디오가 있는 YouTube 또는 Vimeo 채널(보관소 가져오기용)
- YouTube 채널 ID (라이브 스트리밍용 — YouTube 스튜디오 설정에서 찾기)

</div>

## 1단계: 설교 재생목록 만들기

재생목록은 설교를 시리즈와 컬렉션으로 구성하여 쉽게 탐색할 수 있습니다.

[재생목록](../sermons/playlists.md) 가이드를 따르세요:

1. "2024년 일요일 메시지", "결혼 시리즈", "크리스마스 예배" 같은 시리즈/컬렉션 만들기
2. 각 재생목록에 이름, 설명, 게시 날짜, 썸네일 추가

:::tip
설교를 추가하기 전에 재생목록을 만듭니다. 설교를 추가할 때 올바른 시리즈로 구성하기가 쉬워집니다.
:::

## 2단계: 기존 설교 가져오기

YouTube 또는 Vimeo에 설교 비디오가 이미 있으면 모두 한 번에 끌어올 수 있습니다.

[일괄 가져오기](../sermons/bulk-import.md) 가이드를 따르세요:

1. YouTube 또는 Vimeo 채널 URL 입력
2. 가져올 비디오를 미리보기로 확인
3. 가져오기를 클릭하여 모두 한 번에 추가

## 3단계: 개별 설교 추가

일회성 추가 또는 일괄 가져오기의 일부가 아닌 설교의 경우 개별적으로 추가합니다.

[설교 관리](../sermons/managing-sermons.md) 가이드를 따르세요:

1. 비디오 URL 붙여 넣기(YouTube, Vimeo 또는 Facebook)
2. 제목, 설명, 썸네일, 기간이 비디오 메타데이터에서 자동 채워짐
3. 설교를 재생목록에 할당

## 4단계: 라이브 스트리밍 설정

B1 사이트가 자동으로 라이브 스트림을 인식하도록 YouTube 채널을 연결합니다.

[라이브 스트리밍](../sermons/live-streaming.md) 가이드를 따르세요:

1. 스트리밍 설정에 YouTube 채널 ID 입력
2. 정기적인 예배 일정 구성(예: 일요일 오전 10:00)
3. 스트리밍 페이지 탭 사용자 정의: 채팅, 기도 요청, 사용자 정의 링크

:::info
YouTube에서 라이브로 전환하면 B1 스트리밍 페이지가 자동으로 스트림을 인식합니다. 일요일 아침에 추가 단계가 필요하지 않습니다.
:::

## 5단계: 웹사이트에 표시

설교 라이브러리 및 라이브 스트림을 방문자가 쉽게 액세스할 수 있도록 합니다.

[페이지 관리](../website/managing-pages.md) 가이드를 따르세요:

1. B1 웹사이트에 설교 섹션을 추가하여 방문자가 라이브러리를 탐색할 수 있도록 합니다
2. 라이브 보기 환경을 위해 라이브 스트림 섹션을 추가합니다

## 6단계: 모바일 액세스

회원들은 모든 기기에서 설교 및 라이브 예배에 액세스할 수 있습니다.

1. 회원들은 [B1 모바일 앱](../../b1-mobile/content/sermons.md) 및 [B1.church](../../b1-church/content/sermons.md)에서 설교를 볼 수 있습니다
2. 라이브 스트리밍과 채팅 및 기도 요청 기능은 두 플랫폼 모두에서 작동합니다

:::info
라이브 채팅 및 기도 요청 기능을 통해 온라인 회중이 실시간으로 참여할 수 있어 마치 그 자리에 있는 것 같습니다.
:::

## 완료되었습니다!

설교 라이브러리가 온라인에 있고 라이브 스트림이 구성되었습니다. 회원들은 언제든지 과거 설교를 따라 잡을 수 있으며 어디서나 라이브 예배에 참여할 수 있습니다. 새 설교 업로드는 자동으로 동기화될 수 있습니다.
