# 일요일 자원봉사자 관리

<div class="article-intro">

자원봉사자 팀을 설정하고, 주간 예배 계획을 만들고, 위치를 할당하고, 경배 노래가 있는 예배순서를 구축하고, 반복 작업을 자동화합니다. 마지막에 자원봉사자는 B1 웹사이트와 모바일 앱에서 자신의 할당을 보고 직접 수락 또는 거절할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 관리 액세스 권한이 있는 B1 Admin 계정
- 자원봉사자가 [사람 디렉터리에 추가](../people/adding-people.md)됨

</div>

## 1단계: 자원봉사자 팀 만들기

자원봉사자를 팀으로 구성하여 사역 영역별로 할당을 관리할 수 있습니다.

[그룹 만들기](../groups/creating-groups.md) 및 [그룹 회원](../groups/group-members.md) 가이드를 따르세요:

1. 범주 만들기(예: "일요일 아침 팀")
2. "경배팀", "음향 및 미디어", "접대자", "아동 사역 자원봉사자" 같은 그룹 만들기
3. 이름을 검색하고 추가를 클릭하여 각 그룹에 자원봉사자 추가

:::tip
회원 탭의 녹색 키 아이콘을 사용하여 팀 지도자를 지정합니다. 지도자는 자신의 그룹 캘린더를 관리하고 팀을 조정할 수 있습니다.
:::

## 2단계: 노래 라이브러리 구축

예배 계획에 매주 끌어올 수 있도록 경배 노래를 추가합니다.

[노래](../serving/songs.md) 가이드를 따르세요:

1. 제목, 아티스트, 기간이 있는 경배 노래 추가
2. 각 노래에 대해 가사와 키 표시가 있는 편곡 추가
3. YouTube 연주, 악보 또는 악보에 대한 외부 링크 추가

:::info
여기에 추가된 노래는 나중에 모든 예배 계획에 끌어올 수 있습니다. 라이브러리를 한 번 구축하고 매주 다시 사용합니다.
:::

## 3단계: 예배 계획 만들기

계획은 자원봉사자가 필요한 위치를 알려주는 주간 할당입니다.

[계획](../serving/plans.md) 가이드를 따르세요:

1. 봉사, 사역 탭을 선택합니다
2. "계획 추가"를 클릭하고 예배 날짜를 선택합니다
3. 할당 탭에서 각 팀을 확장하고 자원봉사자를 자신의 위치에 할당합니다
4. 자원봉사자는 자신의 할당 상태를 봅니다(요청됨, 확정됨, 대기중)

## 4단계: 예배순서 구축

처음부터 끝까지 예배의 흐름을 설정합니다.

[예배순서](../serving/service-order.md) 가이드를 따르세요:

1. 계획의 "예배순서" 탭으로 전환합니다
2. "항목 추가"를 클릭하여 요소를 추가합니다: 경배 노래(라이브러리에서), 기도, 성경 읽기, 공지사항, 설교, 헌금, 폐회
3. 예배 흐름을 재정렬하려면 항목을 끌어서 놓습니다

:::info
예배순서는 계획별입니다. 여기서의 변경은 다른 계획이나 마스터 노래 라이브러리에 영향을 주지 않습니다.
:::

## 5단계: 반복 작업 설정

주간 책임을 자동화하여 아무것도 놓치지 않습니다.

[작업](../serving/tasks.md) 및 [자동화](../serving/automations.md) 가이드를 따르세요:

1. 주간 정기적인 책임에 대한 자동화를 만듭니다(예: "성찬 요소 준비", "게시판 인쇄", "음향 장비 설정")
2. 각 자동화를 책임자인 사람 또는 그룹에 할당합니다
3. 작업이 B1 대시보드와 모바일 앱에 자동으로 표시됩니다

:::tip
휴일 휴식이나 특별한 시간 중에 자동화를 비활성화로 설정한 다음 준비가 되면 다시 활성화하세요. 삭제하고 다시 만들 필요가 없습니다.
:::

## 6단계: 자원봉사자 자체 신청 활성화(선택 사항)

관리 할당을 기다리는 대신 자원봉사자가 자신에게 위치를 신청하도록 합니다.

1. 계획을 편집할 때 **신청 마감** 시간(예배 전 시간)을 설정하여 자체 신청이 언제 닫히는지 제어합니다. 마감이 없으면 비워둡니다.
2. 선택적으로 토글 **신청 페이지에 자원봉사자 이름 표시** 다른 사람이 신청했는지 알려줍니다.
3. 각 위치에 대해 **자체 신청 허용** 확인란을 선택하여 자체 서비스에 사용 가능하게 만듭니다.
4. 각 위치에 **설명**을 추가하여 자원봉사자가 역할이 무엇인지 알 수 있도록 합니다.

자체 신청 위치가 만들어지면 [B1.church](../../b1-church/serving/volunteer-signup) 및 [B1 모바일 앱](../../b1-mobile/serving/volunteer-signup)의 자원봉사자 신청 페이지에 자동으로 나타납니다. 회원들은 열린 위치를 탐색하고, 남은 자리 수를 확인하고, 한 번의 클릭으로 신청할 수 있습니다.

:::tip
동일한 계획에서 관리 할당 및 자체 신청 위치를 섞습니다. 예를 들어 경배 지도자를 직접 할당하지만 인사자 및 커피 서버를 자체 선택하도록 합니다.
:::

## 7단계: 자원봉사자에게 알림

할당된 자원봉사자는 자동으로 계획을 보고 응답할 수 있습니다.

1. 자원봉사자는 [B1.church](../../b1-church/plans/viewing-plans.md) 및 [B1 모바일 앱](../../b1-mobile/serving/viewing-plans.md)에서 계획을 봅니다
2. 플랫폼에서 직접 할당을 수락 또는 거절할 수 있습니다
3. 사용할 수 없는 주에 대한 차단 날짜를 설정할 수 있습니다
4. 계획을 인쇄하여 리허설용 또는 무대 뒤에 게시할 수 있습니다
5. 자원봉사자는 [자원봉사자 신청 페이지](../../b1-church/serving/volunteer-signup)를 통해 열린 위치를 찾아 신청할 수 있습니다

:::tip
모바일 앱의 푸시 알림은 자원봉사자에게 새 할당을 받았거나 계획이 변경되었을 때 알립니다.
:::

## 완료되었습니다!

자원봉사자 관리 시스템이 설정되었습니다. 매주 새 계획을 만들고 위치를 할당하면 팀이 휴대폰이나 컴퓨터에서 모든 것을 봅니다. 반복 작업은 정기적인 준비 작업을 자동으로 처리합니다.
