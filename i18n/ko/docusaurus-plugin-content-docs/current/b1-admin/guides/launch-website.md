# 교회 웹사이트 출시

<div class="article-intro">

B1.church에는 추가 비용 없이 완전한 웹사이트 빌더가 포함되어 있습니다. 이 가이드에서는 교회 웹사이트를 처음부터 만드는 과정을 안내합니다 -- 홈페이지 설정, 디자인 구성, 주요 페이지 추가, 그리고 선택적으로 온라인 헌금 및 이벤트 등록 양식을 연결하는 방법까지 다룹니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교회 로고를 준비하십시오 (투명 배경의 PNG가 가장 좋습니다)
- 사이트에 사용할 2~3가지 브랜드 색상을 선택하십시오
- 사용자 정의 도메인 (예: yourchurch.com)을 사용하려면 DNS 공급자 (GoDaddy, Cloudflare 등)에 대한 접근 권한이 필요합니다
- 사이트에 온라인 헌금을 넣으려면 먼저 [온라인 헌금 설정](../donations/online-giving-setup.md) (Stripe)을 완료하십시오

</div>

## 1단계: 초기 웹사이트 설정

홈페이지와 기본 사이트 구조를 만드는 것부터 시작합니다.

[웹사이트 초기 설정](../website/initial-setup.md) 가이드를 따라 다음을 수행하십시오:

1. B1 Admin에서 **웹사이트**로 이동합니다
2. 히어로 섹션, 환영 메시지 및 주요 정보가 포함된 홈페이지를 만듭니다
3. 교회 이름과 태그라인을 추가합니다

## 2단계: 외관 구성

사이트의 시각적 아이덴티티를 설정합니다 -- 색상, 글꼴, 로고 및 푸터.

[외관](../website/appearance.md) 가이드를 따라 다음을 수행하십시오:

1. 교회 로고를 업로드합니다
2. 기본 색상과 강조 색상을 설정합니다
3. 내비게이션 바와 푸터를 구성합니다
4. 변경 사항을 미리 봅니다

:::tip
색상 팔레트를 간결하게 유지하십시오 -- 기본 색상 하나와 강조 색상 하나면 대부분 충분합니다. 웹사이트 빌더가 나머지를 처리합니다.
:::

## 3단계: 콘텐츠 페이지 추가

방문자들에게 가장 필요한 페이지를 구성합니다.

[페이지 관리](../website/managing-pages.md) 가이드를 따라 다음과 같은 페이지를 만드십시오:

- **소개** — 교회의 역사, 신앙고백 및 리더십
- **설교** — [설교 라이브러리](../sermons/managing-sermons.md) 링크
- **이벤트** — 예정된 이벤트 및 등록
- **헌금** — 온라인 헌금 페이지 ([Stripe 설정](../donations/online-giving-setup.md) 필요)
- **연락처** — 위치, 예배 시간 및 연락처 정보

## 4단계: 도메인 연결

기본 B1 URL 대신 자체 도메인 이름 (예: yourchurch.com)을 사용하려면:

1. B1 Admin에서 **웹사이트 > 설정**으로 이동합니다
2. 사용자 정의 도메인을 입력합니다
3. 도메인 공급자에서 DNS 레코드를 B1을 가리키도록 업데이트합니다

:::info
DNS 변경 사항이 전파되는 데 최대 48시간이 걸릴 수 있습니다. 사용자 정의 도메인에서 사이트에 즉시 접근할 수 없을 수 있습니다. 이 기간 동안 기본 B1 URL은 계속 작동합니다.
:::

## 5단계: 헌금 및 양식 추가

인터랙티브 요소로 사이트를 향상시킵니다:

- **온라인 헌금** — 교인들이 웹사이트에서 직접 헌금할 수 있도록 헌금 섹션을 추가합니다. 먼저 [온라인 헌금 설정](../donations/online-giving-setup.md)에서 Stripe를 구성하십시오.
- **등록 양식** — 이벤트 등록, 방문자 카드 또는 봉사자 신청을 위한 [독립형 양식](../forms/creating-forms.md)을 삽입합니다. 페이지에 양식 요소를 추가하는 방법은 [페이지 관리](../website/managing-pages.md)를 참조하십시오.

## 완료!

교회 웹사이트가 게시되었습니다. 교인들과 소셜 미디어에 URL을 공유하십시오. B1 Admin 대시보드에서 언제든지 콘텐츠를 업데이트하고, 새 페이지를 추가하고, 외관을 조정할 수 있습니다.
