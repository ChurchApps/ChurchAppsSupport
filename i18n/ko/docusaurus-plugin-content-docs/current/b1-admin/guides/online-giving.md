# 온라인 헌금 설정

<div class="article-intro">

교회에서 온라인 헌금을 받기 위해 필요한 모든 것을 안내합니다 -- 헌금 기금 생성부터, 결제 처리를 위한 Stripe 연결, 교인들에게 헌금 페이지 공유까지. 이 가이드를 완료하면 교인들이 웹사이트와 모바일 앱을 통해 온라인으로 헌금할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 관리자 권한이 있는 B1 Admin 계정 — [역할 및 권한](../people/roles-permissions.md)을 참조하십시오
- Stripe 계정 (필요한 경우 [stripe.com](https://stripe.com)에서 무료로 생성)

</div>

## 1단계: 헌금 기금 만들기

기금은 헌금자가 헌금할 수 있는 범주입니다. 헌금을 받으려면 최소한 하나의 기금이 필요합니다.

[기금](../donations/funds.md) 가이드를 따라 헌금 범주를 설정하십시오:

1. 가장 일반적인 기금을 만듭니다 (예: "일반 헌금", "건축 기금", "선교 헌금")
2. 세금 공제 기금을 적절하게 표시합니다 -- 이는 연말 헌금 명세서에 영향을 미칩니다

:::tip
기금은 언제든지 추가할 수 있습니다. 가장 일반적인 헌금 범주부터 시작하십시오.
:::

## 2단계: Stripe 연결

Stripe가 모든 결제 처리를 담당합니다. Stripe 계정을 B1 Admin에 연결하여 헌금이 은행 계좌로 입금되도록 합니다.

[온라인 헌금 설정](../donations/online-giving-setup.md) 가이드를 따라 Stripe를 연결하십시오:

1. Stripe 대시보드에 로그인하여 Publishable Key와 Secret Key를 가져옵니다
2. B1 Admin에서 설정으로 이동하여 두 키를 입력합니다

:::warning
Stripe는 Secret Key를 한 번만 표시합니다. Stripe 대시보드를 떠나기 전에 복사하여 저장하십시오. 분실한 경우 새로 생성해야 합니다.
:::

## 3단계: 웹사이트에 헌금 페이지 추가

B1 웹사이트에 헌금 페이지를 추가하여 헌금을 쉽게 할 수 있도록 합니다.

[온라인 헌금 설정](../donations/online-giving-setup.md) 및 [페이지 관리](../website/managing-pages.md) 가이드를 따라 다음을 수행하십시오:

1. B1.church 사이트에 "헌금" 탭을 추가합니다
2. 헌금 URL은 다음과 같습니다: `https://yoursubdomain.b1.church/donate`
3. 교인들은 로그인 없이도 헌금할 수 있고 (공개 페이지), 로그인하면 저장된 결제 수단과 헌금 내역을 사용할 수 있습니다

## 4단계: 테스트 헌금 수행

교인들에게 공지하기 전에 모든 것이 작동하는지 확인합니다.

1. 소액 테스트 헌금을 하여 전체 흐름이 정상적으로 작동하는지 확인합니다
2. B1 Admin의 헌금에 해당 헌금이 나타나는지 확인합니다

:::tip
실제 청구 없이 확인하려면 먼저 Stripe의 테스트 모드를 사용한 다음, 교인들에게 공지하기 전에 라이브 모드로 전환하십시오.
:::

## 5단계: 교인들에게 공지

교인들에게 온라인 헌금이 가능함을 알립니다.

1. 웹사이트, 이메일 뉴스레터, 주보 및 소셜 미디어를 통해 헌금 URL을 공유합니다
2. 교인들은 [B1 모바일 앱](../../b1-mobile/giving/)을 통해서도 헌금할 수 있습니다 -- 헌금 기능이 내장되어 있습니다

:::info
로그인한 교인은 결제 수단을 저장하고, 정기 헌금을 설정하고, 헌금 내역을 확인할 수 있습니다. 익명 헌금도 가능합니다 -- 로그인이 필요하지 않습니다.
:::

## 6단계: 지속적인 관리

헌금 기록을 최신 상태로 유지하고 연중 보고서를 생성합니다.

1. 기록을 최신 상태로 유지하기 위해 정기적으로 (주간 또는 월간) [Stripe 거래를 가져옵니다](../donations/stripe-import.md)
2. [헌금 보고서를 확인](../donations/donation-reports.md)하여 기금별 헌금 추이와 합계를 추적합니다
3. 헌금자의 세금 기록을 위해 [연말 헌금 명세서를 생성](../donations/giving-statements.md)합니다

:::tip
기록이 최신 상태를 유지하도록 최소 월 1회 Stripe 가져오기를 실행하십시오. 전체 연말 프로세스는 [연말 보고서 가이드](./year-end-reports.md)를 참조하십시오.
:::

## 완료!

교회에서 이제 온라인 헌금을 받고 있습니다. 교인들은 웹사이트, B1 모바일 앱 또는 웹 브라우저가 있는 모든 기기에서 헌금할 수 있습니다. 모든 헌금은 B1 Admin에서 자동으로 추적됩니다.
