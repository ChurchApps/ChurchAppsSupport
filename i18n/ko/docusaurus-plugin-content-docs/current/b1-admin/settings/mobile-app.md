# 모바일 앱 설정

<div class="article-intro">

모바일 앱 설정 페이지를 사용하면 교회 회원들을 위해 **B1.church 모바일 경험(PWA)**에 나타나는 네비게이션 탭을 구성할 수 있습니다. 어느 탭이 표시될지, 무엇에 연결될지, 어떻게 표시될지를 제어합니다.

</div>

:::info 기본 B1 Mobile 앱은 단종되었습니다
여기서 구성한 탭은 기본 B1 Mobile 앱을 대체한 [B1.church Progressive Web App (PWA)](/docs/b1-church/getting-started/installing-pwa)를 통해 제공됩니다. 회원들은 App Store 또는 Google Play에서 기존 앱을 다운로드하는 대신 https://yourchurchname.b1.church의 모든 페이지에서 PWA를 설치할 수 있습니다.
:::

<div class="prereqs">
<h4>시작하기 전에</h4>

- "교회 설정 편집" 권한이 필요합니다. 접근 권한이 없으면 [역할 및 권한](./roles-permissions.md)을 참조하세요.
- 교회 이름 및 브랜딩을 포함하여 [Church Settings](./church-settings.md)을 먼저 구성하세요

</div>

## 모바일 앱 설정 접근하기

1. 왼쪽 사이드바의 **Settings** 페이지로 이동하세요.
2. 헤더의 **Mobile Apps** 버튼을 클릭하세요.
3. 모바일 앱 설정 페이지가 현재 앱 탭을 표시합니다.

## 새 탭 추가하기

1. 페이지 상단의 **Add Tab** 버튼을 클릭하세요.
2. 탭 세부정보를 작성하세요:
   - **Name** -- 탭에 표시되는 레이블(예: "Sermons" 또는 "Give").
   - **Icon** -- 아이콘 선택기를 클릭하여 탭의 아이콘을 선택하세요. 사용자 정의 이미지를 업로드할 수도 있습니다.
   - **Tab Type** -- Bible, Live Stream, Donation, Website 등의 옵션에서 선택하세요.
   - **URL** -- 탭이 연결되어야 하는 웹 주소를 입력하세요.
   - **Visibility** -- 누가 이 탭을 볼 수 있는지 제어하세요(모두, 회원만 등).
3. **Save Tab**을 클릭하여 앱에 추가하세요.

## 기존 탭 편집하기

1. **App Tabs** 목록의 기존 탭을 클릭하세요.
2. 탭의 이름, 아이콘, URL, 유형 또는 표시 설정을 업데이트하세요.
3. **Save Tab**을 클릭하여 변경사항을 적용하세요.

## 탭 순서 변경하기

모바일 앱에 탭이 나타나는 순서를 변경할 수 있습니다. 목록의 탭을 드래그하여 순서를 변경하세요. 이 페이지에 표시된 순서는 회원들이 앱에서 보게 되는 순서와 일치합니다.

:::info
일부 탭은 특정 조건이 충족될 때 자동으로 나타날 수 있습니다 -- 예를 들어, 스트림이 활성일 때 라이브 스트림 탭이 나타날 수 있습니다. 수동으로 추가된 탭은 항상 회원들이 보는 것을 완전히 제어할 수 있도록 합니다.
:::

:::tip
탭 개수를 관리 가능하게 유지하세요. 대부분의 교회에서는 3~5개의 탭이 잘 작동합니다. 탭이 너무 많으면 회원들을 위해 네비게이션이 혼란스러워질 수 있습니다.
:::

## 이 탭이 나타나는 위치

여기서 구성한 탭은 회원들이 https://yourchurchname.b1.church의 모든 페이지에서 설치하는 **B1.church PWA**에 표시됩니다. 이 페이지에서 변경한 내용은 회원이 앱을 다음에 열 때 반영됩니다. (탭은 기본 [B1 Mobile native app](/docs/b1-mobile/)을 실행 중인 회원들을 위해서도 렌더링되지만, 해당 앱은 단종되었으며 더 이상 업데이트되지 않습니다.)
