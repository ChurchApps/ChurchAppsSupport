# 모바일 앱 설정

<div class="article-intro">

모바일 앱 설정 페이지에서는 교인들이 사용하는 B1 모바일 앱에 표시되는 내비게이션 탭을 구성할 수 있습니다. 어떤 탭을 표시할지, 어디에 연결할지, 어떻게 표시할지를 제어할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- "교회 설정 편집" 권한이 필요합니다. 접근 권한이 없는 경우 [역할 및 권한](./roles-permissions.md)을 참조하십시오.
- 교회 이름과 브랜딩을 포함한 [교회 설정](./church-settings.md)을 먼저 구성하십시오

</div>

## 모바일 앱 설정 접근

1. 왼쪽 사이드바에서 **설정** 페이지로 이동합니다.
2. 헤더에서 **모바일 앱** 버튼을 클릭합니다.
3. 모바일 앱 설정 페이지에 현재 앱 탭이 표시됩니다.

## 새 탭 추가

1. 페이지 상단의 **탭 추가** 버튼을 클릭합니다.
2. 탭 세부 정보를 입력합니다:
   - **이름** -- 탭에 표시되는 레이블입니다 (예: "설교" 또는 "헌금").
   - **아이콘** -- 아이콘 선택기를 클릭하여 탭의 아이콘을 선택합니다. 사용자 정의 이미지를 업로드할 수도 있습니다.
   - **탭 유형** -- 성경, 실시간 스트리밍, 헌금, 웹사이트 등의 옵션 중에서 선택합니다.
   - **URL** -- 탭이 연결될 웹 주소를 입력합니다.
   - **공개 범위** -- 이 탭을 볼 수 있는 대상을 제어합니다 (모든 사람, 교인만 등).
3. **탭 저장**을 클릭하여 앱에 추가합니다.

## 기존 탭 편집

1. **앱 탭** 목록에서 기존 탭을 클릭합니다.
2. 탭의 이름, 아이콘, URL, 유형 또는 공개 범위 설정을 업데이트합니다.
3. **탭 저장**을 클릭하여 변경 사항을 적용합니다.

## 탭 순서 변경

모바일 앱에서 탭이 표시되는 순서를 변경할 수 있습니다. 목록에서 탭을 드래그 앤 드롭하여 재배열합니다. 이 페이지에 표시된 순서가 교인들이 앱에서 보게 되는 순서와 일치합니다.

:::info
특정 조건이 충족되면 일부 탭이 자동으로 나타날 수 있습니다 -- 예를 들어, 스트리밍이 활성화되면 실시간 스트리밍 탭이 표시될 수 있습니다. 수동으로 추가한 탭을 사용하면 교인들에게 항상 표시되는 항목을 완전히 제어할 수 있습니다.
:::

:::tip
탭 수를 적절하게 유지하십시오. 대부분의 교회에는 3~5개의 탭이 적합합니다. 탭이 너무 많으면 교인들이 내비게이션에 혼란을 느낄 수 있습니다.
:::

## 탭이 표시되는 위치

여기에서 구성한 탭은 교인들이 App Store 또는 Google Play에서 다운로드하는 **B1 모바일 앱** (B1.church 앱)에 표시됩니다. 이 페이지에서 변경한 내용은 교인이 다음에 앱을 열 때 반영됩니다.
