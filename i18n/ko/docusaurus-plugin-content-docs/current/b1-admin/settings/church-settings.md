# 교회 설정

<div class="article-intro">

교회 설정 페이지에서는 교회의 기본 정보, 연락처 세부 사항 및 브랜딩을 구성합니다. 이 정보는 B1.church 웹사이트와 B1 모바일 앱을 포함한 모든 ChurchApps 도구에서 사용됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- "교회 설정 편집" 권한이 필요합니다. 접근 권한이 없는 경우 [역할 및 권한](./roles-permissions.md)을 참조하십시오.
- 교회의 주소, 연락처 정보 및 로고를 준비하십시오

</div>

## 교회 정보 편집

1. 왼쪽 사이드바에서 **설정** 페이지로 이동합니다.
2. 헤더에서 **설정 편집** 버튼을 클릭합니다.
3. 다음 필드를 업데이트합니다:
   - **교회 이름** -- 모든 ChurchApps 제품에 표시되는 이름입니다.
   - **주소** -- 교회의 물리적 주소입니다.
   - **연락처 정보** -- 전화번호, 이메일 및 기타 연락처 세부 사항입니다.
4. **저장**을 클릭하여 변경 사항을 적용합니다.

## 서브도메인 설정

교회는 **yourchurch.b1.church** 형태의 무료 서브도메인을 받을 수 있습니다. 이것은 교인들과 방문자들이 교회의 온라인 존재에 접근할 수 있는 웹 주소입니다.

1. 설정 페이지에서 **서브도메인** 필드를 찾습니다.
2. 원하는 서브도메인을 입력합니다 (예: gracechurch.b1.church의 경우 "gracechurch").
3. 변경 사항을 저장합니다.

:::info
서브도메인은 모든 ChurchApps 교회 중에서 고유해야 합니다. 원하는 이름이 이미 사용 중인 경우 도시나 지역명을 추가해 보십시오 (예: "gracechurch-dallas").
:::

## 브랜딩 구성

모든 ChurchApps 도구에서 교회가 표시되는 방식을 사용자 정의합니다:

1. 로고 영역을 클릭하고 이미지 파일을 선택하여 **교회 로고**를 업로드합니다.
2. 웹사이트 및 [모바일 앱](./mobile-app.md)에 사용되는 추가 **교회 이미지**를 추가합니다.

:::tip
최상의 결과를 위해 투명 배경의 PNG 형식 로고를 사용하십시오. 이렇게 하면 밝은 배경과 어두운 배경 모두에서 멋지게 표시됩니다.
:::

## 가져오기 및 내보내기

설정 헤더의 **가져오기/내보내기** 버튼을 클릭하면 새 브라우저 창에서 전용 도구가 열립니다. 이를 통해 다음을 수행할 수 있습니다:

- 다른 교회 관리 시스템에서 교인 데이터를 가져옵니다.
- 백업 또는 마이그레이션 목적으로 ChurchApps 데이터를 내보냅니다.

이 기능은 교회를 처음 설정하고 기존 기록을 ChurchApps로 이전해야 할 때 특히 유용합니다.

:::warning
데이터를 가져오기 전에 항상 기존 기록을 백업하십시오. 가져오기 작업은 시스템에 데이터를 추가하며, 여러 번 실행하면 중복 항목이 생성될 수 있습니다.
:::
