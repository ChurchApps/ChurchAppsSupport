# 설정

<div class="article-intro">

설정 섹션에서는 B1 Admin에서 교회 계정, 브랜딩 및 관리 옵션을 구성합니다. 교회 이름과 서브도메인부터 사용자 권한 및 모바일 앱 설정까지 모든 것을 관리할 수 있습니다.

</div>

## 여기에서 찾을 수 있는 항목

설정 영역은 사이드바에서 접근할 수 있는 네 가지 하위 섹션으로 구성되어 있습니다:

1. **설정** -- 교회의 기본 정보, 브랜딩 및 서브도메인을 구성합니다. 자세한 내용은 [교회 설정](./church-settings.md)을 참조하십시오.
2. **모바일 앱** -- 교인들이 사용하는 B1 모바일 앱에 표시되는 내비게이션 탭을 설정하고 사용자 정의합니다. 자세한 내용은 [모바일 앱 설정](./mobile-app.md)을 참조하십시오.
3. **서버 관리** -- 교회의 서버 수준 설정을 관리하기 위한 고급 관리 도구에 접근합니다.
4. **양식** -- 교인들로부터 정보를 수집하기 위한 사용자 정의 양식을 생성하고 관리합니다.

## 기본 설정 페이지

설정 페이지를 처음 열면 상단에 **교회 이름**과 **서브도메인**이 표시됩니다. 헤더에는 일반적인 작업을 위한 빠른 접근 버튼이 제공됩니다:

- **설정 편집** -- 교회의 이름, 주소, 연락처 정보 및 브랜딩을 업데이트합니다.
- **모바일 앱** -- 모바일 앱 설정으로 바로 이동합니다.
- **역할** -- 팀의 사용자 역할과 권한을 관리합니다.
- **가져오기/내보내기** -- 가져오기/내보내기 도구를 사용하여 시스템 간 데이터를 전송합니다.

:::tip
먼저 **설정 편집**에서 교회 이름과 브랜딩을 구성한 다음, [역할](./roles-permissions.md)을 설정하여 팀원들에게 적절한 접근 권한을 부여하십시오.
:::

:::info
데이터 보호에 대한 정보는 [데이터 보안](./data-security.md)을 참조하십시오.
:::
