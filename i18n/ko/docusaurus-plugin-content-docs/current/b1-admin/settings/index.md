# 설정

<div class="article-intro">

설정 섹션은 B1 Admin에서 교회 계정, 브랜딩, 관리 옵션을 구성하는 곳입니다. 교회 이름과 서브도메인부터 사용자 권한과 모바일 앱 구성까지 모든 것을 관리할 수 있습니다.

</div>

## 여기서 찾을 수 있는 것

설정 영역은 사이드바에서 접근 가능한 다음 세부 섹션으로 구성되어 있습니다.

1. **설정** -- 교회의 기본 정보, 브랜딩, 서브도메인을 구성합니다. 자세한 내용은 [교회 설정](./church-settings.md)을 참고하세요.
2. **캠퍼스** -- 교회의 여러 물리적 위치를 추가하고 관리합니다. 생성된 후 캠퍼스는 교인 프로필, 참석 설정, 통계 대시보드에 나타납니다. 자세한 내용은 [캠퍼스](./campuses.md)를 참고하세요.
3. **모바일 앱** -- [B1.church PWA](/docs/b1-church/getting-started/installing-pwa)에서 교인에게 표시되는 탐색 탭을 설정하고 사용자 정의합니다(같은 탭은 더 이상 사용되지 않는 B1 모바일 네이티브 앱에서도 렌더링됩니다). 자세한 내용은 [모바일 앱 설정](./mobile-app.md)을 참고하세요.
4. **서버 관리** -- 교회의 서버 레벨 설정을 관리하기 위한 고급 관리 도구에 접근합니다.
5. **양식** -- 교인으로부터 정보를 수집하기 위한 사용자 정의 양식을 만들고 관리합니다.

## 메인 설정 페이지

설정 페이지를 처음 열면 상단에 **교회 이름**과 **서브도메인**이 표시됩니다. 헤더에는 일반적인 작업을 위한 빠른 접근 버튼이 있습니다.

- **설정 편집** -- 교회 이름, 주소, 연락처 정보, 브랜딩을 업데이트합니다.
- **모바일 앱** -- 모바일 앱 구성으로 바로 이동합니다.
- **역할** -- 팀의 사용자 역할 및 권한을 관리합니다.
- **가져오기/내보내기** -- 가져오기/내보내기 도구를 사용하여 시스템 간 데이터를 이동합니다.

:::tip
**설정 편집**에서 교회 이름과 브랜딩을 먼저 구성한 후, [역할](./roles-permissions.md)을 설정하여 적절한 접근 권한을 가진 팀원을 초대하세요.
:::

:::info
데이터 보호 방법에 대해 알아보려면 [데이터 보안](./data-security.md)을 참고하세요.
:::
