# 캠퍼스

<div class="article-intro">

교회가 하나 이상의 위치에서 만나면 **캠퍼스**를 사용하여 각 사람과 그룹이 속한 사이트를 추적할 수 있습니다. 구성된 후 캠퍼스는 교인 프로필, 참석 설정, 통계 대시보드에 옵션으로 나타납니다. 다중 캠퍼스 교회는 B1 Admin 전체에서 캠퍼스별로 필터링, 검색, 보고할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 캠퍼스를 관리하려면 **교회 설정 편집** 권한이 필요합니다. [역할 및 권한](./roles-permissions.md)을 참고하세요.

</div>

## 캠퍼스 설정 열기

B1 Admin에서 좌측 사이드바의 **설정**으로 이동하고 설정 탐색에서 **캠퍼스**를 선택합니다. 구성된 모든 캠퍼스의 이름, 위치, 시간대가 표시된 목록이 나타납니다.

## 캠퍼스 추가

1. **캠퍼스 추가**(또는 아직 캠퍼스가 없으면 **+** 버튼)를 클릭합니다.
2. 캠퍼스 세부 사항을 입력합니다:
   - **이름** *(필수)* — B1 Admin 전체에 표시되는 이름(예: "메인 캠퍼스" 또는 "북쪽 캠퍼스").
   - **주소** — 캠퍼스 거리 주소(정보 표시용; 교회 설정의 메인 주소와 다름).
   - **도시 / 주 / 우편번호** — 캠퍼스 위치.
   - **시간대** — 이 캠퍼스의 IANA 시간대(예: *America/Chicago*). 캠퍼스가 다른 시간대에 있을 때 유용합니다.
   - **웹사이트** -- 이 캠퍼스의 자체 웹 존재를 위한 선택적 URL.
3. **저장**을 클릭합니다.

## 캠퍼스 편집

목록의 캠퍼스 행을 클릭하여 오른쪽 패널에서 편집기를 엽니다. 필드를 업데이트하고 **저장**을 클릭합니다.

## 캠퍼스 삭제

편집을 위해 캠퍼스를 열고 **삭제**를 클릭합니다. 확인이 필요합니다. 캠퍼스를 삭제해도 배정된 사람이 제거되지 않습니다. 그들의 캠퍼스 필드는 단순히 공백이 됩니다.

## 캠퍼스에 사람 배정

캠퍼스를 만든 후 직원은 프로필에서 사람을 캠퍼스에 배정할 수 있습니다:

1. **교인**에서 사람의 기록을 엽니다.
2. **편집**을 클릭합니다.
3. **캠퍼스** 드롭다운에서 캠퍼스를 선택합니다.
4. **저장**을 클릭합니다.

교인 페이지에서 캠퍼스를 대량으로 업데이트할 수도 있습니다. 여러 사람을 선택하고, **대량 편집**을 사용하고, 모두에게 캠퍼스 필드를 한 번에 설정합니다.

## 캠퍼스별 필터링

캠퍼스가 설정되면 B1 Admin 전체에서 캠퍼스별로 필터링할 수 있습니다.

- **교인 검색** -- 고급 검색에서 캠퍼스 조건을 추가하거나, 캠퍼스 범위의 [저장된 목록](../people/lists.md)을 불러옵니다.
- **통계** -- [통계 대시보드](../people/demographics.md)는 적어도 하나의 사람이 캠퍼스에 배정되어 있으면 캠퍼스 도넛 차트를 표시합니다.
- **참석 설정** -- 참석의 각 예배 시간을 캠퍼스와 연결할 수 있습니다.

:::tip
단일 위치 교회는 캠퍼스를 구성할 필요가 없습니다. 모든 캠퍼스 기능은 선택 사항입니다. 캠퍼스가 없으면 캠퍼스 필드와 차트가 단순히 나타나지 않습니다.
:::
