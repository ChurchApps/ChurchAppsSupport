# 저장된 목록

<div class="article-intro">

**저장된 목록**은 검색 쿼리를 이름으로 저장했다가 언제든지 한 번의 클릭으로 다시 실행할 수 있도록 합니다. "캠퍼스 노스의 활동적인 교인", "지난 90일간의 방문자" 또는 고급 필터 등의 목록을 한 번 만들면, 쿼리를 다시 만들 필요 없이 사이드바에 남아 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교인 섹션에서 적어도 한 번의 검색을 실행하여 저장할 결과가 있어야 합니다. [교인 검색](./searching-people.md) 또는 [AI 검색](./ai-search.md)을 참고하세요.

</div>

## 검색을 목록으로 저장

1. **교인** 페이지에서 검색을 실행합니다. 빠른 검색, AI 검색, 또는 고급 필터 검색입니다.
2. 결과가 나타나면 교인 페이지 헤더에 **목록으로 저장** 버튼(북마크 아이콘)이 표시됩니다.
3. **목록으로 저장**을 클릭합니다.
4. 목록의 **이름**을 입력합니다(필수).
5. 선택적으로 관련 목록을 사이드바에 그룹화할 **카테고리**를 입력합니다(예: "전도" 또는 "교인권").
6. **저장**을 클릭합니다.

목록이 교회 계정에 저장되고 교인 페이지의 좌측 **저장된 목록** 패널에 나타납니다.

:::info
목록은 교회 전체에서 공유됩니다. 교인 접근 권한이 있는 모든 사람이 목록을 보고 불러올 수 있습니다. 적절한 권한이 있는 직원만 목록의 이름을 바꾸거나 삭제할 수 있습니다.
:::

## 저장된 목록 불러오기

교인 페이지 좌측 **저장된 목록** 패널에서 목록 이름을 클릭합니다. 검색이 즉시 다시 실행되고 저장된 쿼리로 결과 표가 업데이트됩니다.

목록은 카테고리별로 그룹화됩니다(카테고리를 할당한 경우). 카테고리가 없는 목록은 마지막에 나타납니다.

:::tip
목록은 **실시간 쿼리**이며, 스냅샷이 아닙니다. 목록을 불러올 때마다 검색이 현재 데이터에 대해 새로 실행되므로, 조건과 이제 일치하는 새 교인이 나타납니다.
:::

## 목록 이름 바꾸기 및 삭제

관리 권한이 있는 직원은 저장된 목록 패널에서 직접 목록의 이름을 바꾸거나 삭제할 수 있습니다.

- 목록 이름 옆의 **연필 아이콘**을 클릭하여 이름을 바꿉니다.
- **휴지통 아이콘**을 클릭하여 삭제합니다(확인이 필요함).

## 사용 사례

| 시나리오 | 목록을 만드는 방법 |
|---|---|
| 모든 방문자 | 고급 검색: 교인 상태 = 방문자 |
| 특정 캠퍼스의 교인 | 고급 검색: 캠퍼스 = [캠퍼스 이름] |
| 이메일 주소가 없는 사람 | 고급 검색: 이메일이 비어 있음 |
| AI 질문의 결과 | AI 검색에서 질문을 한 후 저장 |
