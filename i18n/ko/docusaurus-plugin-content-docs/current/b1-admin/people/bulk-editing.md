# 사람 일괄 편집

<div class="article-intro">
일괄 편집을 사용하면 여러 사람을 한 번에 업데이트할 수 있어 여러 개인에게 동일한 변경을 수행할 때 시간을 절약할 수 있습니다. 한 번의 작업으로 회원 상태, 결혼 상태, 성별, 수신 거부 기본 설정 및 그룹 구성원 자격을 업데이트할 수 있습니다.
</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 사람 데이터를 관리할 수 있는 권한이 필요합니다. 자세한 내용은 [역할 및 권한](./roles-permissions.md)을 참조하세요.
- 편집하려는 사람을 이미 추가하거나 가져왔어야 합니다. 필요한 경우 [사람 추가](./adding-people.md)를 참조하세요.
</div>

## 일괄 편집할 사람 선택

1. B1 Admin에서 **사람**으로 이동합니다
2. 검색 또는 필터 도구를 사용하여 업데이트하려는 사람을 찾습니다
3. 각 사람의 이름 옆에 있는 확인란을 선택합니다
   - 개별적으로 사람을 선택할 수 있습니다
   - 또는 헤더 확인란을 사용하여 현재 페이지의 모든 표시된 사람을 선택합니다
4. 한 명 이상의 사람을 선택하면 **일괄 작업** 버튼이 나타납니다

:::tip
특정 기준에 따라 많은 사람을 업데이트해야 하는 경우 [AI 검색](./ai-search.md) 기능이나 필터를 사용하여 먼저 목록을 좁힌 다음 일치하는 모든 사람을 선택하세요.
:::

## 사용 가능한 일괄 작업

**일괄 작업** 메뉴는 여러 옵션을 제공합니다.

### 회원 상태 업데이트

선택한 모든 사람의 회원 상태를 업데이트합니다.

1. **일괄 작업** → **회원 상태 설정**을 클릭합니다
2. 새 상태를 선택합니다.
   - **방문자** -- 처음 또는 가끔 참석자
   - **정기 출석자** -- 회원이 아닌 자주 참석하는 사람
   - **회원** -- 공식 교회 회원
   - **직원** -- 교회 직원
   - **비활성** -- 더 이상 참석하지 않는 사람
3. 업데이트 모드를 선택합니다.
   - **모두 덮어쓰기** -- 선택한 모든 사람의 현재 상태 교체
   - **빈 항목만 업데이트** -- 상태가 없는 사람에 대해서만 상태 설정
4. **업데이트**를 클릭합니다

### 결혼 상태 업데이트

결혼 상태를 일괄 업데이트합니다.

1. **일괄 작업** → **결혼 상태 설정**을 클릭합니다
2. 새 상태를 선택합니다.
   - **알 수 없음**
   - **미혼**
   - **기혼**
   - **이혼**
   - **사별**
3. 기존 값을 덮어쓸지 또는 빈 필드만 업데이트할지 선택합니다
4. **업데이트**를 클릭합니다

### 성별 업데이트

여러 사람의 성별 정보를 업데이트합니다.

1. **일괄 작업** → **성별 설정**을 클릭합니다
2. 값을 선택합니다.
   - **지정되지 않음**
   - **남성**
   - **여성**
3. 업데이트 모드를 선택합니다 (모두 덮어쓰기 또는 빈 항목만)
4. **업데이트**를 클릭합니다

### 수신 거부 상태 업데이트

사람들이 커뮤니케이션을 수신 거부했는지 여부를 제어합니다.

1. **일괄 작업** → **수신 거부 설정**을 클릭합니다
2. 선택:
   - **아니요** -- 커뮤니케이션 허용 (수신 거부 제거)
   - **예** -- 커뮤니케이션 차단 (수신 거부 설정)
3. 업데이트 모드를 선택합니다
4. **업데이트**를 클릭합니다

:::warning
수신 거부 상태를 변경할 때 주의하세요. 명시적으로 수신 거부한 사람은 새로운 동의를 하지 않는 한 커뮤니케이션을 받아서는 안 됩니다.
:::

### 그룹에 추가

선택한 모든 사람을 하나 이상의 그룹에 추가합니다.

1. **일괄 작업** → **그룹에 추가**를 클릭합니다
2. 사람을 추가할 그룹을 검색하고 선택합니다
3. 여러 그룹을 선택하여 모든 그룹에 사람을 추가할 수 있습니다
4. **그룹에 추가**를 클릭합니다

각 사람은 선택한 그룹의 일반 구성원으로 추가됩니다. 필요한 경우 나중에 [그룹 구성원](../groups/group-members.md) 페이지에서 개인을 그룹 리더로 승격할 수 있습니다.

### 그룹에서 제거

선택한 모든 사람을 하나 이상의 그룹에서 제거합니다.

1. **일괄 작업** → **그룹에서 제거**를 클릭합니다
2. 사람을 제거할 그룹을 검색하고 선택합니다
3. 여러 그룹을 선택할 수 있습니다
4. **그룹에서 제거**를 클릭합니다

:::info
이 작업은 지정된 그룹에서만 사람을 제거합니다. 사람 레코드를 삭제하지 않습니다.
:::

### 사람 삭제

교회 데이터베이스에서 선택한 사람을 영구적으로 삭제합니다.

1. **일괄 작업** → **삭제**를 클릭합니다
2. 삭제될 사람 목록을 검토합니다
3. 확인 필드에 **DELETE**를 입력합니다
4. **삭제 확인**을 클릭합니다

:::danger
사람 삭제는 영구적이며 취소할 수 없습니다. 다음을 포함한 모든 데이터가 제거됩니다.
- 개인 정보
- 그룹 구성원 자격
- 출석 기록
- 헌금 기록
- 양식 제출

시스템에서 이러한 사람을 제거하려는 것이 확실한 경우에만 이 작업을 사용하세요.
:::

## 일괄 편집 결과

일괄 작업을 완료한 후 다음을 보여주는 요약이 표시됩니다.

- **선택된 총 수** -- 작업에 포함된 사람 수
- **성공적으로 업데이트됨** -- 변경된 레코드 수
- **실패** -- 업데이트할 수 없는 레코드 (해당하는 경우)
- **변경되지 않음** -- 변경이 필요하지 않은 레코드 (예: "빈 항목만 업데이트" 모드를 사용할 때)

업데이트가 실패한 경우 이유를 설명하는 오류 세부 정보가 표시됩니다.

## 모범 사례

- **작게 시작** -- 올바른 변경을 수행하고 있는지 확인하기 위해 먼저 몇 개의 레코드에서 일괄 작업을 테스트합니다
- **필터 사용** -- 사람을 선택하기 전에 필터 또는 AI 검색으로 목록을 좁혀 올바른 개인만 업데이트하도록 합니다
- **선택 항목 다시 확인** -- 일괄 변경을 적용하기 전에 선택한 사람을 검토합니다
- **"빈 항목만 업데이트" 모드 사용** -- 기존 정보를 덮어쓰지 않고 누락된 데이터를 채우려는 경우
- **주요 변경 사항 문서화** -- 나중에 참조해야 하는 경우를 대비하여 일괄 업데이트에 대한 메모를 보관합니다
- **팀과 조정** -- 대규모 일괄 변경을 수행할 때 다른 관리자에게 알립니다

## 일반적인 사용 사례

### 새 회원 업데이트

회원 수업 후 모든 참석자를 회원 상태로 업데이트합니다.

1. 수업에 참석한 사람을 검색합니다
2. 모두 선택합니다
3. **일괄 작업** → **회원 상태 설정** → **회원**을 사용합니다

### 소그룹 구성

새 소그룹에 여러 사람을 추가합니다.

1. 그룹에 포함하려는 사람을 검색합니다
2. 선택합니다
3. **일괄 작업** → **그룹에 추가**를 사용하고 소그룹을 선택합니다

### 데이터 정리

기혼 부부의 누락된 결혼 상태를 채웁니다.

1. 기혼인 사람을 필터링합니다 (가구 정보 사용)
2. 결혼 상태가 비어 있는 사람을 선택합니다
3. **일괄 작업** → **결혼 상태 설정** → **기혼** → **빈 항목만 업데이트**를 사용합니다
