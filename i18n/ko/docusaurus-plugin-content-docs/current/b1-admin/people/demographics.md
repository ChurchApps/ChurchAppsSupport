# 통계

<div class="article-intro">

**통계** 대시보드는 교인의 시각적 스냅샷을 제공합니다. 누가 있는지, 몇 명이 있는지, 교인이 연령대, 성별, 혼인 상태, 캠퍼스 전반에 어떻게 분포하는지를 확인할 수 있습니다. 모든 차트는 상호작용식입니다. 차트의 어떤 영역을 클릭해도 정확히 해당 교인을 필터링한 교인 검색으로 바로 이동합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교인이 명부에 존재해야 합니다. 시작하려면 [교인 추가](./adding-people.md) 또는 [데이터 가져오기](./importing-data.md)를 참고하세요.
- 캠퍼스 차트를 보려면 먼저 [설정 → 캠퍼스](../settings/campuses.md)에서 캠퍼스를 구성해야 합니다.

</div>

## 통계 열기

**교인** 페이지에서 페이지 헤더의 **통계** 버튼(막대 차트 아이콘)을 클릭합니다. 페이지에는 상단에 총 교인 수가 표시되고, 아래에 다음 차트가 표시됩니다.

## 차트

### 연령대 분포

각 연령대(예: 0-9, 10-17, 18-24, 65+)에 해당하는 사람의 수를 보여주는 그룹화된 막대 차트입니다. 각 막대는 **여성**, **남성**, **미지정** 세 시리즈로 나뉘어 각 연령대 내의 성별 분포를 한눈에 확인할 수 있습니다.

### 교인 상태

교회 교인을 교인 상태(예: 교인, 정기 참석자, 방문자)별로 나눈 도넛 차트입니다.

### 성별

명부의 여성, 남성, 미지정 레코드의 비율을 보여주는 도넛 차트입니다.

### 혼인 상태

혼인 상태 값(예: 기혼, 미혼, 사별)별 분포를 보여주는 도넛 차트입니다.

### 캠퍼스

교인이 교회의 캠퍼스 전체에 어떻게 분포하는지를 보여주는 도넛 차트입니다. 이 차트는 캠퍼스가 구성되어 있고 적어도 하나의 교인이 캠퍼스에 배정된 경우에만 나타납니다. 캠퍼스가 배정되지 않은 교인은 **미지정**으로 표시됩니다.

## 데이터 분석

도넛 차트의 어떤 부분이든 또는 연령 차트의 막대 부분을 클릭하여 **교인** 페이지로 이동하고, 해당 교인만 표시하도록 미리 필터링됩니다. 이를 사용하여 빠르게 연락처 목록을 만들거나, 대량 작업을 실행하거나, 세그먼트를 내보낼 수 있습니다. 예를 들어 교인 상태의 **방문자** 부분을 클릭하여 모든 방문자를 보고 워크플로우로 보낼 수 있습니다.

:::tip
필터링된 결과로 드릴다운한 후, 교인 검색에서 **목록으로 저장**을 사용하여 해당 필터를 나중에 사용할 수 있는 목록으로 저장합니다. [저장된 목록](./lists.md)을 참고하세요.
:::
