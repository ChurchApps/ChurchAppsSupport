# 교인 추가

<div class="article-intro">

교인 관리 섹션은 B1 Admin의 기반입니다 — 교회의 교인 데이터베이스입니다. 다른 모든 기능(그룹, 출석, 헌금, 양식)은 인물 기록과 연결됩니다. 이 가이드에서는 데이터베이스에 사람을 추가하고, 세부 정보를 편집하며, 가족 구성원을 가구로 연결하는 방법을 안내합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교인 관리 권한이 있는 활성 B1 Admin 계정이 필요합니다. 접근 권한이 확실하지 않은 경우 [역할 및 권한](roles-permissions.md)을 참조하세요.
- 소수 이상의 사람을 추가하는 경우, [CSV 가져오기](importing-data.md) 도구 사용을 고려하세요.

</div>

## 사람 추가하기

1. B1.church Admin 대시보드로 이동합니다.
2. 왼쪽 사이드바에서 **교인 관리**를 클릭합니다.
3. 오른쪽 상단의 **사람 추가** 버튼을 클릭합니다.
4. 이름, 성, 이메일 주소를 입력한 후 **추가**를 클릭합니다.

해당 사람의 프로필 페이지가 열리며, 추가 세부 정보를 입력할 수 있습니다.

:::tip
다른 교회 관리 시스템에서 마이그레이션하는 경우, [데이터 가져오기](importing-data.md) 기능을 사용하면 CSV 파일에서 전체 명부를 한꺼번에 가져올 수 있습니다 — 한 명씩 추가하는 것보다 훨씬 빠릅니다.
:::

## 세부 정보 편집

1. 해당 사람의 프로필 페이지에서 이름 옆의 **수정 연필** 아이콘을 클릭합니다.
2. 중간 이름, 회원 상태, 날짜, 주소, 전화번호 등 추가 정보를 입력합니다.
3. **저장**을 클릭하여 개인 정보를 저장합니다.

프로필에는 관련 정보를 위한 여러 탭이 포함되어 있습니다:

- **메모** — 해당 사람에 대한 메모 추가 (목회 돌봄, 후속 조치 등)
- **그룹** — [그룹 멤버십](../groups/group-members.md) 조회 및 관리
- **출석** — [출석 기록](../attendance/tracking-attendance.md) 조회
- **헌금** — [헌금 내역](../donations/recording-donations.md) 조회

## 양식 작업

사람의 프로필에서 직접 맞춤 양식을 작성할 수 있습니다. 이러한 양식은 [양식 만들기](../forms/creating-forms.md) 가이드에 따라 직접 작성할 수 있는 사용자 정의 양식입니다.

1. 해당 사람의 프로필에서 **양식** 드롭다운을 클릭하여 양식을 선택합니다.
2. **양식 추가**를 클릭하여 엽니다.
3. 양식 세부 정보를 입력하고 **저장**을 클릭합니다.

:::info
사람의 프로필에 연결된 양식은 **교인** 양식 유형을 사용합니다. 독립형 양식(예: 행사 등록)이 필요한 경우 양식 가이드의 [독립형 양식 옵션](../forms/creating-forms.md)을 참조하세요.
:::

## 가구 관리

가구를 통해 가족 구성원을 함께 연결할 수 있습니다. 이는 부모가 자녀를 한꺼번에 체크인할 수 있는 [체크인](../attendance/check-in.md)에서 특히 유용합니다.

1. 해당 사람의 프로필에서 가구 이름 옆의 **수정 연필** 아이콘을 클릭합니다.
2. 가구 편집기가 열립니다. 현재 사람의 **가구 역할**을 선택합니다 (예: 세대주, 배우자, 자녀).
3. **추가**를 클릭하여 다른 가구 구성원을 추가합니다.
4. 검색 상자에 사람의 이름을 입력하고 **검색**을 클릭합니다.
5. 검색 결과에 해당 사람이 나타나면 **선택**을 클릭합니다.
6. 가구 역할을 선택하고 **저장**을 클릭하여 가구 설정을 완료합니다.
