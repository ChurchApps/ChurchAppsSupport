# 사람

<div class="article-intro">

**사람** 섹션은 B1 Admin에서 교회 관리의 핵심입니다. 여기에서 교회 디렉토리를 유지 관리하고 회원을 검색하고 새 사람을 추가하며 프로필을 보고 편집하고 가구를 추적합니다. 소규모 교회든 대규모 교회든 사람 기록을 체계적으로 관리하는 것이 B1의 다른 모든 기능을 지원하는 기반입니다.

</div>

## 수행할 수 있는 작업

다음은 사람 섹션에서 사용할 수 있는 주요 기능의 개요입니다.

1. **디렉토리 검색** -- 빠른 검색 표시줄을 사용하여 이름으로 누구나 찾거나 고급 필터 및 AI 기반 자연어 검색을 활용하여 정확히 필요한 사람을 찾습니다. 자세한 내용은 [사람 검색](./searching-people.md) 및 [AI 검색](./ai-search.md)을 참조하세요.

2. **새 사람 추가** -- 디렉토리에 개인을 빠르게 추가하고 프로필 세부 정보, 연락처 정보 및 가구 연결을 입력합니다. 단계별 지침은 [사람 추가](./adding-people.md)를 참조하세요.

3. **사람 일괄 편집** -- 회원 상태, 결혼 상태, 성별, 수신 거부 기본 설정 또는 그룹 구성원 자격을 변경하여 여러 사람을 한 번에 업데이트합니다. 여러 개인에게 동일한 변경을 수행할 때 시간을 절약할 수 있습니다. [일괄 편집](./bulk-editing.md)을 참조하세요.

4. **데이터 가져오기 및 내보내기** -- CSV 파일에서 회원 데이터를 가져오거나 Breeze와 같은 다른 교회 관리 시스템에서 마이그레이션합니다. 언제든지 디렉토리를 CSV로 내보낼 수도 있습니다. [데이터 가져오기](./importing-data.md) 및 [데이터 내보내기](./exporting-data.md)를 참조하세요.

5. **프로필 관리** -- 연락처 정보, 가구 구성원, [그룹 구성원 자격](../groups/group-members.md), [출석 기록](../attendance/tracking-attendance.md), [헌금](../donations/recording-donations.md) 및 [맞춤 양식](../forms/creating-forms.md)을 포함한 각 사람의 세부 프로필을 보고 편집합니다.

6. **역할 및 권한 할당** -- 역할을 할당하여 팀의 누가 B1 Admin의 다른 부분에 액세스할 수 있는지 제어합니다. [역할 할당](./roles-permissions.md)을 참조하세요.

## 시작하기

사람 섹션에 액세스하려면 B1 Admin 대시보드의 왼쪽 사이드바에서 **사람**을 클릭합니다. 상단에 검색 표시줄이 있고 아래에 회원 목록이 있는 전체 교회 디렉토리가 표시됩니다.

:::tip
B1을 처음 설정하는 경우 [기존 회원 데이터 가져오기](./importing-data.md)부터 시작하세요. 그런 다음 필요에 따라 프로필을 검토하고 정리합니다.
:::
