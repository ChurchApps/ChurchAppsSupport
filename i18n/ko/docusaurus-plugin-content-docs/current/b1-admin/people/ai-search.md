# AI 검색

<div class="article-intro">

**AI 검색** 기능을 사용하면 기존의 검색 필드와 필터 대신 일상적인 자연어 질문으로 교회 명부를 검색할 수 있습니다. 특정 필터 기준으로 쿼리를 작성하는 대신, 찾고 있는 내용을 일상적인 언어로 간단히 설명하면 됩니다 -- 이를 통해 교인에 대한 복잡한 질문에 빠르게 답을 얻을 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교인 조회 권한이 있는 활성 B1 Admin 계정이 필요합니다. 접근 권한이 확실하지 않은 경우 [역할 및 권한](roles-permissions.md)을 참조하세요.
- 기본 [교인 검색](searching-people.md) 도구에 익숙하면 도움이 되지만 필수는 아닙니다.

</div>

## 작동 방식

AI 검색은 인공지능을 사용하여 질문을 해석하고 교인 데이터베이스에서 일치하는 결과를 반환합니다. 동료에게 질문하듯이 자연스럽게 물어볼 수 있습니다.

1. B1 Admin에서 **교인 관리** 페이지로 이동합니다.
2. 검색 컨트롤 아래의 **AI 검색** 옵션을 찾습니다.
3. 일상적인 언어로 질문을 입력합니다.
4. **Enter**를 누르거나 **검색**을 클릭하여 쿼리를 실행합니다.
5. 결과 테이블이 일치하는 사람들로 업데이트됩니다.

## 쿼리 예시

다음은 질문할 수 있는 유형의 예시입니다:

- "최근 3개월간 출석하지 않은 사람은 누구인가요?"
- "30세 미만의 모든 교인을 보여주세요"
- "청소년 그룹에 자녀가 있는 가정 목록"
- "올해 교회에 등록한 사람은 누구인가요?"
- "이메일 주소가 없는 모든 사람을 보여주세요"
- "75001 우편번호 지역에 사는 교인은 누구인가요?"

:::tip
질문을 가능한 한 구체적으로 작성하세요. 세부 정보를 많이 제공할수록 더 정확한 결과를 얻을 수 있습니다.
:::

## AI 검색 사용 시기

AI 검색은 다음과 같은 경우에 특히 유용합니다:

- 기존 필터로 설정하기 어려운 여러 기준을 조합해야 하는 경우.
- 어떤 필드로 검색해야 할지 정확히 모르는 경우.
- 보고서를 작성하지 않고도 교인에 대한 질문에 빠르게 답을 얻고 싶은 경우.

:::info
AI 검색은 기본 빠른 검색 및 열 필터와 함께 사용할 수 있습니다. 필요에 가장 적합한 방법을 사용하세요. 단순한 이름 검색에는 [빠른 검색](searching-people.md)이 가장 빠릅니다. 보다 복잡한 질문에는 AI 검색이 뛰어납니다.
:::

다른 검색 옵션에 대해서는 [교인 검색](./searching-people.md)을 참조하세요.
