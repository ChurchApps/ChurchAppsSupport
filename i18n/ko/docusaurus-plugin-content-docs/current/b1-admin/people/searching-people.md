# 교인 검색

<div class="article-intro">

**교인 관리** 페이지는 교회 명부를 검색 가능하고 정렬 가능한 테이블로 표시합니다. 교인 중 누구든 빠르게 찾고, 표시되는 정보를 맞춤 설정하며, 결과를 내보낼 수 있습니다. 효율적인 검색은 방문자 후속 조치, 연락처 목록 준비, 교인 기록 관리 등 일상적인 교회 행정 업무에 필수적입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교인 조회 권한이 있는 활성 B1 Admin 계정이 필요합니다. 접근 권한이 확실하지 않은 경우 [역할 및 권한](roles-permissions.md)을 참조하세요.
- 교회 명부에 사람이 등록되어 있어야 합니다. 아직 아무도 추가하지 않았다면 [교인 추가](adding-people.md) 또는 [데이터 가져오기](importing-data.md)를 참조하세요.

</div>

## 빠른 검색

교인 관리 페이지 상단의 검색 바를 사용하면 실시간으로 교인을 찾을 수 있습니다:

1. 교인 관리 페이지 상단의 **검색 상자**를 클릭합니다.
2. 이름, 이메일 또는 기타 키워드를 입력하기 시작합니다.
3. 입력하는 동안 결과가 자동으로 필터링됩니다 (매 키 입력마다 검색이 실행되지 않도록 약 0.5초의 짧은 지연이 있습니다).
4. 아래 테이블이 일치하는 결과만 표시하도록 업데이트됩니다.

:::tip
Enter를 누를 필요가 없습니다. 입력을 멈추면 검색이 자동으로 실행됩니다.
:::

## 결과 정렬

테이블의 열 헤더를 클릭하면 명부를 정렬할 수 있습니다:

1. **열 헤더** (예: **이름** 또는 **이메일**)를 클릭하면 해당 열로 정렬됩니다.
2. 같은 헤더를 다시 클릭하면 정렬 순서가 반전됩니다.

이를 통해 알파벳순, 나이순 또는 기타 표시된 열별로 쉽게 사람을 찾을 수 있습니다.

## 열 맞춤 설정

모든 정보를 한 번에 표시할 필요는 없습니다. 테이블에 나타나는 열을 선택할 수 있습니다:

1. 테이블 상단 근처의 **열 선택기 드롭다운**을 찾습니다.
2. 열을 체크하거나 해제하여 표시하거나 숨깁니다. 사용 가능한 열은 다음과 같습니다:
   - **사진**
   - **이름**
   - **이메일**
   - **전화번호**
   - **주소**
   - **생년월일**
   - **나이**
   - **성별**
   - **회원 상태**
3. 선택 사항이 즉시 테이블에 반영됩니다.

:::info
열 선택은 CSV로 내보낼 때 포함되는 내용에 영향을 줍니다. 필요한 데이터만 정확히 얻으려면 내보내기 전에 열을 맞춤 설정하세요.
:::

## 검색 결과 내보내기

현재 검색 결과를 언제든지 CSV 파일로 다운로드할 수 있습니다:

1. 원하는 검색이나 필터를 적용합니다.
2. 필요한 데이터를 포함하도록 열을 맞춤 설정합니다.
3. **내보내기** 버튼을 클릭합니다.
4. Excel, Google Sheets 또는 기타 스프레드시트 애플리케이션에서 바로 열 수 있는 CSV 파일이 컴퓨터에 다운로드됩니다.

내보내기에 대한 자세한 내용은 [데이터 내보내기](./exporting-data.md)를 참조하세요.

:::tip
최근 3개월간 출석하지 않은 사람을 찾는 것과 같은 고급 쿼리는 일상적인 언어 질문으로 검색할 수 있는 [AI 검색](./ai-search.md) 기능을 사용해 보세요.
:::
