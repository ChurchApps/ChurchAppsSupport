# 사람 검색

<div class="article-intro">

**사람** 페이지는 검색 가능하고 정렬 가능한 테이블로 교회 디렉토리를 표시합니다. 교단의 누구든지 빠르게 찾을 수 있고, 표시되는 정보를 사용자 정의하고, 결과를 내보낼 수 있습니다. 효율적인 검색은 방문자 추적, 연락처 목록 준비, 구성원 기록 관리와 같은 일상적인 교회 관리 작업에 필수적입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 사람을 볼 수 있는 권한이 있는 활성 B1 Admin 계정이 필요합니다. 액세스 수준이 확실하지 않으면 [역할 및 권한](roles-permissions.md)을 참조하세요.
- 교회 디렉토리에 사람이 있어야 합니다. 아직 누구를 추가하지 않았으면 [사람 추가](adding-people.md) 또는 [데이터 가져오기](importing-data.md)를 참조하세요.

</div>

## 빠른 검색

사람 페이지 상단의 검색 막대를 사용하면 실시간으로 구성원을 찾을 수 있습니다:

1. **사람** 페이지 상단의 **검색 상자**를 클릭합니다.
2. 이름, 이메일 또는 기타 키워드를 입력하기 시작합니다.
3. 입력하면 결과가 자동으로 필터링됩니다(검색이 모든 키 입력에서 작동하지 않도록 약 0.5초의 지연이 있습니다).
4. 아래의 테이블이 업데이트되어 일치하는 결과만 표시합니다.

:::tip
Enter 키를 누를 필요가 없습니다. 입력을 중단하면 검색이 자동으로 실행됩니다.
:::

## 결과 정렬

테이블의 열 헤더를 클릭하여 디렉토리를 정렬할 수 있습니다:

1. **열 헤더**(예: **이름** 또는 **이메일**)를 클릭하여 해당 열로 정렬합니다.
2. 같은 헤더를 다시 클릭하면 정렬 순서가 반대가 됩니다.

이를 통해 사람들을 알파벳순, 나이순 또는 다른 보이는 열별로 쉽게 찾을 수 있습니다.

## 열 사용자 정의

모든 정보가 한 번에 표시될 필요는 없습니다. 테이블에 표시되는 열을 선택할 수 있습니다:

1. 테이블 상단 근처의 **열 선택기 드롭다운**을 찾습니다.
2. 열을 확인 또는 선택 취소하여 표시하거나 숨깁니다. 사용 가능한 열은 다음을 포함합니다:
   - **사진**
   - **이름**
   - **이메일**
   - **전화**
   - **주소**
   - **생년월일**
   - **나이**
   - **성별**
   - **구성원 상태**
3. 테이블이 즉시 업데이트되어 선택 사항을 반영합니다.

:::info
열 선택이 CSV로 내보낼 때 포함되는 내용에 영향을 줍니다. 내보내기하기 전에 열을 사용자 정의하여 필요한 정확한 데이터를 얻습니다.
:::

## 페이지 나누기

디렉토리에 많은 기록이 있으면 결과가 여러 페이지로 나뉩니다. 테이블 하단의 **페이지 나누기 컨트롤**을 사용하여 페이지 간을 이동합니다. 현재 페이지와 전체 기록 수가 표시되어 항상 목록의 위치를 알 수 있습니다.

:::tip
한 번에 더 많은 결과를 보려면 검색을 좁혀서 목록을 줄이는 것이 좋습니다.
:::

## 검색 결과 내보내기

언제든지 현재 검색 결과를 CSV 파일로 다운로드할 수 있습니다:

1. 원하는 검색 또는 필터를 적용합니다.
2. 필요한 데이터를 포함하도록 열을 사용자 정의합니다.
3. **내보내기** 버튼을 클릭합니다.
4. CSV 파일이 컴퓨터에 다운로드되어 Excel, Google Sheets 또는 모든 스프레드시트 응용 프로그램에서 열 수 있습니다.

내보내기에 대한 자세한 내용은 [데이터 내보내기](./exporting-data.md)를 참조하세요.

:::tip
지난 3개월간 참석하지 않은 모든 사람을 찾는 것과 같은 고급 쿼리의 경우 [AI 검색](./ai-search.md) 기능을 시도하세요. 이 기능은 일반 언어 질문을 사용하여 검색할 수 있습니다.
:::
