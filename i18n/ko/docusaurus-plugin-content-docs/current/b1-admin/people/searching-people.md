# 교인 검색

<div class="article-intro">

**교인** 페이지는 교인 명부를 검색 가능하고 정렬 가능한 표 형태로 보여줍니다. 교회의 누구든지 빠르게 찾을 수 있으며, 표시할 정보를 사용자 정의하고 결과를 내보낼 수 있습니다. 효율적인 검색은 방문자 팔로우업, 연락처 목록 작성, 교인 기록 관리 등 일상적인 교회 행정 업무에 필수적입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교인을 볼 수 있는 권한이 있는 활성 B1 Admin 계정이 필요합니다. [역할 및 권한](roles-permissions.md)에서 접근 권한을 확인하세요.
- 교인 명부에 교인이 있어야 합니다. 아직 교인을 추가하지 않았다면 [교인 추가](adding-people.md) 또는 [데이터 가져오기](importing-data.md)를 참고하세요.

</div>

## 빠른 검색

교인 페이지 상단의 검색창을 사용하여 실시간으로 교인을 찾을 수 있습니다.

1. 교인 페이지 상단의 **검색창**을 클릭합니다.
2. 이름, 이메일 또는 다른 키워드를 입력하기 시작합니다.
3. 입력하면서 결과가 자동으로 필터링됩니다(키 입력마다 검색이 실행되지 않도록 약 0.5초의 지연이 있습니다).
4. 하단의 표에 일치하는 결과만 표시됩니다.

:::tip
Enter 키를 누를 필요가 없습니다. 입력을 멈춘 후 자동으로 검색이 실행됩니다.
:::

## 결과 정렬

표의 열 헤더를 클릭하여 명부를 정렬할 수 있습니다.

1. **열 헤더**(예: **이름**, **이메일**)를 클릭하여 해당 열로 정렬합니다.
2. 같은 헤더를 다시 클릭하면 정렬 순서가 뒤바뀝니다.

이를 통해 이름 순서, 연령 또는 다른 표시 열로 쉽게 교인을 찾을 수 있습니다.

## 열 사용자 정의

모든 정보를 한 번에 표시할 필요는 없습니다. 표에 표시할 열을 선택할 수 있습니다.

1. 표 상단 근처의 **열 선택 드롭다운**을 찾습니다.
2. 열을 선택하거나 선택 해제하여 표시 여부를 결정합니다. 사용 가능한 열:
   - **사진**
   - **이름**
   - **이메일**
   - **전화**
   - **주소**
   - **생일**
   - **나이**
   - **성별**
   - **교인 상태**
   - **캠퍼스**
3. 표가 즉시 업데이트되어 선택 사항이 반영됩니다.

:::info
열 선택 사항은 CSV로 내보낼 때도 적용됩니다. 내보내기 전에 열을 사용자 정의하여 필요한 데이터를 정확하게 가져올 수 있습니다.
:::

## 페이징

명부에 레코드가 많으면 결과가 여러 페이지로 나뉩니다. 표 하단의 **페이징 컨트롤**을 사용하여 페이지를 이동합니다. 현재 페이지와 전체 레코드 수가 표시되어 목록에서 현재 위치를 항상 알 수 있습니다.

:::tip
많은 레코드를 페이징하기보다는 검색을 좁혀서 목록 크기를 줄이는 것이 좋습니다.
:::

## 검색 결과 내보내기

언제든지 현재 검색 결과를 CSV 파일로 다운로드할 수 있습니다.

1. 원하는 검색이나 필터를 적용합니다.
2. 필요한 데이터가 포함되도록 열을 사용자 정의합니다.
3. **내보내기** 버튼을 클릭합니다.
4. CSV 파일이 컴퓨터에 다운로드되며, Excel, Google 스프레드시트 또는 다른 스프레드시트 응용프로그램에서 열 수 있습니다.

내보내기에 대한 자세한 내용은 [데이터 내보내기](./exporting-data.md)를 참고하세요.

:::tip
지난 3개월간 참석하지 않은 모든 사람 찾기 같은 고급 쿼리의 경우 [AI 검색](./ai-search.md) 기능을 사용하세요. 자연어 질문으로 검색할 수 있습니다.
:::

## 검색을 목록으로 저장

검색을 실행하면 교인 페이지 헤더에 **목록으로 저장** 버튼(북마크 아이콘)이 나타납니다. 이를 클릭하여 현재 쿼리를 이름과 선택적 카테고리로 저장하면 나중에 한 번의 클릭으로 다시 불러올 수 있습니다. [저장된 목록](./lists.md)에서 자세한 내용을 확인하세요.
