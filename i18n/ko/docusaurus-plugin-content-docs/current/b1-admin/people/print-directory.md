# 디렉토리 인쇄

<div class="article-intro">

디렉토리 인쇄 기능은 교회를 위해 형식이 지정된, 인쇄 가능한 구성원 디렉토리를 생성합니다. 구성원을 가구별로 구성하고 연락처 정보와 사진을 포함하며 표준 편지 크기 용지에 인쇄했을 때 전문적으로 보이도록 설계되었습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 사람을 볼 수 있는 권한이 있는 활성 B1 Admin 계정이 필요합니다.
- 디렉토리의 사람이 사진과 연락처 정보를 채워 넣으면 최고의 결과를 얻을 수 있습니다.

</div>

## 디렉토리 인쇄 열기

1. B1 Admin의 **사람**으로 이동합니다.
2. 사람 결과 카드 상단의 도구 모음에서 **인쇄 아이콘**(프린터 아이콘)을 클릭합니다. 내보내기 버튼 옆에 나타납니다.
3. 디렉토리 인쇄가 새 브라우저 탭에서 열리고 디렉토리 데이터를 로드하기 시작합니다.
4. 약 1.5초 후 브라우저의 인쇄 대화 상자가 자동으로 나타납니다.

:::tip
디렉토리가 별도 탭에서 열리므로 사람 페이지를 떠나지 않고 인쇄할 수 있습니다. 인쇄 대화 상자가 나타나면 탭이 자동으로 뒤로 이동합니다.
:::

## 포함되는 내용

인쇄 디렉토리는 교회의 모든 **활성 구성원**을 포함합니다. 구체적으로:

- **비활성** 또는 **방문자** 구성원 상태를 가진 사람은 자동으로 제외됩니다.
- 디렉토리에서 **옵트아웃**한 사람은 제외됩니다.
- 다른 모든 사람은 **가구**별로 그룹화됩니다.

## 디렉토리 레이아웃

인쇄된 디렉토리는 두 부분으로 구성됩니다:

### 표지

첫 번째 페이지는 다음을 포함합니다:
- 교회 이름
- "구성원 디렉토리" 제목
- 현재 연도
- 교회의 위치
- 포함된 총 가구 수

### 구성원 목록

남은 페이지에는 성씨별 알파벳순으로 나열된 가구가 있습니다. 각 가구 카드는 다음을 표시합니다:

- **사진** -- 각 가구 구성원의 원형 썸네일, 아래에 이름
- **가구 표시 이름** -- 예: "Smith 가족", "John & Jane Doe" 또는 개인의 경우 이름만
- **주소** -- 도로명, 도시, 주 및 우편 번호
- **전화** -- 집 또는 휴대폰
- **이메일** -- 주 이메일 주소
- **생일** -- 각 가구 구성원을 위한 개별 날짜로 형식화됨
- **기념일** -- 기록된 경우

가구는 **알파벳 섹션**(A, B, C, ...)으로 구성되며 각 글자 그룹 앞에 장식적인 섹션 헤더가 있습니다. 두 개의 가구 카드가 각 행에 나란히 나타납니다.

## 인쇄 설정

디렉토리는 0.5인치 여백이 있는 **편지 크기 용지**(8.5" × 11")에 맞게 형식이 지정됩니다. 브라우저 인쇄 대화 상자가 나타나면:

- 종이 크기를 **편지**로 설정된 상태로 두세요.
- 섹션 헤더를 색상 스타일로 인쇄하려면 브라우저의 인쇄 설정에서 **배경 그래픽** 활성화(때로는 "배경 인쇄"라고 불림)합니다.
- 세로 방향이 권장됩니다.

:::info
디렉토리는 페이지 나누기를 존중합니다. 가구 카드 및 섹션 헤더는 페이지 간에 분할되지 않습니다.
:::
