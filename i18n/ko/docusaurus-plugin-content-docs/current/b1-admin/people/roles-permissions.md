# 역할 할당

<div class="article-intro">

B1 Admin은 역할 기반 권한 시스템을 사용하여 팀의 각 사용자가 볼 수 있고 할 수 있는 것을 제어합니다. 역할을 할당하여 직원과 자원봉사자에게 필요한 영역에만 정확히 접근 권한을 부여할 수 있습니다 -- 그 이상은 아닙니다. 적절한 역할 관리는 교회 데이터를 안전하게 보호하면서도 팀이 효율적으로 일할 수 있도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin에서 **설정**을 관리할 수 있는 권한이 있는 **도메인 관리자** 접근 권한 또는 역할이 필요합니다.
- 역할을 할당하려는 사람이 이미 명부에 존재해야 합니다. 먼저 추가해야 하는 경우 [교인 추가](adding-people.md)를 참조하세요.

</div>

## 역할 이해하기

역할은 한 명 이상의 사용자에게 할당하는 일련의 권한입니다. 예를 들어, [헌금 기록](../donations/recording-donations.md)에 접근할 수 있는 "재정팀" 역할이나, [출석 기능](../attendance/check-in.md)에만 접근할 수 있는 "체크인 봉사자" 역할을 만들 수 있습니다.

각 역할은 B1 Admin의 특정 영역에 대한 접근을 제어하며, 여기에는 다음이 포함됩니다:

- **교인 관리** -- 교인 프로필 조회 및 편집
- **헌금** -- 헌금 및 재정 보고서 관리
- **출석** -- 출석 데이터 기록 및 조회
- **양식** -- [맞춤 양식](../forms/creating-forms.md) 생성 및 관리
- **그룹** -- [그룹 멤버십](../groups/group-members.md) 및 일정 관리
- **설정** -- 교회 전체 설정 구성

:::warning
**도메인 관리자**는 B1 Admin의 모든 영역에 대한 전체 접근 권한을 가집니다. 이 권한은 편집하거나 제한할 수 없습니다. 이 역할은 주요 관리자에게만 사용하세요.
:::

## 역할 조회 및 관리

1. 왼쪽 사이드바에서 **설정**을 클릭합니다.
2. 상단 내비게이션에서 **역할**을 클릭합니다.
3. 교회에 구성된 모든 역할 목록이 표시됩니다.
4. 역할을 클릭하면 구성원과 권한을 확인할 수 있습니다.

## 역할에 사용자 추가

1. **설정** 다음 **역할**로 이동합니다.
2. 사용자를 추가하려는 역할을 클릭합니다.
3. **구성원** 섹션에서 이름으로 사람을 검색합니다.
4. **추가**를 클릭하여 역할에 할당합니다.

해당 사용자가 다음에 로그인하면 해당 역할에 관련된 모든 권한을 갖게 됩니다.

## 역할 권한 편집

1. **설정** 다음 **역할**로 이동합니다.
2. 수정하려는 역할을 클릭합니다.
3. **권한** 섹션에서 역할이 접근할 수 있는 영역을 체크하거나 해제합니다.
4. **저장**을 클릭하여 변경 사항을 적용합니다.

:::tip
최소 권한 원칙을 따르세요 -- 각 역할에 진정으로 필요한 권한만 부여하세요. 이렇게 하면 데이터를 안전하게 보호하고 실수로 인한 변경 가능성을 줄일 수 있습니다.
:::

## 일반적인 역할 예시

- **사무실 직원** -- 교인 관리, 헌금, 출석, 양식에 접근
- **그룹 리더** -- [그룹](../groups/creating-groups.md)에만 접근
- **체크인 봉사자** -- [출석](../attendance/check-in.md)에만 접근
- **재정팀** -- [헌금](../donations/recording-donations.md) 및 보고서에 접근
