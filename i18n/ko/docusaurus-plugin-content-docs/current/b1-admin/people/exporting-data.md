# 데이터 내보내기

<div class="article-intro">

B1 Admin에서는 교회 데이터를 내보내어 스프레드시트에서 사용하거나, 팀과 공유하거나, 백업을 보관할 수 있습니다. 이름과 이메일의 간단한 목록이든 전체 데이터베이스 내보내기든, 필요에 맞는 옵션이 제공됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 내보내려는 데이터를 조회할 수 있는 권한이 있는 활성 B1 Admin 계정이 필요합니다. 접근 권한이 확실하지 않은 경우 [역할 및 권한](roles-permissions.md)을 참조하세요.
- 전체 데이터베이스 내보내기를 하려면 **설정** 영역에 대한 접근 권한이 필요합니다.

</div>

## 교인 페이지에서 내보내기

명부를 내보내는 가장 빠른 방법은 **교인 관리** 페이지에서 직접 내보내는 것입니다:

1. 왼쪽 사이드바에서 **교인 관리**로 이동합니다.
2. 검색 바나 필터를 사용하여 내보내려는 결과를 좁히거나 (모든 사람을 내보내려면 필터링하지 않은 상태로 둡니다). 필터링 팁은 [교인 검색](searching-people.md)을 참조하세요.
3. **열 선택기**를 사용하여 내보내기에 포함할 열을 선택합니다 (예: 이름, 이메일, 전화번호, 주소).
4. **내보내기** 버튼을 클릭합니다.
5. 테이블에 현재 표시된 데이터가 포함된 CSV 파일이 컴퓨터에 다운로드됩니다.

:::tip
내보내기 전에 열을 맞춤 설정하세요. CSV 파일에는 현재 표시된 열만 포함되므로, 파일을 나중에 편집할 필요 없이 필요에 맞게 내보내기를 조정할 수 있습니다.
:::

## 설정에서 전체 데이터 내보내기

교인뿐만 아니라 모든 B1 데이터를 완전히 내보내려면 설정의 내보내기 도구를 사용하세요:

1. 왼쪽 사이드바에서 **설정**을 클릭합니다.
2. 상단 내비게이션에서 **가져오기/내보내기**를 클릭합니다.
3. **데이터 소스** 드롭다운에서 **B1 Database**를 선택합니다.
4. 데이터 미리보기를 확인하고 **대상으로 계속**을 클릭합니다.
5. 내보내기 대상으로 **B1 Export Zip**을 선택합니다.
6. 모든 항목에 녹색 체크 표시가 나타날 때까지 내보내기 진행 상황을 모니터링합니다.
7. 내보내기 파일이 자동으로 다운로드됩니다. 다운로드 폴더에서 `B1Export` 파일을 찾으세요.
8. 파일의 압축을 풀면 Excel, Google Sheets 또는 Numbers에서 열 수 있는 개별 CSV 파일 (예: `people.csv`)에 접근할 수 있습니다.

:::info
전체 데이터 내보내기에는 교인, 그룹, 헌금, 출석 등 B1 데이터베이스의 모든 데이터가 포함됩니다. 이는 교회 기록의 정기적인 백업을 만드는 좋은 방법이기도 합니다.
:::

## 그룹 데이터 내보내기

개별 그룹의 교인 목록도 내보낼 수 있습니다. **그룹** 페이지에서 그룹을 열고 **다운로드 아이콘**을 클릭하면 해당 그룹의 교인 목록을 내보낼 수 있습니다. 자세한 내용은 [그룹 구성원](../groups/group-members.md)을 참조하세요.

:::info
내보낸 CSV 파일은 Microsoft Excel, Google Sheets, Apple Numbers를 포함한 모든 주요 스프레드시트 애플리케이션에서 사용할 수 있습니다.
:::
