# 다중 통화 지원

<div class="article-intro">

B1의 다중 통화 기능을 사용하면 교회에서 다양한 통화로 헌금을 받고 추적할 수 있습니다. 이것은 국제 회원, 선교사 또는 여러 국가에 캠퍼스가 있는 교회에 특히 유용합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 헌금을 관리할 수 있는 권한이 필요합니다. 자세한 내용은 [역할 및 권한](../people/roles-permissions.md)을 참조하세요.
- 다중 통화 거래를 지원하는 Stripe로 [온라인 헌금](./online-giving-setup.md)을 설정하세요.
- 여러 통화 처리에 대한 교회의 회계 요구 사항을 이해하세요.

</div>

## 다중 통화 활성화하기

다중 통화 지원은 이제 B1에서 기본적으로 활성화됩니다. 활성화되면:

- 회원들이 온라인으로 헌금할 때 현지 통화로 헌금할 수 있습니다
- 모든 통화로 헌금을 수동으로 기록할 수 있습니다
- 헌금 보고서는 원래 통화로 금액을 표시합니다
- Stripe는 온라인 헌금에 대해 자동으로 통화 변환을 처리합니다

## 지원되는 통화

시스템은 다음을 포함한 모든 주요 세계 통화를 지원합니다:

- **USD** -- 미국 달러
- **EUR** -- 유로
- **GBP** -- 영국 파운드
- **CAD** -- 캐나다 달러
- **AUD** -- 호주 달러
- **MXN** -- 멕시코 페소
- **BRL** -- 브라질 레알
- **INR** -- 인도 루피
- **CNY** -- 중국 위안
- **JPY** -- 일본 엔
- 그리고 더 많은 통화...

온라인 헌금에 사용 가능한 통화는 Stripe 계정의 지원 통화에 따라 다릅니다.

## 다른 통화로 헌금 기록하기

### 온라인 헌금

회원이 Stripe를 통해 온라인으로 헌금할 때:

1. 결제 시 선호하는 통화를 선택합니다
2. Stripe가 해당 통화로 결제를 처리합니다
3. 헌금은 원래 통화 금액으로 B1에 기록됩니다
4. Stripe는 계정의 기본 통화로 필요한 통화 변환을 자동으로 처리합니다

### 수동 입력

다른 통화로 현금 또는 수표 헌금을 기록하려면:

1. B1 Admin에서 **Donations**로 이동합니다
2. **Add Donation**을 클릭합니다
3. 통화 드롭다운에서 통화를 선택합니다
4. 해당 통화로 금액을 입력합니다
5. 나머지 헌금 세부 정보를 완료합니다
6. **Save**를 클릭합니다

## 다중 통화 헌금 보기

### 헌금 보고서

헌금 보고서는 원래 통화로 금액을 표시합니다:

- 개별 헌금 기록에는 통화 코드가 표시됩니다(예: "$100.00 USD")
- 합계는 통화별로 계산됩니다
- 특정 통화로 필터링할 수 있습니다

### 헌금 명세서

헌금 명세서를 생성할 때:

- 각 헌금은 원래 통화와 함께 표시됩니다
- 합계는 통화별로 분류됩니다
- 회원들은 각 통화로 정확히 무엇을 헌금했는지 볼 수 있습니다

## Stripe 통합

온라인 헌금의 경우 Stripe가 다중 통화 거래를 처리합니다:

- **자동 변환** -- Stripe는 통화를 계정의 기본 통화로 변환합니다
- **환율** -- Stripe는 현재 시장 환율을 사용합니다
- **수수료** -- 통화 변환 시 추가 Stripe 수수료가 발생할 수 있습니다
- **지급 통화** -- 자금은 계정의 기본 통화로 입금됩니다

:::info
현재 변환율 및 다중 통화 거래와 관련된 수수료를 보려면 Stripe 대시보드를 확인하세요.
:::

## 회계 고려 사항

여러 통화를 사용할 때:

- **기록 보관** -- 정확한 보고를 위해 원래 헌금 금액과 통화를 추적하세요
- **환율** -- Stripe의 변환율은 은행의 환율과 다를 수 있습니다
- **세금 영수증** -- 세금 목적으로 다양한 통화의 헌금을 보고하는 방법에 대해 회계사와 상담하세요
- **기금 할당** -- 통화에 관계없이 특정 기금에 헌금을 할당할 수 있습니다

## 모범 사례

- **기본 통화** -- 대부분의 거래에 대해 교회의 주요 통화를 기본값으로 설정하세요
- **명확한 의사소통** -- 결제 프로세스 중에 기부자에게 어떤 통화로 헌금하는지 알려주세요
- **일관된 보고** -- 원래 통화로 보고할지 요약을 위해 단일 통화로 변환할지 결정하세요
- **정기 조정** -- 통화 변환을 고려하여 Stripe 지급금을 헌금 기록과 조정하세요

## 제한 사항

- 통화 변환은 온라인 헌금에 대해서만 Stripe에서 처리됩니다
- 수동 헌금은 자동 변환 없이 입력된 대로 기록됩니다
- 과거 보고서는 원래 통화로 헌금을 표시합니다
- 총 계산은 통화별로 수행되며 통화 간에 수행되지 않습니다
