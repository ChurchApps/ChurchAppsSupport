# 기금 관리

<div class="article-intro">

기금을 사용하면 목적별로 헌금을 분류하여 교회가 각 사역 분야에 얼마를 받는지 추적할 수 있습니다. 일반 기금, 건축 기금, 선교 기금, 청소년 사역 등이 일반적인 기금이며, 필요한 만큼 많이 만들 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin의 **헌금** 섹션에 접근 권한이 필요합니다
- 교회에 필요한 헌금 카테고리와 각 기금의 세금 공제 대상 여부를 결정하세요

</div>

## 기금 페이지

**헌금 > 기금**으로 이동하면 모든 기금 목록을 볼 수 있습니다. 각 행에는 기금 이름과 **세금 공제** 상태가 표시되며, 이는 해당 기금에 대한 헌금이 연말 [헌금 명세서](giving-statements.md)에 표시될지 여부를 결정합니다.

## 새 기금 만들기

1. 페이지 상단의 **기금 추가**를 클릭합니다.
2. 기금 이름을 입력합니다 (예: "건축 기금").
3. 이 기금에 대한 헌금이 헌금 명세서에 포함되어야 하는 경우 **세금 공제** 체크 박스를 선택합니다.
4. **저장**을 클릭합니다.

새 기금이 목록에 즉시 나타나며 헌금 입력 시 사용할 수 있습니다.

## 기금 편집 및 삭제

- 기금을 편집하려면 옆의 **편집** 버튼을 클릭합니다. 이름을 변경하거나 세금 공제 설정을 전환할 수 있습니다.
- 기금을 삭제하려면 편집 양식을 열고 **삭제**를 클릭합니다.

:::warning
기금을 삭제해도 이전에 해당 기금에 할당된 헌금은 제거되지 않습니다. 해당 헌금 기록은 시스템에 남아 있지만, 새 헌금에는 더 이상 이 기금을 사용할 수 없습니다.
:::

## 기금 상세 정보 보기

**기금 이름**을 클릭하면 상세 페이지가 열립니다. 이 페이지에는 다음이 표시됩니다:

- **헌금 내역** -- 이 기금에 대한 모든 헌금 목록
- **날짜 범위 필터** -- 결과를 특정 기간으로 좁히기
- **요약 통계** -- 총 헌금 수, 고유 기부자 수, 총 금액

이 보기는 특정 헌금 이니셔티브에 대해 보고하거나 모금 목표 진행 상황을 확인해야 할 때 유용합니다.

## 정렬 및 내보내기

- 열 헤더를 사용하여 이름별로 기금을 정렬합니다.
- **CSV로 내보내기**를 클릭하여 기금 목록을 스프레드시트로 다운로드합니다.

:::tip
[헌금 기록](recording-donations.md)을 시작하기 전에 기금을 설정하세요. 모든 기금을 미리 설정해 두면 처음부터 헌금이 올바르게 분류됩니다.
:::

## 다음 단계

기금이 설정되면 일괄 처리에서 [헌금을 기록](recording-donations.md)할 때 기금을 할당할 수 있습니다. [헌금 명세서](giving-statements.md)를 생성하기 전에 적절한 기금이 세금 공제로 표시되어 있는지 확인하세요.
