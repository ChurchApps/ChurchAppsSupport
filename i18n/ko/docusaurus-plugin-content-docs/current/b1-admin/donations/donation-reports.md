# 헌금 보고서

<div class="article-intro">

B1 Admin은 교회의 헌금 데이터를 보고 분석할 수 있는 여러 방법을 제공합니다. 헌금 요약 페이지는 차트와 필터가 포함된 시각적 개요를 제공하며, 보고서 섹션은 더 상세한 헌금 요약 보고서를 제공합니다. 이러한 도구를 사용하여 헌금 추세를 추적하고, 이사회 회의를 준비하거나, 기록을 정산할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 헌금이 올바르게 분류되도록 [기금](funds.md)이 정확히 설정되어 있는지 확인하세요

## 헌금 요약 페이지

**요약** 페이지는 사이드바에서 **헌금**을 클릭하면 가장 먼저 표시되는 화면입니다. 헌금 활동에 대한 개략적인 보기를 제공합니다.

1. 사이드바에서 **헌금**으로 이동하여 요약 페이지를 엽니다.
2. **날짜 범위 필터**를 사용하여 검토하려는 기간을 선택합니다. 이전 날짜를 위에, 최근 날짜를 아래에 설정합니다.
3. 페이지에 주간 헌금 차트가 표시되어 추세를 한눈에 확인할 수 있습니다.
4. **다운로드**를 클릭하면 총 헌금 금액, 헌금 주, 해당 기금이 포함된 CSV 파일을 내보낼 수 있습니다.

:::info
요약 페이지는 집계 헌금 데이터를 보여줍니다. 개별 기부자 이름은 포함되지 않습니다. 기부자 수준의 세부 정보를 보려면 [일괄 처리](batches.md) 페이지를 사용하세요.
:::

## 기부자별 세부 정보 보기

누가 얼마를 어떤 기금에 헌금했는지의 세부 내역을 확인하려면:

1. **헌금 > 일괄 처리**로 이동합니다.
2. **일괄 처리 이름**을 클릭하여 엽니다.
3. 일괄 처리 상세 페이지에 기부자 이름, 금액, 기금, 날짜, 결제 방법과 함께 각 헌금이 나열됩니다.
4. **기부자 이름**을 클릭하면 헌금 횟수와 각 금액의 내역을 확인할 수 있습니다.
5. **헌금 ID**를 클릭하면 해당 개별 헌금의 전체 세부 정보가 표시되는 사이드 패널이 열립니다.
6. **다운로드**를 클릭하면 해당 일괄 처리의 모든 기부자 및 헌금 정보가 포함된 CSV를 내보낼 수 있습니다.

## 헌금 요약 보고서

B1 Admin에는 보고서 섹션에 **헌금 요약** 보고서도 포함되어 있습니다:

1. 사이드바에서 **보고서**를 클릭합니다.
2. **헌금 요약** 보고서를 선택합니다.
3. 필터 (날짜 범위, 기금, 캠퍼스 등)를 선택하고 보고서를 실행합니다.

## 데이터 내보내기

여러 곳에서 헌금 데이터를 내보낼 수 있습니다:

- **요약 페이지** -- 기금별 주간 헌금 합계의 CSV 다운로드
- **일괄 처리 상세 페이지** -- 기부자 세부 정보가 포함된 개별 헌금의 CSV 다운로드
- **기금 상세 페이지** -- 특정 기금의 헌금 내역 다운로드

:::tip
연말 보고를 위해 요약 페이지 내보내기와 [헌금 명세서](giving-statements.md) 도구를 결합하면 집계 추세와 개별 기부자 명세서를 모두 얻을 수 있습니다.
:::
