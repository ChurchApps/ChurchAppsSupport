# 기부금 보고서

<div class="article-intro">

B1 Admin은 교회의 기부금 데이터를 보고 분석할 수 있는 여러 가지 방법을 제공합니다. 기부금 요약 페이지는 차트와 필터를 사용한 시각적 개요를 제공하고, 보고서 섹션은 보다 자세한 기부금 요약 보고서를 제공합니다. 이 도구를 사용하여 기부금 추세를 추적하고, 이사회 회의를 준비하거나, 기록을 조정하세요.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [배치에 기부금이 기록](recording-donations.md)되거나 [Stripe에서 가져오기](stripe-import.md)되었는지 확인합니다
- [기금](funds.md)이 올바르게 설정되어 기부금이 제대로 분류되는지 확인합니다

</div>

## 기부금 대시보드

**기부금 대시보드**는 사이드바에서 **기부금**을 클릭할 때 처음 보는 것입니다. 주요 성과 지표를 포함한 기부금 활동의 높은 수준의 보기를 제공합니다.

1. 사이드바에서 **기부금**으로 이동하여 대시보드를 엽니다.
2. 상단에서 네 개의 **KPI 카드**가 기부금 지표를 한눈에 표시합니다:
   - **총 기부금** -- 선택한 기간에 기부한 총액입니다.
   - **평균 기부금** -- 평균 기부금입니다.
   - **고유 기부자** -- 기부한 서로 다른 사람의 수입니다.
   - **총 기부건** -- 개별 기부건의 총 수입니다.
3. **기간 토글**을 사용하여 **주간**, **월간** 및 **분기별** 보기 사이를 전환합니다.
4. KPI 아래에 차트가 선택한 기간의 기부금 추세를 표시합니다.
5. **다운로드**를 클릭하여 기부금 총액이 있는 CSV 파일을 내보냅니다.

## 기부금 요약 페이지

**요약** 페이지는 더 자세한 집계 기부금 데이터를 제공합니다.

1. 사이드바에서 **기부금**으로 이동하여 요약 페이지를 엽니다.
2. **날짜 범위 필터**를 사용하여 검토할 시간 기간을 선택합니다. 이전 날짜를 맨 위에, 최근 날짜를 맨 아래에 설정합니다.
3. 페이지는 주간 기부금 차트를 표시하므로 한눈에 추세를 볼 수 있습니다.
4. **다운로드**를 클릭하여 기부한 총액, 기부한 주, 기부한 기금이 있는 CSV 파일을 내보냅니다.

:::info
요약 페이지는 집계 기부금 데이터를 보여줍니다. 개별 기부자 이름은 포함되지 않습니다. 기부자 수준 세부 정보는 [배치](batches.md) 페이지를 사용하세요.
:::

## 기부자 수준 세부 정보 보기

누가 기부했는지, 얼마를 기부했는지, 어느 기금에 기부했는지 분석:

1. **기부금 > 배치**로 이동합니다.
2. **배치 이름**을 클릭하여 엽니다.
3. 배치 세부 페이지에는 각 기부건과 기부자의 이름, 금액, 기금, 날짜, 결제 방법이 나열됩니다.
4. **기부자의 이름**을 클릭하여 기부 횟수 및 각 기부금의 금액 분석을 확인합니다.
5. **기부 ID**를 클릭하여 해당 개별 기부건의 전체 세부 정보가 있는 측면 패널을 엽니다.
6. **다운로드**를 클릭하여 해당 배치의 모든 기부자 및 기부 정보가 있는 CSV를 내보냅니다.

## 기부금 요약 보고서

B1 Admin은 보고서 섹션에 **기부금 요약** 보고서도 포함합니다:

1. 사이드바에서 **보고서**를 클릭합니다.
2. **기부금 요약** 보고서를 선택합니다.
3. 필터(날짜 범위, 기금, 캠퍼스 등)를 선택하고 보고서를 실행합니다.

## 데이터 내보내기

여러 위치에서 기부금 데이터를 내보낼 수 있습니다:

- **요약 페이지** -- 기금별 주간 기부금 총액 CSV 다운로드
- **배치 세부 페이지** -- 기부자 세부 정보가 있는 개별 기부 CSV 다운로드
- **기금 세부 페이지** -- 특정 기금의 기부 기록 다운로드

:::tip
연말 보고의 경우 요약 페이지 내보내기를 [기부금 명세서](giving-statements.md) 도구와 결합하여 집계 추세와 개별 기부자 명세서를 모두 얻습니다.
:::
