# 헌금 명세서

<div class="article-intro">

매년 말 기부자들은 세금 신고를 위해 세금 공제 대상 헌금의 요약이 필요합니다. B1 Admin을 사용하면 모든 기부자의 명세서를 한 번에 쉽게 생성할 수 있어, 수동 작업에 드는 수많은 시간을 절약할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [기금](funds.md)이 **세금 공제**로 올바르게 표시되어 있는지 확인하세요 -- 세금 공제 기금에 대한 헌금만 명세서에 표시됩니다
- 모든 헌금이 [기록](recording-donations.md)되었고, 온라인 거래가 [Stripe에서 가져오기](stripe-import.md)되었는지 확인하세요

</div>

## 헌금 명세서 접근

1. **B1 Admin**에서 사이드바의 **헌금**을 클릭합니다.
2. **명세서**를 클릭합니다.

## 명세서 생성

1. 페이지 상단의 드롭다운에서 **연도**를 선택합니다. 현재 연도 또는 이전 5개 연도 중에서 선택할 수 있습니다.
2. 페이지에 해당 연도의 요약 통계가 표시됩니다:
   - **총 기부자** -- 헌금한 사람 수
   - **총 헌금** -- 개별 헌금 기록 수
   - **총 금액** -- 모든 헌금의 합산 금액

## 명세서 다운로드

기부자에게 명세서를 전달하는 두 가지 옵션이 있습니다:

### CSV 파일로 다운로드

**ZIP 다운로드**를 클릭하면 각 기부자별 개별 CSV 파일이 포함된 ZIP 파일이 다운로드됩니다. 명세서를 개별적으로 이메일로 보내거나 다른 시스템으로 가져오는 경우 유용합니다.

### 모든 명세서 인쇄

**모두 인쇄**를 클릭하면 브라우저에서 모든 기부자의 명세서가 인쇄 가능한 보기로 열립니다. 거기서 브라우저의 인쇄 기능을 사용하여 프린터로 전송합니다. 각 명세서는 새 페이지에서 시작되므로 접어서 우편으로 보낼 준비가 됩니다.

:::tip
1월 초에 기록이 정확할 때 명세서를 조기에 실행하세요. 명세서를 생성하기 전에 기금이 세금 공제로 올바르게 표시되어 있는지 다시 확인하세요 -- 세금 공제 기금에 대한 헌금만 포함됩니다.
:::

:::info
헌금 명세서에는 **세금 공제** 설정이 활성화된 기금에 할당된 헌금만 포함됩니다. 기금이 세금 공제로 표시되지 않은 경우, 해당 헌금은 명세서에 나타나지 않습니다. 이 설정은 [기금](funds.md) 페이지에서 관리할 수 있습니다.
:::
