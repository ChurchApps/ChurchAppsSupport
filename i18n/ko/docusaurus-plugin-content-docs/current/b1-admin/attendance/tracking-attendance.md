# 출석 추적

<div class="article-intro">

캠퍼스, 예배 시간, 그룹이 구성되면 B1 Admin에서 출석 데이터를 검토하고 추세를 파악하기 쉽습니다. 출석 페이지는 두 가지 보고 보기를 제공합니다 -- 교회 전체 추세를 위한 **출석** 탭과 그룹별 세부 정보를 위한 **그룹** 탭. 이러한 도구를 사용하여 성장 패턴을 파악하고, 감소하는 참여도를 식별하며, 데이터 기반의 의사 결정을 내릴 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 최소 하나의 캠퍼스와 예배 시간으로 출석 구조가 설정되어 있어야 합니다. 아직 설정하지 않았다면 [출석 설정](setup.md)을 참조하세요.
- 보고서에 결과가 표시되려면 출석 데이터가 기록되어 있어야 합니다. 데이터는 수동 입력 또는 [셀프 체크인](check-in.md)을 통해 수집할 수 있습니다.

</div>

## 출석 추세 보기

1. **B1 Admin**을 열고 사이드바에서 **출석**을 클릭합니다.
2. **출석** 탭을 선택합니다.
3. 탭이 열리면 기본 날짜 범위에 대한 출석 현황이 자동으로 보고서에 표시됩니다.

## 데이터 필터링

페이지 상단의 필터를 사용하여 결과를 좁힐 수 있습니다:

- **날짜 범위** -- 특정 기간에 집중하기 위해 시작일과 종료일을 선택합니다.
- **캠퍼스** -- 특정 장소의 출석만 보려면 캠퍼스를 선택합니다.
- **예배 시간** -- 특정 모임을 상세히 보려면 예배 시간을 선택합니다.

필터를 변경하면 차트와 데이터가 즉시 업데이트되므로 다른 기간이나 장소를 빠르게 비교할 수 있습니다.

:::info
출석 탭을 열 때마다 보고서가 자동 실행되므로, 새로고침 버튼을 클릭할 필요 없이 항상 최신 수치를 확인할 수 있습니다.
:::

## 그룹 출석

**그룹** 탭은 개별 그룹별 출석을 보여줍니다. 전체 예배 수치보다 특정 수업, 사역팀 또는 소그룹을 모니터링하고 싶을 때 유용합니다.

1. **그룹** 탭을 선택합니다.
2. 목록에서 그룹을 선택하여 출석 기록을 확인합니다.
3. 날짜 범위 필터를 사용하여 보고 기간을 조정합니다.

:::tip
그룹 출석은 [소그룹](../groups/creating-groups.md) 리더가 시간 경과에 따른 그룹 내 참여도를 추적하는 데 특히 유용합니다.
:::

## 출석 데이터 활용 팁

- 계절별 패턴을 조기에 파악하기 위해 매월 추세를 검토하세요.
- 캠퍼스별 데이터를 비교하여 어떤 장소가 성장하고 있는지 파악하세요.
- 그룹별 보고서를 사용하여 출석이 감소하는 [그룹](../groups/group-members.md)에 대해 후속 조치를 취하세요.
- 출석 인사이트를 [AI 검색](../people/ai-search.md) 도구와 결합하여 최근 출석하지 않은 사람을 찾으세요.
