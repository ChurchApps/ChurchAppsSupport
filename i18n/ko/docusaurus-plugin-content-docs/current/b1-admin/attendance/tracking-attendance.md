# 출석 추적

<div class="article-intro">

캠퍼스, 예배 시간 및 그룹이 구성되면 B1 Admin을 통해 출석 데이터를 쉽게 검토하고 추세를 파악할 수 있습니다. 출석 페이지는 교회 전체 추세를 위한 **출석** 탭과 그룹 수준 세부 정보를 위한 **그룹** 탭이라는 두 개의 보고 보기를 제공합니다. 이 도구를 사용하여 성장 패턴을 이해하고, 감소하는 참여를 식별하고, 교회를 위한 데이터 기반 결정을 내리세요.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 출석 구조는 최소한 하나의 캠퍼스와 예배 시간으로 설정되어야 합니다. 아직 이를 수행하지 않았으면 [출석 설정](setup.md)을 참조하세요.
- 보고서가 결과를 표시하려면 먼저 출석 데이터를 기록해야 합니다. 데이터는 [수동 항목](recording-attendance.md) 또는 [자체 체크인](check-in.md)에서 나올 수 있습니다.

</div>

## 출석 추세 보기

1. **B1 Admin**을 열고 사이드바에서 **출석**을 클릭합니다.
2. **출석** 탭을 선택합니다.
3. 보고서는 탭이 열릴 때 자동으로 실행되어 기본 날짜 범위 동안의 출석을 보여줍니다.

## 데이터 필터링

페이지 상단의 필터를 사용하여 결과를 좁히세요:

- **날짜 범위** -- 시작 및 종료 날짜를 선택하여 특정 기간에 집중합니다.
- **캠퍼스** -- 캠퍼스를 선택하여 해당 위치에 대한 출석만 확인합니다.
- **예배 시간** -- 특정 모임으로 드릴다운하려면 예배 시간을 선택합니다.

필터를 변경하면 차트와 데이터가 즉시 업데이트되므로 다른 시간 또는 위치를 빠르게 비교할 수 있습니다.

:::info
보고서는 출석 탭을 열 때마다 자동으로 실행되므로 새로고침 버튼을 클릭할 필요 없이 항상 최신 숫자를 볼 수 있습니다.
:::

## 그룹 출석

**그룹** 탭은 개별 그룹별로 분류된 출석을 보여줍니다. 이는 전체 예배 숫자를 보기보다는 특정 클래스, 사역팀 또는 소그룹을 모니터링하려는 경우에 유용합니다.

1. **그룹** 탭을 선택합니다.
2. 목록에서 그룹을 선택하여 출석 기록을 확인합니다.
3. 날짜 범위 필터를 사용하여 보고 기간을 조정합니다.

:::tip
그룹 출석은 특히 시간 경과에 따라 그룹 내 참여를 추적하려는 [소그룹](../groups/creating-groups.md) 지도자에게 유용합니다.
:::

## 출석 데이터 사용 팁

- 월별로 추세를 검토하여 계절 패턴을 조기에 파악합니다.
- 캠퍼스 수준 데이터를 비교하여 어떤 위치가 성장하고 있는지 이해합니다.
- 그룹 수준 보고서를 사용하여 [그룹](../groups/group-members.md)이 출석 감소를 보이는지 팔로우합니다.
- 출석 통찰력을 [AI 검색](../people/ai-search.md) 도구와 결합하여 최근에 참석하지 않은 사람을 찾습니다.

## 관련 페이지

- [출석 기록](recording-attendance.md) -- 그룹 세션의 출석을 수동으로 입력합니다.
- [체크인](check-in.md) -- 자체 체크인을 설정하여 출석이 자동으로 기록되도록 합니다.
