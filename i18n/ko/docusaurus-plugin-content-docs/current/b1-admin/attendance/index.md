# 출석 현황

<div class="article-intro">

B1 Admin의 출석 현황 섹션에서는 교회에 나오는 사람들과 그룹이 시간 경과에 따라 어떻게 성장하고 있는지에 대한 완벽한 개요를 제공합니다. 캠퍼스와 예배 시간 구성부터 동향 검토 및 자동 체크인 설정까지, 출석을 추적하고 이해하는 데 필요한 모든 것이 이 페이지에서 관리됩니다.

</div>

## 페이지 개요

출석 현황 페이지를 열면 교회의 출석 설정에 대한 주요 통계가 포함된 헤더가 표시됩니다:

- **캠퍼스** -- 구성한 실제 장소의 수
- **예배 시간** -- 모든 캠퍼스의 총 예배 수
- **예정된 그룹** -- 특정 예배 시간에 배정된 그룹
- **예정되지 않은 그룹** -- 예배 시간과 독립적으로 출석을 추적하는 그룹

헤더 아래에는 세 개의 탭이 출석 관리 도구를 구성합니다.

## 탭

### 설정

**설정** 탭은 출석 추적의 기본 요소를 구성하는 곳입니다. 여기서 캠퍼스를 정의하고, 각 캠퍼스에 예배 시간을 추가하며, [그룹](../groups/creating-groups.md)을 해당 예배 시간에 할당합니다. 이 작업은 한 번만 수행하면 되지만 일정이 변경될 때마다 업데이트할 수 있습니다. 전체 설명서는 [출석 설정](setup.md)을 참조하세요.

### 출석

**출석** 탭에는 시간 경과에 따른 출석 동향이 표시됩니다. 필터를 사용하여 날짜 범위, 캠퍼스 또는 예배 시간별로 결과를 좁혀서 데이터의 패턴을 빠르게 파악할 수 있습니다. 보고서 및 필터 사용에 대한 자세한 내용은 [출석 추적](tracking-attendance.md)을 참조하세요.

### 그룹

**그룹** 탭에서는 개별 그룹별로 출석을 분류합니다. 교회 전체 수치가 아닌 특정 소그룹, 클래스 또는 사역 팀의 상황을 보고 싶을 때 유용합니다.
