# 출석

<div class="article-intro">

B1 Admin의 출석 섹션은 교회에 누가 참석하고 있으며 그룹이 시간이 지남에 따라 어떻게 성장하고 있는지에 대한 전체적인 뷰를 제공합니다. 캠퍼스와 예배 시간 설정부터 트렌드 확인 및 셀프 체크인 설정까지, 출석을 추적하고 이해하는 데 필요한 모든 것을 이 페이지에서 관리합니다.

</div>

## 페이지 개요

출석 페이지를 열면 교회 출석 설정에 대한 주요 통계가 포함된 헤더가 표시됩니다:

- **캠퍼스** -- 설정된 물리적 장소의 수
- **예배 시간** -- 모든 캠퍼스에 걸친 총 예배 수
- **예정된 그룹** -- 특정 예배 시간에 배정된 그룹
- **비예정 그룹** -- 예배 시간과 독립적으로 출석을 추적하는 그룹

헤더 아래에 세 개의 탭이 출석 도구를 구성합니다.

## 탭

### 설정

**설정** 탭은 출석 추적의 구성 요소를 설정하는 곳입니다. 여기서 캠퍼스를 정의하고, 각 캠퍼스에 예배 시간을 추가하며, [그룹](../groups/creating-groups.md)을 해당 예배 시간에 배정합니다. 이 작업은 한 번만 하면 되지만, 일정이 변경될 때마다 업데이트할 수 있습니다. 전체 안내는 [출석 설정](setup.md)을 참조하세요.

### 출석

**출석** 탭은 시간에 따른 출석 추세를 표시합니다. 필터를 사용하여 날짜 범위, 캠퍼스 또는 예배 시간별로 결과를 좁히고 데이터에서 패턴을 빠르게 파악하세요. 보고서와 필터 사용에 대한 자세한 내용은 [출석 추적](tracking-attendance.md)을 참조하세요.

### 그룹

**그룹** 탭은 개별 그룹별로 출석을 분류합니다. 교회 전체 수치가 아닌 특정 소그룹, 반, 사역팀의 현황을 확인하고 싶을 때 유용합니다.
