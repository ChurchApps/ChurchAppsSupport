# 출석

<div class="article-intro">

B1 Admin의 출석 섹션은 교회에 출석하는 사람과 시간 경과에 따른 그룹 성장 추이를 종합적으로 보여줍니다. 캠퍼스 및 예배 시간 구성부터 추세 검토 및 셀프 체크인 설정까지, 출석을 추적하고 파악하는 데 필요한 모든 것을 이 페이지에서 관리합니다.

</div>

## 페이지 개요

출석 페이지를 열면 교회 출석 설정에 대한 주요 통계가 표시된 헤더를 볼 수 있습니다:

- **캠퍼스** -- 구성한 물리적 장소의 수
- **예배 시간** -- 모든 캠퍼스의 총 예배 수
- **예정된 그룹** -- 특정 예배 시간에 배정된 그룹
- **미예정 그룹** -- 예배 시간과 별도로 독립적으로 출석을 추적하는 그룹

헤더 아래에는 세 개의 탭으로 출석 도구가 정리되어 있습니다.

## 탭

### 설정

**설정** 탭에서는 출석 추적의 기본 구성 요소를 설정합니다. 여기서 캠퍼스를 정의하고, 각 캠퍼스에 예배 시간을 추가하며, 해당 예배 시간에 [그룹](../groups/creating-groups.md)을 배정합니다. 이 작업은 한 번만 하면 되지만, 일정이 변경될 때마다 업데이트할 수 있습니다. 전체 안내는 [출석 설정](setup.md)을 참조하세요.

### 출석

**출석** 탭은 시간 경과에 따른 출석 추세를 표시합니다. 필터를 사용하여 날짜 범위, 캠퍼스 또는 예배 시간별로 결과를 좁히고 데이터의 패턴을 빠르게 파악할 수 있습니다. 보고서와 필터 사용에 대한 자세한 내용은 [출석 추적](tracking-attendance.md)을 참조하세요.

### 그룹

**그룹** 탭은 개별 그룹별 출석을 세분화합니다. 교회 전체 수치보다 특정 소그룹, 수업 또는 사역팀의 현황을 보고 싶을 때 유용합니다.
