# 출석 설정

<div class="article-intro">

출석을 추적하기 전에, B1 Admin에 교회의 물리적 위치, 예배 일정, 각 예배에서 모이는 그룹에 대해 알려주어야 합니다. 이 일회성 설정은 교회 전체의 출석 추적 및 보고를 지원하는 구조를 만듭니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 출석 관리 권한이 있는 활성 B1 Admin 계정이 필요합니다. 접근 수준이 확실하지 않으면 [역할 및 권한](../people/roles-permissions.md)을 참조하세요.
- 예배 시간에 그룹을 배정할 계획이라면, 먼저 [그룹을 생성](../groups/creating-groups.md)해 두세요.

</div>

## 핵심 개념

- **캠퍼스** -- 교회가 모이는 물리적 장소 (예: "본관 캠퍼스," "북부 캠퍼스").
- **예배 시간** -- 캠퍼스에서 정기적으로 열리는 모임 (예: "주일 오전 9시," "수요일 오후 7시").
- **예정된 그룹** -- 특정 예배 시간에 배정된 그룹. 출석은 해당 예배의 맥락에서 추적됩니다.
- **비예정 그룹** -- 예배 시간에 연결되지 않고 자체적으로 출석을 추적하는 그룹.

## 출석 구조 설정

1. **B1 Admin**을 열고 사이드바에서 **출석**을 클릭합니다.
2. **설정** 탭을 선택합니다.
3. **캠퍼스 추가**를 클릭하고 장소 이름을 입력합니다. **저장**을 클릭합니다.
4. 캠퍼스를 선택한 상태에서 **예배 시간 추가**를 클릭합니다. "주일 오전 9시"와 같은 이름을 입력하고 **저장**을 클릭합니다.
5. 해당 캠퍼스의 각 예배 시간에 대해 반복합니다.
6. 예배 시간에 그룹을 배정하려면, 예배 시간을 선택하고 **그룹 추가**를 클릭합니다. 목록에서 그룹을 선택하고 **저장**을 클릭합니다.

### 그룹에서 출석 추적 활성화

그룹의 출석을 기록하려면 먼저 해당 그룹에서 출석 추적을 켜야 합니다.

1. 사이드바에서 **그룹**을 클릭하고 그룹을 선택합니다.
2. **편집** 연필 아이콘을 클릭합니다.
3. **출석 추적**을 **예**로 설정합니다.
4. **저장**을 클릭합니다.

:::tip
이전 단계에서 그룹을 예배 시간에 배정한 경우, 그룹 편집 화면의 **예배 시간 추가** 옵션도 사용하여 올바른 예배에 연결하세요. 이렇게 하면 세션이 올바른 캠퍼스와 시간에 연결됩니다.
:::

:::tip
정기 예배 외에 모이는 그룹(예: 자체 출석을 추적하는 주중 소그룹)은 비예정 그룹으로 남겨두세요. 출석 보고의 그룹 탭에 그대로 표시됩니다.
:::

## 설정 편집

언제든지 설정을 업데이트할 수 있습니다. 캠퍼스, 예배 시간 또는 그룹을 선택하고 **편집**을 클릭하여 세부 사항을 변경하거나 **삭제**를 클릭하여 제거하세요.

:::info
예배 시간을 제거해도 과거 출석 기록은 삭제되지 않습니다. 일정을 변경하더라도 이전 데이터는 보존됩니다.
:::

## 다음 단계
