# 출석 설정

<div class="article-intro">

출석을 추적하기 전에 B1 Admin에 교회의 물리적 위치, 예배가 발생하는 시간 및 각 예배에서 만나는 그룹을 알려야 합니다. 이 일회성 설정은 교회 전체의 모든 출석 추적 및 보고를 구동하는 구조를 만듭니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 출석 관리 권한이 있는 활성 B1 Admin 계정이 필요합니다. 액세스 수준이 불확실한 경우 [역할 및 권한](../people/roles-permissions.md)을 참조하세요.
- 그룹을 예배 시간에 할당할 계획이라면 먼저 [그룹이 생성](../groups/creating-groups.md)되어 있는지 확인하세요.

</div>

## 주요 개념

- **캠퍼스** -- 교회가 만나는 물리적 위치입니다(예: "메인 캠퍼스", "북쪽 캠퍼스").
- **예배 시간** -- 캠퍼스에서의 정기적인 모임입니다(예: "일요일 오전 9:00", "수요일 저녁 7:00").
- **예정된 그룹** -- 특정 예배 시간에 할당된 그룹입니다. 출석은 그 예배의 맥락에서 추적됩니다.
- **예정되지 않은 그룹** -- 예배 시간과 무관하게 자신의 출석을 추적하는 그룹입니다.

## 출석 구조 설정

1. **B1 Admin**을 열고 사이드바에서 **출석**을 클릭합니다.
2. **설정** 탭을 선택합니다.
3. **캠퍼스 추가**를 클릭하고 위치의 이름을 입력합니다. **저장**을 클릭합니다.
4. 캠퍼스를 선택한 상태에서 **예배 시간 추가**를 클릭합니다. "일요일 오전 9:00"과 같은 이름을 입력하고 **저장**을 클릭합니다.
5. 해당 캠퍼스의 각 예배 시간에 대해 반복합니다.
6. 그룹을 예배 시간에 할당하려면 예배 시간을 선택하고 **그룹 추가**를 클릭합니다. 목록에서 그룹을 선택하고 **저장**을 클릭합니다.

### 그룹에서 출석 추적 활성화

그룹이 출석을 기록하려면 먼저 해당 그룹에 대해 출석 추적을 켜야 합니다.

1. 사이드바에서 **그룹**을 클릭하고 그룹을 선택합니다.
2. **편집** 연필 아이콘을 클릭합니다.
3. **출석 추적**을 **예**로 설정합니다.
4. **저장**을 클릭합니다.

:::tip
이전 단계에서 그룹을 예배 시간에 할당한 경우 그룹의 편집 화면에서 **예배 시간 추가** 옵션을 사용하여 올바른 예배에 연결하세요. 이렇게 하면 세션이 올바른 캠퍼스 및 시간에 연결됩니다.
:::

:::tip
평일 소그룹과 같이 정기 예배 외에 만나는 그룹이 있으면 예정되지 않은 그룹으로 둘 수 있습니다. 여전히 출석 보고를 위해 그룹 탭에 나타납니다.
:::

## 설정 편집

언제든지 설정을 업데이트할 수 있습니다. 캠퍼스, 예배 시간 또는 그룹을 선택하고 **편집**을 클릭하여 세부사항을 변경하거나 **삭제**를 클릭하여 제거하세요.

:::info
예배 시간을 제거해도 과거 출석 기록은 삭제되지 않습니다. 일정을 변경해도 과거 데이터는 유지됩니다.
:::

## 다음은 무엇인가요?

캠퍼스, 예배 시간 및 그룹이 준비되면 [출석을 수동으로 기록](recording-attendance.md)하거나 예배를 위해 [자체 체크인](check-in.md)을 설정할 준비가 됩니다.
