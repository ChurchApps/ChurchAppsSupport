# 출석 설정

<div class="article-intro">

출석을 추적하기 전에 교회의 물리적 장소, 예배 시간, 각 예배에서 모이는 그룹을 B1 Admin에 알려야 합니다. 이 일회성 설정으로 교회 전체의 모든 출석 추적과 보고를 지원하는 구조가 만들어집니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 출석 관리 권한이 있는 활성 B1 Admin 계정이 필요합니다. 접근 권한이 확실하지 않은 경우 [역할 및 권한](../people/roles-permissions.md)을 참조하세요.
- 예배 시간에 그룹을 배정할 계획이라면 먼저 [그룹이 생성](../groups/creating-groups.md)되어 있는지 확인하세요.

</div>

## 핵심 개념

- **캠퍼스** -- 교회가 모이는 물리적 장소입니다 (예: "본당 캠퍼스", "북부 캠퍼스").
- **예배 시간** -- 캠퍼스에서 정기적으로 열리는 모임입니다 (예: "주일 오전 9시", "수요일 오후 7시").
- **예정된 그룹** -- 특정 예배 시간에 배정된 그룹입니다. 해당 예배의 맥락에서 출석이 추적됩니다.
- **미예정 그룹** -- 예배 시간에 연결되지 않고 자체적으로 출석을 추적하는 그룹입니다.

## 출석 구조 설정

1. **B1 Admin**을 열고 사이드바에서 **출석**을 클릭합니다.
2. **설정** 탭을 선택합니다.
3. **캠퍼스 추가**를 클릭하고 장소 이름을 입력합니다. **저장**을 클릭합니다.
4. 캠퍼스를 선택한 상태에서 **예배 시간 추가**를 클릭합니다. "주일 오전 9시"와 같은 이름을 입력하고 **저장**을 클릭합니다.
5. 해당 캠퍼스의 각 예배 시간에 대해 반복합니다.
6. 예배 시간에 그룹을 배정하려면 예배 시간을 선택하고 **그룹 추가**를 클릭합니다. 목록에서 그룹을 선택하고 **저장**을 클릭합니다.

:::tip
정규 예배 외에 모이는 그룹 -- 예를 들어 자체 출석을 추적하는 주중 소그룹 -- 은 미예정 그룹으로 남겨둘 수 있습니다. 여전히 출석 보고를 위한 그룹 탭에 표시됩니다.
:::

## 설정 편집

언제든지 설정을 업데이트할 수 있습니다. 캠퍼스, 예배 시간 또는 그룹을 선택하고 **편집**을 클릭하여 세부 정보를 변경하거나, **삭제**를 클릭하여 제거합니다.

:::info
예배 시간을 제거해도 과거 출석 기록은 삭제되지 않습니다. 일정을 변경하더라도 이전 데이터는 보존됩니다.
:::
