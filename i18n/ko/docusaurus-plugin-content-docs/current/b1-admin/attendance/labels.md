# 체크인 라벨 디자이너

<div class="article-intro">

라벨 디자이너를 사용하면 가족이 자녀를 체크인할 때 인쇄되는 이름표와 픽업 슬립 템플릿을 만들고 사용자 지정할 수 있습니다. 각 라벨에 표시되는 정보, 위치 및 모양을 정확히 제어할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [Attendance](setup)를 설정하고 체크인이 활성화된 서비스 시간을 최소 하나 이상 구성
- [Check-In](check-in)을 설정하여 라벨이 인쇄되도록 구성
- Attendance 섹션에 대한 관리 권한 필요

</div>

## 라벨 디자이너 열기

B1 Admin에서 왼쪽 사이드바의 **Attendance**를 선택한 후 **Labels**를 선택합니다. 저장된 라벨 템플릿 목록이 표시되며, **Nametag**과 **Pickup Slip**으로 분류됩니다.

## 라벨 유형

- **Nametag** — 아이에게 인쇄되어 부착됩니다. 일반적으로 아이의 이름, 교실/세션 및 보안 코드가 포함됩니다.
- **Pickup Slip** — 부모 또는 보호자에게 제공됩니다. 일반적으로 보안 코드와 체크인된 자녀 목록이 포함됩니다.

B1은 표준 3.5 × 1.1인치 열감지 라벨용으로 크기 조정된 기본 이름표 및 기본 픽업 슬립 템플릿으로 시작합니다.

## 라벨 템플릿 만들기

1. **Add Nametag** 또는 **Add Pickup Slip**을 클릭하거나 드롭다운을 사용하여 선택합니다.
2. 새 템플릿이 라벨 편집기에서 열립니다.

### 라벨 편집기

편집기는 구성된 크기에 따른 라벨의 축소된 미리보기를 표시합니다. 왼쪽 패널에서 다음을 구성할 수 있습니다:

- **Name** — 템플릿 이름(참조용)
- **Label Type** — Nametag 또는 Pickup Slip
- **Width / Height** — 인치 단위 라벨 크기

### 블록 추가

라벨은 블록으로 구성되며, 각 블록은 라벨 캔버스에 배치된 개별 콘텐츠 조각입니다. **Add Block**을 클릭하여 새 블록을 삽입하고 유형을 선택합니다:

- **Field** — 인쇄 시점에 데이터 값을 가져옵니다:
  - `person.displayName` — 사람의 전체 이름
  - `sessions` — 체크인한 서비스/교실
  - `securityCode` — 임의로 생성된 픽업 보안 코드
  - `children` — 자녀 목록(픽업 슬립용)
  - `person.nametagNotes` — 사람 기록의 특수 메모
  - `campus` — 캠퍼스 이름
- **Text** — 입력한 정적 텍스트(제목, 라벨 또는 지침용)
- **Barcode** — 보안 코드를 인코딩하는 바코드

### 블록 배치

각 블록에는 라벨 캔버스의 비율(0–100)로 표현된 **X**, **Y**, **Width** 및 **Height** 필드가 있습니다. 이를 조정하여 콘텐츠를 정확히 배치합니다. 다음을 설정할 수도 있습니다:

- **Font Size** — 포인트 단위 텍스트 크기
- **Bold** — 굵은 텍스트 설정/해제
- **Align** — 왼쪽, 중앙 또는 오른쪽 텍스트 정렬
- **Condition** — 선택적으로 필드가 비어있으면 블록 숨기기(예: nametagNotes에 값이 있는 경우에만 표시)

### 저장

**Save**를 클릭하여 템플릿을 저장합니다. B1 Checkin에서 라벨을 인쇄할 때 업데이트된 템플릿이 사용됩니다.

## 템플릿 순서 변경

여러 이름표 또는 픽업 슬립 템플릿이 있는 경우, B1 Checkin은 기본적으로 목록의 첫 번째 템플릿을 사용합니다. 템플릿을 드래그하여 순서를 변경합니다.

## 템플릿 삭제

템플릿 행의 삭제 아이콘을 클릭하고 확인합니다. 한 유형의 마지막 템플릿을 삭제하면 기본 제공 템플릿이 복원됩니다.

:::tip
템플릿을 편집한 후 테스트 인쇄를 수행하여 다음 서비스 전에 레이아웃이 올바른지 확인하세요.
:::
