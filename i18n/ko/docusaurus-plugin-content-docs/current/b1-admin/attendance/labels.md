# 체크인 라벨 디자이너

<div class="article-intro">

라벨 디자이너를 사용하면 가족이 자녀를 체크인할 때 인쇄되는 이름표 및 픽업 슬립 템플릿을 만들고 사용자 정의할 수 있습니다. 각 라벨에 표시되는 정보, 위치 및 모양을 정확히 제어할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [출석](setup)을 설정하고 체크인이 활성화된 최소 1개의 서비스 시간을 구성합니다
- [체크인](check-in)을 설정하여 라벨이 인쇄됩니다
- 출석 섹션에 대한 관리 액세스 권한이 필요합니다

</div>

## 라벨 디자이너 열기

B1 Admin에서 왼쪽 사이드바의 **출석**으로 이동하고 **라벨**을 선택합니다. 유형별로 분류된 저장된 라벨 템플릿 목록이 표시됩니다: **이름표** 및 **픽업 슬립**.

## 라벨 유형

- **이름표** — 인쇄되어 아동에게 부착됩니다. 일반적으로 아동의 이름, 교실/세션 및 보안 코드를 포함합니다.
- **픽업 슬립** — 부모 또는 보호자에게 제공됩니다. 일반적으로 보안 코드와 체크인한 자녀 목록을 포함합니다.

B1은 표준 3.5 × 1.1인치 열 감지 라벨로 크기가 조정된 기본 이름표 및 기본 픽업 슬립 템플릿을 제공합니다.

## 라벨 템플릿 만들기

1. **이름표 추가** 또는 **픽업 슬립 추가**(또는 드롭다운을 사용하여 선택)를 클릭합니다.
2. 새 템플릿이 라벨 편집기에서 열립니다.

### 라벨 편집기

편집기는 구성된 크기의 라벨의 축소된 미리보기를 보여줍니다. 왼쪽 패널에서 다음을 구성할 수 있습니다:

- **이름** — 템플릿 이름(참고용)
- **라벨 유형** — 이름표 또는 픽업 슬립
- **너비 / 높이** — 라벨 크기(인치)

### 블록 추가하기

라벨은 라벨 캔버스에 배치된 개별 콘텐츠 조각인 블록으로 구성됩니다. **블록 추가**를 클릭하여 새 블록을 삽입하고 유형을 선택합니다:

- **필드** -- 인쇄 시 데이터 값을 가져옵니다:
  - `person.displayName` — 사람의 전체 이름
  - `sessions` — 체크인한 서비스/교실
  - `securityCode` — 무작위로 생성된 픽업 보안 코드
  - `children` — 자녀 목록(픽업 슬립의 경우)
  - `person.nametagNotes` — 사람 기록에 있는 특수 노트
  - `campus` — 캠퍼스 이름
- **텍스트** -- 입력하는 정적 텍스트(제목, 레이블 또는 지침)
- **바코드** -- 보안 코드를 인코딩하는 바코드

### 블록 위치 지정하기

각 블록에는 라벨 캔버스 백분율(0~100)로 표현된 **X**, **Y**, **너비** 및 **높이** 필드가 있습니다. 이들을 조정하여 콘텐츠를 정확히 배치합니다. 다음도 설정할 수 있습니다:

- **글꼴 크기** — 포인트 단위의 텍스트 크기
- **굵게** -- 굵은 텍스트 토글
- **정렬** -- 왼쪽, 중앙 또는 오른쪽 텍스트 정렬
- **조건** — 필드가 비어 있으면 선택적으로 블록 숨기기(예: 값이 있는 경우에만 nametagNotes 표시)

### 저장하기

**저장**을 클릭하여 템플릿을 저장합니다. 업데이트된 템플릿은 다음에 B1 Checkin에서 라벨을 인쇄할 때 사용됩니다.

## 템플릿 재정렬하기

이름표 또는 픽업 슬립 템플릿이 여러 개 있는 경우 B1 Checkin은 기본적으로 목록의 첫 번째 템플릿을 사용합니다. 템플릿을 드래그하여 재정렬합니다.

## 템플릿 삭제하기

템플릿 행의 삭제 아이콘을 클릭하고 확인합니다. 유형의 마지막 템플릿을 삭제하면 기본 제공 템플릿이 복원됩니다.

:::tip
템플릿을 편집한 후 테스트 인쇄를 수행하여 다음 서비스 전에 레이아웃이 올바르게 보이는지 확인합니다.
:::
