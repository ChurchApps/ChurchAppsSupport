# 출석 기록

<div class="article-intro">

캠퍼스, 예배 시간 및 그룹이 설정되면 각 모임 후에 출석을 수동으로 기록할 수 있습니다. B1 Admin은 **세션** 주변에 출석을 구성합니다 -- 한 그룹당 한 세션입니다. 세션을 생성하고 참석한 사람을 표시하면 데이터가 직접 출석 보고서로 전달됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 캠퍼스, 예배 시간 및 그룹이 구성되어야 합니다. 아직 이를 수행하지 않았으면 [출석 설정](setup.md)을 참조하세요.
- 추적하려는 그룹에는 **출석 추적**이 활성화되어야 합니다. 자세한 내용은 [출석 설정](setup.md)을 참조하세요.

</div>

## 세션 생성

세션은 한 그룹 만남의 한 번의 발생을 나타냅니다 -- 예를 들어 특정 일요일의 K-3학년 클래스입니다.

1. **B1 Admin**을 열고 사이드바에서 **그룹**을 클릭합니다.
2. 출석을 기록할 그룹을 선택합니다.
3. **세션** 탭을 클릭합니다.
4. **새로 만들기**를 클릭하여 새 세션을 만듭니다.
5. 그룹이 예배 시간에 할당된 경우 **예배 시간**을 선택합니다. 예정되지 않은 그룹인 경우 이 필드는 표시되지 않습니다.
6. **세션 날짜**를 선택합니다 -- 오늘, 과거 날짜 또는 미래 날짜가 될 수 있습니다.
7. **저장**을 클릭합니다.

:::tip
아직 기록하지 않은 출석을 따라잡기 위해 과거 날짜에 대한 세션을 생성하거나 그룹이 만날 때 준비되도록 미리 생성할 수 있습니다.
:::

## 출석 표시

세션을 저장한 후 그룹의 회원이 페이지의 오른쪽에 나타납니다.

1. 참석한 각 사람 옆의 상자를 확인합니다.
2. 변경 사항이 자동으로 저장됩니다 -- 출석 표시에 대한 추가 저장 버튼이 없습니다.

:::info
현재 그룹 회원만 세션 목록에 나타납니다. 누군가 참석했지만 아직 그룹에 없으면 먼저 [그룹에 추가](../groups/group-members.md)한 다음 출석을 기록하세요.
:::

## 출석을 스프레드시트로 내보내기

세션 기록을 CSV 파일로 다운로드하여 Excel, Numbers 또는 Google Sheets에서 사용할 수 있습니다.

1. 내보낼 세션을 엽니다.
2. **CSV 다운로드** 버튼을 클릭합니다.
3. 스프레드시트 응용 프로그램에서 다운로드한 파일을 엽니다.

## 기록된 출석 보기

세션을 기록한 후 데이터가 출석 보고서에 나타납니다.

- **출석 탭** -- 시간이 지남에 따른 교회 전체 추세를 보여줍니다. [출석 추적](tracking-attendance.md)을 참조하세요.
- **그룹 탭** -- 개별 그룹별로 분류된 출석을 보여줍니다.

:::tip
방금 생성한 세션이 보고서에 즉시 나타나지 않으면 세션 날짜가 보고서 필터에서 선택한 날짜 범위 내에 있는지 확인하세요.
:::
