# 출석 기록

<div class="article-intro">

캠퍼스, 예배 시간, 그룹이 설정되면 각 모임 후 출석을 수동으로 기록할 수 있습니다. B1 Admin은 **세션** 단위로 출석을 관리합니다 -- 그룹당 모임 날짜마다 하나의 세션입니다. 세션을 생성하고, 참석자를 표시하면 데이터가 출석 보고서에 직접 반영됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 캠퍼스, 예배 시간, 그룹이 설정되어 있어야 합니다. 아직 하지 않았다면 [출석 설정](setup.md)을 참조하세요.
- 추적하려는 그룹에 **출석 추적**이 활성화되어 있어야 합니다. 자세한 내용은 [출석 설정](setup.md)을 참조하세요.

</div>

## 세션 생성

세션은 그룹 모임의 한 번의 발생을 나타냅니다 -- 예를 들어 특정 주일의 유치부--초등부 반입니다.

1. **B1 Admin**을 열고 사이드바에서 **그룹**을 클릭합니다.
2. 출석을 기록할 그룹을 선택합니다.
3. **세션** 탭을 클릭합니다.
4. **새로 만들기**를 클릭하여 새 세션을 생성합니다.
5. 그룹이 예배 시간에 배정되어 있으면 **예배 시간**을 선택합니다. 비예정 그룹인 경우 이 필드는 나타나지 않습니다.
6. **세션 날짜**를 선택합니다 -- 오늘, 과거 날짜 또는 미래 날짜가 될 수 있습니다.
7. **저장**을 클릭합니다.

:::tip
아직 기록하지 못한 출석을 보완하기 위해 과거 날짜로 세션을 생성하거나, 그룹이 모이기 전에 미리 세션을 생성해 둘 수 있습니다.
:::

## 출석 표시

세션을 저장하면 페이지 오른쪽에 그룹 멤버가 나타납니다.

1. 참석한 각 사람 옆의 체크박스를 선택합니다.
2. 변경 사항은 자동으로 저장됩니다 -- 출석 표시를 위한 별도의 저장 버튼은 없습니다.

:::info
세션 목록에는 현재 그룹 멤버만 나타납니다. 참석했지만 아직 그룹에 속하지 않은 사람이 있다면, 먼저 [그룹에 추가](../groups/group-members.md)한 후 출석을 기록하세요.
:::

## 출석을 스프레드시트로 내보내기

세션 기록을 CSV 파일로 다운로드하여 Excel, Numbers 또는 Google Sheets에서 사용할 수 있습니다.

1. 내보내려는 세션을 엽니다.
2. **CSV 다운로드** 버튼을 클릭합니다.
3. 다운로드된 파일을 스프레드시트 애플리케이션에서 엽니다.

## 기록된 출석 확인

세션을 기록하면 데이터가 출석 보고서에 나타납니다.

- **출석 탭** -- 시간에 따른 교회 전체 추세를 보여줍니다. [출석 추적](tracking-attendance.md)을 참조하세요.
- **그룹 탭** -- 개별 그룹별로 분류된 출석을 보여줍니다.

:::tip
방금 생성한 세션이 보고서에 바로 나타나지 않으면, 세션 날짜가 보고서 필터에서 선택한 날짜 범위 내에 있는지 확인하세요.
:::
