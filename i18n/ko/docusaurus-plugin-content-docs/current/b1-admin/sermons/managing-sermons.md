# 설교 관리

<div class="article-intro">

설교 페이지에는 전체 설교 라이브러리가 표시됩니다. 여기서 새 설교를 추가하고, 기존 항목을 편집하며, 재생 목록별로 콘텐츠를 정리할 수 있습니다. 각 설교는 YouTube, Vimeo, Facebook 또는 맞춤 URL에 호스팅된 동영상이나 오디오에 연결할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- **contentApi.streamingServices.edit** 권한이 필요합니다. 접근 권한이 없으면 [역할 및 권한](../settings/roles-permissions.md)을 참조하십시오.
- 설교를 정리할 [재생 목록](playlists)을 최소 하나 만드십시오
- YouTube, Vimeo 또는 Facebook의 동영상 ID나 URL을 준비하십시오

</div>

## 설교 라이브러리 조회

1. B1 Admin에서 왼쪽 사이드바의 **설교**를 클릭합니다.
2. 설교 페이지에 재생 목록별로 정리된 모든 설교 항목이 표시됩니다. 각 설교에는 썸네일, 제목, 날짜가 나타납니다.
3. 설교를 클릭하여 세부 정보를 조회하거나 편집합니다.

## 설교 추가

1. 오른쪽 상단의 **설교 추가** 버튼을 클릭하고 드롭다운에서 **설교 추가**를 선택합니다.
2. 설교를 배정할 **재생 목록**을 선택합니다.
3. **동영상 제공자**를 선택합니다 -- YouTube, Vimeo, Facebook 또는 맞춤 URL. B1 시스템과 가장 잘 작동하는 YouTube를 권장합니다.
4. 동영상 ID 또는 URL을 입력하고 **가져오기**를 클릭합니다. YouTube의 경우, 동영상 ID는 YouTube URL에서 `v=` 뒤에 오는 문자열입니다.
5. **가져오기**를 클릭하면 게시 날짜, 길이, 제목, 설명, 썸네일 등 설교 세부 정보가 자동으로 가져와집니다.
6. 원하는 변경을 하고 **저장**을 클릭합니다.

:::tip
**설교 추가** 드롭다운에서 **영구 라이브 URL 추가**를 선택하여 영구 라이브 스트리밍 URL을 추가할 수도 있습니다. 이는 채널 ID를 사용하여 YouTube 채널의 라이브 스트리밍에 지속적으로 연결합니다. 자세한 내용은 [라이브 스트리밍](live-streaming)을 참조하십시오.
:::

## 설교 편집

1. 라이브러리에서 설교를 클릭하여 세부 정보를 엽니다.
2. 제목, 설교자, 날짜, 설명, 썸네일 또는 미디어 링크를 필요에 따라 업데이트합니다.
3. **저장**을 클릭하여 변경 사항을 적용합니다.

## 설교 세부 정보

각 설교 항목에는 다음이 포함될 수 있습니다:

- **제목** -- 방문자에게 표시되는 설교 이름
- **설교자** -- 설교를 전한 사람
- **날짜** -- 게시 또는 전달 날짜
- **설명** -- 설교 내용 요약
- **썸네일** -- 설교 라이브러리에 표시되는 미리 보기 이미지
- **동영상/오디오 링크** -- YouTube, Vimeo, Facebook 또는 맞춤 호스트의 설교 미디어 URL

## 라이브 스트리밍용 설교 예약

설교를 추가한 후, 라이브 스트리밍 페이지에서 방송을 예약할 수 있습니다:

1. **라이브 스트리밍 시간** 탭으로 이동합니다.
2. 예배를 편집하고 **동영상 설정**에서 드롭다운에서 설교를 선택합니다.
3. 예약된 예배 시간에 설교가 재생됩니다.

:::info
설교를 하나씩 추가하는 대신 여러 개를 한 번에 가져오려면, [일괄 가져오기](bulk-import) 도구를 사용하여 YouTube 또는 Vimeo 계정에서 직접 동영상을 가져오십시오.
:::
