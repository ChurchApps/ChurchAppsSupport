# 재생 목록

<div class="article-intro">

재생 목록을 사용하면 설교를 시리즈 또는 컬렉션으로 정리할 수 있습니다. 방문자가 B1.church 웹사이트에서 설교 라이브러리를 탐색할 때, 재생 목록을 통해 완전한 설교 시리즈를 올바른 순서로 찾아서 따라갈 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- **contentApi.streamingServices.edit** 권한이 필요합니다. 접근 권한이 없으면 [역할 및 권한](../settings/roles-permissions.md)을 참조하십시오.
- 재생 목록의 이름, 설명, 썸네일 이미지를 준비하십시오

</div>

## 재생 목록 조회

1. B1 Admin에서 왼쪽 사이드바의 **설교**를 클릭합니다.
2. 페이지 상단의 **재생 목록** 탭을 클릭합니다.
3. 이름과 설명이 포함된 기존 재생 목록 목록이 표시됩니다.

## 재생 목록 만들기

1. **첫 재생 목록 만들기** 버튼(재생 목록이 없는 경우) 또는 **재생 목록 추가**를 클릭합니다.
2. 재생 목록의 **이름**을 입력합니다 -- 예를 들어, "믿음의 기초" 또는 "시편과 함께하는 여름" 같은 설교 시리즈 제목.
3. 방문자에게 시리즈에 대한 간략한 개요를 제공하는 **설명**을 추가합니다.
4. 재생 목록이 공개되는 시점을 제어하는 **게시 날짜**를 설정합니다.
5. 시리즈를 시각적으로 대표하는 **썸네일 이미지**를 업로드합니다.
6. **저장**을 클릭하여 재생 목록을 만듭니다.

:::tip
설교를 추가하기 전에 재생 목록을 먼저 만드십시오. 이렇게 하면 나중에 재정리할 필요 없이, 진행하면서 각 설교를 올바른 재생 목록에 배정할 수 있습니다.
:::

## 재생 목록에 설교 추가

[설교를 추가하거나 편집](managing-sermons)할 때, **재생 목록** 드롭다운을 사용하여 재생 목록에 배정합니다. 같은 재생 목록에 배정된 모든 설교는 웹사이트에서 함께 그룹화되어 순서대로 표시됩니다.

:::info
[일괄 가져오기](bulk-import) 시에도 설교를 재생 목록에 배정할 수 있습니다. YouTube 또는 Vimeo에서 가져올 때, **가져올 재생 목록** 드롭다운에서 재생 목록을 선택한 후 **가져오기**를 클릭하십시오.
:::

## 웹사이트에서 재생 목록 표시

재생 목록은 B1.church 웹사이트에 탐색 가능한 컬렉션으로 표시됩니다. 방문자는 다음을 할 수 있습니다:

- 사용 가능한 모든 설교 시리즈를 한눈에 확인합니다
- 재생 목록을 클릭하여 해당 시리즈의 모든 설교를 봅니다
- 순서대로 설교를 시청하거나 청취합니다
