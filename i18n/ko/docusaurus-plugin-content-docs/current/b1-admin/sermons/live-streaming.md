# 라이브 스트리밍

<div class="article-intro">

라이브 스트리밍 시간 페이지에서는 교회의 스트리밍 일정을 설정하고, 예배 시간을 관리하며, 시청자 경험을 맞춤 설정할 수 있습니다. 정기 주간 예배나 일회성 행사를 설정하고, 채팅 및 동영상 설정을 구성하며, 스트리밍 시작 시점을 제어합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- **contentApi.streamingServices.edit** 권한이 필요합니다. 접근 권한이 없으면 [역할 및 권한](../settings/roles-permissions.md)을 참조하십시오.
- 자동 라이브 스트리밍을 사용할 계획이라면 YouTube 채널 ID를 준비하십시오
- 스트리밍 소스로 사용할 [설교](managing-sermons) 또는 영구 라이브 URL을 최소 하나 추가하십시오

</div>

이 페이지에는 두 개의 주요 탭이 있습니다: 라이브 스트리밍 일정을 관리하는 **예배**와 스트리밍 페이지를 설정하는 **설정**.

## 예배 관리

### 예배 추가

1. B1 Admin에서 왼쪽 사이드바의 **설교**를 클릭한 다음, **라이브 스트리밍 시간** 탭을 클릭합니다.
2. **예배 추가** 버튼을 클릭하여 새 예약 예배를 만듭니다.
3. **예배 이름**을 입력합니다 (예: "주일 오전 예배").
4. **예배 시간**을 설정합니다 -- 예배가 시작되는 요일과 시간을 선택합니다.
5. **매주 반복**을 정기 주간 예배의 경우 **예**로, 일회성 행사의 경우 **아니오**로 설정합니다.

### 채팅 및 동영상 설정

6. **채팅 설정**에서 예배 전후로 채팅이 활성화될 시간(분)을 설정합니다. 이를 통해 방문자가 예배 시작 전에 채팅을 시작하고 이후에도 계속할 수 있습니다.
7. **동영상 설정**에서 카운트다운이나 예배 전 콘텐츠를 위해 동영상 스트리밍을 얼마나 일찍 시작할지 설정합니다.
8. 드롭다운에서 재생할 설교를 선택합니다:
   - **최신 설교** -- 가장 최근에 추가된 동영상을 자동으로 재생합니다.
   - **현재 라이브 예배** -- 채널 ID를 사용하여 YouTube에서 현재 라이브 스트리밍을 재생합니다.
   - 이미 저장한 특정 설교를 선택할 수도 있습니다.
9. **저장**을 클릭하여 예배를 예약합니다.

:::info
반복으로 설정된 예배는 매주 자동으로 업데이트됩니다. 필요한 만큼 예배를 추가할 수 있습니다. 방문자는 스트리밍 페이지를 방문하면 다음 예정된 예배 시간을 확인할 수 있습니다.
:::

## 스트리밍 페이지 설정

라이브 스트리밍과 함께 표시되는 탭과 링크를 맞춤 설정하려면 **설정** 탭을 클릭합니다.

### 탭 추가

1. **추가** 버튼을 클릭하여 라이브 스트리밍 페이지에 새 탭을 추가합니다.
2. 미리 디자인된 탭(**채팅** 또는 **기도**) 또는 외부 URL이 있는 맞춤 탭 중 선택합니다.
3. 미리 디자인된 탭의 경우, **탭 텍스트** 상자에 이름을 입력하면 설정이 완료됩니다.
4. 링크 탭의 경우, 탭 이름을 입력하고, 아이콘 버튼을 클릭하여 아이콘을 선택하고, URL을 입력합니다.
5. 설정한 탭이 시청자가 추가 리소스와 대화형 기능에 접근할 수 있도록 라이브 스트리밍 페이지에 표시됩니다.

### 스트리밍 미리 보기

**스트리밍 보기** 버튼을 클릭하여 로고, 예배 시간, 설정한 탭을 포함한 라이브 스트리밍 페이지가 방문자에게 어떻게 보이는지 정확히 확인합니다.

## YouTube 라이브 스트리밍 설정

자동 라이브 스트리밍을 위해 YouTube 채널을 연결하려면:

1. **설교**로 이동하여 **설교 추가**를 클릭한 다음, **영구 라이브 URL 추가**를 선택합니다.
2. 동영상 제공자가 기본적으로 **현재 YouTube 라이브 스트리밍**으로 설정됩니다. **YouTube 채널 ID**를 입력합니다.
3. 제목과 설명을 추가한 다음 **저장**을 클릭합니다.
4. **라이브 스트리밍 시간**에서 예배를 만들고 설교 드롭다운에서 영구 라이브 URL을 선택합니다.

:::tip
YouTube 채널 ID를 찾으려면, YouTube 채널의 고급 설정으로 이동하여 채널 ID 값을 복사하십시오.
:::

## 색상 및 로고 맞춤 설정

라이브 스트리밍 페이지는 웹사이트의 [외관](../website/appearance) 설정을 사용합니다:

- 헤더에는 어두운 텍스트와 함께 **밝은 강조 색상**이 사용됩니다.
- 사이드바에는 밝은 텍스트와 함께 **어두운 강조 색상**이 사용됩니다.
- 스트리밍 페이지에 **밝은 배경 로고**가 표시됩니다. 투명 배경과 4:1 비율의 이미지를 사용하십시오.

이를 변경하려면 **웹사이트**, **외관**으로 이동하여 [색상 팔레트](../website/appearance#색상-팔레트) 및 [로고](../website/appearance#로고-및-브랜딩) 설정을 업데이트하십시오.

## 스트리밍 호스트 추가

팀원에게 호스트 기능(채팅 중재, 기도 요청 응답)을 부여하려면:

1. 왼쪽 사이드바의 **설정**으로 이동하여 **역할**을 클릭합니다.
2. 더하기 버튼을 클릭하고 **맞춤 역할 추가**를 선택합니다.
3. 역할 이름을 "스트리밍 호스트"로 지정하고 **저장**을 클릭합니다.
4. 새 역할을 클릭한 다음, 멤버 섹션에서 **추가**를 클릭하여 사람을 추가합니다.
5. **권한 편집**으로 스크롤하여 **콘텐츠** 섹션을 펼치고 **채팅 호스트**를 체크합니다.

호스트가 라이브 스트리밍 페이지에 로그인하면, 채팅 중재 및 기도 요청 관리를 포함한 특별 기능을 사용할 수 있습니다.

:::info
역할 만들기 및 권한 관리에 대한 자세한 내용은 [역할 및 권한](../settings/roles-permissions.md)을 참조하십시오.
:::

## 문제 해결

채널 ID로 "현재 YouTube 라이브 스트리밍" 옵션을 사용할 때 자동 YouTube 라이브 스트리밍이 올바르게 표시되지 않으면, 다음을 시도하십시오:

**증상:**
- 라이브 스트리밍 임베드에 "동영상을 사용할 수 없음"이 표시됨
- 페이지가 로드되지만 동영상이 나타나지 않음
- 직접 YouTube 임베드는 작동하지만 자동 채널 라이브 스트리밍은 작동하지 않음

**해결 방법:**
YouTube 채널에서 이전 또는 예정된 라이브 스트리밍을 확인하고 삭제하십시오:

1. YouTube 스튜디오로 이동합니다.
2. **콘텐츠**, **라이브**로 이동합니다.
3. 이전에 예약된 라이브 또는 예정된 스트리밍을 찾습니다.
4. 이전 또는 예약된 라이브 스트리밍 항목을 삭제합니다.
5. 라이브 스트리밍 페이지를 다시 테스트합니다.

:::warning
YouTube의 자동 채널 라이브 스트리밍 임베드는 채널에 여러 개의 예약된 또는 과거 라이브 스트리밍 항목이 있을 때 차단될 수 있습니다. 이를 제거하면 YouTube가 현재 라이브 스트리밍을 올바르게 식별하고 제공할 수 있습니다.
:::

**추가 요구 사항:**
- 라이브 스트리밍은 **공개**(미등록 또는 비공개가 아닌)로 설정되어야 합니다.
- YouTube 스트리밍 설정에서 임베딩이 허용되어야 합니다.
- **YouTube** 제공자(동영상 ID 사용)가 아닌 **현재 YouTube 라이브 스트리밍** 제공자(채널 ID 사용)를 사용하고 있는지 확인하십시오.
