# 일괄 가져오기

<div class="article-intro">

일괄 가져오기 페이지에서는 YouTube 또는 Vimeo에서 동영상을 가져와 설교 라이브러리를 빠르게 채울 수 있습니다. 각 동영상을 하나씩 추가하는 대신 기존 설교 콘텐츠를 가져오는 가장 빠른 방법입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- **contentApi.streamingServices.edit** 권한이 필요합니다. 접근 권한이 없으면 [역할 및 권한](../settings/roles-permissions.md)을 참조하십시오.
- 설교를 가져올 [재생 목록](playlists)을 최소 하나 만드십시오
- YouTube 채널 ID 또는 Vimeo 계정 정보를 준비하십시오

</div>

## 소스 선택

1. B1 Admin에서 왼쪽 사이드바의 **설교**를 클릭한 다음, **일괄 가져오기** 탭을 클릭합니다.
2. 두 개의 클릭 가능한 카드가 표시됩니다:
   - **YouTube** (빨간색) -- YouTube 채널에서 동영상을 가져옵니다
   - **Vimeo** (파란색) -- Vimeo 계정에서 동영상을 가져옵니다
3. 설교가 호스팅된 플랫폼의 카드를 클릭합니다.

:::tip
두 플랫폼 모두에 동영상이 있는 경우 언제든지 돌아가서 소스를 전환할 수 있습니다.
:::

## 가져오기 전에

동영상을 가져오기 전에, 정리할 재생 목록이 최소 하나 필요합니다. 아직 재생 목록을 만들지 않았다면:

1. **재생 목록** 탭을 클릭합니다.
2. **첫 재생 목록 만들기**를 클릭하고 이름, 설명, 게시 날짜, 썸네일을 입력합니다.
3. **저장**을 클릭한 다음, **일괄 가져오기** 탭으로 돌아옵니다.

재생 목록 만들기 및 관리에 대한 자세한 안내는 [재생 목록](playlists)을 참조하십시오.

## 동영상 가져오기

1. YouTube 또는 Vimeo를 선택한 후, 제공된 필드에 **YouTube 채널 ID** 또는 **Vimeo 계정 정보**를 입력합니다.
2. **가져오기** 버튼을 클릭하여 채널 또는 계정에서 사용 가능한 모든 동영상을 가져옵니다.
3. 가져온 후, 체크박스가 있는 모든 동영상 목록이 표시됩니다.
4. 가져오려는 동영상 옆의 체크박스를 선택합니다. 전체를 선택하거나 특정 항목만 선택할 수 있습니다.
5. 선택적으로, **새 동영상 자동 가져오기**를 활성화하여 향후 업로드를 자동으로 라이브러리에 추가합니다.
6. **재생 목록으로 가져오기** 드롭다운을 클릭하여 이 동영상을 추가할 재생 목록을 선택합니다.
7. **가져오기** 버튼을 클릭하여 일괄 가져오기를 완료합니다.

제목, 설명, 날짜, 썸네일을 포함한 모든 세부 정보와 함께 동영상이 가져와집니다.

:::info
일괄 가져오기는 다른 플랫폼에서 이전할 때 시작하기에 이상적입니다. 앞으로 개별 설교를 추가할 때는 [설교 관리](managing-sermons) 페이지를 사용하십시오.
:::
