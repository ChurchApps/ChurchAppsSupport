# 설교

<div class="article-intro">

B1 Admin의 설교 섹션에서는 교회의 설교 라이브러리와 라이브 스트리밍을 관리할 수 있습니다. 개별 설교를 추가하고, 재생 목록으로 정리하며, 라이브 스트리밍 일정을 설정하고, YouTube나 Vimeo에서 기존 콘텐츠를 일괄 가져올 수 있습니다.

</div>

:::info
설교 섹션에 접근하려면 **contentApi.streamingServices.edit** 권한이 필요합니다. 메뉴에 설교가 보이지 않으면, 교회 관리자에게 적절한 역할을 부여해 달라고 요청하십시오. 권한 관리에 대한 자세한 내용은 [역할 및 권한](../settings/roles-permissions.md)을 참조하십시오.
:::

## 수행할 수 있는 작업

설교 섹션은 네 가지 주요 영역으로 구성되어 있습니다:

### 설교

라이브러리에서 개별 설교 항목을 추가하고 관리합니다. 각 설교에는 제목, 설교자, 날짜, 설명, 썸네일, 동영상 또는 오디오 콘텐츠 링크를 포함할 수 있습니다. 설교를 YouTube, Vimeo, Facebook 또는 맞춤 동영상 URL에 연결할 수 있습니다. 자세한 내용은 [설교 관리](managing-sermons)를 참조하십시오.

### 재생 목록

설교를 시리즈 또는 컬렉션으로 정리합니다. 재생 목록을 통해 교인들이 설교 시리즈를 순서대로 쉽게 탐색할 수 있습니다. 각 재생 목록에는 고유한 이름, 설명, 썸네일이 있습니다. 자세한 내용은 [재생 목록](playlists)을 참조하십시오.

### 라이브 스트리밍 시간

교회의 라이브 스트리밍 일정을 설정합니다. 정기 주간 예배나 일회성 행사를 설정하고, 채팅 및 동영상 설정을 관리하며, 스트리밍 시작 시점을 제어합니다. 방문자는 스트리밍 페이지에서 다음 예정된 예배 시간을 확인할 수 있습니다. 설정 방법은 [라이브 스트리밍](live-streaming)을 참조하십시오.

### 일괄 가져오기

YouTube나 Vimeo에서 동영상을 일괄로 가져와 설교 라이브러리를 빠르게 채웁니다. 기존 설교 콘텐츠를 B1으로 이전하는 가장 빠른 방법입니다. 시작하려면 [일괄 가져오기](bulk-import)를 참조하십시오.

:::tip
처음 설교를 설정하는 경우, [재생 목록](playlists)을 먼저 만든 다음, [설교를 개별로 추가](managing-sermons)하거나 YouTube 또는 Vimeo에서 [일괄 가져오기](bulk-import)하십시오.
:::
