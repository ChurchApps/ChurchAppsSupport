# 봉사

<div class="article-intro">

봉사 섹션은 자원봉사자를 조율하고 교회 예배를 계획하는 허브입니다. 주일 예배, 주중 모임, 특별 행사 등을 진행하든 이 섹션에서 모든 것을 하나로 가져와 팀이 누가 언제 봉사하는지 정확히 알 수 있도록 할 수 있습니다.

</div>

## 봉사 페이지 작동 방식

**봉사** 페이지를 열면 사역이 상단의 탭으로 표시됩니다. 각 사역은 자체 계획 유형과 팀을 가지고 있으므로 여러 사역을 독립적으로 관리할 수 있습니다. 예를 들어, 예배팀, 어린이 사역, 환대팀을 위한 별도의 탭이 있을 수 있습니다.

사역의 구성원인 관리자가 아닌 직원도 봉사 페이지에 접근할 수 있습니다. 그들은 소속된 사역만 볼 수 있으며, 전체 관리자 권한 없이도 자신의 사역에 대해 계획, 배정, 계획 유형, 예배 순서를 만들고 편집할 수 있습니다.

:::tip
관리자인 경우 **모두 표시**를 토글하여 모든 사역을 한 번에 볼 수 있습니다. 여러 팀의 배정을 확인해야 할 때 유용합니다.
:::

## 세부 섹션

봉사 섹션에는 다음 주요 영역이 있습니다.

- **[계획](./plans.md)** -- 특정 날짜의 예배 계획을 만들고 자원봉사자를 팀에 배정합니다. 계획은 누가 언제 봉사하는지를 정리하여 모든 사람이 미리 자신의 역할을 알 수 있도록 합니다. **[계획 개요](./plans-overview.md)**를 사용하여 여러 주의 모든 배정을 한 개의 격자로 확인할 수 있습니다.
- **[찬송](./songs.md)** -- 가사, 편곡, 외부 링크를 포함한 찬송 라이브러리를 관리합니다. 예배팀이 이를 사용하여 다가올 예배를 준비할 수 있습니다.
- **[작업](./tasks.md)** -- 팀원에게 작업을 배정하고 추적합니다. [자동화](./automations.md)를 설정하여 반복 작업을 자동으로 만들면 아무것도 놓치지 않을 수 있습니다.
- **[워크플로우](./workflows.md)** -- 시각적 칸반 보드에서 사람을 일련의 단계로 이동시킵니다(예: 새 방문자 팔로우업 또는 교인 절차). 단계는 자동화된 작업(이메일 발송, 대기, 그룹 추가)을 실행할 수 있으며, 카드는 결과 버튼, 조건부 라우팅 또는 이벤트와 일정 트리거로 진행됩니다. 기본 보고서가 포함되어 있습니다.

## 시작하기

1. B1 Admin의 메인 메뉴에서 **봉사**로 이동합니다.
2. 사역 추가 버튼을 클릭하여 새 사역을 만들거나 "모두 표시"를 토글하여 기존 사역 탭을 봅니다. **사역 탭**을 선택하여 계획 유형과 팀을 확인합니다.
3. "계획 유형 만들기"를 선택하거나 기존 계획 유형을 선택하여 [예배 계획 만들기](./plans.md)를 시작하거나, [찬송](./songs.md) 및 [작업](./tasks.md) 세부 섹션을 살펴봅니다.

:::info
계획을 만들기 전에 봉사 영역에서 사역을 만들고, 각 사역 내에서 팀을 만들어야 합니다.
:::
