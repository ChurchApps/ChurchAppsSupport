# 봉사

<div class="article-intro">

봉사 섹션은 자원봉사자를 조율하고 예배를 기획하는 중심 허브입니다. 주일 예배, 주중 모임, 특별 행사 등 어떤 경우든, 이곳에서 모든 것을 조합하여 팀원들이 언제 누가 봉사하는지 정확히 알 수 있도록 합니다.

</div>

## 봉사 페이지 사용 방법

**봉사** 페이지를 열면, 상단에 탭으로 사역부가 표시됩니다. 각 사역부에는 고유한 계획 유형과 팀이 있으므로, 여러 사역부를 독립적으로 관리할 수 있습니다. 예를 들어, 예배부, 어린이 사역부, 환영부 등 별도의 탭을 가질 수 있습니다.

:::tip
관리자인 경우, **모두 보기**를 켜면 모든 사역부를 한 번에 볼 수 있습니다. 여러 팀의 배정 상황을 확인해야 할 때 유용합니다.
:::

## 하위 섹션

봉사 섹션에는 세 가지 주요 영역이 포함되어 있습니다:

- **[계획](./plans.md)** -- 특정 날짜에 대한 예배 계획을 생성하고 팀에 자원봉사자를 배정합니다. 계획을 통해 누가 언제 봉사하는지 정리하여 모든 사람이 사전에 자신의 역할을 알 수 있습니다.
- **[곡 목록](./songs.md)** -- 가사, 편곡, 외부 링크가 포함된 찬양 곡 라이브러리를 관리합니다. 찬양팀이 다가오는 예배를 준비하는 데 활용할 수 있습니다.
- **[작업](./tasks.md)** -- 팀원에게 작업을 배정하고 추적합니다. [자동화](./automations.md)를 설정하여 반복 작업을 자동으로 생성하면 누락되는 일이 없습니다.

## 시작하기

1. B1 Admin 메인 메뉴에서 **봉사**로 이동합니다.
2. **사역부 탭**을 선택하여 해당 사역부의 계획 유형과 팀을 확인합니다.
3. 계획 유형을 선택하여 [예배 계획 생성](./plans.md)을 시작하거나, [곡 목록](./songs.md) 및 [작업](./tasks.md) 하위 섹션을 살펴보십시오.

:::info
계획을 생성하려면 먼저 설정 영역에서 사역부와 팀을 구성해야 합니다. 봉사 구조 설정에 대한 자세한 내용은 설정 문서를 참조하십시오.
:::
