# 봉사

<div class="article-intro">

봉사 섹션은 자원봉사자를 조정하고 교회 예배를 계획하는 허브입니다. 일요일 경배, 평일 모임 또는 특별 이벤트를 조직하든 이곳이 팀이 정확히 누가 언제 봉사하는지 알 수 있도록 모든 것을 모으는 곳입니다.

</div>

## 봉사 페이지 작동 방식

**봉사** 페이지를 열면 사역이 상단에 탭으로 표시됩니다. 각 사역은 자신의 계획 유형과 팀을 가지고 있어 여러 사역을 독립적으로 관리할 수 있습니다. 예를 들어 경배팀, 아동 사역, 환대를 위한 별도의 탭이 있을 수 있습니다.

사역 회원인 관리자가 아닌 직원도 봉사 페이지에 액세스할 수 있습니다. 그들은 자신이 속한 사역만 보고 전체 관리 권한이 없어도 계획과 할당을 볼 수 있습니다.

:::tip
관리자인 경우 **모두 표시**를 토글하여 한 번에 모든 사역을 봅니다. 여러 팀의 할당을 확인해야 할 때 유용합니다.
:::

## 하위 섹션

봉사 섹션에는 세 가지 주요 영역이 있습니다:

- **[계획](./plans.md)** -- 특정 날짜의 예배 계획을 만들고 자원봉사자를 팀에 할당합니다. 계획은 누가 언제 봉사하는지 정리하여 모든 사람이 미리 자신의 역할을 알 수 있도록 도와줍니다. **[계획 개요](./plans-overview.md)**를 사용하여 여러 주의 모든 예정된 할당을 한 표에서 봅니다.
- **[노래](./songs.md)** -- 가사, 편곡, 외부 링크가 있는 경배 노래 라이브러리를 관리합니다. 경배팀은 다음 예배를 준비하는 데 이를 사용할 수 있습니다.
- **[작업](./tasks.md)** -- 팀 회원에게 작업을 할당하고 추적합니다. [자동화](./automations.md)를 설정하여 반복 작업을 자동으로 만들어 아무것도 놓치지 않도록 합니다.

## 시작하기

1. B1 Admin의 메인 메뉴에서 **봉사**로 이동합니다.
2. 새 사역을 만들려면 사역 추가 버튼을 클릭하거나 기존 사역 탭을 보려면 "모두 표시"를 토글합니다. **사역 탭**을 선택하여 계획 유형과 팀을 봅니다.
3. "계획 유형 만들기"를 선택하거나 [예배 계획](./plans.md) 만들기를 시작하려면 기존 계획 유형을 선택하거나 [노래](./songs.md) 및 [작업](./tasks.md) 하위 섹션을 탐색합니다.

:::info
계획을 만들기 전에 봉사 영역에서 사역을 만들고 각 사역 내에서 팀을 만들어야 합니다.
:::
