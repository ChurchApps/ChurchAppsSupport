# 교실 TV를 B1 Admin 레슨에 연결

<div class="article-intro">

**FreePlay** — ChurchApps의 무료 TV 앱을 사용하여 교실 TV를 B1 Admin 레슨에 연결할 수 있습니다. 연결되면 TV는 B1 Admin에서 예약한 레슨 콘텐츠를 표시하여 교사가 교실 화면에서 사용할 수 있도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin에서 최소한 하나의 레슨을 예약합니다([레슨 예약](./scheduling-lessons.md) 참조)
- 사용하려는 TV 또는 Fire Stick에 **FreePlay** 설치

</div>

## 1단계 — TV에 FreePlay 설치

교실 디스플레이에 **FreePlay** 앱을 설치합니다:

- **Amazon Fire Stick / Fire TV** — Amazon Appstore에서 **FreePlay** 검색
- **Android TV / Google TV** — Google Play Store에서 **FreePlay** 검색
- **Apple TV** — App Store에서 **FreePlay** 검색

전체 설정 지침은 [FreePlay 시작 가이드](/docs/freeplay/getting-started/)를 참조합니다.

## 2단계 — 콘텐츠 제공자로 B1.church 연결

1. TV에서 **FreePlay** 앱을 엽니다
2. **콘텐츠 제공자** 화면으로 이동합니다(사이드바에서 **제공자** 아이콘 선택)
3. 사용 가능한 제공자 목록에서 **B1.church** 찾기 및 선택
4. 화면의 지시에 따라 B1 교회 자격증명으로 로그인합니다
5. FreePlay가 연결 성공 후 **연결됨!** 표시합니다

연결되면 B1.church가 FreePlay 사이드바에 나타납니다. FreePlay는 B1 Admin에서 예약한 레슨 콘텐츠를 끌어와 교사가 교실 TV에서 직접 재생할 수 있습니다.

:::tip
연결 후 교사는 TV 리모컨을 사용하여 레슨이 진행됨에 따라 자신의 속도로 비디오와 슬라이드를 진행할 수 있습니다. 별도 설정이나 USB 드라이브가 필요하지 않습니다.
:::

:::info
콘텐츠 제공자로 인증하는 자세한 단계(QR 코드, 양식 로그인 등)는 [제공자에 연결](/docs/freeplay/content-providers/connecting-providers)을 참조합니다.
:::
