# 작업

<div class="article-intro">

작업은 교회의 개인 또는 그룹에 작업 항목을 배정할 수 있습니다. 일회성 할일이든 반복적인 책임이든 작업은 무엇을 해야 하는지, 누가 책임지는지를 추적하여 아무것도 놓치지 않도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 작업을 배정하려는 사람 또는 [그룹](../groups/creating-groups.md)이 B1 Admin에 존재해야 합니다.
- 작업 페이지에 접근하려면 **봉사** 섹션으로 이동하세요.

</div>

## 작업 보기

**봉사**로 이동하고 **작업** 페이지를 엽니다. 열린 작업 목록이 표시됩니다. 각 작업에는 제목, 담당자, 상태가 표시됩니다.

:::tip
기본적으로 열린 작업만 표시됩니다. **완료된 것 표시**를 토글하면 완료된 작업도 볼 수 있습니다.
:::

## 작업 만들기

1. 작업 페이지에서 **작업 추가**를 클릭합니다.
2. 작업의 **제목**을 입력합니다.
3. **담당자** 상자를 클릭하고 사람의 이름을 입력하거나, **그룹**으로 전환하고 그룹 이름을 입력합니다.
4. **검색**을 클릭하고 결과에서 사람 또는 그룹을 선택합니다.
5. 담당자를 위한 지침이나 세부 사항을 포함한 **메모**를 추가합니다.
6. **저장**을 클릭합니다.

작업이 이제 작업 목록에 나타나며 담당자의 대시보드 및 모바일 앱에서 볼 수 있게 됩니다.

## 작업 관리

작업을 클릭하여 상세 페이지를 엽니다. 여기에서 다음 작업을 할 수 있습니다.

- **상태 변경** -- 상태 드롭다운을 사용하여 **열기**와 **완료** 사이를 전환합니다.
- **작업 재배정** -- 담당자를 다른 사람 또는 그룹으로 업데이트합니다.
- **메모 추가** -- 담당자를 위한 추가 정보 또는 업데이트를 포함합니다.
- **제목 편집** -- 필요에 따라 작업 제목을 업데이트합니다.

## 작업 생명주기

1. 새 작업은 **열기** 상태로 시작합니다.
2. 담당자는 알림을 받고 대시보드, 작업 페이지 또는 모바일 앱에서 작업을 볼 수 있습니다.
3. 작업이 완료되면 담당자 또는 관리자가 상태를 **완료**로 변경합니다.
4. 완료된 작업은 기본 보기에서 제거되지만 **완료된 것 표시**를 활성화하면 여전히 접근할 수 있습니다.

:::info
반복 작업을 자동화하여 일정에 따라 만들어지도록 할 수 있습니다. 설정 방법은 [자동화](./automations.md)를 참고하세요.
:::

:::tip
많은 사람을 같은 일련의 단계로 이동시켜야 할 때(예: 새 방문자 팔로우업 또는 교인 절차) [워크플로우](./workflows.md)를 사용하세요. 워크플로우는 작업을 드래그 앤드 드롭 보드의 카드로 변환하고 단계, 마감일, 라우팅을 추가합니다.
:::
