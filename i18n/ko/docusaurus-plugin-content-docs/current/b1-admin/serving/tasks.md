# 작업

<div class="article-intro">

작업을 통해 교회 내 개인이나 그룹에 해야 할 일을 배정할 수 있습니다. 일회성 할 일이든 반복되는 업무이든, 작업은 무엇을 해야 하는지와 누가 담당하는지를 추적하여 누락되는 일이 없도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 작업을 배정하려는 사람이나 [그룹](../groups/creating-groups.md)이 B1 Admin에 존재하는지 확인하십시오
- **봉사** 섹션으로 이동하여 작업 페이지에 접근하십시오

</div>

## 작업 조회

**봉사**로 이동하여 **작업** 페이지를 엽니다. 진행 중인 작업 목록이 표시됩니다. 각 작업에는 제목, 담당자, 상태가 나타납니다.

:::tip
기본적으로 진행 중인 작업만 표시됩니다. **완료된 항목 표시**를 켜면 완료된 작업도 함께 볼 수 있습니다.
:::

## 작업 생성

1. 작업 페이지에서 **작업 추가**를 클릭합니다.
2. 작업의 **제목**을 입력합니다.
3. **담당자 지정** 상자를 클릭하고 사람 이름을 입력하거나, **그룹**으로 전환하여 그룹 이름을 입력합니다.
4. **검색**을 클릭하고 결과에서 해당 사람이나 그룹을 선택합니다.
5. 담당자를 위한 세부 정보나 지침을 **메모**에 추가합니다.
6. **저장**을 클릭합니다.

작업이 이제 작업 목록에 나타나며, 담당자의 대시보드와 모바일 앱에서도 확인할 수 있습니다.

## 작업 관리

작업을 클릭하여 상세 페이지를 엽니다. 여기서 다음을 수행할 수 있습니다:

- **상태 변경** -- 상태 드롭다운을 사용하여 **진행 중**과 **완료** 사이를 전환합니다.
- **작업 재배정** -- 담당자를 다른 사람이나 그룹으로 변경합니다.
- **메모 추가** -- 담당자를 위한 추가 정보나 업데이트를 포함합니다.
- **제목 편집** -- 필요에 따라 작업 제목을 수정합니다.

## 작업 수명주기

1. 새 작업은 **진행 중** 상태로 시작됩니다.
2. 담당자는 알림을 받고 대시보드, 작업 페이지 또는 모바일 앱에서 작업을 확인할 수 있습니다.
3. 업무가 완료되면 담당자(또는 관리자)가 상태를 **완료**로 변경합니다.
4. 완료된 작업은 기본 보기에서 사라지지만, **완료된 항목 표시**를 활성화하면 계속 접근할 수 있습니다.

:::info
반복 작업을 자동으로 일정에 따라 생성되도록 자동화할 수 있습니다. 설정 방법은 [자동화](./automations.md)를 참조하십시오.
:::
