# Service Order

<div class="article-intro">

서비스 순서는 예배의 요소 순서를 정의합니다. 시작 찬송곡부터 마감 기도까지 모든 것을 배치하여 전체 팀이 예배 흐름을 미리 알 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 서비스 순서를 빌드하려는 날짜에 대해 [service plan](./plans.md) 생성
- 선택적으로 라이브러리에 [songs](./songs.md)을 추가하여 서비스 순서에 직접 연결할 수 있습니다

</div>

## Accessing the Service Order

1. **Serving**으로 이동하고 ministry를 선택합니다.
2. **plan type**을 클릭한 후 특정 **plan**을 엽니다.
3. 계획 세부 페이지의 **Service Order** 탭을 클릭합니다.

## Building Your Service Order

서비스 순서는 예배 중에 발생할 순서대로 배열된 항목 목록입니다. 일반적인 요소:

- 예배 찬송곡
- 기도
- 성경 낭독
- 공지
- 설교 또는 메시지
- 봉헌
- 마감

## Adding Items to the Service Order

1. **Service Order** 탭에서 **Add Item**을 클릭합니다.
2. 추가할 요소의 유형을 선택합니다(예: 라이브러리의 곡, 사용자 정의 항목 또는 제목).
3. 제목 및 메모와 같은 항목의 세부 정보를 입력합니다.
4. **Save**를 클릭합니다.

:::tip
라이브러리에서 직접 곡을 추가할 수 있습니다. 이렇게 하면 가사와 편곡을 포함한 곡 세부 정보가 예배 계획에 연결되어 예배팀이 필요한 모든 것을 가지게 됩니다.
:::

## Reordering Items

각 행의 왼쪽 핸들로 항목을 드래그하여 순서를 재정렬합니다. 항목을 드롭할 파란색 선 또는 항목을 이동할 섹션 헤더에 드롭합니다. 화면에 표시된 순서가 팀이 예배 중에 따를 순서입니다.

## Timing and Media

각 항목과 섹션은 실행 시간과 시작 시간을 보여줍니다:

- 계획에 선택된 단일 예배 시간이 있으면 왼쪽 레일은 각 항목이 시작하는 실제 **clock time**(예: 9:14 AM)을 보여줍니다. 그렇지 않으면 예배 시작부터의 경과 시간을 보여줍니다.
- 섹션 헤더는 내부의 모든 실행 시간을 합산합니다.
- 곡과 사용자 정의 항목은 항목의 편집 화면에 **Minutes**/**Seconds**를 입력하면 이 일정에 포함됩니다.
- 미디어 라이브러리(이미지 또는 비디오 클립)에서 추가한 항목은 **thumbnail**을 표시하고 자동으로 시간이 지정됩니다. 비디오의 지속 시간은 파일 자체에서 측정되며, 이미지는 운영자가 진행할 때까지 화면에 남아 있으므로 약 5:00 계획 예상(이탤릭체로 표시)을 제공합니다. 여전히 둘 다 특정 지속 시간으로 편집할 수 있습니다.

## Editing and Removing Items

- 서비스 순서의 모든 항목을 클릭하여 세부 정보를 **edit**합니다.
- **delete** 옵션을 사용하여 순서에서 항목을 제거합니다.

:::info
서비스 순서는 각 계획별입니다. 여기서 변경한 사항은 선택한 계획에만 영향을 미치고 다른 계획이나 곡 라이브러리를 변경하지 않습니다.
:::

## Including Items in Specific Service Times

계획에 1개 이상의 예배 시간(예: 오전 8시 및 오전 10시 예배)이 있으면 각 항목이 표시되어야 하는 예배를 선택할 수 있습니다. 이는 공지가 한 예배에만 해당하거나 한 예배에서 곡을 부르지만 다른 예배에서는 부르지 않을 때 유용합니다.

1. **Times** 탭에 2개 이상의 예배 시간이 정의된 계획을 엽니다.
2. **Service Order** 탭에서 편집할 항목을 클릭합니다.
3. **Include in Services** 아래에 각 예배 시간에 대한 체크박스가 시간으로 표시되어 있습니다.
4. 항목을 건너뛰어야 하는 예배의 선택을 해제합니다.
5. **Save**를 클릭합니다.

기본적으로 모든 항목이 모든 예배에 포함됩니다. 제외된 예배 시간은 계획을 해당 예배로 필터링하여 인쇄할 때 숨겨져 각 예배가 관련 항목만 포함하는 깔끔한 실행 시트를 받습니다.

:::tip
**Include in Services** 섹션은 계획에 1개 이상의 예배 시간이 있을 때만 나타납니다. 한 예배만 보이면 계획의 **Times** 탭을 편집하여 먼저 추가 예배를 추가합니다.
:::

## Sharing the Service Order

서비스 순서가 완료되면 계획 세부 페이지에서 전체 계획(서비스 순서 포함)을 인쇄할 수 있습니다. 이렇게 하면 팀에 서비스의 전체 개요를 제공합니다.
