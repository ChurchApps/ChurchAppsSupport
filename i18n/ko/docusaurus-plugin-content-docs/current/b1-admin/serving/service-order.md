# 예배 순서

<div class="article-intro">

예배 순서는 예배의 각 요소 순서를 정의합니다. 시작 찬양부터 마무리 기도까지 모든 것을 배치하여, 팀 전체가 예배 흐름을 사전에 파악할 수 있도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 예배 순서를 구성할 날짜에 대한 [예배 계획을 생성](./plans.md)하십시오
- 선택적으로, 예배 순서에 직접 연결할 수 있도록 [곡](./songs.md)을 라이브러리에 추가하십시오

</div>

## 예배 순서 접근

1. **봉사**로 이동하여 사역부를 선택합니다.
2. **계획 유형**을 클릭한 다음, 특정 **계획**을 엽니다.
3. 계획 상세 페이지에서 **예배 순서** 탭을 클릭합니다.

## 예배 순서 구성

예배 순서는 예배 중 진행될 순서대로 배열된 항목 목록입니다. 일반적인 요소는 다음과 같습니다:

- 찬양
- 기도
- 성경 봉독
- 공지사항
- 설교 또는 메시지
- 헌금
- 마무리

## 예배 순서에 항목 추가

1. **예배 순서** 탭에서 **항목 추가**를 클릭합니다.
2. 추가할 요소 유형을 선택합니다 (예: 라이브러리의 곡, 사용자 정의 항목 또는 제목).
3. 제목 및 메모 등 항목의 세부 정보를 입력합니다.
4. **저장**을 클릭합니다.

:::tip
[곡 목록](./songs.md) 라이브러리에서 곡을 직접 추가할 수 있습니다. 이렇게 하면 가사와 편곡을 포함한 곡 정보가 예배 계획에 연결되어, 찬양팀이 필요한 모든 것을 갖추게 됩니다.
:::

## 항목 순서 변경

항목을 드래그 앤 드롭하여 순서를 변경합니다. 화면에 표시된 순서가 팀이 예배 중 따를 순서입니다.

## 항목 편집 및 삭제

- 예배 순서의 항목을 클릭하여 세부 정보를 **편집**합니다.
- **삭제** 옵션을 사용하여 순서에서 항목을 제거합니다.

:::info
예배 순서는 각 계획에 고유합니다. 여기서 변경한 내용은 선택한 계획에만 적용되며, 다른 계획이나 곡 라이브러리에는 영향을 주지 않습니다.
:::

## 예배 순서 공유

예배 순서가 완성되면, 계획 상세 페이지에서 전체 계획(예배 순서 포함)을 인쇄할 수 있습니다. 이를 통해 팀에게 예배의 전체 진행 순서를 제공합니다.
