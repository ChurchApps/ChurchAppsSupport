# 계획 템플릿

<div class="article-intro">

계획 템플릿을 사용하면 계획의 예배 순서 또는 직책의 재사용 가능한 스냅샷을 저장하고 한 번의 클릭으로 미래의 계획에 적용할 수 있습니다. 같은 구조를 매주 다시 만드는 대신 한 번 저장하고 필요한 만큼 많은 예정된 계획에 적용합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 예배 순서 또는 정의된 직책과 함께 최소 하나의 [서비스 계획](plans.md)을 만듭니다
- B1 Admin의 사역 영역에 접근해야 합니다

</div>

## 계획을 템플릿으로 저장

1. **Serving** 영역에서 기존 계획을 엽니다.
2. **Templates** 버튼(계획 페이지 상단)을 클릭합니다.
3. **Save as Template**을 선택합니다.
4. 템플릿에 이름을 지정합니다(예: "Standard Sunday" 또는 "Worship + Sermon").
5. 저장합니다. 템플릿은 현재 예배 순서와 위치를 캡처합니다.

## 템플릿 관리

모든 계획 유형 페이지의 **Templates** 버튼에서 다음을 할 수 있습니다:

- **Rename** -- 템플릿의 표시 이름을 변경합니다
- **Update from plan** -- 계획의 현재 상태로 템플릿의 내용을 바꿉니다(표준 구조를 개선한 후 유용함)
- **Apply** -- 하나 이상의 예정된 계획에 템플릿을 적용합니다
- **Delete** -- 더 이상 필요하지 않은 템플릿을 제거합니다

## 계획에 템플릿 적용

1. 계획 유형 페이지의 **Templates** 버튼을 클릭합니다.
2. 사용할 템플릿 옆의 **Apply** 아이콘(복사 아이콘)을 클릭합니다.
3. 상자를 확인하여 적용할 예정된 계획을 선택합니다.
4. 복사할 내용을 선택합니다:
   - **Order of Service** -- 선택한 계획의 예배 순서 항목을 템플릿의 항목으로 바꿉니다
   - **Positions** -- 선택한 계획의 직책(및 지정)을 템플릿의 직책으로 바꿉니다
5. **Apply**를 클릭합니다.

:::warning
직책을 적용하면 선택한 계획의 모든 기존 직책과 지정이 바뀝니다. 적용하기 전에 이미 자원봉사자 일정을 짜지 않았는지 확인하세요.
:::

:::tip
직책 없이 예배 순서를 적용할 때 일관된 찬양/설교 구조를 원하지만 자원봉사자가 매주 다르게 지정됩니다.
:::

## 템플릿을 최신 상태로 유지

표준 예배 구조가 변경되면 템플릿 관리자를 열고 **Update from plan**을 사용하여 처음부터 다시 만드는 대신 최신 계획에서 템플릿을 동기화합니다.
