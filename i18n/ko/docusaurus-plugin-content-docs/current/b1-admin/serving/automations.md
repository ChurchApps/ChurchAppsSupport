# 자동화

<div class="article-intro">

자동화는 반복 일정에 따라 작업을 자동으로 생성합니다. 매주 또는 매월 동일한 작업을 수동으로 생성하는 대신, 자동화를 한 번 설정하면 시스템이 알아서 처리하므로 누락되는 일이 없습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin에서 [작업](./tasks.md)이 어떻게 작동하는지 숙지하십시오
- 자동화하고 싶은 반복 업무를 파악하십시오

</div>

## 자동화 조회

**봉사**로 이동하여 **자동화** 페이지를 엽니다. 모든 자동화 목록이 표시되며, 각 항목에는 제목, 활성 상태, 반복 패턴이 나타납니다.

## 자동화 생성

1. 자동화 페이지에서 **자동화 추가**를 클릭합니다.
2. 자동화의 **제목**을 입력합니다. 이 제목은 생성되는 작업의 제목으로도 사용됩니다.
3. **반복 패턴**을 설정합니다 -- 작업이 생성될 빈도를 선택합니다 (예: 주간, 월간 또는 특정 요일).
4. 매번 실행될 때 담당할 **개인 또는 그룹**에 작업을 배정합니다.
5. 일정에 따라 작업 생성이 시작되도록 자동화를 **활성**으로 설정합니다.
6. **저장**을 클릭합니다.

## 자동화 편집

1. 목록에서 기존 자동화를 클릭하여 엽니다.
2. 제목, 반복 패턴, 배정 또는 기타 설정을 업데이트합니다.
3. **저장**을 클릭하여 변경 사항을 적용합니다.

## 활성화 및 비활성화

자동화를 삭제하지 않고도 실행 여부를 제어할 수 있습니다:

- **활성** -- 자동화가 일정에 따라 작업을 생성합니다.
- **비활성** -- 자동화가 일시 중지되어 재활성화할 때까지 새 작업을 생성하지 않습니다.

:::tip
휴일이나 특별 시즌 동안 반복 작업이 필요하지 않을 때 자동화를 **비활성**으로 설정하십시오. 설정을 잃지 않고 언제든지 재활성화할 수 있습니다.
:::

## 작동 방식

자동화가 실행되면, 설정된 제목과 배정으로 새 [작업](./tasks.md)이 생성됩니다. 이 작업은 다른 작업과 동일하게 작동합니다 -- 담당자는 알림을 받고 대시보드나 작업 페이지에서 작업을 관리할 수 있습니다.

:::info
자동화는 앞으로의 새 작업만 생성합니다. 과거 날짜에 대해 소급하여 작업을 생성하지 않습니다.
:::
