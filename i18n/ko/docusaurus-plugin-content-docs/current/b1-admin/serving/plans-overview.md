# 계획 개요

<div class="article-intro">

계획 개요는 한 번에 여러 예배 날짜 전체의 모든 자원봉사자 할당을 조감도로 제공합니다. 각 계획을 개별적으로 열기 대신 한 표에서 예정된 주 전체의 모든 위치에서 누가 봉사하는지 볼 수 있으며 채워야 할 공백을 빠르게 파악할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 봉사 영역에서 최소한 하나의 사역 및 계획 유형 만들기
- [예배 계획](./plans.md)을 날짜 및 자원봉사자 할당으로 만들기
- 자원봉사자가 [사람 디렉터리](../people/adding-people.md)에 추가되어 있는지 확인

</div>

## 개요에 액세스

1. B1 Admin의 메인 메뉴에서 **봉사**로 이동합니다.
2. 페이지 상단에서 **사역 탭**을 선택합니다.
3. **계획 유형**을 클릭하여 계획 목록을 엽니다.
4. 페이지 상단 근처의 **개요** 버튼을 클릭합니다.

## 개요 표 읽기

개요는 다음 표를 표시합니다:

- **행**은 각 위치를 나타냅니다(예: "음악: 기타", "기술: 프로젝션"), 범주별로 그룹화됨
- **열**은 예정된 예배 날짜를 나타냅니다(예: "4월 14일", "4월 21일")
- **셀**은 해당 날짜의 해당 위치에 할당된 자원봉사자의 이름을 보여줍니다

**빨간색**으로 강조된 위치는 채워지지 않습니다 — 자원봉사자가 아직 할당되지 않았습니다. 이렇게 하면 각 계획을 개별적으로 열지 않고도 한눈에 인원 배치 공백을 쉽게 볼 수 있습니다.

:::tip
자원봉사자 이름은 표가 많은 위치가 있을 때 간결하게 유지하기 위해 축약 형식(이름과 성 첫 글자, 예: "John D.")으로 표시됩니다.
:::

## 개요 필터링

상단의 필터 제어를 사용하여 개요가 표시하는 항목을 조정할 수 있습니다:

- **시작 날짜 / 종료 날짜** — 기본적으로 개요는 앞으로 12주를 표시합니다. 사용자 정의 날짜를 입력하여 범위를 확대하거나 좁힙니다.
- **사역** — 개요를 벗어나지 않고 다른 사역으로 전환합니다.
- **계획 유형** — 선택한 사역 내에서 특정 계획 유형으로 필터링합니다.

변경 후 **필터**를 클릭하여 표를 업데이트합니다.

## CSV로 내보내기

**CSV 내보내기**를 클릭하여 현재 표를 스프레드시트로 다운로드합니다. 내보내기에는 필터링된 날짜 범위의 모든 위치 및 자원봉사자 할당이 포함되어 사역 지도자와 공유하거나 계획 회의를 위해 인쇄하기가 쉽습니다.

:::info
CSV 내보내기는 현재 적용된 필터를 반영합니다 — 표에 표시되는 날짜 및 계획 유형만 다운로드에 포함됩니다.
:::
