# 계획 개요

<div class="article-intro">

계획 개요는 여러 예배 날짜에 걸친 모든 자원봉사자 지정의 새 보기를 제공합니다. 각 계획을 개별적으로 열지 않고도 한 눈에 여러 주에 걸친 모든 직책에서 누가 사역하는지 볼 수 있으며, 아직 채워야 할 공석을 빠르게 찾을 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 사역 영역에서 최소 하나의 사역 및 계획 유형을 만듭니다
- 날짜 및 자원봉사자 지정과 함께 [서비스 계획](./plans.md)을 만듭니다
- 자원봉사자가 [사람 디렉토리](../people/adding-people.md)에 추가되었는지 확인합니다

</div>

## 개요에 접근하기

1. B1 Admin의 주 메뉴에서 **Serving**으로 이동합니다.
2. 페이지 상단의 **사역 탭**을 선택합니다.
3. 계획 목록을 열 **계획 유형**을 클릭합니다.
4. 페이지 상단 근처의 **Overview** 버튼을 클릭합니다.

## 개요 그리드 읽기

개요는 다음과 같은 그리드를 표시합니다:

- **행**은 각 직책(예: "Music: Guitar", "Tech: Projection")을 카테고리별로 그룹화하여 나타냅니다
- **열**은 예정된 예배 날짜(예: "Apr 14", "Apr 21")를 나타냅니다
- **셀**은 해당 날짜에 그 직책에 지정된 자원봉사자의 이름을 보여줍니다

**빨강색**으로 강조된 셀은 채워지지 않았습니다 -- 자원봉사자가 아직 지정되지 않았습니다. 이렇게 하면 각 계획을 개별적으로 열지 않고도 인력 공석을 한눈에 쉽게 볼 수 있습니다.

:::tip
자원봉사자 이름은 단축 형식(이름 및 성의 첫 글자, 예: "John D.")으로 표시되어 많은 직책이 있을 때 그리드를 컴팩트하게 유지합니다.
:::

## 개요에서 직접 자원봉사자 지정

개별 계획을 열 필요가 없습니다. 그리드의 모든 셀을 클릭하여 해당 직책과 날짜에 대한 지정 패널을 엽니다. 그곳에서 다음을 할 수 있습니다:

- 팀에서 사람을 선택하여 직책에 지정합니다
- 기존 지정을 바꾸거나 제거합니다
- 개요를 떠나지 않고 변경을 저장합니다

이렇게 하면 한 번의 패스에서 전체 일정을 구성할 수 있습니다 -- 개별 계획으로 들어왔다 나갔다 하지 않고 여러 주와 직책에 걸쳐 작업합니다.

## 개요 필터링

상단의 필터 컨트롤을 사용하여 개요에 표시되는 내용을 조정할 수 있습니다:

- **Start Date / End Date** -- 기본적으로 개요는 미래 12주를 표시합니다. 범위를 확대하거나 좁히려면 사용자 지정 날짜를 입력합니다.
- **Ministry** -- 개요를 떠나지 않고 다른 사역으로 전환합니다.
- **Plan Type** -- 선택한 사역 내의 특정 계획 유형으로 필터링합니다.
- **Gaps Only** -- 토글하여 모든 날짜가 이미 채워진 행을 숨기고 아직도 자원봉사자가 필요한 직책에만 집중합니다.

변경한 후 **Filter**를 클릭하여 그리드를 업데이트합니다.

## CSV로 내보내기

**Export CSV**를 클릭하여 현재 그리드를 스프레드시트로 다운로드합니다. 내보내기에는 필터링된 날짜 범위의 모든 직책과 자원봉사자 지정이 포함되어 있어 사역 리더와 공유하거나 계획 회의를 위해 인쇄하기 쉽습니다.

:::info
CSV 내보내기는 현재 적용되는 필터를 반영합니다 -- 그리드에 표시되는 날짜와 계획 유형만 다운로드에 포함됩니다.
:::
