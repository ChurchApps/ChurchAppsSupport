# 예배 계획

<div class="article-intro">

예배 계획은 누가 봉사하고 언제 하는지 정리합니다. 각 계획은 특정 날짜와 사역에 연결되어 주간 단위로 자원봉사자 팀을 쉽게 조정하고 모든 예배가 충분한 인원을 갖추도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 봉사 영역에서 사역 및 팀 설정
- 자원봉사자가 [사람 디렉터리](../people/adding-people.md)에 추가되었는지 확인하고 팀에 할당됨

</div>

## 계획에 액세스

1. 메인 메뉴에서 **봉사**로 이동합니다.
2. 페이지 상단에서 **사역 탭**을 선택합니다.
3. **계획 유형**을 클릭하여 해당 유형의 계획 목록을 봅니다.
4. 특정 계획을 클릭하여 엽니다.

## 계획 만들기

1. 계획 유형 보기에서 **새 계획**을 클릭합니다.
2. 계획 이름을 지정하거나 날짜를 이름으로 사용합니다. 예배의 **날짜**를 선택합니다.
3. 이전 계획에서 복사하려면 위치만 또는 위치 및 할당만 선택합니다. 복사하지 않으려면 아무것도 선택하지 마세요. 이전 계획에서 예배순서를 복사할 수도 있습니다.
4. 계획을 저장합니다. 이제 팀 회원을 할당하고 [예배순서](./service-order.md)를 만들 수 있습니다.

## 계획 세부 페이지

계획을 열면 두 개의 탭이 표시됩니다:

- **할당** -- 이 계획에 할당된 팀 회원을 관리합니다. 기존 팀에서 사람을 추가하고 누가 확정하거나 아직 대기 중인지 볼 수 있습니다.
- **[예배순서](./service-order.md)** -- 경배 노래, 기도, 공지사항, 설교 같은 요소로 예배순서를 구축합니다.

## 팀 회원 할당

1. 계획을 열고 **할당** 탭으로 이동합니다.
2. **위치 추가**를 클릭하여 확장합니다. 위치 추가 양식의 정보를 입력합니다. 범주 이름으로 원하는 범주를 추가합니다.
3. **필요한 사람**을 클릭하고 해당 위치를 채울 자원봉사자를 선택합니다.
4. **추가**를 클릭하여 팀 명부에서 회원을 추가합니다.
5. 할당된 회원이 할당 상태와 함께 팀 아래에 나타납니다.
6. 자원봉사자에게 알림을 클릭하여 B1 앱 또는 이메일로 알립니다.

:::tip
계획을 만들기 전에 사역 설정에서 팀을 설정합니다. 이렇게 하면 할당할 자원봉사자 풀이 준비됩니다.
:::

## 계획 인쇄

팀에 배포할 계획을 인쇄할 수 있습니다. 계획을 열고, 예배순서 탭을 열고, **인쇄** 옵션을 사용하여 할당과 예배순서를 포함하는 인쇄 가능한 버전을 생성합니다. 리허설에서 나눠주거나 공용 영역에 게시하는 데 유용합니다.

:::info
계획은 사역별로 구성됩니다. 계획을 만들거나 보기 전에 올바른 사역 탭에 있는지 확인하세요.
:::
