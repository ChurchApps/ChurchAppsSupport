# 예배 계획

<div class="article-intro">

예배 계획은 누가 언제 봉사하는지를 조직합니다. 각 계획은 특정 날짜와 사역에 연결되어 있어서 주마다 자원봉사 팀을 조정하고 모든 예배가 완벽하게 인력을 배치할 수 있도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 봉사 영역에서 사역과 팀 설정
- 자원봉사자들이 [사람 디렉토리](../people/adding-people.md)에 추가되고 팀에 할당되었는지 확인

</div>

## 계획 접근

1. 메인 메뉴에서 **봉사**로 이동합니다.
2. 페이지 상단의 **사역 탭**을 선택합니다.
3. **계획 유형**을 클릭하여 그 유형의 계획 목록을 봅니다.
4. 특정 계획을 클릭하여 엽니다.

:::info
전체 관리자 접근 권한이 계획을 보는 데 필요하지 않습니다. 사역의 일부인 사역 구성원들은 봉사로 이동하여 관리자 권한이 없어도 자신의 사역에 대한 계획을 볼 수 있습니다.
:::

## 계획 만들기

1. 계획 유형 보기에서 **새 계획**을 클릭합니다.
2. 계획에 이름을 지정하거나 날짜를 이름으로 사용합니다. 예배의 **날짜**를 선택합니다.
3. 이전 계획에서 복사하고 싶으면 직책만 또는 직책과 할당을 선택합니다. 복사하지 않으려면 아무것도 선택하지 않으면 됩니다. 이전 계획에서 예배 순서를 복사할 수도 있습니다.
4. 계획을 저장합니다. 이제 팀 구성원을 할당하고 [예배 순서](./service-order.md)를 작성하기 시작할 수 있습니다.

## 계획 상세 페이지

계획을 열면 두 개의 탭이 표시됩니다:

- **할당** -- 이 계획에 할당된 팀 구성원을 관리합니다. 기존 팀에서 사람을 추가할 수 있으며 누가 확인했거나 아직 보류 중인지 볼 수 있습니다.
- **[예배 순서](./service-order.md)** -- 찬양, 기도, 공지 및 설교와 같은 요소로 예배 순서를 작성합니다.

## 팀 구성원 할당

1. 계획을 열고 **할당** 탭으로 이동합니다.
2. **직책 추가**를 클릭하여 확장합니다. 직책 추가 양식에 정보를 입력합니다. 카테고리 이름에 원하는 카테고리를 추가합니다.
3. **필요한 사람들**을 클릭하여 그 직책을 채울 자원봉사자를 선택합니다.
4. **추가**를 클릭하여 팀 명단에서 구성원을 추가합니다.
5. 할당된 구성원들은 자신의 팀 아래에 할당 상태와 함께 나타납니다.
6. 자원봉사자에게 알림을 클릭하여 B1 앱 또는 이메일을 통해 알립니다.

:::tip
계획을 만들기 전에 사역 설정에서 팀을 설정합니다. 이렇게 하면 할당할 자원봉사자 풀이 준비됩니다.
:::

## 계획 인쇄

팀 배포용 계획을 인쇄할 수 있습니다. 계획을 열고 예배 순서 탭을 열고 **인쇄** 옵션을 사용하여 할당과 예배 순서를 포함하는 인쇄 가능한 버전을 생성합니다. 이는 연습이나 공용 영역에 게시할 때 배포하기에 유용합니다.

:::info
계획은 사역별로 조직됩니다. 계획을 만들거나 보기 전에 올바른 사역 탭에 있는지 확인합니다.
:::
