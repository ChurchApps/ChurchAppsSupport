# 예배 계획

<div class="article-intro">

예배 계획은 누가 언제 봉사하는지를 정리합니다. 각 계획은 특정 날짜와 사역에 연결되어 주 단위로 자원봉사자 팀을 쉽게 조율하고 모든 예배가 충분히 인원이 배치되도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 봉사 영역에서 사역과 팀을 설정하세요
- 자원봉사자들이 [사람 디렉토리](../people/adding-people.md)에 추가되고 팀에 배치되었는지 확인하세요

</div>

## 계획 접근하기

1. 메인 메뉴에서 **Serving**으로 이동하세요.
2. 페이지 상단의 **ministry tab**을 선택하세요.
3. **plan type**을 클릭하여 해당 유형의 계획 목록을 보세요.
4. 특정 계획을 클릭하여 열으세요.

:::info
계획을 관리하기 위해 전체 관리자 접근 권한이 필요하지 않습니다. 사역의 구성원인 누구든지 봉사 영역으로 이동하여 Plans Edit 권한이 없어도 자신의 사역에 대한 계획을 만들고, 편집하고, 일정을 짤 수 있습니다. Plans Edit 역할이 있는 편집자는 모든 사역의 계획을 관리할 수 있습니다.
:::

## 계획 만들기

1. 계획 유형 보기에서 **New Plan**을 클릭하세요.
2. 계획에 이름을 지정하거나 날짜를 이름으로 사용하세요. 예배의 **date**을 선택하세요.
3. 이전 계획에서 복사하려면 직책만 또는 직책과 배치를 선택하세요. 복사하지 않으려면 아무것도 선택하지 마세요. 이전 계획에서 예배 순서도 복사할 수 있습니다.
4. 계획을 저장하세요. 이제 팀 구성원을 배치하고 [예배 순서](./service-order.md)를 구성할 수 있습니다.

## 계획 상세 페이지

계획을 열면 두 개의 탭이 표시됩니다:

- **Assignments** -- 이 계획에 배치된 팀 구성원을 관리하세요. 기존 팀에서 사람을 추가하고 누가 확인했거나 여전히 대기 중인지 볼 수 있습니다.
- **[Service Order](./service-order.md)** -- 찬양곡, 기도, 공지사항, 설교 등의 요소로 예배 순서를 구성하세요.

## 팀 구성원 배치하기

1. 계획을 열고 **Assignments** 탭으로 이동하세요.
2. **add Position**을 클릭하여 확장하세요. 직책 추가 양식의 정보를 작성하세요. 카테고리 이름의 경우 원하는 카테고리를 추가하세요.
3. **People Needed**를 클릭하고 그 직책을 채울 자원봉사자를 선택하세요.
4. **Add**를 클릭하여 팀 명단에서 구성원을 추가하세요.
5. 배치된 구성원은 해당 팀 아래에 배치 상태와 함께 나타납니다.
6. 자원봉사자에게 알림을 클릭하여 B1 앱 또는 이메일을 통해 알리세요.

:::tip
계획을 만들기 전에 사역 설정에서 팀을 설정하세요. 이렇게 하면 배치할 자원봉사자의 준비된 풀을 갖게 됩니다.
:::

## 계획 유형과 그룹 연결하기

계획 유형 페이지의 계획 목록 아래에 있는 **Groups** 섹션을 통해 어느 그룹이 이 계획 유형의 계획을 회원 포털에서 볼 수 있는지 결정할 수 있습니다. 이것은 관리자 접근 권한을 주지 않고도 적절한 팀에 예정된 예배를 표시하는 빠른 방법입니다.

1. 계획 유형 페이지에서 **Groups** 섹션까지 스크롤하세요.
2. **Add Group**을 클릭하고 드롭다운에서 그룹을 선택하세요.
3. **Shows** 열에서 그 그룹의 구성원이 이 계획 유형의 **Past**, **Future**, 또는 **Both** 계획을 볼지 선택하세요.
4. 추가 그룹을 연결하려면 반복하거나 쓰레기통 아이콘을 클릭하여 그룹을 제거하세요.

:::info
**Standard**로 태그된 그룹만 선택기에 나타납니다. 연결된 그룹의 구성원은 자동으로 B1 회원 포털의 [Plans](/docs/b1-church/plans/) 탭에서 이 계획 유형의 계획을 보게 됩니다 -- 선택한 과거/미래/둘 다 창으로 제한됩니다.
:::

## 계획 인쇄하기

팀에 배포할 계획을 인쇄할 수 있습니다. 계획을 열고 예배 순서 탭을 열고 **Print** 옵션을 사용하여 배치와 예배 순서를 포함한 인쇄 가능한 버전을 생성하세요. 이것은 리허설에서 나눠주거나 공용 영역에 게시하기에 유용합니다.

:::info
계획은 사역별로 정리됩니다. 계획을 만들거나 볼 때 올바른 사역 탭에 있는지 확인하세요.
:::
