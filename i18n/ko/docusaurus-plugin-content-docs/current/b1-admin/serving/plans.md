# 예배 계획

<div class="article-intro">

예배 계획은 누가 언제 봉사하는지를 정리합니다. 각 계획은 특정 날짜와 사역부에 연결되어 있어, 매주 자원봉사 팀을 조율하고 모든 예배에 충분한 인력을 배치하기 쉽습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 설정 영역에서 사역부와 팀을 구성하십시오
- 자원봉사자가 [인원 디렉토리](../people/adding-people.md)에 추가되고 팀에 배정되어 있는지 확인하십시오

</div>

## 계획 접근

1. 메인 메뉴에서 **봉사**로 이동합니다.
2. 페이지 상단에서 **사역부 탭**을 선택합니다.
3. **계획 유형**을 클릭하여 해당 유형의 계획 목록을 확인합니다.
4. 특정 계획을 클릭하여 엽니다.

## 계획 생성

1. 계획 유형 보기에서 **계획 추가**를 클릭합니다.
2. 예배 **날짜**를 선택합니다.
3. 계획을 저장합니다. 이제 팀원을 배정하고 [예배 순서](./service-order.md)를 구성할 수 있습니다.

## 계획 상세 페이지

계획을 열면 두 개의 탭이 표시됩니다:

- **배정** -- 이 계획에 배정된 팀원을 관리합니다. 기존 팀에서 인원을 추가하고, 누가 확인했는지 또는 아직 대기 중인지 확인할 수 있습니다.
- **[예배 순서](./service-order.md)** -- 찬양, 기도, 공지사항, 설교 등의 요소로 예배 순서를 구성합니다.

## 팀원 배정

1. 계획을 열고 **배정** 탭으로 이동합니다.
2. **팀**을 클릭하여 펼칩니다.
3. **추가**를 클릭하여 팀 명단에서 구성원을 추가합니다.
4. 배정된 구성원은 해당 팀 아래에 배정 상태와 함께 표시됩니다.

:::tip
계획을 생성하기 전에 사역부 설정에서 팀을 구성하십시오. 이렇게 하면 배정할 수 있는 자원봉사자 풀이 미리 준비됩니다.
:::

## 계획 인쇄

팀에 배포하기 위해 계획을 인쇄할 수 있습니다. 계획을 열고 **인쇄** 옵션을 사용하여 배정과 예배 순서가 포함된 인쇄 가능한 버전을 생성하십시오. 리허설 시 배포하거나 공용 공간에 게시할 때 유용합니다.

:::info
계획은 사역부별로 구성됩니다. 계획을 생성하거나 조회하기 전에 올바른 사역부 탭에 있는지 확인하십시오.
:::
