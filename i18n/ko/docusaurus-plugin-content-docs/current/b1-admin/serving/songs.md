# 찬양

<div class="article-intro">

찬양 페이지는 예배 찬양 도서관을 관리합니다. 모든 찬양 세부사항, 가사 및 편곡을 한 곳에 보관하여 예배팀이 예정된 예배를 쉽게 준비하고 일관된 레퍼토리를 유지할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin의 **Serving** 섹션으로 이동하여 찬양 페이지에 접근합니다
- 선택적으로, [예배 순서](./service-order.md)에 찬양을 직접 연결할 수 있도록 [서비스 계획](./plans.md)을 설정합니다

</div>

## 찬양 도서관 탐색하기

**Songs** 페이지를 열면 찬양이 카드로 표시됩니다. 각 카드에는 찬양의 썸네일 이미지, 제목, 아티스트 및 길이가 표시됩니다. 도서관을 스크롤하거나 검색을 사용하여 필요한 것을 찾습니다.

## 찬양 검색

1. 찬양 페이지에서 **Search** 버튼을 클릭합니다.
2. 찬양 **제목** 또는 **아티스트** 이름을 입력합니다.
3. 입력할 때 결과가 필터링되어 도서관의 일치하는 찬양을 보여줍니다.

## 찬양 추가

1. 찬양 페이지에서 **Add Song**을 클릭합니다.
2. 제목이나 아티스트별로 외부 찬양 데이터베이스를 검색합니다.
3. 추가할 찬양을 선택합니다.
4. 찬양이 기본 세부사항이 미리 채워진 도서관에 추가됩니다.

:::tip
찬양을 추가한 후 클릭하여 세부사항을 검토하고 업데이트하며, 편곡을 추가하거나 YouTube 비디오나 화음 차트와 같은 외부 링크를 첨부합니다.
:::

## 찬양 보기 및 편집

모든 찬양 카드를 클릭하여 세부사항 페이지를 엽니다. 여기서 다음을 할 수 있습니다:

- **찬양 세부사항 편집** -- 제목, 아티스트, 길이 및 기타 메타데이터를 업데이트합니다.
- **편곡 관리** -- 각 찬양은 여러 편곡(예: 다양한 키 또는 버전)을 가질 수 있습니다. 필요에 따라 편곡을 추가, 편집 또는 제거합니다.
- **가사 추가** -- 각 편곡의 가사를 입력하거나 업데이트합니다.
- **키 설정** -- 각 편곡의 음악 키를 지정하여 팀이 어떻게 준비해야 하는지 알 수 있습니다.
- **외부 링크 추가** -- YouTube 비디오, 화음 차트 또는 다른 곳에서 호스팅되는 악보와 같은 리소스에 링크를 첨부합니다.

## 편곡

편곡은 찬양의 서로 다른 버전을 나타냅니다. 예를 들어, 더 빠른 현대 버전과 더 느린 어쿠스틱 버전, 또는 다른 팀을 위해 다른 키의 같은 찬양입니다. 같은 찬양의 각 편곡은 찬양 제목과 아티스트를 공유하지만 자신의 독립적인 설정을 가집니다:

- **Key** -- 편곡이 연주되는 음악 키
- **BPM** -- 분당 박자 수로 계산된 템포
- **Length** -- 초 단위의 길이
- **Meter** -- 박자표(예: 4/4, 3/4, 6/8)
- **Sequence** -- 화음/섹션 구조(예: Intro → Verse → Chorus → Bridge)
- **Lyrics** -- 표시 또는 프로젝션용
- **External links** -- 화음 차트, YouTube 녹음, 악보 또는 기타 리소스

[Service Order](./service-order.md)에 찬양을 추가할 때, 특정 예배에 사용할 편곡을 선택할 수 있으므로 팀이 항상 올바른 버전을 리허설하고 연주합니다.

:::info
도서관의 찬양은 계획 내의 [Service Order](./service-order.md)에 직접 추가되어 찬양 세부사항과 선택한 편곡을 특정 예배에 연결할 수 있습니다.
:::
