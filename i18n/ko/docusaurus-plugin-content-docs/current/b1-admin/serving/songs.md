# 곡 목록

<div class="article-intro">

곡 목록 페이지는 찬양 곡 라이브러리를 관리합니다. 모든 곡의 세부 정보, 가사, 편곡을 한 곳에서 관리하여, 찬양팀이 다가오는 예배를 쉽게 준비하고 일관된 레퍼토리를 유지할 수 있도록 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin의 **봉사** 섹션으로 이동하여 곡 목록 페이지에 접근하십시오
- 선택적으로, [예배 계획](./plans.md)을 설정하여 곡을 [예배 순서](./service-order.md)에 직접 연결할 수 있습니다

</div>

## 곡 라이브러리 탐색

**곡 목록** 페이지를 열면, 곡이 카드 형태로 표시됩니다. 각 카드에는 곡의 썸네일 이미지, 제목, 아티스트, 길이가 나타납니다. 라이브러리를 스크롤하거나 검색을 사용하여 필요한 곡을 찾으십시오.

## 곡 검색

1. 곡 목록 페이지에서 **검색** 버튼을 클릭합니다.
2. 곡 **제목** 또는 **아티스트** 이름을 입력합니다.
3. 입력하는 대로 결과가 필터링되어 라이브러리에서 일치하는 곡이 표시됩니다.

## 곡 추가

1. 곡 목록 페이지에서 **곡 추가**를 클릭합니다.
2. 제목 또는 아티스트로 외부 곡 데이터베이스를 검색합니다.
3. 추가하려는 곡을 선택합니다.
4. 기본 세부 정보가 미리 입력된 상태로 곡이 라이브러리에 추가됩니다.

:::tip
곡을 추가한 후 클릭하여 세부 정보를 검토하고 업데이트하거나, 편곡을 추가하거나, YouTube 동영상이나 코드 차트 같은 외부 링크를 첨부하십시오.
:::

## 곡 조회 및 편집

곡 카드를 클릭하여 상세 페이지를 엽니다. 여기서 다음을 수행할 수 있습니다:

- **곡 세부 정보 편집** -- 제목, 아티스트, 길이 및 기타 메타데이터를 업데이트합니다.
- **편곡 관리** -- 각 곡에는 여러 편곡이 있을 수 있습니다 (예: 다른 키 또는 버전). 필요에 따라 편곡을 추가, 편집 또는 삭제합니다.
- **가사 추가** -- 각 편곡의 가사를 입력하거나 업데이트합니다.
- **키 설정** -- 각 편곡의 음악 키를 지정하여 팀이 준비할 수 있도록 합니다.
- **외부 링크 추가** -- YouTube 동영상, 코드 차트 또는 외부에 호스팅된 악보 등의 리소스 링크를 첨부합니다.

## 편곡

하나의 곡에 다양한 예배나 팀 구성에 맞는 여러 편곡이 있을 수 있습니다. 각 편곡에는 다음이 포함됩니다:

- 표시 또는 프로젝션용 **가사**
- **키** 지정
- 추가 리소스에 대한 **외부 링크**

:::info
라이브러리의 곡은 계획 내 [예배 순서](./service-order.md)에 직접 추가할 수 있으며, 곡의 세부 정보와 편곡이 해당 예배에 연결됩니다.
:::
