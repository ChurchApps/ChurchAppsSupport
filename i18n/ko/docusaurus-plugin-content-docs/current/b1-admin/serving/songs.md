# Songs

<div class="article-intro">

Songs 페이지는 예배 곡 라이브러리를 관리합니다. 모든 곡 세부 정보, 가사 및 편곡을 한 곳에 보관하여 예배팀이 다가오는 예배를 쉽게 준비하고 일관된 레퍼토리를 유지할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin의 **Serving** 섹션으로 이동하여 Songs 페이지에 접근합니다
- 선택적으로 곡을 [service order](./service-order.md)에 직접 연결할 수 있도록 [service plans](./plans.md)을 설정합니다

</div>

## Browsing Your Song Library

**Songs** 페이지를 열면 곡이 카드로 표시됩니다. 각 카드는 곡의 썸네일 이미지, 제목, 아티스트 및 지속 시간을 표시합니다. 라이브러리를 스크롤하거나 검색을 사용하여 필요한 것을 찾습니다.

## Searching for Songs

1. Songs 페이지에서 **Search** 버튼을 클릭합니다.
2. 곡 **title** 또는 **artist** 이름을 입력합니다.
3. 입력하면서 결과가 필터링되어 라이브러리의 일치하는 곡을 표시합니다.

## Adding a Song

1. Songs 페이지에서 **Add Song**을 클릭합니다.
2. 제목 또는 아티스트별로 외부 곡 데이터베이스를 검색합니다.
3. 추가할 곡을 선택합니다.
4. 기본 세부 정보가 미리 입력된 라이브러리에 곡이 추가됩니다.

:::tip
곡을 추가한 후 클릭하여 세부 정보를 검토 및 업데이트하고, 편곡을 추가하거나, YouTube 비디오 또는 코드 차트와 같은 외부 링크를 첨부합니다.
:::

## Viewing and Editing a Song

모든 곡 카드를 클릭하여 세부 페이지를 엽니다. 여기서 다음을 수행할 수 있습니다:

- **곡 세부 정보 편집** -- 제목, 아티스트, 지속 시간 및 기타 메타데이터를 업데이트합니다.
- **편곡 관리** -- 각 곡은 여러 편곡(예: 다른 키 또는 버전)을 가질 수 있습니다. 필요에 따라 편곡을 추가, 편집 또는 제거합니다.
- **가사 추가** -- 각 편곡의 가사를 입력하거나 업데이트합니다.
- **키 설정** -- 가사 옆의 키 선택기를 사용하여 각 편곡의 음악 키를 지정합니다. 키를 변경하면 표시된 코드가 즉시 전치되며, 편곡 페이지와 서비스 계획에서 곡을 열 때 모두 팀이 실제로 연주하는 키의 코드를 항상 봅니다.
- **외부 링크 추가** -- YouTube 비디오, 코드 차트 또는 다른 곳에서 호스팅되는 악보와 같은 리소스에 대한 링크를 첨부합니다.

## Arrangements

편곡은 곡의 별개 버전을 나타냅니다. 예를 들어 더 빠른 현대식 버전 대 느린 어쿠스틱 테이크 또는 다른 팀을 위해 다른 키의 동일한 곡입니다. 같은 곡의 각 편곡은 곡 제목과 아티스트를 공유하지만 자체 독립적인 설정을 가집니다:

- **Key** -- 편곡이 수행되는 음악 키
- **BPM** -- 분당 비트 수의 템포
- **Length** -- 초 단위의 지속 시간
- **Meter** -- 시간 서명(예: 4/4, 3/4, 6/8)
- **Sequence** -- 코드/섹션 구조(예: Intro → Verse → Chorus → Bridge)
- **Lyrics** -- 디스플레이 또는 프로젝션용
- **External links** -- 코드 차트, YouTube 녹음, 악보 또는 기타 리소스

[Service Order](./service-order.md)에 곡을 추가하면 해당 특정 예배에 사용할 편곡을 선택할 수 있으므로 팀은 항상 올바른 버전을 연습하고 연주합니다.

:::info
라이브러리의 곡은 계획 내 [Service Order](./service-order.md)에 직접 추가될 수 있으므로 곡 세부 정보와 선택한 편곡이 해당 특정 예배에 연결됩니다.
:::
