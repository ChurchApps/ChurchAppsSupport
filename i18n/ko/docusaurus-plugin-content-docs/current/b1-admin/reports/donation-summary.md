# 헌금 요약

<div class="article-intro">

헌금 요약 보고서는 교회의 헌금 활동에 대한 명확한 그림을 제공합니다. 헌금 합계를 추적하고, 헌금 추세를 모니터링하며, 리더십 팀을 위한 재정 보고서를 준비하는 데 활용하십시오.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 시스템에 헌금이 [일관되게 기록](../donations/recording-donations.md)되고 있는지 확인하십시오
- 재정 보고서에 접근하려면 적절한 [권한](../settings/roles-permissions.md)이 필요합니다

</div>

## 보고서 실행

1. 메인 메뉴에서 **보고서**로 이동합니다.
2. **헌금 요약** 카드를 클릭합니다.
3. **날짜 범위**를 설정하여 보고 기간을 정의합니다.
4. **기금** 등의 추가 필터를 적용하여 결과를 좁힙니다.
5. 선택한 기간의 헌금 요약이 보고서에 표시됩니다.

## 보고서에 표시되는 내용

헌금 요약 보고서에는 다음이 포함됩니다:

- **총 헌금액** -- 선택한 기간의 전체 헌금 금액입니다.
- **헌금자 수** -- 기여한 고유 헌금자의 수입니다.
- **기금별 금액** -- 해당되는 경우, 다양한 기금에 대한 헌금 내역입니다.

## 실용적 활용

- **재정 계획** -- 헌금 추세를 검토하여 미래 수입을 예측하고 예산을 계획합니다.
- **이사회 보고** -- 명확한 헌금 데이터로 정기 이사회 또는 리더십 회의용 요약을 생성합니다.
- **연말 결산** -- 연간 전체 보고서를 실행하여 연간 재정 요약을 준비합니다.
- **캠페인 추적** -- 기금별로 필터링하여 특별 헌금 캠페인이나 건축 프로젝트의 진행 상황을 추적합니다.
- **계절별 분석** -- 계절적 패턴을 이해하기 위해 다른 기간의 헌금을 비교합니다.

:::tip
교회의 재정 건강 상태를 파악하기 위해 매월 이 보고서를 실행하십시오. 월별 데이터를 비교하면 추세를 조기에 파악할 수 있습니다.
:::

:::warning
헌금 데이터는 시스템에 기록된 기여금에서 가져옵니다. 가장 정확한 보고서를 위해, 온라인 헌금, 수표 처리, 수동 입력 등 모든 방법을 통해 헌금이 일관되게 입력되고 있는지 확인하십시오. 자세한 내용은 [헌금 기록](../donations/recording-donations.md)을 참조하십시오.
:::
