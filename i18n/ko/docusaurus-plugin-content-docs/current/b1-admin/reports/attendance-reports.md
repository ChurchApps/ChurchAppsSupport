# 출석 보고서

<div class="article-intro">

B1 Admin은 교인들이 예배와 그룹에 얼마나 참여하고 있는지 파악하는 데 도움이 되는 세 가지 출석 보고서를 제공합니다. 각 보고서는 출석 데이터에 대해 상위 수준의 추세부터 일별 세부 내역까지 다양한 관점을 제공합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 예배와 그룹에 대해 출석이 [일관되게 기록](../attendance/tracking-attendance.md)되고 있는지 확인하십시오
- [그룹](../groups/creating-groups.md)과 예배가 B1 Admin에 설정되어 있는지 확인하십시오
- 보고서에 접근하려면 적절한 [권한](../settings/roles-permissions.md)이 필요합니다

</div>

## 출석 추세

출석 추세 보고서는 예배의 출석이 시간에 따라 어떻게 변화하는지 보여줍니다.

1. **보고서**로 이동하여 **출석 추세** 카드를 클릭합니다.
2. **날짜 범위**를 설정하여 기간을 정의합니다.
3. **예배** 또는 **그룹**을 선택하여 결과를 필터링합니다.
4. 보고서에 선택한 기간 동안의 출석 수를 나타내는 추세 선이 표시됩니다.

이 보고서는 계절적 감소, 성장 추세, 특별 행사의 영향과 같은 패턴을 파악하는 데 유용합니다.

## 그룹 출석

그룹 출석 보고서는 선택한 기간의 그룹별 출석 합계를 보여줍니다.

1. **보고서**로 이동하여 **그룹 출석** 카드를 클릭합니다.
2. 보고서의 **날짜 범위**를 설정합니다.
3. 검토할 **그룹**을 선택합니다.
4. 보고서에 각 그룹의 총 출석 수가 표시됩니다.

이 보고서를 사용하여 그룹 간 출석을 비교하고, 어떤 그룹이 성장하고 있거나 관심이 필요한지 파악하십시오.

## 일별 그룹 출석

일별 그룹 출석 보고서는 그룹의 출석 데이터를 일별로 세분화하여 제공합니다.

1. **보고서**로 이동하여 **일별 그룹 출석** 카드를 클릭합니다.
2. 보고서의 **날짜 범위**를 설정합니다.
3. 검토할 **그룹**을 선택합니다.
4. 보고서에 범위 내 각 개별 날짜의 출석 수가 표시됩니다.

이 보고서는 세부적인 데이터를 제공하여, 주간 변동을 이해하거나 출석이 비정상적으로 높거나 낮은 특정 날짜를 식별하는 데 도움이 됩니다.

:::tip
전체적인 개요를 보려면 출석 추세 보고서를 사용하고, 특정 날짜를 자세히 살펴보려면 일별 그룹 출석 보고서를 사용하십시오.
:::

## 실용적 활용

- **기획** -- 출석 추세를 사용하여 다가오는 예배의 좌석, 인력 배치, 자원을 계획합니다.
- **봉사 활동** -- 출석 감소 패턴을 조기에 파악하여 교인들에게 후속 조치를 취할 수 있습니다.
- **이사회 보고** -- 사역 건강 상태를 보여주기 위해 정기 리더십 보고서에 출석 데이터를 포함합니다.
- **행사 평가** -- 특별 행사 전후의 출석을 비교하여 그 영향을 측정합니다.

:::warning
출석 데이터는 그룹 및 예배 체크인 과정을 통해 기록됩니다. 출석이 일관되게 기록되지 않으면 보고서가 실제 참여를 정확하게 반영하지 못합니다. 설정 방법은 [출석 기록](../attendance/tracking-attendance.md)을 참조하십시오.
:::
