# 보고서

<div class="article-intro">

보고서 페이지는 교회 데이터를 이해하는 데 도움이 되는 미리 구축된 보고서에 빠르게 액세스할 수 있습니다. 이 보고서를 사용하여 출석 추세를 추적하고, 기부금 활동을 모니터링하고, 생일 축제를 통해 회중과 연결합니다.

</div>

## 사용 가능한 보고서

보고서 페이지에는 각 사용 가능한 보고서에 대한 보고서 카드가 표시됩니다. 아무 카드나 클릭하여 보고서를 필터와 함께 엽니다.

- **[생일 보고서](./birthday-report)** -- 회중 회원의 예정된 생일을 보입니다.
- **[출석 추세](./attendance-reports)** -- 시간이 지남에 따라 예배 출석 패턴이 어떻게 변하는지 확인합니다.
- **[그룹 출석](./attendance-reports)** -- 선택한 날짜 범위에 대해 그룹별 출석 합계를 봅니다.
- **[일별 그룹 출석](./attendance-reports)** -- 그룹 출석 데이터의 일일 분석을 얻습니다.
- **[기부금 요약](./donation-summary)** -- 재정 계획을 위해 기부금 총액 및 추세를 검토합니다.
- **[감사 로그](./audit-log)** -- 교회 관리 시스템 전체의 모든 변경 및 작업을 추적합니다(설정 > 감사 로그 아래에 있음).

## 보고서 사용

1. 메인 메뉴에서 **보고서**로 이동합니다.
2. **보고서 카드**를 클릭하여 엽니다.
3. 각 보고서의 상단에서 **필터**를 사용하여 날짜 범위, 그룹, 예배 또는 기타 기준별로 데이터를 좁힙니다.
4. 보고서에 표시된 결과를 검토합니다.

:::tip
정기적으로 보고서를 실행하여 출석 추세, 예정된 생일, 기부금 패턴을 파악합니다. 이 데이터는 사역을 계획하고 봉사 활동을 더 효과적으로 진행하는 데 도움이 됩니다.
:::

:::info
보고서 데이터는 일관된 데이터 항목에 따라 다릅니다. 팀이 [출석을 추적](../attendance/tracking-attendance.md)하고, [기부금을 기록](../donations/recording-donations.md)하고, [사람 기록](../people/adding-people.md)을 최신 상태로 유지하는지 확인하여 가장 정확한 결과를 얻습니다.
:::
