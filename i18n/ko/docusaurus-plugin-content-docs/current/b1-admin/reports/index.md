# 보고서

<div class="article-intro">

보고서 페이지는 교회 데이터를 파악하는 데 도움이 되는 사전 구축된 보고서에 빠르게 접근할 수 있도록 합니다. 이 보고서를 사용하여 출석 추세를 추적하고, 헌금 활동을 모니터링하며, 생일 축하를 통해 교인들과 소통을 유지하십시오.

</div>

## 사용 가능한 보고서

보고서 페이지에는 다섯 개의 보고서 카드가 표시됩니다. 카드를 클릭하면 필터와 함께 보고서가 열립니다.

- **[생일 보고서](./birthday-report)** -- 교인들의 다가오는 생일을 확인합니다.
- **[출석 추세](./attendance-reports)** -- 예배 전반에 걸친 출석 패턴 변화를 확인합니다.
- **[그룹 출석](./attendance-reports)** -- 선택한 기간의 그룹별 출석 합계를 확인합니다.
- **[일별 그룹 출석](./attendance-reports)** -- 그룹 출석 데이터의 일별 세부 내역을 확인합니다.
- **[헌금 요약](./donation-summary)** -- 재정 계획을 위한 헌금 합계와 추세를 검토합니다.

## 보고서 사용

1. 메인 메뉴에서 **보고서**로 이동합니다.
2. **보고서 카드**를 클릭하여 엽니다.
3. 각 보고서 상단의 **필터**를 사용하여 날짜 범위, 그룹, 예배 또는 기타 기준으로 데이터를 좁힙니다.
4. 보고서에 표시된 결과를 검토합니다.

:::tip
출석 추세, 다가오는 생일, 헌금 패턴을 파악하기 위해 보고서를 정기적으로 실행하십시오. 이 데이터는 사역과 봉사 활동을 더 효과적으로 계획하는 데 도움이 됩니다.
:::

:::info
보고서 데이터는 일관된 데이터 입력에 의존합니다. 가장 정확한 결과를 위해, 팀이 [출석을 기록](../attendance/tracking-attendance.md)하고, [헌금을 기록](../donations/recording-donations.md)하며, [인원 기록](../people/adding-people.md)을 최신 상태로 유지하고 있는지 확인하십시오.
:::
