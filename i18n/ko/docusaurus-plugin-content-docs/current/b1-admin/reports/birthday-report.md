# 생일 보고서

<div class="article-intro">

생일 보고서는 다가오는 생일이 있는 교인을 보여주어 교인들과의 소통을 유지하도록 도와줍니다. 생일 축하 계획, 카드 발송, 예배 중 공지 등에 활용하십시오.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 각 사람의 프로필에 생년월일이 입력되어 있는지 확인하십시오. 자세한 내용은 [인원 추가](../people/adding-people.md)를 참조하십시오.
- 보고서에 접근하려면 적절한 [권한](../settings/roles-permissions.md)이 필요합니다

</div>

## 보고서 실행

1. 메인 메뉴에서 **보고서**로 이동합니다.
2. **생일 보고서** 카드를 클릭합니다.
3. 날짜 필터를 사용하여 확인하려는 생일의 **날짜 범위**를 설정합니다.
4. 해당 범위 내에 생일이 있는 교인 목록이 보고서에 표시됩니다.

## 보고서에 표시되는 내용

생일 보고서에는 다음이 표시됩니다:

- **교인 이름** -- 교회 데이터베이스에 있는 사람의 이름입니다.
- **생년월일** -- 생일 날짜입니다.

결과는 선택한 날짜 범위 내에서 나열되므로, 다가오는 생일이 있는 사람을 쉽게 확인할 수 있습니다.

## 실용적 활용

- **주간 공지** -- 다가오는 주에 대한 보고서를 실행하여 주보나 공지사항에 생일을 포함합니다.
- **생일 카드** -- 월별 목록을 생성하여 팀이 개인 생일 카드를 보내도록 합니다.
- **소그룹 리더** -- 그룹 리더가 보고서를 실행하여 그룹 내 교인의 생일을 기억할 수 있습니다.
- **특별 축하** -- 특정 날짜 범위로 필터링하여 특별 생일 축하를 계획합니다.

:::tip
매월 초에 이 보고서를 실행하여 사전에 계획하십시오. 날짜 범위를 전체 월로 설정하면 카드나 공지를 준비할 시간을 확보할 수 있습니다.
:::

:::info
생일 데이터는 각 사람의 프로필에 있는 생년월일 필드에서 가져옵니다. 정확한 결과를 위해 [교인 기록](../people/adding-people.md)이 최신 상태인지 확인하십시오.
:::
