# 그룹 만들기

<div class="article-intro">

B1 Admin에서 그룹을 만드는 것은 간단합니다. 범주를 설정하고, 그룹의 이름을 지정한 다음 설정을 구성합니다. 그룹은 소그룹, 위원회, 클래스와 같은 의미 있는 단위로 교회를 조직하는 데 도움이 됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 그룹을 관리할 수 있는 권한이 있는 활성 B1 Admin 계정이 필요합니다. 액세스 수준이 확실하지 않으면 [Roles & Permissions](../people/roles-permissions.md)을 참조하세요.
- 그룹의 범주 구조를 결정합니다(예: "Small Groups," "Ministries," "Committees"). 범주는 관련 그룹을 정리하는 데 도움이 됩니다.

</div>

## 새 그룹 추가

1. **B1 Admin dashboard**로 이동합니다.
2. **Groups** 탭을 클릭합니다.
3. **Add Group**을 클릭하고 **Category Name**을 입력합니다. 범주는 관련 그룹을 함께 정렬하는 데 도움이 됩니다(예: "Small Groups," "Ministries," 또는 "Committees"). 범주가 이미 존재하면 목록에서 선택할 수 있습니다.
4. **Group Name**을 입력합니다.
5. **Add**를 클릭합니다. 새 그룹이 선택한 범주 아래의 목록에 나타납니다.

## 그룹 설정 구성

그룹이 생성되면 추가 세부 정보를 입력할 수 있습니다:

1. 목록의 **group name**을 클릭하여 엽니다.
2. **pencil icon**을 클릭하여 그룹 설정을 편집합니다.
3. 다음 옵션을 구성합니다:
   - **Description** -- 그룹이 무엇에 관한 것인지에 대한 간단한 요약. 멤버에게 표시됩니다.
   - **Meeting Times** -- 그룹이 일반적으로 만나는 시간(예: "Wednesdays at 7 PM").
   - **Join Policy** -- 이 그룹에 가입할 수 있는 사람을 선택합니다:
     - **Open** -- 누구나 승인 없이 즉시 가입 가능
     - **Request** -- 사람들이 승인이 필요한 가입 요청을 제출해야 합니다([Group Join Requests](./group-join-requests.md) 참조)
     - **Closed** -- 멤버는 리더 또는 관리자에 의해 수동으로 추가되어야 합니다
   - **Labels** -- 그룹에 하나 이상의 설명적 레이블을 할당합니다(예: "In-Person", "Online", "New Members Welcome"). 레이블은 정의하는 자유 형식 태그입니다. 해당하는 모든 항목을 확인합니다. 레이블은 Groups Browser 웹사이트 요소에서 그룹을 필터링하는 데 사용할 수 있습니다.
   - **Confidential group** -- 공개 페이지, 그룹 찾기 및 비멤버로부터 이 그룹과 그 명단을 숨깁니다. 회복 또는 상담 사역과 같은 민감한 그룹에 이를 사용합니다. 그룹의 멤버와 교회 직원만 볼 수 있습니다.
   - **Attendance Tracking** -- 이 그룹에 대해 [attendance](../attendance/tracking-attendance.md)를 기록하려면 활성화합니다.
   - **Service Times** -- 해당하는 경우 그룹을 특정 교회 예배 시간과 연결합니다. 예배 시간에 대한 자세한 내용은 [Attendance Setup](../attendance/setup.md)을 참조하세요.
4. **Save**를 클릭하여 변경 사항을 적용합니다.

:::tip
명확한 설명과 회의 시간을 추가하면 멤버가 그룹에 가입할 때 무엇을 기대할 수 있는지 알 수 있습니다.
:::

## 그룹 복제

반복되는 클래스 또는 사역의 새 세션을 시작하시나요? 모든 설정을 다시 입력하는 대신 기존 그룹을 복제하세요:

1. 그룹을 열고 그룹 배너(Edit 옆)의 **duplicate** 아이콘을 클릭합니다.
2. 복제를 확인합니다.

복사본은 원본의 설정(범주, 설명, 회의 시간/위치, 가입 정책, 레이블 및 캠퍼스)을 유지하지만 **멤버는 아닙니다**. 새 그룹의 이름은 원본 이름에 " (Copy)"가 추가된 이름입니다. 그룹 설정에서 이름을 변경합니다.

## 그룹 보관

그룹이 더 이상 활성화되지 않지만 삭제하는 대신 그 기록을 유지하려면:

1. 그룹을 열고 **pencil icon**을 클릭하여 설정을 편집합니다.
2. **Archive**를 클릭하고 확인합니다.

보관된 그룹은 메인 Groups 목록에서 제외됩니다. 다시 찾으려면 Groups 페이지의 위쪽에 있는 **Show archived** 토글을 켜고 그룹 옆의 **Restore**를 클릭하여 되돌립니다.

:::info
그룹을 보관해도 멤버, 출석 기록 또는 일정 이벤트는 삭제되지 않습니다. 복원할 때까지 그룹을 기본 목록에서만 숨깁니다.
:::

## 다음 단계

그룹을 만들고 구성한 후 다음을 수행할 준비가 되었습니다:

- **멤버 추가** -- 사람을 검색하고 그룹에 추가합니다. 녹색 키 아이콘을 사용하여 그룹 리더를 지정합니다. [Group Members](./group-members.md)을 참조하세요.
- **일정 설정** -- 그룹에 대한 이벤트 및 반복 회의를 만듭니다. [Group Calendar](./group-calendar.md)을 참조하세요.
- **소통** -- 그룹 페이지에서 직접 모든 그룹 멤버에게 메시지를 보냅니다.
- **데이터 내보내기** -- 다운로드 아이콘을 클릭하여 그룹의 멤버 목록을 내보냅니다.

:::info
모든 교회 그룹은 메인 Groups 페이지의 범주별로 정렬됩니다. 교회가 성장함에 따라 범주를 항상 재정렬하거나 이름을 바꿀 수 있습니다.
:::
