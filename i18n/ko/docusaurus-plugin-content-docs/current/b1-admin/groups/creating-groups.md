# 그룹 만들기

<div class="article-intro">

B1 Admin에서 그룹을 만드는 것은 간단합니다. 카테고리를 설정하고 그룹 이름을 지정한 다음 설정을 구성합니다. 그룹은 소그룹, 위원회, 성경공부반 등 교회를 의미 있는 단위로 구성하는 데 도움이 됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 그룹 관리 권한이 있는 활성 B1 Admin 계정이 필요합니다. 액세스 수준이 확실하지 않으면 [역할 및 권한](../people/roles-permissions.md)을 참조하세요.
- 그룹에 대한 카테고리 구조를 결정하세요(예: "소그룹", "사역", "위원회"). 카테고리는 관련 그룹을 체계적으로 유지하는 데 도움이 됩니다.

</div>

## 새 그룹 추가

1. **B1 Admin 대시보드**로 이동합니다.
2. **그룹** 탭을 클릭합니다.
3. **그룹 추가**를 클릭하고 **카테고리 이름**을 입력합니다. 카테고리는 관련 그룹을 함께 구성하는 데 도움이 됩니다(예: "소그룹", "사역" 또는 "위원회"). 카테고리가 이미 있으면 목록에서 선택할 수 있습니다.
4. **그룹 이름**을 입력합니다.
5. **추가**를 클릭합니다. 새 그룹이 선택한 카테고리 아래의 목록에 나타납니다.

## 그룹 설정 구성

그룹이 생성되면 추가 세부 정보를 입력할 수 있습니다:

1. 목록에서 **그룹 이름**을 클릭하여 엽니다.
2. **연필 아이콘**을 클릭하여 그룹 설정을 편집합니다.
3. 다음 옵션을 구성합니다:
   - **설명** -- 그룹이 무엇에 대한 것인지 간략히 설명합니다. 교인에게 표시됩니다.
   - **모임 시간** -- 그룹이 일반적으로 모이는 시간(예: "수요일 오후 7시").
   - **참여 정책** -- 이 그룹에 누가 참여할 수 있는지 선택합니다:
     - **개방형** -- 승인 없이 누구나 즉시 참여할 수 있습니다.
     - **요청** -- 사람들이 승인이 필요한 참여 요청을 제출해야 합니다([그룹 참여 요청](./group-join-requests.md) 참조).
     - **폐쇄형** -- 지도자 또는 관리자가 수동으로 구성원을 추가해야 합니다.
   - **라벨** -- 그룹에 하나 이상의 설명 라벨을 할당합니다(예: "대면", "온라인", "신규 교인 환영"). 라벨은 자유 형식의 태그입니다. 라벨은 그룹 브라우저 웹사이트 요소에서 그룹을 필터링하는 데 사용할 수 있습니다.
   - **참석 추적** -- 이 그룹에 대해 [참석](../attendance/tracking-attendance.md)을 기록하려면 활성화하세요.
   - **서비스 시간** -- 해당하는 경우 그룹을 특정 교회 예배 시간과 연결합니다. [참석 설정](../attendance/setup.md)에서 서비스 시간에 대한 자세한 내용을 참조하세요.
4. **저장**을 클릭하여 변경 사항을 적용합니다.

:::tip
명확한 설명과 모임 시간을 추가하면 교인들이 그룹에 참여할 때 무엇을 기대해야 하는지 알 수 있습니다.
:::

## 다음 단계

그룹을 만들고 구성한 후 다음을 수행할 수 있습니다:

- **구성원 추가** -- 사람을 검색하여 그룹에 추가합니다. 녹색 열쇠 아이콘을 사용하여 그룹 지도자를 지정합니다. [그룹 구성원](./group-members.md)을 참조하세요.
- **캘린더 설정** -- 그룹에 대한 이벤트 및 반복 모임을 만듭니다. [그룹 캘린더](./group-calendar.md)를 참조하세요.
- **커뮤니케이션** -- 그룹 페이지에서 직접 모든 그룹 구성원에게 메시지를 보냅니다.
- **데이터 내보내기** -- 다운로드 아이콘을 클릭하여 그룹의 구성원 목록을 내보냅니다.

:::info
모든 교회 그룹은 기본 그룹 페이지의 카테고리별로 구성됩니다. 교회가 성장함에 따라 언제든지 카테고리를 다시 정렬하거나 이름을 바꿀 수 있습니다.
:::
