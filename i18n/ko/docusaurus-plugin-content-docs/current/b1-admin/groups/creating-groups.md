# 그룹 만들기

<div class="article-intro">

B1 Admin에서 그룹을 만드는 것은 간단합니다. 범주를 설정하고, 그룹의 이름을 지정한 다음, 설정을 구성합니다. 그룹은 교회를 소그룹, 위원회, 클래스 같은 의미 있는 단위로 구성하는 데 도움이 됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 그룹을 관리할 권한이 있는 활성 B1 Admin 계정이 필요합니다. 액세스 수준이 불확실한 경우 [역할 및 권한](../people/roles-permissions.md)을 참조하세요.
- 그룹에 대한 범주 구조를 결정하세요(예: "소그룹", "사역", "위원회"). 범주는 관련 그룹을 정리하는 데 도움이 됩니다.

</div>

## 새 그룹 추가

1. **B1 Admin 대시보드**로 이동합니다.
2. **그룹** 탭을 클릭합니다.
3. **그룹 추가**를 클릭하고 **범주 이름**을 입력합니다. 범주는 관련 그룹을 한데 모으는 데 도움이 됩니다(예: "소그룹", "사역" 또는 "위원회"). 범주가 이미 있으면 목록에서 선택할 수 있습니다.
4. **그룹 이름**을 입력합니다.
5. **추가**를 클릭합니다. 새 그룹이 선택한 범주 아래의 목록에 나타납니다.

## 그룹 설정 구성

그룹이 만들어진 후 추가 세부 정보를 입력할 수 있습니다:

1. 목록에서 **그룹 이름**을 클릭하여 엽니다.
2. **연필 아이콘**을 클릭하여 그룹 설정을 편집합니다.
3. 다음 옵션 구성:
   - **설명** -- 그룹이 무엇에 관한 것인지 간단히 설명합니다. 회원들에게 표시됩니다.
   - **모임 시간** -- 그룹이 일반적으로 만나는 시간(예: "수요일 오후 7시").
   - **출석 추적** -- 이 그룹에 대해 [출석](../attendance/tracking-attendance.md)을 기록하려면 활성화합니다.
   - **예배 시간** -- 해당하는 경우 그룹을 특정 교회 예배 시간과 연결합니다. [출석 설정](../attendance/setup.md)의 예배 시간에 대한 세부 정보를 참조하세요.
4. **저장**을 클릭하여 변경 사항을 적용합니다.

:::tip
명확한 설명과 모임 시간을 추가하면 회원들이 그룹에 참여할 때 무엇을 기대할 수 있는지 알 수 있도록 도와줍니다.
:::

## 다음 단계

그룹을 만들고 구성한 후 다음을 수행할 준비가 됩니다:

- **회원 추가** -- 사람을 검색하여 그룹에 추가합니다. 녹색 키 아이콘을 사용하여 그룹 지도자를 지정합니다. [그룹 회원](./group-members.md)을 참조하세요.
- **캘린더 설정** -- 그룹의 이벤트 및 정기적인 모임을 만듭니다. [그룹 캘린더](./group-calendar.md)를 참조하세요.
- **소통** -- 그룹 페이지에서 직접 모든 그룹 회원에게 메시지를 보냅니다.
- **데이터 내보내기** -- 다운로드 아이콘을 클릭하여 그룹의 회원 목록을 내보냅니다.

:::info
모든 교회 그룹은 메인 그룹 페이지의 범주로 구성됩니다. 교회가 성장할 때 항상 범주를 재정렬하거나 이름을 바꿀 수 있습니다.
:::
