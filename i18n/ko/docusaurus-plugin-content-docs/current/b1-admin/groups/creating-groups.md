# 그룹 만들기

<div class="article-intro">

B1 Admin에서 그룹을 만드는 것은 간단합니다. 카테고리를 설정하고, 그룹 이름을 지정한 다음, 설정을 구성합니다. 그룹을 사용하면 교회를 소그룹, 사역팀, 위원회, 수업 등 의미 있는 단위로 조직할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 그룹 관리 권한이 있는 활성 B1 Admin 계정이 필요합니다. 접근 권한이 확실하지 않은 경우 [역할 및 권한](../people/roles-permissions.md)을 참조하세요.
- 그룹의 카테고리 구조를 미리 결정하세요 (예: "소그룹", "사역부서", "위원회"). 카테고리는 관련 그룹을 체계적으로 정리하는 데 도움이 됩니다.

</div>

## 새 그룹 추가

1. **B1 Admin 대시보드**로 이동합니다.
2. **그룹** 탭을 클릭합니다.
3. **그룹 추가**를 클릭하고 **카테고리 이름**을 입력합니다. 카테고리는 관련 그룹을 함께 정리하는 데 도움이 됩니다 (예: "소그룹", "사역부서", "위원회"). 이미 카테고리가 있는 경우 목록에서 선택할 수 있습니다.
4. **그룹 이름**을 입력합니다.
5. **추가**를 클릭합니다. 새 그룹이 선택한 카테고리 아래 목록에 나타납니다.

## 그룹 설정 구성

그룹이 생성되면 추가 세부 정보를 입력할 수 있습니다:

1. 목록에서 **그룹 이름**을 클릭하여 엽니다.
2. **연필 아이콘**을 클릭하여 그룹 설정을 편집합니다.
3. 다음 옵션을 구성합니다:
   - **설명** -- 그룹에 대한 간단한 요약입니다. 구성원에게 표시됩니다.
   - **모임 시간** -- 그룹이 주로 모이는 시간입니다 (예: "수요일 오후 7시").
   - **출석 추적** -- 이 그룹의 [출석](../attendance/tracking-attendance.md)을 기록하려면 활성화합니다.
   - **예배 시간** -- 해당하는 경우 그룹을 특정 교회 예배 시간과 연결합니다. 예배 시간에 대한 자세한 내용은 [출석 설정](../attendance/setup.md)을 참조하세요.
4. **저장**을 클릭하여 변경 사항을 적용합니다.

:::tip
명확한 설명과 모임 시간을 추가하면 구성원이 그룹에 가입할 때 무엇을 기대할 수 있는지 알 수 있습니다.
:::

## 다음 단계

그룹을 만들고 구성한 후에는 다음 작업을 할 수 있습니다:

- **구성원 추가** -- 사람을 검색하여 그룹에 추가합니다. 녹색 열쇠 아이콘을 사용하여 그룹 리더를 지정합니다. [그룹 구성원](./group-members.md)을 참조하세요.
- **일정 설정** -- 그룹의 행사 및 정기 모임을 만듭니다. [그룹 일정](./group-calendar.md)을 참조하세요.
- **소통** -- 그룹 페이지에서 직접 모든 그룹 구성원에게 메시지를 보낼 수 있습니다.
- **데이터 내보내기** -- 다운로드 아이콘을 클릭하여 그룹 구성원 목록을 내보냅니다.

:::info
모든 교회 그룹은 메인 그룹 페이지에서 카테고리별로 정리됩니다. 교회가 성장함에 따라 언제든지 카테고리를 재정렬하거나 이름을 변경할 수 있습니다.
:::
