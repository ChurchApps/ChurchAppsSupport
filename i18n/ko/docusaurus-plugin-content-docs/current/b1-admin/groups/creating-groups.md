# 그룹 만들기

<div class="article-intro">

B1 Admin에서 그룹을 만드는 것은 간단합니다. 카테고리를 설정하고 그룹 이름을 지정한 다음 설정을 구성하면 됩니다. 그룹은 소그룹, 위원회, 클래스와 같은 의미 있는 단위로 교회를 조직하는 데 도움이 됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 그룹을 관리할 수 있는 권한이 있는 활성 B1 Admin 계정이 필요합니다. 액세스 수준이 확실하지 않은 경우 [역할 및 권한](../people/roles-permissions.md)을 참조하세요.
- 그룹의 카테고리 구조를 결정하세요(예: "소그룹", "사역", "위원회"). 카테고리는 관련 그룹을 체계적으로 유지하는 데 도움이 됩니다.

</div>

## 새 그룹 추가하기

1. **B1 Admin 대시보드**로 이동합니다.
2. **Groups** 탭을 클릭합니다.
3. **Add Group**을 클릭하고 **Category Name**을 입력합니다. 카테고리는 관련 그룹을 함께 정리하는 데 도움이 됩니다(예: "소그룹", "사역" 또는 "위원회"). 카테고리가 이미 존재하는 경우 목록에서 선택할 수 있습니다.
4. **Group Name**을 입력합니다.
5. **Add**를 클릭합니다. 새 그룹이 선택한 카테고리 아래 목록에 표시됩니다.

## 그룹 설정 구성하기

그룹을 만든 후 추가 세부 정보를 입력할 수 있습니다:

1. 목록에서 **그룹 이름**을 클릭하여 엽니다.
2. **연필 아이콘**을 클릭하여 그룹 설정을 편집합니다.
3. 다음 옵션을 구성합니다:
   - **Description** -- 그룹에 대한 간단한 요약입니다. 회원에게 표시됩니다.
   - **Meeting Times** -- 그룹이 일반적으로 모이는 시간입니다(예: "수요일 오후 7시").
   - **Join Policy** -- 이 그룹에 누가 가입할 수 있는지 선택합니다:
     - **Open** -- 승인 없이 누구나 즉시 가입할 수 있습니다
     - **Request** -- 사람들이 승인이 필요한 가입 요청을 제출해야 합니다([그룹 가입 요청](./group-join-requests.md) 참조)
     - **Closed** -- 리더나 관리자가 수동으로 회원을 추가해야 합니다
   - **Attendance Tracking** -- 이 그룹의 [출석](../attendance/tracking-attendance.md)을 기록하려면 활성화합니다.
   - **Service Times** -- 해당하는 경우 그룹을 특정 교회 예배 시간과 연결합니다. 예배 시간에 대한 자세한 내용은 [출석 설정](../attendance/setup.md)을 참조하세요.
4. **Save**를 클릭하여 변경 사항을 적용합니다.

:::tip
명확한 설명과 모임 시간을 추가하면 회원들이 그룹에 가입할 때 무엇을 기대해야 하는지 알 수 있습니다.
:::

## 다음 단계

그룹을 만들고 구성한 후 다음을 수행할 준비가 되었습니다:

- **회원 추가** -- 사람들을 검색하고 그룹에 추가합니다. 녹색 키 아이콘을 사용하여 그룹 리더를 지정합니다. [그룹 회원](./group-members.md)을 참조하세요.
- **캘린더 설정** -- 그룹의 이벤트와 정기 모임을 만듭니다. [그룹 캘린더](./group-calendar.md)를 참조하세요.
- **소통하기** -- 그룹 페이지에서 직접 모든 그룹 회원에게 메시지를 보냅니다.
- **데이터 내보내기** -- 다운로드 아이콘을 클릭하여 그룹의 회원 목록을 내보냅니다.

:::info
모든 교회 그룹은 메인 Groups 페이지에서 카테고리별로 정리됩니다. 교회가 성장함에 따라 언제든지 카테고리를 재배치하거나 이름을 바꿀 수 있습니다.
:::
