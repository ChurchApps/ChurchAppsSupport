# 그룹

<div class="article-intro">

**그룹** 섹션을 사용하면 교회 교인들을 사역, 소그룹, 성경공부, 위원회 등으로 구성할 수 있습니다. 그룹은 교회의 공동체 생활의 근본이며, 이 섹션은 한 곳에서 모든 그룹을 만들고, 관리하고, 소통할 수 있는 도구를 제공합니다. **팀**은 그룹과 별개입니다. 팀은 봉사 및 자원봉사 일정 조정에 사용됩니다.

</div>

## 개요

그룹은 **카테고리**로 구성됩니다. 카테고리는 "소그룹", "사역" 또는 "위원회"와 같은 상위 수준의 라벨입니다. 각 카테고리는 여러 그룹을 포함할 수 있어 관련 그룹을 함께 유지하기가 간단합니다.

그룹 페이지를 열면 맨 위에 요약이 표시됩니다:

- **총 그룹** -- 모든 카테고리의 그룹 수
- **카테고리** -- 만든 그룹 카테고리 수
- **총 구성원** -- 모든 그룹의 구성원 합계

요약 아래에는 그룹이 카테고리별로 나열되며 각 그룹의 구성원 수가 표시됩니다.

## 할 수 있는 것

그룹 섹션에서 수행할 수 있는 작업은 다음과 같습니다:

1. **그룹 만들기 및 구성** -- 교회 구조를 반영하기 위해 카테고리 내에 새 그룹을 추가합니다. [그룹 만들기](./creating-groups.md)를 참조하세요.

2. **그룹 구성원 관리** -- 구성원을 추가 및 제거하고, 지도자를 지정하고, 그룹과 소통합니다. [그룹 구성원](./group-members.md)을 참조하세요.

3. **이벤트 예약** -- 각 그룹은 모임, 이벤트 및 반복 활동을 위해 자체 캘린더를 가질 수 있습니다. [그룹 캘린더](./group-calendar.md)를 참조하세요.

4. **참여 요청 관리** -- 그룹이 **요청** 참여 정책을 사용할 때, 그룹에 참여하고 싶어 하는 사람들의 들어오는 구성원 요청을 검토하고 승인하거나 거부합니다. [그룹 참여 요청](./group-join-requests.md)을 참조하세요.

5. **그룹 데이터 내보내기** -- 스프레드시트 또는 우편 발송에 사용할 수 있도록 모든 그룹의 구성원 목록을 CSV 파일로 다운로드합니다. 내보내기 지침은 [그룹 구성원](./group-members.md)을 참조하세요.

## 시작하기

그룹에 액세스하려면 B1 Admin 대시보드의 왼쪽 사이드바에서 **사람**을 클릭한 다음 **그룹**을 선택합니다. 카테고리별로 구성된 모든 기존 그룹이 표시됩니다.

:::tip
시작하는 경우 교회 구조에 맞는 몇 가지 카테고리를 만드는 것부터 시작하세요(예: "사역", "소그룹", "위원회"). 그런 다음 각 카테고리 내에 개별 그룹을 추가합니다. 단계별 안내는 [그룹 만들기](./creating-groups.md)를 참조하세요.
:::

:::info
그룹 구성원은 먼저 그룹에 추가되기 전에 [사람](../people/adding-people.md) 디렉토리에 존재해야 합니다. B1을 처음 설정하는 경우 그룹을 만들기 전에 [구성원 데이터를 가져오세요](../people/importing-data.md).
:::
