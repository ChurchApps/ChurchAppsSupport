# 그룹

<div class="article-intro">

**그룹** 섹션을 사용하면 교회 회원을 사역, 소그룹, 성경 공부, 위원회 등으로 구성할 수 있습니다. 그룹은 교회 공동체 생활의 중심이며, 이 섹션은 한 곳에서 모든 그룹을 만들고, 관리하고, 소통할 수 있는 도구를 제공합니다. **팀**은 그룹과 분리되어 있습니다 — 팀은 봉사와 자원봉사 일정에 사용됩니다.

</div>

## 개요

그룹은 **범주**로 구성됩니다. 범주는 "소그룹", "사역" 또는 "위원회"와 같은 최상위 라벨입니다. 각 범주는 여러 그룹을 포함할 수 있으므로 관련 그룹을 쉽게 함께 유지할 수 있습니다.

그룹 페이지를 열면 상단에 다음을 표시하는 요약이 나타납니다:

- **총 그룹** -- 모든 범주의 그룹 수
- **범주** -- 생성한 그룹 범주의 수
- **총 회원** -- 모든 그룹의 회원 수 합계

요약 아래에 그룹이 범주별로 나열되며 각 그룹의 회원 수가 표시됩니다.

## 할 수 있는 것

그룹 섹션에서 다음을 수행할 수 있습니다:

1. **그룹 만들기 및 구성** -- 교회 구조를 반영하도록 범주 내에 새 그룹을 추가합니다. [그룹 만들기](./creating-groups.md)를 참조하세요.

2. **그룹 회원 관리** -- 회원을 추가 및 제거하고, 지도자를 지정하고, 그룹과 소통합니다. [그룹 회원](./group-members.md)을 참조하세요.

3. **이벤트 예약** -- 각 그룹은 모임, 이벤트, 정기적인 활동을 위한 자체 캘린더를 가질 수 있습니다. [그룹 캘린더](./group-calendar.md)를 참조하세요.

4. **그룹 데이터 내보내기** -- 스프레드시트나 메일 발송에 사용할 그룹의 회원 목록을 CSV 파일로 다운로드합니다. 내보내기 지침은 [그룹 회원](./group-members.md)을 참조하세요.

## 시작하기

그룹에 액세스하려면 B1 Admin 대시보드의 왼쪽 사이드바에서 **사람**을 클릭한 다음 **그룹**을 선택합니다. 범주별로 정렬된 모든 기존 그룹을 볼 수 있습니다.

:::tip
이제 시작하는 경우 교회 구조와 일치하는 몇 가지 범주를 만드는 것부터 시작합니다(예: "사역", "소그룹", "위원회"). 그런 다음 각 범주 내에 개별 그룹을 추가합니다. 단계별 안내는 [그룹 만들기](./creating-groups.md)를 참조하세요.
:::

:::info
그룹 회원은 먼저 [사람](../people/adding-people.md) 디렉터리에 있어야 한 후에 그룹에 추가될 수 있습니다. B1을 처음으로 설정하는 경우 그룹을 만들기 전에 [회원 데이터를 가져옵니다](../people/importing-data.md).
:::
