# 그룹

<div class="article-intro">

**그룹** 섹션에서는 교회 교인을 팀, 사역부서, 소그룹, 성경 공부, 위원회 등으로 조직할 수 있습니다. 그룹은 교회 공동체 생활의 근간이며, 이 섹션에서는 모든 그룹을 한 곳에서 생성, 관리, 소통할 수 있는 도구를 제공합니다.

</div>

## 개요

그룹은 **카테고리**별로 정리됩니다. 카테고리는 "소그룹", "사역부서", "위원회"와 같은 최상위 레이블입니다. 각 카테고리에는 여러 그룹이 포함될 수 있어 관련 그룹을 함께 관리하기 쉽습니다.

그룹 페이지를 열면 상단에 다음과 같은 요약 정보가 표시됩니다:

- **전체 그룹** -- 모든 카테고리의 그룹 수
- **카테고리** -- 생성한 그룹 카테고리 수
- **전체 구성원** -- 모든 그룹의 구성원을 합산한 수

요약 아래에는 각 그룹의 구성원 수와 함께 카테고리별로 그룹이 나열됩니다.

## 주요 기능

그룹 섹션에서 수행할 수 있는 작업은 다음과 같습니다:

1. **그룹 생성 및 조직** -- 교회 구조를 반영하는 카테고리 내에 새 그룹을 추가합니다. [그룹 만들기](./creating-groups.md)를 참조하세요.

2. **그룹 구성원 관리** -- 구성원을 추가 및 제거하고, 리더를 지정하며, 그룹과 소통합니다. [그룹 구성원](./group-members.md)을 참조하세요.

3. **일정 관리** -- 각 그룹은 모임, 행사, 정기 활동을 위한 자체 일정을 가질 수 있습니다. [그룹 일정](./group-calendar.md)을 참조하세요.

4. **그룹 데이터 내보내기** -- 스프레드시트나 우편물 발송에 사용할 수 있도록 그룹의 구성원 목록을 CSV 파일로 다운로드합니다. 내보내기 방법은 [그룹 구성원](./group-members.md)을 참조하세요.

## 시작하기

그룹에 접근하려면 B1 Admin 대시보드의 왼쪽 사이드바에서 **교인 관리**를 클릭한 다음 **그룹**을 선택합니다. 카테고리별로 정리된 기존 그룹을 모두 확인할 수 있습니다.

:::tip
처음 시작하는 경우, 교회 구조에 맞는 몇 가지 카테고리를 먼저 만드세요 (예: "사역부서", "소그룹", "팀"). 그런 다음 각 카테고리 내에 개별 그룹을 추가합니다. 단계별 안내는 [그룹 만들기](./creating-groups.md)를 참조하세요.
:::

:::info
그룹 구성원은 먼저 [교인 관리](../people/adding-people.md) 명부에 존재해야 그룹에 추가할 수 있습니다. B1을 처음 설정하는 경우, 그룹을 만들기 전에 [교인 데이터를 가져오세요](../people/importing-data.md).
:::
