# 그룹 건강

<div class="article-intro">

그룹 건강 대시보드는 모든 그룹이 얼마나 잘하고 있는지에 대한 조감도를 제공합니다. 멤버십 추세, 평균 출석 및 지난 90일 동안의 성장 또는 이탈을 모두 한 번에 정렬 가능한 표에 표시합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 의미 있는 데이터를 보려면 멤버가 있는 최소 몇 개의 그룹이 필요합니다. [그룹 만들기](creating-groups)를 참고하세요.
- 출석 데이터는 기록된 세션에서 가져옵니다. [출석](../attendance/index) 섹션을 참고하세요.

</div>

## 그룹 건강 열기

B1 Admin에서 왼쪽 사이드바의 **그룹**으로 이동하고 **건강**을 선택합니다. 대시보드는 그룹당 1개 행씩 포함된 표를 로드합니다.

## 열

| 열 | 표시 내용 |
|--------|--------------|
| **이름** | 그룹 상세 페이지로 연결된 그룹 이름 |
| **카테고리** | 그룹의 카테고리 |
| **멤버** | 현재 활성 멤버 수 |
| **가입함 (90일)** | 지난 90일 동안 가입한 멤버 |
| **탈퇴함 (90일)** | 지난 90일 동안 탈퇴한 멤버 |
| **이탈 (90일)** | 90일 동안의 순 이탈률(백분율) |
| **평균 출석** | 출석 세션당 평균 인원 수 |

표의 열 제목을 클릭하여 해당 열별로 정렬합니다. 다시 클릭하면 정렬 방향이 반대가 됩니다.

## 건강 데이터 사용하기

- **높은 이탈 + 낮은 가입** — 축소되고 있으며 탈퇴한 멤버를 보충하지 않는 그룹입니다. 그룹 리더와 대화할 가치가 있습니다.
- **높은 가입 + 낮은 출석** — 사람들은 등록하지만 나타나지 않습니다. 참여 후속 조치를 고려하세요.
- **높은 평균 출석** — 건강하고 활동적인 그룹입니다. 다른 그룹의 모델이 될 가능성이 있습니다.

:::tip
그룹 이름을 클릭하면 그룹의 상세 페이지로 이동하여 개별 멤버, 출석 기록 및 캘린더 이벤트를 검토할 수 있습니다.
:::
