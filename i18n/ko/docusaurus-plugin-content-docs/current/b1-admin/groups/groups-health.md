# 그룹 상태

<div class="article-intro">

그룹 상태 대시보드는 모든 그룹이 어떻게 진행 중인지에 대한 전체적인 관점을 제공합니다 — 회원 추이, 출석 평균 및 지난 90일 동안의 성장 또는 감소 — 모두 하나의 정렬 가능한 표에 표시됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 의미 있는 데이터를 보려면 회원이 있는 몇 개의 그룹이 필요합니다. [Creating Groups](creating-groups)를 참조하세요.
- 출석 데이터는 기록된 세션에서 가져옵니다. [Attendance](../attendance/) 섹션을 참조하세요.

</div>

## 그룹 상태 열기

B1 Admin에서 왼쪽 사이드바의 **Groups**를 선택한 후 **Health**를 선택합니다. 대시보드는 그룹당 한 행씩 표가 로드됩니다.

## 열

| 열 | 표시 내용 |
|--------|--------------|
| **Name** | 그룹 이름(그룹 상세 페이지로 연결됨) |
| **Category** | 그룹의 카테고리 |
| **Members** | 현재 활성 회원 수 |
| **Joined (90d)** | 지난 90일 동안 가입한 회원 |
| **Left (90d)** | 지난 90일 동안 떠난 회원 |
| **Churn (90d)** | 90일 동안의 순 이탈률(퍼센트) |
| **Avg Attendance** | 출석 세션당 평균 인원 |

열 제목을 클릭하면 해당 열로 표를 정렬합니다. 다시 클릭하면 정렬 방향을 반대로 합니다.

## 상태 데이터 사용

- **높은 이탈률 + 낮은 가입** — 축소되고 있으며 탈락한 회원을 대체하지 않는 그룹입니다. 그룹 리더와 대화할 가치가 있습니다.
- **높은 가입 + 낮은 출석률** — 사람들이 가입하지만 나타나지 않습니다. 참여 후속 조치를 고려하세요.
- **높은 평균 출석률** — 건강하고 활동적인 그룹입니다. 다른 그룹의 모델이 될 수 있습니다.

:::tip
그룹 이름을 클릭하면 그룹의 상세 페이지로 직접 이동하여 개별 회원, 출석 기록 및 달력 이벤트를 검토할 수 있습니다.
:::
