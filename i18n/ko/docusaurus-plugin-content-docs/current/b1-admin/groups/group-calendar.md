# 그룹 일정

<div class="article-intro">

B1의 각 그룹은 자체 행사 일정을 가질 수 있습니다. 그룹 리더는 이를 사용하여 모임, 활동, 정기 행사를 예약하며, 모든 그룹 구성원은 다가오는 행사를 확인하고 관련 대화에 참여할 수 있습니다. 일정을 통해 그룹을 체계적으로 관리하고 모든 사람이 앞으로의 일정을 파악할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin에서 최소 한 명의 구성원이 있는 그룹이 생성되어 있어야 합니다. [그룹 만들기](creating-groups.md)와 [그룹 구성원](group-members.md)을 참조하세요.
- 일정 행사를 추가하거나 편집하려면 그룹 리더로 지정되어 있어야 합니다. 방법은 [그룹 리더 지정](group-members.md)을 참조하세요.

</div>

## 그룹 일정 접근

그룹 일정은 B1.church 회원 앱을 통해 사용할 수 있습니다:

1. B1 Admin에서 화면 오른쪽 상단의 **드롭다운 메뉴**를 클릭합니다.
2. **앱 전환**을 선택합니다.
3. **B1.church**를 선택합니다.
4. 왼쪽 사이드바에서 **그룹**을 클릭합니다.
5. 목록에서 그룹을 선택합니다.
6. 화면 왼쪽에서 **일정**을 클릭합니다.

:::info
그룹 일정은 B1 Admin이 아닌 B1.church (회원 앱)를 통해 관리됩니다. 그룹 리더는 관리자 대시보드에서 앱을 전환하여 접근합니다.
:::

## 행사 추가

1. 그룹 일정에서 **행사 추가**를 클릭합니다.
2. 행사의 **제목**을 입력합니다.
3. 행사 내용에 대한 간단한 **설명**을 작성합니다.
4. **시작 시간**과 **종료 시간**을 설정합니다.
5. **저장**을 클릭하여 일정에 행사를 추가합니다.

## 정기 행사 설정

그룹이 정기적으로 모이는 경우, 행사가 자동으로 반복되도록 설정할 수 있습니다:

1. 행사를 만들거나 편집할 때 **반복** 체크 박스를 선택합니다.
2. 행사가 반복되는 방식을 선택합니다:
   - **간격** -- 반복 간의 일수를 설정합니다. 예를 들어 "7"을 입력하면 7일마다 행사가 반복됩니다.
   - **빈도** -- **일**, **주**, **월** 단위로 반복을 선택합니다.
3. 행사를 저장합니다. 이제 예정된 각 발생 시점에 일정에 표시됩니다.

:::tip
매주 토요일에 모이는 그룹의 경우, 빈도를 "주"로 설정하고 토요일을 선택하세요. 행사가 자동으로 일정에 채워집니다.
:::

## 일정 보기

그룹 일정은 여러 보기 옵션을 제공합니다:

- **월** -- 한 달 전체를 한눈에 봅니다.
- **주** -- 이번 주에 집중합니다.
- **일** -- 하루를 상세하게 봅니다.
- **목록** -- 다가오는 행사를 시간순으로 봅니다.

일정 상단의 버튼을 사용하여 보기를 전환합니다.

## 행사 세부 정보 및 대화

행사를 클릭하면 세부 정보를 확인할 수 있습니다. 행사 세부 정보 화면에서:

- **구성원과 리더**는 행사에 대한 대화를 시작하고 댓글을 남길 수 있습니다.
- **리더**는 행사 세부 정보를 편집하고, 시간을 변경하거나 행사를 삭제할 수 있습니다.

:::warning
행사를 삭제하면 영구적으로 제거됩니다. 정기 행사의 단일 발생만 취소해야 하는 경우 삭제 대신 편집을 고려하세요.
:::

## 일정 구독

그룹 일정을 개인 일정과 동기화할 수 있습니다:

1. 그룹 일정 페이지에서 **구독**을 클릭합니다.
2. 안내에 따라 Google Calendar, Apple Calendar, Outlook 또는 구독을 지원하는 기타 일정 애플리케이션에 일정을 추가합니다.

:::tip
구독하면 B1을 별도로 확인할 필요 없이 개인 일정이 그룹 행사와 동기화됩니다. 그룹 리더가 변경한 내용은 동기화된 일정에 자동으로 반영됩니다.
:::
