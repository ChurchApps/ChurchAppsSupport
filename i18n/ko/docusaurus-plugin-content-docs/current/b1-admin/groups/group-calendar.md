# 그룹 일정

<div class="article-intro">

B1의 각 그룹은 자신의 이벤트 일정을 가질 수 있습니다. 그룹 리더는 회의, 활동 및 반복 이벤트를 예약하는 데 사용하며, 모든 그룹 멤버는 다가오는 이벤트를 보고 이에 대한 대화에 참여할 수 있습니다. 일정은 그룹을 정렬된 상태로 유지하고 모든 사람이 다가오는 항목에 대해 정보를 받게 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin에서 최소한 하나의 멤버를 가진 그룹이 생성되어야 합니다. [Creating Groups](creating-groups.md) 및 [Group Members](group-members.md)을 참조하세요.
- 일정 이벤트를 추가하거나 편집하려면 그룹 리더로 지정되어야 합니다. 지침은 [Designating Group Leaders](group-members.md)을 참조하세요.

</div>

## 그룹 일정 접근

그룹 일정은 B1.church 멤버 대면 앱을 통해 사용할 수 있습니다:

1. B1 Admin에서 화면 오른쪽 위 모서리의 **dropdown menu**를 클릭합니다.
2. **Switch App**을 선택합니다.
3. **B1.church**를 선택합니다.
4. 왼쪽 사이드바의 **Groups**을 클릭합니다.
5. 목록에서 그룹을 선택합니다.
6. 화면의 왼쪽에 있는 **Calendar**를 클릭합니다.

:::info
그룹 일정은 B1 Admin이 아닌 B1.church(멤버 앱)를 통해 관리됩니다. 그룹 리더는 관리 대시보드에서 앱을 전환하여 접근합니다.
:::

## 이벤트 추가

1. 그룹 일정에서 **Add an Event**를 클릭합니다.
2. 이벤트의 **title**을 입력합니다.
3. 이벤트가 무엇에 관한 것인지 짧은 **description**을 작성합니다.
4. **start time**과 **end time**을 설정합니다.
5. **Save**를 클릭하여 이벤트를 일정에 추가합니다.

## 반복 이벤트 설정

그룹이 정기적인 일정에서 만나면 이벤트를 자동으로 반복되도록 설정할 수 있습니다:

1. 이벤트를 만들거나 편집할 때 **Recurring** 상자를 확인합니다.
2. 이벤트를 반복하는 방법을 선택합니다:
   - **Interval** -- 발생 사이의 날 수를 설정합니다. 예를 들어 "7"을 입력하면 7일마다 이벤트가 반복됩니다.
   - **Frequency** -- **day**, **week** 또는 **month**로 반복하도록 선택합니다.
3. 이벤트를 저장합니다. 이제 예정된 각 발생 시 일정에 나타납니다.

:::tip
매주 토요일에 만나는 그룹의 경우 빈도를 "week"로 설정하고 토요일을 선택합니다. 이벤트는 앞으로 일정에 자동으로 나타납니다.
:::

## B1 Admin에서 대량으로 이벤트 생성

B1 Admin의 그룹 페이지에는 직원이 전체 시즌의 회의를 한 번에 예약할 수 있는 **Calendar** 탭도 있습니다:

1. B1 Admin에서 그룹을 열고 **Calendar** 탭을 선택합니다.
2. **Add Events**를 클릭하여 **Add Recurring Events** 대화 상자를 엽니다.
3. 일정(주간, 2주마다, 월간 또는 일회), **First Meeting** 및 **Last Meeting** 날짜, 선택적으로 **Skip holidays**를 선택합니다.
4. 회의 날짜의 미리보기를 검토하고 건너뛸 항목을 선택 해제합니다.
5. 멤버가 이 이벤트에 응답할 수 있기를 원하면 **Allow RSVPs**를 선택된 상태로 두고 저장합니다(아래 참조).

## 이벤트 RSVP

그룹 이벤트는 이벤트의 각 발생에 대해 **Yes**, **No** 또는 **Maybe**인 RSVP를 수집할 수 있습니다.

- **Allow RSVPs**는 대량 추가 대화 상자를 통해 만든 이벤트에 대해 기본적으로 설정됩니다. 응답을 끄려면 선택을 해제합니다. RSVP가 꺼진 이벤트는 일정 탭에서 **Disabled**로 표시됩니다.
- B1 Admin 그룹 일정 탭에서 각 이벤트는 응답 요약을 표시합니다(예: "5 yes · 2 maybe · 1 no"). **RSVP Responses** 명단을 열려면 클릭하여 해당 발생에 대해 yes, maybe, no로 응답한 사람을 나열합니다.

:::tip
RSVP는 리더에게 전체 이벤트 등록의 공식성 없이 인원 파악을 제공합니다. 유료 등록 또는 용량 관리 이벤트의 경우 대신 [event registration](../calendars/creating-calendars.md#enabling-event-registration)을 사용합니다.
:::

## 일정 보기

그룹 일정은 여러 보기 옵션을 제공합니다:

- **Month** -- 전체 월을 한눈에 봅니다.
- **Week** -- 현재 주에 집중합니다.
- **Day** -- 단일 날짜를 자세히 봅니다.
- **Agenda** -- 시간 순서로 다가오는 이벤트 목록을 봅니다.

일정 위쪽의 버튼을 사용하여 보기 사이를 전환합니다.

## 이벤트 세부 정보 및 대화

이벤트를 클릭하여 세부 정보를 봅니다. 이벤트 세부 보기에서:

- **Members and leaders** 이벤트에 대해 대화를 시작하고 의견을 남길 수 있습니다.
- **Leaders** 이벤트 세부 정보를 편집하거나, 시간을 변경하거나, 이벤트를 삭제할 수 있습니다.

:::warning
이벤트를 삭제하면 영구적으로 제거됩니다. 반복 이벤트의 단일 발생을 취소해야 하는 경우 대신 편집하는 것이 좋습니다.
:::

## 일정 구독

그룹 일정을 개인 일정과 동기화할 수 있습니다:

1. 그룹 일정 페이지에서 **Subscribe**를 클릭합니다.
2. 지침을 따라 Google Calendar, Apple Calendar, Outlook 또는 구독을 지원하는 다른 일정 애플리케이션에 일정을 추가합니다.

:::tip
구독하면 B1을 따로 확인할 필요 없이 개인 일정을 그룹의 이벤트와 동기화된 상태로 유지합니다. 그룹 리더가 수행한 모든 변경 사항이 동기화된 일정에 자동으로 나타납니다.
:::
