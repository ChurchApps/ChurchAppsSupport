# 그룹 회원

<div class="article-intro">

그룹을 만든 후 다음 단계는 회원을 추가하는 것입니다. 그룹의 세부 페이지에서 사람을 검색하고, 그룹에 추가하고, 지도자를 지정하고, 메시지를 보내고, 회원 목록을 내보낼 수 있습니다. 그룹 회원 관리는 소그룹, 위원회, 클래스를 조정하는 데 필수적입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin에 최소한 하나의 그룹이 설정되어 있어야 합니다. 아직 만들지 않았으면 [그룹 만들기](creating-groups.md)를 참조하세요.
- 추가하려는 사람은 이미 [사람](../people/adding-people.md) 디렉터리에 있어야 합니다.

</div>

## 그룹에 회원 추가

1. **그룹** 페이지로 이동하고 관리할 그룹을 클릭합니다.
2. **회원** 탭을 클릭합니다.
3. 검색 상자에 추가할 사람의 이름을 입력합니다.
4. 검색 결과에서 사람의 이름 옆에 있는 **추가**를 클릭합니다.
5. 그 사람이 이제 그룹 회원 목록에 나타납니다.

:::tip
검색 상자를 비우고 **검색**을 클릭하여 전체 디렉터리를 탐색합니다. 누군가의 정확한 철자가 확실하지 않을 때 유용합니다.
:::

## 그룹 지도자 지정

그룹 지도자는 특별한 권한이 있습니다. [그룹 캘린더](group-calendar.md)를 편집하고, 이벤트를 관리하고, 그룹을 조정하는 데 도움이 될 수 있습니다.

1. 그룹 회원 목록에서 지도자로 만들 사람을 찾습니다.
2. 이름 옆의 **녹색 키 아이콘**을 클릭합니다.
3. 그 사람이 이제 그룹 지도자로 지정됩니다.

지도자 상태를 제거하려면 녹색 키 아이콘을 다시 클릭합니다.

:::info
모든 그룹 회원은 그룹 캘린더와 이벤트를 볼 수 있지만, 지도자만 캘린더 이벤트를 추가하거나 편집할 수 있습니다.
:::

## 그룹 회원에게 메시지 보내기

B1 Admin에서 직접 그룹의 모든 회원과 소통할 수 있습니다:

1. 그룹 세부 페이지에서 메시징 영역을 찾습니다.
2. 텍스트 상자에 메시지를 입력합니다.
3. **보내기**를 클릭합니다.

메시지가 그룹의 모든 회원에게 전달됩니다.

## 그룹 회원에게 이메일 보내기

그룹의 모든 회원에게 형식이 지정된 이메일을 보낼 수 있습니다:

1. 그룹 세부 페이지에서 **이메일 아이콘**을 클릭합니다.
2. 이메일 보내기 대화가 열리고 이메일을 받을 회원 수와 파일에 이메일 주소가 없는 회원 수를 표시합니다.
3. 선택적으로 드롭다운에서 **이메일 템플릿**을 선택하거나 처음부터 메시지를 작성합니다. **템플릿 관리**를 클릭하여 템플릿을 만들거나 편집합니다.
4. **제목 줄**을 입력합니다. 필드 칩을 클릭하여 병합 필드를 삽입할 수 있습니다: `{{firstName}}`, `{{lastName}}`, `{{displayName}}`, `{{email}}`, `{{churchName}}`.
5. HTML 편집기를 사용하여 **이메일 본문**을 작성합니다. 동일한 병합 필드를 사용할 수 있습니다.
6. **보내기**를 클릭합니다.
7. 요약에 성공적으로 보낸 이메일 수와 건너뛴 회원 수(파일에 이메일이 없음)가 표시됩니다.

:::tip
주간 업데이트, 이벤트 공지, 기도 요청 같은 반복적인 소통을 위해 재사용 가능한 이메일 템플릿을 만듭니다. 템플릿은 시간을 절약하고 일관된 메시징을 보장합니다.
:::

## 그룹 데이터 내보내기

그룹 회원 목록을 파일로 다운로드하려면:

1. 그룹 세부 페이지에서 **다운로드 아이콘**을 클릭합니다.
2. 그룹의 회원 정보가 포함된 CSV 파일이 컴퓨터에 다운로드됩니다.

이는 인쇄된 명부를 만들거나, 데이터를 다른 도구로 가져오거나, 오프라인 기록을 유지하는 데 유용합니다. 더 많은 내보내기 옵션은 [데이터 내보내기](../people/exporting-data.md)를 참조하세요.

## 회원 제거

그룹에서 누군가를 제거하려면 회원 목록에서 이름을 찾아 항목 옆의 **제거** 버튼을 클릭합니다.

:::info
사람을 그룹에서 제거해도 교회 디렉터리에서 삭제되지 않습니다. [사람](../people/adding-people.md) 섹션에 여전히 나타나며 언제든 그룹에 다시 추가할 수 있습니다.
:::
