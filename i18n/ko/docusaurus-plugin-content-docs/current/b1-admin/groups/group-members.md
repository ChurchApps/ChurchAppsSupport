# 그룹 멤버

<div class="article-intro">

그룹을 생성한 후 다음 단계는 멤버를 추가하는 것입니다. 그룹 상세 페이지에서 사람을 검색하고, 그룹에 추가하고, 리더를 지정하고, 메시지를 보내고, 멤버 목록을 내보낼 수 있습니다. 그룹 멤버십 관리는 소그룹, 위원회, 수업을 조율하는 데 필수적입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin에 최소 1개 이상의 그룹이 설정되어 있어야 합니다. 아직 그룹을 생성하지 않았다면 [그룹 생성하기](creating-groups.md)를 참고하세요.
- 추가하려는 사람들은 이미 [사람](../people/adding-people.md) 디렉토리에 존재해야 합니다.

</div>

## 그룹에 멤버 추가하기

1. **그룹** 페이지로 이동하고 관리하려는 그룹을 클릭합니다.
2. **멤버** 탭을 클릭합니다.
3. 검색 상자에 추가하려는 사람의 이름을 입력합니다.
4. 검색 결과에서 해당 사람 이름 옆의 **추가**를 클릭합니다.
5. 이제 그 사람이 그룹 멤버 목록에 표시됩니다.

:::tip
검색 상자를 비워두고 **검색**을 클릭하면 전체 디렉토리를 탐색할 수 있습니다. 이는 누군가의 정확한 이름의 철자를 모를 때 유용합니다.
:::

## 그룹 리더 지정하기

그룹 리더는 특별한 권한을 가집니다. 그들은 [그룹 캘린더](group-calendar.md)를 편집하고, 이벤트를 관리하고, 그룹을 조율하는 것을 도울 수 있습니다.

1. 그룹 멤버 목록에서 리더로 만들려는 사람을 찾습니다.
2. 그들의 이름 옆에 있는 **초록색 키 아이콘**을 클릭합니다.
3. 이제 그 사람은 그룹 리더로 지정됩니다.

리더 상태를 제거하려면 초록색 키 아이콘을 다시 클릭합니다.

:::info
모든 그룹 멤버는 그룹 캘린더와 이벤트를 볼 수 있지만, 리더만 캘린더 이벤트를 추가하거나 편집할 수 있습니다.
:::

## 그룹 멤버에게 메시지 보내기

B1 Admin에서 직접 그룹의 모든 멤버와 통신할 수 있습니다:

1. 그룹 상세 페이지에서 메시징 영역을 찾습니다.
2. 텍스트 상자에 메시지를 입력합니다.
3. **보내기**를 클릭합니다.

메시지는 그룹의 모든 멤버에게 전달됩니다.

## 그룹 멤버에게 이메일 보내기

그룹의 모든 멤버에게 형식이 지정된 이메일을 보낼 수 있습니다:

1. 그룹 상세 페이지에서 **이메일 아이콘**을 클릭합니다.
2. 이메일 보내기 대화상자가 열리고 얼마나 많은 멤버가 이메일을 받을 것인지, 그리고 파일에 이메일 주소가 없는 멤버가 몇 명인지 보여줍니다.
3. 드롭다운에서 **이메일 템플릿**을 선택하거나, 처음부터 메시지를 작성합니다. **템플릿 관리**를 클릭하여 템플릿을 만들거나 편집합니다.
4. **제목 줄**을 입력합니다. 필드 칩을 클릭하여 병합 필드를 삽입할 수 있습니다: `{{firstName}}`, `{{lastName}}`, `{{displayName}}`, `{{email}}`, `{{churchName}}`.
5. HTML 편집기를 사용하여 **이메일 본문**을 작성합니다. 동일한 병합 필드를 여기에서도 사용할 수 있습니다.
6. **보내기**를 클릭합니다.
7. 요약에는 몇 개의 이메일이 성공적으로 발송되었고 몇 명의 멤버가 건너뛰어졌는지(파일에 이메일이 없음) 표시됩니다.

:::tip
주간 업데이트, 이벤트 공지사항, 기도 요청 등 반복되는 통신을 위해 재사용 가능한 이메일 템플릿을 만듭니다. 템플릿은 시간을 절약하고 일관된 메시징을 보장합니다.
:::

## 그룹 데이터 내보내기

그룹 멤버 목록을 파일로 다운로드하려면:

1. 그룹 상세 페이지에서 **다운로드 아이콘**을 클릭합니다.
2. 그룹의 멤버 정보가 포함된 CSV 파일이 컴퓨터에 다운로드됩니다.

이는 인쇄된 명부를 만들거나, 데이터를 다른 도구로 가져오거나, 오프라인 기록을 유지하는 데 유용합니다. 더 많은 내보내기 옵션은 [데이터 내보내기](../people/exporting-data.md)를 참고하세요.

## 그룹 멤버에게 푸시 알림 보내기

기기에 B1.church 앱을 설치했고 푸시 알림을 활성화한 모든 그룹 멤버에게 푸시 알림을 직접 보낼 수 있습니다.

1. 그룹 상세 페이지에서 헤더 툴바의 **벨 아이콘**(이메일과 SMS 아이콘 옆)을 클릭합니다.
2. 그룹의 몇 명의 멤버가 푸시를 활성화했는지 보여주는 대화상자가 열립니다.
3. 알림 세부 정보를 입력합니다:
   - **제목** *(필수)* -- 짧은 요약으로, 최대 80자입니다.
   - **메시지** *(필수)* -- 알림 본문으로, 최대 240자입니다.
   - **링크 열기 또는 전단지 URL** *(선택사항)* -- 상대 앱 경로(예: `/mobile/groups`) 또는 알림이 탭될 때 열 수 있는 전체 `https://` URL입니다.
   - **이미지 URL** *(선택사항)* -- 지원되는 기기에서 알림 옆에 표시되는 이미지의 `https://` URL입니다.
4. 라이브 미리보기는 알림이 기기에서 어떻게 보일지 보여줍니다.
5. **알림 보내기**를 클릭합니다.

:::info
푸시 알림은 B1.church PWA를 설치했고 푸시 알림을 비활성화하지 않은 그룹 멤버에게만 전달됩니다. 등록된 푸시 기기가 없거나 푸시가 꺼진 멤버는 건너뛴 것으로 계산되며, 전송 요약에는 도달한 멤버 수 대 건너뛴 멤버 수가 표시됩니다.
:::

:::tip
전송 후 대화상자는 몇 개의 알림이 성공적으로 대기열에 추가되었는지 보여줍니다. 대부분의 멤버가 건너뛴 것으로 표시되는 경우, 멤버들에게 B1.church 사이트를 방문하고, 홈 화면 앱으로 설치하고, 메시지가 표시될 때 알림을 허용하도록 상기시킵니다.
:::

## 멤버 제거하기

누군가를 그룹에서 제거하려면, 멤버 목록에서 그들의 이름을 찾고 그들의 항목 옆에 있는 **제거** 버튼을 클릭합니다.

:::info
그룹에서 사람을 제거해도 교회 디렉토리에서 삭제되지 않습니다. 그들은 여전히 [사람](../people/adding-people.md) 섹션에 나타나며 언제든 그룹에 다시 추가할 수 있습니다.
:::
