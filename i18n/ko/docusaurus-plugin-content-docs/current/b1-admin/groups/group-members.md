# 그룹 구성원

<div class="article-intro">

그룹을 만든 후 다음 단계는 구성원을 추가하는 것입니다. 그룹 상세 페이지에서 사람을 검색하고, 그룹에 추가하고, 리더를 지정하고, 메시지를 보내고, 구성원 목록을 내보낼 수 있습니다. 그룹 멤버십 관리는 사역부서, 소그룹, 팀을 조율하는 데 필수적입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Admin에 최소 하나의 그룹이 설정되어 있어야 합니다. 아직 만들지 않았다면 [그룹 만들기](creating-groups.md)를 참조하세요.
- 추가하려는 사람이 이미 [교인 관리](../people/adding-people.md) 명부에 존재해야 합니다.

</div>

## 그룹에 구성원 추가

1. **그룹** 페이지로 이동하여 관리하려는 그룹을 클릭합니다.
2. **구성원** 탭을 클릭합니다.
3. 검색 상자에 추가하려는 사람의 이름을 입력합니다.
4. 검색 결과에서 해당 사람의 이름 옆 **추가**를 클릭합니다.
5. 해당 사람이 그룹 구성원 목록에 나타납니다.

:::tip
검색 상자를 비워두고 **검색**을 클릭하면 전체 명부를 탐색할 수 있습니다. 이름의 정확한 철자를 모르는 경우 유용합니다.
:::

## 그룹 리더 지정

그룹 리더는 특별한 권한을 가집니다 -- [그룹 일정](group-calendar.md)을 편집하고, 행사를 관리하며, 그룹을 조율하는 데 도움을 줄 수 있습니다.

1. 그룹 구성원 목록에서 리더로 지정하려는 사람을 찾습니다.
2. 이름 옆의 **녹색 열쇠 아이콘**을 클릭합니다.
3. 해당 사람이 그룹 리더로 지정됩니다.

리더 상태를 제거하려면 녹색 열쇠 아이콘을 다시 클릭합니다.

:::info
모든 그룹 구성원은 그룹 일정과 행사를 볼 수 있지만, 일정 행사를 추가하거나 편집할 수 있는 것은 리더만 가능합니다.
:::

## 그룹 구성원에게 메시지 보내기

B1 Admin에서 직접 그룹의 모든 구성원에게 소통할 수 있습니다:

1. 그룹 상세 페이지에서 메시지 영역을 찾습니다.
2. 텍스트 상자에 메시지를 입력합니다.
3. **전송**을 클릭합니다.

메시지가 그룹의 모든 구성원에게 전달됩니다.

## 그룹 데이터 내보내기

그룹 구성원 목록을 파일로 다운로드하려면:

1. 그룹 상세 페이지에서 **다운로드 아이콘**을 클릭합니다.
2. 그룹의 구성원 정보가 포함된 CSV 파일이 컴퓨터에 다운로드됩니다.

이는 인쇄용 명단 작성, 다른 도구로 데이터 가져오기, 오프라인 기록 보관에 유용합니다. 더 많은 내보내기 옵션은 [데이터 내보내기](../people/exporting-data.md)를 참조하세요.

## 구성원 제거

그룹에서 사람을 제거하려면 구성원 목록에서 해당 이름을 찾고 항목 옆의 **제거** 버튼을 클릭합니다.

:::info
그룹에서 사람을 제거해도 교회 명부에서 삭제되지 않습니다. 해당 사람은 여전히 [교인 관리](../people/adding-people.md) 섹션에 나타나며, 언제든지 그룹에 다시 추가할 수 있습니다.
:::
