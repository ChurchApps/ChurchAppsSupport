# 파일

<div class="article-intro">

파일 페이지는 교회 웹사이트에서 사용되는 모든 미디어와 문서의 중앙 저장소입니다. 여기서 사이트에 필요한 모든 이미지, 문서, 미디어 파일을 업로드, 정리, 탐색, 관리할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 웹사이트의 [초기 설정](initial-setup)을 완료하십시오
- 페이지 로딩 속도를 높이기 위해 업로드 전에 이미지를 최적화하십시오
- 일반적인 웹 형식(PNG, JPG, PDF 등)으로 파일을 준비하십시오

</div>

## 파일 업로드

1. B1 Admin의 **웹사이트** 섹션으로 이동하여 **파일** 탭을 클릭합니다.
2. 업로드 영역에 파일을 직접 드래그 앤 드롭하거나, 업로드 버튼을 클릭하여 컴퓨터에서 파일을 탐색합니다.
3. 파일이 업로드되면 파일 라이브러리에 즉시 나타납니다.

:::tip
최상의 결과를 위해 업로드 전에 이미지를 최적화하십시오. 파일 크기가 작을수록 방문자에게 웹사이트가 더 빠르게 로드됩니다. TinyPNG나 Squoosh 같은 도구로 눈에 띄는 품질 저하 없이 이미지를 압축할 수 있습니다.
:::

## 폴더로 정리

1. 관련 파일을 그룹으로 묶기 위해 폴더를 만드십시오 -- 예를 들어, "설교 그래픽," "행사 배너," "교직원 사진."
2. 라이브러리가 커지면 폴더 간에 파일을 이동하여 깔끔하게 유지하십시오.
3. 팀이 파일을 빠르게 찾을 수 있도록 명확하고 설명적인 폴더 이름을 사용하십시오.

## 탐색 및 검색

- 파일 라이브러리를 스크롤하여 업로드된 모든 파일의 썸네일을 확인합니다.
- 라이브러리가 커지면 **검색** 기능을 사용하여 이름으로 파일을 찾습니다.
- 파일을 클릭하면 파일 이름, 크기, URL 등의 세부 정보를 확인할 수 있습니다.

## 웹사이트에서 파일 사용

1. 파일을 클릭하여 세부 정보를 확인합니다.
2. **파일 URL을 복사**하여 웹사이트 페이지에서 사용합니다. 이 URL을 이미지 요소, 다운로드 링크 또는 파일 URL을 허용하는 어디에든 붙여넣을 수 있습니다.
3. 여기에 업로드한 파일은 모든 웹사이트 페이지와 콘텐츠 블록에서 사용할 수 있습니다.

:::info
[페이지 관리](managing-pages)에서 페이지를 구성할 때, 여기에 업로드된 파일을 URL로 참조할 수 있습니다. 이를 통해 모든 미디어가 하나의 중앙 위치에서 정리됩니다.
:::

## 파일 삭제

1. 제거하려는 파일을 선택합니다.
2. **삭제** 버튼을 클릭하여 라이브러리에서 영구적으로 제거합니다.
3. 삭제하기 전에 해당 파일이 현재 어떤 페이지에서도 사용되고 있지 않은지 확인하십시오. 사용 중인 파일을 삭제하면 해당 파일을 참조하는 링크나 이미지가 깨집니다.

:::warning
삭제된 파일은 복구할 수 없습니다. 파일을 제거하기 전에 어떤 페이지에서도 더 이상 사용되지 않는지 다시 한번 확인하십시오.
:::
