# 웹사이트

<div class="article-intro">

B1 Admin의 웹사이트 섹션은 교회를 위한 완전한 기능의 웹사이트 빌더를 제공합니다. 코드 작성 없이도 페이지를 만들고, 사이트의 디자인을 맞춤 설정하며, 미디어 파일을 관리하고, 행사 캘린더를 게시할 수 있습니다.

</div>

## 수행할 수 있는 작업

웹사이트 빌더는 다섯 가지 주요 영역으로 구성되어 있습니다:

### 페이지

교회 웹사이트의 모든 페이지를 만들고 관리합니다. **페이지** 테이블에는 각 페이지와 함께 상태가 표시됩니다 -- **자동 생성**(시스템에 의해 자동으로 생성됨) 또는 **맞춤**(직접 생성함). 새 맞춤 페이지를 추가하거나, 기존 페이지를 편집하거나, 자동 생성된 페이지를 맞춤 페이지로 전환하여 콘텐츠를 완전히 제어할 수 있습니다.

왼쪽 사이드바에는 내비게이션 링크가 트리 구조로 표시됩니다. 드래그 앤 드롭으로 링크 순서를 변경하여 방문자에게 보여줄 정확한 계층 구조와 순서를 설정할 수 있습니다.

### 블록

여러 페이지에 배치할 수 있는 재사용 가능한 콘텐츠 구성 요소를 만듭니다. 행동 유도 문구, 공지 배너, 연락처 섹션 등 여러 곳에 나타나는 콘텐츠가 있을 때 블록을 사용하면 시간을 절약할 수 있습니다. 블록을 한 번 수정하면 사용된 모든 곳에 변경 사항이 적용됩니다.

### 외관

웹사이트의 테마, 색상, 글꼴, 로고, 간격을 맞춤 설정합니다. 빠른 시작을 위해 사전 구축된 스타일 테마를 적용하거나, 색상 팔레트 편집기, 타이포그래피 설정, 맞춤 CSS로 모든 세부 사항을 미세 조정할 수 있습니다. 자세한 내용은 [외관](appearance) 가이드를 참조하십시오.

### 파일

모든 미디어 자산(이미지, 문서 및 기타 파일)을 한 곳에서 업로드하고 관리합니다. 폴더를 사용하여 정리하고, 사이트 어디에서나 사용할 수 있도록 파일 URL을 복사합니다. 자세한 내용은 [파일](files)을 참조하십시오.

### 캘린더

웹사이트에 행사 캘린더를 게시하여 방문자가 교회에서 진행되는 일을 확인할 수 있도록 합니다. 여러 그룹의 행사를 하나의 뷰로 모아 선별된 캘린더를 만듭니다.

:::tip
웹사이트를 처음 설정하는 경우 [초기 설정](initial-setup)부터 시작하십시오. 도메인 설정, 첫 페이지 만들기, 사이트 외관 설정 과정을 안내합니다.
:::
