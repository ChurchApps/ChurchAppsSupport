# 페이지 관리

<div class="article-intro">

웹사이트 페이지 보기는 교회 웹사이트의 모든 페이지를 생성, 편집, 구성하는 중심 허브입니다. 이 단일 화면에서 페이지 콘텐츠와 사이트 내비게이션을 모두 관리할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [초기 설정](initial-setup)을 완료하여 도메인과 기본 사이트 설정을 구성하십시오
- 콘텐츠와 이미지를 준비하십시오. [파일](files) 관리자를 사용하여 미디어 자산을 먼저 업로드하십시오.

</div>

## 페이지 유형 이해

**페이지** 테이블에는 사이트의 모든 페이지와 상태가 나열됩니다:

- **자동 생성** -- 교회 데이터를 기반으로 시스템이 자동으로 생성한 페이지입니다 (예: 그룹 페이지 또는 설교 페이지). 이 페이지들은 데이터가 변경되면 자동으로 업데이트됩니다.
- **맞춤** -- 직접 만든 콘텐츠와 레이아웃으로 구성된 페이지입니다.

콘텐츠와 디자인을 완전히 제어하려면 자동 생성 페이지를 맞춤 페이지로 전환할 수 있습니다.

## 페이지 추가 및 편집

1. 페이지 테이블 오른쪽 상단의 **페이지 추가** 버튼을 클릭합니다.
2. 페이지 유형(빈 페이지 또는 템플릿)을 선택하고 이름을 지정합니다.
3. 페이지 옆의 **편집**을 클릭하여 페이지 빌더를 열고, 섹션, 텍스트, 이미지 및 기타 요소를 추가합니다.
4. **페이지 설정**을 클릭하여 페이지 제목, URL 경로 및 기타 메타데이터를 업데이트합니다.
5. **미리 보기** 버튼을 사용하여 새 창에서 페이지를 열고 방문자에게 어떻게 보일지 정확히 확인합니다.

:::tip
홈 페이지의 URL 경로는 `/`만 설정하십시오. 다른 모든 페이지에는 `/about` 또는 `/contact` 같은 설명적인 경로를 사용하십시오.
:::

## 내비게이션 관리

웹사이트 페이지 보기의 왼쪽 사이드바에 내비게이션 링크가 표시됩니다. 이 링크는 방문자가 웹사이트에서 보는 메뉴를 제어합니다.

1. **추가**를 클릭하여 새 내비게이션 링크를 만듭니다. 사이트의 페이지나 외부 URL을 지정할 수 있습니다.
2. 링크 순서를 변경하려면 드래그 앤 드롭을 사용합니다. 드롭다운 메뉴를 만들려면 상위 항목 아래에 링크를 중첩할 수도 있습니다.
3. 링크 옆의 **편집** 아이콘을 클릭하여 레이블, URL 또는 위치를 변경합니다.
4. 내비게이션에서 링크를 제거하려면 **삭제** 아이콘을 클릭합니다.

:::info
내비게이션 링크를 제거해도 페이지 자체가 삭제되는 것은 아닙니다. 페이지는 여전히 존재하며 URL로 직접 접근할 수 있습니다 -- 단지 메뉴에 표시되지 않을 뿐입니다.
:::

## 사이트 구성 팁

- 방문자가 빠르게 원하는 것을 찾을 수 있도록 최상위 내비게이션을 다섯 개 또는 여섯 개 항목으로 유지하십시오.
- 관련 하위 페이지에 중첩 링크를 사용하십시오 (예: "소개" 드롭다운 아래에 "우리 팀," "신앙 고백," "연혁").
- **모바일 미리 보기**를 클릭하여 모바일에서 내비게이션을 검토하고 작은 화면에서도 잘 작동하는지 확인하십시오.
- 방문자가 무엇을 찾을 수 있는지 이해할 수 있도록 페이지에 명확하고 설명적인 이름을 지정하십시오.

:::tip
방문자로부터 등록, 기도 요청 또는 기타 정보를 수집하기 위해 페이지에 [양식](../forms/creating-forms.md)을 추가할 수 있습니다.
:::
