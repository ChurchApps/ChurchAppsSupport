# 페이지 관리

<div class="article-intro">

웹사이트 페이지 보기는 교회 웹사이트의 모든 페이지를 생성, 편집, 정렬하기 위한 중앙 허브입니다. 이 한 화면에서 페이지 콘텐츠와 사이트의 네비게이션을 모두 관리할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 도메인과 기본 사이트 설정을 구성하려면 [초기 설정](initial-setup)을 완료하세요
- 콘텐츠와 이미지를 준비합니다. [파일](files) 관리자를 사용하여 먼저 미디어 자산을 업로드합니다.

</div>

## 페이지 유형 이해하기

**페이지** 표에는 웹사이트의 모든 페이지와 그 상태가 나열됩니다:

- **생성됨** -- 교회 데이터를 기반으로 시스템에서 자동으로 생성된 페이지(예: 그룹 페이지 또는 설교 페이지)입니다. 이 페이지들은 데이터가 변경되면 자동으로 업데이트됩니다.
- **사용자 정의** -- 자신의 콘텐츠와 레이아웃으로 직접 생성한 페이지입니다.

자동 생성된 페이지를 사용자 정의 페이지로 변환하여 콘텐츠와 디자인을 완전히 제어할 수 있습니다.

## 페이지 추가 및 편집하기

1. 페이지 표의 오른쪽 상단에 있는 **페이지 추가** 버튼을 클릭합니다.
2. 페이지 유형(빈 페이지 또는 템플릿)을 선택하고 이름을 지정합니다.
3. 페이지 옆의 **편집**을 클릭하여 [페이지 편집기](page-editor)를 열고, 섹션, 텍스트, 이미지 및 기타 요소를 추가할 수 있습니다.
4. **페이지 설정**을 클릭하여 페이지 제목, URL 경로 및 기타 메타데이터를 업데이트합니다.
5. **미리보기** 버튼을 사용하여 새 창에서 페이지를 열고 방문자에게 정확히 어떻게 보일지 확인합니다.

:::tip
홈 페이지의 경우 URL 경로를 `/`로 설정합니다. 다른 모든 페이지의 경우 `/about` 또는 `/contact`와 같은 설명적인 경로를 사용합니다.
:::

## 네비게이션 관리하기

웹사이트 페이지 보기의 왼쪽 사이드바는 네비게이션 링크를 표시합니다. 이 링크들은 방문자가 웹사이트에서 보는 메뉴를 제어합니다.

1. **추가**를 클릭하여 새 네비게이션 링크를 만듭니다. 사이트의 모든 페이지나 외부 URL로 연결할 수 있습니다.
2. 링크를 재정렬하려면 드래그 앤 드롭으로 원하는 순서로 이동합니다. 부모 항목 아래에 링크를 중첩하여 드롭다운 메뉴를 만들 수도 있습니다.
3. 링크 옆의 **편집** 아이콘을 클릭하여 레이블, URL 또는 위치를 변경합니다.
4. 네비게이션에서 링크를 제거하려면 **삭제** 아이콘을 클릭합니다.

:::info
네비게이션 링크를 제거해도 페이지 자체는 삭제되지 않습니다. 페이지는 여전히 존재하며 URL로 직접 액세스할 수 있습니다. 단순히 메뉴에 나타나지 않을 뿐입니다.
:::

## 사이트 정렬 팁

- 상단 레벨 네비게이션을 5~6개 항목으로 유지하여 방문자가 빠르게 찾을 수 있도록 합니다.
- 관련 하위 페이지에 중첩된 링크를 사용합니다(예: "회사소개" 드롭다운에서 "팀", "신앙", "역사").
- **모바일 미리보기**를 클릭하여 모바일에서 네비게이션이 잘 작동하는지 확인합니다.
- 페이지에 명확한 설명적인 이름을 지어 방문자들이 무엇을 찾을 수 있을지 이해할 수 있도록 합니다.

:::tip
[양식](../forms/creating-forms.md)을 페이지에 추가하여 방문자로부터 등록, 기도 요청 또는 기타 정보를 수집할 수 있습니다.
:::

## 사이트 템플릿에서 시작하기

처음부터 사이트를 구축하는 경우, 페이지를 하나씩 만드는 대신 **사이트 템플릿**을 사용하여 부트스트랩할 수 있습니다. 사이트 템플릿은 사전 구축된 페이지 집합(홈, 회사소개, 연결, 기부 등)을 만들고 이미 연결된 자리 표시자 콘텐츠와 네비게이션 링크가 포함되어 있습니다.

1. 페이지 화면에서 **사이트 템플릿** 버튼(**페이지 추가** 버튼 옆)을 클릭합니다.
2. 사용 가능한 템플릿을 찾아보고 하나를 클릭하여 페이지 구조를 미리봅니다.
3. 마음에 드는 것을 찾으면 **템플릿 적용**을 클릭합니다.
4. 아직 존재하지 않는 페이지가 생성되고 네비게이션에 추가됩니다. 기존 페이지는 그대로 유지됩니다.

템플릿을 적용한 후 [페이지 편집기](page-editor)에서 각 페이지를 열어 자리 표시자 텍스트와 이미지를 교회의 실제 콘텐츠로 바꿉니다.

:::info
사이트 템플릿은 페이지 구조와 네비게이션을 만듭니다. 사이트의 색 구성표나 글꼴을 재정의하지 않습니다. 이는 [모양새](appearance)로 제어됩니다.
:::

## 이미지 라이트박스

방문자가 웹사이트의 이미지를 클릭하면 전체 화면 라이트박스 오버레이에서 열립니다. 이를 통해 사람들은 페이지를 떠나지 않고 더 큰 크기로 사진을 볼 수 있습니다. 구성이 필요하지 않습니다. 라이트박스는 페이지 콘텐츠의 이미지에 대해 자동으로 활성화됩니다.
