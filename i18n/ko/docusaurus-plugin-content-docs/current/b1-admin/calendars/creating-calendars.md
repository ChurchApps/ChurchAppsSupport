# 캘린더 생성하기

<div class="article-intro">

B1 Admin에서 캘린더를 생성하면 하나 이상의 그룹을 연결하여 선별된 이벤트 보기를 만들 수 있습니다. 이벤트는 그룹 리더가 각자의 그룹 내에서 관리하며, 캘린더는 해당 이벤트를 한 곳에 표시합니다. 도메인 관리자라 하더라도 이벤트가 속한 그룹의 리더가 아닌 경우 캘린더 섹션에서 직접 이벤트를 추가하거나 편집할 수 없습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 캘린더에 포함할 이벤트가 있는 [그룹](../groups/creating-groups.md)을 설정하세요
- B1 Admin의 캘린더 섹션에 대한 관리자 액세스 권한이 필요합니다

</div>

## 새 캘린더 생성하기

1. B1 Admin에서 **웹사이트**로 이동한 다음 **캘린더** 섹션으로 이동합니다.
2. **캘린더 추가**를 클릭합니다.
3. 캘린더의 **이름**을 입력합니다(예: "청소년 사역 이벤트" 또는 "메인 교회 캘린더").
4. 팀이 이 캘린더의 용도를 이해할 수 있도록 선택적으로 **설명**을 추가합니다.
5. **생성**을 클릭하여 새 캘린더를 저장합니다.

## 캘린더 상세 페이지

캘린더를 생성한 후 클릭하여 상세 페이지를 엽니다. 이 페이지에는 두 가지 주요 영역이 있습니다:

- **왼쪽 열** -- 연결된 그룹에서 가져온 이벤트를 보여주는 캘린더 보기.
- **오른쪽 열** -- 연결된 그룹 목록. 여기에서 이 캘린더에 포함될 그룹을 관리합니다.

## 그룹 연결하기

캘린더에 이벤트가 있는 그룹은 상세 페이지의 오른쪽에 있는 그룹 목록에 자동으로 표시됩니다.

1. 그룹 섹션에서 **추가**를 클릭하여 그룹을 캘린더와 연결합니다.
2. 드롭다운에서 그룹을 선택합니다.
3. 해당 그룹의 **모든 이벤트**를 포함할지 아니면 **특정 이벤트**만 포함할지 선택합니다.
4. **저장**을 클릭합니다.

:::tip
캘린더에 그룹을 연결하는 것은 이벤트를 자동으로 집계하는 강력한 방법입니다. 그룹 리더가 자신의 [그룹](../groups/creating-groups.md)에 이벤트를 추가하면 추가 작업 없이 교회 전체 캘린더에 자동으로 표시됩니다.
:::

:::info
교회 전체의 여러 그룹에서 이벤트를 가져오는 단일 캘린더를 만들고 싶다면 간소화된 접근 방식을 위해 [선별된 캘린더](curated-calendar)를 참조하세요.
:::

## 이벤트 등록 활성화하기

캘린더 이벤트에 대해 등록을 활성화하여 회원들이 B1 웹사이트 또는 모바일 앱을 통해 등록할 수 있습니다.

1. 기존 이벤트를 클릭하거나 새 이벤트를 생성합니다.
2. 이벤트 편집기에서 **등록**을 토글하여 활성화합니다.
3. 등록 설정을 구성합니다:
   - **수용 인원** (선택 사항) -- 최대 등록 인원을 설정합니다. 무제한으로 하려면 비워둡니다.
   - **등록 시작** -- 등록이 가능해지는 날짜와 시간.
   - **등록 마감** -- 등록이 마감되는 날짜와 시간.
   - **태그** -- 등록 가능한 이벤트를 분류하는 데 도움이 되는 쉼표로 구분된 레이블(예: "youth, retreat, vbs").
4. 이벤트를 저장합니다.

등록이 활성화되면 회원들은 [B1 웹사이트](../../b1-church/events/registering) 또는 [B1 Mobile 앱](../../b1-mobile/events/registering)에서 이벤트를 볼 때 **이 이벤트에 등록하기** 버튼을 보게 됩니다.

### 등록 관리하기

이벤트 등록을 보고 관리하려면:

1. B1 Admin에서 **등록** 페이지로 이동합니다.
2. 등록이 활성화된 모든 이벤트의 테이블이 표시되며, 이벤트 제목, 날짜, 현재 등록 수 대 수용 인원, 태그가 표시됩니다.
3. 이벤트를 클릭하여 이름, 회원 수, 상태, 등록 날짜를 포함한 전체 등록 목록을 확인합니다.
4. 상세 페이지에서 다음을 수행할 수 있습니다:
   - 개별 등록 **취소**
   - 등록 영구 **삭제**
   - 모든 등록을 CSV로 **내보내기**

:::tip
수용 인원 진행률 표시줄을 사용하여 이벤트가 얼마나 빨리 채워지는지 모니터링하세요. 이벤트가 수용 인원에 도달하거나 초과하면 표시줄이 빨간색으로 변합니다.
:::
