# 캘린더 만들기

<div class="article-intro">

B1 Admin에서 캘린더를 만들면 사역부, 부서 또는 교회 전체의 이벤트를 관리할 수 있는 중앙 공간이 마련됩니다. 각 캘린더는 하나 이상의 그룹에서 이벤트를 가져올 수 있어 모든 일정을 체계적으로 관리할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 캘린더에 포함하려는 이벤트가 있는 [그룹](../groups/creating-groups.md)을 설정하십시오
- B1 Admin의 캘린더 섹션에 대한 관리자 권한이 필요합니다

</div>

## 새 캘린더 만들기

1. B1 Admin에서 **캘린더** 섹션으로 이동합니다.
2. **캘린더 추가**를 클릭합니다.
3. 캘린더의 **이름**을 입력합니다 (예: "청소년 사역 이벤트" 또는 "교회 메인 캘린더").
4. 팀이 이 캘린더의 용도를 이해할 수 있도록 선택적으로 **설명**을 추가합니다.
5. **만들기**를 클릭하여 새 캘린더를 저장합니다.

## 캘린더 상세 페이지

캘린더를 만든 후 클릭하면 상세 페이지가 열립니다. 이 페이지에는 두 가지 주요 영역이 있습니다:

- **왼쪽 열** -- 캘린더 및 이벤트 편집기입니다. 여기에서 캘린더의 이벤트를 보고, 추가하고, 관리할 수 있습니다.
- **오른쪽 열** -- 연결된 그룹 목록입니다. 이 캘린더에 이벤트가 포함된 그룹이 표시됩니다.

## 이벤트 추가

1. 캘린더 상세 페이지에서 왼쪽의 캘린더 편집기를 사용하여 새 이벤트를 추가합니다.
2. 이벤트 이름, 날짜, 시간 및 기타 세부 정보를 설정합니다.
3. 추가한 이벤트는 캘린더에 표시되며 웹사이트에 게시할 수 있습니다.

## 그룹 연결

캘린더에 이벤트가 있는 그룹은 상세 페이지 오른쪽의 그룹 목록에 자동으로 나타납니다.

1. 그룹 섹션에서 **추가**를 클릭하여 캘린더에 그룹을 연결합니다.
2. 드롭다운에서 그룹을 선택합니다.
3. 해당 그룹의 **모든 이벤트**를 포함할지 **특정 이벤트**만 포함할지 선택합니다.
4. **저장**을 클릭합니다.

:::tip
그룹을 캘린더에 연결하면 이벤트를 자동으로 통합할 수 있는 강력한 방법입니다. 그룹 리더가 자신의 [그룹](../groups/creating-groups.md)에 이벤트를 추가하면, 별도의 작업 없이 교회 전체 캘린더에 자동으로 반영됩니다.
:::

:::info
여러 그룹의 이벤트를 하나의 캘린더로 통합하려면, [큐레이션 캘린더](curated-calendar)에서 간소화된 방법을 확인하십시오.
:::
