# 캘린더 만들기

<div class="article-intro">

B1 Admin에서 캘린더를 만들면 사역, 부서 또는 전체 교회의 이벤트를 관리할 수 있는 중앙 위치가 생깁니다. 각 캘린더는 하나 이상의 그룹에서 이벤트를 끌어올 수 있어 모든 것을 쉽게 정리된 상태로 유지할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 캘린더에 포함시킬 [그룹](../groups/creating-groups.md) 설정
- B1 Admin의 캘린더 섹션에 대한 관리 액세스 필요

</div>

## 새 캘린더 만들기

1. B1 Admin의 **캘린더** 섹션으로 이동합니다.
2. **캘린더 추가**를 클릭합니다.
3. 캘린더의 **이름**을 입력합니다(예: "청년 사역 이벤트" 또는 "메인 교회 캘린더").
4. 팀이 이 캘린더가 무엇을 위한 것인지 이해하도록 도와주는 선택적 **설명**을 추가합니다.
5. **생성**을 클릭하여 새 캘린더를 저장합니다.

## 캘린더 세부 페이지

캘린더를 만든 후 클릭하여 세부 페이지를 엽니다. 이 페이지는 두 개의 주요 영역이 있습니다:

- **왼쪽 열** -- 캘린더 및 이벤트 편집기입니다. 여기서 캘린더의 이벤트를 보고, 추가하고, 관리할 수 있습니다.
- **오른쪽 열** -- 연결된 그룹 목록입니다. 이는 이 캘린더에 이벤트가 포함된 그룹을 보여줍니다.

## 이벤트 추가

1. 캘린더 세부 페이지의 왼쪽에 있는 캘린더 편집기를 사용하여 새 이벤트를 추가합니다.
2. 이벤트 이름, 날짜, 시간 및 기타 세부 정보를 설정합니다.
3. 추가하는 이벤트는 캘린더에 나타나며 웹사이트에 게시할 수 있습니다.

## 그룹 연결

캘린더에 이벤트가 있는 그룹은 세부 페이지의 오른쪽 그룹 목록에 자동으로 나타납니다.

1. 그룹 섹션에서 **추가**를 클릭하여 캘린더와 그룹을 연결합니다.
2. 드롭다운에서 그룹을 선택합니다.
3. 해당 그룹의 **모든 이벤트**를 포함할지 아니면 **특정 이벤트**만 포함할지 선택합니다.
4. **저장**을 클릭합니다.

:::tip
캘린더에 그룹을 연결하는 것은 이벤트를 자동으로 집계하는 강력한 방법입니다. 그룹 지도자가 [그룹](../groups/creating-groups.md)에 이벤트를 추가하면 여러분의 추가 작업 없이 교회 전체 캘린더에 포함될 수 있습니다.
:::

:::info
많은 그룹의 이벤트를 끌어오는 단일 캘린더를 만들려면 [선별된 캘린더](curated-calendar)를 참조하여 간결한 접근 방식을 확인하세요.
:::

## 이벤트 등록 활성화

캘린더 이벤트에 대한 등록을 활성화하여 회원들이 B1 웹사이트 또는 모바일 앱을 통해 가입할 수 있습니다.

1. 기존 이벤트를 클릭하거나 새 이벤트를 만듭니다.
2. 이벤트 편집기에서 **등록**을 토글하여 활성화합니다.
3. 등록 설정 구성:
   - **용량**(선택 사항) -- 최대 등록 수를 설정합니다. 무제한은 비워둡니다.
   - **등록 시작** -- 등록이 가능해지는 날짜 및 시간입니다.
   - **등록 종료** -- 등록이 종료되는 날짜 및 시간입니다.
   - **태그** -- 쉼표로 구분된 라벨(예: "청년, 수련회, 성경학교")로 등록 가능한 이벤트를 분류합니다.
4. 이벤트를 저장합니다.

등록이 활성화되면 회원들은 [B1 웹사이트](../../b1-church/events/registering) 또는 [B1 모바일 앱](../../b1-mobile/events/registering)에서 이벤트를 볼 때 **이 이벤트에 등록** 버튼을 볼 수 있습니다.

### 등록 관리

이벤트에 대한 등록을 보고 관리하려면:

1. B1 Admin의 **등록** 페이지로 이동합니다.
2. 등록이 활성화된 모든 이벤트의 표를 볼 수 있으며, 이벤트 제목, 날짜, 현재 등록 수 및 용량, 태그를 표시합니다.
3. 이벤트를 클릭하여 이름, 회원 수, 상태 및 등록 날짜를 포함한 전체 등록 목록을 확인합니다.
4. 세부 페이지에서 다음을 수행할 수 있습니다:
   - 개별 등록을 **취소**합니다
   - 등록을 영구적으로 **삭제**합니다
   - 모든 등록을 CSV로 **내보냅니다**

:::tip
용량 진행률 표시줄을 사용하여 이벤트가 얼마나 빨리 채워지고 있는지 모니터링합니다. 이벤트가 용량에 도달하거나 초과할 때 표시줄이 빨간색으로 변합니다.
:::
