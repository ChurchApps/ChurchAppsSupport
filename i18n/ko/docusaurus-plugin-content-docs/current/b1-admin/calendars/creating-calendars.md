# 일정 만들기

<div class="article-intro">

B1 Admin에서 일정을 만들면 하나 이상의 그룹을 연결하여 이벤트의 선별된 보기를 만들 수 있습니다. 이벤트는 그룹 리더에 의해 각 그룹 내에서 관리되며, 일정에 해당 이벤트가 한 장소에 표시됩니다. 도메인 관리자도 이벤트가 속한 그룹의 리더가 아니면 일정 섹션에서 직접 이벤트를 추가하거나 편집할 수 없습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 일정에 포함하려는 이벤트의 [그룹](../groups/creating-groups.md)을 설정합니다
- B1 Admin의 Calendars 섹션에 대한 관리 액세스 권한이 필요합니다

</div>

## 새 일정 만들기

1. B1 Admin에서 **Website**로 이동한 후 **Calendars** 섹션으로 이동합니다.
2. **Add Calendar**를 클릭합니다.
3. 일정의 **이름**을 입력합니다(예: "Youth Ministry Events" 또는 "Main Church Calendar").
4. 팀이 이 일정의 용도를 이해하는 데 도움이 되도록 선택사항인 **설명**을 추가합니다.
5. **Create**를 클릭하여 새 일정을 저장합니다.

## 일정 세부 페이지

일정을 만든 후 클릭하여 세부 페이지를 엽니다. 이 페이지에는 두 가지 주요 영역이 있습니다:

- **왼쪽 열** -- 연결된 그룹에서 가져온 이벤트를 보여주는 일정 보기.
- **오른쪽 열** -- 연결된 그룹 목록. 이 일정에 포함될 그룹을 관리하는 곳입니다.

## 그룹 연결

일정에 이벤트가 있는 그룹은 세부 페이지의 오른쪽 그룹 목록에 자동으로 나타납니다.

1. 그룹 섹션에서 **Add**를 클릭하여 그룹을 일정과 연결합니다.
2. 드롭다운에서 그룹을 선택합니다.
3. 해당 그룹의 **모든 이벤트** 또는 **특정 이벤트**만 포함할지 선택합니다.
4. **Save**를 클릭합니다.

:::tip
일정에 그룹을 연결하는 것은 이벤트를 자동으로 집계하는 강력한 방법입니다. 그룹 리더가 [그룹](../groups/creating-groups.md)에 이벤트를 추가하면 추가 작업 없이 교회 전체 일정에 반영될 수 있습니다.
:::

:::info
교회의 여러 그룹에서 이벤트를 가져오는 단일 일정을 만들려면 간편한 방법은 [Curated Calendar](curated-calendar)를 참조하세요.
:::

## 이벤트 등록 활성화

모든 일정 이벤트에 대해 등록을 활성화할 수 있으므로 멤버가 B1 웹사이트 또는 모바일 앱을 통해 등록할 수 있습니다.

1. 기존 이벤트를 클릭하거나 새 이벤트를 만듭니다.
2. 이벤트 편집기에서 **Registration**을 전환하여 활성화합니다.
3. 등록 설정을 구성합니다:
   - **용량** (선택 사항) -- 최대 등록 수를 설정합니다. 무제한으로 두려면 공백으로 두세요.
   - **등록 시작** -- 등록이 가능해지는 날짜와 시간.
   - **등록 종료** -- 등록이 종료되는 날짜와 시간.
   - **태그** -- 등록 가능한 이벤트를 분류하는 데 도움이 되는 쉼표로 구분된 레이블(예: "youth, retreat, vbs").
   - **등록 질문** -- 선택적으로 [양식](../forms/creating-forms.md)을 첨부하여 등록자가 가입의 일부로 추가 질문(식이 제한, T셔츠 크기, 비상 연락처 등)에 답변하게 합니다. 질문을 건너뛰려면 **None**을 선택하세요.
   - **대기 목록 활성화** -- 이벤트가 가득 차면 추가 등록자가 돌려보내지는 대신 대기 목록에 참여하도록 합니다. [Paid Registrations](paid-registrations#waitlist)를 참조하세요.
4. 이벤트를 저장합니다.

유료 이벤트의 경우 동일한 설정 페이지에서 가격이 책정된 **참석자 유형**, 선택사항인 **선택** (추가 항목), **할인 코드**를 정의하고 교회의 기부 제공자를 통해 결제를 수집합니다. 전체 설명서는 [Paid Registrations](paid-registrations)을 참조하세요.

등록이 활성화되면 멤버는 [B1 웹사이트](../../b1-church/events/registering) 또는 [B1 모바일 앱](../../b1-mobile/events/registering)에서 이벤트를 볼 때 **Register for this Event** 버튼이 표시됩니다. 양식을 첨부한 경우 등록자는 등록 중에 **Questions** 단계를 보고 답변이 등록과 함께 저장됩니다.

:::info
등록 질문은 **제한됨으로 표시되지 않은** 양식에서만 작동합니다. 제한된 양식은 표시되지 않고 등록 중에 자동으로 건너뜁니다. 따라서 이벤트에 질문을 첨부할 때는 제한되지 않은 양식을 사용하세요.
:::

### 등록 관리

이벤트에 대한 등록을 보고 관리하려면:

1. B1 Admin에서 **Registrations** 페이지로 이동합니다.
2. 이벤트 제목, 날짜, 현재 등록 수 대 용량, 태그를 보여주는 등록이 활성화된 모든 이벤트의 표를 봅니다.
3. 이벤트를 클릭하여 이름, 멤버 수, 참석자 유형, 결제 상태 및 등록 날짜를 포함한 전체 등록 목록을 봅니다.
4. 세부 페이지에서 다음을 수행할 수 있습니다:
   - **참석자 추가** -- 오프라인 또는 전화로 등록한 사람을 수동으로 등록합니다.
   - 개별 등록 **취소**
   - 등록을 영구적으로 **삭제**
   - 자리가 나면 대기 목록 등록 **승격**
   - **CSV로 내보내기** -- 참석자 유형, 선택 사항, 결제 금액 및 질문 답변을 포함한 모든 등록을 다운로드합니다.

이벤트에 등록 질문이 첨부된 경우 세부 페이지는 아직 답변을 제출하지 않은 등록자를 빠르게 찾기 위한 **Unanswered questions only** 필터와 각 답변된 등록에서 응답을 보기 위한 **View Answers** 버튼도 표시합니다. 유료 이벤트는 **Type** 열, **Paid / Total** 열, 유형별 수, 결제 세부 정보 대화 상자를 추가합니다. [Paid Registrations](paid-registrations#the-registration-roster)을 참조하세요.

:::tip
용량 진행 표시줄을 사용하여 이벤트가 얼마나 빨리 채워지는지 모니터링하세요. 이벤트가 용량에 도달하거나 초과하면 표시줄이 빨간색으로 변합니다.
:::
