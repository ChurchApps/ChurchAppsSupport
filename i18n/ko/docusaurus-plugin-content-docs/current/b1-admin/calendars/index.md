# 캘린더

<div class="article-intro">

B1 Admin의 캘린더 섹션에서는 교회의 이벤트 캘린더를 생성하고 관리할 수 있습니다. 다양한 사역을 위한 캘린더를 설정하고, 여러 그룹의 이벤트를 통합하며, B1.church 웹사이트에 캘린더를 게시하여 교인들이 확인할 수 있도록 할 수 있습니다.

</div>

## 캘린더 보기

1. B1 Admin에서 **캘린더** 섹션으로 이동합니다.
2. 캘린더 페이지에는 모든 캘린더의 이름과 상태가 표시됩니다.
3. 각 캘린더에는 연결된 그룹도 함께 표시됩니다.

## 할 수 있는 작업

- **새 캘린더 추가** -- 추가 버튼을 클릭하여 특정 사역, 이벤트 유형 또는 목적에 맞는 캘린더를 생성합니다. 단계별 안내는 [캘린더 만들기](creating-calendars)를 참조하십시오.
- **캘린더 편집** -- 캘린더를 클릭하여 열고 이벤트와 연결된 그룹을 관리합니다.
- **캘린더 게시** -- 캘린더를 B1.church 웹사이트에 표시하여 방문자들이 교회의 예정된 이벤트를 확인할 수 있습니다.
- **큐레이션 캘린더 만들기** -- 여러 그룹의 이벤트를 하나의 캘린더 보기로 통합하여 교회에서 진행되는 모든 활동을 종합적으로 확인할 수 있습니다. 자세한 내용은 [큐레이션 캘린더](curated-calendar)를 참조하십시오.

:::info
캘린더는 [그룹](../groups/creating-groups.md)과 밀접하게 연동됩니다. 그룹에 일정이 예약되어 있으면 해당 이벤트를 캘린더에 포함할 수 있습니다. 그룹을 캘린더에 연결하는 방법은 [캘린더 만들기](creating-calendars) 페이지를 참조하십시오.
:::
