# 캘린더

<div class="article-intro">

B1 Admin의 캘린더 섹션을 사용하면 그룹의 이벤트를 모아 교회를 위한 선별된 캘린더를 만들 수 있습니다. 그룹 리더는 자신의 그룹 이벤트를 관리하고, 관리자는 해당 이벤트를 B1.church 웹사이트에 표시하여 회중이 볼 수 있도록 캘린더를 만들 수 있습니다.

</div>

## 캘린더 보기

1. B1 Admin에서 **웹사이트**로 이동한 다음 **캘린더** 섹션으로 이동합니다.
2. 캘린더 페이지에는 이름과 상태와 함께 모든 캘린더가 나열됩니다.
3. 각 캘린더는 연결된 그룹도 표시합니다.

## 할 수 있는 일

- **새 캘린더 추가** -- 추가 버튼을 클릭하여 특정 사역, 이벤트 유형 또는 목적을 위한 캘린더를 만듭니다. 단계별 지침은 [캘린더 생성하기](creating-calendars)를 참조하세요.
- **캘린더 편집** -- 캘린더를 클릭하여 열고 포함된 그룹을 관리합니다.
- **캘린더 게시** -- 캘린더를 B1.church 웹사이트에 표시하여 방문자가 교회의 예정된 이벤트를 볼 수 있습니다.
- **선별된 캘린더 만들기** -- 여러 그룹의 이벤트를 단일 캘린더 보기로 가져와 교회에서 일어나는 모든 일을 종합적으로 볼 수 있습니다.

:::info
캘린더는 [그룹](../groups/creating-groups.md)과 긴밀하게 작동합니다. 그룹에 이벤트가 예정되어 있으면 해당 이벤트를 캘린더로 가져올 수 있습니다.
:::
