# 큐레이션 캘린더

<div class="article-intro">

큐레이션 캘린더는 여러 그룹의 이벤트를 하나의 통합된 캘린더 보기로 모아줍니다. 이는 모든 사역과 그룹에서 진행되는 모든 활동을 한 곳에서 보여주는 교회 전체 캘린더를 만드는 데 이상적입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 포함하려는 이벤트가 있는 [그룹](../groups/creating-groups.md)을 설정하십시오
- 그룹 리더들이 각 그룹에 이벤트를 추가했는지 확인하십시오
- [캘린더 만들기](creating-calendars)의 기본 사항을 숙지하십시오

</div>

## 큐레이션 캘린더 만들기

1. B1 Admin에서 **웹사이트** 섹션으로 이동합니다.
2. 상단 내비게이션에서 **캘린더**를 클릭합니다.
3. **캘린더 추가**를 클릭하여 새 큐레이션 캘린더를 만듭니다.
4. 캘린더에 **이름**을 지정합니다 (예: "교회 이벤트" 또는 "이번 주 교회 일정").
5. **만들기**를 클릭하여 저장합니다.

## 캘린더에 그룹 추가

1. 새 캘린더를 클릭하여 엽니다.
2. 오른쪽의 **캘린더 내 그룹** 섹션에서 **추가**를 클릭합니다.
3. 드롭다운 메뉴에서 그룹을 선택합니다.
4. 해당 그룹의 **모든 이벤트**를 추가할지 **특정 이벤트**만 추가할지 선택합니다.
5. **저장**을 클릭합니다.
6. 포함하려는 그룹을 모두 추가할 때까지 이 과정을 반복합니다.

:::tip
활동 중인 모든 사역 그룹을 추가하여 교인들에게 매주 교회에서 진행되는 모든 활동의 전체 모습을 보여주는 종합 캘린더를 만드십시오.
:::

## 캘린더 공유

큐레이션 캘린더가 설정되면 다양한 방법으로 공유할 수 있습니다:

- **웹사이트에 삽입** -- 캘린더를 B1.church 웹사이트 페이지에 직접 표시하여 방문자들이 탐색할 수 있습니다.
- **ICS로 구독** -- **구독**을 클릭하여 ICS 링크를 복사합니다. 교인들은 이 링크를 Google 캘린더, Apple 캘린더, Outlook 또는 기타 캘린더 앱에 붙여넣어 동기화할 수 있습니다.
- **ICS 파일 다운로드** -- 캘린더 파일을 다운로드하여 캘린더 애플리케이션에 가져올 수 있습니다.

:::info
그룹 리더가 자신의 [그룹](../groups/creating-groups.md)에 새 이벤트를 추가하면, 해당 이벤트가 큐레이션 캘린더에 자동으로 나타납니다. 새 이벤트가 추가될 때마다 큐레이션 캘린더를 수동으로 업데이트할 필요가 없습니다.
:::
