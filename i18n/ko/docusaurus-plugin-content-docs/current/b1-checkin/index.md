# B1 Church Checkin 개요

<div class="article-intro">

B1 Church Checkin은 Android 태블릿용 자체 서비스 키오스크 앱입니다. 교회 회원들이 가족을 찾고, 아이들을 그룹이나 클래스에 할당하고, 이름표와 부모 픽업 슬립을 인쇄할 수 있습니다 -- 자원봉사자가 책상에 있을 필요가 없습니다.

</div>

## 작동 원리

로비나 환영 공간에 Android 태블릿을 배치하세요. 회원들이 온 후 전화번호나 성으로 가구를 찾고, 가족을 검토하고, 각 사람이 참석할 그룹을 선택하고, **체크인**을 탭합니다. 앱이 이름표와 픽업 슬립을 자동으로 인쇄한 후 다음 가족을 위해 재설정합니다.

## 주요 기능

- **자체 서비스 회원 조회** -- 전화번호 또는 성으로 검색하여 가구를 빠르게 찾습니다.
- **가구 검토** -- 한눈에 사진과 현재 그룹 할당이 있는 모든 가족 구성원을 봅니다.
- **그룹 할당** -- 각 가족 구성원을 선택한 예배 시간에 대해 올바른 클래스나 그룹에 할당합니다.
- **손님 체크인** -- 한 명과 성을 포함한 방문객을 그 자리에서 가구에 추가합니다.
- **QR 손님 등록** -- 방문객이 스캔할 수 있는 QR 코드를 표시하여 체크인 전에 [자신을 등록](../b1-church/checkin/guest-registration)합니다.
- **라벨 인쇄** -- 아이들을 위한 이름표와 고유한 보안 코드가 있는 부모 픽업 슬립을 인쇄합니다.
- **네트워크 프린터 지원** -- 앱에서 직접 네트워크 프린터를 발견하고 구성합니다.
- **자동 로그인** -- 첫 로그인 후 앱이 자격증명을 기억하고 향후 실행 시 예배 화면으로 바로 이동합니다.
- **다중 교회 지원** -- 계정이 여러 교회에 연결된 경우 체크인할 교회를 선택할 수 있습니다.
- **교회 브랜딩** -- 앱이 교회를 선택한 후 헤더에 교회 로고를 표시합니다.

## 설명서 섹션

- **[시작하기](./getting-started/)** -- 앱 설치, 로그인, 프린터 설정.

:::
