# B1 Church Checkin 개요

<div class="article-intro">

B1 Church Checkin은 Android 태블릿용 셀프 서비스 키오스크 앱입니다. 교회 구성원이 자신의 가족을 조회하고, 자녀를 그룹이나 반에 배정하며, 이름 라벨과 부모 인계 슬립을 출력할 수 있습니다. 데스크에 봉사자가 없어도 모든 과정을 직접 처리할 수 있습니다.

</div>

## 작동 방식

로비 또는 환영 공간에 Android 태블릿을 설치합니다. 구성원이 키오스크 앞에 다가와 전화번호나 성(姓)으로 가정을 검색하고, 가족 구성원을 확인한 후 각 그룹에 참석할 사람을 선택하고 **체크인** 버튼을 탭합니다. 앱이 자동으로 이름 라벨과 인계 슬립을 출력한 뒤 다음 가족을 위해 초기 상태로 돌아갑니다.

## 주요 기능

- **셀프 서비스 구성원 조회** -- 전화번호 또는 성으로 검색하여 가정을 빠르게 찾을 수 있습니다.
- **가정 검토** -- 사진과 현재 그룹 배정 정보를 포함한 모든 가족 구성원을 한눈에 확인할 수 있습니다.
- **그룹 배정** -- 선택한 예배 시간에 맞게 각 가족 구성원을 적절한 반 또는 그룹에 배정합니다.
- **방문자 체크인** -- 이름과 성만 입력하면 방문자를 가정에 즉시 추가할 수 있습니다.
- **라벨 출력** -- 자녀용 이름 라벨과 고유 보안 코드가 포함된 부모 인계 슬립을 출력합니다.
- **네트워크 프린터 지원** -- 앱에서 직접 네트워크 프린터를 검색하고 설정할 수 있습니다.
- **자동 로그인** -- 최초 로그인 후 앱이 자격 증명을 기억하여 이후 실행 시 바로 예배 선택 화면으로 이동합니다.
- **다중 교회 지원** -- 계정이 여러 교회에 연결된 경우 체크인할 교회를 선택할 수 있습니다.
- **교회 브랜딩** -- 교회가 선택되면 앱 헤더에 해당 교회 로고가 표시됩니다.

## 문서 구성

- **[시작하기](./getting-started/)** -- 앱 설치, 로그인, 프린터 설정 방법을 안내합니다.
- **[체크인 워크플로](./check-in/)** -- 예배 선택부터 라벨 출력까지 전체 체크인 과정을 설명합니다.

:::tip
B1 Church Checkin은 가족이 쉽게 접근할 수 있도록 가로 방향으로 거치된 10인치 Android 태블릿에서 가장 잘 작동합니다.
:::

:::info
B1 Church Checkin을 사용하기 전에 교회 관리자가 B1 Admin에서 예배, 예배 시간 및 그룹을 설정해야 합니다. 자세한 내용은 [출석 설정 가이드](../b1-admin/attendance/setup.md)를 참조하십시오.
:::

문의 사항은 [support@churchapps.org](mailto:support@churchapps.org)로 이메일을 보내 주십시오.
