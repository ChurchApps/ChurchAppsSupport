# B1 Church Checkin 시작하기

<div class="article-intro">

B1 Church Checkin 설정은 몇 분이면 완료됩니다. 이 섹션에서는 Android 태블릿에 앱을 설치하고, ChurchApps 계정으로 로그인하며, 필요에 따라 네트워크 라벨 프린터를 설정하는 방법을 안내합니다.

</div>

## 준비 사항

- **Android 태블릿** -- 키오스크 환경에는 10인치 태블릿이 가장 적합합니다.
- **ChurchApps 계정** -- B1 Admin 또는 다른 ChurchApps 제품에서 사용하는 이메일과 비밀번호입니다.
- **네트워크 라벨 프린터** (선택 사항) -- 이름 라벨과 부모 인계 슬립을 출력하는 데 사용합니다. 앱이 로컬 네트워크를 스캔하여 호환 가능한 프린터를 찾을 수 있습니다.

## 설치

1. Android 태블릿에서 **Google Play Store**를 엽니다.
2. **B1 Church Checkin**을 검색합니다.
3. **설치**를 탭하고 다운로드가 완료될 때까지 기다립니다.
4. 앱을 엽니다. 초기화되는 동안 스플래시 화면이 표시됩니다.

## 최초 설정

앱을 처음 실행하면 세 단계를 거치게 됩니다.

1. **[로그인](./logging-in)** -- 이메일과 비밀번호를 입력하여 인증합니다.
2. **[교회 선택](./logging-in)** -- 계정이 여러 교회에 연결된 경우 이 키오스크가 서비스할 교회를 선택합니다.
3. **[프린터 설정](./printer-setup)** -- 라벨 출력을 위한 네트워크 프린터를 설정하거나, 라벨이 필요하지 않은 경우 이 단계를 건너뜁니다.

최초 로그인 후 앱이 자격 증명과 교회 선택 정보를 저장합니다. 이후 실행 시에는 자동으로 예배 선택 화면으로 이동합니다.

:::info
앱은 시작할 때마다 무선 업데이트를 확인합니다. 업데이트가 있으면 로그인 화면에 도달하기 전에 자동으로 다운로드하여 적용됩니다.
:::

:::tip
키오스크를 설정하기 전에 교회 관리자가 B1 Admin에서 예배, 예배 시간 및 그룹을 구성했는지 확인하십시오. 자세한 내용은 [출석 설정 가이드](../../b1-admin/attendance/setup.md)를 참조하십시오.
:::
