# 로그인

<div class="article-intro">

키오스크를 체크인에 사용하기 전에 교회에 접근 권한이 있는 ChurchApps 계정으로 로그인해야 합니다. 이 페이지에서는 최초 로그인 과정, 교회 선택, 자동 로그인 동작, 그리고 로그아웃 방법을 설명합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- Android 태블릿에 B1 Church Checkin을 설치합니다 ([시작하기](./index.md) 참조).
- ChurchApps 계정 이메일과 비밀번호를 준비합니다.

</div>

## 최초 로그인

1. B1 Church Checkin 앱을 엽니다. 스플래시 화면이 지나면 B1 Church Checkin 로고가 있는 로그인 화면이 표시됩니다.
2. 첫 번째 필드에 **이메일 주소**를 입력합니다.
3. 두 번째 필드에 **비밀번호**를 입력합니다.
4. **로그인** 버튼을 탭합니다.

앱이 자격 증명을 확인하고 로그인을 처리합니다. 이메일이나 비밀번호가 올바르지 않으면 화면 하단에 오류 메시지가 표시됩니다.

:::info
이 이메일과 비밀번호는 B1 Admin 또는 다른 ChurchApps 제품에서 사용하는 것과 동일합니다. 체크인 키오스크를 위한 별도 계정은 필요하지 않습니다.
:::

## 교회 선택

로그인에 성공하면 앱이 계정에 연결된 교회 목록을 불러옵니다. 각 교회는 교회 이름과 로고(B1 Admin에서 설정된 경우)가 표시된 카드로 나타납니다.

- 계정이 **하나의 교회**에만 연결된 경우에도 선택 화면이 표시됩니다. 해당 교회를 탭하여 계속 진행합니다.
- 계정이 **여러 교회**에 연결된 경우 이 키오스크가 서비스할 교회를 탭합니다.

교회를 선택하면 앱이 해당 교회의 브랜딩 및 외관 설정을 불러옵니다. 이후 모든 화면의 헤더에 교회 로고가 표시됩니다.

## 재방문 시 자동 로그인

앱은 최초 로그인 성공 후 이메일, 비밀번호, 선택한 교회 정보를 저장합니다. 이후 실행 시에는:

- 저장된 자격 증명으로 자동 로그인됩니다.
- 이전에 교회를 선택한 경우 해당 선택이 복원되어 바로 **예배 선택 화면**으로 이동합니다.
- 저장된 자격 증명이 더 이상 유효하지 않으면 다시 로그인할 수 있도록 로그인 화면으로 돌아갑니다.

:::tip
키오스크는 자동 로그인되기 때문에 초기 설정 시에만 자격 증명을 한 번 입력하면 됩니다. 이후에는 태블릿 전원을 켜기만 하면 바로 사용 준비가 완료됩니다.
:::

## 로그아웃

교회 구성원이 키오스크 사용 중 실수로 로그아웃하지 않도록 로그아웃 기능은 의도적으로 숨겨져 있습니다. 로그아웃하려면:

1. 헤더의 **교회 로고**를 2초 이내에 **일곱 번** 빠르게 탭합니다.
2. 로그아웃 여부를 묻는 확인 대화상자가 표시됩니다.
3. **로그아웃**을 탭하여 확인하거나, **취소**를 탭하여 로그인 상태를 유지합니다.

로그아웃하면 저장된 자격 증명이 모두 삭제되고 로그인 화면으로 돌아갑니다.

:::warning
로그아웃하면 저장된 자격 증명과 교회 선택 정보가 모두 삭제됩니다. 키오스크를 다시 사용하려면 이메일과 비밀번호를 다시 입력해야 합니다. 계정을 전환하거나 키오스크를 재설정해야 할 때만 로그아웃하십시오.
:::
