# 로그인

<div class="article-intro">

키오스크를 체크인에 사용하기 전에 교회에 액세스할 수 있는 ChurchApps 계정으로 로그인해야 합니다. 이 페이지는 첫 로그인 프로세스, 교회 선택, 자동 로그인 동작, 로그아웃 방법을 다룹니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- Android 태블릿에 B1 Church Checkin 설치([시작하기](./index.md) 참조)
- ChurchApps 계정 이메일 및 암호 준비

</div>

## 첫 로그인

1. B1 Church Checkin 앱을 엽니다. 스플래시 화면 후 B1 Church Checkin 로고가 있는 로그인 화면을 볼 수 있습니다.
2. 첫 번째 필드에 **이메일 주소**를 입력합니다.
3. 두 번째 필드에 **암호**를 입력합니다.
4. **로그인** 버튼을 탭합니다.

앱이 자격증명을 검증하고 로그인합니다. 이메일 또는 암호가 잘못된 경우 화면 하단에 오류 메시지가 표시됩니다.

:::info
이것은 B1 Admin 또는 다른 ChurchApps 제품에 사용하는 것과 동일한 이메일 및 암호입니다. 체크인 키오스크에는 별도의 계정이 필요하지 않습니다.
:::

## 교회 선택

성공적인 로그인 후 앱은 계정에 연결된 교회 목록을 로드합니다. 각 교회는 교회 이름과 로고(B1 Admin에서 구성된 경우)를 표시하는 카드로 표시됩니다.

- 계정이 **단 하나의 교회**에만 연결된 경우에도 선택 화면을 볼 수 있습니다. 계속하려면 교회를 탭합니다.
- 계정이 **여러 교회**에 연결된 경우 이 키오스크를 제공할 교회를 탭합니다.

교회를 선택하면 앱이 그 교회의 브랜드 및 모양 설정을 로드합니다. 교회 로고가 이후의 모든 화면 헤더에 나타납니다.

## 반복 방문 시 자동 로그인

앱이 첫 번째 성공적인 로그인 후 이메일, 암호, 선택한 교회를 저장합니다. 향후 실행:

- 앱이 저장된 자격증명으로 자동으로 로그인합니다.
- 이전에 교회를 선택한 경우 해당 선택을 복원하고 **예배 화면**으로 직접 이동합니다.
- 저장된 자격증명이 더 이상 유효하지 않으면 앱이 로그인 화면으로 돌아가므로 다시 로그인할 수 있습니다.

:::tip
키오스크가 자동으로 로그인하기 때문에 초기 설정 중에 한 번만 자격증명을 입력하면 됩니다. 그 후에는 태블릿을 켜면 준비가 됩니다.
:::

## 관리자 설정

관리자 설정은 의도적으로 숨겨져 있어 키오스크를 사용하는 교회 회원이 실수로 구성을 변경할 수 없습니다. 액세스하려면:

1. **2초 이내에** 헤더의 **교회 로고**를 **7번** 빠르게 탭합니다.
2. 키오스크 PIN이 설정된 경우 메시지가 표시되면 PIN을 입력합니다.
3. **관리자 설정** 화면이 다음 옵션으로 열립니다:
   - **예배 변경** — 선택한 예배 시간 전환
   - **프린터 변경** — [프린터 설정](./printer-setup)을 열어 프린터를 스캔하고 선택합니다
   - **PIN 설정/변경** — 관리자 액세스를 보호하는 키오스크 PIN을 설정하거나 업데이트합니다
   - **로그아웃** — 로그아웃하고 로그인 화면으로 돌아갑니다

:::warning
로그아웃하면 저장된 모든 자격증명과 교회 선택이 지워집니다. 키오스크를 다시 사용하려면 이메일과 암호를 다시 입력해야 합니다. 계정을 전환하거나 키오스크를 다시 구성해야 할 때만 로그아웃하세요.
:::
