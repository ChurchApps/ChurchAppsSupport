# 프린터 설정

<div class="article-intro">

B1 Church Checkin은 네트워크 라벨 프린터로 이름 라벨과 부모 인계 슬립을 출력할 수 있습니다. 프린터 설정은 선택 사항으로, 출석 추적만 필요한 경우에는 프린터 없이도 앱을 사용할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Church Checkin 앱에 [로그인](./logging-in)하고 교회를 선택합니다.
- 호환 가능한 네트워크 라벨 프린터를 태블릿과 동일한 Wi-Fi 네트워크에 연결합니다.

</div>

## 프린터 설정 열기

화면 최상단의 **프린터 상태 바**를 탭하면 언제든지 프린터 설정 화면을 열 수 있습니다. 이 얇은 바에는 앱 버전 번호와 현재 프린터 연결 상태가 표시됩니다.

## 프린터 스캔

프린터 설정 화면이 열리면 앱이 자동으로 로컬 네트워크에서 호환 가능한 프린터를 스캔합니다. 스캔이 진행되는 동안 "프린터 검색 중"이라는 메시지와 함께 로딩 표시기가 나타납니다.

스캔이 완료되면 검색된 프린터 목록이 표시됩니다. 각 프린터 카드에는 다음 정보가 표시됩니다.

- **프린터 모델** 이름
- 네트워크상의 프린터 **IP 주소**

목록 하단에는 항상 **프린터 없음** 옵션이 포함됩니다.

:::warning
스캔으로 프린터를 검색하려면 태블릿과 프린터가 동일한 로컬 네트워크에 연결되어 있어야 합니다. 프린터가 표시되지 않으면 두 기기가 동일한 Wi-Fi 네트워크에 있는지, 그리고 프린터의 전원이 켜져 있는지 확인하십시오.
:::

## 프린터 선택

1. 사용할 프린터를 탭합니다. 선택된 프린터 카드에 체크 표시와 함께 강조 표시됩니다.
2. 실제 프린터 없이 사용하려면(테스트 목적 또는 출석 전용 모드) **프린터 없음**을 선택합니다.

## 테스트 인쇄

선택 사항을 저장하기 전에 연결이 올바르게 작동하는지 확인할 수 있습니다.

1. 목록에서 프린터를 선택합니다.
2. 화면 하단의 **테스트 인쇄** 버튼을 탭합니다.
3. 앱이 테스트 라벨을 프린터로 전송합니다. 프린터가 올바르게 연결된 경우 테스트 페이지가 출력됩니다.

**프린터 없음**이 선택된 상태에서 **테스트 인쇄**를 탭하면 프린터가 선택되지 않았다는 알림이 표시됩니다.

:::tip
다음 예배 전에 프린터를 선택한 후 반드시 테스트 인쇄를 실행하여 연결을 확인하십시오.
:::

## 선택 저장

프린터 선택에 만족하면:

1. 화면 하단의 **저장 및 재시작** 버튼을 탭합니다.
2. 앱이 프린터 선택 정보를 저장하고 설정을 적용하기 위해 재시작됩니다.

프린터 선택 정보는 앱을 재시작해도 유지됩니다. 프린터를 변경하려는 경우가 아니라면 다시 설정할 필요가 없습니다.

:::info
키오스크를 다른 위치로 이동하거나 네트워크 설정을 변경한 경우 프린터 설정으로 돌아가서 프린터를 다시 스캔하고 선택 정보를 업데이트하십시오.
:::

## 다음 단계

프린터 설정이 완료되면 체크인을 시작할 준비가 된 것입니다. 전체 프로세스가 어떻게 작동하는지 알아보려면 [예배 선택](../check-in/selecting-service)부터 시작하는 [체크인 워크플로](../check-in/)를 참조하십시오.
