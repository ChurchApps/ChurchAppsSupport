# 프린터 설정

<div class="article-intro">

B1 Church Checkin은 네트워크 라벨 프린터에 이름표 및 부모 픽업 슬립을 인쇄할 수 있습니다. 프린터 설정은 선택 사항입니다 -- 출석 추적만을 위해 프린터 없이 앱을 사용할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [B1 Church Checkin 앱에 로그인](./logging-in)하고 교회 선택
- 호환되는 네트워크 라벨 프린터를 태블릿과 동일한 Wi-Fi 네트워크에 연결

</div>

## 프린터 설정 액세스

언제든지 **관리자 설정** 메뉴를 통해 프린터 설정 화면을 열 수 있습니다:

1. **2초 이내에** 헤더의 **교회 로고**를 **7번** 빠르게 탭합니다.
2. 키오스크 PIN이 설정된 경우 메시지가 표시되면 PIN을 입력합니다.
3. 관리자 설정 화면에서 **프린터 변경**을 탭합니다.

이렇게 하면 프린터를 스캔하고, 선택하고, 테스트 인쇄를 실행할 수 있는 프린터 설정 화면이 열립니다.

## 프린터 스캔

프린터 설정 화면이 열리면 앱이 자동으로 로컬 네트워크에서 호환되는 프린터를 스캔합니다. 스캔이 진행 중일 때 "프린터 스캔 중" 메시지가 있는 로딩 표시기가 나타납니다.

스캔이 완료되면 발견된 프린터 목록이 표시됩니다. 각 프린터 카드는 다음을 표시합니다:

- **프린터 모델** 이름
- 네트워크의 프린터 **IP 주소**

**프린터 없음** 옵션은 항상 목록 하단에 포함됩니다.

:::warning
태블릿과 프린터가 동일한 로컬 네트워크에 연결되어야 스캔이 프린터를 발견합니다. 프린터가 나타나지 않으면 두 기기가 동일한 Wi-Fi 네트워크에 있고 프린터가 켜져 있는지 확인합니다.
:::

## 프린터 선택

1. 사용할 프린터를 탭합니다. 선택한 프린터 카드가 확인 표시로 강조됩니다.
2. 물리적 프린터 없이 인쇄하려면(테스트 또는 출석 전용 모드) **프린터 없음**을 선택합니다.

## 테스트 인쇄

선택을 저장하기 전에 연결이 작동하는지 확인할 수 있습니다:

1. 목록에서 프린터를 선택합니다.
2. 화면 하단의 **테스트 인쇄** 버튼을 탭합니다.
3. 앱이 프린터로 테스트 라벨을 보냅니다. 프린터가 제대로 연결되어 있으면 테스트 페이지가 인쇄됩니다.

**프린터 없음**이 선택된 상태에서 **테스트 인쇄**를 탭하면 앱이 프린터가 선택되지 않았음을 알리는 경고를 표시합니다.

:::tip
프린터를 선택한 후 항상 테스트 인쇄를 실행하여 다음 예배 전에 연결을 확인합니다.
:::

## 선택 저장

프린터 선택에 만족하면:

1. 화면 하단의 **저장 및 다시 시작** 버튼을 탭합니다.
2. 앱이 프린터 선택을 저장하고 설정을 적용하기 위해 다시 시작합니다.

프린터 선택은 앱 다시 시작 시 기억됩니다. 프린터를 변경하려는 경우가 아니면 다시 설정할 필요가 없습니다.

:::info
키오스크를 다른 위치로 이동하거나 네트워크 설정을 변경한 경우 프린터 설정으로 돌아가서 프린터를 다시 스캔하고 선택을 업데이트합니다.
:::
