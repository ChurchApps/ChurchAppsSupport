# 체크인 워크플로우

<div class="article-intro">

B1 Church Checkin의 체크인 프로세스는 간단한 단계적 흐름을 따릅니다. 교회 구성원이 키오스크에 다가가 가족을 찾고, 그룹 할당을 확인하고, 체크인합니다. 모두 1분 이내에 완료됩니다. 이 섹션은 키오스크 체크인 경험의 각 단계를 다룹니다.

</div>

## 한눈에 보는 체크인 흐름

1. **[예배 선택](./selecting-service)** -- 참석 중인 예배(예: 일요일 오전, 일요일 오후, 주중)를 선택합니다.
2. **[가족 조회](./looking-up-members)** -- 전화번호 또는 성을 검색하여 시스템에서 자신을 찾습니다.
3. **[가족 검토](./household-review)** -- 모든 가족 구성원과 현재 그룹 할당을 봅니다.
4. **[그룹 할당](./group-assignment)** -- 각 예배 시간에 각 가족 구성원이 참석해야 할 클래스 또는 그룹을 선택합니다.
5. **[손님 추가](./adding-guests)**(선택 사항) -- 가족과 함께 하는 방문객을 가족에 추가합니다.
6. **[체크인 완료](./completing-checkin)** -- 체크인 버튼을 탭하여 출석을 제출하고 라벨을 인쇄합니다.
7. **[체크아웃](./checking-out)**(관리되는 스테이션) -- 픽업 시간에 보안 코드와 픽업 담당자를 확인하고, 신뢰할 수 있는 픽업 목록, 부모 호출 및 라벨 재인쇄를 사용합니다.

체크인이 완료되면 앱은 성공 화면을 표시하고 자동으로 조회 화면으로 돌아가서 다음 가족을 위해 준비됩니다.

:::tip
전체 흐름은 교회 구성원이 독립적으로 사용하도록 설계되었습니다. 자원봉사자가 키오스크를 운영할 필요는 없지만 처음 방문하는 방문객을 도울 수 있는 사람이 근처에 있으면 항상 좋은 아이디어입니다.
:::

## 시작하기 전에

체크인 키오스크를 사용하기 전에 다음이 B1 Admin에서 구성되었는지 확인하세요:

- **예배 및 예배 시간**은 출석 설정에서 설정됩니다. [출석 설정 안내서](../../b1-admin/attendance/setup.md)를 참조하세요.
- **그룹**이 만들어지고 적절한 예배 시간에 연결됩니다. [그룹 만들기](../../b1-admin/groups/creating-groups.md)를 참조하세요.
- **그룹 설정**이 라벨 인쇄를 위해 구성됩니다. 이름 라벨이 필요한 그룹에 대해 **인쇄 이름표**를 활성화하고 자녀 그룹이 픽업 슬립을 필요로 하는 경우 **부모 픽업**을 활성화합니다.
- **사람**이 가족 연결과 함께 시스템에 추가됩니다. [사람 추가](../../b1-admin/people/adding-people.md)를 참조하세요.

:::info
아직 키오스크 자체를 설정하지 않았으면 설치, 로그인 및 프린터 구성 지침은 [시작하기](../getting-started/)를 참조하세요.
:::
