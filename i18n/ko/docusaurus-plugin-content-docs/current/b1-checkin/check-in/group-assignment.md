# 그룹 배정

<div class="article-intro">

가정 검토 화면에서 예배 시간 버튼을 탭하면 그룹 선택 화면이 열립니다. 여기서 가족 구성원이 해당 예배 시간에 참석할 그룹 또는 반을 선택합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 가정 검토 화면에서 [가정을 검토](./household-review)합니다.
- 배정할 구성원을 펼치고 예배 시간 버튼을 탭합니다.
- 교회 관리자가 [그룹을 생성](../../b1-admin/groups/creating-groups.md)하고 B1 Admin에서 예배 시간에 연결했는지 확인합니다.

</div>

## 그룹 구성 방식

그룹은 **카테고리**별로 표시됩니다. 각 카테고리는 다음 정보가 포함된 카드로 나타납니다.

- **카테고리 이름** (예: "아동부", "청소년부", "성인부")
- 해당 카테고리의 **그룹 수**

카테고리 카드를 탭하면 펼쳐지고 내부의 개별 그룹을 확인할 수 있습니다. 다시 탭하면 접힙니다.

## 그룹 선택

1. **카테고리**를 탭하여 펼칩니다.
2. 해당 카테고리 내의 그룹 목록을 확인합니다. 각 그룹은 이름과 오른쪽의 선택 아이콘으로 표시됩니다.
3. 배정할 **그룹 이름**을 탭합니다.

앱이 선택 정보를 저장하고 [가정 검토 화면](./household-review)으로 돌아갑니다. 해당 구성원의 예배 시간 버튼에 선택한 그룹 이름이 표시됩니다.

## "없음" 선택

이 예배 시간에 가족 구성원을 특정 그룹에 배정하지 않으려면 화면 하단의 **없음** 버튼을 탭합니다. 이렇게 하면 해당 예배 시간의 기존 그룹 배정이 취소되고 가정 검토 화면으로 돌아갑니다.

:::tip
특정 그룹이나 반에 체크인할 필요가 없는 주 예배 참석 성인에게는 **없음** 옵션을 사용하십시오.
:::

## 그룹 배정 변경

이미 완료된 그룹 배정을 변경하려면:

1. [가정 검토 화면](./household-review)으로 돌아갑니다.
2. 그룹을 변경하려는 구성원을 펼칩니다.
3. 예배 시간 버튼(현재 배정된 그룹 이름이 표시됨)을 탭합니다.
4. 다른 그룹을 선택하거나 **없음**을 탭합니다.

:::info
그룹과 카테고리는 교회 관리자가 B1 Admin에서 설정합니다. 그룹이 없거나 잘못된 카테고리에 있는 경우 관리자에게 [그룹 설정](../../b1-admin/groups/creating-groups.md)을 업데이트해 달라고 요청하십시오.
:::

## 다음 단계

그룹 배정 후 [가정 검토 화면](./household-review)으로 돌아가 다음 작업을 계속할 수 있습니다.

- 추가 가족 구성원에게 그룹을 배정합니다.
- 가족과 함께 온 방문자가 있으면 [방문자를 추가](./adding-guests)합니다.
- 모든 배정이 준비되면 [체크인을 완료](./completing-checkin)합니다.
