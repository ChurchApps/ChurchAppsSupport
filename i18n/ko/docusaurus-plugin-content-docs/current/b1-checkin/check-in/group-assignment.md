# 그룹 할당

<div class="article-intro">

가족 검토 화면에서 예배 시간 버튼을 탭하면 앱이 그룹 선택 화면을 엽니다. 여기서 가족 구성원이 해당 특정 예배 시간에 참석해야 할 그룹이나 클래스를 선택합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 가족 검토 화면에서 [가족 검토](./household-review)
- 할당하려는 멤버를 확장하고 예배 시간 버튼을 탭합니다
- 교회 관리자가 [그룹을 만들고](../../b1-admin/groups/creating-groups.md) B1 Admin의 예배 시간에 연결했는지 확인합니다

</div>

## 그룹이 구성되는 방법

그룹은 **카테고리**에 표시됩니다. 각 카테고리는 다음을 표시하는 카드로 나타납니다:

- **카테고리 이름**(예: "어린이", "청소년", "성인")
- 해당 카테고리의 **그룹 수**

카테고리 카드를 탭하여 확장하고 내부의 개별 그룹을 봅니다. 다시 탭하여 축소합니다.

## 그룹 선택

1. **카테고리**를 탭하여 확장합니다.
2. 해당 카테고리 내의 그룹 목록을 검토합니다. 각 그룹은 오른쪽에 선택 아이콘이 있는 이름을 표시합니다.
3. 할당하려는 **그룹 이름**을 탭합니다.

앱이 선택을 저장하고 [가족 검토 화면](./household-review)으로 돌아갑니다. 해당 멤버의 예배 시간 버튼은 이제 선택한 그룹의 이름을 표시합니다.

## 연령 및 학년 지침

교회에서 객실에 연령 또는 학년 범위를 설정한 경우 목록이 올바른 것으로 안내합니다. 해당 사람이 적합한 객실은 **강조 표시**되고 연령이나 학년 범위를 벗어난 객실은 **흐리게** 표시됩니다. 흐려진 객실은 여전히 선택할 수 있지만 직원이 선택을 확인해야 합니다. 연령과 학년은 교회의 학년 승격 날짜에서 계산되므로 학년도 롤오버 전체에서 객실은 정확합니다.

## "없음" 선택

가족 구성원이 이 예배 시간에 그룹에 할당되지 않기로 결정한 경우 화면 하단의 **None** 버튼을 탭합니다. 이렇게 하면 해당 예배 시간에 대한 기존 그룹 할당이 지워지고 가족 검토 화면으로 돌아갑니다.

:::tip
주 예배에 참석하고 특정 그룹이나 클래스에 체크인할 필요가 없는 성인의 경우 **None** 옵션을 사용합니다.
:::

## 그룹 할당 변경

이미 작성된 그룹 할당을 변경하려면:

1. [가족 검토 화면](./household-review)으로 돌아갑니다.
2. 그룹을 변경하려는 멤버를 확장합니다.
3. 예배 시간 버튼(현재 할당된 그룹 이름 표시)을 탭합니다.
4. 다른 그룹을 선택하거나 **None**을 탭합니다.

:::info
그룹 및 카테고리는 B1 Admin의 교회 관리자가 구성합니다. 그룹이 누락되었거나 잘못된 카테고리에 있으면 관리자에게 [그룹 설정](../../b1-admin/groups/creating-groups.md)을 업데이트하도록 요청하세요.
:::

## 다음 단계

그룹을 할당한 후 [가족 검토 화면](./household-review)으로 돌아가서:

- 추가 가족 구성원에게 그룹을 할당합니다
- 누군가 가족과 방문하는 경우 [손님을 추가](./adding-guests)합니다
- 모든 할당이 준비되면 [체크인을 완료](./completing-checkin)합니다
