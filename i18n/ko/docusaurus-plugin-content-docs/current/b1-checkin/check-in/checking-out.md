# 체크아웃 및 어린이 안전

<div class="article-intro">

체크아웃은 어린이 체크인의 루프를 마감합니다. 보호자가 픽업 라벨의 보안 코드를 제시하면 키오스크가 픽업 담당자를 확인하고 어린이들이 체크아웃됩니다. 관리자가 있는 스테이션에는 신뢰할 수 있는 픽업 검증, 보호자 호출 문자, 보안 라벨 재인쇄 및 긴급 방송 등의 안전 도구가 추가로 제공됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 체크아웃은 키오스크 관리자 설정에서 **관리자 모드**로 설정된 스테이션에서 사용 가능합니다.
- 어린이는 보안 코드가 포함된 인쇄된 픽업 라벨과 함께 [체크인](./completing-checkin)되어야 합니다.
- 호출 및 긴급 방송 기능을 사용하려면 교회에서 B1 Admin에 문자 전송 서비스를 연결해야 합니다.

</div>

## 체크아웃 시작하기

1. 관리자 모드 스테이션에서 조회 화면의 **체크아웃**을 탭합니다.
2. 가족의 픽업 라벨에서 4자리 **보안 코드**를 입력합니다. 직접 입력하거나, 화면의 키패드를 사용하거나, USB 또는 Bluetooth 스캐너로 라벨의 바코드를 스캔할 수 있습니다. 4자리가 모두 입력되면 코드가 자동으로 제출됩니다.
3. 키오스크가 해당 코드로 체크인된 어린이들을 표시합니다.

## 픽업 담당자 확인

체크아웃 화면에서 어린이들을 픽업할 사람을 물어봅니다.

- **신뢰할 수 있는 픽업 담당자**는 가구에 대해 사진과 관계가 함께 표시된 선택 가능한 카드로 나타납니다. 앞에 서 있는 사람을 탭합니다.
- **가구 내 성인**도 사진 그리드에 나타납니다.
- **기타** 옵션을 사용하면 목록에 없는 사람의 이름을 입력할 수 있습니다.

입력한 이름이 해당 가구에 대해 **권한 없음**으로 표시된 사람과 일치하면 키오스크는 경고와 함께 체크아웃을 차단합니다. 직원은 **무시하고 진행**을 선택하여 계속 진행할 수 있습니다. 무시한 항목은 해당 사람의 이름과 함께 출석 기록에 기록됩니다.

픽업 담당자가 확인되면 체크아웃을 탭합니다. 픽업 담당자의 이름이 출석 기록과 함께 저장됩니다.

:::info
신뢰할 수 있는 픽업 담당자와 권한 없는 픽업 담당자는 B1 Admin에서 각 사람의 페이지에서 교회 직원이 관리합니다. [체크인 안전](../../b1-admin/attendance/checkin-safety#trusted-and-not-authorized-pickup-people)을 참조하세요.
:::

## 보호자 호출

서비스 중에 보호자가 필요한 경우(기저귀 교환, 울고 있는 어린이 등) 관리자가 있는 스테이션의 직원은 **호출**을 보낼 수 있습니다. 이는 교회의 문자 전송 서비스를 통해 어린이의 부모 또는 보호자에게 문자 메시지를 보냅니다. 문자를 거부한 보호자나 휴대폰 번호가 없는 보호자는 건너뜁니다. 키오스크에는 보낸 메시지 수가 표시됩니다.

## 라벨 재인쇄

이름표 또는 픽업 라벨을 잃어버렸거나 손상된 경우 관리자가 있는 스테이션의 직원은 보안 코드를 입력한 후 체크아웃 화면에서 가족의 라벨을 **재인쇄**할 수 있습니다. 재인쇄는 원래 체크인 시 사용한 것과 동일한 프린터 및 라벨 템플릿을 사용합니다.

## 긴급 방송

긴급 상황에서 직원은 현재 서비스에 대해 **모든 체크인된 어린이**의 보호자들에게 한 번에 문자를 보낼 수 있습니다.

1. 키오스크 **관리자 설정**을 엽니다. (헤더 로고를 빠르게 7번 탭하고, PIN이 설정된 경우 PIN을 입력합니다).
2. **긴급 방송**을 탭합니다.
3. 메시지를 입력한 후 확인 필드에 **EMERGENCY**를 입력합니다. **방송 보내기** 버튼은 입력할 때까지 비활성화됩니다.
4. 키오스크는 메시지를 받은 전화 수와 건너뛴 사람 수(거부하거나 휴대폰 번호가 없는 사람)를 보고합니다.

:::warning
방송은 선택된 서비스에 대해 모든 체크인된 가구로 갑니다. 실제 긴급 상황(대피, 봉쇄, 악천후)에 사용합니다.
:::
