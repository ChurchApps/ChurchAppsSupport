# 가정 검토

<div class="article-intro">

검색 결과에서 사람을 선택하면 앱이 해당 가정 전체를 불러와 가정 검토 화면에 표시합니다. 이 화면은 체크인 과정의 중심 화면으로, 가족 구성원 모두를 확인하고 체크인 전에 그룹 배정을 관리할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 예배 선택 화면에서 [예배를 선택](./selecting-service)합니다.
- [구성원을 조회](./looking-up-members)하고 검색 결과에서 해당 구성원을 선택합니다.

</div>

## 가정 구성원 보기

각 가정 구성원은 다음 정보가 포함된 카드로 표시됩니다.

- **사진** (등록된 사진이 없으면 기본 아이콘 표시)
- **표시 이름**
- **현재 그룹 배정 요약** (카드가 접혀 있을 때 이름 아래 작은 칩으로 표시)
- 오른쪽의 **펼치기/접기 화살표**

## 구성원 펼치기

구성원의 카드를 탭하면 펼쳐집니다. 펼쳐진 상태에서는 선택한 예배의 각 **예배 시간**에 대한 행이 표시됩니다. 각 행에는 다음이 표시됩니다.

- **예배 시간 이름** (예: "오전 9시" 또는 "오전 11시")
- 현재 배정된 그룹 이름이 표시된 **버튼**, 배정된 그룹이 없으면 "그룹 선택"

그룹 버튼을 탭하면 [그룹 선택 화면](./group-assignment)으로 이동하여 해당 사람과 예배 시간의 그룹을 선택하거나 변경할 수 있습니다.

구성원의 카드를 다시 탭하면 접힙니다.

:::tip
모든 가족 구성원에게 그룹을 배정할 필요는 없습니다. 주 예배에 참석하는 성인은 그룹 배정 없이 두어도 됩니다. 특정 반이나 프로그램에 체크인해야 하는 구성원에게만 그룹을 배정하십시오.
:::

## 이 화면에서의 이동

가정 검토 화면에서는 세 가지 작업을 할 수 있습니다.

- **구성원을 펼치고** 예배 시간 버튼을 탭하여 [그룹을 배정하거나 변경](./group-assignment)합니다.
- 구성원 목록 아래의 **방문자 추가 버튼**을 탭하여 가정에 [방문자를 추가](./adding-guests)합니다.
- 화면 하단의 **체크인 버튼**을 탭하여 가정 전체의 [체크인을 완료](./completing-checkin)합니다.

:::info
동일한 예배에 이전에 체크인한 그룹 배정 정보가 자동으로 불러와집니다. 자녀들이 매주 같은 그룹에 참석한다면 변경 없이 바로 확인하고 체크인할 수 있습니다.
:::

## 다음 단계

- 특정 반이나 프로그램에 체크인해야 하는 가족 구성원에게 [그룹을 배정](./group-assignment)합니다.
- 가족과 함께 온 방문자가 있으면 [방문자를 추가](./adding-guests)합니다.
- 모든 배정이 올바르면 [체크인을 완료](./completing-checkin)합니다.
