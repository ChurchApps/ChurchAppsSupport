# 가족 검토

<div class="article-intro">

검색 결과에서 사람을 선택한 후 앱이 전체 가족을 로드하고 가족 검토 화면에 표시합니다. 이는 체크인 프로세스의 중앙 허브이며, 가족의 모든 사람을 볼 수 있고 체크인 전에 그룹 할당을 관리할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 예배 화면에서 [예배를 선택](./selecting-service)합니다
- [멤버를 찾고](./looking-up-members) 검색 결과에서 선택합니다

</div>

## 가족 구성원 보기

각 가족 구성원은 다음과 같은 카드로 나타납니다:

- 그들의 **사진**(파일에 없으면 자리 표시자 아이콘)
- 그들의 **표시 이름**
- **현재 그룹 할당 요약**(카드가 축소되면 이름 아래에 작은 칩으로 표시됨)
- 오른쪽에 있는 **확장/축소 화살표**

## 멤버 확장

멤버의 카드를 탭하여 확장합니다. 확장되면 선택한 예배 내의 각 **예배 시간**에 대한 행이 표시됩니다. 각 행은 다음을 표시합니다:

- **예배 시간 이름**(예: "오전 9:00" 또는 "오전 11:00")
- 현재 할당된 그룹을 표시하는 **버튼** 또는 아직 할당된 그룹이 없으면 "그룹 선택"

그룹 버튼을 탭하여 [그룹 선택 화면](./group-assignment)으로 이동하여 해당 사람 및 예배 시간에 대해 그룹을 선택하거나 변경할 수 있습니다.

확장된 카드는 또한 **Member**, **Guest** 및 **Volunteer** 칩을 표시합니다. 하나를 탭하여 이 사람이 체크인하는 방법을 설정합니다. Member가 기본값입니다. 유형은 어린이 객실에 중요합니다: 자원봉사자는 객실의 자원봉사자-자녀 비율에 포함되고, 손님은 교회에서 설정한 게스트 용량에 포함됩니다.

멤버의 카드를 다시 탭하여 축소합니다.

:::tip
모든 가족 구성원을 그룹에 할당할 필요는 없습니다. 주 예배에 참석하는 성인은 그룹 할당 없이 남겨질 수 있습니다. 특정 클래스 또는 프로그램에 체크인해야 하는 멤버에게만 그룹을 할당합니다.
:::

## 이 화면에서 탐색

가족 검토 화면에서 세 가지 옵션이 있습니다:

- **멤버를 확장하고** 예배 시간 버튼을 탭하여 [그룹을 할당하거나 변경](./group-assignment)합니다.
- **멤버 목록 아래의 Add Guest 버튼을 탭하여** 가족에 [손님을 추가](./adding-guests)합니다.
- **화면 하단의 Check-in 버튼을 탭하여** 가족의 모든 사람에 대해 [체크인을 완료](./completing-checkin)합니다.

:::info
같은 예배로의 이전 체크인의 모든 그룹 할당이 자동으로 사전 로드됩니다. 자녀가 매주 같은 그룹에 참석하면 아무것도 변경할 필요가 없을 수 있습니다. 검토하고 체크인하세요.
:::

## 다음 단계

- 특정 클래스 또는 프로그램에 체크인해야 하는 가족 구성원에게 [그룹을 할당](./group-assignment)합니다.
- 누군가 가족과 방문하는 경우 [손님을 추가](./adding-guests)합니다.
- 모든 사람의 할당이 올바르면 [체크인을 완료](./completing-checkin)합니다.
