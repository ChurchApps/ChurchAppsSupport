# 체크인 완료

<div class="article-intro">

가정을 검토하고 필요한 그룹 배정을 완료했다면 체크인을 마무리할 준비가 된 것입니다. 이 단계가 키오스크 워크플로의 마지막 단계입니다. 앱이 출석을 제출하고 라벨을 출력한 후 다음 가족을 위해 초기 상태로 돌아갑니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 가정 검토 화면에서 [가정을 검토](./household-review)합니다.
- 특정 반이나 프로그램에 체크인해야 하는 가족 구성원에게 [그룹을 배정](./group-assignment)합니다.
- 필요에 따라 가족과 함께 온 [방문자를 추가](./adding-guests)합니다.

</div>

## 체크인 방법

1. **가정 검토 화면**에서 화면 하단의 **체크인** 버튼을 탭합니다.
2. 앱이 출석 데이터를 서버에 제출하고 녹색 체크 표시와 환영 메시지가 있는 **성공 화면**을 표시합니다.

이것으로 완료됩니다. 가족의 출석이 기록되었습니다.

## 라벨 출력

네트워크 프린터가 설정된 경우 체크인 후 앱이 자동으로 라벨을 출력합니다.

- **이름 라벨**은 **이름 라벨 출력** 설정이 활성화된 그룹에 배정된 각 사람에게 출력됩니다. 이름 라벨에는 사람의 이름, 그룹 배정 정보, 그리고 기록된 알레르기/특이 사항 정보가 포함됩니다.
- **부모 인계 슬립**은 **부모 인계** 설정이 활성화된 그룹에 체크인한 사람이 있는 경우 출력됩니다. 인계 슬립에는 자녀 이름, 그룹 배정 정보, 그리고 고유한 **4자리 보안 코드**가 포함됩니다.

:::info
동일한 보안 코드가 자녀의 이름 라벨과 부모의 인계 슬립 양쪽에 모두 표시됩니다. 인계 시간에 봉사자가 코드를 대조하여 올바른 보호자가 각 자녀를 데려가는지 확인합니다.
:::

보안 코드는 체크인할 때마다 새로 생성되며, 부적절한 단어가 만들어지지 않도록 자음과 숫자만 사용합니다(모음은 제외).

:::warning
라벨이 출력되지 않으면 화면 상단의 프린터 상태 바를 확인하십시오. 상태 바를 탭하여 프린터 설정에 접근하고 연결 상태를 확인할 수 있습니다. 문제 해결 단계는 [프린터 설정](../getting-started/printer-setup)을 참조하십시오.
:::

## 체크인 후 진행되는 일

- 프린터가 설정된 경우 앱이 모든 라벨을 출력한 후 자동으로 **조회 화면**으로 돌아가 다음 가족을 위한 준비를 합니다.
- 프린터가 설정되지 않은 경우 성공 화면이 몇 초간 표시된 후 자동으로 **조회 화면**으로 돌아갑니다.

조회 화면으로 돌아가기 위해 별도로 탭할 필요가 없습니다. 앱이 자동으로 전환을 처리합니다.

:::tip
앱은 각 체크인 후 완전히 초기화되므로 한 가족이 다른 가족의 정보를 볼 위험이 없습니다.
:::

## 기록되는 정보

**체크인** 버튼을 탭하면 그룹이 배정된 각 가정 구성원에 대해 다음 정보가 서버로 전송됩니다.

- 체크인하는 **사람**
- 참석하는 **예배**
- 배정된 **예배 시간** 및 **그룹**

이 데이터는 B1 Admin의 출석 섹션에 표시되며, 교회 관리자가 출석 기록을 확인하고 관리할 수 있습니다. 자세한 내용은 [체크인 관리 가이드](../../b1-admin/attendance/check-in.md)를 참조하십시오.
