# 체크인 완료

<div class="article-intro">

가구를 검토하고 필요한 그룹 할당을 한 후 체크인을 마무리할 준비가 됩니다. 이것은 키오스크 워크플로우의 마지막 단계입니다 -- 앱이 출석을 제출하고 라벨을 인쇄하고 다음 가족을 위해 재설정합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 가구 검토 화면에서 [가구를 검토](./household-review)합니다
- 특정 클래스나 프로그램에 체크인해야 하는 가족 구성원에게 [그룹을 할당](./group-assignment)합니다
- 선택적으로 [방문객을 추가](./adding-guests)합니다

</div>

## 체크인 하는 방법

1. **가구 검토 화면**에서 화면 하단의 **체크인** 버튼을 탭합니다.
2. 앱이 출석 데이터를 서버에 제출하고 녹색 확인 표시와 환영 메시지가 있는 **성공 화면**을 표시합니다.

그렇게 간단합니다. 가족의 출석이 기록되었습니다.

## 라벨 인쇄

네트워크 프린터가 구성되어 있으면 앱이 체크인 후 자동으로 라벨을 인쇄합니다:

- **이름 라벨**은 **이름표 인쇄** 설정이 활성화된 그룹에 할당된 각 사람에 대해 인쇄됩니다. 이름 라벨에는 사람의 이름, 그룹 할당, 파일에 있는 경우 알레르기/메모 정보가 포함됩니다.
- **부모 픽업 슬립**은 체크인한 사람이 **부모 픽업** 설정이 활성화된 그룹에 있을 때 인쇄됩니다. 픽업 슬립은 아이들, 그룹 할당, 고유한 **4자 보안 코드**를 나열합니다.

:::info
동일한 보안 코드가 아이의 이름 라벨과 부모의 픽업 슬립 모두에 나타납니다. 픽업 시간에 자원봉사자들은 코드를 일치시켜 올바른 성인이 각 아이를 데려가는지 확인합니다.
:::

보안 코드는 매번 체크인할 때마다 새로 생성되며 자음과 숫자만 사용합니다(모음은 부적절한 단어 형성을 피하기 위해 제외됨).

:::warning
라벨이 인쇄되지 않으면 **교회 로고**를 7번 탭하여 관리자 설정을 열고 **프린터 변경**을 탭하여 프린터 연결을 확인합니다. 문제 해결 단계는 [프린터 설정](../getting-started/printer-setup)을 참조하세요.
:::

## 체크인 후 발생하는 일

- 프린터가 구성된 경우 앱이 모든 라벨을 인쇄한 후 자동으로 **조회 화면**으로 돌아가 다음 가족을 기다립니다.
- 프린터가 구성되지 않은 경우 성공 화면이 몇 초 동안 표시된 후 자동으로 **조회 화면**으로 돌아갑니다.

조회 화면으로 돌아가기 위해 아무것도 탭할 필요가 없습니다 -- 앱이 전환을 자동으로 처리합니다.

:::tip
앱이 체크인 후 완전히 재설정되므로 한 가족이 다른 가족의 정보를 볼 위험이 없습니다.
:::

## 기록되는 내용

**체크인**을 탭하면 앱이 그룹 할당이 있는 각 가구 회원에 대해 다음을 서버로 보냅니다:

- 체크인하는 **사람**
- 참석하는 **예배**
- 할당된 **예배 시간** 및 **그룹**

이 데이터는 B1 Admin의 출석 섹션에 나타나며 여기서 교회 관리자가 출석 기록을 보고 관리할 수 있습니다. 자세한 내용은 [체크인 관리 가이드](../../b1-admin/attendance/check-in.md)를 참조하세요.
