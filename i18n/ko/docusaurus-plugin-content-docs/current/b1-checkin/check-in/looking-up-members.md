# 구성원 조회

<div class="article-intro">

예배를 선택하면 구성원 조회 화면으로 이동합니다. 이 화면에서 가족을 검색하여 체크인을 진행합니다. 앱은 전화번호 또는 이름으로 검색을 지원합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 예배 선택 화면에서 [예배를 선택](./selecting-service)합니다.
- 교회 관리자가 가정을 [시스템에 등록](../../b1-admin/people/adding-people.md)했는지 확인합니다.

</div>

## 검색 모드

조회 화면에서는 두 가지 검색 모드를 제공합니다. 화면 상단의 토글을 탭하여 모드를 전환할 수 있습니다.

- **전화번호** -- 전화번호로 검색합니다. 기본 모드입니다.
- **이름** -- 이름으로 검색합니다.

## 전화번호로 검색

1. 화면 상단에 **전화번호** 검색 모드가 활성화되어 있는지 확인합니다(강조 표시됨).
2. 검색 필드를 탭하고 화면 키패드를 사용하여 전화번호의 **4자리 이상**을 입력합니다.
3. **검색** 버튼을 탭하거나 키보드의 검색 키를 누릅니다.

앱이 입력한 숫자와 전화번호가 일치하는 구성원을 검색합니다. 4자리 이상을 입력하면 전체 번호를 사용하며, 정확히 4자리를 입력하면 등록된 전화번호의 마지막 4자리와 대조합니다.

## 이름으로 검색

1. 화면 상단의 **이름** 토글을 탭하여 이름 검색 모드로 전환합니다.
2. 검색 필드에 사람 이름의 **2자 이상**을 입력합니다.
3. **검색** 버튼을 탭합니다.

앱이 입력한 내용과 이름이 일치하는 구성원을 검색합니다.

## 사람 선택

검색 결과가 검색 필드 아래에 카드로 표시됩니다. 각 카드에는 사람의 **이름**과 **사진**(등록된 경우)이 표시됩니다.

사람의 카드를 탭하여 선택합니다. 앱이 해당 사람의 가정 구성원 전체를 불러와 [가정 검토 화면](./household-review)으로 이동합니다.

:::tip
전화번호 마지막 4자리로 검색하는 것이 가족을 가장 빠르게 찾는 방법입니다. 구성원들이 이 방법을 사용하도록 안내하면 체크인 시간을 단축할 수 있습니다.
:::

:::warning
검색 결과가 없으면 앱이 안내 메시지를 표시합니다. 입력한 숫자나 이름을 다시 확인하거나 다른 검색 모드를 시도해 보십시오. 처음 방문하는 분이라면 봉사자에게 [시스템 등록](../../b1-admin/people/adding-people.md)을 도움을 요청하십시오.
:::
