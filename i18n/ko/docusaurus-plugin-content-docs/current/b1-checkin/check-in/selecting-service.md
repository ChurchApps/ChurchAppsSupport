# 예배 선택

<div class="article-intro">

모든 체크인의 첫 번째 단계는 참석할 예배를 선택하는 것입니다. 예배 선택 화면은 앱 로딩이 완료된 직후 표시되며, 이후 체크인 흐름에서 사용할 수 있는 예배 시간과 그룹을 결정합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1 Church Checkin 앱에 [로그인](../getting-started/logging-in)하고 교회를 선택합니다.
- 교회 관리자가 B1 Admin에서 [예배 및 예배 시간을 설정](../../b1-admin/attendance/setup.md)했는지 확인합니다.

</div>

## 작동 방식

1. 로그인 후(또는 재방문 시 자동 로그인 후) 앱이 **예배 선택 화면**을 표시합니다.
2. 교회에 설정된 모든 예배 목록이 표시됩니다. 각 예배는 예배 이름(예: "주일 오전" 또는 "수요 저녁")이 표시된 카드로 나타납니다.
3. 참석할 예배를 탭합니다.

앱이 해당 예배와 연결된 예배 시간 및 그룹을 불러온 후 [구성원 조회 화면](./looking-up-members)으로 이동합니다.

:::info
예배는 교회 관리자가 B1 Admin의 출석 섹션에서 설정합니다. 원하는 예배가 표시되지 않으면 관리자에게 [출석 설정](../../b1-admin/attendance/setup.md)에서 해당 예배가 생성되었는지 확인을 요청하십시오.
:::

## 배경에서 일어나는 일

예배를 선택하면 앱이 서버에서 다음 세 가지 정보를 가져옵니다.

- 해당 예배의 **예배 시간** (예: 하나의 예배에 오전 9시와 오전 11시 시간대가 있을 수 있습니다).
- 각 예배 시간에 사용 가능한 **그룹** (예: 영아부, 유아부, 초등부).
- 어떤 시간대에 어떤 그룹이 제공되는지를 결정하는 **그룹-예배 시간 연결 정보**.

이 데이터는 로컬에 캐시되어 이후 체크인 과정이 빠르고 원활하게 진행됩니다.

:::tip
교회에 예배가 하나만 설정된 경우에도 계속 진행하려면 해당 예배를 탭해야 합니다. 앱은 예배가 하나뿐이더라도 자동으로 선택하지 않습니다.
:::
