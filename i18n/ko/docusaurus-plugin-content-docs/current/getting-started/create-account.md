# 계정 만들기

<div class="article-intro">

ChurchApps로 시작하는 것은 계정을 만드는 것부터 시작됩니다. 프로세스는 몇 분 정도 소요되며 B1 Admin, B1.church, Lessons.church를 포함한 모든 ChurchApps 도구에 액세스할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 계정 확인을 위해 유효한 이메일 주소를 준비합니다
- [시작하기 개요](./index.md)를 검토하여 ChurchApps가 제공하는 것을 이해합니다

</div>

## 단계별 등록

1. 웹 브라우저에서 [B1.church](https://b1.church)로 이동합니다.
2. **등록** 버튼을 클릭합니다.
3. **이름**, **성**, **이메일 주소**를 입력합니다.
4. 계정에 대한 암호를 만듭니다.
5. 등록 양식을 제출합니다.

## 이메일 확인

등록 후 제공한 주소에서 확인 이메일을 받게 됩니다.

1. ChurchApps의 확인 이메일에 대한 받은 편지함을 확인합니다.
2. 이메일의 확인 링크를 클릭합니다.
3. 계정이 이제 확인되고 사용할 준비가 되었습니다.

:::tip
확인 이메일이 보이지 않으면 스팸 또는 정크 폴더를 확인하세요. 이메일은 ChurchApps에서 오며 도착하는 데 1-2분 정도 걸릴 수 있습니다.
:::

## 교회 설정

이메일을 확인하고 로그인한 후 새 교회를 만들거나 기존 교회에 가입할 수 있는 옵션이 있습니다.

**새 교회를 만들려면:**

1. **새 교회 만들기** 옵션을 선택합니다.
2. **교회 이름** 및 기본 정보를 입력합니다. 일치 항목이 발견되면 등록 양식이 교회 세부사항을 자동으로 채웁니다.
3. 설정 프로세스를 완료합니다.
4. 교회 구성을 시작할 수 있는 **B1 Admin 대시보드**로 이동됩니다.

**기존 교회에 가입하려면:**

1. **기존 교회 가입** 옵션을 선택합니다.
2. 이름으로 교회를 검색합니다.
3. 가입을 요청합니다. 교회의 관리자가 요청을 승인해야 합니다.

:::info
아직 등록되지 않은 이메일로 로그인을 시도하면 계정을 만들 수 있도록 등록 흐름으로 자동 이동됩니다.
:::

:::info
새 교회를 만드는 사람에게는 모든 설정 및 기능에 대한 전체 액세스 권한을 부여하는 **도메인 관리자** 역할이 자동으로 할당됩니다. 나중에 **역할** 페이지에서 추가 관리자를 추가할 수 있습니다.
:::
