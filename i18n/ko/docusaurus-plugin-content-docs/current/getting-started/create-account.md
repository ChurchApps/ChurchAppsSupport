# 계정 만들기

<div class="article-intro">

ChurchApps를 시작하려면 먼저 계정을 만들어야 합니다. 이 과정은 몇 분밖에 걸리지 않으며, B1 Admin, B1.church, Lessons.church를 포함한 모든 ChurchApps 도구에 접근할 수 있게 됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 계정 인증에 사용할 유효한 이메일 주소를 준비하세요
- [시작하기 개요](./index.md)를 검토하여 ChurchApps가 제공하는 내용을 파악하세요

</div>

## 단계별 등록

1. 웹 브라우저에서 [B1.church](https://b1.church)로 이동합니다.
2. **Register(등록)** 버튼을 클릭합니다.
3. **이름**, **성**, **이메일 주소**를 입력합니다.
4. 계정에 사용할 비밀번호를 만듭니다.
5. 등록 양식을 제출합니다.

## 이메일 확인

등록 후, 입력한 이메일 주소로 확인 이메일이 발송됩니다.

1. 받은 편지함에서 ChurchApps로부터 온 확인 이메일을 확인합니다.
2. 이메일의 확인 링크를 클릭합니다.
3. 이제 계정이 인증되어 사용할 준비가 되었습니다.

:::tip
확인 이메일이 보이지 않으면 스팸 또는 정크 폴더를 확인하세요. 이메일은 ChurchApps에서 발송되며 도착하는 데 1~2분이 걸릴 수 있습니다.
:::

## 교회 설정

이메일을 확인하고 로그인하면 새 교회를 만들거나 기존 교회에 참여하는 옵션이 표시됩니다.

**새 교회를 만들려면:**

1. **새 교회 만들기** 옵션을 선택합니다.
2. **교회 이름**과 기본 정보를 입력합니다.
3. 설정 과정을 완료합니다.
4. 교회 구성을 시작할 수 있는 **B1 Admin 대시보드**로 이동합니다.

**기존 교회에 참여하려면:**

1. **기존 교회 참여** 옵션을 선택합니다.
2. 이름으로 교회를 검색합니다.
3. 참여 요청을 보냅니다. 교회의 관리자가 요청을 승인해야 합니다.

:::info
새 교회를 만든 사람은 자동으로 **도메인 관리자** 역할이 부여되며, 모든 설정 및 기능에 대한 전체 접근 권한이 주어집니다. 나중에 **역할** 페이지에서 추가 관리자를 등록할 수 있습니다.
:::
