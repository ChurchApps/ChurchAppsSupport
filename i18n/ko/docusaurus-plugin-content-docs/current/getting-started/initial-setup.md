# 초기 설정

<div class="article-intro">

계정과 교회를 만든 후, 팀을 초대하기 전에 구성해야 할 몇 가지 중요한 단계가 있습니다. 이 가이드는 교회가 첫날부터 준비될 수 있도록 권장 설정 순서를 안내합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [계정을 만들고](./create-account.md) 교회를 등록하세요
- 교회 로고 및 브랜딩 자산을 준비하세요 (선택 사항이지만 권장됩니다)

</div>

## 1단계: 교회 정보 구성

1. 왼쪽 사이드바에서 **Settings(설정)**으로 이동합니다.
2. **Edit Settings(설정 편집)**를 클릭합니다.
3. 교회의 **이름**, **주소**, **연락처 정보**를 입력합니다.
4. **Save(저장)**를 클릭하여 변경 사항을 적용합니다.

## 2단계: 브랜딩 설정

1. **Settings(설정)** 페이지에서 **교회 로고**를 업로드합니다.
2. 색상과 이미지 등 추가 브랜딩 옵션을 구성합니다.
3. 브랜딩은 B1.church 웹사이트와 B1 모바일 앱에 표시됩니다.

:::tip
팀원을 초대하기 전에 로고와 브랜딩을 준비해 두면 그들이 로그인할 때 완성된 첫인상을 줄 수 있습니다.
:::

## 3단계: 역할 구성 및 팀 초대

1. **Settings(설정)** 페이지에서 **Roles(역할)**를 클릭합니다.
2. 팀을 위한 역할을 만듭니다 (예: "목사", "비서", "재무").
3. 각 역할에 적절한 권한을 할당합니다.
4. 팀원의 계정을 검색하고 역할에 배정하여 팀원을 추가합니다.

## 4단계: 핵심 영역 설정

팀이 구성되면 B1 Admin의 핵심 영역을 구축하기 시작합니다:

1. **People(사람)** -- 교회 디렉토리에 회원을 추가합니다. 이것이 다른 모든 것의 기반이 되므로 여기서 시작하세요. 수동으로 사람을 추가하거나 **Settings(설정)**의 **Import(가져오기)** 도구를 사용할 수 있습니다.
2. **Groups(그룹)** -- 소그룹, 사역 팀, 반, 기타 모임을 위한 그룹을 만듭니다.
3. **Attendance(출석)** -- 출석 추적을 위해 캠퍼스, 예배, 예배 시간을 구성합니다.
4. **Donations(헌금)** -- 헌금 기금을 설정하고 온라인 헌금을 받기 위해 결제 게이트웨이(Stripe)를 연결합니다.
5. **Website(웹사이트)** -- B1.church 웹사이트 빌더를 사용하여 공개 교회 웹사이트를 구축합니다.

:::info
모든 영역을 한 번에 설정할 필요는 없습니다. 많은 교회들이 **People(사람)**과 **Groups(그룹)**으로 시작한 후, 플랫폼에 익숙해지면 **Attendance(출석)**와 **Donations(헌금)**을 추가합니다.
:::

## 권장 순서

가장 원활한 설정 경험을 위해 다음 순서를 권장합니다:

1. **People(사람)** -- 먼저 회원과 정기 참석자를 추가합니다.
2. **Groups(그룹)** -- 사람들을 의미 있는 그룹으로 구성합니다.
3. **Attendance(출석)** -- 매주 출석하는 사람을 추적하기 시작합니다.
4. **Donations(헌금)** -- 사람들이 시스템에 등록된 후 헌금을 설정합니다.
5. **Website(웹사이트)** -- 데이터가 준비된 후 마지막으로 공개 사이트를 구축합니다.

이 단계를 완료하면 교회는 시작할 준비가 됩니다. 각 영역에 대한 자세한 가이드는 [B1 Admin](/docs/b1-admin/) 문서를 참조하세요.
