# 서버 관리

<div class="article-intro">

ChurchApps의 서버 관리 기능은 **Server.Admin** 권한이 있는 사용자만 사용할 수 있습니다. 이 도구는 플랫폼 운영, 지원, 시스템 전체의 문제 해결에 사용됩니다.

</div>

:::warning 접근 제한
이 페이지에 설명된 기능은 **Server.Admin** 권한이 필요하며 일반 교회 관리자는 사용할 수 없습니다. 플랫폼 운영자 및 지원 직원용입니다.
:::

## 서버 관리 액세스

Server.Admin 권한이 있는 사용자는 B1 Admin에서 서버 관리 패널에 액세스할 수 있습니다:

1. [admin.b1.church](https://admin.b1.church)에 로그인합니다
2. 주 네비게이션에서 **관리** 탭을 클릭합니다
3. 서버 관리 패널은 교회, 사용자, 시스템 운영을 관리하는 탭을 포함합니다

## 사용자 가장

가장 기능을 사용하면 서버 관리자가 다른 사용자로 로그인하여 지원 및 문제 해결할 수 있습니다. 사용자 보고 문제를 조사하거나 교회가 시스템을 구성하는 것을 도울 때 유용합니다.

### 사용자 가장하는 방법

1. 서버 관리 패널의 **가장** 탭으로 이동합니다
2. 검색 필드에 사용자 이름 또는 이메일 주소를 입력합니다
3. **검색**을 클릭하거나 Enter를 누릅니다
4. 검색 결과에서 가장할 사용자를 클릭합니다
5. 나타나는 대화에서 가장을 확인합니다
6. 해당 사용자로 로그인되고 계정으로 리디렉션됩니다

### 중요 참고

- 가장은 대상 사용자의 권한 및 교회 액세스로 새로운 세션을 생성합니다
- 원래 관리자 세션은 다른 사용자를 가장할 때 종료됩니다
- 가장된 동안의 모든 작업은 감사 추적에 기록됩니다
- 관리자 계정으로 돌아가려면 로그아웃하고 자격 증명으로 다시 로그인합니다
- 가장은 지원 목적으로만 필요할 때 사용하고 항상 사용자에게 지원을 위해 계정에 액세스하고 있음을 알립니다
