# Lessons.church

<div class="article-intro">

Lessons.church는 교회를 위한 완전 무료 커리큘럼 플랫폼입니다. 구독료, 이용료, 사용 제한이 전혀 없습니다. 12개 이상의 공급자로부터 189개 이상의 학습 과정에 걸쳐 1,100개 이상의 영상 기반 공과를 제공하며, 유아부터 성인까지 모든 연령대를 대상으로 합니다.

</div>

## 할 수 있는 것들

Lessons.church를 통해 교회에서 할 수 있는 것들은 다음과 같습니다:

- **콘텐츠 탐색** -- 연령대와 주제별로 정리된 다양한 공급자의 프로그램과 학습 과정으로 구성된 점점 성장하는 라이브러리를 살펴볼 수 있습니다.
- **교실 설정** -- 교회의 사역 구조에 맞는 교실을 만들 수 있습니다 (예: 유아부, 초등부, 청소년부, 성인부).
- **공과 일정 관리** -- 필요한 날짜에 특정 공과를 교실에 배정할 수 있습니다.
- **필요에 맞게 맞춤 설정** -- 교회의 교육 방식과 선호에 맞게 공과 내용을 조정할 수 있습니다.
- **수업 진행** -- 내장된 발표자 모드를 사용하여 수업 시간에 공과를 단계별로 진행할 수 있습니다.

## 콘텐츠 구성 방식

Lessons.church는 커리큘럼을 명확한 계층 구조로 정리합니다:

1. **프로그램** -- 공급자가 제공하는 최상위 커리큘럼 시리즈입니다 (예: 어린이 사역 프로그램).
2. **학습 과정** -- 프로그램 내에서 관련된 공과들을 묶은 그룹입니다 (예: 특정 주제에 대한 4주간의 시리즈).
3. **공과** -- 학습 과정 내의 개별 교육 세션으로, 일반적으로 한 번의 수업 시간에 사용됩니다.
4. **장소** -- 특정 연령대나 환경에 맞게 조정된 공과의 다른 버전입니다 (예: 유아 버전과 초등부 버전).

## 통합 기능

Lessons.church는 다른 ChurchApps 도구와 연동하여 콘텐츠를 전달하는 데 도움을 줍니다:

- **B1.church** -- 봉사자와 학부모가 B1 앱을 통해 예정된 공과와 최근 공과를 확인할 수 있습니다.
- **FreeShow** -- FreeShow 프레젠테이션 소프트웨어를 사용하여 화면에 공과 내용을 표시할 수 있습니다.
- **Venue TV 앱** -- Android TV 앱을 사용하여 TV에 직접 공과를 표시할 수 있습니다.
