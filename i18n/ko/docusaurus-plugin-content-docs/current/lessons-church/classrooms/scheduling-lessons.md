# 공과 일정 관리

<div class="article-intro">

교실이 설정되면, 다음 단계는 봉사자들이 무엇을 언제 가르쳐야 하는지 정확히 알 수 있도록 공과 일정을 잡는 것입니다. 일정 관리는 Lessons.church의 공과 콘텐츠를 교실 캘린더에 연결합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 포털에서 [최소 하나의 교실을 설정](./setting-up-classrooms.md)하십시오
- 가르치고 싶은 프로그램과 학습 과정을 찾기 위해 [콘텐츠 라이브러리를 탐색](../browsing/index.md)하십시오

</div>

## 교실 일정 보기

1. 로그인하여 **포털**로 이동합니다.
2. 일정을 잡으려는 **교실**을 클릭합니다.
3. 일정 화면에 **지난 4개의 공과**와 **다가오는 4개의 공과**가 표시됩니다. 최근 가르친 내용과 앞으로의 일정을 빠르게 확인할 수 있습니다.

## 일정에 공과 추가하기

1. 교실 일정 화면에서 새 공과를 추가하려면 클릭합니다.
2. 공과의 **날짜**를 선택합니다.
3. 가르칠 **프로그램**을 선택합니다 (예: "어린이 성경 이야기").
4. 해당 프로그램 내의 **학습 과정**을 선택합니다 (예: "창조와 초기 이야기").
5. 일정에 넣을 특정 **공과**를 선택합니다.
6. 이 교실에 적합한 **장소**를 선택합니다. 장소는 공과 내용의 연령대별 버전을 나타냅니다 (예: "유아부" 또는 "초등부"). 올바른 장소를 선택하면 봉사자들이 해당 교실의 연령대에 맞는 콘텐츠를 볼 수 있습니다.
7. 일정에 추가된 공과를 저장합니다.

해당 날짜에 교실 일정에 공과가 표시됩니다.

:::info
프로그램, 학습 과정, 공과는 Lessons.church의 콘텐츠 계층 구조입니다. 프로그램에는 여러 학습 과정이 포함되고, 각 학습 과정에는 개별 공과가 포함됩니다. 장소는 같은 공과의 연령대별 변형을 제공합니다. 전체 설명은 [프로그램과 학습 과정](../browsing/programs-and-studies.md)을 참조하십시오.
:::

## 봉사자의 일정 활용 방법

공과 일정을 잡으면, 봉사자들이 포털을 열고 자신의 교실을 선택하여 다가오는 일정을 바로 확인할 수 있습니다. 미리 공과 내용을 검토하고, 필요한 자료를 다운로드하며, 준비된 상태로 올 수 있습니다. 더 이상 다음에 무엇을 가르칠지 고민할 필요가 없습니다.

## 일정 관리 팁

- **미리 계획하기** -- 팀이 미리 준비할 수 있도록 몇 주치 공과를 한꺼번에 일정에 넣으십시오.
- **일관성 유지** -- 프로그램을 순서대로 따르면 아이들이 지난주에 배운 내용을 바탕으로 발전할 수 있습니다.
- **이력 확인** -- 지난 공과 목록을 사용하여 실수로 같은 내용을 반복하지 않도록 하십시오.

:::warning
교실의 연령대에 맞는 올바른 장소를 선택하십시오. 잘못된 장소를 선택하면 봉사자들이 다른 연령대에 맞게 설계된 콘텐츠를 보게 됩니다.
:::
