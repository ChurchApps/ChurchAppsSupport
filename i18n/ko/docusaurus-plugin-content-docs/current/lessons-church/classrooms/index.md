# 교실

<div class="article-intro">

교실은 Lessons.church에서 교회의 교육을 조직하는 기본 단위입니다. 교실은 공과가 진행되는 물리적 또는 가상 공간을 나타냅니다 -- 예를 들어, "유아부 교실", "초등부 K-2", "성인 성경공부" 등입니다. 이 섹션에서는 교실을 만들고 공과를 배정하는 방법을 다룹니다.

</div>

## 교실이 중요한 이유

교실을 설정하면 어린이 사역, 청소년 그룹 또는 기타 교육 환경에 대한 명확한 구조를 만들 수 있습니다. 각 교실은 자체 일정을 가지므로 봉사자들은 언제 무엇을 가르쳐야 하는지 정확히 알 수 있습니다. 종이 일정표를 돌리거나 마지막 순간에 이메일을 보내는 대신, 모든 사람이 포털에서 다가오는 일정을 확인할 수 있습니다.

## 교실로 할 수 있는 것들

- **무제한 교실 생성** -- 교회에서 필요한 만큼 교실을 만들 수 있습니다. 제한이 없으므로 모든 연령대, 캠퍼스 또는 프로그램에 대해 별도의 교실을 설정할 수 있습니다.
- **공과 일정 관리** -- 특정 날짜에 각 교실에 특정 공과를 배정합니다. 일정표에서 과거와 예정된 공과를 한눈에 확인할 수 있습니다.
- **콘텐츠 맞춤 설정** -- 각 교실에 맞게 공과 내용을 조정하여 봉사자들이 자신의 연령대에 관련된 내용만 볼 수 있도록 합니다. 자세한 내용은 [맞춤 설정](../customization/index.md)을 참조하십시오.

## 교실의 연결 방식

교실은 Lessons.church의 공과 콘텐츠를 교회의 실제 교육 일정에 연결합니다. 일반적인 워크플로우는 다음과 같습니다:

1. Lessons.church에서 이용 가능한 프로그램과 학습 과정을 [탐색](../browsing/index.md)합니다.
2. 교회의 교실과 연령대에 맞게 교실을 설정합니다.
3. 다가오는 날짜에 각 교실에 공과를 배정합니다.
4. 봉사자들이 자신의 교실을 열어 배정된 일정을 확인하고 준비합니다.
