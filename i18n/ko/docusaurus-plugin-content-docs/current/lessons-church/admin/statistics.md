# 통계

<div class="article-intro">

통계 페이지는 관리자에게 콘텐츠가 어떻게 사용되고 있는지에 대한 통찰을 제공합니다. 어떤 학습 과정이 가장 인기 있는지, 어떤 교회가 자료를 다운로드하고 있는지, 해당 교회가 지리적으로 어디에 위치하고 있는지 확인할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- Lessons.church에 대한 관리자 권한을 보유하고 있어야 합니다 ([관리 개요](./index.md) 참조)
- 게시된 콘텐츠가 포함된 [프로그램](./managing-programs.md)이 최소 하나 있어야 합니다

</div>

## 통계 접근하기

1. **관리** 영역으로 이동합니다.
2. **통계** 페이지를 선택합니다.
3. 통계를 보고자 하는 **프로그램**을 선택합니다.

## 확인할 수 있는 내용

통계 페이지는 다운로드 데이터의 여러 보기를 제공합니다:

### 학습 과정별 다운로드

선택한 프로그램 내 각 학습 과정의 고유 다운로드 수를 분류하여 보여줍니다. 어떤 시리즈가 가장 많은 관심을 받고 있는지 확인하고 콘텐츠 제작 노력을 어디에 집중할지 결정하는 데 도움이 됩니다.

### 교회 목록

프로그램에서 자료를 다운로드한 교회들의 목록입니다. 대상층을 파악하고 콘텐츠가 얼마나 널리 채택되고 있는지 이해하는 데 도움이 됩니다.

### 지역별 분포

다운로드한 교회의 위치를 지도로 보여줍니다. 커리큘럼의 지리적 도달 범위를 이해하고 콘텐츠가 인기 있는 지역을 파악하는 데 도움이 됩니다.

### 날짜 범위 필터링

모든 통계를 날짜 범위로 필터링하여 특정 기간에 집중할 수 있습니다. 시간에 따른 추세를 확인하거나 새로 출시된 학습 과정의 영향을 측정하는 데 유용합니다.

1. 날짜 선택기를 사용하여 **시작 날짜**와 **종료 날짜**를 설정합니다.
2. 통계가 해당 범위 내의 활동만 반영하도록 업데이트됩니다.

## 통계 효과적으로 활용하기

다운로드 데이터를 활용하는 몇 가지 방법은 다음과 같습니다:

- **인기 콘텐츠 파악** -- 가장 많은 다운로드를 받는 스타일과 주제의 자료를 더 많이 제작하는 데 노력을 집중하십시오.
- **신규 출시 추적** -- 새로운 학습 과정을 게시한 후 날짜로 필터링하여 얼마나 빠르게 채택되고 있는지 확인하십시오.
- **대상층 이해** -- 교회 목록과 지도를 사용하여 누가 콘텐츠를 사용하고 있는지 확인하고 해당 커뮤니티에 연락하는 것을 고려하십시오.

:::info
통계는 고유 다운로드를 기반으로 합니다. 같은 교회가 자료를 여러 번 다운로드해도 한 번으로 계산됩니다.
:::
