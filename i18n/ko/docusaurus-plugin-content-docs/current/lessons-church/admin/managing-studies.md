# 학습 과정 관리

<div class="article-intro">

학습 과정은 프로그램 내의 관련된 공과들의 시리즈입니다. 예를 들어, "선한 사마리아인"이라는 4주간의 시리즈는 더 큰 어린이 사역 프로그램 안의 학습 과정이 됩니다. 이 가이드는 관리 영역에서 학습 과정을 생성하고, 편집하고, 정리하는 방법을 다룹니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 관리 영역에서 [최소 하나의 프로그램을 생성](./managing-programs.md)하십시오
- Lessons.church에 대한 관리자 권한을 보유하고 있어야 합니다 ([관리 개요](./index.md) 참조)

</div>

## 학습 과정 만들기

1. **관리** 영역으로 이동하여 학습 과정을 추가할 프로그램을 펼칩니다.
2. **학습 과정 추가**를 클릭합니다.
3. 다음 항목을 입력합니다:
   - **이름** -- 학습 과정의 표시 이름입니다 (예: "선한 사마리아인").
   - **슬러그** -- URL에 사용되는 식별자입니다 (예: "the-good-samaritan").
   - **설명** -- 이 학습 과정이 다루는 내용과 포함된 공과 수에 대한 요약입니다.
   - **이미지** -- 학습 과정의 썸네일 이미지를 업로드합니다.
4. 학습 과정을 저장합니다.

## 학습 과정 편집하기

1. 관리 목록에서 학습 과정 이름을 클릭합니다.
2. 필요에 따라 항목을 수정합니다.
3. 변경 사항을 저장합니다.

## 카테고리 지정하기

상위 프로그램이 학습 과정 카테고리를 사용하는 경우, 학습 과정에 카테고리를 지정할 수 있습니다:

1. 관리 화면에서 학습 과정을 엽니다.
2. 드롭다운에서 **카테고리**를 선택합니다.
3. 저장합니다.

카테고리는 학습 과정이 많은 프로그램을 사용자가 쉽게 탐색할 수 있도록 도와줍니다. 예를 들어, "구약", "신약", "주제별"과 같은 카테고리를 만들 수 있습니다.

## 학습 과정 내 공과 관리하기

학습 과정은 하나 이상의 공과를 포함합니다. 관리 화면에서 학습 과정을 펼치면 해당 공과를 볼 수 있으며, **공과 추가**를 클릭하여 새 공과를 만들 수도 있습니다. 공과 콘텐츠 작성에 대한 자세한 내용은 [공과 관리](./managing-lessons.md)를 참조하십시오.

## 학습 과정 수준 자료 업로드하기

학습 과정 전체에 적용되는 다운로드 가능한 파일을 첨부할 수 있습니다 -- 예를 들어, 시리즈 개요, 홍보 그래픽, 학부모 소식지 템플릿 등입니다.

1. 관리 화면에서 학습 과정을 엽니다.
2. **자료** 섹션으로 이동합니다.
3. 파일을 업로드합니다.

:::tip
교회 지도자를 염두에 두고 학습 과정 설명을 작성하십시오. 주제, 대상 연령대, 공과 수에 대한 명확한 요약은 교회들이 해당 시리즈를 사용할지 결정하는 데 도움이 됩니다.
:::
