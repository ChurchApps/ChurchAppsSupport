# 공과 관리

<div class="article-intro">

공과는 학습 과정 내의 개별 교육 세션입니다. 각 공과는 한 번의 수업 시간에 일어나는 내용을 나타냅니다. 여기에서 봉사자가 가르칠 때 사용할 상세한 콘텐츠 -- 장소, 섹션, 역할, 활동 -- 를 작성합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [프로그램을 생성](./managing-programs.md)하고 [학습 과정을 추가](./managing-studies.md)하십시오
- Lessons.church에 대한 관리자 권한을 보유하고 있어야 합니다 ([관리 개요](./index.md) 참조)

</div>

## 공과 만들기

1. **관리** 영역으로 이동하여 프로그램과 학습 과정을 펼치고, 공과를 추가할 학습 과정을 찾습니다.
2. **공과 추가**를 클릭합니다.
3. 공과의 **이름** 및 기타 세부 사항을 입력합니다.
4. 공과를 저장합니다.

## 공과 구조 이해하기

각 공과는 여러 계층으로 구성됩니다:

- **장소** -- 연령대나 환경에 따른 공과의 다른 버전입니다. 예를 들어, 같은 공과에 대해 "유아부" 장소와 "초등부" 장소를 만들 수 있습니다.
- **섹션** -- 장소 내의 구간으로, "환영", "영상", "성경 이야기", "활동" 등이 있습니다. 섹션은 수업 시간의 흐름을 제어합니다.
- **역할과 활동** -- 각 섹션 내에서 역할(예: "대표 교사" 또는 "소그룹 리더")을 지정하고 각 역할에 구체적인 활동을 부여합니다.

## 활동 유형

섹션 내의 역할에 지정할 수 있는 네 가지 유형의 활동이 있습니다:

- **말하기** -- 봉사자가 반에서 말하거나 의역해야 할 텍스트입니다.
- **행동하기** -- 수행해야 할 신체 활동이나 지시 사항입니다.
- **참고** -- 봉사자를 위한 배경 정보나 알림입니다 (소리 내어 읽지 않습니다).
- **재생** -- 섹션 중에 재생할 영상 또는 오디오 클립입니다.

## 장소 추가하기

1. 관리 화면에서 공과를 엽니다.
2. **장소 추가**를 클릭합니다.
3. 장소의 이름을 입력합니다 (예: "유아부" 또는 "초등부").
4. 저장합니다. 이제 이 장소에 섹션을 추가할 수 있습니다.

## 섹션 및 활동 추가하기

1. 장소를 펼쳐 섹션을 확인합니다.
2. **섹션 추가**를 클릭하고 이름을 입력합니다 (예: "환영").
3. 섹션 내에서 **활동 추가**를 클릭합니다.
4. **역할**, **활동 유형** (말하기, 행동하기, 참고, 재생)을 선택하고 내용을 입력합니다.
5. 저장합니다.

이 과정을 반복하여 전체 공과 흐름을 작성합니다.

## 공과 수준 자료 업로드하기

공과에 직접 다운로드 가능한 자료를 첨부할 수 있습니다 -- 예를 들어, 해당 주간의 활동지나 색칠 공부 등입니다.

1. 관리 화면에서 공과를 엽니다.
2. **자료** 섹션으로 이동합니다.
3. 파일을 업로드합니다.

:::tip
봉사자의 관점에서 공과를 작성하십시오. 섹션과 활동을 순서대로 살펴보며 처음 봉사하는 사람이 별도의 준비 없이도 따라갈 수 있는지 자문해 보십시오.
:::
