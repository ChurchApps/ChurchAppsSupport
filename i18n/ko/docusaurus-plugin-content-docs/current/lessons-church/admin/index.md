# 관리

<div class="article-intro">

Lessons.church의 관리 영역은 콘텐츠 제작자와 관리자가 공과 라이브러리를 관리하는 곳입니다. 여기에서 커리큘럼 구조를 구축하고, 자료를 업로드하며, 사용 통계를 검토하고, 외부 콘텐츠 공급자를 연결할 수 있습니다.

</div>

:::warning
관리 기능에 접근하려면 적절한 권한이 필요합니다. 관리 옵션이 보이지 않는 경우, 교회 관리자에게 접근 권한을 요청하십시오.
:::

## 할 수 있는 것들

관리 영역에서는 콘텐츠를 완전히 제어할 수 있습니다:

- **프로그램 생성 및 편집** -- 이름, 설명, 이미지, 슬러그를 포함한 최상위 커리큘럼 시리즈를 설정합니다.
- **학습 과정 관리** -- 각 프로그램 내에서 관련된 공과들의 그룹을 정리합니다.
- **공과 작성** -- 장소, 섹션, 역할, 활동이 포함된 개별 공과를 추가합니다.
- **자료 업로드** -- 프로그램, 학습 과정, 공과 수준에서 다운로드 가능한 파일(PDF, 활동지, 이미지)을 첨부합니다.
- **통계 보기** -- 어떤 콘텐츠가 누구에 의해, 어디에서 다운로드되고 있는지 확인합니다.
- **공급자 설정** -- Open Lesson Format을 사용하는 외부 커리큘럼 공급자를 연결합니다.

## 콘텐츠 계층 구조

관리자로서 콘텐츠가 어떻게 구성되어 있는지 이해하는 것이 도움이 됩니다:

1. **프로그램**은 하나 이상의 학습 과정을 포함합니다.
2. **학습 과정**은 하나 이상의 공과를 포함합니다.
3. **공과**는 하나 이상의 장소(연령대별 변형)를 포함합니다.
4. **장소**는 섹션을 포함하고, 섹션은 역할 기반 활동을 포함합니다.

각 수준에는 자체 다운로드 가능한 자료를 첨부할 수 있습니다.
