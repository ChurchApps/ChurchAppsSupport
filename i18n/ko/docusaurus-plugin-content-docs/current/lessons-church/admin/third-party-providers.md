# 외부 공급자

<div class="article-intro">

Lessons.church는 Open Lesson Format을 통해 외부 커리큘럼 공급자를 지원합니다. 이는 다른 조직의 콘텐츠를 가져와서 내장 라이브러리와 함께 제공할 수 있음을 의미하며, 교회들이 한 곳에서 여러 출처의 커리큘럼에 접근할 수 있게 합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- Lessons.church에 대한 관리자 권한을 보유하고 있어야 합니다 ([관리 개요](./index.md) 참조)
- 연결하려는 외부 공급자의 피드 URL을 확보하십시오

</div>

## 공급자 추가하기

1. **관리** 영역으로 이동합니다.
2. **외부 공급자** 페이지를 엽니다.
3. **공급자 추가**를 클릭합니다 (또는 새 연결을 추가하는 해당 버튼을 클릭합니다).
4. 공급자의 **피드 URL**을 입력합니다. 이것은 공급자가 Open Lesson Format으로 커리큘럼 데이터를 게시하는 URL입니다.
5. 연결을 저장합니다.

저장하면 Lessons.church가 공급자의 콘텐츠를 가져옵니다. 해당 프로그램, 학습 과정, 공과가 자체 콘텐츠와 함께 커리큘럼 라이브러리에 표시됩니다.

## 공급자 관리하기

**외부 공급자** 페이지에서 연결된 모든 공급자를 확인할 수 있습니다. 각 공급자에 대해 다음을 수행할 수 있습니다:

- **보기** -- 가져온 콘텐츠가 올바르게 반영되었는지 확인합니다.
- **업데이트** -- 공급자가 엔드포인트를 변경한 경우 피드 URL을 업데이트합니다.
- **제거** -- 해당 공급자의 콘텐츠를 더 이상 포함하지 않으려면 공급자를 제거합니다.

## Open Lesson Format이란?

Open Lesson Format은 커리큘럼 콘텐츠 -- 프로그램, 학습 과정, 공과, 장소, 섹션, 활동 -- 를 구조화된 방식으로 설명하는 표준화된 스키마입니다. 이 형식으로 콘텐츠를 게시하는 모든 조직은 Lessons.church에서 공급자로 추가할 수 있습니다.

Lessons.church를 통해 커리큘럼을 제공하는 데 관심이 있는 콘텐츠 공급자라면, [Open Lesson Schema](https://github.com/LiveChurchSolutions/LessonsScreen) 저장소에서 전체 스키마 문서를 확인할 수 있습니다.

## 외부 공급자를 사용하는 이유

- **교회에 더 많은 선택지** -- 교회들이 플랫폼을 떠나지 않고 더 다양한 커리큘럼을 탐색할 수 있습니다.
- **모든 것을 한 곳에서** -- 봉사자와 코디네이터가 공과를 찾고 일정을 잡기 위해 여러 웹사이트를 방문할 필요가 없습니다.
- **간편한 설정** -- 공급자를 추가하는 데 피드 URL만 있으면 됩니다. 수동 데이터 입력이나 파일 업로드가 필요하지 않습니다.

:::info
외부 콘텐츠는 외부 공급자가 관리합니다. 공급자의 콘텐츠가 누락되거나 오래된 경우, 해당 문제는 공급자 측에서 해결해야 할 수 있습니다.
:::

:::tip
교회 수준의 외부 공급자 설정(관리자 수준이 아닌)에 대해서는 탐색 섹션의 [외부 공급자](../browsing/external-providers.md) 페이지를 참조하십시오.
:::
