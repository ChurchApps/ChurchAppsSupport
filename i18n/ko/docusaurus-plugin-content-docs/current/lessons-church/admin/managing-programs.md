# 프로그램 관리

<div class="article-intro">

프로그램은 Lessons.church에서 커리큘럼의 최상위 컨테이너입니다. 프로그램은 전체 커리큘럼 시리즈를 나타냅니다 -- 예를 들어, "Ark Kids" 또는 "Youth Foundations"와 같습니다. 이 가이드는 관리 영역에서 프로그램을 생성하고, 편집하고, 정리하는 방법을 다룹니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- Lessons.church에 대한 관리자 권한을 보유하고 있어야 합니다 ([관리 개요](./index.md) 참조)
- 프로그램 이름, 설명, 썸네일 이미지를 준비하십시오

</div>

## 프로그램 만들기

1. **관리** 영역으로 이동합니다.
2. **프로그램 추가**를 클릭합니다.
3. 다음 항목을 입력합니다:
   - **이름** -- 프로그램의 표시 이름입니다 (예: "Ark Kids").
   - **슬러그** -- URL에 사용되는 식별자입니다 (예: "ark-kids"). 웹 주소에 사용됩니다.
   - **설명** -- 프로그램이 다루는 내용에 대한 간단한 요약입니다.
   - **이미지** -- 프로그램을 나타내는 썸네일 이미지를 업로드합니다.
4. 프로그램을 저장합니다.

새 프로그램이 관리 프로그램 목록에 나타나며, 학습 과정과 공과가 추가되면 공개 사이트에도 표시됩니다.

## 프로그램 편집하기

1. 관리 목록에서 프로그램 이름을 클릭합니다.
2. 항목(**이름**, **슬러그**, **설명**, 또는 **이미지**)을 수정합니다.
3. 변경 사항을 저장합니다.

## 프로그램 내 학습 과정 정리하기

프로그램이 존재하면 학습 과정을 추가할 수 있습니다. 학습 과정은 관리 화면에서 프로그램 아래에 중첩되어 표시됩니다. 프로그램 옆의 확장 화살표를 클릭하면 해당 학습 과정을 볼 수 있으며, 새 학습 과정을 추가할 수도 있습니다. 자세한 내용은 [학습 과정 관리](./managing-studies.md)를 참조하십시오.

## 학습 과정 카테고리

프로그램에 학습 과정이 많은 경우, **카테고리**로 정리할 수 있습니다. 카테고리는 주제, 연령대 또는 기타 기준에 따라 학습 과정을 그룹화합니다. 이를 통해 사용자가 규모가 큰 프로그램을 쉽게 탐색할 수 있습니다.

## 프로그램 수준 자료 업로드하기

프로그램에 다운로드 가능한 파일을 첨부할 수 있습니다. 이는 특정 학습 과정이나 공과가 아닌 프로그램 전체에 적용되는 자료입니다 -- 예를 들어, 범위와 순서 문서 또는 봉사자 핸드북 등입니다.

1. 관리 화면에서 프로그램을 엽니다.
2. **자료** 섹션으로 이동합니다.
3. 파일을 업로드합니다.

## 캐시 지우기

프로그램을 변경한 후 공개 사이트에 업데이트가 반영되도록 캐시를 지워야 할 수 있습니다. 관리 영역의 **캐시 지우기** 옵션을 사용하여 콘텐츠를 새로고침하십시오.

:::tip
설명이 잘 되는 프로그램 이름을 선택하고 명확한 설명을 작성하십시오. 이것은 교회들이 커리큘럼 라이브러리를 탐색할 때 처음 보게 되는 것이므로, 좋은 요약은 프로그램이 자신들의 필요에 적합한지 판단하는 데 도움이 됩니다.
:::
