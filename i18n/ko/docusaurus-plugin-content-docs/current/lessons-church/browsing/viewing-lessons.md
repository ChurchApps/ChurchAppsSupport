# 공과 보기

<div class="article-intro">

Lessons.church의 각 공과는 구조화된 콘텐츠, 다운로드 가능한 자료, 여러 연령대 지원이 포함된 완전한 교육 세션입니다. 이 가이드는 수업 준비를 위해 공과 페이지를 탐색하고 활용하는 방법을 설명합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [프로그램과 학습 과정](./programs-and-studies.md)을 읽어 콘텐츠가 어떻게 구성되어 있는지 이해하십시오
- 보고 싶은 공과를 찾기 위해 프로그램과 학습 과정을 탐색하십시오

</div>

## 공과에 접근하기

공과를 보려면 콘텐츠 계층 구조를 통해 이동합니다:

1. 홈 페이지에서 **프로그램**을 클릭합니다.
2. 프로그램 페이지에서 **학습 과정**을 선택합니다.
3. 학습 과정 페이지에서 보고 싶은 **공과**를 클릭합니다.

## 표시되는 내용

공과 페이지는 여러 영역으로 구성됩니다:

### 영상

공과에 영상이 포함된 경우 페이지 상단에 표시됩니다. 클릭하면 브라우저에서 바로 재생됩니다. 영상은 특히 어린이 사역에서 주요 교육 부분으로 자주 사용됩니다.

### 섹션과 역할

각 공과는 **섹션**(예: "시작", "성경 이야기", "활동", "마무리")으로 나뉩니다. 각 섹션 내에서 콘텐츠는 **역할**별로 정리됩니다:

- **리더** -- 주요 교사 또는 발표자
- **도우미** -- 보조 봉사자
- 커리큘럼 제작자가 정의한 기타 맞춤 역할

각 역할에는 봉사자가 정확히 무엇을 해야 하는지 알려주는 **활동**이 나열됩니다:

- **말하기** -- 소리 내어 읽을 대본이나 요점
- **행동하기** -- 수행할 신체 활동이나 지시 사항
- **참고** -- 봉사자를 위한 배경 정보나 팁
- **재생** -- 영상이나 오디오 클립 재생 안내

### 다운로드 가능한 자료

공과에는 다운로드하여 인쇄할 수 있는 파일이 포함될 수 있습니다:

- PDF 유인물
- 활동지
- 색칠 공부
- 공작 안내서
- 학부모용 가정 학습지

이러한 자료에 접근하려면 공과 페이지 내의 다운로드 링크를 찾으십시오. 인쇄 자료 준비에 대한 자세한 내용은 [다운로드 및 자료](../presenting/downloads.md)를 참조하십시오.

## 장소 간 전환하기

많은 공과가 유아부, 초등부, 중등부와 같은 다양한 연령대를 나타내는 여러 **장소**를 지원합니다. 장소를 통해 같은 공과 주제를 각 그룹에 적절한 수준으로 조정할 수 있습니다.

장소를 전환하려면:

1. 공과 페이지에서 **장소 선택기**를 찾습니다.
2. 다른 장소 이름을 클릭합니다 (예: **초등부**에서 **유아부**로 전환).
3. 해당 연령대에 맞게 설계된 콘텐츠, 대본, 자료가 페이지에 표시됩니다.

:::tip
여러 연령대를 가르치는 경우, 수업 전에 각 장소를 검토하여 콘텐츠가 다양한 수준에 어떻게 적용되는지 확인하십시오.
:::
