# 콘텐츠 탐색

<div class="article-intro">

Lessons.church는 누구나 탐색할 수 있는 무료 교회 커리큘럼 라이브러리를 제공합니다. 이용 가능한 콘텐츠를 탐색하기 위해 계정이나 로그인이 필요하지 않습니다 -- 사이트를 방문하여 바로 둘러보시면 됩니다. 이 섹션에서는 커리큘럼을 찾고 탐색하는 모든 방법을 다룹니다.

</div>

## 홈 페이지

Lessons.church에 처음 방문하면, 홈 페이지에서 이용 가능한 모든 콘텐츠의 간략한 개요를 확인할 수 있습니다:

- **추천 프로그램** -- 인기 있거나 최근 업데이트된 콘텐츠를 쉽게 발견할 수 있도록 강조된 커리큘럼 시리즈가 상단에 표시됩니다.
- **전체 프로그램** -- 추천 섹션 아래에는 각 프로그램의 공과 수와 학습 과정 수와 같은 통계와 함께 이용 가능한 모든 커리큘럼 프로그램의 전체 목록이 있습니다.
- **공급자 콘텐츠** -- Lessons.church와 외부 커리큘럼 공급자의 콘텐츠가 함께 표시되어 한 곳에서 탐색할 수 있습니다.

## 콘텐츠를 찾는 방법

커리큘럼 라이브러리를 탐색하는 여러 가지 방법이 있습니다:

1. **프로그램별 탐색** -- 홈 페이지에서 프로그램을 클릭하면 해당 학습 과정과 공과를 확인할 수 있습니다. 특정 커리큘럼 시리즈를 살펴보고 싶을 때 가장 좋은 방법입니다. 자세한 내용은 [프로그램과 학습 과정](./programs-and-studies.md)을 참조하십시오.

2. **검색** -- 검색 페이지를 사용하여 전체 라이브러리에서 키워드로 공과를 찾을 수 있습니다. 주제는 알고 있지만 어떤 프로그램에서 다루는지 확실하지 않을 때 유용합니다. 자세한 내용은 [검색](./searching.md)을 참조하십시오.

3. **외부 공급자 탐색** -- 일부 콘텐츠는 Open Lesson Format으로 게시하는 외부 커리큘럼 공급자에서 제공됩니다. 내장 콘텐츠와 동일하게 해당 프로그램과 학습 과정을 탐색할 수 있습니다. 자세한 내용은 [외부 공급자](./external-providers.md)를 참조하십시오.

## 로그인 불필요

탐색은 완전히 개방되어 있습니다. 계정을 만들지 않고도 프로그램, 학습 과정, 공과, 다운로드 자료, 영상을 모두 볼 수 있습니다. 교실에 공과 일정을 잡거나 콘텐츠를 맞춤 설정하려면 그 시점에 [무료 계정을 생성](../getting-started/creating-account.md)할 수 있습니다.

:::tip
특정 학습 과정에 들어가기 전에 홈 페이지에서 몇 가지 프로그램을 살펴보며 커리큘럼이 어떻게 구성되어 있는지 감을 잡으십시오.
:::
