# 외부 공급자

<div class="article-intro">

Lessons.church는 자체 커리큘럼에만 국한되지 않습니다. 외부 커리큘럼 공급자가 **Open Lesson Format**을 통해 콘텐츠를 게시할 수 있으며, 해당 콘텐츠는 Lessons.church의 내장 프로그램과 함께 표시되어 통합된 탐색 경험을 제공합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [프로그램과 학습 과정](./programs-and-studies.md)을 읽어 콘텐츠가 어떻게 구성되어 있는지 이해하십시오
- 공급자 연결을 추가하려면 교회 포털에 접근할 수 있는 [Lessons.church 계정](../getting-started/creating-account.md)이 필요합니다

</div>

## 외부 공급자란?

외부 공급자는 Open Lesson Format -- 플랫폼 간에 공과 콘텐츠를 공유할 수 있는 표준 -- 을 사용하여 교회 커리큘럼을 게시하는 모든 조직입니다. 공급자가 이 형식으로 콘텐츠를 제공하면, Lessons.church가 해당 콘텐츠를 가져와서 자체 프로그램과 동일하게 표시할 수 있습니다.

이를 통해 여러 웹사이트를 방문하지 않고도 더 다양한 커리큘럼에 접근할 수 있습니다.

## 외부 콘텐츠 탐색하기

외부 공급자 콘텐츠는 내장 콘텐츠와 정확히 동일하게 작동합니다:

1. 외부 공급자의 프로그램이 다른 프로그램과 함께 Lessons.church 홈 페이지에 표시됩니다.
2. **프로그램**을 클릭하면 해당 학습 과정을 볼 수 있습니다.
3. **학습 과정**을 클릭하면 해당 공과를 볼 수 있습니다.
4. 동일한 레이아웃 -- 섹션, 역할, 활동, 영상, 다운로드 가능한 자료 -- 으로 공과를 볼 수 있습니다.

외부 콘텐츠를 위한 특별한 과정이나 별도의 섹션은 없습니다. 모든 것이 하나의 라이브러리로 통합되어 쉽게 탐색할 수 있습니다.

## 공급자 연결 추가하기

교회에서 Open Lesson Format을 지원하는 커리큘럼 공급자를 사용하고 있으며 해당 콘텐츠를 Lessons.church 포털에서 이용하고 싶은 경우:

1. Lessons.church 계정에 로그인합니다.
2. 포털에서 **외부 공급자** 페이지로 이동합니다.
3. 외부 공급자의 연결 URL을 추가합니다.
4. 해당 프로그램과 학습 과정을 교회에서 탐색하고, 일정을 잡고, 교실에서 사용할 수 있게 됩니다.

:::info
외부 공급자 연결을 추가하는 것은 Lessons.church 포털을 통해 이루어지며 로그인된 계정이 필요합니다. 플랫폼에 이미 추가된 콘텐츠를 탐색하는 것은 계정이 필요하지 않습니다.
:::

## 콘텐츠 제작자를 위한 안내

Lessons.church에서 콘텐츠를 제공하는 데 관심이 있는 커리큘럼 출판사라면, Open Lesson Format은 프로그램, 학습 과정, 공과를 공유하는 표준화된 방법을 제공합니다. 통합에 대한 자세한 내용은 Lessons.church 팀에 문의하거나 개발자 문서를 방문하십시오.

:::tip
플랫폼 수준에서 외부 공급자 피드를 관리하는 관리자인 경우, 관리 섹션의 [외부 공급자](../admin/third-party-providers.md)를 참조하십시오.
:::
