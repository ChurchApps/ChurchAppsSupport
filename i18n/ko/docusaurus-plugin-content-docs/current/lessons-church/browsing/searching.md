# 검색

<div class="article-intro">

가르치고 싶은 주제는 알고 있지만 어떤 프로그램이나 학습 과정에 포함되어 있는지 확실하지 않을 때, 검색 기능을 사용하면 적합한 콘텐츠를 빠르게 찾을 수 있습니다. 검색은 외부 공급자의 콘텐츠를 포함하여 전체 Lessons.church 라이브러리에서 작동합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 이용 가능한 모든 탐색 옵션을 이해하기 위해 [콘텐츠 탐색](./index.md) 개요를 숙지하십시오

</div>

## 검색하는 방법

1. Lessons.church 사이트 상단에서 **검색 페이지**로 이동합니다.
2. **검색 상자**에 키워드나 구문을 입력합니다 (예: "용서", "다윗과 골리앗", "기도").
3. **Enter** 키를 누르거나 검색 버튼을 클릭하여 결과를 확인합니다.

## 검색 작동 방식

검색은 전체 Lessons.church 라이브러리에 대한 전문 검색으로 구동됩니다. 공과 제목, 설명, 내용을 검색하여 키워드에 맞는 결과를 찾습니다. 결과에는 모든 이용 가능한 프로그램과 공급자의 공과가 포함되므로 특정 주제에 대해 이용 가능한 내용을 종합적으로 확인할 수 있습니다.

## 결과 탐색하기

검색 결과에는 일치하는 공과와 함께 각 공과가 속한 프로그램과 학습 과정의 정보가 표시됩니다. 결과를 클릭하면 해당 공과 페이지로 바로 이동하며, 전체 콘텐츠, 섹션, 역할, 다운로드 가능한 자료를 확인할 수 있습니다.

## 검색을 사용할 때

검색은 다음과 같은 경우에 가장 유용합니다:

- 특정 성경 이야기나 주제를 염두에 두고 있으며 이를 다루는 모든 이용 가능한 커리큘럼을 보고 싶을 때.
- 특별 시리즈를 계획하고 있으며 특정 주제에 대한 공과를 찾아야 할 때.
- 아직 이용 가능한 프로그램에 익숙하지 않으며 제공되는 내용을 빠르게 탐색하고 싶을 때.

:::tip
키워드로 너무 많은 결과가 나오면 더 구체적인 구문을 사용해 보십시오. 예를 들어, "사마리아인"만 검색하는 대신 "선한 사마리아인"으로 검색하면 결과를 좁힐 수 있습니다.
:::

이미 원하는 프로그램을 알고 있다면 프로그램별 탐색이 보통 더 빠릅니다. 콘텐츠 계층 구조 탐색에 대한 자세한 내용은 [프로그램과 학습 과정](./programs-and-studies.md)을 참조하십시오.
