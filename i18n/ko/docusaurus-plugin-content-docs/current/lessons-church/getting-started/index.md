# 시작하기

<div class="article-intro">

Lessons.church는 완전히 무료로 사용할 수 있습니다. 구독료, 체험 기간, 기능 제한이 없습니다. 이 섹션에서는 계정을 만들고, 교회를 연결하고, 포털을 탐색하는 과정을 안내하여 바로 공과 일정을 잡을 수 있도록 합니다.

</div>

## 필요한 것들

Lessons.church를 시작하기 위해 다음을 수행합니다:

1. **계정 만들기** -- [Lessons.church](https://lessons.church)에서 이름, 이메일, 비밀번호로 등록합니다. 이것은 B1.church에서 이미 사용하고 계실 수 있는 ChurchApps 계정과 동일한 계정을 사용합니다.
2. **교회 연결하기** -- 등록 후 기존 교회 프로필에 연결하거나 새 교회 프로필을 생성합니다. 이를 통해 교실과 일정이 교회에 연결됩니다.
3. **교실 설정하기** -- 사역에 맞는 교실을 만듭니다 (예: "유아부", "초등 1-3학년", "청소년부"). 교실은 공과를 배정하고 정리하는 곳입니다.
4. **공과 일정 잡기 시작하기** -- 콘텐츠 라이브러리를 탐색하여 필요에 맞는 학습 과정을 찾고 교실에 공과를 배정합니다.

교실이 설정되고 공과가 배정되면, 봉사자들이 포털을 통해 콘텐츠에 접근하고 수업에서 공과를 진행할 수 있습니다.

:::tip
B1.church에서 이미 ChurchApps 계정을 가지고 있다면, 같은 자격 증명으로 Lessons.church에 로그인할 수 있습니다 -- 다시 등록할 필요가 없습니다.
:::
