# 계정 만들기

<div class="article-intro">

Lessons.church는 B1.church와 동일한 ChurchApps 계정 시스템을 사용합니다. 이미 ChurchApps 계정이 있으면 바로 로그인할 수 있습니다. 그렇지 않은 경우, 등록은 몇 분 안에 완료되며 전체 커리큘럼 라이브러리에 접근할 수 있게 됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 계정 인증을 위한 유효한 이메일 주소를 준비하십시오
- 교회에서 이미 Lessons.church를 사용하고 있다면, 새 교회 프로필을 만들기 전에 관리자에게 자세한 내용을 문의하십시오

</div>

## 새 계정 등록하기

1. 웹 브라우저에서 [Lessons.church](https://lessons.church)로 이동합니다.
2. 상단 탐색 바에서 **로그인**을 클릭합니다.
3. 로그인 페이지에서 **등록**을 클릭하여 새 계정을 만듭니다.
4. **이름**, **성**, **이메일 주소**를 입력합니다.
5. 계정의 **비밀번호**를 만듭니다.
6. 등록 양식을 제출합니다.

등록 후 기존 교회에 연결하거나 새 교회 프로필을 만들라는 안내가 표시됩니다.

## 교회 연결하기

**새 교회를 만드는 경우:**

1. **새 교회 만들기** 옵션을 선택합니다.
2. **교회 이름** 및 기본 정보를 입력합니다.
3. 설정을 완료합니다. Lessons.church에서 교회의 관리자가 됩니다.

**기존 교회에 참여하는 경우:**

1. 교회 이름으로 검색합니다.
2. 결과에서 교회를 선택합니다.
3. 교회 관리자가 적절한 권한을 부여해야 할 수 있습니다.

:::tip
교회에서 이미 Lessons.church를 설정한 사람이 있다면, 교실과 일정을 관리할 수 있는 적절한 권한이 있는지 확인해 달라고 요청하십시오.
:::

## 기존 계정으로 로그인하기

B1.church 또는 다른 ChurchApps 도구에서 이미 ChurchApps 계정이 있는 경우:

1. [Lessons.church](https://lessons.church)로 이동하여 **로그인**을 클릭합니다.
2. 기존 **이메일**과 **비밀번호**를 입력합니다.
3. 교실과 일정을 관리할 수 있는 포털로 이동됩니다.

:::info
ChurchApps 계정은 모든 ChurchApps 제품에서 사용할 수 있습니다. 하나의 계정으로 B1 Admin, B1.church, Lessons.church 등에 접근할 수 있습니다.
:::
