# 맞춤 설정

<div class="article-intro">

Lessons.church는 바로 사용할 수 있는 공과 콘텐츠 라이브러리를 제공하지만, 모든 교회는 다릅니다. 맞춤 설정 기능을 통해 공과 내용을 특정 교실, 교수 방식, 연령대에 맞게 조정하여 봉사자들이 자신에게 관련된 내용만 볼 수 있도록 할 수 있습니다.

</div>

## 왜 맞춤 설정을 하는가?

Lessons.church의 공과 콘텐츠는 유연하게 설계되어 있습니다. 하나의 공과에 여러 연령대를 위한 섹션, 선택적 활동, 모든 교실에 해당하지 않는 역할이 포함될 수 있습니다. 봉사자들에게 해당하지 않는 부분을 머릿속으로 건너뛰도록 요청하는 대신, 공과를 맞춤 설정하여 관련된 내용만 보이도록 할 수 있습니다.

예를 들어, 공과에 유아부에는 적합하지만 5학년 교실에는 맞지 않는 공작 활동이 포함될 수 있습니다. 맞춤 설정을 통해 어린 그룹에는 해당 섹션을 유지하면서 나이가 많은 그룹에서는 숨길 수 있습니다.

## 맞춤 설정할 수 있는 항목

- **섹션 제거** -- 특정 교실이나 연령대에 해당하지 않는 전체 섹션을 숨깁니다.
- **역할 또는 활동 숨기기** -- 필요하지 않은 섹션 내의 개별 역할이나 활동을 제거합니다.
- **콘텐츠 순서 변경** -- 교실에서 선호하는 흐름에 맞게 섹션과 활동의 순서를 재배치합니다.

## 교실별 또는 전체 적용

맞춤 설정을 저장할 때 적용 범위를 선택합니다:

- **단일 교실** -- 맞춤 설정이 현재 교실에만 적용됩니다. 다른 교실에서는 원래 콘텐츠가 표시됩니다.
- **모든 교실** -- 맞춤 설정이 교회의 모든 교실에 적용됩니다.

이러한 유연성을 통해 같은 공과에 대해 연령대별로 다른 보기를 설정하거나, 모든 교실에 한꺼번에 변경 사항을 적용할 수 있습니다.

:::tip
맞춤 설정이 처음이라면, 하나의 교실에서 먼저 시도하여 어떻게 작동하는지 확인한 후 모든 교실에 변경 사항을 적용하십시오.
:::
