# 공과 맞춤 설정

<div class="article-intro">

이 가이드는 교실에 맞게 공과를 맞춤 설정하는 과정을 안내합니다. 맞춤 설정을 통해 콘텐츠를 제거하거나, 숨기거나, 순서를 변경하여 봉사자들이 필요한 내용만 볼 수 있도록 합니다 -- 그 이상도 그 이하도 아닙니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 포털에서 [최소 하나의 교실을 설정](../classrooms/setting-up-classrooms.md)하십시오
- 맞춤 설정하려는 교실에 [공과를 배정](../classrooms/scheduling-lessons.md)하십시오
- 맞춤 설정할 수 있는 항목을 이해하기 위해 [맞춤 설정 개요](./index.md)를 읽으십시오

</div>

## 맞춤 설정 화면 열기

1. 로그인하여 **포털**로 이동합니다.
2. 맞춤 설정하려는 공과가 있는 **교실**을 선택합니다.
3. **배정된 공과**를 클릭하여 엽니다.
4. **맞춤 설정**을 클릭하여 맞춤 설정 화면으로 들어갑니다.

전체 공과 내용이 섹션, 역할, 활동으로 분류되어 표시됩니다.

## 섹션 제거하기

교실에 해당하지 않는 섹션이 있으면 전체를 제거할 수 있습니다. 이는 공과에 여러 연령대의 콘텐츠가 포함되어 있고 하나만 필요한 경우에 유용합니다. 예를 들어, 공과에 "유아부 활동"과 "초등부 활동" 섹션이 모두 있으면, 교실에 맞지 않는 것을 제거할 수 있습니다.

1. 제거하려는 섹션을 찾습니다.
2. 섹션을 **제거** 또는 **숨기기**합니다.
3. 이 공과를 보는 봉사자에게 해당 섹션이 더 이상 표시되지 않습니다.

## 역할 또는 활동 숨기기

섹션 내에서 특정 역할이나 활동을 숨길 수 있습니다. 예를 들어, 섹션에 "찬양 인도자" 역할이 포함되어 있지만 교실에 해당 역할이 없는 경우, 전체 섹션을 제거하지 않고 해당 역할만 숨길 수 있습니다.

1. 섹션을 펼쳐 역할과 활동을 확인합니다.
2. 필요하지 않은 역할이나 활동을 **숨기기**합니다.

## 콘텐츠 순서 변경하기

선호하는 흐름에 맞게 섹션과 활동의 순서를 재배치할 수 있습니다:

1. 섹션이나 활동을 드래그하여 위아래로 이동합니다.
2. 새로운 순서가 봉사자들이 공과를 볼 때 표시됩니다.

## 맞춤 설정 저장하기

변경이 완료되면 맞춤 설정을 저장합니다. 적용 범위를 선택하라는 메시지가 표시됩니다:

- **이 교실만** -- 변경 사항이 현재 교실에만 적용됩니다. 다른 교실은 영향을 받지 않습니다.
- **모든 교실** -- 변경 사항이 교회의 모든 교실에 적용됩니다.
- **맞춤 설정 제거** -- 이전에 이 공과를 맞춤 설정했고 원래 콘텐츠로 돌아가고 싶은 경우, 이 옵션을 선택하여 변경 사항을 지웁니다.

:::tip
하나의 교실에서 하나의 공과를 맞춤 설정하는 것부터 시작하십시오. 과정에 익숙해지면 모든 교실에 더 광범위하게 맞춤 설정을 적용할 수 있습니다.
:::

## 맞춤 설정이 필요한 경우

맞춤 설정은 다음과 같은 경우에 가장 유용합니다:

- 공과에 여러 연령대의 콘텐츠가 있으며 관련 부분만 표시하고 싶을 때.
- 교실에서 특정 역할(전담 찬양 인도자나 기술 봉사자 등)을 사용하지 않을 때.
- 기본으로 제공되는 것과 다른 공과 흐름 순서를 선호할 때.

:::info
모든 공과를 맞춤 설정할 필요는 없습니다. 콘텐츠가 교실에 그대로 적합하다면, 이 단계를 건너뛰고 바로 [수업 진행](../presenting/index.md)으로 이동할 수 있습니다.
:::
