# 발표자 모드

<div class="article-intro">

발표자 모드는 공과를 전체 화면의 프로젝션 친화적 디스플레이로 변환합니다. 수업 시간에 교실의 화면, 프로젝터 또는 TV에 공과 내용을 표시하여 교실의 모든 사람이 함께 따라갈 수 있도록 설계되었습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교실에 [공과를 배정](../classrooms/scheduling-lessons.md)하거나, [콘텐츠 라이브러리](../browsing/index.md)를 통해 공과로 이동하십시오
- 디스플레이(프로젝터, TV 또는 모니터)가 연결되고 설정되었는지 확인하십시오

</div>

## 발표자 모드 시작하기

1. 진행하려는 공과를 엽니다.
2. 공과 페이지에서 **발표** 버튼을 클릭합니다.
3. 공과가 검은 배경과 크고 읽기 쉬운 텍스트가 있는 전체 화면 보기로 전환됩니다.

발표자 모드에 들어가면, 화면의 탐색 컨트롤을 사용하여 공과를 섹션별로 진행할 수 있습니다.

## 화면 모습

발표자 모드는 투사된 화면에서 콘텐츠가 돋보이도록 깔끔한 어두운 배경을 사용합니다. 공과의 각 섹션이 한 번에 하나씩 표시되어 현재 활동에 집중할 수 있습니다. 텍스트, 이미지, 영상 삽입은 교실 반대편에서도 읽을 수 있도록 형식이 지정됩니다.

## FreeShow와 함께 사용하기

Lessons.church는 무료 프레젠테이션 애플리케이션인 [FreeShow](https://freeshow.app)와 통합됩니다. FreeShow는 Lessons.church에서 직접 공과 콘텐츠를 가져올 수 있어 기존 프레젠테이션 워크플로우에 공과 슬라이드를 통합할 수 있습니다. 이미 찬양 가사나 공지 사항에 FreeShow를 사용하고 있다면 특히 유용합니다.

## Venue TV 앱과 함께 사용하기

**Venue TV** Android 앱을 사용하면 별도의 컴퓨터 없이 TV에 공과를 표시할 수 있습니다. Android TV 기기에 앱을 설치하고 Lessons.church 교실에 연결하면, 배정된 공과가 자동으로 표시됩니다. 전용 화면이 있는 교실에 적합한 핸즈프리 옵션입니다.

:::tip
프로젝터를 사용하는 경우, 발표자 모드에 들어가기 전에 디스플레이가 프로젝터에 미러링 또는 확장되도록 설정되어 있는지 확인하십시오. 이렇게 하면 전체 화면 보기가 올바른 화면에 표시됩니다.
:::

## 발표자 모드 종료하기

발표자 모드를 종료하려면 **Escape** 키를 누르거나 화면 모서리의 종료 버튼을 클릭합니다. 일반 공과 보기로 돌아갑니다.
