# 봉사자 가이드

<div class="article-intro">

Lessons.church의 모든 공과에는 특정 봉사자 역할에 맞춘 가이드가 포함되어 있습니다. 각 봉사자에게 전체 공과 계획서를 건네는 대신, 해당 역할에 관련된 안내만 표시하는 가이드를 안내하여 간결하고 집중된 정보를 제공할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [공과 보기](../browsing/viewing-lessons.md)를 읽어 공과 구조를 이해하십시오
- 가이드를 준비하려는 공과로 이동하십시오

</div>

## 가이드 구성 방식

공과는 **섹션**(예: "환영", "성경 이야기", "소그룹 활동")으로 나뉩니다. 각 섹션 내에서 **역할**에 특정 **활동**이 지정됩니다. 네 가지 활동 유형이 있습니다:

- **말하기** -- 봉사자가 읽거나 의역할 대본 텍스트입니다.
- **행동하기** -- 수행할 신체 활동이나 지시 사항입니다.
- **참고** -- 봉사자를 위한 배경 정보나 알림입니다 (소리 내어 말하지 않습니다).
- **재생** -- 반에서 재생할 영상 또는 오디오 클립입니다.

봉사자가 가이드를 열면, 각 섹션이 순서대로 표시되며 해당 역할에 지정된 활동만 보입니다. 이를 통해 간결하고 집중된 정보를 제공합니다.

## 봉사자 가이드 보기

1. 준비하려는 공과를 엽니다.
2. 공과의 장소에 연결된 **역할** 목록을 찾습니다.
3. 역할 이름을 클릭하면 해당 역할의 가이드가 열립니다.

가이드는 각 섹션을 단계별 안내로 표시합니다. 봉사자들은 수업 중에 휴대폰, 태블릿 또는 컴퓨터로 스크롤하며 확인할 수 있습니다.

## 가이드 인쇄하기

봉사자가 종이 사본을 선호하는 경우, 브라우저에서 직접 가이드를 인쇄할 수 있습니다.

1. 원하는 역할의 가이드를 엽니다.
2. 브라우저의 **인쇄** 기능을 사용합니다 (**Ctrl+P** 또는 **Cmd+P**).
3. 페이지를 인쇄하거나 나중에 배포하기 위해 PDF로 저장합니다.

:::tip
봉사자들이 수업 전에 안내를 검토할 수 있도록 가이드 링크를 미리 공유하십시오. 주중에 간단히 읽어두면 주일 아침에 모두가 더 잘 준비될 수 있습니다.
:::

## 다중 역할

일부 공과에는 여러 역할이 있습니다 -- 예를 들어, 대표 교사, 소그룹 리더, 기술 봉사자 등입니다. 각 역할에는 자체 가이드가 있으므로 공과에서 요구하는 만큼 전문화된 가이드를 가질 수 있습니다.

:::info
교실에 해당하지 않는 역할이 있으면, [맞춤 설정](../customization/customizing-lessons.md) 기능을 사용하여 숨길 수 있습니다. 이를 통해 봉사자들이 관련 있는 역할만 볼 수 있습니다.
:::
