# 교재를 위한 교사 보기

<div class="article-intro">
교사 보기는 구조화된 섹션이 있는 교재 콘텐츠를 보기 위한 확장된 전체 화면 인터페이스를 제공하여 교사와 리더가 수업 시간 동안 교재를 더 쉽게 탐색할 수 있도록 합니다.
</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교회는 계획에 예정된 교재가 있어야 합니다. 자세한 내용은 [교재 일정 예약](../../b1-admin/serving/scheduling-lessons.md)에 대한 B1 Admin 가이드를 참조하세요.
- 계획에 할당되거나 B1.church 사이트에서 볼 수 있는 액세스 권한이 있어야 합니다.
</div>

## 교사 보기 열기

교재의 확장된 교사 보기에 액세스하려면:

1. B1.church의 **계획** 섹션으로 이동합니다
2. 가르치려는 교재가 포함된 계획을 엽니다
3. 계획 항목에서 교재를 찾습니다
4. 교재를 클릭하여 세부 정보를 봅니다
5. **교사 보기** 또는 **확장 보기** 버튼을 클릭합니다

교재는 가르치기에 최적화된 전체 화면 인터페이스로 열립니다.

:::tip
교사 보기는 수업이 볼 수 있도록 화면이나 TV에 교재 콘텐츠를 투영할 때 또는 태블릿에서 가르칠 때 특히 유용합니다.
:::

## 교사 보기 기능

### 탭 탐색

교사 보기는 상단의 탭을 사용하여 교재 콘텐츠를 섹션으로 구성합니다.

- 교재의 각 주요 섹션이 별도의 탭으로 나타납니다
- 탭을 클릭하여 해당 섹션으로 직접 이동합니다
- 활성 섹션이 탭 표시줄에서 강조 표시됩니다
- 탭을 사용하면 교재의 다른 부분으로 쉽게 건너뛸 수 있습니다

### 스크롤 가능한 콘텐츠

기본 콘텐츠 영역에 전체 교재가 표시됩니다.

- 교재를 자연스럽게 스크롤합니다
- 스크롤하면 활성 탭이 보고 있는 섹션과 일치하도록 자동으로 업데이트됩니다
- 모든 포맷된 텍스트, 이미지 및 지침이 명확하게 표시됩니다

### 미디어 재생

교재에 비디오, 이미지 또는 기타 미디어가 포함된 경우:

- 미디어 항목을 클릭하여 팝업 플레이어에서 엽니다
- 비디오는 표준 컨트롤이 있는 전용 비디오 플레이어에서 재생됩니다
- 이미지는 전체 화면 보기를 위해 라이트박스에서 열립니다
- 미디어 플레이어를 닫으면 교재 콘텐츠로 돌아갑니다

### 다운로드 가능한 리소스

교재에 다운로드 가능한 파일(활동 시트, 색칠 페이지 등)이 포함된 경우:

- 각 리소스에 대한 다운로드 버튼이 나타납니다
- 클릭하여 장치에 직접 파일을 다운로드합니다
- 파일에는 콘텐츠 제작자가 제공한 PDF, 이미지 또는 기타 자료가 포함될 수 있습니다

## 교실에서 교사 보기 사용

### 수업 전

1. 수업이 시작되기 전에 교사 보기에서 교재를 엽니다
2. 모든 섹션을 검토하고 콘텐츠에 익숙해집니다
3. 필요한 인쇄 가능한 리소스를 다운로드합니다
4. 비디오 또는 미디어를 테스트하여 올바르게 재생되는지 확인합니다

### 수업 중

1. 장치에 교사 보기를 열어 두거나 화면에 투영합니다
2. 탭을 사용하여 교재를 진행하면서 섹션 간에 이동합니다
3. 적절한 시간에 비디오를 재생하거나 이미지를 표시합니다
4. 위치를 잃지 않고 필요에 따라 지침을 스크롤합니다

### 투영된 화면에서

수업을 위해 투영할 때:

- 전체 화면 레이아웃이 주의를 분산시키는 요소를 제거합니다
- 큰 읽기 쉬운 텍스트로 교실 전체에서 콘텐츠를 볼 수 있습니다
- 미디어 항목을 클릭하여 전체 수업에 비디오 또는 이미지를 표시합니다
- 무선 마우스 또는 트랙패드를 사용하여 탐색합니다

## 교사 보기 닫기

교사 보기를 종료하려면:

- 상단 모서리의 **닫기** 버튼 (X)을 클릭합니다
- 또는 키보드에서 **Esc** 키를 누릅니다
- 계획 세부 정보 보기로 돌아갑니다

## 모바일 및 태블릿 사용

교사 보기는 태블릿 및 모바일 장치에서 작동합니다.

- **태블릿** -- 휴대용 교재 가이드를 원하는 교사에게 완벽합니다
- **휴대폰** -- 지원되지만 작은 화면은 가르치기에 이상적이지 않을 수 있습니다
- **가로 모드** -- 모바일 장치에서 최상의 보기 경험을 제공합니다

## 일반 교재 보기와의 차이점

| 기능 | 일반 보기 | 교사 보기 |
|---------|--------------|--------------|
| **레이아웃** | 계획과 함께 인라인으로 컴팩트 | 전체 화면, 전용 보기 |
| **탐색** | 스크롤만 | 자동 강조 표시가 있는 탭 + 스크롤 |
| **미디어** | 페이지에 포함됨 | 집중 보기를 위한 팝업 플레이어 |
| **화면 공간** | 다른 계획 항목 표시 | 교재 콘텐츠를 위해 최대화됨 |
| **최적 용도** | 탐색 및 계획 | 가르치기 및 발표 |

## 모범 사례

- **사전 준비** -- 레이아웃에 익숙해지기 위해 수업 전에 교사 보기에서 교재를 검토합니다
- **섹션 북마크** -- 주요 활동이나 토론이 포함된 탭을 메모합니다
- **미디어 테스트** -- 수업 시간 전에 비디오 및 이미지가 제대로 로드되는지 확인합니다
- **리소스 조기 다운로드** -- 수업 시간까지 기다리지 말고 인쇄물을 다운로드합니다
- **더 큰 화면 사용** -- 최상의 가르침 경험을 위해 투영하거나 태블릿을 사용합니다

## 문제 해결

### 교재가 교사 보기에서 열리지 않음

- 계획에 교재가 제대로 예정되어 있는지 확인합니다
- 계획을 볼 수 있는 권한이 있는지 확인합니다
- 페이지를 새로 고치고 다시 시도합니다

### 비디오가 재생되지 않음

- 인터넷 연결을 확인합니다
- 장치가 미디어 재생을 허용하는지 확인합니다
- 별도의 탭에서 비디오를 열어 봅니다
- 일부 콘텐츠 제공업체는 비디오 재생에 제한이 있을 수 있습니다

### 모바일에서 콘텐츠가 잘림

- 장치를 가로 모드로 회전합니다
- 필요한 경우 약간 축소합니다
- 더 나은 가시성을 위해 태블릿 또는 더 큰 장치를 사용하는 것을 고려합니다
