# 성경

<div class="article-intro">

B1.church는 사이트를 떠나지 않고도 성경을 읽을 수 있는 내장 성경 읽기 도구를 포함하고 있습니다. 읽기 도구가 페이지에 직접 포함되어 있어 다른 교회 도구와 함께 모든 구절에 빠르게 액세스할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 성경 읽기 도구를 사용하려면 로그인이 필요하지 않습니다. 성경 페이지는 공개적으로 액세스할 수 있습니다.
- 교회가 사이트 네비게이션에 성경 페이지를 추가했어야 합니다.

</div>

## 성경 읽기

1. 교회의 사이트 네비게이션에서 **성경** 페이지로 이동합니다.
2. 읽기 도구는 기본적으로 창세기 1:1로 열립니다.
3. 읽기 도구 내의 네비게이션 컨트롤을 사용하여 다른 책, 장 또는 절로 이동합니다.

성경 읽기 도구는 **YouVersion**으로 구동되며 많은 언어로 다양한 성경 번역에 액세스할 수 있습니다.

## 구절 탐색

포함된 읽기 도구는 성경을 통해 이동하기 위한 컨트롤을 제공합니다. 다음을 할 수 있습니다:

- 성경의 특정 책으로 이동
- 특정 장으로 이동
- 장 내의 절을 스크롤
- 버전 선택기를 사용하여 다른 성경 번역으로 전환

읽기 도구는 페이지의 콘텐츠 영역을 채워 편안하게 읽을 수 있는 충분한 공간을 제공합니다.

:::tip
선택한 책, 장 및 번역이 기억되므로 읽기 도구가 떠난 위치에서 다시 열립니다.
:::

:::tip
브라우저에서 성경 페이지를 북마크하여 일일 읽기를 위해 빠르게 돌아올 수 있습니다. [오늘의 말씀](./verse-of-the-day.md)과 함께 사용하여 매일 성경 영감을 받으세요.
:::

## 이용 가능 여부

성경 페이지는 B1.church의 기본 제공 콘텐츠 기능입니다. 교회의 네비게이션에서 **성경** 링크를 볼 수 없으면 교회 관리자가 사이트 메뉴에 추가하지 않았을 수 있습니다. 추가하도록 요청하려면 교회 관리자에게 문의하세요.
