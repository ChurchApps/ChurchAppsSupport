# 그룹 대화

<div class="article-intro">

그룹 페이지의 대화 탭은 그룹 구성원이 메시지를 게시하고 서로 소통할 수 있는 공간입니다. 대화를 사용하여 주제를 토론하고, 업데이트를 공유하며, 모임 사이에 그룹과 연결 상태를 유지하세요.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1.church 계정에 [로그인](../getting-started/logging-in.md)해야 합니다.
- 대화 탭에 접근하려면 그룹의 구성원이어야 합니다. 그룹에 가입하는 방법은 [그룹 세부 정보](./group-details.md)를 참조하세요.

</div>

## 대화 접근하기

1. [그룹 페이지](./browsing-groups.md)에서 그룹을 클릭하여 그룹 세부 정보 페이지로 이동합니다.
2. 로그인되어 있는지 확인합니다.
3. 사이드바에서 **대화** 탭을 클릭합니다.

## 토론 및 공지

대화 탭에는 두 개의 하위 탭이 표시될 수 있습니다:

- **토론** -- 모든 그룹 구성원이 메시지를 게시하고 다른 사람에게 답글을 달 수 있는 공간입니다. 기본 보기이며 항상 사용할 수 있습니다.
- **공지** -- 모든 구성원이 확인해야 할 중요한 그룹 리더의 메시지입니다. 일반 구성원은 공지를 읽을 수 있지만 리더만 작성할 수 있습니다.

공지가 없고 리더가 아닌 경우, 탭 바 없이 토론 보기만 표시됩니다.

## 메시지 게시하기

1. 그룹 페이지에서 **대화** 탭을 엽니다.
2. 탭 바가 보이는 경우 **토론** 하위 탭에 있는지 확인합니다.
3. 텍스트 필드에 메시지를 입력합니다.
4. 메시지를 제출하여 그룹과 공유합니다.

메시지는 그룹의 모든 구성원에게 표시됩니다.

:::tip
그룹 대화는 [타임라인](../community/timeline.md)에도 표시되므로, 각 그룹을 개별적으로 방문하지 않고도 토론을 따라갈 수 있습니다.
:::

## 리더 기능

그룹 리더인 경우 대화 탭에서 추가 기능을 사용할 수 있습니다:

- **공지 게시** -- **공지** 하위 탭으로 전환하여 모든 그룹 구성원이 읽을 수 있는 메시지를 게시합니다.
- **토론 참여** -- 다른 구성원과 마찬가지로 일반 토론에도 참여할 수 있습니다.

:::info
대화는 각 그룹에 한정됩니다. 한 그룹에 게시한 메시지는 다른 그룹의 대화에 나타나지 않습니다.
:::
