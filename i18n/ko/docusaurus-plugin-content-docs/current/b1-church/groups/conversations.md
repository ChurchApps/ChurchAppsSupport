# 그룹 대화

<div class="article-intro">

그룹 페이지의 대화 탭은 그룹 구성원이 메시지를 게시하고 서로 상호 작용할 수 있는 공간입니다. 대화를 사용하여 주제를 논의하고 업데이트를 공유하며 모임 사이에 그룹과 연결된 상태로 유지하세요.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1.church 계정에 [로그인](../getting-started/logging-in.md)해야 합니다.
- 대화 탭에 액세스하려면 그룹의 멤버여야 합니다. 그룹에 가입하는 방법은 [그룹 세부 정보](./group-details.md)를 참조하세요.

</div>

## 대화 액세스

1. [그룹 페이지](./browsing-groups.md)에서 그룹을 클릭하여 그룹 세부 정보 페이지로 이동합니다.
2. 로그인했는지 확인하세요.
3. 사이드바의 **대화** 탭을 클릭합니다.

## 토론 및 공지사항

대화 탭은 두 개의 서브탭을 표시할 수 있습니다:

- **토론** -- 모든 그룹 구성원이 메시지를 게시하고 다른 사람에게 답장할 수 있는 공간입니다. 이것이 기본 보기이며 항상 사용 가능합니다.
- **공지사항** -- 그룹 리더가 게시한 모든 구성원이 봐야 할 중요한 메시지입니다. 일반 구성원은 공지사항을 읽을 수 있지만 리더만 작성할 수 있습니다.

공지사항이 없고 리더가 아니면 탭 막대 없이 토론 보기만 표시됩니다.

## 메시지 게시

1. 그룹 페이지에서 **대화** 탭을 엽니다.
2. **토론** 서브탭에 있는지 확인합니다 (탭 막대가 표시되는 경우).
3. 텍스트 필드에 메시지를 입력합니다.
4. 메시지를 제출하여 그룹과 공유합니다.

메시지는 그룹의 모든 구성원에게 표시됩니다.

## 메시지 편집 또는 삭제

게시한 모든 메시지를 편집하거나 삭제할 수 있습니다:

1. 메시지 위로 마우스를 가져가거나 (모바일에서는 탭) **추가 옵션** 버튼(⋮)을 표시합니다.
2. 버튼을 클릭하여 메시지 메뉴를 엽니다.
3. **편집**을 선택하여 메시지 텍스트를 수정하거나 **삭제**를 선택하여 제거합니다.
   - 편집할 때 메시지 텍스트가 입력 필드로 다시 로드됩니다. 변경하고 제출하여 저장합니다.
   - 삭제할 때 확인 대화가 나타납니다. **확인**을 클릭하여 메시지를 영구적으로 제거합니다.

:::info
본인이 게시한 메시지만 편집/삭제할 수 있습니다. 다른 구성원이 게시한 메시지에는 편집/삭제 옵션이 표시되지 않습니다.
:::

:::tip
그룹 대화는 [타임라인](../community/timeline.md)에도 나타나므로 각 그룹을 방문하지 않고도 토론을 계속할 수 있습니다.
:::

## 리더 기능

그룹 리더인 경우 대화 탭에서 추가 기능을 사용할 수 있습니다:

- **공지사항 게시** -- **공지사항** 서브탭으로 전환하고 모든 그룹 구성원이 읽을 수 있는 메시지를 게시합니다.
- **토론 게시** -- 다른 구성원처럼 정규 토론에도 참여할 수 있습니다.

:::info
대화는 각 그룹에 고유합니다. 한 그룹에 게시한 메시지는 다른 그룹의 대화에 나타나지 않습니다.
:::
