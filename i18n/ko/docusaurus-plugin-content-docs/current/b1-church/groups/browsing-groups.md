# 그룹 탐색하기

<div class="article-intro">

교회의 공개 그룹 페이지에서 관심 있는 그룹을 쉽게 찾을 수 있습니다. 그룹은 카테고리별로 정리되어 있어 다양한 유형의 그룹을 빠르게 둘러보고 자신에게 맞는 그룹을 찾을 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 그룹을 탐색하기 위해 로그인할 필요가 없습니다. 다만, [로그인](../getting-started/logging-in.md)하면 각 그룹의 세부 정보 페이지에서 추가 기능에 접근할 수 있습니다.

</div>

## 그룹 페이지 찾기

교회의 B1.church 사이트에서 **그룹** 페이지로 이동합니다. 이 페이지는 공개적으로 접근 가능하므로 로그인하지 않아도 탐색할 수 있습니다.

## 그룹 정리 방식

그룹은 **카테고리**(라벨이라고도 함)별로 표시됩니다. 카테고리는 교회 관리자가 설정하며 "소그룹", "사역부", "청소년부", "성경 공부" 등의 라벨이 포함될 수 있습니다. 각 카테고리에는 해당하는 모든 그룹이 표시됩니다.

## 그룹 카드 읽기

각 그룹은 다음 정보가 포함된 카드로 표시됩니다:

- **그룹 사진** -- 그룹을 대표하는 커버 이미지입니다. 사진이 설정되지 않은 경우 기본 이미지가 표시됩니다.
- **그룹 이름** -- 사진 위에 눈에 띄게 표시됩니다.
- **모임 시간** -- 그룹이 보통 모이는 시간입니다 (예: "수요일 오후 7시"). 시간이 설정되지 않은 경우 대체 텍스트가 표시됩니다.
- **모임 장소** -- 그룹이 모이는 장소입니다. 장소가 설정되지 않은 경우 대체 텍스트가 표시됩니다.
- **설명** -- 그룹에 대한 간략한 미리보기로, 몇 줄로 제한됩니다. 전체 설명을 읽으려면 그룹을 클릭하세요.

## 그룹 보기

아무 그룹 카드나 클릭하면 전체 그룹 세부 정보 페이지가 열립니다. 거기서 전체 설명, 그룹 리더 정보, 다가오는 일정 행사 및 연락 방법을 확인할 수 있습니다.

:::tip
특정 유형의 그룹을 찾고 있다면 먼저 카테고리 제목을 살펴보세요. 모든 그룹을 스크롤하지 않고 관심 있는 섹션으로 바로 이동할 수 있습니다.
:::
