# 그룹 가입 요청

<div class="article-intro">

일부 그룹은 멤버가 되기 전에 승인이 필요합니다. 그룹이 이 설정을 가지면 가입 요청을 제출합니다 - 선택적으로 그룹 리더에게 메시지를 포함합니다 - 리더가 승인 또는 거절할 때까지 대기합니다. 보류 중인 요청의 상태를 보고 언제든지 취소할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 그룹 가입을 요청하려면 B1.church 계정에 [로그인](../getting-started/logging-in.md)해야 합니다.

</div>

## 가입 vs. 가입 요청

그룹은 관리자가 설정한 다양한 가입 정책을 가질 수 있습니다:

- **공개** -- 그룹의 세부 정보 페이지에서 **그룹 가입**을 클릭하면 즉시 멤버로 추가됩니다.
- **요청 필수** -- **가입 요청**을 클릭하여 그룹 리더가 승인해야 하는 요청을 제출합니다.
- **닫혀 있음** -- 가입 버튼이 나타나지 않습니다. 멤버는 리더가 수동으로 추가해야 합니다.

## 가입 요청 제출 방법

1. 가입하려는 그룹을 찾아보세요. [그룹 찾아보기](./browsing-groups.md)를 참조하여 찾는 방법을 확인하세요.
2. 그룹의 세부 정보 페이지를 엽니다.
3. **가입 요청**을 클릭합니다.
4. 그룹 리더에게 가입하려는 이유를 설명하는 선택 메시지를 작성할 수 있는 대화상자가 나타납니다.
5. **제출**을 클릭하여 요청을 보냅니다.

그룹의 리더는 요청에 대한 알림을 받습니다. 검토한 후 결과에 대한 알림을 받게 됩니다.

:::tip
간단한 메시지(그룹을 어떻게 알게 되었는지 또는 무엇을 얻기를 원하는지)를 포함하면 리더가 더 빨리 결정하는 데 도움이 될 수 있습니다.
:::

## 요청 상태 확인

요청을 제출한 후 **그룹** 페이지에서 상태를 볼 수 있습니다:

1. 교회의 B1.church 사이트의 **그룹** 페이지로 이동합니다.
2. **보류 중인 요청** 섹션으로 아래로 스크롤합니다.
3. 각 보류 중인 요청은 그룹 이름과 제출한 날짜를 표시합니다.

요청이 승인되거나 거절되면 알림을 받게 됩니다 - 승인된 요청은 멤버로 그룹으로 옮깁니다.

## 요청 취소

리더가 요청을 검토하기 전에 마음을 바꾸면:

1. **그룹** 페이지로 이동합니다.
2. **보류 중인 요청** 섹션에서 요청을 찾습니다.
3. 요청 옆의 **취소**를 클릭합니다.

요청이 즉시 철회되고 리더는 더 이상 이를 볼 수 없습니다.
