# 이벤트 참석 여부

<div class="article-intro">

그룹 이벤트는 리더가 누가 참석하는지 알 수 있도록 참석 여부를 수집할 수 있습니다. 그룹의 달력에서 각 예정 이벤트는 "참석하시겠습니까?"를 물어봅니다. 한 번의 탭으로 답변하고, 언제든지 마음을 바꿀 수 있으며, 다른 사람이 몇 명이나 가는지 한눈에 볼 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [로그인](../getting-started/logging-in.md)하고 그룹의 회원이어야 합니다.
- 이벤트에 참석 여부가 허용되어야 합니다. (기본적으로 활성화되어 있습니다. 리더는 이벤트별로 이를 끌 수 있습니다.)

</div>

## 이벤트에 응답하기

1. 회원 포털에서 그룹을 열고 **이벤트** / 달력 보기로 이동합니다.
2. 각 예정 이벤트 카드는 **참석하시겠습니까?** 를 표시하며 세 개의 버튼이 있습니다. **참석**, **미정**, **불참**.
3. 답변을 탭합니다. 선택사항이 강조표시되고 각 버튼은 해당 응답을 제공한 사람 수의 실시간 수를 표시합니다.
4. 강조표시된 답변을 다시 탭하여 지우거나 다른 버튼을 탭하여 응답을 변경합니다.

참석 여부는 특정 이벤트에만 적용됩니다. 주간 모임의 경우 각 주의 이벤트에 별도로 답변합니다.

## 그룹 리더의 경우

- **응답 보기** -- 리더는 각 이벤트 카드에서 **응답 보기** 버튼을 봅니다. 응답 시트가 열리며 모든 사람이 응답별로 나열됩니다. 참석, 미정, 불참. 아직 아무도 답변하지 않았으면 "응답이 없습니다."가 표시됩니다.
- **참석 여부 켜기 또는 끄기** -- 이벤트를 만들거나 편집할 때 **참석 여부 허용** 스위치를 사용합니다. 참석 여부가 꺼진 이벤트는 응답 버튼을 표시하지 않습니다. B1 Admin에서 일괄적으로 이벤트를 만드는 직원은 동일한 옵션을 사용합니다. [그룹 달력](../../b1-admin/groups/group-calendar#event-rsvps)을 참조하세요.

:::tip
참석 여부는 간단한 참석자 수입니다. 이벤트에 가입 세부정보, 수용 인원 관리 또는 결제가 필요하면 [이벤트 등록](../events/registering)을 대신 사용합니다.
:::
