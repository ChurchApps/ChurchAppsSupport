# 타임라인

<div class="article-intro">

**타임라인**은 B1.church 교인 포털의 기본 랜딩 페이지입니다. 그룹 및 교회의 최신 업데이트를 보여주는 활동 피드를 제공하여 모든 것을 한곳에서 편리하게 확인할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1.church 계정에 [로그인](../getting-started/logging-in.md)해야 합니다.

</div>

## 타임라인에 표시되는 내용

타임라인에는 다음을 포함한 게시물과 업데이트 피드가 표시됩니다:

- **그룹 대화** -- 소속된 [그룹](../groups/)의 메시지와 토론입니다.
- **과제** -- 귀하 또는 그룹에 배정된 과제입니다.
- **행사** -- 귀하와 그룹에 관련된 다가오는 행사입니다.
- **공과** -- 그룹과 공유된 교육과정 및 [공과 콘텐츠](../content/lessons.md)입니다.
- **설교** -- 교회에서 게시한 최근 [설교](../content/sermons.md)입니다.

피드의 각 게시물에는 게시물 제목, 해당하는 경우 소속 그룹, 그리고 게시된 시간이 표시됩니다.

## 게시물 세부 정보 보기

타임라인은 마스터-디테일 레이아웃을 사용합니다. 피드는 페이지 왼쪽에 나열됩니다. 아무 게시물이나 클릭하면 오른쪽 패널에 전체 세부 정보가 열립니다. 상세 보기에서 게시물의 전체 내용을 읽고 상호작용할 수 있습니다.

상세 패널 상단의 **업데이트로 돌아가기**를 클릭하면 피드 목록으로 돌아갑니다.

## 연결 유지하기

타임라인은 모든 그룹의 활동을 하나의 피드로 모아주므로, 각 그룹을 개별적으로 확인할 필요가 없습니다. 교회에서 일어나는 일을 파악하기 위해 타임라인을 정기적으로 방문하는 것이 좋습니다.

:::tip
타임라인이 비어 있다면, 아직 어떤 그룹에도 가입하지 않았을 수 있습니다. [그룹](../groups/browsing-groups.md) 페이지를 방문하여 가입할 그룹을 찾거나, 교회 관리자 또는 그룹 리더에게 추가해 달라고 요청하세요.
:::
