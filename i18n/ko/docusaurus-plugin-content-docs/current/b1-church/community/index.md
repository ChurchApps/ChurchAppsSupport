# 커뮤니티

<div class="article-intro">

B1.church의 **커뮤니티** 섹션은 교회 가족과 연결 상태를 유지하는 데 도움을 줍니다. 교인 디렉토리를 통해 동료 교인을 찾고 알아가며, 타임라인 피드를 통해 그룹과 교회에서 일어나는 일들을 파악할 수 있습니다.

</div>

## 이용 가능한 기능

- **[교인 디렉토리](./member-directory.md)** -- 교회 교인 명부를 검색하고 탐색합니다. 이름으로 사람을 찾거나 그룹별로 필터링하여 원하는 사람을 찾을 수 있습니다. 교인을 클릭하면 프로필 세부 정보를 볼 수 있습니다.
- **[타임라인](./timeline.md)** -- 그룹 및 교회의 최신 업데이트를 보여주는 활동 피드입니다. 타임라인은 교인 포털을 방문할 때 기본 랜딩 페이지입니다.

## 시작하기

커뮤니티 기능에 접근하려면 [로그인](../getting-started/logging-in.md)해야 합니다. 로그인하지 않은 상태에서 **커뮤니티** 탭이나 **타임라인**으로 이동하면 로그인하라는 메시지가 표시됩니다.

**타임라인** 탭은 사이드바에서 항상 사용할 수 있으며 교인 포털의 홈 페이지 역할을 합니다. **커뮤니티** 탭은 교인 디렉토리에 접근할 수 있게 해줍니다.

:::info
디렉토리에서 볼 수 있는 교인 및 정보는 교회의 개인정보 설정에 따라 다릅니다. 모든 교인의 전체 프로필을 볼 수 없을 수도 있습니다. 공유되는 정보에 대해 궁금한 점이 있으면 교회 관리자에게 문의하세요.
:::

:::tip
[그룹](../groups/)과 함께 커뮤니티 기능을 활용하여 한 주 내내 교회와 소통하세요.
:::
