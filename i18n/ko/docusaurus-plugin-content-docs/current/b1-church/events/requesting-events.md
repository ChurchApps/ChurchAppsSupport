# 이벤트 및 실 요청하기

<div class="article-intro">

교회에서 예약 가능한 실과 리소스를 추적하는 경우 회원 포털에서 바로 이벤트를 요청할 수 있습니다(필요한 실과 장비 포함). 기존 예약과의 충돌이 선택할 때 플래그가 지정되고, 요청이 교회로 전송되어 승인을 기다리며, 내 요청에서 추적하거나 취소할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 회원 포털에 [로그인](../getting-started/logging-in.md)해야 합니다.
- 교회에서 B1 Admin에 실 또는 리소스를 설정해야 합니다. 요청 바로가기는 설정된 경우에만 나타납니다.

</div>

## 요청 제출하기

1. 회원 포털에서 **내 페이지** 페이지를 열고 바로가기 아래 **이벤트 요청하기**를 탭합니다.
2. 이벤트 기본 정보를 입력합니다: **이벤트 제목**, 선택사항인 **설명**, **시작** 및 **종료** 시간(또는 **종일**), 그리고 **반복**되는지 여부입니다.
3. 필요한 **실** 및 **리소스**를 선택합니다. 리소스의 경우 **수량**을 설정할 수 있습니다.
4. 충돌 경고를 주의하세요. 시간이 실이나 리소스의 기존 예약과 겹치면 *"이 시간은 기존 예약과 충돌합니다."*가 표시됩니다. 시간을 조정하거나 다른 항목을 선택할 수 있습니다.
5. **요청 제출**을 탭합니다.

제출 후 **요청이 제출되었습니다** 요약이 요청한 각 실 및 리소스의 상태를 표시합니다.

- **승인됨** -- 즉시 예약됨 (실 또는 리소스에 승인 그룹이 없음).
- **승인 대기 중** -- 교회의 누군가가 먼저 서명해야 합니다.

## 요청 추적하기

내 페이지에서 **내 요청**을 엽니다. 각 요청은 이벤트, 요청 시간, 상태 배지를 표시합니다.

- **대기 중** -- 결정을 기다리는 중
- **승인됨** -- 예약이 확인됨
- **거부됨** -- 요청이 거부됨

요청이 여전히 대기 중인 동안 **요청 취소**로 철회할 수 있습니다 (확인하라는 메시지가 표시되며, 취소는 실행 취소할 수 없습니다).

:::info
요청은 B1 Admin [승인 페이지](../../b1-admin/calendars/approvals)에서 교회 직원이 검토합니다. 어떤 실과 리소스가 승인이 필요한지는 교회에 따릅니다. [실 및 리소스](../../b1-admin/calendars/rooms-resources)를 참조하세요.
:::
