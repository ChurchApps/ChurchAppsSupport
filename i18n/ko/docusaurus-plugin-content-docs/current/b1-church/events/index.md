# 이벤트

<div class="article-intro">

B1.church의 이벤트 섹션을 통해 예정된 교회 이벤트를 찾아보고 웹사이트에서 직접 등록할 수 있습니다. 교회가 이벤트 등록을 활성화할 때 몇 가지 간단한 단계로 본인과 가족 구성원을 등록할 수 있습니다.

</div>

## 이벤트 찾아보기

이벤트는 교회 일정에 표시됩니다. 등록이 활성화된 이벤트를 열면 이벤트 세부 정보와 함께 **이 이벤트에 등록** 버튼이 표시됩니다.

등록이 활성화된 이벤트는 다음을 표시합니다:

- **이벤트 제목 및 설명** -- 이벤트가 무엇에 관한 것인지.
- **날짜 및 시간** -- 이벤트가 발생하는 시간.
- **등록 버튼** -- 등록 양식으로 이동하는 파란색 버튼.

## 등록 상태

이벤트는 타이밍 및 용량에 따라 다른 상태를 표시할 수 있습니다:

- **열기** -- 등록이 가능하며 등록할 수 있습니다.
- **아직 열리지 않음** -- 등록 기간이 시작되지 않았습니다. 시작 날짜 이후에 다시 확인하세요.
- **닫기** -- 등록 마감일이 지났습니다.
- **가득 찬** -- 모든 사용 가능한 자리가 채워졌습니다.

:::tip
이벤트가 가득 차면 나중에 다시 확인하세요 -- 누군가 등록을 취소하면 자리가 열릴 수 있습니다.
:::

## 필요한 것

- B1.church 계정 (또는 손님으로 등록할 수 있습니다)
- 교회의 B1.church 웹사이트 액세스
