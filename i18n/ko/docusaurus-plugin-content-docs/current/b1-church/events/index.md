# 이벤트

<div class="article-intro">

B1.church의 이벤트 섹션을 통해 예정된 교회 이벤트를 찾아보고 웹사이트에서 직접 등록할 수 있습니다. 교회에서 이벤트에 대한 등록을 활성화하면 몇 가지 단계만으로 자신과 가족 구성원을 가입할 수 있습니다.

</div>

## 이벤트 찾아보기

이벤트는 교회 캘린더에 나타납니다. 등록이 활성화된 이벤트를 열면 이벤트 세부사항과 함께 **이 이벤트에 등록** 버튼이 표시됩니다.

등록이 활성화된 이벤트는 다음을 표시합니다:

- **이벤트 제목 및 설명** -- 이벤트가 무엇에 관한 것인지입니다.
- **날짜 및 시간** -- 이벤트가 발생하는 시간입니다.
- **등록 버튼** -- 등록 양식으로 이동하는 파란 버튼입니다.

## 등록 상태

이벤트는 시간 및 용량에 따라 다양한 상태를 보여줄 수 있습니다:

- **열림** -- 등록을 할 수 있고 가입할 수 있습니다.
- **아직 열리지 않음** -- 등록 기간이 시작되지 않았습니다. 시작 날짜 후 다시 확인하세요.
- **폐쇄됨** -- 등록 마감이 지났습니다.
- **가득 참** -- 모든 사용 가능한 자리가 차있습니다.

:::tip
이벤트가 가득 찬 경우 나중에 다시 확인하세요 -- 누군가 등록을 취소하면 자리가 열릴 수 있습니다.
:::

## 필요한 것

- B1.church 계정(또는 손님으로 등록할 수 있음)
- 교회의 B1.church 웹사이트 액세스
