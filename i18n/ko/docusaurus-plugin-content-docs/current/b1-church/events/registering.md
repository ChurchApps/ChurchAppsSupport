# 이벤트 등록

<div class="article-intro">

교회에서 이벤트에 대한 등록을 활성화하면 간단한 3단계 프로세스를 통해 가입할 수 있습니다. 계정이 있거나 손님으로 방문하는지 여부에 관계없이 자신과 가족 구성원을 등록할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교회에 이벤트에 대한 이벤트 등록이 활성화되어야 함
- 등록 기간이 열려 있어야 함(관리자가 설정한 시작 및 종료 날짜 사이)

</div>

## 1단계: 정보 입력

1. 교회 캘린더에서 등록할 이벤트를 열고 **이 이벤트에 등록**을 클릭합니다.
2. 등록 페이지가 로드되어 이벤트 제목, 설명, 날짜를 표시합니다.
3. **로그인한** 경우 양식이 이름을 표시하고 계정 아래에 등록하는 것을 확인합니다.
4. **손님**(로그인하지 않음)인 경우 다음을 작성합니다:
   - **이름**(필수)
   - **성**(필수)
   - **이메일**(필수)
   - **전화**(선택 사항)

:::info
이벤트에 용량 제한이 있으면 진행률 표시줄이 남은 자리 수를 표시합니다. 이벤트가 가득 찬 경우 등록할 수 없습니다.
:::

## 2단계: 가족 구성원 추가

1. 추가 가족 구성원을 등록하려면 각 사람의 **이름** 및 **성**을 입력합니다.
2. **회원 추가**를 클릭하여 더 많은 사람을 추가합니다(총 10명까지).
3. 모든 회원 옆의 제거 버튼을 클릭하여 등록에서 제거합니다.
4. 자신만 등록하는 경우 이 단계를 건너뜁니다.

## 3단계: 확인

1. 이벤트 세부사항 및 등록 정보를 검토합니다.
2. 등록을 확인합니다.
3. 이벤트 세부사항 및 등록한 모든 사람의 목록과 함께 확인 메시지가 나타납니다.

또한 계정과 연결된 이메일 주소(또는 손님으로 제공한 이메일)에서 **확인 이메일**을 받게 됩니다.

## 등록 취소

등록을 취소해야 하는 경우 교회 관리자에게 연락하세요. 관리자는 B1 Admin 등록 페이지에서 등록을 취소할 수 있습니다.
