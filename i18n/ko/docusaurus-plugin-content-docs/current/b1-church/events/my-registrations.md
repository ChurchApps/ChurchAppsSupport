# 내 등록

<div class="article-intro">

회원 포털의 **등록** 페이지는 가입한 모든 이벤트의 홈입니다. 여기서 상태를 확인하거나, 미납 잔액을 지불하거나, 참석자 및 주문 항목을 편집하거나, 등록을 완전히 취소할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [로그인](../getting-started/logging-in.md)해야 합니다. 게스트 등록은 여기에 표시되지 않습니다.
- 최소 하나의 이벤트 등록이 필요합니다. [이벤트 등록하기](./registering)를 참조하세요.

</div>

## 등록 보기

회원 포털에서 **등록**을 엽니다. 각 등록은 다음을 표시하는 카드로 나타납니다.

- **이벤트 제목 및 날짜**
- **상태** 배지 — **승인됨**, **대기 중**, **대기자 목록**, 또는 **취소됨**
- 등록한 **회원**
- 결제 라인 — 결제된 총액 또는 돈이 아직 남아 있으면 **미납 잔액**

아직 등록한 항목이 없으면 "등록된 항목이 없습니다."가 표시됩니다.

## 결제 완료하기

미납 잔액이 있는 모든 등록에는 **결제 완료** 버튼이 나타납니다. 가장 일반적인 경우는 대기자 목록에서 승격되거나 직원 편집으로 합계가 변경된 경우입니다.

1. **결제 완료**를 클릭합니다.
2. 주문 요약을 검토하고, 저장된 카드를 선택하거나 새 카드를 입력한 후 결제합니다.
3. 잔액이 지불되면 등록이 **승인됨** 상태가 됩니다.

:::tip
대기자 목록 자리가 열리면 교회에서 보낸 이메일에는 이 결제 단계로 바로 가는 링크가 포함되어 있습니다.
:::

## 등록 편집하기

계획이 변경될 수 있습니다. 등록에서 **편집**을 클릭하여 **등록 편집** 대화 상자를 열면 다음을 할 수 있습니다.

- **연락처 세부정보** 업데이트
- 각 회원의 **이름** 및 **참석자 유형** 변경
- **선택 수량** 조정 (T셔츠 또는 식사와 같은 추가 옵션)

편집할 때도 수용 인원이 적용됩니다. 참석자 유형 또는 옵션이 매진된 경우 다른 옵션을 선택하라는 메시지가 표시됩니다. 저장하면 합계가 다시 계산됩니다. 변경으로 인해 미납 잔액이 남으면 카드에 **결제 완료** 버튼이 나타납니다.

:::info
편집할 때 카드가 자동으로 청구되지 않습니다. 새 총액이 더 높으면 **결제 완료**로 차액을 결제합니다. 더 낮으면 교회에 환불에 대해 문의합니다. 환불은 교회 직원이 결제 제공자를 통해 처리합니다.
:::

## 등록 취소하기

1. 카드에서 **등록 취소**를 클릭합니다.
2. 대화 상자에서 확인합니다. 취소는 실행 취소할 수 없습니다.

자리가 즉시 해제됩니다. 이벤트에 대기자 목록이 있으면 목록의 다음 사람이 자동으로 승격됩니다. 지불한 금액은 자동으로 환불되지 않습니다. 교회 사무실에 환불에 대해 문의하세요.
