# 헌금하기

<div class="article-intro">

교회의 B1.church 사이트를 통해 두 가지 방법으로 헌금할 수 있습니다: 공개 헌금 페이지(로그인 불필요)와 교인 헌금 페이지(로그인 필요). 두 가지 모두 Stripe에서 제공하는 안전한 헌금 양식을 사용합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 공개 헌금의 경우 계정이 필요하지 않습니다. 교회의 **헌금** 페이지를 방문하기만 하면 됩니다.
- 내역 추적 및 저장된 결제 수단이 포함된 교인 헌금의 경우, 먼저 계정에 [로그인](../getting-started/logging-in.md)하세요.

</div>

## 공개 헌금 페이지 (로그인 불필요)

계정을 만들지 않고도 누구나 헌금할 수 있습니다:

1. 교회의 **헌금** 페이지로 이동합니다.
2. 페이지 왼쪽에 헌금 양식이 표시됩니다. 헌금할 **기금**을 선택합니다.
3. 결제 정보와 헌금 금액을 입력합니다.
4. 양식을 제출하여 헌금을 완료합니다.

페이지 오른쪽에는 은행 이체를 선호하는 경우 **ACH 결제 안내**로 연결되는 링크가 있습니다. 또한 헌금 내역과 같은 추가 기능에 접근하기 위해 로그인하려는 경우 **로그인** 버튼도 있습니다.

:::tip
공개 헌금 페이지를 방문할 때 이미 로그인되어 있으면, 전체 헌금 내역과 저장된 결제 수단에 접근할 수 있는 교인 헌금 페이지로 자동 리디렉션됩니다.
:::

## 교인 헌금 페이지 (로그인 필요)

로그인하면 추가 기능이 포함된 헌금 환경이 제공됩니다:

1. **헌금** 탭을 클릭합니다 (또는 **내 헌금**으로 이동합니다).
2. 다음 세 가지 섹션이 있는 패널이 표시됩니다:
   - **지금 헌금하기** -- 일회성 또는 정기 헌금을 합니다.
   - **내역** -- 과거 헌금을 확인하고 기록을 내보냅니다.
   - **정기 헌금** -- 예약된 헌금 및 결제 수단을 관리합니다.
3. **지금 헌금하기**를 선택하여 헌금 양식을 엽니다.
4. 헌금할 **기금**을 선택하고 금액을 입력합니다.
5. 기존에 저장된 결제 수단을 선택하거나 새로운 결제 정보를 입력합니다.
6. 양식을 제출하여 헌금을 완료합니다.

헌금이 처리되면 성공 메시지가 표시되고 헌금이 [헌금 내역](./donation-history.md)에 나타납니다.

## 결제 수단

헌금은 **Stripe**를 통해 안전하게 처리됩니다. 신용카드, 직불카드로 결제하거나 ACH 은행 결제를 설정할 수 있습니다. 로그인하면 결제 수단을 저장하여 다음 헌금 시 더 빠르게 이용할 수 있습니다.

:::warning
헌금이 처리되지 않는 경우, 결제 정보를 다시 확인하고 재시도하세요. 문제가 지속되면 교회 사무실에 문의하세요.
:::
