# 체크인

<div class="article-intro">

**체크인** 기능은 B1.church에서 회원 포털에서 직접 교회 예배에 대해 자신과 가구 구성원을 체크인할 수 있습니다. 이것은 물리적 체크인 키오스크를 방문할 필요 없이 출석을 기록하는 편리한 방법입니다.

</div>

## 작동 원리

자체 체크인을 사용할 때 예배를 선택하고 참석 중인 가구 구성원을 선택하고 각 사람을 각 예배 시간에 대해 적절한 그룹에 할당합니다. 확인하면 출석이 자동으로 기록됩니다.

## 시작하기

체크인에 액세스하려면 회원 포털의 [사이드바 네비게이션](../getting-started/navigating.md)에서 **체크인**을 클릭합니다. 이 기능을 사용하려면 [로그인](../getting-started/logging-in.md)해야 합니다 -- 로그인하지 않은 경우 먼저 로그인하라는 메시지가 표시됩니다.

:::info
체크인 기능을 사용하려면 교회가 관리 설정에서 예배 및 출석 그룹을 설정했어야 합니다. 체크인 페이지를 열 때 사용 가능한 예배가 없으면 교회가 아직 이 기능을 구성하지 않았을 수 있습니다. 도움을 요청하려면 교회 관리자에게 연락하세요.
:::
