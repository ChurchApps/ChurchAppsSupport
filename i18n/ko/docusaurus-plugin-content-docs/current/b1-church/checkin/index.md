# 체크인

<div class="article-intro">

B1.church의 **체크인** 기능을 사용하면 교인 포털에서 직접 예배에 본인과 가족 구성원을 체크인할 수 있습니다. 물리적 체크인 키오스크를 방문하지 않고도 편리하게 출석을 기록할 수 있는 방법입니다.

</div>

## 작동 방식

셀프 체크인을 사용하면 예배를 선택하고, 참석하는 가족 구성원을 선택한 후, 각 사람을 각 예배 시간의 적절한 그룹에 배정합니다. 확인하면 출석이 자동으로 기록됩니다.

## 시작하기

체크인에 접근하려면 교인 포털의 [사이드바 내비게이션](../getting-started/navigating.md)에서 **체크인**을 클릭합니다. 이 기능을 사용하려면 [로그인](../getting-started/logging-in.md)해야 합니다. 로그인하지 않은 경우 먼저 로그인하라는 메시지가 표시됩니다.

:::info
체크인 기능을 사용하려면 교회에서 관리 설정에 예배 및 출석 그룹을 설정해 두어야 합니다. 체크인 페이지를 열었을 때 사용 가능한 예배가 없는 경우, 교회에서 아직 이 기능을 설정하지 않았을 수 있습니다. 교회 관리자에게 문의하세요.
:::
