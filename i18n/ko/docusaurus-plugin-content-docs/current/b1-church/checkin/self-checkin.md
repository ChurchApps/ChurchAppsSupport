# 셀프 체크인

<div class="article-intro">

셀프 체크인 프로세스를 통해 몇 단계만으로 본인과 가족의 출석을 기록할 수 있습니다. 교회의 물리적 키오스크를 사용하는 것보다 빠른 대안입니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1.church 계정에 [로그인](../getting-started/logging-in.md)해야 합니다.
- 가족 구성원이 귀하의 계정에 연결되어 있어야 합니다. 가족 구성원이 누락된 경우, 교회 관리자에게 가족에 추가해 달라고 요청하세요.

</div>

## 체크인 절차

체크인은 간단한 3단계 과정을 따릅니다. 시작하려면 [사이드바](../getting-started/navigating.md)에서 **체크인** 탭으로 이동합니다.

### 1단계: 예배 선택

체크인 페이지를 열면 사용 가능한 예배 목록이 표시됩니다 (예: "주일 오전 예배" 또는 "수요일 저녁 예배"). 각 예배 카드에는 예배 이름과 교회에 여러 캠퍼스가 있는 경우 캠퍼스 정보가 표시됩니다.

예배를 클릭하여 선택하고 다음 단계로 진행합니다.

### 2단계: 가족 구성원 및 그룹 선택

예배를 선택하면 가족 구성원 목록이 표시됩니다. 참석하는 각 사람에 대해:

1. 가족 구성원을 탭하여 항목을 확장합니다.
2. 선택한 예배에 사용 가능한 **예배 시간**이 표시됩니다.
3. 예배 시간 옆의 **그룹 선택**을 클릭하여 해당 사람이 체크인할 그룹을 선택합니다 (예: "본당 예배", "어린이실 1", "청소년부").
4. 참석하는 모든 가족 구성원에 대해 반복합니다.

모든 가족 구성원을 체크인할 필요는 없습니다. 실제로 참석하는 사람에 대해서만 항목을 확장하고 그룹을 배정하면 됩니다.

### 3단계: 확인

참석하는 모든 사람의 그룹을 선택한 후, 페이지 하단의 **체크인 완료** 버튼을 클릭합니다.

"체크인 완료!" 메시지와 함께 출석이 저장되었다는 안내가 표시되는 확인 화면이 나타납니다. **내 페이지로 돌아가기**를 클릭하면 [타임라인](../community/timeline.md)으로 돌아갑니다.

:::tip
체크인을 완료하기 전에 그룹 선택을 변경해야 하는 경우, 예배 시간 옆의 **변경** 버튼을 탭하여 다른 그룹을 선택할 수 있습니다.
:::

## 알아두실 사항

- 귀하의 계정에 연결된 가족 구성원만 체크인할 수 있습니다. 가족 구성원이 누락된 경우, 교회 관리자에게 가족에 추가해 달라고 요청하세요.
- 체크인 페이지에 예배가 표시되지 않으면, 현재 교회에서 체크인용 예배를 설정하지 않았을 수 있습니다.

:::warning
체크인은 교회에 활성 예배가 설정되어 있을 때만 사용할 수 있습니다. 예배 목록이 표시되지 않으면 교회 사무실에 셀프 체크인이 활성화되어 있는지 확인하세요.
:::
