# 봉사

<div class="article-intro">

B1.church의 봉사 섹션을 통해 자원봉사 기회를 찾아보고 교회 위치에 가입할 수 있습니다. 할당된 계획 및 예배순서를 보는 것 외에도 채워야 할 위치를 자체 선택할 수 있습니다.

</div>

## 할 수 있는 것

- **자원봉사 기회 찾아보기** -- 자원봉사자가 필요한 열린 위치가 있는 예정된 예배 계획을 봅니다.
- **위치 가입** -- 위치를 선택하고 한 번의 클릭으로 자신을 가입합니다.
- **계획 보기** -- 예정된 봉사 할당 및 과거 [봉사 할당](../plans/viewing-plans)을 봅니다.
- **가입 제거** -- 계획이 변경되면 자체 가입을 취소합니다(마감 전).

## 작동 방식

교회 관리자가 예배 계획을 만들고 특정 위치를 자체 가입에 사용 가능하도록 표시합니다. 이 위치는 자원봉사자 가입 페이지에 "밴드", "환대", "아동 사역" 같은 범주별로 구성하여 나타납니다. 각 위치에 사용 가능한 자리 수를 보고 사용 가능한 것에 가입할 수 있습니다.

:::info
교회 관리자가 자체 가입에 활성화한 위치만 자원봉사자 페이지에 나타납니다. 일부 위치는 직접 할당을 통해서만 채워질 수 있습니다.
:::
