# 자원봉사자 가입

<div class="article-intro">

자원봉사 기회가 있으면 한 번의 클릭으로 위치에 가입할 수 있습니다. 각 위치는 역할의 설명을 포함하여 무엇을 수행할지 정확히 알 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1.church 계정에 [로그인](../getting-started/logging-in)
- 교회 관리자가 자원봉사 위치에 대한 자체 가입을 활성화했는지 확인

</div>

## 열린 자원봉사 위치 찾기

1. B1.church에 로그인합니다.
2. 왼쪽 사이드바에서 **봉사**를 클릭합니다.
3. **자원봉사자 가입** 섹션을 열어 열린 위치를 봅니다.
4. 위치는 범주(예: "밴드", "환대", "아동 사역")별로 구성됩니다.

## 위치에 가입하기

1. 가입하려는 위치를 찾습니다.
2. 위치 카드는 남은 자리 수를 표시합니다.
3. **가입** 버튼을 클릭합니다.
4. 확인 메시지가 표시되어 성공적으로 가입했음을 확인합니다.

:::tip
모든 위치가 자체 가입에 사용 가능한 것은 아닙니다. 일부 역할은 관리자에 의해 직접만 할당됩니다.
:::

## 가입 제거하기

계획이 변경되면 가입을 제거할 수 있습니다:

1. **봉사**로 이동합니다.
2. **내 예정된 위치** 섹션에서 제거할 위치를 찾습니다.
3. **가입 제거** 버튼을 클릭합니다.

:::warning
마감 후에는 자체 가입을 제거할 수 없습니다. 마감 후 변경이 필요하면 교회 관리자에게 연락하세요.
:::
