# 자원봉사 등록

<div class="article-intro">

사용 가능한 자원봉사 직책을 찾아보고 B1.church에서 직접 예정 서비스에 직책에 등록합니다. 직책은 카테고리별로 구성되어 있어 도움이 필요한 곳을 빠르게 찾고 팀에 합류할 수 있습니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- B1.church 계정 -- 등록하려면 로그인해야 합니다.
- 교회 관리자는 최소 하나의 직책에서 자체 등록을 활성화해야 합니다.

</div>

## 사용 가능한 직책 찾아보기

1. 교회의 B1.church 웹사이트의 **자원봉사** 페이지로 이동합니다.
2. 공개 자원봉사 직책이 있는 예정 서비스 계획 목록이 표시됩니다.
3. 각 계획은 **계획 이름**, **서비스 날짜** 및 **서비스 시간**을 표시합니다.

직책은 카테고리별로 그룹화됩니다 (예: "밴드", "접객", "어린이 사역"). 각 직책은 다음을 표시합니다:

- **직책 이름** -- 입력할 역할 (예: "드러머", "맞이하는 사람", "커피 서버").
- **설명** -- 관리자가 제공한 경우 역할에 대한 세부 정보입니다.
- **사용 가능한 자리** -- 채워진 슬롯 수와 남은 슬롯 수를 보여주는 진행 표시줄입니다.

## 등록

1. 자원봉사하려는 직책을 찾습니다.
2. 직책 옆의 **등록** 버튼을 클릭합니다.
3. "등록되었습니다! 자원봉사해주셔서 감사합니다."라는 확인 메시지가 나타납니다.
4. 직책은 이제 등록되었음을 확인하는 녹색 표시기를 표시합니다.

:::info
등록하려면 로그인해야 합니다. 로그인하지 않은 경우 먼저 로그인하라는 메시지가 표시됩니다.
:::

## 등록 제거

계획이 바뀌어 더 이상 봉사할 수 없는 경우:

1. 등록한 직책을 찾으세요 - 녹색 "등록됨" 표시기가 표시됩니다.
2. **제거** 버튼을 클릭합니다.
3. 직책에서 자신을 제거하려는 것을 확인합니다.
4. "이 직책에서 제거되었습니다."라는 메시지가 확인됩니다.

## 마감일

교회 관리자는 각 계획에 등록 마감일을 설정할 수 있습니다. 마감일은 서비스 날짜 전 설정된 시간 수입니다. 마감일이 지나면:

- 새 직책에 등록할 수 없습니다.
- 기존 등록을 제거할 수 없습니다.
- 계획 맨 위의 메시지는 등록 마감일이 지났음을 나타냅니다.

## 직책 용량

각 직책은 제한된 자리 수를 가집니다. 모든 자리가 채워지면:

- 직책은 **가득 찬** 것으로 표시됩니다.
- 등록 버튼이 비활성화됩니다.
- 누군가 등록을 제거하면 자리가 다시 열립니다.

:::tip
자원봉사 페이지를 정기적으로 확인하세요, 특히 주중 초반에. 인기 있는 직책은 빠르게 채워지므로 일찍 등록하면 최고의 선택을 할 수 있습니다.
:::
