# 로그인하기

<div class="article-intro">

B1.church에 로그인하면 타임라인 확인, 그룹 관리, 온라인 헌금 등 교회에서 설정한 모든 도구를 사용할 수 있는 교인 포털에 접근할 수 있습니다. 이 문서에서는 로그인, 사용자 메뉴 탐색 및 로그아웃 방법을 설명합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- ChurchApps 계정이 필요합니다. 계정을 만들려면 [시작하기](../getting-started/)를 참조하세요.
- 교회의 B1.church URL이 필요합니다 (예: `https://yourchurchname.b1.church`).

</div>

## 로그인 방법

1. 교회의 B1.church URL로 이동합니다 (예: `https://yourchurchname.b1.church`).
2. 헤더 오른쪽 상단의 **로그인** 버튼을 클릭합니다.
3. **이메일 주소**와 **비밀번호**를 입력합니다.
4. **로그인**을 클릭합니다.

로그인 후, 로그인 버튼을 클릭하기 전에 보고 있던 페이지로 돌아갑니다. 로그아웃 상태에서 교인 포털 페이지로 직접 이동한 경우, 로그인 후 자동으로 해당 페이지로 이동합니다.

## 교인 포털 접근하기

로그인한 후 페이지 오른쪽 상단의 **이름**을 클릭하고 메뉴에서 **교인 포털**을 선택하면 교인 포털에 접근할 수 있습니다. 교인 포털에는 [타임라인](../community/timeline.md), [그룹](../groups/), [헌금](../giving/) 등 모든 개인 교회 도구가 있습니다.

## 사용자 메뉴

로그인 후 오른쪽 상단에 이름이 칩 형태로 표시됩니다. 클릭하면 다음 옵션이 포함된 메뉴가 열립니다:

- **교인 포털** -- 개인 교인 포털 대시보드로 이동합니다.
- **프로필 편집** -- 계정 설정을 업데이트하거나 교회 프로필을 수정합니다.
- **로그아웃** -- 계정에서 로그아웃합니다.

## 관리자 접근

적절한 권한을 가진 교회 관리자인 경우, 사용자 메뉴에 **관리자 포털** 옵션도 표시됩니다. 클릭하면 교회의 설정, 인원, 그룹 및 콘텐츠를 관리할 수 있는 B1 Admin으로 이동합니다.

:::tip
비밀번호를 잊어버린 경우, 로그인 페이지의 **비밀번호 찾기** 링크를 사용하여 재설정하세요. 새 비밀번호를 만드는 방법이 포함된 이메일을 받게 됩니다.
:::

## 로그아웃

로그아웃하려면 오른쪽 상단의 **이름**을 클릭하고 **로그아웃**을 선택합니다. 로그아웃되면 공개 사이트로 돌아갑니다.
