# 로그인

<div class="article-intro">

B1.church에 로그인하면 시간선을 보고, 그룹을 관리하고, 온라인으로 헌금하고, 교회가 설정한 모든 도구를 사용할 수 있는 회원 포털에 접근할 수 있습니다. 이 문서는 로그인, 사용자 메뉴 탐색 및 로그아웃하는 방법을 다룹니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- ChurchApps 계정이 필요합니다. [시작하기](../getting-started/)를 참조하여 계정을 만드세요.
- 교회의 B1.church URL이 필요합니다(예: https://yourchurchname.b1.church).

</div>

## 로그인 방법

1. 교회의 B1.church URL로 이동하세요(예: https://yourchurchname.b1.church).
2. 헤더의 우측 상단에 있는 **Login** 버튼을 클릭하세요.
3. **이메일 주소**와 **비밀번호**를 입력하세요.
4. **Sign In**을 클릭하세요.

로그인 후 로그인을 클릭하기 전에 있던 페이지로 돌아갑니다. 로그아웃 상태에서 회원 포털 페이지로 직접 이동하면 로그인 후 자동으로 해당 페이지로 반환됩니다.

## 회원 포털 접근하기

로그인되면 페이지의 우측 상단에 있는 **이름**을 클릭하고 메뉴에서 **Member Portal**을 선택하여 회원 포털에 접근할 수 있습니다. 회원 포털은 [time line](../community/timeline.md), [groups](../groups/), [giving](../giving/) 등을 포함한 모든 개인 교회 도구가 있는 곳입니다.

## 사용자 메뉴

로그인 후 이름이 우측 상단의 칩으로 나타납니다. 이를 클릭하여 다음 옵션이 있는 메뉴를 열으세요:

- **Member Portal** -- 개인 회원 포털 대시보드로 이동하세요.
- **Edit Profile** -- 계정 설정을 업데이트하거나 교회 프로필을 편집하세요.
- **Logout** -- 계정에서 로그아웃하세요.

## 관리자 접근

적절한 권한이 있는 교회 관리자인 경우 사용자 메뉴에서 **Admin Portal** 옵션도 볼 수 있습니다. 이를 클릭하면 B1 Admin으로 이동하여 교회의 설정, 사람, 그룹 및 콘텐츠를 관리할 수 있습니다.

## 비밀번호를 잊으셨나요?

비밀번호를 기억하지 못하는 경우:

1. 로그인 페이지에서 **Forgot Password**를 클릭하세요.
2. ChurchApps 계정에 연결된 이메일 주소를 입력하고 제출하세요.
3. **6자리 확인 코드**가 포함된 메시지를 이메일로 확인하세요. 코드는 15분 동안 유효합니다.
4. 확인 화면에서 코드를 입력하여 로그인하세요.
5. 로그인 후 프로필을 방문하여 새 비밀번호를 선택하세요.

:::tip
코드가 몇 분 내에 도착하지 않으면 스팸 폴더를 확인하세요. 5번 틀린 시도 후 코드가 잠깁니다 -- 이 경우 비밀번호 찾기 페이지에서 새 코드를 요청하세요.
:::

## 로그아웃

로그아웃하려면 우측 상단의 **이름**을 클릭하고 **Logout**을 선택하세요. 로그아웃되고 공개 사이트로 돌아갑니다.
