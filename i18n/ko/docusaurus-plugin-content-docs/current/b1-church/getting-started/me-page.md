# 내 페이지

<div class="article-intro">

**내 페이지**는 회원 포털의 개인 대시보드입니다. 여기서 당신을 위해 예정된 모든 것을 시간 순서로 한눈에 볼 수 있습니다. 봉사 할당, 이벤트 등록, 그룹의 예정된 이벤트, 그리고 가장 최근 알림까지 모두 표시됩니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- [로그인](./logging-in.md)해야 합니다. 내 페이지는 로그인한 회원에게만 표시됩니다.

</div>

## 내 페이지 열기

포털 메뉴를 열고 **내 페이지**를 탭합니다. 페이지가 로드되고 개인 정보가 섹션으로 구성됩니다.

## 예정 사항

**예정 사항** 섹션은 날짜 순서로 앞서 있는 모든 항목을 나열하며, 각 항목에는 유형 칩 태그가 지정됩니다.

- **봉사** -- 예정된 예배 계획의 봉사 직책입니다. 하나를 탭하여 계획을 엽니다.
- **등록** -- [등록한](../events/registering) 이벤트입니다. 탭하여 [등록](../events/my-registrations)으로 이동합니다.
- **이벤트** -- 소속된 그룹의 예정된 이벤트입니다. 하나를 탭하여 그룹의 이벤트를 엽니다.

예정된 항목이 없으면 "예정된 항목 없음"이 표시됩니다. 봉사 시간, 이벤트, 등록이 예정되면 여기에 표시됩니다.

## 최근 알림

예정 사항 아래 **최근 알림** 섹션에는 포털의 각 영역을 열지 않고도 최신 상황을 파악할 수 있도록 최신 알림이 표시됩니다.

## 바로가기

교회에서 예약 가능한 실이나 리소스를 보유한 경우 내 페이지에 **바로가기** 섹션도 표시됩니다.

- **이벤트 요청하기** -- 필요한 실과 장비가 있는 이벤트를 개최하도록 요청합니다. [이벤트 및 실 요청하기](../events/requesting-events)를 참조하세요.
- **내 요청** -- 요청 상태를 추적하고 대기 중인 요청을 취소합니다.

교회에서 실이나 리소스를 설정하지 않은 경우 이러한 바로가기는 숨겨집니다.
