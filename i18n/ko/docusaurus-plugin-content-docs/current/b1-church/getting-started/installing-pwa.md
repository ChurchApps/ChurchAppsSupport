# 앱으로 설치 (PWA)

<div class="article-intro">

B1.church는 Progressive Web App (PWA)로 설치할 수 있으며, 이를 통해 네이티브 앱처럼 휴대폰의 홈 화면에 교회의 멤버 포털을 추가할 수 있습니다. 이는 App Store나 Google Play를 거치지 않고 풀 스크린, 앱과 같은 환경을 제공합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교회의 B1.church URL이 필요합니다: https://yourchurchname.b1.church
- 지원 브라우저를 사용하세요: iOS에서는 **Safari** 또는 Android에서 **Chrome**

</div>

## PWA란?

Progressive Web App (PWA)는 기기에 설치할 수 있고 네이티브 앱처럼 작동하는 웹사이트입니다. B1.church를 PWA로 설치할 때 다음을 수행합니다:

- 브라우저 네비게이션 바 없이 풀 스크린으로 열기
- 교회의 아이콘과 이름으로 홈 화면에 나타남
- 교회의 브랜딩 및 테마 색상으로 부드럽게 작동

:::tip
PWA는 App Store나 Google Play에서 별도의 앱을 설치하지 않으려는 경우 좋은 옵션입니다. 브라우저에서 얻는 것과 동일한 멤버 포털 경험을 더 편리한 패키지로 제공합니다.
:::

## 설치 위치

교회의 B1.church 사이트의 **모든 페이지**에서 PWA를 설치할 수 있습니다 - 공개 홈 페이지, 설교 페이지, 로그인 화면 또는 멤버 포털의 어느 곳이든. 기억해야 할 별도의 "설치 URL"은 없습니다. 웹에서 교회를 방문할 수 있으면 거기서 앱을 설치할 수 있습니다.

설치된 앱을 시작할 때 로그인할 준비가 된 멤버 대시보드로 직접 열립니다.

## iPhone에 설치 (iOS — Safari)

1. iPhone에서 **Safari**를 열고 https://yourchurchname.b1.church로 이동합니다 (yourchurchname을 교회의 URL 슬러그로 바꿉니다).
2. 화면 아래의 **공유** 버튼(위쪽을 가리키는 화살표가 있는 상자)을 탭합니다.
3. 공유 시트에서 아래로 스크롤하고 **홈 화면에 추가**를 탭합니다.
4. 이름은 기본적으로 교회 이름으로 설정됩니다. 우측 상단의 **추가**를 탭합니다.

앱 아이콘이 홈 화면에 나타납니다. 이를 탭하여 풀 스크린 모드에서 멤버 포털을 엽니다.

:::info
iOS의 PWA 설치에는 Safari가 필요합니다. iPhone의 Chrome 또는 다른 브라우저를 사용하고 있으면 먼저 Safari로 전환하세요.
:::

## Android에 설치 (Chrome)

1. Android 기기에서 **Chrome**을 열고 https://yourchurchname.b1.church로 이동합니다.
2. Chrome이 자동으로 화면 아래에 **"홈 화면에 추가"** 배너를 표시하거나 우측 상단의 3점 메뉴(⋮)를 탭하고 **홈 화면에 추가**를 선택합니다.
3. 메시지가 표시될 때 **추가**를 탭합니다.

앱 아이콘이 홈 화면에 나타납니다. 이를 탭하여 멤버 포털을 엽니다.

## 설치 후

설치되면 PWA는 멤버 포털로 직접 열리며 전용 풀 스크린 창에서 열립니다. 다음을 수행할 수 있습니다:

- 기존 ChurchApps 계정으로 로그인
- 교회가 활성화한 모든 멤버 기능 - 그룹, 기부금, 체크인, 봉사 계획 등에 액세스
- 홈 화면의 아이콘을 탭하여 언제든지 돌아올 수 있음

앱 아이콘은 읽지 않은 메시지가 있을 때 **알림 배지**(작은 숫자 배지)를 표시하므로 대기 중인 새로운 항목이 있는지 한눈에 볼 수 있습니다.

:::info
PWA는 인터넷 연결이 필요합니다. 일부 네이티브 앱과 달리 오프라인 사용을 위해 모든 콘텐츠를 캐시하지 않습니다.
:::

## 구형 B1 모바일 앱은 어떻게 되나요?

iOS 및 Android의 레거시 네이티브 **B1 Mobile** 앱은 사용 중단되었으며 더 이상 업데이트되지 않습니다. PWA는 향후 휴대폰에서 B1.church 멤버 포털에 액세스하는 지원되는 방법이며 네이티브 앱이 제공했던 모든 멤버 대면 기능을 포함합니다. 이전에 네이티브 앱을 설치했다면 편할 때 PWA로 전환하세요.

사용 중단된 앱에 대한 과거 설명서는 [B1 Mobile 참고 페이지](/docs/b1-mobile/)를 참조하세요.
