# 앱으로 설치하기(PWA)

<div class="article-intro">

B1.church는 Progressive Web App(PWA)로 설치를 지원하므로 교회 회원 포털을 기본 앱처럼 휴대폰의 홈 화면에 추가할 수 있습니다. 이는 App Store나 Google Play를 거치지 않고 전체 화면, 앱과 같은 경험을 제공합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 교회의 B1.church URL이 필요합니다: https://yourchurchname.b1.church
- 지원되는 브라우저를 사용하세요: iOS에서는 **Safari** 또는 Android에서는 **Chrome**

</div>

## PWA란 무엇인가요?

Progressive Web App(PWA)은 장치에 설치되고 기본 앱처럼 작동할 수 있는 웹사이트입니다. B1.church를 PWA로 설치하면:

- 브라우저 네비게이션 바 없이 전체 화면으로 열립니다
- 교회의 아이콘 및 이름으로 홈 화면에 나타납니다
- 교회의 브랜딩 및 테마 색상과 부드럽게 작동합니다

:::tip
PWA는 App Store나 Google Play에서 별도의 앱을 설치하지 않으려는 경우 훌륭한 옵션입니다. 브라우저에서 얻는 것과 동일한 회원 포털 경험을 더 편리한 패키지로 제공합니다.
:::

## 설치할 위치

교회의 B1.church 사이트의 **모든 페이지**에서 PWA를 설치할 수 있습니다 -- 공개 홈 페이지, 설교 페이지, 로그인 화면 또는 회원 포털 내의 어디든지. 기억해야 할 별도의 "설치 URL"은 없습니다. 웹에서 교회를 방문할 수 있으면 그곳에서 앱을 설치할 수 있습니다.

설치된 앱을 시작하면 회원 대시보드로 직접 열려 로그인할 준비가 됩니다.

## iPhone에 설치하기(iOS -- Safari)

1. iPhone에서 **Safari**를 열고 https://yourchurchname.b1.church로 이동하세요(yourchurchname을 교회의 URL 슬러그로 바꾸세요).
2. 화면 하단의 **Share** 버튼(위쪽을 가리키는 화살표가 있는 상자)을 탭하세요.
3. 공유 시트에서 스크롤하여 **Add to Home Screen**을 탭하세요.
4. 이름은 기본적으로 교회의 이름으로 설정됩니다. 우측 상단의 **Add**를 탭하세요.

앱 아이콘이 홈 화면에 나타납니다. 이를 탭하여 회원 포털을 전체 화면 모드로 열으세요.

:::info
iOS에서 PWA 설치는 Safari가 필요합니다. iPhone에서 Chrome 또는 다른 브라우저를 사용 중이면 먼저 Safari로 전환하세요.
:::

## Android에 설치하기(Chrome)

1. Android 기기에서 **Chrome**을 열고 https://yourchurchname.b1.church로 이동하세요.
2. Chrome이 자동으로 화면 하단에 **"Add to Home screen"** 배너를 표시하거나, 우측 상단의 세 점 메뉴(⋮)를 탭하고 **Add to Home screen**을 선택하세요.
3. 메시지가 표시되면 **Add**를 탭하세요.

앱 아이콘이 홈 화면에 나타납니다. 이를 탭하여 회원 포털을 전체 화면 모드로 열으세요.

## 설치 후

설치되면 PWA는 회원 포털에 전용 전체 화면 창으로 직접 열립니다. 다음을 할 수 있습니다:

- 기존 ChurchApps 계정으로 로그인하세요
- 교회가 활성화한 모든 회원 기능에 접근하세요 -- 그룹, 헌금, 체크인, 봉사 계획 등
- 홈 화면의 아이콘을 탭하여 언제든지 돌아오세요

:::info
PWA는 인터넷 연결이 필요합니다. 일부 기본 앱과 달리 오프라인 사용을 위해 모든 콘텐츠를 캐시하지 않습니다.
:::

## 구형 B1 Mobile 앱은 어떻게 되었나요?

iOS 및 Android의 기본 **B1 Mobile** 앱은 단종되었으며 더 이상 업데이트되지 않습니다. PWA는 앞으로 휴대폰에서 B1.church 회원 포털에 접근하는 지원되는 방법이며 기본 앱이 제공했던 모든 회원 대면 기능을 다룹니다. 이전에 기본 앱을 설치했으면 편리할 때에 PWA로 전환하세요.

더 이상 사용되지 않는 앱에 대한 기록 문서는 [B1 Mobile 참조 페이지](/docs/b1-mobile/)를 참조하세요.
