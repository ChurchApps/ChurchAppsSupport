# 앱으로 설치 (PWA)

<div class="article-intro">

B1.church는 Progressive Web App (PWA)로서의 설치를 지원하며, 네이티브 앱과 같이 당신의 휴대폰의 홈 화면에 당신의 교회의 회원 포털을 추가할 수 있습니다. 이것은 App Store 또는 Google Play를 거치지 않고 완전한 화면, 앱 같은 경험을 제공합니다.

</div>

<div class="prereqs">
<h4>시작하기 전에</h4>

- 당신의 교회의 B1.church 모바일 URL이 필요합니다: `https://yourchurchname.b1.church/mobile`
- 지원되는 브라우저를 사용하세요: iOS의 **Safari** 또는 Android의 **Chrome**

</div>

## PWA란?

Progressive Web App (PWA)는 당신의 기기에 설치될 수 있고 네이티브 앱처럼 작동하는 웹사이트입니다. 당신이 B1.church를 PWA로 설치할 때:

- 브라우저 네비게이션 바 없이 전체 화면으로 엽니다
- 당신의 교회의 아이콘 및 이름과 함께 홈 화면에 나타납니다
- 당신의 교회의 브랜딩 및 테마 색상으로 부드럽게 작동합니다

:::tip
PWA는 App Store 또는 Google Play에서 별도의 앱을 설치하지 않으려면 좋은 옵션입니다. 이것은 브라우저에서 얻는 것과 같은 회원 포털 경험을 더 편리한 패키지로 제공합니다.
:::

## iPhone에 설치 (iOS -- Safari)

1. 당신의 iPhone에서 **Safari**를 열고 `https://yourchurchname.b1.church/mobile`으로 이동합니다 (`yourchurchname`을 당신의 교회의 URL slug로 바꿉니다).
2. 화면 맨 아래의 **공유** 버튼 (화살표를 가리키는 상자)을 눌러봅니다.
3. 공유 시트에서 아래로 스크롤하고 **홈 화면에 추가**를 눌러봅니다.
4. 이름은 당신의 교회의 이름으로 기본값으로 설정됩니다. 오른쪽 상단에서 **추가**를 눌러봅니다.

앱 아이콘이 홈 화면에 나타날 것입니다. 전체 화면 모드에서 회원 포털을 열려면 이를 누릅니다.

:::info
iOS의 PWA 설치는 Safari가 필요합니다. iPhone에서 Chrome 또는 다른 브라우저를 사용하고 있다면 먼저 Safari로 전환합니다.
:::

## Android에 설치 (Chrome)

1. 당신의 Android 기기에서 **Chrome**을 열고 `https://yourchurchname.b1.church/mobile`으로 이동합니다.
2. Chrome은 자동으로 화면 아래에 **"홈 화면에 추가"** 배너를 표시하거나 오른쪽 상단의 세 점 메뉴 (⋮)를 눌러 **홈 화면에 추가**를 선택합니다.
3. 메시지가 표시되면 **추가**를 눌러봅니다.

앱 아이콘이 홈 화면에 나타날 것입니다. 회원 포털을 열려면 이를 누릅니다.

## 설치 후

일단 설치되면, PWA는 브라우저에서 모바일 사이트를 방문하는 것과 같이 작동하지만 전용 전체 화면 창에서 작동합니다. 당신은 다음을 할 수 있습니다:

- 당신의 기존 ChurchApps 계정으로 로그인
- 당신의 교회가 활성화한 모든 회원 기능에 접근 -- 그룹, 헌금, 체크인, 봉사 계획 등
- 홈 화면의 아이콘을 눌러 언제든지 돌아올 수 있습니다

:::info
PWA는 인터넷 연결이 필요합니다. 일부 네이티브 앱과 달리 오프라인 사용을 위해 모든 콘텐츠를 캐시하지 않습니다.
:::

## PWA vs. 네이티브 앱

두 옵션 모두 B1.church 회원 포털에 대한 접근을 제공합니다. 올바른 선택은 당신의 선호도에 따릅니다:

| | **PWA** | **B1 Mobile 앱** |
|---|---|---|
| 설치 | 브라우저에서 홈 화면에 추가 | App Store / Google Play에서 다운로드 |
| 업데이트 | 자동 (앱 스토어 업데이트가 필요 없음) | App Store / Google Play를 통해 |
| 푸시 알림 | 지원되지 않음 | 지원됨 |
| 플랫폼 | 브라우저가 있는 모든 기기 | iOS 및 Android |

:::tip
당신의 교회가 그룹 대화를 사용하고 새 메시지 및 활동에 대한 푸시 알림을 원하면 [B1 Mobile 앱](/docs/b1-mobile/)을 대신 사용합니다. 푸시 알림은 현재 PWA에서 지원되지 않습니다.
:::
