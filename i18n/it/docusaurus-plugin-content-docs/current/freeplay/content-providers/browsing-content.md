# Sfogliare e scaricare contenuti

<div class="article-intro">

Una volta connesso a un fornitore di contenuti, puoi sfogliare la sua libreria, scaricare file e riprodurli sul tuo TV. Il browser dei contenuti fornisce una griglia visiva di cartelle e file che navighi con il telecomando TV.

</div>

<div class="prereqs">
<h4>Prima di iniziare</h4>

- Connettiti ad almeno un fornitore di contenuti -- vedi [Connettersi ai fornitori](./connecting-providers)
- Tieni pronto il telecomando TV per la navigazione

</div>

## Il browser dei contenuti

Il browser dei contenuti visualizza gli elementi in una griglia a 3 colonne di schede. Ogni scheda mostra un'immagine in miniatura e un titolo. Ci sono due tipi di elementi:

- **Cartelle** - Rappresentate da una miniatura o un'icona di cartella. Seleziona una cartella per vederne il contenuto.
- **File** - Elementi multimediali individuali (video o immagini). I file video mostrano un'icona di riproduzione sovrapposta sulla miniatura.

L'intestazione nella parte superiore dello schermo mostra il nome della cartella corrente, o il nome del fornitore se sei al livello principale.

## Navigare nelle cartelle

1. Usa il pad direzionale sul telecomando per evidenziare una cartella
2. Premi **Seleziona** per aprirla
3. Il browser carica il contenuto della cartella, che può essere altre cartelle o file riproducibili
4. Premi **Indietro** sul telecomando per salire di un livello

Puoi approfondire quanto la struttura dei contenuti del fornitore lo consente. Il browser tiene traccia del percorso delle cartelle così puoi navigare indietro attraverso ogni livello.

## Scaricare e riprodurre contenuti

Quando selezioni una cartella che contiene file riproducibili (o selezioni un singolo file), FreePlay passa alla schermata di download:

1. Il titolo del contenuto appare sullo schermo con un'immagine di copertina dalla cartella selezionata
2. Una barra di avanzamento mostra **Download dell'elemento X di Y** man mano che i file vengono salvati sul dispositivo
3. Una volta completato il download, appare il pulsante **Avvia** con un'icona di riproduzione
4. Premi **Seleziona** per avviare la riproduzione

:::tip
I contenuti scaricati vengono memorizzati nella cache del TV. Se riproduci lo stesso contenuto in seguito, si carica istantaneamente senza riscaricare.
:::

## Riprodurre contenuti del fornitore

La riproduzione funziona allo stesso modo della riproduzione delle lezioni. Usa questi controlli:

- **Freccia destra** - Elemento successivo
- **Freccia sinistra** - Elemento precedente
- **Seleziona / Play-Pausa** - Metti in pausa o riprendi il video
- **Freccia su** - Apri il selettore dei messaggi per saltare a un elemento specifico
- **Indietro** - Torna al browser dei contenuti

Quando esci dal lettore, torni al browser dei contenuti al livello principale del fornitore corrente.

:::info
Per un riferimento completo di tutti i controlli del lettore, vedi [Riprodurre le lezioni](../classroom-mode/playing-lessons).
:::

## Passare da un fornitore all'altro

I fornitori connessi appaiono come elementi nella barra laterale a sinistra. Per passare a un fornitore diverso:

1. Premi la freccia **Sinistra** sul telecomando per espandere la barra laterale
2. Seleziona il fornitore che vuoi sfogliare
3. Il browser dei contenuti si aggiorna con la libreria di quel fornitore

:::warning
Se una cartella mostra "Nessun contenuto disponibile", il fornitore non ha elementi a quel livello. Prova a navigare in una cartella diversa o controlla più tardi.
:::
