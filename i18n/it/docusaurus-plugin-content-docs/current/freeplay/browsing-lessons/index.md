# Sfogliare le lezioni

<div class="article-intro">

FreePlay include un browser di lezioni integrato che ti permette di esplorare la libreria completa di programmi, studi e lezioni disponibili. Questo è utile quando vuoi scegliere manualmente una lezione da riprodurre anziché affidarti a un programma dell'aula o a un piano.

</div>

## Il flusso di navigazione

Le lezioni sono organizzate in una gerarchia:

1. **Programmi** - Categorie di curriculum di primo livello (es. una serie per il ministero dei bambini)
2. **Studi** - Gruppi di lezioni all'interno di un programma (es. un'unità di più settimane)
3. **Lezioni** - Sessioni di lezione individuali con i propri contenuti
4. **Sedi** - Versioni diverse di una lezione adattate per gruppi di età o contesti specifici

Inizi selezionando un programma, poi approfondisci in uno studio, scegli una lezione e infine scegli una sede per avviare la riproduzione.

## Accedere al browser delle lezioni

Il browser delle lezioni è disponibile tramite la schermata **Sfoglia programmi**. Raggiungi questa schermata usando la barra laterale o selezionando l'opzione di navigazione corrispondente.

:::info
Sfogliare le lezioni richiede una connessione internet per caricare il catalogo dei programmi. Una volta selezionata una sede e avviata la riproduzione, il contenuto viene trasmesso direttamente senza un passaggio di download separato.
:::
