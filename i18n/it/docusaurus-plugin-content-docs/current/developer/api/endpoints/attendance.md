# Endpoint di Partecipazione

<div class="article-intro">

Il modulo Attendance gestisce i campus, i servizi, gli orari di servizio, le sessioni di partecipazione, le visite e le sessioni di visita. Fornisce l'infrastruttura per tracciare chi ha partecipato a quale servizio o riunione di gruppo, supporta i flussi di lavoro di registrazione e offre rapporti sulle tendenze e i riepiloghi della partecipazione.

</div>

Percorso di base: `/attendance`
