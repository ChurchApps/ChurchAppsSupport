# Cambio Chiesa

<div class="article-intro">

Se sei membro di più di una chiesa, B1 Mobile rende facile passare dall'una all'altra. Il tuo account funziona con tutte le chiese che utilizzano ChurchApps, quindi hai bisogno di un solo accesso.

</div>

<div class="prereqs">
<h4>Prima di Iniziare</h4>

- [Installa B1 Mobile](./installing) e [accedi](./logging-in) al tuo account

</div>

## Come Cambiare Chiesa

1. Apri il menu laterale toccando l'**icona hamburger** (tre linee orizzontali) nell'angolo in alto a sinistra.
2. Nell'intestazione del menu laterale, tocca il pulsante che mostra il nome della tua chiesa attuale (o **Trova una Chiesa** se non è selezionata nessuna chiesa).
3. Appare la schermata **Ricerca Chiesa**. Le chiese visitate di recente vengono mostrate in alto per un accesso rapido.
4. Tocca una chiesa recente per passare ad essa, oppure digita nel campo di ricerca per trovare una chiesa diversa.

Quando cambi chiesa, l'app si ricarica con il marchio, i link di navigazione e i contenuti di quella chiesa. La tua [dashboard](../dashboard/), i [gruppi](../groups/) e altri dati rifletteranno la chiesa appena selezionata.

## Ricerca di una Chiesa

Nella schermata Ricerca Chiesa, digita almeno 3 caratteri del nome della chiesa nel campo di ricerca. Le chiese corrispondenti appaiono in un elenco sottostante. Tocca una chiesa per selezionarla.

:::info
Se non appaiono risultati, prova un'ortografia diversa o una porzione più breve del nome della chiesa. La ricerca confronta i nomi delle chiese registrate con ChurchApps.
:::

## Chiese Recenti

La schermata Ricerca Chiesa ricorda le chiese che hai visitato in precedenza. Quando non hai digitato una query di ricerca, le tue chiese recenti vengono visualizzate automaticamente. Questo rende rapido tornare a una chiesa che usi regolarmente.

:::tip
Non è necessario disconnettersi e riconnettersi per cambiare chiesa. Il tuo unico account ChurchApps funziona con tutte le tue chiese. Basta selezionare una chiesa diversa dalla schermata Ricerca Chiesa e sei pronto.
:::
