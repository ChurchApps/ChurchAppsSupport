# File

<div class="article-intro">

La pagina File è il repository centrale per tutti i media e i documenti usati sul tuo sito web della chiesa. Da qui puoi caricare, organizzare, navigare e gestire ogni immagine, documento e file multimediale di cui il tuo sito ha bisogno.

</div>

<div class="prereqs">
<h4>Prima di iniziare</h4>

- Completa la [Configurazione iniziale](initial-setup) per il tuo sito web
- Ottimizza le immagini prima di caricarle per tempi di caricamento più veloci
- Prepara i tuoi file nei formati web comuni (PNG, JPG, PDF, ecc.)

</div>

## Caricare file

1. Naviga alla sezione **Website** in B1 Admin e clicca la scheda **Files**.
2. Trascina e rilascia i file direttamente sull'area di caricamento, o clicca il pulsante di caricamento per navigare nel tuo computer.
3. I tuoi file verranno caricati e appariranno immediatamente nella libreria file.

:::tip
Per i migliori risultati, ottimizza le immagini prima di caricarle. File di dimensioni ridotte aiutano il tuo sito web a caricarsi più velocemente per i visitatori. Strumenti come TinyPNG o Squoosh possono comprimere le immagini senza perdita di qualità percepibile.
:::

## Organizzare con cartelle

1. Crea cartelle per raggruppare file correlati -- ad esempio, "Grafiche sermoni", "Banner eventi" o "Foto staff".
2. Sposta file tra le cartelle per mantenere ordinata la tua libreria man mano che cresce.
3. Usa nomi di cartelle chiari e descrittivi così il tuo team può trovare i file rapidamente.

## Navigare e cercare

- Scorri la libreria file per vedere le miniature di tutti i tuoi file caricati.
- Usa la funzione **search** per trovare file per nome quando la tua libreria diventa grande.
- Clicca su qualsiasi file per visualizzare i suoi dettagli, incluso nome file, dimensione e URL.

## Usare i file sul tuo sito web

1. Clicca su un file per visualizzare i suoi dettagli.
2. **Copia l'URL del file** per usarlo nelle tue pagine web. Puoi incollare questo URL in elementi immagine, link di download o ovunque accetti un URL file.
3. I file caricati qui sono disponibili per l'uso su tutte le tue pagine web e blocchi di contenuto.

:::info
Quando costruisci pagine in [Gestione pagine](managing-pages), puoi riferire qualsiasi file caricato qui tramite il suo URL. Questo mantiene tutti i tuoi media organizzati in una posizione centrale.
:::

## Eliminare file

1. Seleziona il file che vuoi rimuovere.
2. Clicca il pulsante **Delete** per rimuoverlo permanentemente dalla tua libreria.
3. Assicurati che il file non sia attualmente in uso su nessuna pagina prima di eliminarlo, poiché questo romperà qualsiasi link o immagine che lo referenzia.

:::warning
I file eliminati non possono essere recuperati. Verifica che un file non sia più in uso su nessuna pagina prima di rimuoverlo.
:::
