# Report

<div class="article-intro">

La pagina Report ti offre un accesso rapido a report predefiniti che ti aiutano a comprendere i dati della tua chiesa. Usa questi report per monitorare le tendenze di partecipazione, controllare l'attività delle donazioni e rimanere in contatto con la tua congregazione attraverso le celebrazioni dei compleanni.

</div>

## Report disponibili

La pagina Report mostra cinque schede di report. Fai clic su qualsiasi scheda per aprire il report con i suoi filtri.

- **[Report Compleanni](./birthday-report)** -- Visualizza i prossimi compleanni dei membri della tua congregazione.
- **[Tendenza Presenze](./attendance-reports)** -- Scopri come cambiano i modelli di partecipazione nel tempo nei tuoi servizi.
- **[Presenze per Gruppo](./attendance-reports)** -- Visualizza i totali delle presenze per gruppo in un intervallo di date selezionato.
- **[Presenze Giornaliere per Gruppo](./attendance-reports)** -- Ottieni un'analisi giorno per giorno dei dati di presenza dei gruppi.
- **[Riepilogo Donazioni](./donation-summary)** -- Esamina i totali delle donazioni e le tendenze per la pianificazione finanziaria.

## Utilizzo dei Report

1. Naviga su **Reports** dal menu principale.
2. Fai clic su una **scheda di report** per aprirla.
3. Usa i **filtri** nella parte superiore di ogni report per restringere i dati per intervallo di date, gruppo, servizio o altri criteri.
4. Esamina i risultati visualizzati nel report.

:::tip
Esegui i report regolarmente per rimanere aggiornato sulle tendenze di partecipazione, i prossimi compleanni e i modelli di donazione. Questi dati ti aiutano a pianificare i ministeri e le attività di sensibilizzazione in modo più efficace.
:::

:::info
I dati dei report dipendono da un inserimento dati costante. Assicurati che il tuo team stia [registrando le presenze](../attendance/tracking-attendance.md), [registrando le donazioni](../donations/recording-donations.md) e mantenendo aggiornati i [registri delle persone](../people/adding-people.md) per ottenere i risultati più accurati.
:::
