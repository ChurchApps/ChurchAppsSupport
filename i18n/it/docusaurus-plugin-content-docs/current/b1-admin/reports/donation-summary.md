# Riepilogo donazioni

<div class="article-intro">

Il report Riepilogo donazioni ti offre un quadro chiaro dell'attività di donazione della tua chiesa. Usalo per tracciare i totali delle donazioni, monitorare le tendenze delle offerte e preparare report finanziari per il tuo team dirigenziale.

</div>

<div class="prereqs">
<h4>Prima di iniziare</h4>

- Assicurati che le donazioni vengano [registrate costantemente](../donations/recording-donations.md) nel tuo sistema
- Hai bisogno dei [permessi](../settings/roles-permissions.md) appropriati per accedere ai report finanziari

</div>

## Eseguire il report

1. Naviga su **Reports** dal menu principale.
2. Clicca la scheda **Donation Summary**.
3. Imposta l'**intervallo di date** per definire il periodo di reporting.
4. Applica filtri aggiuntivi come **fondo** o altri criteri per restringere i risultati.
5. Il report mostrerà il riepilogo delle offerte per il periodo selezionato.

## Cosa mostra il report

Il report Riepilogo donazioni include:

- **Totale donazioni** -- L'importo complessivo delle offerte per il periodo selezionato.
- **Numero di donatori** -- Il numero di donatori unici che hanno contribuito.
- **Importi per fondo** -- Una suddivisione delle offerte tra i diversi fondi, se applicabile.

## Usi pratici

- **Pianificazione finanziaria** -- Esamina le tendenze delle offerte per prevedere le entrate future e pianificare il tuo budget.
- **Report per il consiglio** -- Genera riepiloghi per le riunioni regolari del consiglio o della dirigenza con dati chiari sulle offerte.
- **Revisioni di fine anno** -- Esegui il report per l'intero anno per preparare riepiloghi finanziari annuali.
- **Tracciamento campagne** -- Filtra per fondo per tracciare i progressi su campagne speciali di raccolta o progetti edilizi.
- **Analisi stagionale** -- Confronta le offerte tra periodi diversi per comprendere i pattern stagionali.

:::tip
Esegui questo report mensilmente per restare informato sulla salute finanziaria della tua chiesa. Confrontare i dati mese su mese ti aiuta a individuare le tendenze in anticipo.
:::

:::warning
I dati delle donazioni provengono dai contributi registrati nel tuo sistema. Assicurati che le donazioni vengano inserite costantemente -- sia tramite donazioni online, elaborazione assegni o inserimento manuale -- per i report più accurati. Vedi [Registrazione donazioni](../donations/recording-donations.md) per i dettagli.
:::
