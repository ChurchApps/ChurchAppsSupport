# Report Presenze

<div class="article-intro">

B1 Admin fornisce tre report sulle presenze per aiutarti a capire come le persone partecipano ai tuoi servizi e gruppi. Ogni report offre una prospettiva diversa sui dati di partecipazione, dalle tendenze di alto livello alle analisi giornaliere.

</div>

<div class="prereqs">
<h4>Prima di iniziare</h4>

- Assicurati che le presenze vengano [registrate in modo costante](../attendance/tracking-attendance.md) per i tuoi servizi e gruppi
- Verifica che i tuoi [gruppi](../groups/creating-groups.md) e servizi siano configurati in B1 Admin
- Hai bisogno dei [permessi](../settings/roles-permissions.md) appropriati per accedere ai report

</div>

## Tendenza Presenze

Il report Tendenza Presenze mostra come la partecipazione cambia nel tempo per i tuoi servizi.

1. Naviga su **Reports** e fai clic sulla scheda **Attendance Trend**.
2. Imposta l'**intervallo di date** per definire il periodo di tempo.
3. Seleziona un **servizio** o **gruppo** per filtrare i risultati.
4. Il report visualizza una linea di tendenza che mostra i conteggi delle presenze nel periodo selezionato.

Questo report Ã¨ utile per individuare modelli come cali stagionali, tendenze di crescita o l'impatto di eventi speciali.

## Presenze per Gruppo

Il report Presenze per Gruppo mostra i totali delle presenze per gruppo in un intervallo di date selezionato.

1. Naviga su **Reports** e fai clic sulla scheda **Group Attendance**.
2. Imposta l'**intervallo di date** per il report.
3. Seleziona il/i **gruppo/i** che desideri esaminare.
4. Il report mostra le cifre totali delle presenze per ogni gruppo.

Usa questo report per confrontare la partecipazione tra i gruppi e identificare quali gruppi stanno crescendo o necessitano di attenzione.

## Presenze Giornaliere per Gruppo

Il report Presenze Giornaliere per Gruppo fornisce un'analisi giorno per giorno dei dati di partecipazione dei tuoi gruppi.

1. Naviga su **Reports** e fai clic sulla scheda **Daily Group Attendance**.
2. Imposta l'**intervallo di date** per il report.
3. Seleziona il/i **gruppo/i** che desideri esaminare.
4. Il report mostra i numeri di partecipazione per ogni singolo giorno nell'intervallo.

Questo report fornisce dettagli granulari, utili per comprendere le variazioni settimana per settimana o per identificare giorni specifici con presenze insolitamente alte o basse.

:::tip
Usa il report Tendenza Presenze per una panoramica di alto livello e il report Presenze Giornaliere per Gruppo quando hai bisogno di approfondire date specifiche.
:::

## Usi pratici

- **Pianificazione** -- Usa le tendenze di partecipazione per pianificare posti a sedere, personale e risorse per i prossimi servizi.
- **Sensibilizzazione** -- Identifica precocemente i modelli di calo delle presenze per poter contattare i membri.
- **Report per il consiglio** -- Includi i dati sulle presenze nei tuoi report periodici alla leadership per mostrare la salute del ministero.
- **Valutazione eventi** -- Confronta le presenze prima e dopo eventi speciali per misurarne l'impatto.

:::warning
I dati sulle presenze vengono registrati attraverso i processi di check-in dei tuoi gruppi e servizi. Se le presenze non vengono registrate in modo costante, i tuoi report non rifletteranno accuratamente la partecipazione effettiva. Consulta [Registrazione Presenze](../attendance/tracking-attendance.md) per le istruzioni di configurazione.
:::
