# Ricerca di Persone

<div class="article-intro">

La pagina **Persone** visualizza la tua directory di chiesa in una tabella ricercabile e ordinabile. Puoi trovare rapidamente chiunque nella tua congregazione, personalizzare quali informazioni vengono visualizzate ed esportare i tuoi risultati. Una ricerca efficiente è essenziale per i compiti quotidiani di amministrazione della chiesa come il follow-up con i visitatori, la preparazione di elenchi di contatti e la gestione dei record dei membri.

</div>

<div class="prereqs">
<h4>Prima di Iniziare</h4>

- Hai bisogno di un account B1 Admin attivo con autorizzazione per visualizzare le persone. Vedi [Ruoli e Autorizzazioni](roles-permissions.md) se non sei sicuro del tuo livello di accesso.
- La tua directory di chiesa dovrebbe avere persone in essa. Se non hai ancora aggiunto nessuno, vedi [Aggiunta di Persone](adding-people.md) o [Importazione di Dati](importing-data.md).

</div>

## Ricerca Rapida

La barra di ricerca in cima alla pagina Persone ti consente di trovare i membri in tempo reale:

1. Fai clic sulla **casella di ricerca** in cima alla pagina Persone.
2. Inizia a digitare un nome, email o altra parola chiave.
3. I risultati filtreranno automaticamente mentre digiti (c'è un breve ritardo di circa mezzo secondo in modo che la ricerca non si attivi ad ogni pressione di tasto).
4. La tabella sottostante si aggiorna per mostrare solo i risultati corrispondenti.

:::tip
Non devi premere Invio. La ricerca viene eseguita automaticamente dopo che smetti di digitare.
:::

## Ordinamento dei Risultati

Puoi ordinare la directory facendo clic su qualsiasi intestazione di colonna nella tabella:

1. Fai clic su un'**intestazione di colonna** (ad esempio, **Nome** o **Email**) per ordinare in base a quella colonna.
2. Fai clic sulla stessa intestazione di nuovo per invertire l'ordine di ordinamento.

Questo rende facile trovare le persone alfabeticamente, per età o per qualsiasi altra colonna visibile.

## Personalizzazione delle Colonne

Non è necessario che ogni informazione sia visibile contemporaneamente. Puoi scegliere quali colonne appaiono nella tabella:

1. Cerca il **menu a discesa del selettore di colonna** vicino alla cima della tabella.
2. Seleziona o deseleziona le colonne per mostrarle o nasconderle. Le colonne disponibili includono:
   - **Foto**
   - **Nome**
   - **Email**
   - **Telefono**
   - **Indirizzo**
   - **Data di Nascita**
   - **Età**
   - **Genere**
   - **Stato di Iscrizione**
3. La tabella si aggiorna immediatamente per riflettere le tue selezioni.

:::info
Le tue scelte di colonna influiscono su ciò che viene incluso quando esporti a CSV. Personalizza le colonne prima di esportare per ottenere esattamente i dati di cui hai bisogno.
:::

## Impaginazione

Quando la tua directory ha molti record, i risultati sono suddivisi in più pagine. Usa i **controlli di impaginazione** in fondo alla tabella per spostarti tra le pagine. La pagina corrente e il conteggio totale dei record vengono visualizzati in modo che tu sappia sempre dove sei nell'elenco.

:::tip
Se desideri vedere più risultati contemporaneamente, affina la tua ricerca per restringere l'elenco piuttosto che spostarti tra una directory di grandi dimensioni.
:::

## Esportazione dei Risultati di Ricerca

Puoi scaricare i tuoi risultati di ricerca attuali come file CSV in qualsiasi momento:

1. Applica qualsiasi ricerca o filtri desiderati.
2. Personalizza le tue colonne per includere i dati di cui hai bisogno.
3. Fai clic sul pulsante **Esporta**.
4. Un file CSV scaricherà sul tuo computer, pronto per essere aperto in Excel, Google Sheets o qualsiasi applicazione di foglio di calcolo.

Per maggiori dettagli sull'esportazione, vedi [Esportazione di Dati](./exporting-data.md).

:::tip
Per query più avanzate -- come trovare tutti coloro che non hanno frequentato negli ultimi tre mesi -- prova la funzione [Ricerca AI](./ai-search.md), che ti consente di cercare utilizzando domande in linguaggio naturale.
:::
