# Ricerca persone

<div class="article-intro">

La pagina **Persone** mostra l'elenco della tua chiesa in una tabella ricercabile e ordinabile. Puoi trovare rapidamente chiunque nella tua congregazione, personalizzare quali informazioni vengono mostrate ed esportare i risultati. Una ricerca efficiente è essenziale per le attività quotidiane di amministrazione della chiesa come il follow-up con i visitatori, la preparazione di liste di contatti e la gestione dei registri dei membri.

</div>

<div class="prereqs">
<h4>Prima di iniziare</h4>

- Hai bisogno di un account B1 Admin attivo con il permesso di visualizzare le persone. Consulta [Ruoli e permessi](roles-permissions.md) se non sei sicuro del tuo livello di accesso.
- Il tuo elenco della chiesa dovrebbe contenere delle persone. Se non hai ancora aggiunto nessuno, consulta [Aggiungere persone](adding-people.md) o [Importare dati](importing-data.md).

</div>

## Ricerca rapida

La barra di ricerca in cima alla pagina Persone ti permette di trovare membri in tempo reale:

1. Clicca sulla **casella di ricerca** in cima alla pagina Persone.
2. Inizia a digitare un nome, email o altra parola chiave.
3. I risultati si filtreranno automaticamente mentre digiti (c'è un breve ritardo di circa mezzo secondo per evitare che la ricerca si attivi ad ogni tasto premuto).
4. La tabella sottostante si aggiorna per mostrare solo i risultati corrispondenti.

:::tip
Non è necessario premere Invio. La ricerca si avvia automaticamente dopo che smetti di digitare.
:::

## Ordinamento dei risultati

Puoi ordinare l'elenco cliccando su qualsiasi intestazione di colonna nella tabella:

1. Clicca su un'**intestazione di colonna** (ad esempio, **Nome** o **Email**) per ordinare per quella colonna.
2. Clicca di nuovo sulla stessa intestazione per invertire l'ordine.

Questo rende facile trovare persone in ordine alfabetico, per età o per qualsiasi altra colonna visibile.

## Personalizzazione delle colonne

Non tutte le informazioni devono essere visibili contemporaneamente. Puoi scegliere quali colonne appaiono nella tabella:

1. Cerca il **menu a discesa del selettore colonne** vicino alla parte superiore della tabella.
2. Seleziona o deseleziona le colonne per mostrarle o nasconderle. Le colonne disponibili includono:
   - **Foto**
   - **Nome**
   - **Email**
   - **Telefono**
   - **Indirizzo**
   - **Data di nascita**
   - **Età**
   - **Genere**
   - **Stato di appartenenza**
3. La tabella si aggiorna immediatamente per riflettere le tue selezioni.

:::info
Le tue scelte di colonne influenzano cosa viene incluso quando esporti in CSV. Personalizza le colonne prima di esportare per ottenere esattamente i dati di cui hai bisogno.
:::

## Esportazione dei risultati di ricerca

Puoi scaricare i risultati della ricerca corrente come file CSV in qualsiasi momento:

1. Applica le ricerche o i filtri desiderati.
2. Personalizza le colonne per includere i dati necessari.
3. Clicca sul pulsante **Esporta**.
4. Un file CSV verrà scaricato sul tuo computer, pronto per essere aperto in Excel, Google Sheets o qualsiasi applicazione di fogli di calcolo.

Per maggiori dettagli sull'esportazione, consulta [Esportare dati](./exporting-data.md).

:::tip
Per query più avanzate -- come trovare tutti coloro che non hanno partecipato negli ultimi tre mesi -- prova la funzione [Ricerca AI](./ai-search.md), che ti permette di cercare usando domande in linguaggio naturale.
:::
