# Stampa Directory

<div class="article-intro">

La funzione Stampa Directory genera una directory dei membri formattata e stampabile per la tua chiesa. Organizza i membri per nucleo familiare, include informazioni di contatto e foto, ed è progettata per sembrare professionale quando stampata su carta di formato standard.

</div>

<div class="prereqs">
<h4>Prima di Iniziare</h4>

- Hai bisogno di un account B1 Admin attivo con autorizzazione per visualizzare le persone.
- Le persone nella tua directory dovrebbero avere foto e informazioni di contatto compilate per i migliori risultati.

</div>

## Apertura della Stampa Directory

1. Accedi a **Persone** in B1 Admin.
2. Fai clic sull'**icona di stampa** (icona della stampante) nella barra degli strumenti in cima alla scheda dei risultati Persone. Appare accanto al pulsante Esporta.
3. La Stampa Directory si apre in una nuova scheda del browser e inizia a caricare i dati della tua directory.
4. Dopo circa 1,5 secondi, la finestra di dialogo di stampa del tuo browser apparirà automaticamente.

:::tip
La directory si apre in una scheda separata in modo che tu possa stamparla senza lasciare la pagina Persone. Dopo che la finestra di dialogo di stampa appare, la scheda si sposta indietro automaticamente.
:::

## Cosa Viene Incluso

La directory di stampa include tutti i **membri attivi** della tua chiesa. Specificamente:

- Le persone con uno stato di iscrizione **Inattivo** o **Visitatore** vengono automaticamente escluse.
- Le persone che hanno **rifiutato** dalla directory vengono escluse.
- Tutti gli altri sono raggruppati per **nucleo familiare**.

## Layout della Directory

La directory stampata ha due parti:

### Pagina di Copertina

La prima pagina include:
- Il nome della tua chiesa
- Il titolo "Directory dei Membri"
- L'anno in corso
- La posizione della tua chiesa
- Un conteggio dei nuclei familiari totali inclusi

### Elenchi dei Membri

Le pagine rimanenti elencano i nuclei familiari in ordine alfabetico per cognome. Ogni scheda del nucleo familiare mostra:

- **Foto** -- Miniature circolari per ogni membro del nucleo familiare, con il loro nome sotto
- **Nome display del nucleo familiare** -- Ad esempio, "La Famiglia Smith", "John & Jane Doe" o solo il nome della persona per gli individui
- **Indirizzo** -- Indirizzo stradale, città, stato e codice postale
- **Telefono** -- Numero di telefono di casa o mobile
- **Email** -- Indirizzo email principale
- **Compleanni** -- Formattati come date individuali per ogni membro del nucleo familiare
- **Anniversario** -- Se registrato

I nuclei familiari sono organizzati in **sezioni alfabetiche** (A, B, C, ...) con un'intestazione di sezione decorativa prima di ogni gruppo di lettere. Due schede del nucleo familiare appaiono una accanto all'altra su ogni riga.

## Impostazioni di Stampa

La directory è formattata per **carta di formato standard** (8,5" × 11") con margini di 0,5 pollici. Quando appare la finestra di dialogo di stampa del browser:

- Lascia il formato carta impostato su **Standard**
- Abilita **Grafica di sfondo** (a volte chiamata "Stampa sfondi") nelle impostazioni di stampa del tuo browser se desideri che le intestazioni di sezione si stampino con il loro stile di colore
- L'orientamento verticale è consigliato

:::info
La directory rispetta i salti di pagina — le schede dei nuclei familiari e le intestazioni di sezione non verranno divise tra pagine.
:::
