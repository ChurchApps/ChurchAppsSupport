# Registrazione presenze

<div class="article-intro">

Una volta configurati sedi, orari dei servizi e gruppi, puoi registrare manualmente le presenze dopo ogni incontro. B1 Admin organizza le presenze intorno alle **sessioni** -- una sessione per gruppo per data di incontro. Crei la sessione, segni chi era presente e i dati confluiscono direttamente nei tuoi report sulle presenze.

</div>

<div class="prereqs">
<h4>Prima di iniziare</h4>

- Le sedi, gli orari dei servizi e i gruppi devono essere configurati. Vedi [Configurazione presenze](setup.md) se non l'hai ancora fatto.
- I gruppi che vuoi monitorare devono avere il **Monitoraggio presenze** abilitato. Vedi [Configurazione presenze](setup.md) per i dettagli.

</div>

## Creare una sessione

Una sessione rappresenta un singolo incontro di un gruppo -- ad esempio, la classe bambini K--3a elementare di una specifica domenica.

1. Apri **B1 Admin** e clicca su **Gruppi** nella barra laterale.
2. Seleziona il gruppo per cui vuoi registrare le presenze.
3. Clicca sulla scheda **Sessioni**.
4. Clicca **Nuovo** per creare una nuova sessione.
5. Se il gruppo è assegnato a un orario di servizio, scegli l'**Orario del servizio**. Se è un gruppo non programmato, questo campo non apparirà.
6. Seleziona la **Data della sessione** -- può essere oggi, una data passata o una data futura.
7. Clicca **Salva**.

:::tip
Puoi creare sessioni per date passate per recuperare presenze non ancora registrate, o crearle in anticipo in modo che siano pronte quando il gruppo si riunisce.
:::

## Registrare le presenze

Dopo aver salvato la sessione, i membri del gruppo appaiono sul lato destro della pagina.

1. Spunta la casella accanto a ogni persona che ha partecipato.
2. Le modifiche vengono salvate automaticamente -- non c'è un pulsante Salva aggiuntivo per le presenze.

:::info
Solo i membri attuali del gruppo appaiono nell'elenco della sessione. Se qualcuno ha partecipato ma non è ancora nel gruppo, [aggiungilo al gruppo](../groups/group-members.md) prima, poi registra la sua presenza.
:::

## Esportare le presenze in un foglio di calcolo

Puoi scaricare un record della sessione come file CSV da usare in Excel, Numbers o Fogli Google.

1. Apri la sessione che vuoi esportare.
2. Clicca il pulsante **Scarica CSV**.
3. Apri il file scaricato nella tua applicazione di fogli di calcolo.

## Visualizzare le presenze registrate

Dopo aver registrato le sessioni, i dati appaiono nei tuoi report sulle presenze.

- **Scheda Presenze** -- mostra le tendenze a livello di chiesa nel tempo. Vedi [Monitoraggio presenze](tracking-attendance.md).
- **Scheda Gruppi** -- mostra le presenze suddivise per singolo gruppo.

:::tip
Se una sessione appena creata non appare subito nei report, assicurati che la data della sessione rientri nell'intervallo di date selezionato nei filtri del report.
:::
