# Profilo

<div class="article-intro">

La sezione Profilo ti consente di gestire le impostazioni del tuo account personale in ChurchApps. Il tuo profilo è condiviso tra tutti gli strumenti ChurchApps, quindi le modifiche apportate qui si applicano ovunque -- da B1 Admin all'app mobile B1.

</div>

## Cosa puoi fare

Dalla sezione Profilo, puoi:

- **Aggiornare nome e email** -- Mantieni aggiornate le tue informazioni di contatto.
- **Cambiare la password** -- Imposta una nuova password per il tuo account.
- **Cambiare tema** -- Alterna tra modalità chiara e scura in base alle tue preferenze.
- **Gestire i dispositivi connessi** -- Visualizza e controlla quali dispositivi ChurchAppsPlayer sono collegati al tuo account.
- **Visualizzare gli account collegati** -- Visualizza e gestisci eventuali servizi esterni connessi al tuo account ChurchApps.

## Sotto-sezioni del Profilo

L'area Profilo contiene le seguenti pagine:

1. **[Gestione del Profilo](./managing-profile.md)** -- Modifica le tue informazioni personali, cambia la password, gestisci gli account collegati, regola le preferenze del tema e gestisci l'eliminazione dell'account.
2. **[Gestione Dispositivi](./devices.md)** -- Visualizza, aggiungi e gestisci i dispositivi ChurchAppsPlayer associati al tuo account.

:::tip
Mantieni aggiornato il tuo indirizzo email. Viene utilizzato per il recupero dell'account e per le notifiche importanti dalla tua chiesa.
:::
