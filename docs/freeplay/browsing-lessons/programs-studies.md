# Programs and Studies

The lesson browser uses a step-by-step flow to narrow down to the exact content you want to play. Each step displays a 3-column grid of cards with images and titles.

## Browsing Programs

1. Open the **Browse Programs** screen
2. Programs appear as a grid of image cards, three across
3. Use the directional pad on your remote to highlight a program
4. Press **Select** to open it

Each program card shows the program's cover image. If no image is available, a placeholder card is displayed.

## Selecting a Study

After choosing a program, the **Select a Study** screen loads with all studies in that program.

1. Study cards appear in a 3-column grid with cover images and study names
2. Navigate to the study you want and press **Select**
3. Press **Back** to return to the program list

## Choosing a Lesson

After selecting a study, the **Select a Lesson** screen displays all lessons in that study.

1. Lesson cards show cover images and lesson titles
2. Navigate to the lesson you want and press **Select**
3. Press **Back** to return to the study list

## Picking a Venue

The lesson details screen shows the lesson name, title, and description overlaid on the lesson's cover image. Below the description, you will see one or more venue buttons.

Venues represent different versions of the same lesson. For example, a lesson might have separate venues for "Preschool," "Elementary," and "Pre-Teen" age groups. Each venue contains the media files appropriate for that setting.

1. Use the directional pad to highlight the venue you want
2. Press **Select** to begin loading the content
3. FreePlay fetches the venue's playlist and starts the player

:::tip
If you only see one venue button, there is just one version of the lesson available. Select it to start playing immediately.
:::

## Navigating Back

At any point in the browsing flow, press **Back** on your remote to return to the previous level:

- From the lesson details screen, **Back** returns to the lesson list
- From the lesson list, **Back** returns to the study list
- From the study list, **Back** returns to the program grid
- From the program grid, **Back** opens the sidebar
